Combined Optimization Scheme for Degree Distributions of LDPC Codes

Jin Soo Park School of Electrical and Electronic Engineering
Yonsei University Seoul, Republic of Korea Email: js.park09@yonsei.ac.kr

Min Kyu Song School of Electrical and Electronic Engineering

Abstract—In this paper, we propose an optimization scheme for degree distributions of LDPC codes that combines differential evolution and iterative simplex algorithm. In the proposed scheme, we find good check and variable nodes degree distribution and threshold by using differential evolution, and then, we further optimize check and variable nodes degree distribution that has enhanced code rate by using an iterative simplex algorithm. An iterative simplex algorithm consists of two simplex algorithms, optimizing ρ(x) and λ(x), respectively and iteratively. Some simulation results show that the proposed scheme finds degree distributions, better than those obtained by the differential evolution only, in terms of threshold and/or code rate. As an application of the proposed scheme, we report some new degree distributions with some new maximum degrees.
I. INTRODUCTION
From the discovery of turbo codes, iterative decoders have attracted a lot of attention. As an example of the attention, LDPC codes, first discovered by Gallager [1], were rediscoverd by Mackay et al. [3], and there have been much results analyzing the performance of the LDPC codes. Threshold phenomenon, that means we can achieve an arbitrary small bit-error probability for any noise level smaller than certain value, is very important. This phenomenon is first observed by Gallager for binary symmetric channels (BSCs) [1], [2], and generalized by Luby et al. [5], Richardson and Urbanke [6].
An LDPC code is specified with variable and check nodes degree distribution and their connection. The degree distribution is a combination of variable and check nodes degree distributions λ(x) and ρ(x), respectively. The design of an LDPC code generally has two steps, ‘code design’ and ‘code construction’ [7]. In the code design step, we should select a good ensemble of LDPC codes by determining variable and check nodes degree distribution. If a degree distribution is given, then we can construct a bipartite graph or realize a parity check matrix using well known construction methods such as PEG [10], and ACE [11]. Here, the much more important step of the designing an LDPC code is the finding a good degree distribution.
A degree distribution can be analyzed by density evolution [8] which uses the expected behavior of an ensemble of LDPC codes with the cycle-free and infinitely long length conditions. The key idea of density evolution is to track the densities of the messages flowed in the Tanner graph of LDPC code

during the decoding stage. By using density evolution [8], the optimization of the degree distribution can be changed to a non-linear optimization problem, and then, the degree distribution can be optimized using differential evolution [12], which has been a well-known non-linear program solver. But obtaining the densities of the messages requires high computational complexity that is a serious problem in practice. Thus, Chung et al. [9] proposed Gaussian approximation to reduce the computational complexity over BI-AWGN channels. In the Gaussian approximation, the message densities are assumed to be Gaussian for the convenience of calculation and the optimization problem of the degree distribution is converted to a linear programming which can be solved by simplex algorithm.
In this paper, we propose an optimization scheme for degree distributions of LDPC codes that combines differential evolution and iterative simplex algorithm. Some limited information for optimizing the degree distribution is given in two papers [8] and [9], from which one can hardly implement the process correctly. On the other hand, the proposed scheme in this paper is straightforward and easily implementable. Further more, the proposed scheme gives good degree distributions and the results show better performance than those in [8].
This paper is organized as follows. In Section II, we review the LDPC code design problem and two optimization schemes are described which minimize the threshold or maximize the code rate. In Section III, we propose a new optimization scheme and compare the results using the scheme with some well-known degree distributions and differential evolution only. Conclusion is given in Section IV.

B. Differential evolution for threshold optimization
In this part, we describe the differential evolution which optimizes the threshold by tuning the degree distribution [8] [12]. It is suitable for non-linear optimization and finds hopefully an optimum solution. The first step is gathering the initial population. They are chosen randomly over the entire parameter space and they should have the given initial code rate. Let xn be the target degree distributions which are the members of the population where n is the index of each target degree distribution and xn is concatenated form expressing check and variable degree distribution together. The thresholds of the target degree distributions are calculated by Gaussian approximation, and we can select the best degree distribution xbest of the target degree distributions. The next step is called mutation. For each target degree distribution, a mutated degree distribution is generated by the following relation:
xbest + (xni − xnj ),
where ni and nj are random indices. And this degree distribution must be normalized. To decide whether or not each mutated degree distribution should become a member of population, each target degree distribution is compared with the corresponding mutated degree distribution. If the mutated degree distribution has smaller threshold than that of the corresponding target degree distribution, then the mutated degree distribution becomes a member of population and the

Thus we can think of a scheme that optimizes R by tuning ρ(x) and λ(x) iteratively for given s∗ by using some linear program solver. This scheme is shown in Fig. 2. D. Application of the iterative simplex algorithm into the code rate enhancement
As mentioned the Section II-C, we should choose ρinit(x) carefully to use proposed code rate enhancement scheme. To analyze the performance of the code rate enhancement by simulation, we choose ρinit(x) from some well-known degree distributions that found by differential evolution in [8], and enhance its code rate with the iterative simplex algorithm for code rate enhancement. From Table II, we can see that (Eb/N0)∗ becomes smaller than well-known degree distribution by the effect of increased code rate for given (Es/N0)∗ = s∗/4. Note that (Es/N0)∗ = R × (Eb/N0)∗.
III. NEW LDPC CODE DESIGN SCHEME COMBINING
DIFFERENTIAL EVOLUTION AND THE ITERATIVE SIMPLEX ALGORITHM
From the results of Section II-D, we can observe that (Eb/N0)∗ will be decreased via the iterative simplex algorithm for code rate enhancement. Thus we can think of a new LDPC code design scheme in Fig. 4. At the first stage in this scheme, we find a good ρ(x) and λ(x) pair, that has low threshold, by using differential evolution in Section II-B. Then, we will enhance the code rate by using the iterative simplex algorithm presented in Section II-C. The result is ρopt(x) and λopt(x) that has enhanced code rate with the same (Es/N0)∗.
This scheme is suitable for any appropriate parameters, so we find some new degree distributions by using proposed new LDPC code design scheme. Table III shows some new degree distributions with dv = 7, 10, 12, 14, 15.
IV. CONCLUSION
In this paper, we propose an iterative simplex algorithm for the code rate enhancement that tunes ρ(x) and λ(x) iteratively. The optimized results of the code rate enhancement scheme show better (Eb/N0)∗ than those obtained by differential evolution only. And we also propose a new LDPC code design scheme, that uses differential evolution for finding good ρ(x) and s∗ and an iterative simplex algorithm for code rate enhancement to enhance code rate.
