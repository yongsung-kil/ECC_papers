2013 8th International Conference on Communications and Networking in China (CHINACOM)
The modified reliability-based iterative majority-logic decoding algorithm with non-uniform quantization

Haiqiang Chen, Lingshan Luo, Youming Sun, Tuanfa Qin and Yunyi, Liu The School of Computer, Electronics and Information, Guangxi University, Nanning, 530004, China. E-mail: haiqiang@gxu.edu.cn

Abstract—This paper investigates the performances of the modified reliability-based iterative majority-logic decoding (MRBI-MLGD) algorithm under different quantization schemes. An efficient non-uniform quantization method is presented for the MRBI-MLGD algorithm. With the non-uniform strategy, the real received symbols will be quantized into integer values before passing to the decoder. The quantized reliability measures exhibit a non-uniform distribution with respect to their original received signals. Furthermore, the quantization resolution can be modified as required. Simulation results show that, when combined with factor correction techniques and a proper quantization parameter r, the MRBI-MLGD with nonuniform quantization performs well with very small quantized bits (3-4 bits) and has very fast decoding convergence speed.
I. INTRODUCTION
Low-density parity-check (LDPC) codes (sometimes called Gallager codes) were first introduced by Gallager in his doctoral dissertation[1]. The research of LDPC codes mainly focuses on construction/design, performance analysis and decoding techniques. At present, there exist two commonly used construction methods for LDPC codes: random construction aided by computer searching [2][3] and systematic construction based on algebraic structures [4][5]. Compared with randomly constructed LDPC codes, the algebraically constructed LDPC codes have some advantages: 1) some of them are cyclic or quasi-cyclic and can be encoded in linear time with the simple feedback shift registers [6]; 2) they perform as well as randomly generated LDPC codes especially at short to moderate block lengths; 3) they are majority-logic decodable LDPC codes. The third property implies that the parity check matrices usually have high column weights, which provides the potential of designing low-complexity majority logic decoders. In this paper, we focus on two classes of binary structured (γ, ρ) regular LDPC codes (with constant row weight ρ and column weight γ), which are constructed based on the properties of finite geometry [4] and finite fields [5].
The decoding algorithms for LDPC codes can be categorized into three general classes:1) hard-decision decoding; 2) probability-based decoding; and 3) reliability-based decoding(also known as hybrid decoding). From an implementation point of view, hard decision decoding algorithms have the lowest complexity, such as the one-step majority-logic decoding (OSMLGD) algorithm [4] and the bit-flipping (BF) decoding algorithm [7]. However, the lowest complexity may cause serious performance loss. The sum-product algorithm (SPA) [8] is the most well-known probability-based iterative

decoding algorithm which provides the best performance but requires high computational complexity due to floating point computations.
Recently, the reliability-based decoding algorithms have attracted considerable attention since they can make efficient trade-offs between decoding performance and computational complexity. The commonly used reliability-based algorithms are the weighted bit flipping (WBF) algorithm [4] and its improved versions [9],[10]. In 2009, Huang et al proposed a reliability-based iterative majority-logic decoding (RBIMLGD) algorithm [11] and a weighted iterative soft-reliability based majority-logic decoding (WISRB-MLGD) [12], which perform well with very low decoding complexity.
In this paper, we present a non-uniform quantization method for the modified reliability-based iterative majoritylogic decoding (MRBI-MLGD) algorithm (presented in our previous work [13][14]). Using a well-defined quantized function, the real received symbols will be transformed into integer reliability values which have a non-uniform distribution. Simulation results show that, when combined with factor correction techniques and a proper non-quantization parameter r, the MRBI-MLGD with non-uniform quantization performs well with very small quantized bits (3-4 bits) and has very fast decoding convergence speed.
II. PRELIMINARY
A. Basic Concepts and Notation
An LDPC code C can be defined as the null space of a sparse parity-check matrix H = (hi,j)m×n. Associated with the parity-check matrix is the Tanner graph [15]. A Tanner graph is a bipartite graph which contains two types of node, the variable node Vj, (0 ≤ j < n) and the check node Ci, (0 ≤ i < m). The variable node corresponds to a column of H and the check node corresponds to a row of H. For any paritycheck matrix, we define the following two sets
Ni = {j : 0 ≤ j ≤ n − 1, hi,j = 1}
for each row i of H and
Mj = {i : 0 ≤ i ≤ m − 1, hi,j = 1}
for each column j of H.

E. Summary of the Algorithm
The reliability-based decoding algorithm for majority-logic decodable LDPC codes with non-uniform quantization can be summarized as follows.
Algorithm:
• Input the received vector y, quantization parameters r, b, the scaling factor α and the maximum iteration number Imax.
• Initialization: quantize the vector y into q according to (1); set the loop counter k = 0; for 0 ≤ j ≤ n − 1, set Rj(0) = qj ;
• Iterations: while k < Imax, do the following steps.
1) for 0 ≤ j ≤ n − 1, make the decision zj(k) = 1 if Rj(k) ≥ 0 and zj(k) = 0 otherwise;

IV. SIMULATION RESULTS
Example 1: We consider the (961, 721) regular QC-LDPC code [5], which is constructed based on finite fields. The parity-check matrix of this code has column and row weights γ = ρ = 30 (majority-logic decodable). The parameters for the MRBI-MLGD algorithm with non-uniform quantization are list as follows: 1)for 3-bit non-uniform quantization, r is set to be 0.745 and α is set to be 0.5; 2)for 4-bit non-uniform quantization, r is set to be 0.88 and α is set to be 1.0. For the purpose of comparisons, we also give the performances of this code decoded by the RBI-MLGD algorithm [11] and MRBIMLGD algorithm [13] and the sum-product algorithm. For all the decoding algorithms, the maximum iteration number is set to be 30. The simulation results are shown in Fig. 2. We have the following observations.
• With 4-bit non-uniform quantization, the MRBIMLGD algorithm achieves a performance gain about 0.3 dB over the RBI-MLGD algorithm around BER = 10−4;
• The MRBI-MLGD algorithm with 4-bit non-uniform quantization performs as well as the MRBI-MLGD algorithm with 8-bit uniform quantization;
• For the MRBI-MLGD algorithm with non-uniform quantization, the performance of the 3-bit quantization is close to that of the 4-bit quantization.
Fig. 3 shows the average numbers of iterations required by the decoding algorithms. We see that the MRBI-MLGD algorithm with 4-bit non-uniform quantization converges as fast as the MRBI-MLGD algorithm with 8-bit uniform quantization. Even with 3-bit non-uniform quantization, the MRBI-MLGD algorithm shows a very fast decoding convergence rate (only requires 2.7 iterations on average at Eb/N0 around 4.0 dB).

V. CONCLUSION
We point out in this conclusion section that, there is not always a monotonic performance improvement with the increased quantization bits. That is, the non-uniform quantization MRBI-MLGD algorithm obtains very little or negligible performance gain when working at other quantization bit larger

than 4. Therefore, taking into account the decoding complexity and the memory load, 3 or 4 non-uniform bits may be more attractive in practice.
