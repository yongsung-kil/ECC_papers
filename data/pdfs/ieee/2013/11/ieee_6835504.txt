Abstract-The construction of low-density parity codes(LDPC) is 
one of the hottest fields in the research of LDPC codes. In this 
paper, we propose a quasi-cyclic(QC) construction method of 
parity-check matrix of LDPC codes, named block-PEG method. 
In this method, a basic matrix is constructed firstly, then the 
none-zero elements in the matrix are extended by circulant 
permutation matrices, and the parity-check matrix is obtained at 
last. The simulation results indicate that the performances of 
LDPC codes constructed by block-PEG method are similar with 
that of PEG algorithm. In linear-time, block-PEG can generate 
LDPC codes with similar performance of the PEG-generated 
codes.

1. 
INTRODUCTION 
Low-density parity-check codes(LDpC)[I] is a special kind 
of linear block codes. A parity-check matrix of LDPC codes is 
composed of a large number of Os and a few number of Is, 
that is, they are very sparse. Encoding of LDPC codes can be 
carried out by the basic method of linear block codes. In this 
method, we need calculate generation matrix by Gauss 
elimination firstly, and then multiply the generation matrix 
with the information sequence to get the codeword. For 
usually both the size of parity-checking matrix and the size of 
non-sparse generation matrix are quite large, the encoding 
method is costly. Richardson proposed an efficient encoding 
method[2] which separates the parity-check matrix into six 
parts and obtains a codeword by matrix manipulation instead 
of by generation matrix. Although computational complexity 
is decreased compared to traditional method, it is still high. 
QC-LPDC codes is a simple and practical codes which is 
described in the thesis [3] and has been researched wideli4-6]. 
Recently Researchers proposed several new methods to 
construct LDPC codes. These methods can be divided into two 
main types: 1) random or pseudorandom constructions, and 2) 
algebraic constructions. Algebraic constructions of LDPC 
codes have two interrelated categories: the first is based on 
finite geometry, and the second is based on circulant 
permutation. Tn fact, the two categories are not independent, 
but interrelated. 
Tn this paper we propose a block-PEG method which is a 
pseudorandom construction method to construct QC-LDPC

(1) Confinn the size of parity check matrix, the size of 
basic matrix M(H) m and n, and the dimension L of circulant 
permutation matrix P. 
(2) Construct a basic matrix M(H) by PEG algorithm. 
(3) Let the values of E(H) corresponding to none-zero 
elements in M(H) be zeros and corresponding to zero 
elements in M(H) be 00.

IV. 
PERFORMANCE SIMULATAION 
We simulate the codeword bit error performances 
constructing by Gallager, Mackay, PEG and block-PEG 
methods by computer, with 1/2 code rate, row weights 6, 
column weight 3, using belief propagation(BP) decoding 
algorithm. The performances of (1152, 576) code and (4000, 
2000) code are presented in Fig.5 and Fig.6, respectively. The 
girths of PEG code and block-PEG code are both eight. 
From Fig.5 and Fig.6, Gallager codes have poor bit error 
performances and are not practical in engineering applications. 
The reason is that 4-length rings exist in the parity-check 
matrix of Gallager codes. If we would like to use them, low 
length rings in the parity-check matrix have to be eliminated. 
The performances of Mackay random codes are followed by 
Gallager codes. The bit error rate performances of PEG and 
block-PEG codes are the highest and similar, and the later 
encoding complexity is lower for linear-time encoding. 
V. 
CONCLUSION 
Because cyclic or QC LDPC codes can be encoded in 
linear-time, constructing this kind of LDPC codes is very 
important. In this paper, a block-PEG method to construct 
quasi-cyclic parity-check matrix is proposed. We construct a 
basic matrix first, and then get parity-check matrix by 
extending it according some special rules. This kind of LDPC 
codes is a member of quasi-cyclic family. The simulation 
results show that the bit error performances of LDPC codes 
constructing by block-PEG and PEG methods are similar. The 
advantage of the former is that it can achieve linear-time 
encoding using circulate shift registers.
