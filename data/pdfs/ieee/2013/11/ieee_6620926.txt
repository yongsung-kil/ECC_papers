Unequal Diversity LDPC Codes for Relay Channels
Paola Pulini, Student Member, IEEE, Gianluigi Liva, Member, IEEE, and Marco Chiani, Fellow, IEEE

Abstract—A novel protograph-based construction of lowdensity parity-check (LDPC) codes for the relay channel is proposed, which provides an enhanced unequal error protection property named unequal diversity. The focus is on quasi-static fading channels and on the high-code-rate (R > 1/2) regimes, for which (according to the Singleton bound) no full diversity can be achieved. In the proposed construction, some nodes (and the corresponding codeword fragment) associated with the code graph enjoy the diversity provided by the relay, whereas the remaining nodes do not experience any diversity. The proposed approach can be thus tailored to transmit information blocks with different priority levels. An extrinsic information transfer (EXIT) analysis is developed, which allows an accurate performance prediction over the considered channel model, and more in general over block-fading channels.
Index Terms—LDPC codes, relay channel, protographs, extrinsic information transfer charts.
I. INTRODUCTION
T HE design of low-density parity-check (LDPC) codes [1] is subject of a renewed interest which stems from the application of coding techniques to wireless block-fading and relay channels [2]–[7].
In this paper, we tackle the problem of analyzing the performance of a high-rate (R > 1/2) distributed LDPC coded scheme for coded cooperation [8], [9] over a quasi-static relay fading channel. The peculiarity of the proposed scheme resides in the so-called unequal diversity (UD) achieved by the different codeword fragments. More specifically, we design distributed LDPC codes which allow achieving full diversity (diversity-2) for a single relay for some codeword fragments, whereas the remaining codeword fragments do not enjoy diversity. The proposed scheme thus trades diversity with coding rate by re-encoding at the relay a subset of the information bits encoded at the source. The proposed scheme is hence of interest for any communication system for which the unequal error protection (UEP) concept is relevant. Full-diversity LDPC codes, referred to as root LDPC codes, where introduced in [5]. Root LDPC codes represent a powerful class of protograph-based LDPC codes, designed to cope with channels affected by block fading. In particular, root LDPC codes provide full diversity under iterative decoding, thus they achieve the outage probability limit on block erasure channels and they approach the limit on the Rayleigh block fading channel [5]. The result is obtained by introducing a specific macroscopic structure in the code parity-check matrix, which resembles the proto-matrix of a protograph LDPC code. In this view, the codes introduced in this contribution are strongly related to root LDPC codes, being both based on structured (protograph-based) Tanner graphs to guarantee a target diversity on block fading channels. With respect to [5], the proposed design targets different diversity levels for different information block parts (i.e., different protograph variable nodes).
The PEXIT analysis has been previously adopted to design bilayer [4] protographs for the relay channel [7]. Nevertheless, in [7] the authors derive via PEXIT analysis the iterative decoding threshold over additive white Gaussian noise (AWGN) channel only for the designed protographs, focusing on the different point-to-point links and hence without any modification of the original analysis of [12], [13]. We extend the definition of outage regions of a code ensemble [3] to distributed protograph ensembles and to protograph variable nodes. Examples of distributed protograph ensembles are provided, with special attention to the high coding rate regime (R > 1/2), where according to the Singleton bound [3] diversity of order 2 cannot be achieved. For this case, ensembles exhibiting an appealing UEP properties are introduced. More specifically, some variable nodes (VNs) of the corresponding protograph achieve a diversity 2, whereas other nodes present a bit error rate (BER) slope which is typical of a scheme without diversity. II. TRANSMISSION SCHEME
Following Figures 1(a) and 1(b), a rate-RS code CS is employed at the source (S) to encode a length-k message vector u which can be split into two sub-blocks (fragments, in the following) as u = [uh|ul], where uh represents the kh bits long high-priority fragment and ul represents the kl bits long low-priority fragment. The length-nS coded block is given by cS = [u|pS], where pS is the mS = nS −k bits parity vector. The coded block is then BPSK modulated, resulting in the modulated sequence xS (with xS ∈ {−1, +1}nS), and transmitted over the first time slot. A corrupted version of xS is received at the destination (D) as
yS = αSxS + nS,
where αS follows a Rayleigh distribution with E[α2S] = 1, and where the elements nS are i.i.d. Gaussian random variables with variance σn2 = N0/2. The instantaneous signal-to-noise ratio (SNR) is given by γS = α2S/2σn2 .
The same sequence xS is received at the relay (R). We will make use of a simplified assumption, i.e. we will consider the link between source and relay as reliable [6]. III. UNEQUAL DIVERSITY PROTOGRAPHS
The code structure presented in the previous section can be suitably realized by means of protograph LDPC code constructions. A protograph [10], [13], [18] is a Tanner graph [19] with a relatively small number of nodes. A protograph G = (V , C , E ) consists of a set of N variable nodes V , a set of M check nodes C , and a set of edges E . Each edge ei,j ∈ E connects a variable node Vj ∈ V to a check node Ci ∈ C . Multiple parallel edges are permitted. A larger graph can be obtained by a copy-and-permute procedure: the protograph is copied q times, and then the edges of the individual replicas
