Bounds on the Size of Parity-Check Matrices for Quasi-Cyclic Low-Density Parity-Check Codes
Kyung-Joong Kim, Jin-Ho Chung, Member, IEEE, and Kyeongcheol Yang, Senior Member, IEEE

Abstract—In this paper, we investigate the cycle properties of quasi-cyclic low-density parity-check (QC-LDPC) codes. Using the sequence representation of a parity-check matrix for a QC-LDPC code, we analyze a necessary and sufficient condition for a cycle of a given length to exist. We then derive bounds which are necessary conditions for a QC-LDPC code to have a given girth in terms of its parameters. We also give a bound which is a sufficient condition for a QC-LDPC code of a given girth to be constructed by a greedy algorithm. The bounds derived here are applicable to any regular or irregular QC-LDPC codes as well as they improve the existing bounds in many classes of regular LDPC codes.
Index Terms—Cycle, girth, low-density parity-check (LDPC) codes, quasi-cyclic codes.
I. INTRODUCTION
L OW-DENSITY parity-check (LDPC) codes, first introduced by Gallager [1], have attracted considerable attention due to their Shannon-limit-approaching performance since the work by MacKay and Neal [2]. As a result, there have been many developments in the construction of LDPC codes. In particular, Richardson et al. introduced density evolution to design an ensemble of LDPC codes that perform extremely close to the Shannon capacity when the code length goes to infinity [3]. However, the finite-length performance of LDPC codes cannot be predicted well by density evolution, due to their heavy dependency on cycles, stopping sets, or trapping sets. For example, short cycles in the Tanner graph [4] of an LDPC code may induce a severe performance degradation. For this reason, several researchers made many efforts to obtain LDPC codes without short cycles [5]–[12].

An interesting class of LDPC codes is quasi-cyclic LDPC (QC-LDPC) codes based on circulant permutation matrices (CPMs) [6]–[18]. They require a smaller amount of memory for storing their parity-check matrices as well as their algebraic properties are easier to analyze than those of random LDPC codes. This motivates us to take an intensive interest in constructing QC-LDPC codes without short cycles.
QC-LDPC codes without short cycles have been extensively studied [6]–[14]. Several methods to construct QC-LDPC codes without short cycles based on an algebraic approach or a finite geometry were presented in [6]–[8]. Fossorier [9] presented a necessary and sufficient condition for a cycle to exist in a QC-LDPC code. Based on this condition, he gave some necessary conditions for a QC-LDPC code to have girth 6 or 8, when its parity-check matrix consists of only CPMs. Recently, Karimi and Banihashemi [10] presented necessary conditions for a QC-LDPC code to have girth 6, 8, or 10, in a QC-LDPC code whose parity-check matrix contains zero matrices as well as CPMs. They also gave bounds on the size of parity-check matrices for regular QC-LDPC codes with girth 6, 8, or 10, when their parity-check matrices consist of only CPMs. In order to construct general QC-LDPC codes whose parity-check matrices contain zero matrices as well as CPMs, simple algorithms [11], [12] based on the progressive edge-growth (PEG) algorithm [5] or extension algorithms using lifting methods [13], [14] were presented. However, no meaningful bounds on the parameters of a general QC-LDPC code of a given girth have been given. Therefore, it is a challenging problem to develop a meaningful bound on the size of a parity-check matrix for a regular or irregular QC-LDPC code of a given girth.
In this paper, we derive bounds which are necessary conditions for a QC-LDPC code to have a given girth, whose paritycheck matrix consists of zero matrices or CPMs. We also give a sufficient condition under which a QC-LDPC code of a given girth can be constructed by a greedy algorithm. The bounds derived here improve the existing bounds in many classes of regular LDPC codes. Furthermore, our results can be applied to any regular or irregular QC-LDPC codes, unlike those in [5], [9], and [10].
The outline of the paper is as follows. In Section II, we describe a sequence representation of a parity-check matrix for a QC-LDPC code and analyze a necessary and sufficient condition for a cycle of a given length to exist. In Section III, we give bounds which are necessary conditions for a QC-LDPC code to have a given girth, and present a sufficient condition for a QC-LDPC code of a given girth to be constructed by a greedy algorithm. The necessary conditions are proved in Section IV, while the sufficient condition is proved in Section V. Finally, we give concluding remarks in Section VI.
