AbstractпЂ­ Several attractive classes of LDPC codes including quasi-cyclic (QC), convolutional, staircase, and spatially-coupled codes are described. We show that these classes of LDPC codes can be derived from QC-LDPC code design. We demonstrate that record coding gains can be obtained from properly designed irregular QC-LDPC codes.

I. INTRODUCTION
The exponential internet traffic growth projections have placed enormous transmission rate demand on the underlying information infrastructure at every level. As the response to these never ending demands for higher data rates, 100 Gb/s Ethernet (100 GbE) has been already standardized, and effort has moved to 400GbE/1TbE (see [1] and references therein). It has become evident that soft-decision FEC represents one of the key enabling technologies for the next generation of optical transport networks.
In this invited paper, we describe several attractive classes of LDPC codes including quasi-cyclic (QC) [2], convolutional [3], staircase [4], and spatially-coupled [5] LDPC codes. We show that various classes of LDPC codes, listed above, can be derived from QC-LDPC code design. These various classes of LDPC codes allow for greater flexibility in avoiding the cycles of short length compared to QC-LDPC code design itself. On the other hand, similar flexibility can be obtained from irregular QC-LDPC codes in which irregular structure can be precisely controlled. One such irregular QC-LDPC code design, based on combinatorial objects known as the pairwise block designs (PBDs), is described in [6]. We demonstrate later in the paper that record coding gains can be obtained using irregular QC-LDPC codes derived from PBDs.

V. CONCLUSIONS
Several attractive classes of LDPC codes including QC, convolutional, staircase, and spatially-coupled codes have been described. It has been shown that these classes of LDPC codes can be derived from QC-LDPC code design and allow for more flexibility in LDPC code design. On the other hand, similar flexibility can be obtained from irregular QC-LDPC codes. It has been demonstrated by simulations that record coding gains can be obtained from properly designed irregular QC-LDPC codes.
