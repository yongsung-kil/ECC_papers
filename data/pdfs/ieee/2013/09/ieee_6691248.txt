Design of Masking Matrix for QC-LDPC Codes
Yang Liu and Ying Li State Key Lab of ISN, Xidian University, Xi’an, China
Email: xdyanger@126.com, yli@mail.xidian.edu.cn

Abstract—The masking matrix plays an important role in constructing new classes of regular and irregular quasi-cyclic low density parity check (QC-LDPC) codes. By coupling two identical graphs in a special way, we present a new structure of the masking matrix, whose Tanner graph can be seen as a protograph. From this perspective, we propose a Gaussian Approximation algorithm for protograph-based LDPC codes to analyze the asymptotic performance of this class of codes. It is shown that, although the proposed structure of the masking matrix has almost the same convergence threshold as the conventional one produced randomly by progressive edge growth (PEG) algorithm, the former converges faster than the latter. Simulation results illustrate that the QC-LDPC code constructed by the proposed structure of the masking matrix has about 0.2dB coding gains compared with the conventional one.
Index Terms—masking matrix, protograph LDPC, Gaussian approximation, convergence rate
I. INTRODUCTION
Quasi-cyclic low-density parity-check (QC-LDPC) code is one of the most promising classes of structured LDPC codes due to its excellent performance over noisy channels and ease of implementation [1]. Authors in [2] show that the theory of finite fields can be successfully used to construct algebraic LDPC codes. And plenty of methods are presented by them to construct QC-LDPC codes. The proposed codes are based on either multiplicative groups or additive groups of finite fields. In [3], the authors propose an approach to construct QC-LDPC codes based on Latin squares over finite fields. Then a matrix theoretic approach for studying QC-LDPC code based on matrix transformations is developed in [4]. Most of the algebraic constructions in the previous works result in regular QC-LDPC codes. The irregular QC-LDPC codes can be constructed by the masking technique. To the best of our knowledge, most of the irregular masking matrices are constructed by computer search based on the variable-andcheck node degree distributions. There are no works focusing on the design of structured masking matrix.
In recent years, spatially coupled codes have received notable attentions due to their excellent performances for a myriad of channels and communication problems [5]. These kinds of codes are constructed by coupling a sequence of identical small structured graphs into a graph chain, together with appropriate boundary conditions. Inspired by this concept, we present a new structure of the masking matrix by coupling two identical small graphs in a special way. To analyze the asymptotic performance, a Gaussian Approximation (GA) algorithm for protograph-based LDPC codes (named as PGA) is proposed to predict the optimistic value of the convergence threshold. It is shown that the proposed structure of the masking matrix has almost the same convergence threshold as the

conventional one produced randomly by PEG algorithm. Both kinds of masking matrix have the same degree distributions. To analyze the convergence rate, we average the means of the loglikelihood ratio of the messages (m − LLRs) over all variable nodes of the masking matrix during the iterative decoding. The results show that the average of m − LLRs of the new masking matrix is always larger than that of the conventional one produced by PEG algorithm [6]. To some extension, this can be explained why the QC-LDPC code constructed by the new masking matrix converges faster than that by the conventional one. Simulation results illustrate that the QC-LDPC code constructed by our proposed structure of the masking matrix has about 0.2dB coding gains.
II. DESCRIPTION OF THE MASKED QC-LDPC CODES
Regular QC-LDPC codes are generally constructed by the algebraic methods based on row-column (RC)-constrained arrays of circulant permutation matrices [2]. The density of an array can be reduced by replacing a set of circulant permutation matrices by zero matrices. The replacement of circulant permutation matrices with zero matrices is referred to as masking which is very effective for constructing regular and irregular QC-LDPC codes. In this section, we first introduce the construction of QC-LDPC codes by masking.

B. Structure of the Base Matrix

In the previous works, there are many different forms of base matrices which are used to construct algebraic QC-LDPC codes over GF (2) using the masking technique [1]- [4]. In this paper, we mainly consider two forms of base matrices. The first type is given in the form of Vandemonde matrix which is a conventional parity-check matrix of a Reed-Solomon (RS) code in terms of the roots of its generator polynomial in a special case [1]. And the base matrix of the second type is a matrix whose entries are given as differences (or sums) of two field elements. The fields for both types of code construction are restricted to be of characteristic 2, i.e., extension fields of GF (2). For simplicity, we denote these two types of base matrices as type-1 and type-2 respectively. Their structures are given in the following.

C. Dispersion
For each nonzero element αi over GF(q) with 0 ≤ i < q−1, define a (q − 1) tuple over GF(2), z(αi) = (z0, z1, · · · , zq−2), where the ith component zi = 1 and all the other q − 2 components are set to zero. z(αi) is called the M-locationvector of αi in [1]. The M-location-vector z(0) of the 0 element of GF(q) is defined as the all-zero (q − 1)-tuple. By right cyclic-shifting M-location-vectors z(αi), we can have a (q − 1) × (q − 1) matrix over GF(2), which is referred to as the (q − 1) − f old dispersion matrix of the field element over
GF(2).

Using this algorithm, more than one new masking matrices can be obtained. In the following section, we will analyze the structure of the masking matrix by the proposed PGA algorithm for the protograph-based LDPC codes. It will be found that the new masking matrix Znew converges faster than the original one Zorig.
IV. GAUSSIAN APPROXIMATION FOR PROTOGRAPH-BASED LDPC CODES
Protograph-based LDPC codes are a subclass of multiedge type LDPC codes, where more than one edges are permitted connecting one variable node to one check node. To analyze this class of codes, PEXIT algorithm is presented in [7]. In this paper, we propose the Gaussian Approximation (GA) algorithm to analyze the protograph-based LDPC codes [8]. Similarly, since the ensemble of protograph-based LDPC codes is different from a conventional LDPC code ensemble, the conventional GA algorithm must be modified in order to accurately predict the performance of the protograph-based LDPC codes.
Here we consider a protograph with a set of ρ variable nodes V, a set of γ check nodes C, and a set of edges E. Each edge ei,j ∈ E connects a variable node Vj ∈ V to a check node Ci ∈ C.

V. SIMULATION RESULTS

In this section, we construct QC-LDPC codes using the masking technique stated in Section II. Firstly, we generate the base matrices in the form of either Vandemonde or Latin square, corresponding to Type-1 and Type-2 respectively. Then masking the base matrix with Z(γ, ρ) yields the masked base matrix:

M(γ, ρ) = Z(γ, ρ) ⊗ H(γ, ρ) = [zij hi,j ]0≤i<γ,0≤j<ρ

(14)

where the masking matrix Z(γ, ρ) can be obtained either by searching with PEG algorithm or by searching in the form of (6). Finally, we disperse M(γ, ρ) and obtain a γ×ρ array Hdisp of circulant permutation and zero matrices of size (q−1)×(q− 1). Then Hdisp is a γ(q −1)×ρ(q −1) matrix with column and row weight distribution close to the given degree distributions. The null space of Hdisp gives an irregular (γ(q − 1), ρ(q − 1)) QC-LDPC code with rate 1 − γ/ρ.
