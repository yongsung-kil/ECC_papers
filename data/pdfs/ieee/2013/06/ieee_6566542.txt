Performance Analysis of QC-LDPC Construction Based on Distance Graph

Lixin LI, Meng ZHU, Fan YANG School of Electronics and Information Northwestern Polytechnical University
Xi'an, China

Yang ZHANG
School of Automation Northwestern Polytechnical University
Xi'an, China

Abstract—According to the fact that the random construction method of LDPC(Low Density Parity Check) codes are lack of structure and are difficult to be implemented in hardware, two construction methods of QC-LDPC(Quasi-Cyclic LDPC) codessequential and random search method based on distance graph are proposed. The two algorithms can produce flexible QCLDPC codes by setting the parameters such as code length, rates and girth according to requirement. The simulation shows that: the random search method performs better than the sequential search method and the QC-LDPC codes constructed by random search method have equivalent BER performance with PEG construction method. Compared with the existing random construction method, the two methods proposed are faster, and can obtain quasi-cyclic codes which are easy to be implemented in hardware.
Keywords—distance graph; QC-LDPC; search algorithm; girth
I. INTRODUCTION
Since LDPC (Low Density Parity Check) codes were found, many experts and scholars have been dedicated to finding good construction algorithms, which requires that the resultant LDPC codes not only have excellent decoding performance, but also achieve low hardware implementation complexity. There are two types of LDPC construction methods, one is random construction, the other is structured construction method. Compared with structured code, random code has a better decoding performance, especially in case of longer code length[1]. Random construction methods like BF[2] and PEG[3] can make the maximum girth, while the code length, rate, girth and other parameters can be selected more flexibly. However the connection between the ranks is uncertain, which greatly increases the hardware implementation complexity. Structured construction method restricts the connection between rows and columns, which reduces hardware encoding and decoding complexity, but the restriction of general structured codes on code rate, code length and girth reduces its scope of application. Thus a search method with flexible code length, code rate, row weight and girth as well as good BER performance is on demand.

These two algorithms ensure that the LDPC codes obtained possess quasi-cycle characteristics and have a given girth, and the parameters like code rate, loop length and girth can be selected in a wide range. Compared with random construction algorithms, the codes constructed with these two algorithms are quasi-cyclic LDPC codes, which can be easily implemented in both hardware encoding and decoding, and can be constructed faster; compared with other structure construction methods, the algorithms are more flexible and hold a wider range of applications.
II. DEFINATION OF DISTANCE GRAPH
Usually H matrix of LDPC code is described with Tanner graph; it can also be described with distance graph. In the distance graph, the vertices represent rows of matrix H, while the columns of matrix H correspond to the links between the vertices in the distance graph. Figure 1(a) is a distance graph whose column weight is 2, row weight is 5 and code length is 15. Each vertex in the distance graph denotes one row of matrix H, while each edge denotes one column, figure 1(b) is the corresponding matrix. This paper has constructed (1500,3,6)-QC-LDPC code, where the code length is 1500, row weight and column weight are 6 and 3 respectively. Random search method and sequential search method are employed to construct LDPC codes with girth 6, 8, 10. In order to compare with the above codes, here a (1500,3,6) random code with girth 8 is constructed in PEG algorithm. The simulation is in AWGN channel with BPSK modulation, the decoding algorithm is Min-Sum decoding algorithm based on the integer arithmetic, where the number of iteration is 20 times. The simulation results are shown in figure 6 and figure 7. V. CONCLUSION
This paper presents two kinds of search methods based on distance graph- sequential and random search method, where the BER performance of random search method is better, which is close to that of the random codes. In addition, the complexity of the search algorithms is proportional to the code length, and is related to the selected algorithm at the same time.
