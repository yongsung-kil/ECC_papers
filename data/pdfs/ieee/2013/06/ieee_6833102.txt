Rate-Adaptive Irregular QC-LDPC Codes from Pairwise Balanced Designs for Ultra-High-Speed Optical Transport

Ivan B. Djordjevic1 and Ting Wang2
1University of Arizona, Department of Electrical and Computer Engineering, 1230 E. Speedway Blvd., Tucson, AZ 85721, USA 2NEC Laboratories America, Princeton, NJ 08540, USA

Abstract: A rate-adaptive, software-defined, coded-modulation scheme based on irregular quasicyclic LDPC codes designed using the concept of pairwise balanced designs is proposed. The proposed scheme outperforms the corresponding scheme based on regular LDPC codes, and eliminates the error floor phenomenon of regular LDPC codes.

1. Introduction
As the response to never ending demands for higher data rates and distance independent connectivity, 100 Gb/s Ethernet (100 GbE) standard has been already adopted, and 400 GbE and 1 TbE have become the research focus of many researchers. One of the key enabling technologies for the next generation of optical transport is the softdecision FEC. In particular, it has been shown that the LDPC coded-modulation based on large girth (t10) LDPC codes provides excellent BER performance [1]. Unfortunately, the codeword lengths are excessively long for quasicyclic (QC) LDPC code design, and corresponding decoders are difficult to implement with currently existing FPGA/VLSI hardware. On the other hand, girth-8 LDPC codes exhibit the error floor phenomenon. To eliminate it, an outer BCH/RS code has been traditionally used [2],[3]. By using this approach the error floor of girth-8 and girth6 LDPC codes can be eliminated. However, the corresponding net coding gains (NCGs) are much below that of large-girth LDPC codes. Another pragmatic approach is to allow a limited number of cycles of short lengths so that corresponding LDPC codes do not lag too much in BER performance compared to large-girth LDPC codes [1].
As an alternative solution, here we propose to use the irregular LDPC codes, in which irregularity can be precisely controlled, while preserving the quasi-cyclic nature of LDPC code design so that the decoder complexity is preserved. The combinatorial mathematics’ objects known as the pairwise balanced designs (PBDs) [4] are used here to provide such flexibility in code design. The complexity of conventional irregular LDPC codes, non-quasicyclic-ones, is too high to be of practical importance at ultra-high-speed. It will be shown later in the paper that irregular QC-LDPC codes of girth-8 and reasonable length significantly outperform the corresponding regular QCLDPC codes and at the same time eliminate the error floor phenomenon.

2. Proposed Rate-Adaptive Coded-Modulation Scheme based on Irregular QC-LDPC codes from PBDs

Since both the identity matrix and the power of permutation matrix have a single 1 per row, the block size of the i-th

block from PBD determines i-th block-column weight. In the example above, the first three block-columns have

column-weight 3, while the last 3 have the column-weight 2. The code rate adaptation is performed by partial

reconfiguration of decoder based on Eq. (1), by changing the size of permutation matrix P and/or varying the

number of employed block-rows, while keeping codeword length fixed. Another alternative to change code rate is to

use the same regular parity-check matrix as template but employ the PBDs corresponding to different code rates.

3. Performance Analysis and Conclusion

The results of simulations are summarized in Fig. 1, for polarization-division multiplexed (PDM) QPSK of

aggregate date information rate of at least 100 Gb/s and 50 iterations in min-sum-plus-correction-term algorithm

(see [1] for details). In Fig. 1(a), comparison of proposed irregular QC-LDPC codes is evaluated against the

corresponding regular QC-LDPC codes. The girth-8 regular QC-LDPC(28432,21327) code of column-weight 4

exhibits the error floor phenomenon. On the other hand, the corresponding irregular QC-LDPC(28180,21135) does not exhibit the error floor phenomenon; it provides the NCG of 9.4 dB at BER of 10-8, while expected NCG at BER

of 10-15 is 12.4dB. This irregular LDPC code performs only 0.09 dB worse than longer girth-10 regular QC-

LDPC(48012,36011) code, and outperforms regular girth-10 LDPC(16908,12683) code. Clearly, this irregular QC-

LDPC code outperforms pragmatic QC-LDPC(16935,13550) code [1] by 0.46 dB. Finally, this irregular QC-LDPC

code outperforms longer regular girth-10 LDPC(48016,39015) code by 0.37 dB. In Fig. 1(b), the proposed QC-

LDPC code design is used for rate-adaptation, for code rates 0.75-0.9. The QC-LDPC code design, given by Eq. (1),

of length 48000 is used. It is evident that QC-LDPC codes of rates ranging from 0.75-0.85 outperform turbo-product

code of rate 0.82, and significantly outperform concatenated RS code of rate 0.82.

In conclusion, the

proposed rate adaptive coded modulation scheme based on irregular QC LDPC codes from PBDs outperforms the corresponding scheme based on regular LDPC codes, and eliminates the error floor phenomenon of regular LDPC codes.
