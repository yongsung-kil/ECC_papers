Adaptive Normalized Min-Sum Algorithm for LDPC Decoding

Liwen Fan, Changyong Pan, Kewu Peng, and Jiachen Huang
Tsinghua National Laboratory for Information Science and Technology Department of Electronic Engineering, Tsinghua University Beijing 100084, China
fanlw11@mails.tsinghua.edu.cn, {pcy, pengkewu}@tsinghua.edu.cn, hjc12@mails.tsinghua.edu.cn

AbstractÔÇöThis paper proposes an adaptive normalized minsum algorithm for the decoding of low-density parity check (LDPC) codes, which utilizes an adaptive normalization factor to improve the accuracy of the soft information transferred during the iterative decoding process, and provides superior performance accordingly. This adaptive normalization factor can be adjusted dynamically and adaptively at each decoding iteration according to a look-up table obtained via training and simulation. Its implementation facility and independence from the channel characteristics make the proposed adaptive normalized min-sum algorithm expect a wide application. Simulation results show that the proposed algorithm can achieve performance much closer to the sum-product algorithm and a coding gain of around 0.2dB compared to the conventional normalized min-sum algorithm for DVB-S2ÔÇÖs rate 2/5 and 3/5 LDPC codes over the additive white Gaussian noise (AWGN) channel.
KeywordsÔÇölow-density parity check codes; iterative decoding; belief propagation algorithm; adaptive normalization factor
I. INTRODUCTION
LDPC codes [1], first discovered by Gallager in the 1960s, and later rediscovered by MacKay and Neal [2], [3], represent a promising prospect for error control coding with the capability of Shannon-limit approaching performance [4], [5].
Soft decoding of LDPC codes is able to achieve good performance, and can be implemented with iterative decoding based on the belief propagation algorithm [6] (also referred to as the sum-product algorithm, SPA [3]). Unfortunately, the hardware implementation of SPA is prohibitive due to its high computational complexity.
In [7] and [8], a complexity-reduced iterative decoding algorithm simplified from SPA has been proposed for LDPC codes and is referred to as the min-sum algorithm (MSA). This algorithm is universal, as it does not need any knowledge about the channel signal-to-noise ratio (SNR) characteristics. In [9], a performance improved algorithm is proposed by introducing a normalization factor with the criterion that makes the soft information value of MSA equal to that of SPA on average sense, and it is referred to as the normalized min-sum algorithm (NMSA). It is also pointed out in [9] that the normalization factor is related with the channel noise characteristics inherently. But for a specific LDPC code, this is usually

fixed for all SNRs and all iterations in practical implementation due to the inconvenience and inaccuracy in measuring channel SNRs, thus requires a good tradeoff between various channels and different SNRs.
To seek solutions achieving better performance than conventional normalized min-sum algorithm while keeping the virtue of low implementation complexity, this paper proposes an adaptive normalized min-sum algorithm (ANMSA) for LDPC decoding, which utilizes a preconfigured normalization look-up table between the normalization factor and the failed parity-check equation proportion , wherein is adaptively determined by at each decoding iteration, and can be easily measured. Hence, the normalization factor changes adaptively according to this normalization look-up table throughout the decoding process. Thus, the accuracy of the soft information delivered in the decoding process is improved at all SNRs and all iterations. The normalization look-up table can be obtained by Monte Carlo training for a specific LDPC code according to the criterion of minimum mean square error (MMSE).

V. CONCLUSION In this paper, we propose an adaptive normalized min-sum algorithm, which can improve the normalized min-sum algorithm by introducing an adaptive normalization factor. We present the approaches to obtain the normalization look-up table by simulations, and finally show that the proposed adaptive algorithm achieves excellent performances accordingly. This adaptive principle is also suitable for offset min-sum algorithm (OMSA) for LDPC decoding, and can be used to bridge the error performance between the conventional maxlog-MAP and log-MAP decoding algorithms of turbo codes or constellation demapping.
