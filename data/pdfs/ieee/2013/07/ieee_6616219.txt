Abstract—A modified algorithm based on the Uniformly Most Powerful Belief Propagation (UMP BP) algorithm are studied for decoding implementation of regular QC-LDPC codes by introducing two factors of mean square difference and ratio in check processing. The decoding algorithm is a combination of soft decision and hard decision to adaptively decrease the oscillation caused by the change of ex-LLRs sign, further to reduce the errors of UMP BP algorithm caused in first iteration, Simulation result shows that implementation of improved DARBFMS algorithm outperform conventional algorithms. Compared with the improved algorithm in irregular LDPC codes, performance in QC-LDPC codes seems better.
Keywords—QC-LDPC codes, UMP BP algorithm, hard decision, soft decision
I. INTRODUCTION
Advanced decoding method is one of the core technologies in 4G&5G communications. Low density parity check (LDPC [1]) codes, first proposed by Gallager in1962, rediscovered with the good error correcting performance by MacKay and Neal,[2], [3] in the late 1990s, have a good advantage over flexibility, lower error floors, and simplicity of decoding, which can achieve entire parallel operation and near Shannon limit error performance[4], [5]. The LDPC codes have been included in IEEE 802.11n/ac standards as optional codes. However，the lack of the methods for designing LDPC codes by random construction presents serious disadvantages on storing and accessing a large parity check matrix. In order to overcome these problems, the QC-LDPC codes are designed with girth as large as possible and combined with the efficient LDPC decoding technologies, such as the improved algorithm of criteria BP algorithm[6], to reduce the decoding complexity and to improve the error correcting performance.
Because LLR BP algorithm has high degree of complexity and is not conducive to the implementation of hardware for the nonlinear operations (hyperbolic tangent function) and multiplication operations. A good tradeoff between the computation complexity and good performance are tried all the time. As an improvement of LLR BP algorithm, the UMP BP algorithm [7], to some extent, avoids a fair amount of

However, the error correcting performance of UMP BP algorithm becomes worse as the iteration number increases, especially, for larger codewords. So the messages transmitted in the first iteration from variable nodes to check nodes are first modified by using a normalized factor [8] in check processing based on UMP BP algorithm, in which the factor chosen from the range of bit error rate is about10-3~10-4 is obtained in the first iteration by Monte Carlo simulation and is used in other iterations. The algorithm[8] utilizes mean square to make the relative error to be small for modifying the value specified in UMP BP algorithm. Later, the algorithm [9] tries to find the coefficient that can produce the minimum mean square error for the best optimum value to get closed to the real value of UMP BP algorithm. In consequence, the algorithm is more desirable than the normalized UMP BP algorithm[8] whether the block length of LDPC codes is long or short. But the iteration number is not small. A new normalized BP_based algorithm[10] with suitable normalized factor in 802.11n system is proposed, which computed the normalized factors under the circumstance of R=1/2,R=2/3,R=3/4,5/6 in the specific system.
In this paper, according to the innate drawback of UMP BP algorithm, a new modification of UMP BP algorithm in regular QC-LDPC codes is presented, which uses the fluctuation decision to illustrate the message fluctuation phenomenon which are caused by the change of ex-LLRs sign before and after an update in check nodes. The proposed algorithm uses difference and ratio factors to adaptively reduce the probability of a sudden change in ex-LLR value over an edge in successive iterations. The factors gained in first iteration of every SNR point are different, and varies as check nodes messages' change, the introducing of which will further reduce the possibility of the change of ex-LLRs' sign. Here an improved algorithm that inherits the merits of hard decision and soft decision is proposed for illustrating its advantage in LDPC codes.

II. BACKGROUND

A. Bisic Concepts and Notation
LDPC code over GF (2) is a linear block code depicted by sparsity matrix HM*N with most of its elements are zeros except parts are ones. The check matrix H can be represented by bipartite graph, which contains a subset of all the variable nodes, a subset of all the check nodes, and a set of the edges that connect the variable nodes and check nodes. Suppose{i∈N(j)} represent the set of nonzero elements (variable nodes) that connects to check node j, {i'∈N(j)\i} represent the set of nonzero elements (variable nodes) that connects to check node j except the variable node i, {j∈M(i)} represent the set of nonzero elements (check nodes) that connects to variable node i, {j'∈(M(i)\j)} represent the set of nonzero elements (check nodes) that connects to variable node i except the check node j.
B. QC-LDPC codes
As one kind of LDPC codes, QC-LDPC codes are widely used in multiple communication systems, such as DVB-S2, 802.16e, 802.11ac, etc., and are defined by very sparse base matrix Hb (J×L). The element in base matrix is -1, 0 or a positive integer and can be expanded to P×P(P extension factor) sub matrix which denotes an all zero sub matrix and identity sub matrix, respectively. The cyclic-shift value for cyclic permutation sub matrix is controlled by the value of shift matrix SH(P). The basic structure of the decoder is only determined by the position of nonnegative integer in base matrix. In this paper, the parity check matrix H in QC-LDPC codes is constructed with row weight L, column weight 3 and the girth which is greater or equal to 6 for extension factor P≥3L2/4, many kinds of code with different length are generated by changing P.

Sparsity check matrix H is composed of Hk×k (information

V. CONCLUSION
In this paper, DARFRMS, the improved algorithm take advantage of hard decision that can reduce the effect of the accumulated errors in codewords after the modification of the change of ex-LLRs sign by introducing two factors in check node processing, and utilize the encoding gain of soft decision, which shows better BER and FER in regular QC-LDPC codes, especially for moderate long codewords. The introduced factors α, β in the proposed algorithm always dynamically change as iteration number increases in different SNR value. It is clear that these figures depicted above show the performance of DARBFMS algorithm, as the increase of the iteration number and code length, exceed conventional algorithms in regular QC-LDPC codes. In contrast with irregular LDPC codes, the proposed algorithm in QC-LDPC codes outperforms it about 0.1dB for same code length. Although the QC-LDPC codes with the revised algorithm have a slightly higher implementation complexity, the QC-LDPC codes are easy to be implemented in hardware using the shift of register by specified value in cyclic permutation sub matrix. What’s more, the performance curve falls quickly and no error floor appears. Referring to the large girth in QC-LDPC codes , the further advancement in iteration decoding is needed in future research for get a better tradeoff between complexity and performance in 4G/5G communication system.
