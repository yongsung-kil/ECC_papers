Combining Spatially Coupled LDPC Codes with Modulation and Detection

Laurent Schmalen and Stephan ten Brink Bell Laboratories, Alcatel-Lucent, Stuttgart, Germany E-mail: {Laurent.Schmalen,Stephan.tenBrink}@alcatel-lucent.com

Abstract—In this paper, we illustrate the performance of terminated convolutional LDPC codes in combination with spectrally efficient modulation (as a generic front-end), and show that classic LDPC design philosophies of optimizing degree profiles (and thus seeking for irregularity) are becoming less relevant for spatially coupled codes. We show that with very simple codes (even if regular) we can observe a “universal” behavior, which means that the code performs well regardless of the channel or the detector characteristic. We give two examples of simple coding schemes with one illustrating the universal behavior and show by a simulative example the performance in a practical scenario. We also try to give a pictorial illustration of convergence mechanisms of spatially coupled codes, and distinguish between two different regimes that we coin micro- and macro-convergence, respectively.
I. INTRODUCTION
Since the rediscovery of low-density parity-check (LDPC) codes [1], this class of codes has received widespread acceptance and has found its way into several new communication standards (e.g., WLAN, WiMAX, DVB-S2) as well as proprietary solutions. With advances in silicon integration, the optimization of iterative detection and decoding loops has become an important aspect of coding research over the past few years. If, for instance, MIMO detectors or equalizers are used to combat channel impairments, LDPC coded modulation [2], [3] has been shown to allow the design of capacity-approaching systems.
In the last few years, the class of spatially coupled code ensembles has emerged and has shown to have appealing properties. LDPC convolutional codes [4], which are one particular instance of spatially coupled codes, have been around for more than a decade, but only recently it has been noticed that the belief propagation thresholds of certain protograph-based LDPC convolutional codes approach the MAP thresholds of the underlying ensemble [5], [6], [7]. Thus, contrary to a common belief, introducing structure into LDPC codes leads to a class of (degenerated) realizations of LDPC codes which show superior performance under belief propagation decoding. This effect of threshold saturation by introducing structure has been analyzed for the BEC in [8] and it has been shown that spatially coupled LDPC codes can asymptotically achieve the MAP threshold of the underlying regular ensemble under belief propagation decoding.
Recently, this result has been extended to more general channels and it has been shown in [9] that spatially coupled ensembles universally achieve capacity over binary-input memoryless symmetric-output channels: most codes of this ensemble are good for each channel realization of this class of channels.

In this paper, we study the performance of spatially coupled protograph-based LDPC codes combined with demodulation and detection. We are especially interested in finding codes that are universally good for a broad class of channels, i.e., for a large variety of different modulation formats and detectors. We first start by recapitulating protograph-based LDPC convolutional codes and giving a pictorial demonstration of their convergence behavior using an EXIT chart. We depict both convergence regions of the code: the macro-convergence, which is governed by the degree profile of the code and the micro-convergence which stems from the spatial coupling and is thus far less accessible for simple analysis. In the second part of the paper, we combine iterative detection and modulation with spatially coupled protograph-based codes and use density evolution based on the protograph and the Gaussian assumption to evaluate different coding schemes. Finally, we compare two coding schemes in the last part of the paper and show that by a careful selection of the protograph, we can realize a coding scheme that works universally well for detectors with different transfer characteristics. Both coding schemes are verified by a simulation example.

III. COMBINATION WITH DETECTION FRONT-END
In the previous section, we have seen that spatially coupled codes constructed from protographs can improve the decoding threshold by the phenomenon of micro-convergence. In this section, we combine spatially coupled codes with a detector front-end and use common analysis techniques to show how the detector front-end influences the performance of the codes. In conventional LDPC code design, usually the code needs to be “matched” to the transfer curve of the detection front-end. If the code is not well matched to the front-end, a performance loss occurs. If the detector front-end has highly varying characteristics, due to, e.g., varying channels and modulation formats, several codes need to be implemented and always the right code needs to be chosen for maximum performance, which appears impractical over, e.g., block fading wireless channels.
On the other hand, in contrast to a random ensemble of the same degree profile, spatially coupled LDPC codes can converge below the pinch-off in the EXIT chart, so even if the code is not well matched to the front-end we can hope to successfully decode due to the micro-convergence. So, even with a varying channel and detector characteristics, we can use a single code which is universally good in all scenarios. This means that the code design can stay agnostic to the channel/detector behavior. Note that in [9] it has been shown that spatially coupled ensembles achieve capacity in binaryinput output-symmetric memoryless channels.

IV. DECODING THRESHOLDS OF SOME BICM-ID SCHEMES
As a generic example, chosen for the ease of reproducibility, we employ the 16-QAM modulation format with varying labelings. Note that we only characterize the labelings by the respective EXIT function fD(IA,DET). We consider three different bit mappings for the 16-QAM constellation, taken from the appendix of [14]:
1) Gray labeling, leading to an (almost) flat, i.e., constant, EXIT characteristic fD(IA,DET) ≈ C of the detector
2) Set partitioning (SP) labeling 3) the optimized M16a labeling, which has been optimized
with the goal to maximize fD(1) Note that in a dedicated system setup with 16-QAM, the capacity can be approached by utilizing BICM (without iterative decoding) with Gray labeling and a capacity-approaching code. However, if other (non-rectangular) optimized modulation formats, such as, e.g., 8-QAM or circular modulation formats are used, Gray mappings do not exist and either BICM-ID has to be employed or other schemes, such as multilevel coding.
We wish to emphasize, again, that this is just a constructed example chosen for simplicity. We employ different labelings to emulate different detector behavior and to have straightforward reproducibility of the results, which can then be easily generalized. Usually, such a scheme is employed in a scenario with, for instance, MIMO detection or demodulation of higher order non-square constellations.

V. CONCLUSION
In this paper, we have first given a pictorial demonstration of
the convergence behavior of spatially coupled codes using an
EXIT chart. We have distinguished between two convergence
regions of the code: the macro-convergence, which is governed
by the degree profile of the code, and the micro-convergence
stemming from the spatial coupling. We have further studied
the performance of spatially coupled protograph-based LDPC
codes combined with demodulation and detection. We have
used protograph-based density evolution using the Gaussian
assumption to evaluate different coding schemes. Finally,
we have compared two coding schemes and showed that,
by a careful selection of the protograph, we can realize
a coding scheme that works universally well for detectors
with different transfer characteristics; this was verified by
a simulation example. This universality is a very desirable
code property which has been mostly overlooked in classic
code design. As a side result, we have observed that with
simple LDPC convolutional codes and an SP mapping for 16-
QAM, we can improve the decoding threshold compared to the
conventionally optimized 16-QAM BICM scheme with some
degree distribution constraints.
