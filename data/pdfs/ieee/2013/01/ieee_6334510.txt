Multi-Edge-Type Low-Density Parity-Check Codes for Bandwidth-Efficient Modulation
Lei M. Zhang, Student Member, IEEE, and Frank R. Kschischang, Fellow, IEEE

Abstract—A method of designing low-density parity-check codes for bandwidth-efficient high-order modulation is proposed. A multi-edge-type LDPC code ensemble is used to improve the correspondence between modulation bit-channel capacity and bit-level protection in a bit-interleaved coded modulation system. A key innovation is the development of a multi-dimensional extrinsic information transfer (EXIT) vector field technique for the analysis and design of multi-edge-type codes. A condition, sufficient for the decoder to converge, is derived on the multidimensional EXIT vector field, and this condition is used as a constraint in code optimization. Code designs indicate that the proposed method produces codes matching the performance of codes designed using best available methods in ensemble threshold, and is capable of achieving identical finite-length error rates with shorter block-lengths. In addition to simplified design complexity, the resulting codes allow for low-complexity implementation and rate-adaptivity, and hence are well-suited for adaptive modulation systems.
Index Terms—binary codes, error-correction codes, iterative decoding, modulation coding, optimization.
I. INTRODUCTION
T HE DESIGN of capacity-approaching low-density paritycheck (LDPC) codes for binary memoryless symmetric channels such as the binary erasure channel (BEC) and binaryinput additive white Gaussian noise channel (BIAWGNC) has been the focus of extensive research in recent years [1]– [3]. In contrast, the design of LDPC codes for bandwidthefficient high-order modulation remains less well understood. In [4], the non-binary high-order modulation channel was decomposed into several binary bit-channels under multi-level coding (MLC) [5] or bit-interleaved coded modulation (BICM) [6]. Density evolution [3] was used to design codes for each binary bit-channel. The overall performance of the constituent LDPC codes was shown to approach channel capacity for very long block-lengths. An alternative non-binary LDPC code design with improved performance for bandwidth-efficient modulation was proposed in [7], but at a much higher decoding complexity.
Recent trends in LDPC code design have shifted away from the ensemble-based infinite block-length regime in favor of practical properties such as lower implementation complexity or error floor. New families of LDPC codes such as protograph-based LDPC codes [8] and variants of repeataccumulate codes [9], [10] have emerged with lower maximum

A common characteristic of these new code families is the presence of structure in the code Tanner graph prior to code optimization. The application of structured LDPC codes to bandwidth-efficient high-order modulation has been reported using protograph-based LDPC codes [11], and irregular repeat-accumulate codes [12].
As a unifying framework for structured LDPC codes, multiedge-type (MET) LDPC codes were introduced in [13] to control the structure of random code ensembles. The general definition of MET LDPC includes both the protographbased and repeat-accumulate based LDPC code families, with on-going work on finding new useful structures. Examples of MET LDPC applications include LDPC unequal errorprotection [14] and BICM de-multiplexer design [15].
In this paper we design MET LDPC codes for bandwidthefficient high-order modulation, motivated by the differences in LDPC bit-level protection and BICM bit-channel capacity. In order to keep the complexity of the MET ensemble parameterization low, a specific MET parameterization for the target system is proposed while retaining all desired MET properties. An innovative design technique called the multidimensional EXIT vector field is developed to optimize the specific MET parameterization. Code design results show ensemble thresholds and finite-length error rates matching those from best available methods, with much lower design complexity.
Prior work on the analysis of MET ensembles using the EXIT chart technique has been reported in [16] in the form of protograph extrinsic information transfer (PEXIT) charts. PEXIT charts can provide an accurate estimate of the threshold of MET ensembles with different sets of variable nodes connected to distinct bit-channels. Similarities exist between the PEXIT charts and the multi-dimensional EXIT vector fields developed in this paper in terms of MET ensemble analysis. The key difference in our work is the focus on code design using multi-dimensional EXIT vector fields, by deriving the sufficient convergence condition and formulating of the code design problem as an efficient optimization problem.
The outline of this paper is as follows. The target system and motivation are described in Section II. The specific MET parameterization for bandwidth-efficient high-order modulation is given in Section III. Section IV, which contains the main results of this paper, develops the multi-dimensional EXIT vector field, the sufficient convergence condition and the optimization problem. Code-design results in terms of ensemble thresholds and finite-length error rates are presented and compared to published results in Section V. Some concluding remarks are given in Section VI.

In binary LDPC design methods, a concentrated check node degree has been shown to simplify the design procedure with negligible loss in optimality [23]. We apply the same simplification by using a concentrated total check node degree, i.e. the number of edges of all types at a check node. Fortunately, for 2 edge-types we have strong evidence to suggest that a symmetrical split between the edgetypes is optimal [25, Sec. 3.1.1]. B. Finite-length simulations
In order to achieve low error-floors, finite-length parity-check matrices were generated using the improved progressive-edge-growth (PEG) algorithm [31]. The i.i.d. channel adapters were included in the simulation to impose channel output symmetry and allow for the use of the all-zeros codeword in simulating the error rate over all codewords. We give examples of rate 3/4 and 1/2 codes under Gray-labelled 16-QAM, with comparison to published results.
1) Rate 3/4 Gray-labelled 16-QAM: We generated paritycheck matrices from the degree distributions of the rate 3/4 16-QAM code in Table II. Fig. 4 shows the bit and frame error rates (BER/FER) with respect to per-bit signal-to-noise ratio Eb/N0 (dB) for block-lengths 4096 (4K), 8192 (8K) and 16384 (16K). Threshold and capacity are marked by solid and dashed vertical lines, respectively. To achieve a BER of 10−6, the SNRs required are 5.78 dB, 5.49 dB, and 5.27 dB for 4K, 8K and 16K block-lengths. For block-length 16K this required SNR is 0.52 dB from the threshold and 0.72 dB from capacity.
An FER error floor is present at approximately 10−4. A concatenated code solution can reduce the error floor to negligible levels as proposed in the DVB-S2 standard [32]. For example, the code can be concatenated with the (255, 239) double error-correcting BCH code [33] through an interleaver of sufficient length to ensure that less than 2 bit errors are present in each BCH codeword.
