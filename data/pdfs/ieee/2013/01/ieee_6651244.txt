Optimum Quantization for Signal Processing and Error Correction in NAND Flash Memory
Dong-hwan Lee, Jonghong Kim, and Wonyong Sung Department of Electrical and Computer Engineering, Seoul National University
Gwanak-gu, Seoul 151-744 Korea Email: ldh@dsp.snu.ac.kr, jhkim@dsp.snu.ac.kr, wysung@snu.ac.kr

Abstract— Conventional error correction employing harddecision decoding algorithms is not sufficient to correct all the bit errors in high density NAND flash memory. Recently, signal processing algorithms as well as soft-decision error correction are widely studied for improving error correcting performance. However, these techniques demand many memory sensing operations and, as a result, can lead to long sensing latency and high access energy. In this paper, we present optimum memory sensing schemes needed for estimation of threshold voltage distribution, cell-to-cell interference cancellation, and soft-decision error correction of NAND flash memory. We show the error performance improvement with each of these algorithms using simulated NAND flash memory.
I. INTRODUCTION
Recently, the signal quality of data stored in NAND flash memory degrades rapidly because of the semiconductor scaling and the use of multi-level cell (MLC) data coding scheme. In order to improve the reliability in NAND flash memory, error correction with hard-decision decoding algorithms has been widely used because they are easy to implement and require relatively short latency for the decoding process. For example, BCH (Bose-Chaudhuri-Hocquenghem) and RS (Reed-Solomon) codes are adopted for commercial NAND flash memory devices [1], [2].
In sub-20 nm NAND flash memory, however, the conventional hard-decision based error correction is not enough to remove all the bit errors. Thus, LDPC (low-density parity check) codes or TPCs (turbo product codes) based softdecision error correction as well as signal processing techniques are demanded to achieve an improved error correcting performance. These techniques demand an increased number of memory sensing operations because NAND flash memory only equips one-bit quantizers for reading threshold voltage, and hence multiple memory access operations are required for high resolution memory sensing output. Thus, while applying signal processing and soft-decision error correction, the total number of memory access needs to be limited to reduce the latency.
In this paper, we present optimum quantization schemes that are needed for applying signal processing and soft-decision error correction in NAND flash memory. The techniques that are explained in this paper are shown in Fig. 1. III. CELL-TO-CELL INTERFERENCE CANCELATION

The cell-to-cell interference becomes one of the major

sources of distortion in NAND flash memory as the feature

size continuously decreases. According to [7], the amount of

CCI becomes 50 % of the total noise for 20 nm process MLC

NAND flash memory. Thus, removing the CCI is crucial to

ensure storage reliability.

The CCI is a deterministic noise that can be modeled as a

linear combination of threshold voltage shifts for neighboring

cells [8]. Therefore, simple FIR filtering like signal processing

techniques can mitigate the CCI if the threshold voltages of

neighboring cells are known accurately, which however de-

mands many memory sensing operations [4], [9], [10]. Among

the previous works, a least squares based CCI canceller (LS-

CCIC) shows an improved BER (bit error rate) performance

with only a small number of memory sensing operations

[4]. The LS-CCIC consists of coefficient training and CCI

removal steps. During the training step, the coefficients, which

are denoted as Ci, can be obtained by minimizing the cost

During the CCI removal step, the amount of CCI is estimated by using the CCI canceller coefficients, and is removed from the memory sensing output signal. Note that the training step is invoked only when there is significant change in the threshold voltage distribution, while the CCI removal step needs to be conducted at every page read operation. In [4], optimal memory sensing schemes for the CCI cancellation were also studied. For the training step, the uniform quantizer can be used for both victim and neighboring cells. In the CCI removal step, the maximizing mutual information (MMI) memory sensing scheme [11] is used for the victim cells, while the uniform quantization is demanded for the neighboring cells.
The error performance of the LS-CCIC is evaluated by using the simulated NAND flash memory model. In this model, the coupling coefficients for the x, y, and xy directions are set to 0.08s, 0.1s, and 0.0065s, respectively, where the coupling coefficient factor, s, varies from 0.4 to 2.0. As the coupling factor increases, the CCI becomes stronger. The value of s represents the generation of NAND flash memory. For example, when s is 0.9, the threshold voltage distribution is close to 20 nm-class one.

IV. LOW-ENERGY SOFT-DECISION ERROR CORRECTION
The post-FEC (forward error correction) BER performance of NAND flash memory improves as the number of memory sensing operations increases, which necessarily demands more read energy consumption. However, unlike memory sensing, soft-decision decoding consumes less energy when the precision of data is high. This is because the number of decoding iterations decreases as the signal quality improves [12]. Thus, it is important to choose the optimal read precision depending on the SNR of the stored data for low-energy error correction. Recently, we have developed an iteration count based precision selection method [5] to this end.
Figure 7 shows the total energy consumption for LSB (least significant bit) pages of 2-bit MLC NAND flash memory, where the total energy consumption was assumed to be the sum of the energy for the read operation and that for softdecision decoding. The energy consumption is computed based on the electrical specifications listed in Micron flash memory data book [13] and the implementation results for the (68254, 65536) LDPC decoder [12]. The vertical dotted lines represent the operating regions that were defined when the target BER is set to 10−7. For example, in the operating region I shown in Fig. 8, all kinds of read precision work, while 7-, 10- and 16level memory sensing achieve the target BER in the region II. From Fig. 7, we can find that decoding with the smallest read precision in each operating region consumes the least energy.

For example, 4- and 7-level memory sensing show the smallest energy consumption in the region I and II, respectively.
The iteration count of the LDPC decoder can be utilized for low-energy error correction of NAND flash memory. Since the iteration count is a function of the RBER (raw BER) and the read precision as shown in Fig. 8, the RBER can be estimated based on the current read precision and the iteration count. For instance, when the iteration count is repeatedly one with 4-level memory sensing, the RBER is in the operating region I, thus 4-level memory sensing is the best for low-energy decoding. However, when the iteration count is repeatedly greater than one with 4-level memory sensing, we need to increase the number of voltage levels from four to seven. Finally, when the average iteration count is greater than 4 with 16-level memory sensing, the threshold voltage distribution estimation introduced in Section II needs to be conducted. The iteration count-based precision selection method avoids extra time and energy for signal quality estimation by increasing the read precision before decoding failures.
V. CONCLUSION
In this paper, we summarize signal processing techniques and optimum quantization schemes developed for reliability improvement of multi-level cell (MLC) NAND flash memory. Since memory sensing is a time and energy consuming process, optimum quantization schemes are very important not only for soft-decision error correction but also for efficient signal processing. The threshold voltage estimation algorithms require about 11 memory sensing operations to find the best-

fit parameters of the modeled distribution. The least squares based CCI cancellation technique can correct bit errors by reducing the deterministic noise with a small number of memory sensing operations. The iteration count based precision selection method can offer optimal precision information of memory sensing output data, which can be used to lower the energy consumption of soft-decision error correction in NAND flash memory.
