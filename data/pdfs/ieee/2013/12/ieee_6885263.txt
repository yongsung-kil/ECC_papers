A New Decoding Algorithm of LDPC for COFDM-MIMO System

Baojian Gao,Luwei Hao*,Qian Wu,Xiaoning Zhao,Yuhua Yang,Liangguang Xu
College of Information Science and Technology Northwest University
Xi’an, Shaanxi, China,710127 *hlwxx@sina.com

Abstract—The features of the received signal in the LDPC-COFDM MIMO system with space-time block coding(STBC) was analyzed in this paper, the signal estimate of the received signal obeys Gaussian distribution and it has nothing to do with the statistical distribution of the specific channel. According to the features of the received signal, a new decoding algorithm of LDPC for COFDM-MIMO system is proposed. The performance of the algorithm is detailed analyzed in the case of different bit rates and compared with other decoding schemes.The simulation results show that the algorithm can greatly reduce the number of iterations, and improve the decoding speed. Furthermore, the algorithm has the good decoding performances and the adaptability of different code rates.
Keywords-LDPC; OFDM; MIMO; orthogonal space-time coding
I. INTRODUCTION
LDPC-COFDM MIMO system is an attractive solution to increase spectrum efficiency and enhance the ability of anti-multipath interference for mobile transmission applications, so this system has become one of the preferred transmission solutions in the future 4G communication system [1].At present, how to provide significant capacity gains in wireless channels, reduce the LDPC decoding complexity, reduce the decoding of excessive reliance on system resources and improve the system performance and other issues have attracted a widespread attention [2,3].
Nowadays, the joint decoding scheme is the mainstream
Natural Science Foundation of Shaanxi Province(Grand No. 2011JM8034).

decoding scheme of LDPC-COFDM MIMO system. In [4],the joint decoding scheme is that the initial bit posterior probability information is calculated by using the relationships between the code symbols in each module and the characteristics of the code symbols in this scheme and then the initial bit posterior probability information is fed into the LDPC decoder decoding. This decoding scheme not only overcome the decoding bottleneck that the decoding performance of each level is limited by the decoding performance of the next higher level in the direct decoding scheme, but also reduces the complexity of decoding algorithm for the whole system to improve the decoding speed. However, related research programs have some defects now. For example, they don’t make good use of the channel characteristics of the MIMO system and the correlation between the mapped symbols in the constellation map. Furthermore, they have difficulty to accurately calculate the initial LLR when they are in the complicated practical channel environment.
In this paper,LDPC-COFDM MIMO system and its application model in LTE, 4G and other practical systems are deeply studied. A new decoding algorithm of LDPC for COFDM-MIMO system is proposed according to the characteristics of the received signal in the system and the associated information between different levels .This algorithm can improve the speed of decoding and reduce bit error rate .The second part focuses on the decoding algorithm. The third part is the results and analysis of the simulation.

B. Design of LDPC-STBC Joint-Decoder
According to the above analysis, whether the spatial channel is orthogonal or non-orthogonal, the estimated values of the received signal can obtain all the space diversity gain of the transmitted signals and only produce a linear influence. At the same time, according to (6), because of the orthogonal properties of space-time block codes, the merged signal estimate satisfies the Gaussian distribution and it is foreign to the statistical distribution of the channel. We only need to know the channel gain weight when we calculate the estimated value of the received signal, and the requirements of software and hardware for channel estimator are greatly reduced.
The main steps of BP decoding in LDPC-STBC joint decoding algorithm are as follows:
Step 1˖Channel estimator in the system is used to get the
channel power information hij between the various
communication links.
Step 2˖According to (4) and (8),the estimated values ˄ ~s or y ˅of the received signals are calculated.
Step 3˖(13) is used to calculate the initial logarithmic likelihood ratio (LLR).
Step 4˖The information bits are outputted by using the Log-BP decoding algorithm.
III. SIMULATED ANALYSIS
In this paper, we adapt QC-LDPC code (Its code length is 2304 and the code rate is 1/2.) in 802.16e[9];the RU fast coding algorithm is used[5]; the number of the decoding iteration is 5 times; the number of sub-carriers is 576 in the OFDM system; the constellation mapping way is 16QAM; the
channel condition is the 2 × 2 MIMO multipath channel
environment of the “EPA” scene in LTE agreement˄The correlation coefficient is 0.5 between the transmitting antennas and receiving antennas.).We simulate the performances of the algorithm in the case of different code rates and compare with the respective decoding algorithm and made a more detailed analysis on the conclusions.
Fig.1 is the BER curves of the proposed algorithm in this paper and the respective decoding algorithm.When the

iterative number is fixed at 5 and Eb/N0 is greater than 5.8dB,
the BER of this algorithm is less than 10−5 . It has high
decoding accuracy and fast convergence speed. When the other parameters are constant, compared to the respective decoding of LDPC and STBC, whether the number of iterations in respective decoding scheme is 5 or 10, the decoding performance is obviously improved by joint decoding scheme .There are two main reasons for the improvement of the performance. On the one hand, the orthogonal space-time code brings the diversity gain. On the other hand, the joint decoding scheme will make full use of the advantages of the space-time decoding, the high order soft demodulation and BP decoding, the characteristics of orthogonal space-time block code and the mutual information of the codes, so it can greatly improve the BP iterative decoding accuracy, reduce the number of iterations, and improve the decoding speed.
Fig.2 is the performance curves of the proposed algorithm with different code rates. When the code rate is 1/2 and 2/3 in the case of five iterations, the algorithm has fast convergence speed. So it reduces the demand for actual channel environment. It can accurately calculate the initial LLR when they are in the complicated practical channel environment and show the good decoding performances and adaptability of different code rates.

IV. CONCLUSIONS The simulated analysis show that the new decoding algorithm of LDPC for COFDM-MIMO system based on the orthogonal space-time encoding makes full use of the orthogonal characteristics of the orthogonal space-time codes and the relationships between the code symbols, so it has excellent decoding performance and efficient decoding speed. The LDPC-COFDM MIMO system based on the proposed decoding algorithm not only can support various code rates, but also can satisfy the requirement of super speed data transmission rate in the future communication system.
