A Reduced-Complexity Successive-Cancellation Decoding Algorithm for Polar Codes

Chao Xing1,2 , Bei Wang1 , and Shengmei Zhao1

1Institute of Signal Processing & Transmission

2College of Information Science& Engineering

Nanjing University of Posts & Telecommunications

Henan University of Technology

Nanjing, China

Zhengzhou, China

Abstract— Polar codes are codes which provably achieve the capacity of arbitrary symmetric binary-input channels with the complexity of encoders and decoders O(N log N ) , where N is the
code block-length. In the paper, we will focus on a lower complexity implementation of decoding algorithm in the loglikelihood ratio domain. We use the update rule proposed by Gallager in the decoding algorithm of low density parity check (LDPC) codes to replace the node update rules used in successive cancellation (SC) algorithm for polar codes. To simplify the logarithmic and the exponential operations in the Gallager’s approach node updates rule for polar codes, we further utilize a piece-wise linear algorithm to approximate the involution transform function, where the piece-wise linear algorithm only uses multiplication and addition operation. It has resulted in a reduced complexity SC decoding algorithm for polar codes. The numerical simulations show that our proposed SC algorithm (Piece-wise approx.) has a lower implementation complexity for polar code decoding, but at the cost about 0.7dB degradation in the bit-error-rate (BER) performance in comparison with the SC algorithm proposed by Arikan when the BER is 10-5. The proposed SC algorithm (Piece-wise approx.) is a tradeoff between the error performance and the decoding complexity.
Keywords: Polar codes; Successive-Cancelation Algorithm; Reduced-Complexity Decoding; Piecewise-Linear Functions
I. INTRODUCTION
Polar codes are recently proved in coding theory that they asymptotically achieve the capacity of binary-input symmetric memoryless channels [1]. Later, polar codes are proved to achieve the channel capacity for discrete memoryless channel and continuous memoryless channel [2]. Their encoding complexity and decoding complexity are O(N log N ) , where N is the length of the code. Currently, they have been applied in many areas due to their achieving channel capacity property [3-6]. It is well known that the decoding algorithm of polar codes will take an important part in their applications. Hence, the study on efficient implementations of decoding algorithms is an attractive topic in the research area.
The basic decoding algorithms of polar codes include successive-cancellation (SC) decoding algorithm [1] and belief propagation (BP) decoding algorithm [7]. For moderate block lengths polar codes, the decoding algorithms may be considered by integration of performance, latency, complexity etc. Recently, many efficient decoding algorithms for moderate length polar codes have been developed from SC to improve

the performance [8], reduce the latency [9], or reduce the complexity and provide an effective hardware implementation [10-14]. Due to SC decoding algorithm lower complexity compared to BP decoding algorithm, we aim to find an efficient hardware implementation of SC decoding algorithm so that polar codes could be applied for the applications.
The original SC decoding was proposed in the likelihood ratio (LR) domain in which the update rules include multiplication and division operations. It was shown that the update rules of SC algorithm in the log-likelihood ratios (LLR) domain could be regarded as sum-product algorithm (SPA) [10]. As it is well known, SPA is one form of BP decoding algorithm for low-density parity-check (LDPC) codes. Besides SPA, BP decoding of LDPC codes has several other forms, for instance, LLR-BP based on Gallager’s form [15]. This approach can be elegantly implemented in software where requires evaluations of the involution transform and additions.
In this paper, we focus on the complexity of the standard SC decoding algorithm implementation. Any improvement made to an SC decoding implementation can benefit more complex decoding algorithms since all polar codes are based on the same recursive construction. We apply the update rule based on Gallager’s approach was proposed in BP decoding of LDPC codes for the node updates of SC algorithm. But the update rule based on Gallager’s approach is complex to implement in hardware. In order to reduce the computational complexity of core operation, we further utilize a piece-wise linear functions to approximate the update rule in Gallager’s approach. The numerical results show that our proposed decoding algorithm could achieve a tradeoff between the bit error rate performance and the decoding complexity.
The rest of the paper is organized as follows: In section II, we review the basic theory of polar codes; In section III, we propose that the LLR messages in node update of SC algorithm can be computed based on Gallager’s approach, after that we use a piece-wise linear method to reduce computational complexity in Gallager’s approach. In Section IV, we present some numerical results of the proposed decoding algorithm. Conclusions are drawn in section V.

V. CONCLUSION
Polar codes have attracted a lot of attention due to their provably capacity-achieving performance with low encoding and decoding complexity. In this paper, we have presented an effective decoding algorithm of polar codes for a reducedcomplexity hardware implementation. We analyze the BER and FER performance for polar code (256,128) using SC decoding algorithms based on different node updates rules by numerical simulations. The results show that the computational complexity of the core operations has been reduced by using a piece-wise linear function when the complicated transcendental functions have been replaced by a simple multiplication and addition operation. The results show that it is a tradeoff between the implementation complexity and the decoding performance. We will do experiments in future work to verify further improvement is obtained in BER (not in FER), if the decoding is performed systematically [18]. Moreover, our proposed decoding algorithm can also be applied for relatively long block lengths.
