Linearly-Encodable Rate-Compatible Punctured LDPC Codes with Low Error Floor

Zhiyong He1,4, Yong Liang Guan1, Chee Cheon Chui2, Lay Khim Bay2, Sebastien Roy3 and Guojun Han1
1 School of Electrical and Electronic Engineering, Nanyang Technological University, Singapore 639798. 2 DSO National Laboratories, Singapore 118230. 3 Department of Electrical and Computer Engineering, University of Sherbrook, Sherbrook City, Quebec, Canada, G1K 2R1. 4 Institute of Modern Physics, Chinese Academy of Sciences, Lanzhou, China, 730000.

Abstract – We propose a class of rate-compatible (RC) low-density parity-check (LDPC) codes that has linear encoding complexity and low error floor. To ensure linear encoding, a deterministic form for the parity part of the mother parity-check matrix is used. To guarantee good performances in the waterfall region for a wide range of code rates, a layered density evolution algorithm is developed to obtain the degree distributions. Then, a layered node-connection algorithm is proposed to lower the error floor by increasing the values of extrinsic-message degrees of the codes. Simulation results show that no error floor is observed at a frame error rate of 10-5 for the proposed RC code.
I. INTRODUCTION
One of the main advantages of using rate-compatible (RC) low-density parity-check (LDPC) codes (e.g. [1]-[5]) is that such codes can be encoded and decoded using single encoder/decoder pair. To encode LDPC codes with multiple rates, an efficient approach is to construct an LDPC matrix in a diagonal or dual-diagonal form followed by puncturing. It has been observed that these efficiently encodeable RC LDPC codes may suffer from high error floors, when high-rate codes are punctured from low-rate mother codes.
The error floor of LDPC codes when the Belief-Propagation (BP) decoding algorithm is applied is mainly due to small-size stopping sets or trapping sets, where a stopping set is a variable-node set in which all neighbors are connected at least twice [6], in the code’s Tanner graph. However, it remains computationally prohibitive to find the trapping sets or the stopping sets of the LDPC code in the code construction. Since low-degree variables are susceptible to channel noise, a parameter called Approximate Cycle Extrinsic (ACE) message degree was proposed in [7] to provide simple and effective connections among the degrees, cycles and stopping sets. Then, we propose a layered node connection algorithm to improve the cycle structure of the code’s Tanner graph. The results indicate that no error floor is observed at a frame error rate of 10-5 for the proposed RC code.

B. Puncturing pattern for ARQ transmission
The mother code represented by (1) has the lowest rate RL with RL = K /(K + M ) . The daughter code with the highest rate RH with RH = K /(K + J ) can be obtained by puncturing parity bit vectors [p J+1, …, pM]. For automatic repeat request (ARQ) transmission schemes, the daughter code with rate RH is transmitted first. If requested, some bits or all bits in the bit vector pm with (J + 1) ≤ m ≤ M are transmitted according to an ascending order from pJ+1 to pM.

III. DESIGN OF RC LDPC CODE
After adopting a deterministic form for the parity part of the mother matrix, the next step in the design of the proposed RC LDPC codes consists of constructing the systematic part which is composed of an M×K array of sub-matrices (the hm column in (1)).
A. Layered density evolution algorithm
A layered density evolution algorithm is proposed to obtain the degree distributions over the range of code rate of interest, from the highest rate RH to the lowest rate RL. Let the mother matrix be divided horizontally into L layers, where the top layer includes J BRs and each layer in the bottom (L-1) layers includes bŁ(M-J)/(L-1) BRs. The preset integer, dmax, is the maximum column degree of the top layer, and d with db, is a parameter applicable to the bottom (L-1) layers. The parameter d can be the same for all (L-1) layers. Let dc(j) be the maximum column degree in the top j layers and Ndc(j) be the number of

BCs with degree dc(j). The algorithm works as follows.
1) Initialization: dc(1)= dmax 2) for Ndc(1) from 1 to K do 3) for j from 1 to L do 4) Increase the maximum degree by d when j2, i.e.
dc(j)=dc(j-1)+d, Ndc(j)= Ndc(j-1) 5) Increase the degree of the rest (K- Ndc(j)) BCs by 0, …,
or b. Then, we get MD=(K - Ndc(j))b+1 sets of degree distributions. 6) Perform density evolution for MD sets of degree distributions to find the best set for the j–th layer. 7) end for 8) If the threshold of the mother code is improved, store the degree distributions of each layer. 9) end for
The detailed description of density evolution in step 6, including the channel assumptions and the constraints, can be found in the literature (e.g. [8]). Finally, the algorithm outputs the stored degree distributions which achieve the best threshold of the mother code in the last layer.
The proposed layered density evolution algorithm is to find the degree distributions that optimizes the threshold (in the waterfall region) for the mother code with the lowest code rate and improves the threshold for the daughter codes with higher code rates. As an example, consider a mother matrix which is composed of a 36×54 array of sub-matrices (i.e. M=36 and K=18). The matrix is divided horizontally into L=11 layers, where the top J=6 BRs are used as the first layer and each layer in the bottom 10 layers has b=3 BRs. B. Layered node-connection (LNC) algorithm
To design a random LDPC code with single rate, a progressive edge-growth (PEG) algorithm had been proposed in [9] to maximize the girth of the code’s Tanner graph. For the proposed RC LDPC codes, a layered node-connection (LNC) algorithm is used to design each sub-matrix in the systematic part by increasing both the girth of the code’s Tanner graph and the ACE values of cycles. V. CONCLUSION
We have designed rate-compatible LDPC codes with linear encoding complexity by proposing a deterministic form for the parity part of the mother matrix. In order to improve the threshold in the waterfall region for the mother code and the daughter codes, we have developed a layered density evolution algorithm to optimize the degree distributions over a wide range of rates. To lower the error floor, we have proposed a layered node-connection algorithm to increase both the girth and the ACE values of cycles for the RC codes at various rates. Simulation results show that no error floor is observed for the proposed RC code at a frame error rate of 10-5. Furthermore, the proposed RC code achieves a throughput which is only about 0.4 dB away from the Shannon limit in the high SNR region.
