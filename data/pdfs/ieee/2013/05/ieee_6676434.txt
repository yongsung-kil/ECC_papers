Abstract— In this paper, we propose an algorithm for removing short-cycles in Quasi-Cyclic (QC) Low-density parity-check (LDPC) codes. First using an algorithm with lower computational complexity, all the cycles in the parity check matrix H are found. Then following a ruler, we adjust the shift value of circulant permutation matrix (CPM) to remove the short-cycles. This algorithm could ensure that the code after cycle removal process preserves the quasi-cyclic structure .Simulation results show that: comparing with the codes (504,252) and (1008, 504), we achieve 0.1dB improvement both.
Keywords: QC-LDPC codes; circulant permutation matrix (CPM); girth; short-cycles
I. INTRODUCTION
LDPC codes were first discovered by Gallager [1] in 1962 and were rediscovered by Mackay in 1990’s [2]. Quasi-cyclic LDPC codes are the most promising class of structured LDPC codes due to their ease of implementation and excellent performance over noisy channels when decoded with message-passing algorithms as extensive simulation studies have shown. The girth is one of key factors affecting the performance of a QC-LDPC code. Short-girth greatly affects the convergence rate. Therefore, QC-LDPC codes with large girth had become a focal point of research.
There exit many ways to construct QC-LDPC codes with large girth, including directly construction [7] and short-cycles removing [3]. The directly construction method [7] is when constructing the parity check matrix, generate the circulant permutation matrix (CPM) in turns. Every CPM obtain the shift value from the CPM before it, the flexibility is decreased. The hill-climbing algorithm is an effective way which could avoid this drawback, but it could only ensure the CPM is local optimal rather than whole optimal; what’s more, the computational complexity is large. The novel short-cycle removing algorithm [6] has lower computational complexity; however, it may affect the performance of the codes when adjusted the shift value of the CPM.
Taking the computational complexity and the performance into consideration, and based on the novel short-cycle removing algorithm, we propose an algorithm which can remove the short cycles and improve the performance of the

In parity check matrix, all the nonzero elements in the first row and the last row form a cycle of length four; in a similar way, the nonzero elements in the second row also form a cycle length four. That is to say the cycles in the mother matrix appear in the form of cycle-set. Take advantage of this nature, we can simplified the search process of the parity check matrix of QC-LDPC codes.

A. Short-cycle Search Algorithm
The process of short-cycle searching algorithm is: first finding the cycles length four and removing them according to the short-cycle removing algorithm; then searching the cycles length six, do the same operation as the cycle length four, until we get the desired girth.

Sequentially numbered all the cycles through the algorithm find as 1, 2……m, and labeled all the CPMs n=JL. We construct an auxiliary binary matrix F with size m × n , whose rows correspond to all the m cycles and whose columns correspond to all the n circulant permutation matrices in H. The element at the intersection of the ith row and the jth column of F is equal to ‘1’ if and only if the ith cycle passes the jth circulant permutation matrix in H. In the auxiliary matrix, the column with the largest column weight corresponding to the CPM which has the most cycles pass through it.

Step 2: Construct an auxiliary matrix, whose rows correspond to all the cycles and whose columns correspond to all the CPMs in H;

Step 4: Set the element “1” in the jth column is “0”;
Step 5: If F is not an all-zero matrix, go to step 2; else stop.
