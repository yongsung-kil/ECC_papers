Deep Learning-Based Decoding of Protograph LDPC Codes for Spin-Transfer Torque Magnetic
Random Access Memory

† School of Electronic and Optical Engineering, Nanjing University of Science and Technology, China

Abstract—Spin-transfer torque magnetic random access memory (STT-MRAM) has attracted significant attention as a promising memory technology for consumer electronics. However, its reliability is severely degraded by process variation and thermal fluctuation, which lead to both write and read errors. An increase in working temperature also results in an unknown resistance offset. In this paper, we first propose novel neural network (NN)based decoding schemes for protograph low-density parity-check (P-LDPC) codes with parallel edges in the base graph. We further develop a transfer learning (TL)-based approach to mitigate the resistance offset, which can reduce the training data size by 90%. Simulation results demonstrate the effectiveness of these approaches in improving decoding performance in the presence of various channel impairments of STT-MRAM.
I. INTRODUCTION
Spin-transfer torque magnetic random access memory (STTMRAM) is an emerging non-volatile memory technology that offers high speed, low power consumption, and excellent endurance, making it highly suitable for consumer electronics applications, such as smart phones, tablets, and wearable devices [1]. But its reliability is severely degraded by process variation and thermal fluctuation. Furthermore, an increase in working temperature also results in a resistance offset that is unknown to the channel decoder [2]. Therefore, novel decoding schemes for error correction codes (ECCs) are highly desirable to effectively mitigate these channel impairments.
In this paper, we first propose novel neural network (NN)based min-sum (MS) decoding algorithms for protograph lowdensity parity-check (P-LDPC) codes with parallel edges in the base graph, an issue that was not addressed in [3]. In particular, we present two schemes for computing the log-likelihood ratios (LLRs) for the parallel edges. Since collecting a large amount of training data is costly, time-consuming, and may also consume additional power, we further develop a transfer learning (TL)-based approach to mitigate the resistance offset, which can reduce the training data size by 90%.

III. NEURAL MS DECODING FOR P-LDPC CODES WITH
TRANSFER LEARNING
P-LDPC codes with standard decoders have demonstrated superior performance over the STT-MRAM channel [5]. An (N ,K) P-LDPC code is defined by an (Nb, Kb) base code, where Nb and Nb − Kb denote the number of variable nodes (VNs) and check nodes (CNs) in the base graph. A protograph code is formed by replicating the base graph Z = N/Nb = K/Kb times and permuting edges among the same type. Utilizing this lifting structure, NN-based decoding algorithms were developed for P-LDPC codes with lengths up to 1k bits [3]. While the P-LDPC codes discussed in [3] used single edges between VNs and CNs, those specifically designed for STT-MRAM may have parallel edges [5]. Hence, we propose novel NN-based MS decoding algorithms for PLDPC codes with parallel edges in the base graph.

On the other hand, as introduced earlier, an increase in working temperature can cause a resistance offset ∆R1 that is unknown to the channel decoder. In this work, we formu-
late the decoding of STT-MRAM channel with the unknown
resistance offset as a TL problem. We propose a TL-based
approach which can greatly reduce the training data size. Let D denote a domain that consists of the features y and
labels x, i.e., D = {yi, xi}Ni=1, with xi and yi being the input user data and the corresponding readback resistance, respectively. We define the source domain Ds as consisting of y and x without the resistance offset, while the target domain Dt consists of y and x with the resistance offset. During TL, we first pretrain the NN decoder using training data in the source domain to obtain αs and βs. In the target domain, the same NN architecture is adopted and the model parameters are initialized by αs and βs. The NN is then further retrained by finetuning the parameters to obtain αt and βt, using a smaller set of training data in the target domain.
