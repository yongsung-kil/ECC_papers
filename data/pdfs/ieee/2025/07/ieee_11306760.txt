Low-Complexity LDPC Decoder for Cloud Based
Architectures
Satya Kumar Vankayala1, Tirthankar Mittra1, Thirumulanathan D2, Srihari Das Sunkada Gopinath1, Varigonda Srinivasa Mohan Kumar1, Seungil Yoon3,
1Network Team, Samsung R&D Institute Bangalore, Bengaluru, India 2Indian Institute of Technology, Kanpur, India
3OAM SW R&D Group, Samsung Electronics, Suwon, South Korea

Abstract—LDPC code structure allows high parallelism to deliver the high demanding data rates, but the LDPC decoder does not perform well concerning the low latency requirements needed for high-speed applications. This is partly due to the soft decoder involving the computation of non-linear functions with high computational complexity. Virtualized radio access networks (VRANs) are a new paradigm that brings software-based network functions to run on the cloud. In this paper, we propose four LDPC soft decoder approximations: a piecewise linear (PL) approximation, a polynomial approximation, a machine learning (ML) based approximation, and a modification of min-sum approximation using regression methods. We show that the approximations retain a comparable BLER performance and, at the same time, bring down the computational complexity of the decoder. We also show that the BLER performance of our approximations is significantly better than the well-known min-sum approximation. Our methods are suitable for direct implementation in cloud based VRAN systems.
Index Terms—LDPC codes, Tanner graph, Low complexity approximations, Neural Network (NN), VRAN.
I. INTRODUCTION
Fifth generation (5G) and beyond (5GB) communication technology [1] promises to serve 100x increased traffic capacity with high data rates and low latency. Forward error correction plays a significant role in high-speed applications and to fulfill the requirements, Low-Density Parity Check (LDPC) codes and Polar codes are the popular choice to use. LDPC codes for data channels [2] and Polar codes [3] for control channels have the potential to be used in the 5G New Radio (NR). LDPC codes have excellent error correction performance enabling parallel implementation characteristics have been considered as one of the most popular FEC for 5G and beyond systems.
Decoding methods of LDPC were initially proposed by Tanner [4] and Richardson et. al [5]. The commonly used method of LDPC decoding involves an iterative process called the belief propagation (BP). The process involves an update of a likelihood function involving complex hyperbolic functions at each stage, making BP a computationally intensive algorithm [6]. We propose three ways to reduce the computational complexity of the BP algorithm.
1) Approximation of likelihood function update formula using piece-wise linear/polynomial functions with low computational cost.

2) Approximation of likelihood function update formula using neural networks.
3) Machine learning-based implementation of a popularly used min-sum approximation.
Some of the salient features of these approximations are:
1) Controllable accuracy: A τ -close approximation of the function can be obtained, where τ > 0 is a parameter that can be chosen depending on system requirements.
2) Simple Hardware: The piece-wise linear approximation only requires adders, multipliers, and comparators. Once trained to desired accuracy, it may be deployed as is.
3) Better space and time complexity: The computation of likelihood function requires either a large look up table or storing of several polynomial coefficients via Taylor approximation. The former has a poor space complexity and the latter, a poor time complexity. The proposed methods mitigate these issues.
LDPC coders’ hardware implementation is challenging due to high throughput and low power consumption requirements. Enhancements in computational complexity are possible if these approximations are implemented on a virtual radio access network (VRAN) or operator defined open RAN (ORAN) [7] architecture. As we move towards cloud-based software functionalities, added flexibility with reduced latency can be achieved with machine learning-based LDPC decoding techniques. Specifically, the O-RAN module features a RAN intelligent controller (RIC) module where these methods may be implemented.
The rest of the paper is organized as follows. In Section II, we review the existing literature on approximations in LDPC decoder operations. Section III provides a brief overview of the conventional LDPC decoder and introduces the relevant notation. We also examine the likelihood function that is approximated by our proposed methods. Section IV presents the details of these approximation techniques. The results derived from these methods are discussed and compared in Section V. Finally, Section VI concludes the paper.

It has a generator matrix G ∈ {0, 1}k×n and a parity check matrix H ∈ {0, 1}(n−k)×n. The matrix H is assumed to be sparse, i.e., it has nr ≪ n ones in each row, nc ≪ (n − k) ones in each column, and zeros everywhere else.

We approximated the computationally intensive belief propagation expression (2) with (i) piecewise linear, (ii) polynomial, (iii) neural network based, and (iv) modified min-sum approximations. We have shown via simulations that the approximations give a comparable BLER performance to the direct evaluation of (2), and also reduce the computation complexity at the same time. We have further illustrated that the BLER performance of our approximations are significantly better than the min-sum approximation method. With the advent of VRAN and O-RAN, these approximation methods are now practically implementable, and may be considered for replacement of existing decoding modules.
