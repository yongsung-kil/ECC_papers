Abstract—Wireless communication is rapidly evolving, and this evolution has led to increasing demands for better error correction, lower latency, and higher adaptability, particularly in Beyond 5G (B5G) networks. Traditional channel coding methods like Low-density Parity-check (LDPC) and Turbo codes often struggle to meet these advanced requirements. To address this challenge, we explore the use of Graph Neural Networks (GNNs) to enhance LDPC decoding. By imposing the structured nature of the Tanner graph, GNNs improve decoding accuracy, optimize bit error rate (BER), and adapt to changing signal conditions in real-time. We implement and evaluate our approach using MATLAB and Python, training our ML-based decoder on varying SNR levels to ensure robustness. The results show that integrating machine learning into LDPC decoding significantly enhances error correction and communication reliability. This work highlights the potential of ML-driven techniques in shaping the future of wireless networks.
Keywords—Adaptive Decoding, Beyond 5G (B5G), Bit Error Rate (BER), Channel Coding, Error Correction, Graph Neural Network (GNNs), Machine Learning (ML), Signal-toNoise Ratio (SNR), Wireless Communication.

Wireless communications are crucial in many sectors like telecommunications, aerospace, and defence, healthcare, IoT, and consumer electronics. Still, these methods are facing difficulties in adapting to the B5G environments [1], [2], [9], [14]. It requires an adaptive mechanism, so we are going to use machine learning (ML) techniques to optimize performance in real life [3], [11], [12].
Machine Learning (ML) is one of the most blooming technologies, to enhance channel coding one of the suitable ML techniques is going to be used, unlike the traditional methods ML algorithm can adaptively learn from network conditions, making real-time adjustments to minimize the errors and maximize throughput because this adaptability is very essential in the B5G networks where dynamic channel conditions require and intelligent coding approaches [4], [10], [13], [15]. So, by integrating the ML technique with channel coding, we aim to develop an efficient and robust B5G communication framework [6], [14], [16].

We are going to focus on performance features like bit error rate (BER), latency, and computational efficiency are analysed to compare ML-enhanced decoding against traditional LDPC decoding techniques [8], [12]. To evaluate the effectiveness of our implemented ML-based approach, we conduct simulations of various network conditions and noise variations, including different SNR levels to train the model. Our results demonstrate the potential of ML-based techniques to achieve superior error correction and adaptive performance, making them a promising solution for nextgeneration wireless communication networks [5], [7], [13].

A. PRELIMINARY CONCEPTS
Low-density parity-check (LDPC) codes are more effective in error-correcting codes, and they are used extensively in modern communication systems such as 5G and Wi-Fi because of their high reliability and low redundancy. They use a sparse parity check matrix and iterative decoding algorithm to detect and correct the errors suitably, which makes them well-suited for high-speed and low-latency applications. LDPC codes can be represented by a Tanner Graph (Bipartite graph with variable and check nodes) or a parity-check matrix, where rows specify parity constraints and columns represent codeword bits. We have enhanced the LDPC decoding for beyond 5G networks using Graph Neural Networks (GNNs) to enhance error correction, optimize bit error rate (BER), reduce latency, and improve computational efficiency. By training the ML model under varying Signal-To-Noise ratio (SNR) conditions, we enable adaptive decoding that outmatches

traditional LDPC approaches. Our implementation is validated through the simulations, demonstrating improved performances in the dynamic wireless environment [6], [8], [10], [12].
Hard-decision and soft-decision decoding are the two approaches used in error correction for digital communication systems. Hard-decision decoding processes the received bits as discrete values (0 or 1) by applying the fixed threshold, making it computationally efficient but in the case of noisy conditions it is less accurate in such cases we use the soft-decision decoding method, processes by considering the checked probability or confidence level of the received bits, using the Log-likelihood ratios (LLRs), leading to better error correction at the cost of increased computational complexity. Studies like [6], [9], [14] shift from traditional forward error correction (FEC) codes to machine learning approaches, which integrate both decoding techniques for improving decoding technique performances. The work by Gallager [7] provides foundational insight into LDPC codes, which use both hard and soft decision decoding. Additionally, recent research explores the use of Graph Neural Networks (GNNs) for channel decoding, showing how deep learning enhances soft-decision decoding techniques. These advancements are crucial beyond 5G networks, where efficient and adaptive channel coding methods are essential for communications [8], [13], [15].
Machine Learning Techniques and graph Neural Networks (GNN’s) are classes of deep learning models mainly designed to process and analyze data representation as graphs. Unlike traditional neural networks that work on structured data like sequences of images, GNN can also handle non-Euclidean data where the connection between the entities is irregular, making them different for graphbased applications. It has transfigured graph-based learning, making it possible to analyze and model complex relational data [6], [11], [16].
The role of GNN is very important in optimizing channel coding performance through the enhancement of standard decoding methods such as Belief Propagation (BP) for LDPC codes. GNNs make use of the Tanner graph structure to learn efficient message-passing rules, which ensures more efficient decoding and noise robustness. Decoders based on GNN have demonstrated superior error correction performance, mainly under high noise levels and generalizability over various error-correcting codes. They also provide potential advantages in end-to-end learned channel coding, supporting adaptive and efficient decoding mechanisms for Beyond 5G (B5G) and future communications [8], [12], [14]. Challenges remain, though, such as scalability, over-smoothing, and hardware implementation to be overcome before practical deployment [4], [10], [16].

IV. SIMULATIONS AND RESULTS
Graph Neural Networks (GNNs) play a crucial role in improving coding and decoding processes, particularly in progressive communication systems such as LDPC decoding. If we take the concept of GNN, data is constructed as graph nodes representing coded symbols and edges representing the relationship between them, enabling efficient data conversion. At the time of decoding GNNs use the new path approaches, as well as the spread of the faith, to improve and modify the received data. Contrary to traditional methods, GNN can learn from different channel conditions and dynamically adapt to different noise levels. This makes a promising solution for improving error correction and reliability for 5G (B5G) networks and future wireless communication technologies.

In our implementation, the GNN-based decoder is designed as a message-passing neural network (MPNN) operating over the tanner graph representation of the LDPC code. The architecture consists of multiple layers, each corresponding to one iteration of message exchange between variable nodes and check nodes. Each layer comprises two main components: message update functions and aggregation functions, both parameterized by trainable neural modules. Specifically, we employ graph convolutional layers with shared weights across iterations to ensure parameter efficiency. The input features to the network are initialized using the LLRs obtained from the channel output. The hidden dimension for node embedding is set to 64, and the network is trained for 10 messagepassing iterations. A sigmoid function is used at the final output layer to generate soft bit estimates. The model is trained using binary cross entropy loss, with Adam optimizer and learning rate scheduling, allowing the decoder to learn robust representations across varying noise levels. this enables the decoder to generalize across different LDPC code instances while maintaining a relatively low computational efficiency and scalability.

V. Conclusion This study is about a novel approach to improving LDPC decoding using Graph Neural Networks (GNNs), addressing challenges in B5G communications. By using the machine learning technique, it enhances error correction, reduces latency, and adapts to complex network conditions. The simulation result confirms the proposed ML-based decoder outmatches the traditional belief propagation (BP) methods, particularly in more noisy environments. Despite the fact that the computational complexity remains a concern, future research can focus on optimising the model for real-world deployment, which includes reducing the consumption of energy and improving scalability. The integration of ML models in wireless communications has the potential to make significant advancements, making next-generation networks more reliable and efficient.
