LDPC Code Optimisation for OTFS Modulation with MP Detection
Chong Zhang1, Shenghui Song1, Chi-Ying Tsui1, and Jinhong Yuan2 1Dept. of Electronic and Computer Engineering, The Hong Kong University of Science and Technology
2The School of Electrical Engineering and Telecommunications, University of New South Wales Email: {eechongzhang, eeshsong, eetsui}@ust.hk, j.yuan@unsw.edu.au

Abstract—The orthogonal time frequency space (OTFS) modulation is a promising technique to provide reliable communications in high-mobility scenarios. However, the performance analysis for coded OTFS systems is not available in the literature. This paper investigates the extrinsic information transfer (EXIT) behaviour of LDPC coded OTFS systems with message passing (MP) detection. Different from conventional EXIT analysis, which is normally obtained by Monte Carlo simulation, the exact distribution of the extrinsic information for MP detection is presented in this paper. Then, the EXIT function for the LDPC decoder is given with the prior information of the MP detector, which illustrates the convergence behaviour of the LDPC coded OTFS systems. Finally, an algorithm is proposed for calculating the decoding threshold for LDPC coded OTFS modulation, which can be used to design LDPC codes for OTFS systems. Numerical results verify the accuracy of the EXIT analysis, and the 5G NR LDPC code optimised by the proposed method demonstrates 1 dB to 1.5 dB performance gains over the original codes.
Index Terms—OTFS, LDPC Code, EXIT Chart, MP Detection, Code Optimisation.
I. INTRODUCTION
Future wireless communication systems are required to provide ultra-reliable communication in a high-mobility environment. However, the orthogonal frequency division multiplexing (OFDM) scheme, widely deployed in modern wireless communication systems, suffers severe performance degradation due to high Doppler spread [1]. Specifically, a doubly selective channel with both Doppler spread and frequencyselective fading will destroy the orthogonality of subcarriers in OFDM systems and introduces severe inter-carrier interference (ICI) [1]. To this end, orthogonal time frequency space (OTFS) modulation was proposed [2] with superior performance over OFDM in doubly selective channels.
The doubly selective channels can be viewed as a 2D quasitime-invariant channel in delay-Doppler domain. Since the information symbols of OTFS modulation are mapped into the delay-Doppler domain, a sparse graph structure is obtained for the OTFS input-output relation (IOR) [1], [3]. Based on this sparse structure, many advanced signal processing algorithms were proposed for OTFS modulation, including a series of iterative message passing (MP) signal detection methods [3],

[4] and several simple ways for channel estimation [5], [6]. These works have laid a sound foundation for the implementation and application of OTFS systems. However, most existing works did not consider channel coding in their analysis.
Channel coding is essential to ensuring reliable communication, especially under unfavourable channel conditions, such as doubly-selective channels. How to choose the code parameters for joint detection and decoding is an important issue to improve the performance of practical OTFS systems [1]. The authors in [7] analysed the performance of coded OTFS systems with convolutional codes and low-density paritycheck (LDPC) codes, demonstrating the benefits of channel coding for OTFS systems. In [8], the extrinsic information transfer (EXIT) chart is used to analyse the element-wise extrinsic information between the detector and decoder, where the authors only considered an iterative soft cancellation for detection without decoding. As both OTFS modulation and LDPC codes have sparse graph structures, the authors of [9] studied a bit-to-symbol mapping problem from LDPC code bits to OTFS symbols, ignoring the LDPC code structure. Unfortunately, the channel code design and optimisation for OTFS systems remains unclear in the literature.
In this paper, we give the EXIT analysis for LDPC-coded OTFS modulation with MP detector and calculate the decoding threshold for LDPC-coded OTFS systems, which can be used to design the LDPC codes for OTFS systems. Firstly, we study the EXIT characteristics of the MP detector and give the exact distribution of the extrinsic mutual information (MI) of the MP detector. The result shows that the output information of the MP detector satisfies the assumption of consistent Gaussian input [10]. Then, we obtain the EXIT function for the LDPC decoder with the prior knowledge of the MP detector to illustrate the decoding thresholds. We demonstrate the EXIT curves for the regular LDPC coded OTFS systems with the MP detection and show the convergence behaviour of these codes. Then we propose an algorithm to calculate the decoding threshold for LDPCcoded OTFS systems. Using this algorithm as the performance metric, we give an example of LDPC code optimisation based on 5G NR LDPC codes for OTFS systems. Finally, the bit error rates (BER) of the proposed codes are presented. The result of regular codes explicitly verifies the previous EXIT analysis, and the optimizing LDPC codes achieve 1 dB to 1.5 dB performance gains over the original 5G NR LDPC codes

In particular, when we only consider the variable node x[c], other variable nodes connected to y[d] are regarded as interferences, and together with Gaussian noise, they form the interference term ζd(i,c). Assuming each transmitted symbol x[c] = aj ∈ A is of equal probability, the received symbols could be viewed as approximately independent for a given x[c] according to the channel sparsity. Therefore, the interference terms ζd(i,c) in (2) are independent for a given x[c]. The interference term ζd(i,c) in (2) is approximately modelled as a Gaussian random variable with mean μd(i,)c and variance (σd(i,)c)2. III. EXIT CHART ANALYSIS FOR MP DETECTION
In this section, we perform the EXIT chart analysis for MP detection. For that purpose, we first present the details for the receiver structure of the LDPC-coded OTFS systems

with MP detection. As illustrated in Fig. 3, the OTFS receiver consists of two iterative loops or processes. In the outer loop, the MP based OTFS symbol detector computes extrinsic information of each coded bit IE,DET based on the received signal ICH as well as the input prior information from the LDPC decoder IA,DEC . This output serves as the input prior information for the LDPC decoder in the same iteration. The LDPC decoder has an inner iterative decoding process that is composed of variable-node decoders (VNDs) and checknode decoders (CNDs). The soft extrinsic information, which describes the uncertainty of each coded bit, is iteratively exchanged between the MP detector and LDPC decoder, as well as in the inner loop between variable node and check node decoders inside the LDPC decoder. In the following, we first study the EXIT properties of the MP detector for OTFS modulation, and then the EXIT function of LDPC decoder is given with a prior information of the MP detector.

V. CONCLUSION
In this paper, we presented the EXIT analysis for the LDPC coded OTFS systems with MP detector under LTV channels. The EXIT charts showed how the channel information and prior information iterate in the MP detector. Based on the EXIT analysis, the decoding convergence behaviour was illustrated for the LDPC coded OTFS systems with MP detection, and an algorithm was proposed to calculate the decoding threshold. This algorithm can be used in the LDPC codes design and optimisation for OTFS systems. We applied the proposed algorithm to optimise 5G NR LDPC codes for OTFS systems, where 1 dB to 1.5 dB performance gain was obtained.
