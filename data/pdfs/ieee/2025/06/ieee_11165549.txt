AbstractέΑΦWith the rapid development of deep learning and its applications in physical layer communication, deep learningbased channel coding has gradually become a research hotspot. In this paper, based on the neural offset min-sum (NOMS) lowdensity parity-check (LDPC) decoding network, we propose a neural adaptive offset min-sum (NAOMS) LDPC decoding network. By considering the current input values of the check nodes when learning the additive offset parameters and introducing an adaptive offset weight vector, we can better leverage deep learning techniques. Experimental results show that, with a slight increase in complexity, the proposed decoder achieves a 0.8dB improvement in bit error rate (BER) over the NOMS LDPC decoding network. Moreover, the NAOMS decoder requires fewer learning offset parameters compared to the NOMS decoder.
KeywordsέΑΦdeep learning, adaptive offset min-sum decoding, LDPC decoding
I. INTRODUCTION
LDPC code was first proposed by Gallager in 1962 [1]. Due to wide applicability and performance approaching the Shannon limit, LDPC code has been a main channel coding and a hot research topic in the field of channel coding and decoding [2]. The belief propagation (BP) algorithm and minsum (MS) algorithm are the two main methods for LDPC decoding. The BP algorithm performs well but has high computational complexity when updating check nodes, making it difficult to implement. In contrast, the MS algorithm simplifies the computation in exchange for significantly reduces hardware complexity, sacrificing the decoding performance. The offset min-sum (OMS) and normalized min-sum (NMS) offer good decoding performance with low complexity [3].
In recent years, deep learning has been applied to channel decoding problems and has gradually become a research hotspot. The reason is that, neural networks can extract latent sequence features, providing stronger error-correction capabilities. And, the computational complexity of the neural network decoding process is low, typically involving a single forward pass without the need for multiple iterations, and its parallel structure is well-suited for hardware implementation. Moreover, the neural decoder can be jointly optimized with other modules in the communication system. Neural networks can be used to implement Joint Source and Channel Coding (JSCC) [4], achieving a globally optimal communication link gain. T. Gruber and S. Cammerer studied the fully connected neural network decoder for LDPC codes [5], which learns a large number of codewords to ensure good decoder performance.
Channel decoding with deep learning is a binary classification problem, where the number of information bits

k determines the number of classes( 2k ) for classification in the network [6]. As the code length increases, training the neural network will require large codeword dataset [7]. To the end, researchers have attempted to reduce the complexity of decoding by minimizing the number of iterations or operations. Nachmani et al. assigned edge weights to the Tanner graph of BCH codes [8][9], and compared to traditional BP decoders, the algorithm achieved similar decoding performance with fewer iterations. The method proposed in [8] requires a large number of logarithmic and multiplication operations. Loren Lugosch et al. proposed the neural offset min-sum algorithm(NOMS), which learns offset parameters for each edge of the check node [10]. The algorithm requires relatively few parameters and achieves error-correction performance similar to the multiplication-weighted method in [8].
The method proposed in [10] solely uses machine learning techniques for parameter optimization, without considering the message-passing characteristics of the OMS algorithm, which may result in suboptimal parameter training. In this paper, we improve the performance of the NOMS LDPC decoding network and propose the NAOMS LDPC decoding network, which takes into account the current input values of the check nodes when learning the offset parameters. Compared with the NOMS LDPC decoding network, the performance of the NAOMS has been significantly improved, with only a slight increase in computational complexity. In addition, considering that the NOMS LDPC decoding network assigns different weights to each edge of the check nodes in each iteration, we share correction factors to reduce the number of parameters.
The rest of this paper is organized as follows: Section έΖκ describes the OMS algorithm and the NOMS LDPC decoding network. In Section έΖλ, we propose a well-design NAOMS LDPC decoding network based on NOMS. In Section έΖμ, we make a set of experiments the proposed network. Finally, we conclude in Section έΖν.
II. SYSTEM MODEL AND BACKGROUND
LDPC code is a class of linear block codes characterized by a sparse parity-check matrixH . For (N, k) LDPC code, the codeword consists of N bits, which are made up of k information bits and (N έΑΥ k) parity-check bits. The LDPC encoder [11] encodes vector x with the k-bit information into N-bit codeword u using the formula u=xG, where G is the generator matrix.

B. Neural OMS LDPC Decoding Network

The learnable offsets serve the dual purpose of reducing the errors caused by the minimum and approximate sum-and-decay cycles in the Tanner graph. The NOMS LDPC decoding network incorporates multiple learnable offset parameters in the messages transmitted along each edge of the Tanner graph, consists of an input layer, hidden layers, and an output layer. The hidden layers are divided into CN layers and VN layers.

The NOMS LDPC decoding network uses mini-batch stochastic gradient descent to learn the offset parameters, since the operations are differentiable. DECODING NETWORK
The NOMS decoding network simply leverages deep learning to add learnable offset parameters based on the OMS algorithm. It is be difficult to achieve optimal training. In this section, we propose a neural adaptive offset min-sum LDPC decoding network based on the OMS algorithm. We will consider the current input values of the check nodes when learning the offset parameters, further eliminating the ambiguity of the offset parameters. By better searching for the offest parameters, the NAOMS decoding network can significantly improve the BER performance, with a slight increase in complexity.

IV. EXPERIMENTAL SETTING AND EVALUATIONS
A. Test Setup
In this section, we train and test the performance of the NAOMS LDPC decoding network. We select LDPC code with a codeword length of 576 and a code rate of 3/4 according to IEEE 802.16e, with the codeword randomly generated. We use mini-batch gradient descent to train the network. The mini-batch size was set to 120, and the training data was generated at various signal-to-noise ratios (SNRs) from 0 to 5 dB, which helps improve the robustness of the data-driven neural network[15]. The network was optimized using the Adam optimizer with an adaptive moment estimation (Adam) and the learning rate of 0.001. It is worth noting that one iteration of the OMS algorithm corresponds to one variable node layer and one check node layer in the network.
B. BER Performance with Number of Network Layers
The experiment analyzes the relationship between the number of network layers and BER performance.
As shown in Fig. 3, the BER performance significantly improves with an increase in the number of network layers. When the network reaches 32 layers(15 iterations), the performance no longer significantly improve, which is consistent with the characteristics of the LDPC decoding structure. It is important to note that an increase in the number of layers leads to higher computational complexity. Therefore, considering the trade-off, we recommend choosing a network with 32 layers (15 iterations).
C. BER Performance Comparison
In this experiment, we compare the BER performance of the OMS, NOMS, and NAOMS LDPC decoding networks. The loss function and training method of the NAOMS LDPC decoding network conforms to the NOMS LDPC decoding network.
As shown in Fig. 4, the BER of the NAOMS LDPC decoding network with 32 layers (15 iterations) reaches 10ΎΑφ8 at SNR = 5 dB, which is approximately 0.8 dB better than the NOMS LDPC decoding network. The NAOMS LDPC decoding network uses adaptive offset weight vector and constrains the offset parameters by considering the minimum input value of the check nodes. This allows the network to better leverage its learning capabilities and find optimal parameters, resulting in better decoding performance.

V. CONCLUSIONS AND FUTURE WORK
In this paper, we have improved the NOMS algorithm by considering the minimum input values of the check nodes and introducing an adaptive offset weight vector. We have proposed the NAOMS decoding network for the first time, which enhances network performance. Simulation results show that, compared to the NOMS network, the NAOMS network achieves better BER performance. Furthermore, the improved NAOMS network has fewer learning parameters, making it easier to train and implement. In future work, we plan to perform simulations in more realistic channels, develop a decoder with relaxed constraints, and implement it on an FPGA.
