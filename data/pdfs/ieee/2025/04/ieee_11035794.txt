Min-Sum Decoding Algorithm for LDPC Codes

Abstract: In modern communication technology, low-density 
parity-check codes (LDPC) are widely used as an extremely 
important channel coding method, and it has significant 
performance advantages when dealing with long codes. However, 
the complexity of LDPC decoding algorithms is high. In response 
to this phenomenon, based on MCMS decoding algorithm, this 
paper proposes an Improved Multi-parameter Compensated Min-
Sum (IMCMS) decoding algorithm. This algorithm adds a 
threshold near zero at the minimum value and chooses a locally 
optimal algorithm when the minimum value is less than the 
threshold to avoid the zero return loss of MCMS decoding 
algorithm. On this basis, a layered scheduling strategy is applied 
to propose a Layered Improved Multi-parameter Compensated 
Min-Sum (LIMCMS) decoding algorithm. Simulation results 
show that the decoding algorithm proposed in this paper not only 
effectively improves bit error performance but also maintains low 
computational complexity and significantly accelerates the 
convergence speed of decoding in academic papers.

I. INTRODUCTION 
Under the background of vigorous development of modern 
communication 
technology, 
the 
demand 
for 
data 
communication has shown an explosive growth trend. In digital 
communication systems, signals are extremely vulnerable to 
various interference factors during transmission, such as noise 
interference and signal fading. These interferences can cause 
data transmission errors and seriously affect the quality and 
reliability of communication. As a key technology to ensure the 
reliable transmission of data, channel coding becomes 
increasingly important [1]. 
Low-density parity-check codes (LDPC) have been widely 
used in recent years due to their excellent error correction 
performance [2]. With continuous research efforts, the 
complexity of LDPC decoding algorithms has gradually 
decreased, which lays a foundation for its broader application. 
LDPC decoding algorithms are mainly divided into hard-
decision and soft-decision categories [3]. Hard-decision 
decoding algorithms use simple bit information to decode, such 
as the Bit Flipping Algorithm, which is only suitable for binary 
symmetric channels (BSC) but has poor computational 
performance [4]. However, with modern communication 
scenarios requiring higher and higher decoding performance, 
hard-decision algorithms have become insufficient, and soft-
decision algorithms have become mainstream. 
Traditional soft-decision algorithms mainly include the 
Belief Propagation (BP) algorithm, the Log-Likelihood Ratio 
Belief Propagation (LLR BP) algorithm, and the Min-Sum (MS) 
algorithm. The BP algorithm is based on graph theory and 
iterates through information passed on the graph for decoding, 
which has high decoding performance but also high 
computational complexity [5]. The LLR BP algorithm is a 
variant of the BP algorithm that simplifies the information 
passing process in the BP algorithm to addition and 
multiplication operations, but its complexity remains relatively 
high [6]. The MS algorithm proposed in literature [7] simplifies 
the calculation by converting the decoding problem into 
solving a minimum sum problem, but it may cause performance 
loss when updating check nodes. To improve the shortcomings 
of the original MS algorithm, numerous scholars have studied 
and introduced various improved algorithm variants such as the 
Offset Min-Sum (OMS) and Normalized Min-Sum (NMS) 
decoding algorithms [8-9]. Compared to the original MS 
algorithm, these improved algorithms have significantly 
enhanced performance but also introduce some additional 
complexity [10]. The existing performance differences between 
MS algorithms and BP (Belief Propagation) algorithms suggest 
there is still potential for further improvements in decoding 
performance [11]. 
In literature [12], a Multi-parameter Compensated Min-
Sum (MCMS) decoding algorithm was proposed, which 
introduced channel and code length information into the MS 
algorithm's check node update calculation. By analyzing the 
characteristics of the tanh(x) image and introducing 
multiplication 
and 
correction 
factors, 
multi-parameter 
compensation was used to reduce check node calculation errors 
and thereby improve decoding performance. However, this 
decoding algorithm uses a flooding scheduling approach, 
resulting in slow convergence speed. Additionally, during 
iteration, this algorithm needs to store all variable node 
information, thus requiring extensive storage space and leading 
to high computational complexity. 
To sum up, in order to improve the convergence speed and 
decoding performance of LDPC decoding algorithms, this 
paper proposes an Improved Multi-parameter Compensated 
Min-Sum (IMCMS) decoding algorithm that optimizes the 
check node formula to more accurately correct errors in MCMS 
algorithms. Based on this, we propose a Layered Improved 
Multi-parameter Compensated Min-Sum (LIMCMS) decoding 
algorithm that combines the Improved Multi-parameter 
Compensated Min-Sum decoding algorithm with layered 
decoding ideas. This changes the traditional decoding

B. Layered Improved Multi-parameter Compensated Min-Sum 
Algorithm 
Addressing the issues of high complexity and slow 
decoding convergence in the IMCMS algorithm, we can 
integrate the IMCMS algorithm with layered decoding ideas to 
propose a Layered Improved Multi-parameter Compensated 
Min-Sum (LIMCMS) decoding algorithm. This algorithm aims 
to optimize the complexity and convergence speed of the 
IMCMS algorithm through layered processing. 
The difference between layered decoding and ordinary 
decoding lies in the scheduling method. In layered scheduling, 
the check matrix is layered first, and updating starts from the 
first layer. The node information of this layer will be updated 
successively, and the results will also be used to update the node 
information of the next layer. In the same iteration, new 
information generated in the upper layer can be immediately 
used by the lower layer. Under this scheduling method, only 
node information of the current layer needs to be stored. 
Compared with the flood scheduling algorithm that stores all 
node information, the required storage space is reduced. The 
decoding convergence speed of the layered scheduling 
algorithm is also faster than that of the flood scheduling 
algorithm, resulting in lower decoding delay. Practice has 
proven that layered scheduling has a twice convergence speed 
compared with flood scheduling.

IV. SIMULATION RESULTS AND COMPLEXITY ANALYSIS 
A. Analysis of Simulation Results 
To verify the superiority of the proposed LIMCMS 
decoding algorithm, a simulation comparison analysis was 
conducted with the algorithm proposed in reference [12]. 
Matlab was used for simulation verification, and LDPC codes 
with code lengths of 2048 and 8192 that meet the CCSDS 
standard were used. During the simulation, the simulation 
parameters involved in each algorithm were kept consistent. 
Under the AWGN channel, BPSK modulation was used, the 
maximum number of iterations was set to 10 times, and the 
code rate was 1/2. The simulation curves of the Bit Error Rate 
(BER) and Signal-to-Noise Ratio (SNR) of different decoding 
algorithms are shown in Figure 3 and Figure 4. The error 
correction performance of these four decoding algorithms is the 
worst for the MS algorithm, followed by the MCMS algorithm, 
and the best for the LIMCMS algorithm. At a code length of 
2048 and a bit error rate close to10−4 , compared with MCMS 
algorithm, the IMCMS algorithm improves the error 
performance by about 0.3 dB, and compared with IMCMS

B. Complexity Analysis 
The LIMCMS decoding algorithm proposed in this paper is 
an improvement based on MCMS decoding algorithm. It only 
sets a threshold at a fixed position without requiring major 
modifications to the decoder hardware, and only requires minor 
adjustments to implement. Based on this, the complexity of this 
decoding algorithm is similar to MCMS decoding algorithm. 
V. 
CONCLUSION 
In the research of LDPC code decoding algorithm, MCMS 
algorithm has the problems of information loss due to zero reset 
at the minimum value and poor flood scheduling performance. 
Therefore, this paper proposes the LIMCMS decoding 
algorithm, which adopts the locally optimal algorithm to reduce 
information loss when the minimum value is close to zero, and 
changes the original flood scheduling to layered scheduling to 
compress the storage space required by the algorithm. 
Simulation results and complexity analysis show that both 
IMCMS and LIMCMS algorithms proposed in this paper 
improve the performance of bit error rate. In addition, the 
LIMCMS algorithm not only improves the bit error 
performance, but also reduces the complexity of algorithm 
implementation, accelerates the convergence speed of decoding, 
and brings lower decoding delay.
