Piecewise Student’s t-distribution Mixture Model-Based Estimation for NAND Flash Memory Channels
Cheng Wang , Zhen Mei , Member, IEEE, Jun Li , Senior Member, IEEE, Kui Cai , and Lingjun Kong, Member, IEEE

Abstract—Accurate modeling and estimation of the threshold voltages of the flash memory can facilitate the efficient design of channel codes and detectors. However, most flash memory channel models are based on Gaussian distributions, which fail to capture certain key properties of the threshold voltages, such as their heavy-tails. To enhance the model accuracy, we first propose a piecewise student’s t-distribution mixture model (PSTMM), which features degrees of freedom to control the left and right tails of the voltage distributions. We further propose an PSTMM based expectation maximization (PSTMM-EM) algorithm to estimate model parameters for flash memories by alternately computing the expected values of the missing data and maximizing the likelihood function with respect to the model parameters. Simulation results demonstrate that our proposed algorithm exhibits superior stability and can effectively extend the flash memory lifespan by 1700 program/erase (PE) cycles compared with the existing parameter estimation algorithms.
Index Terms—NAND flash memory, expectation maximization (EM), piecewise student’s t-distribution mixture model (PSTMM), parameter estimation.

I. INTRODUCTION
W ITH the rapid development of technology, flash memory is widely used in various electronic devices that require high-speed read and write capabilities [1]. However, with the scaling down of the flash memory chips and the increasing of the number of levels in the flash memory cell, various channel interference and noises have become increasingly pronounced [2]. To improve the reliability of flash memories, error-correcting codes such as low-density parity-check (LDPC) codes have been applied in flash memory controllers to recover the stored information [3]. To achieve the a better decoding performance, the LDPC decoder requires accurate channel soft information, such as channel log-likelihood-ratios (LLRs) [4], [5], [6]. However, precise LLRs rely on accurate modeling and estimation of the threshold voltages.
Due to various noises and interference, the flash memory channel is non-stationary and thereby hard to estimate. In the literature, most work focused on parameters estimation for flash memory channel with Gaussian models. The work in [4] proposed an unsupervised method based on the observed data to address this issue. In addition to traditional parameter estimation algorithms, [7], [8] and [9] assumed the flash memory channel to be a Gaussian mixture model (GMM) and the expectation maximization (EM) algorithm was employed for channel parameters estimation.
However, the voltage distribution in real flash memory exhibits heavy-tails, which cannot be captured by the Gaussian distribution models [10], [11], [12]. Hence, exploiting GMMEM algorithm to obtain the parameters is not precise for a realistic flash memory channel. Prior works in [10] and [13] employed non-Gaussian distributions, such as Beta distribution and Gamma distribution, or data-driven machine learning approaches for modeling NAND flash memory channels. Later, a normal-Laplace mixture model was proposed to capture the heavy-tails and asymmetry in the distributions of voltages [12]. Moreover, a piecewise student’s t-distribution with varying tail sizes for flash memory was proposed and the Nelder-Mead (NM) algorithm was used for model parameters estimation [11], [14]. The piecewise student’s t-distribution model has taken into account both complexity and accuracy, making it suitable for the modeling of practical flash memories.
In this paper, we first develop a piecewise student’s tdistribution mixture model (PSTMM) under the EM algorithm framework based on the student’s t-distribution model [11]. As the flash model parameters vary over time, we further propose an PSTMM-EM algorithm to efficiently estimate the channel

III. PSTMM-EM ALGORITHM FOR FLASH MEMORY
To derive the PSTMM-EM algorithm, we first present the maximum likelihood estimation under the PSTMM channel. Then, we derive the E-step and M-step of the PSTMM-EM algorithm based on the observed data likelihood. The PSTMM-EM algorithm iteratively refines parameter estimates by alternately computing the expected values of the missing data (the E-step) and maximizing the likelihood function with respect to the model parameters (the M-step). It starts with an initial parameter vector Ψ(0) and iteratively performs E-step and M-step until convergence.

IV. SIMULATION RESULTS
The degrees of freedom η for left and right halves of voltage distribution for each state are set to (100,6) and the initial η for PSTMM-EM and NM algorithms are set to (50,10). We first estimate the parameters of MLC flash memory model by different algorithms. For GMM-EM algorithm, it models the MLC flash memory channel as a 4-component GMM. For PSTMM-EM and NM algorithms, they consider the MLC flash memory model as a 4-component PSTMM. Then, we optimize the read threshold voltages with these estimated parameters by maximizing the mutual information of the flash memory channel with dynamic programming algorithm [16], [17]. With designed threshold voltages, the LLRs can be calculated and LDPC decoding can be further performed. In addition, the performance with perfect channel parameters is also included as the optimal benchmark.
A 4K-LDPC-code is employed and constructed by progressive-edge-growth algorithm [3]. The code length is 4544 and code rate is 0.9. The variable-node degree distribution (x) = 0.0682x + 0.1822x2 + 0.1329x3 + 0.6167x4 and check-node degree distribution ϑ(x) = 0.22x38 + 0.78x39. The sum-product algorithm is used as the decoding algorithm where the maximum number of decoding iterations is 25.
Fig. 2 plots the mutual information (MI) of different algorithms with 6-level read quantization scheme. In most cases, PSTMM-EM algorithm and GMM-EM algorithm converge within 20 iterations. It is challenging to achieve rapid convergence for NM algorithm due to the fact that it operates by maintaining a geometric figure called a simplex, which is essentially a shape with vertices representing different solutions. The PSTMM of MLC flash memory has 16 parameters to estimate. Therefore, the NM algorithm requires a large number of iterations to search for the optimal model parameters.
As shown in Fig. 2, the MI curve of PSTMM-EM is extremely close to the optimum for each P/E cycle of the flash memory. This indicates that the PSTMM-EM algorithm is capable of accurately estimating the parameters of the PSTMM. Although GMM-EM algorithm can estimate the mean and variance, it

Fig. 3. The FER performance of 4K-LDPC-code over different P/E cycles with 6-level read scheme.
can not optimize the degrees of freedom under PSTMM. Hence, there is a gap between it and the optimal MI. The maximum iterations of NM algorithm are set to 1000 and 3000, respectively. It can be seen that, a part of the MI curve for NM algorithm is close to the optimum while the remaining parts deviate significantly from the optimum. The primary reason is the multitude of parameters that the NM algorithm needs to optimize, making it prone to getting trapped in local optima. Moreover, increasing the number of iterations does not yield significant improvements and may even degrade its performance.
Fig. 3 illustrates the frame error rate (FER) performance of 4K-LDPC-code with 6-level read quantization scheme. It is observed that, our proposed PSTMM-EM algorithm not only outperforms other algorithms significantly in terms of FER performance but also demonstrates notable advantages in stability. At FER = 10−4, the proposed algorithm improves the endurance of MLC flash memory by 1700 P/E cycles compared with other algorithms. Although the NM algorithm is capable of parameter estimation for PSTMM, its stability is comparatively poorer. Due to the numerous parameters in the PSTMM, the NM algorithm tends to get trapped in local optima during the estimation of these parameters. Compared to the above two algorithms, the GMM-EM algorithm lacks the estimation of degrees of freedom, leading to significantly inferior performance compared to these two algorithms.
V. CONCLUSION
In this paper, we have first proposed a PSTMM for flash memories under the EM algorithm framework. It can effectively capture the characteristics of flash memory threshold voltage distributions with asymmetric and heavy-tails. To enable the estimation of the parameters of the PSTMM, we have proposed an PSTMM-EM algorithm. By iteratively estimating the missing values and updating the model parameters, the PSTMM-EM algorithm can accurately estimate the model parameters. Simulation results have demonstrated that our proposed PSTMM-EM algorithm enables the LDPC codes to achieve the near-optimum error rate performance. Compared with the existing algorithms, our proposed algorithm is easy to implement and offers higher accuracy and stability for parameter estimations.
