Rank Analysis and Its Applications for Quasi-Cyclic Low-Density Parity-Check Codes

Po-Chun Yang , Chung-Hsuan Wang , Member, IEEE, and Chi-chao Chao , Member, IEEE

Abstract—In this paper, we develop a new approach for rank analysis of parity-check matrices for quasi-cyclic lowdensity parity-check codes based on the associated polynomials of circulant matrices, applicable to general finite fields and arbitrary circulant sizes. Some formulas on the rank for paritycheck matrices with one, two, and three row-blocks are first derived. For the general case with arbitrary numbers of rowblocks, lower and upper bounds on the rank are presented, and these bounds can be combined to give the exact rank result under certain conditions. We also investigate the effect on the rank by changing the circulant size. Furthermore, we study the relations between the rank of the masked matrix and that of the masking matrix, which can be used to predict the rank of the parity-check matrix after masking. The obtained rank analysis results are then applied to several classes of existing algebraically constructed parity-check matrices, along with their masked matrices. Finally, we demonstrate how rank analysis can be used in the code design procedure.
Index Terms—Low-density parity-check (LDPC) codes, quasicyclic (QC) codes, parity-check matrices, rank, algebraic construction, masking.
I. INTRODUCTION
I N RECENT years, low-density parity-check (LDPC) codes [1] have been widely used owing to the near-Shannonlimit performance with iterative message-passing decoding [2] and sufficiently long block lengths. Quasi-cyclic lowdensity parity-check (QC-LDPC) codes [3], [4], [5], [6], [7], [8], [9], whose parity-check matrices are composed of blocks of circulant matrices, are one of the most well-known subclasses of LDPC codes due to the advantage of efficient encoding and decoding in very-large-scale integration (VLSI) implementation. The performance of well-designed algebraically constructed QC-LDPC codes can be as good as that of other types of LDPC codes [6], [8], such as randomly constructed LDPC codes. Masking [14] is also a common technique to improve the error performance by reducing the number of short cycles in the corresponding Tanner graph.
Determining the code dimension, which corresponds to the number of protected information bits, is usually the first step in code design. Since the code dimension is equal to the code length minus the rank of the parity-check matrix, knowledge of the rank of a parity-check matrix is essential. It was shown in [15] that the row-redundancy can have a good impact on performance for LDPC codes. In fact, quite a large number of parity-check matrices for QC-LDPC codes are not full-rank or even rich in redundant rows. It is hence necessary to know the number of redundant rows in the parity-check matrix to design a QC-LDPC code. Gaussian elimination may be the most popular method to find the exact rank of a determined matrix. However, we may not have a determined parity-check matrix before a code is designed. In this case, we still want a method to find the rank of a matrix with some undetermined parameters, which may be combined with the employed code construction. Hence, even though Gaussian elimination can be employed to compute the exact rank numerically for determined matrices, an analytical method to find the rank of a parity-check matrix is still needed from a theoretical viewpoint. With such a method, the code dimension and row-redundancy can then be designed just like the girth. Furthermore, knowledge of the rank of a parity-check matrix is essential for certain algebraic constructions of LDPC codes, e.g., the variable-rate constructions in [8] and [16]. One method to construct variable-rate QC-LDPC codes is to add column-blocks or remove row-blocks to/from the parity-check matrix of a mother code, and both operations may change the number of dependent rows in the parity-check matrix.
In [17], rank results for LDPC parity-check matrices constructed by Vandermonde matrices and RS codes were presented. There were some research efforts [18], [19], [20], [21], [22], [23], [24], [25], [26] for rank analysis of paritycheck matrices for QC-LDPC codes by Fourier transform, in which analytical results including exact formulas or bounds were provided for several different constructions. However, there is an important limitation on this approach that, for codes over GF(q) and with circulant size e, only when gcd(e, q) = 1 can the method by Fourier transform be applied. Many alge-

In [27], construction of a class of QC-LDPC codes derived from finite affine planes was proposed and rank analysis for the constructed codes was provided; yet the rank analysis method in [27] may not be suitable for general QC-LDPC codes. A formula on the dimension for QC-LDPC codes with no limitation on the circulant size was provided in [28], based on the associated polynomial approach proposed in our preliminary work in [29] different from the previous method by Fourier transform. However, it is necessary to know all the component circulant matrices in the parity-check matrix so that the formula in [28] can be used.
In this paper, based on the approach of associated polynomials proposed in our preliminary conference paper [29], we develop extensive rank analysis for QC-LDPC codes. We first define the associated polynomial for circulant matrices and then introduce the associated polynomial matrix for paritycheck matrices of QC-LDPC codes. The rank of a parity-check matrix can then be determined from the associated polynomial matrix and the circulant size. This approach is applicable to parity-check matrices over general finite fields and with arbitrary circulant sizes.
Our main contributions are summarized as follows:
• Rank analysis for general parity-check matrices: We develop a new approach based on associated polynomials for rank analysis of QC-LDPC codes, applicable to all parity-check matrices over general finite fields and arbitrary circulant sizes. Explicit formulas for the rank are derived for parity-check matrices with one, two, and three row-blocks. For the general case with arbitrary numbers of row-blocks, we present various lower/upper bounds and provide an exact rank result when sufficient structural information is available. In particular, we analyze how varying the circulant size affects the rank, a property not addressed by the conventional Fourier transform approach. Even when only partial information is known, our approach can still yield useful insight into the rank, which is particularly valuable for matrices with unknown or undetermined parameters, as during the code design process.
• Rank analysis with masking: Masking is a common technique used to improve error performance in algebraic QC-LDPC code constructions. We study the relationship between the rank of the masked matrix and that of the masking matrix, which has not been investigated in previous research. We find that in many cases the rank of the masked matrix is determined solely by the masking matrix, regardless of the base matrix. This greatly simplifies the code design process, as a masking matrix with the desired rank can be easily constructed.
• Application to several classes of algebraic constructions: We apply our rank analysis methods to several classes of existing algebraically constructed parity-check matrices [6], [8], [10], [11], [12], [13], producing rank results many of which have not been obtained before. For the constructions in [6] and [8], where prior analyses using

the Fourier transform were available, our method not only recovers similar results under equivalent conditions but also yields additional formulas that can serve as practical design tools. For the constructions in [10], [12], and [13], to the best of our knowledge, the obtained rank analysis results are new. We further extend our analysis to masked versions of these constructions, which have not been previously studied. This enables more flexible use of algebraic constructions in practical code design. • Application to code design: We provide analytical criteria to determine whether a parity-check matrix can exist for a given code length, rate, and circulant size of a QC-LDPC code. We then propose a code design procedure and demonstrate, through examples, how our rank analysis approach can be used in code construction.
The remainder of the paper is organized as follows. In Section II, we introduce the associated polynomials and associated polynomial matrices for QC-LDPC codes. In Section III, rank analysis for different kinds of parity-check matrices is given with bounds and exact rank results in various situations including changing the circulant size. In Section IV, we study the relations between the rank of the masked matrix and that of the masking matrix. In Section V, the rank analysis results are applied to several classes of algebraically constructed paritycheck matrices and the corresponding masked matrices. In Section VI, we illustrate how rank analysis can be used in code design. Finally, the work is concluded in Section VII.

The associated polynomial [30] for a circulant matrix is defined as the corresponding polynomial for its first row so that a circulant matrix can be represented by the associated polynomial. On the other hand, we can reconstruct a vector and a circulant matrix by a polynomial c(x) in the polynomial ring GF(q)[x] and a predetermined circulant size e. The first step is to find the remainder of c(x) divided by xe − 1, i.e., c(x) mod (xe − 1), so that the result has degree less than e. Then by c(x) mod (xe − 1) it is easy to construct the corresponding vector and circulant matrix. Moreover, the multiplication and addition of the e × e circulant matrices over GF(q) are isomorphic to the corresponding operations for associated polynomials in the quotient ring GF(q)[x]/xe − 1. In this paper, we calculate the multiplication and addition of the circulant matrices by computing those of the associated polynomials over GF(q)[x]. Only when we have to reconstruct the circulant matrices we will then map the results from GF(q)[x] to GF(q)[x]/xe − 1.
We can represent an se×te parity-check matrix H(s, t) over GF(q) by replacing all circulant matrices with the corresponding associated polynomials, and the resulting matrix C(s, t) is called the associated polynomial matrix for H(s, t). We may sometimes omit (s, t), i.e., the numbers of row-blocks and column blocks, for simplicity and use H and C to represent a parity-check matrix and its associated polynomial matrix. Moreover, in order to represent the elements in an s × t matrix M, we write M = [mi, j]1≤i≤s,1≤ j≤t, where mi, j is the element in the i-th row and j-th column of M.

III. RANK ANALYSIS OF PARITY-CHECK MATRICES
In this section, we introduce some general methods for rank analysis of parity-check matrices. The basic idea behind the presented methods is to perform elementary row/column operations on the associated polynomial matrices since it is isomorphic to perform operations with circulant matrices in the original parity-check matrices. We first give formulas for the exact rank of a parity-check matrix with at most three row-blocks in terms of entries in the associated polynomial matrices. Lower and upper bounds on the rank for matrices with arbitrary numbers of row-blocks are also given. These bounds can be combined and become an exact formula if the minors of the associated polynomial matrices are given. Moreover, we study the relations between the ranks of parity-check matrices with different circulant sizes but similar associated polynomial matrices.

D. Code Families by Expanding Circulant Size
In many QC-LDPC code constructions, codes are given with some designed parameters, such as the number of row blocks, the number of column blocks, the shifting values of CPMs, the circulant size, etc. We observe that the codes constructed in [34], [35], [36], and [37] can be extended to a series of similar codes by changing the circulant size. Here we study the corresponding effect on rank.
We define a series of codes, called the code family, by generalizing the constructions in [34], [35], [36], and [37]. Consider two parity-check matrices H(s, t) = [hi, j]0≤i<s,0≤ j<t and H (s, t) = [hi, j]0≤i<s,0≤ j<t, both over GF(q), with circulant sizes e and ep, and with associated polynomial matrices C = [ci, j(x)]0≤i<s,0≤ j<t and C = [ci, j(x)]0≤i<s,0≤ j<t, respectively. Then we define that H(s, t) and H (s, t) are in a code family if and only if ci, j(x) ≡ ci, j(x) (mod xe − 1) for all i, j. Moreover, we define that H(s, t) and H (s, t) are in a strict code family if C = C . The constructions in [37] correspond to codes in a code family, while those in [34], [35], and [36] correspond to codes in a strict code family, in which the associated polynomial matrices remain fixed and only the circulant sizes change. We observe that most formulas in Subsection III-C are still applicable with different circulant sizes. Moreover, we obtain that the matrices in the same strict code family may have the same numbers of redundant rows, especially when the circulant sizes are integral multiples of the minimum circulant size. Hence, if matrices H, H in a strict code family with circulant sizes e and ep, over GF(q) of characteristic p, then we can determine the rank of H by that of H. The first result states that the parity-check matrices in the same code family are guaranteed to have full row rank if any one of them has full row rank.

IV. RANK ANALYSIS FOR MASKING
In this section, we first provide some methods to estimate the rank of the masked matrix by that of the masking matrix, which is much easier to determine. We then proceed for rank analysis of the masked matrices with some specific circulant sizes. Several conditions are also given for full-rowrank matrices via masking.
To begin with, we assume that the base matrix is composed of CPMs. Note that if the base matrix B contains all-zero matrices besides CPMs, let the corresponding positions in the masking matrix Z be zeros and thus all the rank results in this section are still applicable. We observe that if the base matrix is composed of CPMs, then we can have rank(C|x=1) = rank(Z) for a masked matrix, where rank(Z) is the rank of the masking matrix Z over the ground field. Then we have the following corollary, which is directly from Theorem 6.
Corollary 6: Consider H(s, t) = Z(s, t) B(s, t), where Z is the masking matrix and B is composed of CPMs with circulant size e. If rank(Z) = rz, then
rank(H(s, t)) ≤ se − s + rz.
This corollary has some implications for code design so that if a full-row-rank masked matrix is desirable, then the masking matrix must be full-row-rank. A typical example for Corollary 6 is that Z is an all-one matrix, which is equivalent to Theorem 6 when H(s, t) is composed of CPMs, and the resulting upper bound is se − s + 1. When there is a submatrix that meets the rank upper bound in Corollary 6, the exact rank of the whole matrix can then be found.

V. APPLICATION TO ALGEBRAIC CONSTRUCTIONS
In this section, we will show how our analytical results can be applied to several classes of algebraic constructions of QC-LDPC codes in [6], [8], [10], [12], and [13]. The rank analysis results for the matrices in [10], [12], and [13] do not appear in the past research. In [19], [20], [21], [22], [23], [24], [25], and [26], there were some rank results for the constructions with restricted circulant sizes in [6] and [8]. Yet our methods can be applied to the matrices in [6] and [8] with arbitrary circulant size and also yield useful results for the restricted cases considered in [19], [20], [21], [22], [23], [24], [25], and [26]. Since there are special structures in all of the above constructions, the bounds in Subsection III-C may become tighter or even exact rank results. Although Theorem 7 is applicable to all parity-check matrices for QCLDPC codes, the calculation may not be easy for general cases. While for the algebraic constructions considered, the rank determination can be much simpler as the parameters in the constructions are specified not arbitrary. Moreover, as it is common to combine the above algebraic constructions with masking, we also provide corresponding rank analysis results.

VI. APPLICATION TO CODE DESIGN
In this section, we will show how rank analysis results can help designing QC-LDPC codes. We first describe the procedure of code design. Then a theorem which can determine whether desired codes exist is provided. Finally, examples are given to illustrate the code design procedure and consideration.
We will focus on algebraic constructions of QC-LDPC codes since they can produce codes of flexible rates and lengths compared with computer searching algorithms such as the progressive edge-growth (PEG) algorithm [41]. Moreover, masking is considered to improve the error performance. For algebraically constructed QC-LDPC codes with masking, in some cases the code rate can be solely decided by the masking matrix so that the code design problem can be greatly simplified.
The proposed procedure of code design is described as follows.
1. Decide the code length and code rate. 2. Choose a proper circulant size and the numbers of
row/column blocks of the parity-check matrix. 3. Construct the parity-check matrix by appropriate alge-
braic constructions with proper rank analysis tools and masking methods.
In traditional code construction, as there are no rank analysis tools available, the rank of the parity-check matrix can be determined only after the whole matrix is specified. In some extreme cases, the desired matrix may not exist. This typically occurs when the circulant size or the number of row/column blocks is improperly chosen relative to the target code length and rate, making it impossible to construct a feasible paritycheck matrix. In contrast, our approach can not only improve efficiency but also prevent such cases by predicting feasibility beforehand.
From Theorem 7, we find that the possible values of the rank of a matrix H(s, t) with circulant size e are restricted.

VII. CONCLUSION
In this paper, we have developed a new approach for rank analysis of parity-check matrices for QC-LDPC codes based on the associated polynomials. Many lower/upper bounds and exact formulas have been proposed for various types of paritycheck matrices. We also analyze the effects on the rank by enlarging the circulant size. Moreover, we provide analysis regarding the relations between the rank of the masked matrix and that of the masking matrix. Our methods have also been applied to several classes of algebraically constructed parity-check matrices and corresponding masked matrices. The adopted approach is applicable to general finite fields and circulant sizes and is hence able to produce results that cannot be obtained by the conventional Fourier transform approach. Finally, we demonstrate how the rank analysis results can be used in code design.

Yet, in our designed codes, the constructed parity-check matrices have rich structures among them.
