Abstract—In this paper, we prove that the generalized ReedSolomon (RS) codes are capacity-achieving over binary-input output-symmetric (BIOS) channels, in terms of frame error rate (FER) under maximum likelihood (ML) decoding. In the finitelength region, we present the ordered statistics decoding with local constraints (LC-OSD) algorithm for the generalized RS codes. In particular, the extended most reliable basis (MRB) is derived based on a systematic matrix calculated by the parallel Lagrange interpolation, accelerating the conventional Gaussian elimination (GE). Additionally, we propose a joint source-channel coding (JSCC) scheme that incorporates generalized RS codes and classified enumerative (CE) coding, where the partition of the source is optimized by the k-means++ clustering algorithm. At the transmitter, we implement the CE coding to encode the source information. Then the variable-length codeword of the CE coding is transformed into a fixed-length codeword by the multiple-rate generalized RS encoding and superimposed with a class label for transmission. At the receiver, parallel LC-OSD is performed to recover the source. Simulation results demonstrate that the proposed JSCC scheme outperforms the double polar JSCC scheme (exhibiting a coding gain of up to 1.4 dB) and the JSCC scheme based on polarizing matrix extension (exhibiting a coding gain of up to 0.6 dB), as predicted by Gallager’s JSCC bound.
Index Terms—Coding theorem, enumerative coding, generalized Reed-Solomon codes, joint source-channel coding (JSCC), ordered statistic decoding with local constraints (LC-OSD) algorithm.
I. INTRODUCTION
R S CODES are optimal in the sense that they are maximum distance separable (MDS) codes, which have been widely used in various systems, such as deep space communication, satellite communication, and data storage, due to their robust error correction capabilities for random and burst errors [1]. An interesting question arises: are RS codes capacityachieving? The answer is positive for generalized RS codes as shown in this paper. To the best of our knowledge, no such a proof is available in the literature. This is primarily because the conventional proof using the error exponent [2] is not applicable to the generalized RS codes due to challenging issues such as the lack of pair-wise independence among codewords and the non-identical distribution of nonzero codewords. In spite of their capacity-achieving property, long generalized RS codes lack efficient soft-decision decoding algorithms. However, in the short-length region, RS codes can be decoded using the ordered statistic decoding (OSD) algorithm [6], [7]. The basic idea of the original OSD is to generate a list of candidate codewords by flipping a small number of bits in the most reliable basis (MRB). To reduce the complexity of the original OSD algorithm by early stopping unlikely unnecessary test patterns, the OSD with local constraints (LC-OSD) algorithm [8], [9] considers an extended subset of reliable positions by including extra positions next to the MRB. The OSD algorithms need to perform the Gaussian elimination (GE) with complexity of order O(k3). Even worse, the GE is a serial algorithm and no efficient parallel implementation is available for a general matrix, inevitably causing extra decoding complexity and latency although the hardware optimization is gradually improved in the literature [10], [11], [12]. This can be alleviated, if the MRB is not mandatory as argued in [13], for RS codes by employing the quasi-OSD, where the GE is implemented in parallel by Lagrange interpolation [14] and for the staircase matrix codes by employing the representative OSD (ROSD) [15], [16], where the GE can be implemented in parallel. Furthermore, the GE process can be skipped by precalculating and storing multiple systematic generator matrices [17], and invoking specific decoding conditions [18]. Actually, by pre-storing a systematic generator matrix, lowcomplexity reduced GE is sufficient for performing modified OSD [19].
In this paper, we focus on the generalized RS codes, which are capacity-achieving (demonstrated by theoretical proof) in the asymptotic regime as the code length tends to infinity and are capacity-approaching (demonstrated by the performance bounds and the numerical results) in the finite-length regime. The main contributions of this paper are listed as follows.
• Coding theorem: In the infinite-length region, we prove that the generalized RS code is capacity-achieving over binary-input output-symmetric (BIOS) channels, in terms of FER under ML decoding.
• Decoding algorithm: We present the LC-OSD algorithm for generalized RS codes, in which the systematic matrix is generated in parallel by the Lagrange interpolation. Furthermore, the extended MRB can be derived through a change-of-basis on the systematic matrix. This implementation accelerates the conventional GE operation and thus reduces the decoding latency.

III. PROOF OF MAIN CODING THEOREM

A. Generalized RS Code Ensemble

To prove the coding theorem for the generalized RS code over BIOS channels, we introduce the following generalized RS code ensemble.
Generalized RS code ensemble: A generalized RS code ensemble transforms u into c by c = uG ∈ Fnq with the generator matrix G = GRSΛΠ of size k × n, where Λ is a diagonal matrix of order n with elements on diagonal being drawn independently and uniformly from Fq \ {0} and Π is a permutation matrix of order n being generated uniformly at random.
It is well known that Theorem 1 holds if we do not restrict our attention to the generalized RS codes. Actually, by introducing a totally random linear code ensemble, we can prove the existence of capacity-achieving linear codes over BIOS channels [2, Theorem 6.2.1]. However, the conventional proof is not applicable to the generalized RS code ensemble due to the following challenging issues.

• The codewords in the totally random linear code ensemble are pair-wise independent, while the codewords in the generalized RS code ensemble are not. Actually, any two distinct codewords in the generalized RS code ensemble have a Hamming distance of at least n−k+1. Particularly, a nonzero codeword in the generalized RS code ensemble must have a Hamming weight greater than or equal to n − k + 1.
• The non-zero codewords in the totally random linear code ensemble are identically distributed, while the non-zero codewords in the generalized RS code ensemble are not. Actually, if a vector u is encoded into an RS codeword of Hamming weight WH (uGRS) = w, then it will be encoded into a random vector uG in the generalized RS code ensemble such that, for any vector c with WH (c) = w,

B. Parallel Interpolation and Change-of-Basis
The main issue of the LC-OSD algorithm is the delay caused by the GE, which has no efficient parallel implementation available for general matrices. To reduce the delay caused by the GE, we first employ the parallel Lagrange interpolation to derive the systematic generator and then apply a method similar to the modified OSD algorithm [19] and the iterative basis update (IBU) method [30] to improve the quality of the basis, resulting in the extended MRB of size K + δ.
Upon receiving y, we calculate the LLR vector r by (20) and the hard-decision vector z by (21). The hard-decision bit sequence z is also written as a hard-decision symbol sequence ζ = (ζ0, ζ1, . . ., ζn−1) = φ(z) with ζp ∈ Fq, 0 ≤ p < n. By sorting the LLR vector r, we can find the most K reliable bits in z, whose indices are denoted collectively by Z∗. Then, for two components ζi and ζj of the hard-decision symbol sequence ζ, we say that ζi is more reliable2 than ζj if ζi contains more bits in Z∗ than ζj does. In the case when two symbols contain the same number of bits in Z∗, the symbol with a greater sum of bit reliabilities is considered more reliable. B. Encoding
The encoding process in our proposed JSCC scheme is
divided into three stages, as shown in Fig. 3.
1) CE Encoding: The CE encoder [38], [21] takes as input a message vector v ∈ F2K generated from a Bernoulli source V with PV (1) = θ (0 < θ < 1) and delivers as output an integer w whenever v ∈ Vw and a vector u ∈ F2Kw that is the binary representation of the rank of v in lexicographic order in the subset Vw. This binary representation works since Kw ≥ log2 |Vw|.
2) Multi-Rate Generalized RS Encoding: Typically, the CE
coding is fixed-to-variable, so we employ the variable-to-
fixed generalized RS coding for ease of synchronization. The compressed sequence u ∈ FK2 w for v ∈ Vw serves as the input to a flexible multiple-rate generalized RS encoder, resulting in a codeword c ∈ C (b)[N, Kw] ⊆ C (b)[N, K] of fixed length N.
3) Class Encoding: The codeword c is superimposed by the w-th class label sw to form a channel input vector x = c+sw.

To reduce the decoding latency, we may discard the type classes with too small probabilities or simply merge them into a incompressible classes [39]. Precisely, given two positive number τp and τb, any type classes with too small probability Pw ≤ τp or incompressible description length Lw ≥ τb is assigned to the subset VJ−1. We say that v ∈ Vw is incompressible if the length Lw is similar to K. Then we apply the k-means++ clustering algorithm [22], one of the most powerful and popular data mining algorithms, to derive the J − 1 subsets Vi (0 ≤ i < J − 1) by merging the unclassified type classes. We treat each unclassified type class w as a data sample, represented as a vector of attribute values: (Lw, − log(Pw), log (Fw)). We normalize these three attributes using min-max normalization [40] and perform the k-means++ clustering algorithm, which employs the Manhattan distance as the distance measure. Given the sensitivity to the selection of the initial clustering centers, we execute the clustering t times and compute the upper bound on JSCC performance (44) for each clustering outcome, delivering the partition result that corresponds to the optimal upper bound. VI. CONCLUSION
We have proved that the generalized RS code is capacityachieving over BIOS channels, in terms of FER under ML decoding. In the finite-length region, we have presented the LC-OSD algorithm for generalized RS codes, where the extended MRB is derived through the change-of-basis on a systematic matrix obtained in parallel by Lagrange interpolation. This implementation accelerates the conventional GE operation and reduces the decoding latency. Additionally, by incorporating generalized RS codes and CE coding, we have proposed a JSCC scheme, which outperforms the existing JSCC schemes by at least half dB at FER around 10−4.
