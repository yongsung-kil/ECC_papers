A Decoder with Reinforcement Learning Feedback
Milad Taghipour, Bane Vasic´, Fellow, IEEE Department of Electrical and Computer Engineering, the University of Arizona, Tucson, AZ
Email: miladt@arizona.edu, vasic@ece.arizona.edu

Abstract—This paper explores the application of reinforcement learning techniques to improve the performance of bit-flipping decoders and finding their optimal decisions. We begin by providing an overview of bit-flipping based decoders and reinforcement learning algorithms. We then outline the methodology for mapping the iterative decoding process into Markov Decision Processes (MDPs). We propose a feedback-based method to exploit and enhance the performance of existing well-established decoders by applying reinforcement learning algorithms after a selected decoder. In essence, we modify the MDP to reduce the number of states, making reinforcement learning algorithms feasible for lowrate and long-length codes. In addition, focusing on the correction of dominant error patterns of a selected decoder improves its error correction capability. Finally, we present experimental results for the Binary Symmetric Channel (BSC) to demonstrate the efficiency of the proposed methods.
Index Terms—Reinforcement Learning, bit-flipping algorithm, Iterative Decoders, trapping sets, linear Block codes, LDPC codes, Error-floor.
I. INTRODUCTION
E RROR correcting codes are critical for reliable communication over noisy channels. While LDPC codes and iterative decoders can asymptotically approach channel capacity, their performance in the finite-length regime remains suboptimal. This limitation is primarily due to trapping sets [1], [2] and pseudocodewords [3], which often have lower weight than the minimum distance of the code [4], leading to error floors at high SNR. These structures hinder iterative decoders from achieving maximum likelihood performance. To mitigate this, several approaches have been proposed, including modifications to belief propagation [5]–[8] and the use of Finite Alphabet Iterative Decoders (FAIDs) [9], [10]. Recent advances in artificial intelligence have motivated interest in designing efficient decoders for finite-length codes. Developing efficient, high-performance decoders with low computational complexity remains challenging for short to medium-length codes. One approach involves parameterized belief propagation (BP) decoders [11]–[14], which retain the advantages of standard BP while improving convergence speed and error-correction performance. The authors in [15] presented the generalization gap of the Neural BP decoder. An alternative approach is to view the problem of decoding as a decision-making problem,

Similarly, [18] formulates belief propagation scheduling as an MDP, optimizing check-node scheduling to improve performance over conventional flooding strategies. In this paper, we propose a feedback decoding strategy designed to further enhance the performance of well-established iterative decoders in the literature. This approach leverages an RL block to address error patterns that a decoder fails to correct, thereby effectively expanding its correctable region. In general, this method can be employed to improve state-of-the-art decoders and strengthen their error correction capabilities. The remainder of this paper is organized as follows. II. CHANNEL CODING BACKGROUND
Let us assume C to be a binary linear code with parameters (n, k, dmin), where n is the code length, k is the code dimension, and dmin is the minimum distance of the code. We define a binary linear code C = {c ∈ Fn2 | cHT = 0}, where H is the parity-check matrix, which is an m × n binary matrix. We also define hci as the i-th column of the parity check matrix H. The error correction capability of a code can be defined on the basis of its minimum distance as t = ⌊(dmin − 1)/2⌋. A codeword c is transmitted through the Binary Symmetric Channel (BSC) as y = (1 − 2c) + z, where y is the received row vector of length n, and z is a row vector of length n generated by i.i.d. Bernoulli random variable with parameter ρ which is the cross-over probability of the channel.
Let y = (y1, y2, · · · , yn) denote the input provided to the decoder, and let xℓ = (xℓ1, xℓ2, · · · , xnℓ ) for ℓ ≤ D represent the output vector at the ℓ-th iteration. The decoder operates until it either finds a valid codeword or reaches the predefined maximum number of iterations D, whichever comes first. As a result, the decoder’s final output will be either a valid codeword or xD. The bit-flipping algorithm is a fast, simple yet effective algorithm in the decoding of error correcting codes such as LDPC codes. The idea is to flip bits that are the least reliable to maximize the number of satisfied checks [19].

A. State and Action Spaces
Given that the code dimension is n, each of these n bits can be erroneous. In other words, the action space A should be designed to allow the flipping of each bit at any time step. Therefore, the set of actions is A = {1, 2, · · · , n}. The decoding process can be terminated based on the value of the syndrome vector, which indicates the positions of unsatisfied checks in the code. Consequently, the state space S is defined by all possible binary syndromes of length m. The initial state

s0 = yHT represents the syndrome of the received vector. Next, we need to define the transition probability function P (s′ | s, a) for an MDP. By taking action a from state s, we transition to state s′ = s + hca with probability 1, indicating that it is a deterministic MDP. Terminal state corresponds to
the all-zero syndrome, indicating that a codeword is obtained
and that the decoding process should be terminated. We also impose a limit of L steps for decoding.

D. Deep Q-learning Algorithm

Deep Q-learning is an extension of Q-learning, a popular
RL algorithm, that uses deep neural networks to approximate
the Q-function [21]. Q-learning stores a real-valued table of size |S| × |A|, which is infeasible for low-rate or long codes. We assume Qθ(s, a) represents the estimation of the Q-function with learnable parameters θ. We store transition tuples (s, a, r, s′) obtained from simulating the MDP in a set B called replay memory. For stable training, we use two neural networks a primary Q-network Qθp(s, a) for selecting actions and a target Q-network Qθt (s, a) to provide more stable Qvalue updates. V. NUMERICAL RESULTS

In this section, we present numerical results for the Tanner code with parameters (155, 64, 20). We select the arbitrary decoder Φ to be the bit-flipping decoder, which, in general, can be any well-established code and decoder pair from the literature. The goal is to enhance its performance through the integration of the RL block. The bounded error weight enumerator of the Tanner code under the BF decoder, around all-zero codeword, is E(w = 4) = 10092825x4 + 154225x3 + 620x2, where w denotes the maximum weight of the error patterns considered during decoding. The multiplicity of xi represents the number of uncorrectable weight-i errors under BF decoder. In addition, no miscorrections are observed for this code when employing the BF decoder for error patterns with weights of at most 4. In Figure 2, we illustrate the performance of the bit-flipping algorithm alongside truncated MDPs with radii w = {2, 3},

trained using the Q-learning, and radii w = {2, 3, 4}, trained using the Deep Q-learning algorithms. We also present the frame error rate (FER) estimates in the error floor region for the feedback model learned using the Q-learning algorithm based on their uncorrectable error patterns. The results demonstrate that, instead of training the RL block to learn the entire code space, we can use the existing bit-flipping algorithm and incorporate the RL block as a feedback architecture to enhance its performance. The Q-learning algorithm with w = {2, 3} achieves improved performance with guaranteed error correction. However, for w = 4, the state space becomes prohibitively large, making it impractical to use a Q-table. In this case, we train only the Deep Q-Network. Using the Deep-Q network, for w = 2, it does not effectively learn the decision boundary due to the small radius of the Hamming ball. However, for w = 3, as shown by comparison with the Q-table results, the network is able to generalize the optimal policy and successfully correct error patterns were not encountered during training. For w = 4, further performance improvement is observed in comparison to w = 3.
VI. CONCLUSIONS
In this paper, we propose a novel feedback method aimed at reducing the number of states in the MDP. Additionally, we focus on enhancing the performance of a chosen decoder by targeting only those error patterns that the decoder fails to correct. In essence, the feedback method seeks to efficiently expand the correctable region of the decoder. In general, our goal is to leverage the performance of existing well-designed decoders and refine their state space and using RL algorithms to further enhance their decoding performance.
