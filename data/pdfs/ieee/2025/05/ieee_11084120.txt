Abstract—Low-Density Parity-Check (LDPC) codes are 
essential for error correction in space communications, 
particularly in CCSDS implementations. Space communication 
channels face interference challenges due to extreme distances 
and signal attenuation. While Belief Propagation (BP) and 
Normalized Min-Sum Algorithm (NMSA) are widely used 
decoding techniques, quantitative comparisons for short-length 
QC-LDPC codes remain insufficient. This paper addresses this 
gap through systematic evaluation. This article implemented 
QC-LDPC encoding structures per CCSDS standards and 
conducted MATLAB simulations using BPSK modulation over 
AWGN channels. Simulations compared 1/2-rate codes across 
different lengths which including 320, 640, 1280 bits, and 
analyzed performance across rates (1/2, 2/3, 4/5) with fixed 
information length. Results show performance improves 
logarithmically with code length. BP outperforms NMSA by 0.2-
0.3dB at low SNRs, while differences narrow at moderate SNRs. 
At high SNRs, NMSA occasionally surpasses BP due to better 
numerical stability. These findings provide reference data for 
spacecraft communication system designers facing power and 
computational constraints. 
Keywords—LDPC, BP, NMSA, BPSK, AWGN  
I. INTRODUCTION 
Low-Density Parity-Check (LDPC) codes have emerged 
as fundamental error correction techniques in modern space 
communications, offering superior performance near 
Shannon's limit [1]. The Consultative Committee for Space 
Data Systems (CCSDS) has standardized Quasi-Cyclic 
LDPC (QC-LDPC) codes for spacecraft telemetry due to their 
exceptional error correction capabilities and hardware-
friendly implementation characteristics. This standardization 
addresses critical challenges in deep space missions where 
signal reliability is compromised by extreme transmission 
distances, resulting in significant signal attenuation and 
vulnerable communication links, necessitating robust error 
correction mechanisms to maintain data integrity under 
severe channel constraints [2]. 
Despite extensive research on LDPC decoding algorithms, 
comparative analyses of Belief Propagation (BP) and 
Normalized 
Min-Sum 
Algorithm 
(NMSA) 
decoders 
specifically for CCSDS standard short-length QC-LDPC 
codes remain insufficient. While BP theoretically achieves 
optimal performance through precise probability propagation, 
NMSA offers computational efficiency with reduced 
hardware complexity [3]. However, the quantitative 
performance gap between these algorithms under various 
code lengths, rates, and signal-to-noise ratio (SNR) 
conditions has not been comprehensively characterized for 
space communication applications, particularly when dealing 
with short code length. 
This paper presents a systematic performance evaluation 
of BP and NMSA decoding algorithms for CCSDS standard 
QC-LDPC codes in space communication channels. We 
implement multiple simulation scenarios: first comparing 
1/2-rate codes across different code lengths (320, 640, and 
1280 bits), then analyzing performance across different code 
rates (1/2, 2/3, and 4/5) with fixed information length (512 
bits). Our contribution includes quantifying the relationship 
between decoding iterations and energy efficiency for both 
algorithms, establishing performance boundaries for short-
length QC-LDPC codes under AWGN channel conditions, 
and providing practical guidelines for algorithm selection 
based on operational SNR regions. This research offers 
reference data for spacecraft communication system 
designers facing stringent power and computational 
constraints.

capabilities, particularly when implemented on FPGA 
platforms where resource consumption is critical[4]. The

The generator matrix is derived from the parity check 
matrix through systematic block multiplication. This process 
involves partitioning the check matrix H into [Q,P] 
components, where P is a square matrix, followed by 
computing W = (P^(-1)Q)^T in GF(2) [6]. The resulting 
generator matrix G = [I,W] in systematic form facilitates low-
complexity encoding operations. 
The CCSDS standard defines specific structures for QC-
LDPC codes with different code rates through a modular 
matrix construction approach. For rate 1/2 codes, the parity 
check matrix consists of a 3×5 grid of sub-matrices, each of 
size M×M, yielding a total matrix dimension of 3M×5M. This 
structure results in a code length N=5M and information 
length K=2M, producing an actual rate of K/N=0.4, slightly 
below the nominal rate. For rate 2/3 codes, the matrix is 
constructed by concatenating a specific 3M×2M sub-matrix 
(H_23) with the rate 1/2 matrix, resulting in dimensions 
3M×7M, code length N=7M, and information length K=4M 
(actual rate 4/7≈0.571). Similarly, for rate 4/5 codes, the 
matrix further expands to include another specific 3M×4M 
sub-matrix (H_45), creating a 3M×11M structure with code 
length N=11M and information length K=8M (actual rate 
8/11≈0.727). The parameter M (SIZE_M in implementation) 
thus directly determines the dimension of sub-matrices and 
consequently scales the overall code length according to the 
chosen rate configuration. 
The encoding implementation incorporates optimized 
computational techniques, including adaptive Monte Carlo 
simulation with enhanced sampling in high signal-to-noise 
ratio regions. This approach findings that proper construction 
of circulant matrices in QC-LDPC codes can eliminate short 
cycles in the Tanner graph, thereby enhancing minimum 
distance properties and improving iterative decoding 
performance under challenging channel conditions without 
requiring complex masking techniques typically employed in 
other QC-LDPC constructions [7]. 
B. Channel Transmission Mode 
The simulation employs Binary Phase Shift Keying 
(BPSK) modulation, transforming the encoded binary 
sequences into bipolar signals through a mapping function: -
2*bit+1, which converts logical 0 to +1 and logical 1 to -1. 
These modulated signals are then transmitted through an 
Additive White Gaussian Noise (AWGN) channel using 
MATLAB's built-in 'awgn' function with the 'measured' power 
option for accurate noise scaling. The performance evaluation 
spans signal-to-noise ratios (SNR) from -1 dB to 4 dB in 0.5 
dB increments, providing a comprehensive assessment across 
challenging 
to 
favorable 
channel 
conditions. 
This 
configuration enables thorough analysis of the LDPC codes' 
error correction capabilities in practical communication 
scenarios where thermal noise is the predominant impairment.

III. MATLAB SIMULATION AND RESULTS ANALYSIS 
A. Simulation Environment Setting 
This paper presents a comparative performance analysis of 
BP and NMSA decoding algorithms for QC-LDPC codes 
under BPSK modulation over AWGN channels. Author 
implemented two simulation configurations: (1) CCSDS QC-
LDPC codes with codeword lengths n=320/640/1280 at rate 
r=1/2, and (2) CCSDS QC-LDPC codes with fixed 
information length k=512 at rates R=1/2, 2/3, and 4/5. 
Author’s adaptive framework employs dynamic Monte Carlo 
simulations that intelligently adjust iteration counts based on 
SNR values. For high SNR regions where errors are rare, I 
dramatically increase simulation runs using an exponential 
growth formula: MtCa_NUM = base_MtCa_NUM × 
10^((SNR-2)/2). This means at SNR=4dB, we run 10 times 
more simulations than at SNR=2dB, ensuring sufficient error 
events are captured for accurate BER estimation. 
The simulation automatically terminates early when 
statistical reliability is achieved, saving computational 
resources. This is determined by two practical criteria: (1) 
collecting enough error events (100-500 minimum bits 
depending on SNR region) and (2) achieving a sufficiently 
narrow confidence interval. The confidence interval is 
calculated using basic probability theory - for each BER 
estimate, we compute how precisely we know the true error 
rate by measuring the spread of possible values (using the 
formula σ=√(BER×(1-BER)/N), where N is the total bits 
tested). When this interval becomes less than 30% of the 
measured BER at 95% confidence level, we conclude the 
result is statistically reliable and stop the simulation. All tests 
were conducted with 15 maximum decoder iterations with 
parallel processing capabilities, providing comprehensive 
performance comparison while significantly reducing 
simulation time.

Figure 3 and Figure 4 present the BER performance curves 
comparing BP and NMSA algorithms under various code 
configurations, clearly demonstrating the performance trade-
offs between these decoding approaches across different SNR 
regions. Examining code lengths (n=320, 640, 1280) at fixed 
rate (R=1/2), both algorithms demonstrate improved 
performance with increasing code length, exhibiting steeper 
waterfall regions and lower error floors. The coding gain 
achieved through increased redundancy shows diminishing 
marginal returns - doubling code length from 320 to 640 yields 
approximately 0.7 dB improvement, while the subsequent 
doubling from 640 to 1280 provides only about 0.5 dB 
additional gain. This logarithmic relationship between code 
length and performance gain aligns with Shannon's channel 
coding theorem, which establishes that for codes approaching 
capacity, the relationship between error probability and code 
length is exponential. Specifically, as code length increases, 
the achievable gap to capacity reduces logarithmically, 
explaining why each doubling of code length yields 
progressively smaller performance improvements in decibels. 
Comparing coding rates (R=1/2, 2/3, 4/5) at fixed code 
length (n=512), lower rate codes outperform higher rate codes 
due to increased redundancy. The performance gap between 
R=1/2 and R=2/3 measures approximately 1-1.5 dB at 
BER=10^-5, while the gap between R=2/3 and R=4/5 is 
around 0.7-1 dB. 
The comparative analysis between BP and NMSA reveals 
an 
intriguing 
performance 
pattern. 
BP 
consistently 
outperforms NMSA in low SNR regions (below 1 dB). As 
SNR increases (1-3 dB), NMSA approaches BP performance. 
Remarkably, in high SNR regions (above 3 dB), NMSA 
occasionally surpasses BP, particularly for shorter code 
lengths. This counterintuitive phenomenon can be attributed 
to several mathematical and computational factors: 
BP algorithm's optimality assumes infinite precision 
arithmetic, but in practical implementations, numerical 
instabilities arise at high SNR where LLR values become 
extremely large. These large values can lead to overflow, 
underflow, or precision loss in floating-point operations, 
particularly in the tanh/atanh calculations used in BP. The 
chosen normalization factor of 0.75 in NMSA effectively 
compensates for the overestimation of check-node messages 
that occurs in the min-sum approximation, providing near-
optimal correction specifically at higher SNR ranges. 
NMSA's simpler computational structure avoids the 
accumulation of rounding errors that affect BP's more 
complex calculations. At high SNR, where few errors occur, 
these accumulated numerical inaccuracies in BP can 
occasionally lead to incorrect convergence or failure to 
converge within the specified iteration limit, while NMSA 
remains robust due to its simplified message update rules. 
These results demonstrate that while BP remains 
theoretically optimal under ideal conditions, properly 
normalized NMSA provides comparable or even superior 
performance in practical high-SNR operating conditions, 
making it an attractive implementation choice for 
communication systems. 
C. Discussion 
The performance analysis has revealed several important 
technical 
implications 
for 
practical 
LDPC 
code 
implementation. The observed performance differences 
between BP and NMSA decoders at varying SNR regions 
suggest optimal decoder selection strategies for different 
operating conditions. 
1) Technical implications 
The simulation results provide clear engineering 
parameter insights for QC-LDPC implementation. For low 
SNR environments (below 1dB), the superior performance of 
BP decoding (approximately 0.2-0.3dB advantage over 
NMSA) justifies its higher computational complexity that BP 
maintains optimality under ideal low-SNR conditions where 
message passing benefits from numerous iterations. However, 
as SNR increases beyond 3dB, NMSA's comparable or 
occasionally superior performance suggests it becomes the 
preferred algorithm. This performance inversion phenomenon 
can be attributed to the numerical precision challenges that 
BP's multiplicative error propagation becomes particularly 
problematic in high-SNR scenarios [15]. 
2) Project strengths, limitations and future improvements 
A key strength of this approach lies in the comprehensive 
statistical verification methodology employed, ensuring 
reliable BER performance measurement through confidence 
interval analysis and adaptive iteration control. This statistical 
rigor addresses a common weakness in communications 
research where insufficient error events can lead to misleading 
performance characterization. 
However, the project has notable limitations. Most 
significantly, it lacks quantitative algorithmic complexity 
analysis and visualization. Future work should include 
detailed 
computational 
complexity 
measurements, 
particularly 
comparing 
operation 
counts 
(additions, 
multiplications, and LUT operations between BP and NMSA 
[7]. Additionally, power consumption analysis would provide 
critical insights for hardware implementation considerations, 
especially for mobile and energy-constrained applications. 
The project could be further enhanced by exploring 
additional decoder variations, such as layered BP or shuffled 
decoding strategies, which might provide intermediate 
complexity-performance 
trade-offs. 
Implementation 
of 
variable node degree optimization, such as 2D-Normalized 
Min-Sum approach, might also yield improved performance 
without significant complexity increases [15]. 
IV. CONCLUSION 
This study established a comprehensive analysis model for 
short-length QC-LDPC codes based on CCSDS standards, 
comparing the performance of BP and NMSA decoding 
algorithms under varying SNR conditions. While CCSDS 
standards typically employ codes with lengths exceeding 3000 
bits, this research specifically investigates performance 
characteristics in the short code length regime (320-1280 bits), 
revealing important insights about decoder behavior in this 
understudied domain. 
The results demonstrate that even with short code lengths, 
QC-LDPC performance improves logarithmically as length 
increases, with diminishing returns when doubling from 320 
to 1280 bits. Under fixed information length (512 bits), lower 
rate codes consistently outperform higher rate variants, with 
the 1/2 rate code showing approximately 1.7-2.5 dB advantage 
over the 4/5 rate code at BER=10^-5. 
BP and NMSA algorithms exhibit distinct performance 
characteristics across SNR regions. At low SNRs (below 1dB),

operating at lower SNRs despite its higher computational

potential for adaptive decoder selection strategies that could 
optimize error correction while managing computational 
resources effectively. 
Future research should explore these short-length codes 
under multi-level modulation schemes (QPSK/OQPSK) and 
complex channel conditions including Doppler effects and 
fading, as well as investigate hardware-specific optimizations 
for power-constrained space applications.
