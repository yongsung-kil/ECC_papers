Generalized Fourier Transform Pair Codes
Yaping Lv , Suihua Cai , and Xiao Ma , Member, IEEE

Abstract— This letter is concerned with Fourier transform pair (FTP) codes over finite fields. Given any element of order n in a finite field, we can construct an FTP code √with length 2n, dimension n, and minimum distance at least 2 n. Distinguishingly, an FTP code defined with any element (if it exists) of order 2, 3 or 5 in a finite field is a maximum distance separable code. In this letter, we extend the FTP codes to generalized FTP (GFTP) codes. To show the superiority of the GFTP codes over the FTP codes, we compute the binary weight spectra of the GFTP codes, approaching more close than the FTP codes to that of the random codes. We also apply the ordered statistic decoding with local constraints (LC-OSD) algorithm to the binary image of GFTP codes. Numerical results show that the GFTP codes perform better than the FTP codes as the code length increases. Numerical results also show that GFTP codes are comparable with existing codes of the same code rate and similar code length.
Index Terms— Fourier transform codes, generalized Fourier transform codes, maximum distance separable codes, ordered statistic decoding with local constraints, URLLC.
I. INTRODUCTION
WITH the advent of 5G, ultra-reliable and low latency communication (URLLC) has emerged as a critical enabler for various applications, such as autonomous driving, industrial automation, and mission-critical Internet of Things (IoT). In particular, URLLC demands stringent requirements on both reliability and latency, which mandates the use of short block-length codes. To this end, designing short codes and/or related decoding algorithms has attracted a great deal of interest [1], [2], [3].
Existing works focused on exploring the short codes that can approach the finite-length capacity [4], such as polar codes, low-density parity-check (LDPC) codes, and Bose-Chaudhuri Hocquenghem (BCH) codes. Polar codes, decoded by successive cancellation list (SCL) decoding with a large list, can approach maximum likelihood (ML) performance. LDPC codes have been shown to perform very close to Shannon’ limit at large block-lengths. BCH codes have excellent error-correcting properties and algebraic structure. However, their application is often limited by the lack of efficient decoding algorithms that can be implemented with low latency, posing a challenge for URLLC.

As a near ML decoding algorithm, the ordered statistic decoding (OSD) [5], [6] algorithm has attracted much attention. For binary transmission, the basic idea of the original OSD algorithm is to generate a list of candidate codewords by flipping a small number of bits in the most reliable basis (MRB). Compared with the original OSD algorithm, which chooses the k most reliable positions as MRB for re-encoding, the OSD with local constraints (LC-OSD) algorithm [7], [8] chooses the k + δ (δ ≥ 0) most reliable positions as an extended MRB. In particular, the reprocessing of LC-OSD can be implemented by performing the serial list Viterbi algorithm (SLVA) [9] over the locally constrained trellis for the k + δ bits, where δ is a parameter the LC-OSD algorithm.
Recently, Fourier transform pair (FTP) code [10] with dimension n and length 2n over a finite fie√ld was proposed, which has a minimum distance at least 2 n. Particularly, an FTP code defined with any element of order 3 in a finite field is proved to be a maximum distance separable (MDS) code. In this letter, we propose a new class of short codes, called generalized FTP (GFTP) codes. Simulation results show that the GFTP codes outperforms FTP codes. Compared with existing codes with the same code rates and similar code length, GFTP codes perform similarly. Compared with the prior work in [10], the contributions of this letter are as follows.
• We prove that the FTP codes with dimension 5 and length 10 are MDS codes, in addition to the MDS property of FTP codes with dimension 3 and length 6.
• We propose the GFTP codes, whose binary weight spectrum is better than that of the corresponding FTP codes.
• To show the superiority of the GFTP codes over the FTP codes, we apply the LC-OSD algorithm to the binary image of GFTP codes. Numerical results show that the GFTP codes perform better than the FTP codes as the code length increases. Numerical results also show that GFTP codes are comparable with existing codes of the same code rate and similar code length.

IV. DECODING OF SHORT GFTP CODES
We have seen that binary image of GFTP codes can have weight spectrum similar to random codes, suggesting good performance. However, no efficient decoding algorithms are available for large m. In this section, we investigate the performance of GFTP codes of short length by performing the LC-OSD algorithms [7], [8].

B. Numerical Results
In this section, we present simulation results to demonstrate the performance of the GFTP codes. All simulations are conducted by assuming BPSK modulation and AWGN channels.
We simulate the GFTP codes CGFTP[10, 5] over F2m , extended BCH (eBCH) code CeBCH[128, 64], LDPC code [15] CLDPC[128, 64] and cyclic redundancy check aided Polar (CA-polar) code CCA-polar[128, 64]. The simulation results are shown in Figs. 2 and 3, from which we see that, for a given m, the codes CG(bF)TP outperform the binary images of codes CFTP[10, 5], exhibiting a comparable decoding complexity and a similar performance to the CeBCH[128, 64] and CLDPC[128, 64] of the same code rate. The results also show that the performance improves as m increases, which is as expected since the code length of the binary image of CGFTP[10, 5] increases linearly with m. In particular, the code CGFTP[10, 5] with m = 12 approaches the random coding union (RCU) bound [4].
V. CONCLUSION We have presented an approach to construction of short codes by transforming GFTP codes into their binary images.
