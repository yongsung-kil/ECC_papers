Optimal Degree Distribution for Two-User Regular LDPC Codes Over GMAC by Fixed-Point Analysis
Meilin He , Mingjue Zhu, Zhaoyang Zhang , Senior Member, IEEE, Rui Guo , and Haiquan Wang

AbstractŌĆö An analytical expression of the check nodes (CNs) degree in a two-user regular Low-Density Parity-Check (LDPC) code is derived over a Gaussian multiple access channel, and a Fixed Point Analysis (FPA) method is developed to determine the optimal degree distribution that maximizes the achievable sum rate of the system. Specifically, firstly, an analytical expression of the CNs degree is derived as a function of mutual information outputs. Secondly, based on this, a reliable region is obtained by taking a complement of a unified unreliable region. Finally, the optimal degree distribution is determined over this reliable region to achieve the optimal sum rate. Numerical results indicate that the proposed method offers superior accuracy and much lower computational complexity compared to the extrinsic information transfer (EXIT) chart.
Index TermsŌĆö LDPC code, FPA method, degree distribution, sum rate, EXIT chart.
I. INTRODUCTION
THE sixth generation (6G) communication presents more stringent requirements in terms of high data rates, high reliability and low latency [1]. The demand of these requirements poses a challenge for existing multiple access technologies. Several promising multiple access technologies, such as interleave-division multiple-access (IDMA) [2], orthogonal frequency division multiple access (OFDMA) [3], and sparse code multiple access (SCMA) [4], over multiple access channel (MAC) have been well studied.
In MAC, such as two-user Gaussian MAC (GMAC), the channel coding scheme plays a crucial role in affecting system performance. It is essential to employ an appropriate channel coding scheme to achieve capacity-approaching performance. Low-Density Parity-Check (LDPC) code is regarded as a powerful channel code for its outstanding coding efficiency and excellent error correction performance [5]. It is known that a well-designed degree distribution of LDPC code can realize a code rate that approaches the Shannon limit. Therefore, it is desired to design an optimal degree distribution of LDPC code.
For equal-power two-user GMAC, extrinsic information transfer (EXIT) chart has been employed to optimize the

For unequal-power two-user GMAC [7], an alternating linear programming (LP) method, where EXIT chart is utilized to derive the constraints, is developed to optimize the degree distribution of LDPC code for approaching the maximum system rate. Specifically, the variable node degree distribution of one user is fixed, while the other userŌĆÖs variable node degree distribution is optimized by solving LP problem alternately. Unfortunately, the conventional EXIT chart and LP method stated above may be not suitable for protograph-based LDPC code [8]. To address this constraint, modified protograph-based EXIT and joint protograph-based EXIT (JP-EXIT) charts [9], [10] are proposed to design the protograph-based LDPC code for two-user GMAC. JPEXIT chart is developed [11] to design double-protograph LDPC code by introducing global and separating strategies for decoding threshold analysis. However, these EXIT-based methods above are limited by numerical precision and rely on iterative operations. Therefore, it is desired to derive the analytic expressions of code parameters.
Motivated by the aforementioned issues, an analytic expression of the repeat number in repeat-accumulate (RA) codes [12], which is formulated as a function of mutual information by fixed-point analysis (FPA), is derived for two-user GMAC to achieve the maximum system rate. For multiuser GMAC, a bivariate FPA [13], [14], which introduces an auxiliary variable to establish the system equation array of the bivariate fixed-point, is developed to obtain the analytic expressions of the repeat number and spreading length. In [15], [16], the upper bounds on the girth of the regular and irregular RA codes are derived as the constraints to design a globally optimized sparse submatrix in the parity-check matrix and a semi-randomly irregular RA code, respectively.
It is worth noting that these works above only focused on RA code, which is regarded as a special case of LDPC code. LDPC code possesses the desirable characteristic of approaching the Shannon limit [17], thus it is potentially insightful to derive analytic expression of its code parameters.
In this letter, we derive an analytical expression of the check nodes (CNs) degree in a two-user regular LDPC code over a GMAC, and develop a FPA method to obtain the optimal degree distribution that maximizes the achievable sum rate of the system. Specifically, firstly, we derive an analytical expression of the CNs degree dkc as a function of final MI outputs for a given channel noise level Žā and the variable nodes (VNs) degree dvk. Secondly, based on the above explicit representation, we get a reliable region R, where the two-user regular LDPC code is decodable, by taking the complement of a unified unreliable region. V. CONCLUSION
We have derived an analytical expression of the CNs degree in a two-user regular LDPC code over a GMAC, and developed a FPA method to determine the optimal degree distribution that maximizes the achievable sum rate of our system. Specifically, we first derived an explicit expression of the CNs degree dkc as a function of the MI outputs. Based on this analytical representation, we then identified a reliable region R, within which the two-user regular LDPC code is decodable, by taking the complement of a unified unreliable region. Finally, leveraging this reliable region R, we selected the optimal degree distribution (dkvo , dcko ) that gives the optimal sum rate ro. The numerical results validate the accuracy of our proposed FPA method and demonstrate its lower computational complexity compared to the EXIT chart, making it a more efficient approach for optimizing LDPC codes in multiple-access systems.
In this letter, we mainly focused on the design of two-user regular LDPC code. In multi-user scenario, it may also be appropriate to consider various factors, such as spreading sequence, user rate assignment (equal-rate and unequal-rate transmission), protograph-based LDPC code, and so on. How to extend our proposed method to multi-user scenario for designing system parameters is an interesting issue for further investigation.
