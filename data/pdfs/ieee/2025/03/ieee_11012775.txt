Bi-Evaluated Chase-like Decoding for QC-MDPC Code

Takuzo Hirata School of computing Institute of Science Tokyo
Tokyo, Japan hirata.t.8111@m.isct.ac.jp

Haruhiko Kaneko School of computing Institute of Science Tokyo
Tokyo, Japan hkaneko@ieee.org

Abstract—This paper presents a decoding algorithm for quasi-cyclic moderate-density parity-check (QCMDPC) code. The method flips certain positions beforehand, and then a decoder for QC-MDPC code is executed. This process repeats if decoding fails. The positions to be flipped are determined using the number of unsatisfied parity checks (NUPCs) and β, which is the sum of NUPCs in suspicious positions after tentatively flipping a position. The proposed method is applied to a variant of bit-flipping (BF), and the block error rate (BLER) is evaluated through computer simulation. The result shows that the proposed method significantly reduces the BLER of the BF variant.
Index Terms—post-quantum cryptography, codebased cryptography, QC-MDPC codes, McEliece PKC, bit-flipping decoding.
I. Introduction
Code-based cryptography is one of the promissing candidate for post-quantum cryptography. Especially, McEliece public-key cryptosystem (PKC) [1], which uses Goppa code, is widly considered secure. QC-MDPC McEliece PKC [2] , which uses quasi-cyclic moderate-density paritycheck (QC-MDPC) code, is a variant of the McEliece PKC, and its secret key and publick key are significantly small compared to the original McEliece PKC.
Decryption in QC-MDPC McEliece PKC is performed using a decoding algorithm for QC-MDPC code such as bit-flipping (BF) [3] and its variants with a non-zero failure probability. The block error rate (BLER), or failure probability, needs to be sufficiently small to prevent attacks that exploit BLER such as [4]. Therefore, it is important to find a decoding algorithm that gives low BLER for QC-MDPC code.
To decrease the BLER of QC-MDPC code, this paper presents a decoding method that we call bi-evaluated chase-like decoding (BCLD). The BCLD flips certain positions in the given ciphertext, and the resulting vector is decoded by any decoder for QC-MDPC code. The positions to be flipped are selected based on the number of unsatisfied parity-checks (NUPCs) and β [5], where the definition of both metrics are introduced in Section III.

Chase decoding [7] is known as a decoding algorithm that adds a vector to a received word before decoding. In chase decoding, the vector added to a received word is called test pattern. Similarly, we refer to the vector added to a ciphertext as a test pattern.
Fig. 3 illustrates the decoding flow of the BCLD. Note that dec(·) denotes any decoder for QC-MDPC code. The ciphertext modified by a test pattern is decoded. If decoding fails, a newly generated test pattern is added to the original ciphertext, and the resulting vector is decoded. This process repeats until decoding succeeds or the number of iterations reaches l, where l is a parameter.

V. Numerical results of BLER We denote the decoder that combines the BCLD and MLABF-1 as BCLD+MLABF-1. Specifically, BCLD+MLABF-1 refers to the BCLD where the MLABF-1 is used as dec(·) in line 22 of Algorithm 3. Similaly, BCLD2+MLABF-1 represents the decoder that combines the BCLD2 and MLABF-1. The BLER of the BCLD+MLABF-1, BCLD2+MLABF-1, MLABF-1, BF and weighted BF (WBF) [8] was evaluated through computer simulation for QC-MDPC code with parameters (n, r, w) = (9602, 4801, 45). The BCLD+MLABF-1 and BCLD2+MLABF-1 significantly reduce the BLER compared to the MLABF1, with the decrease in BLER becoming larger as the number of errors t decreases. The decoder that achieves the lowest BLER is the BCLD+MLABF-1, not the BCLD2+MLABF-1, except when t = 110. This indicates that increasing the number of test patterns contributes more to reducing the BLER than decoding the original ciphertext before adding a test pattern, as long as the total number of decodings remains constant.

This paper also introduced a variation of the BCLD, which we call BCLD2. In the BCLD2, the BCLD is performed only when decoding the original ciphertext fails. The BLER of the BCLD and BCLD2 when applied to MLABF-1, which currently is the best-performing BF variant in terms of BLER, was evaluated through computer simulation. The result show that both the BCLD and BCLD2 significantly decrease the BLER compared to the MLABF-1. The BCLD achieves the lowest BLER among the evaluated decoding algorithms when the number of errors in the ciphertext is relatively small.
The followings remain to be addressed in future work: finding better parameters for the BCLD, especially for (d, q), and evaluating the BLER when the BCLD is applied to the ADDM decoding [9], which currently gives the lowest BLER among decoding algorithms for QC-MDPC code, not just among variants of BF.
