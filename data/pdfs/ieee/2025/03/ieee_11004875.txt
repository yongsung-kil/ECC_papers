Performance Analysis of Normalized Min-Sum Decoder for LDPC using non-uniform Quantization

Pargunarajan K Department of Electronics and Communication Engineering
Amrita School of Engineering, Coimbatore Amrita Vishwa Vidyapeetham, India.

Gandhiraj R Department of Electronics and Communication Engineering

Abstract—Very high data rates and decoding computational complexity place strict constraints in 5G and 6G communications. In this paper, non-uniform Quantized Decoding algorithm is proposed for Low-density Parity Check codes (LDPC). In the proposed algorithm, the check node updating process is carried out using quantization, which reduces complexity. The Sum Product Algorithm (SPA) is iterative message passing decoding techniques offer the optimal performance for LDPC codes. Implementations use message-passing decoding with more precision, results extremely high complexity. In Iterative, message-passing (MP), trapping sets and absorbing sets causes low error floor. In this article non-uniform quantization proposed to overcome dynamic range of values over uniform quantization. Simulation results provide coding gain increases closer to of 0.5 dB in higher SNR and has good error floor at 10−5.
Index Terms—Quantization, Trapping set, Box plus operator, Check node (CN), Variable node (VN), LDPC
I. INTRODUCTION
LDPC codes were first proposed by Gallager and rediscovered by Tanner and MacKay gained more of focus in 5G and 6G communication systems as it can approach the Shannon limit [1] [2]. Numerus works have been carried out to improve the encoding and decoding of Low density parity check codes. Graphical representation (Tanner Graph) of the parity check matrix used in the decoding of LDPC. Bipartite graph drawn between variable node(VN) to check node (CN). VN corresponds to bits in the code word and CN corresponds to the parity check condition Decoding is based on a message passing algorithm on this graph. Sum product decoding techniques depends on message passing between CN and VN. Soft information exchange between check node and variable node in the Tanner graph. The Messages computed should be a high resolution [3]. Decoder provides optimum results if it works on tree. Tanner graph is a bipartite has cycles/loops causes degrading in the performance. To overcome cycles, parity generator matrix constructed larger girth. Presence of stopping sets and trappings sets causes failure in decoding, reduce error correction capabilities and causes error floor problem. Margulis proposed new method for construction of LDPC with minimum trapping sets [4] [5]. This paper

focus on the reducing error floor due to trapping sets and uses quantization of log likelihood values for decoding. Sum product algorithm (SPA) provides better decoding, but they are highly computational complex. Min sum algorithm (MSA) is other method of belief propagation has less complexity in implementation. Check node approximation in the min-sum reduces performance. CN update requires the reliability values derived from intrinsic information [6]. In MSA computation of check-node messages leads to degradation compared to the SPA. Correction methods were proposed [7] [8] in the literatures in order to recover the performance loss of the Min-Sum decoding. Normalized min-sum algorithm proposed by Chen [6]. Normalized factor of min sum value determines performance of the decoder depends on the SNR value. To eliminate high complexity extrinsic information computations are quantized and coded with help of normalization. However, it is well known that the performance degrades drastically due to quantization process [9] [10]. To avoid iteration getting into the specific loops selected check nodes are dropped in next iterations. These nodes are selected from trapping and stopping sets. In the literature, various iterative MP decoding algorithms have been proposed to improve error floor performance in higher SNR.

III. PROPOSED METHOD
A. Quantization
Check node update values exponentially increase as a function of iteration number. Message magnitudes quantized to reduce complexity [11]. Uniform quantization increases the number of levels and not good for exponential growth of check node values. VNs, which is not falling in trapping sets, reaches saturation value in minimum iteration. Uniformly quantized decoders not able to escape from the TS due to minimum saturation value. It results problematic VNs not able converge to correct code word. Incorporating different steps also fails due to saturation values. Non-uniform quantization helps to increase the saturation value and allows converging to correct code word. In this paper, multilevel quantization schemes were proposed. Non-uniform quantization increases the dynamic range of the quantizer. Uniform quantization (N = 2q level) applied for linear increase region. When decoder enters in to TS rapid growth occurs and step size scaled by exponential growing factor d and their powers. This proposal used two different exponential growing regions. In binary number representation last two bits (00,11) differentiate which quantization used and one bit used for sign represent

ion. This proposed method require no change in the check node update rules and reduces error floor.

B. Trapping set
A cycle in a graph is a sequence of connected nodes which form a closed loop where the initial and final node are the same and no edge is used more than once. The shortest cycle in graph is called girth. In Figure 3, ‘•’ denotes a VN, hallow square denotes an even degree CN and dark filled square denotes an odd degree CN. For a given parity check matrix H, where odd number of projections set determines trapping set for an element in a row. Trapping set represented as a subgraph of the Tanner graph. A (5,3) trapping set (TS) denoted as small error pattern in the code where a set of 5 VN are ”trapped” by 3 CN. Decoder fails to correct because of the odd number of connections leading to error floor. Trapping set T (5,3) and (4,4) is a sub graph with VN ’a’ and ’b’ odd degree checks plotted in Figure 3a. TS (5,3) in Figure 3a has critical number 3 and TS (4,4) in Figure 3b with critical number 4. When particular variable node ’a’ trying to correct the value base on the other check node value connected to it. Trapping set causes error in other connected variable node ’b’ and ’c’, due to odd number of connections. In next iteration node ’b’ tries to update it values cause error back in ’a’. In the proposed method allows drops one node based on their LLR values dynamic corrections and decode correctly.

The exponential growing region expressed by two bits (00, 11)

and sign represented by one bit. In iterative decoding process,

decoder toggles between error pattern due loops/ cycles of

parity check matrix. If the same error patten repeats more than

six times, node with higher values are dropped in the updating

process. This allows decoder to comes out of the dead lock.

SPA with 100 iterations plotted. It clearly indicates that SPA

significantly better. Simulation results shows that if SNR is
