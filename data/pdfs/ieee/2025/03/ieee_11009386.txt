AbstractŌĆöIn order to improve the performance and reliability of Low-Density Parity-Check (LDPC) code decoding algorithms, this paper proposes a Self-Correcting Minimum Sum algorithm with Dual Reliability Determination (DRD-SCMS) to address the limitations of the Self-Correcting Minimum Sum (SCMS) algorithm in determining the reliability of variable nodes. Unlike the traditional SCMS algorithm, which relies solely on the symbols from the previous and current iterations to assess the reliability of variable nodes, the DRD-SCMS algorithm combines the number of connections between variable nodes and check nodes with check failures, along with the results of symbol iterations, to provide a more accurate reliability evaluation. By categorizing variable nodes into three groups and prioritizing the updating of the most unreliable nodes, the algorithm optimizes the iteration order in the decoding process. Simulation results demonstrate that the DRDSCMS algorithm significantly reduces the number of iterations and improves decoding efficiency compared to the SCMS algorithm, offering better applicability and reliability.
Keywords-Low-Density Parity-Check Codes; Belief Propagation; Minimum Sum Algorithm; Self-Correcting Minimum Sum Algorithm
I. INTRODUCTION
Low-Density Parity-Check (LDPC) codes are a class of linear block codes introduced by Dr. Gallager in 1962[1]. They represent a coding scheme that approaches the Shannon theoretical limit of performance[2]. LDPC codes gained significant attention in the 1990s and have since been increasingly applied in practical scenarios, including various communication standards, deep space communication, and the coding of Beidou III navigation messages. Moreover, in October 2016, LDPC codes were adopted as the coding scheme for the 5G long-code channel[3].When LDPC codes were first introduced by Gallager, they did not attract much attention due to the hardware limitations of the time, which made decoding long codes very difficult. In the 1990s, MacKay and Neal[4] proved that LDPC codes, when combined with belief propagation (BP) iterative decoding, could approach the

As a result, LDPC codes gradually became a research hotspot in the academic community.By simplifying the check node computation process in the BP algorithm, the Minimum Sum (MS) algorithm can be obtained, which saves significant hardware resources in hardware implementations but also results in some loss of decoding performance. To recover the magnitude of the check node information in the MS algorithm and reduce decoding performance loss, optimizations can be made to the MS algorithm by improving both the check node update process and the variable node update process. For improving the check node update calculation process, two basic correction algorithms for check node information magnitude are the Normalized Minimum Sum (NMS) and Offset Minimum Sum (OMS) algorithms, which introduce normalization and offset factors, respectively[5][6], yielding certain decoding performance gains.In reference[7], the normalization factor is optimized using parameter estimation and linear minimum mean square error, improving the decoding performance of the MS algorithm at the cost of a slight increase in algorithm complexity. Reference[8] employs density evolution theory to determine multiple normalization and offset factors updated iteratively, using weighted averaging to balance complexity and decoding performance. In recent years, Savin[9][10] proposed the SelfCorrected Minimum Sum (SCMS) algorithm and its improvements, which enhance decoding performance and accelerate convergence by considering the variable node messages.
The Self-Correcting Minimum Sum (SCMS) algorithm demonstrates good decoding performance in Low-Density Parity-Check (LDPC) code decoding. However, it relies solely on the symbols from the previous and current iterations to assess the reliability of variable nodes, which results in a lack of more accurate reliability evaluation. This limitation becomes particularly apparent in high signal-to-noise ratio (SNR) scenarios, where the SCMS algorithm exhibits poor convergence. To further improve decoding performance and

II. BP ALGORITHM
The BP decoding algorithm in the probability domain involves a large number of multiplication operations, leading to high computational complexity and numerical instability. Therefore, the BP algorithm is implemented in the logarithmic domain, referred to as the LLR-BP decoding algorithm. This algorithm replaces multiplication operations with addition operations, reducing computational complexity and enhancing data stability. IV. DUAL RELIABILITY DECISION SELFCORRECTING MINIMUM SUM ALGORITHM
The traditional Self-Correcting Minimum Sum (SCMS) algorithm in LDPC code decoding determines the reliability of variable nodes by comparing the symbols from two consecutive iterations. This approach achieves a certain level of performance. However, the SCMS algorithm has some limitations, particularly when determining the reliability of messages. It relies on comparing the symbols from the previous and current iterations, which fails to adequately reflect the actual reliability of the variable nodes, especially in high signalto-noise ratio (SNR) environments. This leads to a significant degradation in the algorithm's convergence. Based on the SCMS algorithm, a Dual Reliability Decision Self-Correcting Minimum Sum (DRD-SCMS) algorithm is proposed. This algorithm assesses the reliability of variable nodes by considering the number of connections between variable nodes and check nodes that fail the check, along with the changes in the symbols from the two consecutive iterations. This method optimizes the iteration order during the decoding process. The

For other nodes that do not meet the above conditions, their reliability label is set to 1, and these nodes will be updated only after the nodes with a reliability label of 2 are updated. This grouping and scheduling strategy based on reliability labels effectively optimizes the update order of nodes during the iteration process, thereby improving the algorithm's convergence speed and decoding performance.

As shown in Figure 2, with the increase in channel SNR, the number of iterations for LDPC decoding gradually decreases. Experimental results show that, compared to the SCMS algorithm, the DSD-SCMS algorithm reduces the average number of iterations by 7.52%, with a maximum reduction of up to 9.09% under optimal conditions, significantly improving decoding efficiency.

VI. CONCLUSION This paper proposes a Dual Reliability Decision SelfCorrecting Minimum Sum (DRD-SCMS) algorithm, aimed at addressing the shortcomings of the traditional SCMS algorithm in the reliability determination of variable nodes. By introducing a dual decision mechanism, which incorporates the connection count between variable nodes and parity-check failed check nodes, as well as the results from the previous and current symbol iterations, the DRD-SCMS algorithm can more accurately assess the reliability of variable nodes. Simulation results demonstrate that, compared to traditional algorithms, the DRD-SCMS algorithm shows significant advantages in decoding performance, convergence speed, and decoding efficiency. Particularly under high signal-to-noise ratio conditions, it exhibits stronger adaptability and reliability.
