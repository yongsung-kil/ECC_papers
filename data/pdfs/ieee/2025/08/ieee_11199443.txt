Abstract—Physical layer key generation (PLKG) offers a lightweight and low-complexity key management solution for future emerging wireless communication networks. However, current PLKG schemes face challenges in effective integration with practical communication systems, hindering their practical deployment. To solve this problem, this paper proposes a key generation scheme integrated with the 5G physical layer protocol, leveraging the Demodulation Reference Signal (DM-RS) to extract channel estimation for PLKG, enabling efficient key generation. Furthermore, a soft-decision quantization method based on Multibit Adaptive Quantization is proposed which makes key reconciliation input compatible with the soft-decision input requirements of the Low-Density Parity-Check (LDPC) decoder in 5G channel coding module. Simulation results demonstrate that the proposed scheme successfully integrates into the 5G standard communication framework while still maintaining favorable key disagreement rate (KDR) performance, the key reconciliation using keys derived by soft-decision quantization method can achieve better key agreement performance than using keys derived by binary quantization.
Keywords- Physical layer key generation; 5G MIMO-OFDM system; Information reconciliation; LDPC coding
I. INTRODUCTION
The fifth-generation mobile communication system (5G) has emerged as a globally recognized critical standard in wireless communications due to its widespread deployment. 5G delivers ultra-high data rates while enabling transformative applications across industries through its low latency, high reliability, and massive connectivity capabilities. These capabilities position 5G as a key enabler for mission-critical domains including smart cities, intelligent transportation systems, telemedicine, and industrial automation [1]. Nevertheless, 5G deployments introduce escalating security challenges, particularly regarding data privacy and transmission security. The inherent resource constraints (e.g., limited computational power and energy) of massive IoT devices in 5G networks expose fundamental limitations in conventional public-key-based security protocols [2], which face systematic challenges including computational

overhead, key distribution complexity, and latency sensitivity [3]. Physical Layer Key Generation (PLKG) technology [4] exploits the inherent reciprocity of wireless channels to extract random characteristics from channel characteristics. By leveraging spatial decorrelation properties, PLKG guarantees source confidentiality while enabling lightweight key distribution and management between communicating parties. For networks with large-scale distributed architectures, PLKG technology can achieve key distribution and management with lower payload overhead, hence PLKG is poised to become a critical approach for implementing key management in future wireless communication networks.
A. Background and Related Work
The fundamental PLKG process comprises four stages: Channel Probing, Quantization, Information Reconciliation, and Privacy Amplification [5]. Firstly, PLKG exploits channel reciprocity to extract highly correlated channel measurements as shared randomness sources. Channel characteristics usually include Received Signal Strength (RSS) [6] and Channel State Information (CSI) [7-8]. The extracted channel characteristics are then quantized by both parties to derive binary key sequences. Typical quantization schemes include Level Crossing Quantization (LCQ) [6] and Multibit Adaptive Quantization (MAQ) [9]. MAQ establishes equiprobable quantization thresholds by computing the Cumulative Distribution Function (CDF) of channel measurements, and employs Gray coding to quantize measurement values falling into different quantization intervals into binary bits. This guarantees balanced bit probabilities in the generated key sequence, with adaptive quantization bit-depth dynamically configured according to channel quality. Due to channel time-variation and noise, the keys obtained by quantization at both parties are not identical. It is essential to correct mismatches in the raw keys by Information reconciliation. Current reconciliation protocols comprise interactive protocols and error-correction-coding (ECC)-based schemes. The Cascade protocol [10] represents the predominant interactive reconciliation method, as this reconciliation protocol achieves low computational overhead, but require frequent bidirectional message exchange to perform key error correction. ECC-based schemes correct key errors by transmitting errorcorrecting codes, utilizing codes like Turbo codes [11], Polar

The current release of 5G provides multiple probing signals for channel state estimation, enabling PLKG to extract reciprocal channel characteristics using standardized signaling resources without dedicated overhead. Among various channel probing signals specified in 5G standards, Channel State Information Reference Signal (CSI-RS), Sounding Reference Signal (SRS), and Demodulation Reference Signals (DM-RS) can all provide accurate channel measurement information, fulfilling the reciprocity requirements for PLKG channel feature extraction. Thus, existing probing signals in 5G standards can be utilized to acquire channel characteristics for PLKG. Simultaneously, the 5G physical layer offers high-performance, low-complexity channel coding and decoding methods for diverse transmission services, ensuring efficient and accurate information transmission, including Polar codes for physical layer signaling and Quasi-Cyclic (QC)-LDPC codes for highthroughput data transmission. The decoding subsystem exploits Log-Likelihood Ratios (LLRs) from demodulation as reliability information, enabling belief propagation decoding to perform near-optimal error correction. which approaches Shannon-limit performance [14]. For ECC-based key reconciliation in PLKG, native 5G channel coding modules can be leveraged to realize high-efficiency reconciliation with minimized complexity. Current 5G standards employ Multiple-Input Multiple-Output (MIMO)- Orthogonal Frequency-Division Multiplexing (OFDM) as the dominant physical layer transmission scheme, where the increased antenna dimensionality and wider communication bandwidth supply enhanced randomness resources for secret key generation, consequently leading to improved secret key capacity. In [15-16] introduce the Degreeof-Freedom (DoF) metric for secret key capacity, and conduct theoretical analysis on the secret key generation potential of MIMO systems, with [17] subsequently deriving a closed-form expression for secret key capacity in Gaussian MIMO channels.
Currently, only a portion of PLKG research considers collaborative design with 5G physical layer protocols. In [12] proposes using SRS for channel feature extraction, eliminating
dedicated pilot overhead，and develops a Polar Code-Based Secret Key Reconciliation Scheme(PSKRS). This scheme outperforms Cascade protocol in error correction with significantly lower communication overhead. However, constrained by Polar codes' block length limitations, LDPC codes exhibit superior performance to Polar codes for large-scale key reconciliation. As 5G LDPC belief propagation decoding requires soft-decision inputs, to adapt reconciliation schemes to 5G LDPC decoding standards, and incorporate bit reliability to enhance reconciliation performance, non-binary quantization schemes should supersede conventional binary quantization, to obtain quantized bit reliability information. In [18] introduced an LDPC reconciliation scheme that quantizes the LLR of keys, constructing error models based on AWGN channel characteristics, which calculates key LLR values based on locally probed channel characteristics and inputs them to the LDPC decoder to achieve key reconciliation. However, this

approach is fundamentally limited to single-bit quantization scenarios. In [19] proposes a partially-encoded key reconciliation method, which classifies key bits into Most Reliable Bits (MRBs) and Least Reliable Bits (LRBs) by computing LLR values during quantization, and applies LDPC coding only to LRBs, thereby significantly reducing encoding/decoding complexity while preserving a low Key Disagreement Rate (KDR). However, practical applications typically require perfectly consistent keys, yet this work overlooks the problem of achieving perfect reconciliation.
B. Contributions
To summarize, although current PLKG schemes exploit channel characteristics for key generation, they typically lack seamless integration with operational communication infrastructures, leading to incompatibility with 5G New Radio (NR) physical layer specifications, and exhibit low resource utilization efficiency during key generation. Furthermore, traditional PLKG approaches incur significant overhead due to dedicated resources, failing to satisfy 5G's stringent demands for ultra-low latency, high reliability, and optimized resource efficiency.
This paper proposes a physical layer key generation scheme integrated with 5G that reuses channel probing resources and coding modules in 5G systems to improve resource efficiency while maintaining high compatibility with 5G physical layer protocols. Specifically, DM-RS signals in 5G MIMO- OFDM systems are leveraged to extract reciprocal channel characteristics for key generation, with a soft-decision quantization based reconciliation scheme designed using 5G channel coding modules. Compared with conventional methods, our approach reduces resource overhead and enhances reconciliation performance via soft-decision quantization. This innovation demonstrates superiority in efficient key generation and KDR reduction. The main contributions of this paper include three aspects:
∂ A DM-RS based reciprocal channel feature extraction method is proposed, exploiting native 5G MIMO-OFDM probing signals for PLKG, where both parties extract multi-channel equivalent CSI from DM-RS signals, using precoding matrix preprocessing to ensure channel reciprocity. Initial keys are then generated through lowpass filtering and multi-bit equiprobable quantization.
∂ A soft-decision quantization scheme compatible with 5G belief-based LDPC decoding is designed, utilizing specialized quantization method on MAQ to generate confidence metrics as soft-bit inputs for LDPC decoding, achieving full compatibility with LDPC coding module in Physical Downlink Shared Channel (PDSCH), and incorporating bit confidence information during quantization, delivering superior error correction versus hard-bit reconciliation.
∂ Simulation results demonstrate that the proposed scheme maintains low initial KDR while adapting to 5G PHY architecture, and adapts to diverse communication scenarios. The soft-decision reconciliation is compatible with 5G belief-based LDPC decoding and outperforms hard-bit methods in error correction.

III. KEY GENERATION SCHEME INTEGRATED WITH 5G SYSTEM
We introduce a key generation scheme integrated with the 5G NR physical layer framework, which enables secret key generation exclusively through native 5G physical layer probing signals and coding mechanisms. The proposed key generation scheme is shown in Fig. 2. Specifically, the scheme first extracts channel measurements from DM-RS signals embedded in 5G

physical layer waveforms and obtains reciprocal channel characteristics through preprocessing, then quantizes the channel characteristics into binary bits via low-pass filtering and Multi-level Adaptive Quantization (MAQ), subsequently, Alice employs the LDPC channel coding system for 5G data transmission to encode the key, and transmits the generated parity bits to Bob, Bob then decodes the data using his locally quantized bits and the received parity bits to recover a key identical to Alice's. The following sections elaborate on each key generation phase.

A. Channel Characteristic Extraction Based On DM-RS
Within 5G NR specifications, SRS and CSI-RS are specifically designed for CSI acquisition, enabling dynamic adaptation of MIMO precoding, power control, and modulationcoding schemes，which can provide accurate CSI information. However, their transmission patterns vary according to service demands, exhibiting non-periodic temporal distribution, as specified in 3GPP, SRS/CSI-RS are dynamically configured via Radio Resource Control (RRC) signaling, or dynamically activated via Downlink Control Information (DCI), leading to inconsistent channel sampling intervals. Conversely, DM-RS primarily facilitate coherent demodulation. All physical channels (data and control) in 5G NR depend on DM-RS for demodulation, making DM-RS ubiquitous in 5G physical resource grids. Compared to SRS and CSI-RS, DM-RS offers periodic channel sampling with stable intervals. Each DM-RS instance generally spans one OFDM symbol, with uniform frequency-domain distribution over allocated bandwidth, delivering wideband channel characteristics. This motivates our selection of DM-RS for CSI harvesting, ensuring efficient and comprehensive channel feature acquisition for PLKG.

D. Privacy Amplification
Due to parity bits carrying key information are transmitted during reconciliation, Eve may acquire partial key-related information, leading to key information leakage, thus privacy amplification is required to enhance key security. Privacy amplification typically employs hash functions [24], where hash mappings exhibit one-way property, thereby enhancing key security.

V. SIMULATION ANALYSIS
To evaluate the efficacy of proposed key generation scheme, we perform Monte Carlo simulations to assess the key generation performance. Simulations utilize functions from MATLAB's 5G Toolbox to implement an end-to-end system simulation conforming to 3GPP PDSCH standards, with specific simulation parameters listed in Table 2. We implement the standardized nrCDLChannel model [26] for 5G channel emulation, evaluating three typical 3GPP propagation scenarios:
1)CDL-A: Urban Microcell (UMi) NLOS with dense multipath scattering.
2)CDL-B: Urban Macrocell (UMa) NLOS featuring larger delay spreads.
3)CDL-C: Rural Macrocell (RMa) LOS with dominant path component.

Under 3GPP TS 38.211, each 10ms radio frame comprises 10 subframes at 30kHz subcarrier spacing (μ=1), with each 1ms subframe containing 2 slots of 0.5ms duration. The simulation configures TDD pattern with uplink/downlink alternately occupying slots, ensuring both Uplink (UL) and Downlink (DL) are included in each subframe, with DM-RS anchored at symbol index 2 per slot, maintaining fixed 0.5ms delay for channel reciprocity calibration. Key generation workflow operates as follows: Every 0.1s, Alice and Bob sample UL/DL signals from co-subframe UL/DL signals, obtaining CFR amplitudes across 624 subcarriers per probe as a CSI acquisition. Each measurement triggers independent quantization and reconciliation, Alice PDSCH RBs, to transmit encoded reconciliation data to Bob, enabling Bob to perform syndromebased error correction. With 1s simulation duration, 10 key generation cycles are executed, repeated over 500 Monte Carlo trials, yielding 5000 total key generations.

A. Performance Analysis of Key Generation Scheme

VI. CONCLUSION
This paper proposes a key generation scheme integrated with 5G physical layer specifications, leveraging DM-RS channel estimation for key generation and presents a soft-decision quantization method compatible with 5G belief-based LDPC decoding algorithm. Simulation results demonstrate the adaptability of the proposed scheme to 5G standard physical layer architecture, while ensuring reliable and efficient key generation, and the soft-decision method enhances reconciliation performance.
