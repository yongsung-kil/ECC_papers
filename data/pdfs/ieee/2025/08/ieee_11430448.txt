Abstract—The high complexity of the Belief Propagation (BP) 
decoding algorithm in LDPC decoding leads to greater resource 
consumption and increased communication link latency. This has 
become a critical challenge in designing efficient communication 
systems. The Box-Plus operation can effectively transform the 
nonlinear equations in check nodes into an addition operation of 
the minimum and the logarithmic functions, but the logarithmic 
function still has high complexity. In this paper, we propose an 
Approximate Box-Plus (ABP) decoding algorithm, which reduces 
decoding complexity while maintaining decoding performance. 
We first deduced that the logarithmic function in addition 
operations essentially represents the error between the BP 
decoding algorithm and the Min-Sum (MS). Based on this insight, 
a low-complexity, high-precision substitute function is designed. 
This function utilizes the boundary control ability and symmetry 
of the min and max functions to effectively express the relative size 
changes, thereby reducing the unreliability in check node message 
passing. The proposed decoding algorithm based on the above 
function reduces the complexity by 38.9% compared to the BP 
algorithm, and the mean square error of the substitute function 
and logarithmic function is only 0.0018. Simulation results 
demonstrate that the decoding performance improves by 
approximately 0.4 dB compared to the classical Normalized Min-
Sum (NMS) and Offset Min-Sum (OMS) algorithms, and the 
complexity is increased by only three and two new operations per 
operation compared to NMS and OMS, respectively. 
Keywords—Box-Plus, low-density parity-check (LDPC) codes, 
iterative decoding, BP  
I. INTRODUCTION  
LOW-DENSITY parity-check (LDPC) codes are a channel 
coding technique based on sparse matrices[1]. It is widely 
adopted 
in 
satellite 
communications[2], 
television 
broadcasting[3], wireless local area networks[4], wireless 
metropolitan area networks[5], and fifth-generation (5G) mobile 
communication technologies[6] due to their error correction 
performance close to the Shannon limit and flexible code rate 
design. However, despite significant advancements in the 
performance and application of LDPC codes, the complexity 
caused by nonlinear equations in their decoding process remains 
a key barrier to further adoption. This challenge is particularly 
pronounced in high data rates and resource-constrained 
autonomous vehicle. So, how to  reduce decoding complexity 
while maintaining performance continues to be a formidable 
task. 
In 1962, Gallager first proposed the Belief Propagation (BP) 
decoding algorithm, which is proved to be the decoding 
algorithm closest to the Shannon limit with the best 
performance[7]. In 1981, Tanner represented LDPC codes with 
a graphical model[8], which made LDPC codes easier to 
understand and spawned a large number of reports on reducing 
the decoding complexity of LDPC codes. Typically, the Min-
Sum Algorithm (MSA) was proposed[9]. By eliminating the 
hyperbolic tangent function, MSA significantly reduced the 
complexity of the BP algorithm. However, this simplification 
replaced the nonlinear multiplication in BP algorithm with sign 
multiplication and minimum operations, leading to accuracy 
loss and degraded decoding performance. To address this issue, 
researchers later introduced the Normalized Min-Sum (NMS) 
and Offset Min-Sum (OMS) algorithms based on MSA[10], 
which respectively use normalization and offset factors to 
balance the differences in the amplitude of node update values. 
Although NMS and OMS have different processing

correction factors in the  NMS and the OMS are empirically 
determined. Moreover, under different channel conditions and 
code structures, these correction factors need to be adjusted to

the nonlinear equation. To address this issue, researchers

logarithmic function while maintaining the performance. 
However, the substitute functions proposed in the current 
research has low accuracy, the mean square error between the 
substitute function and the logarithmic function is large, 
resulting in poor decoding performance[11][12]. Therefore, in 
order to effectively reduce the complexity of LDPC decoding 
algorithm while ensuring the performance, the form of substitute 
function still needs to be further studied. 
To achieve the above purpose, this paper first concludes that 
the error between the MSA and BP decoding algorithms is a 
logarithmic function based on the Box-Plus operation. This 
conclusion  reveals the fundamental reason of error introduced 
by the MSA and provides ideas for further optimization. Then 
summarizes that this logarithmic function is essentially a smooth 
relative difference. Based on the above analysis, a low-
complexity and high-precision substitute function is proposed. 
This function effectively approximates the logarithmic function 
by leveraging the boundary control capability and symmetry of 
the min and max functions, which reduces the unreliability of 
message passing in check nodes. Additionally, the min and max 
functions exhibit low complexity. This substitution ensures 
decoding performance while achieving a reduction in decoding 
complexity. 
(1) Based on the Box-Plus operation, the derivation process 
of the error function between the MSA and BP decoding 
algorithms is presented. 
(2) This paper proposes an Approximate Box-Plus (ABP) 
decoding algorithm, which constructs a novel low-
complexity and high-accuracy substitution function to 
effectively replace the error function. By optimizing the 
check node update equation, the algorithm reduces 
decoding complexity while maintaining decoding 
performance. 
(3) To verify the effectiveness of the ABP algorithm, 
simulations were conducted. Results indicate that the 
performance of the proposed decoding algorithm is 
improved by about 0.4dB compared with the NMS and 
OMS algorithms. Compared to a single check node 
update in the BP decoding algorithm, the OMS algorithm 
reduces operations by 9, the NMS algorithm by 10, and 
the proposed ABP algorithm by 7.

V. SIMULATION RESULTS  
To validate the performance of the proposed ABP decoding 
algorithm, simulation experiments were conducted using LDPC 
codes constructed according to CCSDS standards. As a 
comparison, the performance of the NMS algorithm and the 
OMS algorithm is also simulated. After LDPC encoding, the 
codeword undergoes Binary Phase Shift Keying (BPSK) 
modulation. An Additive White Gaussian Noise (AWGN) 
channel is then simulated on the platform. At the receiver end, 
the decoder assumes that the signal-to-noise ratio (SNR) is 
known. 
The three code types (𝑁, 𝐾) = (2048, 1024), (8192, 4096), 
and (32768, 16384) from the CCSDS standard are considered, 
where K represents the information sequence length, N the total 
code length, and the coding rate R=1/2. For the simulation 
parameters, the maximum number of iterations is set to 10, with 
2000 frames simulated. The Eb/N0 range is set  from 1 to 2.6 dB, 
the bit error rate and the frame error rate of all  algorithm are 
tested every 0.2 dB. In the CCSDS standard, the optimal 
normalization factor for the NMS algorithm is generally set to 
0.75, and the offset factor for the OMS algorithm is set to 0.25.

algorithm, with a difference of 0.1 dB. In conclusion, it can be 
demonstrated that the substitution function constructed in this 
paper effectively compensates for the error between the MSA 
and BP decoding algorithms, resulting in improved decoding 
performance.

To verify the performance of the ABP decoding algorithm 
under different code lengths, simulations were conducted for 
two code structures, (N, K) = (8192, 4096) and (32768, 16384), 
as shown in Fig.5 and Fig.6. The Eb/N0 range was selected from 
1 to 2.6 dB. At a BER of 1E-05, the ABP decoding algorithm 
proposed in this paper provides a gain of at least 0.4 dB 
compared to both NMS and OMS. Furthermore, it can be 
observed that the slope of the ABP decoding algorithm curve is 
similar to that of the BP decoding algorithm, with a performance 
difference of approximately only 0.1 dB. This indicates that the 
ABP decoding algorithm converges faster than NMS and OMS. 
In conclusion, even under different code lengths, the substitution 
function constructed in this paper remains stable in 
compensating for the error between the MSA and BP decoding 
algorithms, and even converges faster in the case of long codes.

VI. CONCLUSION  
This paper proposes an ABP decoding algorithm based on 
the Box-Plus operation. In this algorithm, a low-complexity, 
high-accuracy substitution function is constructed, which 
effectively replaces the error function using the boundary 
control and symmetry properties of the min and max functions. 
This ensures decoding performance while reducing decoding 
complexity. Simulation results show that, compared to the 
existing NMS and OMS decoding algorithms, the ABP 
decoding algorithm provides a 0.4 dB gain. The complexity 
increases by only three and two additional operations per 
iteration compared to NMS and OMS, respectively.
