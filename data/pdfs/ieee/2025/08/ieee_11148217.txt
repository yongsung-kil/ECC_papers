Quasi-Cyclic Spatially Coupled LDPC Codes Construction with Girth Eight
Shikui Guo and Guohua Zhang

Abstract—Small cycles in spatially coupled low-density paritycheck (SC-LDPC) codes have a detrimental effect on the error floor. Recently, an explicit simple method has been proposed to construct a class of 4-cycle free SC-LDPC codes by partitioning a spreading matrix into a number of component matrices. Using a quasi-cyclic (QC) lifting, we propose a component matrices construction method to obtain QC-SC-LDPC codes with a girth eight in this paper. The component matrices of the new method are completely defined by a spreading matrix and an exponent matrix. The spreading matrix is constructed from a finite sequence of integers, and the exponent matrix is constructed from the greatest common divisor (GCD) framework. The novel construction method demonstrates the capacity to entirely eliminate 6-cycles. Compared to the recently proposed explicit structural design, the proposed code achieves a girth of eight and exhibits better bit-error rate (BER) performance.
Index Terms—girth, spatially coupled LDPC codes, greatest common divisor, coupling width.
I. INTRODUCTION
Spatially coupled convolutional codes low-density paritycheck (SC-LDPC-CC) codes [1], [2], [3], [4] are a new class of LDPC code encoding scheme. Its Tanner graph represents a convolutional structure with block-to-block memory, as it is coupled together over time. SC-LDPC codes are generated by truncating SC-LDPC-CCs codes to a finite length. The quasi-cyclic (QC) lifting is then used to obtain the quasicyclic spatially coupled low-density parity-check (QC-SCLDPC) codes [5], [6], which are easy to implement in hardware. The sliding window (SW) decoding algorithm is an improved algorithm based on the belief propagation decoding algorithm [7] that is suitable for SC-LDPC codes. The SCLDPC codes that satisfy the minimum coupling widths w have a low message recovery latency when the SW is used [1]. The long coupling length makes SC-LDPC codes more sensitive to short cycles. It is well known that short cycles and graphical structures containing short cycles, such as trapping sets (TSs) and absorbing sets (ASs), have an adverse effect on the error floor [4]. The girth of a QC-SC-LDPC code refers to the length of the shortest cycle. Therefore, eliminating the 4-cycle and 6-cycle structures to achieve a girth of eight and satisfy the minimum coupling width is a frontier challenge in the field of QC-SC-LDPC code construction.
In [1], an edge spreading procedure is proposed for coupling L block protographs to form convolutional protographs. The

SC-LDPC codes constructed using this procedure can reduce the number of small ASs [4]. In [5], a multi-step design method is proposed to obtain a 4-cycle free SC-LDPC code, which is achieved by decomposing the exponent matrix into w+1 4-cycle free component matrices. In addition, the method uses an exhaustive search algorithm to find the minimum value of w that satisfies the lower bound on the coupling width, but the code generation process with flexible parameters is highly complex. The decoding latency of a QC-SC-LDPC code depends on the product of the coupling width and the lifting degree, such that codewords with smaller coupling widths have lower latency. As an extension of this method, the spreading matrix Ms constructed from “good sequence” in [8] handles two constructions of SC-LDPC codes, depending on whether the elements of the spreading matrix under consideration are continuous or not. Then, the elements in Ms are used to explore whether there are 4-cycles in the parity-check matrix (PCM) of the constructed code. Finally, a new finite sequence of integers is defined, and an explicit method is used to generate a class of SC-LDPC codes without 4-cycles, such that the constructed code satisfies the minimum coupling width requirement in most cases. However, the QC-SC-LDPC codes constructed using this method don’t eliminate the harmful effect of 6-cycles.
In [9], the authors propose a greatest common divisor (GCD) framework and give a simple and flexible method for constructing QC-LDPC codes. The GCD framework can be used to construct QC-LDPC codes with a girth of at least eight and a lower bound on the lifting degree is given. In order to achieve high reliability, GCD framework has been extensively used [10], [11], [12], [13]. In [14], the authors apply GCD framework to construct global coupling (GC) LDPC codes with a girth of eight and improve the performance. In [15], a new method is proposed to construct SC-LDPC codes based on full-length row-multiplier (FLRM) matrices, achieving excellent performance and flexible code rates. These construction methods don’t consider the impact of minimum coupling width on message recovery latency.
In this paper, a new method is proposed for constructing the component matrix and modifying its elements through the introduction of an exponent matrix Mg. The QC-SC-LDPC codes constructed using the exponent matrix Mg and the spreading matrix Ms can completely eliminate the 6-cycles. II. METHOD FOR DESIGNING QC-SC-LDPC CODES
This section introduces the construction method of QCSC-LDPC codes and describes how to construct SC-LDPC codes with a girth of six through the existing edge spreading procedure. Subsequently, QC-SC-LDPC codes are constructed through the QC lifting.
A. Construction of SC-LDPC codes
The spatial coupling structure of the SC-LDPC code based on the protograph is to copy the block protograph into multiple identical copies, and then rearrange the edges of adjacent copies in a specific coupling manner to generate a coupling chain containing multiple nodes. A protograph is a small bipartite graph that usually contains very few variable nodes and check nodes. In a block protograph, there is a set of variable nodes, a set of check nodes, and a set of edges. Multiple block protographs can be coupled into a larger protograph chain through copying and permutation operations [5].
The block protograph matrix B = (bi,j)nc×nv is a nonnegative integer matrix representing a block protograph that contains nv variable nodes {v1, · · · , vnv } and nc check nodes {c1, · · · , cnc }. The bi,j represents the number of edges between check nodes ci and variable nodes vj, where 1 ≤ i ≤ nc, 1 ≤ j ≤ nv [5]. Fig. 1 shows the coupling process of L block protographs to obtain a protograph chain with a length of L. Fig. 1(a) shows a (3 2) block protograph and Fig. 1(b) shows L block protographs. The coupling of L block protographs into a finite length chain of length L is shown in Fig. 1(c). The variable nodes of each copy of block protograph are connected to the check nodes at most w copies of block protograph after it, i.e t, t + 1, · · · , t + w, where w is called coupling width [1].

III. NEW CONSTRUCTION WITH GIRTH EIGHT

Section II introduces the use of the existing “good sequence” [8] to construct the spreading matrix Ms, which is used to construct SC-LDPC codes without 4-cycles through the edge spreading procedure. However, SC-LDPC codes constructed using this method cannot completely eliminate 6cycles. This section presents a new construction method that eliminates the 6-cycle structures in the component matrices.

A. A Novel Approach to Component Matrices Construction
The GCD framework can be utilized to generate the exponent matrix Mg of a QC-LDPC code with a girth of eight. In the event that an ascending sequence S2 = (s0, s1, · · · , sp−1) satisfies the inequalities (sc − sa)/ gcd(sc − sa, sb − sa) ≥ q for all triples (sa, sb, sc), where 0 ≤ a < b < c ≤ p − 1, then S2 satisfies the GCD constraint. The exponent matrix Mg = ST2 · S1 = [s0, s1, · · · , sp−1]T · [0, 1, · · · , q − 1] generated by S2, corresponds to a QC-LDPC code with a girth of eight and a lifting degree P ≥ (sp−1 − s0)(q − 1) + 1 [10]. B. SC-LDPC codes cycle search algorithm
This section presents a cycle search algorithm. Considering the special form of the SC-LDPC codes, as shown in Fig. 3. The cycle search algorithm only needs to search for the number of cycles in specific structure and then use an algebraic method to calculate the total number of cycles in the SC-LDPC codes.
Each coupling block in Fig. 3 has the same structure. For the six 6-cycle structures shown in Fig. 2, the first coupling block will only form 6-cycles with the first w + 1 coupling blocks of the L coupling blocks. The two shaded parts in Fig.3 are the coupling blocks that are furthest apart from each other and can jointly participate in forming the 6-cycle. For each additional coupling block, the resulting 6-cycles structure is the same as the previous one. Therefore, it is only necessary to search for the number of 6-cycles in the first w and w + 1 coupling blocks. For p = 3 and q = 4, the “good sequence” used for searching is (0, 1, 0, 2, 2, 0); for p = 4 and q = 5, it is (0, 0, 2, 1, 2, 0, 0, 2). V. CONCLUSION
This paper presents a construction method for QC-SCLDPC codes based on the GCD framework, achieving a girth of eight. The exponent matrix constructed using GCD constraints replaces the component matrix in the edge spreading procedure, ensuring 6-cycle free structure. Simulation results show that the performance of the new codes is significantly better than that of several existing methods.
