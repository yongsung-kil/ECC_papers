Low Complexity Deep Learning Method for Base Graph of Protograph LDPC Decoding
Shuo Liang1, Xingcheng Liu1,2,3*, Lin Zhang1
1School of Electronics and Information Technology, Sun Yat-sen University, Guangzhou, China 2School of Information and Intelligent Engineering, Guangzhou Xinhua University, Guangzhou, China
3Southern Marine Science and Engineering Guangdong Laboratory (Zhuhai), Zhuhai, China email: isslxc@mail.sysu.edu.cn

Abstract—Deep learning-based decoding algorithms have emerged as a focal point currently in contemporary research. Neural Offset Min-Sum Belief Propagation decoding algorithm is one of the traditional deep learning methods rooted in model-driven approaches. However, with the gradual increase in codeword length, there is an inevitable surge in the training cost and time complexity of such kind of deep learning algorithms. Addressing this issue, a novel deep learning algorithm based on the structural characteristics of the base graph in protograph LDPC codes is proposed. By strategically training the base graph and introducing a unique bit decoding allocation strategy, we endeavor to harmonize the Deep Learning algorithm with the intrinsic structure of the base graph itself. This approach contrasts favorably with several existing methods that utilize conventional segmental deep learning decoding algorithms for protograph LDPC codes. Simulation results demonstrate that, compared to other model-driven based deep learning algorithms, the proposed algorithm significantly reduces the number of parameters required for storage and training, while the loss in decoding performance remains within acceptable bounds.
Index Terms—model-driven neural network, Protograph LDPC Code, Base graph, Error-correction Performance
I. INTRODUCTION
Low-density parity check (LDPC) codes, first introduced by Gallager in 1962 [1], have gained significant interest since their rediscovery in 1995 [2]. This is largely due to their nearapproach to Shannon’s limit of error correction performance when utilizing the belief propagation (BP) algorithm [3]. One of the traditional soft decision decoding algorithms for LDPC codes is the log-likelihood ratio belief propagation algorithm. This algorithm systematically updates all messages between variable-to-check (V2C) nodes and check-to-variable (C2V) nodes in each iteration. While soft decision decoding algorithms offer superior performance compared to hard decision algorithms, they do introduce a significantly higher level of complexity when implemented in hardware. In real-world engineering applications, hard-decision decoding algorithms are more commonly employed. In the case of the Min-Sum

Refinements such as the Normalized Min-Sum (NMS) and Offset Min-Sum (OMS) algorithms [4] have modified the computation of variable messages. By employing correction factors, these algorithms can provide competitive decoding performance while maintaining a low level of complexity. Protograph LDPC codes operate on the premise of expanding a base graph or matrix into a fully developed graph or matrix. Thanks to a thoughtfully designed structure, protograph LDPC codes can deliver superior performance and are more conducive to efficient encoding/decoding implementations.
Over recent years, with the advancing depth of research in deep learning, a growing number of researchers have been attempting to fuse it with traditional channel decoding, which exhibiting significant potential. Tobias Gruber made the first attempt to replace the traditional Tanner graph structure with a fully connected network for information propagation in 2017 [5]. The same year after that, Xu proposed multiple scaled belief propagation (BP) algorithm which can obtain faster convergence and better performance by neural network decoder for polar codes [6]. Beyond the previously mentioned fully connected networks, scholars have also explored the use of other network structures such as Convolutional Neural Networks (CNN) [7], Recurrent Neural Networks (RNN), and Residual Neural Networks (ResNet) as alternatives to the traditional Tanner graph structure. Undeniably, these neural network based decoding algorithms have shown exceptional performance with shorter lengths of codes, however, as the length of the codewords increases, the complexity of these algorithms correspondingly escalates. Considering the massive amount of data parameters demanding training, the time investment and training complexity inevitably rise as significant concerns. Consequently, these neural network-based algorithms pose considerable challenges when it comes to hardware implementation.

B. Protograph LDPC Codes
A technique called protograph-based codes or protograph codes has been proposed to simplify the design and implementation of LDPC codes. This technique involves converting a smaller matrix or graph prototype, known as a protograph, into a full matrix or graph representation [15], [16]. A protograph LDPC code can be constructed using the operation of copy and permute procedure. Figure 1 (a) represents an example of a base graph, which consists of 2 variable nodes and 1 check node. By copying the base matrix 3 times, we can create a derived graph with n = 2 ∗ 3 variable nodes and m = 1 ∗ 3 check nodes, The edge of base graph is named as πij. In the process of constructing a protograph LDPC code, each nonzero element of the base matrix can be represented as a circulate permutation matrix. This implies that the permutation procedure for the H matrix is not arbitrary among the copies. Specifically, the edges of the H matrix can only be permuted within the same type.

C. Model Driven based Decoding Algorithm
Existing Deep Learning Belief Propagation (DL BP) decoding algorithms completely abandon the traditional iterative decoding approach, where information is passed along the edges of the Tanner graph. Instead, they rely heavily on the use of substantial data to train neural networks. This method necessitates the storage and training of considerable amounts of data and labels, leading to heightened resource consumption and training costs.
In contrast, the Model-Driven Deep Learning Belief Propagation algorithms unfold the Tanner graph following the iterative decoding scheme. The expanded network structure is subsequently trained and decoded leveraging the basic theoretical algorithms. While codes devoid of cycles tend to exhibit sub-optimal performance, in practical scenarios, codes with cycles are often utilized. The Bit Error Rate (BER) performance of Model-Driven Deep Learning approaches tends to align more closely with the optimal performance.

As is well known, the degree of VN and CN of H matrix are equal to the degree of VN and CN of base matrix correspondingly, Tanner graph is obtained by copy and permutation operation of base graph. V. CONCLUSION AND FUTURE DIRECTIONS
In this paper, a novel model-driven DL algorithm, rooted in
the structural characteristics of the base graph in protograph
LDPC codes is proposed. Our strategy involves meticulously
training the base graph and implementing a novel bit decoding
allocation approach, with the aim to synchronize the Deep
Learning algorithm with the base graph’s inherent architecture.
This method provides a distinctive advantage over numerous
existing techniques, which employ conventional segmental
deep learning decoding algorithms for protograph LDPC
codes. Subsequent research will primarily address the issue of
early convergence observed in the experiments, investigating
the potential of techniques such as residual networks to further
address this problem.
