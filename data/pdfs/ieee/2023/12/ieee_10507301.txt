A Multi-parameter Compensated Min-Sum Based LDPC Decoding Algorithm

AbstractвҖ”The Minimum Sum (MS) algorithm is a standard Low-Density Parity-Check Codes (LDPC) decoding algorithm. Compared with traditional algorithms, it uses approximate calculations to reduce the computational complexity of the algorithm when calculating parity nodes, resulting in performance loss issues. To improve the LDPC decoding performance, the LDPC decoding algorithm of Multi-parameter Compensated MinSum (MCMS) is proposed. This algorithm introduces relevant information, such as channel and code length, into calculating update verification nodes. Introducing multiplication and correction factors to reduce errors in validation node calculations through multi-parameter compensation. Simulation results show that the MCMS algorithm has superior error rate performance compared with the MS algorithm.
KeywordsвҖ”LDPC decoding, min-sum, multi-parameter compensated
I. INTRODUCTION
At present, the demand for data communication is increasing day by day [1]. With the continuous development of communication technology, channel coding is also constantly evolving and improving. Channel coding refers to encoding and processing data in digital communication to enhance the reliability and anti-interference of data transmission. In digital communication, signals are subject to various interferences and distortions during information transfer, such as noise, fading, etc. These interferences can lead to data transmission errors. The purpose of channel coding is to increase the reliability and anti-interference of data transmission by adding redundant information, enabling the receiving end to detect and correct transmission errors [2], [3]. Low-Density Parity-Check Codes (LDPC) code is a widely used encoding method in recent years, which has good error correction performance. Besides, due to years of research, the complexity of decoding algorithms has gradually decreased [4], [5].
LDPC decoding includes two categories: hard decision and soft decision. Hard decision decoding algorithms decode by passing bit information [6]. Hard decision decoding is a simple LDPC code decoding algorithm proposed in [7]. This algorithm only applies to BSC channels, also known as Bit Flipping Algorithm .Their computational performance is poor, but their computational complexity and decoding algorithm complexity are low .Nowadays, LDPC soft decision algorithm is mainly used. LDPC decoding soft decision algorithms mainly include three types: Belief Propagation (BP) algorithm, Log-Likelihood Ratio Belief Propagation (LLR BP) algorithm [8], and Min-Sum

(MS) algorithm [9]. The BP algorithm is an iterative algorithm based on graph theory, which decodes by passing information on the graph. The advantage of the BP algorithm is its high decoding performance. The LLR BP algorithm is a variant of the BP algorithm, whose basic idea is to transform the information transmission process in the BP algorithm into addition and multiplication operations, thereby simplifying the calculation process. Compared with the Belief Propagation algorithm, although LLR BP reduces the computational complexity to some extent, it is still too complex The MS algorithm is an iterative algorithm based on the minimum sum method, which basic idea is to transform the decoding problem into solving the minimum sum problem, thereby simplifying the calculation process. However, the MS algorithm suffers from performance loss when updating the verification node.
Therefore, this paper proposes a Multi-parameter Compensated Min-Sum (MCMS) algorithm to address the decoding loss problem in MS decoding algorithms. This algorithm introduces channel and code length information into calculating MS algorithm update verification nodes. At the same time, due to the issue of significant errors in verifying node approximation, analyzing the tanh(x) image, it is found that
tanh(x) changes more sharply when the absolute value of the
independent variable is less than 1.5. Therefore, a multiplicative factor and a verification factor are proposed. The combination of the multiplicative factor and correction factor reduce the error in verifying node approximation, thereby improving decoding performance. The simulation results show that compared to traditional MS algorithms, the Eb / N0 of the MCMS algorithm will decrease by 0.1-0.2dB when reaching the same bit error rate (BER).
The rest of this paper is organized as follows. In Section II, we introduced the LDPC soft decision decoding algorithm and process. Then in Section III, we proposed the MCMS algorithm and analyzed its process and steps in detail. The results and analysis are presented in Section IV. Section V draws the conclusion.

V. CONCLUSION
This paper proposes a new LDPC decoding algorithm MCMS algorithm based on the MS decoding algorithm. The MCMS algorithm introduces information such as code length and channel to calculate updating verification nodes and, at the same time, analyze tanh(x) images and introduce correction
factors and multiplicative factors to compensate for the problem of significant errors in updating calibration nodes. The simulation results show that the MCMS decoding algorithm has better BER performance compared to the MS algorithm and has a high application value.
