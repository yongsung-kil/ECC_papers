Abstract—The performance of model-driven channel neural decoding has surpassed that of traditional channel decoding algorithms, but at a higher complexity, making it difficult to implement on resource-constrained communication hardware. In this paper, we propose a quantization scheme for model-driven channel neural decoding, and combine the quantization scheme with TR decomposition and weight sharing algorithms to form different types of multi-way compression methods. Experimental results on LDPC, BCH and Hamming codes show that the proposed quantization and multi-way compression methods can effectively reduce the complexity of channel neural decoding without significant performance degradation.
Index Terms—channel neural decoding, quantization, multiway compression, model-driven
I. INTRODUCTION
In recent years, with the great success of deep learning in computer vision, speech signal recognition, and natural language processing, communication researchers have proposed the model-driven based channel neural decoding algorithm [1]–[3]. The performance of model-driven channel neural decoding has surpassed that of traditional channel decoding algorithms [1]–[4], but at a higher complexity. In order to reduce the complexity of the channel neural decoding algorithm, the authors in [5]–[9] proposed a series of compression methods and achieved excellent compression results. These methods all assume that the input and output values of the channel neural decoding algorithm are continuous, which requires more storage resources. If the input and output values are quantized into discrete values, the storage resources required for channel neural decoding can be reduced.
Quantization is a mapping method that maps input values (often continuous) in a larger data set to output values (often discrete) in a smaller data set [10]. Quantization techniques have broad applications in both machine learning [10]–[13] and communication systems [14]. In the field of deep learning, quantization techniques not only reduce the model size

of neural networks, they can also enable faster speed and require less power [10]. In the field of communication, they can reduce computational complexity and storage resource requirement [14]. The quantization operation is a common step for hardware implementation of channel decoders and has been extensively studied in the past [15]–[20].
Different from the above, the model-driven based channel neural decoding algorithms have both the characteristics of deep learning and communication expert knowledge. Quantization operations involve learning weights, activation functions, decoding messages, etc. The loss of quantization of messages in traditional decoding algorithms will lead to performance degradation. However, in channel neural decoding, since the learning weights have learning ability under the guidance of the loss function, it is possible to make up for the degraded performance loss. The errors after the quantization operation of each part will affect each other, so it is necessary to study the effect of quantization for the model-driven based channel neural decoding algorithm.
In this paper, we propose several quantization schemes for channel neural decoding algorithms, and combine quantization schemes with Tensor Ring (TR) decomposition and weight sharing methods to form different types of multi-way compression methods. Experimental results on BCH, Hamming and LDPC codes show that the proposed multi-way compression schemes with quantization can effectively reduce complexity without causing significant performance loss.
The remainder of this paper is organized as follows: we elaborate the NNMS, NNMS+ and TR-NNMS+ algorithms in Section II. In Section III, we present quantization schemes designed for channel neural decoding algorithms. In Section IV, we incorporate the quantization scheme into different types of multi-way compression methods for channel neural decoding. We present and analyze simulation results in Section V, followed by the conclusions in Section VI.

III. QUANTIZATION FOR CHANNEL NEURAL DECODING

A. Quantization Schemes

In addition to the number of parameters, the number of bits required for each parameter also greatly affects the resources required for hardware implementation of channel neural decoding algorithms. We propose four quantization schemes to verify the impact of different quantization methods on the performance of channel neural decoding. The first scheme uses the rounding method and the second scheme uses the floor method. Both schemes are commonly used in communication systems.
The third quantization scheme can be described as follows [21]: The learning weights are divided into 4 categories, and all learning weights of the same category share the same value, which corresponds to an index. For each learned weight, only the corresponding index needs to be stored in memory.
In the fourth quantization scheme, we consider using a codebook-based quantization method to reduce the number of bits required for each learning weight. The quantization mechanism we use can be described as follows
• In the first step, we quantize n learning weights from floating point to q-bit fixed point [21]. The learning weights in the decoding algorithm are rounded to the nearest value that can be represented by q-bits [22].
• In the second step, based on the quantized learning weights, we design a codebook quantization scheme for weight sharing, which further reduces the number of quantized learning weights.
• In the third step, the codebook is designed based on the quantized q-bit learning weights. For 2q types of quantized learning weights, calculate the frequency of each learning weight.
• In the fourth step, since the learning weights in the codebook are quantized to the closest value and represented by binary indexes, the memory required for each learning weight is further reduced from q bits to c bits.

V. PERFORMANCE EVALUATION AND ANALYSIS
In this section, we present numerical results on the proposed quantization schemes and multi-way compression schemes. We first evaluate the pros and cons of the four quantization schemes on LDPC codes, and then extended the application of the two optimal quantization schemes to BCH and Hamming codes. Finally, we evaluate the performance of different types of multi-way compression methods on LDPC and BCH codes.

A. Quantization for channel neural decoding
We first evaluate the performance of four quantization schemes in channel neural decoding on LDPC codes to select the quantization method with smaller loss. The experimental results are shown in Fig. 1. It can be seen that quantization scheme 4 has the best performance, and quantization scheme 1 has the worst performance. Different quantization schemes bring different information loss, which leads to performance degradation.
We apply two quantization schemes (quantization scheme 3 and quantization scheme 4) with better performance on LDPC codes to BCH and Hamming codes. The experimental results are shown in Fig. (2) ∽ Fig. (5). For quantization scheme 3, the Hamming code has a slight performance drop in the high SNR region. BCH(63,45) and BCH(127,106) codes maintain almost the same performance as before quantization. For quantization scheme 4, the performance of the Hamming code after quantization exceeds the original performance, which due to the learning ability of the learned weights playing a greater role. Similarly, the BCH codes with two code lengths maintain comparable performance to the original NNMS+.

VI. CONCLUSION
Several multi-way compression methods have been proposed to reduce the complexity of channel neural decoding algorithms. We used quantization technology in multi-way compression for the first time, which has great significance for the hardware implementation of channel neural decoding algorithms. We conducted simulation experiments on LDPC, BCH and Hamming codes. Experimental results show that the proposed method can effectively reduce the complexity of channel decoding algorithm without causing significant performance degradation.
