Convergence Optimized Joint Detection and Decoding for LDPC Coded SC-MIMO Systems
Xuanbo Zhang, Kaining Han*, and Jianhao Hu National Key Laboratory of Science and Technology on Communications University of Electronic Science and Technology of China, Chengdu, China
Email: hkaining2014@gmail.com

Abstract—With the increasing requirement for high-speed transmission, multiple input multiple out (MIMO) technologies have been proposed. Beneficial from low peak-to-average power ratio (PAPR), single carrier (SC) MIMO systems are suitable for long-distance transmission. However, one of the significant challenges of SC-MIMO systems is the vast complexity of channel equalization and signal detection, especially for the intersymbol interference (ISI) caused by the frequency-selective fading channels. Due to acceptable complexity and near maximum likelihood (ML) detection performance, the Gaussian approximate interference (GAI) based belief propagation (BP) algorithm has attracted lots of attention. With the help of extrinsic information feedback from low-density parity check (LDPC) codes, iterative detection and decoding (IDD) scheme has been proposed, which has a near to approximate maximum a posterior (MAP) detection performance. Nevertheless, the IDD scheme still suffers from relatively slow convergence resulting from complex connections on the factor graph. Combined with the joint detection and decoding (JDD) scheme and two optimization methods, namely dynamic message damping and layered message updating, a convergence optimized joint detection and decoding (CO-JDD) algorithm is proposed in this paper. Evaluation results show that the proposed CO-JDD algorithm has 2dB bit error rate (BER) performance gains compared with IDD scheme. Besides, the proposed CO-JDD algorithm also has a 70% iteration reduction compared with the IDD scheme.
Index Terms—Single carrier MIMO,LDCP,Inter-symbol interference,Gaussian approximate interference,belief propagation
I. INTRODUCTION
Multiple input and multiple out (MIMO) technology [1] is widely used in wireless wide-band systems, which utilizes spatial multiplexing and diversity to improve system capacity and performance. Orthogonal frequency division multiplexing (OFDM) technology [2] is used in 4G long-term evolution (LTE) [3] and 5G new radio (NR) [4] mobile communication systems to reduce complexity channel equalization in broadband MIMO systems. However, sub-carriers with the same phase are stacked at receiving antenna which causes a high peak-to-average ratio (PAPR) [5]. One solution for the high PAPR of the MIMO system is to use power amplifier fallback technology, but this method will cause a significant loss of system efficiency. Another method is to adopt the single carrier transmission system and use linear algorithms such as ZF and MMSE detection algorithms. However, for some communication systems with high-performance requirements, the performance of linear detection is not satisfied. In recent years, many nonlinear algorithms have been proposed,

such as QR decomposition-based tree-search algorithm [6], Sphere Decoding (SD) algorithm [7], Lattice Reduction Aided algorithm [8]. Most of these algorithms adopt the method of traversal search, resulting in a vast detection complexity.
To achieve the near ML detection performance with acceptable complexity, a Gaussian interference approximation (GAI) method based belief propagation (BP) algorithm is proposed in [9], which equalizes the ISI and processes the MIMO detection simultaneously. To further improve the performance using the extrinsic information feedback of channel codes and achieve near maximum a posterior (MAP) detection, an outer-loop iterative MIMO detection and low density parity check (LDPC) code decoding (IDD) scheme is proposed in [10], [11]. However, the IDD scheme suffers from relatively slow convergence. So that brings significant complexity and processing time overhead.
A convergence-optimized joint detection and decoding (COJDD) algorithm is proposed in this paper, which improves performance and speeds up convergence simultaneously. The contributions of this paper can be summarized as follows.
• A joint detection and decoding (JDD) scheme is first proposed, which processes the MIMO detection and LDPC decoding in a joint factor graph. The extrinsic information of LDPC codes can be feedback to MIMO detection in time, producing a more efficient information exchange. Besides, the early stopping criteria would further significantly speed up the convergence at a relatively high SNR region.
• An Layered message updating strategy is proposed to speed up the convergence further. The updating strategy of JDD scheme is flooding message updating. Variable nodes only exchange messages once in one iteration through the observation node. Several message exchanges in one iteration can be expected using the proposed layered message updating strategy.
• Finally, a dynamic message damping method is proposed. The message damping method is an effective method to improve the performance of the iterative algorithm. However, its disadvantage is that it reduces the convergence speed of the algorithm. The dynamic message damping method is proposed based on the cumulative density function (CDF) analysis of the iterations to accelerate the convergence without performance degradation.

C. Iterative Detection and Decoding Scheme
The error correcting code (ECC) is widely used in wireless communication systems to improve quality and performance. The most direct scheme is to use individual detection and decoding. scheme is to use individual detection and decoding. MIMO detection and LDPC decoding are processed individually, where the soft symbol information after the detector is directly sent to the LDPC decoder. However, the extrinsic information of LDPC codes is wasted by using individual detection and decoding scheme.
In order to utilize the extrinsic information, an iterative detection and decoding (IDD) scheme is proposed and widely used in some research like [10], [11]. After the MIMO detection and LDPC decoding, the extrinsic information of the LDPC decoder is feedback to MIMO detection, forming an outer-loop. IDD scheme has a significant performance improvement compared with individual detection and decoding scheme. However, the IDD scheme suffers from relatively slow convergence. So that brings significant complexity and processing time overhead.

III. JOINT DETECTION AND DECODING SCHEME AND CONVERGENCE OPTIMIZATION
This section first proposes a joint detection and decoding (JDD) scheme to achieve more frequent information exchange. Afterward, two optimization methods, namely dynamic message damping and layered message updating, are proposed to accelerate the convergence further. Moreover, at the end of this section, the convergence optimized joint detection and decoding (CO-JDD) algorithm is summarized as Algorithm III-B.

C. Dynamic Message Damping Method

Traditional constant coefficient message damping is an efficient optimization method to improve the performance of the iteration algorithm. In the above JDD scheme, the process of message damping as

vm(t)l = αvm(t−l 1) + (1 − α)vm(t)l

(15)

where α is the message damping factor, its value would affect the convergence and performance.
Figure 3 shows the BER simulation of the JDD algorithm with different α damping factors for a 16QAM modulated SCMIMO transmission system with antennas M = N = 2. It can be observed that the optimal performance damping factor should be about α = 0.5. Figure 4 shows the cumulative distribution function (CDF) of the JDD algorithm with different α factors. It can be observed that a smaller damping factor would speed up the convergence, but still exists residual bit

errors. Although the JDD algorithm with α ≥ 0.5 can solve the residual bit errors problem, it suffers from relatively slow convergence.
A dynamic message damping method is proposed in this section to achieve faster convergence without residual error problems. The damping factor is initialized and set to α = 0.3, increasing by 0.025 every 5 iterations until α = 0.6. The corresponding CDF curve is also plotted in Figure 4 as a comparison. It can be observed that compared with the fixed damping factor α ≥ 0.5, the dynamic message damping method achieves a significant convergence speedup without residual bit errors.

V. CONCLUSION
This paper first proposes a joint detection and decoding scheme (JDD). By exchanging information in each iteration, the proposed JDD scheme improves performance compared with the individual detection and decoding scheme and speeds up convergence simultaneously compared with the IDD scheme. Afterward, a layered message update strategy and a dynamic message Damping method are proposed to further enhance the extrinsic information exchange and speed up convergence. Evaluation results show that the proposed CO-JDD algorithm has 2dB performance gains compared to individual schemes and about 70% iteration reduction concerning the IDD scheme. The proposed CO-JDD algorithm could be a promising high-performance candidate for the wide-band SCMIMO transmission systems over frequency selective fading
