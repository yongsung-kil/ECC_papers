Design of F-L-QC-LDPC Decoder Based on LNMS Decoding Algorithm

Youyao Liu School of Electronic Engineering
Xi'an University of Posts and Telecommunication Xi’an, China

Haimei Huang School of Electronic Engineering
Xi'an University of Posts and Telecommunication Xi’an, China

Abstract— Low Density Parity Check Code (LDPC) can meet
the decoding requirements of a variety of code lengths and rates in different communication scenarios, but the LDPC code check matrix is irregular, which has problems with difficult storage and reading of the check matrix and high coding complexity. For the decoding requirements of multi-code length, this paper proposes a Quasi Crystal Low Density Parity Check Codes (QCLDPC) code based on Fibonacci Lucas sequence, which can obtain the codewords with different code length and code rate by changing the number of rows and columns of the exponential matrix. In terms of decoding algorithms, the Layered NonMaximum Suppression (LNMS) decoding algorithm, which combines the minimum sum decoding algorithm with the layered algorithm, is proposed to improve the convergence speed of decoding iteration and decoder throughput. Through simulation and comparative experiments, it is proved that this design can support a variety of bit rates, block lengths and sub matrix sizes, and meet the requirements of high throughput and low complexity decoder in modern communication systems.

Keywords: Fibonacci-Lucas sequence, QC-LDPC code, LNMS decoding algorithm.

I. INTRODUCTION

LDPCs in emerging wireless communication systems must be able to support different code lengths and rates. It was pointed out that LDPC codes basically belong to the random construction method, and the constructed codes have good performance [1], but are relatively difficult to decoding. It is said that after years of research by researchers, LDPC code related technology of LDPC code has made breakthroughs in all aspects, and the technology has gradually matured [2], and even its results have been commercialized and entered wireless communication and other related fields. However, the complexity of LDPC code compilation code is large, which is not conducive to hardware implementation. It is proposed that QC-LDPC is an important subset of structured LDPC codes 3]. Its parity check matrix can be divided into multiple square matrices of equal size. Each square matrix is a cyclic shift matrix or all zero-matrix identity matrix, which is very convenient for memory storage and addressing, thus greatly reducing the complexity of encoding and decoding LDPC codes. It is stated that QCLDPC code can achieve a relative balance in decoding performance and decoding complexity to obtain the best decoding [4], which has attracted more and more attention from academia and industry. Therefore, it is essential to design a QC-LDPC decoder that flexibly meets different code lengths and rates in different communication scenarios of 5G New Radio (NR) proposed [5].

B. Interlayer Pipeline Structure

According to the LNMS decoding algorithm, we need to

find the minimum value and the sub-minimum value in the

algorithm. The serial processing method is to read the

variable node information in sequence. Results after the first

and second comparisons are compared with the third. It takes

more clock cycles to complete the operation according to this

method. The parallel processing method reads the variable

node information corresponding to the calibration node in

parallel, and then obtains the minimum value and the sub-

minimum value in groups. Therefore, a partial parallel

structure is adopted in this design to improve the convergence

speed.

Partial parallel design in the decoder refers to layering the

elements of the H check matrix in rows, where parallel

computing is carried out within layers, and there is a serial

structure between layers. As shown in Figure 7, the

calculation of the next layer can only be calculated after all

process calculations of one layer have been completed. This

is because the posterior probabilities of the variable nodes

corresponding to the two layers generally overlap, the next

layer can only be calculated after the posterior probability of

the updated variable node of a layer is returned, so that the

updated posterior probability of the layer can be used to

calculate the next layer, thus accelerating the convergence

speed of iteration. Obviously, when the check matrix has

more layers, the decoding delay will increase.

V. SUMMARY
In view of the fact that the traditional decoder usually uses a basic matrix to realize the decoding of large code length and high code rate, this paper proposes a QC-LDPC decoder based on Fibonacci Lucas sequence, which can select different parity check matrices to realize the decoding function for a variety of code lengths with different code word lengths. Aiming at the problem of limiting the throughput of the LNMS algorithm, this paper designs an inner layer pipeline decoder, which solves the problem of interlayer serialization and improves throughput. With the development of hardware, QC-LDPC decoder based on LNMS algorithm will have a good application prospect in the case of high throughput requirements.
