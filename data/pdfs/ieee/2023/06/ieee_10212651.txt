Abstract—In this paper, we present the design and implementation of a high efficient low-density paritycheck (LDPC) encoder for deep space applications. The proposed encoder utilizes the generator matrix of CCSDS LDPC codes to simplify the encoding process and reduces the complexity of hardware implementation. The proposed encoder has two types. The first type aims to design a low-complexity architecture and flexibility. The second type presents high throughput architecture, allowing the user to choose the appropriate type according to their usage condition. The results of FPGA synthesis show that the proposed LDPC encoders achieve flexibility, low complexity, and high throughput.
Keywords—channel coding, LDPC code, CCSDS, space, satellite
I. INTRODUCTION
Low-density parity-check (LDPC) codes were proposed by Gallager [1], which achieved the Shannon's limit [2]. As a result, LDPC codes have been chosen as channel codes for many standards, such as CCSDS [3] and 5G NR [4]. In this paper, we will focus on deep space communications which utilize QC-LDPC codes. In [5] and [6], the encoders of QC-LDPC code were designed to be more efficient and lower complexity by using a shift register. In [7], the low complexity cyclic encoder was proposed to support the QC-LDPC codes. The research works in [8] and [9] presented the development of recursive convolutional encoder (RCE) [7] to support CCSDS standard. The encoder consists of a series of registers and logic gates. However, it has no flexibility since their hardware implementation supports some configuration.
In this work, we consider the FPGA implementation of CCSDS LDPC encoder. We design 2 types of encoders. The first type is a flexible encoder to support every code length defined by the standard, while retains low complexity. Therefore, it could be an option for some application, which requires flexible encoder modules. The second type will focus on maximizing

throughput which can serve high-speed demand applications.
II. BACKGROUND
A. LDPC Codes
Given that the information bits u = [u1,u2 ,u3 ,...,uK ] are encoded by an LDPC encoder to obtain the codeword bits c = [c1,c2 ,c3 ,...,cN ] , where K and N are the length of information bits and codeword bits, respectively. The codeword c can be obtained by multiply information bits with the generator matrix G as

c = uG.

()

The LDPC code has a parity-check matrix H that represents the relationship among the code bits. The codeword c must satisfy the condition,

cHT = 0.

()

In general, the parity check matrix H of LDPC codes has two different structures, such as regular and irregular codes. The LDPC codes are called regular codes if the degree distributions of rows and columns in H are uniform; otherwise, the LDPC codes are called irregular codes. In CCSDS standard, the parity-check matrix is generally constructed by using quasi-cyclic technique, which the parity-check matrix can be presented by

B. High throughput encoder (Type-B)
High throughput encoder is designed by reducing an encoding latency and increasing parallelism of the encoding process. In this work, we will use the cyclic barrel shifter to match these criteria, since it can perform any circulant shifting within one clock period. However, barrel shifters will consume a significant resource. The structure of cyclic barrel shifter is shown in Fig. 10. where a is an input, y is an output and b is
a shifting distance selector flag.

will use barrel shifter to shift each input in every circulant. The final step is to sum all those shifted input together. We proposed 3 variations of encoder to support different usage scenarios in terms of throughput and resource usage.
1) Type B-1 For type B-1, the number of barrel shifters equals to the amount of sub-matrix columns in GQC . These shifters will shift every circulant from first circulant row of GQC down to the bottom in every set of sub-matrices.

2) Type B-2
For type B-2, the number of barrel shifters equals to the amount of all sub-matrices in GQC . These shifters will shift every circulant of GQC in every set of submatrices.
3) Type B-3
For type B-3, the number of barrel shifters equals to the amount of a summed sub-matrix in GQC . These shifters will serially shift the first single circulant to a last one of each set of sub-matrices in GQC .
