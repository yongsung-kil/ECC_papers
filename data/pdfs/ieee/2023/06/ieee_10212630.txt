Abstract—Low-density parity-check codes(LDPC) have the advantages of high performance, flexibility, parallelized processing and iterative decoding, which make them one of the widely used error correction code schemes in communication and storage systems. In order to improve the decoding convergence speed of low-density parity-check codes, the idea of layered decoding is added to the log-likelihood ratio confidence propagation decoding algorithm, and an improved algorithm based on layered decoding strategy is proposed, which solves the problem of low decoding rate caused by vertical iteration after the completion of horizontal iteration by using the log-likelihood ratio confidence propagation decoding algorithm in each layer, at the same time. The results show that the average decoding rate of the algorithm is improved by about 25.5% compared with that before the improvement without losing or almost losing the error correction performance, and the performance of the improved layered decoding algorithm with log-likelihood ratio confidence propagation decoding is optimal when the signal-to-noise ratio is greater than 4, and the decoding performance is improved.
Keywords-component; Low-density parity-check codes;iterative decoding; layered decoding;signal-to-noise ratio
I. INTRODUCTION
The decoding algorithm of LDPC codes determines whether the error correction capability of the code word can be maximized, i.e., the highest decoding performance can be achieved with the lowest computational complexity.LDPC codes cover a variety of basic decoding algorithms, among which the more widely used algorithms include the belief propagation (BP) decoding algorithm, the logarithmic domain confidence propagation decoding algorithm, and the least-sum decoding algorithm[1]. Among them, the confidence propagation decoding algorithm passes probabilistic information back and forth between the bit node and the check node to obtain the maximum coding gain, but the algorithm has a large number of multiplication operations, so it increases the computational complexity. At the same time, the log-likelihood ratio belief propagation(LLR-BP) decoding algorithm uses the log-likelihood ratio (LLR) as the information metric in the decoding process. The LLR-BP decoding algorithm uses the log-likelihood ratio as the information metric in the decoding process, which is the logarithm of the relative probability of the received signal to the possible values of each bit of the

codeword[2]. Compared with the traditional hard judgment metric, LLR can more accurately express the probabilistic information of bit fetching, which improves the accuracy of decoding, while the LLR-BP decoding algorithm continuously passes and updates the LLR information through a multi-round iterative correction process to improve the decoding performance[3].

B. Improved LLR-BP Decoding Algorithm Based on Hierarc-hical Decoding

Traditional decoding methods usually perform data delivery through horizontal update and vertical update steps. After the horizontal iterative update is completed, the vertical iterative update is then performed[9].Only after all iterations are completed is a complete update of the decoded information completed. However, this process consumes a lot of time and reduces the decoding efficiency. In addition, in the hardware implementation flow, the iterative computation of the decoding algorithm generates a large amount of intermediate quantities that require a large amount of memory to be occupied for storage[10].

IV. SIMULATION EXPERIMENT AND RESULT ANALYSIS
In order to simulate and analyze the layered decoding LLRBP decoding algorithm proposed in this paper, we will use Matlab simulation software for the application. During the simulation, we will consider metrics such as BER, SNR, decoding rate (average decoding time), and further data analysis, including the simulation data analysis of applying the improved decoding algorithm with modulation added. Through these analyses, we will evaluate the performance and effectiveness of the decoding algorithm.
A. BER Performance Simulation
Based on the proposed layered decoding log-likelihood ratio belief propagation (LD-LLR-BP) decoding algorithm, the modulation method is set to BPSK, the maximum number of iterations is set to 10, the code rate is set to 512, the code length is set to 512, the transmission channel is a Gaussian white noise channel, and the performance is compared with the least-sum decoding algorithm (SPA) and the LLR-BP decoding algorithm. For the proposed hierarchical decoding LLR-BP decoding algorithm, simulations are conducted to compare the performance of different modulation methods using switch keying modulation, binary phase shift keying modulation and pulse position modulation, respectively. The LDPC code based on the hierarchical decoding LLR-BP decoding algorithm with a code rate of 1 2 and 10 iterations is used in the simulation, and the rest of the parameters are the same.
As can be seen from Figure 3, at a BER of 5 ´10-5 , there is about 1.5 dB coding gain with binary phase shift keying modulation than with pulse position modulation, and about 1.5 dB coding gain with pulse position modulation than with switch keying modulation. In other words, for the decoding algorithm in this paper, the BER of the channel compilation code of the information sequence using OOK, PPM and BPSK modulation decreases and the decoding effect becomes better in turn.

V. CONCLUSIONS
This paper focuses on the LDPC decoding algorithm with the improved hierarchical decoding log-likelihood ratio confidence propagation decoding algorithm, which is simulated and compared with the least-sum decoding algorithm and LLRBP decoding algorithm with an average decoding time
reduction of 28.9% and 25.5% , respectively, under the same parameters. Also by introducing binary switch keying modulation and pulse position modulation, it is concluded that BPSK modulation is more compatible with the improved decoding algorithm and has better performance, pulse position modulation has the second best performance, binary switch keying modulation has the worst effect and is not applicable to the hierarchical decoding algorithm based on log-likelihood ratio confidence propagation improvement.
