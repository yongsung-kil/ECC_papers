GC-LDPC Coded Modulation
Kun Zhu, Hongwen Yang* School of Information and Communication Engineering
Beijing University of Posts and Telecommunications Beijing, China
*Corresponding author: Hongwen Yang, yanghong@bupt.edu.cn

Abstract— Due to the scarcity of spectrum resources, the combination of forward error correction (FEC) code and highorder modulation, i.e. coded-modulation (CM) is essential to the various high data rate multimedia communication services. In this paper, we propose a new CM scheme, namely the globally coupled (GC) low-density parity check (LDPC) CM where we choose GC-LDPC code as the FEC code and regular rectangular quadrature amplitude modulation (QAM) as the high order modulation. To guarantee the code with at least grith 8, the greatest common division-based method is used to constructe the GC-LDPC code. Simulation results show that, although the GC-LDPC code has poor performance under QPSK modulation compared with the LDPC code defined in 5G (5GLDPC), the proposed GC-LDPC CM outperforms 5G-LPDC CM under 64QAM modulation about 0.5dB. The EXIT charts also theoretically demonstrate the GC-LDPC code the significant gain. Considering other advantages of GC-LDPC such as flexible rate design, parallel decoding and etc., the proposed GC-LDPC CM is attractive to multimedia communication systems which generally requires high spectral efficiency.
Index Terms—GC-LDPC code, coded modulation, multimedia communication
I. INTRODUCTION
Multimedia communication systems generally requires high data rate transmission to support the high quality transmission of audio, vedic, VR/AR and many other services. With fixed channel bandwidth, high rate transmission is typically realized with high order modulation, such as quadrature amplitude modulation (QAM). and delay are crucial indicators for multimedia communication systems. The high-order coded modulation is a conventional technology to raise the spectrum efficiency. On the other hand, high quality communication requires low error rate which can be achieved with various forward error correction (FEC) codes. Coded modulation (CM) is the combination of FEC code and high order modulation which can provide high spectral efficiency communication with high reliability. With the increase of multi-media data transmission traffic, the significance of CM is gradually increasing. CM is a popular design option for various multimedia communication systems [1].
Coded modulation can be divided into two categories depending on the degree of coupling between FEC code bits and M -ary constellation points. With the trellis coded modulation (TCM) Ungerboeck [2] suggested, the mapping between code

bits of convolutional code and the constellation points of multilevel/multi-phase modulation is carefully designed to guarantee good decoding performance. Set partitioning mapping is used in TCM to achieve maximum free Euclidean distance. The TCM performs very well in additive white Gaussian noise (AWGN) channel. However, it only gets the sub-optimal performance in fading channel. On the other hand, with the bitinterleaved coded modulation (BICM) [3], the interleaved FEC code bits are directly mapped to constellations points without considering set-partitioning. The interleaving can improve the diversity performance and hence in the fading channel, the BICM system has better perform than TCM. BICM is now widely adopted in various systems due to its flexibility and good performance.
The performance of BICM mainly depends on the design of FEC codes. In recent decade, low-density parity check (LDPC) codes are popular choice in BICM systems. Spatially coupled (SC) LDPC code [4] and globally coupled (GC) LDPC code [5]–[7] are recent innovations in LDPC code design. As it occurs to the threshold saturation phenomenon, SC-LDPC code outperforms the conventional block code (BC) LDPC code. [5], [8], [9]. GC-LDPC code performs well over the AWGN and binary-erasure channel (BEC). GC-LDPC are generally constructed through algebraic methods. Recent studies utilize the GC-LDPC code in NAND flash [10]. GCLDPC codes have excellent convergence characteristic and the code structure is easy for design and hardware realization.
This paper proposes a new coded modulation scheme, namely the GC-LDPC coded modulation. In the proposed scheme, we use GC-LDPC code to realize BICM. To guarantee the code with at least grith 8, the greatest common division-based method is used to construct the GC-LDPC code [11]. Simulation results show that, although the GCLDPC code has poor performance under QPSK modulation compared with the LDPC code defined in 5G (5G-LDPC), the proposed GC-LDPC CM outperforms 5G-LDPC CM under 64QAM modulation about 0.5dB @BER=1e-6. Considering other advantages of GC-LDPC code such as flexible rate design, parallel decoding and etc., the proposed GC-LDPC CM is attractive to multimedia communication systems which generally requires high spectral efficiency.

III. GC-LDPC CODE
In our proposed scheme, we use GC-LDPC as the FEC code of coded modulation. The unique coupling structure and relatively independent local codes design of GC-LDPC code are the main features that attract the attention of scholars. Globally coupling check constraints connect several relatively independent shortcodes. The high connectivity between local codes ensures the performance of the whole code. In AWGN and BEC channels, GC-LDPC codes have illustrated excellent decoding performance and fast convergence as well. Each local code can be decoded independently, or it can be decoded globally as a part of the whole extended code. This feature allows GC-LDPC code to utilize the [6] local/global twophase decoding technique. Shortcodes have a high possibility of being successfully decoded in the high signal-to-noise ratio (SNR) region, which lowers the overall decoding complexity.

V. SIMLUATION AND COMPARSION
In prior literatures, the study of GC-LDPC code has been limited to BPSK modulation. In this section, the GCDbased GC-LDPC code is utilized in QPSK and 64QAM. To demonstrate its properties in high order modulation, 5G-LDPC code is used as a comparison scheme. The classic construction methods of GC-LDPC are limited to girth 6, and these GC-LDPC codes have similar performance in same parameters. Therefore, for the RS-liked-GC-LDPC code has the most flexibility, it is used in the following simulations as the representative of classic GC-LDPC codes. The type of BP decoding in this paper is selected as MinSum, and the scale factor is 0.5. The iterations in each simulation is set as 50 times.
In the first simulation, the GC-LDPC code, which named GC-GCD-1, is constructed. Firstly, the 5 × 17 FLRM matrix is built. Through the algorithm in [11], the minimum CPM size is Pm∗ in = 357. Setting t = 2, d1,1 = d1,2 = 4 and d2 = 1. Using the Construction 2 in [11], the exponential matrix of GC-GCD-1 is sized as 9 × 34. To loose the column weight, the cyclic mask operation is utilized to each local code with Mn = 1, and the final column weight is 4. Through the CPM dispersion, the parameters of final GC-GCD-1 are n = 12138, r = 0.735.
Afterward, Using the RS-liked construction method to build the GC-RS-1 code in the same parameters. Firstly, the primary GC-LDPC exponential matrix E1. The local parts are RS1(5, 53, 6), and the global part is RS2(1, 265, 1).

Then, arbitrarily delete columns corresponding to three of the local matrices. Adjusting the number of columns in the remaining two local matrices, retaining 24 columns. The remain exponential matrix of GC-RS-1 is E1,which is sized as 13 × 48. The cyclic mask operation is also used in the code to reduce the columns’ weight, the Mn = 3. The final parameters of GC-RS-1 are n = 12720 and r = 0.73.
For comparison, this simulation adopts 5G-LDPC code under similar parameters. For the number of information bits in 5G-LDPC code must be under 8448, it is unable to compare with the code GC-GCD-1 in the same parameters. An approximate set of parameters is used to 5G-LDPC code, named 5G-1. Setting the information length K = 8440, 24 CRC check bits need to be added, and the total length of information bits is greater than 8448. Therefore, the code length of 5862 is divided into two times, including two bits of zero complement. The final parameters of 5G-1 are n = 11722 and r = 0.72.

VI. CONCLUSION
This paper proposed the GCD-based GC-LDPC code over QAM modulations. Through the simulation results, the practicality of GC-LDPC codes to high - order modulation is discovered. The structure of the constructed GC-LDPC code is inferior to that of the carefully designed 5G-LDPC code, which leads the performance of the GC-LDPC code is slightly worse under low-order modulation. However, GC-LDPC codes with the same parameters have obvious advantages over 5G-LDPC codes in 64QAM modulation, and gain is greater than 0.5dB in most cases. This discovery proves an important research significance and direction of GC-LDPC codes. In the future communication, it is inevitable to face the shortage of frequency band resources and the huge amount of data transmission. The friendly properties of GC-LDPC codes for high - order modulation are of great research value. Especially for multimedia and broadcast data with large packet information, the superior performance of GC-LDPC codes under high order modulation provides a new design idea for saving spectrum resources.
