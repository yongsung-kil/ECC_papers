Abstract—In this work, we propose a neural adjusted minsum (NAMS) decoder for low-density parity-check (LDPC) codes. In particular, we improve the traditional normalized min-sum (NMS) decoder by introducing a selection mechanism to adjust the check-node update step, where either the min-sum rule or the belief propagation (BP) rule is selected. Besides, we unfold the modified decoder into a model-driven neural network, where layer-dependent trainable parameters are introduced as weights in the Tanner graph and optimized by gradient descent-based methods during network training. Simulation results demonstrate that the proposed NAMS decoder is able to provide superior error-correction performance as compared to the neural NMS decoder, with only slightly increased computational complexity. Moreover, in certain circumstances, the proposed NAMS decoder even outperforms the neural BP decoder, with much lower computational complexity.
Index Terms—LDPC codes, belief propagation, min-sum decoding, deep learning, deep unfolding, model-driven.
I. INTRODUCTION
L OW-DENSITY parity-check (LDPC) codes which were first discovered by Gallager [1] in the 1960s have been shown to achieve near-Shannon-limit performance under belief propagation (BP) decoding algorithms [2], [3]. BP is a classic decoding algorithm, also known as sum-product algorithm, which is based on the idea of iteratively updating probability information using factor graphs and it is able to achieve performance close to maximum likelihood (ML) decoding. However, the BP decoder involves a large number of multiplication and hyperbolic tangent operations, resulting in high computational complexity. As a result, the min-sum (MS) decoder was introduced [4], which simplifies the update of check nodes in the BP decoder. Compared with BP, the MS decoder has much lower computational complexity at the cost of certain performance degradation.
The performance loss of the MS decoder as compared to the original BP decoder can be partially compensated by using a normalization factor or an offset factor [5], and the resulting decoders are known as normalized MS (NMS) and offset MS (OMS), respectively. Furthermore, these decoders can be further improved by properly adjusting and optimizing the underlying normalization/offset coefficients, as mentioned in [6]. Besides, an adjusted MS (AMS) decoder was proposed in [7], which further improves the OMS and NMS decoders by modifying the check node updating rule and it was shown to achieve close to BP performance.

In recent years, with the vigorous development of machine learning technologies, deep learning has become a popular technique to solve signal processing tasks for communication systems [8]. Specifically, the authors in [9] proposed to use deep neural networks (DNN) for one-shot decoding of random and structured codes, and it was shown that using DNN can achieve near-ML performance for very short codes. In [10], the authors proposed a neural network-aided normalized minsum decoder, where the normalization factors are inferred from the input log-likelihood ratio (LLR) by a pre-trained DNN. However, such data-driven neural networks that are regarded as black boxes usually exhibit poor interpretability, and training such black-box-based networks requires a lot of training time in addition to a huge data set. Therefore, deep unfolding has gained significant attention as a modeldriven deep learning approach that transforms an iterative algorithm into a fixed-depth network while maintaining the original algorithm’s structure [11]. Some parameters of the original algorithm can be treated as trainable parameters and optimized using gradient-based methods. Based on this idea, the work [12] introduced a neural BP (NBP) decoder, which unfolds the traditional BP decoder into a partially connected neural network and it was shown to improve the decoding performance. Besides, linear programming based decoders assisted by deep unfolding were proposed in [13] and [14], which are shown to achieve much better performance than BP-based decoders but with higher computational complexity. To lower the complexity, the authors in [15] proposed a neural OMS (NOMS) decoder and a neural NMS (NNMS) decoder, which are able to achieve comparable performance as compared to those presented in [12]. In additon, the work [16] introduced a NNMS decoder for protograph LDPC codes, which is constructed by a special parameter-sharing mechanism.
In this work, we aim to improve the decoding performance of the NNMS decoder by proposing a neural adjusted MS (NAMS) decoder. Specifically, we first introduce a neural network based selection mechanism to automatically adjust the check node updating rule efficiently. Then, we propose to assign a learnable weight to each check node processing in each iteration and improve the decoding performance via network training. Numerical results show that the proposed NAMS decoder is able to achieve superior performance as compared to the NNMS decoder, and even outperforms the NBP decoder under certain conditions.

A. NAMS Decoder

The structure of the proposed NAMS decoder is illustrated

in Fig. 1. As can be seen, the proposed network is mainly

composed of an input layer, T hidden layers, and an output

layer, where T also represents the maximum iteration number.

The input of the decoder is the received LLR vector as defined

in (1), while the final output is the hard decision result, i.e.,

xˆ. All the hidden layers are of size E, where E denotes the

number of edges in the Tanner graph. For ease of exposition,

we let t denote the hidden layer index, which corresponds to

the t-th iteration in the decoding process. Each hidden layer

further includes two sublayers corresponding to variable node

update and check node update, respectively. The variable node

update sublayer outputs the messages from variable nodes to

check nodes, which are defined in (3), while its inputs are the

output of the previous check node update sublayer as well as

the received LLR. The check node update sublayer, similarly,

outputs the messages from check nodes to variable nodes.

However, in this work, we propose a different strategy for

updating the check nodes.

As mentioned in the previous section, the traditional AMS

decoder employs the BP rule for the edge that carries the

message with the smallest magnitude. This implies that in

the AMS decoder, the check node updating rule is mainly

determined by the input LLR, but is irrelevant to the code

structure (e.g., the loops in the Tanner graph) and the number

of iterations. However, this may lead to the accumulation and

amplification of probability estimation errors, especially dur-

ing the early iterations, which eventually affects the decoding

performance. Moreover, due to the specific code structure,

the approximation errors caused by the MS rule vary for

each check node, i.e., some nodes may experience significant

overestimation, while others exhibit quite precise estimations.

To illustrate this phenomenon, we calculate the check node

outputs using both the BP and MS rules, and then plot the

normalized misestimation level in Fig. 2, which is defined as

To address these issues, we propose to use a neural network aided selection mechanism in the check node update sublayer. In order to automatically select the check node updating rule, a discrete vector λ ∈ {0, 1}T E×1 is introduced as a trainable parameter. In particular, if λ(t, e(i,j)) = 1, where e(i,j) is the index of the edge that passes the message from the i-th check node to the j-th variable node, then the corresponding message in the t-th iteration is calculated via the BP rule; otherwise the MS rule is employed. In this way, the decoder can be trained to learn appropriate check node updating rules, understanding when and where to utilize the BP rule can minimize the performance loss caused by the MS approximation. B. Training the NAMS Decoder

In this subsection, we propose a two-step training approach for the proposed NAMS decoder. In the first step, we train the selection parameters in λ with fixed normalization factors in α. Then, in the second step, with given check node update rule (BP or MS) obtained in the first step, the weights in α are further trained to improve the error-correction performance. 1
Specifically, in the first step, to reduce the computational complexity of the proposed NAMS decoder, the selection parameter vector λ should be a sparse vector with more zeros than ones, such that the MS rule is activated more frequently. Therefore, we propose to add a penalty term in the loss function to penalize the ones in λ. Besides, based on the multi-loss training method [12], our loss function is designed as follows:

In the second step, since λ has been optimized and fixed, the general structure of the proposed NAMS decoder is determined. To further improve the decoding performance, we train the edge weights in the Tanner graph, i.e., {α(t, e(i,j))}. Different from the first step, only the multi-loss term, i.e., the first term on the right hand side of (11), is employed as the loss function. Additionally, we utilize an iterationby-iteration greedy training approach, which means that the proposed NAMS decoding network is growing from a onelayer network with only one iteration to a multi-layer network with T iterations, and each time only the weights in the last layer are learnable.

IV. NUMERICAL RESULTS
In this section, we present simulation results to illustrate the decoding performance of the proposed NAMS decoder. The platform used for neural network training is TensorFlow. In the backpropagation stage, the Adam optimizer is employed to optimize the trainable parameters [18]. In all of our simulations, we assume that the codewords are transmitted over additive white Gaussian noise (AWGN) channels with binary

Generally, training data plays an important role in network optimization. To improve the generalizability of the considered neural network based decoders over the entire signal-to-noise ratio (SNR) regime, especially for the relatively high or low SNRs, the training SNRs are set to [1, 2, . . . , 5] dB for C1 and [1.5, 2.0, . . . , 3.5] dB for C2. Besides, we build the training data set by generating 72000 training samples for each SNR level, and the mini-batch-based method is employed to train the network, where each mini-batch contains 120 blocks of data. All-zero codewords are chosen as the transmission codewords and the learning rate is set to 0.001.

V. CONCLUSION
In this work, we proposed a new neural network based decoder, namely NAMS, which automatically learns the check node updating rule via a properly designed selection mechanism. Simulation results showed that the proposed NAMS decoder provides superior error-correction performance with slightly higher computational complexity as compared to NNMS decoder. Moreover, when the iteration number is large, it even outperforms the NBP decoder in the high SNR regime.
