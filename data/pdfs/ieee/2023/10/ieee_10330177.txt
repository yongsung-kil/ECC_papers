Trapping and Absorbing Set Enumerators for Protograph-Based Generalized Low-Density
Parity-Check Code Ensembles

Emna Ben Yacoub Institute for Communications Engineering Technical University of Munich, Germany
Email: emna.ben-yacoub@tum.de

Abstract‚ÄîGeneralized low-density parity-check (GLDPC) codes have outstanding performance outperforming the standard low-density parity-check (LDPC) codes. However, the performance of these codes under iterative decoding can be degraded because of the presence of trapping, absorbing and fully absorbing sets in the code Tanner graphs. The (elementary) trapping and (fully) absorbing sets of protograph-based GLDPC codes are studied. The finite-length and asymptotic distributions of (elementary) trapping and (fully) absorbing sets for protographbased GLDPC code ensembles are derived and evaluated. The asymptotic distributions can be efficiently computed by solving a system of equations. The relative minimum ‚àÜ-trapping/(fully) absorbing set size can be directly computed by adding one equation to the system of equations. This is useful for the code design where we can restrict the search to ensembles for which the minimum ‚àÜ-trapping and (fully) absorbing set sizes grow linearly with the blocklength. The normalized logarithmic asymptotic distributions of (elementary) trapping and (fully) absorbing sets for an example GLDPC code ensemble are evaluated.
I. INTRODUCTION
The class of GLDPC codes [1] represents a generalization of LDPC codes, where more powerful linear block codes can be used at the constraint nodes (CNs) instead of single paritycheck (SPC) codes. These codes are attractive for applications demanding low rate codes with good waterfall and error floor performance [2]. The growth rate of the weight distribution for unstructured GLDPC code ensembles was derived in [3] and extended to doubly-generalized low-density parity-check (D-GLDPC) in [4]. Protograph-based LDPC and GLDPC codes have a simple design and encoding and are suitable for high-speed decoding. They are used in many communications standards and were studied in [2], [5], [6]. The ensemble weight enumerators for protograph-based GLDPC codes were derived in [7]. The weight distribution of a code can be used to estimate its performance under maximum likelihood (ML) decoding. However, it is important to identify the graphical configurations in the code‚Äôs Tanner graph that may lead to decoding failures under iterative decoding. These harmful structures are referred to as pseudocodewords, stopping [8], trapping [9], [10] and (fully) absorbing sets [11]. Since the enumeration of trapping and (fully) absorbing sets present in a specific code‚Äôs graph is complex, the difficulty can be bypassed by analyzing the average trapping and (fully) absorbing set enumerators of a GLDPC code ensemble. Several works

studied the enumerators of these structures for irregular and protograph-based LDPC code ensembles [11]‚Äì[13]. In [14], definitions of a trapping set (TS), absorbing set (AS), and fully absorbing set (FAS) for GLDPC codes were proposed and the finite-length and asymptotic enumerators of (elementary) trapping and (fully) absorbing sets for irregular GLDPC code ensembles were derived. In this work, we extend the work in [14] to protograph-based GLDPC code ensembles. We derive the distributions of (elementary) trapping, absorbing and fully absorbing sets for protograph-based GLDPC code ensembles using generating functions. We present an efficient method to evaluate the normalized logarithmic asymptotic enumerators.
II. PRELIMINARIES
A. Protograph-Based Generalized LDPC Codes
Protograph-based GLDPC codes are created from a small Tanner graph called a protograph by a copy-permute procedure. The protograph P = (V, C, E ) consists of nP variable nodes (VNs) and mP CNs and e edges. It is characterized by an mP √ó nP base matrix B = [bi,j], where bi,j is the number of edges connecting vj to ci. Each VN/CN/edge in a protograph defines a VN/CN/edge type. Let Evj ( Eci ) be the set of edges connected to vj (ci). The degree dvj of vj (dci of ci) is then equal to |Evj |(|Eci |). For GLDPC codes, each CN ci can be any (dci , ki, dmin,ci ) linear block code Pi. A CN is called a super CN if it is associated with a linear block code different than the SPC code. Further, we assume that dmin,ci ‚â• 3 for super CNs. Q copies of the protograph are generated and their edges are permuted such that the structure imposed by the protograph is maintained. A protograph-based GLDPC code ensemble CnP is defined by the set of length-n GLDPC codes whose Tanner graph is obtained by lifting P (with a specified set of CN types).

VII. CONCLUSION
We derived the normalized logarithmic asymptotic distributions of (elementary) trapping and (fully) absorbing sets

for protograph-based GLDPC code ensembles. The approach
is similar to the ones proposed in [21], [22] to compute the
weight, stopping, and trapping set distributions. Evaluation of
the expressions requires solving a system of equations.
