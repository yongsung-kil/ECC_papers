Trapping and Absorbing Set Enumerators for Multi-Edge Type LDPC Code Ensembles
Emna Ben Yacoub Institute for Communications Engineering Technical University of Munich, Germany
Email: emna.ben-yacoub@tum.de

Abstract—The finite-length and asymptotic (elementary) trapping and (fully) absorbing set enumerators for multi-edge type (MET) low-density parity-check (LDPC) code ensembles are derived. An efficient method for the evaluation of the asymptotic distributions is presented and evaluated.
I. INTRODUCTION
The performance of LDPC codes under iterative decoding can be degraded because of the presence of trapping sets [1] and (fully) absorbing sets [2] in the code Tanner graphs. The enumeration of these graphical configurations is important to study the error floor performance of LDPC codes.
Previous works focused on the enumeration of (elementary) trapping and (fully) absorbing sets of unstructured and protograph-based LDPC code ensembles. The asymptotic trapping set (TS) and absorbing set (AS) enumerators for unstructured LDPC code ensembles were derived in [2]–[5]. The asymptotic distributions of TSs and ASs for protographbased LDPC ensembles were developed in [6]–[10]. MET LDPC were introduced by Richardson and Urbanke. They represent a generalization of unstructured and photographbased LDPC codes. The weight distribution of MET LDPC code ensembles was derived in [11]. In this work, we extend the analysis of [4], [5] to MET LDPC code ensembles. We derive the finite-length and asymptotic distributions of (elementary) trapping and (fully) absorbing sets. The derivation relies on the generating function approach. We present an efficient way to compute the asymptotic distributions of the (elementary) trapping and (fully) absorbing sets. For the case where all the edges are of the same type we reproduce the enumerators of unstructured LDPC code ensembles. We obtain the enumerators of protograph-based LDPC code ensembles if we consider each variable node (VN)/check node (CN)/edge in the protograph as a VN/CN/edge type.
The paper is organized as follows. In Section II, we review the main definitions and known results. In Section III, IV and V we derive the finite-length and asymptotic (elementary) trapping and (fully) absorbing sets enumerators for MET LDPC code ensembles. Numerical results are presented and discussed in Section VI. Conclusions follow in Section VII.

Definition 1. An (a, b) TS Ta,b is a set I of a VNs such that U(I) contains b CNs [3].
Definition 2. An (a, b) AS Aa,b is a trapping set with the additional property that each VN v ∈ I has strictly fewer neighboring CNs from U(I) than from S(I) [2].
Definition 3. An (a, b) fully absorbing set (FAS) Fa,b is a trapping set with the property that each VN v ∈ V has strictly fewer neighboring CNs from U(I) than from S(I) [2].
Moreover, an elementary trapping set (ETS)/elementary absorbing set (EAS)/elementary fully absorbing set (EFAS) is a TS/AS/FAS where each CN c ∈ S(I) is either not connected to any or to two VNs in I and each CN c ∈ U(I) is connected to exactly one VN in I.

VI. NUMERICAL RESULTS
Consider the MET LDPC code ensemble (shown in Fig. 1) with multi-edge degree polynomials ν(z1, z2) = 0.4z13z2 + 0.6z12z22, µ(z1, z2) = 0.1z14z24 + 0.4z15z23. We evaluate the expressions of the normalized logarithmic asymptotic distribution of (elementary) trapping and (fully) absorbing sets derived in Sec. III, IV, V for a fixed ratio ∆ = b/a = β/α ∈ {0.005, 0.05, 0.1, 0.2, 0.3, 0.4}. The results are given in Fig. 2 and Fig. 3. We observe that the gap between the normalized logarithmic asymptotic distributions of TSs, ASs, and FASs vanishes for small values of α. The gap increases with increasing α and ∆. The same holds for the distributions of the elementary sets.

VII. CONCLUSION
We derived the finite-length and asymptotic distributions of (elementary) trapping and (fully) absorbing sets for MET LDPC code ensembles. We presented an efficient method to evaluate the asymptotic enumerators, which requires solving a system of equations.
