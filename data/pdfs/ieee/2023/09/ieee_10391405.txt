Abstract—The alternating direction method of multipliers (ADMM) has proven to be a highly efficient method for implementing linear programming (LP) decoding for low-density parity-check (LDPC) codes. The projection on the parity polytope is the most time-intensive operation involved in the ADMMassisted LP decoding. The existing projection methods involve sorting operations or complicated iterations which are difficult to perform by hardware systems. In this paper, an efficient parity-polytope projection algorithm is proposed by mapping the projection problem onto an isomorphic problem of probability simplex. The projection on probability simplex is obtained by utilizing its recurrent structure. This algorithm does not require sorting operations. Each iteration fixes at least one component of the input vector and has straightforward iteration steps. The simulation results show a noticeable reduction in the processing time for obtaining the projection of a given vector. Due to these advantages, it is beneficial for the effective implementation of ADMM-assisted LP decoding systems.
Keywords—low-density parity-check (LDPC) codes, linear programming (LP) decoding, alternating direction method of multipliers (ADMM), polytope projection
I. INTRODUCTION
L OW-Density Parity-Check (LDPC) codes were initially developed in the 1960s by Gallager [1]. At that time, these codes did not receive much consideration. After MacKay rediscovered these codes in the 1990s [2], a period of profound study on these codes and associated decoding methods began [11] - [14]. Due to their performance being close to the Shannon limit, these codes are used in high-speed communication systems.
A technique named Linear-Programming (LP) decoding was devised to address the decoding of LDPC codes in which the Maximum-Likelihood (ML) problem was converted to a linear program [3]. This decoding method provides a guarantee that if the output is a codeword, it must be the ML codeword. But the complexity of LP decoding poses a significant limitation to its further development. In a recent development, an LP decoding algorithm is put forward that relies on the Alternating

Direction Method of Multipliers (ADMM) [5]. The complexity of this formulation is nearly linear. But, the projection onto the check-polytope greatly affects the computational complexity of the ADMM-assisted LP algorithm, making it a crucial factor in the overall performance.
Numerous approaches for optimizing the projection operations have been put forth [6]- [10]. In [7], the authors forged an innovative approach that requires projection on the probability simplex and determined it using the simplex method. This made the ADMM-LP decoding compatible with the hardware systems. Further in [9], an approximate method is proposed for obtaining these polytope projections. This method requires significantly fewer calculations but provides imprecise results. An iterative method provided by [10] ensures the fixation of at least one component in every step with the reduction of up to 37% of arithmetic operations. However, these algorithms involve the sorting of vectors or iterations that amplifies the complexity of the system.
In this paper, a new polytope projection algorithm is developed that utilizes the recurrent behavior of probability simplex. This algorithm does not require the ordering of vector coordinates and has more simplified iteration steps. Every iteration can fix at least one coordinate of the input vector. Our approach demonstrates a significant improvement in computational efficiency compared to existing projection algorithms. Specifically, this method showcases up to a 22% reduction in the execution time compared to the state-of-theart method.

V. RESULT
In order to inspect the functionality of the proposed projection method, we compared it with the algorithms given in [8] and [10], which are rival algorithms to date having superior performance in the existing literature. It should be noted that the algorithm in [8] provides approximate projection whose precision magnitude relies on a predefined constant ϵ. We set ϵ = 0.001. Fig. 2-3 compares the execution time of the proposed algorithm for different values of d. Each value is derived by averaging the results obtained from a hundred thousand input vectors. The simulation is done with the help of MATLAB R2013a on a personal computer having a 3.4 GHz Intel i5 processor.
As could be noted the proposed algorithm requires less computation time. To explain this, we should take into account the following three points:
• Ordering of vector coordinates is not required for our approach.
• Unlike [8], at least one component of the input vector is fixed in each iteration. This reduces the size of the vector after every iteration.
• Unlike [10], the iteration steps are not dependent on vector θ. Mapping of input vector onto the isomorphic problem using the function Tθ maps the vector θ on zero vector. This effectively decreases the number of comparison operations, multiplications, and divisions from each iteration.
Hence, considering the above facts, it can be said that on the cost of mapping the input vector onto the isomorphic problem, the iteration steps became very simplified and the significant attenuation of computational time is definite. This less complicated projection algorithm can be helpful in the hardware implementation of ADMM-assisted LP decoding.

VI. CONCLUSION
In conclusion, a new parity-polytope projection algorithm based on the projection onto the probability simplex is developed which results in a substantial reduction in decoding complexity. It allows for faster execution times due to the uncomplicated iteration steps and absence of sorts. For future direction, advancements in this algorithm can be made to make it compatible with the non-binary LDPC codes by considering extended parity polytopes.
