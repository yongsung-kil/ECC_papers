Abstract—This paper proposes and verifies a resourceefficient 5-bit non-uniform quantization LDPC code for LLR processing. This low power consumption and low latency LDPC decoding method is promising for future high-speed PON.
Keywords—FPGA, high-speed PON, 5-bit Non-Uniform Quantization, MSA
I. INTRODUCTION
Driven by large bandwidth services such as 4k/8k video, virtual and augmented reality, and industrial Internet, passive optical network (PON) is embracing a new round of data traffic acceleration [1]. The evolution of PON must be compatible with existing optical distribution network (ODN), which puts forward a specific requirement on the PR30 (29dB) link budget of 50G-PON. However, increasing the speed by 50Gb/s per wavelength while maintaining a sufficient power budget is a huge challenge [2]. To meet the power budget demands of 50G-PON, various technologies such as digital signal processing(DSP) [3], optical amplification [4] and soft-decision forward error correction (SD-FEC) are adopted. Among them, low-density paritycheck (LDPC) code can generate higher coding gain to provide the sensitivity of the receiver. A soft-decision lowdensity parity-check (SD-LDPC) method is proposed in [5], which achieves a 1.3dB higher total coding gain than the IEEE 802.3ca 25G-EPON standard. In order to improve the

performance of burst error correction, a rearranging parity check matrix is proposed [6]. The measurement gain of FPGA is over 0.2dB, which can be used for 50G-PON upstream transmission. It has also been reported that a throughput of 90Gb/s has been achieved using 75 FPGAs, with SD-LDPC codes providing gains of more than 1.5dB in 50G-PON [7].
Although LDPC-based SD-FEC can provide high coding gain, it is the most resource-intensive module in the receiver, bringing in high power consumption and latency. This hinders the vision of simultaneous realization of Enhanced Mobile Broadband (eMBB) and ultra-reliable low latency communication (URLLC) in the future access network. Therefore, the resources and processing delay required for LDPC decoding are expected to be reduced.
In this paper, an LDPC decoding method with fewer quantized bits and fewer iterations is proposed. Irregular low density check matrix H (1984,384) is used for decoding, and the LDPC codes of rate is 7/8. FPGA implementation results show that with 5-bit non-uniform quantization, only 5 iterations are needed to achieve better decoding performance.

B. Experiment setup
The overall FPGA implementation procedure is shown in Fig. 2 (a). It is mainly composed of pseudo-random binary sequence (PRBS) generation module, LDPC coding module, Gaussian channel module, 5-bit non-uniform quantization module, LDPC decoding module and error calculation module. The size of generation matrix G used for LDPC coding is (1984,1659). Subsequently, the PRBS module generates 1000 frames of information bit data. After LPDC coding, the information bits are binary phase shift keying (BPSK) modulated, in which each frame contains 1659 bits of information and 325 bits of check information.
The min-sum algorithm (MSA) based on sum-product algorithm (SPA) is adopted in the decoder. the parity node updates only require comparison and addition operations, which can greatly reduce the decoding complexity with trivial difference in performance. Fig. 2(b) shows the FPGA architecture of the 5-bit non-uniform quantized LDPC decoder. In the decoding module, the check node unit (CNU) and variable node unit (VNU) are used to update the information of the check nodes and variable nodes, respectively. The C-V RAM unit stores information that is updated from the check node to the variable node. The V-C RAM unit stores information updated from the variable node to the check node. When the variable nodes update and check nodes update complete a round, the number of iterations is increased by 1. The iteration control module is used to control the CNU and VNU. When the number of iterations reaches the set value, the decoding information is output. Fig. 2(c) shows the FPGA device used in this paper.

A 5-bit quantization approach is adopted in the decoder for less resource occupation and power consumption. Fig. 1(a) shows the distribution of the data at the receiving end. Fig. 1(b) compares three different quantization methods and shows how each one works. The first is the non-uniform quantization method adopted in this paper, The quantization process is as follows: the maximum and minimum received values are first calculated. The interval from 0 to the maximum received value is evenly divided into 15 intervals and each interval is quantified from 1 to 15 in turn. Similarly, the negative axis is quantified from -1 to -15 in turn. There is no quantization value 0. The second quantization method is uniform quantization, which does not contain 0. The third quantization method is uniform quantization, but the quantization value of this method includes 0.

IV. CONCLUSION
We have demonstrated the transmission of 1000 frames of data on an FPGA platform based on 5-bit non-uniformly quantized LDPC code, where a 5-bit nonuniform quantization scheme is employed to reduce the size of memories storing soft messages. Additionally, only 5 iterations of decoding can achieve good performance. The results show that the proposed scheme has the characteristics of low power consumption and low latency, which makes it more suitable for high-speed PON with link budget limitation in the future.
