Receiver in Unitary-Transform Generalized Linear Systems

AbstractвҖ”In wireless communications, the unitary-transform generalized linear system (GLS) has been widely used to evaluate the impact of nonlinear preprocessing on transceivers. Generalized approximate message passing (GAMP) is a state-of-the-art low-complexity signal recovery algorithm, but it is only applicable to independent and identically distributed (IID) Gaussian matrices. To overcome this limitation, generalized orthogonal AMP (GOAMP) has been developed for unitarily invariant matrices, however, its information-theoretic limit analysis is numerical and limited by the high-complexity linear minimum mean-squared error (LMMSE), which is difficult to be analyzed analytically. Meanwhile, it is unable to effectively utilize the properties of unitary matrices, rendering the information-theoretic analysis still complex. To address these issues, in this paper, we provide the achievable rate analysis and optimal coding principle for GOAMP in unitary-transform GLS with arbitrary input distributions, establishing its information-theoretic limit (i.e., maximum achievable rate). Specifically, the simplified variational state evaluation (VSE) of GOAMP are developed using the unitary matrix properties to analyze the achievable rate, and the optimal code principle is derived with goal of maximizing the achievable rate. In addition, it is rigorously proved that GOAMP outperforms GAMP in terms of asymptotic MSE with lower complexity. Furthermore, quantization is used as an example to demonstrate the maximum achievable rate and practical low-density parity-check (LDPC) code design for GOAMP. The optimized LDPC code can approach the threshold limit within 0.8 dB and overcome the decoding deterioration and even divergence of the existing state-of-the-art methods.
Index TermsвҖ”Generalized linear system, generalized approximate message passing, generalized orthogonal/vector AMP

I. INTRODUCTION
A unitary-transform generalized linear system (GLS) is a fundamental model widely used in wireless communication applications to solve the recovery problem of an unknown length-N signal x from a noisy observation signal

To support massive user access in unsourced random access [1], U is selected as a Hadamard matrix for the multi-user codebook at the transmitter. For massive multipleinput multiple-output (MIMO) systems (M, N вҶ’ вҲһ) [2], an independent and identically distributed (IID) Gaussian channel matrix can be regarded as a unitary matrix asymptotically. On the other hand, Q(В·) denotes the common nonlinear preprocessing at the transceiver, such as quantization in the analogto-digital conversion (ADC) of MIMO [3], and peak-clipping operations [4] in OFDM. As a result, the introduction of U and Q(В·) brings new challenges to the design and theoretical analysis of signal recovery algorithms in GLS.
Various generalized approximate message passing (AMP)type algorithms based on the Bayesian framework have been developed for GLS in recent years [5]вҖ“[8]. Generalized AMP (GAMP) is first proposed with low complexity for GLS [5]. Remarkably, the asymptotic mean-square error (MSE) of GAMP can be accurately predicted by a scalar state evolution (SE), which is shown to be Bayes optimal for IID Gaussian matrices by the replica method [9]. However, GAMP may be degraded or divergent for non-IID matrices. As a result, for unitary matrices, a generalized Turbo signal recovery algorithms is proposed [7], which is essentially a simplified generalized orthogonal AMP (GOAMP) [10]. Furthermore, the generalized vector AMP (GVAMP) is proposed to apply to general unitarily invariant matrices [6] and is proved to be Bayes optimal [11]. Due to the equivalence of GOAMP and GVAMP, this paper refers to both as GOAMP. Since GOAMP includes a high-complexity linear minimum meansquared error (LMMSE) detector, it cannot be effectively applied to large-scale GLS. Recently, a low-complexity generalized memory AMP (GMAMP) has been developed in [8] for unitarily invariant matrices, and its Bayes optimality is proven via SE. However, the results in [5]вҖ“[8] are limited to uncoded GLS, where error-free performance is not guaranteed.
For coded systems with error-free recovery, the maximum achievable rate is commonly used to characterize the information-theoretical limit of an algorithm in coded systems. Recently, the achievable rate analysis of GOAMP is first proposed for GLS with unitarily invariant matrices and arbitrary

Due to the complex dual-input dualoutput SE of GOAMP, a simplified single-input single-output (SISO) variational SE (VSE) can be obtained numerically via a sufficiently inner iteration between the linear detection (LD) and the nonlinear detection (NLD), from which the achievable rate can be calculated. However, because the GOAMP is based on LMMSE, it is unable to fully leverage the special structure of unitary matrices, resulting in a more complex achievable rate analysis. Meanwhile, since the VSE analysis is based on Monte Carlo simulation [10], it is difficult to compare other generalized AMP-type algorithms analytically.

In this paper, we present the optimal coding principle and information-theoretical (i.e., achievable rate) analysis of GOAMP in the coded unitary-transform GLS with arbitrary input distributions. Specifically, by utilizing the properties of unitary matrices, the original SE of GOAMP can be analytically simplified to the SISO VSE, from which the achievable rate is derived. Based on the error-free decoding criterion of VSE, the optimal coding principle for GOAMP is established with aim of maximizing the achievable rate. In addition, without a specific form of Q(В·) in GLS, GOAMP under unitary matrices is rigorously proven to outperform GAMP with IID Gaussian matrices in terms of asymptotic MSE. Furthermore, we study the maximum achievable rate of GOAMP in coded GLS and practical LDPC code design using quantization as an example. The main contributions are summarized as follows:

1) The optimal coding principle and maximum achievable rate of GOAMP are obtained by leveraging the properties of unitary matrices.
2) GOAMP with unitary matrices is rigorously proved to be superior to GAMP with IID Gaussian matrices in terms of asymptotic MSE with lower complexity, without the specific form of Q(В·).
3) Using quantization as an example, the achievable rate analysis and practical optimized LDPC code for GOAMP are provided. Numerical results show that the optimized LDPC code can approach the threshold limit within 0.8 dB and overcome the decoding deterioration and even divergence of the existing state-of-the-art methods.

D. Complexity Comparison
From (16a) and (24a), it is noted that the ПҲt(В·) and ПҶt(В·) in (10a) are the same for GAMP and GOAMP. Although GOAMP includes orthogonalizations, the corresponding incremental complexity is ignorable. For GAMP [16], the implementation of LD requires matrix multiplications up to an order of O(M N ). In contrast, GOAMP can exploit the properties of unitary matrices using fast computational techniques such as the fast Fourier transform (FFT) or the Walsh-Hadamard transform with order of O(N log2N ) complexity, while only a small number of elements of the matrices need to be stored.

Note that the optimal code principles for GOAMP in Lemma 5 are applicable for arbitrary input distributions. Fig. 5(c) shows the gaps between BER curves at 10вҲ’5 of the optimized LDPC codes and the corresponding theoretical limits are about within 0.8 dB, which verifies the near optimality of the proposed LDPC codes. Moreover, compared with pointto-point (P2P) regular (3, 6) LDPC codes and well-designed irregular LDPC codes with RLDPC = 0.5 [17], the optimized LDPC codes with GOAMP can achieve about 3.4 вҲј 3.7 dB performance gains. Surprisingly, the optimized LDPC codes with GOAMP under partial DFT matrix can overcome the decoding divergence of the P2P LDPC codes with GAMP under IID Gaussian matrix.
V. CONCLUSION
This paper focuses on the coding principle and informationtheoretic limit of GOAMP in a coded unitary-transform GLS. By virtue of the unitary matrix property, the simplified VSE of GOAMP is developed for the analysis of achievable rate and optimal code design. In addition, it is rigorously proved that GOAMP outperforms GAMP in terms of asymptotic MSE with lower complexity. Numerical results demonstrate the achievable rate advantages of the proposed GOAMP and the BER performance gains of the optimized LDPC code compared to the existing state-of-the-art methods.
