High Performance Error Correction under Low SNR

based on Deep Neural Network

Abstract—The signal-to-noise ratio of long-distance relay free communication systems and quantum secure communication systems is very low, making it difficult to achieve highperformance error correction. In order to achieve effective error correction, longer code words or higher iterations are generally chosen to achieve the highest error correction efficiency as possible. However, this will greatly increase the complexity of the error correction process and reduce real-time performance. To solve this problem, we propose a high-performance error correction algorithm based on deep neural network. In the case of low signalto-noise ratio, the error correction performance of this method is better than that of the classical LLR-BP algorithm for high-speed short codes. At the same time, the number of hidden layers and neurons were adjusted according to the coding length, and the generalization performance of the model was improved. The results show that our proposed DNN based decoding algorithm can achieve better error correction performance than LLR-BP when the signal-to-noise ratio is less than 9dB.
Keywords-Error Correction; LDPC Codes; Deep Neural Network.
I. INTRODUCTION
Communication can achieve effective information transmission between distant parties. Modern communication is mainly based on digital communication, errors often occur during information transmission due to various interferences such as noise. Therefore, error correction plays a very important role in modern communication [1-2]. However, for weak signal systems such as long-distance relay free communication systems and quantum secure communication systems, facing with poor error correction performance. For long-distance relay free communication systems, due to the lack of relay, the signal attenuation is large, resulting in low signal-to-noise ratio and high bit error rate, which affects the reliability of communication. For quantum secure communication systems [3], due to the extremely weak signal itself and the inability to amplify the signal from a security perspective, the signal-tonoise ratio is also extremely low and the error rate is very high. Only high-performance error correction can ensure the normal operation of the system, otherwise effective information cannot be obtained. Therefore, in actual system applications, channel

error correction codes with error correction performance close to Shannon limit, error correction algorithms with high computational complexity, long error correction codes and high decoding iterations are more inclined to be used, which will seriously affect the high-speed real-time performance of the system.
LDPC code [4-5] (Low Density Parity Check code), is a linear block code, which has low implementation complexity, and good performance in error correction, so it has been widely concerned and applied. Although a variety of simplified algorithms for BP (Belief Propagation) decoding algorithm of LDPC codes have been proposed [6-8], including the LLR-BP (Log Likelihood Ratio-Belief Propagation) one converting multiplications into additions and the further simplification version — the Min-Sum decoding algorithm, the performance of these mentioned above deteriorates to a certain extent with the loss of the computational complexity, since the simplified fitting process will sacrifice the accuracy of probability information [9-11].
Therefore, based on the above considerations, we came up with the idea of combining machine learning with the error correction. The hidden layer of neural network can simulate the iterative process of decoding error correction. For the trained network, it can achieve a certain decoding success rate in the case of a given number of iterations (namely a certain number of layers of network), which can avoid the waste of computing resources caused by redundant iterations.
In this paper, we have implemented a high-performance error correction algorithm based on multi-layer neural network by simulating the experimental data required for error correction in actual low signal-to-noise ratio communication systems using professional communication simulation software. Each layer in the model presents an iteration in which check nodes and variable nodes pass and update messages. Additionally, by adjusting the number of hidden layers and neurons according to the code length, the generalization performance of the model can be optimized. This method can to some extent improve the performance of decoding under some certain situations.

II. LDPC CODE
LDPC code can achieve good error correction performance at low SNR. LDPC code is a linear block code with asymptotic performance close to Shannon limit in AWGN channel [12]. Besides, the low density arising from its sparse check matrix [13], that is, the number of non-zero elements is far less than that of the zero elements. Based on the above properties, it has low decoding complexity as well as good decoding performance compared to other error corrected codes.

For a full-rank LDPC matrix with code length N and information bit K, the code rate R satisfies R = K / N. The composition of the code word can be expressed in the form of bilateral Tanner graph or check matrix [14-16], and the two methods are in one-to-one correspondence. Let the check matrix be H for instance, then the dimension of the matrix is expressed as M × N, where M = N − K presents the number of check bits.

The process of BP decoding of LDPC codes can also be further divided into probability BP algorithm and LLR-BP algorithm according to the form of initialization of messages, of which the latter one transforms a lot of multiplication operations into addition operations, reducing the calculation complexity. Additionally, another method with lower computing complexity called Min-Sum decoding algorithm is a variant of LLR-BP at a further sacrifice of accuracy but introduces modified parameters and is suitable for the neural networks.

IV. SIMULATION RESULTS
We established a fully-connected deep neural network decoding model (DNN model) for short codes. The length of the data used in this paper is 212 and they are divided into training sets and test sets in a proportion of 7:3. The output results of the test sets and the BERs (Bit Error Rate) are calculated and compared with that of the classical LLR-BP. All the data used in this paper are generated by a professional simulation software, which can simulate actual low SNR communication systems.
We established two code length codewords to simulate and make comparison. When the length is 10, the hidden layer number is set to 6, and the learning rate is 0.008, the simulation results are shown in Fig. 2.
It can be seen that DNN model performs better in the case of SNRs lower than 7.5dB under relatively high code rate (R=0.6) situations (i.e., high Gaussian noise variance), especially when SNR is 0dB, the BER of DNN model is close to 1E-2, which is one order of magnitude lower than that of the LLR-BP one. The BER curves of both the algorithms decline stably in these SNR ranges. However, when the SNR continues to rise, the BER of LLR-BP algorithm decreases rapidly and is better than DNN algorithm.
Next, the number of hidden layers in the deep neural network is tuned and the results are obtained seeing Fig 3.

V. CONCLUSION In conclusion, for short code with high rate (usually equal to or greater than 0.5), the volume of the check matrix is small and the logic at the check node is relatively simple, contributing to short loops in check matrix and thus poor performance under low SNRs for LLR-BP algorithm. However, DNN model learns the complete code word book does not strictly depend on the verification logic of the code word, so it can perform better than LLR-BP in those SNR cases. At the same time, by adjusting the number of hidden layers and neurons according to the code

length, the generalization performance of the model can be optimized. The results show that our proposed deep neural network based decoding algorithm can achieve better error correction performance than LLR-BP when the signal-to-noise ratio is less than 9dB.
