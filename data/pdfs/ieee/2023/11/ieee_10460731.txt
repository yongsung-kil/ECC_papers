Abstract— In this paper, we propose a new code design 
technique for GLDPC codes which we call partial doping. 
The proposed partial doping technique enables higher 
degrees of freedom in constructing protograph-based 
GLDPC codes. We optimize the partially-doped generalized 
LDPC codes over the binary erasure channels and the finite 
length analysis shows that it outperforms the regular LDPC 
codes for both code rates 1/2 and 1/4. 
Keywords—Binary erasure channel (BEC), Low-density 
parity-check (LDPC) codes, generalized LDPC (GLDPC) 
codes, doping, protograph LDPC codes, typical minimum 
distance, regular LDPC codes. 
I. INTRODUCTION  
Low-density parity-check (LDPC) codes have been 
studied thoroughly until recently due to their capacity-
approaching performances from a reasonably low 
decoding complexity. Furthermore, LDPC codes have 
been selected as the main codes for the data channel for the 
5G new radio (5G NR) standards [1]. While the need for 
the LDPC codes to expand for their effectiveness, the 
study of LDPC codes can be expanded to generalized 
LDPC (GLDPC) codes, where they can be constructed by 
replacing some of the parity checks in the LDPC codes 
with any arbitrary linear codes including Hamming codes, 
Hadamard codes, Reed-Solomon codes, and etc [2]-[4]. 
Since GLDPC codes are the generalized version of the 
LDPC codes, many studies have been made to find 
GLDPC codes with superior decoding performances [5], 
[6].   
While several construction methods of GLDPC codes 
exist, protograph-based GLDPC codes are known to have 
many advantages due to their hardware-friendly structures 
and good asymptotic performances. Protograph-based 
LDPC codes are constructed from a structure, known as a 
protograph, which becomes the building block of the code 
ensemble. One of the well-known construction methods of 
protograph-based GLDPC codes involves the process of 
doping, where a subset of parity check nodes in the 
protograph is replaced by linear codes [7].  
In this paper, we try to apply a new method called 
partial doping over regular LDPC codes. We call it the 
partially-doped GLDPC (PD-GLDPC) codes. It is a well-
known fact that the overuse of degree-2 variable nodes in 
LDPC codes causes error floor phenomenon. One of the 
main reasons for this is the condition for the typical 
minimum distance is not necessarily satisfied for regular 
LDPC codes with degree-2 variable nodes. Thus, we 
applied the proposed partial doping method on these 
LDPC codes to enhance the performance of the underlying 
LDPC codes while at the same time satisfying the typical 
minimum distance condition. Numerical results show that 
the proposed partially-doped GLDPC codes constructed 
from degree-2 regular LDPC codes have better 
performance than that of the (3,6) regular LDPC codes.  
The paper is organized as follows. In Section II, basic 
notations and preliminaries needed for this paper are given. 
In Section III, we provide the proposed construction 
scheme and optimization algorithm. Section IV shows 
both the numerical results and some conclusion remarks 
for the paper.

B. Optimized results 
For this paper, we use the same notation as in [11] in 
order to define the ensemble of the constructed PD-
GLDPC codes. Let a code ensemble (𝐁𝑛𝑐×𝑛𝑣, 𝜇. 𝜅, Χ) , 
where 𝐁𝑛𝑐×𝑛𝑣 is the base matrix with size 𝑛𝑐× 𝑛𝑣 that is 
doped by (𝜇, 𝜅)  linear code. We assume that the 
protograph is doped from the left part of the base matrix, 
where Χ is the number of the doped variable nodes.  
For numerical analysis, we use a (15, 11) systematic 
Hamming code as the component code. For a half-rate 
code, we construct ( 𝐵{200−4𝑦}×400, 15,11,15𝑦) PD-
GLDPC codes, for an integer 𝑦. Likewise, for a quarter-
rate code, we construct a (𝐵{300−4𝑦}×400, 15,11,15𝑦) PD-
GLDPC code ensemble. The code ensembles for both code 
rates were optimized for 𝑦 in terms of the asymptotic 
performances which were held out by the PEXIT analysis. 
The optimal value of 𝑦 were 19 with a threshold of 0.444 
and 10 with a threshold of 0.6935 for half-rate and quarter-
rate, respectively. For all optimized cases, the protographs 
were constructed such that there were no cycles of degree-
2 variable nodes in order to avoid the typical minimum 
distance conditions.  
For finite length analysis, for the code rate 0.5, we use 
(3,6) regular LDPC codes with protograph having a base 
matrix with size 200×400  and lift it by 75, which has a 
threshold of 0.429. Also, we construct an irregular LDPC 
code that has the same degree distribution as the 
constructed GLDPC code. The degree distribution of the 
GLDPC codes considers both the degree in the LDPC code 
part and the degree in the Hamming code part. Both the 
GLDPC codes and LDPC codes were lifted to construct 
(30000,15000) codes. The block error rate (BLER) results 
are shown in Fig. 1. The coding gain of the constructed 
protograph-based PD-GLDPC code is 0.021 and 0.015 
compared to irregular protograph-based LDPC code and 
(3,6) regular protograph-based LDPC code, respectively.

IV. CONCLUSION 
In this paper, we proposed a new design technique for 
GLDPC codes over regular codes. The newly proposed 
partial doping technique over a protograph and the 
corresponding optimization technique made it possible to 
construct the proposed protograph-based PD-GLDPC 
codes. The numerical results show that although the 
underlying base matrix is (2,4) regular LDPC codes, by 
using the optimized partial doping technique, the 
constructed PD-GLDPC codes outperform the (3,6) LDPC 
codes and the LDPC codes with the same degree 
distribution. Extension of the study of PD-GLDPC codes 
to Gaussian noise and other channels is an important topic, 
however, due to the hardness of computing the density 
evolution in a closed form, we leave the topic for the future 
research topic.
