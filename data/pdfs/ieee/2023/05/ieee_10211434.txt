AbstractтАФThis paper proposes a low complexity algorithm for calculating the soft decision metric of forward error correction codes. In order to reduce the complexity of LLR calculation, neural network training can be used to fit the relationship between input and output during exact LLR calculation. The trained network can skip the complex training process and directly estimate the LLR value. We use random signal-to-noise ratio (SNR) and received signals as the input layer of the neural network, and use the precisely calculated soft decision information as the output layer of the network. After training, the network can accurately estimate the LLR of the received symbols. The performance of 16QAM and 64QAM transmission system using the proposed algorithm and the exact LLR calculation algorithm in the AWGN channel with polar code as the encoding algorithm are presented, respectively. The simulation results show that the proposed algorithm has a similar bit error rate performance to traditional exact calculation algorithms, but with much lower complexity. The proposed algorithm can be applied in various decoding situations that require LLR calculation.
KeywordsтАФNeural network, Soft-decision decoding, log likelihood ratio
I. INTRODUCTION
With the development of communication technology, decoders with powerful decoding performance continue to be proposed [1]. Error correction codes, as a common coding method in channel coding, have been successfully applied in a large number of communication systems [2]. Soft decision, as a technique that utilizes digital technology to achieve optimal or near optimal decoding of error correction codes, is commonly used in forward error correction (FEC) codes. Log likelihood ratio (LLR) information, as the initial input to the soft decision decoder, is one of the important factors affecting the performance of bit errors. The exact LLR can be calculated through log-Map algorithm. However, the complexity of logMAP algorithm is too high to be applied in higher-order modulation. To solve this problem, a simplified algorithm named max-log-MAP was proposed. In [3], for SISO bit interleaved OFDM systems with 16QAM and 64QAM constellations, a set of approximate bit metrics for each bit LLR

Deep learning technology has attracted the attention of many researchers due to its advantage in recognition, classification, and data fitting. The combination of neural networks and forward error correction codes has always been a focus of researchers. In [6], a convolutional neural network was used to estimate channel noise, which was preceded by a BP decoder, improving the overall decoding efficiency.[7] proposed a minimum sum decoding algorithm using Linear approximation networks, and achieved good decoding results on quasi-cyclic low-density parity-check codes under the 5G standard. Based on neural network, [8] investigated a rate-compatible punctured polar code decoding scheme which outperforms the traditional punctured Polar code decoder under high SNR.
To solve the problem of LLR computation complexity of log-MAP and estimation error of max-log-MAP in soft decision decoding algorithm, an intelligent LLR estimation algorithm based on neural network is proposed. The LLR is calculated using the strong fitting ability of the neural network, and the performance is verified using polar code. Compared with the exact calculation algorithm of LLR, the algorithm proposed in this paper effectively reduces the computational complexity.
The rest of this paper is organized as follows. A brief overview of the principles of commonly used algorithms for calculating LLR, and introduction of our proposed intelligent LLR estimation algorithm based on neural networks are provided in Section II. In Section III, we introduce the training algorithm of neural networks and analyze the test results of the network. In Section IV, we establish a communication simulation system based on polar codes and verified the performance of the intelligent LLR estimation algorithm in this system. Finally, Section V offers concluding remarks.

In the following part, the specific calculation of log-MAP algorithm and max-logMAP algorithm will be compared, and the intelligent LLR estimation algorithm will be introduced.

C. The Intelligent LLR Estimation Algorithm
As can be seen from the previous section, the approximate algorithm of the simplified max-log-MAP algorithm is to use linear function of different intervals to approximate the LLR value. However, the calculation of LLR exact value is a nonlinear function, and there will be some errors when fitting through linear function. This simplified scheme also has certain requirements for mapping rules. Therefore, this section proposes a algorithm of fitting LLR values using neural networks to learn the calculation rules for exact LLR calculations.

Figure 2 shows the intelligent LLR estimation network designed when using 16QAM and 64QAM, where each layer consists of a fully connected structure. The input layer consists of two neurons, and the input information is the signal-to-noise ratio of the received signal transmitted through the channel and the real or imaginary part of the received signal constellation. For 16QAM, the hidden layer consists of two layers of networks, with network sizes of 4 and 4, respectively. For 64QAM, the hidden layer consists of three layers of networks, with network sizes of 4, 16 and 8. The output layer is composed of two neurons, which output soft decision information of corresponding bits. It is worth noting that the number of neurons in the output layer is equal to the order of the modulation format. In other words, the output layer consists of two neurons when using 16QAM, three neurons when using 64QAM and four neurons when using 256QAM. The loss function selects the mean square error function MSE, and the epoch size is 50.
Through the analysis of this section, for MQAM modulation, the complexity of LLR calculation using intelligent LLR estimation algorithm is proportional to log2 M. As (2) knows, the complexity of calculating LLR using the log-MAP algorithm is proportional to M. In other words, compared to the log-MAP algorithm, the more advanced the modulation format, the more significant the complexity optimization effect of the intelligent LLR estimation algorithm.
III. NEURAL NETWORK TRAINING AND TESTING RESULTS
This section will introduce the accuracy of intelligent LLR estimation algorithm calculates through the training and testing results of neural networks. The network uses a binary random signal generator to generate raw bits, and the source uses a Gray mapped 16QAM signal (This algorithm is not limited to Gray mapping or 16QAM modulation format.) as input data. At SNRs of 1-15dB, 80,000 sets of data were used for network fitting training, and 20,000 sets of data were used to test the degree of network fitting. A back propagation neural network is used as the model. Select the logsig function as the hidden layer node activation function and the purelin function as the output layer node activation function. The training function uses the trainlm function, the weight learning function selects the learngdm function, and the learning rate is set to 0.1.

To verify the validity of the proposed intelligent LLR estimation network, a polar code transmission system is built. When the modulation format is 16QAM, the polar code length is 1024 and the code rate is 0.5. When the modulation format is 64QAM, the polar code length is 1536 and the code rate is 0.33. The specific system block diagram is shown in Figure 5. In the system, the log-MAP algorithm and the intelligent LLR estimation algorithm are used to decode the soft decision output of the channel. B. Performance Analysis
The comparison of bit error rates between different LLR calculation algorithms in 16QAM transmission system is shown in Figure 6. By comparing the bit error rate in polar code decoding tasks, it can be seen that when the SNR is below 2dB, there is no significant difference in the performance of the two LLR calculation algorithms due to the high probability of error bits appearing in each frame. However, as the SNR increases, there are some differences in the LLR values calculated using intelligent LLR networks compared to the log-MAP algorithm. At first, the intelligent LLR estimation algorithm would generate a small amount of loss, but as the channel environment further improves, under high SNR conditions, it gradually approaches the performance of the exact algorithm, achieving similar results to the exact algorithm. Through simulation results, it can be seen that the intelligent LLR estimation algorithm based on neural networks can achieve performance similar to the log-MAP algorithm at the low computational complexity cost of a 2-layer network, effectively reducing the complexity of the decoding algorithm.

V. CONCLUSION
In this paper, a low complexity algorithm for calculating the soft decision metric of forward error correction codes is proposed. This algorithm provides a simple intelligent LLR estimation algorithm and can be used in various soft decision decoding environments. The error bit rate performance of 16QAM and 64QAM transmission system using the proposed algorithm and the exact LLR calculation algorithm in the AWGN channel with polar code as the encoding method are presented, respectively. The results indicate that for the 16QAM system and 64QAM system, the intelligent LLR
