Improving LDPC-Coded PNC Decoders via Informed Dynamic Scheduling
Caizheng Huang , Pingping Chen , Senior Member, IEEE, Zhaopeng Xie , and Yi Fang , Senior Member, IEEE

Abstract— To decode the transmit message pairs and then obtain the exclusive OR (XOR) message, this letter proposes the double-reliability-based decoding (DRB) algorithms with informed dynamic scheduling (IDS) for low-density parity-check (LDPC) coded physical-layer network coding (PNC) systems. The proposed DRB decoding with informed stability (IS-DRB) utilizes a new stability measure to determine the update order of the check nodes. Moreover, by exploiting the principle of PNC decoding, we present an improved IS-DRB (IIS-DRB) decoding to select the check node with unsatisfied PNC mapped checksum and the large stability to update first. This is due to that message propagated from this check node can quickly correct the variable node that causes unsatisfied checksum. Simulation results show that the proposed algorithms can realize significant performance improvement of about 0.5 dB over the conventional DRB decoding with the limited increase in complexity.
Index Terms— Physical-layer network coding, low-density parity-check codes, informed dynamic scheduling.
I. INTRODUCTION
P HYSICAL-LAYER network coding (PNC) can remarkably enhance the throughput of wireless networks [1]. The two-way relay channel (TWRC) is a basic channel model, where a relay helps two users exchange messages. The relay aims to decode the superimposed signal from two user messages into network coding (NC) signal. The simple form of the NC signal is the result of the bitwise XOR operation of two user messages [1], [2], [3]. The low-density parity-check (LDPC) codes [4] have been extensively studied in PNC systems to ensure communication reliability of PNC systems [5], [6]. Two main decoding schemes are available for LDPCcoded PNC, i.e., joint channel decoding and PNC decoding (JCNC) and XOR-based channel decoding (XOR-CD) [7], [8]. The belief propagation (BP)-based JCNC performs better than XOR-CD but with higher computation complexity [5], [7], because the transmit message pair of two users are iteratively updated.

To reduce the decoding complexity, an iterative soft reliability-based majority-logic decoding algorithm (ISRBMLGD) was proposed with only real arithmetics and Galois Field (GF) addition [9]. The changes of variable node stability in decoding are utilized to improve decoding performance [10]. In addition, an informed dynamic scheduling(IDS) strategy was further applied in non-binary LDPC codes [11]. For LDPC-coded PNC, [12] unitized a log-likelihood-ratiobased decoder for the JCNC with the reduced complexity. Reference [13] developed a noncoherent modulation and coding for PNC. Reference [14] utilized extended min-sum (EMS) decoding algorithm for coded-PNC by simplifying the check node update. However, most existing work for coded-PNC still depends on soft information iteration decoding with high computational complexity. Reference [15] proposed a low-complexity improved-ISRB (IM-ISRB) decoding over the PNC-aided TWRC. These algorithms exploit the superposition property of the received signal in PNC systems with low computational complexity, but degrade the performance.
In LDPC-coded PNC systems, we first decode the message pair of two users and then map them into XOR message, which can be seen as nonbinary-like decoding. To improve decoding performance while maintaining low complexity, we propose two algorithms with IDS strategy, i.e., double-reliability-based decoding with informed stability (IS-DRB) and improved ISDRB (IIS-DRB) by exploiting the PNC characteristic. The IS-DRB introduces stability vector to determine the update order of check nodes, while IIS-IDRB prefers to update those check nodes first with unsatisfied PNC mapped checksum. The intuition is that unsatisfied check node with the large stability suggests more information required for convergence, i.e., propagating the message to variable nodes can quickly correct the variable node that causes unsatisfied checksum. Simulation results show that the proposed algorithms can obtain significant performance gains over the conventional DRB and IM-ISRB with the limited complexity burden. In particular, the IIS-DRB shows a narrow performance gap within 1 dB as compared to the conventional BP-JCNC at code rate of 3/4.
The remainder of the letter is organized as follows. Section II briefly reviews the system model of PNC. Section III proposes the decoding algorithms for PNC systems. Section IV shows the complexity analysis and simulation results. Section V concludes the letter.

V. CONCLUSION
In this letter, we propose the IS-DRB and IIS-DRB decoding algorithms with IDS strategy for LDPC-coded PNC. The decoding performance of the PNC systems can be improved at the expense of increased complexity to select the messages to be propagated. In particular, we utilize the stability measures of the variable nodes and check nodes to determine the update schedule. Moreover, by judging the PNC mapped checksum, IIS-DRB is allowed to organize the message schedule according the stability value as well as the checksum result. Simulation results show that the proposed IIS-DRB can achieve significant performance gains over the conventional DRB and the IM-ISRB decoding.

Fig. 7. The average iteration number for decoding convergence.
Figs. 3 and 4, we can see that the proposed IIS-DRB performs worse than DRB at low SNR region, while it outperforms the latter one at a BER of 10−5 at high SNR region. Figs 5 further show the decoding performance of the irregular LDPC code at high code rate of 0.7. We see that the performance gap between the proposed algorithm and the BP-JCNC [7] is smaller, within 1 dB, as the code rate increases. As Fig. 6 shows, the proposed algorithm still outperforms the conventional IM-ISRB [15] by 1 dB in power-balanced channels, and suffers a loss of 1.2 dB as compared to BP-JCNC [7].
To analyse convergence speed, Fig. 7 shows the average iteration number to achieve successful decoding. The proposed IIS-DRB has a smaller number of iterations than the other schemes, since the unsatisfied check nodes with the large stability are quickly updated. It is useful for communication applications with low-latency decoding requirement. This result is consistent with the IDS strategy for binary LDPC decoding, which shows appealing convergence advantage over the standard decoders.
