Abstract—An adaptive LDPC puncturing method suitable for 
L-ACO-SCFDM is proposed. The simulation results show that
it can obtain 6~7dB sensitivity gain compared with conventional
L-ACO-SCFDM and have potential application in short-
distance IM/DD optical transmission system.
Keywords:Adaptive LDPC puncturing; L-ACO-SCFDM; IM/DD 
I.
INTRODUCTION 
With the emergence of new big data services such 
as cloud 
storage,downloads,and live 
broadcasting on 
the Internet, the worth of optical fiber communication in 
the information network has become increasingly significant. 
However,during the signal transmission process,optical fiber 
communication systems are subject to various factors such as 
noise, nonlinearity,and polarization mode dispersion. To 
overcome these interferences, it is necessary to adopt channel 
coding and modulation 
techniques with 
better error 
correction performance to improve these issues. 
The Asymmetrically Clipped Optical-Single Carrier 
Frequency 
Division 
Multiplexing 
(ACO-SCFDM) 
techniques in the  Intensity Modulation/Direct Detection 
(IM/DD) optical transmission system is a modulation method 
that uses OFDM. This technology uses IM/DD to modulate 
and detect optical signals[1]. By using ACO technology to 
clip the optical signal and Discrete Fourier Transform (DFT) 
technology, the OFDM signal is transformed into an ACO-
SCFDM signal, which further reduces the Peak-to-Average 
Power Ratio (PAPR) of the transmission[2], thereby reducing 
the nonlinear 
distortion of 
the 
signal 
in fiber 
optic 
transmission, improving the system's reliability and anti-
interference capabilities, and making the signal meet the 
requirement of positive real numbers so that it can be applied 
to IM/DD systems[3]. In 2000, Chin-Liang Wang proposed 
using Discrete Hartley Transform (DHT) instead of the 
original DFT to reduce the complexity of the OFDM 
system[4]. Mengqi Guo and Ji Zhou proposed the use 
of layered technology in ACO-SCFDM to compensate for the 
low spectral efficiency and also combined with DHT to 
reduce the complexity of the system[5]. 
In a layered ACO-SCFDM system that uses IM/DD, the 
optical signal is susceptible to various interferences during 
transmission, such as fiber dispersion, attenuation, and optical 
noise. Error correction through FEC technology, such as 
LDPC and Turbo codes, is necessary to improve the system's 
anti-error performance[6]. The use of segmented LDPC 
puncturing technology can selectively discard parity bits 
based on channel conditions and data transmission 
requirements, leading to a shorter codeword length and 
improved transmission efficiency. 
Therefore, this paper proposes an adaptive LDPC 
segmented puncturing algorithm suitable for the layered 
asymmetric clipping optical single-carrier frequency division 
multiplexing system. Compared with traditional ACO-
SCFDM technology, the layered asymmetric clipping optical 
single-carrier frequency division multiplexing technology 
with adaptive LDPC segmented puncturing can select the 
appropriate puncturing quantity according to the current 
channel 
state, 
thereby 
balancing 
its transmission 
rate and error correction performance under the premise of 
high spectral utilization. The error performance of this 
puncturing method is simulated and compared with ordinary 
segmented puncturing.

LDPC segmented puncturing is a technique for 
optimizing LDPC codes that can improve the code rate while 
maintaining high error correction performance. It divides the 
LDPC code into multiple segments, each using a different 
parity-check matrix to match the error correction capability 
of different segments. This technique can achieve a balance 
between high code rate and good error correction 
performance in high-speed data transmission. Taking the 
LDPC code in IEEE 802.16e as an example, the traditional 
segmented puncturing principle is shown in Figure 2.

LDPC segmented puncturing in IEEE 802.16e uses 
matrices with different code rates as master codes for 
puncturing to achieve flexible rate control. However, 
increasing the number of punctured bits reduces error 
correction performance. Additionally, LDPC codes obtained 
by puncturing low-rate master codes to achieve high-rate 
codes have weaker error correction performance than the 
corresponding high-rate original master codes, as shown in 
Figure 3. 
Figure 3. Comparison of the bit error performance of the high-bitrate LDPC 
obtained by punching with the original corresponding bitrate LDPC 
In Figure 3, C* represents the high-rate LDPC codeword 
generated by puncturing the low-rate LDPC codeword, while 
C represents the LDPC codeword generated by the 
corresponding rate base matrix. The codeword generated by 
the base matrix performs better in terms of error performance 
compared to the codeword obtained by puncturing the low-
rate codeword with the same rate.

is twice the length of the processed data in the lower layer, 
we use channel evaluation to flexibly adjust the code rates R1 
and R2 of the two layers to control the average code rate by 
puncturing, as shown in Figure 4.

III.
SIMULATION RESULTS AND DISCUSSION
To evaluate the system's performance, a 2PAM 
modulation format was used, and a MATLAB simulation was 
conducted to calculate the bit error rate (BER). The proposed 
layered adaptive LDPC puncturing method was compared 
with the conventional segmented LDPC puncturing method 
under 
AWGN. The simulation results showed an 
improvement of approximately 0.3 to 0.6dB. 
Compared to not using LDPC coding, the proposed 
method showed a minimum improvement of 6.2 dB,as shown 
in Figure 5.The LDPC code used in the simulation had a 
length of 1920 and was based on the IEEE 802.16e standard, 
and a soft-decision decoding algorithm was employed with a 
maximum of 30 iterations. 
Figure 5. Performance comparison of layered adaptive LDPC puncturing 
method and the segmented LDPC puncturing method in AWGN 
A co-simulation was conducted using the same LDPC 
code in both MATLAB and optical simulation software VPI. 
The 
proposed layered 
adaptive 
LDPC 
puncturing 
method was compared with the conventional segmented 
LDPC puncturing method in a fiber optic channel. The 
simulation results showed that the proposed method still 
achieved an improvement of approximately 0.5dB compared 
to the conventional method, and a minimum improvement of 
7dB compared to conventional ACO-SCFDM with potential 
used in short-distance IM/DD optical transmission system.as 
shown in Figure 6.

IV.
CONCLUSION
In this paper an adaptive LDPC puncturing method 
suitable for L-ACO-SCFDM is proposed. This method is 
more suitable for layered structures and can dynamically 
adjust the code rate based on the channel environment, 
resulting in a more efficient and reliable transmission. 
Simulation results that it can obtain 6~7dB and 0.5~0.6dB 
gain respectively compared with conventional L-ACO-
SCFDM and non-layered LDPC puncturing with potential 
used in short-distance IM/DD optical transmission system
