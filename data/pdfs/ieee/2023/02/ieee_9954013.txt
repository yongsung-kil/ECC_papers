A Low-Complexity Ordered Statistic Decoding of Short Block Codes
Jifan Liang , Yiwen Wang , Suihua Cai , and Xiao Ma

Abstract— This letter is concerned with a generalized ordered statistic decoding (OSD) algorithm, called locally constrained OSD (LC-OSD). Instead of order-t reprocessing on the most reliable independent bits, the LC-OSD searches for test error patterns using the serial list Viterbi algorithm (SLVA) over a trellis specified by a local parity-check matrix. We derive several early stopping criteria that can be used to reduce the number of searches. Numerical results show that the LC-OSD algorithm with the proposed stopping criteria has much lower time complexity than the original OSD but incurs negligible performance loss.
Index Terms— Early stopping criteria, ordered statistic decoding (OSD) algorithm, serial list viterbi algorithm (SLVA).
I. INTRODUCTION
T HE ordered statistic decoding (OSD) algorithm was proposed by Fossorier and Lin in [1], which has attracted much attention as the ultra-reliable and low latency communication (URLLC) became an important application scenario in 5G [2]. For a binary linear block code of dimension k and minimum Hamming distance dmin, the original OSD algorithm can approach the maximum-likelihood (ML) decoding performance if order-t reprocessing is implemented with t = dmin/4 − 1 but has a time complexity of O(kt). So the OSD is more promising for short block codes and many efforts have been paid to reduce the complexity [3], [4], [5], [6], [7], [8]. The main ideas of the existing low complexity OSD algorithms include skipping and/or discarding (early stopping) unlikely test patterns to avoid unnecessary calculations.
In this letter, we consider the locally constrained OSD (LC-OSD) algorithm proposed in [9]. In contrast to the original OSD algorithm, which performs order-t reprocessing of the k most reliable independent (MRI) bits, the LC-OSD algorithm reprocesses (k + δ) most reliable but dependent bits of the codeword and then calculates the rest of the coded bits, where δ ≥ 0 is a parameter of the algorithm. In particular, the reprocessing can be implemented by performing the serial list Viterbi algorithm (SLVA) [10] over the locally constrained trellis for the (k + δ) bits. With this setup, we derive and propose several criteria that are used to stop the serial search as early as possible for complexity reduction. The LC-OSD behaves similarly for all simulations, confirming its universality. Also notice that the time complexity of LC-OSD using different stopping criteria decreases with the increase of SNR, as expected.

V. CONCLUSION
We have proposed several early stopping criteria for the locally constrained OSD (LC-OSD) algorithm, by which the computational complexity can be reduced significantly but the performance loss can be negligible when compared with the original OSD algorithm. As a generalization of the original OSD algorithm, the LC-OSD algorithm, which is developed based on the parity-check matrices, does not rely on the structure of code and is applicable to any short block codes. This has been confirmed by simulations and will be discussed in more detail in future work.
