Low Error Floor QC-LDPC Codes Construction Using Modified Cole’s Trapping Sets Enumerating
Vasiliy S. Usatyuk Research and Development Department
T8 Moscow, Russia Email:L@Lcrypto.com

Abstract—In order to count the number of trapping sets (TS), we changed Cole’s Importance Sampling (IS) approach, which led to a message-passing decoder problem. Several concepts have been combined to suggest improvements to Cole’s IS: utilizing short cycles and simultaneous TS impulse tree decomposition, unwrapping of message passing iterations Tanner Graph/Forney’s Normal Graph symmetry’s Graph Authomorphism is a complex yet simple concept. Superior Velasquez-Subramani and KarimiBanihashemi TS enumerating techniques were supported. The proposed technique under PEG (1008, 504) Mackay code for single thread implementation is 43 times quicker than the original Cole’s method and 5027 times (71463 times, multi-thread) faster than the Velasquez-Subramani LP method. For the TS enumerating problem at (2640, 1320), Margulis’s code is 37958 times quicker than the suggested Velasquez-Subramani LP approach for single thread implementation, 82 times faster than KarimiBanihashemi, and 134 times faster than Cole’s original solution. On the basis of the creation of QC-LDPC codes, we demonstrate the effects of improved EMD spectrum and increased hamming distance on the TS spectrum and BER/FER error-floor level.
Index Terms—Extrinsic Message Degree; EMD; Importance Sampling; QC-LDPC; Trapping Sets; Pseudo-codeword;
I. INTRODUCTION
For reliable data storage and transmission over various noisy transmission and storage channels, low-density parity-check (LDPC) codes have been used extensively in optical, wireless, satellite, and quantum communication and storage systems.
As a result of cycles present in the LDPC codes Tannergraph, relaxation of the estimation error caused by messagepassing (Belief propagation) decoder results in an additional error (complexity reduction penalty) compared to the maximum likelihood method. Tanner graphs with linear size cycles and their intersections with linear size Trapping Sets (asymmetrical subgraphs) incur a penalty in the waterfall region. Citing [1], the Covariance evolution approach is used to determine this penalty. ML decoding imposes an errorfloor cost on sublinear size Trapping Sets. Error-floor penalty estimates using Schlegel-Zhang linear model ( [2] for BER, bit error rate), generalized (non-integer Hamming weight) union bound ( [3], [4] for BLER/FER, frame error rate), and require enumeration of sublinear Trapping sets and weighing them to get sublinear part of generalized weigh spectrum, (non-integer hamming distance due Trapping sets pseudocodewords) using Importance Sampling techniques. According

to [5], importance sampling (IS) is a traditional statistical (ML) approach for estimating the likelihood of unusual events.
The general method for trapping sets enumeration and weighting using Monte-Carlo simulation was initially presented in paper [3]. The connection between the pseudocodewords of the Trapping sets, which correspond to the finite unramified covers of the graph, and the fundamental cone in Rn defined by inequalities arising from the paritycheck matrix H using Feldman linear programming decoding, was demonstrated in paper [6]. The search for pseudocodewords can be carried out using linear programming or IS methods. At papers [7]–[9] Cole proposed and applied for LDPC code construction IS enumeration and weighting method. It was one the most well described with application for well know codes and efficient method for TS enumerating. At papers [10]–[13] Stepanov-Chertkov and Vasic proposed several approaches for trapping set enumeration and weighted for error-floor estimation and found phenomenon of messagepassing error due quantization noise. At paper [14] was given full ontology of Trapping sets for column weight 3 LDPC codes, unfortunately not feasible for codes of greater 3 weight. However, the approach to traversing Tanner-graph cycles in order as the size of the trapping set grows used in many effective Trapping set enumeration methods, [15], [16] including our. At papers [17]–[23] was done LP, IS enumeration of trapping sets and it elimination for LDPC code construction. To compare the efficiency of Trapping sets enumerations, we choose Velazquez-Subramani LP and Cole’s, Karimi-Banihashemi, Abu-Surra-Declercq-Divsalar-Ryan IS methods which paper description contain information about the performance for reference PEG(1008, 504) Mackay, Algebraic (2640, 1320) Margulis LDPC codes was given, ( [4], [15], [17], [20]).
In this paper, a modification of the Cole method is proposed, its efficiency is compared with the methods of Vela´zquezSubramani [20] and the original Cole method [4]. Using the example of constructing an QC-LDPC codes, an lowering error-floor of the LDPC code is demonstrated by increasing the code distance, the EMD spectrum, and it relation to Trapping sets spectrum using the proposed modification of the Cole’s IS method.

Let’s have a look at the matrix H Tanner graph’s sub-graph, which is created by the cycles or overlaps of those cycles. The trapping set T S(a, b) of such a subgraph consists of a variable nodes and b odd degree checks. The TS(5,3) generated by the overlap of three 8-cycles and the TS(4,4) formed by cycle 8 in the Tanner graph are shown in Fig. 3. The risk associated with a trapping set depends on the quantity of variable nodes that, in the event of errors, could result in decoding failure.
When there are 3 errors in odd degree check nodes, the TS(5,3) is more hazardous than the TS(4,4), which produces only 4 errors in variable nodes if there are 4 errors in odd degree check nodes and could be interpreted as a weight 4 pseudocodeword. The weight spectrum of a pseudocodeword might be improved, and the likelihood of an error-floor could be reduced, by breaking the worst cycles. Its risk is closely tied to the decoder and may be altered by changing the decoder’s parameters, [26], [27]. The minimum codeword that determines the code distance dmin of the LDPC code corresponds to the T S(a, 0), a = dmin, [14]. TS spectrum are a generalization of the weight spectrum of the code in the case of soft iterative decoding by the message-passing decoders. By improving the EMD (ACE), we improve the weight Spectrum of the LDPC code. Clear that TS spectrum enumeration problem much more complex than weight spectrum enumeration problem. The paper showed the NP complexity of enumerating TS in LDPC codes [28]. The number of check nodes singly connected to the variable nodes included in the cycle is the Extrinsic Message Degree (EMD) of the cycle in the Tanner graph, [29]. Such code will not be interfered with by TS-conditioned pseudo-code words. For example, TS(5,3) has a higher probability of decoding failure than TS(5,5), and the EMD Spectrum of TS(5,3) will contain 3 cycles of length eight with value 14, TS(5,5) will contain 3 cycles of length eight with value 16. An increase in the value of the EMD spectrum will lead to a decrease in the probability of TS pseudocodewords decoding failure.
We eliminate equal-probability trapping sets by building codes with larger minimal EMD values. It indicates that TS(a, b) harm depends simply on the value harm = b/a, subgraph error probability, and TS multiplicity. With this limitation, this purpose is sufficient to create codes free of pseudocodewords. Unfortunately, the decoder’s parameters—such as the scale and offset values, the scale for message quantization, the bitwise comparison of the input and message bits, the restriction of error consideration to information bits, puncturing, shortening, and others—determine the likelihood of TS errors ( [27], [12]). We can adjust the weight of pseudo-codewords by varying these variables. For this reason, the valuable TS damage model calls for estimating the weight of pseudocodewords via importance sampling and applying union-bound to forecast code behavior in the error-floor regime. We employ Cole’s first TS weighing procedures. This weighed technique was greatly generalized by paper [31] using the local convexity property (topology approach from quantum field theory) of the star domain for various channel types (AWGN, BEC, BSC, and etc.) and sizes of TS (sublinear to code length for ”error-floor” and linear for waterfall, Covariance Evolution).

IV. CONCLUSION
Due to the unwrapping of the iteration tree and usage of graph automorphism, the suggested Cole’s IS technique modification for enumerating of Trapping Sets greatly surpasses earlier published approaches in terms of TS enumeration speed and parallel processing capabilities. The creation of five QCLDPC codes as an example illustrates how simulated annealing lifting with code candidates’ improved EMD spectrum and elimination of hazardous TS lowers the error-floor level.
