Abstract—The belief propagation (BP) algorithm for low density parity check (LDPC) codes has proven to have excellent performance and is close to the Shannon limit. However, the high computational complexity is not conducive to realistic use. Although the min sum algorithm (MSA) improves the computational convenience and applicability. It also weakens its decoding capability. In order to improve the decoding performance of MSA and exploit the potential, this paper propose an adaptive exponential correction factor. This can improve the decoding performance of MSA. Adaptive exponential min sum algorithm (AEMSA) demonstrate a noticeable improvement over the normalized min sum algorithm (NMSA) and the BP algorithm in high signal to noise ratio (SNR). AEMSA is also a simple, efficient, low power consuming and highly potential decoding algorithm that can be applied in hardware implementations.
Index Terms—LDPC Codes, Decoding Algorithms, Min Sum, Adaptive, Exponential, Internet of Things (IoTs)
I. INTRODUCTION
Robert Gallag’s Low Density Parity Check (LDPC) codes are recognized as one of the most resilient and versatile forward error correction schemes [1], with decoding performance approaching the Shannon limit [2]. LDPC codes are identified as the data channel coding scheme for 5G eMBB scenarios [3], and have been widely used in WIFI, WIMAX and DVB-S2, etc [4].
The classical decoding algorithm for LDPC codes is the belief propagation (BP) decoding algorithm, which is the preferred algorithm when the LDPC codes are optimally designed and there are no cycles in the matched Tanner graph [1]. However, for finite length LDPC codes, cycles are inevitable, and the BP algorithm has high computational complexity despite its excellent decoding performance. To cope with this problem, the multiplication operation in the BP algorithm can be replaced by an additive operation by adding a hyperbolic tangent function, which is the log likelihood ratio belief propagation (LLR BP) algorithm. And then Fossorier made an approximation to the check node update operation of the LLR BP algorithm to obtain the min sum algorithm (MSA) [5]. MSA has greatly reduced the complexity of the operation, but also reduced the decoding ability, so on the basis of MSA proposed offset min sum algorithm (OMSA) and normalized min sum algorithm (NMSA) [6]. The decoding performance of OMSA and NMSA has some improvement compared with

MSA, but the correction factors of NMSA and OMSA are fixed values determined by simulation, which have limited correction ability and cannot effectively solve the problem of overestimation of the information value of the check nodes of MSA under arbitrary circumstances, so it is difficult to reach the decoding level of LLR BP algorithm.
In order to explore the potential of MSA algorithm, an adaptive exponential min sum algorithm (AEMSA) is proposed in this paper, and the exponential error correction factor is derived through theory, and the computational complexity of AEMSA is slightly increased. By simulating the regular LDPC codes with code rate of 0.5 and code lengths of 204, 816 and 2018, it is proved that the decoding performance of AEMSA is stable and has better decoding ability than LLR BP algorithm. AEMSA is also a more efficient and energy saving decoding solution for future internet of things (IoTs) devices.

The implementation of soft-decision algorithm is much more complex than harddecision algorithm, but the decoding performance of softdecision algorithmis is shown to attain remarkable capacityapproaching error rate performance [8]. At current, soft decision algorithm is the mainstream algorithm of LDPC codes decoding algorithm.

A binary LDPC code can be represented by an M × N sparse check matrix H and the code rate of the LDPC code is R = (N − M )/N . For any check matrix H, there is a Tanner graph corresponding to it [9]. BP algorithm is an iterative decoding algorithm based on the Tanner graph. The main principle of the BP algorithm is that information is continuously sent and exchange along the edges of the Tanner graph. The check nodes information and variable nodes information are updated in turns, with each turn updating represents one iteration. During the iterative process, when the information of the check nodes is updated, the variable nodes connected sends messages to it,
which are updated by the operation and sent back to the original variable nodes. Similarly, when the variable node information is updated, the same operation is performed, then the collected information is aggregated to perform the decoding decision operation and output the decoding result.
During the BP algorithm decoding process, the information generated by the check node updating and variable node updating is called external information, while the initial information received from the channel at the beginning of the decoder operation is called posterior probability. The BP algorithm can be simply divided into 4 steps. The meanings of the symbols of the formulas are listed in Table 1.

III. MIN SUM ALGORITHM
LLR BP algorithm greatly reduces the number of multiplication operations. However, the process of calculating the check nodes information contain a large number of tanh functions, which can be implemented by the lookup table according to the nature of hyperbolic tangent function, which brings the problem of occupying a large amount of logic resources while obtaining a relatively high decoding accuracy. In order to solve this problem, a simplified LLR BP algorithm was developed, which is called the min sum algorithm (MSA) [11].
The MSA is an approximation of LLR BP algorithm, which further optimizes the check nodes update formula, which is the step 2 of LLR BP algorithm. MSA uses the auxiliary functions to transform the complex tanh function and tanh−1 function into basic symbolic value calculation and numerical comparison operation, so MSA has new check nodes update formula [5], that is, Equation 6 can be evolved into Equation 9. MSA is similar to the calculation framework of LLR BP algorithm, except that the update formula of check nodes is more simplified, and then the calculation can be performed sequentially according to the framework of LLR BP algorithm.

The NMSA and the OMSA, although processed in different ways, both have the core idea of compensating for the problem of overestimation of the check nodes information values. However, both NMSA and OMSA have their own drawbacks [12]. For NMSA, the normalization factor α can effectively modify the computational equation of the check nodes and improve the decoding performance, but the decoding process is completely determined by the min value of the edge information. And when the output is almost zero, the decoding performance will suffer a serious degradation. As for the OMSA, the offset coefficient β is constant in the decoding process and does not change adaptively. Therefore, the OMSA can only achieve a limited improvement on the MSA. In fact, NMSA and OMSA can hardly achieve the decoding performance of BP algorithm.

IV. ADAPTIVE EXPONENTIAL MIN SUM ALGORITHM
Although the decoding performance of MSA is inferior to that of BP algorithm, MSA has great potential for improvement. The performance of the MSA can be greatly improved by adding an adaptive exponential factor to the MSA. The exponential factor can increase the nonlinear property of MSA and make the check node formula of MSA approach or even surpass the BP algorithm to some extent.
Analysis of the check nodes information update formula in the LLR BP algorithm shows that it is a multiplication of several (0, 1) values, and the final result can be smaller and smaller with the multiplication operation. The information value of the check node mainly depends on the product of the minimum xmin1 and the second minimum xmin2 value of the edges information. When the difference between the two dif f = xmin2 − xmin1 become smaller and smaller, the product will be closer and closer to x2min1. In addition, a small number of tanh(x) functions can be introduced to increase the nonlinearity of the MSA so that the MSA can better approximate the check nodes update formula of the LLR BP algorithm. V. LDPC CODES AND AEMSA IN IOTS
The Internet of Things (IoTs) will be one of the major trends of the future and billions of devices will be connected to the Internet as shown in Fig 5 [13] [14]. Future IoTs devices require lower energy consumption, lower delay, and higher data transfer rates. In this aspect, LDPC codes are one of the most promising codes among many error correcting codes (ECC) and are preferred as the next generation of IoTs networks [15]. Compared to other ECC, such as Reed Solomon (RS) codes, turbo codes, and Polar codes, LDPC codes have many advantages, such as very low error floor, good industrial support conditions, flexible code structures, diverse encoder and decoder algorithms [16] [17].

and easier to implement in hardware, its decoding performance is also weakened, and NMSA and OMSA can only improve the decoding performance of MSA to a limited extent, making it difficult to reach the decoding level of BP algorithm.
The AEMSA proposed in this paper surpasses the BP algorithm in decoding performance at low and medium code lengths simply by calculating the minimum and second minimum values from the variable nodes to the check nodes. Although AEMSA introduces exponential computation and tanh function, it is still a low power, high efficiency decoding algorithm. So AEMSA is an extremely potential algorithm for IoT devices.
VI. CONCLUSION
This paper presents the decoding principles of BP algorithm, LLR BP algorithm, and MSA algorithm, respectively. Throughout this study, an overestimation problem of the MSA algorithm along with the AEMSA algorithm for LDPC codes is proposed. The proposed algorithms are used the min1 and min2 based on absolute values of the input nodes information and introduces tanh function for non-linearity of the algorithms. It is noteworthy that the proposed AEMSA mainly adds tanh function exponential operation as compared to MSA for better decoding performance. The simulation results show that AEMSA has better decoding performances than LLR BP algorithm in LDPC codes with low and medium code length. For further development of IoTs approach, this study summarizes the advantageous deployment of LDPC codes and potentials of AEMSA towards expansion the wireless communication trade.
