QUASI-CYCLIC LDPC CODES BASED ON D AND Q MATRICES THROUGH PROGRESSIVE EDGE GROWTH

Wei Zhan, Guangxi Zhu, Li Peng and Xi Yan

Wuhan National Laboratory for Optoelectronics, Huazhong University of Science and Technology, Wuhan, China

ABSTRACT
We propose a general method for constructing regular and irregular quasi-cyclic LDPC codes based on the modified PEG (progressive edge growth) graph. The idea of permutation vector is introduced. Through permutation vectors, the QC-LDPC codes based on arbitrary circulant permutation matrices, which are not limited to identity matrix, Q-matrix or D-matrix, can be generated with flexible parameters such as block length, code rate, degree distribution. Simulation results demonstrate that QC-LDPC codes using the new PEG algorithm show an error correction performance comparable to the random LDPC codes based on the traditional PEG algorithm. In addition, the PEG QC-LDPC codes have more hardware-friendly parity check matrices than the random PEG LDPC codes and can fulfill linear-time encoding. The QC-LDPC codes based on D-matrix and Q-matrix are more suitable to be used in QC-LDPC codes than identity matrix because they outperform that of identity matrix by 0.1-0.2 dB in the new PEG algorithm.
Index Terms— low-density parity-check (LDPC) codes, quasi-cyclic (QC) codes, progressive edge growth (PEG)
1. INTRODUCTION
Low-density parity-check (LDPC) codes first discovered by Gallager [1] have a remarkable performance with iterative decoding that is very close to the Shannon limit over additive white Gaussian noise (AWGN) channels [2]. Most methods for designing good LDPC codes are based on random construction, but algebraically structured LDPC codes may be needed for implementation purposes. Random-like LDPC codes [3] are constructed by computer search, based on certain design guidelines and required structural properties of their Tanner graphs [4], such as girth, degree distributions of variable and check nodes, and stopping sets. Structured LDPC codes are constructed based on algebraic and combinatorial tools, such as are given in [5], [6]. In the case of random LDPC codes of large code

length, a significant amount of memory is needed to store their parity-check matrices. Also, it is hard to access the memory and encode data efficiently.
Quasi-cyclic LDPC (QC-LDPC) codes may be a good candidate to solve the memory problem due to their algebraic structures. The required memory for storing QCLDPC codes can be reduced by a factor 1/L, when L × L circulant permutation matrices or the zero matrix are employed. Recently, several coding theorists have proposed some classes of QC-LDPC codes based on circulant permutation matrices and analyzed their properties [7][8]. These codes perform quite well compared to random LDPC codes.
In this paper we propose a general method for constructing regular and irregular quasi-cyclic LDPC codes based on the modified PEG algorithm. These codes generated by this method perform well and have linear-timeencoding complexities.

3. CONSTRUCTING QUASI-CYCLIC LDPC CODES THROUGH PEG GRAPH

A LDPC code is defined as the null space of a sparse paritycheck matrix. Construction of good LDPC codes at short and intermediate block lengths is of great practical importance. Among the existing methods one of the most successful approaches for the construction of finite-length LDPC codes is the PEG algorithm proposed by Hu et al [11]. PEG construction builds up a Tanner graph (TG), or equivalently a parity-check matrix, for an LDPC code on an edge-by-edge basis and by maximizing the local girth at variable nodes in a greedy fashion. It is simple and also flexible in that it can be applied to construct codes of

arbitrary length and rate. In addition, PEG algorithm can be used to construct linear-time encodable LDPC codes which also perform very well.

In the standard PEG algorithm, given the graph parameters, i.e., the number of symbol nodes, the number of check nodes, and the symbol-node-degree sequence, an edge-selection procedure is started such that the placement of a new edge on the graph has as small an impact on the girth as possible. Here two new parameters, i.e., the dimension of a circulant permutation matrix and permutation vector are introduced. As for the traditional PEG graph, edges are added to the Tanner graph node by node. And it is necessary to expand trees for every symbol node. Here the parity check matrix of a QC-LDPC code should be generated by adding the permutation vector constraint to the PEG algorithm. The basic idea is to divide all symbol nodes and check nodes into groups with each group of n nodes. Therefore edges are added to the PEG graph group by group instead of node by node. It is only required to expand trees for the first symbol node in every symbol node group to find its optimized edges. The edges of other symbol nodes in the same group should be automatically determined by the permutation vector constraint. The generation of QC-LDPC codes is sped up by a factor n in this way. Here we take a Q-matrix for example. In a certain selection step, the third check node of the Xth check group is selected to connect the first symbol node of some symbol group as the PEG algorithm, that is to say, the first edge is established. 5. CODE PERFORMANCE

In this section, we study the performance of PEG Tanner graphs applied to binary QC-LDPC codes by means of computer simulations. For comparison purposes, we use the rate-1/2 (N = 1536, K = 768) code of MacKay’s random LDPC, which is based on a regular Tanner graph with ds = 6, dc = 12, and there is no four cycles in the Tanner graph. All the PEG LDPC codes are nearly regular. That is to say, a PEG Tanner graph of 1536 symbol and 768 check nodes is generated with uniform degree 6 for each symbol node. The resulting graph is nearly check-node uniform with degree 12, except for some nodes with a degree of 11 or 13. PEG Random LDPC code is of size (N = 1536, K = 768). 6. CONCLUSIONS
In this paper, a general method for constructing regular and irregular quasi-cyclic LDPC codes based on progressive edge growth graph has been presented. The idea of permutation vector is originally introduced. The main principle of this method is to optimize the placement of a new edge to maximize the local girth length under the permutation vector constraint. Through permutation vectors, the QC-LDPC codes based on arbitrary circulant permutation matrices, which are not limited to identity matrix, Q-matrix or D-matrix, can be generated with flexible parameters such as block length, code rate, degree distribution. Simulation results demonstrate that QC-LDPC codes using the new PEG algorithm show a bit error rate performance comparable to the random LDPC codes based on the traditional PEG algorithm. Using the same algorithm, the QC-LDPC codes based on D-matrix and Q-matrix

outperform the QC-LDPC code based on an identity matrix and PEG random LDPC code, which suggests that D-matrix and Q-matrix are more suitable to be used in QC-LDPC codes than identity matrix. In a word, the proposed general PEG QC-LDPC codes, especially these based on D-matrix or Q-matrix, exhibit good performance and have hardwarefriendly structures for practical application.
