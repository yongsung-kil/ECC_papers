Analysis of Check-Node Merging Decoding for Punctured LDPC Codes
with Dual-Diagonal Parity Structure

Jung-Ae Kim, Sung-Rae Kim, and Dong-Joon Shin Division of Electronics and Computer Engineering Hanyang University Seoul, Korea Email: {punk99, ksr0812}@ccrl.hanyang.ac.kr, djshin@hanyang.ac.kr

Song-Nam Hong Standard Research Lab Telecommunication R&D Center
Samsung Electronics Co. Suwon, Korea
Email: sn7955.hong@samsung.com

Abstract— In this paper, we propose new decoding scheme of punctured LDPC codes with dual-diagonal parity structure by merging check nodes connected to the punctured parity nodes. This check-node merging decoding not only needs smaller number of operations at each iteration but also shows faster decoding convergence speed than the conventional erasure decoding. For the binary erasure channel (BEC) and AWGN channel, we analyze and compare the check-node merging and the conventional erasure decoding schemes of the punctured LDPC codes with dual-diagonal parity structure using density evolution. Analytical results show that the check-node merging decoding gives faster convergence speed for both BEC and AWGN channel. Also, simulation results are provided to confirm the analytical results.
I. INTRODUCTION
In 1996, low-density parity-check (LDPC) codes, originally invented by Gallager [1], were rediscovered by MacKay and Neal [2]. Since then, LDPC codes have been the main research topic because they show the capacity approaching performance with feasible decoding complexity [3]. However, one drawback of LDPC codes is that the encoding complexity is quadratic in the code length, which results in slow encoding. Therefore, efficient encoding algorithms have been proposed [4] and various fast-encodable codes were proposed [5], [6], [7]. Most of the fast-encodable LDPC codes contain the dualdiagonal parity structure, i.e., accumulator, which produces degree-2 parity nodes. We will call such codes as LDPC codes with dual-diagonal parity structure.
Recently, the optimal puncturing scheme of LDPC codes with dual-diagonal parity structure has been extensively studied [8], [9], [10]. Basically, punctured LDPC codes are decoded by giving 0 LLR values to the punctured nodes, which causes the slow decoding convergence speed. Therefore, the performance of punctured LPDC code is worse than that of unpunctured LDPC code with the same threshold when the iteration number is not sufficient (say, < 50). To solve this problem, an efficient check-node merging decoding algorithm for punctured LDPC codes with dual-diagonal parity structure is proposed where the Tanner graph including

punctured parity nodes is simplified by merging check nodes [11], [12]. This algorithm improves the convergence speed and reduces the decoding complexity.
In this paper, we analyze and compare the conventional erasure decoding and the check-node merging decoding for BEC and AWGN channel using density evolution. We verify that both decoding schemes have the same asymptotic performance and the check-node merging decoding shows the faster decoding convergence. Also, simulation results confirm the analytical results.
II. CHECK-NODE MERGING DECODING OF PUNCTURED LDPC CODES WITH DUAL-DIAGONAL PARITY STRUCTURE
For the LDPC codes with dual-diagonal parity structure, the check nodes connected to the punctured parity nodes can be merged as in Fig. 1 [11], [12]. Thus, it is possible for the punctured LDPC codes with dual-diagonal parity structure to be decoded using the sum-product algorithm on the Tanner graph given in Fig. 1 (b) instead of Fig. 1 (a), that is called the check-node merging decoding[11]. The number of edges connected to the parity nodes in Fig. 1 (b) is much smaller than that in Fig. 1 (a).

V. CONCLUSIONS
In this paper, we analyze a new decoding algorithm called check-node merging decoding, which not only needs smaller number of operations but also shows faster decoding convergence speed than the conventional erasure decoding. We analytically verified that both decoding schemes have the same asymptotic performance and investigate the reason why check-node merging decoding shows faster convergence speed than the conventional erasure decoding. This checknode merging decoding can be applied to any LDPC codes with dual-diagonal parity structure.
