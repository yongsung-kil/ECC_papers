Reduced-Complexity Decoding Algorithm for LDPC Codes for Practical Circuit Implementation in Optical Communications

Yoshikuni Miyata, Rui Sakai, Wataru Matsumoto, Hideo Yoshida, and Takashi Mizuochi
Information Technology R&D Center, Mitsubishi Electric Corporation, 5-1-1, Ofuna, Kamakura, Japan Miyata.Yoshikuni@ak.MitsubishiElectric.co.jp
Abstract: For superior, practical decoding of LDPC based FEC, a cyclic approximated delta-minimum algorithm is proposed. Simulation shows that 3-bit soft-decision decoding provides a required Q of 6.2 dB at 10-6 BER with only 16 iterations. ©2007 Optical Society of America
OCIS codes: (060.4510) Optical communication

1. Introduction

Optical transmission systems need increasingly powerful forward error correction (FEC) to ensure the maximum

accuracy of data transmission with the lowest optical power for minimal fiber nonlinearity effects, and to keep to a

minimum the number of repeaters required for a given optical fiber span. In recent years, several types of powerful

FEC have seen intensive development, e.g. the two orthogonally interleaved BCH codes having a net coding gain

(NCG) of 9.4 dB listed in ITU-T G.975.1, and Block Turbo Codes with 3-bit soft decision having an NCG of 10.1

dB [1]. All of these strong FECs operate at 10 Gb/s, and no FEC applicable to systems beyond the 40 Gb/s class and

showing comparable performance has yet been seen.

Low-Density Parity-Check (LDPC) codes are being studied intensively as potential candidates for optical

communications at 40 Gb/s or more. The LDPC code is a linear code, defined by the sparse parity check matrix

invented by Gallager. This technique was ignored for a long time, then was re-discovered by MacKay, leading to

further intensive research for wireless communication systems. Later, studies into applying LDPC to optical

communications were instituted by Vasic and Djordjevic [2]. The irregular LDPC(3367,2821) code with 19%

redundancy was studied [3], and with the iterative soft-decision min-sum algorithm yielded an NCG of 8.1 dB at a post-FEC BER of 10-9. This was improved on by the generalized LDPC(3639,3213) code with 23.6% redundancy, with which a record NCG of 10.9 dB at a post-FEC BER of 10-13 was demonstrated in a Monte-Carlo simulation [4].

The calculations showed that LDPC had the potential to be improved yet further, e.g. the projective geometry

PG(2,26)-LDPC(4161,3431) code with 21.2% redundancy, with which an NCG of 11.27 dB was obtained [5].

Although some simulations reported previously have shown high error correction capability, no experimental

demonstration has been carried out yet. One of the reasons is that it is difficult to implement the proposed algorithms

in LSI in a fully parallel way with a practical circuit size. For example, while an infinite number of soft-decision bits

would yield ideal correction capability, circuit implementation would be impossible. A near-infinite number of

iterations would yield superior error correction performance, but

2. Proposed Decoding Algorithm

Fig. 1 depicts the basic configuration of an LDPC FEC based optical transponder. A received signal is sampled by a k-bit soft-decider. With increasing bit-rate, the implementation of a soft-decider in LSI becomes seriously difficult in terms of limited set-up & hold time, reduced sensitivity, and increased power consumption. An ideal FEC decoder would need hundreds of decoding iterations and a huge circuit to process the numerous operations required each second. The result would be unacceptable latency and a circuit that was impossible to implement. It is very important to realize high error correction capability with however small a number of soft-decision bits.

3. Performance Simulation

The test code is the irregular Quasi-Cyclic LDPC code having code- word and information lengths of 8148 and 6984, respectively. The redundancy is 16.67%. The number of iterations was set to only 16 (including initialization), which is sufficiently small to hold down the latency.

4. Conclusions

We have proposed a novel decoding algorithm for LDPC codes in optical communication systems. The cyclic approximated δ-minimum algorithm has the potential to reduce the circuit complexity drastically without sacrificing error correction capability. The simulations showed an input Q of 6.2 dB could be corrected to a BER of 10-6 with
only 3-bit soft-decision and 16-times iteration. It is anticipated that the proposed algorithm for LDPC codes will
open up new vistas for higher speed, long distance fiber-optic transmission systems with adequate system margins.
