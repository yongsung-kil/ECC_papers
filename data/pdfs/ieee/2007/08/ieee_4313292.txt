Abstract‚ÄĒ Four new messages for decoding of low-density parity-check (LDPC) codes are proposed. An a posteriori probability ratio (APPR) message, an a posteriori probability (APP) message, and an a posteriori probability difference (APPD) message for variable and check nodes are derived for sumproduct decoding of LDPC codes. Also, we propose a hybrid APPD-APPR message, which denotes a hybrid message with an APPD variable node message and APPR check node message. All the proposed messages require only addition, subtraction, multiplication, and division operations for decoding of LDPC codes. We show that the hybrid APPD-APPR message-passing has the lowest complexity among the proposed messages.
I. INTRODUCTION
The low-density parity-check (LDPC) codes constitute a class of iteratively decoded error-control codes whose capacity-approaching performance and simple decoding have resulted in their inclusion in new communications standards such as DVB-S2, IEEE 802.11n, 802.3an, and 802.16 [1]. The LDPC codes are iteratively decoded using a messagepassing algorithm operating between variable and check nodes. The conventional message for binary LDPC codes is the logarithm a posteriori probability ratio (LAPPR), which is the same as the log-likelihood ratio (LLR) when 1 and ‚ąí1 are transmitted with equal probability 1/2 [1]. Calculation of a hyperbolic tangent function is required at check nodes for the conventional LAPPR message,
In this letter, we propose four new messages for decoding of LDPC codes based on different interpretations of the conventional LAPPR message. An a posteriori probability ratio (APPR) message, an a posteriori probability (APP) message, and an a posteriori probability difference (APPD) message for variable and check nodes are derived for sum-product decoding of LDPC codes. Since the proposed messages have the same information as the conventional LAPPR message in binary codes, the decoding using each proposed message is equivalent to the decoding using the conventional LAPPR message. Also, we propose the hybrid APPD-APPR message, which means that the APPD is the output message of the variable node and the APPR is the output message of the check node. None of the proposed messages require calculation of a hyperbolic tangent function, and can be calculated using only four elementary operations: addition, subtraction, multiplication, and division. We show that the hybrid APPD-APPR message has the lowest computational complexity among the proposed messages because the APPR and the APPD

message can be used efficiently at variable and check nodes, respectively.
This letter consists of six sections. In Section II, we describe interpretations of the conventional LAPPR message. In Section III, we propose the APPR, APP, and APPD messages. In Section IV, we propose a hybrid APPD-APPR message. In Section V, we compare the number of computations required for the proposed messages. Finally, some conclusions are drawn in Section VI.

VI. CONCLUSIONS
We have proposed four new messages that require only computation of elementary operations for decoding of LDPC codes. The APPR message, APP message, and APPD message for variable and check nodes were derived for sum-product decoding of LDPC codes. Also, we have proposed the hybrid APPD-APPR message, which denotes APPD variable node message and APPR check node message. By using different messages at variable and check nodes, the hybrid APPD-APPR message-passing has the lowest computational complexity among the proposed messages.
