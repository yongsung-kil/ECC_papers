ABSTRACT
This paper proposes the power-efficient LDPC decoder architecture which features (1) a FIFO buffering based rapid convergence schedule which enables the decoder to accelerate the decoding throughput without increasing the required number of memory bits, (2) an intermediate message compression technique based on a clock gated shift register which reduces the read and write power dissipation for the intermediate messages. Simulation results show that the proposed decoder achieves 1.66 times faster decoding throughput, and improves the power efficiency (which is defined by the power dissipation per Mbps) up to 52% compared to the decoder based on the conventional overlapped schedule.

1. INTRODUCTION
A Low-Density Parity-Check (LDPC) code was proposed by Gallager in the early 1960’s, and was re-discovered by MacKay and Neal in 1996. LDPC code achieves information rates very close to the Shannon limit by using the messagepassing algorithm [1].
The message-passing algorithm performs the row and column operations iteratively, where the intermediate messages α are updated by the row operations and β are updated by the column operations. Originally, the standard messagepassing schedule performs the row and column operations alternatively. In order to increase the decoding throughput and to improve the hardware utilization efficiency, several particular schedule have been proposed in Refs. [2], [3] where the row and column operations are performed simultaneously. However, the decoder in Ref. [2] requires twice number of memory bits for the intermediate messages

in order to decode two frames of codeword simultaneously. The decoder in Ref. [3] overcomes this problem by overlapping the row and column operations (i.e. overlapped schedule), thus, the decoder requires the same number of memory bits as that based on the standard schedule. These conventional decoders increases the decoding throughput by at most twice, in addition the area and power dissipation of memory for the intermediate message would increase about twice compared to the decoder based on the standard schedule since they require the dual-port memory in order to read and write the messages simultaneously. Therefore, the power efficiency (which is defined by the power dissipation per Mbps) is not improved efficiently.
This paper proposes the power-efficient LDPC decoder architecture which features as follows: (1) A FIFO buffering based rapid convergence schedule which enables the decoder to accelerate the decoding throughput more than twice than that based on the standard schedule without increasing the required number of memory bits. (2) An intermediate message compression technique based on a clock gated shift register which enables the decoder to reduce the read and write power dissipation for the intermediate messages. In the proposal (2), the message compression technique reduces the write power dissipation for the message α by reducing the number of write ports. However, this technique increases the read power dissipation for the message α since the bit width of the single compressed message increases more than twice compared to that of the single original message α while the number of read ports is same as that before applying this technique. In order to reduce the read power dissipation, we propose the clock gated shift register based FIFO buffering architecture for the compressed messages α.

In this paper, firstly, we propose a rapid convergence schedule which enables the decoder to increase the decoding throughput more than twice. We focus on the fact that the computational complexity of the column operation is less than that of the row operation. Therefore, the column operations can be performed more frequently than the row operations when the row and column operations are performed simultaneously. Figure 1 shows our rapid convergence schedule for (3,6)-regular QC-LDPC codes. The rapid convergence schedule performs the row and column operations in the following two steps: (Step 1) The row operations are performed determining the position for the column operations. (Step 2) The column operations are performed using every message α which is updated by the row operations. The pipelined operation enables the decoder to perform the row and column operations simultaneously.
The simulation result in Fig. 2 shows that the bit error rate based on the rapid convergence schedule with 6 iterations outperforms that based on the standard and overlapped schedule with 10 iterations. Therefore, the decoder based on the rapid convergence schedule achieves 3.33 times and 1.66 times faster compared to that based on the standard and overlapped schedule respectively.

3. POWER-EFFICIENT DECODER
ARCHITECTURE
In the following section, we propose the power-efficient LDPC decoder architecture based on (1) a FIFO buffering based decoder architecture and (2) an intermediate message compression technique based on a clock gated shift register.

quired number of memory bits to perform the row and col-

umn operations simultaneously. In order to overcome this

problem, we propose a FIFO buffering based decoder ar-

3.3 Clock gated shift register
Originally, the FIFO buffering based rapid convergence schedule requires N sets of 2-stage FIFO buffers which contain N sets of write ports and 2 × N sets of read ports for each of M row blocks. Bit width of every port is Q.
Here, we propose a FIFO buffering based hardware architecture for the proposed message compression technique which is shown in Fig. 4. The proposed architecture contains the FIFO buffer for signs and absolute values of messages α separately. For the absolute values, a single set of (2×N )-stage FIFO buffer which contains a single write port and 2 × N read ports is required since the absolute values are compressed and shared by N submatrices. Bit width of every port is shown in Eq. (8). Therefore, compared to the original FIFO buffer, the total number of write bits for the absolute values can be reduced since the number of write ports is compressed to a single port. However, the total number of read bits for the absolute values increases more than twice since the bit width of the single compressed message increases more than twice compared to that of the single original message α while the number of read ports is same as that before applying this technique. This indicates that the proposed compression technique enables the decoder to reduce the write power dissipation of the compressed message, however the read power dissipation of that

4. SIMULATION RESULTS
We evaluate an area and power dissipation of the proposed FIFO buffering based on the message compression technique, and compare it with that before applying the message compression technique. Table 2 show the synthesis results in TSMC 0.18 μm CMOS using Synopsys Design Compiler and Power Compiler.
Table 2 shows an area and power dissipation results in various coding rates. The result shows that the proposed technique based on the clock gated shift register increases the area by 29% – 5% compared to that before applying the compression technique when the coding rate ranges 1/2 – 2/3. This is because the FIFO buffering based on the proposed message compression technique can not be implemented with SRAMs which is more area-efficient compared to the shift register, since the FIFO buffer have to output N compressed messages simultaneously. However, when the coding rate increases up to 7/10 – 3/4, the proposed technique reduces the area by 2% – 13% compared to that before applying the compression technique since the proposed technique reduces the required total number of memory bits much more effectively as the coding rate increases. Table 2 shows that the proposed technique reduces the memory power dissipation for message α by 52% – 68% compared to that before applying the compression technique.
Table 3 shows the comparisons of LDPC decoders where we designed each of the decoders for 3,072bits rate 1/2 (3,6)regular QC-LDPC codes with Q = 5 bits, and R = 1/2.

Each of the decoders features as follows: (a) the overlapped schedule based on the dual-port SRAM (which is designed based on Ref. [3]), (b) FIFO buffering based rapid convergence schedule. (c) FIFO buffering based rapid convergence schedule with intermediate message compression technique based on a clock gated shift register. As shown in Fig. 2, the rapid convergence schedule with 6 iterations achieves better bit error performance compared to the overlapped schedule with 10 iterations, as the result of which the rapid convergence schedule achieves 1.66 times faster decoding throughput while achieving better bit error performance. Table 3 shows that the proposed rapid convergence schedule accelerates the decoding throughput more efficiently compared to the conventional overlapped schedule in terms of the power dissipation. The proposed decoder architecture (b) improves the power efficiency 41%, and reduces the total cell area by 18% compared to the decoder based on the overlapped schedule. When we apply the normalized min-sum algorithm to the decoder instead of the Gallager function, the proposed decoder architecture (c) improves the power efficiency up to 52% and reduces the total cell area by 17% compared to the decoder based on the overlapped schedule.
5. CONCLUSION
This paper proposes a power-efficient LDPC decoder architecture which features (1) the FIFO buffering based rapid convergence schedule and (2) the intermediate message compression technique based on the clock gated shift register. Simulation results show that the proposed decoder achieves 1.66 times faster decoding throughput compared to the decoder based on the conventional overlapped schedule while reducing the required number of memory bits and the power dissipation for the memory. Accordingly, the proposed decoder architecture improves the power efficiency up to 52% and reduces the total cell area by 17% compared to the decoder based on the conventional overlapped schedule.
