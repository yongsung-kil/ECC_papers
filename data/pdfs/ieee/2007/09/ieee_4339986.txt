Iterative Decoding of L-step Majority-Logic Decodable Codes Based on Belief Propagation
Suquan Ding
Beijing Institutes of Tracking and Telecommunications Technology, Beijing, 100094, China Email: dsq03@mails.tsinghua.edu.cn

Abstract— Iterative decoding of L-step Majority-Logic decodable (LSMLD) codes based on belief propagation (BP) is presented. The orthogonal Tanner graph of LSMLD codes is derived based on the L-step orthogonalized parity checks. The BP decoding algorithm based on the orthogonal Tanner graph (BP-O) is also designed. As the orthogonal Tanner graph is sparse and free of 4cycles, LSMLD codes perform well under BP-O decoding. Simulation results show that BP-O decoding of LSMLD codes achieves a good soft-decision decoding gain over hard-decision decoding and the performance of LSMLD codes under BP-O decoding is equally well as that of LDPC codes with the same block length and rate under BP decoding.
I. INTRODUCTION
It is well known that low-density parity-check (LDPC) codes with belief propagation (BP) decoding can achieve near Shannon limit performance [1], [2]. Also, the connections between BP decoding and Turbo decoding and other well-known algorithms are well understood [3], [4]. Although BP decoding performs well for LDPC codes and Turbo codes, it is hard to achieve good results when applying it to conventional codes, e.g. BCH codes, because these codes usually have high density parity checks and their Tanner graph contains many cycles of short length.
A class of conventional codes which performs well under BP decoding is one-step majority-logic decodable (OSMLD) codes [5]. It is shown that OSMLD codes outperform LDPC codes with the same block length and rate and the performance improvement is mainly due to the redundant sparse constraints [6]. Also, the fact that the Tanner graph of OSMLD codes is free of 4-cycles ensures the codes perform well under BP decoding.
Another class of codes which also has many redundant parity checks is L-step majority-logic decodable (LSMLD) codes [7], [8]. The conventional decoding algorithm is hard-decision L-step majority-logic decoding.

A class of iterative decoding algorithm based on bitflipping (BF) is proposed to exploit the redundant parity checks and achieve near maximum likelihood decoding performance over binary symmetric channel (BSC) [9]. However, as the Tanner graph of LSMLD codes contains many 4-cycles, it is hard to achieve good results when iterative decoding based on BP algorithm is employed directly.
A method of transforming generalized parity check (GPC) matrix is proposed which aims at generating code representations suitable for BP decoding [10]. The performance of BP decoding based on the transformed GPC matrix is improved significantly when the signals transmitted over binary erasure channel but the results are not always so good when transmitted over additive white Gaussian noise (AWGN) channel. Specially, when the method applied to LSMLD codes, the number of check nodes and variable nodes in the Tanner graph corresponding to the constructed GPC matrix is very large compared with the code parameters which means the decoding complexity is large while the performance improvement is not so significant.
We find that the matrix corresponding to the L-step orthogonalized parity checks of LSMLD codes is a GPC matrix which provide a code representation suitable for BP decoding. In this paper, we investigate the performance of BP decoding of LSMLD codes based on such GPC matrix.

III. BP DECODING BASED ON THE ORTHOGONAL TANNER GRAPH

Orthogonal Tanner graph is somewhat different from standard Tanner graph, hence BP-O is also different from standard BP decoding in some aspects.

IV. SIMULATION RESULTS
The simulation results of the BP-O decoding of LSMLD codes are presented in this section. All the simulations are conducted over AWGN channel with BPSK modulation.
The first case studied is the triple-error-correcting (63, 45) BCH code which is also a two-step majority-logic decodable code [8]. There are 63 variable nodes and 18 check nodes in the standard Tanner graph of the code which contains some 4-cycles. On the other hand, there are 71 variable nodes and 56 check nodes in the orthogonal Tanner graph which is free of 4-cycles. It is obvious that the orthogonal Tanner graph is also much sparser than the standard Tanner graph.

V. CONCLUSION
The orthogonal Tanner graph of LSMLD codes is derived from the L-step orthogonalized parity checks. Although the standard Tanner graph of LSMLD codes is dense and contains some 4-cycles, the orthogonal Tanner graph is sparse and free of 4-cycles thus provides a code representation suitable for BP decoding. The BPO decoding is designed to adapt to orthogonal Tanner graph. Simulation results show that BP-O decoding can provide a good SDD gain over HDD. It is also shown that LSMLD codes under BP-O decoding perform equally well as randomly generated LDPC codes with the same block length and rate under BP decoding. It is shown by the case of LSMLD codes that by transforming the Tanner graph properly BP decoding can achieve good

results when applying to the codes which have high density parity checks.
