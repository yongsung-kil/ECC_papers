Cost-Efficient Partially-Parallel Irregular LDPC
Decoder with Message Passing Schedule
Xing LI, Yuta ABE, Kazunori SHIMIZU, Zhen QIU, Takeshi IKENAGA, Satoshi GOTO Graduate School of Information, Production and Systems, Waseda University 2-7 Hibikino, Wakamatsu-ku, Kitakyushu, Fukuoka 808-0135 Japan

Abstract- This paper proposes an improved Message Passing schedule for irregular LDPC decoder. Redundant memory accesses and column operations are removed by utilizing the characteristics of partial-parallel irregular LDPC decoding algorithm. As a result, the memory access frequency and hardware cost are efficiently reduced. According to the experimental results and comparison with existing work, proposed decoder provides a 30% hardware area reduction and a 36% power consumption saving with the same error correcting performance.
1. Introduction
Proposed by Gallager in the early 1960's, LDPC code was re-discovered by Mackay and Neal [1] in 1996. Compared with the traditional Reed-Solomon Codes, sparse graph based LDPC code performs much better on error correction. Also the LDPC decoding algorithm can support high speed parallel processing. So far LDPC code has attracted much attention all around the world and been recognized as the best error correction code in the near future. The irregular LDPC code (code rate 1/2) proposed by Chung[2] demonstrates only 0.04dB close to Shannon limit at BER. Recently, the inherent parallelism of the well-known message-passing algorithm that can realize row and column operations independently for LDPC codes makes it very suitable for LDPC decoder hardware implementation.
There are two kind of parallel LDPC decoders for message-passing algorithm, one is Full-Parallel decoder [3],[4] and the other one is Partial-Parallel decoder. The latter occupies less hardware area and shows better correcting performance than the former [5],[6],[7].
To achieve high speed decoding, some Partial-Parallel decoders utilize concurrent schedule which performs one row and one column operation simultaneously. However the message-passing efficiency between the row and column operations is degraded. To improve the message-passing efficiency, the accelerated message-passing schedule in [8] was proposed for regular LDPC codes. The algorithm performs one row operation, and multiple column operations using concurrent messages updated by the row operations.
For irregular LDPC codes, the schedule proposed in [8] cannot be utilized directly. If use this schedule originally designed for regular LDPC, there will exist lots of redundant computations in the irregular LDPC decoding. Because the row and column weights in irregular LDPC are not fixed, it is very likely to appear that column operations perform over frequently within some local column blocks of parity check matrix, and most messages are already covered before use as

well. The unnecessary computation of column operations for LDPC decoder definitely induces larger hardware area and more power consumption.
In this paper, we propose a new efficient message-passing
schedule for irregular LDPC code. Our approach is based on the schedule designed for regular LDPC code in Ref. [8]. We

We find that messages that come from some column operations are covered directly before the next row operations. In other words, these kinds of column operations are all redundant. It might be possible to omit these redundant column operations in decoder.
Proposed irregular LDPC decoder architecture is
implemented under TSMC 0.18ptm CMOS technology. The
parity check matrix shown in Fig. 1 is employed, which is one
of the parity check matrixes specified by IEEE 802.1 In
Standard. With the equal error correcting performance,
proposed design provides a 30% hardware area reduction and a 36% power consumption saving, compared with the design
utilizing the algorithm proposed in [8].
