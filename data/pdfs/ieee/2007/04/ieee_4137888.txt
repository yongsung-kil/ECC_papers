High-Rate Quasi-Cyclic Low-Density Parity-Check Codes Derived From Finite Affine Planes
Norifumi Kamiya, Member, IEEE

Abstract—This paper shows that several attractive classes of quasi-cyclic (QC) low-density parity-check (LDPC) codes can be obtained from affine planes over finite fields. One class of these consists of duals of one-generator QC codes. Presented here for codes contained in this class are the exact minimum distance and a lower bound on the multiplicity of the minimum-weight codewords. Further, it is shown that the minimum Hamming distance of a code in this class is equal to its minimum additive white Gaussian noise (AWGN) pseudoweight. Also discussed is a class consisting of codes from circulant permutation matrices, and an explicit formula for the rank of the parity-check matrix is presented for these codes. Additionally, it is shown that each of these codes can be identified with a code constructed from a constacyclic maximum distance separable code of dimension 2. The construction is similar to the derivation of Reed–Solomon (RS)-based LDPC codes presented by Chen et al. and Djurdjevic et al. Experimental results show that a number of high rate QC-LDPC codes with excellent error performance are contained in these classes.
Index Terms—Constacyclic maximum-distance separable (MDS) codes, finite affine planes, low-density parity-check (LDPC) codes, one-generator quasi-cyclic (QC) codes, point-line incidence matrix, QC codes.
I. INTRODUCTION
LOW-density parity-check (LDPC) codes, originally invented and investigated by Gallager [13], are defined as null spaces of sparse matrices. The great merit of LDPC codes is that they are relatively easy to decode suboptimally using iterative decoding algorithms [13], [27], [40]. Generally, the computational complexities of iterative decoding algorithms are linearly proportional to the number of -entries in the underlying sparse parity-check matrix, and hence very long codes are feasible. When the sparse matrix is well designed, improvement in error performance can be gained by increasing the number of decoding iterations. It has been shown that randomly constructed LDPC codes with long code lengths perform remarkably close to the Shannon limit [5], [25], [32].
Recently, deterministic approaches to constructing LDPC codes have been developed (cf. [4], [10], [20], [22], [37], [38]). These deterministic constructions are based on block matrices with circulant matrices as blocks and on such combinatorial designs as affine and projective geometries. Codes constructed
