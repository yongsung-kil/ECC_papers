A Modified Belief Propagation Decoding Algorithm

of LDPC Codes for Fast Convergence

Zhu Ai-min
Zhen Jiang Watercraft College of the PLA Zhenjiang, China

Abstract— The Belief-Propagation decoding algorithm for
Low-Density Parity Check Codes is analyzed and the convergence characteristic of this algorithm is researched. A modified Belief-Propagation decoding algorithm aiming at decreasing the number of iteration is proposed. Simulation result shows that the number of iteration of the proposed algorithm is only half of that of the standard one at expense of little loss in performance.
Key words: Low-Density Parity Check (LDPC) codes; Belief Propagation Decoding Algorithm; Iterative Decoding
I. INTRODUCTION
Low-density parity-check codes (LDPC)[1,2], which is the same as Turbo codes, show a good performance achieving Shannon limit and can be applied to almost every channel. It has been focus of researcher for recent years. For linear block codes, the complexity of decoder grows exponentially with the code length and when the code length exceeds a certain degree, the decoding complexity is incontrollable. But for LDPC codes, there exists an effective decoding algorithm, complexity of which grows linearly with the code length because of sparseness of parity check matrix. It also benefit from sparseness of parity check matrix that code bits far away in a long codeword participate in a uniform check, which provides the performance of anti-outburst errors. So in LDPC encoded system, there is no need for interleaver and there is no delay introduced by interleaver.
The decoding algorithm for LDPC codes includes iterative decoding algorithm based on hard-decision and soft-decision. The performance of hard-decision decoding algorithm has a poor performance because it only uses hard-decision message of the channel. Soft-decision iterative decoding always means belief-propagation (BP) algorithm [2,3] on factor graphs, these graphical models encode in their structure a particular factorization of the joint probability

The BP algorithm operates by “message-passing” in a Bayesian network, translates immediately into an instance of the sum-product algorithm operating in a factor graph that expresses the same factorization. The BP algorithm can compute exactly posterior probabilities when there is no cycle in bipartite graph of LDPC codes. The introduction of BP algorithm caused a great glow in research of LDPC decoding algorithm, mainly concerning the following two aspects. On one hand, decrease the computation complexity through simplifying some functions in BP algorithm, which includes Min-Sum. On the other hand, decrease iteration number and decoding delay through some message disposals in LDPC decoder [5,6].
This paper mainly concerns on improvement of BP algorithm. We first gave the standard Belief-Propagation decoding algorithm for Low-Density Parity Check Codes, then analyzed convergence characteristic of this algorithm. A modified Belief-Propagation decoding algorithm aiming at decreasing the number of iteration is proposed based on convergence characteristic of this algorithm. Simulation result shows that the number of iteration of the proposed algorithm is only half of that of the standard one at expense of 0.2dB loss in performance.
II.STANDARD BP ALGORITHM AND ITS CONVERGENCE
CHARACTERISTIC
For each received bit, xi for i=1, 2, …, N, in an N bit block, an LDPC decoder accepts as input, the log likelihood ratio (LLR) of probability of possible values for xi. LDPC codes are often represented by bipartite graphs made up of two families of nodes, bit nodes and check nodes. Bit nodes represent the transmitted bits in the code. Each bit node is connected to a number of check nodes through an unstructured array of edges. Each check node represents a parity check constraint on the set of adjacent bit nodes. LDPC decoders implement the message passing algorithm, which specifies the computation and communication of messages

III. A MODIFIED BELIEF PROPAGATION DECODING ALGORITHM FOR FAST CONVERGENCE
According to above analysis, we notice that it may helpful to make some nonlinear disposal on messages delivered in LDPC decoder, through which we can try to force the decoding process to converge even faster by “pushing” the nodes’ belief towards 0 (or 1, respectively). This obviously introduces artificial noise in the decoding process and can hence be expected to result in higher error rates. But it is useful to decrease number of iterations greatly only at expense of little performance loss.
We first divide the LLR of code digits into three areas. The first area, LLR<|a|, is non-confidence area. The second area, |a|<LLR<|b|, is half-confidence area. The last one, LLR>|b|, is full- confidence area. a,b is constant depending on simulation.
For code digits in non-confidence area, we multiply a constant α less than 1 in order to decrease the range of oscillation, while for code digits in full -confidence area, multiply a constant β greater than 1 in order to speed up convergence. For code digits in half-confidence area, we dispose the message according to standard BP algorithm. To utilize the message of channel fully, the first two iterations should go on in standard BP manner.

IV. CONCLUSION
LDPC codes have been applied in some communication systems because of its excellent performance. However, the BP decoding algorithm used to decode LDPC needs a great deal of iterations, which not only increased hardware resource, but also added decoding delay of LDPC codes. In this paper, we start with statistic characteristic of messages delivered in BP algorithm and make some nonlinear disposals on them, which greatly decrease the number of iterations. It is valuable

in engineering application. But the theoretical deduction of modified algorithm still remains in research.
