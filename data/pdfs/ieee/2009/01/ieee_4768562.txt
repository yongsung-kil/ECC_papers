Low-Density Parity-Check Codes with 2-State Trellis Decoding
Mao-Ching Chiu, Member, IEEE

Abstract—A class of low-density parity-check (LDPC) codes with a simple 2-state trellis structure is presented. For LDPC decoding, the conventional belief propagation (BP) algorithm consists of numerous sub-decoders of single-parity check codes and exchanges information between sub-decoders in an iterative manner. If the single-parity check codes can be constructed and grouped in a proper way, the decoder can be decomposed into few identical 2-state trellis decoders. Therefore, instead of numerous sub-decoders of single-parity check codes, an iterative decoding algorithm based on few sub-decoders over 2-state trellises is proposed. The proposed decoding algorithm improves the efficiency of message passing between sub-decoders and hence provides a fast convergent rate as compared to the standard BP algorithm. Simulation results show that the proposed scheme provides a better performance and a fast convergent rate as compared to those of standard BP algorithm. The result also shows that the proposed algorithm has a similar performance as that of asynchronous replica shuffled BP algorithm and has a slightly inferior performance than that of synchronous replica shuffled BP algorithm. However, complexity analysis shows that our proposed algorithm has complexity that is lower than that of the replica shuffled BP algorithm.
Index Terms—Low-density parity-check (LDPC) codes, iterative decoding, trellis decoding.
I. INTRODUCTION
L OW-DENSITY parity check (LDPC) codes were first proposed by Gallager [1] in 1963 and then later examined by several investigators [2]–[5]. LDPC codes are a class of linear block codes which have attracted considerable attention in the coding community because they provide nearcapacity performance with iterative decoding based on the belief propagation algorithm (commonly known as the sumproduct algorithm (SPA)) [4], [5].
However, approaching near-capacity performance requires hundreds or even thousands of iterations using conventional message passing algorithm over single-parity check codes [6]. This is due to the inefficiency of message exchange between decoders of single-parity check codes, especially for the check nodes in the Tanner graph [2] that constitute a long cycle. The efficiency of the message passing can be improved through proper scheduling of message update. In [7]–[10], horizontal scheduling was proposed to serialize the decoding of LDPC codes, and speed-up of the convergence was achieved using the scheduling process. In [11], [12], the so-called shuffled BP algorithm based on vertical scheduling was presented to speedup the decoding of LDPC codes. To further improve

In [15]–[17], several important scheduling schemes depending on the girth and closed walks of the Tanner graph [15], [16] or reliability information [17] were presented, and were shown to achieve significant performance improvement upon the standard BP algorithm. Another class of algorithms, called generalized belief propagation (GBP) algorithm [18], is to group nodes into regions of nodes and the message is exchanged between regions of nodes. In practice, GBP algorithm can often outperform standard BP algorithms in terms of accuracy or convergence rate [18].
In this paper, we propose another method to speedup the decoding of LDPC code. The idea is similar to the grouping concept proposed in [18]. The efficiency of the message passing can be improved if several check nodes could be grouped and decoded in a jointly optimal manner. It should be noted that all check nodes which do not constitute any cycle in the Tanner graph can be grouped and represented as a simple 2-state trellis. The code can then be decoded using few 2state trellis decoders. However, for a better regularity of the decoder structure, we propose a new construction of LDPC codes which can be iteratively decoded using identical 2-state trellis decoders. The 2-state trellis decoder can be implemented by the Bahl, Cocke, Jelinek, and Raviv (BCJR) algorithm [19]. To further reduce the complexity of the BCJR algorithm, we propose a new decoding algorithm over the 2-state trellis. The new algorithm reduces the storage requirement and the computational complexity by a factor of at least 2 as compared to the conventional BCJR algorithm. We compare the complexity and performance of the proposed algorithm with those of standard BP and replica shuffled BP algorithms. According to simulations, the proposed construction with trellis decoding algorithm provides at least twice as fast as Gallager’s code with the standard BP algorithm and has a comparable performance to that of the replica shuffled BP algorithm. However, complexity analysis shows that our proposed algorithm has complexity that is lower than that of the replica shuffled BP algorithm.

V. CONCLUSIONS
We present a new class of LDPC codes which can be decoded efficiently using few identical simple 2-state trellis decoders. The proposed construction with trellis decoding algorithm improves the efficiency of message passing between sub-decoders and hence provides a fast convergent rate as compared to the conventional BP algorithm. A simplified BCJR algorithm is proposed which reduces the storage requirement and the computational complexity by a factor of at least 2 as compared to conventional BCJR trellis decoding algorithm. According to simulations, the proposed construction with trellis decoding algorithm provides at least twice as fast as Gallager’s code with the standard BP algorithm and has a comparable performance to that of the asynchronous replica shuffled BP algorithm. However, complexity analysis shows that our proposed algorithm has complexity that is lower than that of the replica shuffled BP algorithm.
