Application of Active Current Mirrors to Improve the Speed of Analog Decoder Circuits

Shahaboddin Moazzeni, Glenn E. R. Cowan, Member, IEEE Department of Electrical and Computer Engineering Concordia University Montréal, Québec, Canada
{s_moazze@ece.concordia.ca, gcowan@ece.concordia.ca}

Abstract— Current-mirrors are widely used in the blocks of analog decoder circuits based on the sum-product algorithm. The simple current mirrors, using diode-connected input devices that traditionally have been used are limited in speed when their inputs are loaded with long wires. We present a block diagram for a large (6, 3) LDPC code and an estimation for the largest possible wiring capacitance in this decoder. We show that when active current mirrors are used on the inputs of the equality nodes, their speed is enhanced. The power and speed for a seveninput equality node are simulated with the basic mirrors and with the modified current-mirror circuits. We show through simulations in STMicroelectronics CMOS 90nm technology and with a supply voltage of 0.4V that the power/speed ratio for a 7input equality node can be improved by approximately 50% with the enhanced topology.
I. INTRODUCTION
RECENTLY iterative analog decoders have been found to outperform their traditional digital counterparts in terms of power/speed ratio as well as chip area [1]-[4]. Very large codes up to 5114 bits have been implemented with digital circuits [5], [6], and [7], while the largest semi-iterative (not fully parallel) decoder to date has been reported to be up to 2432 with analog circuits [8].
The Sum-product algorithm is commonly used to implement complex decoding schemes [9]. In order to realize this algorithm, different modules namely equality and check nodes working with probability information for each bit are electrically connected in a configuration that corresponds to Tanner graph for a particular code. Both equality and check nodes have current-mirror circuits at their inputs. For accurate operation of the nodes, these must operate in the weakinversion region. Though this region of operation gives rise to low transconductance, and correspondingly high input resistance, the intrinsic speed of individual nodes is adequate, owing to the small capacitance of the input transistors. However, for large codes, as the number of nodes increases,

wiring the modules that are millimeters apart generates large wiring capacitance which has significant effect on the throughput of an analog decoder.
Increasing the bias currents in a weak-inversion current mirror will proportionally increase its bandwidth. However, since both speed and power will increase equally, this approach fails to improve the power/speed ratio. Active current mirrors at the inputs of nodes allow an improvement in the power/speed ratio for the nodes necessary for implementing the sum-product algorithm.
In this paper, a large Low–density-parity-check (LDPC) code [10] defined by a (1024 x 512) H-matrix is presented based on the sum-product algorithm. The largest possible wiring capacitance for this code which corresponds to the worst case spacing between the modules of the Tanner graph is then estimated. It is then shown that by applying an active current-mirror approach to the input circuits of each node we improve the speed of the analog decoder, and its power/speed ratio.
Current-mode design approaches provide a variety of useful features such as improving the bandwidth [11], [12]. In the aforementioned decoder circuits, we apply a current-mode technique to the input current-mirror circuits of one of the equality nodes. This technique creates a negative feedback loop through an operational transconductance amplifier (OTA) which lowers the input impedance of the current-mirrors, thereby improving the speed of the mirrored currents. However, the increase in the bandwidth is at the expense of higher power consumption. By choosing the parameters of the modified current-mirror circuit properly, it is shown that this extra power consumption is less than the improvement in speed. Therefore the power/speed ratio is decreased by applying the current-mode approach.

II. A LARGE LDPC DECODER
A (6, 3) LDPC code defined by a (1024 x 512) H-Matrix is considered for this example. This requires 512 equality nodes and 1024 check nodes having 6 and 3 edges connected to each node, respectively. Taking into account an extra edge for the received bits there will be a total of 7 bidirectional edges for every equality node. If d indicates the number of edges connected to a bidirectional node, then it can be shown that there must be 3(d-2) two-input unidirectional nodes within each d-input node [13]. As a result, 15 unidirectional equality nodes and 3 unidirectional check nodes are used in the equality and check nodes. This is shown in Fig. 1. It is observed that in a block of equality nodes, there are 7 separate edges for the inputs as well as for the output and each output is generated by the other 6 inputs. In the block of check nodes there will be 3 input edges and 3 output edges and every output is produced by the other 2 inputs.

IV. CONCLUSION To reduce the effect of wiring capacitance on the speed of an analog decoder circuit, a current-mode technique was applied. It was employed in the current-mirror circuits of a block of equality nodes for a large LDPC code defined by a (1024 x 512) H-matrix. Having estimated the worst case wiring capacitance for the code, a significant reduction in the speed of basic current-mirrors was observed. However, replacing the basic mirrors with the enhanced topology showed that the achieved speed in the block of equality nodes is nearly more than twice the speed with the basic currentmirrors, while the total power dissipation is doubled for the modified case. Thus an improvement in the power/speed ratio for the complete decoder is expected if the basic mirrors are replaced with the enhanced mirrors.
