A NEW FORCED CONVERGENCE DECODING SCHEME FOR LDPC CODES

Jianxiao Fan, Hongwen Yang

BUPT Wireless Communication Center Beijing University of Posts and Telecommunications, Beijing
fanjianxiao@gmail.com, yanghong@bupt.edu.cn

Abstract
In this paper, we present a new forced convergence decoding scheme for LDPC codes. We remove the early detected variable nodes from the Tanner graph and thus the parity matrix shrinks iteration by iteration and the decoding complexity can be reduced. When the parity check matrix shrinks to zero before the maximum iteration, we get a chance to detect the wrong deletion and, for the decodable codewords, most of such failure can be recovered by restoring the original Tanner graph and resuming the decoding in the remaining iterations. Simulations results indicate that the proposed method can reduce the decoding complexity significantly while keep the error rate performance unchanged or even better.

1 Introduction
LDPC code [1] has attracted more and more attention for its remarkable performance. For the small terminals such as mobile handsets, decoding the LDPC codes with low complexity is always desirable since low complexity implies low cost of the hardware implementation and also low consumption of battery power. The decoding of LDPC codes is commonly realized by passing messages along the edges between the nodes of Tanner graph and hence the decoding complexity depends mainly on the following three factors:
1) The complexity due to the processing of each node; 2) The number of iterations required to finish the decoding of one codeword; 3. The number of nodes involved in one iteration, or more precisely, the number of edges involved in one iteration, since nodes with different degree have different complexity.
The complexity of the first point mentioned above is mainly contributed by the check nodes updating and there are already a lot of low complexity algorithms for this [2]-[5]. To reduce the number of iterations, some early stopping techniques [6]-[9] can be applied. ________________________________

To address the third point, the early Decision [10], [11] or forced convergence decoding (FC) [12], [13] schemes detect reliable nodes (converged nodes) early and skip the message updates for these nodes.
In this paper, we present a new forced convergence decoding scheme for LDPC codes. In the previous methods [10]-[13], the early detected nodes are deactivated, i.e., the message updates for those nodes are skipped while the edges connecting to them are still reserved to pass the message to the connected active nodes. With the scheme proposed in this paper, the detected variable nodes, together with the connecting edges, are completely deleted from the Tanner graph and then the total number of edges is reduced iteration by iteration. It may happen that we have deleted some nodes with wrong hard decision. However, for the decodable codeword, such failure often leads to an empty Tanner graph before the decoder reaches its maximum iteration number and thus we can restore the original Tanner and recover the failure.
Simulation results show that, compared with conventional scheme, the proposed scheme can achieve about 28%---49% reduction of node updating complexity and 44%---84% reduction of tentative checking. At the same time, the performance is almost the same or even better. Compared with FC [12] [13], our scheme is more effective in reducing the complexity in check node updating and tentative checking. Besides, proposed scheme has no error floor problem which may occur in [11], [13].
The reminder of the paper is organized as follows: Section 1 briefly introduce the Conventional decoding algorithm. The forced convergence scheme is introduced in Section2. Our proposed method is described in Section 3. Section 4 is the simulation results and Section 5 is the conclusion.

4.2 Shrink the Tanner graph
When we have determined the convergence set D, we can shrink the Tanner graph by deleting these nodes, i.e. to set the corresponding columns of H to all zeros. Deleting variable nodes may change the odd/even parity check relation. For example, in equation b1  b4  b17  b117 0 , if b117 is known as 1, then the remaining nodes satisfy b1  b4  b17 1 which is odd parity check. Thus, when we delete some variable nodes, we should also update the parity
check relation p .

It is possible that, at some iteration stage, we will have some check nodes of degree 1 since all other variable nodes connected to it have been deleted. Recall that a variable node is deleted because we have assumed the node is known. Hence, for example, if the check node b1  b4  b17  b117 0 has been reduced to b1 p , then we must have previously

5 Simulation Results
In this section, we use simulation to compare the proposed scheme with the conventional scheme and FC in terms of frame error rate (FER) and the complexity. In the simulation, we use the (1152, 576) LDPC code defined in[15] and the code bits are BPSK modulated and passed through an AWGN channel. We use the horizontal-vertical scheduling [10] for the message passing and the maximum iteration number is 60. The check node updating includes both SPA and MSA. The parameter p0 in the proposed scheme is 0.0001. The parameter pair tv ,tc in FC is (5, 5) which is from [13] and is also a good choice for the code we used here.

When the check node is updated with SPA, the proposed scheme has almost the same FER performance as conventional scheme while FC has obvious error floor. The error floor problem of FC was also reported in [11],[13].

6 Conclusions
In this paper, we proposed a new forced convergence decoding algorithm for LDPC codes. By deleting the converged variable nodes from Tanner Graph, the parity check matrix shrinks iteration by iteration and hence the complexity can be reduced. The proposed scheme also has a mechanism to fix the problem of incorrect deletion. Simulation results indicate that, compared with conventional scheme, the proposed method can reduce node updating complexity by 28%--49% and tentative checking by 44%--84%, while keep the error rate performance unchanged or even better.
