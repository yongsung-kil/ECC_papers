Abstract—Layered Belief-Propagation (LBP) with dynamic schedule for Low-Density Parity-Check (LDPC) codes using variable nodes residual is presented. The proposed schedule outperforms the conventional LBP with predetermined schedule both in terms of convergence speed and error performance. As compared with Node-wised Residual Belief-Propagation (NRBP), it has much lower extra complexity. For LDPC code constructed by PEG, simulation results show that the new schedule offers better error performance and requires much lower extra complexity.
Keywords-LDPC codes; dynamic schedule; layered beliefpropagation; residual
I. INTRODUCTION
Low-density parity-check (LDPC) codes with the ability to approach Shannon limit under low-complexity iterative beliefpropagation (BP) algorithm over the factor graph of the code. The message passing schedule over the factor graph directly affects convergence rate, complexity and error performance. The standard schedule is flooding schedule [1], in which in each iteration all the variable nodes, and subsequently all the check nodes, pass new messages to their neighbors. Besides the flooding schedule, some different types of non-flooding, or sequential schedules for iterative BP decoding were proposed earlier, such as check-node updating schedules presented in[2][4],variable-node updating schedules presented in[5][6]. Several studies[5][7]show that sequential schedules not only speed up the convergence rate in terms of number of iterations but also outperforms the traditional flooding schedule for a large number of iterations.
On the contrary to above strategies with predetermined schedule at each iteration, recently, some dynamic update schedules based on residual were introduced in [8][9], in which the difference between the values of the message before and after an update is used to dynamically update the schedule. Simulation results show that these dynamic schedules require significantly less iterations, and offer better error performance than standard sequential schedules. However, extra residual computation for these dynamic schedules increases the computational complexity.
In this paper, Layered Belief-Propagation (LBP)[3] with dynamic schedule based on variable nodes residual is

proposed. The schedule chose the next check nodes according to the log-likelihood-ratio (LLR) difference of variable nodes before and after an update. Since the residual computation only comes from the LLR difference of variable nodes before and after an update, it significantly reduces the extra residual computational complexity.

Sequential message-passing schedules such as LBP update check nodes sequentially according to a predetermined schedule. However, Dynamic scheduling strategies presented in[8][9] chose the next check node to be updated according to the residual of check-to-variable node message. The residual is the difference between the values of check-to-variable node message before and after an update. Simulations show that RBP LDPC decoding speeds up the convergence rate but its error-rate performance for a large enough number of iterations is worse. Node-Wise schedule inspired by RBP presented in [8] was found to perform better than LBP across all iterations in the waterfall region of several LDPC codes.

III. VARIABLE NODES RESIDUAL BELIEF PROPAGATION
Inspired by NRBP schedule, sine less greedy RBP with better error performance than RBP, we need not propagate the message strictly according to the residual with maximum value. If check-to-variable message has maximum residual after update, the variable message mj will have maximum residual with very high probability, so we can update the message according to variable nodes residual.

A. Algorithm

LBP with dynamic schedule based on symbol residual chose check nodes to be updated next according to symbol residual. We named the new schedule as variable nodes residual belief propagation (VRBP). For a variable node vj, let r(mj) be the residual. The behavior of the new schedule can be explained as follow: Initially, for every symbol vj, the residual are set to the value of their correspondent channel absolute message Cj. then the symbol vi with the maximum residual is chosen, and for every check node ck ∈ N(vi), all check-tovariable message and variable residual r(mj) is updated. Finally vi symbol residual r(mj) is set to zero. This implies that, at the beginning of message propagation, the new schedule will propagate the symbol message with the highest reliability to the symbol with the lower reliability, and then will propagate the symbol message with maximum residual. For a symbol with variable degree dv, since the schedule propagates the message through dv check node, it is a less greedy RBP.

Compared with NRBP, VRBP has two distinguished characters: very low residual computational complexity and dv check nodes update at one dynamic iteration. dv is the variable degree and one dynamic iteration is message propagation between two residual chosen. This also can be explained by the fact presented in[8] that as the Eb/N0 increases, trapping-set errors become dominant, and RBP can conquer trapping-set due to it greedy iteration decoding. For HVRBP, the FER performance is significantly improved at the Eb/N0=2.5dB. This mainly attributes to that hybrid strategy reduces the negative effects of the greediness of RBP. V. CONCLUSION
LBP with dynamical schedule based on variable nodes residual for LDPC codes was presented and the computational complexity was addressed. The proposed schedule chose the next check nodes to be updated based on the posteriori LLR difference of variable nodes before and after an update. It guarantees that the message of variable nodes with the highest reliability will propagate to the variable nodes with the lower reliability message at the initial stage of dynamical iterations. Normal variable residual belief propagation will work at the later stage. Due to the simple residual computation, the new schedule with very low extra computational complexity as compared with NRBP. Simulations performed on the PEG codes have shown that the new schedule outperforms the conventional LBP with predetermined schedule both in terms of convergence speed and frame error performance. Especially for hybrid schedule HVRBP, it offers better FER performance than NRBP while the extra computational complexity is much lower.
