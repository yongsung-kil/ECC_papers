A parallel layered decoding algorithm for LDPC codes in WiMax system

Wang Jun, Yang Shu-hui
College of Photoelectric Information and Communication Engineering Beijing Information &Technology University Beijing, China

Abstract— WiMax presents a quasi-cyclic LDPC codes with excellent performance in decoding, which is suitable to parallel decoding for its structure. And the flexible matrix structure can support more implementation of code with different code-length. To decrease the iterative times of the decoding algorithm and save the hardware resource of the decoder for LDPC codes, the paper presents a parallel layered decoding algorithm based on OMS, which doesn’t reduce the decoding performance. This algorithm proposes a novel scheduling to improve the updating of message. Simulation results show that, for a 2304 bits length of code, the parallel layered decoding algorithm exhibits a better convergent performance than a traditional algorithm. The SNR of parallel layered OMS decreases about 0.05 dB than that of traditional OMS, when the BER is 10-3.
Keywords-WiMax; parallel layered; quasi-cyclic LDPC;OMS
I. INTRODUCTION
Low-Density Parity-Check (LDPC) codes were firstly introduced by Gallager in 1962[1]，which is too complex to implement and had been forgotten for almost 20 years in spite of its excellent performance approximate to Shannon Limit. In 1990s, with the rise of graphic encoding and iterative decoding, the LDPC codes had been discovered again. And sum-product algorithm (SPA) for LDPC codes has been shown to approach the capacity of the AWGN channel with 0.0045 dB [2][3].
For its excellent performance in forward error control, LDPC codes have been widely considered as nextgeneration error-correcting codes for many wireless and wireline digital communication standards, such as DVBS2[4], WiMax (IEEE 802.16e)[5] and 10GBASE-T[6]. WiMax system (Worldwide Interoperability for Microwave Access) is a technical standard of MAC and PHY, which supports fixed and mobile broadband wireless access network system. For transmission of data in low bit-errorratio, LDPC codes have been adopted by WiMax system which has became the forth 3G standard after WCDMA, CDMA2000 and TD-SCDMA.

III. DECODING ALGORITHM FOR LDPC CODES
Iterative decoding based on belief propagation (BP) is suitable for LDPC codes [9][10]. BP algorithm is a soft-in and soft-out decoding method, which analyses each decoding bit in reliability, based on parity-check-sum. The reliability–based analysis may include posteriori probability or conditional probability computation. Then, we put the result of reliability–based analysis into iteration and calculate the new check sum again. Until we get a satisfactory value, the iteration does not stop. Eventually, based on the value of iteration, hard-decision is made for every bit. Due to BP algorithm is based on probability

IV. A FAST CONVERGENT PARALLEL LAYERED OMS IN
WIMAX SYSTEM
In application, we need an improved algorithm to speed convergence. In traditional OMS, variable nodes are updated one by one, and the updating of check nodes starts when the all the variable nodes have been updated. These two points slow the updating of message. So we have to adopt a new scheduling algorithm to update message faster. This paper presents a parallel layered algorithm for OMS. This algorithm updates several variable nodes synchronously and the updating of check nodes start when part of variable nodes had been updated. Considered structure of matrix H in WiMax system, we can divide matrix to several parallel computing process. In this paper, we adopt a matrix H of 1/2 code rate which contains 12×24 sub-matrix. And the size of sub-matrix is 96×96, so there are 96 sub-iteration in one iteration. The parallel layered algorithm for OMS can be illuminated basically as the following: owing to 12 sub-matrix in column, we divide check-to-variable updating into 12 parallel process. And each sub-iteration starts from the first row in each sub-matrix. To the same principal, the variableto-check updating is divided into 24 parallel processes. And each sub-iteration starts from the first column in each submatrix. Then, when the first iteration both in check-tovariable updating and variable-to-check updating is finished, the fresh matrix containing only one row or column fresh message is sent into updating of message. VI. CONCLUSION
Based on WiMax system, this paper presents a parallel layered scheduling used in OMS. For the 2304-bits LDPC code, the SNR of parallel layer OMS decreases about 0.05 dB than that of traditional OMS, when the BER is 10-3. The parallel layered scheduling OMS can decrease the times of iteration and speed the convergence.
