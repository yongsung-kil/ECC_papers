Cooperative Relay Channel with LDPC Codes Constructed from Array Codes

Mingkwan Somphruek, Rolando Carrasco and Stephane Le Goff
School of Electrical, Electronic and Computer Engineering Newcastle University, United Kingdom
{mingkwan.somphruek, r.carrasco, stephane.le-goffjtgmcl.ac.uk

Abstract-Cooperative communication represents a new class of diversity techniques in which multiple nodes each with a single antenna cooperate to generate a virtual multiple-antenna transmission system and thus offer the benefits of spatial diversity. This paper proposes suitable array codes used as lowdensity parity-check (LDPC) codes to be applied in the cooperative amplify-and-forward (AF) network. A mathematical description of the construction of suitable array codes for a relay fading channel is also presented. Compared with quasi-cyclic (QC) LDPC codes, the array codes have similar encoding complexity and can adaptively generate various desired girth values. This paper shows that array codes with large girths can achieve substantial system performance improvements. Simulation results in this paper show that using array codes as LDPC codes for cooperative relay transmission provides a significant performance improvement over direct transmission. Moreover, it is also shown that its performance is superior to some structured LDPC codes such as QC-LDPC codes, particularly, a coding gain of about IdB at BER of 10-3 can be provided over its counterpart.
I. INTRODUCTION
In wireless networks, diversity schemes such as time, frequency and spatial diversity have been ubiquitously employed in communication systems in order to combat the severe effect of fading and path loss. Particularly, multipleantenna systems which provide diversity gain have been widely used in wireless communications since significant improvement in both throughput and range can be achieved without any increase in bandwidth or overall transmit power. However, this scheme may not be practical for some applications, such as mobile devices which may not be able to implement multiple antennas due to the constraints of size and cost. Recently, cooperative communication has been proposed [1-3] in which single-antenna users share their antennas to create a virtual multiple transmit antennas system and achieve a diversity gain. Cooperative schemes [4] can be categorised as amplify-and-forward (AF), where each user simply amplifies the received signal from its partner and retransmits it to the destination and decode-and-forward (DF), where each user attempts to decode the received signal and retransmits the reencoded signal to the destination. AF relaying is one of the most popular cooperative schemes due to its low complexity and good diversity gain. So far, these cooperative schemes have been integrated with various channel coding methods in order to improve the system performance [5-8].

Due to their powerful error-correcting capabilities, lowdensity parity-check codes (LDPC) [9] introduced by Gallager have been found in different applications of communication systems. Recently, LDPC codes have been integrated into relay channels [5, 10, 11] and achieved excellent performance improvement on various channels. Generally, an LDPC code can be described by a Tanner graph and the length of the shortest cycle in a Tanner graph is referred as girth of the graph. It is well known that cycles, particularly short cycles, affect the code performance. Large girth not only leads to more efficient decoding iterations when using sum-product algorithm and also obtains better error floor property than random LDPC codes [12]. Moreover, it has been proved in [13] that a lower
bound on the minimum distance dmin increases with the girth
of the code. Thus, LDPC codes with large girth are preferred in order to improve the code performance.
Among different types of LDPC codes, array codes [14] can be considered as one of the best performing structured LDPC. These codes are quasi-cycle codes based on circulant permutation matrices so they are simple to implement and encode with shift registers[15] and lead them to be one of the most practical LDPC codes. In general, codes of large girth can be obtained via extensive computer search until the conditions of the cycles are met [16]. On the other hand, by shortening an array code [17], array codes with pre-specified length of cycles up to twelve can be simply obtained and this method also provides a less computation. Therefore, a wide range of desired code parameters such as girth and code rate can be easily obtained by array code construction. This paper presents array codes as LDPC codes to be applied to a cooperative network which has not been considered in literature before. The paper focuses on LDPC codes with large girth obtained by shortening an original array code. The performance of array codes with different girths over AF relaying scheme is proposed and evaluated in terms of bit error rate (BER).
The paper is organised as follows. In the next section, the system model of AF relaying network is presented. This protocol operates in a half-duplex mode and uses the time division multiple access (TDMA) scheme. In Section III, the construction of LDPC codes based on array codes and the shortening method that obtains the codes of higher girth will be described. The simulation results of applying good array codes for the relay channel are given in Section IV, and finally conclusion of the paper will be presented in Section V.

The relay receives a noisy version of the transmitted signal from the source, then simply amplifies this noisy version subject to its power constraint and retransmits to the destination. The destination combines the received signals from both nodes and estimates the original messages. In a time division allocation scenario, this cooperative transmission can be divided into two time slots of equal duration, one time slot for the source transmission and one time slot for the relay transmission.

In this paper, coherent detection is employed at each node; hence, the channel state information (CSI) of each link is perfectly known. IV. SIMULATION RESULTS

The performance comparison with other codes of similar code parameters is also provided. The results are examined in various scenarios based on the quality of the uplink channel and the inter-user channel. The .qualities of these channels are represented by the average received SNR. For all codes, the sum-product algorithm is used for decoding with a maximum of 50 iterations. These codes have a designed code rate of 1/2 , column weight of 3 and row weight of 6 with code length of 3606.

B. AF cooperative system using array codes with various girths
Based on the system model described in Section II, it can be seen that the received signals at the destination from source and relay are equivalent to repetition coding which leads to the loss of transmission spectral efficiency. Therefore, in order to obtain identical spectral efficiency as non-cooperative systems and provide a fair comparison, each user has to increase the code rate or use higher order modulation schemes in cooperative mode. In this paper, quadrature phase shift keying (QPSK) is employed for AF scheme instead of increasing the code rate. It results in the system spectral efficiency of 0.5 bit/s/Hz for both non-cooperative and cooperative scenarios. For the symmetric scenario in which both source and relay have a statistically similar uplink channel quality, the performance of the AF scheme by employing array codes with various girth values is illustrated in Fig. 3. It can be observed that better diversity gain can be achieved by improving the inter-user channel quality. Also, array codes with girth of twelve provide the best performance among all codes. For example, with a 10dB inter-user channel at BER of 10-3, it has a 1.3dB and IdB coding gain over girth-six and eight codes, respectively.
C. Comparison ofarray codes and QC-LDPC codes over AF cooperative system
For comparison with other types of LDPC codes, the BER curves of QC-LDPC code constructed from [21] is also shown in Fig. 4(a). This QC-LDPC code has code rate of 1/2 , column weight of 3 and row weight of 6 with code length of 3606. Based on its construction method, the girth of this QCLDPC code can not be pre-specified as array codes and it has actual girth of 6. AF relaying with a QC-LDPC code also outperforms the direct transmission and achieves diversity gain. AF cooperation with girth-twelve array code attains more significant improvements over QC-LDPC code. At BER of 10-3, the array code can provide a IdB and 0.8dB coding gain over QC-LDPC code relaying when the qualities of inter-user channel are 10dB and 20dB, respectively.
Fig. 4(b) shows the performance of the asymmetric scenario in which the qualities of uplink channels for each user are different. For instance, the source has a better uplink channel than the relay node. In this case, the average SNR of the source uplink channel is fixed at 20dB whereas the average SNR of relay uplink channel is varied from OdB to 20dB. At 10dB inter-user channel quality, it can be seen that not only the performance of the user (relay) with a weaker uplink channel is improved significantly but also the performance of the user (source) with a stronger uplink channel has a marked improvement. This shows that the user with very strong uplink channel is encouraged to cooperate, although the other user has a poorer uplink channel. Moreover, these results support the

v. CONCLUSION
In this paper, a coded AF relaying scheme using array codes as LDPC codes has been considered and analysed. These codes are constructed based on an array code construction and larger girth values can be simply obtained by elimination of all short cycle-characterising equations. The simulation results

show that the BER performance of array codes in a slow fading environment can be improved by increasing the girth value of codes. Furthermore, a diversity gain can be achieved in array code cooperative system and codes with large girth value provide the best performance among all codes. Also, large girth array codes can outperform QC-LDPC codes in which a coding gain of about 1dB at BER of 10-3 can be provided over QCLDPC codes. Cooperation is also advantageous for both users not only when users have equal uplink channel qualities but also in the case of different uplink channel qualities.
