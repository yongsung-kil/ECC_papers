Hybrid Burst Erasure Correction of LDPC Codes
Marc Fossorier

Abstract—In this letter, burst erasure correction of low density parity check codes based on a hybrid approach is presented. The proposed method preserves as much as possible the original low density representation and augments it by specific check sums judiciously selected.
Index Terms—Burst erasure correction, low density parity check codes, burst correcting form, block codes.
I. INTRODUCTION
I N [1], an optimum method to correct any burst of up to N − K erasures has been proposed for any (N, K) linear block code. This method can be implemented as a complement to any random error correction scheme. However for long codes (say N ≥ 1000), this method may become quite complex to implement as it requires the storage of an N × N parity check matrix HBCF in burst correction form (BCF). An exception is that of cyclic codes since HBCF corresponds to the cyclic representation of the parity check matrix of the code.
Unfortunately most classes of low density parity check (LDPC) codes are not cyclic and their code lengths are generally quite large. Furthermore HBCF looses the initial sparseness property in general.
In this letter, we investigate how the BCF can increase the burst erasure correction capabilities of LDPC codes by reducing storage and maintaining as much as possible their sparse representation. To this end, we present a two-stage method which first permutes the original LDPC H matrix based the construction of a generalized BCF. This permuted low density matrix is then augmented by a controlled number of check sums selected from the corresponding generalized BCF in order to achieve a guaranteed burst correction capability. For a high rate quasi-cyclic (QC) LDPC code, an example which achieves an efficiency of 88 % for burst erasure correction with the proposed method is presented.

This diagonal structure allows the recovery of any burst of up to N − K erasures [1], [2], thus achieving optimum burst erasure correction.
III. DEFINITIONS
A. Burst correcting capability rM (H) of H
Given an M × N parity check matrix H, we define rM (H) as the maximum value r for which each possible r × M submatrix of H has rank at least rM (H). The value rM (H) is defined as the burst erasure correcting capability of H and can be achieved by Gaussian elimination. It follows that rM (HBCF ) = N − K.
B. Generalized burst correcting form HBCF (L)
We assume that erasure bursts of length L ≤ N − K have to be corrected and therefore construct a N × N matrix HBCF (L) with a diagonal of L − 1 zeroes between two ones. Consequently HBCF = HBCF (N − K) and L ≤ rM (HBCF (L)). The same approach as in [1] can be used to construct HBCF (L).
The construction of HBCF (L) is motivated by the fact that for L < N −K, it is generally easier to obtain HBCF (L) than HBCF (N − K) when following the probabilistic approach of [1]. Furthermore, the next property provides an additional degree of freedom for L < N − K.

• (4) Repeat Step-1 to Step-3 until Lmax = Lmax,it.

VI. CONCLUSION
In this letter, a hybrid method to enhance burst erasure decoding of LDPC codes based on their BCF has been presented. By judiciously adding check sums of the BCF to the corresponding low density representation of H, a guaranteed burst correction capability Lmax,it can be achieved. While the proposed approach considers the design of the BCF independently of the burst erasure capabilities of H in low density form, a joint design which includes these latter in determining the permutation π() should further improve this method.
