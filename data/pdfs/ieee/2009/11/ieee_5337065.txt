An Efficient On-the-Fly Encoding Algorithm for
Binary and Finite Field LDPC Codes
Shayan G. Srinivasa, Member, IEEE, and Anthony Weathers, Member, IEEE
Abstract—We present a general framework on prototype struc-
tures for efficiently encoding low density parity check codes on-
the-fly. The encoding algorithm is applicable to finite field based
codes and can handle a wide range of high code rates and large
block lengths. This structure can be used particularly in data
storage and multimedia applications with increased code rate
and throughput requirements. We also establish the necessary
and sufficient conditions for the existence of such encoders.

I. INTRODUCTION
L
OW density parity check (LDPC) [1] codes enjoy pop-
ularity due to superior error correction capabilities for
several channel considerations. Practical advancements in de-
veloping area and power efficient hardware for decoding these
codes have paved way for realizing an LDPC based system
in magnetic storage and other applications. Though random
LDPC codes are theoretically the best choice to meet channel
performance, they are not viable due to high memory and
computational requirements, suggesting the use of structured
LDPC codes.
While standard encoding and decoding algorithms for
LDPC codes are well established [2], we still need coding
techniques that are architecture-aware. Richardson and Ur-
banke [3] described several efficient constructions based on
matrix restructuring and singular value reduction using a two-
step process. Andrews et al. [4] developed encoders for block-
circulant LDPC codes using erasure decoding techniques. Li et
al. [5] developed efficient encoding structures for quasi-cyclic
based LDPC encoders using simple shift registers. Myung et
al. [6] obtained fast encoding schemes based on the encoding
theme presented in [3] but restricting the parity check matrix to
almost triangular form with additional constraints. In general,
most LDPC encoding techniques are based on some sort
of column permutations of the original parity check matrix
followed by sparse reduction techniques for solving parity bits.
In this paper we will describe an algorithm for systemati-
cally encoding LDPC codewords based on a parity prototype
structure. Our idea generalizes the approach taken by Olcer
and Eleftheriou [7] for solving the parity bits on-the-fly using
an upper triangular matrix and supports a wide range of
code rates and lengths without any degradation in the system
performance while meeting the high throughput requirements.
The structure is systematic, flexible and can incorporate both
regular and irregular geometries over finite fields providing
flexibility in terms of performance and design needs.

We remark that the array code based encoding structure
described in [7] is a special case of the general approach
described in this paper. The upper triangular structure in [7]
is easy to work with since the parity bits can be easily solved
by backward substitution in equations. However, the resulting
code has poor protection due to column weight 1 appearing
on a set of 𝑠variable nodes. This leads to more performance
degradation with quantized LDPC decoding as shown in the
simulations in the next section. It is often appropriate to choose
a higher column weight i.e., greater than or equal to two in
the parity prototype to achieve the desired performance.

to other prototypes that avoid this situation. Thus, the gen-

design of on-the-fly encodable structures with no degradation
in performance relative to a good regular code.
IV. CONCLUSIONS
We developed a framework for on-the-fly encoding of
LDPC codes. The idea is applicable for a wide range of
code rates and code lengths as needed in many magnetic and
optical recording channels. A careful choice of the prototype
can support on-the-fly encoding with no loss in performance
relative to a good random regular code.
