Shortening for Irregular QC-LDPC Codes
Xiaojian Liu, Xiaofu Wu, and Chunming Zhao, Member, IEEE

Abstract—Shortening is a technique to obtain codes of shorter length and lower rate from a given LDPC code by putting infinite reliability on some variable nodes, whose positions are assumed to be available to both encoder and decoder. In this paper, we propose a shortening algorithm suitable for irregular QC-LDPC codes. The efficiency of the proposed algorithm is verified by both theoretical analysis and simulation.
Index Terms—Quasi-cyclic LDPC code, shortening, girth.

I. INTRODUCTION

Fig. 1. The base matrix of (2304, 1152) QC-LDPC code in IEEE. 802.16e.

S HORTENING is a technique to put infinite reliability on some variable nodes for a given Tanner graph, whose positions are assumed to be available to both encoder and decoder. In the decoding process, the shorten variables in the Tanner graph are simply ignored. Consider now a (n, k) binary linear code, which has k information bits and (n−k) party bits. Assuming that k (k < k) information bits were shortened, we obtain a (n−k , k −k ) code, with code length of (n−k ) and rate of (k − k )/(n − k ). Traditionally, shortening is applied to turbo product codes (TPCs) to obtain different encoding block sizes [1]. Recently, a hybrid ARQ scheme based on LDPC codes using shortening has been proposed in [2]. It has been shown that shortening is an effective complementarity of puncturing and extension for adjusting both the rate and the code length of a given LDPC code. Compared with both puncturing and extension, shortening has its advantage on computational complexity, which is preferable for making minor adjustment to the original code.
QC-LDPC codes [3] [4] have encoding advantages over random-constructed LDPC codes as they can be encoded using simple shift-register circuits [5]. They are widely adopted in various standards, such as IEEE. 802.16e and IEEE. 802.16n. The effect of shortening on regular QC-LDPC codes has been discussed in [6] and [7]. With shortening, the number of short cycles, which often governs the performance of regular QCLDPC codes, can be effectively reduced. Nevertheless, for irregular QC-LDPC codes, the degree distribution is a more important factor for determining the decoding performance [8]. Therefore, the criterion for choosing shortening positions

II. SHORTENING FOR IRREGULAR QC-LDPC CODES

A. Base Matrix of an QC-LDPC Code

An irregular QC-LDPC code can be described with a base

matrix Hb = [hi,j], that consists of M rows and N columns. Each “1” in Hb represents a z×z circulant permutation matrix,

and “0” represents a zero matrix. Hb can be divided into the

systematic part Hbs and the parity part Hbp. Usually, the first

column of Hbp consists of three elements, and its rest part is

organized in a dual diagonal form. With such structure, simple

but fast encoding method exists. For example, Fig. 1 shows

the base matrix of the (2304, 1152) QC-LDPC code in IEEE.

802.16e.

Other advantages of the irregular QC-LDPC codes include

fast decoding and small amount of memory acquirements. The

decoding process can be a block-unit operation with

ρ(i)

block memories, whose sizes are uniformly z × z. Here, M

is the number of rows in Hb, and ρ(i) denotes the weight of

the ith row.

IV. CONCLUSION
In this paper, we propose an algorithm to search for the optimal shortening patterns for irregular QC-LDPC codes. This approach is effective and easy to be applied in various standards. The efficiency of the proposed algorithm is verified by both theoretical analysis and simulation.
