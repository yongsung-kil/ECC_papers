2009 International Symposium on Information Engineering and Electronic Commerce

Research of an Improved BP Decoding Algorithm

Guoqiang Wu
Research Center of Satellite technology Harbin Institute of Technology Harbin, China hitwgq@hit.edu.cn
Abstract—That the computation load is big is a shortcoming of traditional BP decoding algorithm. According to this shortcoming we give an improved BP decoding algorithm. The decoding algorithm in the iterative decoding process, only need to update the possibility of a wrong bit of information without having to update the information whose reliability is very high, raises the decoding efficiency and reduces decoding time delay. The simulation result shows: the calculation load of improved BP decoding algorithms descends distinctively with the increasing iteration numbers; Along with the times increasing of LDPC code iterations upper limit, coding performance can be obviously improved.
Keywords- LDPC Code, BP decoding arithmetic, AWGN channel, Decoder, Iterative decoding algorithms
I. INTRODUCTION
LDPC code is a kind of linearity grouping coding which is paid attention to recently years. It was discovered originally by Gallager, so named as Gallager code, too. Passing several years dreariness, with computer hardware and relative theory development, MacKay and Neal rediscover LDPC code. T J Richardson had proved that combining LDPC code with iterations decoding based on BP has the performance [1] of approaching to Shannon limit. Its encoder structure is simple, and it can carry out Parallel calculation and acquire skyscraping throughput with less resource wastage in theory.So it is regarded as one of main channel-coding in future communications field. Recently years, magnitude progress has been acquired for the theory research of LDPC code, engineering application and VLSI realization. Channel coding already adopts LDPC code in DVB-S2 norm which has been recommended by ETSI. The CCSDS131.1-O2(EXPERIMENTAL SPECIFICATION) has been proposed for close ground operation and deep air operation by NASA, and different LDPC code projects have been separately putted forward. Along with WiMAX and etc. communications system has already adopted LDPC code as its channel coding. Soon after, LDPC code will apply massively to communications field [2-5]. The paper puts up an improve BP decoding algorithms according to traditional BP algorithms which brings great calculation quantity and low coding efficiency.

B. Tanner Chart
The chart which is related with checkout matrix A is named Tanner chart. The chart includes two groups’ nodes. One group denotes code word length including N nodes. The nodes are called bit-node. The other group is called checkout-node including M nodes, and denotes parity restriction. If in the mth checkout equation just exists the nth bit, namely Amn=1, then exists link-lines between the nth bit node and the mth checkout node in Tanner chart. Thus, Tanner chart is figure expression of checkout matrix and will be used to research deeply into decoding algorithms. As Tanner chart denotes a checkout matrix of regular code, every bit-node links with wc checkoutnodes and every checkout-node links with wr bit-nodes. III. LDPC CODE PROBABILITY BP DECODING ALGORITHMS
At present the decoding algorithms of LDPC code mainly adopts belief propagation (BP) iterations decoding algorithms, which is proposed by Mackay and Neal based on Tanner chart. Each iteration of BP decoding includes two steps: disposal of checkout node and disposal of variable node. All checkout nodes receive message from nearby variable nodes. The message returns to nearby variable node after treatment. Then all variable nodes carry out the same process. At last, the variable nodes gather all using message to verdict. During the decoding process of LDPC code, each checkout (or variable) node may be seen as a processor. All the disposal of checkout (or variable) nodes may carry out at one time. So using parallel structure may construct high-speed encoder of LDPC code. Convenient for describing, we explain all symbolic meanings involved in BP coding algorithms.
