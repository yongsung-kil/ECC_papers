Large Girth Quasi-Cyclic LDPC Codes Based on the Chinese Remainder Theorem
Xueqin Jiang and Moon Ho Lee, Senior Member, IEEE

Abstract—In this letter, we consider two problems associated with quasi-cyclic low-density parity-check (QC-LDPC) codes. The first is how to extend the code length of a QC-LDPC code without reducing the girth. The second is how to design a QCLDPC code with a prescribed girth easily. We deal with these two problems by using a combining method of QC-LDPC codes via the Chinese Remainder Theorem (CRT). Codes constructed with our proposed method have flexible code lengths, flexible code rates and large girth. Simulation results show that they perform very well with the iterative decoding.
Index Terms—LDPC codes, quasi-cyclic, array codes, shortened array codes, Chinese Remainder Theorem (CRT), girth.
I. INTRODUCTION
L OW-density parity-check (LDPC) codes, first proposed in the early 1960’s [1] and re-discovered in 1996 [2], have recently attracted much attention due to their capacityapproaching performance and low decoding complexity. One parameter that is usually targeted for optimization in the process of designing LDPC codes is the girth of the underlying Tanner graph [1][3][4]. For iterative sum-product decoding of quasi-cyclic (QC) LDPC codes, a part of performance loss is attributed to small values of girth. Array codes which can be viewed as having properties of QC-LDPC codes and with girth 6 were proposed in 2000 [5]. One approach to design larger girth array codes by deleting certain block-rows and block-columns of the parity-check matrices was presented in [6]. As we can see, a shortened array code gets larger girth by sacrificing its code length. In this letter, we propose a method to extend the sacrificed code length of a shortened array code without reducing its obtained large girth. This method need not only be used to extend a shortened array code with a large girth and short length, but can also be used to design a QC-LDPC code with a prescribed girth easily. The simulation results show that the codes constructed with the proposed method perform better than the array codes and the shortened array codes. Chinese remainder theorem (CRT) has applications in many areas, such as computing, coding and cryptography, such as RSA-CRT and secret sharing, and digital signal processing. A combining method to design girth 6 QC-LDPC codes based on CRT was first proposed in [7] and then generalized in [8]. QC-LDPC codes constructed with our method differ from the codes proposed in [7][8] in that our codes have flexible code lengths, flexible code rates and, most importantly, larger girths.

This letter is organized as follows. Section II review QCLDPC codes, array codes and shortened array codes. The generalized combining method via CRT is given in Section III. We propose a method to extend a shortened array code into different lengths based on the CRT in Section IV. A method for the design of a QC-LDPC code with a prescribed girth is given in Section V. Finally, Section VI concludes the letter.

VI. CONCLUSION
Based on the Chinese Remainder Theorem, we introduced how to extend the code length of a shortened array code without reducing the girth and how to design a QC-LDPC code with a prescribed girth easily. Codes constructed with our method differ from the codes in [7][8] in that our codes have more flexibilities (i.e. design code rate, code length) and larger girth. In this letter, only array codes and shortened array codes were considered as component codes, even though other QC-LDPC codes can be naturally employed.
