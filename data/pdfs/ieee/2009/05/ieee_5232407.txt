A Fast Double Layered Decoding Algorithm for Quasi-Cyclic LDPC Codes

Zhixiong Chen Jinsha Yuan
School of Electrical & Electronic Engineering North China Electric Power University Baoding, Hebei, P.R.China, 071003 chenzx1983@sohu.com

Abstract‚ÄîMost Quasi-Cyclic LDPC codes decoders of IEEE 802.11n and IEEE 802.16e standards adopt Layered decoding algorithm recently. A fast double layered decoding algorithm for Quasi-Cyclic LDPC codes in which two row groupings of check nodes are processed synchronously in order to speed the decoding processing is proposed. Compared to BP algorithm and normal Layered decoding algorithm, the proposed algorithm provides a faster decoding speed and more favorable tradeoffs among complexity, decoding speed and error correcting performance. Simulation results show validity and efficiency of the proposed decoding algorithm.
Keywords-iterative message-passing decoding; Quasi-Cyclic LDPC codes; layered decoding; IEEE 802.11n; IEEE 802.16e
I. INTRODUCTION
Quasi-Cyclic LDPC (QC-LDPC) codes [1-2], kinds of structured LDPC codes, are easy to implement because of their block and cyclic interconnections. The required memory to store QC-LDPC codes for both encoders and decoders can be reduced by a factor 1/L, when L√óL square blocks are employed. Moreover, QC-LDPC codes have encoding advantages over pseudo-random LDPC codes as they can be encoded using simple shift-registers, with probable complexity linearly proportional to their code length [3]. As a result, QC-LDPC codes have been recently adopted in several standards: Wireless Local Area Network (IEEE 802.11n)[4] and Metropolitan Area Network (IEEE 802.16e)[5], etc.
In order to keep tradeoffs among hardware complexity, decoding throughout and error-correction performance there are many decoding algorithms for LDPC codes. Belief propagation (BP) algorithm offers excellent performance and fully parallel decoding at the cost of a large decoding complexity. The Min‚ÄìSum (MS) algorithm [6] simplifies check-node operations of BP algorithm to reduce the decoding complexity but decreases decoding performance. Therefore several modifications [7] to the MS algorithm, such as normalized MS and offset MS decoding, with an aim to strike a balance between complexity and performance have been proposed.
However, most decoders of QC-LDPC codes in standards favor Layered decoding algorithm [8-9] which updates variable-node information after every layer operation of check-node group to speed up the decoding process, for the reason that rows of QC-LDPC codes parity-check matrix consisting of square blocks can be grouped into definite layers. The algorithm has a faster convergence speed by a

factor of two in terms of decoding iterations compared with the standard BP algorithm [10]. The overall decoder architecture of the layered decoding can be also reduced by nearly 50% compared to that of BP algorithm [11]. Although the iteration times is decreased by nearly 50%, the total decoding speed of Layered decoding is not fast enough indeed, because the partial parallel Layered decoding process is operated layer by layer and the message update of nodes in every iteration costs more time.
In this paper, we propose a modified decoding algorithm named double Layered decoding algorithm for Quasi-Cyclic LDPC codes to strike a balance between decoding speed and complexity. The outline of the paper is as follows. In Section ‚Ö° , basic concepts of QC- LDPC codes and decoding algorithms are reviewed. In Section ‚Ö¢ , we present the
double Layered decoding algorithm for QC-LDPC codes. The performance of new QC- LDPC codes is investigated by simulation in Section ‚Ö£and concluding remarks are given in Section‚Ö§.

III. FAST DOUBEL LAYERED DECODING
The entire H matrix is composed of the same B(H) with different cyclic shifts, which allows structured decoding and reduces decoder implementation complexity. For IEEE 802.11n and IEEE 802.16e standards, each H matrix of QCLDPC codes has 24 column groupings. Having the same number of column groupings between code rates minimizes the number of different expansion factors that have to be supported. Besides, four rates of QC-LDPC codes in the standards are supported: 1/2, 2/3, 3/4 and 5/6. IV. SIMULATION AND DISCUSSION
The performance of double layered decoding for QCLDPC codes is analyzed over an AWGN channel using computer simulations compared to BP decoding and normal Layered decoding algorithm. The modulation is fixed to BPSK and the maximum number of iteration is set to 50. The QC-LDPC codes used in our simulation are based on [13] in which QC-LDPC codes with a block length of 1800 bits and code rate of 2/3 are constructed using a parity-check matrix H(100,6,18).
Figure 1 shows the simulation result of FER and BER for QC-LDPC codes using BP decoding, layered decoding and double Layered decoding algorithm. It is shown that Layered decoding presents the most excellent performance compared to other two decoding algorithms when the iteration number is 50. The proposed double Layered decoding shows almost the same decoding performance over BP decoding algorithm.
Figure 2 compares the average iteration number for three decoding algorithms in which the maximum number is 50. Compared to normal Layered decoding, the proposed double layered decoding increases iteration number, but it still has convergence advantage over BP decoding.

Because the variable node grouping processing costs the same clock pulses, it is obvious that the speed of double Layered decoding is faster than Layered decoding algorithm.
V. CONCLUTION Decoders of QC-LDPC codes in standards favor Layered decoding algorithm which provides s fast convergence speed

and reduces decoder complexity. However the total decoding speed of Layered decoding is not fast enough indeed
A fast double layered decoding algorithm for QuasiCyclic LDPC codes is proposed. Different from normal Layered decoding algorithm two row groupings of check nodes are processed synchronously to speed the decoding processing. Compared to BP algorithm and normal Layered decoding algorithm, the proposed algorithm provides a faster decoding speed and more favorable tradeoffs among complexity, decoding speed and error correcting performance.
