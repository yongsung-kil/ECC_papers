AbstractвАФBelief propagation (BP) is a powerful algorithm to decode the low-density parity check (LDPC) codes over the additive white Gaussian noise (AWGN) channel. The traditional BP algorithm cannot adapt efficiently to the statistical change of the AWGN channel. Particle filter is a algorithm to estimate a variable of interest as it evolves over time. In this paper, we use particle filter to estimate the noise power and feed back to the BP algorithm in real time. We found that compared with the traditional BP algorithm with fixed estimated noise power, BP algorithm based on particle filter not only give a good real-time estimate for the channel noise, but also achieve a lower decoding error rate.
I. INTRODUCTION
As a type of error-correcting code (ECC), low-density parity-check (LDPC) code was first proposed by Gallager in the early 1960вАЩs [1][2] and revived by Mackay and Neal in 1996 [3]. From that time, LDPC code has raised wide interest in research community, because the performance of LDPC code can make data transmission rates achieve near Shannon Limit[4][5].
LDPC codes can be decoded by using a powerful iterative algorithm known as belief propagation (BP) algorithm [3]. For LDPC decoding over the additive white Gaussian noise (AWGN) channel, the knowledge of the estimated noise power is one of the most important parameters to achieve the best performance of the BP algorithm. Hence, a better initial estimation of the noise power would generate a better performance of the BP algorithm. However, for an unknown AWGN channel, it is very difficult to estimate the noise power ѕГ2 without pilot signal or feedback, where the situation when ѕГЋЖ2 = ѕГ2 is defined as вАЭsignal-to-noise ratio (SNR) mismatchвАЭ. The effect of the SNR mismatch on LDPC codes is studied in [6][7][8], which indicated that the BP algorithm is more sensitive to the underestimation of SNR than the overestimation. Moreover, for a time-varying AWGN channel, to generate a real-time estimated noise power is more difficult. In above two cases, the standard BP algorithm would yield a high bit error rate.
To overcome these problems, we need a nearly correct estimation for the time-varying noise over the AWGN channel. Particle filters, also known as sequential Monte Carlo algorithms, are sophisticated model techniques for estimation based on Monte Carlo simulations [9]. The main objective of

particle filter is to estimate posterior probability distribution of an interest object by sampling a list of random particles with associated weights. Hence, we propose a BP algorithm based on particle filter for LDPC decoding in this paper. In our model, we consider a binary LDPC codes over AWGN channel with a BPSK modulation. That is for an input X which takes values ¬±1, the output Y = X + Z, where Z is a Gaussian random variable N 0, ѕГt2 with mean zero and variance ѕГt2, where ѕГt2 evolves over time. Our proposed algorithm is carried out based on factor graph [4][10], which afford great flexibility in modeling systems. We show that the proposed algorithm no longer depends on the initial estimation of noise power ѕГ2 and offers a good real-time estimation for the noise power of AWGN channel. For different code rates, our algorithm shows a lower decoding error rate than that of standard BP algorithm.
This paper is structured as follows. In Section II, we explain the basic concept of LDPC codes. The belief propagation algorithm in factor graph and the belief propagation algorithm based on particle filter are described in Section III and IV, respectively. Finally, in Section V we present simulation results and in Section VI we draw the concluding remarks.
II. LOW DENSITY PARITY CHECK CODES
A LDPC code can be specified by an M √Ч N sparse paritycheck matrix H, where M and N represent the numbers of check-nodes and bit-nodes respectively. A sparse matrix H means that both the numbers of 1вАЩs per column and the number of 1вАЩs per row are less compared to the numbers of rows and columns, respectively. Regular LDPC codes have the same column weight and row weight, and irregular LDPC codes have different column weight and row weight. Any codeword of a LDPC code X = (x1, x2, ... xN ) must satisfy (1).
LDPC codes can be represented by a Tanner Graph as shown in Fig.1. The factor graph contains two types of nodes: the bit nodes and the check nodes [4]. Each bit node represents a bit in the codeword, and also corresponds to a column in the parity-check matrix H. Each check node corresponds to a row of the parity-check matrix H, which represents a parity-check equation. Edges were used to connect bit nodes and check nodes. VI. CONCLUSION
This paper proposed a BP algorithm associated with particle filter in factor graphs for the decoding of regular and irregular LDPC codes in the presence of real-time AWGN channel noise power. Factor graphs express the factorization structure of a function and offer great flexibility in modeling systems. By introducing the particle filter algorithm in the left side of factor graph in the standard BP algorithm, the noise power of an AWGN channel will be updated step by step for each variable node. A precise estimation of noise in the codeword has been observed. Thus, the performance of our algorithm is not sensitive with the initial estimation of the noise power ѕГ2 and can estimate the noise power in real-time. Finally, we concluded that our BP algorithm with particle filter yields a better performance than that of standard BP algorithm.
