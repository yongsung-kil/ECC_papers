Analysis of Absorbing Sets using Cosets and Syndromes
Emily McMillon∗, Allison Beemer†, and Christine A. Kelley∗ ∗Department of Mathematics, University of Nebraska – Lincoln, Lincoln, Nebraska 68588 †Department of Electrical and Computer Engineering, New Jersey Institute of Technology, Newark, NJ 07103

Abstract—Absorbing sets are combinatorial structures in a Tanner graph that have been shown to characterize iterative decoder failure, and particularly error floor behavior, of LDPC codes. In this paper, we examine the connection between absorbing sets and the syndromes of their support vectors. Using this framework, we provide a new characterization of fully absorbing sets, which have been considered the most harmful for iterative decoders. We also show how the sets of absorbing set support vectors appear as translates of codewords in subspaces of the code. These techniques are used to derive new search methods for absorbing sets.
I. Introduction
Though highly regarded due to their low-complexity implementation algorithms and remarkable performance, iterative decoders of low-density parity-check (LDPC) codes remain prone to decoder failure, particularly at high signal-to-noise ratios (SNR). Wiberg’s dissertation [1] was among the earliest works in characterizing iterative decoder convergence on finite-length LDPC code graphs. In an attempt to capture error patterns that can prevent an iterative decoder from succeeding on the additive white Gaussian noise (AWGN) channel, MacKay and Postol later introduced the concept of nearcodewords, which are binary words that violate a small number of check equations [2]. It was noted that low-weight nearcodewords were most problematic. Around the same time, it was shown that certain graphical (sub)structures in an LDPC code’s Tanner graph can contribute to the failure of messagepassing decoders. In the case of the binary erasure channel (BEC), these substructures are termed stopping sets [3], and are simple to describe. Graphically characterizing the so-called error floor phenomena in the bit error rate (BER) curves of codes for other channels is more challenging. Some structures that have been heavily studied in the literature include trapping sets, pseudocodewords, and absorbing sets [4], [5], [6]. The presence of these structures naturally depends on the choice of Tanner graph representation used in decoding, and so work on locating and removing them is of importance in LDPC code design. In this paper, we focus on absorbing sets, which are combinatorially defined [5], [7] and have been shown to contribute to persistent error floors over a variety of channels [8], [9], [10].
In this work, we examine the connection between absorbing sets and the syndromes of their support vectors. We use this connection to identify which vectors correspond to support vectors of absorbing sets. We give upper bounds on the syndrome weight of vectors corresponding to special cases

of absorbing sets such as fully and small (i.e. low weight) absorbing sets, and we show that fully absorbing sets have support vectors that form complete cosets of the code. In particular, we observe that the set of vectors corresponding to absorbing set supports in each coset are translates of subspaces of the code, and the subspace is equal to the code in the case of fully absorbing set support vectors. The use of subspaces in the search for absorbing sets is a new concept, and we provide an initial algorithm using this approach that differs from existing methods (e.g. [11], [12]).
This paper is organized as follows. In Section II, we provide notation and background on absorbing sets and the standard array of a code. We present a characterization of absorbing sets using subspaces of a linear block code in Section III, along with the special case of fully absorbing sets. In Section IV, we give upper bounds on the syndrome weight of the support vectors of absorbing sets of different types. In Section V, we use the coset framework to give a search algorithm for finding fully absorbing sets, which are the class deemed most harmful. Section VI concludes the paper.
II. Preliminaries
In this section, we introduce relevant notation and terminology. Let C be a binary [n, k] linear code with block length n, dimension k, and minimum Hamming distance denoted by dmin(C). If the code is represented by parity-check matrix H, its associated Tanner graph is defined as the bipartite graph G = (V, W; E), where the vertices in part V correspond to the columns of H, and those in W correspond to the rows of H. These are referred to as variable and check nodes, respectively. There is an edge in E if and only if there is a 1 in the corresponding entry of H.
LDPC codes have sparse parity-check matrices and are amenable to iterative decoders that pass messages back and forth along the edges of a Tanner graph. It has been shown that certain substructures in a code’s Tanner graph can be detrimental to the code’s performance at high SNR, resulting in error floors in BER curves. In this paper, we focus on absorbing sets.
For a subset S of V, let N(S ) ⊆ W denote the set of check node neighbors of S , and let GS = (S , N(S ); ES ) denote the subgraph induced by S ∪ N(S ) in G. Furthermore, let O(S ) denote the subset of check nodes of odd degree in GS .
Definition II.1. An (a, b)-absorbing set is a subset D ⊆ V of variable nodes in a code’s Tanner graph G such that |D| = a,

Definition II.2. An absorbing set D is a fully absorbing set if it has the property that each variable node not in D has strictly fewer neighbors in O(D) than in W \ O(D). An absorbing set D is elementary if all check nodes in GD have degree 1 or 2.
Fully absorbing sets and absorbing sets with small a values have been shown to be more likely to cause decoder failure [9]. Hence, much work has focused on finding and eliminating small and fully absorbing sets in a code’s Tanner graph [13], [14]. Because vectors of weight less than dmin(C) have been shown to be especially problematic in other work [2], we define an analogous subset of absorbing sets that we term low-weight.
Definition II.3. An (a, b)-absorbing set is low-weight if a < dmin(C).
We note that low-weight absorbing sets can be elementary, fully absorbing, both elementary and fully absorbing, or have neither characteristic.
For a subset S of variable nodes, we define xS ∈ F2n to be the word whose support is equal to S ; we refer to xS as the support vector of S . Thus, the support vector of an absorbing set D will be denoted xD. In this paper, when we say a vector corresponds to an absorbing set, we mean the vector is the support vector of the absorbing set.
Since the words in Fn2 can be partitioned into cosets of a code by grouping together words with the same syndrome, we are interested in determining the relationship between a support vector of an absorbing set and its syndrome. Recall the following definitions.
Definition II.4. Let H ∈ Fqm×n be a parity-check matrix for an [n, k] linear code C over Fq, where m ≥ n − k. The syndrome of a vector x in Fnq is the vector defined by syn(x) = HxT ∈ Fmq .
Definition II.5. Let C be an [n, k] linear code over Fq. The distinct cosets x + C of C partition Fqn into qn−k sets of size qk, with each coset corresponding to a unique syndrome syn(x). The weight of a coset is the smallest weight of a vector in that coset. Any vector in the coset whose weight is equal to the coset weight is called a coset leader.
Definition II.6. Let H be a parity-check matrix for an [n, k] binary linear code C. The standard array of H is a 2n−k × 2k array where the first row is comprised of all codewords in C starting with 0, the first column is made up of coset leaders of C (with one coset leader assigned to each coset), and the remaining entries are computed by adding the row’s coset leader to the column’s codeword. In this way, a standard array contains all 2n elements of Fn2.

III. Coset and subspace characterization
In this section, we show that when the set of all absorbing sets is partitioned by syndrome, each of these subsets is the translate of a subspace of the code itself. Further, the subsets which contain fully absorbing sets are translates of the entire code, so that if a coset contains a fully absorbing set, the entire coset is comprised of fully absorbing sets. These results can help us to locate absorbing sets in a Tanner graph by searching cosets in clever ways; this is explored further in Section V.
Theorem III.1. The set of absorbing set support vectors that belong to a single coset of a binary code C is either empty or has the form xD + D, where D is a subspace of C, and D is any absorbing set in that coset.
Proof. If there are only one or two absorbing sets in the coset, this is clearly true, with D being equal to 0 or to the subspace generated by the codeword difference of the two support vectors, respectively.
For cosets with more than two absorbing sets, we claim it is sufficient to show that if D is an absorbing set such that xD is in the coset, and the supports of xD + c1 and xD + c2 both also form absorbing sets, then the support of xD + c1 + c2 is an absorbing set. Once this has been shown, it is evident that the absorbing sets that appear in a single coset have the form xD + D, where D is a subspace of the code C.
Suppose, then, that the supports of xD, xD + c1, and xD + c2 form absorbing sets. The set of odd-degree check nodes in

Call this set of odd-degree neighbors O. Note that every variable node appearing in the support of xD, xD +c1, or xD +c2 has fewer neighbors in O than not in O, by the definition of an absorbing set.
Consider a variable node appearing in the support of xD + c1 +c2. It also appears in the support of either one, or all three, of xD, c1, and c2. If it appears in all three, it appears in the absorbing set D. If it appears in one, it appears in at least one of the absorbing sets given by xD + c1 and xD + c2. Thus, it appears in some absorbing set which has odd check node set O, and must have more neighbors not in O than in O. Since this holds for all variable nodes in the support of xD + c1 + c2, we conclude that, by definition, the support of xD + c1 + c2 also gives an absorbing set.
Remark III.2. In Theorem III.1, the set of absorbing sets contained in a particular coset is equal to xD + D for any choice of absorbing set D in the coset. That is, the subspace D does not depend on the choice of translate vector xD.
Corollary III.3. For binary codes, the number of absorbing set support vectors appearing in any coset is either zero or 2i for some i ≤ k.
Proof. This follows directly from Theorem III.1: if nonempty, the set of absorbing set support vectors in a single coset is the translate of a subspace of the code C.
If a coset contains fully absorbing set support vectors, we may say exactly which subspace has been translated:
Theorem III.4. If D is a fully absorbing set, then the support of every word in the coset containing xD is also a fully absorbing set.
Proof. Let D be a fully absorbing set. Let c ∈ C be a codeword, and let C be its support. Note that C is an absorbing set. Then, D ∆ C is the support of the vector xD + c. Note that O(D) = O(D ∆ C), because xD and xD + c are in the same coset. Because D is fully absorbing, for all v ∈ V, v has fewer neighbors in O(D) than outside of O(D). Since O(D ∆ C) = O(D), this implies that all nodes v ∈ V have fewer neighbors in O(D ∆ C) than outside of O(D ∆ C). We conclude that D ∆ C is a fully absorbing set.
Corollary III.5. An absorbing set D is fully absorbing if and only if all vectors in xD +C correspond to fully absorbing sets.
Proof. The forward direction follows from Theorem III.4, and the backward direction holds because xD ∈ xD + C.
As a result of Theorem III.4, we will call a coset fully absorbing if it contains support vectors of fully absorbing sets.

IV. Syndrome weights and absorbing sets
In this section, we explore the connection between absorbing sets and the weights of the syndromes corresponding to their cosets. In particular, we provide upper bounds on syndrome weights when a coset contains the support vector of a low-weight absorbing set and when a coset contains the support vector of a fully absorbing set. Further, we introduce criteria based on minimum variable node degree that guarantees all cosets with syndromes of sufficiently low weight correspond to fully absorbing cosets. This criterion is utilized in the algorithm presented in Section V.
First, we provide a brief characterization of fully absorbing sets based on their syndrome weights. In Section II.B of [9], it is claimed that D ⊆ V is a fully absorbing set if and only if for all v ∈ V, wt(HxTD∆v) > wt(HxTD) = b. We present a generalization for all absorbing sets.
Theorem IV.1. A subset D ⊆ V is an absorbing set if and only if for all v ∈ D, wt(HxTD\v) > wt(HxTD) = b.
Proof. Let D be an absorbing set and v ∈ D. By removing v from D, we flip the parities of all check nodes adjacent to v. Because D is an absorbing set, v has strictly fewer odd degree than even degree check node neighbors. Hence GD\v has more odd-degree check nodes than GD. So, wt(HxTD\v) > wt(HxTD).
Now, let D ⊆ V be such that for all v ∈ D, wt(HxTD\v) > wt(HxTD). Equivalently, for all v ∈ D, GD\v has strictly more odd-degree check nodes than GD. By removing v from D, we flip the parities of all check nodes adjacent to v. Because there are more odd-degree check nodes in GD\v, it must be the case that v had more even-degree than odd-degree neighbors in GD. Hence D is an absorbing set.

V. Algorithm for finding fully absorbing sets
Previous search algorithms for specifically identifying absorbing sets in the Tanner graph of a code have relied on branch-&-bound exhaustive search algorithms [11], [12], while search algorithms for other graph structures harmful to decoding have operated by searching for related substructures such as cycles [15], [16]. In this section, we present a search algorithm for finding fully absorbing sets that relies on the relationships developed in this paper between absorbing sets, syndromes, and the standard array of a code. This algorithm will allow for the identification of fully absorbing sets, which in turn can facilitate the removal of these harmful substructures in code design.
Theorem V.1. If a coset with syndrome s is not fully absorbing, then any absorbing set with syndrome support containing supp(s) is also not fully absorbing.
Proof. Let x + C be a coset with syndrome s that is not fully absorbing. Let O be the set of check nodes in the support of s. If x + C has no absorbing sets, then by definition, there exists v ∈ V such that v has at least as many neighbors in O as in W \ O. Similarly, if x + C has an absorbing set D, because it is not fully absorbing, there exists v ∈ V \ D such that v has at least as many neighbors in O as in W \ O. In either case, there exists v ∈ V such that v has at least as many neighbors in O as in W \ O. Now, let F be an absorbing set with O(F) ⊇ O. Because O(F) ⊇ O, the variable node v has at least as many neighbors in O(F) as in W \ O(F). So F is not fully absorbing.

VI. Conclusions
This paper provided an analysis of absorbing sets in binary linear block codes using the structure of the standard array and syndrome weights. This included upper bounds on the syndrome weights of cosets containing support vectors of fully and low-weight absorbing sets. We showed that sets of absorbing set support vectors that occur in cosets are translates of subspaces of the code, and when the subspace is the whole code, the vectors in such cosets are precisely the support vectors of fully absorbing sets. We provided a search algorithm for finding fully absorbing sets using these results. This work gives a new avenue for finding absorbing sets via subspaces of the code, and leads to many interesting questions that will be pursued in future work, such as how to predict when a subspace of the code may be translated to absorbing set support vectors, and how to find these sets.
