Variable-Weight Block Dual-Diagonal Structure for Low-Rate QC LDPC Codes With Low Error Floors
Hosung Park , Member, IEEE, Hee-Youl Kwak , Seokbeom Hong, Jong-Seon No , Fellow, IEEE, and Dong-Joon Shin , Senior Member, IEEE

Abstract— Irregular quasi-cyclic (QC) low-density paritycheck (LDPC) codes with the block dual-diagonal (BDD) parity structure are widely adopted in many communication standards because the BDD structure supports an efficient encoding and many degree-2 variable nodes inside are adequate for the construction of mid- to high-rate codes. However, we observe that low-rate irregular QC LDPC codes with the BDD parity structure inherently contain too many degree-2 variable nodes and suffer from error floors in high signal-to-noise ratio (SNR) region. In this paper, a generalized BDD structure including double-weight circulants as well as circulant permutation matrices is proposed for low-rate irregular QC LDPC codes with low error floors which is achieved with a little bit giving up error performance in the waterfall region. When constructing the parity part of a code with the generalized BDD structure, the portion of doubleweight circulants is variable so that the resulting LDPC code can achieve a desired degree distribution including degrees 2 and 3 while supporting the efficient encoding. We show that low-rate QC LDPC codes constructed with the proposed BDD structure have better theoretical properties and lower error floor than those with the conventional BDD structure.
Index Terms— Block dual-diagonal (BDD) structure, circulant matrices, efficient encoding, error floor, girth, minimum Hamming distance, protograph, quasi-cyclic (QC) low-density parity-check (LDPC) codes.

I. INTRODUCTION
L OW-DENSITY parity-check (LDPC) codes [2] have become dominant in recent years for error control in high-speed communication systems because of their capacityapproaching performance and the ease of implementation for parallel decoding. Most practical LDPC codes developed so far usually have mid- to high-rates [3]–[6]. Recently, the need for the construction of low-rate LDPC codes in practice arises for reliable communications [7].
For reliable communications, low-rate LDPC codes have to overcome the error floor problem [8] to achieve low error rates in high signal-to-noise ratio (SNR) region. The error floor phenomenon is closely related to the minimum Hamming distance and some combinatorial structures such as trapping sets [8] and stopping sets [9]. Codes with a small minimum Hamming distance may lead the decoding process to a wrong codeword or trap the decoder. Also, stopping sets and trapping sets of small size can hinder a channel output vector from converging to the correct codeword instead of a pseudo codeword [10] in the binary erasure channel (BEC) and the additive white Gaussian noise (AWGN) channel, respectively. Therefore, it is important to increase the minimum Hamming distance and the size of minimal stopping sets or trapping sets in designing LDPC codes for low error floors.
Most LDPC codes in practice have the quasi-cyclic (QC) property because QC LDPC codes are well suited for hardware implementation using simple shift registers. In particular, irregular QC LDPC codes with the block dual-diagonal (BDD) parity structure are widely adopted in many communication standards such as IEEE 802.16e [3], 802.11n [4], ITU-T G.hn/G.9960 [5], and the core part of 5G LDPC codes [11]. The BDD structure consists of mostly degree-2 variable nodes, which is suitable for a code to have a good degree distribution for mid-to-high rates, and also enables the efficient encoding of LDPC codes [12], [13].
When a rate of a QC LDPC code with the BDD structure is given, the fraction of degree-2 variable nodes in the parity-check matrices cannot be below a certain value. This restriction induces too many degree-2 variable nodes when the code rate is quite low. A large portion of degree-2 nodes can cause low error floors in that the extrinsic message degree (EMD) [14] of a cycle including many degree-2 nodes has a small value, the minimum Hamming distance

It is also noted that the QC LDPC codes in [18] are constructed to avoid too many degree-2 variable codes in the parity structure and empirically shown to have larger values of upper bound on the minimum Hamming distance than the conventional BDD structure.
There have been several research works on designing good parity structures for QC LDPC codes to support efficient encoding. In [18], a class of irregular QC LDPC codes with triangular plus dual-diagonal structure is proposed. The main advantage of these LDPC codes is that the variable nodes in the parity part of parity-check matrices can have various degrees. It is noted that a systematic way of assigning shift values are not provided and only single-weight circulants are used to construct the QC LDPC codes. In [19], irregular QC LDPC codes with triple-diagonal parity structure are proposed. Since the triple-diagonal structure imposes too many degree-3 variable nodes, the flexible design of LDPC codes with respect to the degree distribution is restricted. Another class of QC LDPC codes with an approximate lower triangular parity structure is proposed in [20]. The parity part of the parity-check matrices has the same degree distribution as the BDD structure and these QC LDPC codes show good error performance in waterfall region. In [21], a modified BDD structure is proposed but the restriction on the fraction of degree-2 variable nodes still exists. In [22], a block triplediagonal parity structure with all nodes of degree three is proposed for QC LDPC codes. It is based on single-weight circulants and the degree distribution is restricted due to too many degree-3 nodes.
In this paper, a generalized BDD parity structure is proposed for low-rate irregular QC LDPC codes with low error floors which is achieved with a little bit giving up error performance in the waterfall region. By adopting double-weight circulants, the generalized BDD structure can have flexible variable node degree distributions including degrees 2 and 3. We propose an algorithm to determine the shift values and the locations of the double-weight circulants in the proposed BDD structure in order to support the efficient encoding and maximize the local girth. By properly adjusting the fraction of degree-2 variable nodes, QC LDPC codes constructed with the proposed BDD structure are shown via simulations to have lower error floors than those with the conventional BDD structure. This result is supported by two theoretical values. The typical minimum distance ratio [23] and the upper bound on the minimum Hamming distance [16] of QC LDPC codes with the proposed BDD structure are shown to be larger than those with the conventional BDD structure.

In particular, (J − 1)z variable nodes have degree 2 and z variable nodes have degree 3 in Hp. The fraction of degree-2 variable nodes for rate-R QC LDPC codes with the conventional BDD structure cannot be less than (1 − R) (1 − 1/J) and it becomes very large for small R and large J.
On the contrary, the generalized BDD structure provides the flexibility in controlling the fraction of degree-2 and degree-3 variable nodes by adjusting n2. In this paper, we concern about low-rate QC LDPC codes with too many degree-2 variable nodes and we try to reduce the fraction of degree-2 variable nodes. So we assume that there is no degree-2 variable node in the message part of the QC LDPC codes. Thus the generalized BDD structure provides more flexibility in adopting good degree distributions for QC LDPC codes because good irregular QC LDPC codes generally have many degree-3 variable nodes as well as degree-2 variable nodes as seen in the optimal degree distributions [15] or many standards [3]–[6].

B. Construction of Hp to Maximize Local Girth
It is experimentally shown in [12] that the girth of the subgraph consisting of only variable nodes of low degree (i.e., degrees 2 and 3) in H is desired to be large for good error performance in the error floor region. This observation agrees with the concept of the approximate cycle extrinsic message degree algorithm [14] and its developed versions [24]–[26] that are used to avoid harmful stopping sets and trapping sets. In this literature, for the given S in Theorem 1 or 2, we aim to maximize the girth of Hp by properly determining the ordering of the elements in S and ri, i = 1, · · · , n2, to have low error floor for QC LDPC codes with the proposed BDD structure.
We first briefly investigate how protographs and shift values have an influence on the generation of cycles in Tanner graphs of QC LDPC codes with a graph theoretical approach. Let G = (V, E) denote a graph with a set of vertices V and a set of edges E. Let vi (ei) represent a vertex (an edge) in V (E). A walk is an alternating sequence of vertices and edges v0 e1 v1 · · · vn−1envn, where the vertices vi−1 and vi are the endpoints of the edge ei. The length of a walk w, denoted by l(w), is defined as the number of edges in w. A walk is closed if vn = v0. A walk is non-reversing if ei = ei+1 for i = 1, 2, · · · , n − 1. A non-reversing closed walk is said to be tailless if en = e1.

The girth of a QC LDPC code is clearly determined by the structure of the protograph, lift size, and all shift values assigned to edges according to Lemma 1. However, we can derive an upper bound on the girth of QC LDPC codes lifted from a given protograph without considering the lift size and shift values based on the concept of inevitable cycles.
Definition 1 ( [29], [30]): An inevitable cycle induced by a protograph is defined as the shortest one of the cycles which should appear in the QC LDPC codes lifted from the protograph regardless of the lift size and shift values.

V. CONSTRUCTION OF QC LDPC CODES WITH THE PROPOSED BDD STRUCTURE
In this section, QC LDPC codes with the proposed BDD structure are constructed to have low error floors when J, L, z, and degree distributions in H are given. Definitely, the parity part is constructed by the PPC algorithm in Algorithm 1. For the message part, Hm should be carefully constructed so that the submatrix of H consisting of the columns of weight 2 and 3 can have a sufficiently large girth [12], [14], [24], [25].
In [12], a greedy girth-maximizing algorithm for the construction of good QC LDPC codes with the conventional BDD structure is proposed. The basic idea is that Pm and its shift values are determined stepwise so that the girth of the submatrix consisting of the columns of the lowest weight in H is firstly maximized and then the same procedure is repeatedly performed for the submatrix with the columns of the next lowest weight. This algorithm is called as conventional greedy girth-maximizing (CGGM) algorithm in this paper to distinguish it from the proposed algorithm to appear next.
For QC LDPC codes with the proposed BDD structure, we propose a new greedy girth-maximizing algorithm to construct P, which will be called PGGM algorithm from now on, and assign shift values to its edges. The basic idea of the PGGM algorithm on the construction of the message part is similar to that of the CGGM algorithm. It is assumed that Hm only consists of circulant permutation matrices and zero matrices and its column weights are larger than or equal to 3. After constructing Pp and Hp by the PPC algorithm in Section IV, the part of the lowest column weight in Pm is firstly constructed not to induce the inevitable cycles of short length in H. In designing this part, many subgraph patterns inducing the inevitable cycles of short length should be avoided as shown in [30]. Then, shift values are assigned to edges in the constructed protograph to have large girth in H in a greedy manner. These processes are repeated for the part of higher column weight step by step. The PGGM algorithm is described in Algorithm 2.

VII. CONCLUSION In this paper, a new BDD structure is proposed for low-rate irregular QC LDPC codes to achieve low error floors with a little bit giving up the error performance in the waterfall region. By adopting double-weight circulants, the parity part of parity-check matrices can have flexible degree distributions including degrees 2 and 3, which enables us to avoid too many degree-2 variable nodes in the parity-check matrices of lowrate QC LDPC codes. The proposed BDD structure can support the efficient encoding of QC LDPC codes by determining the set of shift values appropriately. The PPC algorithm based on graph theoretical approach is proposed, which maximizes the local girth of the proposed BDD structure by determining the locations of double-weight circulant in the proposed BDD structure. The PGGM algorithm is provided to constructing irregular QC LDPC codes with the proposed BDD structure. Also, the simplified EMPEG algorithm is considered to

It is shown via numerical analysis that the QC LDPC codes with the proposed BDD structure have better typical minimum distance ratio, upper bound on minimum Hamming distance, and error performance in the error floor region than those with the conventional BDD structure.
To show the potential of large minimum Hamming distance of the proposed BDD structure, we used the upper bound on minimum Hamming distance and the typical minimum distance ratio. To be more precise, we need to derive a tight bound on minimum Hamming distance and we may adopt the method in [33] and [34]. All these works assumed that there is no parallel edges in the protograph which is not the case of this paper. All these analyses are good future works of this paper.
