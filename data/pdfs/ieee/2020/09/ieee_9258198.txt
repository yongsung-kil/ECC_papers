Chen-Xuan Wang and Cheng-Hung Lin Department of Electrical Engineering, Yuan Ze University, Jungli, Taiwan 32003, R.O.C.

Abstract-- The hardware design of the probabilistic min-sum algorithm can be efficiently realized by group searching. Although it can accelerate calculation and reduce hardware complexity, it degrades error correction capability. Thus, an improved method for simplified min-sum algorithm is proposed in this paper. In order to recover the error correction capability, we apply the accurate first minimum for adjusting the reliability of the probabilistic second minimum. Maintaining the value of the first minimum and proportionally combining the first two minima can flexibly improve the reliability of probabilistic second minimum. Compared with the original simplified min-sum algorithm, the improved method not only gains the efficient hardware design but also obtains the better error correction capability.

I. INTRODUCTION
As technique advances, the error correction capability is attached great importance. Low-density parity-check (LDPC) code [1] which possesses forced error correcting capability and fully parallel decoding architecture was proposed by Gallager in 1962. Currently, LDPC codes have been introduced in different communication standards, such as DVB-S2, IEEE 802.11n, and IEEE 802.16e.
In LDPC decoding, the layered messages are alternately transmitted and updated among the variable nodes and check nodes using the iterative belief propagation decoding. Although LDPC codes can achieve required performance, hardware realization and decoding latency are critical problem. In order to alleviate the problem of hardware implementation that causes the computational delay in decoding process, the probabilistic min-sum algorithm (PMSA) proposed in [2] can efficiently relieve the burden of hardware complexity and area cost, and accelerate the decoding speed. Nevertheless, the PMSA only obtains the probabilistic second minimum. This means that the difference between the exact first and second minima probably increases. This leads to the decoding performance degradation that makes the unsatisfied bit-errorrate (BER) requirement. Hence, the work in [3] was proposed to adjust the second minimum with the fixed negative factor to gain better BER performance than that of [2]. Although the work in [3] can efficiently improve the BER performance, it is not suitable for different communication standards with distinct matrix properties. The considerable problem in [3] is how to gain the best factor for the probabilistic second minimum in different standards.

II. PROBABILISTIC MIN-SUM ALGORITHM (PMSA)
In order to achieve a good decoding performance, numerous comparators were proposed to obtain the first two minima. Nevertheless, plenty of operations of comparators prolong the computational latency in decoding process. Therefore, the proposed work in [2] adopts tree structure algorithm using departed groups for minima finder to efficiently reduce the amount of comparators, as shown in Fig. 1. It successfully reduces hardware implementation complexity and accelerates decoding speed. However, the tree structure algorithm with departed groups only determines a accurate first minimum and a probabilistic second minimum. The gap between these two minima in NPMSA is larger than that in NMSA. The difference of gap significantly degrades the error correction capability. It becomes a critical problem to achieve a good balance between improving hardware implementation and maintaining error correction capability.

V. CONCLUSION
INPMSA was proposed to revise the probabilistic second minimum for improving decoding performance by flexibility, simplifying the hardware design by using [2], and providing commonality in different communication standards. The paper focused on the adjustment and the restriction of the second minimum for improving decoding capability. Compared with [3], INPMSA not only can retain the original hardware advantages, but also can obtain similar decoding capability. By clear definition of the weighting factors, the possibility of reversal problem can be completely solved. Compared with [4], the proposed can maintain generality in different standards.
