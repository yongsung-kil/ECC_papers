Abstract—In this paper, parallelly concatenated spatially coupled RA (PC-SCRA) codes was presented based on spatially coupled RA (SCRA) codes with the advantages of simple coding structure, threshold saturation and low decoding complexity. When encoding, the same message bits are respectively encoded by two same SCRA codes to obtain respective check bits. During transmission, the message bits and the obtained parity bits are merged into one codeword for transmission. Besides, density evolution algorithm was proposed to compute the iterative decoding threshold of PC-SCRA code ensemble over the binary erasure channel. Simulation results show that the proposed PC-SCRA code ensemble has excellent BP threshold performance close to Shannon limit and is superior to the spatially coupled LDPC (SC-LDPC) code with same coderate, which provides a new way for construrting capacity-approaching SC-LDPC codes.
Keywords—spatial coupling; RA code; LDPC code; iterative decoding
I. INTRODUCTION
In recent years, spatially coupled low density parity check (SC-LDPC) code has attracted much attention due to its capacity approaching performance. In 2011, Kudekar et al. proved that the convolutional LDPC codes constructed from a block LDPC code can improve the belief propagation (BP) threshold up to the maximum a posteriori (MAP) threshold of the underlying code over the binary erasure channel (BEC) and the binary-input memoryless symmetric-output (BMS) channels and named this phenomenon as threshold saturation[1][2].
How to design SC-LDPC codes with better thresholds has always been a research hotspot. In [3]-[5],Truhachev et al. proposed a method to connect two or more spatial coupling chains to construct SC-LDPC codes by adding extra edges to variable nodes at special positions. In [6], the authors proposed a method of constructing SC-LDPC codes by introducing a check node of degree 2 at each

coupling position to connect two or more coupling chains with the same coderate, and pointed out the constructed SC-LDPC code has excellent performance especially against burst errors. In addition, a method of connecting two or more coupled chains with different coderate in parallel to construct an SC-LDPC code with a larger coderate range through edge exchange was proposed in [7].
In this paper, we follow a different way and consider the parallelly concatenated design method based on spatially coupled repeat accumulate (SCRA) codes. Compared with SC-LDPC code, SCRA code has the advantages of simple encoding structure, better threshold and low decoding complexity[8]. Besides, SCRA code has regular degree distribution (DD) structure and excellent threshold performance, so it is a good choice as a component code to construct concatenated code. When encoding, two identical SCRA codes are used to encode the same message bit to get respective check bits. During transmission, the message bit and the two check bits are merged into one codeword for transmission. Simulation results show that the proposed PC-SCRA code ensemble has excellent threshold performance and is better than SC-LDPC codes with the same coderate.

VI. CONCLUSION
Considering the advantages of SCRA code with simple coding structure, excellent threshold, and low decoding complexity, the PC-SCRA code ensemble is proposed in this paper. Density evolution analysis shows that the BP thresholds of the proposed PC-SCRA code ensemble over the BEC are very close to Shannon limits and are better than the SC-LDPC code ensemble at the same coderate, which provides a new way for constructing the capacity achieving SC-LDPC codes with simple encoding structure. An interesting phenomenon is that all codewords obtained by random puncturing of the PC-SCRA code ensemble are very close to the capacity limit, which also provides a new way for designing rate compatible SC-LDPC codes.
