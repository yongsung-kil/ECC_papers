On the Class of High-Rate QC-LDPC Codes With Girth 8 From Sequences Satisfied in GCD Condition

Marjan Majdzade and Mohammad Gholami

Abstractâ€” Recently, a class of (J, L) quasi-cyclic (QC) low-density parity-check (LDPC) codes with girth eight is constructed based on the greatest-common-divisor (GCD) condition. For L = 3, an equivalence between the sequences generated by a greedy algorithm satisfying in GCD condition and a known class of integer sequences, called Stanley sequences has been proposed as an open problem. In this letter, we solve this problem in a more general case by introducing the class of 3-free sets as a generalization of Stanley sequences and showing an equivalence between 3-free sets and the sequences satisfied in GCD condition. Then, a new algorithm is proposed to find 3-free sets efficiently usually having larger size than the known methods, leading to column-weight 3 QC-LDPC codes with smaller lengths and better 8-cycle distributions. In addition, a new explicit formula is proposed to construct Stanley sequences which results in a class of girth-8 column-weight 3 QC-LDPC codes with high rates. The protograph QC-LDPC codes lifted from the constructed base matrices outperform progressive-edge-growth (PEG), randomlike and some recent QC-LDPC codes with the same girth.
Index Termsâ€” QC-LDPC codes, 3-free sets, stanley sequences, girth, exponent matrix.
I. INTRODUCTION
L OW density parity-check (LDPC) codes are a main class of capacity-approaching codes which can be described by their sparse parity-check matrices (PCMs). One of the most important models to represent an LDPC code with PCM H, is Tanner graph [1], TG(H). The length of the shortest cycle of the Tanner graph, girth, has been known to influence the code performance and the error correction and in general, LDPC codes with large girth and small number of short cycles have good performances [2], [3], so lots of efforts have been put into constructing codes with large girth [4]â€“[15].
Quasi-cyclic (QC) LDPC codes [6] are an important class of LDPC codes that are preferred rather than other types of LDPC codes because of their practical and simple implementations. By a (J, L) QC-LDPC code with CPM-size (Circulant Permutation Matrix-size) P , we mean a linear code whose PCM is a J أ— L array of CPMs of size P . In fact, a (J, L) QCLDPC code can be described by a CPM-size P and a J أ— L matrix, called exponent matrix, of some non-negative integers less than P . B. 3-Free Sets
An arithmetic progression (AP) sequence is a sequence
of numbers such that the difference between the consecutive terms is constant. Let N0 be the set of nonnegative integers. By a 3-free set {a0, a1, آ· آ· آ· , aJâˆ’1}, a0 < a1 < آ· آ· آ· < aJâˆ’1, we mean a subset of N0 containing no AP sequences with three terms, it means that it does not contain any three elements ai < aj < ak with 2aj = ai + ak. For instance, {0, 1, 3, 4, 9, 10, 12, 13, 27} is a 3-free set of size 9. There has been much work [16]â€“[18] on the following question: given n, how large can be the size of a 3-free set contained in {0, آ· آ· آ· , n âˆ’ 1}, which is remained as an open problem.
Interestingly, the following main theorem gives an equiv-
alence between 3-free sets and sequences satisfied in GCD
condition. Theorem 1: For L = 3, the sequence a0, a1, آ· آ· آ· , aJâˆ’1 is
satisfied in GCD condition if and only if {a0, a1, آ· آ· آ· , aJâˆ’1} is a 3-free set.
Proof: To prove the claim, we show that (ak âˆ’ ai)/ gcd(ak âˆ’ ai, aj âˆ’ ai) â‰¤ 2 if and only if 2aj = ai + ak.

Then, a new algorithm was presented to generate 3-free sets having larger size (in some cases) than the last achievements in [4], [7], [13], [18].
