EXIT Chart Analysis of Protograph-based LDPC Codes for Unequal Power Two-User Gaussian Multiple Access Channel

Sirawit Khittiwitchayakul, Watid Phakphisut, Pornchai Supnithi, and Lin Min Min Myint Faculty of Engineering, King Mongkut's Institute of Technology Ladkrabang, Bangkok 10520, Thailand.
Email: {57601155, watid.ph, pornchai.su, linminmin.my}@kmitl.ac.th

Abstract: In this work, we analyze the theoretical performances of protograph-based LDPC codes for unequal power two-user Gaussian multiple access channel (GMAC) by using Extrinsic information transfer (EXIT) chart. We show that the decoding performances of LDPC codes for twousers are improved significantly when the transmitted power of user increases. Moreover, we present the design of protograph structures of the LDPC codes for unequal power two-user GMAC by using the differential evolution (DE). The results show that the designed LDPC codes outperform the traditional LDPC codes.
Keywords: LDPC codes, 2-user GMAC, P-EXIT chart, Decoding threshold, Differential evolution
1. Introduction
Low-density parity check (LDPC) codes [1] are a class of modern channel coding. Nowadays, the LDPC codes are employed in varied systems, such as the magnetic recording (MR) [2], and wireless communications [3, 4]. The Gaussian multiple access channel (GMAC) [5] is one of practical channel in the wireless communication systems. The GMAC consists of two or more sources which transmit the information to a common receiver. In the prior work [6], EXIT chart technique is modified to analyze the performance of the LDPC code for two-user GMAC. Moreover, the code design methodology of LDPC code is introduced. However, the researchers in [6] consider the two-user system with equal power only. The Gaussian mixture (GM) model has been proposed in [7] to calculate the state-to-variable messages. In [8], the EXIT chart technique is modified by removing the equal power constraint, the optimization of LDPC codes over the unequal power two-user GMACs is presented.
In general, LDPC code can be constructed by several approaches. In [9], the protographs are used to generate the QC-LDPC codes which can provide the excellent error performance as well as the fast encoding and decoding. Therefore, in this work, we propose to use the protographbased LDPC codes for unequal power two-user GMAC. The EXIT chart techniques will be modified to analyze the theoretical performances of two-user protograph LDPC codes. In addition, we will design the protograph structure of LDPC codes by using the differential evolution (DE)

algorithm [10]. The results show that the designed LDPC codes have better performances than the traditional LDPC codes.
In the rest of article, we briefly review the LDPC codes constructed from the protograph in Section 2. In Section 3, we describe how to use the EXIT chart for the protographbased LDPC code in two-user GMAC. The design of LDPC codes and their performance evaluations are presented in Section 4. Finally, the simulation results and conclusion are expressed in Section 5 and 6, respectively.
2. Background
2.1 Protograph-based LDPC codes
The LDPC codes can be constructed from a small bipartite graph, known as “protograph” [9]. A protograph can be written in term of base matrix B whose size is m × n, where m and n are the number of check nodes and variable nodes in the protograph, respectively. The parity-check matrix H with various sizes can be obtained through replacing non-zero and zero elements of B by the N × N permutation and zero matrices, respectively. The protographs impose a structure on LDPC code, which can facilitate the design of fast encoder/decoder and efficient LDPC decoding.

5. Simulation results
In this section, the BER performances of proposed codes are presented. Herein, we use the constructions of LDPC code presented in (9) and (10). In addition, the progressive edge growth (PEG) algorithm [14] is used to create the paritycheck matrix (H) from the based matrix (B) by the expansion factor of 500. In Fig. 4, we show the simulation results for the case of P1 = 1.5 and P2 = 1. The simulation results can confirm the theoretical results in Fig. 3 which the proposed codes can perform better than the previous code for both users. Consider at the 1st user, the proposed code provide about 0.5 dB coding gain compared to the previous code at BER = 10-4.

6. Conclusions
In this work, we present the EXIT chart of protographbased LDPC code for unequal power two-user GMAC. We found that the decoding thresholds of 1st user are lower than that of 2nd user when the power P1 > P2 . Whenever we increase the power P1, both the decoding thresholds of user 1 and 2 will decrease. Furthermore, the protograph structures of LDPC codes for unequal power two-user GMAC are well designed. The analytical results show that the decoding thresholds of designed LDPC code will be lower than those of previous LDPC code for both 1st and 2nd users. Our simulation results can confirm the theoretical results which the proposed code outperform the previous code about 0.5 dB coding gain.
