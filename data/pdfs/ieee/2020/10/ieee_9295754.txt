Abstract—To improve the decoding speed and reduce the complexity of low-density parity-check (LDPC) codes, this paper proposes a hybrid LDPC decoding algorithm that combined with the distribution characteristics of log-likelihood ratio (LLR) of all variable node information before each iteration. During the process of decoding, there is a judgment based on an interval value and a threshold value to select the Belief Propagation (BP) algorithm or the Minimal Sum (MS) algorithm to decode, so the proposed hybrid algorithm effectively combines the excellent performance of BP algorithm and the simple operation of MS algorithm. The simulation results show that the algorithm can greatly reduce the computational complexity and achieve similar decoding performance compared with the BP algorithm.
Index Terms—LDPC; log-likelihood ratio; distribution characteristics; interval value; threshold value;
I. INTRODUCTION
As a class of linear block codes, Low-Density Parity-Check (LDPC) codes were proposed by Dr. Gallager [1], which were close to the Shannon limit with infinite length [2]. It has the characteristics of an efficient parallel iterative decoding algorithm and a very Low error level. In particular, the use of structured LDPC codes provides low decoding complexity while ensuring throughput, which is suitable for a variety of parallel decoding structures and beneficial to hardware implementation. Therefore, the LDPC codes have been widely adopted in several industrial standards such as IEEE802.11 (Wi-Fi) [3], IEEE802.16e (WiMAX) [4], 5G [5]. Besides, LDPC codes have been used in satellite television, space communications and other fields.
The Belief Propagation (BP) algorithm [6] is one of the classical LDPC code decoding algorithms. Although the BP algorithm has excellent decoding performance, it requires complex computations. Therefore, more attention should be paid on reducing the implementation complexity and improve the decoding throughput. Minimal Sum (MS) algorithm [7] was generated to optimize the above problem by updating process of approximating variable nodes, but sacrificed part of performance. Then, Normalized Minimal Sum (NMS) algorithm [8] achieves the performance close to the BP algorithm by adding a multiplicity factor, whose value has a great impact

on performance. Then, an improved decoding algorithm [9] based on the danger zone of variable node information value was proposed for dynamic selection, which could apporach the similar decoding performance as BP algorithm. However, due to the strict selection conditions, BP algorithm is still mainly used for decoding iteration in [9], which means very little reduction in complexity.
In this paper, we deeply analyzed the approximate errors of the MS algorithm at different intervals. Using distribution characteristics of log-likelihood ratio (LLR) for variable node information, we set up a parameter group to select which decoding method will be used before each iteration. Then, the decoding performance and computational complexity can effectively balanced.
The rest of the paper is subdivided into four sections: Section II introduces the basic principles of BP algorithm and MS algorithm briefly. An improved hybrid decoding algorithm is proposed in Section III by analyzing the approximate error of MS algorithm in detail, and the determination of core parameters are discussed; The simulation result is described in section IV, and the conclusion is presented in section V.
II. BASIC PRINCIPLES OF BP ALGORITHM AND MS ALGORITHM
A. Belief Propagation Algorithm
Assume (N, K) LDPC code is characterized by a sparse parity check matrix H, N is the number of code length and K is the number of message bits.
Taking binary LDPC code as an example. The output sequence of the LDPC encoder is x = [x1, x2 . . . xi], where i = 1, 2, · · · , N . In our work we consider that the LDPC codes are modulated using BPSK modulation and transmitted over an Additive White Gaussian Noise (AWGN) channel. Before transmission, the codewords is mapped to a signal constellation u to obtain the vector. u = [u1, u2 . . . ui]T . At the receiver, after the demodulator we get the received vector y = [y1, y2 . . . yi]T , where, yi = ui + ni, ni is a random variable with mean zero and variance of N0/2.

D. Analysis of Parameter Group (a, b)
The interval value a and threshold value b jointly determine which algorithm to use in this iteration, and affect the complexity and decoding performance in the whole decoding process. In the limiting case, when a = 0 and b = 0, the hybrid decoding algorithm evolved into the MS algorithm; when a = +∞ and b = ∀, the hybrid algorithm evolved into the BP algorithm. In theory, the range of a is [0, +∞).

A. Simulation Results
In this paper, we use the codewords of 1/2 and 2/3 (B) code rate in the IEEE 802.16e standard [4]. By extending their basis matrix by 24 times and 48 times respectively, we can obtain four types of non-regular LDPC codes with code lengths of 576 and 1152 respectively. Set the maximum number of iterations to 30, the maximum number of frames sent to 60,000, and the maximum number of error frames to 60. we simulate and compare the hybrid algorithm in this paper with the BP algorithm, the MS algorithm and the algorithm in [9]. Typical parameter group (2, 0.09) and (1.4, 0.06) are selected for r = 0.5 and parameter tuples (1.1, 0.026) and (0.86, 0.02) were selected for r = 0.66. Performance simulation results of four types of LDPC codes are shown in Fig.6 – Fig.9.
According to the decoding performance curve, at different code lengths and rates, we can find that when BER = 3×10 - 5, the proposed hybrid algorithm decoding performance with the original BP algorithm and the algorithm of literature [9] keep almost the same, the average gap is less than about 0.08 dB. Beside, when compared with the original MS algorithm, its performance can increase by about 0.2 dB on average, so it can be seen that performance improved significantly.

V. CONCLUSION
In this paper, we propose a hybrid LDPC decoding algorithm by deeply analyzing the theoretical approximation error of the MS algorithm and studying the probability distribution characteristics of variable node information LLR. Besides, the value of interval and threshold are discussed by parametric analysis. This paper analyzes the performance and computational complexity of LDPC codes with different code lengths and rates using the hybrid algorithm under the condition of relatively high SNR. Simulation results show that the hybrid decoding algorithm can reduce the computational complexity of multiplication and division by a large margin, and improve the decoding speed.
