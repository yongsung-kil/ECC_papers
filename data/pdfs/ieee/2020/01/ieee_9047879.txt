ABSTRACT The retention noise in MLC NAND flash memory has a great impact on reliability and lifetime of devices. How to recover information from the data corrupted by retention noise is a challenging problem. The joint processing of channel estimation, detection and low-density parity-check (LDPC) decoding is an effective method to handle this problem, while which is hindered by the limited computational resource of NAND flash memory devices. This paper proposes a discrete retention-failure recovery method, where the reading system only processes integers. The proposed method consists of two major components referred to binary tree searching (BTS) based channel updating and information bottleneck (IB) based LDPC decoding. In channel updating, the proposed learning based BTS algorithm can acquire reference voltages directly; thus it bypasses the intermediate stage of retention noise parameter estimation. The LDPC decoder is realized by IB based message passing, where node operation is achieved by lookup tables, and the messages passed between nodes are integers. Besides, a dynamic quantization level reduction strategy for IB based decoding is proposed to reduce the memory consumption of lookup tables. Finally, using PEG-constructed regular LDPC code (8000, 7200), the numerical results show that the proposed discrete method has much lower computational complexity and similar performance compared with the conventional method.
INDEX TERMS Retention noise, flash memory, information bottleneck, decision trees, computational complexity.

I. INTRODUCTION NAND flash memory is widely used in digital devices. But data in devices powered-off for a long time will suffer from serious noises, among which retention noise predominates [1], [2]. Because of the physical structure of memory cells, the charges inside the floating gates are escaping over time, which is the source of the retention noise [1]. Therefore, how to overcome the retention noise is an important problem in data recovery. On the other hand, prevalent NAND flash memory devices are usually powered by the battery and have limited computational resources, which means that the data recovery method should have low computational complexity and power consumption.
Some conventional retention noise recovery methods utilize the refresh mechanism, where data are frequently read and corrected by error control coding (ECC), then they are written back before the retention noise makes the error

In those methods, extra sensing operation is required, which increases the read latency and power consumption.
Another approach is channel estimation based retention error correction. Moreover, the accuracy of updated reference voltages depends on channel parameters. In decision-directed estimation (DDE) [9], channel parameters were estimated by the decision results of ECC. This method doesn’t need extra memory sensing but requires the correctly decoded data which may be hard to collect under serious retention noise. Detector parameter estimation (DPE)

[10] and RABP assisted channel update (RABP-CU) [11] are iteration based channel estimation methods, where they first constructed a cost function related to channel parameters, then got the estimation result by iteration optimizing algorithms such as gradient descent (GD). Although RABPCU does not need extra sensing operations, it requires a large number of iterations to converge to a proper result, which leads to high complexity. The optimizing algorithm DPE has fast converging speed, but DPE needs extra sensing operations and high quantity of quantization levels. In [12], the authors proposed a two-stage estimation (TSE) method, where the first stage performs the coarse estimation and the second stage tunes it finely. Although TSE has low computational complexity, it needs extra memory sensing operation. Moreover, these methods in [9]–[12] only acquire the channel parameters. The complex reference voltage design methods such as the mutual information maximization [13] are further required to compute the reference voltages based on these parameters.
On the other hand, the strong soft-decision ECC is an essential part of retention error recovery. The low-density parity-check (LDPC) [14] code is generally used as ECC in NAND flash memory, for it can significantly improve the error tolerance capacity compared with conventional Bose–Chaudhuri–Hocquenghem (BCH) [15] code. However, the complexity of float-point message passing is very high for flash memory devices; thus, it is usually replaced by simplified algorithms such as quantized decoder. But the uniform quantization in quantized decoder has usually negative impact on performance [16]. For example, the quantized min-sum decoder [17] has prominent performance loss compared with float-point sum-product algorithm (SPA) decoder.
Recently, Lewandowsky and Bauch have proposed discrete sum-product LDPC information bottleneck (LDPC-IB) decoder [18]–[20]. LDPC-IB is based on information bottleneck (IB), which was first introduced by Tishby et. al. in 1999 [21]. The IB solves the problem that how to effectively extract the information x ∈ X from the related signal y ∈ Y by compressing Y . After that, several IB algorithmic approaches were proposed, such as agglomerative IB (Agg-IB), sequential IB (Seq-IB), deterministic IB (Det-IB) and KullbackLeibler-means IB (KL-means-IB) [22]. Since its invention, IB has widely applied in different fields, such as documents classification [23], neuroscience [24], deep learning [25] and LDPC decoding [18].
In LDPC-IB decoder, the message passing among nodes is achieved by lookup tables (LUTs), which are generated by the IB method offline. With those LUTs, the online decoding procedure only needs to deal with unsigned integers. Compared with the float point decoding, LDPC-IB decoder can avoid float-point calculation and thus has low complexity, and meanwhile has nearly no performance loss. Therefore, it is a proper decoder implementation for NAND flash memory.
In this paper, our goal is to construct a low complexity discrete retention error recovery method, in which all

the processes, including the channel updating and LDPC-IB decoding, can be done only by using integers. In channel updating step, we propose a decision tree based method. In this method, the design of reference voltages is transformed to a classification problem, and the reference voltages are acquired directly without the complex reference voltage computation such as the mutual information maximization method in [13]. Especially, the classification and regression trees (CART) [26] algorithm is utilized here for the classification problem. The inputs of binary tree trained by CART are numbers of cell threshold voltages falling into different quantization levels with raw reference voltages. The output is the index of the selected set of reference voltages. In the LDPC-IB decoder, the LUT is obtained by IB for BPSK-AWGN channel, and a channel re-mapping strategy is given to process the mismatch between MLC flash memory channel and BPSK-AWGN channel. In summary, the contributions of this paper are listed as follows.
1) We propose a decision tree based method to acquire the reference voltages. This method can bypass the complex reference voltage design, and be implemented by simple binary tree searching.
2) In LDPC-IB decoder, a channel re-mapping strategy is proposed to solve the mismatch between MLC flash memory channel and AWGN-BPSK channel. Moreover, a dynamic LUT size reduction method is proposed to save the memory space.
3) The proposed discrete reading system shows good frame error rate performance. Concretely, using PEG-constructed regular LDPC code (8000, 7200), the proposed method has similar performance compared with the benchmark method consisted of TSE channel estimation and float-point SPA decoding.

III. DECISION TREE BASED CHANNEL UPDATING The key components of discrete retention error recovery system are channel updating and LDPC-IB decoder. In the decoding process, the sensing operations are performed with the raw reference voltages designed for the scenario of PE = T = 0. Then, the indexes of quantization intervals are fed to the LDPC-IB decoder. If the decoding successes, the reading process is terminated and outputs the decoded bits; otherwise, the retention error recovery is triggered.
The proposed discrete retention error recovery structure is shown in Fig. 2. We can see that in retention error recovery, the inputs of channel updating procedure are the count numbers of cell threshold voltages falling into different quantization intervals with raw threshold voltages. The channel updating outputs the index of the group of optimized reference voltages. Then, the sensing operation is performed with the new reference voltages, and the subsequent procedures are same as those in normal decoding.
In this section, proposed decision tree based channel updating is introduced, and LDPC-IB decoding will be introduced in next section.

As shown in Fig. 4, one read operation needs Nv reference voltages. Here Nv is set to 6 to achieve a good balance between performance and reading cost [11]. The Nv reference voltages in read operation are collected as v, i.e., v = [v0, v1, . . . , vNv−1]T .
In practical, read reference voltages should be chosen from a set consisting of some discrete values [30]. To reduce the complexity, all the candidate reference voltages should be clustered into several clusters, in which neighbored voltages are represented by the central reference voltages vc. Assuming the amount of vc is Ng, those vc can be stored in a table Tv, i.e., Tv = {vc0, vc1, . . . , vNc g−1}. So the reference voltage design can be viewed as a clustering and classification problem, where the channel information is the feature and the index of reference voltages is the label. The channel information here is defined as the count numbers of cell threshold voltages falling into different quantization intervals with raw threshold voltages during reading process within a block. Next, a decision tree is trained for channel updating.

Before the classification tree training, samples are clustered into Ng groups according to reference voltage vector v. The clustering algorithm employed here is K-means, which is a simple and effective cluster method. In K-means clustering, every sample is placed into the cluster where v has the least L2 distance to the centre of this cluster. The center of each cluster is updated as the mean of samples in this cluster during iterations, and in the first iteration the centers are set randomly. This iteration stops until the centers do not change or reach the max iterations. Then, all the reference voltage vectors in the same cluster are represented by the center of this cluster, and labeled by the index of the reference voltage vector associated with this center. After clustering, a training sample {c, v} can be represented as {c, v}, where v is the label of the cluster including v.
Note that a big Ng may not increase the prediction precision but make the tree overfitting, while a small Ng may lead to underfitting. Here, Ng is set to 50, which shows good balance from simulation tests.

In LDPC-IB decoder [18], the float-point calculation in sum-product algorithm can be replaced by LUTs, and the messages passed between variable and check noses are unsigned integers. In the following, we first introduce the LUT generation method based on BPSK-AWGN channel, then present a method to reduce the size of LUT dynamically. Finally, the channel re-mapping method is given to process the mismatching between MLC NAND flash memory channel and BPSK-AWGN channel.

A. LOOKUP TABLE GENERATION
A channel model is first required to provide probability values in LUT generation process [18]. The MLC NAND flash memory channel can’t be directly used for LUT generation, because the flash memory channel is hard to calibrate to guarantee the decoding performance. Besides, the channel parameters are not fixed due to various PE cycles and retention times and need to be adapted by the channel updating procedure, which means tables designed for a certain set of parameters may not suit the others. Here, BPSK-AWGN channel is selected for the table design, because it’s easy to analyze and calibrate. To bridge the gap between MLC NAND flash memory channel and BPSK-AWGN channel, a channel remapping method will be discussed in Section IV-C.

VI. CONCLUSION In this paper, we proposed a discrete NAND flash memory retention data recovery scheme consisting of channel updating and LDPC-IB decoding, which largely reduced the computational complexity and maintained high performance in the simulation using PEG-constructed regular LDPC code (8000, 7200). The channel updating is transformed into a classification problem, which is solved by CART trees. By this learning-based method, the counts of cell threshold voltages falling into different quantization intervals are directly mapped to reference voltages, which can avoid the complex computation of reference voltages in conventional methods. In the LDPC-IB decoder, the IB LUT was designed based on the BPSK-AWGN channel assumption, and a channel re-mapping strategy was given to process the mismatch between MLC flash memory channel and AWGN-BPSK channel. Moreover, a quantization reduction method was presented to reduce memory consumption. Using PEG-constructed regular LDPC code (8000, 7200), the simulation result shows that the proposed method has good FER performance with low computational complexity and mild memory consumption.
