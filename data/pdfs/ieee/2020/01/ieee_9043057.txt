AbstractŌĆöLow-density parity-check (LDPC) codes have previously been considered as combinatorial optimization problems (COPs) in respect to its decoding. However, after defining it as such, none have gone so far as to convert the LDPC code into a quadratic unconstrained binary optimization (QUBO) problem. Thus, a new method is created: one that converts the LDPC code to a QUBO problem, inputs the QUBO problem into Ising machines (computers based on the Ising model that are designed to solve the QUBO problem), obtains the QUBO solution and converts it to a LDPC solution. By utilizing an actual Ising machine, LDPC solutions with code length of 256-bits have been obtained with an accuracy of 93.9% by average annealing time 214.0ms. The benefit of this newfound methodology goes beyond its theoretical imprint of obtaining LDPC solutions more accurately. It has only been a few years since the Ising machine has been developed. Therefore, in formulating this method, one expands the currently scope of studies involving Ising machines, helping current and future researchers unlock its full range of capabilities and possibilities.

I. INTRODUCTION
Akin to how the LDPC code has evolved past its impracticality from its 1962 inception [1], Ising machines have been developed to satisfy an increasing demand in various fields such as artificial intelligence, machine learning, software engineering, and the like [2]. This demand is in regard to the COPs that are difficult for traditional computers to solve. In utilizing the invention and advancement of Ising machines, a new method to decoding LDPC codes is founded, one in which the LDPC solutions theoretically obtained would be more accurate than currently known methods.
Firstly, LDPC decoding will be defined as a COP and converted into a QUBO problem. Higher-order functions as Exclusive Oring (XORing) operations pose as constraints in this issue. However, variables are introduced to implement XORing and resolve the limitation. Other challenges exist that assume the answer to the QUBO problem as an unfeasible solution to the answer to the LDPC problem. To combat this, a range of hyperparameters is introduced as the coefficient of the constraint term (see Theorem 1) which proves that a feasible solution can be obtained with the hyperparameter within the proposed range. In comparison to the methods in Section II, decoding the LDPC problem in this way will increase the level of accuracy and allow for greater scalability.

II. RELATED WORKS
Previous studies such as [3]ŌĆō[7] reveal to have similarly converted COPs into QUBO problems. Furthermore, there are accepted methods in LDPC decoding such the sum-product decoding algorithm which uses the Bayesian estimation [8] and the bit-flip decoding algorithm [9]. This algorithm is a maximum a posteriori probability (MAP) estimation. However, when applied to a typical LDPC code, it gives an approximation and does not always provide the most precise result. There exist approaches that consider the decoding process of LDPC codes as a COP in [10], [11]. This decoding method, known as the interior-point method (IPM) or the barrier method, is based on linear programming and obtains an approximate solution. Though there are techniques for decoding relaxed LDPC codes using Ising machines [12] via hard-decision (inputting a binary vector from the received signals), the LDPC code is a softdecision decodable code, and as such, this method would likewise only serve approximate answers.
As the computational complexity of the LDPC code decoding problem is NP-hard, conventional methods will not deliver exact solutions. In this new method of solving the LDPC code decoding problem by strictly converting it into a QUBO problem for heuristically operating Ising machines, an exact solution to the LDPC code decoding problem is theoretically ensured if Ising machines minimize the objective function of the QUBO problem.
III. LDPC CODE DECODING PROBLEM
A finite length signal is sent and received over a noisy channel. Since erroneous information is sent when noise is added to a signal passing through a communication channel, it is necessary to perform error-correction coding on the signal. When the LDPC code is used as the error-correcting code, the received data is decoded to correct data. The computational complexity of this decoding process is NP-hard.

IV. LDPC CODE
The LDPC code is an efficient linear error-correcting code in which its error-correcting capability approaches the Shannon limit [13] when the code length is sufficiently large. The LDPC code is configured to be at constant zero when certain bits of the codeword are XORed together. To express this property, a sparse matrix called a parity check matrix is used.

C. Results
The correct solution was obtained in the decoding of a LDPC code that was of code length 256-bits. It was found that unary encoding yielded greater accuracy than binary encoding when the parity check constraint of the LDPC code decoding problem is encoded into the objective function of the QUBO problem (articulated in Tables IV and V). This is due to two factors: the number of satisfying states of the parity check constraint being larger for unary encoding than binary encoding and the ratio of the ground state to the solution space being higher.
VIII. CONCLUSION
The framework of this new LDPC code decoding method can be simply stated as the conversion of LDPC codes into QUBO problems. While others have come so far as to define LDPC codes as COPs, the idea of this kind of conversion has never been suggested. The findings of experiments noted in Section VII demonstrate the theoretical success of the method, while comparisons to previous solutions such as the sumproduct algorithm realize its true potential. The decryption issue has been solved, and LDPC code lengths of up to 256bits have been decoded.
In regards to the fuller analysis of this methodology, a range of hyperparameters are imposed and proven to involve the use of Ising machines. Theoretically, this method proves more accurate than the currently standardized LDPC decoding methods. By using these computers as the hardware central to providing solutions to the converted QUBO problems, this method contributes to unlocking their greater potential. The Ising machine is a recent invention and its technology will only progress with time. Finding this methodology not only creates a new pathway into decoding LDPC codes but can become a reference point for the expansion of Ising machine comprehension.
