Design of Multi-Edge-Type LDPC Codes for High-Order Coded Modulation
Lei Zhang and Frank R. Kschischang Edward S. Rogers Sr. Department of Electrical and Computer Engineering, University of Toronto
Email: {leizhang,frank}@comm.utoronto.ca

AbstractвАФA design method for bandwidth-efficient LDPC coded modulation for 22m-QAM constellations at rate (2m вИТ 1)/(2m) in complex AWGN is presented. A multi-edge-type parameterization is used to exploit the distinct bit-channel capacities unique to high-order modulation using LDPC structures. EXIT analysis is adapted to multi-edge by introducing multidimensional EXIT iterated-function system analysis. Under this conceptualization, a successful decoding condition is developed by estimating fixed points of the dynamical system using its numerical gradient. Optimized ensembles are found for 16QAM with thresholds matching the best known ensembles of equal complexity. For 64 to 1024-QAM, sufficiently high bitchannel capacities allow for extension of lower-order optimized ensembles, leading a practical nested code structure. The nested structure provides flexible rate selection with a single decoder. The gap to constrained, non-iterative capacity of all optimized code ensembles of maximum variable degree of 15 is within 0.21 dB.
I. INTRODUCTION
While the design of capacity-approaching low-density parity-check (LDPC) codes for the additive white Gaussian noise (AWGN) channel under binary phase shift keying (BPSK) modulation has been well studied (e.g., [1]), the application of LDPC codes to the equally important bandwidthefficient regime via high-order modulation has received far less attention. In [2], LDPC ensembles were optimized by applying the techniques in [1] to multi-level coding (MLC) [3] and bit-interleaved coded modulation (BICM) [4] to achieve thresholds within 0.1 dB of modulation constrained capacity. The optimized ensembles in [2] were structurally complex in implementation, while the differential evolution optimization used was heuristic-based and lack intuitive guidance for code designers. Subsequent work focused on reducing design and code complexity, such as Gaussian-approximated design [5], code structures based on irregular repeat-accumulate (IRA) and other protographs [6], [7], and non-binary codes [8]. However, the thresholds of these codes are lower than those of [2] in general.
The key distinction between BPSK and high-order modulation in AWGN is the unequal probability of error experienced by different bit levels in the signal constellation. For a fixed labelling scheme and demapper, the modulation constrained symbol capacity is composed of distinct bit capacities over the bits of each constellation point label [3]. A good inner LDPC code must match its different variable degrees to the bit channel capacities. We achieve this matching by using

multi-edge-type (MET) LDPC ensembles [9] which incorporate distinct channel output densities seamlessly into the ensemble parameterization. The extremely high-dimensional general parameterization of MET ensembles is specialized in our design by imposing judiciously chosen rules on edge-type assignment and right degree types.
Ensemble optimization uses the Gaussian-approximated extrinsic information transfer (EXIT) functions [10]. For the MET setting, we introduce the multi-dimensional EXIT iterated function system (IFS) concept for analysis and design. A successful decoding condition is developed based on fixed points of the dynamical system estimated by its numerical gradient. An optimization problem based on this condition is solved to find good degree profiles for 16-QAM at rate 3/4. For higher-order modulation with higher bit-channels reliabilities, the ensembles optimized for 16-QAM are extended by adding additional edge-types. Given very highquality bit-channels such as 1024-QAM, we found simple linear programming suffices in optimizing the code ensemble for rate 9/10, with decoding success.
The optimized ensembles possess a nested structure extending from rate 3/4 to 9/10. This property reduces practical implementation complexity for adaptive modulation applications where code rate and modulation adapt to different on channel conditions. A single decoder can be used across all required code rates.
Prior work on high-order LDPC coded modulation based on MET concepts are [5], [11], [12]. They fall short of providing the targeted MET parameterization, intuitive and efficient multi-dimensional EXIT analysis, and practical code structures for low-complexity implementation, at best-known thresholds.
In the sequel, Section II provides the high-order modulation system details and notation. Section III outlines the MET framework and our specialized parameterization. Section IV describes the central code optimization. Section V provides examples of optimized code ensembles for 16 to 1024-QAM modulation. Section VI provides conclusions.

C. High-order Extension
The optimization problem can be solved for 16 to 1024QAM. However, as the number of bit-levels increases with constellation size, the highest bit-levels become extremely reliable. The bit-channel mutual informations are estimated from the means of the empirical iid channel adapter output densities. Table I shows the mutual informations corresponding to Gaussian-approximations based on empirical density means. Note these are not actual bit channel capacities due to the Gaussian-approximation.
Note the similarity in Gaussian-approximated bit-channel mutual informations for the same bit level. Furthermore, as constellation size increases, the quality of higher bit levels improves drastically. At 64-QAM the highest bit-level has a Gaussian-approximated mutual information of 0.9994, and for 256 and 1024-QAM the upper levels are virtually error free. We caution that these approximations are based on the mean of empirical densities. The actual densities are not completely error-free as the mutual information suggests.
Nevertheless, from these two insights we propose to design higher-order code ensembles by extending optimized lowerorder ensembles. For example, assuming ensembles have been optimized using the method in Sec. IV-B for 16-QAM, to extend the code to 64-QAM, the same method can be applied

VI. CONCLUSION
In this paper we used multi-edge-type LDPC ensembles to design capacity-approaching codes for high-order bandwidth-efficient modulation. We introduced an intuitive multi-dimensional EXIT iterated-function system technique to analyze and optimize ensemble parameters with excellent results. Decoding convergence was conjectured to be governed by fixed points of the dynamical system and studied using a numerical gradient vector field technique. We believe these new analytical techniques are applicable not only to high-order modulation, but also to other LDPC applications involving distinct channel output densities.
A main strength of our optimized ensembles is its nested structure. A code family with similar properties was reported in [7], with significantly lower thresholds. Direct (non-nested) optimization of 64 to 1024-QAM ensembles using techniques of Sec. IV-B resulted in thresholds within 0.01 dB of the nested ensembles. Therefore, there is no significant performance loss due to the nested structure.
