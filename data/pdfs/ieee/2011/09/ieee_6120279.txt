Abstract—We propose a technique to design complexityoptimized bilayer lengthened LDPC codes by generalizing EXIT charts to bilayer codes and applying a method to estimate the number of decoding iterations. Using this approach, we have designed bilayer lengthened LDPC codes with a relay decoding complexity reduced up to 60% compared to the highest rate codes designed in the literature.
Index Terms—The relay channel, bilayer low-density parity-check (LDPC) codes, complexity-optimization, extrinsicinformation transfer (EXIT) charts, bilayer density evolution
I. INTRODUCTION
In recent years, cooperative communication has attracted significant attention as a class of strategies for achieving higher data rates by providing the communication system with a cooperative diversity gain. Relaying is one of the most widely used techniques of cooperation. A three node relay channel consists of a source node (S), a relay node (R), and a destination node (D) [1]. The idea of relaying is that the relay assists the destination by forwarding to it a function of the information which it receives from the source. Different relaying strategies for the relay channel are categorized based on whether the relay decodes the received signal from the source or merely forwards the received signal to the destination [2]. The decode-and-forward (DF) strategy is a relaying technique where the relay decodes the received signal from the transmitter and then transmits an encoded version of it to the destination. The classic decode-and-forward technique is based on random binning [3], where the relay decodes the source message and sends its bin index to the destination. The destination receives the bin index of the source message and using the entire message received in the previous block decodes the source message. The code design problem for the decode-and-forward strategy for relaying requires designing a code X at the source which is decodable at the relay, as well as designing a code X1 at the relay which is decodable at the destination. The destination first decodes X1 and with the help of the extra information gained by decoding X1 (i.e. the bin index of X), it decodes X [3].
One of the coding techniques which has recently been used to deploy the binning technique for the decode-and-forward strategy is designing bilayer low-density-parity-check (LDPC)

codes [4-7]. A bilayer LDPC code consists of one higher rate code optimized for the source-relay link as well as one lower rate code optimized for the source-destination link. The higher rate code is designed to be decodable at the relay and the lower rate code is designed to be decodable at the destination. Bilayer LDPC codes consist of two general classes of codes called bilayer lengthened LDPC codes and bilayer expurgated LDPC codes. In bilayer lengthened LDPC codes, the lower rate code is designed first and then the higher rate code is designed by keeping the total number of parity check nodes (k) constant and increasing the length of the code (n), while keeping the parameters of the lower rate code unchanged. In bilayer expurgated LDPC codes, first the higher rate code is designed and then by increasing k while keeping n and the parameters of the higher rate code unchanged, the lower rate code is designed [7].
The techniques that have been introduced so far, promise designing very high rate codes for the relay channel. However, these techniques do not address the decoder complexity constraints. This issue is an important problem, particularly at the relay, since on one hand, there are generally stricter hardware and power constraints at the relay and on the other hand, decoding delay at the relay results in a higher decoding delay at the destination.
Inspired by a method to estimate the number of decoding iterations using the extrinsic-information transfer (EXIT) charts to design complexity-optimized conventional LDPC codes ([8]), we propose a technique to design complexity-optimized bilayer lengthened LDPC codes. For this purpose, we shape the overall EXIT chart of the bilayer code— calculated using the bilayer density evolution method— such that the number of the decoding iterations is minimized, while retaining the rate of the rate-optimized bilayer code. Our technique results in designing bilayer lengthened LDPC codes with dramatic complexity reduction compared to threshold optimized codes.

The source-destination link has a lower SNR than the source-relay link and as a result, the lower rate code (represented by the lower layer graph) is optimized considering the effective signal to noise ratio of the source-destination (SD) link, denoted by SNRSD, or equivalently, the effective noise threshold of the SD link, denoted by σSD and the higher rate code (represented by the overall graph) is optimized for the effective signal to noise ratio of the source-relay (SR) link, denoted by SNRSR, or equivalently, the effective noise threshold of the SR link, denoted by σSR.
The relaying technique based on bilayer lengthened LDPC codes is deployed as follows: The source encodes its data using a high-rate LDPC code (characterized by the overall graph) and then transmits the encoded information. The destination receives and only stores this code since it is optimized for SNRSR which is higher than SNRSD. However, the relay decodes this code reliably and then encodes the set of upper variable nodes using X1 and sends it to the destination. It is assumed that the destination can decode this code reliably and as a result gains access to the values of the upper variable nodes. As a result, the destination removes the ambiguity about the upper variable nodes and needs only to decode the LDPC code corresponding to the lower graph which was originally designed based on SNRSD.
Based on the decode-and-forward strategy, as described above, the relay decodes the bilayer code first, and then the decoder at the destination— having received the values of the upper graph variable nodes by the relay— decodes the lower layer LDPC code. Since both codes are very high

C. Bilayer Density Evolution
Traditionally, the density evolution (DE) technique has been used to design LDPC codes and analyze their performance. This technique studies the evolution of the densities of the messages in the iterative message passing decoding of LDPC codes. In the context of bilayer LDPC codes, this method is called bilayer DE [5]. We present a summary of this technique here to show how we can calculate the probability of error for the messages.
Iterative decoding techniques work by successively sending messages from variable nodes to check nodes and vice versa. However, messages at both variable nodes and check nodes are updated according to some update rule. This algorithm is initialized by the densities (or LLR values) of the messages received from the channel and continues by iteratively updating these messages at both sets of nodes. A decoding iteration of the iterative decoding method begins with updating the messages in the check nodes followed by updating them at the lower and upper variable nodes. Considering the fact that the channels and the decoding algorithm are symmetric [9], the negative tail of the density represents the probability of error. Hence, if we denote the message probability density functions at the outputs of degree-i lower and upper variable nodes at the beginning of the l-th iteration by qil and pil, respectively, and their corresponding error probabilities by e(1)(pli, qil) and e(2)(pil, qil), respectively, the overall message error probability at the beginning of the l-th iteration can be calculated by (for more details, refer to [5])

VI. CONCLUDING REMARKS
We proposed a technique to design complexity-optimized bilayer lengthened LDPC codes. Our technique fixes the rateoptimized code of the destination node and targets minimizing the decoding complexity of the relay code by minimizing a function of the upper variable node degree distribution, which approximates N , while retaining the rate of the bilayer code. This was made possible by generalizing EXIT charts to the case of bilayer lengthened LDPC codes in this work and an existing approach to estimate the number of decoding iterations using the EXIT chart of an irregular LDPC code. Using our technique, we designed bilayer codes with noticeable reduction in decoding complexity and delay compared to the rate-optimized codes reported in the literature.
There are more interesting problems for future work in this direction. In this work, we studied the asymptotic behavior of the designed codes. We are also interested in analyzing the performance of these codes while implemented in long block lengths. Moreover, we are interested in studying the impact of using different upper check node degrees. Also, we have reasons to believe that the codes we have designed have a better convergence behavior even at the destination node. In other words, even when the bilayer code is complexityoptimized at the relay level, the destination will benefit from this optimization.
