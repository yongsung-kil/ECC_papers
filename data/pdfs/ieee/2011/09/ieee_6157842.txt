Abstract-Linear programming (LP) decoding is getting more and more attention because of the maximum-likelihood (ML) certificate property. As one of the improved LP decoding methods, multistage LP (MLP) decoding has been proposed to improve the performance of LP decoding but at the cost of higher complexity. In this paper, the problem of achieving a balance between the performance and the complexity of LP decoding is considered. First, a criterion used for searching new constraints adaptively is proposed. Then a modified MLP decoding is presented by incorporating the adaptive decoding algorithm to decrease the complexity. The presented modified MLP decoding can obtain a good tradeoff between the performance and the complexity and is proved also has the ML certificate characteristic. Simulations with moderate block length LDPC codes show that the proposed MLP decoding achieves the same BER performance as the original MLP decoding with far fewer used constraints and faster convergence speed consequently.
Keywords-LDPC codes; linear programming (LP); maximum-likelihood (ML) certificate property; LP decoding; multistage LP (MLP) decoding; adaptive decoding algorithm; modified MLP decoding; adaptive Branch-and-Bound (AB&B) algorithm
I. INTRODUCTION
Low-density parity-check (LDPC) codes were first discovered by Gallager in 1962 [1]. Because of the excellent error-correcting performance, these codes have received a lot of attention. The linear programming (LP) decoding of LDPC codes, as an approximation of maximum-likelihood (ML) decoding, was proposed by Feldman, Wainwright and Karger in [2]. It has gained prominence in the recent years mainly due to its ML certificate property [2] which makes it easier to analyze than message passing decoding algorithms. LP decoding has been improved on the purposes of obtaining either lower complexity [3]-[5] or better performance [6]-[8].
As one of these improved LP decoding methods, multistage LP (MLP) decoding [7] improved the performance based on Branch-and-Bound techniques [9]. It starts by solving the original LP decoding problem, and then the adaptive Branchand-Bound (AB&B) algorithm [7] is used to correct the error results based on a criterion. In this paper, a modified MLP decoding is presented to achieve a good tradeoff between the performance and the complexity. An exact criterion used for finding new constraints is put forward firstly. Then, the adaptive LP decoding introduced in [3] is incorporated into the MLP decoding for the sake of decreasing the complexity.
The presented MLP decoding starts by a simple LP problem and necessary constraints selected by an algorithm are added to LP problem iteratively until the terminated condition is satisfied. Then a special subset of variable nodes with fraction values in the current solution is defined and the space of possible settings to the set is searched according to the criterion proposed. These possible settings are added to the current LP problem as new constraints and consequently a group of LP problems whose solutions may include the optimum is formed. At last, the optimal solution is found by the AB&B algorithm [7].
The modified MLP decoding performs as well as the original MLP decoding but with far fewer used constraints owing to the introduction of adaptive algorithm. Superiorities of the modified MLP decoding are significantly for moderatesized LDPC codes and we will demonstrate it in this paper. Just like the original LP decoder, LP decoder based on this modified multistage algorithm also has the ML certificate property, i.e., whenever it outputs a codeword, it is guaranteed to be the ML codeword.
The remainder of this paper is organized as follows. LP decoding is reviewed briefly in Section II. Section III introduces multistage LP decoding based on the AB&B algorithm. In Section IV we propose a criterion and the modified MLP decoding is described in detail. Then simulations over a moderate-length LDPC code are shown in Section V to demonstrate advantages of the modified MLP decoding. In the end, we conclude the paper in Section VI.

III. MULTISTAGE LP DECODING
The MLP decoding algorithm starts by solving the original LP decoding problem (3). If the LP decoder outputs an integer

solution, the algorithm terminates. Otherwise, the AB&B algorithm is used to correct the error results based on a criterion for searching new constraints.
Suppose the original LP decoder outputs a non-integer solution f ' . A subset of variable nodes with fraction values denoted by T will be found according to some criterion. The space of possible settings to the set T is considered as the group of new constraints. Each possible setting corresponds to a sub-LP problem by adding the possible setting as a set of new constraints to the LP problem. It has been demonstrated that the MLP decoder has the superiority in decoding performance compared with the original LP decoder but at the cost of an increase of complexity [7].

V. SIMULATIONS AND ANALYSIS

In this part, simulations about performance and complexity of different decoding methods for a random (3, 6) regular LDPC code of length n=96 obtained from [10] is carried out over the additive white Gaussian noise (AWGN) channel with BPSK modulation.
In Fig. 1, the BER versus different signal-to-noise ratio (SNR) levels for the MS, BP, LP, MLP and modified MLP decoders are plotted respectively. The maximum number of iterations in the BP decoder is set to 100. The results show that the MS decoder has the worst performance at each SNR and the LP decoder has the similar performances with BP decoder. The modified MLP and the original MLP decoders perform the same and they have the best performance among all of these decoding methods. In the low SNR region, the BP, LP and the modified MLP decoder have the similar performances. As the SNR increases, the superiority of the modified MLP decoding becomes more and more apparent. At the point BER= 102 , the modified MLP decoder brings nearly 0.3 dB gain compared with the BP and original LP decoders.

VI. CONCLUSION AND FUTURUE WORK
In this paper, a modified multistage method for LP decoding of LDPC codes and a criterion used for searching new constraints were presented to make improvements on both the performance and complexity of LP decoding.

In this proposed method, adaptive decoding algorithm was incorporated in the basic decoding part for the sake of lower complexity. Compared to the original MLP decoding, the modified MLP decoding achieved a good balance between the performance and the complexity. Simulations showed that the modified MLP decoding get the same BER as the original MLP decoding with a much lower computation complexity. Besides, the proposed criterion was testified to find the variable nodes whose values are most uncertain in average and search the possible settings to these variable nodes efficiently. The ML certificated property of the modified MLP decoding was also proved in this paper.
In our future work, we will focus on finding more efficient methods to decrease the complexity of the error correcting part unchanging the performance. Besides, more factors would be considered into the adaptive decoding algorithm to try to improve the performance of the basic decoding part.
