A Class of Irregular Repeat Accumulate Code with Flexible Code Length and without Short Circles

Xiang Chen, Huanbing Liu, Hui Jiang
Room 301, Dept. of Information Engineering Hefei Electronic Engineering Institute Hefei, P. R. China xiaxiang83@126.com

Abstract—Based on the good degree distribution of a kind of irregular repeat accumulate (IRA) code, the base check matrix is designed according to quasi-cyclic method, and the length-4 short circles are eliminated. The obtained IRA code has flexible code length from 150 to 6000, with link-level performance close to the LDPC code specified by IEEE802.16e standard. The proposed IRA code family is suitable for multimedia communication system.
Keywords- Low-Density Parity Check (LDPC) Code; Irregular Repeat Accumulate (IRA) code; quasi-cyclic (QC) code;
I. INTRODUCTION
Low-Density Parity Check (LDPC) code and turbo code are two error-correcting codes with Shannon Limit approaching performance. The randomized encoding and iterative decoding methods are two keys of their excellent performance. From the implementation point of view, for LDPC code, decoding algorithm can be of linear complexity, but encoding algorithm is not so. On the contrary, for turbo code, encoding algorithm can be of linear complexity, but decoding algorithm is much more complex.
In 1998, D. Divsalar et al. proposed "Turbo-like" code, i.e. Repeat Accumulate (RA) code [1]. In fact, it is a kind of Serial Concatenated Convolutional Codes (SCCC). In 2000, Hui Jin et al. discovered that RA code is also a type of LDPC code. They designed protograph of Irregular Repeat Accumulate (IRA) code [2], using Belief Propagation (BP) decoding algorithm, thus both encoding and decoding are of linear complexity. The IRA code has simple protograph and excellent degree distribution pair. Unfortunately, no practical code is designed. In this paper, the quasi-cyclic (QC) method is employed to design special sub-check matrices, obtained LDPC code with good error performance and flexible code length.
The rest of the paper is organized as follows. Section II introduces in brief the principles of LDPC code, IRA code and QC-LDPC code. Section III describes the proposed construction scheme of IRA code with flexible code length in detail. The error performance of the constructed IRA code is analyzed in section IV. Finally, Section V concludes the paper.

C. QC-LDPC code
QC-LDPC code [6] donates a kind of (n, k) linear block code, which is still a code word of the linear block code after n0 (n = zn0 , k = zk0 ) times of cyclic shifting. The check matrix of QC-LDPC code is determined by the former k0 rows. The encoder can be implemented by shift register, so the complexity of encoder is reduced significantly. The procedure of right cyclic shifting is shown in fig. 4. The square is a submatrix, and the black lines mean the position of non-zero elements. The original sub-matrix of QC-LDPC code can be identity matrix, or dual-diagonal matrix, as the left one in fig. 4. After n0 times of cyclic right-shifting, the sub-matrix is illustrated as the right one in fig. 4.

C. Eliminating the length-4 circle
The circle of LDPC code is close loop in Tanner graph composed of variable nodes, check nodes and edges that connect them. The number of edges is the length of circle, the minimum length is four. Length-4 circle is of utmost importance, as it determines the error-floor performance of LDPC code [7].

A. The error performance of different code lengths
When the sizes of sub-matrix are selected as 25, 50, 100 and 200, respectively, the corresponding code length are 150, 300, 600 and 1200, these codes are quiet suitable for multimedia communication. The Bit Error Rate (BER) performances of them are compared in fig. 5. It can be observed that when the coding length is changing, the performance is always pretty good, because the degree distribution pair remains the same, and no length-4 circle appears. Obviously, the error performance is getting better with the code length increasing.
B. The improved performance by eliminating length-4 circle
The Block Error Rate (BLER) performance of the proposed IRA code before and after eliminating the length-4 circles are illustrated in fig. 6. Significant performance improvement can be seen when the length-4 circles are eliminated. With the length-4 circles, the error floor of IRA code is relatively higher; on the contrary, the BLER-SNR curve is cliff in higher SNR range when the length-4 circles are eliminated. It verifies that the length-4 circle is a key to the error performance of LDPC code.

C. The performance comparison with WiMax LDPC code
The constructed (600, 300) IRA code is compared with the (576, 288) LDPC code specified by the IEEE802.16e standard. Their BLER performances are shown in fig. 7. Their performance is quiet in neighbor. When the BLER is 0.01, the difference is within 0.5dB. Thus the constructed IRA code is also Shannon-limit approaching error-correcting code.
V. CONCLUSION
Based on the excellent degree distribution pair of a class of IRA code, the practical base check matrix is designed with the Quasi-cyclic method and the length-4 circles are eliminated to lower the error floor. It is verified that the constructed IRA code has flexible code length and pretty good error performance, almost the same with the IEEE802.16e LDPC code.
