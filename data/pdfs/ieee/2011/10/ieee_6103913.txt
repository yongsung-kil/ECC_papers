DESIGN OF RC-LDPC CODES AND ITS APPLICATION IN CODES SC-FDE SYSTEMS
Jianwu Zhang, Huan Yan, Jianrong Bao School of Telecommunication Hangzhou Dianzi University Xiasha Higher Education Zone
Hangzhou, Zhejiang, 310018, P.R.China

Abstract
This paper proposes a new scheme of the rate compatible low-density parity-check (RC-LDPC) coded single carrier-frequency domain equalization (SC-FDE) system where RC-LDPC codes are combined with SC-FDE modulations. To construct a class of different RC-LDPC codes, we take advantage of a base matrix constructed by the progressive edge growth (PEG) algorithm. A special code matrix structure is used and modified to meet requirements of the different code rate. In addition, a pseudorandom sequence instead of traditional cyclic prefix (CP) is proposed to estimate the channel noise variance for LDPC decoding. The simulation results show that the modified RC-LDPC codes have better performance than that of current method and the RCLDPC coded SC-FDE system obtains better performance compared with uncoded SC-FDE system.
Introduction
With rapid development of broadband wireless communications, the requirements of reliability and efficiency in transmitted signals become high. However, frequency selectively fading in wireless channel degrades the quality of wireless communications. Therefore, how to overcome the channel fading has been the research focus. At present, there are several methods to overcome the multi-path effects, such as SC-TDE, SC-FDE and OFDM. Among them, SC-TDE is not suited to be applied in high speed wireless communication systems because of its high complexity [1]. At the same time, OFDM with orthogonal sub-carriers has some deficiencies because it has a high peak-toaverage power ratio and a high sensitivity to phase

noise generated by local oscillator. Meanwhile, it needs obtain accurate frequency synchronous in an OFDM system which can’t be easily solved. Fortunately, SC-FDE performs well in all aforementioned aspects. It has been proved that SCFDE has the same performance and complexity as OFDM [1-3]. The main difference between OFDM and SC-FDE is that the IFFT and FFT operations are performed at receiver in SC-FDE system, while they are located in the transmitter and receiver respectively in OFDM system. In addition, IEEE801.16 has prescribed OFDM and SC-FDE technology as downlink and uplink in a dual mode system respectively [4]. Therefore, SC-FDE has a huge potential in high speed broadband wireless communications.
LDPC code first investigated by Gallager [5] is a linear block code which has Shannon limit approaching performance [6]. Since it can be constructed simply and its code length and rate are much flexible, the LDPC code is gradually considered as an important error-correcting code with high encoding gain. Recently, LDPC code is adopted in many standards, such as IEEE802.16e and DVBS2. Furthermore, it may become one of the channel coding schemes in the 4th Generation wireless communication. RC-LDPC codes with different rate can be obtained by puncturing and extending the bit nodes from its base matrix. Especially some RCLDPC codes can be designed by regular construction. In this paper, we put RC-LDPC codes into the SCFDE system to improve the performance of the whole system. Traditionally, SC-FDE uses CP as a guard interval and removes it before equalization. The construction of RC-LDPC codes
LDPC code is defined by its parity-check matrix H. For a LDPC code, there are very few elements “1” in the parity-check matrix H. The number of columns in parity-check matrix H is also the code length, and it is denoted by n. The number of message bits is denoted by k. If a parity-check matrix H is full rank, the code rate is R=k/n. Since LDPC decoding uses BP algorithm, it is necessary to ensure their independencies in message propagation between variable nodes and check nodes. In other words, it is necessary to guarantee the girth, i.e. the shortest length of rings, as large as possible. Then the PEG algorithm is proposed to search the largest girth and establish the edge between the variable nodes and the check nodes. By this principle, we construct the efficient RC-LDPC codes.
RC codes are widely used in packet data communication where adaptive coding and/or unequal error protection is required [7]. And RCLDPC codes are introduced to adjust the rate flexibly. Generally, some relationship between the base matrix and extended matrix is established to design the structuring the low rate LDPC codes. Equally, the relevant researching algorithms are proposed to determine the optimized puncturing nodes for the high rate LDPC codes. RC-LDPC codes can be adapted to different systems such as HARQ system and SC-FDE system and become a new research focus.

At first, we construct a rate-1/2 (M, N)-LDPC code as a base matrix H M ×N . Fig. 1 is the structure of a typical RC-LDPC code [8].
As seen from Fig. 1, the base matrix H M ×N is a lower triangular matrix with good properties of easily encodable linearity and full rank. So it is the added matrix H M '×M ' , which is a square matrix. To
complement the dependencies between the base matrix and the extended matrices, identity matrices IM '×M ' are added. As we know, the existence
of four-cycles in parity-check matrix depends on that two or more rows(or columns) have more than one position in common that contains a “1” element. And there are two side-by-side identity matrices at the lower left part. So it may exist four-cycle in this parity-check matrix. But this problem isn’t been dealt with in [8]. Therefore, a modification is made as follows. Firstly, each column of the second identity matrix IM '×M ' is continuously shifted right to calculate
and record the number of four-cycle. And the right shift coefficient is corresponded to the minimum number of four-cycles and it obtains the final matrix denoted by I'M '×M ' . Then the elements “1” at the corresponding positions of I'M '×M ' are deleted to remove four-cycle by recording the path of cycle. Though this modification may reduce the row weight of the added rows, the number is very small. In other words, the row weight distribution still keeps a rather uniform. Meanwhile, the computing complexity only increases slightly because the size of I 'M '×M ' is limited.

Because of the special structure of matrices H M ' ×M ' and H M×N , the original PEG algorithm need to be modified for our construction of the code. Firstly, the variable degrees are arranged by rising order according to the density evolution optimized degree distribution [9]. In designing the first lowdegree M variable nodes, the elements in diagonal line must be “1”. At the same time, there is no element “1” in the upper diagonal. However, the local girth may be largest while the distribution of elements “1” in the current column doesn’t satisfy the condition. In such circumstance, the Guassian eliminate algorithm is used to eliminate all fourcycle. In addition, the rightmost column has weight one and there is a deviation from the density evolution. But its influence may be negligible. The construction of matrix HM '×M ' is the same as that of the base matrix. To get a lower rate LDPC code, at each level, corresponding matrices are added into the base matrix. And we puncture the M’ degree-2 variable nodes to increase the rate of LDPC code at a time, which reduces the decoding failure caused by the loss of initial information to the extreme extent.
RC-LDPC coded SC-FDE system
In [1-2], CP is a copied by the last part of each data block. By this transmitting format it avoids the interblock interference and makes the linear convolution between symbols and impulse responses with multipath channel turn to be cyclic convolution in one period of data block. However, it is an unknown sequence since it varies along with the transmitting data. So the estimation of channel parameters doesn’t perform well by using it. But UW can solve this problem efficiently because of its consistency. Furthermore, it has the same merit as CP. The system model of SCFDE can be shown in Fig. 2. Before inserting UW, the RC-LDPC coding can be used to obtain LDPC codes with different rates. The format of inserting UW is represented in Fig. 3. The data symbols with UW are modulated by some modulation method (in this paper we use BPSK modulation for instant) and transmitted by antenna after D/A converter. In this

paper, we use a fixed pseudorandom sequence to obtain channel parameters.

At the receiver, the frequency domain equalization is performed after FFT operation. Zero Forcing (ZF) algorithm and Minimum Mean Square Error (MMSE) algorithm are generally the linear equalization methods. The ZF algorithm may enlarge the noise when there is a deep fading in channel frequency response. But the MMSE algorithm considers the problem of reducing Inter-symbol interference (ISI) and noise enlargement, thus it has a good tradeoff. During this procedure, channel parameters are estimated by pseudorandom sequence. It helps to estimate the maximum likelihood logarithm ratio (LLR) of the initial variable nodes in later LDPC decoding.

Simulation results
In this section, the effectiveness of RC-LDPC codes and its application in SC-FDE system can be verified. We first simulate the performance of RCLDPC codes in AWGN channel with different SNRs. Through observing the BER performance curves of different rate LDPC codes, we draw some conclusion

about the properties of these codes. Then they are put them into the SC-FDE systems and then are analyzed for the difference of the performance between coded system and uncoded system.

The maximum iterations of BP algorithm are 50. And we choose the MMSE as our equalization method. The length of FFT operation should be chosen according to the length of data block and UW. Because of the high coding gain of the LDPC code, the performance of the coded SC-FDE system is improved compared with that of the uncoded system. The SNR of the LDPC coded SC-FDE system improves more than 8dB at the BER of 10-5.This illustrates that good performance of LDPC codes can also perform well in the system. Furthermore, the performance of system with different rate LDPC codes has the same regularity as that of the single LDPC code. Therefore, it can be concluded that the RC-LDPC codes being put into the SC-FDE system can improve the performance greatly.
Conclusions
A RC-LDPC coded SC-FDE system is proposed in this paper. Through slightly modification of the RCLDPC code construction method, we obtain a no four-cycle extended code for the lower rate LDPC codes and for the higher rate codes by puncturing the lowest degree variable nodes. The estimation of channel is also important for LDPC decoding. We use
