CIRCULANT SEARCH ALGORITHM FOR THE CONSTRUCTION OF QC-LDPC CODES

Xiongfei Tao, Xiaofeng Zhou, Deyu Feng, Lixin Zheng

Department of Electronic Science and Technology, Huazhong University of Science and Technology, Wuhan 430074,China taoxiongfei@mail.hust.edu.cn, sec.fengzi@yahoo.com, fengdy0520@sina.com, zhenglixin@mail.hust.edu.cn*

Abstract
A circulant permutation search algorithm is present in this paper, which can be employed to search appopriate circulant permutation to construct QCLDPC codes with desired girth. The algorithm is based on a tree expansion mechanism and a selectand-discard procedure. By proposed method, a practical (3, 4) code with code length n=37368 and girth g=18 is constructed. The experiment shows that the codes construct in this paper have large girth and minimum distance, and the bit error rate performance over AWGN channels is excellent.
Keywords: QC-LDPC codes; tree expansion ˗ search algorithm
1 Introduction
LDPC codes were originally introduced by Gallager [1] in his doctoral thesis. LDPC codes are linear block codes, their bit error rate (BER) performance approaches Shannon limit when belief propagation (BP) algorithm is employed to decode. Since the rediscovery of LDPC codes by Mackay and Neal in 1996 [2], there has been renewed interest in LDPC codes.
The structure of LDPC codes can affect the performance of iteratively decoding algorithm, Tanner [3] analyzed the iteratively decoding algorithm based on graphs and pointed out that the short cycles in a Tanner graph prevent the iteratively algorithm from converging. So the girth, i.e., the shortest cycles, in a Tanner graph is an important factor, which should be taken into account in the design of LDPC codes. If the Tanner graph has larger girth, it will take more iterations until extrinsic information originating from different nodes in the Tanner graph becomes correlated.
Approaches to the construction of LDPC codes can divide into two major classes: random-like constructions and algebraic constructions. Random approaches can construct LDPC codes that closely achieve the Shannon capacity. However, the

significant encoding and decoding complexity of random codes leads to hard implementation in hardware. On the other hand, algebraic LDPC codes have hardware-friendly structure.
The algebraic constructions can be separated into two species. The first species is based on graph theories [4], and the second one is base on circulant permutation matrices[5]. Enlightened by Kou’s idea, some other graph-theoretic methods are proposed. Esmaeili [6] proposed a method for design QC-LDPC codes with girths at most 18. Quasi-cyclic (QC) LDPC codes based on circulant permutation matrices are getting more attention due to their linear complexity with shift register and small requirement of memory.
Originally, a QC-LDPC code is defined as a (J; L) regular LDPC code of length Lp whose paritycheck matrix H is a J×L array of p×p circulant permutation matrices[5]. Fossorier derived a necessary and sufficient condition for the existence of cycles of given length in QC-LDPC codes. Fossorier [5] also showed that these QC-LDPC codes have a girth at most 12. O’Sullivan and etc [7] proposed a construction method of LDPC codes by using a seed matrix which is the concatenation of two incidence matrices for Fano planes and circulant permutation matrices. They showed one special case of constructing an LDPC code with girth 14. Another code with girth 14 has been given in [8]. However, the girth-14 regular (3,5) code given in [8] has a length of 134770 which is too long under the construction parameters.
In this paper, we are concerned with the shift value assigning algorithm for the construction of QCLDPC codes with desired girth. The algorithm is based on a tree expansion in the mother matrix of a QC-LDPC code.

4 Simulation result
In this part, we estimate the bit error rate (BER) performance of some LDPC codes proposed in this paper, and compare the performance between our codes and the Fossorier’s codes. For simulation results we have used an additive white Gaussian noise (AWGN) channel. The decoding algorithm is belief propagation with maximum iteration number 80.

First we construct two (3, 6) LDPC codes with code length n=5964 by our method and Fossorier’s respectively. Both codes have girths of 12, but the Fossorier’s code is construct by a 3×6 mother matrix with all entries is ‘1’ and the mother matrix of our codes is obtained by search with a size of 14*7. From the Figure 2, we can see that the BER/BLER performance of the two codes is almost the same when the SNR is below 1.6dB. While the SNR is higher than 1.6dB, Fossorier’s code starts to show a error floor while our code does not. This implies that althought the two codes have same girth, but the dmin of Fossorier’s code is small which is no more than 24 due to it’s mother matrix structure.

5 Conclusions
An approach to the construction of very large girth LDPC codes with is presented in this paper. The kernel of the approach is a search algorithm based on an expansion tree. The algorithm output not only the exponent matrix but also the sequency of valid block size. In comparison with the Fossorier’s codes, the introduced codes with high girth have better performance.
