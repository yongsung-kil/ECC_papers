Hybrid Linear Programming Based
Decoding Algorithm for LDPC Codes
Telex Magloire Nkouatchah Ngatched, Member, IEEE, Attahiru Sule Alfa, Member, IEEE,
and Jun Cai, Member, IEEE
Abstract—This paper presents a hybrid decoding algorithm
for low-density parity-check (LDPC) codes based on the interior
point method with barrier function for linear programming
(LP) decoding introduced by Wadayama [16]. First, an efficient
implementation of Wadayama’s algorithm is presented. The main
idea behind the modification is to approximate the barrier
function for the fundamental polytope defining the code so that it
contains only one linear constraint for each of the parity-check
constraints. A two-stage hybrid decoding which combines the
interior point decoding and a low-complexity decoding algorithm
for LDPC codes is then proposed. Simulation results show that
the approximations introduced in the proposed algorithms do
not result in any performance degradation, while considerably
reducing the decoding complexity and latency.
Index Terms—Belief propagation (BP) algorithm, convex op-
timization, low-density parity-check (LDPC) codes, finite ge-
ometries, interior point algorithm, weighted bit-flipping (WBF)
algorithm.

Proposed by Feldman et al. [5], linear programming (LP)

and turbo-like codes. The LP decoder is based on a LP
relaxation of the original maximum-likelihood (ML) decoding
problem, and utilizes a set of constraints which contains
all valid codewords of a given code and a linear objective
function. The decoder output is obtained by minimizing the
objective function over the polytope defined by the constraints.
Although the practical performance of LP decoding is com-
parable to message-passing decoding [5]–[7] for LDPC codes

connections allow theoretical insight from the LP decoding

In addition, LP decoding has the ML property; whenever the
result of linear programming decoder is integer, it is provably
the maximum likelihood codeword. None of the standard IMP
algorithms are known to have this desirable property.
However, a major disadvantage of LP decoding concerns
its higher complexity compared to IMP decoding. In the
formulations originally introduced by Feldman et al. [5], the
number of constraints in the LP decoding problem is linear
in the block-length, but exponential in the maximum check
node degree. Even for parity-check matrices of moderate row
weights, the resulting computational load can be prohibitive.
On the other hand, when using a general purpose LP solver
for LP decoding of LDPC codes, the structure of the problem
at hand cannot be taken into account. Moreover, the compu-
tational complexity, although polynomial in the block length,
may not be linear. Several authors have proposed algorithms
for low-complexity LP decoding. In [9], [10], Taghavi and
Siegel introduced “adaptive” linear programming (ALP) to
alleviate the computational burden of LP decoding. Instead
of starting with the full set of constraints that describe a code,
they first solve for the LP optimum of a problem where the
values of the binary codeword symbols are only constrained to
be greater than zero or less than one. At the resulting optimum,
they sequentially and adaptively add more of the constraints
to the problem until either an ML codeword is found or no
further “useful” constraint exists. The result is guaranteed
to match the solution to the original LP, even though only
a small fraction of the original constraints are used. ALP
was recently improved in [11]. A new representation of the
LP decoding problem with a problem size growing linearly
with both the maximum check node degrees and the code
length was proposed in [12] and [13]. This method, however,
has an overall complexity which depends on the underlying
LP solver, and which remains substantially higher than that
of ALP decoding. Other approaches toward improving the
performance of LP decoding include facet guessing, proposed

The design of efficient LP solvers for LP decoding of
LDPC codes using interior-point methods has recently been
investigated in [11], [16], [18]. In particular, the approach
of Wadayama [16] called interior point method with barrier
function yields encouraging results. The method uses a barrier
function to transform the constrained problem resulting from
the LP relaxation of ML decoding into an unconstrained prob-
lem which can be solved using unconstrained optimization
techniques. The barrier function corresponds to the constraints
on the fundamental polytope defined based on an LDPC ma-
trix, and is used so that the trajectory of the search points does
not get close to the boundary of the fundamental polytope.
In this paper, we first study an efficient implementation of
Wadayama’s method. The main idea behind the modification
is to approximate the log-barrier function for the fundamental
polytope defining the code so that it contains only one linear
constraint for each of the parity-check constraints. This leads
to an approximate variation of the Newton method used to
solve the convex optimization problem. A huge reduction in
computational costs is hence obtained, making it possible to
apply interior point algorithms with barrier function for the
decoding of any LDPC code, irrespective of its row weight.
To improve the convergence and reduce the decoding latency,
a two-stage hybrid decoding which combines the interior
point decoding and a low-complexity decoding algorithm is
proposed. In the first stage, the interior point decoding (IPD)
is used to generate a search point. A binary quantization
of the search point is obtained, and the syndrome of the
resulting vector is calculated. If the first stage decoding fails,
the decoding is continued by the low-complexity algorithm
that is initialized by the result of the IPD. The process
is repeated iteratively until either a codeword is found, or
the user-predefined maximum number of outer iteration is
reached. The performance of the proposed algorithms are
demonstrated through simulation, and it is observed that the
approximation does not result in any performance degradation,
while considerably reducing the decoding complexity and
latency.
The rest of this paper is organized as follows. In the next
section, background on LDPC codes and linear programming
relaxation of maximum-likelihood decoding are provided. A
review of the interior point decoding based on the barrier
function method, with an emphasis on its implementation, is
presented in Section III. In Section IV, an efficient imple-
mentation of the barrier function method is introduced. In
Section V, a hybrid decoding algorithm which combines the
interior point decoding based on the barrier function method
and a low-complexity decoding algorithm is proposed. Some
simulations results are presented in Section VI, and finally
some concluding remarks are given in Section VII.

relaxation has the advantage that the description complexity
of the convex hull of any local codeword, and thus 𝒫, is much
less than the description complexity of the codeword polytope.

III. IMPLEMENTATION OF INTERIOR POINT DECODING
Although the LP decoding problem (6) is easier to be solved
than the original MLD problem (1), it is still a computationally
difficult problem. Moreover, using general purpose LP solvers
can be highly inefficient as they do not take advantage of
the particular structure of the optimization problems arising
in LP decoding. In this section, we briefly review the interior
point method (IPM) with barrier function for LP decoding and
outline our implementation of the method.
Interior point methods are a certain class of algorithms
to solve linear and nonlinear convex optimization problems.
These methods have been inspired by Karmarkar’s algorithm
[21], developed by Narendra Karmarkar in 1984 for linear
programming. As implied by its name, and unlike simplex
methods, IPM finds an optimal solution by traversing the
interior of the feasible region. Most interior point algorithms
fall into one of three main categories: affine scaling methods,
potential reduction methods, and path following methods. The
last one, the path following algorithm, has the best known time
complexity and good performance in practice [22].
Consider the constrained optimization problem of the form

applications, like the problem at hand, 𝒮is defined explicitly
by a number of functional constraints. The key idea of interior
point algorithm based on barrier function method is to use
a barrier function to transform the “difficult” constrained
problem (7) into an “easy” unconstrained problem, or into
a sequence of unconstrained problems, which can be solved

E. Complexity
A complexity analysis of interior point methods in terms
of the number of operations required is not straightforward.
Based on the discussion in [23], which exploits the theory of
the self concordant functions, it is shown in [17] that the total
number of Newton iterations required by the standard IPD
scales as 𝑂(𝑛ln 𝑛). This number is given by the product of the
number of centering steps and the number of Newton iterations
required for each centering step. In the proposed IPD, a limit is
placed on the number of outer and inner iterations. Moreover,
due to the approximations introduced, the complexity of the
proposed IPD is much less than that of the standard IPD. It
was observed that, for small block length, the running time
of the proposed IPD is comparable to that of the IDBP. As
the block length increases, however, IDBP is more and more
faster than the proposed IPD.
V. HYBRID DECODING OF LDPC CODES BASED ON
INTERIOR POINT DECODING
As mentioned in Section III, in the interior point decoding
process, it is possible that the current search point approaches
close enough to a codeword vertex. In such a case, it is
advantageous to use a low-complexity decoding algorithm to
complete the decoding instead of waiting for the search point
to converge to a codeword vertex, which in general requires
a much longer computational time. Moreover we observed
through simulation that, though the performance of the IPD
algorithm improves with increasing number of outer iteration,
there is a diminishing return as the number of outer iteration
increases. Actually, there is no significant improvement after
the second outer iteration. In this section, we propose a
two-stage hybrid decoding which combines the interior point
decoding described previously and a low-complexity decoding
algorithm for LDPC codes.
The hybrid algorithm works as follows. In the first stage,
the interior point decoding is used to generate a search point.
A binary quantization of the search point is obtained, and
the syndrome of the resulting vector is calculated. If the
first stage decoding fails, the log-likelihood ratio of each
coordinate of the search point is calculated, and the decoding

Many low-complexity decoding algorithms for LDPC codes
have been proposed in the literature. Examples of these
algorithms include [19], [25]–[30]. Any of these algorithms

very attractive as they are binary message-passing algorithms
and require only logical and integer operations. This feature
simplifies the decoder implementation, as it can be achieved
using simple combinatorial logic circuits. Furthermore, these
two algorithms allow full parallel decoding. From a perfor-
mance point of view, however, the algorithm in [26] offers a
good tradeoff, especially for LDPC codes of short to medium
block lengths. This algorithm, referred to as fast weighted
bit flipping (FWBF), improves the convergence speed of the
modified weighted bit flipping (WBF) decoding of [24] by
flipping multiple bits and thus possibly correcting more than
one bit error at each iteration. Our studies, however, reveal that
the number of bits flip at each iteration of the algorithm of [26]
is overestimated. As a result, some bits are erroneously flipped,
and in some cases cannot be corrected back. As suggested
in [27], a good approximation of the number of bits to be
flipped at each iteration is given by ⌊𝑤𝐻(s)/𝑤𝑐⌋, where s is
the syndrome vector, 𝑤𝐻(a) denotes the Hamming weight of
a, and ⌊𝑥⌋denotes the greatest integer less than or equal to 𝑥.
This approximation is based on the observation that for low-
density parity-check matrices, the syndrome weight increases
with the number of errors on average. Hence, we modify the
fast WBF to obtain Algorithm 4 which we called improved
fast WBF (IFWBF). As shown in the next section, this slight
modification leads to improvement in performance in terms of
both the bit-error rate (BER) and frame-error rate (FER). In
Algorithm 4, 𝛼is a positive weighting factor whose optimal
value is determined through simulation. It should be noted
that Algorithm 4 can be further improved by incorporating
the delay-handling procedure introduced in [28].

VII. CONCLUSION
In this paper, we studied an efficient implementation of
interior point method with barrier function for LP decoding
of LDPC codes. Key to our approach is the approximation of
the barrier function for the fundamental polytope defining the
code so that it contains only one linear constraint for each
of the parity-check constraints. This approximation, together
with the subsequent approximation of the gradient and the
Hessian, makes the interior point method with barrier function
computationally tractable and applicable for the decoding
of any LDPC code. To further reduce the computational
complexity and the decoding latency, a two-stage hybrid
decoding which combines the interior point decoding and
a low-complexity decoding algorithm for LDPC codes is
then proposed. The performance of the proposed algorithm
is demonstrated through simulation. It is observed that the
