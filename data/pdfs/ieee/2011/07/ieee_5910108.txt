Comparisons Between Reliability-Based Iterative Min-Sum and
Majority-Logic Decoding Algorithms for LDPC Codes
Haiqiang Chen, Kai Zhang, Xiao Ma, and Baoming Bai
AbstractвҖ”A modified reliability-based iterative majority-logic
decoding (MRBI-MLGD) algorithm for two classes of structured
LDPC codes is presented based on a recent work by Huang et
al. Compared with the original one, the modified algorithm
has better performance with slightly increased complexity. Then
a reliability-based iterative min-sum decoding (RBI-MSD) al-
gorithm is presented. For the presented RBI-MSD algorithm,
reliability-based integer messages are processed and exchanged
between variable nodes and check nodes. The main computations
include only binary logical operations and integer additions.
Different from the conventional min-sum algorithm, the variable
nodes pass full messages rather than extrinsic messages to check
nodes. This can reduce the memory loads and the computational
complexity but with a little (or negligible) performance degrada-
tion. Simulation results show that, compared with the (M)RBI-
MLGD algorithms, the presented RBI-MSD algorithm achieves
better error performance, faster decoding convergence rate and
fewer quantization bits with moderate increased computational
complexity. Furthermore, the RBI-MSD algorithm is also appli-
cable to decoding random LDPC codes, a distinct difference from
the (M)RBI-MLGD algorithms. Finally, we point out that the
scaling factors employed in the MRBI-MLGD algorithm and the
RBI-MSD algorithm can be optimized using discretized density
evolution.
Index TermsвҖ”Low-density parity-check (LDPC) codes, min-
sum decoding, reliability-based decoding, soft-decision decoding,
structured LDPC codes.
I. INTRODUCTION
L
OW-density parity-check (LDPC) codes, which were
originally proposed by Gallager in 1962 [1], have been
known as a class of capacity-approaching codes. The de-
coding algorithms for LDPC codes can be categorized into:
1) hard-decision decoding; 2) probability-based decoding;
and 3) reliability-based decoding. Among these algorithms,
the sum-product algorithm (SPA) [2], [3] is the most well-
known probability-based iterative decoding algorithm which
provides the best performance but requires high computational
complexity. Several variants of the SPA have been proposed
to reduce the computational complexity, including the conven-
tional min-sum (MS) algorithm [4] and the offset/normalized
BP-based algorithm [5], [6], but they cause performance

decoding algorithms such as the one-step majority-logic de-
coding (OSMLGD) algorithm [7], [8] and the bit-flipping (BF)
algorithm [1], have the lowest decoding complexity but result
in performance loss. The reliability-based decoding algorithms
make efficient trade-offs between decoding performance and
computational complexity. The commonly used reliability-
based algorithms are the weighted bit flipping (WBF) algo-
rithm and its improved versions [7], [9].
Recently, two reliability-based decoding algorithms have
been proposed [10], [11], which are called reliability-based
iterative majority-logic decoding (RBI-MLGD) algorithm and
weighted iterative soft-reliability based majority-logic de-
coding (WISRB-MLGD) algorithm, respectively. These two
reliability-based decoding algorithms perform well for struc-
tured majority-logic decodeable LDPC codes with low decod-
ing complexity.
In this paper, we first present a modified reliability-based
iterative majority-logic decoding (MRBI-MLGD) algorithm
for two classes of structured LDPC codes. The MRBI-
MLGD algorithm achieves better performance with slightly
additional computational complexity. Second, we present a
reliability-based iterative min-sum decoding (RBI-MSD) al-
gorithm, which requires only logical and integer operations.
Different from the conventional belief-propagation algorithm,
the presented RBI-MSD algorithm passes full messages rather
than extrinsic messages from variable nodes to check nodes.
This further reduces the memory loads and the computational
complexity. Simulation results show that the presented RBI-
MSD algorithm achieves better error performance, faster de-
coding convergence speed and fewer quantization bits with
low computational complexity. In addition, unlike the (M)RBI-
MLGD algorithm which is specially devised for majority-logic
decodeable LDPC codes, the presented RBI-MSD algorithm
can also be applicable to decoding random-like LDPC codes.
The rest of this paper is organized as follows. In Sec-
tion II, we first discuss the RBI-MLGD algorithm and then
make a slight modification to improve its performance. In
Section III, we present a reliability-based iterative min-sum
decoding (RBI-MSD) algorithm. In Section IV, we make com-
parisons among these three reliability-based iterative decoding
algorithms in terms of complexities and performances. In
Section V, the scaling factors are optimized using discretized
density evolution. Section VI concludes this paper.

A. Differences Among the Three Reliability-based Algorithms
The presented RBI-MSD algorithm and the (M)RBI-MLGD
algorithms have the following differences with respect to the
reliability messages and the update strategies.
1) The RBI-MSD algorithm is an integer message-passing
algorithm, while both the RBI-MLGD algorithm and
the MRBI-MLGD algorithm are binary message-passing
algorithms. The integer reliability messages in the RBI-
MSD algorithm range from вҲ’(2рқ‘ҸвҲ’1вҲ’1) to +(2рқ‘ҸвҲ’1вҲ’1);
while, for the (M)RBI-MLGD algorithms, the messages
passing between variable nodes and check nodes take ei-
ther 0 or 1. Intuitively, the integer messages may contain
more useful information than binary messages do. Thus
it is possible to achieve better decoding performance.
2) Both the RBI-MSD algorithm and the MRBI-MLGD
algorithm use the initial messages рқ‘…(0)
рқ‘—
in the iterative
procedures, while the RBI-MLGD algorithm uses the
previously updated messages рқ‘…(рқ‘ҳ)
рқ‘—
in the iterative pro-
cedures.
3) Both the RBI-MSD algorithm and the MRBI-MLGD
algorithm use a scaling factor in the iterative procedures
to compensate the performance degradation. This makes
trade-offs between the performance and complexity.

VI. CONCLUSION
In this paper, we have presented a modified reliability-based
iterative majority-logic decoding (MRBI-MLGD) algorithm
based on a recent work by Huang et al. The MRBI-MLGD
algorithm utilizes a scaling factor to compensate the perfor-
mance degradation. Simulation results show that the presented
MRBI-MLGD algorithm obtains better error performance and
faster decoding convergence speed than its original one with
slightly increased complexity.
We have also presented a reliability-based iterative min-
sum decoding (RBI-MSD) algorithm. The main computations
of the RBI-MSD algorithm include only binary logical and
integer operations. Different from the conventional min-sum
algorithm, the variable nodes pass full messages rather than
extrinsic messages to check nodes. This further reduces the
memory loads and the computational complexity. Compared
with the (M)RBI-MLGD algorithms, the presented RBI-MSD
algorithm has three advantages: 1) it achieves better error
performance, faster decoding convergence rates; 2) it requires
fewer quantization bits to achieve the desired performance;
3) unlike the (M)RBI-MLGD algorithms which are specially
devised for structured majority-logic decodeable LDPC codes,
the RBI-MSD algorithm is also applicable to decoding of
random-like LDPC codes.
Finally, we have utilized discretized density evolution to
optimize the scaling factors for the two present algorithms.
