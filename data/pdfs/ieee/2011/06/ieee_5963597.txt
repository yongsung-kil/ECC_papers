Network Coding Scheme based on LDPC Product Codes in Multiple-Access Relay System

Zongyan Li, Mugen Peng, Zhanji Wu and Wenbo Wang
Key laboratory Universal Wireless Communications for Ministry of Education Beijing University of Posts and Telecommunications Beijing, China E-mail: zongyanl@bupt.edu.cn

Abstract—In this paper, we consider a multiple-access relay system with two users, one relay, and one destination. The relay combines the data packets received from these two users and then forwards the data packets to the destination. We consider a novel network coding scheme for information combing at the relay, namely, low-density parity-check (LDPC) product coding. For LDPC product codes, we propose a joint iterative channel decoding (JICD) scheme based on bipartite graph under the loglikelihood ratio belief propagation (LLR-BP) algorithm resist the interference and fading at the destination. Simulation results show that the packet-error-ratio (PER) performance of LDPC product codes based network coding scheme outperforms the traditional XOR-operation based network coding scheme.
Keywords- Network coding; LDPC product codes; Multipleaccess relay channel; LLR-BP algorithm;
I. INTRODUCTION Although the idea of network coding was first developed in [1] to improve the multicast capacity and to attain maximum throughput in wired networks, the application of network coding in wireless networks is being extensively studied [2]-[4], the benefit of network coding has been demonstrated in different communication scenarios, including multiple-access wireless network [5], two-way relay wireless network [6][7]. In this paper, we consider a multiple-access wireless network, which is known as a multiple-access relay channel (MARC) as shown in Fig. 1, in which two users U1 and U2 sharing one relay R. The relay combines the information received from these two users which is the essence of network coding, and then sends the compositive information to the destination D.
Fig. 1. Multiple-access relay model

In order to obtain better diversity gain provided by network coded channel coding, the design of network coded channel coding and decoding has received much attention. For instance, the traditional network coding which is based on XORoperation can be used to provide diversity gain. Furthermore, relay can also employ product coding [9] instead of XORoperation to combine the data from two users and to generate extra redundancy. Product codes can be considered as a set of matrices in which each row and each column are independently coded, the relay decodes the received data packets from two users, then re-encodes and places them along with the columns of coding matrix [11].
In this paper, we consider LDPC product codes based network coding and propose a JICD scheme based on bipartite graph [10] under LLR-BP algorithm at the destination. We will show how to realize the product codes and the main operations of JICD scheme.
This paper is organized as follows. In section II, we describe the system model and the network coding scheme based on LDPC product codes. In section III, we present JICD scheme based on bipartite graph of LDPC product codes. In section IV, simulation results for AWGN and Rayleigh fading channels are given and the performance of proposed scheme is also analyzed. Finally, conclusions are summarized in section V.
II. SYSTEM DESCRIPTION
A. System Model
For simplicity, we consider a time-division MARC system model, the block diagram of the system is depicted in Fig. 2, in which both users communication with destination with the help of one relay. Thus, in Fig. 2 one transmission data packets consists of three time slots, with two users, and relay taking turns to transmit. The destination always listens. In the first two time slots, each user transmits its own data packets. After decoding the different received user data packets, the relay performs network coded channel coding and forwards the coded data packets to the destination. Here, we consider such a system with one half-duplex relay and both users are close to the relay. Then, we can assume that both users have ideal channels to the relay.

IV. SIMULATION RESULTS
Assuming that each user employs LDPC code defined in the IEEE 802.16e (packet length is 480 information bits), iteration number of decoding is 50. We further assume that perfect channel state information (CSI) is assumed at the destination.
Fig. 5 depicts the PER over the Eb N0 of the direct link under AWGN channel. Channel coding rate is 5/6, The modulation scheme used is QPSK. It shows the PER performance comparison for LDPC product codes (LDPC-PC) based network coding, XOR-operation based network coding and the direct transmission link without network coding. For XOR-operation scheme, LDPC coding rate is also 5/6 at the relay. Whereas, for the LDPC-PC scheme, in order to keep the same efficiency with XOR-operation scheme, the ration of the LDPC code at the relay (denoted as the column in the coding matrix) should be about 2/3.

V. CONCLUSIONS
In this paper, we present a network coding scheme based on LDPC product codes at the relay. At the destination, the receiver collects the data packets from the two users, the redundancy data packets received from the relay, we present a JICD scheme based on bipartite graph to decode these received data packets. From the simulation results, we can see that network coding scheme based on product codes provided better diversity gain over conventional network coding scheme based on XOR-operation, where the gain comes from that the produce codes generate extra redundancy over the XORoperation scheme.
