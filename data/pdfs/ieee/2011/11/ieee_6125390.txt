Abstract—We propose an algorithm to design Root-Check
LDPC codes based on Progressive Edge Growth (PEG) tech-
niques for block-fading channels. The proposed algorithm im-
poses some restrictions on the traditional PEG algorithm to
ensure that the LDPC code generated is Root-Check with the
largest possible girth. The performance is investigated by means
of the outage probability. By doing so, the codes generated by
our proposed algorithm are shown to outperform the existing
methods for generating Root-Check LDPC codes.
I. INTRODUCTION
Due to multi-path propagation and mobility, wireless sys-
tems are characterized by time-varying channels with fluctuat-
ing signal strength. In applications with no delay constraints,
the channel can be considered ergodic, where each transmitted
symbol experiences independent fading realizations. Nonethe-
less, for applications subject to delay constraints and slowly
varying channels, only limited independent fading realizations
are experienced. In such non-ergodic scenarios, the channel
capacity is zero since there is an irreducible probability, termed
outage probability [1], that the transmitted data rate is not
supported by the channel. Thus, the block-fading channel
is a simple and useful model that captures the essential
characteristics of non-ergodic channels [2]–[4]. As a result,
we can consider the situation where a codeword with finite
length can be transmitted over only a few independent fading
blocks. Codes designed for block-fading channels are expected
to achieve the limited diversity that the channel provides and
offer good coding gains.
In [5] the authors proposed a family of Low-Density Parity-
Check codes (LDPC) called Root-LDPC codes for non-ergodic
block-fading channels. The Root-LDPC codes are shown to
achieve full diversity on block-fading channels and perform
close to the outage limit when decoded using the iterative
Sum-Product (SP) decoding algorithm. In the bipartite graph
representation of Root-LDPC codes, a subset of connections
are deterministically selected to guarantee full diversity for
information bits and the rest of connections are generated
randomly.
Recently Yueqian Li et al. [6] proposed the construction
of structured Root-LDPC codes by means of tilling circulant
matrices, i.e., by designing Quasi-Cyclic Low-Density Parity-
Check (QC-LDPC) codes. In [6] the authors show that the
QC-LDPC codes can perform as good as randomly generated
Root-LDPC codes over block-fading channels. It is known that
the girth, the length of the shortest cycle present in the graph
of the code, has a significant effect on code performance. For
regular codes, the number of independent messages passed
in SP decoding is proportional to the girth of the code [7].
Consequently, a number of code construction methods aim to
produce codes of large girth. From the promising results of
a number of early heuristic approaches for girth conditioning
on randomly generated codes, this was seen to be an effective
method for improving performance and further effort was
invested in finding methods to produce codes of large girth.
In [6] it is presented an algorithm to obtain QC-LDPC
codes with large girths. This algorithm is based on randomly
selecting the coefficients for the right cyclic shifted square
sub-matrices from a QC-LDPC code. The essence of this
algorithm is to regenerate the random coefficients up until a
code of desired girth is found. Among the algorithms capable
of producing LDPC codes of highest performance for short
to moderate lengths is the Progressive Edge Growth (PEG)
algorithm [8]. Given the code dimensions and variable node
degree distribution, this algorithm constructs the parity-check
matrix of an LDPC code through an edge by edge progression,
maximizing the local girth at each placement. This results in
practical length codes with relatively large overall girth and
with particularly large girth in the left-hand sub-graph of the
codes. Large girth in the left-hand sub-graph is of particular
value when, as is specified for the PEG algorithm, the variable
node degree sequence is presented in non-decreasing order as
this eliminates damaging on short cycles among low-weight
variable nodes. The codes produced by the PEG algorithm ex-
hibit improved performance compared to random construction
methods [8].
The contribution of this paper is to devise a PEG-based
algorithm to design LDPC codes with Root-Check properties,
thus providing Root-LDPC codes with larger girths than those
obtained by previously reported methods for the design of
LDPC codes in block-fading channels. The codes generated
by the proposed PEG-based algorithm for Root-LDPC codes
can achieve a significantly better performance in terms of
Frame Error Rate (FER) than previous works [5], [6]. We also
compare the performance of the Root-LDPC codes generated
by the proposed PEG algorithm to the outage probability of the
block fading channel. Moreover, we conduct some simulation

III. ERROR CONTROL CODING FOR BLOCK-FADING
CHANNELS
The contribution of this paper is the proposal of using a
PEG-based algorithm to generate Root-Check LDPC codes
with large girths. In this section, we will first give a brief
overview of Random Root-Check LDPC codes and Root-
Check QC-LDPC codes.
A. Root-Check LDPC Codes
Root-Check LDPC codes have first been proposed in [5] to
achieve maximal diversity in block-fading channels. The idea
can be best illustrated by means of an example. Consider a
(2,4) regular Root-Check LDPC code defined by the graph in
Fig. 1. The code is divided into two blocks, each of which
contains two systematic bits and two parity check bits. It is
easy to observe that each systematic bit in a block is connected
to a check node (a Root-Check for that systematic bit) which
is further connected only to variable nodes in the other block.
Having such a Root-Check allows for a systematic bit to be
reconstructed from other blocks when it is erased.
It is proved that the code in Fig. 1 achieves the same
performance as the outage probability curve in the case of
a block-erasure channel. Specifically, the systematic bits can
be reconstructed as long as at least one block is not erased.
However, for a general block-fading channel the code can
only achieve the maximal diversity order, while presenting a
performance gap from the outage probability curve [1].

VI. CONCLUSION
In this paper we have proposed a new PEG-based algorithm
to design Root-Check LDPC codes. Based on simulations,
the proposed method was compared to Random Root-Check
LDPC and Root-Check QC-LDPC codes. The results demon-
strate that the PEG-Root-Check LDPC codes generated by our
proposed algorithm outperform the previous published codes
[5], [6] for some 𝐸𝑏/𝑁0 values. Moreover, we are currently
devising a way to adapt the proposed algorithm for Improved
PEG (IPEG) [14], [15] with the goal of obtaining even better
codes, especially for the case of short block lengths.
