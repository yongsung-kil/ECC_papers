QC-IRA-d Codes Based on Circulant Permutation Matrices
Jianhui Wang, Jing Lei, Shaojie Ni, Xiaomei Tang, and Gang Ou
AbstractвҖ”This paper presents a new kind of quasi-cyclic
irregular repeat accumulate (IRA) codes based on circulant
permutation matrices called QC-IRA-d codes. The presented QC-
IRA-d codes have lower encoding complexity than traditional
IRA codes and can be decoded in a partially parallel decoder
like QC-LDPC codes. Simulation results show that QC-IRA-d
codes have good error-correcting performance.
Index TermsвҖ”QC-IRA codes, low encoding complexity, par-
tially parallel decoder.
I. INTRODUCTION
I
RA codes have linear encoding complexity and good
performance [1]. QC-LDPC codes based on circulant per-
mutation matrices [2] can be easily decoded in a partially
parallel way [3]. To combine the advantages of IRA codes
and QC-LDPC codes, we propose a new kind of codes called
QC-IRA-рқ‘‘codes. Due to space restrictions, the encoder and
partially parallel decoder of QC-IRA-рқ‘‘codes will be analyzed
in another paper.

A. QC-IRA-B Codes
To combine the characteristics of IRA and QC-LDPC codes,
a simple method is that let рқҗ»1 have the same form as рқҗ»рқ‘„рқҗ¶.
We call this kind of codes QC-IRA-B codes (base QC-IRA
codes). We will show that QC-IRA-B codes have a lot of low
weight code words, which results in poor performance. The
Tanner graph of a QC-IRA-B code is showed in Fig. 1. The
information nodes in the same dashed circle correspond to the
columns of the same sub-matrix рқҗј(рқ‘ рқ‘–,рқ‘—), and the check nodes
in the same dashed circle correspond to the rows of the same
sub-matrix рқҗј(рқ‘ рқ‘–,рқ‘—). The dimension of each рқҗј(рқ‘ рқ‘–,рқ‘—) is рқ‘қГ— рқ‘қ.
The thick edges in Fig. 1 contain two kinds of cycle
structures which result in low weight code words. These two
kinds of cycle structures are illustrated in Fig. 2 (a) and Fig.
2 (b). In Fig. 2 (a), (b), (c) and (d), every check node has
connections only with two variable nodes inside the cycle
structure (although it may have connection with variable nodes
outside the cycle structure), and every variable node has no
connection with check nodes outside the cycle structure. That
is to say, if the bits of the zero code word corresponding to
the variable nodes in Fig. 2 (a) are flipped from вҖҷ0вҖҷ to вҖҷ1вҖҷ,
the new vector is still a code word whose weight is 4 (due to
the structure of Fig. 2 (a), the check equations at the check
nodes connected to the four flipped bits are still satisfied). So
the cycle structure in Fig. 2 (a) corresponds to a weight-4
code word. Similarly, the cycle structures in Fig. 2 (b), (c)
and (d) correspond to a weight-5, weight-6 and weight-7 code
word respectively. From Fig. 2, we can conclude that degree-рқ‘Ҙ
information nodes of QC-IRA-B codes result in cycle structure
corresponding to a weight-(рқ‘Ҙ+2) code word.

IV. CONCLUSION
The proposed QC-IRA-рқ‘‘codes combine the advantages of
IRA codes and QC-LDPC codes, which have low encoding
complexity and partially parallel decoder. Simulation results
show that QC-IRA-рқ‘‘codes have good performance.
