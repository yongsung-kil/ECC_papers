Quasi Cyclic-LDPC codes based on PEG algorithm 
with maximized girth property

Abstract-In this paper, we propose a method to construct parity­
check matrices of Quasi Cyclic (QC) LDPC codes based on the 
progressive edge-growth (PEG) algorithm with maximized girth 
property. The proposed algorithm can eliminate short cycles and 
improve the decoding performance. Simulation results illustrate 
that the codes constructed with the proposed algorithm have 
superior performance to the previous PEG-QC codes over the 
additive white Gaussian noise (A WGN) channel. At the BER of 
10-7, the coding gain of 0.005 dB. 
Keywords-LDPC codes, quasi-cyclic LDPC codes, progressive 
edge growth (PEG) algorithm 
I. 
INTRODUCTION 
Low-parity density-check (LDPC) codes were invented in 
the early 1960's by Robert Gallager [1]. In 1990's, LDPC 
codes were rediscovered by MacKay and Neal [2]. LDPC code 
is a linear block code which shows a good performance 
approaching the Shannon's limit [2-3] when decoded with the 
soft decision belief-propagation (BA) or the sum-product 
algorithm (SPA). 
LDPC codes can be constructed by random methods. One 
type of the random LDPC codes is a progressive edge-growth 
(PEG) LDPC code [4]. This method constructs a parity-check 
matrix or, equivalently, a Tanner graph, on an edge-by-edge 
basis giving a large girth and shows an impressive decoding 
performance through iterative decoding. Unfortunately, a 
relatively large memory is required for storing the parity-check 
matrices of the random LDPC codes. Therefore, quasi-cyclic 
(QC) LDPC codes have been proposed to reduce the memory 
requirement based on block matrices with the circulant 
architecture [5]. The required memories can be reduced by a 
factor of lip, where p is the dimension of the circulant 
matrices. 
However, QC-LDPC codes have a limitation of 
flexible block sizes and code rates. Consequently, the QC­
LDPC codes base on PEG algorithm or PEG-QC- LDPC codes 
[6] have been introduced to improve the girth property and the 
memory requirement. These methods construct the parity­
check matrices based on the PEG algorithm with established 
circulant sub-matrices instead of edges. These codes establish 
the improved girth property, lower memory requirements and 
comparable decoding performance to the QC-LDPC codes. 
Recently, there are some methods to improve the random 
LDPC codes based on PEG algorithm [7-8]. An extended PEG 
algorithm [8] offers the large girth of random LDPC codes by 
considering the local girth on the symbol nodes before 
implementing the parity-check matrices, resulting in a high 
decoding performance. Motivated by this reason, we propose 
the girth-maximized QC-LDPC codes based on the PEG

check matrix based on the PEG algorithm and compares the 
girth property for every node. Then we select the nodes with 
the maximum local girth to create the circulant sub-matrix and 
insert it to the parity-check matrix. The aim of this method is a 
lower memory requirement due to circulant architecture and an 
improvement of PEG algorithm.

conclusions.

IV. 
SIMULATION RESULTS AND DISCOUSSIONS 
We compare the local girth of PEG, PEG-QC and the 
proposed PEG-QC codes. The dimension of the circulant 
matrix is 128 and the degree of each variable node is 3. The 
information block size is 4096 and the code rates considered 
are 0.91, 0.89, 0.86, 0.84, 0.74, 0.64 and 0.50. The local girth 
property of each code is shown in Table I. For low and medium 
code rates, PEG codes show a superior local girth property to 
the PEG-QC codes. However, for high code rates, both codes 
show a comparable local girth property. Our proposed method 
can maximize a local girth property for every code rate. In the 
addition, our proposed PEG-QC codes are comparable to the 
original PEG codes and superior to the PEG-QC codes. 
In Fig. 4, the performance between the PEG-QC codes and 
the proposed PEG-QC codes are presented. The performance is 
plotted in terms of the bit error rate (BER) versus signal-to­
noise ratio (SNR). In this simulation the code rate is 0.84, the 
dimension of the circulant matrix is 128 and the degree of each 
variable node is 3. At the BER of 10-7, the SNR gain of the 
proposed PEG-QC codes over the PEG-QC codes is about 
0.005 dB. At the BLER of 10-4, the proposed PEG-QC codes 
present the SNR gain about 0.005 dB compare with the PEG­
QC codes.

V. 
CONCLUSION 
In this paper, we propose a local girth optimized PEG 
algorithm for QC codes to construct the parity-check matrix 
with maximum local girth. Simulation results show that the 
proposed PEG-QC codes are superior in the BERIBLER 
performance to PEG-QC codes and show SNR gain of about 
0.005 dB at the BER equal to 10-7.
