Abstract‚ÄĒ A new technique for constructing the Quasi Cyclic Low density Parity Check (QC-LDPC) codes based on splitting rows into sub rows method is proposed. The proposed QCLDPC code, at BER of 10-6 achieves 0.1dB coding gain over the PEG-LDPC codes. Moreover, in the high Eb / N0 region, the proposed code with girths 16 and 20 respectively, achieve 0.056 dB and 0.27 dB at a BER of 10-5 over the randomly constructed code. Simulation results portray that large girth QC-LDPC codes achieve a 0.1dB coding gain over randomly constructed codes and perform 1.3dB from the Shannon-limit at a BER of 10-6 with a code rate of 0.89 for block length of 1332.
Keywords: BER, permutation matrix, QC-LDPC, row, coding

I. INTRODUCTION

LDPC codes are generally designed to be linear and binary block codes. In this case there is a generator matrix G that converts a message vector m into a code vector c by means of a matrix multiplication. The corresponding parity check matrix H has the property that it is constructed with linearly independent row vectors that form the dual subspace of the subspace produced by the linearly independent row vectors of G. This means that every code vector satisfies the condition H ‚ó¶ c = 0.

Quasi-Cyclic class of LDPC codes forms a large type of codes with flexible encoding and decoding properties, these have been comprehensively planned in the sense that their hardware is both cheap and easy to implement. Since both encoding and decoding require less memory, these codes have many advantages for hardware and software implementations. This memory advantage is catered by being able to illustrate the matrices employing a series of short polynomials. Generally QC-LDPC code construction is based on random and algebraic construction. The algebraically constructed QCLDPC codes perform significantly better as compared to the random regular LDPC codes. However, the drawbacks of the algebraic structure codes are limitation in terms of code rate, codeword length and code girth. The codes performance is inferior to the random generated codes when using longer block length.
In work [1], seed matrix for construction of LDPC codes are employed, which is the chain of two relative incidence matrices for Fano planes and circulant permutation matrices. The effect on decoding performance is investigated by employing sum-product algorithm for column weight 3. Authors show by using BER simulations that large girth codes perform better than those with lower girths. A new necessary and adequate condition for determining the girth of QC-LDPC codes is derived in [2], based on the theory of adjacency matrices, without an explicit enumeration of cycles. It is shown from simulation results that the obtained codes are often with performance comparable to the LDPC codes constructed by progressive-edge-growth (PEG) algorithm [3]. The above row splitting results in a new parity check
matrix H with qmk‚ąí1 rows which has the following structural
properties: (1) each row has weight k (2) every column has weight j (3) any two rows (or columns) have at most one 1component in common. Such a sparse parity-check matrix is said to be (j, k) regular and the code generated by it is called (j, k) a regular code. The constraint on the rows and columns of H given by property (3) is called the row-column (RC) constraint. Therefore, the above row splitting results in a new parity check matrix with smaller density. Moreover, when the parity check matrix is systematic, one can easily use it to extract the density of the code.
The main goal of row union is to simplify the support of multiple rates. It provides a simple overall architecture with a smaller chip area required to support all the needed rates. If the message passing is done through a memory, there is a list of memory addresses that tell the processor which variable-tocheck messages to compute in order to compute a check-tovariable message. With row-union, changing the rate of the code can be achieved by replacing the lists for the check nodes with a single list that is the union of those lists, thus changing only the quantity of messages read.

Figure 2 shows the bit error performance comparison of QC-LDPC codes with other renowned LDPC codes. Simulation results show that new QC-LDPC codes with girth 8 and codeword length 200 outperform the LDPC codes constructed by PEG algorithm in [4] with same code rate. The PEG-LDPC codes are using smaller girth (6 in this simulation), than the proposed QC-LDPC codes (girth 8). The proposed QC-LDPC code, at BER of 10-6 achieves 0.1dB coding gain over the PEG-LDPC codes [4] for 40 maximum numbers of iterations. PEG algorithm results freely concentrated check-node degree distributions, while rigorously concentrated check-node degrees are appropriate for the maximization of convergence speed of the codes as well as the hardware implementation of iterative decoder. In the PEG construction all the check nodes in the set have accomplished their maximum degree, which lower the girth of the graph. With the aim to maintain the girth, the check nodes should have the same degree; in the proposed work the 1component selection in the row sustains the girth of the graph. Therefore, it can be concluded that the advantages proposed by the new QC-LDPC codes accumulate without any compromise in the attainable BER performances.

IV. CONCLUSION
In this paper a novel technique for QC-LDPC codes construction is developed. In that method, the restructuring of the interconnections is developed by splitting the rows into sub rows. Such a division reduces the hardware complexity. The new QC-LDPC codes are flexible in terms of girth, code rates and codeword length. This kind of structural design deals the process across a number of computational blocks and is well suited for high speed hardware implementations. The new QC-LDPC codes can be used as the FEC codes for wireless data transmission. Therefore, it can be concluded that the advantages proposed by the new QC-LDPC codes accumulate without any compromise in the attainable BER performances.
