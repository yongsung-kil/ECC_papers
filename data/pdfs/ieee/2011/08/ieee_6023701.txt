I. INTRODUCTION

Low-density parity-check (LDPC) codes, which were originally proposed by Gallager in 1962 [1], have been known as a class of capacity-approaching codes and chosen as the standard codes in many applications such as wireless, optical, satellite communication and digital video broadcast systems. The decoding algorithms for LDPC codes can be categorized into: 1) hard-decision decoding; 2) probabilitybased decoding; and 3) reliability-based decoding. Among these algorithms, the reliability-based decoding algorithms can make efficient trade-offs between decoding performance and computational complexity. The well-known reliabilitybased algorithms are the weighted bit flipping (WBF) algorithm and its improved versions [2], [3].
In 2009, Huang et al proposed a reliability-based iterative majority-logic decoding (RBI-MLGD) algorithm [4], which requires only logical operations and integer additions. For the RBI-MLGD algorithm, specific soft reliability messages are defined from the received quantized symbols and iteratively updated in the decoding process. It has been shown by simulations that the RBI-MLGD algorithm can perform well with very low decoding complexity.
In this paper, we present a modified reliability-based iterative majority-logic decoding (MRBI-MLGD) algorithm for the structured LDPC codes based on the work in [4]. The presented MRBI-MLGD algorithm is an integer message passing algorithm and mainly requires the logical and integer operations. Simulation results show that, compared to the original RBI-MLGD algorithm, the MRBI-MLGD algorithm can achieve a better performance with slightly additional computational complexity.

A. Complexity Analysis
As shown in [4], the RBI-MLGD algorithm requires 28 + n — m binary operations and 6 integer additions per iteration, where 5 = nj = mp denotes the number of nonzero elements in H. Compared to the IRB-MLGD, the presented MIRB-MLGD in section IV requires extra n real multiplications to scale the total extrinsic messages. Actually, this can be implemented by n lookup table operations. Since the size of the lookup table is very small (2 x 27), the additional complexity due to the lookup table operation is neglectable.

VI. CONCLUSION
We have presented a modified reliability-based iterative majority-logic decoding (MRBI-MLGD) algorithm based on the work by Huang et al. The MRBI-MLGD algorithm utilizes a scaling factor to compensate the performance degradation. The main computations of the presented MRBIMLGD algorithm includes only binary logical and integer operations. Simulation results show that the presented MRBI-MLGD algorithm obtains better error performance than its original one with slightly increased complexity.
