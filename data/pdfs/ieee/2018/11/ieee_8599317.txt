The 2018 5th International Conference on Systems and Informatics (ICSAI 2018)
LDPC Code Design via Masking Technology and Progressive Optimization

Dongliang Guo
School of electronics and communication engineering Sun yat-sen university Guangzhou, China

Abstract— A design of low-density parity-check (LDPC) code via the masking technology and progressive optimization is proposed. The code generated by this method has lower computational complexity, and their parity-check matrices can effectively refrain from the girth-4 phenomenon. The proposed method has the superiorities such as better girth-length characteristic and more flexible trim in the length and rate of the code. Experiment results indicate that the error-correction performance of the new code should be better than or as good as the capability of the code which is constructed without been masking. Futhermore, under the condition of short code length, the performance is better than that of LDPC codes via the randomized construction methods.
Key words: LDPC code; mask; progressive optimization; base matrix
I. INTRODUCTION
Under the condition of iterative decoding, low-density parity-check (LDPC) code is the code whose performance in error correction is closest to the famous Shannon capacity limit in known channel coding technology [1, 2]. Therefore, LDPC code has broad application prospects and has been applied to 5G mobile communication, optical communication, deep space communication, digital video broadcasting, data storage system, IEEE 802.16e and other communication fields.
QC (Quasi-Cyclic)-LDPC codes are a very important category of LCPC codes [3, 4]. Because the parity-check matrix of QC-LDPC codes has quasi-cyclic property, its encoding module may be implemented only using cyclic shift registers, which greatly reduces the encoding complexity and memory requirements, and its decoding module is easy to implement parallel computing. Therefore, QC-LDPC code is very practical and is a coding scheme with great potential.
Masking technology can enhance the performance of LDPC codes by changing some elements of parity-check matrix by zero [5]. For QC-LDPC codes, the corresponding parity-check matrix is highly structured and elements in it are also very dense. The density of the matrix could be prominently reduced by superseding part of the circulant permutation matrices (CPMs) with null matrices by masking technique. After masking, the edges of the corresponding Tanner graph of the new generated matrix become less, thus the girth length could be observably increased. The key of masking technology is the optimal design of the vital masking

matrix. Research of constructing LDPC codes by masking technology is still in the exploratory stage, and there is no universally accepted masking matrix selection scheme. In this thesis, a new and novel design method of QC-LDPC code via masking technology and progressive optimization is suggested. The proposed scheme is able to enhance the capability of the generated codes, and the code-length and code-rate of the new generated codes can be adjusted more flexibly.

B. Construction Method The QC-LDPC codes are the most widely studied codes in
the structured LDPC codes [6].
Up to now, among the proposed LDPC code construction methods, the construction methods of QC-LDPC codes include finite geometry method [7], finite field-based construction method [8, 9], Reed-Solomon (RS) code-based construction method [10], balanced incomplete block design (BIBD) method [11] and so on.

III. THE PROPOSED MASKING METHOD
Some CPMs in the base matrix Hb = [hi,j ] can be
replaced by all-zero matrices. This process is called masking.
Masking the pivotal base matrix can make the parity matrix more sparsely, at the same time reduce the quantity of edges in Tanner diagram, thus reducing quantity of short rings, and may get larger girth [5, 12-18].

B. New Construction Method via Masking PEG algorithm [19] or ACE (Approximate Cycle Extrinsic
message degree) algorithm [15] has been applied to optimize
the masking matrix Z in some literatures, but the algorithm is
very complex, another problem is that these methods ignore the influence of the original matrix when considering the construction of masking matrix.
This thesis presents a novel simplified scheme to determine the key masking matrix, and the influence of the original matrix is considered when constructing them.

satisfying the given rate and the given code length according to a certain method, such as finite field based [9], PEG [19] based algorithm and so on.

elements in the set Φ have been tried. In order to avoid over-
centralization of 0 elements, a column-by-column approach is used, when an element of a certain column is assigned zero, continue to search the next column.

B. Analysis of the Parameter w The value range of parameter w is 0 to u . If the value of
parameter w is 0, it means that no matrix element is protected, and if the value of w is u , it means that all matrix elements
are protected, that is, no masking is made.
If the parameter w begins to increase from 0, the
performance of error correcting codes will also increase within
a certain range. However, the value of w is generally less than u / 2 . When w exceeds u / 2 , the performance of the
masking technology is not improved significantly, or even worse.

D. Conclusion In this thesis, a new design scheme of LDPC codes via
masking technology and progressive optimization is proposed. By masking the elements in the base matrix, the quantity of short loops in check matrix is reduced, and the error correction ability of the codes is improved. In addition, the flexible adjustment of code length and encoding rate might be realized by this masking method. The key parameters of the proposed algorithm are analyzed and simulated.
Experiment results indicate that the final error-correction performance of the new designed LDPC codes using this construction method is better than or as good as the capability of the codes constructed without masking. Futhermore, under the condition of short code length, the performance of new codes can exceed the capability of LDPC codes generated by randomized construction method.
