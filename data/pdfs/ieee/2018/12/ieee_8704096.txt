Abstract— Min sum algorithm (MSA) is widely used for decoding Low Density Parity Check (LDPC) codes in many modern Digital Video Broadcasting (DVB) as a simplification for Sum Product Algorithm (SPA) which greatly reduce the implementation complexity. This paper presents a design of modified variable scaling min sum LDPC decoding algorithm for DVB-S2/S2X/T2. The proposed design is a combination between the variable Normalized Min Sum Algorithm (NMSA) and the Offset Min Sum Algorithm (OMSA) which provides a trade-off between decoding complexity and error correcting performance. It improves the use of the Min Sum Algorithm (MSA) by 39%. A proposal for how to implement the proposed algorithm using FPGA is also introduced. The structure used is partially parallel layered decoder architecture.

Keywords— Low Density Parity Check (LDPC), adaptive normalizationfactor, OffsetM in Sum Algorithm (OMSA), DVB S2/S2X/T2.

I.

INTRODUCTION

In early 1960s, Gallager first introduced Low Density Parity Check (LDPC) codes [1,2,3,4,5] and in 1996 were rediscovered by Mackay and Neal [3,5], In recent years, LDPC received much research attention due to its remarkable coding performance and efficiency that is very close to the Shannon limit [1,6,7]. These codes are adopted in many modem video broadcasting standards such as DVB-S2/S2X and DVB-T2 because of their improved Bit Error Rate (BER) performance [1,8]. The Sum Product Algorithm (SPA) is one o f the most efficient algorithm used to realize the LDPC decoder. It achieves an excellent performance on the expense of hardware implementation complexity [1,3]. Due to the limitation in the hardware resources of the Application Specific Integrated Circuit (ASIC) and Field Programmable Gate Array (FPGA), a balance between decoding complexity and error correction capabilities must be adopted [9]. To be able to satisfy that goal and to simplify the high complexity of SPA, Min Sum Algorithm (MSA) was introduced [1,3,6,10,11], The MSA used a simplified way by using summation and comparison operations in updating the check node (CN) to the variable node (VN) message which results in some significant degradation in the decoding performance [6,11]. In order to improve the performance o f MSA, many algorithms were published. The most effective methods are Normalized Min Sum Algorithm (NMSA) and the Offset

Improved NMSA has been proposed in [3], where two normalization factors are used with the same decoding complexity as the NMS algorithm. In [12], rather than subtracting a constant offset factor from CN messages; they simply erase the value of the least significant bit (LSB). The decided scalar parameter which is used in the NMSA and OMSA, corrected the degradation problem of the MSA at the CN [4]. Further improvement on NMSA and OMSA was presented in [13], [14], [6], [1], This enhancement was achieved by changing the scaling parameter, adaptively changed per decoding iteration which improves the performance. Another important modification was the adaptive normalized Min-Sum algorithm which proposed in [1], It proposed a simplified variable scaling min-sum algorithm (SVS-MSA) which utilizes a logical heuristic equation to compute the scaling factor sequence. This paper proposes a modification on the simplified variable scaling Min-Sum algorithm by adding an offset scaling factor to the CN calculations. The proposed algorithm provides a balance trade-off between decoding complexity, error correcting capabilities and easy to implement. The proposed decoding algorithm can be adopted in DVB-T2 & DVB-S2/S2X. The paper provides Matlab simulation results of the proposed decoder algorithm for half rate Irregular (8100, 4050) LDPC code. Finally, the implementation of the MS algorithm using partially parallel layered decoder architecture for the (8100, 4050) LDPC code is also introduced. The paper is organized as follows: section II introduces decoding algorithms for LDPC codes. The proposed algorithm is discussed in Section III. Section IV presents the simulation and results for the proposed algorithm and comparison with other published decoders. Section V discusses the FPGA hardware implementation for the MS algorithm. Finally, the summery and conclusion are in Section VI.

The implementation of the MSA can be divided into three main architectures; Parallel, serial and partially parallel architecture. The partially parallel architecture is the best way to implement the LDPC DVB codes as it is less complex compared to the fully parallel architecture. Moreover, it uses less memory and higher throughput than the fully serial architecture. The proposed design adopts the partially parallel layered decoding architecture. The (8100, 4050) LDPC is divided into 30 layers each layer is (811, 135). Figure 5 shows the top level design for FPGA implementation. The design consists of 8 main blocks.

The LLR RAM is used to store the Log-Likelihood Ratio (LLR) for the initial use. Then the Demux 1 will distribute the LLR values to the variable nodes. The Variable Nodes blocks will take the initial values with the check node values and will calculate the reply message. The Registers are used to shift the values o f the Variable node. The Check Nodes blocks will take the values coming from the Registers and will calculate the reply message for the Variable Nodes block. The reply values will be returned to the Variable Nodes blocks through Demux 2 after storing them in the Check Node RAM.

VI. Summary & Conclusion
An irregular LDPC decoder which is suitable for DVBS2/S2X/T2 is presented in this paper. The proposed LDPC decoder is based on a combination between SVS-MSA and OMSA. The proposed decoder is simulated using Matlab tool for (8100, 4050) which is a shorten code from DVB82 normal frame. The simulations are performed with AWGN channel noise and QPSK modulation technique. The results show that the proposed algorithm shows better error correction performance compared to the other algorithms. It improves the BER by 17% compared to the adaptive NMSA in [1] and 39% from the MSA at S=5. For S=7 the BER is improved by 2% compared to adaptive NMSA and 6% from the MSA. It also shows an improvement by 3% compared to adaptive NMSA for S=10 and 12. By comparing with MSA the proposed algorithm gives 6% improvements for S=10 and 12 % for S=12. A proposal for how to implement the proposed decoder using FPGA is suggested. It is based on partially parallel decoder architecture to compromise between hardware complexity and fast response. The suggested (8100, 4050) LDPC architecture composes o f 30 layered each with (811,135).
