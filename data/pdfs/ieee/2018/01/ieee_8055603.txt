Memory-Reduced Look-Up Tables for Efficient ADMM Decoding of LDPC Codes
Xiaopeng Jiao , Yu-Cheng He , and Jianjun Mu

Abstract—The Euclidean projection involved in the decoding of low-density parity-check (LDPC) codes with the alternating direction method of multipliers (ADMM) can be simplified by jointly using uniform quantization and look-up tables (LUTs). However, the memory requirement for the original LUT-based ADMM decoding is comparatively large. In this letter, a nonuniform quantization method is proposed to save the memory cost by minimizing the mean square error of the outputs of Euclidean projections during quantization. Simulation results over two exemplified LDPC codes show that the proposed method can achieve similar errorrate performances when compared with the original LUT-based ADMM decoding by using significantly less memory units.
Index Terms—Alternating direction method of multipliers (ADMM), Euclidean projection, low-density parity-check (LDPC), look-up table (LUT), nonuniform quantization.
I. INTRODUCTION
R ECENTLY, a promising linear programming (LP) decoding using alternating direction method of multipliers (ADMM), simply noted as ADMM decoding, is proposed for low-density parity-check (LDPC) codes [1]. Compared with the belief propagation (BP) decoding, the ADMM penalized decoding, which includes a penalty term in the objective function, exhibits similar FER performances at waterfall regions and better FER performances at error floor regions [2]. It was also shown that the ADMM decoding has computational complexity comparable to that of the BP decoding [3].
The computational complexity of ADMM decoding is dominated by Euclidean projections [4]. Various methods for simplifying the projection operations have been proposed. In [3], the hyperplane which contains the projection point is first determined by using a cut search algorithm, and then the projection point is found by solving a simple optimization problem. In [5], the projection operation is simplified by using a linear projection algorithm onto simplex, which is effective especially for high

Besides the projection simplification methods above, the techniques of projection reduction [7] and message scheduling [8], [9] have also been investigated for reducing the overall ADMM decoding complexity.
More recently, the computational complexity problem of Euclidean projections is challenged by means of look-up tables (LUTs) [10]. Simulation results have shown that the LUT-based ADMM penalized decoding can achieve a substantial reduction in the decoding time while maintaining the FER performances of ADMM decoding. However, the efficiency advantage is achieved at a comparatively large memory cost, which may prevent this decoding technique from practical applications. In [10], Euclidean projections are approximated with input vectors uniformly quantized. It was shown that a sufficiently large quantization range plays a key role in maintaining the FER performance, and a small quantization interval is more important to improve the convergence performance and hence the decoding time efficiency. As a joint result, the number of quantization levels could be comparably large.
In this letter, we propose a nonuniform quantization scheme for the LUT-based projections in order to further challenge the memory cost problem. Generally stated, nonuniform quantization in a proper range can reduce the number of quantization levels while satisfying the convergence requirements for quantization intervals. For this purpose, a rule of minimum mean square error (MMSE) is employed so that the statistics of projection inputs can best be incorporated in the design.

III. NONUNIFORM QUANTIZATION DESIGN
A. Observation of Projection Input Distribution
Generally stated, uniform quantization can be optimal only when the inputs are uniformly distributed. For nonuniformly distributed inputs, nonuniform quantization using unequally separated levels may produce less distortion. To investigate the distribution of Euclidean projection inputs to be quantized, we consider two example rate-1/2 LDPC codes.
1) The Margulis regular LDPC code with M = 1, 320, N = 2, 640, and check degree d = 6, noted as C1 [11].
2) The IEEE 802.16e irregular LDPC code with M = 288, N = 576, and check degrees d = 6, 7, noted as C2 [12].

It has been shown there exists a small quantization range [a, b], which is code-dependent and can be determined by simulations of the LUT-based ADMM decoding [10]. C. Searching Algorithm With Finite Precision

For the LUT-based ADMM decoding, the design problem of

nonuniform quantization is to find an optimal kernal set that

produces the least MSE metric. Clearly, this problem presents

a continuous search problem over the given range. For small L

values, we may convert it to a discrete search problem by using

a finite precision.

In Algorithm 1, we present a framework for discrete search

with given precision τ and SNR. Let T = (b − a)/τ and T0 =

T /2 . First, the range [a, b] is divided into T intervals of equal

size τ . There must exist T0 − 1 boundaries in the range (a, 0.5).

Then, collect them as a base set of levels. Each combination of

K − 1 levels from the base set constitutes a candidate for the

kernal set. Finally, the candidate that produces the least MSE is

selected as the optimal kernal set Qopt .

The discrete search can be accomplished in an exhaustive

method. The number of search rounds is given by |B| =

T0 −1 K −1

.

In our simulations, we consider τ = 0.02 and the largest K = 4

for practical purposes. For code C1, we use [a, b] = [−1.3, 2.3], which yields T0 = 90 and hence |B| = 113, 564. For code C2 , we use [a, b] = [−1.5, 2.5] instead, then T0 = 100 and hence |B| = 3 764 376. In these cases, the involved computations are

acceptable for exhaustive search.

In case of large K values which make it hard to perform

an exhaustive search, a dynamic programming method in [13]

may be employed. Moreover, instead of being converted into

a discretized problem, the nonuniform quantization design can

also be approached by a continuous search method based on

differential evolution [14], which has been successfully applied

in various coding scheme designs [15], [16].

V. CONCLUSION
In this letter, we have investigated the design of nonuniform quantization for the LUT-based ADMM decoding to achieve a memory cost reduction, in addition to the efficiency advantage. To solve the optimization problem for nonuniform quantization, we have proposed a discretized search method by first introducing the kernal set of quantization levels, and then using a proper precision to dividing the quantization range. Under the MMSE rule, an optimal kernal set for nonuniform quantization is searched with the given precision. Simulation results show that nonuniform quantization have advantages in both error rate performance and convergence performance over uniform quantization. It can also provide a good tradeoff between the FER performance and the memory cost.
