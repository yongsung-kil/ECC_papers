Simulated Annealing Method for Construction of High-Girth QC-LDPC Codes
Vasiliy Usatyuk†, and Ilya Vorobyev‡ †Department of Computer Science, South-West State University, Kursk, Russia ‡Department of Mathematics, Institute for Information Transmission Problems, Moscow, Russia
Email: l@lcrypto.com, vorobyev.i.v@yandex.ru

Abstract—We present a simulated annealing method that construct high-girth quasi-cyclic low-density parity check (QCLDPC) codes. The proposed method is applicable to both regular and irregular protograph codes. The method show improvement in term of minimal circulant size compare with previous described methods: Hill-Climbing and improved PEG. Simulated results are presented to demonstrate performance gain of proposed construction method in error-floor region under base graph 2 (BG2) using 5G eMBB standard length adaption method.
Keywords—circulant permutation matrix (CPM); extrinsic message degree (EMD); girth; quasi-cyclic (QC)
I. INTRODUCTION
Low-density parity-check (LDPC) codes were first discovered by Gallager [1], generalized by Tanner [2], Wibberg [3] and rediscovered by MacKay et al. [4], [5] and Sipser et al. [6].
Quasi-cyclic low-density parity-check (QC-LDPC) codes are of great interest to researchers [7], [9]–[11] since they can be encoded and decoded with low complexity and allow to reach high throughput using linear-feedback shift register [13]–[15]. One advantage of QC-LDPC codes based on circulant permutation matrices (CPM) is that it is easier to analyze their code and graph properties than in the case of random LDPC codes. Recently several methods for construction QC-LDPC codes have been proposed: Bit-Filling [16], Guessand-test [17], Hill-Climbing [18], Progressive Edge-Growth (PEG) [19], Improved PEG [20]. Most of these methods have main objective to avoid short cycles. Short cycles and it union produce subgraph responsibly for performance loss under suboptimal linear complexity marginals estimation using belief-propagation decoding method [21]. The performance of LDPC codes is strongly affected by their graph properties such as the length of the shortest cycle, i.e., girth [17], [18], and trapping sets [22], [23] and code properties, e.g., the distance of the code [24], [25] and the ensemble weight enumerator [26]. Proposed simulated annealing method allow to construct QC-LDPC codes with better cycles properties. This algorithm especially important for construction of high performance QC-LDPC codes. This expurgate codes have extremely low threshold value and require strong cycle broke algorithm to get require level of error-floor produced by Trapping sets pseudo-codewords, [27].
The main contribution of the paper is a novel approach for constructing high-girth quasi-cyclic LDPC codes with

The trapping set (a, b) is a subgraph formed by cycles in

Tanner graph or their overlap with a variable nodes and b odd

degree checks. For example, Trapping sets TS(5,3) formed

by overlap of three 8-cycles, and TS(4,4) formed by cycle 8

in Tanner graph, [22]. The harmfulness of a trapping set is

defined as the number of variable nodes, error in which can

cause suboptimal iterative decoder to fail on this TS. Under

equal probability of error, TS(5,3) more harmful, because 3

errors in odd degree check nodes provide 5 errors in variable

nodes. The TS(4,4) produces 4 errors after 4 errors in odd

degree check nodes, and can be consider as a pseudo-codeword

with weight 4. Broken of the most harmful cycles improve

pseudo-codewords weight spectrum which lower probability

of error-floor. It’s harmfulness are strongly related to decoder

parameters. Parameters modification can change probability of

decoder fail due to TS pseudo-codewords, [28].

III. SIMULATED ANNEALING METHOD FOR CONSTRUCTION QC-LDPC CODES
Consider a QC-LDPC code with mL × nL parity-check matrix H with circulant size L, m×n exponent matrix E(H) = (Eij(H)) and mother matrix M (H). Code length of this code N = L×n. Simulated Annealing method allow to construction of QC-LDPC with different objectives:
1) girth g; 2) minimal EMD value; 3) minimal pseudo-codewords weight.
All objectives are require high-girth QC-LDPC codes construction. Girth maximization is a simple objective because require search of sequence of circulant shift Eij(H) = aij in exponent matrix E(H) corresponding Tanner graph with girth g. For this task you can use greedy search of sequence of circulant shift according (3) or another greedy search methods: PEG, Improved PEG. Greedy search lifting methods not guaranty globally optimal solution of cycles in corresponding Tanner graph. Proposed simulated annealing lifting method allow to found high-girth codes in term of minimal circulant size compare with previous methods.

During application of proposed method the following observations were made:
Observation 1. Converge of simulated annealing lifting is very sensitive to seed value of random number generator. For example, simulated annealing lifting of regular mother matrix with row number m = 3 and column number n = 5 with girth 12 using seed=123 require 193512 second (315770 iterations) and 11807 second (9328 iteration) using seed=777. We propose to run simulated annealing with several different seeds simultaneous to negligible effect of initial parameters.
Observation 2. For some even bigger circulant permutation matrix sizes (circulant values) than obtain in Tables I, II simulated annealing lifting never converge even after 107 iterations. Probably for some circulant permutation matrix sizes parity-check matrix which meet girth requirement exist with very low probability (or even not exist), due algebraic property of equation (3). It is very useful to have mathematical proof of this observation to predict ’catastrophically lengths’. This observation indirectly confirmed by simulation results, peaks on information lengths K = 740, K = 980, K = 1216 Figure 4.

IV. SIMULATION RESULTS
QC-LDPC codes of smaller lengths can be obtained by lifting exponent matrix of QC-LDPC codes of maximal length.

Their performance under sum-product decoder with 50 iterations over an AWGN channel with QPSK modulation was analyzed by computer simulations. Figure 4 shows the SNR value required to achieve 10−5 FER performance for information lengths for QC-LDPC codes with rate 1/3. Code A is the QC-LDPC with mother matrix ”base graph 2” (BG2) from 5G eMBB standard, [35]. The BG2 iterative decoding threshold was optimized for frame-error rate 10−2. To show advantage of our QC-LDPC code construction method we use simulated annealing under same BG2 mother matrix (protograph). BG2 codes show catastrophic performance for some lengths caused by error-floor. Code B is BG2 lifted using same length adaptation from 5G eMBB standard. Our code construction approach improve cycle properties of lifted codes, decrease number of harmful Trapping sets and show from 0.1 to 0.3 dB gain compare to 5G EMBB codes.
V. CONCLUSION
In summary, the proposed QC-LDPC code construction method have high success rate, allow to improve cycles structures. Simulation results are presented to demonstrate the efficiency of simulated annealing method in terms of minimum circulant permutation matrix and error performance. It allow to construct irregular QC-LDPC Codes with 0.1 to 0.3 dB gain compare to 5G eMBB codes.
