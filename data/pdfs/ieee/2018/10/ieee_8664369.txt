Abstract—The frame error rate (FER) of spatially-coupled irregular low-density parity-check (SC-iLDPC) code ensemble in error floor region is estimated. First, the number of codewords of each weight is calculated by the permutations of all the sub-codewords at their coupling positions. Second, the FER is estimated by these numbers of the codewords of each weight. Numerical results show that the FER performances in the error floor regions of the SC-iLDPC code ensembles are superior to the conventional irregular LDPC code ensembles at almost identical belief-propagation (BP) thresholds.
I. INTRODUCTION
Spatially-coupled low-density parity-check (SC-LDPC) code was constructed by connecting multiple regular LDPC codes to a coupled chain. Its belief-propagation (BP) threshold is close to the Shannon limit [1], and its code rate is determined by the length of the coupling chain and the (integer) degrees of both the variable and check nodes of the regular LDPC codes. A finite length of chain results in a rate-loss to a designed rate.
Spatially-coupled irregular LDPC (SC-iLDPC) code has been constructed by assigning each component LDPC block with irregular degree distribution in a finite-length coupled chain [2] [3] [4]. Its parity-check matrix is formed by replacing each non-zero entry in a base matrix with a random, nonsquare, irregular superposition matrix. The desired code rate of the SC-iLDPC code is flexibly obtained by adjusting the size of the superposition matrices. Moreover, its BP decoding threshold is closer to the Shannon limit than some conventional SC-LDPC codes [3] [4]. Note that the best BP threshold is given when the weight of the columns in the base matrix is two [3]. The occurrence of low-weight variable nodes motivated us to estimate the frame error rate (FER) performances in the error floor region of the SC-iLDPC code.

We next compare only the numbers of weight-2 codewords A2 between the SC-iLDPC and conventional irregular LDPC code ensembles, since A2 dominates the error floor performance for a given erasure probability ϵ. We give numerical examples A2 of the SC-iLDPC code ensembles in Table II. For them, their degree distributions of superposition matrices with L = 128 are listed in Table III, and corresponding BP thresholds ϵ∗ are listed in Table II. Note that these distributions and thresholds are optimized by differential evolution [8] and density evolution [9][10]. In Table III, the degrees of check nodes are different at each code rate. The distributions in Table III are not distributions of the SC-iLDPC code.

Figure 5 gives the curves of BP threshold ϵ∗ vs. A2 (corresponding to L) at each code rate of the SC-iLDPC and irregular LDPC code ensembles. The degree distributions of these code ensembles at each marker are optimized by the density evolution. In Fig. 5, there is a trade-off between ϵ∗ and A2, and the values of A2 of the SC-iLDPC code ensemble are always lower than those of the irregular LDPC code ensembles at the same BP threshold ϵ∗. This means that the decoding performance in the error floor region of the SC-iLDPC code ensembles is better than that of the conventional irregular LDPC code ensemble at almost identical BP threshold.

V. CONCLUSION
We estimated the number of codewords of each low weight
in the SC-iLDPC code ensembles and the FER performances.
Numerical results show that the FER performances in the error
floor regions of the SC-iLDPC code ensembles are superior
to the conventional irregular LDPC code ensembles at almost
identical BP thresholds.
