Design of LDPC Codes for Joint Satellite and Terrestrial Broadcasting System

Na Gao, Yin Xu, Dazhi He, Genning Zhang, Wenjun Zhang Cooperative Media Network Innovation Center of Shanghai Jiao Tong University, China

Abstract - In this paper, we introduce a joint satellite and terrestrial broadcasting system which is called Next Generation Broadcasting-Wireless and Satellite (NGB-WS) and will be adopted in Greater China area and some other parts of Asia. This system requires good performance particularly for combined decoding. Under this requirement, we propose an algorithm to globally design a series of QC-IRA LDPC codes based on mother code via punctured GADE(Gaussian Approximate Density Evolution) theory to facilitate the combined decoding. However, in the case of joint decoding, if the satellite and terrestrial broadcasting system uses the same punctured pattern, the combined performance is far away from optimal. To this end, we specially propose an algorithm to select the punctured patterns to better the combined performance based on protograph GADE. Compared with the LDPC codes in DVB-S2, the designed punctured based QC-IRA LDPC codes have a comparable performance in single code rate, and have significant gain when two codes with same/different rates are combined decoded in AWGN and Rayleigh channels.
Keywords-- joint satellite and terrestrial broadcasting system, punctured QC-IRA LDPC, combined decoding

II JOINT SATELLITE AND TERRESTRIAL BROADCASTING SYSTEM
Due to the fact that the satellite broadcasting system or terrestrial broadcasting system cannot provide massive reliable coverage for these cities alone, a joint satellite and terrestrial broadcasting system will be applied to the nationwide broadcasting system as shown in Fig.1. In this joint system, the satellite and terrestrial broadcasting systems will cooperate with each other to transmit signal with high accuracy, alleviating the influence of the obstructions from buildings, tunnels and trees etc. The joint satellite and terrestrial broadcasting system has the five scenarios as follows. z User receives message from only one satellite. z User receives message from more than one satellite. z User receives message from terrestrial broadcasting system. z User receives message from satellite and terrestrial broadcasting system. z User receives message from more than one satellite and terrestrial broadcasting system.
Fig. 1 Joint satellite and terrestrial broadcasting system Nevertheless, this joint system is too expensive to be actually applied if the operation cost is taken into consideration. Therefore, we propose two application scenarios, which are more likely to be applied in practice and can satisfy the users’ requirements. z scenario of single code In most cases, the signal from one transmitter can satisfy the requirements of users. Thus, the LDPC codes used in this system need to have good single performance. z scenario of combined codes If the user needs the high-quality signal and it can be covered by more than one satellite (or by one satellite and terrestrial system), the LDPC codes with the same rate(or different rates) will be translated and combined decoded. Thus, the LDPC codes used in this system need to have good combined performance. To conclude, in this joint satellite and terrestrial broadcasting system, we need to design a series of LDPC codes

which have better combined performance with almost no loss in performance of single code compared with the LDPC codes in conventional broadcasting system.

V OPTIMIZATION ALGORITHM TO SELECT THE PUNCTURED
PATTERNS
Once the mother code is generated, the degree distributions of the single punctured codes are determined. The single performance of these punctured codes is also determined due to the fact that we puncture the parity bits deliberately, which is briefly introduced in Section III. So we need to optimize the combined performance.

However, if the two satellites and the ground use the same punctured pattern base on the mother code in this joint system, some message of parity nodes will be lost, which will influence the combined performance. So we propose to puncture the different parity bits to reduce the overlapped parity message while combined decoding these LDPC codes.

To select punctured patterns for the LDPC codes at different rates, we can apply the protograph to estimate the combined performance of codes with the different punctured patterns. Compared with degree distributions, protograph is also related to the edge connection. That’s why punctured GADE based on protograph is more accurate to select the punctured patterns than punctured GADE based on degree distributions.

Take scenarios of this joint system into consideration, two punctured patterns (mode1 and mode2) will be designed to satisfy the joint system. To estimate the performance of combined LDPC codes, we apply the GADE based on protograph, which helps us to select punctured patterns of the two modes. Compared with the conventional GADE based on protograph which is used to estimate the threshold, the combined LDPC codes are transmitted from different transmitters. Thus, the external LLR message is the sum of LLR message from different transmitters.

VI PERFORMANCE ANALYSIS In this section, the performance of LDPC codes designed for this joint satellite and terrestrial broadcasting system will be briefly introduced. The simulations use the sum-product algorithm based belief propagation decoding over AWGN and Rayleigh channel with 50 iterations and QPSK modulation.
We design a rate 1/2, length 64800 LDPC code as mother code and compare it with same rate and length codes in DVBS2. Besides, we puncture the mother code to construct the punctured code with rate 2/3 and length 48600 and compare it with same rate and 64800 length code in DVB-S2.

The design logic of LDPC codes in this paper is specially to improve the combined decoding performance of the codes, quite different and actually independent from the design logic of single code which is focused on optimization of single code performance. It means the proposed joint design schemes of this paper can be applied to any codes on top of the corresponding single code design algorithm. In specialty, though we use DVB-S2 codes (because DVB-S2 is a satellite system) as reference codes, actually our joint design logic can be applied to any codes (such as ATSC3.0 codes), to improve their joint decoding performance.
1) Single performance
As shown in Fig.3, we will find that the performance of our codes is comparable with that of LDPC codes in DVB-S2 in AWGN channel, while the length of proposed 2/3 code is shorter than 64800. In Rayleigh channel, the proposed code with length 48600, rate 2/3 is slightly worse than the code with length 64800, rate 2/3 in DVB-S2. The shorter length in this system is the main cause of this result. But if the user need highquality signal, the loss can be complemented by this joint system.
2) Combined Performance
We combine the 1/2 rate mother code and 2/3 rate punctured code. As is shown in Fig.4, the system performance is much better than that in DVB-S2, where the LDPC codes are not designed for combined decoding.
Besides, Fig.4 can also prove the superiority of the puncture patterns selected by the Algorithm2. In details, in the case of combined decoding for two same rate codes, if we select different punctured patterns for two codes according to Algorithm2, the performance is much better than the case that two codes using same punctured pattern. As is shown in Fig. 4 there is 0.6dB and 0.75dB gain in the case of 2/3 rate 64800 length codes under AWGN and Rayleigh channel respectively.

VI CONCLUSION
In this paper, we introduce a joint satellite and terrestrial broadcasting system and propose algorithms to design LDPC codes particularly for better combined performance which can better the system performance.
To meet the requirements of this system, algorithm1 is proposed to globally optimize the mother code and puncture codes. The LDPC codes designed by algorithm1 have comparable performance of the single code decoding and good performance of the combined decoding compared with that in DVB-S2. To better the combined performance, this paper also proposes algorithm2 to select the punctured patterns for different transmitters by estimating the combined performance using protographbased GADE. The combined performance of punctured 2/3 codes from different transmitters has 0.6dB and 0.75dB gain in AWGN and Rayleigh channels compared with that of the conventional puncturing method.
