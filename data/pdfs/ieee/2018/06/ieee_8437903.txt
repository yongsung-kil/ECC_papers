AbstractтQuasi-cyclic low-density parity-check (QC-LDPC) codes are an important class of LDPC codes which can be encoded and decoded with low complexity and suitable for many applications. As the code dimension, which describes the number of protected information bits, is equal to the code length minus the rank of the parity-check matrix and the parity-check matrix for QC-LDPC codes is usually not full-rank, determining the rank of the parity-check matrix is of essential importance.
In this paper, we study the rank of the parity-check matrix for QC-LDPC codes based on the associated polynomials for circulant matrices. A formula for the rank of the parity-check matrix with only one row-block is first derived. We then extend the result to matrices with two, three, or more row-blocks. Some bounds are also presented for matrices with arbitrary numbers of row-blocks. Furthermore, the exact rank is determined for a class of algebraically constructed parity-check matrices.
I. INTRODUCTION
Low-density parity-check (LDPC) codes [1] have been shown to have near-capacity performance with iterative message-passing decoding and sufficiently long block length. Quasi-cyclic low-density parity-check (QC-LDPC) codes [2]т [6], whose parity-check matrices are composed of blocks of circulant matrices, are a class of LDPC codes with rich algebraic structures and have the advantages of efficient encoding and decoding in VLSI implementation. Well-designed algebraically constructed QC-LDPC codes can have similar good performance as randomly constructed LDPC codes.
As the code dimension which corresponds to the number of protected information bits is equal to the code length minus the rank of its parity-check matrix, knowing the rank of a paritycheck matrix is essential. Quite a large number of parity-check matrices for QC-LDPC codes are not full-rank. Even though Gaussian elimination can be employed to find the exact rank with a complexity of O(n3) where n is the number of rows (or columns), determining the rank of a parity-check matrix analytically is important for certain algebraic constructions of LDPC codes, e.g., the variable-rate constructions in [7]. A number of research efforts [8]т[14] have been made for rank analysis of parity-check matrices for QC-LDPC codes by Fourier transform; however, the results are subject to some restricted choices of the sizes of circulant matrices. In this paper, based on the associated polynomials for circulant matrices, we develop an alternative approach to study the rank of the parity-check matrix for QC-LDPC codes. Our method can be applied to matrices over general finite fields and with arbitrary sizes of circulant matrices. A formula for the rank of the parity-check matrix with only one row-block is first derived. We then extend the result to matrices with two, three, or more row-blocks. Some lower and upper bounds on the rank are also given for matrices with arbitrary numbers of row-blocks. Furthermore, the exact rank is determined for a class of algebraically constructed parity-check matrices for QC-LDPC codes with good error-correcting capabilities in [4].
The remainder of the paper is organized as follows. In Section II, we introduce the associated polynomials. Rank analysis for different kinds of parity-check matrices are given in Section III, along with some lower and upper bounds. In Section IV, the exact rank of a class of algebraically constructed parity-check matrices is found. Finally, the work is concluded in Section V.
