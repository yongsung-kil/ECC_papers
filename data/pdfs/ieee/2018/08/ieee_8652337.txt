An Improved Belief Propagation Algorithm Based on Exponential Model

Yuhuan Wang, Hang Yin, Zhanxin Yang, Yanling Huang Engineering Research Center of Digital Audio&Video Ministry of Education, Communication University of China, Beijing, China
Email: wangyuhuan@cuc.edu.cn, yinhang@cuc.edu.cn, yangzhanxin@cuc.edu.cn, hyl2015@cuc.edu.cn

Abstract—This paper proposes a new belief propagation algorithm to reduce the computational complexity of Low Density Parity Check codes. In the iterative process of the proposed algorithm, an exponential model is adopted to segmental approximate the hyperbolic tangent function. In addition, two kinds of nonlinear operations are simplified into one nonlinear operation. Simulations reveal that this proposed decoding algorithm can reduce the computational complexity of decoding algorithm with a minimal loss of performance.
Keywords—LDPC, Log Likelihood Ratio Belief Propagation Algorithm, Exponential Model, Segmented Approximation
I. INTRODUCTION
In 1962, Gallager put forward Low-density parity-check (LDPC) codes in his doctoral dissertation [1]. At the simulation experiment level, LDPC codes were demonstrated to be capable of approaching Shannon limit [2]. If the value of element in a matrix is one or zero, and the number of one is far less than the number of zero, the matrix is called a sparse matrix. LDPC codes have a mapping relationship with its parity check matrix. The sparsity of the parity check matrix makes LDPC codes with low complexity and high performance feasible.
LDPC codes are candidate decoding scheme for error control in 5G mobile communication systems, and are also common channel coding method in many communication systems. LDPC decoding algorithm plays a crucial role in its performance. According to the different computing method of information in decoding decision process, the algorithms are divided into hard decision and soft decision algorithms.
Soft decision algorithms are widely used in decoding process, especially belief propagation algorithm. Although BP algorithm has better performance than other algorithms, it needs very heavy nonlinear computation, and its hardware implementation is fussy.
Therefore, many improved BP algorithms were proposed. Using the monotonicity characteristics of function, Fossorier invented Min-Sum algorithm (BP-Based algorithm) [3]. In order to improve the performance, J. Chen designed two algorithms by adding scale operation and offset operation respectively, which were Normalized BP-Based decoding algorithm and Offset BP-Based decoding algorithm [4].
Based on the improvement considerations mentioned above and BER performance requirements, many different algorithms had been proposed [5,6,7]. The transformation between different operations brings about low approximate precision, which negatively affects the decoding performance.
In this paper, hyperbolic tangent function is replaced with the composite exponential functions in decoding process. Compared with LLR BP algorithm, this method can reduce the nonlinear computational complexity with a minimal loss

of decoding performance. In addition, the performance of the proposed BP algorithm is better than Normalized BP-Based algorithm.
The contents of this paper are organized as follows. The theory of LLR BP algorithm is presented in section Ⅱ. In section Ⅲ, the improved BP algorithm based on exponential model is introduced, and the decoding complexity of LLR BP algorithm and the improved BP algorithm are analyzed. Section Ⅳ introduces the performance comparisons of the improved BP algorithm and other three decoding algorithms. Finally, conclusions are drawn in section Ⅴ.

II. LOG LIKELIHOOD RATIO BELIEF PROPAGATION DECODING ALGORITHM
This section describes the Tanner graph representation of LDPC codes and the log likelihood ratio (LLR) BP algorithm. The calculated value of information in LLR BP algorithm is the logarithmic form of that in probability BP algorithm.

A. LDPC Codes The most basic representation of LDPC code is ( N, K ),
where N is code length and K is the number of information bits. The appearance of Tanner graph makes the expression of LDPC code more straightforward.
The Tanner graph is a bidirectional graph model, which is composed of variable nodes and check nodes. The row weights and column weights of the parity check matrix are consistent with the degree of nodes.
For example, (1) shows a parity-check matrix with N=8 and K=4. The code length is 8, and the number of information bits is 4.

IV. ANALYSIS OF ALGORITHM PERFORMANCE Simulations are carried out in order to test the performance of the proposed Exp BP algorithm. The code rate is 1/2 and code length is 2304. Simulations are conducted over the AWGN channel with BPSK modulation. The scale parameter of the Normalized BP-Based algorithm is 0.78.
A linear approximation of the topic nonlinear function in LLR BP algorithm has already been done in [8]. In this paper, the improved algorithm in [8] is called Linear BP algorithm.
Fig. 2 illustrates BER performance comparisons in the condition of the maximum iterative number is 10. The BER performance of Exp BP algorithm is almost the same with LLR BP algorithm, and is better than Normalized BP-Based algorithm and Linear BP algorithm. Beyond that, when BER=10-2, the Eb/N0 of Exp BP algorithm increases 0.1db compared with the Normalized BP-Based algorithm, and also increases 0.05 db compared with the Linear BP algorithm.

The analysis about BER performance, the average iterative number and computational complexity show that when signal-to-noise ratio is low, the performance of BER and the average number of iteration of LLR BP algorithm and Exp BP algorithm under different iteration times have little difference. The Linear BP algorithm has a simple iterative updating formula in check update process, but the average iterative number is higher than other simulation algorithms. Compared with Normalized BP-Based algorithm and Linear BP algorithm, Exp BP algorithm has better BER performance and lower average number of iteration.
V. CONCLUSION In this paper, we investigate an improved BP algorithm based on exponential model. This algorithm reduces the computation complexity by using the composite exponential function to approximate nonlinear function in the check node update process. Two nonlinear operations are simplified into one nonlinear operation. Simulations show that it can reduce the complexity of LLR BP algorithm with a minimal loss of decoding performance. In addition, the performance of the proposed BP algorithm is better than Normalized BP-Based algorithm and Linear BP algorithm.
