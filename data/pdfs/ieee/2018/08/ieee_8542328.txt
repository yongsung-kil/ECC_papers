Efficient Construction for QC-LDPC Convolutional Codes with Periodic Bit-Filling

Ming Zhao, Zhipeng Liu, Ling Zhao
China Academy of Electronics and Information Technology (CAEIT) Beijing, China
e-mail: mysr002@126.com, zhipengliu@live.cn, zhaoling@ee.buaa.edu.cn

Abstract—QC-LDPC-C (Quasi-Cyclic Low Density ParityCheck Convolutional) codes are with low encoding and decoding complexity and can achieve decoding performance approaching the Shannon limit. However, the construction of the parity check matrices need to be free of the length-4 cycles, and the computational complexity of direct construction will increase exponentially without considering the characteristics of the matrices. The construction for QC-LDPC-C codes with periodic bit-filling method is proposed. With the periodicity of the base check matrix, the proposed method firstly fills the deterministic submatrices to realize fast encoding; then it avoids all possible length-4 cycles by periodic filling in the construction of random submatrices. Thus the base check matrix without cycles of length-4 can be obtained, and the girth of expanded check matrix is at least 6. LDPC-C codes with different parameters are used to compare with the constructed QC-LDPC codes. Experimental results show that the codes constructed with proposed method can achieve better performance and lower encoding and decoding complexity.
Keywords-QC-LDPC-C codes; matrix construction; matrix expansion; periodic bit-filling; fast encoding, girth
I. INTRODUCTION AND SUMMARY
LDPC-C (Low Density Parity-Check Convolutional) code is a class of convolutional codes [1-2] defined on a sparse parity-check matrix, which can be used for the encoding and decoding with any length of data continuously, thus it has promising application in the communications with variable frame lengths or with continuous mode [3]. The LDPC-C code has the performance approaching the Shannon limit [1, 2, 4] and the semi-infinite parity-check matrix for QC-LDPC-C (Quasi-Cyclic LDPC-C) code comprises Circulant Permutation Matrices (CPMs), thus it can be encoded and decoded efficiently and is convenient for the hardware implementation [5,6].
The constructions for LDPC-C codes currently are derived from the unwrapping matrices of LDPC block codes. It is proposed that the LDPC-C codes can be constructed from the parity-check matrices for LDPC block codes with the cutting, exchanging and replicating method [1]. The parity-check matrix for LDPC-C code can be constructed with circulant permutation submatrices [5], thus simplifying the complexity of matrix construction. In the construction of LDPC-C codes, the PEG (Progressive Edge Growth) construction for LDPC block codes can also be used [7]. The method is presented that LDPC-C codes construction is achieved from the LDPC block codes with graph-cover-

based approach, and the constructed LDPC-C codes outperform the corresponding LDPC block codes, analyzing the "convolutional gain" [8]. The LDPC-C codes with fast encoding can also be obtained by using the multiplicative groups in the finite field and some related modifications [9]. In addition, there are other methods to construct the paritycheck matrices for LDPC-C codes [10].
The LDPC-C decoding algorithm is accomplished with BP (Belief Propagation) iterative algorithm based on the check rows, therefore the parity-check matrix for LDPC-C code, especially QC-LDPC-C code, needs to consider the problem of short cycles [7]. The LDPC-C code can be derived from the LDPC block code; however it will change the structure of the matrix at this time and there may be length-4 cycles in the matrix for LDPC-C code especially when the parameters are random positive integers. Moreover, the direct construction of parity-check matrix for LDPC-C code will increase the computational complexity of construction algorithm exponentially without considering the structural characteristics of the matrix.
To obtain better decoding performance and improve the girth of the code avoiding length-4 cycles, QC-LDPC-C code construction method based on periodic bit-filling algorithm is proposed. Considering the efficient encoding, in the construction of base check matrix for QC-LDPC-C code, both the numbers of non negative values in each row and column of parity-check portion for each block row are 1s. Based on the periodicity of parity-check matrix for QCLDPC-C code, the proposed construction method firstly fills in the positions of deterministic parity-check portion submatrices in each block row; and then it fills in the corresponding positions with periodical expansion, during which it can be free of all possible length-4 cycles including the cycles formed by the periodically expanded positions and the positions to be filled. Therefore, there are no length-4 cycles in the base check matrix and the girth of expanded check matrix is at least 6. The QC-LDPC-C codes constructed with proposed method can fulfill fast encoding and achieve better decoding performance while with low encoding and decoding complexity.

IV. SIMULATION RESULTS AND DISCUSSION The LDPC-C codes with different parameters are used in the simulations. The experiment is conducted over the Binary Phase Shift Keying (BPSK) modulation and Additive White Gaussian Noise (AWGN) channel, using different Signal to Noise Ratios (SNRs). The maximum number of iterations is 30. For the different codes, the Bit Error Rate (BER) performance with Layered BP (LBP) decoding algorithm is shown in Fig. 3 and Fig. 4.

V. CONCLUSION
The QC-LDPC-C code can obtain approaching Shannon limit decoding performance with the optimized design while with low encoding and decoding complexity, and can encode and decode continuously for any length data. However, the

construction for QC-LDPC-C code needs to avoid length-4 cycles, and the computational complexity is high. QCLDPC-C codes construction based on periodic bit-filling method is proposed. According to the periodicity of base check matrix for QC-LDPC-C code, the proposed method firstly determines the positions of check portion in the block row; and then it needs to remove all length-4 cycles composed of the periodically determined positions and forthcoming positions when one variable node is selected for each column. Thus, the base check matrix without length-4 cycles can be obtained, and the girth of expanded check matrix is at least 6. The computational complexity of proposed construction method is relatively low, which is the three powers of check row numbers. Experimental results show that the base check matrix for QC-LDPC-C code based on the proposed method is free of length-4 cycles, and the code can achieve better decoding performance while with low encoding and decoding complexity. In addition, the proposed construction method can also be used in other convolutional codes.
