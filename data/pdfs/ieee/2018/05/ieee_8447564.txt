Two improved algorithms for layered QC-LDPC decoding algorithm
Xiao Zheng, Qingsheng Hu* , Jun Feng Institute of RF- & OE-ICs Southeast University Nanjing, China

Abstract—Low-Density Parity-Check code is a kind of nearoptimal error correction code. In order to improve the performance of layered decoding algorithms, two algorithms are put forward in this paper. In the first algorithm, additional messages are employed for the messages updating operation of each layer by using another layered part, making it possible to accelerate the speed of convergence. In the second one, a correction operation for certain type of errors is performed to improve the decoding performance. Simulation results show that more decoding gain and accelerated convergence can be obtained compared with the conventional layered decoding algorithms.
Keywords—LDPC, QC-LDPC, layered decoding algorithm, column weight.
I. INTRODUCTION
The low-density parity-check(LPDC) codes which were invented in 1960’s by Gallager[1], were rediscovered in late 1990’s by Mackay and Neal, and had been proved to approach the Shannon-limit performance when the code length is extremely large[2-3]. Quasi-cyclic LDPC (QC-LDPC), one class of LDPC, attracts more and more attention due to its low complexity in encoding and decoding. In fact, the decoding of QC-LDPC is easier compared with its encoding. With the help of tanner graph theory, the decoding matrix of LDPC can be transformed into connected check units and variable units, making the decoding process as the updating and transmitting of check–to-variable (C2V) and variable-to-check (V2C) messages, respectively. Typically, bit-flipping algorithm (BFA) and sum-products algorithm (SPA) presented by Gallager in his thesis are two famous decoding algorithms. Over thirty years later, Mackay and Neal developed a practical beliefpropagation (BP) decoder based on SPA[3]. Although SPA has good decoding performance, it requires complex computations. To solve the problem, in 1999, the min-sum algorithm (MSA) was proposed to reduce the complexity of SPA[5]. Compared with SPA[6], MSA can reduce the complexity of the check node operations significantly at the cost of 0.5 dB performance loss. The modified MSA (including normalized MSA(NMSA) and offset MSA)[7-9], however, has extremely close performance to SPA.
To further reduce the required number of decoding iterations, layered decoding scheme was proposed in [10] and [11] for binary codes and was widely adopted for QC-LDPC. It has been shown that layered decoding schedule converges in approximately half the number of iterations at most signal-to-

With little computing resource, it can improve the decoding performance especially in high SNR area.
II. QC-LDPC AND DECODING ALGORITHM
A binary LDPC can be represented by a sparse parity-check matrix(PCM) H and H•xT=0, where x is a codeword. As one of most important sub-categories of LDPC, QC-LDPC can be viewed as an array of square sub-matrix, each of which is a BuB zero matrix or identity matrix shifted by a non-negative number.
There are two types of message-passing schedules, one is called flooding message-passing schedules, where C2V messages and V2C messages generated from previous iteration are updated simultaneously and the other is sequential message-passing schedules, where the node messages are updated sequentially instead of simultaneously. Although BP and MSA can also be used in the flooding schedule, it has been verified that sequential scheduling can accelerate the convergence speed as well as reduce the number of iterations.
Recently, layered decoding algorithm belonging to sequential schedules has driven much attention by both academy and industry because of its speeding up of convergence and less iterations. There are two layered decoding strategies: row-layered and column-layered decoding[14-15]. The main difference between them is the type of layer. In the former, the rows of PCM are grouped into layers while in the latter the columns of that are grouped. It has been proved that either row-layered or column-layered are theoretically equivalent and have better decoding performance than non-layered BP algorithm[16].
Fig. 1 shows a PCM H of code C partitioned into P(=3) layers each consists of M/P(=6/3) rows of H and the P layers can be viewed as the PCMs of 3 block codes. Generally rowlayered decoding can be implemented by sequentially decoding these P block codes and passing the extrinsic values from one layer to the other. In other words, the C2V messages computed from the decoding of a layer are used to update the V2C messages for the decoding of the next layer right away. III. IMPROVED LAYERED DECODING ALGORITHMS

A. New-messages Added Layered Algorithm
It has been known that layered schemes can update messages from previous layer instead of updating messages from previous iteration, which benefits the convergence of decoding algorithm. However, traditionally the decoding of the next layer can not start until that of the current layer is completed due to the data dependency. In other words, any variable, not contained in last layer, will not be updated in last layer, which means this variable has to use unchanged APP data to participate in update operation in this layer. To speed up the transmitting of new messages, this paper proposes a newmessages added layered algorithm (NALA), in which the check nodes in n-th layer achieve new messages not only from the (n-1)-th layer but also from other layers. Using this method, more new messages can be utilized and more updated variable can join layer decoding as soon as possible.

IV. SIMULATION RESULTS
To compare the performances of the two proposed decoding algorithms with that of some typical algorithms, simulations are performed for (1120,840) irregular LDPC code from IEEE P802.3bn™D3.0 with BPSK modulation over a AWGN channel. Some typical algorithms include NMSA, layered NMSA, and layered NMSA with column-1fixed are simulated. The number of test frames in each Eb/N0 is 50000 and the maximum iteration time is 5. By the way the codes are all encoded by a typical encoding algorithm [17].
Fig.5 shows the performance comparison of different layered algorithm. We can see that the decoding algorithms based on layered schemes perform far better than not layered one. Besides that, NALA has extra 0.1~0.15 dB decoding gain compared with layered NMSA in all Eb/N0 area. What’s more, with the advantage of increased convergence, consumption of calculation resource is also reduced. For column-1 error correction strategy, it works better in high Eb/N0 area. There are less errors emerging in high Eb/N0 area which makes it easier for column-1 error correction strategy to be triggered. We can

V. CONCLUSION We have implemented two algorithms based on layered QC-LDPC decoding algorithms. NALA trades calculation resource to increased convergence and column-1 errors correction strategy focus on certain type errors according to the certain structure of decoding matrix. The simulation result proves that both of them have better performance than typical layered decoding algorithm.
