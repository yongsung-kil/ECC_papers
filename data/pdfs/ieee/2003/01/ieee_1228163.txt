I. INTRODUCTION
Many results on the iterative Q-SP decoding can be found such as in [1]-[31. In this paper, we address the question of how best t o use a limited number of bits t o represent log-likelihood ratio (LLR) messages for the Q-SP algorithm in [2]. For a given maximum iteration number, we practically evaluate the Q-SP decoding with finite-iteration threshold in conjunction with the performances in bit-error-rate (BER), average iteration number as well as error floor.
11. ANALYSIOSF THE Q-SP ALGORITHM
Assume the same quantization with finite levels for LLR messages throughout the quantized decoding of LDPC codes. The step size and the largest level both are important to take into account. The SP algorithm performs better than the minsum algorithm particularly in terms of BER. However, the Q-SP algorithm becomes equivalent to the quantized min-sum (Q-MS) algorithm despite the number of bits used if the step size is greater than the critical value of 2log(d-l), where d is the maximum check degree of the code 131. In order to remove the dependence of the Q-SP decoding on the code itself, we consider the pairwise updating rules at both the variable and the check nodes [2][3]. The critical value is then a constant 210g2. With a probable loss in performance, both regular and irregular codes can be decoded by using the same algorithm with greatly reduced implementation complexity.
Note that the choice of the largest level may affect the convergence of the Q-SP algorithm since quantization overflow generally impairs discrete density evolution. A small largest level may yield a serious quantization overflow, and hence a n error floor of the decoding. The empirical value of the largest level must be at least 15. In that case, quantization overflow may be mitigated and the maximum discrete messages can have enough reliability to improve the convergence of the iterative decoding. This results in a requirement of at least 5 bits for avoiding the degradation of the Q-BP to the Q-MS if uniform quantization is used.

111. NONUNIFORQM-SP ALGORITHMS

Quantization would be finer over a region of small LLRs.
