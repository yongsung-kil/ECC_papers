Results on Non-uniform Error Correction Using Low-Density Parity-Check Codes
Hossein Pishro-Nik, Nazanin Rahnavard, and Faramarz Fekri School of Electrical and Computer Engineering Georgia Institute of Technology Atlanta, Georgia 30332-0250 Email: {hossein, nazanin, fekri}@ece.gatech.edu

Abstract‚Äî We propose a technique for designing low-density parity-check (LDPC) codes over nonuniform channels. In particular, we investigate LDPC codes for volume holographic memory (VHM) systems . We show that the proposed coding scheme for holographic memories has easy design procedure and results in efficient codes. An important property of the proposed technique is that we can design simple codes whose performance is close to the Shannon limit, while they are very good in terms of the error floor effect. We also derive a capacity bound and the stability condition for the proposed codes over the binary erasure channel. We briefly discuss other applications like punctured codes, OFDM systems and multilevel coding.

II. NON-UNIFORM ERROR CORRECTION
A. VHM systems
In some practical applications, such as page oriented memories, we need to perform non-uniform error protection. For example, in holographic data storage, the information is recorded and retrieved in the form of two-dimensional data pages, i.e, two-dimensional patterns of bits. These bits are subject to different sources of noise. However, the SNR decreases as we move from the center to the corner in the page. Typically, raw bit error rate might vary by two orders of magnitude over a page. The common approach to solve the non-uniform error protection problem is to use an interleaver followed by an error correcting code [3]. It has been shown that if we use an LDPC code that is optimized for the nonuniform channel, we can get a much better result [4], [5] . In fact, the simulations show that we can increase the storage capacity of a typical holographic data storage by more than 50 percent if we use the optimized LDPC codes instead of an interleaver and an RS code. III. PRACTICAL CODE DESIGN
Let us consider the problem of designing efficient LDPC codes for the VHM systems. LPDC codes are very good candidates for these systems. This is because as we mentioned before, the whole data in a page is read or stored simultaneously. Therefore, it is possible and desirable to use long LDPC codes. It is known that long LDPC codes can have performance close to the Shannon limit. However, this is not sufficient because it is desirable to have a bit error rate less than 10‚àí12 in the VHM systems. This implies that the code should not have error floor. In fact, this is an important criterion for the code. In this section we show that we can easily design good codes for the VHM systems using the ensemble g(Œõ, œÅ).
It is shown in [6] that if Œª2œÅ (1) < 1, then the minimum distance and the size of the minimum stopping set in the expurgated ensemble increase linearly with the code length. On

the other hand, if Œª2œÅ (1) > 1, these quantities are logarithmic with a high probability. However, it is shown in [7] that for capacity achieving codes we must have Œª2œÅ (1) > 1. Therefore, as it is mentioned in [6], to achieve the capacity, we must have a small minimum distance. This implies that the capacity achieving codes have an error floor . In fact, the capacity achieving codes of practical lengths usually have an error floor before the bit error rate of 10‚àí7. Now, we discuss the code design for the non-uniform error correction. For simplicity, we first consider the binary erasure channel. the problem of the VHM systems in which we have (BIAWGN) channels will be discussed later.

Let discuss the design of codes from g(Œõ, œÅ). First, to prevent the error floor problem we require that Œª2œÅ (1) < 1. Second, we choose the degree distribution to be semi-regular. By a semi-regular degree distribution, we mean a degree distribution in which the variable nodes of the same type (variable nodes corresponding to bits that are transmitted through the same channel) have the same degree. C. Other applications
The approach we took in this paper to design LDPC codes for the nonuniform error correction have several other applications. Punctured LDPC codes, OFDM systems and multilevel coding are among these applications.
Puncturing is one of the most common methods to construct rate-compatible codes. In this method, to change the rate of a code to a higher rate, we puncture (delete) a subset of the codewords bits. It is possible to show that, a punctured code can be

modelled as a code that is used over two parallel channels as Figure 1. In this model, punctured bits are transmitted over the second channel that has a zero capacity. Then, the density evolution formulas given in this paper can be used to find optimum puncturing distributions for LDPC codes.
In OFDM systems we deal with several parallel channels in which some bits experience higher SNR‚Äôs. In this case, we are not usually concerned about the error floor problem, but we have other restrictions. Specifically, we cannot use long codes. It can be shown by simulations that for finite-length cases, codes from the ensemble g(Œõ, œÅ) can have better performance than the ordinary irregular codes over the parallel channels. A similar situation exists in multilevel coding.
IV. CONCLUSION
In this paper we proposed a method to design good LDPC codes over a set of parallel channels. This method is useful for many applications such as the volume holographic memories, OFDM and rate-adaptive coding systems. We showed that the proposed method has several advantage over the conventional method. First, the design procedure is very simple since we do not need to perform the high-complexity optimization algorithms that is necessary for conventional LDPC codes. Second, using the proposed method, we can find codes that have nearShannon limit performance and do not have error floor. Third, in some applications that the code length cannot be large, the proposed codes can have better performance than the ordinary LDPC codes.
