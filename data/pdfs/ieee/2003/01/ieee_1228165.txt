I. INTRODUCTION
Using random permutation matrices, Gallager [l] has shown a procedure to construct LDPC codes without cycles larger than given length in their Tanner graphs. Restricting those permutations to cyclic shifts results in easy description and implementation of the codes and, moreover, we can build paritycheck matrices based on the number of shortest cycles with a little computational effort. Using this method, we can generate not only low rate codes with a small number of short cycles but also high rate codes without cycles of length 4. These codes perform better than some conventional codes based on
algebraic relations [a].
Since cyclic shifts are commutative, the above (3, IC)regular codes have weight 24 codewords for any code length [3]. We present the parity-check matrix structure that consists of Kronecker products of the above cyclic shifts and random permutations. Even if the random permutations are relatively small, we can improve the performance of long codes.
