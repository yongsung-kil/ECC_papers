Efficient Difference-based Decoding Implementations of LDPC Codes
Wu Zhanji, Peng Mugen, Wang Wenbo, and Li Luying

Abstract— Efficient implementations of the sum-product algorithm (SPA) for decoding low-density parity-check (LDPC) codes using difference-based messages between bit nodes and check nodes are presented. As for the updates of check nodes, reducedcomplexity derivatives are also put forward. As compared with the traditional Log-Likelihood-Ratio(LLR)-based decoding implementations, the proposed method has much lower complexity and latency, while it has no obvious loss of the error performance.
Index Terms— LDPC codes, sum-product decoding, differencebased implementation.
I. Introduction
I TERATIVE decoding of low-density parity-check (LDPC) codes using the sum-product algorithm (SPA) can approach the capacity of the additive white Gaussian noise (AWGN) channel. So, efficient implementation of the SPA has become a topic of increasing interest. The direct implementation of the original form of SPA has been shown to be sensitive to quantization effects [2]. Recently, some reduced complexity decoding algorithms that operate entirely in the log-likelihood-ratio (LLR) domain were presented. LLR-SPA based on Gallager’s approach has low parallel computational complexity, but it is very sensitive to quantization and round-off errors, which poses a big challenge for hardware implementations[5].Various reduced-complexity derivatives of the LLR-SPA based on the Jacobian approach are proposed, and both serial and parallel implementations are investigated[3]. For the check-node updates, core operations are implemented, but the computational complexity of direct core operations is still very high. So, for facilitating simple hardware design, the correction terms in these core operations can be executed via look-up tables or piece-wise linear functions, or even by using a single constant[3]. Besides, two main approaches for simplified check-node updates are presented, which are based on the so-called min-sum approximation coupled with either a normalization term or an additive offset term[4], [5]. However, these key terms need to be optimized through the density evolution method, otherwise it will cause the degradation of error performance. In this letter, novel difference-based implementations of SPA (D-SPA) and reduced-complexity derivatives (w-D-SPA) are proposed, which are different from the LLR-SPA implementations totally.

II. Difference-based Implementations of SPA (D-SPA)

We assume a binary [N, K] LDPC code is described by a

sparse M × N parity-check matrix H. The parity-check matrix

H can be viewed as a bipartite graph with two kinds of

nodes:N bit nodes corresponding to the encoded bits, and

M parity check nodes corresponding to the parity checks

represented by the rows of the matrix H. We denote the set of

bits that participate in check j as row[ j]={i:Hi j = 1}. Similarly, we define the set of checks in which bit i participates as

col[i] = { j : Hi j = 1}. We also denote row[ j] \ {i} as the set row[ j] with bit i excluded, and col[i] \ { j} as the set col[i]

with check j excluded.

In the following, we assume the binary phase shift keying

(BPSK) modulation, which maps a codeword X = {xi} into a transmitted sequence Y = {yi}, according to yi = 1 − 2xi, for i ∈ [1, N]. Then Y is transmitted over a channel corrupted

by the additive white Gaussian noise (AWGN).The received

value corresponding to yi after the demodulator is ri = yi + ni,
where ni is a Gaussian random variable with zero mean and variance σ2.
