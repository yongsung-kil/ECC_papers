LDPC Coded Wireless Networks with Adaptive Spectral Efficiency
Zhiyong He and Sébastien Roy Department of Electrical and Computer Engineering, Laval University, Quebec City, Quebec, Canada, G1K 7P4

Abstract— Low-density parity-check (LDPC) codes offer a very powerful error correction technique which allows data transmission in wireless networks at rates near the channel capacity with arbitrarily low probability of error. In this paper, we design a class of linearly encodable LDPC codes with adaptive code rates, i.e. the code rate can be adapted according to channel conditions to maximize the total capacity. Since a unique mother parity check matrix is used to construct LDPC codes with several code rates, the great advantage of the proposed codes is that a single universal encoder (decoder) is adequate to encode (decode) multi-rate codes, which makes it possible to efficiently implement multi-rate LDPC codes in a subscriber station. The implementation results into field programmable gate array (FPGA) devices indicate that a universal layered encoder for LDPC codes with 9 code rates is capable of reaching a throughput above 1.2 Gigabit per second by using 138 exclusiveOR gates and a master clock of 100 MHz. By combining multilevel modulation with the designed multi-rate LDPC codes, a transmission scheme with adaptive spectral efficiency is proposed. The simulation results indicate that the schemes with a spectral efficiency of 1, 2, 3, 4, and 5 bits/symbol/Hz can achieve extremely reliable transmission in a Rayleigh fading channel at signal-to-noise ratios (SNR) per bit of 5, 6, 10, 14, and 16 dB, respectively.
I. INTRODUCTION
Because of their capacity-achieving performance and the existence of effective decoding schemes, low-density paritycheck (LDPC) codes introduced by Gallager [1] have recently received a lot of interest for reliable high speed communication applications. Several standards regarding wireless communication have been recently proposed which suggested the incorporation of LDPC codes, such as broadband wireless access networks [2] and wireless local area networks [3]. By employing multi-level modulation and multi-rate LDPC codes, adaptive modulation and coding can maximize the total channel capacity and provide proper service and quality of service for each user according to channel conditions and transmit power.
An LDPC code with a code length of n and a code rate R is commonly represented by an m×n parity-check matrix, where m is the number of rows and n is the number of columns, or by a Tanner graph which consists of m check nodes and n bit nodes, where a row (column) within the parity-check matrix

corresponds to a check node (bit node) within the Tanner

graph. The code rate R of an LDPC code is determined by the

numbers of rows and columns, i.e.

R = (1 − m / n) .

(1)

Thus, there are two alternative approaches to obtain LDPC codes with multiple code rates. One is the puncturing or extending approach which changes the number n of columns, while the other is the row-removing approach which changes the number m of rows.
The puncturing or extending approach which changes the number of columns has been extensively used in the design of rate-compatible LDPC codes for automatic repeat request (ARQ) protocol. In an ARQ protocol, the incremental redundancy can be achieved by sending the higher-rate codeword first and sending additional parity bits in the lower-rate codeword later if requested from the receiver. The receiver combines all received coded bits to recover the information. The main advantage of the puncturing (extending) approach is its low encoding complexity, since a single encoder is adequate to handle different code rates. The main disadvantage is the degradation of its bit-error-rate (BER) performance, because the puncturing approach decreases the code length n. Since the punctured bits are not transmitted, these punctured bits have to be processed as bit ‘0’ in the decoder of a receiver, though some of these bits may be bit ‘1’. The puncturing approach also increases decoding complexity.
On the other hand, the second approach which changes the number of rows does not change the code length and thus guarantees good performance. However, the critical issue regarding the row-removing approach is its associated encoding complexity. Although turbo codes can be encoded in linear time, a straightforward encoder implementation for LDPC codes has a complexity which is quadratic with the code length. Since the row-removing approach may increase encoding complexity, fewer researches regarding the rowremoving approach are reported in the literature.
In this paper, we report our research results regarding the design of multi-rate LDPC codes using a row-removing approach. To reduce encoding complexity, we propose a class of linearly encodable LDPC codes with several code rates by designing a master parity-check matrix in a triangular plus dual-diagonal form. The code rate is adapted by removing the uppermost rows of the master matrix. Since a single parity check matrix is used to construct a family of codes providing

The decoding process of LDPC codes with adaptive rates is conceptually similar to the encoding process. For the mother code with a rate of 12/24, a joint row-column processor based on the belief-propagation algorithm ([7, 8]) decodes the code from the last layer to the first layer. daughter code with a rate of (12 + i) / 24 , where 1 ≤ i ≤ 8 .

IV. CONCLUSION To efficiently employ the bandwidth in wireless networks, we have proposed a transmission scheme with adaptive
Fig. 6. Bit-error-rate comparison of two schemes with the same spectral efficiency of 3 bits/symbols/Hz in a Rayleigh channel. Two LDPC codes have the same code lengths of 2304 and are decoded with a maximum of 50 iterations.

Thus, the proposed approach makes multi-rate LDPC codes feasible in a subscriber station, such as a cellular phone. The implementation results into FPGA devices indicate that a universal encoder for LDPC codes with 9 code rates attains throughputs up to 3.6 Gbit/s by using 138 exclusive-OR gates and an encoding frequency of 100 MHz.
By evaluating various combinations of the multi-level modulation with the multi-rate LDPC code, a transmission scheme with adaptive spectral efficiency from 1 to 5 bit/symbol/Hz is proposed. The simulation results indicate that the schemes with a spectral efficiency of 1, 2, 3, 4, and 5 bits/symbol/Hz can achieve extremely reliable transmission in a Rayleigh fading channel at signal-to-noise ratios (SNR) per bit of 5, 6, 10, 14, and 16 dB, respectively. Based on periodically fed back channel state information, such as the channel SNR as predicted by the receiver, the proposed adaptive scheme can maximize the total channel capacity and provide proper service and quality of service for each user.
