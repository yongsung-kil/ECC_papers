A Class of Low-Density Parity-Check Convolutional Codes Based on Difference Families
Yu-Cheng He, Christian Cardinal, and David Haccoun Department of Electrical Engineering E´ cole Polytechnique de Montre´al Montreal, Quebec, Canada
Email: {yucheng.he, christian.cardinal, david.haccoun}@polymtl.ca

Abstract— An algebraic construction for a class of low-density parity-check (LDPC) convolutional codes is presented on the basis of difference families associated with the code generator matrix. It can be shown that these codes have girth of at least 10 on the Tanner graph which is independent of either the size of the code generator matrix or the minimum Hamming distance of the codes. The code construction guarantees the independence of the messages exchanged in the belief propagation decoding process during two successive decoding iterations. Computer simulations show that over the additive white Gaussian noise channel, the best error performance of these codes at moderate signal-tonoise ratio values is practically obtained using only three to five iterations.
I. INTRODUCTION
Low-density parity-check (LDPC) codes with algebraic constructions have recently become attractive due to their low complexity encoding and efficient iterative decoding implementation. A number of algebraic constructions, including cyclic, quasi-cyclic, and finite geometry constructions, have been presented, such as [1]–[4]. LDPC convolutional codes can be viewed as a class of quasi-cyclic codes [4].
The Tanner graph model provides a useful tool to describe message passing in terms of belief propagation (BP) – also known as the sum-product algorithm or its approximation, the min-sum algorithm [5], [6]. It is known that the iterative BP decoding algorithm converges to the optimal maximum a posteriori (MAP) solution on a cycle-less graph [5]. However, for graphs with cycles, BP has no such optimality. In order to improve BP decoding, short cycles are avoided in the code design. For example, randomly constructed LDPC codes largely rely on the size and hence the sparsity of the paritycheck matrix to avoid short cycles on the Tanner graph.
The girth of a Tanner graph is the length of the shortest cycle in the graph. Therefore, girth is an important measure for the evaluation of LDPC codes in the graph. The larger the girth, the better the iterative BP decoding algorithm performs for the messages exchanged between the nodes of the graph during BP decoding can keep independent for more iterations. A further method of improving the code structure is to remove as many short cycles as possible in the graph so that the effects of the correlations between the inputs at each decoding iteration are reduced. The improvement in the code structure can also be made from the point of view of the distribution of cycles on the graph. An example is the balanced computation tree

which allows a proper weighing technique to improve the error performance of iterative BP decoding [1], [7].
In this paper, we present a code construction of LDPC convolutional codes based on difference families. Using difference structures in code design is not a new idea. However, in this paper, we extend the ordinary first order orthogonality structure based on difference sets to the second order orthogonality structures based on the difference families. Both encoding and decoding for the codes are completely determined by a generator matrix. Short cycles of lengths 4, 6, and 8 in the Tanner graph are systematically avoided by imposing specific conditions on the difference families associated with the generator matrix. The code construction is combinatorial, but the resulting code structure can be analyzed and explained using an incomplete computation tree. It can be shown that the code construction provides a girth of at least 10 on the Tanner graph which is independent of both the size of the generator matrix and the minimum Hamming distance of the codes. The iterative BP decoding algorithm is exact for two iterations.
The LDPC convolutional codes are decoded using an pipelined iterative BP algorithm which is similar to the algorithm proposed for the recursive time-varying periodic convolutional codes [8]. In order to improve the convergence speed of decoding, a feedback mechanism may also be incorporated [9], [10]. In this paper, the additive white Gaussian noise (AWGN) channel is considered. At moderate Eb/N0 values, the best error performance of these codes is practically obtained using only three to five iterations.
The remainder of the paper is organized as follows. Section II presents the code construction, followed by the iterative BP decoding algorithm in Section III. Simulation results for several rate 1/2 codes are provided in Section IV. Section V concludes the paper.

D. Code Structure
It is clearly seen from (8) that for the NSC codes, each parity checksum is composed of B information symbols and one parity symbol, and each information symbol participates in V parity checksums. Therefore, roughly stated, the semiinfinite parity-check matrix has uniform row weight (B + 1) and uniform column weight V . It can be shown that Condition 1) in Definition 1 reflects a fact that no two columns or no two rows in the entire parity-check matrix have one’s in common

more than once. This orthogonality property of the paritycheck matrix is a basic requirement for iterative BP decoding [1], [2], [6].
Therefore, we note the orthogonality identified by Condition 1) as the order-1 orthogonality. As discussed above, the order1 orthogonality property directly satisfies the requirement of the independence of inputs during one decoding iteration. Similarly, we note the orthogonality identified by Conditions 2) and 3) as the order-2 orthogonality. It can be shown that the order-2 orthogonality is able to guarantee the independence of inputs over two successive decoding iterations.
The structures of the NSC codes can be analyzed in the computation tree for iterative BP decoding which is constructed by iteratively applying the parity constraints of (8). It is easy to show that in the Tanner graph for the NSC codes, Condition 1) removes all the 4-cycles, Condition 2) removes all the 6cycles, and Condition 3) removes all the 8-cycles. It can also be shown that all the messages passing over two successive decoding iterations are independent. Consequently, the girth of the Tanner graph for the NSC codes is no smaller than 10. Therefore, the NSC codes defined above satisfy the properties required by LDPC codes, and hence they are viewed as a class of LDPC convolutional codes.
As a consequence, the NSC-LDPC codes are decodable by the iterative BP algorithm based on (8).

V. CONCLUSION
An algebraic construction for a class of nonrecursive systematic convolutional LDPC codes, noted NSC-LDPC codes, has been presented on the basis of defining two families of differences sets associated with the code generator matrix. In the Tanner graph, the code structure can be well analyzed. The girth of the Tanner graph for these codes is at least 10, and is independent of the minimum Hamming distance of the codes. Simulation results have shown that over the AWGN channel, the best error performance of these codes at moderate Eb/N0 values is practically obtained using only three to five iterations.
