OPTIMALLY QUANTIZED OFFSET MIN-SUM ALGORITHM FOR FLEXIBLE LDPC DECODER
Daesun Oh and Keshab K. Parhi
Department of Electrical and Computer Engineering University of Minnesota, Minneapolis, MN 55455 USA

ABSTRACT
In this paper, we analyze the performance of quantized offset min-sum (MS) decoding algorithm and propose an optimally quantized offset MS algorithm for a flexible low-density paritycheck (LDPC) decoder. It is known that the offset MS decoding algorithm is implemented with simplified hardware complexity and achieves good decoding performance. However, the finite precision effects in decoding LDPC codes result in performance different from floating point. The performance degradation is caused by different dynamic ranges of input data at high signal-to-noise ratio (SNR). The proposed offset MS algorithm uses the received data directly instead of loglikelihood ratio (LLR) data as the intrinsic information. It can achieve better performance than the conventional one since its offset factor is more effective at a wide range of SNR and the intrinsic information is quantized more robustly since it is independent of channel information. Also, it is possible for the proposed scheme to use a same quantization scheme for a flexible LDPC decoder, which can decode several kinds of LDPC codes. Simulation results show that our optimally quantized offset MS algorithms with 5-bits for (1728, 864) and (1728, 1296) irregular LDPC codes achieve better performance compared with the conventional offset MS algorithms with 6-bits quantization scheme.

Furthermore, since the LDPC decoder for the most modern wireless communication systems includes multiple code rates, various block lengths, and different sizes of submatrices, a flexible LDPC decoder needs to decode several kinds of LDPC codes [9][10].
Among a variety of decoding algorithms, the well-known belief propagation (BP) algorithm achieves a good decoding performance but requires a large hardware complexity. There are alternative methods such as several kinds of minsum (MS) algorithms which can significantly reduce the hardware complexity of BP at the cost of acceptable performance degradation, where complex computations at the check nodes are approximated by using simple comparison and summation operations. Recently, the modified MS algorithms using correction factors have been preferred for many practical applications since they offer comparable decoding performance compared to that of BP for regular LDPC codes [11][12]. Also, for irregular LDPC codes, the improved normalized or offset MS algorithms exhibit small performance degradations [13][14]. Specifically, the offset MS algorithm has been implemented for several practical applications due to its better performance and simple computations.

1. INTRODUCTION
Low-density parity-check (LDPC) codes developed by Gallager [1] have recently received a lot of attention because they can achieve a remarkable performance near Shannon limit over the binary symmetric channel (BSC) as well as the additive white Gaussian noise (AWGN) channel [2]. The decoding of an LDPC code allows a high degree of parallelism, which makes it very suitable for high data rate applications such as wide-band wireless multimedia communications and magnetic storage systems [3][4]. Well-designed irregular LDPC codes demonstrate better performance than regular ones [5]. Although the hardware architecture of irregular LDPC codes

2.2. Finite Precision Effects

Due to the high hardware complexity required by LDPC decoding, the finite precision effects for decoding algorithm should be analyzed well in order to decide desired tradeoffs between the available hardware resource and the required decoding performance. The quantization schemes need to be considered for both the intrinsic information and the updating messages. For the finite precision effects of several kinds of MS decoding algorithms for LDPC codes analyzed in [11], it is presented that 6 bits quantization scheme is generally a good tradeoff between hardware complexity and decoding performance for the intrinsic information and the updating messages. The performance loss between 6 bits quantization scheme and floating point in regular LDPC codes seems to be no more than 0.1 dB.

For the finite precision schemes in our works, we use the uniform quantization scheme expressed as (q : f ) in which a total of q bits are used, of which f bits are used for fraction part of the value. For (1728, 864) irregular LDPC code, the offset factor is fixed at βσ = 0.50 optimized at Eb/No = 2.0 dB. For (1728, 1296) irregular LDPC code, the offset factor is set to βσ = 0.60 optimized at Eb/No = 3.0 dB. As shown in Fig. 4, for the performances of (1728, 864) and (1728, 1296) irregular LDPC codes with (6:2) quantization schemes, the quantized offset MS algorithm suffers from large performance degradations and error floors at high SNR area. One feasible reason can be explained with the distributions of LLR intrinsic information.

4. SIMULATION RESULTS
First, we find the optimal offset values with rigorous simulations at specific value of SNR by heuristic methods. For (1728, 864) irregular LDPC code, the offset value, βσ = 0.5, optimized at Eb/No = 2.0 dB, results in the smallest bit error rate (BER) as searched by computer simulation. In case of (1728, 1296) irregular LDPC code, the best decoding performance is obtained with the offset value of βσ = 0.6 optimized at Eb/No = 3.0 dB. To apply one same quantization scheme such as (6:3) or (5:3) for one flexible LDPC decoder, the βn is set to 0.125 and θ is set to 1/(4 · σ22dB · 0.50) ≈ 0.8

or 1/(4 · σ32dB · 0.60) ≈ 1.25 for (1728, 864) or (1728, 1296) irregular LDPC code, respectively.
In Fig. 6, the performances for (1728, 864) irregular LDPC code with (6:3) and (5:3) quantization schemes are shown. The simulation results show that the offset MS algorithm with (6:3) quantization scheme has very similar performance compared to that with floating point although it suffers from a little performance loss at high SNR. When compared to the conventional offset MS algorithm with (6:2) quantization scheme, the performance gain more than 0.2dB is obtained at BER=10−6. The offset MS algorithm with even (5:3) quantization scheme achieves good performance and obtains the performance gain of about 0.1 dB compared to that with conventional (6:2) one.
In Fig. 7, the performances for (1728, 1296) irregular LDPC code with (6:3) and (5:3) quantization schemes are also shown. Similarly, the offset MS algorithm with (6:3) quantization scheme has very close or even better performance compared to that with floating point at wide ranges of SNR. When compared to the conventional (6:2) quantized offset MS algorithm, the performance gain is estimated to be more than 0.2dB under BER=10−6. Also, the proposed (5:3) quantized offset MS algorithm achieves almost same performance and its performance degradation is negligible compared to that of the proposed (6:3) one.
Moreover, even when applying the proposed offset MS algorithms with optimized (5:3) quantization scheme, the error floor at high SNR does not offer until BER=10−7 for both (1728, 864) and (1728, 1296) irregular LDPC codes. Also, note that the proposed scheme can be similarly applied to the normalized MS algorithm, which can lead to better decoding performance under finite precision effects.

5. CONCLUSION
In this paper, we have analyzed the performance of quantized offset MS algorithm and have proposed an optimally quantized offset MS algorithm for design of flexible LDPC decoders. It is known that the offset MS decoding algorithm is implemented with simplified hardware complexity and achieves good decoding performance. However, the finite precision effects in decoding LDPC codes result in different performance from floating point in that the performance degradation caused by different dynamic ranges of input data is not trivial at high SNR. The proposed offset MS algorithm uses the received data directly instead of LLR data as the intrinsic information. It can achieve better performance than the conventional one since its offset factor is more effective for a wide range of SNR and the intrinsic information is quantized more robustly since it is independent of channel information. Also, it is possible for the proposed scheme to use the same quantization scheme for a flexible LDPC decoder, which can decode several kinds of LDPC codes. It is seen from simulation results that the proposed offset MS algorithms under optimized (5:3) quantization scheme achieve better performance compared with the conventional offset MS algorithms with (6:2) quantization scheme for (1728, 864) and (1728, 1296) irregular LDPC codes.
