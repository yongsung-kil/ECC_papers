Abstract
To find the minimum distance computation ofLDPC codes is a NP problem, there has been no simple way to obtain the minimum distance ofLDPC codes because of the code length being very long. To provide a solution to deal with this hard problem, this paper develops an algorithm to estimate the weights and distance of LDPC codes based on generator matrices. The paper establishes the upper bounds of weights and distances of LDPC codes by using the vectors of generator matrices, which is different from the existed probabilitistic search methods for the weights' and distances of LCPC codes. The proposed algorithm can greatly reduce the searching time of weights and distances. Applying the algorithm we can obtain some LDPC codes of great distances and being free girth 4. Simulations verify the algorithms to be valid.

Simulations verify the proposed approaches to be valid.

2 Problem formulation

Good LDPC codes should have large girths and large distances [1-5]. Applied on a Tanner graph of a linear block code [2], the belief-propagation algorithm gives an exact a posteriori probability mass function for a given probability density function of the observed variables [2], but only if the factor graph is girth free. However, [3-6] shows that the LDPC codes without girth 4 free only do not ensure them to have large minimum distances. It is necessary to calculate the minimum weights and minimum distances of LDPC codes, but it is a NP problem [7,8].
Until now, there are three kinds of possible methods of estimating the minimum weights and minimum distances of LDPC codes: a. the algorithm based the parity check matrix ofLDPC codes [2]; b. the algorithm based the resulting codes obtained from generator matrices [4-6, 8-11]; c. the algorithm
