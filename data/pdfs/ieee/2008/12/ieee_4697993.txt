Clustering of Cycles and Construction of LDPC Codes
Xiaofu Wu, Chunming Zhao, Xiaohu You, and Ming Jiang

Abstract— The clustering of cycles to form stopping sets are first observed by T. Tian and et al. As the determination of stopping sets of minimum size is NP-hard, we propose to consider clustering of cycles to avoid the stopping sets of small sizes. In particular, the clustering of two cycles are considered for an improved version of progressive edge-growth construction of LDPC codes.
Index Terms— Stopping sets, message-passing decoding, lowdensity parity-check (LDPC) codes.
I. INTRODUCTION
In recent years, various methods for construction of LDPC codes have been proposed. In general, these methods can be decomposed into two main classes: random or random-like constructions, and algebraic constructions.
Algebraic constructions were initially proposed in [1] and further developed in [2]–[4]. Finite geometries, combinatorial design such as balanced incomplete block design, and finite fields have been extensively exploited to construct structured LDPC codes. These methods can often result into a special kind of parity-check matrices, which are composed of circulant permutation matrices.
Typical random or random-like methods can be found in [5], [6]. These methods only consider regular LDPC codes. Later, the introduction of irregular LDPC codes have shown its power for waterfall performance [7], [8], and now one can push the threshold very close to the Shannnon limit [9] using the tool of density evolution. For iterative message-passing decoding, its optimality can only be ensured when the bipartite graph is cycle-free. Hence, it is natural to try to minimize the influence of the cycles in the iterative decoding process. This idea first appeared in [10], and was extensively exploited by X. Hu, et al. to develop a so-called progressive edge-growth (PEG) algorithm for constructing LDPC codes of large girth [11]. However, for finite-length LDPC codes, there is basically no guarantee that the large girth can be beneficial to good performance for finite-length LDPC codes.

II. CLUSTERING OF CYCLES AND STOPPING SETS
Every linear block code can be defined by a parity-check matrix. The set of codewords for a given linear block code is unique, while the parity-check matrix is basically not unique. The Tanner graph of a linear block code is with one-toone correspondence with the parity-check matrix. Let G = (V C, E) be the Tanner graph of a code C with respect to the parity-check matrix H = [hi,j], where the set of variable nodes V represent the codeword bits (or columns of H) and the set of check nodes C represents the set of parity-check constraints (or rows of H) satisfied by the codeword bits. An edge e ∈ E is pair of (vi, cj), where vi is connected to cj if hi,j = 1.
For each v ∈ V and c ∈ C, let deg(v) and deg(c) denote the degrees of the variable v and the check node c, respectively.
Definition of stopping sets can be referred to [12].
Definition 1: A stopping set S in a code C is a subset of variables in a Tanner graph for C, for which all its neighbors connected to this set at least twice.
Definition 2 (Minimal stopping set): A minimal stopping set is a group of variable nodes which forms a stopping set, however, there is no subset of this group variable nodes which can form a stopping set.
Let HS denote the parity-check matrix induced by S, which is of columns extracted from H corresponding to all variable nodes in S. Let G(S) = (S CS , ES ) denote the graph induced by a stopping set S, which is with one-to-one correspondence of HS .
Theorem 1: Consider a stopping set S for a bipartite graph free of degree-1 variable nodes. For any given variable node in S, there exists at least one cycle in S which passes through this node.

Definition 3 (Cycle): A cycle of length 2d is a set of d variables and d check nodes connected by edges such that there exists a path which travels through every node in the set and connects each node to itself without traversing an edge twice.
Definition 4 (Extrinsic message degree): An extrinsic constraint node of a variable node set is a constraint node that is singly connected to this set. The extrinsic message degree (EMD) of a variable node set is the number of extrinsic constraint nodes of this variable node set.
Definition 5 (ex-Cycle, (d,e)-Cycle): An ex-cycle is a cycle with nonzero extrinsic message degree. An (d, e)-cycle is a cycle of length-2d with the extrinsic message degree of e.
For a Tanner graph of an LDPC code, almost all of cycles are ex-cycles and the only exception is the cycle whose variable nodes are all of degree-2. A cycle of length-2d that consists of d degree-2 variable nodes forms a minimal stopping set, which also corresponds to a codeword of weight-d. In general, a minimal stopping set consists of multiple ex-cycles. These ex-cycles are joined each other to form a set of variables with zero EMD, namely, a stopping set.
Fig. 1 demonstrates the clustering of two ex-cycles to form stopping sets. As shown, if two ex-cycles are of equal EMD, it is possible for them to join together to form a stopping set. Indeed, Fig. 1 is just the same as in [Fig.2, [13]]. Hence, this two ex-cycles join to form the low-weight codewords, which is detrimental to the error-floor performance. The idea of clustering of two ex-cycles can be employed to give a feasible explanation of the construction principle in [13].

For any given cycle-pair (σt, θ), one can compute the number of variable nodes d = τ (σt, θ) and the extrinsic message degree e = ε(σt, θ) of this cycle-pair without much difficulty. However, it is basically difficult to estimate the size of stopping sets containing a given cycle-pair. V. CONCLUSION AND FUTURE WORK
The clustering of cycles are found to be essential to the formation of stopping sets. Therefore, it is of importance to consider the clustering of more than two cycles to form the stopping sets. This idea have been successfully employed to propose a new progressive edge-growth construction method for LDPC codes. The proposed PEG construction considers only the possibility of two cycles for forming stopping sets. Simulations show its superiority in the error-floor region.
In future, there are some variations of the proposed constructions to be considered. Firstly, if the complexity is allowed, it will be beneficial to consider the clustering of more cycles, such as three, four, et al., to form the stopping sets. The constructed LDPC codes will be expected to have lower error-floor in theory. Secondly, the PEG method has one inevitable shortage as the use of the extrinsic message degree of a cycle during construction is not complete, which will be complete when all the edges connected to this cycle have been constructed. This shortage makes the use of EMD of cycles not exact during the construction. Therefore, it is more helpful to consider the construction method for use of exact EMD for all cycles.
