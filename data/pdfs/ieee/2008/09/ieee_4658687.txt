A New Binary Iterative LDPC Decoding Algorithm

Abstract—We introduce a new binary message-passing decoding method for low-density parity-check (LDPC) codes. The proposed decoding principle does not require the degree information of variable nodes and can be applied to both timeinvariant and time-variant decoding algorithms. We present the extrinsic error probability analysis and decoding threshold optimization for the new decoding method in both time-invariant and time-variant decoding scenarios. Both theoretical analysis and numerical results show that the proposed method provides the same performance as the existing methods for both timeinvariant and time-variant decoding, while the proposed new method facilitates efficient circuit implementations of the LDPC decoder since it does not require the degree information.
I. INTRODUCTION
Two binary message-passing LDPC decoding algorithms were introduced in [1], i.e., the Gallager decoding algorithm A (GA), and the Gallager decoding algorithm B (GB) as named in [2]. The GA algorithm is a time-invariant decoding algorithm, i.e., the parameters associated with the algorithm do not change with the decoding iterations. It is later generalized to a majority-vote-based threshold decoding algorithm in [3]. On the other hand, the GB algorithm is a time-variant decoding algorithm, where the majority-based decoding rule can still be applied in every iteration, but the threshold value changes with the iteration number during variable node decoding to minimize the extrinsic error rate. However, for irregular codes, the existing binary decoding algorithms, e.g., GB and majoritybased decoding algorithms, require the degree information of variable nodes. Such requirement can significantly increase the circuit level implementation complexity due to the operations of storing and retrieving the degree information.
To see how the degree information affects the complexity of the circuit implementation, Fig. 1 shows the basic decoding units, including a variable node unit (VNU) and a check node unit (CNU), in an LDPC decoder employing the min-sum algorithm [4]. The VNU outputs the sum of all extrinsic inputs; the CNU outputs the minimum and the second minimum, as well as the sign product, of the extrinsic inputs. It is seen that neither the VNU operation nor the CNU operation requires the degree information. The operations in both VNU and CNU are very simple and can be implemented with a small number of gates. If some parameters dependent on the node degree are involved in decoding, the degree information then has to be retrieved by VNU or CNU to obtain those parameters, which will require additional circuits be added to implement the operations of storing and retrieving the degree information as well as the associated control logics. Moreover, the VNU or

CNU can process the decoding of the variable or check nodes with different degrees. To efficiently implement the LDPC decoder, it is desirable to use the same VNU or CNU for decoding different variable or check nodes with some control units [5], [6], resulting in the minimum number of processing units in the chip. If the degree information is required during decoding, additional control units have to be implemented to retrieve the degree information and send it to the VNU or CNU. Although the example is for min-sum decoding, similar challenge appears when implementing the binary messagepassing decoder. Therefore, for the most efficient implementation of LDPC decoders, it is highly desirable to avoid having degree-dependent parameters in the decoding algorithm.
In this paper, we propose a new LDPC decoding algorithm for the binary symmetric channel (BSC) or any other symmetric channel with hard decision. The proposed decoding algorithm does not require the degree information of variable nodes, leading to a significantly simplified circuit implementation compared with existing binary LDPC decoders. Although the proposed decoding method is very simple, we show that with this new method, the optimized codes under both time-invariant and time-variant decoding do not incur any performance loss compared with the existing decoders that require the degree information of the variable nodes.

V. CONCLUSIONS
In this paper, we have proposed a new binary messagepassing LDPC decoding method that can be applied to both time-invariant and time-variant decoding. In this method, the same flipping threshold is used for decoding all variable nodes. Therefore, it does not require the degree information during decoding.
