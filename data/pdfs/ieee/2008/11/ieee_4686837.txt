LDPC Code Design for Half-Duplex Cooperative Relay
Chuxiang Li, Guosen Yue, Xiaodong Wang, and Mohammad Ali Khojastepour

AbstractÔÇöWe consider the design of LDPC codes for cooperative relay systems in the half-duplex mode. The capacity of halfduplex relay channels has been studied previously but the design of good channel codes for such channels remains a challenging problem. Employing an efficient relay protocol, we transform the half-duplex relay code design problem into a problem of ratecompatible LDPC code design where different code segments experience different SNRs. The density evolution with conventional Gaussian approximation for single user channels, which assumes invariant SNR within one codeword, is not capable of accurately predicting the code performance for this system. Here we develop a density evolution with a modified Gaussian approximation that takes into account the SNR variation in one received codeword as well as the rate-compatibility constraint. We then optimize the code ensemble using a modified differential evolution procedure. Extensive simulations are carried out to demonstrate that the proposed algorithm offers more accurate prediction of code performance in half-duplex relay channels than the conventional methods, and the optimized codes achieve a significant gain over existing codes.
Index TermsÔÇöHalf-duplex relay, LDPC code design, density evolution.
I. INTRODUCTION
R ECENT studies have shown that user cooperation diversity [31], [32] or cooperative diversity [18], [19] can be achieved via the cooperation among different nodes in a relay network. In such a cooperative relay network, multiple terminals can cooperatively form a virtual antenna array providing increased transmission rate in addition to spatial diversity [7], [17]. Although full-duplex relaying can potentially transmit at higher rate, practical wireless systems cannot transmit and receive signals simultaneously in the same frequency band. Thus, half-duplex relaying is of more practical interest, and has been widely studied in recent literature, e.g., [4], [5], [9], [11]ÔÇô[14], [16], [26], [27]. It has been addressed in existing work (e.g., [4], [5], [13], [14]) that similar as that in full-duplex relaying, the optimal coding in half-duplex relaying also has to be performed over multiple blocks, which inevitably involves high complexity and long delay. In this paper, we consider a simple coding strategy involving only one block for half-duplex relay channels from practical and commercial viewpoints.

Low-density parity-check (LDPC) code [10] is a good channel code candidate for half-duplex relay channels because of its capacity approaching performance exhibited in AWGN channel [28] and full-duplex relay channels [20], [21]. Under relaying circumstances, the signal-to-noise ratio (SNR) variation for different sub-blocks within one codeword makes the design of half-duplex relay code quite challenging. In [3], [22], [36], rate-compatible LDPC code design is treated under the assumption that SNR is invariant within one codeword, while in this work, we consider the more general and realistic scenario where SNR is variant within one codeword. The constituent code design treated in this paper essentially falls in the general framework of multi-edge LDPC codes [30]. Note that multi-edge LDPC codes [8], [30] do not explicitly consider rate-compatible constraints, which however have to be considered in relay scenarios and thus is one of the major concerns of this work. Therefore, this work can be viewed as an extension of multi-edge LDPC codes with the ratecompatible constraints in relay channels. It is worth noting the the coding strategy developed in this paper is applicable to incremental redundancy hybrid ARQ (IR-HARQ) systems.
In this paper, the code design problem in half-duplex relay channels is formulated as the ensemble design of ratecompatible LDPC codes. We first present a new ensemble description of the LDPC codes that experience different SNRs within one single codeword. Based on such an ensemble description, we develop a density evolution with a modified Gaussian approximation to accurately predict the ensemble performance of half-duplex relay codes. With the aid of this analysis tool and further considering the rate-compatible constraints in half-duplex relay channels, we can then search for the optimum code ensemble using a modified differential evolution method [25], [28]. Simulations are carried out to show that the proposed techniques offers accurate code performance prediction and powerful LDPC codes for half-duplex relay channels.

B. Relay Protocol for Half-duplex Relay Systems
Relay protocols, which allocates space-time resources to realize cooperative transmissions, are crucial for relay systems and the corresponding code design. The optimum code design for the half-duplex relay channel involves the design of two different sets of codebooks, one for S [xS(2tÔêÆ1) and xS(2t)] and the other for R [xR(2t)]. Similarly as that for fullduplex relay channels [20], [21], the optimum relay coding strategy should be performed over multiple blocks. Note that such a multi-block structure inevitably results in very high computational complexity and long delay in either code design or coding/decoding scheme. From the viewpoint of practical systems which motivates the study of half-duplex relaying,

region. It has been pointed out in [4]ÔÇô[6] that any relay coding strategy can be equivalently viewed as a composite of two fundamental extremes. In one extreme, R and S simultaneously send completely independent information; in the other extreme, R and S simultaneously send identical information (e.g., we in (5)), which corresponds to the relay coding strategy in this paper. Note that the gain brought by relaying is maximal at low SNRs, where the latter extreme performs better than the first extreme [4]ÔÇô[6]. Moreover, the identical codewords from S and R can make the receiver quite simple and reduce the complexity and costs in detection and decoding process. These motivate the study of the simple coding strategy in this work for half-duplex relaying.

V. CONCLUSIONS
In this paper, we have considered the LDPC code ensemble design for half-duplex cooperative relay channels. We have presented a new description of the LDPC code ensemble to characterize the parity-check matrices of half-duplex relay codewords that exhibit specific rate-compatible structure. Based on such an ensemble description, we have developed the density evolution based on the modified Gaussian approximation to accurately predict the ensemble performance of half-duplex relay codes.
