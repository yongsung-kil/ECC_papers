New Sum-Product Algorithm Using Approximation Method for Low Complexity LDPC Decoding

Jae Hee Han and Myung Hoon Sunwoo
School of Electrical and Computer Engineering, Ajou University San 5, Woncheon-Dong, Yeongtong-Gu Suwon, Korea e-mail : sunwoo@ajou.ac.kr

Abstract—This paper proposes a new sum-product algorithm to improve BER performance for low-density parity-check codes. The proposed algorithm can replace multiplications and divisions with additions and subtractions by adding the logarithmic and exponential functions. In addition, the proposed algorithm can simplify both the ln[tanh(x)] and tanh-1[exp(x)] functions by using two quantization tables which can reduce tremendous computational complexity. Moreover, the simulation results show that the proposed SP algorithm can improve maximum of 0.8 dB BER performance compared with the existing modified sumproduct algorithm.
Keywords- iterative decoding; sum-product algorithm; forword error correction; LDPC; piecewise linear function approximation
I. INTRODUCTION
Low-Density Parity-Check (LDPC) codes were proposed by Gallager in the early 1960s [1], however, they were forgotten until the 1990s. LDPC codes were rediscovered by MacKay and Neal in 1996 [2]. LDPC codes achieving close to the Shannon limit performance among the forward error correction (FEC) systems [2] have been adopted by the advanced standards, such as Digital Video Broadcasting – Satellite – Second Generation (DVB-S2), WLAN (IEEE 802.11n) [3] and WiMAX (IEEE 802.16e) [4].
Research works on LDPC decoding algorithms have focused on reducing computational complexity, decreasing the number of iterations and achieving better BER performance. The sum-product (SP) algorithm, also known as the belief propagation algorithm, presents the highest BER performance [5]. In contrast, the min-sum (MS) algorithm aims for reducing computational complexity [6].
In this paper, we propose a new SP decoding algorithm that can concurrently simplify both the ln[tanh(x)] and tanh1[exp(x)] functions. The proposed algorithm can minimize the performance degradation compared with the existing approximation algorithm [7]. Moreover, the proposed algorithm can replace multiplications and divisions with additions and subtractions, and thus, it can dramatically reduce computational complexity compared with the existing SP algorithms [5, 7].
This paper is organized as follows. Section II explains the SP algorithm and modified SP algorithm. Section III describes

the proposed SP algorithm. Section IV presents performance evaluations. Finally, Section V concludes the paper.

II. LDPC CODES AND EXISTING SUM-PRODUCT DECODING ALGORITHMS
This section briefly explains LDPC codes and presents the existing LDPC decoding algorithms such as the SP algorithm [5] and modified SP algorithm [7].

A. LDPC codes
The (N, K) LDPC code is a class of block codes where N and K represent the number of codeword bits and the number of information bits, respectively. The LDPC code is defined by an (N-K)-by-N parity check matrix H, which consists of (N-K) rows and N columns. The code rate is defined as K/N.
LDPC code can be represented by the Tanner graph, also known as a bipartite graph for the decoding process [9]. A single bit node can be connected to check nodes where the elements of a column are ‘1’ in the parity check matrix H. Similarly, a single check node can be connected to bit nodes where the elements of a row are ‘1’ in the parity check matrix H.

The SP algorithm with high BER performance is based on the hyperbolic tangent and inverse hyperbolic tangent functions to compute the check node update. Hence, the check node update requires the highest computational complexity among four steps of the LDPC decoding scheme.

To reduce the computational complexity, the modified SP algorithm has been proposed in [7].

As shown in Fig. 4, the sign bit of the bit nodes can be computed by XOR and only positive values are needed for the check node updates. Both the SP algorithm [5] and modified SP algorithm [7] require multiplications and divisions to update the check nodes. In contrast, the proposed algorithm requires additions and subtractions instead of multiplications and divisions for the check node updates. In general, in terms of hardware complexity and delay, both multiplication and division are much more complicated and longer than addition and subtraction. Moreover, the proposed algorithm uses only positive values of the ln[tanh(x)] and tanh-1[exp(x)] functions compared with the modified SP algorithm [8] which uses both positive and negative values for the tanh(x) and tanh-1(x) functions. Therefore, the proposed algorithm requires much less computational complexity compared with the existing SP algorithm [5, 7]. Moreover, the proposed algorithm can improve BER performance compared with the existing modified SP algorithm [7].
IV. PERFORMANCE EVALUATIONS We have performed the MATLAB simulation to verify the proposed SP algorithm. We apply the proposed algorithm and existing SP algorithms [5, 7] to the WLAN [3] and WiMAX [4] standards. The block size (N, K) = (1296, 864) in the WLAN [3] and (2304, 1152) in the WiMAX [4] have been chosen for performance evaluations. Assume that the encoded data bits are modulated by BPSK and transmitted through the AWGN channel. The number of iterations for LDPC decoding is up to 20. Fig. 3 and Fig. 4 show BER performance comparisons between the proposed SP algorithm and existing SP algorithms [5, 7] in the (1296, 864) LDPC code for the WLAN and (2304, 1152) LDPC code for the WiMAX.

V. CONCLUSIONS
In this paper, we proposed new SP decoding algorithm for LDPC codes. The proposed algorithm can replace both multiplications and divisions by additions and subtractions adding the logarithmic and exponential functions in the check node updates. Two quantization tables which consist of six regions and six quantization values have been selected to compute the ln[tanh(x)] and tanh-1[exp(x)], respectively. Therefore, the proposed SP algorithm can update the check nodes without computing the ln[tanh(x)] and tanh-1[exp(x)] functions. In addition, the proposed SP algorithm can significantly reduce the computational complexity compared with the existing SP algorithms [5, 7] and it achieves less BER performance degradation compared with the existing modified SP algorithm [7]. The simulation results show that the proposed SP algorithm improves maximum of 0.8 dB BER performance compared with the modified SP algorithm [7]. Moreover, the proposed algorithm can be used for various standards, such as the WLAN and WiMAX.
