Interior Point Decoding for Linear Vector Channels based on Convex Optimization
Tadashi Wadayama Nagoya Institute of Technology, Gokiso, Showa-ku, Nagoya, Aichi, Japan
Email: wadayama@nitech.ac.jp

Abstract‚ÄĒ In this paper, a novel decoding algorithm for lowdensity parity-check (LDPC) codes based on convex optimization is presented. The decoding algorithm, called interior point decoding, is designed for linear vector channels. The linear vector channels include many practically important channels such as inter symbol interference channels, partial response channels and MIMO channels. It is shown that the maximum likelihood decoding (MLD) rule for a linear vector channel can be relaxed to a convex optimization problem, which is called a relaxed MLD problem. Approximate variations of gradient descent and Newton methods are used to solve the convex optimization problem.
I. INTRODUCTION
Development of a decoding algorithm for binary linear codes has been one of the central research themes in coding theory. Recent research activities on message passing decoding algorithms are showing remarkable progress and bringing a shift in design principle of a decoder from algebraic algorithms to probabilistic ones. Sum-product algorithm for lowdensity parity-check (LDPC) codes is a distinct example. The combination of the LDPC codes and sum-product algorithm achieves high trade-off between decoding performance and decoding complexity. Especially, for memoryless channel, this combination offers almost satisfactory solution.
In this paper, a possibility of optimization approach to decoding problems for channels with memory, will be discussed. Most of conventional decoding algorithms designed for binary linear codes can be regarded as algorithms for solving a combinatorial optimization problem. Let us consider the following example to make the succeeding discussion concrete. A codeword x in a binary code C ‚äā F2n (F2 is the binary Galois field) is sent over an additive white Gaussian noise (AWGN) channel after binary (0,1) to bipolar (+1, -1) conversion r = (1 ‚ąí 2x) + z, where z represents a Gaussian noise vector and 1 denotes the vector with all-1 elements1. The maximum likelihood decoding (MLD) rule for this channel model is given by xňÜ = arg minx ‚ąąC ||r ‚ąí (1 ‚ąí 2x )||2, where || ¬∑ || is the Euclidean norm. The rule is, indeed, a combinatorial optimization problem because the feasible set C is a discrete set with a combinatorial and also algebraic structure. In general, the MLD problem is a computationally intractable problem when the code length is large. Thus, we need approximation algorithms which yield sub-optimal solution with reasonable amount of computation cost.
1In this paper, the elements in F2, i.e., {0, 1}, can also be regarded as elements in R. Thus, the appropriate arithmetic (e.g., mod-2 sum or addition of real numbers) depend on the context.

As an approach to a hard combinatorial optimization, a technique called relaxation has been well known in the field of combinatorial optimization. The basic idea of the relaxation is to relax the definition of the feasible set from a discrete set (such as set of integers) to a subset of n-dimensional Euclidean space Rn. Elimination of the constraint such that ‚Äúa feasible point is integral‚ÄĚ often leads to a linear programming (LP) problem which can be efficiently solved by a simplex algorithm or an interior point algorithm. Although the solution of the relaxed problem may be different from the solution of the original problem, the approximate solution can be used to find the optimal solution.
The work on LP decoding due to Feldman [1] may be the first application of the idea of relaxation in coding theory. One of the important implications obtained from the work on LP decoding is that the MLD problem can be relaxed to a linear optimization problem defined on Rn. Once a decoding problem is relaxed to an optimization problem, we can exploit an efficient numerical optimization technique to solve the relaxed problem.
So far, most of research activities on LP decoding focuses on the memoryless binary-input output-symmetric (BIOS) channels. This may be because the formulation of the LP decoding is strongly dependent on the memoryless property and the BIOS assumption. Therefore it is challenging and meaningful to consider a relaxed MLD problem for channels with memory. Development of an efficient decoding algorithm for such a channel is one of the important problems in coding theory. Recently, Taghavi and Siegel [4] showed a new relaxation method for PR channels which converts the MLD problem for PR channels to a linear programming problem. This work indicates that the relaxation approaches are effective not only for memoryless channels but also for channels with memory.
In this paper, a novel decoding algorithm for LDPC codes based on convex optimization is presented. The decoding algorithm, called interior point decoding, is designed for linear vector channels. It is shown that the MLD rule for a linear vector channel can be relaxed to a convex optimization problem, which is called a relaxed MLD problem. Approximate variations of gradient descent and Newton methods can be used to solve the convex optimization problem.

A. Linear vector channels and MLD rule
Suppose that H is a binary m √ó n matrix and C(H) is
the binary linear code defined based on H such as C(H) = {x ‚ąą F2n : Hx = 0}. The matrix H is a row-regular sparse matrix whose rows have row weight wr ‚Č• 3. In this paper, a bold-face letter, like x, denotes a column vector.
The linear vector channel, which is the main target channel of this paper, is defined as follows.
Definition 1 (Linear vector channel): A sender chooses a codeword x ‚ąą C(H) according to a message that the sender wishes to send. The codeword is transmitted to the channel and then a receiver obtains a received vector r ‚ąą Rn such that r = Ax + b + z, where A is a non-singular n √ó n real matrix2 (called an interference matrix) and b is a real column vector with length n (called an offset vector). It is assumed that both A and b are known at the receiver. The vector z denotes an additive noise vector. This channel model is called a linear vector channel. The offset vector b represents the offset resulting from the conversion from the code-representation (on F2) to the signalrepresentation (on R). Strictly speaking, the channel defined above should be called an affine vector channel because the transmitted signal s is generated by an affine transformation
s = Ax + b from a codeword x. However, we use more conventional name for this channel since the offset vector is not essential for the following discussion and it can be seen as a part of noises or mean of the noises.
If the vector z is an additive white Gaussian noise vector where the ith element zi(i ‚ąą [1, n]) is an i.i.d. Gaussian random variable with mean 0 and variance ŌÉ2, the channel is called a Gaussian linear vector channel. Note that the notation [a, b] denotes the set of consecutive integers from a to b.
The class of the linear vector channels is fairly wide and it includes many practically interesting channels such as AWGN channels, ISI channels, PR channels and MIMO channels. In order to achieve the best decoding performance with respect to the block error probability, we need to perform MLD for this channel. B. Relaxed MLD rule
We here introduce a relaxed MLD rule which is an approximation of the MLD rule (1). This relaxed rule is the basis of the interior point decoding to be presented in the latter sections. The basic idea of the relaxed rule is to relax the domain of x from C(H) to the fundamental polytope [1] [2] defined based on the parity check matrix H. The fundamental polytope is a polytope contained in Rn and it is a relaxed polytope of the convex hull of C(H). Thus, the set of vertices of the fundamental polytope contains all the codewords of C(H). This relaxation provides a possibility to use a minimization algorithm working on Rn (e.g., a gradient descent algorithm or Newton method) as a decoding algorithm. In the following discussion, we assume that the distance function d(x, y) is a convex function with respect to the variable x and it is a differentiable function as well.

The first term of Ōą(t)(x) is a scaled version of the objective function. The second term is the log-barrier function corresponds to the fundamental polytope. Since the sum of convex functions is also a convex function, Ōą(t)(x) is a convex function. The parameter t is called the scale parameter which controls the relative softness of the constraints represented by the log-barrier function.
The aim of the interior point decoding is to solve the relaxed MLD problem by using an interior point algorithm based on the barrier function method. The interior point decoding consists of the two nested loops: the outer loop and the inner loop. The inner loop corresponds to an approximate gradient descent method or an approximate Newton method which tries to minimize the merit function Ōą(t)(x). The main task of the outer loop is scaling of t. Namely, the scale parameter t is gradually increased from a small initial value to large values while the iteration of a decoding process proceeds. The details of the structure of the interior point decoding are omitted in this paper due to limitation of space6.
After the execution of the inner loop, built-in min-sum decoding7 is performed to obtain an estimate of the transmitted word. This means that the proposed decoding algorithm can be regarded as a hybrid type decoding algorithm based on the interior point algorithm and the conventional normalized minsum algorithm. The roles of the min-sum decoding are the following; (i) it accelerates the speed of convergence which leads to faster decoding, (ii) it enlarges the decision regions of a relaxed ML decoder (the related discussion can be found in the companion manuscript arXiv:0705.3990), (iii) it can save a stacked search point which occurs due to the approximated gradient descent or the approximated Newton process used in the proposed algorithm.

The PR channels are in a class of the ISI channels. In the field of magnetic recording, the channel model is often exploited as a basis of system design. Interferences coming from the neighboring symbols are linearly superimposed to the current symbol and the interferences make decoding process complex. The PR channels can be regarded as linear vector channels with a sparse interference matrix A. Thus, the PR channel is an ideal target of the interior point decoding. C. Bit error probability of the interior point decoding
In this subsection, some simulation results on the BER performance of the proposed algorithm are presented. As a reference decoding algorithm, we used the joint message passing decoding (abbreviated as joint MPD) in this paper. The joint MPD consists of two parts: the BCJR algorithm for the PR channel and the normalized min-sum algorithm (with dumping factor 0.7) for an LDPC code. The BCJR algorithm computes extrinsic information in the standard way and passes them to the min-sum algorithm. The min-sum algorithm uses the output from the BCJR algorithm as a priori information. The extrinsic information generated from the min-sum algorithm is treated as a priori information to the BCJR algorithm. The two parts are iteratively executed in turn. We assume the maximum number of iterations in the min-sum decoder is 10 and overall iterations are limited to 20. For each iteration, parity check procedure is executed.

V. CONCLUSION
In this paper, the interior point decoding for linear vector channels is presented. It may be too early to conclude that the optimization approach is superior to the message passing approach. There are some cases where the joint MPD overcomes the proposed algorithm (e.g, EPR4 case). Furthermore, there exist other configurations of a joint MPD which have not been tested in this paper. However, the simulation results presented in the previous section (and other results) are encouraging and they show the potential of the optimization approach. Compared with the conventional joint MPD, the proposed decoding algorithm achieves better BER performance with less decoding complexity in the case of PR channels in many cases.
