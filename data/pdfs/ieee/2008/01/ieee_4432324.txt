Hybrid Iterative Decoding for Low-Density Parity-Check Codes Based on Finite Geometries
Jian Li and Xian-Da Zhang, Senior Member, IEEE

Abstract— In this letter, a two-stage hybrid iterative decoding algorithm which combines two iterative decoding algorithms is proposed to reduce the computational complexity of finite geometry low-density parity-check (FG-LDPC) codes. We introduce a fast weighted bit-flipping (WBF) decoding algorithm for the first stage decoding. If the first stage decoding fails, the decoding is continued by the powerful belief propagation (BP) algorithm. The proposed hybrid decoding algorithm greatly reduces the computational complexity while maintains the same performance compared to that of using the BP algorithm only.
Index Terms— Low-density parity-check (LDPC) codes, weighted bit-flipping (WBF) algorithm, belief propagation (BP) algorithm, hybrid iterative decoding, finite geometries.
I. INTRODUCTION
L OW-DENSITY parity-check (LDPC) codes [1] can approach Shannon limit with iterative decoding using the so-called belief propagation (BP) algorithm or the sumproduct algorithm (SPA) [2]. Finite-geometry (FG) LDPC codes are algebraically constructed codes and can be encoded by shifting registers in linear time [7]. Besides the commonly used BP decoding algorithm, FG-LDPC codes can also be decoded by serval other simpler decoding algorithms. These decoding methods include: one-step majority-logic (MLG) decoding [7], Gallager’s bit flipping (BF) decoding [1], and weighted BF decoding (WBF) [4]. The main advantage of these simple algorithms are the fact that at each iteration, only a one-dimensional evaluation and memory are required. However, the BP algorithm consists of a two-dimensional evaluation (i.e., horizontal and vertical update) and memory at each iteration. However, these simple algorithms suffer from a prominent loss of coding gain.
Among those one-dimensional algorithms, the standard WBF [4] offers a good tradeoff between performance and complexity. Recently, a modified WBF [6] has been introduced to improve the error performance of the standard WBF. Both the standard and modified WBF algorithms can correct only a single bit error at each iteration. This letter will introduce a fast WBF algorithm that can correct multiple bit errors, which greatly improves the convergence speed of the WBF decoding algorithm and reduces the hardware complexity as well. Combining the fast WBF algorithm and the BP algorithm, we propose a hybrid decoding algorithm aiming to

II. A BRIEF REVIEW OF WBF ALGORITHM
A (N, K)(λ, ρ) FG-LDPC code of length N and dimension K is described by a sparse N × N parity-check matrix H = [Hmn]. The parity-check matrix H has exactly ρ ones in each row and λ ones in each column. Following a notation similar to [2], let us denote the set of bits that participate in check m by N (m) = {n : Hmn = 1}, and the set of checks in which bit n participates as M(n) = {m : Hmn = 1}. In the following, we assume binary phase shift keying (BPSK) modulation, which maps a codeword c = (c1, c2, · · · , cN ) into a transmitted sequences s = (s1, s2, · · · , sN ) according to sn = 2cn − 1 for n = 1, 2, · · · , N . We define the corresponding received sequence as y = (y1, y2, · · · , yN ).
Let z = (z1, z2, · · · , zN ) denote the binary hard decision sequences. For an additive white Gaussian noise (AWGN) channel, the magnitude |yn| can be considered as a simple measure of reliability of the hard-decision digit zn. We describe the standard WBF algorithm [4] as follows:

Step 4) Find the set S of bits for which El is a positive value and flip the bits in set S.
In step 4), both the standard and modified WBF algorithms need to find the maximum value from N values for each iteration, This operation introduces considerable hardware complexity for parallel processing and is also time-consuming, especially for large code length. However, the fast WBF algorithm flips the bits with positive reliability, an operation which is much easier and faster to implement. Both the standard WBF algorithm and the modified WBF algorithm flip one bit for each iteration, possibly correcting only one bit error. In contrast, the fast WBF algorithm flips multiple bits and thus can correct more than one bit error. Therefore,

the ratio of BP decoding participating in hybrid decoding almost equals the WER of fast WBF decoding. In the high SNR regime, the WER of fast WBF decoding is low, i.e., the BP algorithm activates infrequently, thus the computational complexity has been reduced. The fast WBF can be viewed as a simplified version of BP but has much lower error correcting capability than the BP algorithm. Therefore, for most noise patterns, if WBF can decode it correctly, BP can decode also. So one expects that the hybrid decoding has almost the same performance as that of the BP algorithm.
Because the hardware complexity of fast WBF decoder is much lower than that of the BP decoder, the proposed hybrid decoder has comparable hardware complexity to the BP decoder.

VI. CONCLUSION
In this letter, a simple and fast WBF decoding algorithm for FG-LDPC codes was introduced. The fast WBF converges much faster than the standard and modified WBF algorithm but maintains the performance of modified WBF. Then, we proposed a two-stage hybrid decoding which combines the proposed fast WBF and existing BP algorithm. The hybrid decoding can achieve the error performance of BP but has much lower computational complexity than BP, which makes it a promising candidate for decoding FG-LDPC codes.
