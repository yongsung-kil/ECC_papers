New Constructions of Quasi-Cyclic LDPC Codes Based on Special Classes of BIBD’s for the AWGN and Binary Erasure Channels
Lan Lan, Ying Yu Tai, Shu Lin, Behshad Memari, and Bahram Honary

Abstract— This paper presents new methods for constructing efficiently encodable quasi-cyclic LDPC codes based on special balanced incomplete block designs (BIBD’s). Codes constructed perform well over both the AWGN and binary erasure channels with iterative decoding.
Index Terms— Quasi-cyclic LDPC code, BIBD, array dispersion, erasure-burst, zero-covering span.
I. INTRODUCTION
R ECENTLY, BIBD’s (or combinatorial designs in general), [1],[2] were used to construct LDPC codes [3][5]. Several classes of LDPC codes were constructed based on special types of BIBD’s. Codes of these classes are referred to as BIBD-LDPC codes. The parity-check matrix of a BIBDLDPC code is simply the incidence matrix of a BIBD. Several subclasses of BIBD-LDPC codes are quasi-cyclic (QC). QCLDPC codes can be efficiently encoded using simple shiftregisters with linear complexity [6]. This encoding advantage is particularly important in practical applications.
This paper presents new methods for constructing QCLDPC codes based on special types of BIBD’s that are constructed from prime fields. The main concept used in the new constructions is to disperse the entries of a base matrix W over a prime field from which a BIBD is constructed into circulant permutation matrices over GF(2). The dispersion of W results in an array H of circulant permutation matrices, which is a low-density matrix. The null space of any subarray of H gives a QC-LDPC code whose Tanner graph [7] has a girth of at least 6. This new code construction is different from the code construction based on the incidence matrix of a BIBD as proposed in [3]-[5]. Experimental results show that codes of practical lengths constructed using the new methods

Also investigated in this paper are design and construction of LDPC codes for correcting bursts of erasures over a binary burst erasure channel. Two classes of QC-BIBD-LDPC codes for correcting erasure-bursts are constructed. Codes in these two classes also perform well over the AWGN and binary random erasure channels.
A binary regular LDPC code [8] is defined as the null space of a sparse parity-check matrix over GF(2) with the following properties: (1) each row has constant weight ρ; (2) each column has constant weight γ; and (3) no two rows (or two columns) have more than one place where they both have 1-components. Such a parity-check matrix is said to be (γ, ρ)regular and code given by its null space is called a (γ, ρ)regular LDPC code. Property (3) is referred to as the rowcolumn (RC)-constraint. The RC-constraint ensures that: (1) the Tanner graph of the code has a girth of at least 6; and (2) the minimum distance of the code is at least γ + 1 [9], [10]. An LDPC code is said to be irregular if its parity-check matrix has varying column and/or varying row weights. A QCLDPC code is given by the null space of an array of sparse circulants, [9]-[12].
The performance of an LDPC code over the AWGN channel with iterative decoding depends on a number of code structural properties besides its minimum distance. One such structural property is the girth of the code that is defined as the length of the shortest cycles in code’s Tanner graph. For an LDPC code to perform well with iterative decoding, its Tanner graph must not contain short cycles. The shortest cycles that affect the code performance the most are the cycles of length 4. Therefore, cycles of length 4 must be avoided in code construction. This is the case in almost every method of constructing LDPC codes that has been proposed. The RC-constraint on the paritycheck matrix of an LDPC code, automatically rules out cycles of length 4 in the code’s Tanner graph and hence ensures that the girth of the Tanner graph is at least 6. Algebraic methods for constructing LDPC codes with girth greater than 6 have been proposed in [13]-[15].
