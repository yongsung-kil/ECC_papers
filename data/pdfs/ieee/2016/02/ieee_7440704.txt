Abstract‚ÄîFor a given base graph, the protograph can be obtained by a copy-and-permute procedure. If the permutation is cyclic, the protograph corresponds to a quasi-cyclic (QC) protograph LDPC code. The girth of the QC protograph LDPC code is determined by the girth of the base graph and the permutation shifts. Progressive edge-growth (PEG) construction builds up a Tanner graph, or equivalently a parity-check matrix, for an LDPC code by maximizing the local girth at symbol nodes in a greedy algorithm. In this paper, we introduce a modified version of the PEG algorithm which can construct large girth base graph and determine the optimal permutation shifts, simultaneously, for QC protograph LDPC codes. Simulation results show that the QC protograph LDPC codes constructed by the proposed modified PEG algorithm have good frame error rate (FER) performance over the AWGN channel.
I. INTRODUCTION
Low-density parity-check (LDPC) codes first proposed in the early 1960‚Äôs [1] and re-discovered in 1996 [2], are the most promising class of linear codes due to their capacityapproaching performance and low decoding complexity. An LDPC code is specified by a sparse parity-check matrix and its corresponding Tanner graph [3]. An important parameter affecting the performance and determining the efficiency of iterative decoding algorithms for LDPC codes is the girth, which determines the number of independent iterations [1]. Therefore, the construction of LDPC codes of large girth has attract much attention [4][5][6].
Based on the methods of construction, LDPC codes can be classified into two categories: 1) random (or random-like) codes [7] and 2) structure codes [3][8]. Among the existing methods for the construction of random-like LDPC codes, one of the most successful approaches is progressive edge-growth (PEG) algorithm [7]. PEG algorithm establishes each edge of the Tanner graph for an LDPC code by first expanding a subgraph from a symbol node sj to a maximum depth l, and then searches the check node ci which is at the largest distance from sj. If a cycle Œæ is created by establishing an edge between sj and ci, the length of Œæ is of the largest possible length 2(l+2). Obviously, the PEG algorithm has two features:
‚Ä¢ For a given symbol node sj, the PEG algorithm determine the last edge (ci, sj) to form a cycle Œæ.
‚Ä¢ The length of the cycle Œæ is maximized.

Although, LDPC codes constructed by the PEG algorithm are among the best codes with large girth, their major disadvantage is represented by their high implementation complexity that makes them impractical for very long code lengths. On the other hand, structured LDPC codes, especially quasi-cyclic (QC) LDPC codes are well suitable for certain practical applications since they can easily be encoded using simple shift-registers with linear complexity [4]. QC LDPC codes belong to the protograph-based LDPC codes because they can be regarded as being lifted from protographs, also referred to as base graphs, using cyclic permutations [5].
Generally, to construct large girth QC protograph LDPC codes, after constructing the base graph G, the determining of permutation shifts would bring in the cycle searching in base graph [5][9][10]. For example, when the PEG algorithm is used in the construction of the QC LDPC codes, the base matrix is usually generated with the PEG algorithm first, and then extra cycle searchings are needed to determining the permutation shifts [11][12][13]. When code length of basic matrix is short, the algorithm is feasible, but with the code length growth, the cycle searchings will bring high computational complexity.
In this paper, motivated by the cycle searching and girth maximizing features of the PEG algorithm, we introduce a modified version of the PEG algorithm which can construct the large girth base graph and determining the permutation shifts of QC protograph LDPC codes, simultaneously. The advantages of the modified PEG algorithm are as follows.
‚Ä¢ The QC protograph LDPC codes constructed by the PSDPEG algorithm are of large girth.
‚Ä¢ Extra cycle searching is not needed to determine the permutation shifts. Therefore, the computational complexity to construct of QC protograph LDPC codes is reduced.
It should be mentioned that the method in [14] also constructs the Tanner graph of QC-LDPC codes with the PEG algorithm. However, we constructs the base graph and determine the permutation shifts, simultaneously, with the modified PEG algorithm. It is easy to see that the expanding tree is smaller in our modified PEG construction, and therefore the computational complexity is lower.

V. CONCLUSION
In this paper, we introduce a modified version of the PEG algorithm which can construct the base graphs and the permutation shifts of QC protograph LDPC codes, simultaneously. We replace the entries of a base graph of QC protograph LDPC codes with the corresponding permutation shifts, the resulting matrix is referred to as a permutation matrix. Since the PEG algorithm builds up the permutation matrices in an edge-by-edge manner, the girth of the permutation matrices are maximized. By adding an accumulation operation into the original PEG algorithm, the optimal permutation shifts of the permutation matrices are selected. Simulation results show that the proposed QC protograph LDPC codes have good BER performance and FER performance.
