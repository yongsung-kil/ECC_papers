 
 
  Abstract—Multi-level per cell (MLC) /Triple-Level per cell 
(TLC) technique significantly improves storage density with low 
cost increases, which stores more than one bit per cell, but also 
drastically decreases reliability in NAND flash memory. As the 
bit number per cell increases, correspondingly cell-to-cell 
interference (CCI) becomes the major challenge for NAND flash 
memory technology scaling. Recently, low-density parity-check 
(LDPC) code is considered as the most appropriate scheme of 
next generation Error Correcting Code (ECC) in NAND flash 
memory. In this paper, by exploiting the intra-cell characteristics 
in TLC NAND flash memory channels, a low-complexity 
quantization and reliability-Aware iterative majority-logic 
decoding (QR-IMLGD) algorithm for LDPC code is proposed to 
reduce the memory read latency and effectively improve the 
throughput of LDPC decoding. The proposed algorithm takes 
advantage of highly unequal error probability of input 
log-likelihood-ratios (LLRs) in the same cells, employ small 
bit-level quantizer and update only less reliable variable nodes 
that are initiated with higher error probability LLR values. 
Simulation results show that the proposed QR-IMLGD algorithm 
can yield a noticeable improvement in decoding convergence rate 
without compromising the error performance in TLC NAND 
flash memory channels.

Flash-based solid-state drives (SSDs) is an emerging 
trend in modern data storage systems. Flash-based storage is 
superior to legacy magnetic recording in reliability, power 
consumption, latency, read/write throughput, and wide range 
of operating temperatures. Multi-level per cell (MLC) 
/Triple-Level Cell (TLC) technique [1]-[2], which is stored 2 
or 3 logical bit in physical single memory cell, have recently 
been extensively researched to pursue higher storage capacity.  
The low bit cost and high reliability are the most important 
requirement for NAND flash memory systems. Unfortunately, 
the reliability and performance of a NAND flash memory is 
largely destroyed due to scaling down the feature size of 
NAND flash cells [3] and related noise sources, e.g., 
 
 
cell-to-cell 
interference 
(CCI), 
arising 
from 
parasitic 
coupling-capacitance between adjacent cells which has been 
well recognized as one of major noise sources, random 
telegraph noise (RTN), read/program disturb, retention noise 
(RN).  
    In NAND flash memory, the voltage window in which the 
cell’s threshold voltage (Vth) lies determines the logical value 
a memory cell stores [4]. As scaling down of NAND flash 
memory cell and more bits per cell stored, more errors occur 
in determining a cell's value due to the smaller threshold 
voltage window used to represent each value. Therefore, it is 
difficult to decide what logical value a cell's Vth corresponds 
to. Moreover, the introduction of TLC flash substantially 
reduces cost, but drastically decreases reliability, which makes 
the device window narrow. Due to the aggressive technology 
scaling and multi-level per cell technique, some traditional 
hard-decision 
error-correction 
codes 
(ECCs), 
such 
as 
Bose-Chaudhuri-Hocquenghem (BCH) and Reed-Solomon 
(RS) codes, have become increasingly inadequate for SSDs as 
the capacity of MLC/TLC NAND flash memory continues to 
increase and its reliability continues to degrade [5]. Therefore, 
to handle low-cost NAND with high Bit Error Rates (BERs), 
low-density parity-check (LDPC) code [6-7] that has more 
powerful error correction capability is believed to be 
indispensable for MLC/TLC NAND flash-based SSD.  
The large complexity of sum-product algorithm (SPA) [8] is 
a barrier for the implementation. [3] proposed a new 
mathematical model calculate the LLRs for LDPC codes in the 
presence of CCI, where a mass of integral and logarithm 
operations should be made in initialization procedure to 
measure the priori probabilities before executing the SPA. 
Some simplified decoding algorithms, e.g., majority-logic 
decoding (MLGD) and Min-sum algorithms, have been 
presented in [9]-[11] to reduce the complexity of decoding. 
However, it is highly desirable to develop low-cost LDPC 
decoders for flash channel such that its faster read-response 
time is not compromised. In our previous research work [4], 
the modified soft reliability-based iterative majority-logic 
decoding (SRBI-MLGD) algorithm is proposed for the LDPC 
codes over MLC NAND flash memory channels, but no

Motivated by data dependency and intra-cell unbalanced bit 
error probability in the flash memory cell, we propose a 
low-complexity quantization and reliability-Aware iterative

of variable nodes.  Moreover, only the logical and add

the soft information which is determined by quantized value of 
the detected voltage with small bit-lever quantizer for lower 
page.

IV presents the simulation results. Finally, the conclusion is 
drawn in Section V.

IV. SIMULATIONS 
Simulations are conducted to evaluate the properties and 
BER performance of the proposed QR-IMLGD decoding 
algorithm in TLC NAND flash memory system. A rate-0.784 
length 2331 binary quasi-cyclic (QC) LDPC code is employed 
in the simulation. The programmed voltage 𝑉!  levels are 
assumed to 2.7, 3.4, 4.1, 4.8, 5.5, 6.2 and 6.9, respectively. 
The program step voltage ∆𝑉!! is set to 0.2V. We set 𝛾! and 
𝛾!" to 0.08𝑠! and 0.0064𝑠!, respectively.  
Fig. 4 shows the bit error ratio (BER) performance 
comparison between the proposed QR-IMLGD algorithm and 
the modified SRBI-MLGD algorithm in [4] over a TLC 
NAND Flash memory channel. The maximum number of 
iterations is 30. In our QR-IMLGD decoding algorithm, 
two-bit level quantizer is employed using most significant bit 
(MSB) and central significant bit (CSB). Four-bit level 
quantizer is only used for least significant bit (LSB). The 
corresponding mapping between charge values and bit values 
is also given in Fig. 1 [2]. Therefore, average 2.7-bit is employ 
to quantize the threshold voltage, where the number of 
quantized bit is four in [4].  
Monte Carlo simulations demonstrated that the proposed 
QR-IMLGD decoding algorithm achieves better error 
performance and faster decoding convergence rate, and a litter 
better gain can be achieved when cell-to-cell coupling factor 
𝑠! >1.16 due to the intra-cell characteristics in TLC NAND 
flash memory channels when compared with the SRBI-MLGD 
algorithm employed using 4-bit quantizer. 
Our proposed QR-IMLGD algorithm holds the significantly 
reduced complexity but less BER performance loss compared 
with BP decoding algorithm in TLC NAND flash memory 
system, and agrees with our previous research work [4].
