Abstract—LDPC code is a powerful error correcting code used in many communication systems. Efficient decoding algorithm and hardware implementation of LDPC code is an important issue. The paper proposes an improved version of the Normalized Min-Sum (NMS) decoding algorithm named “Girth-Aware NMS”. The principle of the GA-NMS is to determine “offline” the optimal normalisation factor of the extrinsic message using the structure property of the Tanner Graph associated to the LDPC matrix. The idea can be expressed simply: a check node implied in one, or more, cycle of minimum size (the so-called Girth) in the Tanner Graph is more subjected to self-confirmation than other check nodes. Thus, the extrinsic messages generated by this type of check nodes should be more aggressively “normalized” (or attenuated) than other check nodes. In the context of the IEEE P802.16e standard (WIFI standard), simulation results of the GA-NMS shows both improved decoding performance and reduced average number of decoding iterations.
Index Terms—LDPC codes, Girth, Normalized Min Sum, WIMAX code.
1. Introduction
Low Density Parity Check code [1] is one of the most powerful binary codes. LDPC code can be represented by a sparse parity matrix or graphically by a Tanner graph [2], and decoded iteratively using message passing algorithm.
The LDPC codes can be efficiently decoded with the original belief-propagation (BP) algorithm or the (SP) sumproduct algorithm. The BP algorithm is recognized by its good error correcting performance [3] but need many multiplications to update check and variable nodes which make it difficult to use for efficient hardware implementation.
The Min Sum (MS) algorithm [4], [5] is a sub-optimale solution which tries to overcome the computational complexity issue of the SP decoder, at price of performance. This performance penalty is generally considered to be caused by the approximation used within the check node processing

step, which results in an overestimation of check-to-variable messages. In order to mitigate this performance penalty, several correction methods were proposed in the literature [5], [6], [7], [8], [9], [10].
The Normalized Min-Sum (NMS) and Offset Min-Sum (OMS) decoders [6], [7] are slightly modified versions of the MS decoder that rely on the use of either a normalization or an offset factor to compensate the overestimation of checkto-variable messages.
Recently, several approaches are developed in order to design LDPC codes without short cycles [11], or reweight Tanner graph in order to fight against girth problems [12], [13], [14]. Because During iterative decoding process, the existence of short cycles in the code affects independence of the extrinsic messages exchanged [15], [16], which deteriorates overall performance [17].
In this paper, we propose new versions of NMS algorithm which tries to reduce the short cycles effect based on the knowledge of the cyclic structure, the new algorithm is called Girth-Aware-NMS algorithm. The new method consists in attributing to each check node a factor that depends on the girth passed through it.
The reason behind the MS and NMS algorithms choice in this study is that, the MS decoder is widely implemented in real communication systems, and the NMS algorithm is MS reweighted by an constant optimization factor, we suppose that this optimisation factor can take into consideration the cyclic structure of the code.
The evaluation of the proposed algorithm GA-NMS is done using WIMAX code. The comparison results show that the new algorithm is better than NMS and MS standard in error rate performances and convergence behaviors.
The organization of this paper is as follows: Section II introduces the background in terms of understanding how to calculate girth in node and NMS decoder. In Section III, the proposed GA-NMS algorithm is presented in details. Section IV shows the simulation results along with discussions. Finally, Section V concludes the paper.

The structure of the LDPC code determines the decoding performance and the convergence speed of the code. Code construction is a hot topic research. In this paper, we will only consider existing LDPC code and try to adapt the normalisation coefficient γ of (eq. 3) to the local topological structure of the associated check node. More precisely, for each check node m, we will determine the size G(m) of the shortest cycle in the Tanner Graph passing through node m. Note that the girth of a graph is simply G = minm=1···M G(m). The determination of G(m) is straightforward [18], [19]. For self consistence of the paper, a simple algorithm that compute G(m) is presented in Algo. 1.
To determine G(m) of check node m, check m broadcast a “ping” (i.e. a one value) in the Tanner Graph. Each node (variable or check) that receives the “ping” on one of its branch transmits it to its other branches. The “ping” will thus be spread more an more at each iteration in the Tanner Graph. When the “ping” goes back to the initial check node m, a cycle has been found and the algorithm stops. G(m) is thus twice the number of iteration obtained at the end of the algorithm. 3. Girth Aware Normalized Min Sum algorithm
The proposed Girth Aware Normalized MS is to replace the constant normalisation factor γ associated to check node m by a normalization factor γ(G(m)) that is a function of G(m) in order to take into account the actual length of the shortest cycle passing through check node m in the Tanner graph. The determination of the values of γ(G(m)) of the GA-NMS algorithm is an off-line process. The offline γ(G(m)) determination procedure is described in the following section.
3.1. Cyclic structure
The first step of the GA-NMS is the knowledge of the cyclic structure of the code, the study of the IEEE P802.16e [20] code for rate 1/2 shows that this code has a particular structure: there is 144 check nodes with a girth of 4, 120 check nodes with a girth of 6, and finally, 24 check nodes with a girth of 8 as shown in the table below:

The optimization factor γ1, that will be assigned to the check nodes that are connected to girth 4 is calculated in lauing down γ2 and γ3 on the constant value 0.85 already calculated for the NMS and varying γ1 from 0.1 to 1. 4. Simulation and Results
In this study, we focus on the LDPC code of rate 1/2 and size 576 defined in the wimax IEEE P802.16e standard [20]. The new algorithm GA-NMS is examined in terms of convergence speed and in terms of decoding performance. We consider an Additive White Gaussian Noise (AWGN) channel and the maximum number of iterations is limited to 50 iterations.

The bit error rat and frame error rat of the proposed algorithm GA-NMS are compared to NMS and standard MS algorithms, the result shows that the GA-NMS outperforms the NMS and standard MS particularly at error floor region

5. Conclusion
In this paper, the GA-NMS algorithm that modifies the method to find the optimization factor of NMS algorithm was applied to LDPC codes in IEEE P802.16e standard. The new method is to classify the check nodes according to short cycles that crosses and assign an empirically optimization factor for each check nodes class. The results show that the GA-NMS algorithm is better than NMS and MS algorithms in convergence speed and bit error rat performance with a small complexity. The proposed method is suitable for IEEE P802.16e with size (288,576) and can be used for other codes that have the same cyclic structure as the code used in this paper.
