Abstract—This paper proposes a new approach to construct quasi-cyclic low-density parity-check (QC-LDPC) codes of girth 8. The proposed codes are based on a bipartite graph called Moore graph. By decomposing the bit nodes of the bipartite graph, the proposed QC-LDPC codes have variable code lengths and rates. Simulation results show that the proposed QC-LDPC codes have good performance with an iterative decoding over an AWGN channel.
Index Terms—Moore graph, QC-LDPC codes, column decomposing.

Low-density parity-check (LDPC) codes were invented by Gallager in the early 1960’s [1] and Mackay rediscovered them in 1996 [2]. Researchers have paid a lot of attention on LDPC codes because of their capacity-approaching performance and low decoding complexity. Quasi-cyclic (QC) LDPC codes are the most promising class of structured LDPC codes, and the encoding and decoding can be easily done with linear complexity [3]. It is well-known that the iterative sum-product decoding algorithm will converge to the optimal solution when the associated Tanner graph of the code is free of cycles. The algorithm will break down sooner if the graph have shorter cycles. Therefore, parity-check matrices or Tanner graphs with large girth, which is the length of the shortest cycles, are especially preferable, and the girth is among the important parameters which determines the coding performance. Therefore, the construction of LDPC codes of large girth has attracted much attention [4][5][6].
We introduce a class of QC-LDPC codes based on a bipartite graph called Moore (q + 1, 8)-graph with girth larger than or equals to 8 in this paper. In accordance with the structure of the parity-check matrix, which is decided by a Moore (q + 1, 8)-graph, the construction approach of QCLDPC codes in this paper includes three procedures. Firstly, a base matrix B is constructed according to a Moore (q + 1, 8)graph. Secondly, construct the mother matrix HM by using a circulant permutation matrix to replace each ‘1’ entry in B and using a zero matrix to replace each ‘0’ entry in B. V. EXAMPLES AND SIMULATIONS
In the following, two examples are provided to explain the construction process and performance of the proposed QC-LDPC codes. While computing the error performance, we presume BPSK transmission over an AWGN channel in the case of the BER. This paper uses log-likelihood sumproduct algorithm as the decoding algorithm and the maximum iteration number is set to be 30.
Example 3: Let q = 5 and we can obtain an N × N regular matrix B based on the graph Γ5[V0, V1] where N = 1+5+52+ 53 = 156. The columns of B correspond to the vertexes in V0 and the rows correspond to the vertexes in V1. Replace each ‘0’ of B with a ‘∞’ and each ‘1’ of B with an integer in the range [0, 29], then a 156×156 shifting matrix S can be obtained. Let the size L of the permutation matrix be 30 and we can obtain a 4680 × 4680 mother matrix HM by replacing each ‘∞’ of S with a 30 × 30 zero matrix and replacing each integer entry of S with a 30 × 30 permutation matrix. By decomposing 117 block columns of weight 6 into 234 block columns of weight 3 and 39 block columns of weight 6 into 117 block columns of weight 2, we finally obtain a 4680 × 10530 effective matrix HE. The null space of HE contributes to a QC-LDPC code C1 with code rate 0.556 and code length 10530. Figure 3 indicates the BER performance of C1. We compare C1 with a regular geometric LDPC code given in [13] (Fig.6) with code length 10512 and the same code rate as C1. The simulation results

show that the performance of C1 is much better than the one in [13].
Example 4: Consider the 156 × 156 regular matrix B based on the graph Γ5[V0, V1] given in Example 3. Replace each ‘0’ of B with a ‘∞’ and replace each ‘1’ of B with an integer in the range [0, 22], then a 156 × 156 shifting matrix S can be obtained. Let the size L of the permutation matrix be 23 and we can obtain a 3588 × 3588 mother matrix HM by replacing each ‘∞’ of S with a 23 × 23 zero matrix and replacing each integer entry of S with a 23 × 23 permutation matrix. By decomposing each block column of weight 6 into two block columns of column weight 3, we finally obtain a 3588 × 7176 regular effective matrix HE of row weight 6. The null space of HE contributes to a QC-LDPC code C2 with code rate 0.5 and code length 7176. Figure 4 indicates the BER performance of C2. We compare C2 with three different regular QC-LDPC codes provided in [14] (Fig.3 and Fig.4) with the same code length and code rate as C2. They are all of girth 8 and have the same row and column weight. We can see from the simulation results that C2 performances better than the codes in [14] at high SNR region.
VI. CONCLUSION
This paper introduces a new construction approach of QCLDPC codes of girth 8. Firstly, the base matrix B is constructed according to a Moore (q + 1, 8)-graph. Secondly, we obtain the mother matrix HM by replacing each ‘1’ entry in B with a circulant permutation matrix and each ‘0’ entry with a zero matrix. Thirdly, we use column decomposing method to obtain the effective matrices HE with variable rates. Simulation results show that the proposed QC-LDPC codes have good performance with an iterative decoding over an AWGN channel.
