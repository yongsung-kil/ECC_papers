An Improved Scheme for Large Scale Complex Field Network Coding Based on User Clustering and Relay Selection Strategy in Wireless Cooperative Communication

Yuwen Huang, Zhanji Wu, Chengxin Jiang, Kun Lu
School of Information and Communication Engineering Beijing University of Posts and Telecommunications Beijing, China e-mail: wuzhanji@bupt.edu.cn

Abstract—Complex field network coding can earn larger diversity gain, better system reliability and higher network throughput in wireless communication with relay cooperative. An improved complex filed network coding scheme for large scale density network with user clustering and relay selection strategy will be proposed in this paper. The multiuser detector we employ is not based on maximum likelihood algorithm, but based on complex filed belief propagation algorithm improved from belief propagation decode algorithm of low density parity check code. As the same time, network decoding and channel decoding are combined together to form a joint network channel iterative decoder, in which the extrinsic information derive from network decode and that derive from channel decode can work well for each other. Numerical results are given to illustrate that our proposed scheme can provide better reliability, larger network diversity gain and higher throughput, more importantly, it can be well applied to large scale dense network when compared with traditional schemes.
Keywords-complex field network coding; clustering strategy; relay selection; joint channel network decode; cooperative communication
I. INTRODUCTION
In wireless communication, channel code is considered as an effective method to reduce or eliminate the performance degradation result from time-varying or frequency-varying fading channel, whose redundant information in check bits can be used to recover propagate errors of original information sequence [1], [2]. Spatial diversity gains and network coverage can be achieved when employing relay cooperative technique, without packing multiple antennas in each user or base station [3], [4]. However, with the increasing size of network, traditional relay approach become less and less bandwidth efficient. Thanks to the development of new network coding technique in [5], [6], we can well improve the throughput and bandwidth efficiency of the communication system especially in wireless cooperative communication [7].
Stem from physical network coding (PNC) and develop from finite field network coding [8], complex field network coding (CFNC) technique is much more easily to get higher throughput which makes it a promising and practical new technique and research hotspots. Inspired by the operation of

base complex field network coding at user nodes and relay nodes in CFNC scheme as in [9], the multiuser detection technique for decoding message from the mixing signal in [10], the traditional iterative decoder in [11] and the network decode module and the channel decode module in iterative multiuser decoder [12], we put forward and highlighted an enhanced CFNC scheme in large scale dense network.
The major advance of this paper is the employment of user clustering strategy and relay selection strategy as well as jointing network decoding and channel decoding together in multiuser access cooperative relay network. Basing on complex filed belief propagation (BP) algorithm, we also improve the traditional iterative decoder for network and channel decoding. The extrinsic information between the two part of iterative multiuser decoder (the network decode module and the channel decode module) was able to utilize each other in an iterative way. By combining the channel coding and the base CFNC scheme together, and with new improved complex filed belief propagation network decoding algorithm, and user clustering and relay selection strategy, our simulation results turn out that the proposed CFNC scheme can greatly improve the comprehensive performance of communication system, especially in large scale dense wireless network system.
The organization of this paper is as follows. In Section II, the description of complex field network coding system model in large scale dense network will be display. The new strategy adopted in our scheme, including the improved complex filed belief propagation algorithm, user clustering and relay selection strategy, along with joint networkchannel iterative decode is pursued in Section III. To validate the performance of large scale complex field network coding based on our scheme, we will present some simulation results in Section IV. Finally some conclusions and further research subject inspired by this paper are summarized in Section VI.

B. User Clustering and Relay Selection Strategy
Initially clustering algorithm is one of the most effective methods for network topology control in wireless sensor networks, which can significantly reduce energy consumption and improve the network throughput. In our subject, we introduce user clustering algorithm to handle the user's access problem in large scale dense network using complex field network coding technology. The user clustering algorithm, by dividing all users into several clusters, can greatly simplifies the process and complexity of complex field network coding. Thanks to user clustering algorithm, it makes it possible for the application of complex field network coding technology extends from small scale network to large scale network.
Relay selection strategy means that we will select the best (or several) relay according to the channel state information of the link of relay-destination. Also this strategy can greatly reduce the complexity of message process at relay nodes and destination. IV. PERFORMANCE ANALYSIS
In this section, we provide some simulation results to validate the excellent performance of our CFNC scheme in various scenarios for practical wireless channel environment. Simulation condition and some parameters are as follow. We consider the Turbo code now applied in LTE-A with generator matrix of (13, 15), code rate 1/2, code length 256bit. The modulation and channel type is binary phase shift keying and fully-interleaved Rayleigh fading channel with unit variance (fast fading), respectively. And each user and relay uses the same channel code and modulation style. Also we assume that the magnitude of fading variance (SNR) of the three link source-relay, relay-destination, sourcedestination, are the same. As described above, improved BP network decoding, joint network channel iterative decode, user clustering and relay selection strategy are all considered in our new approach.
With reference to Fig. 5, we compare the bit error rate of large scale complex filed network coding and traditional small scale complex filed network coding. In large scale complex filed network coding there are 120 users and 12 relays, and all the users are clustered into several clusters with 2, 4, 6 or 8 users in each cluster respectively (from LSCFNC UCS1 to LS-CFNC UCS 4). While in small scale complex filed network coding there are 4 relays and 2, 4, 6 or 8 users in each test respectively (from SS-CFNC user=2 to SS-CFNC user=8). The reliability of large scale CFNC can reach that of small scale CFNC and even much better, thanks to the user clustering strategy and relay selection strategy. In general, LS-CFNC get about 0.5~1.0dB gain over SS-CFNC

when the BER is 10-5. It’s because the relay selection strategy selects the best relay, therefore the reliability is higher than that of the average of good, medium and even bad relay provide. The use of user clustering strategy makes it possible that the application scenarios of CFNC can extend from small scale network to large scale dense communication network.

V. CONCLUSIONU
An improved scheme was proposed in this paper for large scale complex field network coding based on user clustering and relay selection strategy in large scale dense wireless cooperative networks. For the proposed scheme, a novel joint channel network iterative decoder based on improved complex filed belief propagation algorithm was presented. The joint use of BP network decoding, network channel decoding, and user clustering and relay selection strategy, all make it possible that the application scenarios of complex filed network coding technology can extend from small communication network to large scale dense network. Simulation illustrates that our new scheme LS-CFNC has about 0.5~1.0 dB gain over SS-CFNC, providing an excellent performance improvement, effectively combating the adverse effects of wireless communication environment. And it meets the development needs of large scale dense network in the future very well.
