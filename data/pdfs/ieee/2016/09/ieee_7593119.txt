Abstract-There exist lots of (3, L )-regular quasi-cyclic (QC) 
LDPC codes constructed from finite fields, protographs, array 
codes, and computer search under some design rules. For a 
given code length and rate, how to select the best one from these 
codes is considerable. In this paper, we study (3, L )-regular QC 
LDPC codes from the perspective of graph isomorphism, and 
non-isomorphic (3, L )-regular QC LDPC codes are determined. 
By analyzing the cycle structures of the resulting non-isomorphic 
codes, an efficient algorithm for counting cycles is presented. 
Also proposed is a simplified exhaustive search of non-isomorphic 
(3, L )-regular QC LDPC codes free of cycles of length less than 
go, where go is the estimated optimal girth value for a given code 
length. Based on these two algorithms, we can easily construct a 
(3, L )-regular QC LDPC code with optimized cycle distribution 
for a given L and code length. Numerical results show that the 
constructed codes have better performance under the iterative 
decoding algorithms. 
I. INTRODUCTION 
Quasi-cyclic (QC) low-density parity-check (LDPC) codes 
have better performance with the iterative algorithms [1], [2]. 
For hardware implementation, they can be not only efficiently 
encoded using the simple shift registers [3], but also decoded 
in partially parallel benefiting from their QC-structure [4]. 
Therefore, much research effort was made in construction, 
analysis, encoding, and decoding of QC LDPC codes [5]-[9]. 
This paper focuses on construction of QC LDPC codes, 
especially (3, L)-regular QC LDPC codes, where 3 and L 
are the column weight and row weight of their parity-check 
matrices, respectively. There are many QC LDPC codes con­
structed based on algebraic tools (such as finite fields, finite 
geometries) [10], protographs [11], array codes [12], and 
computer search [13], [14]. For a given degree distribution 
and code length, how to pick out an optimal QC LDPC code 
from these existing codes (or how to construct an optimal QC 
LDPC code) should be considered. 
To the best of our knowledge, A. Tasdighi et al. [15] first 
employed the isomorphism of LDPC codes to find the shortest 
(3, L)-regular QC LDPC codes constructed based on cyclic 
liftings of fully connected base graphs for a given degree 
distribution and girth. However, they focus on (3, L )-regular 
QC LDPC codes whose block lengths satisfy the lower bound 
on the lifting degree of cyclic liftings with a given girth. In 
this paper, we discuss (3, L )-regular QC LDPC codes from the 
perspective of their Tanner graph [17] isomorphism, and non­
isomorphic (3, L)-regular QC LDPC codes of length pL are 
determined, where P is an arbitrary positive integer. Notice 
that isomorphic QC LDPC codes have same performance 
decoded with the iterative algorithms. By analyzing the cycle 
structures in the Tanner graphs of QC LDPC codes, we present 
an algorithm for counting cycles based on their exponent 
matrices. Furthermore, we propose a simplified exhaustive 
search of non-isomorphic QC LDPC codes, and all possible 
non-isomorphic QC LDPC codes free of cycles of length less 
than 90 are derived, where 90 is the estimated optimal girth 
value. Thus, by counting the cycles of these non-isomorphic 
QC LDPC codes, we can obtain a QC LDPC code with 
optimized cycle distribution for a given degree distribution and 
code length. Numerical results show that the constructed codes 
with optimized cycle distribution have better performance with 
the iterative decoding algorithms. 
The remainder of this paper is organized as follows. Section 
II provides basic notations and definitions. Section III investi­
gates the cycle structures of (3, L )-regular QC LDPC codes. In 
Section IV, an algorithm for counting cycles of (3, L )-regular 
QC LDPC codes is presented, and a simplified exhaustive 
search of non-isomorphic (3, L)-regular QC LDPC codes is 
proposed in Section V. In Section VI, construction of (3, L)­
regular QC LDPC codes with optimized cycle distribution is 
proposed. Finally, Section VII concludes this paper.

The Tanner graph of H is a bipartite graph composed of 3p

pL variable nodes (representing code symbols). There exists 
an edge connecting the check node i and the variable node 
j if and only if the entry at row-i and column-j of H is I. 
The length of the shortest cycle is referred to as the girth 
of the Tanner graph (or LDPC code). Generally, short cycles, 
especially the cycles of length 4, can result in a degradation 
of the decoding performance when decoded with the iterative 
algorithms based on message passing. A constraint, well­
known as row-column constraint (or RC constraint) [10], 
imposed on H can ensure that the girth of an LDPC code 
defined by the null space of such H is at least 6.

VII. CONCLUSION 
In this paper, we presented an algorithm for counting cycles 
in the Tanner graphs of (3, L)-regular QC LDPC codes. 
Also proposed was a simplified exhaustive search of non­
isomorphic (3, L)-regular QC LDPC codes of length pL, 
where p is the size of CPMs in their parity-check matrices. 
On the basis of these two algorithms, we can easily construct 
a (3, L)-regular QC LDPC code with optimized cycle distri­
bution for a given degree distribution and code length. Note 
that the proposed construction method is also applicable to 
the construction of (.1, L )-regular QC LDPC codes for .1 W 4. 
Simulation results show that the constructed (3, L )-regular 
QC LDPC codes offer better performance over the AWGN 
channel.
