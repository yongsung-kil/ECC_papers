Optimization of LDPC Codes for Bit-Patterned Media Recording with Media Noise
Lingjun Kong†∗, Shengmei Zhao∗, Ming Jiang† and Chunming Zhao† †National Mobile Communications Research Laboratory, Southeast University, Nanjing, China ∗College of Communication & Information Engineering, Nanjing University of Posts & Telecommunications, Nanjing

Abstract—Challenges in bit-pattered magnetic recording (BPMR) include two-dimensional (2D) inter-symbol interference (ISI) constituted by both one-dimensional ISI and inter-track interference (ITI), and media noise arising from island position jitter (PJ) and island size fluctuations (SF) in both the downtrack and cross-track directions. A LDPC code optimization approach is proposed for BPMR channel with media noise, where a reduced-complexity 2D detector based on the iterative rowcolumn soft detection feedback with Gaussian approximation (IRCSDF-GA) detector is employed. Also, the threshold of the proposed LDPC codes is calculated for different percentage of media noise in BPMR. Simulations results show that about 0.3 dB performance gain can be achieved when BPMR is only suffered from the PJ.
Index Terms—bit-pattered magnetic recording (BPMR), twodimensional (2D) inter-symbol interference (ISI), LDPC code.
I. INTRODUCTION
Due to continued strong demand for higher storage areal densities, bit-patterned magnetic recording (BPMR) and twodimensional magnetic recording (TDMR) have recently been extensively researched [1]-[2]. In order to increase the storage density, reducing the separation between recording tracks will result in more cross-talk between the tracks, called intertrack interference (ITI). The read-back signal of both the BPMR and TDMR can be modeled as a signal corrupted by two-dimensional (2D) inter-symbol interference (ISI) which is constituted by both one-dimensional (1D) ISI and ITI [3]-[5].
In BPMR, the magnetic domains that store the bits are isolated and ordered on the media, thus the well-defined magnetic islands are thermally more stable as the magnetic domains are larger, and the SNR is maintained. One more challenge for BPMR during the readback process is the media noise, which is mainly caused by fabrication imperfections. Main sources of media noise include island position jitter (PJ) and island size fluctuations (SF), in both the along-track and cross-track directions [3].
In our previous work [6], a first-ever 2D ISI optimized LDPC code was designed based on Extrinsic Information Transfer (EXIT) curve fitting of modified EXIT charts to combat the 2D ISI and its simulation results shown that LDPC codes with block length 104 bits optimized for 2D ISI channel corresponding to 4T b/in2 recording density achieve bit error rate 10−5 at signal-to-noise ratio (SNR) 0.33 dB away from the symmetric information rate (SIR). However, the capacityapproaching code designs in [6] did not include media noise

considerations. Further, the coding gains of LDPC codes have not been fully investigated for BPMR with media noise.
In 2D ISI channels, the computational complexity of the BCJR 2D detector (2D-DET) is prohibitive, hence iterative row-column soft decision feedback algorithm (IRCSDFA) was proposed in [7] as a reduced-complexity 2D-DET. To further cut down the complexity, Gaussian approximation (GA) is utilized to replace the strip detection in IRCSDFA by a single row or column detection, the resultant simplified detector is referred as “IRCSDFA-GA” [8]-[9].
In this paper, an analytical 2D island pulse response for BPMR channel with media noise[3] is used for LDPC code design, where a reduced-complexity IRCSDFA-GA detector is employed instead of the full BCJR 2D-DET.
The rest of this paper is organized as follows. Section II introduces the channel model for BPMR with media noise. Then in Section III, LDPC codes design for BPMR channel with media noise is proposed based on the modified EXIT chart. Simulation results are presented in Section IV. Finally, Section V renders some conclusions.

III. CODE OPTIMIZATION FOR BPMR WITH MEDIA NOISE
We define an ensemble of LDPC codes with the variable and check node degree sequences λ = (λ2, λ3, · · · , λDL ) and ρ = (ρ2, ρ3, · · · , ρDR ), where λi and ρj denote the fraction of edges connected to variable nodes of degree i and check nodes of degree j, respectively. And let DL and DR be the number of different variable node degrees and check node degrees, respectively. The optimized LDPC codes are designed by selecting the best λ sequences to fit the CND EXIT curve as closely as possible to the EXIT curve of the combined 2DDET and VND, while ensuring that they do not cross over.

IV. SIMULATIONS
Simulations are conducted to evaluate the properties and BER performance of the proposed LDPC codes which are optimized for BPMR with media noise. The simulated system is shown in Fig.1. The maximum number of iterations between the 2D-DET and the LDPC belief-propagation (BP) decoder is set to 10, and the maximum number of BP iteration in LDPC decoder to 30. All codes have rate 8/9.
In Fig.2, we compare the threshold of the LDPC codes detected by RCSDFA-GA detector for different percentage of media noise in a 2D ISI channel with recording density of 4Tb/in2. It can be seen from the Fig.2 that PJ make the LDPC code having a higher threshold, and the threshold gap increases when the percentage of media noise increases. The threshold gap represents the performance loss incurred by the media noise.
Fig.3 shows that the code with 13K code length that we designed for 4T b/in2 BPMR with SF achieves about 0.2dB over the code optimized for AWGN channel at the bit error rate (BER) of 10−5. About 0.3 dB performance gain can be achieved when BPMR is only suffered from the PJ. All schemes use joint RCSDFA-GA detector [8]. and BP LDPC decoder. This is the first LDPC code successfully optimized for BPMR with media noise.

V. CONCLUSION
In this paper, we reviewed the major impairments in BPMR channel, namely 2D ISI and media noises (which include island position jitter and size fluctuations ) from a coding and signal processing perspective. To suppress 2D ISI with media noise, new LDPC codes have been optimized based on modified EXIT charts, which incorporate low complexity IRCSDFA-GA 2D-DET, to recover the resultant performance loss by coding re-design. Monte Carlo simulations demonstrated that the new LDPC codes used in conjunction with the low-complexity IRCSDFA-GA detector can achieve 0.3 dB performance gain over the code optimized for AWGN channel when BPMR is only suffered from the position jitter. Along with the optimization of LDPC codes, advanced signal processing schemes need to be employed to further improve the potential performance of system and remove the error floor caused by the position jitter.
