AbstractŌĆöIn this paper, a non-uniform quantization scheme is proposed for the decoding of Low-Density Parity-Check (LDPC) codes with the Sum-Product Algorithm (SPA). With the proposed scheme, the quantization-range and quantization-interval for the soft Log-Likelihood-Ratio (LLR) values of the decoder input, the variable nodes, and the check nodes are set separately, and are also adjusted according to the evolution of the probability density distribution of the LLR values in the iterative decoding procedure to minimize the difference between the decoding values before quantization and that after quantization. Simulation results show that decoding with 3-bit non-uniform quantization of the proposed scheme may obtain a BER performance slightly better than that of decoding with 6-bits quantization of the traditional uniform quantization method.
Keywords- non-uniform quantization; Low-Density ParityCheck (LDPC); Sum-Product Algorithm (SPA)
I. INTRODUCTION
In wireless communication systems, transmission signals are exposed to random noise, fading, and interference, which may greatly deteriorate the quality of the received signal and thus results in demodulation errors in receiving the signal. Channel coding techniques are effective schemes to correct channel errors and eliminate interference, which have been widely applied to promote the transmission reliability of wireless communications systems.
Low-Density Parity-Check (LDPC) code is a kind of linear block code that uses a random sparse matrix to be its parity check matrix. This kind of code was firstly proposed by Robert G. Gallager in 1962 [1], and rediscovered by David Mackay in the 1990s [2]. Due to its near-Shannon-Limit error-correcting capability, low error-floor, and easy design with parallel decoding, LDPC codes are widespread used in wireless communication systems [3].
However, with the fast development of wideband wireless communication technologies, the LDPC decoders need to be designed with higher speed, better error-correcting performance, and less hardware complexity. But generally speaking, the higher the speed and the better the performance are, the more complex the decoder is. Hence, how to make a

new balance between the decoding speed, the error-correcting performance, and the hardware implementation complexity remains an intractable problem.
Among the decoding methods of LDPC codes, sum-product algorithm (SPA) [4] based on the belief propagation (BP) is most widely used for its excellent decoding performance. When speaking to hardware implementation of LDPC decoders, such as Field Programmable Gate Array (FPGA) or Application Specific Integrated Circuit (ASIC), all numerical values in every step are treated quantitatively. Uniform quantization method is applied to this field widely for its simple realization and low complexity in application. In order to reach similar performance for non-quantization decoding, a larger maximum limit, smaller minimum limit and narrower quantization interval are required, which means that more quantization bits are necessary for representing all the uniform quantized values. As for applying LDPC decoders on hardware, more resources are occupied, which severely restricts its use.
So far, non-uniform quantization methods for the decoding of LDPC codes have received more attention. An adaptive nonuniform quantization with min-sum approximation decoding is introduced [5]. Another non-uniform quantization function is given for decoder messages, which is effective for an M-QAM system [6]. In [7], the quantization step is replaced by a finite alphabet on binary symmetric channel (BSC). In the sense of maximizing mutual information, lookup tables are derived for low complexity in hardware [8] and an optimal quantization of many outputs on discrete memoryless channel (DMC) is proposed [9].
In this paper, a non-uniform quantization method for the decoding of LDPC codes with the SPA is introduced. With the proposed scheme, the soft information of each received LDPC code, the soft information of variable nodes and check nodes during an iterative decoding process are treated in different ways, and different quantized values are chosen for the three kinds of variables to make them closer to their real magnitudes. The spaces between every two adjacent quantized values are not the same. In this way, the interval insensitive to decoding process disappears and the important space which contributes more to better performance remains. We use each bit

II. THE SUM-PRODUCT ALGORITHM OF LDPC CODES
The LDPC code belongs to a type of linear block code. In its sparse check-check matrix H , all non-zero elements in one row make a single parity-check constraint, which is defined as a parity check node. All non-zero elements in one column constitute a repetition-code constraint, which is set as a variable node. When given a check matrix of an (N, K)
LDPC code, all (N, K) parity check nodes and N variable
nodes are known. Every non-zero point in H satisfies both the parity check constrain in row and variable-node constrain in column. So the outputs of the variable nodes, which are also called their extrinsic information, are transferred to check nodes and are regarded as the inputs of check nodes. The outputs of check nodes are transmitted in the opposite direction to variable nodes.

When considering the hardware implementation of LDPC decoding process, the question of numerical quantization can not be ignored. With the decoding process continuing, the computing results of every variable node and check node in each iteration should be expressed in a finite width of bits, in which the uniform quantization method is most commonly used. III. THE PROPOSED NON-UNIFORM QUANTIZATION SCHEME
Tracking the extrinsic information of variable nodes and check nodes with SPA, they trend differently. In Fig.1, as the decoding process develops, the extrinsic information of variable nodes trend to the maximum limit gradually, while the extrinsic information of check nodes trends to small value. As to the soft information of each received LDPC code, it does not change during the iterative process, while with the uniform quantization, the parameters are selected large enough for the three variables, which results in a big cost in practice.
Note that with non-uniform quantization, the quantized values are unequally distributed in the field of real numbers. Therefore, the quantization intervals are not always the same. The redundant quantized values, which are not sensitive for real numbers, are deleted so that all the remaining quantized values have great importance for the decoding.
In our proposed non-uniform quantization method, the three kinds of variables are treated individually as their real values belong to diverse regions. This method is based on the uniform quantization parameters with more quantization bits. Then the number of quantized values of three variables is reduced in proper order until getting the least number needed. Once the quantized values at the same level of the three

variables are determined, a step of combined adjustment within the three variables is carried on for quantization error correction.

VI. CONCLUSIONS
In this paper, we have proposed a non-uniform quantization method for the decoding of LDPC codes with SPA. We treat the LLR of LDPC codes, the soft information of the variable nodes, and the soft information of the check nodes separately to make the quantized values close to their own real numbers. The mutual information between the source code and the quantized value is introduced and then the mean squared error of the quantized values is used to measure the quantization method. On the AWGN channel with SDD and HDD, we propose a 3-bits quantization scheme respectively. Simulation results show that the BER performance with 3-bits non-uniform scheme is similar to that with 6-bits uniform scheme and the 3-bits scheme performs a little better than the
6-bits scheme at high Eb N0 regions.
