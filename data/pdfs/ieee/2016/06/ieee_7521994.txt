On the performance of Fixed-length Spatially Coupled LDPC Code
Shuang Chen, Kewu Peng, Huangpin Jin, Jian Song Tsinghua National Laboratory for Information Science and Technology, Department of Electronic Engineering, Tsinghua University, Beijing 100084, China
Email: pengkewu@tsinghua.edu.cn

AbstractŌĆöThis paper evaluates the rate loss of fixedlength spatially-coupled low-density parity-check (SCLDPC) code compared to the asymptotic performance of such code. In previous works, SC-LDPC with sufficient length of codeword shows excellent asymptotic performance and attracts a lot of attentions. However, when the length of codeword is limited, the performance of SCLDPC code with limited codeword-length is not as well studied as its counterpart with infinite codeword-length. In this paper, the rate loss of a fixed-length SC-LDPC will be analyzed from two aspects: the rate loss due to finite length of the uncoupled code, and the rate loss due to finite length of the coupling chain. Furthermore, through trade-off between the length of the coupling chain and that of the uncoupled code, the rate loss of the SC-LDPC is optimized. The optimization results show that the trend of the rate loss of a length-N SC-LDPC is O(NŌłÆ1/3) and is different from that of a conventional LDPC code [1], which is O(NŌłÆ1/2). In other words, for achieving similar performance of a conventional length-N LDPC code, the length of an SC-LDPC code should be increased to O(N3/2).
KeywordsŌĆöchannel coding, low-density paritycheck (LDPC) codes, spatially coupled LDPC codes, finite-length code performance
I. INTRODUCTION
Low-density parity-check (LDPC) code, has been widely deployed in many broadcasting and wireless communication standards [2], [3] owning to its capacityapproaching capability with acceptable decoding complexity [4]. By carefully designing the variable node degree distribution and check node degree distribution, LDPC code is able to approach the channel capacity of any binary memoryless symmetric (BMS) channel. However, as mentioned in [5], [6], conventional LDPC code does not have the property of universality with beliefpropagation (BP)-based decoding, e.g., an LDPC code optimized for a specific channel is usually suboptimal for other channels.

Recently, spatially coupled LDPC (SC-LDPC) was proposed and gained a lot of attentions in academic circles as a class of structured multi-edge type LDPC code or a class of convolutional LDPC code. In [7], Kudekar et al. showed that the asymptotic performance of a spatially coupled LDPC code under BP-based decoding is essentially equal to that of maximum aposteriori (MAP) decoding. Furthermore, [7] also proved that the MAP decoding threshold of an almost-regular spatially coupled code is capacity-approaching. Hence, without carefully designing the degree distribution, spatially coupled LDPC code with regular degree distribution has the near-capacity performance for any binary memoryless symmetric (BMS) channel [8] under BPbased decoding. Furthermore, Kudekar et al. reveals the universality of SC-LDPC in [7]. More precisely, for a group of BMS channels with equal capacity, it is possible to construct a single SC-LDPC code to approach the capacity for all these BMS channels simultaneously.
In previous works [7], SC-LDPC with sufficient length of codeword shows excellent asymptotic performance. However, when the length of codeword is limited, e.g. due to limited decoding resources or in the case of burst transmission, the performance of SC-LDPC code with limited codeword-length is not as well studied as its counterpart with infinite codeword-length. Recently, the finite-length performance of SC-LDPC code for binary erasure channel (BEC) was discussed in [9]. The author analyzed the block error probability behavior of SCLDPC code ensembles.
However, the asymptotic and finite-length analysis in [9] does not consider the rate loss due to termination and smoothing parameter, which caused an unfair comparison and analysis. As described in [7], the design rate of an SC-LDPC is a little different from the uncoupled rate (also referred as true rate). If the codeword length is unlimited, the difference between design rate and the true rate trends to zero when the length of the coupling

Section IV discusses the optimization results in section III and presents some more discussions about spatial coupling.

IV. RESULTS AND DISCUSSION
Based on the analysis before, we can derive that the trend of the rate loss of a length-N SC-LDPC is O(NŌłÆ1/3). However, as a comparison, the trend of the rate loss of a conventional LDPC code and that of the finite length channel capacity are both O(NŌłÆ1/2) according to the analysis in [1] and [10]. In other words, although the asymptotic performance of an infinitelength SC-LDPC code is excellent and has the property of universality, the finite-length behavior of SC-LDPC is much inferior to conventional LDPC code or other capacity-approaching channel codes, e.g. polar code, turbo code, etc. For example, for achieving the same performance of a conventional length-N LDPC code, the length of an SC-LDPC code should be increased to O(N3/2). And vice versa, the finite length performance of a length-N SC-LDPC code is only comparable to a conventional length-O(N2/3) channel code.
Furthermore, the derivation in this paper does not utilize much information about the code structure of the uncoded code. For example, the O(LŌłÆ1) rate loss due to finite length of the coupling chain can be interpreted as the tailing effect of spatial coupling, which also exists in some other spatially-coupled channel codes, e.g. spatially-coupled polar code [11] or multi edge-type SCLDPC code [12], etc. In this case, the trend of the rate loss of such length-N spatially-coupled channel code will also be O(NŌłÆ1/3), if the rate loss due to finite length of the uncoupled code is O(MŌłÆ1/2) which is usually true for all channel codes as the rate loss of finite-length channel capacity is O(MŌłÆ1/2).
Even though, the benefit of SC-LDPC is still attractive, especially the universality property. And if the length of the codeword is allowed to be sufficient large, the rate loss due to finite length will no-doubtfully trends to zero and can be ignored, for example, in the case of highspeed continuous communication.
