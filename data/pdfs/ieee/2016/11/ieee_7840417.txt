Spatially-Coupled Irregular LDPC Codes by Non-Square Superposition Matrices
Kengo Shibata, Masakazu Yoshida, and Jun Cheng Dept. of Intelligent Information Eng. and Sci., Doshisha University, Kyoto 610-0321 Japan
ken.shibata612@gmail.com, jcheng@ieee.org

Abstract—Spatially-coupled irregular LDPC codes by nonsquare superposition matrices are proposed. Starting with a binary base matrix and replacing non-zero entries in the base matrix by superposition matrices produces a parity-check matrix and its corresponding code ensemble. The proposed codes have two features: a) flexible code rates because the size of the superposition matrix is arbitrarily adjusted, and b) guaranteed iterative decoding performance because the degree distributions of the superposition matrix are optimized, even in a finite coupling width. The numerical analysis of density evolution and Monte-Carlo simulations show that our proposed code ensembles approach the Shannon limits at arbitrary code rates and outperform the conventional codes.
I. INTRODUCTION
Spatially-coupled (SC) low-density parity-check (LDPC) codes are constructed by connecting multiple disjoint, or uncoupled, regular LDPC code Tanner graphs into a single coupled chain. It has been demonstrated that the belief propagation (BP) threshold of the SC-LDPC code ensemble asymptotically achieves the maximum a posteriori probability (MAP) threshold of the corresponding regular LDPC code [1]. The threshold saturation of SC-LDPC has attracted a great deal of attention over the past few years.
To obtain capacity-approaching codes at a wide range of code rates, punctured SC-LDPC code ensembles have been proposed [2] that achieved a wide and continuous rate range. The BP thresholds of the code ensembles depend on those of the unpunctured code ensemble. Due to random punctuation, the BP thresholds are slightly away from the Shannon limit with the increase of the code rates. The parity-check matrices of spatially-coupled regular LDPC code ensembles were extended to obtain code ensembles with different rates [3]. The codes have finite, discrete rates since check node degree K in the uncoupled regular LDPC codes should remain constant. Although a larger K can provide more discrete rates, it also leads to high decoding complexity and a larger BP threshold gap to the Shannon limit in the low rate region for a given coupling length. More recently, rate-flexible SC-LDPC code ensembles [4] were constructed by parallely connecting multiple different coupled chains, each of which is associated with different SC-LDPC component code ensembles. The connection structure is realized by exchanging the edges of the variable nodes between different chains at the same coupling position. Note that in the above works [2] [3] [4], the SCLDPC codes with flexible rates are constructed from regular LDPC codes. The codes can in principle achieve every rational

rate. Because of threshold saturation, sufficiently large node degrees and coupling widths are chosen.
Another attempt to design rate-flexible codes was reported [5], where the uncoupled code is an irregular LDPC code. The BP thresholds of the code ensembles with slight irregularity are near the Shannon limit. Note that the irregularity in this construction was not introduced for improving the iterative decoding thresholds but for increasing the flexibility to achieve different rates while maintaining low variable node degrees.
In this paper, we propose spatially-coupled irregular LDPC codes by non-square superposition matrices. To generate a parity-check matrix of spatially-coupled irregular LDPC code ensembles, we start with a binary base matrix. Replacing non-zero entries in the base matrix by superposition matrices produces parity-check matrices and corresponding code ensembles. The superposition matrices employed are non-square, and have no-constant weights of columns and rows. Our proposed codes have the following two features: a) flexible code rates because the size of the superposition matrix is arbitrarily adjusted, and b) guaranteed iterative decoding performance because the degree distributions of the superposition matrix are optimized, even in a finite coupling width. The numerical analysis of the density evolution over a binary erasure channel (BEC) shows that our proposed code ensembles approach the Shannon limits at arbitrary code rates and outperform the conventional codes [2] [3] [4] [6]. Monte-Carlo simulations over the additive white Gaussian noise channel (AWGNC) show that our proposed codes with moderate code length are also close to the Shannon limits.

Remark 1: It is interesting to compare the constructions of the proposed code with the conventional SC-LDPC codes, e.g., previously reported code [1]. Both codes are constructed from the same base matrix. The difference is that the superposition matrices used in the proposed code are (M − k) × M irregular matrices with no-constant weights of columns and rows, while the superposition matrices in the SC-LDPC codes are M × M permutation matrices where the weight of every row and column is 1. The code rates of the SC-LDPC code are the sum of the first and second terms of Eq. (2). In our work, employing irregular superposition matrices makes it possible to construct rate-variable, capacity-approaching codes by setting and optimizing the distribution of the superposition matrices.

IV. PERFORMANCE EVALUATION
A. Numerical Analysis
In this section, we provide the numerical results for our code ensemble optimizations over BEC. The degree distributions we considered in our numerical evaluation are limited to λ(x) = λ1 + λ2x and ρ(x) = ρj x(j−1) + ρj+1xj .
For given L, J, and code rate R, we give the threshold of Eq. (7) by optimizing degree distributions λ(x) and ρ(x). Figs. 3 and 4 are the curves of the threshold versus code rate. In Fig. 3, by setting L = 16, we observe that at each code rate, the thresholds closely approach the Shannon limit εsh = 1 − R when J decreases from 4 to 2. A large threshold is provided by J = 2. The result is contrary to the conventional SC-LDPC code ensemble, where the threshold becomes large when J increases due to the rate loss [13]. In our work, even for different values of J, we can maintain code rate R as a constant by adjusting the degree distributions, and no rate-loss occurs. Explaining why a small J provides a large threshold remains an open question.
In Fig. 4, we set J = 2 and observe that the thresholds of our optimal code ensemble are close to the Shannon limits with the increase of L from 8 to 100. Even for a small spatial width, e.g., L = 8, the thresholds of our code are close to the Shannon limit. This result benefits from introducing irregular superposition matrices and optimizing their degree distributions.

At every code rate, gap G = εsh − ε∗ between each Shannon limit εsh = 1 − R and threshold ε∗ is below 0.011. As we expected, our proposed SC-iLDPC code ensembles are capacity-approaching at any code rate when L → ∞.

V. CONCLUSION
We proposed spatially-coupled irregular LDPC codes by non-square superposition matrices. We started with a binary base matrix. Replacing the non-zero entries in the base matrix by superposition matrices produces a parity-check matrix and its corresponding code ensemble. Our proposed codes have the following two features: a) flexible code rates because

the size of the superposition matrix is arbitrarily adjusted, and b) guaranteed iterative decoding performance because the degree distributions of the superposition matrix are optimized, even in a finite coupling width. The numerical analysis of density evolution over BEC shows that our proposed code ensembles approach the Shannon limits at arbitrary code rates and outperform previous conventional codes [2] [3] [4] [6]. Monte-Carlo simulations over AWGNC show that our proposed codes with moderate code length are also close to the Shannon limits.
