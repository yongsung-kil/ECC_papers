Genetic Optimization of a Short Block Length LDPC Code Accelerated by Distributed Algorithms
Jan Broulim, Sima Davarzani, Vjaceslav Georgiev, Jan Zich

Abstract—In this paper we propose an application of genetic optimization algorithms to the construction of LDPC codes with interesting correcting capabilities. A short block length (64,56) code is constructed by the accelerated genetic optimization. A highly significant acceleration of the convergence speed is achieved by the coarse grained parallelization algorithm. The best optimized (64,56) LDPC code and its comprehensive evaluation are given in the paper.
Index Terms—error correcting codes, genetic optimization, LDPC, low redundancy codes, Tanner graph.
I. INTRODUCTION
T HE past decades have seen the increasingly advances in the field of error correcting codes. Today, the Low Density Parity Check (LDPC) codes are very popular because of their good performance. LDPC codes are defined in terms of sparse parity check matrices. Regular forms of parity check matrices were introduced by Gallager in 1963 [1], but his codes were overshadowed by other error correcting codes for over 30 years. In 1995, David MacKay rediscovered LDPC codes and highlighted their excellent correcting performance [2]. Since this work, there has been an increasing interest in the field of LDPC codes.
A decoding performance of an LDPC code is significantly affected by the parameters in the associated Tanner graph [3]. This led to the extensive surveys for novel construction algorithms. Gallager’s original approach produces a random-based regular structure of the parity-check matrix and associated Tanner graph is composed of short cycles. This approach has been shown to be ineffective in the past two decades. Irregular degree distributions conclusively surpassed regular structures in the work of Luby, Mitzenmaches, Shokrollahi and Spielman [4] from 1998. Good degree distributions are usually found with the use of the density evolution algorithm proposed by Chung, Forney, Richardson and Urbanke [5]. The Progressive Edge Growth algorithm [6] for finding graphs of large girths

(large lengths of cycles) is a highly respected standard for designing good LDPC codes.
Although the density evolution algorithm produces a fast estimation of the correcting performance by means of a threshold bound, it is not convenient for short block lengths. The density evolution produces a threshold bound for particular degree distribution. The threshold bound determines the correction capabilities, assuming the codeword length n → ∞. In other words, it provides a maximum achievable performance, supposing cycle free graphs for associated degree distributions. Because of the cycle free assumption, the density evolution is not convenient for evaluating short block length codes. Thus, complex simulations are needed for measuring correction capabilities of such codes. Tanner graphs for short block length codes should be constructed by different methods. Genetic optimization algorithms can be shown to be very effective when applied to the construction of short block length codes [7].
Genetic algorithms are a part of heuristics in artificial intelligence and mathematical optimization. The algorithms have been theoretically studied, since their appearance in 1950s [8], for several applications. The greater number of applications has occurred since the 1980s and improved results have been achieved since this time. Obtaining these results by analytical methods is problematic or even impossible.
This paper proposes to a design of short block length LDPC codes by a genetic algorithm accelerated with a coarse grained parallelization. Short block length LDPC codes were chosen intentionally, because methods based on the density evolution fail for such codes and also the convergence of decoding algorithms is not optimal (occurence of short cycles in the Tanner graph). Thus, genetic algorithms in generating short LDPC codes are convenient, especially when applied for designing very interesting codes with specified correcting capabilities. Our work related to genetic algorithms and LDPC code construction is discussed in [7], [9].

During every certain time period of generations (denoted as K), an exchange of best solutions is made among threads performing the genetic optimization. This method significantly improves the convergence speed when applied in the optimization process performed by Tanner graph mutations, which are used for the reproduction of next generations in the optimization process. However, the improvement depends on both the number of computing threads and the period of generations. In the implementation, all optimization threads regularly send their best solutions to the thread responsible for the maintenance. When a certain period of generation is reached, each thread sends a query for a code created from the combination of all best solutions. After receiving the best solution, the solution is included in the optimization process performed by the particular thread.
III. PERFORMANCE EVALUATION AND CODE DESIGN
The performance of the accelerated LDPC code genetic construction was evaluated by the optimization experiment performed on LDPC (64,56) code. The code construction algorithm proposed in [9] had generated 50 different codes, which were included in the initial population. The optimization was held until a code with a capability of correcting all single bit errors using the Sum-Product (SP) decoder [12] was reached. The number of required generations (denoted as N ) is measured for different number of optimization threads and different sharing periods.

B. Best code
In this section, we present the best code achieved by the optimization algorithm. The code was selected from all optimization experiments according to the best AWGN channel simulation. The minimum distance of the code is 3. The code guarantees a 100% correction capability of single bit errors and a 100% detection capability of single and double bit errors. Furthermore, the code is able to correct 11.1% of double burst errors and detect all burst errors. Sum-Product decoding was considered. The results of correcting and detecting capabilities are shown in Table IV and V. The code constructed by our genetic optimization algorithm is presented in terms of the parity check matrix and the generator matrix in Tables VI and VII. Although LDPC codes are usually considered as codes with relatively long block length, the presented short block length code performs well on Sum-Product decoding and can find applications, where the computing resources are crucial. 100% correcting capability of single bit errors, 11% correction capability of double burst errors and detection capabilities of all burst errors are provided by the constructed code. The generator and the parity-check matrix of the best optimized (64,56) code was provided in this letter. The presented genetic optimization algorithm can be used for designing codes of a wide variety of lengths and codes with specified correcting capabilities.
