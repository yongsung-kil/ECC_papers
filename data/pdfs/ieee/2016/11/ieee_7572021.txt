A Grouping Based on Local Girths for the Group Shuffled Belief Propagation Decoding
Akiko Manada, Member, IEEE, Kyohei Yoshida, Hiroyoshi Morita, Senior Member, IEEE, and Ryuichi Tatasukawa

Abstract— Low-density parity-check codes have been known to exhibit good error correcting performance by using with the belief propagation (BP) decoding. The group shuffled BP decoding, which is a modification of the BP decoding, can exhibit a better performance than BP decoding by suitably grouping variable nodes in a Tanner graph. In this letter, we propose a new grouping of variable nodes based on local girths and evaluate its performance.
Index Terms— Low-density parity-check codes, the group shuffled belief propagation decoding, local girth.
I. INTRODUCTION
L OW-DENSITY Parity-Check (LDPC) codes [1] have been attracting coding theorists these days because of their high error correcting performance. Indeed, a parity check matrix of an LDPC code can be presented by a bipartite graph called a Tanner graph consisting of the bipartition of the variable node set V and the check node set C, and the error correcting performance based on the Belief Propagation (BP) decoding [2], a message passing decoding between nodes in Tanner graphs, is known to get closer to the Shannon limit as the codeword length goes to infinity [3].
In the BP decoding, all variable nodes pass the messages to check nodes in parallel at each iteration. On the other hand, the Group Shuffled Belief Propagation (GSBP) decoding [4], [5], a modification of the BP decoding, first partitions the variable nodes into some groups so that at each iteration 1) message passing within the same group is executed in parallel; and 2) message passing between groups is executed in sequence. Thus, the GSBP decoding allows to use updated massages within the same iteration. Hence, more reliable messages than ones at the BP decoding can be obtained in the early stage of iteration, which results in acceleration of the convergence speed (i.e. the GSBP decoding can decrease the number of iterations until a resulting word ends up a codeword).
In the GSBP decoding, how to partition variable nodes can affect its performance, so it is desirable to find suitable groupings for the decoding. Indeed, a grouping based on the degrees [6] or one based on the magnitude of Log Likelihood Ratio (LLR) [7] have been proposed in order to increase the performance. II. THE GROUP SHUFFLED BELIEF PROPAGATION DECODING

The Belief Propagation (BP) decoding is a symbol Maximum A posteriori Probability (MAP) decoding which efficiently computes the posterior probability of each bit using marginalization based on message passing in Tanner graphs.
In the BP decoding, all variable nodes execute message passing in parallel at each iteration. On the other hand, the Group Shuffled Belief Propagation (GSBP) decoding, a modification of the BP decoding, first partitions variable nodes into some groups and then decodes so that at each iteration

• message passing within the same group is executed in parallel; and
• message passing between groups is executed in sequence

to increase the convergence speed. It is possible to partition variable nodes into groups with
different size. However, in our setting, we assume that each group Gk of variable nodes has the same size NG , and execute message passing between groups in ascending order of group indices. In [5], when an LDPC code of codeword length N is given (and hence, there are N variable nodes v1, v2, . . . , vN ), each Gk is defined as

Belief Propagation (SBP) decoding [4], [5].

However, if updated massages are incorrect, then the incorrect messages are propagated at an early stage, which might invoke some delay of the converging speed. The original GSBP decoding does not consider the point, but it is desirable to prevent such a situation by suitably partitioning variable nodes.
The local girth of a variable node can be a guidepost of a suitable partition, where the local girth of a node v is defined to be the length of a shortest cycle containing v. Indeed, it is widely known that the BP decoding performs very well when Tanner graphs do not contain many short cycles, and a good code construction exploiting local girths is also proposed [8]. Furthermore, for irregular LDPC codes, executing message passing of variable nodes of higher degrees in first can increase the convergence speed in the GSBP decoding [6]. We, therefore, propose a grouping employing the degrees and the local girths.
Now suppose that there are N variable nodes v1, v2, . . . , vN . We denote by d(v) and g(v) the degree and the local girth of variable node v, respectively, where a sufficiently large number is assigned as the local girth when a variable node is not included in any cycle. Algorithm 1 explains how the N variable nodes are partitioned into r groups G1, G2, . . . , Gr of equal size NG = N/r .
Observe that for variable nodes v, w and groups Gk and Gl satisfying v ∈ Gk and w ∈ Gl ,
1) if d(v) > d(w), then k ≤ l (i.e., message passing of variable nodes of higher degree is executed earlier).
2) if d(v) = d(w) and g(v) > g(w), then k ≤ l (i.e., for variable nodes of the same degree, message passing of those with higher local girth is executed earlier).

V. CONCLUSION
In this letter, we proposed a grouping based on local girths for the GSBP decoding. We also confirmed that the proposed grouping can increase the error correcting performance and the convergence speed of the GSBP decoding especially at a small number of iterations.
As a future work, we aim to further analyze connections between variable nodes so that updated messages are better propagated. In so doing, we consider the Extrinsic Message Degree (EMD) of a variable-node group and the Approximate Cycle EMD (ACE), an upper bound on the EMD [13].
