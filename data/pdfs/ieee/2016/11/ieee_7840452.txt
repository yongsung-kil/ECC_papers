Abstract—In this paper, new constructions of variable-rate quasi-cyclic low-density parity-check (QC-LDPC) codes are proposed. The criteria which guarantee that a higher-rate QC-LDPC code can be obtained by adding column-blocks to or removing row-blocks from the parity-check matrix of a given lower-rate QC-LDPC code are derived. Based on these criteria, new families of variable-rate QC-LDPC codes can be constructed. Simulation results show that the constructed codes provide a variety of errorprotection capabilities against different channel conditions.
I. INTRODUCTION
Quasi-cyclic low-density parity-check (QC-LDPC) codes [1]–[6] are LDPC codes [7], [8] with parity-check matrices composed of blocks of circulant matrices. Owing to their structured parity-check matrices, QC-LDPC codes have the advantages of efficient encoding/decoding and allowing for high-speed very large scale integration (VLSI) implementation. Moreover, with iterative message-passing decoding [9], good QC-LDPC codes can still achieve near-Shannon-limit performance as randomly constructed LDPC codes.
Practical communication systems are often designed to operate over a wide range of channel conditions and to provide a variety of services. The transmission data rate strongly depends on the required service and channel condition. Hence, variable-rate error-correcting codes are desirable, which find applications in many systems. For example, in the IEEE 802.11ac wireless local area network (WLAN) standard [10], codes of rates 1/2, 2/3, 3/4, and 5/6 are used.
In recent years, a number of research efforts on variable-rate LDPC codes and QC-LDPC codes were made in [6], [11]– [17]. Usually, codes of multiple code rates in one system are derived from a single mother code. Then variable-rate codes can be obtained by properly modifying the mother code. The ability to use a single decoder for a sequence of variable-rate codes is also desirable to reduce the hardware cost. In [11]– [14], variable-rate LDPC codes and QC-LDPC codes were designed by puncturing. Given a mother code, the higher-rate child codes are obtained by periodically deleting certain bits of codewords of the mother code. The selection of punctured bits plays an important role on the puncturing performance. For LDPC codes and QC-LDPC codes, the complexity of

puncturing is very high due to the large size of the paritycheck matrix. Therefore, optimization of the mother code and puncturing pattern is usually carried out separately to reduce the design complexity, which results in large performance gaps to the capacity. In [15]–[17], higher-rate QC-LDPC codes were obtained by linearly combining and splitting the rows of the parity-check matrix of a lower-rate QC-LDPC code. However, some other factors which also influence the error performance of LDPC codes are not well considered in this approach; hence the performance of higher-rate and lower-rate codes is not optimized simultaneously, which induces large performance gaps to the capacity. In [6], higher-rate QC-LDPC codes were obtained by adding column-blocks to the parity-check matrix of a mother code. However, this method may produce lowerrate child codes unexpectedly. There is no criterion given in [6] to show how to properly add column-blocks to the paritycheck matrix of a mother code to obtain higher-rate QC-LDPC codes. In this paper, we make efforts to design the strategy which adds column-blocks to the the parity-check matrix of a mother code to obtain higher-rate QC-LDPC codes. We also consider how to properly remove row-blocks from the paritycheck matrix of a mother code to construct higher-rate codes.
New constructions of variable-rate QC-LDPC codes are proposed in this paper. In the first part, the lower bound on the number of the column-blocks which should be added to the parity-check matrix of a mother code to guarantee a higherrate QC-LDPC code is derived. We also give the corresponding criterion to obtain a higher-rate QC-LDPC code no matter how many column-blocks are added to the parity-check matrix if the rank of the parity-check matrix is large enough. Based on these results, we know how to properly add column-blocks to the parity-check matrix of a given QC-LDPC code and then a family of variable-rate QC-LDPC codes can be constructed. In the second part, similar criteria which state how to properly remove row-blocks from the parity-check matrix of a mother code to obtain higher-rate QC-LDPC codes are derived. Based on the criteria, we can hence construct a family of variablerate QC-LDPC codes by properly adjusting the row-blocks of a given parity-check matrix. Computer simulations are conducted to verify the code performance. The results show that our constructed codes can have different error correcting capabilities. V. CONCLUSION
In this paper, we provide the criteria to guarantee that adding
column-blocks to a given parity-check matrix of a QC-LDPC
code results in a new higher-rate QC-LDPC code. Based on
the criteria, a family of variable-rate QC-LDPC codes can be
obtained. Similar results on the construction of a new family of
variable-rate QC-LDPC codes by removing row-blocks from
a given parity-check matrix are also given. Simulation results
show that our constructed codes can provide different error
protection capabilities against different channel conditions.
