Abstract ü LDPC convolutional research has far-reaching significance. Its parity check matrix is periodic, which greatly reduces the encoding process and is easy to implement. And different convolutional LDPC codes can carry on the unified coding to the information blocks of the length transformation. The information transfer efficiency can be improved obviously. In the paper, based on the key problem about the construction of the protograph LDPC convolutional codes, the paper studies boundary extension method and matrix extension algorithm. A series of termination convolutional codes are constructed and the performance simulation is carried out. The experimental results show that the structure of terminated LDPC convolutional codes has a better performance than the protograph LDPC codes in the bit error rate.
Keywords—convolutional LDPC; boundary extension method; matrix extension algorithm; protograph LDPC;
I. INTRODUCTION
LDPC convolutional codes are proposed on the basis of LDPC. It can be seen as an infinite size loop matrix version of QC-LDPC, that is, the expansion of semi-infinite matrix is completed by grouping lifting LDPC. In the equivalent form of parity check matrix, the H matrix is equivalent to LDPC codes by copying and rewiring process coupled together, so this form of parity check matrix operation is called the corresponding code space coupling. Because of LDPC semiinfinite parity check matrix removes head and tail, LDPC block codes have the same degree distribution, it is also the sparse matrix. Because spatial position constraints and the BP algorithm characteristics, the kind of code can be decoded by the BP decoder based on the sliding window structure. In an iterative process, information transfers independent characteristic, except for the spatial constraint distance. The decoder can be implemented by a plurality of simple sub processors of a pipeline structure, so the time delay of the encoding and decoding is small. Thus it is especially beneficial to applications which are sensitive to end-to-end latency [1-4]. In addition, the decoder is easy to implement.
The performance analysis of convolutional LDPC codes has been widely studied in recent years, Lentmaier and Costello, etc. using the density evolution technique, on the convergence of LDPC convolutional codes in BEC and AWGN channel, BP decoding is systematically analyzed. It is pointed out that the threshold decoding is better than that of the LDPC block code. And Lentmaier, Sridharan, Zigangirov and Costello explain the reason of this phenomenon through

density evolution. Kudekar, Richardson and Urbanke give more analysis results. It is proved that the BEC in binary LDPC convolutional codes of BP decoding threshold can reach the underlying LDPC codes of optimal MAP decoding threshold, recently, a lot of research work shows that space coupling technology of numerous channel (BEC, BMS, MAC, relay) and communication problems have shown excellent performance.
This paper is now widely used to expand protographs LDPC, applying in the convolution LDPC. Study on the termination codes of protograph LDPC convolutional, applying this idea to the R4JA protograph. The idea structures the terminated convolutional codes, analyzing the influence of decoding performance on the different edge expansion.

V. CONCLUSION
This paper takes R4JA protographs as the prototype, constructed LDPC convolutional codes. The extension method and the matrix extension algorithm are further explored, constructed a series of termination LDPC convolutional codes, the performance experiment was carried out.
Analysis and simulation results show that the code length L is shorter, the performance of the code is similar to the performance of the protograph codes, but the complexity of the decoder is lower. When the code length is longer, that is, L is larger, T is larger The termination code has obvious advantages. First , spatial coupling codes have a threshold value in theory, which is more approximate to the Shannon limit, compared to traditional construction method. Spatial coupling method based on the local optimization of the BP threshold in the performance is similar to the global optimization of the threshold. Secondly, the complexity of the hardware implementation is much lower than that of the same code length in the storage space. By increasing the L, a long code word is constructed to achieve better performance without increasing the hardware complexity and the overhead of storage space. The termination code is suitable for the occasions where the data length is longer.
