An Efficient Algorithm for Determining the Girth and ACE Distributions in LDPC Code Tanner Graphs

Umar-Faruk Abdu-Aguye, Marcel Adrian Ambroze, and Martin Tomlinson
School of Computing, Electronics, and Mathematics Faculty of Science and Environment Plymouth University Plymouth, UK
Umar-Faruk.Abdu-Aguye@plymouth.ac.uk, M.Ambroze@plymouth.ac.uk, and M.Tomlinson@plymouth.ac.uk

Abstract— We propose a simple and efficient algorithm for evaluating the girth and approximate cycle extrinsic message degree (ACE) of all short cycles through symbol nodes in short-to-medium length low-density parity-check (LDPC) code Tanner graphs. The algorithm uses single-edge tree-apex (SETA) subgraph expansions to find all the short cycles in a code graph in a direct manner, unlike existing algorithms which use message-passing algorithms or trellis representations of code graphs. The complexity of the algorithm, when applied to a parity-check matrix H of dimension m × n, is O(n m). Due to the fact that both girth and ACE are features of cycles, we adopt parameters similar to those used for girth descriptions for ACE descriptions. Accordingly, we define the ‘local ACE’ at a symbol node or the ‘symbol node ACE’, as the minimum ACE of the short cycles through the symbol node in the graph. This basic parameter enables the local ACE distribution, minimum and maximum local ACE, modal ACE, and the average ACE of an LDPC code graph to be determined using SETA subgraph expansions. The average ACE of an LDPC code graph is a concise single-valued measure of the overall connectivity of cycles in the graph. The proposed algorithm determines the local girth and ACE distributions in irregular short length LDPC codes in less time than existing algorithms. The algorithm is not intended for quasi-cyclic (QC) LDPC codes.
Keywords—Approximate cycle extrinsic message degree (ACE), bipartite graph, cycle connectivity, girth, graph expansion, low-density parity-check (LDPC) codes, PEG algorithm, single-edge tree-apex (SETA) subgraph expansion, Tanner graph.
I. INTRODUCTION
The presence of cycles in Tanner graphs of low-density parity-check (LDPC) codes has a very significant impact on their decoding performance under iterative message-passing decoding algorithms such as the belief propagation or sum-product algorithm (BP/SPA). The performance of an LDPC code depends not only on its global girth, i.e. the length of the shortest cycle in the graph, but also on the number of shortest cycles [1], [2]; therefore, the performance depends on local girth distribution in the graph. On the one hand, it is evident that under iterative message-passing decoding algorithms, the presence of short cycles cause the edges (and symbol nodes) involved in the cycles to become correlated after a few decoding iterations which significantly reduces decoding efficiency [3]. On the other hand, cycle-free codes,

or codes with very large girths, have poor decoding performance not only under iterative decoding schemes but also under maximum likelihood (ML) decoding [4]. Therefore, it is clear that cycles are necessary in the bipartite (Tanner) graphs of capacity-approaching LDPC codes [5]. Accordingly, in order to identify the features of good practical-length LDPC codes, it is imperative to determine not only their global girths but also their local girth distributions.
Another important parameter which significantly affects the performance of LDPC codes, especially the error floor of short-to-medium length codes, is the approximate cycle extrinsic message degree (ACE) of cycles in code graphs. The ACE, a measure of the connectivity of a cycle in a graph to the rest of the graph, was first introduced in [6], [7]. Xiao and Banihashemi [10] modified the standard progressive edge-growth (PEG) algorithm [8], [9] in order to construct LDPC codes with improved ACE. Subsequently, Vukobratović and Šenk [11], [12] proposed algorithms which further improved the ACE properties of LDPC codes. Improved ACE in LDPC codes generally results in lower error floor without sacrificing performance in the waterfall region.
There are many algorithms for enumerating cycles in bipartite graphs of global girth . Recent examples of such algorithms are: i) an algorithm developed by Halford and Chugg [13] for counting cycles of length , + 2, and + 4, ii) an efficient message-passing algorithm for counting short cycles of length , + 2, . . . , 2 – 2, by Karimi and Banihashemi [14], and iii) an improved message-passing algorithm for counting short cycles by Li et al. [15], which reduces the complexity and computing time of the algorithm in [14] by a factor of two. These algorithms use rather indirect methods to find the cycles in bipartite graphs.
Although numerous studies have utilized the ACE parameters of code graphs to improve the design of LDPC codes for lower error floor performance, there are only a few algorithms dedicated to enumerating and quantifying the ACE parameters in LDPC code graphs. The most noteworthy of such algorithms was developed by Vukobratović et al. [16]. Unfortunately, published literature on cycle connectivity present ACE spectra in LDPC code graphs using complex notation and conceptualizations.

The proposed algorithm is based on a subgraph expansion technique that is a slight modification of the subgraph expansion used in the standard PEG algorithm [8], [9]. In the remainder of this paper, we will refer to the modified subgraph expansion technique as the single-edge tree-apex (SETA) subgraph expansion. In Section II, standard definitions and notations used to describe LDPC codes and graphs are presented, the SETA subgraph expansion technique for finding short cycles in graphs is explained, and a description of how SETA subgraph expansions are used to determine the girth and ACE of short cycles is given. III. THE SETA SUBGRAPH EXPANSION BASED ALGORITHM FOR DETERMINING THE GIRTH AND ACE DISTRIBUTIONS IN LDPC CODE TANNER GRAPHS
In terms of computational load, the proposed SETA subgraph expansion based algorithm for determining the local girth and ACE distributions in LDPC code Tanner graphs is very similar to the standard PEG LDPC code construction algorithm. Consequently, based on complexity estimations for the standard PEG algorithm in [9], the worst case computational complexity of the proposed algorithm is O(n m). Therefore, the complexity of the proposed algorithm is considerably less than those of the algorithms in [13], [14], and [15]. The proposed algorithm is summarized as follows.

The MPA and IMPA were implemented using the MATLAB 2013a software in a 3.4GHz CPU machine with 8 GB RAM. Code I is an irregular LDPC code with maximum symbol node degree of 15 and a maximum check node degree of 9. Code II is a (3, 8)-regular code. V. CONCLUSION
The proposed algorithm for determining the girth and ACE distributions in LDPC code Tanner graphs is based on SETA subgraph expansions. The algorithm is easy to implement and finds all the short cycles through symbol nodes in graphs in a direct manner. It is faster than existing algorithms when applied to irregular short length LDPC codes. The algorithm is not intended for application to QC-LDPC codes.
LDPC code parameters, such as global girth, maximum local girth, minimum and maximum local ACE, average ACE, and ensemble average ACE, provide valuable insight into the internal structure of code graphs. In addition to code performance simulations, these parameters can be used to identify optimal short-to-medium length LDPC code designs.
