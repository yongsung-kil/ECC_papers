Spatially Coupled Repeat-Accumulate Codes over Half-Duplex Relay Channel

Abstract—In this paper we study the design and analysis of spatially coupled repeat-accumulate (SC-RA) codes over a threeterminal half-duplex relay channel. The channels between the terminals are considered binary input additive white Gaussian noise (BIAWGN) channels, while decode-and-forward (DF) relaying is considered at the relay. We present low complexity density evolution analysis for the SC-RA code based relay code, which leads to simple code design procedure. For a wide range of design rates, we show that SC-RA code based relay code asymptotically approaches the capacity of a given relay channel. We observe that this relay code outperforms existing channel codes optimized for half-duplex DF relaying protocol. We also show the finite-length performance of the presented code through simulation results.
I. INTRODUCTION
Cooperative communication has attracted much attention for enhancing the reliability of wireless communication by obtaining diversity gain. Among different protocols studied for cooperative communication, decode-and-forward (DF) relaying strategy is one of the well-known protocols. In [1], Cover and Gamal proved that it is possible to attain the capacity of the degraded relay channel through the DF strategy.
With DF relaying protocol, the design of error correcting codes for relay channel has been broadly investigated in previous literature including [2–8]. Most of the previous works adopt the parity bit forwarding approach, where additional parity bits are transmitted by the relay to help the destination with the assumption that source’s message is correctly decoded at the relay. To transmit the additional parity bits form the relay, two different approaches were considered in previous works. In [2–6], the additional parity bits are assumed to be protected by an independent error correcting code, while [7] does not use a separate code to protect the additional parity bits. In the former case, the decoding process at the destination involves two steps: (i) decoding of the additional parity bits, (ii) decoding the source’s message with the help of additional parity bits. In [2–6], the end-to-end analysis does not take into account the effect of the relay-destination link, since the additional parity bits are assumed to be correctly decoded at the destination. Although the error free assumption in the relay-destination link is reasonable for asymptotic analysis, the assumption becomes invalid in finite-length applications. In contrast, the later approach presented by [7] takes into account the relay-destination channel, and the destination performs one joint decoding operation. Thus, the relay code design,

presented in [7], is more practical, since the code design does not consider error-free assumption of the relay-destination link. [7] utilizes irregular low-density parity-check (LDPC) codes to design a DF relay codes, which leads to capacity approaching performance. However, to achieve those capacity approaching relay codes, an extensive optimization procedure is required. Moreover, the optimization procedure becomes very complex for a variety of channel conditions [9].
Similar to the irregular LDPC codes, spatially coupled codes exhibit capacity approaching performance over point-topoint communication [10]. Thus, spatially coupled codes are potential candidates for relay code design. Moreover, using spatially coupled codes, one can avoid complicated degree optimization for different coding rates [9], since spatially coupled codes constructed from regular protographs. These facts motivate us to design relay channel code based on the spatially coupled codes while following the relaying strategy of [7]. Two prominent classes of spatially coupled codes are namely known as SC-LDPC codes and SC-RA codes. In this paper, we are particularly interested in the SC-RA code, since SC-RA code provides much simpler encoding technique than the SC-LDPC code while maintaining similar capacity approaching performance.1
In this paper, we propose relay code design using spatially coupled repeat-accumulate codes for the DF relaying protocol. We present low complexity density evolution analysis for the proposed spatially coupled relay codes considering that the variable nodes experience non-uniform SNRs. We show a simple procedure to obtain an optimized spatially coupled relay code. With numerical examples, we show that the proposed relay codes are capacity approaching for a wide range of design rates. We also show that proposed spatially coupled relay codes give better performance than existing optimized relay codes.
The remainder of the paper is organized as follows. We present brief backgrounds about the system model and the SCRA codes in Section II. Using SC-RA codes, the relay code design and analysis is presented in Section III. We present the numerical results for the proposed relay codes in Section IV. Finally we conclude in Section V.

V. CONCLUSION
In this paper, we have presented the design and analysis of practical relay code using SC-RA codes. We have demonstrated the performance of the proposed code through low complexity density evolution analysis. From the numerical results, we observe that proposed SC-RA relay code asymptotically outperforms existing relay codes. Moreover, SC-RA relay codes allow simple encoding and optimization procedures. It will be interesting to see how the proposed scheme performs over fading channels.
