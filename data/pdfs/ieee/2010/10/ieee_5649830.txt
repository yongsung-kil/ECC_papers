Adaptive Quantization for Low-Density-Parity-Check Decoders

Cha-Hao Chung, Yeong-Luh Ueng Dept. of Electrical Engineering National Tsing Hua University Hsinchu, Taiwan, R.O.C.

Ming-Che Lu Novatek Microelectronics Corporation,
Hsinchu, Taiwan, R.O.C.

Mao-Chao Lin Dept. of Electrical Engineering
National Taiwan University Taipei, Taiwan, R.O.C.

Abstract—For the implementation of low-density parity-check (LDPC) decoders, the associated error performance and the complexity are significantly affected by the number of quantization bits used. In this paper, we propose an adaptive quantization scheme, which uses different quantization schemes at different iteration numbers based on a fixed number of quantization bits. Simulation results show that the proposed adaptive quantization can reduce the number of quantization bits without error performance degradation.
Keywords— Iterative decoding, low-density parity-check (LDPC) codes, adaptive quantization, quantization, LDPC decoder.
I. INTRODUCTION
Low-density parity-check (LDPC) codes were originally discovered by Gallager [1] and were rediscovered by MacKay [2]. LDPC codes are defined by sparse parity-check matrices (PCMs) and can be represented by Tanner graphs [3] which are bipartite graphs consisting of variable nodes and check nodes. LDPC codes are attractive as a result of their nearcapacity performance [4][5][6] and their potential for highlyparallel decoder implementation [7][8]. LDPC codes for several applications have previously been discussed. Many recent communication standards, such as IEEE 802.3an [9] and DVBS2 [10] have included LDPC codes. However, the very-largescale integration (VLSI) implementation of an LDPC decoder remains a big challenge, since an LDPC decoder has complex interconnections and huge storage requirements [11][12].
It is known that the number of quantization bits or equivalently quantization error, which is the discrepancy between the fixed-point extrinsic message and the floating-point extrinsic message, directly affects the error performance and implementation complexity including both the interconnection complexity and storage complexity [20][21][22]. Consequently, quantization is a critical issue that should be resolved for an LDPC decoder. For example, there are 64800 intrinsic messages (channel values) and 226800 extrinsic messages (check-to-variable messages) for the (64800, 32400) LDPC code specified in the DVB-S2 standard. If we can reduce one bit for each intrinsic (or extrinsic) message, the storage complexity can be significantly reduced. In [13][14][23], several quantization schemes were presented. In [13], the authors proposed using different quantization schemes at different signalto-noise ratios (SNRs) for intrinsic messages. In [14][23],

intrinsic messages are scaled down as the iteration number increases in order to compensate the quantization errors caused by finite precision. It is worth noting that both quantization schemes presented in [13] and [14] focused on the quantization of the intrinsic messages.
In contrast with the quantization schemes presented in [13][14][23], in this paper, we propose a quantization scheme for extrinsic messages instead of intrinsic messages. We propose using different quantization schemes at different iteration numbers instead of at different SNRs based on fixed number of iterations. It is shown through extrinsic information transfer (EXIT) charts and the investigation of quantization error that the proposed adaptive quantization can achieve better performance compared to the conventional fixed quantization based on the same number of quantization bits. In addition, a simple but effective switching criterion is proposed such that an LDPC decoder can determine when to switch quantization scheme. To implement this switching criterion, we require only a counter to count the number of satisfied check equations at each iteration and the associated increase in computation complexity is slight. Simulation results show that the proposed adaptive quantization can reduce the number of quantization bits without error performance degradation.
The remainder of the paper is organized as follows. Section II presents the decoding algorithm adopted in this paper. Section III presents the issue of quantization in LDPC decoders. Section IV presents the proposed adaptive quantization. Section V concludes this paper.
II. LDPC DECODING ALGORITHMS
An LDPC code can be decoded using message-passing decoding (MPD) through its Tanner graph [3]. The sumproduct algorithm (SPA) [2] and the min-sum algorithm (MSA) [15] can be adopted in the MPD. Compared to the SPA, the MSA has worse error performance but is less complex in hardware implementation. Both the offset MSA (OMSA) presented in [16] and the λ-3-Min algorithm presented in [17] can achieve better error performance at the cost of slightly increased complexity compared to the MSA. Since the λ-3Min algorithm can achieve a slightly better error performance compared to the OMSA, the λ-3-Min algorithm is adopted in this paper. In addition, we modify the λ-3-Min algorithm in order to avoid the estimation of noise variance.

III. ISSUE OF QUANTIZATION IN LDPC DECODERS
In an LDPC decoder, both the intrinsic messages (channel values) and extrinsic messages (C2V messages) have to be stored. If we can use a less number of quantization bits, the storage complexity will be significantly reduced. In addition, the logic complexity (or computational complexity) of checknode and variable-node units can be significantly reduced as well. Moreover, the interconnections among the check-node units, the variable-node units, and the memory units can be simplified if a less number of quantization bits can be adopted. In summary, using a less number of quantization bits can greatly simplify the hardware implementation. However, a less number of quantization bits will degrade the error performance of an LDPC decoder. In this section, we attempt to find the

A. Perspective Based on Quantization Error
We use the notation (q : f ) to represent a quantization scheme in which the number of total bits is q and the number of fractional bits is f . A signed fixed-point representation is used and has a quantization precision of 2−f . For the (q : f ) quantization scheme, the minimum value is −2q−f−1 while the maximum value is 2q−f−1 − 2−f . When the input value of the quantizer is greater than 2q−f−1 − 2−f , the value is quantized to 2q−f−1 − 2−f . Similarly, when the input value of the quantizer is smaller than −2q−f−1, the value is quantized to −2q−f−1. We now consider three quantization schemes respectively called schemes A, B, and C. For schemes A, B, and C, quantization schemes (5:3), (5:2), and (5:1) are adopted, respectively. For scheme A, the output ranges from -2 to 1.875 and the quantization precision is 0.125. Although the quantization range for scheme C is from -8 to 7.5, its quantization precision is only 0.5.
Fig. 2 shows the absolute values of C2V messages using floating-point and fixed-point representations for the (64800, 32400) LDPC code, where schemes A, B, and C are considered. It can be seen that as the iteration number increases, the absolute value of C2V message increases. In addition, when the absolute value of extrinsic message is large, using scheme A will result in a larger quantization error compared to using scheme C. On the other hand, when the absolute value of extrinsic value is small, using scheme C will result in a larger quantization error compared to using scheme A. Moreover, we found that using scheme A or B will achieve a smaller quantization error compared to using scheme C when the iteration number is smaller than 20. On the other hand, using scheme C will achieve a smaller quantization error compared to using scheme A or B when the iteration number is larger than 25.
From the discussion given above, it can be seen that for a quantization scheme, there exists a trade-off between quantization range and quantization precision. At the beginning of iterative decoding, the absolute value of extrinsic value is small and hence we use a quantization scheme which is focused on the fractional part in order to represent the value precisely. As the iteration number increases, we use a quantization scheme which uses less fractional bits and more integer bits in order to achieve a smaller quantization error.

IV. PROPOSED ADAPTIVE QUANTIZATION
A. Switching criterion based on quantization error
In order to optimize the error performance based on a fixed number of quantization bits, we must find suitable criterion of switching quantization schemes when performing the iterative MPD. In LDPC decoding, the quantization error is likely to cause decoding error. Intuitively, a quantization scheme, which has a smaller quantization error, will have a better error performance. Consequently, we can switch the quantization scheme based on the quantization error. In other words, we choose the quantization scheme which will achieve the minimum quantization error among all the possible quantization schemes for each iteration.
Fig. 4 shows the simulation results for the (64800, 32400) LDPC code using adaptive quantization based on the quantization-error-based switching criterion. It can be seen that using the 5-bit adaptive quantization scheme achieves a better BER performance compared to using the 5-bit fixed quantization scheme although the same number of quantization bits is used. Furthermore, using the 5-bit adaptive quantization scheme can achieve almost the same BER performance compared to using the 6-bit fixed quantization scheme.

B. Switching criterion based on mutual information
Since a quantization scheme which has a better CND EXIT curve, will have a better error performance, we can switch the quantization scheme based on the mutual information for the extrinsic messages. At each iteration, we calculate the average mutual information IE of all quantized extrinsic messages using different quantization schemes and then choose the quantization scheme which will achieve the largest mutual information among all the possible quantization schemes. In order to reduce implementation complexity, we can use the threshold of mutual information to determine which quantization scheme should be used in each iteration. It can be seen from Fig. 3 that if the mutual information IA is smaller than 0.8, using the quantization scheme (5:3) achieves larger mutual information IE compared to using (5:1) while if the mutual information IA is greater than 0.8, using the quantization scheme (5:1) achieves larger mutual information IE compared to using (5:3). Consequently, when IA < 0.8, the quantization scheme used is (5:3) while when IA ≥ 0.8, the quantization scheme used is (5:1). Fig. 5 shows the simulation results for the (64800, 32400) LDPC code using adaptive quantizationbased on the mutual-information-based switching criteria. It can be seen that the switching criterion using either the threshold of mutual information or the maximum of mutual information achieves almost the same error performance. In addition, using the 5-bit adaptive quantization can achieve almost the same BER performance compared to using the 6-bit fixed quantization.

C. Switching criterion based on check equations
Since the quantization-error-based switching criterion and the mutual-information-based switching criterion are difficult to be implemented, we now present a switching criterion with lower computational complexity. We found that the number of satisfied parity-check equations (or, equivalently, parity-check constraints), denoted as Ns, is a good criterion to determine which quantization scheme should be used. For the (64800, 32400) irregular LDPC code, where the total number of check equations is 32400, the proposed quantization scheme can be implemented as follows. We use a counter to calculate the number of satisfied parity-check equations, i.e., Ns, at each iteration. When Ns <29850, the quantization scheme used is (5:3). When 29850 ≤ Ns < 31500, the quantization scheme used is (5:2). Finally, the quantization scheme used is (5:1) when Ns ≥ 31500. The switching thresholds are determined by correlating the quantization errors and the Ns values. The threshold value of Ns is the average number of satisfied paritycheck equations when we change quantization scheme based on the criterion of quantization error. Also included in Fig. 5 are the BER results for the (64800, 32400) irregular LDPC code using the check-equation-based switching criterion. From Fig. 4 and Fig. 5, it can be seen that using the check-equationbased criterion can achieve almost the same error performance compared to using the quantization-error-based switching criterion and the mutual-information-based switching criterion.

V. CONCLUSION
We have presented an adaptive quantization scheme for
extrinsic messages in an LDPC decoder. Based on a fixed
number of total quantization bits, the advantage of using
adaptive quantization is verified through the EXIT curves. A
simple but effective check-equation-based switching criterion
has been proposed in order to realize the adaptive quantization.
Based on the same BER performance, the proposed adaptive
quantization scheme requires a less number of total quanti-
zation bits compared to the conventional fixed quantization
scheme. Consequently, the proposed adaptive quantization
scheme can be used to reduce the number of quantization bits
without error performance degradation.
