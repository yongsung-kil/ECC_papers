I. INTRODUCTION
For belief propagation decoding algorithm of LDPC code, when the MATLAB is used in emulation, it is common to adopt for statement. When the iteration is being conducted, it is necessary to visit the non-zero element in the check matrix from up to down and from left to right and then calculate the probability and value of belief degree according to formulas. Thus, it will cost much visiting time. Especially, when the block code length n is very huge, for example the block code length n=103 or more, the time for one iteration is nearly I hour. Very huge block code length n will defer the emulation process, thus influencing the emulation performance.
If the belief propagation decoding algorithm of LDPC code adopts vectorization algorithm to complete, the computing time can be reduced remarkablely. Take the emulation with the block code length of 20 as example. The computing time is 33.5 s by for loop and the time is 13 s by vectorization operation. If the block code length is very huge, for example n=10008, the time is 12 h by for loop, but the average time is only 52.6 s by vectorization operation. For these reasons, this method is of great significance to decoding process of LDPC code.
