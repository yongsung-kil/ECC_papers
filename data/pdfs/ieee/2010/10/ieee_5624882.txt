IMPACT OF LLR SATURATION AND QUANTIZATION ON LDPC MIN-SUM DECODERS
N. Kanistras, I. Tsatsaragkos, I. Paraskevakos, A. Mahdi and V. Paliouras
University of Patras Computer Engineering and Informatics Department 26500 Patras, Greece

ABSTRACT
In this paper we quantify the power of noise due to quantization and saturation of the LLRs. Subsequently a model is constructed using the obtained noise power expressions that can be used to estimate the performance of various LLR quantization schemes. The model is validated by comparing the estimation with experimental BER results for an LDPC-based system that uses the min-sum layered decoding algorithm.
Index TermsŌĆö LLR quantization, soft-decoding, ┼┐nite-word length effects, LDPC decoding.
1. INTRODUCTION
Low-density parity-check (LDPC) codes were originally invented by Gallager in 1960s [1] and rediscovered in late 1990s by MacKay and Neal [2]. LDPC codes show excellent error correction performance, very close to the Shannon limit. Recently, LDPC codes have been considered for many industrial standards of next generation communication systems as Digital Video Broadcast (DVB) DVBS2, IEEE 802.3an (10GBASE-T) and IEEE 802.16e (WiMAX). An LDPC code is a linear block code characterized by a sparse parity check matrix and can be fully represented by a bipartite graph (known as Tanner graph) with two sets of nodes, called the check nodes and the variable nodes. The decoding of an LDPC code allows a high degree of parallelism, which makes it very suitable for highdata-rate applications such as wide-band wireless multimedia communications. LDPC code is called regular if the degrees of all nodes in each set are the same, while degrees of an irregular LDPC code are calculated according to some degree distributions. Although the hardware implementation of irregular LDPC codes is a bit complex, many practical applications have adopted or are considering the use of irregular LDPC codes due to their better performance [3]. LDPC codes can be decoded using the message passing algorithm. It iteratively exchanges soft information between variable and check nodes, a procedure called belief propagation (BP). Updating the nodes can be done with a canonical, two-phased scheduling (two-phase BP). The ┼┐rst phase updates all variable nodes, while the second phase updates all check nodes. The processing of individual nodes within one phase is independent and can thus be parallelized. Layered decoding [4] applies a different message schedule than the two-phase decoding. The basic idea is to process a subset of check nodes and to pass the newly calculated messages immediately to the corresponding variable nodes. The variable nodes update their outgoing messages within the same iteration. The next check-node subset will thus receive newly updated messages which improves the convergence speed [5]. There are a variety of decoding algorithms, such

as the sum-product algorithm (SP), the Log-SP algorithm, the MinSum algorithm (MS), and the normalized Min-Sum algorithm. The Log-SP algorithm has a good decoding performance but requires a large hardware complexity. The Min-Sum algorithm can significantly reduce the hardware complexity at the cost of performance degradation [6].
The quantization scheme strongly inŲĆuences the performance of an LDPC decoder architecture. While simulations are normally done with ŲĆoating point computation, any practical implementation (ASIC or FPGA) will be built with ┼┐xed-point computation. The ┼┐nite word length analysis in implementing an LDPC decoder is a very important factor since it directly impacts the size of memory to store the intrinsic and extrinsic messages and the overall hardware area. The ┼┐nite word length of the soft information affects the complexity of computation logic, routing complexity, and the decoding performance of an LDPC code. The determination of the optimum word lengths required to represent data is a tedious and time consuming procedure, which requires extensive simulations. Targeting at the objective of reducing the corresponding design time, analytical models that relate word lengths to LDPC decoding performance are sought.
Several authors have studied the impact of data quantization both at the waterfall region as well as at the error ŲĆoor. Sometimes, excellent performance is only observed up until a moderate bit error rate (BER), because at lower BER, the waterfall curve changes slope leading to a so-called error ŲĆoor [7]. Such error ŲĆoors are a major factor in limiting the use of LDPC codes in high-throughput applications. The error ŲĆoor is commonly attributed to the suboptimality of the iterative decoding algorithms on graphs with cycles, and past work has studied concepts such as near-codewords [8], trapping sets [9], stopping sets [10] and absorbing sets [11]. Zhang et al. have shown that quantization has a strong impact on which absorbing sets dominate the error-ŲĆoor region [12]. Zarubica et al. study the quantization effects for 3, 4 or 5-bit architectures, using density evolution and decoder simulation [13]. Oh and Parhi study the impact of nonuniform quantization on decoding performance of Min-Sum decoder architectures [14]. A work of great interest in [15] investigates the way in which scaling affects the dynamic range of the LLR values, considering the ┼┐nite precision for hardware implementation. They propose an adaptive quantization approach which provides the optimal performance in selecting suitable input LLR values to the decoder as far as the tradeoffs between error performance and hardware complexity are concerned. Yazdani and Ardakani ┼┐nd the optimum linear approximation of LLRs in the sense of maximum achievable transmission rate on a fading channel, assuming that the power of the additive noise is known [16]. Zhang et al. manage to improve the error ŲĆoor performance substantially,

2.3. Proposed Model
Although the saturation error power is in general greater than the corresponding roundoff error power, its contribution to the decoding performance degradation due to the information loss, that LLRs quantization causes, is in general less than the contribution of the roundoff error. The reason is that LLRs express the reliability of the corresponding bit; the larger the amplitude of the LLR, the greater the certainty of the decoder for the value of the corresponding bit. Saturation error impacts on LLRs of large amplitude, while roundoff changes the values of LLRs that have smaller absolute values. In order to estimate the impact of ┼┐nite-word-length effect to channel information loss, a weighted summation of roundoff and saturation error should be consider, with the main contribution given by the roundoff error.
Fig. 2 depicts the weighted summation of the error power, which results by taking into account 98% of roundoff error and 2% of saturation error. The desired quantization scheme is the one that contributes the less weighted power error within the noise level region that we are interested on. This means that in case, for example, of 3.5dB < Eb/N0 < 4.5dB, the quantization scheme with 5 integer and 1 fractional bits, impacts the less error power to LLRs and it is expected that this scheme achieves, within this region, the best decoding performance among the quantization schemes that issue six representation bits. The question that proposed model aspires to answer is, ŌĆ£which is the best representation scheme for channel information, namely LLRs, in case that a ┼┐xed number of representation bits is available?ŌĆØ and is almost always encountered on real applications.

The system has been designed in VHDL and mapped into FPGA using Xilinx design ŲĆow. RNG module is a Random Number Generator implemented as a Linear Feedback Shift Register (LFSR). The decoder implements the min-sum layered decoding algorithm. For noise levels higher than 5dB, (6,0) quantization scheme tends to overcome the others, as the proposed model predicts.
