QSN—A Simple Circular-Shift Network for Reconfigurable Quasi-Cyclic LDPC Decoders
Xiaoheng Chen, Shu Lin, Life Fellow, IEEE, and Venkatesh Akella

Abstract—There is an increasing need for configurable quasicyclic low-density parity-check (QC-LDPC) decoders that can support a family of structurally compatible codes instead of a single code. The key component in a configurable QC-LDPC decoder is a programmable circular-shift network that supports cyclic shifts of any size up to a predefined maximum submatrix size. This paper presents a QC-LDPC shift network (QSN), which has two key advantages over state-of-the-art solutions in recent literature. First, the QSN reduces the number of stages in the critical path, which improves the clock frequency and makes it scalable, particularly in a field-programmable gate array (FPGA)-based implementation where an interconnect delay is dominant. Second, the QSN’s control logic is simple to generate and occupies a significantly smaller area. The QSNs for a variety of codes suitable for emerging applications are implemented, targeting both a 180-nm Taiwan Semiconductor Manufacturing Company Ltd. complimentary metal–oxide–semiconductor library and a Xilinx Virtex 4 FPGA. The proposed implementation is shown to be 2.1 times faster than the best known implementation in literature and requires almost eight times less control area. Furthermore, this paper presents analytical models of the critical-path and datapath complexity for arbitrary-sized submatrices and proves that the QSN indeed generates all the output combinations required for implementing reconfigurable QC-LDPC decoders.
Index Terms—Benes network, error correction codes, quasicyclic low-density parity-check (QC-LDPC) codes, very large scale integration, WiFi, WiMAX.
I. INTRODUCTION
L OW-DENSITY parity-check (LDPC) codes, discovered by Gallager in 1962, were rediscovered and shown to approach the Shannon capacity in the late 1990s. The quasicyclic (QC) LDPC codes [1] have received significant attention, since their structures ease the hardware implementation, and have excellent error performance over noisy channels. The parity-check matrices of the QC-LDPC codes consist of circulant permutation matrices (CPMs) and/or zero matrices of the same size, which determine the interconnection of the checknode processing units (CNUs) and the variable-node processing units (VNUs). For the QC-LDPC codes, the interconnection network is highly structured and can be characterized by the submatrix size and the circular-shift value of each CPM. Thus,

masking, row, and column decomposition [1]. The dominance

The output of the QSN can be divided into two parts: 1) the left part, i.e., for 0 ≤ i < p − c, O[i] = I[i + c]; 2) the right part, i.e., for p − c ≤ i < p, O[i] = I[i − (p − c)]. Based on the observation, the output of the QSN can be viewed as the combination of two shifted arrays of the inputs. Thus, the generation of the circular-shifted array has three steps, which are listed below:
Step 1) Left shift: Generate the left part of the final output messages by performing left-shift operation on the array; let L[i] be the left-shift output, then L[i] ← I[i + c].
Step 2) Right shift: Generate the right part of the final output messages by performing right-shift operation on the array; let R[i] be the right-shift output, then R[i] ← I[i − (p − c)].
Step 3) Merge: Extract the useful part from the left-shift output and the right-shift output. When 0 ≤ i < (p − c), O[i] ← L[i]; when (p − c) ≤ i < p, O[i] ← R[i].
Steps 1 and 2 are independent and, thus, can be performed in parallel. Step 3 depends on the output of step 1 and step 2. The overall architecture is shown in Fig. 2.

B. Proof of Correctness
Next, we prove that the QSN implementation can actually generate all the required output combinations, given by O[i] = I[(i + c) mod p] for all 0 ≤ i < PM − 1. The output of the left-shift network is L[i] = I[i + c] for 0 ≤ i < (p − c). The output of the right-shift network is R[i] = I[i − (p − c)] for (p − c) ≤ i < p. Based on our merge-network control-signal generating function, the final output would be O[i] = I[i + c] for 0 ≤ i < (p − c) and O[i] = I[i − (p − c)] for (p − c) ≤ i < p. As I[(i + c) mod p] = I[i + c] when 0 ≤ i < (p − c), I[(i + c) mod p] = I[i + c − p] = I[i − (p − c)] when (p − c) ≤ i < p. Hence, it is proven.

V. CONCLUSIONS
We have presented the architecture and the implementation of the QSN, a simple circular-shift network that can be used to implement reconfigurable QC-LDPC decoders efficiently. Unlike existing solutions to this problem in the research literature, the QSN has not been derived from a Benes topology and has hence resulted in simpler control logic and fewer stages in the critical path. Consequently, the proposed network is suitable for both the ASIC and FPGA implementation of low-overhead and fast circular-shift networks for decoding QC-LDPC codes.
