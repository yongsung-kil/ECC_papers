Abstract-A bit-flipping (BF) algorithm is a decoding algorithm for LDPC codes that can be efficiently implemented by electronic circuits. Although its error correcting performance is rather poor, we show that a BF algorithm with bias terms has good error correcting performance. We also introduce two more parameters to improve the performance further. We also propose a variant of candidate bit based BF (CBBF) algorithm by introducing bias terms. We see that these algorithms having bias terms exhibit very good error correcting performances.
Index Terms-low-density parity-check (LDPC) codes, bitflipping decoding, WBF decoding, CBBF decoding
I. INTRODUCTION
Low-density parity-check (LDPC) codes are linear codes having very sparse parity check matrices. Iterative decoding for LDPC codes using soft information, e.g. sum-product (SP) algorithm, has very good decoding performance [1]. It was shown by computer simulation that if the code length is very long then the performance is very close to the Shannon limit.
There is a simple decoding algorithm called bit-flipping (BF) decoding algorithm, which uses only hard information, and therefore, it can be efficiently implemented by electronic circuits. Since the algorithm does not use any soft information, the performance of the algorithm is inferior to that of the SP algorithm.
Although, a weighted BF (WBF) algorithm can be regarded as a variant of the BF algorithm [7], the WBF algorithm uses not only hard information but also soft information during its decoding process.
A quantized modified bit-flipping (QMBF) is also a variant of the BF algorithm [3]. In the algorithm, if a received signal is relatively close to 0, a bit corresponding to the signal is considered to be unreliable. Thus, a bias term is added to a value which is used in making a decision to flip the bit. It has been shown that the performance of the algorithm is rather better than that of the BF algorithm.
A candidate bit based bit-flipping (CBBF) algorithm is a variant of the BF algorithm [4]. In the algorithm, the reliability of a parity check equation is calculated in addition to the reliability of each bit. The error correcting performance of the CBBF algorithm is also better than the BF algorithm.
In this paper, two decoding algorithms with bias terms are proposed and it is shown that these algorithms are better than the original algorithms having no bias terms.

VI. SUMMARY
A modified bit-flipping algorithm with bias terms (4 parameters) has been proposed (QMBF2) . A CBBF with bias terms (QCBF) has also been proposed. It has been shown that both of them have better error correcting performances than original algorithms.
These results may show that it is important to keep the initial belief about received signals during decoding operations. We may apply this idea to other decoding algorithms.
