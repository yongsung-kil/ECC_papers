A Class of QC-LDPC Codes with
Low Encoding Complexity and Good Error Performance
Wai M. Tam, Francis C. M. Lau, Senior Member IEEE, and Chi K. Tse, Fellow, IEEE
Abstract—We propose a novel approximate lower triangular
structure for the parity part of the parity-check matrix of
QC-LDPC codes. About half of the non-zero elements in the
parity part are set to locate on the upper diagonal while the
remaining non-zero elements can be located almost anywhere
within the lower triangular area, provided certain rules are
observed. Compared with the typical dual-diagonal structure,
the proposed structure requires very similar encoding complexity
and produces lower error rates over an AWGN channel.
Index Terms—Approximate lower triangular structure, encod-
ing complexity, QC low-density-parity-check (LDPC) codes.
I. INTRODUCTION
D
UE to their near-Shannon-limit performance, LDPC
codes have become promising candidate codes for cur-
rent and future-generation wireless communication systems
and storage systems. To facilitate the implementations of the
LDPC encoder/decoder, a class of structured codes consisting
of square sub-block matrices, named as quasi-cyclic (QC)
LDPC codes, has been proposed [1], [2]. QC-LDPC codes
possess several advantages over the randomly constructed
LDPC codes. First, the encoding of QC-LDPC codes can be
accomplished by using simple shift registers with complexity
linearly proportional to the number of parity bits, compared to
random LDPC codes which requires quadratic complexity [3].
Second, QC-LDPC codes can be represented by a base parity-
check matrix, in which each element is a square sub-block
matrix of size 𝑧by 𝑧. When compared with the parity-check
matrix, a base parity-check matrix has a much smaller size
(reduced by 𝑧× 𝑧) and therefore requires much less memory
for storage. Furthermore, a base matrix can be easily and
flexibly expanded to a parity-check matrix 𝑯for different
code lengths by using sub-block matrices with appropriate
sizes.
Moreover, the parity portion of the parity-check matrix of a
QC-LDPC code is usually established in an approximate lower
triangular (ALT) form to achieve low encoding complexity.
One typical ALT structure is a dual-diagonal structure, which
is widely used in practical systems [4], [5], [6]. Additionally,
some efficient encoding methods for QC-LDPC codes with
dual-diagonal structure have been explored [5], [6], [7]. To re-
duce the encoding complexity, the parity portion of the parity-
check matrix has also been designed directly in some special
forms, such as the triangular plus dual-diagonal structure [8]
and the triple-diagonal structure [9].

In this paper, we propose a novel ALT structure for a class
of QC-LDPC codes that allows low encoding complexity. In
contrast to the almost completely deterministic format in the
dual-diagonal structure, the proposed codes are constructed in
a semi-deterministic manner. Thanks to the flexibility of the
design, the proposed ALT structure can provide lower error
rates than the dual-diagonal structure.

In summary, the parity portion 𝑯2 is constructed in a semi-
deterministic manner. The positions of about half of the non-
zero sub-block matrices in 𝑯2 are deterministic while those of
the other non-zero elements are allowed to be located arbitrar-
ily within the lower trapezium area. When compared with the
dual-diagonal structure, the semi-deterministic structure can
increase the design flexibility of QC-LDPC codes.
Also, in the design of LDPC codes, the optimized row
weights are found to be nearly-uniform with only two or
three consecutive weight values [10]. For the proposed ALT
structure, since all of the sub-block matrices within the lower
trapezium area of 𝑯2 can be assigned as 0, 𝑰or 𝑷, the
matrix 𝑯2 can have uniform or non-uniform row weights.
Correspondingly, the row weights of 𝑯1 should be designed
to be nearly-uniform or non-uniform such that the overall row
weights of the parity-check matrix 𝑯can be maintained at
nearly-uniform.

Finally, we compare the error performance of the codes
under an additive white Gaussian noise (AWGN) channel.
Belief propagation (BP) decoding is used with a maximum
number of 80 iterations. Simulations are performed until we
get at least 100 block errors for each bit-energy-to-noise-
power-spectral-density ratio (𝐸𝑏/𝑁0). Fig. 2 shows the block
error rate (BLER) and the bit error rate (BER) of the A code,
B code and the IEEE 802.16e code. The A code, having
the same column-weight and row-weight distributions as the
IEEE 802.16e code but with non-uniform row weights of 𝑯𝑏1
and 𝑯𝑏2, outperforms the IEEE 802.16e code. Moreover, the
B code, which possesses a slightly different column-weight
distribution compared with the A code and the IEEE 802.16e
code, shows superior error performance over the A code and
the IEEE 802.16e code.
IV. CONCLUSION
We have proposed a novel ALT structure for a class of QC-
LDPC codes. A corresponding efficient encoding method has
been given. Simulations results have shown that the proposed
class of QC-LDPC codes outperforms the IEEE 802.16e code
with dual-diagonal structure under an AWGN environment.
