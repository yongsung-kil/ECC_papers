On the Design of Irregular
LDPC Code Ensembles for BIAWGN Channels
Hamid Saeedi, Member, IEEE, and Amir H. Banihashemi, Senior Member, IEEE
Abstract—Existing design methods for irregular Low-Density
Parity-Check (LDPC) codes over the additive white Gaussian
noise channel are based on using asymptotic analysis tools such
as density evolution in an optimization process. Such a process is
computationally expensive particularly when a large number of
constituent variable node degrees are involved in the design. In
this paper, we propose a systematic approach for the design of
irregular LDPC codes. The proposed method, which is based on
a pre-computed upper bound on the fraction of edges connected
to variable nodes of degree 3, is considerably less complex than
the conventional optimization approach. Through a number of
examples, we demonstrate that using our method, ensembles with
performance very close to those devised based on optimization,
can be designed. In addition to having very good performance,
the number of constituent variable node degrees in the designed
ensembles is only three or four. This, in some cases, is much
smaller than the corresponding number for optimization-based
designs with similar performance.
Index Terms—Low-density parity-check codes, LDPC codes,
irregular LDPC codes, AWGN channel, code design, systematic
design, stability condition.
I. INTRODUCTION
L
OW-density parity-check (LDPC) codes [1], which ex-
hibit very good performance under iterative decoding al-
gorithms, has been an active area of research in the past decade
or so. An important topic in this area is code design, defined,
at very large block lengths, as the design of variable and check
node degree distributions (𝜆(𝑥), 𝜌(𝑥)) for an ensemble of
codes [2]. Concentration results [3] would then guarantee that
any sufficiently long code taken from the ensemble performs
identical to the average performance of the ensemble, which
is predicted by asymptotic analysis tools such as density
evolution [2], [3] as well as simpler one-dimensional tools
like Extrinsic Information Transfer (EXIT) charts [4] and
Gaussian approximation [5], [6]. For most well-known channel
models such as the binary erasure channel (BEC), the binary
symmetric channel (BSC) and the additive white Gaussian
noise (AWGN) channel, asymptotic analysis tools have been
developed and widely used. For such channels, the code
design can then be carried out based on the asymptotic tools
through an optimization process [2]. The process is however

For the code design, there are two main categories in
general:
1) For a given channel parameter, we look for an ensemble
with maximum rate and negligible average probability of error,
2) For a given rate, the ensemble capable of providing a
reliable transmission for the worst possible channel parameter
is designed.
The second category, corresponding to finding the best ensem-
ble threshold for a given rate, is of more practical interest,
while the first category is usually easier to design. For a given
set of constituent variable and check node degrees, and for a
given channel parameter, the ensemble (𝜆(𝑥), 𝜌(𝑥)) is called
convergent if the average probability of error tends to zero
as the number of iterations tends to infinity. A convergent
ensemble with the highest rate is called the optimum ensemble
for the first category of code design.
In this paper, we propose a systematic method for the code
design over the BIAWGN channel. The channel has 1 as the
input, and the additive Gaussian noise has zero mean and
variance 𝜎2. We first focus on the first category of code
design. The design begins with a given channel parameter
𝜎2, and a given check node degree distribution and aims at
devising a variable node degree distribution which maximizes
the rate for reliable transmission. The maximization of rate
is performed in a greedy fashion by first maximizing 𝜆2, the
percentage of edges connected to variable nodes of degree
2, and then maximizing 𝜆3, while ensuring the convergence
of the ensemble. At the core of the proposed method is
an upper bound on 𝜆3, which is pre-computed to reduce
the complexity. We show that by using only variable node
degrees 2, 3, and 𝐷𝑣(the maximum variable node degree),
and in some cases, a fourth degree between 3 and 𝐷𝑣, one
can design ensembles with close-to-optimum performance
with considerably less computational complexity compared
to the conventional optimization-based designs. The proposed
method for the first category of code design is then extended
to the second category. Although in this paper, we focus

for 𝐷𝑣, are also given. In the proposed algorithm, we first
set the values of 𝜆2 and 𝜆3 equal to their maximum values.
We then search for the smallest 𝐷𝑣for which the resulting
ensemble is convergent. Based on the discussions of Section
III, this results in the largest code rate for a convergent
ensemble with the three constituent variable node degrees 2,
3 and 𝐷𝑣, and with the constraints that 𝜆2 = 𝜆2,𝑚𝑎𝑥and
𝜆3 = 𝜆3,𝑚𝑎𝑥. If the resulting 𝐷𝑣happens to be equal or

4 < 𝑗< 𝐷𝑣, to the distribution,2 and increase 𝜆𝑗as far as
the ensemble remains convergent. Alternatively, the resulting

VI. CONCLUSION
In this paper, we propose a systematic approach to design
LDPC codes for BIAWGN channels. Our method, which
is based on a pre-computed upper bound on the fraction
of edges connected to degree 3, is computationally much
less complex than the conventional optimization-based design
methods. Compared to some of the best ensembles designed
by optimization, the proposed ensembles perform practically
the same and yet have a smaller number of constituent variable
and check node degrees as well as smaller average variable
and check node degrees.
