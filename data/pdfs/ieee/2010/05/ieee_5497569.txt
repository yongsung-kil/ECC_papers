Abstract—An adaptive-normalized min-sum (AN-MS) algo-
rithm for decoding low-density parity-check (LDPC) codes is
proposed. Unlike the normalized min-sum (NMS) algorithm, the
normalization factor is adaptively adjusted according to the state
of check nodes in each iteration. Simulation results show that the
proposed AN-MS algorithm can perform better than the NMS
algorithm while still preserving the low complexity of the min-
sum algorithm.

I. INTRODUCTION
Iterative decoding of low-density parity-check (LDPC)
codes has received much attention in recent years ( [1]–
[4], and references therein). The standard sum-product (SP)
algorithm can achieve excellent performance but often with
heavy implementation complexity [1]. In particular, the SP
algorithm requires to estimate the signal-to-noise ratio (SNR),
which is undesirable in practice. The min-sum (MS) algorithm
, as an approximation to the SP algorithm, perform worse
than the SP algorithm. However, the MS algorithm has the
advantage of both low implementation complexity and no
requirement for SNR estimation.
The sub-optimality of the MS algorithm in performance
comes from the overestimation of the message at check
nodes, therefore, the improvement can be achieved with a
normalization or offset operation at check nodes. The resulted
algorithm is the normalized MS (NMS) or the offset MS
(OMS) algorithm [3], which has received much interest in
practice. In this paper, we propose an adaptive-normalized
min-sum (AN-MS) algorithm, which can further improve the
performance of the NMS algorithm.

II. ADAPTIVE-NORMALIZED MIN-SUM ALGORITHM
A. Motivation
In this section, we based on the NMS algorithm to make a
further improvement. We notice that the decoding iteration can
terminate immediately if all the checks are satisfied. However,
if this is not the case, there should be some check nodes which
are checked in error. For a error check node, there must exist at
least one variable node which passes the wrong message to this
check node. By the rule of min-sum algorithm, the messages
updated at this check node is not reliable and preferable to be
suppressed.
Based on this observation, we propose an adaptive-
normalized min-sum algorithm. For this new NMS algorithm,
the normalization factors at check nodes can be adjusted
according to the state of the check nodes. Here, the state of
a check node is defined as the check sum of the incoming
messages.

D. Computational Complexity
Compared to the NMS algorithm, increase in computational
complexity can be well expected. In general, there are two
aspects for additional complexity. One of which is the decision
of the state of check nodes, and the other is the use of
two normalization factor. If the dynamic early termination of
decoding is used for both the NMS and AN-MS algorithms,
the decision of the state of check nodes should be performed
at each iteration for both algorithms. Hence, the decision of
state of check nodes at each iteration incur no additional imple-
mentation complexity. Therefore, the use of two normalization
factors dominates the additional computational complexity.
For an ASIC implementation of the algorithm, we argue
that the adaptive use of two kinds of normalization factors
does not incur much more computational complexity. In fact,
the multiplicative operation can be equivalently implemented
as the series of both addition and shift operations. For exam-
ple, the multiplication of 𝑥with 0.875 can be implemented
as 0.875 × 𝑥= 𝑥/2 + 𝑥/4 + 𝑥/8, which includes three
shift operations and two additions. Hence, the value of the
normalization factor responds to a group of shift coefficients
in implementation. Therefore, the AN-MS algorithm requires
two groups of shift coefficients, which are adaptively selected
according to the decision of the state of check nodes. Hence,
there is a minor increase in memory and control logic for the
AN-MS algorithm compared to the NMS algorithm.

IV. CONCLUSION
We have proposed an adaptive-normalized min-sum de-
coding algorithm for LDPC codes. Compared to the NMS
decoding, the proposed AN-MS decoding can achieve better
performance with a minor increase in complexity. Therefore,
we can expect that it is very competitive in practice.
Future work includes a theoretical determination of normal-
ization factors.
