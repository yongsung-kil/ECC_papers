It is a dynamic scheduling belief propagation using residuals, and has some advantages, such as fast decoding, good performance, and low complexity. It is similar to residual belief propagation (RBP),but has some diffe rence in computing the residual message. Simulation shows that it outperforms with only a maximum of ten iterations by about 0.28 dB compared with RBP at an BER of 10-4•

Low density parity code[l] is practical application and the error correction code that the performance close to Shannon limit . Thus it is paid widely attention, and has a vital effect for wireless communication. B ecause there is large numbers of information transmission between variable nodes and check nodes in decoding, in order to reducing the LDPC decoding complexity and the power consumption of decoder, the key is to reduce computation transmit information.
Researchers have proposed some excellent decoding algorithm, which residual information BP decoding algorithm [2] is an effective dynamic scheduling decoding algorithm, it is decoding faster than non-dynamic scheduling decoding algorithm under a few number of iterations, but the performance of BER is not good and has high complexity. The VC - RBP decoding algorithm researched in the paper has good B ER performance, low complexity and faster decoding performance compared with RBP decoding algorithm, the difference between them is the computing of the residual message. The computing of the RBP residual message is before and after the message that transport from check nodes to variable nodes ,but the computing of the VC­ RBP residual message is before and after the message that
transport from variable nodes to check nodes.

II. THE LDPC CODE lHAT HAS THE CHARACTER I sr I C OF QUASI CYCL IC
Although LDPC code has good performance and low complexity compared with Turbo code, coding is more complex. Turbo code can code in linear time , while the time complexity of LDPC encoder is the quadratic of code length. In order to reduce the complexity of LDPC code, IEEE802.

I6e provides quasi cyclic LDPC code[3 ] ,the LDPC coding
rules in standards of IEEE802. 16e is fol low ing: H matrix is composed by block matrix. Each block matrix is shown by a number, and when the number is equal or greater than :lero ,it means the digit that the unit matrix moving to right circularly , when the number is less than :lero, it means :lero matrix. Thus, we can expand model matrix into a very large sparse matrix. The QC-LDPC code use shift register to code, greatly reducing the coding complexit y, avoiding the matrix inverse operation, saving the storage space of check matrix, and ma k ing the coding process simplifY.

From the performance aspect of error correction, VC­ RBP guarantese better performance than RBP. RBP tends to propagate first the message to the less reliable variable nodes because the message which has the largest residual is based on only one check equation. The greediness of RBP can generate new errors that need a large number of message updates to be corrected. VC-RBP propagates first the message which has the largest residual based on all check equations. So, VC-RBP is a less greedy algorithm , VC-RBP solves the trapping set more effectively by updating the check nodes and then simultaneously all the variable nodes representing the check equation.
