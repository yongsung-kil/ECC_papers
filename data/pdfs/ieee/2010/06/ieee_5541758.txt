Abstract-In order to improve the performance and the decoding convergence speed of the improved weighted bit­ flipping algorithm (IWBF), an algorithm based on the characteristics of parallel bit-flipping algorithm that updates multiple bits in each iteration is presented here. Also this algorithm is used in the image transmission. Simulation results show that: improved algorithm not only enables significant improvement in bit error performance , the decoding convergence speed has also been greatly improved, and the image transmission in communication can obtain high speed and high equality.

I. INTRODUCTION (HEADING 1)
For low-density parity-check (LDPC) codes, one class of linear block codes, was proposed by Gallager[l ] in 1962. MacKay[2] discovered and proved that LDPC codes are kinds of good codes which could approach the Shannon limit. The performance of LDPC codes using iterative soft information decoding method even better than the Turbo codes, this is by far a hot spot in the fields of information and communication. The algorithms in [3] [ 4] , which we called improved weighted bit-flipping (IWBF) algorithm, use soft information of the received sequence weighted on the flipping criterion, and flip the bit with the largest flipping criterion. This algorithm provides a good performance and complexity tradeoff. In the class of bit-flipping algorithms based on iterations, the average number of iterations in large measure affects the decoding efficiency. As the IWBF algorithm is a serial decoding algorithm with a common drawback of only one bit flipped in each iteration, this is not conducive to the fast convergence. This paper takes into account the performance and the convergence speed, and puts forward an improvement based on the IWBF algorithm: depending on the iterative characteristics of the IWBF algorithm, multiple bits were updated in each iteration (called MIWBF), which is similar to parallel algorithms, so that the number of decoding iterations could be significantly reduced and the performance could be improved. Simulation shows that the MIWBF algorithm not only gets a certain degree of improvement in performance, but also greatly

III. PROPOSED ALGORITHM FOR IWBF (MIWBF)
The convergence speed of IWBF algorithm is slow mainly because of its serial mechanism. In order to speed up its convergence rate, multiple bits can be flipped in each iteration according to a certain rule. The idea borrowed from [6] is to determine the bit flipping criterion from a threshold so as to build a neighboring of bits to be flipped. V. CONCLUSION
Depending on the iterative characteristics of IWBF algorithm, MIWBF algorithm updating multiple bits in each iteration is proposed. The simulation results demonstrate that the performance of MIWBF algorithm is better than the standard IWBF algorithm about O.SdB. When SNR is larger than 4.SdB, the average number of iterations for each code word is greatly reduced. Experiments proved that MIWBF algorithm can obtain very fast decoding convergence speed for quality assurance when MIWBF algorithm is applied to image transmission. The image restoration quality of the MIWBF algorithm at the iteration limit of 2 times is better than other types of bit flipping algorithms at the iteration limit of IS times so that the effectiveness and reliability of the image transmission could be greatly improved.
