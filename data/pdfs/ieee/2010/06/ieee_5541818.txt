Abstract-A modified Min-Sum algorithm for low-density 
parity check codes is proposed in this paper. The inaccurate 
check node messages of the Min-Sum algorithm compared to 
the belief propagation (BP) algorithm are compensated by 
modified variable node messages, which is different from the 
normalized 
Min-Sum 
algorithm. 
The 
modified 
factor 
is 
obtained 
before 
decoding 
program 
and 
only 
one 
extra 
multiplication 
is 
needed 
in one 
cycle. 
So 
the 
increased 
complexity is fairly 10w.The simulation results show that the 
bit error rate (BER) and average number of iteration of the 
modified 
Min-Sum 
algorithm 
are 
very 
close 
to 
the 
BP 
algorithm, the modified algorithm can improve the BER 
performance compared to the Min-Sum algorithm while added 
complexity slightly.

INTRODUCTION 
Low density parity check (LDPC) codes are one of the 
error correcting codes which were first discovered in 
nineteen sixties by Gallager in his doctoral dissertation [1]. 
In the last twenty years, many researchers focus on the 
LDPC codes[2] [3]. 
The main decoding algorithms of LDPC codes include 
soft-decision such as Belief Propagation (BP) algorithm and 
hard-decision one. Both versions are iterative decoding 
schemes. Performance and complexity are contradictive. 
Compared with the hard-decision algorithm, the soft­
decision one has better performance with higher complexity. 
The BP algorithm is a kind of common used decoding 
algorithm which Wiberg and Davey were used to LDPC 
codes[4] [5] with fairly good performance. To reduce the 
complexity of the BP algorithm, Fossorier proposed Min­
Sum algorithm[6] which is also called UMP BP-Based 
algorithm with a signification reduction of complexity, but at 
the expense of about I-dB degradation in error performance 
with respect to the BP algorithm at the bit error rate 10-5 • 
Chen and Fossorier proposed to improve the accuracy of the 
soft values by normalization factor[7], named normalized 
Min-Sum algorithm, which obtained a good performance­
complexity tradeoff. 
The compensation to soft value of the normalized Min­
Sum algorithm is processed at check node. But in the final 
analysis, the codeword can be output after hard decision of 
variable node messages from decoding program in Section II.

modified Min-Sum algorithm improves compared to the

The Performance of the modified Min-Sum algorithm

CONCLUSION 
In this paper, we consider that the variable node message 
in the Min-Sum algorithm can be modified to be close to BP 
algorithm. When computing variable message, a modified 
factor a can be added to compensate the inaccuracy which 
is induced by approximate value in check message. The 
optimal value a should be confirmed before the decoding 
process, so the increased complexity is fairly small during 
the decoding process. The simulation result shows that the 
modified algorithm can 
improve the bit 
error rate 
performance of the Min-Sum algorithm and it is a useful 
decoding algorithm.
