Abstract—In this paper, we introduce the concept of correctable error set for the BSC, which allows to generalize sum-product decoding for this channel. As a result, better error performance or faster convergence can be achieved. Furthermore, the correctable error set allows to evaluate the error performance of generalized sum-product decoding with a given iteration number for the BSC.
I. INTRODUCTION
LDPC codes are known as a class of capacity approaching codes in the sense of Shannon’s limit, when decoded with sum-product decoding [1]. Computer experiments have shown LDPC codes can achieve a good error-correcting performance. On the other hand, there are no fomulas for an accurate error-rate evaluation of sum-product decoding of a given LDPC code.
One research trend on LDPC codes is the theoretical analysis of code space and its related structures, e.g. “minimum distance” [2], [3], [4], “weight distribution” [5], [6], [7], “stopping set” [8], and so on. These approaches provide not only bounds or approximate values of error-rates but also guidelines for the construction of good performance LDPC codes.
In this paper, we generalize sum-product decoding by introducing an additional parameter “initialization”. In the process, we introduce the concept of correctable error set.
II. THEORETICAL CONTRIBUTION
Consider the following general communication scenario: a sender chooses a codeword c of a binary error-correcting code associated with an LDPC matrix H and sends to a receiver over a noisy channel. If the noisy channel is a binary symmetric channel (BSC) with cross over probability q, each bit of the codeword c flips with the probability q and the receiver obtains a bit sequence y. The receiver inputs the parameters H, lmax, q and the received word y to a sum-product decoder SPDec, where lmax is a parameter called the maximum iteration number. The communication succeeds if the output is c, and it fails otherwise. The word error rate is the failure probability of the communication scenario. In this paper, we analyze

the word error-rate of LDPC codes with sum-product decoding over a BSC.
We introduce a simple idea for analyzing the word error-rate. The idea is to introduce another value p to SPDec as the estimated cross over probability of the channel. We call p the initialization of the decoder. As a result, the word error rate is regarded as a function of two variables p and q for fixed parameters H and lmax. Let it be denoted by WERH,lmax (p, q). With this notation, the word-error rate of the original sum-product decoding is equal to WERH,lmax (q, q).
The following theorem presents some properties of WERH,lmax (p, q):
Theorem II.1. Let H be a parity-check matrix and WERH,lmax (p, q) be the word error rate for an initialization p, a cross over probability q, and the maximal iterative number lmax. Then we have the following:
1) For a fixed initialization p0, WER(p0, q) is a polynomial of q. 2) For any initialization p, WER(p, 1/2)×2N is an integer, where
N is the length of H. Furthermore, WER(p, 1/2) is a discrete function of p. 3) For some parity-check matrix H, WER(p, p), the word errorrate of the original sum-product decoding, is not a continuous function of p.
Theorem II.1 3) implies that there may be no polynomial representation of the word-error rate WER(p, p) as a function of p for some LDPC codes. On the other hand, by Theorem II.1 1) for a fixed initialization p0 the exists a polynomial representation of the word error rate WER(p0, q) as a function of q for any LDPC code. This fact motivates us to investigate sum-product decoding with a fixed initialization.

IV. CORRECTABLE ERROR SET
Assuming s = 0, we denote the output of the sum-product decoding as SPDec[H, y, p, lmax]. Then the following statements are equivalent:
• SPDec[H, y, p, lmax] = 0, • SPDec[H, c + y, p, lmax] = c,
where c is any codeword defined by H. It suggests that we can define the correctable error set for an LDPC code under fixed initialization p and maximal iteration number lmax as the set of error patterns corrected by SPDec. VIII. CONCLUSION
In this paper, we introduced the concept of correctable error set, by noticing that for the BSC, the sum product decoder with a given iteration number only depends on the initialized probability of error p0. Although this value has been conventionally selected as the BSC crossover probability, we showed that other selections can provide better performance or faster convergence. We also proved that for any fixed initialization p0 (i.e. any given correctable error set), the word error rate can be represented as a polynomial of the BSC crossover probability . This suggests that the word error rate can be analytically derived from the (total or partial) knowledge of the correctable error set.
While these results have been derived for the BSC, the same concepts can be extended to any discrete input - discrete output channel model, in particular quantized versions of the AWGN channel.

VII. WORD ERROR RATE ESTIMATION AND CORRECTABLE ERROR SET FOR QUANTUM LDPC CODES
We introduce an application to evaluate the word-error rate for quantum LDPC codes. It is known that the experiment for quantum CSS codes is implementable over classical computer [11]. A quantum LDPC code of type CSS is a pair of LDPC codes associated with parity-check matrices Hx and Hz such that HxHzT = 0. A quantum codeword of the CSS code is characterized by a complex linear combination of quantum states defined by d∈D⊥ |c + d , where D⊥ is the dual code associated with Hz and c is a codeword defined by Hx.
The experiment is composed of the following processes:
1 Set a quantum Pauli channel p(I)+p(X)+p(Z)+p(XZ) = 1. 2 Generate a pair of error vectors ex, ez related to the Pauli
channel randomly. 3 Calculate syndromes sx, sz by sx = Hxex, sz = Hzez. 4 Input syndromes sx and sz to a sum-product syndrome decoder
and obtain outputs eox, eoz. 5 Decoding succeeds if ex − eox ∈ Hz and ez − eoz ∈ Hx .
Decoding fails otherwise, where H is a code space generated by H.
