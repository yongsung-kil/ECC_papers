Using Time-Aware Memory Sensing to Address Resistance Drift Issue in Multi-Level Phase Change Memory
Wei Xu, and Tong Zhang ECSE Department, Rensselaer Polytechnic Institute, NY

Abstract—Because of its great scalability potential and support of multi-level per cell storage, phase change memory has become a topic of great current interest. However, recent studies show that structural relaxation effect makes the resistance of phase change material drift over the time, which can severely degrade multi-level phase change memory storage reliability. This paper studies the potential of using a time-aware memory sensing strategy to address this challenge. The basic idea is to keep track of memory content lifetime and, when memory is being read, accordingly adjust the memory sensing configuration to minimize the negative impact of time-dependent resistance drift on memory storage reliability. Because multi-level phase change memory may demand the use of powerful error correction code (ECC) whose decoding can request either harddecision or soft-decision log-likelihood (LLR) memory sensing, we discuss both hard-decision and soft-decision time-aware memory sensing in details. Using BCH code and LDPC code as ECC for 4-level/cell and 8-level/cell phase change memory, we carry out simulations and the results show that, compared with time-independent static memory sensing, time-aware memory sensing can increase allowable memory content lifetime by several orders of magnitude.
Keywords—Multi-level phase change memory, structural relaxation, time-aware sensing, BCH, LDPC
I. Introduction
Past a few years experienced a resurgence of interest in search for highly scalable universal memory [1, 2] fueled by the wellrecognized scalability limits of current mainstream memory technologies such as SRAM, DRAM and flash memories. Phase change memory is one of the most promising candidates that have attracted a lot of attentions (e.g., see [3–7]). In phase change memory, data storage is realized by configuring the chalcogenide material, usually Ge2Sb2T e5 (GST), into either a crystalline or an amorphous phase with resistances differing by several orders of magnitude [8]. Due to the large resistance margin between crystalline and amorphous phases, phase change memory can realize multi-level per cell storage based upon incomplete phase transition [3, 4]. However, as demonstrated in recent studies [9–12], resistance of phase change material may drift over the time with significant variability in terms of drift speed. This clearly poses a severe reliability challenge, particularly for multi-level phase change memory.
In this work, we investigate the potential of using time-aware memory sensing to address such resistance drift issue in multilevel phase change memory. The basic idea is very intuitive: if we have the resistance drift statistical model and know the lifetime of each word (i.e., how long each word has been held in memory), we should be able to estimate the resistance drift

characteristics and accordingly adjust memory sensing configurations to maximize memory sensing accuracy. Assuming the availability of phase change memory cell resistance drift statistical model, this paper studies one possible realization of such time-aware memory sensing. In particular, when each word is being written to multi-level phase change memory, we append the current time as a time stamp to the word; when each word is being read, we can obtain the word lifetime by comparing the current time and word time stamp, based on which we can estimate the cell resistance statistical distribution parameters (e.g., the mean and deviation of each storage level) and accordingly adjust how to sense the memory word. It is evident that error correction code (ECC) will inevitably play an essential role in ensuring the storage integrity of multi-level phase change memory. Such time-ware memory sensing strategy can work well for both classical ECC such as BCH code, for which memory sensing only needs to deliver memory cell content hard decision, and advanced iterative ECC such as low-density paritycheck (LDPC) code, for which memory sensing must provide soft probability information of memory cell content.
To demonstrate the potential of this intuitive time-aware memory sensing strategy, we use hypothetical 4-level/cell and 8-level/cell phase change memories as test vehicle with memory cell resistance drift statistical parameters from recent device studies [11,12]. With respect to phase change memory ECC, we consider the use of both BCH code and LDPC code. Simulation results show that such time-aware memory sensing strategy can significantly improve the overall system reliability over timeindependent static memory sensing. This can directly translate to a much longer allowable memory content lifetime, e.g., our simulation results suggest that the word lifetime in 4-level/cell memory may improve by 4 and 7 orders of magnitude when BCH and LDPC codes are being used, respectively.
II. Background
As shown in Fig. 1(a), one phase change memory cell consists of an access control device (e.g. an nMOS transistor) and a storage element, i.e., a chalcogenide layer (Ge2Sb2T e5, or GST) between a top metal contact and a resistive bottom electrode (BE). The top of GST is connected to the bit-line (BL) through the top metal and the bottom of GST is connected to the drain of the nMOS transistor. Being heated by current pulses with different amplitudes and durations, the GST can switch between the amorphous (RESET) state and crystalline (SET) state. When the chalcogenide material is amorphous (or crystalline), the GST shows low (or high) resistance. The resistance may differ by several orders of magnitude, leading to excellent nonvolatile storage capability.

Moreover, due to the large resistance margin between crystalline and amorphous states, phase change memory can support multi-bit per cell storage by realizing incomplete phase transition, e.g., researchers have recently demonstrated 4-level/cell (i.e., 2-bit/cell) phase change memory cells with four distinguishable resistance states [3]: very amorphous, amorphous, semi-crystalline and crystalline. Given its obvious storage density and hence cost advantages, successful development of multi-level phase change memory is highly desirable and even can be critical for phase change memory technology to enter mainstream market.
However, phase change memory, particularly multi-level phase change memory, is subject to a severe storage reliability issue: while the crystalline phase of phase change chalcogenide material is very stable over the time, the amorphous phase (and other phases obtained by incomplete phase transition in multilevel phase change memory) is metastable and can experience structural relaxation [9], which results in resistance drift over the time. III. Proposed Time-Aware Sensing Strategy The basic idea of time-aware memory sensing is very intuitive
and can be illustrated in Fig. 2. It shows the resistance drift of 4-level phase change memory cells. Since a larger initial resistance tends to incur a larger drift exponent, higher storage levels with larger initial resistance are subject to bigger resistance drift, as illustrated in Fig. 2. Assume memory cells are written at t = 0 and are read at t = T . As illustrated in Fig. 2, if we use a static memory sensing, i.e., the memory sensing thresholds are fixed, a noticeable amount of memory read errors will be induced. On the other hand, if we can keep track of the lifetime of memory cells, we can estimate the resistance drift values based upon resistance drift statistical model, and hence dynamically adjust the sensing thresholds to minimize the memory read errors, as illustrated in Fig. 2.
Fig. 2. A simple illustration of time-aware memory sensing.
As pointed out in Section II, drift exponent ν has a large degree of variability, e.g., let µν and σν represent the mean and deviation of drift exponent, σν can be over 30% of µν as shown in [11]. Therefore, resistance window of each storage level not only shifts but also becomes wider over the time. This may result in increasingly non-negligible overlaps between adjacent storage levels over the time, leading to storage reliability degradation. Therefore, appropriate use of ECC becomes almost indispensable to ensure multi-level phase change memory storage integrity. In this work, we consider two types of ECC, including (i) classical hard-decision ECC such as BCH code, where the decoding only needs hard decision of memory content (i.e., memory sensing only needs to make binary estimation on each bit), and (ii) advanced soft-decision ECC such as LDPC code, where the decoding needs probability information of memory content (i.e., memory sensing should not only make binary estimation on each bit but also provide the probability or confidence of the decision). Clearly, the time-aware sensing as illustrated in Fig. 2 is only sufficient for hard-decision ECC. If soft-decision ECC is being used, we have to use more sensing thresholds with a finer granularity and carry out more sophisticated computation to estimate soft decision for each bit, which will be elaborated later this section.

Fig. 4 shows the simulated raw BER performance of harddecision sensing before ECC decoding in the case of both timeindependent static sensing and time-aware sensing. In the case of time-aware sensing, we consider different finite precisions of the sensing thresholds calculated using (5), i.e., we use 3 integer bits and 2 to 4 fractional bits as shown in Fig. 4. Simulation results clearly show that time-aware sensing can largely reduce the raw BER for both 4-level/cell and 8-level/cell scenarios. As shown in Fig. 4, when using time-aware sensing, the raw BER slightly decreases at the beginning and then moderately increases over the time. This counterintuitive behavior can be intuitively explained as follows. Since memory cells with higher initial resistance value have larger resistance drift exponents, the distance between the mean of adjacent resistance levels will increase over the time, which tends to reduce the raw memory hard-decision sensing BER. On the other hand, the resistance distribution of each level spreads over the time due to the variability of resistance drift exponent, which tends to increase distribution overlaps between adjacent storage levels and hence increase the raw memory hard-decision sensing BER. At the beginning, the resistance distribution spread is relatively small and hence its impact on raw BER can be off-set by the increased distance between mean of adjacent resistance levels, leading to a net slight decrease of raw BER. However, the resistance distribution spread continues to accumulate over the time and quickly become dominant compared with increased distance between mean of adjacent resistance levels, leading to continuous net increase of raw BER.

We set ECC code rate as 0.94. We construct a binary (34816, 32768, 128) BCH code over GF (216), where each BCH codeword contains 34816 bits and can correct up to 128-bit errors. In this study, we construct a length-34816, rate-0.94 QC-LDPC code whose parity check matrix contains an array of 2 × 34 circulant matrices with size of 1024, where all the circulant matrices have a column weight of 2 and are constructed randomly subject to the 4-cycle free constraint. Fig. 5 shows the corresponding sector error rate (SER), i.e., ECC decoding failure rate, for both 4-level/cell and 8-level/cell phase change memory. Because simulation of LDPC code decoding is very computation intensive and the simulation time

significantly increases as we reduce target SER, the solid parts of LDPC performance curves are obtained from simulations while the dashed parts are simply extrapolated, as shown in Fig. 5. Using time-independent static sensing, the SER quickly increases over the time, leading to very short allowable memory content lifetime. For example, the 4-level/cell phase change memory can only achieve a SER of 10−3 before the memory content lifetime reaches 104s (i.e., less than 3 hours) even using the powerful LDPC code as shown in Fig. 5(a), and it becomes much worse for the 8-level/cell phase change memory as shown in Fig. 5(b). Results in Fig. 5 clearly show that timeaware sensing can significantly increase the allowable memory content lifetime by several orders of magnitude. As shown in Fig. 5, when BCH code is being used, the SER with time-aware hard-decision sensing achieves 10−6 at 106s and 105s for 4level/cell and 8-level/cell phase change memory, representing about 4 orders of magnitude improvement over their static sensing results. When LDPC code is being used, the SER with timeaware soft-decision sensing achieves 10−6 at 1010s and 108s for 4-level/cell and 8-level/cell phase change memory, representing about 7 orders of magnitude improvement. Finally, we note that, although both 4-level/cell and 8-level/cell phase change memory have the same raw BER at the beginning, the 4-level/cell phase change memory can allow a much longer memory content lifetime compared with its 8-level/cell counterpart, which suggests a clear trade-off between the multi-level phase change memory storage density and memory content lifetime.
V. Conclusions
This paper presents a time-aware memory sensing strategy to improve multi-level phase change memory reliability. Due to the inherent structural relaxation effect, phase change material is subject to resistance drift with significant variability over the time, which inevitably results in storage reliability degradation and severe constraint on allowable memory content lifetime.
