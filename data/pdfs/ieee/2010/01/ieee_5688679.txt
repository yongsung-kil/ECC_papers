Modified Decoding Algorithm for IRA Code in MIMO-OFDM Communication System

Meng Zhang, Weifeng Sun, Meng Xu, Chuanhai Zhou, Hao Liu National ASIC System Engineering Research Center Southeast University, Nanjing 210096, P.R. China E-mail: zmeng@seu.edu.cn

AbstractвҖ”Irregular Repeat Accumulate(IRA) code has drawn a lot of attention recently to its prominent decoding algorithm, whose performance approaches to Shannon limit. Belief propagation decoding algorithm is the best decoding algorithm for IRA code, which has great hardware complexity when implemented. Min-sum decoding algorithm modifies belief propagation decoding algorithm at the cost of decoding performance. A modified decoding algorithm is proposed to simplify belief propagation decoding algorithm with a little of loss in decoding performance according to minimum mean square error rule. The modified decoding algorithm can be applied to MIMO-OFDM communication system to decrease Inter-Carrier-Interference. Simulation results show that if MIMO-OFDM system needs SNR to be 10б§©5, the encoding gain
of modified decoding algorithm can be 0.25dB and 0.15dB better than normalized decoding algorithm and offset decoding algorithm respectively. And the modified decoding algorithm also has lower hardware complexity.

I. INTRODUCTION
How to improve transmission speed of data and utilization efficiency of frequency spectrum has been a hotspot in wireless communications field[1]. Current research shows space multiplex technology of unite system can heighten transmission speed of data without improving transmission power and bandwidth. However, transmission performance of MIMO system becomes worse in channel of great frequency selectivity[2]. By partitioning channel into many sub-channels, OFDM system can resist frequency selectivity fading effectively. Therefore, the strategy of joining MIMO and OFDM has been paid on a lot of attention. In multipath environment, many sub-carriers can be checked correctly, but the performance of MIMO-OFDM communication system will be worse because of some sub-carriers of great frequency selectivity, due to Inter-Carrier-Interference (ICI)[3]. This problem can be solved by encoding for correcting.
IRA(Irregular Repeat Accumulate) code[4][5][6]is a kind of linear block codes approaching to Shannon limit, which has lower complexity than LDPC codes. BP (Belief Propagation) decoding algorithm is the best decoding algorithm, but BP decoding algorithm is very complex when implemented because it refers hyperbolic tangent function and its inverse function. V. CONCLUSION
Making the best of the ideas of normalized decoding algorithm and offset decoding algorithm, a modified decoding algorithm is proposed according to MMSE rule in this paper. And the modified decoding algorithm has been used in MIMOOFDM communication system in order to achieve good decoding performance.
