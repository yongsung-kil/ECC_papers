Abstract- A new memory mapping technique using a VLSI
circuit based on recursive quadratic polynomial equations is
described. The proposed methodology allows parallel processing
elements, such as used in Turbo and Low Density Parity Check
(LDPC) decoders, to work independently across memory
segments, thus enabling parallel, high throughput, and power
efficient LSI circuits.
I. INTRODUCTION
Two-phase Turbo and LDPC decoders have shown error correcting performance approaching channel capacity and are being adopted in several high throughput systems. A number of recent industry standards such as the high throughput wireless LAN standard IEEE 802.11n, the high rate wireless PAN standard 802.15.3 and the high rate wireless MAN standard IEEE 802.16 have opted to include them as channel decoding schemes.
A high throughput decoder suitable for such standards may be obtained by parallelizing the operating blocks. With an effective contention free interleaving, the increase in throughput such decoders provide gets proportional to the number of processors used. For example, in [1] several log­ MAP processors operate on smaller-sized windows within each received frame. A significant hardware problem in such parallel processors is how to handle memory contentions, which occur when multiple processors try to access the same memory simultaneously [1].
An efficient contention resolving strategy will therefore require use of contention free interleavers. Interleavers for Turbo codes [2] - [8] have been extensively reported. Takeshita et al. in [9] suggested the use of permutation polynomial-based (QPP) interleavers over integer rings. The QPP interleavers have an error correcting performance equivalent to random interleavers [10], [11] and in addition provide contention free access to memories.
In this paper we present an efficient recursive implementation for parallel interleavers based on the QPP formulation. This new formulation allows processing elements

such as those used in two-phase decoders to work in parallel. As a result the interleaved address storage is replaced with an 'on the fly' calculation, which saves circuit area and power consumption.
This paper is organized as follows; section II describes the binomial interleaver and associated interleaved memory addresses using a simple example. In section III we describe our recursive address computation algorithm mathematically. In section IV an application specific integrated circuit (ASIC) implementation of the polynomial interleaver is described. Section V compares the results when interleavers are fabricated on a 90nm CMOS process and compared to memory based solutions.

CONCLUSION
A new methodology for recursive QPP interleaving that allows a higher level of parallelism is disclosed. Using this methodology a 3 Gb/sec parallel QPP interleaver is implemented in 90nm CMOS process. The interleaver occupies only 0.05 mm2 of area and consumes 12 mW of power using 12 parallel blocks. The interleaver can be deployed in parallel turbo and LDPC decoders at a fraction of the power and area required for interleaver memories. The power consumption can be traded with throughput by having lesser number of parallel interleaver blocks.
