Selecting Error Patters Based on Symbol Reliability For OSD Algorithm

DONG Zi-jian QIAO Guo-lei
School of Electronic Engineering Huaihai Institute of Technology Lianyungang, Jiangsu 222005
zjdong@hhit.edu.cn

Abstract—A new scheme of selecting error pattern is proposed. OSD algorithm processes certain MRIPs of a received sequence, performs bits flip among MRIPs, forms error patterns, and then generates candidate codeword. The bits flip is equal probability among MRIPs in OSD algorithms, which leads to too many candidates generated, and seriously affects the implement of algorithm. This paper proposes an improved algorithm which adopts unequal probability strategy utilizing the reliability of received symbols. The simulation results show that the proposed algorithm is effective. With little performance degradation, the computational complexity of algorithm can be reduced by 60% or more for one class of LDPC codes.
Keywords-ordered statistic decoding (OSD); most reliable independent positions (MRIPs); error pattern
I. INTRODUCTION
Ordered statistics decoding (OSD) and related algorithms such as the box and match algorithm (BMA) are all maximum-likelihood decoding algorithms to decode linear block codes. OSD can also be cascaded with belief propagation (BP) to decode LDPC (Low Density Parity Check) codes with medium/short code length. OSD algorithm processes certain most reliable independent positions (MRIPs) of a received sequence, performs bits flip among MRIPs, forms error patterns, and then generates candidate codeword. The bits flip is now equal probability among MRIPs. This strategy leads to so many candidates generated, and seriously affects the implementation of algorithm. Combined with the reliability of received symbols, we can adopt unequal probability strategy to select the error patterns, which means that, if symbol reliability is high, it can be ignored, or selected with lower probability to be as an “error bit”; if less reliable, then the higher the probability of selection.

V. CONCLUSION
We propose a scheme for the selection of error patterns in OSD algorithm based on the reliability of received sequence. This method can save a lot of computing time with slightly performance degradation. Simulation results show that the proposed algorithm is viable. The algorithm can further achieve a good tradeoff between computational complexity and decoding performance with suitable choice of MRIPs and reasonable arrangements for the error patterns.
