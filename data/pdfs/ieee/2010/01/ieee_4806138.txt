Low-Complexity Switch Network for Reconfigurable LDPC Decoders
Daesun Oh and Keshab K. Parhi

Abstract—In this paper, we propose an efficient low-complexity switch network design for reconfigurable low-density parity-check (LDPC) decoders. The proposed architecture leads to significant reductions in hardware complexity. Since the structured quasi-cyclic (QC) LDPC codes for most modern wireless communication systems include multiple code rates, various block lengths, and different sizes of submatrices, a reconfigurable LDPC decoder is desirable and the barrel shifter needs to be programmable. The Benes network cannot be optimized as the barrel shifter for a reconfigurable LDPC decoder when the input size of barrel shifter is not a power of 2. Also, it is not trivial to generate all the control signals on-the-fly for numerous 2 2 switches in the switch network. In this paper, a novel low-complexity switch network design is proposed, which can be used efficiently when the input size of barrel shifters is not a power of 2. Furthermore, we propose a novel algorithm to generate all the control signals, which can be implemented with a small size of lookup table (LUT) or a simple combination logic on-the-fly, using the properties that both the full-size switch network can be broken into two half-size switch networks and the barrel shifters for the structured QC LDPC decoders require only cyclic shifts. Compared with conventional Benes networks using a dedicated LUT or a complicated signal generating algorithm, the proposed architectures achieve significant hardware reductions in implementing the barrel shifters for reconfigurable LDPC decoders. In synthesis result using the TSMC 0.18- m standard cell CMOS technology, the proposed switch network for a reconfigurable LDPC decoder of IEEE 802.16e and IEEE 802.11n can be implemented with an area of 0.772 mm2, which leads to a significant area reduction.
Index Terms—Barrel shifter, Benes network, low-density paritycheck (LDPC) decoder, reconfigurable, switch network, VLSI.
I. INTRODUCTION
T HE highly parallel implementations and remarkable error correcting capabilities of low-density parity-check (LDPC) codes [1] have led them into modern wireless communications standards, such as IEEE 802.11n [2], IEEE 802.16e [3], and DVB-S2 [4]. Recently, many efforts have been attempted to simplify the VLSI implementation of the related decoders called “block-LDPC codes” [5] or “architecture-aware LDPC codes” [6]. Based on these design approaches, the structured quasi-cyclic (QC) LDPC codes have received significant attention due to their efficient hardware implementations and good decoding performances [7]. In the structured QC LDPC codes, the parity check matrix

is composed of cyclic permutation submatrices that determine

the interconnection network between the check node updating

units (CNUs) and the variable node updating units (VNUs). The

barrel shifting network for interconnections between CNUs and

VNUs can be predetermined and optimized for a specific given

code. However, since LDPC decoders for most modern wire-

less communication systems must accommodate multiple code

rates, various block lengths, and different sizes of submatrices,

the barrel shifting networks need to be dynamically reconfig-

urable [8]–[10].

Generally, a barrel shifter is optimized for a given permu-

tation matrix with the well-known

Benes network

[11], [12], which is capable of performing any sort of permuta-

tion over data units where is a power of 2. To properly

control all the 2 2 switches in the Benes network, the con-

figurable bits must be provided for all of the output com-

binations. However, since only cyclic shifts are required in the

barrel shifter of the structured QC LDPC decoders, the number

of output combinations is reduced to

. Typ-

ically, when implementing the barrel shifter for a reconfigurable

LDPC decoder, these control signals of the Benes network are

stored in a dedicated lookup table (LUT) [8], [9]. However,

the straightforward implementation such as LUT may make the

large size of Benes networks inefficient or impractical.

Recently, a reconfigurable shuffle network architecture has

been presented in [13] and [14], which is based on the use of

two Benes networks: one shuffles the data while the other con-

figures the first one on-the-fly. Even though this can reduce the

hardware complexity compared to direct implementations using

-to-1 multiplexers, it still requires a huge hardware com-

plexity. Also, a newly designed controller for the Benes network

has been proposed in [15], which offers a significant area reduc-

tion by designing the control logic. However, using the Benes

network as barrel shifter for the reconfigurable LDPC decoder

may not be optimal when the largest input size of barrel shifter

is not a power of 2.

The logarithmic barrel shifter [16] composed of modular cells

or the multisize circular shifting networks [17] consisting of sev-

eral flexible barrel shifters and adaptation networks with multi-

stage multiplexer can support not arbitrary but predefined ex-

pansion factors for specific LDPC decoders such as the IEEE

802.16e or IEEE 802.11n systems. Also, their precomputed con-

trol signals need to be stored in a dedicated LUT, which re-

quires a large implementation area. The shifter-based permuta-

tion structures are implemented in [18] with small area, which

In this paper, we propose a novel efficient switch network design for reconfigurable LDPC decoders, which can support any kind of cyclic shifts equal to or less than the largest input of switch networks. The proposed design can be implemented with lower hardware complexity than original Benes networks when the input size of switch network is not a power of 2. In addition, an efficient algorithm to generate all the control signals of the switch network is proposed by making use of two valuable properties: one is that the full-size switch network can be broken into two half-size switch networks, and the other is that the barrel shifter in the structured QC LDPC decoder requires only cyclic shifts.
This paper is organized as follows. Section II briefly reviews the structured QC LDPC codes and decoders. Also, the switch networks and barrel shifters for reconfigurable LDPC decoders are discussed. In Section III, we propose a novel low-complexity switch network design. In Section IV, an efficient algorithm to generate all the control signals for the proposed switch network is proposed, which can be implemented with much less hardware complexity. Also, the hardware complexity of proposed designs is compared with those of the direct implementations using LUT and the design presented in [13]. Finally, Section V concludes this paper.

V. CONCLUSION

In this paper, we have proposed a low-complexity switch net-

work design for reconfigurable LDPC decoders. The Benes net-

work cannot be not optimized as the barrel shifter for reconfig-

urable LDPC decoders when the input size of barrel shifter is

not a power of 2. Also, it is not trivial to generate all the control

signals on-the-fly for a lot of 2 2 switches in the switch net-

work. A low-complexity switch network and its control signal

generators are proposed, which can be used efficiently when the

input size of barrel shifters does not belong to a power of 2.

For example, when a 96-input barrel shifter for the reconfig-

urable LDPC decoder is required, the proposed 96 96 switch

network can be applied efficiently. The 96 96 switch network

and its control signal generator can be implemented with a low

area of 0.772

mm and a maximum clock fre-

quency of 94 MHz for 8 bit word length compared to the conven-

tional ones requiring a implementation area of 2.171 mm and a

maximum clock frequency of 85 MHz. Therefore, our proposed

designs achieve a significant area reduction of 67% in imple-

menting switch networks for reconfigurable LDPC decoders.
