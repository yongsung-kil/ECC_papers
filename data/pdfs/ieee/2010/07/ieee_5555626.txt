Improved Syndrome-based Ordered Statistic Decoding Algorithm for LDPC Codes

DONG Zijian, LIU Fei, SONG Yaolian
College of Communication and Information Engineering Nanjing University of Posts and Telecommunications Nanjing, China zjdong@hhit.edu.cn

Abstract—Ordered statistics decoding (OSD) algorithm and syndrome–based decoding algorithm are investigated in this work, based on which, a syndrome-based ordered statistics decoding (SOSD) algorithm for LDPC codes is proposed. The indexed systematic form of parity check matrix is obtained based on the least reliable independent positions (LRIPs) by performing Gaussian eliminations, which promises that the SOSD processing need not the re-encoding process, but only several binary vector additions, therefore achieves noticeable computation complexity reduction.
Keywords- low-density parity-check codes; reliability-based decoding; syndrome-based ordered statistics decoding.
I. INTRODUCTION
For medium and short length LDPC codes, the performance of belief propagation (BP) decoding, though effective, is regarded as suboptimal, therefore the ordered statistics decoding (OSD) [1] concatenated to BP is proposed [2]. For such a BP-OSD combined decoding, OSD is called in each BP iteration if the stopping criterion is not satisfied, which may results in an intolerable decoding complexity. A simple and intuitive scheme is proposed [3], in which OSD is performed just once, if no valid codeword is returned after BP decoding with maximum iterations. The reliability information delivered from the last BP iteration may be inaccurate due to the LLR oscillation phenomenon [4], which may results in a faulty OSD decoding. In [3], the accumulated LLR information is used to reconstruct ordered information sequence, instead of the last BP LLR output, which achieves a noticeable performance gain.
The syndrome-based ordered statistic decoding algorithm (SOSD) is also a MLD algorithm for linear group code [5], in which, the parity check matrix is supposed to be full rank, and got by matrix transformation of the systematic generator
matrix, which is obtained by first searching out the k
MRIPs, and then implementing Gaussian eliminations on original generator matrix. This process is complicated, and more important, the parity check matrix of LDPC code may be non-full rank. This letter proposes a modified SOSD algorithm, which can be adapted to all LDPC codes with full rank or not. The process for obtaining systematic form parity check matrix is also simplified by matrix transformation from the original parity check matrix, instead from the original generator matrix. This process is more convenient for LDPC code which is defined by parity check matrix.

IV.

CONCLUSION

We propose an improved BP-SOSD algorithm for LDPC codes, which can save much complexity compared with the existing BP-OSD. For an error pattern, the candidate codeword is obtained by only several binary vector additions in the proposed algorithm, instead of the re-encoding process in general BP-OSD algorithm. Simulations show that our proposed method is viable and effective.
