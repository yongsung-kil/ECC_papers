Performance of Adaptive Low-Density Parity Check Codes in a Low Cost Spectrum-Sliced WDM Network

Shaobo Sun, Mark. S. Leeson School of Engineering, University of Warwick, Coventry, CV4 7AL, UK Tel: +44 (0)24 7652 3908, Fax: +44 (0)24 7652 2129, e-mail: {s.sun.1;mark.leeson}@warwick.ac.uk
ABSTRACT In this paper, regular low-density parity-check (LDPC) codes have been applied for the first time to a low cost spectrum-sliced WDM system. The Gaussian approximation of the noise in such system has been commonly used in previous studies, and there is a standard LDPC code for symmetric additive white Gaussian noise (AWGN) channels. However, in the system considered here the actual noise character is asymmetric and chisquare distributed. Due to the specific channel property, an adaptive decoding algorithm based on the precise chi-square noise statistics has been proposed. Simulations are conducted for various code rates. The results show that such a forward error correction scheme provides significant coding gain for a dedicated system and the adaptive algorithm outperforms decoding with conventional Gaussian approximations. Keywords: Forward error correction (FEC), low-density parity check codes, optical fiber communication,
spectrum slicing, wavelength division multiplexing (WDM).

1. INTRODUCTION
High capacity wavelength division multiplexed passive optical networks (WDM-PONs) have been considered as a promising approach in next generation access applications for providing sustained bit rates beyond 2.5 Gb/s. The approach of spectrum slicing (SS) has been investigated as a lower cost alternative to conventional WDM systems [1]. In SS-WDM, individually modulated spectral slices of a cheap broadband incoherent source are employed. However, beat noise and chromatic dispersion have been the main impairments to SS-WDM operation by inducing power penalties. As a result, there have a number of investigations dedicated to mitigating the power penalty and enhancing the system capacity, [2]-[4]. Recently, low-density parity check (LDPC) codes [5] have become of great interest due to their achievable near Shannon limit error performance [6]. A regular LDPC code is a linear code featuring a binary sparse parity-check matrix which contains only a small weight per column and a uniform weight per row. It has been shown that LDPCs are able to provide 10dB coding gain at a BER of 10-9 for high-speed (> 40 Gb/s) long-haul WDM channels [7]. With such features, LDPC codes have been considered strong candidates for high-speed optical communications. It is of a great interest to investigate the performance of LDPC codes in a low-cost but noisy and dispersive SS-WDM system.
In the existing literature, the BER performance of LDPC codes has been assessed under the assumption that noise is white and zero mean symmetric Gaussian distributed [8]. However, in SS-WDM the noise character is not well modelled by such an assumption [2]. Hence, it is necessary to estimate the probability density functions of SS channel outputs and thus to assign initial likelihoods accordingly in the soft iterative decoding scheme.
In this paper, an LDPC based error correction scheme intended to combat dispersion and other impairments in SS-WDM is developed. The design is adapted to the specific noise distribution in the relevant access network. The simulation uses a realistic model taking account of all the major impairments in SS-WDM systems. In Section 2, the channel models employed are described; Section 3 presents the LDPC codes adapted to the SS-WDM asymmetric channel in systems; simulation results form Section 4; the conclusion is the fifth section.

In an SS-WDM system, the asymmetric distribution of “ON” and “OFF” after passing through the receiver leads to different noise variances and hence the asymmetric probability density functions (pdf) P1 and P0. In the following, the chi-square and the Gaussian channel models will be described and compared based on the result in [2]. The dispersion will accordingly be considered in the calculation for an optical preamplifier receiver (OPR).

3. LDPC CODES ADAPTED TO THE ASYMMETRIC CHANNEL IN SS-WDM SYSTEMS
LDPC codes can be specified by a Non-Systematic Sparse Parity-Check Matrix, H, so that an (n, k) LDPC code has a matrix with (n-k) rows and n columns. All short cycles in the matrix graph should be eliminated through row operations. For each transmitted data array s = [s1 s2 s3 … sk], the emitted codeword u = [u1 u2 u3 … un] is obtained by u = s × G in the encoder. The process of encoding adds (n-k) redundant bits at the end of the information data decreasing the data rate by the code rate factor R = k/n. The coded data intensity modulates the spectrum sliced light source and is then multiplexed. At the receiver, Belief Propagation (BP) decoding for

Monte Carlo simulations were conducted with the universal system parameters to in Table 1 validate the LDPC adaptive decoding algorithm using equiprobable OOK and chi-square distributed global noise. The validation of the new algorithm is shown in Fig. 2, where it may be seen that it can provide considerable BER gains for the LDPC with various redundancies, for example an order of magnitude enhancement is demonstrated with LDPC (414,207) at Q = 1.8. The BER improvement is more significant with increasing Q; the modified algorithm outperforms the conservative Gaussian approximation.
Fig. 3 illustrates the optimum system parameters decided by both beat noise and dispersion. It is seen that for a given BER, the beat noise dominates the global noise at small values of m while dispersion dominates at the other end in terms of increasing the required power leafing to an optimum m value for minimum signal power.

Fig. 4 shows the BER performance using LDPC and OPRs against average received power, showing LDPC coding gain of ~9.5 dB and improvements with the length of the LDPC codes. The adaptive decoding scheme gives ~3.5 dB greater coding gain with the (1024,512) code compared to the (414,207) code. Fig. 4 also shows the BERs derived from LDPC(1057,813) and LDPC(1057,990), which give smaller coding gains (~7 dB and ~4.5 dB, respectively) but reduced redundancies (23% and 6.3% respectively) than at R = 0.5. The performance is superior to any FEC scheme so far applied to SS-WDM systems with the same code rate [4]. For OPRs, an obvious error floor of 10-10 is observed for m = 20 because of the tradeoff between large amplifier ASE noise from the high-gain condition and the dominating beat noise from the low value of m. Therefore, even increasing the signal power provides little further improvement in the SNR to make the BER meet the requirement. It is seen that increasing the value of m may eliminate the error floor but associate with larger optical bandwidth per channel hence lead to more severe dispersion and fewer available channels. In contrast, LDPC provides an effective means to enhance the receiver sensitivity at low value of m, without any error floor observed.

5. CONCLUSIONS
A novel regular LDPC code decoding algorithm has been is designed for the SS-WDM system. It accommodates beat noise and dispersion, which are the main causes of penalty in the systems considered. The analysis made use of the exact chi-square noise distributions rather than the Gaussian approximation, producing superior BER performance estimates. Codes with rates from 0.5 to 0.937 and block lengths from 414 to 1057 were investigated. The adaptive LDPC outperformed other FEC codes on SS-WDM so far, providing almost 10dB of coding gain using an algorithm tailored to asymmetric channels in contrast to previous AWGN LDPC analyses.
