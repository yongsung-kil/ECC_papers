Design of A Novel Multi-Rate QC-LDPC Decoder

Pengjun Wang and Fanglong Yi
Institute of Circuits and Systems Ningbo University Ningbo, China
wangpengjun@nbu. edu.cn

Xiaofang Zhou
ASIC & Systems State Key Laboratory Fudan University Shanghai, China
xiaofangzhou@fudan. edu.cn

Abstract- Based on layered decoding and semi-parallel
structure, a novel architecture of multi-rate QC-LDPC decoder is proposed in this paper. It can support any code rates without any changes in hardware to achieve higher hardware utilization. The update of current layer and the comparison of next layer are processed simultaneously to improve the throughput. The check-to-variable messages are stored indirectly to reduce the storage space. Based on the proposed architecture, a QC-LDPC decoder of 2304 bits and 6-encodings style is presented. Then ModelSim SE6.0 simulation results verify that the proposed decoder is correct and effective. Finally, the decoder is
synthesized by Synopsys Design Compiler on SMIC 0.18 ILm CMOS technology, and the maximum throughput can achieve 318 Mbps at 145 MHz and 15 iterations.

Keywords- multi-rate; QC-LDPC; decoder

I.

INTRODUCTION

Low-density parity-check (LDPC) code, a linear block code defined by a sparse parity-check matrix, was firstly invented by Gallager in 1962 [1]. LDPC code has engaged tremendous attention due to its near Shannon limit performance and its low-complex decoding algorithm. Therefore, it is being widely used in communication standards, such as DVB-S2, IEEE 802. 16e and IEEE 802. 1In. In practice, Quasi-cyclic LDPC (QC-LDPC) code is a special kind of LDPC code, which has excellent performance and is well suited for hardware implementation owing to its regular parity-check matrix [2].

In communication systems, the interferences which are introduced by transmission medium affect the reliability of communication to different extent under different environment. When the signal-to-noise ratio (SNR) is low, a channel coding of low code rate is needed to protect the validity of the signal; On the contrary, a channel coding of high code rate is needed [3]. Thus, in order to adapt to complex rates changing in communication systems, it is important to design multi-rate LDPC decoder which can process both high code rate and low code rate, and it becomes a research hotspot in channel coding [4-5].

Through the analysis and research on parity-check matrices of QC-LDPC codes which are defined by IEEE 802. 16e, a novel multi-rate QC-LDPC decoder is proposed based on layered decoding and semi-parallel structure. And any code rates can be processed by the decoder without any changes. The reminder of this paper is organized as follows. Section II briefly reviews QC-LDPC code and the decoding

algorithm. Section III presents the architecture of multi-rate QC-LDPC decoder. Section IV shows the implementation results and comparison, and Section V concludes the paper.
II. QC-LDPC CODE AND THE DECODING ALGORITHM
A. QC-LDPC Code
QC-LDPC code is a special type of LDPC code, whose generator matrix and parity-check matrix have a quasi-cyclic character. As the QC-LDPC code can be encoded in the way of shifting to reduce encoding complexity, it has been used in various communication systems. The parity-check matrix of QC-LDPC code is constructed by MxN sub-matrix. And each sub-matrix is a zxz cyclic-shifted identity matrix or zxz zero-matrix.
IEEE 802. 16e is an emerging broadband wireless access technology to provide high-speed connection for Internet access. To adapt to different communication environments, it provides QC-LDPC codes of 4-code rates and 6-encodings [6]. Because the six parity-check matrices have similar structures, the decoding processes are similar. B. The Decoding Algorithm ofLDPC Code
So far, researchers have proposed a variety of decoding algorithms of LDPC code. Out of all these methods, best results are achieved with belief propagation (BP) decoding [7]. But the process needs to compute the complex exponential function and logarithmic function, which is difficult to achieve by hardware. Min-Sum (MS) algorithm is an approximation of BP algorithm [8], which uses the minimal function instead of the exponential function and logarithmic function to reduce the complexity. However, its performance IS worse.

Normalized min-sum (NMS) algorithm compensates the loss performance of MS algorithm by normalization factor [9]. Therefore, better performance is obtained with small cost.

There are two formats of LDPC decoding process. One is the traditional two phase message propagation (TPMP) decoding [2], and the other is layered decoding [10] which is suitable for QC-LDPC code. In TPMP decoding, the delay is serious and the throughput is limited, because row operations and column operations are implemented separately. Moreover, a lot of memories are required to store the check-to-variable messages and the variable-to-check messages. However, column operations are merged into row operations to reduce the delay in layered decoding. Furthermore, the more reliable messages of variable nodes from the previous layer are immediately passed to next layer to speed up convergence. Consequently, NMS algorithm based on layered decoding is adopted in this paper.

III. THE ARCHITECTURE OF MULTI-RATE DECODER
How to design the computing units and the storage structure according to the similarity of parity-check matrices is the key issue. A multi-rate LDPC decoder based on semi-parallel structure has been presented in [3]. It sets the number of check nodes processors (CNP) and variable nodes processors (VNP) according to the largest parity-check matrix (lowest code rate). However, some code rates require less number of CNP when the decoder processes other code rates, which means some of the CNP are not always full utilized. Another multi-rate QC-LDPC decoder is presented in [11]. It requires a lot of memory blocks and large storage space.

V. CONCLUSION
A low-complexity and efficient QC-LDPC decoder has been proposed, which adopts NMS algorithm based on layered decoding. The decoder can support multi-rate QC-LDPC codes without additional hardware. Two adjacent sub-matrices are regarded as a basic unit and implemented simultaneously. QC-LDPC codes of different rates share the same storage and hardware to enhance utilization of hardware. Additionally, the VNU of current layer and the CNU of next layer are implemented simultaneously to improve the throughput. Furthermore, indirect storage reduces the required storage space. Finally, a QC-LDPC decoder of 2304 bits is designed and the maximum throughput can achieve 318 Mbps at 145 MHz and 15 iterations on SMIC 0. 18 /-lm CMOS technology. Therefore, the proposed decoder has a great potential and is suitable for modem digital communication systems.
