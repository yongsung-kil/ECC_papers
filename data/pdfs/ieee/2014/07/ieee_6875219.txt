B. Parameter Optimization of the Precoder
The precoding ratio α is one of the parameters which have important influences on the water-fall performance of the precoded Kite codes. We now optimize α by analyzing the convergence behavior of precoded Kite codes under iterative decoding based on the EXIT chart. To facilitate the EXIT analysis, the two component codes at the left side of the interleaver in Fig. 2, i.e., the accumulator and repetition codes (denoted as Acc-Rep for short) and repetition codes (denoted as Rep ), can be viewed as a mixture code. Then a precoded Kite code can be analyzed using a two dimensional EXIT chart, where one EXIT curve is for the mixture code and the other is for the single parity-check (SPC) code concatenated with an accumulator (denoted as SPC-ACC). Given an ensemble of RC-Kite codes, we can adjust the precoding ratio α to make two EXIT curves fitting for optimal design of the codes. This can be easily done by a brute force search because it is a one-dimensional optimization problem.
In the following, we will explain the EXIT functions in detail. Consider the AWGN channel with BPSK signaling. Assume that the signal-to-noise ratio is given by Eb/N0, and the code rate is R.
