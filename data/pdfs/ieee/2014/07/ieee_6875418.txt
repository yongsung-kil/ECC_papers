2014 IEEE International Symposium on Information Theory

On the Error-Correcting Capabilities of Low-Complexity Decoded Irregular LDPC Codes
Pavel Rybin Inst. for Information Transmission Problems
Russian Academy of Science Moscow, Russia
Email: prybin@iitp.ru

Abstract—This paper deals with the irregular binary lowdensity parity-check (LDPC) codes with the constituent single parity check (SPC) codes and the error-correcting iterative lowcomplex decoding algorithm. The lower bound on the error fraction, guaranteed corrected by the considered iterative algorithm, was obtained for the irregular LDPC code for the first time in this paper. This lower bound was obtained as a result of analysis of Tanner graph representation of irregular LDPC code. The number of decoding iterations, required to correct the errors, is a logarithmic function of the code length. The numerical results, obtained at the end of the paper for proposed lower bound achieved similar results for the previously known best lowerbounds for regular LDPC codes and were represented for the first time for the irregular LDPC codes.
I. INTRODUCTION
The error correcting capabilities of Gallager’s low-density parity-check (LDPC) [1] for a binary-symmetric channel (BSC) were studied in [2], where it was shown that such Gallager’s LDPC code exists that capable of correcting a linear portion of errors, with decoding complexity O(n log n), where n – LDPC code length. Then in [3] a new analytical easier to compute lower-bound on error correcting capabilities of Gallager’s LDPC code decoder was obtained. But the numerical results in most cases don’t exceed the numerical results, obtained by old lower-bound from [2]. It should be noted, that the decoding algorithm, considered in [3], differs from the algorithm, described in [2]. In this paper we consider the algorithm similar to the algorithm from paper [2].
LDPC code with constituent Hamming code (H-LDPC code) was considered in [4]. Then the code distance and the “soft” decoding of H-LDPC code was investigated in [5], [6]. It was shown in [7], that ensemble of H-LDPC codes contain such codes, that the minimal code distance almost reaches Varshamov – Gilbert bound. Then the result similar to the result from [2] for the first time was obtained for H-LDPC code in [8] by generalization of the methods developed in [2]. After that, using generalized methods from [8] similar results were obtained for q-ary LDPC code while transmitting over qary symmetric channel in [9] and for generalized LDPC codes while transmitting over the binary erasure channel in [10]. The authors of work [11] obtained a novel lower-bound on the fraction of guaranteed corrected errors for the binary codes on graph while transmitting over BSC. The numerical results of the lower-bound [11] for H-LDPC codes exceed similar results

of the lower bound from [8]. Then in [12], [13] a new lower bound for generalized regular LDPC codes was obtained. It was shown that numerical results of the lower bound [12], [13] for H-LDPC codes exceed similar results of the previously known best lower-bounds on the error correcting capabilities of H-LDPC codes [11].
This work was inspired partly by [14] and partly by [13]. In this paper we consider the irregular LDPC codes and iterative low-complex hard-decision decoding algorithms similar to the algorithm from [2], [13]. We obtained a new lower-bound on fraction of guaranteed corrected errors generalizing and combining the methods, developed in [13], [14]. Numerical computation for various choices of LDPC code shows that the proposed lower-bound achieves similar results for the previously known best lower bounds for regular LDPC codes [13]. The numerical results for the irregular LDPC codes are represented for the first time.

III. DECODING ALGORITHM
A. Decoding algorithm description
The decoding algorithm is similar to the algorithm from [2] and [13]. The main idea is that the weight of the syndrome decreases on each iteration of the decoding algorithm. On each iteration we invert one symbol after another that corresponds to the variable node with left degree l that is connected with more than l/2 check nodes corresponding to unsatisfied paritychecks (SPC). It is clear that inverting such symbols reduce the syndrome weight, because more than l/2 constituent (SPC) code becomes satisfied.

Consider iterative hard-decision decoding algorithm A, which decoding iteration i, i = 1, 2, ..., imax, consists of the two following steps:
1) consider the symbols of tentative sequence r(i), where r(1) is received sequence r, one after another:
• determine the number of unsatisfied parity-checks corresponding to check nodes connected to variable node corresponding to the considering symbol;
• invert the current symbol if the variable node with the degree l, corresponding to this symbol, is connected with more than l/2 check nodes corresponding to unsatisfied parity-checks;
• proceed to the next symbol of the current sequence; 2) consider the updated sequence r(i) obtained on the
previous step:
• if the syndrome of LDPC code for the updated sequence becomes zero (i.e. there is no constituent code with unsatisfied checks), the algorithm returns the updated (corrected) sequence r(i), sets a successful decoding flag and stops execution;
• otherwise if the syndrome weight for the updated sequence decreases, the algorithm proceeds to the next iteration i + 1 with the sequence r(i+1) that is equal to the updated sequence r(i);
• otherwise the algorithm sets a decoding failure flag and stops execution.
Remark 1: It is important to note that according to the description of the algorithm A new errors can be added to the tentative sequence while decoding. So, the algorithm A doesn’t guarantee that only erroneous symbols are corrected, new ones may be added.
