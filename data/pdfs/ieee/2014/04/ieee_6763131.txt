Deterministic Girth-Eight QC-LDPC Codes with Large Column Weight
Jianhua Zhang and Guohua Zhang

Abstract—For any row weight L, several novel classes of (J = 5, L) and (J = 6, L) quasi-cyclic LDPC codes are deterministically constructed with girth eight. From these results, it is proved that (5, L) QC-LDPC codes with girth eight exist for any circulant permutation matrix (CPM) size P ≥ (2L + 3)(L − 1) + 1, and that girth-eight (6, L) QC-LDPC codes exist for any P ≥ 2(L + 5)(L − 1) + 1. The two novel bounds remarkably improve the existing bounds of L2(L − 1) + 1 and (L2 + 1)(L − 1) + 1, respectively. Moreover, for any column weight J and any row weight L, a construction for (J, L) QCLDPC codes with girth eight is also proposed. This is the first deterministic and systematic construction which can generate girth-eight QC-LDPC codes with J ≥ 7.
Index Terms—Circulant permutation matrix (CPM), girth, low-density parity-check (LDPC) codes, quasi-cyclic (QC) linear block codes.
I. INTRODUCTION
A (J, L) quasi-cyclic (QC) low-density parity-check (LDPC) code is defined by the null space of a paritycheck matrix (PCM) which is a J × L array of circulant permutation matrices (CPMs) of the same size, namely, P ×P . The girth of the Tanner graph of an LDPC code is defined as the length of the shortest cycles in the graph. Since QCLDPC codes with large girth have low encoding complexity and may outperform random LDPC codes for moderate block lengths, they have recently attracted much attention [1-12]. The construction of QC-LDPC codes can be classified into two categories: random methods [3,5,8,9] and explicit methods [2,4,6,10]. While random methods are generally based on combination of some algebraic techniques and computer search methods, explicit methods are deterministic in the sense that no computer search is required. The explicit methods are particularly preferred in some applications (such as the CRT method [6,7]) where a shorter code with large girth is required to construct a larger target code, with nondecreasing girth.
For the trivial case J = 2, it is obvious that (2, L) QCLDPC codes with girth 8 can be constructed for any CPM size P ≥ L. For J = 3, (3, L) QC-LDPC codes with girth 8 have been constructed for any P ≥ L{L+[L (mod 2)]}/2+1 [4]. For J = 4, (4, L) QC-LDPC codes with girth 8 can be constructed for any P ≥ 3L2/4 + L − 1 [6]. For J = 5 and J = 6, the existing best results are that (J, L) QC-LDPC codes with girth 8 can be constructed for any P ≥ L2(L − 1) + 1 and any P ≥ (L2 + 1)(L − 1) + 1, respectively [2].

In this letter, several novel classes of QC-LDPC codes are explicitly constructed, showing that for J = 5 and J = 6, (J, L) QC-LDPC codes with girth 8 exist for any P ≥ (2L + 3)(L − 1) + 1 and any P ≥ 2(L + 5)(L − 1) + 1, respectively. In addition, a construction for girth-8 (J, L) QC-LDPC codes is proposed for any J and any L.

B. Application of the new codes
The new girth-8 codes can not only provide some tighter lower bounds in respect of the CPM sizes for QC-LDPC codes with girth at least eight, but also serve as a good starting point or a basic module in some general skills, such as [3,7,11,12]. To be specific, the new codes can be used as component matrices to construct other types of QC-LDPC codes with girth at least eight, by the CRT method [6,7] or the row-weightextension method [11]; the new codes with L = J can be directly used to construct girth-16 (2, L) QC-LDPC codes by the method in [12]; and just like [3], the so-called type-1 QCLDPC codes with girth at least eight can be easily generated by combining the new codes and some well-designed masking matrices.
In what follows, we give an example which illustrates the application of the new codes combined with masking matrices.

It should be pointed out that Construction 5 is the first deterministic and systematic method which can generate girth8 QC-LDPC codes with J ≥ 7.

VI. CONCLUSION
For column weight J = 5, 6 and an arbitrary row weight L, we have explicitly proposed several types of girth-8 (J, L) QC-LDPC codes and constructively proved that girth-8 (5, L) and (6, L) QC-LDPC codes exist for any CPM size P ≥ (2L+ 3)(L − 1) + 1 and any P ≥ 2(L + 5)(L − 1) + 1, respectively. The two lower bounds are much tighter than the up-to-date bounds presented in [2]. In addition, an explicit construction for girth-8 (J, L) QC-LDPC codes is also proposed for any J and any L.
