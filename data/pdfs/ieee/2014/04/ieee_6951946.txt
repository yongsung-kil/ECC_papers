Abstract—The construction and performance of full diversity Root-LDPC codes for block fading channels are the hot spot at present. There are two important factors, the girth and the cycles connectivity, can affect the frame error rate performance of Root-LDPC codes. The original PEG algorithm is an effective method to provide longer girth of cycles, but ignores the cycles connectivity. The contribution of this paper is to present two algorithms to design Root-LDPC codes for block fading channels. The first algorithm is the modified PEG algorithm (MPEG). We apply the Approximated Cycle Extrinsic Message Degree (ACE) metric, which is used to measure the cycles connectivity, to modify the traditional PEG algorithm to ensure having lower frame error rate. However, MPEG algorithm is one of the greedy algorithms. With the increase of code length, the encoding/decoding complexity of MPEG algorithm will enhance. Meanwhile the QC algorithm has low complexity because of its quasi cyclic feature. Then we combine the modified PEG algorithm with QC algorithm, which guarantees not only the code performance but also the quasi cyclic feature. The novel algorithm achieves a tradeoff between the frame error rate performance and the encoding/decoding complexity, so that it is more suitable for the practical application. Simulation results demonstrate that the algorithms we proposed can achieve full diversity and make the average required number of iterations decrease from medium to high SNR, suitable to different fading blocks.
Keywords—Cooperative Root Low-Density Parity-Check (RootLDPC); Block fading channels; PEG; ACE; QC
I. INTRODUCTION
The block fading channel model was first proposed in [1]. Due to the fact that multi-path propagation and mobility exist in many wireless systems, information transmission over the wireless channels is confronted with a significant challenge. The block fading channels model slowly varying fading, where a codeword span a fixed number of F of fading degrees of freedom. In such non-ergodic scenarios, the channel capacity is zero since there is an irreducible probability, is called outage probability [2]. The block-fading channel is a simple and useful model that captures the essential characteristics of nonergodic channels [3]. Codes designed for block fading channels are expected to achieve the maximum diversity order and that the channel provides good code gain. Recently, code design for block fading channels has been widely studied [3]-[6].
Boutros et al. proposed a family of full diversity low density parity check (LDPC) called Root-LDPC codes for nonergodic block fading channels in [7]. The Root-LDPC codes

can achieve full diversity and perform close to the outage limit when decoded using the iterative belief-propagation (BP) decoding algorithm on block fading channels. Yueqian Li et al. proposed the structured Root-LDPC codes by average of tilling circulant matrices and designed Quasi-Cyclic LDPC (QC-LDPC) codes [8]. The generator matrices of QC-LDPC codes are in systematic (SC) form with the requirement that the parity check matrices are full rank. The memory cost for encoding of QC-LDPC codes is greatly reduced and the encoder can be implemented using simple shift registers. The QC-LDPC codes perform similarly to the Root-LDPC design by Boutros.
In order to improve the girth of the Root-LDPC codes, the PEG Root-LDPC codes [2], which are designed in PEG based technique have been developed. The Root-LDPC codes generated by PEG algorithm can save up to 0.5dB in terms of signal to noise ratio (SNR) to achieve the same FER or BER compared to the previous algorithms.
Tian et al. proposed that not only the girth of cycles, but also the connectivity, of the cycles plays an important role on the performance [9]. The metric of Approximate Cycle Extrinsic message degree (ACE) is used to describe the connectivity of cycles, which is identified as a simple and effective connection between the cycles and the stopping sets. The better the cycles connectivity is, the better the code performs. Not all short cycles are equally harmful [9]. Some longer cycles with poor graph connectivity are more harmful than shorter cycles with good graph connectivity. This paper first presents an improved PEG algorithm (MPEG) with Approximate Cycle Extrinsic message degree (ACE) metric for block-fading channels. The approach can maximize the girth of the cycles, improve the cycles connectivity.
However, MPEG algorithm is one of the greedy algorithms which have the high complexity. Another contribution of this paper is to present a MPEG-based algorithm to design QCLDPC codes with Root-Check properties for block-fading channels. The proposed algorithm combines the MPEG algorithm with QC algorithm, which guarantees not only the performance but also the quasi cycle feature. Simulation results demonstrate that the algorithm we proposed can achieve full diversity and significantly better code performance, suitable to different fading blocks. From medium to high SNR, the average required number of iterations can be decreased.

III. MODIFIED PEG ALGORITHM (MPEG)
In this section, we present a method to design Root-LDPC codes based on modified PEG algorithm for block fading channels. The resulting structured MPEG Root-LDPC codes can achieve full diversity and significantly better code performance. PEG algorithm is a construction method to maximum the girth of cycles. Although the algorithm guarantees the girth of cycle maximum, the cycles connectivity is ignored. Tian et

Tian et al. defined a stopping set, Extrinsic Message Degree (EMD) and Approximate Cycle EMD (ACE) to measure the cycles connectivity.

Definition 1 (A stopping set): A variable-node set is called a stopping set if all its neighbors are connected to this set at least twice.

Definition 2 (EMD): An extrinsic constraint node of a variable-node set is constraint node that is singly connected to this set. The EMD of a variable-node is the number of extrinsic constraint nodes of this set.

In general, the EMD of this cycle is calculated by i(di − 2),where di is the degree of the ith variable in the cycle when there are no variable nodes in a cycle that share common constraint nodes outside of the cycle. Otherwise, the EMD is reduced through constraint node sharing. For easy of calculation, we neglect constraint node sharing and define ACE.

Definition 3 (ACE): The ACE of a length 2d cycle is i(di − 2), where di is the degree of the ith variable in this cycle. We also say that the ACE of a degree-d variable node is d-2, and the ACE of any constraint node is zero.

VI. CONCLUSION
Two novel algorithms have been proposed to design RootLDPC codes for block fading channels. The MPEG algorithm is presented to enhance the coding gain. PEG algorithm progresses on an edge-by-edge basic. Edge placements are such that when a cycle is created, that cycle is of maximum possible length. But the PEG algorithm does not considerate the cycles connectivity. We use the ACE metric to modify the traditional PEG algorithm to ensure having better performance. Then a MPEG-based algorithm to design QC-LDPC codes with Root-Check properties for block-fading channels is proposed, which guarantees not only the performance but also the quasi cycle feature. Based on simulations, the proposed method was compared to MPEG Root- LDPC and QC Root-LDPC codes. The results demonstrate that the MPEG-QC Root- LDPC codes generated by our proposed algorithm outperform in a wide range of SNR values.
