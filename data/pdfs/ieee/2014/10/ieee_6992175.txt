2014 Sixth International Conference on Wireless Communications and Signal Processing (WCSP)
Lower Error Floor of LDPC Codes Based on Trapping Sets Elimination

Le Dong, Ziyu Zhao, Jing Lei, Er-bao Li
College of Electronic Science and Engineering National University of Defense Technology Changsha, China dongle818@sina.com

Abstract—LDPC codes with excellent performance have great potential in applications of wireless mobile communications. However, the error floor phenomenon existing in LDPC codes limits its further application in disk storage and deep-space communications which require very low bit error rate (BER) system. The researchers find that trapping sets are the major reason why the LDPC codes have error floor. In this paper, we proposed an elimination algorithm to reduce small trapping sets. It was applying node rearrangement method to optimize the neighborhood of target nodes. In this way, cycles will be broken and the small size of trapping sets can be eliminated. The experimental results show that it can lower the error floor to improve the performance of LDPC codes. Besides, we also can keep the degree distribution, code rates and girth features of the original code with this method.
Keywords—Low-density parity check(LDPC) codes; trapping sets; elimination algorithm; structure optimization; error floor
I. INTRODUCTION
Since LDPC codes were rediscovered and regarded to be the "good" code which close to Shannon capacity, the researches about structure of LDPC codes, encoding and decoding methods, performance analysis and appliance in modern communication systems have become the hot topic. However, it was found that many LDPC codes with middle length have the error floor phenomenon, which means the original BER curve suddenly become flat when it reached to a high SNR region. This limits the further application of LDPC codes in the communication systems with the requirement of very low BER, such as application in disk storage and deepspace communications.
Richardson [1] studied the error floor of LDPC codes in AWGN channel. He pointed out that the decoding error of iterative decoder on the error floor region mainly caused by the topology of nodes sets which called trapping sets in check matrix. It have impact on starting point and the height of the error floor. Thus, trapping sets become the key point to solve the problem of LDPC error floor. An effective way to reduce the error floor is to eliminate and avoid the trapping sets producing. This method include constructing the LDPC codes with good topology and improving the decoding algorithm.
As in [2], the method they proposed consists of taking two (or more) copies of the same code and swapping edges between the code copies in such a way that the most dominant trapping sets are broken, so the structure of the LDPC code will have a

good topology. Since cycles are generally exist in topology of trapping sets, cyclic liftings are propose to reduce the error floor of LDPC codes by optimizing the structure of cycles in the Tanner graph in [3]. In [4], the paper shows that we can add new check node, and connect it with the same variable node in adjoining trapping sets, thus, the multiple trapping sets can be simultaneously eliminated. In general, the advantage of this graph structure optimization method is improving performance only with the cost of increasing code length, and it can be applied to any codes, any channel models, any decoding algorithms. It provides a way to lower the error floor by restraining the generation of error trapping sets. But the disadvantage is that it will change the original parameter of codes and increase the complexity of code design.
So in this paper, we will find a way to optimize the performance of LDPC codes without changing the parameter of codes. We present a trapping sets elimination algorithm based on node rearrangement with the premise of ensuring the same original code rate and degree distribution. By replacing the neighbor node of variable node, we can achieve the purpose of ‘broking cycle’ which eliminates trapping sets, then the error floor become lower and the performance of LDPC codes is improved.
The remainder of this paper is organized as follows. In Section II, We make a brief description of the trapping sets, and analyze the relationship between cycles and trapping sets, then we briefly discuss their influence on decoding performance. Section III shows the algorithm how to optimize the code structure which include the identification algorithm and elimination algorithm of trapping sets based on node rearrangement. In Section IV, the proposed elimination algorithm using two kinds of different LDPC codes and BER performances are illustrated in this Section. Finally, Section V concludes the paper.

III. THE CONSTRUCTION ALGORITHM OF LDPC CODES BASED ON
TRAPPING SETS ELIMINATION
Due to the impact of special structure of trapping sets on the result of decoding, we put forward the proposal on the elimination of trapping sets mainly from the perspective of the structure of the LDPC codes. Optimization algorithm proposed in this paper contains two steps, the identification of trapping sets and trapping sets elimination algorithm which is based on node rearrangement.
A. The Identification of Trapping Sets
The search methods of trapping sets are divided into two categories: the search method based on iterative decoding process [11] and graph search method [12]. This paper uses the graph search method to determine the distribution of the trapping sets, which has serious influence on decoding performance. The basic idea is to search and identify a set of variable nodes with certain structural features in the overlapping of different length of a short cycle, which is based on the structural relationship between trapping sets and cycles.
Graph search method is different from the search method which is based on the iterative decoding process. Its identification result is not the trapping set which combined with a specific channel conditions, but those special structure was hidden in the Tanner graph. Although these trapping sets does not necessarily reflect in the final decoding results, they are all potential factors that will cause the failure of decoding. Under the influence of noise, it is likely to become a major factor from the underlying causes of decoding error factor. Therefore, graph search method is applied to determine the comprehensive trapping sets, it can find out all subgraph that meet structural characteristics for a specific trapping set.

V. CONCLUTION
This paper presents a new method to lower the error floor of LDPC codes. This method focuses on analyzing the reasons for the failure of the trapping sets, which is based on the important characteristics of this special structure of trapping sets, and optimizing the design from the perspective of code, then presenting a kind of new node-based rearrangement algorithm to eliminate trapping sets, with the premise of maintaining the original code rate and degree distribution the same, by replacing the variable trapping neighbor node in order to achieve the purpose of ‘broken cycle’, thereby improving the structure connectivity of trapping sets’ graph, to achieve the elimination of trapping sets. Through the simulation result, it is shown that the LDPC code which was optimized by the proposed algorithm can achieve about 0.5 dB coding gain in high SNR than the original code, then the existing error floor region has been mitigated effectively.
