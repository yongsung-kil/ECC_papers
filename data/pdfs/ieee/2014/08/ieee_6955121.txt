An Efficient Complexity-Optimizing LDPC Code Design for the Binary Erasure Channel

‚ÄÝVahid Jamali, ‚Ä°Yasser Karimian, ‚ÄÝJohannes Huber, ‚Ä°Mahmoud Ahmadian, ‚ÄÝFriedrich-Alexander-University Erlangen-N√ºrnberg (FAU), Erlangen, Germany
‚Ä°K. N. Toosi University of Technology (KNTU), Tehran, Iran

Abstract‚ÄîThe complexity-performance trade-off is a fundamental aspect of the design of low-density parity-check (LDPC) codes. In this paper, we consider LDPC codes for the binary erasure channel (BEC), use code rate for performance metric, and number of decoding iterations to achieve a certain residual erasure probability for complexity metric. The available complexity-optimizing problems in the literature for the BEC are either non-convex or belong to the class of semi-infinite problems which are computationally challenging to be solved. Hence, in this paper, we first propose a lower bound on the number of iterations for the BEC. Moreover, a simple but efficient utility function corresponding to the number of iterations is developed. Using this utility function, an optimization problem w.r.t. complexity is formulated to find complexity-optimized code degree distributions. We prove that the considered problem with the proposed utility function falls into the class of semi-definite programming (SDP) and thus, the global solution can be found efficiently using available SDP solvers. Numerical results reveal the superiority of the proposed code design compared to existing code designs from literature.
I. INTRODUCTION
A significant research effort has been dedicated in the past decade to designing low-density parity-check (LDPC) codes which maximize the code rate as the performance metric [1], [2]. These codes are referred to as performance-optimized codes. However, for a performance-optimized code, convergence of the decoder usually requires a large number of iterations. This leads to high decoding complexity and processing delay which is not appropriate for many practical applications. In contrast to performance-optimized codes, codes which are designed to minimize the decoding complexity for a given code rate are denoted by complexity-optimized codes [3]‚Äì[5].
Unfortunately, characterizing the number of decoding iterations as a function of code parameters is not straightforward. However, for the binary erasure channel (BEC), several bounds and approximations for the number of decoding iterations have been proposed in literature [3]‚Äì[5]. Simple lower bounds on the number of iterations required for successful messagepassing decoding over BEC were proposed in [3]. In [4], an approximation of the number of iterations was proposed and used to formulate an optimization problem for complexity minimization. However, the resulting optimization problem is non-convex and is solved iteratively, only. A similar approach to design a complexity-optimized code was investigated in [5]. We note that one of the essential constraints that has to be considered for LDPC code design is to guarantee that the residual erasure probability decreases after each decoding

iteration. In general, this successful decoding constraint for the BEC leads to an optimization problem which belongs to a class of semi-infinite programming, i.e., the problem has infinite number of constraints [6], [7].
In order to avoid the computational complexity of the code design in the previous works, we first propose in this paper a lower bound on the number of decoding iterations based on a quite accurate approximation of the number of iterations in [8]. The insights offered by this lower bound motivate us to develop a simple, but efficient utility function corresponding to the number of decoding iterations. By this, an optimization problem is formulated such that for any given check degree distribution and code rate, a variable degree distribution with a finite maximum variable node degree is found by which the proposed utility function corresponding to the number of decoding iterations to achieve a certain target residual erasure probability is optimized. We show that by applying this utility function, the optimization problem is convex but still falls into the class of semi-infinite programming. However, using the theory of semi-definite programming (SDP), we prove that the considered problem has also an SDP representation. Hence, the global solution of the considered optimization problem can be found efficiently using available SDP solvers such as CVX [9]. Numerical results reveal the effectiveness of the proposed code design compared to existing code designs from literature.

V. CONCLUSION
The design of complexity-optimized LDPC codes for the BEC is considered in this paper. To this end, a utility function corresponding to the number of iterations is developed. Using this utility function, a complexity-optimizing problem is formulated to find a variable degree distribution with a finite maximum variable node degree for a given check degree distribution and code rate. It is shown that the optimization problem has an SDP representation and thus can be solved much more efficiently compared to available semi-infinite problems. Numerical results confirmed the effectiveness of the proposed methods for code design.
