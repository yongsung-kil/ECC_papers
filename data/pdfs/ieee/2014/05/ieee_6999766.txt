Abstract— In this paper, a new fast convergence hybrid decoding algorithm based on multi-dimensional searhing is proposed for decoding LDPC codes. The main idea of this algorithm is flipping variable multi bits in each iteration, change in which leads to the syndrome vector with least hamming weight. To this end, the algorithm does multidimensional searching between all possible bit positions that could flip in each iteration to select the best choices. Simulation results illustrate that the proposed algorithm converge significantly faster and have reduction in iteration number and also have excellent performance but with little performance difference than the robust SumProduct algorithm.
Keywords- LDPC codes; hybrid decoding; multi-dimensional searching; finite geometries
I. INTRODUCTION
Low-density parity-check (LDPC) codes, first introduced by Gallager in 1962 [1] and then rediscovered in late 1990’s [2,3]. LDPC codes have recently received tremendous attentions because these can achieve near Shannon limit performance over the binary symmetric channel as well as the additive white Gaussian noise (AWGN) channel [3]. Ever since their rediscovery, a great deal of research effort has been expended in design, construction, structural analysis, efficient encoding and decoding, performance analysis, decoder architecture and implementation, generalizations and applications of LDPC codes. Numerous papers have been published on these subjects. Many LDPC codes have been chosen as the standard codes for various next-generation communication systems and they are appearing in recent data storage products.
The rapid dominance of LDPC codes in applications requiring error control coding is due to their capacity approaching performance which can be achieved with practically implementable iterative decoding algorithms. The

LDPC codes possess many advantages including parallelizable decoding, coherent interleaver, low error-floor, and lower decoding complexity compared with turbo codes but the greatest advantage of these codes, regardless of regular or irregular types, is their relatively simple and efficient decoding algorithm with parallel processing that allows the resulted message (extrinsic information) passing between the associated variable and check nodes in a graph simultaneously [12].

There have been wide variety methods for decoding of LDPC codes, ranging from low to high complexity and from reasonably good to very good performance. These decoding methods may be classified into three categories: hard-decision, soft-decision and hybrid decoding schemes. These decoding algorithms include: OneStep Majority-Logic (OSMLG) decoding [14,15], Gallager’s Bit Flipping (BF) decoding [1], Weighted MLG decoding (WMLG)[16], Weighted BF decoding (WBF)[17], and other modified versions of WBF algorithm like Improved Weighted Bit Flipping (IWBF)[18], Modified Weighted BitFlipping (MWBF)[19] and Improved Variable Multi Weighted Bit Flipping (IVMWBF) [25], A Posteriori Probability (APP) decoding[1], iterative decoding based on Belief Propagation (IDBP) (commonly known as SumProduct algorithm (SPA))[20], and approximations of the IDBP like Min-Sum[21], Normalized / Offset-BP [22-24] algorithms. OSMLG and BF decoding belong to harddecision decoding methods. APP, IDBP/SPA decoding and its approximations are soft-decision schemes. WBF, IWBF, Modified WBF and IVMWBF algorithms are hybrid decoding methods.
Each iterative decoding algorithm provides a distinct trade-off between complexity and performance. It goes without saying that the soft-decision algorithms in which message passing is used to carry out their decoding procedures can achieve better performance compared with those based on hard-decision. However, due to their higher complexity in terms of calculations and the hardware implementation of the relevant decoders, the use of these algorithms is unfeasible. The performance of hard-decision algorithms is approximately 3dB lower than that of softdecision algorithms. Hybrid decoding algorithms which are in fact based on both soft and hard decisions have been developed to bridge the performance gap between these two groups of algorithms (those based on soft and hard decisions). Results obtained from analytical simulations and studies indicate that there is still a wide performance gap between hybrid algorithms and those based on soft decision.
The present work is an effort to propose a new fastconvergence high-performance low-complexity hybrid decoding algorithm. In order to achieve this, we shall first

investigate original version of algorithm based on 1dimensional searching, which while having the capacity for flipping one bit per iteration. Then, in order to faster convergence, lower complexity and reduction in iteration number of original version, we shall propose our hybrid algorithm based on multi-dimensional searching that is new and highly efficient algorithm with an acceptable performance compared with the Sum-Product algorithm and less complexity and fewer iterations required. The remainder of the paper is organized as follows: Section II presents notation and basic definitions. Section III introduces the decoding algorithms based on searching and section IV provides some simulation results to illustrate the performance of the proposed algorithm. Finally, Section V concludes this paper.

B. The Proposed Algorithm Based on Multi-Dimensional Searching
Our algorithm established upon multi-dimensional searching is proposed based on the idea that in the case of LDPC matrices, generally the syndrome weight increases with the number of errors until error weights much larger than half the minimum distance. This algorithm flips P (k)
bit(s) in kth iteration and these bits are selected in such a way that reduces the weight of the syndrome vector. Obviously, the number of bits must be flipped in each iteration is different. Furthermore, its value in each iteration decreases toward the previous iteration.

Recently, we have proposed IVMWBF algorithm and inside it utilized a new equation for flipping multi variable bits in each iteration. For conciseness, we refer readers to [25] for details. Here, we intend to expand a new hybrid decoding algorithm based on multi-dimensional searching to find a case that will lead to the best answer by the use of that equation for determining the number of the bits to be flipped in each iteration.

B. Choosing the Optimal Value α
Accurate studies on simulation results for different LDPC codes have shown that the optimal weighting factor
of α depends on the SNR and the code structure. However, choosing the α at BER = 10−5 and keeping constant it for optimal value of α for all values of SNRs
does not cause significant degradation in performance.
Therefore, the optimal weighting factor α can be
considered to be only code-dependent.

V. CONCLUSION In the present work, we have introduced a new hybrid iterative decoding algorithm based on Multi-Dimensional searching for regular LDPC codes. The proposed algorithm uses a new equation to determine the number of bits to be flipped in each iteration. This equation presented based on the observation that there is a strong correlation between the number of bits to be flipped and the hamming weight of the syndrome vector in each iteration. Our simulation results substantiate that compared to the Existing algorithm based on 1-dimensional searching, the proposed algorithm converge very much faster and outperform it. Furthermore, for different types of LDPC codes it was shown that the proposed algorithm perform very close to Sum-Product decoding, but with faster convergence rate and very less iteration number.
