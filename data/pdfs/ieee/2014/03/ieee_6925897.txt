*Department of Telecommunications Engineering, KMITL, Bangkok, Thailand, kpnattak@kmitl.ac.th '''Department of Computer Engineering, Thammasat University, Pathumthani, Thailand, kaewprapha@gmail.com

Abstract-In data communications system including wired and wireless transmissions, the reliability, integrity, speed, and cost efficiency are the main issues that people desire. An error correcting code is one of the efficient and nearly-optimal methods to improve the system performance. Based on the idea of constructing MDS codes called CGR codes in [3] for disk arrays, this paper introduces a new perspective and construction of LDPC codes due to its sparsity and advantage complexity in implementation. The performance of this code in Additive White Gaussian Noise (AWGN) and fading channels gains about 1.0 dB at the 10-3 of bit error rate (BER) compared with an EVENODD code.
Index Terms-maximum distance separable (MDS) codes, array codes, low-density parity-check (LDPC) codes
I. INTRODUCTION
Since a huge amount of data transferred via communications systems and also stored in storages everyday, data storages continue to be an important media for massive storage, inÂ­ cluding near-line storage and backup. It is a rather mature technology, and provides the storage volume, speed, reliability and cost people desire. However, hard disks are not faultÂ­ free. Both latent errors and salient errors may occur lead to disk failures. The latent errors can be protected by a driver's internal error control mechanisms, while the salient errors cannot be detected or repaired by the disk drive itself and may cause more insidious disk errors.
The key challenge of disk drives is not only a high caÂ­ pacity, but also fault tolerance and capability to recover disk failures to reliable protect all stored information. To achieve higher storage capacity, disk arrays is the solution where a lot of cheap disks are grouped/stacked together in order to redundant all stored information. The RAID (redundant arrays of independent/inexpensive disk) technology is an example of this solution. Instead of using a high performance disk which is costly, it uses an array of inexpensive disks while still maintaining the same capability to protect disk failures. Generally, there are two ways to introduce redundancy in order to prevent data loss due to disk errors/ failures: Replication and error/erasure coding. Replication is simple, but wasteful in space. Erasure codes can provide better space efficiency,

Fault tolerance (or disk failures) is usually protected/ recovered by coding techniques such as maximum distance separable (MDS) code, low-density parity check (LDPC) codes, or parity array codes. Various techniques for increasing storage reliability have been actively exploited, including powerful error correction codes applied inside each block/sector of a disk to protect against bit errors or bit loss, and, more recently, efficient erasure codes applied between disks (or blocks and sectors) to protect against a disk (block/sector) failure [2]- [4].
In case of LDPC codes, the limitation of applying LDPC codes with disk arrays is that it is an asymptotically code and requires a long code length to achieve a good performance. LDPC codes, likewise, are well-known as the good randomly codes with their performance is closed to Shannon bound. However, to construct LDPC codes for disk arrays in order to recover disk failures is still not easy to achieve at a low bit error rate (BER) since the short LDPC code does not give a good performance, while disk arrays require short code length for the sake of speed and capacity limit. Tornado code [5] is a class of irregular LDPC codes applied for fault tolerant storages with a simple linear time decoding algorithm, but still need a long code length.

In this work, we will introduce a new perspective and construction of low-density parity check (LDPC) codes for erasure channels due to its sparsity and advantage complexity in implementation based on the structure of CGR codes in [3]. Not only for the disk arrays, but also for the wireless communications systems, we will indicate the relation of CGR codes and LDPC codes that can achieve MDS property which reaches space optimality. Then, this code will be described as low density code since it has the smallest possible updated complexity for its parameters [6], also we will describe its performance compared to other MDS codes such as repetition codes and EV ENODD codes in Additive White Gaussian Noise (AWGN) and fading channels.

V. CONCLUSION
This paper introduces the performance of LDPC codes which are converting from an MDS array code for disk storages in order to tolerate/protect disk failures for the sake of general applications. Not only for the disk arrays, this code can also have another rigid structure for any erasure channels. This MDS-LDPC code shows the relation between MDS array code and LDPC code in terms of their structures and performances. Applying the advantage of decoding algorithm of LDPC code, the performance shows that the MDS-LDPC code outperforms the repetition code and EvenOdd code at the same rate of bit error for a low rate of SNR. Its SNR gain is approximately 1 dB at the 10-3 of BER.
