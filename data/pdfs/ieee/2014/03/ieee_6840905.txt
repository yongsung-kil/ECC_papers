Abstract—Shuffled decoding is known to provide efficient and high-throughput implementation of LDPC decoders. In this paper, a new grouping technique of variable nodes to accelerate the message-passing rate is presented. In this algorithm, unlike other group-shuffled decoding methods, we consider for the first sub-group a set of variable nodes that has a low value of the intrinsic information, where the variable nodes of the code graph are divided into sub-groups called layers to perform group-bygroup message-passing decoding and each iteration is broken into multiple sub-iterations. This paper proposes a simplification of the Variable Node group-shuffled Belief Propagation (Optimized VN group-shuffled BP) to lower the complexity of the check node update rule. Simulation results, verify that the Optimized VN group-shuffled BP does yield a faster convergence rate than both Belief Propagation (BP) and Check Node group-shuffled Belief Propagation (CN group-shuffled BP) decoder.
Keywords—Shuffled decoding; CN group-shuffled BP; VN group-shuffled BP; Reduced Complexity decoding; Optimized VN group-shuffled BP; Correction factor.
I. INTRODUCTION
Low Density Parity Check (LDPC) codes, first introduced by R. Gallager [1] in the early 1960s, deliver very good performance when decoded with the belief-propagation (BP) [15] or the sum-product algorithm [2-4]. As LDPC codes are being considered for use in a wide range of applications, the search for efficient implementations of decoding algorithms is being pursued intensively. The BP algorithm, however, suffers from low convergence rate and high implementation complexity. To improve the rate of convergence and reduce implementation cost, the BP algorithm can be simplified using the so-called Min-Sum (BP-based) approximation [15]. But this simplification is made at the expense of a substantial loss in performance. In [11], an improvement is made to the Min-Sum (BP-based) algorithm by using a correction factor in the check node update rule. It is denoted by offset BPbased algorithm when the correction factor is subtracted to the minimum value, or normalized BP-based algorithm when it is multiplied by the correcting factor. Recently, several papers [5, 20] have investigated different types of scheduling strategies in BP LDPC decoding. With sequential scheduling, the messages are generated sequentially using the latest available information. Sequential scheduling was introduced as a sequence of check-node updates (CN updates) in [5,6] and a sequence of variable-node updates (VN updates) in [7,8]. It is also presented in [20] and [24] under the

name of Layered BP (L-BP), in [9] and [12] as serial schedule, in [13] as row message passing and column message passing and in [20] as serialized BP decoding. Group-Shuffled BP decoding algorithms partition either the variable nodes (VNs) [21] or the check nodes (CNs) [22] of the corresponding bipartite graph into small sub-groups called layers and each iteration is broken into multiple sub-iterations [24]. In each sub-iteration one layer of check nodes (resp. variable nodes) and their neighboring variable nodes (resp. check nodes) are processed [20]. In [20], an improvement is made to the Group-Shuffled BP decoding algorithm by using an efficient variable node layering strategy that significantly increases decoding convergence of LDPC codes as compared to both BP and CN Group-Shuffled BP algorithms. In this paper, we consider Group-Shuffled BP decoding and propose an Optimized VN Group-Shuffled BP algorithm for LDPC code which has better performance not only than BP algorithm but also than CN Group-Shuffled BP algorithm. The rest of the paper is organized as follows. Section II introduces the LDPC representation with bi-partite graph. Section III presents the iterative decoding algorithm. In section IV, a new decoding strategy for updating extrinsic information is proposed based on variable node layering. The section V focuses on simplifying the check-node update rule to obtain reducedcomplexity Variable-Node Layered BP (VL-BP) derivatives that achieve near-optimum performance. The simulation results are discussed in section VI, and finally, conclusions are drawn in section VII.
II. GROUP SHUFFLED BELIEF PROPAGATION DECODING
A. Basic definitions and notations
A binary (N, j, k) regular LDPC code is a linear block code of length N having a small fixed number ’j’ of ones in each column of the parity check matrix H, and a small fixed number ’k’ of ones in each rows of H. A sparse M × N parity-check matrix H can be viewed as a Tanner graph. In the Tanner graph the variable nodes are denoted by vn, corresponding to the encoded symbols and the parity-check nodes are denoted by cm corresponding to the parity checks. A variable node vn is connected to a check node cm if and only if H(m, n) has a non-zero entry. The Tanner graph representation of LDPC codes is very useful since their decoding algorithms can be explained by the exchange of information along the edges of these graphs. The notations related to the Tanner graph and an important hypothesis will be hereafter detailed.

B. Variable Node Group Shuffled Belief Propagation decoding
Group Shuffled Belief Propagation decoding is a modification of the Belief Propagation (BP) algorithm. It divides the Tanner graph of an LDPC code into smaller subgraphs, called layers [21]. In this paper, we focus on the Variable Node Group Shuffled Belief Propagation (VN Group Shuffled Belief Propagation) decoding algorithm. In this algorithm, each sub-graph consists of a set of variable nodes and all their neighboring check nodes. Each variable node appears in exactly one layer, while check nodes can appear in multiple layers. In each sub-iteration the check node and variable node updates are calculated in one layer. The decoding then progresses sequentially through layers by performing message updates sub-iteration by sub-iteration. A parity check test over the entire codeword is performed at the end of each sub-iteration. VN Group Shuffled Belief Propagation requires the processing of tow messages transmitted from nodes to nodes: T(n,m) and E(n,m). Let v = {v1, v2, ..., vN } denotes the set of all variable nodes and let c = {c1, c2, ..., cM } denotes the set of all check nodes. T(n,m) denotes the information which is sent by a variable node vn, n = 1, . . . , N to its connected check node cm, m = 1, . . . , M and E(n,m) denotes the information which is sent by a check node cm to its connected variable node vn [18].

III. OPTIMIZED VN GROUP SHUFFLED BP ALGORITHM
Our work is based on the improvement in error performance of group shuffled BP algorithm. We propose an approximation for the VN group shuffled BP algorithm based on the so-called min-sum approximation coupled with an additive offset term β used for each layer. Unlike other layered decoding methods, we consider, for each sub-group a set of variable nodes that has a low absolute value of the intrinsic information In for the bit vn. This technique increase the reliability of the decision of the bit vn. The advantage of the proposed method is that the optimization factor β reduce the performance loss and the complexity of decoding compared to min-sum approximation.

V. CONCLUSION
This paper discusses a solution to accelerate convergence of LDPC decoding algorithm. We propose a new simplification of Group Shuffled Belief propagation algorithm, for updating extrinsic information that finds good variable node layering. This technique increase the reliability of the decision of variable nodes which benefit from receiving updated-messages within a decoding iteration, accelerating the message-passing
