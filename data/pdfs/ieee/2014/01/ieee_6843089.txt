A New Soft Decoding Method for Systematic LT Codes
Meixiang Zhang and Sooyoung Kim Division of Electronics Eng. Chonbuk National University Jeonju, Korea
Email: mei1986xiang@live.cn, sookim@jbnu.ac.kr

AbstractвҖ”In this paper, we propose a method to construct the modified Tanner graphs for systematic Luby transform (LT) codes. The modified Tanner graph is drawn in terms of bit nodes and check nodes, instead of encoding nodes and information nodes as in the conventional Tanner graphs for LT codes. By using this modified Tanner graph, we can use a more simplified soft iterative decoding algorithm, and also the decoding performance can be improved, compared to the conventional method. Most importantly, we can simply derive the corresponding parity check equation with the modified Tanner graph, and thus the iterative decoding process can be terminated easily. The average number of iterations can be largely reduced. Simulation results in this paper reveal that our proposed method can achieve better performance than the conventional iterative soft decoding with much less complexity.
KeywordsвҖ”rateless codes, systematic LT codes, iterative decoding, soft decoding, Tanner graph, sum-product algorithm
I. INTRODUCTION
Rateless codes, also known as fountain codes, were introduced to obtain capacity-achieving performance for the channel without any feedback and without knowing the erasure conditions. With rateless codes, every receiver can access the ongoing broadcast session at arbitrary time and receive the message with the required quality of service (QoS), independent of the channel condition. Luby transform (LT) codes were the first class of rateless codes [1]. The decoding complexity of LT codes is generally infeasible when the length of the code is long. Raptor codes were designed to solve this problem, as an extension of LT codes [2]. A raptor code is a concatenated code of an LT code as an inner code and a precode as an outer code, and generally, a low density parity check (LDPC) code is used for a precode. It has been verified that raptor code outperforms LT codes on a wide variety of noisy channel [3].
In earlier stages, these rateless codes have almost exclusively been investigated on wired internet systems, and therefore, investigated on erasure channels with a hard decision decoding method. The channel condition of wireless communication systems are usually much more severe, and thus attempts

In addition, by using the fountain-like characteristics of rateless codes, they were considered for incremental redundancy application [7][8]. Since most of the wireless channels have time-varying characteristics, an adaptive decoding scheme was introduced to reduce the average complexity of soft iterative decoder for LT codes [6].
As far as soft iterative decoding is concerned, systematic codes have advantages because they can provide initial log likelihood ratio (LLR) values from the channel, resulting in efficient belief propagation [7]. Systematic LT codes are generated by the systematic generator matrix which can be produced by expanding generator matrix of non-systematic LT codes with the aid of a unity matrix [9].
In this paper, we modify the conventional Tanner graphs of systematic LT codes so that we can draw them in terms of check nodes and bit nodes as in the case of LDPC codes, instead of encoding nodes and information nodes as in the case of the conventional LT codes. There have been studies on modified encoding scheme for LT codes, mainly for efficient unequal error protection [10][11]. In comparison with these previous studies on modified encoding scheme, the purpose of our study is not in the modification of encoding process itself, but the main purpose is in increasing the efficiency and performance of decoding process.
By using the proposed modification of the Tanner graph, we can derive the corresponding parity check matrix and apply the min-sum algorithm used for LDPC codes [12]. By using this approach, there are a number of advantages compared to the conventional soft iterative decoding for LT codes [4][5]. First, the performance can be enhanced. Second, we can utilize a simple early stopping condition using the parity check matrix, and thus the average number of iterations can be largely reduced. Third, in case of raptor codes using systematic LT codes, the hardware structure of the iterative decoder can be implemented in a much simpler way, because basically the same decoding algorithm can be used in both of systematic LT and LDPC codes.

II. SOFT DECODING FOR LT CODES
A. System Model
Figure 1 shows the transmission flow using a systematic LT code. At the transmission part, information u with length k is encoded to c of length n by the systematic LT encoder with a predetermined degree distribution О© [1]. Then, c is modulated and transmitted through the channel.
At the receiver part, the receiver starts the demodulation and decoding process if a sufficient number of received symbols are collected. The LT decoder takes the demodulated soft information, r of length n as prior information, and produces soft information, uЛҶ of length k, by iterative log-likelihood (LLR) estimation. The ACK signal will be sent to the transmitter if the decoding is successful. Otherwise, more received symbols will be collected for the next decoding attempt.
B. Soft iterative decoding algorithm
The left side of Fig. 2 shows the conventional Tanner graph for systematic LT codes [4], compared to the modified Tanner graph in the right side. The soft iterative decoding of the LT codes is performed using the belief propagation algorithm for LT code [4]. In the iterative decoding, the estimation of the LLR between the nodes is performed iteratively.
