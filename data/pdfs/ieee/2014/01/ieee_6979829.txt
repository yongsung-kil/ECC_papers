An Improvement of Approximate BP Decoding

Abstract—In this study, we present new methods for piecewise linear approximation of Min-Sum (MS) decoding algorithm with good trade-off between performance and complexity. By analysis based on density evolution, the performance of the proposed method is identical to that of the Belief-Propagation decoding algorithm. The increment of the complexity of the proposed algorithm is small compared with the MS decoding algorithm. Moreover simulation result also shows effectiveness of the proposed algorithm.
I. INTRODUCTION
Low-density parity-check (LDPC) codes and the BeliefPropagation (BP) decoding algorithm [1] have received a lot of attention because of their attractively high performance [3], [5], [15]. From the theoretical analysis of this coding scheme called density evolution [4] has shown its promising capability. The performances of irregular LDPC block codes [5] and the LDPC covolutional codes [18] (spatially-coupled LDPC codes [20]) achieve very close to the Shannon limit.
Exact calculation of the BP decoding algorithm may be difficult to implement in a hardware circuit, since it requires large cost operations, i.e. exponential and logarithmic operations. Therefore efficient algorithms of BP decoding have been studied so much. The most well-known algorithm is the Min-Sum (MS) decoding algorithm [2] which carries only comparison and addition operations. Unfortunately, this leads performance degradation especially for codes with large length. Based on MS decoding algorithm, several variants of algorithms have been studied, i.e. improvement of the MS decoding algorithm [7], [6], [8], [9], [11], [13], [14] or quantization of values in the decoder operations [10], [12], [19], [21], [22], i.e. normalized BP based decoding, off-set BP based decoding [7], or MS decoding with linear approximations [6], [8], [9], [11], [13], [14], [17]. In this study, we focus on the MS decoding with linear approximations [6], [8], [9], [11], [13], [14], [17]. These decoding algorithms have an advantage that they require no optimization for the parameters in the algorithm which depend on a class of LDPC codes as in [7]. Moreover they have performance higher than [7].
In this study, we present new methods for piece-wise linear approximations of MS based decoding algorithm with good trade-off between performance and complexity. To reduce approximation loss, the proposed method is also k-pieces of linear approximations, but k keeps small. By analysis based on density evolution, the performance of the proposed method is identical to that of the BP decoding algorithm with a small increase of the complexity. Moreover simulation result also shows effectiveness of the proposed methods.

V. SIMULATION RESULTS
In order to compare each approximation, we use LDPC codes by simulations. First we show a result using LDPCBC with N = 40000, dv = 4, dc = 16 of R = 0.75 in Fig. 3. The horizontal and the vertical axises represent Eb/N0 and bit error rate (BER), respectively. From Fig. 3, all the methods have almost identical performance, since effectiveness of performance can be seen for codes with long code length. This phenomenon can also be found in other literatures, e.g. [11]. Table VII shows thresholds comparison of (4, 16) LDPC-BC ensemble by the density evolution. From this table, performance differences of the methods can be verified as well as (3, 6) LDPC-BC ensemble in previous sections.
Next we show a reuslt using Lentmaier’s LDPC covolutional code (LDPC-CC) [18] with N = 120000, dv = 3, dc = 6, band width number L = 150 of rate R = 0.49, and the size of permutation matrix 400 × 400 in Fig. 4. From Fig. 4, App2 exhibits almost the same performance as the BP decoding algorithm and is better than other approximations. Table VIII shows thresholds comparison of (3, 6) LDPC-CC ensemble. From Table VIII, gap of thresholds can also be verified as well as the simulation result in Fig. 4 and threshold comparison of LDPC-BC presented in Sect. III and IV. Note that we also examined code performance for short block length ((3,6) LDPC-BC with N = 10000), but significant performance difference cannot be verified as well as Fig. 4.

VI. CONCLUSION
In this study, new methods for piece-wise approximation of MS decoding algorithm are devised. From the analysis based on density evolution and simulation results, we find that the performance of the proposed method is identical to that of the BP decoding algorithm with a small increase of the complexity. From simulation results, effectiveness of the proposed approximations can be verified for codes with large length.
