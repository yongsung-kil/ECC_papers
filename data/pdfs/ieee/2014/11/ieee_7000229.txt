Dynamic Decoding Algorithms of LDPC Codes Based on Simulated Annealing

Ru Cui, Xingcheng Liu
Department of Electronic and Communications Engineering Sun Yat-sen University Guangzhou, China
E-mail: isslxc@mail.sysu.edu.cn

Abstract-The dynamic scheduling (DS) strategy based decoding algorithm for low-density parity-check (LDPC) codes is explored from a novel point of view. In the view of dynamic selection strategies, intelligent search algorithms are taken into consideration. As is known, the dynamic decoding algorithm is a greedy one, where the message with the maximal-residual is always propagated, which causes unfairness in decoding scheduling when updating messages. To cope with the problem, the simulated annealing (SA) method is first introduced to provide assistance in dynamic scheduling-based decoding. Then, a more accurate dynamic selection strategy is presented based on the SA-assisted dynamic scheduling strategy, SA-DS. When the Metroplis acceptance criterion is satisfied, the SA-DS algorithm can select the most unreliable node accurately and give a priority to update it.
Simulation results demonstrate that the proposed algorithm behaves better than any other algorithm of interests.
Index Terms-low-density parity-check (LDPC) codes; dynamic scheduling; simulated annealing
1. INTRODUCTION
With the development of modern communication systems, a faster and more accurate decoder is needed in such systems. Many decoding algorithms for low-density parity-check (LDPC) codes have been studied due to their perfect performance since first introduced in 1962 [1].
In the view of message scheduling strategies, there are three general categories. The first one is the flooding strategy, among which is the log likelihood ratios belief propagation (LLR-BP) [2]. Tn this algorithm, all the variable nodes can be updated but the updated message can only be used in the next iteration. The second category is the serial scheduling strategy (SSS), where messages are updated in a serial manner according to a fixed order as presented in [3]. It is shown that the serial scheduling not only has a better performance but

also converges twice as fast as the flooding since the SSS can use the latest available information in the consecutive iterative decoding. The last one is the dynamic scheduling strategy, called the residual belief propagation (RBP) [4], where the unreliable message measured by residual is updated in a dynamic way. Tn the informed dynamic scheduling strategy [5] [6], the message with the maximum residual is updated first. It is demonstrated that the dynamic decoding algorithm has the fastest convergence speed and the best error-correcting performance among the three. Since then, some other decoding algorithms using dynamic scheduling strategy, such as the variable-to-check RBP (VC-RBP) [7] and the informed variable-to-check RBP (TVC-RBP) [8], were further studied. However, it also has some shortcomings because of its greediness [6]. Then, how to reduce the greediness was studied extensively [6][9][10]. All the strategies not only retain good property of dynamic algorithms, but also reduce the greediness to some extent.
Simulated annealing (SA) algorithm was first successfully introduced to combinatorial optimization in [II]. The biggest advantage of SA is that it can accept worse solutions at a certain probability during the process of searching the optimal solution and reduce the probability of trapping into a local optimum solution. As for the dynamic decoding algorithm, we noticed that the dynamic selection strategy plays a very important role in dynamic decoding. So, we first combine dynamic decoding algorithm with SA algorithm and then suggest a novel dynamic decoding algorithm for LDPC codes, named SA-DS. In the proposed algorithm, a new accurate dynamic selection strategy based on the simulated annealing is employed.

A. Dynamic selection strategy
The dynamic algorithm outperforms the flooding and the SSS owmg to the dynamically-updating order [5][6]. Before decoding, the dynamic selection strategy is used to determine which edge or node will be updated first. Usually, the dynamic decoding algorithm updates the least reliable message preferentially according to the residual computed in (3). However, the message reliability that just relies on the residual is not so accurate in some way. There are two cases that a large residual can be generated. One case is that the error-information becomes corrected after an update. The other is that the previously-corrected information turns back into an error due to node oscillations or some other reasons. If the message with the maximum residual generated by the former, it will be a waste of computation resources and the decoding algorithm will be hard to get converged.

IV. SIMULATION RESULTS
In this section, the performance of the proposed SA-DS decoding algorithm for various types of LDPC codes is presented. In simulations the signaling with BPSK modulation was transmitted over the AWON channel. In the proposed method, the initial value of the control parameter tk is set to to=lO. The final value is not set to tk and the attenuation factor a is set to a=0.99 because the number of iterations in our simulations is relatively small. The LDPC codes
in simulations are denoted with (n, k), where n is the length of codes and k is the number of
information symbols in the codes. In order to examine the error-correcting performance of the proposed SA-DS algorithm, three other decoding algorithms are used for comparison. The maximum number of iterations is set to 5 for all the decoding algorithms. Fig. 2 shows the perfonnance of different decoding algorithms of the regular LDPC code (504, 252), which is constructed with the method in [12]. Fig. 3 presents the perfonnance of the (576, 288) LDPC code, which is constructed based on the IEEE 802.16e standard [13]. It is clearly demonstrated that the proposed SA-DS algorithm has better BER performance than any other algorithms, especially at the relatively high signal-to-noise ratio region. The BER perfonnance vs the number of iterations for (576, 288) LDPC code at EtlNo= 2.5dB is presented in Fig. 4. It is shown that the SA-DS algorithm converges faster than any other algorithms within about 20 iterations.
