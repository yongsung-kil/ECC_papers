Simplified Forced Convergence Decoding Algorithm for Low Power LDPC Decoders

Byung Jun Choi and Myung Hoon Sunwoo
School of Electrical and Computer Engineering, Ajou University
San 5, Woncheon-Dong, Yeongtong-Gu Suwon, 443-749, Korea
bjchoi0117@ajou.ac.kr, sunwoo@ajou.ac.kr

Abstract—This paper proposes a simplified forcedconvergence (SFC) algorithm to reduce the computational complexity for low-density parity-check (LDPC) decoding. To reduce the computational complexity, the proposed algorithm uses the modified check node (CN) operation that does not use a condition for deactivating CNs. Therefore, the proposed SFC algorithm uses only one threshold value while the existing forcedconvergence (FC) algorithm uses two threshold values. The simulation results show that SFC achieves a bit error rate (BER) performance close to min-sum (MS) algorithm. SFC can reduce the computational complexity of check node by approximately 22.21% compared to FC.
Keywords—low-density parity-check codes, belief-propagation decoding algorithm, forced-convergence.

I. INTRODUCTION
Low-density parity-check (LDPC) codes were first introduced by Gallager in 1962 [1], [2]. LDPC codes have been researched because of their linear decoding complexity and near Shannon limit performance. These characteristics make LDPC an attractive candidate for error correcting code applications in communication systems. Recently, LDPC codes have been adopted as FEC codes in the emerging communication standards such as Wi-Fi (IEEE 802.11n [3], 802.11ac [4]), WiGig (802.11ad) [5], 10Gbase-T Ethernet (802.3an) [6], etc. Even though LDPC codes achieve outstanding performance, they require large codeword lengths that sharply increase the computational complexity. Hence, reducing the computational complexity is a key issue for LDPC codes. To reduce the computational complexity of LDPC decoders, the forced convergence (FC) has been proposed in [7].
LDPC decoding algorithms are based on a message-passing iterative decoding algorithm such as the belief-propagation (BP) algorithm [8]. The BP algorithm consists of check-nodeupdate (CNU) and variable-node-update (VNU), which exchange soft messages along the edges between variable nodes (VNs) and check nodes (CNs) in Fig. 1. FC is a technique to reduce the number of active nodes per iteration, which in turn can alleviate the overall decoding complexity. FC exploits the fact that a large number of VNs converges to a strong belief after few iterations, since these VNs have been reliably decoded. Thus, FC can skip updating their belief in subsequent iterations. FC uses a threshold rule to decide if a variable node should update its information in subsequent iterations or be considered as converged [9-13].
tv and tc are the threshold values used in [9]. FC results in the deactivated nodes that are satisfied with two conditions using tv and tc in each node. It can reduce the computational complexity by deactivating nodes.
In this paper, we propose the simplified FC (SFC) to further reduce the computational complexity. SFC uses a modified CN operation without a condition of CN. Therefore, the proposed algorithm uses only tv instead of tv and tc in FC [9]. Since the proposed algorithm efficiently produces the deactivated nodes, it can significantly reduce the computational complexity and memory accesses resulting in low power consumption.

III. SIMPLIFIED FORCED CONVERGENCE ALGORITHM
This section proposes the SFC algorithm to further reduce the computational complexity of CN. The computation of CN accounts for a large portion of the LDPC decoding algorithm. If the maximum number of iterations (Imax) is less than ten, the

existing FC [9] using two threshold values cannot reduce the computational complexity of CN. In contrast, SFC using one threshold value can further reduce the computational complexity of CN.

IV. SIMULATION RESULTS
In this section, we use Visual C to perform the bit error rate
(BER) performance simulations and to compare the computational complexities. We use the (6, 32) regular (2048, 1723) LDPC code and the AWGN channel with BPSK modulation. We assume that the maximum number of iterations (Imax) are five and ten. In the simulations, SFC uses only one threshold value tv = 7. FC uses two threshold values tv = tc = 7.
Fig. 3 shows the BER performances of MSA, FC, and SFC. For Imax = 5, SFC achieves a BER performance close to both MSA and FC. However, for Imax = 10, the results show that the difference between SFC and FC is approximately 0.05 dB at BER = 10-7. Moreover, SFC has a 0.05 dB lower performance than MSA. In summary, SFC shows comparable BER performance to MSA.

V. CONCLUSIONS
We proposed SFC to reduce the computational complexity of LDPC decoders, which shows a BER performance close to MSA. SFC can be implemented with a small modification to the existing FC with minimal hardware overhead. On average, compared with FC, the proposed SFC algorithm can reduce up to 22.21% of the computational complexity of CN at Eb/N0 ranging from 2 dB to 4 dB for tv = 7.
