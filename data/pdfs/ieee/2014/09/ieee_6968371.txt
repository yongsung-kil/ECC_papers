Abstract-In the field of communication, coding the data is the most crucial part. Since transmission channel is very much susceptible to noise, it becomes imperative to channel code the data to be transmitted. This paper presents an improved and faster Low Density Parity Check (LDPC) based algorithm, for channel coding grayscale images of 'n*n' dimension. It constructs the encoded image, by arranging bits into 8n data vectors and modulating it with BPSK modulation technique.The novel approch for designing an adaptive matrix as a function of input image size is described in the paper. The decoding algorithm is well equipped with a probabilistic approach. Further analyzing the performance of the algorithm shows improved restitution of images, that are affected by AWGN noise. Hence, LDPC codes prove their dominance over other coding schemes viz. Turbo codes, BCH etc, for applications such as medical imaging, deep­ space communications and multimedia.
Keywords:- Channel coding, Low Density Parity Check (LDPC), Grayscale Images, image processing,adaptive matrix, probabilistic approach, AWGN noise, Image restoration.

I. INTRODUCTION
A communication system is said to be reliable when it supports faster data transmission and faithful reconstruction of data at the receiver end, with minimum noise affecting the system. In digital communication systems, where images are transmitted over wireless link, the noise affecting these images should be minimized. In various applications, the necessity of broadcasting high quality images reliably and efficiently over the link has given rise to challenging channel coding techniques. These channel coding schemes are basically error correcting codes (ECC), used to improve the performance for digital systems and its storage. ECC engages systematic addition of extra bits to the data transmitted. These extra check bits do not convey any information by themselves, but they make it possible to sense and correct the errors. It also helps to ameliorate the effects of decoherence.
The information capacity of a channel is given by;

C = max[I(X; Y)] & H(X) :::; C

where; 'X' is the input to the channel i. e data transmitted whereas, 'Y' is the output of the channel i. e data received. In information theory, the Shannon coding theorem states that, 'If the entropy 'H(X) , of the symbol stream is equal to or less than the channel capacity 'C', then there exist a coding

technique that enables transmISSIOn over the channel with arbitrarily very small error.
However, this theorem does not provide any information as to which technique should be used specifically. Many coding schemes such as, Block codes, Hamming codes, BCH codes, Turbo codes, Reed-Solomon codes etc. have been pro­ posed. However, the most promising error correcting scheme which has gained widespread application in various fields, is the Low-Density Parity Codes (LDPC) [1]. It exhibits near Shannon limit performance when interpreted via iterative probabilistic algorithm.
In this paper, we present a complete Image Transmission System (ITS) for transmitting and restoring images sent on the wireless link. This system is based on channel coding grayscale 'n*n' dimension images, using LDPC scheme, creat­ ing '8n' data vectors and modulating them with Binary Phase Shift Keying (BPSK) technique. This algorithm facilitates high quality restoration of images, when affected with Additive White Gaussian Noise (AWGN). The decoding Algorithm also helps in achieving better bit-error-rate.
This paper is organized as follows; Section II discusses LDPC codes, its constructing, properties and its dominance over other coding schemes. Section III presents the imple­ mented Image Transmission System employing LDPC En­ coding and Decoding. Section IV elaborates the results and analysis of the algorithm, to prove its efficiency. The final Section concludes the paper.
II. Low DENSITY PARITY CODE (LDPC)
R. G. Gallagar fathered LDPC codes in 1962 at MIT [2] in his Phd thesis. Since then, LDPC codes have been implemented in wide range of emerging application fields such as error resilient data transmission over wireless channel, co-operative and satellite communication area etc. The per­ formance of existing coding and modulation scheme demon­ strated in [3] can be enhanced using LDPC properties.

2) Bit-Flipping Algorithm: Bit-flipping is a type of hard-decision message passing scheme for decoding LDPC codes.The detector takes the hard decision for each received bit and passes it to the decoder. The messages passed along the tanner graph edges are binary. In the decoding process, following steps are executed; i) A bit node 'i' sends a message declaring if it is a 1 or O. ii) Each check node 'j' transmits a message to each connected bit node 'i', declaring what value the bit is, based on the information available to the check node. iii) The check node determines that eq.no.3 is satisfied, if the modulo-2 '&/ sum of the incoming bit is O. iv) If majority messages received by the bit node are different from its received value, then the bit node flips its current value. v) This is an iterative process which continues until the parity­ check equation (eq.no.3) is satisfied or the detector runs out of its maximum iterations.
This algorithm has advantages that; additional iterations are avoided once a solution has been found, and a failure to converge to a codeword is always detected. In this technique, the codeword corrects itself and no complex computation is required.

The n*n dimension input image can be represented with 8n2 data bits. These bits are then arranged in the form of [8nxn] data vectors. Hence, the size of matrix representing the number of blocks is 8n and the block length is n. Since, each block length is [lxn], it contains nl8 pixels. These data vectors are multiplied with the Generator (G) matrix of [nx2n] size. Hence, we obtain LDPC encoded code vectors of dimen­ sions [8nx2n]. These code vectors are modulated with BPSK modulation scheme. Other sophisticated modulation schemes can also be implemented, but for simplistic approach, BPSK modulation is deployed in our system. The final encoded image

LDPC decoder and BPSK demodulator processes the re­ ceived encoded image at the recipient end. Set of 8n code vectors are processed by LDPC decoder. Further the code vectors of [8nx2n] dimension are multiplied with the transpose of parity check matrix H' having [nx2n] dimension, in order to retrieve the original data vectors. The process continues till syndrome becomes zero. If a non-zero syndrome occurs Bit-flipping (BF) algorithm is executed. Ultimately, 8n data vectors are recovered and are arranged to retrieve n*n image. A 512*512 gray-scale image is taken as an input test image. Algorithm 1: Algorithm for Adaptive LDPC Matrix Creation
A. LDPC encoding Algorithm for Image transmission
• As the input data is segmented in an image of 512*512 size, it contains 262144 pixels. Each pixel can be rep­ resented uniquely with 8 bits, which results into the requirement of 262144*8 bits.
• In order to encode such a huge amount of data, it is essential to rearrange them into a set of data vectors. Since the Generator matrix 'G' is of size 512*1024, the input data vector size is restricted to 512 bits.
• Hence, 262144*8 bits are arranged into 4096 data vectors, each having size of 512 bits.
• For obtaining 'G' matrix from parity check matrix, the row transform-echelon conversion method is used, out of the two techniques mentioned in [11].
• Rearrange the input data of 262144*8 bits into 4096 data vectors having 512 bits each.
• These 4096 data vectors are provided to LDPC encoder Generator matrix (G).
• Set of 4096 code vectors is generated, each appended with parity bits of 512 bits i. e it has 4096*1024 size.

• These channel encoded code vectors are modulated with Binary Phase Shift Keying (BPSK) modulator and trans­ mitted over a link.
B. LDPC decoding Algorithm for Image Restoration
• The received data is processed by LDPC decoder and its demodulated with Binary Phase Shift Keying (BPSK) demodulator.
• Code vectors 4096*1024 and transpose of parity check matrix H' 512*1024 are multiplied to retrieve data vectors and to generate the syndrome.
• If generated syndrome is zero, data is received faithfully whereas nonzero syndrome indicates corrupted data
• The Bit Flipping (BF) algorithm is implemented for decoding [12] as explained in earlier section.
• For non-zero syndrome, it searches out for corresponding failed parity check equations. For corresponding received code vector it flips error bits. Above procedure is repeated till the syndrome becomes zero.
• Finally, 4096 data vectors are recovered at the output. • Bit rearrangement is carried out to retrieve the 512*512
pixels image.

V. CONCLUSION
Eliminating distortion from images is one of the intricate problems. This paper presents an image transmission system that rectifies the data in spatial domain using LDPC error correction codes. LDPC codes have proved their dominance over other coding schemes in the field of communication. Since, the decoding procedure of LDPC involves an iterative structure, high coding gains can be achieved. The size of generator matrix (G) and parity check matrix (H) affects the processing time required. Hence, in our proposed system the dimension of H and G is flexible, depending upon the input image. For analyzing the accuracy and effectiveness of the algorithm, it is compared with standard image filters.

The results show improved visual quality which causes the significantly high Peak Signal to Noise Ratio (PSNR) values. Furthermore, to evaluate the performance of the decoding algorithm over AWGN channel, the pixels in error and BER are estimated for various SNR values.The algorithm is not restricted to a particular application area, since it can be implemented in deep-space communication, medical imaging and multimedia field. Hence without introducing much com­ plexity, a flexible system is implemented with the designing of adaptive matrices.This renders faster encoding and decoding of the data.
