I. INTRODUCTION

s2

Low-density parity-check (LDPC) codes were invented by Robert Gallager in 1961 in his Ph.D. thesis [1]. One method to construct LDPC codes is based on the construction of base matrix and the following replacement of each entry with permutation submatrix or zero submatrix to obtain paritycheck matrix [1]. This method without big losses can be limited to considering only circulant permutation submatrices, and a code constructed in such a way will be quasi-cyclic (after some permutation of symbols) [2]. Such codes are called QC-LDPC codes, constructions from this class are widely used in telecommunication nowadays. QC-LDPC codes are adopted for several standards: Wi-Fi (802.11) [3], WiMAX (802.16) [4], 5G [5], digital video broadcast (DVB), Advanced Television System Committee (ATSC) and others.
The conventional hardware implementation of QC-LDPC encoder or decoder includes several barrel shifters which are used to perform circular shifts of data blocks of size q. A barrel shifter is a digital circuit which consists of m = log2 q layers of multiplexers, k-th layer, 1 ≤ k ≤ m, is responsible for circular shift by 2k−1 positions. So, barrel shifter is able to perform circular shift by any integer value from 0 to q − 1. A schematic of barrel shifter for q = 4 is showed in Figure I. However, QC-LDPC encoder and decoder do not need to support circular shifts by any value. Defining the set of supported shift values, one can try to replace the standard basis set consisting of powers of two by another basis set of smaller cardinality, but still preserving the property: every supported shift value can be expressed as a sum of several elements from new basis set. Moreover, we can costlessly add a non-identical fixed connection on the input or the output of barrel shifter, it means that a wider property may be considered: every supported shift value can be expressed as a sum of a fixed constant and several elements from the basis set. So, in some cases the barrel shifter can be replaced by a similar circuit

From a practical point of view one need to find a basis set of minimal size for a fixed set of supported shift values. In this paper we concentrate on the theoretical extremal problem specified in the abstract. Particularly, knowing the number ζ(q, m) allows to conclude that for arbitrary set of supported shift values of cardinality < ζ(q, m) the improvement of barrel shifter is always possible.
The problem specified above is closely related to some problems in additive number theory. A basis set of size m can cover up to 2m elements and to cover exactly 2m elements the basis set must satisfy the property that all subset sums are different. For more than 50 years mathematicians research a problem of the minimal number N such that there exists a set of m numbers ≤ N with distinct subset sums [6], [7]. Another observation is that a minimal-size set requiring basis set of cardinality ≥ m should not consists of subsequent elements or more generally should be a distinct difference set, i.e. all differences between two elements are distinct modulo q [8], [9]. Search of basis set of size < k for a set S of size < ζ(q, k) can be simplified if one element of S is the sum of several others modulo q or sum of several elements of S equals 0 modulo q. III. RELATIONS BETWEEN FUNCTIONS ζ AND ζ+
At first sight, consideration of basis sets with constant shift provide significantly more freedom, but actually it increases function ζ(q, k) by only 1.

Proposition 1. For any k, 1 ≤ k ≤ m, the following equality

holds:

ζ+(q, k) = ζ(q, k) + 1.

Fix a subset S, S ⊂ Zq, of size |S| = n.
Definition 1. A set G, G ⊂ Zq, is called a basis set for set S if for any s ∈ S there exists subset Gs, Gs ⊂ G, such that: s = g∈Gs g, where summation is taken modulo q. In other words, S ⊂ Σ(G).
It is obvious that for any set S, S ⊂ Zq, there exists a basis set of cardinality not more than m, because set {1, 2, . . . , 2m−1} is the basis set for whole ring Zq.
As explained in the introduction, for the considered optimization problem of shift module, we can costlessly add additional constant shift to all shift values supported by some basis set. Thus we consider the following generalization of basis sets.
Definition 2. A pair (G, c), G ⊂ Zq, c ∈ Zq, is called a basis set with constant shift for set S if for any s ∈ S there exists subset Gs, Gs ⊂ G, such that: s = c + g∈Gs g, where summation is taken modulo q. In other words, S ⊂ c + Σ(G). By cardinality of basis set with constant shift (G, c) we mean the cardinality of G, i.e. |G|.
Example 1. Let q = 8. Set S = {2, 3, 5, 6} obviously cannot have basis set of cardinality ≤ 2 because it has 4 nonzero elements. At the same time there exists basis set with constant shift of cardinality 2: (G, c) = ({1, 3}, 2).

For several particular cases function ζ(q, k) is exactly known. However for general case, even asymptotic behaviour is not clear. For the most interesting case q = 2m and k = m, from Proposition 4 and Theorem 1 we only know that
Corollary 2. For q = 2m and m → ∞, the following asymptotic inequality holds:
(log2 q)(1 + o(1)) ≤ ζ(q) ≤ (log2 q)2(1 + o(1)).
