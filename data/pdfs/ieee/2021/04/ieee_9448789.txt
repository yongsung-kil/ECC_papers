Dynamic Dual Quantization-Domain Decoding for NR LDPC Codes

Taehyun Kim Network Business Samsung Electronics Co., Ltd. Suwon-si, Korea th8978.kim@samsung.com

JooSung Park Network Business Samsung Electronics Co., Ltd. Suwon-si, Korea jsjoo.park@samsung.com

Abstract—In this paper, we propose a low-density paritycheck (LDPC) decoder with a dynamic dual quantization-domain (DQD) that check nodes are operated in a 3-bit domain while variable nodes are operated in a higher bits domain. The proposed mapping to transform messages from a domain to the other domain gradually changes during the LDPC decoding process and is optimized by differential evolution and machine learning. In addition, for performance improvement, the variable node performs self-correction (SC) that the unreliable messages are erased. Simulation results show that the proposed dynamic DQDSC decoder has better error correction performance compared to the previously proposed hybrid DQD decoder.
Index Terms—low-density parity-check code, non-linear mapping, quantization, low-complexity, machine learning
I. INTRODUCTION
Low-density parity-check (LDPC) codes, proposed by Gallager [1], are error correction codes well known for the error correction performance close to the Shannon limit and the high parallelization in hardware. Due to these characteristics, LDPC codes are adopted for various communication systems including the data channel of the enhanced mobile broadband (eMBB) scenario in 5G new radio (NR) [2]. For the battery-powered communication systems, there were several schemes for reducing the computational complexity of the LDPC decoding algorithm without significant performance loss. The min-sum algorithm [3] is a low complexity decoding algorithm using a mathematical approximation in the check node operation. Various schemes were proposed to improve the error correction performance of the min-sum algorithm by compensating for errors due to approximation [4].
A dual quantization-domain (DQD) LDPC decoder [5] was proposed for reducing the implementation complexity of the min-sum algorithm. In DQD decoding, the check nodes operate in a 3-bit domain while variable nodes operate in a higher bits domain. To reduce performance loss due to low precision, a non-linear mapping that is optimized through differential evolution [6] is used to transform messages from a domain to the other domain. On FPGA and ASIC platform, the DQD decoder shows the improvement in power consumption and reduction in logic cost.
Similarly, finite alphabet iterative decoders (FAIDs) using a simple check node update rule with low bit precision

were presented for the binary symmetric channel (BSC) [7]– [9]. Recently, authors in [9] use recurrent quantized neural networks to optimize the message update look-up tables. In some LDPC codes, their FAIDs show superior performance than the floating point belief propagation (BP) decoder in the waterfall region.
In this paper, we propose a dynamic DQD self-corrected (DQD-SC) decoder. The main goal of this work is the performance improvement of the DQD decoder proposed in [5]. The basic operation of the DQD decoder and their method for optimizing mapping parameters in [5] are used in the proposed decoder. Compared to the previously proposed DQD decoder, the dynamic mapping parameters are used to consider varying log-likelihood ratio (LLR) during the decoding process. The mapping parameters for each iteration are optimized through differential evolution and machine learning. In addition, the self-correction (SC) [10] is performed at variable nodes to improve error correction performance by erasing the unreliable messages.
The remainder of this paper is organized as follows. In Section II, we review the DQD decoder. The details of the proposed dynamic DQD-SC decoder are explained in Section III. The simulation results are discussed in Section IV. Finally, Section IV concludes this paper.
II. DUAL QUANTIZATION-DOMAIN LDPC DECODER
A binary (N, K) LDPC codes is identified by a M × N parity check matrix H, where M is equal to N −K. In general, LDPC decoding algorithm is represented as the exchange of messages between N variable nodes and M check nodes. Let LV 2C (m, n) and LC2V (m, n) denote, respectively, the LLR of propagated message from the m-th check node to the n-th variable node and the LLR of propagated message from the nth variable node to the m-th check node. The LLR of received information by the n-th variable node from the channel is denoted by Lch(n). The LLRs LV 2C propagated from the nth variable node are initialized to Lch(n).

III. DYNAMIC DUAL QUANTIZATION-DOMAIN SELF-CORRECTED LDPC DECODER
A. Dual Quantization-Domain Self-Corrected LDPC Decoder
The proposed dynamic DQD-SC decoder uses the SC method to improve error correction performance. SC is one of the schemes to improve the performance of the minsum algorithm. In dynamic DQD-SC decoder, if the sign of calculated L(Vl)2C (m, n) and the sign of LV(l−2C1)(m, n) are different, the information in LV(l)2C (m, n) is considered as unreliable and erased (i.e., L(Vl)2C (m, n) = 0). If L(Vl−2C1)(m, n) is zero, the L(Vl)2C (m, n) is always updated to the calculated value regardless of its sign. Note that the optimized mapping parameters for the DQD-SC decoder are different from that of the DQD decoder. The reason is that the erased message affects the probability distribution of LLR in the LDPC decoder.

B. Dynamic Dual Quantization-Domain

The distribution of LLRs in the LDPC decoder changes during the iterative decoding process. The mapping which is not suitable for LLR distribution can cause performance degradation. In order to avoid significant performance loss, we use the dynamic mapping that changes during the decoding process considering the change in LLR.

IV. SIMULATION RESULTS
Simulations are performed over an AWGN channel with binary phase-shift keying (BPSK) modulation. For the performance comparison, we simulate the hybrid DQD decoder [5], DQD-SC decoder, hybrid DQD-SC decoder, and dynamic DQD-SC decoder. Following the optimization method represented in [5], the DQD mappings for DQD-SC decoder and hybrid DQD-SC decoder are optimized for two SNR points around BER= 10−4 and 10−6 region. In addition, we also represents the performance of the floating point SC decoder. It shows how much performance degradation can be reduced by the proposed methods. The NR BG2 LDPC codes with K = 520 and code rate R = 1/3 and 1/2 are used for the performance evaluation. In all simulations, the maximum number of iterations is 30.
The size of the training data set used in this paper is 1000. The batch size is 100 and the number of epochs for training scaling factors is 20. The mapping parameters for hybrid DQD-SC decoder and dynamic DQD-SC decoder using NR BG2 LDPC codes with K = 520 and code rate R = 1/3 are shown in Fig. 2. The first half iterations of the hybrid DQD-SC decoder use the mapping optimized for the low SNR region, and the mapping optimized for the high SNR region is applied to the rest of the iterations. In general, the average magnitude of LLRs in the LDPC decoder is greater when the SNR is higher. Therefore, the mapping parameters for the high SNR region are greater than that for the low SNR region. As a result, the hybrid DQD-SC decoder has the effect of using mapping parameters considering the reliability increase in messages exchanged during the decoding process. The dynamic DQD-SC decoder with various mapping rules that are determined by machine learning shows the best error correction performance. It shows the performance much closer to the floating point SC decoder than the others. Furthermore, the dynamic DQD-SC decoder converges faster than the hybrid DQD decoder. Figure 7 and 8 show the effect of the number of epochs on the error correction performance. The performance of dynamic DQD-SC decoders represented in Fig. 7 and 8 is better than the hybrid DQD decoder, but there is a difference in performance according to the number of epochs. When the number of epochs is 10, the regression is underfitting. It means that the number of epochs is not enough for the model to be learned. On the other hand, overfitting occurs when the number of epochs is 30. In this case, the model is over-trained to the training data set and loss generality. Therefore, the performance is degraded in the simulation where the training data is not included. The appropriate number of epochs depends on the LDPC structure and training data, so it should be experimentally determined.

V. CONCLUSION
A dynamic DQD-SC LDPC decoder is proposed in this paper. The check nodes operate in a low precision quantization domain while the variable nodes operate in a high precision quantization domain. The mapping function to transform messages from a domain to the other domain changes during the decoding process considering the LLR change. We use differential evolution and machine learning to optimize the dynamic mapping parameters. The proposed regression model is very simple, but sufficient to get the mapping parameters. In addition, the SC method is applied to the variable node for the improvement of the error correction performance.
