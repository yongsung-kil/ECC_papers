AbstractвҖ”A novel Neural Offset Min-Sum(NOMS) Belief 
Propagation(BP) decoding algorithm based on model-driven is 
proposed which applied to LDPC decoding. NOMS is improved 
multiplication in Neural Normalized Min-Sum(NNMS) into 
addition operation to reduce the complexity of calculation., a 
better Bit Error Rate (BER) performance is simultaneously 
achieved in the same condition. Secondly, considering that there 
are still many multiplication operations in NOMS, we propose a 
novel Shared Offset Min-Sum(SNOMS) to reduce the number of 
weights in the network by sharing parameters. Finally, codebook-
based quantization is used to further reduce the memory 
consumption. Simulation experimental results show that the 
proposed method has a better BER performance, and the 
decoding accuracy of the decoder is 0.65dB higher than that of the 
NNMS after 5 iterations. In addition, SNOMS decoding method 
achieves almost the same decoding performance comparable to 
that of NOMS, but requires less complex calculation. Proposed 
quantization of code-book method reduces memory requirement 
significantly with slight performance loss. 
KeywordsвҖ”Deep learning, offset min-sum decoding, model-
driven neural network, belief propagation 
 
I. INTRODUCTION 
In the 1960s, Gallager firstly proposed Low-density parity-
check(LDPC) in[1]. Because it has close to Shannon limit[2] 
and can be implemented in parallel decoding algorithm[3], and 
then, scholars delving into the LDPC code, obtained many 
excellent results. LDPC decoding includes based on hard 
decisions algorithm and the soft decision. Hard decision 
algorithm is simple to operate and easy to implement, but its 
performance is poor. The soft decision algorithm performance 
is well, but the hardware implementation complexity is high. 
For BP algorithm, Normalized Min-Sum(NMS) and Offset 
Min-Sum(OMS) algorithm further revised the calculation of 
variable message[4], that employs correction factors can offer 
competitive decoding performance with a low complexity. 
In recent years, Deep Learning(DL) technology has been 
applied in a variety of detection, including image classification, 
object detection [5], speech signal processing [6], machine 
translation [7] and games[8].  
DL technology has shown extraordinary potential in the field 
of channel coding, such as error-correction codes [9-11]. In 
these literatures, fully connected network or Convolutional 
Neural Networks(CNN), Recurrent Neural Networks(RNN) 
network is used as the DL framework, and the performance 
equivalent to or even beyond conventional decoding is achieved. 
When the codeword is short, the DL training effect is well. But 
as the data grows in length, there is an exponential increase in 
complexity. Those literatures abandoned the theoretical 
knowledge of signal encoding and decoding, and only use a 
large number of data to train numerous parameters, thereby 
converging slowly and suffering heigh complex computation 
which difficult to implement in chips. These methods do not 
result in any reduce the delay and resource consumption on the 
basis of improving performance. 
In order to solve the above problems, Model-driven DL has 
been proposed in recent years. Nachmani et al. in [12-14] 
proposed to expand Tanner factor graph into a network layer 
plus weight coefficient to realize model-driven DL network (BP) 
decoding, which greatly improves the performance of the 
decoder. Lugosch et al. in [15] proposed the iterative expansion 
of the OMS algorithm as the BP DL nerve to realize the BP 
model driven neural network of linear codes. Xu et al. [16] 
implemented BP decoding DL algorithm based on Polar code. 
Abotabl et al. realized the OMS decoding model-driven DL 
method, whose [21] framework can be easily extended to any 
decoding scheduling. Wang et al. realized the normalized

The main contributions of this paper are as follows: 
A model-driven LDPC decoding algorithm named NOMS is 
proposed, in which Check Nodes(CNs) learn an additive offset 
parameter for each edge, rather than a multiplicative weights in 
NNMS. NOMS reduces complex operations, while get superior 
Bit Error Ratio(BER) performance than NNMS in [17] with 
same condition. 
Considering that there are many multiplication operations in 
NOMS, we propose a Shared Offset Min-Sum(SNOMS) to 
reduce the number of weights in the network by sharing weights. 
SNOMS significantly reduces complexity, while achieve a 
slight reduction in BER performance than NOMS. 
In the end, codebook-based quantization is used to further 
reduce the memory complexity. All of parameters is quantized 
from float32 to codebook, this method can effectively reduce 
the memory bit without visible BER performance degradation. 
The rest of the paper is structured as follows: part II 
describes LDPC decoder. III describes the proposed technique 
NOMS and SNOMS which is simulate in Section IV. Finally, a 
conclusion is provided in section V. 
II. PRELIMINARIES 
AпјҺ LDPC Decoder 
LDPC code is a linear block code, which achieve near 
Shannon channel capacity with long block length. The parity 
check matrix рқҗ»рқҗ» [18] is sparse with most of вҖҳ0вҖҷs, and very few 
вҖҳ1вҖҷs. BP decoding algorithm is used to decoder LDPC 
code(рқ‘Ғрқ‘Ғ, рқ‘ҳрқ‘ҳ), which contain N bit, рқ‘ҳрқ‘ҳ bit information in рқ‘Ғрқ‘Ғ. 
When рқ‘ҳрқ‘ҳ bit data is encoded and passed through the channel, a 
lot of noise is added, and the original signal needs to be 
recovered according to the Variable Nodes(VNs) and CNs in 
the decoding process. The content of LDPC decoding is that the 
maximum likelihood ratio of the signal passing through the 
channel superimposed noise is used as the decoded information, 
and information is transmitted to the CNs through the VNs, and 
CNs feeds back the information. Repeat those process, the 
decoding is finally realized from the information of the VNs. 
BP iterative algorithm includes Min-Sum(MS), NMS, and 
OMS with sum-product algorithm.

CпјҺ Model-driven BP methods 
DL BP decoder of data-drive uses a large amount of data to 
train network, that is like a black box, which unable to improve 
BER performance, on the contrary, bring higher latency and 
resource consumption. They need a lot of data and labels, but 
they're not easy to get. 
DL BP decoder of model-drive is using mathematical model 
which is unfold each iteration. Each iteration of BP decoder is 
unfolded as few layer of network, which is computed according 
mathematical decoder based on basic theory algorithms. In 
addition, network structure and parameters used properly, 
makes superior BER performance than conventional algorithms 
with less iterations. It's hard to find рқҗ»рқҗ» in LDPC without a circle 
in the Tanner graph, BP decoding often performs well for codes 
but it is not guaranteed to be optimal. Although cycle-less codes 
have poor optimal performance, codes with cycles are 
frequently used in practice [22]. BER performance of Model-
driven DL methods are closer to the optimal performance.

V. CONCLUSION 
A model-driven NOMS LDPC decoding method is proposed 
for DL networks. The multiplication operation of NNMS is 
changed to the addition operation in NOMS, which improving 
the BER performance and reducing the computational 
complexity. And through the sharing of weight coefficient, 
SNOMS further reduce complexity. Simulation results show 
that the proposed SNOMS decoder has similar decoding 
performance with lower computational complexity. In addition, 
through codebook-based quantization method, memory 
consumption is further reduced with invisible BER performance 
degradation.
