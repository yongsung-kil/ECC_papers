Abstract—Low-Density Parity-Check (LDPC) code is a 
linear block code approaching the Shannon limit. The 
improvement of the decoding performance of LDPC codes is a 
hot research direction today. To improve the decoding 
performance of the self-correcting min-sum (SCMS) algorithm, 
an improved self-correcting min-sum (ISCMS) algorithm is 
proposed based on this algorithm. In the iterative decoding 
process of the ISCMS algorithm, the update rule of the check 
node (CN) is optimized to the update rule of the CN of the IOMS 
algorithm, which improves the accuracy of decoding from the 
processing of the CN. Simulation results show that the proposed 
ISCMS algorithm has 
better 
error 
performance 
and 
convergence performance than the SCMS algorithm with a 
small increase in computational complexity.   
Keywords—Low-Density Parity-Check, belief propagation, 
self-correcting minimum sum algorithm 
I. INTRODUCTION  
Error control coding is a key technology to improve the 
reliability of information transmission, and it occupies an 
essential part in most digital communication systems today [1]. 
In 1962, Gallager first proposed low-density parity-check 
(LDPC) codes. LDPC codes have strong error correction 
capabilities [2], but due to the limitations of hardware 
technology at that time, the decoding of long codes is very 
difficult. So people did not do much research on it. In the 
1990s, MacKay and Neal proved that the performance of 
LDPC codes can approach the Shannon limit [3], and LDPC 
codes have re-entered the public's field of vision and become 
a research hotspot in academia. LDPC has the characteristics 
of simple encoding and decoding, parallel decoding, and 
strong flexibility. Therefore, it was adopted by the long code 
coding scheme of the data channel in the new radio (NR) of 
the 5th generation mobile communication system as an 
enhanced mobile broadband (eMBB) scenario [4]. 
Belief propagation (BP) decoding is one of the main 
decoding algorithms for LDPC codes. The disadvantage of the 
BP algorithm is the high computational complexity. 
Especially when the message quantization bit is low, the 
performance is severely degraded. On this basis, Fossorier 
simplified the CN message calculation formula, replaced the 
look-up table operation with a comparison operation, and 
proposed the minimum sum (MS) algorithm [5]. Although the 
MS algorithm reduces the computational complexity, the 
decoding performance suffers a greater loss. Later, Savin et al. 
proposed the self-corrected minimum sum (SCMS) algorithm 
[6], which improved the decoding performance and 
accelerated the convergence of the decoding from the 
perspective of variable node (VN)messages. 
The SCMS algorithm has excellent decoding performance, 
but its CN message update rule uses the traditional MS 
algorithm, which has poor performance in the CN processing. 
To improve the performance of LDPC code, this paper 
proposes an improved self-corrected minimum sum (ISCMS) 
decoding algorithm based on the SCMS algorithm, which 
optimizes the message of the CN. The calculation rules 
improve the decoding performance. 
II. BACKGROUND 
A. LLR BP Algorithm 
LDPC code is a special type of linear block code 
characterized by a check matrix, so its decoding method is 
different from other linear block codes. The BP algorithm is 
the most primitive LDPC decoding algorithm [7]. BP 
algorithm, also known as probability domain BP algorithm, is 
an iterative probability decoding algorithm. BP algorithm has 
high computational complexity, which consumes more 
hardware resources and time in practical applications, and it is 
very difficult to implement in engineering. To reduce the 
computational complexity of the BP decoding algorithm, the 
probability operation in the algorithm is converted to the 
logarithmic domain, and a large number of multiplication 
operations are simplified into addition operations. The 
simplified algorithm is called the log-likelihood ratio (LLR) 
BP algorithm [8].

can usually be represented by a check matrix [9]. For the

B. MS Algorithm 
Although the LLR BP algorithm has simplified many 
operations based on the probability domain BP algorithm, the 
update calculation of its CN contains the hyperbolic tangent 
function and its inverse function, which is still very 
complicated 
in 
implementation. 
Therefore, 
Fossorier 
simplified it, proposed the MS algorithm. 
Since the hyperbolic tangent function and its inverse 
function are both odd functions and monotonic, the MS 
algorithm verifies the node message update as in (6)

This calculation method uses the minimum value and the 
next minimum value to replace the actual amount of 
messages transmitted by the CN. Therefore, the result 
obtained by the MS algorithm is an approximation of the 
actual result of the LLR BP algorithm [10]. This causes the 
MS algorithm to inevitably sacrifice part of its performance 
but greatly simplifies the computational complexity. There 
are only "sum" operations and "comparison" operations in the 
decoding process, which is more conducive to hardware 
implementation.

The SCMS algorithm follows the update of the MS 
algorithm at the CN message and adds a reliability judgment 
to the update of the VN message. If the sign of a VN is 
reversed before and after an iteration, the reliability of the VN 
message is low, and it can be regarded as an error message. 
In order to prevent the error information from being passed 
to the next level, the SCMS algorithm directly sets it to 0, thus 
speeding up the iteration. 
D. IOMS Algorithm

algorithm can also be achieved by modifying the information 
volume of the CN. By comparing the difference between the 
output message of the CN in the MS algorithm and the LLR 
BP algorithm, it can be found that the message symbols 
output by the CN is the same, but the output message of the 
MS algorithm at the CN is greater than the value of the LLR 
BP algorithm.

offset (OMS) algorithm. As shown in (12), this algorithm 
subtracts an offset 𝛽 from the CN to make the amount 
calculated by the CN closer to the value of the LLR BP 
algorithm.

messages and improves the performance of the algorithm.

which makes the decoding performance of the SCMS 
algorithm poor. Therefore, in order to make up for its 
performance loss in the message processing of the CN, this 
paper adopts the IOMS algorithm to improve the performance 
of the decoding algorithm from the processing of the CN.

B. Density Evolution 
The density evolution method is a new method to analyze 
the asymptotic performance of modern and efficient error 
correction coding and decoding. Using the density evolution 
method, the average asymptotic performance of the sum-
product algorithm decoding of LDPC codes can be calculated 
numerically. In addition, we can get a threshold that can 
preliminarily predict the performance of this algorithm and 
whether it meets the set BER requirements [14]. 
For the sake of simplicity, first, fix γ to 0.2, and obtain the 
best η of 0.93 from density evolution. The density evolution 
result is shown in Fig. 1. 
 
Fig. 1. Parameter η optimization results. 
In order to make the calculated result closer to LLR BP, 
fix η to 0.93, and use density evolution to get the best γ value 
of 0.3. The density evolution result is shown in Fig. 2. 
 
Fig. 2. Parameter γ optimization results. 
Therefore, we use η=0.93 and γ=0.3 as the best factors for 
the ISCMS algorithm to CN message update.

Compared with the BP algorithm, the MS algorithm 
reduces many multiplication and division operations and 
converts them into addition operations. The SCMS algorithm 
only completes the operation of discarding VN messages with 
low credibility, so it does not increase calculation. The 
ISCMS algorithm adds a few additional operations and

iterations. Overall, the ISCMS algorithm improves the 
performance of the decoding algorithm. 
V. CONCLUSION 
An improved self-correcting minimum sum algorithm 
based on the SCMS algorithm was proposed in this paper. 
This algorithm uses two correction factors to correct the 
information in the update rule of the CN, making the 
information of the CN closer to the LLR BP algorithm. The 
simulation results show that with a small increase in 
computational complexity, the ISCMS algorithm can 
effectively improve the system's error characteristics and 
convergence characteristics.
