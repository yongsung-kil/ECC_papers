Abstract—In this paper, we propose an algorithm for improving performance of LDPC decoders, which we called the Mutational LDPC (MLDPC). This algorithm is based on different parity-check matrices produced by slight mutations. An iterative decoding process is performed using mutated matrices in several decoders running in parallel and an overall codeword estimation is then given by information from all decoders. This approach leads to a significant improvement in terms of Bit Error Rate.
Index Terms—belief propagation, error control, mutations, LDPC, MLDPC.
I. INTRODUCTION
E RROR correction coding have found many application in communications and reliable electronic systems. Over the past few decades, there has been a rapid development in the theory and applications of coding techniques. Low Density Parity Codes (LDPC), firstly proposed by Gallager [1], have become very popular in the past two decades because of their excellent performance.
LDPC codes are decoded by a probabilistic iterative algorithm, called Belief Propagation (BP) [2]. Several methods for improving the original BP algorithm have been proposed [3][7]. In this letter, we propose a method based on tracking a decoder convergency and utilization of several parity-check matrices produced by the mutational algorithm. Information from decoders using different parity-check matrices is combined to produce one overall codeword estimation. The proposed method improves Bit Error Rate, which is presented in simulations for different LDPC codes.
II. PRINCIPLE OF THE METHOD
The LDPC code C defined by the optimized parity-check matrix H and the unique generator matrix G, such GHT = 0. However, the last equation could be solved in a choice of different H matrices with different convergency properties. Therefore, we introduce a method which uses a set of H matrices produced by mutations of the optimized one. We call it the Mutational LDPC decoding (MLDPC), described in Section II-A.
All codewords of the code C satisfy the condition cHT = 0, where H consists of (n − k) row vectors, where n is the number of code bits and k is the number of information bits.

III. EXPERIMENTAL EVALUATION
The performance of the proposed MLDPC decoding were compared using several widely known LDPC codes, generated according to WiMAX [9] and CCSDS [10] standards. Performance results are shown in Fig. 3 and Table I.
The efficiency of this algorithm can be seen in the comparison of Bit Error Rate in Fig. 3 and Table I. For WiMAX 2304 code, the Bit Error Rate was up to 89 times lower, when four additional mutational decoders were used. One of the main advantages of the proposed algorithm is its natural parallelism, which can be utilized in applications.
IV. CONCLUSIONS
We proposed a method for improving performance of LDPC decoding, callled MLDPC. The method is based on using the set of parity-check matrices produced by mutations. All paritycheck matrices are used for decoders running in parallel. The decoders produce probability vectors, which are combined to get the Overall Probability Vector (OPV). According to the
