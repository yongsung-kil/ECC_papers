The optimization of error floor in M-QAM multilevel coded modulation scheme based on LDPC code
Mao Ge, Liqian Wang*, Zhihui Wang, Runqiu Gao State Key Laboratory of Information Photonics and Optical Communications, Beijing University of
Posts and Telecommunications, Beijing 100876, P. R. China *E-mail (Corresponding author): lqwangwang@bupt.edu.cn

Abstract—In M-QAM multilevel coded modulation scheme with LDPC, the system error floor can be effectively reduced and the coding gain can be improved by adjusting the component code rate based on the capacity rule.
Keywords: MLC; LDPC; error floor
I. INTRODUCTION
Joint coded modulation technology can improve the system performance by exchanging the complexity of the equipment for coding gain without reducing the spectral efficiency and power utilization. Its emergence is marked by trellis coded modulation (TCM) proposed by Ungerboeck in 1976[1], [2] and multilevel coded modulation (MLC) proposed by Imai and Hirakawa in 1977[3]. TCM was later summarized as a special case under MLC framework [4]. MLC has attracted much attention for it has many advantages. Such as it can choose FEC code type and modulation format flexibly, its system capacity is not constrained by the mapping rules nor the modulation order. Besides, the system implementation complexity of MLC is lower than other schemes. We choose LDPC code as the component code of M-QAM (M-ary quadrature amplitude modulation) multilevel coded modulation scheme for its performance has been proved to be close to Shannon limit [5] [6] and LDPC has been selected as the eMBB data channel coding scheme of 5G technology standard.
There are several labeling strategies for MLC scheme, such as block partitioning (BP), mixed partitioning (MP), Ungerboeck set partitioning (UP) and Gray labeling. The latter two have prompted more research in additive white Gaussian noise channel. the set partitioning strategy introduced by Ungerboeck maximizes the minimum intrasubset Euclidean distance while Gray labeling ensures that there is only a bit difference between adjacent constellations. [4], [8] propose that Ungerboeck set partitioning performs better than other labeling strategies like BP, MP and widely used Gray labeling. In addition, it also proposes that Ungerboeck set partitioning can effectively reduce the error propagation caused by multistage decoding (MSD) in[4]. However, in our research we find that when using irregular QC-LDPC as the component code of M-QAM multilevel coded modulation scheme and allocating the code rate according to the capacity rule, the system with Ungerboeck set partitioning exhibits serious error floor phenomenon and its performance is worse than the system with Gray labeling and other labeling strategies. By analyzing the bit error rate

of equivalent sub-channels at each level and then adjusting the component code rate, we finally reduce the system error floor and improve the system coding gain. The outline of this paper is as follows. In Section II, we introduce the theoretical basis and system construction of MLC with MSD. Besides, we briefly introduce the capacity rule and calculate the equivalent subchannel capacity of each level in 16QAM scheme. In Section III, we firstly verify the performance of the component codes in BPSK and then apply them to 16QAM system for simulation, after that we analyze the results and adjust the component codes rate. Section Ⅳ concludes the paper.
II. THEORETICAL BASIS AND SYSTEM CONSTRUCTION
The MLC scheme with MSD is shown in Fig.1, taking 16QAM as an example. The MLC scheme divides the real physical channel into log2 (M) equivalent subchannels, each bit bi (i=0,1,2,3) is protected by different independent LDPC code. When multistage decoding is adopted at the receiver side, the LLR value’s calculation of equivalent high-level subchannels will refer to the decoding results of low-level subchannels. The use of mutual information between subchannels makes the decoding results more accurate, but it inevitably leads to error propagation, which is particularly prominent at low SNR.

that the performance of the first three equivalent subchannels is normal, while the BER of the fourth equivalent subchannel remains at 1e-5 at high SNR. After comparison, we find that the code rate applied to the fourth equivalent subchannel is 1, which means the fourth equivalent subchannel is not protected by LDPC. When calculating the BER of the whole system, the number of error bits of all equivalent subchannels should be considered. The information bits of four equivalent subchannels are in the same order of magnitude, while the error bit of magnitude of the fourth equivalent subchannel is significantly higher than that of other equivalent subchannels. On the other hand, the capacity of the fourth equivalent subchannel is always infinitely close to 1 with the increase of SNR, so the system with Ungerboeck set partitioning exhibits serious error floor phenomenon. In the system with Gray labeling, the result shows that the first and third equivalent subchannel’s BER much higher than the others while the preBER (BER before decoding) is opposite. It means that the component codes don’t have enough protection for the first and the third equivalent subchannel compares to the other two equivalent subchannels.
In order to reduce the system error floor with Ungerboeck set partitioning and improve the system performance with Gray labeling, we adjusting the component codes rate applied to all equivalent subchannels. In the system with Ungerboeck set partitioning, we set the fourth component code rate equals to the third one and remain others unchanged, which makes the total transmission rate is almost constant. In the system with Gray labeling, we reduce the component codes rate of the first and third equivalent subchannel and increase the other two. Through simulation, it can be found that this method can greatly reduce the error floor of the system with Ungerboeck set partitioning and improve the coding gain about 0.3 dB at a BER of 1e-7 in the system with Gray labeling, as shown by the dotted line in Figure 3. In addition, we find that the similar method can be applied to 64QAM MLC-MSD scheme to reduce the error floor and improve the system coding gain, but the results will not be shown int this paper due to the limited space.

IV. CONCLUSION
In application, when applying irregular QC-LDPC as the component code to M-QAM multilevel coded modulation scheme and allocating the code rate strictly according to the capacity rule, the system performs not good as theory and exhibits serious error floor phenomenon. By adjusting the component codes rate, the error floor of the system with Ungerboeck set partitioning could be greatly reduced and the coding gain could be improved about 0.3 dB at a BER of 1e7 in the system with Gray labeling in 16QAM. The similar conclusion can be extended to 64QAM and other systems.
