Joint Coding/Decoding Optimization for DC-BICM System: Collaborative Design
Zhiping Xu , Graduate Student Member, IEEE, Lin Wang , Senior Member, IEEE, and Guanrong Chen , Life Fellow, IEEE

Abstract— Differential chaotic bit-interleaved coded modulation (DC-BICM) system can achieve coding gain from channel code without loss of bandwidth in the differential chaos shift keying (DCSK) system, but the current DC-BICM system using the belief propagation decoding algorithm has high decoding complexity, which brings burdens to resource-limited applications. In this letter, a novel collaborative design framework based on joint coding and decoding optimization is proposed for the DC-BICM system over multipath Rayleigh channels. Firstly, the column-weight-based (CWB) shuffled scheduling decoding is introduced into the DC-BICM system to lower the decoding complexity, called shuffled-based DC-BICM system. Then, a CWB extrinsic information transfer (CWB-EXIT) algorithm is developed for the theoretical analysis of this system. With the CWB-EXIT algorithm, a low-storage protograph low-density parity-check (P-LDPC) code is designed for this system, and the column-weight distribution is taken as an important constraint in the collaborative design process. In this way, the storage space used for storing the scheduling order in the CWB shuffled scheduling can be saved, whose size is proportional to code length. Theoretical analysis and simulations show that the shuffled-based DC-BICM system with the low-storage P-LDPC code can achieve lower decoding complexity, less storage requirement and better performance than the existing systems.
Index Terms— DCSK, BICM, joint coding and decoding, protograph LDPC, shuffled scheduling, extrinsic information transfer, multipath Rayleigh fading channel.
I. INTRODUCTION
D IFFERENTIAL chaos shift keying (DCSK) systems [1]–[5], [9], [10] can achieve remarkable performances over multipath fading channels with intrinsic spread spectrum properties, which is particularly suitable for complex communication environments [2], [10]. To enhance the reliability of the DCSK systems, error correcting coding can be adopted at the price of increasing bandwidths [3], [4]. Coded modulation is suitable for band-limited transmission [6], since it can achieve coding gain without loss of bandwidth. To improve the performance over Rayleigh

fading channels, bit interleaved coded modulation (BICM) was proposed in [7], and further analyzed in [8]. BICM was firstly introduced to improve the performance of the DCSK system in [9], named differential chaotic BICM (DC-BICM) system, which was then optimized over powerline channel [10]. The DC-BICM system adopted the protograph low-density paritycheck (LDPC) code [11] and variable degree matched mapping (VDMM) rule [12], which is optimized in [13] and generalized in [14]. And the protograph extrinsic information transfer (EXIT) algorithm can be utilized to analyze the decoding threshold of the protograph LDPC (P-LDPC) code of this system [15]. However, the belief-propagation (BP) decoding used in the DC-BICM system has high decoding complexity.
The decoding complexity in LDPC-related communication systems can be measured by O(EI), where E represents the sum of edges in the LDPC codes and I represents the iteration number [16]. The former is determined by the structure of the LDPC codes and the latter is mainly decided by the decoding algorithm. When a specific LDPC code is adopted, the decoding complexity is mainly determined by the decoding algorithm. The current research on DC-BICM system is developed with the BP decoding, also named flooding scheduling decoding, where the variable node (VN) and check node (CN) update messages in parallel. Differing from the flooding scheduling decoding, the serial scheduling decoding can converge faster than the parallel scheduling decoding, and efficiently reduce the decoding complexity [17], [18]. The shuffled scheduling decoding [17], [18], as one kind of serial scheduling decoding, is a VN-based algorithm. To further improve convergence speed, the column-weight-based (CWB) shuffled scheduling was proposed in [18]. The flooding and original shuffled decoding are scheduled at a fixed order of the LDPC code without requiring extra storage [19]. But the CWB shuffled scheduling needs extra storage to store the scheduling order with higher control complexity, which is not friendly to hardware implementation. The question is whether it is reasonable to lower decoding complexity at the price of storage. That is, is it a good solution to optimize performance in one aspect by sacrificing another aspect performance to only optimize the decoding of the system? To our knowledge, this question has not been answered.
The main contributions are summarized as follows. 1) A novel collaborative design framework based on joint coding and decoding optimization is proposed for the DC-BICM system over the multipath Rayleigh fading channels. In this framework, the property of the decoding algorithm is used to guide code design, thereby avoiding drawbacks from the one-sided optimization.

The main idea of the collaborative design is that some properties of the decoding algorithm can be used to optimize the code design so as to avoid performance loss from the decoding side. The specific process considers the characteristic of the DC-BICM system. The code design can be regarded as an optimization problem with the objective function considering the column-weight distribution, expressed by

1) Low storage requirement: In hardware implementation,

the CWB shuffled decoding algorithm needs to restore the

scheduling order of the VNs with degrees in the high-to-

low order. When the column weight distribution of the code

adopted has been designed in the descending order, the storage

can be saved in the CWB shuffled decoding algorithm.

2) Low decoding threshold: A good protograph base matrix,

which has a low decoding threshold, includes a high-degree

VN with punctured, several degree-2 VNs, and one or more

degree-1 VNs (also called precoder) [11].

3) Linear minimum distance growth: The linear mini-

mum distance growth (LMDG) property is directly related

to the maximum number of degree-2 VNs, Nd2, in the base matrix [11], which can be calculate as Nd2 = mB − mp − 1, where mp represents the number of degree-1 VNs.
4) Optimization space and search complexity: In the code

design, the size of the base matrix determines the optimization

space. The design process needs to consider the balance

between the optimization space and the search complexity.

V. CONCLUSION
In this letter, a novel collaborative design framework is proposed for the DC-BICM system. In this framework, both coding and decoding are jointly considered to optimize the system performance. Firstly, the CWB shuffled scheduling decoding is applied to optimize the decoding complexity for the DC-BICM system. To analyze the performance of the shuffled-based DC-BICM system, a CWB-EXIT algorithm is

developed. By using the CWB-EIXT algorithm, a low-storage
P-LDPC code is designed for the shuffled-based DC-BICM
system, and the order of the column-weight distribution is
taken as an important constraint in the collaborative design process. Through this process, the storage space of size O(N ) can be saved in the CWBHL shuffled scheduling decoding. The simulation results show that the proposed scheme can
perform better in the terms of decoding complexity, storage
and performance than the counterparts. These demonstrate
that collaborative design can overcome the limitations of the
conventional one-sided optimization.
