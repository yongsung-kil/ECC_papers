FPGA Implementation for LDPC Decoders Using A Novel Memory Effective Decoding Algorithm

Hongyuan Li, Zhenghong Yu*, Wanjun Zheng, Haijie Feng, Ziqian Ma College of Robotics Guangdong Polytechnic of Science and Technology,
Zhuhai 519090,China *Corresponding author: gditlhy@163.com

Abstract—In this paper, a novel memory effective algorithm for decoding Low-Density Parity-Check (LDPC) codes is proposed with a view to reduce the implementation complexity and hardware resources. The algorithm, called check node selfupdate (CNSU) algorithm, is based on layered normalized minsum (LNMS) decoding algorithm while utilizing iteration parallel techniques to integrate both variable nodes (VNs) message and aposterior probability message into the check nodes (CNs) message, which eliminates memories of both the variable node and the aposterior probability message as well as updating module of aposterior probability message in CNs unit. Based on the proposed CNSU algorithm, design of partially parallel decoder architecture and serial simulations followed by implementation on the Stratix II EP2S180 FPGA are presented. The results show that the proposed algorithm significantly reduces hardware memory resources and chip area while keeping the benefit of bit-errorrate (BER) performance and speeding up of convergence with LNMS.
Keywords-low density parity check (LDPC); decoding algorithm; FPGA implementation; memory effective
I. INTRODUCTION
Error correction codes (ECCs) are widely applied in modern wireless communication systems to reduce the signalto-noise ratio (SNR) required to reliably send data through a noisy channel, which introduce redundancy that can be used by a decoder to correct errors. In the past few years, significant progress has been observed in ECCs, for example, turbo-like codes, LDPC codes, polar codes, and fountain codes. In particular, owing to excellent error-correction performance and capacity approaching capability, LDPC codes have been widely adopted in many communication standards and applications, such as IEEE802.16e, IEEE802.11ad, IEEE802.11n/ac/ax, fiber optical communications, storage systems, deep-space communication systems, quantum key distribution and ATSC3.0. Recently, LDPC codes are adopted in the enhanced mobile broadband (eMBB) scenario in the 5th generation mobile networks (5G)[1-6].
Typically, LDPC decoding algorithms take iterative message passing manner, where the decoding messages are iteratively exchanged between VNs and CNs in each iteration. Among numerous decoding algorithms, the belief-propagation (BP) algorithm could provide the best decoding performance but it is hardly used in practical due to the high computation complexity. By simplifying the CNs-update function or reordering the CNs-update and VNs-update, min-sum algorithm significantly reduces the decoding complexity while layered algorithm reduce the required number of decoding iterations. However, the min-sum algorithm suffers from a

considerable performance loss, therefore, offset min-sum and normalized min-sum (NMS) algorithms were proposed , which greatly reduce the performance loss by involving a correction factor. The LNMS decoding algorithm, which combine NMS algorithm with layered algorithm, is proposed to achieve both the reduction of complex computations and the benefit of BER performance and speeding up of convergence[7-9].
However, the decoding algorithm involved in LDPC decoding requires extensive memory and calculation, and the memory consumes majority of the power. Hence, reducing the memory is necessary for low power decoder. Numerous studies[10-21] have focused on memory optimization. For example, researchers [10-11] used a layered algorithm to avoid variable node message storage, thus saving memory. The splitrow algorithm is proposed in research [12], which reduces intermediate message storage. Based on the advantages of optimizing the algorithm, studies have focused on other aspects of memory use. Researchers [13] adopted the check node message compressing storage to reduce the memory. Nonuniform quantization is utilized [14] to achieve a balance between performance and memory resources. The decoders above improve their parallel degree of the CNs unit to increase performance. This horizontal parallel strategy allows the decoder to update the message of more nodes simultaneously, which shortens the iteration. However, although throughput is improved, memory can not be decreased further.
Iteration parallel, on the other hand, can reduce the memory effectively without sacrificing the performance of the decoder. A mathematical rule on the iteration is utilized to merge several calculation steps into one. The architecture of CNs unit can be simplified to effectively optimize the operation and cut down on power consumption. In this paper, a novel check node selfupdate algorithm based on the traditional layered min-sum (LMS) algorithm and the idea of iteration-parallel is proposed. This novel iteration-parallel-based algorithm can eliminate memories of both the variable node and the posterior information as well as updates the module of posterior information in CNU. Also, it takes advantage of the check node information to compress storage to significantly reduce the decoder memory and the power consumption without losing performance.
The structure of this paper is as follows: the second part introduces deduction of the CNSU algorithm; the third part introduces design of the proposed LDPC decoder; the fourth part provides results of simulation and implementation; and the final part draws a conclusion.

Both the VNs message and the aposterior probability message in the LDPC decoding are integrated into CNs message without depreciation in performance. As a result, the algorithm eliminates the storage of VNs message and a-posterior probability message, and the usage of memory is reduced significantly.

Row degree of the LDPC Check Matrix is 6 and the column degree is 3, which means the matrix has six ‘1’ in every row and three ‘1’ in every column.

The control module is the center of the decoder. It controls other modules according to the timing requirement and provides the read-write address message of the memory in different periods. It also controls input of the external soft message in the initialization and output of the code word.
The CNs message memory module is used to store the logarithm likelihood message. With 6 bits quantization scheme, each row of CNs message needs 19bits by using the memory compressing technology. The 85.5Kbits CNs message memory module is composed of 9 dual-ports Block RAM, the depth and bit-width of which is 256 and 38 bits.
The intrinsic message memory module is used to receive the quantized soft message from the channel. In this LDPC decoder, it is composed of 36 single-port Block RAM with 256bits depth and bit-width of 4bits.
CNU is the key module of the whole LDPC decoder. Based on the iteration parallel, the proposed CNU is different from the traditional one that designed with LNMS algorithm, as showed in Figure 6 and Figure 7.
The read network is used to choose the specific intrinsic and CNs message.

IV. RESULTS OF SIMULATION AND IMPLEMENTATION The proposed algorithm maintains the benefit of BER performance and speeding up of convergence of the LNMS algorithm. With the same quantification strategy, Figure 8 shows a comparison among the decoding performance of the proposed algorithm, the traditional BP algorithm, and the LNMS algorithm. The proposed algorithm performs as well as LNMS, and loses only 0.05 dB,@BER=10-6 than the BP algorithm.
Figure.8. The decoding BER performance of LDPC decoder
The designed LDPC decoder is implemented using Verilog HDL base on the Stratix Ċ EP2S180 FPGA. The Top-level interface and simulation result, which can been seen in Quartus II and ModelSim SE, are respectively shown as Figure 9 and Figure 10.
In the implemented LDPC decoder, on account of reduction of the a-posterior probability message memory, the proposed CNSU algorithm reduces by 20% memory compared with the LNMS algorithm.
The memory, if using PingPong structure in the intrinsic message memory to guarantee the decoding continuity and high throughput, is 157.5 Kbits. Without using the PingPong structure, the memory usage is 121.5 Kbit.
The decoder in paper [21], which has been taped out and found a 0.23 mm2 logic area and 8.32 mm2 memory area by Design Compiler, is designed with a LMNS algorithm. Compared with LDPC decoder in paper [21], whose Memory usage is 197 K, the proposed LDPC decoder in this paper reduces memory by 20%, which can also reduce chip area. significantly.

V. CONCLUSIONS In this study, base on layered normalized min-sum (LNMS) decoding algorithm and utilizing iteration parallel techniques, a novel CNSU algorithm is proposed. The proposed CNSU algorithm eliminates memories of both the variable node and the a-posterior probability message as well as updating module of a-posterior probability message in CNs unit by integrating both variable nodes (VNs) message and a-posterior probability message into the check nodes (CNs) message. Based on the proposed CNSU algorithm, design of partially parallel decoder architecture and serial simulations followed by implementation on the Stratix Ċ EP2S180 FPGA are presented. The results show that the proposed algorithm significantly achieve the reduction by 20% hardware memory resources and chip area while keeping the benefit of BER performance and speeding up of convergence with LNMS algorithm.
