Piecewise Linear Interpolation based LOG-BP algorithm for 5G LDPC codes
Bin Xu, Yin Xu, Dazhi He, Hao Ju, Na Gao, Wenjun Zhang Cooperative Media Network Innovation Center (CMIC), School of Electronics Information and Electrical Engineering, Shanghai Jiao Tong University, China Emails: {zjfh20150911, xuyin, hedazhi, juhao237, gaonaa, zhangwenjun}@sjtu.edu.cn,

AbstractвҖ”BP algorithm is one of the most prominent decoding algorithms based on message passing. However, it is not commonly adopted in hardware implementation for the high computational complexity. The normalized min-sum (NMS) algorithm was then proposed to solve this problem but with performance flaws. To satisfy the comprehensive requirements of complexity and accuracy in 5G systems, we introduce an approximated algorithm based on BP in log-domain and name it piecewise linear interpolation based LOG-BP (PLI-LOG-BP) algorithm. In our novel algorithm, the non-linear mapping function in the horizontal step of BP is replaced by several linear functions for simplification. Meanwhile, we also propose an MMSE-based selection algorithm of segment points that improves the approximation accuracy. To verify the validation of the proposed algorithm, we apply 5G LDPC codes in our simulation and obtain BLER performance results. According to these results, PLI-LOG-BP algorithm decoding performance is significantly better than the NMS algorithm and even almost as good as the BP algorithm.
Index TermsвҖ”LDPC, BP, NMS, PLI-LOG-BP, MMSE based, 5G, BLER
I. INTRODUCTION
Low-density parity-check (LDPC) is a kind of linear block code with the sparse parity-check matrix. Since first introduced by Gallager in 1962 [1] and rediscovered by MacKay and Spielman et al in 1990вҖҷs [2], [3], LDPC code has captured worldwide attention for its Shannon-limit-approaching performance. With the development in recent years, it has been widely applied into different communication systems and standards such as DTTB [4], WLAN (IEEE 802.11n), WiMAX (IEEE 802.16e) [5] and ATSC3.0 [6].
The 5th-Generation (5G) wireless system has been gradually equipped in our daily life. Compared with 4G, 5G system accomplishes an all-around improvement. It provides enhance mobile broadband (eMBB) [7], massive machine type of communication (mMTC) [8] and ultra-reliable low latency communication (URLLC) [9] to satisfy various applications and soaring communication demand. As one type of the forward error correction (FEC) codes in 5G, LDPC code provides reliable support for precise communication. Different

Though iterative algorithms can achieve an error performance close to the Shannon limit, the cost of computation resources is exceptionally high. The belief propagation (BP) algorithm is proposed in [12] based on Bayes rules but is not suitable for hardware. Then some LLR BP-based algorithms are introduced to reduce the computation requirement of BP [13], [14]. For further simplification, Fossorier invented the min-sum (MS) algorithm [15], replacing exponential calculation with comparison operations while sacrificing the performance because of its over-estimation. Normalized min-sum (NMS) algorithm [16] is later proposed to revise this overestimation with a multiplicative correction factor.
This paper introduces an improved BP algorithm for binary LDPC code in log-domain and names it piecewise linear interpolation LOG-BP (PLI-LOG-BP) algorithm. In the traditional BP algorithm, the mapping function in horizontal steps is non-linear and complex. By replacing it with piecewise linear functions in several intervals, the approximation significantly reduces the complexity and makes it easier for hardware implementation. To further improve accuracy, we also propose a selection algorithm of segment points based on the minimum mean squared error (MMSE) criterion. Owing to this algorithm, we can obtain a function approaching enough to the original.

Simulation results of BLER performance of PLILOG-BP, BP, NMS algorithms are shown in section IV. The conclusion is in section V.

Although the LOG-BP algorithm reduces the resources consumed in multiplication, it still has massive exponential and logarithmic operations. We need to further simplify its approximation.

V. CONCLUSION
In this paper we mainly introduce the derivation and application process of the PLI-LOG-BP algorithm to simplify the horizontal step in traditional BP algorithm. The PLILOG-BP algorithm significantly reduces the complexity by replacing the non-linear operations with linear operations. Besides, to achieve enough accuracy, we propose an MMSEbased selection algorithm to determine the segmented points.
Compared to the BP algorithm, our novel PLI-LOG-BP algorithm costs lower than 20% calculation resources while provides a close performance for 5G LDPC codes, which is obviously better than the NMS algorithm. Under different code rates and sub-matrix sizes, PLI-LOG-BP can maintain a stable performance. In addition, the replacement by linear operation leads to easier hardware implementation. Both circuit area and storage size are optimized compared to the original. For 5G communication, which requires higher throughput and lower latency, the PLI-LOG-BP algorithm is a component to carry out.
