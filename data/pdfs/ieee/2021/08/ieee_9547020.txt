MET-DE Aided Design of Low-Rate DTMB-A LDPC Codes
Jian Song1, Zhitong He2, Kewu Peng3, Chao Zhang4 Department of Electronic Engineering, Tsinghua University, BNRist, Beijing 100084, China
1, 3, 4Peng Cheng Laboratory, Shenzhen, 518055, China Email: 1jsong@tsinghua.edu.cn, 2hezt20@mails.tsinghua.edu.cn, 3pengkewu@tsinghua.edu.cn, 4z c@tsinghua.edu.cn

Abstract—DTMB-A adopts LDPC codes with three code rates 1/2, 2/3 and 5/6, and low-rate LDPC codes are needed by DTMB-A to support scenarios which require low spectral efficiency and carrier-to-inference ratio. In this paper, a potential design of low-rate DTMB-A LDPC codes is presented. The newlydesigned LDPC codes employ a lower-triangular Raptor-like base matrix, which is carefully optimized by multi-edge-type density evolution (MET-DE). A more detailed cycle classification method is proposed to mitigate the error floor of the newly designed codes, while progressive lifting is employed to keep the feature of length-scalability. BER and BLER simulation results are presented, which demonstrates that the newly-designed lowrate DTMB-A LDPC codes have better performance than ATSC 3.0 Type-A LDPC codes with similar code lengths, within both waterfall region and error floor region, and over both BI-AWGN and Rayleigh fading channels.
Index Terms—DTMB-A, multi-edge-type density evolution, error floor, cycle classification, progressive lifting
I. INTRODUCTION
Digital terrestrial television multimedia broadcasting advanced (DTMB-A) [1], [2], as the new digital terrestrial television broadcasting (DTTB) system proposed by China, have been accepted by International Telecommunications Unions (ITU) [3] as the international DTTB standard in July, 2015. The cascade structure with outer code and inner code is used as the capacity-approaching FEC coding of DTMB-A, and the inner code is LDPC codes with three code rates of 1/2, 2/3 or 5/6 and two code lengths of 61440 bits or 15360 bits, respectively.
Nevertheless, various emerging applications have posed great challenges for the FEC coding schemes in future DTTB systems. As an example, WiB [4], [5], as a new system concept for DTTB, requires relatively low spectral efficiency and Carrier-to-Interference ratio (C/I) close to 0 dB. Hence low-rate DTMB-A LDPC codes is necessary to be designed for DTMB-A to meet these requirements.
Recently, Raptor-like structure [6], as a new multi-edgetype (MET) LDPC structure with simple encoding structure and capacity-approaching performance at low code rates, has been successfully applied in many communication standards, such as Type-A LDPC codes of ATSC 3.0 [7] and LDPC Codes of 5G new radio (5G-NR) eMBB data channel [8]. However, there still exist several shortages in these designs of Raptor-like LDPC Codes. For instance, ATSC 3.0 TypeA LDPC Codes are independently designed over two code

lengths, without the feature of length-scalability. Constrained by its lower-triangular Raptor-like structure, ATSC 3.0 TypeA LDPC Codes have error floor under low bit error rate (BER), and outer BCH codes are required to lower the error floor. Under the constraint of compact base graph and the feature of rate-adaptability and length-scalability, the decoding performance and error floor performance of 5G-NR LDPC codes are also restricted.
In this paper, a potential design of low-rate DTMB-A LDPC codes is presented, which could be separated into the following two steps. At the first step, the lower-triangular Raptor-like base matrix is employed to reduce the encoding complexity and increase the minimum distance, and the base matrix is optimized by MET density evolution (MET-DE) as is proposed in [9]. At the second step, progressive lifting is employed to keep the feature of length-scalability, while a more detailed cycle classification method is proposed to mitigate the error floor during lifting. The design example of rate-1/3 LDPC codes is shown in this paper, and compared to rate-1/3 ATSC 3.0 Type-A codes with similar code length.
The rest paper is organized as follows. In section II, the design of base matrix is introduced. The progressive lifting and cycle classification method is presented in section III. The simulation results compared with ATSC 3.0 Type-A LDPC codes are shown in section IV. Finally, the conclusion is drawn in section V.

The base matrix of the newly-designed low-rate DTMBA LDPC codes is carefully designed to reduce the encoding complexity and increase the minimum distance. To be specific, compared to ATSC 3.0 Type-A LDPC codes, the base matrix of A is designed as a full-1 matrix with m1 = 3, to simplify the design of the pre-code while maintain the linear-distance property with the variable node degree not less than 3. The base matrix of C is designed as a full-1 lower-triangular matrix rather than a dual-diagonal matrix to increase the variable node degree within the pre-code, while the rightmost column in the base matrix of E is designed full of 1, to mitigate the error floor caused by the inevitable degree-1 variable nodes within the pre-code. Hence the design freedom lies in the Luby code HL despite the rightmost column, which is assumed quasiregular and optimized by MET-DE which will be introduced in the next subsection.
B. MET-DE Aided Design
Density evolution can predict the asymptotic performance of the given degree distribution when code length tends to infinity. Traditional DE treat all the edges in the Tanner graph as the same edge type, which is called single-edgetype (SET). SET-DE has been widely employed to optimize the degree distributions and obtain the capacity-approaching irregular codes [12]. However, SET-DE is not accurate to analyze structural LDPC codes such as Raptor-like LDPC codes.
Different from SET-DE, MET-DE [13] treat the edges in the Tanner graph as several different edge types, and tracks the exchange of probability density function (PDF) of the information among all the edge-type combinations. METDE can optimize the multivariate degree distributions, whose variables denote different edge types. Hence MET-DE has more constraints than SET-DE and can predict the asymptotic performance of structural LDPC codes more accurately.

III. OPTIMIZED LIFTING
Given the optimized base matrix structure, the lifting parameters are of great importance for finite-length LDPC code design as well, especially for short code length. On the one hand, bad lifting parameters could lead to harmful trapping sets, and thus result in severe error floor. On the other hand, the decoding performance at the waterfall region of short LDPC codes would be affected as well. Hence the lifting parameters should be optimized.
In the optimization of the lifting parameters, the main target is to eliminate the most harmful trapping sets that lead to error floor. Since the analysis of trapping sets is complicated, the simplified method of eliminate the shortest cycles to improve the girth of LDPC codes has been widely employed. For irregular LDPC codes, cycles with the same length might make different contributions to the error floor performance, and the extrinsic message degree, which measures the cycle connectivity, is proposed in [14] to further classify the cycles with the same length. However, for MET-LDPC codes, the cycle classification methods mentioned above are still not precise enough.
In this paper, a more accurate cycle classification method for MET-LDPC codes, which can eliminate the most harmful cycles more efficiently. As shown in Fig. 2, the cycles are classified into Type-1 cycles, Type-2 cycles and Type-3 cycles. Type-1 cycles denote the cycles completely in the pre-code, which are the most harmful cycles due to the Raptor-like structure. Type-2 cycles denote the cycles across the pre-code and the Luby code, which are less harmful than Type-1 cycles but more harmful than Type-3 cycles. Moreover, progressive lifting technique is employed to reduce the search space and introduce the length-scalability, as 5G-NR LDPC codes. As an example, direct choice of lifting parameters with lifting size Z = 128 is hard, because the search space is so large that we have to employ random construction skills. Nevertheless, the lifting parameters with lifting size Z = 2 could be constructed first, whose search space could be traversed. Then we can progressively choose lifting parameters with lifting size Z = 4, 8, ..., 64, 128 which are compatible with previous lifting size. Owing to the relation between these lifting sizes, the search space would be largely reduced while the performance might be comparable with or even better than the direct random search case with lifting size Z = 128. IV. SIMULATION RESULTS
In this section, the performance of the newly-designed DTMB-A rate-1/3 LDPC codes with the optimized base matrix in section II and the optimized lifting factors in section III is shown. The lifting sizes Z = 32, 64, 128, 512 are chosen, which result in the code length n = 3780, 7560, 15360, 61440. The simulation results of rate-1/3 ATSC-3.0 Type-A long LDPC code with code length 64800 and short LDPC code with code length 16200 are provided as benchmark. Both AWGN and Rayleigh fading channel are considered and the iteration number is set 30 under layer decoding schedule.
As Fig. 3 shows, the newly-designed DTMB-A codes have better decoding performance than ATSC 3.0 Type-A LDPC codes with similar code lengths. Specifically, the newlydesigned DTMB-A codes show better error floor performance near code length 16k and better decoding performance over Rayleigh fading channel. Moreover, the newly-designed DTMB-A codes shows no evident error floor near code length 4k and 8k over both AWGN and Rayleigh fading channel. Hence the newly-designed DTMB-A rate-1/3 codes are demonstrated having low error floor and robust performance, over both AWGN and Rayleigh fading channels and over a wide range of code lengths.

V. CONCLUSION
In this paper, a MET-DE aided design of low-rate DTMBA LDPC codes is presented. The QC Raptor-like structure is chosen and slightly modified from ATSC3.0 Type-A LDPC codes to increase the minimum distance. MET-DE is employed to optimize the base matrix structure for better asymptotic performance. The progressive lifting is adopted for lengthscalability, while a more accurate cycle classification method is proposed to lower the error floor. The newly-designed DTMBA rate-1/3 codes shows robust performance and low error floor, over both AWGN and Rayleigh fading channels and over a wide range of code lengths, which demonstrate the effectiveness for the proposed design method.
