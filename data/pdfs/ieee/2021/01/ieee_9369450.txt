Dynamic Dual Quantization-Domain Self-Corrected LDPC Decoder

Taehyun Kim Network Business Samsung Electronics Co., Ltd. Suwon-si, Korea th8978.kim@samsung.com

JooSung Park Network Business Samsung Electronics Co., Ltd. Suwon-si, Korea jsjoo.park@samsung.com

Abstract—In this paper, we propose a low-density paritycheck (LDPC) decoder with a dynamic dual quantization-domain (DQD) that check nodes are operated in a 3-bit domain while variable nodes are operated in a higher bits domain. The mappings to transform messages from a domain to another domain gradually changes during the LDPC decoding process and is optimized by differential evolution and machine learning. In addition, for performance improvement, the variable node performs self-correction (SC) that the unreliable messages are erased. Simulation results show that the proposed dynamic DQDSC decoder has better error correction performance compared to a previously proposed hybrid DQD decoder.
Index Terms—low-density parity-check code, non-linear mapping, quantization, low-complexity, machine learning
I. INTRODUCTION
Dual quantization-domain (DQD) low-density parity-check (LDPC) decoder [1] was proposed for reducing the implementation complexity of the min-sum algorithm [2]. In DQD decoding, the check nodes operate in a 3-bit domain while variable nodes operate in a higher bits domain. To reduce performance loss due to low precision, a non-linear mapping that is optimized through differential evolution [3] is used to transform messages from a domain to the other domain.
In this paper, we propose a dynamic DQD self-corrected (DQD-SC) decoder. In order to achieve better performance than the previously proposed DQD decoder, dynamic mapping parameters and self-correction [4] are applied to the proposed decoder. The dynamic mapping parameters are used to consider varying log-likelihood ratio (LLR) during the decoding process. The mapping parameters for each iteration are optimized through differential evolution and machine learning. The self-correction is performed at variable nodes to improve error correction performance by erasing the unreliable messages propagated from variable nodes to check nodes.

To minimize the loss function, we use Adam optimizer [5] with learning rate= 10−3. After the scaling factor for the l-th iteration is optimized, the dynamic DQD-SC decoder performs one iteration with optimized mapping parameters for the l-th iteration to obtain the input of neural network for the (l+1)-th iteration.
At the variable node in the proposed dynamic DQD-SC decoder, self-correction method is used to improve error correction performance. If the sign of calculated L(Vl)2C (m, n) and the sign of LV(l−2C1)(m, n) are different, the information in LV(l)2C (m, n) is considered as unreliable and erased (i.e., LV(l)2C (m, n) = 0). If LV(l−2C1)(m, n) is zero, the LV(l)2C (m, n) is always updated to the calculated value regardless of its sign.

IV. CONCLUSION
A dynamic DQD-SC LDPC decoder is proposed in this paper. We use differential evolution and machine learning to optimize the mapping parameters. In addition, the selfcorrection is applied to the variable node for additional performance improvement. Simulation results show that the dynamic DQD-SC decoder outperforms the previously proposed hybrid DQD decoder.
