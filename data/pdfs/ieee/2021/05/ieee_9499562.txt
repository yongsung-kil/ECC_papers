A Low Complexity Parallel QC-LDPC Encoder

Abstract—In this paper, a low complexity parallel QC (quasicyclic)-LDPC (low-density parity-check) Codes encoding algorithm and encoder architecture is introduced. In traditional encoder architectures, the complexity of multiplication operation of information sequence and generator matrix is very high. The proposed encoding algorithm bit selection (BS) directly selects the valid information bits according to position set of non-zero elements in the first column of the sub-matrix, which avoids the operation of invalid information bits. The proposed parallel QCLDPC encoder architectures is realized for FPGA implementation based on CCSDS (7154, 8176) LDPC codes.
Index Terms— QC-LDPC, low complexity, BS, FPGA, parallel encoder.
I. INTRODUCTION
LDPC (low-density parity-check) codes [1] are a class of linear block codes with sparse parity check matrix H, which can achieve performance close to Shannon's limit. However, the complexity of LDPC encoding is essentially quadratic in the block length. To mitigate this problem, Chen et al. [2-3] proposed a special class of LDPC codes called QC (quasicyclic)-LDPC codes, for which parity check matrix is composed of circulant permutation matrix. The encoding complexity of QC-LDPC can be effectively reduced because the generator matrix and parity check matrix of QC-LDPC have cyclic shift characteristics [4-5]. At present, a large number of studies on low-complexity LDPC encoders have been conducted [6-10]. In [6], Li proposed three classical encoder architectures based on SRAA (shift register adder accumulator). Wei Yan et al. [8] proposed an efficient LDPC encoding scheme with low power and low parallel, the valid bit “1” among information bits is encoding only. But with the rapid development of satellite Internet and distributed star clusters, the demand for inter-satellite and satellite-ground data transmission is rising sharply. Due to the contradiction between the limited resources of space communication and the demand of high speed and large capacity communication, the research of high-performance signal processing and realization technology has important application value.
In this work, an area efficient parallel LDPC encoding scheme is proposed for QC-LDPC code. Similar to SRAA, the information sequence can be segmented firstly. Then, the encoding bits of valid information are selected according to position set of non-zero elements in the first column of the submatrix and the hardware complexity of the multiplication of

information sequence and generator matrix is reduced. Under the same parallelism, the logic AND operation is completely eliminated, and the number of logic XOR operation is reduced by about 50%. Finally, the proposed encoder architecture is implemented on the Xilinx platform based on CCSDS (7154, 8176) LDPC code, and the encoding complexity is moderately reduced compared with existing works.
The reminder of the paper is organized as follows: Section II briefly introduces the construction of QC-LDPC code and traditional SRAA encoder architecture. Section III details the proposed encoding algorithm and hardware implementation architecture, also algorithm complexity and implementation results are analyzed. Finally, conclusions are drawn in Section IV.

The serial SRAA encoder requires a complexity linearly proportional to the number of parity check bits, but the throughput is significantly limited that cannot meet the demand of high-speed data transmission. The parallel encoder does not work until the information bits are fully collected, which consumes a large number of registers. Two-stage encoder uses fewer logical resources but require that the parity check matrix must be full rank, which leads to application scenario constraints. Wei Yan proposed an improved parallel encoder based on SRAA, only the valid bit ‘1’ among information bits is coded. The encoding power is reduced but the encoding delay is variable because the number of valid bits is not determined.

IV. CONCLUSION
In this work, a low complexity parallel QC-LDPC encoder is proposed. The idea of the proposed algorithm is to select the valid information bits according to position set of non-zero elements in the first column of the submatrix, and then encoding. The operation of invalid information bits is avoided. By analyzing encoding complexity, the logic AND operation units is completely eliminated, and the number of logic XOR operation is reduced by about 50%. Implementation result of CCSDS (7154, 8176) LDPC code reveals that resources of the proposed encoder architecture are reduced compared with existing works, and the throughput can reach up to 5.84Gbps.
