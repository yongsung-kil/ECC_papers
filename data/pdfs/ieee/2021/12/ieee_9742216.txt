IV. DYNAMIC MAXIMUM ITERATIONS SCHEME FOR LDPC DECODING
As mentioned earlier, PDSCH data can be split up to 152 blocks, and it would be time consuming to perform LDPC decoding with a fixed maximum number of iterations for each block. Especially in the low SNR environment, the cha=el environment of code blocks with different frequencies is different. Some PDSCH code blocks can be decoded correctly before reaching the maximum number of iterations, while some code blocks ca=ot be decoded correctly even after using the maximum number of iterations. In fact, it is possible to properly decode code blocks of extra time to other code blocks used. It is even stronger requirement when the PDSCH retransmission occurs.
LDPC decoding selects 256QAM Modulation and Coding Scheme (MCS) in the highest speed code stream. When Modulation and Coding Scheme is 27, the fixed maximum iteration number can reach 30 times. However, when Modulation and Coding Scheme is 20, only 17 iterations can be achieved on average. If the first transmission of code block is wrong, the minimum number of iterations may be as low as 8 during the combine retransmission, and a lower maximum number of iterations will incur a performance penalty. When LDPC decoding load is high with high Modulation and Coding Scheme and large bit error rate, the dynamic maximum iteration number scheme can be used to improve the performance loss caused by insufficient maximum iteration number. It can reduce the parallelism demand of LDPC decoding, and get the benefits of lower cost and lower power consumption.
The basic operation mode of dynamic iteration number is that there is a maximum iteration number of high energy
I H and a maximum iteration number of low energy I L .
When time is sufficient, it uses the maximum number of
iterations I H ; Otherwise, uses I L .

VI. CONCLUSION
This article begins with a brief introduction of the LDPC encoding of PDSCH data. Then, a layered minimum sum algorithm with offset factor is implemented to decode LDPC. The problems are that the processing delay of fixed maximum iterations is too long in LDPC decoding, the semiconductor chip area of terminal equipment is too big, and power consumption is too high. A dynamic maximum iterations scheme is proposed. Finally, the simulation results compare LDPC decoding with fixed maximum iterations and dynamic maximum iterations. The bit error rate is close to each other. The whole process of the algorithm is easy to implement, which can provide a design idea for 5G wireless communication terminal equipments.
