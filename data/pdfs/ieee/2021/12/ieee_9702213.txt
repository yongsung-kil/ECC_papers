AbstractвЂ”BP decoding algorithm is one of the commonly 
used decoding algorithms for LDPC codes. To adapt LDPC 
codes to different 5G scenarios and further improve the 
decoding performance of short LDPC codes, a scheme 
combining model-driven deep learning with a traditional BP 
decoding algorithm is proposed. With the advantages of model-
driven, this solution expands the decoding iteration process 
between the check node and the variable node into a neural 
network and proposes a parameter normalization optimization 
solution to solve the problem of the program with many 
training parameters, the edge weights of the optimized Tanner 
graph are re-assigned and bound. Simulation results show that 
the proposed scheme can improve the decoding performance of 
LDPC codes with short lengths.

I. INTRODUCTION  
Low-density parity-check code (LDPC) code is a well-
known linear error correction code, first proposed by 
Gallager in 1962 [1]. Because of the excellent error 
correction performance close to the Shannon limit and high 
decoding throughput, it has become a new research hotspot 
[2]. So far, the Belief Propagation (BP) decoding algorithm 
is one of the main LDPC decoding algorithms. The BP 
decoding algorithm can obtain decoding performance close 
to the Shannon limit when the code length is very long. 
In recent years, deep learning has gradually been widely 
used in communication technology, and has made 
outstanding progress. Deep learning has a strong ability to 
learn and extract feature attributes, and is suitable for dealing 
with high-complexity problems. Researchers have gradually 
begun to use deep learning to solve the decoding problem in 
communication. Literature [3] assigns appropriate weights to 
the transmission information in the Tanner graph, which can 
reduce the iteration of the BP decoder with the same 
decoding performance. Literature [4] proposed a BP-CNN 
decoding model for the decoding problem in correlated noise 
channels. With the help of image noise reduction, the neural 
network is used to estimate the channel correlated noise to 
improve the decoding performance. With the rapid 
development of 5G systems, the demand for ultra-high 
reliability and low-delay communication is becoming 
stronger and stronger. For example, in smart grids and 
intelligent transportation systems, reliable transmission is 
required to be completed within 1~100 ms, and the block 
error rate is no more than 10-3. The short LDPC codes are 
one of the alternative codes of URLLC, which can meet the 
requirements of the application scenario of the URLLC, so 
the research on the decoding algorithm of the short LDPC 
code is meaningful. 
To adapt LDPC codes to different 5G scenarios, it is 
necessary to study the decoding performance of shortcode. 
Aiming at the decoding problem of short codes, in order to 
design a low-complexity LDPC decoder, this paper proposes 
a new neural network BP decoding structure. The model-
driven deep learning is combined with the BP decoding 
algorithm, the hidden layer information transfer of the neural 
network is used to simulate the iterative process of the BP 
algorithm, and the edge weights in the Tanner graph of the 
LDPC are re-assigned to improve the decoding performance. 
The rest of this paper is organized as follows. In Section 
II, we introduce BP decoding and the characteristics of 
model-driven neural network. Then in Section III, the 
specific steps of the proposed neural network decoding 
algorithm are introduced. The results and analysis are 
presented in Section IV.

B. Model-Driven Deep Learning  
Most deep learning is data-driven, characterized by the 
use of large amounts of data and black-box models to train 
the parameters of neural networks. But this type of model 
cannot deeply analyze the relationship between the 
algorithm and the network structure, so it is uninterpretable 
and unpredictable [8].  
The model-driven approach starts from the priori of the 
target mechanism, forms a learning cost function, and then 
minimizes the cost function to solve the problem [9]. 
Therefore, model-driven deep learning can use task-specific 
models and existing algorithm knowledge to design 
corresponding interpretable networks. In addition, model-
driven deep learning relies on less training data. In the field 
of communication, model-driven deep learning can build a 
network architecture based on existing communication 
models. At present, this method has been successfully used 
in communication, including the improvement of the 
channel decoding algorithm [10]. 
 
III. NEURAL BP STRUCTURE  
A new type of LDPC decoding network is proposed 
based on the BP decoding algorithm and model-driven deep 
learning. In this section, we explain the method and 
improvement of expanding the LDPC decoding algorithm 
into a neural network and give the definitions and formula 
expressions of several key parameters below. 
First, Fig. 1 is a block diagram of the proposed neural 
network-based decoder. At the transmitting end, the 
information bit x is encoded by the encoder and then 
modulated by BPSK, and then the information y is generated 
through the channel of additive Gaussian noise. At the 
receiving end, the decoder first calculates the LLR value of 
the received information and then inputs it into the neural 
network. The task of the decoder is to use the constructed 
neural network model to estimate the transmitted information 
bits.

neural BP network. 
The channel output value is accurately calculated by the 
channel LLR value module to obtain the LLR value 
information, and then decoded by the deep learning decoder 
module. The channel LLR value module corresponds to the 
accurate LLR layer in the decoding model. The proposed 
decoding network structure is shown in Fig. 2.

IV. NUMERICAL RESULTS 
The experiment in this section is based on the 
TensorFlow framework for training. The training data set is 
a codeword with a length of 256 and a rate of 1/2.  
The network training uses the mini-batch gradient 
descent algorithm, where the mini-batch size is 120, the 
Adam optimization method is used to search for the optimal 
network parameters with a learning rate equal to 0.03. 
Use the varying SNR to generate training data to test the 
robustness of the decoder. An SNR of 1dB means that the 
channel conditions used for data generation are very poor, 
and many errors will be generated in the data set, which will 
make the network unable to correctly learn the structure of 
the decoder, resulting in an inappropriate network. An SNR 
of 5dB means that the channel conditions for data 
generation are very good, and there is little error data 
feedback input into the network, which will make the 
network generalization ability poor and cannot eliminate 
noise correctly.

Fig. 6 and Fig. 7 are comparisons of the proposed BP 
decoding based on deep learning and traditional BP 
decoding. The results show that the performance of the BP 
decoding algorithm based on deep learning is better than the 
traditional BP decoding algorithm. When the bit error rate is 
10-3 in Fig. 6, compared with BP decoding, NNBP decoding 
can achieve a gain of about 0.9dB. 
Compared Fig. 6 with Fig. 7, it can be found that the BP 
decoding algorithm based on deep learning is more effective 
in high code rate codewords. This is because the rate will 
affect the number of short loops in the Tanner graph. The 
advantage of the BP decoding algorithm based on deep 
learning is that the re-assignment of the side weights of the 
Tanner graph reduces the influence of the short loop in the 
short codes decoding process, which can improve the 
decoding performance. When the code rate is 1/2 as shown 
in Fig. 6, the number of short loops with a code rate of 3/4 is 
reduced compared to Fig. 7, thus reducing the influence of 
the correction factor on the decoding performance, and 
therefore the performance improvement effect is small. 
V. CONCLUSION 
In this paper, a model-driven based decoding scheme for 
short LDPC codes is proposed. The BP decoding algorithm 
is combined with deep learning to improve the decoding 
performance. Using the advantages of model-driven, the BP 
decoding algorithm is extended to a neural network, and the 
learnable correction factors are trained. Aiming at the high 
complexity problem caused by many training parameters in 
the scheme, a parameter normalization optimization scheme 
is proposed. The simulation results show that, compared 
with the traditional BP decoding algorithm, the BP decoding 
algorithm based on deep learning achieves better decoding 
performance with less complexity increase. 
In future research, we will further explore the influence 
of short cycles in the Tanner graph of LDPC codes. We plan 
to compare and analyze the performance of NBP and NNBP, 
analyze the impact of this algorithm on short codes of 
different lengths, and calculate the complexity.
