A Robust Uplink Transmission Scheme for Interactive Services in Future Broadcasting Systems

Abstract—A dedicated return channel (DRC) for terrestrial broadcasting systems is considered to support interactive services. Compared with orthogonal multiple access based random access, non-orthogonal multiple access (NOMA) based random access technology is capable of supporting massive connectivity with higher reliability. In this paper, quasi-cyclic spatially coupled lowdensity parity-check (QC-SC-LDPC) coded interleave-division multiple access (IDMA) is proposed as a robust uplink physical layer solution for DRC in future terrestrial broadcasting systems. More specifically, QC-SC-LDPC codes, which have been recently proven to be universally capacity-achieving over different channels, are constructed for IDMA to exploit the inherent universality against the variations of spectral efficiencies, while the practical constraints of finite code length and number of decoding iterations are taken into account. Finally, simulations are carried out to verify the performance and robustness of the constructed QC-SC-LDPC coded IDMA scheme.
Index Terms—Spatially coupled low-density parity-check (SCLDPC) code, interleave-division multiple access (IDMA), dedicated return channel, terrestrial broadcasting.

Furthermore, Polar and Turbo codes have also been combined with Faster Than Nyquist signaling to improve the system performance against the inherent InterSymbol Interference [7]. Different from scheduled access, the user number J and sum spectral efficiency ηs vary over different slots in random access, and usually unknown at the transmitter side. The uncertainty of J and ηs poses significant challenges to the corresponding physical-layer scheme design.
Recently, quasi-cyclic spatially coupled low-density paritycheck (QC-SC-LDPC) codes have been recently proven to be universally capacity-achieving over different channels [8]– [10], which are promising to provide robust performance against the uncertainty of J and ηs. In this paper, QC-SC-LDPC coded IDMA is proposed as a robust uplink physical layer solution for DRC in future terrestrial broadcasting systems. QC-SC-LDPC codes are constructed for IDMA to obtain the robust performance at a wide range of ηs, which ensures the capability of accommodating random arrivals and ηs for IDMA based random access scheme.

The demand of interactive services is rapidly increasing in terrestrial broadcasting systems. With the users’ feedback, personalized services can be provided to enhance user experiences, and dynamic network planning can also be facilitated to maximize network capacity. To support the uplink transmission, a dedicated return channel (DRC) has been considered for existing terrestrial broadcasting standards, like Digital Video Broadcasting - Return Channel Terrestrial (DVB-RCT) [1] and Advanced Television Systems Committee (ATSC) 3.0 [2], where orthogonal multiple access (OMA) based random access schemes are adopted.
However, for conventional OMA based random access schemes, an outage may occur when more than one user contend for the same channel resource at the same time, leading to high outage probability for high user load. Meanwhile, by allowing multiple users to share the same time-frequency resources, non-orthogonal multiple access (NOMA) is naturally suitable for the collision resolution in random access. Thus, as a capacity-approaching NOMA technique, interleave-division multiple access (IDMA) [3] can be considered as a promising physical layer solution for random access in DRC.

At the receiver, chip-by-chip multi-user detection (MUD) is performed for each received symbol, and the output extrinsic information for each user, in the form of log-likelihood ratio (LLR), is de-interleaved and sent to the corresponding repetition and QC-SC-LDPC decoders. Otherwise, with a pseudo-random interleaver for all coded bits of each user, the QC-SC-LDPC code will behave more like a conventional LDPC code, and lose the universal advantage in IDMA system.

Then BSC is lifted with lifting factor Z to construct the parity check matrix, i.e., the zeros and ones in BSC are replaced by Z ×Z zeros matrices and Z ×Z circulant/general permutation matrices, respectively. Accordingly, the code length of QC-SC-LDPC code is nSC = NSCZ, and the uncoupled code length is nunc = NuncZ.

With sufficiently large L, w, nunc and number of decoding iterations TDEC, it has been proved that the threshold of QC-SC-LDPC code under belief-propagation decoding can saturate to the maximum a posteriori (MAP) thresholds of the uncoupled codes [8]–[10]. This implies the universality over different channels, including uplink multiple access channel, i.e., capacity-approaching performance could be achieve with a single QC-SC-LDPC code/ensemble. However, in practical systems, the total code length nSC is usually limited, and accordingly, L, w and nunc are finite. TDEC is also limited due to the computational-resource limitation and power-consumption consideration. Therefore, the above constraints are taken into consideration in this paper, and the QC-SC-LDPC code is constructed via two stages, the base matrix design and twostep lifting [14], [15].

IV. SIMULATION RESULTS
BLER simulations are carried out to demonstrate the robustness of the constructed QC-SC-LDPC coded IDMA scheme against different user numbers J and sum spectral efficiencies ηs. J-user Gaussian multiple access channel with BPSK modulation and repetition code rate of Rrep = 1, i.e., no repetition code, are considered for simulations, and the theoretical threshold SNRtheo with constellation-constrained input is calculated for each case as the reference.

Note that, since the SNR loss due to the rate loss for each case can be retrieved by increasing L, the gaps SNRsim − SNRtheo can be further decreased. Specifically, the SNR loss caused by rate loss varies from 0.21 to 0.50 dB for the cases of J = 1 to 9. According to the asymptotic analysis and simulations results in [6], [16], e.g., Fig. 10 of [16], conventional LDPC coded IDMA is capable of showing good performance at the target ηs, towards which the LDPC codes are optimized, but the threshold significantly degrades as ηs changes. In contrast, as shown in Fig. 4, good threshold performance can be achieved by the constructed QC-SC-LDPC coded IDMA scheme for a wide range of ηs, i.e., from 0.48 to 4.34 bits/chip. Furthermore, though repetition code is not considered for the simulations in this subsection, previous results [6], [16] have shown that the supported user number J of IDMA could be increased with the aid of repetition code, e.g., J could be quadrupled with Rrep = 1/4, while the threshold could be kept close to that without repetition code at equivalent ηs.
Moreover, for conventional LDPC codes, the average column weight usually needs to be relatively low for good EXIT matching at high ηs, and might lead to relatively high error floor, e.g., at BLER= 10−1 to 10−2 in [6]. Since this limitation does not exist for QC-SC-LDPC coded IDMA, better finitelength performance can be achieved as demonstrated in Fig. 3.

V. CONCLUSION
In this paper, QC-SC-LDPC codes are constructed for IDMA to support robust uplink transmission with different user numbers and sum spectral efficiencies. The practical constraints of finite coupling length, smoothing parameter, uncoupled code length and number of decoding iterations are considered in code construction. BLER simulation results show that good threshold performance can be achieved at a wide range of sum spectral efficiencies and different user numbers. This validates that QC-SC-LDPC coded IDMA is capable of accommodating random arrivals with varying sum spectral efficiencies for DRC in future terrestrial broadcasting systems.
