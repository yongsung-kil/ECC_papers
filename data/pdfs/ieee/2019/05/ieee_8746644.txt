Construction of Replacement Set in the GC-LDPC Codes Based on BIBDs

Wei Zhou, Lei Cheng, Lijun Zhang School of Electronics and Information Engineering
Beijing Jiaotong University Beijing, China
{16120194, chenglei, ljzhang}@bjtu.edu.cn

Abstract—In the GC-LDPC code, the replacement set plays an important role in the construction of the parity-check matrix H. Considering BIBDs with λ = 1, a replacement set can be structured which satisfies RC-constraint. The parity-check matrix H of the GC-LDPC code can be straightforwardly constructed from the replacement set. Simulation results show that the performance of BIBDs based CN-GC-LDPC code surpasses finite field based CN-GC-LDPC code.

Once the H matrix satisfies RC-constraint, the Tanner graph has girths of at least 6. In [9], a replacement set whose matrix can satisfy RC-constraint is a key component to construct an LDPC code. For t cyclic matrices of (q, n, g, r, 1)-BIBD, they satisfy RC-constraint, which leads to a new structure of the replacement set in the GC-LDPC code.

I. INTRODUCTION
Globally coupled low-density parity-check (GC-LPDC) codes were first proposed by Juane Li et al. [1]. For the GC-LDPC code, there are two kinds of global coupling structure. One is based on a group of check-nodes (CNs) to provide globally coupled connections and we call this code CN-based GC-LDPC code (CN-GC-LDPC). Likewise, the other is based on a group of variable-nodes (VNs) to provide globally coupled connections and we call this code VN-based GC-LDPC code (VN-GC-LDPC).
Due to the lower complexity, the construction of the GC-LDPC code has become the focus of researchers. Such as, in 2016, the GC-LDPC codes based on finite fields (FF) were proposed in [1] and in 2017, the GC-LDPC codes were constructed based on RS codes [2]. From [1] and [2], the simulation results show that these codes perform well.
Apart from the above algebraic constructions, balanced incomplete block designs (BIBDs) is an important class in combinatorial designs. Over many years, many classes of BIBD have been designed [3]. Suppose Z = (z1, z2, · · · , zq) is a set with q objects. A BIBD of Z refers to a set containing n g-subsets, denoted B1, B2, · · · , Bn. They satisfy the following structural characteristics: (1) each objects zi appears in exactly r of n blocks; and (2) every two objects appears in exactly λ of the n blocks. Hence, a BIBD is characterized by five parameters, q, n, g, r and λ, it is also called a (q, n, g, r, λ)-BIBD. A special case of BIBD with λ = 1 has been used to constructed the LDPC code [4-6]. In (1), if R is an m × n matrix and X is an s × nk matrix, Hgc,cn is an (mk + s) × nk matrix. III. REPLACEMENT SET BASED ON BIBDS
To ensure good performance, the replacement set of the GCLDPC code should satisfy the RC-constraint. And interestingly enough, t cyclic matrices of (q, n, g, r, 1)-BIBD also can satisfy RC-constraint. We have an idea that a replacement set of the GC-LDPC code can be constructed by t cyclic matrices. Next, we introduce two classes of the most commonly used BIBD with λ = 1 and the replacement sets based on two classes replacement sets. For other classes of BIBD with λ = 1, the reader can construct replacement sets based on them in the same way.

For Class-I Bose-BIBD, B(1), t base blocks are described Bi,0 = {0, α2i, α2i+4t, α2i+8t}, 0 ≤ i < t. We add 12t+1 elements of GF(12t + 1) to each base block Bi,0 and get t(12t+1) blocks Bi,j = {j, α2i + j, α2i+4t + j, α2i+8t + j}, 0 ≤ i < t, 0 ≤ j < 12t + 1. We can obtain 12t +
1 blocks, Bi,0, Bi,1, · · · , Bi,12t, which are called co-blocks of the base block Bi,0. These 12t + 1 blocks can form a translate class, Qi. The t(12t+1) blocks in the t translate classes, Q0, Q1, · · · , Qt−1, can form a Class-I (12t+1, t(12t+1), 4, 4t, 1)-Bose-BIBD B(1).

Form the j-th block Bi,j in the i-th translate class Qi of a Class-I Bose-BIBD with 0 ≤ j ≤ 12t, we design a (12t+1)tuple over GF(2), vi,j = (vi,j,0, vi,j,i, · · · , vi,j,12t), whose components correspond to the 12t+1 elements of GF(12t + 1), where the l-th components vi,j,l = 1 if l is an element in Bi,j, otherwise vi,j,l = 0. This (12t+1)-tuple vi,j,l is the incidence vector of the block Bi,j and it has weight 4. For 0 ≤ i < t, a (12t + 1) × (12t + 1) circulant Gi matrix is formed with the incidence vector vi,0 of the base block Bi,0 of Qi as the first column and its 12t downward cyclic-shifts as the other 12t columns. For 0 ≤ i < t, the column and row weights of Gi matrix are both 4. These t cyclic matrices form Class-I replacement set denoted by GI = [G0, G1, · · · , Gt−1].
B. Class-II Replacement Set
Similar to the previous section, t is a positive integer. 20t + 1 is prime and GF(20t + 1) is a prime field. Let α be a primitive element of GF(20t + 1) and the condition α4t − 1 = αc should be satisfied, where c is an odd integer less than 20t+1. Under this condition, over GF(20t + 1), these parameters t, c, and α will also be detailedly listed in Table I. Consequently, (q, n, g, r, 1)-BIBD can be constructed with q = 20t + 1, n = t(20t + 1), g = 5, γ = 5t, and λ = 1. For Class-II Bose-BIBD, denoted by B(2), t base blocks are described as follows: Bi,0 = {α2i, α2i+4t, α2i+8t, α2i+12t, α2i+16t}, with 0 ≤ i < t. We can get 20t+1 co-blocks from each Bi,0, where the j-th co-blocks is Bi,j = {j +α2i, j +α2i+4t, j +α2i+8t, j +α2i+12t, j +α2i+16t}, j ∈ GF (20t+1). The t(20t+1) blocks in the t translate classes, Q0, Q1, · · · , Qt−1, form a Class-II (20t+1, t(20t+1), 5, 5t, 1)Bose-BIBD B(2). According to the same theory, the column and row weights of Gi (0 ≤ i < t) matrix are both 5. These t cyclic matrices form Class-II replacement set denoted by GII = [G0, G1, · · · , Gt−1].
IV. CODES AND THEIR BEHAVIOR
Next, we will take advantage of Class-I replacement set and Class-II replacement set to construct the GC-LDPC code. As we know, progressive edge growth (PEG) algorithm have higher complexity, but the LDPC code by PEG algorithm has better performance [10]. Therefore, in each of the following examples, we use the LDPC code by PEG algorithm as the comparison.

V. CONCLUSIONS
The method for constructing the replacement set in the GCLDPC code is proposed. Class-I replacement set and ClassII replacement sets are obtained through GF(12t + 1) and GF(20t + 1), respectively. Considering BIBDs with λ = 1, two classes replacement sets satisfy RC-construction. The subset of two classes replacement sets can be used as local matrix even global matrix. In addition, a row of identity matrices can also be used as global matrix. Consequently, the paritycheck matrix H of the GC-LDPC code can be straightforwardly constructed. The behavior of the CN-GC-BIBD-LDPC code not only performs similarly to the PEG code, but surpasses the CNGC-FF-LDPC code over the AWGN channel. Compared with the PEG code, the CN-GC-BIBD-LDPC code and the VN-GCBIBD-LDPC code do not have flexible code length and code rate. However, owing to quasi-cyclic and global-coupling, the proposed method faces a promising perspective.
