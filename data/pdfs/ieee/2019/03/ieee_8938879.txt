AbstractÔÇö This paper presents the performance analysis of orthogonal frequency division multiplexing (OFDM) system using a mixed-scheduling belief propagation (MBP) algorithm for LDPC decoding where a double fading channels is considered. The MBP method is designed based on the LBP algorithm for the left-to-right direction (LBP-LR) and right-toleft direction (LBP-RL). Additionally, the bit error rate (BER) performance comparison of the BP, LBP, and MBP algorithms for LDPC decoding is proposed. The simulation results showed that the bit error rate (BER) performance of the MBP gave better BER than the BP and LBP algorithms, respectively.
KeywordsÔÇö OFDM system, LDPC decoding, MBP algorithm, LBP algorithm, BP algorithm, double fading.
I. INTRODUCTION
Orthogonal frequency division multiplexing (OFDM) system has been widely employed in wireless communication systems because of its high data rate transmission [1]. This system uses a multi-carrier modulation (MCM) technique in the parallel communication. It can make the available bandwidth at full capacity [2]. The OFDM system has been applied to different modulation formats, the Binary phaseshifted keying (BPSK) gives better error rate performances than other modulation methods, and fundamental performance have been evaluated [3]. However, this system is affected by the fading channel, which leads to data distortion and reduced system performance. The double fading channels are very interesting because of their applicability in satellite mobile communication [4], multiple input multiple output (MIMO) wireless communication [5], and mobile-to-mobile communication [6]. It is another important model to characterize the fading channel [7]. To improve the error rate

performance of OFDM system, Low-density parity-check (LDPC) code is essential.
LDPC codes have attracted much attention. LDPC codes were proposed by R. Gallager in 1962 [8], they are a kind of linear block codes defined by sparse parity-check matrix H, and the performance approaching Shannon limit likes Turbo codes. The belief propagation (BP) algorithm is a general iterative decoding algorithm used to decode the LDPC code [9]. The BP algorithm for LDPC decoder for OFDM system has already been studied in depth [10]. Many researches have focused on improvement of the convergence speed of LDPC decoding. Sequential scheduling also called the layered belief propagation (LBP) algorithm is also presented [11]. Recently, in [12], the mixed-scheduling belief propagation (MBP) algorithm has been proposed for magnetic recording systems. The MBP gives the fastest convergence speed and improves the bit error rate (BER) performance in the system.
In this work, we propose the LDPC decoders using MBP in the OFDM system with BPSK modulation as the used format. This method is adapted from the LBP algorithm in two directions, which consists of the left-to-right (LR) and right-toleft (RL) directions. The methods analyze the reliabilities from the sequential updates in both directions. The simulation results are compared, where the BER performances of the BP, LBP, and MBP algorithms are considered under double fading channels. The BER performances of the proposed algorithms show that the MBP achieves better BER performances than other algorithms in the OFDM system.

Fig. 3 shows the block diagram of the LDPC code in the OFDM system model, which is composed of the BP, LBP and MBP algorithms for LDPC decoders. The data flow of the block diagram is summarized as follows:
1) At the transmitter, information bits are encoded into codeword respectively by an LDPC encoder.
2) The encoded data are mapped into a sequence of modulation on N subcarriers which will be converted from serial into parallel (S/P).
3) The OFDM subchannel modulation is implemented by using an inverse fast Fourier transform (IFFT) for conversion into the time domain and inserted cyclic prefix (CP) by adding the copied part of the end of OFDM symbols to the front.
4) After that, the OFDM symbols are passed through the fading channel and additive white Gaussian noise is added.
5) The processing at the receiver is contrasted to the transmitter. In LDPC decoders, we use the different algorithm for LDPC decoding.

VI. COCLUSION
In this paper, we proposed a mixed scheduling belief propagation of LDPC decoder to improve the BER performance in the OFDM system. The MBP method was designed based on the LBP algorithm for the left-to-right direction and right-to-left directions. The simulation results showed that MBP gave better bit error rate performances than the BP and LBP algorithms for LDPC decoders in the OFDM
system. All algorithms for LDPC decoder had enhanced the
BER performance of the OFDM system under double fading channels, where the BPSK was adopted. However, LDPC coded in OFDM system gave better BER performance than uncoded in OFDM system, where the SNR > 10 dB.
