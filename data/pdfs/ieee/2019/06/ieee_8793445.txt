Capacity Enhancement of Asymmetric Multi-Level Cell (MLC) NAND Flash Memory using Write Voltage Optimization
Chatuporn Duangthong, Watid Phakphisut and Pornchai Supnithi Faculty of Engineering, King Mongkut’s Institute of Technology Ladkrabang,
Bangkok 10520, Thailand. {58601143, watid.ph, pornchai.su}@kmitl.ac.th

Abstract
In NAND flash memory, the number of bits per cell is increased to enhance the storage density, however, the reliability of NAND flash memory will be degraded. The current research only focuses on the read voltage optimization and advanced error-correction codes. The investigation of the write voltages optimization is limited. Since the write voltages significantly affect the channel capacity, we propose the write voltage optimization to enhance the channel capacity of MLC NAND flash memory. The differential evolution algorithm is applied to find the optimal write voltage that provide the maximum channel capacity.
Keywords: MLC/TLC NAND flash memory, write voltages, maximizing mutual information, asymmetric channel
1. Introduction
The NAND flash memory technology is broadly used in a vast number of data storage systems, including mobile phones and solid-state drives. To increase the density of NAND flash memory, the memory cell must store the multiple bits and rearrange its structure as 3D flash memory [1]. However, many noise sources are increased, making the unreliability of stored data [2-3]. To address these problems, in several works [4-6], the error-correction codes (ECCs) are considered to apply in NAND flash memory. The soft-decision decoding is required for advanced error-correction to achieve the channel capacity, for example, the belief-propagation algorithm of low-density parity-check (LDPC) codes. To obtain the soft information of NAND flash memory, the multiple reads of NAND flash memory is contributed in [7-9]. The multiple reads cause the quantized threshold voltage. Therefore, the read voltage optimization needs to be investigated [7-9]. In channel codes theory, the channel capacity leads the error-correction capability of ECCs. Alternatively, to enhance the performance of ECCs, the write voltage which significantly affect the channel capacity should be emphasized. Therefore, we propose the write voltage optimization to enhance the channel capacity of MLC NAND flash memories. By fixing the minimum and maximum write voltage, the optimal write voltages can be obtained by maximizing the mutual information. The simulation results show that

the capacity of optimal write voltages at high PE is greater than the fixed write voltages.
This paper is organized as follows. The channel model of NAND flash memory and the channel capacity are explained in section 2. The write voltage optimization by maximizing the mutual information is described in section 3. In section 4, we show the simulation results. Finally, we summarize our research work in section 5.

For the MLC NAND flash memory, we set the parameter as n = 0.00025(PE)0.62 ,  p = 0.05 ,

For the first simulation, the differential evolution algorithm is investigated. We observe the convergence of the differential evolution algorithm at 10,000 PE cycles. In the differential evolution algorithm, we set the number of write voltage vectors as 100 vectors. The parameter F and CR are set to be 1 and 0.5, respectively. 5. Conclusion
In this work, the write voltages are optimized to enhance the channel capacity of MLC NAND flash memory. We use the differential evolution algorithm to find the optimal write voltage. As a result, the capacity of optimal write voltages is greater than that of fixed write voltage; The capacity gain is 2.94% at PE = 50,000. By the differential evolution algorithm, the size of search space can be reduced from 9×108 to 4500 sets; The FER performance of optimized write voltage with maximized mutual information and minimized error probability are lower than that of the fixed write voltage at the high PE cycle.
