Improved Min-Sum Algorithm for Raptor-Like LDPC codes in ATSC 3.0

Hao Ju, Yin Xu, Genning Zhang, Dazhi He, Na Gao, Wenjun Zhang Cooperative Media Network Innovation Center (CMIC), School of Electronics Information and Electrical Engineering, Shanghai Jiao Tong
University, China Email: juhao237, xuyin,zhanggenning , hedazhi, gaonaa, zhangwenjun@sjtu.edu.cn

Abstract—In order to improve the performance of low-rate LDPC codes, ATSC 3.0 adopted raptor-like structure. Min-sum (MS) algorithm and improved versions can be used directly, but their performance on raptor-like LDPC codes are disappointing. In this paper, we introduce an improved min-sum(IMS) algorithm for raptor-like LDPC codes in ATSC 3.0. Simulation results of frame error rate show that the IMS algorithm not only has a noticeable gain in waterfall region over normalized min-sum (NMS) algorithm and offset min-sun (OMS) algorithm, but also has almost the same time complexity. Different from OMS algorithm, the proposed algorithm is not sensitive to the imperfect channel estimation. Meanwhile, the IMS algorithm is easier to realize than those variable factors based algorithms.
Keywords—raptor-like, LDPC decoding, min-sum, IMS, ATSC 3.0
I. INTRODUCTION
Low-density parity-check (LDPC) codes were first introduced by Gallager in the 1962 [1] and rediscovered in the 1990’s. In recent years, LDPC codes have been applied in plenty of systems, such as DTMB system [2], DVB related standards [3], Cloud Transmission systems [4], Wi-Fi (IEE 802.11n) [5], Wi-Max (IEEE 802.16e) [6].
LDPC codes can be specified by a sparse parity-check matrix, which makes the encoding and decoding more briefly, and special sparse parity-check matrix structure can bring benefits. Quasi-cyclic(QC) LDPC codes were proposed in [7] , in order to simplify the complexity of hardware implementation. Then irregular repeat accumulate(IRA) structure was introduced for QC-LDPC codes, which provides a linear-time encoding complexity [8]. However, the decoding performance of IRA structure on low-rate LDPC codes still has space to be improved. The raptor-like structure was proposed in [9], for purpose of better performance on low-rate LDPC codes.
The Advanced Television Systems Committee (ATSC) has proposed the next generation digital broadcast television standard, known as ATSC 3.0 [10]. ATSC 3.0 employed two structures, IRA structure for high-rate LDPC codes and raptor-like structure for low-rate LDPC codes [11].As we all know, the belief propagation (BP) algorithm [12] is one of the best performing iterative decoding algorithms for LDPC codes,

and its error performance is close to the Shannon limit [13]. However, with high computational complexity, BP algorithm is hard for hardware implementation. Then the min-sum (MS) algorithm was presented by Fossorier [14], which decreases the complexity but degrades decoding performance as a result of the over-estimation. Thereafter, normalized min-sum (NMS) algorithm [15] and offset min-sum (OMS) algorithm [16] were proposed to revise this overestimation and behaved well on regular LDPC codes. By the way, OMS requires perfect channel estimation, which can hardly be achieved in practice [17].To further improve the performance of MS algorithm, variable scaling min-sum(VMS) [18] algorithm was introduced, but the algorithm cannot fit irregular codes well. To address the problem, a measure based on generalized mutual information (GMI) [19] was provided, which requires lots of hardware resource.
An outstanding algorithm, named 2-D normalized MS (2-D NMS) algorithm with good performance on irregular LDPC codes, was proposed by Juntan Zhang [20]. For the raptor-like structure LDPC codes in ATSC 3.0, the performance of 2-D NMS algorithm can be improved.
In the paper [17], NMS algorithm and OMS algorithm are applied for ATSC 3.0 LDPC decoders, but the performance of NMS and OMS algorithm can be improved. In this paper, we introduce an improved min-sum algorithm (IMS) based on 2-D NMS, which is suitable for the decoding of raptor-like LDPC codes. We also compare the performance among different kinds of decoding algorithms when applied in raptor-like structure LDPC codes in ATSC 3.0, such as BP, MS, NMS, OMS and IMS. We aim to offer more choices for a better implementation of the ATSC 3.0 system, considering both performance and complexity.
The rest of this paper is organized as follows: in Section II, the structure of raptor-like LDPC codes are introduced briefly. Our IMS algorithm is proposed in Section III. Simulation results of the structure of raptor-like LDPC codes in ATSC 3.0 and the comparisons of different kinds of algorithms are demonstrated and discussed in Section IV. Finally, Section V concludes this paper.

The complexity of hardware implementation of IMS algorithm is obviously lower than that of other algorithms, which require factors changing with iteration. Achieving significant performance improvement compared to NMS algorithm, IMS algorithm only requires an extra multiplication operation in vertical and decision step. In addition, IMS algorithm only needs a small amount of space to store the selected set instead of requiring a large amount of degree storage like 2D-NMS.
Considering the simplification of implementation, IMS algorithm requires multiplication operations that are only up to 15 percent more than the NMS algorithm. And the storage space that IMS needs to increase is almost negligible. Consequently, IMS algorithm can be the common algorithm for raptor-like structure LDPC codes in consideration of the trade-off between performance and complexity.

V. SUMMARY
In this paper, we introduce an improved min-sum(IMS) algorithm with low complexity applied in raptor-like structure, which simplifies 2-D NMS and adds a judgment step in order to satisfy the raptor-like structure. Compared with IMS algorithm, we find that variable scaling with iteration has little gain in performance but a huge raise in complexity. In consideration of the trade-off of that, the simplified algorithm works better in practice. We also provide the performance comparison of some widely used LDPC decoding algorithms, such as BP, MS, NMS and OMS, with IMS. Simulation results of frame error rate(FER) show that IMS algorithm works obviously better than NMS and OMS, and has almost the same time complexity. In addition, considering about the performance of OMS algorithm in practice, IMS algorithm can output more stably good performance for raptor-like structure LDPC codes in ATSC 3.0.
