AbstractIn this paper the Bit Error Rate (BER) performance
of a Low-Density Parity-Check (LDPC) code operating over a
Middleton's Class A is improved by modifying the initialization
step of the classic Sum-Product (SP)decoding algorithm for
LDPC codes, implemented in its logarithmic form. The im-
provement is obtained by initializing the corresponding decoder
withinitialization values calculated using the pdf function of
the Middleton's Class impulsive noise channel, evaluated over
channel samples for the two possible values“0’and‘1'.This
procedure requires from the knowledge of channel parameters,
which can be estimated by different methods. The best BER
performance obtainedcorresponds to theinitializationdonewith
perfect knowledge of channel parameters.
Estimation methods considered in this paper are the Kanemo-
to, Zabin -Poor and Expectation-Maximization (EM) methods.
Wehave evaluated these estimation methodsbyusing the
estimatedvalues ofchannelparametersneeded tocalculatethe
values of theMiddleton's Class Aimpulsivenoise channelpdf,
for the two possible values ‘0’ and ‘1'. We have found that
none of the estimation methods utilized in its original form
provided good enough channel parameters estimations, to make
the corresponding decoder perform as in the case of perfect
knowledge of the channel parameters.
Averaging procedures have to be applied over the initial
estimatesobtainedwiththedifferentmethodstomakethe
mance. Among the different methods, averaging over estimations
provided by the Zabin-Poor method appears to be the best option
if implementation complexity and performance are both taken
into account.

1. INTRODUCTION
In this paper we propose an improvement on the BER
performance of an efficient error-control code operating over
an impulsive noise channel. Among the different options
for efficient error-control codes, like Turbo codes, LDPC
codes[1],wehaveselectedthelatter duetotheirexcellent
BER performance. The classic decoding algorithm for these
codes is the well known SP iterative decoding algorithm,
usually implemented in its logarithmic form. For a given
error-correcting code, the way the corresponding decoding
978-1-7281-2363-9/19/$31.002019IEEE
algorithm is implemented makes a difference with respect to
its BER performance. The proposed decoding algorithm shows
an improvement in BER performance when the initialization
step of the iterative SP decoding algorithm takes into account
the particular characteristics of the channel over which the
code operates. When the decoder makes use of this sort
of information,it is said that it operates with channel side
information (CSI).
Onthe other handmost of thewell-known error-control
techniques are usually studied under the presence of Additive
White Gaussian Noise (AWGN). This channel properly models
several communication scenarios present in practice. There
are however other real communications environments that are
characterized by other sources of noise, generated by natural
causes (e.g., lightning strikes) or by human activities (such as
vehicle ignition systems, home appliances, power transmission
lines,etc.).These channels are usually characterized by the
presence of impulsive noise that adds to the effect of the classic
AWGN. These channels are part of the so called non-Gaussian
channels. In the case of impulsive noise channels, one of
the best models is the Middleton's Class-A model [2], which
is especially suitable for modeling modern communication
systems like power line communications (PLC).
A.The coding technique:Brief description of LDPC codes
LDPC codes are a very efficient class of linear block codes.
H related to a generator matrix G, which is used to encode a
message vector m with k elements in a code word c, with
n elements.A word that belongs to the code satisfies the
syndrome condition H.c = o. This matrix satisfies certain
conditions that provide optimum performance in terms of BER
[1], [3], [4]. The aim of the decoding algorithm is to find the
vector d, considered as an estimation of the transmitted vector,
able to satisfy the condition H.d = O. The most commonly
used decoding algorithm for decoding LDPC codes is the
so called SP algorithm, usually implemented in a simplified
mode as in the Logarithmic SP iterative decoding algorithm,

The aim of ths paper is to implement a LogSP decoding
algorithm that is initialized with estimates of parameters of
the Middleton's Class A impulsive noise channel A and T,
by utilizing different estimation strategies. We make use of
three different methods for estimating Middleton's Class A
impulsive noise channel parameters A and F: the Kanemoto,
Zabin - Poor, and Expectation-Maximization Algorithm (E-
MA) methods. The better option seems to be the Zabin-Poor
estimator, for which the BER performance obtained is close to
that of an initialization with perfect knwledge of parameters
A and I, showing the least complexity in their calculation, in
comparison to other estimators.
In the following section we will describe different esti-
mation algorithms for determining the main parameters of a
Middleton's Class A impulsive noise channel, which will be
used at the initialization step of the decoding algorithm for
LDPC codes.

IHI. INITIALIZATION
LDPC codes are usually decoded using the SP algorithm
that in practice is implemented in its logarithmic form as the
LogSP decoding algorithm. The classic LogSP iterative decod-
ing algorithm for LDPC codes starts with an initialization step.
There are different alternatives for initializing the decoding
algorithms under study for the decoding of LDPC codes:
a) Initialization with statistical values corresponding to the
Gaussian Channel, independently of the channel over which
thecodeoperates.In this caseestimation of thevalue of the
noise dispersion o is required.
b) Initialization with statistical values corresponding to the
channel over which the code operates. (In this case statistics
based on the Middleton's Class A impulsive Noise channel,
its pdf, expression (1) and related parameters, have to be
determined). This is suggested in [12] for LDPC codes over
the Rayleigh fading channel.In this paper we propose a
BER perormance improvement for decoding LDPC codes over
the Middleton's Class A impulsive noise channel, based on
estimation of parameters of this channel, by using an improved
algorithm that combines different estimator algorithms like
those presented in section II. An averaging procedure is
necesary for obtaining reliable estimations of the channel, that
will be used for initializing the decoding algorithm for LDPC
codes.

noise channel with A = 0.1,F = 0.1 and M = 10. The
initialization step in the LogSP decoding algorithm is done
by using estimates provided by the mentioned estimators. The

 without averaging, provided satisfying

implemented with av = 100, for which the ideal BER perfor-
