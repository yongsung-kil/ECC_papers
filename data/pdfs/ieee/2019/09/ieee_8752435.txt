A Fast Approximate Check Polytope Projection Algorithm for ADMM Decoding of LDPC Codes
Qiaoqiao Xia , Yan Lin, Shidi Tang, and Qinglin Zhang

Abstract— Simplifying the Euclidean projection onto check polytope is an efficient way to reduce the computational complexity of alternating direction method of multipliers (ADMM) decoding algorithm for low-density parity-check (LDPC) codes. Existing algorithms for check polytope projection require sorting operation or iterative operation, which happens to be the most complex part of the projection. In this letter, a novel and fast projection algorithm is proposed without sorting and iterative operations. In the proposed algorithm, line segment projection replaces check polytope projection to approach approximate Euclidean projection at low computational complexity. Simulation results show that the proposed algorithm can substantially reduce the projection time while maintaining the frame error rate (FER) performance. In particular, the proposed algorithm can save the average projection time by 43% compared with cut search algorithm (CSA) when the dimension of the input vector is 20.
Index Terms— Alternating direction method of multipliers (ADMM), low-density parity-check(LDPC) codes, check polytope projection, line segment projection algorithm.
I. INTRODUCTION
L INEAR programming (LP) decoding was proposed by Feldman in [1] as an alternative decoding algorithm for LDPC codes. However, the development of LP decoding is limited by its high complexity. Recently, Barman proposed the LP decoding algorithm based on alternating direction method of multipliers (ADMM-LP) in [2]. Though ADMM-LP provides an efficient way to solve the LP decoding model, the decoding algorithm can be further simplified. There has been a flurry of research activities focusing on the computational complexity of ADMM-LP dominated by check polytope projection.
Currently, reducing the complexity of ADMM-LP algorithm can be considered from two aspects: reducing the number of projection and simplifying the projection operation. In [3], Wei et al. proposed an algorithm to avoid useless projection so as to reduce the computational complexity of ADMM-LP. Moreover, a main thrust has been to simplify the projection operation of a vector onto check polytope. In [4], Zhang et al. used cut search algorithm (CSA) to simplify the Euclidean projection. Subsequently, G. Zhang et al. proposed Euclidean projection algorithm based on the simplex [5]. Besides, Wasson and Draper proposed to combine CSA with simplex projection

More recently, Jiao et al. proposed to simplify the Euclidean projection by building look-up table and quantizing the input vector, which reduced the computational complexity of Euclidean projection at the expense of huge memory [7]. In [8], Wei and Banihashemi proposed an iterative check polytope projection algorithm which obtained the approximate projection by iterative operation. In this letter, a line segment projection algorithm (LSA) without sorting operation and iterative operation is proposed to approach approximate Euclidean projection. The complexity of the proposed algorithm is linear. What’s more, the algorithm can maintain the FER performance and convergence rate of the ADMM-LP decoder with exact Euclidean projection. It significantly reduces about 43% projection time that of CSA when the dimension of projection vector is 20. According to the simulation results, the saving of projection time becomes prominent as the dimension of the projection vector increases. Besides, the proposed algorithm refers simple arithmetic operations which are convenient for hardware implementation.

IV. THE PROPOSED ALGORITHM
An efficient computation of the approximate projection can dramatically reduce the complexity of check polytope projection. An example of such a simplification is to use the even-vertex (the vertex has an even number of 1s) closest to the input vector as the approximate projection (EVA, even-vertex algorithm ). However, the simulation results show that, EVA yields a poor FER performance. The precision of the approximate projection depends on the target FER and increases by decreasing the target FER [8]. It indicates that the approximate projection does not need to always maintain a high level of projection accuracy and the precision of the projection can be increased as the number of decoding iterations increases. Subsequently, we propose an efficient line segment projection algorithm without loss of the FER performance.

V. SIMULATION RESULTS
In the simulations, the additive white Gaussian noise (AWGN) channel with binary phase shift keying (BPSK) modulation is assumed. Moreover, the adopted codes are the regular (576, 480) rate 5/6 code C1, irregular (576, 432) rate 3/4 code C2, (576, 384) rate 2/3 code C3 from IEEE 802.16e standard [9] and regular (2640, 1320) rate 1/2 Margulis code C4. The check node degree of C1 − C4 is 20,{14, 15}, {10, 11} and 6, respectively.
We adopt ADMM-LP decoder with l2 penalty method. Penalty coefficient μ are set to 5.0, 4.2, 4.4 and 3.8 for C1−C4, respectively. Parameter α are set to 2.0, 1.8, 1.4 and 1.6 for C1−C4, respectively. Besides, the over-relaxation parameter is set to 1.0. The maximum number of iterations for ADMM-LP decoder is set to 80. The points plotted in all FER curves are obtained by generating at least 100 error frames.
Figure 1 shows the FER performance of C1 − C4 for the ADMM-LP decoder with CSA, EVA and the proposed algorithm. CSA obtains the exact projection so that the FER performance of CSA can be considered as the reference object. It is suggested that the FER performance of EVA is worse than that of CSA about 0.3 dB. However, the FER performance for the decoder with the proposed LSA at the simulated SNR regions is basically the same as that of CSA. VI. CONCLUSION
To summarize, we propose LSA for efficient implementation of approximate check polytope projections. On one hand, it avoids sorting operation and iterative operation. Using simple arithmetic operations can get approximate projection without loss of FER performance of decoder. Meanwhile, simple operations mean less resource required for its implementation in hardware. On the other hand, LSA can save the average projection time by nearly 7% to 43% which is determined by degree of check nodes. For the codes with 512-degree check nodes, the proposed algorithm can save the average projection time roughly 95%. It leads to a considerable reduction of the projection time.

Combining these results, we conclude that the execution time for each projection of LSA is shorter than that of CSA, especially for codewords with high check node dgree.
