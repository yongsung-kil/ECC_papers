Low Complexity Decoding Scheme of Raptor-like LDPC Code in Sufficient SNR Scenarios
Chao Zhang, Yin XuвҲ—, Na Gao, Dazhi He, Hao Ju, Genning Zhang, Yizhe Zhang, Yu Zhang and Wei Xie Cooperative Media Network Innovation Center (CMIC), School of Electronics Information and Electrical Engineering, Shanghai Jiao Tong University, China Academy of Broadcasting Science, NRTA, China

AbstractвҖ”In this paper, A low complexity decoding scheme of raptor-like low-density parity-check code (LDPC) in sufficient SNR scenarios such as in an over-engineered network or core layer of layered-division multiplexing (LDM), is proposed. This scheme is meaningful to emerging power-limited devices, such as IoT and handheld devices. Based on the proof of boundedness and convergence properties of raptor-like LDPC code, the proposed decoding scheme selectively skips some check node operations during iterative decoding. Furthermore, we propose a corresponding algorithm to choose a proper set of skipped check nodes, which ensures an acceptable performance with much lower complexity. Simulation results show that the proposed scheme is good enough to meet the performance requirements, while yielding attractive power benefits over conventional scheme. The decoding complexity can be reduced by up to 80% in the best case while the SNR is sufficient.
Index TermsвҖ”raptor-like, LDPC, sufficient SNR, decoding, SPA, NMSA, skipped check nodes
I. INTRODUCTION
Recently, the research on multimedia broadcast multicast services(MBMS) has been enhanced due to emerging applications such as venue casting and software updates [1]. To ensure normal operation of all receivers, traditionally, the broadcaster may transmit signals in an excessive mode to ensure that every user receives a sufficient signal without knowledge of the userвҖҷs carrier to noise ratio (C/N), which leads to an overengineered network [2]. As for multicast services in cellular network, a joint beamforming design algorithm is proposed in [3] for minimizing the total transmit power by using LDM. LDM may provide significant gains when the superposed signals exhibit large differences in terms of signal-to-noise ratio (SNR), which means the SNR of the core layer will be much larger than the threshold. Besides, the SNR at the center of the cell is much larger than it is on the edge [4]. In such scenarios mentioned above, the received SNR tends to be more sufficient than required. However, in practice, some devices such as handheld and IoT equipments are often quite power

This demand is more urgent in LDM due to its requirement of a limited increase in complexity [5]. Since the LDPC decoder occupies almost 50% chip area in ASIC of the receiver, it is an effective way to reduce the complexity of LDPC decoder.
The low density parity check (LDPC) code was first presented by Gallager [6] in 1962, which has been demonstrated that irregular LDPC codes can perform close to the Shannon limit. Recently, a novel structural LDPC code called raptor-like LDPC code [7] has been discovered, which has been adopted in both 5G and ATSC3.0 standards [8]. It has lots of degreeone variable nodes. Based on this characteristics, we propose a low complexity LDPC decoding scheme. Currently, the most popular LDPC decoding algorithms are belief propagation (BP) based algorithms, which include sum-product algorithm (SPA) [9] and its simplified algorithms, such as widely-used normalized min-sum algorithm (NMSA) [10]. These traditional BP-based LDPC decoding algorithms strictly transmit the messages between variable nodes and check nodes. Whereas, some transmissions in raptor like LDPC code may be omitted through our derivation.
In this paper, we firstly prove that in terms of raptor-like LDPC code, some messages from particular check nodes to core variable nodes are always finite during any iteration and the decoder will converge as long as core variable nodes converge. Based on this theoretical situation, we infer that some check nodes can be skipped appropriately to reduce the complexity with an acceptable performance when the SNR is much sufficient in practice. A novel low complexity decoding scheme for raptor-like LDPC code is thus proposed. Meanwhile, we propose a corresponding algorithm which can obtain a proper set of skipped check nodes for the proposed decoding scheme based on different SNR. Finally, we evaluate the performance of the proposed decoding scheme by numerical simulations and analyze their computational complexities.

II. BOUNDEDNESS AND CONVERGENCE PROPERTIES OF RAPTOR-LIKE LDPC CODES
In this section, we first introduce the structure of Raptor-like LDPC codes, and then prove its convergence and boundedness properties of some specific message transmissions. The results show that the decoder will converge as long as core variable nodes converge.

III. NOVEL DECODING SCHEME OF RAPTOR-LIKE LDPC CODES
Section II shows that the messages from check nodes that are connected to degree-one variable nodes to core variable nodes are bounded in magnitude during any iteration l. And the decoder will converge as long as core variable nodes converge when iteration l tends to infinity. According to this corollary, we can infer that some LLR messages from check nodes connected to degree-one variable nodes to core variable nodes may be omitted when the initial LLR is large. Since this is very similar to the scenario where the iteration tends to infinity. Therefore, it is feasible to skip some check node operations in sufficient SNR scenarios
In this section, firstly, a low complexity raptor-like LDPC decoding scheme which skips those unnecessary check nodes that are connected to degree-one variable nodes is proposed. Then, we propose a corresponding algorithm for choosing the set of skipped check nodes.

IV. NUMERICAL RESULTS
In this section, we analyze the performance and complexity of the proposed decoding scheme based on one of the raptorlike LDPC codes in ATSC 3.0, where LDM is regarded as a key technology [11]. During simulation, we transmitted over BI-AWGN channel by using QPSK modulation. The maximum iteration was set to 50 and at least 20 erroneous blocks were attained to eliminate contingency. And the proposed scheme was applied to both SPA and NMSA. The detailed parameters of this LDPC code are shown in Table I.
Since the proposed decoding scheme skips some unnecessary operations, it can reduce the complexity. Therefore, we need to compare the complexity of the traditional BP-based LDPC decoding scheme and the proposed low complexity raptor-like LDPC decoding scheme. The computational complexity of the SPA is calculated by applying [12]. In order to update a degree-dc check node in the SPA, 2dc look-up tables

(LUTs), 2(dc вҲ’ 1) exclusive-ors (XORs), and 2(dc вҲ’ 1) real additions (RAs) are required approximately. For an update in a degree-dv variable node, 2dv вҲ’ 1 RAs are required. As for NMSA, it is the same as SPA when updating a variable node. However, we need 2(dc вҲ’ 1) XORs and 2dc вҲ’ 3 comparisons (CPs) to update a check node.

V. CONCLUSION
This paper aims to reduce complexity of raptor-like LDPC code in sufficient SNR scenarios, which is meaningful to emerging power-limited devices or LDM system. We strictly proved that some check node operations of raptor-like LDPC code can be skipped. Because some messages from check nodes that are connected to degree-one variable nodes to core variable nodes are bounded during any iteration and the decoder will converge as long as core variable nodes

converge. We proposed a low complexity decoding scheme
for raptor-like LDPC code based on above conclusion. And
a corresponding algorithm for choosing skipped check nodes
was proposed based on different SNR.
The simulation results showed that the proposed decoding
scheme of raptor-like LDPC code and its corresponding factor
selection algorithm are good enough to meet the performance
requirements in sufficient SNR scenarios and reduce the complexity by up to 80% in the best case, compared with the
traditional BP-based decoding scheme.
