Offset min-sum Optimization for General Decoding Scheduling: A Deep Learning Approach
Ahmed Abotabl, Member, IEEE, Jung Hyun Bae, Member, IEEE, and Kee-Bong Song, Member, IEEE

Abstract‚ÄĒDeep learning has shown an unprecedented success in many fields such as computer vision and speech recognition, providing solutions to intractable problems. Deep learning has also provided solutions to several intractable problems in communications systems. This paper uses a deep learning approach to optimize the analytically intractable offset value in the offset-minsum (OMS) algorithm. OMS algorithm is a very attractive lowcomplexity algorithm that is used in the belief propagation decoding of linear codes. The contributions of this paper are: First, providing a low complexity offset optimization framework based on gradient descent and back-propagation on the original Tanner graph. Our proposed algorithm has comparable complexity and similar operation as the forward belief propagation decoding algorithm and hence, can be trained much more efficiently. Second, the framework can be easily extended to any decoding scheduling such as flooding or layered scheduling. Training results show that the proposed framework can find the optimal offset value under different decoding scheduling with the same complexity of the belief propagation algorithm.
I. INTRODUCTION
Deep learning was shown to provide many solutions to open problems in fields such as computer vision [1], speech recognition [2] and language processing [3]. Consequently, recent contributions in the communications systems physical layer have relied on deep learning in solving intractable problems [4] or forming a better understanding [5]. In order to realize the advantage of deep learning over the existing systems, one may realize that the real gain of deep learning appears in problems that are either analytically intractable or their optimal solution is too complex to be practical [6]. In this paper, we use the deep learning tools to optimize the decoding of linear codes such as low-density parity-check (LDPC) codes [7] under offset-min-sum algorithm [8] which, to the best of our knowledge, is a problem that does not have analytical solution.
Deep learning tools have been applied for the decoding of error correction codes. A fully connected neural network that is trained to perform the decoding of linear codes was presented in [9]. However, because the fully connected neural network does not recognize the structure of the code, it is necessary to perform the training over so many possible codewords and noise realizations which makes the problem infeasible for long codes. Further simplifications using the syndrome detection was applied to this method to reduce the training overhead [10] but it does not simplify the network structure which is way more complex than the traditional decoders such as the belief propagation decoder [11]. The authors of [13], [14] have used the same trellis architecture to learn the offset value of the OMS algorithm. The main drawbacks in this trellis architecture is first, the very high complexity since the number of nodes in the hidden layer is the same as the number of edges of the corresponding tanner graph of the code. Second, this trellis architecture performs the flooding scheduling only so it can not be extended to other decoding scheduling. Using deep learning ideas, similar approaches of building a partially connected neural network that recognizes the structure of the code have also been studied for polar codes [15], [16].
This paper provides a framework for optimizing the offset value in the OMS algorithm. Our approach is based on deriving the gradient via back-propagation of the belief propagation algorithm. This results in a neural network that is much simpler than [13], [14] since the number of nodes in each hidden layer is either equal to the number of variable nodes or check nodes. Another complexity reduction compared to [13], [14] comes from the resemblance of back-propagation to the belief propagation algorithm under the proposed framework. Due to the low complexity, the proposed framework does not require extensive processing tools such as GPU. It is also shown that our framework can be extended to optimize the offset value under any decoding scheduling such as flooding and layered scheduling. Even though the gradient calculations are presented in this paper for the additive offset optimization, using the same approach, the gradient calculations can be performed similarly for a multiplicative scaling parameter.

An unfolding of this procedure gives a partially connected neural network with number of layers depending on the number of iterations and number of nodes depending on the number of check nodes and the number of variable nodes. Remark 1: The gradient calculations under flooding or lay-

ered scheduling resemble a belief propagation algorithm. In

the back-propagation phase, the variable-to-check messages are ‚ąÜ(őĪl,)ij and the check-to-variable messages are ‚ąÜ(ő≤l,)ij. These
messages are calculated in a descending order of l starting with

l = L. This is a significant complexity reduction compared to

the conventional neural networks or even the networks in [13],

[14]. Moreover, because of this resemblance with the belief

propagation decoding, parallelism can be realized across the

different protographs of a QC-LDPC codes leading to faster

training.

Remark 2: For a numerical comparison between the com-

plexity of the proposed framework and [13], [14], consider the

trellis representation depicted in [13], [14] for the BCH(15, 11)

code. This trellis structure has a total of 800 edges for three

iterations while our framework constructs a network with the

same number of layers and a total of only 160 edges. Longer

codes are used in the NR standard, for instance, a QC-LDPC

code chosen according to base-graph 1 and a lifting size 384

and a code rate of 0.9167 has 26Z variable nodes, 4Z check

nodes and 76Z edges where Z is the lifting size. Assuming

10 iterations, the approach in [13], [14] would require a trellis

with about 6 √ó 106 edges in total while our approach requires less than 6 √ó 105 edges in total.

VI. CONCLUSION
Based on a deep learning approach, a framework for learning the offset value in the offset-min-sum algorithm is presented. The presented framework has a very low complexity and works under any decoding scheduling. Results show that the presented framework is capable of finding the optimal offset value with a belief propagation algorithm complexity. Different error functions that optimize the bit-error-rate and the block-error-rate were also presented. Simulation results show that optimizing the offset value results in a significant gain.
