An Unequal Error Protection of Quasi-Cyclic Low Density Parity Check (QC-LDPC) Codes Based on Combinatorial Designs

Bijaya Paudel College of Engineering, IT and Environment
Charles Darwin University Northern Territory, Australia e-mail: bijaya.paudel@cdu.edu.au

Sina Vafi College of Engineering, IT and Environment
Charles Darwin University Northern Territory, Australia e-mail: sina.vafi@cdu.edu.au

Abstract—This paper presents a novel method for constructing high performance unequal error protection (UEP) quasicyclic low-density parity-check (QC-LDPC) codes. These codes are highly efficient and designed with flexible rates, lengths and number of protection levels. The given QC-LDPC codes are based on the combinatorial design. Each circulant constituting the parity check matrix of the UEP code is determined from subsets with the distinct difference between elements. Simulation results indicate that proposed UEP codes have equivalent performance to UEP codes constructed with the higher girth and lower rate, while achieving superior performance than the codes constructed by conventional equal error protection (EEP) techniques.
Keywords-Forward Error Correction, Unequal Error Protection, QC-LDPC, COmbinatorial Design
I. INTRODUCTION
Recently, there has been an unprecedented growth in broadband multimedia services such as mobile television, video sharing and recorded systems. These services are generally bandwidth intensive and error sensitive. To have a proper delivery of these services to users, implementing an unequal error protection (UEP) technique is considered, which protects different parts of bitstream based on their importance [1]. This can be done by a forward error correcting (FEC) code, which is recognised as a highly efficient, lowcomplexity method for protecting the original information against the noise of channel. For the next generation of broadband transmission systems, it is vital to construct a FEC code, which satisfies reliability and low latency requirements of the transmission systems [2].
Quasi-Cyclic Low-Density Parity Check (QC-LDPC) codes are introduced as advanced and highly efficient FEC codes, which were considered in different global standards such as DVB-S2, IEEE 802.11n (Wi-Fi) and IEEE 802.16e(WiMAX) [3]. They are also a high candidate for the next generation of wireless networks including the fifth generation of mobile networks (5G). This is because of their high performance, rich algebraic structure, low design complexity and a capability to produce a high throughput for the network [4]–[6].

QC-LDPC codes are usually designed for equal error protection (EEP) of the information. However, there are different methods for their construction with UEP capability. Puncturing of codewords can be done, which removes certain number of parity bits from the original codeword and produces a new code with a higher rate [3], [7]. An advantage of puncturing is implementing one decoder for all codewords formed by different rates. However, many puncturing algorithms are complex and difficult to implement by the hardware [3]. Zhao et. al proposed a simple puncturing algorithm [7], but the bit error performance became significantly lower compared to the non-punctured codes with same rate [8]. For improving the performance of punctured codes, a technique based on progressive increment of information variable nodes and check nodes was suggested. It provided high performance for the QC-LDPC code [3]. However, this improvement engendered high complexity in the code design. Alternatively, UEP QC-LDPC codes can be formed by a masking technique conducted on its parity check matrix (PCM) [9]. By masking a selected number of submatrices of the PCM, the density of the main matrix is reduced resulting a code with unequal error correction capability. This technique required specific constructions of base and masking matrices, which should satisfy a number of conditions [9]. Furthermore, for high performance, constituent submatrices of the main PCM must correspond to the PCMs of the QC-LDPC code with good error correcting capability.
This paper proposes a scheme of UEP QC-LDPC codes based on combinatorial design. The circulants applied in the structure of PCM is designed from defined subsets, similar to [8]. The PCM has an ability to form multiple QC-LDPC codes with different rates. Considering this specification of the PCM, a modification is conducted in order to construct a PCM suitable for an UEP code. In comparison with previously proposed methods, both encoder and decoder are designed with low-complexity, without using any masking technique in construction of the code. Conducted simulations confirm that proposed UEP codes provide better performance compared to their EEP capabilities.

IV. SIMULATION RESULTS
Computer based simulations are conducted to verify the performance of proposed UEP codes. The type of channel considered for simulation is Additive White Gaussian Noise (AWGN) channel. The codewords are modulated by using Binary Phase Shift Keying (BPSK) and decoded by sum product algorithm (SPA) with 50 iterations.
For two levels of protection, (2620, 1572), (3730, 2638), (9890, 5934) codes with rate 0.6 are utilized. Out of 1572 message bits of (2620, 1572) code, 524 bits are protected with the high priority, while remaining are protected with low priority. The ratio of most important bits to less important bits is 1 : 2 for all three codes. Figure 1. shows the performance comparison of proposed (3730, 2638) and (2620, 1572) UEP codes with (3728, 1864) code presented in [9]. It is noticeable that both levels of protection have close performance to the UEP code constructed in [9]. In the waterfall region, the proposed codes exhibit better performance than (3728, 1864) code. It should also be noted that the rate of the code constructed in [9] is considerably lower than proposed codes. Therefore, the proposed (3730, 2638) code, which has almost same length as the (3728, 1864) code is able to transmit more bits, while providing better error protection. In addition, the (3728, 1864) code has girth eight, while the proposed codes have girth six.

V. CONCLUSION
In this paper, a new scheme of UEP QC-LDPC code with girth six was provided. The constituent circulants of the UEP code were based on combinatorial design. Simulations results indicated that these UEP codes achieved a significant improvement in the protection of high priority information than conventional EEP schemes. In addition, the flexibility of these proposed codes enable them to accommodate different needs for error protection in future broadband transmission systems. In order to further improve the performance of these codes for all levels of protection, their construction based on a girth greater than six is recommended.

Figure 2. Performance of the designed (9890,5934) UEP code.
Figure 2. shows the performance of the proposed (9890, 5934) code, which is compared with (9872, 4936) code constructed in [9]. The high protection level for both codes have similar performance. The performance of the low protection levelis slightly better for (9872, 4936) code. This is because (9872, 4936) code has girth eight and low rate compared to the proposed code. For similar bandwidth, (9890, 5934) code is able to transmit around 20% more bits compared to (9872, 4936) code.
For three levels of protection, (5958, 3972) code with rate 0.67 is utilized. In this case, 662, 1324 and 1986 bits are applied in the first, medium and lowest level of protection, respectively. The ratio between the number of bits in different levels of protection is 1 : 2 : 3. In Figure 3, the performance of this code is compared with EEP (5355, 4338) code and other EEP codes [12], [13]. The first and second level of protections significantly outperform the EEP code. At the BER of 10−6, the first level of protection outperforms EEP code by almost 2.0 dB. Similarly, the second level of protection provides a better performance than considered EEP codes, which are amongst the highest performing QC-LDPC codes. Therefore, it can be concluded that the proposed UEP scheme indeed provides better performance than advanced EEP codes.
