Construction of High Performance Block and Convolutional Multi-Edge Type QC-LDPC codes

Vasiliy Usatyuk Department of Computer Science
South-West State University Kursk, Russia

Abstract—We present code sieving method for construction of long-length high performance Block and Convolutional MultiEdge Type QC-LDPC codes (MET QC-LDPCC) under the fixed point (Quantized) reduced complexity box-plus (QRCBP) decoder. Proposed method use iterative decoding threshold protograph sieving and simulated annealing method for QC lifting with cycles EMD maximization. Using proposed method we construct base matrix of MET-LDPC code with size 92 · 20, maximal column weight 17, with weight one circulants, code rate 0.8, 2 variable nodes punctured which have iterative decoding threshold Eb/N o = 2.333 dB under 200 iterations of QRCBP over an AWGN channel with QPSK modulation on Bit Error Rate below 10−15, 0.289 dB from Shannon limit. Simulation results of Block with code length N = 261924 and MET QC-LDPCC codes, equivalent block length N = 785772 constructed using simulated annealing lifting of base matrix under 32 iterations of QRCBP show Eb/N o = 2.56 dB, Eb/N o = 2.478 dB below BER 10−15, respectively. Cole’s importance sampling under QRCBP show that proposed codes error-floor BER ≈ 10−16.
Keywords—Extrinsic Message Degree (EMD); Multi-Edge Type Low Density Parity-Check (MET-LDPC); Quasi-Cyclic (QC); Quantized Reduced Complexity Box-Plus Decoder (QRCBP); Spatially-Coupled MET QC-LDPC code (QC-LDPCC)
I. INTRODUCTION
Multi-edge Type (MET) LDPC codes generalize class of irregular LDPC codes. MET-LDPC code approach allow to make precious estimation of iterative decoding threshold according protograph structures. Traditional node perspective, edge perspective or equivalent interleaver representation, define general ensemble of LDPC codes. MET-LDPC codes graph edges from variable node have special type defined according structures of connected through a CPMs. It define specified graph from the general ensemble. Each edge type has a different message distribution under each iteration of decoding method. Another important advantages of METLDPC codes readily admit is the addition punctured variable nodes. This punctured nodes useful both for improving iterative decoding thresholds and for lowering error floors, due to properties of cycles cancellation [1].
Improving of iterative decoding threshold method using puncturing was known for Turbo-code and used in LTE codes, 6% of variable nodes in circular buffers punctured, [2]. METLDPC codes was proposed by Divsalar and R. J. McEliece (called as Protograph based codes, IRA turbo-like, [3]). Final

general description and fixed point converge condition was proof by Richardson and Urbanke, [4].
MET QC-LDPC codes, by cost of the relative small, compare to current hardware resource, increasing numbers of decoder iterations to converge, improve iterative decoding threshold compare to irregular codes with the same size of protograph. Protograph size defined complexity of barrel shifter network, smaller size allow to decrease complexity and increase throughput. MET approaches allow to use small size of protograph, lower column weight to reach comparable error correcting capability as random irregular LDPC codes.
Multi-Edge Type QC-LDPC codes currently apply or consider as strongest candidate at many modern high performance error correcting proprietary solutions and standards: from 5G eMBB [5], TV physical layer standard ATSC 3.0 [6], Deep Space Communication [7], 50G-EPON IEEE P802.3ca, [8].
To obtain a soft iterative decoder with a linear complexity, which call min-sum decoder, the Jacobi approximation rule for loopy Belief Propagation (Sum-product decoder) is applied. Min-sum decoder solve the LDPC factor graph marginalization problem with linear complexity depend from iterations number and parity-checks numbers, [9].
Exist several way to compensate loss due Jacobi approximation. One of the way is use different values of scales and offset coefficient in min-sum decoder [10], [11], make special Tanner-graph traversal message scheduler, like SC scheduler used at Polar Codes, to improve converge and decrease/eliminate influence of low weight Trapping Sets (TS) pseudocodewords, use optimal processing of subcodes (GLDPC/DGLDPC, Stair-case, Turbo codes) or graph extension methods (automorphism group decoder, [12]–[14], polar list decoder), use different approximation for checknode processing, [15]. We not consider graph traversal and graph extension approaches, even layered scheduler, which allow to decrease number of iteration (depend from structures of protograph, around twice) by the cost of decoder stalling in non-orthogonal parity-checks, [16]. To simplification of our proposed MET QC-LDPC codes construction method we use QRCBP decoder which with around 12.5% of complexity overhead from Min-Sum decoder allow to reach sum-product decoder performance without fine-tune optimization of decoder parameters for TS, [17].

The main contribution of the paper is a novel approach for constructing long length high performance Block and MET QC-LDPCC codes with code (iterativer decoding threshold) and graph connection properties (girth, EMD) which allow under limited iteration number (32 iterations vs 2000 at [27]) of quantized (bitwise for input log-likelihood ratio 7 bits, variable message 7 bits, check message 5 bits) QRCBP decoder (12.5 % complexity overhead compare to min-sum) achieve BER ≤ 10−15 with reasonable barrel shifter complexity by using protograph with size of 92 variable nodes, 20 check nodes, maximal variable node weight 17.

The performance of Block and MET-QCLDPCC codes obtained by the proposed method is investigated by simulations in Section V.

For the equal error probability TS(5,3) is more dangerous: with 3 errors in odd degree check nodes it will provide 5 errors in variable nodes while the TS(4,4) produces only 4 errors in variable nodes if there are 4 errors in odd degree check nodes and could be consider as a pseudo codeword of weight 4. The tearing of the most harmful cycles could improve pseudo codeword’s weight spectrum and decrease the probability of error-floor. It’s hazard is strongly related to the decoder and could be changed by the modification of decoder parameters, [30].
The metric Extrinsic Message Degree (EMD) of a cycle in the Tanner graph is defined as the number of check nodes singly connected to the variable nodes involved in the cycle. The EMD value of a code is an important characteristic, because each cycle is a trapping set. The EMD metric estimates how strongly subgraph of cycle is connected with the rest of the Tanner graph. Calculation of the Tanner graph EMD is a hard task, because it requires to determine if the edge is extrinsic edge or cut edge, [31].
By construction codes with bigger minimal EMD value we make equal-probability trapping sets elimination. It mean that if TS(a, b) harm depend only from value harm = b/a, probability of error in subgraph and multiplicity of this TS. Under this restriction this objective enough to construct codes which not suffer from pseudo-codewords. Unfortunately probability of TS error depend from parameters of decoder: scale and offset values in decoder, scale for message quantization, bitwise of input bits and message bit, consider error only in information bits, puncturing, shortening and etc, ( [30], [32], [33]). By variety this values we can change weight of pseudocodewords. This is reason why precious TS harm model require estimate pseudo-codeword weight using importance sampling and applied union-bound to predict behavior of code in error-floor regime. Special structures of protograph and degree distribution can simplify this model, ( [34], [35]).
QRCBP decoder allow to use flat model of Trapping set: for which higher values of EMD have more strong correlation of higher weight of pseudocodewords compare to Min-Sum decoder without scale-offset optimization. TS(a, b) harm depend mostly from value harm = b/a, probability of error in subgraph and multiplicity of this TS. It greatly simplify code construction optimization, by decreasing dependency from decoder parameters.
Performance of long length MET QC-LDPC codes under QRCBP decoder shall depend mostly from iterative decoding threshold (asymptotically code properties), EMD (graph properties), soft metric noise normalization for log-likelihood ration, quantized scale to map values from continues float representation to fix point) and quantized grid (for example regular fix point quantization grid, Fibonacci grid).
Code distance properties of long length MET QC-LDPC codes could be analyzed using different approaches: linear distance grown [36], Spectral Shape [37], [38] double exponentially fall [39]. We shall use indirect approach to improve code properties by decreasing iterative decoding threshold using modified Protograph EXIT-chart method, [40].

V. SIMULATION RESULTS
Using proposed method we construct base matrix of METLDPC code with size 92 · 20, maximal column weight 17, with weight one circulants, code rate 0.8, 2 variable nodes punctured, which have iterative decoding threshold Eb/N o = 2.333 dB under 200 iterations of QRCBP on BER < 10−15, 0.289 dB from Shannon limit. For example, iterative decoding threshold of AR4JA code, Eb/N o = 2.4095 dB, [41]. Coles importance sampling method predict error-floor at BER ≈ 10−16. Simulation results of Block MET QC-LDPC code with code length N = 261924 and MET QC-LDPCC codes, equivalent block length N = 785772 constructed using simulated annealing lifting of base matrix under 32 iterations of QRCBP over an AWGN channel with QPSK modulation show Eb/N o = 2.56 dB, Eb/N o = 2.478 dB below BER 10−15, respectively, Fig. 7.
