Constructing QC-LDPC codes based on dimension reduction for physical layer security

1st Manfei Zhang College of Computer Science and Engineering
Hunan University Changsha, China manfeizhang@hnu.edu.cn

2nd Renyong Wu College of Computer Science and Engineering
Hunan University Changsha, China renyongwu@hnu.edu.cn

Abstract—As a recently proposed physical layer security scheme, the two-dimensional transmission scheme prevents eavesdropping, but the bit error rate (BER) of the receiver becomes high. A new coding scheme for Quasi-Cyclic Low-Density Parity-Check (QC-LDPC) codes to improve the performance of the receiver is proposed and named as dimension reduction coding. This new scheme constructs the base matrix, which is necessary to obtain the parity check matrix of QC-LDPC codes, from the channel coefficient matrix of two-dimensional transmission scheme. Then, by analyzing the relationship between the base matrix and the parity check matrix of QC-LDPC codes, we find out the conditions that the base matrix must satisfy so that the corresponding parity check matrix has no shortest cycle in Tanner graph. Furthermore, a dimension reduction algorithm is presented to modify the base matrix and obtain parity check matrices with good performance. Finally, simulation results are given and validate the feasibility of this method.
Index Terms—Base matrix, dimension reduction coding, quasi-cyclic (QC) low-density parity-check (LDPC), twodimensional transmission
I. INTRODUCTION
With the emergence of wireless communication applications, the interest in extending block-fading wiretap channels to cases where the terminals have multiple antennas (e.g., [1], [2], and references therein). Ref. [3] explored practical characterizations for a specific scenario in which the sender and eavesdropper have multiple antennas, but the intended receiver has a single antenna, i.e., a multi-input, singleoutput, multi-eavesdropper (MISOME) case. Afterward, LI H [4] proved that the eavesdropper could use multiple antennas by the MUSIC-like algorithm to obtain received signals whose quality is better than that of the authorized receiver dose. To solve this problem by cracking the MUSIC-like algorithm, a two-dimensional (2D) transmission scheme [5] was proposed and used for secure transmission [6]. However, the BER performance of the authorized receiver becomes poor in the two-dimensional transmission scheme.
In the past few decades, QC-LDPC codes [7,8] have been greatly developed as an important class of LDPC codes [9]. Li et al. [10] proposed a method for encoding QC-LDPC codes with linear time and space complexities in the length of the code, which have made the design of QC-LDPC codes an attractive research area; meanwhile, numerous methods, including algebraic methods, were proposed for constructing QC-LDPC codes in [11,12]. In [13,14], the great error

performance of randomly structured LDPC codes has been

presented. In particular, the length of the shortest cycles in

the graph, girth, was identified as an important measure of

the codes performance. Furthermore, the use of group rings

to construct and encode QC-LDPC was considered in [15].

On the basis of the above situation, we propose a scheme

that improves the performance of the authorized receiver of

2D scheme in wireless communication systems. This scheme

constructs base matrix, which is necessary to obtain the

parity check matrix of QC-LDPC codes, from the channel

coefficient matrix of 2D scheme and this scheme is named as

dimensional reduced coding (DRC). To mitigate the system

BER performance degradation problem resulting from the

short girth in the parity check matrix, we exploit the available

implementations of the dimension reduction algorithm, there-

by increasing the efficiency of the proposed coding method.

The rest of this paper is organized as follows. Section

II describes the channel coefficient matrix of 2D scheme,

base matrix, and the conditions that need to be satisfied

without girth-four in parity check matrix of QC-LDPC codes.

In section III, the proposed DRC strategy is described in

detail. Section IV is concerned with numerical and simulation

results. Finally, Section V concludes this paper.

2) Determine the Elements That Need to Change in D:
According to the definitions of GF and difference matrix, the case of forming GF in D can be expressed by difference matrix that the elements of a certain column of Ds, with 1 ≤ s ≤ S − 2, have the same value. In this situation, we think that certain elements in the s-th column in D need to
be changed. Next, we determine which elements need to be changed in column s of D.
Drs,p represents the element of row r and column p in Ds, here 0 ≤ s ≤ S −2, 1 ≤ r ≤ R−1, 1 ≤ p ≤ S −1−s. When s = 0, Dr0,p represents the element of row r and column p in D. For Ds, with 1 ≤ s ≤ S − 2, if the elements of certain
rows in a column have the same value, we think that these row elements of s-th column in D cause GF to form. That is, Drs,p = Dsr ,p, which can be seen as a GF formed by rows r and r under the combination of column s and (s + p) in D. Here, 1 ≤ s ≤ S − 2, 1 ≤ r, r ≤ R − 1, r = r , 1 ≤ p ≤ S − 1 − s. We can prevent a GF from forming by changing the elements in rows r and r of s-th column in D.
Thus, to prevent all GF from being formed by elements in s-th column, we record all the rows with the same elements of every column of Ds and the elements of these rows in s-th column of D are need to be reset.

3) Selectable Elements to Replace the Elements That Need to Be Changed in D: Alternative elements should also be carefully considered. In Section II-C, the first two conditions that there is no GF in D have been discussed: Elements can not be zero; Elements in the same row or column can not be the same. The elements that are not represented in U until the last part can be considered as the substitutes to satisfy the above two conditions. And we define the set φo = φe − φl without 0 to mark these elements.
4) Decision Condition: Criteria for judging the appropriateness of the selected elements is that the number of GF in D decreases after changing all elements that form GF by selected elements. If the number of GF in D decreases, we decide that the selected elements are appropriate and remove them from φo; If there is no reduction, we consider these selected elements inappropriate. In this case, we reset D, rearrange these elements, replace variables again, and get a new D. If the entire arrangement of the selected elements are not appropriate, we will select a new element from φo to replace one of the selected elements.
Finally, we have the reason to set the number of iteration T to repeat the above process and describe the dimension reduction algorithm in detail:
Step 1: Set the number of iterations to T . The number of iterations t is initialized to 1.
Step 2: Given D, we obtain all difference matrices. The number of GF in D is counted and denoted by n1.
Step 3: Find out all Drs,p = Drs ,p in D and determine the elements that need to be changed.
Step 4: Arrange all elements selected from φo and choose an unused permutation to replace the elements that need to be changed in Step 3. From this, a new D is obtained and the new difference matrices of D are obtained. The number of GF in D is calculated and denoted by n2.
Step 5: If n2 = 0 or t = T , export D; If n2 < n1 and n2 = 0, remove the selected elements in Step 4 from φo, t = t+1, and jump to Step 2; If n2 ≥ n1 and all permutations in Step 4 have been used, we reset D, select another element from φo instead of an existing selected element, and jump to Step 5.
Thus, matrix D and base matrix U are obtained. Substitute U into (2), H with code length M = zLJ can be achieved.

V. CONCLUSION
This paper presents the DRC scheme, which constructs QC-LDPC codes to improve the performance of authorized receivers in wireless communication systems. When the proposed DRC scheme is applied to the system, the receiver performs excellently. Moreover, the numbers of fading blocks and antennas of 2D scheme and the size of the unit matrix in parity check matrix positively affect the performance of communication systems to varying degrees. Specifically, the number of antennas and the size of the unit matrix have some influence on the performance of the receiver, whereas the number of fading blocks has considerable influence. The proposed DRC scheme can naturally be extended to finitelength code constructions and is suitable for communication systems with large numbers of antennas and higher performance requirements. The size of the codeword is determined by the number of antennas, the number of fading blocks, and the size of the unit matrix. Under this situation, it is easy to support different QC-LDPC codes, given its flexibility in code length.
