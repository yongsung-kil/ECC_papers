Abstract—Time-varying spatially-coupled low-density paritycheck convolutional codes (SC-LDPC-CCs) exhibit excellent features, but their representation requires a very large number of parameters. On the other hand, the description of timeinvariant SC-LDPC-CCs is very convenient and their error rate performance, though usually worse, is often satisfactory. In this paper we investigate the girth properties of these codes, showing that the time-invariant ones have some weaknesses, which can be compensated by introducing a small periodicity in the code. By considering periodically time-varying codes, we achieve considerable improvements in the girth properties using few more degrees of freedom with respect to the time-invariant case.
I. INTRODUCTION
Low-density parity-check (LDPC) codes were introduced by Gallager in [1]. After their rediscovery in the mid ’90s, they gained increasing attention thanks to their capacity approaching performance. Recent studies have shown that spatiallycoupled LDPC (SC-LDPC) codes are capable of achieving the channel capacity for a large number of channels [2]. Spatially coupled LDPC convolutional codes (SC-LDPC-CCs) were first proposed in [3] with the name of LDPC convolutional codes (LDPC-CCs), as the convolutional counterparts of LDPC block codes. Many properties of an LDPC-CC have been shown to be at least as good as for the underlying LDPC block code. This also occurs for the girth of the associated Tanner graph.
In general, the properties of a time-varying SC-LDPC-CC can be improved by increasing its block length a, its memory ms and/or its period T . Normally, either the block length or the memory are kept small, in order to maintain small decoding latency and complexity. In [3], good codes were designed by considering a small block length, and letting the period increase with the memory. The approach in [3] was later generalized in [4], where larger block lengths are allowed and better performance is achieved. However, such codes need a large number of parameters for their representation.
Time-invariant SC-LDPC-CCs [5], characterized by unitary period, are very attractive in that they can be represented with a relatively small number of parameters. In [6], it is shown via simulations that time-invariant codes exhibit excellent error rate performance on the binary erasure channel for increasing values of the memory and fixed block length. The

girth properties of time-invariant codes are investigated, for

example, in [7], [8] and in the references therein.

In this paper we show that just increasing the value of the

memory is not sufficient to improve the girth of time-invariant

codes, while slight increases of the period result in significant

improvements in the girth. In particular, we prove that, if we

aim at achieving larger girths with respect to the time-invariant

case, for the same (small) block length, it is necessary and

sufficient to consider T > 1.

The design of quasi-cyclic LDPC (QC-LDPC) block codes

with large girth proposed in [9], [10] is easily adaptable to

the convolutional scenario. Nevertheless, the analysis carried

out in these papers does not take into account the periodicity

of the code. Authors in [11] design memory efficient QC SC-

LDPC-CCs and investigate the relationship between girth and

period. However, their analysis holds only for small values of

In this paper, we show that the girth of time-invariant SC-

In [12], the authors propose a way of describing cycles of

time-varying SC-LDPC-CCs based on a new symbolic repre-

sentation. By exploiting this description, they also introduce

a cycle-counting algorithm which allows cycle enumeration

new results on the girth properties of general time-varying SC-

LDPC-CCs, which are not bounded to any particular structure

of the parity-check matrix and hold for any code rate. We

also design practical codes characterized by small values of

the block length and period, but relatively large girths.

We define the exponent matrix P as the matrix containing the exponents of the entries of H(D). We also define W(·) as the function that returns the weight of its argument. If the argument of the function is a polynomial matrix, W(·) results in a matrix containing the number of non-zero coefficients of its entries. For the sake of simplicity, in the rest of the paper we denote the weight matrix W(H(D)) as W. We define an ensemble of codes E(W) as the collection of all the codes characterized by the same W. In the time-varying case, if W(Hi(D)) = W(Hj(D)), ∀i = j, we assume without loss of generality that W = W(H0(D)).

B. Cycles
The parity-check matrix of a code is the bi-adjacency matrix of its Tanner graph; thus, cycles of length λ can be represented over H as closed loops with λ/2 vertical edges and λ/2 horizontal edges, passing through λ different symbols 1, each associated to an edge in the code Tanner graph. The girth g of a code is defined as the length of the shortest cycle in H. In general, cycles can be grouped in two categories: the avoidable ones and the unavoidable ones. The avoidable cycles can be avoided with a proper choice of the entries of P, whereas unavoidable cycles only depend on W and are thus an intrinsic characteristic of each ensemble.

A. Limits of time-invariant SC-LDPC-CCs Lemma 1 Given a c × a (dv, dc)-regular weight matrix, such that dc > dv > 1, then each symbol 1 is involved in at least a cycle.

Lemma 2 If two cycles of length l1 and l2 in a weight matrix W share r edges, then there is an unavoidable cycle of length 2(l1 +l2 −r) in the parity-check matrix of the codes in E(W).
In the following Lemmas 3 and 4 we rephrase some results from [15].
Lemma 3 Given a c × a (dv, dc)-regular weight matrix W, such that dc > dv > 1, if it contains:
• a 3, then the girth of any code in E(W) is upper bounded by 6;
• a couple of 2 in the same row or column, then the girth of any code in E(W) is upper bounded by 8;
• a 2, then the girth of any code in E(W) is upper bounded by 10 if dv > 2, and by 12 if dv = 2.
Lemma 4 If the weight matrix W contains: • an m × n submatrix with m = n + 1 containing two rows or two columns with weight greater than 2, or • an m × n submatrix with m = n + 1 containing one row or one column with weight greater than 3, or • an m × n submatrix with m = n containing one row and one column with weight greater than or equal to 3,
and all the other rows and columns have weight greater than or equal to 2, then the girth of any time-invariant code in E(W) is upper bounded by 2(m + n + 1).
Lemma 5 A c × a (dv, dc)-regular weight matrix W, such that dc > dv > 1, contains at least two cycles.
Theorem 1 Given a c × a (dv, dc)-regular weight matrix W, such that dc > dv > 1, where dc and dv are finite, the girth of any time-invariant code in E(W) is upper bounded by a finite value, say g∗, corresponding to the length of an unavoidable cycle, for any value of ms.

V. CONCLUSION
We have shown that periodically time-varying codes with
period much smaller than their memory are able to achieve
better girth properties than time-invariant codes with the same
block length. Moreover, time-varying codes with these features
can be obtained by considering few additional parameters than
those required to describe time-invariant codes.
