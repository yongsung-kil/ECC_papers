A Study on Iterative Decoding With LLR Modulator Using Neural Network in SMR System

M. Nishikawa 1, Y. Nakamura 1, H. Osawa1, Y. Okamoto1, and Y. Kanai 2
1Graduate School of Science and Engineering, Ehime University, Matsuyama 790-8577, Japan 2Department of Information and Electronics Engineering, Niigata Institute of Technology, Kashiwazaki 945-1103, Japan

This paper focuses on the shingled magnetic recording (SMR) as the recording system for further high-density hard disk drives. Previously, we applied a log-likelihood ratio (LLR) modulator to the low-density parity check coding and iterative decoding system in SMR to improve the decoding performance. In this paper, we propose a new LLR modulator using neural network. Furthermore, we clarify that the proposed LLR modulator using neural network realizes effective iterative decoding in a read/write channel with pattern-dependent medium noise.
Index Terms— Log-likelihood ratio (LLR), neural network, shingled magnetic recording (SMR).

I. INTRODUCTION

S HINGLED magnetic recording (SMR) has attracted attention as an effective recording method for further increasing the density in a hard disk drive (HDD) [1]. Previously,

we studied the performance improvement of the low-density parity check (LDPC) coding and iterative decoding system which combines the sum-product (SP) decoder for LDPC

code with the a posteriori probability (APP) decoding for partial response (PR) channel in SMR [2]. Then, we proposed

a log-likelihood ratio (LLR) modulator using parity check information to utilize more reliable information and support the SP decoding. Furthermore, we clarified the effectiveness of the LLR modulator in the iterative decoding by computer simulation [3]. Also, we have reported that performance improvement can be obtained by various signal processing methods introducing a neural network. For example, we proposed an inter-track interference canceller using a neural network for waveform equalization [4]. In this paper, we propose a new LLR modulator using neural network for decoding process. It is expected that the new LLR modulator reduces the effect of pattern-dependent medium noise and makes the iterative decoding system more effective. We evaluate the performances of the new LLR modulator by computer simulation and clarify the impact on the iterative decoding system for HDD.
II. READ/WRITE SYSTEM
Fig. 1 shows the read/write system of LDPC encoding and iterative decoding with the LLR modulator. The input sequence passes through a 128/130 (0, 16/8) run-length limited (RLL) [5] encoder and an LDPC encoder to obtain the recording sequence. Here, the input sequence is LDPC encoded based on the sector format of 4096 bytes/sector. In this paper, 30 sectors of user data sequence are used to evaluate the bit error rate (BER) performance. The APP decoder calculates LLR by APP decoding for the PR1 channel from the 2D-FIR filter output sequence [11]. The LLR modulator selects the more reliable LLR sequence and gains it for the SP decoder to work well. The SP decoder iteratively decodes for the LDPC code until the maximum iteration number isp times. Furthermore, the external LLR sequence obtained by the SP decoding, is returned to the APP decoder again, and the turbo equalization is performed with the maximum iteration number iglobal times. The SP decoder output sequence after the turbo equalization passes through the hard decision unit and the RLL decoder, and results in an output sequence. Finally, BER is provided by comparing input and output sequences.

V. CONCLUSION
In this paper, we have proposed the LLR modulator using the neural network and compared with the performance of the conventional LLR modulator. As a result, the probability of selecting the correct LLR increases by using the proposed LLR modulator and more effective iterative decoding is performed for the pattern-dependent medium noise.
