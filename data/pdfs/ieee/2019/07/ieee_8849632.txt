AbstractÔÇöOne of the most efficient approaches to improving the frame error rate (FER) performance of low-density paritycheck (LDPC) codes is based on adding both auxiliary variable nodes (AVN) and the corresponding redundant parity checks (RPC) to the binary parity-check matrices of the code. It is known that for the LDPC codes, whose Tanner graphs contain length four cycles, this technique allows for substantial improvement in the FER performance of the belief-propagation (BP) decoding on the binary erasure channel (BEC) channel. The AVN-based technique followed by adding orthogonal redundant paritychecks is known to be efficient for both the BEC and additive white Gaussian noise (AWGN) channels. In this paper, firstly, the AVN-based approach is generalized to efficiently removing cycles of length larger than four. Secondly, the AVN-based technique is reformulated in terms of labeled base matrices of quasi-cyclic (QC) LDPC codes. An improved iterative decoding of QC LDPC codes is proposed, which is applied to the labeled base matrices of QC LDPC codes extended by the AVN-RPC technique.
I. INTRODUCTION
It is well known that iterative decoding performance of quasi-cyclic (QC) LDPC codes of a finite length is still rather far from their maximum-likelihood (ML) decoding performance. On the other hand, compactness of their description, as well as existence of efficient coding procedures, are among the reasons for QC LDPC codes being widely used in communication standards. For those reasons, improving errorcorrecting capability of these codes is an important problem.
Low-complexity suboptimal decoding techniques for LDPC codes on the additive white Gaussian noise (AWGN) channel are often based on one of the following two approaches. In the first approach, a post-processing is used in case of belief-propagation (BP) decoding failure [1]ÔÇô[3]. This postprocessing consists of erasing less reliable symbols in the codeword. Thereby, decoding on the AWGN channel is reduced to the ML decoding on a BEC, which is equivalent to solving a system of linear equations of order ╬Ż > 0, where ╬Ż denotes the number of erasures. In [4], in order to further reduce the error probability, a basis of a linear space of solutions is constructed if the solution of the linear system is not unique. Then, a search over a restricted list of solutions from the constructed linear space is performed. We refer to this approach as the list-erasure decoding (LED) algorithm. In the second approach, redundant parity checks (RPC) and auxiliary variable nodes (AVN) are added to the original code

parity-check matrix (see, for example, [5], [6]). The goal of this procedure is to remove length 4 cycles from the code Tanner graph. We refer to the algorithms in this class as AVNRPC based decoding algorithms. The BP decoding of short LDPC codes with redundant parity-checks and convergence of its error performance to the performance of the ML decoding both on the BEC and AWGN channels is considered in [7], [8]. The two aforementioned techniques allow for improving of the frame error rate (FER) performance of the BP decoding at the cost of increasing the number of decoding attempts if the BP decoder fails to recover a valid codeword. From the practical point of view, the advantage of the AVN-RPC approach compared to the LED algorithm is its relatively lower implementation complexity, although, as we can see from [5]ÔÇô [8], the size of extended parity-check matrices can be rather large. A practical approach to decoding of QC LDPC codes using RPC-like ideas was proposed in [9].
In this paper, we modify the AVN-RPC technique in the following aspects. First, we generalize this technique in order to eliminate length 6 and 8 cycles in the Tanner graphs of binary LDPC codes. Then, we reformulate this approach in terms of labeled base matrices of QC LDPC codes, which dramatically reduces implementation complexity. Finally, the BP decoding applied to the extended parity-check matrices of QC LDPC codes is tested experimentally.
We investigate the behavior of coding gain as a function of a number of decoding attempts (number of different extended parity-check matrices used by the BP decoder). Our simulation results show that if the number of decoding attempts is large enough, then the AVN-RPC based decoding performance is superior to that of the LED algorithm. However, the efficiency of the AVN-RPC decoding heavily depends on the choice of a particular code. For most of the considered QC LDPC codes, even for a small number of decoding attempts, the coding gain is noticeable.

Theorem 1: A generalized AVN-RPC based procedure applied to the QC LDPC code C produces a code equivalent to C. The subgraphs determined by the original sets of variable and check nodes of the Tanner graph of C have no cycles of length six and eight.

The presented procedure destroys cycles in the original Tanner graph similarly to its counterpart in [5]. However, it could generate new cycles involving both original and auxiliary variable nodes and modified original and auxiliary check nodes. For example, the extended parity-check matrix (9) contains the new cycle V = {v2, u1, u2}, C = {c╦ť3, c4, c5}, where c╦ť3 denotes the modified check c3. Theorem 2: The AVN-RPC based procedure does not increase the number of the stopping sets in the parity-check matrix.

Although the auxiliary variable nodes can belong to a new cycle, the appearance of this cycle does not increase the number of stopping sets. Since the first g/2 Ôłĺ 1 rows of the extended submatrix of the code parity-check matrix are in bidiagonal form, all auxiliary variables can be found by using these g/2 Ôłĺ 1 parity-check equations. V. AVN-RPC BASED DECODING ON THE AWGN CHANNEL
In this section, we present the decoding algorithm based on the AVN-RPC approach, which is presented as Algorithm 1. For a given QC LDPC code, first, the list of cycles of lengths 4, 6 and 8 in its labeled Tanner graph is constructed. Let L denote the total number of such cycles. A quasi-cyclic structure of the code simplifies the search and reduces M times the memory requirements for storing the list of cycles. For a QC LDPC code, it is sufficient to find and store the pair (V, C) corresponding to each cycle in the labeled Tanner graph.

In the beginning, the conventional BP decoding is applied to the received word. If the BP decoder fails, then the candidates from the pre-constructed list of cycles are successively excluded from H by the AVN-RPC based procedure, and the BP decoding is applied to the modified matrix. In this decoder, the initial LLRs for the auxiliary variables are set to zero. The decoding stops either if the codeword is found, or if the list of candidates is exhausted.
