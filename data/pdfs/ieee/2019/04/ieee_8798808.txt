Read Voltage Optimization in MLC NAND Flash Memory via the Density Evolution
Chatuporn Duangthong, Watid Phakphisut and Pornchai Supnithi Faculty of Engineering, King Mongkut's Institute of Technology Ladkrabang, Bangkok 10520, Thailand.
Email: {58601143, watid.ph, pornchai.su}@kmitl.ac.th

Abstract—The multi-level-cell (MLC) NAND flash memory can typically use multiple reads for obtaining soft information for the LDPC decoder. The multiple reads give the soft information which is quantized to a certain level. The major challenge is that the read voltages must be precisely selected to provide better soft information. In the previous work, the read voltages are selected so that the soft information has the maximum mutual information (MMI). However, the error-correction capability of LDPC decoder is not considered. Therefore, in this work, we analyze the performance of LDPC decoder by density evolution whereby the soft information is quantized. Then the optimal read voltages for given LDPC codes are obtained. As a result, for a regular LDPC code with the read voltages optimized by density evolution can provide the lower BER performance compared with the MMI technique.
Keywords—MLC NAND flash memory, read voltage optimization, LDPC code, density evolution
I. INTRODUCTION
The NAND flash memory is the omnipresent storage technology belonging to mobile consumer. The first generation of NAND flash memory uses a single-level-cell (SLC) technology in which one data bit can be stored in a memory cell. Since the demand for high capacity storage has rapidly increased, the multiple bits per memory cell called multi-levelcell (MLC) flash memory have been alternatively used. As the manufacturing cost of the integrated circuit chip has been continuously decreased, the NAND flash memory is expected to replace the storage technologies that have the moving parts in the enterprise products.
Although the multi-level-cell technology can provide the high capacity storage, the memory cell incurs the severe distortions such as random telegraph noise (RTN) [1] and cellto-cell interference [2]. To overcome several noisy sources, the low-density parity-check (LDPC) codes, the near Shannon limit error-correcting codes, have been adopted. The LDPC codes require the soft information from a memory cell, therefore, the multiple reads of a memory cell must be used. However, the multiple reads cause the read signal quantized and influences the performance of LDPC decoder. The authors in [2] proposed the soft information is obtained by nonuniform quantization. The threshold voltage dominating overlap region will be sensed by the multiple read voltage. Recently, in [3], the read voltage is optimized by using the maximum mutual information (MMI). The advantage of MMI technique is that the number of reads can be assigned easily through the discrete memoryless

channel (DMC) model. However, in [4], the simulation results show that optimized read voltage obtained from MMI method does not guarantee that the bit error rate of LDPC decoder is lowest. The authors in [4] propose the entropy approach for finding the optimal read voltage. The obtained entropy value will provide the read voltage sets. For each read voltage set, the performance of LDPC decoder will be evaluated by Monte Carlo simulation. The optimal read voltage from the entropy technique reveals that the read voltages dominating overlap region must be wider. Although the entropy technique can give the lower BER performance, the read voltage sets obtained from the entropy value must be evaluated and the number of read voltages is not flexible.
Since the optimized read voltage obtained from MMI method does not guarantee that the bit error rate of LDPC decoder is lowest, we propose the use of density evolution in read voltage optimization to find the optimal read voltage for LDPC codes. We consider the situation which the existing LDPC code is applied in the 2 bits/cell NAND flash memory. the bit-interleaved coded modulation is used [5] as the coded modulation scheme of MLC NAND flash memory. Herein, we separate the quantized signals of 2 bits/cell NAND flash memory into the soft information corresponding to the most significant bit (MSB) and the least significant bit (LSB). Then the probability density function of each soft information will be tracked during the iterative processing of LDPC decoder. II. BACKGROUND
A. The Basics of NAND Flash Memory
In the NAND flash memory, the memory cell uses the transistor technology in which the amount of charges in the transistor technology represent the data bits stored in the memory cell. The threshold voltage vth of transistor is related to the specific voltage for turning on the transistor. Before the memory cell is written, the memory cell must be erased, i.e., the charges in the transistor are removed. To write the memory cell, the write voltage will be applied causing the threshold voltage to be shifted to a particular voltage level. For example, given 2 bits/cell NAND flash memory, the threshold voltage can be configured to 4 levels μ11 , μ10 , μ00 and μ01 which represent the symbols ‘11’, ‘10’, ‘00’ and ’01,’ respectively. To read the memory cell, the read voltage (γ) must be applied at the control gate. The sense amp comparator will identify the drain current as the stored information bits. For example, the read voltage γ 1 , γ 2 and γ 3 are needed for 2 bits/cell NAND flash memory, where γ 1 has the value between μ11 and μ10 for determining the symbols ‘11’and ‘10’, γ 2 is defined between μ10 and μ00 for the symbols ‘10’and ‘00’ and γ 3 is given between μ00 and μ01 for the symbols ‘10’and ’00.’ To get the soft information, at least 6 levels {γ1, γ2, γ3, γ4, γ5, γ6} are required to compute the likelihood of each symbol as shown in Fig. 1.
B. Quantized Channel Model
In [6], the threshold voltage vth of 2 bits/cell NAND flash memory is modeled as the write voltages affected by the major noise sources such as the erase operation noise, the write operation variation, the program and erase (PE) cycles and

retention time. However, in this work, the threshold voltage vth of 2 bits/cell NAND flash memory is modeled as a symmetric Gaussian distribution [7]. Considering the 2 bits/cell NAND flash memory in Fig. 1, each threshold voltage is modeled as Gaussian noise with variance σ2. It can be observed that there are three overlap regions. The read voltages located at the first and third overlap region will affect the bit error rate of the least significant bit (LSB). The read voltage located at the second overlap region will influence the most significant bit (MSB).
When the read voltages {γ1, γ2, γ3, γ4, γ5, γ6} is applied, the Gaussian channel model in Fig.1 can be viewed as the discrete memoryless channel (DMC). The DMC consists of the input symbol X and the output symbol Y, whereby the input X is the set of {‘11’, ‘10’, ‘00’, ‘01’} and the output Y is the set of {‘11’, ‘e1’, ‘10’, ‘e2’, ‘00’, ‘e3’, ‘01’}. Fig. 2 (a) shows the DMC of MSB. The line represents the probability set p1 of ‘1’ and the dot line is the probability set p0 of ‘0’. The p1 consist of {p31, p32, p33, p34, p35, p36, p37, p41, p42, p43, p44, p45, p46, p47} and p0 consist of {p11, p12, p13, p14, p15, p16, p17, p21, p22, p23, p24, p25, p26, p27}. C. The Quantization of Flash Memory Channel using Mutual Information

In the previous work [3], the read voltages are obtained by maximizing the mutual information of the single level cell (SLC) flash memory. For SLC flash memory which stores one bit over the memory cell, the channel is modelled as a mixture of 2 independent and identically distributed (i.i.d.) Gaussian p1 and p2. For the written bit ‘0’ (resp. ‘1’), the threshold voltage is modeled as the Gaussian distribution with the variance N0/2 and mean -1 (resp. +1).
For 2 reads, the threshold voltage is divided into 3 regions by 2 symmetric read voltages γ and -γ. These regions {p11, p12, p13} are given by (1). C. Read Voltages Optimization
In this work, we use the bit-interleaved coded modulation (BICM) [5] as the coded modulation schemes of MLC NAND flash memory. Fig. 4 shows the BER estimation after LDPC decoding of 2 bits/cell NAND flash memory. The optimization algorithm start by setting the initial read voltages γ = {γ1, γ2, γ3, γ4, γ5, γ6} and the degree distribution dv and dc of LDPC code are set to be a constant. Then the probability density function of LLR of MSB and LSB are computed under assumption that the channel is symmetric. At the density evolution process, the BER performance of MSB and LSB are estimated separately and then both estimated BER performance are averaged. To find the optimal read voltage, the γ1 is lightly changed while γ2, γ3, γ4, γ5, γ6 are fixed. Then the read voltage γ1 that provide lowest BER performance is stored. The read voltages γ2, γ3, γ4, γ5, γ6 are updated in the same manner as the read voltages γ1

IV. SIMULATION AND RESULTS
In this section, we will compare the performance of LDPC decoder in 2 bits/cell NAND flash memory whereby the read voltages are selected by the maximizing mutual information (MMI) and density evolution (DE). The signal-to-noise ratio of Gaussian channel model is SNR (dB) = 10log10(Eb/N0) where Eb is a bit energy and N0 is a noise spectral density. The degrees of bit nodes and check nodes of LDPC codes are dv = 3 and dc = 27, respectively. The iteration number of the LDPC decoder is 50 iterations.
Fig. 5 shows the performance comparison of MMI and density evolution quantization on the Gaussian channel model with 6, 7, 8 and 9 reads. The research found that the BER performance is going up when the read number is increased and the density evolution provides the BER curve below MMI at the high SNRs.
In Fig. 6, we study the read voltages of the MMI and the density evolution. We found that the read voltages of density evolution will be wider than that of MMI techniques when the SNR is increased.

V. CONCLUSION
This paper investigates the optimal read voltages for the LDPC decoder in MLC NAND flash memory. We determine the read voltages obtained by density evolution of LDPC decoder. Therefore, the optimal read voltages for given LDPC codes can be found. As a result, the read voltages of density evolution will be wider than that of MMI techniques when the SNR is increased. Therefore, the read voltages optimized by density evolution can provide the lower BER performance than the MMI technique. Moreover, we show that the read voltage is dependent on dv, dc and code rate. At the low SNR, the LDPC decoder with optimal read voltages from DE method gives hard decision.
