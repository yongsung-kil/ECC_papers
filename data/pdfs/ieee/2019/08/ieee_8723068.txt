Adaptive Soft Detection for Flash Memory
Magnus Sandell , Senior Member, IEEE, and Amr Ismail , Member, IEEE

Abstract— To provide reliability of flash memory, error-correcting codes are usually used which can be further enhanced by moving on from hard-decision to soft-decision decoders. This requires multiple readings to obtain a reliability measure of the stored data, which is complicated by the fact that a posteriori probability distributions change as the memory ages through retention and wear-out. The existing methods for reading a flash memory either assume knowledge of these distributions or require a large number of readings for accurate estimation. In this letter, we propose a low-complexity algorithm that tracks the soft-decision boundaries and estimates the corresponding log-likelihood ratios (LLRs). It does not require any additional readings and only involves simple calculations using already available variables. The efficacy is verified through the simulations and is shown to provide near-optimal performance.
Index Terms— Flash memory cells, demodulation, decoding.
I. INTRODUCTION
N AND flash memory [1] is a ubiquitous storage medium that has become essential in many consumer products, such as flash drives, mobile phone etc. To protect the data, powerful error-correcting codes are used and to increase its reliability, the trend is to move away from hard-decision codes, e.g., BCH, to soft-decision codes such as LDPC. This requires more readings of the flash memory cells to provide a more nuanced estimate of its content. As the device changes over time (retention) and usage (wear-out), an adaptive memory controller is necessary to read the content of the cells with a low error probability.
One of the biggest challenges when providing soft readings from a flash memory cell is where to put the voltage thresholds, as this is usually more critical than finding the log-likelihood ratios (LLRs) [2]. Methods based on mutual information [3] and entropy [4] have been proposed but they have assumed full knowledge of the a posteriori distribution of the cell content. These distributions can be estimated [5] but require multiple readings and extra processing; it is also difficult to capture the varying nature of the distributions. Blind LLR estimation, i.e., assuming no prior knowledge of the distributions, is possible. A scheme exploiting the parameters of the LDPC code was proposed in [6]; however it involves a costly optimization step and still requires the determination of the quantization intervals.

To overcome these problems, we propose another approach. Instead of looking at the values of the PDFs, consider the probability of a decision error which is indicated by the shaded area in Figure 1. Just as the ratio, this is an indication of the overlap of the two PDFs. However it is much easier to estimate; in fact it is already done when the LLRs are estimated and is linked through (2). Hence we can devise an adaptive algorithm that tries to keep this quantity fixed. In the next subsection we will describe this in more detail.
A. Estimating Boundaries
The hard decision boundaries can easily be tracked adjusting them according to the bits read from the cells. Assuming that all stored bits are equiprobable (if this is not the case, it can easily be achieved with a scrambler), the boundary is set to have half of the estimated bits to be 0. For the LSB of a four-level MLC cell, there are two hard boundaries, V1(LSB), V2(LSB). By using the same symmetry argument, we need to set V1(LSB) such that a quarter of the cells have voltages less than this and likewise V2(LSB) such that a quarter have higher voltages. This principle can be extended to any number of bits per cell.
The soft decision boundaries can be assumed to be symmetrically placed around the hard decision boundaries since two neighboring cell states have very similar distributions (with the possible exception of the erased state). Hence we can place them a distance Δ on either side of the hard decision boundary2, see Figure 1. This separation Δ is estimated by exploiting the estimates of the error probability pn.

IV. SOFT DETECTION ALGORITHM
It is now possible to formulate the complete soft detection algorithm. Starting from an initial estimation of the boundaries and LLRs, the cells are read and decoded. The hard and soft boundaries are then updated as described in Section III using the read values and the LLR look-up table is updated with the procedure described in Section II3. Hence this will not involve any additional readings apart from the one needed to obtain the stored data. Note that the boundaries and LLRs will change quite slowly over time and usage, so it is not necessary to update them every read cycle. Our simulations show that computing new values every few hundreds program/erase cycles is sufficient, keeping the complexity very low.

V. NUMERICAL RESULTS
To evaluate the performance of the proposed scheme, we simulated it with a four-level cell (two bits) and a regular (9216,8192) LDPC code with variable node degree 3 and constraint node degree 27 for both the MSB and LSB. The threshold voltage of erased cells is modeled with a Gaussian distribution [8]. In order to program a cell to a nonerased state, we use the Incremental Step Pulse Programming (ISPP) approach [8]. In our simulations, we simply apply an incremental program step voltage until the required threshold voltage has been reached. The voltage levels were {1.1, 2.35, 3.05, 3.75} V with the erased state standard deviation 0.35 and an ISPP step size of 0.3 V. For the dynamical boundary tracking, we used fixed adjustment sizes dhard = dsoft = 0.01 and the target BER was set to pn(target) = 0.15; the adaptation is done every 100 P/E cycles.

VI. CONCLUSIONS
In this letter we have presented a blind tracking algorithm for soft-decision reading of flash memory cells. The hard and soft decision boundaries are adapted to keep the error probability in the regions next to the hard decision constant. No additional readings are required and only already computed variables are needed for the adaptation. Simulations show that its performance is close to the genie-aided one with perfect knowledge of the a posteriori distributions.
