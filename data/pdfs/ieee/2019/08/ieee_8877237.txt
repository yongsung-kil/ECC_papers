Neighbor-Cell-Information Based Detection for LDPC Coded MLC NAND Flash Memory
Yanfu Li, Guojun Han, Zishuai Peng, Guofa Cai, and Yi Fang

Abstract—As the feature size of NAND flash memory decreases, multi-level cell (MLC) technique is emerging to dramatically increase the storage density. In such scenarios, the cell-to-cell interference (CCI) arising from parasitic coupling-capacitance between adjacent cells becomes more severe, which will significantly distort the threshold voltage and deteriorate the data-storage reliability. In this paper, through analyzing the characteristics of threshold voltage distribution of low-density parity-check (LDPC) code MLC NAND flash memory, a post-processing detection scheme is proposed to mitigate the CCI. The proposed scheme utilizes both the neighborcell information and the neighbor-a-priori information, and thus is referred to as the Mixed-NCI detector. A salient feature of the proposed detection scheme is that it can acquire very accurate readback voltages by exploiting the neighbor-cell information of the successfully decoded cells. Analyses and simulation results show that the proposed Mixed-NCI detector achieves remarkable performance gains compared with the state-of-the-art counterparts. The above advantage makes the Mixed-NCI detector extremely attractive for MLC NAND flash memory.
Index Terms—Cell-to-Cell interference (CCI), Neighbor-Cell information, NAND flash memory, Channel detector.
I. INTRODUCTION
NAND flash memory has been widely applied in a myriad of consumer electronic products due to high storage density and low cost, thus dominating the global flash memory market [1]. With the development of manufacture technology, the multi-level-cell (MLC) technique, which allows a single flash cell to store multiple bits, has been proposed to dramatically increase the storage density of flash memory. Nowadays, NAND flash memory with sub-20-nm has already been employed in the realistic productions. Unfortunately, when more than two bits are programmed per cell and flash cells scale below 20 nm [2], the data-storage reliability severely degrades.
As the NAND flash memory cell scales down, the couplingcapacitance between the adjacent memory cells increases. The circuit-level impairment introduces the cell-to-cell interference (CCI) phenomenon inside the flash cell array, which is widely recognized as one of the major noise sources causing the thresholdvoltage distortion [3]. In fact, the CCI will dramatically degrades the data-storage reliability, and thus making the channel detection much more difficult. To overcome the aforementioned weakness, error-correction codes (ECCs) have been introduced to NAND flash memory systems, especially MLC NAND flash memory systems. Therefore, devising effective ECCs has become one of the most important research tasks to ensure the data-storage reliability. However, some hard-decision ECCs, such as BoseChaudhuri-Hocquenghem codes [4] and Reed-Solomon codes

Against this background, soft-decision ECCs that enable stronger anti-noise capability are carefully investigated. In recent years, the excellent error performance of low-density parity-check (LDPC) codes has been demonstrated in a variety of communication systems [8], [9]. Thus, such codes exhibit great potential to be a competitive candidate in the NAND flash memory system. The error-correcting capability of the soft-decision decoding algorithms highly depends on the accuracy of channel initial log-likelihood-ratio (LLR) information [10], [11]. Due to the above reason, designing channel detectors tailored for MLC flash memory is inevitably required.
In the existing literature, there have been several programming approaches to minimize the CCI effect [14]. Specifically, a CCI minimization scheme has been proposed by exploiting the different impacts on the multiple bits in [14]. However, these schemes only reduce the CCI effect during programming process while they cannot be efficient to mitigate the CCI effect.
To further improve the data-storage reliability for NAND flash memory, a myriad of pre-processing and post-processing techniques have been conceived in [3], [7], [10], [12], [13], [15]. In [3], the pre-distortion technique has been first applied to the MLC NAND flash memory systems to mitigate CCI effect. Following the advancement of [3], a new pre-processing scheme has been developed in [12] by estimating CCI strength before programming and subsequently subtracting from the desired write voltage. However, these pre-processing schemes are not able to mitigate the CCI effect from the erased-state cells.
With an aim to mitigating the CCI effect from the erasedstate cells, many post-processing detection schemes have been proposed. In [7], a variety of channel detectors have been conceived by adaptively applying the read-voltage level. Nevertheless, in these detectors the CCI cannot be removed from the read-back voltage, thus the flash memory system cannot achieve desirable error performance. Recently, some more effective post-processing schemes have been proposed in [3], [12], in which the CCI strength not only can be estimated, but also can be removed from the read-back voltage. Moreover, a new CCI-mitigation scheme that utilizes reduced symbol pattern of interfering cells has been proposed in [13]. Likewise, a trellis-based detection scheme, which exploits a high-complexity method to calculate soft-input-soft-output information, has been developed in [10]. In flash memory systems, the read-back voltage is influenced by CCI, which leads to overcompensation phenomenon and results in suboptimal error performance. As a further advance, a new detection scheme using the neighbor-a-priori information has been proposed in [15] to partially resolve the overcompensation problem. The main idea of this work is to remove the CCI component from the interfering cells before CCI cancelation from the victim cell. However, there’s still a nontrivial gap between the

II. CHANNEL MODEL OF MLC NAND FLASH MEMORY
NAND flash memory is composed of a number of floating-gate transistors and can be represented by an array that consists of rows and columns. In particular, each row and column are referred to as a word-line and a bit-line, respectively. Conventional single-level cell has two levels of charge amount, thus it can only store one bit in each cell, while MLC has 2m levels of charge amount and it can store m bits in each cell [3]. One can exploit different charge amounts of floating gate to distinguish the values of storage bits in a flash memory cell. Data storage in MLC NAND flash memory can be realized by programming the value of voltage in floating gate into 2m nonoverlapping voltage windows. Each memory cell must be erased before programming.

The main storage device of flash memory cells is the floating gate transistor, and it can be programmed by changing its corresponding charge amount. In the initial state, all the cells should be erased and preserve the erased state. IV. PERFORMANCE ANALYSIS
Fig. 4 illustrates the PDFs of the threshold voltages after programming, CCI, and Post-Comp, All-AP and Mixed-NCI detection schemes in a 3-bits/cell NAND flash memory. It can be observed that threshold-voltage distribution is severely distorted to the right side under the CCI. The negative effect of CCI is obviously mitigated after performing Post-Comp [3], thus making the identification of different storage states eaiser. Furthermore, by exploiting the All-AP detection scheme, the voltage signal can be significantly recovered and the overcompensation phenomenon can be partially eliminated. Last but not least, because the threshold-voltage PDFs of the proposed Mixed-NCI detection scheme is closest to that of the programmed threshold voltages, the proposed Mixed-NCI detection scheme is superior to the AllAP detection scheme, and the Post-Comp detection scheme.

VI. CONCLUSION In this paper, a new post-processing-based detection scheme, referred to as Mixed-NCI detection scheme, has been proposed for LDPC-coded MLC NAND flash memory systems. The proposed detection scheme exploits the neighbor-cell information of neighbor cells to significantly mitigate the CCI influence of flash memory. In the Mixed-NCI detector, if the neighbor cells successful-decoding, the neighbor-cell information is calculated; Otherwise, the neighbor-cell information is replaced by the apriori information.
