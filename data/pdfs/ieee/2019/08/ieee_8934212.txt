ABSTRACT
In this paper, we consider the construction of irregular QC-LDPC codes by jointly optimizing the girth, the number and approximate cycle extrinsic message degree (ACE) of short cycles to further improve the error-floor performance of irregular QC-LDPC codes with given degree distributions. By selecting the best edges, each of which must make the cycles consisting of the current variable node have the best lower bound of ACE or the minimum number of cycles with ACE lower bound, for each variable node via the progressive Edge growth (PEG) approaches, irregular QC-LDPC codes with better tradeoff between girth, the shortest cycle number and ACE could be constructed. Simulation results show that the proposed codes achieve better error-floor performance compared with the codes using the same degree distributions constructed by the recent proposed methods.
Keywords: Irregular quasi-cyclic (QC) LDPC codes, Masking, Progressive Edge Growth (PEG), Girth, Approximate Cycle Extrinsic Message Degree (ACE).
1. INTRODUCTION
With the emergence of soft decision concept, the soft decision FEC codes, such as LDPC code suggested by the OIF white paper, becomes one of the strong candidates for the third generation FEC scheme [1].
It has been reported that well-constructed large girth regular QC-LDPC code can achieve 11.3dB net coding gain (NCG) at the BER of 10 [2] with 10Gb/s high throughput. Although the large girth regular QC-LDPC code achieves very high NCG, the required SNR still has a 1dB gap between the Shannon limit. The 1dB gap mainly caused by its regular property, which will lead to the unequal protection (UEP) to be ineffective. Benefit from the UEP property, irregular LDPC code can achieve the convergence threshold very closing to the Shannon limit. However, the high error-floor of irregular LDPC code prevents its application in high speed optical communication systems.

Recently, some new code design and construction methods are proposed to lower the error-floor of the irregular LDPC code. By properly adjusting the degree distributions, the error-floor can be pushed down as low as possible by costing small convergence thresholds [3]. Furthermore, some masking [4] and heuristic searching algorithms [5] with different design metrics, such as girth, the number of the shortest cycle and approximate cycle extrinsic message degree (EMD) (ACE) [6], are proposed to improve the error-floor performance of irregular QC-LDPC codes with given degree distributions.
Inspired by the joint design and construction idea [7], we consider the construction of irregular QC-LDPC codes by jointly optimizing the three-important metrics that referred to the girth, the number of the shortest cycle and ACE to further improve the error-floor performance of irregular QC-LDPC codes with given degree distributions. The main idea is summarized as follows: Firstly, Adding the required number of edges for each variable node via the PEG approaches in the variable node weight descending order; Secondly, selecting the best edge, which makes the cycles consisting of the current variable node have the best lower bound of ACE or the minimum number of cycles with ACE lower bound, from the edges that both satisfying the PEG and degree distribution constraints to be the k-th edge of the current variable node; Finally, after the edge-adding
process, the null space of the parity check matrix could give the required degree distribution irregular QC-LDPC code with better tradeoff between girth, the shortest cycle number and ACE. Simulation results show that the proposed codes perform better error-floor performance compared with the codes using the same degree distributions constructed by the recent proposed methods when decoded with SPA over the AWGN Channel.

PEG WITH ACE OPTIMIZATION

It is well known that short cycles in the Tanner Graph will degrade the performance of SPA. Therefore, enlarging the girth of the Tanner Graph becomes one of the effective methods to improve the performance of LDPC codes. In addition, the connectivity of cycles, measured by EMD or ACE, play important roles in error-floor performance of LDPC codes. The PEG algorithm can naturally make the girth of the Tanner Graph as large as possible. When using PEG to construct
QC-LDPC codes with given degree distributions,
