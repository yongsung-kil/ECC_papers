Burst Erasure Correction by 2D LDPC Codes

Hiroshi Kamabe Gifu University 1-1, Yanagido, Gifu, 501–1193, Japan Email: kamabe@ieee.org

Shan Lu Gifu University 1-1, Yanagido, Gifu, 501–1193, Japan Email: slu@ieee.org

Abstract—Two dimensional magnetic recording systems are investigated extensively to achieve higher recording density. Since the recording density is very high, a small physical defect may cause many two dimensional errors. Hence many two dimensional signal processing of magnetic recording systems are also proposed and investigated. Matcha et al. have proposed a two dimensional LDPC code and investigated the error correcting performance of the code. In this paper we propose a code which is a variant of their code but we can show that the theoretical lower bound of the burst erasure correction capability of our code is better than that of Matcha’s code.
I. INTRODUCTION
Since very high speed mobile communication systems, e.g. 5G, and very high resolution broadcast are now available, the demand for digital storage systems with very large capacity increases constantly.
Wood et al. proposed the concept of two dimensional magnetic recording (TDMR) systems to achieve very high recording density [1]. Two dimensional signal processing is one of key techniques of TDMR. The size of a reading head in TDMR is larger than the width of a recording track and there may be no gap between adjacent tracks. Since the width of a recording track in the TDMR system is very narrow, a joint track detecting system is proposed [2]. Hence a physical defect of a recording medium may cause two dimensional errors. Although there are several algebraic 2D error correcting codes [3], we consider LDPC codes to apply the generalized belief propagation algorithm [4].
Matcha et al. have proposed a two dimensional LDPC code and shown a lower bound of the burst erasure correcting capability of the code [5]. By computer simulation they also have shown that the code can correct larger burst erasures than the lower bound.
In this paper we improve their lower bound. Strictly speaking, a code we investigate here is little bit different from the code defined in [5] but the structures of these two codes are almost the same.
II. PRELIMINARIES
We assume that p is an odd prime number throughout this manuscript.
For a finite set A, #A means the number of elements in A. The set of integers is denoted by Z. For two positive integers a and b, a mod b is the remainder when a is divided by b, that

is, the smallest non negative integer r such that a = qb + r for some integer q. We define mod′ by
a mod′ b = ((a − 1) mod b) + 1.

For any positive integer k, if k is not a multiple of p, then we have

{ik mod p|1 ≤ i ≤ p} = {ik mod p|i ∈ Z} = {0, 1, . . . , p − 1}.

If (a − b) mod p = 0, then we write a ≡ b mod p. We define a circular permutation pm(i) by
pm(i) = (i + 1) mod′ p.

In this paper we index positions of a matrix with positive integers, that is, the set of positions of the matrix of size n×m is {(i, j) | 1 ≤ i ≤ n, 1 ≤ j ≤ m}.
For a real number x, ⌊x⌋ is the greatest integer less than or equal to x.

The code introduced by Matcha et al. can correct every burst erasure of size p × p. The structure of our code is almost the same to that of their code but our code can correct every burst erasure of size p × (2p − 1).
Consider a two dimensional magnetic recording (TDMR) system [1] and suppose that a reading head of the system reads signals on n tracks simultaneously [2]. Assume that the cross and down track directions correspond to the j and k directions, respectively. When the head accidentally touches a recording medium, a burst erasure error may be induced. If n ≤ p and the contact is rather small, the size of the erasure may be smaller than p×p and the erasures can be corrected by the code introduced by Matcha et al. If the contact is little bit heavy then the size of the erasure pattern for the down track direction may increase. Our code can correct the erasure errors when the size of the erasure pattern is less than p × (2p − 1).
