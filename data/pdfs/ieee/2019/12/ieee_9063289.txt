Joint Channel-Network Coding In Unequally Error Protected Multi-way Wireless Relay Systems
Nuwan Balasuriya, Member, IEEE and Chandika B. Wavegedara, Member, IEEE

Abstract-Joint-channel physical layer network coding (JCPNC) has attracted much attention recently as a promising technique to achieve a good trade-off between the error performance and the spectral efficiency in multi-way wireless relay systems. However, in most of the practical multi-way relay systems, the channels between the nodes and the relay are heterogeneous. In particular, some channels can be of relatively poorer quality than the other channels which require a higher error protection. Meanwhile, providing the same high level of error protection in high-quality channels results in a poor spectral efficiency. Thus, in this paper, we propose a novel unequal error protection scheme in order to achieve a better trade-off between the error performance and spectral efficiency. In the proposed scheme, low-quality channels are provided higher protection using a full-length low density parity check (LDPC) channel code, while high-quality channels are given lower protection using punctured LDPC codes. We also develop a novel joint LDPC decoding and network coding algorithm to be employed at the relay. As demonstrated by the simulation results, the proposed error protection scheme together with the joint decoding algorithm provides an excellent spectral efficiency-error performance trade-otT in practical multi-way wireless relay systems.
Index Terms-Multi-way relay channel, unequal error protection, Joint channel-physical layer network coding
I. INTRODUCTION
Multi-way relay systems, where multiple nodes exchange their information via a single relay, are one of the most common relay configurations in wireless networks. There is an array of applications of multi-way relay systems ranging from wireless sensor networks to wireless teleconferencing networks. One of the main concerns in multi-way relay communication is the exchange of total amount of information at multiple nodes within a minimum number of time-slots. In particular, two-way relay systems benefit from physical layer network coding (PNC) where the two nodes can exchange their information using only a half of the time-slots required for conventional relaying [1, 2]. This results in a 100% throughput enhancement. PNC can be extended to multi-way relay systems too and several such extensions have been proposed in [3-5]. At the same time, the quality of the information transfer in terms of bit error rate (BER) should be kept at a minimum. To meet a trade-off between the BER performance and the time-slot usage in multi-way relay systems, an opportunistic

As in most of the other practical wireless communication systems, in order to further reduce the BER, channel coding can also be employed in multi-way wireless relay systems. However, to yield the full potential benefits of the channel coding one needs to carry out channel decoding and network coding jointly at the relay [9, 10]. Joint decoding algorithms were proposed for multi-way relay systems using pair-wise opportunistic information exchange [11-13]. All these algorithms employ the same channel code in all the channels between each node and the relay. However, the condition of each channel between a node and the relay can be different. Therefore, employing a single channel code for the protection of all channels may not be very effective. Employing a channel code of a high code rate may not be able cater for poor quality channels introducing high number of bit errors. On the other hand, employment of a low rate code reduces the throughput without adding a significant error performance improvement. Therefore, it is clear that asymmetric error protection offers a better trade-off between the throughput and error performance and hence, more desirable.
Motivated by the above reasons, we propose an unequal error protection scheme for the multi-way wireless relay system employing PNC. Moreover, we develop a joint channel decoding-PNC algorithm which can be employed at the relay of unequally error protected multi-way relay systems to provide a trade-off between the error performance and the spectral efficiency.

In Section V, we elaborate on the scaling of the proposed decoding algorithm to a higher number of nodes. Section VI is dedicated for simulation results and a discussion while Section VII concludes the paper highlighting several avenues for further research.

V. SCALING THE UNEQUAL ERROR PROTECTION SCHEME

(M > 3) number of nodes wish to exchange information via a
single relay, the same unequally error protection scheme can be applied. Let us name the M nodes as N 1 , N 2 , •.• , N M such that the nodes N 1 and N M are associated with the two best quality channels. During the first time-slot, N 1 and
N 2 transmit n + k1 dimensional code vectors to the relay.
During the second time-slot nodes N2 and N 3 transmit n+k2 dimensional code vectors to the relay. During the third time-
