Abstract
A kind of new parallel concatenated LDPC codes (NPCGC) is presented which use two same degree distribution pairs of irregular LDPC codes constructed by the progressive edge-growth (PEG) algorithm in turbo structure. The PEG algorithm for constructing Tanner graphs with large girth by progressively establishing edges or connections between symbol and check nodes in an edge-by-edge manner is used to construct a sparse parity check matrix and a simple encoding method is used to encode the irregular LDPC codes. The advantages ofNPCGC over PCGC, irregular LDPC code and turbo code are demonstrated by extensive simulation results on code performance.
1. Introduction
Parallel concatenates code are constructed in parallel concatenating two or more relatively simple component codes. This make it possible to use the iterated decoding algorithm at decoding end and make the perfornance of Parallel concatenated codes is largely improved compared with the component codes. The process of exchanging information between the component decoders continues until both decoders converge to valid codewords, or a maximum number of super iterations is reached. In the latter case, the output from the second component decoder is declared as the best estimate of the transmitted sequence.

7. Conclusion
The turbo concatenated codes using two same degree distribution pairs of irregular LDPC codes constructed by the PEG algorithm as the components has very good BER performance at low SNR, compared with PCGC, turbo codes and irregular LDPC codes, we may obtain 0.2dB improved in BER at BER= 10-5 at same code length and code rate. The complexity in encoding is reduced compared with PCGC, the complexity in decoding is reduced compared with turbo codes .
