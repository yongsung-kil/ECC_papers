A Probabilistic Subspace Approach to the Minimal Stop­ ping Set Problem
Xiao-Yu Hu and Evangelos Eleftheriou
IBM Research, Zurich Research Laboratory, 8803 Riischlikon, Switzerland Email: {xhu. ele}<Czurich.ibm.com

Abstract
Determining the minimum size of stopping sets of a linear code is an important and yet difficult combinatorial problem for which the exhaustive search requires exponentially growing complexity. We propose a probabilistic subspace approach that can be applied to discover stopping sets of small size in a linear code, particularly for low-density parity-check (LDPC) codes.

L OW-DENSITY parity-check (LDPC) codes were originally introduced by Gallager [1]. Interest in them has recently been revived because of the suc­ cess of iterative decoding for TUrbo codes [2]. LDPC codes are competitors of these codes, as their perfor­ mance with iterative decoding closely approaches the Shannon limit [3�5].
A binary (n, k) LDPC code, in its broader defini­ tion, is a linear block code with dimension k described by a sparse m x n parity-check matrix H, i.e., H
has fewer Is than Os. Each symbol (variable node) is checked by a small number of parity checks, and each parity check (check node) includes a small number of symbols.
The success of LDPC codes lies in the strength of the iterative decoding (or alternatively message­ passing) algorithm, i.e., its low complexity. The al­ gorithm operates locally on a so-called Tanner graph
representation [6] of H, which is extremely efficient.
However, because it acts locally, the algorithm is only suboptimal and sometimes cannot correct error pat­ terns that a maximum likelihood (ML) decoder can. In the case of the binary erasure channel (BEC), the decoding performance of LDPC codes under the iter­ ative decoding algorithm has a purely combinatorial characterization. Certain subsets of variable nodes in the Tanner graph of the LDPC code, called stop­ ping sets [7], are precisely those subsets that can lead to undetermined variable nodes after iterative decod­ ing. Given the knowledge of the stopping sets and the erasure probability of the BEC channel, the per­ formance of the code is completely determined. In particular, the performance in the error floor region is dominated by the minimum-size stopping sets. While stopping sets do not behave in exactly the same man­

codewords, see [8�1O].
To understand the performance of a linear code un­ der ML decoding, it is crucial to know the minimum­ weight codewords. In the case of iterative decoding over the BEC, it turns out to be necessary to know the minimum-size stopping sets. For (semi-) randomly generated nontrivial codes, an exhaustive search is currently unfeasible. A naive Monte-Carlo approach would be to run repeated iterative decoding simula­ tions of the code over BEC with low erasure prob­ ability and to record the stopping sets encountered. This method can indeed reveal information on stop­ ping sets to some extent if the simulation process is properly tuned; however it is inefficient as most sim­ ulation runs return no stopping sets at low erasure probability. Furthermore, as the stopping sets appear in the simulation in a probabilistic way, the probabil­ ity of the size of a specific stopping set being encoun­ tered or captured is as important as the stopping set itself. Unfortunately that information is missing in the Monte-Carlo simulation approach.
This paper proposes a probabilistic subspace ap­ proach to discover stopping sets in a linear code with a specific parity-check matrix representation, partic­ ularly LDPC codes. It not only provides an efficient method to find small-size stopping sets, but also gives the probability that any presumed stopping set of size s is missed by the approach. Another important fea­ ture of this approach is its simple inherent parallelism, i.e., it can be run individually with independent ran­ dom seeds and the results can easily be combined to handle large block-length, high-dimension codes, ex­ ploiting the emerging economical grid-computing in­ frastructure.

5 Conclusion
The minimum-size stopping sets essentially play the same role in iterative decoding over the BEC as the minimum-weight codewords do in ML decoding. Not­ ing that the set of codewords is a subset of the set of stopping sets, the minimum-weight codewords can potentially be obtained based on a good knowledge of stopping sets. Moreover, the size of the minimal stopping set, with a very low miss probability, can at least serve as a lower bound on the minimum distance of the code in a probabilistic sense.
This paper proposes a probabilistic subspace ap­ proach to the minimal stopping set problem. The minimum size of stopping sets is obtained in the form of miss probabilities for smaller presumed stopping sets. The probabilistic subspace approach consists of a random sampler, which essentially converts a larger subspace into a smaller manageable one, and of an algorithm to find stopping sets in a subspace that ex­ ploits the efficiency of the well-known iterative erasure decoding. The probabilistic subspace approach can be applied recursively and in a massively parallel way.
