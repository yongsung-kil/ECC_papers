RATE-COMPATIBLE PUNCTURING FOR FINITE-LENGTH LOW-DENSITY PARITY-CHECK CODES WITH ZIGZAG PARITY STRUCTURE
Song-Nam Hong, Jaeweon Cho, and DS Park Advanced Research Lab
Telecommunication R&D Center Samsung Electronics Co. Suwon, Korea {sn7955.hong, jaeweon.cho, dspark}@samsung.com

ABSTRACT
In this paper we investigate the puncturing scheme of lowdensity parity-check code with zigzag parity structure (ZLDPC code) using the fact that two check nodes can be merged if they are connected to the same parity node. By applying this result to Tanner graph of punctured Z-LDPC code, we can obtain simple Tanner graph, called effective Tanner graph (eTanner graph), which does not include the punctured parity nodes. Based on degree distributions of eTanner graph of punctured Z-LDPC code, we propose a simple algorithm to design good rate-compatible puncturing for finite-length Z-LDPC code. It is shown that the proposed algorithm is the optimal for Z-LDPC code and the designed rate-compatible Z-LDPC code outperforms the ratecompatible Turbo code adopted in 3GPP.
I. INTRODUCTION
Low-density parity-check (LDPC) codes were introduced by Gallager [1] and rediscovered by MacKay and Neal [2]. Richardson, Shokrollahi, and Urbanke showed that irregular LDPC codes could have better performance on the AWGN channel than Turbo codes, which was nearly close to the Shannon limit. Also, the decoding complexity of LDPC codes is less than that of Turbo codes since LDPC decoding is based on sum-product algorithm. However, the encoding complexity of LDPC codes is quadratic in the block length, which results in slow encoding. Therefore, efficient encoding algorithm has been studied [3] and various fast-encodable LDPC codes with zigzag parity pattern were proposed [4], [5], [6], [7], which are denoted by Z-LDPC codes in this paper.
Puncturing is a simple and efficient scheme to obtain higher code rates from a mother code. In [8], good puncturing schemes for irregular LDPC codes have been proposed, which are based on degree distributions of irregular LDPC codes and it was shown that there exist capacity-approaching puncturing patterns over a wide range of code rates. In [9], [10], the puncturing patterns for moderate block lengths are studied, in which the locations of punctured bits were arbitrary chosen. Recently, a systematic puncturing scheme for LDPC codes with small lengths was proposed [11], in which the locations of punctured bits are determined such that the punctured nodes receive the so-called reliable messages from the unpunctured neighboring nodes in the smallest number of iterations.
In this paper, we verify that two check nodes of Z-LDPC code can be merged if they are connected to the same punctured parity node. By applying this to the Tanner graph

of punctured Z-LDPC code, we can remove all punctured parity bits and obtain simple Tanner graph which can be used to derive the effective degree distributions of punctured ZLDPC codes. We call this new Tanner graph an effective Tanner graph (eTanner graph) of punctured Z-LDPC code. Based on these degree distributions, we propose an algorithm to find the optimal locations of punctured parity bits. It is shown that proposed algorithm is more suitable to Z-LDPC code than the previous algorithm [11] for finite-length LDPC code. As an example, rate-compatible Z-LDPC code achieving code rates 1/3, 1/2, and 2/3 is constructed, which shows better performance than rate-compatible Turbo code adopted in 3GPP2.
The remainder of this paper is organized as follows. In Section II, we introduce an effective Tanner graph of punctured Z-LDPC code. In Section III, we propose a simple but efficient algorithm to find the locations of punctured parity bits in finite-length Z-LDPC codes. In Section IV, by using proposed algorithm, we design rate-compatible ZLDPC code which is compared with previous algorithm and rate-compatible Turbo code adopted in 3GPP. Section V presents conclusions.

V. CONCLUSIONS
In this paper, we verify that two check nodes can be merged if they are connected to the same punctured parity node. Based on the fact, we introduce an eTanner graph of punctured Z-LDPC code, which shows effective degree distributions of Z-LDPC code. By using eTanner graph, we propose a criterion to find the optimal puncturing patterns. By criterion, we propose a search algorithm to find the optimal locations of punctured nodes in Tanner graph for finite-length Z-LDPC code. We show that our algorithm is better than pervious algorithm for finite-length LDPC code. Furthermore, it is shown that rate-compatible Z-LDPC code constructed by proposed algorithm outperforms the rate-compatible Turbo code at code rates 1/3, 1/2, and 2/3.
