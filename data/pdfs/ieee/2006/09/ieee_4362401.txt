Optimized Puncturing Distributions for long LDPC
Codes and Different Channels
G. Richter, A. Hof, and C. Huppert

Abstract- In this paper, we optimize puncturing distributions for long low-density parity-check (LDPC) codes for transmissions over different channels. For this purpose, we use a generalization of the discretized density evolution. We show by the thresholds calculated with the generalized discretized density evolution and by simulations that even for the same LDPC code the optimized puncturing distributions vary considerably for transmissions over different channels.

The paper is organized as follows: In Section I, some important definitions and notations for LDPC codes are given, while Section II recalls the generalized discretized density evolution described in [9]. After optimizing the puncturing distributions for an LDPC code and transmissions over different channels, we show some simulation results in Section IV. Finally, we conclude our paper in Section V.

INTRODUCTION
In mobile communications, the data bits are transmitted over a time-varying channel. One strategy to achieve high throughput and reliability is to adapt the rate of the forward error correction code according to the quality of the channel. Therefore, codes of different rates are needed, which can be effectively achieved by using one mother code with a low rate and puncture it to get higher rates.
Low-density parity-check (LDPC) codes are considered as candidates for such codes. They were originally invented by Gallager in 1963 [1] and rediscovered by Mackay and Neal [2] and also by Wiberg [3] in 1996. It is shown [4], [5], [6] that especially irregular LDPC codes are capable of reaching a performance close to the channel capacity at low complexity, when they are decoded by an iterative decoding algorithm, the so-called belief propagation algorithm. To optimize irregular codes for any memoryless channel, a method is introduced in [5], called density evolution, which can be effectively implemented as a discretized density evolution [7].
Ha et al. [8] describe a modification of the density evolution with Gaussian approximation such that asymptotic thresholds can be calculated for an ensemble of punctured LDPC codes. This ensemble is defined by the variable node degree distribution, the check node degree distribution, and the puncturing distribution. Furthermore, optimized puncturing distributions are given for infinite long LDPC code ensembles.
In this paper, we use the method described in [9] to calculate the thresholds for punctured LDPC code ensembles and a given channel. Furthermore, we optimize the puncturing distributions for one LDPC code ensemble for transmissions over different channels. Here, a punctured LDPC code ensemble is defined by the generalized check node degree distribution and the puncturing distribution. The threshold of such a punctured LDPC code ensemble is calculated with a generalization of the discretized density evolution.

I. DEFINITIONS AND NOTATIONS
An LDPC code of length n and dimension k can be represented as Tanner graph [10]. This graph consists of two sets of nodes connected by edges. One set, the variable nodes, corresponds to the n columns of the parity-check matrix H and the other set, the check nodes, represent the rows of H. The number of rows in H is denoted by m, where m > n- k. A one in the parity-check matrix in row j and in column i corresponds to an edge between the i-th variable node vi
and the j-th check node cj. The number of edges incident to
the i-th variable node vi is called the variable node degree
d(vi), which is equal to the number of ones in column i.
Similarly, the number of edges connected with the j-th check
node cj is called the check node degree d(cj) and is equal
to the number of ones in row j. This is demonstrated in the following example.

V. CONCLUSIONS
In this paper, we modified the generalized discretized density evolution such that thresholds for punctured LDPC codes can be calculated not only for a transmission over the additive white Gaussian noise channel, but also for the uncorrelated Rayleigh channel with ideal side information and for the binary symmetric channel. We found out that even for the same LDPC code ensemble defined by the generalized check node degree distribution the optimized puncturing distributions vary considerably for transmissions over different channels. It was illustrated by simulations and by calculating the thresholds that puncturing distributions leading to a good performance for a transmission over one channel, result in very bad error rates for a transmission over another channel.
