High Performance Simplified LDPC Decoder without Knowledge of Multi-path Rayleigh Channel

Tsuguo Maru
System Platform Research Laboratories, NEC Corporation 1753 Shimonumabe, Nakahara-Ku, Kawasaki, Kanagawa 211-8666, Japan Tel: +81-44-396-2422, Fax: +81-44-431-7609, E-mail: maruibcjp.nec.com

Abstract This paper presents a high performance simplified LDPC decoder which doesn't need knowledge of multi-path channels and is robust over level mismatches between demodulator and decoder. This decoder also has the benefit of avoiding a high error floor caused by the approximation in the Max-Log domain known as the min-sum algorithm in categorized Tanner graphs. To overcome error floor and to get the robustness, we apply some individual coefficients to LLR values in every category. Simulation results in AWGN and multi-path fading channels show that even though categorized low profile and approximations are used, good performance is achieved, which can not be achieved by ordinary single coefficient methods (e.g., normalized BP algorithm). Unlike the recently developed simplified algorithms which use the noise variance of the channel estimate, simulation results show that the proposed multi-coefficient method which doesn't use the noise variance is robust over varying fading conditions.
I. INTRODUCTION
LDPC decoders which use iterative techniques provide large coding gains and thus are attractive for mobile communications systems. As the number of users increases, a capacity improvement of the radio interface will be required for enabling high-speed/large-capacity services such as content delivery services, high quality multi-media communications and so on [1]. That is, we should consider not only increasing user capacity which relates to coding gains, but also increasing peak data rate per user and total cell throughput in the low decoder complexity.
LDPC codes, which have the best known theoretical performance (0.0045dB to Shannon limit) [2], are great candidates for mobile communication systems. Though the performance of irregular type LDPC is closest to the Shannon limit, this performance is obtained at the expense of very large frame lengths (1 06_ 107) and high degree orders(1 02), which means that the complexity of the LDPC decoder is very high and infeasible for mobile communication equipment.
Some categorized methods which specify the numbers of ones in disjointly divided square sub-matrices of the parity check matrix have been proposed as in Fig.1 [3], and the recent advent of a new framework brings the possibility of achieving good performance while featuring a low complexity with the help of a socket model using categorized methods as in Fig. 1 [5] and Fig.2 [4].
From the tuning process of this socket model using the sum-product algorithm, the simulation result shows that the

performance of these categorized methods outperforms the well tuned 3GPP turbo decoder with the several thousand
frame length and a few degree orders.

II. AN APPROACH FOR ACHIEVING HIGH PERFORMANCE
LDPC DECODER FOR MOBILE COMMUNICATION
Spectrum efficiency reflects the bit cost of the cellular systems and the coding gain is one of the major factors of spectrum efficiency. To achieve the large coding gain, we took the
general framework and density evolution strategy in [4] to find the suitable code for mobile communication using categorized
Gaussian approximation shown in Section 3. We should consider not only increasing the coding gain but also reducing the complexity with the help of low degree profiles. Several categorized density evolution methods have been proposed [4] [5] and good performance is obtained with a few degree orders. However, the greatly approximate algorithm in the Max-Log domain called normalized BP algorithm [7] causes error floors. To combat the error floor and to achieve good performance, we propose a multi-coefficient approach in the categorized Tanner graph shown in Section 2.A which is a general method and not limited to the code design such as [4] and [5], but this method can not be used by the standard irregular LDPC because of its randomly distributed degree profile. The procedure for achieving high performance LDPC design for mobile communication is summarized as follows.

[Multi-coefficient approach] Use GA in the Max-Log domain to find multi-coefficients in every cate-
gory to overcome error floor. (rough estimation of multi-coefficients)

[Tuning the Multi-coefficients] Optimum multi-coefficients tuned by Monte Carlo simulation are obtained
around the above GA results. (tuning the multi-coefficients)

The proposed method operates in a relative manner. It doesn't use absolute values and the noise variance 02 from the
varying fading channel. Unlike the recently developed simplified algorithms which
use the 02, the proposed multi-coefficient method is robust over the varying fading conditions.
To confirm that only one multi-coefficient parameter set is adequate even in the fading conditions, OFDM based Monte Carlo simulations have been conducted. Table 2 shows the
parameters. We evaluated the performance using an exponential decay
model with eighteen-path Rayleigh fading profile with the fading maximum Doppler frequency of 20Hz, in which the average signal power was reduced by 1 dB in descending order as shown in Fig.12.
From the R= 1/3 and 1/2 simulation results shown in Fig. 13, the proposed multi-coefficient LDPC is nearly equal to the ideal belief propagation LDPC. The ideal BP LDPC uses knowledge about channel characteristics but the proposed multi-coefficient LDPC doesn't use it. The proposed methods can achieve more than 3dB gain (R= 1/3) and 1db gain (R=1/2) respectively to the well turned single coefficient LDPC.

VI. CONCLUSION
Multi-coefficient methods provide large coding gain and a low degree profile LDPC decoder which is suitable for mobile communication systems to increase user capacity and peak data rate through low complexity. Since this method does not require any knowledge of the channel noise, it brings good performance in fading conditions. This method can not be used by the standard irregular LDPC because of its randomly distributed degree profile. Hence, this method is suitable for the categorized Tanner graph.
