Improved Decoding of LDPC Coded Modulations

Yaniv Nana, Eran Sharon, Student Member, IEEE, and Simon Litsyn, Senior Member, IEEE

Abstract— A Coded Modulation Belief Propagation (CMBP) decoder is proposed for decoding LDPC codes with multilevel modulations. The decoder takes into account statistical dependencies among bits originating in the same symbol, providing better performance than the Marginal BP (MBP) decoder. Asymptotically it converges to MAP decoding. The CMBP decoder is based on a Single-Level Coding (SLC) scheme and does not suffer from practical disadvantages of Multi-Level Coding (MLC) schemes. Furthermore, the CMBP decoder can close the capacity gap of the Bit Interleaved Coded Modulation (BICM) SLC scheme. The BICM capacity gap increases when the modulation size increases and in scenarios where Gray mapping is not possible.
Index Terms— LDPC codes, bit interleaved coded modulation, iterative decoding.
I. INTRODUCTION
L DPC codes can approach the capacity on various channels at relatively low complexity using iterative decoding algorithm [4], [3]. Various coding schemes and decoding strategies were proposed for LDPC coded modulations in order to achieve both power and bandwidth efficiency.
Use of non-binary LDPC codes was considered in [1]. This approach provides good performance but incurs high decoding complexity compared to binary LDPC codes. A common approach when using binary codes is Multi-Level Coding (MLC) using different rate compatible LDPC codes for encoding each constellation mapper input bit stream, together with Multi-Stage Decoding (MSD) [11], [6], [7]. Although MLC/MSD scheme is theoretically optimal, in practice it suffers from several disadvantages: 1) The component codes’ rate design depends on the Signal-to-Noise Ratio (SNR) conditions of the channel. Thus, the MLC/MSD approach does not fit well to channels with changing SNR conditions; 2) Appropriate component rates are hard to design, resulting in error propagation between decoding levels; 3) The component codes are shorter compared to a Single-Level Coding (SLC) approach, resulting in degraded LDPC code performance; 4) MSD incurs high decoding latency.
In [11], [2] it was shown that MSD can be replaced with Parallel Independent Decoding (PID) at each level separately with a very small loss in performance if Gray mapping is employed, due to low statistical dependence between label bits. As shown in [2], assuming ideal interleaving the pragmatic Bit Interleaved Coded Modulation (BICM) SLC scheme is equivalent to MLC/PID, providing good performance when Gray mapping is used. The SLC BICM scheme does not suffer from practical disadvantages of MLC schemes. However, the gap between BICM capacity (which is equal to the MLC/PID

For example, while the capacity gap for a 4-PAM modulation at spectral efficiency of 1 bit/symbol is only 0.16dB, the capacity gap for a 32-PAM modulation at spectral efficiency of 2.5 bit/symbol is 0.91dB. Furthermore, due to high sensitivity of BICM to the mapping, it will not work well in scenarios where Gray mapping is not possible, such as 32-QAM and 128QAM modulations. For example, for a 32-QAM modulation with a quasi-Gray labelling the BICM capacity gap is 2.6dB.
In this paper we propose a pragmatic SLC coding scheme that can approach the modulation constraint capacity (i.e. is not limited by the BICM capacity). In BICM based on LDPC codes a Belief-Propagation decoder can be used, initialized with marginal bit probabilities (or LLR values). We refer to this decoding scheme as Marginal BP (MBP) decoding. BICM is suboptimal since MBP decoding ignores statistical dependencies between bits mapped to the same symbol. Motivated by the MBP decoding sub-optimality we derive an improved BP decoding algorithm which combines decoding and demodulation, called Coded Modulation BP (CMBP). CMBP decoding asymptotically (in the code length) achieves MAP performance. Similar ideas, combining decoding and demodulation, were used in [9] for LDPC convolutional codes and in [5] for the multiple antenna channel. We compare the performance of SLC-CMBP and SLC-MBP decoding for finite length codes using simulations and show increasing performance improvement when the constellation size increases and when non-Gray mapping is used.
