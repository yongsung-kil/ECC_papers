Optimized Asymptotic Puncturing Distributions for Different LDPC Code Constructions
G. Richter, S. Stiglmayr, and M. Bossert
University of Ulm Department of Telecommunications and Applied Information Theory
Albert-Einstein-Allee 43 D-89081 Ulm, Germany {gerd.richter,stephan.stiglmayr,martin.bossert}@uni-ulm.de

AbstractвАФ In this paper, we describe a method, how to optimize the asymptotic puncturing distributions for low-density paritycheck codes constructed with different algorithms. Therefore, we generalize the discretized density evolution such that we can take care of the structure of the code. We show by density evolution and by simulations that even for the same degree distributions the optimized asymptotic puncturing distributions vary considerably for different construction algorithms. Furthermore, we demonstrate the performance gain by using the designed puncturing distributions compared to known puncturing distributions.
I. INTRODUCTION
In mobile communications, the data bits are transmitted over a time-varying channel. One strategy to achieve high throughput and reliability is to adapt the rate of the forward error correction code according to the quality of the channel. Therefore, codes of different rates are needed, which can be effectively achieved by using one mother code with a low rate and puncture it to get higher rates.
Low-density parity-check (LDPC) codes are considered as candidates for such codes. They were originally invented by Gallager in 1963 [1] and rediscovered by Mackay and Neal [2] and also by Wiberg [3] in 1996. It is shown [4], [5], [6] that especially irregular LDPC codes are capable of reaching a performance close to the channel capacity at low complexity, when they are decoded by an iterative decoding algorithm, the so-called belief propagation algorithm. To optimize irregular codes for any memoryless channel, a method is introduced in [5], called density evolution, which can be effectively implemented as a discretized density evolution [7].
It is clarified experimentally [10] and by a generalized density evolution [11], [12], [13] that the performance of irregular LDPC codes in the waterfall region can be improved by additionally taking care of the structure of the code.
Ha et al. [8] describe a modification of the density evolution with Gaussian approximation such that asymptotic thresholds can be calculated for an ensemble of punctured LDPC codes. This ensemble is defined by the variable node degree distribution, the check node degree distribution, and the puncturing distribution. Furthermore, optimized puncturing distributions are given for infinite long LDPC code ensembles.
In this paper, we describe another method, which is a generalization of the discretized density evolution, to obtain asymptotic thresholds for irregular punctured LDPC codes.

In contrast to [8], we take not only the check node, the variable node, and the puncturing distribution into account, but also the structure of the LDPC code that follows from different construction algorithms. We optimize the puncturing distributions for different LDPC codes and show that the optimized puncturing distribution strongly depends on the construction algorithm. For all codes the optimized puncturing distributions result in better thresholds and in lower error rates compared to the distributions obtained by Ha et al. in [8].
The paper is organized as follows: In Section II, some important definitions and notations for LDPC codes are given, while Section III deals with the generalization of the discretized density evolution. After optimizing the puncturing distributions for different LDPC codes and calculating the thresholds in Section IV, we show some simulation results in Section V. Finally, we conclude our paper in Section VI.
II. DEFINITIONS AND NOTATIONS
An LDPC code of length n and dimension k can be represented as Tanner graph [9]. This graph consists of two sets of nodes connected by edges. One set, the variable nodes, corresponds to the n columns of the parity-check matrix H and the other set, the check nodes, represent the rows of H. The number of rows in H is denoted by m, where m вЙ• n вИТ k. A one in the parity-check matrix in row j and in column i corresponds to an edge between the i-th variable node vi and the j-th check node cj. The number of edges incident to the i-th variable node vi is called the variable node degree d(vi), which is equal to the number of ones in column i. Similarly, the number of edges connected with the j-th check node cj is called the check node degree d(cj) and is equal to the number of ones in row j. This is demonstrated in the following example.

IV. OPTIMIZATION AND THRESHOLDS
In this paper, we use an algorithm similar to that described in [7] to optimize the puncturing distributions ѕА(0)(x) for LDPC codes with a given generalized check node degree distribution dc and a given p(0). We maximize the thresholds of the LDPC codes while maintaining the following constraints:
1) The puncturing distribution must be chosen such that (3) is fulfilled.
2) The new ѕА(0)(x) is slightly different from the old ѕА(0)(x).
3) The new ѕА(0)(x) results in a higher threshold.
To get comparable results, we use an LDPC code ensemble of dimension k = 50000, length n = 100000, and with the variable and check node degree distribution taken from [8], i.e., ќї(x) = 0.25105x + 0.30938x2 + 0.00104x3 + 0.43853x9 and ѕБ(x) = 0.63676x6 + 0.36324x7. The total fraction of punctured nodes is p(0) = 0.25, which results in an LDPC code with rate R = 2/3. The first LDPC code (Random) was constructed completely randomly avoiding cycles of length 4. The second LDPC code (Zigzag) was constructed by connecting all degree-2 variable nodes in a zigzag manner and by choosing the other edges randomly, while avoiding cycles of length 4. The third LDPC code (PEG) was constructed with the Progressive-Edge Growth algorithm described in [14].

VI. CONCLUSIONS
In this paper, we introduced a method, how to optimize the asymptotic puncturing distributions for LDPC codes constructed with different algorithms. Therefore, we generalized the discretized density evolution such that it takes not only the variable and the check node degree distribution into account, but also the structure of the LDPC code described by the generalized check node degree distribution. We found out that the optimized puncturing distributions vary considerably for LDPC codes that belong to the same ensemble defined by the conventional degree distributions, but are constructed with different algorithms. It was illustrated by simulations and by calculating the thresholds that puncturing distributions leading to a good performance for one code, result in very bad error rates for another code. Furthermore, using the optimized puncturing distributions showed a performance gain compared to using existing puncturing distributions.
