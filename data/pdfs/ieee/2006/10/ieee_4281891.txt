Channel Mismatch Effect on Performance of Low Density Parity Check Codes

LIU Qi, GONG Chen, CUI Huijuan, TANG Kun Department of Electronic Engineering Tsinghua University, BeiJing, P.R.China 100084
Phone: +86-10-62781417, E-mail:

Abstract - Effect of channel mismatch on decoding performance of low density parity check (LDPC) codes over binary symmetric channel (BSC) under sum-product algorithm (SPA) is analyzed in this paper. Density evolution is employed in theoretical analysis to get optimal initial value for SPA. Based on the results, a modified SPA algorithm is proposed. When decoding using this modified SPA algorithm, no channel information is needed at the decoder side. This will be of significant value for time-varying channels. Theoretical claims from density evolution are confirmed by extensive simulations. Our simulations show that by fixing the initial value for all channel conditions, decoding performance can be improved. One channel state estimation algorithm is proposed in this paper based on the error detection ability of LDPC codes. Maximum number of iterations taken when decoding a LDPC code can be dynamically adjusted according to the estimated channel state. Further performance improvements can be achieved by using the dynamic iteration number adjusting scheme together with the modified sum-product algorithm.
Keywords: channel mismatch; sum-product algorithm; binary symmetric channel; low density parity check codes; channel estimation
I. INTRODUCTION
Low density parity-check (LDPC) codes, originally invented and investigated by Gallager [1], lay dormant for nearly 30 years, except for papers by Zyablov and Pinsker [2], Margulis [3], and Tanner [4]. Iterative decoding, as one innovations by Gallager but forgotten for decades, are capable of achieving a significant fraction of channel capacity at low complexity. Interest in LDPC codes was rekindled by the discovery of turbo codes and LDPC codes were independently rediscovered by both MacKay and Neal [5] and Wiberg [6]. During the past few years many new developments have been brought in this area. In several papers, Luby, Mitzenmacher, Shokrollahi, Spielman, and Stemann introduced new tools for the investigation of message-passing decoders for the binary-erasure channel (BEC) and the binary-symmetric channel (BSC) under hard-decision message-passing decoding [7] [8], in which Gallager's definition of LDPC codes is extended to include irregular codes. Density evolution, introduced in [10] to analyze capacity of LDPC codes under message-passing decoding algorithm, can be used to search for good capacity approaching codes.
For sum-product algorithm (SPA) decoding LDPC

codes, knowledge of channel state is necessary to achieve the ultimate performance of the algorithm. Channel mismatch occurs when we encounter an error in estimating the channel state parameter, which will be addressed in this paper. Performance of SPA with imperfect channel estimation was addressed in different papers [1 1-13] for Turbo codes and in [14-15] for LDPC codes. Simulation results for finite block length codes have been presented in [14]. In this work, we perform asymptotic analysis of SPA with channel mismatch for both regular and irregular codes. We demonstrate that our asymptotic results are in consistent with finite-length simulations. Based on analysis and simulations, a modified SPA algorithm with initial value independent of channel error probability is proposed. Compared with SPA, the modified algorithm shows meaningful performance improvements while holding less sensitivity to channel conditions. Also proposed is one channel state estimation algorithm utilizing the error detection ability of LDPC codes. Maximum number of iterations can be adjusted dynamically according to channel conditions. Further performance improvements can be achieved by combining this scheme with the optimal initial value. SPA with initial value independent of channel conditions serves as a good candidate for communication in time-varying channels.
The paper was organized as follows, in Part II the modified SPA algorithm was introduced, and channel mismatch effect on LDPC codes decoding using SPA was analyzed in Part III. Simulation results were given in Part IV. And Part V concludes this paper.

At the decoder side, when a codeword is received, first
use Hk =z to check how many check nodes failed, thus
can get the channel state information at the decoding time. As shown by Mackay, short cycles in the parity check matrix H have detrimental effects on the performance of LDPC codes. At relatively good channel state compared with the threshold, fewer iterations is required for successful decoding, if after some iterations (i.e. 10 from our experiment), the decoder declares a failed decoding, then even we increased the number of iteration, it's mostly likely that the decoder will still failed the decoding. Since this fail is dude to the short cycles in the parity check matrix. According to this property of LDPC codes, we increase the iteration number only when it's necessary. So the average iteration number is decreased without the decoding performance being impaired. Since computation complexity is proportional to the average iteration numbers, the computation complexity is reduced when the average iteration numbers is decreased. In figure 3, we present the simulation result comparing the performance of
LDPC code when the initial value of SPA is set as the
actual channel crossover probability, the optimal value determined in Part III of this paper and the optimal value together with dynamic adjustment of the maximum iteration number. From our simulation result, we found that when using the optimal initial value for SPA, the performance of LDPC code was slightly improved. It's worthy to be noted that when using the optimal value to initialize p0 in SPA, no channel information is needed. If combined with the strategy of dynamic adjusting the
maximum iteration number based check failed nodes of
the received message, the performance can be further improved. In our experiment, the maximum iteration number is 0 when the channel state is good, and improved to 20 when it gets worse. According to our

experiments, when more than 15 2000 check nodes failed the check, we say that the channel state is worse and will improve the maximum iteration number.

V. CONCLUSION
In this paper, we analyzed the channel mismatch effect on the performance of LDPC codes. And density evolution is employed to help our analysis. The effect of the initial value in SPA is investigated and an optimal
value for initializingp is achieved. Based on our
analysis, a modified SPA algorithm is proposed. When using the modified SPA algorithm, not only is the
channel state statistics no more needed at the decoder
side, but also better decoding performance can be achieved. Besides, a dynamic iteration number adjusting algorithm based on error detection ability of Low Density Parity Check codes is proposed. Simulation results show that when decoding with the optimized initial value together with the dynamic iteration number adjusting algorithm, the performance of LDPC codes can be further improved.
