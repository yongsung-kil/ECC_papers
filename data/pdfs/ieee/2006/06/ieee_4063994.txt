Multi-Rate Quasi-Cyclic LDPC Codes
Zhichu Lin, Zhixing Yang, Changyong Pan Department of Electronic Engineering, Tsinghua University, Beijing, 100084, China

Abstract- A new method called fractional row combination is proposed to design a group of Multi-Rate Quasi-Cyclic LDPC codes in this paper. Unlike the original row combination method, it guarantees all the codes under different rates have concentrated check-node degree distributions, hence better performance under iterative decoding. Moreover, a combinatorial method based on BIBD is applied to our design. Together with fractional row combination, a family of multi-rate quasi-cyclic LDPC codes can be constructed. A low-complexity architecture of the universal decoder for all rates is also introduced.

Communication systems usually need to work on different rates to provide different qualities of service. In order to reduce the complexity of decoder, it is desirable to design a family of error correction codes with different rates, which share some common structural properties. One of the approaches to this goal with low-density parity-check (LDPC) codes [1] is to use Rate-Compatible LDPC codes. Rate-compatible LDPC codes can be constructed by either puncturing parity bits of a lower rate code, or extending the parity-check matrix of a higher rate code [2] [3]. Both methods will result in a family of codes with different rates and different code lengths, and the code with the shortest code length may suffer a large degradation in its performance. An alternative approach is to generate higher rate codes from a lower rate code (called "mother code"), by linearly combining rows of its parity check matrix, as proposed in [4]. In this way, all the codes with different rates have the same code length and variable-node degree distribution. A disadvantage of this method is that it may generate a higher rate code with a non-concentrated checknode degree distribution, which may degrade its performance under iterative decoding, according to the results of density evolution [5].
In this paper, we introduce fractional row combination, which is more flexible than the original row combination method, and it guarantees that the higher rate LDPC codes always have check-node degree distributions as concentrated as possible. A simplified architecture of universal decoder is also proposed, which has a very small increase in complexity compared with a decoder for a single-rate code. Furthermore, a quasi-cyclic mother code is designed using a combinatorial construction based on Balanced Incomplete Block Design (BIBD). A degree distribution is said to be concentrated if every node has the same degree. Although the mother code Hs has a concentrated check-node degree distribution, H6 1 does not maintain a concentrated check-node degree distribution. It is because not all the rows in H6 1 are generated by combining the same number of rows. Using density evolution, it has been shown that the check-node degree distribution should be as concentrated as possible, in order to provide best performance. So we introduce our method of fractional row combination, which can guarantee a concentrated check-node degree distribution in that case.
Fractional row combination is more flexible than the original row combination method in that it allows combining part of some row with another row. Generally, fractional row combination is done by splitting some rows into more subrows, and each sub-row will be combined with some original row in the mother code, if they do not contain more than one "1-components" in the same column. The number of "1components" in each sub-row is selected so that the new code after fractional row combination will have a checknode degree distribution as concentrated as possible. For example, we can construct a rate-6/9 code from the mother code using fractional row combination like this: split the first
row into three sub-rows with one "1-component" in each
sub-row, and combine them with the remaining three rows respectively. Every row in the new parity check matrix H6 2 has a row weight of 4, thus a concentrated check-node degree distribution is maintained.

III. COMBINATORIAL CONSTRUCTION OF MULTI-RATE QUASI-CYCLIC LDPC CODES
A. Multi-rate Quasi-Cyclic LDPC Codes
A circulant matrix is a square matrix in which each row is the right-cyclic shift of the previous row, and the first row is the right-cyclic shift of the last row. An example of circulant matrix is given in (10). If the parity-check matrix of an LDPC
code is an array of circulant matrix, it is a Quasi-Cyclic (QC) LDPC code [8]. QC-LDPC codes have a lower implementation complexity in both encoding and decoding compared with random LDPC codes, due to their special structural property, and well-designed QC-LDPC codes can be as good as their random counterparts. An example QC-LDPC code is given in (11), in which the sub-matrix is either a circulant or a zero matrix. A row of circulant matrices in the parity-check matrix of QC-LDPC code is called a row-block.

If a QC-LDPC code is used as mother code, a group of multi-rate QC-LDPC codes can be constructed using fractional

row combination. In order to keep the structural property of the circulant matrix, fractional row combination has to be carried out in a block-wise manner:
* Only combination of row-blocks is allowed. When combining 2 row-blocks B1 and B2, it means the ith row of B1 is combined with the ith row of B2, for all i;
* Only splitting of a row-block is allowed and it can only be split at the boundaries of circulant matrices.

According to the structural property of BIBD, W has a constant row weight r and column weight k, and any two
rows have exactly A "1-components" in the same positions. Generally, the Tanner graph of an LDPC code should contain no length-4 circle. For a BIBD with A = 1, there is exactly one common "1-component" between two rows, so the Tanner graph of its incidence matrix is free of length-4 cycle, and its null space gives a regular LDPC codes with row weight r and column weight k.
Furthermore, QC-LDPC codes can also be constructed using certain kinds of BIBD. We are specially interested in the design method called Bose Design, which was introduced by [9] , because using such a method to design our mother code, we can guarantee that after fractional row combination, all the higher-rate codes are free of length-4 cycle.
We introduce a method of Bose Design following [9]. For any positive integer t, if 12t + 1 is a power of a prime, a Galois field GF(12t + 1) exists. If there exists a primitive element x of GF(12t + 1), and an odd integer c less than 12t + 1, such that x4t_ 1 = x', then a BIBD with parameter
v = 12t + 1,k = 4,A 1,,r = 4t and b = t(12t + 1)) exists.

A problem that may arise from row combination is that even if the mother code is free of length-4 cycle, a new matrix after row combination may contain length-4 cycle.
A simple example is our previously constructed matrix H7,

An advantage of using S as mother code is that the Tanner graph of S, or any higher-rate code after a block-wise fractional row combination of S, which we call S', does not contain any length-4 cycle. It is because if S' or S has a length-4 cycle, it would have two "1-components" in the same positions between two rows. Note that Q can be recovered by combining four row-blocks of S or S into one row-block using block-wise row combination, so Q would also have two "1-components" in the same positions between two rows. But it contradicts that Q is an incidence matrix of a BIBD with A = 1.
It means that if S is selected as the parity-check matrix of mother code, we can construct a group of multi-rate QCLDPC codes by block-wise fractional row combination, and all the codes will have girth length at least 6. No extra methods of eliminating length-4 cycles are needed. All the codes are quasi-cyclic, thus can be encoded and decoded efficiently, and they can be decoded using our universal decoder discussed in Section II-C.

V. CONCLUSION
Multi-rate LDPC codes can be constructed by the method of fractional row combination, which guarantees a concentrated check-node degree distribution for all the codes under different rates. A universal decoder for all rates can be efficiently implemented with a complexity comparable to a single-rate decoder. Furthermore, a combinatorial construction based on BIBD is used to design the quasi-cyclic mother code, which can prevent the existence of length-4 cycle during row combination. By applying fractional row combination to the QC mother code, a group of multi-rate QC-LDPC codes can be constructed.
