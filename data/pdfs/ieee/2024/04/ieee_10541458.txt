Abstract—For some linear codes, neural networks have been found to be a way to enhance traditional belief propagation (BP) decoder. When deploying such methods in practice, quantization issues also need to be considered. Considering that the messages of variable/check nodes are variable during the BP decoding process, this paper proposes a dynamic quantization method to predict the optimal bit-width for the node’s message quantization in different decoding iterations. We build a shallow neural network as a bit-width predictor and train it jointly with a neural belief propagation decoding network. During actual decoding, the bit prediction network will generate appropriate bit-width for quantizing the messages of each layer of nodes. Finally, simulation experiments validate the effectiveness of the proposed quantization method.
Index Terms—belief propagation, dynamic quantization, LDPC codes, neural decoder
I. INTRODUCTION
Low Density Parity Check (LDPC) codes were first introduced by Gallager in 1962, and later Tanner studied LDPC codes from a graph-theoretic perspective [1], which revealed their superior performance. Nowadays LDPC codes have been widely used in communication systems. The recommended decoding algorithm for LDPC codes is belief propagation (BP) algorithm, which can achieve near-optimal decoding performance for long code lengths. For short codes, some recent work has introduced the combination of feedforward neural networks to extend the BP algorithm and its improved algorithms, and obtained performance improvements on some linear block codes [2], [3]. For hardware implementation, the information processed by LDPC decoders is usually quantized, as lower bit-width messages result in lower complexity. There has been a lot of research on this topic, which can be divided into dynamic quantization and static quantization based on whether the quantization parameters change during the iterative process. For dynamic quantization, a (q+1)-bit quasi-uniform quantization method was used to quantify the traditional message iteration decoder of LDPC codes in [4], which can adapt to the dynamic range of messages, i.e., the later messages will use larger bit widths to accommodate

larger message ranges as the decoding iteration progresses. Other studies use adaptive quantization methods with different quantization step sizes to quantize messages [5]. A quantization method proposed in [6] trains a surrogate model of the quantization function using neural network methods to obtain the quantization step sizes of different nodes in the iterative process. Fixed quantization methods include the finite alphabet iterative decoder (FAID) [7], which describes the variable and check node update behavior of BP decoder with a customized update function. Work presented in [8] demonstrated training linear FAID and RQNN parameters using a recurrent quantized neural networks. They performed well on an additive white Gaussian noise (AWGN) channel. Other fixed quantization treats the update of node messages as a binary input discrete memoryless channel (DMC) model [9], and then maximizes the mutual information between quantizer inputs and quantizer outputs by designing update rules [10].
Recently, researchers at Peking University proposed a method for quantizing the weights and activation values of residual neural networks. This method predicts the optimal quantization bit-width for each layer of the network using a bit predictor with low complexity, and then implements dynamic quantization [11]. Inspired by this work, this paper proposes a generic quantization method for quantizing Neural BP (NBP) decoders of linear (LDPC) codes. Different quantization bitwidths are allocated for different code blocks at different decoding iterations to implement dynamic quantization. This method can be used to find the optimal bit-width at each iteration for most fixed quantization schemes. Specifically, a lightweight fully connected neural network is used to predict the number of quantization bits for each iteration of node messages in NBP decoding. The loss function is designed to jointly train the network with the NBP decoding network to obtain more suitable outputs. We verify our method by using a uniform quantizer to quantize the NBP decoder in experiments. The main contributions of this paper can be summarized as follows:
1) The quantization problem of NBP decoders is considered and a uniform quantization of 5 bits is used to quantize the node outputs of the NBP decoder with floating-point precision

II. PRELIMINARIES

A (n, k) LDPC code is generally defined by a parity-check matrix H of size m ∗ n. Each row of the matrix H represents a check equation of the LDPC code, hence it is also referred to as check nodes (CNs). The columns of the matrix H correspond to the encoded code bits, which are known as variable nodes (VNs). By examining the positions of “1”s in the H matrix, a Tanner graph can be constructed. In the Tanner graph, the set of VNs is defined as I {i|i = 1, 2, · · · , n}, and the set of CNs is defined as J {j|j = 1, 2, · · · , m}. The set of VNs connected to CN j is denoted by Nc (j), and the set of CNs connected to VN i is denoted by Nv (i).

A. Fixed-Point Quantization of Neural BP Decoder
There has been a significant amount of work investigating the use of finite-precision messages instead of floating-point messages passed between different nodes (CNs or VNs) in BP decoders, which usually leads to FAID or lookup tables designed for hardware implementation. Such quantization can typically be described by a Markov chain X → Y →Q Z, where X is a node (CN or VN), Y is its observation value, Q is the quantization function, Z is the quantized value, and then minimizing the conditional entropy of X and Z [12] or maximizing the mutual information of X and Z [13], using the quantized value Z instead of the observation value Y during actual decoding. In this section, we do not consider how to design the optimal quantizer Q, but instead use a simple uniform quantizer Q (q, ∆, x) to perform the above process, which is determined by the quantization interval ∆ and the bitwidth q. B. Dynamic Quantization of Neural BP Decoder Messages
The quantization of messages in BP decoders is mostly done using fixed bit-width, although there are a number of dynamic quantization schemes but they mainly focus on quantization parameters other than bit-width. In fact, the range of node messages during BP decoding iterations is dynamically changing and the distribution of messages is also related to signal-to-noise ratio (SNR). To study such dynamically changing messages with iterative decoding, we propose dynamic quantization to dynamically allocate quantization bits according to the parity check matrix of different codewords

and different iterations. Assuming that we provide K bit-width candidates, denoted as q1, q2, · · · , qK , in a certain transmission channel environment, the quantization of messages during a

D. Network Training
In order to enable the newly added bit prediction network to perceive quantized decoding at different iterations, we have carefully designed the length of output logits to be 2 · tmax · K. This allows each quantization layer to correspond to its respective quantization bits. The third term denotes a regularization term from bit prediction network, where wb refers to the trainable parameters within that network. Regularization is employed to constrain the complexity of network, as during the simulation process, it was observed that bit prediction network converges faster, and excessive training can lead to overfitting. α and β are balance coefficients designed to allow the first term to dominate training process.

IV. SIMULATION RESULTS
We constructed the proposed model in python and all the simulation experiments were performed based on Tensorflow. We utilized Adam optimizer with a learning rate of 0.001 for training. After encoding, the coded bits were modulated using binary phase shift keying (BPSK) and passed through an AWGN channel. Sufficient datasets for training and testing were obtained at receiver. The neural decoder was fed with observation values from 2400 code blocks for each training, and after repeating the training for 5000 epochs, the loss function stabilized. During performance testing at least 1000 code block errors occur or at least 12000000 code blocks are simulated before stopping, considering that the maximum number of iterations for conventional BP decoding is either 5 or 10. The quantization interval ∆ = 1 in (10).
The choice of benchmark quantization bit bbenchmark is determined by the selected base quantizer, here 5 bits are chosen when uniform quantization is used. The candidate bits for dynamic quantization are chosen from around bbenchmark, and for this simulation, {4, 5, 6} are selected. Choosing not to use too small bit is because uniform quantization incurs significant performance loss at low bit quantization.
A. Results on LDPC codes We tested the proposed decoding framework on LDPC codes of different lengths, including LDPC(32, 16), LDPC(96, 48), and LDPC(128, 64). All the channel codes used in this paper are from [16]. Each LDPC code was tested for its BER performance using four decoding models: 1) traditional BP decoder, 2) Neural BP (NBP) decoder, 3) Dynamic Quantization NBP decoder based on bit prediction network (DQ-NBP), and 4) fixed 5-bit uniform quantization NBP decoder. As can be seen from the figures, full-precision NBP decoder exhibits varying degrees of performance gain compared to traditional BP decoder on LDPC codes. Similar performance to full-precision NBP decoding can be achieved by using fixed 5-bit uniform quantization. Dynamic quantization based on bit prediction networks, on the other hand, has a small BER loss only at the highest SNR, and can effectively track the performance of fixed quantization at the rest of the SNRs, but requires the same or fewer bits in relative terms. B. Results on BCH codes Since NBP decoder was initially proposed mainly for BCH codes, we provide the BER performance of dynamic quantization NBP decoder on BCH codes with different error correction capability T . The training dataset is obtained from SNR ranging from 1 to 8 dB. All simulations are performed with 5 iterations. The results are shown in Fig. 5 - Fig. 7. It can be seen that the BER curve of NBP decoder using uniform quantization has a significant gap compared to the un7quantized case, but the proposed dynamic quantization framework can still effectively track the performance of fixed quantization, especially for small T there is almost no performance loss compared to fixed quantization. Note that uniform quantization may not be the best base quantization method in this test, but it is simple to implement. Using uniform quantization to test our dynamic quantization scheme has the benefit of being adaptive to the dynamic range of the iterative messages, and we also recommend interested readers to test the proposed quantization framework with other more advanced quantization methods.
V. CONCLUSION
In this paper, we proposed a dynamic quantization scheme for a neural decoder of LDPC codes, where the quantization bit-width of variable/check node messages varies with the dynamic range of messages in different iterations. We used a three-layer fully connected neural network for bit predic-

tion and jointly trained it with the main network of neural
decoder by designing a loss function to predict the optimal
bit-width for decoding. We simulated the proposed dynamic
quantization method using uniform quantization and verified
its effectiveness.
