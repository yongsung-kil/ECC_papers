New Constructions of MDS Array Codes and Optimal Locally Repairable Array Codes
Weijun Fang , Jingjie Lv , Bin Chen , Member, IEEE, Shu-Tao Xia , and Xiangyu Chen

Abstract— MDS array codes have been extensively studied due to their applications in storage systems. In this paper, we first propose a novel method of constructing MDS array codes by deleting one row and one column from the circulant matrices associated to some polynomials. Several new classes of MDS array codes with flexible parameters are constructed. In particular, we give a new algebraic presentation of the Blaum-Roth codes with sparser parity-check matrices. We also obtain a family of MDS array codes over finite fields with even characteristics whose parity-check matrices have the lowest density. Furthermore, based on these new MDS array codes, we give a general construction of optimal locally repairable array codes (LRACs) achieving the Singleton-type bound. Additionally, we obtain some new optimal LRACs of long lengths. Finally, we present a scheduled algorithm for syndrome computations of binary optimal LRACs with redundancy 4, which can tolerate three failures. The number of XORs per data bit required in our algorithm approaches 2 as the length approaches infinity, which is the same as the MDS codes tolerating three failures. However, the number of nodes required during the repair of a failed node in our optimal LRACs is only about half of that in MDS array codes.
Index Terms— MDS array codes, locally repairable array codes, circulant matrices, Blaum-Roth codes, syndrome computation.

I. INTRODUCTION
I N THE Redundant Arrays of Inexpensive Disks (RAID) system [13], [33], [34], single-disk failures are tolerated in RAID-4 or RAID-5. In order to increase data reliability, erasure codes have been widely used in modern distributed storage systems. From a coding point of view, to recover r failures (or erasures), we need at least r parities, which is known as the famous Singleton bound. An erasure code achieving the Singleton bound is called a maximum distance separable (MDS) code. As we know, Reed-Solomon codes ( [36]) are the most important family of MDS codes, which have various applications in practice. However, the encoding and decoding procedures of Reed-Solomon codes are usually performed as operations over a large finite field, which significantly increases the computational complexity. It is desirable that the encoding and decoding can be done by simple cyclic shifts and exclusive OR (XOR) operations without finite field operations. Array codes were introduced to solve this problem [7], [15], which store data into a twodimensional array of bits. The symbols of classical codes correspond to the columns of such an array. Array codes cannot only reduce computational complexity but also are easily implementable in both software and hardware.
The EVENODD codes [3] and the row-diagonal parity (RDP) codes [14] are designed to recover any two erasures. To tolerate three or more failures, the EVENODD codes (resp. the RDP codes) were extended in [4] and [5] (resp. [2], [16]). In [21], Huang and Xu also extended the EVENODD codes to STAR codes. For correcting an arbitrary prescribed number of failures, Blaum and Roth [9] proposed the Blaum–Roth codes, which can be interpreted as Reed-Solomon codes over the residue ring Fq[x]/⟨1 + x + · · · + xp−1⟩. In [18], [19], and [20], some other MDS array codes were constructed based on certain variants of the Blaum-Roth codes.
For a code-based distributed storage system, the update complexity of a linear code is defined as the maximum number of coded symbols that must be updated when any single information symbol is changed ([1]). Specifically, in a linear encoding scheme using a generator matrix, the update complexity is the maximal row weight of the generator matrix ( [30]). To reduce the update complexity, it is desirable to design linear codes with a sparse generator matrix. A generator matrix of a linear code has the lowest density if it has the smallest possible number of nonzero entries. Previous research [5], [10], [27], [40] has focused on investigating the

In [35], based on measurements from Facebook’s warehouse cluster in production, the researchers pointed out that node failures are common and on average, 98.08 % of the failures involve exactly one failed node. Therefore, it is critical to present efficient repair schemes for repairing a single failed node while providing fault tolerance for multiple node failures. To minimize the number of helper nodes during the repair of a single failed node, the authors in [17], [31], and [32] introduced the locally repairable codes (LRCs) and present the well-known Singleton-type bound for such codes. Since then, optimal LRCs achieving the Singleton-type bound have been widely studied. Analogous to Reed-Solomon codes, most of the optimal LRCs were constructed over large finite fields. Therefore, it is meaningful to design new storage codes with both advantages of LRCs and array codes. In [23], the authors introduced the locality of array codes, which we called locally repairable array codes (LRACs). A Singletontype bound for LRACs was also presented. In [37], the authors gave some constructions of LRACs via subspaces. Very recently, Expanded-Blaum-Roth (EBR) [6] codes and Expanded-Independent-Parity (EIP) codes [8] with symbol locally repairable property were also proposed, which can repair one failed symbol within one node by adding some parity symbols. Nevertheless, if some symbols in a node fail, usually, the remaining symbols will no longer be reliable. It is very interesting to further investigate the properties and constructions of LRACs.

A. Our Contribution
In this paper, we first present a novel method of constructing MDS array codes. Specifically, we consider the associated circulant matrix of a polynomial over finite fields and introduce the definition of the deleted circulant matrix of such a matrix by deleting its last row and last column. Then we provide a general construction of new MDS array codes with flexible parameters using the parity-check matrix based on the deleted circulant matrices of specific polynomials with certain properties (see Theorem 2). In the context of an [n, K, d, b]q array code C, a file of size K is encoded into an array c ∈ C in a distributed storage system, and each column of c is stored in a storage node. The minimum distance d of the code ensures that any failure of up to d − 1 nodes can be corrected.
If we read the symbols of codewords column by column, and from the first to the last symbol within each column, then each b × n matrice can be flattened as a row vector of length bn. B. Locally Repairable Array Codes

In [17] and [32], Gopalan et al. introduced the locality of linear codes to minimize the number of nodes participating in a repair process when a node fails. In [23], the authors generalized the locality to array codes.
Definition 2 [23]: Suppose C is an (n, k, d, b)q-array code and i ∈ [n]. The i-th code node of C is said to locality r, if there exists a subset Ri ⊆ [n] such that
• i ∈ Ri, |Ri| ≤ r + 1; • d(C|Ri ) ≥ 2, where C|Ri is the punctured code of C on [n]\Ri. C is called a locally repairable array code (LRAC) with locality r if all nodes of C have locality r, and denote it by an (n, k, d, b; r)q-LRAC. Equivalently, we give an alternative definition. V. CONCLUSION
In this paper, we propose a novel method of constructing MDS array codes by deleting the last row and last column of the circulant matrices associated to some polynomials. We construct a new family of MDS array codes whose paritycheck matrices are more sparse than the famous BR codes. Under certain conditions, we can obtain a family of Fq-linear (n + 1, k, d, p − 1) MDS array codes over Fqp−1 with n that can go up to qp−1. Similarly to classical Reed-Solomon codes, when q is even, we extend them to MDS array codes of length n + 2 and normalized dimension k = 3. Then we apply these results to construct new optimal LRACs achieving the Singleton-type bound with flexible parameters. By using binary constant weight codes, for d = 5, 6, we obtain new optimal LRACs over Fqp−1 of length n = Ωr(q2(p−1)). Finally, we propose an efficient algorithm of syndrome computations for binary LRACs with redundancy 4. In our algorithm, the required number of XORs per data bit approaches 2 as the length tends to infinity.
Reed-Solomon codes are widely recognized as the most crucial class of erasure codes in the field of engineering. The BR codes can be seen as the RS codes over residue rings. The parity-check matrices of these codes are Vandermonde matrices. In [10] the authors presented the construction of MDS array codes with the lowest density by utilizing array Vandermonde matrices. It is worth noting that the parity-check matrix in our Construction 1 is intrinsically

different from the Vandermonde-type matrix, since A[j] ≠ (A[1])j. We summarize some advantages of our new constructions of MDS array codes and optimal LRACs as follows.
• The parameters of our new MDS array codes are flexible. • We present a new algebraic construction of BR codes,
which has a sparser parity-check matrix. • We present a new construction of MDS array codes with
a lowest-density parity-check matrix. • The application of our new MDS array codes allows for
the construction of new optimal LRACs with flexible parameters and long code lengths. • Compared to the MDS array codes with redundancy 3, our optimal LRACs only need about half the number of nodes required to repair a single node, while the complexities of syndrome computations are asymptotically the same. • Compared to RS codes with redundancy 4, our optimal LRACs outperform theirs in terms of both syndrome computational complexity and the number of nodes required to repair a single node failure. The only tradeoff lies in a minor compromise of erasure correction capability. To sum up, our methods and results present an interesting and potentially meaningful research avenue for future studies on the construction of new regenerating codes and low-density parity-check (LDPC) codes.
APPENDIX A PROOF OF THEOREM 5
Firstly, we briefly review some basic results on binary constant weight codes. A binary (n, M, d, w) constant weight code is a set of binary vectors of size M and length n, such that each vector contains w ones and n−w zeros, and any two distinct vectors differ in at least d positions. It is easy to verify that there is a binary (n, M, d ≥ 2w − 2t, w) constant-weight code if and only if there is a family of subsets {Si}iM=1 of the coordinate set {0, 1, · · · , n − 1} such that
|Si| = w and |Si ∩ Sj| ≤ t, for all 1 ≤ i ≠ j ≤ M.
