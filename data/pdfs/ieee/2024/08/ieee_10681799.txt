Construction of Spatially Coupled GC-LDPC Code for NAND Flash Memory
Jinhong Mo, Xiongfei Zhai, Min Yu, Yi Fang, and Guojun Han School of Information Engineering, Guangdong University of Technology, Guangzhou 510006, China Email: 1393044175@qq.com, zhaixiongfei@gdut.edu.cn, 844535734@qq.com, fangyi@gdut.edu.cn, gjhan@gdut.edu.cn

Abstract—With the continuous improvement of data memory density, as the common solution for error correction of NAND flash, the low-density-parity-check (LDPC) code faces more challenges, such as the increasing code length requirements, the worse raw bit error rates (RBER), and the frequently varied channels. In this paper, a new (37536, 33672) spatially coupled and globally coupled LDPC (SC-GC-LDPC) code is constructed to address the above challenge, which outperforms the traditional globally coupled LDPC (GC-LDPC) code with the gaps of 0.065 dB and 0.08 dB by exploiting the layered sumproduct algorithm (SPA) and the layered normalized min-sum (NMS) algorithm, respectively. Moreover, in the flash channel, the SC-GC-LDPC code also shows superior error correction performance as compared with the conventional GC-LDPC codes, which can effectively adapt to flash channels in different pages.
Index Terms—NAND flash memory, GC-LDPC, P/E cycles, SPA, NMS.
I. INTRODUCTION
Since the 21st century, an explosive growth of data has been shown due to the rapid development of information technology. It is indicated that the global data volume is expected to increase from 163 ZB to 175 ZB by 2025, while the corresponding storage capacity will only reach 22 ZB at that time [1]. Thus a significant challenge in terms of storage capability is now an urgent topic. Over the past decade, flash memory media is extensively utilized in the smart phones and the computer storage systems due to its high-density storage, low power consumption, and non-volatility. In tandem with the rapid advancements in NAND flash technology, the storage capacity of a single NAND flash unit has progressively increased from single-level cell (SLC) to multi-level cell (MLC), trinarylevel cell (TLC), and even quad-level cell (QLC). In order to further enhance the density of flash storage, the NAND flash manufacturing process has transitioned entirely from 2D planar technology to 3D vertical stacking technology. The increase in storage capacity and density also renders flash memory units more susceptible to various types of noise, thereby introducing a significant challenge to the reliability of NAND flash memory [2].
Error correction code (ECC) technology serves as an effective approach to enhance the reliability of stored data. The Bose-Chaudhuri-Hocquenghem (BCH) code was the first one to be employed in the solid state disk (SSD) master, which is a hard-decision decoding technique for the channel error correction in SLC and MLC flash memories [3]. The error correction capability of BCH code, however, is limited and falls short in addressing the excessively high raw bit error rates

(RBER). Recently, the low-density-parity-check (LDPC) code is extensively employed for ensuring data reliability, owing to its inherent advantages of facilitating soft decision and exhibiting robust error correction capabilities. In the industry, there is a common consideration of exploiting the gradual transition towards 2KB LDPC as the key ECC technique for the NAND flash memory. However, when the NAND flash memory comes to be TLC and QLC, its error correction capability fails to meet the stringent requirements of enterprise-class SSDs in terms of data reliability. Consequently, 4KB LDPC has emerged as the predominant choice.
In 2016, Juane Li proposed globally coupled LDPC (GCLDPC) codes, employing full-valued matrices for constructing the global matrix of GC-LDPC and the base matrix of local matrices, while utilizing finite fields to construct the shift value matrix [4]. Due to its flexible decoding methods, the GC-LDPC code was soon applied to the flash memory. In 2019, Yen-Chin Liao constructed a new (21150, 19050) GCLDPC code for NAND flash applications [5]. In 2023, Binhao Bao proposed an improved adaptive granularity progressive LDPC decoding (IAGPD) by taking advantage of the (21150, 19050) GC-LDPC code, which can significantly reduce flash read latency [6]. However, the aforementioned works primarily focus on 2KB GC-LDPC codes in the flash channel, which fails to meet the requirement of code length in the rapid development of NAND flash. For encoding, the structure of the check matrix leads to high encoding complexity. Furthermore, the features of flash channels and the unique structure of check matrix are not considered in the above works, which may be exploited to significantly improve the decoding algorithm. To the best of our knowledge, there is still no works reported in the literature to properly address the above issues.
To address the above issues and achieve dynamic error correction as well, we propose a spatially coupled GC-LDPC (SC-GC-LDPC) code. The main contributions of this work are summarized as follows:
• A GC-LDPC code with 4KB code length is constructed by the traditional method, which achieves good error correction ability.
• By exploiting the spatially coupled structure for the construction of the global matrix, a (37536,33672) SC-GCLDPC code is proposed. As compared with the previously GC-LDPC codes, it is shown that the SC-GC-LDPC code exhibits superior error correction features.
• Experimental results of the NAND flash channel demonstrate that the SC-GC-LDPC code can effectively adapt to flash channels with excellent error correction capability

B. GC-LDPC Codes

In recent years, LDPC codes have emerged as a popular choice for NAND flash systems, where the GC-LDPC code is one of the recent innovations in the code design. Based on a sparse check matrix structure, the GC-LDPC code achieves the reduced coding complexity and fast convergence speed. The definition, structure, and characteristics of the GC-LDPC code are briefly elucidated in this subsection.
The GC-LDPC code is composed of multiple shorter LDPC codes, which can be regarded as a specialized QC-LDPC code. The check matrix can be succinctly described by four parameters, namely the dimensions of the local matrix with m rows and n columns, the number s of rows in the global matrix, and the number t of local matrices. In this section, an example of 2KB GC-LDPC code is firstly presented, followed by a similarly constructed 4KB one. Then, the proposed SC-GC-LDPC code is introduced in detail.
A 2KB GC-LDPC code: Considering the finite field GF(151), a matrix S1 of size 150 × 150 can be obtained as shown in (6). Since q−1 can be factored into 151−1 = 3×50, we can construct a GC-LDPC code with three local codes, where the first 4 rows and the first 47 columns of S1 form the identical local matrix. Rows 5 and 6 of matrix S1 are used to construct the global part Bglobal. The resulting matrix is shown in the Fig. 3. Then each column of the matrix is randomly masked, that is, one of the non-zero elements is randomly forced to be zero. Finally, it is extended with a lifting factor 150. It is worth noting that the zero elements in the matrix, need to be extended to the all-zero matrices. This results in a (21150,19050) GC-LDPC code with a code rate of 0.9007.

IV. EXPERIMENT RESULTS
In this section, the error correction performance of proposed (37536,33672) SC-GC-LDPC code in the AWGN channel and NAND flash memory channel is evaluated via the Monte Carlo method.
The discussed 2KB and 4KB GC-LDPC codes in Section III are considered as the benchmarks. Both the sum-product algorithm (SPA) and the normalized min-sum (NMS) algorithm are exploited. To speed up the convergence of decoding and achieve high efficiency of hardware resource utilization, layered decoding [11] is also considered in the experiments. Besides, the maximum number of iterations is set to 20 and the normalization factor of NMS is 0.7 among the simulations, which is applied to both the AWGN channel and NAND flash memory channel.
In our work, random data write is performed to a block within the NAND flash chip and subsequently the numbers of memory cells falling into different threshold voltage ranges can be obtained by exploiting the NanoCycler test platform as shown in Fig. 7, which enables us to acquire the threshold voltage distribution of the actual NAND flash chip. According to the test results, we consider the following default parameter setups. For erase and programming operations, we set the means of the eight threshold voltages to -1.20V, 0.40V, 1.20V, 2.05V, 2.88V, 3.72V, 4.59V, 5.60V, respectively. The corresponding voltage standard deviations are 0.35V, 0.06V, 0.06V, 0.06V, 0.06V, 0.06V, 0.08V, 0.13V, respectively. For the noise factors, ART N = 1.81 × 10−4, aIT = 0.62, γy = 0.04, γxy = 0.003, At = 5.3 × 10−5, Bt = 2.9 × 10−4, aOT = 0.3, K1 = 1, K2 = 0.3.
In flash memory, the process of soft decoding typically involves step-by-step soft decision. The higher the decision level is, the more accurate the LLR provided by the channel should be. However, this comes at the cost of increasing read delay [13]. During the early and middle stages of flash life, a high soft decision level is unnecessary due to low RBER. To simplify the channel model, level 1 soft decision analysis is employed.

V. CONCLUSION
Compared with conventional LDPC codes, the GC-LDPC codes can significantly improve the flexibility of decoding and adapt to dynamically changing flash channels. Motivated by this advantage, this paper proposed a 4KB GC-LDPC code by traditional method and a (37536,33672) SC-GC-LDPC code by exploiting the spatially coupled structure for the construction of the global matrix. We conducted decoding experiments in AWGN channel and flash memory channel to verify the constructed code. It is shown that the SC-GC-LDPC code can achieve excellent decoding performance under the layered decoding algorithms. In the AWGN channel, compared with the proposed 4KB GC-LDPC code in this paper, the SC-GCLDPC code can achieve 0.065 dB and 0.08 dB performance gain by exploiting SPA and NMS, respectively. Finally, in the flash channel, the SC-GC-LDPC code also shows better error correction performance than the conventional GC-LDPC codes with 25% improvement. Based on these results, it is verified that the (37536,33672) SC-GC-LDPC code is an attractive option for NAND flash memories by providing the stronger ability to deal with channel deterioration in NAND flash memory.
