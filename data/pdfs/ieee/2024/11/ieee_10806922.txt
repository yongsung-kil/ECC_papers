I. INTRODUCTION
Low-density parity-check (LDPC) codes are capacityapproaching codes that are extensively utilized in various systems, such as Wi-Fi, optical microwave, and storage, and have been adopted as the coding method for 5G eMBB data channels [1], [2]. Quasi-cyclic LDPC (QC-LDPC) codes are particularly noteworthy due to their exceptional error-correction capabilities and the efficiency of their hardware implementation [3], [4]. Nonetheless, LDPC codes under iterative decoding are susceptible to a problematic phenomenon known as the error floor, which is evidenced by the diminishing rate of error reduction as the channel quality improves.
The error floor behavior of LDPC codes is principally attributed to certain graphical structures within the code’s Tanner graph, which are identified as trapping sets [5]. An (a, b) trapping set is defined as an induced subgraph containing a variable nodes and b check nodes with odd degrees, along

with any number of check nodes with even degrees, within the

Tanner graph. An elementary trapping set (ETS) is a specific

type of trapping set in which all check nodes have degrees of

either 1 or 2; these are considered the most harmful among the

trapping sets [6], [7].

Nevertheless, enumerating the non-isomorphic structures of

ETSs with varying values of a and b is challenging. Indeed,

McGregor et al. [8] have proven that determining the minimum

size of trapping sets within a Tanner graph of an LDPC code is

an NP-hard problem. For the sake of simplicity, our focus will

be on ETSs of variable-regular Tanner graphs in this paper.

Then we can use a specialized graph known as the variable

node (VN) graph [9] to simplify an ETS. Furthermore, in the

case of variable-regular Tanner graphs, there exists a one-to-one

correspondence between an ETS and its VN graph.

Owing to the diversity of ETSs, many researchers have

endeavored to impose constraints on Tanner graphs with the

intention of eliminating small ETSs. Tao et al. [10] constructed

QC-LDPC codes with the variable-regular degree dL(v) = 3 and the girth g = 8. These codes are free of all (a, b)-

ETSs for a ≤ 8 and b ≤ 3, achieved through the purposeful

elimination of certain 8-cycles. In [11], [12], Banihashemi et

al. substantiated through computer programming that any ETS

short cycles or non-cycle graphs with basic structures being

theta graphs and dumbbell graphs. By avoiding the 8-cycle

girth g = 6. These codes are free of all (a, b)-ETSs where

II. DEFINITIONS AND PRELIMINARIES
A graph G = (V, E) consists of a non-empty vertex set V and an edge set E as unordered pairs of vertices. Two vertices u and v are adjacent, denoted as uv ∈ E if they are connected by an edge. Denote the neighborhood of a vertex v as N (v) = {u ∈ V |uv ∈ E}. Given a subset S ⊆ V , the neighborhoods of S are denoted by N (S). The induced subgraph generated by S is defined as G[S] = (S ∪ N (S), E′), where E′ = {vc ∈ E(G)|v ∈ S, c ∈ N (S)}. A graph without loops or multiple edges is simple. A simple graph is complete, denoted Kn, if there is an edge connecting every pair of its n vertices. Additionally, a simple graph is bipartite if its vertices can be partitioned into two sets V1, V2 with no edge within the same set. A bipartite graph is complete, denoted by Km,n, if every vertex in V1 is connected to every vertex in V2, with m and n vertices in each part respectively.
A path of length k, denoted by Pk, is a sequence of vertices v0v1v2 . . . vk, where v0, v1, . . . , vk are all distinct and each pair of consecutive nodes (vi, vi+1) forms an edge in G, i.e., vivi+1 ∈ E for all 0 ≤ i ≤ k−1. We say that two or more paths are internally disjoint if they have no common vertices except

the endpoints. The distance d(u, v) is the length of the shortest path between u and v. The diameter of the graph G is defined as diam(G) = max{d(u, v)|u, v ∈ V (G)}. A cycle of length k, denoted by Ck, is a path that starts and ends at the same vertex v0 = vk with all other vertices in the sequence being distinct. Then, we introduce the following definition about theta graph, which is the main object we consider in this paper.
Definition 1 ([14]): A theta graph, denoted by θ(a, b, c), is formed by three internally disjoint paths with the same pair of endpoints, of length a, b, c respectively, where a ≤ b ≤ c.
Note that if θ(a, b, c) is a simple graph, then b ≥ 2 as there are no multiple edges.
A Tanner graph G = (L ∪ R, E) is a bipartite graph derived from the parity-check matrix H of an LDPC code—–introduced in [15]. In this graph, the set L represents variable nodes, each corresponding to a column in H, and the set R represents check nodes, each corresponding to a row in H. An edge adjoins the i-th check node to the j-th variable node if the entry Hij in the matrix is 1. The degree of a vertex v ∈ G, denoted as dG(v), is the total number of its neighbors. Specifically, the graph is variable-regular if every variable node v ∈ L has the same degree dL(v) = γ. The girth g of a Tanner graph is the length of the shortest cycle within the graph and is a critical factor that impacts the performance of LDPC codes.

For a given ETS, a variable node (VN) graph GV N = (VV N , EV N ) is constructed by first removing all check nodes of degree 1. The vertices, VV N , of this graph are defined as the variable nodes in the ETS. The edges, EV N , correspond to the pairs of variable nodes that are connected through check

nodes of degree 2 in the ETS [9].

In a variable-regular Tanner graph where each variable node

v ∈ L has the same degree dL(v) = γ, there exists a one-to-one correspondence between an (a, b)-ETS and its corresponding

VN graph. The VN graph is derived from the (a, b)-ETS by

definition; conversely, one can reconstruct the (a, b)-ETS from

a given VN graph by placing a check node on each edge to

represent the check nodes of degree 2. Additionally, for every

variable node u ∈ VV N where the degree dGV N (u) is less than γ, we can attach γ −dGV N (u) check nodes of degree 1, thereby obtaining the (a, b)-ETS.
