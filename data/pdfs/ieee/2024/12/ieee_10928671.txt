Abstract—Resistive Random Access Memory (ReRAM), as a non-volatile storage technology, has shown great potential in the storage field due to its high density and fast storage capabilities. However, its crossbar array structure gives rise to the issue of Sneak Path Interference (SPI) that significantly impacts data reliability. To address this challenge, this paper refines the general model of multiple sneak paths based on the number and combination types of these paths. Furthermore, an across-array coding scheme under multiple sneak paths is proposed. This scheme allocates low-density parity-check (LDPC) codewords to multiple memory arrays, reducing the correlation of channelcoded bits used for decoding, thereby achieving a diversity gain. Simulation results show that, under the multiple sneak path model, the cross-array coding scheme with LDPC coding can effectively improve the system's error rate performance.
Keywords- ReRAM; multiple sneak paths; across-array coding
I. INTRODUCTION
Resistive Random Access Memory (ReRAM) is a novel type of non-volatile memory that stores information by changing resistance. The principle is that when current passes through the memory cell, the resistance of the material changes, typically by applying voltage to switch the material between a high-resistance state or a low-resistance state, thereby representing binary data. Due to its memristors being located at each row-column intersection of a crossbar array, ReRAM achieves high storage density. It combines the advantages of traditional random-access memory and flash memory, featuring fast read/write speeds and low power consumption [1]. These advantages make ReRAM a crucial research direction for emerging memory technologies [2].
However, the crossbar array structure of ReRAM is prone to causing sneak path(SP) issues, which refer to unintended current paths parallel to the target cell being read [3]. The current flowing through these SPs can lower the resistance of the measured cell, significantly affecting data retrieval. To mitigate this issue, selectors are typically serially connected to each array cell. However, due to manufacturing defects, the selector may fail, leading to the recurrence of SP problems.

This study investigates the effects of SPs following selector failures.
Most existing research typically relies on a single SP model, treating it as interference and correcting the errors it induces using error correction codes (ECC) [4-8]. However, in practical applications, there may be multiple selector failures within a memory array that can result in the occurrence of two or more SPs at the target cell. A single SP model is inadequate to accurately describe the complexity of parallel resistance paths, and thus, a more precise multi-path model is required to analyze the effect of SPs on memory performance.
This paper uses LDPC codes to correct errors induced by the general model of multiple SPs. Additionally, a cross-array coding scheme is combined, where codewords are allocated to multiple independent memory arrays. This approach reduces the correlation of sneak path interference (SPI) within the codeword, thereby achieving a diversity gain during decoding, which effectively improves the BER.
The structure of this paper is as follows. In Section II, the multiple SPs in the ReRAM channel and the cross-array coding scheme are introduced. Section III presents the specific coding scheme and the related simulation results. Section IV concludes the paper.

The SPI is assessed by computing the parallel resistance of the readback cell. In ReRAM's crossbar structure, the parallel resistance is influenced not only by the number of SPs but also by the specific combination of these paths. When multiple selectors in the array fail, it may lead to multiple parallel SPs simultaneously existing at the target cell. These additional paths lower the overall parallel resistance, so, compared to a single SP, multiple SPs have a more significant impact on the target cell. This change not only affects the current distribution but may also lead to instability in the performance of the ReRAM array, impacting the reliability of storage operations. For example, in Figure 1, at the target cell (3,1), if the selectors at location (2,2) and (2,4) both fail, there will be two parallel SPs: (3,1)→(3,2)→(2,2)→(2,1)→(3,1) (red line) and (3,1)→(3,4)→(2,4)→(2,1)→(3,1)(blue line). Therefore, the type of SPs at the cell (3,1) would be l = (2;1, 2) , corresponding to the two parallel paths. B. Across-Array Coding Scheme
The across-array coding scheme refers to encoding a data vector u = (u1,¼,uK ) of length K into a codeword x = (x1,...xN ) of length N . The codeword x is then divided into T equal-length sub-vectors, with each sub-vector being assigned to a separate memory array, i.e., xt = (x1t ,¼, xNt /T ) ,where the j  th bit xit of the i  th memory array satisfies xit = x(t-1)N /T +i , i = 1,¼, N / T , t = 1,¼,T .Since the size of each memory array is m´n , each memory array can store up to mnT / N codewords. In this scheme, the interference from SPs is treated as i.i.d. noise, and the correlation between the channel input and SPs is ignored. As a result, each codeword x = (x1,¼, xT ) is independently
decoded based on its readback signal y = ( y1,¼, yT ) , which
yields the estimated data uˆ , where yt is the readback signal
from the t  th memory array, and xt is the original data vector, and t = 1,¼,T [6].

As the codeword is allocated to T independent memory arrays, the probability of SP occurrence is averaged across these arrays. According to the law of large numbers, as T  ¥ , the probability of SP occurrence converges to its average value with probability 1, thus reducing the instability of the channel. Furthermore, the readback signals of the encoded bits from different arrays are independent of each other, effectively reducing the correlation of SPI within a codeword [6].

III. ACROSS-ARRAY CODING SCHEME FOR MULTIPLE SNEAK PATHS
This section aims to reduce the instability of the ReRAM channel and decrease the correlation of multiple SPI within a codeword by applying the across-array coding scheme, based on the model of multiple SPs, thereby improving the BER performance.

The codeword x is divided into T equal-length sub-vectors with each sub-vector’s encoded bits sequentially assigned to T independent memory arrays. Each array passes through an independent ReRAM channel, which helps reduce the correlation between ReRAM channels, thereby improving decoding performance. To accurately measure the resistance of the target cell, this study considers the scenario where multiple selectors fail in the array, i.e., the general case where multiple SPs may occur at the target cell. During the detection process, we read the readback signal y after passing through the ReRAM multiple SP model channel and, by considering the actual probability of SP occurrence in each array, use it as an input to the elementary signal estimator (ESE). This allows us to estimate the loglikelihood ratio (LLR) of the xi, j . In the decoding process, the LDPC decoder uses the soft information provided by the ESE as input to perform soft decision decoding, ultimately obtaining the estimated value of the data uˆ . The proposed encoding scheme is also adaptable to other types of error correction codes.

D. Simulation Result
This paper uses LDPC codes under the Consultative Committee for Space Data Systems(CCSDS) standard to correct errors caused by the occurrence of multiple SPs, with N = 5120 , K = 2048 , R = 1 / 2 ,and the redundancy generation parameters of the parity-check matrix is M = 1024 . Soft-decision BP decoding is used, with a maximum of 20 decoding iterations. The memory array size is m´n = 72´72 , with reference to the data in[6], R(0) = 1000W , R(1) = 100W , q = 0.5 , p f = 0.0005 .

Figure 4. Performance comparison of across-array coding under the multiple SP model
As shown in the Figure 4, as the number of arrays T increases, the BER gradually improves, with the improvement being more significant in low noise conditions. This improvement is due to the diversity gain from distributing the codewords across several memory arrays. Additionally, the correlation of SPI within a codeword decreases, and the

channel's variability is reduced, thereby enhancing the overall system's interference resistance and lowering the BER. In summary, the cross-array coding scheme for multiple SPs proposed in this paper achieves significant performance improvement with only a slight increase in decoding time, effectively enhancing system performance. Furthermore, the scheme is also effective in addressing performance fluctuations under different array structures and noise conditions, demonstrating strong robustness.

IV. CONCLUSIONS This paper investigates the across-array coding scheme under the multiple SP model. By introducing a more precise multiple SP model, the study deeply analyzes the impact of SPs on memory performance and combines this with a across-array coding scheme to effectively decrease channel instability. Building on this, the ESE detector is used to compute the LLR for each encoded bit under different SP types for decoding. Simulation results show that after adopting the across-array coding scheme, as the number of arrays T increases, a slight extension of the decoding time can significantly improve the BER, thereby substantially enhancing system performance.
