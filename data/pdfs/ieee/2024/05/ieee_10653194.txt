AbstractвҖ”LDPC decoding algorithm based on the alternating direction method of multipliers (ADMM) technique has increased much attentions because of its outstanding decoding performance. To enhancing performance advantages of ADMM decoding in practical application, a main challenge is to design an efficient penalty function and then customize an accelerating decoding algorithm. To address this issue, in this paper we propose an improved penalized decoding problem based on a segmented penalty function (SPF), and then develop an efficient ADMM algorithm via optimizing variable updating to solve this SPFbased decoding model. Experimental simulations demonstrate that the proposed algorithm effectively reduces the number of iterations while achieving better error-correction performance, when compared to the state-of-the-art ADMM-based decoding algorithms.
Keywords- low-density parity-check codes; alternating direction method of multipliers; penalty function; variable node updating
I. INTRODUCTION
Low-Density Parity-Check (LDPC) codes, as a channel coding method approximating the Shannon limit, have been widely used in various fields such as 5G and satellite communications. The design of efficient decoding algorithms plays a crucial role in improving the error correction performance of LDPC codes.
Recently, linear programming (LP) decoding algorithm based on alternating direction method of multipliers (named ADMM-LP for short) has emerged and owned significant potential in reducing computational complexity and improving error correction performance [1] [2]. However, they suffer from a difficulty of inferior decoding performance at low signal-tonoise (SNR) ratios [3].

To improve error-correction performance of ADMM-LP decoding at low SNR regions, Liu et al. [4] [5] proposed an ADMM-based penalized decoding algorithm by adding a penalty function to LP decoding mode. The inclusion of penalty terms makes the decoding results deviate further from fractional codewords while getting closer to integer codewords.
Despite the advancements in decoding algorithms for LDPC codes, particularly in low signal-to-noise ratio conditions, there remains a challenge in achieving an optimal balance between decoding efficiency and error correction performance. Addressing this challenge is crucial for the continued improvement of LDPC-based communication systems.
Reference [6] points out that the design of segmented penalty functions results in a minimal impact of the penalty term on the variable node values when they are close to the decision threshold of 0.5. This enables the decision of variable nodes to rely primarily on the information transmitted by the channel and the associated check nodes, thus enhancing decision accuracy. Inspired by this, in this paper we propose an improved penalized decoding problem based on a segmented penalty function (SPF). Moreover, an efficient ADMM-based solving algorithm to address SPF-based decoding problem is suggested by optimizing variable updating for further improving decoding efficiency.

A. Model Construction Designing an excellent penalty function for the LDPC code
decoding model based on the ADMM algorithm is essential to enhance error correction performance. Besides, it's crucial to understand that different penalty functions will have varying effects on the same model and lead to differences in error correction performance. Therefore, developing a suitable penalty function is necessary for the ADMM-based LDPC decoding to improve its error correction performance.
The l1 norm and l2 norm are two commonly used forms of penalty functions. By analyzing the impact of the slope characteristics of these two penalty functions on decoding performance [6], the following conclusion can be drawn: when the variable node xi is near 0.5, the slope of the penalty function should be chosen to be small; whereas when the variable node xi is far from 0.5, the slope of the penalty function should be chosen to be large for achieving stronger error correction performance. C. Algorithm Analysis By observing equations (12) and (13), it can be analyzed
that the ADMM-IPF algorithm proposed in this paper does not increase the complexity order in comparison with the ADMMPD decoding algorithm [5]. Therefore, the computational complexity of the proposed model grows linearly with the LDPC code length n.
Additionally, because the penalty function proposed in this paper satisfies the following two properties: (1) the penalty function is increasing over the interval (0,0.5); (2) for x пғҺ [0,1], the penalty function is symmetric about 0.5. Therefore, the decoding algorithm proposed in this paper satisfies the ML assumption and the all-zero assumption [5].

The compared algorithms include the ADMM-PD-l1 algorithm [5], ADMMPD-l2 algorithm [5], and ADMM-Lpbox algorithm [11]. V. CONCLUSIONS
Designing appropriate penalty functions to optimize the solution space is a common method for enhancing the error correction performance of ADMM decoding algorithms. However, the addition of penalty functions often results in a certain loss of decoding complexity. Therefore, in this paper we propose an improved ADMM-IPF decoding algorithm by optimizing the ADMM update iterations to improve the decoding efficiency of ADMM decoding. In comparison to the state-of-the-art ADMM-based decoders, the approach introduced in this paper requires fewer number of iterations, while achieving superior decoding performance. Finally, the effectiveness of the proposed ADMM-IPF decoding algorithm is validated through experimental simulations.
