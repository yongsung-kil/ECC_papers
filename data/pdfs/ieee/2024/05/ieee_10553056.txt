Disjoint Difference Sets and QC-LDPC Codes With Girth 10
Farzane Amirzade, Mohammad-Reza Sadeghi and Daniel Panario, Senior Member, IEEE

2024 12th Iran Workshop on Communication and Information Theory (IWCIT) | 979-8-3503-8709-4/24/$31.00 ©2024 IEEE | DOI: 10.1109/IWCIT62550.2024.10553056

Abstract—A method to obtain an exponent matrix of a QCLDPC code with column weight 4 appeared recently in which the entries of the matrix belong to the sets of a (v, k, 2) disjoint difference set (DDS). It was also shown that the girth of the Tanner graph in these codes is at least 8. In this paper, we prove the existence of an 8-cycle in their Tanner graph which shows that the girth of these codes is exactly 8.
The merits of a QC-LDPC code from a DDS motivated us to study these combinatorial designs. However, the disjoint difference sets existing in the literature are limited to small values of the parameters k and t. We present a novel approach to obtain a (v, k, t)-DDS with 3 ≤ t ≤ 5 and different values of k. Our method is based on a new relation between QC-LDPC codes with girth 10 and disjoint difference sets.
Index Terms—Disjoint difference set (DDS), QC-LDPC codes, Tanner graph, girth.
I. INTRODUCTION
Disjoint difference sets (DDS) and disjoint triangle sets are combinatorial designs that are used to construct codes such as quasi-cyclic and convolutional codes [1, 2]. Such application of these mathematical concepts in coding theory motivates many researchers to study the properties of these designs [3] and to find their relation with other combinatorial concepts such as difference families, Golomb rulers as well as Sidon sequences [4].
Let 0 < k < v and G be a group of order v. A collection {B1, B2, . . . , Bt} of k-subsets of G, where Bℓ = {aℓ1, . . . , aℓk} for 1 ≤ ℓ ≤ t, forms a (v, k, t)-DDS if all the differences {aℓi − aℓj : 1 ≤ ℓ ≤ t, 1 ≤ i ̸= j ≤ k} modulo v form a set of tk(k − 1) distinct elements.
A low-density parity-check (LDPC) code is a binary linear code whose parity-check matrix is sparse. There are different methods to obtain an LDPC code. Quasi-cyclic (QC) LDPC codes form an important class of LDPC codes. They have simple implementation. These codes associate a small m × n matrix, the exponent matrix, whose elements are from {0, 1, . . . , N − 1} to an (m, n)-regular QC-LDPC code. The parameter N is called lifting degree of the QC-LDPC code. Efficient schemes to identify the entries of an exponent matrix are given in [5–7].
Recently, the construction of QC-LDPC codes based on a DDS has been given [8, 9]. In [8], a method to obtain an exponent matrix of a QC-LDPC code with girth at least 8 is provided; the entries of the exponent matrix are the elements of a (v, k, 2)-DDS. An advantage of the resulting codes over other QC-LDPC codes in the literature is that experimentally they have better performance curves [8]. Then, in [9], a new method to construct a (v, k, 2)-DDS is proposed.
In this paper, we first prove that the exact girth of the Tanner graph of the QC-LDPC codes in [8] is 8, and then we prove that these codes outperform other ones in the literature. We present a general method to obtain a (v, k, t)-DDS with t ≥ 3. Our method is based on a new relation between an (m, n)regular QC-LDPC code with lifting degree N and girth 10 and an (N, n, m − 1)-DDS.

The rest of the paper is organized as follows. Section II presents some notations and definitions. In Section III, we analyze the QC-LDPC codes proposed in [8] in terms of their girth as well as their performances. Section IV proposes the relation between a girth-10 QC-LDPC code and a DDS. We summarize our results in Section V.

II. PRELIMINARIES
Let B = [bij] be an m×n exponent matrix of a quasi-cyclic LDPC (QC-LDPC) code with lifting degree N , where bij ∈ {0, 1, . . . , N −1} or bij = (∞). If integer numbers, bijs, in the exponent matrix are replaced by N × N circulant permutation matrices (CPMs), whose first row has 1 in the bij-th column and zero in other columns, and (∞) is replaced by an N × N zero matrix (ZM), then we obtain a parity-check matrix. The null space of this parity-check matrix forms a QC-LDPC code [5]. The number of 1s in a column of the parity-check matrix is its column weight and the number of 1s in a row of the paritycheck matrix is its row weight. If the parity-check matrix has a constant column weight m and a constant row weight n, then it is an (m, n)-regular QC-LDPC code. If all elements of the exponent matrix are integers, then the obtained parity-check matrix results in a fully connected QC-LDPC code. Since all QC-LDPC codes in this paper are fully connected, we prefer to use “QC-LDPC code” instead of “fully connected QC-LDPC code”.

There are several numerical and analytical results [4] regarding the existence of a (v, k, λ) difference family for different parameters of v, k, and λ. In general, the results for (v, k, 1) difference families focus on 3 ≤ k ≤ 9. Using the existing difference families we cannot obtain a DDS for large k. Moreover, when constructing a (v, k, t)-DDS with the smallest scope v, using a (v, k, 1) difference family is not a good choice if t is much smaller than b. Indeed, in a difference family we have v ≥ bk(k − 1) + 1 but for a DDS the lower bound on the scope is tk(k − 1) + 1, where t ≪ b. For example, there exists a (449, 8, 1) difference family [4] for which the number of sets is b = 8. Thus, the first three sets yield a (449, 8, 3)-
DDS, while using our method, which we explain in Section IV, we can obtain a (181, 8, 3)-DDS.
Another combinatorial design that can be used to obtain a DDS is modular Golomb ruler that we present next.

IV. CONSTRUCTION OF A DDS FROM A GIRTH-10 QC-LDPC CODE
As explained in Section II, constructing a (v, k, t)-DDS with small 3 ≤ t ≤ 5 and smallest scope v is not possible if we use difference families, MGR as well as Theorems 1 and 2. In fact, as mentioned in Section II, if we use a modular Golomb ruler (Theorem 1 and Table 1), then we have few possibilities for a (v, k, t)-DDS with a particular t, and if we use difference families, then the existing numerical results are only for 3 ≤ k ≤ 9. Thus, these two methods cannot yield a (v, k, t)-DDS for parameters such as k = 14 and t = 4. In this section, to tackle this problem, we present a novel method which is based

Remark 4: We assume an (N, n, 3)-DDS with three sets R1,2, R1,3, R2,3 is obtained from Theorem 7. Our goal is to obtain an (N ′, n, 3)-DDS with N ′ < N , which is a

In order to simplify this process we can only compute three sets Q1,2, Q1,3, Q2,3 modulo N ′, where 3n(n − 1) + 1 ≤ N ′ ≤ N . If for an N ′ < N , the set Q1,2 ∪ Q1,3 ∪ Q2,3 is of size 3n(n − 1) and free of repeated elements, then we have an (N ′, n, 3)-DDS with three sets R1,2, R1,3, R2,3 modulo N ′.

V. CONCLUSION
In this paper, we present the exact girth of a (4, n)-regular QC-LDPC code obtained from a (v, k, 2) disjoint difference set (DDS) in [8]. We prove that the girth of these codes is 8. Then, we provide an analytical discussion regarding the merit of these codes in terms of their simulation results. We show

We also introduce a relation between a QC-LDPC code with girth 10 and a DDS. Then, we show how to construct a DDS from a girth-10 QC-LDPC.
