Generalization of Floor Lifting for QC-LDPC Codes

Abstract
The given paper presents a method for constructing a QC-LDPC code of shorter length by length adaption from a given QC-LDPC code of maximal length. The proposed method can be considered as a generalization of floor lifting. Making some offline calculation it is possible to construct a sequence of QC-LDPC codes with different circulant sizes generated from a single exponent matrix of QC-LDPC code which has the largest length. The only parameter stored in memory is a constant needed for scaling in the lifting procedure. Simulation results show that the proposed lifting method achieves around a 0.1dB coding gain on short lengths over previous lifting approaches.
1. Introduction
Low-density parity-check (LDPC) codes were first discovered by Gallager [1], rediscovered by MacKay [2] and Sipser et al. [3].
Quasi-cyclic Low-Density Parity-Check (QCLDPC) codes are of great interest to researchers since they can be encoded and decoded with low complexity suboptimal iterative decoding algorithms and allow reaching high throughput using linear-feedback shift register, [4-6]. Suboptimal iterative decoding algorithms converge to the ML decoding if states of all check nodes connected to any arbitrary variable node in Tanner graph of LDPC-codes are independent.
One advantage of QC-LDPC codes based on circulant permutation matrices (CPM) is that it is easier to analyze their code and graph properties than in the case of random LDPC codes. The performance of LDPC codes under suboptimal iterative decoding algorithms is strongly affected by their code and graph properties, such as the length of the shortest cycle, i.e., girth [7], trapping sets formed by overlap of cycles and cycles itself [8-10], code distance [11-12] and the

ensemble weight enumerator [13]. The trapping sets formed by overlap of cycles prevent suboptimal iterative decoding algorithms to converge the ML. Instead of convergence to codeword from ensemble weight enumerator, decoder converges to pseudocodeword formed by the trapping set.
A large number of code construction methods have been proposed to eliminate harmful Trapping sets formed by short cycles: Guess-and-Test, PEG, Hillclimbing [7, 9, 14]. Based on these methods using the lifting equation a QC-LDPC code can be constructed in such way to have length adaption. Making some offline algorithm it is possible to construct a sequence of QCLDPC codes with different circulant sizes generated from a single exponent matrix of QC-LDPC code having the largest length. The main contribution of the paper is an approach for constructing a quasi-cyclic LDPC code of shorter length by lifting the largest length QC-LDPC code. This method allows eliminate harmful short cycles. The proposed method can be considered as a generalization of floor lifting method introduced in [15].

The trapping set (a,b) is a subgraph formed by cycles in Tanner graph or their overlap with a variable nodes and b odd degree checks. For example, Figure 2 shows Trapping sets TS(5,3) formed by overlap of three 8-cycles, and TS(4,4) formed by cycle 8 in Tanner graph, [9].
The harmfulness of a trapping set is defined as the number of variable nodes, error in which can cause suboptimal iterative decoder to fail on this TS.
Under equal probability of error, TS(5,3) more harmful, because 3 errors in odd degree check nodes provide 5 errors in variable nodes. The TS(4,4) produces 4 errors after 4 errors in odd degree check nodes, and can be consider as a pseudocodeword with weight 4. Harmfulness is strongly related to decoder parameters. Their modification can change probability of decoder fail due to TS pseudocodewords.

The metric Extrinsic Message Degree (EMD) of a cycle in the Tanner graph is defined as the number of check nodes singly connected to the variable nodes involved in the cycle. The EMD value of a code is an important characteristic, because each cycle is a trapping set. The EMD metric estimates how strongly subgraph of cycle is connected with the rest of the Tanner graph. Calculation of the Tanner graph EMD is a hard task, because it requires to determine if the edge is extrinsic edge or cut edge. Instead of EMD, Approximated Cycle EMD can be used. 4. Code sieving method for design length adaptive QC-LDPC codes
The design of length adaptive QC-LDPC code with code lengths Nk = n × Lk ,1 ≤ Lk ≤ L0 and rate R = 1− n − m , where p -number of punctured variable
n− p nodes (columns) consists of two stages: mother code (protograph) choice M (H ) , lifting for every of require circulant sizes {Lk } .

Choice of mother code matrix is based on minimization of threshold using Density Evolution method or it approximations: Gaussian approximations (GA, [17]), Reciprocal Channel Approximation (RCA, [18]), Protograph Exit-chart (PEXIT-chart, [19]).
For short length addition assumption is usually used based on statistical physics optimization which controls of parity-check dense like Finite-length scale (Covariance evolutional, [20]).
The proposed mother matrix search method uses PEXIT-chart and it covariance evolutional modification for short length, [19, 21].
For construction of length adaptive QC-LDPC code for each supported code length Nk = n × Lk sieving code candidates method is used:
1. A set of lifted QC-LDPC codes for current circulant value Lcurrent with different scale value r is generated using simulated anneling method. The goal of simulated anneling method is minimal number of cycles for which ACE equals EMD. A comparison of simulated anneling method with Hill Climbing method, [14] is shown on Table 2.
2. If require level of frame error rate is lower than 10−4 following are additional performed. Pseudocodewords formed by TS are weighted using suboptimal decoder, [8]. The QC-LDPC codes are reordered according to code distance upper bound estimated using Smarandache-Vontobel theorem modified by Butler-Siegel method [11, 12]. Example of code distance upper-bound distribution under 50 code candidates is represented on histogram, Figure 3.
3. Subset of QC-LDPC code candidates is simulating according to the order. The best one QCLDPC code is chosen.
4. Repeat all steps for every supported circulants Nk = n × Lk .
5. Simulation Results
Below on figure 4 simulation result is represented for SNR required to reach 10−2 block error rate for different information length of QC-LDPC code with rate 1/2 under AWGN-channel, QPSK modulation using 15 iteration of NOMS decoder. Code A is from 5G proposal [22], code B is from 5G proposal [23] and code C is the QC-LDPC code, defined by mother matrix Table 3, constructed using the code sieving method. Code C have 0.1 dB coding gain on information length less than 2000 and near same performance till 8000.
For family C the floor modular scale lifting was applied. Layered normalized offset min-sum decoder

with 15 iterations was used in simulations. Normalize and offset factors were optimized to improve waterfall performance [24]. In summary, the proposed lifting scheme supports fine granularity and avoids catastrophic cases of performance degradation causes by code distance properties [10, 11, 25] and graph properties [8, 26] for different lengths.

5. Conclusion

The given paper presents a method for constructing a QC-LDPC code of shorter length by length adaption from a given QC-LDPC code of maximal length. The proposed method can be considered as a generalization of the floor lifting. Making some offline calculation it is possible to construct a sequence of QC-LDPC codes with different circulant sizes generated from a single exponent matrix of QC-LDPC code which has the largest length. The only parameter stored in memory is a constant needed for scaling in the lifting procedure. The proposed lifting method allows to make improvement with respect to the number of the shortest cycles. It was determined in such way to increase cardinality of posible CPM value choices and shows improved performance over the previous approaches.
