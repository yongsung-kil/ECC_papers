The Design and Analysis of Protograph LDPC Codes over Rayleigh Fading Channels
Zhaojie Yang, Yi Fang, Guofa Cai, Zibing Yao, and Guojun Han School of Information Engineering, Guangdong University of Technology, Guangzhou, China, 510006 Email: zjyang0131@163.com; {fangyi, gjhan}@gdut.edu.cn; caiguofa2006@126.com; 1798653396@qq.com

Abstract—In wireless communication systems, the error performance in the high signal-to-noise-ratio (SNR) region determines the reliability and quality of data transmission. Among existing error-control techniques, channel coding is of particular interest to enhance the overall performance of such system. In this paper, we investigate the error performance of the protograph low-density parity-check (LDPC) codes over Rayleigh fading channels. To obtain robust error-correction capability, we construct two improved ARA (i.e., IARA1 and IARA2) codes based on the extrinsic information transfer (EXIT) algorithm and the asymptotic weight distribution (AWD) function. We further analyze the asymptotic performance of the IARA1 code and the IARA2 code from the perspective of decoding threshold, typical minimum distance ratio (TMDR), and bit-error rate. Both the analytical and simulated results show that the IARA1 code and IARA2 code outperform the well-performing accumulate-repeatby-4-jagged-accumulate (AR4JA) code in the high SNR region. As a consequence, the IARA1 code and IARA2 code appear to be better candidates for use in wireless communication applications with fast fading.
Index Terms—protograph low-density parity-check (LDPC) code, extrinsic information transfer (EXIT) algorithm, asymptotic weight distribution (AWD) function, Rayleigh fading channels.
I. INTRODUCTION
Forward error correction (FEC) technique is a powerful solution which has been exploited to enhance the reliability and quality of data transmission. It is well known that lowdensity parity-check (LDPC) codes can achieve Shannonlimit-approaching performance over the additive white Gaussian noise (AWGN) channel [1]. During the past decade, a significant amount of attention has been given to the LDPC codes due to their superior error performance. In wireless communication environments, most applications require excellent performance down to a bit-error-rate (BER) of 10−6. Compared with other existing error-correcting codes, LDPC codes can achieve very desirable error performance, especially in the high signal-to-noise-ratio (SNR) region. For this reason, LDPC codes have been widely applied in wireless communication systems [2], [3]. In [4], LDPC codes have been researched in Rayleigh fading channels and some optimized LDPC codes have exhibited strong robustness against signal fading. Meanwhile, much effort has been given to the design and analysis of LDPC codes. For example, the density evolution [5], [6] and the extrinsic information transfer (EXIT) algorithm [7], [8]

have been exploited to analyze the characteristics of LDPC codes in order to construct the best candidate for digital communication systems.
LDPC codes can realize a near-capacity performance under the iterative decoding. However, the nonlinear encoding complexity and the high error floor become two major of such LDPC codes drawbacks that prevent them from being applied in practical communication systems. In order to solve this problem, a great deal of research work has been devoted to reducing the error floor of short-block-length LDPC codes which can attain excellent error performance over flat Rayleigh fading channels [9]. Structured LDPC codes, such as quasi-cyclic LDPC (QC-LDPC) codes [10], have also been investigated over such transmission scenarios. The encoding complexity of QC-LDPC codes is lower than the conventional LDPC codes. Moreover, as a novel class of LDPC codes, protograph LDPC codes have also received much attention since they can produce outstanding performance with low complexity [11], [12]. In [13], two types of protograph LDPC codes, namely the accumulate-repeat-by-3-accumulate (AR3A) code and the accumulate-repeat-by-4-jagged-accumulate (AR4JA) code, which possess simple protograph structures to realize linear encoding and decoding, have been constructed. With respect to the AR3A code, the AR4JA code is a preferable choice for wireless communications since it can achieve very desirable error performance in both low and high SNR regions [14]. The protograph EXIT (PEXIT) algorithm has been conceived in [15] to calculate the decoding thresholds of protograph LDPC codes over AWGN channels. Following this work, a modified version of the PEXIT algorithm has been developed in [16] to estimate the thresholds of protograph LDPC codes over Rayleigh fading channels. On the other hand, the asymptotic weight distribution (AWD) function, which can be utilized to estimate the typical minimum distance ratio (TMDR) and characterize the linear-minimum-distancegrowth property, has also been developed [17]. Recently, there is a large set of work in the field of protograph LDPC codes [12], [18]. To be specific, the rate-compatible protograph LDPC codes have been designed in [19], [20]. The BER expressions of protograph LDPC codes over Nakagami-m fading relay channels with different relay protocols have been derived in [21]. Besides, a design methodology has been conceived to construct protograph LDPC codes in order to accomplish fast decoding [22], [23]. Nevertheless, to the best of our knowledge, there is a lack of good protograph LDPC

In this paper, we investigate the performance of protograph LDPC codes in the high SNR region over the Rayleigh fading channel. Based on the PEXIT algorithm and the AWD function, we construct two improved ARA codes, namely the IARA1 code and IARA2 code, by exploiting a computersearch-based method. In particular, the IARA1 code and the IARA2 code have the same encoding complexity with respect to the AR4JA code due to the same number of edges in their protographs. Furthermore, we compare the decoding thresholds, TMDRs, and simulated BERs of the IARA1 code, the IARA2 code, and the AR4JA code. Theoretical and simulation results illustrate that the IARA1 code and IARA2 code are superior to the AR4JA code in the high SNR region. Accordingly, the IARA1 code and IARA2 code stand out as better alternatives in fast-fading transmission scenarios.
The remainder of this paper is organized as follows. In Section II, the protograph-coded system model over a Rayleigh fading channel is introduced. Section III describes the basic principles of protograph LDPC codes. In Section IV, the protograph design method for the IARA1 code and IARA2 code is described. Moreover, we investigate the decoding thresholds and TMDRs for the AR4JA code, the IARA1 code, and the IARA2 code. Section V provides several simulation results. Finally, Section VI concludes this paper.

II. SYSTEM MODEL
The system model being considered in this paper is described in Figure 1. Referring to this figure, the information bits (Info bits) are frist encoded by a protograph LDPC encoder. Meanwhile, some binary coded bits are punctured in order to increase the code rate. Then the binary coded bits vj ∈ {0, 1} (j = 1, 2, . . .) are passed to the binary-phase shift-keying (BPSK) modulator, whose output can be described as xj = (−1)vj ∈ {+1, −1}. The modulated signal will be transmitted through an uncorrelated flat Rayleigh fading channel which can be modeled by a fading channel with AWGN. III. PROTOGRAPH CODES

A protograph, which has been described in [11], is a Tanner graph consists of a relatively small number of nodes. A protograph G = (V, C, E) is composed of three different sets, i.e., a variable node (VN) set V, a check node (CN) set C, and an edge set E. The cardinalities of V, C, and E correspond to n VNs, m CNs, and E edges, respectively. Furthermore, each edge ei,j connects a VN vj and a CN ci. The protograph codes are different from conventional LDPC codes since parallel edges are allowed in the protograph. Therefore, the mapping ei,j → (vj, ci) may not be one-toone. Based on the aforementioned definition, the code rate of a protograph code becomes R = (n − m)/(n − np), where the np is the number of punctured VNs. We define an adjacency matrix of the protograph as a base matrix B = (bi,j), where the bi,j denotes the number of edges connecting ci with vj. A larger protograph, referred to as the derived graph, can be obtained by performing the “copy-and-permute” operation on a given protograph. The LDPC code corresponding to a derived graph is called as protograph LDPC code.
A simple example of generating the derived graph is presented in Figure 2. This protograph, which has 5 different edge types, consists of 3 VNs, 2 CNs, and 6 edges. The derived graph is constructed by copying the protograph 2 times and permuting the edges belonging to the same type. The relevant base matrix is given by

The decoding threshold of protograph codes, which can be accurately estimated by the PEXIT algorithm, is capable of indicating the error performance in the waterfall region over the AWGN channel. B. Improved Protograph LDPC Codes
In order to design good protograph codes, we concentrate on the search of protograph which not only has a desirable threshold but also can possess the linear-minimum-distancegrowth property. Based on the computer search, it can be easily obtained that the largest size of the search space can be described by (1 + emax)mn − 1, where emax denotes the maximum integer entry in the base matrix. Considering the exponential growth of the search space, we can conduct an initialized base matrix based on some constraints.
Firstly, these elements in the base matrix have been limited to no larger than 3 so as to reduce the search space and ensure the low-complexity property of protograph codes. More importantly, with an aim to keep the encoding and decoding complexities unchanged, we assume that the overall number of edges involved in the improved protograph codes is the same as that in the AR4JA code. In addition, because a proper proportion of degree-2 VNs can improve the error performance of protograph LDPC codes, we further reduce the search space by means of setting one degree-2 VN. A. Error Performance over an AWGN Channel
The BER performance curves of the AR4JA code, IARA1 code, and IARA2 code over an AWGN channel are shown in Figure 5. It can be seen that the IARA1 code is the best-performing code while the AR4JA code is the worstperforming code in such a scenario. Specifically, these three type of protograph codes perform similarly in the low SNR region (Eb/N0 ≤ 0.6 dB), and thus one can not identify the difference among them. In the medium SNR region, the IARA1 code is superior to the IARA2 code, which performs better than the AR4JA code. As the SNR further increases, the gap between IARA1 and IARA2 codes becomes negligible, and these two codes achieve a remarkable gain over the AR4JA code. It is because the error performance in the high SNR region is dominated by the TMDR value. For example, at Eb/N0 = 2 dB, the IARA1 and IARA2 code achieve a BER of 10−7, while the AR4JA code only accomplishes a BER of 5 × 10−7.

B. Error Performance over a Rayleigh Fading Channel
The BER performance curves of these three protograph codes over the Rayleigh fading channel are depicted in Figure 6. Referring to this figure, the AR4JA code is the worstperforming code compared with the IARA1 code and the IARA2 code. Furthermore, these BER curves corresponding to the IARA1 code and the IARA2 code are very close. For example, at Eb/N0 = 4.0 dB, the IARA1 code and the IARA2 code achieve a BERs of 6×10−6, 7×10−6, respectively, while the AR4JA code accomplishes a BER of 10−4. Moreover, at a BER of 2 × 10−6, both the IARA1 code and the IARA2 code have a significant performance gain of about 0.4 dB over the AR4JA code. In addition, we find that the error floor does not appear when the BER is less than 10−6. From this perspective, a large gain can be expected at a lower BER.
In summary, we conclude that the IARA1 code and IARA2 code can provide outstanding error performance in the high SNR region over both AWGN and Rayleigh fading channels.
VI. CONCLUSIONS
In this paper, we have studied the performance of the protograph LDPC codes over both AWGN and Rayleigh fading channels. We have constructed two improved ARA codes, called as the IARA1 code and IARA2 code based on a PEXIT-AWD-based design method. Analytical and simulated results have suggested that the IARA1 code and IARA2 code can achieve a remarkable performance gain over the AR4JA code over Rayleigh fading channels. In the future, we plan to develop the design approach of quasi-cyclic protograph codes in order to further reduce the implementation complexity.
