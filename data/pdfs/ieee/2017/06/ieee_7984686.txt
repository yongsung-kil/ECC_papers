A Novel lRA-like Codes Construction Method Based on Cyclic Matrix Permutation

Wei Wang, Yingxiang Liu, Xiaomei Tang, Zhe Liu, Guangfu Sun
School of Electronic Science and Engineering National University of Defense Technology
Changsha 410073,China e-mail: 1993wongwei@outlook.com

Abstract-Due to the structure limitation of navigation message, the channel coding used by the satellite navigation system should have as high a coding gain as possible under short code lengths. At the same time, in order to reduce the cost of the receiver, the channel coding suitable for the navigation system should have the lowest complexity for coding and decoding. A well-designed structured LDPC code can reduce the complexity of the codec significantly without the loss of performance. In this paper, a method based on cyclic matrix permutation is proposed to improve IRA-like codes, which is combined Irregular Repeat Accumulation Codes (IRA) with Quasi-Cyclic LDPC codes. Simulation results show our method has good iterative decoding performance in decreasing bit error rate and greatly reducing the number of iterations required for decoding as weil as error platform.
Keywords-LDPC; lRA-like codes; channel coding; cyclic matrix permutation
I. INTRODUCTTON
In order to improve the reliability of the satellite navigation receiver message demodulation, the existing major satellite navigation systems are used forward error correction (FEC) channel coding. In the weak signal condition,the carrier-to-noise ratio of the received signal is low, and the performance on the conventional channel coding and demodulation is poor. With the development of high-gain channel coding techniques, such as Low Density Parity Check (LDPC) codes [1,2] and Turbo codes [3],it is possible to obtain a higher message demodulation sensitivity by increasing the coding gain. LDPC codes have higher coding gain under the condition of long code length, low coding efficiency,better error floor performance [4,5],and are not patented.
However, when LDPC codes are applied to navigation systems, they must meet the following requirements: the constraints on the frame structure of navigation message,and the length of the encoded information packet can not be too long. In order to reduce the cost of the receiver,the coding and decoding complexity should achieve to be as low as possible. Therefore, we propose a method which constructs IRA-like codes with higher gain and lower the complexity of coding and decoding to improve the robustness of the navigation signal with the above constraints.
In this paper, a feasible and simple matrix permutation construction method based on IRA-like codes, which is

constructed by the Irregular Repeat-Accumulate Codes [6] and QC-LDPC [7],is proposed.
This paper is organized as folIows. In Section 2, the structure of IRA-like codes is introduced. Detailed cycle analysis of the QC-part in H will be presented in Section 3. Section 4 shows a feasible method to avoid short cycles generated by circulant permutation matrix. Besides,we will discuss the construction steps of IRA-like codes in details. Section 5,the decoding performance and iterations of IRA­ like codes on the AWGN channel is validated. Finally,the main conclusion of the paper is drawn in section 6.

VI. CONCLUSION
In this paper, a feasible and simple matrix permutation method is proposed to eliminate low-weight code words. Beside,IRA-like codes have the advantages of linear coding complexity and low-usage of memory,which is significant to reduce the cost in satellite navigation receiver. The result is this paper can be applied to the existing GPS and BeiDou satellite navigation system and have a good performance.
