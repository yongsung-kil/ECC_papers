On the Design of Good LDPC Codes with Joint Genetic Algorithm and Linear Programming Optimization

Ahmadreza Amirzadeh, Mohamed Haj Taieb, and Jean-Yves Chouinard Departement of Electrical and Computer Engineering
Laval University, Quebec City, Quebec G1V 0A6, Canada Email: ahmadreza.amirzadeh.1@ulaval.ca, mohamed.haj-taieb.1@ulaval.ca, and jean-yves.chouinard@gel.ulaval.ca

Abstract—In communication systems, the transmitted data is corrupted by channel perturbations, such as noise and fading, which affect the reliability of the received data. Error correction codes are employed to mitigate channel perturbations. However, design and implementation of good and efficient error correction codes remains an open problem. In this paper, Low Density Parity Check (LDPC) codes are considered as they provide a reasonable trade-off between computational complexity and reliability. Good LDPC codes should ideally provide low complexity, close to capacity acheivable transmission rate, high coding threshold, and high decoding stability. In this paper, we investigate a joint LDPC code optimization algorithm using Genetic Algorithm (GA) and Linear Programming (LP) to determine the variable nodes and check nodes degrees distributions. EXIT chart analysis and Frame Error Rate (FER) performance are used to validate the proposed method.
I. INTRODUCTION
There are two families of channel coding methods which enable transmitted messages handle impairments caused by the channel impariments such as noise, fading and interference [1]. The first family of coding method is the Automatic Repeat Request (ARQ) codes [2] that automatically requests for the retransmission of received corrupted frames. The second one, Forward Error Correction (FEC) codes, encodes the messages with redundant but structured data such that the receiver can reconstruct the erroneous bits without the need for retransmission.
There are several channel coding approaches associated with FEC: convolutional codes, block codes (e.g. Hamming codes, Reed-Solomon codes), Turbo codes, and the Low Density Parity Check codes (LDPC) introduced by Gallager [3]. MacKay and Neal [4] showed that LDPC codes can achieve near Shannon limit performance (channel capacity). LDPC codes are among the most useful error correcting codes today as they have the dual advantages of relative simplicity and efficiency.
In this paper, we consider LDPC codes which are linear block codes using the sparse parity-check matrices. There are two different types of LDPC codes, namely regular and irregular LDPC codes. In the parity check matrix of a regular LDPC code, the column weight wc is constant for every column as well as the row weight wr for each row. Otherwise, if H does not have the same number of 1s for each row or for each column, it is called an irregular LDPC code. The generalization

of the Gallager’s regular LDPC codes to irregular LDPC codes have been investigated by several researchers [5–7]. Typically, irregular LDPC codes outperform the regular LDPC codes and can lead to the better performances with respect to the best turbo codes [8]. However, the main problem in designing the good irregular LDPC codes is to find suitable variable node degree distributions, λ(x), and check node degree distributions, ρ(x). This procedure is called code design [9]: ideally, one would like to optimize both the variable node and check node degree distributions jointly, but this leads to a complex optimization problem which is hard to implement [7, 10, 11].
In this paper, we investigate the optimization process of irregular LDPC codes over Binary Erasure Channels (BEC). Variable node degree distribution is optimized using a Genetic Algorithm (GA) and then the algorithm searches for the best check node degree distribution using Linear Programming (LP), until a good LDPC code with desirable characteristics over the BEC is found. These LDPC code characteristics include low complexity, high rate and threshold, low multiplicative gap between code design rate and capacity, and high decoding stability.
This paper is organized as follows. In section II, the desired characteristics of LDPC codes are described. The proposed joint optimization method for variable and check node degree distributions for irregular LDPC codes is presented in section III. Simulation results are provided in section IV and a discussion on the computational complexity follows in section V.

D. High threshold
Good LDPC codes should have the ability to correct errors for the maximum value of channel erasure probability ε in a BEC. In other words, one search for codes which provide reliable transmission over the worst BEC channels, i.e. maximum channel erasure probabilities. EXIT chart analysis is a tool that can be used for the calculation of the channel threshold using a Belief Propagation (BP) method. From equation (2), one can consider that f (ε, x) = ελ(1 − ρ(1 − x)) is the combination of two functions: vε(x) = ελ(x) which shows the action of the variable nodes, and c(x) = 1 − ρ(1 − x) which shows the action of the parity nodes, such that f (ε, x) = vε(c(x)). The LDPC decoding convergence condition is bounded as [14]:

c(x) < vε−1(x), x ∈ [0, 1]

(6)

The graphical interpretation of (6) is that c(x) must be strictly less than vε−1(x) for x ∈ [0, 1], and εBP is the supremum of all possible values of ε which satisfy (6). c(x) and vε−1(x) can be considered as channel capacity and designed code rate. They are very close together with the maximum value of εBP . Codes with higher threshold exhibits better
waterfall performance in terms of (bit, symbol or frame) error
rates [15].

E. Minimum distance from channel capacity
Channel capacity is the upper bound on the rate at which the information can be reliably transmitted over a communications channel. There is a difference between the Shannon channel capacity and the maximum rate which can be obtained by (λ, ρ) optimization. In other words, the optimization of the (λ, ρ) pair achieves a (1 − δ) fraction of the capacity, where δ is called the multiplicative gap.
Let (λ, ρ) be a degree distribution ensemble with the threshold εBP (λ, ρ), which is obtained by using BP [14]. In this case, the design rate of LDPC code can be expressed as:

The main weakness of proposed method relative to the LP and the GA algorithms is that GA-LP method requires a higher computation time at each cycle of optimization. This is mainly due to the degree-two model for the check node degree distributions: larger number of edges leads to more complex node degree distributions. For instance, the codes for case 1 (λ(x) with 7 degrees of freedom), the parity check matrices H9L9P9×2000, H9G9A9×2000, and H9G9A9×−2L0P00 include 9891, 15524, and 61803 edges, respectively. Similarly, for the codes used in case 2 (λ(x) with 12 degrees of freedom), the parity check matrices H9L9P9×2000, H9G9A9×2000, and H9G9A9×−2L0P00 involve 56758, 59638, and 67641 edges, respectively.
VI. CONCLUSION
In this paper, a joint GA and LP optimization method is proposed for the design of good LDPC codes over binary erasure channels. The proposed method tries to meet five coding criteria: higher rate, higher threshold, better stability, and lower multiplicative gap, by iterating between variable node degree distribution optimization using GA and check node degree distribution optimization using LP. Simulation results show the advantages of the proposed method but this at the cost of a more complex model for the check node degree distribution. There is a trade-off between computational complexity and performance: as we allow for an increase in complexity, better LDPC codes can be found. The FER performance curves of the proposed method shows better performance in both the waterfall and in the error floor regions. EXIT chart analysis has been used for the calculation and visualization of the threshold using the belief propagation algorithm.
