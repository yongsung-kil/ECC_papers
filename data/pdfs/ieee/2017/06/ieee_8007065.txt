Message Alignment for Discrete LDPC Decoders with Quadrature Amplitude Modulation
Jan Lewandowsky, Maximilian Stark, Gerhard Bauch Hamburg University of Technology Institute of Communications 21073 Hamburg, Germany
Email: {jan.lewandowsky,maximilian.stark,bauch}@tuhh.de

Abstract—Recent works describe the design of discrete decoders for low-density parity-check codes by application of mutual information maximizing clustering algorithms in discrete density evolution. In the resulting discrete message passing decoders only integers are exchanged and node operations become simple lookup operations. Earlier works only describe discrete decoders for binary modulation schemes. This paper presents a new technique called message alignment which enables to design discrete decoders for higher-order modulation schemes. First, we design a channel output quantizer for quadrature amplitude modulation with the Information Bottleneck method. The quantizer attempts to preserve the relevant information on the modulation symbols. Afterwards, we illustrate that the assignment of several bits to one modulation symbol does not allow straightforward decoder design with the available discrete density evolution technique. The proposed message alignment solves this problem. The resulting discrete decoder is compared with state-of-the-art decoders using bit error rate simulations.
I. INTRODUCTION
Low-density parity-check (LDPC) codes are powerful error correcting codes, but decoding of LDPC codes is a bottleneck in modern communication receivers. Belief propagation decoders have high memory and computational complexity. The complexity of belief propagation decoding can be reduced by application of approximate check node operations and by using short bit width representations of log-likelihood-ratios (LLRs) [1], [2]. However, reduced complexity is achieved at the expense of a performance degradation. Several recent works [2]–[7] describe alternative methods to design fully discrete LDPC decoders for regular LDPC codes with tools from information theory. Mutual information maximizing clustering algorithms are applied in a discrete density evolution scheme originally introduced in [8] and used in several modified variants in [2]–[7]. The aim is finding message mappings of integer messages that serve as node operations in the decoder. The resulting discrete decoders have performance very close to belief propagation decoding for quantized output additive white Gaussian noise (AWGN) [3], [5], [6] and also for Rayleigh fading channels [7]. For example, in [5]–[7] losses of only fractions of a decibel over Eb/N0 in comparison to double precision belief propagation decoders were achieved with a 4 bit message representation. Moreover, the complete decoder can be implemented using only integer arithmetic and lookup tables [3]–[7]. Numerous mutual information maximizing clustering algorithms exist (e.g., [9]–[12]). In

this paper we use the Information Bottleneck method [11], [12] for mutual information maximizing clustering in discrete density evolution, as we already did in [5]–[7]. The prior works on discrete LDPC decoders investigate transmission schemes for binary input channels and binary phase shift keying (BPSK) modulation. To achieve high data rates and more flexibility, it is desirable to also use discrete decoders with higher-order modulation schemes, for example, quadrature amplitude modulation (QAM). Perhaps surprisingly, in this context several problems occur that we aim to solve in this paper. In a first step, a suitable mutual information maximizing channel output quantizer for QAM under AWGN has to be designed. The quantizer shall preserve the relevant information on the transmitted QAM symbols. We also use the Information Bottleneck method [11], [12] to design a quantizer with the desired characteristic. The resulting quantizer outputs unsigned integer indices intended to be processed by a discrete decoder. In contrast to BPSK, for M -QAM a received sample carries information about more than one, i.e., log2 M bits. We show that the inherent assignment of several bits to one modulation symbol in QAM complicates design of a discrete LDPC decoder with the discrete density evolution scheme used in [5]–[7]. The main contribution of this paper is a new technique called message alignment which enables to design a discrete LDPC decoder for QAM. The paper is structured as follows: Section II introduces the system model and some prerequisites. Section III presents the proposed quantizer design and the new message alignment technique. Section IV illustrates exemplary results of message alignment and compares the resulting discrete decoder with state-of-the-art decoders for LDPC codes using bit error rate simulations. We conclude the paper in Section V. We use the following notation: Random variable X, realization x, probability distribution p(x), joint distribution p(x, y), conditional distribution p(x|y), mutual information I(X; Y ), column vector y, transposed vector yT, complex x = xre + jxim, set S. A tilde indicates a continuous variable, e.g., x˜. More specific notations are introduced on appearance.

B. The Information Bottleneck Method
The Information Bottleneck method [11], [12] is a generic framework from the field of machine learning. It attempts to design a compression mapping p(t|y) which maps an observation Y to a compression variable T . The key idea is to design p(t|y) in a way which preserves the maximum possible amount of mutual information I(T ; X) ≤ I(X; Y ) under a constraint for the cardinality of the event space of T . Variable X is termed the relevant random variable. In this paper only deterministic mappings p(t|y), i.e., p(t|y) ∈ {0, 1} ∀(t, y), are considered. Such mappings can be interpreted as lookup tables mapping y to t. Several Information Bottleneck algorithms aiming to find an optimum compression mapping p(t|y) exist [11], [12]. All Information Bottleneck algorithms input joint distribution p(x, y) and yield the compression mapping p(t|y) and joint distribution p(x, t) = p(x|t)p(t). More details on the Information Bottleneck method can be found in [11], [12].

C. Message Alignment
We propose a preprocessing of the quantized indices rkre and rkim at the quantizer output, enabling to use equations (1) and (2) for design of the node operations despite the distinctness of p(ck,ν , rkre) and p(ck,ν , rkim) for different ν. In particular, we design deterministic mappings p(zk,ν |rkre) and p(zk,ν |rkim). Given a received pair (rkre, rkim) these mappings yield transformed unsigned integers zk,ν ∈ T , ν ∈ {0, 1, 2, 3} for each bit ck,ν which are passed to the check nodes. The proposed method is motivated by the following exemplary observation in Figure 4: In the upper part of Figure 4, rkre = 8 for bit ck,0 corresponds to L(ck,0|rkre) ≈ 0. In the lower part, rkre = 3 and rkre = 12 correspond to almost the same LLR L(ck,1|rkre) ≈ 0 for bit ck,1. Since a check node cannot distinguish to which index ν an incoming message corresponds, the messages zk,0
and zk,1 passed to the check nodes for decoding should be identical for both bits when these respective rkre appear. In this manner, we want to achieve that whenever a certain
integer message zk,ν is passed to any check node, this integer corresponds to (approximately) the same LLR, independent
of the bit label index ν. To indicate that all bits bl in the

ck,ν, we add subscript ν to posterior and joint distributions of bl, yielding pν (bl|.) and pν (bl, .), if bl = ck,ν in the following. A received rkre implies posterior distributions p(ck,0|rkre) and p(ck,1|rkre) for the respective bits ck,0 and ck,1. Our aim is now to align p0(bl|zk,0) and p1(bl|zk,1) for zk,0 = zk,1. A. Exemplary Illustration of Message Alignment
Figure 5 shows the result of the message alignment tech-
nique. In addition to Figure 4, arrows indicate which integer indices rkre from the quantizer are mapped onto which zk,1 by p(zk,1|rkre). Going back to our previous observation that for bit index ν = 1 received samples rkre = 3 and rkre = 12 correspond to the same (almost zero) LLR L(ck,1|rkre), we note that the message alignment algorithm relabeled these rkre to zk,1 = 8 as assumed. This is highlighted using red arrows. Due to symmetry of L(ck,1|rkre) always two integers rkre are mapped onto the same zk,1. Finally, the upper part of Figure 5 also shows LLRs L(bl|yl) calculated from p(bl, yl) = p(bl|yl)p(yl) in (7) as horizontal line markers. The markers indicate that LLRs L(bl|yl) do not significantly differ from LLRs L(ck,0|rkre). We conclude that the proposed technique aligns p0(bl|zk,0) and p1(bl|zk,1) for zk,0 = zk,1 as desired.

B. Bit Error Rate Performance

Figure 6 shows the simulated bit error rate performance of a discrete decoder with the considered 16-QAM over Eb/N0. It is compared with the performance of several reference decoders. We used the length Nv = 8000 LDPC code labeled 8000.4000.3.483 from [14]. The code has (dv, dc) = (3, 6) and rate Rc = 1/2. All decoders performed a maximum of 50 iterations. A 4 bit quantizer, i.e., quantization to |T | = 16 levels and also 4 bit messages in the discrete decoder were used. The discrete decoder was generated only once for Eb/N0 = 3.5 dB, but used for all Eb/N0. The quantizer and the message alignment mappings were matched to Eb/N0. As an ultimate reference, performance of a double precision belief
propagation decoder working on continuous channel LLRs is
shown (black curve). To illustrate the inevitable loss resulting
from the proposed quantizer, a curve for the same decoder applied to quantized LLRs, calculated from p(ck,ν |rkre) and p(ck,ν |rkim) is shown (blue curve). The loss induced by the proposed quantizer is very small. The blue curve is a fair
reference because here the double precision belief propagation
decoder and the discrete decoder suffered from the same
information loss due to channel output quantization. The green
curve refers to the discrete decoder with the proposed message alignment. A loss of just 0.15 dB in comparison to the belief propagation decoder working on quantized channel LLRs is highlighted for bit error rate ≈ 10−6. We also included a comparison to the well known min-sum decoder (red curve).
The min-sum decoder used double precision LLRs calculated
from the continuous AWGN channel output. Hence, it does
not suffer from a mentionable quantization loss. Despite this
privilege, the min-sum decoder is outperformed by the discrete
decoder. To illustrate that message alignment is the key to
achieve the shown performance, a curve for the discrete
decoder without message alignment is included (brown curve). Here all mappings p(zk,ν |rkre) and p(zk,ν |rkim) were identity mappings. In this case decoding failed completely.

V. CONCLUSION
Discrete LDPC decoders for QAM and regular codes were
designed by application of a new technique called message
alignment. Using message alignment, the meanings of the
integer channel messages in terms of the posterior distributions
of a bit given this integer can be aligned. Message alignment
is necessary because different to LLRs, integer messages do
not represent this posterior distribution inherently. Simulation
results showed that message alignment enables to design
discrete decoders for QAM with discrete density evolution.
Similar to earlier investigated BPSK schemes, the performance
is very close to belief propagation decoders. In future work
we plan to apply message alignment to discrete decoding of irregular LDPC codes. Here a very similar problem exists:
During decoding, a particular node cannot consider the degree
of a connected node delivering an integer message, but the
respective posterior distributions of the codeword bits given
the passed integer messages depend on this degree.
