Abstract²In this paper, we studied the irregular row weight problem of DVB-S2 LDPC short frame. This problem will cause the failure of decoding. Based on a special structure of check matrix address table, we have achieved a simple and common solution to the problem.
Keywords-DVB-S2; LDPC short frame; irregular row weight
I. INTRODUCTION
The DVB-S2 standard has been extensively studied due to its superior performance. Its performance advantage mainly comes from BCH cascaded LDPC forward error correction (FEC) codec system. The LDPC in DVB-S2 has normal frames with the codeword length of 64800 bits and short frames with the codeword length of 16200 bits [1]. The normal frame supports 11 code rates, the short frame is 10. For convenience in referring to code rates, each short frame code rate is named to the nearest normal frame code rate, even if the actual rate is slightly different. III. ANALYSIS AND SOLUTION OF IRREGULAR ROW WEIGHT
PROMBLEM IN SHORT FRAME
A. Quantitative analysis
The column-based MATs can be converted to row-based form [7]. Similar to the column-based form, row-based MATs can also use the regularity of DVB-S2 LDPC for compressed storage. Using the q-value given by DVB-S2 standard, the (N-K) rows of an matrix are divided into 360 sets, each containing q rows. The first q rows consist of the 0th set, the second q rows consist of the 1st set 1, and so on. To store the matrix, simply store the non-zero elements address for each row in the 0th set. The non-zero elements address of the other sets can be obtained by a simple translation of the 0th set. Divide the matrix from left to right by 360 columns as one block. In order to save storage overhead, the row-based MATs can be divided into two parts. Block table storing the block index of the non-zero elements in the 0th set. And offset table storing the right offset value of the same elements as block table. The row weights and their distributions in of different rates can be also obtained from row-based MATs, just as column form. By analyzing the row-based MAT, we can find that the following 5 rates of short frame are irregular row weight: 1/4, 1/2, 3/4, 4/5, 5/6. Now, take 5/6 short frame as an example to analyze. The block and offset table of 5/6 short frame are shown in Fig.1 and Fig.2, respectively.

Obviously, for short frames with irregular row weight, a logical error will occur if the normal algorithm is used to decode by the maximum row weight. Or some zero elements will be treated as nonzero elements. These two cases will lead to decoding error. If the rows with different weights are processed separately, it is necessary to store the position and the row weight information of all irregular rows. And logical judgment and calculation will also increase. This will undoubtedly increase the complexity of system, especially for multi-rate systems.

V. CONCLUSION
In this paper, the irregular row weight problem of DVB-S2 LDPC short frame is analyzed in detail, and a method to solve this problem is proposed by reconstructing the row-based MATs. This solution effectively improves the degradation of decoding performance due to the irregular row weight problem. The method is easy to implement and applies to all code rates, while the system complexity is almost no increase.
