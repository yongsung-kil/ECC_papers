EXIT Chart Aided LDPC Code Design for Symmetric Alpha-Stable Impulsive Noise

Bin Dai, Rongke Liu, Member, IEEE, Yi Hou, Ling Zhao, and Zhen Mei

Abstract— In this letter, an iterative analysis method based on an extrinsic information transfer (EXIT) chart and quantized density evolution is developed for low-density paritycheck (LDPC) codes over channels with symmetric alpha-stable (SαS) impulsive noise. Based on the proposed scheme, a method to optimize ensembles of LDPC codes under different levels of impulsiveness is presented and it is shown that the information rates of our optimized code ensembles can achieve 95.4% and 94.7% of the channel capacity for α = 1.8 and α = 1, respectively. Furthermore, experimental results, including EXIT curves, thresholds, and bit-error rate performance of optimized code ensembles, are obtained to verify the effectiveness of our analysis.
Index Terms— LDPC codes, EXIT chart, impulsive noise, symmetric alpha-stable distribution.
I. INTRODUCTION
L OW-DENSITY parity-check (LDPC) codes are a class of linear block codes which received a lot of attention due to their near-capacity performance. In addition, they are widely applied to different recent standards such as WiMAX and DVB-S2, owing to their high error-correction capability and low decoding complexity.
Conventionally, Gaussian distributions are used to describe noises in communication systems. However, some communication systems exhibit non-Gaussian noise which appears impulsive in nature, namely, wireless networks [1], shallow water communications [2] and powerline communications (PLC) [3]. These impulsive noises last for a very short duration and have much greater amplitudes than Gaussian noises. In particular, under very general assumptions, the first-order distribution of impulsive interference follows a symmetric alpha-stable (SαS) law [4]. Hence, the nonGaussian impulsive noise channel can be modeled by additive white SαS noise (AWSαSN).
However, as a class of powerful error-correction codes, the performance of LDPC codes is not well studied over impulsive noise channels, especially for the code design. There are several approaches, such as density evolution (DE) [5], quantized DE (QDE) [6] and extrinsic information transfer (EXIT) charts [7], that have been employed to analyze the asymptotic

Compared with DE, EXIT charts not only provide a visible tool to predict LDPC code performance, but can also be used to design LDPC codes by matching the transfer curves of variable-node (VN) decoder and checknode (CN) decoder. To our best knowledge, there is no literature on EXIT chart analysis for LDPC codes over AWSαSN channels. Since the probability density function (PDF) of SαS distribution is not given in closed-form, the analytic expression of EXIT functions is difficult to obtain.
In this letter, an EXIT chart analysis for LDPC code design over AWSαSN channels is proposed. Our contributions are divided into two aspects: 1. An EXIT chart analysis based on QDE for LDPC codes is proposed. 2. We propose a code optimization method which based on EXIT chart analysis to design capacity-approaching irregular LDPC codes. IV. RESULTS
In this section, two optimized code ensembles are designed for α = 1.8 and α = 1, respectively and proposed optimized code ensembles are given in part A. In addition, EXIT curves of our optimized code ensembles and the referential code ensemble optimized for additive white Gaussian noise (AWGN) channels [11] are compared in part B. To validate our analysis, the bit-error rate (BER) performance of codes constructed from each ensemble is also obtained.
A. Optimized Code Ensembles
To verify the performance of proposed optimized code ensembles, we compare proposed optimized code ensembles with the referential code ensemble (λ(x) := 0.30013x + 0.28395x2 + 0.41592x7 and ρ(x) := 0.22919x5 + 0.77081x6) optimized for AWGN channels [11]. To make a fair comparison, the maximum variable node degree of optimized code ensembles is also set to be 8. We note that, code rates of optimized code ensembles are equal to or greater than the code rate of the referential code ensemble (R = 0.5). Without loss of generality, two optimized code ensembles are designed for slightly (α = 1.8) and strongly (α = 1) impulsive noise channels.
By using the proposed method, the optimized code ensemble 1 designed for α = 1.8 is λ(x) := 0.2943x + 0.2523x2 + 0.1733x5 + 0.2801x7 and ρ(x) := 0.1637x5 + 0.8363x6. The optimized code ensemble 2 designed for α = 1 is λ(x) := 0.2923x + 0.2492x2 + 0.1701x5 + 0.2884x7 and ρ(x) := 0.1642x5 + 0.8358x6. The channel capacities of α = 1.8, γ = 0.62 and α = 1, γ = 0.42 are 0.527 and 0.528, respectively. The capacity calculation for AWSαSN channels can refer to [12]. Hence, code rates of ensemble 1 and ensemble 2 are 0.503 and 0.5 which reach 95.4% and 94.7% of the channel capacity for AWSαSN channels, respectively.
B. Simulation Results of Optimized Code Ensembles
In this section, EXIT curves of the referential code ensemble and proposed optimized code ensembles are represented for α = 1.8 and α = 1 (Fig. 2). In both plots, the inset shows EXIT curves for VN (solid lines) and CN (dashed lines) in detail. It is shown that there is a tunnel between VN and CN curves. Under same channel conditions, the wider the tunnel between VN and CN curves is, the lower the decoding threshold becomes.
Fig. 2(a) shows EXIT curves for slightly impulsive noise (α = 1.8). It can be seen that VN and CN curves of the referential code ensemble intersect at the (0.533, 0.890) point, while these two curves of ensemble 1 meet at the (1, 1) point. Ideally, the decoding threshold of LDPC codes can be obtained by observing whether EXIT curves intersect before reaching the (1, 1) point at a specific Eb/N0 value. Note that QDE can also be used to find the threshold γ ∗ and its corresponding Eb/N0. Hence when α = 1.8, the decoding threshold of ensemble 1 is 0.86 dB (γ ∗ = 0.62) and that of the referential code ensemble is 1.0 dB (γ ∗ = 0.61).

V. CONCLUSION
In this letter, an EXIT chart scheme for LDPC codes is introduced for AWSαSN channels. In addition, we have proposed an efficient optimization method for LDPC codes based on the EXIT chart analysis. The information rates of optimized code ensemble 1 and ensemble 2 can achieve 95.4% and 94.7% of the channel capacity, respectively. Moreover, experimental results including thresholds and BER performance show that the optimized code ensemble 1 and ensemble 2 can achieve 0.1 dB (α = 1.8) and 0.2 dB (α = 1) gains over the referential code ensemble. Hence, our proposed optimization method can be used to design LDPC codes which approach the channel capacity for AWSαSN channels with different levels of impulsiveness.
