A Systematic Approach For Achieving Low Bit Error Rate of LDPC Decoder Using MWD Algorithm
Shruti Wahane,1 Sandeep Kakde,2 Atish Khobragade3 and Wael Elmedany4
1,2Dept. of Electronics Engg. Y C College of Engineering, Nagpur, India 3Dept. of Electronics Engg. Rajiv Gandhi College of Engg. Nagpur, India
4Dept. of Computer Engg. University of Bahrain Manama, Bahrain Email : 1wahaneshruti@gmail.com 2sandip.kakde@gmail.com 3atish_khobragade@rediffmail.com 4welmedany@uob.edu.bh

Abstract—One of the most significant barriers for using Low Density Parity Check (LDPC) codes in many wireless communication system and magnetic storage devices is the bit error rate floor phenomena with their complex message passing iterative algorithm. LDPC codes aremostly used in communication system because of its outstanding performance achievement. There are various decoding algorithm for LDPC decoder. In this paper, we presented the mathematical analysis of Minimum Weight Decoding (MWD) algorithm and its error rate performance over Additive White Gaussian Noise (AWGN) channel. A comparison of bit error rate performance over signal to noise ratio (dB) is carried out for different codes such as hamming code, BCH code and LDPC code. This paper shows bit error performance of decoder using minimum weight decoding algorithm. The main significance of this paper is that it shows efficient performance of Minimum Weight Decoding (MWD) algorithm based LDPC Decoder. We presented three classes of codes (1) hamming code, (2) BCH code and (3) LDPC Codes. We demonstrate the effectiveness of LDPC decoder over other two code decoders for the code rate 1/2 and block length (16,8) which is tarnished for its error floors.
Index Terms—LDPC codes; BCH Codes; Hamming codes; Minimum Weight decoding (MWD) algorithm; communication system; AWGN; BPSK; error correcting capability.

I. INTRODUCTION
Low density parity-check (LDPC) code is an error correcting code invented by RobortGallager in 1960. LDPC codes used in noisy communication system that minimizes the probability of loss of information. Forward error correction (FEC) codes are widely used to encounter data errors caused through noisy channel during transmission. Assured count of error bits can be detected as well as corrected by adding some parity check bits in end of information message bit vectors, with fewer retransmission. Low density parity check is dominant forward error correction coding technique which has excellent bit error performance over noisy channels [1]. Moreover, LDPC code is used in the decoding technique as they have achieved low complexity in comparison with other correction codes. With advancement in processing supremacy, LDPC codes are widely used in many high speed communication system such as digital video broadcasting, Wi-MAX, 4G/5G wireless systems and many more. Low-density paritycheckcodes are type of linear block codes. The name came

into existence that it has a number of 1’sless than the number of 0’s in their parity-check matrix. LDPC codes had achieved enormous responsiveness in the coding technique due to their excellent performance of error correction ability. More or lessun systematically constructed LDPC codes, measured in Bit Error Rate (BER), come very close to that for the AWGN channel (within 0.05 dB) [1]. This paper presents the minimum weight decoding algorithm for LDPC decoder. The basic communication system is as shown in Fig. 1.
Hamming code was invented by Richard Hamming in 1950. While decoding Hamming code can detect two bit error and correct single bit error, due to limited redundancy that hamming code add to the data, they can only detect and correct error when error rate is lower. On other hand BCH Code were invented by Bose, Chaudhuri and Hocquenghen in 1960. This BCH code provides multiple error correction. But the code length for BCH can not be chosen randomly. BCH Code length always has an odd value. BCH code with long code word is less practical. On other hand a lot of LDPC modification exist, LDPC Codes are classified as

Minimum weight decoding algorithm is one of the message passing technique. When a code word is received, very first task is to divide the received codeword by the generator polynomial vector, to check if there exists an error or not. If the result is not zero then the weight (W ) of that remainder (syndrome) result is calculated and then compared with the error correcting capability of the code word. If the weight (W ) is less than or equal to the error correcting capability, then the syndrome itself is the error pattern of the received word. If the weight (W ) is not less or equal to the error correcting capability of the code, then the process of cyclically shifting (rotation) and division is carried out and decoding process is repeated till the weight of the remainder reaches less than or equal to the error correcting capability of the code, or the syndrome of the remainder is to be repeated. If we reach the case, where error message is detected then the decoding process is stopped by adding the error message with the received word to correct the occurred errors.

When transmitted code word are received they are not same as the transmitted one. So in order to achieve the exact and correct code word. If the weight is not less than error correcting capability of code (t) then the process of cyclic shift(rotation) and division is repeated until weight should be less than or equals to error correcting capability of code (t).
