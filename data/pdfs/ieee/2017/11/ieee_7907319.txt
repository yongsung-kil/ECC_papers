Iterative Channel Detection With LDPC Product Code for Bit-Patterned Media Recording
Seongkwon Jeong and Jaejin Lee
Department of ICMC Convergence Technology, Soongsil University, Seoul 06978, South Korea

Because of the explosion in data growth, the requirement for high-density storage systems has increased. Bit-patterned media recording (BPMR) is a candidate for the next-generation magnetic recording systems, and its many advantages facilitate the achievement of recording densities of 1 Tb/in2 and beyond. In BPMR, each information bit is represented by a magnetic island; however, due to the small spacings between the along- and across-track islands that are for the achievement of a high areal density, severe extents of the inter-symbol interference and inter-track interference appear. These error factors degrade the system performance of the recording system. In this paper, an iterative channel detection scheme with a low-density parity check (LDPC) product code for which the extrinsic information between the soft output Viterbi algorithm and the LDPC product code is used for the BPMR is proposed. For the improvement of the BPMR performance, the modified extrinsic information data are exploited.
Index Terms— Bit-patterned media recording (BPMR), burst error, low-density parity check (LDPC) code, product code.

I. INTRODUCTION
B IT-PATTERNED media recording (BPMR) is one of the promising technologies for overcoming the limitation of the conventional storage technology since it exhibits the superparamagnetism phenomenon [1]. In BPMR, since each bit is recorded in a single magnetic domain surrounded by a nonmagnetic region between the islands, BPMR can accomplish high densities up to 4 Tb/in2 [2]. Furthermore, BPMR has the strength of an easy track search, a decreased nonlinear transition shift, and a simple timing recovery due to the nonmagnetic region. To achieve higher areal recording densities, the period between the bit islands in the across track and along track must be reduced.
When the inter-island interval is decreased, 2-D interference that is constituted with inter-symbol interference (ISI) and inter-track interference (ITI) is increased. Also, media noise and written errors are problematic in terms of BPMR [3], [4]. These error factors severely degrade the systemic performance of BPMR. From the perspective of signal processing, it is necessary to eliminate the errors by using appropriate equalizers, detection algorithms, error-correction codes, and modulation codes.
To mitigate the ISI and ITI, Nabavi and Kumar [5] proposed a 2-D equalizer with optimized equalizer coefficients and a partial-response (PR) target. An iterative 2-D equalizer that is based on contraction mapping is introduced to reduce the ISI and ITI [6]. Also, to alleviate the ITI, a modified trellis for BPMR was introduced [7]. In holographic data storage, to moderate the 2-D ISI, the 2-D soft output Viterbi algorithm (SOVA) with low-density parity check (LDPC) codes was proposed [8]. In addition, a high code rate modulation code was presented to avoid the ISI and ITI [9].
The above-mentioned schemes pay attention to the removal of the ISI and ITI, but the burst error that is generated

For example, when physical impacts of the same size occur in the storage systems of 4 Tb/in2 and 1 Tb/in2, respectively, for the 4 Tb/in2 system, this impact will result in a very long burst error, in contrast to the 1 Tb/in2 system. A burst detector and an error-correction capability are, therefore, needed for the long burst error [10]. To inhibit the burst error, a product code was introduced by Elias [11] in 1954. The Reed–Solomon (RS) product code of DVD systems is mostly used for the correction of random and burst errors [12]. Besides the RS product, various product schemes have been proposed [13], [14].
The proposal of this paper is an iterative channel detection with the LDPC product code for which the extrinsic information between the SOVA and the LDPC is used for the BPMR to mitigate the ISI and ITI via the 2-D SOVA, and where the product code is used to counteract the burst error.
The rest of this paper is organized as follows. In Section II, the product code and the proposed model are explained and presented. Section III briefly explains the BPMR-channel model and shows the simulation results. The conclusion is presented in Section IV.
II. PROPOSED LDPC PRODUCT CODE
A. Product Code
To enhance the error-correction capability for the burst error, a product code comprises several error-correction codes. In general, the product code is composed of the two errorcorrection codes: the inner code c1 (n1, k1) and the outer code c2 (n2, k2). Information bits are set in an array of the k2 by the k1. An inner code with the minimum number of parities is used to detect the burst error and correct most random errors and short burst errors, but the inner code cannot accurately decode the long burst error. To notify the location of the long burst error, the inner decoder declares the erasure information and informs the location to the outer decoder with the maximum number of parities. B. Iterative Channel Detection With LDPC Product Code
Fig. 1 shows the block diagram of the proposed parallel concatenated LDPC product code, which is composed of two LDPC encoders of the horizontal and vertical directions. The data flow of the block diagram is summarized as follows.
1) The input data are encoded by the LDPC codes of horizontal and vertical directions, respectively.
2) After the encoded data a[ p, q] are passed through the BPMR channel, the received data r [ p, q] are processed by the 2-D equalizer and the 2-D SOVA.
3) After the LLR value d[ p,q] that is received from the 2-D SOVA is decoded by the LDPC decoder of the horizontal direction, the horizontal LDPC decoder transmits the extrinsic information eh1[ p,q] of the horizontal LDPC decoder to the vertical LDPC decoder.
4) The LDPC decoder of the vertical direction decodes the LLR value d[ p,q] of the 2-D SOVA with the extrinsic information eh1[ p,q] of the horizontal LDPC.
5) After the vertical LDPC decoder sends the extrinsic information ev [ p,q] to the horizontal LDPC decoder,
the LLR value d[ p,q] of the 2-D SOVA with the extrinsic information ev [ p,q] of the vertical LDPC is decoded by the horizontal LDPC decoder. 6) Then, the extrinsic information of horizontal and vertical LDPC decoders is averaged, and the averaged extrinsic information value ea[ p,q] is scaled down. (This means the value of the extrinsic information is multiplied by α.)

Then, this value em[ p,q] is added to the equalizer output value and processed by the 2-D SOVA. 7) The procedure from 3) to 6) is repeated three times, and the vertical-direction LDPC decoder outputs the estimated data a[ p,q].
For instance, the 2-D equalizer and the 2-D SOVA received the channel output r [ p, q] and the equalizer output z[ p, q].
The two LDPC decoders use the same input data d[ p,q] from the 2-D SOVA. After the horizontal LDPC decodes the data from the 2-D SOVA, the vertical LDPC receives the extrinsic information eh1[ p,q] from the horizontal LDPC. After the vertical LDPC decodes the data from the 2-D SOVA and the eh1[ p,q], the horizontal LDPC receives the ev [ p,q] from the vertical LDPC. Then, the horizontal LDPC decodes the data from the 2-D SOVA and the ev [ p,q] once again. The island pulse-response shape is applied to the perpendicular magnetic recording for the BPMR with and without the soft underlayer (SUL) [15], [16]. To achieve 1 Tb/in2, it is assumed that the length of the square magnetic island is 12.5 nm, and the along-track PW50 values are 21.2 and 18.8 nm with and without the SUL, respectively. For the fixed island width, the across-track profile of the island is similar to the Lorentzian pulse [15]. In the present case, the PW50 is 31.2 nm. The pulse response can be written as follows:
