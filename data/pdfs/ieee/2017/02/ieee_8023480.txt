Ahstract-A new class of bit flipping algorithms for low­
density parity-check codes over the binary symmetric channel 
is proposed. The algorithms employ multiple bits at a variable 
node to represent its reliability, and multiple bits a check node 
to capture the sequence of its syndrome values. The check node 
update function thus requires a simple bit-shift operation, while 
the variable node updates require a nonlinear Boolean function. 
This class of multi-bit flipping (MBF) algorithms is enhanced by 
the probabilistic gradient descent (PGD) algorithm. The gradient 
descent algorithm minimizes the variable node energy function 
which, in addition to the classical term which quantifies the 
discrepancy between the variable estimate and channel value, 
also involves an additive term defined as a weighted sum of 
neighboring check node states. Only the variable nodes with 
the maximal value of energy are eligible for updating, but 
the updates are not done by default but probabilistically. The 
resulting probabilistic gradient descent multi-bit flipping PGD­
MBF algorithm combined with rewinding improves the codeword 
probability of error while keeping the complexity lower than that 
of the state-of-the-art algorithms of comparable throughput. 
I. INTRODUCTION 
Extremely high-speed applications such as memory, storage, 
and optical conununications dictate using only hard decisions 
from a cOlmnunication channel. For such binary synunetric 
channels (BSCs), low-density parity-check (LDPC) codes and 
their iterative algorithms derived from Belief Propagation (BP) 
offer an attractive and so far unsurpassed tradeoff between 
error performance and low complexity. Iterative decoders 
operate by passing messages along the edges of the Tanner 
graph of a code, and low decoding complexity results from 
the sparseness of the Tanner graph and locality of message 
updating. This locality is both "spatial" - a message from a 
given node depends only on the messages from the neighbor­
ing nodes- and "temporal" - a message in a given iteration 
depends only on messages from the previous iteration and the 
bit obtained from the channel. 
A posteriori probability (APP) decoder [1] is a suboptimal 
but memory-efficient derivative of the BP decoder. In the APP 
decoder, the nodes in a Tanner graph send messages in a local 
and "intrinsic" manner. A single message - node value or node 
state - is sent to all neighboring nodes. The node value is 
a posteriori probability used to estimate that variable. Thus, 
APP decoders require memory proportional to the number of 
nodes in the Tanner graph of the code, rather than to the 
number of edges. The suboptimality of APP decoders can be 
remedied to a great extent by using the message reweighting 
parameter which depends on the node and iteration [2]. To 
further improve the complexity of APP decoding, the precision 
of messages can be reduced but at the expense of degrading 
decoding performance [3], [4], especially in the error floor 
regime. 
On the extreme side of the spectrum of APP-like decoders 
are the traditional bit flipping (BF) algorithms, which are the 
fastest and least complex as they use single bit precision of 
node values. The check node operations of these algorithms 
are modulo-two additions while the variable node operations 
involve only majority logic. The parallel bit flipping algorithm 
can correct a linear number of errors in the code length [5], [6] 
for variable-regular LDPC codes with sufficient variable node 
degree. For finite-length codes these algorithms do not have 
sufficient error correction capability, and numerous improve­
ments have been proposed [7]-[10]. Unfortunately almost all 
of these algorithms require soft information from a channel. 
The first notable modification which does not require soft 
information but provides significant performance improvement 
is the two-bit flipping (TBF) algorithm [11], which employs 
two-bits to represent the APP values of variable nodes and two 
bits indicate the check node reliability. However, the TBF is 
not a quantized version of the APP algorithm because its check 
node update function involves only simple Boolean operations 
and does not require arithmetic operations required in the APP 
algorithm. The TBF algorithm permits characterization of the 
set of uncorrectable patterns and the corresponding subgraphs 
in the code Tanner graph using a recursive procedure [12]. 
All the above decoders involve only local computations. 
Recently, a new class of bit-flipping decoders that in addition 
to local computation also involve global computation was 
proposed in [13]. This decoder prescribes a rule for flipping 
a value of a bit in a given iteration based on the so-called 
energy function of that variable. Global minimization of the 
energy functions coupled with the gradient descent algorithm 
was shown to greatly improves convergence. This makes this 
algorithm, know as Gradient Descent Bit-Flipping (GDBF), 
superior to the BF. As opposed to other iterative decoders 
which employ only local computations, a condition for flipping 
a bit in GDBF decoder involves computations involving energy 
functions of all bits. Despite its performance that is much 
better than the BF, the GDBF still exhibits error floor. We have

hardware complexity burden required to perform these global 
computations is compensated by simplicity of the node update 
computations, resulting in overall complexity significantly 
lower than that of BP-like algorithm such as min-sum of the 
same or lower throughput [16]. 
This fact motivates the approach taken in this paper, which 
combines several components. In APP algorithms, a state of 
a check node in some iteration is a measure of the reliability 
that this check node was satisfied at the end of the previous 
iteration, and is obtained by computing hyperbolic tangent -
or its approximation - of the reliabilities of the neighboring 
variable nodes, which is computationally costly. To ensure 
high speed, our algorithm does not require any arithmetic 
operations to update check node state. Variables simply pass 
their estimates to neighboring check nodes, and each check 
node computes its the syndrome value. Consequently, the only 
information based on which a reliability of the check node at 
the end of a given iteration can be computed is the sequence of 
past syndrome values of this check node. The number of such 
past syndrome values determines the check node memory, Pc. 
This sequence defines the state of a check node at the end of 
a given iteration, and the check node update function requires 
a simple bit shift operation. This is the first key component 
of our algorithm. As in the APP algorithm, the variable node 
values are represented by Pv bits which define its strength, but 
unlike the APP the variable update function 1> is a nonlinear 
function of neighboring check node states. This is the second 
component of the algorithm. In this paper we are interested 
in multi-bit flipping algorithms (MBF) for which Pc ;:: 1 and 
Pv ;:: 1. Clearly the number of such algorithms is huge, and 
we present our initial findings on a heuristic used to identify 
good ones. 
The third component of the proposed algorithm is prob­
abilistic gradient descent. We use the term "probabilistic" 
to distinguish it from the well know stochastic (on-line) -
approximation of the (batch) gradient descent algorithm for 
classification. The variable update function 1> of the MBF is 
only a condition for flipping. The second condition for flipping 
is the energy function of a variable node, which is explained 
in detail in the paper. 
The rest of the paper is organized as follows. Section II gives 
the necessary background. Section III introduces the class of 
MBF algorithms. The probabilistic gradient descent version of 
MBF algorithms is given in Section IV. Numerical results are 
given in Section V and Section VI concludes the paper.

Before introducing the gradient descent version of the 
introduced algorithm, we would like to recall the observation 
made in [15]. Consider a setting of dv check nodes that 
are connected to a variable node v in which the number

also takes advantage of this observation but with a different 
approach. 
However neigther PBFA nor TBFAI are incapable of evalu­

The MBF decoders owe their effectiveness to successive 
construction of a trapping set profile for a given (dv, dc)­
regular Tanner graph. Here we rely on a method proposed in 
[11], which enable selection of different MBF decoders with 
acceptable performance-complexity tradeoff, when applied 
on 3-left-regular graphs. Namely, we test the improvement 
provided by employment of probabilistic gradient descent to 
three decoders VI, V3, V4, labeled as in [11]. The VI has 
been constructed after careful characterization of trapping 
sets, among 41,472,000 decoders that satisfy the symmetry 
and irreducibility conditions, described in Section III. VI is 
capable of correcting all weight-three error patterns, and has 
the highest probability to correct a random error pattern of 
weight-four among other MBF decoders. On the other hand, 
the decoders V3, V4, are not individual decoders, rather a 
collection of different MBF decoders run in parallel, that 
complement each other. Furthermore, the construction method 
has also been directed to selecting component decoders based 
on their fitness to a specific code, for example (155,64) quasi­
cyclic code, usually called Tanner code [19]. It has be shown 
that extracting harmful structures from the trapping set profile 
that are not contained in the Tanner graph of (155,64) code 
leads to design of V3, a decoder with four component MBF 
decoders and better error-correction capability, compared to 
VI. Finally, V4 is composed of 35 different decoder and 
exibits superior performance compared to other presented 
decoders.

VI. CONCLUSIONS 
We proposed a novel class of bit flipping algorithms for 
LDPC codes which uses probabilistic gradient flipping prin­
ciple, while allowing multiple bit representations of messages 
that are exchanged between nodes of the corresponding Tanner 
graph. Although this paper contains our initial results, we 
were able to show significant improvement of the proposed 
technique compared to previous MBF solutions which do not 
employ probabilistic gradient descent. Moreover, since our 
solution is only sub-optimal in a large search space, there are 
possibilities for further improvement.
