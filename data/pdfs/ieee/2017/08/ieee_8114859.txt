Irregular QC-LDPC Based Multi-level Coded Modulation Scheme for the Next Generation Optical
Communication Systems

Dongdong Wang1, Liqian Wang1, Xue Chen1, Chen Ju1, Zhirong Wang1, Aimei Fei2, Huitao Wang2, and Qi Zhang2
1State Key Lab of Information Photonics and Optical Communications Beijing University of Posts and Telecomunictaions, Beijing, China
2Pre-research & Standard Department ZTE Corporation Wireline R&D Insititute, Beijing, China

Abstract—In this paper, we mainly focus on the optimization of multi-level coded modulation scheme in terms of sub-channel code rate optimal assignment and irregular QC-LDPC code construction with the aid of EXIT chart analysis. Simulation results show that the proposed MLCM scheme can achieve additional 0.1dB NCG compared with the UEP coded modulation scheme.
Keywords-Irregular QC-LDPC code; EXIT chart; mutul information; Multi-level coded modulation
I. INTRODUCTION With current emerging digital coherent technologies [1] and advanced digital signal processing techniques [2], coded modulation, that refers to the combination of soft decision forward error correction (FEC) codes with high order modulation formats, becomes one of the key enabling technologies for the next generation optical communication systems which can achieve higher spectral efficiency transmission with reliable bit error rate (BER) performance [3].
Recently, QC-LDPC code based multi-level coded modulation (MLCM) scheme has been verified through FPGA emulation system [4], additional 0.5dB coding gain can be achieved by employing the unequal protection (UEP) strategy compared with the conventional coded modulation scheme. However, limited by the regular QC-LDPC code based rateadaptive scheme, which can only achieve some fixed code rates due to the array size of the mother code cannot be extended to the large size which will dramatically reduce the regular QC-LDPC code BER performance. Therefore, the code rate assigned to each sub-channel may not be the optimal, which will lead to some code rates exceed their corresponding sub-channel capacities, that will cause the theory limit performance degradation of the MLCM system.
In order to solve this problem, we propose an irregular QCLDPC code construction method, by which the constructed irregular QC-LDPC codes do not only have very flexible code rate, but also have the BER performance approaching the Shannon limit. Furthermore, with the aid of MI and GMI analysis, the optimal code rate for each sub-channel of MLCM system is obtained. Then, by assigning each EXIT chart analysis aided irregular QC-LDPC code with the optimal code rate to its corresponding sub-channel, the resulting irregular

QC-LDPC code based 16QAM MLCM scheme shows excellent BER performance, compared with the corresponding UEP coded modulation scheme, additional 0.1 dB coding gain can be achieved.

IRREGULAR QC-LDPC CODE CONSTRUCTION
In order to demonstrate the effectiveness of the proposed MLC-PID scheme, good irregular QC-LDPC codes with all the above mentioned code rates are needed, which will be used in the above mentioned coded modulation schemes.
A. Mother Code Construction
For fairly comparison, all the four code rates, 5/6, 4/5, 7/10, 2/3, must be exactly achieved, so the array size (J , L) of the mother code should be chosen carefully, according to the relationship R = 1 − J / L between code rate and array size, L must be at least equal to the LCM of 6, 5, 10 and 3. Therefore, L is chosen to be 30, then J can be determined by the lowest code rate 2/3 due to it has the largest array size, other code rate can be realized by shortening the array size. The array size (J , L) is set to (10, 30) , and the size p of circulant permutation matrix is set to 540 , that will lead to a mother code with code length 16200 bits. Then we adopt the finite geometry based local optimal heuristic searching algorithm to generate the mother code with code parameter (10, 30, 540) . The searching process is started with the smallest array size (5, 30) , and will be changed to (6, 30) , which is generated by
adding a new row to (5, 30) , until cycles in (5, 30) cannot be
reduced further. Repeat until cycles in (10, 30) can no longer be reduced. This approach can ensure that all the needed codes with different array size have good performance in terms of girth and cycle numbers.

B. EXIT Chart Analysis and Irregular Code Construction
EXIT chart [6] is used for analyzing the optimized column weight distribution λ (x) for each code, the smallest column weight is 3 and the column weight number for each code is L − 2 . Note that, although the threshold of the optimized λ (x) with column weight 2 is better than the optimized
λ (x) without column weight 2, it will prematurely exhibits the error floor phenomenon, therefore column weight 2 is not suitable for the QC-LDPC code design for high speed optical systems. Combined the randomly generated masking matrix [7] with the mother code parity check matrix, irregular QC-LDPC codes with the optimized λ (x) that listed in table I can be constructed. SIMULATION RESULTS
In this section, we verified both the BER performance of the proposed MLC-PID scheme and the UEP scheme in [4], as a reference, in 16QAM AWGN channel, where MLC-PID scheme adopts irregular QC-LDPC codes with code rate 5/6 and 2/3 as its component codes, UEP scheme adopts irregular QC-LDPC codes with code rate 4/5 and 7/10 as its component codes. Both the two schemes adopt the same total code rate 3/4 and same code length 16200 bits.
Fig. 4 shows the BER performance of the two coded modulation schemes, where the proposed scheme outperforms the UEP scheme 0.1dB at the BER of 10−8 , this benefit mainly comes from the flexible code rate adaptive construction method in section III, which can achieve the more flexible code rates via the array size expansion. In addition, the code BER performance can be further improved by adding the irregular degree distribution property to the original parity check matrix, where the expansion will lead to the degradation of girth or the increase of cycle numbers, however, the degree distribution with high accuracy will overcompensate the performance loss caused by girth degradation and cycle number increase.

CONCLUSION
In this paper, we propose an irregular QC-LDPC code based MLC-PID coded modulation scheme, where the code rate assignment and code construction should be designed jointly, simulation results show that it can achieve additional 0.1dB coding gain compared with the UEP coded modulation scheme. Therefore, the proposed coded modulation scheme represents one of the promising candidates for the nextgeneration optical communication systems.
