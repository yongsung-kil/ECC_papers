I. INTRODUCTION
Low-density parity-check (LDPC) codes were first introduced by Gallager in 1962 [1], [2]. LDPC codes have attracted much attention as the good error correcting codes achieving the near Shannon limit performance. These characteristics make LDPC an attractive candidate for error correcting code applications in communication systems. Recently, LDPC codes have been adopted as forward error correction codes in the emerging communication standards such as Wi-Fi (IEEE 802.11n, 802.11ac) [3-4], WiGig (802.11ad) [5], 10Gbase-T Ethernet (802.3an) [6], etc.
Even though LDPC codes achieve outstanding performance, LDPC codes require a large number of computational complexity. Hence, reducing the computational complexity is the one of key issue in LDPC codes. To reduce the computational complexity of LDPC decoders, the forced convergence (FC) has been proposed in [10]. The FC algorithm is a technique to reduce the computational complexity by generating the number of deactivated variable nodes (VNs). The FC algorithm exploits the fact that a large number of VNs converges to a strong reliability after few iterations, i.e., these highly reliable VNs have already been reliably decoded. Thus, the FC algorithm can skip their reliability in subsequent iterations. The FC uses a threshold value to decide whether to update VNs or not [9]. To lower computational complexity, Ernesto et al. use two threshold determinants to deactivate VNs and check nodes (CNs) [10-13].

However, the probability of satisfying the VN condition is extremely low. Moreover, additional hardware complexity is required. To solve these problem, the simplified forced convergence (SFC) algorithm is proposed to lower the computational complexity and hardware complexity using only one threshold value [14].
In this paper, we propose the adaptive forced convergence (AFC) algorithm to further reduce the computational complexity using only one adaptive threshold value. The threshold value is dynamically determined to deactivate the highly reliable VNs. Since the AFC algorithm can lower the threshold value of VNs, the number of the deactivated VNs increases. Therefore, the AFC algorithm can significantly reduce the computational complexity compared with existing algorithm [9-14].
This paper is organized as follows. In Section II, the existing FC for the min-sum algorithm is introduced. Section III presents the proposed SFC algorithm. The simulation results are presented in Section IV. Finally, conclusions are given in Section V.

We use MATLAB to perform the bit error rate (BER) performance simulation and compare with the existing algorithm [10]. We use the IEEE 802.11ad standard LDPC code and the AWGN channel with BPSK modulation. We
assume that the maximum number of iterations Imax are five.

Figure 2 shows the BER performance comparisons with the MS algorithm, the FC algorithm [10], and the proposed AFC algorithm. Fig. 2 is simulated in the environment with the code rate of 1/2, t is eight, and the predefined constant for the
dynamic threshold value is 0.75. The proposed AFC algorithm has little degradation of BER performance compared with the existing FC algorithm [10].

V. CONCLUSIUON
G
This paper proposes the AFC algorithm to reduce computational complexity of CNs using the only one adaptive threshold value. The AFC algorithm employ the modified CN operation to deactivate the CNs using the adjacent VNs. Therefore, AFC can simultaneously reduce the computational complexity of CNs and VNs without using CN threshold value. Moreover, AFC lowers the threshold value of VNs, the number of the deactivated VNs rapidly increases. Therefore, when Eb/N0 = 5 dB, 6 dB, and 7 dB, AFC can significantly reduce the computational complexity of CNs by 29.04%, 24.53%, and 14.74% respectively, compared with the FC algorithm [10].
