Design of LDPC Decoder Using Message Passing Algorithm

Prerana Dhanorkar, Monica Kalbande

 Abstract— Low density parity check (LDPC) codes are most favorable error-correcting codes and also in terms of high coding gain, high throughput and low power dissipation LDPC code is advantageous. In this paper, for designing LDPC decoder the Min Sum Algorithm for decoding the LDPC codes is suggested. Min Sum algorithm is belong to soft –decision decoding techniques The hardware implementation of check node and variable node using a traditional technique based on HDL code is very complex and time consuming. Here a new approach for designing of LDPC decoder using Simulink modelling have been proposed. For check node and variable node the hdl code have been generated using the Simulink hdl coder in matlab.
Index Terms—-LDPC codes; Min sum algorithm; Communication Channel; AWGN; error correcting capacity.
I. INTRODUCTION
he Low-Density Parity-Check (LDPC) codes are error correcting code pre-owned in noisy communication channels to diminish the probability loss of statistic .In 1962 LDPC code was discovered by Gallager but these codes does not much popular for a few spans because of high realization convolution [1]. However it, gets popular in 1997 when it was again re-introduced by MacKay and Neal in 1997[2]. Forward error correction (FEC) codes have been extensively cast-off to switch data errors caused by transmission of data through a noisy channel. A certain number of bit errors can be identified and altered without retransmission by adding some of the parity bits to message bits. LDPC codes have huge benefits over turbo codes, with reduced execution complexity, improved bit error rate (BER) at a low signal to noise ratio (SNR). Hence LDPC codes have become progressively popular and because of the progresses in computing power it is use in high speed communication standards such as WLAN,

WiMax and Digital Video Broadcasting - Second Generation (DVB-S2).Sum-Product Algorithm (SPA) has high design complexity and belongs to soft-decision decoding which gives the unbeatable decoding performance. To clarify the node operation several refinement is made in SPA. The check node behaviour is simplified by diminishing the non-linear function to an approached quantization table and even to logarithmic functions. The check node behaviour in SPA is restructured in min-sum algorithm is restructured.

II. LDPC CODES
LDPC codes be owned to a class of linear block codes .As their name recommend, its attain parity-check matrix (H) reside of hugely minor number of non-zero elements. The sparseness of H regulate the decoding complexity and the minimum distance of the code [3] The number of nonzero entries in each of the rows and columns of parity matrix is mutually known as degree of distribution. The degree distribution of rows and columns is static if the parity matrix is said to be regular, otherwise it is said to be Irregular. Tanner graph of parity check matrix can be represented by bipartite graph. A cycle in the graph is a sequence of connected nodes, which start and end at the same node. A regular (3, 6) parity-check matrix for 10-bit code word length in each columns number of ones is three and in row number of ones is six is shown in Fig. 1 (a) and the corresponding Tanner graph of the parity check Matrix is shown in Fig. 1 (b).

2) MIN SUM ALGORITHM At the start of the decoding process, LLR values are given to the variable node and pass that to the check node. A variable node sum all the information that is received at the end of iteration. At the end of the process syndrome is checked. V. CONCLUSION
In this paper a original design methodology for modelling, synthesis and implementation for the designs of Check node and Variable node based on Simulink has been presented. The proposed technique deals with reduced design complexity, less effort and less time. The higher level of abstraction offered by the Simulink modelling environment is utilized by the new design methodology. The proposed new technique shortens the testing strategy very rapidly. This technique also helps to reduce the time of evaluating and testing the check node and variable node designs.
