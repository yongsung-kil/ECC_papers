IEEE ICC 2017 Wireless Communications Symposium
Spatially Coupled Code Design for Three-Phase Bidirectional Relaying

Md. Noor-A-Rahim and Yong Liang Guan Centre for Infocomm Technology (INFINITUS) Nanyang Technological University, Singapore
Email: {narahim, eylguan}@ntu.edu.sg

Abstract—In this paper, we propose a design of spatially coupled repeat-accumulate (SC-RA) codes over a three-phase bidirectional relay. The channels between the nodes are assumed binary input additive white Gaussian noise (BIAWGN) channels, while decode-and-forward (DF) relaying is considered at the relay. We present low complexity density evolution analysis for the proposed bidirectional relay code, which leads to a simple code design procedure. The performance of the proposed relay coding scheme significantly outperforms the performance of the relay code based on regular RA block code. We show that the decoding threshold of the proposed relay code approaches the theoretical limit for a wide range of design rates. We also show the finite-length performance of the proposed bidirectional relay code through simulation results.
I. INTRODUCTION
The bidirectional/two-way relaying [1–4], where two end nodes exchange their information via a relay node, has received much attention in recent years due to its ability to enhance the performance of relay communication. The half-duplex bidirectional relaying can be performed in four, three or two phases/time slots. In the four-phase scheme, which consists of four traditional point-to-point (P2P) communication, the two end nodes take turn to send their information to the relay node and the relay node takes two phases to forward the information to the other end node. In the three-phase scheme [1], the relay node takes single phase to forward the coded information to both end nodes. In the two-phase scheme [2–4], the end nodes transmit simultaneously at the first phase and at the second phase, the relay node broadcasts a signal generated based on the superimposed signal. Among different protocols studied for bidirectional relaying, decode-and-forward (DF) relaying [5] strategy is one of the well-known protocols. In this paper, we focus on the design of error correcting code for bidirectional relaying while considering DF protocol.
Low-density parity-check (LDPC) codes are potential candidates for bidirectional relay code design, since they exhibits capacity approaching performance for P2P scenario, while maintaining low complexity decoding. The bidirectional relay code design based on LDPC or LDPC-like codes (such as repeat accumulate (RA) codes) has been broadly investigated in recent literature. Among them, code design for the twophase scheme is investigated in [6–10], while code design for the three-phase scheme is investigated in [11]. Althouh,

Moreover, the optimization procedure of irregular LDPC code becomes very complex for a variety of channel conditions [12].
Similar to the irregular LDPC codes, spatially coupled codes [13], [14] are also potential candidates for bidirectional relay code design due to their threshold saturation1 property over P2P scenario. Moreover, simple optimization procedure can be formulated for different channel conditions, since spatially coupled codes are constructed from regular codes. Besides the widely studied P2P scenario [15], [16], the threshold saturation property of spatially coupled code is also proved over other channels including multiple access channels [17], [18], one way relay channels [19–21]. Motivated from these facts, we aim to design three-phase bidirectional relay code using spatially coupled code. The rationale behind the choosing of three-phase scheme is as follows. Although two-phase scheme provides superior throughput, this scheme exhibits very high decoding complexity at the relay and perfect synchronization is required to achieve optimum result. On the other hand, three-phase scheme is simple to implement since traditional P2P decoding technique can be used at the relay and does not need to handle the synchronization issue since no superimposed signal is involved. To the best of our knowledge, spatially coupled code design for the three-phase bidirectional relaying has not been studied previously.
Two prominent classes of spatially coupled codes are namely known as spatially coupled LDPC (SC-LDPC) codes and spatially coupled RA (SC-RA) codes. We are particularly interested in the SC-RA code, since SC-RA code provides much simpler encoding technique than the SC-LDPC code while maintaining similar capacity approaching performance. In this paper, we propose SC-RA code design for half-duplex three-phase bidirectional relaying with the DF protocol. We present low complexity density evolution analysis for the proposed relay codes and show a simple procedure to obtain an optimized spatially coupled bidirectional relay code. With numerical examples, we show that the proposed relay code significantly outperforms its counterpart conventional regular RA block code and achieves capacity approaching performance

II. PRELIMINARIES

A. Bidirectional Relay Channel Model

The bidirectional relaying scenario, considered in this paper, is illustrated in Fig. 1. This relay network consists of two end nodes (A and B) and one relay node R. We assume a half-duplex relaying scenario where the exchange of information between the end nodes take place in three phases with normalized phase duration τ1, τ2, and τ3, where τ1 + τ2 + τ3 = 1. In the first phase, node A encodes and transmits the encoded message, which is received by the relay and node B. In the second phase, node B encodes and transmits the encoded message, which is received by the relay and end node A. We assume that the end nodes are unable to decode the other end’s encoded message due to the worst channel condition, while the relay can decode the encoded messages received in phase 1 and phase 2. In the third phase, the relay node R broadcasts a message, which is generated based on the decoded message. At the end of phase 3, node A (node B) jointly decodes the message of node B ( node A) by using its own codeword, signals received in phase 2 (phase 1), and signals received in phase 3. C. Optimization
Recall that the bidirectional relay code design, presented in subsection III-A, becomes the joint design of code CA, code CB and the overall code such that the relay can correctly decode the codeword cA and cB in phase 1 and phase 2, respectively, while node A (node B) can successfully recover codeword cB (codeword cA) in phase 3. To design a good SC-RA bidirectional relay code for a given relay channel, we present the following two step optimization procedure. In the first step, we optimize code CA and code CB separately, such that they approach the capacity of the AR and BR channels, respectively. This optimization step is same as the conventional SC-RA code optimization for P2P communication scenario. In the second step, we optimize the code parameters that are used to generate the codeword segment cR. For the sake of simplicity, we consider WRA = QRA and WRB = QRB . Then we search for the degree pairs (QRA , SRA ) and (QRB , SRB ) such that the overall code will be decodable at both end nodes. V. CONCLUSION
In this paper, we have proposed the design of threephase bidirectional relay code using spatially coupled repeataccumulate codes. Density evolution analysis of the proposed code is presented and a simple optimization procedure is given to obtain good SC-RA bidirectional relay code for a given bidirectional relay channel. The performance of the proposed code is compared with the conventional regular RA block code and shown that the proposed code outperforms the conventional RA code for both asymptotic and finite-length scenarios. Using numerical results obtained from density evolution, it is shown that the proposed code approaches the Shannon limit for a wide range of code rates.
