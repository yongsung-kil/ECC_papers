Abstract—This paper proposes a new coded modulation technique, called generalized low-density parity-check (GLDPC) coded modulation, where individual redundant modulation mapping nodes are connected to code constraint nodes in a pseudorandom manner, and according to predesigned degree distributions. The main contribution of this paper is to derive a compact solution for the distribution of the squared Euclidean distance (SED) of a GLDPC coded modulation ensemble in the general case. In order to allow for the lack of symmetry of the distribution with respect to the transmitted constellation symbol sequence, the SED is averaged with respect to all possible transmitted constellation symbol sequences. This result can be used directly as a design tool for targeting a good system performance in the error floor region. Since the proposed GLDPC coded modulation scheme is also amenable to extrinsic information transfer (EXIT) chart analysis, a high degree of design freedom is thus facilitated.
I. INTRODUCTION
Since the rediscovery in the 1990s of Gallager’s low-density parity-check (LDPC) codes [1], research on code (ensemble) design has focused on characterizing the decoding threshold [2] as well as the codeword weight [3] and stopping set size [4] distributions. Generalized LDPC (GLDPC) codes, first proposed in [5], allowed for each check node (CN) to implement arbitrary linear block code constraints, and doublygeneralized D-GLDPC codes [6] allowed generalization also of the variable node (VN) constraints; the corresponding analyses of the codeword weight and stopping set size distributions were performed in [7] and [8]. The results of these analyses are useful for the design of low-density coding schemes for use with binary modulation.
Modern high-data-rate communication systems however require the use of higher-order modulation, e.g. phase-shift keying (PSK) or quadrature amplitude modulation (QAM), as well as a pseudorandom code structure which is amenable to iterative constellation demapping and decoding. The most prominent such technique is bit-interleaved coded modulation with iterative detection and decoding (BICM-ID) [9], [10], which is known to outperform the original trellis coded modulation (TCM) scheme of [11] over fading channels, and also to have excellent performance over the additive white Gaussian noise (AWGN) channel [12]. Extrinsic information transfer (EXIT) charts have been successfully used to design

advanced BICM schemes for use over AWGN, e.g., in [13], [14], and [15]. In these works, the constellation mapping was optimized based on EXIT chart analysis together with local Euclidean distance considerations. In each case however, the optimization is limited to the use of a single constellation mapping as well as to the use of convolutional coding.
In this paper, we propose a new approach to coded modulation called GLDPC coded modulation. This allows the use of different modulation mappings and indeed different constellations at the VNs, so that the structure can be truly irregular and the designer has a greater degree of control over the connectivity between the redundant modulation mappers and the code constraints. We analyze the squared Euclidean distance (SED) distribution of this coded modulation approach in the general case; this is possible through the fact that the modulation mapping is embedded in the VN structure. In contrast to LDPC coding with BPSK, a lack of codebook symmetry necessitates that the SED distribution is averaged with respect to all possible transmitted constellation symbol sequences. The proposed coded modulation structure may be viewed as a generalization of the scheme of [6] to coded modulation, and also as a generalization of LDPC-coded BICM.

Definition 2.2 (Paired assignment): A paired assignment
is an ordered pair (S1, S2) of subsets of the edges of the Tanner graph. The weight of a paired assignment is equal to n = (n1 n2 n3) where n1 is the number of edges in S1 only, n2 is the number of edges in S2 only, and n3 is the number of edges in S1 and S2. A paired assignment is said to be check-valid if the following condition holds: for each i ∈ {1, 2}, supposing that each edge of Si carries a 1 and each of the other edges carries a 0, each CN recognizes a valid local codeword.

Definition 2.3 (Paired constellation symbol assignment):
A paired constellation symbol assignment is an ordered pair
(s1, s2) of length-n constellation symbol sequences on the VN side. Any such paired constellation symbol assignment
induces a paired assignment (S1, S2) through the local labeling functions L(t) at the VNs (each input sequence si induces an output Si). The weight of the paired constellation symbol assignment (s1, s2) is given by (u, v1, v2, v3) where u = s1 − s2 2 is the SED between the symbol sequences s1 and s2, v1 is the number of edges in S1 only, v2 is the number of edges in S2 only, and v3 is the number of edges in S1 and S2. Finally, the paired constellation symbol assignment (s1, s2) is said to be check-valid if its induced paired assignment (S1, S2) is check-valid.
