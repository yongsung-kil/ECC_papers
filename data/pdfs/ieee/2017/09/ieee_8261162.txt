Performance of Protograph LDPC Codes over Ergodic Nakagami Fading Channels
Yi Fang∗†, Guofa Cai∗, Zhaojie Yang∗, Pingping Chen§, and Guojun Han∗ ∗School of Information Engineering, Guangdong University of Technology, Guangzhou, China †Department of Electronic and Information Engineering, Hong Kong Polytechnic University, Hong Kong
§Department of Electronic Engineering, Fuzhou University, Fuzhou, China Email: {fangyi, gjhan}@gdut.edu.cn; caiguofa2006@126.com; zjyang0131@163.com; ppchen.xm@gmail.com

Abstract—The protograph low-density parity-check (LDPC) codes have drawn much attention in the past decade due to their simple structures and outstanding error performance. Unfortunately, the conventional protograph codes designed for additive white Gaussian noise (AWGN) channels can not perform well in ergodic Nakagami fading channels. To address this issue, we construct a new rate-1/2 protograph code, called improved accumulate-repeat-accumulate (IARA) code, by exploiting a modified PEXIT algorithm. We compare the decoding threshold and bit error rate (BER) of the IARA code with the existing protograph codes, the regular LDPC code and the irregular code over ergodic Nakagami fading channels. Both analytical and simulated results show that the proposed IARA code can achieve the lowest BER in the high signal-to-noise-ratio region. Hence, the IARA code can be considered as a good candidate for use in wireless communication systems with fast fading.
Index Terms—Channel state information, extrinsic information transfer (EXIT) algorithm, protograph low-density parity-check (LDPC) code, ergodic Nakagami fading channel.
I. INTRODUCTION
Forward error-correction coding has been a very important technique for improving the reliability of wireless data transmission. Among the error-correction codes, low-density parity-check (LDPC) code can work very close to the Shannon limit under the message-passing algorithm over additive white Gaussian noise (AWGN) channels [1], [2]. However, LDPC codes that perform well in the AWGN channel may not do so in fading environments, e.g., ergodic and non-ergodic fading channels. In [3]–[5], LDPC codes have been studied over Nakagami fading channels and some optimized LDPC codes have shown strong robustness against signal fading. For example, in an ergodic fading channel, the distance between the Shannon capacity and an optimized rate-1/2 LDPC code proposed in [3] is found to be 0.07 dB only. Subsequently, much research effort has been devoted to modifying the conventional optimization methods such as density evolution [6], [7] and extrinsic information transfer (EXIT) function [5], [8]. The objective is to make use of the modified techniques to design capacity-approaching LDPC codes over ergodic Nakagami fading environments [9]–[11].
Yet, many of the capacity-approaching LDPC codes are irregular and hence suffers from two main drawbacks —

high error floor and nonlinear encoding. On one hand, some research has successfully reduced the error floor of shortblock-length LDPC codes which can then attain outstanding performance down to an error rate of 10−5 over ergodic fading channels [12]. On the other hand, structured LDPC codes, including quasi-cyclic LDPC (QC-LDPC) codes [13] and multilevel-structured LDPC codes [14], have also been studied over ergodic and non-ergodic fading channels. These low-complexity codes can also accomplish error performance comparable to other optimized LDPC codes. In addition, a novel type of LDPC codes, called protograph LDPC codes, has been invented by Thorpe [15]. Protograph codes has received much research interest because of their excellent error performance and low complexity [15]–[17]. Furthermore, two protograph codes, namely the accumulate-repeat3-accumulate (AR3A) code and the accumulate-repeat-by-4jagged-accumulate (AR4JA) code, which can realize linear encoding and decoding, have been constructed [18], [19]. In [20] and [21], the protograph EXIT (PEXIT) algorithms have been developed so as to facilitate the asymptotic-performance analysis over AWGN and ergodic fading channels. However, to date, there is no related work touching upon the design of protograph codes over ergodic Nakagami fading channels.
In this paper, we investigate the feasibility of protograph codes over ergodic Nakagami fading channels. We first present a modified PEXIT algorithm to derive the decoding threshold of protograph codes in such scenarios. We further alter the structure of the accumulate-repeat-by-3-accumulate (AR3A) code [19] and construct an improved ARA (IARA) code. The IARA code has the same complexity as the AR3A code, but possesses a lower proportion of degree-2 variable nodes. Theoretical and simulation results over a Nakagami fading channel have shown that the proposed IARA code can accomplish the best BER performance in the high signalto-noise-ratio (SNR) region compared with other protograph codes, regular LDPC code and optimized irregular LDPC code. Thus, the IARA code is a good candidate for use in fast-fading wireless communication applications.

II. SYSTEM MODEL

Fig. 1 shows the system model considered in this paper. As seen, the information bits (Info bits) are firstly encoded by the (punctured) protograph code. Afterwards, the binary coded bits vj ∈ {0, 1} (j = 1, 2, . . .) are passed to a binary phase-shiftkeying (BPSK) modulator, whose output is given by xj = (−1)vj ∈ {+1, −1}. The modulated signal is further passed through an ergodic Nakagami fading channel. The output of the channel, denoted by yj, can be written as

III. PROTOGRAPH CODES AND MODIFIED PEXIT ALGORITHM
A protograph G = (V, C, E) consists of three sets V , C and E corresponding to the variable nodes, check nodes and edges, respectively [22]. In a protograph, each edge ei,j ∈ E connects a variable node vj ∈ V to a check node ci ∈ C, and parallel edges are allowed. Furthermore, a protograph with N variable nodes and M check nodes can also be characterized by a base matrix B of size M × N , where the (i, j)th element bi,j denotes the number of edges connecting the variable node vj to the check node ci. A large protograph, called derived graph, corresponding to the protograph code can be obtained by performing the “copy-and-permute” operation on a given protograph.

EXIT function [23] can be used to trace the convergence behavior of the iterative decoding schemes and to efficiently estimate the decoding threshold of different codes. In [20] and [21], protograph EXIT (PEXIT) algorithms have been introduced to facilitate the analysis and design of protograph codes over AWGN and ergodic fast-fading channels. In the following, we describe the modified PEXIT algorithm for protograph codes over ergodic Nakagami fading channels.

B. Design of an Improved Protograph Code
Referring to the structure of the AR3A code shown in (13), there is a very high percentage (40 percent) of degree-2 variable nodes in the protograph of AR3A code, which may restrict the performance of the code in the high-SNR region [6], [7]. Aiming to improve the performance of the AR3A code, we reduce the proportion of degree-2 variable nodes by adding extra connections to one type of degree-2 variable nodes. For every edge added, moreover, one edge will be removed from a variable node with the largest degree so as to keep the overall complexity (number of connections) of the code unchanged.
To achieve this goal, we randomly select one of the two degree-2 variable nodes (corresponding to the fourth column and the fifth column) in the AR3A code shown in (13). We then add a new edge to connect it to a check node, and we delete an edge that connects the selected check node to the largest-degree variable node. V. SIMULATION RESULTS AND DISCUSSIONS
In this section, we present the simulation results of the proposed IARA code, the AR3A code, the AR4JA code, the regular (3, 6) code, and the optimized irregular LDPC code in [3] with a code rate R = 1/2. The channel being considered is the ergodic Nakagami fading channel with fading depth m = 1.
Fig. 3 depicts the BER curves of the proposed IARA code, the AR3A code, the AR3A code, the AR4JA code, the regular (3, 6) code, and the irregular LDPC code over an ergodic Nakagami fading channel with different information lengths. We can observe that the AR4JA code and the regular (3, 6) code are two worst-performing codes. Moreover, as shown in Fig. 3(a), the irregular LDPC code obtains a gain of about 0.1 dB over the IARA code at a BER of 10−4, which possesses better performance than the other two protograph codes and the regular LDPC code. However, the BER performance of the irregular LDPC code has little improvement when Eb/N0 exceeds 3.75 dB upon which the error floor emerges. In the same figure, we observe that the IARA code and the AR3A code have excellent error performance for the range of Eb/N0 under study. For example, at Eb/N0 = 4.0 dB, the regular code, the AR4JA code, and the irregular LDPC code achieve BERs of 5 × 10−4, 1.5 × 10−4 and 2 × 10−5, respectively; while the AR3A code and the IARA code accomplish BERs of 4 × 10−6 and 2 × 10−6, respectively. In addition, the IARA code has performance gains of about 0.1 dB, 0.4 dB, 0.5 dB, and 0.6 dB, respectively, over the AR3A code, the irregular code, the AR4JA code and the regular code at a BER of 10−5. Moreover, a large gain can be expected at a lower BER. Similar conclusions can be obtained from Fig. 3(b), in which information length is increased to 2048. As seen from Fig. 3(b), the irregular code still exhibits an error floor and performs worse than the IARA code and the AR3A code when the Eb/N0 exceeds 3.40 dB. Consequently, we conclude that the performance advantage of the IARA code and the AR3A code can be maintained as the information length is increased.
Remark: We have also performed simulations on other three realizations of the improved ARA code and have obtained similar results.
