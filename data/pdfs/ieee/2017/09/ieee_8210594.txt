Estimation of channel state information for non-volatile flash memories

∗Institute for System Dynamics HTWG Konstanz, University of Applied Sciences, Germany

†Faculty of Power Engineering and Telecommunications Georgian Technical University, Georgia Email: sshavgulidze@gncc.ge

Abstract—Error correction coding based on soft-input decoding can significantly improve the reliability of flash memories. Such soft-input decoding algorithms require reliability information about the state of the memory cell. This work proposes a channel model for soft-input decoding that considers the asymmetric error characteristic of multi-level cell (MLC) and triplelevel cell (TLC) memories. Based on this model, an estimation method for the channel state information is devised which avoids additional pilot data for channel estimation. Furthermore, the proposed method supports page-wise read operations.
I. INTRODUCTION
Non-volatile flash memories are becoming more and more important for mass data storage. The flash cells keep their electrical charge without a power supply. However, errors may occur while the information is read. In order to ensure a reliable information storage, error correction coding (ECC) is required. In recent years, mostly BCH codes with hardinput algebraic decoding were used for error correction [1], [2], [3]. The performance of error correction coding can be improved if reliability information about the state of the cell is available [4]. Soft-input decoding algorithms are required in order to exploit the reliability information. For instance, low-density parity-check (LDPC) codes and generalized concatenated codes with soft-input decoding can provide stronger error correcting performance in NAND flash memories [5], [6], [7], [8], [9], [10].
In contrast to algebraic hard-input decoding, soft-input decoding algorithms typically require channel state information, e.g. the bit error probability or the noise variance. Basically, the flash channel can be considered as binary input additive white Gaussian noise (AWGN) channel, where the channel output is quantized using a small number of bits [11]. But for MLC and TLC the probability that an input 0 will be flipped into a 1 differs from the probability for a flip from 1 to 0 [12], [13]. This error characteristic is not considered by the simple AWGN channel model. Moreover, the probabilities vary during the life time of a flash memory, e.g. the error probabilities depend on the number of program/erase cycles and the number of reads after programming.

In the work, we present a channel model that reflects the asymmetric error characteristic of the flash channel by considering different variances for the different charge levels of the flash cell. Moreover, we present a method to estimate the channel state information for the different quantization levels. Our approach is histogram based, similar to the methods proposed in [14], [15], where the complete distribution of all charge levels is estimated simultaneously. In contrast to [14] and [15], we consider a setup where only statistical data for a single decision threshold is available, i.e. a page-wise read operation. Furthermore, we assume that the reading thresholds are pre-determined, e.g. in order to optimize the soft-input decoding performance, whereas in [15] an adaptive bin placement is used to optimize the channel estimation.
The remainder of the paper is organized as follows. We describe the channel model in section II. The proposed channel estimation method is presented in section III. In section IV, we present simulation results for a generalized concatenated code with soft-input decoding. We compare the performance with perfect channel information to that with the proposed estimation method. Conclusions are drawn in section V.
II. CHANNEL MODEL
In flash memories, the cells store data in the form of a charge level. The basic memory element is a floating gate transistor, where the floating gate can be charged and erased. These floating gates keep their electrical charge without a power supply. To read out the data, a threshold voltage is applied to turn on the transistor. The stored charge level changes the threshold voltage of the transistor. Hence, the charge level can be inferred by applying different threshold values. There are different flash technologies that store two charge levels (single-level cell, SLC), four levels (multi-level cell, MLC), or eight levels (triple-level cell, TLC). The estimated RBER values are calculated based on the theoretical model using σ0 and σ1. The measured RBER values are bit error rates obtained from the measurements. The estimated RBER values are in good agreement with the measured values. Note that σ0, σ1 correspond to normalized mean values such that µ0 = −1 and µ1 = 1. For symmetric distributions, i.e. for σ0 ≈ σ1, the reference threshold is in the centre of the interval between µ0 and µ1 (in the point of origin for the normalized mean values). For asymmetric charge distributions, the reference threshold is not necessarily in the centre of the interval. The offset value in Tabel I indicates a shift of the threshold value with respect to the point of origin.
The objective of the channel estimation is to determine the LLR values for the different quantization intervals. Basically, this could be achieved by using pilot data that is stored in the flash and counting bit errors conditioned on the observation interval for the pilot data. However, the conditional error probabilities are typically very low. Therefore many pilot bits would be required to estimate the corresponding LLR values.
Based on the statistical model presented in section II, LLR estimation corresponds to estimating the mean and standard deviation for each charge level. We present an estimation approach that does not require pilot data. The estimation is based on the number of observations in each quantization interval. Note that relative frequencies of observations in each quantization interval should correspond to the probabilities qi according to Eq. (3). By measuring the relative frequency q˜i of observations for each interval li, we obtain an estimate of the probability distribution qi with i ∈ {0, ..., k}. The probability distribution qi is determined by the mean values and standard deviations of the corresponding charge levels.
Note that the threshold voltage sensing operation that is required to obtain the soft information incurs a larger energy

consumption and a latency penalty [21]. Hence, the soft information might only be used for blocks where the decoding without reliability information fails [9]. Furthermore, the LLR values are typically represented with a small number of bits, e.g. three or four bits per LLR value [9]. Due to the low resolution of the reliability information, we assume that the channel state information is only required for a small number of different channel states.
Based on this model, the channel estimation can be performed by a table lookup procedure. For each considered channel state, we calculate the corresponding distribution of qi and the corresponding LLR values. These values are stored in a lookup table. During the reading process we count the relative frequencies q˜i of observations for each interval li. In order to determine the best table entry, we use the Kullback– Leibler divergence, which is a non-symmetric measure of the difference between two probability distributions. Let qi and q˜i be two probability distributions. The Kullback-Leibler divergence is defined as

V. CONCLUSION
Soft-input decoding algorithms require reliability information about the state of the flash memory cells. In this paper, we have presented an estimation method for the channel state information. The channel state information depends on the charge level which may change due to read disturb and other effects. The proposed channel model considers different voltage distributions for different charge levels. The different distributions model the asymmetric error characteristic of multi-level cell and triple-level cell memories.
