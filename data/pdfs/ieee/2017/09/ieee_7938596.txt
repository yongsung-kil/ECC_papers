A Parity Structure for Scalable QC-LDPC Codes With All Nodes of Degree Three
Xiaoning Wu, Student Member, IEEE, Ming Jiang, Member, IEEE, and Chunming Zhao, Member, IEEE

Abstract— A novel block triple-diagonal (BTD) parity structure with all nodes of degree three is proposed for quasicyclic (QC) low-density parity-check (LDPC) codes. The BTD structure without low node degree can effectively improve the error floor performance and, meanwhile, enable low complexity encoding. With this structure, we use a modulo-lifting technology to construct a family of QC-LDPC codes, which supports scalable code lengths with a simple and flexible encoding solution. The simulation results show that our coding schemes for short and moderate lengths achieve better performance in the error floor region.
Index Terms— Parity structure, scalable code lengths, low error floor, efficient encoding, quasi-cyclic (QC) LDPC codes.
I. INTRODUCTION
T HANKS to the excellent performance in the waterfall region and the support for rate compatible transmission [1], quasi-cyclic low-density parity-check (QC-LDPC) codes have been chosen as the data channel coding scheme in 5G enhanced mobile broadband (eMBB) scenarios. Due to the inherent parallel encoding/decoding architectures and lower error floor, QC-LDPC codes are also considered as one of the most promising error correction technologies for 5G ultra reliable and low-latency communications (uRLLC) services.
While algebraic QC-LDPC codes [2] have low error floors and efficient encoding algorithms [3], [4] for corresponding code lengths, the flexibility of encoding with scalable code lengths is also significant for the uRLLC applications. The encoding flexibility can be achieved by block dualdiagonal (BDD) parity structures, which have been used in the QC-LDPC codes in 802.16e [5] with efficient encoding [6], [7]. But the codes with this structure may suffer from an inevitable error floor problem caused by a large proportion of degree-2 nodes. To lower the error floor, a parity structure with all node degrees at least 3 is preferred. There exist some excellent parity structures [1], [8], [9], but they can not meet our specific requirement of the degree distribution. An improved BDD (IBDD) structure introduced in [10] can satisfy the degree distribution with all nodes of degree three. However, the direct extension of this structure will lead to an intricate design and large data storage for encoding with scalable code lengths.

In this letter, we propose a block triple-diagonal (BTD) parity structure composed of all degree-3 variable nodes. Another important kind of BTD structure has been proposed in [9], which usually includes ultra low-weight variable nodes and is not appropriate for the scenario considered here. To avoid short cycles of length 4 and 6, and achieve linear encoding complexity for scalable code lengths, a greedy algorithm is developed to search proper shift values in our BTD structure. With the BTD structure, a family of scalable QC-LDPC codes is constructed by a combination of the error minimization progressive edge growth (EMPEG) algorithm [11] and the modulo lifting [12]. Simulation results show that the codes with the BTD structure outperform the existing scalable codes, such as the QC-LDPC codes in 802.16e [5] and Samsung 5G proposal [13], in the error floor region.

For convenience, we use x −i to denote x −i mod Z for i = 0, 1, · · · , Z − 1 in a binary polynomial. For example, a binary polynomial h(x ) = 1 + x + x 2 + x −1 + x −2 means that its shift values should be converted into {0, 1, 2, −1, −2} mod Z . respectively. For clear expression in the following context, we use Hp(x ) and Hp(Z)(x ) to denote the parity blocks before and after the modulo operations with Z , respectively.

In order to mitigate the computational complexity, we employ some simple methods for the search space reduction of the vector q. Firstly, we narrow the search space by some constraints which guarantee the girth g = 8 in the structure. Then, for each possible vector in this narrowed space, calculate the corresponding function (x, q) and get (x). If (x) is a polynomial with single nonzero term, then we keep this vector, otherwise delete it. From the selected vectors, take one vector as the proper values of q by the cycle distribution. The details of the greedy algorithm are described as follows.
1) Through an exhaustive search, generate all possible combinations of J + 2 integer values qi ∈ [−v, v], i = 0, . . . , J + 1 in q and select the solutions of q which satisfy with the constraints in Proposition 1.
2) According to every vector q in the selected solutions, calculate the corresponding function (x, q) and get (x). Only keep and store the solutions of q which can make (x) be a polynomial with only one nonzero term.
3) From the stored solutions, find the proper vector q by the cycle distribution of the parity structure.
Here, the integer value v and the number of elements J + 2 determine the complexity of the greedy algorithm, where the search space of q is (2v + 1)J +2 in the first step. Generally, v is set to a small number, v ≤ 3, to restrict the search space and to enlarge the scale of Z ≥ 3v + 1 in Proposition 1. The existence of such q is quite difficult to be proved for all the sizes. B. Construction for Scalable QC-LDPC Codes

According to a given Hp(x) with the proposed BTD parity structure, we can further construct Hm(x) of a QC-LDPC code

by the EMPEG algorithm in [11], and make it scalable by the

modulo lifting in [12]. Using the modulo lifting, Hm(x) with Zk, k = 0, 1, · · · , t − 1, denoted by Hm(Zk)(x ), are constructed progressively and finally Hm(Zt−1)(x ) is obtained. On the base of Hm(Zt−1)(x ), the shift values in Hm(Zk)(x ) can be determined by the modulo operation on that in Hm(Zt−1)(x ) with Zk . Then, Hm(Zk)(x ) is got by lifting the shift values with Zk. In order to generate Hm(Z)(x ) for Zk < Z < Zk+1, we directly set the shift values in Hm(Z)(x ) to those in Hm(Zk)(x ). Next, lift the shift values with Z and get Hm(Z)(x ).
Similar to Hm(Z)(x ), Hp(Z)(x ) can be also obtained by the modulo operation on the shift values in Hp(x) with Z . Therefore, using Hm(Zt−1)(x ) and Hp(x ), we can construct a family of QC-LDPC codes for arbitrary Z , Z ∈ [Z0, Zt−1], which

supports scalable code lengths with the information granularity

of (L − J ) bits. An example is given as follows.

V. CONCLUSION
In this letter, we propose a BTD parity structure which can support code length scaling and linear complex encoding. With this structure, all column weights can be set to 3, that can further lower the error floor. Although with all degree-3 nodes, our BTD structures are as flexible as the BDD structures that can support various code lengths and code rates. For the given BTD parity structure, we combine the EMPEG algorithm and the modulo lifting to get a family of QC-LDPC codes, which can be simply described by a shift-value array. Simulation results show that the codes with the BTD structures obviously outperform the existing codes in the high SNR region.
