Design of Check-Hybrid LDPC Codes for Data
Communications over Helicopter-Satellite Channels

Abstract—A class of check-hybrid low-density parity-check
(CH-LDPC) codes is designed and implemented to mitigate
the problem of periodical signal blockage over the helicopter-
satellite channels. The CH-LDPC code is derived by replacing
one layer check nodes of a quasi-cyclic LDPC (QC-LDPC)
code by simplex code constraints or first-order Reed-Muller
(RM) code constraints, based on the property that the check
matrix of the QC-LDPC code has a layered structure. Moreover,
extrinsic information transfer (EXIT) functions are established
to analyze the iterative decoding performance of the CH-LDPC
codes. Simulation results show that, compared with the QC-
LDPC coding scheme, the CH-LDPC coding scheme designed
in this paper achieves more than 25% bandwidth efficiency
improvement over the helicopter-satellite channels.
I. INTRODUCTION
Helicopter-satellite communications can provide critical
communication links between the helicopter and the ground
stations via a communication satellite, enabling helicopter-
satellite communications to apply in various areas such as
rescue missions after disasters and forest fire monitoring. How-
ever, one of the toughest obstacles in implementing helicopter-
satellite communications is that the transmitted signal from/to
the communication antenna is intermittently interrupted by
rotor blades of the helicopter, leading to periodical interruption
of the received signal [1].
Forward error control (FEC) coding with appropriate inter-
leaving method is an indispensable modular that can help to
recover the transmitted data in a periodical blockage impaired
channel [2]. Recently, there has been considerable interest in
applying LDPC codes over the helicopter-satellite channels,
due to their powerful performance and low decoding com-
plexity. In [3], the well-documented LDPC concatenated Bose-
Chaudhuri-Hocquenghen (BCH) codes from the Digital Video
Broadcast via Satellite 2nd generation (DVB-S2) waveform
specification [4], featured by long block lengths (16200 bits
and 64800 bits), are employed to mitigate the rotor blades
blockage. The work of [5] adopts a very low rate LDPC
coding scheme (1/9 and 2/9) to deliver uninterrupted data
transmission through intermittent blockages caused by rotary
wings. Shorter code length for fast re-acquisition at a lower
rate in [5] achieves the same decoding performance as the
rate-1/2 mode of DVB-S2, inevitably at the expense of great
bandwidth expansion.
One particularly attractive class of structured LDPC codes
for practical applications is the class of QC-LDPC codes [6].
However, QC-LDPC codes are not strong enough in helping to
correct the erasure errors in helicopter-satellite channels, since
most of the QC-LDPC codes are designed for additive white
gaussian noise (AWGN) channels. The work of [7] replaces
the single parity check (SPC) code constraints of a LDPC code
by the extended Hadamard (first-order RM) code constraints,
and additionally introduces the degree-1 nodes attached to
the communication channel, which can be considered as a
special kind of generalized LDPC (GLDPC) codes. Such
kind of GLDPC code may better handle the erasure errors,
with much more powerful SPC-constrained component codes
involved. However, they suffer an overall code rate loss, which
is attributed to the higher error correcting capability of the
component codes.
In this paper, a class of CH-LDPC codes is designed to mit-
igate the periodical rotor blades blockage over the helicopter-
satellite channels. The CH-LDPC code is a generalization from
the QC-LDPC code, and it has hybrid check code constraints:
the SPC code constraints and the component code constraints.
The CH-LDPC code is a different version of the previous
work on the GLDPC code of [7], embodied in the following
aspects: the base code of the CH-LDPC code is a QC-LDPC
code whose check matrix has a layered structure, and only one
layer SPC code constraints of the base code are generalized to
moderate the code rate loss of the GLDPC code. Except for
the first-order RM codes, simplex codes are also introduced
as the component codes. Unlike the first-order RM codes, the
simplex codes need no data preprocessing during component
codes encoding and decoding. Moreover, the EXIT curve of
the check nodes are depicted based on the approximate EXIT
functions of the component codes derived in this paper, which
is more flexible and easier to use than the Monte Carlo method
used in [7].
Taking both bandwidth utilization and power consumption
into account, the proposed FEC coding scheme over the
helicopter-satellite channels in this paper includes two CH-
LDPC codes: one with 843 information bits and code rate
1/4, and the other with 122 information bits and code rate
2/7. Simulation results show that CH-LDPC codes show better
capability to deal with blockage than QC-LDPC codes, and the

II. HELICOPTER-SATELLITE CHANNEL MODEL
Due to the installing restriction of the helicopter, the com-
munication antenna is usually mounted on the tail boom or
the rear fuselage, which is just under the rotor blades. Once
hovering or in flight, the rotor blades will periodically block
the antenna, consequently generating the periodical fading of
the received signal. The received signal fading can be modeled
as a rectangular window fading, as shown in Fig. 1. The
maximum reduction from the mean signal level can reach
10dB to 30dB, which is far beyond the accommodating ability
of link budget.
The time when two blades one behind the other sweep over
the antenna is called the blockage period 𝑇𝑝, which is inversely
proportional to the product of the rotor angular speed 𝑉𝑟and
the number of the blades 𝑁, that is

airframe orientation 𝛽. Fig. 2 shows the blockage ratios of
a helicopter with 4 pieces of blades. As shown in Fig. 2, 𝛾

25%. Therefore, a qualified FEC coding scheme for helicopter-

IV. DESIGN OF THE CH-LDPC CODES
The check matrix of an (𝑎, 𝑏, 𝑝) QC-LDPC code has an 𝑎-
layers structure, that is 𝑯QC = [𝑯0; 𝑯1; . . . ; 𝑯𝑎−1]. The CH-
LDPC code proposed in this paper is a generalization from the
QC-LDPC code where only one layer SPC code constraints
of the base QC-LDPC code are replaced by the component
codes constraints. This is more easily understood in Fig. 3,
which presents a different rendition of the Tanner graph. When
considering which layer to be replaced, the middle layer is
often the best choice, which can enable the component codes
to join in as many cycles, especially short cycles, in the Tanner
graph of the base code as possible, since it is well known that
short cycles degrade the the error correction performance of
LDPC codes.
In [7], the first-order RM codes are adopted as the compo-
nent codes. However, only half of the first-order RM codes
satisfy the SPC constraint directly, the other half have to
use nonsystematic generator matrices, which have to execute
data preprocessing before component codes encoding and
decoding. Besides the first-order RM codes, we also adopt
the simplex codes as component codes. Since the (2𝑟−1, 𝑟)
simplex code is always the dual code of the (2𝑟−1, 2𝑟−1−𝑟)
Hamming code, the generator matrix of the simplex code is
just the parity check matrix of the Hamming code due to
the dual code property for linear block codes [8]. Since a
systematic (2𝑟−1, 2𝑟−1−𝑟) Hamming code has a parity check
matrix 𝑯Ham whose columns consist of all nonzero binary
vectors of length 𝑟, each used once, we will always see an
all-ones column in 𝑯Ham. Therefore, all simplex codes satisfy
the SPC constraint, and no data preprocessing is needed.
Just replacing one layer check nodes of the (𝑎, 𝑏, 𝑝) QC-
LDPC code by simplex codes or first-order RM codes (𝑛𝐶,
𝑘𝐶), we get the CH-LDPC code. By puncturing a certain check
bits generated by component codes encoding, the code rate
loss of GLDPC codes can be further alleviated. If 𝑝𝐶bits
of each component code are punctured, the code rate of the
CH-LDPC code will be

V. NUMERICAL RESULTS
In this section, we demonstrate the decoding performance
of the CH-LDPC codes designed for helicopter-satellite chan-
nels via binary phase-shift keying (BPSK) modulation. The
maximum signal fading level during blockage period is set to
30dB. 𝐾bits represents the number of information bits.
The proposed Code 1 is a rate-1/4 CH-LDPC code with
𝐾bits = 843. The base code is a (5, 8, 281) irregular QC-
LDPC code, and the component code is the (16, 5) first-order
RM code with 6 check bits punctured for each component
code. Fig. 4 shows the decoding performance of Code 1 at
25% blockage, compared with performance over no blockage
channel, i.e. the AWGN channel. For 25% blockage, the
degradation of Code 1 at frame error rate (FER) of 10−4 is
about 1.49dB. The performance for a rate-1/4 and 𝐾bits = 900
QC-LDPC code constructed by [14] is also illustrated as
comparison. Fig. 4 shows that Code 1 outperforms the QC-
LDPC code by a fairly large margin over both channels.
The proposed Code 2 is a rate-2/7 CH-LDPC code with
𝐾bits = 122. The base code is a (3, 5, 61) regular QC-
LDPC code, and the component code is the (15, 4) simplex
code with 8 check bits punctured for each component code.
The decoding performance of Code 2 at no blockage, 10%
blockage and 25% blockage is shown in Fig. 5, compared
with the performance of a rate-2/9 and 𝐾bits = 200 QC-LDPC
code, whose parameters are equivalent to the LDPC code
adopted for periodical blockage channels in [5]. Furthermore,
the performance a rate-2/7 and 𝐾bits = 142 QC-LDPC code
constructed according to [6] is also illustrated as comparison.
For 10% and 25% blockage, the degradations of Code 2 at FER
of 10−4 are 0.59dB and 1.52dB respectively. Code 2 shows
better decoding performance than the rate-2/9 QC-LDPC code
over both AWGN channels and blockage channels. From an
engineering point of view, compared with the rate-2/9 QC-
LDPC FEC coding scheme, bandwidth efficiency improves
28.6% of our rate-2/7 CH-LDPC FEC coding scheme.

VI. CONCLUSION
Periodical signal fading arising from rotor blades blockage
is one of the most severe impairment over the helicopter-
satellite channels. We study a class of CH-LDPC codes to
mitigate such signal fading. The EXIT charts have been
presented as an engineering tool for the iterative decoding
analysis of the CH-LDPC codes. Simulation results show that,
our FEC coding scheme based on CH-LDPC codes achieves
28.6% higher bandwidth efficiency than that of QC-LDPC
codes according to the latest FEC solutions over the helicopter-
satellite channels.
