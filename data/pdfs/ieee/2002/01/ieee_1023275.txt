We derive an alternative recursion which, for regular codes, can be used to evaluate the block error probability using O ( n 3 ) operations, and the bit error probability using O(n4) operations. At the cost of additional operations, the recursion can also be applied t o evaluate the block error probability of leftirregular right-regular codes.
We use the recursion to obtain the block error probability of several codes with various block lengths. We also relate the block error probability to the code’s lowest left degree.
