I. INTRODUCTION
Error control coding has undergone rapid development in the previous ten years. Following the introduction of turbo codes in 1993 [I] and renewed interest in low-density parity check (LDPC) codes [2], [3], [4], [ 5 ] ,the advancement of error control coding climaxed with the use of irregular-LDPC codes [6], [71, [8], [9]. These systems, all based to some degree on random constructions, can be improved for practical implementations by employing deterministic matrix constructions as the basis for these codes. The a-rotation codes introduced in [IO] and [I I] provided a means to avoid random construction methods and build a deterministic form for (quasi-) regular-LDPC codes of arbitrary length and good performance. In this paper we extend the results in [IO] and [l I] to create multi-rate irregular-LDPC codes with correspondingly improved performance.

The H matrix not only defines the code but is also important in the decoding of LDPC codes. The decoding procedure uses the message passing algorithm [13], 1141, [15], 1161. Briefly,
the decoder is an iterative process that evolves to the codeword decision based on shared information from different portions of the received vector. The information sharing is based on the
pattem in the parity check matrix and thus H plays a key role in
the decoding process.

The n-rotation codes are derived from the semi-random parity check matrix introduced by Li Ping, W. K. Leung, and Nam Phamdo in [4], or can be thought of as an extended form of RA codes introduced by Dariush Divsalar, HUI lin and Robert J. McEliece in [12]. The irregular n-rotation codes extend the original s-rotation codes by using the simple encoding circuit in an iterative fashion to produce an irregular column weight in the underlying parity check matrix.
We begin this paper with a review of some basic features of a-rotation LDPC codes and a brief description of the a-rotation code encoding circuit. We then show how the n-rotation code can be extended and punctured to produced codes of various rates and irregular column weight without modifying the fundamentals of the encoding circuit. We then show how the original encoding circuit is used to produce irregular n-rotation codewords directly from the information bits. We conclude with results from a sampling of codes created using this method.

IX. CONCLUSIONS
In this paper we extend the r-rotation LDPC check codes to include irregular column weight in the parity check matrix. The
advantages of the n-rotation pattern - namely the three digit de-
scription of the permutation matrix and efficient encoding circuitry are maintained. The performance of these codes is very good for many configurations. The flexibility of this encoding scheme and good performance make this code attractive for implementation in many practical communication products.
