For difference set cyclic (DSC) codes, the shuffled 
DP decoding also presents similar advantages com- 
pared to the standard DP decoding, but the savings 
are not as imporiant as for LDPC codes. A possible 
reason is the fact that the parity check matrix of a 
DSC code has a higher density than that of an LDPC 
code. When decoded with the DP algorithm, the cor- 
relation between iterations for DSC codes is higher 
than that for LDPC codes. The sequential property 
of the sln~ffld BP increases these correlations and the 
gain when decoding a DSC code wit,]> the shuffled BP 
algorithm becomes smaller than for an LDPC code.
