Abstract - Asymptotic iterative decoding performance of low-density parity-check (LDPC) convolutional codes [l]is analyzed. An important role within the analysis plays a theorem on the minimal length of cycles in the Tanner graph describing a code.
Homogeneous LDPC convolutional codes, the convolutional counterpart of Gallager’s LDPC block codes, were introduced in [2] and further developed in [l]. The maximumlikelihood ensemble performance of this codes, including a lower bound on the free distance, was investigated in [3]. In this paper we study iterative decoding of LDPC convolutional codes.
The iterative limit [4][5],is a fundamental limit specifying when reliable transmission is possible with a particular code construction and iterative decoding. Lower bounds for iterative limits of Gallager’s LDPC block codes were derived in [4]. Here we prove that this results can be extended t o LDPC convolutional codes.
Our analysis consists of two steps. In the first step we prove the existence of LDPC convolutional ( J ,K)-codes for which the minimal length of a cycle in the Tanner graph, describing a code, is greater than 210, where 10 is proportional t o the logarithm of the memory of the LDPC convolutional code. Since the number of independent decoding iterations for such a code is not less than l o , we call this code lo-nondegenerated.

In the second step we analyze iterative decoding of Eonondegenerated LDPC convolutional codes when the number of iterations equals lo. To obtain lower bounds on the iterative limits, we use a two-phase decoding algorithm. In the first phase the original probabilistic Gallager algorithm or a simplified suboptimal algorithm is used. This phase is analyzed numerically. In the second decoding phase we use a simple majority decoding algorithm, which can be investigated analytically. This allows us t o carry out an asymptotic analysis and rigorously lowerbound the iterative limits like in [4].

It can be observed, that the required memory of an LDPC convolutional code is essentially smaller than the length of a corresponding LDPC block code achieving the same bit error probability.
The paper [l]also discusses a convolutional version of turbo codes, which is an example for nonhomogeneous LDPC convolutional codes. We prove that the results given in [5] can be extended to this convolutional version of turbo codes [6].
