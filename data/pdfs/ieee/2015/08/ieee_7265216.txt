Rate-Compatible LDPC Codes Based on the PEG Algorithm for Relay Communication Systems
Yangzhao Zhou, Xueqin Jiang, and Moon Ho Lee

Abstract: It is known that the progressive edge-growth (PEG) algorithm can be used to construct low-density parity-check (LDPC) codes at finite code lengths with large girths through the establishment of edges between variable and check nodes in an edge-byedge manner. In [1], the authors derived a class of LDPC codes for relay communication systems by extending the full-diversity rootLDPC code. However, the submatrices of the parity-check matrix H corresponding to this code were constructed separately; thus, the girth of H was not optimized. To solve this problem, this paper proposes a modified PEG algorithm for use in the design of large girth and full-diversity LDPC codes. Simulation results indicated that the LDPC codes constructed using the modified PEG algorithm exhibited a more favorable frame error rate performance than did codes proposed in [1] over block-fading channels.

I. INTRODUCTION
WHEN communicating over fading channels, the use of transmit diversity-that is, the transmission of signals carrying the same information over different paths in time, frequency, or space-can enhance frame error rate (FER) performance and improve power savings. Recently, a new network protocol called cooperative communication [2]–[6] has been used to take advantage of the broadcast nature of wireless transmission and enable transmit diversity by the use of a singleantenna device in a multi-user environment. The most elementary example of cooperative communication is the relay channel [7]–[9]. The relay channel can be generalized to a cooperative multiple access channel, where two users transmitting data to a single receiver cooperate by alternately serving as relays for each other [1], as shown in Fig. 1. In a repetition based protocol, such as decode-and-forward, each user sends the same data over two independent channels to the relay and destination. This protocol automatically makes full use of diversity offered by the relay channel.
In [10], a family of low-density parity-check codes (LDPC) called root-LDPC codes were proposed for non-ergodic block-

In [1], rate-compatible root (RCR)-LDPC codes were derived by extending the root-LDPC code to the block fading (BF) relay channel, which is rate-compatible. Because of its flexibility and favorable performance, RCR-LDPC codes are attractive for wireless relay communications that require slowly varying fading. However, the submatrices of the corresponding parity-check matrix H are constructed separately. Therefore, when the submatrices are combined into H, the girth of H is not optimal.
It is known that girth has a significant effect on the performance of LDPC codes [12]. For LDPC codes, the number of independent messages passed in SP decoding is proportional to the girth of the code [13], which means that a more favorable performance can be achieved when codes with larger girths are used. Consequently, a number of code construction methods have aimed to produce codes of large girth. Among the existing methods for the construction of large girth LDPC codes, one of the most successful approaches is the progressive-edge-growth (PEG) algorithm [14].
In this paper, to solve the girth problem of RCR-LDPC codes for relay communications, we imposed constraints on the PEG algorithm and introduce a modified PEG algorithm for use in the design of large-girth RCR-LDPC codes.

A. Progressive Edge-Growth Algorithm
The original PEG algorithm is introduced as follows. The variable node (VN) degree sequence Ds is defined as the set of column weights of the H designed; it is prescribed by the variable node degree distribution λ(x) as described in [16]. The PEG algorithm constructs H by progressively operating on variable nodes to place the edges required by Ds. The variable node of interest is labeled vj and the candidate check nodes are individually referred to as ci. The PEG algorithm chooses a check node, ci, to connect to the variable node of interest, vj, by expanding a sub-graph from vj up to a maximum depth, l. The set of check nodes found in this sub-graph is denoted by Nvlj ; the set of check nodes of interest, those not currently found in the subgraph, is denoted by N¯vlj . For the PEG algorithm, a check node is chosen at random from the minimum weight check nodes of
this set.

V. CONCLUSION
This paper’s contribution lies in its introduction of a modified PEG algorithm for the design of a large-girth LDPC codes for relay communication systems. A strategy that imposes constraints on the modified PEG algorithm was devised. The FER performance of codes generated using the proposed algorithm was shown to be superior to the performance of codes proposed in previous works [1]. In terms of SNR, compared with the LDPC codes in [1], regular and irregular LDPC codes of a rate of 1/3 constructed using the proposed algorithm both can saved up to 0.5 dB to achieve FER = 10−3. Further generalization of our algorithm from two users to more users is possible; we leave this issue open for our future research.
