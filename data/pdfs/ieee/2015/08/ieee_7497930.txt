Improved Decoding ofLDPC Codes by Variable-to-check Residual Belief Propagation

SONG Lingyan, HOU Shujuan
School ofInformation and Electronics Beijing Institute of Technology Beijing, P.R. China
e-mail:songlingyan9006@163.com. shujuanhou@bit.edu.cn

Abstract-This paper proposes an improved variable-to-check residual belief propagation algorithm, called Lazy-RBP, in which a new informed dynamic schedule (IDS) combined with the lazy schedule (LS) is adopted. Firstly, Lazy-RBP exploits the probabilistic girth-based approach, one of the LS strategies, to decide the set of updated variable nodes in each iteration. Then among this set, a new IDS scheme is introduced, where both the unreliable variable nodes and the residual of the variable-to-check messages are utilized to locate the message to be propagated first. Simulation results show that the LazyRBP algorithm can not only improve the error performance but accelerate the convergence compared with the VC-RBP algorithm.
Keywords-LDPC codes; lazy residual belief propagation (Lazy-REP); lazy schedule (LS); informed dynamic schedule (IDS)
I. INTRODUCTION
Low-density parity-check (LDPC) codes constitute a class of capacity-achieving error-correcting codes [1], which makes them popular channel coding solutions for modern wireless communication system. Belief-propagation (BP) is the dominant decoding algorithm for LDPC codes. The standard decoding schedules include the flooding schedule [2], in which all the variable nodes, and subsequently all the check nodes, pass new messages to their neighbors in each iteration. Beside the flooding schedule, several sequential schedules for iterative BP decoding, are proposed, such as shuffled schedule [3][4] and check-shuffled schedule [5][6] . It has been proved that all sequential schedules have the similar performance results and converge twice as fast as the flooding schedule at no cost [7]. Given the similarities, the sequential schedule considered for comparison in the simulation for this paper is named as the Layered BP (LBP) [3].
Recently, the informed dynamic schedules (IDS) named as the residual belief propagation (RBP) are employed for general sequential message passing [8] [9]. Then, two other dynamic schedules, called EDS-LBP [10] and VC-RBP [11] are raised. All these dynamic schedules require significantly

less iterations at the cost of the increased complexity, and yield better error performance than the standard sequential schedules. Since then another variant of the sequential schedule, referred to as the lazy schedule (LS), are introduced such as variable-node LS-LBP (VLS-LBP) [12], check-node LS-LBP (CLS-LBP) [13], and LS-LBP [14]. The LS strategy speeds up the BP convergence while keeping the extra complexity very low according to the simulation results [14].
However, the dynamic schedules stated above always face the difficulty in looking for a better sequence of message to be updated first. An appropriate updating ordering of message is very helpful to accelerate the convergence and/or achieve a good error performance. This paper proposes the Lazy Residual Belief Propagation (LazyRBP) algorithm, which combines the idea of IDS with the lazy schedule. Firstly, the proposed algorithm presents a new IDS scheme, which employs the residual of variable-tocheck message and unreliable variable node to locate the first updated variable-to-check message. The intuition behind the aided decision of unreliable variable node is that the unreliable variable node means its binary decision will be changed, which needs more information to correct itself. Therefore, updating the unreliable variable node with the largest residual will increase the convergence speed of the decoding algorithm. In addition, when the selected variableto-check message is indicated, the outgoing check-tovariable messages connected to the corresponding variable node should be generated and propagated first, for the sake of increasing the reliability of the selected variable-to-check message. Here, the lazy schedule of proposed algorithm exploits the probabilistic shuffled approach [15]. Simulations show that the Lazy-RBP decoding algorithm has faster convergence and better error performance than the flooding BP, the LBP and the VC-RBP.

III. THE PROPOSED ALGORITHM
For the proposed Lazy-REP algorithm, the new informed dynamic scheduling strategy utilizes the unreliable variable nodes, defined as ones with opposite sign of loglikelihood ratio (LLR) before and after an update. As the decoding algorithm converges, the LLR sign of the variable node will remain the same before and after an update. In other words, the codeword bit corresponding to the unreliable variable node tends to be in error in decoding decision. Thus, the outgoing message of the unreliable variable, with the largest variable-to-check residual, should be updated first to accelerate the convergence. If an unreliable variable node has the largest outgoing message residual, then the check-to-variable messages incoming to this unreliable variables also have the largest residual.

IV. SIMULATION RESULTS
In order to evaluate the performance of the proposed Lazy-RBP algorithm, we consider the CCSDS (2048,1024) LDPC code [16]. The error and convergence performance of the flooding BP, the LBP, the VC-RBP and the Lazy-RBP algorithms over AWGN channel using BPSK modulation and 5 quantization bits are demonstrated in this section.
Fig. 1 and Fig. 2 show the BER and FER performance at different signal-to-noise ratios (SNR) of these four algorithms at most 10 iterations, respectively. It can be seen that the Lazy-RBP outperforms the other three decoding algorithms both in BER and FER performance for a small number of iterations, and particularly achieves 0.16dB coding gain compared with VC-RBP when BER equals 10-6. This can be explained that with the help of unreliable variable node and the pre-computations of check-to-variable messages, the error bits are more likely to be located and then be corrected.
Fig. 3 and Fig. 4 show BER and FER performance vs. different number of iterations at Eb / No = 1.2 dB for these
algorithms, respectively. The figures show the Lazy-RBP

converges faster than other algorithms for about 25 iterations and have better performance than that of VC-RBP for a large nwnber of iterations. This advantage attributes to co-functioning of the new IDS scheme and the probabilistic lazy schedule.

V. CONCLUSIONS
This paper proposes an improved variable-to-check RBP for LDPC codes, called Lazy-RBP algorithm. Lazy-RBP algorithm combines the ideas ofLS with IDS strategy, where LS utilizes the probabilistic variable-shuffled approach. The new IDS strategy adopts unreliable variable as aided decision to locate the error variable nodes and pre-computes the corresponding check-to-variable messages to make the selected variable-to-check message more reliable.
