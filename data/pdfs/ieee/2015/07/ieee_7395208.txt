Abstract-In this paper, an adaptive decoding algorithm for 
majority-logic decodable LDPC codes is present based on a 
recent work by Chen et al. 
Different from the 
RBI-MSD 
algorithm, 
the 
presented 
adaptive 
reliability-based 
iteration 
min-sum decoding algorithm does not need to predesignate the 
scaling factor, which is optimized by discretized density evolution. 
In the adaptive RBIMSD algorithm, an adaptive scaling factor is 
induced for each check node to modify the extrinsic message. The 
adaptive scaling factor is computed by the ratio of the second 
minimum and the maximum value among the message, which is 
received by the check nodes. Simulation results show that, the 
performance of the adaptive RBI-MSD algorithm is almost as 
well as that of the original RBI-MSD algorithm but with a little 
(or negligible) computation increase.

I. 
INTRODUCTION 
Low-density parity-check (LDPC) codes were originally 
proposed by Gallager in 1962 [1], but they had been 
unnoticed for a long time due to the hardware of the time 
could not support effective decoder implementations. In the 
late 1990's, they were rediscovered beginning with the 
work of MacKay [2][3] and enhanced to irregular LDPC 
codes by Richardson et al: [4] [5] in 2001. Currently, LDPC 
codes have been known as a class of Shannon capacity 
approaching codes. 
Generally, LDPC codes can be classified into two 
categories depending on their construction methods: the 
random-like LDPC codes [3][5][6] and the structured 
LDPC codes [7][8]. The random-like LDPC codes are 
usually generated by computer searching to guarantee their 
structural properties such as girth, degree distributions, 
stopping sets and hence. The structured LDPC codes which 
generated with algebraic and combinatorial tools, however, 
may have more advantages in encoding complexity and 
integrated circuits(IC) decoder implementation due to their 
cyclic symmetry properties [9]. 
LDPC codes can be decoded by several well known 
decoding algorithms including: 1) soft-decision decoding; 
2) hard-decision decoding; and 3) reliability-based hybrid 
decoding 
schemes. 
Among 
these, 
the 
sum-product 
algorithm 
(SPA) 
[10][11] 
is 
the 
most 
well-known 
soft-decision iterative decoding which provides the best 
performances. The Well-known hard decision decoding 
algorithms such as the one-step majority logic-decoding 
(OSMLGD) [7][12] and the bit-flipping(BF) algorithms [1] 
have the lowest decoding complexity but all suffer 
performance 
degradation 
at 
different 
levels. 
The 
reliability-based hybrid decoding schemes are possible to 
make efficient tradeoffs between coding performance and 
computational complexity, such as Weighted Bit Flipping 
(WBF) decoding algorithm and its improved versions 
[7][13], 
the 
iterative 
reliability-based 
majority-logic 
decoding(IRB-MLGD) 
algorithm 
[14] 
and 
the 
reliability-based iterative min-sum decoding (RBI-MSD) 
algorithm [15], etc. Reliability-based decoding algorithms 
for nonbinary LDPC codes have also been investigated, see 
[16] [17] and references therein. 
Recently, an adaptive decoding algorithm for LDPC 
codes has been proposed in [18]. There, all the check nodes 
are classified into active nodes and silent nodes according 
to certain criteria, and only active check nodes are involved 
in the iterative procession. Simulation results show that this 
adaptive decoding algorithm perform well for structured 
majority-logic decodable LDPC codes. 
In this paper, another adaptive strategy for structured 
majority-logic decodable LDPC codes is present based on the 
RBT-MSD algorithm. Different from the original RBT-MSD 
algorithm, the presented adaptive reliability-based iteration 
min-sum decoding algorithm does not need to predesignate the 
scaling factor, which is optimized by discretized density 
evolution. In the adaptive RBI-MSD algorithm, an adaptive 
scaling factor is induced for each check node to modify the 
extnnsiC 
message. 
Simulation 
results 
show 
that, 
the 
performance of the adaptive RBI-MSD algorithm is almost as 
well as that of the original RBI-MSD algorithm but with a 
little (or negligible) computation increase.

CONCLUSION 
Tn this paper, we have presented an adaptive RBT-MSD 
algorithm for majority-logic decodable LDPC codes based on a 
recent work by Chen et al. Compared with the original 
RBI-MSD algorithm, an adaptive scaling factor is induced in 
the adaptive RBT-MSD algorithm for each check nodes to 
modify the extrinsic message. The adaptive scaling factor is 
computed by the ratio of the second minimum and the 
maximum value among the message, which is received by the 
check nodes. Simulation results show that, the performance of 
the adaptive RBI-MSD algorithm is almost as well as that of 
the original RBT-MSD algorithm but with a little (or negligible) 
computation increase.
