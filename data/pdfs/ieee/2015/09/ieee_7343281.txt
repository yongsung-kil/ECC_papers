Recursive Encoding Of Spatially Coupled LDPC Codes With Arbitrary Rates

Junyang Ma, Zhongwei Si, Zhiqiang He, and Kai Niu Key Lab of Universal Wireless Communications, Ministry of Education Beijing University of Posts and Telecommunications(BUPT), Beijing, China

Abstract—Spatially coupled LDPC codes have attracted much attention due to the promising performance. Recursive encoding with low delay and low complexity has been proposed in the literature for selected node degrees. To realize the recursive encoding of spatially coupled LDPC codes with arbitrary rates, we propose in this paper a modified structure of the parity-check matrix and implement the encoding using a shift-register regardless of the node degrees. By rearranging the edge connections, the parity bits at each coupling position can be jointly determined by the information bits at the current position and the encoded bits at former positions. Performance analysis in terms of design rate and density evolution has been provided. It can be observed that the modified code structure leads to a better belief-propagation threshold. Finite-length simulation results are provided, which verify the theoretical analysis.
I. INTRODUCTION
Low-density parity-check (LDPC) codes [1] were the most researched channel coding in the last decade due to the good performance and the reasonable complexity. Their convolutional counterparts, the LDPC convolutional codes, were invented in [2] and further developed in, e.g., [3], [4]. Different constructions and analytical results have been provided in [3]– [7]. For example, [3] proposed a realization of LDPC convolutional codes and conjectured the capacity-achieving performance under belief-prorogation (BP) decoding. The codes were generalized to spatially coupled LDPC codes in [4], and the capacity-achieving property of the code family has been proved theoretically [8], [9]. The good features triggered the application of spatially coupling in many scenarios [10]–[13].
To realize the promising performance in practice, the implementation of spatially coupled LDPC codes is worth investigation. The most common implementation of spatially coupled LDPC codes is the one introduced in [6]. The paritycheck matrix of the spatially coupled LDPC code is made up of randomly permuted identity matrices. Based on the parity-check matrix, a recursive encoder using a shift-register was proposed. The encoder requires only a small number of memory units and the computational cost per bit is proportional to the check node degree. This is a clear advantage compared with the encoding of LDPC block code regarding storage requirements and computational costs. Combined with the pipeline decoding or the sliding-window decoding [6], [14], spatially coupled LDPC codes provide a low-delay and lowcomplexity realization of channel coding.
The recursive encoder in [6] only applies to the case that the ratio of the check node degree dc and the variable node degree dv is an integer. To realize the recursive encoding of

spatially coupled LDPC codes with arbitrary rates, we propose in this paper a modified structure of the parity-check matrix and implement the encoding using a shift-register regardless of the choice of dc and dv. By rearranging the edge connections, all the parity bits at each coupling position can be jointly determined by the information bits at the current position and the encoded bits at former positions. The requirement of memory units and the computational complexity retain similar to the recursive encoder in [6]. Performance analysis in terms of design rate and density evolution is provided. It can be observed that the modified code structure converges faster in the BP decoding than the existing structure. Consequently, the modified spatially coupled LDPC codes in general present better BP thresholds and smaller gaps to the Shannon limits. Bit erasure rate (BER) performance with finite code lengths is provided as well, which verifies the analysis of the BP thresholds.
The remainder of the paper is organized as follows. An overview of spatially coupled LDPC codes and the recursive encoding proposed in [6] is given in Section II. In Section III, we propose the modified structure of spatially coupled LDPC codes and illustrate the recursive encoding for the case that dc/dv is not an integer. Performance analysis based on design rate and density evolution is provided in Section IV. Section V presents the finite-length performance evaluation in the binary erasure channel (BEC). Section VI concludes the paper.

III. RECURSIVE ENCODING OF SPATIALLY COUPLED LDPC CODES WITH ARBITRARY RATES
In this section, we will give a detailed introduction of spatially coupled LDPC codes with the modified structure and realize the recursive encoding accordingly.

A. Spatially Coupled LDPC Code With The Modified Structure
To illustrate the modified structure of the spatially coupled LDPC code with arbitrary rate, we first consider the original spatially coupled LDPC code [6] for the case that dc/dv is not an integer. As an example, the protograph of a (dv, dc, L) = (4, 6, L) spatially coupled LDPC ensemble is provided in Fig.2(a). The notations are the same as in Fig.1. Without loss of generality, we assume M = 1. The protograph consists of dc variable nodes and dv check nodes at each position. The variable node degree equals dv, and the check node degree is dc. At position i in Fig.2(a), vi,0 denotes the information bits while vi,1 and vi,2 represent the parity bits. It can be found out that vi,1 can not be calculated only depending on vi,0 and the encoded bits at former positions, as the two variable nodes associated with vi,1 and vi,2 connect to the same check node. Therefore, the recursive encoder proposed in [6] cannot be used for this case.
In order to realize the recursive encoding, we modify the structure of the protograph and illustrate it in Fig.2(b). At position i, vi,0 represents the information bits, and vi,1, vi,2 are the parity bits. It is easy to find out that vi,1 can be calculated depending on vi,0 and the encoded bits at former positions while vi,2 can be obtained through vi,0, vi,1 and the encoded bits at former positions. It can be concluded that, after the modification, the parity bits can be calculated depending on the information bits and the encoded parity bits, and the code can be encoded recursively. We have to emphasize that the modified structure shown above is not unique for recursive encoding the (dv, dc, L) spatially coupled LDPC code. Other structures are not included in this paper due to the limited space.
The corresponding parity-check matrices of (dv, dc, L) = (4, 6, L) spatially coupled LDPC codes are illustrated in Fig.3. Both matrices are composed of submatrices which are indicated by the dashed ellipses. For the original structure in Fig.3(a), each submatrix has dc = 3 rows of permutation matrices and dvms = 4 columns of permutation matrices. V. SIMULATION RESULTS
In this section, we evaluate the performance of the proposed structure numerically. We compare the BP thresholds of the spatially coupled LDPC code after the modification with the BP thresholds of the corresponding original structure in [6]. In addition, the bit erasure rates for both two structures over the BEC are provided.
The BP thresholds of the modified spatially coupled LDPC code and the original implementation in [6] over the BEC are shown in Table I, which are represented by εBP and εBP , respectively. Besides, εSh denotes the Shannon limit corresponding to the design rate. Here, we set the coupling number L = 50 for all the ensembles and vary node degrees. Comparing εBP with εBP , we can find that the BP thresholds get improved after the modification. This verifies the density evolution in the previous section. It also can be seen that, for the modified spatially couple LDPC codes, the gaps between the BP thresholds and the Shannon limits are in general very small.
Bit erasure rates for the spatially coupled LDPC codes with the modified structure and the original structure over BECs are provided in Fig.5. Here, we set the lifting number M = 500 and the coupling number L = 50. We also include the BP thresholds as the dashed lines and the Shannon limits as the dotted lines. Comparing the BER curves, we can find that the spatially coupled LDPC codes with the modified structure clearly outperforms the original spatially coupled LDPC code in [6]. Note that, in addition to the good performance, the proposed structure allows for recursive encoding which facilitates the application of spatially coupled LDPC codes in practice.
VI. CONCLUSION
In this paper, a modified structure for the spatially coupled LDPC code has been proposed, which enables the recursive encoding using a shift-register for arbitrary rate. The algorithm significantly reduces the computational complexity and memory requirement for the encoding. Performance analysis in terms of design rate and density evolution has been provided. The belief-prorogation decoding threshold of the modified spatially coupled LDPC code is improved due to the edge rearrangement. Finite-length simulation results verify the theoretical analysis.
