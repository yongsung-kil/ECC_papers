Reliability-Based Iterative Proportionality-logic Decoding of LDPC Codes with Adaptive Decision
Youming Sun, Haiqiang Chen, Xiangcheng Li, Lingshan Luo, and Tuanfa Qin

Abstract: In this paper, we present a reliability-based iterative proportionality-logic decoding algorithm for two classes of structured low-density parity-check (LDPC) codes. The main contributions of this paper include: 1) Syndrome messages instead of extrinsic messages are processed and exchanged between variable nodes and check nodes, which can reduce the decoding complexity; 2) a more flexible decision mechanism is developed in which the decision threshold can be self-adjusted during the iterative process. Such decision mechanism is particularly effective for decoding the majority-logic decodable codes; 3) only part of the variable nodes satisfying the pre-designed criterion are involved for the presented algorithm, which is in the proportionality-logic sense and can further reduce the computational complexity. Simulation results show that, when combined with factor correction techniques and appropriate proportionality parameter, the presented algorithm performs well and can achieve fast decoding convergence rate while maintaining relative low decoding complexity, especially for small quantized levels (3–4 bits). The presented algorithm provides a candidate for those application scenarios where the memory load and the energy consumption are extremely constrained.
Index Terms: Adaptive decision, decoding function, low-density parity-check (LDPC) codes, proportionality-logic, reliability-based decoding.
I. INTRODUCTION
RECENTLY, the reliability-based decoding algorithms have attracted considerable attention since they can make efficient trade-offs between decoding performance and computational complexity. The well-known reliability-based algorithms are the weighted one-step majority-logic decoding (WOSMLGD) algorithm [1], the weighted bit flipping (WBF) algorithm [1] and their improved versions [2]–[5].

It is shown that these reliability measures can be ameliorated by the iterative updating strategy (rather than simple flipping operation) to achieve better performance. Relevant works are the reliabilitybased iterative majority-logic decoding (RBI-MLGD) algorithm and the weighted iterative soft-reliability based majority-logic decoding algorithm proposed in [6] and [7]. These two algorithms are both binary message-passing reliability-based algorithms and require only logical operations and integer additions, resulting significantly low decoding complexity while maintaining excellent performance. Ngatched et al. presented an improvement of the RBI-MLGD algorithm by introducing the reliability measure for each check-sum [8]. In [9], we extended the RBI-MLGD algorithm to integer message-passing and presented the reliability-based min-sum decoding (RBI-MSD) algorithm, which achieves better performance but suffering more computational complexity. To further reduce the complexity, a new approach is presented in [10], where the check nodes are classified into active nodes and silent nodes and only active check nodes are involved in the iterative procession. Reliabilitybased decoding algorithms for nonbinary low-density paritycheck (LDPC) codes have also been investigated, (see [11] and [12]) and references therein.
Motivated by these work, we present a reliability-based iterative proportionality-logic decoding algorithm in this paper. The main contributions of this paper include: 1) At check nodes, syndrome messages instead of extrinsic messages are calculated and passed back to variable nodes; while at variable nodes, the syndrome messages are directly involved in the voting process. This is different from the existing reliability-based algorithms, such as the modified RBI-MLFD (MRBI-MLGD) algorithm and the RBI-MSD algorithm and can reduce the computational complexity. 2) An more flexible decision mechanism is developed in which the decision threshold can be self-adjusted during the iterative process. The developed decision mechanism can be viewed as the generalized case of the existing reliability-based decoding algorithms and is more reasonable when decoding the majority-logic decodable codes with large column weight. 3) Unlike the RBI-MLGD and RBI-MSD algorithms where all the variable nodes are required to be processed at each iteration, only part of the variable nodes satisfying the pre-designed criterion are involved for the presented algorithm, which is in the proportionality-logic sense. This can further reduce the computational complexity. Simulation results show that, when combined with factor correction techniques and appropriate proportionality parameter, the presented algorithm can achieve good bit error rate (BER) performance and fast decoding convergence rate while maintaining low decoding complexity. II. BASIC CONCEPTS, NOTATION AND SYSTEM MODEL
A. Basic Concepts and Notation
An LDPC code C can be defined as the null space of a sparse parity-check matrix H = (hi,j)m×n. Associated with the parity-check matrix is the Tanner graph [13]. A Tanner graph is a bipartite graph which contains two types of node, the variable node Vj (0 ≤ j ≤ n−1) and the check node Ci (0 ≤ i ≤ m−1). The variable node corresponds to a column of H and the check node corresponds to a row of H. For any parity-check matrix, we define the following two sets
Ni = {j : 0 ≤ j ≤ n − 1, hi,j = 1}
for each row i of H and
Mj = {i : 0 ≤ i ≤ m − 1, hi,j = 1}
for each column j of H. We focus on two classes of binary structured (γ, ρ) regular
LDPC codes, which are constructed based on the properties of finite geometry [1], [14] and finite fields [15]. Their parity-check matrices have the following structures: 1) Each row has constant weight ρ; 2) each column has constant weight γ; and 3) no two rows (or two columns) can have more than one position where they both have 1-components. The third property is referred to as RC-constraint, which guarantees that the code can be represented by a Tanner graph [13] with girth of at least 6.
B. System Model
Encoding: Let u = (u0, u1, · · ·, uk−1) ∈ F2k be an information sequence to be transmitted. The information sequence u is encoded by a binary LDPC encoder into a codeword c = (c0, c1, · · ·, cn−1) ∈ F2n;
Mapping: The simple BPSK modulation is employed in this paper. That is, the codeword c is transformed into a bipolar signal vector x = (x0, x1, · · ·, xn−1), where xj = 2cj − 1.
Channel: Assume that the signal vector x is transmitted through an additive white Gaussian noise (AWGN) channel. The received vector is denoted by y = (y0, y1, · · ·, yn−1), where yj = xj + nj and nj is a white Gaussian noise sample.
Quantizing/Decoding: The receiver attempts to recover the transmitted data u based on the received signal y. For some

application scenarios (such as to guarantee the integer operation in the iterative decoding process), the real received symbols may require to be quantized into integer values before passing to the decoder.

B. Adaptive Decision Mechanism
For the OSMLGD algorithm described above, the received bit is assumed to be erroneous if the number of unsatisfied checksums orthogonal on this bit is greater than a threshold, say, fifty percent of the column weight. This threshold depends on the code parameters ρ, γ, and the signal-to-noise ratio (SNR). Actually, the threshold can be chosen to optimize the error performance while minimizing the computational complexity [1]. In this paper, we present an adaptive decision mechanism in which the decision threshold can be self-adjusted with the iterations. Such mechanism can make the decoding decision more reliable/conservative in the iterative process and achieves performance gain.

A. Complexity Analysis

In this section, we analyze the complexity of the proposed algorithm. Let δ(= nγ = mρ ) denote the number of nonzero elements in H. For the presented algorithm, at each iteration, Step 1 requires n tests of the sign of the updated reliability mea-
sures associated with the received vector. Testing the sign of an integer is actually a logical operation; Step 2 requires δ − m
XOR operations (or modulo-2 additions) to compute the syndrome of a received vector; Step 3 requires δ(ρ − 2) integer comparisons to find the least reliable (extrinsic) messages at the check nodes; Step 4 requires δ integer additions to compute the
decoding function; Step 5 includes the following operations: 1) n integer additions and n real multiplications to compute the decision threshold η; 2) n integer comparisons to compare the decoding function with the decision threshold; 3) n′ real multiplications to scale the decoding function, n′ binary operations to obtain the signs of the updated reliability measures and n′ integer additions to update the reliability measures. B. Simulation Results
1) Selection of the parameter λ: As mentioned above, the parameter λ is of importance for the presented algorithm, which may affect the BER performance and the computational complexity. We take the (961, 721) regular QC-LDPC code as an example to discuss the selection of λ by simulations. Fig. 1 and Fig. 2 show the effect of λ on the BER performance and the average number of iterations, respectively.
It can be seen that the performance is not very sensitive when λ is lower than a certain value λ∗ (obviously, λ∗ is about 0.35 for this example). When λ is larger than λ∗, we observe an evident degradation of the BER performance and a more faster increase of the iterations. Therefore, the selection of the parameter λ is required to be not larger than λ∗. Besides, for the presented algorithm, lower λ indicates that more variable nodes are involved to be processed and thus leads to higher computational complexity. Based on the discussions above, it is reasonable to set λ = λ∗ = 0.35. The same observations can be made for the (4095, 3367) cyclic Euclidean geometry (EG)-LDPC code.

VI. CONCLUSION
We have presented a reliability-based iterative proportionality logic decoding algorithm in this paper. Different form the conventional (W)OSMLGD decoding algorithms, the presented algorithm performs the updating rather than flipping operation at each iteration. We have also developed an adaptive decision mechanism in which the decision threshold varies and is self-

adjustable during the iterative process. This enables the decoding decision to be more conservative. With a well-designed proportionality parameter, only part of the variable nodes are required to be processed at each iteration, which is different from most of the existing reliability-based decoding algorithms and can further reduce the decoding complexity. The main computations for the presented algorithm include only binary logical operations and integer additions. Simulation results show that, when combined with factor correction techniques and appropriate proportionality parameter, the presented algorithm can achieve good BER performance and fast decoding convergence rate while maintaining relative low decoding complexity. In particular, the presented algorithm can even outperform the RBIMSD algorithm for low quantized levels (3–4 bits). The presented algorithm may find application in those scenarios where the memory load and energy consumption are extremely constrained.
