Improved ADMM Penalized Decoder for Irregular Low-Density Parity-Check Codes
Xiaopeng Jiao, Haoyuan Wei, Jianjun Mu, and Chao Chen

Abstract—Based on linear programming (LP) decoding with alternating direction method of multipliers (ADMM), Liu et al. proposed an ADMM penalized decoder for low-density parity-check (LDPC) codes that can improve the error rate performance at low signal-to-noise ratios. In this letter, we propose a new ADMM penalized decoder to improve the error rate performance further for irregular LDPC codes. The proposed decoder modifies the penalty term of the objective function by assigning different penalty parameters for different variable node degrees. Simulations over three irregular LDPC codes show that the proposed decoder performs considerably better than the ADMM penalized decoder.
Index Terms—Alternating direction method of multipliers (ADMM), linear programming (LP) decoding, penalized ADMM decoder.
I. INTRODUCTION
L INEAR PROGRAMMING (LP) decoding is an alternative decoding method for low-density parity-check (LDPC) codes. A major research problem for LP decoding is how to reduce its computational complexity. Recently, Barman et al. proposed an efficient algorithm for LP decoding based on the alternating direction method of multipliers (ADMM) [1]. In [1], the authors presented a “two-slice” characterization for the parity polytope. Using this characterization, they developed an efficient Euclidean projection algorithm which is required for the ADMM-based LP decoding. Based on the cut search algorithm introduced in [2], Zhang et al. proposed a reduced-complexity algorithm for the Euclidean projection operation [3]. Later, the Euclidean projection operation is simplified further by using an efficient projection algorithm onto the simplex [4]. It should also be noted that the ADMM-based LP decoder can be interpreted as a message passing algorithm that passes messages between variable nodes and check nodes on a Tanner graph [1].
Though the computational complexity of ADMM-based LP decoding is low, simulation results in [1] reveal that the error rate performances of the simulated LDPC codes are worse than that of the belief propagation (BP) decoder at low signal-tonoise ratios (SNRs). To address this problem, Liu et al. proposed an improved ADMM decoder by introducing a penalty term to the objective function of the LP decoding problem,

called ADMM penalized decoder [5]–[7]. The goal of the penalty term is to make pseudocodewords more costly. For the ADMM penalized decoder, only one penalty parameter is used for all the variable nodes involved in the penalty term. However, for irregular LDPC codes, using only one penalty parameter may not be enough since the number of messages received from neighbor check nodes are different for variable nodes with different degrees. The value of variable node can be decided more reliable if more messages are received. Therefore, we should differentiate variable nodes with different degrees. This motivates us to introduce more penalty parameters into the penalty term for variable nodes with different degrees.
In this letter, we propose an improved ADMM penalized decoder that modifies the penalty term by using a number of penalty parameters. Since the optimization of these penalty parameters in the continuous real space is a difficult task, we use a heuristic search method called differential evolution (DE) [8] to optimize the penalty parameters. Simulation results show that the proposed decoder performs considerably better than the ADMM penalized decoder.
II. PRELIMINARIES
Consider an LDPC code C with a parity check matrix H. Its Tanner graph has n variable nodes indexed by I = {1, 2, . . . , n} and m check nodes indexed by J = {1, 2, . . . , m}. The element Hji = 1 in H if there is an edge between variable node vi and check node cj. Otherwise, Hji = 0. Let the degree of variable node vi be dvi which represents the number of 1 s in the ith column of H. Similarly, let the degree of check node cj be dcj . Let Nv(i) = {j ∈ J : Hji = 1} denote the index set of neighbors of variable node vi, and similarly let Nc(j) = {i ∈ I : Hji = 1} denote the index set of neighbors of check node cj. If all variable nodes have degree dv and all check nodes have degree dc, then we call this LDPC code a (dv, dc) regular LDPC code. Otherwise, an irregular LDPC code is obtained.

From above examples we can see that the FER performances for the ADMM penalized decoder are sensitive to the choice of penalty functions. However, for the improved ADMM penalized decoder, the FER performances for the l1 and l2 penalty functions are similar. It can also be seen that I-ADMM-PD suffers from an error floor in high SNRs. Two questions may arise from the simulations: (1) Is the error floor related to the penalty coefficients which are optimized at the waterfall region? and (2) How to choose penalty coefficients in high SNR regimes? Based on instanton analysis, Liu et al. investigate the relationship between the instanton norm and the penalty coefficient for the ADMM penalized decoder [7]. It provides a method to choose penalty coefficient in high SNR regimes. For the improved ADMM penalized decoder, a number of penalty parameters are introduced and this complicates the choice of penalty coefficients in high SNR regimes. Thus we leave it as a future research problem.

V. CONCLUSION
To summarize, we have proposed an improved ADMM penalized decoder for irregular LDPC codes by introducing several penalty parameters to the penalty term of the optimization problem. These penalty parameters are optimized by the DE method. Simulation results show that the proposed decoder performs considerably better than the original ADMM penalized decoder.
