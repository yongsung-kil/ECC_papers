Novel Reliability Evaluation Method for NAND Flash Memory

Jiang Xiao-bo, Tan Xue-qing, Huang Wei-pei
School of Electronic and Information South China University of Technology
Guangzhou, China

Abstract—The evaluation of error correction code (ECC) for NAND flash memory is increasingly complicated by the increasing bit error rate in memory. The concept of error-free information capacity is proposed to evaluate the performance ECC of NAND flash memory. The new method simultaneously considers the capacity and reliability of NAND flash memory. Low-density parity-check (LDPC) codes with a medium code rate can improve the integrated performance of NAND flash memory in order of magnitudes. Observations provide guides for the development of ECC schemes in NAND flash memory in future.
Keywords—NAND flash memory; Error-free information capacity; ECC structure;
I. INTRODUCTION
NAND flash memory has been widely used as a storage medium for digital systems, such as mobile phones, pads, solid-state drives, and cloud storages. The market of NAND flash memory increases yearly. However, the increasing bit error rate (BER) limits the development of NAND flash memory. As the storage density increases, BER in NAND flash has become increasingly serious [1], [2]. The program/erase (P/E) cycles of NAND flash memory have decreased from 100,000 to several hundreds, which has greatly affected the development of NAND flash memory.
BER of NAND flash memory has attracted considerable research and experienced a rapid improvement. The development of error correction code (ECC) in NAND flash memory considers capacity first and then performance. Singleerror detection/correction codes, such as Hamming codes, used to be sufficient to enhance the reliability of single-level cell NAND flash memory systems [3]. As technology is scaling and given the multi-level cell (MLC) technology, the singleerror correction capacity of Hamming code is no longer sufficient. Long linear block codes with high error correction capacities are then used in NAND flash memory, such as Bose–Chaudhuri–Hocquenghem (BCH) and Reed–Solomon codes [4]-[5]. Several ECC schemes based on concatenation of BCH codes are recently proposed [6]-[8]. Owing to near Shannon capacity performance in the presence of soft information, low-density parity-check (LDPC) codes have received great attention. Compared with traditional BCH codes,

LDPC codes can obtain a more significant coding gain[9]-[11]. LDPC codes applied to NAND flash memory have been proposed [9], [10].
Given that they consider capacity first, ECCs, including Hamming, BCH, and LDPC codes, are based on correction codes with a code rate higher than 0.9 in NAND flash memory. ECCs based on BCH codes with a code rate higher than 0.9 have been used in several commercial MLC NAND flash memories. Low complex rate-0.929 concatenated BCH codes have been presented to use in MLC NAND flash memories [7], [8]. Rate-0.9 and rate-0.95 quasi-cyclic-LDPC codes are proposed in [12]. References [13] and [14] presented a rate0.96 Euclidean geometry-LDPC code and its very-large-scale integration for high-throughput NAND flash memory systems.
However, with increasing BER in NAND flash memory, studies have focused on the reliability of NAND flash memory. Studies on novel coding themes, such as write-once memory (WOM)-codes and rank-modulation [15]-[22], have sacrificed NAND flash memory storage capacity and generate considerable P/E cycles in return. WOM-codes, which are designed for memory devices with irreversible state transition [15], can be used to write information multiple times before an erase operation is required. WOM-codes have been proposed for flash memory [16], and coding themes based on polar codes or coset codes have been presented [18]-[19]. Jiang et al. [20] proposed the application of rank-modulation scheme for flash memories. The main advantage of the scheme is that erasing cells is not required because any permutation can be reached from any other through adding charge to an appropriate subset of cells. Rank-modulation schemes have attracted considerable research [21], [22]. WOM-codes and rank-modulation are both coding themes that achieve multiple writes before an erase operation. These two coding themes sacrifice capacity in each write. They sacrifice NAND flash memory storage capacity and obtain numerous P/E cycles in return. Endurance (P/E cycles) is no longer enough to evaluate the coding theme in NAND Flash, because it doesn’t considerate the varying storage capacity in flash.
For NAND flash memory at a high error rate, our study focuses on the contradiction between capacity and reliability. The concept of error-free information capacity (EIC) is

II. EIC
The key indicators of NAND flash are capacity and reliability (P/E cycles). When BER is small, capacity is the main factor requiring consideration. However, as BER increases and P/E cycles decrease to several hundreds, reliability becomes more important than capacity. Hence, capacity is sacrificed, and reliability is achieved in return. Both capacity and reliability should be considered in designing an ECC in NAND flash memory. The concept of error-free information capacity (EIC), which is an evaluation method that simultaneously considers reliability and capacity, is proposed. EIC is used to evaluate whether a coding theme can improve both capacity and reliability in NAND Flash.

This difference is due to the fact that the proposed EIC considers reliability and stable information capacity in NAND flash memory. EIC is determined by P/E cycles and code rate. In the two channel models, flash can obtain the most P/E cycles at code rate-1/2. However, as code rate decreases, information written in NAND flash memory decreases as well. A low code rate therefore leads to high P/E cycles, but EIC is not high. With the same kind of ECC, the highest EIC is obtained in different code rates. This result is associated with the channel models, and it is realistic. Considering that some systems require reliability and long-term stability of capacity, EIC can be used to evaluate ECC in NAND flash memory. EIC analyzed by different forward error correction architectures can obtain a high overall performance of flash memory with a low code rate. This finding breaks the understanding of using a highrate ECC in NAND flash memory. This result is also consistent with many flash applications. Compared with storage systems with high initial but continuously declining capacity, the long-term capacity of stable storage systems are a better choice.
IV. CONCLUSION
We propose the concept of EIC to evaluate the overall performance of NAND flash memory of ECC. Analysis of the EIC of different ECCs in NAND flash memories indicates that it can achieve integrated performance of both NAND flash capacity and reliability after decreasing the code rate. EIC provides guides for improving NAND flash ECC in future.
In this paper, we present an adaptive ECC structure based on LDPC codes. The structure can achieve higher EIC than current ECC schemes in order of magnitudes.
