Performance of the Modified SRBI-MLGD Algorithm for LDPC Codes in MLC NAND
Flash Memory

Jiaying Wen, Guojun Han, Xusheng Lin and Yi Fang
School of Information Engineering Guangdong University of Technology
Guangzhou, China, 510006 Email: gjhan@gdut.edu.cn

Abstract—With continued scaling down of the NAND flash memory, the multi-level cell (MLC) technique emerges to dramatically increase the storage density. Meanwhile, this technique also leads to worse error performance owing to the cell-to-cell interference (CCI). Recently, low-density parity-check (LDPC) code has appeared to be a promising solution to combat the performance degradation of NAND flash memory. Although LDPC codes can provide stronger error performance, the conventional belief propagation (BP) decoding algorithm requires calculating soft-decision log-likelihood (LLR) information and hence possesses a massive computational complexity. In this paper, by exploiting the characteristics of MLC NAND flash memory channels, we present a modified soft reliability-based iterative majority-logic decoding (SRBI-MLGD) algorithm for the LDPC codes over MLC NAND flash memory channels. The modified decoding algorithm has less computational complexity as compared with the sum-product decoding algorithm (SPA). Moreover, the simulation results show that the decoding algorithm offers an effective trade-off between complexity and performance over high-column-weight LDPC coded MLC NAND flash memory channels.
Index Terms—cell-to-cell interference (CCI), low-density parity-check (LDPC) code, NAND flash memory, majority-logic decoding, soft reliability-based decoding.
I. INTRODUCTION
FROM consumer electronics to personal and enterprise computers, NAND flash memory has been widely applied in a myriad of diverse applications because of high access speed, low power consumption and large capacity. The multi-level cell (MLC) technology can store more than 1 bit in each memory cell by pulling up the voltage into one of several voltage windows when programming. Due to the high density of storage, MLC flash memory has seized a growing share of global memory market.
To ensure the integrity of data that stored with many noise

Therefore, finding effective ECCs has become one of the most important research tasks in the flash memory community. Some traditional hard-decision error correcting codes such as Bose-Chaudhuri-Hocquenghem (BCH) and Reed-Solomon (RS) codes have been used for NAND Flash memory [1] [2]. With regard to other ECCs, soft-decision based error correction codes such as low-density parity-check (LDPC) code is a promising choice to some applications thanks to its outstanding error performance. The sum-product algorithm (SPA) [3] [4] is considered as the most well-known decoding algorithm based on iterative soft-decision. However, the large complexity of such a decoding algorithm, especially the calculation of log-likelihood (LLR) information, is still a barrier for the implementation. In [5], a new mathematical model has been proposed to calculate the LLRs for LDPC codes in the presence of CCI. From this model, we can observe that a mass of integral and logarithm operations should be made in initialization procedure to measure the priori probabilities before executing the SPA. In order to reduce the complexity of decoding, some simplified decoding algorithms, e.g., majority-logic decoding (MLGD) and Min-sum decoding algorithms, have been presented in [6] and [7]. For LDPC codes with high column-weight, the author propose two reliability-based iterative majority-logic decoding algorithms [8], which can greatly reduce the decoding complexity while keeping good error correction performance.
In this paper, taking into consideration the erase and program characteristics of MLC flash memory channels, we present a modified soft reliability-based iterative majority-logic decoding (SRBI-MLGD) algorithm, in which we establish mathematical formulation to approximately model the threshold voltage distribution of storing cells effected by CCI. II. MULTI-LEVEL CELL NAND FLASH MEMORY
A. NAND Flash Memory
Each NAND flash storage cell is composed of a semiconductor with a floating-gate MOS transistor. The value of voltage in floating gate is varied during programming process. Conventional single-level cell (SLC) has two levels of charge amount which means it can only store one bit in each cell, while multi-level cell (MLC) has ʹ௕ಾಽ಴ levels of charge amount and hence it can store ܾெ௅஼ bits in each cell. Although MLC memory possesses relatively higher storage density, it is more vulnerable to errors because of small variation of the charge and the parasitic capacitance-coupling effect while programming. There are many noise factors such as retention noise and random telegraph noise, but some specific analytical methods are designed to work out with them, so cell-to-cell interference is the major distortion considered in this paper.

B. Cell-to-cell Interference
The channel model of NAND flash memory is described in [9]. The voltage shift of one floating gate transistor can influence the voltage of its neighboring floating gate transistors because of the parasitic capacitance coupling effect. This effect is commonly called as cell-to-cell interference (CCI) in NAND flash memory. As the MLC technology keeps scaling down, CCI is served as a major noise source in flash memory model. II. CONCLUSION
In this paper, by exploiting the characteristics of MLC NAND flash memory channels, we present a modified SRBI-MLGD algorithm for high column weight LDPC coded MLC NAND flash memory channels. Since the modified algorithm still can be performed by logical and add operations, which leads to less computational complexity as compared with the standard SPA. Simulation results show that the modified SRBI-MLGD algorithm can provide a desirable trade-off between performance and complexity in NAND flash memory channels.
