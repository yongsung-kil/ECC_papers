Abstract—State-of-the-art NAND flash chips experience significant channel noise impairments and raise several data reliability issues. To overcome these issues, low-density parity-check (LDPC) codes are becoming the main stream error-correctingcodes in flash memory controllers. Consequently, the long beliefpropagation (BP) decoding latency starts to deteriorate the system performance. In this paper, a low-complexity quantization-aware belief-propagation (QA-BP) decoding scheme is introduced for meeting the requirement of faster BP convergence speed. Using the shuffled BP algorithm, the proposed QA-BP scheme takes advantage of highly unequal error probability of input loglikelihood-ratios (LLRs), transpired as a result of non-uniform channel quantization, and update only less reliable variable nodes that are initiated with higher error probability LLR values. For realizing the QA-BP scheme, neither additional runtime operations nor real-valued comparisons are required. Using simulation results, it is shown that the proposed QA-BP scheme can yield a noticeable improvement in BP convergence speed without compromising the error performance.
I. INTRODUCTION
With increasing usage of NAND flash memory in digital electronic products and high-performance enterprise data applications, it has become the most dominant data storage medium, owing to its prominent features such as low cost, high storage capacity, low power consumption and faster read-response time. However, due to the continuous node-scaling along-with multi-level-cell (MLC) technology used for satisfying the evergrowing demand of storage size, several channel noise and interference effects have been emerged over the past several years [1]–[4], thereby, degrading the reliability of flash data. As a direct consequence, the operational lifetime of a modern flash memory chip has become strictly limited with respect to wear-out (number of program and erase (PE) operations) and retention age (time duration of data storage) [5]. This finite number of PE cycles that the flash memory can handle is also referred as memory endurance. These reliability issues cause significant hindrance towards further proliferation of flash usage in practical applications.
Modern flash memory controllers widely employ errorcorrecting-codes (ECCs) to improve the data reliability. However, as the storage capacity requirement tends to increase

incessantly, traditional ECC schemes, such as Bose-ChaudhuriHocquenghem (BCH) code with hard-decision decoding, are turned-out to be inadequate to ensure the on-chip data reliability. Thus, more powerful codes with soft-decision decoding are inevitably required for handling large number of flash channel errors. In particular, the low-density parity-check (LDPC) codes have gained significant attention due to their well-proven superior error-correction capability and have appeared to be a promising candidate for flash channel [6]–[8]. However, it is highly desirable to develop low-complexity LDPC decoding algorithms for flash channel such that its faster read-response time is not compromised. In this direction, we propose a simplified decoding scheme to reduce the computational complexity of LDPC decoder.
In general, LDPC codes are decoded using beliefpropagation (BP) algorithm by iteratively exchanging message updates between the variable and check nodes (mvÑc and mcÑv) along the edges of the factor (Tanner) graph of the code [9]. The message passing can be performed either simultaneously by updating all the variable/check nodes inparallel using the flooding schedule, or sequentially (serially) by updating nodes one at-a-time. However, the serial BP schedules have shown to converge (produce decoding result) faster as compared to the flooding schedule [10]. Considering that, in this paper, we focus on the shuffled BP schedule [11] where individual variable nodes (and the connected check nodes) are updated in sequential order. For shuffled BP decoding, the number of message propagations required for convergence can be reduced by selectively updating partial set of variable nodes as reported in forced-convergence (FC) [12] scheme. In FC decoder, at each iteration, the reliability of each variable node is compared with a pre-deterministic threshold and only less reliable variable nodes are selected for further updates. Alternatively, variable nodes can also be updated based on their convergence probability as discussed in lazy schedule (LS). However, LS decoder requires extra run-time operations to compute the convergence probability. Moreover, both FC and LS schemes make real-valued comparisons after every decoding iteration which itself add more computational

In CW based scheduling, variable nodes are sorted in descending order of their column weights once in the offline mode, and processed such that high column weight nodes are updated first followed by low column weight variable nodes.
Utilizing the benefits of CW based scheduling for the BP decoding of binary LDPC codes, in this paper, we present a low complexity quantization-aware belief-propagation (QABP) decoding scheme for MLC NAND flash memory. In view of the highly asymmetric error probability of log-likelihoodratios (LLRs), realized as a consequence of non-uniform quantization of flash channel, we execute the shuffled BP decoder using column-weight scheduling such that only variable nodes associated with high LLR error probability are updated in each BP iteration. Via computer simulations, we demonstrate the increase in convergence speed without degrading the system error performance.

To this end, we define a binary state vector U “ tui|i “ 1, ..., N u for selectively updating the variable nodes, where N (code-word length) is the total number of variable nodes in the code graph. Here, ui “ 1 flags the skip operation and vice-versa. If stopping criteria

(parity-check verification) is not satisfied within α iterations, we reset the status flag (ui “ 0) and update all the variable nodes in subsequent iterations. In this work, we select the

iteration parameter α based on numerical simulations. Besides,

to further improve the convergence speed, we follow the

high-to-low (H-L) column-weight (CW) based scheduling [14]

V. CONCLUSION
Considering the importance of simplified decoders for timesensitive NAND flash channel, we present a low complexity quantization-aware belief propagation (QA-BP) decoding scheme that can achieve a faster decoding convergence. Using an irregular LDPC code, we propose to update partial set of variable nodes, leveraging on the asymmetric nature of input log-likelihood-ratio (LLR) values.
