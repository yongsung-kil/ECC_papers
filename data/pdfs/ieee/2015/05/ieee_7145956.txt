Novel Extending Scheme for the Construction of Rate-compatible IRA-like Codes

Wenwen Li, Jing Lei, Er-bao Li, Baoguo Li
College of Electronic Science and Engineering National University of Defense Technology Changsha, China liwenwennudt@outlook.com

Abstract— In this paper, a novel extending algorithm is proposed for the construction of rate compatible (RC) irregular repeat accumulate like (IRA-like) codes. Based on the structure characters of IRA-like codes, we propose an extending algorithm with uniform check node degree distribution and strong dependency in parity check matrix. Lower rate codes constructed through this method are efficiently encodable and well performed. Starting from initial code rate 8/16, the designed extending algorithm combining with existing puncturing algorithm realizes a series of RC IRA-like codes with rate R = 8/24, 8/23… 8/9. Simulation results show that the constructed RC IRA-like codes outperform those of similar algorithms from 0.04dB to 0.4 dB.
Keywords—Rate compatible (RC); Low Density parity check codes (LDPC); Irregular Repeat Accumulate (IRA) like Codes; extending algorithm; Quasi cyclic (QC)
I. INTRODUCTION
High throughput is a requirement in modern wireless communication system. Fixed coding scheme in traditional wireless communication system limits the throughput due to the time varying character of wireless channel. Unlike fixed coding scheme, rate-compatible (RC) codes are capable of providing both flexibility and efficiency in data transmission. This technique begins with convolutional codes [1], and then it is introduced to LDPC codes.
Several approaches for the construction of RC-LDPC codes, for example puncturing [6-10], extending [2][10-12], splitting [13], inserting dummy bits [14] and so on, have been presented during last years. Among all of them, puncturing and extending are main methods applied in type-II hybrid automatic repeat request (ARQ) protocol. Puncturing is adopted to obtain higher rate codes while extending on the contrary. Neither of them is able to provide good RC family codes alone. As for puncturing, there exist several efficient algorithms, which can provide higher rate codes with ideal performance. Compared with that, the research of extending algorithm still remains an open problem.
Generally, lower rate RC-LDPC codes can be acquired by adding rows and columns to parity check matrix of mother code simultaneously. When adding new parts to original matrix, two conditions are desired in construction of lower rate codes with good performance. One is uniform check node

degree distribution; another is strong dependency between original parity check matrix and newly added parts [11]. The extending algorithm for RC-LDPC codes with lower triangular structure is proposed by using progressive edge growth (PEG) method in [11]. However, its average row weight decreases in newly added parts. An optimal technique which adopts irregular repeat accumulate (IRA) codes [3] as mother code is given in [10] based on bit filling algorithm. Though it provides ideal performance and encoding convenience, it suffers from relatively high construction complexity. Recently, another approach to realize RC-IRA codes by using quadratic congruential extension sequences is offered in [12]. The method attains both good performance and low complexity. Nevertheless, it fails to create good dependency between newly added parts and original matrix.
In this paper, a new extending algorithm to realize RCLDPC is proposed. In our RC scheme, IRA-like codes [5] are adopted as mother code. They are a special modification of IRA codes, possessing quasi-cyclic (QC) structure and zigzag pattern both. Grounded on these structure characters of IRAlike codes, our extending algorithm is able to balance check node degree distribution while provide strong dependency between original and new parts. Together with the puncturing method for IRA code given in [10], the extending algorithm can obtain a series of well-performed RC-LDPC codes.
The remaining parts of the paper are organized as follows. In section II, a brief review of IRA-like codes and some of the relevant concepts are given. Then the proposed extending algorithm for RC IRA-like codes is discussed in detail in section III. Section IV provides some simulation results and comparisons to evaluate the effectiveness of our extending algorithm. And Section V concludes the paper.

III. EXTENDING ALGORITHM FOR IRA-LIKE CODES In this section, the extending algorithm for IRA-like code is presented and analyzed in detail. Based on the structure of mother code, the extending profile is given in Fig.3
Fig.3 Profile of extending scheme
Hm stands for parity check matrix of IRA-like codes acting as mother code. Z1, Z2 and Z3 are all zero sub-matrix. Thus the parity check matrix retains lower triangular structure after extension. Therefore, the added parity bits can be attained by the given information bits and existed parity check bits, providing encoding efficiency for lower rate codes.
As for B1 and B2, both of them are set as identity matrix since extension with all degree-2 variable nodes turns out to be undesirable. Large amount of degree-2 nodes have great impact on code performance [10]. The problem can be settled by introducing degree-1 variable nodes. In [10], it proves that there exists an optimal proportion of degree-1 and degree-2 for a certain code through density evolution theory and simulations. However, this proportion can’t be possessed by all rates and it suffers from high construction complexity. Grounded on these considerations, the proportion of degree-1 nodes in B1 and B2 is set as 100%. Though it is a little inferior to that of optimal proportion, it provides low complexity as well as ideal performance for all code rates.

V. CONCLUSIONS
A new extending algorithm to construct RC-LDPC codes is proposed in this paper based on QC structure and zigzag pattern of IRA-like codes. A set of RC family codes with efficient encoding and good error correction capability is constructed through the algorithm. In conclusion, three main factors guarantee the good performance of the constructed lower rate codes: 1) the balance of row weight distribution in added parts; 2) the good dependency between newly added parts and original matrix; 3) the avoidance of short cycles through the careful selection of shift value in QC structure. Simulation results verify that the new method has better performance compared with existing algorithms.
