STRUCTURED LDPC OVER URN MODEL CHANNELS WITH MEMORY

Vijay Nagarajan and Olgica Milenkovic Electrical and Computer Engineering Dept.,University of Colorado,Boulder
E-mail: [ nagaraja, milenkov) @colorado.edu

Abstract
In this paper we investigate the pelformance of structured LDPC codes over a class of channels with memory. The channels under consideration are based on Polya’s urn model [I] and can be viewed as a practical representative of channels with burst errors. Structured rather than random-like LDPC codes are considered due to thefact that theformer lead to significant reduction in storage complexity and simple hardware architectures. The bit error rate (BER) cnrves for iterative decoding with chnnnel estimation presented in this work show that random-like codes and a new class of regular and irregular codes based on parity-check matrices of the form of block-circulants have comparable performance for codelengths of the order of several thousands. Furthermore, the complete convergence region within the parameter space of the urn channel model is determined by using a new techniquefor generating sideinformation termed state XORing. This technique also allows for complete characterization of the convergence region of another important bursty channel model, namely the Gilbert-Eliot scheme. Some possible applications of the results described in this paper are for designing reliable coding schemes for communication overfading or storage channels.
Key Words: Channels with Memory, Structured LDPC Codes, Urn Models
1. Introduction
LDPC coding for discrete, symmetric, memoryless channels is a well-understoodproblem with known closeto-optimal solutions. Several capacity approaching code design tools for such channels exist such as density
evolution [SI. On the other hand, many practical and
important channels, such as fading channels, have memory. The problem of code desigdanalysis for channels with memory has only recently become the focal point of communication theory research. Among the few channel models from this class, analyzed in the setting of joint channel-state estimation and iterative decoding, is the Gilbert-Eliot (henceforth, GE) model [21. It was shown in [4] that a class of channels based on

Polya’s urn scheme represents a model more accurate then the GE system for hard-decision frequency-shift keying demodulated correlated Rayleigh and Rician fading channels. The quality of approximation in this case is measured in terms of the autocorrelation function and variational distance. The finite memory version of the communication channel described in terms of the Polya’s urn model for contagion [l] (henceforth, the PU channel) depends only on three parameters. For the special case of memory of length one, the model reduces to a simple Markov chain with two states. For reasons of clarity, we will restrict our attention to this case only, although all results presented in the paper can be extended to apply to the more general case. In this paper, we will first describe the message-passing algorithm and density evolution for the PU channel. Based on these results, we will evaluate the performance of random codes and a special class of structured LDPC codes over this channel. In this case, iterative decoding is performed jointly with channel state estimation, utilizing the standard BCJR algorithm. We will also present a complete characterization of the zero-error-probability (convergence) region in the underlying parameter space, based on a new side-information technique referred to as state XOR-ing. State XOR-ing can be used to complete the characterization of the convergence region for the GE channel described in [Z]. The outline of the paper is as follows: Section 2 gives a brief description of the PU model. Section 3 contains a short overview of the construction techniques for LDPC codes under consideration. The iterative channel estimation/deccding algorithm is described in Section 4. Section 5 presents a complete characterization of the convergence region of the PU channel. Section 6 provides a description of capacity achieving code designs. Section 7 contains the simulation results, while concluding remarks are made in Section 8.

6.Design of Capacity achieving codes

As described in Section 4 and in [Z], we first write
the density evolution equations for the channel estimator and the decoder assuming a stable input density. It has been proven in [2] that for sufficiently large number of iterations of the estimator per iteration of the decoder, a stable estimator output density is available. As for the GE channel, for any given urn model channel, the density of the outputs of the PU subgraph after N iterations (N sufficiently large) is pre-calculated based on quantized
extrinsic information. Using this as stable density input to the decoder, DE is performed over the LDPC sub-
graph using an approximate technique as the S e d Gaussian approximation (SGA). The degree distribution is determined using a linear programming technique with the objective of maximizing the achievable rate under the constraint that the transfer functions never intersect for the entire parameter range. 8. Conclusions
In this paper, we compared the performance of structured LDPC codes based on cycle-invariant difference sets with the performance of random-like codes for communication over a simple channel with memory described in terms of the PU model. We derived the equations for belief propagation for the combination of a channel state estimator and iterative decoder for the PU channel. We showed that these structured codes have almost the same performance as the random-like codes, but with significantly smaller storage complexity. We

also provided a complete characterization of the convergence region of the PU channel and completed the characterization of the convergence region for a GE channel, by developing a new analytic technique termed state XOR-ing.
