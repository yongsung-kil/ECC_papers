Performance Analysis of BP-based Algorithms for Irregular Low-Density
Parity-Check Codes On Fast Rayleigh Fading Channel
Akinori Ohhashi † and Tomoaki Ohtsuki †† † Graduate School of Science and Technology, Tokyo University of Science
††Faculty of Science and Technology, Tokyo University of Science 2641 Yamazaki, Noda, Chiba 278 – 8510 Japan
E-mail : † oohashi@pc.email.ne.jp, †† ohtsuki@ee.noda.tus.ac.jp

Abstract— In this paper, we analyze the performance of irregular lowdensity parity-check (LDPC) codes with three belief propagation (BP) based decoding algorithms, namely the uniformly most powerful (UMP) BP-based algorithm, the normalized BP-based algorithm, and the offset BP-based algorithm on a fast Rayleigh fading channel by using density evolution (DE). We also optimize the code construction of the irregular LDPC codes with three BP-based algorithms, and determine optimum parameters for the normalized BP-based algorithm and the offset BP-based algorithm on the fast Rayleigh fading channel. From the numerical results, we show that the performance of the irregular LDPC codes with offset BP-based algorithm can be very close to that with BP algorithm on the fast Rayleigh fading channel. We also show that the irregular LDPC codes with offset BPbased algorithm can achieve very good trade-off between the performance and decoding complexity.
I. INTRODUCTION
Low density parity check (LDPC) codes, proposed by Gallager [1] in the 1960’s, and later rediscovered by MacKay and Neal [2] [3], appear as a class of codes that can yield very good performance on a binary symmetric channel (BSC) as well as on an additive white Gaussian noise (AWGN) channel. It has been shown that these LDPC codes with iterative decoding based on belief propagation (BP) algorithm achieve a remarkable error rate performance that is close to the Shannon limit. The BP algorithm can be simplified to three BP-based algorithms, namely the uniformly most powerful (UMP) BP-based algorithm that is identical to the min-sum algorithm [4], the normalized BP-based algorithm, and the offset BP-based algorithm [5]–[7]. Three BPbased algorithms achieve very good trade-off between the performance and the decoding complexity. Density evolution (DE) has been proposed to analyze the capacity of regular and irregular LDPC codes with BP algorithm [8] [9]. In the BP algorithm, the irregular LDPC codes with the optimized code construction outperform the regular LDPC codes. In [10] the regular and irregular LDPC codes with BP algorithm have been analyzed on Rayleigh fading channel using DE. DE has been applied to regular LDPC codes with three BP-based algorithms in [7]. The normalization factor and the offset value of the normalized BPbased algorithm and offset BP-based algorithm for the regular LDPC codes have been optimized, respectively. However, the irregular LDPC codes with three BP-based algorithms have not been analyzed yet.
In this paper, we analyze the performance of irregular lowdensity parity-check (LDPC) codes with three belief propagation (BP) based decoding algorithms, namely the uniformly most powerful (UMP) BP-based algorithm, the normalized BPbased algorithm, and the offset BP-based algorithm on a fast Rayleigh fading channel by using DE. We also optimize the code

construction of the modified irregular semi-random LDPC codes with three BP-based algorithms, and determine optimum parameters for the normalized BP-based algorithm and the offset BPbased algorithm on the fast Rayleigh fading channel. From the numerical results, we show that the performance of the irregular LDPC codes with offset BP-based algorithm can be very close to that with BP algorithm on the fast Rayleigh fading channel. We also show that the irregular LDPC codes with offset BP-based algorithm can achieve very good trade-off between the performance and decoding complexity.

II. LOW-DENSITY PARITY-CHECK (LDPC) CODES

A low density parity check (LDPC) code is a linear error-

correcting code defined by a parity-check matrix that contains

mostly zeros and only a small number of ones. As a linear block

code, an LDPC code can be represented by a Tanner Graph [11].

If the parity-check matrix H has N columns and M rows, the

codewords consist of sequences c of N bits that satisfy a set of

M parity checks defined by the parity-check equation Hc = 0.

If the parity-check matrix H is full-rank, the number of mes-

sage bits is K = N − M, and the code rate is R = K/N. The

corresponding Tanner Graph consists of N bit nodes, M check

nodes, and a certain number of edges. Each bit node represents a

bit of the codeword. Each check node represents a parity check

of the code. An edge exists between a bit node and a check node

if and only if there is a “1” in the corresponding entry in the

parity-check matrix.

Regular LDPC codes are those for which all the bit (check)

nodes have a uniform degree dv (dc), where the degree of a

node refers to the number of edges for which it is a neigh-

bor node. On the other hand, the irregular LDPC codes are

those for which all the bit and the check nodes can have dif-

VIII. NUMERICAL RESULTS
We have performed numerical calculations of the code optimization and the threshold for the irregular LDPC codes with BP, UMP BP-based, normalized BP-based, and offset BP-based algorithms. For comparison, we show the thresholds for three ensembles of rate-1/2 regular LDPC codes with BP, UMP BPbased, normalized BP-based, and offset BP-based algorithms on the AWGN channel and the fast Rayleigh fading channel in Tables I and II.
In Table III, we show the optimized degree distributions and those thresholds for the irregular LDPC codes with BP, UMP BP-based, normalized BP-based, and offset BP-based algorithms on the AWGN channel where we set to R = 0.5 and dl = 20. We use the modified semi-random irregular LDPC codes. For each decoding algorithm, we can see that the threshold of the irregular LDPC code is better than that for the regular LDPC codes on an AWGN channel. We can also see that the BP algorithm is 0.51 dB away from the UMP BP-based algorithm, 0.41 dB away from the normalized BP-based algorithm, and 0.28 dB away from the offset BP-based algorithm, respectively.
In Table IV, we show the optimized degree distributions and those thresholds for the irregular LDPC codes with BP, UMP BP-based, normalized BP-based, and offset BP-based algorithms on the fast Rayleigh fading channel where we set to R = 0.5 and dl = 20. We use the modified semi-random irregular LDPC codes. For each decoding algorithm, we can see that the threshold of the irregular LDPC code is better than that of the regular LDPC code on the fast Rayleigh fading channel. We can also see that the BP algorithm is 0.62 dB away from the UMP BP-based algorithm, 0.45 dB away from the normalized BP-based algorithm, and 0.23 dB away from the offset BP-based algorithm, respectively.
We can also see that the optimized degree distribution for each decoding algorithm is different on the AWGN channel and the fast Rayleigh fading channel. From Tables I and II, in the regular LDPC codes on the AWGN channel and the fast Rayleigh fading channel, the normalized BP-based algorithm outperforms the offset BP-based algorithm. However, in the irregular LDPC codes on the AWGN channel and the fast Rayleigh fading channel, the threshold of the offset BP-based algorithm with the optimized degree distribution and the optimal offset value is better than that of the normalized BP-based algorithm with the optimized degree distribution and the optimal normalization factor. IX. CONCLUSION
In this paper, we analyze the performance of irregular LDPC codes with UMP BP-based algorithm, the normalized BP-based algorithm, and the offset BP-based algorithm on a fast Rayleigh fading channel by using DE. We also optimize the code construction of the modified irregular semi-random LDPC codes with three BP-based algorithms, and determine the optimum parameters for the normalized BP-based algorithm and the offset BP-based algorithm on the fast Rayleigh fading channel. From the numerical results, we showed that the performance of the irregular LDPC codes with the offset BP-based algorithm can be very close to that with the BP algorithm on the fast Rayleigh fading channel. We also showed that the irregular LDPC codes with the offset BP-based algorithm can achieve very good tradeoff between the performance and decoding complexity.
