UMP BP-based Algorithms on Fast Rayleigh Fading Channel

Abstract—In this paper, we analyze and evaluate the performance of the regular LDPC codes with the normalized and UMP BP-based algorithms on a fast Rayleigh fading channel. For the short regular LDPC codes, we derive the formula of the normalization factor theoretically by using the probability density function (pdf) of the initial likelihood information for the normalized BP-based algorithm on the fast Rayleigh fading channel. For the long regular LDPC codes, we derive the optimal normalization factor for the normalized BP-based algorithm on the fast Rayleigh fading channel by using density evolution (DE). We also analyze the performance of the long regular LDPC codes with the UMP BP-based algorithm on the fast Rayleigh fading channel by using DE. From the numerical and simulation results, we show that the optimum normalization factor for the short regular LDPC codes on the fast Rayleigh fading channel is different from that on the AWGN channel. We also show that for the short regular LDPC codes, the normalized BP-based algorithm outperforms the UMP BP-based algorithm, and has the performance very close to those of the BP algorithm. Furthermore, for the long regular LDPC codes, we show that the normalized BP-based algorithm outperforms the BP algorithm and the UMP BPbased algorithm on the fast Rayleigh fading channel. Therefore, for the short and long regular LDPC codes, the normalized BP-based algorithm is shown to be suitable as the decoding algorithm on the fast Rayleigh fading channel.
I. INTRODUCTION
Low density parity check (LDPC) codes can achieve a much better performance, decoding with the BP algorithm or the sumproduct algorithm. Recently, various iterative decoding algorithms based on the BP algorithm have been proposed. The uniformly most powerful (UMP) BP-based algorithm [1] , which is identical to min-sum algorithm [2], needs only real additions and is said to be universal as it does not need any a priori information about the channel state. However, the simplification causes degradation in the performance. In [3] the normalized BP-based algorithm has been proposed to improve the UMP BPbased algorithm. The normalized BP-based algorithm with reduced complexity can achieve nearly the same error rate performance as the BP algorithm on an AWGN channel. In addition, the normalized BP-based algorithm conserves the important universal property of the UMP BP-based algorithm on an AWGN channel.
The performance of LDPC codes can be analyzed by the technique known as density evolution (DE). The DE was applied to the BP algorithm [4] and the BP-based algorithm [5] to determine the thresholds of LDPC codes.
LDPC codes have recently been expected to be applied to wireless communications. Some papers have applied LDPC codes to wireless communications where the BP algorithm is employed as a decoding algorithm [6][7]. As a decoding algorithm particularly in wireless communications, a simpler decoding algorithm is preferred. As written above, the normalized BP-based algorithm can achieve nearly the same error rate performance as the

BP algorithm on an AWGN channel. In addition, the normalized BP-based algorithm is universal as it does not need any a priori information about the channel state. This universal property is attractive in wireless communications where the channel state changes frequently. However, the normalized BP-based algorithm has not been derived on the Rayleigh fading channel.
In this paper, we analyze and evaluate the performance of the regular LDPC codes with the normalized BP-based algorithms on the fast Rayleigh fading channel. For the short regular LDPC codes, we derive the formula of the normalization factor theoretically by using the pdf of the initial likelihood information for the normalized BP-based algorithm on the fast Rayleigh fading channel. For the long regular LDPC codes, we derive the optimal normalization factor for the normalized BP-based algorithm on the fast Rayleigh fading channel by DE. We also analyze the performance of the long regular LDPC codes with the UMP BP-based algorithm on the fast Rayleigh fading channel by using DE. From the numerical and simulation results, we show that the optimum normalization factor for the short regular LDPC codes on the fast Rayleigh fading channel is different from that on the AWGN channel. We also show that for the short regular LDPC codes, the normalized BP-based algorithm outperforms the UMP BP-based algorithm, and has the performance very close to those of the BP algorithm. Furthermore, for the long regular LDPC codes, we show that the normalized BP-based algorithm outperforms the BP algorithm and the UMP BP-based algorithm on the fast Rayleigh fading channel. Therefore, for the short and long regular LDPC codes, the normalized BP-based algorithm is shown to be suitable as the decoding algorithm on the fast Rayleigh fading channel.
II. LOW DENSITY PARITY CHECK CODES
A low density parity check (LDPC) code is a linear errorcorrecting code defined by a parity-check matrix that contains mostly zeros and only a small number of ones. As a linear block code, an LDPC code can be represented by a Tanner Graph [8]. If the parity-check matrix H has N columns and M rows, the codeword consists of sequences c of N bits that satisfy a set of M parity checks defined by the parity-check equation Hc = 0. The number of message bits is K ≥ N − M , and the rate of the code is R = K/N . The corresponding Tanner Graph consists of N bit nodes, M check nodes, and a certain number of edges. Each bit node represents a bit of the codeword. Each check node represents a parity check of the code. An edge exists between a bit node and a check node if and only if there is a “1” in the corresponding entry in the parity-check matrix.

III. THE SYSTEM MODEL

We consider BPSK modulation that maps a codeword c = (c1, c2, · · · , cN ), with cn = 0, 1, into a transmitted signal x = (x1, x2, · · · , xN ), according to xn = 2cn − 1, for n = 1, 2, · · · , N . After a transmitted signal passes through the Rayleigh fading channel, a signal y = (y1, y2, · · · , yN ) is received at the receiver, described by

VII. NUMERICAL AND SIMULATION RESULTS

We assume each bit to be affected by an independent Rayleigh fading and no information for the channel state is available at the receiver.
We have performed numerical calculations of the thresholds for the normalized BP-based algorithm with different normalization factors α and different code ensembles with the code rate 1/2 on the fast Rayleigh fading channel. The results are shown in Fig. 1. Also, the optimal thresholds of the normalized BP-based algorithm on the fast Rayleigh fading channel are summarized in Table 1. In the three code ensembles, the thresholds of the normalized BP-based algorithm are about 0.7 ∼ 1.5 dB lower than those of the UMP BP-based algorithm on the fast Rayleigh fading channel, and are about 0.1 ∼ 0.7 dB lower than those of the BP algorithm. In Fig. 2, the BER of the (8000, 4000) regular LDPC code with dc = 6 and dv = 3 is shown. We can see that for the long regular LDPC codes, the normalized BP-based algorithm outperforms the BP algorithm and the UMP BP-based algorithm on the fast Rayleigh fading channel in terms of the thresholds and the simulation results.
Fig. 3 shows the simulation and theoretical results of the normalization factor α on the fast Rayleigh fading channel where we set dc = 6. We can see that the theoretical curve is almost identical to the simulation one up to Eb/N0 = 7 dB. We choose the normalization factor in the range of Eb/N0 values that correspond to the BER of 10−4 to 10−5. For instance, the BER of (504, 252) regular LDPC code with the BP algorithm is between 10−4 to 10−5 at Eb/N0 = 6.5 dB on the fast Rayleigh fading channel. In Fig. 3, α is 1.2 at Eb/N0 = 6.5 dB. We note that since the normalization factor α on the AWGN channel is 1.4 [3], the optimal normalization factor changes with the channel.
Figs. 4 and 5 show the BER of the (504, 252) LDPC code with the normalized BP-based algorithm for some values of α on the fast Rayleigh fading channel. In Fig. 4, the normalized BP-based algorithm with α = 1.2 achieves the best BER. In Fig. 5, we can see that the BER of the normalized BP-based algorithm with α = 1.2 is almost identical to those with optimal α at each Eb/N0, obtained by the theory and by the simulation, respectively. Therefore, we use α = 1.2 in the following evaluation.
Fig. 6 shows the BER of the (504,252) regular LDPC code with the BP algorithm, the UMP BP-based algorithm, and the normalized BP-based algorithm on the fast Rayleigh fading channel, where α = 1.2. Note that the maximum number of iterations for each algorithm is set to the smallest value for which the near convergence performance is achieved: 1000 for the

BP algorithm, 200 for the UMP BP-based algorithm, and 200 for the normalized BP-based algorithm. We can see that at the BER= 10−4, the normalized BP-based algorithm can achieve about 0.3 dB larger coding gain than the UMP BP-based algorithm. We can also see that the BER of the normalized BP-based algorithm is almost identical to that of the BP algorithm.
Therefore, for the short and long regular LDPC codes, the normalized BP-based algorithm is suitable as the decoding algorithm on the fast Rayleigh fading channel.
VIII. CONCLUSIONS
In this paper, we analyzed and evaluated the performance of the regular LDPC codes with the normalized BP-based algorithms on the fast Rayleigh fading channel. For the short regular LDPC codes, we derived the formula of the normalization factor theoretically by using the pdf of the initial likelihood information for the normalized BP-based algorithm on the fast Rayleigh fading channel. For the long regular LDPC codes, we derived the optimal normalization factor for the normalized BP-based algorithm on the fast Rayleigh fading channel by DE. We also analyzed the performance of the long regular LDPC codes with the UMP BP-based algorithm on the fast Rayleigh fading channel by using DE. From the numerical and simulation results, we showed that the optimum normalization factor for the short regular LDPC codes on the fast Rayleigh fading channel is different from that on the AWGN channel. We also showed that for the short regular LDPC codes, the normalized BP-based algorithm outperforms the UMP BP-based algorithm, and has the performance very close to those of the BP algorithm. Furthermore, for the long regular LDPC codes, we showed that the normalized BP-based algorithm outperforms the BP algorithm and the UMP BP-based algorithm on the fast Rayleigh fading channel. Therefore, for the short and long regular LDPC codes, the normalized BP-based algorithm is suitable as the decoding algorithm on the fast Rayleigh fading channel.
