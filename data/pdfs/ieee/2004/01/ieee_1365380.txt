Belief-Propagation with Information Correction: Improved Near Maximum-Likelihood Decoding of Low-Density Parity-Check Codes
Nedeljko Varnica† and Marc Fossorier‡ †Division of Engineering and Applied Sciences, Harvard University ‡Department of Electrical Engineering, University of Hawaii at Manoa
e-mail: varnica@deas.harvard.edu, marc@spectra.eng.hawaii.edu

Abstract — We propose an improved beliefpropagation (BP) decoder for low-density paritycheck (LDPC) codes based on channel information correction. We show that our algorithm achieves sizeable performance gains (in waterfall and error floor regions) compared to the standard BP decoder. We verify by examples that the proposed decoder almost achieves the maximum-likelihood decoding performance for short LDPC codes.

I. Improved Decoding Algorithm

The graphs of finite-length low-density paritycheck (LDPC) codes [1] with good distance properties contain a large number of cycles. The presence of cycles in LDPC code graphs can introduce a noticeable loss in performance if the standard belief-propagation (BP) decoding is performed instead of the maximum likelihood (ML) decoding.
The BP-to-ML performance gaps exist for both short and long LDPC codes; large gaps for long codes are typically observed in the error floor region. To bridge these two types of gaps we propose a method based on information correction in the BP decoding algorithm.
Denote the transmitted codeword by x. Let r be the received observation of this vector at the output of an AWGN channel. We denote by CS(L) the set of unsatisfied check nodes (SUCN) after L iterations of BP decoding. Let VS(L) be the set of variable nodes connected to CS(L) by at least one edge in the LDPC code graph G.

Definition: The SUCN code graph G(SL) = (VS(L), CS(L)) ⊆ G is defined as the graph induced by the SUCN CS(L).

Denote the degree of a variable node v with respect to the SUCN code graph G(SL) by dGS (v). Clearly, dGS (v) = 0 if v ∈/ VS(L) and dGS (v) > 0 if v ∈ VS(L).

Observation 1: The higher the degree dGS (v) of a variable node v ∈ VS(L), the more likely is v to be in error.

With this decoding strategy we enforce the correct values on the selected variable nodes and eliminate pseudocodewords [3]. For example, if our method selects the node vi, and xi = 1, then (assuming O(vi) is set to −S) all the pseudocodewords which have a non-zero entry in the i-th position are eliminated. Hence, the set of pseudo-codewords is significantly reduced and the pseudo-distance spectra is improved.

II. Results
In Fig. 1 and 2 we demonstrate the performance of our method for the (155,64) Tanner code [4] and the (2640,1320) Margulis code [5] on the AWGN channel.
For the Tanner code our algorithm (with jmax = 4, L = 100 and K = 10) outperforms the original BP with the same maximum number of 400 iterations (and less iterations than the original BP on average) at word error rate W ER = 10−3 by about 0.35dB. When we allow a large number of iterations our method closely approaches the ML performance, with significantly lower complexity. For the Margulis code our decoding strategy improves the performance in the error floor region by at least two orders of magnitude, see Fig. 2.
Our approach can be extended in a straightforward manner to other channels (e.g., binary erasure, binary symmetric, fading and partial response channels). Our methods can be applied to other iterative decoders as the correction algorithm is independent of the iterative decoding algorithm considered.
