The Semi-Algebra Low-Density Parity-Check Codes
Yu Yi, Gi Yean Hwang and Moon Ho Lee
Institute of Information and Communications, Chonbuk National University, ChonJu, 561-756, Korea Email: yuyi1020@hotmail.com, infoman@mail.chonbuk.ac.kr, moonho@chonbuk.ac.kr

Abstract- In this paper, the construction of semi-algebra Low-density parity-check (LDPC) code with an arbitrary block length is presented. The encoding circuit is based on the original semi-algebra design and users can have the choice of using the matrix pattern and various code rates to design for different communication applications. Especially, a novel girth scheme named Adaptive Slope Group (ASG) is proposed for the semi-algebra code design with the large girth and maximum minimum distance. We can compare the performance of this novel semi-algebra LDPC code against the Shannon limit for several rates. The simulation results show our codes can perform a large coding gain at high SNRs with BPSK modulation over AWGN channels. However, column weight j is not random, only for j=2.
I. INTRODUCTION
Low-density parity-check (LDPC) codes were discovered by Gallager [1] in early nineteen sixties, in which two ideas are explored: the constrained random code construction and iterative decoding algorithms. In spite of their excellent properties, LDPC codes have been forgotten for about thirty years. Recently, LDPC codes with the sum product decoding algorithm are now recognized as the good error correcting codes approaching the binary Shannon limit and outperform turbo codes [2]. Especially, for code rate 1/2, they can achieve within 0.04dB of the Shannon limit at a bit error rate of10 −6 using a block length of 107 [3].
However, main drawbacks of LDPC codes are the higher encoding complexity [4] and short cycles such as cycle-4 in Tanner graph [5]. These disadvantages result in difficulties of hardware implementation and poor performance with the iterative decoding algorithm. Many works have been done in these two fields. In [6], algebra LDPC codes with at least cycle-6 such as BIBD and quasi-cyclic codes are described. These codes have the regular construction, but they are transformed by Gaussian elimination (GE) which can increase the encoding complexity. For the short cycles, nowadays some researchers proposed the good methods to eliminate girth and search good LDPC codes for different communication systems. A heuristic search for the good LDPC codes at short block lengths is proposed by Y. Mao in [7]. Progressive edge growth (PEG) [8] is used to design good irregular codes with the larger girth than random codes proposed by David J. C. Mackay in 2001. However,

In order to get a good trade off between the hardware implementation and performance, Semi-Algebra LDPC (SA-LDPC) code with the efficient encoding scheme is presented. This code is one of deterministic LDPC codes based on a dual diagonal sub-matrix. The pattern is similar fashion to [10]. However, the original SA-LDPC code is subject to the block length. As described in [11], the parity check H matrix of SA-LDPC code can be divided into two sub-matrices, H p and H d . H p is an (n-k) × (n-k) square matrix and H d is an (n-k)×k matrix. The H p sub-matrix has a so called dual-diagonal pattern which is defined by identity matrix, I and D, dual matrix. Dual matrix can be obtained from I matrix by removing the first row and appending a bottom row of zeros. An example is shown,

IV. CONCLUSIONS
In this paper, modified semi-algebra LDPC code based on a novel girth design named ASG is proposed. This method results in an irregular LDPC codes with an arbitrary block length. Designed codes in this class have a large girth, then

maximum minimum distances. These codes can perform well at high SNRs with the 20 numbers iterative decoding.
