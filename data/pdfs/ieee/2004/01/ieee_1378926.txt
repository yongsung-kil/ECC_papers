(LDPC) Coded Space-Time Transmit Diversity (STTD) System

Abstract‚Äî Space-time transmit diversity (STTD) and space-time block codes (STBC) are the attractive techniques for high bit-rate and high capacity transmission. Recently, low-density parity-check (LDPC) codes have attracted much attention as the good error correcting codes achieving the near Shannon limit. The concatenation schemes of LDPC codes and STBC (LDPC-STBC), and Turbo codes and STBC (Turbo-STBC) have been proposed. The performance of the LDPC-STBC is better than that of the Turbo-STBC on a flat Rayleigh fading channel. The performance of LDPC code can be analyzed by density evolution (DE). In this paper we analyze the performance of the LDPC-STBC by using DE. Also, we optimize the irregular LDPC codes for the LDPC-STBC by using DE. Furthermore, we evaluate the error rate performance of the optimized irregular LDPC codes and the regular LDPC codes for the LDPC-STBC. From the numerical and simulation results, we show that the LDPC-STBC with the irregular LDPC codes achieves the better error rate performance than the LDPC-STBC with the regular LDPC codes.

Space-time transmit diversity (STTD) and space-time block code (STBC) have attracted much attention as wireless communication schemes achieving high capacity and high bit rate transmission. The STBC for two transmit antennas, invented by Alamouti [1], can achieve the transmit diversity by orthogonalizing the signals for each transmit antenna.
Recently, low-density parity-check (LDPC) codes have attracted much attention as the good error correcting codes achieving the error rate performance close to the Shannon limit. In [2], [3] the concatenation scheme of LDPC codes with the capability of powerful error correcting and STBC based on Alamouti scheme (LDPC-STBC) has been proposed, and the LDPC-STBC was shown to achieve the better error rate performance than the Turbo-STBC on a flat Rayleigh fading channel. In [4] LDPC-STBC is also evaluated. In [5] LDPC-based space-time coded orthogonal frequency-division multiplexing (OFDM) system (LDPC-based STC-OFDM) is proposed and shown that LDPC-based STC-OFDM exhibits lower receiver complexity and more flexible scalability than turbo-code-based STC scheme. However, they are not looking at optimizing LDPC codes (degree distributions). The error correcting codes with iterative decoding algorithm such as LDPC codes and Turbo codes are analyzed by using density evolution (DE) [7] and EXIT chart [6]. DE can analyze the capacity of regular and irregular LDPC codes with the BP algorithm by deriving the probability density function (pdf) of the log likelihood ratios (LLR) obtained by the iterative decoder [7]. DE and EXIT chart can be used for designing code for iterative decoding. We can see that these thresholds are the smallest in the three ensembles of the regular LDPC codes. Also, the threshold of the regular LDPC codes becomes larger as dc and dv becomes larger.

In Fig. 1 we compare the simulation results and the thresholds for the LDPC-STBC with two transmit antennas and one receive antenna. We use the irregular LDPC code with the optimized degree distribution in Table II. We set the block length to N = 8000 for both regular and irregular LDPC codes. LDPC code. Also, for the LDPC-STBC with the irregular LDPC code, at BER = 10‚àí4, the difference between them is about 0.7 dB. VIII. CONCLUSION
In this paper we have analyzed the performance of the LDPCSTBC by using DE. Also, we have optimized the irregular LDPC codes for the LDPC-STBC by using DE. Furthermore, we have evaluated the optimized irregular LDPC codes and the regular LDPC codes for the LDPC-STBC. From the numerical and simulation results, we showed that the LDPC-STBC with the irregular LDPC code achieves the better error rate performance than the LDPC-STBC with the regular LDPC code. Also, we showed that in the LDPC-STBC, the difference between the simulation results and the threshold for the regular LDPC code is almost identical to that for the irregular LDPC code.
