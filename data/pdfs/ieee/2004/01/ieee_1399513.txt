A Class of Good Quasi-Cyclic Low-Density
Parity Check Codes Based On Progressive
Edge Growth Graph
Zongwang Li and B.V.K. Vijaya Kumar
Department of Electrical and Computer Engineering, Carnegie Mellon University 5000 Forbes Ave, Pittsburgh, PA 15213
Email: zwli@andrew.cmu.edu, kumar@ece.cmu.edu

Abstract-We present a new class of quasi-cyclic low density parity check (QC-LDPC) codes, whose quasi-cyclic nature makes them attractive for implementation. Both regular and irregular QCLDPC codes are designed by modifying the progressive edge growth (PEG) graph with a quasi-cyclic constraint. Simulations show that these QC-LDPC codes offer hardware-friendly parity check matrices and have as good error correction performance as random LDPC codes and other good QC-LDPC codes. In addition, the proposed QC-LDPC codes offer a much more flexible set of parameters compared to the traditional designs.
I. INTRODUCTION
Low density parity check (LDPC) codes, first invented in early 1960's by Gallager [1], were rediscovered in late 1990's and shown to form a class of Shannon limit approaching codes [2-6]. These codes decoded with iterative decoding method, such as the sum-product algorithm (SPA) [3], achieve excellent performance. One class of LDPC codes that admits low-complexity encoding and decoding is the class of quasicyclic LDPC codes with parity check matrices in circulant form [7]. Encoding of a quasi-cyclic (QC) LDPC code can be achieved with simple shift-registers and decoding can be done based on the circulant architecture in parallel or partially parallel manner. Recently, several classes of QC-LDPC codes have been constructed [7-10]. They have been shown to perform as well as computer-generated random LDPC codes. Thus QC-LDPC codes have attracted considerable attention because of their good performance and hardware-friendly architecture. The existing classes of QC-LDPC codes are designed based on finite geometry or algebraic methods [7-10]. All those methods are used to design QC-LDPC codes by avoiding cycles of length four and six. The code parameters such as code rate and block size are limited by the combinatorial algebraic structure or finite geometry.
In this paper, we present a class of QC-LDPC codes based on progressive edge growth graph. Parity check matrices of these new QC-LDPC codes are in circulant form and the parameters of these new QC-LDPC codes are flexible. The rest of this paper is structured as follows. In Section II, we will present the method of designing these new QC-LDPC codes and simulation results will be given in Section III. We will conclude in Section IV.

Based on the original PEG algorithm, the QC-LDPC code with its parity check matrix in circulant form can be generated by adding the quasi-cyclic constraint to the PEG algorithm. In the PEG algorithm for random LDPC codes, edges are added to the Tanner graph node by node. And we need to expand trees for each variable node. To generate a QC-LDPC code with circulant size n × n , first we need to divided all variable nodes and check nodes into small groups with each group has n nodes. Then we add edges to the Tanner graph group by group instead of node by node. For each variable node group, we only need to expand trees for the first variable node in the group to find its optimized edges. The edges of other variable nodes in the same group can de determined by the circulant

constraint automatically. In this way, the design of QC-LDPC codes is sped up by a factor n. Fig. 2 shows an example of generating a 4 × 4 circulant of QC-LDPC code. The dark circles represent the variable nodes and the dark squares represent the check nodes. Each set of 4 variable nodes and 4 check nodes are grouped together. Once an edge of the first variable node in a group is determined (shown as the solid line in Fig. 1), edges of the other variable nodes in the same group can be determined automatically according to the circulant constraint (shown as the dash lines in Fig. 1).

By modifying parameters n, c, t, (dc, ds ), both regular and

irregular QC-LDPC codes can be generated with flexible sizes

variable degree distribution.

III. SIMULATION RESULTS
All following results are obtained by simulation of an additive white Gaussian noise (AWGN) channel. Fig. 4 compares the bit error rate (BER) and block error rates (BLER) of this new QC-LDPC code based on PEG algorithm to the random PEG-LDPC codes [12], Mackay’s random LDPC code [3], and the QC-LDPC code based on Euclidian geometry (EG) [7]. All these LDPC codes are regular. The
PEG codes are of size (N , K ) equaling (8192, 7168) and
circulant 128 ×128 , where N is the codeword length and K is the information bit number. The Mackay’s code and EGQC-LDPC code are of size (8176, 7156) and the latter with circulant 511× 511. All these codes are with variable degree 4 and code rate of about 0.875. It is shown that the PEG-QCLDPC code offers comparably good performance, especially in low bit error rate/block error rate (BER/BLER) regions while being more hardware-friendly because of its quasicyclic structure and flexible parameters. Fig. 5 presents the simulation results for the PEG-QC-LDPC code and comparison with the random LDPC code and the disjoint difference set (DDS) LDPC code [9]. All these codes are regular and with size (4608, 4096), variable degree 3 and code rate 8/9. The DDS-LDPC code and the PEG-QC-LDPC code

IV. CONCULSION
This paper proposed a method to generate both regular and irregular quasi-cyclic LDPC codes based on progressive edge growth graph. Simulation results show that the proposed PEG-QC-LDPC codes exhibit bit error rate performance comparable to the random LDPC codes and other quasi-cyclic LDPC codes generated by algebraic method or finite geometry. The QC-LDPC codes generated by the proposed method are much more flexible than the codes generated by algebraic method or finite geometry. Thus the proposed PEG-QC-LDPC
