Abstract- We propose a modified hyperbolic tangent (tanh) function that is used in the sum-product algorithm (SPA). This is done in order to approximate the infinite argument of the tanh function, when decoding low-density parity-check (LDPC) codes. As shown by computer simulation results, the proposed modification has a significant impact on the error floor reduction of the code and to the coding gain improvement with respect to the SPA with no modification. Computer simulation results with approximations using both piecewise linear function and coarse quantization are also presented.
Keywords- iterative decoding, low-density parity-check codes, sum-product algorithm
1. INTRODUCTION
Low-density parity-check (LDPC) codes were proposed by Gallager in the early sixties [1], but being the technology not ready for their practical implementation, they were forgotten for many years up to the nineties, when they were rediscovered by MacKay [2]. LDPC codes can achieve near Shannon limit performance over the binary erasure channel (BEC) as well as over the additive white Gaussian noise (AWGN) channel [3] at reasonable decoding complexity. As a practical application, they have recently been adopted by the new version of the Digital Video Broadcasting by Satellite (DVB-S2) standard. More specifically, an inner LDPC code is incorporated with adaptive coding and modulation (ACM) techniques and optimized high order modulation schemes for satellite signals, such as the amplitude phase shift keying (APSK). The resulting DVB-S2 broadcasting system capacity is improved by 30% compared to the existing standard, i.e. DVB-S [4].
The design of a LDPC code can be based on either the encoding part [2], [3] or the decoding part [5], [6]. The first case includes, for example, fast encoding methods and smart parity-check matrix construction for regular/irregular LDPC codes with small number of short cycles. The second case includes reduced complexity iterative decoding algorithms based on the message passing principle, which a description can be found in [7].
In this paper, we focus our attention to the decoding part. Inspired by [6], we first adapt the sum-product algorithm (SPA). Then, we propose a modification to the hyperbolic

tangent (tanh) function to improve the bit error rate (BER) performance of LDPC codes. Furthermore, we use two approximations ofthe tanh function to reduce its computational complexity.
The organization of the paper is as follows. After this introduction, the sum-product algorithm (SPA) for LDPC decoding is described in Section II. In Section III the proposed modification to the tanh function is introduced as well as its two approximations based on piecewise linear function and coarse quantization. Computer simulation results are shown in Section IV and the concluding remarks are made in Section V.

V. CONCLUSIONS
We verify that the modification to the tanh function of the SPA is necessary to improve the performance of LDPC codes. Piecewise linear function with seven regions and also quantization with 3-bits look-up table were used in order to approximate further the modified tanh function.
As future work, it is interesting to investigate the SPA as from (6). Finally, optimization of the quantization table and the

piecewise linear function regions could be investigated, so as to
minimize the RMSE.
