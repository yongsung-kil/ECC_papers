Constructions of Quasi-Cyclic LDPC Codes for the AWGN and Binary Erasure Channels Based on Finite Fields and Affine Mappings
L. Lan, L.-Q. Zeng, Y. Y. Tai, S. Lin and K. Abdel-Ghaffar Department of Electrical and Computer Engineering University of California, Davis, Davis, CA 95616, U.S.A.
Email: squash, lqzeng, yytai, shulin, ghaffar @ece.ucdavis.edu

Abstract— This paper presents two algebraic methods for constructing efficiently encodable quasi-cyclic (QC) LDPC codes that perform well on both the AWGN and binary erasure channels with iterative decoding in terms of bit-error performance, block error performance and error-floor, collectively. The constructions are based on the cyclic subgroups of the multiplicative groups of finite fields and affine mappings.
I. Introduction
LDPC codes, discovered by Gallager in 1962 [1], were rediscovered and shown to form a class of Shannon limit approaching codes in the late 1990’s [2], [3]. These codes decoded with iterative decoding based on belief propagation achieve very good error performance. Ever since their rediscovery, construction, decoding, performance analysis, and applications of these codes in digital and storage systems have become focal points of research. Various methods for constructing LDPC codes have been proposed. Based on the methods of construction, LDPC codes can be divided into two categories: (1) random (or random-like) codes [3], [4]; and (2) structured codes, such as finite geometry LDPC codes [5]. Long random LDPC codes in general perform closer to the Shannon limit in the waterfall region than structured LDPC codes of the same parameters, however, they usually do not have sufficient structure to allow simple encoding. On the other hand, structured codes in general have encoding advantage over random codes, especially cyclic- and QC-LDPC codes [5], [6], which can be encoded with simple shift-registers with linear complexity. In fact, well designed QC-LDPC codes can perform as well as random LDPC codes in terms of biterror performance, block-error performance, and error-floor, collectively [7]–[9].

The performance of an LDPC code with iterative decoding depends on a number of code structural properties. One such structural property is the girth of the code that is defined as the length of the shortest cycle in the code’s Tanner graph. For a code to perform well with iterative decoding, its Tanner graph must not contain too many short cycles. The shortest cycles that affect code performance the most are cycles of length . Therefore, cycles of length 4 must be prevented in code construction.
Let Î be a set of variable nodes in the Tanner graph Ì of an LDPC code and Ë be a set of check nodes in Ì that are adjacent to the nodes in Î, i.e., each check node in Ë is connected to at least one variable node in Î. The nodes in Ë are called the neighbors of the nodes in Î. A set Î of variable nodes is called a stopping set of the Tanner graph Ì if each check node in the neighbor check set Ë of Î is connected to at least two variable nodes in Î [12]. The Tanner graph Ì of an LDPC code may contain stopping sets of various sizes (the size of a stopping set is defined as the number of nodes in the set). A stopping set of minimum size is called a minimal stopping set (not unique). The performance of an LDPC code on a binary erasure channel (BEC) with iterative decoding is also determined by the stopping sets of its Tanner graph, especially the size of its minimal stopping set [12]. For good performance, the size of a minimal stopping set should be made as large as possible. The size of a minimal stopping set in a Tanner graph is proved to be related to the girth of the Tanner graph [13]. For a ´­ µ-regular LDPC code whose parity-check matrix satisfies the RC-constraint, the size of a minimal stopping set in its Tanner graph is at least ­ · ½.

V. Conclusion
In this paper, we have presented two algebraic methods for constructing QC-LDPC codes for both the AWGN and binary erasure channels. Experimental results show that codes constructed by these methods perform very well with iterative decoding and have low error-floors. Since the codes are quasicyclic, they can be encoded with simple shift-registers with linear complexity [6]. This encoding advantage is not shared by other types of LDPC codes. Since the parity-check matrix has regular structure and consists of circulant permutation matrices, these structural properties also simplify the (VLSI) hardware decoder implementation, especially wire routing.
The masking technique proposed in [15] can be applied to the arrays of circulant permutation matrices constructed in Section III to construct irregular QC-LDPC codes based on the variable- and check-node degree distributions of a Tanner graph for a given code rate designed based on density evolution [4]. This construction improves the code performance in the waterfall region while still maintains the
