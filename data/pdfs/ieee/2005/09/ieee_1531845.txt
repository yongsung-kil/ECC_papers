Abstract— We investigate the finite-length scaling methodology for irregular LDPC code ensembles when transmission takes place over the binary erasure channel (BEC). We first show how the necessary computations, namely the covariance evolution and the computation of the finite-length shift, can be accomplished in the irregular case. We then investigate how the obtained approximation can be used to predict the performance of irregular code ensembles and to optimize the degree distributions for finitelength codes.
I. INTRODUCTION
Finding the exact error probability of an LDPC code ensemble of fixed length n is a computationally non-trivial problem. For the BEC, a combinatorial approach to compute this probability was introduced in [2] for regular ensembles. This approach was generalized in [5] to irregular ensembles. Further improvements were then suggested in [4]. Despite all progress, the complexity of the computation grows like a power of the blocklength, where the exponent equals the number of degrees (irregularity) of the system. Therefore, in particular for the optimization of degree distributions, which involves quite a few degrees of freedom, the exact approach is typically too costly.
In [1], an alternative approach was proposed. Rather than computing the exact error probability, the authors in [1] suggested to study the behavior of the error probability close to the threshold as a function of the blocklength. This leads to the notion of a scaling law which (in a way that can be made precise) gives an asymptotic (in the blocklength) expansion of the error probability. Although this expansion is only guaranteed to become exact in the limit of large blocklengths, it was shown that it gives fairly accurate predictions already for moderate sizes and can therefore be used as a basis for finite-length optimization. The result is that for unconditionally stable ensembles (ensembles for which ¼´0µ ¼´1µ 1) having only a single critical point (point at which the density evolution curve touches zero at the threshold), the block error probability curves of an LDPC code ensemble of fixed length n can be

For the general irregular case it is necessary to work in a space whose dimension equals the number of variable degrees plus the highest check node degree minus one.
In Section II we explicitly state all the terms necessary to accomplish the covariance evolution for irregular ensembles. In Section III we then accomplish the computation of the shift parameter. Finally, in Section V we outline what remains to be accomplished before the presented approach can be used as a performance prediction and optimization tool.

II. COVARIANCE EVOLUTION
A. Underlying Principles
The finite-length scaling approach can be seen as an extension of the celebrate density evolution analysis of Luby et al. in [3]. Whereas density evolution only tracks means, the

Both approaches consider a version of the belief propagation algorithm where one variable node is decoded at each time instant and removed from the graph with all its connected edges. For the regular case, it was shown in [1] that it was necessary to track, through the decoding process, the evolution of only three variables: the number of variable nodes still unknown, denote it by “v”, the number of check nodes of residual degree one, call it “s”, and the number of check node of residual degree bigger or equal to two, let it be “t”. The state of the decoder ´v t sµ was tracked throughout the decoding process. Note that the decoding process is unsuccessful if and only if the variable s takes on the value 0 before all variables have been determined. The distribution of the state was shown to tend to a Gaussian distribution whose mean could be tracked through Density Evolution and its covariance matrix through Covariance Evolution. Knowing the mean and covariance matrix allows one to find a simple approximation of the block and bit error probability curves. Further, a refinement of this approximation results in adding a finite-length shift parameter.

V. OPTIMIZATION VIA SCALING: AN OUTLINE

Let us briefly outline what remains to be done before the

above scaling approach can be used as an everyday tool for

finite-length analysis and optimization.

First, so far we have only dealt with ensembles that are

unconditionally stable ( ¼´0µ ¼´1µ 1) and exhibit a single

critical point. Unfortunately, degree distribution that have been

optimized to have a high threshold typically have several

critical points. Such an example is shown in Fig. 3. How

can we extend the scaling law to these cases? Clearly, one

should take into account the effect of all the critical points. It

is tempting to conjecture that the proper generalization of the

scaling law to ensembles with two critical points is

Once all the above components have been integrated, we can perform finite-length optimization.
We note that there are many other open challenges and interesting opportunities. First, it would be very pleasing to find an analytic solution to the covariance evolution set of differential equations in the same way as the solution found by Luby et. al. [3] for the density evolution equations.
Next, the scaling approach makes it possible to investigate the speed of convergence to the promised threshold. It is intuitively clear that distributions with large degrees have a slower such convergence as degree distributions with only small degrees. The scaling methodology is the right setting to put those intuitive notions onto a sound analytic basis.
Finally, the biggest open challenge is without doubt the extension to general binary memoryless channels (AWGN, BSC,...).
