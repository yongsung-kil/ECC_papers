IEEE 2005 CUSTOM INTEGRATED CIRCUITS CONFERENCE

An 80-Mb/s 0.18-Pm CMOS Analog Min-Sum Iterative Decoder for a (32,8,10) LDPC Code

Saied Hemati, Amir H. Banihashemi, and Calvin Plett Carleton University, 1125 Colonel By Dr., Ottawa, Canada, {hemati, ahashemi}@sce.carleton.ca and cp@doe.carleton.ca

Abstract- Analog current-mode circuits are presented for implementing min-sum (MS) iterative decoders, which are used for decoding low-density parity-check (LDPC) codes and turbo codes. While previously reported analog decoders rely on the exponential characteristics of bipolar or subthreshold MOS transistors, proposed circuits can also be used for designing strongly inverted CMOS analog decoders. A proof-of-concept 80-Mb/s CMOS MS decoder is designed and fabricated that consumes the least reported energy per bit and operates at least six times faster than previously reported CMOS analog decoders.
I. INTRODUCTION
Analog continuous-time iterative decoding was proposed a few years ago to improve the power/speed ratio of the chips that decode low-density parity-check (LDPC) codes and turbo codes. Traditionally, digital discrete-time circuits are used for implementing decoders for these capacity achieving codes but
power consumption can be prohibitively high [1]. Further-
more, in [2] it was shown that analog continuous-time decoding has different dynamics than that of conventional digital discrete-time decoding and its steady-state error correcting performance surpasses that of a conventional digital discretetime iterative decoder. It has been shown that belief propagation (BP) decoding algorithm can be implemented by analog
BiCMOS or subthreshold CMOS circuits [3]-[7]. BiCMOS circuits are costly and subthreshold circuits are rather slow. In
[8]-[9] it was shown that strongly inverted CMOS transistors can be used for implementing affordable high-speed analog min-sum (MS) decoders and the functionality of the proposed circuits was verified by circuit simulation.
In this paper, we present measurement results for improved
versions of the circuits in [9]. We also show that major blocks in an analog min-sum iterative decoder can be implemented by using current-mirrors. This implies that our new circuits can be implemented in state-of-the-art BiCMOS or CMOS fabrication technologies. Proposed approach was successfully used for designing and fabricating an analog MS decoder for a (n,k,dmin)=(32,8,10) LDPC code in 0.18-Pm CMOS technology. This decoder chip consumes 5-mW from a single 1.8-V power supply at 80-Mb/s throughput.

This figure also shows that in a current mirror, a suitable biasing voltage (Vbias) corresponding to the input current is generated that is used for regenerating the input current (Iin) at the output. It is worth noting that n-type (p-type) current mirrors can be used for positive (negative) currents. Since messages could be positive or negative, we use both types of current mirrors simultaneously in our blocks. We call this combination a current buffer.
Fig. 3(a) shows a current buffer and demonstrates how it can duplicate currents. This figure also shows that a leakage current (Ileak) flows through the loop when we construct a current buffer. While this leakage current does not degrade the accuracy of the current buffer, it increases the power consumption. Therefore, it is desirable to minimize this current by choosing proper p-type and n-type current mirrors that can not be on at the same time for the given supply voltage. However, when the input current is zero, this leakage current keeps transistors out of the cutoff region and thus increases the speed of the circuit. To improve the accuracy of the current buffers, it is possible to use cascode current mirrors for making a current buffer. In the fabricated decoder chip in 0.18-Pm CMOS technology, current buffers shown in Fig. 3(b) and Fig. 3(c) are used and width-length ratio for p-type transistors and n-type transistors are 2Pm/0.4Pm and 1Pm/1Pm, respectively. IV. BASIC MODULES OF PARITY-CHECK NODES

In this section, the proposed approach for implementing parity-check nodes is presented. As an example, in Fig. 1 that has twenty four similar 4-input parity-check nodes, we focus on implementation of the parity-check node #1. The operation in a parity-check node is given by (2) and Fig. 5 shows the necessary modules for implementing this node. First, sign and magnitude of all incoming messages are extracted in RTAS (real to absolute value and sign converter) modules. Then the magnitude of each outgoing message to each variable node is computed by finding the minimum value among the magnitudes of all the received messages from other variable nodes that are connected to the parity-check node. This operation is performed in the minimum winner-take-all (WTA) modules.
The sign of the outgoing message to each variable node is obtained by multiplying the sign of the messages that have been received from other variable nodes. In hardware implementation, it is simpler to work with a sign bit that represents the polarity of the message. A. Implementation of RTAS Modules
Fig. 6 illustrates the RTAS module. Fig. 6(a) shows a current rectifier, the output of which is always a positive current, regardless of the polarity of its incoming current. This circuit exploits the fact that n-type current mirrors can only be used for positive currents and p-type current mirrors can only be used for negative currents. Fig. 6(b) shows a sign extractor. In this circuit if Iin is positive, n-type current mirror will be active and it will discharge the input capacitance (Cin) of the NOT gate and the output will be zero. Similarly if Iin is negative, the output will be one.
B. Implementation of WTA Modules
WTA circuits consist of a few cells that are competing with each other in duplicating their input at the output. This implies that there should be some kind of mirroring in these circuits. By exploiting this idea and by using current mirrors, WTA circuits can be devised. Fig. 7(a) shows how a min WTA can be designed by minor modification to the current mirror of Fig. 2(c). Diodes in this figure guarantee that the feedback path is closed only for the minimum input current that generates maximum Vout. In real circuits, there are better ways than using diodes to implement a WTA circuit. Fig. 7(b) shows a min WTA circuit that is based on a high-swing cascode current mirror and was used in our analog decoder.
C. Implementation of ASTR Modules
The structure of the ASTR module is illustrated in Fig. 8. Input current is received from the WTA module and the sign is generated in XOR gates and these two are combined together to generate the outgoing message of the parity-check node.

It should however be noted that for higher signal to noise ratios, the performance of the analog decoding chip is deteriorated compared to the ideal analog decoding (SR curves). Fig. 11 shows the error rate performance of the analog MS decoder chip for different throughputs. It is observed that by increasing the throughput, the coding gain is reduced and eventually at 80-Mb/s the loss in coding gain is about 2dB at BER of 10-3 compared to the steady-state curve (320-kb/s). The average power consumption of this chip at 80-Mb/s is about 5-mW. This chip is the first analog MS decoder and also the first successful implementation of an analog decoder for an LDPC code. Other reported CMOS analog decoder chips use BP decoding algorithm and decode turbo codes [5], [7] or the (8,4,4) Hamming code [6].
For analog decoders in [5],[6],[7] that use older CMOS technologies (in [5],[7] 0.35-Pm and in [6] 0.5-Pm) and bias transistors in weak or medium inversion, power–speed ratio is 185-mW/13.3-Mb/s=13.9-nJ/b, 1-mW/2-Mb/s= 500-pJ/b (16PW/20-kb/s=800-pJ/b), and 10.3-mW/2-Mb/s=5.1-nJ/b, respectively. In our chip, this ratio is only 5-mW/80-Mb/s=60pJ/b. Thus, not only is this analog chip the fastest reported CMOS analog iterative decoder, but also it consumes the least amount of energy per bit. It should be noted that part of this improvement is due to utilizing a better fabrication technology.

VI. CONCLUSION
In this paper, current-mode processing modules for implementing analog min-sum iterative decoders were presented that are designed based on current mirrors. Therefore, in any fabrication technology, CMOS or BiCMOS, that accurate current mirrors can be implemented, we expect the proposed approach can be used for implementing analog MS decoders. A proof-of-concept analog min-sum decoder chip was designed and fabricated. Error correcting performance of this chip in steady state and in the low signal to noise ratio region is close to simulation results based on ideal continuous-time iterative
decoding [2] and surpasses that of conventional discrete-time decoding.
