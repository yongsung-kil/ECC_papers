High-Rate Quasi-Cyclic Low-Density Parity-Check Codes Derived from Finite Affine Planes
Norifumi Kamiya Internet Systems Research Laboratories, NEC Corporation
Kawasaki, Kanagawa 211-8666, Japan E-mail: kamiya@bc.jp.nec.com

Abstract— This paper shows that several attractive classes of quasi-cyclic (QC) low-density parity-check (LDPC) codes can be obtained from affine planes over finite fields. One class of these consists of duals of one-generator QC codes. Here, for codes contained in this class, the exact minimum-distance and a lower bound on the multiplicity of the minimum-weight codewords are presented. It is shown that the lower bound on the multiplicity provides a very accurate indication of the bit error performance at moderate and high signal-to-noise ratios, and thus error-floor behavior can be easily predicted. Also discussed is a class consisting of codes from circulant permutation matrices. An explicit formula for the rank of the parity-check matrix is presented for these codes. Furthermore, it is shown that each of these codes can be identified as a code constructed from a constacyclic maximum distance separable (MDS) code in a similar manner to the RS-based LDPC codes presented by Chen et al. and Djurdjevic et al. Experimental results show that a number of high rate QC-LDPC codes with excellent error performance are contained in these classes.

I. INTRODUCTION

Recently, deterministic approaches to constructing LDPC

codes have been developed (cf. [2], [11], [18]). These de-

terministic constructions are based on block matrices with

circulant matrices as blocks and on such combinatorial designs

as affine and projective geometries. Codes constructed using

deterministic approaches will be quasi-cyclic (QC), and hence

they will be amenable to low-complexity encoding [2], [11]

and to efficient implementation of iterative decoding algo-

rithms. Moreover, for high rates and moderate code lengths,

they are able to outperform randomly constructed codes.

The code construction methods presented in this paper are

deterministic, combinatorial, and based on finite affine planes.

It is well-known that the incidence matrix of points other than

the origin and lines not passing through the origin in an affine

plane AG(2, q) can be put in a circulant matrix of order n = q2 −1. Since n is a composite number (n = ab), by permuting

the columns and rows appropriately, the circulant matrix can

be put in a b × b block matrix of circulant matrices of order a. We denote such a block matrix by H(a,b). It is apparent that

Next, we show how the matrix H(q−1,q+1) relates to a
constacyclic MDS code. The family of constacyclic codes is
a natural generalization of cyclic codes. In general, a linear
code of length N over Fq is said to be constacyclic modulo xN − λ (or λ-constacyclic) for λ ∈ Fq× if it is invariant under the transformation σλ, where σλ(c0, c1, · · · , cN−1) = (λcN−1, c0, c1, · · · , cN−2). An (N, K, d) constacyclic code is called a constacyclic MDS code if d = N − K + 1. More
details on constacyclic MDS codes can be found in [12]. For 0 < u q − 1 and γ α−(q+1) ∈ Fq, we define
a (q + 1) × (q + 1) matrix H(q−1,q+1)(γu) over Fq by H(q−1,q+1)(γu) (fi,j (γu))0 i,j q. The (i, j)-th entry of
the matrix is therefore the value of the polynomial fi,j(x) at x = γu. Now we have the following theorem.
Theorem 5 ([8]): Let C(u) be the q-ary code spanned by the row vectors of H(q−1,q+1)(γu) over Fq. If u = 2l − 1 for some 0 < l s, then C(u) is a (q + 1, 2l, q − 2l + 2) constacyclic MDS code modulo (xq+1 − γu) over Fq.
From Lemma 2 and Theorem 5 it follows that the row space C(1) of H(q−1,q+1)(γ) has (q − 1)(q + 1) codewords of weight q, and thus C(1) is a constant weight code. Each row vector of H(q−1,q+1)(γ) can be regarded as a representative of a uniform class [2] of minimum-weight codewords of C(1). Thus, Ct(q−1,q+1) can be identified as a code constructed from a (q + 1, 2, q) constacyclic MDS code in a manner similar to
the derivation of the RS-based LDPC codes [2], [3].

V. CONCLUSION
In this paper, we have shown that several attractive classes
of QC-LDPC codes can be obtained from affine planes over
finite fields. Among them, we have primarily investigated two
classes. The first is derived from block matrices consisting of
only one row block, and the second is from block matrices
consisting of all zero and circulant permutation matrices. For
codes in the first class, we have presented a lower bound
on the multiplicity of the minimum-weight codewords, which
provides a very accurate indication of the bit error performance
at high SNRs. For codes in the second class, we have presented
an explicit formula for the rank of their parity-check matrices
and have drawn a connection to constacyclic MDS codes.
We have shown through computer simulations that QC-LDPC
codes in these classes exhibit excellent error performance
under a normalized min-sum algorithm.
