RATE-COMPATIBLE PUNCTURING FOR LOW-DENSITY PARITY-CHECK CODES WITH
DUAL-DIAGONAL PARITY STRUCTURE

Eoiyoung Choi, Seung-Bum Suh and Jaehong Kim
Communication & Network Lab. Samsung Advanced Institute of Technology Mt. 14-1, Nongseo-Ri, Giheung-Eup, Yongin-Si, Gyeonggido, Korea 449-712
Email: ugam.choiiasamsung.com

Abstract - We investigate suitable puncturing patterns for circulant sub-matrices based quasi-cyclic low-density parity-check codes whose binary base matrices have dual-diagonal parity structure with a single weight-3 column. The proposed method develops the existing puncturing criteria, and exploits the structure of the parity-check matrix. The performance of the proposed puncturing patterns is quite competitive with the performance of dedicated codes for individual rates.
I. INTRODUCTION
Recently, low-density parity-check (LDPC) codes [1] have emerged as more promising forward error correction (FEC) codes than turbo codes [2] in various fields. In mobile wireless communications, IEEE 802.16e adopted LDPC codes as one of optional FEC codes [3] and TGn Sync and WWiSE, two remaining working groups of IEEE 802.1 In have proposed LDPC codes as the advanced FEC codes [4,
5]. Since the rediscovery of LDPC codes [6], they have
received considerable attention due to several attractive advantages such as Shannon-limit approaching capability, low error floor characteristic [7, 8, 9] and inherently fully parallelizable decoding with the belief propagation (BP) algorithm. However, their randomly constructed large parity-check matrix resulted in serious drawbacks such as high encoding complexity and large memory requirements. To overcome these problems, several algebraic methods of
constructing LDPC codes have been suggested. Especially, well-designed quasi-cyclic (QC) LDPC codes based on
circulant sub-matrices enable an efficient VLSI implementation and show comparable performance to random codes as well [10, 11, 12]. These made LDPC codes not only powerful alternatives to turbo codes, but also attractive for diverse standard activities [3, 4, 5]. If the base parity-check matrices of QC LDPC codes have dualdiagonal parity structure with single weight-3 column under some constraints, the efficient encoding shown in [13] can
be realized in linear time with the complexity proportional

to the codeword length n [3, 4].
On time-varying channel, code rates of FEC codes are
required to be adaptable based on the channel state
information (CSI) in order to maximize throughput. If the
parity bits of higher rate codes in a sequence of rates are embedded in those of lower rate codes, the codes are called
rate-compatible. Additionally, block size should be flexible according to the packet size from medium access control
(MAC) layer. Puncturing a low rate code (mother code) can be utilized to achieve rate-compatibility and block size
flexibility. Punctured codes have the following advantages: 1) they need only one encoder/decoder pair for their mother
codes; 2) they enable us to transmit redundancies progressively for hybrid automatic repeat request (ARQ)
protocols [14]. In [15], it was shown that capacity-approaching
puncturing patterns for irregular LDPC codes exist across a wide range of code rates. These results, however, assumed
very large block size. For practical regular LDPC codes with a few thousands block length, Ha et al. proposed good
puncturing algorithm by introducing two definitions, 1-step recoverable (1 -SR) node and k-step recoverable (k-SR) node
as described in Fig. 1 [16]. A punctured variable node (Pa,) is
called a 1-SR node if it has at least one neighboring check node (called survived check node) whose neighboring variable nodes are all unpunctured except for Pa, which can be extended to k-SR node. A punctured variable node (Pb) is called a k-SR node if it has at least one survived check node
that has at least one (k-l)-SR node and m-SR nodes (0 < m < k-1). On binary erasure channel (BEC), k-SR nodes can be recovered after kth iteration. The proposed algorithm in [16] gives us a set of groups {Gj, G2 ,.G Gn}, where Gk denotes the group of k-SR nodes, guaranteeing only one survived check node for each k-SR node to maximize the size of each Gk. Since a kl-SR node is more reliable than a k2-SR node for 0 < ki < k2 < oo, the punctured LDPC codes are designed to puncture the symbols in lower indexed group first. Although it shows much better performance than random puncturing for arbitrary regular LDPC codes, it cannot exploit more reliable k-SR nodes with multiple

IV. CONCLUSIONS
In this paper, we searched for puncturing criteria through several simulation results and proposed rate-compatible puncturing patterns for a dual-diagonal parity structure with single weight-3 column. This parity structure not only enables efficient encoding but also is very suitable for puncturing by k-SR node concept. We modified existing puncturing criteria in [16] in order to exploit the properties of this parity structure.
The proposed puncturing patterns show better

performance than those by [ 16] and comparable performance to dedicated LDPC codes.
