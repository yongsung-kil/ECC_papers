Construction of LDPC Codes for AWGN and Binary Erasure Channels Based on Finite Fields
Lingqi Zeng, Lan Lan, Ying Y. Tai, Shu Lin and Khaled Abdel-Ghaffar Department of Electrical & Computer Engineering University of California, Davis Davis, CA 95616
Email: lqzeng, squash, yytai, shulin, ghaffar@ece.ucdvis.edu

Abstract— This paper presents a method for constructing structured LDPC codes based on finite fields and affine mappings. The Tanner graphs of these codes have girth at least six. Experimental results show that codes constructed based on this method with iterative decoding perform well on both the AWGN and binary erasure channels. Furthermore, codes constructed based on prime fields are quasi-cyclic.
I. INTRODUCTION
In the late 1950’s and early 1960’s, finite fields were successfully used to construct linear block codes with large minimum distances for hard-decision decoding, such as BoseChaudhuri-Hocquenghem (BCH) codes [1], [2] and ReedSolomon (RS) codes [3]. In this paper, we will show that finite fields can also be successfully used for constructing LDPC codes that perform well with iterative decoding on both the AWGN and binary erasure channels.
À A binary regular LDPC code [4] is defined as the null
space of a sparse parity-check matrix over GF(2) with the following structural properties: (1) each row has constant weight ; (2) each column has constant weight ­; (3) no two rows (or two columns) have more than one 1-component
À in common; and (4) both and ­ are small compared with
the code length. is said to be ´­ µ-regular. Property (3) is referred to as the column-row (RC) constraint. The RCconstraint ensures that: (1) the Tanner graph [5] of the code is free of cycles of length ; and (2) the minimum distance of the code is at least ­ ·½ [6]. An LDPC code is said to be irregular if its parity-check matrix has multiple column weights and/or multiple row weights. A quasi-cyclic (QC) LDPC code is given by the null space of an array of sparse circulants [6], [7], [8].
The performance of an LDPC code with iterative decoding depends on a number of code structural properties. One such structural property is the girth of the code that is defined as the length of the shortest cycle in the code’s Tanner graph. For a code to perform well with iterative decoding, its Tanner graph must not contain too many short cycles. The shortest cycles that affect code performance the most are cycles of length 4. Therefore in code construction, cycles of length 4 must be prevented. Many experimental results show that the error-floor of an LDPC code with iterative decoding very much depends on the minimum distance of the code and hence it is necessary to keep the minimum distance of an LDPC code reasonably

large. For an irregular code with iterative decoding, its error performance also depends on the degree distributions of the variable- and check-nodes of its Tanner graph [9],[10].
Let Î be a set of variable nodes in the Tanner graph Ì of an LDPC code and Ë be a set of check-nodes in Ì that are adjacent to the variable-nodes in Î, i.e., each check-node in Ë is connected to at least one variable-node in Î. Ë is called the neighbor set of Î. A set Î of variable nodes is called a stopping set of the Tanner graph Ì if each check-node in its neighbor set Ë is connected to at least two variable-nodes in Î. The Tanner graph Ì of an LDPC code may contain stopping
sets of various sizes. A stopping set of minimum size is called a minimal stopping set (not unique). The performance of an LDPC code on the binary erasure channel (BEC) with iterative decoding is also determined by the stopping sets of its Tanner graph, especially the size of its minimal stopping set [11]. For good performance, the size of a minimal stopping set should be made as large as possible. It has been proved in [12] that the sizes of minimal stopping sets of Tanner graphs with girth , and are ¾, ­ ·½, and ¾­, respectively. Experimental results show that a code with girth 4 performs poorly on the BEC with iterative decoding and has a very high error-floor. For a ´­ µ-regular code whose parity-check matrix satisfies the RC-constraint, the size of a minimal stopping set in its Tanner graph is at least ­ · ½. So for a ´­ µ-regular LDPC code, to attain large minimum distance and large size of a minimal stopping set in its Tanner graph, one way is to keep the column weight of its parity-check matrix large.

V. CONCLUSION
In this paper, we have presented an algebraic method for constructing a large class of structured LDPC codes based on the additive groups of finite fields and affine permutations. This class contains a subclass of QC-LDPC codes. Experimental results show that codes constructed by this method perform very well on both the AWGN and binary erasure channels. All the codes constructed have very low error-floors. LDPC codes can also be constructed based on the multiplicative groups of finite fields [17].
