Abstract—This paper investigates a Low-Density Parity-Check (LDPC) coded Orthogonal Frequency-Division Multiplexing (OFDM) wireless communication system based on IEEE 802.11a standard. And an initialization algorithm is proposed for the decoding of LDPC coded modulation in OFDM system. The LDPC decoding procedure is simplified without the estimation of channel noise power. Simulation results show that this algorithm is effective and the decoding performance is satisfied when maximum iteration number is 10.
Keywords- IEEE 802.11a standard; LDPC coded
modulation; OFDM system; decoding initialization algorithm
I. INTRODUCTION
Orthogonal Frequency Division Multiplexing (OFDM) is a very attractive technique for high-bit-rate data transmission [1] in frequency selective channel because it can eliminate the intersymbol interference (ISI). And this technique is applied in some Wireless Local Access Networks (WLAN) standards such as IEEE 802.11a in USA [2] and HiperLAN/2 in Europe. On the other hand, Low-Density Parity-Check (LDPC) codes, first introduced by Gallager in 1962, have near Shannon limit performance when decoded using an iteratively probabilistic algorithm [3-5]. In this paper, we studied the LDPC coded OFDM wireless communication system based on IEEE 802.11a standard. And a simplified LDPC decoding initialization algorithm is proposed. Simulation results show that this algorithm is effective and can make decoding rapidly.
The paper is organized as follows. In Section 2, we give a brief overview of LDPC coded OFDM system. In Section 3 we present the decoding initialization of LDPC coded modulation in OFDM system. And then we focus on the simulation results and analysis in section 4. Finally, the
conclusion is made.

The BER performance versus Signal-Noise-Ratio (SNR) of different data rate is shown in Fig. 2(solid curves). In general, when the data rate or the spectral efficiency increases the power loss increases. For example, for 6Mbits/s data rate with 0.3bits/s/Hz spectral efficiency, the BER is 10-6 when SNR is near to 6.2dB. But for 36Mbits/s data rate with 1.8bits/s/Hz spectral efficiency, only when SNR exceeds 16dB the BER can achieve 10-6. In addition, we observe that the scheme of 12Mbits/s with code rate 1/2 and QPSK modulation outperforms the scheme of 9Mbits/s with code rate 3/4 and BPSK modulation. Here the influence to performance by the code rate is larger than that of by the modulation. For instance, when BER is 10-5, 12Mbits/s scheme has a power gain of 2dB
than 9Mbits/s scheme.

IV. SIMULATIONS AND RESULTS
We construct three regular LDPC codes with code length 1536 and variable node degree is 3 because this class of codes can achieve good tradeoff between performance and complexity [4-5]. The code parameters are showed in Table 3. Then we study the performance of the system by applying the proposed initialization algorithm. Simulation channel is an equal power 2-path Rayleigh fading channel and the length of the cyclic prefix is larger than the time delay of the channel. At the same time, the maximum of LDPC decoding is 50.

V. CONCLUSION
The LDPC coded OFDM wireless system is considered in this paper. To obtain the data rate available in IEEE 802.11a standard, we construct three code rate LDPC codes with code length 1536 and variable node degree 3, which balance performance and complexity well. Furthermore we propose a simplified algorithm based on the Euclidean measure to calculate the LDPC decoding initialization messages. Results prove that this initialization algorithm is effective and can fasten the decoding procedure. Performance is satisfied with 10 maximum iterations.
