I. INTRODUCTION
Currently the cellular wireless market of circuit-based networks copes with the wireless Internet market of packetbased networks using IP (Internet ProtocoI) power. One of the most prevailing standards for the broadband wireless Internet market is IEEE 802.16. One of physical layer options is OFDM, OFDMA, and LDPC. The LDPC codes show outstanding performance and maximum throughput based on parallel implementation [I]. Although LDPC codes have such advantages, there is a disadvantage like encoding complexity. The Repeat Accumulate (RA) code [Z] is a simple SCC (repetitioncode and accumulator) which has low encoding.

In this paper, we present a new LDPC code which has a structured parity check matrix. The proposed LDPC code can be represented by a Serially Concatenated Codes (SCCs) with a repetition code and tree codes [3]. Therefore we call it a treeLDPC code hereafter. It means that the encoding complexity of tree-LDPC codes is much less than that of regular LDPC codes. In the application of IEEE 802.16 broadband wireless internet, the input block size is about 500 to 2000 bits where the regular LDPC codes can not show its maximum performance advantage. Therefore we propose the new treeLDPC code with OFDM for IEEE 802.16 application.
