Iterative Reliability-Based Decoding of Linear Block Codes with Adaptive Belief Propagation
Aditi Kothiyal, Oscar Y. Takeshita, Wenyi Jin, and Marc Fossorier

Abstract— In this letter, an iterative decoding algorithm for linear block codes combining reliability-based decoding with adaptive belief propagation decoding is proposed. At each iteration, the soft output values delivered by the adaptive belief propagation algorithm are used as reliability values to perform reduced order reliability-based decoding of the code considered. This approach allows to bridge the gap between the error performance achieved by the lower order reliability-based decoding algorithms which remain sub-optimum, and the maximum likelihood decoding, which is too complex to be implemented for most codes employed in practice. Simulations results for various linear block codes are given and elaborated.
Index Terms— Belief propagation, maximum likelihood decoding, order statistics, linear block codes, iterative soft-decision decoding.
I. INTRODUCTION
G OOD traditional codes such as BCH and Reed-Solomon (RS) codes are defined in a way that does not readily permit a sparse graph representation. The graphs have many short cycles which cause error propagation in belief propagation (BP). Hence if BP is performed over such a code the bit reliabilities would not converge to the a posteriori probabilities (APP). Recently however, an adaptive belief propagation (ABP) algorithm has been proposed [1] wherein the graph of the code is adaptively modified at each iteration to render it suitable to BP. A variant of the sum-product algorithm is then used to update the bit reliabilities. In [2], another ABP algorithm has been proposed with a different bit reliability update than [1]. However, ABP suffers from the disadvantage of having error floor-like characteristics at high signal-tonoise ratios (SNRs). In fact, it has been shown in [3] that at medium to high SNR values, ABP roughly performs like ordered statistics decoding (OSD) of order-1, which achieves near maximum likelihood decoding (MLD) only for very high rate codes like long Hamming codes.
In [4], an iterative reliability-based decoding (RBD) algorithm has been proposed for LDPC codes of medium to long block lengths. This algorithm combines BP with RBD. It has been shown via simulations that an improvement in word error rate (WER) performance for medium length LDPC codes is obtained even in the error floor region. The author chooses OSD of orders 0 and 1 to keep manageable complexity.

In this paper, we use the motivation obtained from [1], [2] and [4] and propose a decoding algorithm for linear block codes which combines ABP with RBD like OSD and box and match algorithm (BMA). Each iteration of OSD or BMA is completed by a bit reliability update using ABP. The updated bit reliabilities are treated as the input reliability values for OSD/BMA. With each iteration the log likelihood ratios (LLRs) of the bits converge towards the a posteriori LLRs. Our hope is that as the bit reliabilities converge toward the direction of the optimal codeword, the number of errors in the most reliable basis (MRB) decreases and it is possible for a smaller order OSD or BMA to correct them as compared to the order that would be required when only the channel information is used.
We propose a two-step algorithm as follows. Each iteration of the decoding begins with a RBD algorithm. In the first part of the paper we employ OSD [5] and in the latter part we use BMA with iterative information set reduction (IISR) called IT-BMA [6], [7]. Sufficient conditions [8] are applied to test whether the optimum codeword has been found. If not, then an iteration of ABP is performed to update the reliability values of the bits. Now using the updated reliabilities a new MRB is constructed and OSD/IT-BMA is repeated over the new MRB. In order to decrease the decoding complexity, phase-0 and phase-1 [4] reprocessing of OSD/BMA algorithm is chosen. Since the MRB is already computed in OSD/BMA, determining the least reliable basis (LRB) requires no additional complexity and O(K) candidate codewords are processed at each iteration. This approach allows us to achieve the performance of higher order OSD/BMA using an iterative OSD(1)/BMA(1) with updated MRB at each iteration. ABP serves as the tool which provides the updated MRB.

B. Adaptive Belief Propagation
The first step at each iteration m of ABP is to reduce the parity check matrix of the code H to Hm, such that the columns of Hm corresponding to bits with the least absolute reliability (based on the LLR L(m−1)(ci) from the previous iteration) have weight one. Note that it is always possible to reduce (N − K) columns of H to weight one even though these (N − K) bits may not have the least reliabilities. Let {i1, i2, · · · , iN } denote the indices of the bits corresponding to sorting |Lm(ci)| in ascending order. We start the matrix reduction from the i1th column and proceed sequentially down the list until (N − K) columns have been reduced to weight one. This construction increases the likelihood that bits in error
are moved to leaves on the graph. In other words, it decreases
the probability that a bit in error participates in any loops.
Thus error propagation is limited. We refer the reader to [2]
for implementation details. It has been shown in [3] that under

V. CONCLUSIONS
In this letter, we have proposed an iterative soft-decision decoding algorithm for linear block codes based on ABP and OSD/BMA. The improvement in performance is obtained with a complexity that scales linearly with the number of iterations performed, as opposed to using higher order OSD/BMA in which case the complexity scales exponentially with the order. Extensions to other RBD algorithms such as that of [11] are straightforward.
