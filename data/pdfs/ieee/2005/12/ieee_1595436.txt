RATE-COMPATIBLE PUNCTURED LOW-DENSITY PARITY-CHECK CODES WITH BIT-INTERLEAVED CODED MODULATION

Jingbo Gao, Manli Zhou Department of Electronics & Information Engineering, Huazhong University of Science and Technology, Wuhan 430074, P. R. China

ABSTRACT
We design bit-interleaved coded modulation (BICM) schemes based on low-density parity-check (LDPC) codes. On time-varying channels, it is necessary to adapt the code rate according to channel conditions. Rate-compatible punctured-code (RCPC) is such an efficient and flexible scheme that it uses a single code which can be punctured in rate compatible fashion across a range of code rate. Therefore, when the capacity rule is used to determine the rate of component LDPC codes, we construct ratecompatible component LDPC codes by carefully selecting degree distributions, followed by optimal parity puncturing. Simulation results show that our system performs within 1 dB above the capacity at bit error rate 10-6 across a range of rates.
1. INTRODUCTION
On bandwidth-constrained channels, it is desired to improve the performance of the communication system without sacrificing bandwidth. A natural way to achieve this target is to device an effective way to jointly design modulation and coding to maximize the minimum Euclidean distance. Using multilevel codes (MLC) with multistage decoding (MSD) is a common approach to achieve this target [1]. Shortly speaking, in MLC scheme, modulation symbols are divided into several levels and are protected separately by corresponding error correcting codes. And in MSD, decoding process starts from the lowest level and then up to the higher levels taking into account the decisions of the levels below it. Although MLC with MSD has been proved to be sufficed to approach the channel capacity under the condition that the length of the code approaches infinity, this scheme has a lot of delays. On the contrary, it was shown that the capacity of Gray-mapped BICM is very close to the channel capacity [2]. Furthermore, BICM scheme uses only a single code. Since it is recognized that powerefficiency of LDPC codes increases greatly with the code length, it might be advantageous to use BICM when such

coding schemes like LDPC codes are used as component codes.
As we all know, the characteristics (or states) of communication channel is actually changed because of the variation of environment. Therefore, it is necessary to adapt the code rate according to channel conditions. Ratecompatible punctured-code (RCPC) [3] is such an efficient and flexible scheme that it uses a single code which can be punctured in rate compatible fashion across a range of code rate. The idea of RCPC has been applied to various codes, such as convolutional codes [3], turbo codes [4] and LDPC codes [5]. Among them, LDPC codes have more freedom in choosing puncturing patterns. In [5], Ha et al. proposed an efficient method to determine good puncturing patterns by linear programming. Moreover, they also show that punctured codes have small performance degradation compared with the threshold.
LDPC codes were original introduced by Gallager in the early 1960's [6] and rediscovered by Mackey and Neal in the late 1990's [7]. Nowadays, LDPC codes are one of the hottest topics in coding theory since they have been shown to achieve reliable transmission at signal-to-noise ratios (SNR) with 0.0046 dB of the Shannon limit on the additive white Gaussian noise (AWGN) channel with relatively low decoding complexity [8]. LDPC codes are ideal candidates for BICM schemes since we can construct LDPC codes with almost arbitrary code rate.
In this paper, we explore the use of LDPC codes as the component codes of BICM schemes designed to approach the channel capacity. The idea RCPC is applied to adjust code rate in such way that the punctured codes keep threshold optimality across a range of rates. The outline of this paper is as follows. In section II, we first introduce the system model and the capacity rule. In section III, we apply Gaussian approximation to find good puncture pattern of the proposed BICM scheme. In section IV, we present simulation results. And the conclusion is presented in section V.

When the message-passing algorithm is used as decoding algorithm for LDPC codes, Richardson et al. proved that threshold exists for LDPC codes using a numerical technique called density evolution [10]. Density evolution enables the determination of the thresholds of a LDPC code ensemble based on the assumption of the Tanner graph is cycle-free. Furthermore, density evolution also enables us to track the evolution of the message distribution to analyze the performance of LDPC codes. However, since using density evolution to calculate thresholds and optimize degree distributions is very complex, a suboptimal but simple and tractable codedesign method using Gaussian approximation was proposed in [11].
In [5], Ha et al. studied puncturing of LDPC codes for AWGN channels. They firstly grouped variable nodes of a bipartite graph according to their edge degrees. Then they

5. CONCLUSION

In this paper we study the code design and optimal puncturing of the component LDPC code of our BICM

scheme for AWGN channel. Simulation results show that the proposed system can achieve close to capacity performance with relatively low complexity. Namely, our proposed puncturing BICM scheme is good in terms of large coded gains. Furthermore, since block fading is a case of AWGN with different instantaneous SNR across different coherence periods, this method can be applied to block fading channel as well.
