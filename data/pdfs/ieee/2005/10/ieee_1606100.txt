1. INTRODUCTION
Low Density Parity Check (LDPC) codes tested by Gallager[1] in the 1960s were too short to approach Shannon capacity, and longer codes were not proposed.
Mackay [2] re-established the LDPC codes in the 1990s. Richardson et al. [3] developed the density evolution algorithm to analyze the degree distribution in asymptotically large random bipartite graphs whose associated bipartite graphs have a tree-like structure in different channel conditions.
Density evolution determines the threshold for infinitely long codes whose bipartite graph is tree-like, however bipartite graphs of finite length codes with no singly connected nodes have cycles which are unavoidable. We know that cycle connectivity is an important parameter in lowering the error floor apart from the length of the cycle, conventional girth conditioning on these non tree-like structures increases the shortest cycles length [4][5]. The method of selectively avoiding harmful cycles and conditioning on a cycles of length shorter than a prescribed length, as proposed by Tao Tian et al.[6], lowers the error floor effectively under the AWGN channel without compromising the irregular structure of parity check matrices. The parity check matrix creation from the present schemes cannot avoid cycles and is hard to implement at encoder-decoder de-

signing. It requires large power and storage space because of its larger size, while the Quasi Circulant matrices, which can be created by the shift registers at the decoder, are comparatively easier to implement and do not contain cycles of length 4. This motivates us to apply the ACE algorithm proposed by Tao Tian et al. [6], on the less complex Quasi Circulant matrices to condition the higher length cycles and lower the error floors under the fading channels. This paper presents the performance of ACE conditioned Quasi Circulant matrices under untested Jakes fading channels. A significant gain of тЙИ 5 dB is observed at bit error rate 10тИТ4 for rate 0.8, (244,1220) ACE QC-LDPC in comparison with Quasi Circulant, irregular and regular matrix (with or without cycle conditioning) under the fading channel model and for rate 0.5, ACE QC-LDPC (5000,10000) the lowest error floor is observed.

Then there is a non-tree like bipartite graph or loop for a stopping set. For cycle conditioning, bipartite graph is given the levels with respect to some variable node v4 [6], which follows the structure according to the parity check matrix, e.g., v4 given in Fig.1. Odd levels are assigned to the constraint nodes, and even levels are assigned to the variable nodes. If a cycle occurs in a support tree, at least one node will appear twice at the same level. To detect a cycle of length 2d in the support tree, expansion of support tree must be done at least up to d levels. In an ACE application, we try to make a node set cycle free, if possible or we try to extend the extrinsic message degree of the cycle according to the threshold.
A variable node set is said to be a cycle-free set if no cycle exists among its constituent variables. [6]
A degree-d variable node whose constraints are {c1, c2, c3, ┬╖ ┬╖ ┬╖ cd} can be conceptually replaced by a cluster of degree-2 nodes, whose constraints are , {c1, c2}, {c2, c3}, ┬╖ ┬╖ ┬╖ {cdтИТ1, cd} respectively. Since the indices of the constraints in a cluster are ordered, any cycle involving the degree-d node is equivalent to a cycle involving some of degree-2 nodes in the cluster replacing the original node. As an example, Fig. 2 shows cycle free variable set for a possible (9,3) code, with cycle clustering for the variable node v1.
Extrinsic Message Degree(EMD) : An extrinsic constraint node of a variable node set is a constraint node that is singly connected to this set. The EMD of a variable-node set is the number of extrinsic constraint nodes of this set and the EMD of a stopping set is zero.[6]
Approximate Cycle EMD (ACE) : The ACE of a length 2d cycle is i(di тИТ 2), where di is the degree of the ith variable in this cycle. ACE of a degree-d variable node is d тИТ 2, and the ACE of any constraint node is zero.[6]
ACE is an upper bound on EMD. For QC-LDPC set of variable nodes with large EMD will require additional concatenation of nodes to become a stopping set. Also the cycle of length four has already been avoided by the circulant design. We have concentrated on the higher length cycle conditioning by increasing the extrinsic message degree that ensures all cycles less than a given length have an EMD greater than a given value. This technique statistically increases the smallest stopping-set size. Cycles whose length are above a threshold are ignored.

The ACE detection algorithm is in context of cycles, and their associated ACE values are explained in a manner that each node will propagate to its descendants only if the path leading to this node has the lowest ACE value that any path to this node has had so far. Initially the path ACEs can be set to тИЮ, so that they are unvisited. A cycle ACE equals the sum of the previously lowest path ACEs to a node and the current path ACE to the node, with the substraction of the doubly counted root and the child ACE. i.e. an ACE value of a cycle formed by connecting two distinct paths from vi to ┬╡t is: ACE = ptemp + p (┬╡t) тИТ ACE (vi) тИТ ACE (┬╡t) where ptemp and p (┬╡t) are the ACEs of the two paths from vi to ┬╡t.

4. CONCLUSION

We have constructed a Regular ACE QC-LDPC code by

applying the ACE algorithm on Regular QC-LDPC according

to the distribution of non-zero positions of a circulant matrix in

QC-LDPC. The code construction complexity of the ACE QC-

LDPC in comparison with the construction of the QC-LDPC,

Irregular codes with or without ACE algorithm, and Regular

codes with or without ACE algorithm is least. Our simula-

tion results show ACE QC-LDPC, for rate RтЙе

cycle conditioning can lead to a poor performance of the ACE-

QC-LDPC codes. For (244,1220) ACE QC-LDPC 5 dB gain is achieved at BER of 10тИТ4 with the lowest error floor and
