Design of 4-cycles Free Short-length LDPC Codes Based on the Kirkman Triple Systems

Abstract Short-length codes may play a crucial role in delay-sensitive scenarios. Low-density parity-check (LDPC) codes, as a class of Shannon limit approaching codes, perform well with long code length. However, the design of short-length LDPC codes with good performance is still challenging. This paper investigates the construction of short-length LDPC codes based on the combinatorial design theory. Specifically, Kirkman triple systems (KTS), as a type of balanced incomplete block design (BIBD) techniques, are used to construct the parity-check matrices of LDPC codes, which are guaranteed to be 4-cycles free. Some other properties of such designed LDPC codes are also investigated. For instance, a lower bound on the girth, the code rate and minimum code distance. The simulation results show that the designed short-length LDPC codes based on KTS outperform that of randomly constructed ones.
Keywords LDPC codes, Kirkman triple systems, balanced incomplete block design
I. INTRODUCTION
Low-density parity-check (LDPC) codes are a class of linear block codes approaching the Shannon limit in wellknown channel coding techniques [1-3]. After years of development, LDPC codes have become more and more mature, and have been widely as a standard technique in many wireless communication fields, such as 5G mobile communication [4], deep space communication [5], data storage system [6], digital video broadcasting, IEEE 802.16e [7], etc.
A LDPC code can be represented by the parity-check matrix, which determines the length distribution of the cycles on the Tanner graph and determines the coding performance. Therefore, the construction of parity-check matrix is the key to obtain good performance. There are usually two types of construction methods: random construction methods and algebraic or structural construction methods.
The parameter selection of LDPC codes constructed by random method is flexible. If the code length is long enough (say close to or more than 10000), the performance can be very close to the Shannon limit. It is well known that LDPC codes with 4-cycles cannot perform well due to the short loops in belief propagations. For short-length LDPC codes, it is difficult to avoid the 4-cycles in Tanner graph if they are constructed by the random methods. Besides, since it has no definite code structure, the coding complexity is very high. Using algebraic methods to construct LDPC codes can increase the cycle length in Tanner graph, reduce the coding complexity, and perform better when the code length is relatively short (say less than 1000) [8].
Algebraic construction methods use algebraic tools such as finite geometry, finite fields, balanced incomplete block

design (BIBD), etc. We are interested in the construction of LDPC codes based on BIBD. In 2001, S. J. Johnson and S.R. Weller proposed a construction method of regular LDPC codes using Kirkman triple systems [9, 10]. In 2004, B. Vasic and O. Milenkovic constructed LDPC codes suitable for iterative decoding by using combinatorial design theory [11]. In 2006, D. Suquan, Y. Zhixing and S. Jian, et al presented a method to construct regular LDPC codes of short block length based on BIBD [12]. In [13-15], LDPC code design schemes based on combinatorial design method were proposed. These LDPC codes designed based on BIBD are limited by BIBD itself, and their code rate is fixed, so it is difficult to adjust the code rate flexibly according to the application needs. In addition, most of them focused on the long or medium block length LDPC codes.
This paper considers the construction of short-length LDPC codes based on the Kirkman triple systems (KTS) and analyzes the properties of the obtained codes, such as girth and coding rate. In the following context, the LDPC codes designed from KTS is called KTS-LDPC. The main contributions are summarized as follows.
A construction method based on the KTS is proposed to generate short-length LDPC codes, which is shown to be 4-cycles free. Further, a lower bound on the girth of the constructed KTS-LDPC codes is obtained.
A method to adjust the code rate by deleting parallel classes of KTS is presented, which can be used to further improve the performance.
Coding rate and minimum distance properties of the constructed LDPC codes are analyzed and performance is verified by the simulations.
The remainder of this paper is organized as follows. Section II briefly review the background on LDPC codes and presents the BIBD construction process, especially the KTS. Properties of the short-length LDPC codes constructed by KTS are analyzed in Section III. The performance of the constructed codes is verified by simulations presented in Section IV. Section V concludes the paper.

A cycle in Tanner graph is a path that starts from a node and returns to the original node without passing through repeated edges. The length of a cycle is the number of edges in the cycle. The decoding algorithm of LDPC codes can achieve excellent performance only when there is no cycle in Tanner graph. However, the actual Tanner graph inevitably has cycles. The cycle will cause the repeated transmission of decoding information, so that the messages in the decoding process do not meet the independence assumption, thus affecting the performance of iterative decoding algorithms, such as belief propagation decoding algorithm [16]. For long block length LDPC codes, it can be approximately regarded as cycle-free, but for short block length LDPC codes, short cycles tend to appear and will deteriorate the decoding performance, and it is generally required that there should be no short cycle with length of 4 [17].

III. SHORT-LENGTH KTS-LDPC CODES
The construction of binary LDPC codes based on STS or KTS has been proposed in the literatures, though these codes are generally aimed at long LDPC codes. While long codes obtain excellent performance, they also increase the decoding complexity and delay. For applications requiring high realtime performance, short codes have the advantages of low decoding complexity and short decoding delay. Since the block designed based on KTS can be divided into parallel classes and the row and column weights can maintain unchanged while deleting some parallel classes, it has advantage over STS in code rate adjustment. This paper studies the construction of short LDPC codes based on KTS, and analyzes the performance parameters such as girth, minimum code length and minimum code distance.

Note that rates of LDPC codes designed from KTS are fixed. This paper proposes the following method to flexibly adjust the rate by deleting one or more parallel classes of KTS: in order to flexibly adjust the coding rate, the columns corresponding to a parallel class can be deleted from the parity-check matrix at the same time. Since the row weight of each parallel class is 1, the row weight of the deleted paritycheck matrix is still constant, which can further reduce the coding rate if necessary. Since KTS generally contains a large number of parallel classes, the code rate can be adjusted in a large range.
C. Minimum Code Distance
The minimum code distance determines the ability of error correction and detection. Recall that the minimum code distance is equal to the minimum number of columns for which the sum of the column vectors of its parity-check matrix is zero vector. The column weight of the parity-check matrix designed from the incidence matrix of KTS is always 3, and

no two columns contain more than one check node. Therefore, at least 4 columns are required to make the sum of column vectors zero, so the minimum code distance is at least 4.
Note that after deleting some columns of the parity-check matrix by the above method of adjusting the coding rate, the minimum code distance will not be reduced, but will be maintained or increased instead, since the number of columns whose sum is zero vector increases monotonically.

IV. SIMULATION RESULTS
In order to analyze the performance of short length LDPC codes constructed from KTS, a typical simulation experiment has been done as follows and the results are presented in this section.

The LDPC code digital signal is modulated by BPSK under additive white Gaussian noise (AWGN) channel. The decoding algorithm adopts belief propagation decoding algorithm [16, 19], which is also known as sum-product decoding algorithm. The amount of simulation data is 5000 frames, i.e. 5000n bits.

The extremely short block-length (less than 100) LDPC code is suitable for occasions that require high real-time performance, such as the data link of autonomous driving or unmanned aerial vehicle (UAV) system, where short packets, like speed and location, need to be frequently communicated.

V. CONCLUSION In this paper, a class of LDPC codes with short code length constructed based on Kirkman triple systems is studied. The codes constructed in this way are regular LDPC codes and can be encoded efficiently. The required code length to avoid a cycle of a certain length in Tanner graphs is derived, and the lower bound on girth of Tanner graph is proved. The code rate and minimum code distance of this class of codes are analyzed. In addition, a flexible rate adjustment method is proposed by deleting the parallel classes of the parity-check matrix designed from Kirkman triple systems. Experimental results

indicate that the performance of LDPC codes designed via KTS is better than that of random LDPC codes, in the scenario with short code length.
