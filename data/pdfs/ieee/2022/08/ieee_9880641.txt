Abstract—In this paper, a transmission scheme based on Quasi-Cyclic Low-Density Parity-Check (QC-LDPC) codes with an invertible quasi-cyclic Matrix Q is proposed over Additive White Gaussian Noise (AWGN) channel in order to obtain faster convergence of hard-decision decoding. Combined with the soft information from channel, another improved Reliability-based Q (RQ) decoding algorithm is proposed. The new design can also be applied to QC-LDPC codes in IEEE 802.16e Standards. The simulation results show that the proposed scheme with Q or RQ decoding algorithm has better error correction performance and lower average number of iterations than Bit-Flipping (BF) or weighted Bit-Flipping (WBF) decoding algorithm.
Index Terms—QC-LDPC, hard-decision decoding, Q decoding algorithm, RQ decoding algorithm
I. INTRODUCTION
Low-density parity-check (LDPC) codes were first proposed by Gallager in 1962 [1], and then revived by Mackay and Neal [2]. As a kind of linear block codes theoretically approaching Shannon’s limit, it has always been a research focus and adopted by many advanced systems such as satellite communications (DVB [3]), the wireless systems (5G NR [4], WiMAX [5]), data storage [6], etc. The LDPC decoding methods are divided into hard-decision decoding and soft-decision decoding according to different forms of messages transmitted during iteration process. The soft-decision decoding, for example, the Sum-Product Algorithm (SPA) [7] can achieve excellent decoding performance while its implementation complexity is much higher. On the contrary, the hard-decision decoding algorithms such as Bit-Flipping (BF) algorithm are easier to implement but have worse performance. Many variants of BF decoding algorithms are proposed to reduce the performance

gap between BF decoding algorithm and soft-decision algorithms. The weighted Bit-Flipping (WBF) algorithm [8] and the modified weight Bit-Flipping (MWBF) [9] were proposed as a basis category of soft-information-assisted BF algorithms. The gradient-descent Bit-Flipping (GDBF) and noise aided GDBF algorithms in [10], [11] flip bits adaptively according to the entropy. Besides, with the aim of improving the decoding performance or reducing the computational complexity, some multi-stage algorithms are proposed which combine two or more existing LDPC decoding algorithms sequentially [12]– [14].
In the field of code-based cryptography, an efficient decoder of QC-LDPC codes was proposed by Baldi to improve the decryption success probability [15]. This method was denoted as Q-decoder which exploits the correlation between the matrix Q and the error vector to obtain better decoding performance. Inspired by the concept, we propose a more reliable communication transmission scheme based on QCLDPC codes with matrix Q over AWGN channel in this paper. Through adding matrix Q in the encoding process, the parity check matrix is changed from H to H · Q, and the check and variable degree profile is also changed. Actually the BF decoding algorithm is more efficent for LDPC codes with a denser variable degree profile. The proposed scheme with Q decoding algorithm, a well-performed hard-decision decoding algorithm, can obtain a better error correction performance than the classical transmission scheme with BF decoding algorithm. In addition, an improved Reliability-based Q (RQ) decoding algorithm is proposed in this paper. Assisted by soft information, RQ decoding algorithm determines the final positions of error bits more accurately. Since most calculations of RQ decoding algorithm are still over the integer domain, compared with Q decoding algorithm, only few extra compu-

The equation e = e · QT can be written as the following form: e = j∈p(e) qj, where qj denotes the j-th row of QT and p(e) denotes the sequence of 1’s positions in error vector e. The equation above shows that e is obtained by adding some rows (where the corresponding position in vector e is 1) of matrix QT . With both very sparse matrix Q and vector e, the j-th row of matrix QT will have stronger correlation with e if j ∈ p(e), which means the value of R(t) at the same position is larger. Corresponding to vector e, the values are more likely to be 1 in these positions.
In BF decoding algorithm, whether to flip a bit or not is decided by the number of unsatisfied parity-check equations. These n variable nodes are computed independently, so that the elements between the flipping reference vector are independent and unconnected. In contrast, elements of R(t), the flipping reference vector of Q decoding algorithm, are correlated with each other. Since e consists of the elements of certain columns in matrix Q, we estimate the position of 1s in error vector e based on the correlation of e and Q. Such an estimation is more accurate than separately analyzing the error likelihood of each variable node in BF.
III. IMPROVED DECODING ALGORITHM OF QC-LDPC CODES WITH MATRIX Q
To further improve the performance of Q decoding algorithm, this section introduces a Reliability-based Q decoding algorithm, called RQ decoding algorithm, which introduces the reliabilities of received real value sequence into decoding process.
Assume that the received real value sequence, as described in Section II, is y = [y1, y2, . . . , yN ]. For AWGN channel, a simple measure of reliability of a received symbol yi is its magnitude, |yi|. The larger the magnitude |yi| is, the larger the reliability of its corresponding hard-decision bit has. Based on above principle, the bits with smaller magnitudes are selected as the candidate flipping bits in RQ decoding algorithm. A ratio η (0 ≤ η ≤ 1) is defined to determine the size of candidate flipping set which is denoted as ζ. For example, if η = 1/10, it means the first n/10 bits with lowest reliabilities are selected as the candidate flipping bits. If η = 0, the RQ decoding algorithm becomes Q decoding algorithm described in Section II. After initially obtaining the set ζ in RQ decoding algorithm, another set of flipping positions called ξ(t) will be obtained from the t-th iteration of Q decoding algorithm. In RQ decoding algorithm, the elements of a error vector are finally flipped in each iteration according to the set Γ(t), which is the intersection of set ζ and set ξ(t). Γ(t) can be achieved only if the intersection of set ζ and ξ(t) exists. If not, the set ξ(t) will be selected as the final flipping set. Algorithm 2 describes how RQ decoding algorithm works.
Note that RQ decoding algorithm possesses the same decoding steps as Q decoding algorithm in Section II, except

that ξ(t) in Step 3 of Algorithm 1 is replaced with Γ(t) in Algorithm 2. Through introducing some soft information detected from received real value sequence, RQ decoding algorithm generates a set of candidate flipping bits before implementing Q decoding algorithm. With the assistance of candidate set, RQ decoding algorithm takes advantage of soft information, and adds only a few operations to perform better.

V. CONCLUSION
In this paper, an emergent hard-decision algorithm called Q decoding algorithm is applied to a newly proposed QCLDPC codes based transmission scheme, and an improved RQ decoding algorithm is proposed to further improve the error correction performance. Through introducing a quasicyclic matrix Q in the encoding process, the changed parity

check matrix is more suitable for decoding based on bitflipping. In the decoding process, the proposed transmission scheme with Q decoding algorithm expands the vector e into a new error vector e by e = e · QT. The expansion enhances the connection between elements of flipping reference vector, which provides significant decoding performance gain against classical transmission scheme with BF decoding algorithm. Based on Q decoding algorithm, RQ decoding algorithm locates more accurate positions for flipping according to combining with soft information. The numerical results demonstrate that the proposed transmission scheme with Q decoding algorithm has lower BER than classical transmission scheme with BF decoding algorithm, and RQ decoding algorithm in the proposed scheme also performs better than WBF decoding algorithm, especially at high SNRs. The newly designed transmission scheme is still valid for QCLDPC codes structured by IEEE 802.16e standards. Besides, the proposed RQ decoding algorithm can achieve a better performance than Q decoding algorithm at the cost of an extra small amount of computation. MWBF decoding algorithm has better error correction performance than WBF decoding algorithm by considering both the check constraint messages and the intrinsic message for each bit. Correspondingly, the RQ decoding algorithm can also be further improved by combining the check constraint messages and soft information. The work will be investigated in the future.
