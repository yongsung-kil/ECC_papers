On the woven-like concatenation of LDPC codes

Abstract—In this paper we consider the special construction of concatenated codes. Our scheme consists of outer LDPC code and a specially designed class of inner LDPC code with interleaver between inner and outer codes. An inner code in the proposed construction is obtained using a specific optimization algorithm that allows to minimize SNR for a given target frame error probability. The decoding of the proposed codes is based on the idea of woven codes decoding. Simulation results show that suggested code construction being decoded by woven-based decoding algorithm outperforms low-rate 5G LDPC codes.
Keywords—error correcting codes, concatenated codes, LDPC, error reducing codes, low-rate codes, optimization of codes
I. INTRODUCTION
At present there are different ways to improve the quality of transmission over noisy channel using error correcting codes. Our research combines two known effective approaches: LDPC codes and concatenated codes.
Low-density parity-check codes were suggested by R. W. Gallager in 1962 in [1]. In the beginning, LDPC codes have not found wide application after their discovery due to the insufficient development of computer technology. Research of LDPC codes resumed in the late 1990s after MacKay’s publication [2]. Nowadays some constructions of LDPC codes are used in 5G standard as error correcting codes on channel level of URLLC scenario [3], 5G EMBB standard [4]. Standard IEEE 802.11n/ac (Wi-Fi) [5], standard of digital television DVB S2 [6] are also example of modern standards which utilizes LDPC codes.
It is useful to represent LDPC codes using their Tanner graph representation [7]. Tanner graph is a bipartite graph with left nodes linked with the code-word bits and right nodes linked with the parity-check equations. Such representation is used in the belief propagation (BP) algorithm for decoding of the received word from a channel. This algorithm is softdecision, and it is based on message passing between nodes in the Tanner graph.
There are different approaches to construct LDPC codes. For example, there are methods based on progressive edge growth algorithm (PEG) [8], combined algebraic constructions [9]-[11]. Among many existing methods for constructing LDPC codes, one of the most effective ones is as follows: small core matrix called a protograph, that can be found by exhaustive search or by applying density-evolution [12], is expanded by substitution of each non-zero position in protograph with a permutation matrix or circulant. For example, ACE algorithm [13] could be used to expand protograph. As a result, full parity-check matrix of LDPC code is obtained.
The concept of concatenated codes was introduced in [14]. There are several approaches to construct such codes from a given set of codes. Every approach differs in the choice of components of the entire scheme and the type of concatenation which is used. In serial concatenation [15] information is

encoded successively, wherein channel, inner coder and decoder form a superchannel, whereas parallel concatenation technique applies two or more encoders separately to the information vector and then combines obtained parity-checks with initial information word. Furthermore, permutation is applied to the outer codeword before second encoding in parallel concatenation. The parallel concatenation method was suggested independently by Nilson et al in [16] and Benedetto et al in [17].
One of the most important issues during designing of concatenated codes is to choose appropriate code components of concatenated construction. There are some methods which use different modern codes as polar codes and convolutional codes as components of concatenated construction. In paper [18] authors use LDPC code as outer code and polar code as inner code. Such a method has limited performance due to the BP decoder of polar code. The papers [19]-[23] use the same approach.
In this paper we propose a new construction of concatenated codes, based on two accurately chosen LDPC codes. We examine scheme with serial concatenation, but it inherits the idea of permutation from parallel concatenation for both encoding and decoding. The key difference between previous results and our one is the usage of inner code not for forward error correction, but for error reducing. We use the inner code for improvement channel for the decoder of outer code. The most suitable candidates for error reducing codes are convolutional and LDPC codes. However, the convolutional codes have no enough parameters for optimization in contradistinction to LDPC codes. Thus, inner LDPC in our construction is used for error reducing and outer for error correction. We depict suitable encoding and decoding schemes and describe special algorithm to improve the inner LDPC with fixed outer code. Finally, we demonstrate simulation results to compare the performance of our scheme and LDPC codes from 5G standard.

III. OPTIMIZATION ALGORITHM FOR INNER ENCODER
In our concatenated system we use special error reducing inner LDPC code. First, we describe optimization algorithm, that gives the protograph of this inner code. As a result, the expanding version of this protograph is the parity-check matrix of error reducing inner LDPC code. We describe the properties of such codes later.
A. Optimization algoritm for inner code
The main goal of the optimization procedure is to get the error reducing inner LDPC code that gives the lowest possible signal-to-noise ratio (SNR) region (of a given width) on which the whole concatenated code has target frame error rate (FER).
The inputs of the optimization algorithm are:
• initial core matrix/protograph 4 - , (for our purposed we choose the protograph of 5G LDPC with required rate as initial protograph.)
• fixed outer LDPC code
• target FER value or region
• initial SNR range
The output of the optimization algorithm is a new protograph of inner LDPC code that being decoded gives target FER for lowest found SNR.
Proposed algorithm is iterative, it changes the initial core parity-check matrix of inner LDPC code by flipping some positions in it at each iteration to observe superiorly results of the entire concatenated code checking this by simulation in the AWGN channel. The positions of the protograph to be changed are chosen randomly and after each flip the observed protograph of the inner code is checked for performance together with some fixed outer LDPC code and random or fixed permutation. The SNR(target FER) characteristic is compared to the one of the previous best result and, if it is lower, the current protogaph becomes a new best protograph 4 5, for the algorithm on the next iterations. The outer code of the concatenated scheme is fixed, and it does not change during the optimization process.

B. Error reducing LDPC codes Inner code in our concatenated construction is error
reducing LDPC code that improves channel in terms of BER. Furthermore, such code has a very poor FER characteristic on almost full SNR range. This code allows to improve channel for outer decoder by for decreasing a number of bit errors. As a result, due to improvement of the superchannel, the outer LDPC code is able to decode the received information.
IV. SIMULATION RESULTS
In this section we demonstrate simulation results which prove the effectiveness of the application of our code construction. We use the following simulation setup:
• AWGN channel and BPSK modulation
• 10 iterations of iterative decoder of the concatenated code construction and 5 BP iterations for decoders of inner and outer codes. Thus, the total number of BP iterations is 50 for inner and outer codes.
• ACE algorithm serves as a lifting method for outer code and lifting algorithm from [27] is used for inner codes
We consider the following parameters of the proposed concatenated construction:
• N=4420, R≈1/4, Rinner≈0.85, Router≈0.29
• N=8840, R≈1/4, Rinner≈0.85, Router≈0.29
To compare our coding method with existing solutions we use 5G NR LDPC codes of rate R=1/4 and lengths N=5120, N=8000. We utilize BP-50 decoder for this code.
We present simulation results for LDPC (5120,1280), R=1/4 and proposed scheme with length N=4420, R≈1/4 in Fig. 7.

Fig. 7. Simulation results for N=4420 and R≈1/4
As we can see, the error probability per frame of the suggested concatenated code is lower than the error probability per frame of LDPC code from the 5G standard for the full SNR range. The gap is approximately 0.2 dB at FER level 0.01 that was our target performance. The important feature of the proposed concatenated code is the sharp slope of the FER characteristic. It was expected for this type of concatenation. In contrast to the actual construction of LDPC from 5G standard, our concatenated code has a sharper slope due to the significant minimum distance, that for serial concatenation is equal to the product of minimum distances of component codes.
Now let us present the simulation results of the concatenated code scheme with length 8840 and compare this with LDPC (8000,4000). These results are demonstrated in Fig. 8.
Fig. 8. Simulation results for N=8000 and R≈1/4
It can be found that the difference between the concatenated codes and LDPC codes is not so noticeable as for lower length. The proposed construction begins to outperform LDPC code at FER level 0.001. We can note the sharp slope of the FER compared to the LDPC code in this case as well, which is inherent in serial concatenation.

V. CONCLUSION
In this paper we considered new concatenated code construction that consists of outer LDPC code and inner error reducing LDPC code. The usage of inner code with bit-level reducing property refines the transmission in the sense of decrease the probability of error per frame. We proposed optimization algorithm that allows to obtain protograph of error reducing inner code with target FER on the lowest possible SNR. We also demonstrated encoding and decoding algorithms of our concatenated construction. The described decoder is based on one of the decoders of woven codes. The comparison of the FER characteristics of our concatenated scheme and LDPC from the 5G standard verify the effectiveness of usage of the proposed method in transmission systems.
