Deep Unfolding-Aided Sum-Product Algorithm
for Error Correction of CRC Coded Short Message
Qilin Zhang∗, Shinsuke Ibi∗, Takumi Takahashi†, and Hisato Iwai∗ ∗ Graduate School of Science and Engineering, Doshisha University, Kyoto, Japan
E-mail: ctwg0301@mail4.doshisha.ac.jp, {sibi, iwai}@mail.doshisha.ac.jp † Graduate School of Engineering, Osaka University, Osaka, Japan E-mail: takahashi@comm.eng.osaka-u.ac.jp

AbstractŮThis paper proposes a deep unfoldingaided sum-product algorithm (SPA) for error correction decoding of cyclic redundancy check (CRC) coded short message. SPA is a practical decoding algorithm for linear codes without requiring enormous computational complexity. However, if the SPA is used as it is for CRC codes, belief correlation and outliers will be induced in the iterative decoding process, resulting in lousy correction capability. To compensate for this drawback, we design a SPA-based decoding process for CRC code that incorporates a data-driven design based on deep learning and learning optimization of internal trainable parameters. Considering the operation principle of soft-decision decoder, a novel loss function based on a weighted average of negentropy, which is a key measure to evaluate the Gaussianity, and BCE of the decoder output is proposed. Numerical results show that the proposed algorithm improves the bit error rate (BER) performance with deep unfolding and negentropy-aware loss function.
I. Introduction
The error correction capability of low-density parity check (LDPC) codes has proved to be capable of approaching the Shannon limit with the aid of decoding methods based on sum-product algorithm (SPA), a kind of belief propagation (BP) [1]. Nevertheless, LDPC codes demonstrate high error correction capability only when the code length is sufficiently long, and the capability will signiĄcantly reduce when the code length is short. The Internet of things (IoT), which plays an important role in the next-generation wireless communication infrastructure, tends to use smaller packet sizes. Therefore, high error correction capability cannot be expected even if LDPC code is utilized for IoT communication. Some conĄgurations, which deliberately do not use error-correcting codes, are sometimes used as standard, such as Bluetooth low energy (BLE) 4.0.
Even in such conĄgurations, the cyclic redundancy check (CRC) code is typically utilized to detect packet error events. Since parity bits of about 24 bits are appended to the information bit sequence of the CRC code, the coding rate will be high when the length of the information bit sequence is very long. Therefore, it cannot be expected to function as an error-correcting code in general. However, on the premise that IoT communication has a small packet

size, it is possible to use the CRC error detecting code as an error correction code. For example, if the number of information bits is 48 bits, the coding rate of the CRC with 24 parity bits is 2/3, which means that the coding rate of the error correction code is relatively low. In this case, high error correction capability can be expected.
Based on the above, this paper investigates the use of CRC code as an error correction code. SPA is a practical decoding algorithm for linear codes without requiring enormous computational complexity. However, the CRC code contains many short cycles in the parity check matrix if the SPA is used as it is. In this case, belief correlation and outliers will be induced in the iterative decoding process, resulting in lousy correction capability. To address this issue, this paper aims to design an SPA-based decoding algorithm for CRC code that incorporates a data-driven design based on deep learning and learning optimization of internal trainable parameters [2]. Appropriate adjustment of trainable weight parameters is effective in mitigating the harmful effects of belief correlation and outliers.
In recent years, data-driven algorithm design based on deep learning techniques has rapidly attracted attention in the Ąeld of wireless communications [3], [4]. In particular, many investigations using deep unfolding (DU) have been reported in statistical signal processing, where analytical mathematical models are given with relatively high accuracy [5], [6]. DU is the application of the error backpropagation and stochastic gradient methods used in conventional deep learning to a signal Ćow graph similar to a feed-forward neural network (FFNN), given by expanding the iterative structure of the algorithm. DU optimizes the trainable parameters embedded in the signal Ćow graph so as to minimize designed cost functions. If the trainable parameters can be successfully tuned, the algorithm can be designed while enjoying the advantages of both modeldriven and data-driven approaches [5], [7].
Since the forward error correction (FEC) is usually optimized for additive white Gaussian noise (AWGN) channels, their error correction capability is maximized when the log-likelihood ratio (LLR) input to the softdecision decoder follows a Gaussian distribution [8]. That is, the proposed SPA decoder should be trained with a loss function that can enhance the Gaussianity of the output

B. Sum-Product Algorithm
In SPA, the decoding proceeds by exchanging belief messages calculated by fundamental variable and function node operations over the Tanner graph of the parity check matrix H ∈ ¶0, 1♢M×N , iteratively, where M = N − L is the number of parity checks. An example of Tanner graphs is depicted in Fig. 2. In the graph, N variable nodes and M function nodes are connected by edges deĄned by H [10]. Let hn,m be the (n, m) element of H. If hn,m = 1, there exists the edge between the n-th variable and the m-th check nodes.
As shown in Fig. 3, βn(i→) m denotes the belief message sent from the variable node cn to the function node um along the edge at the i-th iteration step. On the other hand, αm (i)→n denotes the message sent from the function node um to the variable node cn at the i-th iteration step. Let Am be the set of column indices n in the m-th row of H with hn,m = 1. Similarly, Bn be the set of row numbers m in the n-th column with hn,m = 1. The exchange of messages shall be iterated at most I times.

In a Tanner graph, a path starting from a variable node and returning to the same node is called a loop (or cycle). When a Tanner graph contains short loops, the decoding performance by SPA is degraded. In particular, loops of length 4 seriously affect decoding performances, so it is desirable to construct the parity check matrix so that such short loops are avoided as much as possible [11].
As shown in Fig. 2, if a matrix is expanded into a Tanner graph, there are several loops of length 4, for example, indicated by the bold line between the variable and function nodes. When such short loops are included in the graph, the outliers of the belief message are gradually ampliĄed by the nature of the SPA, degrading decoding performance. Therefore, the DU described in the next section is introduced to compensate for SPA and mitigate the effect of outliers.

III. Deep Unfolding-Aided Sum-Product Algorithm
A. Deep Unfolding Model for SPA This section describes DU, which applies deep learning
techniques to SPA. An FFNN is a network in which information is propagated only in one direction from the input side to the output side, as shown in Fig. 4. In addition, the FFNN is divided into three sections: input, intermediate, and output layers, which are composed of multiple nodes connected by edges. The intermediate layer can have multiple layers, and training a network with deep intermediate layers is called deep learning. Each node has an Şactivation function,Ť and edges have Şweights.Ť The output value of each node is calculated by the linearly weighted values of the nodes in the previous layer connected to it via the activation function [12].
Fig. 5 illustrates the signal Ćow graph given by unfolding the iterative structure of SPA, which exchanges message beliefs αn(i→) m and βm(i)→n. Now, let us recall (3) and (4). The variable node operation of (3) has a linearly weighted structure, and the function node operation of (4) can be regarded as an activation function that uses a hyperbolic tangent function. Thus, regarding the function node operation followed by the variable node operation at the i-th iteration step as the node operation of deep FFNN, the similarity can be found between Figs. 4 and 5.

B. Loss Function

The trainable parameters embedded in the deep FFNN are optimized to minimize loss function for reducing bit error rate (BER), improving convergence property of iterative behavior, and so on. Training data sets of the code vector c and received LLR vector λ are prepared in advance, and data-driven tuning of the trainable parameters ζη(i) is performed. Note that the code vector c and corresponding LLR λ are perfectly known at the optimizer in the training phase, whereas c is unknown at the receiver in the test phase.

Error correction is performed by M = 24 bits of CRC parity when the length of information vector L is 16, 40, and 64 bits, respectively. The generator polynomial of the CRC-24 is deĄned by [20]

G = x24 + x10 + x9 + x6 + x4 + x3 + x + 1.

B. Hyper-Parameters of Training Es/N0 and Weight w for DU-Aided SPA
Let training and test Es/N0 be Es/N0 at the training and test phases, respectively. In the training phase, the trainable parameters ζη(i) are tuned by data-driven optimization on the basis of loss functions. On the other hand, in the test phase, BER is evaluated by using the tuned parameters ζη(i).
Fig. 6 characterizes BER performance when given arbitrary hyper-parameters of training Es/N0 and weight w. The test Es/N0 are 3, 4, and 5 dB for L = 16, 40, and 64, respectively, for tuning the BER performance around BER = 10−4. Note that loss functions of w = 0 and w = 1 correspond to BCE and NEG criteria, respectively. According to these results, the error correction capability of DU-aided SPA can be improved by using the trained parameters of training Es/N0 and weight w, which minimize BER in the test phase.
Tab. I summarizes the achievable minimum BER and optimal training Es/N0 and weight w in Fig. 6. In cases of L = 16 and 40, the loss function of NEG achieves smaller BER than BCE. V. Conclusions
In this paper, we designed a DU-aided SPA for error correction decoding of CRC-coded short messages. In the proposed method, the trainable parameters effectively solve outlier problems caused by short cycles in the parity check matrix. In addition, we also designed a loss function, which is given by a weighted average of BCE and negentropy. Computer simulations demonstrate the proposed method can improve the BER performance, and the negentropyaware loss function design is effective, especially in shorter packet sizes.
