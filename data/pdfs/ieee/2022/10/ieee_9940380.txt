Abstract—A communication transmission system with channel coding and deep neural network (DNN)-based decoding is considered. A DNN-based decoding scheme is proposed for reliable transmission. The decoding scheme is accomplished by efficient local decoding at all the neurons and interactions in the input, hidden and output layer. Specifically, firstly, the nonlinear operations at each neuron and the linear operations of the weights and biases at each edge are performed by the local decoding. Secondly, the weights and biases are updated by gradient descent (GD) algorithm, based on the estimated loss value. This process above is performed iteratively until the message sequence has been recovered. Simulation results show that our proposed decoding scheme performs well. Moreover, our decoding scheme performs significantly better than the conventional hard decision.
Index Terms—DNN-based decoding, local decoding, hard decision.
I. INTRODUCTION
Through the development of the times, communication technology has made great development in many fields. Among various communication technologies [1], channel coding technology has always been the focus of attention. Channel coding technologies continue to approach the Shannon limit [2], pushing the communications field past one pinnacle after another.
Nowadays, channel coding is also used all the time around us. Hamming code [3] was proposed by Richard, which is a type of linear block code. It can still be found in data storage [4], information protection [5] and image processing [6]. The low-density parity-check (LDPC) code proposed by Gallager is selected as the standard codes for the fifth generation

enhanced Mobile Broad Band scenario [7]. From the earliest Hamming code to the current LDPC code, the channel coding performance has been greatly improved. However, with the change of channel coding methods, the problems of their higher decoding complexity and longer decoding time have always existed. Finding a decoding method that can solve this problem is an important issue for researchers in the communication field to consider.
In 1958, Frank Rosenblatt proposed a network composed of two layers of neurons, named the perceptron [8]. However, perceptron was proved unable to complete the simple linear inseparable problems such as XOR. Later, with the idea of multilayer perceptrons being put forward and the continuous improvement of computer computing power, neural network [9] came into people’s field of vision. In neural network, traditional shallow models are not good at processing massive data, while deep neural network (DNN) can use more neurons to process data, based on its flexible structure. Therefore, our starting point is to combine DNN and channel decoding.
In this letter, we propose a DNN-based decoding scheme over Additive White Gaussian Noise (AWGN) channel. Specifically, firstly, we divide the signal from the AWGN channel into several equal groups. Each group performs an independent local decoding process. The local decoding is performed in the input layer, hidden layers and output layer of the DNNbased decoding scheme, respectively. There are two types of operations in local decoding: linear and nonlinear. The process of linear operation is a linear superposition of the weights and biases on each edge and the input messages of the neurons. The process of nonlinear operation is to introduce a nonlinear function on the output messages of the neurons. After several local decodings, the loss value is obtained by employing the

The DNN-based decoding scheme is accomplished by efficient local decoding at all the neurons and interactions. During a decoding iteration, each neuron acts once to perform local decoding and updates the message on the edges to each of its adjacent neurons. A decoding iteration starts from the local decoding at the neurons in input layer. Based on the received signal y, each neuron performs a local decoding. This local decoding employs a nonlinear operation by an activation function. Based on these activation function results, a local decoding is performed on each edge, which connects the neurons of the input layer and the first hidden layer. This local decoding employs a linear operation of the weights and biases. A similar process is performed in hidden layers and output layer. For each group, the code length divided by the signal is n, and the message length after decoding is k. In the hard decision, the received codeword is compared with the codebook, and the codeword that gives the minimum Hamming distance is selected. To compare the minimum Hamming distance, the size of the Hamming distance martrix is (1 × 2k). The codebook has 2k rows with n lines per row. Therefore, the total space of hard decision is 2k(n + 1).
In the DNN-based decoding scheme, it does not require a comparison with the Hamming codebook. As n changes, it only corresponds to the weights between the input layer and the first hidden layer. If the number of neurons in the first hidden layer is P , the size of the weight matrix is (P × n). As k changes, it only corresponds to the biases at the output layer. The size of the bias matrix is (1 × k). No extra space is created during the decoding iteration due to the changes of n or k. Thus, the total space of DNN-based decoding scheme is (P n + k), as shown in TABLE III.
IV. SIMULATION RESULT
In this section, we present some simulation results. The training platform of DNN-based decoding scheme is built by Tensorflow [14]. The number of neurons in the input layer of the scheme is 7, the number of layers in the hidden layer is 3, the number of neurons in each hidden layer is 140, and the number of neurons in the output layer is 4. The initial weights and biases of each edge are randomly generated, and the activation function is the Sigmod function.
The DNN-based decoding scheme is affected from different learning rates and different loss functions. To observe the performance variation of the decoder, the simulation analysis is divided into three parts.
A. Effect of Different Learning Rates on BER Performance
Let the η in the GD optimization function be 0.1, 0.08, and 0.01 respectively for simulation. The effects of different learning rates on the BER performance of the DNN-based decoding scheme are plotted in Fig. 4. It shows that as the Eb/N0 increases, the magnitude of the BER varies more and more significantly with the η. When and only when other parameters are constant, the BER is lower if the η = 0.08. Moreover, too large or too small η also has an impact on BER performance. When the η is too small, the process of finding the optimal solution is prolonged and requires a huge number of iterations. When the η is too large, the optimal solution is easily skipped and non-convergence occurs.

We gave an example of DNN-based decoding scheme. Based on the complete process of classical Hamming coding and decoding, we transmitted the signal from AWGN channel to the input layer of the DNN-based decoder. After several local decodings, to measure the number of errors in transmission, the estimated loss value of this iteration was obtained by the MSE loss function. The GD algorithm was used to update the weights and biases based on the loss value. After a large number of iterations, the near-optimal message sequence was recovered. The simulation results show that our proposed scheme is superior to the conventional hard decision.
