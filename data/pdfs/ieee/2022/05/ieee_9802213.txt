A Novel LDPC Construction Scheme for NAND Flash Memory

Hongyuan Li 1, Xiaobo Jiang *2, Zhenghong Yu 1, Wanjun Zheng1 1 Guangdong Polytechnic of Science and Technology, Zhuhai, 519090, China
2 South China University of Technology, Guangzhou, 510641, China
*Corresponding author e-mail: gditlhy@163.com

Abstract—The storage capacity of NAND Flash memory has increased by scaling down to smaller cell size and using multi-level storage technology, but data reliability is degraded by severer retention errors. As adopting a very powerful error-correcting code gradually becomes a strategic demand for the endurance of nowadays NAND Flash memory, Low Density Parity Check (LDPC) codes are recently proposed due to their outstanding error correcting capability. Herein, a novel construction scheme of LDPC for NAND Flash memory is proposed. By using the proposed scheme, a high code-rate, high performance of Bit Error Rate (BER), low error floor Quasi Cyclic Low Density Parity Check (QC-LDPC) code is constructed to meet the needs of NAND Flash memory. In the proposed LDPC construction scheme, iterative cycle elimination technique is introduced to ensure that the checksum matrix is cycle-4 free and has minimal cycle-6, which is beneficial to achive high performance of BER and low error floor for high code-rate LDPC. A diagonal coding structure is used in the QC-LDPC code to achieve linear-time coding and meet the high throughput requirements of NAND Flash memory. Simulation results show that NAND Flash memory can be used more 1800 times for Program/Erase (P/E) cycle by using the proposed QC-LDPC codes compared with Euclidean-Geometry LDPC codes. The error floor of the constructed QC-LDPC codes is below 10−12.
Index Terms—LDPC; NAND flash memory; construction scheme; iteratively cycle elimination
I. INTRODUCTION
NAND Flash memory has been widely adopted as storage devices in mobile devices, embedded systems, and data centers [1]. The advances in process and technology drive the NAND Flash memories’ capacity to grow rapidly. Due to the shrinkage of memory cells, multi-level-cell representation, e.g. the MLC (2bit), TLC (3bit) or even QLC (4bit), and advanced memory architectures such as the 3D NAND, the stored information are even prone to errors while information density grows [2]. For the purpose of overcoming the reliability issues of the NAND Flash memory, it is essential to employ the technology of Error-Correction-Code (ECC). Traditional ECC, e.g., Hamming code, Reed Solomon code, or Bose Chaudhuri Hocquenghem (BCH) code, cannot satisfy the NAND Flash memory [3]. With strong error correction capability through soft decoding, low-density parity-check (LDPC) codes have been applied for the state-of-the-art

NAND Fash memory [1-8]. The high capacity and low cost of NAND Flash memory
bring up lots of strict requirements for LDPC code. Under existing numerous short cycles for high code-rate LDPC, the LDPC code is required to have high performance and low complexity with flexible code rate and code length. The error floor of the LDPC code should also be below 10−12 to 10−15[4]. The current construction schemes of LDPC codes seldom consider the construction of high-performance high code-rate LDPC codes under numerous short cycles.
Current studies on LDPC codes for NAND Flash memory mainly cover the finite-geometry construction schemes[5-6], computer search schemes[7], progressive edge growth (PEG) combined QC-LDPC schemes[8], ACE schemes[9], and other construction schemes that seldom consider the code construction at high code rates, which are unsuitable for NAND Flash memory. New construction schemes need to be provided. References[5-6] adopted a Euclidean geometry (EG)-LDPC code with a 96% code rate to study the LDPC code for NAND Flash memory. EG-LDPC code has no cycle-4 in the checksum matrix but has a QC characteristic. However, the special structure of EG-LDPC makes the column weight heavy under high code-rate. This feature may result in the existence of numerous cycle-6, which significantly influences the performance of the LDPC code. The checksum matrix of the unfilled rank also hinders the application of this LDPC code underfast codes, thus limiting the practical applications. Reference [7] adopted the construction schemes from[10] to construct a BLOCK LDPC code with a four-column weight and 95% code rate for NAND Flash memory channels and error-correction control. The BLOCK LDPC code constructed in this schemes has a QC characteristic and linear-time coding. On the basis of the given parameters and random search by a computer, a group of BLOCK LDPC codes, which arefree from cycle-4,are obtained.The one with the best performance among all BLOCK LDPC codesis selected. However, the search conducted in the beginning stage, as well as the one-by-one comparison, requires considerable calculations and does notensure the optimization performance of the code. Reference [11-12] proposed the progressive edge-growth (PEG) schemes, which is a widely used and effective scheme for constructing large girth random codes. Combined with the

The ACE schemes [14] can effectively decrease the error floor of the LDPC code but has high complexity in realizing a random code; thus, this schemes is impractical. The ACE schemes may also exhibit poor performance under high code-rate because of the introduction of redundant short cycles. The cycle-4 and cycle-6 elimination algorithms in [15]–[17] can construct a QC-LDPC code with a large girth and excellent performance. However, in case of high code-rate, cycle-6 cannot be eliminated. Therefore, cycle-4 and cycle-6 elimination algorithms cannot be used to construct LDPC codes for NAND Flash memory.
This paper proposes a construction scheme for the fast iterative cycle elimination of LDPC codes on the basis of the requirement of NAND Flash memory, particularly the high code-rate requirement. All factors (e.g., low complexity, high performance, and low-error floor) are considered. We also construct a high-performance QC-LDPC code for NAND Flash memory. A random permutation matrix is introduced such that the distribution of non-zero elements in the checksum matrix is random. To optimize the performance of the constructed QC-LDPC codes, the distribution of short cycles is considered by iteratively fixing the shift factor of the permutation matrix to ensure that the checksum matrix is cycle-4 free and has minimal cycle-6. A diagonal coding structure is used in the QC-LDPC code to achieve linear-time coding and satisfy the high throughput requirements of NAND Flash memory. The construction scheme proposed in this paper also yields a flexible code length and code rate.
The structure of this paper is as follows. After the introduction, Section II discusses the requirements of NAND Flash memory for LDPC codes and presents the proposed construction scheme for a high-performance QC-LDPC by using iterative cycle elimination techniques, while Section III presents simulation results of the LDPC code constructed using proposed scheme NAND Flash memory. Section IV concludes this paper.

The main idea of the algorithm is to start from an all-zero shift factor index matrix P1 and fix the values of the elements in P1 continuously to reduce the quantity of short cycles gradually. The values of elements in P1 are roughly adjusted. When the quantity of cycle-4 is reduced to certain degree, the values of elements in P1 are adjusted in a small range until the quantity of cycle-4 becomes zero and the quantity of cycle-6 cannot be less, i.e., reaching the critical threshold value. We can then obtain the final shift factor index matrix P1. The algorithm adjusts the distribution and quantity of cycles by adding one random number, thus increasing the degree of randomness of the distribution of non-zero elements in the checksum matrix.This characteristic not only improves the error-correction performance but also reduces the occurrence of low code and decreases the error floor of the constructed code.

IV. CONCLUSION
Rapid technological development has bring up the need for a high-performance LDPC code for NAND Flash memory. Specific to the numerous short cycle of the LDPC code for NAND Flash memory in case of high code-rate, this study proposes a construction scheme based on the fast iterative cycle elimination techique of the QC-LDPC code to eliminate the influences of short cycles on the BER performance and error floor. Combined with the structuralization of the QC-LDPC code and the random distribution of non-zero elements of the random LDPC code, a diagonal coding structure is used in the QC-LDPC code in this study to achieve low coding latency to meet the high throughput requirements of NAND Flash memory.
The simulation results show that the service life of the constructed QC-LDPC code (68800,66048) is 1,800 times more than that of EG-LDPC(69615,66897) and 5,000 times more than that of BCH code under the same rate. The error floor of the constructed QC-LDPC code is below 10−12.
