Abstract—Controlling small size trapping sets and short cycles can result in LDPC codes with large minimum distance dmin. We prove that short cycles with a chord are the root of several trapping sets and eliminating these cycles increases dmin. We show that the lower bounds on dmin of an LDPC code with chordless short cycles, girth 6 and column weights γ is 2γ. This is a significant improvement compared to the existing bounds γ + 1.
Several exponent matrices of protograph-based LDPC codes with chordless short cycles are proposed for any type of protographs, single-edge and multi-edge. These numerical results as well as simulations show that the removal of short cycles with a chord improves previous results in the literature.
Index Terms—LDPC codes, girth, Tanner graph, elementary trapping set, chordless cycles, minimum distance.
I. INTRODUCTION
Low-density parity-check (LDPC) codes as dominant channel codes for high speed communications have excellent error-correcting performance and high parallelism in decoder implementation [1], [2]. The most practical LDPC codes are protograph-based LDPC codes [3].
A protograph is a small size bipartite graph. The adjacency matrix of the protograph is taken as a base matrix of protograph-based LDPC codes. Since these codes are associated with a base matrix and an exponent matrix, they give a simple description of codes and enable hardwarefriendly implementation. The Tanner graph of a protographbased LDPC code is generated by lifting a protograph, that is copying the protograph several times and then permuting the copied edges originated from the same edge. If we use cyclic permutations for lifting, then the resulting codes are quasi-cyclic (QC) LDPC codes. It is known that the length of the shortest cycles of the Tanner graph, girth, influences code performance. Graphical structures including detrimental trapping sets are known to be key factors of error floor behavior of LDPC codes.
An (a, b) trapping set is a set of a variable nodes in the Tanner graph which induce a subgraph of the Tanner graph with b check nodes of odd degrees and an arbitrary number of even degree check nodes. An (a, b) trapping set is elementary (ETS) if all check nodes are of degree 1 or 2.
The authors were partially funded by Natural Sciences and Engineering Research Council of Canada (NSERC).

Removing trapping sets up to a certain size which are subgraphs of (a, 0) trapping sets, yields a code with improved minimum distance dmin. In [4], it is shown that controlling specific cycles contributes to the removal of harmful trapping sets. In [5]–[7] we used an edge-coloring technique to provide a sufficient condition for an exponent matrix to give QC-LDPC codes with different column weights γ and free of small size trapping sets. In [5], we considered (3, n)-regular algebraicbased QC-LDPC codes with girth 6, column weight γ = 3, row weight n and free of (a, b) ETSs, 4 ≤ a ≤ 5 and b ≤ 2. In [6], the conditions to construct all protograph-based LDPC codes with g = 6 (g = 8) and γ = 4, 5, 6 (γ = 3, 4) and free of small size ETSs are given. Then, in [7], we presented another approach to improve the performance of a QC-LDPC code. We used some concepts from graph theory such as rainbow cycles to show that for large column weights the removal of all 8-cycles but the ones we call rainbow 8-cycle causes the elimination of several small size trapping sets.
Most methods mentioned above and others in the literature to remove harmful trapping sets are applied to fully-connected protographs. There are not many results regarding multi-edge protographs. Moreover, in this area the effect of removing trapping sets on minimum distance of the code is rarely investigated.
In this paper, our approach to improve the performance of an LDPC code and construct an LDPC code with large minimum distance is based on cycles with a chord and chordless cycles. Our main contributions are for QC-LDPC codes with girth 6 and column weight γ.
• We show that short cycles with a chord are the root of several trapping sets and eliminating these cycles increases dmin.
• We prove that avoiding 8-cycles with a chord results in an LDPC code in which the lower bounds on the size of smallest (a, b) ETSs, b < a, are equal to the minimum sizes of ETSs in a girth-8 LDPC code.
• We prove that the lower bound on dmin of QC-LDPC codes with a Tanner graph free of 8-cycles with a chord is increased from γ + 1 to 2γ.
• We provide a new method to control 8-cycles with a chord which is applicable to any exponent matrix.

Definition 4: A cycle is a chordless cycle if outside the cycle

Lemma 2: Let an LDPC code with column weight γ = 4

and girth 6 be given. The Tanner graph is free of an (8, 0) ETS

which contains 6-cycles and its 8-cycles are all chordless.

Proof : We prove that if an ETS with chordless 8-cycles

contains a 6-cycle, then the smallest size is more than 8. Since

b = 0, all variable nodes of the VN graph have degree 4.

Suppose v0 is a variable node connected to four check nodes

c1, c2, c3, c4. Since all check nodes have degree 2, each check node is connected to other variable node. Let vi be connected

to ci. Without loss of generality, suppose v0, v1, v2 are the

variable nodes of a 6-cycle. If a check node connects v1 or

v2 to one of the variable nodes v3, v4, then an 8-cycle-wc is

obtained which is impossible. Thus, there must be variable

nodes v5 and v6 for two different check nodes connected to

v1. If there is a check node between v2 and v5 or between v2 and v6, then we have an 8-cycle-wc with variable nodes

v0, v1, v5, v2 or v0, v1, v6, v2, respectively. Therefore, there is

no check node between v2 and any of the variable nodes

v3, v4, v5, v6. Hence, there must be two variable nodes v7, v8 for two different check nodes connected to v2. This process

guarantees the non-existence of an (8, 0) ETS containing 6-

cycles and chordless 8-cycles.

V. CONCLUSION
We define a cycle with a chord in the Tanner graph of an LDPC code and analyze the impact of removing short cycles with a chord on the lower bound of the size of small size trapping sets and dmin of a code. We prove that without increasing the girth we can improve several important factors of an LDPC code which significantly influence the performance curve of a code. For any protograph, single-edge

or multi-edge, we provide exponent matrices free of these
cycles. The performance of the resulting QC-LDPC codes and
a comparison with the codes containing short cycles with a
chord show the merit of the removal of these cycles in practice.
