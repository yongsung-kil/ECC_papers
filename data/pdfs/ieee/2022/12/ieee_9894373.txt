A Low-Complexity Neural Normalized Min-Sum LDPC Decoding Algorithm Using Tensor-Train Decomposition
Yuanhui Liang, Chan-Tong Lam , Member, IEEE, and Benjamin K. Ng , Member, IEEE

Abstract— Compared with traditional low-density parity-check (LDPC) decoding algorithms, the current model-driven deep learning (DL)-based LDPC decoding algorithms face the disadvantage of high computational complexity. Based on the Neural Normalized Min-Sum (NNMS) algorithm, we propose a lowcomplexity model-driven DL-based LDPC decoding algorithm using Tensor-Train (TT) decomposition and syndrome loss function, called TT-NNMS+ algorithm. Our experiments show that the proposed TT-NNMS+ algorithm is more competitive than the NNMS algorithm in terms of bit error rate (BER) performance, memory requirement and computational complexity.
Index Terms— Model-driven LDPC decoding, tensor-train decomposition, neural normalized min-sum, syndrome loss.
I. INTRODUCTION
I N RECENT years, deep learning (DL) has been applied to channel decoding algorithms. In particular, DL was applied to improve the belief propagation (BP) decoding algorithm for linear codes in [1]. The iterative form of the BP algorithm is unrolled into a neural network form. This method adds training parameters by assigning weight coefficients to the Tanner graph, and then uses the DL network and a loss function to train these weight coefficients. In [2], the authors proposed a neural offset min-sum (NOMS) decoding algorithm, which unfolded the OMS decoding algorithm into the form of a neural network. In [3], the authors extended the unrolling method to the normalized min-sum (NMS) decoding algorithm and proposed a neural normalized min-sum (NNMS) decoding algorithm, which performs 1.5 dB better than the NMS decoding algorithm. Model-driven DL-based decoding algorithms typically use a cross-entropy loss function for parameter optimization. The authors in [4] proposed a syndrome loss-based neural error correction decoder, which achieves better decoding performance than decoders based on the crossentropy loss function. The syndrome loss-based decoding algorithm can be trained without requiring the knowledge of the transmitted codewords, which can be a useful tool for online adaptation to the changing channel conditions.
By assigning weights to the Tannar graph, the neural BP (NBP) decoding algorithm shows better decoding performance than the traditional BP decoding algorithm, at the expense of increasing the complexity of the decoding algorithm. In order

to lower the high complexity of the NBP decoding algorithms, [5] and [6] introduced the pruning algorithm and proposed the pruning-based NBP algorithm, which prunes unimportant network connections.
In addition to pruning, low-rank decomposition, such as Tensor-Train (TT) decomposition [7], can also be used to reduce the number of parameters by decomposing the weight tensors. In [8], the TT decomposition was first applied to fully-connected deep neural networks (DNN) to reduce the computational complexity. The experiment was run on the MNIST dataset for the task of handwritten digit recognition. The parameters of the DNN were reduced by 200,000 times, and the size of the entire network was reduced by 7 times. In [9], the TT decomposition is applied to the convolutional neural network (CNN). The authors in [10] applied TT decomposition to the Recurrent Neural Network (RNN). In [11], a Tensor-Train deep neural network (TT-DNN) based channel estimator was proposed.
In this letter, we first improve the performance of the model-driven NNMS algorithm proposed in [3], by using variable weights [12] when computing the message transmission between variable nodes (VNs) and check nodes (CNs), called NNMS+ algorithm. Comparing with the original NNMS algorithm, the NNMS+ algorithm leads to larger memory requirement. We then apply TT decomposition to the NNMS+ decoding algorithm to further reduce its memory requirement and computational complexity, called the TT-NNMS+ algorithm, which is the main contribution of this letter. In addition, we apply syndrome loss function to the TT-NNMS+ algorithm, which makes it possible to train the decoding network without knowing the transmitted codewords.
The remainder of this letter is organized as follows. In Section II, we provide the preliminaries, including NNMS and TT decomposition. In Section III, we describe the NNMS+ decoding algorithm. We then describe the TT-NNMS+ decoding algorithm and analyze its complexity in Section IV. The experimental results are provided in Section V, followed by the conclusion in Section VI.

The NNMS+ network is different from the NNMS network where each edge has the same weight. Therefore, the weights multiplied by the message of an edge can be different, depending on the output message being calculated. The NNMS+ algorithm can be shown to have better BER performance than the NNMS algorithm, at the expense of higher memory requirement and computational complexity. In practice, the neural decoding process can be implemented in parallel with a batch of codewords for efficiency. TT-decomposition can be used to lower the complexity. This can be an optional tool for tracking the varying channel conditions without sending pilots.

VI. CONCLUSION
We have proposed a low-complexity NNMS LDPC decoding algorithm with variable weights using TT decomposition, called TT-NNMS+ algorithm. We have also proposed to use the syndrome loss function for the TT-NNMS+ algorithm for long LDPC codes, which requires no knowledge of the transmitted codewords for the training of the learning algorithm. The experimental results show that the proposed TT-NNMS+ algorithm is more competitive than the NNMS algorithm in terms of BER and FER performance, memory requirement and computational complexity.
