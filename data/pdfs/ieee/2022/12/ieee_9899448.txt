Fully Absorbing Set Enumerators for Protograph-Based LDPC Code Ensembles
Emna Ben Yacoub , Graduate Student Member, IEEE, and Gianluigi Liva , Senior Member, IEEE

Abstract‚ÄĒ The average finite-length (elementary) fully absorbing set enumerators for binary protograph-based lowdensity parity-check (LDPC) code ensembles are derived. The method relies on generating functions, and it provides an efficient mean to obtain the normalized logarithmic asymptotic distribution of (elementary) fully absorbing sets. Using these results, the asymptotic distribution of (elementary) fully absorbing sets is evaluated for some protograph-based LDPC code ensembles. Numerical results show how the distributions can be used to estimate the error floor performance of protograph LDPC codes.
Index Terms‚ÄĒ Fully absorbing sets, protograph-based LDPC codes, asymptotic enumeration.

In this work, we derive the (elementary) FAS enumerators for binary protograph-based LDPC code ensembles, as well as the associated normalized logarithmic asymptotic distribution. For the latter, we rely on the efficient technique presented in [13] for the analysis of the weight distribution of protograph LDPC code ensembles, extending its application to the analysis of FASs. T HE enumeration of trapping sets (TSs) [1], [2], absorbing sets (ASs) and fully absorbing sets (FASs) [3] within low-density parity-check (LDPC) code graphs is needed to study the performance under iterative decoding in the low error probability regime under iterative decoding [4]. However, a complete enumeration of the TSs, ASs and FASs present within a specific LDPC code graph is impractical already at moderate block lengths. This difficulty can be circumvented by analyzing the average TS, AS and FAS enumerators of an LDPC code ensemble, rather than analyzing a specific code. While this approach captures only an average ensemble behavior, it identifies code ensembles that may yield codes with bad trapping/absorbing set properties. The asymptotic TS, AS and FAS enumerators for regular and irregular binary LDPC ensembles were derived in [3], [5], [6], [7], and [8]. In [9] and [10], an elegant way to obtain the TS enumerators for binary and nonbinary protograph-based ensembles [11] using the average weight enumerators was presented. The method assigns the value one to a variable nodes (VNs) and adds auxiliary flag VNs to each check node (CN) to obtain a modified protograph. A flag VN is assigned the value zero if its corresponding CN is satisfied and is assigned the unique symbol that satisfies the corresponding CN otherwise. The number of size (a, b) trapping sets for the original protograph is then equal to the number of codewords of VN weight a and flag VN weight b for the modified protograph. It is not clear if this method can be adapted to obtain the AS and FAS enumerators because each VN must fulfill a condition on the number of its connected satisfied CNs. II. PRELIMINARIES
We consider binary LDPC codes derived from protographs. A protograph P is a small bipartite graph consisting of a set
of n0 VNs, a set of m0 CNs, and e edges. Each VN/CN/edge in a protograph defines a VN/CN/edge type. We adopt the convention of denoting by Ex the set of edges connected to a (variable/check) node x, and by dx = |Ex| the corresponding node degree. The bipartite graph G of an LDPC code can be
derived by lifting the protograph. In particular, the protograph is copied Q times (where Q is referred to as the lifting factor),
and the edges of the protograph copies are permuted under the following constraint: if an edge connects a type-vj VN to a type-ci CN in P , after permutation the edge should connect one of the Q type-vj VN copies with one of the Q type-ci CN copies in G. In the following we will denote by V and C the sets of CNs and VNs in G, respectively. The lifted graph G defines the m √ó n parity-check matrix H, where m = m0Q and n = n0Q. A protograph-based LDPC code ensemble Cn(P ) is the set of length-n LDPC codes whose bipartite graph G is obtained by lifting P . To provide a compact
representation of a protograph, we will use its associated base matrix, which is an m0 √ó n0 matrix B = [bi,j] where bi,j is the number of edges that connects vj to ci in P .
We now summarize standard definitions of TSs, FASs and elementary fully absorbing set (EFAS). Consider first a set I of VNs in G. We denote by N (I) the set of its neighboring CNs. Furthermore, we denote by U(I) the set of CNs in N (I) that are connected to I with an odd number of edges, and S(I) the set of CNs in N (I) that are connected to I with an even number of edges. Note that if we assign the value 1 to the VNs in I and the value 0 to the VNs outside I, U(I) would be the set of unsatisfied CNs in N (I) and S(I) would be the set of satisfied CNs in N (I). We have that an (a, b) TS Ta,b is a set of a VNs I for which U(I) has cardinality b (i.e., it comprises b CNs) [6]. An (a, b) FAS Fa,b is a TS where each VN v in G has strictly less neighboring CNs from U(I) than from C\U(I) [14]. Finally, an EFAS FaE,b is a FAS where each CN c ‚ąą S(I) is connected to exactly two VNs in I, and each CN c ‚ąą U(I) is connected to exactly one VN in I [3].

V. CONCLUSION
In this letter, the finite-length and asymptotic average distributions of (elementary) fully absorbing sets for protographbased low-density parity-check (LDPC) code ensembles have been derived. The complexity of the calculation of the asymptotic distributions has been analyzed and reduced by exploiting symmetries introduced by parallel edges. Numerical results show how the derived distributions can be used to estimate the error floor performance of binary protograph LDPC codes.
