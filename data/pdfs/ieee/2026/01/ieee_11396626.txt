ABSTRACT The storage density of NAND flash memory has significantly increased due to multi-leveling and downscaling technologies. As a side effect of enhanced storage capacity, flash memory becomes vulnerable to circuit-level noise. To enhance data reliability, NAND flash memory employs error correction codes such as low-density parity-check (LDPC) code with soft-decision (SD) decoding based on log-likelihood ratio (LLR) information. Conventional SD read uses pre-defined uniform quantization, which is not optimized for varying threshold voltage distributions caused by physical noise, degradation, and process variation (PV). We propose an intelligent read framework employing a non-uniform quantizer that adjusts read reference voltages (RRVs) and LLR values according to the memory channel state. The optimization of RRVs and LLRs requires information on threshold voltage distributions. To accurately estimate the threshold voltage distribution from limited channel information, we propose a machine learning-based estimation model. Moreover, we incorporate model-agnostic meta-learning, enabling precise estimation of threshold voltage distribution across different flash chips addressing PV. Numerical results demonstrate significant improvements in read performance, validating the effectiveness of the proposed framework.
INDEX TERMS NAND flash memory, read reference voltage, threshold voltage estimation, meta-learning.

I. INTRODUCTION NAND flash memory is a type of non-volatile memory that has a large storage capacity. It is widely used in various applications, including solid-state drives and mobile devices, due to its swift read/write operations [1]. To meet the ever-increasing demand for greater storage capacity, NAND flash memory adopts multi-leveling and downscaling technologies. In addition, 3D NAND flash memory stacks cell arrays vertically to overcome the limitation of size reduction [2]. As flash memory becomes smaller and denser, the stored data becomes vulnerable to physical noise, leading to deterioration of reliability. To enhance the reliability and extend the lifetime of flash memory, error recovery schemes

such as read reference voltage (RRV) re-adjustment [3], [4] and error correction code (ECC) [5] are applied.
NAND flash memory stores data as a voltage level of the memory cell, i.e., threshold voltage. In read operation, the threshold voltage is compared with pre-determined RRVs to detect the stored data symbol. However, applying a fixed RRV optimized to a specific threshold voltage distribution may cause significant errors, since the distribution shifts and is distorted due to data retention and circuit-level noise. To reduce bit error, recalibrating the RRV for the memory state being read is essential. The sensing error is minimized if the RRVs are set as the intersections of two adjacent threshold voltage distributions. In [3] and [4], read-retry methods that adjust RRVs and retry to read repeatedly until successful data retrieval have been proposed. Errors in sensing are inevitable even if RRVs are determined ideally due to the shift and widening of threshold voltage distributions as degradation of NAND flash memory. ECC can enhance data reliability by encoding messages with parity bits that allow the correction of possible errors. Therefore, modern flash memory adopts ECC such as low-density parity check (LDPC) codes due to its strong error correction capability. LDPC code can be decoded by a message-passing decoding method where variable nodes and check nodes iteratively exchange the likelihood information with each other. The decoding can be conducted in both harddecision (HD) and soft-decision (SD) modes, which can be applied flexibly based on the memory channel states [5].
SD decoding of LDPC code utilizes soft information on the stored data symbol which can be acquired by sensing with multiple RRVs [7]. By optimizing RRVs involved in sensing for SD decoding, the decoding performance can be significantly improved [8]. However, optimizing multiple RRVs for SD decoding in the consideration of memory channel states, i.e., threshold voltage distributions, is difficult. The relations between memory channel states, RRVs, and decoding performance such as frame error rate (FER) are highly non-linear and difficult to model. The studies in [9] and [10] have optimized RRVs to maximize mutual information (MI) between input data symbol and output of sensing operation, instead of optimizing RRV to minimize FER. It has been shown that soft information obtained from sensing using optimized RRVs improves decoding performance. To model MI and optimize RRVs, the threshold voltage distributions of the stored data should be known. However, these works do not address the online procedure to obtain threshold voltage distribution information required for the RRV optimization. It is difficult to capture the threshold voltage distribution, often requiring intensive sensing, and only limited information is accessible from the controller. Moreover, optimizing RRVs for multi-level NAND flash memory is complicated since multiple pairs of threshold voltage distributions need to be considered jointly. In this case, a data-driven approach is useful, allowing optimization of RRVs with datasets.
Machine-learning (ML) algorithms can effectively learn highly non-linear input and output relationships from data. ML has become a popular technology for designing error management schemes for NAND flash memory systems due to its high effectiveness and utility. For instance, the work of [11] has proposed a neural-modulation framework to mitigate inter-cell interference. The neural modulator pre-distorts threshold voltages corresponding to the data symbol to be programmed considering the interference of adjacent cells. Although this strategy can enhance the reliability of flash memory considerably, utilizing neural networks (NNs) for every programming and read operation introduces significant

latency. In addition, read-side optimization, such as RRV and LLR calibrations are not considered here. In [12] and [13], RRV adaptation schemes based on ML without prior knowledge of memory channel state have been proposed. However, these methods are limited to optimizing RRVs for HD sensing. In particular, the approach in [13] leverages sparse histograms obtained from multiple sensing operations to infer the memory channel state and optimize the RRVs, which may result in increased latency. Alternatively, it has been shown that ML models such as NN can effectively extract information on memory channel states from input features that can be obtained with low overheads [14], [15]. Based on this rationale, if threshold voltage distributions can be effectively estimated, RRVs can be optimized for better error-correcting capability of the system. In [16], an unsupervised learning approach is applied to estimate channel parameters and reset the HD RRVs to reduce the symbol error rate. However, this method does not address the issue of generalization across different memory chips. In general, ML-based threshold voltage estimation suffers from limited applicability across devices, as a single model often fails to generalize due to process variation (PV).
PV is an unexpected error that occurs during fabrication, inducing differences in threshold voltage distribution and reliability [17], [18], [19], [20]. These variations occur between different chips, blocks, or wordlines (WLs). Designing a general error management scheme that encompasses all flash memory states is challenging due to the unpredictability of the PV. In particular, in the case of ML-based schemes, the PV causes large performance deviations between memories. Moreover, optimizing the ML model for each memory is costly in large-scale manufacturing. Meta-learning is a learning algorithm that can quickly adapt a model to a new task using information acquired from a previously learned task [21]. Model-agnostic meta-learning (MAML) can adapt to a variety of different tasks with gradient descent with small overheads [22]. In the process of learning multiple different but similar tasks, a common bias is learned and used as a shared variable. Later, when applying the model to a new task, the model is adapted by acquiring a context variable that represents meaningful embedding of the task [23]. Due to the adaptability of meta-learning, it has been shown that a model trained on several flash memory chips can effectively adapt to a new memory chip [15]. In this spirit, meta-learning enables the development of a universal error management scheme, even in the presence of the PV.
Motivated by these observations, we propose an intelligent read framework for NAND flash memory systems under the influence of PVs. The framework optimizes RRVs for SD sensing to maximize the MI between input data symbols and the soft sensing output. Moreover, log-likelihood ratio (LLR) values assigned to each sensing output are precisely adjusted. Since the optimizations of RRVs and LLRs require information on threshold voltage distribution, we adopt a threshold voltage estimation model with NN structures. The

• We characterize the distribution of threshold voltage in TLC NAND flash memory, incorporating a comprehensive examination of factors influencing the distribution ranging from physical noise to the complexities introduced by PV. Subsequently, the non-uniform quantizers and LLR mapping are optimized based on the estimated distribution for improved read performance.

A MAML structure, specifically CAVIA, is utilized due to its fast adaptability via a single stochastic gradient descent (SGD) update with small overheads. This approach contributes to advancing the framework to be a universal strategy that can enhance read performance in the presence of PV.

ARCHITECTURE OF NAND FLASH MEMORY A NAND flash memory chip is a composite of thousands of memory blocks, which are two-dimensional arrays of flash cells. Flash cells in the same row within a block share a WL, which contains 32 k to 64 k cells [25]. The flash cell stores data in the form of threshold voltage trapped in a floating gate (FG) of a metal-oxide-semiconductor field-effect transistor.
The number of threshold voltage levels determines the bit of symbol that can be stored in a single memory cell. For

instance, a single-level cell (SLC) flash memory can store 1-bit data per cell by using two levels of threshold voltages. Likewise, a multi-level cell (MLC) and triple-level cell (TLC) flash memory stores 2 and 3-bit data symbols with 22 and 23 threshold voltage levels, respectively. We particularly focus on TLC NAND flash memory in this paper. Fig. 1 illustrates the threshold voltage levels and assigned 3-bit data symbols with Gray coding. Note that the three bits of Gray coded symbol are represented as most significant bit (MSB), central significant bit (CSB), and least significant bit (LSB) pages depending on their logical locations. The considered Gray coding enables swift page-wise read operations minimizing read latency [26].

B. INTERFACE OF NAND FLASH MEMORY
Three basic operations of a typical NAND flash memory system are program, erase, and read. The programming operation is a process of storing data by injecting charges into FGs. The programming is carried out using an incremental step pulse programming (ISPP) scheme [27]. The ISPP scheme involves iterative injection of charges and verification of threshold voltage level until it reaches the desired level for the corresponding data symbol. Next, the erase operation removes charges from cells in a block to configure all cells to erasure state for reprogramming. The read operation of NAND flash memory employing LDPC code retrieves the data stored in a WL by performing sensing followed by decoding.
Read modes of LDPC codes in NAND flash memory are defined as HD and n-bit SD according to the precision of the sensing operation. HD read mode quantizes the threshold voltages into two windows by a single sensing operation on each page. In the case of n-bit SD, 2n voltage windows are obtained from 2n − 1 sensing operations. Therefore, LLR values with higher precision are assigned to each voltage window, which significantly improves the error correction capability of message-passing decoding. However, the latency due to additional sensing operations may limit the throughput of memory.

III. OPTIMIZATION OF SOFT-DECISION READ The encoded data symbol X is stored as a threshold voltage in a flash memory cell during the programming operation. When the data is read, the observed threshold voltage Y follows the distribution described in (4). To read the stored data, RRVs are applied to the control gate and compared with the threshold voltage stored in the cell to determine the corresponding voltage window. As a result, quantized information Z on the stored data is acquired from the sensing operation. Here, the resolution of quantized information is determined according to the precision of the sensing. The process of converting continuous information, such as a threshold voltage Y , into quantized information Z is called quantization, and the reference of the quantization is quantizer, i.e., RRVs. This procedure can be interpreted as an example of the information bottleneck method, which seeks to compress observed data (threshold voltage, Y ) into a compact representation (quantized information, Z ) while preserving information about the target variable (original stored data, X ) [32].
In order to improve the reliability of memory by reducing read error, quantizers must be optimized according to the distribution of threshold voltages. Optimal RRV for HD sensing is determined as an intersection between two adjacent threshold voltage distributions, where the raw bit error rate

(RBER) is minimized. On the other hand, SD RRVs are generally placed at fixed intervals relative to the optimal HD RRV as shown in Fig. 2 (a). Next, fixed LLR values are assigned to Z for decoding without concerning the state of flash memory. However, these uniform quantizers with fixed LLR mapping become inefficient when the threshold voltage distribution changes as the memory wears out or is corrupted by noise. Optimizing the SD quantizer can enhance read performance, but directly modeling the relationship between soft sensing outputs and the frame error rate (FER) after decoding remains challenging because of the iterative and nonlinear nature of the message-passing decoding algorithm. FER depends on the final hard decisions made from the summed LLRs at each variable node. Since this involves discrete thresholding operations, FER is a non-differentiable metric and cannot be optimized using gradient-based methods. Moreover, small changes in input LLR values can propagate through the entire decoding process, making performance evaluation computationally intensive. Optimizing quantizers and LLRs with respect to FER would require full decoding simulations for each design iteration and would depend heavily on the specific code and decoder architecture.
Alternatively, the information bottleneck method addresses this problem by selecting quantizers that maximize relevant information. In particular, [9], [33] proposed non-uniform quantizers that maximize the MI between the input symbol X and the quantized output Z from the soft sensing operation, as illustrated in Fig. 2 (b). The MI-maximizing design is decoder-agnostic and analytically tractable, enabling gradient-based optimization. While maximizing MI does not directly minimize FER, it serves as an effective surrogate objective, leading to substantial error rate reductions in practice.
Note that, the number of RRVs required for sensing increases with the number of bits stored per cell. Specifically, HD sensing uses 2n − 1 RRVs for n-bit cells, while 2-bit and 3-bit SD sensing require 2n+1 − 2 and 3 · 2n+1 − 6 additional RRVs, respectively. Optimizing multiple RRVs in consideration of the threshold voltage distribution is a burden on the memory controller. In addition, the threshold voltage distribution of each state for the data symbol is not uniform. Rather, their standard deviation varies because the impact of data retention varies depending on the voltage levels. Thus, the RRVs should be optimized individually by pages in consideration of differences in the deviations of the distributions. Considering these challenges, we describe the optimization of the RRVs for SD sensing given distributions of threshold voltages in Section III-A. Next, Section III-B suggests the optimal LLR values for quantized information Z , given optimal RRVs and threshold voltage distributions.

B. LLR MAPPING After the acquisition of quantized information Z , proper LLR values have to be assigned to Z for decoding of LDPC codewords. The LLR values of Z are relatively determined according to the distribution of the threshold voltage and the RRV. Since the LDPC decoding algorithm corrects errors in the codeword through an iterative update process based on the input LLR value, assigning accurate LLR values to Z is important to achieve better error-correcting capability.
Decoding of LDPC code corrects errors with a message-passing algorithm where check nodes and variable nodes iteratively exchange LLR information [35]. In particular, the input LLR from the channel should be carefully determined because it provides information on the reliability of the received bit. Unlike communication systems where LLR is determined based on the received signal, NAND flash memory acquires quantized LLR values assigned to the sensing outputs.

IV. INTELLIGENT READ FRAMEWORK Optimization of non-uniform quantizers and LLR mapping functions proposed in Section III requires information on the threshold voltage distributions of flash memory. Therefore, it is essential to determine the threshold voltage distributions before optimizations. Capturing the threshold voltage distribution of flash memory influenced by various factors is challenging. In the studies of [33] and [37], threshold voltage distributions have been estimated by fitting histogram data on the threshold voltage stored in the memory to a specific distribution model. Here, the histogram data was obtained from repetitive sensing by increasing the RRV at a fixed interval. This method can obtain high-resolution information on the threshold voltage distribution, but it is not practical because numerous sensing operations result in significant latency. Furthermore, due to the varying characteristics of memory across different products, it is challenging to model the threshold voltage distribution with high accuracy using a generalized mathematical model. Instead, the threshold voltage distribution can be estimated by obtaining indirect information on the state of memory deterioration. Nevertheless, the information available from the memory controller is limited and it is impossible to model the complex and non-linear relationship between the indirect information and the threshold voltage distribution.
Data-driven approaches, specifically, NN models can learn the complex and non-linear input-output relationship from the data [38]. A model trained with sufficient data can derive a meaningful output for a new input. In this section, we propose an intelligent read framework that leverages machine learning models to enhance the performance of SD read. This framework estimates threshold voltage distributions and optimizes non-uniform quantizers and LLR mapping functions based on the obtained information.
Fig. 6 illustrates the operational procedure of the proposed intelligent read framework. Once soft reading with optimized quantizers and LLR mapping is activated, the flash memory controller first acquires the input feature vector x. The distribution estimator module then estimates the parameters of the threshold voltage distribution for the current channel. The design of the input feature and the distribution estimator is detailed in Sections IV-A and IV-B. Next, based on the estimated parameters, the HD and SD RRVs are optimized. Using the optimized RRVs, soft sensing is performed, the LLR values are mapped to the sensed results, and soft

VOLUME 14, 2026

M. Hwang et al.: Intelligent Read Framework With Meta-Learning for NAND Flash Memory

decoding is initiated. The optimization procedures for the RRVs and LLR values are detailed in Section IV-C.

A. ESTIMATION OF THRESHOLD VOLTAGE DISTRIBUTIONS As a first step, the proposed framework estimates the distributions of threshold voltages using a distribution estimator module. Since the threshold voltage distributions are approximated as Gaussian, the mean and standard deviation are the target of the estimation. The distribution estimator module is designed as a multi-layer NN regression model.

where gθl (x) = σ (Wl x + bl ) is the lth layer of the NN with non-linear activation function σ (·), e.g., a rectified linear unit (ReLU) or a hyperbolic tangent function, for l = 1, . . . , L. Note that the output layer, i.e., l = L, does not use non-linear activation. θ = {θl }l=1,...,L is the parameters of the model where θl = {Wl , bl } with the weight matrix Wl and the bias vector bl. The output of the network is the mean and variance vectors of the threshold voltage distributions, i.e., µ and σ.
The model is trained to acquire the non-linear relationship between input features and the distributions of threshold voltages. The appropriate feature should contain sufficient information on the state of flash memory for accurate prediction of threshold voltage distributions. Ideally, the model is trained using features including chip, block, WL identifiers, as well as the number of P/E cycles and retention time. While the controller has access to P/E cycle counts and WL location at access time, it cannot directly observe retention time or physical variations like inter-chip and inter-block differences. To address this limitation, we use practical features including the number of P/E cycles N , index of WL w, and on-cell ratios r, i.e., x = [N , w, r]T to predict the flash memory channel [14], [15]. The number of P/E cycles gives information on the wearing degree of flash memory. The inter-WL variation tends to be consistent in different blocks and it can be inferred from WL index. However, the overhead of obtaining r is insignificant because the on-cell ratio can be obtained by sensing only one specific WL for each block as the variation between WLs for each block tends to be consistent [20]. Note that multiple on-cell ratios can be utilized as input features to further improve estimation performance. In this work, we use two on-cell ratios obtained from two sensing operations with different fixed reference voltages. These on-cell ratios can

B. META-LEARNING BASED DISTRIBUTION ESTIMATOR
Accurate estimation of threshold voltage distributions is crucial for the optimization of the soft read. However, the non-linear relationship between the input feature and the distribution varies across flash memory chips due to PVs. For this reason, the estimation performance is significantly degraded when a distribution estimator module trained on a specific chip is applied to another chip with a different distribution. In addition, if the model is trained with a large number of chips, its size may become huge or it may not be able to learn the input-output relationships. To address this problem, we adopt a meta-learning approach that enables the model to adapt to the diverse memory characteristics of different flash chips. Various meta-learning methods can be used for this purpose, including MAML, First-Order MAML, and Reptile [39]. In our work, we specifically utilize the CAVIA architecture for the distribution estimator module. Unlike conventional MAML-based methods that require updating the entire set of network parameters, the CAVIA model enables rapid adaptation to new flash memory chip environments by updating only a small set of input-dependent parameters [23].
The distribution estimator based on CAVIA has the same architecture with (22). The only structural difference is that the CAVIA model receives context variable as an additional input. Thus, the concatenated input for the CAVIA model becomes x¯ = [xT , φ]T . The training of CAVIA is carried out in two phases: an offline meta-training and an online adaptation.
The meta-training phase is an offline process for learning an initial state model to be applied to each chip before the adaptation phase. In particular, shared variable which is the parameter θ of the model is trained using data from multiple source memory chips. Consider datasets {DS1 , . . . , DSK } from K different source chips. Note that the generalizability of the meta-trained model depends mainly on the diversity of chip characteristics encountered during the meta-training

In summary, the operation of the meta-learning-based distribution estimator proceeds as follows. First, the memory producer collects multi-chip threshold voltage distribution data for meta-training. Meta-training is then conducted offline using a sufficient amount of data. The resulting meta-trained model is subsequently deployed to the actual product. Upon the initial activation of the flash memory chip, the model performs online context variable adaptation using pilot data samples, which can be obtained as a histogram from multiple sensing operations. Finally, the adapted model can infer the threshold voltage distribution for any given input query in real time.
Although the meta-trained model can adapt to various flash memories under variations that are not directly observable, performance can degrade if the model is applied to flash memory produced using an entirely new manufacturing process or subjected to different aging conditions. This is because data-driven models are highly dependent on the training data distribution and might not be effective when the threshold voltage distribution at inference time differs significantly from the training phase. In such cases, the model weights should be fine-tuned using data obtained from the device under the expected usage conditions.
C. OPTIMIZATION OF NON-UNIFORM QUANTIZER AND LLR MAPPING FUNCTION In the following step of the threshold voltage estimation, non-uniform quantizers are optimized based on the estimated distribution. The quantizers for HD sensing are determined as the cross-points of adjacent distributions of threshold voltage using HD reference voltage calculation module following (12). Subsequently, SD sensing quantizers are determined to maximize MI between the input data and the sensing output, which is the solution of (9). However, due to the high complexity of the objective function of (9), it is difficult to implement an algorithm to find a solution on a flash memory controller. Therefore, we can predict the SD sensing quantizers by designing the SD reference voltage optimizer module as a NN model as well. The input features for the model are the means and variances of the threshold voltage distributions of the WL to be read. Then, the model predicts the optimal SD quantizers for the threshold voltage distribution with input means and
VOLUME 14, 2026

M. Hwang et al.: Intelligent Read Framework With Meta-Learning for NAND Flash Memory

variances. Note that the model can be trained offline using datasets composed of synthetic threshold voltage distribution and the corresponding optimal SD quantizers which is the solution of (9). This can ease the burdens of obtaining data for training considerably.
Fig. 7 shows an example of optimal and predicted quantizers for 2-bit soft reads on TLC flash memory. The predicted quantizers are based on the estimated threshold voltage distribution, while the optimal quantizers are based on the true distribution. Since the estimation of threshold voltage distribution must contain an error, the preceding prediction on quantizers has errors due to error propagation. Nevertheless, the predicted quantizers are very close to the optimal ones. Typically, the RBERs of HD sensing using optimal and predicted HD quantizers are 0.014432 and 0.014455, respectively. For 2-bit SD sensing, the MI values for the optimal and predicted quantizers are 2.9324 and 2.9322, respectively, and the corresponding FERs are 0.00065 and 0.00128. Note that a threshold voltage deviation due to data retention is larger with a larger threshold voltage. For this reason, even within the same cell, the size of the interval between SD quantizers that maximize MI varies by page. For example, the gap between quantizers is set to be narrower between threshold voltage distributions corresponding to x = 0, 1 than that of x = 6, 7 because the size of error regions is larger between threshold voltage distributions corresponding to x = 6, 7. Therefore, compared to the existing SD technique using uniform quantizers, the proposed read scheme greatly improves decoding performance, especially on the MSB pages.
Next, LLR mapping function of the LLR calculation module is optimized considering the estimated threshold voltage distributions and optimized quantizers following (18)(20). As a result, accurate LLR values can be assigned to each quantized state after sensing so that the decoding performance improves.
Our framework adopts a pipelined structure, offering flexibility to modify each module to meet diverse requirements, in contrast to end-to-end designs. However, due to the structural dependencies, estimation errors in the threshold voltage distribution may propagate through subsequent stages. To address this issue, future research could explore an intermediate refinement step to reduce noise and bias in the estimated distributions before quantizer optimization. Potential strategies include ensemble averaging, which aggregates results from multiple estimators, or autoencoder-based postestimation denoising to suppress residual errors. Although these techniques are not evaluated in this study, they represent promising directions for enhancing robustness. Their additional computational overhead and latency, however, should be carefully considered in future design trade-offs.
In practical implementations, the proposed framework is triggered upon detecting reliability degradation at the page or wordline level. In a typical NAND flash memory system employing LDPC codes, the controller first attempts HD decoding and sequentially applies SD decoding with
VOLUME 14, 2026

increasing sensing resolutions. The proposed distribution estimation and subsequent calibration of RRVs and LLR mappings can be initiated if the SD decoding with the highest precision fails. Beyond decoding failures, more proactive trigger mechanisms can be employed to better balance the latency-reliability trade-off. For instance, recalibration can be triggered when the cyclic redundancy check error count exceeds a predefined threshold or when hardware level indicators, such as the on-cell ratios measured at specific reference voltages, deviate significantly from their nominal values. While the optimal selection of these trigger thresholds depends on specific hardware characteristics and application level requirements, these strategies provide a feasible approach for real time reliability monitoring and adaptive management.
V. EXPERIMENTS AND DISCUSSIONS In this section, we present and discuss experimental results to verify the effectiveness of the proposed intelligent read framework. For the experiments, we generated threshold voltage distributions corresponding to various TLC flash memory states following the method in [15]. The standard deviations of initial threshold voltage distributions vary across WLs and chips. Initial threshold voltage distributions for multi-WL flash memory are extracted by emulating the memory channel of a 64-stack TLC vertical NAND [40]. Random dopant fluctuation and oxide trap fluctuation result in chip-wise variations in the standard deviation of initial threshold voltages. We generated a hundred chip variation values and selected 9 samples at 10 % intervals after arranging them in ascending order. These samples are specified as the variations of chip 1 to chip 9. For the variation between blocks, 10,000 random wearing degree values were generated following the wearing degree model in [41]. Next, nine samples are selected in the same manner as wearing degree of block 1 through block 9. For training of the distribution estimator module and the SD reference voltage optimizer module, we used 9, 000 data samples collected over the following range: chips [1, 3, 5, 7, 9], blocks [1, 5, 9], WLs [1, 6, 8, 29], P/E cycle counts [1000 : 200 : 2000], and retention days [2 : 2 : 50]. This dataset is generated from our channel model described in Section II-C and in our previous work [15], which incorporates variations across WLs, blocks, and chips. The initial multi-WL threshold voltage distribution, programmed using ISPP, was modeled based on channel data extracted by emulating a 64-layer stacked TLC vertical NAND, following the process in [40]. We adopted LDPC code with a payload length of 1 kB and code rate of 0.9, which represents a typical configuration used in modern NAND storage systems. A single data frame consists of 24 codewords. The maximum iteration numbers for HD decoding and SD decoding algorithms were set at 50 and 250, respectively.

Compared to other models (Beta, Gamma, Log-Normal, Weibull, etc.), the Gaussian model has less deviation in modeling errors for the threshold voltage distribution corresponding to each data symbol in flash memory with multi-leveling [34]. The ISPP scheme iteratively injects charge and verifies the threshold voltage until it reaches the target value corresponding to each data symbol. However, after extensive P/E cycling, the programming process becomes prone to errors, leading to misprogramming events. As a result, the threshold voltage distribution develops thicker tails and exhibits multi-modality due to these misprogramming effects. enhancement in error-correcting capability through rigorous optimization of the soft-decision read scheme. The optimization includes the adjustment of non-uniform quantizers to maximize the MI and precise calculation of LLRs for decoding on the basis of flash memory channel states. In order to obtain information on the flash memory channel, we have introduced an effective threshold voltage estimation model based on a neural network. Crucially, our framework stands out for its practicality, utilizing features accessible through a memory controller without heavy overheads. The incorporation of meta-learning, particularly the CAVIA structure, has proven effective in reducing the impact of PVs, significantly improving estimation accuracy for various memory chips. The numerical results have attested to the effectiveness of the proposed approach, showcasing substantial improvements in read performance.
