Abstract—Spatially-coupled (SC) LDPC codes are constructed from a set of L regular sparse codes of length M . In the
asymptotic limit of these parameters, SC codes present an exce-
llent decoding threshold under belief propagation (BP) decoding,
close to the maximum a posteriori (MAP) threshold of the
underlying regular code. In the finite-length regime, we need LM
both dimensions, and , to be sufficiently large, yielding a
very large code length and decoding latency. In this paper, and for
the erasure channel, we show that the finite-length performance
of SC codes is improved if we consider the tree-structured
expectation propagation (TEP) algorithm in the decoding stage.
When applied to the decoding of SC LDPC codes, it allows using
shorter codes to achieve similar error rates. We also propose
a window-sliding scheme for the TEP decoder to reduce the
decoding latency.
Index Terms—convolutional LDPC codes, finite-length analy-
sis, tree-structured expectation propagation.
I. INTRODUCTION
Spatially-coupled (SC) codes [1] are constructed by coupling a chain of L regular (l, r) low-density parity-check (LDPC) ensembles of code length M , together with appropriate boundary conditions. This construction exhibits a belief propagation (BP) threshold close to the maximum-a-posteriori (MAP) threshold of the regular (l, r) LDPC ensemble [2]. Spatially-coupled codes are terminated by incorporating additional check nodes of low degree, which are intimately related with the good asymptotic decoding properties, but produce an undesirable rate loss, proportional to 1/L in the finite-length regime [1], [3]. As a consequence, long chains of LDPC codes (i.e. large L) are needed to mitigate this rate loss, yielding codes with excessive code lengths and decoding latency, i.e. an unreasonable delay associated to the acquisition and decoding of a new received word [4]. We may classify SC LDPC codes attending to how the coupling between LDPC ensembles is performed. Low-density parity-check convolutional (LDPCC) ensembles, first introduced in [5], are the best known example of SC codes. Unlike irregular capacity achieving (c.a.) LDPC ensembles [6], LDPCC codes achieve channel capacity with a minimum distance that linearly grows with the constraint length [3], [7].
The tree-structured expectation propagation (TEP) decoder [8], [9] improves the BP algorithm solution by using pairwise marginal estimates to improve each bit posterior probability estimate. When used for LDPC decoding, the TEP algorithm

is able to break some fairly common stopping-sets (SSs) that limit the BP decoder [10], improving the performance. For finite-length LDPC ensembles, we show in [10] that the TEP either provides a gain in error rate for a given code length, or a reduction of the block length needed to achieve a target error rate. However, this gain vanishes with the code length. When working over the binary erasure channel (BEC), the TEP complexity is linear with the code length, as for the BP decoder [9].
In this work, we extend the TEP algorithm for LDPCC decoding over the BEC [9]. We focus our study on the gains in terms of word error rate (WER) of the TEP decoder compared to BP decoding, or similarly on the reduction of the code length for a desired WER. The TEP decoder improves BP for LDPCC codes of interest, because it reaches the desired performance with lower M . Even for extremely large code lengths (L ⇥ M ), we show that the TEP decoder exhibits a significant gain with respect to BP. This fact makes the TEP decoder an attractive tool to exploit LDPCC codes in erasure coding and data storage applications [11], [12], [13]. To address the decoding delay, a windowed scheme, wTEP, is also proposed to provide a simple, yet efficient, way to tradeoff decoding performance and reduced latency [2], [14].

VI. CONCLUSIONS
For the BEC and finite-length LDPC codes, the TEP decoder exhibits a gain in performance compared to the BP, with linear complexity. In this letter we show that for LDPCC codes, a representative example of spatially-coupled codes, this gain is achieved for much longer codes. This can be exploited either to improve the WER or the latency by reducing the code length. For a given WER, under TEP decoding the code length is reduced by significantly decreasing the number of variables per LDPC code in the LDPCC. We also prove that TEP decoding can be successfully applied in an sliding window scheme, improving the performance of the windowedBP or, alternatively, reducing the window size or the total code length. The same conclusions can be extended to other variants proposed to date for convolutional-structure LDPC coding. In particular, we have presented results for diminished rate-loss LDPCC codes, where we allow low degree check nodes at only one side of the code Tanner graph, reducing the rate loss to the half. These results are of major interest, because practical LDPCC codes are limited by run-time complexity and decoding latency.
