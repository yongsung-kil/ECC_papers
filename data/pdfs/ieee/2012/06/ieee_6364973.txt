Performance of Rate 0.96 (68254, 65536) EG-LDPC Code for NAND Flash Memory Error Correction
Jonghong Kim, Dong-hwan Lee, and Wonyong Sung Department of Electrical Engineering and Computer Science
Seoul National University Seoul, Korea
E-mail: {jhkim, ldh}@dsp.snu.ac.kr; wysung@snu.ac.kr

Abstract—As the process technology scales down and the number of bits per cell increases, NAND Flash memory is more prone to bit errors. In this paper, we employ a rate-0.96 (68254, 65536) Euclidean geometry (EG) low-density parity-check (LDPC) code for NAND Flash memory error correction, and evaluate the performance under binary input (BI) additive white Gaussian noise (AWGN) and NAND Flash memory channels. The performance effect of output signal quantization is also studied. We show the strategies for determining the optimum quantization boundaries and computing the quantized loglikelihood ratio (LLR) for the NAND Flash channel model that is approximated as a mixture of Gaussian distributions. Simulation results show that the error performance with the NAND Flash memory channel is much different from that with the BIAWGN channel. Since the distribution of NAND Flash memory output signal is not stationary, it is important to accurately assess the stochastic distribution of the signal for optimum sensing. Index Terms—NAND Flash memory, LDPC codes, quantization.
I. INTRODUCTION
NAND Flash memory is used for many applications such as digital cameras, smart phones, and solid-state drives because of fast access speed, low power consumption, and high capacity. Today’s 20nm technology NAND Flash memory has the capacity of 64Gbit with the page size of 8KB [1]. Although the capacity of Flash memory has increased very dramatically by aggressive process scaling and multilevel cell (MLC) technology, NAND Flash is more prone to bit errors because of program-erase (PE), data retention, and cell-to-cell interference (CCI) problems. Due to the MLC technology, the gap between adjacent voltage levels becomes narrower, and the threshold voltage distribution may be shifted or even overlapped by the CCI and increased PE cycling. The number of electrons in the floating gate also decreases due to the feature size reduction. Furthermore, since electrons in the floating gate of a cell slowly leaks as time passes, the voltage distribution becomes wider and shifted to the left direction (negative voltage region).
Traditionally, hard decision based error correction such as Bose-Chaudhuri-Hocquenghem (BCH) and Reed-Solomon

(RS) codes have been used for NAND Flash memory error correction [2]-[4]. However, in order to compensate for the performance degradation in MLC NAND Flash memory, strong error correcting codes are demanded. One approach is to employ soft-decision error correction codes that represent the signal level in multiple bits, instead of just 0 or 1. In this work, we employ low-density parity-check (LDPC) codes for error correction of NAND Flash memory.
LDPC codes [5] show excellent error correcting capability close to the Shannon limit. Since practical LDPC codes have a finite code length and employ a finite iteration for decoding, their error correcting performance is slightly degraded. However, it still offers good error performance when belief-propagation (BP) algorithm [6] with soft-decision information is used for the decoding process. Meanwhile, the error performance of LDPC decoding is closely related to the precision of log-likelihood ratio (LLR) information [7]. In order to increase the precision of LLR information, multiple voltage sensing operations are required because only a simple quantization circuit is used inside in NAND flash memory. However, in this case, their memory access latency increases because of multiple sensing operations and the limited number of I/O ports. Therefore, it is desired to use the minimum number of bits to ensure high-speed performance. Moreover, the voltage distribution of NAND Flash memory is not Gaussian and the distribution is strongly affected by various factors such as CCI, PE cycling, and data retention. Thus, error performance simulation with the binary input (BI) additive white Gaussian noise (AWGN) channel is not enough to evaluate the performance under the NAND Flash memory channel.
In this paper, we assess the performance of a rate-0.96 (68254, 65536) shortened EG-LDPC code under BI-AWGN and NAND Flash memory channels. The optimal output quantizer proposed in [7] is used for determining the threshold levels of the BI-AWGN channel, while the non-uniform quantization scheme proposed in [8] is adopted for the NAND Flash memory channel. Note that LLR computation of Flash memory channel follows the method in [7]. The purpose of this work is not only to assess the error performan-

II. EG-LDPC CODES FOR NAND FLASH APPLICATION
EG-LDPC codes are algebraically generated and form a class of finite geometry (FG)-LDPC codes [9]. The FGLDPC codes have either cyclic or quasi-cyclic structure for their parity-check matrices, which allows efficient encoder implementation. For example, the encoder can be implemented with linear feedback shift registers. Moreover, since the parity-check matrix of an FG-LDPC code has some redundant rows (i.e., the number of rows is larger than the rank of the matrix) and all the rows are used for computing check sums, they show good error performance when decoded with the BP algorithm [9]. In addition, they have no harmful trapping sets with the size smaller than their minimum weights, thus resulting in good error-floor performance [10]. Recently, it is reported that (1024, 781) EG-LDPC code has the errorfloor below the bit error rate (BER) of 10-23 [11].
In this paper, a rate-0.96 (69615, 66897) EG-LDPC code is adopted for error correction in NAND Flash memory. Its parity-check matrix consists of 17 sub-matrices as shown in Fig. 1 and each sub-matrix is a cyclically right-shifted 4095× 4095 matrix with row and column weights of 16. In order for the (69615, 66897) EG-LDPC to fit in the page size of NAND Flash memory, a rate-0.96 (68254, 65536) shortened EG-LDPC code is considered as shown in Fig. 2. In this work, a page consists of 8KB user data and 5% spare bits that correspond to approximately 3450 bits. The first 1361 information bits are shortened and the remaining 65536 bits (8KB) are used as the user data. After encoding the information bits, 65536 user data bits and 2718 parity bits are stored as one page. The remaining 732 (=3450-2718) spare bits can be used for other purposes such as memory signal processing or the operation of Flash translation layer (FTL).

IV. SIMULATION RESULTS
The error performance of the (68254, 66897) EG-LDPC code is evaluated for BI-AWGN and MLC NAND Flash memory channels. We used a 2-bit MLC NAND Flash memory cell model described in Section III. We assume that the erase operation results in a Gaussian distribution whose mean and standard deviation are 1.0 V and 0.32 V, respectively. During the programming, the target voltages of 2.6 V, 3.2 V, and 3.8 V are used for the symbol 01, 00, and 10, respectively. Quantization boundaries of the BI-AWGN channel are obtained by referring to [7], while those of NAND Flash memory are calculated by solving (4) and (5), where the probability ratio ܴ was set to 8. The value of ܴ was determined by exhaustive search. The LDPC decoding employs the BP algorithm with floating-point precision, and the maximum iteration limit is set to 16. Note that, in the

V. CONCLUSION
In this paper, we employ a rate-0.96 (68254, 65536) EGLDPC code for error correction of NAND Flash memory. We assessed the error performance of NAND Flash memory by using the infinite-bit, 16-, 10-, and 4-level output quantizers for the signal obtained from the NAND Flash memory channel model. From the simulation, the performance gaps between the BI-AWGN and the NAND Flash memory channels are estimated as 0.14, 0.09, and 0.18 dB for the 3-, 2-, and 1-bit LSB quantizers, and 0.25, 0.24, and 0.13 dB for

the 3-, 2-, and 1-bit MSB quantizers, respectively. This reveals that the error performance under the conventional BIAWGN channel is significantly different from that under the NAND Flash memory channel. It is very needed to find an optimum sensing scheme that can adapt to non-stationary characteristics of the NAND Flash memory channel.
