Insertion/Deletion/Substitution Error Correction using Adaptive Inversion of Synchronization Marker
Masato Inoue and Haruhiko Kaneko Graduate School of Information Science and Engineering,
Tokyo Institute of Technology 2-12-1, Ookayama, Meguro-ku, Tokyo 152-8552, Japan. Email: inoue@fuji.cs.titech.ac.jp, hkaneko@fuji.cs.titech.ac.jp

Abstract—This paper proposes an adaptive inversion method of synchronization marker to design efficient insertion/deletion/substitution error correcting code. Unlike the conventional marker coding in which marker patterns are selected deterministically, the proposed method selects marker patterns adaptively. That is, the first bit and the last bit of each marker are conditionally inverted depending on the neighboring δ bits of an LDPC codeword, where δ is a shift resistance of the marker. This paper also shows a forward-backward algorithm to calculate the symbol-by-symbol likelihood of the adaptive marker coding. Evaluation shows that the adaptive marker coding reduces the bit error rate. For deletion/insertion error probability Pi = Pd = 0.01, substitution error probability Ps = 0.01, and code rate 0.44, the decoded BER of conventional coding is 9.29 × 10−4, while that of the proposed coding is 2.67 × 10−4.
I. INTRODUCTION
Insertion and deletion errors are observed in some practical systems. For example, bit-patterned media (BPM) recording [1] requires the precise synchronization of write signal timing and the position of a bit island. However, it is hard to achieve perfect synchronization because of the jitter in the internal clock, head vibrations, disk speed variations, and inaccurate positioning of the bit islands or variable bit island shapes. Accordingly, the BPM recording suffers from insertion/deletion errors in the recorded data.
Conventional error-correcting codes, such as the BCH code, Reed-Solomon code, and LDPC code, however, cannot correct insertion/deletion errors efficiently. Although Levenshtein code [2] has insertion/deletion/substitution (IDS) error correction capability, this code cannot correct multiple IDS errors. To correct multiple IDS errors, concatenation of LDPC and watermark codes has been proposed in [3]. Also, concatenation of LDPC and marker codes has been presented in [4], [5], in which insertion/deletion errors are reduced to burst errors of moderate length by the marker code, and then the errors are corrected by the LDPC code. In [6], the authors have proposed a method which adaptively selects a marker depending on neighboring two bits of an LDPC codeword. This adaptive selection method is applicable for a specific marker patterns, namely {(0, 1, 0), (1, 0, 1)}, of length ns = 3.
This paper proposes a generalized adaptive marker selection method which is applicable for any marker code of length ns ≥ 2, and for any marker pattern. In the proposed method,

the first bit and the last bit of each marker are conditionally inverted depending on the neighboring δ bits of an LDPC codeword, where δ is a shift resistance of the marker.
This paper is organized as follows. Section II presents related works. Section III proposes an adaptive selection method of synchronization marker. Section IV shows simulation results, and Section V concludes this paper.

III. ADAPTIVE SYNCHRONIZATION MARKER
If the decoder cannot detect the marker correctly, the FB algorithm does not provide good likelihood. Figure 3 shows example cases of missynchronization, that is, the values of marker bits, indicated by shaded part, coincide with those of neighboring bits, indicated by underline. This situation might lead to poor synchronization performance of FB algorithm. To reduce the probability of such missynchronization, this section proposes an adaptive marker and decoding algorithm.

can be explained as follows. In the adaptive marker 2 with

value, which negate the gain of the adaptive inversion. Hence

the adaptive marker 2 has lower synchronization performance

compared to the adaptive marker 1, which has two determin-

istic values in the marker.

V. CONCLUSION
This paper has proposed the adaptive inversion method of synchronization marker. In order to select marker patterns adaptively, the shift resistance δ has been introduced for each marker. This paper has also shown the decoding algorithm for adaptive marker. Evaluations have shown that the adaptive marker reduces the bit error rate.
Further research will be required to determine optimal base marker set W1 for a given channel. Also, more efficient FB algorithm should be constructed to reduce the decoding complexity.
