A New Construction of
Structured Binary Regular LDPC Codes
Based on Steiner Systems with Parameter 𝑡> 2
Hossein Falsafain and Morteza Esmaeili, Member, IEEE
Abstract—This paper presents a novel method for constructing
structured regular low-density parity-check (LDPC) codes based
on a special type of combinatorial designs, known as Steiner
systems. This code design approach can be considered as a
generalization of the well-known method which uses the point-
block incidence matrix of a Steiner 2-design for the code
construction. Though the given method can be applied on any
Steiner system 𝑆(𝑡, 𝑘, 𝑣), in this paper we focus only on Steiner
systems with 𝑡≥3. Furthermore, we show that not only a Steiner
system (𝑋, ℬ) itself, but also its residual design with respect to
an arbitrary point 𝑥∈𝑋can be employed for code construction.
We also present a technique for constructing binary and non-
binary QC-LDPC codes from Steiner systems. The Tanner graph
of the constructed codes is free of 4-cycles and hence the codes
have girth at least six. Simulation results show that the so
constructed codes perform well over the AWGN channel with
iterative message-passing decoding.
Index Terms—LDPC codes, combinatorial design, Steiner
quadruple system (SQS), inversive geometry.
I. INTRODUCTION AND BACKGROUNDS
A. Introduction
A
low-density parity-check (LDPC) code is a linear
block code represented by a sparse parity-check matrix
𝐻[1], [2]. LDPC codes are capable of performing very
close to the Shannon’s capacity limits for a large collection
of data-transmission and data-storage channels [3]-[7]. These
codes can be classified, from construction method perspective,
into computer-generated pseudo-random codes and structured
codes. Various combinatorial and algebraic methods for con-
structing well-structured LDPC codes have been proposed [8]-
[22].
Combinatorial design theory is a rich field of powerful tools
for constructing LDPC codes. For instance, various types of

Due to the nature of iterative decoding, the presence of
short cycles in the Tanner graph [23] of a given LDPC code 𝒞
has normally a negative effect on the code performance [24].
Therefore, in designing LDPC codes, it is desired to construct
codes free of short cycles, in particular length-4 cycles.
A parity-check matrix 𝐻is said to satisfy the row-column
(RC)-constraint if for any two rows (columns) r and r′ of 𝐻
the number of positions wherein both r and r′ are non-zero
is at most one. This is equivalent to saying that the bipartite
Tanner graph TG(𝐻) representing 𝐻has no 4-cycles. Hence
the code has girth at least 6. A binary (𝐽, 𝐿)-regular LDPC
code is defined as the null space of a sparse parity-check
matrix 𝐻over 𝔽2 with constant column-weight 𝐽and constant
row-weight 𝐿. If the parity-check matrix 𝐻of a (𝐽, 𝐿)-regular
LDPC code satisfies the RC-constraint, then the code has
minimum distance at least 𝐽+ 1 [25], [26].
Steiner systems have been widely used in earlier works
to construct LDPC codes. All of the previously proposed
construction methods based on Steiner systems use Steiner
systems with 𝑡= 2 to generate the required parity-check
matrices. For instance, various types of LDPC codes have
already been constructed based on 2-(𝑛, 3, 1) designs or
Steiner triple systems (see [11], [18]), 2-(𝑛2, 𝑛, 1) designs or
affine planes (see [11], [19], [21]), 2-(𝑛2 + 𝑛+ 1, 𝑛+ 1, 1)
designs or projective planes (see [11], [15], [18], [19], [25]),
2-(𝑛3 + 1, 𝑛+ 1, 1) designs or unital designs (see [16], [18],
[19]), 2-(𝑛(𝑛−1)/2, 𝑛/2, 1) designs or oval designs (see [17]–
[19]), and Bose-BIBDs (see [20], [22]).

signs. In particular, we consider two special families of Steiner
systems: Steiner quadruple systems and inversive geometries.
The first family gives a class of regular LDPC codes with
various lengths, high rates, girths at least 6, and column-weight
6. The second family like the first one, gives a class of regular

turning to higher values of 𝑡does not solely guarantee that the
resulting code unavoidably will perform better.
