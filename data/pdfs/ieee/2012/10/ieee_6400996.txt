Abstract‚ÄĒIn this paper, a generalized construction of quasicyclic low-density parity-check (QC-LDPC) codes for unequal error protection (UEP) is proposed based on the masking technique. We can obtain UEP QC-LDPC codes as long as the Tanner graph of the masking matrix satisfies a general condition which can include our previous UEP construction as a special case. A sufficient condition to have flexible choice of protection levels is then presented. Simulation results show that the constructed codes can indeed provide codeword bits with different bit-errorrate performance.
I. INTRODUCTION
Unequal error protection (UEP) [1], [2] means that codeword bits can be divided into several parts and each part has different error-correcting capability. The message part which is more important can be provided with more error protection. UEP is wildly used in many practical applications such as broadcast channels, packet-switching networks, visual communication systems, etc.
Quasi-cyclic low-density parity-check (QC-LDPC) codes [3]‚Äď[6] are LDPC codes [7] with parity-check matrices usually composed of permutation matrices. They can achieve nearShannon-limit performance, with the advantage of efficient encoding. In [6], an algebraic method based on masking was proposed, which can be applied to any parity-check matrices of QC-LDPC codes. The obtained codes in [6] can have a reduced number of short cycles, along with near-Shannon-limit performance and low error floor.
There are two common constructive methods for UEP LDPC codes: one by puncturing [8], [9] and the other based on the code structure [10]‚Äď[12]. In [12], we proposed a new construction of UEP QC-LDPC codes based on UEP block codes with stairway parity-check matrices. The UEP QC-LDPC codes obtained in [12] can have low error floor and UEP capability. However, only certain (3, t)-regular QCLDPC code family with even t can be used as the block component in the construction in [12], and only codes of certain rates can be obtained. These restrictions rather limit the applicability of the construction in [12].
In this paper, we propose a generalized construction of UEP QC-LDPC codes based on masking. By properly select the masking matrix, the stairway structure in [12] can be

We can have more flexible choices for the block component and can construct more UEP QC-LDPC codes of different rates than those in [12]. The above product procedure can be viewed as masking out a selected set of submatrices of the corresponding parity-check matrix. Since it reduces the density of the parity-check matrix, the resulting code is still a QCLDPC code.
The masking procedure can also be viewed as reducing the number of edges in the Tanner graph of the base matrix. Therefore, the Tanner graph of the masked code has a smaller number of short cycles. If B satisfies the row-column (RC) constraint, the masked matrix M also satisfies the RC constraint with any masking matrix Z. Furthermore, if the girth of the Tanner graph of Z is g, the girth of the Tanner graph of M is at least g. The masked code with a larger girth can be constructed by choosing the masking matrix Z with a larger girth. It is much easier to construct the masking matrix Z with a large girth due to its smaller size than that of the base matrix B.
Based on these properties, the masked code can have outstanding error performance with a properly chosen masking matrix. The rate and length of the resulting code depend on the choice of the base matrix B.

VI. CONCLUSION
In this paper, a generalized construction of UEP QC-LDPC codes is proposed. Based on masking, we can choose the quasi-cyclic base matrix and the masking matrix with desired properties on its Tanner graph for UEP. Moreover, a sufficient condition to ensure strict UEP is provided. Simulation results show that the constructed QC-LDPC codes can indeed

achieve UEP and each protection level can have better BER
performance than that of traditional time-sharing codes.
