In Praise of Nontrivial Redundant Rows of Algebraic LDPC Codes
Qin Huang, Mu Zhang, Xujing Guo and Zulin Wang School of Electronic and Information Engineering Beihang University, Beijing 100083, China
Email: qhuang.smash@gmail.com; calering ococ@163.com; guoxujing@126.com; wzulin@vip.sina.com

Abstract—One of the most interesting features of some algebraic LDPC codes is that their RC-constrained parity-check matrices have several or many redundant rows. This paper focuses on the positive impacts of these redundant rows on the belief propagation decoding algorithms. First, the analysis indicates that they are able to eliminate small trapping sets which cause high error-floors in BER curves. Second, their positive impacts on waterfall thresholds of BER curves are verified and explained. An LDPC code with such redundant rows is constructed, which outperforms the corresponding PEG-LDPC code 0.2 dB at BER around 10−7.
I. INTRODUCTION
Algebraic low-density parity-check (LDPC) codes [2], [8], [9], [10], [11] form important subclasses of LDPC codes. They enjoy wonderful features, e.g., cyclic or quasi-cyclic feature, redundant rows, and RC-constraint, from their inherent algebraic structures. As a result, they not only perform well with the belief propagation decoding algorithms, e.g., the sum-product algorithm (SPA) [1], but also are suitable for hardware implementations of encoders and decoders. Thus, some communication systems, e.g., IEEE 802.3 10G BasedT and NASA Landsat, and solid state drives have adopted algebraic LDPC codes as their standards.
A certain or large number of redundant rows in paritycheck matrices are one of the most interesting features of some algebraic LDPC codes that is seldom found in other classes of LDPC codes. Since these redundant rows keep the row-column (RC) constraint of parity-check matrices, they are suitable and helpful for the belief propagation decoding algorithms. It has been shown that algebraic LDPC codes with large number of redundant rows enjoy low error-floors due to its large trapping sets. Moreover, it has been noticed that algebraic LDPC codes have better waterfall thresholds than other LDPC codes with the same variable node degree distributions.
Thus, we would like to explore the effects of such redundant rows on the belief propagation decoding algorithms and the inherent mechanism. First, we call such redundant rows ‘nontrivial’ and prove that they are able to eliminate small trapping sets which may cause high error-floors in BER curves. It is verified by increasing the redundant rows of an Euclidean geometries (EG) LDPC code. Second, their positive effects on waterfall thresholds of BER curves are explained by the notation of similar Tanner graph. It is verified by increasing

the redundant rows of the EG-LDPC code and comparing it with the corresponding progressive edge-growth (PEG)LDPC codes [7]. In addition, an LDPC code is constructed with nontrivial redundant rows, which achieves better error performance than its corresponding PEG-LDPC code.
The rest of this paper is organized as follows: Section II presents the background of algebraic LDPC codes with nontrivial redundant rows. In Section III, we analyze trapping sets of algebraic LDPC codes. Section IV shows the waterfall threshold improvement from nontrivial redundant rows. The last section concludes the paper.
II. BACKGROUND
A. LDPC codes
A binary LDPC code is given by the null space of a sparse matrix H over the finite field GF(2). If H, called as the paritycheck matrix, has constant column weight γ and constant row weight ρ, the code given by the null space of H is said to be (γ,ρ)-regular, otherwise it is said to be irregular.
In almost all of the proposed constructions of LDPC codes, the following constraint on the rows and columns of the paritycheck matrix H is imposed: no two rows (or two columns) can have more than one place where they both have identical nonzero components. This constraint on the rows and columns of H is referred to as the row-column (RC)-constraint [11]. This RC-constraint ensures that the Tanner graph of the LDPC code given by the null space of H has a girth of at least 6 such that it is suitable for the belief propagation decoding algorithms. It also guarantees that the minimum distance of the code, if (γ,ρ)-regular, is at least γ + 1. A parity-check matrix H that satisfies the RC-constraint is called an RC-constrained paritycheck matrix and the code given by its null space is called an RC-constrained LDPC code.
B. Nontrivial redundant rows
Given an (n, k) linear block code, its parity-check matrix must contain at least n−k independent rows to span the whole null space of size of 2n−k. In other words, any parity-check matrix of m rows must have r m − n + k redundant rows, where 0 ≤ r ≤ 2n−k − n + k.

III. NTRR IMPACT ON TRAPPING SETS AND ERROR FLOOR
In this section, we will show that nontrivial row-redundancy is able to eliminate the small trapping sets of LDPC codes. Thus, it makes the iterative decoding of an LDPC code suffer a lower error-floor.
Suppose that the parity-check matrix of an LDPC code has the minimum column weight γ, i.e., the minimum variable node degree is γ. If the parity-check matrix satisfies the RCconstraint, from [13], any subgraph with κ variable nodes must contain at least τ = κ(γ − (κ − 1)) odd degree check nodes. If the lower bound κ(γ − (κ − 1)) on the number of odd degree check nodes is larger than 4κ, then it is not small and, therefore, not harmful for the belief propagation decoding algorithm in high SNR region. With nontrivial redundant rows, the parity-check matrix still keeps the RC-constraint, but its minimum variable node degree will increase to γ ≥ γ. Thus, any subgraph with κ variable nodes must contains at least κ(γ − (κ − 1)) odd degree check nodes. Clearly, nontrivial redundant rows increase the lower bound on odd degree check nodes by κ(γ − γ). The ratio τ /κ increases by γ − γ.

IV. NTRR IMPACT ON WATERFALL THRESHOLD
Actually, some regular algebraic LDPC codes have better waterfall thresholds of BER curves than corresponding regular LDPC codes designed by pseudo-random matrices [7] with the same variable node degree distributions. However, irregular algebraic LDPC codes perform close to irregular pseudo-random LDPC codes. In the following, we would like to summarize the phenomenon as the effect of nontrivial redundant rows on the belief propagation decoding algorithm.
We call two Tanner graphs similar if they have the same girth and degree-distributions. It is well known that the belief propagation algorithms performs closely under two similar Tanner graphs at the same signal power per code bit to noise ratio (SNRcb). Without loss of generality, suppose that the parity-check matrix of the first code has row-redundancy and the second does not. Thus, the null space of the first one defines a higher-rate code than the null space of the second one. Therefore, the first code requires lower signal power per information bit to noise ratio (SNR) than the second code to achieve the same performance. In another word, the row-redundancy will improve the performance of LDPC codes. Most regular algebraic LDPC codes have nontrivial redundant rows, but most irregular algebraic LDPC codes have no redundant rows. Thus, we see this interesting phenomenon from simulations.
Consider the (255, 175) EG-LDPC code constructed in [2], whose parity-check matrix has at most 175 nontrivial redundant rows and 0.6863 row-redundancy. In Figure 1, we increase its row-redundancy from 0 to 0.6838 step by step to show its effect on error performance over the AWGN channel with BPSK modulation. As the row-redundancy increases, it is clear that the performance of the SPA decoding algorithm of the EG-LDPC code improves as the row-redundancy of H increases. From the above analysis, we have shown the advantages of nontrivial redundant rows. Here we would like to construct an quasi-cyclic (QC) LDPC code by array dispersion of the base matrix B over GF(q) with such redundant rows to verify its performance.
Suppose that α is a primitive element over GF(q). For each entry bi,j in B, 1) if bi,j = 0, let bi,j = αl with 0 ≤ l < e, replace bi,j by an e × e circulant permutation matrix whose first row has a single 1-component at position l, 2) if bi,j = 0, replace bi,j by an e × e zero matrix. The above construction of the RC-constrained array H directly from the base matrix B is referred to as e-fold dispersion of B [12].

V. CONCLUSION AND REMARKS
This paper verified the positive effects of nontrivial redundant rows on the belief propagation decoding algorithms, such as lower error-floors and waterfall thresholds. We also proposed to construct good QC-LDPC codes whose paritycheck matrices contain nontrivial redundant rows.
