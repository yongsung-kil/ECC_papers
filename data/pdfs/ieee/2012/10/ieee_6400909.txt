AbstractРђћIn this paper, we propose a new encoding algorithm applicable to any linear codes over arbitrary finite field whose computational complexity is O(w(H)) where w(H) denotes the number of non-zero elements in a parity check matrix H of a code. The proposed algorithm is essentially equivalent to the linear time encoding algorithm presented by Lu et al. when the maximum column weight ╬┤РѕЌ of H is less than or equal to 3, and is regarded as a natural generalization of the algorithm when ╬┤РѕЌ > 3. Moreover, the proposed algorithm can encode any linear codes defined by sparse parity check matrices, such as LDPC codes, with O(n) complexity where n denotes the code length.

Low-density parity-check (LDPC) codes are celebrated for
their excellent performance. By exploiting the sparseness of
parity check matrices, LDPC codes have decoding algorithm
with O(n) computational complexity. On the other hand, com-
putational complexity of a straightforward encoding method for LDPC codes is O(n2). For example, solving (2) by multiplying (AP)Рѕњ1 to both sides of it requires O(n2) computation, because (AP)Рѕњ1 is not sparse in general even if AP is sparse.
Multiplying a generator matrix by an information vector also requires O(n2) computation, as generator matrices are not

always sparse for LDPC codes. In order to overcome the imbalance in encoding and decoding complexity, a considerable number of researches have investigated efficient encoding of LDPC codes [1, 6]. Recently, Lu et al. [4] succeeded to develop a linear time encoding method for binary LDPC codes.
In this paper, we propose a new method to solve a system of linear equations over arbitrary field. The proposed method first arranges A of (1) into a matrix that defines an equivalent system of linear equations by row and column permutations. The permutations are carefully designed so that the resulting matrix has special properties, referred to as block-triangular and ╬┤-approximate lower triangular. By employing these properties, we succeed to develop an algorithm to solve (2) with O(w(A)) computational complexity where w(A) denotes the number of non-zero elements in A. This implies that the proposed method realizes a linear time encoding algorithm if a parity check matrix A is sparse like LDPC codes.
This paper is organized as follows. In Section II, we propose an algorithm to solve (2) under some assumption about the coefficient matrix A. In Section III, we show that it is always possible to transform a given coefficient matrix into a matrix that defines an equivalent system of linear equations and satisfies the assumption. Concrete procedure to obtain such matrices is also presented. Computational complexity of the proposed algorithm is investigated in Section IV and the relation between the proposed and LuРђЎs methods is discussed in Section V. The concluding remark is given in Section VI.

V. RELATION BETWEEN THE PROPOSED AND LUРђЎS ALGORITHMS
The linear time encoding algorithm by Lu et al. [4] is described as a message passing like algorithm over Tanner

graph. Before the algorithm is executed, Tanner graph is decomposed into several classes of subgraphs and the dependency relation between those subgraphs is determined. Roughly speaking, this decomposition of Tanner graph and the dependency relation of its subgraphs correspond to the block-triangularization of a parity check matrix introduced in our proposed method.
For a parity check matrix of a binary linear codes with ╬┤РѕЌ РЅц 3, both algorithms are essentially equivalent. However, the approaches of two algorithms are completely different if ╬┤РѕЌ > 3.
By introducing ╬┤ Рѕњ 3 additional rows and columns for each column of weight ╬┤ > 3, we can transform a parity check matrix H with ╬┤РѕЌ > 3 into a matrix H that has maximum column weight less than or equal to 3 and is still equivalent to H in some sense [4]. Hence encoding for H results in encoding for H , and computational complexity of LuРђЎs and the proposed algorithms for H becomes O(w(H )) by Theorem 1.

VI. CONCLUSION
In this paper, we have proposed a new method to solve a system of linear equations over arbitrary field that can be executed with O(w(A)) complexity. Moreover, by applying the proposed method to encoding, any linear codes defined by sparse parity check matrices, such as LDPC codes, can be encoded with O(n) complexity where n denotes their code length.
