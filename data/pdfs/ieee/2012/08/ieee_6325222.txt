Adaptive Decoding Algorithms for LDPC Codes with Redundant Check Nodes
Kai Zhang∗, Haiqiang Chen†, and Xiao Ma∗ ∗Department of ECE, Sun Yat-sen University Email: maxiao@mail.sysu.edu.cn
†School of Computer Electronics and Information Guangxi University

Abstract—This paper is concerned with decoding of algebraic low-density parity-check (LDPC) codes that are constructed based on finite fields and finite geometries. The parity-check matrices of such codes usually have redundant rows. Equivalently, their Tanner graphs have redundant check nodes. Based on this property, we propose an adaptive decoding algorithm. In the adaptive decoding algorithm, all the check nodes are classified into active nodes and silent nodes according to certain criteria, and only active check nodes are involved in the iterative procession. That is, each variable node collects messages from active check nodes and passes the extrinsic messages to its active neighbors. Two approaches to select the active check nodes are presented. Simulation results show that, the adaptive decoding algorithm can make a trade-off between the complexity and the performance.
I. INTRODUCTION
Low-density parity-check (LDPC) codes, which were first proposed by Gallager [1], have been known as a class of capacity-approaching codes. Generally, LDPC codes can be constructed by two methods: random construction aided by computer searching [2], [3] and systematic construction based on algebraic structures [4], [5], [6]. Compared with randomly constructed LDPC codes, the algebraically constructed LDPC codes usually have the following advantages: 1) some of them are quasi-cyclic and can be encoded in linear time with the simple feedback shift registers [7]; 2) they perform as well as randomly generated LDPC codes especially at short to moderate block lengths; 3) they are majority-logic decodable LDPC codes [8]. The third property implies that the paritycheck matrices usually have high column weights, which may be more attractive since it admits low-complexity majoritylogic decoders [9] [10]. The decoding algorithms for LDPC codes can be classified into three categories: 1) hard-decision decoding; 2) probability-based decoding; and 3) reliabilitybased decoding. From an implementation point of view, harddecision decoding algorithms have the lowest complexity, such as the one-step majority-logic decoding (OSMLGD) algorithm [5] and the bit-flipping (BF) decoding algorithm [8]. However, the lowest complexity results in performance loss compared with the soft-decision decoding algorithms, such as the sum-product algorithm (SPA) [11], which provide better performance but require higher computational complexity.
Recently, several reliability-based decoding algorithms have

been proposed to offer a trade-off between computational complexity and performance. The reliability-based iterative majority-logic decoding (RBI-MLGD) algorithm and weighted iterative soft-reliability based majority-logic decoding (WISRB-MLGD) algorithm for binary LDPC codes recently proposed in [12] [13] offer a trade-off between complexity and performance. Similarly, reliability-based decoding algorithms for nonbinary LDPC codes have also been investigated, see [9] [14] and references therein.
In this paper, a new approach is presented to further reduce the decoding complexity for algebraically constructed LDPC codes. The algorithm is based on the fact that the Tanner graphs of algebraically constructed LDPC codes usually have redundant check nodes. Given a criterion, all the check nodes are classified into active nodes and silent nodes accordingly. In the iterative procession, only active check nodes are involved. That is, each variable node collects messages from active check nodes and passes extrinsic messages to its active neighbors. In principle, the presented adaptive decoding algorithm can be adapted to any existing decoding algorithms for LDPC codes with redundant check nodes. In this paper, we will take the reliability-based iterative min-sum decoding (RBI-MSD) algorithm [10] as an example to illustrate the advantages of the adaptive decoding algorithm.
II. GENERAL DESCRIPTION OF THE ADAPTIVE DECODING ALGORITHM
A. Basic Concepts and Notation
An LDPC code C can be defined as the null space of a sparse parity-check matrix H = (hi,j)m×n. Associated with the parity-check matrix is the Tanner graph [15]. A Tanner graph is a bipartite graph which contains two types of node, the variable node Vj, (0 ≤ j < n) and the check node Ci, (0 ≤ i < m). The variable node corresponds to a column of H and the check node corresponds to a row of H. For any paritycheck matrix, we define the following two sets
Ni = {j : 0 ≤ j ≤ n − 1, hi,j = 1}
for each row i of H and
Mj = {i : 0 ≤ i ≤ m − 1, hi,j = 1}
for each column j of H.

C. Possible Applications
Imagine that a transmitter attempts to broadcast a common message to a couple of receivers over different AWGN channels. Assume that all the receivers are energy-limited and have a performance requirement around BER = 10−5. Suppose that the message is encoded by a majority-logic decodeable LDPC code, say, the (961, 721) code we have simulated above. When one receiver collects the the noisy codeword, it can start decoding by selecting partial rows of the matrix. If the channel is good, then it can start the decoding with, say, r1 = 500 rows. If the decoding fails, the receiver can try the decoding with incremental rows, say r1 = 750. On average, this adaptive decoding algorithm can reduce energy consumption in the decoding process when the channel condition is good.

IV. CONCLUSION
In this paper, we have presented an adaptive decoding algorithm for LDPC codes with redundant check nodes. Two approaches are introduced to distinguish the check nodes between active nodes and silent nodes. One is dependent of the received noisy codewords. The other is just simply selecting partial consecutive rows from the original parity-check matrix. In the adaptive decoding algorithm, only active check nodes are involved in the iterative procession.
