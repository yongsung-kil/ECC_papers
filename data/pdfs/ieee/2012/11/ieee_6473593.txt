A Novel FPGA Implementation of Mirror-Paradigm RS-Based QC-LDPC Decoder for NVM Channels

AbstractŌĆöThe scrutiny of the class of Reed-Solomon (RS) based quasi-cyclic (QC) low-density parity-check (LDPC) codes has inspired the authors to propose a memory efficient mirrorparadigm (MP) RS-based QC-LDPC code by exploiting the geometrical properties from the RS-based QC-LDPC nomenclature. Without any loss in performance, the proposed MPRS-based QC-LDPC code delivers discernible memory savings that address the concerns of hefty H-matrices associated to lengthy codewords for non-volatile memory (NVM) applications. Besides, the MPRS-based QC-LDPC codes are not confined to any particular hardware implementation and are compatible with various decoder architectures to complement other optimization schemes.
Keywords-Reed-Solomon(RS); low-density parity-check(LDPC) codes; mirror-paradigm(MP); non-volatile memory(NVM).
I. INTRODUCTION
The renaissance of low-density parity-check (LDPC) codes marks a new chapter in the history of modern channel coding. Renowned for its near-capacity performance on a diversity of data transmissions and storage channels, concerted endeavors of coding theorists and practitioners have derived state-of-theart LDPC code designs that offer LDPC codes with lowcomplexity encoding, good waterfall regions and low error floors [1]-[5]. Among these propitious LDPC codes, the class of Reed-Solomon (RS) based quasi-cyclic (QC) LDPC nomenclature [6] is a subclass of QC-LDPC codes based on the structure of RS codes with two information symbols. RSbased QC-LDPC codes perform well with iterative decoding [7] over the AWGN channel and exhibit fast decoding convergence, with good minimum distances and girth properties to boot, which are crucial features in high-speed communication system, such as non-volatile memory (NVM) channels, where very low error rates are required. In particular, a RS-based QC-LDPC code has been adopted by 10 Gigabit Ethernet over copper (10GBASE-T), which is standardized by the IEEE 802.3an task force [8].
The parity check matrix, otherwise known as the H-matrix, of a RS-based QC-LDPC code is simply a compilation of

circulant permutation matrices, which admits significant memory savings as its H-matrix can be unambiguously described by the first row of every circulant permutation matrix via cyclic shifting. However, the codewords for NVM applications tend to be lengthy, ranging from 4000 bits to 32000 bits. As a result, its corresponding LDPC decoder may encounter memory conundrums. Hence, a new methodology for designing memory efficient RS-based QC-LDPC codes for the likes of NVM is highly demanded.
In this paper, a novel mirror-paradigm (MP) RS-based QCLDPC code is derived. The intent of the MPRS-based QCLDPC code is to reduce the storage requirements of its Hmatrix by exploiting mirror-paradigms, where adjacent row generators are mirror images of one another. Moreover, since the RC-constraint is not violated, the performance of the proposed MPRS-based QC-LDPC code is on par with the original RS-based QC-LDPC code across various channels.
This paper is organized as follows. Section II presents one of many code design techniques based on RS codes. The proposed design of a memory efficient MPRS-based QCLDPC code, along with its implementation and advantages, are covered in Section III and Section IV, respectively. Last but not least, the conclusions are given in Section V.

IV. ARCHITECTURE FOR MPRS-BASED QC-LDPC DECODER
In this section, a case study of a lengthy MPRS-based QCLDPC that is viable for NVM channels is presented to exemplify the advantages of exploiting the mirror-paradigms.

Consider Žå = 67 such that a (67,2,66) RS code Cb over the prime field GF (67) based on the set of polynomials of degree 1 or less over GF (67) is generated. With this base RS code, a 67 ├Ś 67 array H rs of 67 ├Ś 67 circulant permutation matrices can be obtained. In addition, for m = 6 , a 6 ├Ś 67 array of 67 ├Ś 67 circulant permutation matrices can be elicited to form a 402 ├Ś 4489 MPRS-based QC-LDPC code by selecting any pair of mirrored-row generators. From (9), only 199 elements instead of all 804 elements are required to span the entire H-matrix of this LDPC code for decoding. Consequently, due to the mirror paradigms, the memory addresses generated through cyclic shifts can be transposed and shared among mirrored elements, rendering a drastic curtailment in decoder resource as enumerated in Table II. V. CONCLUSION
The MPRS-based QC-LDPC code, a subclass of the RSbased QC-LDPC code, has been presented in this paper. Apart from addressing the storage requirements of its H-matrix, the MPRS-based QC-LDPC code reduces the amount of resources required by the decoder by more than 75% in most cases as compared to the original RS-based QC-LDPC code in [6], without sacrificing any of its propitious attributes or code performance. Furthermore, it works well with various decoder architectures and can be integrated with other optimization schemes to invoke concerted improvements.
