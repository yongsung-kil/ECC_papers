Iterative Detection Scheme with LDPC Codes for Two-Dimensional Interference Channels
Jun Yao, Kah Chan Teh, and Kwok Hung Li School of Electrical & Electronic Engineering, Nanyang Technological University, 639798 Singapore
Email: { yaoj0003, ekcteh, ekhli }@ntu.edu.sg

Abstract‚ÄîIn this paper, an algorithm that performs joint iterative detection and decoding for 2D interference channels is proposed. Low-density parity-check (LDPC) codes are adopted as the error control codes in this system. The density evolution technique is further employed to track the message densities in the iterations. Using the density evolution technique, a threshold can be calculated which can be used to predict the performance of the system with long code length. Simulation results demonstrate that the proposed detection scheme outperforms some existing schemes. The density evolution technique is effective in analyzing the system performance.
I. INTRODUCTION
Two-dimensional(2D) inter-symbol interference (ISI) channels represent a class of channels in data storage and communication systems in which the channel memory comes from two directions. Some examples of these channels are two-dimensional magnetic recording (TDMR) [1], 2D optical storage (TwoDOS) [2] and patterned media storage [3] in data storage systems, and multiple-input multiple-output (MIMO) [4] and filter bank multi-carrier (FBMC) [5] in communication systems.
It has been proved that the optimal maximum-likelihood sequence detection (MLSD) for 2D ISI channels is NPcomplete [6], which is impossible to be implemented in practice. The performance bound of MLSD for 2D interference channels has been analyzed in [7]. Inspired by the great success of turbo codes [8], the iterative technique has been applied to many circumstances and achieved satisfying performance [9]-[11]. Several iterative detection algorithms have been proposed in [12]-[15]. Specifically, in [14] and [15], the authors proposed an iterative row-column soft decision feedback algorithm (IRCSDFA) which is a concatenation of two soft-input/soft-output (SISO) detectors. The soft information exchanged between the two detectors is scaled down by a weight factor which is normally a function of the iteration number. In [16], a Markov chain Monte Carlo (MCMC) based 2D detection algorithm was proposed. In this work, the a posteriori probability of the desired binary bit is converted to the expected value of a function. Using the Monte Carlo techniques, the calculation of the expectation is approximated by taking an average of the samples. It has been shown in [16] that the MCMC-based algorithm outperforms the IRCSDFA which has been reported to have better performance than other previous detection algorithms.
In this paper, we propose a multiple joint iterative detection/decoding scheme with low-density parity-check (LDPC)

However, unlike the IRCSDFA in which the two constituent detectors are concatenated in parallel, in our detection algorithm, the two constituent detectors are serially concatenated and the soft information exchanged between the two constituent detectors is not weighted. Simulation results show that our proposed iterative detection scheme outperforms the MCMCbased method and the IRCSDFA. Moreover, in this paper, we also employ the density evolution technique [17], [18] to analyze the performance of our proposed detection scheme. The threshold calculated using density evolution technique can be shown to well approach the simulation result when the code length is large.

VI. CONCLUSION
In this paper, we have proposed a joint iterative detection/decoding scheme for a generalized discrete 2D interference channel. The channel detector is composed of two constituent detectors that work in the down-track direction and cross-track direction, respectively. LDPC codes have been adopted to further improve the system error performance. Iterations can be conducted between the two constituent detectors and among the two constituent detectors with LDPC decoder to achieve better performance. We have also adopted the density evolution technique to track the message densities in the iterations. Simulation results have shown that our proposed iterative detector outperforms the MCMC-based method and the IRCSDFA. The threshold calculated using density evolution technique has also been shown to accurately predict the system performance for large code length.
