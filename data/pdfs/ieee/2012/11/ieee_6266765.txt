LDPC-Based Lossless Compression of Nonstationary Binary Sources Using Sliding-Window Belief Propagation
Yong Fang

Abstract— Low-density parity-check (LDPC) codes have been used to implement lossless distributed or conventional source coding. However, block-wise LDPC codes are difficult to adapt to varying source statistics as traditional symbol-wise entropy coding techniques. In this paper, we propose the sliding-window belief propagation (SWBP) algorithm which is able to simultaneously recover the source and refine the estimate of varying source statistics. The SWBP is easy to implement and performs well in simulations.
Index Terms—Lossless source coding, LDPC, belief propagation, correlation estimation, Slepian-Wolf coding.
I. INTRODUCTION
T HIS letter researches lossless distributed/conventional source coding (DSC/CSC) using LDPC codes. As shown in [1], the LDPC-based lossless DSC problem [2] can be converted into an LDPC-based lossless CSC problem [3] by XORing the syndrome of side information with the syndrome of source. For this reason, we will concentrate only on LDPCbased lossless CSC problem in the rest of this letter.

The principle of LDPC-based lossless source coding [3] is to multiply source x with an m × n parity-check matrix H to get syndrome s = {si}m i=1 = Hx. At the decoder, the belief propagation (BP) algorithm is run to recover x from s [2], provided that variable nodes are seeded with local bias probabilities p.
For block-wise LDPC codes, it is difficult to obtain online estimates of local bias probabilities as symbol-wise entropy coding techniques, e.g. adaptive arithmetic coding. In [1], [4], the authors address the estimation of global crossover probability between source and side information (SI) for SlepianWolf coding (SWC) [5], which can be directly applied to lossless CSC. In [6], [7], the particle-based belief propagation (PBP) algorithm is proposed to estimate channel noise adaptively. Considering its complexity, a parallel implementation

In this letter, we propose the sliding-window belief propagation (SWBP) which first seeds variable nodes with a coarse estimate of global bias probability (not necessary accurate) and then refines the estimate of local bias probability of each variable node from the overall beliefs of its neighbors after each standard BP iteration. Thus as the decoding proceeds, the estimates of local bias probabilities become more and more accurate. The SWBP is very easy to implement and shows near-optimal performance in experiments. In addition, we find that the performance of the SWBP is insensitive to the initial setting. These merits make the SWBP very attractive in practice. Compared with the PBP, the main advantage of the SWBP lies in its low complexity.

Finally, Section V concludes this letter.
II. SLIDING-WINDOW BELIEF PROPAGATION
A. Notations
Before introducing the SWBP algorithm, we define the following notations:
• r = {ri}ni=1: overall (including both intrinsic and extrinsic) beliefs of variable nodes;
• x˜ = {x˜i}in=1: estimate of x; • p˜ = {p˜i}in=1: estimate of p; • Ji: set of the indices of check nodes connected to xi; • Ij: set of the indices of variable nodes connected to sj; • aij : belief propagated from xi to sj; • bji: belief propagated from sj to xi.

III. ADAPTIVE WINDOW-SIZE SETTING
As we see, window-size w plays a key role in the SWBP. The setting of w is a tradeoff: on one hand, w should be small enough so as to quickly track varying source statistics; on the other hand, as p˜i is obtained by averaging random variables ri in a size-w window around xi, w should be large enough to depress the fluctuation of p˜i. In this section, we first describe our algorithm for window-size setting and then justify it theoretically. Finally, some practical implementation isssues of the SWBP is discussed.

V. CONCLUSION
This paper proposes the SWBP to simultaneously recover source and estimate local bias probabilities. We also propose an adaptive algorithm to set sliding-window size in the SWBP. The SWBP is simple and easy to implement. Experimental results show that the SWBP performs very well and is insensitive to initial settings. These merits make SWBP very attractive in practice. Moreover, due to the duality, the SWBP can also be applied to SWC and channel decoding.
