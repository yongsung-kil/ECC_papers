Abstract— Puncturing is an effective method to achieve rate compatible. However, the existing puncturing algorithms are complex and difficult to implement by hardware. This article researches on puncturing algorithm for QC-LDPC codes with dual-diagonal structure. Taking LDPC codes under IEEE802.16e standard as an example, this paper proposes a simple puncturing algorithm to achieve bit-rate compatible. Based on dual-diagonal structure, the puncturing algorithm chooses variable nodes with low degree for deletion and follows the rule that the deleted variable nodes should not be in the same calibration equation as far as possible. The simulation shows that the proposed puncturing algorithm is simple and effective, and the sub-codes obtained through the puncturing have better BER performance than that of random puncturing and order puncturing. In addition, because the puncturing matrix does not need to send to the receiver, it reduces a lot of overhead. Overall, the proposed puncturing algorithm is very practical to achieve rate compatible.
Keywords- Low Density Parity-Check Code; dual-diagonal; rate compatible; puncturing
I. INTRODUCTION
Adaptive technology can guarantee the communication quality and increase the spectral efficiency and throughput as far as possible. Constructing the excellent channel codes which possess the performance of adaptively changing the code rate is the basis of adaptive modulation and coding (AMC). This article chooses LDPC codes as the channel coding due to its good performance of approaching the Shannon limit [1, 2] and its excellent properties. The methods of constructing rate compatible codes include the check matrix expansion and puncturing. This paper mainly researches on puncturing method.
The puncturing method points that a number of bits (usually parity bits) of the encoded code word (mother code) are deleted and the remaining code words are transmitted. And then resume all code words at the receiver. The higher bit rate code words can be got through puncturing, by which codes with different error correction capabilities can be obtained. By puncturing at the sending end, we can achieve the required error protection capability. This method is simple and flexible; code words with superior performance can be generated through the rational designing of the puncturing matrix

(puncture pattern). The existing puncturing algorithms possess high complexity both for theory and hardware implementation. On the basis of reducing the delay and computational complexity, this paper presents a simple puncturing algorithm for QC-LDPC codes with dual-diagonal structure. Now we analyze several different puncturing methods firstly.
II. ANALYSIS OF SEVERAL DIFFERENT PUNCTURING ALGORITHMS
Puncturing is one of effective ways to achieve variable code rate. Hagenauer applied puncturing method to convolutional codes, and first proposed rate compatible punctured convolutional code (RCPC) [3]. The concept that RCPC adjusts the bit rate through puncturing provides ideas for the subsequent error correction code development, such as rate compatible of punctured Turbo Codes (RCPT) [4] and rate-compatible LDPC codes (RC-LDPC) [5]. From the idea of puncturing algorithm, the receiver needs to

Although puncturing can achieve bit rate compatible, it

accompanies by the loss of error correction performance. In

order to minimize losses, it is critical to search a puncturing

matrix with excellent performance. The existing puncturing

methods consist of random puncturing, order puncturing,

grouping and sorting program proposed by Ha [6] and its

improvement programs. Another scholars apply puncturing

with the best puncturing or the puncturing matrix obtained by

density evolution. Although they possess excellent

performance, they are very complex both for theory and

hardware implementation. A brief introduction is given for

several common puncturing methods as below.

A. Random puncturing algorithm
Random puncturing, the position of deleted bits is randomly selected to construct high-rate LDPC codes from the low bit-rate mother code. Although this puncturing can guarantee that the bits are deleted with the same probability, there may lose important nodes to make the BER performance worse. What’s more, the receiver needs to know the position of deleted bits (puncturing matrix). So puncturing matrix should be sent to the receiver in each transmission, which will lead a waste of system resources. It is not practical.

B. Order puncturing algorithm
Based on the analysis of the decoding performance for irregular LDPC codes, from the view of variable nodes, the higher degree of variable nodes is, the more parity information is obtained from the check nodes to achieve faster and reliable decoding. Therefore, some scholars have put forward that removing the variable nodes with low degree has a slightest impact on decoding, which made the order puncturing method [7]. The basis of order puncturing is to construct parity check matrix with column weight increasing, and the variable nodes are deleted from ones corresponding to the minimum column weight sequence. This puncturing method is relatively simple, which does not need to transfer puncturing matrix to the receiver. But it has performance loss relative to the random puncturing.

C. Grouping and Sorting algorithm Ha and others proposed Grouping and Sorting program
through the study of the process of node recovery in LDPC decoding. It utilizes optimized search algorithm to determine the best deleting nodes. Grouping algorithm divides variable nodes into different collections and maximizes the number of nodes in each collection in order. In each collection, the order of the deleted nodes is determined by the Sorting algorithm. In order to describe the transmission of information in the recovery process of the puncturing nodes, [8] introduces the concept of recovery tree. Figure 2 shows an example of a recovery tree for a 3-SR node.
Figure 2. a recovery tree for a 3-SR node
Under the same conditions, the more k-SR nodes are, the less number of iterations is required in the recovery and decoding process, which relieves the problem of reliability descending of the high-k-SR node due to the correlation in the recovery process. So it obtains better performance of the puncturing.
D. Improved puncturing algorithms The program of Grouping and Sorting proposed by Ha
makes the number of low k-SR nodes maximization, with which error correction performance of the sub-codes has a great gain relative to random puncturing. However, this program promises a survival check nodes, and the structure of check matrix will limit the number of nodes deleted result in the smaller dynamic range of bit rate. Based on the above problems, many scholars proposed new puncturing algorithms. Through studying the functions of several survival check nodes, [9] proposed the MSCN puncturing program to maximize the number of survival check nodes of the punctured nodes. In order to solve the problem of limitation of the highest punctured code rate, [10] proposed the hierarchical selection algorithm.
There are scholars researching on puncturing algorithm for dual-diagonal structure of LDPC codes, such as [11] studies the puncturing algorithm for LDPC codes under DVB-S2 standards and [12] for LDPC codes with block type. Although some puncturing methods above have excellent punctured performance and a wide range of applications, the theory is more complex and with high complexity of hardware implementation. So this article presents a simple puncturing algorithm of LDPC codes with dual-diagonal structure. Before introducing the new puncturing algorithm, the basic theory on

III. THE QC-LDPC CODES WITH DUAL-DIAGONAL STRUCTURE

QC-LDPC (Quasi-Cyclic LDPC codes) is a type of structured construct LDPC codes, which is typical and practical. The parity check matrix of QC-LDPC code is constituted by a group of cyclic matrixes, whose quasi-cyclic nature makes it easy to be efficiently encoded and decoded. And the algebraic structure of codes makes it easily implemented in hardware. The QC-LDPC codes with dual-diagonal structure can guarantee the parity check matrix full rank, which were applied to DVB-S2 standard, IEEE802.16e and IEEE802.11n standard and so on. This article researches on puncturing algorithm for LDPC codes with dual-diagonal structure taking LDPC codes under IEEE802.16e standard as an example. IV. NEW METHOD OF PUNCTURING
The new puncturing program based on the following principles: 1) Removing the variable nodes with low degree has a
slightest impact on decoding; 2) The punctured variable nodes should not be adjacent to
the same check node as far as possible. It is that the deleted variable nodes should not be in the same calibration equation as far as possible to reduce the impact to the recovery of the punctured nodes; 3) Maximize the number of Low-k-SR nodes. The right 11 columns of the base matrix (rate 1/2) under IEEE802.16e standard construct the dual-diagonal structure, whose column weight is 2 corresponding to the part of variable nodes with lowest degree. Therefore, this article
locates the range of punctured codes at the right 11* p bits
of the code word (rate 1/2), which is shown in Figure 4.

Figure 4. the range of punctured codes
Based on the dual-diagonal structure, we find
that b(1) and b( p 1) are in the same calibration equation,
which are adjacent to the first common check nodes. Similarly,
b(2) and b( p  2) are adjacent to the second common check
nodes, and so on. This paper presents a simple puncturing algorithm based on the principle of puncturing optimization. Take LDPC code (rate 1/2) under IEEE802.16e standard as the mother code. The punctured variable nodes are determined in blocks:

In addition, because the puncturing matrix does not need to send to the receiver, it saves a lot of overheads. The following simulation proves the feasibility and superiority of this new puncturing algorithm.

VI. CONCLUSIONS
Constructing the excellent channel codes which possess the performance of adaptively changing the code rate is the basis of adaptive modulation and coding. This article researches on puncturing algorithm for QC-LDPC codes with dual-diagonal structure. Taking LDPC codes under IEEE802.16e standard as an example, this paper proposes a simple puncturing algorithm to achieve bit-rate compatible. The puncturing algorithm chooses variable nodes with low degree for deletion and follows the rule that the punctured variable nodes should not
