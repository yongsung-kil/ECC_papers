Abstract: 
For some communication services, a class of channel 
coding corresponding to the variable code length is needed. 
Variable 
length 
LDPC 
codes 
construction 
algorithm 
is 
proposed for the need, which determines the shift factor in the 
parity check matrix according to short cycles and extrinsic 
information degree of the tanner graph, effectively avoiding 
the cycle of length 4 and cycle of length 6. Compared with 
802.16e LDPC codes, rate of 112 codeword constructed by the 
algorithm has lower bit error rate and larger code length 
dynamic range, Which provides a valuable encoding scheme 
for communication service of variable information length.

1. 
Introduction 
Low density parity check code was first proposed by 
Gallager in 1962 [1]. But it didn't get enough attention 
because of its computation complexity. Mackay and Neal 
rediscovered 
the 
value of LDPC 
in 1995 
[2]. 
Then 
researchers started to focus on it. As the research becomes 
deeper, LDPC has been widely used. GPS L 1 C signal 
published in 2006 uses 112 rate LDPC. 802.16e standard 
adopts structured quasi cyclic LDPC (QC-LDPC). The 
special structure of check matrix has two advantages: 1. It 
supports fast encode algorithm with linear complexity. 2. 
The base matrix can be expanded as check matrix for 
different code length. Fossorier has plenty of research of 
QC-LDPC [3], which focuses on the influence of girth. Tao 
Tian proposed the defmition of extrinsic message degree 
(EMD) [4]. Constructed code words based on EMD has low 
error floor. Published code constructing algorithm [7], [8], 
[9] usually can't support code length variable LDPC. 
There are two methods of LDPC construction. The 
first one is deterministic construction. The check matrix can 
be directly constructed by some math expression. It's 
method 
is 
valuable 
in 
theory 
research, 
but 
codes 
constructed by this method usually have poor performance. 
The second method is constructing codes by computer 
search in some constraint conditions. Sometimes constraint 
conditions are too strict to find out proper codes. If the 
constraint conditions are set up properly, codes of good 
performance 
will 
be 
find 
out. 
We 
have 
proposed 
a 
QC-LDPC construction algorithm in the second method. It 
constructs code words (GEO-LDPC) based on girth and 
EMD. Compared with LDPC in 802.l6e, GEO-LDPC has 
lower bit error rate and larger code length dynamic range.

The edge of Tanner graph corresponds to I in the 
check matrix, which stands for a connection between 
variable nodes and check nodes. The real lines in Fig.1 
construct a close path, which is called a ring. The minimum 
length of ring is girth. LDPC uses iterative decoding, But 
the existence of short girth deteriorates the performance of 
iterative decoding. Different girth has different influence on 
the performance. Commonly, girth of 4 and 6 have worse 
influence on the performance than other length. During the 
process of iterative decoding, message passes in a close 
path, in which error will cumulate and lead to decoding 
failure.

based on girth and EMD. Code constructed by this method 
is GEO-LDPC code. The base principle of the algorithm is 
using density evolution and PEG construction algorithm to 
decide the location of nonzero value in base matrix. Choose 
best shift value for each nonzero value element according to 
the information of girth and EMD. For rate 112 GEO-LDPC,

4. 
Simulation and analysis 
We have simulation of GEO-LDPC code and LDPC 
code in 802.16e. For minimum and maximum code length 
of LDPC code in 802.16e, GEO-LDPC have better bit error 
rate. We have simulation by using minimum sum decoding 
with normalized factor 0.8. Simulate 100000 code words 
for each code length and iteratively decode 100 times for 
each code. The simulation result is shown in FigA and 
Fig.5. 
As the result shown in Fig.5, GEO-LDPC has lower 
decoding converges threshold by comparison with LDPC of 
802.16e. GEO-LDPC has better degree distribution which 
offers a better code word space. The construction algorithm 
finds a set of code with best capability in the space. Fig.6 
has shown the bit error rate performance of GEO-LDPC in 
its dynamic code length range. For 3200 bits code length, 
GEO-LDPC has converged to error floor in 2.2 dB and the 
error floor can be below 10-9•
