Modified BP Decoding Algorithm for Satellite Communication

Wang Yu, Li Guangxia, Zhu Hongpeng
Institute of Communication Engineering PLA University of Science and Technology
Nanjing China magicwy87@gmail.com

Abstract—For regenerative transfer satellite communication system which uses LDPC code, Receivers suffer a bad decoding performance because of failing to estimate the channel correctly. It proposes a satellite communication system model and modifies traditional belief propagation decoding algorithm based on logarithm likelihood ratio. By using proper initial logarithm likelihood ratio, the system gets a lower total bit error rate. When the bit error rate is 10−5 , the modified decoding algorithm offer 0.4 dB encoding gain.
Keywords-component; regenerative transponder; low density parity check; belief propagation;logarithm likelihood ratio;
I. INTRODUCTION
Transponder is a key component of satellite communication system. Telephone, telegraph, data and photo are transferred from one user to the destination by transponders. Number of communication satellite transponder is a important factor to indicate the capacity of communication. Current communication satellite usually own one hundred scale transponders. Transponders include two kinds. The first one is transparent transponder. It amplifies or changes the frequency of the received signals. It just fulfils the task of transferring without processing the signal. So it’s transparent to all the signals. The other kind is regenerative transponder. It transfer the signal meanwhile processes the received signals. Broadband ‘Skyplex’ of European satellite communication organization, IPStar and Spaceway3 broadband communication satellite all use regenerative system. Regenerative transponders have the following advantages.
a) It demodulate the signal to regenerate the original date, which will eliminate the noise added to the uplink signals. The quality of the whole communication system will be improved.
b) By regerating signals, forward link is independent from reverse link. Links can be disigned separatly. It’s convenient to realise different modulation and multiple access.
c) Resources like channel, frequence, power, and beam can be assigned dynamicly. It will help to make the best use of finite satellite resources.
d) With processing signal on-broad, links between satellites can be built to realise interplanetary network. It’s important for users which can’t communicate with others

through one satellite. It’s alsp helpful to the ground management center to control the whole system.
Low density parity check (LDPC) code was first proposed by Gallager in 1962 [1]. It’s a method of channel coding with outstanding error correction ability. But it didn’t get enough attentions because of is computation complexity. Mackay and Neal rediscovered the value of LDPC in 1995. Plenty of research started to focus on it after that. Long LDPC codes can achieve perfect coding gain which approximates the Shannon limit. LDPC codes use iterative decoding , which performs iterative message independence assumption. When the iteration number is smaller than the size of the smallest cycle, the assumption is valid. Unfortunately, LDPC codes with good distance properties still include cycles of different sizes. However, when the code block length is short, the presence of cycles of LDPC codes graph brings terrible loss in performance, which leads a obvious gap between iterative decoding and maximum likelihood(ML) decoding. However, for large block lengths, LDPC codes decoded by iterative decoder has achieved the capacities of a variety of channels.
Using LDPC code in regenerative system can reduce the bit error rate in receiver. LDPC code adopts belief propagation (BP) decoding algorithm [2,3]. Though BP algorithm offers good decoding performance, it has quite high computation complexity. If regenerative system doesn’t need to decode LDPC code on the satellite, it will be easier to manufacture the satellite and guarantee the reliability of the satellite communication payload. If the payload doesn’t decode code words on the satellite, the receiver need to estimate the initial channel state correctly to make sure decoding successfully. [4]~[6] has researched LDPC decoding algorithm and modified the estimation of the noise power. But they are not proper for regenerative satellite communication system. [7] has modified the initial likelihood value of probability field. There is no work on modifying the initial likelihood value of logarithm likelihood ratio (LLR). Compared with decoding algorithm based on probability field, decoding algorithm on LLR field owns better numerical stability, and it’s easier to achieve by hardware. We proposed modified decoding algorithm based on LLR field to improve the performance of the regenerative satellite communication system.

IV. THE PERFORMANCE OF MODIFIED DECODING ALGORITHM
Minimum sum (MS) is an approximate method of BP decoding algorithm [8]. MS reduces computation complexity of BP algorithm at the cost of little decoding capacity loss. Simulation uses MS decoding algorithm. The max iteration time is 80. Test 10000 words under different signal to noise ratio(SNR), respectively. Simulation adopts QC-LDPC code of rate 1/2 and code length 720 bits [9]. In satellite communication systems, uplink and downlink usually have the same SNR condition to keep the balance. As the simulation result shown in figure 2, modified decoding algorithm has clear superiority over the original one at low SNR. This is because that modified initial LLR is more accurate than the original LLR estimation, which helps decode correctly. When the SNR condition becomes better, the efficient capacity of LDPC codes plays a powerful role so that the differences between two algorithms are not obvious.
Figure 3 shows the performance of different length LDPC codes. With the code becoming longer, the encoding gain it offers becomes more and more unconspicuous. It’s because that long LDPC codes have obvious advantage in encoding gain. It will help to resist the bad influence of imprecise initial LLR value. Though the précising LLR value makes a difference at the beginning, as the iteration processing the initial influence will be reduced . In view of the above, modified decoding algorithm affect the best at low SNR for LDPC codes of short length.

V. CONCLUSION
Modified decoding algorithm improves the performance of LDPC code by accurately estimating initial LLR value. It’s suitable for the regenerative transfer satellite system. The model proposed in part 3 just use hard decision of simple rules, which affects the decoding performance in some degrees. In further research, hard decision of low performance loss and low complexity should be considered [10,11]. It’s also important to keep the balance of performance and complexity.
