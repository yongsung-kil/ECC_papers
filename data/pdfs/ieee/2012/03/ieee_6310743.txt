Abstract-In this paper, we design an extended class of complexity-optimized bilayer lengthened low-density parity­ check (LDPC) codes by considering upper check nodes with dif­ ferent degrees. We show that by implementing a low complexity relay decoder, our designed codes outperform the rate-optimized codes in a wide range of Eb/No's.
Index Terms-Relay channels, bilayer low-density parity-check (LDPC) codes, complexity measure, complexity-optimized codes, rate-optimized codes
I. INTRODUCTION
The classical relay channel model was originally introduced by van der Meulen in [1]. The next major work was published by Cover and EI Gamal who studied and proved a class of capacity theorems for the degraded relay channel in [2]. More­ over, they proposed a technique for relaying which is called decode-and-forward (DF) and was based on random binning and block Markov superposition coding. In this technique, the source transmits a message both to the destination and the relay in the first time instance. The destination stores the received sequence. The relay decodes the received message and transmits the bin index of this message to the destination in the second time instance. The destinations receives the bin index and with the help of this extra information from the relay, decodes the message of the source.
In recent years, relay channels attracted significant attention and different cooperation strategies and coding schemes were proposed to exploit the cooperative diversity and achieve the achievable rates of relay channels [3-5]. A coding technique which has been proposed to target the achievable rates of the decode-and-forward relaying strategy is based on bilayer low-density parity-check (LDPC) codes. In [7] two classes of bilayer codes were designed for the Gaussian relay channel. These codes are called bilayer expurgated LDPC codes and bilayer lengthened LDPC codes. Expurgation in the context of bilayer LDPC codes translates into decreasing the rate of the original code while maintaining the codeword length. This is achieved by increasing the parity check bits of the code and reducing the information bits by the same amount. Lengthen­ ing means increasing the rate of the original code by increasing

the number of information bits while keeping the number of parity check bits unchanged. In [8], bilayer lengthened LDPC codes were designed and reported for Rayleigh fading relay channels.
In [6], a class of irregular complexity-optimized LDPC codes were introduced by formulating an optimization problem to minimize the decoding complexity through estimation of the number of decoding iterations. This work was extended to the class of bilayer lengthened LDPC codes in [9] where complexity-optimized bilayer codes were designed that re­ duced the decoding complexity at the relay considerably.
In this work, we design a broader class of complexity­ optimized bilayer lengthened LDPC codes by considering dif­ ferent upper check node degrees and study their performance when implemented over codes with long block lengths. The rest of this paper is organized as follows. In Sec. II, we briefly review our channel model as well as the structure of the codes deployed in this work. In Sec. III, we review the optimization problem and propose how it can be further improved. Numerical and simulation results are reported in Sec. IV. We conclude this paper in Sec. V.

V. CONCLUSIONS
In this work, we designed a wide class of complexity­ optimized bilayer lengthened LDPC codes with different upper check node degrees in the bilayer graph and studied their measure of complexity over a wide range of channel noise thresholds. Our numerical results show that these codes require much fewer number of iterations to converge compared to the rate-optimized code reported in the literature. When simulated over codes with long block lengths, these codes also demon­ strate better performance in terms of bit error rates especially at mid-to-high signal to noise ratios.

B. Results on Performance
We have simulated our designed codes over a Gaussian relay channel by generating bilayer lengthened LDPC codes with blocks of length equal to 105, which were transmitted using BPSK signaling. We have shown the bit-error-rate (BER)
vs. Eb/No plots for three pair of codes, (C1, F) , (C2, F) , and (C3, F) in Fig. 4. The number of iterations for each
pair is the number at which the complexity-optimized code
(Ci, i = 1, 2, 3) is optimum, i.e., we have set the maximum number of iterations for decoding (C1, F) to 43, for decoding (C2, F) to 27, and for decoding (C3, F) to 2l.
The simulation results suggest that at low values
of Eb/No, code (F) performs slightly better than the complexity-optimized codes. However, as Eb/No increases (even at bit error probabilities higher than 10-3) , these codes outperform the rate-optimized code (F) . For example, at BER = 10-4, codes (C1) , (C2) , and (C3) achieve a coding gain equal to 0.05dB, 0.16dB, and 0.18dB over code (F) , respectively. Interestingly, as Eb/No increases, the difference
between the coding gains of the complexity-optimized codes
and code (F) increases as the BER curves of these codes
experience steeper waterfall regions in comparison with the
BER curves of code (F) . This phenomenon is due to the
fact that the complexity minimized codes have decreased
fraction of degree-2 variable nodes compared to code (F) .
Additionally, this results in a more open EXIT chart near the origin for complexity-optimized codes which explains the fast
convergence of these codes compared to code (F) .
Although we did not study the convergence behavior of the lower graph code of the destination in this work, we believe that the decoding complexity of this code is less than the complexity of the corresponding rate-optimized code.
