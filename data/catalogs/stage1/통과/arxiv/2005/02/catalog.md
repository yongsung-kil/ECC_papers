## Pseudo-Codewords of Cycle Codes via Zeta Functions

- **Status**: ✅
- **Reason**: 사이클부호(바이너리 LDPC 특수형) pseudo-codeword/fundamental cone 분석, error floor 관련 코드설계 이론(E)으로 이식 가능
- **알고리즘 기여**: ✅ 알고리즘/코드 기여
- **ID**: arxiv:cs/0502033v1
- **Type**: preprint
- **Published**: 2005-02-06
- **Authors**: Ralf Koetter, Wen-Ching W. Li, Pascal O. Vontobel +1
- **PDF**: https://arxiv.org/pdf/cs/0502033v1
- **Abstract**: Cycle codes are a special case of low-density parity-check (LDPC) codes and as such can be decoded using an iterative message-passing decoding algorithm on the associated Tanner graph. The existence of pseudo-codewords is known to cause the decoding algorithm to fail in certain instances. In this paper, we draw a connection between pseudo-codewords of cycle codes and the so-called edge zeta function of the associated normal graph and show how the Newton polyhedron of the zeta function equals the fundamental cone of the code, which plays a crucial role in characterizing the performance of iterative decoding algorithms.

## Idempotents, Mattson-Solomon Polynomials and Binary LDPC codes

- **Status**: ✅
- **Reason**: 바이너리 LDPC 구성 신규 알고리즘(sparseness/rate/최소거리 제어), 코드설계 기법(E) 포함
- **알고리즘 기여**: ✅ 알고리즘/코드 기여
- **ID**: arxiv:cs/0502024v1
- **Type**: preprint
- **Published**: 2005-02-04
- **Authors**: R. Horan, C. Tjhai, M. Tomlinson +2
- **PDF**: https://arxiv.org/pdf/cs/0502024v1
- **Abstract**: We show how to construct an algorithm to search for binary idempotents which may be used to construct binary LDPC codes. The algorithm, which allows control of the key properties of sparseness, code rate and minimum distance, is constructed in the Mattson-Solomon domain. Some of the new codes, found by using this technique, are displayed.
