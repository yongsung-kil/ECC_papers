### B. 알고리즘 요약

1. 시스템: 헬리콥터-위성 채널에서 rotor blade가 주기적으로 안테나를 가려 사각창(rectangular-window) fading(10~30dB)을 유발, blockage ratio γ는 10~25% 범위.
2. 문제: AWGN용으로 설계된 QC-LDPC는 이런 주기적 erasure 오류에 약하고, 기존 GLDPC(모든 체크노드를 성분코드로 치환)는 심각한 rate loss로 실용성 낮음.
3. 채널 모델: `r = h·s + n`, 블록키지 구간과 free 구간을 서로 다른 SNR을 갖는 두 AWGN 인스턴스 (`(Eb/N0)_0`, `(Eb/N0)_1`)로 취급.
4. 핵심기법: `(a,b,p)` QC-LDPC의 층상(layered) 구조에서 **한 층(가급적 중간층)의 SPC 체크노드만** simplex 또는 1차 RM 성분코드 제약으로 치환 (super check node).
5. rate 식 `R=(b-a)/(b+n_C-k_C-p_C-1)`: 한 층만 일반화+puncturing으로 GLDPC 대비 rate loss 완화 (예: 전면치환 0.057 vs CH 0.133).
6. 확장/트릭: simplex 코드는 항상 SPC 제약(all-ones 열)을 만족해 RM과 달리 데이터 전처리 불필요; 성분코드 일부 check bit puncturing으로 rate 추가 회복.
7. 복호: variable/SPC node는 sum-product, super check node는 trellis 기반 MAP(simple MAP로 저복잡도화) 반복 message passing.
8. 분석: 성분코드 AWGN EXIT `I_{E,C}`를 BEC EXIT와 split-encoder 모델로 폐형식 근사(식 7) — Monte Carlo(기존 [7]) 대체로 EXIT 차트 작성 간편화.
9. 결과: rate-1/4(K=843, base (5,8,281) + (16,5)RM)와 rate-2/7(K=122, base (3,5,61) + (15,4)simplex). 25% blockage에서 FER 10^-4 열화 1.49/1.52dB, QC-LDPC 대비 대역효율 28.6% 향상.
10. 한계: HW 미설계, 시뮬만, 위성 blockage 채널 특화 초저rate, super check node MAP 복호 부담.
