### B. 알고리즘 요약

1. 대상은 AWGN 채널 무선 통신용 rate-compatible LDPC의 puncturing 패턴 설계 (mother code: PEG 구성 (3,6)-regular, rate 1/2, length 1024).
2. 문제: puncturing으로 rate 가변성을 얻지만 성능 손실 발생, 특히 저rate에서 punctured node 복구 실패.
3. 배경 정의: punctured node가 이웃 check node로부터 non-zero LLR을 받으면 복구(k-SR), 그 check node를 survived check node(SCN)로 정의. 기존 Ha et al. grouping/sorting은 SCN을 단 1개만 보장.
4. 핵심 관찰: `(k+1)-SR` node의 LLR mean `mu(V')=sum_i mu,i` (식 9) — SCN이 여러 개면 LLR이 커지고 빠르게 evolve하여 error prob 하락.
5. 핵심 식 (10): `Pe(V')=Q(mu(V')/sqrt(2))` — SCN 다수가 오류확률을 낮춤을 정량화.
6. 핵심 기법 MSCN: puncturing 시 각 node의 SCN 수를 최대화. 새 punctured node는 이전 punctured node들의 SCN 감소를 최소화하도록 선택 (SCN이 매번 최대 1만 감소하도록 state (l,p) 관리).
7. check node state (l,p) 도입: l=복구 level, p=SCN 개수. state 기반으로 candidate 집합 갱신 (Update 1/2).
8. 분석 도구: density evolution with Gaussian approximation으로 LLR mean evolution 유도.
9. 결과: BER 10^-5에서 rate 0.6/0.65에서 grouping/sorting 대비 각 0.15/0.2dB SNR gain. 고rate에서는 기존 기법과 성능 근접.
10. 한계: HW 미설계, 시뮬만, AWGN 무선 특화, rate-compatible puncturing 목적으로 NAND 고정 고rate ECC와 무관.
