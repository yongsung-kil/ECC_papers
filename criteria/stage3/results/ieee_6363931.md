### B. 알고리즘 요약

1. 시스템: binary-input(J=2) discrete memoryless channel(DMC) 뒤에 출력 알파벳 크기 I를 K(<I)개 레벨로 줄이는 양자화기 Q를 두는 구조.
2. 문제: 기존 연구는 mutual information 기준으로 양자화기를 최적화했으나, 유한 블록길이 코드의 오류확률 상한과 더 밀접한 random coding exponent(RCE) 기준의 최적 양자화기는 미해결이었음.
3. 핵심 가정: 채널은 이산 무기억(DMC)이며 연속 채널은 충분히 세밀한 양자화로 근사 가능하다고 가정.
4. 핵심 기법 1단: Gallager 함수 `E0(ρ,T) = -log Σ_j p_j (Σ_k T_k|j^{1/(1+ρ)})^{1+ρ}` 를 양자화기 Q에 대해 재정의한 `Ẽ0(ρ,Q)`를 최대화(= `F̃0(ρ,Q)` 최소화)하는 것이 목표.
5. 핵심 식 의미: RCE `Er(R) = max_p max_ρ [Ẽ0(ρ,Q) - ρR]` 은 블록길이 n에서 디코딩 오류확률의 상한 `Pe ≤ 2^{-nEr(R)}` 를 결정하므로, 이를 최대화하는 양자화기가 유한 길이 코드 성능에 더 적합.
6. 핵심 기법 2단: 목적함수가 Qk|i에 대해 concave이므로 최적해는 feasible region의 extreme point, 즉 deterministic 양자화기(각 입력이 확률 1로 하나의 출력에 매핑)이며(Lemma 2), 채널 출력을 특정 순서로 정렬하면 최적 파티션은 연속된 구간(consecutive quantizer)임(Lemma 3).
7. 부수 트릭: 정렬 조건 `P1|1/P1|2 < ... < PI|1/PI|2` 하에서 partial objective function `ι(a→a')`를 정의해 trellis 형태의 동적계획법(dynamic programming)으로 상태값 `Sk(a) = min[Sk-1(a') + ι(a'→a)]` 재귀 계산.
8. 최적화 절차: [13]에서 제안된 mutual-information용 quantization algorithm을 동일하게 재사용(메트릭만 RCE로 교체), Arimoto-Blahut 알고리즘으로 입력분포 p 최적화 병행.
9. 결과: I=5, K=3 수치예제에서 RCE 기준 최적양자화기가 MI 기준 최적양자화기와 다르며, RCE 기준으로 설계 시 오류확률이 더 빠르게 감소함을 확인(예: DMC #2에서 차이 0.01678). 복잡도는 최대 O(I^3), brute-force O(I^{K-1}) 대비 개선.
10. 한계: 이론·수치예제(합성 DMC, 소규모 I/K)뿐이며 실제 LDPC 코드나 디코더에 적용한 시뮬레이션·BER 곡선이 없음. HW 구현이나 NAND 채널 실측 데이터도 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| soft-read 채널 양자화기(quantizer Q) 설계 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | LLR 레벨 임계값 설계 이론과 개념적으로 연결되나 RCE 기준 최적화 알고리즘은 별도 구현 필요 |
| Gallager random coding exponent 계산 | 대응 없음 | Prime ECC는 시뮬레이터로 이런 이론적 exponent 계산 모듈이 없음 |
| 동적계획법 기반 양자화 경계 탐색 | 대응 없음 | 코드에 대응하는 양자화기 설계 알고리즘 모듈 없음 |

> 적용 가치: 낮음. soft-read 임계값 설계와 이론적 연관은 있으나 LDPC 코드/디코더 실험 검증이 전혀 없고 순수 채널 양자화 이론이라 Prime ECC에 즉시 이식할 구체적 알고리즘·수치가 없음.
