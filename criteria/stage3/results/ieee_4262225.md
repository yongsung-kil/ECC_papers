### B. 알고리즘 요약

1. 대상은 longitudinal/perpendicular 자기기록 채널로, ISI·transition noise·electronic noise·media nonlinearity(partial erasure, NLTS)를 포함한 advanced channel model 위에서 QC-LDPC(column weight 3) 부호를 평가.
2. 기존에는 AWGN 채널에서 LDPC의 near-capacity 성능만 검증되었으나, 실제 자기기록 readback 신호는 ISI·transition noise·매체 비선형성 등 복합 손상을 겪으므로 이를 반영한 평가가 필요하다는 문제의식.
3. 채널 모델: 고립 전이 응답을 Lorentzian pulse(PW50, 진폭)로 특성화하고, jitter noise·width variation·AWGN을 포함한 SNR 정의, PE ratio `r`로 partial erasure, NLTS factor `ε`로 전이 이동을 모델링.
4. 핵심 구조는 SOVA(soft-output Viterbi algorithm) 기반 PR(EPR4) 검출기와 QC-LDPC 복호기 사이의 channel iteration(외부 반복)과 LDPC 내부 iteration(inner iteration)을 결합한 2단 반복 복호 시스템.
5. 시스템 전체(SOVA+LDPC+channel model)를 VHDL로 작성해 Xilinx Virtex II 8000 FPGA에 구현, PR target으로 EPR4 사용.
6. 별도의 확장 기법 없음(SOVA+LDPC iterative decoding 구조 자체가 핵심).
7. 구현 디테일: 40MHz 클록으로 동작, HW 특화 최적화(양자화 등)는 본문에 상세 기술 없음.
8. 학습/최적화 절차 없음(고정 QC-LDPC 부호 + 채널 모델 파라미터 스윕으로 평가).
9. 결과: 25 inner iteration + 2 channel iteration 시 정규화 기록밀도 2에서 SNR 17.5dB에 BER 10^-8 도달(BER 10^-11, FER 10^-8까지 평가), throughput 약 37Mbit/sec; inner iteration을 25→5로 줄이면 throughput 5배 증가하지만 BER 성능은 약 1dB 손실.
10. 한계: 자기기록 채널(ISI, transition noise, PE, NLTS) 전용 평가이며 부호율/길이 등 구체 스펙 본문 미기재, NAND 채널·플래시 오류 모델과 무관, 순수 평가/구현 보고로 새로운 디코딩 알고리즘 제안은 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QC-LDPC inner iteration 복호 | `decoder.cpp` `LDPC_Decoder()` | QC-LDPC라는 점에서만 접점, 구체 알고리즘(min-sum 등) 기술 없음 |
| SOVA 기반 channel iteration(외부 검출-복호 반복) | 대응 없음 | NAND ECC는 별도 채널 검출기(Viterbi)와 결합되지 않음 |
| 자기기록 채널 모델(ISI, PE, NLTS) | `channel.cpp` | AWGN/RBER/fixed-error 모델과 근본적으로 다른 채널 특성으로 직접 대응 불가 |
| iteration 수 축소에 따른 throughput-BER trade-off | `decoder.cpp` (iteration 루프) | 조기종료/iteration 감소 설계 시 참고 가능한 정성적 트렌드 정도 |
