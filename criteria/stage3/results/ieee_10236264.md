> 총평: 광 IM/DD 계층 ACO-SCFDM 변조에 특화된 적응형 LDPC puncturing rate-control 논문으로 NAND binary LDPC ECC 이식가치 거의 없음.

### B. 알고리즘 요약

1. 시스템: IM/DD 광전송용 Layered ACO-SCFDM(L-ACO-SCFDM), DHT로 DFT 대체해 복잡도 감소, PAPR 저감·정실수 신호화, 적응형 LDPC 부호화.
2. 문제: puncturing으로 고rate를 얻으면 정정능력이 저하되고, 저rate 모부호를 punc해 만든 고rate 부호는 동일 rate 원래 모부호보다 성능 열세.
3. 채널/구조: L-ACO-SCFDM에서 레이어γ 데이터 길이는 레이어γ+1의 2배(γ=1~R), Yi가 anti-symmetry라 음의 부분 clipping해도 정보손실 없음.
4. 핵심기법: 계층 구조에 맞춘 적응형 분절(segmented) puncturing. 채널평가로 상위·하위 두 레이어 rate `R1,R2`를 별도로 조절해 평균 rate 제어.
5. 식 의미: 평균 rate는 레이어 길이비(2:1) 가중평균, 예 `0.6≈(2/3·2+1/2)/3` — 상·하위 레이어에 서로 다른 rate 모부호를 써 필요한 puncturing 비트를 줄임.
6. 확장: 모부호 rate에 근접할 때 상·하위 레이어가 서로 다른 rate로 punc하여 과도한 puncturing에 의한 PCM 복호 손상 회피.
7. 구현 디테일: IEEE 802.16e 표준 LDPC(rate 1/2~5/6 모부호군)를 기반으로 rate별 검사행렬 선택 puncturing, 2PAM 변조.
8. 최적화: 별도 학습절차 없음(채널평가 기반 rate 선택 규칙).
9. 결과: LDPC 길이 1920, soft-decision 복호 최대 30 iteration. AWGN에서 기존 분절 puncturing 대비 0.3~0.6dB, 광채널(VPI 공동시뮬) 0.5dB, 기존 L-ACO-SCFDM 대비 6~7dB(무부호 대비 최소 6.2dB) 개선.
10. 한계: HW 미설계, MATLAB/VPI 시뮬뿐, 광 IM/DD ACO-SCFDM 계층변조 전용, 표준 802.16e 부호 재사용.
