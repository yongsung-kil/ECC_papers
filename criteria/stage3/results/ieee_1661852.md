### B. 알고리즘 요약

1. 시스템: 이진영상(binary image expansion)된 (n,k) RS 부호(GF(2^m))를 SPA 기반으로 반복 SISO 복호, BPSK/AWGN·Rayleigh-fading 채널 다수 실험(RS(31,25), RS(255,239), RS(255,127) 등).
2. 문제: RS 부호의 패리티체크행렬 `H_b`는 밀도가 높아(dense) SPA를 직접 적용하면 신뢰도 낮은 비트들이 대부분의 check를 saturate시켜 pseudo-equilibrium(수렴 정체)에 빠짐.
3. 핵심 가정: 채널 LLR 크기순 정렬 후 신뢰도 낮은 비트(unreliable bits, set `B_L`)에 대응하는 서브행렬만 Gaussian elimination으로 sparse(주로 열가중치 1~2)하게 변형.
4. 핵심 기법: 매 반복 `l`마다 LLR `L^(l)`을 기준으로 패리티체크행렬을 `H_b^(l) = adapt(H_b, |L^(l)|)`로 재구성 후 SPA로 extrinsic LLR 계산, `L^(l+1) = L^(l) + α·L_ext^(l)` (α는 damping 계수).
5. 기하학적 해석: tanh 도메인 변환 `Φ(L)=tanh(L/2)`로 정의된 potential function `J(H_b,T)`를 최소화하는 2단계 gradient descent로 재해석, `H_b`를 바꿔가며 다른 gradient 방향을 얻어 정체점 탈출.
6. 확장: Degree-2 random connection(비신뢰 비트도 열가중치 2로 확장해 extrinsic 정보 보강), symbol-level adaptation(Forney's algorithm으로 병렬화 가능한 행렬 생성), min-sum 근사(MS ADP)로 tanh 연산 대체.
7. 부수 기법: incorporated HDD(매 반복 HDD 결과도 후보에 포함), partial reliable bits updating(신뢰영역 일부 M비트만 갱신해 연산량 절감).
8. 학습/최적화 없음(고정 알고리즘, damping 계수 α는 실험적으로 튜닝).
9. 결과: RS(255,127)에서 ADP(80,50)&HDD가 KV 알고리즘 대비 FER=1e-4에서 약 1.65dB 개선, RS(31,25)에서 표준 BP는 거의 이득 없으나 ADP&HDD는 HDD 대비 2~4dB 이득.
10. 한계: RS(대수적 HDPC) 부호 전용 설계로 저밀도 LDPC 구조와 무관, HW 설계·합성 없이 순수 알고리즘/시뮬 결과뿐, 장부호·고SNR에서 이득 감소.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SPA/min-sum 기반 반복복호 | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | 표준 min-sum·SPA는 이미 보유한 기법, RS 특유의 행렬 적응 절차는 대응 없음 |
| 패리티체크행렬 신뢰도 기반 적응(Gaussian elimination) | 대응 없음 | Prime ECC는 고정 QC-LDPC H-matrix(`Load_PCM()`) 사용, 반복마다 행렬을 재구성하는 개념 자체가 없음 |
| tanh/gradient-descent 기하학적 해석 | 대응 없음 | binary LDPC decoder와 다른 해석 프레임워크로 이식 대상 아님 |

> 적용 가치: 낮음 — RS(HDPC) 부호 전용 patiry-check 행렬 적응 기법으로 Prime ECC의 고정 sparse QC-LDPC 구조·min-sum 디코더에 이식할 근거가 없음.
