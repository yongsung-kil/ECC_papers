> 총평: 고전 선형블록코드(Golay, RM, BCH, RS 등)가 4-cycle-free Tanner graph를 가질 수 없음을 증명하는 그래프이론 논문으로, LDPC 부호설계/디코더/HW와 직접 관련이 없어 이식 가치가 낮음.

### B. 알고리즘 요약

1. 이진 선형블록코드 `[n,k,d]`와 그 dual code `C^⊥`(minimum distance `d^⊥`)의 관계에서, 코드 C가 4-cycle-free Tanner graph로 표현 가능한지 여부를 다룸(변수: `p=n-k`).
2. 문제: LDPC 코드는 iterative decoding 성능 향상을 위해 4-cycle 없는 Tanner graph가 바람직한데, 고전 블록코드(Golay/RM/BCH/RS 등)의 parity check matrix는 대체로 4-cycle을 포함함 — 이 현상을 일반적으로 증명하는 방법이 없었음.
3. 핵심 가정: Tanner graph는 parity check matrix `H`에 대응하는 이분 그래프(U=check node, W=variable node)로 정의.
4. 핵심 기법: 그래프이론의 **Reiman's inequality**(Theorem 3, 4-cycle-free 이분그래프의 최대 edge 수 상한)를 이용해 C가 4-cycle-free Tanner graph를 가지려면 `p·d^⊥ ≤ sqrt(np(p-1)) + n²/4 + n/2` 를 만족해야 함을 증명(Theorem 4).
5. 핵심 식의 의미: parity check matrix의 각 행(row)은 dual code의 codeword이므로 최소 `d^⊥`개의 1을 가지며, 전체 weight `wt(H) ≥ p·d^⊥`; 이것이 4-cycle-free 그래프의 최대 edge 수(Reiman bound)를 넘으면 4-cycle-free 표현이 불가능.
6. 확장/따름정리: 이 부등식을 만족하지 못하는 경우 임의의 Tanner graph에 존재하는 최소 4-cycle 개수의 하한(Corollary 6)도 유도.
7. 부수 결과: 등식이 성립하는 그래프는 Steiner system `S(2,k;n_u)`의 incidence graph와 일치함(Neuwirth 결과 인용); `C_{S(2,2;v)}` 코드 계열로 tightness 예시 제시.
8. 학습/최적화 절차 없음 — 순수 조합론적 증명(Cauchy-Schwarz 부등식 활용).
9. 결과: [23,12,7]/[24,12,8] Golay, rate≥1/2인 다수 RM코드(3≤m≤9), rate≥1/2인 BCH코드(3≤m≤8), 특정 rate 범위의 RS코드 binary image가 모두 4-cycle-free Tanner graph를 가질 수 없음을 표로 확인.
10. 한계: 필요조건만 제시(충분조건 아님), 실제 디코더 성능/HW 실험 없음, 대상은 대수적 고전 블록코드이며 QC-LDPC 등 이미 4-cycle-free로 설계되는 현대 LDPC 부호 구성과는 직접 관련 없음.
