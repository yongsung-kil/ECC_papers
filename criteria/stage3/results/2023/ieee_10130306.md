### B. 알고리즘 요약

1. 시스템: SMR HDD, 4 Tbit/in² 기록밀도, 128/130 RLL + (30,3)-regular LDPC(code length 4096 byte, col weight 3, row weight 30) 인코딩.
2. 문제: 기존 NN LLR modulator [2]-[4]는 NN 구조·반복횟수가 커서 계산효율이 나쁨 → 반복 디코딩 가속 필요.
3. 채널 모델: PR1 채널, TD-FIR(3×Nt=45 tap), APP 디코더와 SP 디코더 간 turbo equalization(`isp`, `iglobal` 반복).
4. 핵심 기법: trellis 전이확률 `ln γn`(n=1~3 sub-matrix)을 3-layer NN(N1=3,N2=5,N3=1)이 신뢰도 판정 후 선택적 갱신.
5. 갱신 규칙: NN 출력>0.9 또는 <0.1이면 `ln γn' = ln γn × 1.34`(갱신), 아니면 ×1.00(비갱신)으로 신뢰 message 강화.
6. 학습: 오차함수 `E=½(ŷ - y)²`, back-propagation으로 가중치 `w` 갱신(training signal = 기록열).
7. 부수 디테일: `LLRmn`은 `ln γn', ln αn, ln βn`으로 Max-log-MAP 계산, extrinsic에 `Wext=1/col_weight=1/3`, `WLa=0.7` 감쇠로 error propagation 억제.
8. NN 정확도: 올바른 `ln γn` 입력 시 98.4% 갱신 선택, 오류 입력 시 99.9% 비갱신 선택, `isp=7,iglobal=13`에서 정확도 ≈1.00.
9. 결과: NN 적용 시 `isp=7, iglobal=13`에서 no error 달성(기존 [2]-[4]는 isp=15,iglobal=10 필요), SNRS 0.6 dB 개선.
10. 한계: 컴퓨터 시뮬만, HW 미설계, PR 채널·turbo equalization·trellis 전제로 NAND read 채널과 상이.
