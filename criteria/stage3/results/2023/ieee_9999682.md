### B. 알고리즘 요약

1. 시스템: 이진 LDPC, `m×n` PCM `H`, AWGN+BPSK. 시뮬 부호 C1=(2640,1320) Margulis regular, C2=(2048,512) 5G irregular.
2. 문제: ADMM 기반 MP(수학계획) 복호는 복잡도 저감 연구는 많으나 수렴속도(iteration 수) 개선은 미흡, BP 대비 수렴특성 열위.
3. 모델: penalized LP 복호식 `min γ^T x - (α/2)||x-0.5||^2, s.t. P_j x = z_j ∈ check polytope` (norm-2 penalty로 정수해 유도).
4. 핵심기법1: ADMM x-update(2a)와 z-update(2b) 각각에 추가 Bregman divergence `B_ψ(·)`를 삽입한 G-BADMM 프레임워크(식 5a~5c).
5. 핵심식/의미: 추가 divergence는 새 업데이트 `x^{k+1},z^{k+1}`이 직전 값 `x^k,z^k`에서 급변하지 않게 억제 → 수렴 안정·가속.
6. 핵심기법2: `B_ψx=½||x-x^k||²`, `B_ψz=½||z-z^k||²` 선택 시 x-update가 component-wise 폐형해(식 8), z-update는 스케일된 check-polytope projection(식 9)로 병렬 처리.
7. 구현 디테일: ML test(정수해면 penalty 없는 1회 추가 iteration으로 ML 여부 확인), all-zeros assumption 성립, over-relaxation 계수 1.9.
8. 복잡도: iteration당 x-update에 n회, z-update에 2m·dc회 추가 곱셈 → code length n에 선형.
9. 결과: 경쟁 ADMM 계열(VNP/CNP/p-box/QP) 대비 평균 iteration 최소, 복호시간 최소, FER은 약간 더 우수(C1/C2). 파라미터 μ=5, (ρx,ρz)=(6,1), Imax=100.
10. 한계: HW 미설계(향후 layer scheduling 결합 언급), 시뮬만, 통신용 부호(rate 0.5/0.25)로 NAND 고rate 미검증.
