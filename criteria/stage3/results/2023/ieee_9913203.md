### B. 알고리즘 요약

1. 시스템: (N,k) binary LDPC(주로 IEEE 802.16e 576/rate 3/4, 그 외 802.11n 648/324 등), BPSK+AWGN 및 bursty noise 채널.
2. 문제: BP는 정확하나 log/곱셈 많고, MS/NMS는 저복잡도지만 성능 손실. 기존 NN 디코더는 파라미터 과다로 length>100 장부호에 적용 곤란.
3. 핵심가정: model-driven — NMS 반복복호를 Tanner graph 위상 그대로 feed-forward NN 계층으로 unfold(각 edge에 학습 가중치).
4. 핵심기법 1 — NNMS: CN neuron `L(rji)=∏sgn·min(α·|L(qi'j)|)`, VN neuron `L(qij)=L(ui)+Σβ·L(rj'i)` 처럼 edge별 correction factor `αij,βji`를 iteration마다 학습(cross-entropy loss, Adam).
5. 핵심식 의미: NMS의 normalization factor `α`(DE로 1/α≈1.25)를 고정 대신 back-prop으로 학습해 과대추정 메시지를 보정.
6. 핵심기법 2 — SNNMS-LR-Q: layer별로 `α(l-1)`,`β(l)`를 전 edge가 공유(파라미터 수·곱셈기 감소), LeakyReLU를 VN layer 뒤에 삽입(비선형 적합), 입력에 12bit 양자화(3σ rule).
7. 부수 디테일: T iteration을 `2·lmax+2` 계층으로 unfold, 실험상 22 layers/10 iter가 성능-복잡도 균형, 12bit가 최적 양자화(unquantized보다도 우수).
8. 학습/최적화: mini-batch(120 blocks) gradient descent, SNR 0~5dB 다양 데이터로 학습(다양성↑→robustness↑).
9. 결과: NNMS가 NMS 대비 BER 약 1.5dB 개선(더 적은 iter), SNNMS-LR-Q가 32 layer(15 iter) BER 1e-7에서 NNMS 대비 최대 0.4dB 추가 개선, bursty noise에서도 NMS/MSA/SPA 대비 우위.
10. 한계: HW 미설계(TensorFlow 시뮬만), 학습 가중치가 특정 check matrix에 종속(H 바뀌면 성능 붕괴 — 논문 명시), NAND/flash 언급 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| NNMS/SNNMS unfolded NN 디코더 전체 | 대응 없음 | Prime ECC는 고정 min-sum 복호기, NN/딥러닝 디코더 미지원(프로파일 §6 대응없음 예시) |
| CN min-sum + learned normalization factor | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | min-sum 자체는 이미 보유(중복), learned α를 상수 스케줄로 뽑아 넣는 정도만 부분 활용 |
| iteration별 correction factor(α/β) 스케줄 | [ecc_data.h](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `PARAM_LLR`, [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `Get_VNU_Table_Idx()` | 우리도 iteration 구간별 LLR 테이블 보유(중복), 개념적 참조만 |
| 12bit 입력 양자화 | [channel.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) | 12bit는 우리 hard-1bit/soft-2~3bit 범위 초과(soft-4bit+), NAND read 비용상 부적합 |
| LeakyReLU 비선형 모듈 | 대응 없음 | min-sum 파이프라인에 비선형 활성함수 개념 없음 |

적용 가치: **낮음** — min-sum/iteration별 정규화·양자화는 이미 보유(중복) 기법이고, 핵심인 unfolded NN 디코더는 특정 H-matrix 종속에 NAND 무관·HW 미설계라 Prime ECC binary QC-LDPC 파이프라인에 이식 가치가 낮다.
