### B. 알고리즘 요약

1. 대상: (n,k) LDPC 앙상블, edge 양끝 variable/check 차수의 결합분포 `P(Xe=x, Ye=y)`를 독립 가정에서 상관 있는 이변량 분포로 일반화, 채널은 소거확률 `δ`의 BEC.
2. 문제: 기존 density evolution은 edge 양끝 차수가 독립이라 가정 → degree-degree 상관이 성능에 미치는 효과가 미탐구.
3. 모델: 두 종류 stub(type1/type2)로 나눈 configuration model 확장, `G = n/(n-k) = E[Y]/E[X]`.
4. 핵심 기법1(block construction): stub을 차수 내림차순으로 `b` 블록 분할, 블록 내 `q` 비율을 type1로 지정 후 순열 `π`로 블록 매칭 연결 → 상관계수 `ρ(Xe,Ye) = -q`.
5. 식 의미: `π(1)=2,π(2)=1`이면 큰차수-작은차수를 잇는 음(negative) 상관, `π(1)=1,π(2)=2`이면 양(positive) 상관 유도.
6. 핵심 기법2(general construction): 임의의 이변량 분포 `P(Xe=x,Ye=y)`에서 주변분포·차수분포를 역산해 지정된 상관을 갖는 코드 생성(multi-edge type LDPC의 특수형).
7. Density Evolution: 조건부확률로 여러 edge type의 밀도를 단일 node type으로 평균화 → 상관 반영 비선형 재귀식 `α_x^(i)`, `β_y^(i)` 유도, 독립 경우를 특수 케이스로 포함.
8. 이론 결과: 최대 허용 소거확률 `δ*` 하한 정리(`d_v,min≥2`, `δ<1/(d_c,max-1)`).
9. 수치 결과: block(G=3,d=3)에서 `q=0.37` 음상관 시 `δ*=0.3066`(독립 `q=0`은 0.2741); general로 기존 최고기록 `δ*` 0.49553→0.49568, 0.47410→0.48077로 개선. UEP 설계에도 음상관 유리.
10. 한계: BEC 점근이론만, AWGN/NAND·유한길이·복호 HW 검증 없음, soft-decision/정정능력 논의 부재.
