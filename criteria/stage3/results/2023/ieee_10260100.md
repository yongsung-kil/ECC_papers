### B. 알고리즘 요약

1. 시스템: C-band IM/DD 광섬유 위 Gb/s PAM 신호 전송(SSMF, 수 km), 수신 DSP에 LDPC 기반 SD-FEC 사용.
2. 문제: 대역폭 제한·분산으로 인한 ISI가 용량 증가의 주요 한계이고, 성능 좋은 soft-output MLSE(sMLSE)는 복잡도가 hard MLSE의 2배로 구현 부담이 큼.
3. 모델: memory length `L`의 trellis, branch metric `gl(s',s) = (yl - ys)^2 / (2*d^2)` (가우시안 잡음), forward metric `al(s)`와 backward metric `bl(s)`를 max-log BCJR로 재귀 계산.
4. 핵심 기법: threshold detector로 발생확률 높은 상위 3개 상태(`P` states)만 남겨 trellis state 수를 `M^L`에서 `P^L`로 압축.
5. 의미: 상태 압축으로 forward/backward 양방향의 branch metric 계산량을 동시에 줄여 sMLSE 총 복잡도를 낮춤.
6. 결과: 곱셈 연산이 `M^L`에서 `P^L` 수준으로 감소, 계산 복잡도 약 50% 감소하면서 conventional sMLSE와 거의 동일한 BER 성능.
7. 실험: AWG(64 GSa/s)→RFA→MZM→SSMF→PD→DPO 오프라인 처리, SD-FEC 및 HD-FEC 임계 도달 확인.
8. 확장: 고차 변조에서 상태 압축 효과가 더 커져 복잡도 절감폭 증가.
9. LDPC: overhead가 있는 LDPC를 SD-FEC로 채택했을 뿐 부호 설계/복호 알고리즘 기여는 없음.
10. 한계: HW 미설계, 광통신 채널 특화, LDPC 자체 개선 없음.
