## B. 알고리즘 요약

1. 시스템은 single-cell massive MU-MIMO 상향링크, `M'` UE 대 `N'` BS 안테나, QPSK, 채널코딩 미적용, 복소모델을 PAM 실수모델로 변환.
2. 문제: Bayes-optimal AMP 검출은 배정밀도 연산 필요로 전력·지연이 커서, 유한정밀도(정수) 메시지 패싱 구현이 요구됨.
3. 모델: 간섭+잡음을 CLT로 실수 Gaussian 근사, 유효분산 `ψ(k)`로 belief 계산, tanh 기반 Bayes denoiser로 soft replica 갱신 (Alg.1).
4. 핵심 기법: AMP의 모든 내부 연산(덧셈/곱셈)을 2-to-1 LUT로 분해해 계층적(다층) referential AMP 구조 구성, 정수 라벨만 교환.
5. 핵심 식/의미: 양자화 함수 `u = Q(a)` (식5)와 대표값 설계가 성능을 결정 - 누적 양자화 오차 최소화가 관건.
6. 확장: 합산 연산을 tree 구조(깊이 `T`)로 표현해 LUT 수를 `2^{2B}·⌊log2 J⌋`로 축소, 상관채널에는 ASB(`µ(k)=κ1·(k/K)^κ2`) 도입.
7. 부수 트릭: 비균일 양자화 채택, 오프라인 LUT 설계로 구성 복잡도는 무시.
8. 학습/최적화: DDT로 각 층 실측 히스토그램 추적 후 k-means++ 초기화 + Lloyd-Max 클러스터링으로 MSE 최소 임계값 결정(비지도).
9. 결과: (M',N',B)=(64,96,5) 시 307KB 메모리, B=5에서 배정밀도 대비 BER=1e-4에서 약 1.0dB 열화, B=6 점근, 상관채널에서도 error floor 대폭 저감, 수렴속도 EPQ/IB 대비 빠름.
10. 한계: HW 미설계, 시뮬만, 무선 채널(Rayleigh/one-ring/finite path) 특화, LDPC ECC 대상 아님.

| DDT 기반 LUT 양자화 임계값 설계 | [ecc_data.h](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `PARAM_LLR`, [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `Get_VNU_Table_Idx()` | 양자화 테이블 설계 개념은 유사하나 대상 알고리즘(AMP)이 달라 직접 이식 불가 |
| 다층 LUT 메모리 사용량 평가 | 대응 없음 | Prime ECC는 min-sum magnitude 양자화 테이블 방식으로 접근이 다름 |
