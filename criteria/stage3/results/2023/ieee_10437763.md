> 총평: 무선 URLLC용 GRAND 계열 universal 디코더로 부호·복호 패러다임이 달라 NAND binary QC-LDPC 이식성 하.

### B. 알고리즘 요약

1. 시스템: 짧은 블록 `n=128` 코드(CA-Polar/RLC/CRC/PAC), BPSK, complex Gauss-Markov(상관 `ρ`) 채널, URLLC 목표, rate `R=0.5~0.95`.
2. 문제: soft 디코더는 per-bit 독립 가정 → 실제 상관 노이즈를 깨기 위한 interleaver가 latency 증가+capacity 손실 유발.
3. 모델: 수신 심볼열을 `b`심볼 비중첩 블록으로 나누고 블록 간 근사독립(product measure) 가정, CSI로 블록 posterior `p(X_i^b|Y_i^b)` 평가.
4. 핵심기법(ORBGRAND-AI): hard-demod 블록 대비 대체 심볼블록의 신뢰도(exceedance)를 rank order → ORBGRAND landslide 알고리즘으로 노이즈효과 패턴을 우도 내림차순 생성, codebook check `Φ`로 정지.
5. 핵심식: 블록독립 근사 posterior가 각 블록 대체 시 우도 penalty를 곱하는 형태 → symbol-level ORBGRAND와 동일 형태로 환원.
6. 이론근거: Gauss-Markov 미분엔트로피율 분석상 블록크기 `b=2`만으로도 상관이득의 절반 이상 확보(다수 성능이득 가능).
7. 부수: BPSK·`b=4`, 128비트에서 32블록×15대체 → 480 신뢰도 정렬, 부호구조 무관(어떤 moderate redundancy 코드도 복호).
8. 최적화: 없음(오프라인 학습 불필요), abandonment threshold `τ`로 조기포기.
9. 결과: `ρ=0.75`에서 rate 0.86~0.95 코드가 interleaved rate-1/2 CA-Polar(CA-SCL) 대비 >2 dB 이득, target BLER 10^-3에서 평균 query <10.
10. 한계: HW 미설계(ORBGRAND 기존 칩 인용만), 무선 상관채널/CSI 전제, LDPC 반복복호와 무관.
