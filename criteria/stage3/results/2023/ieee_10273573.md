> 총평: QLDPC(양자 CSS) syndrome-based 복호 실패의 그래프 구조(absorbing/trapping/failure-inducing set)를 분석한 순수 이론 논문으로, HW·시뮬 수치 없고 대상이 양자 부호라 NAND binary LDPC 이식성·추천도 하(error-floor 이론 관점만 참고).

### B. 알고리즘 요약

1. 대상: 양자 CSS QLDPC 부호, stabilizer `HS=(HX|HZ)`로 `[[n,n-k]]` 부호 정의, X/Z 에러를 2개 독립 BSC로 분리 취급.
2. 문제: syndrome-based 반복복호는 유한길이 QLDPC에서 suboptimal — 수렴 실패, 잘못된 syndrome, 또는 logical operator만큼 다른 error(degenerate) 추정으로 실패.
3. 모델: Tanner graph(VN=코드좌표, CN=검사식), 입력 syndrome `σe=(HZeX^T, HX eZ^T)`, degenerate error(`e+f`가 stabilizer면 성공 취급)라는 양자 특유 개념 도입.
4. 핵심 기법1(복호): syndrome-based Gallager-B(BSC) — 초기 all-zero error 가정, CN 출력 = extrinsic VN 메시지 XOR + syndrome, VN 출력 = incoming CN 다수결(동점이면 0), `σ̂ℓ` 매칭까지 반복.
5. 핵심 정의: `(a,b)`-trapping set과 그 failure-inducing set(초기 에러 시 실패 유발하는 VN 부분집합), critical number `μ(T)`·strength.
6. 핵심 기법2(연결): `(a,b)`-absorbing set(각 VN이 odd degree보다 even degree CN 이웃이 많음) 정의 후 absorbing↔failure-inducing 관계 규명.
7. Thm1: `b≥1`(odd degree CN 존재) absorbing set은 항상 failure-inducing set → trapping set (복호 syndrome이 항상 `0⃗`이라 mismatch).
8. Thm2~4: `(a,0)` even-degree-only absorbing set은 `1⃗`의 rowspace 포함 여부로 degenerate/logical error 분기; acyclic `(a,0)` absorbing set은 trapping set(Thm3), cycle 구조는 a 홀/짝에 따라 failure-inducing singleton/특정 3-set(Thm4).
9. 결과(정성): 거의 모든 absorbing set graph가 failure-inducing set을 가져 trapping set이 됨; weight-4/5 에러가 mismatched syndrome 또는 oscillation으로 실패(Example 2, Table I). 수치성능 없음.
10. 한계: 순수 이론(HW/시뮬/BER 곡선 없음), 양자 CSS·degenerate error 전제, non-acyclic `(a,0)` with `1⃗∈rowspace`는 open, hard-decision Gallager-B(BSC) 국한.
