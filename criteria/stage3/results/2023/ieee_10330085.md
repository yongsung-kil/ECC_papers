### B. 알고리즘 요약

1. 시스템: AWGN, `k=5540, n=6000` 내부부호(OH 8.24%), 외부 staircase HDD(OH 6.25%)와 연접(총 OH 15%), 목표 `D=4.7e-3` output BER.
2. 문제: 6G/광통신 1Tb/s급에서 long-SDD는 전력·지연 과다, HDD는 성능부족 → 짧은 SDD "error-reducing" 내부부호로 채널 품질만 개선.
3. 모델: `y = c + w`, `w~N(0,I)`; error-reducing 조건 `P[g(y)∈B_k(u,Dk)]≥1-ε`, FER=1인 SNR에서도 목표 BER 달성이 핵심.
4. 핵심기법 1단(SPARC): design matrix `A`(n×ML), L 섹션 각 M열 중 1열 선택 → `c=Ab`, AMP 복호(Algorithm 1: residual `z`, Onsager term, section-wise nonlinear `η`).
5. 핵심 식: `η_i(s)`는 section 내 softmax 형태(식4)로 valid message 제약 `b∈B_{M,L}` 반영, `τ²`는 residual로 추정.
6. 핵심기법 2단(LDPC superposition): `c=√P1 f1(m1)+√P2 f2(m2)`, L=2 BPSK-LDPC를 다른 전력으로 중첩(≈PAM-4), soft SIC 복호(Algorithm 2).
7. 구현 디테일: soft SIC는 간섭 `ρ_ℓ=ΣP_j tanh(γ_j/2)` 추정, SINR 재계산 후 각 LDPC 복호 반복(outer 20회 × sum-product 20회), 랜덤 인터리버로 간섭 백색화.
8. 최적화: 전력분할이 rate분할보다 민감(최적 `P1*≈0.277`), decoder-in-the-loop로 base PCM(14×24, factor 250 lifting, 불규칙 CN degree, Table II) 최적화.
9. 결과(Table III): 목표 BER 달성 SNR - ER-opt LDPC 3.39 → 2×superposition ER-opt LDPC 2.77 → SPARC M=2^10 2.16 dB; SPARC로 1.23 dB, LDPC superposition으로 0.62 dB gain.
10. 한계: HW 미설계·시뮬만, SPARC는 Gaussian 변조/복잡도 과다, superposition은 고차변조(PAM-4)·연접 전제로 NAND ECC와 무관.
