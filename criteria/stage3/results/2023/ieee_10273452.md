### B. 알고리즘 요약

1. 대상은 regular `(dv, dc)` binary LDPC, 채널은 BSC이며 parity bit와 message bit가 서로 다른 crossover probability(각각 `p̲`, `p`)를 보는 UDP(Unequal Data Protection) 설정.
2. 문제: 고전 UDP는 message bit를 보호했으나 고rate 코드는 message가 많아 비효율 → parity bit를 대신 보호하는 것이 message로의 정보 확산에 유리한지 이론적으로 규명.
3. 핵심 가정: 부호구성은 그대로 두고, 각 message VN이 CN마다 `x = dv`개의 신뢰성 높은 parity VN에 연결되어 고신뢰 메시지가 확산된다고 모델링.
4. 핵심 기법 1단: Gallager-A DE 방정식을 UDP용으로 재유도. `p₋₁` 갱신에서 신뢰 VN 기여를 `(α, β) = (x, dc−1−x)`로 분리하여 반영(식 (2)).
5. 핵심 식 의미: doping 극단(parity 완전신뢰, `p̲=0`)에서 exact polynomial의 `k = dc−1−x`로 바뀌어(식 (4),(7)) threshold를 상향.
6. 핵심 기법 2단: 평균 threshold `γ̄ = 1/[(dv−1)(dc−1−x)] − dv/dc`(식 (11))로 doping의 closed-form 근사 유도 (기존 [12]는 수치해만 제공).
7. 부수 트릭: 3-level(errors-and-erasures) 양자화 디코더로도 동일 UDP를 적용, `q0/q1/q₋1` DE(식 (14),(15)) 재유도.
8. 최적화 절차: DE로 수렴 만족하는 최고 평균 `(p̲*, p*)` 쌍을 Newton법 등으로 탐색해 threshold 산출.
9. 결과: 3-level에서 최대 38.2%, BP에서 최대 42.7% threshold gain; (5,10) 코드 평균 iteration 21.13→15.13 등 수렴속도 개선.
10. 한계: BSC(hard) 이론·시뮬만, HW 미설계, 고rate(0.80~0.89)에서 gain이 3.4~15.8%로 작고 parity 신뢰성 확보의 실제 구현수단은 본 논문 범위 밖.
