### B. 알고리즘 요약

1. 시스템: 상향 power-domain NOMA(BPSK, AWGN, `nu`명 사용자), 단블록 부호 `C(n,k)`(예 (64,30)/(30,16,8) eBCH), URLLC용 ≤150b.
2. 문제: OSD기반 결합복호(JD)의 LLR 분포 해석 도구 부재로 DS·decoding order 최적화가 불가.
3. 모델: PIC(다중사용자 검출)와 SOSD(연판정 OSD)가 교대하며 LLR 교환, decoding switch(DS)로 SOSD 개입 제어.
4. 핵심기법1(Dual-OSD): SOSD의 extrinsic LLR을 `δi = V0 - V1`(phase-0/phase-1 최소 WHD 차)로 표현하는 분석용 변형 알고리즘 도입.
5. 핵심식: `δi = Σ|ℓj|(cj(i:1)) - Σ|ℓj|(cj(i:0))`, WHD 가중 해밍거리 기반.
6. 핵심기법2(density-transform): random code(binomial weight) 가정하에 `V0`, `V1` pdf를 Gaussian WHD 근사로 유도, `Di=V0⊗V1^(-)` (Thm1).
7. 트릭: order-(m0,m1) Dual-OSD ≈ order-m SOSD, MRB/non-MRB별 `N0,N1` 경계로 SOSD·LC-SOSD 근사(Prop2/3).
8. DE 프레임워크: JD를 bipartite graph로 표현, PIC/SOSD 노드별 density-transform으로 반복 간 LLR 밀도 추적, 수렴점 `ge(ξ*)=gd(ξ*)`.
9. 결과(이론/수치): 2·3사용자 시스템에서 DE가 LLR 진화·BER·수렴점을 정확히 예측(중고 SNR), DS로 SIC 대비 반복수 감소 재확인.
10. 한계: 순수 해석·수치(HW 없음), 단블록 eBCH·random code 전제, non-LDPC(OSD) 디코더이며 NAND soft-read/error-floor와 무관.
