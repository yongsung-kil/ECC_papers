### B. 알고리즘 요약

1. 시스템: 6G/MIMO(NT=NR=2, QPSK, MAP 검출, Rayleigh fading) 위 GC-LDPC 부호화 전송, rate 0.49/0.66, length 64800/63360, Z=384.
2. 문제: 5G LDPC(정보열 22, max lifting 384)로는 Tbps급 초고throughput·대길이를 못 맞춤 → 병렬성 높은 새 구성/복호 필요.
3. 모델: GC-LDPC를 local(대각 `Ri`)+global(`Xi`) 연접부호로 보고, local 부호를 각 안테나로 전송·병렬 복호, global로 초고신뢰 확보.
4. 핵심 기법1(구성): superposition construction으로 base matrix `Bsp`의 각 원소를 `Z×Z` CPM `Q(pi,j)`로 치환, 5G LDPC base를 `Ri`로 채택하고 `Xi=[Ai,0]`의 `Ai`만 설계.
5. 식: `Ai`는 PEXIT 알고리즘으로 행별(top→bottom) 최적화, 탐색공간 `O`(식8)는 kb≤22·작은 degree로 exhaustive 가능.
6. 핵심 기법2(확장): t=2용 `Bi`를 임의 t의 최소단위로 취급 → 탐색공간 축소·scalability(Table II에서 t 증가해도 capacity gap 거의 불변).
7. 트릭(cycle 설계): girth 조건(식9)으로 cycle type `Td` 판정, GC용 cycle metric `Wgc(P)`(식10) 최소화로 short cycle 회피, `Xi` exponent matrix 구성.
8. 복호: 2단계 TPD — local SPA 복호, 실패 시 global 복호 활성화. 수정 TPD는 성공한 local의 `CPM(Ri)`를 pruning(Alg.1)해 부분 검사행렬 `Hsp`로 global 복호.
9. 결과: 성공 local 병렬화로 throughput 4×/5× 증가, 고SNR에서 global iteration `IG`→0, global 복잡도를 TPD의 ~1/4·1/5로 감소, DVB-S2 BCH-LDPC 대비 0.05/0.15 dB 이득(FER 10^-5).
10. 한계: 무선 MIMO 특화 채널 가정, HW 미설계, throughput은 "배수"만 제시(Gbps 절대값 없음).
