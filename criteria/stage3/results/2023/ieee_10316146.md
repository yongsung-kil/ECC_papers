### B. 알고리즘 요약

1. 시스템: AWGN/BPSK 채널, 외부(정정용) LDPC + 내부(오류감소용) LDPC의 직렬 연접(serial concatenation) + 코드 간 랜덤 인터리버(permutation `P`); 예시 `N=4420, R≈1/4`, `N=8840, R≈1/4`, `N=4350, R≈1/2`.
2. 문제: 저rate에서 5G LDPC의 error-floor·복호 복잡도를 낮추면서 정정성능을 유지/개선하려 함.
3. 모델: 내부 LDPC는 디코드가 아니라 outer 디코더용 super-channel의 BER을 낮추는 "error-reducing" 역할, outer는 error-correcting 역할로 분리.
4. 핵심기법 1단: woven-like 반복 복호 - inner/outer 모두 SISO(soft-in soft-out) 디코더, extrinsic LLR `L_out`을 다음 iteration inner 입력으로 되먹임.
5. 핵심 식: inner 입력 = `[L_ch | L_out(prev)]` 연결, `L_in = D_in(...)`, `L_out = D_out(P^-1(L_in))` 교대; extrinsic 교환으로 super-channel 개선.
6. 확장: inner/outer 각각의 protograph를 최적화 대상으로 삼음(이전 논문[7]은 inner만, 본 논문은 outer도 최적화).
7. 구현 디테일: outer는 ACE 리프팅, inner는 [10] 리프팅; outer의 parity 심볼을 5G식 puncturing(수신 LLR을 0 세팅)해 rate 조정.
8. 최적화 절차: protograph의 랜덤 위치 flip → 확장 → AWGN에서 target FER 달성 SNR 측정 → best 갱신, 지역최소 회피 위해 flip 개수 증가.
9. 결과: `R≈1/4`에서 Eb/No 0.9 dB 이상부터 5G LDPC 능가, error-floor 없음; woven-like 복호는 동일 iteration 대비 BP 대비 최대 35% 복잡도 절감(Table I: 25~36%); 3단계 최적화로 약 1.2 dB gain.
10. 한계: HW 미설계, 시뮬만, 저rate(1/4~1/2) 통신 특화로 NAND 고rate ECC와 무관.
