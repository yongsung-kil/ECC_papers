### B. 알고리즘 요약

1. 대상: `(J,L)`-regular binary QC-LDPC, exponent matrix `E = S2^T · S1`(`S1=[0,1,...,L-1]`, `S2=[α0,...,α_{J-1}]`), circulant 크기 `P`; BPSK/AWGN/SPA(iter=50) 시뮬.
2. 문제: 기존 GCD 방법은 girth-8(4·6-cycle 제거)을 보장하지만 circulant size `P`가 임계값 `(α_{J-1}-α0)(L-1)`보다 커야만 성립 → 짧은 부호에서 성능/유연성 제약.
3. 핵심: circulant/정수의 새 lemma(Lemma 3~5)로 modulo 방정식을 분석해, 임계값보다 훨씬 작은 `P`로도 girth-8을 보장하는 새 tuple `S2`를 명시적으로 구성.
4. J=3: 기존 `[0,1,L]`은 `P>L(L-1)` 필요(Theorem 1)임을 증명; 새 tuple `[0,2,L]`은 `P=L^2-2L+4`(Theorem 2), `[0,1,3L/2]`·`[0,1,(3L+1)/2]`는 LBccs의 약 1/2 크기(Theorem 3).
5. J=4: `[0,x,yL,x+zL]`(gcd 조건)로 `P=L^2` 일반 구성(Theorem 5, `[0,1,L,L+1]`을 특수 케이스로 포함); `[0,L/2,L+1,3L+1]`은 `mod(L,8)=0`에서 `P=L^2-L/2`(Theorem 6).
6. J=5: `[0,1,L,L+1,2L+3]`(`mod(L,6)∈{2,4}`, `P=(L+2)^2-1`, Theorem 7), `[0,2,L,2L+1,2L+2]`(`mod(L,6)∈{1,3}`, `P=2L(L-1)+1`, Theorem 8) 등 기존 대비 축소.
7. 등가성 도구: Lemma 5로 `S2`와 `[α_{J-1}-...]`가 등가 부호임을 보여 탐색범위를 `1<=α1<=⌊L/2⌋`로 축소.
8. 확장: masking(Eq.(7) in [12])·column-splitting으로 irregular 부호 파생 가능, masking 적용 시 성능 크게 개선.
9. 결과: (480,261) 새 부호가 기존 (512,276)보다 BER=1e-5에서 0.20dB 우수; masked (1305,580)이 masked array (1341,596)보다 BER=1e-4에서 약 1.0dB 우수; 짧은 새 부호가 더 긴 기존 부호와 거의 동일 성능.
10. 한계: 순수 부호 구성 이론(HW 미설계, SPA 시뮬만), 위성/지상 통신 타깃, 저rate·regular 소형 부호로 NAND 실험 없음; 다수 결과가 conjecture(미증명)이며 trapping/absorbing set 제거는 future work.
