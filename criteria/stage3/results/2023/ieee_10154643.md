### B. 알고리즘 요약

1. 대상은 QC-LDPC(base matrix + CPM `I(p_i,j)`, lifting factor Z) 부호구성으로, base 구조는 고정하고 CPM shift 값만 선택해 error-floor를 낮춘다.
2. 문제: 낮은 column weight 부호는 threshold/throughput은 좋지만 소형 trapping set 등 harmful substructure로 error-floor가 심각.
3. 기존 LETS/AS 탐색법(LSS, DQL)은 중간결과가 방대하거나 수백~수천 회 반복이 필요해 부호 생성에 부적합.
4. 핵심: LETS와 AS 특징을 결합한 신규 표적구조 LEAS(leafless elementary absorbing set)를 정의하고, 이를 최소화하도록 shift 값을 열별로 순차 최적화.
5. 표적지표 `τ_{a,b}` — (a,b) 크기의 harmful substructure 집합으로, `(a_min,b_min)`가 최소이고 그 개수가 적은 shift를 선택(작을수록 error-floor 유리).
6. 핵심 기법 1단 cycle combination: `C_2l` 사이클 라이브러리에서 CHE(공유 check node)로 사이클을 결합, `a = l1+(l2-x)+...+(ln-x)`(x=공유 VN 수=0/2/3)로 basic topology(BT) 생성.
7. 핵심 기법 2단 leaf cutting: 각 VN에 flag `f_i=(d_max-2)` 초기화 후 외부 공유 CN 연결마다 감소시켜 `b=Σf_i` 산출, 비동형 구조를 b로 분류(비완전탐색, 저복잡 근사 커버).
8. 구성 프레임워크: 열별로 shift `p_cur` 후보를 0~Z-1 순회하며 Alg.1로 LEAS 분포를 평가해 최소 τ를 주는 shift로 확정.
9. 결과: WiMAX R=2/3에서 FER=10^-6에 약 0.5dB 이득·error-floor를 10^-5→10^-7로 저감, R=5/6 단부호 10^-5에서 0.6dB 이상, 5G-NR(11/14, Z=224)은 이득 1자릿수 미만.
10. 한계: shift 값 선택뿐(base 구조 유지), HW 미설계, 검증은 72코어 SIMD/AVX CPU 시뮬(2Gbps) 한정.
