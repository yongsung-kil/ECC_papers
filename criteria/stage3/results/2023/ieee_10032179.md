### B. 알고리즘 요약

1. 대상은 GLDPC 부호(CN이 SPC 대신 Hamming/BCH 등 `(nτ,kτ,dmin,τ)` 선형블록 component code)와 non-binary(GF(q)) irregular LDPC 앙상블.
2. 문제: LDPC/GLDPC의 error floor를 좌우하는 harmful graph 구조(trapping/absorbing set)를 GLDPC로 확장 정의하고 그 개수 분포를 앙상블 평균으로 셈.
3. 참조 디코더는 `PBF`(Parallel Bit Flipping, 하드결정 BSC): super-CN은 BDD, SPC-CN은 미충족 시 flip, VN은 flip 메시지가 `dv/2` 초과면 반전.
4. TS `Ta,b`=1값 VN a개가 미충족 CN b개를 만드는 집합, AS는 추가로 각 VN이 `n_f(v)<dv/2`, FAS는 전체 VN이 `n_f(v)<dv/2` (PBF로 정정 불가).
5. 핵심 도구: 각 VN/CN의 포함 경우를 생성함수 `f(t,s)`(VN측), `g(x,y)`(CN측)로 인코딩하고 `coeff(·)` 곱으로 유한길이 평균 개수 `E(a,b)` 산출 (Lemma 4/6/7).
6. 점근 분포 `G(θ,γ)=lim (1/n)ln E(θn,γn)`는 Hayman 다변수 공식(Lemma 3)으로 계산 → CN type 수와 무관한 방정식계의 해로 귀결 (Theorem 1/2/3).
7. small-θ 근사(Corollary 1): 상대최소 ∆-trapping set 크기 `θ*_TS`의 닫힌형 근사식 (23) 도출, 방정식계 풀지 않고 평가 가능.
8. Appendix E: 동일 생성함수 기법을 non-binary LDPC로 확장해 TS/AS/EAS enumerator (q-ary, `q=4` 예시) 유도.
9. 결과: (155,80) regular GLDPC(component (31,26) Hamming)에서 1000 error frame 수집 → 모든 수렴 실패가 FAS, dominant는 `(2,4)` FAS이며 고SNR FER/BER 지배; 앙상블 평균과 랜덤부호 다중도 일치. Example들에서 component code의 `dmin`↑(Hamming→BCH)가 TS 성질 개선.
10. 한계: 순수 앙상블 조합론/점근 이론 - HW·throughput 없음, 하드결정 PBF 전용, binary QC-LDPC + soft min-sum 대상 아님.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| TS/AS/FAS 조합구조로 error-floor 예측 | 대응 없음 | Prime ECC는 부호구조 고정, error-floor를 앙상블 enumerator로 설계하는 경로 없음 |
| PBF(하드결정 bit-flipping) 디코더 | 대응 없음 | Prime ECC 디코더는 min-sum soft message passing(`decoder.cpp` `LDPC_Decoder()`)로 알고리즘 계열 상이 |
| GLDPC super-CN(Hamming/BCH) BDD | 대응 없음 | Prime ECC는 SPC-CN binary QC-LDPC, super-CN 없음 |
| non-binary(GF(q)) LDPC enumerator | 대응 없음 | Prime ECC는 binary 전용 (프로파일 §5) |

적용 가치: **낮음** - GLDPC/non-binary/PBF 하드결정 이론으로, binary QC-LDPC soft min-sum 디코더·HW·NAND 어느 레이어에도 직접 이식 접점이 없다.
