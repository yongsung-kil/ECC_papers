### B. 알고리즘 요약

1. 시스템: PLDPC-coded GSMPPM(일반화 공간 MPPM) FSO 링크, `Nt=4/Nr=4/Na=2`, MPPM `(l,la)`, 약난류(lognormal) 채널, rate-1/2.
2. 문제: 기존 GSMPPM은 공간·신호 도메인 성상도 크기를 2의 거듭제곱으로 강제해 안테나그룹·MPPM 심볼 다수가 idle → 자원 낭비.
3. 채널모델: lognormal 약난류 `h_ij`, IM/DD, AWGN `σ²=N0/2`, `SNR = la·Pt²/(2Rm σ²)`, 수신은 Max-log-MAP 검출 후 BP 복호.
4. 핵심기법1(ADM 성상도): 2의 거듭제곱 제약 제거, `ΨA`(effective 안테나그룹, 크기 `MA`)+`ΨB`(추가 안테나그룹, 크기 `MB<MA`)로 full MPPM 심볼·모든 안테나그룹을 최대 활용 (Algorithm 1/2).
5. 핵심식: 최대 Hamming distance 기준으로 label↔MPPM 심볼 재라벨링, `Dij=2la`일 때 label 간 거리 d 최대화.
6. 핵심기법2(I-PLDPC): PEXIT로 decoding threshold 최소화하는 rate-1/2 4x7 base matrix `BI` 탐색, pre-coding 구조 + degree-2 VN 비율 `1≤pv,2≤pc/2` 제약으로 TMDR(error floor 방지) 확보.
7. 구현 디테일: 병렬 edge 최대 3, punctured VN 1개, PEG lifting으로 대형 PCM 생성; codeword length 4500(임계) / 정보 k=1800(BER 시뮬).
8. 최적화: constellation-constrained AMI(CM/BICM) 계산으로 성상도 capacity 비교, AWD 함수로 TMDR 측정.
9. 결과: ADM이 기존 성상도 대비 BER 1e-5에서 약 0.3~0.96dB 이득; I-PLDPC가 AR4JA/code-B/regular-(3,6) 등 대비 threshold 0.1~0.99dB 이득, error floor 없음.
10. 한계: 순수 시뮬레이션, HW 없음, 특정 FSO 약난류+공간변조 채널 특화, soft-read 양자화·NAND read 비용 개념 부재.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| PEXIT 기반 protograph base matrix 탐색 | 대응 없음 | Prime ECC는 QC-LDPC H-matrix 고정([Load_PCM](../Prime_ECC_3.1_Claude/paper_screening_profile.md)), protograph 설계 경로 아님 |
| ADM(공간+신호) 성상도 설계 | 대응 없음 | Prime ECC는 변조·성상도 계층 없음(NAND read/RBER 채널) |
| lognormal 약난류 채널/Max-log-MAP 검출 | 대응 없음 | Prime ECC 채널은 AWGN/RBER/fixed-error(`channel.cpp`)로 상이 |
| BP(belief-propagation) 복호 | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 HW용 min-sum 근사, 논문은 full-BP 알고리즘 층위만 언급 |

적용 가치: **낮음** - FSO 광통신 공간변조 성상도와 protograph 부호를 함께 설계하는 통신 특화 논문으로, NAND binary QC-LDPC soft min-sum 디코더·HW에 떼어 쓸 요소가 없다.
