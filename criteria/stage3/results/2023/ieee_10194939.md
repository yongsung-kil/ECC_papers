### B. 알고리즘 요약

1. 시스템: 자율 수중 플랫폼(AUV) 텔레메트리를 LDPC(IEEE802.11 시뮬 r=2/3,3/4,5/6 및 실측 (4098,984) 전용부호)로 보호, BSC/AWGN 채널.
2. 문제: 표준 ECC는 소스 압축 후 균일 사전분포를 가정하나, 센서 데이터엔 시간적 상관에 기인한 비균일 자연 잉여(NR)가 남아 있어 이를 버리는 것이 비효율.
3. 모델: 각 정보비트 `bi`를 Bernoulli `B(pi)`로 모델링, BSC 비트반전 `f`로 수신비트 `yi = bi ⊕ wi`, `vi = pi - f(2pi-1)`.
4. 핵심기법1(추정): N개 수신코드워드로 `vi`의 ML 추정 후 `p̂i = (v̂i - f)/(1-2f)`로 무잡음 비트분포 온라인 추정(슬라이딩 윈도우 유한메모리).
5. 핵심식 의미: 채널 반전확률을 역보정해 구조적 비트의 참 편향 `p̂i`를 복원 - 이것이 사전정보(prior)가 됨.
6. 핵심기법2(디코딩): 비트 엔트로피 `hi < Hthr`인 비트를 "구조적 비트"로 판정, 채널 LLR에 모델 LLR `LLRm = log((1-p̂i)/p̂i)`를 confidence `c`(로지스틱) 가중해 더해 BP에 입력.
7. 구현 디테일: 코드워드가 다수 구조를 담으면 병렬 다중 추정기 + 온라인 클러스터링(Bernoulli mixture, Hellinger 거리) 분류기로 코드워드 유형 분리.
8. 학습: 사전 훈련 없이 수신 중 온라인 학습, 디코딩 성공 시 estimate로, 실패 시 hard-decision으로 모델 갱신.
9. 결과: 모든 rate에서 FER 개선, 2022/2023 실측에서 BP 실패 코드워드의 20~21% 복구, 학습 후 정보비트의 26%가 구조적 비트로 추론(r=3/4).
10. 한계: HW 미설계(순수 알고리즘/실측), 고rate 코드일수록 이득 크고 저rate에선 NR 비율 감소, 데이터의 시간적 상관 전제에 의존.
