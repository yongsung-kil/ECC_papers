### B. 알고리즘 요약

1. 대상: 일반 binary LDPC(적용 예 DTMB-A, code rate 1/2·2/3·5/6, length 61440, QPSK/AWGN)의 고정소수점 message-passing 복호기.
2. 문제: 양자화 bit-width가 극소(1~2bit)일 때 성능 급락, 기존 MSA 개선기법들이 양자화 효과를 고려 안 하고 LUT를 iteration 번호로 인덱싱해 부호/채널 변화에 취약.
3. 핵심 기법 1단(비대칭 양자화): CNU 입출력은 저bit `wL`, VNU 입출력은 고bit `wH`로 분리하고, 둘 사이를 QCM(quantization conversion module) `QCM[w2,T1,T2]`로 변환 (식 4).
4. 핵심 인덱스(check-ratio): iteration 번호 대신 unsatisfied check-node 비율(check-ratio)을 LUT 인덱스로 사용해 양자화 interval `Tcur`를 매 iteration 적응 결정 (VNU는 고정 `T0`, CNU만 적응).
5. 핵심 기법 2단(AQNMSA): CNU 직후 정규화를 결합한 QCNM `QCNM[w2,T1,T2,α]=QUAN(LLR·α·T1)` (식 5)로 양자화+정규화 factor `α`를 동일 check-ratio LUT `(T,α)`로 함께 조회.
6. 부수 트릭(smooth): 유한 부호길이로 check-ratio가 요동쳐 `Tcur/αcur`가 불안정 → smooth factor `sT,sα`로 이전값과 지수평활 (식 6/7).
7. 학습/최적화: MET-DE(multi-edge-type density evolution)로 asymptotic BER를 예측해 LUT를 설계(Alg.1 FindBetterLUT), Monte-Carlo 대비 설계 복잡도 대폭 저감. 단 smooth 적용 시 MET-DE 정밀분석 불가, `sT,sα`는 시뮬로 최적화.
8. 파라미터: `wH=9, wL=4, wT=4, N=wα=5`, 최종 LUT는 Table I(check-ratio→T,α, T·2^wα, α·2^wα).
9. 결과: rate 1/2에서 float SPA 대비 threshold 손실 0.2dB에 불과하며 error-floor 낮음. 3개 rate 모두 optimal float-point NMSA보다 우수, 4bit 양자화로 SPA에 근접.
10. 한계: 시뮬(BER)만이고 HW 설계·gatecount·throughput 수치 없음. 타깃은 DTMB-A 방송(QPSK/AWGN), NAND 직접 실험 없음.
