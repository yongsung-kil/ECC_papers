### B. 알고리즘 요약

1. 시스템: 자기기록(MR)용 circulant-based(CB) SC-LDPC, 시뮬 파라미터 `κ=11`, `z=67`, `γ=3`, memory `m=1`, coupling length `l=5`, 부호길이 3685 bit.
2. 문제: LDPC 설계가 asymptotic(threshold) 관점과 finite-length(cycle) 관점으로 분리 연구돼 저SNR·고SNR 영역을 동시에 최적화하지 못함.
3. 모델: AWGN 및 PR(transition jitter+전자잡음, density 1.4, 등화타깃 [8,14,2]) 채널, BP/min-sum + BCJR pattern-dependent noise prediction 검출.
4. 핵심기법1: SC-LDPC 성능이 작은 `partitioning matrix P`의 row/column 순열 등가류에만 의존함을 증명(Lemma 3), full PCM 대신 `γ×κ` P만 탐색.
5. 핵심식/의미: cycle-6 개수 `F(B)`를 overlap parameter로 표현, threshold는 EXIT 방법 `σ*(B)`로 산출 — 두 지표로 후보를 평가.
6. 핵심기법2: non-equivalent binary matrix 집합 `K_{κ,3}`의 closed-form 개수 도출(γ=3), 등가 다중도 제거로 탐색공간 대폭 축소(예: κ=11에서 전체 대비 1.41×10^6배 감소).
7. 구현 디테일: cycle-6 오름차순 정렬 후 threshold 기준 inferior 후보 제거 → cycle-driven(CD)/threshold-driven(TD) 두 대표 후보 출력.
8. 최적화 절차: CB lifting `c_{i,j}=α·i·j (mod z)`로 cycle-4 제거, circulant power 최적화는 본 논문 범위 외.
9. 결과: CD가 CV 대비 lifted graph cycle-6를 54% 감소, PR 채널 SNR=14 dB에서 약 1 order BER 개선, AWGN 저SNR에서 TD가 약 0.5 order 우위(SNR=2.5 dB). 복호 20 global × 200 local iteration.
10. 한계: HW 미설계, 시뮬만, SC-LDPC 특화(부호율 본문 명시 없음), 상용 MR 채널 적용은 future work.
