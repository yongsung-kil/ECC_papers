### B. 알고리즘 요약

1. 대상은 CPM 기반 binary QC-LDPC(base matrix + circulant shift), 예시로 4×20 (2560, 2048) 5개 부호와 PEG(1008,504), Margulis(2640,1320) 벤치.
2. 문제: LDPC error-floor를 좌우하는 sublinear trapping set(TS)을 정확·고속 열거해야 하는데, 기존 Cole IS·Velazquez-Subramani LP법은 큰 부호에서 매우 느림.
3. 모델: message-passing 복호에서 cycle/TS가 pseudocodeword를 유발, generalized union bound(FER)와 Schlegel-Zhang linear model(BER)로 error-floor를 추정.
4. 핵심기법1: Cole의 unrolled BP 계산 트리(깊이 3, `VN1-CN1-...-CN3`)에 error impulse `εi·h`(h=채널분산)를 주입해 TS 후보를 검출.
5. 핵심 식: exponent chain이 `Σ(-1)^i·a_i ≡ 0 mod L`이면 Tanner graph에 cycle 형성(Prop.1) — cycle 열거를 정수 연산으로 환원.
6. 핵심기법2: subgraph separation 함수 도입 — `TS(VN)`(새 cycle 추가 VN 합집합), `Tree(VN)`(tree-free 최대 부분집합)으로 harmful ordered sieving 수열 `S1,S2`를 girth/2부터 max까지 정렬.
7. 부수트릭: `b/a` 비 기반 TS 순서화, graph automorphism(×128) 활용, 여러 단계 스케줄러로 병렬화·RAM·iteration 수 조절.
8. 최적화: Simulated Annealing lifting법[24]으로 동일 base matrix에서 EMD 스펙트럼/code distance가 다른 5개 QC-LDPC 후보 생성.
9. 결과: 제안법이 (1008,504)에서 LP 대비 5027배·Cole 대비 43배, (2640,1320)에서 LP 대비 37958배 빠름; EMD/dmin 개선이 TS 스펙트럼과 BER/FER error-floor 저감으로 이어짐(20 iter Quantized Normalized Layer Min-Sum, LLR=4/C2V=4/V2C=6 bit).
10. 한계: HW 미설계·시뮬레이션/열거 도구뿐, error-floor 저감은 부호 재구성 통해서만 얻어짐(디코더 알고리즘 신규 기여 아님).
