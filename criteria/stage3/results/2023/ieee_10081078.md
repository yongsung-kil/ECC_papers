### B. 알고리즘 요약

1. 대상은 QC-LDPC (base matrix + `Z×Z` circulant, `M×N` 블록)의 row layered 복호이며, 각 layer가 여러 행을 병렬 처리하는 Type-2 partition 스킴을 다룬다.
2. 문제: 각 layer의 최대 column weight `ω(HTl)`가 layer 내부 계산지연(`O(log2 ω)`)과 HW 복잡도를 좌우하는데, 최적 Type-2 분할이 체계적으로 정립되지 않음.
3. block cyclic shift property(`HTl = ϕlS(HT0)`)를 유지하면서 `ω(HT0)` 최소화를 최적화 문제(Problem 1)로 정식화, 분할을 `(S, T0)` 쌍으로 유일 표현.
4. 모든 feasible solution을 s-cyclic shift class / LS-class 조합으로 특성화(Theorem 2)하고, 하한 `ωLB = ⌈ω(H)/L⌉` 도출(Theorem 5).
5. 연속 layer 간 데이터 의존성(같은 VN 공유로 인한 pipeline 대기)을 줄이려 `layer distance d(S,T0)` 지표 정의: 연속 k개 layer의 누적 column weight ≤1이 되는 최대 k.
6. Problem 4(ωLB=1 달성)가 k-partite 그래프의 k-clique 탐색과 등가임을 증명 → 다항시간 최적해 없음. 따라서 enumerative(Algorithm 2)·greedy(Algorithm 3) 분할 알고리즘 제시.
7. Theorem 4로 `ω(·)` 계산을 `j mod Z < LS` 열로만 축소해 복잡도를 `Z/(LS)`배 절감.
8. ωLB/원하는 layer distance를 분할로 못 얻는 경우, QC-PEG의 CN 선택 전략에 조건(식 16/19)을 추가(Strategy 2/3)하여 straightforward 분할 `(1, ∪Cm,0,0)`을 갖는 PCM을 직접 구성.
9. 결과: 5G LDPC와 동일 파라미터(M=5,N=27,Z=384 / M=4,N=8,Z=384)로 구성한 부호가 layered SPA/MSA(최대 10 iter, AWGN/BPSK)에서 5G 부호보다 FER 우수·평균 iteration 감소.
10. 한계: HW 미설계(게이트카운트/throughput 수치 없음), 이득은 분할·부호구성 수준이며 CN 업데이트 알고리즘 자체는 불변, 5G 대상이라 NAND soft-read/error-floor 특화는 아님.
