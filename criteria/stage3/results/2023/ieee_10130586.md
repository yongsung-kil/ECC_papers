### B. 알고리즘 요약

1. 시스템: AWGN+BPSK 채널의 binary LDPC(주로 CN/VN degree 6/3 regular), `(96,48)~(2400,1202)` 및 일부 QC-LDPC/irregular, 최대 iteration 20.
2. 문제: Tanner 그래프의 short-cycle(특히 4-cycle)이 message 독립성을 깨 BP가 이론성능(식 11의 상관계수 `µ`)에 도달 못함.
3. 모델: cycle 존재 시 CN 확률식이 `P=0.5+0.5·Π µi(1-2pi)`처럼 상관계수 `µ`로 보정되어야 하나 `µ`를 정확 계산 불가.
4. 핵심기법: BP의 horizontal(CN)·vertical(VN) 단일노드 update만 각각 소형 NN(`f`,`g`)으로 학습해 cycle 보정된 message update를 근사.
5. 핵심: 네트워크가 노드 degree(`Dc`,`Dv`)에만 의존하고 code length·특정 Tanner그래프와 무관 → 같은 학습모델로 여러 LDPC 디코딩(일반화).
6. 확장: 출력 판정용 network `z`(softmax) 추가, RNN 방식으로 iteration간 동일 가중치 공유, end-to-end BCE loss(식 15) 학습.
7. 구현 디테일: irregular 코드는 learnable padding(`PCN`은 tanh후 곱셈 무영향 위해 ~0.8, `PVN`~0)으로 tensor degree 통일.
8. 학습: Mixed-SNR(1~5dB) 데이터가 Single-SNR보다 일반화 우수, 코드별 BER 1e-1~1e-5 구간 SNR 범위에서 균일 샘플링, Adam optimizer.
9. 결과: 4-cycle 많을수록 BP 대비 BER 이득 커짐, GPU에서 iteration당 시간 20%↓·전체 ~40% time gain, high-SNR에서 avg iteration 소폭 감소.
10. 한계: HW 미설계·시뮬만, 연산량 자체는 BP보다 많고 GPU 병렬성에 의존, high-irregularity(5G-LDPC)·고degree(20) 노드에서 성능 저하.
