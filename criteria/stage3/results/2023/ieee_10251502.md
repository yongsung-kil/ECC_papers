### B. 알고리즘 요약

1. 대상은 5G-NR BG1 MET/QC-LDPC(irregular, raptor-like, punctured VN, deg-1 VN), lifting `z=384`, BI-AWGN, code rate 0.33~0.89.
2. 문제: 고throughput(20Gb/s)용 저bit 양자화 min-sum은 성능 열화 크고(5bit SPA 대비 0.70dB↓), 기존 DE/RCA는 양자화·min 연산 예측 부정확.
3. 핵심모델: LLR을 bit-width `w`, 양자화 간격 `T`로 균일 양자화 `QUAN(w,T)`, VNU=합·CNU=min·sign 쌍연산으로 표현.
4. 핵심기법1 modified MET-DE: PMF를 실제 양자화 파라미터로 정확 추적(`ψInput/ψV/ψV2C/ψC/ψC2V`), check-ratio를 unsatisfied CN 확률 `P(k)`로 근사해 적응 알고리즘 예측 가능케 함.
5. 핵심식: `P_i=½(1-Π(1-p_j))`, `p_j=Σ_{n≥0}PMF(n)` — VNU 출력 부호확률로 check-ratio를 DE에서 산출(수치 시뮬 없이 LUT 인덱스 확보).
6. 핵심기법2 비대칭 양자화: 입력/VNU는 고bit `wH=9`, VNU↔CNU 전달은 저bit `wL=4`, QCM으로 도메인 변환해 라우팅 부담 축소.
7. 부수트릭: check-ratio+edge-type을 LUT 인덱스로 `T`·정규화 `α` 동시 적응(QCNM), smooth operation(`sT,sα`)으로 check-ratio 요동 안정화.
8. 최적화 절차: modified MET-DE 기반 LUT 설계(Alg.1) — SNR을 waterfall 영역에 맞추고 파라미터를 단위씩 증감·단조성 유지하며 BER 최소 후보 채택.
9. 결과: 5G-NR BG1에 4bit로 적용 시 float NMSA와 동등/우수, float SPA 대비 0.36dB 손실·error floor 없음, QANMSA 대비 0.25dB 이득; |dSNR|>1.5dB에서 SPA보다 강건.
10. 한계: HW 미설계(시뮬만), gatecount/throughput 수치 없음, 5G MET/BG1·BI-AWGN 특화, 4bit+ 소프트 전제.

```
| 적응 양자화/정규화 LUT(check-ratio,edge-type) | ecc_data.h PARAM_LLR, decoder.cpp Get_VNU_Table_Idx() | 우리 iteration별 LLR 테이블 인덱싱을 check-ratio 인덱스로 대체 검토 가능 |
| VNU 합산·양자화 처리 | decoder.cpp VN_Cal_HD() 등, decoder.h Get_VNU_* | 비대칭 wH/wL 양자화 개념 이식 지점 |
| soft-read LLR 양자화/threshold | channel.cpp Set_LLR_Th(), Set_R_Offset() | 입력 LLR 양자화 간격 T 설계와 대응 |

```
