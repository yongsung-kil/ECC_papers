### B. 알고리즘 요약

1. 대상: (3,6)/(3,9)/(4,6)-regular SC-LDPC(=LDPC convolutional) code, AWGN+BPSK, sliding window decoding(SWD) window size `W`, lifting `M`.
2. 문제: 저지연(작은 `W`) + capacity 근접(LDPC-BC BP threshold 이하 SNR)에서 SWD가 드물지만 심한 decoder error propagation(연속 block error burst)을 겪음.
3. 모델: multi-state Markov 모델(random state `S0`, 중간 `S_i`, burst `S_J`), 각 상태 BLER `q_i`가 `q_J≥…≥q_0`; `q_J→1`이면 무한 전파.
4. 핵심 기법1 (CN doping): 결합 그래프 곳곳에 reduced-degree(degree-4) CN을 삽입해 강한 local code 형성, error propagation을 절단(디코딩 절차 window shift 수정 필요).
5. 핵심 식: 무한전송 평균 BLER `PB = rJ/(1−qJ+rJ)`; doping이 `1−qJ≈2/s`(period `s`)로 만들어 burst 탈출 확률을 키움.
6. 핵심 기법2 (VN doping): 특정 VN 비트를 known 값 0으로 고정(LLR=+10 주입), 디코딩 절차 변경 없이 CN doping 효과 모사; adaptive doping은 feedback channel로 `η` threshold 실패 시 doping 요청.
7. 구현 디테일: systematic VN doping — systematic 비트만 fractional(`δ≤R`) doping해 encoding 사전 처리 및 정보열 복원 단순화(rate loss 최소화).
8. 최적화: 주기적 doping이 최적(구간 길이 균등 시 BLER 최소), doping 위치 수 `d` 증가로 추가 개선.
9. 결과: 관심 SNR 구간에서 doped code가 undoped 대비 BER 약 2 orders, BLER 약 1 order 개선; doping rate loss < encoder termination rate loss; systematic doping은 full doping과 거의 동일 성능(<0.025 dB 손실).
10. 한계: HW 미설계, AWGN 시뮬만(NAND/실측 없음), SC-LDPC/SWD 특유 구조 전제로 QC-LDPC block decoder엔 비적용.
