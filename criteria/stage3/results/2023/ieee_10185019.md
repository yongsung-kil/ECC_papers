## B. 알고리즘 요약

1. 대상은 MLC/TLC/SLC NAND flash로, Gaussian 및 empirical(PE/CCI/RTN/RT noise) 채널 모델을 특성함수로 closed-form 유도하고 multiple read를 quantizer로 본다.
2. 문제: 읽기 속도 제약으로 soft 정보 해상도가 제한되며, 균일/MMI/entropy read voltage 최적화는 LDPC 복호기 거동을 고려하지 않아 최적이 아니다.
3. 핵심모델: read voltage `r`로 threshold voltage를 영역 분할해 MSB/LSB LLR을 이산화하고, density evolution(DE)로 CN/VN LLR 분포를 추적해 오류확률 `Perr`을 추정한다.
4. 핵심기법1(multiple set): 각 PE cycle마다 `r*=argmin_r Pavg(r,...)`로 read voltage를 오류확률 최소화하도록 최적화.
5. 핵심기법2(single set): PE cycle별 최적치가 달라 `r*=argmax_r PE*(...)`로 복호임계(decoding threshold)를 최대화하도록 최적화.
6. DE는 BP·min-sum·NMS(`gNMS=sign·min·α`)·OMS(`gOMS=sign·max(min-β,0)`)를 모두 지원, 최적 read voltage 탐색은 differential evolution(mutation/crossover/selection)로 수행.
7. 부호설계: read voltage에 맞춰 irregular LDPC의 degree distribution `(λ,ρ)`와 protograph base graph `b(i,j)∈{0,1,2,3}`(AR4JA 계열, rate 0.9)를 복호임계 최대화로 설계.
8. 공동설계: read voltage와 LDPC 부호를 동시에 `argmax PE*`로 최적화, multiple decoding(1차 hard→실패 시 2차 재읽기)용 read voltage 설계 규칙도 제시.
9. 결과: (3,30)/(3,6) 및 irregular/protograph에서 MMI·Entropy 대비 FER=10^-3에서 1,000~3,000 PE cycle 이득, 공동설계는 MMI 대비 약 4,600 cycle, AR4JA 대비 4,300 cycle 이득.
10. 한계: 순수 DE 기반 offline 설계 이론/시뮬 논문으로 HW 설계·gatecount·throughput 없음, 부호는 PEG/degree-distribution 기반 non-QC.
