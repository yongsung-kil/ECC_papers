### B. 알고리즘 요약

1. 시스템: non-adaptive quantitative group testing - n개 항목 중 결함 확률 γ, m<n개 테스트로 syndrome `s=xA^T` 관측(테스트당 결함 개수를 정확히 셈, noiseless).
2. 문제: 기존 GLDPC 기반 GT[6,7]는 CN에 t-오류정정 BCH를 두는데, 강한 코드(큰 dv,t)가 iterative 복호에서 최적이 아니고 irregular 최적화가 민감함.
3. 모델/가정: test matrix A를 (dv,dc)-regular LDPC의 PCM으로 사용, VN=항목/CN=테스트인 bipartite graph.
4. 핵심기법 1단: CN을 BCH(t≥1) 대신 단순 SPC(t=0)로 두는 LDPC 기반 GT + peeling decoder - CN syndrome `s=0`이면 전부 non-defective, `s=d_c`이면 전부 defective로 해소.
5. 핵심식: LDPC-GT rate `Ω=dv/dc`((2)에서 t=0), DE 방정식 (3)~(6)으로 defective/non-defective VN 메시지 미해소 확률 `p0,p1,q0,q1` 추적.
6. 핵심기법 2단: spatial coupling - L개 복사본을 배치, VN이 위치 [τ,τ+w] 범위 CN에 연결(coupling memory w)하여 threshold saturation·wave-like 복호 유도.
7. 부수 디테일: 결합 rate `Ω_SC=(1+w/L)Ω`, sliding window(W=15) 복호로 latency를 uncoupled와 동일하게 맞춰 비교.
8. 학습/최적화: 별도 학습 없음, DE로 threshold Ωth(고정 γ 최소 rate)와 γth(고정 Ω 최대 prevalence) 산출.
9. 결과: LDPC 기반 GT가 GLDPC 기반[6]을 크게 능가; coupling으로 threshold·error floor 개선, w 증가 시 threshold가 상수로 수렴; Ω=5%에서 γth가 GLDPC 대비 대폭 향상(Table III).
10. 한계: HW 미설계, 시뮬/DE만, ECC가 아닌 진단 응용, coupled scheme의 latency(nbL)가 크게 증가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LDPC PCM을 test matrix로 사용 | 대응 없음 | ECC 복호가 아니라 결함항목 검출용 test matrix로, `Load_PCM()`의 부호 PCM과 목적이 다름 |
| syndrome peeling decoder(s=0/s=dc) | 대응 없음 | BEC-류 peeling으로 min-sum message passing([decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `LDPC_Decoder()`)과 연산·채널이 무관 |
| spatial coupling(SC-LDPC) | 대응 없음 | 프로파일 부호는 고정 QC-LDPC block code로 SC 구조 미채택 |

적용 가치: 낮음 - LDPC 그래프/spatial coupling을 정량적 group testing 진단에 적용한 연구로, NAND용 binary QC-LDPC ECC의 부호설계·디코더·HW 어느 레이어와도 이식 접점이 없다.
