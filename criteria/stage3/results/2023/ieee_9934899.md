### B. 알고리즘 요약

1. 대상: (J,L)-regular binary QC-LDPC, PCM은 P×P circulant 배열, exponent matrix `E`(J×L)와 circulant size `P`로 결정, BPSK/AWGN/SPA 시뮬.
2. 문제: Tanner graph의 4-cycle/6-cycle 제거(girth≥8)가 성능 핵심인데 기존 명시/준랜덤법(GCD, ZSW, Sidon, SYM)은 circulant 크기·후보 수 제약.
3. 도구: (v,k,t)-DDS - t개 집합의 모든 차분 `di(r)-di(s)`가 modulo v로 상이, `v≥tk(k-1)+1`.
4. 핵심기법 I: t=2 DDS `{d1,d2}`로 `e(0,r)=0, e(1,r)=d1(r), e(2,r)=d2(r), e(3,r)=d1(r)+d2(r)` 정의 → (4,L)-regular, 4/6-cycle free (Theorem 1).
5. 핵심식: cycle-governing 식이 Property 1(i)(iii)(집합 내/간 차분의 유일성)에 의해 모두 불성립 → girth≥8 보장.
6. 핵심기법 II: t=1 DDS(=Sidon sequence) `d1`에 `[a,b,c]^T·d1`로 (3,L)-regular column-weight-3 단길이 코드, `[0,1,2]`가 8-feasible (Theorem 2).
7. 유연성: Property 4/5로 circulant P 하한이 GCD/Sidon 대비 훨씬 작음(L=10일 때 DDS>180 vs Sidon>380), 한 집합 permute로 L! 비등가 exponent matrix 생성.
8. 최적화: 별도 학습 없음, DDS는 대수적/랜덤 생성, future work로 trapping set 기반 최적 exponent 선택 예고.
9. 결과: 고SNR(>2.5dB)에서 GCD/ZSW 대비 성능 우수, Sidon/Sidon-M/SYM과 거의 동일, KKS 대비 SNR>4dB에서 우수(BER 수치는 그림에만 있어 미상).
10. 한계: HW 미설계, 시뮬만, column-weight 3/4·rate 0.5~2/3 예제뿐(고rate 미검증), error-floor 정량 없음.
