### B. 알고리즘 요약

1. 대상: binary QC-LDPC (CPM-dispersion superposition, `Hgc=CPM(Bgc)`), GC-LDPC(local code + global CN 결합)을 확장한 N-fold MGC-LDPC.
2. 문제: 기존 GC-LDPC의 degree/cycle 분포·threshold를 더 개선하고, 여러 blocklength·rate를 하나의 구조로 지원(rate-compatible)하고 싶다.
3. 핵심정의: N층 Matryoshka-style code가 `Cmgc(1)≺Cmgc(2)≺…≺Cmgc(N)` nested chain을 이뤄 상위층이 하위층을 포함 → 자연스러운 rate-compatible (Theorem 1).
4. 핵심기법1(구성): base matrix `Bmgc(N)`를 하삼각 array로 두고 `Bgc,n`(대각=GC-LDPC), `Bcm,n1,n2`(층간 연결) 조립; truncating(효율적) / masking(유연, cycle·degree 최적화) 두 방식.
5. 핵심식: 2x2/3x3 SM-constraint로 girth≥6/8 보장, masking `Bmgc=Bsm⊙Mmgc`로 short-cycle 축소.
6. 핵심기법2(인코딩): parity-check matrix를 DLT(double lower triangular) form `[[HU1,HU2,HU3],[HL1,HL2,0]]`으로 변환, back-substitution으로 `pL,pU` 계산.
7. 확장(recursive encoding): local code M-1개는 독립 인코딩, 마지막 local code만 방정식 풀이; n층 코드워드는 (n-1)층 `vmgc(n-1)`에서 재귀 생성 (Theorem 3, Algorithm 2).
8. 복잡도: 전처리 `O(l^3+kn l^2+Mn l kn^2)`, 인코딩 `O(kn(Mnl-kn)+l^2)`.
9. 결과: MGC가 동rate/length GC-LDPC 대비 short-cycle 대폭 감소, P-EXIT threshold 0.21~0.37dB 개선, AWGNC BER 1e-8~1e-9까지 error-floor 없음, 50-iter MSA 복호, 4-fold 예제서 rate 5/6~2/3 연속 지원.
10. 한계: 순수 시뮬(HW 없음), min-sum flooding 복호만 검증(전용 저복잡 디코더는 future work), NAND read/soft-quant 미고려, non-binary 미포함.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| MGC/GC base matrix·H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | binary QC-LDPC H-matrix 로드 지점이나, Prime ECC는 특정 QC-LDPC 고정이라 rate-compatible 다층구조 통째 대체는 재검증 부담(프로파일 §4 "높음") |
| CPM-dispersion / circulant shift 구조 | `ecc_data.h` `PCM_b`(base+shift) | 동일 QC 계열이나 global-coupling·nested layer 개념 없음 |
| MSA(min-sum) 복호 | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | 복호는 min-sum으로 동일 계열, 이미 보유(중복) |
| recursive/DLT 인코딩 | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | Prime ECC는 dual-diagonal 고정, DLT recursive 방식은 상이 |
| masking으로 short-cycle 감소 (waterfall) | `ecc_top.cpp` `Load_PCM()` | H-matrix 설계 단계에서 waterfall 개선 아이디어로 부분 참고 여지 |

적용 가치: **중간** - binary QC-LDPC + min-sum + waterfall/threshold 개선이라 Prime ECC와 계열은 일치하나, rate-compatible 다층 global-coupling 부호구조는 특정 QC-LDPC 고정인 우리 코드에 통째 이식하기엔 재검증 부담이 크고, short-cycle 억제 masking 설계 아이디어 정도가 부분 차용 대상이다.
