### B. 알고리즘 요약

1. 대상: systematic (N,K) Reed-Solomon 코드를 GF(2^m) 원소의 이진 표현으로 확장한 binary parity check matrix `H_b` ((N-K)m x Nm)에 BP(belief propagation)를 적용, BPSK+AWGN 채널 가정.
2. 문제: RS 코드의 원래 이진 parity check matrix는 밀도가 약 50%로 높아 factor graph에 4-cycle이 많고 표준 BP가 correlation/error propagation으로 성능이 나쁨.
3. 핵심 가정: 대칭군(GF 원소의 companion matrix 기반 field isomorphism)으로 만든 이진 `H_b`를 저밀도화(듀얼코드의 저가중치 codeword 탐색)하고, 다시 row/column을 추가로 확장해 4-cycle을 제거(밀도 15%까지 감소).
4. 핵심 기법1: RS 코드의 순환성(cyclic property)을 이용해 수신 LLR을 `φ` 심볼만큼 순환 시프트한 뒤 동일한 `H` 행렬로 BP를 수행 - outer round마다 다른 `φ`로 factor graph의 형상을 바꿔 결정론적 오류(에러 정체)를 회피.
5. 핵심 식: potential function `J(H_b,T) = -Σγ_j` (식 13)이 checks 만족 시 최소화(`J_min=-(n-k)`)되며, `H_b`가 고정이어도 `T`(bit reliability 배열)의 시프트가 다르면 `J`가 다른 궤적을 그려 pseudo-equilibrium point를 벗어남.
6. 알고리즘 구조: outer round(순환 시프트 변경)와 inner round(BP iteration)의 이중 루프, 매 BP iteration 후 하드결정 결과를 Berlekamp-Massey(BM) 디코더에도 입력해 성공 시 후보로 저장, 최종적으로 수신벡터와 유클리드 거리 최소인 후보를 선택.
7. 부수 트릭: damping coefficient `θ`(0<θ≤1)로 LLR 업데이트를 완화, extra stopping criterion(`d≤r`, `r=√(Nm)σ²`)으로 BM 성공 시 거리 조건 만족하면 조기 종료해 평균 iteration 수 감소(0.2dB 성능 손실과 trade-off).
8. 학습/최적화 절차 없음 - 결정론적 BP+BM 결합 반복 알고리즘.
9. 결과: RS(15,11)에서 KV(algebraic soft decision) 대비 1.5dB, BM(hard decision) 대비 2.7dB coding gain, ADP 방법 대비도 0.5dB 우위. SNR 증가 시 평균 BP iteration 수 대폭 감소(예: RS(15,11) Eb/N0=2dB에서 253회 → 5dB에서 5회).
10. 한계: 순수 시뮬레이션(HW 미설계), 대상이 Reed-Solomon(비-LDPC, GF(2^m) 기반)이라 LDPC ECC와 부호 구조가 근본적으로 다르고, 짧은 코드(N≤63)로만 검증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| RS 코드의 이진 확장 parity check matrix + BP 디코딩 전체 | 대응 없음 | Prime ECC는 binary QC-LDPC 전용이며 RS(비-LDPC) 부호 자체를 다루지 않음 |
| 순환 시프트로 pseudo-equilibrium 회피(outer round) | `decoder.cpp` `LDPC_Decoder()` (개념적 유사성만) | LDPC min-sum이 stall/oscillation에 빠질 때 메시지 재배열로 탈출하는 아이디어와 개념적으로 유사하나 RS 순환성에 의존하는 기법이라 QC-LDPC에 직접 이식 불가 |
| Berlekamp-Massey 병행 디코딩 | 대응 없음 | 대수적 디코더로 Prime ECC의 min-sum 구조와 무관 |

> 적용 가치: 낮음 — Reed-Solomon 코드 전용 BP 기법으로 Prime ECC의 binary QC-LDPC 부호·디코더 구조와 근본적으로 대응되지 않으며, 이식 가능한 아이디어(순환 재배열을 통한 정체 회피)도 RS의 순환군 구조에 강하게 의존.
