### B. 알고리즘 요약

1. 대상은 array-based `(γ,p)`-regular QC SC-LDPC(예: `C(3,61,100,[15,30,45])`, `R_SC=0.9503`)로, edge spreading으로 구성한 컨볼루셔널 구조.
2. 문제: 양자화(유한정밀) 디코더의 error-floor를 시뮬 없이 예측 - absorbing set이 error-floor의 주원인.
3. 모델: `(a,b)` absorbing set (SPA는 `(4,2)`, MSA는 `(6,0)`이 지배적), AWGN 채널, 양자화 message passing.
4. 핵심 기법: 이전 연구[8]의 LDPC-BC용 code-independent 하한 `λ`를 SC-LDPC로 확장. absorbing set에 최대 LLR을 주입한 실패 확률로 `λ` 산출.
5. 식 `FER ≥ Nλ`: 부호 그래프 내 absorbing set 다중도 `N`에 단일 지배 absorbing set 하한 `λ`를 곱해 전체 하한 근사.
6. 확장: 윈도우 디코더용으로 target symbol absorbing set 다중도 `T(j)`를 정의, `BLER ≥ (λ/L)Σ Ti(L-i+1)`.
7. 디테일: absorbing set이 걸치는 블록 수 `i`별 다중도 `Ni`를 세어 표준/윈도우 하한을 동일하게 만드는 조건 도출(W≥3에서 동일).
8. MSA용 하한은 extra-edge LLR이 iteration마다 단조 증가하는 수정 가정으로 계산.
9. 결과: SC-LDPC 하한이 LDPC-BC 대비 `α=5.08`(SPA)/`8.535`(MSA) 배 우수, 시뮬과 일치. quasi-uniform 5-bit MSA가 uniform 6-bit SPA보다 우수.
10. 한계: HW 미설계, 순수 이론+시뮬, array-based SC-LDPC 구조에 특화된 분석 도구.
