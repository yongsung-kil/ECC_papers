### B. 알고리즘 요약

1. 대상은 IEEE 802.3an 표준의 regular (2048,1723) rate `0.8413` binary LDPC, 64×64 순열행렬 블록 구조의 (6,32) 정칙 부호.
2. 문제: LDPC error-floor를 지배하는 trapping set(여기선 absorption set) 거동을 시뮬 없이 정확 예측하는 것.
3. absorption set 정의: 각 변수노드의 이웃 체크노드 다수가 짝수 번 연결된 변수노드 집합. `(a,b)` = 크기 `a`, EMD(홀수 연결 체크 수) `b`.
4. 핵심 기법: 반복 message-passing에서 absorption set 내부 메시지를 선형 동역학 모델 `x^(l+1)=A x^(l)+extrinsic`로 표현, 행렬 `A`의 최대 고유값 `λ`(=set gain)이 성장 속도 결정.
5. 스펙트럼 정리로 초기 geometric 성장 후 largest eigenvector 방향으로 수렴, 이후 bit-flipping 고정점 도달하는 2단계 거동 규명.
6. (8,8) set의 경우 `λ=4`, 고유벡터는 all-1; 지배 set은 14,272개 존재.
7. 확장: 체크노드의 Taylor 근사로 "check node gain" `μ` 도입, extrinsic 신호의 오류극성 반전 확률을 raw BER `p`로 보정.
8. Gaussian density evolution으로 extrinsic 평균값을 대입, union bound에 multiplicity(14,272)와 8/1723 곱해 BER 추정식 유도.
9. 결과: 해석식이 importance sampling·FPGA 하드웨어 시뮬과 일치, (8,8) set이 error-floor를 지배함을 확인(E_b/N_0=5dB 부근).
10. 한계: 순수 해석·시뮬 이론, HW 미설계, 특정 표준 코드 대상, 디코더 성능 개선책이 아니라 예측 도구.
