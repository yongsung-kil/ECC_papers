### B. 알고리즘 요약

1. 대상: short/moderate 길이 dense parity-check matrix를 갖는 순환형(cyclic/extended-cyclic) 대수부호 (예: [24,12,8] extended Golay, [31,16,7]·[127,64,21] BCH, [47,24,11] QR).
2. 문제: 대수부호는 H가 dense하고 short cycle이 많아 표준 BP 성능이 나쁨. LDPC와 겸용 가능한 soft-in soft-out BP 복호기를 원함.
3. 핵심 아이디어: 한 부호는 구조적으로 다양한 여러 parity-check matrix(dual code의 base=cog)를 가지며, 서로 다른 error pattern을 검출 → 병렬 복호 후 결합(MBBP).
4. base 선택: BEC의 `stopping set` 개수(크기 ≤d)를 성능지표로 cog family를 최적화. cog=cyclic orbit generator, m=n square matrix로 균등 오류보호.
5. 핵심 도구: automorphism group의 affine 순열(`θ:i→q·i+ω mod n`, gcd(q,n)=1)로 한 cog에서 같은 stopping-set 분포의 redundant base를 생성(Lemma III.3, Claim III.4/III.5) → 저장비용 절감.
6. 복호 구조1 MBBP-NX: l개 BP를 서로 독립 병렬 실행 후 종료 시점에 least-metric selection(NX-S) 또는 first-success(NX-FS)로 최종 선택.
7. 복호 구조2 MBBP-X: 매 Np iteration마다 extrinsic 정보를 교환(probability-averaging/highest-reliability/information-combining).
8. 학습/최적화: cog family를 stopping-set 수로 선별(오프라인). Golay는 759개 최소무게 codeword를 33 orbit→3 family(각 11 cog)로 분할.
9. 결과: extended Golay에서 MBBP-NX-S(l=11)가 표준 BP 대비 약 0.75dB 개선, ML bound에 근접. NX-FS는 평균 iteration 대폭 감소하며 NX-S에 근접.
10. 한계: 순수 시뮬레이션(AWGN), HW 미설계. l개 BP 병렬로 복잡도 l배, [127,64,21]에선 Chase Type-2에 열세. 대상이 sparse LDPC가 아닌 dense 대수부호.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 병렬 BP 복호 / 결합 | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `LDPC_Decoder()` | 복호 대상이나 Prime ECC는 단일 H-matrix min-sum, 다중 base 병렬 구조와 불일치 |
| stopping set 기반 base 선택 | `ecc_top.cpp` `Load_PCM()` | H-matrix 로드 지점이나 순환 대수부호 전용이라 QC-LDPC에 직접 대응 없음 |
| full-tanh Sum-Product BP | 대응 없음 | Prime ECC는 min-sum 근사만 지원, full BP 미채택 |
| information-combining 결합 | 대응 없음 | Prime ECC에 다중 디코더 결합 로직 없음 |

적용 가치: **낮음** — 다중 base 병렬 BP는 short dense 대수부호(Golay/BCH/QR) 전용이며, Prime ECC의 단일 sparse QC-LDPC + min-sum 구조와 정합하지 않고 l배 복호기 병렬은 NAND 컨트롤러 면적 예산에 부적합.
