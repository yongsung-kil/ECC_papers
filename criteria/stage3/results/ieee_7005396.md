### B. 알고리즘 요약

1. 대상 시스템은 PM(편광다중) 장거리 광섬유 BICM 링크, PM-16/64/256-QAM 성상도에 SC-FEC를 결합.
2. 문제: 성상도의 서로 다른 modulation bit는 보호수준(crossover prob `p_i`)이 달라, 기존 sequential/random bit mapper는 이 비균일 보호를 활용하지 못함.
3. 두 부호군: (1) protograph 기반 SC-LDPC + BP soft-decision, (2) SC-GLDPC(각 비트를 두 BCH 성분부호로 보호) + iterative hard-decision(BDD).
4. 핵심 기법: bit mapper 행렬 `A=[a_ij]` (공간위치 j의 coded bit를 modulation bit i에 비율 할당, `Σ_i a_ij=1`)를 DE 임계 기준으로 최적화.
5. HDD DE 재귀 `q_j^(l)`(식 (2)/(7))에서 채널확률 `p`를 위치별 유효 crossover `ε_j=Σ a_ij p_i`로 치환해 bit mapper 효과를 반영.
6. SDD는 modified P-EXIT(`[16, Algorithm 1]`)로, 각 protograph column의 할당비율을 반영해 반복성능 예측.
7. 최적화는 differential evolution으로 decoding threshold 최소화 → 최적행렬 `A*` 산출.
8. 결과: tailbiting SC 부호는 앞쪽 공간위치를 좋은 modulation bit에 몰아 인공 wave decoding 유발, 종단(terminated) 대비 낮은 OH로 유사 capacity gap 달성.
9. 수치: SC-LDPC, R=0.75, PM-64-QAM, 길이 360000에서 BER 1e-5 기준 약 0.55 dB 이득, 비선형 전송에서 전송거리 약 13%(3 span) 연장.
10. 한계: 부호설계는 다루지 않고 문헌 파라미터 사용, HW 미설계, 시뮬/DE 뿐, 성상도 UEP 없으면(BPSK/QPSK) 이득 없음, error-floor 미분석.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| bit mapper(coded bit→modulation bit 할당) | 대응 없음 | Prime ECC는 성상도/BICM 없는 NAND binary 채널 → 개념 부재 |
| SC-GLDPC + BCH 성분부호 HDD | 대응 없음 | binary QC-LDPC min-sum 구조와 부호군 자체가 상이 |
| SC-LDPC BP + windowed decoder | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `LDPC_Decoder()` | 원리상 message passing이나 SC/windowed 구조 미보유, 이식 실익 낮음 |

적용 가치: **낮음** — 광통신 coded-modulation의 성상도 UEP 기반 bit mapper 최적화로, NAND의 binary(성상도 없음) QC-LDPC에는 대상 개념 자체가 존재하지 않음.
