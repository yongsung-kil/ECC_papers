### B. 알고리즘 요약

1. 대상은 perpendicular magnetic recording(HDD) 채널이며, PR1/PR3 partial response 등화 후 BCJR 알고리즘으로 LLR을 생성하고 이를 PG-LDPC(rate 0.931~0.949, length 4296~5797) SPA 디코더에 입력하는 구조.
2. 문제: pattern-dependent noise(2-D 패턴 의존 잡음) 환경에서는 인접 비트에 따라 LLR 분포가 크게 달라져(3비트 패턴 "010","101"이 특히 표준편차가 큼) 기존 LLR이 애매한(ambiguous) 샘플을 다수 포함, 반복 복호 효율이 저하됨.
3. 가정: 3비트 조합("000"~"111")별 LLR을 3차원 공간에 매핑하면 패턴별로 서로 다른 분포(클러스터)를 형성한다는 실측 관찰에 기반.
4. 핵심 기법: SVM(RBF 커널 `K(x_i,x)`)로 인접 패턴 쌍 사이 경계에 hyperplane(S1~S9, 총 9개 곡면)을 생성, 판별함수 `f(x)=Σα_i y_i K(x_i,x)+b`.
5. 각 비트의 LLR을 해당 hyperplane까지의 거리 기반으로 재조정(rescale)하여 패턴 의존 편향(bias)을 제거 — 재조정된 LLR이 기존 LLR보다 오류샘플을 더 명확히 분리.
6. 훈련 데이터 선정 조건 3가지(분포 중심과의 거리, 오류비트 제외, 대부분 오류비트가 hyperplane 근방에 오도록 스케일링)로 학습 품질 확보.
7. 부수 디테일: 6-bit ADC, 9-tap transversal filter equalization, 실제 2.5" HDD(600Mb/s, ~1000kfci)에서 채취한 실측 데이터(BER~10^-3) 사용.
8. 학습 절차: SVM quadratic programming으로 support vector만 남기고 나머지 계수는 0으로 만들어 연산량 절감.
9. 결과: PG-LDPC rate 0.93 코드에서 SPA 반복 횟수를 기존 LLR 대비 약 절반으로 감소, rate 0.949 코드는 기존 LLR로 9회 반복에도 오류 미정정이었으나 재조정 LLR로는 7~9회에 전부 정정.
10. 한계: 3차원(3비트 패턴) 공간에 국한된 실험이며(저자는 확장 가능하다고만 언급), HW 구현/합성 없음, LDPC 부호·디코더 자체는 기존 것을 그대로 사용(전처리 단계만 기여), 자기기록 채널 특유의 pattern-dependent noise 모델에 특화되어 NAND 채널(cell-to-cell interference, retention 등)과는 잡음 모델이 상이.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SVM 기반 LLR 재조정(hyperplane 전처리) | `channel.cpp` `Set_LLR_Th()` | 개념적으로는 soft-read LLR 임계값 설계와 유사한 위치이나, SVM 학습 기반 접근은 Prime ECC의 threshold 테이블 방식과 근본적으로 다르고 자기기록 잡음 모델 전용 |
| LDPC SPA 디코딩 자체 | `decoder.cpp` `LDPC_Decoder()` | 본 논문은 기존 PG-LDPC SPA 디코더를 그대로 사용, 디코더 알고리즘 개선 없음 |
| (대응 없음) 자기기록 pattern-dependent noise 모델 | 대응 없음 | Prime ECC는 NAND RBER/fixed-error 채널 모델(`channel.cpp`)을 사용하며 자기기록 특유의 인접비트 패턴 잡음 모델은 대응 없음 |

적용 가치: `낮음` — NAND 채널과 무관한 자기기록(HDD) 특화 잡음 모델 기반 전처리 기법이며, LDPC 부호/디코더 자체에 대한 기여가 없음.
