### B. 알고리즘 요약

1. 대상: irregular binary LDPC 코드의 랜덤 앙상블 `H^m_{c,r,n}` (열/행 weight 분포 `κ_i`, `χ_i`로 파라미터화), 채널 비특정(일반 대칭 채널).
2. 문제: 개별 코드의 trapping set을 전수 열거하는 것은 계산상 intractable하므로, 앙상블 평균으로 통계적 분포를 구하는 확률론적 접근이 필요.
3. 핵심 정의: `(a,b)` trapping set — `a`개 variable node로 유도된 부분그래프에 `b`개의 홀수차수 check node가 존재; `b`가 모두 degree 1·2인 경우는 elementary trapping set.
4. 핵심 기법: Sanov's theorem(대편차 이론)과 Varadhan's lemma를 이용해 `a=θn, b=λn`(선형 크기)인 trapping set 개수의 점근 로그 성장률 `e(θ,λ)`, `t(θ,λ)`을 지수함수 형태로 유도.
5. 핵심 식: `e(θ,λ) = sup[Σκ_jH(θ_j/κ_j) − ζI(λ,θ̃) − Σκ_jsⱼH(θ̃/Σκ_jsⱼ)]` — 열/행 weight 분포, 상대엔트로피 `I(λ,θ̃)`로 표현되는 앙상블 평균 trapping set spectrum.
6. 확장: elementary trapping set(Theorem 3.1)에 이어 제약을 완화한 general trapping set(Theorem 3.2)에 대해서도 동일 방법으로 점근분포 유도.
7. 구현 디테일: 고정점 방정식(`ρ`, `A`/`τ`, `θ̃`, `y`, `z`)을 수치적으로 풀어 점근 성장률 계산 — closed-form이 아닌 수치 최적화 필요.
8. 최적화 절차: Lagrange multiplier를 이용한 제약조건 하 상대엔트로피 최소화(각 theorem의 proof에서 전개).
9. 결과: 정성적으로 열/행 weight 분포 및 코드율이 trapping set 점근 개수에 강하게 영향을 미침을 확인(수치 예제·시뮬레이션 비교는 본문에 없음).
10. 한계: 순수 이론(closed-form 점근식과 수치적 고정점 방정식)뿐이며 구체적 코드 예제, 시뮬레이션 검증, HW/디코더 설계는 전혀 다루지 않음.
