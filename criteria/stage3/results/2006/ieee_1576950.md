### B. 알고리즘 요약 (10줄 내외)

1. 시스템: binary LDPC, irregular(1268,456) 및 regular(504,252) 코드, AWGN+BPSK 채널, BP(LR domain)와 MS(LLR domain) 두 알고리즘 대상.
2. 풀려는 문제: 기존 반복복호(flooding schedule)는 fixed-point 문제 `x=F(x)`를 successive substitution(SS)으로 푸는 것과 동일한데, 사이클이 있는 그래프에서 SS의 수렴 속도/정확도가 최적이 아님.
3. 핵심 가정: 연속시간 비동기 아날로그 디코더의 동작을 노드 간 전파지연(propagation delay) `τ`를 갖는 1차 비선형 미분방정식 `τ dx/dt = F(x) - x`로 모델링(Elmore delay 근사).
4. 핵심 기법 1단: 이 미분방정식에 forward Euler를 적용하면 `x_{k+1} = (1-λ)x_k + λF(x_k)`의 successive relaxation(SR) 재귀식이 되며, `λ=Δt/τ`가 relaxation factor. `λ=1`이면 기존 SS(동기 디지털 flooding)와 동일.
5. 핵심 식 의미: `λ<1`(underrelaxation)일수록 아날로그 극한에 가까워지고 수렴 정확도가 개선되나, 동기 플랫폼에서는 평균 iteration 수가 증가하는 복잡도-성능 트레이드오프가 생김.
6. 확장: `λ→0, N(최대iteration)→∞` 극한이 "이상적 아날로그 디코더"의 steady-state 성능을 대표하는 벤치마크로 제시됨.
7. 부수 트릭: 노드 간 지연 분포를 절단 가우시안(truncated Gaussian)으로 가정해 다양한 분산값에서 실험한 결과, steady-state 성능은 지연 분포 분산에 거의 무관함을 확인.
8. 학습/최적화 절차: 없음(수치해석적 SR 방법론이며 DE/GA/NN 미사용); backward/forward Euler 두 적분법으로 비교 검증.
9. 결과: (1268,456) 코드에서 relaxed BP가 SS 대비 0.7dB, relaxed MS가 0.5dB 개선(높은 Eb/N0 영역); 특정 λ 구간에서는 평균 iteration 수를 유지하거나 오히려 줄이면서 성능 개선(예: λ=0.7에서 iteration 유지, 0.4dB 개선).
10. 한계: 회로 비이상성(mismatch, 유한 동적범위) 미고려한 "이상적" 아날로그 모델일 뿐이며, HW(RTL/ASIC) 설계나 실측 검증 없음; 동기 디지털 플랫폼 적용 시 λ가 작을수록 iteration 수가 증가해 실제 HW 이득은 제한적.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Successive relaxation(damped) VN/CN 업데이트 | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | 기존 message-passing 업데이트를 이전 값과의 가중합으로 대체하는 스케줄링 변형, 낮은 난이도로 이식 가능하나 부가 저장(이전 iteration 값) 필요 |
| CN min-sum 연산(relaxed MS) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | MS relaxation 적용 시 해당 함수의 업데이트 규칙 수정 필요 |
| 아날로그 회로 구현 자체(비동기 연속시간) | 대응 없음 | Prime ECC는 디지털 bit-exact 클록 기반 시뮬레이터로 아날로그 회로 모델과 근본적으로 불일치 |
| 지연분포 기반 steady-state 분석 | 대응 없음 | 아날로그 회로 특화 분석으로 디지털 HW 시뮬레이터에 대응 없음 |
```

적용 가치: `중간` — 핵심 기여인 아날로그 회로 구현 자체는 Prime ECC(디지털 bit-exact)와 무관하지만, relaxation(damped update) 아이디어는 동기 디지털 min-sum 디코더에도 부분 적용 가능한 성능 개선 기법으로 참고 가치 있음.
