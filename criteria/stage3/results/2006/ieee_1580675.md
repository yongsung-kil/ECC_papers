> 총평: 여러 structured LDPC 코드 클래스(EG, RS-based, array, TPC/SPC, RA/IRA/eIRA) 간 등가성을 증명하는 순수 부호이론+시스템레벨(SFR/RS concatenation) 논문으로, magnetic recording 채널·outer RS 코드 최적화에 초점이 맞춰져 있어 NAND binary QC-LDPC HW 이식과는 거리가 있음. 부호 구성 이론적 배경지식으로만 참고 가치.

### B. 알고리즘 요약

1. 시스템: perpendicular magnetic recording 채널(PR/partial response) + 반복 검출/디코딩(SISO detector + LDPC 디코더) + outer Reed-Solomon 코드로 구성된 concatenated 시스템.
2. 문제: 다양한 structured LDPC 코드군(FG, RA/IRA/eIRA, Array, Combinatorial(BIBD), RS-based, TPC/SPC)이 독립적으로 제안되어 있으나 상호 관계가 불명확했고, BER만으로는 저장장치 시스템 성능(섹터 실패율)을 제대로 평가할 수 없었음.
3. 핵심 가정/모델: readback signal `r_k = Σ a_i h(t-iT) + n_j + n_g` (transition jitter noise `n_j` + AWGN `n_g`)의 linear superposition 모델, transition response는 perpendicular 임펄스 응답으로 근사.
4. 핵심 기법 1: structured LDPC 코드 간 수학적 등가성 증명(Lemma 1~4) — 예: extended RS-based LDPC 코드가 EG-Gallager LDPC(유클리드 기하 위 직선의 incidence vector)와 동치, lattice/array LDPC와도 파라미터 매핑으로 동치.
5. 핵심 식 의미: 이 등가성은 서로 다르게 "발명"된 구성법들이 실제로는 같은 코드족이라는 것을 보여, 구현/이론 분석을 통합할 수 있게 함(엔지니어링적으로는 새로운 코드 성능이 아니라 분류 정리).
6. 핵심 기법 2: 시스템 평가지표로 BER 외 SFR(sector failure rate)을 도입, global iteration(1회 검출+N회 LDPC 디코딩) 단위로 시뮬.
7. 부수 기법: outer RS 코드의 오류정정능력 T를 변화시키며 SFR을 "bathtub" 곡선으로 관찰, overall redundancy(RS+LDPC parity) 기준 최적 T 탐색.
8. 최적화 절차: 별도 학습기법 없음(그리드 서치 방식 시뮬레이션으로 최적 T, code 선택).
9. 결과: 무작위(random) LDPC 코드가 대부분 SNR 구간에서 최고 BER, structured 코드 중 RS-based/array 코드가 SFR 측면에서 random과 근접, EG(column weight 8)는 고SNR에서만 우수. BER 우수 코드가 SFR은 더 나쁠 수 있음을 확인.
10. 한계: HW 설계·게이트카운트·양자화(soft-bit) 논의 없음, magnetic recording 채널 특화(NAND 채널 아님), 순수 시뮬레이션 기반 시스템 레벨 비교로 알고리즘적 디코더 개선은 없음(min-sum 등 언급 없음).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| structured LDPC(RS-based/EG/array) 부호 구성 등가성 | `ecc_top.cpp` `Load_PCM()` | H-matrix 구성법 이론이나, Prime ECC는 이미 특정 QC-LDPC(base+circulant) 고정이라 재구성 부담 큼 |
| outer RS 코드 concatenation 및 T 최적화 | 대응 없음 | Prime ECC는 순수 LDPC ECC 엔진이며 outer RS concatenation 구조 없음 |
| SFR(섹터 실패율) 시스템 평가 지표 | 대응 없음 | Prime ECC 통계는 iteration/FER 중심, SFR 개념 자체가 스토리지 섹터 단위 평가로 별도 프레임워크 필요 |

> 적용 가치: 낮음 — magnetic recording 특화 시스템 레벨 분석이며 부호 구성 등가성은 이론적 흥미 수준. HW/디코더 알고리즘 개선이 없어 Prime ECC 3.1 이식 실익이 낮음.
