### B. 알고리즘 요약

1. 자기기록 채널(EPR4+AWGN)용 고rate(8/9~15/16) regular QC-LDPC 코드, 두 개 Altera Stratix-II FPGA로 구성된 전용 하드웨어 시뮬레이터로 초저 sector error rate(~10^-9)까지 평가.
2. 기존 컴퓨터 시뮬레이션은 sector error rate 10^-4 수준까지만 도달 가능해 HDD 요구사양(10^-12)의 error floor 검증이 불가능한 문제를 해결하려 함.
3. 채널 모델은 EPR4 신호+AWGN, Max-Log-MAP detector(sliding window)와 QC-LDPC 디코더가 최대 4회 detector-decoder iteration, 디코더는 매회 4회 내부 iteration을 수행.
4. parity-check 행렬 `H`는 `p x p` circulant 서브행렬 `Hij`(GF(2))의 배열이며, circulant weight `w`(0/1/2)를 갖는 랜덤 구성 regular QC-LDPC, 4-cycle-free 제약.
5. 핵심 관찰: circulant 크기 `p`가 작으면(p=64) error floor가 심하고, `p`가 충분히 크면(128, 256 이상) 관측 가능한 영역에서 error floor 미발생 — column weight 4가 최적으로 제시됨.
6. 디코더 개선: circulant 크기 `p`에 반비례하던 기존 병렬성을, `v`(p를 나누는 값, `h=p/v`)개씩 그룹화해 CNU/VNU 당 h배 병렬성 향상.
7. 메모리 구조: DMMB(decoding message memory block, w개/circulant)와 CMMB(channel message memory block, n개)로 나눠 binary counter+barrel shifter로 read/write address 생성, 각 iteration을 2v clock cycle(CN phase v + VN phase v)로 처리.
8. 별도 학습/최적화 절차 없음; 랜덤 구성 후 시뮬레이션 기반 경험적 설계 규칙(circulant 크게, weight 0 회피) 도출.
9. rate 8/9, N=4608, column weight 4 코드에서 p=64는 error floor 발생, p=128/256은 SNR 3.5~7dB 구간에서 error-floor-free(sector error rate 10^-9까지); rate 15/16(N=16384)도 error floor 미관측.
10. 한계: gatecount/throughput 등 HW 정량 수치는 본문 서술에 없음(그림/표만 존재); 자기기록 채널·Max-Log-MAP 결합 시나리오로 순수 LDPC 채널 코딩과 다르며 검증이 관측 가능한 SNR 구간에 한정됨.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 고rate QC-LDPC 랜덤 구성 + circulant 크기 기반 error-floor 설계 규칙 | `ecc_top.cpp` `Load_PCM()` | H-matrix 구성법 자체 이식은 난이도 높으나 error-floor 대책 규칙(circulant 크게, weight 0 회피)은 참고 가치 |
| 디코더 병렬성 h배 향상 아키텍처(CNU/VNU 그룹, DMMB/CMMB) | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | Prime ECC는 z_sb=32 고정 병렬단위이며 이 논문의 가변 h 그룹핑과 구조 다름, 참고만 가능 |
| soft-output 6bit 양자화(3 fractional) | `channel.cpp` `Set_LLR_Th()` | soft-2~3bit를 넘는 통신채널용 양자화로 NAND read 비용 관점과 다름 |
| Max-Log-MAP detector + LDPC iterative detection-decoding | 대응 없음 | 자기기록 채널 특화 joint detection 구조로 NAND ECC와 무관 |

> 적용 가치: 중간 | error-floor 설계 규칙과 circulant-size 기반 디코더 병렬화 개선은 참고 가치가 있으나 자기기록 채널 특화 및 정량 HW 수치 부재로 직접 이식보다는 아이디어 참고 수준.
