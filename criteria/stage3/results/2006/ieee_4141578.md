### B. 알고리즘 요약

1. DVB-S2 표준의 구조화(structured) QC-LDPC(p=360 circulant component matrix) 코드, code length 16200/64800, rate 1/5~9/10을 대상으로 하는 partly-parallel 프로그래머블 디코더.
2. 목표: full-parallel(면적↑)과 full-serial(성능↓) 사이에서, 다양한 code rate/length에 걸쳐 성능 저하 없이 재구성 가능한 디코더 구현.
3. Belief propagation(log-domain sum-product) 기반 반복복호, check node 메시지는 비선형 Gallager 함수 `Φ(x)=log((e^x+1)/(e^x-1))`와 그 역함수 `Φ^-1`를 LUT로 계산.
4. 핵심 기법: 기존 균일양자화(9비트) 대신 floating-point(FP) 양자화 `f(x) = 2^(x-1)×(2^f+y)×d` (x=exponent, y=mantissa f비트, d=최소step)를 `Φ(x)` 출력에 적용.
5. FP 양자화는 0 근처를 미세 step으로, 큰 값은 넓은 step으로 표현 - 작은 비트수(5비트, exponent 3+mantissa 2)로 큰 dynamic range(0~448d) 확보, 9비트 균일양자화와 동일 정밀도.
6. 변형/확장: Zhang의 variable-precision(VP) quantization(2단계 step)을 개선해 threshold/step 파라미터를 임의 설정 가능하게 일반화한 것이 FP 양자화의 전신 비교 대상.
7. 구현 디테일: CNOD 내부에서 FP↔uniform 변환기(selector+rounding adder)를 두어 덧셈은 uniform 도메인에서, LUT는 FP 도메인에서 수행. 수신 LLR/CNOD 메시지 6비트, VNOD 메시지 7비트 균일양자화(2's complement) 별도 채택.
8. 최적화 절차 없음(고정 설계), syndrome check 모듈로 조기 종료(early termination) 지원.
9. 결과: FP(3,2) 5비트 양자화가 9비트 균일양자화 대비 0.02dB 손실에 불과, LUT당 197게이트+딜레이메모리 최대 1080게이트 절감(CN degree 30 가정), 총 180개 CN processor에서 180k게이트 절감. 전체 0.18µm 0.92Mgates, throughput 150Mbps, 구현손실 0.03~0.04dB.
10. 한계: DVB-S2 AWGN 채널 전용 시뮬/실측(VLSI 테스트보드)이며 min-sum이 아닌 정확한 sum-product 계산 방식, NAND read 채널이나 error-floor 대책은 다루지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Gallager 함수(Φ) FP 양자화 LUT 최적화 | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC는 min-sum(비선형함수 LUT 불필요)이라 직접 대응 없음, sum-product 전용 기법 |
| soft-info 양자화(6~7비트 균일) | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | 양자화 비트폭 설계 아이디어는 참고 가능하나 목적(LUT압축)이 다름 |
| p=360 circulant 기반 QC-LDPC HW 매핑, barrel shifter | `ecc_top.cpp` `Load_PCM()` | QC-LDPC 구조 개념은 유사하나 DVB-S2 고정 매트릭스로 재사용성 낮음 |
| syndrome check 조기종료(SYND) | `partial_CRC.cpp` `partial_crc_HW_T4()`, `full_CRC.cpp` | 목적(반복 조기종료) 유사하나 CRC가 아닌 syndrome=0 검사 방식으로 세부 구현 상이 |

> 적용 가치: 낮음 - 핵심 기여(Gallager 함수 FP 양자화)는 sum-product 디코더 전용이며 Prime ECC는 이미 min-sum(비선형함수 LUT 없음)을 채택해 직접 이식 대상이 없음. QC-LDPC HW 매핑 개념만 참고 가치.
