> 총평: 유한 길이 irregular LDPC의 degree distribution·girth를 동시에 고려한 실용적 구성법(MEBF)이지만, semi-random 구조와 dv/dc 최적화 중심이라 QC-LDPC 고정 구조인 Prime ECC로의 직접 이식성은 낮음.

### B. 알고리즘 요약

1. 유한 길이(N=1024) irregular LDPC 부호를 AWGN/BPSK 채널, rate 1/8~7/8 5종에 대해 설계.
2. 무한 길이 점근 최적화로 얻은 degree distribution은 유한 길이에서 성능이 나쁠 수 있다는 문제(선행연구[4] 인용)를 보완하려 함.
3. 채널 모델은 AWGN(표준편차 σ), Eb/No = 1/(2Rσ²), Gaussian approximation[8]으로 threshold 계산.
4. 1단계(구조 선택): variable/check node degree 종류를 2~3개로 제한하고 column/row linear density를 5% 이내로 제한하는 semi-random 2-form 구조(`λ(x)`, `ρ(x)`, Eq.9,10) 채택.
5. 핵심 식: semi-random parity check matrix(Eq.7)는 결정적 대각 부분 + 랜덤 부분으로 구성되어 fast-encoding(Eq.8, 밀집 생성행렬 없이 인코딩)을 가능케 함.
6. 2단계(점근 최적화): 각 form에서 독립 파라미터(i, 또는 c/v/i)만 threshold 최대화로 탐색해 후보 (λ,ρ) 집합 선정.
7. 3단계(선별): Modified EBF(MEBF) - 기존 EBF[7]에 row target weight를 오름차순 정렬 후 순차 제한을 추가해 row weight까지 제어하며 4-cycle 등 short cycle 회피.
8. 최적화 절차: girth 우선 생성(EBF/MEBF) 후 후보 코드들을 BER 시뮬레이션으로 비교해 최종 코드 선정(설계 자체가 학습 절차).
9. 결과: 동일 length(1024) 기준 규칙(regular) 코드 대비 rate 1/8,1/4,1/2,3/4,7/8에서 각각 약 5dB,1dB,0.25dB,0.5dB,0.75dB 이득(BER 10⁻⁷ 기준), MEBF가 PEG/EBF 대비 우수(Fig.2).
10. 한계: 순수 시뮬레이션 기반 부호 구성법 논문으로 디코더 HW/복잡도/throughput 분석 전혀 없음, QC 구조 아닌 semi-random 구조라 하드웨어 정합성 별도 검증 필요.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| semi-random parity check matrix 구성(degree distribution + MEBF) | `ecc_top.cpp` `Load_PCM()` | 부호설계 자체이나 QC-LDPC가 아닌 semi-random 구조라 base matrix+circulant shift 방식과 이질적, 재검증 부담 큼 |
| fast-encoding(semi-random 이중대각 구조) | `encoder.cpp` `Generate_PCM_encoding()` | 개념적으로 dual-diagonal 인코딩과 유사하나 코드가 다름 |
| girth 제어(MEBF) | 대응 없음 | Prime ECC는 고정 H-matrix 사용, 동적 girth 최적화 파이프라인 없음 |
| 디코더/HW | 대응 없음 | 논문에 디코더 알고리즘·HW 설계 없음 (표준 BP 시뮬레이션만 가정) |

> 적용 가치: 낮음 — 유한 길이 irregular LDPC 구성법으로서 학술적 가치는 있으나 QC-LDPC 고정 구조인 Prime ECC와 부호 형식이 근본적으로 달라 이식 난이도가 높고(§4 "부호설계" 항목 최고난도), NAND ECC급 고rate 영역(rate 1/8~7/8 중 저rate 위주)과도 초점이 다름.
