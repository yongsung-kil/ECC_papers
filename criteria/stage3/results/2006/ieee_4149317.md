### B. 알고리즘 요약

1. 짧은 block length(`N<1000`)·고rate(`R>2/3`)의 regular QC-LDPC 부호를 조합론적 설계(BIBD)로 구성하는 문제를 다룸.
2. 기존 구성법은 length-4 cycle을 없애기 위해 λ=1 BIBD만 사용했으나, 짧은 길이·고rate 조건에서는 λ=1로 원하는 파라미터를 만들 수 없는 경우가 있어 성능이 제한됨.
3. λ=2 BIBD를 순환 차분족(cyclic difference family)으로 구성하면 Tanner graph에 일부 length-4 cycle이 남지만, 고rate 조건에서는 그 개수(`N_c=C(v,2)`)가 상대적으로 작아 성능 저하가 미미하다고 가정.
4. `t`개의 `k`-부분집합 base block `B_i`를 abelian group `G=Z_v`에서 구성, 각 원소 차분이 λ번씩 나타나면 difference family가 되고 이를 평행이동해 `(b,v,r,k,λ)` BIBD를 형성. incidence matrix `Q`의 전치가 검사행렬 `H=Q^T`.
5. `H`는 각 `Q_i`가 순환(cyclic)이므로 QC 구조가 되며 `(k,r)`-regular; rank는 `k`가 홀수면 `v`, 짝수면 `v-1`(식 4)이고 rate `R≈1-v/b`(식 5) — v가 작을수록(고rate) length-4 cycle 수가 적음.
6. 예시: `(19,4,2)` 순환 차분족으로 `(57,19,12,4,2)` BIBD를 만들어 `(4,12)`-regular QC-LDPC, `N=57, K=39, R=39/57`, cycle 수 `N_c=171` 구성.
7. 특별한 부수 트릭 없음. Table I에 다양한 `(N,K)` 조합(57~724)과 대응 BIBD 파라미터 목록 제공.
8. 학습/최적화 절차 없음. 조합수학적 direct construction.
9. AWGN+BPSK, sum-product 복호(최대 iteration 50) 시뮬 결과: `(155,125)` 코드는 λ=1 free-cycle 코드와 성능이 거의 동등(고SNR)하며 무작위 구성 대비 0.7dB 이득; `(252,224)`(R=0.89)는 무작위 대비 1dB 이득; `(255,204)`는 무작위 코드에서 발생하는 error floor가 구조코드에서는 나타나지 않음.
10. HW 미설계, 시뮬레이션만 수행. length가 최대 724로 작아 대형 NAND 코드로의 확장성은 미검증하며, length-4 cycle 허용이 항상 이득인지는 rate·length 의존적.
