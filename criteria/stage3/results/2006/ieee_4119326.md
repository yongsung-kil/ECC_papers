### B. 알고리즘 요약 (10줄 내외)

1. 시스템: 2-user 협력 무선 통신(User1, User2가 base station과 통신), inter-user 채널과 각 user-to-BS 채널은 독립 flat Rayleigh fading + AWGN.
2. 문제: 소형 모바일 단말은 다중 안테나 탑재가 어려워 transmit diversity 확보가 곤란 → coded cooperation으로 업링크 다이버시티 확보.
3. 채널모델: `r_ij(n) = a_ij(n)*s_i(n) + z_j(n)`, `s_i(n)=sqrt(E_b)*b_i(n)` (BPSK), fading coefficient `a_ij`는 slow/fast fading 가정, inter-user link는 reciprocal.
4. 핵심 기법: Parallel Concatenated LDPC(PC-LDPC) 코드 — S개 component LDPC 코드를 인터리버 없이 병렬 연접, 전체 rate `1/R = 1/r1+1/r2+...+1/rS-(S-1)`.
5. 부호어 `C=[d p1 p2 ... pS]`, 전체 parity-check 행렬은 블록대각+공유 정보비트 열 구조(식(1),(3)); `H1d=[1], Hi^p=[1 1]`로 두면 semi-random LDPC(Li Ping)로 환원.
6. 협력 전송: 각 user는 자신의 codeword를 rate `rf`(1st frame)와 rate `rf2`(2nd frame) 두 세그먼트로 분할, 파트너의 1st frame을 성공 복호하면 파트너 몫의 추가 parity 비트를 2nd frame에서 송신(실패 시 non-cooperative 모드로 자기 parity만 송신).
7. 파트너 노드 복호: source-relay(inter-user) 링크 품질이 좋다는 가정 하에 간단한 bit-flipping(BF) 복호만 사용(저지연·저전력), 목적지(base station)는 표준 sum-product 반복복호 사용.
8. 별도 학습/최적화 절차 없음(랜덤 LDPC 2개를 rate 1/2 component code로 사용).
9. 결과: K=512, 전체 rate 1/3, rate-1/2 component 2개 조합. BER=10^-3에서 비협력 대비 3.5~9dB 이득(inter-user 채널 품질 5dB~perfect에 따라), 2-antenna 수신 시 3.8~8dB 이득.
10. 한계: HW 미설계, Rayleigh fading 무선 협력 시나리오 전용 시뮬레이션(단일 채널·시스템 모델), NAND/저장장치 관련성 없음.
