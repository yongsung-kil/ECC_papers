### B. 알고리즘 요약

1. 대상은 perpendicular magnetic recording을 모사한 AWGN partial-response 채널(impulse response (4,6,4,2)).
2. 기존 post-Viterbi processor는 약한 패리티 코드용, turbo equalization은 강한 LDPC 코드용으로 쓰이는데, 본 논문은 약한 패리티 코드에 turbo equalization을 적용해 복잡도는 낮추고 postprocessing 성능을 확보하려는 문제를 다룸.
3. 채널 모델은 이상화된 AWGN partial-response 채널이며, 복호는 채널검출용 BCJR과 패리티코드용 message-passing을 번갈아 수행하는 turbo iteration 구조.
4. 핵심 기법은 rate `R=(n-1)/n` interleaved single-parity-check(SPC) 코드로, 검사행렬 `H_0 = [I_M I_M ... I_M]` (identity matrix 연결) 구조를 가짐.
5. 이 구조 덕에 message-passing 그래프에서 parity 부분은 메시지 N개뿐이고 degree-1 bit node는 연산이 필요 없어 LDPC 대비 복잡도가 크게 낮음.
6. 확장으로 UEP(Unequal Error Protection) 패리티 코드 `H_1`을 제안: RS 코드워드 경계(i mod q = 0)에 해당하는 컬럼은 weight r로 강보호, 나머지는 weight 1로 약보호.
7. UEP 코드는 H_0에서 시작해 강보호 컬럼에 pseudo-random하게 비트를 추가하며 row/column weight 제약을 만족시키는 방식으로 구성.
8. 별도의 학습/최적화 절차는 없음(구조적 구성 + multinomial 분석으로 섹터 오류율 계산).
9. 결과: R=50/51 SPC 코드가 3회 turbo iteration 후 BER 10^-4/10^-5에서 각각 1.3dB/1.6dB 이득(R=64/65는 1.2dB/1.5dB), UEP 코드는 동일 rate SPC 대비 섹터오류율에서 유의미한 이득.
10. 한계: LDPC가 아닌 단일 패리티 코드 기반이며 자기기록/partial-response 채널 시뮬레이션에 한정, HW 미설계, NAND 채널·오류 모델과 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SPC/UEP 패리티 코드 message-passing 복호 | 대응 없음 | binary QC-LDPC min-sum 디코더와 근본적으로 다른 저복잡도 SPC 구조 |
| BCJR 기반 turbo equalization | 대응 없음 | NAND ECC는 partial-response 채널 검출기와 결합되지 않음 |
| UEP 컬럼 weight 차등 보호 | 대응 없음 | Prime ECC의 QC-LDPC H-matrix 설계와 무관한 개념 |
