> 총평: 분산 파일 스토리지 시스템에서 노드 장애 내성을 위한 소규모 LDPC 소거부호 설계·프로토타입 논문으로, NAND 바이너리 채널코딩 ECC와는 적용 계층(파일시스템 vs 채널코딩)과 부호 목적(소거정정 vs 오류정정)이 달라 이식 가치가 낮음.

### B. 알고리즘 요약

1. 대상 시스템은 commodity 컴퓨팅 클러스터(P2P overlay) 위의 분산 스토리지: 파일을 n개 데이터 청크 + m개 코드(패리티) 청크로 분할 저장.
2. 문제: 기존 방식은 완전 replication(높은 오버헤드) 또는 Reed-Solomon 소거부호(최적이지만 재구성 시 O(n·m) 연산, XOR만으로 부족)에 의존; 소규모(n,m<1000) 영역에서 좋은 LDPC 부호 구성법이 알려져 있지 않음.
3. 핵심 가정/모델: 노드(서버) 장애를 확률 `1-μ`의 독립 소거(erasure)로 모델링, 저장 오버헤드 `S=(n+m)/n`, 부호율 `R=n/(n+m)`.
4. 핵심 기법: Monte-Carlo 탐색으로 소규모(n,m ≈ 8~20) LDPC 이분그래프(패리티 `c_i = XOR of connected d_j`)를 생성하고, 최악의 경우 복구 가능 청크 수(`f_max·n`)를 극대화하는 그래프를 선택.
5. 핵심 식: 가용성 `A_e(μ,n,m) = Σ_{i=n}^{n+m} C(n+m,i) μ^i(1-μ)^{n+m-i}` (소거부호), 이를 replication 가용성 `A_r(μ,S)`와 비교해 동일 오버헤드에서 LDPC가 압도적으로 우수함을 보임(예: μ=0.95, S<1.5에서 LDPC가 replication보다 여러 자릿수 낮은 실패율).
6. 확장: 표준 greedy 알고리즘(재귀적 알려진 청크로부터 미지 청크 복원)으로 디코딩; degree 분포는 `0.4<p<0.6` 범위에서 랜덤 생성 후 최적 그래프를 Monte-Carlo로 선별.
7. 구현 디테일: n=8, m=6 (R=4/7) 채택 — 3개 이상 미싱 청크까지 견디며 오버헤드/가용성 트레이드오프가 우수; 클라이언트가 결정론적 해시로 청크명→저장 노드를 결정, HTTP 기반 전송, gossip(rumor-mongering) 프로토콜로 host list 일관성 유지.
8. 최적화 절차: 서로 다른 파라미터(n,m) 조합에 대해 무작위 이분그래프를 다수 생성·평가하여 최악 오버헤드 `f_max`가 가장 작은(즉 최소 개수의 청크만 있어도 복구 가능한) 그래프를 채택.
9. 결과(실측): lxplus(CERN, ~100노드)와 Muenster 클러스터(~50노드) 프로토타입에서 집계 처리율(aggregated rate) 최대 ~110MB/s / ~380MB/s, 노드 3개 실패까지 정상 복구 확인, R=2/3와 R=4/7의 실패율 비교 시 R=4/7이 유사 오버헤드로 실패율 대폭 개선(10^-6 이하 수준 유지).
10. 한계: 순수 소프트웨어/시스템 실측이며 채널코딩 HW 설계 없음, 부호는 대칭 이진 대신 파일 청크 XOR 조합(소거정정 전용, LLR/soft decision 개념 없음), 접근제어·명명공간 관리 등 미구현.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 분산 스토리지용 소규모 LDPC 소거부호 구성(Monte-Carlo 그래프 탐색) | 대응 없음 | binary QC-LDPC 채널코딩 H-matrix 설계와 목적(소거정정 vs 오류정정)·구성법이 상이하여 대응 모듈 없음 |
| greedy 디코딩(알려진 청크로 미지 청크 XOR 복원) | 대응 없음 | min-sum 반복복호가 아닌 단순 XOR 기반 소거정정으로 `decoder.cpp`와 알고리즘적 접점 없음 |
| P2P 시스템 아키텍처(gossip, HTTP 전송, host list) | 대응 없음 | 시스템/네트워크 계층 설계로 ECC 코드베이스와 무관 |

> 적용 가치: 낮음 — 파일 시스템 레벨의 소거부호(erasure code) 및 분산 스토리지 아키텍처 논문으로, NAND 바이너리 QC-LDPC 채널코딩(부호설계/디코더/HW)과는 문제 영역과 부호 구성 방식이 근본적으로 달라 이식할 요소가 없음.
