### B. 알고리즘 요약

1. 배경: Feldman et al.의 LP(linear programming) decoding은 codeword polytope을 완화한 relaxed polytope `P` 위에서 ML 근사 최적화를 수행.
2. 문제: LP 디코더는 fractional pseudocodeword가 ML codeword보다 높은 likelihood를 가지면 실패하며, 유한 블록길이에서 성능 예측이 어려움.
3. 가정: `(dc, dv)`-regular bipartite graph가 `(α, δ)`-expander라는 확장성(expansion) 가정 하에 분석.
4. 핵심 결과(Prop.1): expander code의 모든 pseudocodeword는 fractional support(비정수 비트 수)가 블록길이에 선형(`|Vfrac| ≥ αn`)으로 큼.
5. 핵심 정리(Thm.1): codeword의 active set 크기 `|A(xcw)| = γcw·n`이 fractional pseudocodeword의 active set `|A(xpc)| ≤ γpc·n` 보다 항상 크다(`γpc < γcw`) — 즉 codeword가 더 많은 facet에 인접.
6. 핵심 기법(facet-guessing): LP 디코딩 실패 시(fractional 결과) facet `Fi`를 하나 골라 `P' = P ∩ Fi`로 축소한 뒤 재차 LP를 풀어 최적 vertex를 얻는 후처리 반복.
7. 두 변형: 전수 탐색(EFG, `N=|A\Axpc|`, 다항시간)과 무작위 샘플링(RFG, `N=20` 등 상수회, LP 디코더와 동일 복잡도 오더).
8. 학습/최적화 절차 없음 (이론적 확률 하한 `p_RFG ≥ (γcw−C1γpc)/(2^(dc-1)(1-R)+2)` 유도).
9. 결과: Tanner의 group-structured (155,64) 코드와 random (3,4) n=200 코드에서 RFG(N=20)가 sum-product 대비 고SNR에서 최대 0.5dB 이득.
10. 한계: 짧은/중간 블록길이 실험뿐, HW 미설계, LP 솔버 반복 호출이 필요해 실시간 NAND ECC 디코더 구조와 거리가 멂.

| pseudo-codeword 분석(expander code) | 대응 없음 | 이론적 다면체 기하 분석으로 HW 디코더 이터레이션 루프와 무관 |

> 적용 가치: 낮음 — LP decoding 프레임워크 전용 후처리 알고리즘으로, min-sum 기반 Prime ECC 디코더 구조에 이식할 수 있는 구체적 모듈이 없음.
