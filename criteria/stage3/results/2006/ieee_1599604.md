### B. 알고리즘 요약

1. 대상은 memoryless compound channel(예: unknown transition probability를 갖는 BSC, 위상 오프셋을 모르는 complex AWGN)에서 regular LDPC 코드(예: (3,6), (4,8), rate≈0.5)의 반복 복호.
2. 풀려는 문제: 채널 파라미터(위상/전이확률)를 모르는 상태에서 LDPC 복호와 채널 추정을 동시에 수행해야 하고, 기존 방법들은 엄밀한 성능 분석(density evolution)이 어려움.
3. 핵심 가정: 채널 파라미터 노드를 다수의 비협력(noncooperative) estimator node로 분할해 tree property를 강제로 성립시켜 density evolution 분석을 가능하게 함.
4. 핵심 기법: variable-check 2-layer Tanner graph에 estimator node 계층을 추가한 3-layer 그래프 정의, variable-to-estimator/estimator-to-variable 메시지 교환을 좌-좌-우-우 병렬 스케줄로 반복.
5. 핵심 식: estimator 출력 메시지는 ML 기준 `η* = argmax_η f(m|η)`(식 4)로 각 iteration마다 재설계되며, 이는 density evolution으로 계산된 입력 밀도함수에 맞춰 iteration-varying transfer function을 사전 설계하기 위함.
6. 확장: compound BSC에서는 KT(Krichevski-Trofimov) 추정기 기반 하드/소프트 확률 추정식(식 3) 적용, unknown-phase AWGN에서는 QPSK I/Q 실수 variable node 쌍을 하나의 estimator node에 연결하는 4-layer 구조로 확장.
7. 구현 디테일: 대칭성(symmetry) 조건 하 all-zero codeword 가정이 깨지는 채널에서는 Friedlander-Burshtein 기법을 빌려 1-type/0-type 두 개의 조건부 밀도를 별도 추적.
8. 최적화 절차: density evolution으로 각 iteration의 입력 density를 추적하고, 이를 기반으로 estimator transfer function을 ML criterion으로 사전 계산(오프라인 설계, GA/NN 없음).
9. 결과: unknown-phase AWGN + QPSK에서 제안 알고리즘은 위상 오프셋 50도까지 안정적 복호 가능(기존 EM 기반 [12] 대비 강건), rate≈0.5 규칙 LDPC 코드 기준.
10. 한계: HW 미설계, 시뮬레이션/density-evolution 이론 분석뿐, 위상불확실성·BSC 전이확률불확실성 등 무선 특화 채널 모델 전제로 NAND flash 채널과 직접 관련 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Estimator node 계층을 통한 joint channel estimation + LDPC 복호 | 대응 없음 | Prime ECC는 채널(RBER/fixed-error)을 사전 정의된 파라미터로 다루며 반복 복호 중 온라인 채널 파라미터 추정 메커니즘이 없음 |
| Density evolution 기반 estimator transfer function 설계 | 대응 없음 | Prime ECC 코드베이스에 density evolution 도구 없음(§3 Min-Sum 등 고정 알고리즘) |
| Compound BSC/unknown-phase AWGN 특화 알고리즘 | 대응 없음 | 무선통신 위상 추정 문제로 NAND read channel 모델과 무관 |

> 적용 가치: 낮음 — NAND 채널은 위상 불확실성이나 미지의 전이확률 문제가 없는 잘 특성화된 채널이라 joint channel-estimation 프레임워크의 필요성이 낮고, decoder/HW 개선 기여도 없음.
