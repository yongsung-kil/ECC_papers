### B. 알고리즘 요약

1. LDPC 부호와 채널별 coded modulator(CM)를 결합한 LDPC-coded modulation 수신기를 block A(CM-SISO+VND)와 block B(CND)로 분해하여 EXIT chart로 분석한다.
2. 기존 EXIT 해석은 MI가 1에 도달하는 지점만으로 수렴을 판정하나, 유한 반복에서 실제 BER과의 관계가 불명확하다는 문제를 다룬다.
3. 이진 등확률 전송을 가정하고, MI `I(X;Y)`와 BER `Pe`의 관계를 Fano 하한과 MAP 상한(정리 1: `Pe ≤ H^{-1}(1-I)`)으로 각각 유도한다.
4. Block A의 EXIT 함수 `IA`를 degree distribution `{λi}`의 선형결합으로, block B의 `IB`를 `{ρj}`의 선형결합으로 표현하고 점(1,1) 근방에서 1차 Taylor 근사를 구한다.
5. 이 근사에 BEC information-combining 함수(`I_V^(i)(IB,IS)=1-(1-IS)(1-IB)^{i-1}` 등)를 대입해 수렴 조건 `λ1(2∂IS/∂IV)+λ2(1-IS) < 1/Σj(j-1)ρj` (식 14)를 유도한다.
6. 이 부등식은 기존 AWGN 채널용 안정성 조건(Richardson-Shokrollahi-Urbanke)을 CM/채널 특성이 반영된 형태로 일반화한 것이다.
7. BPSK/AWGN, DE-QPSK, ISI 채널(4PSK) 세 가지 예제로 degree-1(`λ1`), degree-2(`λ2`) 계수 허용범위가 채널 특성(`IS(1)`)에 따라 달라짐을 보인다.
8. 별도의 최적화 알고리즘(GA/DE)은 사용하지 않고, 유도한 부등식 자체를 설계 제약으로 제시한다.
9. 목표 BER과 반복 횟수 관계식(식 17, `BER_{n+n0} ≤ (ab)^n H_{n0}^2`)을 유도해, 주어진 반복 수 내 목표 BER 달성을 위한 degree distribution 설계 부등식(식 19 근방)을 제시한다.
10. 코드 그래프의 cycle은 고려하지 않은 asymptotic 해석이며, 실제 HW 구현이나 유한장 코드 시뮬레이션 검증은 없다.
