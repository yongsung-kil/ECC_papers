### B. 알고리즘 요약 (10줄 내외)

1. 부호 스펙: irregular LDPC, block length `N=64800`, rate 1/4~9/10(11개 rate), QPSK/8-PSK/16-APSK/32-APSK 변조와 결합해 DVB-S2 표준화.
2. 풀려는 문제: partially-parallel LDPC 디코더 HW에서 `b`개 물리 노드가 논리 노드를 재사용 처리할 때, edge 정보가 메모리에 무작위 분산되어 있어 충돌 없는 동시 접근이 어려움(기존 해법은 다중 소형 메모리 블록 분할 → 면적 비효율).
3. 핵심 가정/모델: parity-check matrix `H=[H1 H2]` 형태, `H2`는 staircase 하삼각(각 행이 이전 행과 한 칸씩 시프트된 이중대각 구조).
4. 핵심 기법: `b`개 bit node로 이뤄진 그룹마다 첫 bit node에 연결된 check node 인덱스 집합 `{i1,...,idv}`를 정하면, `j`번째(`0≤j<b`) bit node의 연결은 `(i_k + j·q) mod (M-g0)`로 순환 이동(`q=(M-g0)/b`)해 결정 — 이렇게 하면 그룹 전체 edge가 RAM의 정해진 행들에 함께 배치됨.
5. 핵심 식 의미: 이 순환-시프트 규칙 덕분에 check-node 처리 시 top/bottom edge-RAM에서 각각 `b`개 연속 행만 fetch하면 그룹 전체 edge를 한 클록에 읽을 수 있어, "구조화된 배치 + 유사난수성 유지"를 동시에 달성.
6. 확장: `H2`(staircase) 서브행렬 덕분에 별도 generator matrix 없이 `Hx^T=0`을 재귀적으로 풀어 인코딩 가능(quadratic 인코딩 복잡도 회피), 동시에 degree-2 bit node만으로 구성된 short cycle을 원천 차단해 성능 개선.
7. 부수 트릭: check node 그룹 첫 노드의 free constant `{i1,...,idv}`를 cycle-4 free, cycle-6 최소화 조건으로 탐색 선택; edge RAM을 top(연결 degree>2)/bottom(degree≤2)으로 분리해 저장.
8. 학습/최적화 절차: 명시적 DE/GA는 아니고, free constant 탐색(search)으로 짧은 사이클 최소화; degree distribution은 Richardson-Shokrollahi-Urbanke 방식(참고문헌) 활용.
9. 결과: `b=360` 예시 기준 throughput `R = b·f_clk/(q_a·d̄)` 관계식 제시(예: 100Mb/s급), Shannon 한계 대비 0.7~0.8dB 이내, DVB-S1(RS+콘볼루션) 대비 35% 이상 용량 개선, ROM table 면적 <0.1mm², 전체 실리콘 <10mm².
10. 한계: ASIC 합성/실측 결과 없이 이론적 면적/속도 추정치만 제시, 특정 `N=64800`(및 축소판 16200) 길이·DVB 변조 방식에 특화, 순수 binary AWGN 채널 가정으로 NAND 채널(비대칭 오류, RBER) 특성은 미반영.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| H-matrix 구조 설계(그룹 순환-시프트, staircase 하삼각) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 QC-LDPC(circulant shift) 고정 구조라 이 논문의 그룹-순환 규칙과 유사한 원리이나 재검증 부담 큼 |
| 충돌 없는 병렬 edge-RAM 배치(top/bottom RAM 분리) | 디코더 HW 모델 (z=32, HCU, GT) | Prime ECC도 `z_sb=32` row 단위 병렬 처리로 메모리 접근을 구조화하고 있어 설계 철학은 유사, 구체 매핑은 상이 |
| Quadratic-complexity-free 인코딩(staircase 재귀 해법) | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | Prime ECC도 dual-diagonal 방식으로 유사한 목표(저복잡도 인코딩) 달성, 대체 구조로서 참고 가치 |
| Degree-2 bit node 배치로 short cycle 차단 | `ecc_top.cpp` `Load_PCM()` | H-matrix 설계 시 이식 가능한 설계 규칙이나 Prime ECC의 특정 QC-LDPC 구조 재설계 필요(난이도 높음) |
```

적용 가치: `중간` — DVB-S2 표준급 부호설계 원조 논문으로 메모리 충돌 회피와 인코딩 단순화 아이디어는 유용하나, Prime ECC는 이미 QC-LDPC circulant 구조로 다른 방식의 병렬화를 달성하고 있어 전면 이식보다는 설계 철학 참고 수준.
