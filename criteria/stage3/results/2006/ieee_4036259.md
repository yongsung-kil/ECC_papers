### B. 알고리즘 요약

1. 대상은 IEEE 802.16e/802.11n 스타일 structured LDPC(QC-LDPC): base matrix `Hb`(`mb×nb`)를 `z`배 확장(`z×z` shifted identity/all-zero 블록)해 만든 `H`(`M×N`, `M=mb·z`, `N=nb·z`).
2. 표준 SBP(belief propagation)는 구조를 활용하지 못해 고속화 여지가 제한적이라는 문제의식에서, GRA(Generalized Repeat-Accumulate) 코드처럼 처리 가능함을 보이려 함.
3. 채널은 BPSK/QPSK-modulated AWGN, 표준 LDPC 코드(rate 1/2, 2/3, 3/4) 그대로 사용.
4. `H = [H_GRA | H_p2]`로 분해(`식 (4)`), 비-dual-diagonal 열(parity bit `x6`)을 먼저 계산 후 나머지 parity bit는 repetition-SPC-ACC(반복→인터리브→single-parity-check→accumulator) 구조로 back-substitution.
5. 핵심 파라미터 `Q`(열별 반복횟수), `J`(행별 SPC 입력수), 인터리버 `ρ`(`식 (5)`, 길이 `W`=`H_GRA`의 1의 개수)로 GRA-like 인코더/디코더를 완전히 정의.
6. 디코더는 두 constituent decoder(반복 디코더 + trellis 기반 SPC-ACC 디코더)가 extrinsic LLR을 (역)인터리버로 교환하는 turbo-like 반복 구조(TLD).
7. structured LDPC 확장 시 `z`개의 base-matrix GRA 인코더/디코더가 vector interleaver(shift 값 `R_bm` + 열치환 `ρ`)로 연결되며, 이는 contention-free(CF) inter-window shuffle 인터리버로 재해석되어(`식 (8),(9)`) `z`-way 병렬 프로세서에서 메모리 충돌 없이 동작.
8. 별도 학습/최적화 절차 없음(구조적 유도, 기존 표준 코드에 적용).
9. IEEE 802.16e N=2304(z=96) 코드로 20회 반복 FER 비교 시 저rate(1/2)에서 TLD가 SBP를 능가하며, rate가 오를수록(2/3, 3/4) trellis puncturing이 늘어 이득이 감소.
10. HW 구현이나 gatecount/throughput 수치는 제시하지 않으며(순수 알고리즘/FER 시뮬), min-sum 계열 디코더와의 직접 비교도 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| structured LDPC(base matrix + shift, QC-LDPC) 구조 | `ecc_top.cpp` `Load_PCM()` | Prime ECC의 `PCM_b`(base+shift) 구조와 개념적으로 동일 |
| turbo-like decoder(SPC-ACC trellis 기반 반복 디코딩) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 기존 min-sum CN 연산과 다른 대체 디코딩 방식으로, 도입 시 CNU 로직 자체 교체 필요 |
| contention-free 인터리버(병렬 메모리 접근) | 디코더 HW 모델(z=32 병렬, HCU) | 병렬 처리 시 메모리 충돌 회피 개념은 Prime ECC의 row 단위 병렬 구조와 유사하나 직접 대응 모듈 없음 |
| GRA-like 인코딩(dual-diagonal 기반) | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | dual-diagonal 인코딩 개념은 유사하나 partial dual-diagonal 처리 방식은 상이 |

적용 가치: 중간 — QC-LDPC 구조 자체는 Prime ECC와 유사하지만, 제안된 turbo-like(SPC-ACC trellis) 디코딩은 기존 min-sum 기반 CNU를 대체하는 별개 알고리즘이라 이식에는 상당한 재설계가 필요하고, 고rate(NAND급) 영역에서는 이득이 줄어드는 한계가 있음.
