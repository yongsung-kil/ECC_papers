### B. 알고리즘 요약

1. Tanner 코드는 LDPC(단순 SPC constraint node 다수)와 turbo 코드(복잡한 convolutional constraint node 소수)의 중간 스펙트럼을 아우르는 일반화된 그래프 부호이며, 본 논문은 저rate(R≤1/2) 영역에서 낮은 error-floor와 빠른 복호구조를 동시에 갖는 quasi-cyclic Tanner 코드 설계를 다룬다.
2. 기존 LDPC는 저rate에서 좋은 threshold와 낮은 floor를 동시에 얻기 어렵고, turbo 코드는 parallel형은 floor가 높고 serial형은 threshold가 나쁘다는 문제를 해결하고자 한다.
3. Adjacency 행렬 `Γ`를 block-circulant(순환치환행렬 `q×q` 블록)로 구성해 quasi-cyclic 구조를 만들고, 각 constraint node의 component code(SPC 또는 Hamming)의 parity-check 행렬을 전개해 최종 H를 얻는다.
4. 핵심 기법은 "code doping": 프로토그래프의 일부 SPC 노드를 shortened (6,3) Hamming 노드로 치환(HD-LDPCC)해 ARA(accumulate-repeat-accumulate) 프로토그래프 기반 rate-1/6 Tanner code I/II를 구성한다.
5. Doping으로 threshold는 소폭 열화되나(Tanner code I: -0.3dB → Tanner code II: -0.33dB, 부가 edge로 추가 열화) error-floor가 크게 낮아지는 트레이드오프가 핵심 관찰이다.
6. 복호는 constraint node별로 BCJR 트렐리스(Hamming/accumulator 노드) 기반 soft-output 계산과 표준 log-domain sum-product(SPA, H-matrix 전개 시)를 혼용한다.
7. Puncturing으로 프로토그래프 노드(S0, P4 등)를 제거해 rate 1/6→1/5→1/4→1/3 등으로 다양화하며, 이때 density evolution으로 threshold를 재계산한다(Table I, II).
8. 학습/최적화 절차는 프로토그래프 수준 density evolution을 통한 threshold 계산뿐이며, 자동 최적화(GA/DE) 없이 수작업 doping 설계이다.
9. 결과: (2044,1022) rate-1/2 코드는 FER 5×10^-8까지 floor 없음(random coding bound 대비 1dB 이내), rate-1/6 Tanner code II는 SPA 대비 BCJR 손실 0.1dB, rate-1/5 Tanner code I은 CCSDS rate-1/6 turbo code와 유사 성능(FER=10^-3에서 random bound 대비 0.5dB 이내)이나 자체 floor 존재.
10. 한계: 소프트웨어 디코더가 3GHz Pentium IV에서 약 200kbps 처리(순수 알고리즘 시뮬 수준), HW 합성/FPGA 검증 없음, 저rate(1/6~1/2)·큰 Imax(최대 200)로 NAND 고rate ECC와는 반복수·복잡도 특성이 상이함.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Doping 기반 Tanner 부호설계(SPC→Hamming 치환) | `ecc_top.cpp` `Load_PCM()` | 특정 protograph(ARA) 기반 비-LDPC Tanner 구조로 이식 난이도 높고, Prime ECC는 순수 SPC 기반 binary QC-LDPC 전용 |
| Hamming node BCJR soft-decoding | 대응 없음 | Prime ECC decoder.cpp는 min-sum CN 연산만 지원, component-code(Hamming) 트렐리스 디코더 미보유 |
| SPA(sum-product) 기반 H-matrix 디코딩 | `decoder.cpp` `LDPC_Decoder()` | 개념적으로 유사하나 본 논문은 tanh 기반 full SPA이며 Prime ECC는 min-sum 근사 사용, doping node 특유 구조 재현 불가 |

> 적용 가치: 낮음 — 저rate Tanner 코드(SPC+Hamming 혼합) 설계 기법으로 NAND 고rate binary QC-LDPC 구조 및 Prime ECC의 순수 min-sum 디코더와 근본적으로 다른 계열.
