> 총평: IEEE 802.16e 다중 rate/블록사이즈를 파라미터화된 router/reverse-router로 지원하는 layered BP 디코더 HW 구조 논문. Prime ECC는 이미 유사한 fixed QC-LDPC 파이프라인·HCU·GT를 보유해 재구성 라우팅 아이디어의 이식가치는 제한적.

### B. 알고리즘 요약

1. IEEE 802.16e 표준 QC-LDPC(rate 1/2, 2/3A, 2/3B, 3/4A, 3/4B; 19종 block size Z=24~96; nb=24, nr=15)를 대상으로 layered BP 디코더 설계.
2. 기존 [1,2]의 column-grouping 방식은 특정 H-matrix 1개에만 최적화되어 다중 rate/블록사이즈(95 조합) 지원 시 재설계 부담이 큼이 문제.
3. layered(row-serial) BP를 사용하며 `L(qmj) = L(qj) - Rmj`, check node에서 `Amj = sum φ(L(qmn))`, `Rmj = -Smj·φ(Amj)`로 갱신(식 2~6).
4. router/reverse router를 shuffle network - alignment bank - distribution network 3단으로 분리하여 컬럼 그룹핑 없이 임의 shift value/블록 배치를 지원.
5. `φ(·)` 함수는 자기역함수 성질(`φ(φ(x))=x`) 이용해 sign-magnitude 변환 후 LUT 크기를 절반으로 축소.
6. Configuration ROM에 code type, expansion factor(Z/4), block row number를 입력하면 shift value와 row weight를 출력, 파라미터만으로 95개 조합 전환.
7. R memory·column sum memory를 dual bank two-port 구조로 두어 동시 read/write, alignment/reverse alignment는 mux/shifter/register 5요소로 구성.
8. 학습/최적화 절차 없음(고정 아키텍처 설계).
9. TI 90nm ASIC 합성 결과 parallel factor 12, 8bit(5:3) 정밀도, 100MHz에서 15 iteration 기준 최대 throughput 256Mbps; 100MHz는 80MHz 대비 gatecount 5.6%, 동적전력 14.8% 증가.
10. 정정성능(BER/error-floor) 개선은 다루지 않고 순수 HW 아키텍처/재설계 용이성에 집중, 총 gatecount 절대값은 본문에 미기재(percent breakdown만 표1 제공).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| layered BP 이터레이션 루프 (식 2~6) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 동일 계열 min-sum/φ 기반 layered 알고리즘, Prime ECC는 이미 min-sum으로 구현되어 φ-LUT 방식은 참고용 |
| CN 갱신 (Amj, Smj, Rmj) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | 유사 연산 구조, 이미 min-sum 근사로 대체 보유 |
| router/reverse router 재구성 라우팅 | 대응 없음 | Prime ECC는 z_sb=32 고정 QC 구조+HCU 기반, 임의 다중 H-matrix 지원 목적의 range router는 대응 모듈 없음 |
| Configuration ROM (다중 표준 파라미터화) | `ecc_top.cpp` `Load_PCM()` | 부분 대응; Prime ECC는 단일/소수 H-matrix 로드 방식으로 95조합급 파라미터화는 없음 |
| dual-bank column sum memory | 대응 없음 | Prime ECC 메모리 뱅크 구조는 본 프로파일에 상세 기재 없음 |

적용 가치: **낮음** — 정정성능 개선이 없고, 다중 표준/rate 지원용 범용 라우팅 구조는 Prime ECC의 고정 QC-LDPC 특화 구조(HCU/GT)와 목적이 달라 이식 실익이 적음.
