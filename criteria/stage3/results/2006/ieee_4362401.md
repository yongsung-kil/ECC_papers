### B. 알고리즘 요약

1. 시스템: rate 1/2 mother code(n=100000, 랜덤 구성, 4-cycle 회피)를 puncturing하여 rate 2/3 부호 생성, BSC/AWGNC/RCISI 세 채널 대상.
2. 문제: 기존 puncturing distribution 최적화는 단일 채널(주로 AWGNC) 기준이라, 실제 이동통신처럼 채널이 바뀌면 최적성이 깨질 수 있음.
3. 모델: 각 채널의 LLR 확률밀도함수 `f_ℓ(y)`를 puncturing fraction `π^(0)_ℓ`을 반영해 정의(식 4~6, AWGNC/BSC/RCISI 각각).
4. 핵심 기법: generalized check node degree distribution `p_d`(식 1)와 puncturing distribution `π^(0)(x)`(식 2)를 이용한 generalized discretized density evolution으로 threshold `σ_A`, `σ_R`, `ε_B` 계산.
5. 핵심 식: check-node 업데이트 연산자 `T(a,b)`(식 8, quantized tanh-rule)와 결합 연산 `F(p_v1,p_v2)`(식 9)로 반복적으로 확률질량함수를 합성해 검사노드 출력 계산.
6. 확장: 서로 다른 variable node degree별 확률질량함수를 평균화하지 않고 개별 유지(`p^(j)_c,ℓ`)하여 정밀도 향상.
7. 구현 디테일: LLR을 구간 `[-B,+B]`로 제한하고 양자화 간격 `Δ`로 quantize(식 7 부근)하여 discretized density evolution 수행.
8. 최적화 절차: 제약조건(총 puncture 비율 유지, 이전 분포와 근접, threshold 개선) 하에 puncturing distribution `π^(0)(x)`를 반복 갱신하는 최적화 알고리즘([9] 재사용).
9. 결과: BSC 최적 분포 사용 시 BER 10⁻⁴에서 AWGNC 최적 대비 Δε≈0.04, RCISI 최적 대비 Δε≈0.06 이득. AWGNC에서는 BSC 대비 0.18dB, RCISI에서는 AWGNC 대비 0.1dB 등 채널별 상이한 이득 확인.
10. 한계: 무한 길이 코드 가정(cycle-free) 하의 asymptotic 이론/시뮬레이션뿐이며 HW 설계 없음, NAND read channel(soft quantization 등)과 무관한 이동통신 채널 모델 사용.
