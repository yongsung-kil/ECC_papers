### B. 알고리즘 요약

1. (λ,ρ)-regular bipartite 그래프(변수노드 n개, 체크노드 r개, edge N=nλ=rρ)를 대상으로, 각 edge의 우측 라벨을 2차 다항식 `f(x)=f1x+f2x^2 (mod N)`으로 정의하는 QC-LDPC 구성법.
2. 기존 랜덤/PEG 구성법은 그래프가 비구조적(unstructured)이라 구현 복잡도가 크고, 대수적 구성법(Margulis 등)은 최소거리 상한이 작다는 한계를 해결하려 함.
3. 채널모델은 AWGN + BPSK, iterative sum-product 복호(최대 80회) 가정.
4. 핵심 기법은 `f(x)`가 permutation polynomial이 되는 필요충분조건(N의 소인수 분해와 f2의 관계, Proposition 1)을 이용해 코드를 `(n,k,λ,ρ,f1,f2,N)` 소수 파라미터만으로 유일하게 정의.
5. `f2`를 N의 서로 다른 소인수의 곱(f2min)으로 설정하면 그래프 automorphism 등가류 수가 최소화되어 girth가 최대화된다는 설계 규칙(Theorem 1, Lemma 1/2) 도출.
6. 확장으로 그래프 isomorphism(Proposition 2,3)을 이용해 `f1` 탐색범위를 `1~2*f2*α`로 축소, 탐색 효율화.
7. 부수 절차로 NNCS 방법을 이용한 최소 해밍거리 상한 알고리즘 계산 및 parity-check matrix rank 확인(코드 VIII은 rank-deficient로 rate가 1/2에서 벗어남).
8. 최적화 절차: `(λ,ρ)`와 `(n,k)` 선택 → `f2` 초기값 설정 → girth 최대화하는 `f1` 탐색 → `f2` 확장 반복(6단계 그리디 탐색 절차, 일반 PC에서 신속 수행).
9. 결과: girth 최대 12 달성, (1008,504) (3,6)-regular 코드가 동일 크기 girth-8 PEG 코드와 동등한 BER/FER 성능(10^-7까지 error-floor 미관측).
10. 한계: 순수 시뮬레이션(이론+시뮬)이며 인코더/디코더 HW 구현 이점은 논문에서 다루지 않음, quasi-cyclic 구조의 실제 HW 이식 성능은 미검증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QPP 기반 QC-LDPC H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | H-matrix 생성 자체를 대체하는 부호설계로 기존 고정 QC-LDPC를 대체해야 함 (이식 난이도 높음) |
| Girth 최대화를 통한 error-floor 개선 | 대응 없음 | Prime ECC 디코더 알고리즘이 아닌 부호설계 단계 기여이므로 코드 레벨 대응 모듈 없음 |
| Quasi-cyclic circulant 구조 | `ecc_data.h` `PCM_b` (base+shift) | Prime ECC도 base+circulant shift 구조이므로 개념적 정합 가능성 있음 |

> 적용 가치: **중간** — Prime ECC가 이미 QC-LDPC 구조이므로 QPP 기반 구성법 자체는 개념적으로 이질적이지 않으나, 실제 도입은 H-matrix 전면 재설계와 재검증이 필요해 난이도가 높음.
