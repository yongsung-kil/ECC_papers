### B. 알고리즘 요약

1. QC-LDPC 부호(base matrix + square permutation 확장)로 irregular rate 1/2, regular 5/8, regular 7/8 세 코드율을 하나의 9k-bit FPGA 디코더로 지원.
2. 기존 base matrix 확장 기법(Zhang-Parhi 등)은 확장행렬 girth가 base matrix girth 이상임만 보장하고, 특정 girth 목표값 달성은 보장하지 못함.
3. base matrix의 length-`2g` cycle이 순환이동값(`p`)의 수직/수평 shifted-value drop 합이 `0 mod L`일 때만 확장 후에도 동일 길이 cycle을 만든다는 lemma를 증명(L=permutation 크기).
4. 이 lemma를 바탕으로 cycle elimination(CE) 알고리즘: 각 1의 위치에 제약조건 리스트를 만들고, cycle을 발견할 때마다 순환이동값 `p`를 증가시켜 제약을 만족시키되 값 증가가 최소인 원소만 채택.
5. CE로 구성한 코드는 rate 1/2, 5/8, 7/8에서 평균 girth 10, 10, 8 달성 (기존 ZP 방식은 8.06, 8.25, 5.91).
6. check-node 비선형 함수(`log(1+e^-|x|)`) 근사에 LUT 대신 min-sum with correction(MSC) 방식을 채택, irregular/high-rate 코드에서 LUT 대비 손실을 줄임(양자화 6:1 사용).
7. 디코더는 partially-parallel 구조(VNU/CNU + configurable router + memory bank), 두 개 FPGA 핀으로 세 모드 전환.
8. 최적화 절차: PEG(progressive edge growth)로 irregular 코드 변수노드 차수분포 결정 후 CE로 girth 최적화.
9. Xilinx XC2V8000 FPGA, 100MHz, rate 5/8·7/8에서 40Mbps(24회 반복), rate 1/2에서 15Mbps(60회 반복); CE 코드는 block error rate 10^-7 @ SNR 1.8dB, ZP 코드 대비 error floor 유의미하게 억제.
10. CE-구성 코드와 ZP-구성 코드의 코딩이득 차이는 0.1dB 이내로 성능저하 없이 error floor만 개선; 다만 short cycle 제거가 stopping set 기반 error floor의 근본해결책은 아님(저자 언급, 향후 과제).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Cycle Elimination(CE) 알고리즘, base matrix girth 최적화 | `ecc_top.cpp` `Load_PCM()` | H-matrix/부호구조 로드 단계에 해당하나 CE 자체는 오프라인 부호 생성 도구라 코드 내 직접 대응 모듈 없음 |
| Min-Sum with Correction(MSC) check-node 근사 | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC는 이미 min-sum 기반이라 MSC 보정항 추가 여부 검토 가능 |
| 양자화(6:1 fixed-point) | `channel.cpp` `Set_LLR_Th()` | soft-read LLR 양자화 설계와 관련 |
| Configurable VNU/CNU router (multi-rate 지원) | 대응 없음 | Prime ECC는 특정 QC-LDPC 고정 구조로 멀티레이트 라우터 개념 없음 |
| Partially-parallel 구조, memory bank | 디코더 HW 모델(z=32, HCU, GT) | 병렬화/메모리 구조 철학은 유사하나 세부 대응은 낮음 |

적용 가치: `낮음` - 핵심 기여인 CE 알고리즘은 부호설계(base matrix 구성) 단계의 오프라인 기법으로 Prime ECC의 고정 QC-LDPC 구조에 이식하려면 H-matrix 재설계가 필요해 이식 난이도가 높고, 나머지 MSC/양자화 기법은 이미 Prime ECC가 유사 기법을 보유.
