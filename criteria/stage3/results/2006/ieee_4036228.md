### B. 알고리즘 요약

1. protograph LDPC 코드는 작은 base graph를 N배 copy-and-permute하여 얻어지는 LDPC 앙상블이다.
2. 목표는 protograph 앙상블의 유한/점근 weight enumerator `A_d`를 계산해 minimum distance가 block size에 선형으로 증가하는 조건을 규명하는 것.
3. 각 variable/check node를 constituent code로 보고 serial concatenation/turbo-like code 이론의 uniform interleaver 기법을 적용해 `A_d`를 계산한다 (`식 (1)-(3)`).
4. degree-3 check node의 vector weight enumerator는 multinomial theorem으로 닫힌 형태 `A_{w1,w2,w3}`로 유도되며(`식 (4)`), 고차 degree check는 이를 concatenation으로 확장한다(`식 (8)`).
5. 점근적으로는 정규화 로그 weight distribution `r(δ)`를 정의하고, 첫 zero-crossing `δmin`을 typical minimum distance ratio로 정의한다.
6. `r(δ)`로부터 ML decoding의 최소 Eb/No 임계값 상한(`식 (9)`)을 계산하며, 이를 iterative decoding threshold `γiter`와 비교한다.
7. Precoding(accumulator를 degree-1 variable node에 삽입)이 rate를 바꾸지 않으면서 `δmin`과 threshold를 개선함을 예제로 검증(예: (3,6) regular LDPC `δmin=0.023→0.033`, RJA→ARJA `δmin=0.013→0.015`).
8. 결과적으로 unstructured irregular LDPC의 선형 minimum distance 충분조건(`λ'(0)ρ'(1)<1`)이 protograph 코드에는 필요조건이 아님을 보임(Example 4, `δmin=0.004`로도 양수 유지).
9. 모든 결과는 시뮬레이션 없이 해석적 유도로 rate 1/2 예제(3,6 regular, RJA/ARJA) 중심으로 검증됨.
10. HW 구현이나 디코더 알고리즘 개선은 다루지 않으며, 순수 부호 앙상블의 거리 스펙트럼 이론에 국한됨.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| protograph 부호 구성/weight enumerator 이론 | `ecc_top.cpp` `Load_PCM()` | H-matrix 자체가 아닌 앙상블 이론이라 직접 대응 약함 |
| precoding(accumulator)을 통한 minimum distance 개선 | 대응 없음 | Prime ECC는 QC-LDPC 고정 구조, accumulator 기반 precoding 미지원 |
| 디코더/HW 관련 내용 | 대응 없음 | 논문에 디코더 알고리즘·HW 서술 없음 |
