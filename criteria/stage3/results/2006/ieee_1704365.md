### B. 알고리즘 요약

1. 시스템: longitudinal magnetic recording 채널, PR4 target으로 등화된 ISI 채널, transition jitter(media noise) 포함, 코드화 시스템에서는 `(1107,986,123,3,27,0.89)` LDPC를 outer code로 사용.
2. 문제: 기존 BP(belief propagation) 채널 검출기는 상관된(correlated) media noise를 고려하지 못하고, 기존 NPBP(noise predictive BP)도 데이터 패턴별 노이즈 상관구조 차이를 반영하지 못함.
3. 채널/노이즈 모델: transition response는 Lorentzian pulse, jitter는 truncated Gaussian 랜덤 시프트로 모델링되며 noise는 AWGN + data-dependent media noise filter를 통과한 성분으로 구성. media noise 수준이 높을수록 패턴별 노이즈 상관구조가 크게 달라짐.
4. 핵심 기법: factor graph 기반 BP의 function node에서 계산되는 a posteriori probability에, 각 데이터 패턴 `s`마다 별도의 FIR 예측필터 `p_s`와 예측오차분산 `sigma_s^2`을 적용하는 Pattern-Dependent Noise Predictive BP(PD-NPBP).
5. 핵심 식 의미: 패턴별 예측필터로 추정한 과거 노이즈 샘플을 이용해 채널출력을 백색화(whitening)한 뒤(식 (3)) 이를 a posteriori 확률 계산(식 (4))에 반영, 패턴 의존적 노이즈 상관을 반복적으로 제거.
6. 확장: inner iteration(검출기 내부 반복, 20회)과 outer iteration(검출기-LDPC 디코더 간 soft 정보 교환)을 구분하여 수행.
7. 구현 디테일: 패턴 길이 `L=3`, 예측필터 길이는 과거/미래 2, 0, 2개 탭(총 8종 패턴별 필터) 사용.
8. 최적화 절차: average mutual information(AMI) 기반 수렴 특성 분석으로 PD-NPBP가 MAP/BP/NPBP 대비 더 높은 mutual information을 달성함을 확인 (별도 학습 알고리즘은 없음).
9. 결과: 9% jitter 조건에서 PD-NPBP가 NPBP 대비 약 0.35dB, BP(무 NP) 대비 약 1.25dB 성능 개선. LDPC outer code 결합 시 PD-NPBP가 가장 큰 코딩이득 제공.
10. 한계: 시뮬레이션만 수행(HW 미설계), 자기기록 채널의 특정 media noise 모델(Lorentzian pulse/jitter)에 국한되며 LDPC 자체의 알고리즘 개선은 다루지 않음.
