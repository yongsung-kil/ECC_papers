### B. 알고리즘 요약

1. 시스템: 광/자기 저장장치의 Bliss scheme(RLL 부호와 ECC 결합 순서를 바꿔 error propagation 방지) 기반, LDPC 부호(rate 0.9~0.95, length 6000 예제)를 scaled min-sum으로 복호.
2. 문제의식: Bliss scheme에서는 SISO RLL 디코더의 error propagation으로 인해 LDPC 코드워드의 parity부가 systematic부보다 BER이 높아, 기존 density evolution(균일 채널 가정)을 그대로 적용할 수 없음.
3. 핵심 가정: systematic/parity 두 부분이 서로 다른 분산 `σ1²≤σ2²`의 독립 biAWGN 채널을 통과한다고 가정, 두 채널 간 차이를 error propagation factor `s_EPF=Q(1/σ2)/Q(1/σ1)`로 정의.
4. 핵심 기법 1단(regular): scaled min-sum 메시지 규칙 `u_C→V=k·sign(∏v)·min(|v|)`(k=scaling factor)을 systematic/parity 두 그룹으로 나눠 별도 PDF(`f_v(1)`, `f_v(2)`)를 반복 추적하는 density evolution 유도.
5. 핵심 식: check-to-variable 메시지 PDF는 `G(fa,fb)` 연산(부호별 convolution 조합, 식(7)(8))으로 계산, scaling factor `k`가 곱해진 변수 `kv`에 대해 계산 — 균일 채널 DE를 비대칭 2-채널로 확장한 핵심.
6. 핵심 기법 2단(irregular): systematic/parity bit·check 노드의 degree distribution을 4개 그룹(`λ1,λ2,ρ1,ρ2`)으로 분리 추적, IRA(irregular repeat-accumulate) 부호에 특화된 `λ2(x)≈x, ρ2(x)≈x` 근사 적용.
7. 부수 트릭: differential evolution(DE) 최적화로 degree distribution 탐색(최대 50세대, 세대당 최대 200 iteration DE).
8. 최적화 절차: 주어진 code rate/k에서 threshold `σ*`를 최대화하는 degree distribution을 DE로 탐색(Table III).
9. 결과: k=0.8이 최적 scaling factor로 확인(Table II), s_EPF가 클수록(비대칭 심할수록) threshold 저하; length 6000, rate 0.9 시뮬레이션에서 density evolution 예측과 waterfall 영역 성능이 잘 일치했으나 error floor는 threshold와 무관하게 다를 수 있음(Case1 vs Case2).
10. 한계: 점근적 density evolution 분석 도구이며, HW 미설계·순수 시뮬레이션. Bliss scheme 고유의 비대칭 채널 가정에 특화되어 일반 NAND 채널(단일 균일 채널)에는 직접 적용 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| scaled min-sum 메시지 규칙 (`k` scaling) | `decoder.cpp` `CNU_Update_New_Mag()` | Prime ECC의 min-sum도 magnitude 양자화 테이블 기반 스케일 개념 존재하나, 본 논문의 scaling factor 값 자체는 비대칭 2-채널 특화라 직접 재사용 불가 |
| systematic/parity 비대칭 채널 density evolution | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | NAND는 read voltage 기반 균일 채널이라 Bliss scheme의 두 채널 분리 모델과 대응 약함 |
| irregular LDPC(IRA) degree distribution 최적화 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 고정 QC-LDPC 구조, degree distribution 재설계 대상 아님 |

> 적용 가치: 낮음. 광/자기 저장 Bliss scheme 특유의 비대칭 채널 가정에 특화된 density evolution 분석으로, NAND 균일 채널 기반 Prime ECC의 디코더/채널/부호설계 어느 모듈에도 직접 이식할 요소가 없음.
