### B. 알고리즘 요약

1. 표준 LDPC는 generator matrix 기반 encoding 복잡도가 block length에 제곱(`O(N^2)`)으로 증가하는 문제가 있음; Richardson-Urbanke(RU)는 parity-check 행렬을 approximately lower triangular(ALT) 형태로 행/열 치환해 `O(N+G^2)` 선형복잡도 인코딩을 달성.
2. 기존 RU 방법은 주어진 부호에 대해 매번 sparse ALT 형태로 변환하는 전처리(preprocessing) 단계가 필요하고, gap `G`를 줄이는 것이 최적화 대상.
3. 이 논문은 전처리 없이 애초에 ALT 제약을 만족하도록 부호 앙상블(bipartite graph ensemble) 자체를 구성하는 `CALT(λ,ρ,g)` 앙상블을 제안 (variable node i는 check node `G+i`에 고정 엣지로 연결, 상위 check node와는 연결 금지).
4. 핵심 식: 매칭 유효조건 `Σ(Xj-1) ≤ Σ Yj` (식(1), i=1..L-G)를 만족하는 그래프만 유효; 위반 시 소수 노드의 degree를 `lmin`으로 강제 조정하는 expurgated 앙상블 `Ĉ_ALT(λ,ρ,g)` 정의.
5. Theorem 1~3: `N→∞`일 때 ALT 앙상블의 거의 모든 그래프가 expurgated 조건을 만족하며(w.p. 1-o(1)), tree assumption(2l-neighborhood가 cycle-free)이 표준 앙상블과 동일하게 성립함을 증명 → density evolution 임계값(threshold)이 표준 앙상블과 점근적으로 동일.
6. gap `g=0`인 특수 경우를 lower triangular(LT) 앙상블 `CLT(λ,ρ)`로 정의: parity bit를 뒤에서부터 재귀적으로 계산하는 완전 선형 인코딩, RU의 역행렬 계산·특수 연산 순서 불필요.
7. 부수 트릭: gap 크기 `g`를 자유롭게 선택 가능해 인코딩 복잡도(`O(N+g^2N^2)`)와 성능 사이 트레이드오프 조절 가능; `g=O(√N)`이면 선형 인코딩 달성.
8. 학습/최적화 절차: 별도 최적화 없이 기존 density evolution으로 설계한 degree profile을 그대로 ALT 제약에 맞춰 재사용 가능.
9. 결과: rate 1/2, (3,6)-regular 및 λ2=0.251/λ3=0.309/λ4=0.002/λ10=0.438, ρ7=0.637/ρ8=0.363 irregular 앙상블에서 N=1000, 10000 시뮬레이션 결과 `G=√N`인 ALT 앙상블과 `G=0`인 LT 앙상블 모두 unconstrained 앙상블과 거의 동일한 BER(waterfall) 곡선을 보임(그림 4~7, 수치는 그래프 전용이라 본문 서술 없음).
10. 한계: 이론/시뮬 수준(AWGN/BSC, N≤10000)에 국한되고 HW 구현·복잡도 실측 없음; error-floor 영역 데이터 없음(짧은 코드 waterfall 위주); LT 앙상블은 작은 인덱스의 variable node 근처에서 cycle이 많아지는 단점 언급.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| ALT/LT 앙상블 기반 선형시간 인코딩 구조 | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | Prime ECC는 이미 dual-diagonal 고정 인코더를 사용하며 이는 LT 앙상블(g=0)의 특수 사례와 유사 개념 — H-matrix 재설계 없이는 직접 대체 어려움 |
| 부호 앙상블(degree distribution) 설계 | `ecc_top.cpp` `Load_PCM()` | H-matrix 자체 재구성 필요, 이식 난이도 높음 |
| 디코더/HW 성능 요소 | 대응 없음 | 논문은 인코딩 복잡도·threshold만 다루고 디코더/HW 미다룸 |

적용 가치: **중간** — dual-diagonal과 개념적으로 유사한 선형 인코딩 구조 이론이나, Prime ECC는 이미 선형시간 인코더를 보유하고 있어 신규성보다는 이론적 배경 참고 수준.
