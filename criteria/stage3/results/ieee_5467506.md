### B. 알고리즘 요약

1. 시스템: PR(EPR4 등) + SOVA 검출 + interleaver + iterative LDPC 복호 구조의 자기기록 채널에서 long LDPC의 error-floor(FER ~1e-12 영역) 추정.
2. 문제: AWGN용 trapping-set 추정법은 있으나 data-dependent(jitter) noise가 있는 현실 자기기록 채널의 error-floor는 미해결.
3. 채널: transition response(error function) + Taylor 전개로 pulse shift 모델링, jitter noise는 transition 시에만 발생하는 data-dependent Gaussian.
4. trapping set 정의: 오류 상태의 fixed point가 되는 variable node 집합(홀수-degree 이웃 적은 subset), error-floor에서 decoding failure 대부분을 지배.
5. 탐색: 고SNR에서 FPGA HW 시뮬로 60 iteration 후 error symptom(오류 상태 VN 집합)을 기록해 trapping set 직접 추출.
6. 실패율 추정: 전체 FER = 각 trapping set별 실패율의 합, `P(TS_i)` 개별 계산.
7. 핵심 근사: trapping set 입력에 영향 주는 소수 noise 변수만 고려, 노이즈 벡터를 정규직교기저 투영 스칼라 `y`(Gaussian, 분산 유도) 하나로 축약 → Monte Carlo 차원 대폭 축소.
8. data-dependent 확장: jitter로 차원별 분산 상이 → asymmetric linear transformation으로 정규화 후 투영 파라미터로 in-situ 시뮬.
9. 결과: interleaver가 TS 내 VN을 짧은 error event로 묶어 동일 그래프 구조 TS도 FER 크게(수 자릿수) 차이, transition 많은 데이터 패턴이 FER 높음. 추정치가 직접 시뮬 대비 0.3 dB 이내.
10. 한계: 부호/복호 자체를 개선하지 않는 평가 방법론, HW 디코더 설계 아님(FPGA는 TS 탐색용), 자기기록 PR/interleaver 채널 특화.
