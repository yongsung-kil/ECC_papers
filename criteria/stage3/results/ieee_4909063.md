### B. 알고리즘 요약

1. 시스템은 OOK(On-Off Keying) 광통신 링크 + BIBD(balanced incomplete block design) 구성 LDPC 코드(예: LDPC(1810,1630), LDPC(2715,2535), LDPC(6176,5983))로, 반복 Belief Propagation(BP) 디코딩 수행.
2. 기존 연구는 AWGN 또는 이상적 central chi-square 채널(“0” 데이터 전력=0)을 가정했으나, 실제 광채널은 ASE(amplified spontaneous emission) noise로 “0” 심볼도 non-null 전력을 가져 부정확했던 문제를 다룸.
3. 채널 모델은 photocurrent를 order `2ns`의 non-central chi-square 분포(신호+AWGN 성분 `2ns`차원 투영합)로 정의, extinction ratio `ER = P1/P0`로 특성화.
4. 핵심 기법은 이 chi-square pdf(식 6)와 Bayes 규칙으로 초기 a priori LLR(식 8, 12)을 일반형으로 해석적 유도 — BP 디코더의 초기화(variable-to-check message) 정확도를 결정.
5. 식 (12)는 `ER→∞`(이상적 케이스)로 단순화되며 기존 문헌([7])의 특수식과 일치함을 보여 일반화 관계를 확인.
6. BIBD-LDPC 구성은 각 컬럼이 소수 `p` 길이 벡터의 순환 이동으로 구성되어 4-cycle을 회피(girth≥6)하는 기존 구성법(코드 자체는 논문 기여 아님).
7. 부수 트릭: 정확한 chi-square LLR 대신 등가 노이즈분산으로 AWGN 근사 LLR(식 4)을 쓰는 경우와 비교해 성능 차이 정량화.
8. 별도 학습/최적화 절차 없음(닫힌 형식 통계 모델 + 시뮬레이션 검증).
9. 결과: ER=∞, chi-square 정확 LLR 사용 시 미부호화 BER 1e-2에서 부호화 BER 1e-9 도달하나 AWGN 근사 LLR 사용 시 동일 조건에서 BER 1e-5에 그침(정확한 LLR 필요성 입증). Waterfall region은 Q-factor 7.5~8.2dB 부근.
10. 한계: 광통신(ASE noise, chi-square 통계, extinction ratio) 특화 채널 모델링·시뮬레이션뿐이며 HW 미설계, NAND RBER 채널과 통계적 성격이 상이해 직접 이식 불가.
