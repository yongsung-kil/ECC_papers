### B. 알고리즘 요약

1. 시스템은 BIAWGN 채널 위 BP 복호 irregular LDPC(rate 1/2, 시뮬 길이 n=50000), degree distribution `λ(x)`, `ρ(x)`로 정의.
2. 문제: BP는 채널 노이즈 분산 `σ²`을 정확히 알아야 최적 성능을 내는데, 오추정 시 초기 LLR 분포가 `N(2/σ²_es, 4σ²/σ²_es)`로 consistency 조건(분산=2×평균)이 깨져 성능이 저하됨(SNR mismatch).
3. 핵심 가정: mismatch 하에서도 channel symmetry 및 check/variable node symmetry는 유지되므로 all-one codeword 가정 하 상호정보량 기반 분석이 가능.
4. 핵심 기법: 기존 EXIT chart(ten Brink)는 consistent Gaussian 가정에 의존해 mismatch 시 적용 불가 → density evolution으로 실제 message 분포를 구하고 `I(X;Y)` 공식(식 2)으로 상호정보량을 산출하는 mismatch용 EXIT curve 생성법 제안.
5. 핵심 식: `I(X;Y) = -∫f_Y(η|X=1)log2(2f_Y(η|X=1)/(f_Y(-η|X=1)+f_Y(η|X=1)))dη`(consistency 없이 성립, 식 2), variable node 출력 `I_Ev = Σλ_i J_cv(I_Av, m_ch)`(식 9).
6. 확장: 관심 offset 구간 `[η1,η2]`에서 최대 threshold `a(η)`를 최소화하는 min-max 부호설계(2절 문제정의) — 이를 위해 check node degree distribution도 최적화 대상에 포함.
7. 부수 구현: EXIT curve는 density evolution으로 얻은 점들을 차수 6 다항식으로 피팅해 생성(폐형이 아닌 수치적 근사).
8. 학습/최적화: fixed check-node curve에 variable-node curve를 매칭하는 방식에서 나아가, check node degree도 함께 최적화하며 목표 offset(예: -2dB, +3dB)에서 threshold 최소화.
9. 결과: 제안 코드 C2는 offset 0dB에서 성능손실 0.1dB 이하이면서 [-2,2]dB 구간 threshold 0.8dB 이내 유지(기존 C1 대비 [-0.5,0.5]dB 밖에서 우세), 유한길이(n=50000) 시뮬에서 BER 1e-5 기준 mismatch 허용범위가 C1 [-1.5,1.5]→C2 [-1.7,1.9]dB(SNR=1dB), [-2.7,5.4]→[-2.7,7.5]dB(SNR=1.6dB)로 확장.
10. 한계: 순수 알고리즘/이론 설계로 HW 구현 없음. 채널은 AWGN 특화이며, mismatch(SNR 오추정) 문제 자체가 NAND의 전형적 실패모드(RBER 변화, retention/read-disturb)와 어떻게 대응하는지는 논문에서 다루지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| EXIT chart 기반 degree distribution min-max 최적화 (부호설계) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC-LDPC(H-matrix 고정) 구조라 irregular degree distribution 재설계는 난이도 높음(§4 "부호설계" 항목) |
| 채널 SNR mismatch 강건성 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | NAND의 RBER 오추정/threshold voltage shift 대응 LLR 테이블 설계에 개념적으로 참고 가능하나 채널모델 자체는 대응 없음 |
| Min-Sum robust to mismatch (관련 논의) | `decoder.cpp` `CNU_Update_New_Mag()` | 이미 Min-Sum 보유, 본 논문은 BP 대비 Min-Sum의 mismatch 강건성만 부가 언급 수준 |

> 적용 가치: 중간 | 채널 오추정에 강건한 degree distribution 설계 개념은 NAND의 read-voltage/threshold 추정 오차 상황에 아이디어 차원에서 참고할 수 있으나, Prime ECC는 H-matrix가 고정된 특정 QC-LDPC라 degree distribution 전면 재설계는 이식 난이도가 높음.
