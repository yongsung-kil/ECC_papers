### B. 알고리즘 요약

1. 시스템: 2bit/cell MLC NAND, 각 워드라인은 MSB/LSB 두 논리페이지, QC-LDPC `(18432,16384)`, `R=0.8889`, PEG로 girth-6 생성.
2. 문제: ICI·retention에 의한 패턴의존 에러는 저장 데이터에 종속 — RLL은 rate 손실 크고, 인접 워드라인 재읽기[2]는 read latency 과다.
3. 채널모델: victim cell 임계전압 교란 `∆V(i,j)`를 5개 간섭셀(x/y/xy 방향)의 `∆V` 선형결합(`α`, `γx/γy/γxy`)으로 모델링.
4. 핵심기법: MSB 복호 실패 시 MSB 다중읽기 대신 **페어드 LSB 페이지를 1회 추가 read**하여 인접셀 심볼정보를 획득 — 전압값보다 다루기 쉬운 심볼레벨 간섭완화.
5. 핵심식: LLR `Li = log( Σ_{xi∈P0} Pr(y|x) / Σ_{xi∈P1} Pr(y|x) )`, 채널을 `Pr(y_{i-1}^{i+1}|x_{i-2}^{i+2})`로 조건부 모델링해 인접 프로그램레벨 추정.
6. 확장: 프로그램레벨을 노이지 관측 `y`만으로 추정, `x̃_{i-2}^{i+2}∈P^4` 256조합 marginalize (`1/256` 균등가정).
7. 구현: 2x nm 기준 전압센싱 40µs, 8KB 페이지 전송 10µs — MSB read 90µs, LSB read 50µs.
8. latency: 3-read MSB(+180µs) 또는 1.5bit 2-read(+90µs) 대비 제안기법은 +50µs만 소요.
9. 결과: `α=0.15`에서 제안기법이 CS(2-read)와 동등, CH(hard) 대비 WER 약 3자릿수 개선; 간섭 심할수록 CS도 크게 능가.
10. 한계: 시뮬만(FPGA/ASIC 없음), 2bit/cell 특정 가정, 디코더/부호 무변경 — 순수 채널·read 전략 기법.

적용 가치: **중간** — 디코더·부호는 그대로이고 채널/LLR 계층에만 닿는 기법이나, NAND 특유 ICI·패턴의존 에러를 소프트 read 비용 증가 없이 다루는 아이디어는 우리 채널모델·LLR 테이블 확장에 참고할 만하다.
