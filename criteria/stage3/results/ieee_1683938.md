### B. 알고리즘 요약

1. 대상은 `[n,k,d]` binary linear block code 및 그 dual code `C⊥`이며, Tanner graph 표현의 4-cycle 존재 여부를 다룸.
2. 기존 연구들이 4-cycle-free Tanner graph를 가진 LDPC 설계가 반복복호 성능에 유리함을 지적했으나, classical block code는 대체로 4-cycle을 포함하는 경향이 있어 그 이유를 규명하고자 함.
3. 채널/노이즈 모델 없음 — 순수 그래프이론적 구조(이분그래프의 크기, 차수) 분석.
4. 핵심 정리(Theorem 4): parity-check 행렬 `H`(p×n, p=n-k)의 Tanner graph가 4-cycle-free이려면 `wt(H) ≤ -np(p-1)/2 + n²/4 + n/2` (근사식, dual 최소거리 `d⊥` 관련 항 포함)를 만족해야 함.
5. 이 부등식은 Reiman's inequality(이분그래프의 4-cycle-free 시 간선수 상한)를 Tanner graph의 행-열 구조에 적용해 유도됨 — 4-cycle이 없으면 그래프 크기(간선수)가 제한된다는 조합론적 사실이 핵심.
6. Corollary 5로 4-cycle-free Tanner graph가 존재하지 않는 부호에 대해, 임의의 Tanner graph가 반드시 포함해야 하는 4-cycle 개수의 하한도 유도.
7. Steiner system `S(2,k,v)`의 incidence matrix가 Theorem 3의 등식 조건(bound tight)을 만족하는 특수 사례로 다뤄짐 — 부호 구성이 아닌 존재성 분석 도구로 사용.
8. 최적화/학습 절차 없음 — 순수 대수적/조합론적 증명.
9. 결과: Golay [23,12,7]/[24,12,8], Reed-Muller (`3≤m≤9`), primitive BCH (`3≤m≤8`, R≥1/2), RS 부호의 binary image(`3≤m≤5`, `1/2≤R≤1-2/n`) 등 다수의 classical code가 4-cycle-free Tanner graph를 가질 수 없음을 증명.
10. 한계: 알고리즘/HW 설계 전혀 없음, 시뮬레이션도 없는 순수 이론 논문이며 LDPC(희소 부호) 자체가 아닌 classical dense block code에 대한 결과.

