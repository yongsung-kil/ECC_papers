### B. 알고리즘 요약

1. Multi-dimensional(MD) 그래프 부호는 M개의 1차원(OD) 부호 사본을 auxiliary matrix `X_1,...,X_{M-1}`로 결합해 `H_MD`를 구성(M은 소수), NAND Flash·자기기록(MR) 등 다중 채널 간섭/불균일성 완화를 목표로 함.
2. 기존 기법(선행연구[11])은 circulant를 relocation하여 short cycle 수를 줄이는데 그쳤으나, 실제 error-floor는 cycle이 결합된 absorbing set(AS)/unlabeled absorbing set(UAS)이 지배하므로 cycle 최소화만으로는 불충분함.
3. UAS를 basic cycle(`n_b`개, `n_b = (a(γ-2)-d1+2)/2`)의 벡터공간(GF(2)) 표현으로 정의하고, relocation `R(E_i,j) ∈ {0,...,M-1}` 하에서 cycle이 활성 상태를 유지하는 조건은 `Σ(-1)^w R(E_iw,jw) ≡ 0 (mod M)`.
4. 핵심 정리(Theorem 1): UAS가 relocation 후에도 활성 유지되려면 이 조건이 UAS의 **모든 basic cycle**에서 성립해야 함 — cycle보다 UAS를 비활성화하는 것이 훨씬 어려움(관용도 낮음)을 증명.
5. 이 어려움 덕분에 cycle 대신 UAS를 표적으로 relocation 옵션을 절약할 수 있음(saving 공식 `S1`, `S2` 유도) — 예시에서 최대 80% relocation 옵션 절약.
6. Algorithm 1: 지배적 (a,d1) UAS 인스턴스를 포함하는 circulant를 순차 선택, 해당 UAS의 basic cycle을 비활성화하는 relocation 옵션에 투표(voting) 후 최다 득표 옵션으로 relocate, auxiliary matrix 간 sparsity 균형도 고려.
7. NAND Flash 채널은 sub-20nm MLC NLM(normal-Laplace mixture) 모델, 3-read soft info, FFT-QSPA 디코더(max 50 iteration)로 시뮬레이션.
8. 2차 분석(Section VI)으로 작은 패턴(β×α submatrix)이 relocation을 통해 더 큰 cycle로 뭉치는 조건과 그 발생 확률이 cycle-to-cycle 재결합 확률보다 현저히 낮음을 증명, Algorithm 1에서 이 2차 효과를 무시해도 됨을 정당화.
9. 결과: MD Code 1(NAND Flash, 회로 7% relocation)이 동일 rate/length OD 부호 대비 waterfall 영역 UBER~10^-7에서 P/E cycle 1200회, error-floor UBER~10^-9에서 1800회 수명 이득. MD Code 2(MR)는 SNR 0.4dB(waterfall)/1.1dB(error-floor) 이득, error-floor 완전 제거.
10. HW 구현/gatecount는 없으며 시뮬레이션(FFT-QSPA soft 디코더)에 한정, 상당수 예제가 non-binary(GF(4)) SC 부호이고 알고리즘은 한 번에 하나의 지배적 object만 다룸(다중 objects 동시 최적화는 future work).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| UAS/absorbing-set 기반 circulant relocation 부호설계 (Algorithm 1) | `ecc_top.cpp` `Load_PCM()` | H-matrix/PCM 구조를 결정하는 부호설계 레이어라 이식 난이도 높음(특정 QC-LDPC 고정, 재검증 부담) |
| Basic cycle 조건(식 3) 기반 absorbing set 비활성화 | 대응 없음 | Prime ECC 프로파일에 error-floor 대책용 부호설계 최적화 도구가 명시되어 있지 않음 |
| NAND Flash NLM 채널 모델 + soft-read(3-read) | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | soft-read 양자화·threshold 설계와 관련되나 이 논문은 채널 모델 자체가 아닌 부호구조 최적화가 핵심 |
| FFT-QSPA (non-binary) 디코더 | 대응 없음 | Prime ECC는 binary min-sum 전용, non-binary QSPA는 프로파일상 판정 기준에서 이식성 `하` 대상 |

적용 가치: **중간** — NAND Flash를 직접 타깃으로 error-floor를 실측(P/E cycle) 개선한 점은 관련성이 높으나, 기여가 H-matrix 구성(부호설계) 레이어에 집중되어 있어 Prime ECC의 고정 QC-LDPC 구조에 이식하려면 `Load_PCM()` 대상 재설계·재검증이 필요함.
