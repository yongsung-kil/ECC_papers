### B. 알고리즘 요약

1. AWGN 채널, sum-product 알고리즘(최대 50 iteration)으로 복호하는 QC-LDPC의 일반화인 APM-LDPC(affine permutation matrix 기반) 부호 구성법 논문이다.
2. 기존 랜덤 LDPC는 parity-check 행렬 저장에 메모리가 많이 들고 encode가 어려워, 대수적으로 구조화된 QC-LDPC가 상용에 유리하지만 girth(최소 cycle 길이, 특히 4-cycle)를 크게 만들기 어려운 문제를 다룬다.
3. 각 서브블록을 순수 circulant(`x+b`)가 아닌 일반 affine map `f(a,b)(x)=ax+b` (`a∈Z_L*`, `b∈Z_L`)로 정의한 `L×L` 순열행렬 `P_f(a,b)`로 구성하는 것이 핵심 모델이다.
4. block-cycle의 함수사슬(function chain) `(f1,...,f2l)`의 특성함수 `z(x)=f2l∘f2l-1^-1∘...∘f1^-1(x)`을 정의하고, `z^r(x0)≡x0 mod L`을 만족하는 최소 `r`이 존재하면 Tanner graph에 길이 `2lr`의 cycle이 생긴다(Theorem 1).
5. `z^r(x)=a^r x+(a^(r-1)+...+1)b`이며, `a=1`(QC-LDPC 특수해)이면 기존 결과(Corollary 2)로 환원되어 이 식이 girth 제어 여부를 판정하는 근거가 된다.
6. 서로 소인 길이 `L1, L2`를 갖는 두 APM-LDPC 부호(동일 mother matrix)를 중국인의 나머지 정리(CRT)로 결합해 `L=L1L2` 크기의 function matrix를 만드는 CRT-combining 절차(3단계)를 제안한다.
7. Theorem 5로 결합 후 girth를 결정하는 `r=lcm(r1,r2)`임을 증명해, 결합 부호의 girth가 원본 부호들의 girth 이상임을 보장한다(성능 저하 방지의 핵심 논거).
8. 별도 학습/최적화 절차는 없고, array code `C(p,j)`(`Fik=x+(i-1)(k-1)`) 여러 개를 CRT로 결합하는 구체적 구성 예시로 대형 부호를 만든다.
9. (4,9)/(4,12)/(5,12)-regular girth-8 소형 부호는 랜덤 girth-6 LDPC 대비 BER=10^-6에서 뚜렷한 성능 저하 없음. rate 0.9688인 (49280,47743,4)(5,7,11) 고rate 대형 부호는 BER=10^-6에서 Shannon 한계 대비 약 0.55dB 이내.
10. HW 구현/합성 없음, column weight(≥4) 조건에서만 성능 저하가 없음이 확인되어 저 column-weight(예: NAND ECC의 degree 구조)에는 일반화가 제한적일 수 있음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Affine permutation 기반 base matrix + CRT 결합 구성법 | `ecc_top.cpp` `Load_PCM()` | QC-LDPC(`PCM_b` base+shift)를 일반 affine(ax+b)로 확장하는 구성법으로 H-matrix 재설계에 해당, 이식 난이도 높음 |
| girth 제어(4-cycle 회피) 이론 | 대응 없음 | Prime ECC는 고정 H-matrix 사용, girth 설계 파이프라인 없음 |
| CRT를 이용한 소형→대형 코드 결합(길이 확장) | 대응 없음 | Prime ECC의 lifting(`z_sb=32`)과는 다른 방식의 길이 확장 기법 |

> 적용 가치: 중간 — QC-LDPC를 일반화한 affine permutation 구조와 girth 보존 결합법은 Prime ECC가 이미 사용하는 base matrix+circulant shift 철학과 개념적으로 가깝고 대형 고rate 부호 구성에 참고 가치가 있으나, H-matrix 자체의 재설계·재검증 부담이 크다.
