### B. 알고리즘 요약

1. 대상은 CSS(Calderbank-Shor-Steane) quantum LDPC 코드로, X/Z 두 종류 parity-check(`HX`, `HZ`, `HX HZ^T=0`)를 갖는 stabilizer 코드이며 i.i.d. depolarizing 채널(X/Z/Y 오류 각 확률 `p/3`)을 가정.
2. 문제: QLDPC의 Tanner graph는 CSS 제약상 4-cycle을 필연적으로 포함해 plain BP 성능이 열화되며(symmetric stabilizer trapping set), 기존 post-processing(random perturbation, enhanced feedback, matrix augmentation, OSD)은 시행횟수가 많이 필요.
3. 핵심 채널/부호 모델: quaternary BP(BP4)가 X/Z 오류 상관관계를 반영해 알파벳 `{I,X,Y,Z}` 전체에서 직접 동작(`ΛiX, ΛiY, ΛiZ`의 LLR 벡터).
4. 핵심 기법: BP4 반복 사이에 학습 가능한 "feedback GNN"을 삽입(Algorithm 1, Sandwiched BP GNN Decoding)해, 이전 BP 라운드의 사후 LLR `Λpost`로부터 다음 BP 초기화용 `Λ′`를 산출.
5. 핵심 식: check node LLR `∆j→i=(-1)^{sj}·⊞λi′→j`(Gallager boxplus, 식 2-3)와 GNN 메시지 `m_{cj→vi}=f^{CX/Z→V}(hcj||hvi)`(식 9)로 CN/VN 특징을 정의; VN 업데이트는 X/Z 타입 메시지 평균(식 10)을 MLP(`f^VN`)에 통과시켜 `Λ′` 생성.
6. GNN 구조는 X-type/Z-type CN마다 가중치를 공유하는 은닉유닛 40의 MLP(tanh), 총 3923개 파라미터의 경량 1회 업데이트 구조(반복 GNN 아님).
7. 학습은 boxplus loss(식 6-8의 `Γ`로부터 계산되는 binary cross-entropy)를 사용하며 4단계 절차: (1) 64회 BP 실패 패턴으로 easy dataset 수집 → (2) coarse GNN 학습(16+16 BP 샌드위치, RTX4090 약 10분) → (3) coarse GNN 실패 패턴으로 hard dataset 수집(수만 개, 며칠 소요) → (4) easy+hard 혼합(hard 50배 반복)으로 refined GNN 학습, 10개 모델 중 최선 선택.
8. 최종 디코더는 (64,16,16,16) BP 반복 구성으로 GNN 가중치를 공유하며 최대 3~5회 attempt.
9. 결과: `[1270,28,≤46]`, `[882,24,≤24]` 두 medium-size QLDPC 코드에서 random perturbation/enhanced feedback/matrix augmentation 대비 훨씬 적은 attempt 수로 error-floor를 유의미하게 낮춤(logical error rate 그래프 기준, 구체 dB/배수는 그림에만 존재해 본문 수치 미상).
10. 한계: 32-bit float 시뮬레이션뿐 HW 미설계, non-binary/양자 CSS 코드 전용 구조라 다른 코드에 적용 시 데이터셋을 매번 새로 수집해야 하며(며칠 소요) MLP 행렬곱 오버헤드가 16-iteration BP와 유사한 복잡도.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP4(quaternary belief propagation) | 대응 없음 | non-binary(4진 알파벳) 양자 코드 전용, Prime ECC는 binary LDPC만 지원 |
| feedback GNN 피드백 LLR 재초기화 | `channel.cpp` `Set_LLR_Th()` | LLR 재초기화 개념만 유사하나 GNN(NN) 자체는 대응 없음 |
| boxplus 연산 기반 신뢰도 계산 | `decoder.cpp` `C2V_Cal()` | min-sum이 아닌 sum-product(tanh) boxplus 방식으로 근사 방식이 다름 |
| CSS stabilizer 부호 구조 / trapping set 대응 | 대응 없음 | QLDPC stabilizer 특유 구조로 NAND binary QC-LDPC와 무관 |

적용 가치: **낮음** — 양자 CSS LDPC와 quaternary BP라는 non-binary 특수 구조에 강결합된 GNN 피드백 기법으로, binary QC-LDPC 기반 Prime ECC와 부호·복호 알고리즘 계열 자체가 근본적으로 다름.
