### B. 알고리즘 요약

1. 시스템은 noiseless 2-user unsourced binary adder channel(U-BAC)을 frame delay `τ`로 확장한 AU-BAC(`Yi = Xi^(1) + Xi^(-2)τ`), 두 사용자가 동일 LDPC 코드 사용, rate 목표는 `r_d < 3/4`.
2. 문제: 두 사용자의 Tanner graph를 multiple-access node(MN)로 연결한 joint graph에서, peeling decoding 시 degree-1 VN 4개로 구성된 최소 stopping set(4SET)이 특정 delay τ에서 형성되어 PUPE(per-user error) error-floor를 유발.
3. 핵심 가정: 채널은 BEC와 유사하게 모델링(수신값 `Yi=0`이면 소거로 간주), peeling decoding으로 두 코드워드를 동시복호. 단일 사용자에서는 코드비트 순서가 무관하지만, 비동기 2-user 환경에서는 PCM 컬럼 순서가 성능에 직접 영향.
4. 핵심 기법: CN마다 연결된 degree-1 VN들의 위치 간 거리(distance multiset) `D_j`, 전체 `D = ⊎D_j`를 정의. Theorem 1: `D`의 모든 원소가 unique하면 어떤 delay τ에서도 4SET이 형성되지 않음(4SET-free PCM).
5. Theorem 3: degree-1 VN을 모든 CN에 고르게 분산(`|V_u|-|V_w|≤1`)시키면 `|D|`가 최소화되어 4SET-free 조건 달성이 쉬워짐. 평균 CN당 degree-1 VN 수 `V := L1/(1-r_d)`가 핵심 변수.
6. 확장 기법: `V≤1`이면 균등분산만으로 충분, `V≤2`이면 결정적 구성식(`V_j = (n+1)/2 ± (j-1/2)`) 존재, `V>2`이면 Algorithm 1(확률적 컬럼 재배치로 거리 중복 회피)과 이를 반복하는 Algorithm 2로 임의 크기 stopping set까지 제거.
7. 부수 트릭: 제안 알고리즘은 PCM의 컬럼(변수노드)만 순열하므로 단일사용자 관점의 부호 특성(degree distribution 등)은 전혀 변경하지 않음. Progressive Edge Growth(PEG) 구성이 degree-1 VN을 CN에 자동으로 고르게 분산시키는 부수효과 발견.
8. 최적화 절차: [7]의 density evolution 기반 degree distribution 최적화로 앙상블 선정(Table I) 후, Even-RCC/PEG로 초기 PCM 생성, Algorithm 1/2로 후처리(별도 학습·NN 없음).
9. 결과: n=1600에서 PEG+stopping set 제거가 plain RCC 대비 PUPE를 84.2배, plain PEG 대비 9.29배 감소. Even-RCC만으로도 RCC 대비 error-floor 1자릿수(0.03→0.0035) 개선.
10. 한계: 잡음 없는(noiseless) 2-user 채널 가정, HW 구현 없음(순수 알고리즘·시뮬레이션), density evolution이 원래 무제약 앙상블 기준이라 제안 구조화된 코드에는 근사적으로만 유효, 3-user 이상 확장은 다루지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 4SET(degree-1 VN stopping set) 제거용 PCM 컬럼 순열 | 대응 없음 | Prime ECC는 min-sum BP 디코더(soft-decision)이며 peeling decoding/BEC stopping set 개념 자체가 적용되지 않음 |
| PCM/H-matrix 구성 (degree distribution, PEG) | `ecc_top.cpp` `Load_PCM()` | 부호구조 로드 지점은 대응하나, 이 논문의 multi-user joint graph 구성법은 단일사용자 QC-LDPC와 목적이 상이해 실질적 이식 난이도 높음 |
| stopping set 회피 이론(trapping set과 유사 개념) | `decoder.cpp` `LDPC_Decoder()` | 개념적으로 trapping-set 회피와 유사하나, 이 논문은 peeling decoding(erasure) 전용이라 min-sum 반복복호의 trapping set과 메커니즘이 다름 |

적용 가치: **낮음** — noiseless multi-user unsourced access용 peeling decoding stopping set 제거 기법으로, NAND 단일사용자 soft-decision min-sum LDPC ECC와 채널모델·디코딩 알고리즘이 근본적으로 달라 직접 이식 불가.
