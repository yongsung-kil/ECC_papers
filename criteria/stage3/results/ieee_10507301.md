### B. 알고리즘 요약

1. 대상은 일반 binary LDPC를 QPSK로 AWGN에 전송, code length `512/864/1024/2048`, rate `1/2, 2/3, 3/4`.
2. 표준 min-sum(MS)이 CN 갱신에서 `∏tanh(L/2)`를 최소항으로 근사해 성능 손실이 있는 문제를 개선.
3. LLR-BP의 CN 식 `L(r_ij)=2 tanh⁻¹(∏ tanh(L(q_ij)/2))`를 부호부와 절댓값부로 분해한 MS를 출발점으로 삼음.
4. 근사오차가 code length·channel·iteration에 의존한다고 보고 CN 항을 `α·min(tanh(½L(q_ij))) − β` 형태로 보상(식 9).
5. `tanh(x)`가 `|x|<1.5`에서 급변하는 특성을 근거로 곱셈인자 `α`, 보정인자 `β`를 도입.
6. 조건부 스위칭: `min|L(q_ij)|≥1.5`이면 `α=1, β=0`, `<1.5`이면 `α=0.75, β=0.125` 적용(Fig.5 flowchart).
7. `α`, `β` 값은 iteration 수 최소화 기준으로 실험적으로 결정(`β≈0.125`, `α≈0.75`, Fig.3-4).
8. 결과: MS 대비 동일 BER 도달 `E_b/N0` 약 0.1~0.2dB 개선, code length 512에서 약 0.15dB, 1024 0.12dB, 2048 0.08dB.
9. rate별(864, 1/2·2/3·3/4)에서도 MS 대비 약 0.12dB 개선.
10. 한계: 인자가 고정 상수이며 code length·channel 기반 동적조정은 미구현, HW 미설계·gatecount/throughput 미보고.
