### B. 알고리즘 요약

1. 시스템/부호: IEEE 802.16e WiMAX QC-LDPC, `N=2304`, 서브행렬 `p=96`, 코드레이트 `1/2, 2/3, 3/4`, base matrix는 circulant permutation + 이중대각(staircase) 패리티 구조.
2. 문제: 표준 row layered decoding이 순차 처리로 자원·throughput 병목이라 다중 rate 고throughput 디코더가 필요.
3. 채널/모델: BPSK 변조, AWGN 채널, LLR 기반 soft decision.
4. 핵심 기법: row layered `min-sum` — CN 업데이트 `Rij = α·∏sign(Lij)·min|Lij|` (식 2), `min1/min2` 방식, `α=0.5` correction factor.
5. VN 업데이트 `Lij = Iv + Σ Rij` (식 5), posterior `Qj = Iv + Σ Rij` (식 6)로 hard decision, parity check 0이면 종료.
6. 확장: 패리티 검사 행렬의 행을 재배열(reordering)하여 두 개 이상 행을 동시에 병렬 처리(modified layered scheduling, Fig.5).
7. 구현 디테일: 행 간 data dependence를 줄여 조기수렴; p-level 병렬(p개 동일 constituent decoder)에 행 병렬을 추가.
8. 학습/최적화: 없음 (스케줄 재배열만).
9. 결과: 1/2 rate에서 표준 대비 약 `0.1 dB` 조기수렴, `BER 10^-6 @ 2.5 dB`, 최대 iteration 15~20.
10. 한계: HW 미설계(MATLAB 시뮬만), 실제 throughput 수치 없음, 특정 WiMAX 부호에 종속, 기여가 표준 재사용 수준.
