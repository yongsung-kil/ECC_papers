### B. 알고리즘 요약

1. 대상: 이진 정규 QC-LDPC 부호, AWGN·binary random erasure·binary burst erasure 세 채널 대상, rate 0.50~0.90, 길이 2040~16352 bit.
2. 문제: 기존 LDPC 구성법은 주로 AWGN 채널에만 초점, erasure/burst-erasure 채널 대응 구성법은 드묾. 다중 채널에 범용적으로 잘 동작하는 QC-LDPC가 필요.
3. 채널 모델: AWGN(BPSK, SPA 반복복호), binary erasure channel(BEC, 확률 `p`로 소실), burst erasure channel(연속 구간 소실).
4. 핵심 기법 1단: `m`-차원 Euclidean geometry `EG(m,q)`의 line 교차 구조를 이용해 circulant permutation matrix 배열 `H`(RC-constraint 만족)를 구성, 임의 부분배열(subarray) 선택으로 다양한 rate/length의 QC-LDPC 코드 생성 (Theorem 1).
5. 핵심 식: RC constraint(두 행/열이 공통 1을 최대 1개만 가짐)로 길이-4 cycle 제거→girth≥6 보장, 이는 minimum distance `≥ γ+2`(짝수 column weight `γ`)를 보장하여 error-floor를 낮춤.
6. 핵심 기법 2단: burst-erasure 대응 위해 "zero-covering span" `s(H)` 정의(Theorem 2) - PC행렬의 각 열이 커버하는 최대 zero-span 길이로 erasure-burst 정정 하한을 보장.
7. 부수 기법(dispersion): 특정 subarray를 circulant/zero 행렬의 확장 배열로 "dispersion"하여 zero-covering span을 인위적으로 늘려 burst-erasure 정정능력을 강화(Section VII, Example 3/4).
8. 최적화 절차: 별도 학습 없음. subarray 선택은 파라미터(row/col weight pair) 지정 후 실험적으로 rank/성능 확인.
9. 결과: (16352,14338) rate=0.8768 코드가 AWGN에서 Shannon limit 대비 0.8dB, BEC에서 threshold 대비 0.01 이내 성능; (10200,8191) 코드는 이론 하한 1021보다 큰 실제 1366-bit burst 정정 가능.
10. 한계: HW 미설계, 시뮬레이션(반복복호 최대 100회)만 수행, 구성이 특정 유한기하 구조(EG(m,q))에 종속되어 임의 rate/length로 자유 설계 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Euclidean geometry 기반 H-matrix/부호구조 구성 | `ecc_top.cpp` `Load_PCM()` | 부호설계 레이어로 이식 난이도 높음(특정 QC-LDPC 고정, 재검증 부담 큼) |
| RC constraint 기반 girth-6 보장 (error-floor 개선) | `ecc_top.cpp` `Load_PCM()` | H-matrix 설계 시 girth/minimum distance 기준으로 참고 가능하나 코드 구조 자체 이식은 아님 |
| burst-erasure 대응 zero-covering span / dispersion 기법 | 대응 없음 | Prime ECC는 NAND read-error 채널(RBER/fixed-error) 대상이라 erasure-burst 채널 대응 기법과 직접 연결점 없음 |

> 적용 가치: 중간 - 대수적 QC-LDPC 부호설계 원리(girth-6 보장, RC constraint)는 참고 가치가 있으나 erasure 채널 특화 기법은 NAND 환경과 무관하고, 부호구조 자체 이식은 난이도가 높음.
