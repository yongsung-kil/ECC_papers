### B. 알고리즘 요약

1. 시스템은 AWGN 채널 상 rate 1/3~7/8의 protograph 기반 turbo-like 코드(ARA)이며 LDPC의 부분집합으로 표현 가능.
2. 풀려는 문제는 기존 RA/IRA 코드나 unstructured irregular LDPC보다 낮은 iterative decoding threshold(용량 한계에 근접)를 얻는 것 — 특히 max variable node degree 제약 하에서.
3. 핵심 가정: repetition code(반복부호)의 extrinsic SNR 거동이 low-SNR 영역에서는 우수하지만 high-SNR 영역에서 부족하다는 관찰에서 출발, accumulator를 precoder로 추가하면 high-SNR 거동이 개선됨(precoding gain).
4. 핵심 기법: ARA = Accumulate(precoder) + Repeat(반복 3~4) + Accumulate(inner, puncturing 적용) 직렬연쇄(serial concatenation) 구조, protograph(소형 Tanner graph)로 표현하여 copy-and-permute로 임의 크기 코드 생성.
5. 핵심 식: FER 상한(Divsalar bound) `P_F <= sum A_w * f(w,...)`(식 1~4)로 ML 디코딩 성능/threshold를 distance spectrum(weight distribution, IOWE)으로 산출 — protograph 구조가 iterative/ML 양쪽 성능을 결정.
6. 확장: puncturing pattern(주기 `p`)을 가진 accumulator의 IOWE를 2-D z-transform으로 계산(`W(Y,Z)`), 상위 rate 코드 패밀리(1/2~7/8)를 protograph 추가 puncturing으로 구성.
7. 부수 트릭: "doping" — accumulator를 거치지 않는 일부 비트를 시스템 비트로 그대로 전송해야 iterative decoding이 시작 가능(비율 `q` 파라미터).
8. 최적화 절차: Gaussian density evolution(및 정확 density evolution on protograph)으로 threshold 계산, `q` 값에 대한 threshold 최소화(2차 미분 음수 확인).
9. 결과: rate-1/2 ARA(degree≤5) threshold 0.08dB(용량한계 대비), 기존 unstructured irregular LDPC(동일 degree 제약) 0.72dB 대비 큰 개선. ML 성능도 random code에 근접(예: ARA(4,4) threshold 0.31dB, random code 0.308dB).
10. 한계: 이론적 threshold/시뮬레이션 분석에 그치며 HW 구현, 복잡도, throughput은 다루지 않음; message-passing 표준 알고리즘을 그대로 사용(디코더 자체 개선 없음).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| ARA protograph 부호 구성(H-matrix 대응) | `ecc_top.cpp` `Load_PCM()` | 완전히 다른 turbo-like 직렬연쇄 구조로 기존 QC base matrix 교체 필요, 이식 난이도 높음 |
| circulant permutation 기반 인코더(참고문헌) | `encoder.cpp` `Generate_PCM_encoding()` | circulant 구조라는 점만 유사, ARA의 accumulator 직렬연쇄 구조는 dual-diagonal 인코더와 상이 |
| message-passing 디코딩 | `decoder.cpp` `LDPC_Decoder()` | 표준 기법 사용, 논문 고유 개선 없음(이미 보유) |
| Divsalar FER/BER bound, weight distribution 분석 | 대응 없음 | 코드 이론 분석 도구로 HW/디코더 구현과 무관 |

> 적용 가치: 낮음 — turbo-like ARA/protograph 코드 설계 및 threshold 이론 분석 논문으로, Prime ECC의 고정된 QC-LDPC base matrix·dual-diagonal 인코더 구조와 근본적으로 다른 코드 패밀리이며 HW 이식 요소가 없음.
