### B. 알고리즘 요약

1. 대상 시스템은 고속이동(최대 4860km/h Doppler) 환경의 MIMO-AFDM(affine frequency division multiplexing) 통신으로, 채널코드는 5G NR LDPC(base graph 2, length 512, rate 0.5)를 사용한다.
2. 기존 turbo-IDD(반복 검출-복호) 구조는 VI 검출 그래프와 LDPC Tanner 그래프가 분리되어 있어 interleaver/deinterleaver, 심볼-비트 LLR 변환 오버헤드로 지연·복잡도가 크다는 문제를 다룬다.
3. 채널모델은 P개 경로의 doubly-selective 채널(지연 `dp`, 도플러 `νp`)이며, VI(변분추론)로 사후확률 `p(x|y)`를 근사하는 베이지안 검출 문제로 정식화한다.
4. 핵심 기법은 VI 검출 인수그래프(AVN-FN)와 LDPC Tanner그래프(LVN-CN)를 하나의 이분그래프로 병합한 Joint Sparse Graph(JSG)이며, BP-JSG(KL(p|q))와 EP-JSG(KL(q|p))를 유도한다.
5. 핵심 식은 VN 갱신식 `L_{α_i→β_j} = L^a_{α_i,LDPC} + Σ L_{β_j'→α_i}`로, LDPC 사전 LLR과 VI 그래프의 메시지가 하나의 노드에서 동시에 합산되어 별도 외부 반복 없이 정보가 교환된다.
6. 확장 기법인 E-JSG(enhanced JSG)는 linear constellation encoding `x=gu`를 이용해 심볼VN 대신 비트VN(BVN)만 사용, interleaver/deinterleaver와 심볼-비트 LLR 변환을 완전히 제거한다.
7. 부수 트릭으로 eSNR(effective SNR) 기반 sparse channel 근사를 도입: 채널계수가 작은 다수 그래프 엣지를 단일 엣지로 근사(threshold `eSNR_th=-12dB`)해 FN degree를 줄인다.
8. 학습/최적화 절차는 없으며 전부 폐형 메시지패싱 유도(VI/BP/EP)와 시뮬레이션 파라미터 튜닝(eSNR threshold 그리드 탐색)이다.
9. 결과: MMSE-LDPC 대비 coded BER 기준 약 2dB(SISO)~4dB(2x2 multiplexing) 이득, 반복수는 BP/EP-JSG 7회, E-JSG 9회로 MMSE-LDPC(11회) 대비 감소, EP-turbo-IDD 대비 3 outer iteration에서 약 2dB 이득을 보였다.
10. LDPC 자체는 5G NR 표준 코드를 변경 없이 사용하며(부호설계 기여 없음), HW 구현/합성 없이 MATLAB 런타임 기반 정규화 복잡도 비교와 시뮬레이션에 국한된다.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LDPC Tanner그래프와 VI 검출그래프 병합(JSG) | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 단일 NAND 채널 디코더로 별도 외부 MIMO 검출기와의 그래프 병합 구조 자체가 대응 없음 |
| BP(KL(p|q)) 기반 CN/VN 메시지 갱신 | `decoder.cpp` `C2V_Cal()` | Sum-Product 계열 BP이며 Prime ECC의 min-sum 근사와 알고리즘 계열이 다름 |
| eSNR 기반 sparse channel 엣지 근사 | 대응 없음 | MIMO 다중경로 채널의 그래프 엣지 축소 기법으로 NAND 채널·PCM 구조와 무관 |
| 5G NR LDPC(BG2) 코드 자체 | `ecc_top.cpp` `Load_PCM()` | 표준 5G 코드를 그대로 사용, 이 논문의 기여가 아니므로 참고용 |

> 적용 가치: 낮음 — 무선 MIMO 채널의 검출기-LDPC 그래프 병합 수신기 아키텍처로 부호설계·디코더 알고리즘 자체 개선이 없고 NAND 단일채널 ECC 구조와 접점이 없음.
