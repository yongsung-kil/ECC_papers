### B. 알고리즘 요약

1. 대상은 primitive rateless (PR) code — GF(2) 위 primitive polynomial 기반 m-sequence로 생성되는 rate-compatible 짧은 블록 부호(길이 32~128, URLLC/mMTC 타깃), LDPC와는 별개의 대수적 부호.
2. 문제: 대수적 부호(RM, BCH, PR)는 PCM이 조밀해 4-cycle 등 짧은 사이클이 많아 BP 성능이 저하되고, 기존 해법(MBBP)은 다중 PCM 구성이 계산량이 크고 비확장적임.
3. 채널 모델은 BI-AWGN + BPSK, 디코더는 damped normalized min-sum(NMS): `L_c(l→v) = λL_c(l-1→v) + γ(1-λ)·min|L_v'|·sgn(...)` (스케일 γ, 댐핑 λ=0.7).
4. 핵심 기법 1단: primitive polynomial `p(x)`의 support 집합이 Golomb ruler(모든 pairwise 차분이 유일)이면 임의의 rate에서 표준 PCM이 4-cycle-free임을 이용, 최대 허용 weight `wp/2 < k` 조건 하 탐색 알고리즘(Algorithm 1)으로 해당 다항식을 구성.
5. 핵심 기법 2단(double-bases): 수신 LLR 크기 내림차순 정렬 permutation `π1`을 표준 PCM `H1`에 적용 후 Gaussian elimination으로 `H2=[P̃|I_{n-k}]` 생성 — 신뢰도 낮은 비트를 degree-1 변수노드로 만들어 국소적으로 쉽게 정정.
6. 두 PCM(`H1`, `H2`)에서 각각 BP를 최대 `Imax` 반복 병렬 수행해 두 후보 codeword를 얻고, 수신신호와 유클리드 거리가 더 짧은 쪽을 최종 선택.
7. 복잡도: 표준 BP는 `O(|E|·Imax)`, MBBP는 N개 PCM에 비례하는 `O(ΣN|Ei|Imax)`인 반면, double-bases는 정렬/GE 비용 `O(n log n)+O(n·min(k,n-k)^2)` + 2개 PCM BP 비용만 필요해 PCM 수를 2개로 고정.
8. 학습/최적화 절차 없음(구성적 설계 + 결정론적 GE).
9. 결과: PR(24,64)에서 double-bases BP가 near-ML 성능 달성(MBBP는 동등 성능에 4개 PCM 필요), 표준 BP 대비 약 1dB 이득, PR(64,128) 576 edges가 RM(64,128) 1246 edges 대비 우수, PR(20,128)이 Polar CA-SCL(L=32) 대비 낮은 SNR에서 우수.
10. 한계: LDPC가 아닌 PR code 전용(NAND 표준 LDPC와 부호 구조 자체가 다름), 매우 짧은 블록길이(≤128) 및 URLLC/mMTC向 시뮬레이션뿐이며 HW 구현·gatecount·throughput 분석 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| double-bases BP(두 PCM 병렬 디코딩 + 유클리드 거리 선택) | `decoder.cpp` `LDPC_Decoder()` | 개념적으로 multiple-decoding-attempt 구조와 유사하나, PR code 고유의 두 번째 PCM 생성 방식(LLR 정렬+GE)은 QC-LDPC 고정 PCM 구조와 맞지 않음 |
| damped normalized min-sum (NMS) | `decoder.cpp` `CNU_Update_New_Mag()` | 이미 보유한 min-sum 계열, 댐핑 인자만 추가된 변형으로 중복 기여 |
| Golomb ruler 기반 4-cycle-free PCM 구성 | `ecc_top.cpp` `Load_PCM()` | PR code 전용 대수적 구성법으로 QC-LDPC 구조에 이식 불가 |
| Gaussian elimination 기반 2번째 PCM 생성 | 대응 없음 | PR code의 rateless 특성에 특화된 기법, 고정 rate QC-LDPC에는 부적합 |

> 적용 가치: 낮음 — LDPC가 아닌 primitive rateless code 전용 기법이며, NAND 표준 binary QC-LDPC의 고정 PCM 구조와 근본적으로 다른 부호 계열이라 이식 실익이 적음.
