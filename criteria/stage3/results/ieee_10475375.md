### B. 알고리즘 요약 (10줄 내외)

1. 시스템: OAM(orbital angular momentum) 기반 mode division multiplexing(MDM) LoS 채널 위의 protograph LDPC(PLDPC)-coded index modulation(IM) 시스템, rate `Rp=1/2~5/6`, codeword length `n_e=2400`.
2. 문제: UCA 정렬 오차(각도 `θ`)로 인한 inter-mode interference 때문에 채널 검출기 출력 extrinsic LLR이 비대칭·비가우시안(skewed) 분포를 가져, 기존 PEXIT/EXIT 이론 도구를 그대로 적용할 수 없음.
3. 가정: 채널은 `y=Hs+n` (AWGN + OAM mode 간 간섭), LLR 분포는 이론상 `N(σ²/2, σ²)`를 가정하나 실측 시 skewed distribution으로 나타남(θ≠0일 때).
4. 핵심 기법 1단: 원 데이터 `ς`를 절대값 기반으로 변환하는 customized Box-Cox transform(CBCT, 식 15) `l^CBCT=|l|^λt·sign(l)`로 skewed LLR을 대칭 가우시안 분포로 근사.
5. 핵심 식: CBCT 변환은 원본과 부호(sign)를 보존(`l^CBCT×l>0`)하여 BP 디코딩 정확도를 해치지 않으면서 EXIT 분석/부호설계만 단순화한다는 것이 핵심 논리.
6. 핵심 기법 2단: CBCT로 얻은 대칭 가우시안 LLR을 입력으로 하는 CBCT-PEXIT 알고리즘(Algorithm 1)으로 decoding threshold를 추정, 이를 이용한 computer-search로 rate-1/2 IUP(unpunctured)·IP1/IP2(punctured) protograph 코드를 구성.
7. 부수 트릭: IP1/IP2 코드에 degree-3 VN 패턴을 반복 추가(식 24-25)해 rate `Rp=(e+1)/(e+2)`로 확장하는 rate-compatible puncturing 구조를 채택.
8. 최적화 절차: 제약 기반(VN/CN degree, element 범위) enumeration + CBCT-PEXIT decoding threshold 비교로 최적 base matrix `B_real`을 선택하는 computer-search(Algorithm 1).
9. 결과: 4×4 OAM-IM 시스템에서 IUP/IP1/IP2가 regular-(3,6), AR4JA 대비 BER=10⁻⁵에서 약 0.1~0.3dB 이득, shaping index 적용 시 추가로 약 2.2~2.5dB 이득(spectral efficiency 손실 0.5bpcu, 약 6~8%).
10. 한계: 시뮬레이션(이론 EXIT 분석 + Monte-Carlo BER)만 수행되어 HW 미설계이며, 부호 구성법과 임계값 예측 도구(CBCT-PEXIT) 자체가 OAM/LoS 채널의 비가우시안 LLR 특성에 전적으로 종속됨.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| protograph 기반 base matrix 구성/rate-compatible puncturing | ecc_top.cpp Load_PCM() | Prime ECC도 QC-LDPC(base+circulant) 구조이나, H-matrix 고정 재구성은 이식 난이도 "높음"(프로파일 §4). CBCT-PEXIT 임계값 최적화가 OAM 채널 특화라 그대로 재사용 불가 |
| CBCT (비가우시안 LLR→대칭 가우시안 변환) | channel.cpp Set_LLR_Th() | NAND 채널(AWGN/RBER/fixed-error)은 이미 대칭적 모델을 가정하므로 OAM 특유의 inter-mode interference로 인한 skewed LLR 보정 기법은 대응 없음에 가까움 |
| shaping index modulation | 대응 없음 | OAM mode 활성화 조합을 이용한 변조 기법으로 NAND ECC 디코더/인코더 범위 밖 |
```

적용 가치: `낮음` — protograph LDPC라는 부호 종류는 Prime ECC(QC-LDPC)와 결이 다르지 않지만, 핵심 기여인 CBCT-PEXIT 최적화 도구와 shaping index가 OAM 무선채널의 비가우시안 간섭 특성에 전적으로 결합되어 있어 NAND binary LDPC로 이식할 요소가 거의 없음.
