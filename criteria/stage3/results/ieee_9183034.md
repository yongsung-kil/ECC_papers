### B. 알고리즘 요약

1. 대상 채널은 2-user Gaussian Multiple Access Channel(GMAC)로 두 사용자가 서로 다른 파워(P1, P2)로 동시에 신호를 전송, 공통 수신단에서 결합 신호 `y = √P1 x[1] + √P2 x[2] + w`를 관측.
2. 기존 연구(동일 파워 2-user GMAC EXIT 분석, 비대칭 파워로 확장한 선행연구)는 랜덤 LDPC만 다뤄, 본 논문은 빠른 인코딩/복호가 가능한 protograph 기반 LDPC로 확장하는 문제를 다룸.
3. protograph는 m×n base matrix `B`로 정의되고 각 원소를 N×N 순열/영행렬로 치환해 parity-check 행렬 H를 생성(QC-LDPC 생성 방식).
4. 수신단 joint decoding은 두 사용자의 protograph를 "state node"로 연결한 그래프에서 변수노드-체크노드-상태노드 간 LLR 메시지(`m_cv`, `m_vc`, `m_vs`, `m_sv`)를 병렬 방식으로 교환하며 최대 IterLDPC회 반복.
5. 상태-변수 메시지 `m_sv`는 두 사용자 신호의 결합확률(파워 P1,P2, 잡음분산 σ² 포함)로 계산되는 식 (2)가 핵심이며, 상대 사용자의 LLR을 조건부로 반영해 다중접속 간섭을 분리.
6. 이론 성능은 protograph-EXIT(P-EXIT) chart로 평가하며, 각 iteration에서의 상호정보량(MI) `I_sv, I_vc, I_cv`를 J-함수 기반 recursion(식 3~7)으로 갱신, 평균 MI가 1에 수렴하는 최소 Eb/N0를 디코딩 임계값으로 정의.
7. protograph 설계는 differential evolution(DE) 알고리즘으로 3×6 base matrix(원소 최대값 2)의 원소를 최적화해 디코딩 임계값을 최소화.
8. 결과: P1=1.5,P2=1 조건에서 기존 코드는 1st/2nd 사용자 임계값 4.03dB/10.13dB인데 DE 설계 코드는 2.65dB/6.50dB로 개선, P1=2 조건에서는 0.71dB/1.00dB까지 개선.
9. BER 시뮬레이션(PEG 알고리즘으로 확장계수 500의 H 생성, 단일채널 B_AWGN 코드 기준)에서 제안 코드가 기존 코드 대비 BER=10^-4에서 약 0.5dB 코딩이득.
10. 한계: 순수 이론(EXIT chart)+시뮬레이션 수준으로 HW 설계 없음, 2-user 다중접속 채널 특화 구조라 단일 채널 NAND 저장장치에는 state node/joint decoding 자체가 적용 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| protograph 기반 QC-LDPC 구성(base matrix B) | `ecc_top.cpp` `Load_PCM()` | Prime ECC도 QC-LDPC(base+circulant shift) 구조이나 본 논문의 base matrix는 다중접속용으로 재설계 필요, 직접 재사용 불가 |
| P-EXIT chart 기반 코드 설계(DE 최적화) | 대응 없음 | Prime ECC는 H-matrix 고정 로드만 지원, 코드 설계/최적화 파이프라인 없음 |
| state node 기반 joint LDPC decoding(다중접속) | 대응 없음 | Prime ECC는 단일채널 NAND 디코더로 다중사용자 개념 자체가 없음 |

> 적용 가치: 낮음 - 2-user GMAC 특화 protograph 설계 및 이론 분석으로 단일채널 NAND binary LDPC 이식에는 구조적으로 부적합.
