### B. 알고리즘 요약 (10줄 내외)

1. 대상: BCH/polar/turbo/LDPC 등 short block code(n=31~64) decoding을 위한 transformer 기반 신경망 디코더(ECCT)의 확장 아키텍처.
2. 문제: 기존 ECCT는 단일 PCM에서 유도한 mask matrix 하나만 사용해 코드워드 비트 간 관계의 다양성을 충분히 포착하지 못함.
3. 채널 모델: BPSK 변조 후 AWGN(`y=xs+z`), syndrome 기반 전처리(`y˜=[|y|,s(y)]`)로 all-zero codeword만으로 학습(overfitting 회피).
4. MM ECCT: `m`개의 서로 다른 PCM `Hi`에서 유도한 mask matrix `Mi`로 병렬 masked self-attention 블록을 구성, 각 블록 출력을 concat 후 FC layer로 결합.
5. 마스크 생성 함수 `g(H)`(Algorithm 1)는 PCM의 0이 아닌 원소 위치를 attention 허용(비마스크)으로 매핑, `softmax(QK^T/√d + g(H))V` 형태로 attention에 반영.
6. Systematic mask: PCM을 Gaussian elimination으로 systematic form `Hsys=[I_{n-k} P]`로 변환 후 마스크 생성 -> identity 부분이 sparse attention을 유도해 성능·복잡도 동시 개선.
7. Complementary mask: cyclic code 성질(`Hcp(i,j)=Hsys(i,(j-p(n-k)) mod n)`)을 이용해 identity 위치가 다른 PCM을 추가로 결합(CDM), systematic 파트의 낮은 BER 특성을 다른 비트 구간까지 확장.
8. 학습: all-zero codeword, Eb/N0 3~7dB 구간 노이즈로 Adam, 1000 epoch, cosine decay learning rate; NVIDIA RTX 3090 GPU 사용.
9. 결과: (31,16) BCH에서 MM ECCT+CDM이 ML 성능에 근접, (63,30) BCH에서 MM ECCT(m=2) 대비 BER 1자릿수 이상 개선; 단 LDPC는 PCM이 이미 sparse해 systematic mask가 오히려 attention을 밀집화시켜 개선 제한적(Table VI).
10. 한계: 전용 HW 설계 없음(GPU 시뮬만), 파라미터/FLOPs가 `m`배 증가, short block code(n≤64) 위주로 NAND ECC급 고rate·긴 length QC-LDPC와 상이한 문제 영역.
