### B. 알고리즘 요약

1. 시스템: DVB-S2 LDPC(위성방송), QPSK/8PSK/16APSK/32APSK 변조별 code rate `1/4~9/10`, 대표 code length `N=64800`(및 `(8100,4050)` irregular QC-LDPC), AWGN·Rayleigh fading 채널.
2. 문제: DVB-S2 FPGA 구현을 위해 Log-BP와 저복잡도 Min-Sum 중 적절한 알고리즘/아키텍처를 선정.
3. 모델: BPSK/QPSK 기반 soft-decision 복호, AWGN 및 Rayleigh fading 채널(등화기·rake 없음).
4. 핵심기법 1: Log-BP(sum-product) — CN 갱신 `Lj→i = 2·tanh⁻¹(∏ tanh(λk→j/2))` (식1).
5. 핵심기법 2: Min-Sum 근사 — `Lj→i = |λmin|·∏ sign(λk→j)` (식2), 비선형 tanh 제거로 HW 단순화(수 dB 성능 손실).
6. 결과(rate/BER): AWGN 6dB rate `3/5`에서 BER 0.0009216, `1/2`에서 0.0002997, `1/4`에서 0.0001426 — 저rate일수록 성능 우수.
7. 결과(채널): rate `3/5` BER 0.01 기준 SNR AWGN 4.747dB vs Rayleigh 9.769dB — Rayleigh가 실제환경에 근접하나 성능 열화.
8. 결과(MS vs Log-BP): (8100,4050) rate `1/2` AWGN 6dB에서 Log-BP BER 0.0003111 vs MS 0.03885 — MS가 열등하나 HW상 유리.
9. 결과(iteration): MS AWGN 6dB에서 iteration 2→25로 늘리면 BER 0.03885→0.006406 개선.
10. 한계: 신규 알고리즘 없음, matlab 시뮬만(FPGA는 계획 단계), gatecount/throughput 미제시, semi-parallel 설계는 미구현.
