### B. 알고리즘 요약

1. 대상은 `(J,L)` QC-LDPC 부호(circulant permutation matrix 기반), 특히 `L=3`, column-weight 3인 girth-8 부호의 exponent matrix `E(a0,...,a(J-1))` 설계.
2. 문제의식은 기존 GCDg8 그리디 알고리즘이 생성하는 수열과 Stanley sequence(3-free set의 일종)의 동치성이 미해결 문제였고, 더 큰 3-free set을 찾아야 girth-8 조건을 만족하는 더 작은 CPM-size(lower-bound `P_low`)로 더 높은 rate의 부호를 만들 수 있다는 점.
3. 핵심 가정은 채널 독립적인 조합론적 구성: GCD condition `(a_k-a_i)/gcd(a_k-a_i, a_j-a_i) ≥ L`을 만족하면 girth ≥ 8이 보장됨(기존 결과 인용).
4. 핵심 기법 1단: Theorem 1로 `L=3`일 때 GCD condition을 만족하는 수열이 정확히 3-free set(3개 원소로 등차수열 `2a_j=a_i+a_k`을 이루지 않는 집합)과 동치임을 증명.
5. 핵심 식 `Plow=(a_(J-1)-a_0)(L-1)+1`은 girth-8을 보장하는 최소 CPM-size이며, 3-free set이 클수록(같은 J에 대해 최대값 `a_(J-1)`이 작을수록) `P_low`가 작아져 더 조밀한(고rate) 부호가 가능해짐.
6. 핵심 기법 2단: Algorithm 1(랜덤 그리디로 큰 3-free set 탐색, `num1/num2` 반복으로 최적 lower-bound 선택) + Theorem 2(Stanley sequence `S(0)`의 원소를 이진표현(BR) 기반 명시적 폐형 공식으로 유도).
7. 부수 트릭: `H(D)` 형태의 convolutional PCM 표현과 대응 generator matrix `G(D)`를 유도해 LFSR 기반 인코딩이 가능함을 보임(구현 디테일, HW 미설계).
8. 최적화 절차는 Algorithm 1의 랜덤 그리디 탐색(반복 횟수 `num1,num2,num3`로 품질-복잡도 트레이드오프 조절)이며 별도 학습(DE/GA/NN) 없음.
9. 결과: 구성된 J≥5 부호들이 기존 구성법([4],[7],[13],[18]) 대비 더 나은(작은) lower-bound와 더 적은 8-cycle 수(N8)를 보이며, (2016,1008)/(2910,2330)/(98400,79950) girth-8/10 QC-LDPC 부호가 random-like, PEG, 기존 QC-LDPC 대비 BER 성능 우위를 시뮬레이션(Fig.1)으로 보임.
10. 한계는 column-weight 3(매우 저밀도) regular 구조에 국한되고, HW 설계·게이트카운트·throughput 분석은 없으며 girth 최적화 위주라 error-floor 대책이나 디코더 알고리즘 개선은 다루지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Exponent matrix / H-matrix 구성(girth-8, 3-free set 기반) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC-LDPC(column weight 17, N_b=129) 고정 구조라 저밀도(weight 3) 구성법을 그대로 적용하기 어려움(이식 난이도 높음) |
| LFSR 기반 convolutional 인코딩(`G(D)` 유도) | `encoder.cpp` `Generate_PCM_encoding()` | Prime ECC는 dual-diagonal 인코딩 고정이라 대응 없음에 가까움 |
| Girth/cycle 최적화(N8 감소)를 통한 BER 개선 | `ecc_top.cpp` `Load_PCM()` | 부호설계 단계에서 cycle 구조 개선 개념은 참고 가능하나 구체 알고리즘은 저밀도 규칙 부호 전용 |

> 적용 가치: 낮음 — column-weight 3 규칙(regular) QC-LDPC의 girth-8 최적화 구성법으로 조합론적으로는 흥미롭지만 Prime ECC의 고밀도 비규칙(irregular, degree 17) 구조와 근본적으로 다르며 특정 QC-LDPC 재검증 부담이 커 이식 난이도가 높음.
