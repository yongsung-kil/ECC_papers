### B. 알고리즘 요약

1. 대상은 5G eMBB 표준의 Polar 코드(N=256/512/1024/4096) BP 디코더이며, 성능 개선을 위해 짧은 (3,6)-regular LDPC(N˚=32~96)를 outer code로 concatenate한 Pconcat(N,K,N˚,K˚) 구조도 포함.
2. 기존 normalized Min-Sum(NMS)은 brute-force로 구한 단일 정규화 인자 α를 쓰는데, 코드 길이가 바뀌면 재탐색이 필요하고 성능 손실이 큼 — 이를 해결하고자 함.
3. Polar BP는 factor graph에 short cycle이 있어 LDPC용 density evolution을 적용할 수 없다는 가정하에, 딥러닝 기반 파라미터 최적화로 대체.
4. 핵심 기법: 2-D offset Min-Sum(2-D OMS) — right-to-left/left-to-right propagation에 서로 다른 오프셋 `βL`, `βR`을 적용해 `g(x,y) ≈ sgn(x)sgn(y)·max(min(|x|,|y|)-β, 0)` 형태로 근사.
5. Polar BP 반복 구조를 feed-forward 신경망처럼 unfold하여 BCE loss 기반 역전파+mini-batch SGD(Adam)로 `{βL, βR}` 또는 concat. OMS의 `{βL, βR, βLDPC}`를 학습.
6. 확장: Pconcat 구조에서 outer LDPC 디코딩(fully-parallel CNP/VNP, offset=0인 MS)을 R2L-L2R propagation 사이에 삽입하는 concatenated OMS 알고리즘(Algorithm 2).
7. HW 친화 디테일: 양자화 QLDPC=6bit, Qpolar=7bit, offset factor는 2 fractional bit 고정소수점(예: β=0.25)으로 단순화, G-matrix 기반 조기종료, double-column PE array 구조.
8. 학습 절차: AWGN Eb/N0=1dB/2dB에서 mini-batch(M=400) 샘플링, Adam(lr=0.01), 5-iteration unfold로 학습.
9. 결과: N=1024/4096에서 2-D OMS가 exact BP와 성능 격차 없음(0.1~0.2dB 개선), concat. OMS는 CA-SCL(L=2)에 근접한 성능. ASIC(65nm) 1332k gates, 270MHz, 637mW, N=1024에서 5.4Gb/s, N=256에서 최대 8.5Gb/s(파이프라인+조기종료).
10. 한계: 주 기여는 Polar 코드 BP 디코더이고 LDPC는 짧은 outer code(3,6-regular, N˚≤96)로 부수적 역할만 하며, NAND용 고전 binary QC-LDPC의 단독 부호설계/디코더 개선과는 거리가 있음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Offset Min-Sum CN 연산(`g(x,y)≈sgn·max(min-β,0)`) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | offset MS 근사 방식 자체는 Prime ECC의 min-sum 계열과 유사하나, 본 논문은 Polar factor graph 대상으로 직접 이식 어려움 |
| DL(역전파/SGD) 기반 오프셋 파라미터 최적화 | 대응 없음 | Prime ECC는 LLR 테이블 기반 magnitude 양자화이며 DL 학습 파이프라인 없음 |
| Outer LDPC fully-parallel MS 디코더(N˚=32~96) | `decoder.cpp` `LDPC_Decoder()` | 짧은 코드용 MS 구조로만 유사, QC-LDPC 구조·규모 상이해 직접 대응 어려움 |
| ASIC 파이프라인/조기종료(G-matrix 스킴) | `partial_CRC.cpp` `partial_crc_HW_T4()` | 조기종료 개념은 유사하나 Polar 전용 G-matrix 검증 방식이라 Prime ECC의 CRC 기반과 상이 |

> 적용 가치: 낮음 — 핵심 기여가 Polar 코드 BP 디코더(DL 최적화 offset MS)이며, binary QC-LDPC는 짧은 outer concatenation 코드로만 부수 등장해 Prime ECC 이식 가치가 제한적.
