> 총평: 고차 변조(BICM) 대상 irregular LDPC degree distribution을 EXIT+LP로 설계하는 code-design 이론 논문으로, NAND 고rate binary QC-LDPC 디코더/HW 이식성은 낮다.

### B. 알고리즘 요약

1. AWGN 상 고차 변조(M-PAM `M=4,8`, 8-PSK)용 BICM에 맞춘 irregular LDPC degree distribution `(λ,ρ)`를 설계하는 방법론.
2. 문제: BICM은 레벨별 등가 이진채널이 비대칭이라 all-zero codeword 가정의 Density Evolution(DE)을 직접 못 쓴다.
3. 해결: iid adapter([3])로 송수신 양측 동일 seed 난수와 XOR을 써서 전체 채널을 대칭화 → all-zero codeword로 DE 분석 가능.
4. 입력 LLR PDF `p0`를 심볼 alphabet·bit mapping·noise variance로부터 시뮬레이션으로 추정(식 3), PDF 비대칭·spike로 DE 정확도가 관건.
5. bit/check node별 `(i,r)`/`(l,j)`-regular ensemble DE로 EXIT 곡선상의 점(`IA,IE` 상호정보)을 계산하고 선형보간으로 전체 EXIT 곡선 구성.
6. 수렴조건: EXIT chart에서 bit-node 곡선이 check-node 곡선 위에 있어야 하며, 최소 간격 `δ`(=0.001)를 강제.
7. rate 최대화는 `Σ λi/i` 최대화이며, EXIT 조건 `Σ λi IE,V(x) ≥ min(Σ ρj IA,C(x)+δ, 1)`가 `λi`에 선형이므로 선형계획(LP)으로 풀린다.
8. check degree 분포 `ρ(x)`는 연속 2항으로 고정한 뒤 `ρj`를 스캔, 각 경우 LP로 최적 `λ(x)` 탐색; right-regular는 절차가 더 단순.
9. 결과: 4-PAM rate 0.5003 threshold 2.62dB(PID capacity 0.35dB 이내), 8-PSK rate 0.78(기존 [5] rate 2/3보다 높음), block length 20k~25k 시뮬로 threshold 근접 확인.
10. 한계: EXIT 곡선 보간 오차로 [3] 대비 BER 다소 열세, HW 미설계, 무선 고차 변조 전제라 NAND 비변조 채널과 직접 무관.
