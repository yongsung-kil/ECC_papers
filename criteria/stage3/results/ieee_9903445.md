### B. 알고리즘 요약

1. 시스템: 선형 ISI + AWGN 채널 `y=Hc̆+w`, 심볼 검출(equalization), 실험은 BPSK/16-QAM, block `K=500`, 표준 ISI 채널 CA/CB/CC.
2. 문제: 최적 MAP(BCJR) 복잡도가 채널 memory `L`·심볼당 비트 `m`에 지수적; 저복잡도 factor-graph 검출(UFG)은 그래프 cycle 때문에 suboptimal·error-floor.
3. 모델: Ungerboeck observation model `G=HᴴH, x=Hᴴy`로 APP를 factor `Fk`·`Ik,ℓ`(degree-2)로 인수분해(13), 복잡도 `L`에 선형.
4. 핵심 기법1 (NBP+FN 강화): SPA를 layer로 unfold, VN↔FN 메시지에 학습 가중 `w_v,w_f` + FN 내부 곱 가중 `κ,λ` 도입해 cycle의 overconfidence(`1/σ²`) 억제.
5. 핵심 식: 일반화 FN `F̃k`,`Ĩk,ℓ`에서 `κ1`이 `1/σ²`를 감쇠 → SPA extrinsic 위반 완화(파라미터 `|PNBP|=NK(5L+3)`).
6. 핵심 기법2 (GAP, 동적 factor-graph 전이): FIR 전처리 `x=Py`로 observation model 자체를 매 stage마다 바꿔 cycle 반복 회피, `S` stage × `B` 병렬 branch 결합(14).
7. 구현 디테일: log-domain SPA(`max*` Jacobian), 전처리 `P=P̃Hᴴ` 구조로 채널 일반화 확보, degree-2 FN으로 복잡도 `O(SBNKLM²)`.
8. 학습: end-to-end, Adam+backprop, 목적함수는 BMI(bit-metric decoding 기반 achievable rate)와 multiloss(16), `Eb/N0` 범위 샘플링으로 과적합 완화.
9. 결과: 채널 CB 16-QAM에서 GAP가 GFG 대비 +0.95, LMMSE 대비 +1.75 bit/channel-use BMI 이득으로 MAP 근접; shortened-BCJR 대비 복잡도 10배↓(X=68825 vs 851968).
10. 한계: 순수 시뮬(HW 미설계), 통신 ISI 채널 전제, LDPC/ECC 디코딩과 무관, 특정 `Eb/N0` 훈련 의존.

| NBP/FN 가중·deep unfolding 신경 강화 | 대응 없음 | 낮음 — NN/딥러닝 기반, 프로파일상 명시적 "대응 없음" |
