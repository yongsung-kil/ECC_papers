### B. 알고리즘 요약

1. 시스템: binary QC-LDPC (GC-LDPC), 예제 rate 0.6875~20/21, length 14976~16272bit, CPM 크기 126~226, BPSK, MSA 디코더(max iter 50).
2. 문제: 기존 GC-LDPC는 유한체 기반 구성·시뮬만 있고, 구조와 반복복호 threshold의 관계·파라미터 선택법이 부재.
3. 채널 모델: AWGN에 burst-erasure를 결합한 BuEC-G(Random 또는 Gilbert-Elliott 소거 모델), 소거 시 LLR을 0으로 처리.
4. 핵심기법1: `global edge spreading` — 여러 disjoint block protograph의 local CN edge 일부를 global CN(`c_gp,k`)으로 spread해 GC protograph 생성 (cascaded=type-1, interleaved=type-2).
5. 핵심식: 설계율 `Rd = 1 - m/n - s/nt`(type-1), `1 - 1/2l - s/2lft`(type-2)로 파라미터(m,n,t,s)/(l,f,t,s)가 rate·length를 결정.
6. 확장: type-2는 inter-column permutation(식 5)으로 irregular protograph를 interleaved GC 구조로 재배치.
7. 분석도구: GA와 P-EXIT를 BuEC-G로 일반화 (소거확률 `ε_ch`를 MI/mean 갱신식에 삽입, 식 6~21), 안정성 조건·capacity `C_GEC-G=(1-ε_ch)C_AWGN` 증명.
8. 최적화 절차: 주어진 rate/length 범위에서 모든 파라미터를 열거해 capacity-threshold 가중 gap `W_GA/P-EXIT`를 최소화하는 spreading 형태 탐색.
9. 결과: 제안 부호가 기존 well-designed GC-QC-LDPC보다 threshold 우수, GEC-G에서 BER 10^-7까지 error-floor 없음, capacity 대비 약 1.0~1.2dB.
10. 한계: HW 미설계·시뮬만, 타깃이 자기/광 기록의 burst-erasure 채널로 NAND flash와 무관, non-Gaussian/impulse noise는 미논의.
