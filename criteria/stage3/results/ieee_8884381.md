### B. 알고리즘 요약

1. 시스템: moderate length MET QC-LDPC 코드, 정보길이 600 / codeword 1800 bit, rate 1/3, AWGN + QPSK, SPA 50 iteration 복호.
2. 문제: Scattering PEXIT-chart(K<200)와 Covariance Evolution(rate<0.9, K≈200~600) 사이 length 구간을 커버하는 finite-length MET protograph 구성법 부재.
3. 모델: CPM `P^k`(circulant permutation) 블록 행렬 `H`, exponent matrix `E(H)`, base graph `M(H)`, exponent chain이 `Σ(-1)^k a_k ≡ 0 mod L`(식1)이면 Tanner graph에 cycle 형성.
4. 핵심기법: degree distribution/greedy로 protograph 후보 ensemble 생성 후 balanced cycle 등 graph 특성으로 다단 sieving (Alg.1).
5. 핵심식: code distance 상한 `dmin(C(H)) ≤ min Σ wt(perm(H_S\i))` (Vontobel-Smarandache Th.7 / Butler-Siegel Th.12) — permanent 기반 최소거리 상한으로 후보 걸러냄.
6. 확장: balanced cycle(각 셀이 even/odd step 동수 참여)은 식1을 `0=0`으로 만들어 girth 상한 결정, girth 8 목표로 code distance 확보.
7. 트릭: trapping set TS(5,3)/TS(4,4) 등 harmful cycle 파괴로 pseudo-codeword weight spectrum 개선, EMD metric으로 cycle 연결강도 평가.
8. 최적화: union bound `PUB≈mult_dmin·Q(√dmin)`로 후보 평가, simulated annealing으로 girth 8·EMD 10·code distance 68 protograph lifting.
9. 결과: 제안 Code A가 5G eMBB BG2(Code B) 대비 BLER 10^-6에서 0.2 dB coding gain, decoding threshold SNR=-1.57 (BER=10^-8).
10. 한계: 시뮬레이션(computer simulation)만, HW 미설계, rate 1/3 저rate·MET/5G 특화, circulant 60 lifting 예제 중심.
