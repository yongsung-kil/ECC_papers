### B. 알고리즘 요약 (10줄 내외)

1. NAND flash TLC 채널(RTN/CCI/retention noise 컨볼루션 모델)을 대상으로 4kB급 (37536, 33672) rate `R=0.9` QC-LDPC 코드(SC–GC-LDPC)를 구성.
2. 기존 GC-LDPC(2kB/4kB)는 코드 길이가 최신 NAND 요구를 못 따라가고, 글로벌 커플링 구조 때문에 인코딩 복잡도 `O(N^2)`가 높다는 문제.
3. 채널은 erased/programming state의 Gaussian 근사 `p(x)`에 RTN(exponential), CCI(선형 결합), retention noise(Gaussian) 컨볼루션 `P(x)=p(x)⊗pr(x)⊗F⊗pt(x)`로 모델링.
4. 핵심 기법은 tail-biting SC 구조를 global matrix 구성에 도입(로컬 매트릭스는 대각 배치, 글로벌 체크는 finite field `GF(139)` 기반 shift matrix `S`로 CPM 확장)해 국소부호 4개(local code)를 갖는 SC–GC-LDPC 완성.
5. 인코딩은 `P1=LS`, `P2=GS+G'P1` 형태의 희소 행렬곱으로 처리되어 전통 GC-LDPC의 Gaussian elimination(`O(N^2)`) 대비 `O(N)`으로 감소.
6. 디코딩은 두 단계: 개선 2-레벨(ITLA, level1 로컬→level2 글로벌 순차)과 개선 2-페이즈(ITPA, 로컬↔글로벌 phase 순환)로, 로컬 부호 1개만 실패 시 인접 로컬부호와 관련된 global matrix 일부(부분 행)만 사용해 정정.
7. HW 친화 트릭: 로컬 디코딩 첫 라운드 이후엔 성공한 로컬부호를 재복호하지 않고 LLR 전달만 수행, 실패한 로컬부호에는 인접 로컬부호 관련 global matrix 부분행만 순회(전체 global matrix 대신).
8. 최적화 절차는 없음(수동 파라미터 선택: layered SPA/NMS, NMS 정규화계수 0.7, `(TL,TG)`=(20,10), `(TL,TG,Imax)`=(3,2,7) 등 그리드 튜닝).
9. 결과: AWGN에서 SPA 0.065dB, NMS 0.08dB 성능이득(대비 GC-LDPC-4kB); NAND flash 채널에서 RBER 기준 약 25% 성능개선; ITLA/ITPA는 단일 부블록 오류 시 CNU를 6.64~31.14% 감소(성능 저하는 소폭).
10. 한계: HW(RTL/ASIC) 미설계, 실 SSD 실측이 아닌 NanoCycler 실측 임계전압 분포 기반 채널 모델 시뮬레이션, 고정 max iteration(적응형 iteration은 향후 과제로 언급만 됨).

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| ITLA/ITPA 로컬-글로벌 2단계 스케줄링 | decoder.cpp LDPC_Decoder(), C2V_Cal() | 이터레이션 루프 내 부분 check matrix만 순회하는 스케줄링 개선으로 min-sum 이터레이션 루프에 직접 이식 가능성 있음 |
| CNU 감소(부분 global matrix 활용) | decoder.cpp CNU_Update_New_Mag() | CN 업데이트 대상을 서브셋으로 제한하는 아이디어가 CNU 연산 축소에 대응 |
| SC–GC-LDPC 부호 구성(finite field 기반 shift matrix, tail-biting) | ecc_top.cpp Load_PCM() | 특정 QC-LDPC 고정 구조라 이식 난이도 높음(프로파일 §4 기준) |
| 인코딩(희소 행렬곱 O(N) 인코딩) | encoder.cpp LDPC_encoder_4KB(), Generate_PCM_encoding() | dual-diagonal 고정 구조와 다른 인코딩 방식이라 중~높음 난이도 |
| layered SPA/NMS + 정규화계수 0.7 | decoder.cpp CNU_Update_New_Mag() | Min-Sum 정규화 계수는 이미 보유 기법과 중복(§3), 신규 기여 아님 |
```

> 적용 가치: 높음 — NAND flash 4kB급 QC-LDPC 부호설계와 로컬/글로벌 부분 스케줄링으로 CNU(연산량)를 실질적으로 줄이는 디코더 알고리즘을 직접 제안해, Prime ECC의 이터레이션 루프·CNU 스케줄링에 낮은~중간 난이도로 이식 가능.
