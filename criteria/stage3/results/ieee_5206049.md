### B. 알고리즘 요약 (10줄 내외)

1. 시스템: MT×MR MIMO-BICM(bit-interleaved coded modulation) 무선통신 시스템, max-log demodulator가 code bit별 LLR `Λ_l = (1/σ²)(min_{x∈X_l^0}‖y-Hx‖² - min_{x∈X_l^1}‖y-Hx‖²)` 계산.
2. 문제의식: 실제 구현은 유한 비트 LLR 양자화가 불가피한데, 기존 최적 양자화([2])는 AWGN+BPSK 특수 케이스에만 해석적으로 풀려 일반 채널/변조로 확장이 어려움.
3. 핵심 가정: 양자화된 LLR을 이진입력 K진출력의 등가 이산채널(equivalent discrete channel)로 모델링, crossover 확률 `p_bk = Pr{d=λ_k|c=b} = ∫_{I_k} f_{Λ|c}(ξ|b)dξ`.
4. 핵심 기법: `I(c;d) ≤ I(c;Λ)`(data processing inequality)이므로 직접 최적화 대신 `I(Λ;d) = H(d)`를 최대화 - 즉 양자화 출력 `d`가 균등분포(equiprobable)가 되도록 경계 `i_k* = F_Λ^{-1}(k/K)`를 설정(누적분포함수 역함수).
5. 핵심 식: 양자화 레벨(대표 LLR값)은 `λ_k* = log(p1_k/p0_k)`로 설정해 디코더(BP)에 정확한 신뢰도 정보를 전달 - 경계는 capacity에만 영향, 레벨값은 디코더 성능에 영향.
6. 확장: SISO-BPSK-Rayleigh의 경우 LLR 분포를 카이제곱 분포 조합으로 닫힌형 유도(Bessel 함수 `K0`); MIMO/고차변조는 해석식이 어려워 Monte-Carlo(10^5 채널 realization)로 경험적 분포 추정.
7. 부수 트릭: 실전 온라인(on-the-fly) 파라미터 추정을 위해 `f_{Λ|c}`를 2-parameter 지수 혼합모델(`αβ/(α+β)·exp(αξ)` 등)로 근사, 2개 bin의 관측확률로 α,β를 수치적으로 역산.
8. 학습/최적화 절차: 없음(폐형 경계식 + 파라미터 fitting).
9. 결과: rate-1/2, length-64000 regular LDPC 검증에서 SISO-BPSK 1-bit/2-bit/3-bit 양자화가 무양자화 대비 각각 6.2dB/1.1dB/0.4dB 손실; MIMO-16QAM은 0.6dB(2,3-bit)/1dB(1-bit) 손실; 1-bit 양자화 시 레벨 선택에 BER이 매우 민감(부적절 시 2자릿수 열화).
10. 한계: MIMO/Rayleigh 페이딩 무선 채널 전용, LDPC는 코드 자체 설계·복호 알고리즘 개선이 아닌 검증용 채널코드로만 사용, HW 구현/양자화 회로 설계는 다루지 않음(계산 방법 자체도 "out of scope"로 명시).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Equiprobable 출력 기반 LLR 양자화 경계 설계 | `channel.cpp` `Set_LLR_Th()` | soft-read LLR threshold 설계라는 목적은 유사하나, 대상 채널이 MIMO Rayleigh 페이딩이라 NAND read(전하분포 기반) 채널모델과 근본적으로 달라 파라미터·분포 유도식 자체는 이식 불가, 설계 "철학"만 참고 가능 |
| 양자화 레벨(대표 LLR)을 log-likelihood ratio로 설정 | `channel.cpp` `Set_R_Offset()`, `ecc_data.h` `PARAM_LLR` | LLR 테이블 값 설계라는 개념은 공통이나 계산 근거(무선 페이딩 통계)가 NAND와 무관 |
| 온라인 파라미터 추정(2-parameter 지수모델 fitting) | 대응 없음 | 무선 채널 통계 실시간 추정 기법으로 NAND 컨트롤러의 오프라인 LLR 테이블 방식과 목적이 다름 |

> 적용 가치: 낮음 — MIMO-BICM 무선 페이딩 채널의 soft-demodulator 양자화 설계로, LDPC는 검증용일 뿐이며 채널 모델·목적 모두 NAND ECC와 거리가 멀다.
