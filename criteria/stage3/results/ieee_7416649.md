### B. 알고리즘 요약

1. 시스템: 2bit/cell MLC NAND, 4개 write-level(`Vmin,V1,V2,Vmax`)+3~6개 read-level. binary LDPC 두 종: 4K-code(irregular, rate 0.90, `4096/4544`), 8K-code(regular col-weight 4, rate 0.92, `7360/8000`), PEG 구성, shuffled BP 복호(`Imax=25`).
2. 문제: MLC 채널이 PE cycle·retention에 따라 비정상(non-stationary)이라 고정 전압·균일 양자화는 error-rate·복호수렴이 나쁨.
3. 채널모델: programming noise + RTN(`σn=0.00025(PE)^0.62`) + retention noise + cell-to-cell interference(CCI)를 convolution으로 합성해 4개 심볼 분포 도출.
4. write 최적화: 채널 오류확률 `Pe`(식15)를 목적함수로 gradient-descent(식21)로 `V1*,V2*` 탐색, PE별 LUT 저장(오프라인).
5. `Pe = (1/4)ΣP(e|sp)` - 인접 분포 교점(`R1,R2,R3`)을 decision boundary로 심볼오류 최소화.
6. read 양자화: voltage entropy 함수 `H(v)`(식24) 도입, `H(Rn)=θ`로 erasure 영역 폭을 파라미터 `θ`로 제어(controlled erasure).
7. 트릭: erasure 영역 LLR을 0으로 리셋(가장 불확실) → 심볼오류↓ 대신 erasure확률↑ 트레이드오프, MMI 양자화 대비 유리(`θ=0.35`가 최적).
8. LLR: `Rn-1≤v≤Rn` 구간별 MSB/LSB LLR을 분포 적분비 log로 계산(식22,23).
9. 결과: 제안 entropy 양자화가 MMI·uniform 대비 FER 우수(12-level uniform보다 6-level non-uniform이 더 좋음), 복호 latency 초기 16%·EOL 22% 감소, retention 6/12/24개월서도 우수. EXIT 곡선으로 최소 read-level 선택(14K PE 이후 3→6level).
10. 한계: HW 미설계(모델기반 시뮬만), θ 최적값의 해석적 해는 future work, 디코더는 표준 BP.
