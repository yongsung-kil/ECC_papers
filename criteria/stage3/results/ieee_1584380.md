### B. 알고리즘 요약

1. 대상은 BEC 및 BIAWGNC 채널에서 length-1000~10000의 irregular LDPC ensemble, degree distribution `λ(x)`(bit node), `ρ(x)`(check node)로 정의.
2. 문제: threshold 최적화(density evolution)만으로 degree distribution을 설계하면 finite-length에서 error floor 성능이 매우 나빠짐 — 두 목표가 상충.
3. 핵심 모델: BEC에서 degree-2 bit node로 구성된 서브그래프 `T2`의 stopping set(=cycle)이 error floor를 지배한다고 가정, girth `g`인 코드는 크기 `g/2` stopping set이 지배적.
4. 핵심 기법: word error rate 근사식 `Wg ≈ (λ2ρ'(1))^(g/2) / g` (BEC), BIAWGNC에서는 `Wg ≈ e^(-g/4σ²)(λ2ρ'(1))^(g/2)/g`를 유도.
5. 이 식이 필요한 이유: λ2(2차 bit node 비율)가 stopping set/저중량 codeword 확률을 지수적으로 좌우하므로, λ2를 제한하면 error floor를 직접 제어 가능.
6. 확장 기법: 기존 stability constraint(`λ2 ≤ ρ'(1)^-1 · ε*` 등)를 스케일 인자 `E`로 조정(`E=(g·Wg)^(2/g)`)해 목표 WER floor `Wg`를 만족하는 λ2 상한을 도출, `E=1`이면 기존 threshold 최적화와 동일.
7. 구현 디테일: differential evolution으로 제약 조건 하 최적 degree distribution 탐색, BIAWGNC threshold는 Gaussian approximation density evolution으로 계산.
8. 학습/최적화 절차: differential evolution(GA 계열) 기반 degree distribution 탐색, 제약 `E` 값을 바꿔가며 threshold-floor trade-off 곡선 생성.
9. 결과: rate-1/2 BEC/BIAWGNC 코드에서 `E`를 1→0.02로 낮추면 threshold 손실은 작지만 error floor(BER)가 최대 4자릿수(10^-4 order) 개선.
10. 한계: 무한 길이 ensemble 기반 이론/시뮬레이션뿐, HW 미설계, degree-2 노드 중심 근사(고차 stopping set 무시), QC-LDPC 구조 특화 설계는 다루지 않음.
