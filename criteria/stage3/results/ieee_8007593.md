### B. 알고리즘 요약

1. 시스템: 초고밀도 자기기록(BPMR/SW/TDMR)의 2D-ISI 채널, rate 8/9 LDPC, code length `Nc = 4608, 9216`, 기록밀도 4Tb/in².
2. 문제: 2D-ISI 최적화 LDPC는 특정 2D 검출기(2D-DET)에 종속되어 검출기가 바뀌면 최적화를 재수행해야 하고, 2D-DET의 EXIT 곡선은 폐형식이 없어 Monte Carlo로만 평가 가능.
3. 핵심 가정: 2D-DET 출력 L-value 분포를 대칭 가우시안 `N(μ, 2μ)`으로 근사 가능(대칭 조건).
4. 핵심 기법: 특정 2D-DET의 a-posteriori 상호정보량(mutual information)을 계산하고, 이를 이용해 L-value 분포를 `N(μ, 2μ)`으로 재모델링.
5. 핵심 식 의미: 등가분산 `2μ`가 ISI 채널과 AWGN 채널 사이의 "다리" 역할을 하여, AWGN용 코드 최적화 기법을 그대로 재사용 가능하게 함.
6. 확장: 원래 분산 대신 2D-DET에서 유도한 새 L-value 분산을 EXIT 차트에 대입하여 차수분포(degree sequence) 최적화.
7. 구현 디테일: 4Tb/in² BCJR 2D-DET에서 전송 코드비트 0/1의 조건부 PDF가 재모델링된 대칭 가우시안과 잘 일치함을 Monte Carlo로 확인.
8. 결과: 제안 코드가 AWGN 최적화 코드(Scheme 3) 대비 BER 10⁻⁵에서 약 0.2 dB 이득, 두 길이 모두. 완전 2D-ISI 최적화 코드보다 error floor가 낮아 고SNR에서 우세.
9. 결과2: 복잡도가 훨씬 높은 최적화 방식(Scheme 2) 대비 손실은 작으면서 더 낮은 error floor.
10. 한계: HW 미설계, 시뮬레이션만, 자기기록 2D-ISI 채널 및 특정 2D-DET 가정에 강하게 종속.
