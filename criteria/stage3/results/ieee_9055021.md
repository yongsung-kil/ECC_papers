### B. 알고리즘 요약

1. MLC NAND (2bit/cell), Quasi-Cyclic LDPC 사용, code length 2KB, column weight 4, row weight 36, code rate 8/9, page size 16KB.
2. Program interference(hot data)와 retention error(cold data)가 서로 반대 방향으로 Vth를 이동시켜 상쇄 효과가 있음에도 기존 전처리 기법(DPA/AC/NRC)은 이를 고려하지 않아 RBER 감소가 최적이 아니라는 문제의식.
3. 채널모델: erased state는 Gaussian(`p_e(x)`), program state는 ISPP 균등분포, program interference는 RTN+cell-to-cell interference(`V_CCI`), retention error는 `N(µ_d, σ_d²)` (PE cycle, retention time 종속).
4. 1단계(쓰기): CeSR - 데이터 세그먼트를 hot/cold × 0-dominant/1-dominant(H0/H1/C0/C1)로 분류 후 각기 다른 비트 플리핑 규칙 적용, hot data는 `11>10/00>01`, cold data는 `10>11/00>01` 분포를 목표로 state 재배치.
5. 2단계(읽기): Assisted LDPC - LSB가 먼저 복호되므로 그 결과(flip 여부, HD 값)를 MSB 복호의 초기 LLR/HD에 반영. 예: LSB flip 검출 시 MSB의 LLR을 큰 양수(=100)로 설정, LSB=1이면 대응 MSB의 LLR을 `1.85×LLR`로 강화, HD를 `±3`만큼 보정.
6. 핵심 근거: NAND 셀 상태 전이는 무작위가 아니라 hot data는 좌→우(전하감소→증가), cold data는 우→좌(전하누설) 방향으로 편향되므로, 이 통계적 상관관계를 LLR/HD 초기값 보정에 활용해 check equation 수렴을 가속.
7. hot/cold 데이터별로 각 4개/3개의 관측 규칙(observation)을 정의해 Table 4/5의 조건-보정값 매핑표로 구현, 알고리즘 자체 추가 계산 오버헤드는 거의 없음(단순 초기값 대입).
8. 학습/최적화 절차 없음 — 관측 기반 휴리스틱 규칙(고정 보정값)이며 NAND 오류모델 파라미터는 문헌값[13][17] 사용.
9. 결과: CeSR은 NRC 대비 RBER을 hot 20.30%, cold 85.13% 감소. Assisted LDPC는 cold data 복호성공률 최대 97%p 향상(2%→99%), 평균 iteration을 hot 31.6%, cold 73.9% 감소. Assisted LDPC 자체는 추가 연산 오버헤드 없음.
10. 한계: MATLAB 시뮬레이션뿐 HW 미설계, hot data는 LSB/MSB 동시 버퍼 유지 필요(최대 63페이지 갭, 32 open block 기준 31.5MB 추가 버퍼) 및 MSB 읽기 시 LSB 선행 read로 latency 약 7% 증가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LSB/MSB 상관관계 기반 초기 LLR/HD 보정 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | soft-read LLR 양자화/threshold 설계에 초기값 보정 로직으로 결합 가능, NAND 페이지 간 상관정보 활용은 신규 요소 |
| MSB 복호 시 LSB 결과 반영 (조건부 HD 보정) | `decoder.cpp` `VN_Cal_HD()`, `Get_VNU_Table_Idx()` | VN 초기화 단계에 조건부 보정 로직 삽입 가능, LSB/MSB 페이지 순서 의존성은 시스템 레벨 스케줄링 변경 필요 |
| CeSR 데이터 전처리(cell state remapping) | 대응 없음 | ECC 외부의 데이터 전처리 단계로 Prime ECC decoder/encoder 범위 밖 |
| iteration 수 감소(정확한 초기정보로 조기 수렴) | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프, `partial_CRC.cpp` `partial_crc_HW_T4()` | 초기 LLR 정확도 향상이 조기종료(partial CRC) 도달을 가속하는 효과와 상호보완적 |

> 적용 가치: 중간 — NAND 직접 타깃이며 LSB/MSB 상관관계를 이용한 LLR/HD 보정은 decoder의 초기정보 단계에 비교적 낮은 난이도로 이식 가능하나, CeSR 전처리와의 결합·페이지 순서 의존성 등 시스템 레벨 전제조건이 있어 순수 decoder 모듈만 분리 이식하기는 다소 제약적.
