### B. 알고리즘 요약

1. 대상은 고차 QAM(64/256/1024-QAM) 변조를 사용하는 2-level MLCM(multilevel coded modulation) 시스템으로, 각 code block(CB)은 5G-NR 호환 SC-LDPC 성분 부호(component code)로 구성됨.
2. 기존 5G-NR BICM(bit-interleaved coded modulation)은 고차 변조에서 복잡도-처리량 균형이 어렵다는 문제를 해결하고자 함.
3. 채널은 AWGN(`y = x + n`)이며 변조 심볼의 bit-level 신뢰도(HB/LB) 차이를 이용해 서로 다른 rate의 성분 부호를 매핑.
4. 핵심 기법은 SPC(single parity check)-aided SC-LDPC(SSC-LDPC): 여러 transport block(TB) 내 마지막 CB의 메시지 벡터를 `b_{T-1} = b_0 ⊕ b_1 ⊕ ... ⊕ b_{T-2}`로 XOR 구성해 블록 간 결합 중복성을 추가, error floor 저감.
5. 두 성분 부호 간 결합 서브행렬 `B_{i,i+1}`은 simplified EMPEG 알고리즘으로 최적화된 exponent matrix `P_{i,i+1}`을 사용해 circulant permutation 구조를 형성.
6. 부수 기법은 Approximate-Min* BP 디코더로 완전 BP 대비 낮은 복잡도의 근사 복호를 수행하며, 1단 복호 성공 시 2단에서 서브행렬(`H_{0,1}`)을 제외해 복잡도를 절감.
7. 학습/최적화 절차는 수정 PEXIT(protograph EXIT) 분석: BER 매칭 기반으로 각 bit level의 등가 SNR/MI를 계산해 posteriori MI가 1에 도달하는 최소 `(Eb/N0)*`를 threshold로 정의, 메시지 길이 비율 `K0/K1`을 스윕해 threshold를 최소화하는 배분 선택.
8. "LB-aided HB" 매핑이 일관되게 최저 threshold를 달성(예: 64-QAM에서 BICM 대비 0.3dB, 256-QAM 0.5dB, 1024-QAM 최대 0.4dB 개선).
9. 결과: BLER=10^-3 기준 BICM 대비 최대 0.5dB 코딩이득, check-node "⊞" 연산 수 기준 최대 34.0% 복잡도 감소(64-QAM, SPC 적용 시).
10. 한계: 순수 시뮬레이션 검증(HW 미설계)이며, 고차 QAM multilevel 변조·5G-NR 표준 base graph에 강하게 종속된 설계로 NAND의 이진/soft-read 채널과 직접 호환 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SPC 기반 블록 간(TB-level) XOR 결합으로 error floor 저감 | `decoder.cpp` `LDPC_Decoder()` | 개념적으로 다중 코드워드 간 중복성 추가 아이디어이나, Prime ECC는 단일 코드워드 단위 복호이며 TB 개념·XOR 결합 없음 |
| 수정 PEXIT 분석(메시지 길이 배분 최적화) | `ecc_top.cpp` `Load_PCM()` | H-matrix 설계/threshold 예측 도구이나 QAM multilevel 변조 전제이므로 NAND의 hard/soft read 채널과 무관 |
| Approximate-Min* BP 디코더 | `decoder.cpp` `CNU_Update_New_Mag()` | Min-Sum 근사와 유사 계열이나 이미 Prime ECC가 Min-Sum을 보유(§3), 중복 기여 |

> 적용 가치: 낮음 — 고차 QAM MLCM 변조 특화 설계로 Prime ECC의 NAND binary QC-LDPC 단일 코드워드 구조와 근본적으로 다르고, HW 레이어 기여 없음.
