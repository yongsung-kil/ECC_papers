### B. 알고리즘 요약

1. 시스템: binary 2-write WOM(1→0 불가 제약), n 셀에 write당 k-bit 저장, read 오류는 bit-flip 확률 `p`인 BSC로 모델링.
2. 문제: 2-write WOM 용량영역(무잡음 `R<1-δ`, read오류 `R<1-δ-h(p)`)을 저복잡도 message-passing으로 달성 — 기존 polar/최적화 irregular LDGM 대비 실용 인코딩·디코딩·오류정정 동시 확보.
3. 핵심 부호: compound LDGM/LDPC (Hsu-Anastasopoulos) 부호 `(dv,dc,dv',dc')` — 상단 LDGM이 codeword↔정보비트, 하단 LDPC가 parity 제약. parity 그룹 `P1`(메시지 운반)과 `P2`(오류정정)로 coset 분해.
4. 핵심 관찰: 2번째 write 인코딩은 binary erasure quantization(BEQ) 문제로 환원 → 선형복잡도 iterative quantization algorithm으로 코드워드 탐색.
5. iterative quantization(Algorithm 1): peeling decoder 쌍대 — degree-one LDPC bit-node를 유일 check-node와 pairing 후 제거, 역순으로 paired parity 만족하게 값 설정.
6. 확장: regular degree에서는 degree-one bit-node 부족으로 실패 → spatial coupling의 boundary termination이 필요한 degree-one 노드를 자연 공급(irregular 설계 불요).
7. 구현 디테일: chain length `L`, coupling window `w`로 SC 구조 구성, 첫 `w-1` section LDPC bit-node를 0으로 shortening해 encoding/decoding wave 시작 + threshold saturation 유도.
8. read-error 처리: 메시지 복호는 `P1` 제외한 코드북 `C`로 BSC channel MAP 복호, degree profile로 정정 가능한 최대 `p*=h^{-1}(1-code-rate)` 계산.
9. 결과: L=30, single blocklength≈24000(유효 720000)에서 무잡음 threshold δ가 최적값에 근접(w=3~4), blocklength 1200 SC(3,3,3,6)에서 δ<0.43까지 10^5 인코딩 무실패 — polar 대비 우수 시사(단 10% rate loss).
10. 한계: HW 미설계, 시뮬만, 거대 blocklength 필요, first write와 weight 제약 second write는 BPGD로도 미해결(open).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| compound LDGM/LDPC 부호구조 / SC 구성 | `ecc_top.cpp` `Load_PCM()` (부분) | Prime ECC는 고정 QC-LDPC H-matrix 로드이며 LDGM+SC compound 구조 미지원, 재설계 부담 큼 |
| iterative quantization(peeling) 인코딩 | `encoder.cpp` `LDPC_encoder_4KB()` (부분) | Prime ECC 인코더는 dual-diagonal 고정, WOM용 BEQ peeling 인코딩과 목적·알고리즘 상이 |
| BSC read-error 채널복호(message-passing) | `decoder.cpp` `LDPC_Decoder()` | 표준 BP 채널복호로 min-sum과 개념 근접하나 논문이 제공하는 신규 디코더 기법 없음 |
| WOM constraint / rewrite | 대응 없음 | Prime ECC는 재기입 제약 미모델링, NAND 오류정정 전용 |

적용 가치: **낮음** — WOM 재기입 용량달성용 SC compound 부호설계로, NAND 고전 binary QC-LDPC 오류정정 디코더/HW에 떼어 쓸 신규 기법이 없고 부호구조·blocklength·문제정의가 모두 상이함.
