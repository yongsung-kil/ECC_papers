### B. 알고리즘 요약

1. 대상 부호는 NAND용 (18396,16416) QC-EG-LDPC(유클리드기하 기반), rate ~0.892, 2KB 유저데이터+4B CRC(K=16416), CPM(circulant) size=511, dv=8/dc=72를 dual-circulant로 분해해 dv=4/dc=36 구조로 확장.
2. 문제: 대부분 QC-LDPC 디코더는 라우팅 혼잡 때문에 작은 CPM을 쓰는데, large-CPM(511) EG-LDPC를 써도 고throughput 디코더가 가능한지를 실증.
3. 모델: Bi-AWGN 채널, RBER 1% 근방을 3D TLC/QLC end-of-life 기준으로 삼고 UBER<1e-10, error-floor 없음 목표.
4. 핵심기법1: normalized min-sum(NMS, Algo 1) — VN에서 `Lpost=Lch+ΣLc2v`, CN에서 min1/min2+global sign, `Mag=α×min`으로 C2V 생성, syndrome 조기종료.
5. 핵심식: `Lv2c(i,j)=Lpost(i)-Lc2v(j,i)`(extrinsic), scaling `α=0.75`로 min-sum 과대추정 보정.
6. 핵심기법2: column-shuffle 스케줄링(36 cycle/iter)과 block-parallel layer(72 cycle/iter) 정량 비교 후 shuffle 채택, array-dispersion 패턴을 활용한 Dual VNU Cluster로 병렬도 2배.
7. 구현 디테일: 입력 2-bit soft, 노드간 메시지 4-bit, magnitude 3-bit 포화, 8세트 MUX 기반 shifting network, shuffle 스케줄용 min1/min2 force-refresh 로직.
8. 학습/최적화: Monte-Carlo bit-true 시뮬로 α=0.75·4bit 양자화가 최소 marginal cost임을 탐색.
9. 결과: TSMC 28nm, core 2.97mm², RBER=1.456%에서 avg 7.342 iter, 38.64Gbps, ONFI 5.0 2채널 지원, RBER=1.245%까지 error-floor 없이 UBER≈1.178e-10.
10. 한계: 실리콘 실측 아님(합성+P&R, PrimePower 전력), gatecount 본문 수치 없음, large-CPM 특정 부호·스케줄링에 arch가 강결합.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| NMS 반복복호 전체 | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 우리도 min-sum 반복복호 보유 — 알고리즘 기여는 중복 |
| CN min1/min2 + α scaling | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | α=0.75 정규화 factor 값은 참고 가능하나 계열 동일 |
| VN Lpost/Lv2c + 3bit 포화 | `decoder.cpp` `VN_Cal_HD()` 등, `decoder.h` `Get_VNU_*` | 2bit soft 입력/4bit 메시지 양자화가 우리 2SD/3SD와 비교 대상 |
| syndrome 조기종료 | `partial_CRC.cpp`/`full_CRC.cpp` | 우리는 CRC 기반 조기종료 — syndrome 방식과 상이 |
| MUX 기반 shifting network | `GT.cpp` `Make_GT_HW()` | edge/MUX 라우팅 개념은 유사하나 large-CPM 8세트 barrel 구조는 z=32와 다름 |
| QC-EG-LDPC H-matrix 로드 | `ecc_top.cpp` `Load_PCM()` | EG기반 large-CPM(511) 부호는 우리 고정 QC-LDPC(z=32)와 구조 상이 |

적용 가치: **중간** — NAND 직접 타깃의 고rate QC-LDPC 디코더 실구현 레퍼런스로 라우팅/플로어플랜 통찰은 유용하나, 복호 알고리즘(NMS)·조기종료는 이미 보유하고 HW arch가 large-CPM(z=511) 부호에 강결합되어 z=32 Prime ECC로의 직접 이식은 제한적.
