### B. 알고리즘 요약

1. 대상은 column-weight-2 LDPC(=cycle code)로, rate가 m 증가에 따라 1로 수렴(rate 0.17~0.99, length 270~70000)하는 고rate 구성.
2. 문제: 기존 large-girth cycle code(cylinder-type rate 1/e, geometric girth-24 rate 1/3)는 rate가 고정·낮아 유연성 부족 -> 최소길이·flexible-rate로 개선.
3. 모델: m×m 체스판 `Pm`의 흑칸 좌표로 2-set block `Bm`을 만들어 (m,b)-design의 incidence matrix `Hm`을 parity-check로 사용.
4. 핵심기법: `Cm`(mother matrix `Hm`)이 girth-8이고 같은 행규칙성 cycle code 중 최소길이(`m^2/4`)임을 Gallager bound로 증명(Theorem 2.5).
5. 식 의미: girth-8 최소길이 `n≥k+k(k-1)+...`(Gallager bound)를 달성 -> mother matrix로서 최적 출발점 보장.
6. 확장: `Hm`을 mother로 하여 circulant permutation `I_N^s`로 lifting, slope vector `S`로 (m,N,S)-code 생성, 최대달성 girth는 최소 `6g=24`.
7. 구현 디테일: cycle의 slope 합 `s_{j0,j1}+...≡0 mod N` 조건으로 목표 girth 미만 사이클을 배제하는 결정론적 slope 탐색 알고리즘(단계 1~11).
8. 학습/최적화: N을 1부터 키우며 backtracking으로 girth≥2g를 만족하는 모든 slope vector 탐색.
9. 결과: block-size 300·length 4800·rate 0.5 이진 C(24)가 PEG·RA·random 대비 우수(QPEG과 유사); 128-array 비이진 NC(24)(length 270, rate 0.33)가 NPEG·NRA 능가 (BER/FER 곡선만, 그림).
10. 한계: column-weight-2라 최소거리 로그증가(고신뢰 부적합), HW 미설계·시뮬만, 비이진 GF(q) 상당부분 포함.
