> 총평: 부호 연접(local/global coupling)과 2-phase 복호로 trapping set을 상호 해소해 error-floor를 없애는 부호설계 논문으로, 아이디어는 참신하나 Prime ECC의 단일 고rate QC-LDPC HW 구조와 이질적이라 이식성은 낮다.

### B. 알고리즘 요약

1. 시스템: 이진 LDPC 연접부호. Type1 GC-FG/FF는 로컬=cyclic FG-LDPC `CFG`, 글로벌 커플링=QC FF-LDPC `CFF`; Type2 GC-FF/FF는 둘 다 QC FF-LDPC. AWGN/BPSK, MSA 복호.
2. 문제: FG 부호는 최소거리 크고 small trapping set 없으나 length/rate 유연성 부족; FF 부호는 유연하나 열가중치 작으면 small trapping set 존재. 두 장점 결합 필요.
3. 모델: PCM을 `[O | diag(HFG,...,HFG); HFF,left | HFF,right]` 글로벌 형태로 구성 - `c`개 로컬 그래프를 글로벌 CN이 연결(CN-based globally coupled).
4. 핵심 기법 1: 2-phase 반복복호(global/local 또는 local/global) - 로컬 디코더가 갇힌 trapping set을 글로벌 디코더가 풀고 그 역도 성립. `d0-1`보다 작은 trapping set은 결코 2-phase 디코더를 가두지 못함.
5. 핵심 식 의미: 로컬 그래프별 trapping set VN 수 `κi < d0-1`이 보장되어 각 로컬 그래프에 유해 trapping set이 형성되지 않음.
6. 핵심 기법 2: FF 부호는 `B(m,n)=[αik+ηαjl]` 기반 2×2 SM 제약으로 girth≥6, CPM-dispersion으로 QC 구조 획득. GC-FF/FF는 단일필드 시 RC 제약 만족→1-phase 복호도 가능.
7. 구현 디테일: 로컬 FG 복호는 저복잡도 revolving iterative(RID/MSA)로 `HFG` 부분행렬만 사용; 글로벌 커플링 복호 시 로컬 codeword 신뢰도 공유(information sharing).
8. 부수: interleaved GC-FG/FF로 BEC burst erasure까지 대응(EPL peeling, 최대 `c(d0-1)` erasure 정정), HARQ 적용 여지.
9. 결과: 예1 (10000,7317) GC-PG/FF rate 0.7317, MSA(Igc/Ilocal/Imax=5, scale 0.625/0.275), error-floor 없음; 예2 GC-FF/FF (10000) rate 0.84, MSA 20 vs 100 iter 격차 0.1dB, 1-phase≈2-phase.
10. 한계: 이론+시뮬(BER)뿐, HW/공정 미설계, 대형 순환 FG 부호(1057)와 GF(101) FF 부호로 NAND용 단일 고rate QC-LDPC와 구조 이질적.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|

| 2-phase(global/local) 반복복호 스케줄 | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | 단일 부호 대상 이터레이션 루프와 달리 2단 디코더 오케스트레이션 필요, 직접 대응 없음 |
| scaled MSA (scale 0.275/0.625) CN 연산 | `decoder.cpp` `CNU_Update_New_Mag()` | Prime ECC min-sum과 동류, scaling factor 개념은 참고 가능 |
| trapping set 상호 해소(2-phase) | 대응 없음 | Prime ECC에 연접/2-phase trapping set 해소 메커니즘 없음 |
| 인코딩(local + global coupling 2단) | `encoder.cpp` `LDPC_encoder_4KB()` | Prime ECC dual-diagonal 단일부호 인코더와 구조 상이 |

적용 가치: **낮음** - error-floor를 부호 연접 구조와 2-phase 복호로 제거하는 접근은 아이디어로는 흥미로우나, Prime ECC의 단일 고rate binary QC-LDPC + 단일패스 파이프라인 HW와 근본적으로 이질적이라 떼어 쓰기 어렵다.
