### B. 알고리즘 요약

1. 대상: 병렬엣지 없는 base graph의 cyclic lifting으로 얻는 QC-LDPC `H`(circulant 배열, `s_{i,j}∈{0..P-1,∞}`), `N=JP`, `R=1-I/J`, binary/nonbinary(GF(q)) 모두.
2. 문제: 고정 base matrix·lifting degree `P`에서 최적 shift `s_t` 탐색은 조합폭발이라 기존은 heuristic/pseudo-random local 최적에 그침 → 전역최적 필요.
3. 핵심 개념: base graph의 closed walk가 lifted graph 사이클의 원천, 사이클 비존재 조건은 `Σ(-1)^{k-1} s_{tk} ≡ 0 mod P`(Theorem 1).
4. 핵심 기법: 각 closed walk의 mod-P 부등식을 이진변수 `y_i`와 큰 상수 `Z`로 선형부등식화(식 13)해 girth 보장 strict constraint 구성.
5. 목적함수: girth 이상 closed walk는 제약 대신 `α(d)·Σy_i` 항으로 목적함수에 넣어 제거 사이클 수 최대화, `α(d)`에 EMD/ACE 기반 harmfulness 가중.
6. 정식화: 전 closed walk 계수를 행렬 `A`, `b`, `c`로 vectorize(식 18-19)해 상용 IP solver(예: Matlab `intlinprog`)에 직접 투입.
7. 복잡도 절감: 등가부호 normalization으로 첫 nonzero submatrix를 P0 고정, reduced search로 합을 `-P~P` 범위로 제한해 closed walk당 부등식 5개·변수 2개로 축소.
8. 절차: PEG로 base matrix 생성 → support tree(trellis)로 closed walk 검출(`Dmax` 상한, 선형복잡도) → IP 풀이 → nonbinary는 GF(q) 계수를 행 단위로 선택·greedy 최적화(Algorithm 1).
9. 결과: rate-1/2~3/4 binary(576~1296) 및 GF(8/64/256) 부호에서 WiMAX/WiFi/[41]/[35] 대비 주로 error-floor 영역 약 0.1~0.2dB 이득, 동일 girth에서 더 작은 P 달성.
10. 한계: HW 미설계, 이득은 대부분 error-floor(그래프 최적화 특성상), 검증은 통신용 저~중rate 및 nonbinary 중심.
