### B. 알고리즘 요약

1. 대상은 MET-LDPC 코드 앙상블 설계로, 채널은 BEC와 BI-AWGN, rate `0.1~0.9` 전 구간을 다룬다.
2. 문제: 기존 MET-LDPC 설계는 check node degree를 concentrated form으로 가정해 탐색공간을 제한 -> sub-optimal 앙상블.
3. 앙상블은 edge distribution `E=[Ev Ec]`와 node distribution `ζ=[ν μ]`로 표현되며, 둘은 socket count equality(SCE)로 상호 제약.
4. 핵심: 주어진 edge distribution에 대해 SCE+rate+received 제약을 선형동차계 `A[ν;μ]=0` (식 12,13)로 정식화.
5. Dines 정리로 이 계를 단일 방정식까지 축약 후 positive solution을 재귀 복원 -> valid node distribution을 곧바로 산출(Algorithm 1).
6. 확장: joint 최적화를 nested DE로 구성 - outer DE는 edge distribution, inner DE는 node distribution 탐색(NP hard 순서 고정).
7. 트릭: `nv+nc-(ne+1)=1`로 잡으면 null space 차원 1 -> inner DE 없이 단일해 확보, 저rate엔 degree-1 VN 조건 `g1,i<=1` 부과.
8. 최적화 절차: DE 제어변수 outer(NP=1000, F=0.5, Cr=0.5, Gmax=1000), inner(NP=50, Gmax=100), target=density evolution threshold.
9. 결과: Algorithm 1이 random search 대비 valid node distribution 탐색 약 100배 빠름, 설계 앙상블이 기존 대비 capacity gap<0.02, C8이 rate 0.6667에서 BER 1e-5 기준 [29] 대비 0.3dB / AR4JA 대비 0.2dB 우세.
10. 한계: 앙상블 threshold 이론 수준 - 유한길이 실 코드/QC 구성/디코더/HW 미포함, NAND 저장장치는 응용 언급에 그침.
