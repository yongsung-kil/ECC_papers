### B. 알고리즘 요약

1. 대상: MLC(2bit/cell) NAND flash 채널, 비대칭 전압분포(µ11=0, µ10=3.25, µ00=4.55, µ01=6.5; δ11=2δ01=4δ10=4δ00). rate 0.81, length 4096 QC-LDPC.
2. 문제: AWGN용 LDPC는 flash 비대칭·양자화 채널에 최적이 아니고, 양자화 LLR은 비Gaussian이라 표준 EXIT chart 직접 적용 불가.
3. 모델: 3-bit 양자기로 read한 신호에서 MI를 최대화(MMI), 채널 출력 LLR을 히스토그램으로 추정.
4. 핵심기법 1단: 양자화 LLR의 MI `I(ζ,X)`를 식(3)으로 계산하고, 이를 분산 `δ_mlc²`를 갖는 Gaussian으로 강제 매핑해 flash 채널을 AWGN-like로 연결.
5. 핵심식: VND EXIT 함수 `IE = J(√((dv-1)[J⁻¹(IA)]²+δ_mlc²))` (식4) — `δ_mlc²`로 flash 채널을 EXIT chart에 편입.
6. 핵심기법 2단: 차수열 `λ`를 선형계획(식5)으로 최적화 (VND·CND EXIT 곡선을 근접시키되 교차 금지).
7. 구현 디테일: QC 제약을 유지하도록 PEG를 변형(girth 최대화 + QC 제약)해 base matrix(nested, n=4)와 circulant 배치 생성 (Algorithm 1).
8. 확장(별개 기여): rate-adaptive 극(polar) 부호 — cell bit Bhattacharyya 파라미터를 반복 계산(식10,11)해 큰 값 위치를 puncturing(식12)으로 rate 조정 (Algorithm 2).
9. 결과: 제안 protograph QC-LDPC는 EG-QC LDPC 대비 flash에서 BER 10⁻⁶에서 0.7dB, AWGN에서 1.2dB gain; polar은 puncturing 대비 BER 2~5 order 개선.
10. 한계: HW 미설계(gatecount/throughput 없음), 시뮬만, 특정 MLC 전압분포 가정, error-floor 미평가.
