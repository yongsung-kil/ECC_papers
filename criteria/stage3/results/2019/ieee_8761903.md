### B. 알고리즘 요약

1. 시스템은 3-track TDMR(2D 자기기록)로 GFP media 모델 파형 입력, IRA-LDPC를 채널디코더로 사용(code rate `0.66~0.705`).
2. 문제: 1D-PDNP의 2D 확장은 trellis state가 `4^(∆+I+L)`로 3-track 이상에서 비현실적 복잡도.
3. 모델: readback `yU=αU+Σαi`로, 주변 8 bit cell의 local area influence(LAI) `α` 합으로 target bit 결정, grain-flipping-probability media 모델 가정.
4. 핵심기법 1단(LAIP): offline training으로 LAI를 3×3 이웃 read값 조건부 PMF LUT에 저장, 검출 시 LUT 조회+convolution으로 `P(αtotal)` 산출.
5. 핵심식: `LLR(U)=log[(P(αtotal<yU)+Povw/2)/(P(αtotal>yU)+Povw/2)]`로 coded bit LLR 산출(overwrite 확률 `Povw` 반영).
6. 핵심기법 2단(BCJR): LAIP LLR을 a priori로 받는 trellis BCJR가 장거리 ISI/ITI 처리, PR mask 채널 기반.
7. 부수: 2개 MMSE 선형 등화기(FR target=1은 LAIP용, PR mask는 BCJR용), LLR에 weight `w1,w2` 곱·clip(overflow 방지).
8. 학습: 512개 3×3 패턴을 각 10301회 write/read, Lloyd-Max 40-bin 양자화로 조건부 PMF 학습, anti-symmetry 강제.
9. 결과: 1D-BCJR/PDNP 대비 center-track 6.8%·3-track 1.2% density gain(U/G), 2D-PDNP 대비 track1/2 BER 0.55×/0.08×.
10. 한계: 순수 시뮬(GFP media), HW 미설계, 자기기록 전용 채널모델, one-shot decoding(turbo 미적용), LDPC 자체 기여 없음.
