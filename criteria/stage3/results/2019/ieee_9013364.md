### B. 알고리즘 요약

1. 대상: LDPC FAID를 QNN으로 설계, 예제는 Tanner code `(155,64)`와 QC-LDPC `(1296,972)` rate 0.75, BPSK/AWGN, 최대 5 iteration.
2. 문제: BP/MS를 학습하는 기존 DNN 디코더는 floating-point weight라 HW에 부적합, 저정밀 activation은 gradient가 거의 0이라 backprop 불가.
3. 모델: Tanner graph를 "펼친" QNN - 3 hidden layer가 1 iteration(VN update/CN update/likelihood 추정), 각 뉴런=edge 또는 VN.
4. 핵심기법: activation 갱신규칙 `Φ`(VN), `Ψ`(CN=min-sum), `Υ`를 MS에서 취하고 양자화기 `Qmsg`(bmsg-bit), `Qc`(bc-bit)로 저정밀화(식1). weight/bias는 full precision.
5. 핵심 트릭: 저정밀 activation의 zero-gradient 문제를 straight-through estimator(STE)로 해결 - `∂Qmsg/∂Φ`를 clipped-ReLU STE(식12,14), sign은 sigmoid계 STE(식13,15)로 대체.
6. 양자화기: `Qc`는 MI 최대화로 설계(식4), `Qmsg`는 `Ac`의 대칭 부분집합 + 스칼라 `Smsg`로 threshold 유도(식5).
7. 목적함수: BCE 대신 실제 BER(Hamming, 식8)을 최소화 - BCE 최소가 BER 최소를 보장 못함을 반례로 증명.
8. 학습: all-zero codeword 가정(대칭성), ADAM+mini-batch, gradient는 full precision 누적, 학습은 오프라인.
9. 산출물: 학습된 `{W(3ℓ),b(3ℓ)}`로 iteration별 갱신규칙 `f(ℓ),g(ℓ)`를 정수 LUT로 구현(식6,7) → FAID. regular이면 weight 공유로 파라미터 축소.
10. 결과: 3bit 메시지/4bit 채널 FAID가 5 iteration 내 float MS 및 4bit MS 대비 평균 0.2dB coding gain(Fig.2,3). HW 미설계, gate/throughput 없음.
