### B. 알고리즘 요약

1. 시스템: N=8 user IDMA(NOMA) GMAC/Rayleigh 채널, 각 user가 동일 SC-LDPC(rate `Rc`) + 외부 repetition(rate `Rr=1/dr`) 직렬연접, 총 rate `Rtot=Rc·Rr`, sum-rate 목표 `Rsum=1`.
2. 풀려는 문제: 기존 LDPC-IDMA는 MUD(채널 front-end)에 맞춰 degree profile을 매칭해야 해 채널/user수/SNR마다 여러 부호 재설계 필요 — 실용성 저하.
3. 핵심 모델: SC-LDPC의 threshold saturation과 채널 front-end에 대한 **보편성(universality)** — 부호 파라미터 고정으로 채널 변해도 재설계 불필요.
4. 핵심 기법 1단: protograph 기반 SC-LDPC 구성 `B_{L,W}`(coupling window `W`, replication `L`, lifting `Z`)로 `H_{L,W,Z}` 생성; 규칙적 저degree(`dv=3`)로 error-floor 완화.
5. 검출/복호 결합: SoIC 기반 저복잡 MUD가 다른 user 간섭을 MMSE(`x̂_i=tanh(L/2)·e^{jφ}`)로 제거, 잔차간섭을 Gaussian으로 근사해 LLR `L_{ME}` 산출.
6. 핵심 기법 2단: repetition code를 "power equalizer"이자 user수 유연성 수단으로 사용 — `dr` 증가로 고정 SC-LDPC가 넓은 user 범위(N∈[8,64]) 지원.
7. 구현 디테일: windowed detector/decoder(활성창 `Wd≥W`, wavelike 수렴)로 복잡도·latency 억제, sub-block interleaver로 spatial locality 유지(full interleaver는 locality 파괴).
8. 학습/분석: Gaussian approximation density evolution으로 GMAC threshold 산출(식4~5, `φ(·)`), EXIT chart로 open tunnel 없이도 micro-convergence 확인.
9. 결과(시뮬): NCW=4×10^5, Wd=10 sub-block(각 8000 symbol), Imax=40; BER 10^-6에서 DE threshold 대비 ~0.2dB, GMAC capacity 대비 (3,4)~1dB/(3,6)~1.7dB, Rayleigh 손실 ~0.4dB.
10. 한계: HW 미설계·시뮬만, multi-user 무선/광 NOMA 특화, 매우 긴 블록길이(4×10^5)와 windowed decoding으로 NAND ECC와 응용/구조 상이.
