### B. 알고리즘 요약

1. 대상: regular LDPC `(408,816)`(rate 0.5 추정), BPSK/AWGN, 최대 iteration 100, hard-decision bit-flipping 복호 계열 비교.
2. 문제: bit-flipping(BF)은 복잡도 낮으나 성능 나쁨 → 성능/복잡도 trade-off 개선이 목표.
3. 모델: `y=c+z`, `z`는 평균0·분산 `σ²` AWGN. syndrome `Si=∏_{j∈N(i)} x_j`, `Si=+1`이면 무오류.
4. 핵심기법(기존 개관): WBF 반전함수 `Δk = Σ_{i∈M(k)} β_i ∏ x_j` (`β_i=min|y_j|`); MWBF는 `α|y_k|` 항 추가; GDBF는 ML 상관 최대화 목적함수의 편미분으로 `Δk = x_k y_k + Σ ∏ x_j`.
5. 핵심식: GDBF 목적함수 `f(x)=Σ x_k y_k + Σ_i ∏_{j∈N(i)} x_j` - hard-decision 코드워드와 수신값 상관 + syndrome penalty 동시 최대화, local maxima가 sub-optimality 원인.
6. 핵심기법(제안): NGDBF는 noise `q_k`(평균0, 분산 `η²N0/2`) 추가로 local maxima 탈출; 제안 GDBF1은 `Δk = 1.73·var(y)·x_k y_k + var(y)·Σ ∏ x_j + q_k` 로 수신열 분산 var(y)을 가중치에 도입해 SNR 영향 반영.
7. 구현 디테일: 반전함수 최소 bit를 한 번에 하나 flip(single-bit flipping), 분산으로 noise 분포 규모 조절.
8. 학습/최적화: 없음(휴리스틱), `α`는 Monte Carlo로 결정한다고만 언급.
9. 결과: NGDBF(1.73×var)가 SNR 5dB에서 BER `10^-3`, GDBF는 `10^-2.8`(Fig.1) - 분산 가중이 소폭 개선, iteration도 감소한다고 서술.
10. 한계: 단일 부호 시뮬만, HW 미설계, 수식/기호 손상 많고 var(y)·1.73 계수 정의·유도 근거 불충분.
