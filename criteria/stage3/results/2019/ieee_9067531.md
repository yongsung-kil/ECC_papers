### B. 알고리즘 요약

1. 시스템: rate `R=(n-m)/n` binary time-varying strongly-coupled QC-SC-LDPC, sub-matrix `H_i(t)` 크기 `m×n`에 lifting factor `S` circulant, syndrome former memory `μ`; 예제 `dv=4`,`S=40`,`T=5`,`W=120`,`μ=24`,`m=1`,`n=4`, rate 3/4, `L=100000`.
2. 문제: SC-LDPC의 semi-infinite 시변구조는 block 인코딩/복호가 부적합하고, 인코딩과 sliding window 디코딩을 동기화하며 large girth를 확보해야 함.
3. 인코딩 핵심: 주기 `T` 시변 QC-SC-LDPC 행렬을 grouping해 더 큰 time-invariant `H_new_SC`(식5, `H'_i` 크기 `TmS×TnS`, `μ'=⌈μ/T⌉`)로 재편 → 인코딩·복호 동기화.
4. 인코딩 알고리즘(Alg.1): `H_0^T`의 마지막 `m`행을 단위행렬로 두어 full-rank 보장, register 기반으로 partial syndrome `r1/r2/r3`를 계산·합산해 parity `v_t`를 구하고 `TnS`bit씩 좌shift.
5. Protograph 구성: base protomatrix `B=[B_0^T,...,B_μ^T]^T`(m=1이라 행벡터), lower-right로 semi-infinite 복제해 convolutional protograph `B_conv`(식7) 생성.
6. cycle 조건: distance parameter `B_{i,k}^{(j)}`(열 j의 두 '1' 행인덱스 차)로 4-cycle/6-cycle 조건 정의; no-4-cycle 위한 `μ` 하한 식8.
7. 탐색 축소: base protomatrix 첫 행을 all-'1'로 고정해 상태수를 `μ=15,dv=3`에서 약 `1.3×10^10`로 축소(열 재정렬 불필요).
8. 부호탐색: 4개 기준(C1=4-cycle수, C2=열간 공통위치, C3=6-cycle수, C4=3위치 이상 공통 triple)을 중요도순 greedy 적용해 protomatrix 구성 후, QC lifting으로 GP/SWG(sliding-window girth) 기준으로 girth 최적화(dim ≥ `(2μ+T)mS×(μ+T)nS`).
9. 결과: `dv=4` SC-LDPC가 동일 iteration(`Ie=15`)에서 `dv=3` BC-LDPC 대비 BER 1e-6에서 0.2dB gain; SC-LDPC의 delay/complexity가 d3 BC-LDPC(Ie=30) 대비 각 50%/67.5%로 유사성능; Shannon limit과 1.74dB 격차.
10. 한계: HW 미설계(gatecount/throughput 없음), AWGN·QPSK 가정, error-floor 영역 데이터 없음, strongly-coupled QC-SC-LDPC 한정.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| protograph 4-criteria greedy + QC lifting girth 탐색 | ecc_top.cpp Load_PCM() + H-matrix | 개념만 참고 가능, SC 구조라 직접 이식 부담 큼 |
| grouping 기반 시변→time-invariant 인코딩 | encoder.cpp LDPC_encoder_4KB(), Generate_PCM_encoding() | dual-diagonal block 인코더와 이질(컨볼루션 구조), 대응 약함 |
| Log-BP layered 디코딩 | decoder.cpp LDPC_Decoder() 이터레이션 루프 | Prime ECC는 min-sum, Log-BP는 full-tanh라 대응 약함 |
| sliding windowed decoding | 대응 없음 | block QC-LDPC엔 window 개념 없음 |
| GP/SWG girth 최적화 | ecc_top.cpp Load_PCM() (부호설계) | base matrix shift 설계 착상만 부분 참고 |
```

적용 가치: 낮음 — strongly-coupled QC-SC-LDPC 부호탐색·grouping 인코딩·windowed decoding이 핵심이라 block QC-LDPC 기반 Prime ECC에 직접 이식이 어렵고, protograph/girth 탐색(C1~C4, GP/SWG) 착상만 base matrix 설계에 부분 참고 가능.
