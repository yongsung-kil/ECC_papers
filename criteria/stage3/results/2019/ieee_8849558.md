### B. 알고리즘 요약 (10줄 내외)

1. 대상은 `(l, r, t)` SC-LDPCL protograph (coupling width `T=2`)를 `M`개 sub-block(SB)으로 나눈 랜덤액세스 부호, 채널은 BEC.
2. 문제: 긴 블록 전체 global 복호는 latency/복잡도가 크고, 순수 local 복호는 threshold가 낮아 정정력 부족.
3. 모델: coupling check(CC)로 인접 SB만 연결, `t`가 SB 간 연결 CN 수(=locality-coupling trade-off)를 제어.
4. 핵심 기법: "semi-global(SG) 복호" - target SB를 `d`개 helper SB만 접근해 helper phase→target phase 2단으로 복호.
5. helper phase는 helper를 국소 복호하며 한 방향으로만 메시지 전달, target phase는 양측 helper 정보로 target 복호 (window decoding과 달리 window 겹침 없음, target 근처에서 시작).
6. SG graph에 VN/edge labeling 함수를 정의해 동일 label edge끼리 DE가 일치 → DE 방정식 수를 축소.
7. `t=1`은 target graph의 edge type 4개를 실질 2개(`x^(1)`, `x^(2)`)로 축소, 2차원 fixed-point로 표현 (식 (1)~(3)).
8. SB varying-BEC(각 SB가 독립 BEC(E_m) 통과) 채널로 확장, 성공확률 `p_d(δ1,δ2)`를 재귀식(Thm 1)·양자화 하한(Thm 2)으로 계산.
9. 결과: `(5,12,t=3)` `M=11`에서 global threshold 0.375 대비 SG(d=10) 0.361(3.7% 손실)로 복잡도는 27% 절감; 양측 SG가 단측보다 항상 우수.
10. 한계: BEC 전용 asymptotic DE 이론만, AWGN/soft channel·유한길이·HW 구현·gate/throughput 수치 전무.
