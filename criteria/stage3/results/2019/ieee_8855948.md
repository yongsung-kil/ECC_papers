### B. 알고리즘 요약

1. 시스템: FSO(무선 레이저) 통신용 QC-LDPC, `(4008,3507)`, rate `0.875`, base matrix `3×24`, sub-matrix `167×167`, max col weight 3, BPSK.
2. 문제: column-serial layered 디코더는 CNU가 `nc`개 직렬 입력의 min1/min2를 다 처리해야 출력 → 다른 모듈(LS/AS/SUB/ADD/BS)이 layer당 `Call-nc` clock을 idle로 낭비.
3. 채널/모델: AWGN 기반 LLR 초기화 `Λ = 2y/σ²`, NMSA 기반 layered decoding(LDA).
4. 핵심기법: Inter-Frame Pipeline(IFP) — REC 모듈이 2프레임 LLR을 ping-pong으로 수신, 첫 프레임이 iteration 루프에 진입한 뒤 둘째 프레임을 idle clock에 삽입해 layer당 소비 clock을 `2nc`로 유지하며 유휴 제거.
5. 근거식: 절감 clock `Cearn=(Call-nc)×mL×mI`, 소요 clock `Cneed=nc×mL×mI+Δ` — idle 구간을 valid 처리로 전환해 유효 clock 밀도 최대화.
6. 확장: DMFP[14]가 3프레임 이상 저장으로 지연 폭증하는 것과 달리 IFP는 정확히 2프레임만 저장해 지연 최소화.
7. 구현 디테일: CNU는 min1(`vmin`)/min2(`vsmin`)/index(`vid`)/sign 추출, ADD·SUB의 특수 로직(Algorithm 1)으로 Rcv 복원, Sign 모듈은 redundant delay 없이 pipeline 참여.
8. 최적화: 별도 학습 없음(구조적 스케줄링 기법).
9. 결과: Xilinx VU13P, f=220MHz, niter=5, Δ=3 → throughput `2429Mbps`(IFP 없을 때 1825Mbps 대비 `+604Mbps`, 약 30%↑), delay `3.28us`.
10. 한계: HW는 FPGA만(ASIC/gate count 없음), 정정성능·error-floor 무개선, latency +1080ns 및 RAM 2배 소모, FSO 고rate 코드 특화.
