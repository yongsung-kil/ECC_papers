### B. 알고리즘 요약

1. 대상은 systematic LDPC `(16,8)`, rate `0.5`, BPSK, impulsive noise(SαS) 채널 평가용 (부호는 평가 도구일 뿐 NN이 부호 특성에 의존 안 함).
2. 문제: AWGN 기반으로 설계된 FNN/BP 디코더가 impulsive(non-Gaussian) 노이즈에서 성능 저하.
3. 모델: 잡음을 symmetric alpha-stable `SαS(α,γ)`로 모델링, `α∈[1,2]`, α=2가 AWGN, 성능척도는 GSNR.
4. 핵심 기법: 은닉층 뉴런을 LSTM 유래 gate 뉴런으로 재설계 — 입력게이트 `i`, 출력게이트 `o`, cell `c`; `i=μg(Wiv+bi)`, `c=i⊙μh(Wcv+bc)`, `y=o⊙μh(c)`.
5. 식 의미: 입력게이트 값이 0에 가까우면 outlier(impulse) 입력을 차단해 학습·추론을 안정화.
6. 확장: partially connected layer(PCL) — generator matrix `G`의 0/1 패턴대로 h1-h2 연결을 제거해 파라미터 수 축소.
7. 구현 디테일: forget gate·hidden state 제거(FNN용 단순화), sigmoid 게이트 + tanh cell 활성화, MSE 손실 + backprop 학습.
8. 학습: 무잡음 코드워드를 입력, 정보비트를 라벨로 지도학습, GTX 1080 Ti에서 1회성 학습.
9. 결과: near-ML(gap ≤0.1 dB), BP 대비 α=1.8에서 ≥0.8 dB, α=1.2에서 ~1.3 dB 개선(BER 10^-3), FNN 대비 ≥4 dB, ML보다 디코딩 시간 크게 단축.
10. 한계: 지수적 학습복잡도로 짧은 부호만 가능, HW 미설계·시뮬만, impulsive 채널 특화.
