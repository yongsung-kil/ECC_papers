### B. 알고리즘 요약

1. 대상: 5G eMBB Base Graph 2(BG2) 기반 MET QC-LDPC, protograph size `m=32, n=22, p=2`, code rate `1/3`, AWGN+QPSK.
2. 풀려는 문제: 5G eMBB 부호는 BLER `10^-2`에 최적화돼 BLER `10^-5`에서 error-floor 발생, 특히 정보길이 `k<232` 단구간에서 catastrophic behavior.
3. 핵심 가정: QC-LDPC를 circulant permutation matrix `P^k` 블록으로 표현, exponent matrix `E(H)` + mother matrix `M(H)`로 girth/block-cycle 정의.
4. 핵심 기법 1단 - modular lifting(식1): `E_ij(H_k)=E_ij(H_0) mod L_k`로 원부호에서 여러 circulant size `L_k`의 length-adaptive 부호 파생, 8 cyclic group family `L_k=a·2^i`.
5. 식의 의미: 하나의 base graph로 100개 이상 다른 길이를 compact하게 구성하면서 mother matrix 불변 유지.
6. 핵심 기법 2단 - simulated annealing lifting `SALift`: 각 `L_k`마다 Tanner graph girth와 EMD를 최대화(Hill-Climbing/improved PEG 대비 우수).
7. 부수 트릭 - k-step recoverable 인터리버: punctured VN의 s-SR 회복성으로 variable node를 Group&Sort(Alg.1) 하여 rate-matching용 최적 protograph permutation 탐색.
8. 최적화 절차: girth/EMD 목적의 simulated annealing + puncturing chain 탐색(num/it 반복), 소스코드 공개(github Lcrypto/puncturing-optimization).
9. 결과: `k>600`에서 5G eMBB 대비 `0.1~0.3 dB` gain(FER `10^-5`), `k<600`은 near-same, 104개 길이 시뮬(200 frame error/길이).
10. 한계: HW 미설계, sum-product 50 iteration 시뮬만, 5G BG2/MET/QPSK 전용, 저rate(1/3) 대상.
