### B. 알고리즘 요약

1. 대상 시스템은 지상파 방송의 dedicated return channel(DRC) 상향링크로, IDMA(NOMA) 위에 QC-SC-LDPC를 얹은 J-user Gaussian 다중접속 채널.
2. 문제: OMA 랜덤액세스는 충돌 시 outage가 크고, 기존 LDPC-coded IDMA는 타깃 `ηs`에만 최적화돼 사용자수 `J`·sum spectral efficiency `ηs` 변동에 threshold가 급락.
3. 핵심 아이디어: threshold saturation으로 채널 무관 universally capacity-achieving인 QC-SC-LDPC를 IDMA에 결합해 넓은 `ηs`(0.48~4.34 bit/chip)에서 강건성 확보.
4. 부호 구성: 각 uncoupled code base matrix `Bunc`(크기 1x2, true rate `Rtrue=1/2`)를 edge-spreading pattern `Bpattern=[B0..Bw]`(smoothing `w=7`)으로 coupling length `L=32` 결합해 `BSC` 생성(식 2).
5. termination rate loss는 최종 base matrix 마지막 6행 제거로 1/64로 축소.
6. 확장: two-step lifting - pre-lifting(general permutation, `Zpre=5`)으로 설계 자유도, post-lifting(circulant, `Zpost=128`)으로 QC 구조 유지, 총 `Z=640`.
7. intra-uncoupled-code interleaver `Πj`가 SC의 local 구조를 보존해 wave-like decoding을 MUD-복호 반복 중 유지(일반 랜덤 interleaver면 SC 이점 소실).
8. 수신: chip-by-chip MUD ↔ QC-SC-LDPC BP 복호 반복(`TMUD x TDEC`, 예 80x4, 강간섭 J=8은 160x4).
9. 결과: BLER 1e-2 기준 SNRsim이 SNRtheo 대비 `ηs`=0.48/2.42/3.88에서 1.42/1.38/1.96 dB gap, 넓은 `ηs`·J=1~9에서 강건, 기존 LDPC-IDMA의 high error-floor 문제 회피.
10. 한계: BLER 시뮬 검증뿐(HW 미설계), BPSK/Gaussian MAC·IDMA 특정 시나리오 가정, 복잡도/throughput 미제시.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QC-SC-LDPC edge-spreading/coupling 부호구성 | ecc_top.cpp Load_PCM() | Prime ECC는 고정 binary QC-LDPC(비 SC)로 spatial coupling 구조 미지원 |
| two-step lifting(pre/post permutation) | encoder.cpp Generate_PCM_encoding() | post-lifting circulant은 QC와 유사하나 SC coupling+general permutation은 대응 없음 |
| BP 복호 + MUD 반복(wave-like decoding) | decoder.cpp LDPC_Decoder() | min-sum 이터레이션은 있으나 IDMA MUD 연동·wave-like 스케줄은 대응 없음 |
| soft LLR(chip-by-chip MUD 출력) | channel.cpp Set_LLR_Th() | LLR 양자화 개념은 공유하나 다중접속 extrinsic LLR 교환은 범위 밖 |
```

마지막에 한 문장으로 **적용 가치**: `낮음` - SC-LDPC의 threshold saturation·wave-like 복호는 이론적으로 흥미로우나 Prime ECC는 고정 binary QC-LDPC(비 SC)이고 대상이 무선 IDMA 다중접속이라 NAND 단일사용자 ECC로의 이식 접점이 없다.
