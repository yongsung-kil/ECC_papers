### B. 알고리즘 요약

1. 시스템: 5G eMBB용 QC-LDPC(BG2 base matrix), `K=520`, `Z=52`, rate `R=0.2/0.25/0.33`, `N=1560~2600`, BPSK/AWGN, row-layered 스케줄링, `Itmax=15`.
2. 문제: 5G LDPC는 극도로 irregular하고 상당수 VN이 `degree-1`(extension의 identity 파트 I). quantized OMS는 이 저차수 VN에서 성능이 크게 나빠짐.
3. 모델: 메시지를 `q`비트 정수 `A=(-Q..+Q)`로 양자화, APP `γ̃`는 `q̃>q`비트로 유지, 채널 gain factor `µ`로 스케일.
4. 핵심기법: Adapted Min-Sum(AMS) — VN degree 그룹별로 서로 다른 offset `λ` 적용. degree-1 VN 이웃 CN엔 `λ=0`(MS), 나머지 CN엔 `λ=1`(OMS).
5. 핵심식: CN 업데이트 `β = sgn·max(min|α|-λ, 0)` 에서 정수 양자화상 `λ=1`이 저차수 VN 신뢰도를 과도하게 깎는 것을 `λ=0`로 회피.
6. 확장: BG2에선 degree-1 VN을 갖는 CN이 layer `l≥5`에 위치 → `λ(l)=1 if l≤4 else 0`으로 layer index 조건으로 단순 구현.
7. 구현 디테일: quantized(4,6) 즉 message 4비트, APP 6비트. CNU에 dc개 2:1 MUX + 비교기 하나 추가로 offset 분기 (Fig.4).
8. 학습/최적화: 없음(offset 값은 정수 양자화 하 고정 룰).
9. 결과: degree-1 VN 오류확률 OMS 2.4e-2 → AMS 3.2e-5로 급감. FER은 quantized AMS가 floating-point OMS를 능가하고 SP에 근접(R=0.25/0.3).
10. 한계: HW 상세 회로 미제시(장문판 이연), FER≈5e-4에서 error-floor, 무선(5G) 채널 가정.
