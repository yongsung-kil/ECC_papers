> 총평: 저비트(1~2bit) message-passing으로 디코더 내부 데이터플로우/라우팅 congestion을 줄이는 min-sum 계열 변형, NAND soft-read 비용 절감 관점에서 참고가치 있으나 SC-LDPC·광변조 특화라 부분 이식.

### B. 알고리즘 요약

1. 대상: 광통신(dispersion-uncompensated long-haul, AWGN 모델) 고속 FEC용 protograph 기반 SC-LDPC, higher-order modulation(M-ASK/QAM)+PAS, SE 1.0~1.5 bpcu, `Rc ∈ {2/3,3/4,5/6}`.
2. 문제: 1 Tbps급 처리량에서 BP 복호의 내부 데이터플로우 `F = 2·nc·q·d̄v`가 라우팅 congestion을 유발 → 메시지 비트폭 `q`를 1~2bit로 줄이려 함.
3. 모델: bit-metric decoding(BMD) bit-channel `lk(y)=log P(0|y)/P(1|y)`를 VN 입력 soft-info로 사용, PAS 채널 비대칭은 scrambling으로 대칭화.
4. 핵심기법(QMP): CN→VN, VN→CN 메시지를 4-ary `{-H,-L,+L,+H}`로 양자화(저/고 신뢰 구분), CN은 min-sum 규칙(부호는 곱, 크기는 최소).
5. 핵심식: VN 갱신 `mv→c = Ψ(ldec + Σ sign(mc→v)·w^(l)_{|mc→v|})`, 신뢰도별 가중 `wL, wH` 2세트 사용 (식 (13),(56)).
6. 확장: BMP(1bit)/TMP(3-ary 2bit)와 동일 데이터플로우에서 QMP가 4-ary로 더 세밀한 신뢰도 표현.
7. 구현 디테일: 임계 `T`(예 T=1.3) 고정, 가중치는 iteration별로 공유 가능(성능 저하 미미).
8. 최적화: 각 알고리즘의 DE(density evolution)로 asymptotic threshold와 최적 가중치 `wL,wH`를 유도(식 (32),(33)); surrogate AWGN 채널로 초기화 근사.
9. 결과: 4U-0.50에서 unquantized BP 대비 gap BMP 2.39/TMP 1.14/QMP 0.9 dB; QMP가 TMP 대비 저rate 0.24 dB, 고rate 0.1 dB 이득; finite length(nc=60000, S=50) FER 10^-4에서 QMP가 BMP 대비 ~0.8 dB, full BP 대비 gap ~0.75 dB.
10. 한계: HW 미구현(gatecount/throughput 없음), 광변조·PAS·SC-LDPC 특화, avg iteration 감소는 다루지 않음(max 1000 iter).
