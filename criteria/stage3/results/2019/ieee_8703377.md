### B. 알고리즘 요약 (10줄 내외)

1. 시스템: MLC(2bit/cell) NAND flash, gray-code 매핑 4-state, 고rate LDPC 예 `(3780,3402)`,`(8000,7360)`,`(12000,10800)` + GF(4) `(2304,2048)`.
2. 문제: soft-decision BP(특히 flooding/serial)의 수렴이 느려 latency 큼, IDS는 residual 계산으로 복잡도 과대.
3. 채널 모델: 4-state threshold voltage 분포(erf 기반) + 6개 non-uniform read reference로 7 region(4 data + 3 overlap E1~E3), region별 MSB/LSB LLR 산출.
4. 핵심 발견: overlapping region의 `blurry` 노드(E1·E3 LSB, E2 MSB)만 오류확률 높음 → 노드를 blurry/non-blurry로 분류(LLR magnitude 그룹 `G1<G2`).
5. 핵심 기법1(MP): 초기 iteration에 blurry 노드의 불신뢰 V2C 전파를 차단, 신뢰 V2C로 blurry 노드 우선 갱신(T0~Tk check node 타입 기반 3-step).
6. 핵심 식: SBP식 (9)(10)(11) 그대로 쓰되 T1 타입(신뢰) check node의 C2V만 사용해 `domination`(부호 강화/정정) 유도.
7. 핵심 기법2(OB-ASS): MP 후 진동노드(부호가 이전 iteration과 다른 노드) 집합 O를 별도 재갱신(alternate selection).
8. 비이진 확장(NVNBP-MP): GF(4) 심볼=cell, region으로 blurry 분류, 확률영역 연산, oscillation을 심볼 변화로 정의.
9. 결과: MLC 채널서 SBP/QABP/LLRBP 대비 FER 최우수, `(4032,3264)`는 FER=1e-4서 SBP 대비 ~1400 PE cycle 이득, throughput 상회, Imax=5. 추가 복잡도는 `α·Nbv·dbv` 비교연산 수준(IDS의 N(N-1)보다 훨씬 작음).
10. 한계: HW 미설계·시뮬만, serial 스케줄 기반(노드 재정렬), 채널모델이 실제 NAND와 다소 괴리, MSB/LSB gray-code 가정에 의존.
