### B. 알고리즘 요약

1. 시스템: 농업용 무선센서망(WSN, 5G/URLLC 문맥) LDPC, IRA(irregular accumulation) 부호, rate `1/2 ≤ γ ≤ 5/6`, `1296-bit` 랜덤 생성 LDPC, FSK/ASK 변조.
2. 문제: 선행 [6]의 parity trimming(fake error) 방식이 복호 iteration·복잡도를 오히려 증가시켜 실시간 데이터 처리량을 제한.
3. 모델: parity의 일부(3/5/6/7%)를 trim해 message 자리에 분산 배치(fake error), 수신 후 재구성하며 SNR 채널에서 복호.
4. 핵심기법 1(인코더): 인접 message bit `m2n−1 ⊗ m2n+1` XOR로 매핑코드 생성(식1), message codes 앞에 붙여 전송 - 데이터 압축이 아니라 복호 복잡도 감소용.
5. 핵심식: 매핑코드 bit이 diagonal(1)/direct(0) 패턴을 지정, bit-node 갱신 시 SPA 결정에 순차 반영해 짝수 check/bit node를 매핑패턴으로 단순 결정.
6. 핵심기법 2(디코더): 수정 SPA(Alg.1) - check node 갱신 `Lγ(rji)=∏sign·φ(∑φ(Lγ(gij)))`(식3), 매핑코드 `mpi=1`이면 XOR 적용, 짝수 node는 매핑패턴으로 계산 생략.
7. 구현 디테일: bit-node 갱신 `Lγ(gij)=L(gij)+β∑(rji)`(식4)에 trimming factor `β(0<β≤1)`로 trim 보정, scenario 1/2/3(매핑코드를 message/혼합/parity에만 배치)로 구분.
8. 최적화 절차: 별도 학습 없음, scenario 비교로 message-only(scenario 1)가 최적임을 실험 확인(scenario 3은 parity 부족으로 실패).
9. 결과: 제안 scheme이 SNR 8~11 구간에서 normal 대비 복호 iteration을 최대 54% 감소, 3% trim에서 최소 복잡도, [6] 대비 복잡도·패킷에러 개선(수치는 그림 기반, 본문 확정치 제한적).
10. 한계: 순수 알고리즘/시뮬(C++·MATLAB, HW 미설계), trimming으로 실제 code length가 단축되며 WSN 패킷전송 패러다임에 강결합.
