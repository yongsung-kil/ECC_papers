### B. 알고리즘 요약

1. 대상은 variable-regular LDPC 부호의 Tanner graph, 변수노드 차수 `dv`, girth `g`(고려 범위 `3≤dv≤6`, `6≤g≤16`, `0≤b≤5`).
2. 문제: trapping set(특히 LETS)이 error floor의 주원인인데, 주어진 `dv`,`g`에서 가장 작은 trapping set 크기의 상한이 정립되지 않음.
3. 핵심 관찰: `(a,b)` trapping set 중 `b=0`은 무게 `a`의 비영 codeword이며, cage(주어진 girth에서 노드 최소인 정규그래프)와 trapping set을 연결할 수 있음.
4. 핵심 기법: cage `G`에서 시작해 모든 간선을 subdivision(`G^(1/2)`)하여 Tanner graph로 변환한 뒤, 노드/간선 추가·제거로 원하는 `(a,b)` 클래스 trapping set을 구성.
5. 이 구성으로 LETS 최소크기 `T_LETS^b(dv;g)`의 정확값/상한을 cage 크기 `n(dv;g/2)`로 표현(예: `T_LETS^0(dv;g)=n(dv;g/2)`).
6. 확장: ETSL(leaf 있는 ETS), NETS(non-elementary), 그리고 `b=0`의 elementary/non-elementary codeword로 유형별 상한 유도.
7. 부수 결과: 유도된 상한이 known lower bound와 다수 일치 → tight, 즉 실제 최소 trapping set 크기 및 최소거리 확정.
8. 이론적 함의: worst-case 최소거리는 블록길이 `n`과 무관하게 `dv`,`g`만의 함수(상수)임을 보임.
9. 결과: 여러 `(dv,g)`에서 unique한 최소 trapping set 구조(예: dv=3,g=10의 (10,0) LETS = Petersen graph) 식별, 이를 부호설계에서 회피·탐색 지표로 활용 가능.
10. 한계: 순수 그래프이론 상한 유도로 실제 부호 구성·복호기·시뮬레이션·HW 없음, variable-regular에 국한.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| trapping set/최소거리 상한 기반 부호구조 회피 | `ecc_top.cpp` `Load_PCM()` (H-matrix 로드) | Prime ECC는 특정 QC-LDPC(dv=17) 고정, 본 이론은 variable-regular general 결과라 직접 반영 어려움 |
| error-floor(trapping set) 분석 | 대응 없음 | Prime ECC는 수렴실패 추적(`corner.cpp`)은 있으나 trapping set 상한 계산 모듈 없음 |
| min-sum 복호/양자화 | 대응 없음 | 본 논문은 복호 알고리즘·양자화 미다룸 |

적용 가치: 낮음 — error-floor의 근원인 trapping set 최소크기 이해에는 참고가 되나, 순수 이론 상한으로 Prime ECC의 특정 QC-LDPC 부호설계나 디코더에 떼어 이식할 실체가 없다.
