### B. 알고리즘 요약

1. 시스템: coherent 광통신의 probabilistically shaped(PS) QAM(PS-16QAM), FEC는 soft-decision LDPC, code rate `RC=0.75~0.875`, 12.5GBaud 실험.
2. 문제: PS 시스템에서 exact LLR(ELLR) 계산은 성상점 전체 합·지수 연산으로 비효율적이고, PS는 심볼별 사전확률 p(s)가 불균등해 별도 LLR 필요.
3. 채널/모델: AWGN 채널 + Maxwell-Boltzmann 분포(PAS, 저에너지 성상 빈번 전송), 수신신호 `r=(x,y)`.
4. 핵심 기법 1단: dual-maxima 규칙 - 각 합에서 지배적 최대항만 취해 `L(Bi) ≈ -(1/σ²)[min_{S0}|r-s|² - min_{S1}|r-s|²]` (ALLR, Eq.3).
5. 핵심 식 의미: 지수항 중 한 항이 합을 지배하므로 최대항 대체로 정확도 손실 없이 지수·합 연산 제거.
6. 핵심 기법 2단: 사전확률 반영 ALLR-P(Eq.5) - min 항에 `-σ²·ln p(s)` 보정 추가해 PS의 불균등 확률 반영.
7. 부수 트릭: bit position별 S0/S1 집합 정의, ELLR-P(Eq.4)와 대비하여 근사 정당화.
8. 학습/최적화: 없음.
9. 결과: 시뮬레이션에서 ALLR/ELLR 간 BER penalty 거의 없음, 동일 net rate 기준 PS-16QAM이 uniform 대비 post-BER 1.25dB(시뮬)·1.17dB(실험) 이득.
10. 한계: HW 미설계, 광통신 PS-QAM 특화, gate count·throughput·iteration 지표 없음, 성상도 기반 soft-info라 NAND read 모델과 무관.
