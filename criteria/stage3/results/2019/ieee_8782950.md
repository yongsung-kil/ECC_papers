### B. 알고리즘 요약

1. 대상: 3D-TLC NAND flash, code rate `9/10`, max iteration `30`의 LDPC ECC, read-hot(읽기 반복) 데이터 정정.
2. 문제: 기존 ANN-LDPC는 read-cold(data-retention 오류) 최적화라 read-hot의 read-disturb(R.D.) 오류를 정정 못함.
3. 모델: read-hot에서 R.D.는 `VTH` 상승/하강 모두 유발(A~E state 상승, F/G 하강), WL별 BER 상이(hole 크기 차이).
4. 핵심 기법: NN(지도학습)으로 1bit 단위 `BER`를 예측 → `LLR(0)=ln{(1-BER)/BER}` 계산에 사용.
5. 식 의미: 정확한 BER→정확한 LLR→LDPC 복호가 빠르고 효과적이 됨(디코더 자체는 표준).
6. 확장: 입력 파라미터를 D.R. time 대신 read cycle `Nread`로 교체, target VTH-state/page type/WL번호/WL(n±1) VTH-state 등 사용.
7. 구현 디테일: 연속값(`NW/E`, `Nread`)은 linear 또는 log scaling으로 정규화(Case 1~4), 이산값은 binary 인코딩.
8. 학습: forward/back propagation, hidden layer ReLU, output layer softmax, loss `E=-{L1 ln(y1)+L2 ln(y2)}`, read-hot 데이터 11.6GB 학습.
9. 결과: 예측 BER를 측정 BER에 4.3배 근접(정확도 92% vs 21%), acceptable BER 2.1배, acceptable read cycle 10배(300→3000) 확장.
10. 한계: LDPC 디코더/HW 미설계, NN 전처리(LLR 예측)에 국한, 특정 3D-TLC 실측 채널 가정.
