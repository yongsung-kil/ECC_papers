### B. 알고리즘 요약

1. 대상 부호는 직접 구성한 rate `0.9`, 길이 `2KB` binary QC-LDPC, layer min-sum(보정계수 `0.75`, 최대 iter `10`, 초기 LLR 6-bit 부호양자화 [-31,31]).
2. 문제: 3D TLC NAND에서 부정확한 LLR이 LDPC 정정능력을 떨어뜨림 -> LLR 정확도 향상이 목표.
3. 모델: 64층 3D TLC 16칩 실측(P/E~5000, medium retention)에서 각 state Vth가 가우시안에 잘 맞음(R-square>0.995), 간섭은 대칭적 균일 강하로 관찰.
4. VaLLR 핵심: Vth 분포 + reference voltage 배치로 floating LLR 계산 후 quantization+saturation 2단계로 고정소수 LLR화, offline LUT로 구현.
5. 양자화식 `q(x)=β×x/min(x)+γ`, saturation `Q(x)`는 `±max=2^(q-1)-1`로 포화 — 작은 LLR 정밀도 보존이 핵심.
6. SDRV 배치: 각 HDRV 좌우 대칭 배치가 최적(관찰1), 6개 overlap 구간에 공통 `Δi`(0.135) 사용해도 성능 손실 없음(관찰2), P0-P1만 비대칭.
7. reference voltage 개수 전략 A(3개)/B(5개)/C(9개) 비교: 3->5는 수 자릿수 이득, 5->9는 미미 -> 3~5개 권장.
8. LDPC 디코더가 작은 LLR에 민감하므로 `β,γ`로 작은 LLR 증폭(일부 큰 LLR은 포화 감수), 시뮬 기반 수치근사로 `β,γ` 최적화.
9. 결과: strategy A에 `β=2,γ=0` 적용 시 FER 2 자릿수 개선(추가 구현복잡도 없음), strategy B도 개선되어 C에 근접 -> reference 5개로 우수 성능.
10. 한계: HW 미설계(순수 시뮬), offline LUT가 특정 64층 3D TLC 실측 종속, layer min-sum 소프트웨어 SIMD 가속 전제.
