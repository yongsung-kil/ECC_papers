### B. 알고리즘 요약

1. 시스템: MLC 2bit/cell NAND flash, LDPC soft-decision 복호용 read reference 전압 최적화, rate-`0.9014 (4544,4096)` QC-LDPC + min-sum, `Imax=30`.
2. 문제: 기존 read-voltage 최적화([5]~[8])는 채널 노이즈 하 threshold-voltage 분포 추정을 요구하고, ROR[9]은 wordline별 전압 shift a-priori 정보를 안 써 BER가 미흡.
3. 모델: erased state 정규분포, programmed state 균등분포(ISPP), CCI shift `F=ΔVtk·γk`, retention noise `N(μt,σt²)`, `μt=(x-x0)[At·PE^αi+Bt·PE^αo]log(1+t)`, `σt=0.3|μt|`.
4. 핵심기법1(CSD-TVD): overlap 영역을 폭 `Δ` sub-window로 분할, read 전압을 왼쪽으로 슬라이딩해 셀 수가 최소가 되는 지점에서 wordline별 전압 shift `ΔVik` 검출.
5. 의미: 최소셀 sub-window가 두 상태 분포 교차점에 대응하므로 분포 전체를 추정하지 않고도 최적 shift를 직접 잡아냄.
6. 핵심기법2(분포 활용): wordline별 `ΔVik`가 정규분포 `N(μΔV,σ²ΔV)`를 따름을 확인, 그 평균 `μΔV`를 최적 shift로 삼아 read reference 전압 정확도 향상.
7. 구현디테일(LL-CSD-TVD): retention상 상위 state가 더 빠르게 shift(`ΔV1<ΔV2<ΔV3`)함을 이용, `(i+1)` 영역 시작을 `Vread^(i+1)-ΔVi`로 앞당겨 read 연산 수를 절반으로 절감.
8. 최적화: 파라미터 추정으로 `ΔVopt`를 read 전압에서 감산해 채널 노이즈 보정(분포 추정 불필요).
9. 결과: `FER=1e-3, PE=20K`에서 retention age 700h(ROR)→1500h(CSD-TVD), ROR 대비 약 1자릿수 이득, 평균 iteration 감소(수렴 가속), LL-CSD-TVD read 횟수 절반.
10. 한계: HW 미설계·시뮬만, 특정 retention/CCI 노이즈 모델·MLC 2bit/cell 가정, 디코더 자체가 아닌 read 전단 검출 기법.
