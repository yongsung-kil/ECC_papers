### B. 알고리즘 요약

1. 시스템은 QPSK 변조 OFDM UWA 통신 링크로, code rate `R=1/2` regular LDPC를 결합하고 최대 디코딩 iteration은 50회로 고정한다.
2. 풀려는 문제: regular random matrix에 존재하는 girth-4 short cycle이 BP 디코딩의 message가 2 iteration 후 자기 노드로 되돌아와 error propagation을 유발해 성능을 저하시킨다.
3. 채널 모델은 지수분포 multipath delay + Rayleigh 진폭(평균전력 지수감쇠)의 linear time-invariant UWA 채널, 최대 multipath delay 약 10ms.
4. 부호 구성: column weight `dv=3` 예시로 matrix를 3개 sub-matrix로 수평 분할, 첫 sub-matrix는 diagonal-block 패턴, 2·3번째는 random column 치환.
5. 코드율 식 `R = 1 - dv/dc` (regular (dv,dc) 코드), 여기서 short cycle이 BP의 tree 가정을 깨 MAP 근사를 열화시키는 것이 핵심 동기.
6. Method 1: length-4 cycle 검출 시 cycle 위 edge 하나를 직접 0으로 flip하여 cycle을 끊는다(단, node degree가 변함).
7. Method 2: edge 하나를 0으로 flip한 뒤 같은 column의 다른 1을 0으로 flip하고 원래 위치를 보정해 row weight·column weight를 보존한 채 cycle 제거.
8. 디코딩은 log-domain BP(tanh 기반 CN 업데이트, `Lrkl`/`Lqlk` 메시지)로 OFDM demodulation 후 LLR을 초기화값으로 사용.
9. 결과: AWGN에서 Method 2 gain 약 0.8dB, UWA 시뮬 채널에서 short cycle 미제거 대비 약 1dB gain(Method 1은 고SNR에서 오히려 열화).
10. 한계: HW 미설계, 순수 시뮬, UWA/OFDM 특정 채널 가정, gate count·throughput 등 구현 지표 전무.
