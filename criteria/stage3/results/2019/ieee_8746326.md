### B. 알고리즘 요약

1. 시스템: finite-field(GF(512)) 기반 (3,6)-regular QC-LDPC `(8176,4088)`, rate 1/2, QPSK gray mapping, `Itmax=100`, Markov Gaussian(버스트 임펄스) 채널.
2. 문제: 임펄스 노이즈는 버스트성이라 AWGN만 가정한 SPA는 성능이 크게 붕괴. 기존 기법은 임펄스 통계를 알아야 해 계산량 과다.
3. 모델: 2-state(Good=AWGN, Bad=임펄스, 전력비 `R`) 1차 Markov 채널, 전이확률 `P_GG,P_GB,P_BG,P_BB`. 수신기는 실제 통계 대신 근사 파라미터 `(d)` 사용.
4. 핵심기법: 두 개체 캐스케이드 — (a) forward-backward(F-B) MAP 채널상태 추정기, (b) revised SPA 디코더. Bad로 추정된 표본 `y_j`는 clipping, Good이면 그대로 통과.
5. 핵심식: 결합확률 `G(c_j,s_j,s_{j+1})=Pr(s_{j+1}|s_j)P(n_j=y_j-c_j|s_j)`, bad 상태 PDF를 유효 PDF 대신 상수 offset provider `1/(πR^(d)N0)`로 치환 → clipping과 등가.
6. 확장: 외부 루프(F-B 상태추정)와 내부 루프(SPA)가 VN 메시지 `v_jx`와 소프트정보 `f_nx`를 상호 교환. 유효 codeword가 두 번 나오거나 최대 반복 시 종료.
7. 구현 디테일: masking matrix `Z` 보정 규칙 3가지(연속 1 금지, 좌우 밀도 배치, 대각 제약)로 버스트 잔류 임펄스 억제.
8. 학습/최적화: 없음(통계 추정 회피가 핵심). clipping threshold는 sparse 신호의 정규화 파라미터와 등가.
9. 결과: 최적 수신기(완전 통계 가정) 대비 BER 10^-5에서 약 0.1dB gap. 외부 루프 1회로 단순화해도 손실 미미. 보정 masking한 finite-field QC-LDPC이 5G NR 부호보다 임펄스 하에서 낮은 error floor.
10. 한계: HW 미설계, 외부 MAP 루프로 복잡도 과중, 무선/전력선 버스트 임펄스 채널 특화(NAND 무관).
