### B. 알고리즘 요약

1. 시스템: nested code(여러 sub-code Ci가 global code C에 포섭)를 array-based SC-LDPC로 구성, 무선 multi-terminal 물리계층 XOR 중첩 전송용.
2. 문제: array-based LDPC의 Tanner graph에 존재하는 dominant absorbing set(ABS)이 BP 디코더 실패·error-floor를 유발, nested 구조는 sub-code PCM이 겹쳐 공동 제약 최적화가 어려움.
3. 모델: `(γ,p)` AB-LDPC-BC는 base가 all-ones, 순환행렬 `σ^z`(shift z mod p)로 구성, spatial coupling은 `B=B0+...+Bm`로 edge-spreading.
4. 핵심기법1: ALC(adapted line-counting) - 임의의 column-weight-3 부분행렬 region `R`에서 6-cycle을 다항시간 `O(p^2)`로 열거 (기존 cycle counting `O(gγ^2 p^3)` 대비).
5. 핵심식: Lemma 2의 부등식 (3)이 `c3` 범위를 `c1,c2`와 `λ,q2,q3`로 표현, 6-cycle 존재를 정수격자점 카운팅으로 환원.
6. 핵심기법2: M=2 nested 최적화 - 먼저 global `H(3,p,L)` 최적화(Method 1) 또는 sub-code `Hi(4,p,L)` 먼저(Method 2), 제약을 물려 row-group 단위 순차 edge-spreading 최적화.
7. 구현디테일: terminal lift(lifting factor `J`, 순환행렬)로 QC-LDPC 화 및 잔여 6-cycle 제거.
8. 최적화: 6-cycle 수를 목적함수로 하는 수치 최적화, `Imax` 반복 상한.
9. 결과: Method 2에서 p=5는 column-weight 3/4 nested code 모두 6-cycle 0 → (3,3)·(4,4) ABS 부재, J=5 terminal lift로 p=5,7(m=1,2)·p=11(m=2)의 모든 6-cycle 제거.
10. 한계: HW 미설계, BER 실측 없이 6-cycle 카운트만, 소규모 소수 p 예제, 무선 nested 채널 가정.
