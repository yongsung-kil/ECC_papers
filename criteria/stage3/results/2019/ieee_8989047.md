### B. 알고리즘 요약

1. 대상: 투영평면 `PG(2,q)`(q=2의 거듭제곱)의 line-by-point incidence matrix로 정의되는 cyclic 투영평면 LDPC 부호 `[q²+q+1, q²+q−3log₂q, q+2]`.
2. 문제: 잉여(redundant) 패리티식이 유해 substructure를 줄여 성능을 높이지만, 잉여식 수를 적게 유지하며 stopping set을 회피하도록 패리티행렬을 설계하기 어려움.
3. 모델: erasure 채널의 표준 반복복호에서 stopping set이 복호 실패를 완전히 규정 → 작은 stopping set 회피가 목표.
4. 핵심결과(Theorem 1.1): PG(2,q) q짝수 패리티행렬은 최소무게 부호어(무게 q+2)를 제외하면 크기 q+3 이하의 stopping set이 존재하지 않음.
5. 의미: erasure가 최소거리를 2 이상 초과하지 않는 한, ML 디코더가 실패하지 않는 이상 표준 반복복호도 실패하지 않음.
6. 증명(기하): H(S) 부분행렬에서 각 점이 q+1 line에 등장하는 성질과 각 line이 q+1 point를 포함하는 성질로 무게별 행 개수 li의 등식/부등식을 세워 |S|≥q+4 유도.
7. 증명(스펙트럴): r-regular 이분그래프의 두번째 Tanner 고윳값(=q, Theorem 3.2)과 확장성 부등식(Theorem 3.1)으로 |S|≥q+3을 대안 증명.
8. 부수결과: generic (r,m)-erasure correcting set 크기 상한을 greedy 분석으로 Theorem 4.1보다 (floor 함수 유지로) 근소하게 개선(Theorem 4.3).
9. 결과: PG(2,q) 부호의 stopping redundancy가 부호길이 q²+q+1 이하임을 재확인하고, 일반 상한(Theorem 2.2) 대비 잉여식이 극히 적음을 강조(Table I: q=16에서 273 vs 411672).
10. 한계: 순수 이론(증명만), HW·시뮬 없음, erasure 채널·투영평면 cyclic 부호 전용, greedy 구성은 비효율적이라 실용 아님.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| PG(2,q) 투영평면 부호 패리티행렬 설계 | 대응 없음 | Prime ECC는 QC-LDPC(base+shift) 고정, 비QC cyclic 투영평면 부호와 구조 상이 |
| stopping set / erasure 복호 성능 | 대응 없음 | Prime ECC 채널은 AWGN/RBER/fixed-error, erasure 채널·stopping set 분석 미대응 |
| generic erasure correcting set 상한 | 대응 없음 | 이론적 상한 결과로 코드베이스 이식 대상 아님 |

적용 가치: **낮음** — erasure 채널의 stopping set에 대한 순수 조합론 증명이며 대상 부호가 비QC 투영평면 cyclic 부호라, NAND binary QC-LDPC(Prime ECC) 디코더/부호설계 어디에도 직접 이식할 요소가 없음.
