### B. 알고리즘 요약

1. 대상은 quantum stabilizer code `⟨n,k,d⟩` (toric code, MacKay bicycle code), depolarizing channel을 두 개의 독립 BSC(crossover `2p/3`)로 근사해 binary normalizer label code `N`에서 syndrome 기반 복호.
2. 문제: quantum LDPC의 degeneracy 때문에 표준 SPA 성능이 크게 열화 — 정정능력이 normalizer label code 최소거리 `dN`에 묶임(부호 최소거리 `d`가 아님).
3. graph-cover pseudocodeword로 SPA 고정점을 분석: SPA fixed point는 Bethe free energy 정류점에 대응하고 pseudocodeword `ω`를 유도.
4. 이론적으로 non-degenerate/degenerate ML 디코더 실패 최소 오류무게가 각각 `tN+1`, `t+1`임을 증명(Thm 2,3).
5. toric code에서 SPA가 실패하는 최소 오류무게는 2이며(Thm 4), `⟨2L²,2,L⟩`에서 그 개수는 `12L²`, 일반 cycle code는 `dN` 이하에서 실패(Thm 5).
6. 개선 1 — RR-SPA: 메시지 계산을 랜덤 구간에서 뽑은 가중치로 재가중.
7. 개선 2 — IMR-SPA(Alg.1): 초기 LLR `γi`를 랜덤 `αi∈[a,b]`로 재가중(`αiγi`)해 syndrome 만족 벡터를 얻을 때까지 반복, Bethe free energy 프레임으로 분석 가능.
8. 개선 3 — SPA+PCWD(Alg.2,3): SPA 실패 시 pseudocodeword `ω`를 경로들로 분해(가장 큰 성분 edge를 따라감), 비용 `(1−ω̂i)·|Pi|` 최소 경로들로 valid syndrome 벡터 구성(LP 변형은 SPA+LPPCWD).
9. 결과: toric code에서 SPA+LPPCWD는 블록 길이 증가 시 오히려 성능 개선, neural BP·MCMC와 유사; `⟨256,32⟩` bicycle code IMR-SPA는 `p≈10⁻²`에서 neural BP 대비 낮은 WER.
10. 한계: quantum 전용, HW 미설계, 시뮬만, iteration/latency/throughput 수치 없음.
