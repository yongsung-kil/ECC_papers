### B. 알고리즘 요약 (10줄 내외)

1. 대상은 rate `R=b/c` binary QC-LDPC(circulant permutation `P`, base matrix `B`, degree matrix `W`), 채널은 AWGN, 실험은 IEEE 802.11n `n=648`(`c=24`, `M=27`).
2. 문제: 유한길이 QC-LDPC의 BP 복호가 ML과 격차가 크고, 짧은 사이클(길이 4/6/8)이 FER를 열화시킴.
3. 기존 AVN-RPC는 길이 4 사이클만 제거했는데, 이를 길이 6·8까지 일반화하고 QC의 라벨 베이스행렬 수준으로 재정식화.
4. 핵심 기법: 사이클을 이루는 `(V,C)` 부분행렬에 보조변수노드(AVN) `u_i=v_1+v_2 ...`와 잉여패리티체크(RPC)를 `g/2-1`개씩 추가해 원 서브그래프의 해당 사이클을 제거.
5. 핵심 조건: 라벨 사이클은 degree matrix에서 `Σ(-1)^i μ_i = 0 mod M`(식 (19))를 만족, 이 two-diagonal 구조 덕에 확장행렬을 해석적으로(식 (13)(20)(21), 계수 `a_i` 재귀식 (22)) 구성.
6. 정당성: Thm1(확장 부호는 원 부호와 등가·해당 사이클 제거), Thm2(stopping set 수 불변→BEC에서 오류확률 비증가); AVN은 bidiagonal 구조로 항상 결정되어 stopping set 안 됨.
7. QC 재정식화로 라벨 그래프의 사이클 하나가 `M`개 사이클을 대표 → 사이클 리스트 저장 메모리 `M`배 절감.
8. 복호(Algorithm 1): 우선 일반 BP, 실패 시 정렬된 사이클 리스트에서 `L_max`개까지 하나씩 골라 H를 확장 후 BP 재실행(AVN LLR 초기값 0), 성공하거나 리스트 소진 시 종료.
9. 결과: FER≈1e-4에서 최대 이득 약 0.5dB(시도 8회면 ~0.25dB), 저rate일수록 이득 큼; 최대 시도 시 LED(31 trial) 대비 FER·복잡도 모두 우위.
10. 한계: 시뮬레이션(FER 곡선)뿐, HW/gate/throughput 없음, 이득이 부호·rate 의존적이고 재시도 기반이라 최악 latency 증가.
