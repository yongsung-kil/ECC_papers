### B. 알고리즘 요약

1. 대상: BPSK/AWGN 채널의 irregular QC-LDPC 부호(광통신 FEC 3세대 후보), 주어진 degree distribution 하에서 error-floor를 낮추는 부호구성이 목표.
2. 문제: irregular 부호는 UEP로 Shannon 한계에 근접하나 error-floor가 높아 고속 광통신 적용이 어려움.
3. 관찰: Tanner graph의 짧은 cycle과 cycle 연결성(EMD/ACE)이 error-floor를 좌우 → girth만 키우는 PEG로는 부족.
4. 핵심기법: PEG로 변수노드에 edge를 추가하되, girth·degree distribution 제약을 만족하는 후보 중 ACE 하한이 최선이거나 ACE 하한 미달 cycle 수가 최소인 edge를 선택(Algorithm 2).
5. ACE 정의: 2i-cycle의 ACE = `∑(d_i − 2)`(`d_i`=cycle 내 i번째 변수노드 degree), 값이 클수록 cycle의 외재정보 연결이 좋아 error-floor에 유리.
6. 확장: 대수적(algebraic) CPM 좌표 표현으로 변환해 Algorithm 1로 각 cycle의 ACE type과 개수를 검출, graph 표현과 교대 수행하며 edge 추가.
7. 구현 디테일: 변수노드를 column weight 내림차순으로 처리하고, 최저 degree check node 집합에서 ACE 기준 최적 edge 선택.
8. 최적화: EXIT chart로 각 그룹의 최적 check-node degree distribution `ρ(x)`를 먼저 결정(G1/G2 두 그룹).
9. 결과: 동일 degree distribution의 [8][10] 대비 ACE 우수+최단cycle 적음, [9] 대비 ACE 우수하나 최단cycle 많음에도 시뮬(SPA 50 iter)에서 error-floor 영역 WER/BER 최우수.
10. 한계: HW 미설계, 시뮬만, rate 0.75/0.833의 소형 예제(7×28×72, 6×36×36)뿐, waterfall 영역 이득은 불명.
