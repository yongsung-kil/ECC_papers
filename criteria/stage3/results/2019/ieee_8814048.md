### B. 알고리즘 요약

1. 시스템: 위성/무선 통신용 FEC 코덱을 FPGA(Xilinx Artix-7 xc7a200)로 구현, 내부 DVB-S2 LDPC(codeword `64800`bit) + 외부 RS 코드 캐스케이드.
2. 문제: binary LDPC 복호 후 남는 잔여오류(error-floor) 제거 필요 - 순수 LDPC로는 낮은 오류율 영역 성능 부족.
3. 채널 모델: 가우시안 분포 오류를 인코딩된 패킷에 주입(AWGN 유사), 최대 50 iteration의 soft-decision 복호.
4. 핵심 기법 1단: DVB-S2 LDPC 디코더 - 6개 code rate 지원, `90`개 처리블록의 부분병렬 구조, modified min-sum으로 parity check 계산.
5. 핵심 아이디어: LDPC 복호 후 잔여오류(선택 rate 1/4,1/2,3/4에서 5bit 이하)를 외부 RS로 정정 - 캐스케이드로 error-floor 소거.
6. 핵심 기법 2단: 외부 RS `(4095,4083)` 코드(12bit 심볼, 최대 6워드 정정), Vivado IP core 사용, 설정 가능 codeword 길이 모드로 합성.
7. 부수 트릭: 인코딩 후 codeword bit 셔플(인터리버)로 burst error를 codeword 전체에 분산.
8. 학습/최적화: 없음(고정 표준 부호).
9. 결과: FPGA 합성 후 210MHz 동작, 로직셀 63% 점유, 복호 품질은 MATLAB sum-product 대비 0.4dB 열화, throughput R=3/4에서 24.228Mbit/s.
10. 한계: DVB-S2 방송 표준 특화, 부분병렬 throughput 낮음(Mbps급), gate count·65nm 급 지표 없음, NAND 무관.
