### B. 알고리즘 요약

1. 대상: 2차원 자기기록(TDMR)용 2D-LDPC 부호, 홀수 소수 `p` 기반, 코드워드를 `hp×wp` 평면으로 취급, binary erasure.
2. 문제: TDMR은 read head가 여러 track을 동시에 읽어 물리적 결함이 2차원 burst erasure를 유발 → Matcha의 2D-LDPC 하한(`p×p`)을 넘는 정정 보장을 원함.
3. 모델: 크기 `p×p×p` 기본 cube `Id`(대각선만 1)를 순열 `P`,`Q`로 변환해 `c×h×w` 직육면체로 쌓고, 각 bit layer가 하나의 parity check 방정식을 정의.
4. 핵심기법: Matcha 구성의 `b0`(식3)를 `b(I,J,K)=⌊(I-1)/p⌋·ϕ(J+1)ϕ(K+1)`(식5)로 교체, `ϕ(x)=x mod'(p-1)`.
5. 핵심 식: `g(i,j,k)=Id(i, j+I+1, k+Iϕ(J+1)ϕ(K+1))`(식8) - cube 순열이 bit position에 미치는 효과를 `Id` 인수로 환원, 정정 증명의 기반.
6. 정정 전략: parity check 행렬 좌상단 identity 구조를 이용, erased bit를 layer별 방정식으로 순차 복구(좌반부 `p×p` → 우반부 `p×(p-1)`).
7. 핵심 정리: `c≥p³+p²+p`이면 크기 `p×(2p-1)` burst erasure를 정정, code rate = `(hwp-c)/hwp`, hw→∞일 때 rate→1.
8. 증명 도구: Lemma 2~8(각 layer에서 목표 위치만 1이고 나머지 0인 `i`의 존재를 modular 산술로 보장).
9. 결과: Matcha 부호의 `p×p` 대비 `p×(2p-1)` burst erasure 정정 하한 개선(이론적 하한, 시뮬레이션 없음).
10. 한계: 복호는 generalized belief propagation을 향후 시뮬로 평가 예정이라 명시, 디코더/HW/성능수치 전무, NAND 무관.
