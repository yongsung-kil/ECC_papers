### B. 알고리즘 요약

1. 시스템: 5G eMBB BG2(base graph 2) 기반 MET QC-LDPC, rate 1/3(BG2 nested m=32,n=22) 및 1/5(m=52,n=42), 정보길이 k=80~1600, AWGN+QPSK, SPA 50 iteration.
2. 문제: 5G eMBB QC-LDPC는 BLER 10^-2 최적화라 BLER 10^-5에서 error-floor 발생, 무선용 length granularity(transport block) 대응 필요.
3. 모델: CPM `P^k` 블록의 `H`, exponent matrix `E(H)`, base graph `M(H)`, exponent chain `Σ(-1)^k a_k ≡ 0 mod L`(식1)로 cycle 판정, EMD로 cycle 연결강도 평가.
4. 핵심기법: modular lifting `E(Hk)=E(H0) mod Lk`(식2)로 length adaptation, 5G의 8 cyclic group `Lk=a·2^i`(식3, a∈{2,3,5,7,9,11,13,15})로 family 구성.
5. 핵심식: simulated annealing lifting `E(H)=argmax_{girth,EMD} M(H)`로 각 circulant size별 girth·EMD 최대화 exponent matrix 8종 생성.
6. 확장: 짧은 길이(rate 1/3 k<232, rate 1/5 k<380)의 0.6 dB gap 해소를 위해 Hamming distance 기준 sieving(Alg.1)으로 code distance 개선(예: CPM32에서 31→44).
7. 트릭: Number Geometry 확률적 code distance 추정(Alg.2) — 생성행렬을 lattice에 embedding 후 BKZ/LLL basis reduction + Kannan embedding으로 SVP 근사 풀어 최소 Hamming weight 추정.
8. 최적화: Butler-Siegel(Th.12) permanent 기반 code distance 상한을 검색범위로, random permutation으로 탐색영역 Exp/Var 최소화(GPU QR-decomposition).
9. 결과: k>600에서 5G eMBB 대비 rate 1/3 0.1~0.3 dB, rate 1/5 약 0.3 dB coding gain (FER 10^-5), k<600은 near-same 성능.
10. 한계: computer simulation만, HW 미설계, rate 1/3·1/5 저rate·5G 무선표준 특화, code distance 추정이 NP-hard(확률적 근사로 완화).
