> 총평: 자기기록(PR/ISI) 채널의 SC-LDPC turbo equalization 스케줄링 개선으로, NAND binary QC-LDPC ECC와 채널·부호구조가 달라 이식성 낮음.

### B. 알고리즘 요약

1. 시스템: PR 채널(PR1=[1 1], PR2=[1 2 1])에서 `(3,21)`-SC-LDPC 코드(rate `R=6/7`), 결합길이 `L=20`, 확장인자 `N=35`, memory `mcc=2` 사용.
2. 문제: 기존 turbo equalization(C-TE)은 SC-LDPC만 슬라이딩윈도우로 복호하고 BCJR은 전체 코드워드 복호 후에야 동작해 수렴이 느림.
3. 채널 모델: `y_j = Σ h_m x_{j-m} + n_j` (AWGN), BPSK 변조, 코드율 반영 SNR 정의식 (3).
4. 핵심기법1(SW-TE): BCJR 검출기와 SC-LDPC 디코더 **둘 다** 동일 윈도우 크기 `W`로 슬라이딩, 이전 복호된 노드 LLR(`L_out`)을 다음 윈도우 복호에 재사용.
5. 의미: 앞서 확정된 노드 정보를 재주입해 C-TE보다 빠른 수렴과 BER 개선을 얻음.
6. 핵심기법2(RC-SW-TE): turbo iteration `l`이 커질수록 윈도우 내 위치 `p = 1..W-l+1`만 갱신, 나머지 trellis/variable 노드 갱신 생략(타깃노드 `p=1`은 항상 갱신, `ITURBO ≤ W-1`).
7. 구현 디테일: 복잡도를 평균 trellis/variable 노드 갱신 수 `U_avg`(식 4)로 정량화.
8. 학습/최적화: 없음(구조적 스케줄 변형).
9. 결과: SW-TE가 C-TE 대비 약 0.4~0.5 dB coding gain, 수렴 지점 `U_var` 약 60% 감소; RC-SW-TE는 SW-TE 대비 약 20% 추가 복잡도 감소(BER 손실 미미). `W=8`, `ILDPC=20`.
10. 한계: HW 미설계, 시뮬레이션뿐, PR(ISI) 채널·BCJR 검출기 전제.
