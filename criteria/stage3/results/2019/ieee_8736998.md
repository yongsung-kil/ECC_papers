### B. 알고리즘 요약

1. 대상: 랜덤 variable-regular / irregular LDPC 앙상블(예제 rate 0.5, girth 6, `dv=3` 또는 `dc=15`), 블록길이 `n=500~20000`.
2. 문제: error-floor를 지배하는 Tanner graph 부분구조(TS/ETS/LETS/ABS/EABS/SS)의 개수를 세는 것은 NP-hard지만, 점근적 평균 개수는 예측 가치가 큼.
3. 모델: bin-cell configuration 모델로 랜덤 Tanner graph를 균등 생성, degree distribution `λ(x), μ(x)`만으로 기대값 계산.
4. 핵심기법(Theorem 1): 임의 local 구조 `S`의 기대 개수 `E(X_S) = Θ(n^{|V(S)|-|E(S)|})` — 노드수-에지수만으로 점근 차수 결정.
5. 의미(Lemma 1, Corollary 1): 구조의 사이클 수가 0/1/≥2이면 평균 개수가 각각 ∞ / 양의 상수 / 0으로 수렴 → 상수 케이스(사이클 정확히 1개)만 error-floor에서 유의미.
6. 확장: 상수로 수렴하는 구조를 "단일 simple cycle + 변수노드에 붙은 tree"로 특성화, 재귀적 카운팅을 Catalan number 일반화로 해결.
7. 세분화: 기존 연구는 (a,b) class 전체를 다뤘으나, 본 논문은 동일 class 내 non-isomorphic 개별 구조가 서로 다른 점근 거동(∞/상수/0)을 보일 수 있음을 규명.
8. 부수 결과(Theorem 4/5): 특정 길이·노드 degree 조합의 사이클 기대 개수 공식을 유도, LETS/ABS/SS/ETS multiplicity 계산에 활용.
9. 결과: (3,6)·(4,8) regular 및 irregular(`λ=0.4286x²+0.5714x³, μ=x⁶`) 앙상블에서 유한길이 exhaustive search 결과와 점근 예측값이 짧은 블록길이에서도 잘 일치.
10. 한계: HW·디코더 미설계, 순수 이론, ≥2 사이클 구조(유한길이 지배 TS 다수)는 점근적으로 0이라 유한길이 예측 불가, 랜덤 앙상블 전용(QC-LDPC 미포함).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| trapping/absorbing set 열거 이론 | 대응 없음 | Prime ECC는 고정 QC-LDPC 시뮬레이터, 앙상블 열거 기능 없음 |
| error-floor 구조 분석 | `corner.cpp` (수렴실패 추적) | 개념적 인접뿐, 이 논문의 점근 열거식과 직접 대응 없음 |
| H-matrix / 부호구조 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC H-matrix 고정, 논문은 랜덤 앙상블이라 정합 안 됨 |

적용 가치: **낮음** — 순수 점근 열거 이론이고 랜덤(비-QC) 앙상블 대상이라, 고정 QC-LDPC를 쓰는 Prime ECC의 부호설계/디코더/HW 어디에도 직접 이식할 기법이 없다.
