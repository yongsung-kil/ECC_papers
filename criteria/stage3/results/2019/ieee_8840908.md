### B. 알고리즘 요약

1. 시스템: PR(partial response) 채널 = memory `D`의 FIR `(h0..hD)` + AWGN, ISI 존재. binary LDPC(regular, rate 0.5~0.82, length 204~4095), BPSK, 자기기록/통신 대상.
2. 문제: PR 채널 LDPC 복호를 QP/LP로 풀면 IPD·LP solver가 채널 memory 길이에 지수/큰 복잡도, ADMM은 memoryless만 연구됨(PR 미해결).
3. 모델: 수신 `ri = Σ_t h_t(1-2x_{i-t}) + n_i`, 인접 심볼이 ISI로 상관, ML 복호를 QP `min f(x)` s.t. check polytope `PP` 제약으로 완화.
4. 핵심기법(ADMM-PD): penalized objective `fp = f + g`에 augmented Lagrangian 적용, x-update/z-update/λ-update 교대 반복. z-update는 check polytope로의 Euclidean projection.
5. 핵심식 의미: x-update (4)가 PR 채널 항 때문에 내부 QP를 다시 반복해야 하는 2중 loop-nesting 구조 -> latency 병목.
6. 확장/핵심 기여: l1/l2 penalty로 x-update를 closed-form (14)/(15)로 유도, 인접 `x_{i±q}`는 이전 iteration `x^{k-1}` 값으로 근사해 내부 loop 제거.
7. 구현 디테일: `f` gradient의 inner/outer summation을 O(ND)로 사전계산·저장, `si`는 상수항 활용, x/z/λ update 병렬 수행 가능.
8. 최적화: penalty parameter `μ, α1, α2`는 시뮬로 튜닝(참고문헌 [13] 방식).
9. 결과: 전체 복잡도 O(ND + M·dc)로 채널 memory `D`·`dc`에 선형(Table I에서 유일). waterfall에서 TE(BCJR+BP)와 comparable, 고SNR에서 iteration 증가 시 TE 능가. PR16 채널에서 BCJR(2^16 states) 대비 복호시간 대폭 단축. Imax 900~2000 필요.
10. 한계: HW 미설계·시뮬만, PR/ISI 채널 특화, 수백~수천 iteration 필요, penalty 파라미터 시뮬 튜닝 의존.
