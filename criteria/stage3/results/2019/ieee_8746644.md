### B. 알고리즘 요약

1. 시스템: Globally-coupled LDPC(GC-LDPC), CN형/VN형 두 구조. 예제 부호 `(1629,908)` rate 0.5574, `(3372,2249)` rate 0.67, `(2810,1967)` rate 0.7, AWGN, SPA 50 iteration.
2. 문제: GC-LDPC의 H는 replacement set이 핵심인데, RC-constraint(두 열/행이 공통 1을 정확히 1개)를 만족해야 girth≥6 보장. 기존 finite-field 구성 대안이 필요.
3. 모델: `(q,n,g,r,λ=1)`-BIBD의 t개 순환행렬이 RC-constraint를 만족한다는 관찰을 이용.
4. 핵심기법: Bose-BIBD 두 클래스 — Class-I는 `GF(12t+1)`, `g=4,r=4t`; Class-II는 `GF(20t+1)`, `g=5,r=5t`. base block `B_{i,0}={α^{2i},...}`의 incidence vector를 첫 열로 하는 `(12t+1)×(12t+1)` circulant `G_i` 생성.
5. 핵심식: 조건 `α^{4t}-1=α^c`(c는 홀수) 하에서 primitive element α로 base block 정의 → co-block 평행이동으로 translate class `Q_i` 형성, t개 순환행렬 `G_I=[G_0..G_{t-1}]`가 replacement set.
6. 확장: 이 replacement set의 부분집합을 local matrix R, global matrix X로 무작위 선택해 `H_gc,cn`(식1)/`H_gc,vn`(식2) 구성. 4-cycle 회피 위해 대각 nonzero를 0으로 치환(`G̃_i`).
7. 구현 디테일: 인코딩 복잡도 절감을 위해 X를 identity 행렬 한 줄로 대체 가능. VN-GC는 결합이 sparse해 iteration당 정보교환↓, 수렴에 불리.
8. 학습/최적화: 없음(순수 대수적 구성). 비교군은 PEG 알고리즘 부호.
9. 결과: CN-GC-BIBD-LDPC가 PEG 대비 0.15dB 개선(Fig.1) 또는 유사(Fig.2), CN-GC-FF-LDPC(PEG보다 0.2dB 열세)보다는 우수. AWGN 기준.
10. 한계: code length/rate 유연성 없음, HW 미설계, error-floor·정정능력 직접 보증 없음, AWGN SPA 시뮬만.
