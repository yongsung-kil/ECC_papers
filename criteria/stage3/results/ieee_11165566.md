### B. 알고리즘 요약

1. 5G-NR LDPC BG1, code rate 1/3, code length 26112, QC-LDPC 구조, AWGN 채널(오픈소스 HDL AWGN 코어 사용), block-parallel 유니버설 디코더.
2. 기존 universal 디코더는 다양한 base graph를 지원하려는 유연성 때문에 성능 저하가 있고, 특히 저비트폭(4-bit) 양자화 시 NMSA는 severe error-floor 발생.
3. 채널/노이즈 모델은 표준 AWGN, 별도 NAND 특화 모델 없음(무선 5G-NR 표준 대상).
4. 핵심 기법: Adaptive Quantized and Normalized MSA(AQNMSA) — 매 iteration마다 현재 노드의 `check-ratio`(현재 edge-type 노드 중 unsatisfied check 비율)와 `edge-type`을 인덱스로 LUT에서 양자화 구간 `T`와 정규화 인자 `α`를 조회해 내부 신호에 적용.
5. 핵심 식: VNU→CNU 변환 시 `qL, T_cur`로 high-bit VNU 출력을 저비트 CNU 입력으로 변환하고, CNU→VNU 변환 시 `qH, T0, T_cur`로 복원 후 `α_cur`를 곱함 — 반복별 통계(check-ratio)에 따라 quantization 손실을 최소화.
6. LUT `((check-ratio, edge-type), (T, α))`는 multi-edge-type density evolution(MET-DE) 툴로 부호별 사전계산되어 디코더에 프리로드됨.
7. HW 구현: Q_Cal/A_Upd 모듈에 양자화 변환·정규화 단계 추가, Exit_Chk 모듈이 check-ratio 계산 및 LUT 조회 담당, 6-stage 파이프라인(IP-BS-QCa-QCo-AU-AW), QSN(QC-LDPC shift network)으로 lifting block 내 데이터 시프트.
8. 최적화 절차: LUT 파라미터는 MET-DE로 사전 설계(오프라인), 런타임 학습 없음.
9. 결과: 4-bit 메시지 비트폭에서 FER 10^-4 도달 SNR이 AQNMSA -1.03dB vs 고정양자화 NMSA -0.47dB(약 0.56dB 개선), NMSA는 error-floor 뚜렷한 반면 AQNMSA는 steep descent.
10. 한계: 리소스 사용량(Combinational ALUTs 116066 vs NMSA 4-bit 96216)과 최대주파수(71.24MHz vs NMSA 4-bit 93.93MHz)가 동일 비트폭 NMSA 대비 오히려 열화, 5G-NR 표준 부호로만 검증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 적응형 양자화/정규화(check-ratio 기반 T, α LUT) | `decoder.cpp` `CNU_Update_New_Mag()`, `ecc_data.h` `PARAM_LLR` | 기존 iteration별 정적 LLR 테이블을 반복 중 실시간 check-ratio 기반 동적 LUT로 확장하는 아이디어로 직접 대응 |
| VNU→CNU/CNU→VNU 비트폭 변환(Q_Cal, A_Upd) | `decoder.cpp` `VN_Cal_HD()` 등, `decoder.h` `Get_VNU_*` | 저비트폭 양자화 변환 로직이 VN/CN 처리 단계와 직접 대응 |
| check-ratio 계산 및 LUT 조회(Exit_Chk) | `decoder.cpp` `Get_VNU_Table_Idx()` | iteration-index 기반 테이블 조회를 check-ratio 기반으로 확장하는 방식으로 관련 |
| block-parallel 파이프라인(6-stage, QSN) | HW 디코더 pipeline (z=32, HCU, GT) | 파이프라인 단계 수·역할이 다르나 lifting 병렬 처리 구조 자체는 유사한 계열 |
| MET-DE 기반 LUT 사전설계 | `ecc_top.cpp` `Load_PCM()`(주변) | 오프라인 설계 툴 자체는 대응 없음, LUT 결과만 로드하는 구조는 유사 |

> 적용 가치: 높음 — check-ratio 기반 적응형 양자화/정규화는 저비트폭 min-sum의 error-floor를 개선하는 실증된 기법으로, Prime ECC의 정적 LLR 테이블을 동적화하는 방향으로 이식 시 soft-read 비용 절감과 정정능력 향상을 동시에 노릴 수 있음.
