### B. 알고리즘 요약

1. 시스템: convolutional encoder(g=[111;001;100]) + bit interleaver + 8-PSK 매핑(subset partitioning), AWGN 채널, 부호화 비트열 길이 `Lc=6000`.
2. 문제: 반복복호(iterative decoding)는 원래 최적화 문제로 도입되지 않아 수렴성 분석이 어려움.
3. 가정: demapper와 SISO decoder가 교대로 사후확률(APP)을 교환하는 turbo-like 구조, 채널은 AWGN.
4. 핵심 기법 1단: 반복복호를 두 개의 임베디드 최소화 과정으로 재정식화, Fermi-Dirac divergence(Bregman divergence) `D_FD(p,q)` 사용.
5. 핵심 식: demapper는 `min_λ2 D_FD(p_{Bλ1+θm}, p_{B(λ1+λ2)})`, decoder는 `min_λ1 D_FD(p_{Bλ2+θc}, p_{B(λ1+λ2)})` 최소화 - 반복복호의 각 서브블록이 실제로 거리 최소화를 수행함을 증명.
6. 확장: proximal point penalty term `μm, μc` 추가(Hybrid Proximal Point, HPP)로 매 반복마다 지정 criterion이 감소함을 보장.
7. 부수 기법: Hybrid Minimum Entropy Algorithm(HMEA)은 엔트로피 `E_{B(λ1+λ2)}` 최소화 제약을 추가, 함수 `f_η(p)` 도입(η는 step-size, 0.05 사용).
8. 최적화 절차: 별도 학습 없음, closed-form 업데이트 식(그래디언트 zeroing)으로 각 iteration의 유일해 도출.
9. 결과: HPP는 classical iterative decoding과 동일 BER/iteration 수, HMEA는 중간 EbN0 구간에서 BER 개선(반복 횟수는 classical 이하이나 계산복잡도는 더 높음).
10. 한계: convolutional/BICM-8PSK 대상 시뮬레이션뿐이며 LDPC 실험 없음, HW 설계 없음, 계산복잡도(`f_η` distortion function) 증가 트레이드오프 존재.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 반복복호 수렴 재정식화(Fermi-Dirac divergence) | 대응 없음 | Prime ECC는 min-sum 기반 메시지 전달이며 정보기하학적 재정식화 개념 자체를 사용하지 않음 |
| Hybrid Minimum Entropy Algorithm (HMEA) | `decoder.cpp` `LDPC_Decoder()` | 반복종료/신뢰도 기준으로 원용 가능성 있으나 BICM-ID demapper 구조 전제라 직접 이식 어려움 |
| 8-PSK demapping / SISO decoder 구조 | 대응 없음 | NAND binary LDPC는 demapper 개념이 없어 대응 모듈 없음 |

> 적용 가치: 낮음 — BICM-ID(convolutional+8PSK) 대상 이론 논문이며 binary QC-LDPC 디코더 구조와 직접 대응되는 요소가 없음.
