### B. 알고리즘 요약

1. 대상은 이진 선형 코드의 Tanner 그래프이며, AWGN 채널에서 binary antipodal modulation으로 전송된 codeword의 decoding failure(non-codeword output)를 다룸.
2. 풀려는 문제는 iterative message-passing 디코딩(sum-product, min-sum)이 왜 codeword가 아닌 출력을 내는지를, LP decoding·graph cover decoding·computation tree 분석이라는 세 이론 틀 사이의 관계로 설명하는 것.
3. 핵심 가정은 Wiberg의 computation tree 모델(디코더는 유한 깊이 계산 트리 위의 최소비용 configuration을 출력)과 Feldman의 LP decoding(`fundamental polytope` `P(H)` 위 vertex 탐색)을 동시에 사용.
4. minimal pseudocodeword(`fundamental cone`의 edge에 놓인 것)와 irreducible pseudocodeword(다른 unscaled pseudocodeword의 합으로 못 쓰는 것)를 구분하고, 문헌에서 두 개념이 혼용되던 오류를 반례(Example 3.4)로 정정.
5. `Proposition 3.7`: extended polytope 상의 rational point가 minimal realization되는 cover의 degree는 좌표들을 정수로 만드는 최소 양의 정수와 같음 — LP pseudocodeword를 실현하는 최소 cover 차수를 정확히 산출.
6. `Theorem 4.4`: 평균 variable node degree `a_v`와 평균 check node degree `a_c`가 특정 부등식(`a_v ≥ 2`, `a_c > 2` 등)을 만족하면 fundamental polytope의 모든 rational point가 connected cover 위에서 minimal realization 가능함을 증명.
7. 증명은 spanning tree 위 두 edge를 "cross"시켜 서로 다른 connected component를 하나로 합치는 구성적 알고리즘(Lemma 4.3, 정리 4.4 proof) 사용.
8. 학습/최적화 절차 없음 — 순수 그래프이론/조합론적 증명.
9. 결과는 정성적(정리·반례)이며 수치 실험/시뮬레이션 결과 없음; Example 4.7에서 graph cover pseudocodeword와 computation tree pseudocodeword 간 불완전한 대응관계를 보이는 반례 제시.
10. 한계: 알고리즘·부호설계·HW 구현이 전혀 없는 순수 이론 논문이며, computation tree pseudocodeword가 항상 graph cover pseudocodeword에서 유도되는지는 추측으로만 남김.
