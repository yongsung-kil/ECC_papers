> 총평: QC-LDPC 구조(HW 친화적)를 유지하면서 PEG 기반 무작위 구성법에 가까운 error-floor 성능을 얻는 부호 구성 알고리즘으로, 부호설계 이식 관점에서는 참고가치가 있으나 H-matrix 구조가 이미 고정된 Prime ECC에 이식하려면 재검증 부담이 큼.

### B. 알고리즘 요약

1. 대상은 QC-LDPC 부호(base circulant matrix `I(p_{i,j})` 크기 `L×L`)이며, AWGN 채널과 PR4-equalized 자기기록채널(MRC, user density SNR 2.0/2.5)에서 시뮬레이션.
2. 문제: 무작위 구성(PEG 등)은 error-floor가 낮지만 H-matrix가 완전 랜덤이라 VLSI 구현 복잡도가 높고, 반대로 QC-LDPC는 HW에 유리하나 기존 랜덤 QC 구성(Li et al.)은 특히 고rate·MRC 채널에서 정정성능이 랜덤 PEG보다 열등함.
3. 가정/모델: Tanner graph 상 short cycle(특히 girth가 짧은 cycle)이 반복복호 시 extrinsic information 독립성을 해치고 error-floor를 유발한다는 표준 LDPC 가정.
4. 핵심 기법: 4단계 구성 알고리즘 — (1) PEG 기반 후보 check node 집합 탐색, (2) 미래 edge growth에 미치는 영향(생성될 cycle 수·길이)을 고려한 check node 선택, (3) EMD(extrinsic message degree)가 최대인 cycle을 만드는 check node 집합 산출.
5. 핵심 식: 노드의 `unreliable factor λ = Σ(1/l_k)` (l_k는 참여하는 k번째 cycle의 길이), cycle의 unreliable factor는 참여 노드 λ의 합 — 값이 클수록(짧은 cycle 다수 참여) 정보 신뢰도가 낮음을 의미.
6. 핵심 기법 2단(step 4): unreliable factor가 최소인 cycle을 형성하는 check node 집합에서 최종 연결 노드를 무작위 선택해 신뢰도 낮은 정보 전달을 최대한 회피.
7. 부수 디테일: critical column/relative column 개념으로 QC 구조(cyclic-shift identity matrix `I(p_{i,j})`)를 하나의 열(critical column)만으로 결정, sub-matrix 내부 cycle을 원천 차단(limit check node set으로 sub-matrix 내부 edge 제한).
8. 별도 학습/최적화 절차 없음(탐욕적/휴리스틱 그래프 구성 알고리즘, GA/NN 미사용).
9. 결과: (1008,504) 부호에서 고 SNR 영역 랜덤 PEG 대비 약 0.1dB 이득 및 error-floor 유의미하게 감소, (4608,4096) 부호에서 PR4-equalized MRC 채널(SNR 2.0/2.5)에서도 랜덤 PEG·랜덤 QC(Li et al.) 대비 우수한 BER/FER.
10. 한계: HW 구현/합성 결과 없음(순수 시뮬레이션), 인코딩은 Gaussian elimination으로 생성한 G-matrix 사용(구조적 인코딩 아님), 자기기록 채널(PR4-MRC) 특화 성능평가로 NAND 채널 특성(RBER, retention 등) 검증은 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| PEG+unreliable factor 기반 QC-LDPC H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC-LDPC(base matrix+circulant shift, N_b=129) 고정 구조이므로, 새 구성법 적용 시 H-matrix 전체 재설계·재검증 필요 (§4 "부호설계: 높음"에 해당) |
| short cycle 제거로 error-floor 개선 | `decoder.cpp` `LDPC_Decoder()` (간접) | 부호설계 단계 개선이라 디코더 알고리즘 자체 수정은 불필요하나, 개선된 H-matrix를 쓸 경우 min-sum 성능(특히 error-floor)이 간접적으로 향상됨 |
| EMD/unreliable factor 기반 check node 선택 휴리스틱 | 대응 없음 | Prime ECC 코드베이스에 H-matrix 생성/최적화 도구에 대응하는 모듈 언급 없음(고정 PCM 로드만 존재) |

적용 가치: `중간` — QC-LDPC 구조를 유지하며 error-floor를 낮추는 부호 구성법은 개념적으로 고rate NAND ECC에도 유효한 방향이지만, Prime ECC의 H-matrix가 이미 고정되어 있어 이식 난이도가 높고(§4 "부호설계" 항목 참고) 실제 NAND 채널 검증도 없음.
