### B. 알고리즘 요약

1. 대상 코드는 논문 서두 예시인 (10,5) 소형 binary irregular LDPC(Tanner graph 예시), AWGN 채널, BPSK 미기재.
2. 기존 SP/APP 등 확률 기반 디코더 대신, 확률 계산 없이 신경망 학습으로 디코딩하는 새로운 접근을 제안.
3. 채널모델: 수신벡터 `r = c + n` (표준 AWGN 가산잡음).
4. 핵심 기법: 입력층(수신벡터)-출력층(체크노드) 2계층 MLP를 Tanner graph/H-matrix 구조 그대로 구성, 각 체크노드는 연결된 변수노드들의 XOR 관계를 근사.
5. XOR 근사식 `x⊕y = x(1-y)+y(1-x)`를 differentiable하게 정의해 gradient descent로 오차함수 `E = 1/2 Σ Oi^2`(목표값 0)를 최소화하도록 수신벡터 성분을 반복 갱신.
6. 확장 없음 — 단일 2계층 구조만 제시.
7. 부수 트릭: iteration 종료조건으로 `E ≤ 0.2` 임계값 사용(최대 200회), 수렴 후 수신벡터 성분을 0/1로 하드 매핑.
8. 학습/최적화: gradient descent로 매 반복마다 입력 성분(사실상 LLR 아닌 벡터값) `ΔCj = -η·∂E/∂Cj`를 갱신(가중치 학습 아닌 입력 최적화).
9. 결과: η=0.008~0.04, SNR=11~14dB에서 BER 4.2e-3~2.75e-5 수준으로 SPA(3.7e-3~2.1e-5) 대비 전반적으로 열세.
10. 한계: (10,5) 초소형 코드 1개만 검증, HW/파이프라인/양자화 전무, 저자 스스로 "simple prototype"이라 명시하며 성능 개선 여지가 크다고 언급.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| MLP 기반 XOR 근사 디코더 전체 | 대응 없음 | 프로파일에 "NN/딥러닝 디코더는 대응 없음"으로 명시된 판정 필터 대상 |
| gradient descent 기반 수신벡터 보정 | 대응 없음 | min-sum/message-passing 구조와 근본적으로 다른 방식 |
```

적용 가치: 낮음 — Prime ECC는 min-sum 기반 message-passing HW 시뮬레이터로, NN 전면 대체 디코더는 이미 판정 필터에서 이식성 하로 분류됨.
