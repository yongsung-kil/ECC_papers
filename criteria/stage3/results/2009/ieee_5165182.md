> 총평: array LDPC code의 stopping distance/minimum distance를 수학적으로 완전 규명하는 순수 이론 논문으로, Prime ECC 3.1의 H-matrix 구성이나 디코더 알고리즘에 직접 이식할 여지 없음.

### B. 알고리즘 요약

1. 대상은 array LDPC code(QC-LDPC의 한 클래스): 홀수 소수 `q`와 정수 `1<=m<=q`로 정의되는 이진 `qm x q^2` parity-check matrix `A(m,q)` 및 그 일반화 `PA(m,q)`(proper array), `IA(m,q,S)`(improper array).
2. 풀려는 문제: iterative decoding(BEC) 성능을 좌우하는 stopping distance `s(H)`는 parity-check matrix 선택에 의존하며, 일반적으로 `d_min(C) <= s(H)`인데 임의 코드에 대한 stopping set 완전 특성화가 어려움.
3. 핵심 가정/모델: binary erasure channel(BEC) 상 iterative decoding, Tanner graph 관점에서 stopping set을 정의(모든 이웃이 2회 이상 연결된 variable node 부분집합).
4. 핵심 기법: circulant permutation matrix `P` (identity를 순환이동)로 구성된 array를 `m x q` block 구조로 정의, cyclotomic coset을 이용해 improper array code(IA)의 block-row 상한을 결정.
5. 핵심 식의 의미: PA/IA 코드의 parity-check matrix 구조(등차수열 vs 임의 정수열)에 따라 4-cycle/6-cycle 존재 여부를 조합론적으로 분석, girth-6 성질을 이용해 stopping set 크기 하한을 도출.
6. 확장: `A(m,q)`, `PA(m,q)`, `IA(4,q,S)` 각각에 대해 `m=2,3`과 `(m,q)=(4,5)`, `(m,q)=(4,7)` 케이스별로 stopping distance를 정확히 산출.
7. 부수 트릭: 특정 크기(4,5,6,7,8열)의 submatrix가 stopping set을 이루는지 여부를 modular arithmetic(mod q) 연립방정식의 모순 여부로 판별하는 증명 기법 반복 사용.
8. 학습/최적화 절차: 없음(순수 수학적 증명, 시뮬레이션/최적화 없음).
9. 결과: `m=2,3`일 때 `s(A(m,q))=d_min=2(m+1)`, `(m,q)=(4,5)`일 때 `s=d_min=8`이 등식으로 성립함을 증명; 하한 `s(A(m,q))>=(m+1)(m+2)/(m-1)` (m>3, q>3)도 제시.
10. 한계: 특정 (m,q) 범위에만 완전한 결과, 일반 m,q에 대한 stopping distance는 open problem으로 남김; 실제 디코더 성능 시뮬레이션이나 HW 구현 전혀 없음.
