### B. 알고리즘 요약

1. 채널은 AWGN, BPSK 변조, iterative message-passing 복호를 전제로 한 이진 QC-LDPC 부호 구성법 논문.
2. 기존 Euclidean geometry 기반 QC-LDPC(정규 부호)는 column/row splitting으로 irregular 확장 시 QC 구조가 깨지는 문제가 있었음.
3. EG(d, p^s)의 μ-flat(하이퍼플레인) 개념으로 base matrix `B`를 정의하고, `μ1`-flat과 `μ2`-flat의 포함관계로 row/column weight를 결정.
4. 정규 base matrix `B`를 column decomposition(및 row decomposition)하여 irregular base matrix `B̄`를 얻음 (최대 `D=μ2-μ1-1`개 column weight, `T=μ1+1`개 row weight).
5. `B̄`의 각 '1' 원소를 shift 값 `a(m,n)∈[0,q-1]`으로 치환한 shift matrix `S`를 만들고, 순환치환행렬(circulant permutation matrix)로 전개해 QC 행렬 `H`를 구성 (girth는 Theorem 2.1에 의해 `μ1=μ2-1`이면 6, 아니면 4).
6. 짧은 사이클을 깨기 위해 block-cycle 조건식 `Σ(a(mk,nk)-a(mk+1,nk))≡0 mod q`을 피하도록 shift 값을 선택.
7. degree distribution 최적화는 EXIT chart curve fitting: irregular VND/CND 전이함수(`IE,V`, `IE,C`)를 `λ`, `ρ` 가중합으로 구성하고 CND 곡선 위에 VND 곡선이 오도록 `av,i` 조정.
8. error-floor 개선을 위해 `λ2ρ'(1)>1`이면 사이클 수가 늘어나므로(식 21), degree-2 variable node 비율 `λ2`를 최적값 이하로 제한 (waterfall SNR은 소폭 악화).
9. 실험: EG(6,2)에서 4288×8040 정규 코드 C1(rate 0.4667)을 기반으로, degree-2 제한 없는 C2(waterfall 우수), 제한한 C3(waterfall+error-floor 균형)를 구성. C3는 기존 방법(C4, [2])대비 BER=10^-6에서 3dB 코딩이득.
10. 한계: 순수 시뮬레이션(AWGN)뿐이며 HW 설계, 복잡도, throughput 분석 없음. rate가 NAND ECC 목표(~0.9)보다 낮은 예제(0.4667~0.7419)만 제시.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| μ-flat 기반 base matrix 구성 및 QC 확장 | `ecc_top.cpp` `Load_PCM()` | H-matrix 자체를 교체해야 해 재검증 부담 큼 |
| EXIT chart curve fitting 기반 degree distribution 최적화 | 대응 없음 | Prime ECC는 특정 고정 QC-LDPC(N_b=129, VN degree 17)를 사용하며 degree distribution 재설계 도구 없음 |
| degree-2 노드 비율 제한(error-floor 개선) | 대응 없음 | trapping-set/error-floor 대책이지만 순수 부호설계 레벨이라 decoder.cpp와 무관 |
| block-cycle 회피용 shift 값 선택 | `ecc_top.cpp` `Load_PCM()` | girth 확보 로직이나 Prime ECC의 shift 값은 이미 고정 |

> 적용 가치: 낮음 — 부호설계(H-matrix) 전용 이론 논문으로 Prime ECC의 고정 QC-LDPC 구조 교체는 이식 난이도가 가장 높은 레이어이며, target rate/length도 NAND 대역과 불일치.
