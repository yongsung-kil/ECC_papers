> 총평: probe-based storage(AFM) 채널의 jitter/noise 통계를 반영한 연판정 계산과 (1,7)-RLL+LDPC 결합복호 벤치마크 논문으로, LDPC 자체는 표준 sum-product이고 채널이 NAND와 무관해 이식 가치가 낮음.

### B. 알고리즘 요약

1. 시스템은 probe-storage(AFM 기반) 장치: 내부 (1,7)-RLL 변조코드 + 외부 `(495,433)` LDPC 코드의 직렬 연접 구조, symbol interval `T`.
2. 문제: probe-storage 채널은 write/read jitter와 colored Gaussian noise(전자/매질 잡음)가 지배적 손상 요인이며, 이를 무시하면 연판정 정확도가 저하됨.
3. 채널 모델: read-back 신호 `r(t) = Σ x_i h(t−iT−J_i^w) + w(t)`, write/read jitter `J^w, J^r`는 상관된 colored Gaussian(표준편차 `σ_J`, truncated Gaussian PDF), 잡음 `w(t)`는 전자+매질 잡음의 합성 colored Gaussian.
4. 핵심 기법: 채널 출력의 조건부확률 `p(y|x_k)`를 jitter·noise 통계를 명시적으로 반영해 해석적으로 유도(등화기 출력 triangular pulse 가정, 식 (3)~(7)).
5. 이 조건부확률이 MCMD(metric computation and modulation decoder)의 (1,7) 코드 forward-backward 연판정 입력이 되어, 기존 Gaussian-근사 방식보다 정확한 a posteriori probability를 제공.
6. 확장: (1,7)-RLL 복호(forward-backward)와 외부 LDPC 복호(sum-product) 간 반복적 피드백(최대 10회 inner-outer iteration, LDPC 자체는 최대 25 iteration)으로 성능 개선.
7. 부수 디테일: MMSE partial-response 등화 target `[-0.1,1,-0.1]` 채택, jitter 진폭은 `|J|≤γT`로 bounded truncated Gaussian 모델링.
8. 별도 학습/최적화 절차 없음 — 실험적으로 추출한 noise/jitter 스펙트럴 특성을 시뮬레이션 파라미터로 사용.
9. 결과: 심한 jitter(σ_J up to 13.2%T)에서 jitter-aware 연판정이 Gaussian-근사 대비 1dB 이상 BER 이득, RS 하드복호 대비 LDPC 연판정이 BER 10⁻⁵에서 1.5~2.5dB 이득.
10. 한계: 특정 probe-storage 채널(AFM indentation, 2~3 symbol ISI)에 특화된 채널모델이며, LDPC 부호/디코더 자체는 표준 sum-product를 그대로 사용해 알고리즘적 개선은 없음(HW 미설계, 시뮬레이션 검증뿐).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LDPC sum-product 복호 | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum 계열이며 본 논문은 표준 sum-product를 그대로 사용해 알고리즘 기여 없음 |
| jitter/colored noise 기반 연판정(soft-info) 생성 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | probe-storage 특유 jitter 통계식이라 NAND read 채널(AWGN/RBER)과 물리적 기반이 달라 직접 이식 어려움 |
| (1,7)-RLL 변조코드 및 결합복호 | 대응 없음 | NAND ECC 스택에 RLL 변조코드 개념이 없음 |

> 적용 가치: 낮음 — probe-storage 특화 채널 통계 모델과 표준 sum-product LDPC 결합 논문으로, NAND binary QC-LDPC 디코더/부호설계에 이식할 알고리즘적 기여가 없음.
