### B. 알고리즘 요약

1. GF(q) 위에서 binary/nonbinary QC-LDPC 코드를 하나의 프레임워크로 구성; 예제 코드는 rate `0.59~0.91`, length `567~12750`(binary), nonbinary는 `GF(2^6)/GF(2^8)` 심볼 기반.
2. 기존 연구가 binary LDPC 구성에 집중되어 nonbinary LDPC 구성/설계 연구가 부족했던 문제를 통합 프레임워크로 동시에 해결하고자 함.
3. 핵심 가정: `m×n` base matrix `W`가 "α-multiplied row-constraint 1, 2"(모든 2×2 부분행렬이 nonsingular)를 만족하면, 이를 `(q-1)-fold` dispersion(binary는 `Ab(δ)`, q-ary는 `Aq(δ)`)으로 전개해 RC-constraint(short cycle 없음, girth≥6)를 만족하는 QC-LDPC parity-check matrix `Hb`/`Hq`를 얻는다는 것이 핵심 정리(Lemma 1, 2).
4. 핵심 기법 1단: 원시원소(primitive element) 기반 구성 - GF(q)의 모든 원시원소 `{α^u1,...,α^uK}`로 `W(1)` 행렬을 만들고, 대각선의 zero 서브행렬을 피해 `γ×ρ` 서브어레이를 취해 QC-LDPC 코드를 얻음 (`Cb(1)`, rate ≥ `(ρ-γ)/ρ`, girth≥6, 최소거리 ≥ `γ+1`).
5. 핵심 기법 2단: 덧셈 부분군(additive subgroup) 기반 구성 `W(2)`(GF(p^m)의 두 서로소 부분공간 coset 이용) 및 곱셈 순환부분군(cyclic subgroup) 기반 구성 `W(3)`(서로소 `k, m`로 `q-1=km` 분해); 세 방법 모두 동일한 dispersion 절차로 binary/nonbinary 코드 각 1개 클래스씩 생성(총 6개 클래스).
6. 부수 트릭: 대각선 zero 서브행렬 위치를 피해 서브어레이를 선택하는 것이 RC-constraint 유지의 핵심 구현 디테일; regular/irregular 여부는 base matrix의 row/column weight 균일성에 따라 결정.
7. 학습/최적화 절차 없음(수학적 폐형 구성, DE/GA 미사용); 성능은 AWGN+BPSK, decoding은 binary SPA 또는 FFT-QSPA(50 iteration)로 시뮬레이션.
8. 결과 예시: binary (8160,7159) QC-LDPC(rate 0.877)가 BER 10⁻⁶에서 Shannon limit 대비 0.95dB, MacKay 랜덤 LDPC와 거의 동일 성능; binary (12750,11553) rate 0.906 코드가 0.9dB 갭; error floor는 Richardson 방법으로 (2331,2007) 코드에서 BER 10⁻¹⁶ 이하로 추정.
9. nonbinary 코드(GF(2^6)/GF(2^8))는 동일 rate/length의 shortened RS 코드(hard BM 또는 KV soft decoding) 대비 WER 10⁻⁵에서 1.7~2.7dB 코딩이득.
10. 한계: 순수 이론/시뮬레이션 논문으로 decoder 알고리즘 개선이나 HW 구현이 전혀 없으며, 6개 클래스 중 다수가 nonbinary(GF(q)) 코드로 NAND용 binary LDPC와는 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| binary QC-LDPC base matrix 구성(원시원소/부분군/순환부분군) | `ecc_top.cpp` `Load_PCM()` | 새로운 H-matrix 구성법 후보이나 특정 rate/length(원소체 크기 종속) 고정이라 재검증 부담 큼 |
| RC-constraint(girth≥6) 보장 설계 | `ecc_top.cpp` `Load_PCM()` | girth 보장 설계 원리는 참고 가능하나 코드 자체 대응 없음 |
| nonbinary(GF(q)) QC-LDPC 구성 | 대응 없음 | Prime ECC는 binary 전용이라 nonbinary 클래스는 이식 불가 |
| FFT-QSPA(q-ary 디코딩) | 대응 없음 | non-binary LDPC 기법으로 binary min-sum 구조와 무관 |

> 적용 가치: 낮음 | 순수 부호구성 이론 논문으로 decoder/HW 요소가 없고, 6개 클래스 중 상당수가 non-binary이며, binary 서브클래스도 특정 유한체 크기에 종속된 고정 rate/length라 Prime ECC의 기존 QC-LDPC H-matrix를 대체할 유인이 크지 않음.
