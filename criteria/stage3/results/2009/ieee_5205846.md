### B. 알고리즘 요약

1. 대상은 이진 선형블록부호(LDPC, BCH, WIMAX)이며 BIAWGN 채널에서 정수계획법(IP) 기반 ML 디코딩을 다룬다.
2. LP 완화(RIPD)로 얻은 비정수해를 분리(cut generation)로 정수해에 근접시키는 기존 NSA(New Separation Algorithm)의 한계를 개선하는 것이 목표.
3. 채널모델은 BIAWGN이며 cost vector `c_j = log(P(y_j|x_j=0)/P(y_j|x_j=1))`로 LLR을 사용.
4. 핵심 기법은 forbidden-set(FS) 부등식을 확장한 ICGA2(Improved Cut Generation Algorithm 2)로, `|A_i|≥2`인 특수 케이스까지 cut을 생성 (`k_i` = 홀짝, `j_max` = 최대 비정수 성분).
5. 부등식 (5),(6)은 기존에 놓치던 비정수해를 추가로 잘라내는 valid cut으로, 최악 복잡도 `O(mn)`을 유지.
6. RSA(Revised Separation Algorithm)는 NSA에 CGA3(대체 파리티검사행렬 기반 cut)까지 순차 적용하는 확장판.
7. 새로운 Minimum Distance(MD) 부등식(9)을 도입, 부호의 최소거리 `d(C)`를 IP로 계산 후 cut 생성에 활용(CGA4).
8. 별도 학습/최적화 절차 없음. CPLEX(상용 IP solver)로 RIPD 및 최소거리 IP를 직접 풀이.
9. (64,32)/(160,80) irregular LDPC, (155,64) Tanner LDPC, (63,39) BCH에서 RSA가 NSA 대비 0.2~0.5dB 개선, BCH는 ML에 근접. (204,102) LDPC 최소거리=8, (576,288) WIMAX 최소거리=13 계산.
10. HW 미설계, 시뮬레이션(소형 부호, CPLEX 기반)만 수행되어 실용적 대규모 QC-LDPC/HW 디코더로의 확장성 불명.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| IP/LP 기반 cut generation 디코딩 | 대응 없음 | min-sum iterative decoder(`decoder.cpp`)와 근본적으로 다른 조합최적화 알고리즘 |
| 최소거리 IP 계산 | 대응 없음 | 코드 설계 단계 오프라인 분석 도구로 HW 디코더와 무관 |

적용 가치: 낮음 — IP/LP 기반 ML 디코딩은 Prime ECC의 min-sum HW 파이프라인 구조와 전혀 다른 패러다임이며 HW 구현이 사실상 불가능.
