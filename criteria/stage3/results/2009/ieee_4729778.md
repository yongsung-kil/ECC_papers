### B. 알고리즘 요약

1. 대상은 binary erasure channel(BEC) 상의 (3,6)-regular LDPC 코드(rate 1/2, length 32/200/1000)의 finite-length 앙상블 평균 성능 분석.
2. 풀려는 문제: 기존 finite-length(FL) 분석[4]-[6]은 Tanner graph에 반복 edge(같은 bit-check 쌍을 잇는 edge가 2개 이상)를 허용하는 앙상블을 다뤄, 실제 구현되는(반복 edge 없는) LDPC 코드보다 비관적인 성능 예측을 내림.
3. 핵심 가정: BEC에서 전송 심볼은 정확히 수신되거나 완전 소거(erasure)되며, message-passing 디코딩 실패는 정확히 stopping set(연결된 모든 check vertex가 그 set 내 다른 bit에도 연결된 bit 집합)의 존재로 결정됨.
4. 핵심 기법: 반복 edge를 배제한 새 앙상블 정의 하에서, subgraph를 한 bit vertex씩 재귀적으로 확장하며 degree-1/degree-2 check vertex 개수(`s`,`t`)를 추적해 stopping set 개수 `A(n,v,e)`를 조합론적으로 계산.
5. 핵심 식: 반복 edge 배제 조건에서 상한/하한 bound(`Ω_up`, `Ω_low`)를 유도해 `A_v,e`, `B_v,e` 재귀식(식 12–13)으로 word/bit erasure rate를 계산 — 반복 edge 자체를 카운팅에서 원천 배제하는 것이 핵심.
6. 확장/보정: 초기화 단계에서 이미 expurgate된(더 작은) stopping set을 포함하는 subgraph를 중복 계산하지 않도록 재귀적 초기화 함수 `Q(·)`를 도입(§II-A의 initialization modification).
7. 부수 트릭: 상한 계산은 기존 edge가 check vertex에 균등 분배되었다고 가정, 하한 계산은 가장 불균등한 분배를 가정해 두 bound를 유도.
8. 학습/최적화 절차 없음 — 순수 조합론적 recursion.
9. 결과: 제안 FL 알고리즘이 기존 FL 대비 계산량이 적음("half as many summations")이며, (3,6)-regular length 32/200/1000 코드에서 Monte Carlo 시뮬레이션과 정확히 일치하는 예측 성능을 보임(수치는 그래프로만 제시, 본문 텍스트에 구체 수치 없음).
10. 한계: BEC 전용 분석 도구이며 실제 부호설계·디코더 알고리즘·HW 구현이 전혀 없는 순수 이론(앙상블 평균 성능 예측) 논문.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BEC 기반 finite-length 앙상블 분석 | 대응 없음 | Prime ECC는 특정 고정 QC-LDPC H-matrix의 bit-exact 시뮬레이터로, 앙상블 평균 이론 분석 도구와 무관 |
| stopping set 기반 실패 예측 | 대응 없음 | AWGN/RBER 채널 기반 Prime ECC(NAND read)와 채널 모델 자체가 다름(BEC vs soft-read LLR) |
| message-passing 디코딩 | `decoder.cpp` `LDPC_Decoder()` | 개념적 상위 카테고리만 공유(반복 message-passing), 본 논문은 알고리즘이 아닌 성능 예측 이론이라 실질 대응 없음 |

> 적용 가치: 낮음 — BEC 전용 조합론적 성능 예측 이론으로 Prime ECC의 부호설계/디코더/HW 어느 레이어에도 이식 가능한 구체적 알고리즘·구조가 없음.
