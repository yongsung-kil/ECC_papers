> 총평: LDPC-like 코드 앙상블의 소가중치 codeword growth rate(error floor 원인)를 계산/판정하는 순수 이론적 해석 프레임워크로, 구체적 부호 설계나 디코더/HW 이식 요소가 없어 Prime ECC와 관련성 낮음.

### B. 알고리즘 요약 (10줄 내외)

1. LDPC/GLDPC/DGLDPC/turbo/RA 코드를 통칭하는 "LDPC-like codes"의 Tanner graph 프레임워크(variable/check node degree distribution `λ, λ̃, ς` 및 `ρ, ρ̃`)를 정의.
2. 풀려는 문제: error floor는 소가중치 codeword/작은 stopping set에서 기인하므로, 코드 길이 무한대(asymptotic) 극한에서 이 codeword의 등장 growth rate를 빠르고 정확하게 계산하는 방법이 필요.
3. codeword weight enumerating function(IOWEF) `P^v_j(W,X)`를 정의하고, Stirling 근사로 asymptotic weight enumerator(AWE) `A^v(δ,ω)`를 지수함수 극한으로 유도.
4. 핵심 기법: primal 최적화 문제(엔트로피 최대화, 식 (6)-(7))를 dual 변수 `μ, γ`(변수노드), `τ`(체크노드)를 이용한 unconstrained geometric programming(식 (10)-(11))으로 변환해 Newton's method로 효율적으로 계산.
5. 이 dual 변환은 기존 [12]의 근사법보다 정밀하고, 긴 코드의 체크노드를 "분해"할 필요가 없다는 것이 장점.
6. 소선형 가중치(δ, ω→0) 극한에서 `A(δ)`를 전개해 `t > q/(q-1)`이면 negative AWE(정정 가능), `t < q/(q-1)`이면 positive AWE(error floor 유발)라는 조건(Theorem 5.1)을 도출; `t=q=2`인 특수케이스는 `RL < 1/4` 조건(Theorem 5.2)으로 별도 판정.
7. N-layer 코드(GLDPC 일반화)로 확장: 각 층을 순차적으로 등가 check node side로 재귀 처리하여 negative AWE 조건을 재귀식(식 (40), (45))으로 일반화.
8. Nonbinary DGLDPC(GF(Q))로 확장: 모든 nonzero element가 등확률이라는 가정 하에 binary case와 동일한 형태의 dual 문제로 환원, 조건은 `RL < (Q-1)^2/4`.
9. Convolutional 컴포넌트 코드(turbo/RA 코드 포함)로 확장하여 parallel/serial turbo 코드의 word error probability 관련 IGE conjecture를 뒷받침하는 결과 도출.
10. 한계: 전부 이론적 유도이며 수치 실험/시뮬레이션/특정 코드 설계 예시가 전혀 없음. 순수 asymptotic(n→∞) 분석으로 유한길이 코드 성능과의 괴리 가능.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Asymptotic weight enumerator / degree distribution 최적화 이론 | 대응 없음 | Prime ECC는 특정 H-matrix(PCM_b) 고정 시뮬레이터로 앙상블 이론 분석 기능 없음 |
| Error floor 원인(소가중치 codeword) 분석 | 대응 없음 | 코드 설계 단계(`ecc_top.cpp` `Load_PCM()`) 이전의 이론 툴이라 코드베이스에 직접 대응 없음 |
| Nonbinary DGLDPC 확장 | 대응 없음 | Prime ECC는 binary QC-LDPC 전용 |
```

> 적용 가치: 낮음 — 순수 asymptotic 앙상블 이론 논문으로 특정 부호 구성이나 디코더/HW 기법을 제시하지 않아 Prime ECC 코드베이스에 이식할 구체적 요소가 없음.
