### B. 알고리즘 요약

1. 무선 채널(PBNJ, partial band noise jamming) 대상 QC-LDPC 부호를 diversity 기법과 결합한 forward error correction으로 사용.
2. 기존 구조적(structured) 부호는 HW 구현비용은 낮지만 대형 코드 길이에서 랜덤 부호 대비 성능이 열세이고, 기존 QC-LDPC 구성법[14]은 최소거리 제한으로 error floor가 발생하는 문제를 해결하고자 함.
3. column weight `j=2` 고정, row weight는 그룹 크기에 종속(`k = Σr_i`, 식(1))되도록 설계해 정규(regular) 부호를 유지.
4. Row Division Method: 행들을 그룹으로 균등 분할하고, 각 그룹이 `k`번 등장하도록 `2k`개의 row-group pair를 구성 — 개별 행/열 연결 대비 하드웨어 구현 비용·복잡도를 낮춘다고 주장(정량화 없음).
5. 두 번째 이후 그룹의 행을 내림차순으로 배치하여 최소 원하는 거리(distance)를 그룹 내에서 탐색하도록 함(girth 확보 전략).
6. array dispersion 기법으로 prime field 상 circulant permutation matrix 배열을 구성, 부분배열의 null space로 QC-LDPC 부호 도출 — 인코딩은 shift register로 parity bit 수에 선형 비례하는 복잡도.
7. 별도 학습/최적화 절차 없음 — 행 선택은 랜덤 또는 순차 탐색.
8. 결과: girth-12 QC-LDPC(N=155, N=11555)가 랜덤 부호 및 [14] 대비 우수, block length 2662 부호가 랜덤 부호 대비 BER 10⁻⁵에서 약 0.7dB 이득(45 iteration), column weight 2 vs 3 비교에서 fractional bandwidth ρ=1일 때 약 1dB 이득.
9. 디코더 최대 iteration 60회로 설정된 시뮬레이션만 수행.
10. 한계: 무선(PBNJ/anti-jamming) 채널 전용 평가, HW 설계·합성 없음, 부호 구성법이 특정 그룹 분할 파라미터에 의존해 rate/length가 예제별로 상이.
