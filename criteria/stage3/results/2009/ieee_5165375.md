### B. 알고리즘 요약 (10줄 내외)

1. 대상은 AWGN 채널, BPSK, sum-product 디코더(200 iteration 상한) 기반 regular LDPC 코드(예제: (96,48), (2000,1000)); 목표는 매우 낮은 BER(예: 10^-20) 영역의 error floor를 실용적 시간 내 추정.
2. 기존 문제: Monte Carlo(MC) 시뮬레이션은 low-BER 영역에서 필요한 시행 횟수가 기하급수적으로 증가해 사실상 불가능(예: BER 10^-20 추정에 8x10^17 루프 필요).
3. 핵심 가정: 매우 낮은 BER에서 디코더 실패는 특정 오류비트 조합인 "trapping set" `(x, y)` (x=오류비트 수, y=미충족 parity 수)에 의해 지배된다는 모델.
4. 핵심 기법 1단 - trapping set 탐색: 비트파그래프(Tanner graph)에서 각 비트가 속한 최단 사이클(cycle-6 등)을 나열한 표를 만들고, 각 행(비트)마다 사이클 비트들에 편향(bias=-1)을 주어 IS 시뮬레이션(`Ls`=1000 loop)을 돌려 실제 디코더 실패로 이어지는 trapping set을 검출.
5. trapping set은 짧은 사이클들의 중첩으로 형성되며, 탐색 비용은 `N*Ls` 회의 IS 시뮬레이션으로 제한되어 코드 길이에 크게 의존하지 않음(기존 [3]의 MC 기반 탐색 대비 대폭 절감).
6. 핵심 기법 2단 - 클래스화: 발견된 trapping set들을 크기(비트 수) 기준 "class"로 묶고, 클래스당 대표 1개만 IS로 정밀 시뮬레이션(동일 클래스 내 BER 근사 동일 가정), 클래스별 BER x 클래스 내 개수를 합산해 전체 BER 추정.
7. 부수 트릭: 편향 값은 각 parity check에서 총 편향이 2가 되도록 cycle-6 비트에 분배하여 설정(구현 세부).
8. 최적화/학습 절차 없음(그래프 탐색 + IS 통계기법, ML/GA 미사용).
9. 결과: (96,48) 코드에서 BER 10^-14까지 시뮬레이션 gain ~10^11; (2000,1000) 코드에서 BER 10^-20까지 gain 5.3x10^14 (MC 대비 필요 루프 수를 8x10^17 → 1.5x10^3로 축소).
10. 한계: HW 미설계, 순수 시뮬레이션/통계 기법(코드나 디코더 알고리즘 자체는 변경 없음); regular LDPC·cycle-4 없는 무작위 구성 코드에 한정해 검증, QC-LDPC/protograph 구조에서의 재현성은 논문에서 다루지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| trapping set 기반 error-floor 가속 시뮬레이션(IS) | 대응 없음 | Prime ECC는 bit-exact 결정론적 HW 시뮬레이터로 MC 채널 주입(`channel.cpp`) 방식이며, 편향(biasing) IS 프레임워크는 미보유 — 신규 검증 도구로 추가 이식 가능 |
| 채널 오류 주입(bias 적용 대상) | `channel.cpp` (AWGN/RBER/fixed-error 주입) | 기존 오류 주입 로직에 trapping-set 편향 옵션을 추가하는 형태로 이식 가능 |
| 비트그래프 사이클/trapping set 탐색 | `ecc_top.cpp` `Load_PCM()` (H-matrix 구조 접근) | H-matrix 구조에서 사이클 탐색 로직은 신규 구현 필요, 기존 로드 함수는 구조 접근 지점으로만 활용 |
| decoder 실패(non-convergence) 판정 | `decoder.cpp` `LDPC_Decoder()` 반복 루프의 수렴 실패 판정, `corner.cpp` (수렴실패 추적) | trapping set 여부 판정 시 기존 수렴실패 추적 로직 재사용 가능 |

> 적용 가치: 중간 — 디코더/부호 자체 개선은 아니지만, Prime ECC의 error-floor 실측 시뮬레이션(현재 MC 기반으로 추정)에 IS 가속 기법을 추가하면 매우 낮은 BER 검증 시간을 대폭 단축할 수 있어 검증 인프라 관점에서 실용적 가치 있음.
