### B. 알고리즘 요약

1. 대상은 protograph 기반 unterminated, periodically time-varying **LDPC convolutional code**(rate `R=1-nc/nv`) 앙상블, block LDPC 코드와 달리 sliding window 구조를 가짐.
2. 문제의식: iterative decoding의 error floor는 Tanner graph 상의 작은 trapping set이 지배하므로, 이를 미리 점근적으로 예측하고 싶음.
3. `(a,b)` general trapping set을 변수노드 `a`개, 이 중 홀수차수 체크노드 `b`개로 정의하고, 보조 flag 변수노드를 추가한 modified protograph에 두-부분 weight enumerator를 적용해 enumerate.
4. 핵심 변수 `Δ = b/a`에 대해 정규화된 두-부분 asymptotic spectral shape function `r(α,β)`(`α=a/n`, `β=b/n`)의 첫 zero-crossing을 `δts(Δ)`(Δ-trapping set growth rate)로 정의.
5. tail-biting LDPC convolutional code(unwrapping factor `Λ`)를 block code로 취급해 `δ_ts^(Λ)(Δ)`를 계산하고, `Λ→∞`일수록 원래 unterminated convolutional code의 성장률에 대한 하한이 개선됨(Theorem 1).
6. 하한식은 `δ_ccts(Δ) ≥ δ_ts^(Λ)(Δ)/v_s` (v_s = decoding constraint length)로 주어짐.
7. 부수 기법: `Δ=0`일 때는 기존에 알려진 최소거리 성장률 문제로 환원되어 Gallager의 (3,6)-regular 결과(`δmin=0.023`)와 일치함을 확인해 검증.
8. 별도 학습/최적화 절차 없음(닫힌형 조합론적 계산).
9. 결과: (3,6)-regular 예제에서 Λ≥3일 때 하한이 수렴하고, block code 대비 convolutional code의 trapping set 성장률이 유의미하게 큼을 확인; irregular RJA 예제(`δmin=0.013`)도 유사 경향.
10. 한계: 유한 길이/실제 부호 구현이나 BER 시뮬레이션 없이 앙상블 점근적(N→∞) 성질만 다룸, elementary trapping set 분석은 후속 연구로 남김.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LDPC convolutional code (protograph) 구조 | 대응 없음 | Prime ECC는 QC-LDPC block code만 다루며 convolutional/sliding-window 구조 없음 |
| trapping set 성장률 점근 분석 | 대응 없음 | 코드가 아닌 순수 이론적 앙상블 분석, 특정 H-matrix 설계 도구 아님 |
| error floor 예측 | `decoder.cpp` `LDPC_Decoder()` (참고용) | 개념적으로 error-floor와 관련되나 이 논문은 구체적 H-matrix나 디코더 변경을 제시하지 않음 |

적용 가치: `낮음` — convolutional 코드 구조와 순수 점근적 이론에 국한되어 QC-LDPC block 코드 기반 Prime ECC 3.1에 이식 가능한 구체적 기법(디코더/부호설계 모듈)이 없음.
