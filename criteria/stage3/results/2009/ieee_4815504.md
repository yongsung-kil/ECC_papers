### B. 알고리즘 요약

1. 시스템은 spectral-amplitude-coding(SAC) optical CDMA로, modified Welch–Costas(MWC) 코드 기반 인코더/디코더(AWG router, FBG, TFF)를 사용해 2.5 Gb/s 데이터 전송.
2. 데이터율이 높아질수록 PIIN(phase-induced intensity noise) 및 열잡음이 증가해 BER이 악화되는 문제를 LDPC로 보완하려 함.
3. 채널 모델은 PIIN과 thermal noise를 주 잡음원으로 하는 Gaussian 근사 기반 BER 해석.
4. 핵심 기법은 홀수 소수 `p`와 원소 `S(y)` 두 랜덤 시퀀스로 `p×p` 예비행렬 `S`를 만들고, 순환치환행렬(circulant permutation matrix) 블록 `q×q` 단위로 배열해 `H`를 구성하는 (γ,ρ)-regular QC-LDPC 구성법.
5. 식 (15)-(16)의 block sequence 조건으로 Tanner graph에 길이4 사이클이 존재하지 않음(girth≥6)을 증명, `p`가 GF(p)의 원시원소 성질을 이용.
6. 예시로 [248,127] (4,8)-regular QC-LDPC 코드(p=31, q=8) 구성, rank 121, code rate ≈0.5.
7. 디코딩은 log-domain sum-product algorithm(SPA), 최대 반복 50회 사용(표준 SPA, HW 변형 없음).
8. 별도 학습/최적화 절차 없음(닫힌 형식 구성법 + girth 정리 증명).
9. 결과: rate 0.94, [3992,1999] QC-LDPC 코드의 coding gain이 RS(255,239) 대비 약 4.5dB, uncoded 대비 약 8dB 높음(BER 1e-9 기준). [248,127] 코드는 RS 대비 약 2dB, uncoded 대비 약 6dB.
10. 한계: 광통신 OCDMA 채널(PIIN, thermal noise) 특화 시뮬레이션뿐이며 HW 미설계, NAND 채널 모델과 무관한 이론적 구성법 논문.
