### B. 알고리즘 요약

1. 대상은 AWGN 채널 iterative(message-passing) 복호를 가정한 irregular LDPC, 블록길이 `n=3200`, code rate 3/8·1/2·0.6 예제.
2. 문제: 중·단블록 LDPC의 degree 분포 최적화에 쓰는 density evolution(DE)은 시간이 오래 걸리고, 짧은 블록에선 점근 가정이 부정확.
3. 착안: PEG로 Tanner graph를 구성하면 check-node degree는 (거의) 집중분포로 자동 결정되므로 최적화 변수는 symbol(variable)-node degree 분포 `λ(x)` 하나로 축소.
4. 핵심 기법: 제약형 Downhill Simplex(비선형, cost function만 사용)로 `(l-1)`차원 확률벡터 `λ`를 탐색, 각 simplex 정점이 하나의 유효 degree 분포.
5. 비용함수: `λ`대로 PEG 그래프를 만들고 고정 `Eb/N0`에서 Monte-Carlo로 block-error rate를 직접 측정(오류블록 `$=50`개 검출 시 종료) → DE-threshold 대신 단일 시뮬 값 사용.
6. 근거: 고정 `Eb/N0`를 BER `10^-1~10^-3`가 나오도록 낮게 잡아 시뮬을 빠르게 종료, 다중 시뮬 없이 비용 산출 → 탐색 효율↑.
7. 결과: DE 대비 약 절반 시간에 near-optimal degree 분포 생성, rate 3/8·블록 3200에서 전 SNR 구간 0.2dB 이상, 저rate에서 BER `10^-4` 기준 ~0.3dB 우수.
8. rate 의존성: Fig.4에서 code rate가 낮을수록 DE 대비 성능격차가 커지고, rate 0.6에선 <0.1dB로 격차 소멸.
9. 결론: 중·단블록·저rate LDPC degree 분포 설계에서 DE보다 효율·성능 모두 우수.
10. 한계: HW 미설계·시뮬만, error-floor/절대 정정능력 분석 없음, 고rate QC 구조엔 이득 없음.

```
| PEG Tanner graph 구성 | 대응 없음 | Prime ECC는 QC circulant 구조로 PEG 기반 irregular 구성에 대응 모듈 없음 |

```
