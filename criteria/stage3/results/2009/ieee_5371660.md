### B. 알고리즘 요약

1. 대상: binary LDPC의 parity-check matrix `H`((n-k)×n) → Tanner graph의 short cycle(girth) 검출, 예제는 regular (8,4,2), rate 1/2.
2. 문제: Tanner graph의 cycle이 belief-propagation의 메시지 통계 독립성을 깨 복호 정확도를 떨어뜨림 → 구성 단계에서 cycle(특히 length>4)을 검출·제거 필요.
3. 핵심 발상: 통신망 노드 간 전체경로 산출용 logic algebraic 알고리즘[6]을 개량해 cycle 검출에 적용.
4. 상대행렬 구성: 각 edge에 라벨 `L1..Ls` 부여, `H'`(edge 라벨 행렬)로부터 식(1) `M=[[I,H'],[H'^T,I]]` (n×n 상대행렬) 구성.
5. 통합·삭제: 식(2) `m'_ij = m_ik·m_kj + m_ij` (logic and/or)로 source node에서 타 노드로의 모든 route 산출, source당 n-1회 반복.
6. cycle 추출: n-1회 후 얻은 `M(n-1)`에서 single-edge item `Lg`을 갖는 원소를 골라 식(3)(4) `C_ij = m(n-1)_ij·Lg − Lg`로 `Lg` 제거 → node i·j를 지나는 cycle 집합.
7. 전체 cycle: 모든 variable node를 순차 source로 반복하면 그래프의 모든 cycle 획득.
8. 계산량: 코드길이 n, 정보비트 k에 대해 통합·삭제 n-1회 + 수정 k회로 한 source의 cycle 확보(논문 주장).
9. 결과: (8,4,2) 예제에서 node 1의 cycle(C19, C110 등)을 손계산으로 열거, 타 방법과 동일 결과로 정확성만 검증.
10. 한계: 초소형 toy 예제뿐, HW·복호성능·BER 없음, 대형 행렬 확장성/복잡도 미검증, non-binary·QC 특화 없음.
