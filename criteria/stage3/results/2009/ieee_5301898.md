### B. 알고리즘 요약

1. AWGN 채널, HARQ 시스템에서 사용할 rate 1/2 다중엣지타입(MET) LDPC 모체부호(k=600, 6-edge-type)를 정의.
2. 기존 랜덤 펑처링은 짧은 블록길이·고rate에서 성능 손실이 커 GS(grouping-and-sorting) 기반 펑처링이 필요하다는 문제의식.
3. 모체부호는 degree-1/degree-2 parity bit 두 종류만 존재하고 state information bit는 첫 이터레이션부터 soft 정보를 가짐(threshold/error-floor에 유리).
4. 핵심 기법: 단순화 GS 알고리즘 - degree-1 parity bit는 무작위 펑처링(모두 1SR 노드), degree-2 parity bit는 언펑처링 비트 사이에 정해진 개수만큼 펑처링 비트를 삽입.
5. 핵심 식 `N_punc = N - k/R_t`(펑처링할 비트 수), `N_im = None/(N_punc-None) - (N_punc-None) - 1`(degree-2 구간 삽입 위치 계산용)로 목표 rate에 맞춰 펑처링 비트 수·위치를 결정.
6. MET-LDPC 구조적 특성(그룹 구분이 이미 되어 있음) 덕분에 기존 GS 알고리즘의 grouping 단계를 생략할 수 있어 펑처링 처리 복잡도 감소.
7. 부수 트릭: degree-2 parity bit는 RA(repeat-accumulate) 코드와 동일한 인코딩 구조라 forward/backward likelihood로 펑처링 비트의 우도 정보를 빠르게 갱신 가능.
8. 별도 학습/최적화 절차 없음(모체부호 자체는 기존 [8] 설계를 그대로 채용).
9. 결과: BER 10^-6 기준 RC 불규칙 LDPC 대비 rate 0.6/0.7/0.8에서 각 0.1/0.2/0.5dB 이득, E2RC LDPC 대비 rate 0.5~0.8에서 0.15~0.3dB 이득; HARQ 처리량은 SNR>0.5dB에서 우수.
10. 한계: AWGN·짧은 블록길이(k=600) 시뮬레이션에만 국한, HW 설계·복잡도 분석 없음, error-floor 관찰은 BER 10^-6까지만 확인.
