### B. 알고리즘 요약

1. Binary Erasure Channel(BEC) 위에서 peeling algorithm으로 복호되는 irregular LDPC 코드 앙상블(`n`, edge-perspective 차수분포 `λ(x)`, `ρ(x)`)을 대상으로 함.
2. 풀려는 문제: irregular LDPC는 variable node 차수에 따라 보호수준이 다른(UEP) 특성이 있는데, 이를 유한길이에서 해석적으로 정량화하는 기존 방법이 부족함.
3. 채널/디코더 모델: BEC(소거확률 `ε`)와 전통적 peeling 디코더(degree-1 check node를 찾아 순차 제거)만 가정.
4. 핵심 기법: Amraoui et al.의 유한길이 스케일링 기법을 확장, degree-1 check node 개수 과정 `X_t^(0)`을 파라볼릭 드리프트를 갖는 two-sided Brownian motion `w(θ)=w(0)+B(θ)+θ²/2`로 근사.
5. 핵심 식: 차수 `d_v` variable node의 bit erasure rate `P_E^(d_v) = Prob{decoding failure}·E{L_dv(T)|failure}/(d_v n Λ_dv)` — 첫통과시간(first passage time) `T`의 분포로부터 유도.
6. 확장: `g(y)=Prob{w(θ)>0, ∀θ<y}`를 Airy 함수 기반 폐형식으로 y≤0/y>0 두 경우로 유도, 실제 UEP 성능 근사에 사용.
7. 부수 트릭: 수치적분 복잡도 절감을 위해 `g(y)`의 상한/하한 bound를 추가로 유도(식 (34), (37)).
8. 별도 학습/최적화 절차 없음 - 순수 해석적 근사와 몬테카를로 시뮬레이션 비교.
9. 결과: `n=1000~8000` 길이의 degree-3/degree-9 variable node에 대해 스케일링 근사가 몬테카를로 시뮬레이션과 근접, 길이 증가 시 상하한도 시뮬레이션에 수렴.
10. 한계: BEC/peeling 알고리즘 전용 분석이며 HW 설계·구현은 전혀 없고, AWGN이나 NAND read-noise 채널·min-sum 디코더에는 직접 적용 불가.
