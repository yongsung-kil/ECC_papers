### B. 알고리즘 요약 (10줄 내외)

1. 대상은 유한 blocklength array-based structured LDPC 코드((2209,1978), (2209,2024) 등)로, quantized sum-product 반복복호(예: (4.2)/(4.4)/(4.5) fixed-point) 및 AWGN/Gaussian-mixture 채널을 가정.
2. 풀려는 문제: 중간 blocklength LDPC 코드의 error floor는 density evolution/EXIT 같은 asymptotic 기법으로 예측이 부정확하고, Monte Carlo 시뮬레이션은 10^-20 이하 저확률 영역에서 계산량이 폭발적으로 증가.
3. 핵심 가정: error floor는 `(a,b) absorbing set`(각 bit node가 unsatisfied보다 satisfied check를 더 많이 갖는, bit-flip에 안정적인 조합적 객체, 채널·디코더 무관 정의)에 의해 지배된다는 관점.
4. 핵심 기법 1단: 특정 디코더(양자화/포화/최대 반복수 고정)에 대한 `absorbing region R(A)` (그 흡수집합으로 수렴하는 LLR 입력공간)을 정의하고, bit node를 satisfied/unsatisfied check 개수 `(s:u)` 타입별로 그룹화해 저차원(2D 등) 근사.
5. 핵심 식: `P[E(L)] ≥ (1-o_σ(1)) card(A_{a,b}) Q_A[L∈R(A)]` (Theorem 1) — absorbing set 개수와 채널-특화 확률 `Q_A`(흡수집합에만 노이즈를 둔 보조분포)의 곱으로 error probability의 결정론적 하한을 구함.
6. 확장 기법: `Q_A[L∈R(A)]`를 근사영역(halfspace 교집합 등)의 내부/외부 근사로 상·하한 계산, 고SNR에서 이 하한이 실제 오류확률에 점근적으로 수렴함을 증명.
7. 부수 트릭: 채널 독립적 관점을 채택해 absorbing region은 한 번만 계산하고, 채널(AWGN/mixture Gaussian)별 `Q_A` 적분만 다시 계산하면 되므로 여러 채널에 재사용 가능.
8. 비교 기법: mean-shifted Gaussian importance sampling(흡수집합 중심으로 노이즈 평균 이동, 식 (15)-(16))으로 stochastic 추정치를 별도로 구해 결정론적 하한과 대조.
9. 결과: (8,6)/(4,8)/(6,4) absorbing set에 대해 결정론적 추정이 importance sampling·하드웨어 emulation FER 곡선과 근접 일치하며, 고SNR 영역에서 error probability를 10^-30까지 예측 확장 (하드웨어 emulation은 훨씬 좁은 범위만 가능).
10. 한계: 순수 시뮬레이션/이론 검증(HW 설계 없음), array-based LDPC라는 특정 구조 클래스 및 (a,b) absorbing set 개수(card)를 해석적으로 셀 수 있는 코드에 한정, 일반 QC-LDPC에는 card 계산이 추가 연구 필요.
