### B. 알고리즘 요약

1. 시스템: IEEE 802.11n 기반 QC-LDPC, codeword length 1944, subblock size Z=81, rate 1/2 예시(H는 n×m 서브블록 배열, n̄=24, m̄=12), BPSK/AWGN 채널.
2. 문제: LDPC 인코딩은 quadratic 복잡도가 기본이며, 기존 Richardson 알고리즘(H를 `[[A,B,T],[C,D,E]]`로 approximate lower triangulation, gap `g`)은 O(n+g²)로 g가 커지면 복잡도가 늘어남.
3. 가정: 파리티체크행렬 H의 행이 선형독립이며, 행/열 순열만으로 approximate lower triangular 형태(Fig.1)로 변환 가능(sparsity 유지).
4. 핵심 기법: H를 `[[M,T],[N,P]]`(M: t×(n−t), T: t×t 하삼각 대각1, N/P는 나머지)로 변환한 뒤, 파리티 `p^T = T^{-1} M s^T`를 두 단계(① `M s^T` sparse 곱 O(n), ② `T^{-1}[M s^T]`를 후진대입으로 O(n))로 계산해 전체 O(n) 달성.
5. 핵심 식: `M s^T + T p^T = 0` (식 5)에서 파리티는 오직 M, T 서브행렬만으로 결정되므로(행 rank가 t), 나머지 N, P 서브행렬은 파리티 계산에 기여하지 않아 후처리(postprocessing)로 삭제 가능 → 결과 H_p = `[M T]`만 남김.
6. 확장: 802.11n 표준 H-matrix(A,B,T,C,D,E 6분할)에 동일 아이디어 적용, `-ET^{-1}B+D=I`가 되는 구조적 성질을 이용해 파리티 벡터 `p1^T=(-ET^{-1}A+C)s^T`, `T p2^T = As^T+Bp1^T`로 계산.
7. 부수 트릭: N,P 삭제로 H의 열 수는 유지되나 행 수가 줄어 codeword length는 동일하되 유효 message length/rate가 변함(adaptive message length, AML) — 예: r=1/2 → 13/24로 증가.
8. 학습/최적화: 별도 학습 없음. 결정론적 행렬 재구성(순열 + 서브행렬 삭제) 알고리즘.
9. 결과: 802.11n H-matrix 기준 modulo-2 덧셈 수 비교에서 Richardson 10368회 vs 제안 AML 4617회(약 55% 감소); BER은 Eb/N0=1dB에서 Richardson(r=1/2) 0.02 vs AML(r=13/24) 0.015로 유사하거나 오히려 낮음(단, rate가 다름).
10. 한계: HW 구현/게이트카운트 분석 없음(순수 알고리즘 복잡도 비교), decoder iteration은 100으로 고정한 시뮬레이션뿐이며 code rate가 원래 표준과 달라지는 부작용 존재.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 파리티 벡터 계산 O(n) 선형 인코딩(ALT+후처리) | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | Prime ECC는 dual-diagonal 고정 구조라 ALT 기반 방식으로 교체 시 인코더 전면 재설계 필요, 이식 난이도 높음 |
| H-matrix를 근사 하삼각형으로 재배열(순열) | `ecc_top.cpp` `Load_PCM()` | H-matrix 로드/구조 변경 지점과 관련되나 부호 재검증 부담 큼 |
| 서브행렬(N,P) 삭제로 인한 rate 변화 | `encoder.cpp` `Column_Reordering_Dual_Update()` | column 재배치 개념은 유사하나 목적(복잡도 절감 vs dual-update)이 달라 직접 대응 아님 |
| BER 성능 비교(Richardson vs AML) | (대응 없음) | 순수 인코딩 복잡도/rate trade-off 분석, 디코더 정정능력과 무관 |

적용 가치: **낮음** — 인코딩 복잡도 절감 기법이나 Prime ECC는 이미 dual-diagonal 인코더로 고정되어 있고(§4 이식 난이도 "중~높음"), 본 기법 도입 시 rate 변화까지 수반되어 재검증 부담 대비 이득이 제한적.
