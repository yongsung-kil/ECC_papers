### B. 알고리즘 요약

1. QC-LDPC 코드 `H_QC`는 circulant permutation matrix 또는 all-zero matrix 블록으로 구성되며, 각 블록의 1-성분 위치는 첫 열 위치 하나로 완전히 특정 가능.
2. 기존 PEG는 girth만 최대화할 뿐 cycle 간 connectivity(=error-floor에 영향)는 고려하지 않아 최선이 아님; ACE(Approximate Cycle EMD) 지표로 stopping set과 cycle을 연결해 error-floor를 개선하려는 것이 목적.
3. 채널/모델: BPSK, AWGN, log-BP decoding, 50 iteration (교과서적 표준 채널, NAND 특화 없음).
4. 핵심 기법: Generalized ACE 제약 PEG의 constraint node 선택 기준을 QC 구조에 맞춰 변형(Transformed PEG) — 각 열 블록 `j`에서 첫 열(`(j-1)*b+1`)만 대상으로 constraint node를 선택하고, 선택된 위치를 기준으로 순환치환행렬을 "expansion procedure"로 확장.
5. 선택 기준 순서: 미도달/최장경로 노드 우선 → 최대 minimum path ACE metric → 최소 ACE 최단경로 수 → 최소 총 최단경로 수 → 최소 degree. ACE 값은 `Σ(d_s - 2)` (cycle 내 심볼노드 차수 합, 식 없음이나 본문 정의).
6. 확장 없음(단일 변형 PEG 알고리즘).
7. 부수 트릭: expansion node(같은 열에서 이미 확장에 쓰인 constraint node)는 후속 선택에서 제외 — 이로 인해 일부 성능 손실 발생함을 저자가 명시.
8. 최적화 절차: 별도 학습 없음, degree distribution은 기존 원본 PEG(DE 최적화) 코드와 유사하게 설정.
9. 결과: n=1008, rate 1/2, b=7·t=144·c=72 QC-LDPC에서 원본 PEG 대비 error-floor 영역(BER 10^-6~10^-8) 개선, 단 저SNR(waterfall) 영역은 소폭 성능 저하("a little depress").
10. 한계: 시뮬(BER/FER 곡선)뿐 HW/복잡도 미검증, rate 0.5·단일 (b,t,c) 조합만 실험, 고rate NAND ECC급 확장성 불명, waterfall-errorfloor 트레이드오프 존재.
