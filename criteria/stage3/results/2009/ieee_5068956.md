### B. 알고리즘 요약

1. 시스템은 3-노드 amplify-and-forward(AF) 협력 릴레이 채널(source-relay-destination), quasi-static Rayleigh-형 페이딩, TDMA half-duplex 2-slot 구조.
2. 기존 랜덤 LDPC는 girth를 사전 지정하기 어렵고 QC-LDPC는 girth가 자동으로 6에 머무는 문제를 해결하려 함.
3. Array code parity-check matrix `H`는 p×p identity `I`와 permutation `P`의 거듭제곱으로 구성되는 (r,p)-regular 구조 (p: odd prime, r: column weight).
4. 특정 block-row 집합 `{0,1,3}`을 고정하고, cycle-characterising 방정식(`v1(u1-u2)+...=0 mod p`)을 만족하는 block-column들을 제거(shortening)하여 girth 6/8/12를 선택적으로 확보.
5. 식 (12)는 Tanner graph에서 길이 2l 사이클이 존재할 조건(permutation 곱이 항등행렬)을 block-column 인덱스 v들의 선형결합으로 표현한 것으로, 이 조건을 피하는 column만 남기면 해당 길이의 cycle이 사라짐.
6. r=3, R=1/2 코드에서 p=601(길이 3606)로 girth 6/8/12에 대응하는 retaining block-column 집합을 그리디 탐색으로 산출.
7. 최소거리 dmin의 하한/상한을 girth·column weight로부터 유도(식 16, 17)하여 girth-12 코드의 dmin이 [14,24] 범위임을 제시.
8. 최적화 절차는 별도 학습 없이 그리디 서치로 조건을 만족하는 column 조합을 탐색.
9. 결과: girth-12가 girth-6/8 대비 BER 10^-2~10^-3에서 0.5~2.1dB 코딩이득, AF 협력 시 QC-LDPC 대비 girth-12 array code가 약 1dB 이득.
10. 한계: 순수 시뮬레이션(50 iteration sum-product)만 수행, HW 설계 없음, 채널 모델이 무선 페이딩 특화라 NAND 채널과 무관.
