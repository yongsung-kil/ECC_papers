### B. 알고리즘 요약

1. 채널모델 `y = r·x + z` (r: 임의 pdf의 fading gain, z~N(0,σz²)), 수신단에 순시 fading gain 정보(SI) 없음.
2. 문제: SI 없을 때 true LLR은 채널출력의 복잡한 비선형 함수 → 기존은 `α = 2E[r]/σz²` 선형근사(E[r] 사용)를 쓰나 BER 최적이 아님.
3. 핵심: 대칭 선형근사 `l̂ = α·y`에서 근사 LLR의 pdf `f_L̂`로 용량형 측도 `Ĉ = 1 - E[log2(1+e^-L̂)]` 정의, 이를 정확도 척도로 사용(MCLA).
4. 정리1: `Ĉ`는 `r̂`에 대해 concave → 유일 최댓값을 볼록최적화(steepest descent 등)로 효율적 계산, true LLR pdf 불필요.
5. 증명: 이산/연속 MBISO 채널을 M개 병렬 BSC로 보고 각 `Ĉ_i`가 true LLR에서 최대임을 보여, `Ĉ`가 true LLR에서만 최대가 됨을 증명(정확도 측도 정당화).
6. 잡음전력 σz도 미지인 경우: 각 σz의 `α_Ĉ(σz)`를 구하고, density evolution + binary search로 수렴범위를 최대화하는 `α`(=최대 threshold) 선택.
7. 부호설계: MCLA 하에서 threshold 최대화 / rate 최대화(고정 ρ(x)면 linear programming)로 irregular LDPC degree distribution 최적화(Table III).
8. 결과1: threshold가 E[r] 대비 개선 — (x²,x⁵)에서 4.06→3.81dB, (x³,x¹⁵)에서 7.69→6.93dB, opt와 거의 동일.
9. 결과2: BER 개선 약 0.3dB(x²,x⁵)~0.75dB(x³,x¹⁵) @1e-5, true LLR 대비 손실 <0.02dB, **추가 복호비용 없음**; σz 미지여도 성능 거의 동일.
10. 한계: HW 미설계, 무선 fading 채널·채널추정 시나리오 특화, SPA·DE 기반 이론/시뮬만.
