### B. 알고리즘 요약

1. BSC(binary symmetric channel) 위에서 column-weight-4 regular LDPC 부호를 대상, 메시지 알파벳 `M={-S,-W,W,S}` (strong/weak 0·1) 4-level(2-bit) 표현을 사용.
2. Gallager A/B(1-bit hard decision)는 error-floor에서 최적 이하이고, soft BP(다중 bit)는 복잡도가 높아 그 중간 지점의 저복잡도 디코더가 필요.
3. 채널은 BSC로 단순화, 수신값 `r_v∈{0,1}`을 `R_v∈{C,-C}`로 매핑, 코드/채널/디코더 대칭성으로 all-zero codeword 가정.
4. 디코더는 `(C,S,W)` triplet으로 파라미터화되는 voting 방식: 이웃 메시지의 부호별(0/1) 투표 총합 `t_j(v,c) = Σ w_{j-1}(u,v) + R_v`으로 값과 강도(strong/weak) 결정, check-to-variable은 XOR 부호 규칙과 강한 메시지(S) 존재 시 특수 처리.
5. `t_j(v,c)` 크기가 `S` 이상이면 strong, `0<|t_j|<S`이면 weak로 quantize — 4-level 양자화가 error-floor를 낮추는 핵심.
6. 확장(expansion) 조건 기반 정리: girth-6, `(C,S,W)=(2,2,1)` 디코더가 4→11, 5→12, 6→14, 8→16, 9→18 neighbor 확장조건을 만족하면 3-iteration 내 모든 weight-3 오류패턴을 정정함을 증명(Gallager B의 대응 조건보다 더 완화된 조건).
7. Density evolution으로 다양한 `(C,S,W)` 조합의 BSC threshold를 계산, iteration마다 `C`를 최적화하는 dynamic two-bit decoder도 도출.
8. 최적화 절차는 density evolution 기반 threshold 계산(수치 최적화), 코드 구성은 PEG(progressive edge growth) 변형으로 가능하다고 언급(직접 구성 실험은 없음).
9. 결과: rate 0.875, p=32 기준 dynamic 디코더 threshold 0.00953 vs Gallager A 0.00585, Algorithm E 0.00935 — 1-bit 대비, 심지어 erasure 포함 Algorithm E보다도 우수.
10. 한계: 시뮬레이션/HW 구현 없이 순수 threshold·조합론 증명, column-weight-4·BSC로 한정, 실제 코드 구성·유한장 성능은 미검증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 2-bit 메시지 message-passing 알고리즘 | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | soft-2~3bit 표현과 유사한 low-resolution quantization 아이디어, min-sum 대체 후보로 검토 가능하나 column-weight-4·BSC 전용이라 이식엔 재검증 필요 |
| VN 투표(voting) 기반 양자화 규칙 | `decoder.cpp` `VN_Cal_HD()` 등 (HD/2SD/3SD) | 기존 HD/2SD 사이 대안 양자화 레벨 설계에 참고 가능 |
| error-floor 3-error 보정 확장조건(Tanner graph) | 대응 없음 | 부호 설계(H-matrix 구성) 단계의 조합론적 조건으로 `ecc_top.cpp` `Load_PCM()`과 직접 연결되지 않음 |
| dynamic threshold 최적화(반복별 C 값 변경) | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | iteration별 LLR/threshold 테이블 갱신 아이디어와 유사한 개념 |

> 적용 가치: 중간 — 2-bit quantized message-passing이 NAND soft-read(2~3bit) 상황과 개념적으로 맞닿아 있으나, column-weight-4 regular code·BSC 채널 한정 이론 논문이라 실질 이식은 재설계 부담이 큼.
