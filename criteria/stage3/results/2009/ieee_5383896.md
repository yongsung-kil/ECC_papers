### B. 알고리즘 요약

1. 대상은 IEEE 802.11n/802.16e 표준의 QC-LDPC 디코더로, expansion factor `z`(24~96)와 code length `N`(576~2304), rate 0.5 등 다중 파라미터를 지원해야 하는 상황.
2. 문제: QC-LDPC 디코더는 LLR 메모리에서 처리요소(PE)로 데이터를 옮길 때 서브행렬 크기(z)만큼 순환 시프트하는 permutation network(PN)가 필요한데, 기존 Benes Network(BN) 기반 PN은 한 번에 한 그룹만 시프트 가능해 코드 길이가 짧을수록(z가 작을수록) throughput/PE 사용률이 크게 저하됨.
3. 배경 가정: QC-LDPC PCM은 identity 행렬을 순환 시프트한 서브행렬(크기 z×z)들의 조합인 base matrix로 표현됨(예: N=12, M=6, 서브행렬 4×4).
4. 핵심 기법: BN의 재귀적 구성 및 대칭성(symmetric property)을 활용해 데이터 그룹 크기가 입력 크기의 1/2 또는 1/4 이하일 때 최대 4개 그룹을 동시에 순환 시프트하는 새 control algorithm(`ALGORITHM I`, 함수 `CS(z, s, N, m)`) 제안.
5. 핵심 식/절차: 각 스테이지의 crossbar switch를 z(홀짝), s(시프트량 홀짝), m(그룹수 1/2/4)에 따라 BAR/CROSS로 설정하고 재귀적으로 `CS([z/2],[s/2],N/2,m)`를 호출하는 방식 — 수학적 증명 대신 802.11n/16e에서 가능한 모든 (z,s,128,m) 조합을 C 프로그램으로 전수 검증.
6. HW 아키텍처: N=128 크기의 main PN을 사용해 LLR 메시지(8bit word)를 메모리에서 PE로 라우팅, code length에 따라 1/2/4개 독립 코드 블록을 동시 디코딩하는 3가지 모드로 동작.
7. 부수 구현: LLR 메모리는 1024bit(8bit×128) wide, 24 deep으로 구성해 다양한 z에 대응.
8. 별도 학습/최적화 절차 없음(알고리즘 설계 + 전수 검증뿐).
9. 결과: 기존 PN 대비 최대 4배 병렬성 확보, 면적 0.676mm²@180nm, 최대 주파수 177MHz(타 연구 대비 면적/주파수 균형 우수), 짧은 코드 길이(768/1536bit)에서 기존 대비 최대 1.33배 unified throughput.
10. 한계: 특정 두 표준(802.11n/16e)의 고정 파라미터 세트에 최적화되어 있고, 정확한 절대 throughput(Gbps) 수치는 미제시(unit 상대값만 제공), decoder 알고리즘 자체(min-sum 등)는 다루지 않고 순수 라우팅/HW 구조만 다룸.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BN 기반 재구성 가능 permutation network (LLR 메모리→PE 라우팅) | 디코더 HW 모델 (z=32, HCU, GT) | Prime ECC는 z_sb=32 고정 단일 병렬 구조로 다중 code length 지원 목적이 없어 직접 적용 대상은 아니나, 메모리-PE 라우팅 병목 관점에서 참고 가능 |
| Graph Thinning(GT)의 MUX 기반 edge 선택 | GT.cpp Make_GT_HW() | 본 논문의 crossbar switch 기반 데이터 셔플과 개념적으로 유사한 영역(라우팅 최적화)이나 논문 내 직접 대응 언급 없음 |
| LLR 메시지 8bit 폭 처리 | ecc_data.h PARAM_LLR | soft-4bit+ 수준 양자화로 Prime ECC의 HD/2SD/3SD보다 넓은 폭이나 논문에서 양자화 최적화 자체는 다루지 않음 |
```

적용 가치: 낮음 — 다중 code length(576~2304)/다중 rate 지원용 HW 라우팅 구조 논문으로, Prime ECC는 단일 고정 z=32 구조라 재구성성(reconfigurability) 자체가 이식 목적과 맞지 않음. 다만 향후 Prime ECC가 가변 code length를 지원할 계획이 있다면 메모리-PE 라우팅 설계 참고 가치는 있음.
