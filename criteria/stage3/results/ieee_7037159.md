### B. 알고리즘 요약

1. 대상은 sub-20nm(1x-nm) MLC NAND flash의 threshold voltage(VTH) 분포 — E/A/B/C 4상태, LSB(lower)/MSB(upper) 2단계 프로그래밍 구조.
2. 문제: 고P/E-cycle에서 VTH 분포가 비대칭·지수꼬리·multi-modal로 변해 기존 Gaussian(혼합) 모델이 실측 BER을 과소추정 — 고급 soft-decision ECC 설계에 부정확.
3. 히스토그램 추출: read voltage를 `∆`만큼 shift하며 read sweep → 각 셀을 `N_bin=6n+4`개 bin으로 라벨링, 상태별 조건부 확률 `P_k(X)` 추정(식 3,4).
4. 핵심 모델: Normal-Laplace 혼합 — `V_ideal(A)=µ_A+σ_A W+α_A^{-1}Z1−β^{-1}Z2` 형태로 Gaussian(`W`)+좌우 지수꼬리(`Z1,Z2`) 결합, E/C는 shifted-exponential(식 9~12).
5. 지수꼬리 파라미터 `α`(우측), `β`(좌측)가 P/E cycle 마모로 인한 비대칭 exponential 꼬리를 표현 — Gaussian이 못 잡던 tail을 포착.
6. programming error 확장: `Φ_X(z)=Σ_Y λ_{X,Y} φ_Y(z+δ_{X,Y})` — 셀이 의도 상태 X 대신 Y로 프로그램될 확률 `λ_{X,Y}`로 multi-modal bump 표현(식 13~15), 고전하 방향만 허용해 반복적 fitting.
7. Fitting: 실측 bin 확률과 모델 bin 확률 간 Kullback-Leibler divergence 최소화(식 8,16), Nelder-Mead simplex로 최적 파라미터 추정.
8. 파라미터화: `λ`는 정규화 P/E cycle `x`에 지수 증가, `µ`는 power-law, `σ`는 상수, `1/α·1/β`는 선형 — Table I로 compact 모델 제공(외삽 가능).
9. 결과: 10L P/E cycle까지 실측 분포·BER과 거의 정확히 일치(Gaussian은 저BER 영역에서도 과소추정), programming error 제거 가정 시 BER=0.01 endurance가 9L→15L(1.7배) 향상 예측.
10. 한계: 채널 VTH 분포 모델링일 뿐 ECC/디코더 설계 없음, cell-to-cell interference·retention·read-disturb는 미반영(future work), HW 없음.
