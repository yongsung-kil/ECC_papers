> 총평: 고rate QC-LDPC layered normalized min-sum에 정렬 스케줄링+payload CRC 조기종료를 얹어 iteration을 줄이는 실용 기법이나, 조기종료는 Prime ECC가 이미 보유(Partial/Full CRC)해 신규성은 정렬 스케줄링 정도.

### B. 알고리즘 요약

1. 대상 시스템은 위성방송 표준 ABS-S로, 32×32 순환천이 서브행렬로 구성된 QC-LDPC, 고정길이 `Nldpc=15360`, 8개 rate(1/2~9/10)를 단일 디코더로 지원.
2. 문제: layered 복호에서 반복(iteration) 수와 그로 인한 전력을 줄이면서 8 rate 호환 저비용 VLSI를 만드는 것.
3. 복호는 layered(수평) 스케줄링 + 정규화 min-sum: 각 layer에서 갱신된 bit LLR을 다음 layer CN 평가에 즉시 사용.
4. CN 갱신식은 `Lr = (∏ sign(Lq))·min(|Lq|)·k`로, `k`는 min-sum 과추정 보정 scaling factor.
5. bit node 갱신은 `Lq_i,m+1 = Lq_i,m + (Lr(n) - Lr(n-1))`로 layer 간 extrinsic 차분만 누적.
6. 핵심 트릭1 "sorted scheduling": 1st iteration 후 각 layer의 실패 check 방정식 수를 세어, 실패가 적은 layer부터(불확실 bit가 최신 정보를 받도록) 처리 → 평균 iteration 0.5~1 감소.
7. 핵심 트릭2 "payload CRC 조기종료": irregular 부호라 마지막 반복에서 오류가 parity부에 몰림 → 실패 check 수가 임계 T(=50) 미만이면 payload 32bit CRC를 활성화, payload가 통과하면 parity 실패가 남아도 종료.
8. HW: Altera Stratix III FPGA, 135MHz, 32개 SISO CNU 병렬, LLR 메모리 뱅크와 extrinsic 메모리 뱅크 2개, 스케줄/종료는 main controller가 담당.
9. 결과: sorted scheduling+CRC 조기종료로 총 iteration 8~10% 감소(예: QPSK 3/4 4.2dB에서 10271→9267), FPGA ALUT 38%/메모리 22% 사용.
10. 한계: gatecount/throughput/BER 곡선 수치 없음(FPGA 자원표만), soft quant bit 폭 미기재, 정정능력(waterfall/error-floor) 개선은 없음(iteration/전력 절감 목적).
