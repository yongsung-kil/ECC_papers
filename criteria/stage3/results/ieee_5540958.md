### B. 알고리즘 요약

1. 대상: WiMAX(IEEE 802.16e) irregular QC-LDPC, base matrix `Hb`(M×N sub-matrix, expansion `b`=96), 예시 (2304,1152) rate 0.5; 114 modes(19 length×6 rate) 호환.
2. 문제: 기존 partially-parallel layered 디코더는 prior message용 별도 FIFO와 sum RAM 접근 충돌로 면적·throughput 손실.
3. 알고리즘: 개선 TDMP(layered) — 각 block row를 constituent code로 보고 sub-iteration 간 메시지 직접 전달, TPMP 대비 수렴 iteration 50% 절감.
4. SISO 엔진은 normalized min-sum 식 `(1)` 사용(정규화 계수 `α`), CNU/VNU 두 phase로 분해, 메시지 성분 `C(m,i)`=1st/2nd min·pos·sign·flag만 저장.
5. 핵심 기법: sum memory를 odd/even 두 bank로 분할해 variable node sum과 prior message를 함께 저장 → 13,440bit FIFO 제거, 약 20K gate 절감.
6. 확장: base matrix 전처리 3단계(block column permutation, block row permutation, nonzero sub-matrix reordering)로 매 클록 서로 다른 bank(odd/even) 접근 보장, 접근 충돌 제거.
7. 구현 디테일: symmetrical 4-stage pipeline + read-write bypass로 CNU end-stage와 VNU start-stage 오버랩, two-port sum RAM 활용, 7-bit 양자화.
8. 96-way configurable permutation network(2단 log shifter)로 114 mode의 cyclic shift 처리, 약 14.3% HW 절감.
9. 결과: SMIC 0.13um, 410k gate, iteration당 78~90 클록, 150MHz/15iter에서 248~287Mb/s, 282mW(66pJ/bit/iter), core 1.57×1.57mm²; 7-bit 양자화 시 float 대비 약 0.4dB 손실.
10. 한계: WiMAX 특화 multi-mode 유연성 중심, 정정능력 자체 개선은 없음, soft 7-bit 입력 가정.
