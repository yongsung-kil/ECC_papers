> 총평: NAND retention noise 복구를 위한 채널추정(결정트리)+IB 기반 정수 LUT 복호 결합 기법으로 NAND 직접 타깃이나, 복호 알고리즘 자체가 min-sum과 이질적인 IB-LUT 방식이라 Prime ECC decoder 이식은 제한적, 채널/참조전압 갱신 아이디어의 활용 가치가 더 큼.

### B. 알고리즘 요약

1. MLC NAND 2bit/cell, retention noise(전하 누설)로 인한 threshold voltage shift 문제를 다룸, PEG-constructed regular LDPC (8000,7200), `dc=51`, `dv=5`, 최대 50 iteration.
2. 기존 채널추정(DDE/DPE/TSE 등)은 반복최적화나 추가 센싱이 필요해 복잡도가 높고, 부동소수점 sum-product는 저사양 NAND 컨트롤러에 부적합하다는 문제의식.
3. threshold voltage 모델을 ISPP noise/program noise/RTN/retention noise 합으로 정의(`V = Vs+nu+np+nw-nr+ni`), retention noise는 `PE cycle`, `retention time T`에 종속된 가우시안 모델.
4. 1단계: CART 결정트리로 quantization interval별 cell count `c`를 입력받아 참조전압 인덱스 `v`를 직접 분류(회귀 없이 분류 문제로 단순화), 채널파라미터 추정 단계를 생략.
5. 2단계: information bottleneck(IB) 기반 LDPC 복호, `min I(Y;Z) - βI(X;Z)` 압축 원리로 check/variable node 연산을 오프라인 생성된 LUT 참조로 대체, 온라인 복호는 정수(unsigned integer)만 처리.
6. LUT는 BPSK-AWGN 채널 기준(Seq-IB)으로 설계되며, `dc-1` 입력 트리구조(`log2(dc-1)+1`개 서브테이블)로 메모리 절감, NAND 채널과의 불일치는 closest-LLR 매칭 기반 channel re-mapping으로 보정.
7. dynamic quantization level reduction: iteration 진행에 따라 LUT 양자화 레벨을 16→14→12→10으로 단계적으로 줄여 메모리 약 25% 절감, mutual information이 목표 iteration에서 1로 수렴하도록 SNR 설계.
8. 학습: K-means로 참조전압 벡터를 Ng=50개 클러스터로 그룹화 후 CART 알고리즘(Gini 최소화 분할)으로 143-node 이진트리 오프라인 학습.
9. 결과: 참조전압 추정오차 0.02 이하, FER 성능이 TSE-SP(부동소수점 SPA) 벤치마크와 유사, LDPC-IB 전체 메모리 57170 Bytes, 결정트리 773 Bytes, 트리탐색 복잡도 O(log n)~O(n).
10. 한계: HW(RTL) 설계 없이 순수 시뮬레이션뿐, IB LUT가 min-sum이 아닌 별도 정수 message-passing 구조라 기존 파이프라인과의 통합 난이도, gate count/throughput 등 HW 지표 부재.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| IB 기반 LUT 정수 message passing (min-sum 대체) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | min-sum과 다른 복호 알고리즘이나 반복 메시지 패싱 루프 위치는 대응, 대체 시 재검증 부담 큼 |
| iteration별 동적 양자화 레벨 축소 | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | iteration별 LLR 테이블 개념과 유사하나 목적(메모리 절감 vs 성능 튜닝)이 다름 |
| 참조전압(reference voltage) 결정트리 기반 채널 업데이트 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | NAND soft-read threshold 설계와 직접 연관, 이식 가치 있음 |
| 채널 재매핑(NAND ↔ BPSK-AWGN LLR 매칭) | `channel.cpp` | soft-read LLR 양자화 테이블 설계에 참고 가능 |

> 적용 가치: 중간 — NAND 직접 타깃이며 참조전압/채널 업데이트 아이디어는 이식 가치가 있으나, 핵심 복호 알고리즘(IB-LUT)은 Prime ECC의 min-sum 구조와 이질적이라 통째 이식은 어려움.
