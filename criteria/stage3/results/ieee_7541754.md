### B. 알고리즘 요약

1. 대상: 변수정규(dv-regular, dv≥3, g>4) 이진 LDPC의 Tanner 그래프에서 (a,b) 트래핑셋(TS) 중 check degree가 1/2뿐인 elementary TS(ETS), 특히 normal graph가 leafless인 LETS.
2. 문제: 기존 LSS(layered super set) 특성화는 (1) 모든 ETS가 cycle의 LSS는 아니어서 비완전, (2) girth보다 훨씬 긴 cycle 열거가 필요해 multiplicity가 수천만 규모로 폭발.
3. 핵심 아이디어: 모든 LETS를 `simple cycle`에서 출발해 세 가지 확장으로 생성 — depth-one tree(`dot_m`), path(`pa_m` open/closed), lollipop(`lo_m` = cycle c + path d).
4. 각 확장은 변수정규 제약(추가 edge는 degree<dv인 노드에만, root degree 2~dv)을 만족해야 유효한 LETS normal graph가 됨.
5. Theorem 1: 임의 (a,b) LETS(단순 cycle 제외)는 dot/path/lollipop 조합으로 simple cycle에서 생성 가능 → 특성화가 exhaustive.
6. Theorem 2: 중간 sub-structure가 모두 LETS인 minimal 특성화는 반드시 dot/path/lollipop 세 확장만으로 구성됨.
7. Theorem 3: 주어진 dv,g,amax,bmax에서 dpl은 필요한 simple cycle 최대길이 K와 중간 b값 범위(bmax')를 최소화 → optimal minimal 특성화.
8. 탐색: 특성화 표(Table I: dv=3,g=8,a≤12,b≤4)에 명시된 prime simple cycle(s4,s5,s6)만 열거 후 확장 적용 → 관심 범위 내 구조만 탐색.
9. 결과: Tanner(155,64) C1, PEG (3,6)-정규 n=504/1008 C2/C3에서 a≤12,b≤4 LETS 완전 열거를 442/561/157초에 수행. LSS-기반은 하루 내 완료 실패(a≤10에서도 dpl 28초 vs LSS 392초).
10. 한계: HW 미설계, 오프라인 그래프 분석 전용, 변수정규 코드 한정(비정규·QC 구조 특화 최적화 없음), 디코더/BER 성능은 직접 다루지 않음.
