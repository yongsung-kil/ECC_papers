### B. 알고리즘 요약

1. 대상은 SPC check를 (n,k) block code(주로 (7,4)/(15,11) Hamming)로 대체한 GLDPC와 이를 시공간 결합한 SC-GLDPC 앙상블(protograph+lifting factor M 기반).
2. 풀려는 문제: irregular LDPC(waterfall 좋음/error-floor 나쁨)와 regular LDPC의 trade-off를, 강한 GC constraint + spatial coupling으로 동시에 개선.
3. 채널은 BP threshold/유한길이 분석에 BEC(erasure prob `ε`), 메시지는 LLR, GC node는 component code의 APP(BCJR) 또는 ML 디코더 사용.
4. threshold 분석: GLDPC density evolution `q(ℓ)=f_b^{Ci}(p(ℓ-1))`로 GC node transfer function 확장, terminated SC-GLDPC의 `ε_BP`가 L→∞에서 underlying block code의 `ε_MAP`에 수렴(threshold saturation).
5. 핵심식: BP EXIT `h_BP(ε)`의 면적정리로 `ε_MAP` 상한 산출(예: (2,7)-regular Hamming GLDPC에서 `ε_MAP=0.856`, `ε_BP=0.756`).
6. 거리 분석: terminated/tail-biting 앙상블의 spectral shape `r(δ)`로 최소거리 growth rate 계산, unterminated 앙상블의 free distance growth rate를 상·하한으로 bound(예: A7에서 `δ_free=0.805` ≫ block `δ_min=0.186`).
7. 유한길이: generalized peeling decoding(GPD)을 도입, type vector 기반 residual graph 진화를 Wormald 미분방정식으로 기술.
8. scaling law: critical point `τ*`에서의 decodable node 통계로 block error `P_Block≈Q(α√n(ε*-ε))`(block), coupled는 covariance decay θ 포함한 (37) 식으로 예측.
9. 결과: SC-GLDPC은 threshold saturation으로 near-capacity, asymptotically-good 대거리, 유한길이 scaling도 GLDPC block보다 우수(수치는 BEC 시뮬로 검증, M=500~4000).
10. 한계: 순수 이론(BEC/점근/유한길이 scaling)뿐, AWGN·NAND·HW 미대상, GC component 디코더 복잡도 증가.
