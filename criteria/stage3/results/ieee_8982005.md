### B. 알고리즘 요약 (10줄 내외)

1. 시스템은 AWGN + BPSK 채널에서 (N,K) Polar code, 수신단이 one-bit quantizer(one-bit ADC)만 사용하는 zero-delay 통신 모델(5G eMBB 제어채널 맥락).
2. 문제: one-bit 양자화가 강한 비선형성을 유발해 표준 polar BP 디코더 성능이 크게 저하됨.
3. 채널 모델: BIAWGN 출력을 1비트로 양자화하면 등가적으로 binary symmetric channel(BSC)로 근사(`P_ij` 전이확률로 정의).
4. 핵심 기법: unfolded polar BP factor graph의 각 엣지에 학습 가능한 가중치 `w`를 곱하는 NND(Neural Network Decoder). 표준 BP는 전체 가중치가 1인 특수 경우.
5. 핵심 식 `g(x,y)=sgn(x)sgn(y)min(|x|,|y|)`(min-sum 근사)에 가중치를 곱한 `f(af,bf,b̄f,wf)=wf·g(bf,af+b̄f)` 형태의 S2C/C2S 뉴런으로 메시지 업데이트.
6. 확장: layer-based weighting scheme으로 동일 iteration·stage의 뉴런들이 가중치를 공유하게 하여 학습 파라미터 수를 대폭 축소(뉴런별 개별 가중치 대비).
7. 부수 트릭: NND가 대칭구조(symmetry)임을 S2C/C2S 뉴런 단위로 증명 → 전-영(all-zero) 코드워드만으로 학습 가능(지수적 학습 복잡도 회피).
8. 학습: cross-entropy loss, mini-batch SGD/역전파, TensorFlow 구현. all-zero codeword로 학습, 랜덤 코드워드로 테스트.
9. 결과: (64,32) Polar에서 5회 iteration 기준 BP-5-1bit 대비 1.2dB, BP-5-2bit 대비 0.2dB 이득; (256,128)에서 BP-5-1bit 대비 2.0dB 이득; 표준 BP보다 빠른 수렴.
10. 한계: Polar code·AWGN·무선 one-bit ADC 시나리오에 특화, 실제 HW 설계·합성 없음, LDPC/NAND 채널에 대한 검증 전무.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Polar BP 알고리즘 전체 구조 | 대응 없음 | Polar code 전용 factor graph이며 Prime ECC는 QC-LDPC min-sum 구조로 부호 자체가 다름 |
| min-sum 근사 g(x,y)=sgn·min(|x|,|y|) | `decoder.cpp` `CNU_Update_New_Mag()` | Prime ECC도 동일한 min-sum 근사를 사용하나, 본 논문은 이를 Polar factor graph에 적용한 것이라 직접 이식 불가, 근사식 자체는 이미 보유 |
| 가중치 학습 기반 메시지 스케일링(layer-based weighting) | `decoder.cpp` LDPC_Decoder() 이터레이션 루프 (LLR 스케일 지점) | NN 학습형 weighted min-sum 개념은 LDPC decoder의 normalized/offset min-sum 대체 아이디어로 참고 가능하나 본 논문은 Polar 대상 검증뿐 |
| one-bit ADC/quantizer 채널 모델 | `channel.cpp` `Set_LLR_Th()` | Prime ECC도 hard-1bit 지원하나 채널이 무선 AWGN이라 NAND RBER 채널과 상이, 직접 재사용 불가 |
```

적용 가치: `낮음` — 대상 부호(Polar), 채널(AWGN 무선, one-bit ADC), 검증 방식(순수 시뮬, HW 미설계)이 모두 Prime ECC 대상과 어긋나며, NN 학습형 weighted BP 아이디어는 개념적 시사점 이상의 즉시 이식 요소가 없음.
