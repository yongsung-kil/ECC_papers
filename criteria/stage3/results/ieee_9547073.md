### B. 알고리즘 요약

1. 시스템: 5G-NR LDPC(BG2), BPSK/AWGN, rate 1/5·1/4·1/3·1/2·2/3·5/6 대상 MET(protograph) LDPC 복호 알고리즘.
2. 문제: MSA는 CN 연산의 `tanh`를 `min`으로 치환해 출력이 실제 LLR이 아니게 되어 belief propagation이 왜곡, MET-LDPC에서 성능손실이 더 큼. 기존 NMSA/ANMSA factor는 Monte Carlo 학습 기반이라 오차·샘플부족 문제.
3. 모델: MET-DE로 각 edge의 iteration별 출력 PDF `PDF_{i,j}`를 추적, AWGN a-priori LLR PDF는 식(1).
4. 기법1(오차정량화): MSA row 연산 출력의 PDF와 "실제 LLR" PDF 사이 편차를 Kullback-Leibler divergence `D_KL(P||Q)`(식9)로 정량화.
5. 핵심식: normalization factor는 `α = argmin_α D_KL(P(α·x)||P(x_j))`(식11) — 정규화 출력 PDF가 실제 LLR PDF에 가장 근접하도록 하는 factor.
6. 기법2(factor 선택): failed parity-check 비율인 check ratio `ρ`를 index로 사용, `L_mn = α(ρ,d_c)·L_msn`(식10)로 row 출력 보정. iteration별 `ρ`는 출력 PDF+PCM으로 계산.
7. 기법3(edge classification): edge를 check-node degree 등 파라미터로 clustering해 factor LUT 공유 → type 수와 정확도의 trade-off를 완화. 기존 ANMSA/Enhanced ANMSA는 이 방식의 특수 케이스.
8. 최적화: Monte Carlo 학습 대신 MET-DE로 각 edge factor 직접 산출 → 샘플부족(margin of error) 회피, fitting은 smooth spline.
9. 결과: BG2에서 CN degree별 6 type으로 분류(Fig.1/2), BLER=1e-3에서 제안(DE-MSA1/2) 성능이 SPA/Adj-MSA에 근접, SPA와의 gap 최대 0.3344dB(rate-1/4)·최소 0.0085dB(rate-5/6).
10. 한계: 순수 알고리즘·시뮬(BG2/AWGN), HW·throughput·복잡도 수치 없음, spline fitting numerical error 잔존.
