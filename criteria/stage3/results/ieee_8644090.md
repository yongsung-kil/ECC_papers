### B. 알고리즘 요약

1. 시스템: uplink NOMA, V=6 user가 F=4 resource element에 비직교 다중화, 각 user는 BCH(63,45)로 채널부호화 후 SCMA codebook 매핑, AWGN/fading 채널.
2. 문제: NOMA 수신단은 다중사용자 검출(MPA)+채널복호(BP)를 결합해야 하며, 검출만으로는 부호 이득을 못 얻어 두 단을 함께 최적화할 필요.
3. 모델: NOMA 구조를 factor graph(VN=user, FN=RE)로, 선형부호를 Tanner graph로 표현 — MPA와 LDPC BP가 본질적으로 동일한 message passing임을 활용.
4. 핵심 기법: factor/Tanner graph의 edge에 학습 가능한 가중치 `w`를 부여한 weighted MPA/BP를 다층 sparse NN(2L hidden layer=L iteration)으로 전개.
5. 핵심 식: 검출단 `Θf(x)=-(1/N0)|y_f - Σ h_{v,f}x_{v,f}|^2`, 복호단 `I_{i,(v,c)}=tanh((1/2)(w·l_v + Σ w·I_{i-1}))`, 출력 `Λ(b)=max{Q(x^1)}-max{Q(x^0)}` — `max*` 대신 `max`로 근사(지수연산 회피).
6. 확장: 검출 NN 출력 LLR을 복호 NN에 cascade하여 전체를 하나의 큰 NN으로 학습.
7. 구현 디테일: 한 neuron이 2^J sub-neuron으로 구성(multi-ary 변조), 정규화로 max 차감, 병렬 구조는 ⌈N/J⌉ detector + V decoder.
8. 학습: cross-entropy loss + multiloss, RMSPROP, lr=0.01, SNR을 mini-batch마다 랜덤(10~16dB), epoch 10000; MPA는 codeword 독립성이 없어 단일 codeword 학습 불가.
9. 결과: 기존(MPA+BP) 대비 AWGN에서 BER=1e-4에 2dB, fading에서 약 1dB 이득; 학습 후 forward 복잡도는 표준 MPA/BP와 동일(학습은 offline).
10. 한계: HW 미설계(gatecount/throughput 없음), BCH·NOMA·multi-ary 변조 종속, 검출단이 codebook에 강하게 의존.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| weighted-BP NN 복호기 (tanh 기반) | 대응 없음 | Prime ECC는 min-sum HW 디코더이며 NN/딥러닝 full-tanh BP 복호는 모듈 대응이 없음(프로파일 §6 명시 "대응 없음") |
| edge 가중치 학습 | 대응 없음 | Prime ECC에 학습 파라미터 개념 없음 |
| MPA 다중사용자 검출단 | 대응 없음 | NOMA 검출은 NAND ECC 파이프라인과 무관 |
| BP iteration 루프 (개념적) | decoder.cpp LDPC_Decoder() | 반복 message passing 구조만 유사, 실제 연산(NN weighted-BP)은 이식 불가 |
```

적용 가치: **낮음** — NN 기반 weighted-BP 전면 대체 + BCH 부호 + NOMA/5G multi-ary 변조 특화로, 프로파일 §5 판정필터상 NN 전면 대체·무선 특화는 관련성 낮음이며 Prime ECC의 binary QC-LDPC min-sum HW 엔진에 떼어 쓸 요소가 없음.
