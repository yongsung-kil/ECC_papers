### B. 알고리즘 요약

1. BSC 채널(crossover probability `α`), 실사용 QC-LDPC 코드 2종(dv=4, rate `R=0.903`, circulant `Z=32`, `N=9216`/`18432`, girth 6) + IEEE 802.3an(2048,1723, dv=6) 대상, quantized MS/OMS(4-bit) flooding 디코더로 검증.
2. 풀려는 문제: NAND flash·광통신처럼 FER<10^-12가 요구되는 응용에서 MC 시뮬레이션이나 기존 importance sampling은 harmful trapping set(TS)에 대한 사전지식이 필요해 강한 디코더(min-sum, FAID)·고차수(dv≥4) 코드에 적용 불가.
3. 핵심 가정/모델: TS 외부에서 들어오는 메시지는 "correct"로 가정하고 computation tree로 근사(식 (4)) — 실제 전체 그래프에서 동작하는 메시지를 국소 서브그래프로 정확히 모사하는 것이 핵심.
4. 핵심 기법 1단(expansion): girth `g`, `g+2` 길이의 짧은 cycle 목록에서 시작해, degree-1 CN의 외부 VN이 다른 CN과 연결되면 반복적으로 편입시켜 서브그래프 `G_EXP`를 확장(임계값 `κ`로 크기 제한).
5. 핵심 식: FER `≈ Σ_{μ,s} sΓ_{μ,s}α^μ(1-α)^{N-μ}` (식 (2)) — critical number `μ`(최소 오류 weight)와 strength `s`(해당 weight 오류패턴 개수)가 EF 기울기·위치를 결정하므로, 이 두 값을 정확히 구하는 것이 목표.
6. 핵심 기법 2단(contraction): 확장된 서브그래프 `G_EXP`의 모든 변수노드에 weight 1부터 exhaustive하게 오류패턴을 주입해 실제 디코더 `D`로 복호 실패 여부 확인, 실패 패턴들의 support 합집합이 곧 TS(`V_CONTR`)이자 critical number/strength 산출.
7. 부수 트릭: QC-LDPC의 준순환 구조를 이용해 non-isomorphic cycle만 열거(circulant 크기 `Z`만큼 복잡도 축소), CN degree>1은 내부 오류 메시지가 항상 최소이므로 외부 메시지 모델링을 degree-1 CN에만 국한해 단순화.
8. 학습/최적화 절차 없음 — exhaustive search(전수탐색) 기반 결정론적 절차.
9. 결과: QC 2KB 코드(N=18432)에서 MC 대비 약 100만 배(10^6), cycle enumeration 기반 기존 기법([20]) 대비 약 3×10^5배 빠르게 FER=10^-10대 error-floor 정확 추정, OMS(Y=5,γ=1)의 critical number `μ=4`, strength `s=2`인 TS를 직접 특정.
10. 한계: BSC·quantized MS/OMS 검증에 한정(AWGN·양자화 soft 채널 확장은 future work), 부호나 디코더 알고리즘 자체를 개선하지 않는 순수 사후 분석·검증 도구이며 HW 설계·throughput 언급 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QC-LDPC 준순환 구조 이용 cycle 열거 축소 | `ecc_top.cpp` `Load_PCM()` | Prime ECC의 base+circulant PCM(`PCM_b`) 구조에 그대로 적용 가능한 non-isomorphic cycle 열거 최적화 |
| Quantized MS/OMS 디코더 exhaustive contraction | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC의 min-sum 로직을 그대로 서브그래프에 적용해 TS 탐색 가능(로직 변경 없이 검증용 wrapper로 재사용) |
| Trapping set profile을 이용한 dominant TS 식별 | 대응 없음(신규 검증 도구) | Prime ECC에는 error-floor 사전탐지/검증 파이프라인이 없어 신규 도구로 추가 이식 여지 |
| BSC 채널·hard-decision 모델링 | `channel.cpp` `Set_R_Offset()` | Prime ECC의 hard-1bit(HD) 모드와 채널 가정이 일치 |

> 적용 가치: 높음 — NAND를 명시적으로 타깃 응용으로 언급하고 Prime ECC와 동일한 high-rate(~0.9) QC-LDPC + quantized min-sum 조합에서 error-floor를 trapping set 사전지식 없이 100만 배 빠르게 정확 추정 가능해, 부호/디코더 검증·설계 피드백 파이프라인으로 직접 이식 가치가 큼(단, 부호·디코더 자체 개선은 아님).
