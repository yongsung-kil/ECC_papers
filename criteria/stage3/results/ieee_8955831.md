### B. 알고리즘 요약

1. 대상 채널은 2차원 자기기록(TDMR) 시스템으로, 다운트랙·크로스트랙 양방향 간섭(ISI+ITI)을 갖는 3×3 CIR 행렬(`H4T`, 기록밀도 4 Tb/in²)을 사용하는 turbo-equalized 시스템(BCJR 검출기 + LDPC 디코더).
2. 기존 SC-protograph(SC-P) 부호는 메모리리스 채널(BEC/AWGN)에서는 우수하나 TDMR 채널에서는 성능이 검증되지 않아, 재설계가 필요하다는 문제의식.
3. 채널 모델: readback signal `y_ij = Σ h_st x_(i-s,j-t) + n_ij`, AWGN 노이즈, 경계는 `-1` 패딩.
4. 핵심 기법 1단: turbo-like EXIT 알고리즘으로 BCJR 검출기와 LDPC 디코더 간 상호정보(MI) 수렴 임계값(decoding threshold)을 산출 — `I_Ed = 1 - E[log2(1+e^{-u_j l_j})]`.
5. 이 EXIT threshold는 유한장 BER 시뮬레이션의 워터폴 영역과 일치해 코드 성능을 사전 예측하는 지표로 사용.
6. 핵심 기법 2단: AWD(asymptotic-weight-distribution) 분석으로 최소거리 성장률(TMDR `δmin`)을 계산, `dmin = N × δmin`로 선형최소거리 특성 판별.
7. 부수 설계 기법: (1) uniform-weight edge-spreading으로 coupling width `w=1`일 때 rate loss 최소화, (2) 모든 CN에 연결되는 고차수 VN 1개 + 필요 시 degree-2 VN 추가로 rate compensation(원래 protograph rate와 일치시킴).
8. 최적화 절차: 새 VN의 CN별 edge 수를 turbo-like EXIT 분석으로 탐색·결정(별도 GA/NN 없음, 반복적 threshold 비교).
9. 결과: enhanced (3,6) SC-P 코드가 기존 SC-P 대비 BER 10⁻⁵에서 약 0.1dB, 원 protograph 대비 약 0.2dB 이득, error-floor 미관측(BER 10⁻⁶까지).
10. 한계: HW 미설계, 이론+시뮬만 존재, 채널이 magnetic recording ISI/ITI 전용(NAND read channel과 무관), 코드워드 길이 4800 고정 예제.
