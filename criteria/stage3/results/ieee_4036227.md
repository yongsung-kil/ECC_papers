> 총평: degree-2 variable node 비율이 최대(n2=m)인 특정 구조 LDPC 코드 앙상블의 minimum distance가 sub-linear임을 증명하는 순수 조합론 논문. capacity-approaching(용량 근접) 성능과 낮은 error-floor 사이의 트레이드오프를 이론적으로 뒷받침하나, NAND ECC 코드설계에 직접 이식할 요소 없음.

### B. 알고리즘 요약

1. 배경: LDPC 코드에서 degree-2 variable node는 용량에 근접하려면 그 비율이 커야 하지만(용량-achieving sequence 조건 `λ2ρ→1/p`), 비율이 커지면 error-floor가 악화되는 상충 관계가 있음.
2. 문제: degree-2 variable node 수 `n2`가 parity-check 수 `m`과 같은(`n2=m`) 극단적 구조(선형시간 인코딩 가능)에서 minimum distance가 어떻게 거동하는지 미해결.
3. 모델: parity-check matrix `H=[Cm|M]`, `Cm`은 degree-2 노드들이 하나의 사이클을 이루는 `m×m` 행렬, `M`은 열 weight `c`(degree-c variable node), 행 weight `d`로 고정된 앙상블.
4. 핵심 결과(Thm.1-i): `M`을 무작위 균등 선택 시, `dmin ≤ αn^(1-2/c)`일 확률이 `O(α^(c/2))+O(n^(2/c-1))`(c 짝수) 또는 `O(α^c)+O(n^(2/c-1))`(c 홀수)로 상한.
5. 핵심 식의 의미: minimum distance의 전형값이 `O(n^(1-2/c))` 수준(c=3이면 `n^{1/3}`, c=4이면 `√n`)으로, c가 클수록(즉 degree-c 노드 차수가 높을수록) error-floor가 완화됨을 시사.
6. 핵심 기법(Thm.1-ii): weight-c 벡터 공간에 metric `dm,c`를 정의하고 sphere-packing(구 채우기) 논증으로 `dmin ≤ O(n^(1-1/c))` 상한을 별도 유도(무작위성 가정 없이 항상 성립).
7. 증명 도구: 코드워드 가중치 분포의 기댓값(`a_{1...t}`)을 Stirling 근사와 생성함수(`f(x)`, `g(x)`) 기법으로 상한 계산.
8. 학습/최적화 절차 없음(순수 확률론적/조합론적 증명).
9. 결과: 수치 시뮬레이션 없음. 이론적으로 `c=3`일 때 `dmin~n^{1/3}`, `c=4`일 때 `dmin~√n` 수준의 sub-linear 거동을 도출.
10. 한계: 순수 이론(증명)뿐이며 디코더 성능, HW, 실제 코드 구성 예시 없음. serial turbo code와 유사한 거동이라는 정성적 비유만 제시.

| minimum distance 점근 분석 | 대응 없음 | 코드 자체는 시뮬레이터로 특정 H-matrix의 실측 성능만 확인하며 점근적 조합론 분석 기능 없음 |
| error-floor/waterfall trade-off 논의 | `decoder.cpp` `LDPC_Decoder()` | 목적 의식은 관련 있으나(error-floor 관심) 본 논문은 부호 구성 단계의 조합론적 상한 증명이라 디코더 모듈과 무관 |

> 적용 가치: 낮음 — 특정 앙상블(degree-2 node가 정확히 parity-check 수와 같은 구조)의 minimum distance 점근 상한을 다루는 순수 이론 논문으로, Prime ECC의 고정 고rate QC-LDPC 구조나 디코더 알고리즘에 이식할 구체적 기법이 없음.
