### B. 알고리즘 요약

1. 대상은 F개 fading block을 span하는 non-ergodic block-fading 채널용 full-diversity Root-LDPC 코드(BPSK, AWGN 잡음, Rayleigh fading `α_f`).
2. 문제: 기존 PEG는 cycle `girth`만 최대화하고 cycle connectivity를 무시하여 FER 성능이 제한됨.
3. 모델: 부분 Hamming weight `w_j(c)`로 diversity order `d=min|{w_j(c)≠0}|` 정의, full diversity는 `d=F`.
4. 1단 MPEG: PEG 트리 확장 시 최소weight check node 후보들 중 `ACE` 최대인 것을 선택해 edge 배치.
5. `ACE = Σ(d_i − 2)` (cycle 내 variable node degree 합), stopping set 연결도를 근사 측정 → connectivity 개선으로 error rate 감소.
6. 2단 MPEG-QC: circulant sub-matrix의 cyclic shift 값을 MPEG로 결정, Root-Check 구조(`I_ij`) 유지하며 QC화.
7. 구현 디테일: 지시벡터 `Z1,Z2`(F=2), `z1..zF`(F=3,4)로 Root 구조 위치 제약, submatrix 내 first edge 후 circulant shift로 나머지 배치.
8. 부호구조: R=1/3, R=1/4용 PCM을 명시적으로 제시, degree distribution `λ(x)`,`ρ(x)` 지정.
9. 결과: N=900, 20 iteration BP에서 MPEG-QC가 QC보다 FER 우수(outage limit 근접), full diversity 달성, medium~high SNR에서 평균 iteration 수 감소.
10. 한계: HW 미설계, 시뮬만, 저rate(1/3,1/4) block-fading 전용, NAND용 고rate AWGN QC-LDPC와 정합 안 됨.
