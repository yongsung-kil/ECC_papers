### B. 알고리즘 요약

1. 시스템: SISO uncorrelated Rayleigh fading 채널(`r = hs + n`), 고차 QAM(256-QAM), 표준 rate-1/2 LDPC 부호와 결합.
2. 문제: 메모리 제약 수신단에서 LLR을 최소 비트로 저장해야 하며, sufficient statistic `(G, r̃_r, r̃_i)`를 그대로 scalar 양자화하면 `G` 때문에 심한 error floor 발생.
3. 핵심 개념: 한 심볼의 LLR은 3차원 sufficient statistic으로 완전 복원 가능 → latent 차원을 3으로 고정한 deep autoencoder로 압축.
4. 핵심 기법: soft bit `Λi = tanh(Li/2)`를 입력, encoder `f`가 3차원 latent `z`로 매핑, decoder `g`가 복원(식 6,12).
5. 핵심 식: 가중 손실 `D = Σ|Λj−Λ̃j|²/(|Λj|+ε)`(식 9)로 작은(불확실) LLR 복원을 강조 → 디코딩 error propagation 억제.
6. 확장: encoder/decoder 마지막 층 `tanh`로 `z ∈ [−1,1]³` 제약, 동일 양자화기 재사용·에너지 무한증가 방지(암묵 정규화).
7. 구현 디테일: 학습 시 미분가능한 Gaussian noise layer `z̃ = z + n`로 양자화 잡음을 근사, 추론 시 `Nb`비트 uniform scalar 양자화(식 11, `δ=0.8`).
8. 학습: encoder/decoder 각 3개 FC hidden layer(폭 `4K`), ReLU+tanh, Adam(lr 1e-3), 여러 SNR의 codeword LLR 혼합 학습(2000 epoch).
9. 결과: 256-QAM single-tx에서 scalar 대비 2.7배 압축, full-precision 대비 0.15dB 손실; HARQ(rate 3/8)에서 15bit로 32bit scalar 수준(2.13배 압축).
10. 한계: 시뮬레이션만(HW 미설계), `h` 분포 바뀌면 전체 재학습 필요, 무선 fading·고차 QAM 특화라 NAND 채널 가정과 상이.
