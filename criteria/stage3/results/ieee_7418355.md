### B. 알고리즘 요약

1. VLC 시스템에서 OOK 변조 + regular (3,6) LDPC 부호(rate 0.5, block length 20000)를 사용하는 APD 수신기 모델.
2. 기존 LLR 계산은 signal-independent Gaussian noise(SIGN)를 가정하지만 실제 APD 채널은 조명 강도에 비례한 shot noise로 인해 constellation point별 노이즈 분산이 다른 signal-dependent Gaussian noise(SDGN) 채널이다.
3. 핵심 가정: shot noise 분산 `σ_sd^2 = γ·h·x` (수신 신호 x에 비례), 신호독립 잡음 `σ_si^2`와의 비율 `f = σ̄_sd^2/σ_si^2`로 채널 특성을 파라미터화.
4. 최적 LLR `Λ_opt`(식 9)를 SDGN 채널의 사후확률비로 유도, κ0/κ1(식 8)로 constellation point별 분산 스케일 반영.
5. `Λ_opt`는 σ_n 추정이 필요해 실용성이 낮으므로, MS 디코딩용 근사 LLR `Λ̃_opt`(식 12)를 σ_n 없이 유도해 계산 복잡도 절감.
6. correction factor `α`(기존 문헌값 0.8, rate 0.5용)를 근사 min-sum에 적용해 성능 격차를 축소.
7. EXIT chart로 optimal/sub-optimal detection의 decoding threshold를 계산해 BER 시뮬레이션 없이 빠르게 성능 예측.
8. 별도 학습/최적화 절차 없음(채널 모델 유도 + EXIT 임계값 계산만).
9. 결과: BER 10^-6 기준 modulation depth m=1, f=1.4에서 SIGN 대비 SDGN 최적검출로 0.7dB 이득. σ_n 과대추정 시 BP가 오히려 근사 MS보다 성능 열화.
10. 한계: APD 광통신 특유의 shot noise 채널 모델(NAND read 채널과 무관), HW 미설계, 부호 자체는 표준 regular LDPC 예제일 뿐 개선 대상 아님.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SDGN 채널 LLR 계산/양자화 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | 채널 모델 자체가 VLC APD shot noise 전용이라 NAND RBER 채널에 직접 대응 안 됨, 개념적 참고만 가능 |
| min-sum correction factor α | `decoder.cpp` `CNU_Update_New_Mag()` | scaled min-sum의 α factor는 이미 일반적으로 알려진 기법(Chen & Fossorier)이라 신규성 낮음 |
| 부호(regular (3,6) LDPC) | 대응 없음 | 부호 자체는 개선 대상이 아닌 예제 사용 |

> 적용 가치: 낮음 — VLC 광통신 채널 특유의 signal-dependent noise 모델에 국한된 LLR 유도로, NAND flash 채널 특성과 무관하며 부호/디코더 알고리즘 자체의 신규 기여도 없음.
