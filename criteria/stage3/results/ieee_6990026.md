### B. 알고리즘 요약

1. 시스템: 100 Gb/s 이상 광전송(PDM-QPSK, 25 GS/s), soft-decision FEC용 GLDPC 부호 설계.
2. 문제: 기존 단일 local code 기반 GLDPC는 rate가 낮거나 성능/복잡도 균형이 부족.
3. 모델: LZ-codes(Lentmaier-Zigangirov)를 채택 - global LDPC의 각 row를 local code의 parity-check matrix로 치환.
4. 핵심 기법 1단: global code의 i-번째 row를 s-번째 local code `Hs`의 PCM으로 치환, 여러 component code(Hamming, BCH, RM)를 혼용.
5. 핵심 식: rate `R`은 `1-(W/n)-(W/d)(1-(k-1)/n) ≤ R ≤ RG`, `RG=1-W/n`. 파라미터 `d`(매 d번째 row만 (n,k) local code) 조정으로 rate 연속 튜닝.
6. 확장: 나머지 row는 단일 parity-check로 두어 낮은 rate 문제 해소, MAP 복호 trivial.
7. 구현 디테일: local RM code는 FHWT 기반 MAP 복호로 복잡도 `n log n`, sum-product 대비 낮음.
8. 최적화: 별도 학습 없음, global code로 girth-8/10 column-weight-3 QC-LDPC/LDPC 사용.
9. 결과: GLDPC(53168,42373,0.7969) NCG 9.65 dB @ BER 1e-9(예상 12.1 dB @ 1e-15), LDPC(16935) 대비 복잡도 1.66배 낮음.
10. 한계: HW 미설계, Monte Carlo 시뮬만, 광채널(OSNR) 가정, local block code 기반이라 binary QC-LDPC 디코더와 이질적.
