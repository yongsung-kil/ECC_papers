> 총평: 2비트 coarse-quantization MP 디코딩의 DE·error-floor 이론과 QMP 전용 부호설계를 제시. 메시지 2비트라는 점은 NAND soft-read 비용 절감과 통하지만 알고리즘 자체가 우리 min-sum과 상이해 부호설계 제약(degree 분포·trapping set) 관점의 참고가 주된 가치.

### B. 알고리즘 요약

1. 대상: binary-input AWGN, unstructured irregular LDPC 부호앙상블. 내부 메시지를 2비트(quaternary) 알파벳 `M={-H,-L,+L,+H}`로 표현하는 QMP 디코딩.
2. 문제: 메시지 4~5비트면 수백 Gbps 가능하나 데이터플로우가 비트폭에 선형 → 1~2비트 coarse quantization으로 더 높은 rate 지향, 단 부호/디코더 재설계 필요.
3. 모델: CN→VN 메시지를 extrinsic DMC의 hard observation으로 보고, VN은 전이확률(DE로 추정)을 알면 soft 메시지로 변환.
4. 핵심 기법(QMP): 초기 VN 메시지 `Ψ(L_ch)` (식1-2, 임계값 `T`로 4레벨 양자화), CN 갱신 `m_c->v = min|·|·prod sign` (식3), VN 갱신 `m_v->c = Ψ(L_ch + Σ L_ex(m_c->v))` (식4).
5. 핵심식: extrinsic LLR `L_ex(m)=w^(l)(|m|)·sign(m)`, 신뢰도항 `w^(l)(L)=ln(q_L^+/q_L^-)`, `w^(l)(H)=ln(q_H^+/q_H^-)` (식6) — DE로 iteration별 추정하되 SNR 전역 고정.
6. DE 분석: unstructured irregular 앙상블에 대해 `p(m)`, `q(m)` 확률 추적식 유도, 반복 임계값(Eb/N0) 산출. 고rate일수록 Shannon 갭 작음(R=7/8에서 0.6dB).
7. Error-floor 분석: degree-2/3 VN만으로 된 cycle이 elementary trapping set `Ta,b` 형성. CEM/CIM 개념 도입, Lemma 1로 QMP 실패 조건 규명 (channel LLR·CIM 모두 오류+CEM 정확 시 정정 불가).
8. 부호설계: DE(differential evolution)로 R∈{1/2,...,9/10} degree 분포 최적화, 제약식 `3λ2+4λ3 < 6/d̄c` (식7)로 유해 cycle 억제.
9. 결과: 설계 부호 C2 (5120,4096, R=4/5)가 5G LDPC 대비 BP 기준 0.55dB 내 성능. 5G 부호는 QMP 시 0.8dB 손실+error floor(>1e-4). Trapping-set VN의 채널 LLR을 20 iter 후 0으로 재설정하는 완화로 1e-8에서 ~1dB 이득.
10. 한계: HW 미설계, AWGN/광통신 대상 시뮬·이론, 내부 2비트 QMP 전용 알고리즘.
