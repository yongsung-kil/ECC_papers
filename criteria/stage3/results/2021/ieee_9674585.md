### B. 알고리즘 요약

1. 대상은 binary QC-LDPC (base matrix `B[J×L]` + circulant shift, lifting size `Z`)로, 5G NR-LDPC(BG1/BG2, raptor-like)와 WiMAX 부호. rate는 BG1 R=1/2, BG2 R=1/3, WiMAX R=2/3.
2. 문제: 하나의 base matrix를 여러 rate·length에 공용하면 SNR 구간별 성능 trade-off가 불가피, 특히 저rate 단부호가 error-floor를 겪음.
3. 채널/복호 모델: AWGN + BPSK, layered normalized min-sum(LNMS) 50 iteration, 부호별 normalized factor 개별 최적화.
4. 핵심 기법: Genetic Algorithm으로 각 row의 특성집합 `d_j={p_j,e_j}`(CPM 위치·shift 값)을 진화. row weight `ρ_j`는 고정, 열 범위 `[L1,LV]` 내에서만 최적화.
5. 최적화는 row 단위 순차 진행: `j=J1..JC`, 각 row 최적화 결과를 다음 row에 반영해 전체 matrix 개선.
6. 확장(multi-length): 최대 lifting size `Z_S`로 population 진화 후 식(4) `mod(e_j,Z)`로 작은 Z 부호 유도, 여러 Z를 동시 평가.
7. 부수: mutation(위치 재추첨·교환, shift 값 치환), crossover(위치는 중복 시 skip, shift는 제한 없음) 두 축으로 분리 설계.
8. Fitness: 다중 length를 단일 기준으로 평가하려 통계량 `η_k,q = Σ_s (γ0_s−γk_s)/γ0_s` (design SNR별 FER 개선율 합) 정의, `f=1/η`.
9. 결과: BG1 Z=6에서 FER=1e-5에서 약 0.5 dB 이득, BG2 평균 0.5 dB, WiMAX FER=1e-4에서 약 0.1 dB 이득, error-floor 제거.
10. 한계: HW 미설계, 순수 시뮬(SIMD 가속), waterfall 영역 이득은 거의 없거나 소폭 손실, 대상 부호가 무선 표준 부호에 한정.
