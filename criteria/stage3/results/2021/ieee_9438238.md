### B. 알고리즘 요약

1. 시스템: 무선(5G/위성) 단블록 통신, codeword length `256`, code rate `1/2, 3/4, 7/8`, BPSK/QPSK/8PSK 변조. H 차원 128×256, 64×256, 32×256.
2. 문제: 대블록 LDPC는 성능 좋으나 latency 큼. 단블록에서 latency-성능 trade-off를 개선하는 구성법 부재.
3. 모델: Euclidean geometry `EG(m,2^s)` 위에서 minimal polynomial과 parity polynomial `p(x)`로 부호 정의, girth≥6 보장.
4. 핵심 기법: Type-II EG QC-LDPC + circulant decomposition으로 hybrid QC-LDPC(HQC-LDPC) 구성. H를 decomposition matrix `D`, permutation matrix `P`, distribution matrix `M` 세 서브행렬로 설계.
5. 핵심 식: `g(X)=LCM{φ1..φn}`로 generator polynomial, `p(X)=(X^n−1)/g(X)`로 parity polynomial 산출. 연속 power는 짧은 cycle 유발하므로 회피/permute하여 girth≥6.
6. 확장: 파라미터 `Dc`(column decomposition factor), `Ds`(distribution factor), weight `w`로 서브행렬 개수/차원(32×32 또는 64×64) 제어.
7. 구현 디테일: reference column에 zero 부착으로 31→32, 63→64 차원 정합. 서브행렬 `Nij`는 reference column의 downward cyclic shift로 생성.
8. 최적화: 별도 학습 없음. reference index(even/prime power) 규칙과 permutation check로 결정론적 구성.
9. 결과: SPA 디코딩 시뮬에서 1/2 rate BPSK BER 10^-2 @ 1.55dB, error floor @ 3dB. BG1/BG2 및 dispersion&masking 대비 우수, 대블록 algebraic/PEG와 경쟁적(훨씬 작은 블록으로).
10. 한계: HW 미설계, 시뮬(SPA)만, 무선 PSK 채널 특화. 기존 random LDPC용 encoder/decoder 재사용 가능하다는 점 외 신규 복호 기여 없음.
