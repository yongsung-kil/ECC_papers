### B. 알고리즘 요약

1. 시스템은 16QAM(및 64QAM) MLC-MSD, AWGN 채널, component code로 irregular QC-LDPC(PEG 구성, `N=5400`) 사용, 16QAM은 `R=3 bit/symbol`(overhead 33%).
2. 문제: 용량규칙(capacity rule)대로 component rate를 배분하면 Ungerboeck set partitioning(UP)에서 심각한 error floor, Gray labeling에서도 error floor 경향 발생.
3. MLC는 물리채널을 `log2(M)`개 등가 서브채널로 분할, MSD 수신기가 저레벨 복호결과를 고레벨 LLR 계산에 참조(상호정보 이용, 대신 오류전파 유발).
4. 상호정보 chain rule (식1)과 AWGN 전이확률(식2)로 각 등가 서브채널 용량 계산 → Table I (UP: C1=0.282, C4=1).
5. 원인 분석: UP에서 4번째 서브채널의 rate가 1(LDPC 미보호)이라 고SNR에서 BER이 1e-5에 정체 → 전체 시스템 error floor 유발.
6. Gray labeling에서는 1·3번째 서브채널 보호가 부족(preBER와 반대 경향).
7. 해법(UP): 4번째 component rate를 3번째와 동일하게 조정, 나머지 유지 → 총 전송률 거의 불변으로 error floor 대폭 감소.
8. 해법(Gray): 1·3번째 rate를 낮추고 2·4번째를 높여 재배분.
9. 결과: UP error floor 크게 감소, Gray labeling 16QAM에서 BER 1e-7 기준 약 0.3dB 코딩이득 개선. 64QAM에도 확장 가능(지면상 미표시).
10. 한계: HW 미설계·시뮬만, 디코더/부호 자체 개선이 아닌 MLC 서브채널 rate 배분 튜닝, 광통신 MLC 특화.
