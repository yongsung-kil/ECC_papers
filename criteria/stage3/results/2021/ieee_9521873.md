### B. 알고리즘 요약

1. 대상: `(l, r, w, L, M)` finite-length regular SC-LDPC 앙상블, 채널은 BEC(erasure prob `ε`), 부호율 `R = 1 - l/r - (l/(r))·(w-1)/L`(식 1), 예시 rate ~0.46~0.5.
2. 문제: 기존 scaling law는 coupling width `w = l` 제약을 두어 `w` 효과를 독립 분석 불가, 주어진 spec(rate/frame/복잡도)에 맞는 파라미터 최적화 미해결.
3. 모델: peeling decoding의 degree-1 CN 개수 `r1(τ)` 진화를 Ornstein-Uhlenbeck 과정으로 근사, scaling parameter `(γ, δ, θ)`로 특성화.
4. 핵심 기법 1: `w = l` 제약 제거한 일반 edge-spreading(`S1..Sw`)로 expected graph evolution 미분방정식 확장 → `(γ, δ, θ)` 계산.
5. 핵심 식: `α ≜ γ/√δ`가 FER 곡선 기울기를 지배(식 2/3), refined scaling law로 FER 예측.
6. 핵심 관찰: `w`가 클수록 `α`↑(가파른 slope)이나 rate-loss↑ → w는 rate와 scaling의 trade-off.
7. 트릭: rate 맞춤 비교 위해 random puncturing(`ρ=1-R/Rtarget`)으로 목표율 0.5 정렬, puncturing loss `(1-ε)ρ`.
8. 최적화 절차: 목표 FER(1e-4)에서 모든 `(l,r,w)`의 달성 `ε`을 scaling law로 계산, 최대 `ε` 파라미터를 최적으로 선택(시뮬 불필요).
9. 결과: 최적 `(l,r,w)`는 `L,M`에 따라 달라 단일 우세 파라미터 없음 — 소`M`은 α 지배(3,6,3), 대`M`은 BP threshold 지배(4,8,5) 등 영역 분할.
10. 한계: BEC·regular SC-LDPC·peeling에 국한, AWGN/BSC(NAND 실채널) 미검증, HW·복호기 알고리즘 무변경, 새 부호류 제안 아님(기존 파라미터 튜닝).

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SC-LDPC (l,r,w,L,M) 부호구조 | 대응 없음 | Prime은 block binary QC-LDPC (ecc_top.cpp Load_PCM), SC(convolutional) 구조 미지원 |
| scaling law 기반 파라미터 최적화 | 대응 없음 | BEC peeling 이론 — 디코더/부호 로드 모듈과 접점 없음 |
| random puncturing로 rate 정렬 | ecc_top.cpp Load_PCM() (H-matrix 종속) | 개념만 근접, Prime은 punct_col_num=2 고정 구조라 직접 이식 아님 |
```

적용 가치: **낮음** — Prime ECC는 고rate block QC-LDPC(AWGN/RBER)이고 본 논문은 BEC 위 SC-LDPC 앙상블 파라미터 최적화 이론이라 부호구조·채널·디코더 계층이 모두 어긋난다.
