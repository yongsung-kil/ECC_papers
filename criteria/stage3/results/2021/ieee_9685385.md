### B. 알고리즘 요약

1. 시스템: RS-SPC product code (행=RS `C1(n,k,δ)` over GF(2^p), 열=binary SPC `C2`). AWGN·BPSK. 예제: `P(255,239,4,32)` rate 0.93, `P(63,51,1,15)` 6048bit rate 0.759.
2. 문제: RS 단일부호의 SDD는 MLD와 격차 큼; RS PCM은 비희소(HDPC)라 BP 직접적용 곤란 (ABP는 매 iteration Gaussian소거로 복잡도 과함).
3. 부호구성: RS의 이진 image expansion `Hb`(밀도 ~0.5, sparsification으로 ~0.3)와 SPC를 결합해 전체 이진 PCM `H(n,k,w,L)` 구성 (식(6): 대각 `Δ(Hb,L)` + 순환 `R` + `I`).
4. 2단계 복호: local decoding(LD)=성분별 BM-HDD, global decoding(GD)=전체 이진 PCM에 BP 반복. LD 저지연 유지 + GD로 큰 블록급 정정능력 확보.
5. 조기판정: soft weight `W=Σ|y_i|(y_i^H⊕ĉ_i)/Σ|y_i|` (식(9), 0≤W≤1), 임계 `Wθ`보다 작으면 정답으로 간주하고 해당 RS 성분을 freeze(LLR 큰값 고정, active set 제거).
6. CN 업데이트: check node를 `M1`(RS부분)/`M2`(SPC부분)로 분할, `M1`엔 SSID식 순환 shift `Θ(L,µ)` 적용. extrinsic LLR은 MSA `sgn·min` (식(14))로 계산 (APP 기반).
7. VN 업데이트: `L^{κ1+1}=L0+α1·L_ext,1+α2·L_ext,2` (damping factor `α1,α2`). inner/outer iteration `κ1,κ2`, `N2=1`(LCS)/`N2=n`(HCS) 두 모드.
8. 복잡도: CN당 실수곱 2회, per-bit 정규화 복잡도를 Chase-Pyndiah TPC와 비교 (Table I). 큰 SNR에서 `Iavg≈2`로 수렴.
9. 결과: BER 10^-7에서 `P(255,239,4,32)` HCS는 TPC `PR(63,61)` 대비 0.4dB 이내; LD 성능은 TPC LD보다 1.9dB 우수. 큰 SNR에서 TPC보다 낮은 복잡도.
10. 한계: HW 미설계(시뮬만); 성분부호가 비이진 RS, PCM이 HDPC라 희소 binary LDPC와 구조 상이; undetected error 제어(Wθ 튜닝) 필요.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| CN extrinsic LLR = MSA sgn·min (식14) | decoder.cpp CNU_Update_New_Mag(), C2V_Cal_New_Sgn() | min-sum CN 연산 자체는 공유하나 우리는 희소 QC-LDPC, 논문은 HDPC 이진확장 RS |
| VN 업데이트 / damping α1,α2 | decoder.cpp VN_Cal_HD(), decoder.h Get_VNU_* | VN 합산 구조 유사, damping/2단 extrinsic은 우리 스케줄과 상이 |
| soft weight Wθ 조기판정·freeze | full_CRC.cpp, partial_CRC.cpp partial_crc_HW_T4() | 조기종료 목적 유사하나 우리는 CRC 기반, 논문은 ML-metric 임계 |
| 이진확장 PCM/부호구조 로드 H(n,k,w,L) | ecc_top.cpp Load_PCM() | 로드 지점은 대응하나 HDPC·RS 기반이라 우리 QC-LDPC H-matrix와 부호설계 상이 |
| LLR 초기화 L(c_i)=2y_i/σ² | channel.cpp Set_LLR_Th() | LLR 산출 개념만 공유 |
| product code 인코딩(RS×SPC) | encoder.cpp | dual-diagonal QC-LDPC 인코더와 부호구조 상이 |
```

적용 가치: **낮음** — min-sum(MSA) CN 연산은 이미 보유(중복)이고, 부호가 비이진 RS 성분의 HDPC 이진확장이라 NAND binary QC-LDPC의 부호설계·디코더에 떼어낼 재사용 가치가 낮다.
