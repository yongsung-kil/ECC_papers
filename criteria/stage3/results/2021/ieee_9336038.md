### B. 알고리즘 요약

1. 대상은 CCSDS Near-Earth C2 QC-LDPC `(8176,7154)`, rate 0.875, H는 2×16개 `511×511` 순환 부행렬로 구성.
2. 문제: 기존 C2 인코더가 고throughput에서 논리자원·레지스터를 과다 사용 — 면적 효율 개선 필요.
3. 인코딩은 direct method: 패리티 `p = sB` (`p` 1022bit, `B`는 14×2개 `511×511` 순환 부행렬), 14블록 병렬 곱셈.
4. 기본 구조: 부행렬 첫 행 `bi`만 저장하고 순환 우시프트를 shift register의 좌시프트로 모사, 첫 행 원소가 1이면 XOR로 합산.
5. 레지스터 최적화: 동기 입력 시 합산 순서를 재배열(식 12) — 14블록을 먼저 합하고 511 순환시프트 → 임시저장 레지스터 대량 절감, 대형 가산기 제거.
6. 논리 최적화: 출력 `Om,On`이 공유하는 공통 부분식(CS) `bi,m×si,f`를 추출, 회로를 common network + private network 2계층으로 분리해 중복 제거.
7. 그룹핑 알고리즘(Alg.1): 1022개 출력을 CS 최대 쌍으로 511그룹화, CS<10인 4그룹은 3-출력 그룹으로 재편해 CS>10 보장.
8. 전처리/제어: 병렬도 14가 2의 거듭제곱이 아닌 문제를, 2bit 제어명령(C0,C1)을 14bit 입력에 삽입해 16bit화, counter/FSM 대신 configuration register로 제어.
9. 결과: Virtex-5 FPGA 구현, 1658 LUT / 1038 FF, 335MHz, 4.69Gbps, 인코딩 511 clock; [8] 대비 논리 76%↓, [13] 대비 FF 87%↓.
10. 한계: C2 특정 부호에 결속, 복호/정정능력 무관, 부호설계 기여 없음, ASIC 아닌 FPGA 검증.
