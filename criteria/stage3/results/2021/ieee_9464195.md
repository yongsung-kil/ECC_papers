### B. 알고리즘 요약

1. 시스템: CCSDS near-earth type-II QC-LDPC `(8176,7154)`(rate≈0.875), `H`=2 layer × 16 circulant, circulant `511×511`, weight `Wt=2`.
2. 문제: 다중 weight circulant에서 TDMP layered 복호는 `r`번째 update가 `(r-1)` update LLR을 필요로 해 memory conflict → 병렬도·pipeline 깊이 제한(Liao `P×DepthP≤Omin`, Omin=104, P=26이면 depth≤4).
3. 모델: NMSA(scaling `α=0.75`) 기반 layered TDMP, 한 iteration = H 전 행 처리.
4. 핵심기법: single parsing으로 CN/BN 동시 갱신 update rule `L_l,r^i = Q_l,r^(i-1)+E_l,r^i+D_l,r-1^i` (식3), `D`는 1차 update에서 생성해 2차 update에 사용 → 2번째 update만 메모리 기록.
5. 핵심식: 새 병렬도 제약 `P×K≤Omin` (식4), K=D값 write 후 read까지 cycle수 → K를 작게(구현 3) 하여 conflict 회피.
6. 확장: RCU는 NMSA(min1/min2 압축저장, sign updater)로 E를 압축 형태로 저장(E-memory RAM), min-finder는 Wey tree 구조 + 단별 pipeline.
7. 구현 디테일: L-memory를 shift-register(L_01/L_10 교대, read alignment만) + MUX 선택망(matlab로 H 분석해 하드와이어), D-memory도 shift-register + MUX forwarding, 미접근 구간은 SRL LUT primitive로 구현.
8. 최적화: RCU 13-stage 최대 pipelining으로 critical path 3.95ns → 253MHz.
9. 결과: XCVU9P FPGA, 253MHz, `P=26`, 10 iteration에서 throughput 2.65Gbps(Liao 1.6Gbps 대비↑), LUT 194479/FF 252306, BRAM 감소.
10. 한계: 부호정정 성능 개선 아님(표준 NMSA), post-layer stall cycle 존재, gate count/공정 및 LLR 양자화 bit수 미기재.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| NMSA(min1/min2, sign) CN 연산 | decoder.cpp CNU_Update_New_Mag(), C2V_Cal_New_Sgn() | 이미 보유한 min-sum과 중복 |
| layered TDMP 단일 파싱 update rule | decoder.cpp LDPC_Decoder() 이터레이션/스케줄 | dual-update 교대 스케줄과 유사 개념 |
| shift-register L/D memory + MUX forwarding | 디코더 HW 모델(pipeline col_M1~P4), GT.cpp Make_GT_HW() | memory conflict 회피·edge/MUX 선택망 관점 참고 |
| pipeline 심화(13-stage)·min-finder tree | 디코더 pipeline 6-stage 모델 | HW throughput/critical path 개선 참고 |
| QC-LDPC H-matrix 구조 로드 | ecc_top.cpp Load_PCM() | 부호구조 상이(511 circulant, Wt=2) |
```

적용 가치: **중간** — memory conflict를 shift-register + pipe-stage forwarding으로 제거해 pipeline 깊이 제약을 푸는 HW 아이디어는 Prime ECC의 고throughput 디코더 설계에 참고 가치가 있으나, 복호 알고리즘(NMSA)은 이미 보유 기법이고 511-circulant·Wt=2·2-layer 특정 구조 전제라 z=32/HCU/GT 구조와의 정합 재설계가 필요하다.
