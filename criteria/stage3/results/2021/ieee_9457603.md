### B. 알고리즘 요약

1. 대상은 QC-LDPC 인코더/디코더 HW의 barrel shifter (`m = ceil(log2 q)`층 MUX로 임의 순환 shift 수행) 복잡도 감축.
2. 문제: 지원해야 할 shift 값 집합 `S ⊂ Zq`가 전체가 아니면, 표준 basis `{1,2,...,2^(m-1)}` 대신 더 작은 basis set `G`로 대체 가능 → 층수 감소.
3. basis set 정의: 모든 `s∈S`가 `G`의 부분집합 원소 합(mod q)으로 표현되면 `G`는 `S`의 basis set (`S ⊂ Σ(G)`).
4. constant shift 일반화 `(G,c)`: `s = c + Σ` 형태 허용 (barrel shifter 입출력에 고정 배선 무비용 추가 가능).
5. 핵심 함수 `ζ(q,k) = min{|S| : dim_q S = k}` — dimension이 정확히 `k`인 최소 크기 집합. 관심은 `ζ(q) = ζ(q,m)`.
6. Proposition 1: `ζ+(q,k) = ζ(q,k) + 1` (constant shift는 자유도를 1만 증가).
7. Proposition 2 (단조성): `ζ(2q,k+1) ≥ ζ(q,k) + 1`.
8. 특수해: `k≤3`에서 `ζ(q,k)=k`, `q=2^(m-1)+1`이면 `ζ(q,m)=q-1`. q=2^m 등 exhaustive 컴퓨터 탐색 표(TABLE I/II) 제시.
9. 하한 `ζ(q,k) ≥ k`, 상한 `ζ(q,k) ≤ 2^(k-1)+1`, 확률론적 방법으로 `q=2^m`에서 개선된 상한(Theorem 1) 유도.
10. 한계: HW 설계·구현·복잡도 수치 없음, 일반해·점근 거동 미해결(`log2 q ≤ ζ(q) ≤ (log2 q)^2`만 확인).
