### B. 알고리즘 요약

1. 시스템: 2-user Gaussian CFMA(MAC), BPSK, QC-LDPC 두 부호 `R1=0.900`/`R2=0.875`, 동일 길이 `N=5080`, 채널이득 `h1=√3, h2=1`.
2. 문제: 기존 [5]의 row-merging nested 부호는 VN 고립·랜덤성으로 성능보장 불가, HW 비친화(랜덤부호).
3. 모델: 수신 `y=h1x1+h2x2+n`, `n~N(0,1)`. 2-stage 복호(1단: `u1⊕u2` 조합 복호, 2단: side-info로 `u1` 복호).
4. 핵심기법1: 1단 입력 LLR `LLR(sj)`를 Gaussian 근사, root-mean-square `σch=√(2·E[LLR(sj)])`를 Monte Carlo로 산출.
5. 식 의미: `σch`가 CFMA 조합부호 채널의 초기 신뢰도를 나타내어 P-EXIT 입력으로 사용.
6. 핵심기법2: modified P-EXIT(Algorithm 1) — EMI `IE,V`,`IE,C`,`IAPP`를 base matrix `Hb(ρ,γ)` 위에서 반복 갱신, 수렴 시 iterative decoding threshold `P` 반환.
7. 부수: masking `M=W⊗H`(GF(2) masking matrix로 regular→irregular), girth 보존(g>6이면 masked girth≥g).
8. 학습/최적화: P-EXIT로 masking degree distribution `λ(X)=0.097X+0.581X²+0.322X³`, `ρ(X)=X³⁰` 설계, improved PEG로 masking matrix 생성.
9. 구성: RS-over-GF(128) 기반 `Hb(127,127)` 서브어레이 → masking → extending(꼬리에 새 행 추가)으로 nested M1(4,40)/M2(5,40), CPM(127×127) expansion.
10. 결과/한계: masked 부호가 threshold 6.170dB(0.08dB gain), BER 10⁻⁶에서 Shannon Bound 1.186dB gap; HW 미설계·시뮬만·CFMA 전용, threshold gain 미미.
