### B. 알고리즘 요약

1. 시스템: nonergodic block-fading(BF) 채널, `nc`개 독립 페이딩 블록, BPSK, rate `R=1/nc` (R=1/2,1/3,1/4 다룸), N=1800(R=1/3)/2400(R=1/4).
2. 문제: AWGN용 일반 LDPC는 BF 채널 outage bound에 도달 못함 → BF 특화 Root-LDPC 필요.
3. 목표 속성: MDS + full diversity + **nested**(수신 블록 수/구성이 달라도 동일 diversity & equal coding gain 보장) - "carousel"형 방송(GNSS 등)용.
4. 핵심 기법: Root-LDPC의 base matrix `HBr`에 대칭 조건(예 R=1/3: `h3,1=h6,4=h2,7` 등)을 강제해 정보비트 노드 관점의 uniform graph topology 확보.
5. 이 대칭이 왜 필요한가: iterative(BP) 복호 시 모든 블록에 동일 local connection을 주어 `IAP,k(γ,α)`가 블록 permutation에 불변 → equal coding gain.
6. 확장(irregular): degree 짝수 제약 없는 실용 irregular Protograph Root-LDPC를 대칭 조건 하에서 설계.
7. 최적화: 계수 `∗∈{0,1,2,3}`를 대칭 base matrix 집합에서 열거하고 PEXIT chart로 복조 threshold `Thi` 계산.
8. 선택 기준: `ΔdB = Σ|Thcapa,i − Thi|` 최소화하는 protograph 선정 (R=1/3에서 Th2=0.92dB, Th3=0.75dB).
9. 결과: 최적화 irregular 구조가 regular보다 outage bound에 근접, 대칭 덕에 FER이 수신 블록에 무관하게 균일.
10. 한계: 시뮬만, HW 미설계, Rayleigh BF 채널·저rate 전용, 두 블록 erase 시 복호 불가(비-MDS 케이스 존재).
