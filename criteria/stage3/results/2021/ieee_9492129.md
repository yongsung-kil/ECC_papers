### B. 알고리즘 요약

1. 시스템: QC-LDPC(`mb×nb` CPM 배열, 크기 `p`), AWGN/BPSK, row layered(수평) SPA로 각 iteration을 `mb`개 layer로 직렬 sub-iteration.
2. 문제: layered decoder의 error-floor 거동이 이론적으로 미규명 - 기존 상태공간 모델은 flooding 스케줄에만 한정.
3. 모델: 에러의 주 원인인 LETS(leafless ETS)를 대상으로, 채널·미충족CN 메시지를 입력으로 하는 선형 상태공간 모델(`x(ℓ)=Ax(ℓ-1)+...`)을 layered 버전으로 확장.
4. 핵심 기법 1: state variable을 layer별 systematic labeling으로 배치 → layer별 전이행렬 `Aj`, 이득행렬 `Gj(ℓ)`로 layer 갱신을 표현, 비재귀식(30)으로 `x̃(ℓ,J)` 유도.
5. 핵심 식 의미: 지배 고유값 `r̃`(전이행렬 `Ã_J→1`의 spectral radius)이 TS 내부 오류메시지 성장률을 결정 - `r̃`이 클수록 실패확률↑.
6. 핵심 기법 2: TS layer profile(TSLP) - 동일 topology TS라도 CN이 속한 layer 조합에 따라 유해성이 달라짐을 정량화(같은 TSLP끼리 system matrix 동일).
7. 구현 디테일: `r̃`은 layer 순열의 cyclic shift·reverse에 불변(Prop 2·3) → distinct `r̃` 개수는 `(J-1)!/2`로 상한.
8. 최적화 절차: `mb!` 스케줄을 2단계 탐색 - (1)평균 DE 근사로 낮은 error-floor 후보 선별, (2)정확 DE로 최저 error-floor 스케줄 확정.
9. 결과: C1 (640,192) `dv=5`, `z=64`에서 최적 순열 (2,3,1,7,4,5,6)이 최악 대비 error-floor를 크게 낮추고 flooding(Imax=60)보다도 우수; `r̃` 8그룹 12.402~16.125; C2는 Wimax (576,432) `z=24`. min-sum/offset/normalized MSA에서도 동일 상대순위 유지.
10. 한계: 이론+Monte Carlo 시뮬뿐, HW 미설계, saturating SPA 기준(box-plus), min-sum 이득은 정성 관찰만.

| CN box-plus / SPA 연산 | `decoder.cpp` `C2V_Cal()`, `CNU_Update_New_Mag()` | Prime ECC는 min-sum 기반이라 box-plus와 직접 매칭은 아니나 스케줄 상대성 결과는 참고 가능 |
| error-floor 유발 LETS/TSLP 분석 | 대응 없음 | Prime ECC에 error-floor 분석·TSLP 최적화 도구는 없음(코너케이스 추적 `corner.cpp`만 존재) |
| soft-info / 포화 LLR 양자화 | `channel.cpp` + `ecc_data.h` `PARAM_LLR` | 논문은 saturation level이 error-floor에 직접 영향 - LLR 포화 테이블 튜닝과 연결 |

적용 가치: **중간** - HW·부호 변경 없이 layer 갱신 순서만 조정해 error-floor를 낮추는 아이디어는 Prime ECC 스케줄 튜닝에 이식 여지가 있으나, box-plus SPA·특정 학술 부호 기반이라 min-sum·NAND 고rate 부호에서 재검증이 필요하다.
