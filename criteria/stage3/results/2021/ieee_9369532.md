### B. 알고리즘 요약

1. 대상: 5G NR BG2 LDPC (`K=260/520`, rate `1/2, 1/3, 1/4`), AWGN+BPSK, sum-product 복호, `Imax=30`.
2. 문제: max iteration까지 돌려도 실패하는 undecodable 블록의 불필요한 반복을 조기 종료로 줄이려 함.
3. 기존 한계: 신뢰도변화(CMM)류는 평균 reliability 계산 복잡도↑, syndrome변화(CSVP)류는 첫 몇 iteration에서 조기판단 불가.
4. 핵심 기법: `l`번째 iteration의 unsatisfied parity check 수 `N_PC(l)`(전체의 1/4만 관찰)를 입력으로 로지스틱 회귀 분류.
5. 다변수 다항 모델 `x = θ0·l + θ1·N_PC + θ2·l·N_PC + θ3·l² + θ4·N_PC² + θ5` 사용 (결정경계 비선형이라 2차항 포함).
6. `y = 1/(1+e^-x)` 시그모이드; `y>0.5`이면 undecodable로 분류(추론 시엔 `x`의 부호만으로 판정 가능).
7. 손실은 balanced cross-entropy, 가중 `α=0.1`로 설정해 type II 오류(실제 decodable을 실패로 오판)를 줄이도록 학습.
8. 학습(AdaGrad)은 offline 1회, 계수는 memory에 저장 → 디코더 런타임 추가복잡도 없음. threshold 사전계산 시 반복별 `N_th(l)` 비교로 더 저복잡.
9. 결과: type II 오류 없이(α=0.1) FER는 conventional과 동일, 고에러영역 평균 iteration 최소값 1까지 감소(CMM/CSVP보다 빠름), 복잡도 CMM/CSVP 대비 크게 낮음.
10. 한계: HW 미설계, 시뮬만, sum-product 가정, gate count/throughput 수치 없음, NAND 검증 부재.
