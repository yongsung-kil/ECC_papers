### B. 알고리즘 요약

1. 5G eMBB용 multi-edge raptor-like QC-LDPC 대상, QPSK 채널, 예: `k=120,R=0.2,N=600`부터 `k=3840,N=19200`까지, base graph `Kb=6/10`.
2. 문제: short LDPC는 small trapping set으로 성능 저하; 일반 dense NN은 codeword 수 지수증가(차원의 저주)로 학습 불가.
3. 해법 원리: 기존 BP 복호와 결합 - Nachmani의 Tanner-graph 기반 NN 디코더(각 Tanner edge = NN 정점)에 학습가능 가중치 삽입.
4. 핵심기법 1 - Min-Sum activation: `Rj→i = sign·wj→i·max(min|Qi→j|−βj→i,0)`, VN은 `Qi→j = w_ch·Li + ΣRj→i`, 가중치 `w,β`가 1,0이면 표준 MS.
5. Weight sharing: 5G의 QC 구조 활용, circulant 하나당 학습 가중치 1개만 두어 파라미터를 circulant size배 축소, 학습 수렴 가속(성능 유지).
6. 핵심기법 2 - residual connection: `Ql_i→j`를 이전 여러 variable layer 출력의 선형결합으로 shortcut(식 4), vanishing gradient 완화·학습 가속(short code에서 유효).
7. 구현 디테일: input mapping(채널 LLR 복제), output mapping(가중치 없는 VN층), zero-codeword 훈련, FER 최소화를 위해 cross-entropy에 max 적용.
8. 학습: SGD, 배치 내 SNR 랜덤(FER 10⁻¹~10⁻⁴), 긴 부호는 residual 비활성 후 iteration별 loss 가중합 `L=Σγi·Li`(γ 점감)로 vanishing gradient 대응.
9. 결과: neural MS가 동일 iteration Sum-Product 성능 달성, 일부 구성에서 약 0.3 dB 이득; length adaptation은 lifting size만 바꿔 가중치 재사용, rate adaptation은 0.1 dB 손실기준 3세트로 23 rate 커버.
10. 한계: HW 미설계, 시뮬만, 학습 필요(긴 부호 훈련 난이도↑), 5G/무선 타깃, error-floor 아닌 waterfall 영역 개선.

적용 가치: **낮음** - 학습형 NN 디코더는 Prime ECC의 고정 HW min-sum 파이프라인에 그대로 이식 불가하며, circulant당 offset/scaling 발상만 학습 배제하고 참고할 여지가 있다.
