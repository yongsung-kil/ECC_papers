### B. 알고리즘 요약

1. 시스템: AWGN + M-QAM(16/64-QAM, 802.11 bit labeling) + random `(d_v,d_c)`-regular LDPC (16QAM: (3,6) rate 0.5 len 8000, 50 iter / 64QAM: (7,20) rate 0.65 len 12000, 40 iter).
2. 문제: SPA/box-plus 기반 복조·복호는 실수 LLR을 고비트폭으로 표현·연산해 전력·면적 병목 → 저비트폭 단순 연산 필요.
3. 모델: 이산출력 채널 `p(y_k|b_k)`, `y_k∈{0..2^q-1}`, 관심변수 `X=B0⊕B1` (check node XOR)에 대한 extrinsic 정보 추출.
4. 핵심기법1: Information Bottleneck 방법으로 `β→∞`에서 결정적 압축 매핑 `t=f(y0,y1)`를 설계, 실수 LLR 없이 정수 `t`만 교환하는 LUT 노드 연산.
5. 핵심식: 라그랑지안 `L=I(Y;T)−βI(X;T)` 최소화 → `I(X;T)≤I(X;Y)` 최대 보존하는 양자화(관련정보 최대 보존이 설계목표).
6. 핵심기법2: 동일 원리를 QAM 복조로 확장 — 채널샘플 `r̃`를 IB 양자화(`q_quant`)→압축매핑 `p(t_{k,l}|r_k)`로 bit label 정보 `t_{k,l}` 산출.
7. 구현 디테일: LUT는 offline/1회 설계(고정 design-`Eb/N0`), 실제 `Eb/N0`에 mismatched로 사용; 디코더는 [7]의 IB LDPC 디코더(정수만 교환) 사용, message alignment[8]로 복조 출력 정렬.
8. 학습/최적화: KL-means IB 알고리즘[14]으로 LUT 설계, 손수 최적화한 고정 design-`Eb/N0`.
9. 결과: 16QAM 4bit IB 시스템이 double-precision BP 대비 waterfall 약 0.2 dB 손실, double-precision min-sum은 오히려 능가; 64QAM은 6bit 양자화/5bit 채널·4bit 내부 메시지로 BP 근접·min-sum 능가.
10. 한계: BER 시뮬만, HW/gate/throughput 미제시, regular LDPC·QAM 특화, NAND QC-LDPC·hard read 시나리오 미대상.
