### B. 알고리즘 요약

1. 대상: BEC에서 variable node가 type-1/type-2(다층) check에 연결되는 bilayer(및 L-layer) LDPC ensemble, 여러 복호 모드(layer1, layers1+2, ...)를 동시 보장.
2. 문제: 기존 [7]은 layer1+2를 bivariate product degree distribution으로 직접 최적화 -> 변수수가 층 최대차수 곱(→L에서 지수적)으로 폭증.
3. 모델: 각 층을 독립적인 uni-variate degree distribution `λ^(i), ρ^(i)`로 지정, `P0=Λ0^(2)`(type-2 degree 0 분율)로 rate 제어.
4. 핵심기법: 2차원 density evolution `x_l=f(ε,x,y)`, `y_l=g(ε,x,y)` (식 5a/5b)로 layer1+2 threshold `ε2*`를 비자명 (f,g)-fixed point 부재 상한으로 정의(Theorem 4/6).
5. 의미: fixed point 부재 조건이 성공복호 임계값이므로, 층별 threshold를 독립 계산 가능케 함.
6. 확장(Construction 1): 목표 threshold `ε1,ε2` 입력 -> off-the-shelf layer1 코드 선택 -> `as(ε2)` 계산 -> layer2를 `ε2·as(ε2)`용으로 선택 -> `P0=ε1/ε2`. 최적화 불필요.
7. 구현 디테일: gap-to-capacity 상한 `δ ≤ δ1 + δ2(1-P0)` (식 23), L층은 `δ ≤ Σ δi(1-P0^(i))`로 선형 확장.
8. 최적화: type-2 iteration 수 `N2` 최소화 스케줄(식 30, η 파라미터)로 layer2 접근을 stuck 시점에만 수행, 최적성 증명(Lemma 11).
9. 결과: Tornado 코드로 `R=0.79`에서 `ε2*≥0.2` 달성; layer1 단독 복호 시 평균차수 30.16% 감소; rate-vs-N2 트레이드오프 예시(N2 570→11).
10. 한계: BEC 점근 이론 전용, 유한블록·AWGN 미검증, HW/양자화/NAND 전무.
