### B. 알고리즘 요약 (10줄 내외)

1. 대상: 유한길이 binary LDPC (`Tanner (155,64)` col-wt 3, `QC-LDPC (1296,972)` col-wt 4)의 반복 min-sum 디코더를 NN으로 최적화.
2. 문제: AWGN에서 기존 반복 디코더는 수렴이 느려 작은 최대 반복수(`Lmax=5`)로는 성능이 부족 — waterfall 성능을 소반복에서 확보하려 함.
3. 구조: Tanner 그래프를 `K=2·Lmax+1` 은닉층 MLPNN으로 "unwrap", 홀/짝 은닉층이 VN(`Φ`)/CN(`Ψ`) 갱신을 모사(edge-based / node-based).
4. 핵심 제약: 한 뉴런의 nonzero weight를 모두 동일값 `w(ℓ)`로 강제(대칭성 유지) → 전 codeword 대신 all-zero codeword의 noise 실현만으로 학습 가능.
5. 학습식: cross-entropy loss `Γ(u,x)`(식2)를 Adam으로 최소화, 반복별 스칼라 weight `{w(ℓ)}`와 bias `{b(ℓ)}`만 학습(파라미터 수 최소).
6. 환원: 학습된 `{w,b}`로 최적 갱신규칙 `Φ_opt=Φ(b·y, w·m)`(식3)을 conventional VN/CN 식으로 되돌림 → HW에서 NN 불필요.
7. 양자화: 메시지를 `2l+1=7` 레벨 알파벳 `A_nn,5`와 threshold `T`로 3bit 양자화(식4), VN 갱신을 3차원(Tanner)/4차원(QC) LUT 집합으로 표현.
8. 관찰: 학습된 LUT는 초기 반복에서 message magnitude를 attenuate(빠른 성장 억제)하며, BSC용 FAID의 trapping-set 기반 attenuation과 유사한 효과.
9. 결과: 5회 반복에서 `Dnn`이 floating MSA 대비 0.4dB(Tanner)/0.45dB(QC) 이득, `Dnn`(5it)≈`Dproto`(10it) 즉 수렴 2배 가속; 3bit `Dnn,Q`가 NMSA·MSA 초과.
10. 한계: HW 미구현(면적/throughput 없음), AWGN·waterfall 한정, error-floor·NAND soft-read 미검증, weight의 code 구조 의존성은 open.
