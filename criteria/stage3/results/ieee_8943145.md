### B. 알고리즘 요약

1. 대상은 일반 이분그래프(및 LDPC Tanner graph)이며, girth `g`를 갖는 그래프에서 길이 `g`, `g+2`, `g+4`인 짧은 사이클의 개수를 세는 문제를 다룬다.
2. 기존 message-passing 알고리즘(복잡도 `O(g|V|^2Δ^2)`)보다 더 빠르게 짧은 사이클을 세는 것이 목표.
3. 채널/디코더 모델은 없음 — 순수 그래프 조합론 문제로, LDPC 코드는 "Tanner graph"라는 응용 사례로만 언급됨.
4. 표준 BFS를 변형해 각 노드의 부모집합 `P(u)`, 자식집합 `C(u)`, 루트까지의 최단경로 수 `a(u)`, 보조량 `b(u)`를 계산하는 Algorithm 2를 제안.
5. `a(u) = Σ a(u')` (부모 합), `b(u) = Σ a(u') - a(u)` (자식 합에서 자기값 차감)로 정의되며, 이를 조합하여 사이클 개수를 닫힌 보행(closed walk) 수 – 롤리팝(lollipop) 개수로 구한다(정리 1~7).
6. 일반 이분그래프에서 g-cycle과 (g+2)-cycle 계산은 `O(|V|^2Δ)`, (g+4)-cycle은 `O(|V|^2Δ^2)`(bi-regular 그래프는 `O(|V|^2Δ)`로 축소).
7. QC 구조 그래프에서는 순환군(circulant block) 내 모든 노드가 동일 파라미터를 가지므로 대표 노드 1개만 BFS 실행하면 되어 복잡도가 lifting degree `N`배 감소.
8. 별도 학습/최적화 절차 없음 — 순수 결정론적 조합 계산.
9. rate-1/2 LDPC 코드 4종(길이 1008~20000)에 대해 기존 message-passing 알고리즘 대비 최대 한 자릿수 이상 빠른 실행시간 달성(정확한 초 단위 수치는 표에만 있어 본문 서술 기준 확인 불가).
10. 그래프의 girth를 사전에 알아야 하며, HW 구현/디코딩 성능 개선과는 무관한 순수 오프라인 분석 도구(코드 평가·설계 단계의 보조 도구 성격).
