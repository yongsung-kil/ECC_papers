### B. 알고리즘 요약

1. 대상: 유한기하(EG/PG) 기반 FG-LDPC 부호, 예시로 `EG(2,2^6)` 기반 (4095,3367) cyclic 코드(rate 0.8222, 열/행무게 64, dmin=65).
2. 문제: 반복복호 LDPC의 error-floor는 작은 harmful trapping set이 지배 - 큰 최소거리·작은 trapping set 부재 부호 구성이 조합적으로 어려움.
3. 핵심가정: `(κ,τ)` trapping set = κ개 VN이 유도한 subgraph에 홀수차 CN이 τ개, harmful은 작은 κ와 작은 `τ/κ`.
4. 핵심기법: trapping set을 유한기하의 subgeometry로 표현 - κ개 점과 이를 지나는 line 집합 `Φ(Λ)`, 홀수개 점을 지나는 line 수가 곧 τ.
5. 핵심식: point-line pair 이중계수로 `τ ≥ (γ+1-κ)κ` 하한 도출 (γ=VN degree).
6. 확장: EG(2,q) parallel bundle 구조로 하한을 `τ ≥ qκ - κ^2 + Σκ_l^2` 로 개선(center/non-center line 분해).
7. 결과: `κ < γ-3`이면 `τ/κ ≥ 5`이므로 최소거리-3 미만 크기의 harmful/small trapping set 부재, absorbing set 최소크기 `γ/2+2`.
8. 예시 수치: (4095,3367) 코드는 크기 32 trapping set이 최소 1088개 홀수차 CN을 가져 error-floor가 매우 낮음.
9. 확장성: type-0/1 cyclic 및 QC-EG-LDPC(`H_EG,qc,1` CPM 배열)로 조합적 동치, 동일 trapping set 구조 상속. FG-LDPC로 BER 10^-15 이하 보고.
10. 한계: 순수 이론(증명뿐), 대수적 FG 부호에 국한, 복호 알고리즘/HW/시뮬 결과 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| trapping set/error-floor 보증하는 H-matrix 구조(RC-constraint, row-redundancy) | `ecc_top.cpp` `Load_PCM()` | 부호구조 로드 지점에서 error-floor 기준 참고 가능하나 FG 대수부호는 우리 QC 구조와 상이 |
| QC-EG-LDPC(CPM 배열) 구성 | 대응 없음 | Prime ECC는 특정 고정 base+shift QC-LDPC, FG 유래 CPM 배열과 직접 정합 불가 |
| 반복복호(SPA/MSA)의 trapping set 취약성 분석 | `decoder.cpp` `LDPC_Decoder()` | 복호는 min-sum 보유, 논문은 알고리즘 변경 없이 부호구조로 error-floor 억제 |

적용 가치: 중간 - error-floor를 최소거리로 지배시키는 trapping set 부재 증명 아이디어는 고rate QC-LDPC 부호설계 검증 기준으로 참고가치가 있으나, FG 대수부호 한정이라 Prime ECC H-matrix에 그대로 이식은 어려움.
