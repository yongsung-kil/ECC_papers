### B. 알고리즘 요약

1. 시스템: 외부 분포정합기(DM, constant-composition) + 내부 protograph LDPC로 구성한 MacKay-Neal(MN) 부호, biAWGN 채널, 예제 rate `0.1~0.666`, 유한길이 예제 `n=1200`.
2. 문제: 통신표준은 rate 유연성을 위해 다수의 LDPC PCM을 두는데, 내부부호를 바꾸지 않고 rate만 가변하고 싶음.
3. 모델: 균일 소스에 CCDM을 붙여 empirical 분포 `ω`(v의 fractional Hamming weight)를 지정, 전체 rate `R = R_O·R_I ≈ Hb(ω)·R_I`.
4. 핵심기법: 내부 mother code는 base matrix `B=[B1|B2]` protograph를 lifting(계수 `ℓ`)한 것, 앞 `h0` 타입 VN은 punctured(state VN).
5. 핵심식: BP 초기화에서 punctured VN엔 prior `Li=Δ=ln((1-ω)/ω)`, 전송 VN엔 채널 LLR `Li=2y/σ²` 주입 — 외부 CC 제약 대신 marginal 분포만 이용하는 mismatched metric.
6. 확장: 비선형 부호를 분석하기 위해 scrambler 도입 후 등가 병렬채널(EPC) 모델로 변환, all-zero 기준 분석 가능.
7. 구현 디테일: a priori 채널을 crossover `ω`의 BSC로 근사(길이↑ 시 오류율 `ω`로 집중).
8. 최적화: PEXIT/양자화 DE로 iterative threshold `γ⋆(R)` 계산, worst-case loss `Δ(P)`를 differential evolution으로 최소화하여 base matrix 탐색(base 원소 최대 3).
9. 결과: 예제 base matrix들의 threshold가 넓은 rate 범위에서 Shannon limit 1dB 이내, `n=1200` FER 시뮬과 union bound(TUB)로 error-floor 예측 일치.
10. 한계: HW 미설계, 순수 부호설계·threshold 이론, biAWGN 통신 특화(NAND read/soft-quant 무관).
