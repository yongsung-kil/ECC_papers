### B. 알고리즘 요약 (10줄 내외)

1. AWGN 채널 대상, rate `R=(n-1)/n` CSOC(Massey, 1963)를 convolutional protograph로 취급해 permutation matrix로 lifting, high-rate SC-LDPC 코드를 구성.
2. 기존 SC-LDPC 구성(classical edge spreading)은 girth 6 보장을 위해 별도 search가 필요한 문제를 CSOC의 orthogonality 성질로 자동 해결하려 함.
3. 핵심 가정: CSOC의 parity-check matrix는 orthogonality 조건에 의해 girth≥6, `dfree=J+1` (J=orthogonal check 수)를 원천적으로 보장.
4. 핵심 기법: systematic CSOC는 degree-1 VN(파리티 열)이 성능을 저해하므로, degree-1 열을 제거한 non-systematic CSOC(`Hns`, 식 3~4)를 제안, `rate=(n-2)/(n-1)`로 재구성하되 RSC(`Gsys(D)`)로 systematic encoding 유지.
5. Theorem 1: lifting된 non-systematic CSOC protograph도 `girth≥6`, `dfree≥J+1`이 그대로 유지됨을 증명(4-cycle이 생기면 기저 protograph도 4-cycle을 가져야 하므로 모순).
6. 부수 기법: circulant(quasi-cyclic) lifting을 쓰면 구현 복잡도를 낮추면서도 성능 저하가 거의 없음(Code V vs VI 비교, girth 8 circulant lifting).
7. 디코딩: iterative BP 기반 sliding window decoding(SWD), 윈도우 크기 `W` constraint length, latency `η=W·M·n(m+1)`, `W=4`가 일반적으로 충분.
8. 결과(수치): Code I(R=13/14→Rt=0.915, J=3)과 Code II(R=10/11→Rt=0.880, J=5)가 BER 10⁻⁵에서 finite-length bound 대비 각각 0.55dB, 0.60dB 이내. CSOC 기반 코드가 classical edge spreading 대비 약 0.1dB 우수(동일 latency).
9. PEXIT 문턱값 분석: L=200에서는 classical이 CSOC보다 gap-to-capacity 0.16~0.32dB 낮지만, L=1000에서는 거의 동일해짐(threshold saturation과 유사한 대형-L 수렴).
10. 한계: 순수 AWGN 시뮬레이션 및 PEXIT 이론 분석뿐, HW 구현·게이트 설계·양자화 비용 분석 없음. NAND 특유 read 채널이나 error-floor 세부 측정도 없음.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| CSOC 기반 protograph lifting으로 girth≥6/dfree≥J+1 보장하는 H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | H-matrix 구조 설계 아이디어이나 Prime ECC는 SC-LDPC가 아닌 QC-LDPC block code로 구조가 다름, 이식 난이도 높음 |
| circulant(quasi-cyclic) lifting으로 구현 복잡도 절감 | `encoder.cpp` `Generate_PCM_encoding()` | Prime ECC도 QC 구조(circulant shift)이므로 아이디어 참고 가능 |
| BP 기반 sliding window decoding(SWD), latency η | `decoder.cpp` `LDPC_Decoder()` | SC-LDPC 전용 SWD 개념으로 block code인 Prime ECC 디코더 루프와 구조적 대응 낮음 |
```

적용 가치: `중간` — 부호설계 관점(girth/dfree 보장 구성)은 참고 가치가 있으나 SC-LDPC 특유 구조(convolutional, sliding window)라 QC-LDPC block code인 Prime ECC에 직접 이식은 어려움.
