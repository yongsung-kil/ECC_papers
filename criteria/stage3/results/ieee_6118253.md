### B. 알고리즘 요약

1. 대상: 이진 LDPC의 binary message passing(BMP) 디코더 — VN↔CN 간 1bit 메시지 교환(Gallager A/B 계열), 저메모리·저복잡도(광통신 등 고속용).
2. 문제: BMP는 복잡도 이득 대신 성능 손실 큼. 채널에서 hard 판정만 받는 기존 Gallager B의 성능 gap을 채널 soft 정보로 줄이려 함.
3. 모델: VN은 모든 입력을 L-value(LLR)로 변환해 L-domain에서 계산 후 hard decision → symmetry/isotropy 조건 자동 충족. BIAWGN 채널 가정.
4. 핵심기법: CN EXIT `ε_ec=fc(ε_ac;dc)=(1-(1-2ε_ac)^{dc-1})/2`(식3), VN EXIT는 채널/사전정보 신뢰도 `Dch,Dav`와 임계 `t`(식6,7)로 이항누적분포로 표현(식8).
5. 핵심 식 의미: EXIT가 binary message에 대해 근사가 아닌 exact(상호정보=메시지 밀도 정확기술) → 해석적 부호설계 가능.
6. 확장: 채널 출력을 `W+1`개 sub-BSC로 분해(식13~17), W=2(2bit, BSQC)면 관측 L-value가 `{±Dch,1,±Dch,2}` 4레벨 → Gallager B 대비 >1.0dB 이득.
7. 구현 트릭: L-domain 대신 majority decision rule로 대체(복잡도↓), 미충족 체크 비율 `s=Me/M`로 사전확률 채널을 온라인 추정(precomputed sequence 불필요).
8. 최적화: rate `R=1-(1/dc)Σλi/i` 최대화를 λi에 대한 선형계획(LP)으로 — stability 조건 `(λ2+2ε_ch λ3)(dc-1)<1`(식19)과 degree-2/3 사이클-free 조건 `3λ2+4λ3 ≤ 6(1-R)`(식24)을 선형 제약으로.
9. 결과(EXIT+BER 시뮬): 채널 1bit 추가로 rate0.5에서 3.67→2.69 dB, 광통신 rate0.9375(dv=7,dc=112, 40Gbps)에서도 ~1.0dB 개선; rate0.9 soft는 capacity 0.5dB 이내. dv<4 정칙부호는 shallow error-floor 불가피(정리2,3).
10. 한계: HW 미설계·시뮬/이론뿐; VN↔CN 메시지는 여전히 binary(1bit)라 완전 soft LLR 전파 대비 gap 존재.
