### B. 알고리즘 요약

1. 시스템: beyond-100Gb/s coherent 광전송, 16/64/256QAM 고차변조 + SD-FEC; LDPC-CC(convolutional) 기반 BICM(-ID).
2. 문제: 긴 블록 LDPC(N≈20000)는 NCG 11.3dB·error-floor 없음을 얻지만 **wire routing congestion**이 심함(wire수 ∝ N·q).
3. 모델: 광채널 SD-FEC, 양자화비트 `q`=3~4bit(QPSK 기준 floating-point 대비 0.2dB 이내).
4. 핵심기법: LDPC block 대신 **LDPC-CC**로 짧은 constraint length(10032)에서 동등 이득 → N 절반으로 routing 완화.
5. 의미: bit-serial/stochastic 복호는 wire를 줄이나 throughput·성능 손실(>0.2dB)이라 부적합 → LDPC-CC가 N 축소의 실용해.
6. 확장: LDPC-CC에 **layered decoding** 적용, message-passing 대비 절반 iteration으로 동일성능 → 면적·latency 절반.
7. 코디드모듈레이션: LDPC-CC 내장 인터리빙으로 별도 bit interleaver 생략, Gray/Gray-like 매핑; BICM(-ID)에 IPQ/NU-QAM/U-QAM shaped constellation 적용.
8. 최적화: irregular degree distribution을 EXIT chart로 설계(BICM-ID용); 단 irregular는 error-floor로 outer code 필요.
9. 결과: 37.14% overhead regular LDPC-CC(N=9262, rate-0.75 shortening), 13 iteration; FPGA에서 12iter layered로 NCG 11.5dB @ BER 10⁻¹⁵, 256U-QAM BICM은 NCG 14.7dB.
10. 한계: HW 미상세(FPGA emulation 수준), 광통신 고차QAM 특화, demapper FLOPb가 k에 지수증가라 4-D/BICM-ID 실용성 제한.
