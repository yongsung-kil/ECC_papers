### B. 알고리즘 요약

1. 대상은 lifted-product QLDPC 부호 `[[1054, 124, 20]]` (classical (155,62) Tanner code 기반, circulant size `l=31`), depolarizing 채널.
2. 문제: BP/min-sum이 stabilizer-induced trapping set(특히 symmetric stabilizer TS) 때문에 QLDPC에서 error floor가 높음.
3. 모델: memoryless Pauli(depolarizing) 채널, syndrome-based decoding — `σ = xHx^T`, `x⊕x̂ ∈ rowspace(Hz)`면 성공(degenerate error 허용).
4. 핵심기법: edge type별로 서로 다른 VN update 규칙 부여. CN은 min-sum 공통(식(3)), VN은 `m_{v→c}=b_i + w·Σ m_{c→v}` (식(4))로 edge-type bias `b_i` 사용.
5. 식 의미: `b_i=log((1-p)/p)`, `w=1`이면 표준 min-sum 복원 — bias만 조정해 TS 대칭 파괴.
6. 확장: decoder diversity — 여러 decoder `D_k`(각기 다른 bias 벡터 `b_k`)를 sequential/parallel로 운용, 수렴하는 첫 디코더 채택.
7. 구현 디테일: RNN 3 hidden layer, depth `I_k`(iteration 수), activation을 min-sum update 형태로 고정 — iteration 간 규칙 재사용(recurrent) → HW 친화.
8. 학습: `D_0`=min-sum 실패 error 집합 `P_w`(expansion-contraction으로 획득)를 훈련셋으로, degenerate 고려한 loss `L=Σ sin(0.5π σ̂x)` 사용. QC 대칭으로 훈련셋/파라미터를 `l`배 축소.
9. 결과: weight-5 uncorrectable가 `D0` 136811 → `D4` 6으로 감소, 100 iteration에서 normalized min-sum 대비 logical error rate 개선.
10. 한계: 양자 QLDPC 전용(depolarizing/syndrome), HW 미설계, gate/throughput 수치 없음, classical NAND LDPC와 직접 연결 안 됨.
