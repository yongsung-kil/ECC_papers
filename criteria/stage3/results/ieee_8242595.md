### B. 알고리즘 요약

1. 대상: binary LDPC(N=204/504/1008, rate 0.5 regular) + BPSK, PR 자기기록 ISI 채널(PR4/EPR4/modified-EPR4).
2. 문제: ADMM LP 복호는 BSC/AWGN 무기억채널에서만 검증됨, ISI 채널에서의 동작 미지.
3. 채널모델: `r_j = Σ_d h_d(1-2x_{j-d}) + n_j` (식7), PR 전달다항식 `h(D)=Σ h_i D^i` (식8), n_j는 i.i.d. 가우시안.
4. 핵심기법: turbo equalization — BCJR equalizer와 ADMM 복호기를 outer loop로 교대, extrinsic 정보 교환(Fig.2b).
5. 핵심식: ML을 LP로 완화 `min γᵀx s.t. x∈Ω=∩(C_j)` (식2), replica 변수로 재정식화(식3) 후 check polytope P_{d_j}로의 Euclidean projection.
6. ADMM 반복: x-update(식4, [0,1] clamp), z-update(식5, check polytope projection), dual y-update(식6), penalty μ=7.0.
7. 구현 디테일: ADMM 출력 x∈[0,1]을 확률로 보고 `L_post=log((1-x)/x)`(식12)로 LLR 변환해 BCJR과 정보교환, l1 penalized decoder(param 0.25), over-relaxation 1.9.
8. 스케줄: outer O_max=5/10, inner I_max=10, 조기종료 `Hx^T=0`; inner 10회에서 BER 수렴, outer>5부터 개선 미미.
9. 결과: ADMM이 BP 대비 BER=1e-5에서 modified-EPR4 0.5dB / PR4 0.4dB / EPR4 0.1dB 이득, interior-point 대비 크게 우수.
10. 한계: HW 미설계, 시뮬만, BCJR 직렬 detector 의존, 소~중 길이 부호, ADMM check-polytope projection HW 비용 미평가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| ADMM 복호 반복(x/z/y update) | [decoder.cpp](../Prime_ECC_3.1_Claude) `LDPC_Decoder()` | 복호기 위치는 같으나 LP 최적화 프레임, min-sum 이터레이션과 알고리즘 상이 |
| check polytope Euclidean projection | 대응 없음 | Prime의 CN min-sum(`CNU_Update_New_Mag()`)과 근본적으로 다른 연산 |
| BCJR turbo equalization | 대응 없음 | Prime은 채널 등화기(BCJR) 미보유, AWGN/RBER 직접 LLR |
| PR/ISI 채널 모델 | 대응 없음 | Prime channel.cpp는 AWGN/RBER/fixed-error, ISI 다중경로 무관 |

적용 가치: **낮음** — ADMM/LP 복호는 Prime의 min-sum HW 근사와 별개 패러다임이고 채널(PR/ISI)·부호(소길이 rate-0.5)도 NAND 고rate QC-LDPC와 어긋나 이식 실익 없음.
