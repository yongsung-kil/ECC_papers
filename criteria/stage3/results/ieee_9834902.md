### B. 알고리즘 요약

1. 대상은 Tanner 코드 `T(G,{C1..Cm})`: 이분그래프 `G=(L∪R,E)`의 각 체크노드 `j`에 내부코드 `Cj`를 부여, 이웃 제약 `c|N(j)∈Cj`를 만족하는 부호.
2. 풀려는 문제: 지역복구부호(LRC)는 단일 소거는 국소 복구되나 다중 소거 시 전체 심볼이 필요 → 계층적 지역복구(HLRC)로 소거 개수에 따라 복구 계층(tier)을 나눠 접근 심볼 수를 완화.
3. 모델: 소거채널(erasure) + 그래프기반 message-passing(peeling) 디코더, 유한체 `Fq`(비바이너리 허용).
4. 핵심기법1: 내부코드를 LRC로 선택하면 Tanner 코드가 girth `g(G)` 조건(=4 vs ≥6)에 따라 지역성 `r`과 가용성 `a`를 상속(Thm 1,2)함을 증명.
5. 핵심식: `s` 소거 시 복구셋 크기가 `r = s·max{rij}`로 s배 증가(girth≥6) → 다중소거에서 단순 LRC의 한계를 정량화.
6. 핵심기법2: Tanner 코드를 2~3레벨 HLRC로 구성하고 각 레벨의 지역성 파라미터 `(t̃_k, ρ̃_k)`와 가용성 `τ̃_k`를 유도(Thm 3,4).
7. 구현 디테일: 그래프에 4-cycle이 없을 때(girth≥6) 이웃 복구그룹이 서로소가 되어 가용성이 유의하게 증가.
8. 학습/최적화 절차: 없음(조합론적 증명).
9. 결과: 계층별 stopping set 정의(Def 4,5)와 크기 경계 `min ρi ≤ smin(G3) ≤ smin(G2) = min dmin(Ci) ≤ smin(G1)`(Thm 5) 제시 → 레벨 인덱스를 낮춰도 정정 가능 소거 수가 줄지 않음.
10. 한계: 순수 이론(HW·시뮬 없음), 소거채널·분산저장 문맥, 구체적 rate/length·정정성능 수치 없음.
