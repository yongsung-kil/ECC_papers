### B. 알고리즘 요약

1. 시스템: protograph LDPC(ARJA) + square-constellation M-ary DCSK를 1회 비트인터리빙으로 결합한 DC-BICM, band-limited 전송용, 3-path Rayleigh fading 채널.
2. 문제: DCSK는 참조샘플 전송으로 data rate·대역효율 낮고, 채널부호 redundancy가 band-limited 전송에 불리해 coding gain을 대역손실 없이 얻어야 함.
3. 모델: 수신 `r(t)=Σαl·δ(t-τl)⊗s(t)+η(t)`, Rayleigh 계수 PDF, ISI 무시(`τmax≤β`) 가정 하 결정변수 `za,zb`는 Gauss.
4. 핵심기법: ARJA protograph에서 non-punctured VN 수 `ñ=log2 M`, 코드비트를 constellation labeling에 water-filling(비트평면 MI 비례 VN degree 매핑)으로 배정하는 VDMM.
5. VDMM은 고차 VN이 더 많은 노드에 영향을 주므로 MI가 큰 비트평면에 고차 VN을 배치(최적은 아니나 유효).
6. 카오스 참조 `xk+1=1-2xk²` 생성 후 Hilbert 변환으로 직교 신호쌍 `cx,cy` 확보, `ms=as·cx+bs·cy`로 심볼 형성.
7. 부분 CSI만 사용: 파일럿(1,0)으로 `h=Σαl²` 채널전력만 LS 추정, LLR은 Gaussian approximation `LLR[i]=log Σp(z|sm)`로 계산.
8. 분석: P-EXIT로 protograph별 디코딩 threshold 도출(최대 100 iter), demod LLR이 β 클 때 `N(u,2u)` Gaussian 근사.
9. 결과: 동일 스펙효율(2bit/symbol)에서 uncoded 4-ary DCSK 대비 약 12.5dB coding gain(BER 1e-4, β=64), code length↑·iter↑로 개선(50/100 iter 차 작음), M↑·β 최적점 존재.
10. 한계: HW 미설계, 시뮬만, gate/throughput 없음, 카오스 DCSK·multipath Rayleigh 무선 특화, sim-threshold gap 큼(추가 최적화 여지).
