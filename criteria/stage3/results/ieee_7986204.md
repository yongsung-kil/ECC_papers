### B. 알고리즘 요약

1. 시스템: 방송(ATSC 3.0) 시그널링 보호용 저rate raptor-like QC-LDPC, rate `1/5`, length `16200`.
2. 문제: 시그널링은 수신기가 프레임 복원에 필수인데 저SNR에서 복호 실패 시 프레임 손실. 시그널링 비트 대부분(약 80%)은 오래 안 바뀌는 fixed bits라는 특성 미활용.
3. 모델: 코드워드를 `y = [changeable bits, fixed bits]`로 구분. fixed information bits + fixed parity bits = fixed bits.
4. 핵심기법(중첩복호): 첫 프레임에서 fixed bits를 정확히 복호하면 다음 프레임들의 changeable bits 복호 시 known-bit으로 사용. 저SNR에서는 여러 연속 프레임의 fixed bits를 누적 중첩(overlap)해 복호.
5. 식 의미: 검사식 `H*y=0`을 `A*y_i + C*y_pc = 0`, `B*(y_i,y_f) + D*y_pd = 0`으로 분할해 fixed parity bit 판정.
6. 확장(선택 알고리즘): fixed parity bit 수를 최대화하는 fixed information bit 위치조합 `s_j`를 base matrix 기준으로 탐색(algorithm1).
7. 구현 디테일: raptor-like QC-LDPC의 circulant/zero 블록 구조 덕에 mother/base matrix 단위로 계산해 복잡도 축소.
8. 최적화 절차: 모든 위치조합에 대해 fixed parity bit 개수 `n_j` 카운트 후 최대값 조합 선택.
9. 결과: 양호채널서 다음 프레임에 약 2.3dB 이득, 악조건서 첫 프레임 희생으로 threshold 1dB(2프레임 희생 시 1.6dB) 하강. 선택 알고리즘은 랜덤 대비 0.4~0.5dB 추가 하강.
10. 한계: HW 미설계, 시뮬만, 프레임간 fixed-bit 불변이라는 방송 시그널링 특수 가정에 전면 의존.
