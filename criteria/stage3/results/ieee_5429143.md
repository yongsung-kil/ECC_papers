### B. 알고리즘 요약

1. 대상은 MBIOS 채널에서 ML 복호 하 "정보비트당 에지 수"(그래프복잡도)를 유계로 유지하며 용량을 달성하는 부호족 존재성 문제.
2. 기존 정규 LDPC는 용량 접근 시 그래프복잡도가 무한대로 발산([10])하는 한계 → 유계 복잡도 부호 필요.
3. 먼저 IRA 앙상블의 AAWD(점근 평균 무게분포)를 유도하고 Divsalar 한계로 BIAWGN 성능평가.
4. nonsystematic regular IRA가 같은 그래프복잡도의 systematic IRA·정규 LDPC보다 우수, Shannon 한계 대비 최대 `0.124 dB`.
5. 그러나 IRA는 최소거리가 선형 증가하지 않아([40]) 용량달성 엄밀 증명 불가.
6. 이를 우회해 LDPC-GM 부호(외부 Gallager LDPC + 내부 rate-1 정규 LDGM) 직렬연접을 제안.
7. 핵심 식: rate-1 LDGM 내부부호가 외부 LDPC의 고무게 부호어를 제거하면서 저무게 부호어를 소멸시켜 AAWD를 random 앙상블 상한으로 유계화.
8. state node 도입이 유계 그래프복잡도 용량달성의 열쇠(정규 LDPC엔 state node 없음).
9. 결과: `k`(내부 LDGM 반복도)가 유한 임계값 이상이면 임의 rate에서 ML 용량달성 + Gilbert-Varshamov 한계 달성; BEC에서는 BP로도 유계복잡도 용량달성(Th.4/5, 펑처링 확장 Th.6).
10. 한계: MBIOS 채널에서 BP 복호 복잡도 유계성은 미해결(DE 해석 불가), ML 복호 자체는 지수복잡도, HW·유한길이 설계 없음.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LDPC-GM 부호구조(외부 LDPC+내부 LDGM 연접) | 대응 없음 | Prime ECC는 고정 binary QC-LDPC(dual-diagonal 인코딩)로 LDGM 연접구조 미대응 |
| 그래프복잡도/AAWD 이론분석 | 대응 없음 | 순수 점근 부호이론, 코드 대응 없음 |
| ML 복호 성능한계 | 대응 없음 | Prime ECC는 min-sum BP만 사용, ML 복호 없음 |
```

적용 가치: 낮음 — MBIOS 용량달성 존재성 증명에 그친 이론이라 NAND binary QC-LDPC 디코더/HW로 떼어낼 실용 요소가 없음.
