### B. 알고리즘 요약

1. 시스템: BPSK, `N_T`(1 또는 2)송신·1수신, time-varying fading(Jakes 모델, Doppler `f_D`). SC(주파수 non-selective)·OFDM(frequency selective, L=8 multipath) 두 전송모드.
2. 문제: LDPC를 다이버시티와 단순 연결한 기존연구는 LDPC 고유구조(SPC+REP 결합)를 활용 못함. REP를 다이버시티로 재해석해 결합 필요.
3. 모델: LDPC를 SPC+REP+랜덤 인터리버 결합으로 보고, REP를 half-rate invertible code로 대체해 정보블록 `u`와 패리티블록 `b`(`b=f(u)`, 유일역 `u=f⁻¹(b)`)가 모두 다이버시티+코딩 이득 제공.
4. 핵심기법1: PCTD 인코딩 — 다이버시티차수 `Ω`(2/3). Ω=3은 RC(재부호화 `c=(u,b,d)`)·DC(복제 `c=(u,u,b)`) 두 방식. Space-time형은 Alamouti/STBC와 결합.
5. 식 의미: half-rate invertible SP-QC-LDPC를 GF(2⁷) base matrix + masking + CPM-dispersion으로 구성 → `(3,6)-regular (1016,508)` rate 0.5 부호(Example 1).
6. 핵심기법2: CC-MPA — EGC로 `u`·`b` LLR을 상호 결합(식22~25), 시스템형을 RC-constraint 비시스템형 `v_ns`로 변환 후 SPA 복호. Ω=3 RC는 `b`를 공유 VN으로 하는 cross-iteration(CI) MPA(Alg.2).
7. 트릭: 증가 복잡도는 초기 LLR 갱신에만 의존, CN/VN 각각 `O(η_u²·N)`·`O(ρ_u·N)`로 부호길이 선형(η_u,ρ_u=1~6).
8. 이론: fast fading BPSK ergodic capacity(식28~32)로 Shannon limit 유도 — Rc=1/Ω일 때 `(Eb/N0)_min` (fading 1.831dB vs AWGN 0.188dB @R=0.5).
9. 결과: SC non-selective, fDTs=0.1, BER 10⁻³서 REP 대비 8.7dB(Ω=2)·최대 10.9dB(Ω=3 DC) 이득, SPA 대비 0.7dB↑, Shannon limit와 3.2dB gap. OFDM은 threshold 12dB 초과서 waterfall.
10. 한계: HW 미설계·시뮬만, 무선 fading/다이버시티 전제. half-rate·invertible 제약, DC가 RC보다 우수(채널 다이버시티 이득 > 반복 코딩 이득).

```

```
