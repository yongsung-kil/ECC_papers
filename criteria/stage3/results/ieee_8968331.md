### B. 알고리즘 요약

1. 문제 설정: `k`-sparse 비음수 실수 신호 `x ∈ R^n`을 소수의 선형측정 `y = Ax`(`A`는 `m×n` measurement matrix, `m<n`)로부터 복원하는 compressed sensing 문제, IPA(interval-passing algorithm)는 저복잡도 반복 재구성 알고리즘.
2. 기존 IPA 실패조건은 LDPC 디코딩의 stopping set이 실패조건의 부분집합임만 알려져 있었고, 완전한 실패조건 특성화는 미해결.
3. 핵심 가정: 측정행렬 `A`가 nonnegative 실수 행렬이며, variable node(신호 성분)와 measurement node(측정치) 사이 bipartite Tanner graph로 표현 - IPA는 각 노드에 상/하한 구간(`μ`, `M`) 메시지를 반복 전달.
4. 핵심 기법: 먼저 IPA가 `x`(실수)를 복원 실패하는 것은 동일 support의 이진벡터 `z`를 복원 실패하는 것과 동치임을 증명(Theorem 1), 이로써 이진 measurement matrix 분석으로 문제를 축소.
5. 핵심 정의: "termatiko set" `T` — `IPA(Ax_T, A)=0`이 되는 support(즉 완전복원 실패의 필요충분조건 집합). Theorem 2는 `T`와 그 이웃 measurement node 집합 `N`, 부가 변수집합 `S`에 대한 그래프이론적 조건으로 termatiko set을 정확히 특징화.
6. 확장/부수 기법: column-regular(열 degree `a`) measurement matrix가 4-cycle이 없으면 termatiko distance(최소 termatiko set 크기) `≥ a`(Theorem 5); array LDPC 부호 `H(q,a)`(permutation-matrix 기반 QC 유사 구조)에 대해 termatiko distance 상한(최소거리의 절반)과 column weight 3일 때 정확한 값·다중도를 유도.
7. 부수 알고리즘: 최소가중치 codeword의 support를 두 개의 절반 크기 termatiko set으로 분할하는 greedy 2-coloring 휴리스틱(Algorithm 2), redundant row(측정치 선형결합) 추가로 termatiko set을 제거하는 정수계획법 기반 탐색(Section V).
8. 학습/최적화 절차: 정수선형계획(ILP)으로 redundant row 후보를 생성하고, termatiko set 제거 개수를 score로 하는 greedy 알고리즘으로 반복 채택.
9. 결과: protograph 기반 (3,6)-regular LDPC 앙상블(200개 행렬, lifting 100)에서 평균 최소거리 6.84 대비 평균 termatiko distance 3.90; redundant row 추가로 일부 measurement matrix의 termatiko distance 증가 및 FER 개선 확인(시뮬).
10. 한계: 채널코딩 디코더가 아닌 compressed sensing 신호복원 문제(전송된 codeword의 오류정정이 아닌 sparse 신호 재구성)이며, HW 설계·양자화·throughput 등 구현 관점 전무, IPA 자체가 sum-product/min-sum과 무관한 별도 알고리즘.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| IPA(interval-passing algorithm) | 대응 없음 | Min-Sum/BP 기반 LDPC 디코더가 아닌 compressed sensing 전용 별도 반복 알고리즘 |
| termatiko set(실패집합) 그래프이론 분석 | 대응 없음 | trapping-set 개념과 형식적으로 유사하나 대상이 codeword 오류정정이 아닌 신호 support 복원이라 `decoder.cpp`의 min-sum 알고리즘과 직접 대응 불가 |
| array LDPC parity-check matrix 구조 | `ecc_top.cpp` `Load_PCM()` (간접) | array LDPC(H(q,a))는 permutation-matrix 기반 QC-LDPC 유사 구조이나 본 논문에서는 measurement matrix 용도로만 사용되어 부호설계 관점 참고가치 낮음 |

적용 가치: `낮음` — compressed sensing 신호복원 알고리즘(IPA)의 실패집합 이론으로, NAND binary LDPC ECC의 디코더·부호설계·HW 어느 레이어와도 직접 접점이 없음.
