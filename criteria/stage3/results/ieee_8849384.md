### B. 알고리즘 요약

1. 기본 부호는 rate 1/2 (3,6)-regular binary LDPC-BC `C[n,k]`(PEG로 구성, length 1024~2400), AWGN·BPSK/16-QAM 전송.
2. 문제: 기존 SC-LDPC 구성은 성능은 좋으나 인코더/디코더 HW가 기저 LDPC-BC HW와 느슨하게 연결돼 실제 구현이 불편.
3. 핵심 아이디어: LDPC-BC 부호어 `v(t)=u(t)G`를 block Markov superposition transmission(BMST) 방식으로 encoding memory=1로 결합.
4. Partial superposition: selection matrix `S`(랜덤 순열 permutation의 `n(1-α)` 열을 0으로) 사용, `c(t)=v(t)+v(t-1)S`. `α`는 superposition fraction(최적화 대상).
5. `α<1`(부분 결합)이 catastrophic error propagation을 완화 — 기존 BMST(full superposition, α=1) 대비 핵심 차별점.
6. 대수적 표현: 생성행렬 `G_SC`는 upper-banded block, 패리티행렬 `H_SC`는 lower block-triangular → BMST-LDPC가 SC-LDPC의 특수 클래스임을 보임.
7. 디코딩: sliding-window(SW) 반복 message passing, 각 layer에 `+/=/S/LDPC` 노드, 내부 `LDPC` 노드는 기존 SISO SPA(Imax=20) 재사용, forward/backward recursion + successive cancellation.
8. BER 분석: genie-aided(GA) bound — 시뮬 GA(L=1)와 추정 GA `f_BMST(γ) ≥ f_basic(γ+10log10(1+α))`(기본 부호 곡선을 좌로 shift)로 예측.
9. 결과: length 1024, α=0.3에서 LDPC-BC 대비 BER 1e-5에서 0.7dB coding gain; 16-QAM에서 window d=2/3/4로 BICM-LDPC 대비 0.5/0.6/0.7dB gain.
10. 한계: HW 미설계(시뮬만), decoding latency `dn`으로 증가, AWGN/무선 채널 가정, NAND read/soft 양자화 미고려.

```
|---|---|---|
| BMST superposition 인코딩 (G_SC, selection matrix S) | encoder.cpp LDPC_encoder_4KB() | 대응은 인코더지만 dual-diagonal 고정 구조라 superposition 결합 이식 부담 큼 |
| sliding-window SISO 반복 복호 (LDPC 노드=기존 SPA) | decoder.cpp LDPC_Decoder() | 내부 컴포넌트 디코더는 유사하나 우리는 min-sum, 논문은 SPA + SW 스케줄이라 정합 낮음 |
| 부호구성 (H_SC, SC-LDPC 결합) | ecc_top.cpp Load_PCM() | Prime ECC는 QC-LDPC 블록 PCM 고정, spatial-coupling 구조 미지원 |
```
