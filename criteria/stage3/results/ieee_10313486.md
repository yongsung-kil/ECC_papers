### B. 알고리즘 요약

1. 대상: Multi-Edge Type(MET) LDPC 앙상블 `C^{ν,μ}_n` (VN/CN/edge 타입 다수). 예제는 rate `1/2`, `nv=nc=ne=2` 앙상블.
2. 문제: iterative 복호의 error-floor는 Tanner graph의 trapping set(TS)/absorbing set(AS)/fully absorbing set(FAS)에 좌우되므로, 이들의 개수 분포를 앙상블 수준에서 세는 enumerator가 필요.
3. 정의: `(a,b)` TS는 a개 VN 집합 I에서 홀수번 연결된 CN(unsatisfied)이 b개; AS/FAS는 각 VN이 U(I)보다 S(I) 이웃 CN이 더 많다는 추가조건 (elementary는 CN 연결수 제한).
4. 핵심 기법: generating function(더미변수 `x,y` 및 `s,t`)으로 CN/VN 배치의 경우의 수를 다항식 계수(`coeff`)로 표현, VN weight vector `ε`와 edge weight vector `w`로 유한길이 평균 개수(Lemma 4~6) 유도.
5. 핵심 식: 점근 분포 `G(α,β)=lim (1/n) ln E(αn,βn)`를 이항엔트로피 `H(p)`와 다변수 계수 점근공식(Lemma 3)로 표현.
6. 확장: Theorem 1/2/3에서 TS/AS/FAS의 점근 `G_TS/G_AS/G_FAS`를 `ne+nv+2`개 미지수 방정식계의 양의 해로 계산; elementary 변형은 `A_i` 생성함수만 치환.
7. 부수: relative minimum size `α*_TS/α*_AS/α*_FAS`는 정규화 로그분포의 두 번째 영점(`G(α,Δα)=0`)으로 정의, 방정식 하나 추가로 구함.
8. 특수화: 모든 edge 동일타입이면 unstructured LDPC enumerator, protograph 각 VN/CN/edge를 타입으로 보면 protograph enumerator를 재현.
9. 결과: rate 1/2 예제 앙상블에서 `Δ=b/a∈{0.005~0.4}`에 대해 정규화 로그 점근분포 수치평가(Fig.2/3), α가 작으면 TS/AS/FAS 격차 소멸, α·Δ 증가 시 격차 확대.
10. 한계: 순수 이론(부호 구성·디코더·HW·BER 실험 전무), non-binary도 다루는 일반이론이라 NAND binary QC-LDPC 특화 산출물 없음.

### C. Prime ECC 관련 모듈 핀포인트

|---|---|---|

| MET/protograph 앙상블 이론 | [ecc_top.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `Load_PCM()` (H-matrix) | 부호설계 계층에 개념적 접점이나 고정 QC-LDPC라 재검증 부담 큼 (높은 난이도) |
| 수렴실패/trapping-set 추적 | [corner.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) | 우리는 실행 중 수렴실패 추적, 논문은 앙상블 평균 이론 - 계층 상이 |

적용 가치: 낮음 - error-floor 분석 이론으로 우리 부호를 MET 앙상블로 사상하는 대규모 재작업이 선행돼야 하고, 직접 이식할 부호/디코더/HW 산출물이 없음.
