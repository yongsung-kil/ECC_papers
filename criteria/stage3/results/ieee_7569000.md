### B. 알고리즘 요약

1. 시스템: 1-D 자기기록(MR) 채널, non-binary GF(4) CB(circulant-based) LDPC 기반 SC 코드, rate 0.83~0.87, 길이 1058~8410 bit.
2. 문제: SC 코드의 asymptotic 분석은 유한길이·유한cycle 실제 설계에 직접 적용 불가, 특히 MR 채널의 error floor를 지배하는 구조가 AWGN과 다름.
3. 모델: MR 채널은 ISI+jitter+전자잡음, PR target `[8 14 2]`, density 1.4, BCJR detector와 q-ary FFT-SPA LDPC decoder 간 global iteration(10)·local iteration(20).
4. 핵심기법1: parity-check matrix `H`를 cutting vector `ζ=[ζ0..ζγ-1]`로 `H0/H1`으로 partition 후 `L`개 복제를 staircase로 coupling해 `HSC` 생성.
5. 핵심식: region 길이 `πn`과 search matrix `An,β`(β=a(γ-1)/2+1)로 반복구조 활용, 전체 그래프 대신 소수의 sub-matrix만 탐색(복잡도 L 무관).
6. 핵심기법2: MR에서 dominant error는 balanced AS(BAS); γ=3은 (6,0)/(8,0)/(10,0), γ=4는 (8,2)/(8,4)/(10,0)/(12,0)이 지배적(AWGN과 상이).
7. 트릭: 이들 BAS의 "common denominator" substructure를 γ=3은 (4,4) AS, γ=4는 (4,8) TS로 식별해 이 하나만 최소화(둘 다 cycle-8).
8. 최적화: Algorithm 1로 모든 valid cutting vector를 탐색해 cycle-8 최소 `ζopt`를 찾고, non-binary는 edge weight 최적화([15][20])를 추가 적용.
9. 결과: 최적 cutting vector가 degenerate 대비 common denominator 약 60% 감소, error floor에서 block 대비 2 orders 이상, unopt SC 대비 최대 3.5 orders 개선.
10. 한계: HW 미설계·시뮬만, non-binary GF(4)·MR 채널 특화(binary/NAND 아님).
