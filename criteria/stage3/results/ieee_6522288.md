### B. 알고리즘 요약

1. 시스템: BPMR/TDMR 고밀도 자기기록의 2-D ISI 채널(1-D ISI + intertrack interference)에서 동작하는 rate 8/9 irregular LDPC, code length 13824.
2. 문제: AWGN 최적화 LDPC는 2-D ISI + 등화 환경에서 성능 저하 → 2-D ISI 채널 전용 capacity-approaching 부호가 필요(기존 미해결).
3. 모델: 이진 데이터 2-D 배열이 채널응답행렬 `H`(1/4 Tb/in² 밀도 예시)와 AWGN으로 왜곡, 목표는 SIR(symmetric information rate) 근접.
4. 핵심기법: 수정 EXIT chart 기반 차수최적화. 통상과 반대로 2-D-DET+VND 결합 EXIT 곡선을 먼저 구하고, CND EXIT 곡선을 여기에 정합.
5. 핵심식: CND EXIT는 duality theorem으로 근사(식 6~8), 2-D-DET+VND는 식 10~11로 혼합 차수분포 곡선 산출.
6. 확장: 최적화는 두 EXIT 곡선이 교차하지 않으면서 최대한 밀착하도록 하는 선형계획(LP) 문제(식 12)로 정식화.
7. 구현 디테일: 2-D-DET EXIT 곡선은 폐형이 없어 Monte Carlo로 산출, 3차 다항식으로 근사해 식에 대입.
8. 검출기: symbol-based BCJR 2-D 검출기와 BP LDPC 디코더가 반복(2-D-DET↔LDPC 10회, LDPC 내부 30회) 정보 교환.
9. 결과: 4 Tb/in²에서 BER 1e-5를 SIR 대비 0.33 dB 이내 달성, AWGN 최적화 부호 대비 약 0.2 dB 우수(1 Tb/in²는 0.1 dB).
10. 한계: 순수 시뮬레이션, HW 미설계, 자기기록 2-D ISI 채널 전용이라 NAND AWGN/RBER로 직접 이식 불가.
