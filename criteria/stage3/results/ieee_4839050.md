### B. 알고리즘 요약

1. 대상은 column-weight-3, `(3,γ)`-regular LDPC 코드 앙상블, Gallager A 알고리즘(및 이와 동치인 bit-flipping)으로 BSC 상에서 복호.
2. 기존 expander-graph 기반 분석(Burshtein-Miller 등)은 variable node degree ≥6 필요, degree 3(column-weight-3)에서는 상수 오류분율 정정 가능성이 미해결이었음.
3. all-zero codeword 전송 가정 하에 trapping set, inducing set, fixed set, fixed point 개념으로 복호 실패를 특성화.
4. 핵심 정리(Theorem 1): variable node 집합 `S`가 유도 부분그래프에서 (a) 모든 variable node가 `S` 내 홀수차수 check node에 2개 이상 연결, (b) `S` 밖 공통 variable node에 연결된 `S`내 홀수차수 check node 쌍이 없음 을 만족하면 `S`는 fixed set(=inducing/trapping set).
5. Girth `g`인 코드는 최단 사이클을 이루는 `g/2`개 variable node가 크기 `g/2`의 fixed/inducing set이 됨(Lemma 5) → 이 코드는 `⌈g/4⌉`개 초과 오류를 항상 정정하지 못함.
6. 메인 정리(Theorem 2): 표준 `(3,γ)`-regular 앙상블에서 임의 `ε>0`에 대해 girth 상한 `O(log n)`이 성립하므로, 충분히 큰 블록길이 `n`에서는 어떤 코드도 `εn` 이하의 모든 오류패턴을 정정할 수 없음.
7. 결과는 parallel/serial bit-flipping 알고리즘에도 동일하게 확장됨(Theorem 3, Corollary 2: Gallager A의 fixed set은 bit-flipping에도 fixed set).
8. 최적화/학습 절차 없음 — 순수 조합론적 증명(귀납적 girth-4/6/8 케이스 분석, Appendix I/II).
9. 결과는 수치 실험이 아닌 점근적 한계 증명뿐; 구체 gate count/BER 수치 없음.
10. 한계: 완전 이론 증명이며 시뮬레이션·HW·특정 부호 구성 실험이 전혀 없고, soft-decision이나 실제 NAND 채널 모델도 다루지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Gallager A / bit-flipping 오류정정 한계 증명 | 대응 없음 | Prime ECC는 min-sum 기반 soft-decision 위주이며 hard bit-flipping 알고리즘 자체를 사용하지 않음 |
| girth-기반 trapping set 이론 | 대응 없음 | 코드 구성 시 girth 고려는 `ecc_top.cpp` `Load_PCM()`이 로드하는 고정 H-matrix 설계 단계 문제이나, 본 논문은 구성법이 아닌 이론적 불가능성 증명 |

적용 가치: `낮음` — column-weight-3(즉 매우 낮은 rate/저밀도) 정규 LDPC의 점근적 한계를 증명하는 순수 이론 논문으로, Prime ECC의 고rate QC-LDPC 설계나 디코더 구현에 직접 이식할 알고리즘/HW 요소가 없음.
