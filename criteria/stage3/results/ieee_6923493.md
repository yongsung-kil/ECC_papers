### B. 알고리즘 요약

1. 대상 부호는 IEEE 802.15.3c HSI 모드의 4종 irregular QC-LDPC-BC(길이 672, rate 1/2·5/8·3/4·7/8), rate 7/8 base matrix에서 파생, 32개 그룹 21×21 circulant.
2. 문제: 완전병렬 LDPC 디코더의 실질 병목은 게이트가 아니라 CNU↔VNU 메시지 교환 배선 congestion이며, 다중rate 지원 시 더 복잡해짐.
3. 채널/모델: BPSK/16-QAM over AWGN, 채널값을 LLR→확률→stochastic bit stream으로 변환(RNG+comparator).
4. 핵심기법: stochastic 복호에서 CNU=XOR, VNU=flip-flop+control로 구현, 1-bit 메시지 교환(DC 단위), counter로 통계 수집 후 hard decision.
5. 핵심식: `P=∑ b_i·2^{-i}` 형태로 RCA-TFM 각 자리에 base-2 가중치를 부여해 regenerative bit 확률 추정.
6. 확장1: 재구성형 32-input CNU가 concealed syndrome checking 내장, degree-32를 4개 8-input XOR로 분해해 (672,336)/(672,588) 공유.
7. 확장2: 재구성형 VNU가 degree-2/3/4를 2-input sub-VNU 캐스케이드로 처리, latching 문제는 RCA-TFM 기반 PTU로 완화.
8. 부수트릭: RCA-TFM 값을 0.5로 고정해 adder/mux 제거, layer re-permutation+input reallocation으로 V-to-C mux 완전 제거·C-to-V mux 축소.
9. 결과: 90nm 760.3K gate, 768MHz에서 7.92Gb/s, 437.2mW(55.2pJ/bit), routing wire [7] 대비 90.7% 절감, 90% chip utilization.
10. 한계: HW 검증은 실측이나 stochastic 복호가 min-sum 대비 저SNR·고차변조에서 성능 손실, decoding cycle 최대 1000으로 latency 큼.
