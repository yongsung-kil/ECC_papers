### B. 알고리즘 요약

1. 시스템: HDD/TDMR 자기기록, codeword `N`을 `L`개 부분으로 나눠 `L`개 partial-response(PR) 채널로 BPSK 전송, 채널별 BCJR 검출기 출력을 결합해 LDPC 복호. 예: rate 8/9, length 4608, PEG로 구성.
2. 문제: 서로 다른(unequal) ISI target 채널에 대한 LDPC 부호설계가 문헌에 부재, 전체 codeword를 고오류 채널로 보내는 것을 회피하고자 함.
3. 채널 모델: `y_{l,k}=Σ h_{l,j} x_{l,k-j}+n_{l,k}`, 채널별 `SNR_l`과 인접 채널 SNR 차 `Δ=SNR_{l+1}-SNR_l` 정의.
4. 핵심기법: variable node degree 분포 `λ(x)`를 `L`개 부분별 `λ_{li}`로 쪼개, 각 부분이 담당하는 채널에 맞춰 개별 최적화(수정 EXIT chart).
5. 핵심식: BCJR 검출기 상호정보 `I_{E,DET}`(Monte Carlo로 추정, 3차 다항식 근사)를 VND 상호정보 식 (12)에 결합.
6. 확장: check node degree `ρ(x)`는 부분간 동일 가정, rate `R`을 `λ_{li}`,`ρ_j`로 표현.
7. 최적화: 주어진 `SNR_l`,`ρ`에서 CND EXIT 곡선이 VND 위에 오도록(`I_{A,CND}<I_{E,VND}`) `Σλ_{li}/i` 최대화 선형계획.
8. 학습: DE 대신 수정 EXIT chart 기반 degree 최적화, `dvmax=15`.
9. 결과: PR1/PR2 겸용 부호가 개별 설계 부호 대비 FER 2×10⁻⁴에서 약 0.2 dB, 40% media noise 수직채널에서 FER 10⁻³에 약 0.08 dB coding gain.
10. 한계: 시뮬만·HW 미설계, media noise는 preceding soft detector가 처리한다고 가정, 단일 채널 NAND에 부적합.
