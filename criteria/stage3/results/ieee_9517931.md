> 총평: cycle·흡수집합 최소화로 error-floor/waterfall 동시 개선하는 QC-SC-LDPC 부호설계 방법론으로 아이디어 가치는 있으나, 대상이 spatially-coupled(고정 block QC-LDPC 아님)라 Prime ECC로의 직접 이식은 부호 재설계 부담이 큼.

### B. 알고리즘 요약

1. 시스템: QC 구조 spatially-coupled(SC) LDPC, protograph `HP` + circulant size `z`, all-one base `Π`(γ×κ), partitioning matrix `P`·lifting matrix `L`로 표현; AWGN 채널.
2. 문제: memory `m`이 클수록 성능↑이나 최적 고memory SC 부호 구성은 조합폭발(OO/CPO 지수복잡도)로 난해.
3. 핵심 지표: 성분행렬 edge 분포 `p`를 coupling polynomial `f(X)=Σpᵢ X^aᵢ`로 표현, cycle-6/8 후보 기대개수를 `f`의 계수로 표현(Thm 1,2).
4. 핵심 기법(GRADE): `P6`, `N8` 최소화의 KKT 필요조건(Lemma 3,4)으로부터 gradient descent로 (국소)최적 edge 분포 `p` 도출(Algorithm 1).
5. 핵심 관찰: 고memory 최적 SC 부호의 edge 분포는 **비균일**(양 끝 치우침), 균일 partition은 준최적.
6. 구성(AO): 도출된 `p` 분포의 random partition에서 시작해 L1/L∞ 거리 제약 semi-greedy 탐색 + CPO로 국소최적 `P` 확보(Algorithm 2) → GD 부호.
7. 확장(TC): pseudo-memory `mt<m` 부호로 낮은 복잡도에서 고memory 효과, brute-force로 최적 coupling pattern `a` 탐색 후 OO+CPO.
8. 검출대상: trapping set/absorbing set 대신 공통 부분구조 small cycle(길이 4,6,8) 최소화.
9. 결과: (4,29) GD 부호가 UNF 대비 cycle-8 51.4%↓, FER 약 0.25 dB 이득; (3,17) GD는 weight-6 흡수집합 0개(UNF 6개), error-floor 개선. 복호는 FFT 기반 SPA, 최대 50 iter.
10. 한계: HW 미설계, rate/length/gatecount 본문 미명시, SC(convolutional)·windowed decoding 전제로 block QC-LDPC와 구조 상이.
