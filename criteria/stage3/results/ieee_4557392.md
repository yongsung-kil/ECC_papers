> 총평: QC-LDPC parity-check 행렬 구성을 위한 generalized Kronecker product 기반 girth 확장/연결성 증명 논문으로, 디코더나 HW와 무관한 순수 부호 구성 이론이라 이식성 낮음.

### B. 알고리즘 요약

1. 대상은 type I (J,L,s) QC-LDPC 코드로, parity-check 행렬 `H`는 circulant permutation matrix `I_s(p_j,l)` 블록들로 구성되며 `H = W ⊗_g P` (weight matrix `W`와 circulant 블록 `P`의 generalized Kronecker product)로 표현.
2. 풀려는 문제: 기존 `D(m,q)` 그래프 기반 `LU(m,q)` 코드는 `m>6`에서 비연결(disconnected)이 되어 실제 응용에 어려움이 있고, girth를 임의로 키우는 일반적 QC-LDPC 구성법이 부족함.
3. 핵심 가정/모델: Tanner graph의 cycle을 `H` 원소의 index/shift-value 시퀀스로 표현하고, 길이 `2i` cycle 존재 조건을 `Σ(p_jk,lk - p_jk+1,lk) ≡ 0 (mod s)` (discriminant)로 정식화(Theorem 1, Fossorier[14] 결과 재사용).
4. 핵심 기법: `generalized Kronecker product` `A ⊗_g B`(정의 2)를 기존 girth `2g`인 QC-LDPC 행렬 `H`에 반복 적용해 `H' = H ⊗_g P'`를 구성, cycle을 class I(discriminant 항등적 0)/class II(비항등적 0)로 분류.
5. 핵심 식(Theorem 6): girth `2g`인 `(J,L,s)` QC-LDPC 코드에서 시작해 유한 번의 generalized Kronecker product를 적용하면 길이 `2g`의 class II cycle이 없는 코드를 얻을 수 있음 — 이를 재귀 적용하면 임의로 큰 girth의 QC-LDPC 코드 구성 가능.
6. 확장: prime `q`에 대해 기존 `LU(m,q)` 코드가 이 construction의 특수 사례임을 증명(`H(m,q) = H(m-1,q) ⊗_g P`).
7. 부수 결과: Tanner graph 연결성 조건(Proposition 7, Corollary 8) — `d = Σ(p_jk,lk - p_jk+1,lk) mod s`에 대해 `gcd(d,s)=1`이면 연결(connected) Tanner graph 보장, 아니면 `gcd(d,s)`개의 isomorphic 컴포넌트로 분리됨을 증명.
8. 별도 학습/최적화 절차 없음(대수적/그래프이론적 구성과 증명 위주). 실제 블록 순열 행렬 선택을 위한 효율적 방법은 "아직 없음"이라고 명시(V. Other Issues).
9. 결과: 수치 시뮬레이션 없음. 인코딩 복잡도만 정성적으로 언급 — SRAA 인코딩 스킴([17]) 기준 `(L-J)s` 클록 사이클, `2Js` flip-flop, `Js` XOR/AND 게이트 필요(대형 weight matrix·소형 `s` 조합이 복잡도상 불리하지 않을 수 있다는 서술).
10. 한계: BER/디코딩 성능 시뮬레이션 전무, HW 구현 없음, 순열 행렬(shift value) 선택을 위한 효율적 알고리즘 부재를 스스로 명시.
