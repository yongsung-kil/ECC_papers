> 총평: LUT 기반 threshold-attenuated min-sum으로 하드웨어 비용 거의 없이 error-floor 개선을 달성한 실용적 기법으로, 기존 min-sum HW에 저비용으로 얹을 수 있어 이식성이 높다.

### B. 알고리즘 요약

1. 대상 코드는 (155,64) QC Tanner LDPC(3×5 블록의 31×31 circulant identity 시프트 행렬), AWGN+BPSK 채널.
2. 기존 MSA는 SPA 대비 근사 오차로 성능 저하가 있고, AMSA/OMSA는 저·중 SNR에서만 개선되며 고SNR(error-floor)에서는 trapping set 문제로 효과가 제한적.
3. 핵심 가정: 고SNR 영역의 복호 실패는 Tanner 그래프 내 국소적 trapping set에 기인한다고 보고, 이를 완화하려 한다.
4. 핵심 기법: check node 업데이트 시 incoming LLR들의 min 값이 임계값 `τ`보다 작으면 attenuation factor `α`를 곱하고, `τ` 이상이면 attenuation 없이 그대로 전달하는 `TAMSA`(식 6).
5. 식 (6)은 신뢰도가 낮은(작은) LLR만 선택적으로 감쇠시켜 과대추정을 억제하면서도, 신뢰도 높은 메시지는 보존해 고SNR에서 AMSA/MSA보다 나은 BER을 얻게 한다.
6. 확장으로 `layered TAMSA`를 제시: 식 (7)-(9)로 열(variable node) 메시지를 이전 값에서 감산·재계산하며, 행을 31개씩 3배치로 나눠 순차 갱신하는 layered 스케줄 적용.
7. 구현 디테일: 4-bit 크기 + 1-bit 부호로 LLR을 양자화(총 5비트)하고, attenuation은 실시간 곱셈 대신 사전 계산된 LUT(Table I/II) 매핑으로 대체해 HW 비용을 최소화.
8. 최적화 절차: α=0.8, τ=1.425는 시뮬레이션을 통해 선택(논문 내 별도 DE 최적화 절차 서술 없음, 이전 연구([17])의 quantized DE 인용만 있음).
9. 결과: layered TAMSA는 BER=10⁻⁹에서 MSA/AMSA 대비 약 0.4dB, TAMSA 대비 0.1dB 이득. 저SNR에서 평균 반복횟수(ANI)도 MSA 대비 유의하게 감소. FPGA(Xilinx Kintex-7, Cadence 0.18um) 합성 결과 layered TAMSA는 AMSA 대비 추가 면적/전력 없음, MSA 대비 leaf cell 0.07%만 증가.
10. 한계: (155,64) 소형 코드 단일 예시로만 검증(대형/고rate QC-LDPC 미검증), pipeline 미적용(완전병렬 구조 특성상), α/τ 선택 기준에 대한 이론적 최적화 절차는 상세 기술되지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Min-Sum CN 업데이트 attenuation(threshold 기반) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | min-sum 근사값 보정 로직에 직접 삽입 가능한 위치 |
| LLR 양자화(4-bit+sign) 및 attenuation LUT | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | 기존 iteration별 LLR 테이블 구조에 threshold LUT를 추가 확장하는 형태로 이식 가능 |
| Layered 스케줄(31행 단위 순차 갱신) | `decoder.cpp` `LDPC_Decoder()` (Dual-Update 옵션과 유사 개념) | 이미 Dual-Update(even/odd 교대) 보유, layered TAMSA는 이와 유사한 스케줄 변형이라 중복 가능성 있음 |
| 조기종료(수렴 판정) | `partial_CRC.cpp` `partial_crc_HW_T4()` | 논문은 codeword 검증만 언급, CRC 방식 상세 없어 대응 약함 |
| FPGA 완전병렬 CNU 구조(최소/차소 계산 회로) | 대응 없음 | Prime ECC는 z=32 lifting 기반 row-parallel 구조로 논문의 5-CNU 완전병렬 구조와 스케일이 다름 |

> 적용 가치: 높음 — 기존 min-sum 디코더에 LUT 기반 threshold attenuation만 추가하면 HW 비용 거의 없이 error-floor 개선과 iteration 감소를 동시에 얻을 수 있어, Prime ECC의 decoder.cpp CN 연산에 낮은 난이도로 이식 가능.
