> 총평: SC-LDPC의 local/global 코드 설계 이론 논문으로 NAND 고전 binary QC-LDPC 이식 관점에서는 참고 가치가 낮음.

### B. 알고리즘 요약

1. SC-LDPC(m=1, coupling-length L) 코드에 sub-block locality를 부여: 코드워드가 L개 sub-block으로 나뉘고 각 sub-block은 local code로 독립 복호 가능, 전체는 더 강한 global code.
2. 기존 SC-LDPC 설계(threshold 최적화 후 cycle 최적화)는 locality 조건에서 불충분하다는 문제의식([6])에서 출발, 본 논문은 local과 global을 동시에 finite-length(cycle-6 개수)와 asymptotic(EXIT threshold) 관점으로 재설계.
3. 채널은 AWGN, 디코더는 BP(EXIT 방법으로 근사) 가정.
4. Local 설계: γL×κ protograph에서 ν개 zero-circulant를 넣는 두 구성 balanced(`BB`, 식(4))와 unbalanced(`BU`, 식(5))를 제안 — 동일 rate, 서로 다른 irregularity 분포.
5. Proposition 2: `κ-⌈ν/γL⌉ ≤ ν`이면 `σ*(BU) ≤ σ*(BB)` (unbalanced가 threshold 열세), Proposition 3/4: γL=3에서는 cycle-6 개수 `F(BU) ≤ F(BB)`(unbalanced 우세)이나 γL=4에서는 balanced가 threshold·cycle 둘 다 우세(trade-off 소멸).
6. Global 설계: coupling CN(CCN) 배치를 위한 partitioning 방법 3종(Cutting-Vector, Locality-Blind Optimal, Locality-Aware Optimal(LAO))을 비교, LAO는 [5]의 optimal overlap partitioning을 locality 제약 하에 확장.
7. Lemma 5: 정규 local code(PL=0)일 때 독립 overlap parameter 수가 γC에만 의존, 즉 locality 도입이 LAO partitioning 복잡도를 증가시키지 않음.
8. 최적화 절차는 조합적 cycle-6 개수 최소화(discrete optimization)와 EXIT 기반 threshold 계산의 조합, 별도 ML/GA 학습 없음.
9. 결과: LAO가 CV 대비 cycle-6 21% 감소(LBO는 5% 미만), irregular 도입(ν=10)으로 LAO 대비 추가 63%(balanced)/55%(unbalanced) 감소, global BER은 SNR=7dB에서 CV 대비 1.5 orders of magnitude 개선.
10. 한계: HW 설계 없음, 순수 시뮬레이션(AWGN, Monte Carlo, 최소 50 frame error)뿐이며 error floor 영역까지는 SNR 제약으로 확인 못함(각주1).

| Locality-Aware Optimal partitioning(cycle-6 최소화) | 대응 없음 | Prime ECC는 이미 확정된 H-matrix를 사용, cycle 최적화 부호 재설계 단계 없음 |
| EXIT 기반 asymptotic threshold 분석 | 대응 없음 | 시뮬레이터는 bit-exact HW 모사이며 asymptotic 이론 분석 도구 없음 |
