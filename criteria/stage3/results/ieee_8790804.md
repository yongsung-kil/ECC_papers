### B. 알고리즘 요약

1. 시스템: 지하무선센서망(WUSN)의 자기유도(MI) 통신 링크에 QC-LDPC 부호적용, 부호율 1/2·길이 2304(code A1) 등, sub-matrix `L=96`.
2. 문제: MI 채널의 오류전파를 완화할 채널부호가 필요하나 기존 랜덤 LDPC는 HW 부적합·고에너지, MI 전용 부호최적화 연구 부재.
3. 모델: MI 채널을 변압기 등가회로로 모델링, 경로손실 `LMI(d)=6.02+60lgd+10lg(Nt Nr a_t^3 a_r^3)`로 거리 `d` 종속 잡음분산 `σ_ch,j^2=8R·f(d)` 정의.
4. 핵심기법1: MI-PEXIT — protograph EXIT를 MI채널로 확장, 변수/체크노드간 상호정보 `IEv/IEc/IAPP`를 반복추적하여 `IAPP→1` 수렴여부로 "거리임계값 `d*`" 산출.
5. 식 의미: `d*`는 부호가 성공전송 가능한 최대 코일거리로, PSO의 적합도(fitness)로 사용 — 성능을 거리 하나의 스칼라로 요약.
6. 핵심기법2: 이산(binary) PSO — 최적화 변수는 정보부 base matrix `B(HI)`를 벡터화한 `Wb`, 속도갱신 `v=λv+δ(pbest)+δ(gbest)`, 시그모이드 확률로 비트 결정.
7. 구현 디테일: 저부호화복잡도 위해 parity부 `HP`(dual-diagonal, 802.16e류)는 고정, `HI`만 탐색 → H로부터 재귀적 직접 인코딩 가능.
8. 최적화 절차: `Pnum=20` 입자, `Imax=30` 반복으로 `B(HI)` 탐색, exponent matrix 비영항은 [0,L-1] 랜덤부여.
9. 결과: code A2가 거리임계값 8.42m로 rate/length 절충 최적, code A1이 802.16e 대비 BER 10^-4에서 약 4.4m 전송거리 개선.
10. 한계: MI 직접통신 채널·거리모델에 국한(도파관 미포함), HW/RTL 설계 없음, 순수 시뮬레이션, error-floor·복호기 개선 아님.
