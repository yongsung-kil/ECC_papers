### B. 알고리즘 요약

1. 시스템: AWGN 채널, 고rate(R≈0.89) GC-LDPC(QC-LDPC의 특수 masking 형태), 예제 block length `N=18308~20148`.
2. 문제: 기존 GC-LDPC(finite field 기반)는 finite field 선택이 col/row degree·block length·CPM size를 암묵 고정 -> 원하는 rate/length에 맞는 파라미터 탐색이 어려움.
3. 구조: GC-LDPC = 여러 disjoint local code(대각)+bottom의 globally-connecting part `HG`. `HG`가 모든 VN을 연결해 high connectivity -> 반복복호 수렴 촉진.
4. 기법1 (RS-like): `γs-1 = t·p·e` 인수분해 없이 임의 prime `γ`,`s`가 존재한다 가정하고 RS matrix `RS(a,b,d)`로 base matrix 구성 -> 파라미터 자유도 확대.
5. cycle-4 회피 조건: `d1<p/t`, `d2<t`(2x2 submatrix constraint [2])로 girth 보장.
6. 기법2 (directly masking): 기성 QC-LDPC의 base/binary PCM에 GC-masking `MGC`를 element-wise 곱 -> off-the-shelf LDPC를 GC-LDPC로 변환, girth 감소 없음.
7. 구현 디테일: locally-joint 변형(overlapping rows/cols, Fig.4)으로 local code 결합도↑ -> unequal-SNR 시나리오 성능 개선.
8. 복호: row-layered shuffle min-sum, scaling 0.5, 최대 20 iteration (논문 자체 기여 아님, 차용).
9. 결과: 4~6개 예제 코드 BLER 비교 - 제안 GC-LDPC가 conventional GC-LDPC/QC-LDPC와 comparable; unequal-SNR에서 connectivity 높은 code(Code-3m)가 우세.
10. 한계: HW 미설계, gatecount/throughput 없음; 정정능력·error-floor 정량 개선 미제시, 순수 구성법+시뮬.
