### B. 알고리즘 요약 (10줄 내외)

1. 대상은 hard-decision 반복 디코딩(GA, GB, MB 계열)으로 복호되는 LDPC 부호의 BSC(binary symmetric channel, crossover probability `ε`) 상 성능 추정.
2. 기존 몬테카를로 시뮬레이션은 저오류율(낮은 p) 영역에서 계산 비용이 매우 크다는 문제를 해결하려 함.
3. 디코더가 정정 못하는 최소 가중치 오류패턴의 개수 `|EJ|` (가중치 `J`)만 전수 열거하면 나머지 오류패턴들의 실패 확률을 조합론적으로 근사할 수 있다는 가정(Assumption a, b)이 핵심 모델.
4. `Si`를 `EJ`의 원소를 부분집합으로 포함하는 것과 아닌 것으로 분할하고, `|Si'| ≈ |EJ| * C(n-J, i-J)`로 근사(식 3)해 FER 상/하한 `FERL(N)`, `FERU(N)`를 유도(식 4, 5).
5. 실패 확률이 급격히 증가하는 임계 가중치 `N0`를 몬테카를로(고FER 영역, p≈0.01~0.1)로 소수 실험해 결정하면 `FERU(N0)`가 전 구간에서 정확한 FER 추정치가 됨.
6. BER은 `J ≤ i < N0` 구간에서 평균 비트오류수를 `J`로, `N0 ≤ i ≤ n` 구간에서는 `N0` 가중치 오류패턴의 평균 비트오류수 `M`(몬테카를로로 별도 산출)로 근사(Assumption c, d)해 식 (6)으로 결합.
7. 트래핑셋 구조를 직접 식별할 필요 없이 임의의 degree distribution·임의의 hard-decision 알고리즘에 보편 적용 가능하다는 것이 기존 Richardson [7] 접근 대비 장점.
8. 별도 학습/최적화 절차 없음 (조합론적 근사 + 소량 몬테카를로 캘리브레이션).
9. Code 1(200,100) 기준 목표 FER `p=10^-7`에서 몬테카를로 대비 계산량 η=750배 절감(p=10^-8 가정 시 8250배), 4개 code(200~1998bit, regular/irregular) 모두 FER/BER 곡선이 시뮬레이션과 잘 일치.
10. HW 미설계, hard-decision 알고리즘(GA/GB/MB) 전용이며 soft-decision min-sum 계열 디코더에는 직접 적용되지 않음; 트래핑셋 타입(fixed/oscillatory/random-like) 분류는 실험적 관찰에 근거.
