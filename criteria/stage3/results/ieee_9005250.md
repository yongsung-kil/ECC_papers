### B. 알고리즘 요약

1. 대상은 이진 LDPC 부호(AWGN, BSC), SPA/MSA/normalized-MSA 등 대칭 반복 메시지패싱 디코더 전반, quantized 및 unquantized 모두 포함. 검증에 사용된 코드는 rate 0.587~0.84, length 63~2640의 QC/random/irregular/finite-geometry 등 7종.
2. 풀려는 문제: 기존 linear state-space model([10])은 quantized(비선형) 디코더에 적용 불가하고, 선형화(0 근처 Taylor)로 인해 error floor를 과대추정하는 한계가 있음.
3. 핵심 가정: trapping set(TS) 주변 Tanner graph는 tree로 근사, TS로 들어오는 외부 메시지(unsatisfied/missatisfied check node 경유)는 density evolution(DE)으로 얻은 분포를 따름.
4. 핵심 기법: Local Importance Sampling(LIS) - 전체 Tanner graph 대신 TS가 유도하는 부분그래프에서만 실제(비선형) message-passing을 반복 수행해 실패확률 `Pe(a,b) = (1/N)Σ I_z ω_z` 추정, biasing으로 실패율 10~50%가 되도록 조정해 rare event 시뮬 가속.
5. missatisfied check node로 들어오는 메시지는 반복 `ℓ < g/2`(g=girth)까지는 DE 분포에서 i.i.d. 샘플링, `ℓ ≥ g/2`부터는 상관관계 근사를 위해 매 반복 1개 샘플을 모든 missatisfied check node에 공유("correlation model") - 이 상관을 무시하면(LIS-DEnc) error floor를 과소추정함을 실험으로 확인.
6. 이론 분석(선형화 모델 확장): missatisfied check node의 random gain에 분산·상관계수를 도입, 상관 `κij≥0`이면 항상 실패율이 증가함을 Theorem 1로 증명 - 기존 linear state-space model의 과대추정 원인이 선형근사(0 근처 Taylor) 자체에 있음을 규명.
7. 복잡도: 부분그래프 반복당 메시지 수는 `a·dv - b`로 전체 그래프(`n·dv`) 대비 `n/a` 배 절감, 코드 독립적(디그리 분포만 필요, 전체 Tanner graph 불필요)이라는 것이 IS 대비 강점.
8. 결과: 7개 코드(QC/Margulis/random/Wimax/RS-LDPC/EG-LDPC/PG-LDPC), 다양한 quantization(3~8bit MSA/SPA)에서 LIS-DEc 추정치가 Monte-Carlo 시뮬과 정확히 일치, 기존 linear state-space model은 최대 한 자릿수(order of magnitude) 과대추정.
9. 비균일 quantization(MI 최대화 기반)에도 적용 가능함을 보임, dominant TS class가 SNR/quantization bit 수에 따라 바뀌는 것도 정확히 추정(예: C3에서 quantization 6bit→5bit시 dominant class가 (5,1)→(4,2)로 전환).
10. 한계: 이론/시뮬 검증뿐(HW 없음), saturating decoder에만 적용 가능(non-saturating 디코더는 모델 밖), TS enumeration(dominant ETS 탐색)은 별도 알고리즘 필요 - 이 논문은 이미 알려진 TS 구조의 기여도만 추정.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| trapping-set별 error-floor 기여도 추정(LIS-DEc) | 대응 없음 | Prime ECC 코드에는 error-floor 분석/진단 전용 모듈 없음, 오프라인 검증 도구로만 이식 가능 |
| quantized min-sum/normalized-MSA의 error-floor 분석 대상 | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC의 min-sum quantization 파라미터가 이 방법론의 입력(quantization scheme)이 됨 |
| DE 기반 메시지 분포 계산 | `ecc_data.h` `PARAM_LLR` | 논문의 DE는 LLR 테이블과 무관하게 디그리 분포 기반 별도 계산, 직접 대응 없음 |
| dominant TS 구조 및 multiplicity | 대응 없음 | Prime ECC의 실제 H-matrix(QC-LDPC, base+shift)에서 dominant ETS를 찾는 별도 탐색 알고리즘([11],[33]-[36] 참조 필요) 부재 |

> 적용 가치: 중간 — 디코더/부호를 직접 바꾸는 기여는 없으나, Prime ECC의 quantized min-sum 디코더에 대해 (오프라인으로) error-floor를 코드 독립적/저비용으로 예측·검증하는 진단 도구로 활용 가치 있음(HW 이식 대상 아님).
