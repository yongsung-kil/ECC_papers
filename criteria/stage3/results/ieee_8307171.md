### B. 알고리즘 요약

1. 시스템: 고bit-rate 광통신 OTN용 concatenated FEC — inner soft-decision LDPC + outer hard-decision staircase code, QPSK/AWGN 채널, 목표 BER `<10^-15`, OH 15~25%(rate ~0.83).
2. 문제: soft-decision 디코더가 hard 대비 1dB coding gain을 주지만 전력/복잡도가 한 자릿수 크다 → data-flow(복잡도) 최소화 필요.
3. 핵심 아이디어: inner LDPC ensemble에 **degree-0(uncoded)·degree-1 variable node를 허용** — degree-0은 inner 복호 복잡도 0, degree-1은 data-flow 불변이라 복잡도에 무기여.
4. 핵심 변수: 설계 파라미터 `(L0, Λ)` (L0=uncoded 비율, Λ=variable degree 분포), inner 복잡도 score `ηi=(1-Rin)(dc-ν)I / Rin`.
5. 의미: uncoded bit를 늘리면 inner가 처리할 edge가 줄어 data-flow가 감소하고, outer staircase가 나머지 BER를 `10^-15`까지 낮춤.
6. 확장: check-degree를 1~2개 인접 degree로 집중, degree-1 노드 평균수 `ν=dc·λ1`를 EXIT 계산에 반영.
7. 구현 디테일: EXIT chart로 inner 성능 예측(반복수 `I≈∫dp/(p log(p/fΛ(p)))`), 이를 SQP로 convex 최적화, staircase table을 loop해 (SNR, ηi) Pareto frontier 도출. inner-outer 간 diagonal interleaver로 오류 상관 완화.
8. HW 친화화: 랜덤 구조 대신 **girth-8 QC-LDPC**(length 6000~30000) 구성 → energy-efficient, layered message-passing schedule로 flooding 대비 복잡도 최대 50% 추가 절감.
9. 결과: 기존 설계 대비 inner 복잡도 15% OH에서 71%, 20%에서 52% 절감하며 동일 NCG 달성, Shannon limit에 0.11~0.21dB 근접. 예제 inner code 8~16 iteration.
10. 한계: HW 미설계(gatecount 없음), 시뮬만, outer staircase 의존 concatenated 구조, latency 14μs(200Gb/s), QC 구조 최적성 미보장.
