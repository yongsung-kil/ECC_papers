### B. 알고리즘 요약

1. 대상은 protograph 기반 generalized spatially-coupled LDPC (GSC-LDPC) 앙상블로, GLDPC convolutional 부호를 유한 길이로 종단(termination)하여 구성한다.
2. 문제: GSC-LDPC은 우수한 iterative decoding threshold(MAP threshold 수렴)는 알려졌으나 최소거리(distance) 성질이 미확인이었고, error floor 예측에 필요하다.
3. 모델: `(2,7)-regular` GLDPC block code를 예로 삼고 constraint node를 `(7,4)` Hamming code로 둔다(SPC 대신 일반 선형부호 constraint).
4. 구성: convolutional protograph base matrix `B[0,∞]`를 edge spreading으로 `B0+B1+...+Bms = B` 분할 후, 종단인자 `L`로 잘라 `B[0,L-1]` 유한 base matrix를 만든다.
5. 종단으로 인해 rate는 `RL = 1 - (L+ms)bcMC/Lbv - Δ`가 되며 `L→∞`이면 블록부호 rate로 수렴한다.
6. tail-biting 종단(GTB-LDPC)도 병행하여 종단인자 `λ`에 대해 rate `1/7`을 유지하는 앙상블을 얻는다.
7. Abu-Surra/Divsalar/Ryan의 weight enumerator 기법으로 앙상블 평균 asymptotic spectral shape function `r(δ)`를 계산해 asymptotic goodness(δmin 존재)를 판정한다.
8. GSC-LDPC와 GTB-LDPC의 최소거리 성장률로 periodically time-varying GLDPCC 앙상블의 free distance 성장률 상·하한을 유도(식 (11),(12)).
9. 결과: `(2,7)`-regular Hamming-constraint 예에서 δmin=0.186, T≥10에서 상·하한이 일치하여 free distance 성장률 δfree=0.805로 블록부호보다 크게 나옴.
10. 한계: 디코더·HW 미설계, 수식 이론뿐이며 실측/시뮬 BER 없음. non-SPC(generalized) constraint + convolutional 구조라 고전 QC-LDPC와 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| GSC-LDPC / GLDPCC 부호 구성 (spatial coupling, Hamming constraint) | 대응 없음 | Prime ECC는 QC-LDPC block, generalized/convolutional 구조 미지원 |
| 최소거리/free distance 성장률 이론 분석 | 대응 없음 | 앙상블 존재성 증명은 코드 모듈에 대응 없음 |
| protograph base matrix / lifting | `ecc_top.cpp` `Load_PCM()` (부분적 유사만) | H-matrix 로드 개념은 있으나 convolutional 종단 구조와 불일치 |

적용 가치: 낮음 — generalized/spatially-coupled convolutional 앙상블의 순수 최소거리 이론으로 고전 binary QC-LDPC block 코드베이스에 떼어 쓸 알고리즘·HW 요소가 없음.
