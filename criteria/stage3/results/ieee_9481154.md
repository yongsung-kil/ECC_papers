### B. 알고리즘 요약

1. 대상: `(J, L)` full-length row-multiplier(FLRM) QC-LDPC 코드, 지수행렬 `Ef=[ai·r]`, CPM 크기 `P`.
2. 풀려는 문제: GCD 제약이 girth-8을 보장(충분)하는 것은 알려졌으나, 그것이 *필요*조건인지 미해결.
3. 모델: Tanner 그래프의 length-6 cycle 존재 여부를 지수행렬 정수관계식으로 환원.
4. 핵심 기법 1: GCD 제약 `(ak−ai)/gcd(ak−ai, aj−ai) ≥ L` 이 위배되면 반드시 length-6 cycle이 존재함을 대우로 증명(Lemma 2) → 필요조건 확립.
5. 핵심 식 의미: 위 부등식이 임의 CPM 크기에서 6-cycle을 막는 정수 조건이며, 위배 시 특정 열 인덱스 삼중항이 cycle을 형성.
6. 핵심 기법 2: Theorem 1로 필요충분성 확정 → GCD 제약 검증만으로 최단 girth-8 FLRM 코드 탐색 가능.
7. 구현 디테일: Algorithm 1은 boundary 성질(`ai<ai+1`, `ai≤ai+2−L`, `ai≥⌊i/2⌋L+mod(i,2)`)로 탐색공간을 좁혀 상한 없이 `aJ−1`을 1씩 증가.
8. 부가 결과: base expansion(BE) 방법이 GCD 방법의 특수 경우임을 증명(Theorem 2), L=3에서 ES(no-three-term) 시퀀스와 동치.
9. 결과: `J=5~8`, `L=J~20`에서 기존 [10]보다 더 작은 CPM 크기 획득; 예시로 rate 3/4 (3,12,368) GCD+CS 코드(길이 8832)가 girth-10 VGB 코드와 BER/BLER 거의 동등(SPA 50 iter, AWGN/BPSK).
10. 한계: 부호 구성 이론뿐, 디코더/HW/NAND 채널 미논의, small trapping set 회피는 향후 과제로 남김.
