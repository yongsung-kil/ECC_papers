> 총평: DVB-S2 error-floor를 degree분포/짧은루프 관점에서 진단하는 분석 방법론으로 통찰은 있으나 신규 부호·디코더·HW가 없고 특정 표준 종속이라 Prime ECC 이식 대상이 아님.

### B. 알고리즘 요약

1. 대상: DVB-S2 표준 rate `3/4`, length `N=64800` (K=48600) irregular LDPC, 8-PSK, AWGN, BP 복호.
2. 문제: BP는 loopy Tanner graph에서 sub-optimal이며 trapping set이 error-floor를 유발 → error-floor의 원인을 부호 구조로 규명하려 함.
3. 구조 분석: H가 3영역으로 분할 (`dv=12` for i≤5400, `dv=3` for 5400<i≤48600, `dv=2` for i>48600 = 스테어케이스 패리티).
4. connectivity 정의: `|Xi| = dv(i)·(dc−1)` (dc=14). 지역밀도만 보면 dv=12가 우세하나, 실제 연결도는 dv=3 노드가 더 높음.
5. loop extractor: BFS로 `Gi(d)` 구성 후 DFS로 길이 `l≤10` 루프 탐색 (d=5). 조건 `l<2d` (길이 l 루프는 l/2 iteration 후 영향).
6. 핵심 관찰: 최단 루프는 길이 6; degree-2 비트노드가 6-loop에 2배 더 관여하고 이웃 2개뿐이라 루프 탈출 불가 → BP 복호 취약.
7. forced-LLR 실험: 특정 subgraph에 매우 큰 음의 LLR `−L` 강제 주입, 복호오류/채널오류 비 `γdv` 측정 → dv=2가 dv=3보다 2배 더 실패.
8. error 분류: persistent(BP 전후 오류) vs created(BP 후 새로 생긴 오류); created 오류가 주로 degree-2에서 발생 → error-floor(구간 I3) 지배.
9. 모델: SNR별 degree별 오류비 `ρdv`와 평균 error event 크기 `wH(x̂)`로 `ndv = ρdv·wH(x̂)` 시뮬레이터 구축.
10. 결과·한계: DVB-S2 외부 BCH(최대12정정)가 error-floor에서 부족함을 보이고 degree-2 노드 재설계를 제언; 신규 부호·디코더·HW 없음, 순수 분석·특정 표준 종속.
