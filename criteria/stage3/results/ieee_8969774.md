### B. 알고리즘 요약

1. 시스템은 다중안테나 2D(pre-overlapping) 보안 전송 채널이며 QC-LDPC로 인가 수신자 BER을 개선, rate `r=1/2` 고정.
2. 문제: 2D 전송이 도청은 막지만 인가 수신자 BER이 나빠져 이를 부호로 보완해야 함.
3. QC-LDPC parity check matrix `H`는 base matrix `U`(z×z 순환치환 offset 행렬)와 circulant size `z`로 정의.
4. 핵심: Tanner 그래프에 girth-4(길이 4 사이클)가 없도록 `U`를 설계 - 사이클 조건 `sum(u_rt,st - u_rt+1,st)=0 mod z`.
5. girth-4를 형성하는 4원소 패턴을 `GF`로 정의, `u_g,f - u_g,n = u_m,f - u_m,n`이면 GF 발생.
6. GF 없는 3조건: 비-첫행·열 원소는 0 불가 / 같은 행·열 원소는 서로 다름 / 부분행렬 `D`의 열간 차이가 서로 다름.
7. 2D 채널계수 행렬 `h_AB^H`를 양자화·정수화하여 벡터 `alpha`를 만들고 이를 순환 이동시켜 `U`의 첫 행부터 구성.
8. dimension reduction 알고리즘: difference matrix `Ds`로 GF 유발 원소 탐지 후 여유원소집합 `phi_o`에서 대체, GF 수 `n2<n1`이면 채택, 최대 `T`회 반복.
9. 결과: MacKay 부호 대비 BER 곡선 급경사, SNR 8dB에서 MacKay BER이 DRC의 10배; 복잡도는 `(LJ)^3`에 비례.
10. 한계: HW 미설계, BPSK+BF 시뮬만, 무선 2D 전송 특화, r=1/2 고정, 부호 length 명시 없음.
