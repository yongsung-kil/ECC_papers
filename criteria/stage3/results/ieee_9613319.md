### B. 알고리즘 요약

1. 시스템: 2-user downlink NOMA, BS가 `X=√P1 x1+√P2 x2` 중첩 송신, 근접(UE1)·원거리(UE2) 사용자, BPSK/AWGN, protograph LDPC 부호(길이 `n=4800`, rate 1/4~2/3, base matrix 4×8).
2. 문제: 전통 irregular LDPC는 인코딩/디코딩 복잡도가 높고 NOMA 채널용 정확한 threshold 예측이 어려움.
3. 모델: 채널 `y_l=h_l(√P1 x1+√P2 x2)+n_l`, `P2>P1`(원거리 사용자 고전력), SIC로 co-channel 간섭 제거.
4. 핵심기법 1단(E-SIC): 두 decoder가 매 iteration마다 외부정보 `Pr_l(ŝ)`를 교환. x2 먼저 복호→soft output을 decoder1의 사전정보로, 다시 x1 soft output을 decoder2로 (식 (1),(2) 초기 LLR).
5. 핵심기법 2단(TU-EXIT): P-EXIT를 2-user용으로 수정, state-to-variable 갱신에 `F±[l]` 함수(식 (5),(6),(12),(13))로 상대 사용자 신호 간섭·전력 반영, mutual info `J(σ)` 기반.
6. 핵심식: 상태노드 업데이트 `I_SV`가 두 사용자 전력 `P1,P2`·`σ²`에 의존 → protograph 구조를 반영한 정확한 decoding threshold 예측.
7. 부수: PEG로 base matrix를 확장인자 400으로 expand해 H 생성, degree-2 VN은 waterfall 개선/error-floor 유발이라 비율 제어, 일부 VN degree≥3로 최소거리 선형성장 보장.
8. 최적화: TU-EXIT threshold 최소화로 base matrix 4종(rate 1/4~2/3) 설계.
9. 결과: 저rate(P1=0.55,P2=0.75)에서 irregular 대비 >0.3dB(BER 10⁻⁵), 고rate(P1=1.8,P2=3)에서 0.8dB 이득, AR3A/AR4JA보다 우수(AR3A는 degree-2로 error-floor). threshold 표(0.572~3.997dB).
10. 한계: 순수 시뮬, HW 미설계, 2-user NOMA·SIC 정보교환 구조에 종속.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| protograph base matrix 설계 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 QC-LDPC(base+shift) 고정이라 protograph 부호계열·설계규칙 상이 |
| E-SIC 사용자간 외부정보 교환 | 대응 없음 | 단일 채널 NAND ECC에 다중사용자 SIC 개념 부재 |
| TU-EXIT threshold 분석 | 대응 없음 | EXIT는 설계도구, Prime ECC 코드에 대응 모듈 없음 |
| 초기 채널 LLR `L(b|y)` | `channel.cpp` `Set_LLR_Th()` | NOMA 중첩신호 LLR로 full-soft, hard/2~3bit 양자화와 불일치 |
| degree-2 VN 비율/error-floor 규칙 | `decoder.cpp` `LDPC_Decoder()` (간접) | 부호 degree 설계 규칙은 개념적 참고만 가능(직접 이식 아님) |

적용 가치: **낮음** — NOMA 2-user SIC·EXIT 설계 종속의 protograph 부호로, 단일채널 고rate binary QC-LDPC NAND ECC와 채널·부호계열이 어긋남. degree 분포·error-floor 억제 설계 규칙 정도만 개념 참고.
