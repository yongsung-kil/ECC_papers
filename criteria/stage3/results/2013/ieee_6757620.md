### B. 알고리즘 요약

1. 시스템: 채널디코딩용 이진 H-matrix(M×N)의 GF(2) 가우스소거를 FPGA로 구현, ABP/OSD/LP 등 신형 소프트디코더가 요구하는 reduced row echelon 변환 가속.
2. 문제: SMITH는 N·M 셀로 자원 과다(RS(255,239) 128×2040→261,120셀), systolic은 M² 셀이나 사이클(`3M+N-2`) 많고 H-matrix의 high aspect ratio·가변 designated column 특성 미반영.
3. 모델: 선형부호 H에 행 mod-2 가산으로 designated column을 단위벡터화(부호 자체 불변), 완전소거 불필요 가정.
4. 핵심 기법: M개 소거단계를 각각 하드웨어 column eliminator로 매핑해 파이프라인 array 구성, eliminator마다 pivot 위치 고정.
5. 핵심 식/구조: array 크기가 행 수 M에만 의존(열 N 무관) → N≫M(고aspect)일수록 유리, latency `M` 클록.
6. 확장: 2-phase 동작 — Setup phase(designated M열 투입해 각 eliminator의 가산 결정), Elimination phase(나머지 열 통과, M 클록 후 소거열 출력).
7. 구현 트릭: pivot을 priority encoder 없이 파이프라인이 암묵 탐색 → encoder 회로 제거, setup 후 임의 개수 열을 추가 자원 없이 소거.
8. 최적화 절차: 없음(아키텍처 설계).
9. 결과: Virtex7 place&route, 제안 방식 RS(255,239) 16.6k LUT/32.9k FF(systolic 81.7k/98.6k), WiMAX(576,480) 9.4k/18.5k, 사이클 `M+N`으로 systolic `3M+N-2`보다 적어 throughput ~20%↑(matrices/s).
10. 한계: GE 자체는 FER 미개선(소프트디코더 enabler), throughput이 matrices/s 단위, 외부 메모리 M×N 필요, RS/짧은 LDPC 사례만.
