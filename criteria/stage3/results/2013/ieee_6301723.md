### B. 알고리즘 요약 (10줄 내외)

1. 대상은 OFDM 시스템의 채널추정+검출+디코딩을 하나의 factor graph에서 결합하는 일반 프레임워크이며, 특정 LDPC 코드 스펙은 정의하지 않는다.
2. 기존에는 채널추정(MF)과 검출/디코딩(BP)을 별도로 heuristic하게 결합해 왔고, extrinsic 값 vs APP 값 선택에 이론적 근거가 없었다.
3. factor graph를 hard-constraint(코드 제약 등)를 포함하는 "BP part"와 conjugate-exponential model인 "MF part"로 분할한다는 것이 핵심 가정이다.
4. region-based free energy approximation(Yedidia et al. 방식)에 두 파트를 위한 서로 다른 counting number를 부여해 하나의 constrained Lagrangian `L_A[b]`을 구성한다(식 20).
5. Theorem 2: Lagrangian의 stationary point가 BP-MF 결합 메시지패싱 고정점(식 21, 22)과 일치함을 증명 — 이는 BP part 메시지는 extrinsic, MF part로 가는 메시지는 APP이어야 함을 이론적으로 도출한다(Remark 3.3).
6. Lemma 2로 BP part factor가 항등적으로 0이 될 수 있는 hard constraint(코드 제약 등)까지 이론을 확장한다.
7. Algorithm 1: BP part가 사이클 없고 특정 기술조건(식 26)을 만족하면, MF belief 갱신과 BP part의 forward-backward 알고리즘을 교대로 수행하는 방식으로 수렴이 보장된다.
8. 응용 예시(Algorithm 2)에서는 convolutional code + interleaving + 고차 변조를 BP part로, 채널이득 파라미터 추정을 MF part(conjugate-exponential Gaussian)로 분리해 pilot 기반 반복 채널추정/디코딩에 적용한다.
9. 결과: BER 관점에서 제안 Algorithm 2가 "BP + Gaussian 근사" 방식보다 우수하며, 저밀도 pilot 환경에서도 완벽한 CSI를 아는 경우에 근접하는 성능을 보인다(Fig. 2, 수치는 그래프뿐 본문 서술 없음).
10. 한계: LDPC를 직접 다루지 않고 convolutional code 예시로 개념을 검증했으며, HW 설계나 복잡도/게이트카운트 분석은 전혀 없는 순수 이론+시뮬 논문이다.
