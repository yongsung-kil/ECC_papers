### B. 알고리즘 요약

1. 대상 시스템: sub-lithographic nanowire 기반 나노메모리로, 저장셀뿐 아니라 encoder/decoder(corrector) 주변 회로도 transient fault에 취약하다고 가정.
2. 풀려는 문제: 기존 fault-tolerant 메모리는 주변 회로를 fault-free로 가정 → 나노스케일에서는 무효. 회로 자체 오류까지 정정 필요.
3. 핵심 개념: FSD-ECC(fault-secure detector capable ECC)라는 제한적 ECC 정의 — parity-check matrix 구조가 수신 codeword와 detector 회로 오류를 동시에 검출 보증.
4. 핵심 기법: EG-LDPC(Euclidean Geometry LDPC)가 이 FSD 성질을 만족함을 증명하고, 그 parity-check matrix로부터 fault-secure detector를 자동 생성.
5. 정정 메커니즘: encoder/corrector 출력에 오류 검출 시 해당 유닛이 연산을 재시도(retry)하여 올바른 출력 생성 → 완전 fault-tolerant 메모리.
6. 신뢰성 부품: 전 유닛을 fault-prone 나노회로로 구현하고, syndrome 누적용 OR 게이트 2개만 신뢰성 회로로 구현.
7. 부호 효율: EG-LDPC rate를 Gilbert-Varshamov 상한/Hamming 하한과 비교 → FSD 성질을 code compactness 희생 없이 달성.
8. 시스템 설계: memory banking, scrubbing 주기, defect threshold 등 파라미터로 reliability/area/performance 균형 최적화.
9. 결과: 나노메모리 시스템의 area/performance/reliability를 보고(단, 추출본에서 수치 표 손상으로 확인 불가).
10. 한계: hard-decision majority-logic 계열 결함검출 회로 중심으로, soft-decision iterative message-passing 복호 성능 개선과는 무관.
