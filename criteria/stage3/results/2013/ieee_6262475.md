### B. 알고리즘 요약

1. 시스템: protograph 기반 LDPC convolutional code(LDPC-CC) 앙상블 - regular/irregular, RJA/ARJA 계열, syndrome former memory `ms`, decoding constraint length `nu=(ms+1)b` 구조.
2. 문제: LDPC-CC는 반복 메시지패싱 복호에서 좋은 성능을 보이나, 점근적으로 free distance가 constraint length에 비례해 선형 성장하는지(asymptotically good)와 error-floor를 좌우하는 trapping set 크기가 함께 성장하는지가 미해결.
3. 핵심 가정: convolutional protograph를 "edge spreading"으로 블록 protograph에서 유도, tail-biting LDPC-CC(유한 termination factor `M`)를 무한장 LDPC-CC의 근사체로 사용.
4. 핵심 기법 1단: tail-biting 코드워드로 원본 무한장 convolutional 코드워드를 모듈러 index vector로 "wrap-back"시켜, tail-biting 최소거리가 unterminated 코드의 free distance 하한이 됨을 증명(Theorem 4/Corollary 6,8).
5. 핵심 식: free distance growth rate `alpha = lim d_free/nu`가 tail-biting LDPC block code의 minimum distance growth rate `delta(M)`으로 하한(`alpha >= delta(M)/M` 유사 관계, 식 (9)(12)(17)).
6. 핵심 기법 2단: edge spreading 설계규칙(Design Rule 1/2, "stairway" cutting vector)으로 decoding constraint length `nu`를 최소화해 성장률 `alpha`를 최적화.
7. 부수 트릭: irregular 앙상블(RJA/ARJA protograph)은 punctured variable node를 포함해 좋은 반복복호 threshold와 distance growth rate를 동시 확보.
8. 학습/최적화 절차: 각 앙상블에 대해 tail-biting termination factor `M`을 늘려가며 spectral shape function으로 `delta(M)`을 수치 계산, `M` 증가에 따른 leveling-off로 하한 수렴 확인.
9. 결과: (3,6)-regular 기준 block code growth rate `delta=0.0116` 대비 convolutional 하한 `alpha≈0.0250`으로 2배 이상 개선, ARJA 기반 예시는 constraint length 증가 시 waterfall이 반복복호 threshold 대비 0.13dB 이내로 근접, error floor 미관측(시뮬).
10. 한계: 전 과정이 앙상블 평균에 대한 점근적 하한 증명 + 소규모 BP 시뮬 검증뿐, 특정 HW용 QC-LDPC block code 설계나 디코더 구현은 다루지 않음, 부호는 QC-LDPC가 아닌 LDPC convolutional(spatially-coupled류) 계열.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| protograph/edge-spreading 기반 부호설계 이론 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 고정 QC-LDPC(base+shift) 구조라 convolutional edge-spreading 설계기법 자체는 이식 난이도 높음(H-matrix 구성법 상이) |
| trapping set 성장률/error-floor 하한 분석 | 대응 없음 | 이론적 앙상블 평균 분석으로 특정 코드의 trapping-set 구조 개선 기법이 아니라 직접 대응 모듈 없음 |
| min-sum/BP 반복복호 자체 | `decoder.cpp` `LDPC_Decoder()` | 논문은 표준 BP/sliding-window 복호를 시뮬 검증에만 사용, 복호 알고리즘 개선 기여 아님 |

적용 가치: **낮음** — LDPC convolutional code 앙상블의 점근적 distance/trapping-set 성장률을 증명하는 순수 정보이론 논문으로, QC-LDPC block code인 Prime ECC의 부호설계·디코더·HW 어느 레이어에도 직접 이식할 요소가 거의 없음(error-floor를 우려하는 설계 철학만 참고 가능).
