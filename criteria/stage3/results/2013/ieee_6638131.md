### B. 알고리즘 요약

1. 대상: NVMS(SSD/MLC 플래시)용 binary QC-LDPC, 프로토타입은 rate-0.896 `(M,N,b)=(6,58,160)`, 최대 codeword 9280bit, circulant b×b 순환구조.
2. 문제: 기존 QC-LDPC(TDMP) 디코더는 codeword 크기(페이지=수 KB)에 비례하는 FIFO 메모리 때문에 면적 비용이 과대, NVMS엔 통일 규격도 없음.
3. 핵심 가정: circulant 내 행 간 데이터 의존성이 없어 block row의 b개 행을 병렬 복호 가능.
4. 핵심 기법 1단: TDMP(layered, row-by-row) 복호 - 각 행 4단계(read&subtract → SISO decode → CN write-back → BN write-back), prior `ρ=γ(Ii)-λi`.
5. 핵심 식: `Λj=0.5·{XOR{sgn(ρi')}·min{|ρi'|}}` - normalized min-sum(0.5 scaling)로 SISO 함수 구현, tanh 없이 최소값·부호로 CN 갱신.
6. 핵심 기법 2단: re-arranged FIFO-free 아키텍처 - posterior γ와 prior ρ 둘 다 저장하지 않고 prior ρ만 저장, γ는 permuter/CNU로 직접 전달해 timing 완전 overlap.
7. 구현 디테일: fixed-point 분석으로 γ/ρ/Λ를 각각 8/7/6bit 양자화, scalable datapath는 M,N 경계조건 run-time programmable, CNU/BNU를 b값 enable로 on/off, arbitrary shift permuter.
8. 학습/최적화: 없음(구조·양자화 최적화, exhaustive 시뮬로 bit width 결정).
9. 결과: TSMC 90nm 1P9M, core 2.52mm², 138.8MHz, 393Mb/s@8iter, gate count 748K→609K(18.5% 절감), NAE 최고.
10. 한계: 정정능력 개선 없음(순수 면적/메모리 최적화), 처리량 393Mbps로 고속 NAND엔 부족, BER 곡선은 그림에만 존재.
