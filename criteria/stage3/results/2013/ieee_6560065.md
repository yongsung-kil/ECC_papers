### B. 알고리즘 요약

1. 시스템: binary regular QC-LDPC, 패리티검사행렬 `H`가 z×z circulant의 단일 행 `H=[H1...HL]`, col-weight `dv=3` 또는 `4`, design rate `R=(L-1)/L` (0.9~0.95).
2. 문제: 고rate 단·중길이 LDPC는 dense해 short cycle이 생기기 쉽고, 기존 고rate QC-LDPC는 rate/length 파라미터가 매우 제한적.
3. 모델: AWGN 채널, sum-product 복호(최대 100 iteration)로 성능 평가.
4. 핵심 기법 1: `(v,k,1)` Cyclic Difference Family(CDF)와 Perfect Difference Family(PDF)의 원소 `Bi`를 circulant shift 값 `sij`로 사용 (`sij=bij`).
5. 식 의미: CDF의 차분이 서로 distinct하면 길이-4 cycle이 없음 → 정리 3 `z≥2δmax+1`이면 girth≥6 보장.
6. 핵심 기법 2: PDF가 없는 `dv=3, L≡2·3 mod 4` 경우 hooked Skolem sequence로 `(6L+1,3,1)` CDF를 구성해 파라미터 커버리지 확장 (단 `z=6L+2`는 정리 4로 girth-6 불가 증명).
7. 구현 디테일: `z=dv(dv-1)L+1`일 때 H가 Steiner system incidence matrix가 되어 이론적 최소 code length 달성.
8. 랭크/인코딩: dv=3에서 full rank, dv=4에서 redundant row 1개(거의 full rank); 최소 1개 full-rank circulant 존재로 systematic generator `G` 단순 인코딩 가능.
9. 결과: rate-0.9167(1020,935)/0.9333(2115,1974)/0.9006(1640,1477) 부호가 AG/array/PEG LDPC 대비 BER 성능 동등 또는 소폭 우수, 최소길이 커버 개수는 CONV/ARR 대비 대폭 증가.
10. 한계: HW 미설계, 시뮬뿐. girth 6·최소거리 낮음 고정, 성능이 기존 대비 "거의 동일"에 그침.
