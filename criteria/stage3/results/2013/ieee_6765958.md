> 총평: 2D 자기기록 ISI 채널에 맞춘 irregular LDPC threshold 계산·부호 최적화 도구로, 채널·검출기(2D-DET) 의존이 강해 NAND binary QC-LDPC 이식성은 낮음.

### B. 알고리즘 요약

1. 대상은 고밀도 자기기록(BPMR/TDMR)의 2D ISI+AWGN 채널, rate 8/9 irregular LDPC, 길이 `Nc`=4608/9216/13824.
2. Density Evolution은 ISI/AWGN 채널에서 계산 부담이 커, EXIT 곡선 피팅으로 threshold를 구하는 저비용 대안이 필요.
3. 채널 모델: `LM×LN` 채널행렬 `H`로 down-track ISI와 cross-track ITI를 합친 2D ISI, 등가잡음을 중심극한정리로 Gaussian 근사.
4. 핵심 기법: 결합 2D-DET+VND의 EXIT 함수 `IE,VND(IA)`를 CND EXIT 곡선 `IE,CND`에 교차 없이 최대한 근접시키는 최소 `Eb/N0`를 threshold `s*`로 정의.
5. 식 (10)~(13): `λi`(VN degree 분포)로 가중한 VND transfer 곡선과 CND 곡선의 cost `Fcost`를 계산, `s*=sup{s, Fcost≤Tcost}`.
6. 확장: 최적 symbol-based BCJR 2D-DET와 저복잡도 IRCSDFA-GA 2D-DET 두 검출기를 모두 threshold·BER에 적용.
7. 구현 트릭: 2D-DET EXIT 곡선을 3차 다항식으로 근사(TABLE I 계수), IRCSDFA-GA는 GA로 trellis 상태공간을 `2^(LN-1)`로 축소.
8. 최적화: Algorithm 1이 `Eb/N0`를 `Δs`씩 증가시키며 `Fcost≤Tcost` 만족 최소값 탐색.
9. 결과: 1Tb/in² threshold `s*=3.5dB`, 4Tb/in² `s*=4.0dB`; IRCSDFA-GA는 BCJR 대비 threshold 0.23dB·BER 0.3dB(@10⁻⁵) 열화, threshold gap이 BER gap과 일치.
10. 한계: HW 미설계, 2D 자기기록 채널·검출기 전용, QC/protograph 적용은 future work.
