### B. 알고리즘 요약

1. 대상은 pairwise Markov random field에서의 marginalization; 각 변수는 `d`개 상태를 가지며 BP 메시지 업데이트는 `d×d` 호환행렬-벡터 곱으로 반복당·edge당 `O(d^2)`.
2. 문제: 이미지 비전 등에서 상태차원 `d`가 수천 규모라 BP의 `O(d^2)` 계산과 `d`개 실수 전송(통신)이 과도.
3. 핵심 착안: edge 메시지 업데이트가 정규화 호환행렬 열들의 기댓값(식 (7))으로 표현됨 → 전체 기댓값 대신 확률(식 (9))로 열 하나만 랜덤 추출.
4. SBP 업데이트(식 (10)): step size `γ_t`로 damping된 랜덤 열 갱신. 반복당 계산 `O(d)`, edge당 전송은 열 index `log d` bit만.
5. 열/정규화상수 선계산(offline)으로 저장, 이후 랜덤 index 생성은 uniform 난수 1개로 수행.
6. 이론 보증: tree에서 거의 확실 수렴(Thm 1, nilpotent 행렬로 `l∞` 오차 `O(1/t)` 감쇠), contraction 조건 만족하는 일반 그래프에서 정규화 MSE `O(1/t)` 감쇠(Thm 2).
7. Robbins-Monro 확률근사 + Azuma-Hoeffding/martingale 집중부등식으로 고확률 오차 경계 유도.
8. Potts model에 대한 contractivity 충분조건(step size `γ` 구간)을 node/edge potential로 명시(Prop 1).
9. 결과: SBP는 반복당 시간이 `d`에 선형(BP는 `d^2`), 큰 `d`에서 유의미 speedup; 이미지 denoising(200×200, 256 gray)·스테레오 비전에서 BP 대비 짧은 시간에 동등~약간 우수. 단 SBP는 반복수가 더 필요.
10. 한계: HW 미설계, 순수 이론+시뮬(비전 응용). LDPC는 배경 예시일 뿐이며 nonbinary LDPC factor graph 확장은 future work로만 언급.
