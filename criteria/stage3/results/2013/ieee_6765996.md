### B. 알고리즘 요약

1. 대상은 BI-AWGN 채널의 irregular LDPC 코드로, 변수·검사노드 차수분포 `λ(x)`, `ρ(x)`와 threshold `s*`를 설계.
2. 문제: density evolution 직접 계산은 복잡도가 높아 실용적 최적화가 어렵고, 기존 [8][9]는 구현 정보가 부족.
3. 모델: 무한길이·cycle-free 가정 하 Gaussian approximation으로 LLR 메시지를 가우시안으로 근사, 평균 `m`만 추적(분산 `2m`).
4. 핵심 기법 1단: differential evolution(휴리스틱)으로 given rate에서 threshold `s*`를 최소화하는 `ρ(x)`,`λ(x)` 탐색(mutation `xbest+(xni-xnj)` + selection).
5. 핵심 식: CN→VN 메시지 평균 갱신 `t(l)=g(s,t(l-1))`, 수렴조건 `t<g(s,t)`가 `pe→0`을 보장.
6. 핵심 기법 2단: given `s*`에서 `ρ(x)`, `λ(x)`를 교대로 simplex(LP)로 최적화하는 iterative simplex로 code rate `R` 향상.
7. 구현 디테일: `ρ(x)=ρx^(dc-1)+(1-ρ)x^dc` 형태 가정 시 code rate 문제가 LP로 선형화되고, r/t 부등식을 양자화.
8. 최적화 절차: Fig.4 — DE로 저threshold 분포 확보 후 iterative simplex로 rate 향상, 동일 `(Es/N0)*` 유지.
9. 결과: 기존 Richardson [8] 분포 대비 code rate 향상으로 `(Eb/N0)*`가 감소, dv=7,10,12,14,15에 대한 새 차수분포 표 제시.
10. 한계: ensemble 수준 이론 결과로 유한길이 코드 구성/디코더/HW 검증이 전혀 없음.
