### B. 알고리즘 요약

1. 대상: QC protograph LDPC 부호. base graph(작은 Tanner 그래프)를 circulant permutation으로 `N`-lifting하여 lifted 부호를 얻는 구조.
2. 문제: 주어진 base graph·lifting degree `p`에서 달성 가능한 girth의 상한(또는 특정 girth에 필요한 `p` 하한)을 기존 inevitable-cycle 기반보다 더 tight하게 규정.
3. 핵심 도구: base graph의 tailless backtrackless closed walk의 permutation shift(식 `(1)`) 의존관계를 분석. shift가 0이면 lifted 그래프에 cycle 생성.
4. girth≥6 조건(Lemma 2): `p > m·(m-1)/2` 형태, base matrix 두 열/행 간 최대 겹침 `η`로 결정.
5. girth≥8 조건(Lemma 3,4): 한 edge를 공유하는 4-cycle 수 `g_e` 기반, `p ≥ 2·g_e+1` 류. Fully connected base엔 폐형식.
6. girth≥10 조건(Lemma 6,7): 한 node를 공유하는 4-cycle 수 기반 하한. TSF(155,64) 예: 이 base로 girth>8 불가(상한 8) 증명.
7. shift 의존관계(식 `(3),(5)`): 4-cycle들의 shift가 서로 다르고 서로의 additive inverse와도 달라야 더 큰 girth 확보.
8. 새 구성(§V): fully connected base(좌차수 3,4,5,6)로 girth 8 QC-LDPC 명시적 permutation shift matrix(식 `(6)~(9)`) 제시, rate 예: 좌차수별 상이.
9. 부가 결과: 좌차수 3 girth-8 구성이 minimum distance≥8(Lemma 10), 2행 permute 시 ≥10(Lemma 11) 보장 — array-based LDPC(d_min=6)보다 우수.
10. 한계: 순수 이론(BER/error-floor 시뮬 없음), HW 미설계, girth≥12 유도는 tedious.
