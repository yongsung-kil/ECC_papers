### B. 알고리즘 요약

1. 시스템은 Free-Space Optical(FSO) 링크용 네트워크 브리지로, rate-1/2 regular LDPC(H=8x16, length 16)를 encoder/decoder로 Virtex-5 FPGA(ML505)에 실장.
2. 문제: FSO/hybrid 링크의 single-bit error를 저복잡도 HW로 확실히 정정 -> 소형 부호로 BER=0 목표.
3. 모델: `c=kG`로 인코딩, `Hc^T=0`으로 검증, `s=Hy^T` 신드롬으로 오류비트 위치 판단(hard-decision).
4. 핵심기법: bit-flipping 복호 - 신드롬이 0이 아니면 오류비트를 flip, 신드롬 0 또는 최대반복(=열 수)까지 반복.
5. 식 의미: 신드롬 `s=Hy^T`가 위반된 check row를 지목 -> 해당 비트 flip으로 single-bit error 복원.
6. 부호구성: MATLAB 랜덤분포로 H 생성 후 short cycle 제거, full-rank화, `H=[P^T I]`로 변환해 `G=[I P]` 도출.
7. 구현 디테일: encoder는 4-state, decoder는 6-state 유한상태기계(VHDL), 행렬계수는 Memory Block Generator IP core LUT에 저장.
8. 결과: 모든 single-bit error에서 BER=0(MATLAB 전수 시뮬), encoder bit rate 57.14~81.57 Mbps@125MHz, 단일오류 복호 최소 176 ns(22 clock), 무오류 시 11 clock.
9. HW 자원: Virtex-5에서 slice 1%, LUT 30% 사용.
10. 한계: 8x16 토이 부호·랜덤탐색 설계, hard bit-flipping만, 새 알고리즘/부호설계 기여 없음, FSO 특화.
