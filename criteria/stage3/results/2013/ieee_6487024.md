### B. 알고리즘 요약

1. 시스템: DVB-T2 방송용 LDPC 코드(`N=64800`, rate `1/3`), AWGN 채널, BPSK 가정.
2. 문제: 표준 min-sum(및 scaled/offset 변형)은 `ln cosh(x)`를 `x-ln2` 1차 근사로 대체해 구현은 쉽지만 낮은 LLR(`x≈0`) 영역에서 최대 `ln2(≈0.693)` 오차로 성능 저하.
3. 가정: check node update는 `CHK(Λ1,Λ2)=ln cosh((Λ1+Λ2)/2)-ln cosh((Λ1-Λ2)/2)`(식 2)이며, 채널 LLR `Λ=2r/σ²`, LLR은 uniform 분포로 가정.
4. 핵심 기법: `ln cosh(x)`를 2단 계단함수로 근사(`x if x≤ln2, else x-ln2`, 식 8)해 min-sum 업데이트에 4구간 조건부 offset `C_offset=σ_op²·ln2/4` 적용(식 9).
5. 핵심 식: `CHK(Λ1,Λ2)≈sgn(r1)sgn(r2)[min(|r1|,|r2|)∓C_offset]` (구간별 `|r1±r2|` 대소에 따라 offset 부호/적용 여부 결정) — 기존 offset-MSA의 고정 `β`보다 정밀한 조건부 보정.
6. 확장: 없음(check node update 단일 개선).
7. 부수 트릭: 곱셈/나눗셈/LUT 메모리 불필요, 정수 덧셈·비교·비트시프트만으로 구현 가능; SNR/채널 정보 불필요(σ_op는 사전 오프라인 시뮬로 고정값 결정).
8. 최적화 절차: 별도 학습 없음, `σ_op`(동작점 노이즈 분산)를 컴퓨터 시뮬레이션으로 탐색해 상수로 고정.
9. 결과: iteration 50회 고정, BER=1e-5에서 제안기법이 SPA 대비 0.02dB, Offset-MSA 대비 0.14dB, Scaled-MSA 대비 0.23dB 우수(Shannon limit -0.495dB 대비 근접).
10. 한계: HW 미설계(gatecount/throughput 없음), 단일 채널(AWGN)·단일 코드(rate 1/3, N=64800) 시뮬 검증뿐, `σ_op` 최적값을 사전 탐색해야 하는 튜닝 부담.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| CN min-sum 연산(check node update 근사) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | 논문의 2-step `ln cosh` 근사 check node update는 Prime ECC의 min-sum CN 연산과 직접 대응되는 후보 개선점 |
| offset/scaled min-sum과의 성능 비교 | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | Prime ECC가 이미 min-sum 기반이므로 대체 알고리즘으로서 이터레이션 루프에 삽입 검토 가능 |
| 채널정보(SNR) 불필요 설계 철학 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | soft-read LLR 양자화 threshold 설계와 관련되나, 논문은 AWGN 연속값 기반이라 NAND 이산 soft-read LLR 테이블 적용 시 재검증 필요 |

적용 가치: **중간** — min-sum CN 업데이트 근사 개선이라는 점에서 Prime ECC의 디코더 알고리즘 레이어(`decoder.cpp`)와 직접 대응되나, 연속 LLR/AWGN 가정과 곱셈 없는 정수 연산이라는 장점이 NAND의 이산 soft-read LLR 양자화 체계와 얼마나 정합되는지 재검증이 필요하고 HW 검증(gatecount/throughput)이 전무함.
