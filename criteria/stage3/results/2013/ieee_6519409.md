### B. 알고리즘 요약

1. `(v,b,3,1)`-packing design의 incidence matrix `Mm`(girth 6)을 mother matrix로 삼아 column-weight 3 QC-LDPC를 구성한다.
2. 문제: 기존 voltage-graph 기반 girth-20 QC-LDPC는 부호 길이가 매우 길어 비효율적 — 더 짧은 base matrix가 필요.
3. `m≥7`에 대해 홀/짝 정수 집합 `Om,Em`으로 관계 `R`을 정의, 동치류 `Ri`로 블록집합 `Bm`(packing) 생성.
4. `Bm`의 incidence matrix `Mm`(크기 `v×b`, `v=3⌊m/2⌋`, `b=⌊m/2⌋⌈m/2⌉`)의 각 1을 순환치환행렬 `I, I^{sj}`로 lifting.
5. slope vector `S`와 lifting size `N`으로 `H(m,N,S)` 생성 → QC-LDPC `C(m,N,S)`, rate `R=1-(3⌊m/2⌋)/(⌊m/2⌋⌈m/2⌉)`.
6. Theorem 3.1: `Mm`이 금지 부분행렬 `P12,P14,P16,P18` 및 전치를 포함하지 않아 최대 girth `= 20` 보장.
7. slope vector는 girth 조건 `Σs ≡ 0 (mod N)` 회피하도록 결정론적 알고리즘(column-by-column 탐색)으로 생성, `N`은 컴퓨터 탐색.
8. 결과: rate 0.25/0.4/0.5에서 길이 158240/6250000/72000000, voltage-graph 대비 각각 76/4.9/7.2배 짧음.
9. BER 시뮬(sum-product, 최대 iter 20)에서 girth-20 코드가 random/PEG/RA/QPEG/S-STS(9)보다 우수.
10. 한계: column-weight 3 저rate 위주, HW 미설계, 채널은 AWGN 시뮬만.
