### B. 알고리즘 요약

1. 대상: 실수 AWGN 채널, streaming 데이터 전송용 신규 변조 포맷(FEC 부호는 별도, LDPCCC 권장).
2. 문제: 기존 PAM/QAM·multi-level coding은 용량 근접에 다중 rate 최적화·interleaver·turbo 피드백 필요 → 단순하며 용량달성하는 변조를 원함.
3. 핵심 구성: 이진 부호화열 `v ∈ {−1,1}`을 multiple convolutional permutor(LDPCCC의 semi-infinite H, 행 M개·열 L개 ones)로 복제·치환.
4. 변조신호 `x = vH`를 실수영역(모듈로-2 아님) 합으로 계산 → L개 데이터 스트림이 offset을 두고 superposition(=spatial coupling).
5. 각 bit `v_j`는 M번 복제되어 `T(j)` 위치에 퍼지고, 수신 `y = sqrt(P/L)x + n`.
6. 복호는 2단계: (1)반복 bit 추정 + 병렬 간섭제거, (2)FEC 디코딩(피드백 루프 불필요).
7. 추정: 잡음+간섭 벡터 `ξ_j`를 CLT로 Gaussian 근사, MMSE 필터 `w_j=(I+h*R_j^-1 h)^-1 h*R_j^-1`로 `z_j` 산출, `v̂_j = tanh(z_j γ_j)`.
8. 성능분석: SNR 재귀 `γ_{i,t}=(P/L)Σ 1/σ^2`, 간섭전력 `σ^2_{i,t}=α(P/L)Σg(γ_{i-1})+1`(`g`=MMSE 함수), 디코딩 성공조건 `γ_{I,t}>θ`(FEC threshold), threshold saturation 성립.
9. 결과: modulation load α∈[2,6]에서 통신 rate가 2^M-PAM 용량 초과, SNR 커질수록 AWGN 용량과의 gap 소멸(Thm.1, `α*≈C_AWGN`, 이진부호 rate≈1).
10. 한계: 순수 이론(시뮬 곡선만), HW 미설계; ECC 디코더/부호설계가 아니라 변조 포맷이며 NAND와 무관.

### C. Prime ECC 관련 모듈 핀포인트

대상이 other(변조 포맷)이므로 C섹션 전부 N/A.

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| spatially coupled 변조 / MMSE 간섭제거 | 대응 없음 | 실수영역 변조·통신 채널 기법으로 binary LDPC 복호와 무관 |
| LDPCCC permutor(H) 구조 | 대응 없음 | QC-LDPC block 부호(Prime ECC)와 다른 convolutional 구조 |

적용 가치: **낮음** — AWGN 통신용 변조 이론으로, NAND binary QC-LDPC ECC 디코더/부호설계에 이식할 요소가 없다.
