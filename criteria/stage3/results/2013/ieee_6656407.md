### B. 알고리즘 요약

1. 대상은 100+ Gb/s 광전송(PDM-QPSK, OSNR 채널)용 soft-decision FEC, QC-LDPC(rate 0.75~0.81, 길이 16908~48016).
2. 문제: 정칙 QC-LDPC는 short-cycle 회피가 제한적이고 error-floor가 발생 → 더 유연한 설계 필요.
3. 핵심 관찰: convolutional/spatially-coupled/staircase LDPC는 모두 하나의 template QC-LDPC의 H-matrix 재배열로 유도됨 (permutation matrix `P`의 거듭제곱 `S[i]` 선택).
4. cycle 회피: 길이 `2k` cycle 조건식 `S_{i1,j1}+...+S_{ik,jk} = S_{i1,j2}+...+S_{ik,j1} mod B`를 검사, 위배 정수를 집합 S에서 제거하며 탐색.
5. 부호 파라미터: rate `R ≥ 1-r/|S|`, 길이 `|S|·B`, `B`는 소수.
6. 확장(핵심): pairwise balanced design PBD(v,K,λ)로 비정칙 구조를 정밀 제어 — 블록 크기 집합 K가 col-block weight 결정 (지시함수 `I(bij)`로 template submatrix 보존/제거).
7. 예: PBD(6,{2,3},1) → 앞 3개 block-column은 weight 3, 뒤 3개는 weight 2인 비정칙 QC-LDPC.
8. PBD는 대수적 구조라 비정칙 부호의 설계 복잡도가 정칙 QC-LDPC와 동등 이하.
9. 결과: girth-8 비정칙 QC-LDPC(28180,21135)는 정칙 대비 error-floor 없고 NCG 9.4 dB@BER 1e-8(예상 12.4 dB@1e-15), pragmatic 정칙 대비 0.46 dB 우위, min-sum-plus-correction 50 iter.
10. 한계: HW 미설계, 시뮬만, 광통신 초장문 부호·고 iteration으로 NAND 컨트롤러 제약과 무관.
