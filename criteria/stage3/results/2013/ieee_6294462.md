### B. 알고리즘 요약

1. 대상은 LDPC 부호 복호가 아니라 long-span/sparse multipath ISI 채널(수중음향·HDTV 등)의 심볼 검출기로, QPSK(M=4), 채널 30탭을 가정.
2. 문제: 최적 ML(Viterbi) 검출은 채널길이에 지수적 복잡도라 긴 sparse 채널에 비현실적.
3. 채널모델은 `y[k]=Σh[i]x[k-i]+w[k]` (복소 AWGN), sparse 계수(`Dh << Lh`).
4. 핵심 기법 1단 PRE(partial response equalizer): FIR 필터로 CIR을 nonzero 계수 소수(`Dg`개)만 남긴 TIR `g`로 등화, 계수는 dual-LMS(MMSE)로 적응 계산(`Lf≈3Lh=100`).
5. 핵심 기법 2단 BP 검출: TIR로 만든 Toeplitz factor graph에서 CTV/VTC 메시지를 log-sum-exp(Jacobian 근사)으로 교대 갱신, `x̂[n]=ψ(argmax Λn[i])`.
6. BP CTV 연산은 Euclidean 공간이라 `3·M^Dg` 곱셈 필요(Dg=3,M=4 → 192 곱셈)로 `Dg`에 지수적, 그래서 `Dg`를 작게 유지.
7. HW: layered(sequential) BP를 pipeline화(layer 간 무의존), sliding-window cache + CTV/LLR-sum 메모리, folded FIR로 PRE 구현. 8-bit 고정소수점(float 대비 0.2dB).
8. LMS는 프레임당 1회만 필요해 DSP(TMS320C6748)에서 처리, PRE/BP는 ASIC.
9. 결과: Dg=3,nit=5에서 linear equalizer 대비 SER 10^-5에서 3.5dB 개선, TSMC 90nm에서 206Mb/s, 3.162mm2, 1.096W.
10. 한계: 채널행렬이 시변·non-binary(Euclidean)라 QC-LDPC식 병렬화 불가, LDPC ECC 복호가 아니며 부호설계·정정능력 카테고리 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP 심볼검출(CTV/VTC, log-sum-exp) | 대응 없음 | Euclidean 공간 검출기로 GF(2) min-sum 복호와 상이 |
| layered/sequential BP pipeline | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | 개념(layer 수렴가속)만 유사, 검출기 전용 구현 |
| sliding-window cache + 메모리 라우팅 | 대응 없음 | Toeplitz 시변행렬 전용 인터커넥트 |
| PRE(FIR/LMS 등화) | 대응 없음 | 채널 등화는 우리 ECC 파이프라인에 없음 |

적용 가치: **낮음** — LDPC 복호가 아닌 sparse ISI 채널 심볼검출기로, BP·layered 개념은 공유하나 non-binary Euclidean 연산·시변 Toeplitz 구조라 NAND binary QC-LDPC 디코더/HW에 떼어 쓸 요소가 없다.
