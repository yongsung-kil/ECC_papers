### B. 알고리즘 요약

1. 시스템: 단일반송파 SISO UWB 전송, 주파수영역 등화기 + soft demapper + LDPC 디코더의 turbo equalization 수신기, ARJA rate-1/2 기반.
2. 문제: block-code 최적화 protograph가 ISI 증가 시 성능 저하 → convolutional 확장(LDPCC)으로 threshold 개선하되, 부호 구성 파라미터 영향이 미정.
3. 채널 모델: block-fading multipath, 지수감쇠 PDP `h(i)=exp(-(i+1)/10)`, i=0..9 로 significant ISI 유발, AWGN 분산 σ².
4. 핵심 기법: base matrix `B`(b_c×b_v)를 syndrome former memory `m_s`로 부분행렬 `B_0..B_{m_s}`로 분해해 band matrix로 unwrapping, termination length `L`로 잘라 terminated convolutional protograph 생성.
5. 핵심 식: `N = b_v·L·ω` (식7) — 부호길이는 permutation size ω, L, protograph 폭 b_v의 곱; rate는 `R_L=1-((L+m_s)/L)(1-R)` (식6).
6. 확장: copy-and-permute로 protograph→Tanner graph, permutation matrix 크기 `ω`가 code length를 직접 결정(식2 `N=b_v·ω`).
7. 구현 디테일: permutation matrix는 PEG/ACE 알고리즘으로 결정, punctured VN 포함 rate 계산.
8. 분석 절차: EXIT-chart(Monte-Carlo 측정 + protograph용 density evolution 두 방법)로 threshold 예측, m_s 증가는 낮은 a-priori 영역 extrinsic↑, L 증가는 반대.
9. 결과: convolutional 변형이 block/standard convolutional 대비 우수, ω 증가 시 low-BER 영역 성능↑(decay point는 L에 종속), 동일 block length 비교(예 L=10·ω=300 vs L=30·ω=100)로 rate/성능 trade-off 정량화.
10. 한계: HW 미설계, DE는 무한 blocklength 가정, 무선 ISI 채널 전용이며 부호length 절대수치·gate/throughput 없음.
