### B. 알고리즘 요약

1. 시스템: randomized linear network coding 하 잡음 네트워크. 패킷은 헤더(systematic 부분)로 선형 인코딩 계수를 기록, 오류는 codeword의 저랭크(rank `t`) 섭동으로 모델링.
2. 문제: 기존 error-trap(Silva-Kschischang-Kötter) 방식은 최소 rank distance가 1이라 단일 정보심볼 오류를 못 고침. 용량달성 + 저복잡도 부호가 필요.
3. 채널 모델: `Y = X + T`, `T`는 GF(q)상 rank `t` 이하 균등랜덤 행렬(systematic network coding channel `C(n,t)`).
4. 핵심기법: 변수노드 degree 2 고정 sparse-graph(Tanner graph)로 codeword를 제약, 각 nonzero를 GF(q) 가역행렬로 lift한 부호.
5. 핵심식: 각 check node에서 affine subspace 교차/합으로 message 갱신 `V_{i→a} = ∩ (...)`; 부호의 최소거리가 `n→∞`에서 발산함을 증명해 정정력 확보.
6. 확장: message가 affine subspace이며 (offset vector, basis matrix) 쌍으로 표현, Gaussian elimination으로 교차·합 연산.
7. 구현 디테일: 인코딩은 sparse 행렬을 상삼각화해 `O(n^{3/2})`로 축소; degree profile을 rescaled density evolution으로 최적화.
8. 최적화: `nt`가 `l`의 정수배일 때 density evolution이 스칼라 재귀로 붕괴 → 최적 degree distribution `λ` 해석적 도출(Lemma 4.3).
9. 결과: 최적화 부호가 채널 용량을 달성, 복호 복잡도는 정보비트 수에 대해 제곱·오류확률에 대해 sublogarithmic.
10. 한계: 순수 이론(HW·시뮬 없음), 채널이 GF(q) 저랭크 섭동이라 binary NAND LDPC와 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| rank-metric affine-subspace 메시지패싱 복호 | 대응 없음 | binary min-sum LDPC 복호와 연산이 다름 |
| GF(q)-lift sparse-graph 부호 구성 | 대응 없음 | non-binary·네트워크코딩 전용 |
| degree-profile density evolution 최적화 | 대응 없음 | 부호설계 대상 채널이 무관 |

적용 가치: 낮음 — non-binary rank-metric 네트워크코딩 채널 전용 이론으로 NAND binary QC-LDPC ECC에 떼어 쓸 요소가 없음.
