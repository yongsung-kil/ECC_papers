> 총평: HDD PMR 채널용 단순 array-type H-matrix 구성법으로, 소수 GF(103) 기반이라 Prime ECC의 binary QC-LDPC 구조에 직접 이식 곤란하고 기여도 교과서 수준.

### B. 알고리즘 요약

1. 대상: HDD PMR 채널용 rate 0.9 LDPC, PR2 target `[1 2 1]` + SISO(SOVA) 등화기와 연접, AWGN 및 PMR 채널에서 평가.
2. 문제: 부호 성능은 H-matrix 구조에 크게 의존 → array/QC 대비 더 단순한 H-matrix 구성이 필요.
3. 모델: Fan의 array code → Eleftheriou/Olcer의 modified array code(MAC)를 채택, permutation matrix `σ`는 항등행렬 `I`의 1칸 오른쪽 shift.
4. 핵심 기법: 2개 결정변수 `a`, `b`(GF(p), a,b∈{2..p-1})로 permutation 지수를 정의해 H를 구성 `H = [a^i · b^j]` 블록 배열(식 4,6).
5. 식 의미: `j=4, k=40, p=103`으로 설정, 부호 rate `R = 1 - (pj-j+1)/p²` ≈ 0.9로 고rate 구성.
6. 확장: Tanner의 circulant 기반 대수적 구성을 MAC에 적응(random-like 구조).
7. 복호: BP(식 1, tanh 기반)와 Min-Sum(식 2, abs·sgn 근사) 두 디코더로 벤치마크, 5/20 iteration 비교.
8. 시스템: LDPC 외부부호 + SOVA 내부부호가 soft-info 교환(inner 3 iter, outer 5 iter).
9. 결과: AWGN에서 iteration↑ 시 BER/BLER 개선, SNR>3dB에서 BP가 MS보다 우수; PMR 채널에서는 iteration 효과 미미. 수치 BER 곡선은 그림에만 존재(미상).
10. 한계: HW 미설계, error-floor 분석 없음, MS 성능이 BP보다 열위(MS 개선을 향후과제로 제시).
