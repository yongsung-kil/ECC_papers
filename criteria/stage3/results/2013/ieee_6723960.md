### B. 알고리즘 요약

1. 대상은 irregular QC-LDPC 코드(WiMax rate-1/2, 예: (2304,1152), z0=96)의 base matrix 구조가 반복복호 성능에 미치는 영향 분석.
2. 문제: 같은 degree distribution pair라도 base matrix 구조에 따라 waterfall 성능이 달라지나, 기존 density evolution(DE)은 VN-CN 연결성(H_b 구조)을 반영 못함.
3. 모델: AWGN + BPSK 가정, base model matrix `H_bm`(원소 {-1,0,...,z-1})를 z×z circulant permutation으로 확장, `H_b`가 연결성 특성화.
4. 핵심 기법 PEXIT chart: base graph `G_b`의 각 edge에 두 mutual information(VN→CN, CN→VN)을 부여해 개별 edge별 밀도 진화를 추적.
5. VND EXIT 함수 `I_VND,i = J(sqrt(Σ_{j≠i}[J^-1(I_CND,j)]^2 + [J^-1(I_ch)]^2))` (식 2)로 edge별 외부정보 갱신, 모든 MI가 1로 수렴하면 복호 성공.
6. CND EXIT는 duality로 `I_CND,j ≈ 1 - J(...)` (식 3) 근사.
7. 실험: degree distribution (식 4) 고정 하에 6×12(전수 604개), 12×24·18×36(각 5000개 샘플) base matrix의 threshold 분포 생성.
8. 결과1: base matrix 차원 증가 시 threshold 평균 감소(0.7908→0.7782→0.7598dB), 18×36에서 최적 0.609dB로 ensemble threshold 0.641dB 능가 가능.
9. 결과2: WiMax rate-1/2 개선 base matrix 설계 시 최적 threshold(0.681dB) 코드 C1은 waterfall 우수하나 error-floor는 오히려 악화(C2가 더 나은 trade-off).
10. 한계: threshold 하나로 error-floor 예측 불가, base matrix 전수 열거법 부재, HW·NAND 미설계, 시뮬(BER≥10^-6)만 검증.
