### B. 알고리즘 요약 (10줄 내외)

1. 대상은 Gray-labelled 2^n-QAM(16~1024-QAM)에서 BICM 시스템의 rate `(n-1)/n` LDPC 부호이며, 각 비트 위치(bit-level)마다 서로 다른 bit-channel capacity를 갖는다.
2. 기존 irregular LDPC는 균일한 채널을 가정해 모든 codeword bit에 동일한 방식으로 최적화되지만, 고차 변조의 bit-channel은 output-symmetric이 아니고 bit-level별 신뢰도가 크게 다르다는 것이 문제(Fig. 1, 16-QAM에서 b0/b2 vs b1/b3 LLR 평균 차이).
3. 핵심 가정은 각 bit-channel(edge-type) `k`에 서로 다른 variable node degree 분포 `λ_i^k`를 할당해 신뢰도 낮은 채널엔 고차수(반복코드 보호강함) variable node를, 신뢰도 높은 채널엔 저차수 node를 배정한다는 것.
4. 핵심 기법은 multi-edge-type(MET) LDPC 앙상블 파라미터화(식 1, 5): edge-type마다 별도 degree distribution `L(x1,...,xT,r1,...,rS)`, `R(x1,...,xT)`를 정의하고 socket-count(식 2)·channel-ratio(식 3) 제약을 둔다.
5. 핵심 식은 다차원 EXIT vector field `I^l = f_{I0}(I^l-1)`(식 7~10)로, 단일 스칼라 mutual information 대신 edge-type별 벡터 `(I1,...,IT)`를 추적해 MET 앙상블의 수렴 궤적을 근사한다.
6. 확장 기법으로 backward-difference vector field `∇f_{I0}(I)`(식 13)를 이용한 sufficient convergence condition(Theorem 4.1, `‖∇f_{I0}(I)‖2 > 0 ∀I∈D̂`)을 유도해 이를 code rate 최대화 최적화 문제의 quadratic 제약으로 사용.
7. 구현 디테일: check node degree를 T개 edge-type 사이에 균등 분할(2 edge-type일 때 대칭분할이 최적)하는 concentrated check degree 단순화로 최적화 변수 수를 크게 줄임.
8. 최적화 절차: MATLAB `fmincon`(interior-point + log-barrier)으로 200 step 이내 수렴, 고차 변조(64/256/1024-QAM)는 저차 앙상블에 새 edge-type을 추가하는 확장(nesting) 절차로 탐색공간을 축소.
9. 결과: 16-QAM rate 3/4에서 capacity와 0.199dB gap(기존 최선 대비 동일), block-length 16K에서 threshold 대비 0.52dB로 BER 10^-6 달성; 동일 정보 블록길이에서 AR4JA 대비 0.1dB 우수, 동일 성능을 더 짧은 블록길이로 달성해 구현 복잡도(node/edge 수) 절감.
10. 한계: HW 설계·게이트카운트·throughput 언급 없는 순수 이론+시뮬(density evolution/PEG 구성) 논문이며, 대상 채널이 고차 QAM 무선통신으로 NAND의 단일 binary(hard 1-bit read) 채널 특성과 근본적으로 다르고 FER error-floor(~10^-4)는 외부 BCH concatenation으로 해결을 제안할 뿐 본 논문 범위 밖.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Multi-edge-type 부호 구성(다중 bit-channel별 degree 분포) | `ecc_top.cpp` `Load_PCM()` | 특정 QC-LDPC 고정 구조인 Prime ECC와 달리 다중 edge-type 앙상블 구조 자체는 재검증 부담이 큰 새 부호설계이며 이식 난이도 높음 |
| 다차원 EXIT vector field 최적화/수렴조건 | 대응 없음 | density evolution 기반 오프라인 앙상블 설계 도구로 Prime ECC의 HW 디코더 루프와 무관 |
| bit-channel 신뢰도별 LLR 차등 보호 아이디어 | `channel.cpp` `Set_LLR_Th()`, `ecc_data.h` `PARAM_LLR` | NAND MSB/LSB page별 신뢰도 차이에 대한 개념적 유사성은 있으나 본 논문은 QAM 변조 특화라 직접 대응 함수 없음 |

> 적용 가치: **낮음** — 고차 QAM 변조를 위한 부호구성·EXIT 최적화 이론이며 NAND binary read channel과 채널 모델이 다르고 특정 QC-LDPC 구조가 고정된 Prime ECC에 새 부호설계를 이식하는 것은 재검증 부담이 커 활용도가 낮음.
