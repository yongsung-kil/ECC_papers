### B. 알고리즘 요약

1. 시스템: base matrix `B`(J×L) + circulant shift `p_{j,l}`로 `p×p` CPM을 배치한 binary QC-LDPC, 예제 `QC(252,504)`(p=84,J=3,L=6), `QC(504,1008)`(p=168).
2. 문제: girth가 작으면 short-cycle이 message-passing 수렴속도·성능을 저하 → 기존 직접구성/hill-climbing은 유연성 낮거나 복잡도 큼.
3. 모델: BPSK, AWGN 채널, sum-product 반복복호(최대 100 iteration).
4. 핵심기법 1단(cycle-set 탐색): mother matrix를 J개 section으로 나눠 cycle이 cycle-set `h(i)` 형태로 반복됨을 이용, tree graph 탐색으로 length-2l cycle을 저복잡도로 검출.
5. 핵심식 의미: cycle 존재조건 `∑(h_{jv,kv}-h_{jv,kv+1})=0 (mod p)`로 shift값 합이 0이면 길이 2l cycle 형성.
6. 핵심기법 2단(제거): 가장 많은 cycle이 지나는 CPM을 auxiliary matrix `F`(m×n)의 최대 column weight로 찾아 그 CPM의 shift값을 조정.
7. 구현 디테일: dead set `D`(cycle 유발 shift값 집합)를 (5)로 계산, candidate set `S=GF(p)-D`에서 값을 골라 length-4 cycle 회피, QC 구조 보존.
8. 최적화 절차: length-4 먼저 제거 후 length-6로 girth 목표까지 반복, `F`가 전부 0이 될 때까지 진행.
9. 결과: 복잡도가 CPM 크기 p와 무관(기존 [4]는 p 비례); `QC(504,1008)`이 MacKay random 대비 BER `10^-5`에서 0.1dB, WER `10^-3`에서 0.1dB 개선.
10. 한계: HW 미설계, 시뮬만, 개선폭 0.1dB로 작고 저rate short-block 예제뿐.
