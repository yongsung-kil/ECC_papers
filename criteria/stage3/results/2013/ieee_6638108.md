> 총평: MLC NAND retention/PI 오류 비대칭을 이용한 BCH/Hamming+even-parity 곱부호 HW 논문으로 NAND는 직접 관련이나 min-sum LDPC와 무관해 알고리즘 이식 가치는 낮음. 다만 오류타입 비대칭 통찰은 LLR 설계 참고용.

### B. 알고리즘 요약

1. 시스템: 2KB/page 2bit MLC NAND Flash. 오류를 retention(>1일 지배) vs program interference(PI, <1일 지배) 두 유형으로 구분.
2. 문제: 두 오류 유형은 flip 방향(0->1 vs 1->0)과 상태 종속 확률이 다름(Table 1: retention `00->01` 46%, PI `11->10` 70%). 저장시간에 무관한 강한 BCH는 오버헤드 과다.
3. 모델: [14]의 실측 오류패턴 사용. P/E cycle↑ 오류↑, 저장시간↑ retention↑. all-bit-line 구조 가정.
4. 1단계: Gray coding + 2bit interleaving으로 한 page를 MSB/LSB sub-page(각 n/2)로 분리 -> 각 sub-page에서 단일 flip 유형 지배(예: retention 시 MSB 0->1 88%, LSB 0->1 97%).
5. 2단계: 곱부호 - 행은 선형블록코드(BCH 또는 Hamming), 열은 even parity check. even parity는 latency/저장 오버헤드 최소.
6. 부호 GF: sub-page 단위(1024bit) 연산으로 `GF(2^11)`(vs 전체 페이지 `GF(2^12)`) 사용해 면적·critical path(0.72->0.65ns) 축소.
7. 오류위치 선택: 행복호에서 t 초과 오류 행 marking + 열 even parity로 오류 열 검출 -> 교점이 후보(m*n). 열당 1오류 가정, 각 열에서 "지배 유형 방향 1을 가장 적은 행에서 flip"하는 그리디 선택 알고리즘(a=1..T).
8. 구성 예: >1일 = 2×BCH(1046,1024,t=2)+even parity(9,8), <1일 = 2×Hamming(1036,1024)+even parity.
9. 결과: BER 성능은 BCH(2084,2048,t=3)/BCH(2072,2048,t=2)와 동등하며, even parity 추가로 오류율 약 0.8 decade 감소. HW(45nm 합성): Scheme2가 Scheme1 대비 에너지 50%·latency 60% 절감(에너지 2186->1060pJ, latency 3676->1520ns).
10. 한계: hard-decision·비반복 복호. raw BER>10^-3에서 목표 10^-9 미달(t=6 필요, 비실용) -> refresh 전략 필요. Scheme4는 latency 18.3ns지만 면적 52118µm² 급증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BCH/Hamming+even-parity 곱부호 복호 | 대응 없음 | Prime ECC는 min-sum 반복복호 LDPC(`LDPC_Decoder()`), 대수적 BCH/곱부호 아님 |
| Gray코딩+2bit MSB/LSB 인터리빙 | 대응 없음 | 페이지->sub-page 매핑은 컨트롤러 계층, Prime ECC 코드모듈에 없음 |
| retention/PI 오류 비대칭(값 종속 flip 확률) | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | soft-read threshold/LLR 설계 시 비대칭 오류 모델 참고 가능(간접) |
| 오류타입 지배 기반 위치선택 알고리즘 | 대응 없음 | 곱부호 전용 후처리, min-sum 반복복호와 무관 |

적용 가치: **낮음** — NAND 직접 대상이나 부호가 LDPC가 아닌 BCH/Hamming+even-parity 곱부호이고 hard-decision 비반복 복호라 Prime ECC 디코더/부호설계에 알고리즘 이식 불가. retention/PI 오류 비대칭 모델만 LLR/soft-read threshold 설계에 개념적 참고 가치.
