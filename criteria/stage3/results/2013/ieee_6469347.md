### B. 알고리즘 요약

1. 시스템: 16-QAM 계층적 변조(hierarchical modulation, Gray-coded) 신호에서 발생하는 2개의 equivalent binary modulation class(outer/inner QPSK, MSB/LSB)를 갖는 채널을 위한 irregular LDPC 부호 설계, `N=4096`, `dvmax=30`.
2. 문제: 표준 LDPC 설계는 모든 비트가 하나의 균일 채널을 거친다고 가정하지만, 계층적 변조는 심볼 내 비트 위치별로 서로 다른 error probability(noise variance)를 가지므로 새로운 설계가 필요.
3. 채널/모델: 계층적 16-QAM의 두 modulation class(`M1`, `M2`)별 비트오류확률 `Pb,c1`, `Pb,c2`을 등가 BPSK 잡음분산 `σ_cj^2`로 변환(constellation parameter `α=d0/d1`로 비대칭 정도 조절).
4. 핵심 기법: check node degree distribution `ρ(x)`는 고정한 채, variable node degree distribution을 modulation class별 sub-degree distribution `λ_Mj(x)`로 분할하는 multi-edge-type 구성.
5. 핵심 식: Density Evolution의 variable→check 갱신식 `x_vc^(l) = Σ_j Σ_i λ_Mj,i J(2/σj^2 + (i-1)J^-1(x_cv^(l-1)))` — 클래스별 잡음분산을 반영해 상호정보량 갱신.
6. 확장: 코드율 최대화를 목적함수로 하는 Linear Programming(LP) 알고리즘으로 `λ_Mj,i`를 최적화(수렴조건, stability condition, 클래스별 비율제약 `β` 포함).
7. 구현 디테일: 최적화된 sub-degree distribution에 따라 H matrix를 PEG 알고리즘으로 구성 후, 각 variable node를 degree/클래스 정보 기준으로 두 modulation class 중 하나에 배정.
8. 학습/최적화: LP 기반 rate-maximization + DE로 degree distribution 도출(외부 학습 없음, 폐형 최적화).
9. 결과: 계층적(α>1) 경우가 균일(α=1) 대비 코드율이 대체로 더 낮거나 유사(R≈0.48~0.54), BER 성능도 큰 차이 없음 — 저자 스스로 "계층적 변조가 LDPC 설계를 유의미하게 개선하지 못한다"고 결론.
10. 한계: 이론/시뮬레이션만, 무선 방송(UEP) 특화 시나리오로 NAND의 균일 read 채널과 근본적으로 다른 문제 설정이며 성능 개선 효과 자체가 불확실.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 계층변조 클래스별 sub-degree distribution 최적화(LP+DE) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 단일 균일 NAND read 채널의 고정 QC-LDPC라 다중 modulation class 개념 자체가 대응 없음 |
| 클래스별 soft LLR 산출(σ_cj) | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | NAND read는 균일 채널이라 계층변조 특화 LLR 분리 로직은 관련성 낮음 |

적용 가치: 낮음 — 무선 계층변조(UEP) 환경 전용 부호 설계로 NAND 균일 채널 ECC와 문제 설정이 다르고, 논문 자체 결론도 유의미한 성능개선이 없음.
