### B. 알고리즘 요약

1. 시스템: 자기저장(hard drive) PR4/EPR4 partial-response 채널 + rate 8/9, length 495 LDPC outer code, AWGN, message-passing 검출/복호.
2. 문제: 반복 검출/복호에는 soft-in/soft-out이 필요해 MLSD(Viterbi, hard-out)가 부적합 — 통상 APP(BCJR) detector 사용.
3. 모델: PR4를 두 interleaved dicode(`y_i=x_i-x_{i-1}`)로 분해, EPR4는 5-level 출력·8-state 채널로 모델.
4. 핵심 기법 1: dicode 채널을 입력비트 노드/출력심볼 노드의 factor graph로 표현, PR 다항식 의존성에 따라 edge 연결 후 bit-probability를 병렬 message-passing.
5. 의미: input/output 노드가 동시 갱신되어 스케줄이 병렬이며, iteration↑ 시 MAP detector 성능에 수렴.
6. 핵심 기법 2: EPR4는 bit-node 그래프가 short cycle 다수로 성능 열위 → state node(8-state) 도입한 state-based message-passing으로 APP 성능에 수렴.
7. 구현 트릭: 소수 iteration에서는 채널 역변환(mod-2) precoding이 성능 개선.
8. 결합 detector/decoder: dicode 입력/출력/LDPC parity의 3종 노드 단일 그래프, serial 스케줄(dicode T회 → LDPC S회 → U회 반복).
9. 결과: `(T+S)U≈24~25` 고정 시 dicode는 `T=S=1,U=12`, EPR4는 `T=2,S=1,U=8`이 최저 BER.
10. 한계: HW 미설계, 시뮬만, PR/ISI 자기저장 채널 전용, gate/throughput 수치 없음.
