> 총평: 랜덤 LDPC 대신 Steiner/BIBD 조합설계로 short-cycle-free 고rate(R≥0.9) QC-LDPC를 파라미터 몇 개로 결정론 구성하는 부호설계 논문으로, girth-6 보장·QC 구조가 Prime ECC의 H-matrix 설계에 참고가치는 있으나(이식성 중), column-weight 3 고정·자기저장 타깃이 제약.

### B. 알고리즘 요약

1. 대상: 랜덤 LDPC의 "H-matrix 위치 명세에 큰 정보량 필요→HW 복잡" 문제를 피해, 파라미터 소수로 완전 결정되는 구조적 LDPC를 조합설계로 만든다.
2. 채널/응용: 자기기록(magnetic recording) PRML(EEPR4 계열 `q(D)=5+4D-3D²-4D³+2D⁴`), 미디어잡음 지배, >1Gbps read channel, 고rate(≥8/9) 요구.
3. 핵심 도구: BIBD(v,k,λ) — v점·블록크기 k·index λ; k=3, λ=1이 Steiner triple system(Kirkman). 점=parity check, 블록=bit로 두면 block-point 근접행렬 `Aᵀ`가 Gallager 부호의 H가 된다.
4. girth 조건: λ=1 ⇔ 두 점이 한 블록에만 ⇔ H의 어떤 두 열도 같은 위치 1을 2개 안 가짐 ⇔ Tanner 그래프에 length-4 cycle 없음(girth≥6).
5. rate 하한식: `R=(λv(v-1)/k(k-1) - v)/(λv(v-1)/k(k-1))`(식1.2); k=3 고정, R>0.9면 b>500 필요(Fig.2).
6. 구성법: Abelian군 Zv의 cyclic difference family(CDF)의 base block을 g∈V로 순환 이동시켜 orbit(블록) 생성 → 각 orbit이 v×v circulant 서브행렬 `Hi` → `H=[H1…Ht]`가 quasi-cyclic 구조.
7. CDF 생성: Netto 구성(v=prime power, v≡1 mod6), GF(v) 곱셈군 생성원 ω로 `{ωⁱΨ_2t}`가 Kirkman (6t+1,3,1) DF 정의; TABLE 2에 v=7~43 base block 제시.
8. 인코딩: circulant/QC 구조를 t개 생성다항식으로 효율적 수행(shift register), self-orthogonal QC 부호와 유사.
9. 복호/결과: SOVA↔MPA(min-sign 근사, 5bit 정밀도) 최대 3라운드 concatenation, extrinsic 0.5 스케일링; Kirkman(81,3,1) 길이1040·R=0.922 코드가 Viterbi 대비 2dB↑ coding gain(cbd=2.8).
10. 한계: HW 미설계·시뮬만, 자기저장 채널 특화, syndrome 재계산 latency 문제·짧은 base block에 의한 rate 손실 미해결.
