### B. 알고리즘 요약

1. 대상: Gallager LDPC block code의 convolutional 판(LDPC convolutional (J,K)-code), BSC 및 AWGN 전송.
2. 문제: 반복복호가 신뢰전송 가능한 한계(iterative limit)를 convolutional 부호에 대해 엄밀히 하한 짓기.
3. 1단계: Tanner graph의 최소 cycle 길이가 `2·l0`(`l0`은 memory `m_s`의 로그에 비례) 초과인 `l0`-nondegenerated 부호의 존재 증명(Theorem 1).
4. cycle-free 독립 반복 횟수가 `l0` 이상이어야 반복복호 분석이 유효 — cycle 길이 정리가 그 근거.
5. 2단계: 반복 횟수를 `l0`로 두고 two-phase 복호 분석. 1상은 확률적 Gallager(또는 간이 suboptimal) 알고리즘을 수치 분석.
6. 2상은 majority decoding으로 해석적 분석 가능 → 점근 분석으로 iterative limit `p0`를 엄밀 하한.
7. 결과: (3,6)/(4,8) 부호의 BSC crossover probability에 대한 하한 `p0(3,6)`, `p0(4,8)`을 그림으로 제시(수치 본문 없음), AWGN에도 유사 결과.
8. 관찰: 동일 BER 달성 시 LDPC convolutional code에 필요한 memory가 대응 LDPC block code 길이보다 훨씬 작음.
9. 확장: turbo code의 convolutional 판(nonhomogeneous LDPC convolutional)에도 결과 확장 가능.
10. 한계: 순수 점근 이론 요약(1쪽), HW·구현·구체 부호 파라미터·정량 BER 없음.
