### B. 알고리즘 요약

1. 대상: 임의 rate/length의 binary LDPC용 Tanner graph를, girth를 크게 만드는 일반적(비대수적) 구성법 PEG로 생성(예: rate-1/2, n=504/1008).
2. 문제: 무작위 그래프는 short cycle(girth) 보장이 없어 short/medium length에서 iterative 복호 성능이 나쁨; Gallager 구성은 regular·대규모 length에서 복잡도가 큼.
3. 핵심기법: symbol node를 순서대로 처리하며 새 edge를 추가할 때마다 그 노드의 local girth가 최대가 되도록 best-effort로 check node를 선택(edge-by-edge).
4. edge 선택: symbol node에서 depth `l`까지 tree를 확장, 도달 못한 check node 집합에서 degree가 가장 낮은 노드를 골라 연결 → check-node degree를 최대한 균일하게.
5. 이론: 결과 그래프의 girth 하한과 최소거리 하한을 symbol/check node 수와 최대 degree로 표현(Theorem 1, Lemma 1~4); girth 상한은 임의 Tanner graph에도 성립.
6. girth histogram: PEG는 각 symbol node의 local girth를 개별 최대화 → 평균 local girth가 MacKay/random 대비 높음(n=504에서 8.01 vs 6.74 vs 6.42).
7. 선형시간 인코딩: redundant subset에 미리 "zigzag" 패턴(triangular PCM) edge를 깔고 나머지 정보 노드만 PEG로 연결 → 반복적 O(n) 인코딩 가능(IRA의 zigzag와 결합).
8. 결과: 80 iteration에서 PEG (504,252)는 BLER 기준 MacKay 대비 약 0.2 dB 우수, random graph 대비 큰 폭 우수.
9. 통찰: 전역 girth뿐 아니라 girth histogram 분포가 iterative 복호 성능을 지배함.
10. 한계: HW 미설계, 시뮬만; 원형 PEG는 비구조적이라 QC/circulant 제약과 결합하려면 별도 변형 필요.
