### B. 알고리즘 요약

1. 대상: Bayesian network/Markov random field(MRF)로 표현된 임의 위상 그래피컬 모델에서 MAP 추정을 위한 max-product(=log영역 min-sum) belief propagation.
2. 문제: max-product는 tree에서만 최적해 수렴이 증명됐고, 루프 그래프(turbo/LDPC 복호 포함)에서의 고정점 해 품질에 대한 이론이 거의 없었다.
3. 핵심 가정: 모든 clique를 pairwise로 전처리한 MRF, Markov blanket property; 메시지는 병렬 동시 갱신, tie 없음(유일 MAP).
4. 핵심 기법: 고정점 assignment `x*`가 posterior의 "neighborhood maximum"임을 증명 - `x*`에서 tree 또는 단일 loop를 이루는 임의 노드 부분집합을 바꿔도 posterior가 감소.
5. 핵심 식: SLT(single loops and trees) neighborhood 정의 - `P(x*)>=P(x)` for all `x` in SLT(`x*`) (Claim 1); local maximum보다 강하고 global보다 약함.
6. 증명 도구: unwrapped tree(computation tree)와 universal covering - 루프 그래프의 local topology를 보존한 tree에서 max-product가 global optimum이라는 성질과 isomorphic-subtree/infinite-chain property를 이용.
7. 응용 예: Gaussian graphical model이면 unimodal이라 neighborhood max=global max -> BP 사후평균이 정확(Corollary 1).
8. 부호 예: turbo code에서 max-product 고정점은 MAP이거나 Hamming distance>=3 이상 떨어짐(Corollary 2) - "거의 맞는" 오복호 불가.
9. 결과(실험): length-7 turbo(exhaustive MAP 대조)에서 max-product는 수렴 시 항상 MAP 발견; 25x25 grid에서 99% MAP, 틀려도 2등해(greedy는 44%).
10. 한계: 순수 추론/그래피컬모델 이론(부호는 turbo·toy·cycle code 예시), HW/유한성능/양자화 없음; 수렴 자체는 보장 안 함(수렴하면 최적성 보장).
