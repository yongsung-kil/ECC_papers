### B. 알고리즘 요약

1. Wiberg형 일반화 state realization을 대상으로, 심볼변수 degree=1, state변수 degree=2인 `normal realization`을 정의한다.
2. 기존 factor graph 대신 변수를 edge로, 제약(local code)을 vertex로 표현하는 `normal graph`(leaf edge=심볼, 보통 edge=state)를 도입한다.
3. 임의의 state realization은 replication 절차로 정규화 가능하며, 위상·복호복잡도가 본질적으로 불변임을 증명한다(Thm 3.3).
4. cut-set과 State Space Theorem으로 cycle-free 그래프의 최소(canonical) realization 존재를 보이고, Cut-Set Bound로 state space 하한을 제시한다.
5. 그래프에 cycle이 있으면 성능-복잡도 tradeoff가 우월할 수 있으나(예: tail-biting Golay는 state 수 256→16), sum-product는 근사·반복적이 된다.
6. 핵심 결과인 duality theorem: normal linear/group realization의 dual(문자군 알파벳+직교 제약)은 동일 위상 그래프로 dual code를 생성한다.
7. 이 이중성은 dual state space, dual trellis, dual Tanner graph, dual I/O 시스템 등 다수 응용을 낳는다.
8. sum-product 알고리즘을 normal graph 상에서 edge/cut 중심으로 재유도(upstream/downstream 분해).
9. 고rate 부호는 dual 그래프+Fourier 변환으로 복호를 단순화할 수 있음을 제시한다.
10. RM/Golay/turbo/LDPC/RA 예제로 realization을 예시하나 정정능력 수치·HW 설계는 없다(순수 이론).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| sum-product / 그래프 복호 이론 | 대응 없음 | 일반 이론일 뿐, min-sum HW 구현과 직접 연결 안 됨 |
| normal graph / factor graph 표현 | 대응 없음 | 부호 표현 이론, 코드베이스에 별도 모듈 없음 |
| dual code / Fourier 복호 | 대응 없음 | Prime ECC는 primal binary QC-LDPC 고정 |
