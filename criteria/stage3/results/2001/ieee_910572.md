### B. 알고리즘 요약

1. 시스템: 다변수 global 함수의 factorization을 bipartite factor graph(variable node + factor node)로 표현, Tanner graph의 함수 일반화.
2. 문제: BP/BCJR/Viterbi/Kalman/FFT 등 서로 달라 보이는 알고리즘들을 하나의 marginalize-product-of-functions(MPF) 문제로 통합 설명.
3. 핵심 기법: sum-product 알고리즘 - 노드가 인접 메시지의 곱을 해당 변수로 summary(not-sum)해 전달하는 단일 규칙 (식5 VN, 식6 factor node).
4. 핵심식: VN은 입력 메시지 곱(식5), factor node는 local function × 입력곱을 summary(식6) - cycle-free 그래프에서 정확한 marginal 계산.
5. 의미: distributive law로 partial sum 재사용해 marginal을 효율 계산, cycle 있으면 iterative(turbo/LDPC 복호)로 근사.
6. 확장(semiring): max-product/min-sum semiring으로 일반화 → MLSD(Viterbi)는 negative-log 도메인 min-sum, branch metric은 Euclidean/Hamming.
7. 이진 단순화(§V-E): binary 변수 + parity/repetition 제약에서 LR/LLR/LD/SLLD 파라미터화, LLR 도메인에서 check node 갱신이 tanh-rule이며 그 근사가 정확히 **min-sum 갱신 규칙**.
8. 응용: forward/backward(BCJR), Viterbi, Kalman filter, turbo/LDPC/RA 복호, factor-graph 변환(clustering/stretching/spanning tree)으로 FFT까지 SPA 인스턴스로 유도.
9. 결과: 통합 이론 프레임워크 제시 (정량 성능·부호 설계 없음), min-sum이 LLR SPA의 계산 근사임을 명시.
10. 한계: 순수 튜토리얼/이론 (HW·시뮬·NAND 무관), 신규 부호나 디코더 최적화 기법 없음.
