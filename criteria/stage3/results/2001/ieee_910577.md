### B. 알고리즘 요약

1. 대상: `(d_v, d_l)`-regular LDPC 앙상블(예: `(3,6)`, design rate `1 - d_v/d_l = 0.5`)을 임의 binary-input memoryless 대칭 채널(BSC/BIAWGN/BIL)에서 message-passing 복호.
2. 문제: Gallager식은 특정 부호(큰 girth) 구성이 어렵고 irregular로 가면 거의 불가능 → 개별 부호 대신 **앙상블 평균 성능**을 분석해 threshold를 정한다.
3. 세 축의 정리: (a) concentration(거의 모든 부호가 평균처럼 행동), (b) cycle-free 수렴(유한 iteration에서 이웃이 tree-like), (c) density evolution + threshold.
4. 핵심 기법: 반복 `l`회의 오류 메시지 분포를 tree-like 가정하에 추적하는 `density evolution`. 유한 알파벳이면 결합 재귀식, 무한 알파벳(BP)이면 밀도 자체를 갱신.
5. 대칭 가정(channel/check/variable node symmetry)으로 오류확률이 송신 codeword에 무관 → all-one codeword 전송 가정으로 분석 단순화.
6. BP density evolution: variable node는 LLR 합(실수 convolution → Fourier), check node는 `GF(2)×[0,∞)` 표현에서 sign·log-magnitude 분해 후 덧셈(FFT로 cyclic convolution).
7. 유한 알파벳 복호기(Gallager A/B, 소거 허용 E, 3-bit 양자화)를 예로 들어 quantized BP로 해석 — 3-level만으로도 BP에 근접.
8. threshold 존재성: physical degradation으로 정렬되는 채널족에서 항상 존재. BSC `(3,6)` threshold ≈ 0.084, BIAWGN `(3,6)` `σ* ≈ 1.0` 부근(raw BER ~13%).
9. 결과: threshold를 임의 정밀도로 계산하는 효율 알고리즘 제시, rate-1/2 `(3,6)` 부호가 BIAWGN에서 turbo에 근접(약 1.4 dB @ BER 목표).
10. 한계: 무한 길이 앙상블 평균의 점근 이론 — HW 미설계, 유한길이/부호구성·NAND soft-read 등 실무 적용은 범위 밖.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| density evolution / threshold 계산 | 대응 없음 | 부호 성능 이론 분석 도구, 런타임 디코더 코드와 무관 |
| BP(sum-product) check node full-`tanh`/FFT 갱신 | 대응 없음 | Prime ECC는 min-sum 근사 사용, full BP는 미지원 |
| 대칭 가정 → all-one codeword | 대응 없음 | 이론 가정, 구현 요소 아님 |

적용 가치: **낮음** — 교과서화된 DE/threshold 이론이며 Prime ECC 3.1(min-sum 기반 binary QC-LDPC HW 시뮬)에 새로 이식할 알고리즘·구현 요소가 없음.
