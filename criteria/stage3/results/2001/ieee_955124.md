### B. 알고리즘 요약

1. 시스템: multilevel coding(MLC)에 parallel independent decoding(PID)을 결합, Gray-mapped 8-PSK(m=3레벨), 전체 rate `R=2 bits/symbol`, AWGN 채널.
2. 문제: MLC/PID의 이점은 알려졌으나 capacity에 근접하는 성분부호를 실제로 설계하는 구성법이 없었음.
3. 모델: PID로 시스템을 m개 병렬 binary-input 채널로 분해, 각 채널 `i`는 입력 `c_i`·출력 `y`, PID capacity `C̃ = Σ I(C_i;Y)`.
4. 핵심기법: 각 레벨 `i`의 등가 이진채널에 density evolution 적용해 반복 `l`후 오류메시지 비율 `p_i(l)` 추적, noise threshold `(σ*)^i` 산출.
5. 핵심식: `f_l^i = f_0^i ⊗ λ^i(Γ^{-1}(ρ^i(Γ(f_{l-1}^i))))` — 비대칭 등가채널이라 all-ones 가정 대신 심볼 등확률 가정으로 초기밀도 결정.
6. 확장: 각 레벨 부호가 (거의) 동일 성능을 갖도록 rate와 degree분포 `(λ^i,ρ^i)`를 공동최적화(threshold 균등화).
7. 최적화: density evolution + differential evolution(비선형 최적화)으로 최대 threshold 갖는 degree분포 탐색, 최대 bit degree `d_v,max=50` 제약.
8. 결과: 최적 성분 rate `0.51/0.745/0.745`, PID capacity와 gap 0.072 dB, 채널 capacity와 0.142 dB 이내.
9. 검증: 길이 `10^6` 성분부호 BER 시뮬로 threshold 예측 정확성 확인(두 rate-0.745 부호 성능 동일).
10. 한계: 대역효율 변조 특화 이론/시뮬만, HW 미설계, NAND soft-read·binary QC-LDPC 구조와 무관.
