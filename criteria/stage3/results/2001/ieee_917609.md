### B. 알고리즘 요약

1. 시스템: 고밀도 자기기록 채널(Lorentzian, EPR4 등화)용 부호. 대표 rate 8/9, block size 4896.
2. 문제: turbo는 인코딩 쉽고 복호 무겁고, LDPC는 반대. 둘의 장점을 합쳐 인코딩·복호 모두 저복잡한 부호를 원함.
3. 채널모델: 1차 position jitter/width variation(PJWV) transition noise + AWGN, `SNR` 정의는 symbol density 독립적으로 정의.
4. 핵심기법: 여러 개의 single-bit parity check block code를 랜덤 인터리버로 **병렬 연접**(PCPC). 인코더 구조가 곧 **low-density generator matrix(LDGM)**가 됨.
5. 의미: PCPC 인코딩 구조에 대응하는 parity check matrix `H = [I | A]` 형태가 sparse → LDPC식 message passing으로 복호 가능.
6. 확장: 인터리버 미사용 경로에 odd parity를 강제해 연속 동일비트(runlength) 제한, rate 손실 없이 timing recovery 개선.
7. 구현 디테일: 각 단일패리티 성분부호는 XOR 하나로 구현. rate 8/9 예시는 rate 32/33 성분부호 4개를 병렬 사용.
8. 복호: 채널디코더(pattern-dependent noise-predictive BCJR, 64-state, 4-tap) + bipartite graph belief propagation, 채널로 soft info 재귀(iterate) 옵션.
9. 결과: 100% AWGN에서 uncoded EPR4 대비 ~3dB, 90% media noise에서 block 4896시 8.5dB SNR gain. LDPC와 거의 동일 성능, error floor 미관측.
10. 한계: 자기기록 채널 특화, HW 미설계, LDGM 점근성능 불량, 성능은 채널디코더 최적화에 크게 의존.
