### B. 알고리즘 요약

1. 시스템: rate 8/9(주력, 4/5·16/17 비교), 블록 4896-bit LDPC를 고밀도 자기기록(Lorentzian, user density 2.5) 채널에 적용.
2. 문제: 기존 iterative 복호 연구는 자기기록 기준 너무 낮은 rate를 씀 — 고rate에서 media noise 하 성능을 규명해야 함.
3. 채널 모델: 1차 position-jitter 모델로 signal-dependent transition noise를 표현, `r(t)=Σ h(t+j_k-kT)+n(t)`, `j_k`는 독립 Gaussian.
4. 채널 복호: pattern-dependent noise-predictive(PDNP) BCJR — `γ_k(s',s)` 계산에 예측기(4-tap)로 조건부 평균 `n̂_k`를 반영, `α_k`,`β_k`는 AWGN과 동일.
5. 의미: transition noise가 신호의존적이라 표준 Gaussian `γ`를 예측잔차 분산 `σ²` 기반으로 교체해야 최적 soft 정보를 얻음.
6. LDPC 복호: bipartite graph 상 message passing, check 갱신에 `tanh`/`tanh⁻¹`(식 7, 표준 sum-product), 이후 log-APP `Λ_j = λ_j + Σ C_ij` 갱신, 유효 codeword 시 종료(최대 20 iter).
7. 구현 디테일: soft 정보를 채널로 되돌리는 turbo-equalization 대신, LDPC 내부만 iterate해도 손실이 약 0.5dB뿐 — HW 복잡도·latency 절감.
8. 학습: 예측기 계수·오차분산을 긴 training sequence로 획득(a priori 가정 안 함).
9. 결과: 90% media noise + 64-state PDNP + LDPC로 uncoded EPR4 대비 약 7.5dB 이득; 8/9가 rate 최적, 블록 축소 시 최대 1dB 손실.
10. 한계: HW 미설계, 시뮬레이션만, 자기기록 채널 특화(NAND read/soft-quant 무관), 2차 모델 시 이득 감소.
