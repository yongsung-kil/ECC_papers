### B. 알고리즘 요약

1. 대상: IEEE 802.11n / 802.16e 표준 irregular LDPC(rate 1/2~2/3, N 648~2304), QPSK·AWGN 채널, layered OMS(offset min-sum) 복호.
2. 문제: LDPC 복호의 높은 계산복잡도를 모바일 저전력 제약 하에서 성능 손실 최소로 더 줄이기.
3. 기법 골격: forced convergence — VN LLR `Qv` 크기가 임계값 `θ`를 넘으면 해당 VN 업데이트를 중단(Inact 집합), + 반복마다 syndrome check 조기종료.
4. 핵심: 수신기 설계를 `minimize Γ(δ,σ,ρ) s.t. Π(δ,σ,ρ)≥Πcon` 최적화로 정식화, 조정변수 `δ=θ`, 환경 `ρ=SNR`.
5. 복잡도 함수 `C̄t(θ,SNR)=α·N̄it + 4·N̄nodes` (α=Amax+|c|·Ψ, Ψ는 min1/min2 탐색비용), FER1·N̄들은 Monte-Carlo로 추정.
6. Case1 (relaxed FER=1e-2): `min C̄t s.t. FER1≤FERrelax` → 최적 `θopt=f(SNR)` 채널적응 임계값 도출.
7. Case2 (tight): forced-convergence 오류 블록을 보조 디코더로 재복호, `min C̄t + FER1·C̄a` 로 두 디코더 합계 복잡도 최소화점 탐색(Imax 도달=오류블록 판정).
8. 관찰: θ가 크면 node activity가 reference(Amax)로, θ가 작으면 반복수 증가 → 비용함수는 중간에 최소.
9. 결과: reference OMS(syndrome check O, forced-conv X) 대비 복잡도 최대 35% 절감, 채널적응 θ가 상수 θ보다 우수. FER 곡선상 θ 작을수록 error floor 상승.
10. 한계: 복잡도는 add/compare 개수 근사(wordlength·memory access 무시), HW 미설계, 무선 표준 코드·AWGN 가정.
