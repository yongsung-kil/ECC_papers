### B. 알고리즘 요약

1. 대상은 개별 부호가 아닌 **Tanner-graph 앙상블** `C_n^{λ,ρ}` (변수/체크 edge degree 분포 `λ`, `ρ`, block length `n`), rate는 `1-∫ρ/∫λ`로 상한.
2. 문제: BEC 반복복호에서 실패는 소거집합이 nonempty **stopping set**을 포함할 때만 발생 → stopping set 분포가 성능(특히 error floor)을 결정.
3. 모델: 랜덤 순열로 socket 연결한 앙상블 평균을 다루며, stopping set = 모든 체크가 단일 edge로 연결되지 않는 변수노드 집합.
4. 핵심 기법 1: 다항 특성함수(polynomial characteristic function)로 크기 `s` stopping constellation 수를 세고, Stirling·saddle-point 부등식으로 **정규화 평균 stopping set 분포** `γ(α)`의 지수적 성장률 유도(정리 2, 5).
5. 핵심 식: **critical exponent stopping ratio** `α_c` = 크기 `α·n` stopping set이 지수적으로 많아지는 최소 비율. 앙상블의 likely stopping number를 규정.
6. 핵심 기법 2: **stopping number**(최소 stopping set 크기)가 block length에 선형으로 성장할 확률 분석 — 변수 degree `l≥3`면 확률→1, degree-2 노드 존재 시 `λ'(0)ρ'(1)` 값으로 갈림(정리 13).
7. 구현 디테일: expurgation(작은 stopping set 부호 제거)으로 앙상블에서 선형 stopping number 보장, cycle 구조·inclusion-exclusion으로 degree-2 노드 기여 bound.
8. 최적화: 없음(해석적 유도, Lagrange multiplier로 지수 최대화만 사용).
9. 결과: `R=1/2` 정규 앙상블에서 `c` 증가 시 `α_c → h⁻¹(...) ≈ 0.11`; error-floor threshold `p̃`(예 `d_v=3` 정규에서 `p̃=0.326`) 이하면 선형 크기 stopping set 기여가 지수적으로 소멸.
10. 한계: **BEC 전용**·앙상블 평균·순수 이론, HW/시뮬 산출물 없음, AWGN이나 NAND soft-read로의 직접 확장 없음.
