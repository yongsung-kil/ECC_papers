### B. 알고리즘 요약

1. 대상은 메시지가 이진값 `{0,1}`인 binary message-passing(BMP) 디코더로, BSC에서 LDPC를 복호하며 EXIT chart(오류확률 기반)로 수렴을 분석한다.
2. 풀려는 문제: 이진 메시지 제약 아래 어떤 갱신규칙이 메시지 오류율을 최소화하는 "최적" 디코더인가, 그리고 그 디코더용 최대rate 부호를 어떻게 설계하는가.
3. 핵심 모델: 대칭 BSC + 등확률 codeword + computation tree(들어오는 메시지 독립) 가정.
4. 핵심 기법 1단: 최적 CN/VN 갱신규칙은 반드시 symmetry 조건과 isotropy 조건을 만족해야 함을 증명(Thm 1~4).
5. 핵심 식: CN 최적규칙은 입력의 modulo-2 합, `1-2p⊕ = ∏(1-2p_i)`로 각 `p_i<1/2`면 `p⊕<1/2` 보장.
6. 핵심 기법 2단: 이 성질로부터 regular 부호에서 Gallager `Algorithm B`가 모든 BMP 중 최적 threshold임을 증명, irregular도 이웃차수 미활용 시 최적성 유지.
7. 구현 디테일: Algorithm B를 고정 `b`의 elementary algorithm 스위칭으로 보고, EXIT chart(`p_in` vs `p_out`) 형상화를 선형계획으로 해결.
8. 최적화 절차: check degree distribution 고정 시 irregular 설계는 EXIT chart 여러 `g_i`의 선형결합을 여는(open) 선형계획으로 환원.
9. 결과: Algorithm B rate 상한식(12) 유도, 저오류 BSC(`p0=0.001`)에서 `d_v=3` 집중 설계로 rate `0.9051`(상한 `0.9076`) 달성, 최적 check degree는 1~2개 차수에 집중.
10. 한계: 전부 점근 이론(유한길이·HW·시뮬레이션 없음), BSC hard-decision 전용.
