### B. 알고리즘 요약

1. 고rate LDPC를 sum-product로 복호 시 Shannon 한계에 근접하며, Tanner graph의 최단 사이클 길이(girth `g`)가 클수록 복호 효율과 최소거리가 향상됨을 배경으로 함.
2. 풀려는 문제: 유연한 code rate를 가지면서 큰 girth를 보장하는 구조화 LDPC 부호를 체계적으로 설계.
3. 구성법(PS-LDPC): check node 집합을 크기 `p`의 `N_c`개 부분집합, bit node 집합을 `N_b`개 부분집합으로 분할(`n = N_b·p`), 각 부분집합 내 노드를 `0~p-1`로 인덱싱.
4. 연결 규칙: 각 check node는 서로 다른 `k`개 bit-node 부분집합의 노드와, 각 bit node는 서로 다른 `j`개 check-node 부분집합의 노드와 연결(regular `(j,k)`).
5. 핵심(shift): `α`번째 check 부분집합의 인덱스 `X` 노드는 `β`번째 bit 부분집합에서 `W = X ⊕ S_{α,β}` (modulo-`p` 덧셈) 인덱스 노드와 연결; 모든 shift를 `N_c×N_b` shift matrix `S`로 수집 → circulant-shift 구조.
6. 핵심 정리(Theorem 1): Tanner graph에 `2t`-cycle이 존재 ⇔ shift matrix `S`에 길이 `2t`의 closed path가 있고 그 `2t` corner들의 교대 modulo-합이 0.
7. 설계 절차: Theorem 1 조건을 위반하도록 shift `S_{α,β}`를 선택하면 임의 girth의 PS-LDPC 부호 설계 가능(예: 6-cycle 제거).
8. 예시 부호: `(6075, 3, 27)` PS-LDPC, rate `r = 8/9`, girth 8 (column weight 3).
9. 결과: EPR4 채널에서 girth-8 PS-LDPC를 동일 파라미터(column weight 3, length 6075, rate 8/9)의 4-cycle-free 랜덤 LDPC와 비교; 고SNR에서 랜덤 부호는 BER≈3×10^-? 부근 error-floor 진입 후 PS-LDPC가 앞서며 아직 floor 미도달.
10. 한계: 1페이지 digest로 HW/디코더 미설계, EPR4 자기기록 채널 시뮬만, rate 유연성의 실제 범위와 구현 비용 미제시.
