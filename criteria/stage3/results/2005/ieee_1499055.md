### B. 알고리즘 요약 (10줄 내외)

1. 대상: BPSK-AWGN 상의 rate-1/n 이진 비재귀 systematic **convolutional self-orthogonal(CSOC)** 및 **self-doubly-orthogonal(CSO2C)** 부호 (memory m=17~1835 예시).
2. 문제: 반복복호(BP/threshold)에서 code graph의 short cycle과 관측 상관이 성능·수렴을 저해 → 부호 구조로 이를 최소화.
3. 도구: difference set과 computation tree로 부호 구조를 분석 (self-orthogonal set `A`, 차분집합, 차분의 차분집합).
4. 핵심1(독립관측): double orthogonality가 2회 연속 iteration에서 독립 관측 수(`|D2|` 성격)를 최대화 → 수렴 가속·초기지연 감소.
5. 핵심2(cycle 최소화): computation tree 상 6-cycle·8-cycle 중 회피 가능한 것을 double orthogonality가 모두 제거 → 최소 개수 달성.
6. 식 의미: 짧은 cycle이 BP 성능을 저해하므로(널리 알려진 사실) 이를 구조적으로 배제.
7. 균형트리: double orthogonality가 computation tree를 엄밀히 balanced하게 만들어 최적 weighing factor(출력 감쇠) 결정 가능.
8. 복호식: check message는 sum-product(식5) 또는 그 근사인 add-min(min-sum, 식6) 사용, 최종 message에 weighing factor 적용(식7).
9. 결과: CSO2C의 threshold decoding이 BP에 근접(구현복잡도는 BP one-step의 약 1/J), CSO2C의 effective iteration이 CSOC보다 크게 작음(BP 기준 ~4회), CSOC에서 BP가 threshold 대비 BER 1e-5에서 ~0.5dB 우세.
10. 한계: HW 미설계, gate count/throughput 없음, convolutional 부호 전용 이론이며 LDPC로의 확장은 향후 과제로만 제시.
