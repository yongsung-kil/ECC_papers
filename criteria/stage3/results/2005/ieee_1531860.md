### B. 알고리즘 요약

1. 목표: binary cyclic 부호를 유한체 변환(Mattson-Solomon, MS) 도메인에서 구성해 sparse H-matrix를 갖는 LDPC로 활용, BPSK/AWGN 전송·iterative(수정 BP) 복호.
2. 문제: Shibuya-Sakaniwa의 cyclotomic-coset 기반 4-cycle-free 구성은 min-distance·code-rate 보장이 없음.
3. 핵심: idempotent `u(x)`의 MS 다항식 `ζ(z), ξ(z)`에서 부호의 3대 특성(idempotent weight, n번째 단위근 근의 수, BCH bound)을 직접 읽어냄.
4. `wt(u(x))` = `ξ(z)`의 차수 → 낮은 weight idempotent 선택이 곧 sparse H-matrix.
5. 부호 dimension = `u(x)`의 근인 단위근 수 = `n − wt(ξ(z))` → 이를 통해 code-rate를 제어.
6. BCH bound = `ξ(z)`의 연속 non-zero 계수 최대 길이 → min-distance 하한을 설계단계에서 보장.
7. 탐색: primitive idempotent들을 차수 오름차순 정렬 후 weight 증가 순 재귀 tree-search(Algorithm 1), sparse(`deg≤√n`)·rate(`≥Rmin`)·min-distance(`≥δ`) 세 조건으로 가지치기(exhaustive `O(2^t)`를 축소).
8. 성공 시 역 MS 변환으로 idempotent 복원해 cyclic 부호 확정; DSC·Euclidean/Projective Geometry 부호 등 기존 부호도 재생성 가능.
9. 결과: 구성 부호 FER이 이진 sphere-packing bound에 근접, 동등 irregular 부호 대비 coding gain 확보하고 high min-distance로 error-floor 부재(Fig.1~4).
10. 한계: HW 미설계·시뮬만; error-floor 개선이 waterfall(SNR threshold) 악화로 상쇄; cyclic 구조(비-QC)이고 장블록에서 threshold 손실.
