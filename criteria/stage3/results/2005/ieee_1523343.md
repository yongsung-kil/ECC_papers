### B. 알고리즘 요약

1. 대상: (6561,4096) irregular LDPC(China HDTV, rate 0.62), AWGN, binary 변조, soft-input 복호.
2. 문제: sum-product(SPA)는 경험적이라 loop 그래프에서 수렴 이론이 취약하고 error floor가 높음(BER 10^-5 부근 floor).
3. 모델: ML 복호를 제약최적화 `max Σ a_i(2x_i-1) s.t. Hx=0`을 unary constraint `f_i`+parity m-ary constraint `f_{Xj}`의 비제약 최소화 `min Σf_i + Σf_{Xj}`(식12)로 변환.
4. 기법(cooperative optimization): 비용함수를 변수별 sub-cost `E_i`로 분해, 각 agent가 sub-problem을 soft decision `c_i(x_i)`로 풀고 이웃과 절충.
5. 핵심 식(6): `c_i^k(x_i)=min_{xj∈Xi\xi}[(1-λk)E_i + λk Σ_j w_ij c_j^{k-1}(x_j)]` — `λk`(cooperation strength)로 자기 sub-problem vs 이웃 해 가중.
6. 이론 보장: propagation matrix `W`(irreducible, 열합=1) 하에서 유일 equilibrium(Thm2.4)·지수수렴 rate λ(Thm2.5), consensus면 global optimum(Thm2.3), `Σc_i`는 하한 함수(Thm2.1).
7. 그래프 분해: Tanner 그래프를 변수별 tree-like sub-graph로 분해(simple decomposition), 분해 방식이 성능 좌우. classical min-sum은 더 복잡한 분해의 특수경우로 해석.
8. 확장: min-sum semiring 외 다른 semiring으로 일반화 가능(sum-product 유사).
9. 결과: SPA 대비 error floor 1자릿수 이상 저감(BER<10^-7 @2.0dB), waterfall도 더 급하강, iteration 120이지만 per-iter 저비용으로 시뮬 10배 이상 빠름.
10. 한계: HW 미설계, 단일 코드 실험, 복잡도/gate 정량 없음, iteration 수는 SPA보다 많음.

