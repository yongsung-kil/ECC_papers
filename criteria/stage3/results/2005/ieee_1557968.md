### B. 알고리즘 요약

1. 시스템: 2x2 MIMO(fast Rayleigh fading), QPSK, LDPC rate `R=1/2`, random edge interleaver, `y=Hs+n`(식1).
2. 문제: AWGN 최적 degree profile은 채널이 바뀌면 성능이 나빠지고, MIMO는 memory 채널이라 symbol-by-symbol 검출이 최적 아님.
3. 구조: SISO MIMO decorrelator(APP 검출기)가 LLR 생성 → LDPC 디코더와 concat해 turbo iterative detection(검출기·variable node를 하나의 decoder로 간주, Fig.2).
4. 핵심: LLR이 비-Gaussian이라 Gaussian approximation(분산=2·평균) 부정확 → mutual information을 품질척도로 사용, 등가평균 `µ=(J⁻¹(I))²/2`(식2).
5. VN/CN mutual information transfer function 식3·식4로 message passing 해석, 검출기 곡선은 직선 근사(다항식).
6. 최적화: EXIT chart에서 (검출기+VN) 결합곡선을 CN 곡선에 정합. 결합곡선이 degree별 곡선의 선형결합(식6, `λ_i`)이라 curve fitting을 linear programming으로 해결.
7. 절차: 검출기 EXIT 곡선 근사 → 좌/우 degree profile을 LP로 번갈아 최적화하며 목표 rate 만족까지 SNR 조정(7단계).
8. 결과(시뮬, 2x2 QPSK, 100 iter): EXIT 최적 profile waterfall ~2.2dB(fitting point 2.0dB에서 0.2dB), 규칙 (3,6) 대비 ~1dB 우수; AWGN 최적 profile은 오히려 (3,6)보다 0.3dB 나쁨.
9. 시사: irregularity 자체가 아니라 채널 transfer 곡선 정합이 목적이며, 한 채널 최적화가 다른 채널에서 좋음을 보장하지 않음.
10. 한계: 순수 시뮬(HW 없음), MIMO fading·joint detection 전용, code length 본문 표기(n=105) superscript 손상 의심.
