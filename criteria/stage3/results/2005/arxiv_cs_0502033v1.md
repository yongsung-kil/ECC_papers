### B. 알고리즘 요약

1. 대상: 모든 변수(bit)노드 degree가 2인 특수 LDPC인 cycle code를 memoryless 채널·iterative decoding 관점에서 분석.
2. 문제: iterative decoder는 Tanner graph의 finite cover 위 codeword(pseudo-codeword)를 구분 못해 복호 실패가 발생 — 이 실패를 기하학적으로 특성화하고자 함.
3. 핵심 개념: parity-check matrix `H`의 fundamental cone `K(H)` = pseudo-codeword들이 이루는 원뿔(Euclidean cone), iterative/LP decoding 성능을 지배.
4. cycle code의 normal graph `N(H)`에서 codeword = simple cycle(또는 그 대칭차)에 대응하며 `dmin` = `N`의 girth(최단 cycle 길이).
5. edge zeta function `ζ_N(u_1,...,u_n)` = normal graph의 primitive·backtrackless·tailless cycle들을 열거하는 유리함수, `ζ_X⁻¹ = det(I − UM(X))` (Stark-Terras).
6. 핵심 결과(Th. V.4): 단항식 `u_1^{p_1}...u_n^{p_n}`이 `ζ_N`에서 비영 계수를 가질 필요충분조건은 지수벡터 `(p_1,...,p_n)`이 unscaled pseudo-codeword인 것.
7. 따라서(Cor. V.5) `ζ_N`의 Newton polyhedron(Taylor 항 지수들이 span하는 다면체)이 fundamental cone `K(H)`와 정확히 일치.
8. 증명 아이디어: normal graph의 backtrackless·tailless cycle ↔ 유한 cover 위 simple cycle의 사영(π)이 일대일 대응함을 cycle 길이 귀납으로 증명.
9. 결과 형태: Code A[4,2]/Code B[7,2] 두 toy 예제로 fundamental cone·zeta function·pseudo-codeword 대응을 명시(수치 성능 없음).
10. 한계: cycle code(degree-2) 전용, dmin이 로그로만 성장, general LDPC로의 zeta function 존재는 추측에 그침, HW/시뮬 전무.
