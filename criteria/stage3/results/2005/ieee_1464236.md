### B. 알고리즘 요약

1. 자기기록(magnetic recording) PR 채널에서 LDPC의 error-floor를 저BER 영역까지 관측하기 위한 FPGA 시뮬레이션 플랫폼을 제안.
2. 문제: C 시뮬레이션은 순차적이라 PR 채널 10^-9 BER 관측에 2GHz PC로 수개월 소요 → HW 가속 필요.
3. 구조: write processor(RDG→LDPC ENC→interleaver→ideal PR 채널+AWGN)와 read processor(SOVA→LDPC decoder)로 구성.
4. SOVA(soft-output Viterbi)가 초기 soft 정보를 생성하고 LDPC 디코더가 개선 → 한 번의 채널 이터레이션(turbo-equalization) 구성.
5. 다음 채널 이터레이션에서 SOVA가 LDPC 출력을 받아 prior 정보를 재계산하는 반복 구조.
6. 모듈러 설계: bit width, iteration 수, SNR을 재구성 가능하게 하여 유연성 확보.
7. 단일 디코더로 block length·code rate·code structure를 바꿔가며 구조화 LDPC(QC-LDPC)를 복호 [ref3].
8. 평가 대상: Array code와 Disjoint Difference Set(DDS) QC-LDPC 부호, 낮은 column weight, 채널 iteration=2.
9. 결과: DDS 부호(column weight j=5, LDPC 25 iter, 채널 2 iter, EPR4 채널)의 error-floor 관측; 샘플 처리속도 100M samples/s(채널 1 iter), 2 iter시 절반; 1.5시간에 5E-R 관측.
10. 한계: 부호설계·디코더 알고리즘 자체 개선 없음, 자기기록 PR/SOVA 전용 평가 툴, NAND와 무관.

| SOVA turbo-equalization | 대응 없음 | PR/자기기록 전용, NAND(AWGN/RBER) 채널과 무관 |
