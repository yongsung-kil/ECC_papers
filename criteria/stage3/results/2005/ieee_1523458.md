### B. 알고리즘 요약

1. `p`(홀수 소수)로 `p×p` 순환치환행렬 `P`와 하삼각형 `T` 정의. `M=pj`, `N=p(j+k)`, 부호길이 `N`, 차원 `N-M`.
2. 패리티검사행렬 `H=[H(p) | H(d)]`: `H(p)`는 `T`와 `I` 블록의 bi-diagonal형(RA류) 패리티부, `H(d)=(P^{(l+1)i})`는 RS형(quasi-cyclic) 정보부.
3. 문제: 정칙(regular) 대수 LDPC는 HW친화적이나 성능이 irregular보다 낮음 → "대수적으로 구성 가능한 고성능 irregular LDPC"를 목표.
4. 성질: `H(d)`는 length-4 cycle 없음(Fan [4]) → Tanner graph girth = 6 (Thm 2.4), 최소거리 ≥ `j+1` (Thm 2.5).
5. 효율 인코딩: 정보벡터를 `p`-bit 블록 `k`개로 나눠 Horner법으로 `v_i = Σ_l P^{i(l+1)} u_{d,l}` 계산(RS 신드롬 계산과 유사) → `u_{p,i}+u_{p,i+1}=v_i` 후진 대입 → `T u_{p,0}=...` feedback accumulator로 마무리.
6. 인코딩 회로: `p`-bit DFF + 순환시프트(shift by `i`) + feedback accumulator (Fig 3,4)로 구성, HW친화적.
7. Fossorier형 일반화: `H(d)`를 girth 최대 12까지 설계 가능한 Fossorier QC-LDPC(`P^{q_{l,m}}`)로 대체 → `C_F` (Thm 2.6).
8. irregular화: mask법(Chen/Hocevar) — 목표 degree분포 `L(x),R(x)`에 맞춰 `j×k` masking matrix `W`를 설계, `H(d)`에 `W∘A`로 masking → 임의 degree분포 달성 (Thm 3.1: min-dist ≥ lmin+1).
9. 결과: (a) 길이 1044 rate 1/2, SP 100 iter → R&U irregular보다 약간 우수. (b) 길이 4148 rate 0.8235 → R&U보다 우수, YRL과 거의 동등(YRL은 BER 1e-6에서 error floor).
10. 한계: HW 합성/게이트수 없음(회로 개념도만), 보증 최소거리 약함(j+1)·girth 6, error-floor 개선을 실측으로 입증하지 않음.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| RS형 순환행렬+mask 기반 irregular QC-LDPC 구성 | ecc_top.cpp Load_PCM() | H-matrix 재설계 필요, Prime ECC는 고정 QC-LDPC라 이식 부담 큼 |
| RA형 효율 인코딩 (Horner + feedback accumulator) | encoder.cpp LDPC_encoder_4KB(), Generate_PCM_encoding() | dual-diagonal 인코딩과 유사 → 이미 보유, 중복 |
| min-dist≥j+1 / girth 6 보증 | 대응 없음 | 보증이 약함, Prime ECC 부호 대비 이점 불명 |
```

적용 가치: 낮음 — binary 고rate QC-LDPC 구성이라 도메인은 정합하나, 효율 인코딩(dual-diagonal)과 QC 구조를 이미 보유해 중복이고 보증(girth 6 / min-dist j+1)도 약하다.
