### B. 알고리즘 요약

1. 대상: 유한체 `GF(q)` 곱셈군 기반 binary QC-LDPC. AWGN(SPA)+BEC(iterative) 양 채널, BPSK, 최대 iteration 50/100.
2. 문제: 랜덤 LDPC는 waterfall 우수하나 인코딩 복잡, 구조적 QC-LDPC는 shift-register로 선형복잡도 인코딩 가능 → 성능·error-floor·인코딩을 동시에 만족하는 대수적 구성이 목표.
3. 위치벡터: `GF(q)` 비영원소를 `(q-1)`-tuple로 표현하면 `αδ`의 위치벡터는 `δ`의 순환천이 → circulant permutation matrix 생성 기반.
4. 구성1: 곱셈군 순환부분군 `{β^0..β^{m-1}}`(β=α^k)에 affine mapping `y=α^i·x+1` 적용해 `(q-1)×m` 행렬 `M` 생성, 각 원소를 위치벡터로 치환 → circulant/zero 배열 `H`.
5. 핵심 성질(Theorem 1): `α^i·w_i`와 `α^l·w_j`는 최대 1위치만 일치 → RC-constraint 만족 → Tanner graph에 길이-4 cycle 없음(girth≥6).
6. 부분배열 `H(γ,ρ)` 선택으로 다양한 rate/length: null space가 열무게 `γ`, 최소거리 `≥γ+1`(짝수 γ면 `γ+2`), minimal stopping set `≥γ+1`인 regular QC-LDPC.
7. 구성2: `2×2` 부분배열은 girth-8이라 상삼각(diagonal `2×2` 부분배열)+하단 identity 행 구조로 `(3,ρ)`-regular girth-8 QC-LDPC 생성(BEC에 특히 효율).
8. 확장: masking 기법([15])으로 density-evolution 설계된 degree 분포를 배열에 적용해 irregular QC-LDPC 생성 → waterfall 개선하며 QC 구조 유지.
9. 결과: 예1 (961,721) rate0.75 SPA로 BER1e-6에서 Shannon 1.7dB, error-floor 1e-13 이하 추정. 예2 (2048,1027)~(32768,31747) rate0.50~0.968 고rate 코드군, (16384,8192) masked irregular는 Shannon 1dB.
10. 한계: HW 미구현(circulant→routing 이점만 언급); error-floor는 추정치이며 저BER 실측 없음; column weight↑로 error-floor↓ 확인은 정성적.
