> 총평: difference family의 m 제약(prime power ≡1 mod w(w-1))을 EDF로 완화해 code size 유연성을 얻는 girth-6 고rate QC-LDPC 구성법으로, 부호설계 아이디어는 흥미로우나 Prime ECC는 특정 QC-LDPC 고정이라 부호 교체 이식부담이 크다.

### B. 알고리즘 요약

1. 대상: rate `(t-1)/t` (≥0.9) high-rate short/medium QC-LDPC, column weight `w`=3~5, AWGN·binary signaling.
2. 문제: 기존 difference family(DF) 기반 QC 구성은 `m = t·w(w-1)+1`이 prime power여야 해 code size 선택폭이 좁음.
3. 정의: `(w,t)-EDF`는 `w`개 정수 집합 `t`개로, 모든 intra-set distance(거리)가 서로 다른 집합족; DF와 달리 group 원소 차가 아닌 자연수 거리만 요구해 제약이 느슨.
4. 핵심기법: 각 `Bl`로 `m×m` circulant `Hl`를 만들어 `H=(H1,...,Ht)` parity-check 구성, `hl(x)`는 `Bl`로 결정.
5. 핵심 조건: `m ≥ 2·dmax+1`이면 length-4 cycle 부재(girth≥6) 보장 (Lemma 3).
6. dmax 하한: `dmax ≥ tw(w-1)/2` (Lemma 2), dmax 최소화가 더 유연·고rate 설계에 유리.
7. 탐색: 각 `Bl={0,l}`에서 시작해 거리 중복 없는 최소 정수를 순차 삽입하는 heuristic 알고리즘으로 EDF 탐색(최적성 미보장).
8. minimum distance: `w+1 ≤ dmin ≤ 2w`, weight-2w 코드워드 최소 `(t choose 2)·m`개.
9. 인코딩: circulant 하나(`Ht`)를 lower-triangular로 강제해 `HtpT=cT`를 재귀적으로 O(m) 복잡도로 풀이(shift register 기반).
10. 결과: rate-0.9 (63,3,10)/(139,4,10)/(247,5,10) [길이 630/1390/2470] 코드를 100 iter message-passing로 시뮬, importance sampling으로 Pcw=1e-8까지 관측해도 뚜렷한 error floor 미관측; HW 미설계.

```
| girth-4-free QC-LDPC 부호설계 | ecc_top.cpp Load_PCM() (H-matrix) | 우리도 QC-LDPC지만 특정 부호 고정 - 이식 난이도 높음 |
| lower-triangular Ht 재귀 인코딩 | encoder.cpp LDPC_encoder_4KB(), Generate_PCM_encoding() | 우리는 dual-diagonal 인코딩 - 방식 상이 |

```

적용 가치: **중간** — 고rate·girth-6 QC-LDPC 구성 유연성 확보라는 방향은 NAND ECC와 결이 맞으나, Prime ECC는 z=32 특정 QC-LDPC가 고정이고 부호 교체는 프로파일 §4에서 이식 난이도 "높음"으로 분류된 영역이라 직접 적용가치는 제한적이다.
