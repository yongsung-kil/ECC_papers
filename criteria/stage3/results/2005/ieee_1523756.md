### B. 알고리즘 요약

1. 대상: `(v,k,1)`-cyclic difference family(CDF)에서 유도한 binary QC-LDPC, AWGN memoryless 채널, SPA 복호(최대 100 iteration).
2. 문제: column weight 3 QC-LDPC는 codelength가 길면 성능 악화 → girth·최소거리를 키워 고rate(≥0.8)·moderate length(<10000) 자기저장/광통신용 코드가 필요.
3. 기반: `(v,k,1)`-BIBD의 point-block incidence matrix를 H로 쓰면 λ=1이라 길이-4 cycle 없음(girth≥6); `gcd(v,k)=1`이면 각 base block이 `v×v` circulant → QC 구조.
4. 핵심 기법: base block `Di`를 `Di,j`/`Di,j`로 분할, 확장행렬 `M^t(Di)`(`t×t` circulant 배열)를 만들어 `H̄=[M1^t...Ml^t]` 구성 → codelength `vlt`, rate `≥(l-1)/l`.
5. 핵심 식(girth-8 조건): `M^t(D)`는 `t(mod 3)≠1 and t>3`이면 6-cycle 없음; 여러 블록 결합 시 `2ΔDi∩ΔDi'=∅`, `(ΔDi+ΔDi')∩ΔDi''=∅` (식10~12) → girth≥8 (Lemma 1).
6. 최소거리(k=3): 상한 `dmin≤2(k-1)k`=12 (Lemma 2), 하한 `dmin≥10` (Lemma 3, weight≤8 codeword가 6-cycle을 강제해 존재불가) → `10≤dmin≤12` (Theorem 1).
7. 트레이드오프: `|D|`가 커지면 girth-8 조건 만족이 어려워 rate>0.9 girth-8 코드는 사실상 구성 불가(저자 명시).
8. 학습/최적화: 없음(대수적·조합적 구성, DE/GA/NN 미사용).
9. 결과: 예2 C1 (9905,8490) rate0.857 girth-8 dmin≥10 → C2(girth-6)가 BER 1e-4 error-floor인 반면 C1은 1e-6 이상까지 error-floor 없음. 예3 C5 (9640,8435) rate0.875 girth-6.
10. 한계: HW 미설계; girth-8이 고rate(>0.9)에서 불가; column weight 3 고정으로 dmin≤12 제한; SPA 시뮬만.
