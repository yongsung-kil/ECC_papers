### B. 알고리즘 요약

1. 시스템: BIAWGN + BPSK, binary regular/irregular LDPC. 테스트부호 RS-LDPC(2048,1723), PEG-LDPC(2048,1664), EG-LDPC(1023,781).
2. 문제: IDBP/SPA는 성능 최고지만 반복마다 곱셈·나눗셈·exp·log 다수 필요 — 복잡도 과다.
3. 모델: 채널 LLR `Li=ln(P(ci=1|yi)/P(ci=0|yi))`, 신뢰도 `|Li|`, 초기 경판정 `z(0)`.
4. Algorithm I(경판정): 매 iteration 각 비트 i에 대해 `S_i=wt((z+ui)·H^T)` 계산 후 syndrome weight 최소가 되는 한 비트만 flip.
5. 근거: low-density H에서 syndrome weight가 평균적으로 오류 수에 비례 증가 → 감소 방향 flip이 정정에 유효.
6. Algorithm II(연판정): `S_i`에 신뢰도 페널티 `α·wc·|Li|` 가산 — 신뢰도 높은 비트의 flip을 억제. α=0이면 I과 동일.
7. 구현 디테일: α는 목표 BER(10^-5)에서 SNR별 시뮬로 최적화. 실수 덧셈은 code symbol당 1회뿐, 나머지는 이진 연산.
8. Loop detection: `j(k)=j(k-1)`이면 진동 → 차선 `S_i` 비트 선택. `n_j`, `w(k')` 반복적 계산으로 loop 탐지, block length N 무관 비용.
9. 결과: BER 10^-5에서 Alg I이 BF 대비 0.2~0.5dB 개선, Alg II(LD)가 IWBF 대비 0.1dB, IDBP 대비 0.53dB 열세. LD가 0.3~0.4dB 추가 개선.
10. 한계: HW 미설계, 시뮬만, kmax=200으로 반복수 큼(IDBP 50), BP 대비 여전히 성능 열세.
