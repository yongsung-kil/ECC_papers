### B. 알고리즘 요약

1. 시스템: EPR4-equalized 자기기록채널(MRC)에서 random noise+erasure를 정정하는 systematic irregular LDPC (`PCM` 512×4096, column weight 3, PEG 구성).
2. 문제: LDPC 복호는 에러 비트가 많을 때 성공 판정까지 iteration이 오래 걸림 → 속도·CER 저하.
3. 모델: AWGN prior + 다중 CRC 프레임 포맷([5][6]). systematic part를 여러 세그먼트로 나눠 각 세그먼트에 개별 CRC 부착.
4. 핵심기법: IBP(iterative belief propagation) 루프 중 각 세그먼트의 checksum을 계산, checksum=0인 세그먼트의 변수노드를 "임시 정확"으로 판정.
5. 핵심식(5): 정확 판정된 세그먼트의 변수노드 메시지를 결정론적(large) LLR 값으로 고정 → 이웃 노드로 강한 메시지 전파해 수렴 가속.
6. 확장: updatable indicator를 0으로 두면(신뢰 CRC) 해당 세그먼트 LLR 재계산을 생략해 연산량 절감, 1로 두면(불신 CRC) 연산은 유지하되 large message 효과만 사용.
7. 구현 디테일: 복호는 이중 종료 판정 — Step5(CRC 정상복호), Step6(syndrome=0 오류복호 종료).
8. 최적화: CRC 선택은 미검출확률 P_ud (식(6), 세그먼트 길이·BER·생성다항식 weight 종속) 기준, [7, Table I]의 검증된 CRC 사용(전 코드공간 탐색은 생략).
9. 결과: 제안법이 동일 rate에서 CER·iteration 수 모두 개선, 긴 CRC로 P_ud가 매우 작을 때 error floor가 사라짐(Fig.4), erasure 증가 시 이득 확대.
10. 한계: HW 미설계, 시뮬만; erasure 채널 특화; 다중 CRC로 인한 rate 손실; 정량 gatecount/throughput 없음.
