### B. 알고리즘 요약

1. 대상은 short·low-rate(rate<0.5) 이진 AWGN LDPC로, 짧은 부호의 심한 error floor를 낮추는 부호구조 제안.
2. 문제: 긴 부호는 좋은 LDPC 설계가 쉽지만 short·low-rate에선 error floor가 심해 최소거리 확보가 어려움.
3. 핵심 기법: LDPC/ARA 이분그래프의 일부 SPC constraint node를 Hamming 성분부호 node로 치환("Hamming doping") → generalized LDPC(Tanner code, D-LDPCC/HD-LDPCC).
4. Hamming 성분부호(최소거리 3)로 doping하면 전체 부호 최소거리↑ → error floor↓, rate는 `Rc ≥ 1 - m/n` (m=성분노드 총 redundancy).
5. 복호: message-passing 골격은 LDPC와 동일하나 doped node의 soft-output을 성분부호 trellis의 BCJR로 정확 계산(복잡도 `2^(n'-k')`).
6. 성분복호 대안: modified-Chase 또는 성분 PCM에 local BP — `(6,3)` shortened Hamming은 cycle-4 free라 local BP 손실 <0.1dB.
7. 설계도구: Gaussian approximation 기반 density evolution으로 doped node 출력 p.d.f.를 히스토그램으로 특성화, mean 대신 SNR/mutual-info를 추적해 threshold 최적화.
8. `(512,160)` HD-LDPCC-I(rate 0.3125, `(8,4)` ext. Hamming 32개 doping): FER 3e-7까지 floor 없음, EG LDPCC(511,139) 대비 0.1dB↑, constraint node 364개 vs EG 4599개.
9. `(512,140)` HD-LDPCC-II(rate 0.273, `(6,3)` Hamming), `(2048,928)` HD-LDPCC(DE 설계, threshold Eb/N0=0.68dB): 모두 FER 5e-7 이하까지 floor 없음.
10. 한계: HW 미설계, rate 하락 수반, doped node 성분복호(BCJR/Chase)가 SPC min-sum보다 무겁고 저SNR서 Gaussian 근사 정확도 저하.
