### B. 알고리즘 요약

1. 대상: iterative(LDPC/turbo) 디코더 내 "information combining" 연산자(variable node=LLR 합, check node=parity 결합)를 정보이론적으로 분석.
2. 풀려는 문제: EXIT chart는 mutual information 한 파라미터만 추적하며 입력 메시지 밀도에 모델 가정을 두어 rigor 부족 → 극단 입력 분포로 엄밀한 bound 대체.
3. 채널 모델: (i) binary-input symmetric parallel broadcast channel(단일 비트의 다중 관측 = variable node), (ii) parity-constrained-input parallel channels(패리티 제약 = check node).
4. 핵심 결과 1(Thm 1,2): broadcast 채널에서 주어진 입력 mutual info에 대해 출력 정보를 최대화하는 극단 채널은 `BEC`, 최소화는 `BSC`.
5. 의미: 최적 결합이 LLR 도메인 합이라는 사실과 "Mrs. Gerber's lemma"로 극단분포가 다른 구성채널 분포와 무관하게 결정됨(놀라운 결과).
6. 핵심 결과 2(Thm 3): check node(parity 제약)에서는 반대로 `BEC`가 최소·`BSC`가 최대 정보 채널.
7. 응용(Thm 4~6): BEC/BSC 치환으로 EXIT 곡선의 best/worst bound → per-symbol MAP(sum-product) 복호 임계값의 상한·하한 도출, Corollary 2로 특정 앙상블의 capacity 미달성 판정.
8. 확장: 입력이 memoryless 아니고 1차 균등분포여도, 또는 단일 비대칭 구성채널이 있어도 정리 성립(대칭성이 필요조건임을 Z-channel 반례로 확인).
9. 결과(수치): BIAWGN에서 하한은 정확 임계값보다 ~1.2 dB 아래, regular 부호 상한은 정확값에서 ~0.5 dB, Burshtein-Miller [11] bound와 비교; irregular 부호는 상한과 6~7.5 dB 격차.
10. 한계: 순수 이론(정리·증명), 실제 디코더 구현·부호 구성·HW·시뮬 없음, min-sum 등 suboptimal combiner는 미분석.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| variable node 정보결합(LLR 합) 분석 | `decoder.cpp` `VN_Cal_HD()` 등 | 이론적 극단분포 분석일 뿐 min-sum VN 연산 개선 아님 |
| check node 정보결합 분석 | `decoder.cpp` `C2V_Cal()`, `CNU_Update_New_Mag()` | check node bound 이론, 실제 CN 연산 변경 없음 |
| 복호 임계값 상·하한/EXIT bound | 대응 없음 | Prime ECC는 임계값 해석 도구 미탑재, 이론 결과 소비처 없음 |

적용 가치: **낮음** — LDPC 복호 임계값의 정보이론적 상·하한을 유도한 순수 이론으로, NAND binary QC-LDPC 디코더/부호설계/HW에 떼어 쓸 알고리즘 산출물이 없다.
