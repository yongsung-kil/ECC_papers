> 총평: 4bit min-sum 양자화·clipping·offset/조건부 보정으로 BP 근접시키는 고전 min-sum 개선의 원류 논문, decoder 루프에 직접 이식 가능하나 offset은 이미 표준화된 기법.

### B. 알고리즘 요약

1. 대상: 이진입력 AWGN 채널, 단·중길이 LDPC (코드 I: irregular (1268,456); II: (273,191); III: (3,6)-regular (8000,4000))를 min-sum(MS)으로 복호.
2. 문제: MS는 BP 대비 수 dB의 십분의 몇 열세이고, 양자화 시 성능 저하 + clipping이 early error floor 유발.
3. 핵심 관찰: check node 출력의 과신뢰(overconfident) magnitude를 임계값 `c`로 clipping하면 MS-BP 성능 격차가 부분적으로 닫힘. 최적 `c`는 SNR에 둔감하고 코드별로 다름(예: (8000,4000) 표준 MS는 `c≈1.25`).
4. 양자화: 수신값을 `±c`로 대칭 clipping 후 균일 양자화(`b` bits), 정수 연산. 결과적으로 **4bit면 이상적 성능에 근접, 때로는 상회** (BP는 LLR domain에서 6bit 필요).
5. 보정 개념: BP의 check node 연산에서 MS와의 차이는 correction factor `f(d)`(두 입력 magnitude 차의 함수, 항상 음수)로, MS 출력 magnitude에 더하면 BP와 근사됨.
6. 조건부 보정(Conditional A/B): 최소 magnitude 메시지 집합에서 반복적으로 `c1` 미만이면 `c2`만큼 magnitude를 낮춤(부호 불변). A는 매 반복, B는 한 번만 적용. 파라미터 시뮬 최적화.
7. 무조건 보정(Unconditional): 모든 outgoing magnitude에서 상수 `y`를 빼는 offset-MS(=Chen/Fossorier "offset BP-based"와 동일). 복잡도 낮음.
8. 트레이드오프: 4bit 이하 coarse 양자화에서는 조건부 보정이 무조건보다 우수, bit 수가 크면 둘이 근접 → 무조건(offset)이 더 매력적.
9. 결과: 코드 I에서 무조건보정이 BER 10^-5에서 BP를 약 0.2 dB 상회; 수정 MS는 최대 200 iteration, 코드별 최적 `c`(예 modified: 3/2/2.5)로 error floor 개선. 코드 I,III에서 4→5bit로 약 0.2 dB 추가 이득.
10. 한계: 순수 시뮬(HW 미설계), throughput/gate count 없음, gain은 코드/파라미터 최적화 종속.
