### B. 알고리즘 요약

1. 목표: capacity-achieving 부호 디코더의 power/speed 비 개선을 위해 아날로그 연속시간 반복 복호를 current-mirror 기반 current-mode 회로로 min-sum 구현.
2. 문제: 기존 아날로그 BP 디코더는 BiCMOS(고비용) 또는 subthreshold CMOS(저속)라 고속·저비용 아날로그 MS 디코더가 필요.
3. 부호/기술: `(n,k,dmin)=(32,8,10)` regular LDPC(VN degree 3, CN degree 4)를 0.18um CMOS로 구현.
4. 메시지 표현: LLR을 전류(부호=전류 방향)로 표현하고 current mirror가 크기 복제+부호 반전, n/p-type 조합을 `current buffer`로 사용.
5. VN 구현: Kirchhoff 전류법칙으로 입력 전류를 합산해 식(3)/(4)의 VN 출력 계산.
6. CN 구현: RTAS(부호/크기 분리) → min-WTA(winner-take-all로 최소 크기 선택) → sign XOR → ASTR(부호·크기 재결합)로 식(2) min-sum 수행.
7. 칩 아키텍처: 24 CN, 32 VN, 96 analog switch, on-chip `32*6b` memory + 5b current-steering DAC, 32:1 MUX 출력; 스위치 개방 시 메모리 자동 클리어.
8. 입력 양자화: `6b`(5b 크기+1b 부호)를 `5b`로 축소하고 DAC gain 2배 → 소전류 정확도 개선으로 성능 향상.
9. 실측: 저SNR steady-state에서 conventional SS-MS를 상회하고 SR-MS(successive relaxation) 모델에 근접; 24Mb/s에서 BER `10^-3` coding gain 손실 ~0.3dB(더 빠르면 최대 ~1dB), 전력 ~5mW, 코어 630x910um.
10. 한계: 고SNR에서 mismatch 등 imperfection으로 성능 열화; throughput 24Mb/s로 낮음; 아날로그 연속시간이라 디지털 이식 불가.
