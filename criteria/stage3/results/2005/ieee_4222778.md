### B. 알고리즘 요약

1. 시스템: `Nt`/`Nr` 안테나·`Nc` sub-carrier MIMO-OFDM BLAST에 고rate irregular LDPC를 channel code로 결합, ~1Gbps 무선 전송 목표.
2. 문제: OFDM-BLAST-a(전체 stream 부호화)는 성능 좋으나 긴 code length·높은 processing rate 필요; BLAST-b(sub-stream 독립부호화)는 공간 다이버시티 못 씀.
3. 제안: OFDM-BLAST-c — 변조심볼을 공간/주파수/시간 자원배열에 indexing set `S`(식 `(1)`)로 분산해 BLAST-a 수준 다이버시티를 짧은 code length로 달성.
4. LDPC 규칙1: 고rate에서 degree-2 v-node만의 short cycle 방지, `NV(2)=N-K-1`로 density evolution으로 `λ(x),ρ(x)` 최적화.
5. 규칙2: parity 부분 `p2`를 bi-diagonal 구조로 → 저복잡도 인코딩 + degree-2 short cycle 제거.
6. 규칙3: info 부분 `p1`을 ACE 알고리즘으로 cycle 회피 구성.
7. 부호: `R=3/4`, `λ(x)=0.125x+0.446x²+0.4078x^10+0.0213x^11`, `ρ(x)=x^15`.
8. 복호: sum-product 알고리즘 최대 100 iteration; BLAST 검출은 list sphere decoding.
9. 결과: 6-path Rayleigh fading에서 OFDM-BLAST-c가 OFDM-BLAST-a와 동일 PER, OFDM-BLAST-b보다 우수(특히 고 `Eb/No`).
10. 한계: 무선 MIMO-OFDM 전용, LDPC 구성은 기존 기법 조합, 시뮬레이션만(HW/양자화/gate count 없음).
