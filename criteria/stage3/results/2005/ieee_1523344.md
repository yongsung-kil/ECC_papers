> 총평: LLR을 유한 support로 truncate하면 suboptimal LP 복호가 오히려 개선됨을 pseudo-codeword로 증명한 순수 이론. LLR 클리핑의 이론적 정당화로 통찰은 있으나 Prime ECC가 이미 LLR 양자화를 하므로 신규 이식 산출물은 없음.

### B. 알고리즘 요약

1. 대상: 길이 n, rate r=1-dv/dc의 (dv,dc)-regular binary LDPC, BI-AWGNC, BPSK, LP 복호.
2. 문제: LP/BP 같은 suboptimal 복호는 실제 codeword가 아닌 `pseudo-codeword`(fundamental polytope P의 fractional vertex)를 끌개로 오판 → error-floor 유발.
3. 세팅: 수신 LLR `λ_i`에 매핑 `μ`를 적용해 수정 LLR `λ'_i=μ(λ_i)`로 복호. 최적 복호면 `μ_triv`가 최선(data-processing)이나 suboptimal이면 다름.
4. 핵심 질문: LP 복호 전 LLR을 truncate/quantize하면(=`μ_{T,W}` 유한 절단, `μ_{Q2,L}` 2-level) 도움이 되는가.
5. canonical completion(Def.2): 변수노드 root에서 tier `2t`의 bit에 `ω̃_i=1/(dc-1)^t` 부여한 pseudo-codeword ω, `wp_AWGNC(ω)=||ω||1²/||ω||2² ≤ β_{dv,dc}·n^{β_{dv,dc}}`, `β<1`.
6. 결과1(trivial map): `μ_triv` 사용 시 LP 복호 reliability function이 임의 고정 SNR에서 0으로 수렴(ML은 양수 유지) → LP가 pseudo-codeword로 약함.
7. 결과2(quantize): `μ_{Q2,L}`(1-bit)은 BSC와 동치, expander 기반 [6] 결과로 `P_err ≤ n·exp(-K'n)` → error exponent 양수.
8. 결과3(threshold, Thm2): 임의 유한 `W≥1`, 임의 rate r, 충분히 작은 σ²에서 `μ_{T,W}` 하에 error exponent가 양수인 (dv,dc)-regular expander 부호족 존재 → 이득의 본질은 양자화가 아니라 LLR을 유한 support로 제한하는 것.
9. 증명 도구: (αn,δdv)-expander, δ-matching, dual witness(feasible edge weight τ) — `dv>4(4W+2)`, `δ>1-1/(4W+2)` 조건으로 LP 성공을 `1-2^{-Ω(n)}` 확률 보장.
10. 한계: 순수 asymptotic 존재성 이론, 시뮬/HW/구체 부호 없음, LP 복호 전용(min-sum은 관계만 인용).

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LLR truncate/thresholding 매핑 μ | channel.cpp Set_LLR_Th(), Set_R_Offset() | Prime ECC가 이미 LLR 클리핑/양자화 수행, 신규 매핑 아님 |
| LLR 양자화 테이블 | ecc_data.h PARAM_LLR, decoder.cpp Get_VNU_Table_Idx() | 유한 support 이득이 양자화 테이블 설계 정당화에 통찰 |
| LP 복호 / pseudo-codeword | decoder.cpp LDPC_Decoder() | Prime ECC는 min-sum(LP와 관계 인용), LP 미구현 |
| (dv,dc)-regular expander 부호 | ecc_top.cpp Load_PCM() | Prime ECC는 QC-LDPC, expander-random regular와 부호구조 상이 |
```

적용 가치: 중간 — LLR을 유한 support로 클리핑하는 것이 suboptimal 복호에서 error-floor(pseudo-codeword)를 개선한다는 이론적 정당화는 LLR 양자화 테이블 설계 통찰이 되나, LP 복호 순수이론이고 Prime ECC가 이미 LLR 양자화를 수행해 직접 이식할 신규 기법은 없음.
