### B. 알고리즘 요약

1. 대상: binary QC-LDPC, `L×L` circulant permutation matrix `P` 기반, exponent matrix `E(H)=(aij)`로 표현(exponent coupling `H=E∘P`).
2. 문제: 큰 길이 QC-LDPC를 직접 설계하기 어렵고 짧은 cycle이 성능을 저하 → 작은 부호들을 결합해 girth 보장하며 확장하려 함.
3. 핵심가정: 모든 component `Hk`의 mother matrix `M(Hk)`가 동일하고 `gcd(Ll,Lk)=1` (서로소).
4. 핵심기법: CRT로 exponent 결합 `cij ≡ aij·A1L2 + bij·A2L1 mod L1L2` (식5), 여기서 `A1L2≡1 mod L1`, `A2L1≡1 mod L2`.
5. 식 의미: 서로소 lifting size의 부호들을 CRT로 합쳐 길이 `L1L2`의 단일 circulant 구조로 사상.
6. girth 정리(Thm 2): block-cycle이 유발하는 cycle 길이가 `r=r1r2`가 되어 결합 부호의 girth ≥ 각 component의 girth → 4-cycle 제거 보장.
7. 적용: array code `C(p,m)` (`Eij=(i-1)(j-1)`, girth 6)를 component로 사용, 서로 다른 소수 `p1,p2`로 결합 시 `mp1p2 × p1²p2²` 부호가 4-cycle 없음(Thm 3).
8. 구현 이점: QC 구조로 PCM 저장 메모리 `1/L`로 축소, 인코딩·메모리 접근 효율.
9. 결과: AWGN에서 BER/FER 시뮬, random regular LDPC 대비 성능 저하 미미, column weight>3이면 FER=1e-4까지 error floor 없음.
10. 한계: row block 수 고정으로 저rate 대형 부호 직접 구성 불가, 성능이 random 대비 "거의 동일"(gain 없음), HW/디코더 미설계, 정확 rate/length는 본문 미기재.
