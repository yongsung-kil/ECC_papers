> 총평: 선형 최소거리를 보장하는 저rate LDPC/터보 하이브리드(TLDPC) 구성법. 이론적으로 흥미로우나 고rate NAND QC-LDPC로 직접 이식은 불가.

### B. 알고리즘 요약

1. `Cbase`(base code) + 랜덤 이분그래프로 코드 `C`를 정의하는 일반 구성법 제시: LDPC(단일패리티 base) / 병렬터보(convolutional base)를 특수 케이스로 포함.
2. 풀려는 문제: 캐패시티에 근접하는 부호(irregular LDPC/turbo)는 최소거리가 작아 error-floor가 높음 → "점근적으로 good(최소거리 ∝ 길이)"이면서 저복잡도 반복복호가 되는 부호를 찾음.
3. 핵심 도구: `graph of codewords of weight 2` (degree-2 비트와 base code weight-2 codeword로 유도되는 그래프). 이 그래프의 짧은 cycle이 낮은 최소거리를 유발.
4. 조건: 이 그래프에서 sub-linear 크기 cycle을 회피(길이 `l ≥ δn`인 cycle 제거)하면 weight-w codeword 평균수 `a_w`가 `o(1)`로 낮아져 대부분의 부호가 asymptotically good.
5. 구현: tail-biting trellis base code 두 종류 — Family A(section당 2bit, `C0={00,11}`), Family B(section당 3bit). EXIT chart가 직선에 가까워 degree-2 비트 다수 사용 가능.
6. degree 분포 `Λ(x)`를 최적화(예 Family B: `0.5x + 0.182x^3 + 0.069x^12 + 0.249x^13`), Family A는 `λ2<1`, Family B는 `λ2=1/2`로 cycle 회피.
7. 증명: 생성함수 + 조건부확률로 `a_w` 명시식 유도 → 지수 `α(ω)=Cω log ω + O(ω)` (`C>0`) 도출 → Markov 부등식으로 최소거리 `≥ Cn`인 부호 비율이 `1-o(1)`.
8. 결과1: Family-A rate 1/3, 길이 15330, 100 iteration으로 동급 8-state 터보(길이 15432) 대비 BER 우위.
9. 결과2: Family-B rate 1/2, 길이 16000이 WER `2·10^-4`에서 duo-binary 터보 대비 0.6 dB 우수, 평균 iteration 25~40, iteration당 복잡도는 터보의 약 1/3.
10. 한계: HW 미설계, 저rate(1/2,1/3)만 예시, AWGN 시뮬 위주, tail-biting trellis 구조가 표준 LDPC와 이질적.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| TLDPC 부호 구성 (base code + 이분그래프) | 대응 없음 (`ecc_top.cpp` `Load_PCM()`) | QC-LDPC base+circulant shift와 완전히 다른 트렐리스 base-code 구성, 이식 불가 |
| 트렐리스 base code 반복복호 | 대응 없음 (`decoder.cpp` `LDPC_Decoder()`) | min-sum CN/VN 연산과 다른 트렐리스형 복호, 대응 없음 |
| weight-2 그래프로 최소거리/error-floor 보증 | 대응 없음 | Prime ECC는 고정 QC-LDPC H-matrix, 부호 재설계·재검증 부담 큼 |

적용 가치: 낮음 — 저rate 트렐리스 하이브리드 부호로 고rate NAND binary QC-LDPC와 구조가 이질적이고 HW 설계도 없어 이식 실익이 없음.
