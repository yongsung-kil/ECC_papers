### B. 알고리즘 요약

1. 시스템: binary irregular LDPC, `H=[H1|H2]`, `H2`는 dual-diagonal → systematic·efficiently-encodable. rate 0.5/0.75/0.9375, n=3728~6416.
2. 문제: 전통 IR-LDPC는 양측 degree 패턴을 컴퓨터탐색으로 정해 구성복잡도가 높음 → 탐색 없이 구성하려는 목표.
3. 모델: sparse `H1`을 벡터로 표현하는 `V-Matrix`(각 행이 1의 위치를 지정하는 정수열) 도입.
4. 핵심기법1: 소수 `N=4q+1`에 대해 `S=Λ·Λ^T`, 원소별 `mod N`으로 V-Matrix `V` 생성 후 각 행을 N회 cyclic shift → circulant 블록 `V'`.
5. 의미: cyclic shift와 loop graph 규칙으로 girth-4(정사각형 구조)를 배제 → BER 열화/error-floor 유발 short cycle 제거.
6. 핵심기법2: 짝수번째 circulant(`V2,V4,...`)의 행순서를 reverse하고 마지막 두 열 제거 → 인접중복 제거, girth-4 완전 배제.
7. 디테일: 각 V-Matrix 정수를 1×N permutation matrix로 매핑 → sparse `H1`; `H2` dual-diagonal 결합. 열가중치 W, rate R로 부분행렬 선택.
8. 최적화: 학습 없음, 대수적 구성(모듈러 곱+cyclic shift)만 사용해 탐색 제거.
9. 결과: AWGN sum-product 복호, BER 10^-6에서 Shannon gap 2.7/2.5/1.7/1.3 dB (rate 0.5/0.5/0.75/0.9375); rate 0.9375에서 π-rotation 부호 대비 고SNR 우세.
10. 한계: HW 미설계·시뮬만, error-floor/girth 수치 미제시, 정정성능은 기존 π-rotation과 대동소이.
