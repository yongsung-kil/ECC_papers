### B. 알고리즘 요약

1. 대상: 순환치환행렬(`P^a`, ∞=zero)로 구성된 `mL×nL` QC-LDPC PCM, mother matrix `M(H)`(m×n binary) + exponent matrix `E(H)=(a_ij)`로 표현. rate `1-m/n`.
2. 문제: 유한길이 QC-LDPC 성능은 girth·cycle에 좌우되나, 큰 길이 부호를 매번 새로 설계·저장하기 비효율적.
3. cycle 이론: 2l-block-cycle의 exponent chain이 `r·Σ(-1)^(i-1)a_i ≡ 0 mod L`을 만족하는 최소 r로 길이 2lr cycle 유발(Fossorier Prop.1).
4. girth 상한 (Theorem 2/4): mother matrix에서 두 block-cycle이 p개 overlap → girth ≤ `2(2l+2k-p)`; p blocks로 연결 → cycle 길이 `4(l+k+p)`. 즉 mother matrix 구조가 girth를 근본적으로 제한(Corollary 3: zero 없는 regular는 girth≤12).
5. 핵심 의미: L·exponent를 아무리 키워도 mother matrix의 overlap/connection이 있으면 girth를 무한정 못 늘림 → mother matrix 신중 선택 필요.
6. lifting: `L_{k+1}=q_k·L_k` 조건에서 작은 부호 `E(H_k)`로부터 `a_ij^(k+1) = a_ij^(k)+c_i·L_k` 후보 중 girth 최대·최단cycle 최소를 저degree 열부터 선택해 큰 부호 생성.
7. 부수: 모든 `C_k`가 같은 mother matrix 공유 → `E(H_k) ≡ E(H_s) mod L_k`, 따라서 `E(H_s)` 하나만 저장하면 modulo-L_k로 모든 길이 재생성(메모리 효율).
8. Theorem 6: lifting 시 `r_{k+1}`은 `r_k`의 배수 → girth는 절대 감소하지 않음(비감소 보장). 단 Step2 exponent 제약으로 suboptimal.
9. 결과(AWGN 시뮬): 24×48 mother irregular 및 28×48 MET(rate 1/2) 부호, (80000,40000,2000)은 FER=10^-4에서 error-floor 없이 Shannon 대비 0.39dB; lifting 많이 할수록 성능 소폭 저하·메모리 효율 향상(trade-off).
10. 한계: HW 미설계, AWGN 시뮬만, rate 1/2 무선/OFDM 맥락, lifting 반복 시 exponent suboptimality로 성능 저하.
