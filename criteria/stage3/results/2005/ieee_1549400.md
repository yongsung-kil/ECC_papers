### B. 알고리즘 요약

1. 문제: 랜덤 LDPC는 waterfall은 좋지만 error-floor·인코딩 복잡도 문제, 구조적 LDPC는 이를 완화하나 AWGN/BEC 양쪽 동시 우수 부호가 드묾.
2. 채널·목표: BPSK/AWGN(SPA, 최대 100 iter)과 BEC(erasure 복호) **양쪽에서 동시에 잘 동작**하는 regular/irregular LDPC를 대수적으로 구성.
3. 핵심 기법 1단: m차원 Euclidean geometry `EG(m,q)`의 connecting line들을 parallel bundle로 분해(`geometry-decomposition`)하여 `n x n` permutation 행렬들의 `n x q` 배열 `A`를 얻음.
4. 성질: `A`는 RC-constraint(두 행/열이 1을 최대 1개 공유) 만족 → Tanner graph의 length-4 cycle 제거, girth>=6 보장.
5. 부호 생성: `A`의 `γ x ρ` 부분배열 `H(γ,ρ)`의 null space가 `(γ,ρ)`-regular LDPC. 길이 `N=ρn`, rate>=`(ρ-γ)/ρ`, 최소거리>=`γ+1`(odd)/`γ+2`(even).
6. 성질의 의미: 큰 column weight `γ` → 큰 최소거리 + 큰 minimal stopping set(>=`γ`) → AWGN error-floor와 BEC 성능을 동시에 하한 보증.
7. 핵심 기법 2단(masking): base matrix `H`의 일부 permutation 블록을 masking matrix `Z(γ,ρ)`(binary)로 0으로 치환 → `M=Z⊗H`. degree분포를 임의(irregular)로 조절, 밀도 낮춰 short cycle 추가 감소.
8. Masking matrix 설계: primitive γ-tuple 또는 distance-3 Hamming codeword로 circulant 생성 → 원하는 `v(X)/c(X)` degree분포에 근사.
9. 결과: (1024,781) rate0.763는 AWGN error-floor<10^-12(추정<10^-25), Shannon 2dB 이내; (8192,*) 3종은 error-floor 없이 Shannon 1.1~1.5dB; (16128,8064) rate1/2 irregular는 BER 10^-6에서 Shannon 0.83dB, error-floor 없음(turbo 대비 우수).
10. 한계: 순수 부호구성(HW/디코더 아키텍처 없음), soft-read/양자화 논의 없음, degree분포는 채널별 재설계 필요(AWGN 최적≠BEC 최적).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| EG-decomposition으로 permutation 배열 PCM 구성 | `ecc_top.cpp` `Load_PCM()` | H-matrix/부호구조 로드 지점 — 우리 고정 QC-LDPC 대체 후보이나 재검증 부담 큼 |
| masking으로 degree분포/rate 조절한 base matrix | `ecc_top.cpp` `Load_PCM()` | base+shift 형태 PCM(`PCM_b`) 생성에 대응 가능하나 permutation 배열은 circulant-shift와 표현 상이 |
| girth>=6 / stopping-set 보증(error-floor) | 대응 없음 | 부호설계 단계 성질로, 디코더 코드 모듈엔 직접 대응 없음 |
| SPA 반복 복호 | `decoder.cpp` `LDPC_Decoder()` (단, 우리는 min-sum) | 논문은 full SPA, 우리는 min-sum 근사 — 알고리즘 자체 이식 아님 |

적용 가치: **낮음** — error-floor 하한을 대수적으로 보증하는 부호구성법이지만, EG 기반 permutation 배열은 우리 고정 QC-LDPC(z=32, base+shift) 구조와 정합되지 않고 HW·soft-read 논의가 전무해 실이식 가치는 낮다.
