### B. 알고리즘 요약

1. 대상: rate-1/2 IPP(irregular partitioned permutation) LDPC 2종, AWGN, block length 3104/8102 (global matrix M=16x32, permutation size p=97/251).
2. 문제: irregular LDPC는 coding gain은 크나 error floor가 높고, 유한정밀 양자화 시 waterfall 손실·조기 error floor가 심함.
3. 부호구성: global matrix M의 각 1을 pxp 순환-shift permutation, 0을 zero로 치환 -> H (QC 구조). degree 분포는 M에서 최적화, degree-2 all cycle 회피.
4. 핵심: H의 cycle이 M의 closed path에서 유래 -> `ACE_C = ACE_P (mod)` 관계와 `S = sum (-1)^i s_i` 유도. 모든 small-ACE closed path에 대해 `S != 0`이 되도록 cyclic shift 선택 -> stopping set 확대 -> error floor 개선 (cycle length 18까지).
5. 복호: 표준 BP(log-domain), 유일 비선형 `phi(x)=-ln(tanh(x/2))`를 LUT로 사전계산.
6. 양자화: 6-bit(sign 포함) 균일 양자화는 irregular에서 dynamic range 부족으로 overflow·high floor. 3-level variable-precision non-uniform 양자화 고안: magnitude 5-bit를 b4/b3 상위비트로 3구간 나눠 radix point를 2bit씩 우측shift(step 6, 46, 166), best step ~0.032 -> double-float 대비 <0.1dB 손실.
7. 아키텍처: [4] 유사 partly-parallel. global CNU/VNU를 p개 노드가 time-multiplex 공유 -> 연산유닛 p배 감소. EMU(edge mem)/IMU(초기메시지 mem) 사용, VNU 접근주소는 cyclic shift만큼 offset 후 wrap.
8. HW 디테일: VNU에 Mapping(5bit->9bit) + SM-to-2's + adder tree + Saturate&Quantize, 3-stage pipeline.
9. 결과: Altera Cyclone II EP2C70에서 33043/68416 LE 사용, 98MHz, 30 iter시 throughput ~50Mbps.
10. 한계: arithmetic unit(adder-tree) 미최적화로 throughput 낮음, 무선용 rate-1/2 irregular 부호이며 NAND ECC급 고rate 아님.
