### B. 알고리즘 요약

1. 목적: 기하/대수 구성(EG/PG/BIBD)은 code length·weight가 파라미터에 강하게 묶여 유연성이 낮음 — length/rate를 자유롭게 고를 수 있는 QC-LDPC 구성법 제안.
2. 문제: Tanner graph의 short girth(특히 4-cycle)가 성능을 크게 저하 → PCM이 RC-constraint(임의 두 열의 공통 '1' 수 λ=0 또는 1, girth≥6)를 만족해야 함.
3. 모델: PCM을 CSPBIBD(Circulant Symmetric PBIBD) incidence matrix들의 결합 `H=[Q1,...,Qt]`로 구성, 각 `Qi`는 base incidence vector `Bi`와 그 cyclic shift로 생성.
4. 핵심 기법 1: base block 판정 — block `B`의 원소 차분 `d_ij=b_i-b_j mod v`가 v 내에서 최대 1회 등장하면 base block(Deduction 1). 여러 base block의 difference set `D=[D1,...,Dt]` 원소가 모두 distinct면 RC-constraint 만족(Deduction 2).
5. 의미: difference set의 distinct 조건이 4-cycle 부재(RC-constraint)를 대수적으로 보장 → error-floor 억제의 구조적 근거.
6. 핵심 기법 2: y개 난수(<v)를 뽑아 mod-v 차분이 모두 distinct한지 검사하는 컴퓨터 탐색으로 base block 수집(Fig.1 flow), base block 수 상한은 Theorem 1 `T≤⌊(v-1)/(y(y-1))⌋`.
7. 구현: 결과 PCM은 `v×tv`, row weight `ty`·column weight `y`, null space가 QC-LDPC(length `tv`, rate≈`(t-1)/t`), feedback shift register로 인코딩 가능.
8. 확장(circulant decomposition): rate가 너무 높으면 row splitting으로 각 circulant을 c개 descendant로 분해 → rate를 `(t-c)/t`로 낮추고 error-floor를 하강.
9. 결과: 예2 (9360, rate≈7/8) BER=10⁻? at 4.2dB(Shannon 1.3dB), 원 부호는 이른 error-floor 발생; 예3 circulant decomposition으로 (3066, rate≈1/2)는 10⁻? 까지 error-floor 없음, Shannon gap 약 2.3dB — PG/Mackay 부호와 성능 대등.
10. 한계: 순수 구성법+Sum-Product 시뮬, HW/인코더 미설계, 원 PCM error-floor를 decomposition으로 완화하나 rate 손실 동반, gate/throughput 부재.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |

| QC-LDPC PCM 구성(base block+cyclic shift) | [ecc_top.cpp](../Prime_ECC_3.1_Claude) `Load_PCM()` | H-matrix/부호구조 로드 지점 — 특정 QC-LDPC 고정이라 새 구성법 도입은 재검증 부담 큼 |
| circulant / base matrix + shift 구조 | `ecc_data.h` `PCM_b`(base+shift) | Prime의 base matrix + circulant shift와 개념 동형, 다만 z=32 lifting과 v(소수) 기반 CSPBIBD의 정합 필요 |
| RC-constraint(4-cycle 제거)·error-floor | [decoder.cpp](../Prime_ECC_3.1_Claude) `LDPC_Decoder()` | 부호구조가 error-floor에 기여 — 디코더 로직 변경은 아님, 부호설계 레이어 |
| 인코딩(feedback shift register) | [encoder.cpp](../Prime_ECC_3.1_Claude) `LDPC_encoder_4KB()` | Prime는 dual-diagonal 고정 — 구성법이 바뀌면 인코더 PCM 재생성 필요 |

적용 가치: **낮음** — QC base+shift라는 골격은 Prime과 동형이나, 부호설계 레이어(프로파일 §4에서 이식난이도 '높음') 교체는 재검증 부담이 크고, circulant decomposition은 rate를 낮추는 방향이라 NAND 고rate 요구와 상충. error-floor를 구조적으로 보증하는 RC-constraint/difference-set 아이디어만 개념 참고.
