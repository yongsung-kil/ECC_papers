### B. 알고리즘 요약

1. LDPC-coded MIMO(M송·N수신 안테나, flat fading, 송·수신 모두 CSI 미지). 외부 LDPC(rate `R=k/n`) + 내부 space-time 인코더(interleaver·변조·pilot 삽입).
2. 문제: 기존 2단(pilot 채널추정→coherent 검출)은 데이터 심볼의 채널정보를 못 써 fast fading·저SNR에서 성능 저하.
3. 채널 `Yi=Xi·Hi+wi`, 미지 H를 통계분포만 이용하는 memoryless vector channel로 취급 → 명시적 채널추정 불필요.
4. Optimal soft MIMO detector: `p(Y|X)` (식6)로 각 coded bit의 posterior LLR `Lpos`→extrinsic `Lext` (식8-9), 복잡도 `2^{Td·M·log2|X|}`(지수).
5. Sub-optimal detector: 두 행씩 처리해 partial extrinsic 선형결합, 복잡도 `Td`에 선형.
6. Butterfly detector: FFT 유사 다단 구조로 `log2(Td)·2^{2M·log2|X|}` 수준으로 추가 감소.
7. 새 수신기 구조: soft MIMO detector + LDPC VND = super soft decoder, CND가 다른 컴포넌트 → 단일 iterative loop(BICM-ID 대비 복잡도↓).
8. EXIT 닫힌형: super decoder `IE,VND=fs(IA,VND)` (식23), CND `IE,CND=fc(IA,CND)` (식26).
9. degree profile 최적화: `max Router` (식27)를 λ/ρ 교대 선형최적화, concave 목적+선형제약 → 전역최적·임의 초기화 수렴 보장(수동 curve fitting 대비 우수).
10. 결과(2×2, T=6, Tτ=2, BPSK): optimal detector가 MMSE 대비 >1.5dB, 최적화 profile이 regular(3,6) 대비 ~0.6dB gain. BER 시뮬만, HW 없음.
