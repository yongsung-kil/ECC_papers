> 총평: DVB-S2 IRA-LDPC decoder를 0.13um에서 합성한 초기 IP core로 부분병렬 매핑·메모리충돌 회피·zigzag PN 최적 업데이트가 유용하나, 부호/병렬구조가 NAND QC-LDPC와 달라 직접 이식보다 아키텍처 참고 가치.

### B. 알고리즘 요약

1. 시스템: DVB-S2 표준 IRA-LDPC(11개 rate `R=1/4~9/10`, block `N=64800`), 목표 throughput 255Mbit/s, 30 iteration.
2. 문제: 64800 VN·반복당 ~30만 메시지 처리/재정렬로 fully-parallel은 라우팅 혼잡, 부분병렬 아키텍처 필수.
3. 부호구조: parity check가 random part(IN↔CN, permutation Π) + fixed zigzag(PN degree-2 accumulator)로 분리, IRA라 인코딩 선형.
4. 복호: message passing(BP), VN는 `λk=λch+Σλl`(식4), CN는 `tanh(λk/2)=∏tanh(λl/2)`(식5, SPA tanh) - fixed-point 6bit 양자화 손실 ≤0.1dB.
5. degree-2 PN 최적화: zigzag 연결에서 CN을 순차(forward) 처리해 갱신 메시지를 다음 CN에 직접 전달, backward는 병렬 업데이트.
6. 효과: 이 스케줄로 통신성능 개선 → 동일 성능에 40 대신 30 iteration(10 iter 절감), 저장 메시지도 2개→1개.
7. HW 매핑: `P=360` 기능유닛(식2에서 360 edge가 단일 x로 결정), 각 유닛이 `q*(k-2)` edge 처리(식6), 단순 shuffling network로 Tanner 연결성 제공.
8. 최적화: single-port SRAM read/write 충돌 회피 위해 IN 메모리를 4 RAM 분할 + simulated annealing으로 addressing 최적화(버퍼 1개로 전 rate 커버).
9. 결과(0.13um CMOS 합성): 최대 270MHz, 6bit 양자화 기준 total area 22.74mm2(RAM 9.12/logic 10.8/shuffling 0.55), Tanner 연결성 저장은 0.075mm2에 불과, 255Mbit/s 달성.
10. 한계: DVB-S2 IRA 부호 전용(N=64800 대형), SPA tanh 기반 6bit LLR, 정정능력 개선이 아닌 HW 실현/스케줄링 최적화.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP 복호 루프(VN/CN 업데이트) | decoder.cpp LDPC_Decoder(), C2V_Cal() | 이터레이션 루프는 대응하나 논문은 SPA tanh, 우리는 min-sum |
| degree-2 PN forward/backward 스케줄 | decoder.cpp LDPC_Decoder() (스케줄링) | Dual-Update(even/odd 교대)와 유사 개념, iteration 절감 아이디어 참고 |
| shuffling network / P=360 부분병렬 매핑 | 디코더 HW 모델(z=32, GT) / GT.cpp Make_GT_HW() | 부분병렬·라우팅 구조 참고하나 z=32 QC 구조와 병렬단위 상이 |
| 6bit LLR 양자화 | ecc_data.h PARAM_LLR, channel.cpp | 양자화 손실 데이터는 참고 가능, 우리는 2~3bit soft |
| RAM 충돌 회피(simulated annealing 매핑) | ecc_top.cpp Load_PCM() / 디코더 메모리 모델 | 메모리 access 스케줄 최적화 개념 참고 |
```

적용 가치: **중간** - 부분병렬 decoder 아키텍처·메모리충돌 회피·degree-2 zigzag forward update(iteration 절감)는 참고 가치가 있으나, DVB-S2 IRA(N=64800, SPA tanh, 4bit+ LLR) 특화라 Prime ECC의 고rate QC-LDPC(z=32, min-sum, 2~3bit)와 부호·병렬구조가 달라 직접 이식은 제한적.
