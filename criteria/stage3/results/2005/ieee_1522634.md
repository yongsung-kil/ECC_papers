### B. 알고리즘 요약

1. 대상은 코드가 아니라 cycle 있는 그래프의 pairwise MRF에서 `argmax p(x;θ)` MAP 구성을 찾는 일반 문제.
2. 문제: cycle 그래프 MAP은 NP-hard, 표준 max-product(min-sum)는 최적 보장이 없고 틀린 답을 낼 수 있음.
3. 아이디어: 원 분포 θ를 여러 spanning tree 분포들의 볼록결합 `θ=Σμ(T)θ(T)`로 분해, Jensen으로 MAP값 상한을 얻음.
4. 핵심정리: 이 상한은 모든 tree가 공통 최적 구성을 가질 때(tree agreement)에만 tight, 그때 공통 구성이 원 문제 MAP.
5. 상한 최소화의 Lagrangian dual이 `LOCAL(G)` (node·edge marginal 정합) 위의 LP 완화임을 증명(Thm.1) - Bethe 변분 제약과 동일.
6. tree-reweighted max-product(Alg.2): 메시지를 edge 출현확률 `ρ_e`로 재가중, `ρ_e=1`이면 표준 max-product로 환원.
7. OS(optimum specification) 기준 만족 시 고정점이 원 MAP를 특정하고 LP dual 최적해가 됨(Thm.2).
8. Corollary: tree에서는 `ρ_e≡1`이라 표준 max-product(Viterbi)가 곧 정확 LP의 dual을 푸는 것.
9. 결과: Ising 격자(n=400)에서 edge/tree-based 업데이트 비교, attractive 결합은 항상 MAP 도달, mixed는 80%+ 변수 정확.
10. 한계: 부호/디코더 특화 아님, HW·양자화 없음, 강한 결합서 tree-based는 진동(수렴 미보장), coding 응용·pseudocodeword 해석은 인용문헌으로 위임.
