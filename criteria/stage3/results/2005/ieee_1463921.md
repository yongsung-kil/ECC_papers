### B. 알고리즘 요약

1. 대상 시스템은 트랙이 겹치는 자기기록(HDD) 매체로, 사용자 데이터를 LDPC로 부호화해 기록한다.
2. 풀려는 문제: 트랙 오버랩으로 인한 inter-track interference(ITI)와 downtrack ISI가 결합된 2D ISI 채널에서 신뢰성 있는 복호와 사용자 비트밀도 확보.
3. 채널 모델: 재생신호 `R = h ** X + W`, `h`는 2D PSF(마이크로마그네틱 시뮬로 추정), `W`는 zero-mean AWGN(분산 σ²).
4. PSF 추정: 3개 인접 트랙에 알려진 패턴을 기록 후 중앙 트랙 재생신호를 선형 채널 가정 하에 분석해 PSF를 산출.
5. 핵심 기법: 채널 IS I 그래프와 error-control code 그래프의 joint graph 위에서 message-passing을 수행하는 full graph 알고리즘([2])으로 equalization 후 부호화를 통합 복호.
6. 오버랩 파라미터 α는 read head가 보는 중앙 트랙 비율(예: 10% 오버랩 → α=0.8), ITI는 10% 오버랩 시 ~11%, 20% 오버랩 시 ~33%.
7. SNR 정의: `SNR = 10 log10(...)`에 LDPC rate `c`를 반영, 사용자 비트밀도는 rate `c`에 비례하고 α에 반비례.
8. 결과: 20% 오버랩은 무오버랩 대비 약 2.5 dB SNR 손실, 10% 오버랩은 약 0.5 dB 손실; 저rate LDPC(0.5) 사용 시 밀도를 높게 유지하면서 성능 이득이 밀도 손실을 상회.
9. bit aspect ratio가 2까지 낮아도 신뢰 복호 가능하며 1로 수렴 시에도 잘 동작할 것으로 기대.
10. 한계: 1페이지 digest로 HW 미설계·시뮬만, 부호 설계/디코더 구조 세부 없음, HDD 2D ISI 전용.
