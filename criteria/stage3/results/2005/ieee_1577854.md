### B. 알고리즘 요약

1. 대상은 `Mb×Nb` base matrix의 각 1을 `L×L` 순환치환(identity를 `Pi,j`만큼 shift) 행렬로 확장한 binary QC-LDPC(rate 1/2 irregular, 5/8·7/8 regular, 부호길이 ~9k).
2. 문제: base matrix 확장 시 short cycle이 복제되어 girth가 제한되고 error floor 발생, 기존 girth 개선법(bit-filling/PEG)은 HW 매핑을 고려하지 않음.
3. 모델: Tanner graph girth가 낮으면 BP의 이웃 독립 가정이 깨져 수렴 저하·오류; VLSI 지향 구조(base+circulant)를 유지하며 girth를 올려야 함.
4. 핵심 기법: cycle elimination(CE) — base matrix의 length-`2g` cycle을 따라 shift값 drop 합 `ΣΔP`가 `kL`이 되지 않도록 `Pi,j`를 설정.
5. Lemma: base matrix의 `2g` cycle은 `ΣV(ΔP)`(또는 `ΣH`) `= kL`일 때만 확장행렬에 `L`개의 `2g` cycle을 생성 → 이 조건을 회피하면 확장 후 해당 cycle 제거.
6. CE 절차: 열을 좌→우로 훑으며 current column 우측 관련 cycle을 탐색, 각 1의 `P`를 최소값부터 증가시켜 제약 만족(step 1~4).
7. 디코더: partial-parallel 아키텍처(`Nb` VNU, `Mb` CNU를 time-multiplex), CNU는 4CU→8/12CU→24/36CU 다층 트리로 병렬 check 연산, VNU/global/reverse router로 멀티레이트 구성.
8. 양자화: LUT 대신 Min-Sum with Correction(MSC, `Ψ(x)=log((1+e^-|x|)/(1-e^-|x|))`)로 high-rate/irregular의 dynamic range-precision 트레이드오프 손실 회피.
9. 결과: CE로 평균 girth 8→10(1/2), 8.25→10(5/8), 5.9→8(7/8) 개선; irregular 1/2가 regular 1/2 대비 0.5dB↑, SNR 1.8dB에서 BLER<10^-7(error floor 소멸), coding gain 손실 0.1dB 이내. Xilinx XC2V8000 FPGA, 100MHz, 66/30Mbps.
10. 한계: CE는 short cycle만 제거하나 error floor 주원인은 stopping set이라 논문이 인정, throughput(≤66Mbps)은 무선/OFDM급이며 NAND급엔 부족.
