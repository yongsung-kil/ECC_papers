### B. 알고리즘 요약

1. 대상: `(λ,ρ)` regular bipartite graph로 정의되는 binary QC-LDPC, 예제 9종 대부분 (3,6)-regular rate-1/2, n=504~32768.
2. 문제: LDPC error-floor의 두 원인(작은 dmin·저weight near-codeword) 완화하면서 대수구조+의사랜덤성을 겸비한 실용 구성 필요.
3. 핵심 아이디어: edge permutation `f(·)`를 정수환 `Z_N` 위 2차 permutation polynomial(QPP) `f(x)=f1·x+f2·x²(mod N)`로 정의 — 그래프가 유일 결정됨.
4. permutation 조건: Prop.1이 QPP가 permutation이 되는 `f1,f2`의 정수론적 필요충분조건 제시(소인수분해 기반).
5. 그래프 isomorphism(Prop.2/3)·automorphism(Thm.1) 규명으로 `f1` 탐색범위를 `1~2f2·α`(`α=lcm(λ,ρ)`)로 축소, girth 계산도 equivalence class 대표만 검사.
6. QPP의 비선형성이 그래프에 automorphism을 유발 → 코드가 quasi-cyclic(shifting constraint `β`)이 되어 circulant 부분행렬로 PCM 표현 가능(예: 84×84 circulant).
7. dmin 상계: type II(circulant 행/열 weight 0,1,2) weight matrix `A`로 확장한 Thm.3이 기존 Fossorier(≤24)/MacKay보다 큰 상계 제공(code II에서 dmin≤62), NNCS로 실제 저weight codeword 탐색.
8. dmin이 블록길이에 따라 (선형) 증가한다는 추측을 여러 길이의 상계로 뒷받침, turbo QPP interleaver와의 통일적 연결 규명.
9. 결과: (3,6) (1008,504) code II가 80/200 BP iteration에서 girth-8 PEG 코드를 고SNR에서 상회, code IV/VI는 FER 3e-7까지 error-floor 없음; code II만 (12,2) near-codeword로 ~5e-8 floor 예상.
10. 한계: rate 1/2·regular만 시연(irregular은 future work), HW/양자화 미설계, contention-free 병렬성은 성질 주장 수준.
