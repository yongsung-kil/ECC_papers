> 총평: 전력선 통신 임펄스 노이즈(Class A)에 맞춰 sum-product의 LLR 입력을 재정의한 채널특화 논문, 알고리즘은 표준 SP 그대로라 NAND binary QC-LDPC 이식 가치 낮음.

### B. 알고리즘 요약

1. 시스템: 전력선 통신(PLC) 홈네트워킹, regular LDPC (column weight 3, 길이 1200, rate≈1/2), BPSK, log-domain sum-product 복호.
2. 문제: 전력선은 가전제품 유래 임펄스 노이즈가 지배적이라 AWGN 가정으로 설계된 기존 sum-product가 큰 성능 열화(BER 10^-5에서 약 13 dB 열화).
3. 채널 모델: Middleton Class A 노이즈 = 배경 Gaussian + Poisson 분포 임펄스원의 합. PDF `pA(z)`는 파라미터 impulsive index `A`, GIR `Γ=σG²/σI²`, 총전력 `σ²`로 규정.
4. 핵심 기법: sum-product 입력 LLR을 AWGN용 `λG=2yn/σ²` 대신 Class A PDF 기반 `λA(yn)=ln[pA(yn-1)/pA(yn+1)]`로 교체 (식 12).
5. 식 의미: `λA`는 무한급수라 `L=max{2,⌈10A⌉}` 항까지 근사. 수신값이 작으면 LLR이 신호에 비례하나 큰 값이면 임펄스 가능성이 높아 LLR을 억제(clip 유사 효과).
6. 확장: 파라미터 `A`, `Γ`, `σ²`는 수신 envelope의 2/4/6차 모멘트에서 추정.
7. 구현: 복호 알고리즘(Step1~4 log-domain SP, `f(x)=ln((e^x+1)/(e^x-1))`)은 표준 그대로, LLR 입력만 채널특화.
8. 학습/최적화: 별도 학습 없음, 채널 파라미터 추정 기반 해석적 LLR.
9. 결과: 제안 SP는 AWAN(A=0.1,Γ=0.1)에서 max iter 20일 때 기존 SP 대비 BER 10^-5에서 약 11 dB 이득; 평균 iteration도 급감(Eb/N0=5 dB에서 기존은 20 iter로 복호 실패, 제안은 평균 약 7 iter로 성공).
10. 한계: 순수 시뮬, HW 미설계, 전력선 임펄스 채널 전용, GIR=1.0 근처에서는 개선 효과 미미.
