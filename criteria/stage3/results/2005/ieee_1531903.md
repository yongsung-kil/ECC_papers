> 총평: 유한체·affine mapping으로 girth≥6과 큰 최소거리/stopping set을 보장해 저 error-floor 고rate binary QC-LDPC(소수체 사용 시)를 대수적으로 구성하는 방법으로, Prime ECC의 부호설계 참조가치는 있으나 고정 H-matrix 교체 부담으로 이식성 중.

### B. 알고리즘 요약

1. 목표: 유한체 `GF(q)`의 가법군과 affine permutation으로 girth≥6·RC-constraint를 만족하는 구조적 binary (γ,ρ)-regular LDPC 구성(소수체 사용 시 QC-LDPC), 채널은 AWGN/BEC, 복호는 반복 SPA.
2. 문제: iterative 복호 성능은 Tanner graph의 short cycle(특히 length-4)과 최소거리/stopping set 크기에 좌우, length-4 cycle과 작은 최소거리는 높은 error-floor 유발.
3. 핵심 도구: `GF(q)` 원소 `α^i`를 q-tuple location vector `z(α^i)`(단일 1-성분)로 표현 → q×q 순열행렬, affine mapping `x→a·x+b`.
4. 기법 1단: 원소 `w(k)` 벡터들(Theorem 1: 두 행은 모든 위치에서 다르거나 최대 1위치만 동일)로 q×q 행렬 W 구성, 각 항을 location vector로 치환해 q개 순열행렬의 q×q array `H`(q²×q², 행·열 weight q) 생성.
5. 핵심 성질: Theorem 1로 `H`가 RC-constraint 충족 → Tanner graph girth≥6, 최소거리≥γ+1(짝수 γ면 γ+2), minimal stopping set 크기≥γ+1 보장.
6. 기법 2단(부분행렬): `H`의 γ×ρ 부분array `H(γ,ρ)`로 다양한 length·rate·최소거리의 부호족 생성, rate≥(ρ-γ)/ρ, column weight γ를 키우면 최소거리↑→error-floor↓(대신 복호 복잡도↑).
7. QC 확장: 소수체 `GF(p)` 사용 시 각 순열행렬이 circulant(location vector가 modulo-p로 한 칸 순환)가 되어 array 전체가 circulant permutation array → 부분array가 QC-LDPC, shift-register 인코딩(패리티비트 수에 선형).
8. 실험 부호: (1024,781)/(1024,821)(GF(2^5)), (2048,1723)/(5256,4823)/(16448,15423)(GF(2^6) 계열), 소수체 예 GF(73)/(GF(2^6)) 등, max iteration 100, BPSK.
9. 결과: AWGN에서 BER 10^-6에 Shannon limit 대비 약 1~1.5 dB, error-floor는 10^-12~10^-15 이하로 매우 낮음; BEC에서 threshold 대비 0.003~0.014, 10^-10까지 error-floor 없음. (16448,15423)은 IEEE 802.3 10G BASE-T 표준 RS-based LDPC와 동일.
10. 한계: HW 미설계(gate/throughput 없음), 고 column weight는 복호 복잡도 증가, error-floor는 [15] 방법 추정치(장프레임 실측 아님).
