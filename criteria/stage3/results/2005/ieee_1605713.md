### B. 알고리즘 요약

1. binary protograph LDPC 출력을 8PSK/16QAM 등 고차 변조에 매핑(VDMM 직접 매핑 또는 BICM 채널 인터리버), AWGN 채널 대상.
2. 문제: 우수한 iterative decoding threshold와 낮은 error floor를 동시에 만족하는 구조적 LDPC가 필요.
3. 핵심: protograph를 copy-and-permute(lift factor `N`)로 확장, degree-2 VN 비율이 threshold vs 최소거리 성장의 트레이드오프를 좌우.
4. 최소거리 선형증가 보장을 위해 asymptotic ensemble weight enumerator `r(δ)`를 계산, 첫 non-trivial zero crossing `δmin`으로 정규화 최소거리를 판정.
5. `dmin = δmin × n`; `δmin>0`이면 최소거리가 블록길이 `n`에 선형 증가 → low error floor의 이론적 근거.
6. precoding: accumulator(degree-1 VN)를 삽입해 rate를 유지한 채 threshold를 낮추면서 동시에 `δmin` 증가(예: (3,6) `δmin` 0.023→0.033).
7. ARJA(Accumulate-Repeat-Jagged-Accumulate): ARA의 degree-2 VN 일부를 degree-3으로 교체해 최소거리 성장을 확보.
8. rate-compatible ARJA family(rate 1/2~4/5)를 구성, VDMM은 bit-plane 상호정보 `I(xb;y)`에 비례해 VN degree를 배분(waterfilling).
9. 결과: k=4096 rate-1/2 QPSK에서 FER 10^-8 / BER 10^-10 @ Eb/No 1.50 dB, FPGA 시뮬로 극저 error floor 확인.
10. 한계: gate count/throughput 미제시, coded modulation·AWGN 가정, 표준 BP 디코더로 복호 알고리즘 자체 기여는 없음.

| weight enumerator 기반 최소거리 설계 | 대응 없음 | 설계 이론, 대응 모듈 없음 |
| coded modulation / demapper LLR | 대응 없음 | 8PSK/16QAM 변조 LLR은 NAND read 비용과 무관 |
| protograph-slice 병렬 디코더 아키텍처 | 디코더 HW 모델(z=32) | 병렬 개념은 유사하나 QC circulant 병렬과 구조 상이 |
