### B. 알고리즘 요약

1. 대상: 유한길이 systematic IRA 부호(= accumulator 구조 irregular LDPC), AWGN, rate 1/4·1/2·3/4, length ~1000, sum-product 복호(최대 100 iter).
2. 문제: 기존 IRA는 accumulator가 `M`개의 weight-2 열을 강제 → 저rate/저error-floor용 최적 차수분포(weight-2 열 < M 요구)를 못 맞춤.
3. 모델: IRA를 `H=[H1,H2]`로 표현. `H2`는 accumulator(`p_i=p_{i-1}⊕r_i`)로 weight-2 열 밴드, `H1`은 인터리버가 결정하는 반복부호 열.
4. 기법1(차수분포): density/differential evolution로 임계값 최적화, 단 `λ2 ≥ 2(1-R)/R·Σ...` 제약(식4)으로 accumulator에 충분한 weight-2 열 확보.
5. 의미: 안정조건(식5) `λ'(0)ρ'(1)<e^{1/2σ²}`이 `λ2` 상한을 주므로 저rate에서 제약 충돌 → 밀도(임계값) 감소 불가피.
6. 기법2(w3IRA): accumulator를 `1/(1+D+D^{g+1})`로 변경, `H2` 각 열이 `i,i+1,i+1+g` 행에 non-zero → weight-3 열 도입, `g`가 남는 weight-2 열 수·cycle(size `2(g+1)`) 제어.
7. 트릭: `g=623` 등으로 length-1000 rate-1/4에서 LDPC용 차수분포를 accumulator 유지한 채 달성.
8. 인터리버: Steiner 2-design 접합행렬 `N`의 행/열 치환으로 RA 형식 매핑(Thm 2), Construction 1로 girth 6(4-cycle-free) 보장.
9. 결과: 조합적(Steiner) IRA/w3IRA가 random 인터리버 대비 error floor 개선, LDPC ensemble 평균에 근접(BER 곡선, Figs.2~4).
10. 한계: HW 미설계, AWGN 시뮬만, gate/throughput 없음, 부호족이 accumulator 기반이라 dual-diagonal QC-LDPC와 상이.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| IRA/w3IRA H-matrix 구성·인터리버 | ecc_top.cpp Load_PCM() | Prime ECC는 고정 QC-LDPC 로드, IRA 구성 직접 대응 없음 |
| accumulator 기반 간단 인코딩 | encoder.cpp LDPC_encoder_4KB() | Prime ECC는 dual-diagonal 인코딩, accumulator와 구조 상이 |
| 차수분포/error-floor 최적화 | ecc_top.cpp Load_PCM() (H-matrix) | 부호설계 레이어, 고rate QC-LDPC 재검증 부담 큼 |
| sum-product 복호 | decoder.cpp LDPC_Decoder() | Prime ECC는 min-sum 보유, 복호 신규성 없음 |
```

적용 가치: 낮음 — 부호족(IRA/accumulator)·rate·인코딩 방식이 Prime ECC의 고정 고rate QC-LDPC/dual-diagonal과 달라 조합적 인터리버 아이디어 외 직접 이식할 것이 거의 없음.
