### B. 알고리즘 요약

1. 대상: BPSK/AWGN용 protograph 기반 LDPC 부호 계열(RA/IRA/ARA→ARJA/AR4JA), rate 1/4~8/9, 입력블록 `k=1024,4096`.
2. 문제: 정규 LDPC는 error floor 낮으나 threshold 높고, ARA류는 threshold 좋으나 앙상블 min-distance가 `n`에 비례 증가하지 않아 error floor 취약.
3. 모델: 앙상블 평균 weight enumerator로 정규화 로그 점근 분포 `r(δ)=ln(A_d)/n` (`δ=d/n`)의 첫 zero crossing `δmin>0` 존재 여부로 선형 min-distance 성장 판정.
4. 핵심기법1(precoding): constraint node와 고차 variable node 사이에 degree-1 variable node(accumulator)를 삽입, rate 불변으로 iterative threshold 저하.
5. 의미: 예 (3,6) 정규 LDPC를 25/50% precoding하면 threshold가 최대 0.23dB 개선(DE 예측과 FPGA 실측 일치).
6. 핵심기법2(ARJA): ARA의 degree-2 variable node 일부(50%)를 degree-3로 교체해 "jagged" accumulator 구성 → min-distance가 `n`에 선형 증가(`δmin=0.015`).
7. 확장(AR4JA/AR3JA): repetition-4 추가 및 puncturing으로 rate-compatible 고rate 계열 생성, protograph를 ACE 알고리즘으로 lifting해 각 edge를 circulant로 배치.
8. 저rate 확장: ARJA protograph에 LDGM(단일 parity-check) 병렬 경로를 hybrid concatenation으로 붙여 rate 1/3, 1/4 부호 생성(threshold가 capacity에 근접).
9. 결과: k=1024 AR4JA는 짧은 블록에도 우수 threshold, error floor 미관측; (3,6) precoding gain 0.23dB, capacity와의 차이 rate 7/8에서 ~0.36dB.
10. 한계: 부호설계·threshold/error-floor 초점, HW gate 미설계(기존 JPL FPGA 디코더로 시뮬), 저~중rate·AWGN 가정.
