### B. 알고리즘 요약

1. 대상: 임의의 binary linear block code `C(N,K)` (실험은 BCH(127,71), eBCH(128,64), RS(31,25)), BPSK/AWGN.
2. 문제: BCH/RS는 PCM이 고밀도·다수 cycle이라 BP가 error propagation으로 수렴 실패 → ML에 못 미침.
3. 공통 1단계: 채널 LLR `|L|`을 오름차순 정렬, least-reliable(LR) bit부터 해당 컬럼을 weight-1(단위벡터)로 만들도록 `H`에 row operation → 오류 bit를 그래프 leaf로 이동 `H_j = Φ(H,|L_j|)`.
4. ABP: 변형된 `H_j` 위에서 SPA 1회 실행해 extrinsic `L_ext` 산출, `L_{j+1} = L_j + α L_ext` (damping `α`, gradient descent 형태). soft-syndrome 비용함수 `J(H,T)=−ΣS_i` 최소화로 해석.
5. BGA(제안): 매 iteration마다 원본 `H`에서 다시 출발해 `L_{j-1}` 기준 변형 후 BP 수행. 그래프가 매번 바뀌어 CN-VN 연결이 없으면 채널 LLR `L_p^0`을 대신 전달.
6. BGA 핵심: 과거 연결이 끊긴 메시지 `VCg`를 저장했다가 재연결 시 사용 → 정보 손실 방지, 수렴 개선.
7. 부작용: 저장된 메시지가 "pseudo-loop"를 만들어 positive feedback·error propagation 유발 → 확률적 bit-flip류로 억제 필요(미구현).
8. 결과(WER): BGA가 ABP 대비 BCH(127)/eBCH(128)에서 `10^-3~10^-4`에서 1dB 이상 gain, 단 ML에는 미달.
9. 결과(iteration): BGA의 평균 iteration은 ABP보다 많음(특히 저SNR), 최대 300 iteration 설정.
10. 한계: HW 미설계·시뮬만, 고SNR에서 error-floor 유사 특성, 긴 블록에서 gain 감소, 매 iteration PCM 재변형(Gaussian elimination) 비용.
