### B. 알고리즘 요약

1. 대상은 CSS 양자 QLDPC 코드([[n,k,d]] stabilizer code, 예: bivariate bicycle, surface, hypergraph product, generalized bicycle), depolarizing channel에서 X/Z 오류를 독립적으로 신드롬 기반 디코딩.
2. 문제: quantum degeneracy(여러 오류 패턴이 동일 신드롬을 유발) 때문에 표준 BP(scaled min-sum)가 수렴 실패하는 경우가 많음.
3. 채널 모델은 물리적 오류율 `p`의 depolarizing channel, 입력 LLR `p_in = log((1-p)/p)`.
4. 핵심 기법(RB): 첫 root BP(`T_root` 반복) 실패 시, 출력 LLR을 오름차순 정렬하여 가장 낮은(불확실한) LLR 큐비트부터 순서대로 에러를 강제 인가(`e_bar`)하고 신드롬·입력LLR을 갱신 후 branch BP(`T_branch` 반복)를 재실행하는 branch-and-bound 방식.
5. 핵심 식은 조기종료 조건: distance-specific(`w(ê) ≤ t`이면 최소가중치 해 확정)과 defect-specific(`w(s)/ξ > t`이면 최소가중치 해 자동 인정) — 왜 필요한가: OSD 없이도 최소가중치 보장 및 불필요한 branch 탐색 생략.
6. 확장: branch 개수 `η`(성능-지연 trade-off 파라미터)를 코드별로 exhaustive search로 튜닝해 코드 거리 `t`까지의 오류 패턴을 항상 정정하도록 보장(결정론적, RelayBP의 확률적 방식과 대비).
7. 구현 디테일: 각 branch는 최대 `t-1`번의 BP 인스턴스로 구성되며, η개 branch는 서로 독립적이라 병렬 HW 매핑 가능(worst-case 복잡도 `O((tT_branch+T_root)n)`로 병렬화 시 감소).
8. 별도 학습/최적화 절차 없음 — η는 코드별 exhaustive search로 사전 결정.
9. 결과: bivariate bicycle([[144,12,12]] gross code), surface([[85,1,7]]), hypergraph product([[145,5,6]]), generalized bicycle([[48,6,8]]) 코드에서 BP/BPGD/RelayBP 대비 codeword error rate 개선, 대부분 코드에서 RelayBP보다 평균 BP iteration 수 감소(Table I).
10. 한계: depolarizing channel 시뮬레이션만 수행(회로 레벨 노이즈 미반영), HW 미설계, quantum stabilizer 코드 전용이라 classical binary LDPC와 신드롬/오류모델 구조가 다름.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP(scaled min-sum) 반복 디코딩 전체 | `decoder.cpp` `LDPC_Decoder()` | 알고리즘 골격(check-to-variable min-sum)은 유사하나 quantum 신드롬 기반 오류 추정이라 직접 대응 아님 |
| restart 시 최저-LLR 큐비트 강제 에러 인가 및 branch 재실행 | 대응 없음 | Prime ECC는 stuck 상황에서 branch-and-bound식 재시작 로직이 없음(현재는 Partial/Full CRC 조기종료만 존재) |
| distance/defect-specific 조기종료 | `partial_CRC.cpp` `partial_crc_HW_T4()` | 조기종료라는 목적은 유사하나, 판정 기준(코드 거리 기반 weight bound)이 CRC와 무관해 개념적 유사성만 있음 |
| non-binary/quantum stabilizer 부호구조 | 대응 없음 | Prime ECC는 binary QC-LDPC 전용, quantum CSS 부호 개념 자체가 대응 없음 |

> 적용 가치: 낮음 — quantum LDPC 전용 디코더로 부호·채널 모델이 Prime ECC(binary classical QC-LDPC)와 근본적으로 다르며, branch-and-bound 재시작 아이디어도 신드롬 기반 quantum 오류정정에 특화되어 있어 직접 이식보다는 개념적 참고 수준.
