> 총평: (14,8) extended Hamming code와 column splitting으로 cycle-free에 가까운 LDPC 구성법을 제안한 이론/시뮬 수준 논문으로, HW·디코더 변경이 없어 Prime ECC 이식 가치는 낮음.

### B. 알고리즘 요약

1. 대상은 AWGN 채널(BPSK 변조)에서 SPA(sum-product algorithm) 복호되는 이진 LDPC 부호이며, base matrix(PEG 또는 Gallager)와 (14,8) extended Hamming code(EHC)를 결합한 hybrid 구성법을 제안.
2. 문제의식: 기존 LDPC parity-check matrix의 Tanner graph에 짧은 cycle(특히 4-cycle)이 존재하면 반복복호 성능이 저하됨.
3. 핵심 가정: (7,4) Hamming code(dual code generator `1+u^2+u^3`)의 circulant에 column splitting을 적용하면 cycle이 거의 없는 (14,8) EHC를 얻을 수 있다고 가정.
4. 핵심 기법: base matrix `H`(m×n, column weight `γ`, row weight `ρ`)의 각 1-성분을 (14,8) EHC의 대응 행(row)으로, 0-성분을 all-zero tuple로 치환해 `mn×n` 크기의 최종 parity-check 행렬을 구성.
5. Rate 계산식 `R=(n-m')/(mn)` (m'은 가우스 소거 후 rank)로 hybrid code의 code rate를 도출 — 표 자체가 손상되어 구체적 수치는 확인 불가.
6. 두 종류의 base matrix(PEG 기반, Gallager 기반)로 각각 hybrid code를 구성해 성능을 비교(PEG-H, Gallager-H).
7. 부수 트릭: (14,8) EHC 자체는 cycle-free이며, base matrix의 column weight를 EHC의 row 수와 일치시켜야 하는 제약(구성 조건)이 있음.
8. 별도 학습/최적화 절차 없음 — PEG 알고리즘(edge-growth, girth 최대화 휴리스틱)은 base matrix 생성에 재사용(신규 기여 아님).
9. 결과: PEG 기반 hybrid code가 Gallager 기반보다 SNR 임계값(약 2dB) 이상에서 BER/FER 모두 우수하며, Shannon limit 대비 약 1dB 이내로 접근. 두 코드의 평균 반복 횟수는 유사(최대 iteration 50).
10. 한계: 순수 시뮬레이션 결과만 제시(HW 구현 없음), 디코더 알고리즘 자체는 표준 SPA 그대로이며 새로운 정정능력 개선은 부호구조(cycle 감소)에서만 기인.

| PEG/Gallager base matrix 생성 | 대응 없음 | Prime ECC는 이미 특정 QC-LDPC H-matrix를 고정 사용, base matrix 재설계는 범위 밖 |
| SPA(sum-product) 복호 | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum 기반이라 full SPA 자체는 미사용, 개념적 대응만 |

> 적용 가치: 낮음 — 특정 대수적 구성(Hamming code 기반 column splitting)으로 QC-LDPC 구조와 무관하며, 디코더/HW 개선이 없는 순수 code-design 논문.
