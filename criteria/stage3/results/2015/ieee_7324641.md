### B. 알고리즘 요약 (10줄 내외)

1. 채널 모델: Bernoulli-Gaussian(BG) 임펄스 노이즈. `nk = wk + bk·gk` (wk: 배경 가우시안, gk: 임펄스 성분, bk: 발생확률 p의 베르누이 변수, IGR Γ = 임펄스/가우시안 전력비).
2. 문제: 기존 clipping 기반 디코더(MEVA 등)는 노이즈 통계 근사 가정이 고SNR에서 깨져 BER 곡선에 error floor 발생.
3. 대상 시스템: (2,1,6) convolutional code + Viterbi 유사 디코딩(LDPC 아님).
4. 핵심 기법: clipping threshold `t`를 노이즈 통계 추정 없이 exact PEP 최소화 문제로 공식화. 결정 지표는 `m(t,r,x)`, 비트 metric difference `Δ(t,ni)`.
5. 핵심 식: PEP는 moment generating function `Φ(s,t)`의 Chernoff bound `min_{s,t} Φ(s,t)`로 근사, `∂Φ/∂t=0`, `∂Φ/∂s=0` 연립.
6. 도출 결과: 최적 clipping threshold는 `t = 2√E`로 수렴(E: 신호 에너지)함을 수치적으로 확인.
7. 부수 트릭: 기존 MEVA는 Chernoff bound 그래프에서 파라미터를 눈대중으로 선택했으나, 본 논문은 3D plot으로 global minimum 위치를 직접 검증.
8. 최적화 절차: closed-form 미분 조건 유도 후 수치적 3D 플롯으로 global minimum 확인(학습/DE 없음).
9. 결과: Γ∈{15,100,400}, p∈{0.001,0.01,0.1} 범위에서 OMCD는 MLD 성능에 근접, MEVA는 SNR 15~25dB에서 error floor(10⁻¹⁴~10⁻¹⁸) 발생.
10. 한계: 이론/시뮬레이션 수준(HW 설계 없음), convolutional code 기준 검증, LDPC나 NAND 채널에 대한 직접 검증 없음.
