### B. 알고리즘 요약

1. 시스템: 가시광통신(VLC) 채널(백색 LED, OOK 변조, AWGN), QC-LDPC rate=0.8, length=3075(615×3075 PCM).
2. 문제: 기존 VLC는 RS 부호(rate 0.8)를 사용하나 신뢰성 개선을 위해 LDPC 도입 필요, 또한 LLR-BP는 HW 구현 부담이 큼.
3. 채널 모델: VLC 임펄스 응답 `h(t)`(직접 성분 + 반사 성분)와 AWGN 결합, 신호 `Y(t)=RX(t)⊗h(t)+N(t)`.
4. 부호 구성: multiplicative group 기반 인덱스 행렬 `P`(원소 `P_i,j=a^j b^i`)로 circulant 서브행렬 생성, `H=[H1 H2]`(H1은 multiplicative 구조, H2는 dual-diagonal 구조).
5. Girth-4 검사: `O = H^T H - [h(1)...h(N)]^T[h(1)...h(N)]`의 비대각 원소가 모두 0/1이면 girth-4 없음을 보장.
6. Improved MS: check node 업데이트에 곱셈인자 `μ`(0~1) 적용, `L_mn = μ·sign(·)·min(·)` 형태로 MS의 magnitude 과대추정을 완화.
7. 추가 근사: variable node 처리 시 `min` 함수를 `Φ(x)` 함수로 대체하여 나눗셈 연산 감소(LLR-BP 대비).
8. 최적화 절차: SNR별 시뮬레이션으로 μ 스윕(0.5~0.8) 후 μ=0.65가 최적임을 실험적으로 결정.
9. 결과: iteration=10, μ=0.65일 때 Improved MS는 SNR>4dB에서 LLR-BP보다 우수, RS 대비 coding gain 최대 2dB. 곱셈 연산 수는 LLR-BP 대비 약 1/10(1.53×10^5 vs 1.47×10^6).
10. 한계: MATLAB 시뮬만 수행, HW 구현/합성 없음, VLC/AWGN 채널 특화로 NAND 채널 모델과 무관.
