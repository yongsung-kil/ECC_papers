> 총평: ADMM 기반 LP decoding의 핵심 서브루틴(parity polytope/probability simplex projection)을 정렬 네트워크로 HW화한 기초 연구로, min-sum 기반 Prime ECC와는 디코딩 패러다임 자체가 다름.

### B. 알고리즘 요약

1. LDPC 코드의 LP(linear programming) decoding, 특히 ADMM 기반 분해 기법에서 반복적으로 필요한 두 유클리드 투영(parity polytope `PP_d`, probability simplex `S_d`)을 다룸(코드 rate/length 특정 없음, 범용 알고리즘 논문).
2. 기존 parity polytope projection 알고리즘들(정렬 기반, cut search 기반, partial sort 기반)은 반복적 구조/입력 의존적 지연으로 HW 부적합.
3. 채널 모델 없음 — 순수 최적화(투영) 알고리즘 논문이며 오차/노이즈는 고정소수점 양자화 오차로만 평가.
4. 제안 Algorithm 1: unit cube 투영 `v̂=Π_[0,1]^d(v)` 후 임계값 `1/2` 기준 이진 벡터 `f` 생성, cut search 스타일 테스트(Theorem 1)로 이미 `PP_d` 내부인지 판별, 아니면 `T_f` 변환 후 probability simplex 투영 `S_d(·)`으로 해결.
5. 핵심 식: `w = T_f(Π_{S_d}(T_f(v)))` — parity polytope 투영을 probability simplex 투영으로 환원(Theorem 2), sort/desort 대신 argmin+cut-search로 대체해 HW 친화화.
6. probability simplex 투영(Algorithm 2)은 내림차순 정렬 `μ=descendingSort(v)` + prefix sum `s_i` + threshold index `ρ` 기반, 최종 `w_i = max(v_i - s_ρ, 0)`.
7. HW 구현 디테일: argmin은 min-tree(선형 복잡도/log 지연), 정렬은 Batcher's odd-even merge-sort 기반 sorting network(입력 무관 고정 지연), prefix sum은 Ladner-Fischer parallel prefix로 O(log d) 지연 구현. 고정소수점(1 sign + 가변 fraction bit) 사용.
8. 별도 학습/최적화 절차 없음(알고리즘적/구조적 HW 매핑).
9. Altera Stratix V FPGA 합성 결과 면적 O(d(log d)^2), 지연 O((log d)^2) 스케일링 확인, 8-bit 고정소수점(6 fraction bit)에서 정확도-면적 트레이드오프 실험.
10. LDPC 디코더 전체에 통합·BER 시뮬 없이 투영 모듈만 독립 검증했으며, 저자 스스로 "ADMM LP decoding 적용은 다음 연구 단계"라고 명시(개념 증명 이전 단계).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| parity polytope/simplex projection (ADMM-LP decoding 서브루틴) | 대응 없음 | Prime ECC는 min-sum 기반 message-passing 디코더이며 LP/ADMM decoding 프레임워크 자체가 없음 |
| sorting network / argmin HW 구조 | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `CNU_Update_New_Mag()` (min1/min2 sorter) | min-finding 개념은 유사하나 목적(투영 vs min-sum CN 연산)이 달라 직접 대응 아님 |
| 고정소수점 양자화 실험 | [channel.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) LLR 테이블(`PARAM_LLR`) | 양자화 정확도-면적 트레이드오프 방법론만 참고 가능, 직접 이식 요소 아님 |

적용 가치: **낮음** — ADMM/LP decoding 패러다임 전용 투영 회로로 Prime ECC의 min-sum 기반 디코딩 구조와 근본적으로 다른 알고리즘 계열이며, 저자도 LDPC 디코더 통합을 향후 과제로 남김.
