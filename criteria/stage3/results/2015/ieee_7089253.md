### B. 알고리즘 요약

1. 시스템: WiGig(802.11ad) 표준의 QC-LDPC block code(Nb=672bit, lifting factor Z=42, rate 1/2 및 13/16 예제)를 기반으로 채널은 AWGN.
2. 문제의식: 기존 LDPC block code는 BP 반복복호에 다수 iteration(8~100+)이 필요해 전력·복잡도·latency가 크고, 표준의 큰 block size는 입력 granularity(코드워드 생성에 필요한 최소 입력 비트수)를 거칠게 만든다.
3. 핵심 가정: QC-LDPC block PCM `H=[Hsys, Hpar]`(Hpar는 하삼각)에 trellis 기반 convolutional 종속성을 도입하면 원래 block code와 동일한 앙상블 성능을 유지하면서 구조를 바꿀 수 있다는 가정.
4. 핵심 기법: PCM을 Z×Z 서브행렬로 그룹화(`Hzsys`, `Hzpar`)하고 이를 신드롬 형성 행렬 `H^T`의 convolutional 표현(식 (1)~(4))으로 재구성해 TQC-LDPC convolutional code를 유도. constraint length `λ=LK+1`(L=syndrome memory, K=systematic row weight).
5. 식 (4)는 GF(2) 위에서 현재 parity 그룹이 이전 입력 그룹들과 이전 parity 그룹들의 XOR 합으로 결정됨을 나타내며, 이는 block code의 유효 등가 convolutional 형태를 규정한다.
6. 코드율 증대를 위한 2단 기법: single-row encoding(Hpar 대각화 후 비중첩 행 기반 basis set, R=q/(q+M))과 multi-row encoding(Hpar 하삼각 유지, 중첩 행 결합 후 생성다항식 재설계).
7. 부수 트릭: 최적 생성다항식(예 λ=10, R=1/2에서 G=[1467,1751,1325,1173] octal)을 표준 convolutional code table에서 선택해 BER 개선, puncturing(π)으로 최종 rate 조정.
8. 복호기: 반복 BP/min-sum 대신 non-iterative QC-Viterbi 디코더(Z개 병렬 Viterbi, 각 constraint length λ의 trellis, ACS butterfly 연산) 제안.
9. 결과: λ=10, R=1/2 QC-Viterbi가 동일 BER·SNR에서 scaled min-sum(flooding/layered) 대비 정규화 complexity 3배 절감, FPGA(Virtex-7)/65nm ASIC 합성에서 QC-Viterbi가 min-sum 대비 logic cost 34~61%, 동작주파수 5배 이상.
10. 한계: 무선(WiGig 60GHz) AWGN 채널 전용 검증이며 λ 증가 시 trellis state 수(2^(λ-1))가 지수적으로 증가해 낮은 SNR 영역(Eb/No<2.5dB)에서는 비현실적이라고 저자 스스로 명시.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QC-Viterbi 트렐리스 복호(반복 BP/min-sum 완전 대체) | 대응 없음 | Prime ECC는 min-sum 기반 iterative decoder(`decoder.cpp` `LDPC_Decoder()`)만 보유, Viterbi/trellis 구조 자체가 없음 |
| TQC-LDPC convolutional 부호 변환(PCM diagonalization, single/multi-row encoding) | `ecc_top.cpp` `Load_PCM()` | 기존 고정 QC-LDPC block PCM 구조를 convolutional로 재설계해야 하므로 이식 난이도 높음 |
| 인코딩(single-row/multi-row) | `encoder.cpp` `Generate_PCM_encoding()` | dual-diagonal 고정 인코더와 구조가 달라 직접 대응 어려움 |
| 복잡도/전력 비교 프레임워크(DieSize·Stages 기반) | 대응 없음 | Prime ECC는 gatecount/throughput을 별도 방식으로 산출, 해당 metric 프레임 없음 |

> 적용 가치: 낮음 — 반복 min-sum 복호 자체를 비반복 Viterbi 트렐리스 복호로 치환하는 구조적 전환이라 Prime ECC의 QC-LDPC min-sum 파이프라인과 이질적이며, 검증도 무선 AWGN 채널 한정.
