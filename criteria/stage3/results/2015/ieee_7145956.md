> 총평: 무선통신 HARQ용 rate-compatible IRA-like LDPC 확장(extending) 구성법으로, NAND 고전 고정-rate binary QC-LDPC 이식과는 목적·적용 영역이 달라 이식 가치가 낮음.

### B. 알고리즘 요약

1. 시스템은 BPSK/AWGN 무선통신 채널에서 type-II hybrid ARQ 프로토콜용 rate-compatible(RC) LDPC 부호를 다룸.
2. 기존 puncturing은 고rate화만 가능하고 extending(저rate화) 알고리즘은 상대적으로 미성숙한 문제를 해결하고자 함.
3. 모(mother) 부호로 QC 구조와 zigzag 패턴을 동시에 갖는 IRA-like 부호(`H=[H1,H2]`)를 채택.
4. 확장 시 `Hm` 아래에 `[C1,C2;Z1,B1,Z2,Z3,B2]` 형태로 행/열을 추가하되 하삼각 구조를 유지해 인코딩 효율성 확보.
5. `B1,B2`는 degree-1 노드 100% 비율의 항등행렬로 설정(degree-2 과다 문제 회피, [10] 대비 저복잡도).
6. `C1,C2`는 평균 행가중치 `RWave=dc` 를 만족하도록 non-zero 위치를 선택하고, right shift 값은 `Σ(-1)^(k+1) s_αk,βk = 0 mod p` 조건을 피해 length-4 short cycle을 배제하도록 설계.
7. 구성 절차: (A) H1을 QC-LDPC로 구성 → (B) non-zero 위치 선택(girth≥6 보장) → (C) shift 값 랜덤 배정 후 cycle 조건 검사·재시도.
8. 별도 학습/최적화 없이 결정론적 알고리즘 + 랜덤 시도 기반 구성.
9. 결과: mother code `k0=1016, R0=8/16`에서 `R=8/9~8/24`까지 확장, FER=10^-2 기준 기존 기법 대비 0.04~0.4dB 성능 이득, BER<10^-6까지 error floor 없음.
10. 한계: 무선 HARQ 시나리오 전용(AWGN/BPSK), HW 설계·복잡도·throughput 언급 없음, NAND에 필요한 고rate 고정 부호 설계와 무관.
