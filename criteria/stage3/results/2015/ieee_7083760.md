### B. 알고리즘 요약

1. 시스템: Thompson VLSI 모델(평면 그리드, 노드=연산·와이어=통신, 에너지 `Eproc = ξtech·Ac·τ` = 면적×클록수)에서 디코더 회로의 에너지 하한을 유도.
2. 문제: 기존 area-time 연구는 특정 부호/블록길이 최적화만 다뤄 "블록길이 n 증가에 따라 복호 에너지가 어떻게 스케일하는가"의 근본 한계가 미해결.
3. 채널: binary erasure channel(소거확률 ε), BSC로도 확장(`ε=2p_ch` degradation). 부호율 `R=k/n`.
4. 핵심 기법 1단: r단 nested minimum bisection으로 회로를 2^r 서브회로로 분할, 각 서브회로에 전달된 비트수 `Br`과 에너지·에러확률을 연결.
5. 핵심 식: `Aτ ≥ K·(√β/2^r)·Br` (Lemma 6) — 서브회로가 책임진 비트보다 주입 비트가 적으면 최소 1비트 추측→에러≥1/2 (Lemma 2).
6. 핵심 기법 2단: 병렬(Thm1)→직렬(Thm2, epoch 시간분할)→출력핀 가변(Thm3, 시공간 동시분할) 3케이스로 일반화.
7. 결과 하한: 병렬 `Ω(n(log n)^1/2)` (bit당 `(log n)^1/2`), 직렬 `Ω(n log n)`, 가변핀 `Ω(n(log n)^1/5)`.
8. 귀결: 용량접근(η=R/C→1) 시 bit당 에너지가 `1/log(1/(1-η))`로 발산 → capacity 근접이 에너지 최적이 아님.
9. LDPC 상한 예제: Tanner graph를 와이어로 배치(면적 `≤2|E|²`), regular LDPC는 `O(n²)`, Lentmaier의 `O(loglog n)` iteration 결과 결합 시 총에너지 `O(n²loglog n)`.
10. 한계: 순수 점근 이론(Big-Ω), HW 미구현·시뮬 없음, adiabatic/multi-layer/저에너지 메모리 회로는 모델 밖, 유한 블록 NAND ECC에 정량 적용 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Tanner graph 와이어 배치·라우팅 면적(2|E|²) | 대응 없음 (HW 배선 모델 무관, GT는 edge 선택 테이블) | 개념 참고만 — 우리 z=32 고정 QC-LDPC 라우팅 설계와 무관 |
| iteration 수 τ vs 에너지 | [decoder.cpp](../Prime_ECC_3.1_Claude/) `LDPC_Decoder()` 이터레이션 루프 | 간접 — iteration 감소가 에너지 절감이라는 정성적 방향뿐 |
| min-sum/VN 연산 노드 면적 | 대응 없음 (노드를 degree² black-box로만 취급) | 논문은 연산노드 내부를 다루지 않음 |
| 부호설계(area-limited Tanner graph) | 대응 없음 | 랜덤 Tanner graph 가정, 고정 H-matrix와 무관 |

적용 가치: **낮음** — 근본 하한 정보이론으로 특정 디코더/부호 설계 지침이 없고 HW 이식 대상 모듈이 없다. 유한길이 고rate NAND QC-LDPC에는 스케일링 결과가 직접 닿지 않는다.
