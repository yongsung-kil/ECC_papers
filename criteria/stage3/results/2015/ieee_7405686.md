### B. 알고리즘 요약

1. 대상 채널은 Rayleigh fading(BPSK, `y_j = D_j*x_j + n_j`, `D_j`는 Rayleigh 분포 곱셈성 잡음), 부호는 PEG로 생성한 42×147 표준 LDPC의 각 행(비0원소 7개)을 (7,4) Hamming code check matrix로 치환해 만든 (147,126,21,7,3,4) Hamming-GLDPC.
2. 문제의식: 표준 LDPC의 SPC(single parity check) check node는 단일 오류만 검출 가능해 constraint가 약함; Hamming code로 대체하면 더 강한 제약을 줄 수 있고, 표준 LDPC를 괴롭히는 4-cycle이 Hamming 블록 내부에 갇혀 성능에 영향을 덜 준다는 것이 핵심 가설.
3. 채널 모델: 채널 전이확률 `Pr(x_j=x|y_j) = [1+exp(-2*D_j*y_j*x/σ²)]^-1`로 정의, 이를 디코더 초기 정보로 사용.
4. 핵심 기법: Pearl의 belief propagation과 BCJR을 결합한 반복 디코딩(Algorithm 1) — VN은 REP 제약 유지, HCN(Hamming Check Node)은 성분부호의 codeword 확률 `q_l^j = Π p_i^j`(식 (2))을 모든 `l=1..2^k` codeword에 대해 exhaustive 계산 후 MLD로 갱신.
5. 핵심 식: HCN 출력 `p_i^j(0)/p_i^j(1)`은 해당 비트가 0/1인 codeword들의 `q_l^j` 합(식 (3))으로, 표준 LDPC의 단순 XOR parity 갱신보다 훨씬 무거운 exhaustive marginalization.
6. 확장: 부호 표기를 `(N,M,K,n,m,k)` 일반화 형태로 정의해 임의의 Hamming 성분부호(길이 n, 정보비트 k)로 확장 가능함을 제시(본 논문은 (7,4)만 실험).
7. 부수 분석: weight enumerating sequence(WES)로 최소 Hamming distance(=35) 및 codeword weight 분포(주로 60~100)를 도출해 error-floor가 매우 낮을 것으로 예측(수치 시뮬은 BER 10^-7까지만 확인, error floor 미관측).
8. 별도 학습/최적화 절차 없음 — base LDPC는 PEG로 1회 생성.
9. 결과: 동일 check matrix의 표준 LDPC 대비 BER 10^-7에서 최대 약 2dB 이득, 평균 iteration도 항상 더 적음; 4-cycle을 제거한 표준 LDPC와 비교해도 iteration 3회 이하 구간에서는 GLDPC가 더 빨리 수렴(표준 LDPC는 Eb/N0<9dB, iter<3에서 미수렴이나 GLDPC는 iter=2로 수렴).
10. 한계: 계산복잡도가 표준 LDPC 대비 훨씬 높음(저자가 결론에서 명시, 향후 효율적 복호 알고리즘 개발이 과제), rate 21/147(≈0.14)로 매우 낮아 NAND 고rate ECC와 스펙 상이, HW 구현·gate count 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Hamming 성분부호 HCN 업데이트(MLD, exhaustive codeword marginalization) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | Prime ECC는 min-sum 기반 단순 XOR/CN 근사 연산이라, 2^k codeword 전수조사형 HCN 갱신은 구조가 전혀 다르고 연산량도 급증해 이식 난이도 매우 높음 |
| Rayleigh fading 채널 모델(`Set_R_Offset`류 채널 파라미터) | `channel.cpp` `Set_R_Offset()` | Prime ECC는 AWGN/RBER/fixed-error 채널만 지원, 곱셈성 페이딩 채널은 대응 없음 |
| GLDPC 부호 구성(H-matrix에 Hamming block 삽입) | `ecc_top.cpp` `Load_PCM()` | 특정 QC-LDPC 고정 구조와 무관한 대수적 구성이라 재검증 부담 큼 |

> 적용 가치: 낮음 — 저rate 무선 페이딩 채널 대상 GLDPC로 NAND 고rate binary QC-LDPC와 부호 스펙·채널 모델이 모두 상이하며, HCN 업데이트 복잡도가 Prime ECC의 min-sum 대비 과도하게 높음.
