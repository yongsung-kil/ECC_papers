### B. 알고리즘 요약

1. multi-edge type(MET) LDPC 앙상블로 rate-compatible spatially-coupled LDPC 코드를 모델링, incremental redundancy(IR) 스텝마다 variable/check node를 추가해 확장하는 구조.
2. 문제: 기존 regular rate-compatible SC-LDPC(문헌 [6])는 낮은 rate로 갈수록 전체 규칙성(regular) 제약 때문에 variable node degree `J`가 선형적으로 증가해, 작은 coupling width `w`에서 성능·복잡도가 나빠짐.
3. 가정: BEC 및 AWGN(BIAWGN) 채널, density evolution(DE) 기반 iterative decoding threshold 분석, coupling length `L→∞`.
4. 핵심 기법: check node degree `K`를 고정하지 않고 매 IR 스텝마다 감소시키는 "relaxed degree evolution" 구성 도입, 확장 variable node degree `Ja,p`는 작게(Ensemble B는 `Ja,p=1`) 고정.
5. 핵심 식: 복잡도 `C = J̄/R` (식 9) — 평균 variable node degree와 rate의 비율로 BP 디코딩 복잡도(edge 수 기반)를 정의, degree 증가가 곧 복잡도 증가로 직결됨을 보임.
6. 두 가지 relaxed 앙상블(A: `Ja,p=Ka,p`, B: `Ja,p=1`)을 비교, B가 낮은 rate에서 더 낮은 복잡도와 더 나은 threshold를 보임.
7. 부수 트릭 없음(순수 앙상블 구성 파라미터 조정, HW/양자화 관련 디테일 없음).
8. 학습/최적화 절차 없음(수동으로 degree table 설계, DE로 threshold 수치 계산).
9. 결과: `w=3`, `R=0.2`에서 relaxed Ensemble B는 Shannon limit 대비 `ΔEb/N0 ≈ 2.2dB` gap, regular(K=25)는 `≈5.5dB` gap으로 relaxed 구성이 유의하게 우수. `w=10`에서는 거의 포화(Shannon limit 근접).
10. 한계: 순수 asymptotic(density evolution) 분석으로 유한 length 시뮬레이션·HW 구현 전혀 없음. HARQ/IR 통신 채널(BEC/AWGN) 대상이며 NAND flash 채널 모델과 무관.
