### B. 알고리즘 요약

1. 대상은 AWGN 채널, BPSK 변조, SPA(sum-product) 반복복호를 쓰는 일반 이진 LDPC 부호 구성 문제.
2. 기존 대수/조합 구조 기반(BIBD, finite geometry, partial geometry 등) 구성법을 확장해 다양한 rate/length를 동시에 얻을 수 있는 새 조합적 도구(RGDD)를 제안.
3. RGDD(k, λ=1, g; υ)는 point 집합 X, group 분할 G, block 집합 B로 정의되며 block들이 parallel class로 분할 가능(resolvable)한 group divisible design.
4. RGDD의 υ×l incidence matrix I(`I_{i,j}=1` if point i ∈ block j)에서 임의의 ρ개 parallel class(≤전체 p개)를 골라 만든 υ×(ρυ/k) 부분행렬 H를 parity-check matrix로 사용.
5. RC-constraint(임의 두 행/열이 공통 1을 2개 이상 갖지 않음)가 RGDD 구조상 자동 성립해 4-cycle이 없고 girth≥6 보장; 코드 rate는 대략 `1-k/ρ`, length는 `ρυ/k`.
6. ρ(사용하는 parallel class 수)를 늘리면 rate가 오르는 방식으로 하나의 RGDD에서 다양한 rate/length 부호열을 생성(부호설계 확장 트릭).
7. H의 rank는 실험적으로 거의 full-rank(υ에 근접)임을 수치로 확인, 별도 최적화(GA/DE 등)는 없음.
8. 결과: (5166,4915) rate 0.951 부호가 50 iteration에서 BER 10⁻⁶·10⁻⁷·10⁻⁸ 지점에서 Shannon limit 대비 각각 1.2/1.25/1.3dB, BER 10⁻⁸까지 error floor 미관측; (1008,757) 부호는 동일 rate QC-LDPC[16] 대비 BER 10⁻⁶에서 0.65dB 이득이나 QC 구조가 없어 인코딩/디코딩 시간이 더 걸림.
9. 20 iteration으로도 50 iteration과 성능차 0.04dB 이내로 수렴이 빠름(정성적 언급, HW 이터레이션 감소 수치는 없음).
10. 한계: 순수 이론/시뮬 논문으로 HW 미설계, non-QC 구조라 실제 인코더/디코더 구현 복잡도가 높고 동일 code length에서 여러 rate를 자유롭게 고를 수 없는 제약(RGDD 파라미터에 rate·length가 종속).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| RGDD 기반 H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | non-QC 구조라 이식 난이도 높음, 재검증 부담 큼 |
| SPA(sum-product) 복호 | 대응 없음 | Prime ECC는 min-sum 근사만 지원, full-tanh SPA 미지원 |
| regular LDPC 부호설계 | 대응 없음 | Prime ECC는 QC-LDPC(base+circulant) 고정 구조 |

적용 가치: 낮음 — non-QC regular 부호 구성법으로 Prime ECC의 QC-LDPC 고정 구조와 근본적으로 맞지 않고, 복호도 SPA 기준이라 min-sum HW와 직접 비교 어려움.
