> 총평: LDPC factor-graph를 희소 측정행렬로 차용한 compressed sensing BP 재구성의 양자화 vs 가우시안근사 성능 분석(DE)으로, 실수 신호 추정 문제라 NAND binary QC-LDPC 오류정정에 이식할 요소가 없어 이식성·추천도·NAND관련성 모두 하/낮음.

### B. 알고리즘 요약

1. 문제: `y=Φx+w` (실수 신호 `x`, 측정노이즈 `w~N(0,σ_W^2)`, `m<n`)의 compressed sensing 재구성을 factor-graph BP로 수행 — MAP/MMSE 추정.
2. 신호모델: Bernoulli-Gaussian prior `fX(x)=(1-ε)δ0(x)+εφ(x/σ)` (sparsity `ε`), 측정행렬 `Φ`는 `(wc,wr)`-regular Gallager-Rademacher ensemble(비영엔트리 ±1 등확률).
3. 핵심 대비: BP 메시지가 실수 위 분포라 실제 구현에 (a) 양자화 PMF (b) 가우시안근사(RBP) 두 방식 사용 — 본 논문은 이 둘을 density evolution(DE)으로 정량 비교.
4. 양자화(Algorithm 2): `[-10σ,10σ]` 범위를 243개 등간격 점으로, FFT circular convolution의 aliasing/zero-padding 문제 지적, 실효 support는 `√(wr-1)`로만 성장.
5. 가우시안근사(RBP): 각 메시지를 `(m,v)` 쌍으로 추적, factor-node 갱신식 (6)(7) `v̂=Φ^{-2}(σ_W^2+ΣΦ^2 v)`, `m̂=Φ^{-1}(y-ΣΦ m)`, VN은 prior 곱으로 비가우시안이나 mean/variance만 수치계산.
6. 핵심 기법(shift 연산): 신호가 랜덤이라 DE가 자동확장 안됨 → genie가 참신호 `x*`로 `x=x̃+x*` 병진 → 관측 `ỹ=0`인 shifted system으로 변환, prior를 PDF 위 분포 `F(z)`로 정의(식 3).
7. 이론 뒷받침: 원 시스템과 shifted system의 MMSE가 동일함을 증명(식 4,5) → shift 후 DE가 유효.
8. DE 알고리즘: Monte Carlo population dynamics(Algorithm 3 CSDE / Algorithm 4 CSDE-GA), 메시지를 N개 샘플 bag으로 표현, damping 계수 `γ`로 RBP 수렴 보정(식 13,14).
9. 결과: `n=5000`, `(3,6)/(5,10)/(7,14)` ensemble에서 DE가 시뮬 곡선과 일치, `ε_BP≈0.28` 근처 sharp transition(LDPC BP-threshold 유사), CSBP-GA가 다른 방식보다 약 200배 빠름(작은 wc,wr에선 가우시안근사 부정확).
10. 한계: HW 미설계, 시뮬만, 작은 node degree에서 CSDE-GA 불안정(open), 실수 신호 재구성 전용.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP 메시지 양자화(PMF) / 가우시안근사 | `decoder.cpp` `VN_Cal_HD()` 등 양자화 (부분) | Prime ECC는 정수 magnitude 양자화 min-sum이며, 실수 PMF/가우시안 메시지 표현은 목적·표현이 상이 |
| factor-graph BP(FNOP/VNOP) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 메시지 전파 개념만 공통, 본 논문은 tanh/convolution 기반 CS 추론이라 min-sum 정정과 무관 |
| Gallager-Rademacher 측정행렬 | 대응 없음 | LDPC 그래프를 CS 측정행렬로 차용한 것이며 부호설계(H-matrix 로드)와 무관 |
| density evolution / shift 분석 | 대응 없음 | Prime ECC에 DE 분석 모듈 없음, 실수 신호 MMSE 분석은 이식 대상 아님 |

적용 가치: **낮음** — LDPC factor-graph를 차용한 compressed sensing 실수 신호 재구성 분석으로, NAND binary QC-LDPC 오류정정 디코더/HW에 떼어 쓸 신규 기법이 없음.
