### B. 알고리즘 요약

1. 대상 채널은 BPMR/HAMR/TDMR 등 초고밀도 자기기록에서 발생하는 2D 층간간섭(ITI)+ISI가 결합된 2D-ISI 채널.
2. 기존 full 2D BCJR 검출기는 최적이지만 복잡도가 과도하고, random-like 최적화 LDPC([4])는 성능은 좋으나 구조가 불규칙해 인코딩 복잡도가 블록길이에 이차적으로 증가.
3. 채널 모델은 4Tb/in2 밀도의 2D-ISI, 검출기로 저복잡도 2D GA-IRCSDFA(Gaussian approximation 기반 iterative row-column soft decision feedback)를 사용.
4. 4Tb/in2 2D-ISI 채널용 degree sequence(기존 EXIT-chart 기반 설계에서 일부 수정)를 base matrix로 사용한 protograph 기반 QC-LDPC 부호 설계.
5. QC 구조 덕분에 저복잡도 quasi-cyclic 인코더와 병렬화 가능한 protograph 디코더 구조를 확보.
6. 부호율은 전 코드 공통 8/9, BP 디코더 최대 30회 반복, 검출기-디코더 간 joint iteration 최대 10회, GA-IRCSDFA 내부 component detector 간 반복 3회.
7. 세 가지 코드길이(Nc=4608, 9216, 13824)로 몬테카를로 시뮬레이션 수행.
8. 별도 학습/최적화 절차 언급 없음(기존 degree sequence를 변형 사용).
9. BER 10^-5 기준, 제안 부호가 AWGN 최적화 부호(Scheme 3) 대비 약 0.55dB 이득, 훨씬 복잡한 random-like 2D-ISI 최적 부호(Scheme 2) 대비 0.05dB 손실에 불과.
10. HW 미설계, 시뮬레이션에 국한, 특정 자기기록 2D-ISI 채널·검출기 조합에 종속된 결과.
