> 총평: run-time 실수비교 없이 low-error-probability VN을 skip하는 스케줄링으로, 기존 Prime ECC의 iterative message passing 루프에 조건부 skip 로직만 추가하면 되는 저비용·저리스크 이식 후보.

### B. 알고리즘 요약

1. 2-bit/cell MLC NAND 채널을 프로그래밍 노이즈/RTN/retention noise의 합성 가우시안 모델로 정의하고, 6-level 비균일 양자화(R1~R6)로 4개 data region(D1~D4)과 3개 erasure region(E1~E3)을 구성.
2. Shuffled BP는 flooding보다 빠르지만, 기존 FC/LS 기법은 매 iteration마다 real-valued threshold 비교가 필요해 추가 연산 부담이 있음.
3. 채널 비균일 양자화로 인해 입력 LLR의 오류확률이 region별로 크게 비대칭적이라는 관찰(`Pe(Lmsb∈D1)`, `Pe(Lmsb∈D4)`가 다른 region보다 수 자릿수 낮음)이 핵심 가정.
4. 이진 상태벡터 `U={u_i}`를 정의하여, `Lmsb∈D1` 또는 `D4`인 variable node는 `u_i=1`로 플래그하고 초기 α iteration 동안 메시지 업데이트를 skip.
5. α iteration 내 parity-check(정지조건)를 만족하지 못하면 전체 `u_i=0`으로 리셋 후 모든 VN을 업데이트 재개 — 임계값 비교 없이 미리 정한 채널 region 분류만으로 skip 여부 결정.
6. High-to-low column-weight(CW) 기반 스케줄링[14]과 결합하여 QA-BP + CW 방식(`I_QA-BP,H-L`)으로 추가 수렴가속.
7. 공정 조건 동일화를 위해 QA-BP와 conventional 방식의 최대 VN 업데이트 횟수를 동일하게 고정하여 공정 비교.
8. 학습/최적화 절차 없음 — α, Imax는 수치 시뮬레이션으로 경험적 선택(α=5, Imax=40).
9. 결과: rate-9/10 irregular LDPC(N=4544, PEG 구성)에서 QA-BP 단독 최대 25%, QA-BP+CW 결합 최대 44% 평균 iteration 감소(정지, PE=12.5K 기준), FER 성능 저하 없음.
10. 한계: HW 설계/gatecount·throughput 미제시, 시뮬 검증만 수행, 2-bit/cell 특정 채널 모델과 특정 부호(N=4544)에서만 검증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Shuffled BP VN skip 스케줄링 (조건부 업데이트) | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | VN 업데이트 순서/조건에 skip flag(`u_i`)만 추가하면 되는 낮은 난이도 이식 대상 |
| 채널 region 기반 LLR 오류확률 분류(D1~D4) | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | soft-read quantization threshold 설계와 직접 연결, LLR 테이블 기반 region 매핑에 활용 가능 |
| High-to-low column-weight 스케줄링(CW, 결합기법) | `decoder.cpp` `C2V_Cal()` (스케줄 순서) | VN 업데이트 순서 결정 로직에 결합 가능하나 별도 정적 정렬 필요 |
| iteration별 LLR 테이블/threshold | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | 기존 적응형 LLR 테이블 구조와 유사한 개념으로 재사용 가능 |

> 적용 가치: 높음 — 추가 real-valued 비교 없이 기존 iteration 루프에 저비용 조건부 skip만 추가하는 구조라 Prime ECC의 min-sum 루프에 낮은 리스크로 이식 가능.
