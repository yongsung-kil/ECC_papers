### B. 알고리즘 요약

1. 2-D ISI 자기기록 채널(4 Tb/in² 밀도)에서 turbo equalization 구조(내부 2-D-DET + 외부 LDPC 디코더) 사용, 부호율 8/9, 길이 4608~10k.
2. 기존 AWGN 최적화 protograph(AR4JA, AR3A)나 punctured-node 구조가 2-D ISI 채널에서 성능 저하 및 추가 puncturing 구현 부담 발생.
3. 채널은 `L_M × L_N` 2-D ISI 응답 행렬 `H`와 AWGN 잡음 `n(i,j)`로 모델링, BCJR 2-D-DET 및 저복잡도 IRCSDFA-GA 2-D-DET 두 검출기 사용.
4. EXIT 차트 분석으로 protograph 설계: 내부 검출기와 외부 디코더의 EXIT 곡선이 서로 겹치지 않고 근접하도록 decoding threshold 최소화.
5. 점근 가중치 분포 `r(δ)` 계산(식 5, 6)으로 최소 거리 성장률 `δmin`을 평가해 error-floor 특성 예측.
6. Degree-2 variable node 비율 조절이 핵심 트릭: 늘리면 waterfall 개선, 줄이면 error-floor 개선 — 두 상충 목표의 균형점 탐색.
7. Punctured node를 사용하지 않는 설계 규칙 채택(2-D ISI 시스템에서 인코더/디코더 puncturing 컴포넌트 회피 목적).
8. 2-D-P1(degree-2 노드 일부 유지)과 2-D-P2(degree-2 노드 완전 제거, edge 추가) 두 가지 protograph를 EXIT/weight-distribution 분석으로 도출.
9. 결과: 2-D-P2가 기존 최적화 irregular LDPC 대비 BER 10⁻⁶에서 0.22 dB 이득, δmin은 2-D-P1 0.023 / 2-D-P2 0.031.
10. 한계: 순수 시뮬레이션/이론 분석뿐이며 HW 미설계, 2-D ISI 자기기록 채널 전용 가정으로 NAND 채널(비ISI, RBER 기반)과 직접 대응 안 됨.
