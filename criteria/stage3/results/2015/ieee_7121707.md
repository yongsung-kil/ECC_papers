### B. 알고리즘 요약

1. 시스템: coherent 광통신 QPSK/16-QAM, phase slip(cycle slip)에 강인한 iterative differential decoding + LDPC FEC, 목표 rate `r=0.8`(25% OH).
2. 문제: 위상복구 시 phase slip 발생 -> 기존 differential decoder는 채널 mismatch로 큰 error floor 발생.
3. 모델: slip-extended AWGN `z[t]=(x[t]+n[t])·exp(...)`, slip state `s̃[t]=s̃[t-1]+s[t]`, `P(S=0)=1-Pslip`.
4. 핵심 기법 1단: differential code trellis를 확장(`V`상태 + phase-slip 전이 추가)하여 slip-resilient BCJR 구성. EXIT 분석으로 mismatched trellis가 비단조 -> error floor 원인임을 규명.
5. 핵심 식/의미: 확장 trellis 시 EXIT 특성이 단조증가 -> low error-floor 복호의 전제조건 충족.
6. 핵심 기법 2단: protograph 기반 terminated SC-LDPC 설계(`μ=2`, `dv=3`, `dc=15`), rate loss `rL=1-((L+μ)/L)(m/n)`, `L=200`으로 rate loss 무시.
7. 부수: 1837개 coupled protograph 중 threshold 상위 50개 선정 -> index 3 채택 (`B0=(1 1 1 1 1)`, `B1=(1 0 0 0 0)`, `B2=(1 2 2 2 2)`). `P=40` lifting 후 50×50 circulant QC 부호 생성, `Hi` 크기 2000×10000.
8. 최적화: windowed decoding 3종(`w=6/I=3`, `w=9/I=2`, `w=18/I=1`) — differential decoder 실행횟수(18)·비트당 연산 동일.
9. 결과: `Pslip=0`에서 `w=9` setup이 Code(1)을 능가, `Pslip=0.01`에서 단일 SC 부호가 error floor 없이 관찰됨(universality). 참조 LDPC는 여러 부호 필요.
10. 한계: 시뮬만, HW 미설계. 광통신 phase-slip 채널·differential decoding에 특화된 부호설계.
