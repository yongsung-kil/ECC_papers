### B. 알고리즘 요약

1. 시스템은 noncoherent M-FSK(`M=4,8`) 물리계층 network coding(PNC) 양방향 릴레이(TWRC)의 multiple-access phase, DVB-S2(`N=64800, R=3/5`)·WiMAX(`N=2304, R=2/3`) LDPC 부호 사용.
2. 문제: 표준 LDPC는 이 릴레이 수신(두 소스 신호의 전자기 합 + Rayleigh fading + noncoherent 복조) 조건에 최적화되어 있지 않아 error-rate 개선 여지가 있음.
3. 채널/모델: flat Rayleigh fading `Y = X1H1 + X2H2 + N`, 복조는 super-symbol probability mapper + DNC-SOMAP, 디코더와 반복적으로 extrinsic 교환(BICM-ID), CSI(full/partial/no) 케이스 구분.
4. 핵심 기법: Tanner graph variable-node degree distribution `V={dv,i:oi}`를 EXIT 차트 curve-fit으로 최적화 — VND/CND 전달특성이 최저 SNR에서 맞닿는 분포 선택.
5. 핵심 식: VND 곡선 `IE,VND = Σ bi·J(√((dv,i-1)J^-1(IA,VND)^2 + [J^-1(IE,DET)]^2))` (식 (16)-(17)), CND는 `IE,CND = 1 - J(√(dc-1)·J^-1(1-IA,CND))` — 두 곡선의 EXIT threshold(맞닿는 최고 SNR)를 최소화.
6. 제약: eIRA 부호 클래스 유지(systematic encoding 보장) — `H=[H1|H2]`에서 eIRA인 `H2`는 표준 그대로 두고 `H1`의 VN degree만 최적화, check-regular(`dc=11` DVB-S2, `dc=10` WiMAX) 유지.
7. detector 특성: DNC-SOMAP 출력과 network-coded bit 간 mutual info `IE,DET`를 Monte Carlo로 구해 3차 다항식 `fDET(x)`로 fit, VND 결합특성 계산에 사용.
8. 절차: `M`·CSI 4개 구성마다 dv 조합 전수 탐색(dv2∈{3..100} 등)으로 EXIT threshold 최소 분포 선정, 해당 degree로 PCM 랜덤 생성.
9. 결과: DVB-S2는 `M=8` partial CSI에서 약 1.2dB, WiMAX는 최대 0.4dB 개선(표준 대비 0.3~1.2dB), iteration은 100회 고정.
10. 한계: HW 미설계·시뮬만, noncoherent FSK PNC/Rayleigh 채널 특화라 최적 degree가 SOMAP detector 특성에 종속되어 다른 채널로 이전 불가.
