### B. 알고리즘 요약

1. 시스템: D-GLDPC(doubly-generalized LDPC) 부호를 AWGN(및 BEC) 채널에서 설계·분석, base matrix M×N, SVN/SCN이 성분부호로 치환됨.
2. 문제: 기존 D-GLDPC는 check component(Hamming/BCH/Hadamard) 강화로 정정력을 얻지만 부호율 손실 발생, VN측 개선 여지 미탐구.
3. 모델: SVN에 `(n_i, n_i-1)` SPC 부호(최소거리 2)를 systematic(S)/cyclic(C)/anti-systematic(A) 3형태로 배치, SCN에는 2-D SPC product code(SPC-PC, dmin=4) 사용.
4. 핵심기법: SPC 부호의 생성행렬 형태별로 weight-2 codeword 분포 `B^i_{2,u}`가 달라져 stability polynomial `P_i(x)`가 달라짐.
5. 핵심식: A형은 `P_i^A(x)`의 역함수가 `x`(length-2 반복부호)와 (0,1)에서 교차 → stability bound 기여로 threshold가 크게 상승(부록에서 교차 증명).
6. 확장: SPC 최소거리 2가 EXIT 곡선 간 area gap을 메워 반복부호 대비 부호율 손실을 보완.
7. 구현 디테일: split information function `e_{g,h}`(부분행렬 rank 합)로 SPC 형태별 EXIT 함수 `I_E`를 폐형식으로 유도.
8. 최적화 절차: EXIT chart로 SVN/SCN degree distribution 최적화, threshold 0.141 dB로 고정해 3형태 area gap 비교.
9. 결과: C형 SPC가 stability bound·decoding threshold 최소(최우수), S형 차선, A형 최악; 표 예시 (3,2)²SPC-PC+(9,8)SPC rate 0.375, threshold 0.727~1.726 dB.
10. 한계: 순수 이론(EXIT/stability bound), HW·양자화·throughput 없음, generalized LDPC라 성분부호 복호 필요.
