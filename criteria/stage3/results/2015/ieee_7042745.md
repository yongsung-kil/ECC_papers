### B. 알고리즘 요약

1. 대상: rate `R=b/c` (ms,J,K)-regular LDPC 컨볼루션 부호(LDPC-CC, spatially-coupled)의 주기적 puncturing.
2. 문제: puncturing 패턴 후보 중 복구 불가 노드(∞-SR)를 포함해 성능 저하시키는 패턴을 효율적으로 배제.
3. 핵심 도구: `M`배 확대한 변환 다항식 syndrome former matrix `H̃T(D)`와 그 weight matrix `B`로 recovery tree 연결을 행렬 대수로 치환.
4. 복구성 판정: 변환 puncturing 패턴 `ã`와 `B`의 곱 `b̃=ã·B`의 각 엔트리로 check node 연결 상태 분류(0/1/≥2)를 판별.
5. `b̃k=1`이면 해당 punctured 노드는 1-SR(1-step recoverable), `≥2`이면 unreliable check로 이어짐.
6. Algorithm 1: `b̃`에 "1"이 사라질 때까지 반복하며 각 iteration `m`에서 m-SR 노드 수 `Nm-SR=|A4|` 집계, 남은 집합이 ∞-SR.
7. 이 알고리즘은 QC/비구조 LDPC-BC에도 적용(패턴·weight matrix를 `aBC`·`H^T_BC`로 치환).
8. 상한(Theorem 2): `R∞ ≤ Rf = bP/(cP − rank(sgn(B)) − 1)` — 이 rate 이상이면 ∞-SR 불가피.
9. 결과: 예제부호 recoverability를 Algorithm 1 0.000143s vs 그래프법 0.168s; (1205,482) QC-LDPC-BC는 0.0615s vs 4.47s(>70배 빠름). 예제서 `Rf=4/5`.
10. 한계: 순수 이론(복구성 판정 알고리즘)만, BER/디코더 성능·HW 없음, rate-compatible puncturing 시나리오 전용.
