### B. 알고리즘 요약

1. LDPC block code protograph(작은 bipartite graph, base matrix `B`)를 M-fold graph lifting하여 임의 크기 Tanner graph를 얻는 표준 구성법을 다룸.
2. 기존 (J,K)-regular 코드는 waterfall에서 capacity 대비 부족, 최적화된 irregular 코드는 error-floor에 취약하다는 trade-off를 해결하려 함.
3. 채널 모델은 BEC(erasure probability `ε`)와 binary-input AWGNC 두 가지, 모두 이론적 density evolution(DE) 분석 대상.
4. 핵심 기법은 "Edge Spreading Rule": block protograph를 시간축으로 L개 복제한 뒤, 각 variable node의 edge를 coupling width `w`에 걸쳐 이웃 시간 인스턴스의 check node로 분산시켜 convolutional protograph `B[-∞,∞] = Σ B_i`를 구성.
5. 이 convolutional protograph를 길이 `L`로 termination하면 SC-LDPC-BC 앙상블(base matrix `B[0,L-1]`)이 되며, 경계 check node의 degree 감소가 낮은 오류확률의 "wave" 전파를 일으켜 BP 임계값을 향상시킴.
6. Tail-biting 구성(`B[0b,L-1]`)도 대안으로 제시, rate loss 없이 convolutional 성질을 유지.
7. DE 재귀식 `p(i) = ε λ(1-ρ(1-p(i-1)))`(BEC) 및 protograph별 edge-perspective DE 재귀(식 15-16)로 BP threshold `ε*`를 계산.
8. 최적화/학습 절차는 없음(수치 DE 반복 계산과 asymptotic weight enumerator 조합론적 유도만 사용).
9. 결과: C(3,6,L) 앙상블은 L≈20에서 threshold가 언더라잉 (3,6)-regular 코드의 MAP threshold(0.4881)에 수렴(BP만으로 0.429→0.488), gap-to-capacity 0.012까지 감소; 모든 variable node degree>2 조건에서 오류확률이 반복횟수에 대해 최소 이중지수적으로 감소.
10. 한계: 전부 무한 lifting factor(M→∞) 가정의 asymptotic ensemble 분석이며, 실제 유한 length 코드의 BER 시뮬레이션, 양자화, HW 구현/합성은 다루지 않음(windowed decoding 등 구현 이슈는 참고문헌으로 우회).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Protograph 기반 코드 구성(edge spreading, base matrix) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC-LDPC 고정 구조라 protograph 앙상블 재설계는 이식 난이도 최상급, 실질 대응 낮음 |
| Asymptotic DE/threshold saturation 분석 | 대응 없음 | Prime ECC는 bit-exact 유한 length 시뮬레이터이며 이론적 무한 앙상블 DE 분석 모듈 없음 |
| BP(flooding) 디코딩 스케줄 | `decoder.cpp` `LDPC_Decoder()` | 본 논문은 min-sum이 아닌 이론적 BP(erasure/LLR) 분석이라 direct 대응 약함 |

> 적용 가치: 낮음 — 순수 asymptotic 이론(무한 lifting, DE, weight enumerator)이며 유한 length·양자화·HW 요소가 전혀 없어 binary QC-LDPC HW 이식과는 거리가 멂.
