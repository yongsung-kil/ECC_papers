> 총평: 우주 telecommand용 short rate-1/2 LDPC에 ML근사(MRB) 디코딩을 붙이는 통신 특화 연구로, 고rate·긴 NAND QC-LDPC에는 복잡도상 이식 불가.

### B. 알고리즘 요약

1. 대상 부호는 NASA/CCSDS TC 링크용 short binary LDPC 두 개: `LDPC(128,64)`, `LDPC(512,256)`, 모두 rate `Rc=1/2`, BPSK/AWGN 채널.
2. 풀려는 문제: 기존 BCH(63,56) hard-decision은 CER=1e-5에서 Eb/No≈9.1dB 필요해 미래 미션에 부족, SPA/MS/NMS도 ML 한계와 격차가 큼.
3. 핵심 기법: MRB(Most Reliable Basis, 순서통계 기반) 디코딩 — 수신 신뢰도 상위 `k`개 비트 선택 후 G를 가우스-조던 소거로 systematic `G*`로 변환.
4. `G*`로 후보 codeword `c*=v*G*` 생성, order `i`까지 Hamming weight `w<=i`인 test error pattern(TEP)들을 더해 Euclidean 거리 최소 후보로 갱신.
5. `i=k`면 ML 디코딩과 동일하나 `2^k`개 벡터로 비현실적 → order `i`와 acceptance threshold `A`, 확률순 TEP 리스트로 평균 테스트 수 `N_TEP` 축소.
6. 하이브리드: 먼저 IA(SPA-LLR/NMS)로 복호, 실패(유효 codeword 미발견) 시에만 MRB 호출. MRB 호출률 α = IA의 CER.
7. 구현 디테일: MRB는 IA 마지막 iteration의 soft값을 쓰지 않고 원 수신어에서 시작(열화된 soft값 회피가 더 효과적).
8. 양자화: mid-tread 선형/로그 2종 비교, `q=4,5,6` bit. 하이브리드는 로그법이 선형 대비 1bit 절약.
9. 결과: LDPC(128,64)에서 MRB(4)/하이브리드가 CER=1e-5 기준 SPA 대비 약 1.6dB 이득, union bound에 근접. LDPC(512,256)은 이득 0.15dB로 미미. q=6이면 거의 이상적.
10. 한계: HW 미설계(칩 면적·전력 향후 과제), 복잡도는 짧은 코드에서도 SPA 대비 약 10배, 긴 코드에선 MRB order 제약(i=3)으로 이득 급감.

### C. Prime ECC 관련 모듈 핀포인트

| IA(SPA/MS/NMS) 반복복호 | [decoder.cpp](../Prime_ECC_3.1_Claude/) `LDPC_Decoder()` | Prime ECC는 min-sum 사용, SPA-LLR와는 다른 근사 |
| 양자화(선형/로그 q비트) | [ecc_data.h](../Prime_ECC_3.1_Claude/) `PARAM_LLR` | LLR 양자화 테이블 대응하나 논문은 short-code 특화 |
| 하이브리드 조기종료/재호출 트리거 | [partial_CRC.cpp](../Prime_ECC_3.1_Claude/) `partial_crc_HW_T4()` | 유효 codeword 판정 개념만 유사, MRB fallback 자체는 대응 없음 |
