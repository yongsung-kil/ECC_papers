### B. 알고리즘 요약

1. 시스템/채널: Binary Erasure Channel(BEC), 시스템릭 ARA(accumulate-repeat-accumulate) 부호를 protograph 체인으로 spatially coupling한 SC-ARA 코드, 예제 rate `R=0.5`, block length 8184(coupled)/8192(self-matched).
2. 문제: 기존 capacity-achieving LDPC/IRA/ARA 부호는 gap-to-capacity가 줄수록 정보비트당 복잡도(density)가 무한정 커짐 — bounded complexity를 유지하며 capacity에 접근하는 구성이 필요.
3. 핵심 가정/모델: proto-graph 기반 ARA 부호(accumulator-repetition-interleaver-combiner-accumulator 직렬 연결)를 `2L+1`개 체인으로 spatially coupling, punctured bit node를 좌우 인접 proto-graph에 연결하는 구조.
4. 핵심 기법: 시스템릭 ARA의 "repeater-combiner" 단(punctured bit ↔ parity check B node)에 spatial coupling 적용, 메시지 전달 확률 변수(예: `p_i`, 고정점 오식률 `x_i`)로 density evolution 폐형식(식 4, 7, 8) 유도.
5. 핵심 식: 고정점 DE `x_i = f(p, λ, ρ, w, q, a, ...)`(식 4/7)로 iteration별 erasure 확률 수렴 여부를 판정, tilted degree distribution `λ~, ρ~`(식 14/15)로 graph reduction 후 등가 LDPC로 변환.
6. 확장: NSIRA(non-systematic IRA) 코드에도 동일 spatial coupling을 적용한 변형 DE(식 8) 제시, 코드 rate `R(q,a,L,w)→1/(1+q/a)`(식 9), density `D→q/(1+q/a)`(식 10) (`L→∞`).
7. 부수 트릭: cycle-4 및 small stopping set에 의한 error floor 완화를 위해 rate-1 outer code(균등 랜덤 선택)를 부가, ensemble 평균 성능(entropy 수렴)만 시뮬레이션.
8. 학습/최적화 절차: 없음(닫힌 형식 DE + 안정성 조건(식 5/6)으로 threshold 유도, degree distribution truncation으로 bounded complexity 확보).
9. 결과: `(8,4,16,w=16)` SC-ARA(n=8184)가 자기정합(self-matched) ARA(n=8192, rate 0.5)보다 waterfall 영역 및 threshold에서 우수, BP threshold가 randomize parameter `w` 증가 시 Shannon limit(1/2)에 근접(작은 gap 존재).
10. 한계: 순수 BEC 채널 이론/DE 분석뿐(HW 미설계, ASIC/FPGA 없음), NAND에서 쓰는 binary symmetric/AWGN soft-read 채널과 불일치, decoder 복잡도·throughput 수치 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SC-ARA 부호 구성(protograph spatial coupling) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC-LDPC 고정 구조라 새 protograph 계열(ARA) 도입은 재검증 부담 큰 "높음" 난이도 항목 |
| BEC 상 density evolution 폐형식 | 대응 없음 | Prime ECC는 AWGN/RBER/fixed-error 채널 대상이며 BEC DE 이론은 직접 대응 모듈 없음 |
| 조기종료/error floor 대책(rate-1 outer code) | `partial_CRC.cpp`, `full_CRC.cpp` | 개념적으로 조기종료·오류검출과 유사하나 outer code 자체는 별도 구성요소로 이식 시 encoder/decoder 전면 재설계 필요 |

적용 가치: **낮음** — BEC 채널 전용 이론적 부호 구성·DE 분석이며 HW/디코더 알고리즘 개선이 아니어서 NAND binary QC-LDPC decoder/encoder에 직접 이식할 요소가 없음.
