### B. 알고리즘 요약

1. 대상은 SC-LDPC(spatially coupled LDPC) 부호이며 채널은 BEC(binary erasure channel), 평가지표는 BP 반복복호의 iterative decoding threshold다.
2. 문제: 기존 단일 체인 SC-LDPC나 [4]-[6]의 결합 방식은 rate 가변성이 제한적이거나 특정 rate 구간에서 성능이 열화된다.
3. 핵심 가정: `C`개의 서로 다른 rate를 가진 `(J,K,L)` 커플드 체인을 각 위치별로 변수노드/체크노드 edge를 교환(exchange)해 병렬 연결한다(base chain 정의).
4. 핵심 기법: 각 위치 `(j,k)`에서 `m`개 변수노드, `n`개 체크노드를 무작위 선택해 다음 체인 `(j, (k mod C)+1)`과 edge를 교환, PC-MSC-LDPC(Parallelly Connected Multiple SC-LDPC) 앙상블을 구성한다.
5. rate>1/2 앙상블은 `C(J, tJ, L)` 체인들(`t=2,3,...`)을 연결해 설계율 `R_C^up = 1 - (...)·(1/L_up)` 식(2)로, rate<1/2는 `C(J,K,L)`(`tJ=(t-1)K`) 연결로 식(4)를 유도한다.
6. 확장: BEC 상에서 위치별 소거확률 `x_(j,k)^l`(V2C), `y_(i,k)^l`(C2V)에 대해 edge 교환 비율(`n/J_k`, `m/K_k`)을 반영한 density evolution 재귀식(식 5,6)을 유도해 BP threshold를 계산한다.
7. "smoothing" 파라미터 `ω_k`로 각 체인의 edge 연결을 무작위화(랜덤화)하는 `(J,K,L,ω)` 앙상블을 기본 구조로 사용한다.
8. 별도 학습/최적화 절차는 없고, 체인 개수 `C`, 커플링 길이 `L`, base chain의 `t` 값을 바꿔가며 threshold를 수치 계산(최대 반복 1e4, 종료조건 소거확률 1e-6)한다.
9. 결과: 제안 앙상블의 BP threshold는 Shannon limit에 근접하며 단일 체인 대비 소폭 개선, 단 `C≥5`이면 gap이 다시 커지는 경향(교환 edge 비율 감소로 인한 지지력 약화)을 확인했다.
10. 한계: 전 분석이 BEC + density evolution 점근적 이론에 국한되며, AWGN 채널 시뮬레이션·HW 구현·정량적 error-floor 분석이 전혀 없다.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 다중 SC-LDPC 체인 병렬 edge-exchange 구성 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 QC-LDPC(H-matrix 고정) 기반이라 SC-LDPC의 체인 결합 구조 자체가 대응 없음에 가까움 |
| Density evolution 기반 threshold 분석 | 대응 없음 | 점근적 앙상블 이론 분석이며 bit-exact HW 시뮬레이터와 무관 |
| BP(sum-product) 복호 | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum 기반이며 논문은 BEC 전용 BP(소거확률) 갱신식이라 직접 대응은 아님 |

> 적용 가치: 낮음 — SC-LDPC 부호 구성(convolutional-like 체인 결합) 및 BEC 전용 이론 분석이라 QC-LDPC 고정구조의 binary NAND ECC(Prime ECC)에 이식할 요소가 거의 없음.
