### B. 알고리즘 요약

1. DVB-S2 LDPC 부호(length 64800, rate 2/3)는 severe error floor를 보이며(FER 4×10^-3, BER 2×10^-7 부근), sum-product 디코딩·AWGN 채널에서 시뮬레이션.
2. Error floor의 원인은 harmful "trapping set" — `(a,b)` trapping set: a개 variable node로 이루어진 유도 부분그래프에서 홀수 차수 check node가 b개인 집합. a, b가 작을수록 더 해롭다.
3. DVB-S2 부호는 sequence representation `(c_0,...,c_{K-1})`으로 정의되며, 각 `c_j`는 360j번째 열의 nonzero 행 인덱스 벡터; parity part는 dual-diagonal 구조.
4. Trapping set 탐지: dual-diagonal parity 구조 때문에 information part 열의 두 nonzero 원소 차이가 `a-1 mod m`이면 degree-3 variable node 하나를 포함한 길이 `2a` 사이클(=(a,0)/(a,1) trapping set) 발생 — `c_{l1,j} - c_{l2,j} ≡ a-1 (mod m)`, 여기서 `d_j=3`.
5. 확장: degree-3 variable node 2개(또는 3개) 조합의 trapping set은 각 노드에서 나온 edge 경로를 길이 `2a-4`까지 추적해, 세 경로 P1,P2,P3의 총 variable node 수가 `a-2`인 조합을 탐색.
6. 별도 학습/최적화 없음 — 결정론적 그래프 탐색(사이클/경로 추적)으로 dominant trapping set 후보(block index)를 특정.
7. 실측 결과: 원본 부호에서 SNR 3.1dB 기준 프레임 오류의 90.3%가 (9,0) trapping set(block index 23, 68) 기인 확인 → sequence representation의 숫자 1개만 수정(modified code A)해 a≤9 trapping set 제거.
8. 이어서 (10,1) trapping set(block index 28, 54, 80, 프레임 오류의 92.9%)도 동일 방식으로 추가 제거(modified code B) — 총 480개 숫자 중 단 2개만 수정.
9. 결과: modified code B는 FER 10^-5까지 error floor 징후 없음(원본은 FER 4×10^-3, BER 2×10^-7에서 severe error floor).
10. 한계: HW 미설계, 시뮬레이션(순수 이론/구성법 검증)뿐, degree-3 variable node 중심의 특정 trapping set 구조만 다루며 DVB-S2의 dual-diagonal parity·sequence representation에 특화된 탐지 방식.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Dominant trapping set 탐지·제거 (H-matrix 미세 수정) | `ecc_top.cpp` `Load_PCM()` | 특정 QC-LDPC 고정 구조(Prime ECC PCM_b)에 이 탐지법을 재구현하려면 High 난이도지만, error-floor 개선 목적과 직결되어 관심도 높음 |
| Error-floor 관련 trapping set 구조 분석 | 대응 없음 | Prime ECC 코드는 trapping-set 분석/탐지 모듈을 별도 보유하지 않음(코드는 시뮬레이터, 부호 구성 도구 아님) |
| Sum-product 디코딩 시뮬레이션 (min-sum 아님) | `decoder.cpp` `CNU_Update_New_Mag()` | Prime ECC는 이미 min-sum 사용, sum-product 자체는 이식 대상 아님 |

> 적용 가치: 중간 — trapping-set 탐지·제거를 통한 error-floor 개선이라는 목표는 NAND LDPC 부호설계에 직접 관련되나(§5 판정 필터의 "관심 높음" 항목), DVB-S2 특유의 sequence representation 기반 방법이라 Prime ECC의 QC-LDPC PCM 구조에 이식하려면 별도 알고리즘 재설계가 필요.
