> 총평: QC 구조(선형 인코딩, 병렬 디코더)를 갖춘 protograph LDPC를 2-D ISI 자기기록 채널 전용으로 설계한 구성법 논문. 부호 구조는 QC-LDPC이나 채널·검출기 전제가 NAND와 근본적으로 달라 이식 실익 낮음.

### B. 알고리즘 요약

1. 초고밀도(4 Tb/in²) bit-patterned/2-D 자기기록 채널의 2-D ISI를 대상으로, 완전 2-D BCJR 검출기 대비 저복잡도인 IRCSDFA-GA(iterative row-column soft decision feedback with Gaussian approximation) 검출기와 결합되는 protograph 기반 QC-LDPC 부호 설계.
2. 문제: 기존 2-D-ISI 최적화 random-like LDPC[12]는 완전 2-D BCJR 검출기 대비 성능은 좋으나 비구조적(random) 특성상 HW 구현 복잡도가 높고 인코딩 복잡도가 블록 길이에 대해 2차로 증가.
3. 채널은 `L_M × L_N` 2-D ISI 응답 행렬 `H_4T`(4 Tb/in² 밀도)와 AWGN으로 모델링, IRCSDFA-GA는 중심극한정리로 ITI+채널 간섭을 등가 가우시안 잡음으로 근사해 상태공간을 축소.
4. 핵심 기법: 2-D-DET과 결합된 variable node decoder(VND)의 결합 EXIT 함수 `I_E,VND`(식 9, `λ_i`, `J^{-1}` 함수 활용)를 이용해 degree sequence를 최적화.
5. 식(9)의 의미: 2-D-DET의 EXIT 곡선(3차 다항식 근사)과 VND의 결합 mutual information을 기반으로 각 degree별 변수노드 비율 `λ_i`를 결정해 decoding threshold를 최소화.
6. 확장: 최적화된 degree sequence로 protograph base matrix `H_p`를 만들고, QC 제약을 만족하도록 수정된 PEG(progressive edge-growth) 알고리즘으로 girth를 최대화하며 J개 복제본에 대해 순환 시프트 위치를 결정(Algorithm 1).
7. 구현 디테일: 복제 계수 `J`에 따라 J-1개 열은 순환 시프트로 위치 결정(`L^l_j` 재귀식), 첫 열만 local girth 기준으로 PEG 선택 — QC 구조 유지가 핵심 트릭.
8. 학습/최적화 절차: EXIT 차트 기반 degree sequence 최적화(기존 [12] 방식 채택) + 수정 PEG로 QC 제약 하 girth 최대화.
9. 결과: 제안 protograph-QC-LDPC(rate 8/9, length 4608~13824)가 AWGN 최적화 random code 대비 BER 10⁻⁵에서 ∼0.55dB 이득, 2-D-ISI 최적화 unstructured random code 대비 0.05dB 손실에 그침(거의 대등).
10. 한계: 순수 시뮬레이션뿐이며 HW 미설계, IRCSDFA-GA 검출기와의 turbo joint decoding 구조 및 4 Tb/in² 특정 채널 응답 행렬에 종속된 설계.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |

| EXIT 기반 degree sequence 최적화 + 수정 PEG로 QC 제약 부여 | `ecc_top.cpp` `Load_PCM()` | protograph/QC base matrix 설계는 H-matrix 로드 영역과 대응되나 QC-LDPC 고정 구조 재검증 부담 큼 |
| QC 구조 기반 저복잡도 선형 인코더 | `encoder.cpp` `Generate_PCM_encoding()` | QC 순환 시프트 구조 자체는 유사하나 dual-diagonal 고정 인코더와 설계 철학 다름 |
```

적용 가치: **낮음** — 2-D ISI 자기기록 채널 전용 검출기·채널 모델에 종속된 부호 설계이며, QC-LDPC 구조 자체는 Prime ECC와 유사하나 rate 8/9 고정 설계를 특정 채널(4 Tb/in²)에 맞춰 재최적화한 것이라 NAND 채널로 직접 이식할 근거가 부족.
