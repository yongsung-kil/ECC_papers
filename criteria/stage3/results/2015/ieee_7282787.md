### B. 알고리즘 요약

1. 대상은 (dv,dc)=(3,6) regular LDPC, finite-precision min-sum 디코더(sign-magnitude, b비트 양자화, b=4~7이 실무 범위).
2. 문제: 공정 스케일링으로 embedded memory(디코더 내부 메시지 저장 셀)의 신뢰성이 저하되어, 저장된 메시지 비트가 오류(BSC bit-flip, 확률 ρ)를 겪는 상황에서 잔여 오류율을 낮추고자 함.
3. 채널/오류 모델: 통신채널은 AWGN(σ), 하드웨어 오류는 각 메모리 셀에 독립적인 BSC(cross-over probability ρ)로 모델링. check node 메모리는 노이즈 없음으로 가정, variable node 메시지 저장 셀만 오류 대상.
4. 관찰: 기존 density evolution(Balatsoukas-Stimming & Burg) 분석 결과, 반복이 진행되면 메시지 대부분이 큰 매그니튜드로 수렴하고 잔여오류는 주로 부호비트(sign bit) flip에서 발생, 큰 매그니튜드 메시지의 LSB 오류는 잔여오류율에 거의 영향 없음.
5. 핵심 기법: MSB(부호+최상위 매그니튜드 비트)가 1일 때 부호비트를 `(3,1)` 반복부호로 인코딩해 원래 LSB 2비트 저장공간에 나머지 2 중복비트를 저장(추가 메모리 셀 불필요, redundancy-free). MSB가 0이면 원본 그대로 저장.
6. 핵심 식: 노이즈 왜곡을 선형변환 `p^(ℓ) = B·A(ρ)·G·p̂^(ℓ)`(식 4)로 표현, G(인코딩)/B(디코딩) 행렬은 등가클래스(equivalence class) 매핑으로 구성.
7. 이론적 결과(Lemma 1): 통신노이즈·ρ가 충분히 작을 때, robust min-sum의 잔여오류율은 nominal min-sum 잔여오류율의 제곱 오더(`p̆_r,rb ≈ (p̆_r,nm)²`) — sign bit 오류 유발에 최소 2회 bit flip 필요해지기 때문.
8. 확장 가능성: b≥8이면 (7,4) Hamming code로 대체 가능(본문은 분석 단순화를 위해 (3,1)만 다룸), 학습/최적화 절차는 없음(closed-form density evolution만).
9. 결과: DE 예측과 유한장 시뮬(N=2640, (3,6)-LDPC) 모두에서 robust min-sum이 nominal 대비 잔여오류율 크게 감소(Fig.1,4에서 ρ에 대해 제곱 오더 개선 확인), phase transition point도 noise-free 임계값(0.811)에 근접(nominal 0.796 → robust 0.803).
10. 한계: check node 메모리·연산유닛(logic) 오류는 미고려, HW 구현/게이트카운트 없음, 시뮬은 이론 검증 목적의 중간 code length에 그침.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| min-sum 디코더 variable-to-check 메시지 저장/재구성 로직 | `decoder.cpp` `VN_Cal_HD()` 등, `decoder.h` `Get_VNU_*` | 메시지 sign-magnitude 저장 및 갱신 구조에 부호비트 보호 스킴을 추가하려면 이 경로에 인코딩/디코딩 단계 삽입 필요 |
| 메시지 양자화 비트폭(b) 설계 | `ecc_data.h` `PARAM_LLR` | 논문의 (3,1) 반복부호는 sign-magnitude b비트 표현 전제라 Prime ECC의 LLR 양자화 테이블 구조와 정합 필요 |
| 하드웨어 결함(soft error) 모델링 자체 | 대응 없음 | Prime ECC는 NAND read 채널(AWGN/RBER/fixed-error)만 모델링하며 내부 메모리 셀의 BSC 결함 모델은 없음 |

> 적용 가치: 낮음~중간 — 기법 자체(부호비트를 redundancy-free하게 보호)는 decoder.cpp 메시지 경로에 이식 가능하지만, Prime ECC가 대응하는 오류원(NAND read noise)과 이 논문이 대응하는 오류원(디코더 내부 SRAM/레지스터 결함)이 달라 직접 필요성은 낮음.
