### B. 알고리즘 요약

1. 대상: SSD NAND Flash의 BF(bit-flipping) 계열 LDPC 디코더(NBF, GDBF 등 hard-decision 알고리즘), 4kB QC-PEG-LDPC, rate 9/10, column weight 3 또는 5.
2. 문제: BF 계열 디코더는 짝수 개 비트 flip 시 parity가 정상으로 보이는 오검출과, 잘못된 flip이 연쇄적으로 다른 비트에 전파되는 두 가지 고질적 약점이 있어 반복 횟수·실패율 증가.
3. 채널 모델: 4-level MLC를 nine-read AWGN 가우시안 혼합모델로 표현, hard-decision threshold voltage 주변의 `(−q3,−q2)`, `(−q1,q1)`, `(q2,q3)` 3개 구간을 "dominating error region"으로 정의(대부분의 랜덤 비트오류가 여기 분포).
4. 핵심 기법: ADC로 읽은 Vth가 dominating error region 안에 속하는지 comparator(CMP1~3)로 판별해 flag bit 생성, region 밖(hard-decision이 신뢰되는) 비트는 BF 반복 연산(카운팅·최댓값 탐색)에서 제외.
5. 핵심 식: mutual information `I(X;Y)`(식 1, X는 4-level 입력, Y는 9-read 양자화 출력)를 q의 함수로 세우고 `dI/dq=0`으로 최적 q를 유도(MMI 기반 해석적 결정, q∈[0,1]).
6. 확장: DERS는 NBF·GDBF 등 기존 BF 알고리즘 위에 plug-in 형태로 결합 가능(알고리즘 자체 대체가 아닌 전처리 필터).
7. 구현 디테일: q는 Fig.5/6 실험으로 검증된 범위 [0.4, 0.6] 사용, DERS는 comparator 3개 + OR 게이트로 극히 단순한 회로.
8. HW 검증: Verilog로 회로 모델링, Altera Stratix V FPGA 합성(ALM 10개, 레지스터 0개, 기존 BF 디코더 대비 로직 사용량 약 0.03%); throughput은 Synopsys 65nm ASIC 합성 + PrimeTime 타이밍 분석(QC-PEG-LDPC parallel factor 32, 250MHz)으로 post-layout 시뮬레이션.
9. 결과: DERS 적용 시 uncorrectable BER 대폭 감소(특히 column weight 3에서 더 크게 개선), iteration 최대 약 60회 절감(W3, RBER=0.005 부근), decoding throughput 30~48% 향상(clock cycle/iteration이 NBF 15→10, GDBF 24→16으로 감소).
10. 한계: 소규모 column weight(3,5) PEG-LDPC로만 검증, error-floor 영역·대형 NAND ECC 코드(고weight, 다양한 rate)에서의 일반화는 미확인. Table I의 정확한 MB/s 수치는 표에만 있어 확정 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BF/NBF/GDBF hard-decision 반복 루프 | `decoder.cpp` `LDPC_Decoder()`, `VN_Cal_HD()` | DERS는 HD 디코딩 루프에 결합되는 전처리 필터로, Prime ECC의 HD 경로에 직접 적용 가능한 개선 지점 |
| Vth 기반 dominating error region 판별(comparator) | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | soft-read threshold 설정 로직과 유사한 위치에 comparator 기반 flag 생성 로직 추가 가능 |
| 반복 중 검증된 비트 제외로 iteration/clock 절감 | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | 조기종료(Partial CRC)와 별개로 비트 단위 skip을 통한 반복당 연산량 절감 아이디어 |
| 짝수 flip/오류전파 억제 | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | min-sum/CN 갱신 자체는 아니나, HD 디코더의 flip 대상 축소라는 점에서 관련 |

> 적용 가치: 높음 - NAND SSD 직접 타깃, hard-decision 디코더에 매우 저비용(게이트 0.03%)으로 결합 가능한 성능/latency 개선 기법이며 FPGA/ASIC 합성까지 검증됨.
