### B. 알고리즘 요약

1. 대상은 protograph LDPC를 정의하는 base matrix `θ`(m×m 비음행렬)와 그 M-graph cover에 해당하는 P-lifting `θ↑P` (P는 M×M 순열행렬 블록).
2. 풀려는 문제: Vontobel의 추측 `perm(θ↑P) ≤ perm(θ)^M`과 degree-M Bethe permanent `permM,B(θ) ≤ perm(θ)`를 조합론적으로 증명하는 것.
3. permanent-product `Aτ = Π (θ↑P)_{iτ(i)}`를 exponent matrix `Rτ`(행/열 합이 M인 비음정수 행렬)로 환원하는 decomposition algorithm 정의.
4. Birkhoff-von Neumann 정리로 `(1/M)Rτ`를 순열행렬들의 볼록결합으로 분해 → `Aτ`를 base `θ`의 permanent-product 거듭제곱들의 곱으로 재작성(standard decomposition).
5. same-index decomposition: `θ↑P`의 첫 M열 boxed 엔트리를 같은 행 index끼리 묶어 M개 permanent-product 곱으로 유일 분해.
6. illegal sub-product(θ의 permanent-product가 아닌 것)를 legal한 것으로 유일 매핑(standard mapping, Lemma 1) → `θ↑P` permanent-product와 M개 θ-permanent-product 곱 사이 일대일 대응 성립.
7. 고정 exponent matrix R에 대한 계수를 multinomial 계수 `M!/Πt_{j,σ}!`(Lemma 2)로 상한 계산.
8. 이를 `perm(θ)^M`의 다항 전개와 비교하여 주정리 `perm(θ↑P) ≤ perm(θ)^M` 도출(Theorem 2).
9. 결과: base matrix에 대한 Bethe permanent 부등식 `permB(θ) ≤ perm(θ)`의 조합론 기반 대체 증명(corollary), P-lifting permanent 계산 복잡도 감소 알고리즘 가능성 시사.
10. 한계: 순수 이론 논문 — 구체 부호설계·복호기·HW 없음, LDPC 성능과의 연결은 "may help explain" 수준의 정성적 시사뿐.
