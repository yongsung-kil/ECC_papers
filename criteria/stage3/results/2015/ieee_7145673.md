### B. 알고리즘 요약 (10줄 내외)

1. 대상은 regular binary QC-LDPC 코드(예: (16213,14124) rate 0.8712, (31903,29814) rate 0.9345, (2407,2078) rate 0.8633, (2656,1328) rate 0.5, (4380,4091) rate 0.9340, (5256,2629) rate 0.5002), AWGN 채널, BPSK, decoding은 SPA(Sum-Product) 최대 50 iteration.
2. 기존 base matrix + dispersing/masking 구성법은 4-cycle-free를 보장하지만 무작위(MacKay)식 구성 대비 error-floor 발생 문제를 겪는데, 본 논문은 이를 개선할 대수적 구성법 2종을 제안.
3. 핵심 가정: 소수체 `GF(p)`의 곱셈군은 순환군(cyclic)이며, 원소 `i`를 p-tuple location-vector `v(i)`(cyclic shift)로 임베딩해 additive p-fold matrix dispersion으로 `p×p` CPM(circulant permutation matrix)을 얻음.
4. 구성법1(§II): 순환군의 생성원(generator) 집합 `{l1,...,l_K}`(`K=(p-1)∏(1-1/p_i)`, Euler 공식)로 `(K+1)×(K+1)` base matrix `W(1)=[l_i l_j mod p]`를 만들고, `W(1)`이 additive row constraint 1·2를 만족함을 증명(Lemma 1)해 RC-constraint(4-cycle-free) 보장.
5. `W(1)`을 additive p-fold dispersion하여 `H(1)` 배열 생성 후 `γ×ρ` 부분배열을 취하면 length `ρp`, rate `≥(ρ-γ)/ρ`인 QC-LDPC 코드; 최소거리 `≥γ+1`(γ 홀수) 또는 `γ+2`(짝수).
6. 구성법2(§III): 두 생성원 `l_μ, l_ν`의 곱으로 `p×p` base matrix `W(2)=[l_μi·l_νj mod p]`를 구성(Lemma 2로 constraint 증명), girth ≥6까지 보장하는 확장.
7. masking 기법(`M = Z⊗H(γ,ρ)`, `Z`는 0/1 마스킹 행렬)을 병용해 파리티체크 행렬을 더 희소화하고 rate 0.5 근처 코드도 구성 가능.
8. 별도 학습/최적화 절차 없음(순수 대수적 구성, Euler 공식과 순환군 이론 기반).
9. 결과: (16213,14124) rate 0.8712 코드는 BER `5×10^-7`에서 MacKay 코드와 유사하나 저BER 영역서 error-floor 없이 우세; (31903,29814) rate 0.9345 코드는 BER `10^-6`에서 Shannon limit 대비 0.665dB; (4380,4091)은 iteration 50→5로 줄여도 BER `10^-6`서 0.36dB 열화에 그쳐 빠른 수렴 입증.
10. 한계: 모두 소프트 SPA 디코더 시뮬레이션(하드웨어 미설계, gatecount/throughput 실측 없음)이며 부호 파라미터(rate/length)가 예제마다 달라 단일 대표값으로 확정 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 대수적 base matrix(GF(p) 생성원) + CPM dispersion 구성법 | `ecc_top.cpp` `Load_PCM()` | H-matrix/부호구조 생성 방식이 다르지만 4-cycle-free QC-LDPC 구성 아이디어는 신규 H-matrix 설계 시 참고 가능 |
| masking 기법(sparser PCM으로 error-floor 억제) | `ecc_top.cpp` `Load_PCM()` | 부호 rate 조정·희소화 기법으로 신규 부호 설계 시 참고 가능 |
| SPA 기반 빠른 수렴(iteration 감소) 결과 | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum이라 SPA와 다르지만, 부호 구조 자체가 빠른 수렴을 유도한다는 통찰은 참고 가능 |

> 적용 가치: 중간 — 대수적 QC-LDPC 구성법은 특정 소수체 기반 H-matrix를 요구해 Prime ECC의 고정 QC-LDPC 구조(z=32, N_b=129)에 직접 이식은 재검증 부담이 크지만, error-floor 억제·빠른 수렴 부호설계 아이디어는 신규 부호 개발 시 참고할 만함.
