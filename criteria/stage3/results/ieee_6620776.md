### B. 알고리즘 요약

1. 대상은 임의의 binary cycle code(패리티검사행렬 모든 열의 Hamming weight가 2인 LDPC), 예제는 `[9,4,3]` 코드 `C9`.
2. 문제: 유한길이 MPI/SPA 복호의 동작(pseudo-codeword)을 해석적으로 특성화하고 싶다.
3. 채널은 memoryless binary-input, LLR `γ_i = log W(y_i|0)/W(y_i|1)`, 모든 성분 유한 가정.
4. 핵심: normal factor graph(NFG) `N(H)`의 Bethe free energy function `F_B(ω)`를 특성화하고 SPA의 전역 수렴을 증명.
5. SPA 메시지 갱신 연산자 `S_γ`를 max-norm 하에서 strict contraction으로 보임(Theorem 10) - check node degree ≥ 3 가정.
6. 유도 Bethe entropy는 concave(Thm 6), 유도 Bethe free energy는 convex(Thm 7)임을 Hessian 분석으로 증명.
7. 행렬 `Z_γ = M·U|_{U_i=exp(-γ_i)}`의 spectral radius `ρ_γ`로 최소점 조건 판정(Cor 9): `ρ_γ<1`이면 all-zero가 유일 최소.
8. Theorem 12: SPA가 all-zero codeword로 수렴 ⟺ `ρ_γ < 1`.
9. 결과: 무한반복 SPA = symbolwise graph-cover decoding(Cor 14), 수렴영역 = edge zeta function `ζ_N(U)`의 수렴반경(Cor 15). 수치실험 없음.
10. 한계: 최소거리 낮은 cycle code 전용 이론, HW/시뮬 전무, 일반 LDPC 확장은 지면상 생략.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SPA(sum-product) 반복 복호 수렴 분석 | [decoder.cpp](../Prime_ECC_3.1_Claude/) `LDPC_Decoder()` | Prime ECC는 min-sum 근사 사용 - full tanh SPA 이론이라 직접 대응 약함 |
| edge zeta / graph-cover pseudo-codeword 이론 | 대응 없음 | 코드베이스에 해당 분석 모듈 없음, error-floor 진단에 간접 참고 정도 |
| cycle code(column weight 2) 구조 | 대응 없음 | Prime ECC는 VN degree 17 고rate QC-LDPC로 구조 상이 |

적용 가치: 낮음 - column weight 2 cycle code의 tanh-SPA 수렴을 다룬 순수 이론이라 min-sum 기반 고rate NAND QC-LDPC 엔진에 이식할 알고리즘/HW 요소가 없다.
