### B. 알고리즘 요약

1. 시스템: 2Y-nm(20-24nm) 2-bit MLC NAND flash, read-retry 기능으로 read reference voltage를 미세 조정해 Vth 분포를 실측 특성화(erase state 제외 P1/P2/P3).
2. 문제: BCH hard-decision ECC의 정정능력 한계 → LDPC 같은 soft-decoding 도입 필요, 그러려면 실측 기반 Vth 분포 모델이 필수인데 공개 문헌에 없음.
3. 방법론: FPGA 기반 테스트 플랫폼에 Set REF/Get REF 컨트롤러 명령 추가, REF를 bin 단위로 스캔해 각 programmed state의 Vth 히스토그램 수집.
4. 밀도추정: 비모수(Gaussian kernel density estimation, 식1)와 모수(MLE, 식2) 접근 비교 - 5개 분포(Gaussian/Beta/Gamma/Lognormal/Weibull) RMSE 적합도 평가.
5. 결과1(적합도): 평균적으로 Beta가 최적이나 Gaussian이 세 state에 고르게 적합(RMSE 약 22%) → 실용상 Gaussian 근사 채택(closed-form이라 MSP/LDPC soft-info 산출에 유리).
6. 결과2(독립성): 서로 다른 위치 셀의 Vth는 상관 <0.05로 거의 독립, 동일 셀의 과거 P/E age와는 상관 → ideal wear-leveling 하 memoryless.
7. 채널모델: 셀을 `Vth = Vth_signal + Vth_noise`로 분해, program noise의 PSD 평탄·auto-correlation 0-lag 스파이크 → **white noise** → ideal wear-leveling 시 **AWGN 채널**로 모델링.
8. P/E 노이즈 모델: ISPP over-programming으로 분포가 우측 이동·확산, 평균/표준편차를 `A + B·e^(C·PEcycle)` 지수식(식4)으로 모델링 - 정확도 96.5~98.1%(45k cycle까지).
9. 결과(SNR): 2-bit MLC를 4-state PAM으로 보면 SNR이 P/E cycle에 선형 감소(3k→45k에서 11dB→5.5dB, 1k당 약 0.13dB) → P/E 적응형 ECC 강도 조절 근거.
10. 한계: 부호/디코더/HW 설계는 명시적으로 범위 밖(future work), 특정 2Y-nm 칩 실측이라 다른 공정 일반화 미검증, erase state는 음전압 제약으로 미측정.
