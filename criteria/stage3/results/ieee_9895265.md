### B. 알고리즘 요약

1. 시스템: m차원 SPC product code (예제 3차원 `(125,64)`, rate≈0.51), BMS 채널(BEC, B-AWGN) 대상 이론 분석.
2. 문제: SPC product code에 대한 BP 복호는 단블록·저오율 영역에서 ML과 gap이 크다.
3. 핵심관찰: SPC product code = frozen bit 제약을 준 multi-kernel polar code로 표현 가능(kernel `Kn` 식11, frozen 집합 `A_PC` 식13).
4. 핵심기법1(SC): polar식 successive cancellation 스케줄로 bit likelihood를 재귀 근사(식17), Elias 디코더는 병렬 근사(식21)로 대비.
5. 핵심식: `P_SC ≤ Σ P(B^GA_SC,i)` union bound(식19), SC ≤ Elias 오율 증명.
6. 핵심기법2(SCL): list size `L`로 경로 확장, `L=4`로 TSB 근접, `L=8`로 ML 하한 근접.
7. 트릭: 외부 고rate 부호(CRC) 연접 후 list에서 검사해 최종 선택 → 거리분포 개선.
8. 분석: joint weight enumerator 없이 IOWEF 계산법 제시(Thm3/4), uniform interleaver 앙상블 평균 weight로 Poltyrev TSB 산출.
9. 결과(시뮬): B-AWGN에서 SCL(L=4)이 BP(100 iter) 능가, CRC 연접 시 BLER 10^-4에서 원 product 대비 +1.5dB, RCU와 gap≈0.7dB, (128,64) 5G-NR LDPC(BP) 대비 +0.6dB.
10. 한계: 순수 이론·시뮬(HW 없음), 초단블록 전용, non-LDPC(product+polar) 부호이며 NAND soft-read/error-floor 논의 없음.
