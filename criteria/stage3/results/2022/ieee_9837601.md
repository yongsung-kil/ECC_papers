### B. 알고리즘 요약 (10줄 내외)

1. 대상: rate-7/10, `N=7650`, `dv=3`, `dc=10`, lifting `Z=765` regular QC-LDPC, BPSK AWGN 채널, layered 스케줄, `imax=10`.
2. 문제: IB(Information Bottleneck) LUT 디코더는 min-sum보다 정정능력이 좋지만, LUT 크기가 메시지폭 `w`·노드차수에 지수적으로 커져 HW 설계 가이드가 부재.
3. 기법1(LUT 분해): `dc`입력 CN LUT(`2^{w(dc-1)}` 엔트리)를 `dc-2`개의 2입력 캐스케이드 LUT `Φ^c_{i,l,k}(t_k, y_{k+1})`로 분해해 복잡도를 `2^{2w}·(dc-2)`로 축소. VN도 동일.
4. 핵심식: `t_{k+1}=Φ^c_{i,l,k}(t_k,y_{k+1})`, `t_0:=y_0`. 각 LUT는 상호정보 `I(X_{k+1};T_{k+1})` 최대화로 설계(이산 DE + IB).
5. min-LUT 변형: CN에서 최소근사 사용 `Φ^c=sgn(t_k)sgn(y_{k+1})min(|t_k|,|y_{k+1}|)` — min-sum과 유사하나 IB로 양자화.
6. 대칭성 활용: 채널 대칭으로 부호비트 무시한 magnitude만 저장 → CN `(dc-2)·2^{2(w-1)}`, VN `(dv-1)·2^{2w-1}` 엔트리로 추가 축소, sign은 DeMorgan 논리로 복원.
7. HW 구현: 2계층 메모리 — 중앙 ROM(BRAM)에 `dv·imax` LUT 세트 저장, 노드별 RAM(LUTRAM)에 현재 layer/iter 세트만 캐시, layer마다 broadcast remapping.
8. 최적화: layer 간 LUT 값 변화가 작다는 관찰로 base-delta(B+Δ) 압축 적용, nonzero delta만 기록해 RAM write cycle 단축(3bit는 remapping이 message-passing에 은닉).
9. 결과: FPGA(ZCU104)에서 3bit min-LUT가 4bit OMS 대비 24.2% 면적절감·1.39x TAR, 성능손실 ≤0.08dB; 정규화 throughput 최대 13.45 Gbps/iter, 실효 1.345 Gbps@10iter.
10. 한계: 단일 rate-7/10 코드만, 통신(AWGN) 타깃, gate count 없음, remapping stall 존재, dv=3/dc=10로 NAND 고rate(dv 큼) 대비 차수 작음.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| IB-LUT 기반 CN/VN 노드 연산(디코딩 알고리즘 전체) | decoder.cpp LDPC_Decoder(), C2V_Cal() | min-sum 산술을 LUT로 대체 — 우리 CNU/VNU를 통째로 교체하는 급진적 변경 |
| min-LUT의 min 근사(CN) | decoder.cpp CNU_Update_New_Mag(), C2V_Cal_New_Sgn() | 우리 min-sum과 등가 형태, IB 양자화만 다름 |
| VN 처리 / 양자화(2~3bit LUT) | decoder.cpp VN_Cal_HD() 등, decoder.h Get_VNU_* | soft 2/3bit 지원 범위와 정합, LUT 매핑값 주입 필요 |
| iteration/layer별 LUT 매핑 세트 | ecc_data.h PARAM_LLR, decoder.cpp Get_VNU_Table_Idx() | 우리 iteration별 LLR 테이블과 개념 유사(테이블 인덱싱) |
| 2계층 ROM/RAM + B+Δ remapping 인프라 | 대응 없음 | 우리 구조엔 런타임 LUT 재구성 메모리 계층 없음 |
```

적용 가치: **낮음** — min-LUT은 우리가 이미 가진 min-sum과 성능 격차가 ≤0.13dB로 작고, IB-LUT의 이득을 살리려면 iteration/layer별 LUT 세트 생성·2계층 재구성 메모리라는 무거운 인프라를 통째로 이식해야 하므로 고rate NAND ECC 관점에서 비용 대비 효과가 낮다.
