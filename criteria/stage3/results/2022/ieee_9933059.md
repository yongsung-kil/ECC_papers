### B. 알고리즘 요약

1. 대상: 3D TLC charge-trap NAND flash, NplusT raw chip tester로 read disturb(RD)와 endurance를 측정. LDPC 부호 스펙(rate/length)은 논문에 없음.
2. 문제: 기존 수명예측은 대부분 ML 기반이라 대량 테스트·다수 파라미터(동작시간, Vth 등)가 필요 → 더 단순한 예측 모델 요구.
3. 핵심 관찰: 조기단계 FBC(`FBC1r00`=100회 읽기 시 FBC, `FBC1p0`=10 P/E cycle 후 FBC)와 후기단계 FBC 사이에 근사 선형관계 존재.
4. 기법 1: `FBC_kr = S(k)*FBC1r00 + I(k)`, `FBC_np = H(n)*FBC1p0 + P(n)` — slope/intercept를 read time·P/E cycle의 함수로 표현.
5. 기법 2: slope/intercept를 지수합으로 피팅 `S(k)=Σ a_i·exp(k·b_i)` 등, 상수 `a_i,b_i,c_i,d_i`는 칩·환경 종속.
6. 최종형: 두 시점 사이 증분식 (식 3,4)으로 단기예측 정확도 향상 (short-term > long-term).
7. 응용: STWS(short-term warning system)가 주기적으로 FBC 예측 후 "가장 적합한 LDPC 코드"를 미래 encoding/decoding용으로 선택.
8. 결과: RD·endurance 예측 모델 정확도 손실 10% 이하가 각각 99.16%, 97.63% 샘플에서 달성.
9. 목적: 초기 PE 단계에 사용자 데이터 공간 절약 + NAND 수명 연장.
10. 한계: WIP 2쪽 논문, LDPC 부호/디코더 설계 없음, "LDPC 코드 선택"은 개념적 언급뿐 구체 코드셋·전환 메커니즘 미제시.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| FBC 단기 수명예측 모델 (선형·지수피팅) | 대응 없음 | 컨트롤러 수명관리 로직, ECC 코어 외부 |
| STWS 기반 LDPC 코드 동적 선택 | 대응 없음 | rate-adaptive 코드 스위칭은 상위 SSD 정책, 부호설계/디코더 아님 |
| RD·endurance 실측 특성화 | [channel.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) (RBER 주입) | 간접: 에러율 모델링과만 느슨히 연관, 이식 대상 아님 |

적용 가치: **낮음** — LDPC 부호설계·디코더·HW 어느 레이어도 다루지 않는 NAND 수명예측/코드선택 정책 논문이라 Prime ECC 3.1에 이식할 요소가 없음.
