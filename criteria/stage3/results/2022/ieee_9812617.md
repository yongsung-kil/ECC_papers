### B. 알고리즘 요약

1. 대상은 고속 optical transport network(100Gbit/s+, BER 1e-15)용 FEC, rate 0.9372, binary primitive BCH를 성분부호로 쓰는 spatially coupled product code.
2. 문제: 기존 staircase code는 약한 BCH 성분부호로 miscorrection 확률이 높아 waterfall·error-floor 성능 한계.
3. 핵심 기법(SR-staircase): 선행 code block `B_{i-1}`을 `q`개 등크기 sub-block으로 분해→전치(transpose)→재배열해 message matrix를 구성, code block 크기를 conventional staircase의 `1/q`로 축소(식 1~5).
4. 핵심 효과: block 크기를 유지하면서 더 강한(큰 `t`) BCH 성분부호를 쓸 수 있어 같은 rate·block size로 정정능력 향상; coupling width `w`로 추가 개선.
5. 핵심 식: code rate `R = 1 - (1/2)(ν1 t1/m1 + ν2 t2/m2)` (식 7), 성분코드 파라미터로 rate 결정.
6. 복호: sliding window iBDD(iterative bounded distance decoding, algebraic hard-decision) - 비-반복 message passing, BSC/AWGN hard-decision 입력.
7. 임계 분석: deterministic 구조에 density evolution 적용, effective channel quality `M̄`과 BSC threshold `p̄=M̄/2m` 도출(Table I~II), staircase보다 큰 threshold 조건(Theorem 1) 제시.
8. Error floor 분석: minimum stall pattern 크기 `smin`과 multiplicity `Amin`을 union bound로 유도(식 18~30), 큰 `w`가 error floor 낮춤(Theorem 2/3).
9. 결과: AWGN 시뮬(최대 10 iteration)에서 제안 부호가 벤치마크 staircase(SC1/SC2) 대비 waterfall·error-floor 모두 우수, `w=5` 코드가 miscorrection-free에 근접.
10. 한계: HW 미설계, hard-decision iBDD 전용, LDPC/soft-decision 미적용, optical 타깃(향후 NAND용 outer code는 별개 [46]).

### C. Prime ECC 관련 모듈 핀포인트

대상이 code-design이며 복호가 algebraic iBDD(비 message-passing)라 C섹션(디코더 성능)은 전부 `N/A`. 참고로 부호구조/복호 계열이 Prime ECC와 상이해 모듈 대응은 아래와 같이 제한적.

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SR-staircase 부호구조 (product/staircase) | 대응 없음 | Prime ECC는 binary QC-LDPC(base+circulant), staircase product 아님 |
| iBDD(algebraic BDD) 복호 | 대응 없음 | Prime ECC는 min-sum message passing, algebraic BDD 미지원 |
| BCH 성분부호 인코딩 | [encoder.cpp](../Prime_ECC_3.1_Claude/) `LDPC_encoder_4KB()` | 인코더 모듈은 존재하나 dual-diagonal LDPC 전용, BCH 성분부호 아님 |
| DE 기반 threshold 설계 | 대응 없음 | 부호설계 이론 도구, 코드베이스 무관 |

적용 가치: **낮음** - staircase/product BCH + iBDD hard-decision 계열로 Prime ECC의 binary QC-LDPC + min-sum과 부호·복호 계열이 근본적으로 달라 직접 이식 요소 없음. NAND용 outer/concatenation 관점은 별도 검토 대상.
