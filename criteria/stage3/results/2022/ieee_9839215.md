### B. 알고리즘 요약

1. 대상: AWGN 채널, rate 4/9 부호. 대형 `(n,k)` 부호를 소형 성분 `(n1,k1),(n2,k2)`로 분해(`n=n1n2`, `k=k1k2`), 예 `(15,10)²→(225,100)`, `(21,14)²→(441,196)`.
2. 문제: 채널부호의 차원 문제(`2^k` 코드워드)로 `k>100` 대형 신경부호는 직접 학습이 사실상 불가.
3. 모델: 실수값 코드워드, 전력정규화 `c=√n·c/||c||₂`로 평균전력 1, `SNR=1/σ²`.
4. 핵심 기법: 2D product code의 각 행/열 encoder를 별개 FCNN `E1(Φ1),E2(Φ2)`로, `I`회 반복 복호의 각 pair를 별개 FCNN 쌍으로 대체해 총 `2I`개 NN 디코더 구성.
5. 핵심 식: 성분 파라미터의 곱 `X=∏Xm (X∈{n,k,d,R})`, `G=G1⊗G2⊗…` — 작은 부호로 큰 부호를 얻는 product 구조가 학습 복잡도를 관리 가능하게 함.
6. 확장: 임의 `M`차원 product로 일반화(2D는 복잡도-성능 절충).
7. 구현 트릭: 채널출력 injection, feature size `F`(다중 soft-info 추정), soft-info 차분 입력 — 세 가지 병용이 학습 성능 크게 개선.
8. 학습: encoder/decoder 교대 스케줄(`Tenc=100`, `Tdec=500`), Adam, SELU, BCEWithLogitsLoss, 대형 배치(`B≥5000`)+gradient accumulation, decoder는 SNR 범위 `[γ-2.5,γ+1]`.
9. 결과: `(15,10)²` 및 `(21,14)²`가 동일 길이 polar(SC)를 전 SNR에서 BER로 능가, TurboAE(600,200) 대비 BER 10⁻⁶에서 0.5dB 이득, 고전 LDPC/TBCC/polar도 상회.
10. 한계: HW 미설계, 시뮬만, `k≤196` 규모만 검증, 비이진 비선형 실수 부호.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| FCNN 신경 encoder/decoder 전체 | 대응 없음 | NN 디코더는 Prime의 min-sum 파이프라인 밖 (프로파일 §6 "대응 없음") |
| product code(Kronecker) 부호구조 | 대응 없음 | Prime은 단일 QC-LDPC, non-binary 실수부호와 무관 |
| 반복 복호 스케줄(2I NN 반복) | decoder.cpp LDPC_Decoder() 이터레이션 루프 | 반복 개념만 유사, 알고리즘 실체 상이 |
| soft-info 차분/feature 처리 | decoder.cpp C2V_Cal() (메시지 패싱) | 개념적 유사, NN 가중치라 이식 불가 |
```

적용 가치: **낮음** — 비이진 비선형 FCNN 부호로 min-sum 기반 binary QC-LDPC NAND ECC(부호설계→디코더→HW)와 근본적으로 이질적이며, HW·양자화도 미설계다.
