### B. 알고리즘 요약

1. 시스템: 2-user Gaussian MAC(GMAC), 수신신호 `y_j = √P1·x_j[1] + √P2·x_j[2] + n_j`, BPSK, equal/unequal power.
2. 문제: 기존 최적화된 irregular LDPC는 HW 구현 복잡·error floor 높음 → 저복잡 protograph 부호를 GMAC용으로 설계 필요.
3. 모델: 두 사용자 protograph를 state node로 연결한 joint protograph, joint BP로 4종 메시지(`m_cv,m_vc,m_sv,m_vs`) 교환.
4. 핵심기법1: state-to-variable 메시지 `m_sv[k]`의 unequal/equal-power 갱신식 유도, 밀도를 대칭 Gaussian으로 근사.
5. 핵심 식: `J(σ_m)` 상호정보 함수 기반으로 joint protograph 엣지를 따라 MI를 추적하는 JP-EXIT chart 도출(식 9~14).
6. 핵심기법2: JP-EXIT로 decoding threshold(수렴 최소 SNR) 예측 → 두 사용자 base matrix `B[1],B[2]`를 각 rate별로 설계.
7. 설계규칙: puncture node 회피, degree-2 VN 비율 제어(waterfall↓ but error floor 위험), stability condition (15) 만족, VN degree 초기 3으로 최소거리 선형성장 유지.
8. 최적화: 설계규칙으로 탐색공간 축소 후 JP-EXIT 가이드 단순 탐색으로 최저 threshold base matrix 선택.
9. 결과: 제안 부호가 최적화 irregular LDPC 대비 equal-power에서 0.5dB, unequal-power에서 0.4dB BER 이득(L=15000/10000, iter=200), sum-rate가 용량 경계에 근접.
10. 한계: 2-user GMAC 전용, HW 미설계, 시뮬레이션 수준, 단일사용자 NAND 채널 미대응.
