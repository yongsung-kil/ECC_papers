### B. 알고리즘 요약

1. 시스템: 2-user unsourced binary adder channel(UBAC) - 두 사용자가 같은 codebook 사용, 채널출력 `Yi = X1,i + X2,i`(실수 덧셈), IoT 랜덤액세스 대상. 예제 부호 `n=800, R=0.35`.
2. 문제: `Yi=1`은 두 입력 애매 -> erasure로 처리. erasure 패턴이 codeword support와 일치해 stopping set 형성, 표준 BEC threshold 기반 DE 설계가 안 통함.
3. 모델: erasure 집합 `E=S1={j:yj=1}`, plain BP는 반드시 실패 -> pivot(erasure VN 하나 값 고정) 필요.
4. 핵심 기법: pivot decoding + good-pivot 분석. irregular ensemble `(L,R)` degree 분포에서 induced graph의 degree-2 CN 부분그래프에 Molloy-Reed 기준 적용해 good pivot 비율 `#D` 계산.
5. 핵심 식: `Rind(x)=R((1+x)/2)+R((1-x)/2)` (induced CN 분포), `fλ(x)=x−λ(1−ρind(1−x))<0` 만족 시 `#D>0` -> DE로 erasure 소멸 보장.
6. 확장: `#D` 분석과 DE 사이 연결 발견 - 이 기준으로 degree 분포 최적화, regular 한계 R=1/4를 irregular로 R=1/2(선형부호 최적)까지 확장.
7. 최적화: 고정 `ρind`에 대해 `fλ<0`가 λ에 선형 -> linear program으로 design rate 최대화 λ 탐색, ρ도 대칭 LP, 교대최적화. Table I에 degree 분포 예.
8. AWGN: BPSK AWGN adder MAC에서 noise가 대칭 깨서 오히려 도움 - interference cancellation + pivoting 구조. 10~15dB에서 12dB 부근 error 최소(반직관적).
9. 결과: multi-edge type LDPC 구성 (UBAC용 C1 + BEC capacity-achieving C2, `n2−k2=k1`)으로 R=1/2 한계 증명적 달성. 시뮬 BLER(n=800,R=0.35) 제시.
10. 한계: 순수 이론/시뮬, HW 미설계, 2-user 간섭채널 특화라 NAND single-user 채널로 전이 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| pivot decoding / good-pivot 분석 | 대응 없음 | UBAC 간섭채널 전용, single-user AWGN/RBAC 디코더에 무의미 |
| irregular degree 분포 LP 최적화 | 대응 없음 | Prime ECC는 QC-LDPC 고정 H-matrix, 부호설계 재검증 부담 큼 |
| multi-edge type 부호구성 | 대응 없음 | NAND용 dual-diagonal/QC 구조와 무관 |

적용 가치: **낮음** - 2-user 간섭채널(BAC) 특화 정보이론이라 NAND single-user binary QC-LDPC ECC에 떼어 쓸 요소가 없음.
