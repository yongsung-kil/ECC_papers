### B. 알고리즘 요약 (10줄 내외)

1. 시스템: 2D 자기기록(TDMR) 채널, Voronoi 매체 모델(픽셀 nm 단위), rate-8/9 modulation code + LDPC(rate `0.9`) 결합, 4개 트랙 동시 기록.
2. 문제: 초고밀도에서 ISI/ITI 등 2D 간섭으로 신호 열화, 기존 Viterbi 기반 검출기로는 개별 신호 검출조차 곤란.
3. 채널 모델: readback 신호 = grain magnetization과 read-head sensitivity 함수의 2D convolution + AWGN(평균0, 분산 σ²).
4. 핵심 기법: MLP 기반 검출기가 readback 시퀀스를 입력받아 soft-output LLR `λ_i,k`를 생성 (Viterbi 대체).
5. 구조: input layer(벡터화) + FC unit 3개(ReLU) + output unit(softmax + cross-entropy), 은닉노드 수는 본문 손상.
6. 확장(2-pass turbo): 2차 pass에서 MLP가 LDPC 디코더가 만든 LLR을 추가 입력받아 검출기-디코더 간 정보 교환(turbo).
7. 구현 디테일: 인접 트랙(complementary) readback 시퀀스를 함께 입력, rate-modulation code로 고품질 인접신호 확보.
8. 학습: Adam optimizer, 학습률·epsilon 지정, mini-batch/epoch 설정, 데이터 셔플로 분산 감소.
9. 결과: SNR 특정 지점에서 제안 시스템이 BER 10^-x로 수렴, Viterbi 기반 Conv/Coded 시스템 대비 우수, pass가 진행될수록 개선.
10. 한계: 시뮬레이션만(Voronoi 모델), HW 미설계, LDPC 자체 개선 없음, 자기기록 특화 채널로 NAND 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| MLP 기반 soft-output 검출기 | 대응 없음 | NN 검출기 - Prime ECC에 대응 모듈 없음 |
| turbo 정보교환(검출기↔LDPC) | 대응 없음 | LDPC 외부 turbo 루프, Prime ECC는 standalone 복호기 |
| LDPC 복호기(외부부호) | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `LDPC_Decoder()` | 논문이 LDPC 내부를 건드리지 않아 이식할 것 없음 |

적용 가치: **낮음** — 2D 자기기록 채널의 MLP 검출기 논문으로 LDPC는 표준 외부부호에 불과, NAND binary LDPC 부호설계/디코더에 이식할 요소가 없다.
