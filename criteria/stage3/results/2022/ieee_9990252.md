### B. 알고리즘 요약

1. 시스템: SISO-OFDM, 256-QAM Gray mapping, 128 sub-carrier, MATLAB 2019 시뮬. LDPC는 trellis shaping 뒤에 concatenate.
2. 문제: OFDM의 높은 PAPR(`PAPR(x)=max|xk|^2 / E|xk|^2`)를 줄이면서 BER도 개선.
3. 채널/모델: shaping convolutional code `Cs` (rate `1/ns=1/2`), 심볼당 MSB=shaping, LSB=passthrough.
4. 핵심 기법 1: trellis shaping depth(constraint length)를 4→8→16→32→64 states로 키워 PAPR 감소 개선.
5. 의미: `z = s(H^-1)^T` 로 MSB 인코딩 후 codeword `y`와 modulo-2 합, syndrome former로 `(z⊕y)H^T=s` 복원 — shaping gain은 trellis depth에 비례(Forney).
6. 핵심 기법 2: irregular LDPC를 concatenate, `λ(x)`, `ρ(x)` degree 분포 다항식으로 표현.
7. 구현 디테일: QAM 심볼 내 각 비트의 에러확률(union-bound 근사)을 이용해 variable node degree 분포 최적화 (Table-1).
8. 학습/최적화: shaping code별 비트 에러확률로 VN degree 분포 최적화 (수치 최적화, NN 없음).
9. 결과: window `∆=32`에서 32-state가 4-state 대비 PAPR 1.2 dB 개선(8-state 0.6, 16-state 0.8), BER 페널티 0.4 dB, irregular가 (3,21) regular 대비 우수.
10. 한계: OFDM/QAM 특화, HW 미설계, rate/length 미명시, hard decision 복호, 시뮬레이션만.
