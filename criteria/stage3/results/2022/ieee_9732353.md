### B. 알고리즘 요약

1. 대상은 protograph 기반 SC-LDPC(base matrix `B`, coupling length `L`, coupling width `w`, lifting `M`), window decoder(window size `W`)로 복호. 예제 rate 1/2·3/4, `L=100`, `M=1000`.
2. 문제: 기존 WD 코드설계는 asymptotic 지표(WD threshold)만 최적화하나, 유한길이 성능을 충분히 설명 못함(비슷한 threshold라도 성능 상이).
3. 모델: BEC 소실확률 `ε` 중심, DE로 CN/VN 소실확률 추적(식(2)~(5)), WD schedule 하 window configuration에서 갱신.
4. 핵심기법: BP mean parameter `γBP`의 WD 대응물인 window mean parameter `γWIN(W)` 제안 - degree-1 CN 정규화 개수 `r1(ℓ)`의 최소값(FER 곡선 falling slope와 직결).
5. 핵심식: `γWIN(W)`가 threshold로 설명 안 되는 유한길이 우열(특히 slope)을 예측 - threshold 동일해도 성능차 해명.
6. 확장: 단일목적함수화한 수정지표 `γ̄WIN(W)`(고정 계산점 `εCA_WIN(W)−0.01`)로 최적화 가속, non-uniform coupling width로 rate-loss 없이 설계공간 확장.
7. 구현 디테일: `B0`에 항등행렬 포함 강제(linear-time encoding 조건), 목표 window 작으면 `B0`에 edge 집중, 크면 큰 `w`로 좌경계 CN degree↓(left decoding wave 강화).
8. 최적화 절차: differential evolution(mutation/crossover/selection, `Np=100`, `G=1000`) - 목표 window size `W*`별 앙상블 family `{P6..P24}` 산출, 1회 최적화 <1시간.
9. 결과: BEC에서 기존 SC-LDPC(CA/CB/CC) 대비 FER 최대 2자릿수 개선(W=18,24), AWGN에서도 우세. 목표 window size에 맞춰 쓸 때만 이득(W<W*면 급격 열화).
10. 한계: HW 미설계, 이론+시뮬만, error-floor·trapping set 미다룸, SC-LDPC/window-decoder 전용.
