### B. 알고리즘 요약

1. 시스템: BPSK over AWGN/Rayleigh, 5G NR QC-LDPC(BG1, `kb=22`), rate `1/3, 1/2, 2/3`, `z=12`, PCM `264×792/528/396`, K=256bit.
2. 문제: MS 계열 저복잡도 디코더의 성능 손실을 복잡도 큰 증가 없이 만회하고자 함.
3. 모델: 심볼주기 내 Nyquist보다 빠르게 오버샘플링(`OSR=M`)하여 M-folded 반복심볼 `y_{N,i}=he^{jφ}x_N+n_{N,i}` 생성, noise i.i.d.
4. 핵심기법 1단: 외부 SISO repetition 디코더가 M-folded 심볼을 결합 우도로 처리, `LLR(y)=Σ_{i=1}^{M} LLR(y_i)`로 신뢰도 향상.
5. 식 의미: M개 샘플 LLR 합산이 유효 SNR을 높여 내부 디코더 초기 신뢰도를 강화.
6. 핵심기법 2단: 강화된 LLR을 내부 OMS(offset `β=0.6`) layered 디코더의 VN 초기값으로 투입, layered QC-LDPC 복호.
7. 구현: 오버샘플링에 다중 ADC 채널, SRRC 필터(roll-off `α=0.3`) 사용, 코딩레이트는 반복이 수신단이라 불변.
8. 복잡도: 전체 `≈O(K/R)`, 추가항 `(M+1)`만 붙어 conventional 대비 유의미 증가 없음.
9. 결과: OSR=2에서 BER 10^-3에 OMS 대비 ~3dB(이상), SRRC 시 ~2.4dB 이득; iteration 10→5 감소, BER 10^-5서 0.9~1.9dB.
10. 한계: HW 미설계, MATLAB 시뮬만, 이득이 수신단 오버샘플링에 의존.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 내부 OMS layered 복호 | decoder.cpp LDPC_Decoder(), C2V_Cal() | Min-Sum 계열은 이미 보유, 중복 |
| offset min-sum 연산 | decoder.cpp CNU_Update_New_Mag() | offset은 기존 min-sum과 유사, 신규성 낮음 |
| 외부 SISO repetition / 오버샘플링 | 대응 없음 | NAND read에 오버샘플링 개념 부재, 이식 불가 |
| LLR 초기화(soft prior) | channel.cpp Set_LLR_Th() | soft 입력 경로는 있으나 오버샘플 합산 LLR은 무관 |
```

적용 가치: **낮음** — 이득의 핵심이 수신단 오버샘플링 반복이며 NAND 플래시 read에는 존재하지 않는 물리계층 가정이라 떼어낼 부분이 없음.
