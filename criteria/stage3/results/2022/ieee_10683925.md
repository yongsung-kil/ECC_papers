### B. 알고리즘 요약

1. `GF(2^m)`(characteristic 2)에서 `α^i+α^j+α^k=0`을 만족하는 triplet 집합 `S`(sum-constraint)를 정의.
2. `S`를 line, `GF*(2^m)` 원소를 point로 보면 `PaG(2^{m-1}-1, 3, 3)` 부분기하(T-PaG)가 성립함을 증명.
3. 각 line 발생벡터(weight 3)를 cyclic-shift하여 `(2^m-1)×(2^m-1)` circulant `H_i`를 만들고, 세로/가로 배열로 line-point/point-line incidence 행렬 `H`,`G` 생성.
4. `H`,`G`는 RC-constraint(두 행이 최대 1곳에서 겹침)를 만족 → Tanner graph girth≥6 보장.
5. `H`의 null space = 길이 `2^m-1`, 최소거리 `2^{m-1}`인 cyclic (binary) LDPC 부호; `G`의 null space = QC-LDPC 부호.
6. 부수적으로 line/point 발생벡터가 optimal cyclic constant-weight 부호 `C_cw(2^m-1,3,1)`를 형성, 1's-complement dispersion으로 QC-balanced 부호도 구성.
7. nonbinary 확장: base matrix `B(3,τ1)` 각 원소를 uniform/α-multiplied CPM으로 dispersion → `(3,k)`-regular `2^m`-ary QC-LDPC (rate≥(k-3)/k).
8. 이 nonbinary 배열의 nonzero를 1로 치환하면 binary QC-LDPC/QC-CW 부호도 얻음(구성만 언급).
9. 결과: 64-ary (378,197) QC-LDPC(rate 0.521)를 FFT-QPSA로 5/10/50 iter 복호, protograph 기반 64-ary (540,270) 대비 더 높은 rate·짧은 길이로 우수(Fig.1).
10. 한계: HW 미설계, 성능 검증은 nonbinary 코드에만, binary LDPC는 구성 가능성만 서술.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| T-PaG 기반 binary cyclic/QC-LDPC H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | 이론상 새 base matrix 후보이나 Prime ECC는 특정 QC-LDPC 고정, 재검증 부담 큼 |
| nonbinary(GF(2^m)) CPM-dispersion / FFT-QPSA 복호 | 대응 없음 | Prime ECC는 binary min-sum 전용, nonbinary 미지원 |
| constant-weight / QC-balanced 부호 구성 | 대응 없음 | ECC 디코더/인코더 파이프라인과 무관 |

적용 가치: **낮음** — nonbinary 중심 순수 구성법이며, binary QC-LDPC 부분은 girth≥6 성립만 보이고 고rate/성능/HW 검증이 없어 Prime ECC(고rate binary QC-LDPC)에 떼어 쓸 근거가 부족하다.
