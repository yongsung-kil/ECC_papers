### B. 알고리즘 요약

1. 시스템/코드: BPSK→memory ν ISI 채널(CH-I ν=1, CH-II ν=2, CH-III ν=4)+AWGN, turbo equalization(검출기↔LDPC 반복). 설계율 R=1/2.
2. 문제: 고전 irregular LDPC는 특정 채널용 degree distribution 최적화 필요 → 채널이 바뀌면 성능 급락(예: CH-II용 코드가 CH-III에서 SIR 대비 ~3dB 열화), 송신단이 채널을 알아야 함.
3. 핵심주장: 정규 SC-LDPC는 threshold saturation으로 BP threshold가 uncoupled 코드의 MAP threshold(≈SIR)에 근접 → 단일 코드로 여러 ISI 채널을 채널지식 없이 universal하게 커버.
4. 핵심기법1(SC 구성): (dv,dc) 정규 LDPC를 L개 spatial position에 복제, coupling memory m으로 위치 t의 VN을 CN [t..t+m]에 연결. (5,10),(6,12) 정규코드 사용.
5. 식 의미: DE 업데이트 `p(Lv→c)=p(LH→v)⊛λ(p(Lc→v))` 등으로 결합 BP threshold 계산, threshold saturation이 MAP=SIR에 접근함을 보임.
6. 확장(LMMSE): 최적 BCJR 대신 suboptimal LMMSE 검출기에서도 coupling 시 BCJR에 근접(CH-III에서 gap 5dB→수분의 1 dB).
7. 부수: BCJR용 GEXIT area bound가 LMMSE엔 무의미 → EXIT curve positive-gap 조건 기반 근사 threshold `γArea` 제안(식 1~3, J(σ) 근사).
8. 최적화: irregular 코드는 EXIT chart 리스트업 후 DE로 best threshold 선택하는 2단계 탐색(비교군).
9. 결과: (6,12) m=6, L=500, N=10000(R=0.494), window W=30(latency WN=300000 symbol), IC=30/ID=20 반복. SC 코드가 CH-II·CH-III 모두 강건, irregular 최적코드보다 threshold 우수.
10. 한계: 순수 threshold/DE/EXIT 이론+시뮬만, HW 미설계, ISI/turbo equalization 특화(NAND memoryless 채널과 무관), SC-LDPC는 Prime ECC 고정 QC 구조와 이질.
