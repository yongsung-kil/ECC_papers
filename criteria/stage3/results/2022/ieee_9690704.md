### B. 알고리즘 요약

1. 대상: DTTB 방송(ATSC 3.0/DVB)용 QC-SC-LDPC(spatially-coupled) coded transmission, coupling length `L`, smoothing `w`, lifting `Z=768`, `nSC=92160`, `nconst=16128`, `Rtrue=2/3`.
2. 문제: SC-LDPC full BP 복잡도가 `L`에 비례해 과대 → sliding window decoding으로 `L` 무관하게 줄이되 window size `W`(복잡도·latency ∝ `W`)를 최적 선택해야 함.
3. 모델/스케줄: CN-centered window + layered scheduling(flooding 대비 ~50% 복잡도↓), 고정 복잡도 `Ī1` 제약 하에서 `W` 최적화.
4. 핵심 기법 1: QC-MET-DE(protograph/edge-type별 PDF convolution `(7)~(9)`)로 `W` vs SNR threshold gap 곡선을 구해 최적 `Wopt` 도출.
5. 핵심 결과: `Wopt`가 constraint length의 2배 미만 가능 → 기존 경험칙(3~5×) 완화, edge spreading pattern 구조가 `Wopt`에 영향.
6. 핵심 기법 2: PC(parity-check) 기반 조기종료 - Type-1(고SNR, window 내 PC식이 수 iteration 연속 만족 + LLR 부호 불변 시 종료), Type-2(저SNR, 미충족 PC식 비율 `p=0.0625` 초과 시 전체 복호 중단).
7. 구현 디테일: soft-BER 추정 `BERj=1/(1+e^|Lj|)`은 LLR 기반이라 지수연산 필요 → HW 친화한 PC 기반 채택, non-integer iteration은 확률혼합으로 처리.
8. 학습/최적화: NN 없음 - MET-DE 해석 + 시뮬로 `W=16`, ET 파라미터는 경험적 설정.
9. 결과: `W=16`, `Imax=2`에서 Type-1+Type-2 ET로 총 복잡도 72% 이하, SNR loss ≤0.025dB. ATSC 3.0 LDPC 대비 comparable threshold, 복잡도는 conventional의 최대 2배.
10. 한계: HW 미설계, 방송 채널(AWGN/Rayleigh, NUC) 타깃, SC-LDPC convolutional 구조 전용, 향후 quantized BP 예고만.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| sliding window / SC-LDPC 복호 | 대응 없음 | Prime ECC는 binary QC-LDPC block 부호, spatial-coupling 미지원 |
| layered scheduling BP | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | 개념적으로 유사(Dual-Update 스케줄 보유), 이미 보유라 중복 |
| PC 기반 조기종료(Type-1/2) | `partial_CRC.cpp` `partial_crc_HW_T4()`, `full_CRC.cpp` | 조기종료 목적 동일하나 Prime ECC는 CRC 기반 이미 보유 |
| QC-MET-DE 부호설계 해석 | `ecc_top.cpp` `Load_PCM()` (H-matrix) | SC용 base+edge spreading 설계라 block QC-LDPC에 직접 대응 안 됨 |

적용 가치: **낮음** - SC-LDPC/방송 전용 sliding window 기법이라 Prime ECC의 binary QC-LDPC block 구조와 정합되지 않고, 재사용 가능한 layered scheduling·PC 조기종료는 이미 보유(중복).
