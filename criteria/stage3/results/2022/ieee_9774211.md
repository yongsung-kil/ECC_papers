> 총평: 5G NR BG2 dual-diagonal QC-LDPC 인코더의 barrel shifter를 MUX 공유로 절반 줄인 area 최적화 — 인코더가 dual-diagonal인 Prime ECC에 아이디어 차원으로만 참고 가능.

### B. 알고리즘 요약

1. 시스템: 5G NR URLLC용 short/low-rate binary QC-LDPC 인코더, BIAWGN 채널, 3GPP BG2(mb=42, nb=52), lifting `Z`.
2. 문제: URLLC는 초저지연·초고신뢰 동시 요구 → short block이 유리하나 인코더 area/속도가 관건.
3. 모델: `HC^T=0`, codeword `C=[d, Pa, Pc]` (message + dual-diagonal parity Pa + single-parity-check 확장 Pc).
4. 핵심기법: dual-diagonal 구조 이용, parity Pa를 `δx = Σ a_xy d_y` 누적과 cyclic shift(`Pa1^(1)=Σδx`)로 구함 — Gauss/RU 대비 sparse 유지·저복잡도.
5. 식 의미: dual-diagonal의 back-substitution을 단순 circulant shift 연쇄로 환원해 인코더를 barrel shifter+XOR tree로 구현.
6. 확장: Pc는 C1,C2 (single parity check extension) 행렬로 `Pc_i = Σ c d + Σ c Pa` 계산 (Raptor-like 구조).
7. 구현 디테일: barrel shifter는 log배럴(N-bit → N·logN MUX), 입력 shift register와 barrel 사이에 MUX 단을 삽입해 barrel shifter 수를 `kb+s+1` → `kb/2+s+1`로 절반 감소.
8. 메모리: shift 계수를 A_ROM/C1_ROM/C2_ROM(비영 계수만), δ값은 single-port RAM 저장.
9. 결과: kb=10,Z=16 예시에서 barrel shifter 16→10, MUX 1024→645 (37% 감소), clock cycle 불변(속도 손실 없음), Xilinx Virtex-7 구현.
10. 한계: 인코더 area만 다룸, 복호·BER 성능·gatecount·throughput 수치 없음, 5G BG2 특화.
