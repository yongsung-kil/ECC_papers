### B. 알고리즘 요약

1. 대상: 조합설계(combinatorial design) 기반 short-length regular LDPC 부호 구성, BPSK+AWGN, BP(sum-product) 복호, iter 50.
2. 문제: short-length(<1000) LDPC는 random 구성 시 4-cycle 회피 어렵고 부호화 복잡도 높음. 기존 BIBD/STS 구성은 rate 고정·주로 long code 대상.
3. 배경: Tanner graph의 4-cycle은 BP 메시지 독립성 가정을 깨 성능 저하; girth(최단 cycle) 크게 하는 것이 핵심.
4. 핵심기법: BIBD의 incidence matrix 전치를 H로 사용. `λ=1`(BIBD의 pair 파라미터) → 두 VN이 동일 두 CN에 함께 걸리지 않아 4-cycle 원천 배제 (Theorem 1).
5. 구조: KTS(Kirkman Triple System)는 decomposable STS(k=3,λ=1)로 block을 parallel class로 분할 가능; H의 column weight 항상 3, (3,r)-regular.
6. 확장: cycle 길이 c 회피 최소 code length 유도, girth 하한 6 증명 (Theorem 2).
7. rate 조정: parallel class(row weight 1) 삭제로 row weight 유지한 채 rate를 넓은 범위(0.25~≈1)로 가변; 삭제 시 min distance 유지·증가.
8. 성질: min code distance ≥ 4 (column weight 3, 두 열이 CN 1개 이하 공유).
9. 결과: (35,20) KTS-LDPC(rate 0.571)가 random (36,18) 대비 BER/FER 우수(iter 50); parallel class 삭제 (30,15) rate 0.5로 추가 개선. 구성 복잡도 ~linear (PEG의 O(n²) 대비).
10. 한계: HW 미설계, girth 하한 6에 그침, non-QC regular, short-length 특화(자율주행·UAV 링크 등).

| 4-cycle-free/girth 보장 부호설계 | 대응 없음 | 부호구조 자체 교체는 재검증 부담 큼 |
