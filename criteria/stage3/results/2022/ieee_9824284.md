### B. 알고리즘 요약

1. 대상은 5G-NR LDPC(BG2), BPSK/AWGN, code rate 1/5~5/6, `k=10, b=384` MET(multi-edge-type) 구조.
2. 문제: MSA는 scale-invariant·저복잡도지만 `tanh`를 min으로 대체해 horizontal 출력이 진짜 LLR이 아니게 되어 belief 전파가 왜곡되고, NMSA류는 MET-LDPC에서 성능·안정성을 동시에 못 잡음.
3. 모델: AWGN `y=x+n`, a-priori `LLR = 2y_i/σ²`; MSA 출력을 확률분포 관점에서 real LLR과 비교.
4. 핵심기법: rescaling — horizontal 출력 `l_i=ζ`의 PDF(`PDForg`)를 real LLR의 PDF에 맞도록 출력값을 보정(식 (5)~(9)), SPA를 template로 삼지 않음.
5. 식 의미: `L(l)=log(p0/p1)` 로 정의된 집합 `Φ(ξ)`를 통해 각 출력을 real LLR ξ로 사상, min으로 깨진 LLR 정합성을 복원.
6. 확장: asymptotic optimal(각 edge 실시간 보정, SPA급이나 HW 불가) → suboptimal(least KL divergence로 factor α, 식 (10)~(11)) → 실용판 CL-RANMSA.
7. 구현 트릭: check-ratio(5bit) 기반 edge 분류, CN degree>6 여부로 2 type 분류, factor는 [0,1]을 M 구간 이산화한 LUT.
8. 최적화: MET-DE로 decoding threshold `SNRth`를 목적함수로 삼아 `K×2^N` factor LUT를 greedy 탐색(algorithm 1), Monte Carlo 샘플 한계 회피.
9. 결과: BG2에서 제안법이 asymptotic optimal과 threshold 차 0.098dB, SPA 대비 +0.244dB이며 rate-1/5로 얻은 factor가 다른 5개 rate에도 near-capacity로 전이.
10. 한계: HW 미설계, error-floor·gate count·throughput 미보고, layered/MET 구조 및 SNR 추정에 의존.
