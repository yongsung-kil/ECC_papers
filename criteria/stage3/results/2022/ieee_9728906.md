### B. 알고리즘 요약

1. 시스템: source-relay(MIMO-RF, Rayleigh fading) + relay-destination(point-to-point FSO, Gamma-Gamma turbulence + pointing error)로 구성된 double-hop DF 협력 릴레이. 코드 rate/length 미명시.
2. 문제: FSO는 안개·난류·정렬오류에 취약하고 RF는 스펙트럼 부족. 혼합 RF/FSO 릴레이로 신뢰도·커버리지 개선을 노림.
3. 채널 모델: S-R은 Rayleigh, R-D FSO는 path loss `h_l` + 난류 `h_a`(Gamma-Gamma) + pointing error `h_p`. IM/DD 및 heterodyne 검출 SNR 식 정의.
4. 핵심 기법: 정보를 GC-LDPC로 부호화, PPM 변조. 첫 홉에서 Interference Alignment(IA) + OR-ML 알고리즘으로 간섭 제거.
5. GC-LDPC 구조: 여러 짧은 local code(대각 상단) + 하단 global connection part로 구성된 QC-LDPC. Latin square + superposition으로 구성.
6. 복호: 2단계 local-global 복호 - local phase(local code 독립 복호, `I_L`회) ↔ global phase(`I_G`회) 전환, 수렴 조건까지 반복.
7. 대안 direct-acting 방식: 모든 local decoder 독립 복호 후, 실패 시에만 전체 `H` 기반 level-2 복호로 전환(`I_L`=3,6,10 비교, `I_2`=10).
8. 학습/최적화: 없음(구성법 + 시뮬 분석).
9. 결과: BER-SNR 곡선에서 GC-LDPC가 SC-LDPC/QC-LDPC 대비 부호이득 우위, coupling 길이 증가 시 성능 향상. 안테나 수 증가 시 MIMO 이득. 대부분 global 교환 4회 이내 수렴, level-2 진입 후 1~2 iter 수렴.
10. 한계: HW 미설계, Matlab 시뮬만, 부호 rate/length/gate 수치 전무. 광무선 채널 특화로 NAND 무관.

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |

| 2단계 local-global 복호 스케줄 | decoder.cpp LDPC_Decoder() 이터레이션 루프 | 낮음 - 스케줄 아이디어는 있으나 GC-LDPC 전용, min-sum 세부 없음 |
| MIMO-RF/FSO 채널 모델 | 대응 없음 | 무관 - 광무선 채널은 NAND RBER/AWGN와 무관 |
| IA / OR-ML 간섭제거 | 대응 없음 | 무관 - 통신 물리계층 기법 |
| PPM 변조 | 대응 없음 | 무관 |
```

적용 가치: **낮음** - LDPC를 통신 링크의 블랙박스 ECC로만 사용했고, 부호설계/디코더 알고리즘 세부·수치·HW가 없어 NAND binary QC-LDPC ECC로 떼어낼 요소가 사실상 없다.
