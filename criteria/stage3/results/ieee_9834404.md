### B. 알고리즘 요약

1. 대상은 SC-LDPC(spatially-coupled LDPC): base Tanner graph를 coupling length `L`, coupling width(memory) `m`으로 edge spreading 하여 결합한 부호.
2. windowed decoder는 창 길이 `W`(연속 `W`개 constraint position + 인접 variable node)만 보고 좌→우로 슬라이딩 복호한다.
3. 핵심 정의: 시각 `i`에서 디코더가 보는 부분부호를 `window code C_Wi`로 정의, 대부분 동일한 것을 `typical window code W(C)`라 부른다.
4. 문제의식: iterative decoding은 cycle-free 그래프에서 최적(stopping/absorbing set 회피)이나 cycle-free 부호는 distance가 나쁨(Etzion bound) → window code만 cycle-free로 만들면 두 이점을 절충.
5. Lemma IV.3: cycle-free window의 nonzero 수 `wt(H_W) ≤ (W+m)nv + Wnc - 1`.
6. Lemma IV.6 + Corollary IV.7: `nv ≤ (Wnc-1)/(W-m)`, base matrix 최대 design rate `R = 1 - (nv+1)/(2nv)`.
7. Theorem IV.8: cycle-free window code면 `nv ≤ 2nc-1`, `R ≤ 1 - (nv+1)/(2nv) < 1/2` (memory·window 길이 무관).
8. Corollary IV.12: `nv` 홀수 & `m+1 ≤ W ≤ 2m`일 때 최대 rate 달성 가능(Lemma IV.9 구성으로 존재 증명).
9. Theorem IV.13/Corollary IV.14: `W > m(nc+1)-1` 이면(충분히 큰 coupling length) window code에 반드시 cycle 발생 → cycle-free window code를 가진 SC-LDPC 코드가 전체적으론 cycle을 가질 수 있음을 보임.
10. 한계: 순수 존재성 증명(성능/BER/HW 없음), 저자도 결과 부호가 "good code 예시가 아니다"라고 명시.
