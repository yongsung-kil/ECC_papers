### B. 알고리즘 요약

1. 대상 부호: binary `QC-LDPC`, CPM-array 구조. rate 0.5~0.975, length 1016~66049 (예: (1016,508) rate0.5, (7240,6519) rate0.9, (16129,15372) rate0.953, (32382,16191) irregular rate0.5).
2. 문제: partial geometry로 대수·기하 구조가 풍부해 waterfall·error-floor가 좋은 LDPC를 얻되, QC 구조로 HW 구현성까지 확보.
3. 핵심 정의: partial geometry `PaG(γ,ρ,δ)` — 각 점이 γ 라인, 각 라인이 ρ 점, 라인 밖 점이 δ개 라인으로 연결. line-point adjacency matrix H가 CPM array면 `PaG(γ,ρ,ρ-1)`(net) (Theorem 1,2).
4. 구성 1(prime field): `Bp=[i·j mod p]` (식7) base matrix → 각 원소를 p×p CPM으로 dispersion → `QC-PaGp(p,p,p-1)`. 부분행렬 `Bp(k,r)`로 다양한 rate `≥(r-k)/r`·length `rp` 부호.
5. 구성 2(cyclic subgroup): GF(q)의 소수 order t 부분군 `S={1,β,…,β^(t-1)}`로 `Bc=[β^(ij)]`(식10) → `QC-PaGc(t,t,t-1)`. GF(2^7)/t=127, GF(2^16)/t=257 예시.
6. masking: CPM을 ZM으로 치환(Hadamard product Z∘H)해 short cycle 제거 → girth 6→8, cycle 수 감소. (3,6)-biregular나 density-evolution 최적 degree분포 irregular 부호 생성.
7. 핵심 결과(정정능력/error-floor): GF(127) (1016,508) 부호 error-floor 없이 BER 10^-15까지. GF(2^7) (16129,15372) rate0.953 부호 (FPGA 6bit 양자화 MSA) BER 10^-15, BLER ~10^-12 error-floor 없이 waterfall 우수. PEG 부호와 동등하나 QC 구조로 HW 복잡도 낮음.
8. iteration 수렴: 큰 row weight(예 127) → VN 연결도 높아(각 VN이 756 VN과 path-2 연결) MSA 5·10·50 iteration 거의 겹침 → 빠른 수렴.
9. 이론 경계: trapping-set 크기 하한 `τ≥(γ+1-κ)κ` + net 개선 경계(Theorem 3,4), 인접행렬 second eigenvalue `√(γ+ρ-1-δ)`로 expander 성질, min distance/rank 경계(식17,18). δ=γ-1(net)일 때 expansion 최적.
10. 한계: 순수 이론/시뮬(AWGN·BPSK), HW gate count/throughput 없음, 부호길이가 소수 p·t에 강하게 종속되어 임의 lifting/길이 자유도 낮음, NAND 채널 미검증.
