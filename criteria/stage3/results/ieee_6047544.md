### B. 알고리즘 요약

1. 부호설계 전용 논문: Steiner system `S(t,k,v)` (t≥3)와 그 residual design `res_x(D)`로부터 (t-1)-incidence matrix `N_{t-1}(D)`를 생성해 parity-check matrix로 사용.
2. 기존 연구는 t=2(BIBD)만 사용했으나, 본 논문은 t≥3(특히 Steiner quadruple system, inversive/spherical geometry)로 확장해 girth-6을 보장하는 새로운 code family를 제시.
3. 핵심 정리: `N_{t-1}(D)`가 RC-constraint(임의 두 행이 공통 1을 최대 1개 위치에서만 가짐)를 만족하면 Tanner graph에 4-cycle이 없어 girth≥6이 보장됨(Lemma 1, 2).
4. 구성된 부호는 `(J,L)`-regular이며 column-weight `J`, row-weight `L`, 최소거리 하한이 조합론적 파라미터(v,k,t)로 결정됨(Theorem 2, 3).
5. 확장: 유한체 F_q 위의 원소를 (q-1)-fold matrix dispersion으로 치환해 non-binary/binary QC-LDPC 부호도 동일 방식으로 구성 가능(Section III).
6. Steiner quadruple system(SQS(v), t=3,k=4)에서 column-weight 6인 high-rate regular LDPC family 생성(Proposition 1); v 증가 시 rate→1.
7. inversive/spherical geometry(S(3,q+1,q^n+1))에서 다양한 column-weight를 갖는 또 다른 family 생성(Proposition 2); q→∞ 시 rate→1, 최소거리→∞.
8. 부수 트릭 없음: 별도의 양자화/HW 근사/최적화 절차 없음, 순수 조합론적 구성 및 정리 증명.
9. 결과: AWGN+BPSK, SPA(binary)/FFT-QSPA(non-binary) 반복복호, 최대 20 iteration으로 시뮬레이션. 제안 부호가 random-like 및 기존 finite-field 기반 QC-LDPC[8]와 동등하거나 우수한 BER 성능을 보임(구체 dB 수치는 그림에만 있어 본문엔 없음).
10. 한계: 순수 이론/구성법 논문으로 HW 설계 없음, 디코더 알고리즘 개선 없음(표준 SPA/FFT-QSPA 사용), 특정 파라미터(v,k,t) 조합에서만 부호 존재 가능.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Steiner system 기반 H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | 부호구조 자체를 특정 조합설계로 고정하는 것이라 기존 QC-LDPC(base+circulant) 구조와 직접 호환 어려움 |
| non-binary QC-LDPC 확장(matrix dispersion) | 대응 없음 | Prime ECC는 binary 전용, non-binary(GF(q)) 기법은 이식성 하 |
| SPA/FFT-QSPA 복호 | 대응 없음 | 표준 sum-product 복호이며 Prime ECC는 이미 min-sum 채택(중복) |

> 적용 가치: 낮음. 순수 조합론적 부호 구성법(binary/non-binary Steiner-design LDPC)으로 디코더·HW·양자화 개선이 전혀 없고, Prime ECC의 고정된 QC-LDPC 구조 재설계는 부담이 커 이식 실익이 낮음.
