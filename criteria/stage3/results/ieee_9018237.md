> 총평: 무한 codeword length SC-LDPC 앙상블의 BP 디코딩 웨이브 전파 속도를 계산하는 순수 이론(density evolution) 논문으로, NAND 유한장 binary QC-LDPC 이식 가치는 낮음.

### B. 알고리즘 요약

1. 대상은 일반 binary memoryless symmetric(BMS) 채널 위에서 BP 복호되는 SC-LDPC(λ, ρ) 앙상블(무한 coupling width `w→∞` continuum limit 가정).
2. 기존 coupled density evolution(CDE)은 정확하지만 계산비용이 매우 크고, Gaussian approximation 기반 EXIT 근사는 빠르지만 오차가 큼 — 이 둘 사이 trade-off 문제를 다룸.
3. 핵심 관찰(ansatz): decoding profile(디코딩 웨이브)은 density space 상에서 uncoupled DE의 fixed point(FP) 몇 개를 순차로 지나가는 경로로 볼 수 있고, 각 구간(transition region)은 일정 속도로 solitonic하게 전파.
4. 제안 기법 interpolated density evolution(IDE): decoding profile을 uncoupled DE의 fixed point들 사이를 잇는 1차원 interpolation function `α(t)`로 근사.
5. VN/CN transfer function `˚ρ`, `˚λ`을 정의해 interpolation function의 반복 갱신 recursion을 구성, 이로부터 웨이브 속도 `v̂_j`를 저비용 산출.
6. 다중 transition region이 공존하는 경우, 속도 `{v̂_j}`의 단조증가성 체크(Algorithm 1/2)로 몇 개의 non-skipped fixed point를 유지할지 자동 판별.
7. IDE 기반 threshold `h_IDE`가 CDE 기반 threshold `h_CDE`와 Ansatz 1/2 하에서 일치함을 정리(Theorem 1)로 증명 — 채널 엔트로피가 BP threshold에 가까울수록 근사 정확도 보장.
8. 별도 학습/최적화 절차 없음 (해석적/수치적 recursion만).
9. BSC/BIAWGNC에서 regular·irregular SC-LDPC 앙상블에 대해 IDE가 CDE와 거의 일치하는 threshold·속도를 제공하며 GA-EXIT 대비 오차가 훨씬 작음을 수치예제로 확인.
10. 유한 codeword length 코드나 실제 HW/디코더 구현은 다루지 않으며, 시뮬레이션도 없이 순수 이론적(밀도진화) 수치계산에 그침.
