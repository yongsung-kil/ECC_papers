> 총평: WSN 분산 네트워크-채널 코딩(LDGM)의 4-cycle 제거 구성법으로 NAND QC-LDPC ECC와 접점이 거의 없다.

### B. 알고리즘 요약

1. 시스템: 대규모 WSN에서 `M`개 source node + `N-M`개 relay node가 공통 destination과 통신, Rayleigh fading 채널, BPSK.
2. 코드: destination에서 broadcast/cooperative 두 phase 패킷을 결합해 systematic irregular LDGM 코드(rate `1/2`, length `2000`)를 on-the-fly로 생성.
3. 문제: 분산(decentralized) 구성 특성상 short cycle(특히 4-cycle) 제거가 어려워 code 성능(error floor) 저하.
4. 설계 degree 분포: `Λ(x)=0.23x+0.4573x^4+0.3127x^19`, relay는 `P(x)=x^8`; source=variable node, relay=check node에 대응.
5. 핵심 기법: 네트워크를 cluster로 분할, 각 relay가 자기 retrieval-set `F_j`에서 source node를 선택하되 4-cycle을 유발하지 않는 후보만 채택하는 분산 트리 확장 알고리즘.
6. 두 4-cycle 정의: intercluster(서로 다른 cluster의 relay 두 개가 관여) / intracluster(같은 cluster 내).
7. 두 전략: Strategy 1(partial NTK, intracluster만 제거), Strategy 2(complete NTK, inter+intra 모두 제거).
8. degree 관리: counter `du(s_i)`로 각 source가 지정 degree `d(s_i)` 이상 선택되지 않게 제어, signalling으로 갱신.
9. 복잡도: Strategy 1 `O(|C|(ds-1)dr^2+n)`, Strategy 2 `O(n(ds-1)dr^2+n)` — relay 수 `n`의 선형 함수.
10. 결과: SNR=5dB에서 Strategy 2가 error floor를 `3×10^-5 → 6×10^-6`로 저감; cluster 클수록 제거되는 4-cycle 증가.
11. 한계: HW 미설계, WSN relay 협력 가정(서로 완벽히 청취) 의존, 시뮬만.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 4-cycle 제거 code 구성 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 고정 QC-LDPC H-matrix를 로드; 분산 릴레이 구성법은 대응 불가 |
| LDGM/systematic 부호 생성 | 대응 없음 | Prime ECC는 dual-diagonal QC-LDPC 인코딩, LDGM 릴레이 구성과 무관 |
| SPA(sum-product) 복호 | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum 사용, 본 논문은 복호기 개선 없음 |

적용 가치: **낮음** — WSN 분산 LDGM 릴레이 구성 특화라 NAND 중앙집중 binary QC-LDPC ECC에 떼어 쓸 요소가 없다.
