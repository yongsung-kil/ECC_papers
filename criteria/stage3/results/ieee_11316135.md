### B. 알고리즘 요약

1. 시스템은 FTN(faster-than-nyquist) 신호(의도적 ISI 도입)로 전송되는 5G 스타일 protograph-based QC-LDPC 부호(BG1/BG2, rate-compatible raptor-like 구조)를 다룸.
2. 기존 5G LDPC는 Nyquist 채널에 최적화되어 있어 FTN의 ISI 환경에서 성능이 저하되는 문제를 해결하고자 함.
3. 채널 모델은 `y = Gx + η` (G는 ISI를 나타내는 Toeplitz 행렬, η는 컬러드 노이즈), Ungerboeck observation model 기반 SPA 검출기를 사용.
4. 핵심 기법은 PEXIT(protograph EXIT) 차트 분석으로, FTN 검출기의 입력-출력 상호정보량(MI) 함수를 `f(Eb/N0, IA,Det)` 다항식 최소자승 피팅으로 근사해 반복 호출 비용을 제거.
5. `IE,Det ≈ f(Eb/N0, IA,Det)` 근사로 decoding threshold를 예측 가능케 하고, 이분탐색으로 최소 필요 Eb/N0(threshold)를 구함.
6. 관찰 결과 FTN에서는 5G-NR식 정보 puncturing이 오히려 성능을 저하시키고, 비교 가능한 앙상블 내에서 parity-check matrix가 상대적으로 sparse할수록 유리함(단, 과도한 sparsity는 역효과).
7. 이에 기반해 5G base matrix의 A/E 영역에 random masking(Bernoulli-p, diagonal I/D 구조는 보존)을 적용, EXIT threshold가 최소인 masking 패턴을 다수 시행 중 선택.
8. 최적화 절차는 masking 확률 `p ~ U[pmin,pmax]`를 매 iteration 무작위 추출해 EXIT threshold를 평가하는 grid/random search(유전 알고리즘 등은 향후 과제로 명시).
9. 결과: 표준 5G LDPC 대비 FTN에서 0.4dB 이상 코딩이득, 동일 스펙트럼효율 기준 Nyquist 대비 최대 0.6dB 이득, 기존 FTN 전용 고성능 부호 대비도 우세.
10. 한계: 순수 시뮬레이션(BER/threshold) 검증뿐이며 HW 구현·양자화(soft 다단 LLR, turbo-equalization 반복 구조 전제) 없음, FTN 고유의 ISI 가정에 종속된 설계로 일반 채널(NAND)에 직접 적용 불가.
