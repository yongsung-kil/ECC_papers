### B. 알고리즘 요약

1. 대상: binary QC-LDPC, `H`는 순환치환행렬(CPM) 블록. AWGN+BPSK, 최대 50 iteration sum-product 복호.
2. 문제: RS기반 QC-LDPC는 girth≥8은 보장하나 최소 해밍거리 상계가 낮아 error performance에 한계.
3. 기법1(RS-QC-LDPC): GF(2^s)의 order `n` 원소 `β`로 `BRS(d,n)` RS 패리티행렬 구성, 지수를 순환 shift량으로 사상해 `HRS(d,n)` 생성 → 4-cycle free.
4. girth≥8 조건: 선택 열 인덱스집합 `Λt`의 임의 세 원소가 식(3)의 9개 조건 모두 만족 시 shortest cycle ≥8 보장.
5. 기법2(protograph 결합): weight matrix에 병렬간선(entry ≥2)을 도입한 protograph를 `BRS(d,t)` 위에 얹어 `M_{i,j}=Σ m_{i,j,s}·H_{i,j}` 순환행렬로 구성(식4).
6. 의미: 병렬간선으로 weight matrix 값을 키우면 protograph 기반 최소 해밍거리 상계(Smarandache-Vontobel)가 커져 word error 성능 개선.
7. 구성전략 두 가지: (i) check/variable node degree를 regular하게, (ii) irregular하게. 각각 `HP_RS,1`, `HP_RS,2`.
8. 실험 파라미터: RS over GF(2^8), `BRS(4,85)`, `Λ8={2,5,7,13,20,32,54,60}`, `N=255` → half-rate length 2040; `BRS(4,12)`, `Λ12` → 2/3-rate length 3060.
9. 결과: half-rate(construction ii) BER 이득 ≈0.05dB@1e-5; 2/3-rate BER ≈0.15dB@1e-5, BLER ≈0.12dB@1e-4. 고rate일수록 이득 증가 경향.
10. 한계: 시뮬만(HW·복잡도·throughput 없음), 저rate·소길이, error-floor 영역 그림 손상으로 미확인, 이득 폭 작음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| RS기반 QC-LDPC H-matrix 구성 (girth≥8) | `ecc_top.cpp` `Load_PCM()` | 우리 부호는 고정 QC-LDPC로 로드만, 재구성은 재검증 부담 큼 |
| protograph 병렬간선/weight matrix 부호설계 | `ecc_top.cpp` `Load_PCM()` + H-matrix | 부호설계 레이어 교체는 이식 난이도 높음 |
| sum-product 복호(50 iter) | `decoder.cpp` `LDPC_Decoder()` | 우리는 min-sum; 논문은 복호 알고리즘 개선 아님 |
| 최소 해밍거리 상계/error-floor 개선 | 대응 없음 | 부호구성 이론뿐, 우리 HW 모듈에 직접 대응 없음 |

적용 가치: **낮음** — 순수 부호구성법으로 저rate·소길이 시뮬만 제시하고 HW·고rate·error-floor 검증이 없어, 특정 QC-LDPC로 고정된 Prime ECC에 재구성·재검증 부담 대비 이득이 작다.
