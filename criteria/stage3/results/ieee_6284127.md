### B. 알고리즘 요약

1. 대상: 순환행렬(circulant) 배열 `H`의 null space로 정의되는 binary QC-LDPC (예제 (3654,3335) rate 0.9126, (6,58)-regular, CPM 크기 63×63).
2. 문제: GT/대수적 QC-LDPC의 rank(=차원)와 특히 row-redundancy(중복행 비율)를 일반적으로 예측·설계하는 알고리즘 부재.
3. 모델: AWGN + BPSK, 유한체 `GF(q=2^r)`, order `e=q-1` 순환행렬을 Fourier 변환으로 대각화.
4. 핵심 기법: 순환행렬 `A`를 Vandermonde `V,V^-1`로 `AF=VAV^-1=diag(b0..be-1)` 대각화, base matrix `B=[b_ij]`의 conjugacy 제약 `B(2t)e=B^◦2` 활용.
5. 의미: 켤레류(conjugacy class)로 묶어 `rank(H)=μ0+Σ c_i μ_i`로 rank를 분해 계산.
6. 확장: Hadamard power 경계 `rank(B^◦t)≤μ1·τ(t)`로 base matrix만으로 `rank(H)` 상한 (Theorem 5) → row-redundancy 하한 `R(H)` 도출.
7. 구현 디테일: 2×2 SM-제약 base matrix면 RC-제약 만족(girth≥6), CPM/ZM 배열로 e-fold dispersion.
8. 구성법: 유한체를 두 disjoint 부분집합 `G1,G2`로 랜덤 분할, `Brp=[λi+δj]`로 base matrix 생성 → 유연한 code 차원 + 풍부한 redundant row.
9. 결과: row-redundancy 0.6863이 0보다 (255,175) EG-LDPC에서 BER 10^-5 기준 약 0.6 dB 우수; (3654,3335) 코드는 MSA 50 iter에서 BER 10^-6에 Shannon 1.2 dB 근접.
10. 한계: HW 미설계, 이론+AWGN 시뮬만, 특정 대수 구성에 종속, 비이진 케이스/증명 다수는 arXiv:1202.0702v1로 이관.
