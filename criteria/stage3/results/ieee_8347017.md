### B. 알고리즘 요약

1. 시스템: AWGN + 강한 위상잡음(Wiener 모델 `θk=θk-1+Δk`) 채널, 8PSK/64-QAM 고차 성상, LDPC(code A: 400x800 R=0.5, code B: 737x4095 R=0.82) 또는 turbo 부호.
2. 문제: 위상잡음 하 exact SPA는 연속 확률변수라 구현 불가, DP-BCJR는 고복잡도, Tikhonov 근사는 고rate에서 error floor 발생.
3. 가정: 파일럿이 간격 `P`로 삽입, `θk-θ̂k`가 작다는 전제로 1차 Maclaurin 전개 `e^{jx}≈1+jx` 적용.
4. 핵심기법: 사전 pilots 기반 위상 기준 `θ̂k`를 얻고, 위상 지수를 그 기준 주변에서 선형근사해 factor `fk(ck,θk)`를 가우시안 정준형(GCF) `g(C2, σ²/|ck|², θk)`로 축약.
5. 핵심식: `zk=rk*e^{jθ̂k}` 정의로 `fk`가 가우시안 곱 형태가 되어, SPA forward/backward 메시지가 평균·분산 `(μk,σk²)`, `(ηk,νk²)`만 전파하는 재귀식으로 단순화.
6. 확장: 여러 가우시안 혼합을 단일 가우시안으로 붕괴(collapsing)하는 근사, 위상 기준을 iteration마다 갱신하지 않아 cycle slip 내성 확보.
7. 구현 트릭: Max-Alpha/m-Max-Alpha(지배적 α 항만 사용), 정상상태 상수분산 사용으로 추가 복잡도 절감.
8. 위상기준 획득: DDL, Viterbi&Viterbi, finite-lag Wiener filter 비교 - pilots-only Wiener + 선형보간이 최선.
9. 결과: GCF가 Tikhonov 대비 우수, 고rate/고차 성상에서 Tikhonov의 error floor를 제거, 복잡도는 Tikhonov보다도 낮음(Max-Alpha 8M+27 op/M-PSK).
10. 한계: HW 미설계, 시뮬레이션만, 위성/무선 위상잡음 채널 특화로 NAND AWGN 시나리오와 무관.
