### B. 알고리즘 요약

1. 대상: 일반 binary linear (n,k) 블록부호 C를 BSC(오류확률 p) 상에서 sparse parity-check 행렬 H로 복호.
2. 문제: 채널 용량에 접근하려면 최소거리 절반(≈n/4)을 넘는 오류를 정정해야 하는데, 최적/최소거리 복호는 min(k,n-k)이 크면 불가능.
3. 기법: Iterative Probabilistic Decoding Algorithm(IPDA) - 각 비트의 사후 오류확률 `P_i^(j)`를 orthogonal parity check로 갱신하고 `P_i>0.5`면 해당 비트를 반전(APP threshold 복호).
4. 사후확률비 `q_i`(식(2))는 prior 오류확률 `p_i`와 만족된 parity check 수 `s_i(w)`로 계산; orthogonal check 가정 하 곱셈형.
5. 핵심 모델: IPDA를 "기대 bit-error rate의 self-composition" `P^(l)=f(P^(l-1))`(식(10)(11))로 근사 - 반복 과정의 수렴을 단일 함수 반복으로 환원.
6. Theorem 1: 임계값 `P0`(식(14)) 존재 - `0<p<P0`이면 self-composition이 0으로 수렴, `P0<=p`이면 발산. `P0`는 parity-check 가중치별 개수 `J(w)`로 결정.
7. 의미: `J(w)`가 작은 w에서 클수록(=sparse H) `P0`가 최소거리 절반보다 훨씬 커짐 -> sparse 부호가 절반 이상 오류를 정정하는 이유를 설명.
8. 구현 트릭: reset 있는 IPDA(round 단위, 임계 `ε`/최대반복 `jmax`/최대라운드 `rmax`) + 마지막에 information-set decoding(sliding window)으로 잔여오류 정정.
9. 결과: (5000, k=32/64) truncated simplex 부호, weight 3/5/7 parity-check 다항식으로 실험 - 이론 임계 잡음률이 경험적 임계값의 (비교적 tight한) 상한을 제공.
10. 한계: 평균적/독립오류 가정 기반 이론, HW 미설계, 부호가 암호(correlation attack)용 저rate simplex 부호로 NAND ECC와 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| APP threshold 반복 복호(IPDA) | 대응 없음 | Prime ECC는 min-sum message passing, 본 논문의 확률-반전 threshold 복호와 알고리즘 계열이 다름 |
| 수렴 임계 잡음률 해석 | 대응 없음 | Density Evolution류 이론 분석 도구, 코드베이스에 대응 모듈 없음 |
| information-set decoding 후처리 | 대응 없음 | Prime ECC 후처리(partial/full CRC 조기종료)와 무관 |

적용 가치: **낮음** - BSC 상 저rate 암호용 simplex 부호의 반복 확률 복호 수렴을 다룬 순수 이론으로, NAND용 고rate binary QC-LDPC min-sum 디코더에 이식할 알고리즘/HW 요소가 없음.
