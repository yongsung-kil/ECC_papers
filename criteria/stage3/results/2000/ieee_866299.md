### B. 알고리즘 요약

1. 대상은 Wiberg-type graphical code realization의 특수형인 **normal realization**: symbol variable 차수=1, state variable 차수=2로 제한한 그래프 표현.
2. 문제: 일반 Wiberg-type 실현은 그래프 topology가 자유로워 duality 등 일반 정리를 세우기 어려움.
3. 핵심 모델: normal graph는 degree-1 leaf edge(=symbol `Ak`), degree-2 ordinary edge(=state `Sj`), vertex(=local constraint `Ci`)로 구성되며 sum-product(message-passing)로 복호 가능.
4. 핵심 결과 1: 임의의 Wiberg-type 실현은 변수 복제(replication)로 normal form으로 변환 가능하며 그래프 모양·복호 복잡도가 본질적으로 불변.
5. 핵심 결과 2(duality 정리): group/linear normal realization의 dual은 각 변수를 character group으로, 각 local code를 dual code로 치환하고 ordinary edge마다 sign inverter를 삽입하면 얻어짐.
6. 이 dual 실현은 그래프 topology(사이클 포함)에 무관하게 dual code를 실현 - Mittelholzer의 dual trellis 결과를 일반화.
7. 의미: primal 코드와 동일 그래프·동일 state space로 dual 코드를 실현 가능함을 보임.
8. 학습/최적화 절차 없음(순수 이론).
9. 결과: 수치 실험/BER/gate count 없음 - 정리 증명이 결과물.
10. 한계: 복호 알고리즘 성능·HW·부호설계 수치 전무, 이식 관점의 구체 기법 없음.
