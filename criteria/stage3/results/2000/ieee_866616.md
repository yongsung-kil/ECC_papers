### B. 알고리즘 요약

1. 문제: memoryless binary-input AWGN에서 sum-product LDPC threshold를 full density evolution(DE)로 구하면 무겁고 직관이 부족함.
2. 핵심 가정: variable/check node의 extrinsic LLR 분포가 근사적으로 가우시안, consistency 조건 `sigma^2 = 2*mean` 강제 → 분포를 평균 하나로 파라미터화.
3. 핵심 기법: message 갱신을 `t_l = f(s, t_{l-1})` 1차원 재귀로 축소(`s = m_u0` 입력 LLR 평균, `t_l` = l-번째 iteration check node 출력 mean), 초기값 `t_0=0`.
4. 핵심 식: threshold `s*` = `t_l(s)`가 무한대로 수렴하는 최소 `s`; Lemma 1: `t < f(s,t)`이면 수렴. → threshold를 고속 계산.
5. 안정성 조건(Theorem 1): `lambda_2 < lambda_2*` (`lambda_2* = e^{1/(2*...)} / sum_j (j-1)p_j` 형태)일 때만 수렴 → degree-2 비율 상한 제시.
6. 오류확률 수렴율: `lambda_2 < lambda_2*`에서 `P_l`이 점근적으로 특정 형태로 감소(상수 a,b는 `lambda(x),rho(x),s` 종속), DE 실측과 잘 부합.
7. degree 최적화: 주어진 `rho(x)`와 rate에서 threshold 최대화하는 `lambda(x)`를 선형계획(LP)으로 탐색, `rho(x)` 최적형은 1~2개 인접 degree에 집중.
8. 학습/최적화 절차: LP 기반 degree sequence 최적화(GA로 DE 대체해 고속화).
9. 결과: rate>0.99에서 Shannon 한계 0.05dB 이내 degree sequence 설계 성공(저rate도 양호), 온라인 데모 제공.
10. 한계: 무한 블록길이 asymptotic 이론뿐, HW/복잡도/유한길이 성능 없음, sum-product(tanh) 기반이라 min-sum HW 직접 무관.
