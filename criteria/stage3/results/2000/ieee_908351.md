### B. 알고리즘 요약

1. 시스템: rate 16/17 QMTR 채널코드 + LDPC(rate ~0.94, length 4376/4352) 직렬연접, ME²PR4 등화 Lorentzian 자기기록 채널 (user density 2.8).
2. 문제: 기존 turbo/convolutional 연접 대비 강력한 블록부호(LDPC)로 자기기록 채널 이득 극대화.
3. 채널모델: precoded ME²PR4 부분응답(PR) 채널 + 등화기 전단 AWGN(variance σ²), SNR ∝ isolated pulse 진폭/σ.
4. 복호 기법 1단: Log-BP(LLR 도메인 belief propagation) - CN/VN 메시지를 `sign·magnitude` 분리 표현, syndrome `z` 확인으로 조기종료.
5. 핵심식: pseudo-posteriori LLR (식3)에서 extrinsic LLR을 추출해 채널 APP 디코더로 피드백 (turbo equalization의 근간).
6. 복호 기법 2단: 이중 반복루프 - LDPC 내부루프(최대 50 iter) + 채널루프(PR채널 APP↔LDPC, 최대 100 iter), LDPC codeword 유효 시 전체 종료.
7. 구현 디테일: LDPC check bit에 guard bit 삽입해 run-length 제약 만족, valid codeword 미발견 시 error flag 제공(undetected error 미관측).
8. 부호설계: H를 codeword-bit↔check 다대다 매핑 permutation S로 보고, 랜덤 생성 후 4-cycle 발견 시 매핑 교환 반복으로 girth 확보, rate>0.95/단길이/col weight>3에서 수렴 저하.
9. 결과: 제안 시스템이 uncoded EPR4 대비 5.9 dB, uncoded PR4 대비 7.5 dB 이득 (BER 10⁻⁵). 채널 iteration 3회면 1회 대비 0.3 dB 추가이득이나 총 LDPC iteration 7배 증가.
10. 한계: HW 미설계, 시뮬만, PR/ISI 자기기록 채널 특화 turbo equalization으로 NAND 무기억 채널과 구조적 부정합. QC 구조 없는 랜덤 col-weight-3 부호.
