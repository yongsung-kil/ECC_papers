> 총평: 부호 구성 단계에서 EMD·short-cycle 메트릭으로 error floor를 사전 스크리닝하는 분석 기법으로, 부호설계(H-matrix)에 이식 시 error-floor 개선에 참고할 이론적 가치는 있으나 NAND 실측·HW 요소는 전무.

### B. 알고리즘 요약

1. 대상은 rate `R=1/2`, length `N=648~1344`의 irregular LDPC 부호 5종(MacKay 랜덤, PEG, IEEE 802.11 Wi1, IEEE 802.16e Wi2, RAVIS)을 AWGN/SPA 디코딩(최대 20 iteration)으로 비교.
2. 유한 길이 irregular LDPC의 실제 성능은 iterative(BP) 디코더의 loopy graph 위 suboptimality로 인한 error floor에 제한되며, 이를 코드 설계 단계에서 미리 식별하는 것이 목적.
3. Error floor는 최소 부호거리(d0)와 Tanner graph의 stopping/trapping set 존재에 의해 좌우된다는 가정 하에, 이를 간접적으로 반영하는 girth·cycle 개수·EMD를 지표로 사용.
4. 핵심 기법: 각 cycle length `g0`~`g0+4`의 개수 `n(g)`와, 각 cycle의 extrinsic message degree `η = Σ(di - 2)` (i=1..g, di=해당 cycle 내 VN degree)를 계산.
5. EMD가 낮은(0에 가까운) cycle이 stopping/trapping set 형성에 취약 - EMD=0이면 stopping set과 동치.
6. 실험적으로 "최소 EMD 값이 3 미만이면 error floor 수준이 낮다(나쁘다)"는 경험적 관계를 관찰; girth를 6→8로 늘려도 EMD가 작으면 error floor 개선 효과 없음.
7. 부수 디테일: cycle 카운팅은 별도 참조 알고리즘([16], Voropaev girth-accounting)으로 계산, 별도 근사/양자화 기법 없음.
8. 학습/최적화 절차 없음 (기존 부호 앙상블들의 사후 구조 분석·비교).
9. 결과: EMD≥3인 부호(PEG, RAVIS 일부)가 EMD<3인 부호(MacKay, Wi1/Wi2) 대비 error floor 영역에서 BER 수 자릿수 개선 관찰.
10. 한계: 무선통신 표준용 저rate(0.5) unstructured/semi-structured 부호 비교이며 HW 구현·복잡도·throughput 분석 없음, QC-LDPC 고rate NAND 부호에 대한 검증 없음.
