### B. 알고리즘 요약

1. 대상은 BEC 상의 binary QC-LDPC로, 패리티검사행렬 `H`가 `l×l` CPM(circulant permutation matrix) 배열이며 소거가 CPM-section(길이 `l`) 단위 버스트로 발생한다고 가정한다.
2. 문제: 랜덤하게 분포한 다중 CPM-phased 소거버스트를 최소 패리티로 완전 정정(최적 η=1)하는 부호를 대수적으로 구성하는 것.
3. 기본 구성: GF(2^s)의 order `n`(prime) 원소 β로 `d×n` RS matrix `BRS(d,n)=[β^{l_ij}]`를 만들고, 각 항을 `n×n` CPM으로 dispersion하여 `HRS(d,n)`을 얻는다.
4. `BRS`의 2×2 SNS-constraint가 `HRS`의 RC-constraint(girth≥6)를 보장하여 length-4 cycle이 없다.
5. Class 1: `d=2`로 `(n^2, n^2-2n+1)` (2,n)-regular RS-QC-LDPC를 구성; Tanner graph의 국소 사이클이 길이 `4n` 단일 cycle임을 증명.
6. 이 단일 사이클 구조 덕에 두 CPM-section에 걸친 mutually semi-solid 소거(최대 `2n-1`개=패리티 수)를 항상 정정 → η=1.
7. 정정 알고리즘은 checksum에서 미지 소거심볼 하나를 modulo-2 합으로 복원하는 peeling 반복(연산=모듈로-2 덧셈뿐).
8. Class 2: `k`개의 local code를 `n^2` global CN(identity 배열 `G(n^2,k)`)으로 묶는 globally-coupled RS-QC-LDPC `(kn^2,(k-1)(n-1)^2)` 구성.
9. Class 2는 local phase↔global phase를 교대하는 2단 iterative 소거복원으로 국소·전역 분산 버스트를 정정, 최대 `n^2+(k-1)(2n-1)`개=패리티 수(η=1).
10. 결과: 예1 `(5329,5184)` rate 0.9737, 예2 `(15376,13500)` rate 0.878, 완전 소거된 local codeword 1개까지 복원. 한계: BEC 전용·순수 이론(HW·AWGN·soft 미포함), non-binary 확장만 언급.

```
| Globally-coupled 2단 소거정정 프로세스 | 대응 없음 | BEC peeling 소거복원, min-sum LLR 디코더(decoder.cpp)와 알고리즘 상이 |

```
