### B. 알고리즘 요약

1. 대상은 GC(generalized constraint) 노드와 SPC 노드를 함께 갖는 protograph 기반 GLDPC 코드(짧은 길이 4종 G1~G4), AWGN 채널.
2. 기존 APP 디코딩은 GC 노드에서 component code 차원에 지수적인 복잡도로 실용성 낮고, GC를 SPC로 expand한 MS는 dense graph에서 성능이 열화됨.
3. 각 GC 노드를 component PCM `C(t)`로 vertical expand하여 VN·SPC만 남는 확장 PCM `H`로 만들고 그 위에서 MS를 수행.
4. 핵심1: NMS - MS의 min 연산에 edge별 가중치 `wc→v`를 곱하는 neural min-sum. Tanner graph를 NN으로 unfold하여 가중치 학습.
5. 가중치 공유(같은 CN, protograph 대응 z개 CN 공유)로 총 가중치 수를 `m×ℓ`로 축소해 HW 친화·학습 안정화.
6. 핵심2: component PCM 최적화 - 코드 불변인 overcomplete PCM `Cfull`에서 `mstd`행을 골라 최적 `S`를 구성.
7. 최적화는 differential evolution(Alg.1, mutation/crossover/selection, FER_MS 기준)로 `S(0)` 결정 후, greedy로 redundant 행을 추가하며 `S(r)` 확장.
8. 학습은 SGD + soft-BER 손실 + multi-loss(전 iteration 반영, discount `η`), 하이퍼파라미터는 Table II.
9. 결과: G1은 NMS+최적 PCM이 FER 10^-5 이하에서 APP 상회, G2는 APP에 근접(ℓ=7.64), G4(RM(32,16))는 MS 대비 약 3.5dB 개선.
10. 한계: HW 미설계, 시뮬(FER)만, 짧은 GLDPC 코드 전용, gate/throughput 수치 없음.
