### B. 알고리즘 요약

1. 시스템은 BPSK 변조된 (dv=3, dc=5) regular LDPC 부호(rate 0.4, length 1055, circulant matrix 구성), AWGN 채널 + 잔류 주파수 오프셋(`f_o r_s < 0.03%`) 및 균일분포 위상 오프셋 `θ_0`을 가정.
2. 풀려는 문제: 서브블록 디코딩(sub-block decoding)은 잔류 주파수 오프셋에 강인하지만, 각 서브블록마다 blind phase estimator(BPE)가 π 라디안의 위상 모호성을 남겨 매 서브블록마다 별도의 phase ambiguity resolution(PAR)이 필요.
3. 핵심 가정: 홀수 차수(odd degree) check node는 위상이 π만큼 뒤집히면 parity check가 깨진다는 성질을 이용(even parity 원리).
4. 핵심 기법 'localized decoding': H-matrix의 열(column) 교환으로 같은 서브블록 내 변수노드만 연결된 'local check node' `c_k`를 생성, 노이즈리스 가정 시 이 체크의 만족 여부로 서브블록의 위상 모호성(0 또는 π) 판정.
5. 핵심 식: local check node의 LLR `L(a_k) = ln[(1+∏tanh(L_i/2))/(1-∏tanh(L_i/2))]` — sum-product 알고리즘의 check-node LLR 계산식과 동일한 형태를 위상 모호성 변수 추정에 적용.
6. 확장: 서브블록 내 여러 local check node의 LLR 기댓값 `E[L(a)]`을 평균해 위상 모호성 추정 신뢰도를 높임(`E[L(a)]>0`이면 a=0, 아니면 a=π).
7. 구현 디테일: 열 교환(column-swapping)만으로 local check node 최대 개수 `C_max = codeword_length/check_node_degree`를 구성, 행 연산(row operation)은 사용하지 않음.
8. 별도 학습/최적화 절차 없음(구성적 열 교환 + 시뮬레이션 검증).
9. 결과: 최대 211개 local check node 확보, 작은 주파수 오프셋(`f_o r_s<0.0003`)에서 coherent 성능 대비 약 0.1~0.3dB 손실만으로 위상 모호성 해소.
10. 한계: HW 설계 없음(순수 알고리즘/시뮬), regular LDPC·특정 길이(1055) 코드에 국한된 실험, LDPC 디코더 자체의 정정능력 개선이 아니라 통신 동기화(위상/주파수) 보조 목적이라 NAND 저장 채널에는 해당 문제 자체가 존재하지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Local check node LLR 계산(sum-product 변형) | 대응 없음 | Prime ECC는 min-sum 기반이며, 위상 모호성 문제 자체가 NAND 채널에 존재하지 않음 |
| H-matrix 열 교환을 통한 local check node 구성 | 대응 없음 | PSK 통신 동기화 특화 기법으로 QC-LDPC base matrix 설계와 무관 |
| Sub-block decoding / phase ambiguity resolver | 대응 없음 | NAND ECC 파이프라인에 위상 동기화 단계가 없음 |

적용 가치: 낮음 — PSK 통신 채널의 위상/주파수 동기화 문제 해결이 목적으로, NAND 플래시 저장 채널 ECC와는 문제 도메인 자체가 다름.
