### B. 알고리즘 요약

1. 대상은 write 해상도 > read 해상도인 저장장치를 모델링한 `symbol-pair read channel`(Cassuto-Blaum)로, 출력이 인접 심볼의 겹치는 쌍 `π(x)_i=(x_i,x_{i+1})`이다.
2. 기존 symbol-pair 코드(대수적 구성/신드롬 복호)의 한계를 넘어 capacity-approaching인 binary LDPC를 이 채널에 쓰기 위한 복호기가 필요하다.
3. 채널 모델: pair 오류가 각 위치 `i`에서 독립적으로 확률 `P<1/2`로 발생 `p(y|x)=∏ p(y_i|π(x)_i)`.
4. ML 복호 `min -ln p(y|x) s.t. x∈C`를 지시함수 `T_{i,(a,b)}(x)`로 재작성하고, `x∈C`를 fundamental polytope `Q(H)`로, `T∈{0,1}`을 `T∈[0,1]`로 완화한 LP (4)를 푼다.
5. 완화 LP는 `2n + Σ 2^{d_j-1}` 부등식으로 표현되며 LDPC(작은 `d_j`)면 `n`의 다항시간에 풀린다.
6. Theorem 1: LP 출력이 정수해면 ML 부호어임을 보증(ML certificate 성질)하며, cycle-free 그래프에서는 LP가 ML 복호와 정확히 일치한다.
7. 성능 분석을 위해 all-zeros 가정 성립을 Theorem 2로 증명(오류확률이 입력 부호어에 무관).
8. fractional pair distance `d_fp = min_{x∈V_Q\{0}} Σ max{x_i,x_{i+1}}`를 정의, `d_f ≤ d_fp ≤ d_p` 관계를 보인다.
9. Theorem 3: LP 디코더는 최대 `⌈d_fp/2⌉-1` pair 오류를 정정하며, facet별 볼록최소화로 `d_fp`의 하한을 계산한다.
10. 결과: variable degree 3, check degree 4 랜덤 LDPC(길이 ≤100)에서 `d_fp`가 블록길이에 선형 증가하고 fractional distance의 약 1.8배. HW 미설계, BER 곡선 없음, pair-error 독립 가정 의존.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LP 완화 기반 복호(4) | 대응 없음 | Prime ECC는 min-sum message passing이라 LP 디코더 계열 자체가 부재 |
| symbol-pair read 채널 모델 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | 우리는 AWGN/RBER soft-read; pair-read 채널은 미지원(개념만 원거리 대응) |
| fractional pair distance 분석 | 대응 없음 | 부호 거리 이론 분석 도구로 코드베이스 대응 모듈 없음 |

적용 가치: **낮음** — LP 디코딩과 symbol-pair 채널 모두 min-sum 기반 NAND binary QC-LDPC 엔진과 계열이 달라 떼어 이식할 구성요소가 없다.
