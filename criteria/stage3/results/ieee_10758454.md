### B. 알고리즘 요약

1. 시스템: 5G mMTC(짧은 코드) 타깃, 5G LDPC 코드 [N=600, K=120] (Kb=6, k=120), rate 1/5, 40 punctured bits, BPSK, AWGN 채널.
2. 문제: LDPC는 짧은 길이에서 성능 저하가 크고, neural BP/MS 디코더는 성능은 개선하지만 연산·메모리 부담(32-bit float 메시지)이 커서 IoT 저전력 기기에 부적합.
3. 채널모델: `y = x + z`, `z ~ N(0, σ²I)`, LLR `L_i = 2y_i/σ²` (표준 AWGN LLR).
4. 핵심 기법 1단(trainable min-sum): 기존 min-sum에 스케일 `ω_{j→i}`, 오프셋 `β_{j→i}`, 채널 LLR 가중치 `ω^{ch}_{i→j}`를 학습 파라미터로 추가 — `R_{j→i} = sign(·)·ω_{j→i}·max[min|Q|-β_{j→i}, 0]`.
5. 이 스케일/오프셋은 min-sum 근사 오차 보정과 짧은 girth의 영향 완화를 위해 학습되며, Tanner graph를 unfold한 sparse NN 구조(변수/체크 레이어 반복)로 학습.
6. 확장(residual connection): variable-node 업데이트에 과거 iteration 메시지를 가중합으로 더하는 residual term `Σ ω_{i→j}^{(l-t)} Q_i^{l-t→j}` (깊이 D)를 추가해 vanishing gradient 완화 및 수렴 안정화.
7. 부수 트릭(neural quantizer): differentiable soft-staircase 함수 `f(r;S;σ²)`로 메시지를 양자화(온도 파라미터 σ²=t^η로 anneal), check/variable node에 각각 별도 양자화기(CQuant 4-bit, VQuant 6-bit) 적용, overparameterization(hidden vector 내적)으로 양자화 레벨 학습.
8. 학습 절차: 2단계 학습(batch 128→32, lr 1e-3→1e-4), zero-codeword + AWGN(SNR U(-5.5,-5.0)dB) 학습, BCE loss, 20 iteration 고정.
9. 결과: NN-20(비양자화)이 SP 대비 FER=1e-2에서 0.125dB 이득, 공동학습 양자화(QNN-20)는 NN-20 대비 0.3dB 이내 손실(균등양자화 대비 약 0.5dB 개선), 메시지 메모리 81.25%(6bit)/87.5%(4bit) 절감(32bit float 대비).
10. 한계: HW 미설계·시뮬레이션뿐이며 특정 5G 짧은 LDPC(N=600, rate 1/5)·AWGN 채널에 국한, iteration 20 고정 학습이라 NAND의 가변 iteration/조기종료 구조에는 별도 검증 필요.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| trainable min-sum (scaling/offset) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | min-sum CN 연산에 학습된 scaling/offset factor 적용 아이디어로 연결 가능 |
| 채널 LLR 가중치 / VN 업데이트 | `decoder.cpp` `VN_Cal_HD()`, `ecc_data.h` `PARAM_LLR` | variable-node update에 학습 가중치를 적용하는 부분과 대응 |
| residual connection (과거 iteration 메시지 활용) | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | 기존 iteration 루프 구조에 과거 메시지 재사용 로직 추가가 필요 - 현재 미보유 |
| neural quantizer (soft-staircase, 비균등 양자화 레벨) | `channel.cpp` `Set_LLR_Th()`, `ecc_data.h` `PARAM_LLR` | LLR threshold 테이블 설계에 학습 기반 비균등 양자화 레벨 아이디어 참고 가능 |
| GNN/hypernetwork 기반 디코더(비교 대상, 논문 내 인용) | 대응 없음 | 본 논문 자체 기법이 아닌 관련연구 인용부 |

> 적용 가치: 중간 | min-sum 스케일/오프셋 학습과 비균등 양자화 레벨 설계 아이디어는 decoder.cpp/channel.cpp에 개념적으로 이식 가능하나, 신경망 학습 파이프라인 자체는 Prime ECC의 bit-exact C++ 시뮬레이터 구조와 이질적이라 재구현 부담이 있음.
