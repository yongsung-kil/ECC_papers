### B. 알고리즘 요약

1. 시스템: 화성-지구 synodic period(780일) FSO 채널, L-ary PPM(LPPM) 변조, staircase LDPC 코드(컴포넌트 코드 C1: `N=128,K=64`, C2: `N=96,K=48`) 사용.
2. 문제: 태양 섬광(solar scintillation)으로 인한 Gamma-Gamma(GG) 페이딩이 심할 때 기존 BP-SWD(sliding window decoder)는 error floor가 발생.
3. 채널 모델: 무섬광(AWGN)/섬광(GG, 약함·중간·강함)/아웃티지 3상태로 분류, 섬광지수 `σI²` 임계값(0.01, 0.99)으로 구분.
4. 핵심 기법1(SC-OSD): staircase 구조의 재부호화 비트(reencoded bits)에 신뢰도 계수 `μ>1`을 곱해 OSD의 신뢰도 시퀀스 `o_i`를 수정, MRB(most reliable basis) 정확도를 높임.
5. 핵심 식: TEP(test error pattern) 스킵 기준 `θ_TEP`으로 likelihood `L_e`가 임계값 `L_thr` 미만일 때만 재부호화 수행 → 불필요한 reencoding 생략, 복잡도 절반 감소.
6. 핵심 기법2(SOSD-SWD): SOSD(soft-output OSD)를 컴포넌트 디코더로 하는 이중 반복(inner iteration `I`, outer iteration `J`) SWD 구조, 윈도우 크기 `w`.
7. 부수 트릭: 이전 블록 추정치의 신뢰도를 정량화하기 위한 μ 파라미터 튜닝(SNR에 따라 최적값 변화), extrinsic LLR `ψ^e` 3단계 정보 교환(초기화/블록간/반복간).
8. 이론 유도: 정렬통계량(order statistics)의 PDF/CDF 유도 및 중심극한정리 기반 가우시안 근사로 BLER 하한 `P_lower` 및 임계값 도출.
9. 결과: SC-OSD가 OSD 대비 reencoding 연산 약 50% 감소(Eb/N0=6dB), 강섬광 조건에서 SOSD-SWD가 BP-SWD 대비 error floor 극복하고 미부호화 대비 약 30dB 이득(BER 10⁻⁴ 기준).
10. 한계: 알고리즘/시뮬레이션 수준(HW 미설계), staircase code 구조 특유의 재부호화 절차에 종속적이며 컴포넌트 코드가 QC-LDPC인지 불명확.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SC-OSD (OSD 기반 디코딩) | 대응 없음 | Prime ECC는 min-sum 기반 BP 디코더이며 OSD(Gaussian elimination + TEP 재부호화) 알고리즘 자체가 없음 |
| TEP 스킵 기준(복잡도 절감) | 대응 없음 | 조기종료는 Partial/Full CRC(`partial_CRC.cpp`, `full_CRC.cpp`) 방식과 원리가 다름(TEP 우도 임계값 vs CRC) |
| SOSD-SWD 이중 반복 스케줄 | 대응 없음 | staircase 블록간 sliding window 구조로 Prime ECC의 단일 QC-LDPC 블록 디코딩 루프(`decoder.cpp` `LDPC_Decoder()`)와 구조 자체가 다름 |
| staircase LDPC 부호 설계 | `ecc_top.cpp` `Load_PCM()` | staircase 구조는 BCH/LDPC 컴포넌트 결합으로 QC-LDPC H-matrix 로드 방식과 무관 |

적용 가치: 낮음 — 심우주 광통신용 staircase code + OSD 소프트 디코딩 기법으로 NAND binary QC-LDPC min-sum 디코더 구조와 알고리즘/응용 모두 이질적이며 이미 보유한 기법과 중복되지 않으나 이식 난이도가 매우 높음.
