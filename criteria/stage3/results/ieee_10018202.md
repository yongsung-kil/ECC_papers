### B. 알고리즘 요약

1. 시스템/부호: rate-1/2 `(3,6)`-regular, rate-2/3 `(3,9)`-regular LDPC(길이 1024~1536)를 payload로, BPSK-AWGN 채널, SPA/BP 복호.
2. 문제: GC-LDPC/product/terminated SC-LDPC 결합부호는 성능 개선하나 추가 parity/global check로 rate 손실 발생, 대역폭 제약 시 불리.
3. 핵심가정: BIOS 대칭채널에서 coset 부호는 원부호와 동일 WER, syndrome이 genie로 주어지면 결합부호와 동성능.
4. 핵심기법1(free-ride): 부가비트 `f`를 free-ride 부호 `Cf[n,ℓ]`로 인코딩한 `g`를 payload 코드워드 `v`에 XOR 중첩 `c=v+g`(식1), 추가 전력·대역폭 없이 소수 부가비트 전송.
5. 핵심식: syndrome 채널 `s=gH^T`(payload 간섭 `vH^T=0`으로 소거) 관점의 accessible capacity `Ca=(1/n)maxI(S;Y)`(식6) 상·하한 유도로 신뢰전송 타당성 입증.
6. 핵심기법2(적용): 결합 check 비트(GC global check / product column syndrome / SC partial syndrome)를 부가비트로 free-ride 전송해 rate 손실 없이 GC-LDPC·partial product-LDPC·terminated SC-LDPC 구성.
7. 구현 디테일: free-ride 부호는 신뢰도 높은 1차 RM 부호 사용, `Gf H^T=GRM`(식4), L-fold Cartesian product+interleaver로 diversity gain.
8. 복호: 2단 successive cancellation - (1) soft syndrome `Γ(cH^T)`(식2)로 RM 복호해 부가비트 복원 후 효과 제거(식3), (2) payload LDPC(SPA) 복호, 필요시 global/column/window 복호.
9. 결과: implicit GC-LDPC 최대 0.8dB coding gain, partial product-LDPC는 WER `10^-2→10^-6`@~2dB(최대 1.0dB), terminated SC-LDPC는 conventional 동성능이나 rate 손실 0, 모두 성분부호와 동일 rate.
10. 한계: HW 미설계(free-ride enc/dec 복잡도 정성 언급만), 전송링크·대역폭 제약 시나리오 지향, 대부분 `(3,6)`-regular·low row-weight 가정, AWGN 시뮬만.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| payload LDPC BP/SPA 복호 | `decoder.cpp` `LDPC_Decoder()` | LDPC 복호 자체는 대응되나 논문은 full-tanh SPA(min-sum 아님)·regular 부호, 우리 min-sum QC-LDPC와 이질 |
| soft syndrome `Γ(cH^T)` (tanh 계산) | 대응 없음 | full-`tanh` 기반 syndrome LLR 계산, 우리 min-sum 근사와 불일치 |
| RM 기반 free-ride enc/dec | 대응 없음 | RM 부호 중첩전송은 결합부호 구성용, NAND ECC 코드베이스에 대응 모듈 없음 |
| 결합부호 구성(GC/product/SC H-matrix) | 대응 없음 | 특정 binary QC-LDPC H-matrix 고정, 결합/product 구성 미대상 |
| column syndrome→BCH HDD(product) | 대응 없음 | 곱부호 열복호(BCH)는 우리 QC-LDPC 단일부호 구조와 무관 |

적용 가치: 낮음 — binary LDPC 기반이라 관련성은 간접이나, free-ride/coupling으로 부가비트를 rate 손실 없이 전송하는 것이 목적이며 부호설계·min-sum 디코더·HW 어느 레이어에도 떼어 이식할 핵심요소가 없고, syndrome LLR도 full-tanh 기반이라 우리 min-sum과 불일치.
