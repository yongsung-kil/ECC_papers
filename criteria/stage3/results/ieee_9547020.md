### B. 알고리즘 요약

1. 시스템: DTMB-A 방송용 LDPC, rate 1/3(신규 저rate), lifting size `Z=32,64,128,512` → code length `n=3780,7560,15360,61440`, QC Raptor-like 구조.
2. 문제: DTMB-A는 rate 1/2·2/3·5/6만 지원, 저 spectral efficiency/저 C/I(WiB) 시나리오용 저rate 부호가 없음. ATSC 3.0 Type-A는 length-scalability 없고 error floor 존재.
3. 구조: PCM `H=[[A,C,0],[D,E,I]]`, `HU=[A,C]`=pre-code, `HL=[D,E]`=Luby code. `A`=full-1(m1=3), `C`=full-1 lower-triangular, `E` 최우열 full-1로 degree-1 VN의 error floor 완화.
4. 기법1(base matrix): 자유도는 Luby code에 두고 quasi-regular로 가정, MET-DE로 다변수 degree distribution 최적화.
5. MET-DE: 같은 base matrix edge에 대응하는 edge만 같은 type으로 분류해 PDF를 edge-type별로 추적 → 구조적 LDPC의 asymptotic 성능을 SET-DE보다 정확히 예측.
6. 결과(threshold): 80×120 base matrix가 ATSC 3.0(120×180)보다 SNR loss 작음(BER=1e-10 기준, AWGN/Rayleigh, t=25/1000 모두).
7. 기법2(lifting): trapping set 유발 최단 cycle 제거. MET용 cycle을 Type-1(pre-code 내부, 가장 유해), Type-2(pre-code↔Luby), Type-3(Luby 내부, 최소 유해)로 분류해 유해도 순 정렬 제거.
8. progressive lifting: `Z=2`부터 전수 탐색 후 `Z=4,8,...,128`로 호환 확장 → 탐색공간 축소 + length-scalability 확보.
9. 결과(BER/BLER): 유사 길이 ATSC 3.0 Type-A 대비 waterfall·error floor 모두 우수, 4k/8k 길이에서 뚜렷한 error floor 없음, AWGN·Rayleigh 모두 robust (iteration 30, layered decoding).
10. 한계: 순수 부호설계+DE 이론, HW 미설계, 저rate·방송채널 전용, gate/throughput/복잡도 수치 없음.
