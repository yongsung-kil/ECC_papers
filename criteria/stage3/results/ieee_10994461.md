### B. 알고리즘 요약

1. 시스템: BPMR(비트패턴 자기저장) 채널에 QC-LDPC 코드([52] Tanner circulant) + serial MAP 검출을 결합, 데이터 페이지 `1200×1200 bit`, H 예시 `123×1353`.
2. 문제: AD 증가로 2D 간섭(ISI 다운트랙·ITI 크로스트랙) 심화, 기존 sum-product용 H행렬의 suboptimal 구조가 복호성능을 저하.
3. 모델: 2D Gaussian pulse 채널 식(2), TMR·position fluctuation·size jitter·AWGN noise 모델, `Lx=Lq=1`.
4. 핵심기법1(검출): GPR 타깃 `G`·equalizer `F`를 식(12~14)로 추정 후, horizontal MAP(6심볼/36상태 trellis) → vertical MAP(2심볼/4상태) serial 검출.
5. 핵심식 의미: 분기확률 `γk=exp(-(z-Zi)²/2σ²)·Pe(Ri)`로 extrinsic정보 `Pe` 반영, LDPC 출력 LLR을 `Pe(a)=li(t)`로 vertical MAP에 피드백.
6. 핵심기법2(부호설계): shift 규칙 `n_{j,k}=κ·n_{j,0}·n_{0,k}-(1-κ)`, `κ=0`(확률 α)이면 zero-matrix 삽입 → 1-bit 밀도 감소(MALT form).
7. 부수트릭: Algorithm 2로 행별 1-bit 히스토그램 정규화(식22)·이산 gradient(식23) 평균 `β<0.001`가 될 때까지 재생성해 1-bit 균등 재분포.
8. 최적화: α는 0~1 스윕, `α=0.7`에서 BER 최적; short cycle(특히 length-4) 감소로 belief propagation 개선 주장.
9. 결과: 피드백 1~2회로 BER 포화, BER `10^-6`에서 기존 대비 `0.7 dB` 이득; position fluctuation 18%까지 강건.
10. 한계: HW 미설계·시뮬만, BPMR 2D 채널·MAP 특화, sum-product(full tanh) 복호, rate/AD 가변 예시라 고정 NAND 부호로의 직접 이식 부적합.
