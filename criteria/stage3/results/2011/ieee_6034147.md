### B. 알고리즘 요약

1. 대상: GF(2) 위 QC 부호로, PCM `H`가 `e×e` circulant 행렬의 `m0×n0` 배열(`e`는 홀수, `q-1`이 `e`로 나누어짐).
2. 문제: QC-LDPC PCM은 redundant row가 많아 rank(=코드 dimension)가 하한만 알려져 있고, 정확값은 매 부호마다 Gaussian elimination(컴퓨터)으로 구해야 함.
3. 핵심 아이디어: circulant은 공통 Vandermonde 변환 `V=[α^{-ij}]`로 대각화됨 (Lemma 1) — Fourier 변환의 행렬판.
4. 배열 전체에 `diag(V,...)H diag(V^{-1},...)`를 적용하면 `HF`는 대각행렬들의 배열이 됨 (Lemma 2).
5. 행/열 치환 `π_{m0}, π_{n0}`로 `HF,π = diag(B0,...,B_{e-1})` (블록 대각) 형태로 변환 — rank 보존.
6. 핵심 결과 Theorem 3: `rank(H) = Σ_{t=0}^{e-1} rank(Bt)`, circulant permutation/zero 배열이면 `Bt = B^{∘t}`(Hadamard t승, base matrix `B` 기준).
7. 세 부류에 적용: (a) field element 거듭제곱 base `B=[β^{ij}]` → `rank=e/p + (e-e/p)min{m0,n0}` (식 2).
8. (b) prime field 원소 base `W=[ij]` → `rank = 1+(p-1)min{m0,n0}` (array code 특수화 포함).
9. (c) field element 차이 base `B=[δi-ωj]` → Lucas 정리로 `rank(Bt) ≤ min{m0,n0,2^{w(t)}}`, 특수 경우 `3^r` 계열 폐형식.
10. 한계: 이론뿐 — 디코더/HW/BER/양자화 전혀 없음. rate/length는 구성 파라미터 종속으로 확정값 없음.
