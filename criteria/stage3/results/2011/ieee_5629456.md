### B. 알고리즘 요약

1. 시스템: 2-bits/cell all-bit-line MLC NAND, rate-19/20 (34520, 32794) QC-LDPC(column weight 4, girth 6), min-sum 복호, gray code 매핑.
2. 문제: soft-decision LDPC는 fine-grained sensing이 필요해 sensing latency·page buffer·전송지연이 급증 → sensing 정밀도를 최소화하면서 정정성능 유지 필요.
3. 모델: 문턱전압 분포를 erased(Gaussian)/programmed(uniform)로 모델링, cell-to-cell 간섭을 이웃 셀 프로그램 후 shift `ΔV_int = Σ γ_k·ΔV_k`(coupling ratio γ)로 정의.
4. 핵심 기법 1: 간섭 후 victim-cell 문턱전압 분포 `P(V)`를 uniform 합 분포의 closed-form으로 유도(식 (17)-(19)), 이를 LLR 계산의 pdf로 사용.
5. 핵심 식: `LLR_k = ln[Σ_{s∈S_k^1} P_s(V) / Σ_{s∈S_k^0} P_s(V)]` (식 (7)) — 각 state pdf 기반으로 bit LLR 산출, table lookup 선계산.
6. 핵심 기법 2: 엔트로피 `H(V)`가 큰 영역(인접 state 경계=hard-decision 기준전압 부근)에만 sensing을 집중하는 nonuniform sensing.
7. 트릭: dominating overlap region을 확률비 `η`(=512)로 결정, LLR을 6bit로 양자화, 참조전압 조합별 LLR을 사전계산해 런타임 lookup.
8. 확장: 간섭셀도 sensing하면(known interference) victim 분포를 좁혀 더 정확한 LLR — consecutive page read 시 자연히 확보, reverse programming으로 read latency 제거.
9. 결과: 15-level nonuniform ≈ 31-level uniform 성능(~50% sensing latency 절감), 9-level nonuniform ≈ 15-level uniform(~40% 절감), multistep progressive sensing으로 추가 43%·27% 절감. LDPC가 BCH 대비 PER 우위.
10. 한계: HW 디코더/sensing 회로 미구현(시뮬만), coupling ratio를 상수로 근사, 특정 2bit/cell·간섭모델 가정 의존.
