### B. 알고리즘 요약

1. 대상: 일반 (G)LDPC 부호의 Linear Programming(LP) 복호와 minimum/fractional distance 하한 계산; 실험은 Gallager (3,6)-regular 앙상블(length 60~200), 예제 GLDPC rate 1/7.
2. 문제: LP 복호는 ML certificate 성질(정수해=ML)이 있으나 복잡도가 BP/min-sum 대비 크고, fractional distance 정확 계산도 일반 LP solver로 다항이지만 이차보다 나쁨.
3. 기여1(relaxation tightening): Tanner graph의 check node들을 병합(merge)하면 LP polytope가 finer relaxation이 되어 복호 성능·distance 하한 개선(Prop.1-5); cycle-free 집합 병합은 무변화라 cycle 포함 후보만 병합.
4. 핵심 도구: Vontobel-Koetter류 선형복잡도 iterative LP 디코더(`Algorithm 1`, log-sum-exp로 max 근사, trellis forward/backward 재귀)를 GLDPC로 확장.
5. 기여2(minimum distance 하한): `Algorithm 2` — 각 check node에 nonzero local codeword를 강제한 coset code 위에서 Algorithm 1을 반복 실행, weak duality로 하한 확보, 총 복잡도 `O(n^2)`.
6. bar method: 거리 2의 check node 쌍에 local codeword를 공동 강제해 하한을 refine(bisection으로 최고 attainable bar 탐색).
7. 기여3(fractional distance 하한): polytope facet별 최소 norm을 penalty 항 Problem-P_l로 환원해 `O(n^2)`에 하한 산출(기존 LP solver 기반보다 저복잡도).
8. 기여4(fundamental polytope): double description method(cdd+)로 GLDPC·nonbinary 부호의 fundamental polytope facet 표현 도출; (7,4) Hamming 지역부호는 70열 행렬.
9. 결과: BSC 시뮬서 check node merging이 LP 복호를 약간 개선하나 여전히 BP보다 열세(crossover>0.045); distance 하한은 length up to 200에서 exact 값에 근접.
10. 한계: 순수 이론, HW 미설계, LP 복호 자체가 min-sum 대비 고복잡·저성능, non-binary/GLDPC 포함.

| nonbinary/GLDPC polytope | 대응 없음 | Prime ECC는 binary QC-LDPC 전용 |

적용 가치: **낮음** — LP 복호 프레임워크와 distance 하한 계산 이론으로, NAND용 binary min-sum QC-LDPC의 복호기/HW에 떼어 쓸 알고리즘이 없고 non-binary/GLDPC 지향이라 필터상 이식성 하.
