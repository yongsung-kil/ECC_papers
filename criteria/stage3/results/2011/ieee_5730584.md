### B. 알고리즘 요약

1. 시스템: BPSK 입력 partial-response(PR/ISI) 채널의 ML 검출을 memoryless 이진복호 문제로 변환, LDPC(rate-1/4, length-200, (3,4)-regular) Tanner graph와 결합.
2. 문제: BCJR/SOVA 기반 등화는 채널 메모리 길이에 지수복잡도, LP decoding은 ISI의 quadratic 목적함수(nonconvex)에 직접 적용 불가.
3. 모델: PR 채널 `y=Hx+n`(Toeplitz H), ML은 IQP(정수 이차계획). auxiliary 변수의 modulo-2 합을 parity-check처럼 도입.
4. 핵심기법: quadratic 목적함수를 선형화 - `t_{i,j}=x_i XOR x_j` state bit 도입해 IQP를 이진 memoryless 복호로 변환, code layer + PR layer의 단일 Tanner graph 구성.
5. 핵심식: PR layer check node는 degree-3(정보비트 2 + state비트 1), state bit 계수 `q`는 채널 임펄스응답만의 함수, 정보비트 계수는 matched-filter 출력(LLR 역할).
6. 확장: Cycle Condition(CC) - 모든 cycle이 짝수 개 negative check를 포함하면 uncoded LP relaxation이 항상 ML 정수해(=proper channel), 2-D ISI에도 적용.
7. 트릭: Selective Message Passing(SMP) - PR layer의 짧은 cycle(길이 6) 양성 피드백 억제를 위해 정보비트 노드에서 layer별로 메시지 선택 결합.
8. 분석: LP distance `d` 척도 도출 - `d≤0`이면 improper(SNR 무관 nonvanishing WER), Viterbi trellis로 dominant error event 계산.
9. 결과: proper(CH1)에서 LP=ML, EPR4(improper) uncoded LP는 SNR 무관 BER floor. coded MSA는 LP 대비 EPR4에서 BER 단조감소, SPA는 MSA 대비 약 2dB gain, SMP는 MSA에 추가 0.5dB.
10. 한계: HW 미설계, 시뮬만. ISI/자기기록 채널 특화로 부호복호 자체는 표준 MSA/SPA.

### C. Prime ECC 관련 모듈 핀포인트

|---|---|---|

| PR/ISI 채널 등화 그래프 계층 | 대응 없음 | Prime ECC 채널모델(AWGN/RBER, `channel.cpp`)은 ISI 없음, PR layer 결합 불필요 |
| LLR/양자화 | `channel.cpp` `Set_LLR_Th()` | 논문은 matched-filter 계수를 LLR로 쓰나 NAND soft-read 양자화와 무관 |
