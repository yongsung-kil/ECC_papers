> 총평: circulant 분해로 큰 최소거리·낮은 error-floor를 보증하는 다양한 cyclic/QC-LDPC 부호군을 대수적으로 생성하는 이론 논문으로, 고rate·저error-floor 부호설계 아이디어는 참고 가치 있으나 Prime ECC의 고정 H-matrix 이식엔 재검증 부담.

### B. 알고리즘 요약

1. 대상: GF(q) 위 n×n circulant `W=Ψ(w)` (첫 행 w가 생성자)로 주어진 cyclic 부호를 분해해 cyclic·QC descendant 부호군 생성.
2. 문제: cyclic FG-LDPC(EG/PG-LDPC)는 큰 최소거리·낮은 error-floor의 장점이 있으나 부호군이 작음 → 다양한 length/rate 확보 필요.
3. 핵심 기법(분해): n=c·l로 인수분해 후 column/row permutation `π`(식 (2)(3))로 W를 l×l circulant의 c×c doubly-cyclic array `Φ(w)`로 변환(Theorem 1).
4. descendant 종류: type-1(단일 descendant circulant `Ψ(wi)`의 null space, 길이 l), type-2(여러 circulant 열결합 `Hcol,k`, 식 (5)), type-3(masking으로 일부 circulant→zero matrix 치환 후 `π⁻¹` 역변환).
5. 생성다항식 특성화: type-1의 근은 `σe,j=0` 조건으로 결정(Theorem 2), type-2는 type-1 생성다항식들의 LCM(Theorem 3) — 근으로부터 descendant 부호 스펙 예측.
6. FG-LDPC 적용: 2D Euclidean geometry EG(2,q) incidence vector로 RC-constraint 만족 circulant `HEG` 구성(girth≥6, 최소거리 ≥ column weight+1), 이를 분해해 cyclic·QC-EG-LDPC 부호군 획득.
7. CPM 분해·masking: `HEG`를 CPM/ZM의 c×c array `π(HEG)cpm`로 분해(Theorem 4), γ×ρ subarray 추출 + masking matrix `Z(γ,ρ)`⊗base로 edge/short-cycle 축소 → 정규/비정규 QC-LDPC 생성.
8. 다차원 확장: EG(m,q), m≥3에서 column/row block splitting(e-fold expansion)으로 더 sparse한 array `MEG` 구성.
9. 결과: 예제 다수 — (4095,3367 rate 0.82), (1365,765 rate 0.56 min-dist 17), (32768,31747 rate 0.969, Shannon 0.6dB), (65536,32768) QC-EG가 DVB S-2 능가, (8176,7156 rate 0.875)은 BER 10⁻¹⁴까지 error-floor 없음(HW 디코더 제작됨), scaled MS가 SPA와 근접·빠른 수렴.
10. 한계: 순수 대수적 구성 이론(SPA/MS 시뮬만), HW 세부·gatecount·throughput 미기재, EG/PG 특정 기하구조 종속.
