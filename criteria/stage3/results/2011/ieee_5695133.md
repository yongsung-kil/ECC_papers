### B. 알고리즘 요약

1. 대상: LDPC block code에서 LDPC convolutional code(SC-LDPC)를 유도하는 두 unwrapping 기법(Tanner / Jiménez-Feltström-Zigangirov, JFZ)을 graph-cover 관점에서 통합.
2. 문제: 두 기법이 서로 무관해 보이나 실은 tightly connected이며, convolutional 변환 시 나타나는 "convolutional gain"의 근원과 비용이 불명확.
3. 핵심도구 `graph cover`: proto-matrix를 GCC1(중간행렬 `A⊗I` 후 `A_l`로 치환)·GCC2(copy-and-permute) 두 방식으로 M-fold cover를 구성, 두 그래프는 isomorphic.
4. Tanner unwrapping = quasi-cyclic block을 modulo 연산 제거로 time-invariant convolutional로 "cut"; JFZ unwrapping = PCM을 `H=Σ H_i` 분해 후 diagonal cut. Tanner는 JFZ의 특수경우임을 예시로 증명.
5. 반복 graph-cover(iterated GCC1+GCC2)로 time-varying LDPC convolutional code 생성(Definition 17): 유한 permutation로 GCC1 -> bi-infinite Toeplitz로 GCC2 -> 음의 time index shortening.
6. "random cut": diagonal cut 대신 임의 분할, 계산복잡도(degree distribution) 불변이나 constraint length 증가로 작은 추가 이득.
7. 결과: `(3,5)`-regular 기반 time-varying convolutional code가 BER 1e-5에서 underlying block 대비 2.0~2.8dB gain (Fig.7); irregular proto-graph 기반은 0.6~0.9dB gain (Fig.9).
8. 분석: unwrapping이 Tanner graph 짧은 cycle을 "break"(girth 증가)하고 pseudocodeword를 감소시켜 고SNR·error-floor에 유리(pseudoweight 보존 lemma).
9. 비용: 계산복잡도/bit은 동일(node degree 동일)하나 pipeline 디코더가 iteration을 공간 다중화 -> 메모리·지연이 M배(block은 1배).
10. 한계: AWGNC/BPSK 시뮬 및 이론뿐, HW 미설계, 양자화·min-sum·NAND read 논의 없음, 저-중SNR gain의 정량 해석은 open.
