### B. 알고리즘 요약

1. 대상: 임의 유한길이 LDPC의 parity-check matrix `H`에서 소형 error-prone substructure(EPS: stopping/trapping/absorbing set)와 fully absorbing set(FAS)를 **완전열거**.
2. 문제: 소형 EPS 완전열거는 NP-complete이며, error-floor(10^-12~10^-15, data storage 요구)는 Monte-Carlo로 측정 불가 → 트래핑셋 열거로 준해석적 추정 필요.
3. EPS 정의(그래프 이론적, 채널·복호기 독립): 유도 부분그래프에서 변수노드 `a`개, 홀수차수 이웃 check `b`개인 `(a,b)` 집합, `s=Ht`의 weight가 b.
4. 핵심: branch-and-bound로 candidate vector `f∈{0,1,*}^n`(*=미결정)를 이진트리 탐색, 각 노드에서 `α=|supp(f)|`, `β=wt(H⊗f)` 계산.
5. Bound step(Alg.1): 하한 `ω(f)`를 IP `min Σx_i s.t. Ĥx≥1`을 LP 완화(`x≥0`)로 풀어 구함 → `α+β+ω(f)>amax+bmax`이면 가지치기.
6. Branch step: 가장 많은 이웃 CN이 가장 적은 미결정 VN에 연결된 VN 위치 `p`를 lexicographic 순서로 선택.
7. FAS 알고리즘(Alg.2): FAS 조건 `d_k−|U_k|>|U_k|`을 강제하는 별도 LP 하한 `μ(f)` 사용, 복잡도는 `amax`만으로 결정(Alg.1은 `amax+bmax`).
8. QC 가속: t-section QC 부호는 EPS의 cyclic-shift도 EPS이므로 STACK을 t개 고정위치 candidate로 초기화 → 탐색공간 대폭 축소(Tanner 155부호에서 6.8×10^6→6.0×10^5).
9. 결과: Tanner(155,64), M816/M1008, CCSDS(256,128), PEGR(504,252)에서 완전열거 검증; FAS가 종종 trapping set 집합과 정확히 일치. PEGR504 FAS 탐색을 EFSA[13] 6h55m→3min(Alg.2)로 2오더 이상 단축.
10. 한계: 순수 부호분석 도구(GLPK LP 솔버 오프라인), 복호기·HW·부호설계 절차 자체는 제공 안 함.
