### B. 알고리즘 요약

1. 시스템: DPSK 변조 + flat Rayleigh fading 채널에서 QC-LDPC 성능 평가, rate 0.89, block length 1332.
2. 문제: 무작위 구성 LDPC는 short cycle/error-floor 문제와 높은 메모리 요구가 있어 구조적(QC) 고girth 부호가 필요.
3. 채널 모델: multipath 임펄스 응답 `h(t,τ)` 기반 flat Rayleigh fading, Doppler shift 포함, DPSK 복조.
4. 핵심 기법: 각 H 행을 `s`개 sub-row로 분할(row-splitting)하고 "ones"를 rotating 방식으로 재분배하여 RC-constraint(두 행/열이 최대 1개 공통 1) 만족하는 `(γ,ρ)` regular 부호 생성.
5. 핵심 식: circulant permutation matrix 배열로 H 구성, shift 값만 저장해 메모리를 `1/b` 배 축소.
6. 확장: 여러 rate 지원 위해 행들의 UNION 연산으로 multiple coding rate(0.25/0.5/0.89)를 한 구조에서 지원.
7. 구현 디테일: weight-1/weight-2 circulant을 다항식(첫 행 1의 위치)으로 표현, shift 값만 저장.
8. 최적화: girth conditioning + code optimization으로 short cycle 제거, girth-12 부호 확보.
9. 결과: 무작위 부호 대비 0.1dB coding gain, BER 10^-6에서 Shannon-limit로부터 1.3dB, error floor 완화; FPGA(Xilinx Spartan-3E XC3S500E) 프로토타입.
10. 한계: 무선 Rayleigh/DPSK 특화, HW 정량치(gate/throughput/면적) 본문 미제시, min-sum 등 디코더 알고리즘 기여 없음.
