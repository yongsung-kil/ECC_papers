### B. 알고리즘 요약

1. 시스템: 2-user binary-input Gaussian MAC `Y = h[1]X[1] + h[2]X[2] + N`, 채널이득 `h[1],h[2]` 송신단 미지, 두 유저가 동일 rate `R`(예제 rate 0.5) LDPC 사용.
2. 문제: 채널이득을 모를 때 이론적으로 가능한 모든 이득에서 신뢰통신하는 universal 부호 필요 — 특정 이득에 최적화된 irregular LDPC는 비대칭 이득에서 성능 급락.
3. 모델: joint iterative decoder — 두 유저 Tanner graph를 function node로 연결, 비대칭 function-node 메시지규칙 `m_{f→v}` (식 3,4), 유저교대 반복.
4. 핵심 기법: density evolution(DE)으로 joint decoder의 BP-ACPR(달성 가능 채널이득 영역) 계산, MAC-ACPR과 비교.
5. 핵심 도구: GEXIT curve `g(α)`와 area theorem으로 MAP boundary의 outer bound 계산 (`∫gBP dα = 2r(λ,ρ)`로 MAP threshold `ᾱ` 도출).
6. 확장: spatial coupling `(l,r,L,w)` ensemble(protograph 기반)로 부호 구성, 위치별 variable/check node 배치 + 무작위 permutation.
7. 구현 디테일: `(3,6,L)`은 wiggle로 threshold saturation 불완전 → smoothing 파라미터 `w` 포함 `(l,r,L,w)` 사용, coupled DE 식(8) fixed-entropy DE로 안정/불안정 fixed point 계산.
8. 학습/최적화: 별도 최적화 없음 — DE/GEXIT 수치 계산으로 threshold saturation 실증.
9. 결과: `(3,6,64,5)`,`(4,8,64,5)` coupled ensemble의 BP-ACPR이 MAC-ACPR에 근접 — near-universal 실증 (rate 0.5 기준).
10. 한계: HW 미설계, 유한길이 성능 없음, threshold saturation의 해석적 증명은 open problem, MAC 다중접속 특화.
