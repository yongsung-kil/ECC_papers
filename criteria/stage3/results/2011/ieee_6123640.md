> 총평: LDPC 디코더 PE의 min-offset 연산을 charge-recovery logic(pNBL)으로 저전력화한 순수 트랜지스터/클록 회로 논문으로, 알고리즘·부호설계·양자화와 무관하여 Prime ECC(C++ 시뮬레이터) 이식성 하.

### B. 알고리즘 요약

1. 대상: LDPC 디코더의 Processing Engine(PE) — message passing algorithm(MPA)에서 LLR 결합·min 계산·offset 감산을 수행(입출력 5-bit signed = 1 sign + 4 bit).
2. 문제: 무선통신 LDPC 칩의 전력이 배터리 수명 직결, 기존 저전력화는 아키텍처/알고리즘 개선 중심(회로 복잡도 상승 수반).
3. 접근: 아키텍처/알고리즘이 아닌 회로 기법 — charge-recovery logic으로 CV² 한계 이하 에너지 목표.
4. 핵심 기법: pseudo-NMOS boost logic(pNBL) — boost logic family, PBL의 PUN을 단일 PMOS(clk 게이트)로 치환, evaluation/boost 2-stage 동작.
5. 개선 포인트: 복잡한 pull-up network 제거로 회로 단순화, crowbar current 감소로 PBL 대비 에너지 ~28% 절감(식5 대비 식4).
6. 에너지식: `E_pNBL ≈ 5π²RbCL/T · CL·Vth²`(식4) — 주 소모는 boost part, logic part 기여는 무시가능.
7. 인터페이스: static CMOS↔pNBL 변환 버퍼(DC→AC)와 peak-sampling A/D 컨버터로 기존 CMOS 호환.
8. 전원 클록: LC-tank(3nH on-chip inductor)로 에너지 재활용, 2-phase non-overlap power clock을 blip 회로로 생성(VCC 0.8V).
9. 결과: 0.18µm test chip, 시뮬 403MHz에서 1pJ/cycle(static CMOS 대비 36%), 실측 404MHz 1.1pJ~609MHz 2.1pJ/cycle, 기능 최대 1.5GHz 정상 동작; PE 면적 230×161µm.
10. 한계: 부호·rate·알고리즘 성능 개선 전무, LC-tank/off-chip capacitor·전용 power clock 필요, 회로(트랜지스터) 레벨 기법이라 RTL/C++ 모델에 대응 없음.
