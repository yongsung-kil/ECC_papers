### B. 알고리즘 요약

1. 대상: separable circulant-based(SCB) regular QC-LDPC. `f(i,j)=a(i)·b(j)` 분리형, mother matrix `fm(i,j)=i·j`. array-based/많은 QC 코드 포함. high-rate(0.85~0.92), N~2209, r=4/c=47.
2. 문제: data storage 등 저FER 요구 시스템에서 absorbing set(trapping set의 bit-flip 안정 부분집합)이 error-floor의 주원인.
3. 모델: absorbing set의 각 사이클이 cycle consistency 식 `Σ(i_{m-1}-i_m)u_m = 0 mod p`를 만족해야 존재. 이를 행렬식 `M·u=0`으로 집약.
4. 핵심 도구 CCM: absorbing set 그래프의 선형독립 사이클마다 한 행을 갖는 Cycle Consistency Matrix `M`. absorbing set 존재의 필요조건 = `M`이 full column-rank가 아님.
5. 핵심 아이디어: 행/열 선택으로 dominant absorbing set의 CCM을 full column-rank로 강제하면 해당 구조가 부호에 존재 불가 → 해석적(비휴리스틱) 부호설계.
6. 3계층 부호: EAB(elementary array-based) → SR(선택행 a(i), row-selection) → SSR(추가 열선택 b(j)=shortening). shortening은 새 absorbing set 도입 안 함(Corollary 1).
7. case study r=4: 최소 dominant는 (6,4) absorbing set 3형. Fig.1형은 모든 SR에 존재(det(M)=0 항상), Fig.2/3형은 적절한 RSF `[0,1,3,4]`로 회피.
8. 알고리즘: greedy column-cutting(Algorithm 1) — Fig.1형 (6,4)을 만드는 벡터집합 `W`에서 가장 빈번한 열 j를 반복 제거해 `V∩Ṽ=∅` 달성 → 전 (6,4) 제거.
9. 등가류: row-mapping의 차이행렬 기반 equivalence class로 탐색공간 대폭 축소(r=4는 [0,1,x,y]로 환원).
10. 결과: 200 iter, Q4.2 고정소수점, soft-xor 디코더. SR/SSR 부호가 EAB 대비 error-floor 기울기 급화, low-FER에서 최소 1 order 개선(Table I 오류 프로파일이 (6,4) 소멸 뒷받침). HW 미설계, AWGN 시뮬만.
