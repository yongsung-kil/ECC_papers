### B. 알고리즘 요약

1. 시스템: RC-LDPC(rate-compatible LDPC)를 SC-FDE(single carrier-frequency domain equalization) 무선시스템에 결합, BPSK 변조, ITU Pedestrian A 멀티패스 채널.
2. 문제: rate-compatible 부호가 필요한 적응 코딩/HARQ 환경에서 하나의 base matrix로 여러 rate를 구성하되 4-cycle을 제거해야 함.
3. 채널/모델: AWGN + 멀티패스; CP 대신 unique word(UW, M-sequence 64 symbol) 의사난수로 채널 계수 `γ`와 잡음분산 `σ²` 추정 후 LDPC 초기 LLR 계산.
4. 핵심 기법: rate-1/2 (256,512) base matrix `H_M×N`(lower triangular)에 확장행렬 `H_M'×M'`(size 64)과 identity `I_M'×M'`를 계층적으로 추가 → 저rate 코드 생성, degree-2 VN 펑처링 → 고rate 코드 생성.
5. 4-cycle 제거: 확장부의 두 번째 identity 행렬을 열단위 우측 shift하며 4-cycle 수를 최소화하는 shift 계수 선택, 해당 위치 "1" 삭제(경로 기록).
6. 부호구성: 변수노드 degree를 density evolution 최적 분포(`λ(x)=0.30013x+0.28395x²+0.41592x⁷`)로 오름차순 배치, PEG로 girth 최대화; 조건 미충족 시 Gaussian elimination으로 4-cycle 제거.
7. 채널추정: UW 수신값으로 `E[â]`, `E[|â|²]`로부터 `σ²=E(|â|²)-[E(â)]²`, `γ` 산출.
8. 복호: 표준 BP 알고리즘, 최대 iteration 50, MMSE 등화.
9. 결과: 수정 방식이 원논문[8] 대비 rate-2/5에서 약 0.3dB 개선; SC-FDE 시스템은 uncoded 대비 BER 10⁻⁵에서 8dB 이상 개선.
10. 한계: 시뮬레이션만(HW 미설계), non-QC irregular PEG 부호, 무선 SC-FDE 채널/UW 추정 특화.
