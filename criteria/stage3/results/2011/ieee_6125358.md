### B. 알고리즘 요약

1. 대상은 SeIRA(structured extended IRA) QC-LDPC, rate 1/2, length 576~2304, mother matrix 12×24(802.16과 동일), BPSK/AWGN, sum-product 복호(max 50 iter).
2. 문제: short/moderate 길이에서 짧은 cycle을 전부 제거 불가하고, cycle 연결성이 error-floor를 좌우 — 기존 girth-only 구성으로 부족.
3. 모델: mother matrix 고정 후 각 '1'을 `s×s` 순환천이 identity로 확장, shift 행렬 H의 shift 값을 탐색 대상으로 삼고 trapping/stopping set 크기로 error-floor를 판단.
4. 핵심기법: cycle 길이와 ACE 연결성을 **결합한 신규 metric** — circle `C_2at`의 복호 실패확률을 `Pe^t·[(dc-1)Pe]^a`로 근사하고 이를 최소화하는 shift 값을 선택.
5. 식 의미: `Pe^t·[(dc-1)Pe]^a`는 cycle 내 VN이 모두 초기 오류이고 extrinsic 메시지도 오류일 때 복호가 실패할 확률 근사 — 낮을수록 error-floor에 덜 해로운 cycle.
6. 확장/이론: circle condition(Theorem 1)으로 mother matrix의 cycle만 세어 확장행렬 cycle 분포(`w=2t`, 개수 `s/w`)를 산출, spanning tree로 layer `L_t`까지 cycle 탐색.
7. 구현: shift entry를 `C(dv,2)` 클러스터(2개씩)로 나눠 equation set 구성, VN을 저degree 우선으로 그래프에 점진 추가.
8. 최적화: Greedy(Alg1)·Partial Greedy(Alg2)를 교대하는 Iterative Search(Alg3, max 10회)로 metric 최대값 최소화하는 sub-optimal shift 해 탐색, `Pe`는 대상 BER 구간에서 0.01 간격으로 최적값 선정.
9. 결과: 동일 mother matrix에서 확장한 802.16 표준 부호 대비 FER≈1e-5 부근에서 0.15~0.4dB 이득(FER 우수, BER 최적은 아님).
10. 한계: HW 미설계, sum-product 시뮬만, throughput/gatecount 없음, SeIRA(dual-diagonal) 인코딩 전제.
