### B. 알고리즘 요약

1. 시스템은 3-노드 Gaussian relay 채널(S-R-D), decode-and-forward, bilayer lengthened LDPC(하위=저rate 목적지코드 길이 n1, 전체=고rate relay코드 길이 n=n1+n2, 공통 check k개).
2. 문제: 기존 bilayer 설계는 rate만 최적화하고 relay의 HW/전력·지연 제약(복호 복잡도)을 무시함.
3. 채널모델: `Y1=X+Z1`(relay), `Y=X+X1+Z2`(dest), effective SNR은 SR>SD 링크.
4. 핵심 기법: bilayer density evolution으로 iteration `l`의 메시지 오류확률 `e(p^l,q^l)=Σ{ηλi^(1)e^(1)+(1-η)λi^(2)e^(2)}` (식5) 계산, `η=dc/(dc+dc')`.
5. 하위(목적지) code(λ^(1)) 고정 후 상위(relay) degree분포 λ^(2)만 LP로 최적화; 각 iteration 오류확률 감소 제약 (식6).
6. bilayer EXIT chart로 확장: `e^(l+1)=f(e^(l))` (식7), `f(e)=Σ{ηλi^(1)fi^(1)+(1-η)λi^(2)fi^(2)}` (식8) - chart 개방도가 수렴속도 결정.
7. 복잡도척도: 정보비트당 `K=NE/(nR)` (식9), iteration 수 `N≈∫ de/(e·ln(e/f(e)))` (식12)를 EXIT chart로 근사.
8. 최적화: rate 유지하면서 `N`(=복잡도) 최소화하는 λ^(2)를 연속최적화로 도출 (식13), µ(h)를 점증시켜 threshold-최적 초기값에서 개선.
9. 결과: 목표rate(0.5,0.7)에서 code F(133 iter@σ=0.7066) 대비 설계 C1은 52 iter로 60.9% 복잡도↓; C2/C3도 48~58% 감소 (DDE 12-bit 양자화, 목표 BER 1e-6, max 600 iter).
10. 한계: HW 미설계, asymptotic(무한길이) degree분포 설계뿐, 유한길이/실제 성능·타 check degree 미검토, relay 2-layer 구조 전제.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| bilayer degree분포 최적화 | 대응 없음 | 2-layer relay 부호구조 - 우리 단일 고정 QC-LDPC와 맞지 않음 |
| EXIT chart로 iteration 최소화 설계 | 대응 없음 | 설계시 asymptotic 도구, 런타임 디코더 모듈 대응 없음 |
| 부호구조/H-matrix 로드 | `ecc_top.cpp` `Load_PCM()` | 개념상 부호설계 레이어에 닿으나 QC 고정·재검증 부담 큼(이식난이도 높음) |
| 반복복호 message-passing | `decoder.cpp` `LDPC_Decoder()` | iteration 감소 목표는 공유하나 방법(DE degree설계)이 다름 |

적용 가치: **낮음** - relay 채널 bilayer 부호의 asymptotic degree분포 설계로, NAND 단일링크 binary QC-LDPC의 고정 H-matrix·HW 디코더에 떼어 쓸 접점이 없다.
