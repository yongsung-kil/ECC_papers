> 총평: SCB(Separable Circulant-Based) QC-LDPC의 (6,4) absorbing-set을 provably 제거하며 rate/length 유연성을 높인 부호설계 논문. error-floor 개선은 확실하나 부호구성 레이어라 Prime ECC 이식은 재검증 부담이 큼.

### B. 알고리즘 요약

1. 시스템: circulant 기반 구조화 LDPC 중 **SCB(Separable Circulant-Based)** 부호 계열, 열무게 `r=4`, 내부 circulant 크기 `p`(예제 p=17~37), rate 0.5556~0.8950 / length 209~4953bit.
2. 문제: moderate block length에서 well-designed LDPC도 **error floor**가 남으며, 이는 Tanner graph의 **absorbing set**(near-codeword/trapping-set류) 구조 때문.
3. 기존 방식 한계: 선행연구[7]의 **greedy column elimination**은 과도하게 많은 열을 제거해 rate 상한이 지나치게 보수적 → rate/length 유연성 부족.
4. 핵심 통찰: (6,4) absorbing set을 6원소 집합 `Si`로 매핑, 이를 모두 hit하는 최소 열집합 찾기 = **hitting set problem**(NP-complete)로 정식화.
5. 핵심 식: `min ||X||_1  s.t. M·X ≥ B`, `M`은 |S|×p 지시행렬(`Mi,j=1`이면 열그룹 j가 absorbing set Si에 속함), `X`는 제거할 열그룹 지시벡터.
6. 알고리즘 1단: moderate length는 **BLP(binary linear programming)** 로 최적해; 대규모는 **근사 BNB(branch-and-bound)**.
7. 근사 트릭: 각 열에 reputation 값 유지, 첫 `f`층은 exact BNB로 reputation 누적, `f+1`층부터 reputation>0이면 강제 cut(`ReputationReduction`) → **다항시간** 보장(원래 Θ(2^p)).
8. 상·하한: 해의 하한 `|S|/||M||_1`, 상한은 greedy 알고리즘(`UpperBound()`, 선형시간).
9. 결과: block ~1700bit에서 greedy는 rate 0.8261 고정, 신규 BNB는 0.8261~0.8462; target rate 0.895에서 length 3783~4953bit로 선택폭 확대. FER 성능은 greedy/optimal 동등(Table I), random circulant 대비 최소 1자릿수 우수. (4,4)/(5,4)/(6,4) absorbing set 완전 제거(Table II).
10. 한계: 순수 부호구성 알고리즘, HW/디코더 미설계, AWGN Q4.2 고정소수점 시뮬만(SOFT-XOR 디코더), NAND read/soft 채널 무관.
