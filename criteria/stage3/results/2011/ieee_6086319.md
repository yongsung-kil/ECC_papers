### B. 알고리즘 요약

1. 대상: rate 1/2 QC-LDPC(Block-LDPC), 부호길이 504~2016 bit, binary-input AWGN 채널, BP 복호(최대 50 iteration).
2. 문제: 기존 구성법은 짧은 사이클 제거만 고려해 error floor가 높고, ACE만 고려하면 girth와 ACE 분포 균형 실패.
3. 핵심 개념: girth `lcycle`, ACE 값 `CA=Σ(di-2)`(사이클 참여 VN degree `di` 기반), stopping set 차원.
4. PEGPC-ACE(Algorithm): PEGPC(사이클 다항식 계수)에 ACE 제약 추가, 새 edge 생성 시 `8≤lcycle≤10, CA≥CACE` 만족.
5. 동일 최소계수 후보 check node 중 최대 ACE 노드를 선택 → 최단 girth 수 감소 + 평균 girth 증가 + 사이클 연결성 개선.
6. QC 구성: PEGPC로 basis matrix 생성 후 shift coefficient matrix `P`로 확장, Property 1/2(`Σ(-1)^a·pia,ja=0 mod A`)로 girth-6 사이클 검출.
7. shift 계수 배정: 3번째 행부터 girth-6 검출, 발견 시 랜덤값 재배정, 실패 시 backtracking, 초기 0 원소 최대 보존으로 큰 사이클 유지.
8. 결과1: PEGPC-ACE(504bit)가 PEGPC 대비 BER 2e-5에서 0.1dB 이득, error floor 낮음.
9. 결과2: QC-LDPC(576bit)가 랜덤 Mackay 대비 8e-5에서 0.2dB 이득.
10. 한계: HW 미설계, 시뮬만, 복잡도/게이트카운트 없음, rate 1/2 저rate 무선용이라 고rate NAND와 거리.
