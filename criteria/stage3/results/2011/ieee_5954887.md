### B. 알고리즘 요약

1. 대상: Chinese DTMB QC-LDPC(base matrix + 127x127 circulant), rate 0.4/0.6/0.8, 원 length 7493.
2. 문제: 방송 시스템은 rate가 이산적(rate granularity 제한)이라 seamless rate 적응 불가.
3. 해법 개념: mother code에 known-bits를 padding = codeword의 일부 정보비트를 known(0)으로 고정 → H에서 해당 column 삭제, G에서 해당 row/column 삭제. `(n,k) → (n-m, k-m)`.
4. 등가성: padding m bits ≡ H에서 m개 column 제거 ≡ bipartite graph에서 variable node 제거. residual graph에서 복호.
5. 핵심 파라미터: 어떤 column을 지울지가 residual code의 degree distribution과 decoding threshold를 결정.
6. 기준1 MRWC: row-weight sum(RWS) 최대 column 삭제. `E(i)=Σ(ρ(j)-1)`로 정의된 extrinsic-sum과 등가이나 저자는 RWS 해석이 더 타당하다고 주장.
7. 후보집합 차이: MRWC(A)=최저 degree column만 후보 / MRWC(B)=모든 degree column 후보 → threshold 기반 최적 패턴 탐색.
8. 기준2 MRWVC: 삭제 후 row weight variance 최소 → residual matrix의 row degree를 최대한 regular하게(throughput·error-floor 유리).
9. 최적화: 정수 partition Np로 각 degree별 삭제 개수 조합을 탐색, 각 조합의 threshold(EXIT/DE) 계산 후 최저 threshold 패턴 선택.
10. 결과: BAWGN 시뮬에서 MRWC(B)/MRWVC가 MRWC(A) 대비 threshold 개선(예 rate0.5: 1.56→0.98dB, rate0.75: 2.37→2.21dB). HW 미설계, threshold/BER 시뮬만.
