> 총평: 고차변조 BICM-ID에서 demapper↔decoder 반복 시 외부정보를 Gaussian factor(<1)로 곱해 상관을 줄이는 통신 특화 기법. NAND에는 복조 반복 loop 자체가 없어 이식 가치 낮음.

### B. 알고리즘 요약

1. 시스템: LDPC 부호 BICM-ID(bit-interleaved coded modulation with iterative decoding), AWGN + high-order 변조(8PSK/16QAM), inner/outer iteration 각 10회.
2. 문제: BICM-ID에서 short/moderate LDPC의 Tanner graph short cycle로 디코더 출력 외부정보의 독립성 가정이 깨져 `Lch(c)` 계산이 부정확.
3. 모델: 외부정보 `W`를 Gaussian random variable로 모델링, `wi = ci·μw + ni'`, `ci∈{+1,-1}`.
4. 핵심기법: 디코더가 내보낸 외부정보 `La(c)`에 인수(<1)를 곱해 데모듈레이터로 feedback → 상관 완화, 신뢰도 향상.
5. 핵심식: `La,i ≈ (2μw/σw²)·wi`, 곱해지는 인수 `2μw/σw²`를 "Gaussian estimate factor"로 정의(식 4).
6. 파라미터: `μw`(식5), `σw²`(식6)는 통계 프레임·반복 수 곱 `Nt` 위에서 추정.
7. 구현 디테일: high-SNR에서 factor가 고정값으로 수렴 → offline 초기화 시 상수 `α`로 대체, 온라인 계산 불필요(복잡도 증가 미미).
8. SISO 복조/복호는 Max-Log-MAP 사용.
9. 결과: (504,252) regular 및 (1056,528) 802.16e irregular LDPC에서 각 0.6dB / 0.4dB 개선; WiFi 부호(1944, rate 1/2·3/4·5/6)로 factor(0.4~0.9)별 FER 비교, factor≈0.5~0.6 부근 최적.
10. 한계: HW 미설계, 순수 시뮬, factor 최적값 결정법 미해결, 고차변조 복조 반복에 종속.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| demapper↔decoder 외부정보 feedback loop | 대응 없음 | Prime은 단일 채널(NAND read)만, 고차변조 복조 반복 loop 자체가 없음 |
| 외부정보 Gaussian scaling factor | [channel.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `Set_LLR_Th()` (약한 유사) | LLR 스케일링 개념만 겹치나 반복 복조가 전제라 직접 대응 아님 |
| Max-Log-MAP SISO 복조 | 대응 없음 | Prime은 min-sum 기반 LDPC 복호만 |

적용 가치: **낮음** — 기법의 전제(고차변조 BICM-ID의 데모듈레이터-디코더 반복)가 NAND ECC에 존재하지 않으며, LDPC 내부 디코더 개선이 아니라 외부 SISO loop scaling에 국한된다.
