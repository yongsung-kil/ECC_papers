### B. 알고리즘 요약

1. 대상: array code 기반 QC-LDPC, PCM은 `p×p` circulant 순열행렬 `P`의 거듭제곱으로 구성(식1), `p` 소수, column weight `r`, rate `R ≥ 1−r/p`.
2. 문제: girth-10 이상 고rate QC-LDPC의 체계적 구성은 미해결 문제(광통신용 저redundancy·극저 error-floor 요구).
3. 모델: cycle-2k는 cycle-governing equation `Σcᵢxᵢ ≡ 0 (mod p)`(식3,5)의 proper solution 존재와 등가 → shortening으로 이를 회피할 column index 부분집합 `S(p;Ω)` 탐색.
4. 핵심기법 1: 남는 최대 column 수 `s(p;Ω)`의 일반 하한 유도(Theorem 2, `Θ(p^{1/(k-1)})`).
5. 핵심식 의미: `s(p;Ω)`가 클수록 달성 rate가 높으므로 이 상하한이 구성 가능한 rate 범위를 결정.
6. 핵심기법 2: girth-8은 상한 `p(log log p)^{-c(v)}`, girth-10↑는 대칭방정식 `x₁+x₂≡x₃+x₄`에 Lemma 1 적용해 상한 `√(6p)=O(√p)` 유도(Corollary 1).
7. 부수 결과: column-weight-3 girth-10은 Theorem 4로 더 tight한 하한 `β√p − α√log p` → 상한과 동차수.
8. 구성 절차: 제안 알고리즘 — 소수 `q`로 집합 `X` 구성 후 4변수 cycle 방정식 proper solution 회피 부분집합을 초기화하고 greedy 확장(random 구성과도 비교).
9. 결과(Table II): 동일 rate(0.727~0.842) 달성에 필요한 최소 `p`가 greedy 대비 comparable하거나 훨씬 작음(예 R=0.833: pproposed=2999 < pgreedy=3407).
10. 한계: 순수 이론(구성법·상하한)만, 실제 BER/error-floor 시뮬 없음, 디코더·HW 미설계, 광통신 지향, column-weight-3 저degree.
