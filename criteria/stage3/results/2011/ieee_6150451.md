### B. 알고리즘 요약

1. 대상은 AWGN 채널용 regular binary QC-LDPC 부호설계로, circulant permutation matrix 기반, rate 0.25/0.5/0.89, block length 200~2947(대표 1332), girth 8~20 예제.
2. 문제: 대수적 QC-LDPC은 rate/length/girth가 제한적이고, 랜덤/PEG 코드는 row-column 연결이 거의 무작위라 디코더 복잡도가 높음.
3. 채널/모델: 표준 AWGN, sum-product(iterative) 복호 가정.
4. 핵심 기법: 코드워드 row를 group size에 따라 sub-row로 분할(row-splitting)하여 새 H를 구성 — 각 sub-row에서 1-component를 선택해 concentrated degree distribution 유지.
5. 핵심 제약: RC(row-column) constraint — 각 row weight `k`, 각 column weight `j`, 임의 두 row/column이 공유하는 1은 최대 1개(식 property 3)로 short cycle 억제.
6. 확장: row-union으로 여러 rate를 단일 아키텍처에서 지원 — check node 메시지 리스트를 union으로 교체해 rate 변경 시 읽는 메시지 양만 조정.
7. 구현 디테일: `H_QC`는 `H_cir`(circulant permutation)와 zero block `Z`의 블록 배열; concentrated check-node degree가 수렴속도·HW 디코더에 유리.
8. (학습/최적화 절차 없음 — 결정론적 구성, 랜덤코드는 guess-and-test로 girth 탐색.)
9. 결과: BER 10⁻⁶에서 PEG-LDPC 대비 0.1dB gain; 고 Eb/N0에서 girth 16/20이 랜덤코드 대비 0.056/0.27dB gain(BER 10⁻⁵); rate 0.89·length 1332에서 Shannon-limit 1.3dB 근접, error floor도 낮춤.
10. 한계: HW 미설계(구조적 주장만), wireless FEC 타깃, coding gain 0.1dB로 작고, 정량 수치 상당수가 그림/표에만 존재.

적용 가치: **낮음** — wireless FEC 대상의 부호설계 논문으로, Prime ECC의 고정 QC-LDPC 부호를 교체할 실익이 작고(이득 0.1dB), HW는 구조적 주장에 그쳐 이식 근거가 부족함.
