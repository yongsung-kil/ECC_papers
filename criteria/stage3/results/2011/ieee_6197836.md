### B. 알고리즘 요약

1. 목표는 조합적 객체 `difference sequence`로 (3,k)-regular binary QC-LDPC(DS-LDPC)를 대수적으로 구성, girth ≥ 8 보장.
2. 문제: 랜덤 구성은 저장·구현 부담이 크고, 구조화된 QC-LDPC라도 girth 확보가 성능(BER/error-floor) 핵심.
3. 구성: 2차 다항식 `hn = a n^2 + b n + c`의 0/1/2차 차분수열을 3개 행으로 삼아 3×k shift matrix `S(H)` 생성(`Δ^3 hn = 0`).
4. 각 원소 `Δ^j h_i`를 p×p 항등행렬을 그만큼 순환 shift한 CPM `I(Δ^j h_i)`로 치환해 3p×kp PCM H 완성(col weight 3, row weight k).
5. rate는 `R ≥ 1 - (3p-2)/kp` (각 CPM 행 블록이 all-one row로 합쳐져 rank ≤ 3p-2).
6. girth 조건: 2×2 부분행렬에서 `(b-a) mod p ≠ (d-c) mod p`면 4-cycle 없음(girth≥6, 정리 3.2/3.3, `b>0, a≠b`).
7. girth 8: 6-cycle 6개 패턴을 pairwise difference set `D(1),D(2)` 비교로 배제하는 탐색 알고리즘으로 좋은 (a,b,c)·p 선택.
8. 검증: BI-AWGN, BPSK, SPA 복호, 최대 50 iteration 시뮬.
9. 결과: (402,203) rate0.5, (2552,1862) rate0.73, (854,491) rate0.57 예제 — PEG와 대등/약간 우세, MacKay 대비 0.5~0.8dB, array 대비 1.7~2dB 우세, MacKay가 보이는 error floor 없이 BER 1e-8까지 내려감.
10. 한계: HW 미설계, column weight 3 고정, rate 0.5~0.73(고rate 영역 미검증), 순수 부호 구성법.
