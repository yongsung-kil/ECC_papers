### B. 알고리즘 요약

1. 대상은 CMMB(China Mobile Multimedia Broadcasting) LDPC, 부호길이 `9216`, rate `1/2`와 `3/4` 두 종을 지원하는 저전력 멀티레이트 디코더.
2. 풀려는 문제: CMMB 검사행렬의 non-zero 원소가 랜덤 분포라 대용량 메모리·읽기/쓰기 충돌·큰 전력소모가 발생 - 이를 저비용으로 해결.
3. 복호는 modified layered min-sum(offset/scaled). 정량화: soft 입력 4bit, posterior(bit node) 6bit(부호1+크기5), check node 4bit(부호1+크기3).
4. 파라미터: 시뮬레이션으로 4bit 양자화가 성능/메모리 최적(4->5bit는 0.03dB뿐), iteration `10`회(10->15는 0.1dB뿐), correction factor `0.75`(0.8이 최적이나 x0.75가 구현 편의).
5. 핵심 기법1(메모리 압축): CNU가 최소/차소값만 쓰므로 각 RAM 주소에 |min|·|2nd min|·입력 MSB·min 위치만 저장 -> 메모리 37.5% 절감. check 메모리는 dual-port RAM(depth 2304, width 30bit).
6. 핵심 기법2(충돌 해결): 열가중치 2 서브행렬(1/2 rate 9행23열, 3/4 rate 3개)은 한 clock에 2 read + 2 write 필요 - dual-read/write RAM 회피 위해 동일내용 2개 dual-port RAM backup 구조로 교대 읽기/쓰기.
7. 구현 디테일: PROF(2x2 pseudo rank order filter) 기반 6입력 CNU(min1/min2/index 출력), 1/2와 3/4 rate CNU를 재사용(module reuse)해 자원 절감. 총 39개 dual-port RAM(6bit x 256).
8. 결과: EP2S180(Altera Stratix II) FPGA 검증, 메모리 201.2Kbit(칩 4%), 전통 대비 메모리 ~50% 절감, 문헌 대비 최소 combinational logic/register.
9. 합성: SMIC 0.18um, 로직셀 0.23mm2 + dual-port RAM 7.38mm2 = 총 7.6mm2, 전력 132.8mW(문헌 중 최저 수준).
10. 한계: 정정성능 개선 없음, throughput/gate count 본문 미기재, CMMB 표준의 랜덤분포·열가중치2 충돌에 특화된 메모리 구조.
