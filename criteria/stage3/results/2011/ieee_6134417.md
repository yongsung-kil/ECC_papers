### B. 알고리즘 요약

1. 시스템: NAND SLC/MLC(4-level) flash. threshold voltage를 m-PAM+AWGN으로 모델링, `Es`/`N0` 파라미터. 실험은 rate `0.9021` irregular LDPC (`n=9118`, `k=8225`, ACE+stopping-set 최적화), 비교군 BCH(`n=9152`).
2. 문제: flash는 관례적으로 hard 1-bit(sense-amp comparator)만 제공. LDPC의 이점을 살리려면 soft 정보 필요하나, 균등 간격 word-line 전압은 성능이 나쁘다.
3. 모델: 각 read는 word-line 전압에서 1비트만 제공. 여러 word-line 전압으로 다중 read 시 threshold voltage를 여러 구간으로 양자화 → 등가 read 채널을 확률전이행렬로 표현.
4. 핵심 기법: 각 양자화(read 수)에 대해 입력 X와 출력 Y 사이 상호정보 `I(X;Y)=H(Y)-H(Y|X)`를 최대화하는 word-line 전압 `q*`를 수치 탐색.
5. 핵심 식 의미: `I(X;Y)` 최대화가 곧 양자화가 전달하는 정보량 최대화 → 주어진 양자화 레벨에서 근사 최적 LDPC 복호 성능 보장. `q*`는 `Es/N0`의 함수.
6. 확장: SLC 2-read(1.6bit)/3-read(2bit), 4-level MLC는 기본 3 word-line에 3개 추가해 6-read(2.8bit, 7구간). Gray labeling(00,01,11,10)로 raw BER 최소화.
7. 복호: 최적화된 전이행렬로 표준 belief-propagation(sequential BP) 복호.
8. 결과: 2-read soft가 1-read 대비 대부분의 full-soft gap 회복, 6-read MLC는 full-soft에 근접. soft 정보 사용 LDPC가 동급 rate/length BCH를 명확히 상회(hard-only에선 격차 작음).
9. 한계: i.i.d. Gaussian threshold 모델 단순 가정(ICI/retention/tail 비대칭 미반영, 확장 가능하다고만 언급). HW 설계 없음, error-floor는 future work.
