### B. 알고리즘 요약

1. 대상은 여러 개 regular LDPC block을 결합한 terminated LDPC convolutional(=spatially-coupled) 부호. 예로 `(3,6)`, `(4,8)`-regular, base matrix `B=[3,3]`/`[4,4]`, rate 1/2.
2. 문제: 용량접근 irregular LDPC는 degree-2 VN이 많아 error floor 높고 수렴 느림. 결합 부호로 이를 회피하면서 threshold saturation을 얻되 복잡도 대가를 분석.
3. 채널은 BEC 및 AWGN, 성능은 discretized density evolution으로 threshold 계산.
4. 핵심 기법: edge spreading으로 시각 t의 VN 엣지를 `t..t+mcc` CN에 분배해 convolutional protograph base matrix `B[1,L]` 구성, 조건 `sum_i B_i = B` 유지.
5. 결과 명제: `Jmin>2`이면 iteration `I>Ibr` 이후 오류확률이 이중지수적으로 감소(식 7), lifting `N`에 대해 `Pb < exp(-a'N^γ)`, 블록오류 `O(N log N)` 복잡도로 소멸.
6. Threshold saturation: L→∞일 때 결합 앙상블 BP threshold가 기저 block 앙상블의 MAP threshold로 수렴(끝단 저차수 CN 덕분).
7. 초기 iteration `Ibr`은 저SNR 영역(`σ_blk<σ<σ*`)에서 L에 선형 증가, 고SNR 영역에서는 상수로 수렴(Fig.2).
8. 복잡도 절감: 효율적 message passing schedule(MPS-I~IV)로 불필요 노드 갱신 생략, MPS-III는 effective iteration `Ieff`가 L에 무관.
9. 실용 구현: sliding window decoder(MPS-IV, window `W`)로 latency·크기가 L에 무관해짐, `(3,6)-B W=6`, `(4,8)-D W=4`가 좋은 복잡도-latency 트레이드오프.
10. 한계: 순수 점근 이론, 유한길이 시뮬/BER 곡선·HW 없음, 부호구조가 convolutional/coupled로 block QC-LDPC와 근본적으로 다름.
