### B. 알고리즘 요약

1. 대상은 GF(2) 위 `e×e` 순환행렬 `m0×n0` 배열로 된 QC 부호의 parity-check matrix `H` (e는 홀수 가정).
2. 문제: 순환배열 H의 구조적 성질(girth, binary 조건)을 직접 다루기 어려워 통일적 분석 틀이 필요.
3. 핵심 도구: `q-1`이 e로 나눠떨어지는 GF(q)에서 order e인 원소 α로 Vandermonde `V=[α^{-ij}]` 정의.
4. Lemma 1: 임의 순환행렬 `A`는 `VAV^{-1}`로 대각화되며, 대각벡터가 첫 행의 Fourier 변환 `bt=Σ α^{tl}a_l`.
5. Lemma 2-3: 배열 전체에 `diag(V,...)H diag(V^{-1},...)` 적용 후 행/열 치환 π로 `HF,π = diag(B0,...,Be-1)` 대각 배열 획득.
6. Theorem 1: H ↔ HF,π는 일대일이며 rank(H)=Σ rank(Bt) 보존.
7. Theorem 2: H가 binary일 필요충분조건은 conjugacy constraint `B_{(2t)e}=Bt^{∘2}` (Hadamard 제곱).
8. Theorem 3: H가 순환 permutation/zero 행렬 배열일 필요충분조건은 `Bt=B1^{∘t}`이며, base matrix `B=B1`로 부호가 완전 규정됨(masking `Bmask=Z⊗B` 포함).
9. Theorem 4·Cor 1-3: girth≥g 조건을 base matrix B의 `g'/2×g'/2` 부분행렬 determinant 전개(동일 비영항 부재)로 특성화 — RC제약(girth≥6)은 2×2 부분행렬이 zero 포함 or 비특이.
10. 결과: RS 기반 base로 (5080,4575) rate 0.9006, (16129,15372) rate 0.953, masked (8128,4064) rate 0.5 QC-LDPC를 SPA 50회로 AWGN에서 시뮬(BER 10^-6서 Shannon 한계 0.95~1.2dB); HW/디코더 미설계.
