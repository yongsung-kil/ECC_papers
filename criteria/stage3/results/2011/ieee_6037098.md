### B. 알고리즘 요약

1. 시스템: rotating OFDM + QPSK, AWGN 채널, LDPC 부호 rate `1/2`, length `1152`, 최대 iteration 50.
2. 문제: LLR BP(SPA)는 체크노드 갱신에 `tanh`/`tanh^-1`·곱셈·나눗셈이 많아 복잡도 높고, 짧은 LDPC의 Tanner graph loop 때문에 변수노드 간 상관으로 성능 저하.
3. 모델: 표준 AWGN(평균 0, 분산 σ²), 수신 LLR 초기화 `L(P_i)=2y_i/σ²`.
4. 핵심기법 1단(체크노드): SPA의 `2 tanh^-1(∏ tanh(·))`를 부호(sign) 분리 후 `∏ sgn·min(|L(q)|)` min-sum으로 근사.
5. 핵심 식: 여기에 offset factor `β`를 적용 `L(r_ji)=∏ sgn(L(q))·max(|L(q)|-β, 0)` (식10) — BP-based 과대추정 진폭을 β만큼 깎아 신뢰도 보정.
6. 핵심기법 2단(변수노드): additive correlation factor `X`로 loop 상관 완화. `Y=L(P_i)+Σ L(r)`가 `Y>X`면 `X`를 빼고, 아니면 그대로 (식11).
7. 구현 디테일: 개선 알고리즘은 덧셈·비교 연산만 사용(곱셈/나눗셈 제거), Table 1에서 single-iteration 연산량 비교 제시.
8. 학습/최적화: 없음. β=0.55, X=0.7 고정값 사용.
9. 결과: BER 10^-3에서 LLR BP 대비 약 0.5 dB 이득, SNR 2 dB에서 평균 iteration 20% 감소.
10. 한계: HW 설계 없음, MATLAB 시뮬만, rotating OFDM 특정 시스템, 부호종류(QC 여부) 미기재, 표/그림 손상으로 세부 수치 확인 어려움.
