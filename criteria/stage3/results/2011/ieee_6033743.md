### B. 알고리즘 요약

1. 대상: BSC 상 열가중치 `λ` regular binary LDPC, hard-decision **bit-flipping(BF)** 디코딩. 보장 정정능력 이론 분석.
2. 문제: 기존 결과는 flipping 임계값을 `λ/2`로 두거나 expander 논증에 의존 — 정정능력 하한이 느슨함.
3. 설정: 신드롬 `s = Hе`, BF는 매 반복 `i = argmin_j w_H(s - h_j)`로 불만족 검사합이 최대인 열 `h_i`를 뒤집음(`s ← s - h_i`).
4. Lemma 1: `h_i∈오류집합`, `h_k∉오류집합`일 때 `w_H(s-h_i) < w_H(s-h_k)` ⇔ `|Is(s-h_i,h_i)| + |Is(s,h_k)| < λ` (Is = support 교집합).
5. girth ≥ 6: 각 열쌍이 최대 1개 공통 1비트 → 오류수 `t < (λ+1)/2`면 BF가 항상 정확한 오류 위치를 찾음(Theorem 1).
6. 핵심 정의: VNA 그래프 `G_VNA(v1..vt)` — 공통 검사노드를 공유하는 변수노드쌍에 간선; maximal clique(size≥3)는 트리로 축약(Step 2)해 Tanner girth 반영.
7. Property 1: girth-8 Tanner → VNA girth ≥ 4; `deg_G(h_i)`가 `h_i`와 타 오류 변수노드를 함께 검사하는 검사합 수의 상한.
8. Theorem 4(핵심): girth ≥ 8이면 `t ≤ λ-1`일 때 `i0 = argmin deg_G(h_i)`에 대해 `w_H(s-h_i0) < w_H(s-h_k)` — 즉 BF가 λ-1 오류까지 보장 정정.
9. 함의: `1/2·n0(λ/2, g/2) = λ/2` (girth 8)를 넘어 λ-1로 강화; girth ≥ 8 부호는 `d_min ≥ 2λ-1`. LU(3,q) 부호는 random 정정능력까지 BF로 도달.
10. 한계: 순수 이론 증명, 시뮬/HW 전무; BSC·hard-decision·regular·girth-8 가정에 국한.
