> 총평: girth·최소거리를 높이는 일반 regular LDPC 구성법으로 error-floor 개선 방향은 유효하나, QC 제약·고rate·HW 검증이 없어 Prime ECC 고정 QC-LDPC에 이식성 하.

### B. 알고리즘 요약

1. 대상은 `(d_V, d_F)`-regular LDPC (예: (3,6)), block length `N`=504/1008/1512, rate 0.5, AWGN 채널·BP 복호(최대 80 iter).
2. 문제: PEG/ACE-PEG는 순차적 edge 고정으로 girth만 국소 최대화 → 최소거리 하한이 제한적.
3. 각 variable node `v_i`에 뿌리를 둔 **independent tree**를 만들어 최소거리 하한을 계산 (Tanner graph 유효 codeword는 tree에서도 유효).
4. 핵심 기법: leaf→root로 모든 VN에 LLR cost +1.0을 주고 **min-sum(MS)** 연산 → root 출력이 최소무게 deviation `d_ITB,i`.
5. Theorem 2: `d_min >= d_ITB-min = min_i d_ITB,i` (regular·irregular 모두 성립).
6. Proposition 3: variable-node-degree regular 코드에서 `d_ITB,i`는 girth 기반 하한 `d_G-min`보다 크거나 같음(즉 더 tight).
7. 구성(Algorithm 4): 랜덤 초기 코드에서 공통 check를 공유하지 않는 edge 쌍의 check 연결을 교환, `d_ITB-min` 개선/동률 시 유지, 악화 시 되돌림 (edge 교환이 없을 때까지 반복).
8. 트릭: `N`개 independent tree 구성을 병렬화해 구성 복잡도 완화 가능(정성적 언급만).
9. 결과: N=1008 ITB 코드가 girth 8→10, `d_ITB-min` 6→10으로 상승, 고SNR(`P_b<1e-5`)에서 PEG/ACE-PEG 대비 BER 최저.
10. 한계: HW 미설계·시뮬만, non-QC 일반 regular 코드 대상, 구성 복잡도 PEG보다 큼.
