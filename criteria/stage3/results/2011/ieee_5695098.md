### B. 알고리즘 요약

1. 시스템: 정규 이분/하이퍼그래프 정점에 local code를 배치한 generalized LDPC(graph/hypergraph) 부호 앙상블, `d`-정규 그래프 정점수 무한대 점근 분석.
2. 문제: 기존 expander code 이론은 spectral gap 때문에 local code 최소거리가 그래프 degree의 제곱근 이상이어야 해 Hamming 같은 작은 거리 부호를 배제함.
3. 모델: spectral gap에 의존하는 explicit expander 대신 **랜덤 그래프** 앙상블을 사용해 이 거리 제약을 제거.
4. 핵심기법 1단(Algorithm I, 이분그래프): 두 정점 파트를 교대로 병렬 복호, local code를 최소거리 보장 반경까지 복호하는 반복 알고리즘.
5. 정리 4.2: local code가 2개 이상 오류를 정정하면 앙상블 내 거의 모든 부호가 부호길이에 선형 비례하는 오류 패턴을 정정(지수적으로 작은 예외 비율).
6. 핵심기법 2단(Algorithm II, 하이퍼그래프): 각 파트를 부분절차로 복호해 후보 리스트 생성, 상수 iteration 내 최소 한 후보의 오류를 목표 비율로 감축 후 다른 저복잡도 디코더로 잔여오류 제거.
7. 부수: local code가 1-오류정정 Hamming이면 `s <= 2`일 때 4-cycle 문제로 수렴 실패, Golay/BCH 등 예시로 정정 비율 폐형 계산.
8. 학습/최적화: 없음(엔트로피 함수 Lagrangian 최적화로 정정 비율 경계를 해석적으로 도출).
9. 결과: Hamming local code 네트워크가 트리 가정 없이 일정 비율 오류 정정, flipping 알고리즘 대비 최소 한 자릿수 나은 추정치(예: rate ~0.375에서 δ 등 예제값).
10. 한계: 순수 점근 이론, 유한길이 구성·복잡도 상수·HW·soft 정보·NAND 실험 전무.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| generalized LDPC / hypergraph 부호구성 | 대응 없음 | Prime ECC는 특정 binary QC-LDPC(base+circulant) 고정, graph-code local-code 구조 아님 |
| 반복 local-code 복호(Algorithm I/II) | 대응 없음 | min-sum 메시지패싱과 무관한 조합론적 iterative bounded-distance 복호 |
| 정정 오류 개수 점근 경계 | 대응 없음 | 이론적 정정능력 증명일 뿐 디코더/부호 파라미터에 직접 이식할 산출물 없음 |

적용 가치: **낮음** — 순수 점근 정보이론 결과로 유한길이 QC-LDPC 부호설계나 min-sum 디코더 어느 레이어에도 직접 이식할 구체 산출물이 없음.
