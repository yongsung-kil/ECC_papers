### B. 알고리즘 요약

1. 시스템: base LDPC(Tanner graph)를 cyclic lifting(degree `L`)하여 QC-LDPC를 만들고, BSC/Gallager A/B 및 BIAWGN/BP·min-sum에서 error-floor 개선.
2. 문제: 유한길이 LDPC는 고SNR에서 dominant trapping set 때문에 error-floor가 발생. 기존 random lifting은 앙상블 평균만 개선.
3. 모델: `(a,b)` trapping set = odd-degree check를 갖는 variable node 집합. BSC에서 FER floor는 최소 critical number `m`과 그 개수 `κ`로 추정(FER 기울기 = `m`).
4. 핵심기법: dominant trapping set을 구성하는 short cycle을 lifted graph에서 더 긴 cycle로 매핑해 제거. edge에 circulant permutation shift 부여.
5. 핵심식: cycle의 permutation shift 합 `σ = Σ ±b_i (mod L) ≠ 0` (Thm.1) 이면 base cycle이 lifted graph에서 길이 `L·length/o`의 더 긴 cycle들로 분해 -> 원 cycle 소멸.
6. 확장: Lemma 3 - degree `L=L1·L2` cyclic lifting은 L1-lifting의 L2-lifting으로 분해, rate/min-distance bound(Thm.3) 도출.
7. 트릭: Deterministic Edge Swapping(DES) - trapping set을 critical number 오름차순 정렬 후 greedy하게 최소 edge를 골라 `σ≠0` shift 할당(Algorithm 1).
8. 성질 보존: lifting은 base code의 degree distribution을 그대로 유지, full-rank PCM이면 rate도 유지.
9. 결과: Tanner (155,64) 5-lifting에서 FER 기울기 3→5로 증가(trapping set 제거), random lifting/random code 대비 error-floor 크게 우수. 다양한 예제(regular (504,252), irregular (200,100) 13-lifting rate-1/2 등)에서 일관된 개선.
10. 한계: HW 미설계, 시뮬만. dominant trapping set을 미리 알아야 하고 block length가 `L`배 증가. Gallager A/B용 설계지만 BIAWGN/BP·MS에도 효과 확인.
