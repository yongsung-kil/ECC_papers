### B. 알고리즘 요약

1. 대상은 802.11n/802.16e irregular QC-LDPC (base matrix `Mb`, rate 1/2·2/3, length 648~2304).
2. 문제: 알려진(고정값) 정보비트를 무한신뢰로 처리하는 shortening에서 "어느 정보열을 제거할지" 기준이 성능을 좌우.
3. 가정: irregular QC-LDPC 성능은 girth보다 degree 분포가 지배; EXIT chart로 Eb/No threshold 예측(무한길이 가정).
4. 기존 LES: 최소 column-degree 열을 우선 후보로, 그중 extrinsic sum `e(j)=Σ(ρ(i)-1)` 최대 열 제거.
5. 저자 주장: 최소 degree 우선은 average row degree 증가로 전체 복호력 저하 가능 → trade-off라 근거 약함.
6. 제안 SRVP: 모든 정보열을 동등 평가, degree별 그룹 조합(category)을 나눠 각 category에서 제거 후 **row-degree variance**가 최소인 패턴을 선택.
7. 최종 선택: 각 category 대표 패턴들에 대해 EXIT chart로 threshold 계산, 최저 threshold 패턴을 최종 shortening 패턴으로 확정.
8. 결과(이론): 원부호 대비 LES 0.212~0.847dB, SRVP 0.365~1.041dB gain; SRVP가 LES보다 0.009~0.414dB 우수.
9. 결과(시뮬): 유한길이라 이론보다 gain 작음; degree-2 parity열이 short 후 최대 80% 차지 시 성능 붕괴.
10. 한계: rate 1/2에서 정보비트 67% shortening 시 waterfall이 평탄해져 고SNR에서 원부호보다 나빠짐; HW 미설계, 시뮬만.
