### B. 알고리즘 요약

1. 대상: rate `R=b/c` binary convolutional code를 tailbite한 `(J,K)`-regular QC-LDPC block code. PCM은 monomial `h_ij(D)=D^wij` 항으로 구성.
2. 문제: 반복복호 효율을 좌우하는 Tanner graph의 girth를 크게 만든 짧은/중간 길이 QC-LDPC 부호를 찾기(short length에서 랜덤부호보다 우수).
3. 핵심 표현: full-rank PCM = 이분그래프의 biadjacency matrix(Tanner graph). tailbiting 길이 `M→∞`이면 convolutional PCM `H(D)`로 unwrap, 그 무한 그래프 girth를 `gfree`로 정의.
4. 핵심 도구(voltage graph): base graph `GB`(=B=H(D)|_{D=1})의 각 edge에 voltage `µ_ij=w_ij mod M` 부여. lifted graph의 cycle ⟺ voltage graph에서 voltage 합=0인 cycle.
5. 핵심 아이디어: (free)girth 탐색 = "voltage 합이 0인 최단 cycle 찾기". voltage 배정이 곧 `H(D)` monomial 차수 선택과 동치.
6. 탐색 절차: base graph에서 girth ≥ g 만족하는 비음정수 voltage 배정을 찾고, girth 유지하며 mod M residue로 M 최소화 → tailbiting해 rate `R=Mb/Mc` QC-LDPC 획득.
7. 복잡도 축소 트릭: 모든 길이-g cycle 탐색 `(J-1)^g` → tree 표현으로 `(J-1)^{g/2}`. 각 symbol node root로 c개 subtree를 depth `(g-2)/2`까지 성장, 같은 subtree 내 동일 number·depth·다른 parent인 node쌍의 path-voltage 차 `f=0`이면 짧은 cycle 존재.
8. 두 탐색 알고리즘: A(불평등식 리스트 L 기반, `NT+NL`), B(정렬 기반, `NT·log2 NT`). girth ≤ 10은 A, 점근적으로는 B 우수(NL≈NT^2).
9. 결과: STS(n) 기반 shortened base matrix로 girth 14~18, 짧은 QC부호 재적용으로 girth 20~24 부호 구성 → 기존[7] 결과 개선. 예: (1812,453) g=14, (39.6M,...) g=24.
10. 한계: 순수 구성법(BER/디코딩 실험·HW·복잡도 gate 없음), girth 20~24 부호는 block length 수백만~수억으로 저자도 "비실용적"이라 명시, rate 대부분 0.2~0.5.
