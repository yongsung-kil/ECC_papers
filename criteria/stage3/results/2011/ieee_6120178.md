### B. 알고리즘 요약

1. 대상은 이산 상태 `d`개를 갖는 pairwise MRF의 marginalization; BP/sum-product의 CN 갱신이 `d x d` 행렬-벡터곱이라 iteration당 `Θ(d^2)` 비용.
2. 문제: `d`가 큰 경우(컴퓨터비전 disparity, 센서추적, 대형 alphabet 부호) BP 메시지 계산/저장이 비싸 소규모로 제한됨.
3. 핵심 관찰: 메시지 `F_uv(m)`는 정규화 컬럼 `Γuv(:,J)/βuv(J)`의 기대값 `E_{J~puv}[·]` (식8)로 표현됨.
4. 핵심 기법(SBP): 매 라운드 기대값을 직접 계산하는 대신, 확률질량 `puv(j)` (식7)로 컬럼 인덱스 `J`를 1개 샘플링.
5. 메시지 갱신 `m^{t+1} = (1-α_t)m^t + α_t·Γuv(:,J)` (식9), step size `α_t=O(1/t)`로 시간평균해 기대값 근사.
6. 복잡도: edge당 `(ρmax+6)d+3` 연산으로 `Θ(d^2)->Θ(d)` 한 자릿수 절감 (컬럼/β는 사전계산·저장).
7. 이론 보장1(트리): SBP가 유일 BP 고정점으로 a.s. 수렴, 절대오차 `E|m^t-m*| <= 4(I-2A)^{-1}/sqrt(t)` (nilpotent A, degree=diam(G)).
8. 이론 보장2(일반 그래프, contraction `1-μ/2`): a.s. 수렴 + MSE `E[||ε||^2]/||m*||^2 = O(1/t)`, 고확률 오차 상한 `O(t^{-1/2} log t)`.
9. 결과: Potts 모델 사슬(n=100)·10x10 grid에서 `d in {128,256,512,1024}` MSE가 1/t(log-log 기울기 -1)로 수렴, iteration당 복잡도 크게 감소.
10. 한계: HW 미설계, 큰 alphabet `d`에서만 이득 - binary(`d=2`) LDPC는 `Θ(d^2)=Θ(d)`라 이득 소멸; LDPC는 예시 언급뿐 실험 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| stochastic 메시지 갱신(SBP) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 개념적으로 message-passing 루프에 닿으나 binary min-sum과 다른 패러다임 |
| d x d 행렬-벡터곱 복잡도 절감 | 대응 없음 | 우리 CN은 min-sum(min1/min2+sign) - d=2라 O(d^2) 병목 자체가 없음 |
| non-binary/대형 alphabet BP | 대응 없음 | non-binary LDPC/일반 MRF 전용, binary QC-LDPC 코드베이스와 무관 |

적용 가치: **낮음** - 복잡도 이득이 큰 alphabet `d`에 비례하는 알고리즘이라 binary GF(2) QC-LDPC(`d=2`)에서는 이득이 사라지고, 우리 min-sum HW 디코더와 정합점이 없다.
