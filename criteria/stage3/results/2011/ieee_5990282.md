### B. 알고리즘 요약

1. 대상은 AWGN/Rayleigh 채널의 medium/short binary QC-LDPC 부호 구성(rate 0.5는 (504,252)/(1008,504), rate 0.33은 (816,544)).
2. 풀려는 문제: PEG로 직접 만든 random LDPC는 구조 비고정이라 인코딩·저장 비용이 크고, 기존 PEG 기반 QC 확장은 6-cycle 일부만 제거하거나 girth 증가를 보장 못 함.
3. 채널·모델: 표준 AWGN 및 Rayleigh, 별도 간섭 모델 없음.
4. 핵심 기법 1단: PEG 알고리즘으로 작은 `base matrix`를 만들고, 각 "1"을 `p×p` circulant permutation matrix로 확장.
5. 핵심 식: cycle 형성 조건 `∑(-1)^{k+1} S_{αk,βk} = 0 (mod p)` (식 1) — 한 loop 안 shift값 교대합이 0이면 `2t` 길이 cycle 발생.
6. 핵심 기법 2단: Dijkstra 기반 cycle-search로 모든 길이 `L` cycle을 찾고, 식(1)을 위반하도록(=0이 안 되도록) shift값을 열 단위로 선택해 short cycle 제거 → girth +2.
7. 구현 디테일: 열별 cycle 수 내림차순 재정렬 후 shift 선택, 종료 후 원래 열 순서 복원; block이 `L-1`개 미만 cycle에 포함되면 해당 cycle 전부 제거 보장.
8. 학습/최적화: 없음(결정적 탐색 절차).
9. 결과: PEG-E-LDPC의 BER/FER이 random PEG-LDPC와 근접하고 원본 PEG-OE 및 단순확장 PEG-PE보다 우수(정성적, 그림 기반).
10. 한계: HW 미설계, 시뮬만, cycle-search 계산량·저장량 커서 작은 base matrix에만 실용적.
