### B. 알고리즘 요약

1. 대상은 protograph 기반 binary QC-LDPC 부호; base matrix `B`(nc×nv)를 circulant permutation으로 lifting해 QC 코드를 만든다.
2. 문제: binary(0/1) base matrix의 circulant lifting은 최소거리가 `(nc+1)!`로 상한이 걸려 girth·dmin이 낮게 고정된다.
3. 제안 2단계 lifting: (1) `m`-fold pre-lifting으로 pre-lifted base matrix `B'` 생성(m 작음), (2) `B'`를 `r`-fold circulant lifting → 최종 QC PCM `H`.
4. 핵심은 일부 permutation을 `r×r` circulant의 sub-array(block-circulant)로 구성하는 것: circulant 구조를 유지하면서도 원 protograph의 QC 상한을 넘긴다.
5. 상한 계산은 Smarandache-Vontobel의 permanent 기반 bound(Thm 1, `dmin ≤ min* Σ perm(B_{S\i})`)로 pre-lifted `B'`에 적용.
6. covering graph 선택이 중요: disjoint 1-cover 등 harmful substructure는 dmin을 낮추므로 sieve 원리로 걸러낸다(예: m=3에서 36→5→2 후보).
7. (2,3)-regular 사례: m=1,2,3,4에서 dmin=6,10,12,14 / girth=12,20,24,28로 단조 증가(표 I); 예 [54,19,8] girth16, [120,41,10] girth20.
8. (3,4)-regular 사례: circulant 상한 dmin≤24를 2-cover pre-lifting으로 dmin≤116까지 확장; r=17에서 [136,36,26] girth8, MAGMA로 상한 검증.
9. 탐색공간 대폭 축소: block-circulant P는 `r^m` 선택지(vs 임의 순열 `(mr)!`), 예 m=2,r=9면 81 vs 18!.
10. 한계: 순수 구성·이론(dmin/girth 상한만, BER 복호 시뮬 없음), HW 미설계, 예제는 저rate·짧은 블록에 국한.
