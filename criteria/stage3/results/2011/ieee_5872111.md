### B. 알고리즘 요약

1. 시스템: 복소 AWGN + `2^2m-QAM`(Gray labeling) 고차변조, rate `(2m-1)/(2m)` (m=2~5, 즉 3/4~9/10), PID(parallel independent demapping) + iid 채널 어댑터.
2. 문제: 고차변조는 비트레벨별 오류확률(비트채널 용량)이 불균등 → 내부 LDPC의 variable degree를 비트채널 용량에 정합해야 함, 기존 [2]는 구조 복잡·heuristic DE.
3. 모델: MET(multi-edge-type) 앙상블로 서로 다른 채널출력 밀도를 파라미터화, variable node는 edge-type 1개만, edge-type은 check node에서만 혼합.
4. 핵심기법: Gaussian 근사 EXIT를 다차원으로 확장 → mutual information 벡터의 IFS(iterated function system) `{I_v^{k,l}=f^k(...)}`, `f:D→D`.
5. 식 의미: 성공 복호는 IFS가 D 내에서 `1`에만 유일 fixed point 존재로 판정, 실패는 `1`이 아닌 fixed point가 지배 → 수렴조건 `|∇f|²≥ε`.
6. 확장: numerical gradient vector field `∇f`를 elementary EXIT 함수 `g_i^k`로 표현해 최적화, 이차 수렴제약을 최적화 문제에 편입.
7. 구현 디테일: 고차(64~1024-QAM)는 상위 비트레벨이 매우 신뢰도 높아, 저차(16-QAM) 최적앙상블에 edge-type 추가로 확장 → nested 구조, 최상위는 linear programming만으로 충분.
8. 최적화: 16-QAM은 Sec.IV-B(ε=2.0e-6), 64-QAM은 이차제약 확장, 256/1024-QAM은 LP 확장.
9. 결과: 16-QAM은 [2]의 best 앙상블(max var degree 15)과 threshold 정합, 16~1024-QAM 전 constellation에서 constrained PID capacity 대비 gap ≤0.21dB, nested vs 직접최적화 차 0.01dB 이내.
10. 한계: 순수 앙상블 설계 이론(DE threshold만), 실제 BER/HW 미검증, 고차 QAM 대역효율 무선 특화.
