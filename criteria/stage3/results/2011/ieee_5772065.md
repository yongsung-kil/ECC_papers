### B. 알고리즘 요약

1. 대상은 유한길이 concatenated code(예: (7,4)Hamming∘(6,2)SPC 직렬, (3,4) LDPC len60 rate1/4)의 MP 디코딩 성능 분석.
2. 문제: EXIT/density evolution은 무한길이 가정이라 단·중 길이 부호의 MP 성능 예측 불가.
3. 핵심 도구는 `pseudo-weight`(fundamental polytope의 vertex = pseudo-codeword) 기반 유한길이 분석.
4. `dual optimization decoder`(Algorithm 1): 등호 제약을 Lagrangian으로 이완, `M`개 부분문제로 분해, subgradient(식13) 또는 ellipsoid로 dual variable `λ` 갱신.
5. Theorem 2/3: dual decoder = generalized LP(GLP) decoder = MP decoder가 `SNR→∞`에서 목적함수 값 동일(강쌍대성).
6. Theorem 4: max-fractional weight `w_m`이 Hamming weight와 같으면 pseudo-weight spectrum gap>0 → ML 성능 수렴.
7. Theorem 5/7: 직렬·병렬 concatenated code의 max-fractional weight 하한을 component code weight 곱으로 유도.
8. Section IV 핵심 기법: `H_new=[H; h̄^T]`로 redundant parity-check 추가해 `w_m` 증대. `effective parity-check constraint`(Def.4) 개념 + Algorithm 2~4(ordered systematic matrix, Type-1/2 탐색)로 효율 생성.
9. 결과: (3,4)LDPC len60에 15개 redundant PC 추가 시 max-fractional weight 증가·BER 개선(랜덤 추가는 개선 없음/열화), 복잡도는 redundant 수에 선형.
10. 한계: 길이 36~60의 소규모 부호·시뮬만, HW 미설계, LP/dual은 반복 최적화라 실시간 디코더 부적합.
