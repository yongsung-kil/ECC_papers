> 총평: projective plane(PG(2,q))과 RS 기반 LDPC의 등가성을 GF(q) 연산표로 간명하게 보인 대수적 부호구성 이론 — high-rate QC-LDPC를 낳지만 디코더/HW/성능 실험이 없어 NAND ECC 이식가치는 낮음.

### B. 알고리즘 요약

1. 대상: `PG(2,q)`(q=p^s) 위 projective plane의 `N×N` incidence matrix(N=q^2+q+1)로 정의되는 `(J,L)`-regular LDPC 부호.
2. 문제: projective-plane LDPC와 RS(2 정보심볼) 기반 Gallager-type LDPC라는 두 부호류가 사실상 같은 부호임을 간명히 증명.
3. 핵심 표현: GF(q)의 덧셈표 `A`·곱셈표 `M`으로 incidence matrix `H_PG(2,q)`를 블록 구성(식(2)). 핵심 부분행렬 `HAM`은 `M`의 각 원소 `m_ij`를 `q×q` 특성행렬 `A(m_ij)`로 치환.
4. 핵심 성질: `A(y)`는 permutation matrix이며 GF(q) 가법군과 동형(`A(δ)·A(γ)=A(δ+γ)`). `(HAM)^2 = J_q`인 대수 구조.
5. bit-wise 표현: `HAM={h_(i,k),(j,l)}`가 특성함수 `χ`로 기술(식(5)) → PG(2,q) 위 1의 위치를 대수식으로 명시.
6. QC 구조: q가 prime이면 `C_i,j = A(a^{i+1}a^{j+1})`가 circulant → `HAM=[C_i,j]`로 만든 LDPC가 quasi-cyclic. 각 circulant는 첫 행의 1 위치 `(i·j) mod q`로 결정. girth 6, circulant 기반 부호는 girth ≤ 12.
7. RS 기반 부호(§III): RS `[q-1, q-p+1, p-1]` 부호를 2 정보심볼로 shorten한 `C^b`를 coset 분할, map (6)으로 이진화해 `Jq×pq` `(J,L)`-regular Gallager PCM `H_GA` 구성(length Lq, rate ≥ (L-J)/L).
8. 등가성 증명(§IV): modified RS 부호의 coset 분할·map으로 얻은 `h_(i,k),(j,l)` 표현이 PG incidence matrix의 식(5)와 (첫 열 제외) 동일 → `H_GA`는 `HAM`의 부분행렬, 역방향도 성립. 원 RS→modified RS 변환·coset 선택은 결과 부호에 무영향(열밴드 내 bit permutation).
9. 결과(예제): GF(2^6) full-length RS → RS기반 LDPC `(4032,3307)`, rate 0.82. 잉여비트 725 (이론상 상한 730 = 3s+1, s=6) 근사 일치. 성능은 [1]의 AWGN 시뮬 인용(capacity-approaching).
10. 한계: 순수 대수적 부호구성·등가성 증명(본 논문에 디코더·HW·자체 BER 실험 없음), girth 6~12 상한, 일반화(유한환·군쌍) 가능성만 언급.
