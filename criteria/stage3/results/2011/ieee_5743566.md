### B. 알고리즘 요약

1. 대상은 rate R=1/2 근처 (J,2J)-regular 및 ARJA protograph 기반 terminated LDPC convolutional code(SC-LDPC) 앙상블, lifting factor `N=6000`.
2. 문제: block LDPC의 BP threshold 개선을 위해 convolutional 구조를 termination하면 slight irregularity로 threshold가 MAP 근접, 이를 BEC 넘어 AWGN까지 확장 검증.
3. 구성: block base matrix `B`의 edge를 `B0+...+Bms=B`로 나누는 edge-spreading으로 convolutional base matrix `B[-∞,∞]` 구성, L 시점 termination해 `B[0,L-1]` 유한 앙상블 획득.
4. 예: (3,6)-regular는 `B=[3 3]`, `B0=B1=B2=[1 1]`, `ms=2`; termination rate `RL=1-((L+ms)/L)(1-R)`로 L↑시 R로 수렴(rate loss 존재).
5. AWGN threshold는 density evolution 대신 reciprocal channel approximation(RCA)로 근사 계산.
6. min-distance 성장률 `δmin`은 Divsalar 앙상블 weight enumerator 기법으로 계산 — L↑시 threshold는 개선되나 `δmin`은 감소하는 trade-off.
7. Trapping set 분석: (a,b) trapping set을 modified protograph의 codeword로 대응시켜 two-part asymptotic spectral shape `r(α,β)` 계산, `Δ=b/a` 고정 시 first zero-crossing `δts(Δ)`가 trapping-set 성장률.
8. `δts(0)=δmin`이며, terminated (3,6)-regular 앙상블은 모든 Δ≥0에서 `δts(Δ)>0` — 최소 non-empty trapping set이 block length에 선형 성장.
9. 결과: L 중간값에서 gap-to-capacity 작으면서 `δmin` 유지, L↑시 `δts(Δ)` 감소(L1>L2면 δts(L1)≤δts(L2)); (4,8)·(5,10)는 큰 L서 TARJA와 comparable threshold.
10. 한계: 앙상블 점근 이론뿐(HW/디코더 미설계), termination rate loss와 매우 큰 block length(N=6000~200000) 전제, waterfall·error-floor 개선은 시뮬/이론적 논거.
