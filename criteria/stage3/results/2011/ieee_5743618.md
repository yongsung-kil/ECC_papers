### B. 알고리즘 요약

1. 대상: separable circulant-based(SCB) QC-LDPC — `f(i,j)=a(i)·b(j)`로 분리되는 지수를 갖는 circulant 배열 부호, Tanner 구성(`f=a^i·b^j`) r=4(column weight 4), rate 0.86~0.92.
2. 문제: 중~고rate LDPC의 error-floor는 iterative decoding이 갇히는 absorbing set(bit-flip에 stable한 trapping set의 부분집합)이 주원인 — 이를 대수적으로 판별하고자 함.
3. 모델: SCB mother matrix `fm(i,j)=i·j`에서 행/열 선택으로 daughter 부호 생성, 1의 위치는 `k+ij=l mod p`(pattern consistency) 만족.
4. cycle consistency: length-2t cycle은 `Σ im(j_{m+1}-jm)=0 mod p` (식 2), 열그룹 차 `um=jm-j1`로 재작성해 `Σ(i_{m-1}-im)um=0` (식 3).
5. CCM 정의: absorbing set의 모든 cycle 식을 모아 `M·u=0 mod p`, M의 행은 binary cycle space(incidence matrix null space, `Dbcs=|Eas|-rank(Bas)`)의 선형독립 cycle에 대응, 열은 `|Vas|-1`개.
6. 핵심 결과(Theorem 1): absorbing set 존재의 필요조건은 CCM이 full column-rank가 아닐 것(+bit/check consistency); variable-node graph가 non-extensible이면 충분조건도 됨.
7. (4,4) absorbing set: variable-node graph의 incidence matrix rank 3, `Dbcs=3`, CCM det = `(z-x)(w-y)+(z-y)(w-x)=0 mod p`이면 존재 (Lemma 1).
8. Lemma 2: `a(i)=α^i, o(α)=4`인 Tanner r=4 부호는 `det M=2(1-α)(1+α^2)`이고 `α^2=p-1`이라 항상 `det M=0` → (4,4) absorbing set이 항상 존재.
9. 결과: 5개 고rate Tanner QC-LDPC (N=812~2756, rate 0.86~0.92)를 200 iter·Q4.2 fixed-point·soft-xor 디코더로 시뮬, error-floor 프로파일이 (4,4)·(6,2) absorbing set 지배 확인.
10. 한계: 부호 성능을 개선하는 것이 아니라 error-floor 유발 구조의 존재를 진단·증명하는 툴이며, HW 미설계·정정능력 직접 향상 없음(관련 개선은 [11][12] 후속연구).
