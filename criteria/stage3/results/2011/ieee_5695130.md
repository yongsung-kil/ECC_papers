> 총평: spatial coupling threshold saturation의 landmark 이론 논문이나 BEC 전용 asymptotic 증명이라 NAND용 고정 binary QC-LDPC 디코더/HW에 떼어 쓸 요소가 없다.

### B. 알고리즘 요약

1. 대상은 Felström-Zigangirov의 **convolutional/spatially-coupled LDPC 앙상블**(`(d_l,d_r,L)` 및 smoothing 파라미터 `w` 포함 `(d_l,d_r,L,w)`), rate 1/2 예제는 (3,6)-regular 기반.
2. 문제: 결합 LDPC의 BP threshold가 관측적으로 매우 높으나(용량 근접) 그 근본 메커니즘이 미증명이었음(Lentmaier-Fettweis 추측).
3. 모델: binary erasure channel(BEC), density evolution FP 방정식 `x = ε·(1−(1−x)^{d_r−1})^{d_l−1}` 및 EBP/BP EXIT 곡선.
4. 핵심 기법: L개의 `(d_l,d_r)` 앙상블을 좌우로 결합(coupling)하면 경계효과로 rate loss(design rate `≈(1−d_l/d_r)−O(1/L)`)를 치르는 대신 BP threshold가 상승.
5. 핵심 결과(Thm 10): `L,w→∞` 극한에서 결합 앙상블의 BP threshold `ε^BP(d_l,d_r,L,w)`가 underlying 앙상블의 **MAP threshold `ε^MAP(d_l,d_r)`로 포화**(threshold saturation).
6. 증명 확장: unimodal FP(중앙 constant, 빠른 transition, 경계에서 0) 존재 증명 → one-sided FP를 두 개 붙여 구성(Thm 27).
7. 구현/증명 트릭: FP 족을 보간(interpolation)해 인공 EXIT 곡선 구성, 채널 파라미터를 position별로 변화 허용하고 최소값으로 하한(느슨한 bound), Area Theorem으로 MAP threshold 상한 도출.
8. 최적화: smoothing 파라미터 `w`를 키워 EBP EXIT 곡선의 "wiggle"을 임의로 작게 만들어 연속계 근사.
9. 결과: (3,6) 예제에서 `ε^BP(∞)=0.4881`, `ε^MAP=0.4881`로 일치; L↑에 따라 BP threshold가 MAP threshold(rate 1/2에서 용량 0.5에 지수적 근접)로 수렴하는 수치 제시.
10. 한계: 순수 이론(HW/디코더 시뮬 없음), BEC·(3,6)-regular 한정 엄밀증명(일반 BMS/불규칙은 추측), 수렴속도 `O(1/L)`로 매우 긴 블록·큰 rate loss 필요.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| spatial coupling / convolutional LDPC 부호구조 | 대응 없음 | Prime ECC는 고정 block QC-LDPC([Load_PCM()](criteria/stage3/Prime_ECC_3.1_Claude/paper_screening_profile.md)) - SC-LDPC 부호구조 미지원 |
| density evolution / EXIT threshold 분석 | 대응 없음 | 앙상블 asymptotic 분석 도구이며 실제 min-sum 반복복호 연산이 아님 |
| BP threshold saturation (부호설계) | 대응 없음 | BEC 전용 이론, NAND 고전 binary QC-LDPC 부호설계에 직접 이식 불가 |

마지막 적용 가치: **낮음** — spatial coupling threshold saturation의 이론적 landmark이나 BEC·convolutional 앙상블 전용 asymptotic 증명으로, NAND용 고정 binary QC-LDPC block code의 디코더/인코더/HW에 이식할 구체 요소가 없다.
