### B. 알고리즘 요약

1. 시스템: AWGN 채널 무선통신, WiMAX(802.16e) QC-LDPC, code rate 1/2, `n=2304`, block matrix size `p=96`, soft-demapping 후 LLR 입력.
2. 문제: 표준 WiMAX irregular H는 복호 HW(비교기·연결배선)가 커서 저복잡도 구현이 필요.
3. 방법(부호): H를 A,B,T,C,D,E 6블록으로 분할하고 dual-diagonal 구조로 선형 인코딩(식 (1)~(2)), 일부 원소를 제거해 "modified WiMAX" irregular H를 구성(Fig.3).
4. 복호: belief propagation을 min-sum(MSA)으로 근사 — CN은 `sign` 곱 × `min|Lr(q)|` (식 (3)), BN은 LLR 합산 (식 (4)), 정지판정 (식 (5)).
5. 식 의미: CN의 min 근사로 `tanh` 없이 최소·차선최소만 계산 → HW 비교기로 구현.
6. HW: BNU(8-bit adder + clipping 6-bit, sign-magnitude↔2's complement 변환), CNU(sign XOR + CMP로 min/2nd-min 추출), one/two-port buffer, BNU/CNU Connect가 H 블록주소로 라우팅.
7. 핵심 트릭: CMP6/CMP5를 CMP2 단위로 합성 — 표준은 8-CMP6+4-CMP7=116 CMP2, modified는 11-CMP5+1-CMP6=86 CMP2. 비교단에 레지스터 삽입해 파이프라인화.
8. 양자화: 8-bit adder, 출력 clipping 6-bit; fixed-point 분석으로 8-bit wordlength면 충분(Fig.12).
9. 결과: modified WiMAX BER는 표준과 근접(Fig.10), 8 iteration에서 수렴 시작, 면적 약 26% 절감(결론).
10. 한계: 정정능력 개선 없음(표준 동등), 시뮬레이션만(실합성/게이트카운트/throughput 없음), WiMAX H 전용.
