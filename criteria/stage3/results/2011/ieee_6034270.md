### B. 알고리즘 요약

1. 대상은 AWGN 채널용 유한길이 irregular QC-LDPC로, waterfall·error-floor 모두 우수한 부호를 설계한다.
2. 문제: waterfall 최적화된 irregular degree distribution은 랜덤 구성 시 error floor가 높아지며, trapping set 완전 특성화는 (특히 irregular) 사실상 불가.
3. 모델: Tanner graph의 cycle이 iterative decoding 성능 저하 원인, cycle 유해도 지표로 ACE(approximate cycle extrinsic message degree) `ACE(ξ)=Σ(di-2)` 사용.
4. 핵심 기법: 원하는 degree distribution을 담은 작은 프로토그래프에서 출발해 순환(cyclic) N-lifting으로 QC 부호 생성, edge에 circulant permutation shift `I(dij)` 할당.
5. ACE 스펙트럼 `η=(η2,...,η2dmax)`(각 길이 2i cycle의 최소 ACE)을 목표로 설정, extremal ACE 스펙트럼이 좋은 error floor 함의.
6. 리프팅 설계: base graph의 TBC(tailless backtrackless closed) walk 중 inverse image가 저-ACE 짧은 cycle이 되는 "problematic" walk를 식별(Lemma 2~5), walk order `O(W)=N/gcd(N,d)`로 판정.
7. 알고리즘: problematic TBC walk에 최소 edge만 골라 permutation shift를 재할당(cyclic edge swapping)해 목표 ACE 스펙트럼 위반 제거, greedy로 η2부터 순차 최대화.
8. 부수: 작은 base code는 compact 기술(간단한 enc/dec)에 유리하나 큰 lifting degree N 필요, degree distribution 정확 구현엔 tradeoff.
9. 결과: 예제1 rate1/2 n=990에서 ACE 스펙트럼 (∞,∞,17,10,5)로 [9][10] 대비 개선, BP(100iter) FER/BER에서 error-floor 우수; 예제2 rate 1/2~7/8 rate-compatible에서 [2] 대비 특히 고rate·error-floor에서 크게 우수.
10. 한계: HW 미설계, AWGN 시뮬만, 알고리즘 세부 미공개, gate/throughput/양자화 정보 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| ACE 제약 순환 리프팅 H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | Prime ECC의 QC-LDPC H-matrix(base+circulant shift) 설계 단계에 error-floor 개선 리프팅 접목 가능 |
| QC circulant permutation shift 배치 | `ecc_data.h` `PCM_b`(base+shift) | 부호 구조가 동일(base matrix + circulant shift)이라 shift 설계 지침으로 활용 가능 |
| error-floor(trapping set/cycle) 개선 목표 | `decoder.cpp` `LDPC_Decoder()` (성능 수혜 측) | 부호개선이 min-sum 복호의 error-floor에 직접 기여 |

적용 가치: **중간** — Prime ECC와 동일한 QC-LDPC(base+circulant) 구조를 대상으로 error-floor를 낮추는 부호설계라 `Load_PCM` 단계에 원리 접목 가치가 있으나, 알고리즘 세부 미공개·AWGN 시뮬만·고rate(0.9+) 미검증으로 재설계·재검증 부담이 있다.
