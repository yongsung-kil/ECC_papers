### B. 알고리즘 요약

1. 대상은 circulant 기반 QC-LDPC의 Tanner 그래프에서 나타나는 `balanced cycle`(각 edge가 짝수 번 등장하는 cycle) 구조 분석이다.
2. 문제: 유한 길이 LDPC 성능은 girth·stopping set 등 cycle 성질에 좌우되나, 랜덤 부호는 이를 명시적으로 분석·예측하기 어렵다.
3. QC-LDPC는 mother matrix(base matrix)에서 lifting되므로 cycle 성질을 대수적으로 더 쉽게 분석할 수 있다는 점을 활용.
4. balanced cycle을 정의: 길이 `2k` cycle에서 임의 edge `e`가 짝수 번 등장 → 최단 balanced cycle 길이를 `B-girth` `g_B(M)`로 정의(최소값 12).
5. Lemma 1~10으로 path/cycle 대수(경로 결합, 역경로, cyclic path 분해)를 세운 뒤, balanced cycle이 `(|J1|,|J2|,|J3|)`-cycle 또는 `(|C1|,|C2|,|J0|)`-cycle 형태로 구성됨을 보인다.
6. Theorem 1~2: `B-girth`가 유한 ⇔ simple cycle의 배수가 아닌 cycle 존재 ⇔ 서로 비등가인 연결된 두 simple cycle 존재 등 필요충분조건을 등가로 제시.
7. Lemma 11: 정수 `a,b,c` 조건 (5)(6)으로 minimal 행렬 `S(a,b,c)`를 명시 구성하고 그 B-girth를 `4c` 또는 `2(a+b+c)`로 폐형식 도출.
8. Lemma 12~13, Corollary 1~2, Theorem 3: 임의의 유한 B-girth 행렬은 `S(a,b,c)` 형태의 BC*-minimal 부분행렬을 가짐을 증명(최소 B-girth 판정 알고리즘 근거).
9. 결과: mother matrix에서 B-girth를 판정하는 간단한 방법을 이론적으로 확립(수치·시뮬 결과 없음).
10. 한계: 정리·증명뿐이며 부호설계 절차, 디코더, 성능곡선, HW 어느 것도 없음.
