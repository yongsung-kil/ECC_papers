> 총평: non-binary LDPC의 binary image 성능격차를 RPC로 메우는 BEC 이론 논문으로, binary QC-LDPC 기반 NAND ECC에 이식 가치 낮음.

### B. 알고리즘 요약

1. 시스템: BEC(binary erasure channel), non-binary LDPC over `GF(q=2^b)`. 예제: `C4`(GF(4), nq=96, kq=32, dl=2,dr=3), `C8`(GF(8), nq=100, kq=50, PEG 구성).
2. 문제: NB-LDPC는 성능 우수하나 BP 복호 복잡도 큼. 복잡도 낮은 basic binary image(BP(HB))로 복호하면 성능 열화. 원인 규명·회복이 목표.
3. 핵심 관찰: NB-LDPC를 동형사상 `ΦB`로 basic binary image `HB`로 변환하면 짧은 cycle이 많아지고 stopping set 수가 증가 → BP 성능 저하.
4. 이론 기법: extended binary image `HE = [HPM Inq⊗HS(b)]` (simplex code로 각 심볼 확장, `ΨM(h)`는 permutation 행렬). BP(HE)가 BP(Hq)와 등가임을 이용.
5. 핵심 명제 5: `SE ⊂ SB` (extended image의 stopping set이 basic image의 부분집합) → non-binary 우수성의 분석적 설명.
6. 핵심 기법: `SB \ SE`의 각 stopping set에 대해 basic binary image용 redundant parity-check(RPC) `hS`를 찾는 알고리즘(Algorithm 1). HE 상의 부분 peeling 복호와 computation tree 순회로 RPC 도출.
7. 구현 디테일: 저weight stopping set부터 RPC 추가(상위 weight set도 함께 제거 가능). RPC의 check degree 최소화 최적화는 미구현.
8. 결과: C4는 weight≤12/≤15까지 RPC 추가(93/154개), C8은 weight≤8/≤10까지(60/185개). enhanced binary image의 FER/BER가 non-binary 부호에 근접하면서 basic image 대비 낮은 복잡도 유지.
9. 한계: BEC 전용 이론. AWGN 등 error 채널로의 확장 미해결(논문 결론에서 open). HW/throughput/gatecount 없음, non-binary 대상.
