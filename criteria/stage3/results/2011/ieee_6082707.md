### B. 알고리즘 요약

1. 시스템: DVB-S2 위성 수신, AWGN + 시변 carrier phase noise, DVB-S2 64K LDPC(표준부호)를 채널복호기로 사용, PL header/pilot 필드 활용.
2. 문제: 저가 LNB/LO의 위상잡음으로 PLL 기반 위상추적이 cycle slip을 겪어 수신 성능 저하, 기존 CBC 연구는 실구현 절차 미기술.
3. 채널 모델: 위상잡음을 Wiener random walk `θ_{k+1}=θ_k+Δ_k`, `Δ_k~N(0,σ_Δ)`로 모델링.
4. 핵심 기법: joint iterative detection & decoding — 채널제약(위상)으로부터 `LLR_up`을 매 디코더 iteration 전에 재계산, LDPC 외인정보 `Pr_down(s)`를 되먹여 갱신.
5. ML 접근: 위상을 `L`값으로 이산화한 trellis에서 BCJR(α,β forward-backward)로 위상 marginalize, `p(k→ℓ)≈p_Δ((ℓ-k)2π/L)`.
6. 확장: trellis length reduction(시간 aggregation, 식13)으로 느린 위상변화를 병합, state 수는 못 줄임.
7. Sub-ML 기법: CBC 알고리즘 — canonical(Tikhonov) 분포로 파라미터화해 BCJR을 1차원 forward-backward 복소수 재귀 `α_k, β_k`로 축소(식 14~15).
8. 절차/구현: (1)축소 trellis BCJR→(2)심볼확률 marginalize→(3)bit LLR deinterleave 후 LDPC decoder에 투입(horizontal shuffle 스케줄 권장, extrinsic 감산)→(4)CBC 재실행 반복.
9. 결과: QPSK/8PSK/16APSK에서 기존 DPLL 대비 SNR 이득, horizontal shuffle 스케줄이 vertical 대비 실구현상 최선(정성 PER 곡선, 수치 본문 서술 없음).
10. 한계: 위성 carrier recovery 특화, LDPC는 기성부호 외부결합, HW 미설계·시뮬만, gate/throughput 정량 없음.
