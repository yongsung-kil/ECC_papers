### B. 알고리즘 요약

1. 대상은 AWGN 채널용 binary QC-LDPC 부호설계로, circulant sub-matrix 크기 `p=128`, VN degree 3, info block 4096(코드워드 4864), rate 0.50~0.91 예제.
2. 문제: PEG-QC-LDPC은 circulant 구조로 메모리는 `1/p`로 줄지만 원본 PEG 대비 girth(짧은 cycle) 특성이 열화됨.
3. 채널/모델: 표준 AWGN, sum-product(BP) 복호 가정 — 별도 채널 모델 없음.
4. 핵심 기법: PEG 알고리즘으로 심볼노드마다 edge를 확장하되, check node degree가 최소인 노드를 골라 `p×p` circulant sub-matrix를 생성해 그룹 단위로 H에 삽입.
5. 핵심 아이디어: 매 심볼노드에서 local girth(트리 확장 깊이)를 계산하여 collected check node들과 비교, **최대 local girth** 조합을 선택함으로써 짧은 cycle 제거.
6. 확장: 이미 수집된 check node 시퀀스와 유사하면 girth 비교 후 최대치 선택, 다르면 새로 저장하고 해당 edge 삭제 후 전 과정을 반복.
7. 구현 디테일: H는 `H_{z,j}`가 `p×p` circulant 또는 all-zero인 블록 행렬 형태로 표현, 첫 컬럼이 전부 0이면 null matrix 배치.
8. (학습/NN 절차 없음 — 결정론적 그래프 구성.)
9. 결과: local girth 분포(표I)에서 제안법이 모든 rate에서 girth 최대화; rate 0.84에서 BER 10⁻⁷ 기준 PEG-QC 대비 SNR gain 약 0.005 dB, iter 50에서 BER 개선.
10. 한계: HW 미설계, SNR gain이 0.005 dB로 극히 작고, 고rate(0.89~0.91)에서는 기존 대비 girth 개선이 거의 없음(표I).
