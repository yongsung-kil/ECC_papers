> 총평: bit-flipping(hard-decision) 디코딩의 이론적 실패율/redundancy 분석 논문으로, NAND binary LDPC ECC 이식 관점에서 실용적 가치가 낮음.

### B. 알고리즘 요약

1. 대상은 길이 n의 binary linear code (left-regular Tanner graph, 열 가중치 c 고정), parity-check matrix H가 full-rank일 필요 없음(r ≥ n-k).
2. 문제의식: BIKE 등 post-quantum crypto에 쓰이는 moderate-density parity-check(MDPC) 부호에서 bit-flipping 복호 실패율(decoding failure rate) 분석이 Tanner graph의 cycle 때문에 점근적 분석으로 다루기 어려움.
3. Gallager의 bit-flipping 알고리즘: 각 변수노드의 불만족 이웃 체크노드 수 `ui`를 계산, `ui > c/2`인 변수노드 중 최대값을 뒤집고 syndrome을 갱신, 모든 `ui ≤ c/2`이면 종료.
4. 핵심 정의(pseudoredundancy `ρ`): 최소거리 d인 부호 C에 대해, bit-flipping이 `⌊(d-1)/2⌋`개 오류까지 항상 정정 성공하도록 하는 parity-check matrix의 최소 행 수.
5. Expander code 이론(Sipser-Spielman)을 적용: `(c,d,c/2)`-expander이면 최소거리 > d, `(c,d,3c/4)`-expander이면 bit-flipping이 `d/2`개까지 정정.
6. Finite geometry(projective plane 등, no 4-cycle을 갖는 partial geometry) 기반 부호에서 Proposition 1: 블록(열) 교집합 크기 제약으로 bit-flipping이 `c/2`개 오류까지 정정 가능함을 증명, 이를 이용해 Hamming/simplex code의 pseudoredundancy 상한(`ρ ≤ n`, `ρ ≤ n(n-1)/6`)을 유도.
7. 보조 도구로 Tanner의 eigenvalue bound(H^T H의 λ1, λ2로 최소거리·확장성 하한)를 적용해 projective plane 부호 예시 검증.
8. t=2, t=3 소수 오류 개수에 대해 블록 교집합 크기(`s12, s13, s23` 등) 조건으로 bit-flipping 성공/실패 경계를 조합론적으로 분석(closed-form 없음, 케이스별 증명).
9. 결과는 수치 실험이 아닌 상한식(예: `[15,7,5]`, `[21,11,6]` 부호의 유한 pseudoredundancy 존재 증명)뿐, BER/iteration 등 정량 성능 지표 없음.
10. 한계: 순수 이론(정리·증명) 논문으로 시뮬레이션·HW 구현 전무, 결론에서도 무한 pseudoredundancy를 갖는 구체적 부호 예시나 향상된 bit-flipping(black-gray decoder 등) redundancy는 열린 문제로 남김.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| bit-flipping 하드결정 디코딩 실패율 이론 | 대응 없음 | Prime ECC는 min-sum soft decoder 중심이며 순수 hard-decision bit-flipping과 다른 알고리즘 계열 |
| finite geometry 기반 부호(projective plane 등) H-matrix | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 QC-LDPC 고정 구조라 finite-geometry 부호 자체는 대응 없음 |
| pseudoredundancy(정정 실패 없는 최소 parity 행 수) 이론 | 대응 없음 | 코드베이스에 해당 이론적 redundancy 분석/검증 모듈 없음 |

> 적용 가치: 낮음 — hard-decision bit-flipping의 이론적 redundancy 분석으로 Prime ECC의 min-sum soft 디코더·QC-LDPC 구조 어디에도 직접 대응되지 않음.
