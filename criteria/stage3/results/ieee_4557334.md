### B. 알고리즘 요약 (10줄 내외)

1. Binary message-passing LDPC 디코더(check/variable node가 1-bit 메시지만 교환)를 EXIT chart로 분석; hard decision 채널에서는 Gallager B와 동치.
2. 기존 Gallager A/B는 메모리·복잡도가 낮지만 성능 손실이 크다는 문제; variable node가 모든 입력을 L-value 도메인으로 변환·합산 후 hard decision하는 일반화된 구조로 확장.
3. 채널 모델: BIAWGN을 K-ary quantizer(`ζ = [ζ0,...,ζK-1]`)로 양자화, 부채널(sub-channel) `K+1`개의 BSC로 분해(`ε_ch,k`, `p_k`).
4. Check node EXIT 함수: `f_c(ε_ac; d_c) = (1-(1-2ε_ac)^{d_c-1})/2` (degree `d_c`); Variable node EXIT 함수는 binomial CDF `B(k; d_v-1, ε_av)` 기반 (식 8-9).
5. 핵심 정리(Theorem 1): 대칭조건 `g(-l)=e^l g(l)` 만족 시 평균 EXIT 함수=평균분포의 EXIT 함수이나, binary message-passing에서는 quantization(비선형 연산)으로 이 성질이 깨져 crossover probability 도메인에서 선형 평균(`E_ev = ΣΛ_i E_ev,i`)해야 함.
6. 코드 최적화: variable node degree distribution `Λ(x)`를 `d_v,max=100`, `d_c∈[2,100]` 범위에서 EXIT 부등식 제약(`f_v(ε) > f_c^-1(ε)`)을 선형계획법(LP)으로 풀어 rate 최대화.
7. 부수 트릭: 실제 구현 시 extrinsic channel 파라미터를 모른다는 문제를 decoding trajectory로 crossover probability 시퀀스를 예측해 대체(genie-aided 지식 불필요).
8. 채널 예시로 BSC(hard), BSEC(erasure), BSQC(4-symbol quaternary), soft 채널 4종의 quantization 파라미터(`ζ0=1.69`, `ζ1=1.90` 등)를 EXIT 면적 최대화로 최적화.
9. 결과: rate 0.5 코드(N=10^4, PEG 구성)에서 BSC→BSQC(1비트→2비트급 quantization) 전환 시 1.0dB 이상 이득, unquantized soft 대비 gap 0.25dB로 축소. rate 0.9 soft 코드는 capacity 대비 0.5dB 이내.
10. 한계: 디코더는 여전히 1-bit(binary) 메시지 교환 구조로 min-sum 계열과 근본적으로 다른 알고리즘이며, HW 구현·게이트카운트·throughput 분석은 전혀 없음(순수 이론+시뮬).

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 채널 K-ary quantization / soft-read threshold 설계 | channel.cpp Set_R_Offset(), Set_LLR_Th() | Prime ECC의 hard-1bit/soft-2~3bit LLR 임계값 설계와 개념적으로 유사한 quantization 최적화 기법이나 EXIT 기반 해석적 도출 방식은 별도 적용 필요 |
| Degree distribution 최적화(LP) | ecc_top.cpp Load_PCM() (H-matrix 설계 관련) | Prime ECC는 고정 QC-LDPC(VN degree 17) 구조라 irregular Λ(x) 재설계는 이식 난이도 높음 |
| Binary message-passing 디코더(Gallager B) | decoder.cpp LDPC_Decoder() | Prime ECC는 min-sum 계열이라 1-bit message-passing 알고리즘 자체는 대응 없음(별개 계열) |
```

적용 가치: `중간` - 디코더 알고리즘(binary message-passing)은 Prime ECC의 min-sum과 다른 계열이라 직접 이식은 어렵지만, 채널 quantization 단계별 EXIT 기반 최적화 개념(soft-read 비트수 증가에 따른 이득 정량화)은 LLR 테이블/threshold 설계 시 참고 가치가 있음.
