### B. 알고리즘 요약

1. 시스템은 SMR(HDD)로 128/130 (0,16/8) RLL + (3,30) regular LDPC 인코딩, granular medium(Voronoi) 모델에 기록, 3-reader array head로 target·인접 트랙 동시 read.
2. 문제: BAR(bit aspect ratio) 증가 시 ISI 영향이 커져 반복복호 신뢰도가 떨어짐 - 기존 LLR modulator는 자기전이 정보를 안 씀.
3. 채널 모델: PR1 target equalization(LPF+TD-FIR Nt=15), 각 파형에 AWGN(system noise, `SNRS,3` 정의), APP 디코더와 SP 디코더 간 turbo equalization(`iglobal`=25, `isp`=15).
4. 핵심 기법: APP 디코더 출력 LLR 시퀀스 `xk`를 3층 신경망(sigmoid)으로 변조하는 LLR modulator - 입력은 APP LLR `{ak}`와 자기전이 정보 `{ck}` 두 종류.
5. 핵심 식: `ck = bk − bk−1` (hard-decision `{bk}`에서 자기전이 추출) - ISI를 유발하는 전이를 명시적 특징으로 넣어 신뢰도 판단에 반영.
6. 확장: 3-level 판정 `{dk}`(NN 출력 `yk(3)`>0.9→1, <0.1→0, else 0.5)와 coincidence detector로 SP 입력 `{fk}={ak}·{ek}` 생성(일치 시 1.6배 가중).
7. 구현 디테일: 소형 NN(`N1APP=16`, `N1TR=16`, `N2=6`, `N3=1`), backprop 학습.
8. 학습: 기록 시퀀스 `yˆk`를 교사신호로 오차 `E` 역전파(식 3,4)로 가중치 갱신.
9. 결과: 제안 modulator가 BAR≥2.0에서 무LLR변조 대비 약 1.0dB, 기존 modulator 대비 BAR≥3.0에서 약 0.5dB SNR 개선(no-error 기준).
10. 한계: HDD/SMR 채널 특화 시뮬레이션만, HW 미설계, iteration 수 감소는 목표 아님(고정 반복).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LLR 변조(NN) | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `Get_VNU_Table_Idx()`(개념적) | LLR 조정이라는 접점뿐, NN/자기전이 특징은 대응 없음 |
| SP(sum-product) 디코더 | 대응 없음 | Prime ECC는 min-sum 사용, full sum-product 아님 |
| 자기전이 정보 `{ck}` 특징 | 대응 없음 | SMR 자기기록 고유 - NAND 채널에 존재하지 않음 |
| APP/turbo equalization | 대응 없음 | HDD PR 채널 이퀄라이제이션, NAND ECC 범위 밖 |

적용 가치: **낮음** - NN이 SMR 자기전이 특징에 의존해 LLR을 변조하는 채널특화 기법이라 NAND binary QC-LDPC min-sum 디코더에 떼어 쓸 요소가 없음.
