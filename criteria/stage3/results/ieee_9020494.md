### B. 알고리즘 요약

1. 반복 최적화 알고리즘을 고정 반복수 `T`로 unfold하여 각 반복을 NN 계층으로 매핑하고 학습 파라미터를 삽입하는 deep unfolding 패러다임을 정리한 서베이.
2. 문제의식: 실시간 통신은 반복수(<=10)가 극히 제한되어 step-size 등 파라미터 수동 튜닝이 열위 -> 학습으로 최적 파라미터 확보.
3. MIMO 검출/프리코딩: projected gradient/OAMP 반복을 unfold, 학습 스칼라(`gamma_t`,`theta_t`) 도입해 near-ML 성능을 저복잡도로 달성.
4. BP 복호(핵심 관련 파트): Tanner 그래프 message-passing `(22)~(25)`에 학습 가중치 `w_v^t`,`w_c^t`를 곱해 BP 개선 `(26)` (Nachmani [19]).
5. 개선식 의미: 채널 LLR `l_v`와 이전 CN 메시지에 학습 가중치를 부여해 표준 BP 대비 BCH 등에서 유의미한 이득.
6. 확장: RNN 구조([20][21])로 반복 간 가중치 공유, relaxation으로 수렴 개선.
7. min-sum 변형: normalized MS `(28)`에 학습 `w`, normalized-offset MS `(30)`에 per-message/per-iteration 학습 `alpha`,`beta` 도입.
8. 학습: multi-loss/BCE 손실, soft bit error 손실 `(31)`, incremental/windowed training으로 vanishing gradient 완화.
9. 결과 수치: unfolded normalized MS가 polar에서 표준 대비 ~0.5dB, finite-alphabet 3bit 양자화 unfold가 (1296,972) QC-LDPC에서 최대 0.25dB 이득.
10. 한계: 서베이라 자체 HW 설계·gate/throughput 없음, 대부분 무선/polar 대상이고 HW 복잡도는 미고려로 명시.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| unfolded BP/min-sum 학습 가중치 | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 이론상 min-sum CN/VN 루프에 학습 스케일 삽입 가능하나 학습 파이프라인 부재로 원거리 |
| normalized/offset MS 파라미터 | `decoder.cpp` `CNU_Update_New_Mag()` | Prime ECC는 min1/min2 min-sum 보유, 학습 없는 고정 테이블이라 개념만 겹침 |
| finite-alphabet 양자화 LLR 학습([30]) | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | LLR 테이블 자동최적화(`local_opt.cpp`)와 목표 유사, 다만 NN unfold 방식은 대응 없음 |
| MIMO 검출/프리코딩 unfold | 대응 없음 | NAND ECC와 무관 |

적용 가치: **낮음** — 서베이라 이식할 구체 기법·수치가 없고 학습 기반 NN unfold는 Prime ECC의 고정 QC-LDPC/테이블 구조와 정합 부담이 크다. 인용 원논문 추적 인덱스로만 유용.
