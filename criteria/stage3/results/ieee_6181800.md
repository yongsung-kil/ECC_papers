### B. 알고리즘 요약

1. binary QC-LDPC 부호를 CPM/ZM 배열의 null space로 정의하고, 그 배열을 Fourier 변환 도메인에서 구성한다 (rate/length는 예제별: (63,37), (1023,877), (4095,3367)).
2. 기존 finite-field 대수적 구성들을 하나의 변환 도메인 프레임워크로 통합, 랭크·girth 계산을 단순화하려는 것이 목표다.
3. 순환행렬 `A`의 Fourier 변환 `AF = VAV⁻¹`(Vandermonde `V`)는 대각행렬이 되고, 생성벡터의 Fourier 변환이 대각원소 `bt`를 준다.
4. base matrix `B` (GF(2^r) 위)에서 `HF,π = diag(B∘0,...,B∘(2^r−2))` Hadamard power 배열로 CPM/ZM 배열 `H`를 array dispersion으로 직접 생성한다.
5. RC-constraint(girth≥6, 최소거리≥γ+1)는 `B`의 모든 2×2 부분행렬이 zero를 포함하거나 non-singular일 때(2×2 SM-constraint) 보장된다.
6. 2차원 Euclidean geometry `EG(2,2^r)`의 line incidence vector를 c=2^r+1 섹션으로 나눠 permute하면 (2^r+1)×(2^r+1) base matrix `B`를 line 하나로 얻는다.
7. m차원(≥3) EG로 확장하면 K개 cyclic class에서 여러 base matrix `Bi`를 쌓아 다양한 rate/length의 대규모 부호족을 만든다.
8. 특정 subclass(`CEG,qc`)는 RC-constrained (γ,ρ)-regular이며 최소거리 미만 크기의 τ/κ<1 유해 trapping set이 없음을 Theorem 2로 보증한다.
9. 결과: (4095,3367) 부호는 최소거리 65, 5/10/50 MSA iteration 간 BER 10⁻⁶에서 0.1dB 차이(빠른 수렴), 50 MSA는 SPA 대비 0.1dB 이내.
10. 한계: HW 설계 없음, AWGN 채널·SPA/MSA 시뮬만, error-floor는 실측이 아닌 최소거리 논거로 추정.
