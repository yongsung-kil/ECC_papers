### B. 알고리즘 요약 (10줄 내외)

1. 시스템: 3-D TLC/QLC NAND(64-stacked, 8-state ER~PV7), page(LSB/CSB/MSB) 구성, retention·P/E에 따른 Vth 분포 변화 대상.
2. 문제: retention Vth 분포 실측은 시간·비용 과다, 다양한 변동원(전자요동/디바이스/데이터패턴)을 물리모델로 분리하기 어려움.
3. 채널/모델: read voltage optimization(RVO)은 정확한 Vth 분포에 의존 → 분포를 데이터 기반으로 예측.
4. 핵심 기법1: 입력 `f1=P/E cycle p`, `f2=retention time t` (n=2), 은닉 1층 1000노드, 출력 261노드(Vth bin별 cell 수)인 BpNN으로 분포 회귀.
5. 핵심 식: `ms=h1(Σ fi·w1+a)`(sigmoid), `yk=h2(Σ ms·w2+b)`(linear), MSE 목표로 가중치 학습, PV 레벨별 별도 학습(PV1~PV6).
6. 핵심 기법2(전처리): 출력범위가 넓어(0~1000+) 로그 변환 `tk'=Namp·log10(tk)`으로 crossover 영역 variability 억제.
7. 후처리: BpNN 고유 ripple 제거 위해 max-based·increment-based boundary detection 필터(kernel 1×f, f=10)로 좌우 경계 추출.
8. 학습 데이터: 12 group 학습 / 8 group 테스트(Table I), 온도 85℃ 조건.
9. 결과: 1.6~20.56h retention, 3K~10K P/E에서 예측 분포가 실측과 잘 일치, 예측 분포로 산출한 optimal read voltage(Vopt)가 실측 기반과 거의 동일(soft는 14 reads 양자화 모델).
10. 한계: LDPC 복호/HW 미포함, SSD 컨트롤러 임베딩은 제안 수준(가중치 테이블), 특정 3-D TLC 측정에 종속.

| Vth 분포 기반 read voltage / soft-read threshold(14 reads) | [channel.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `Set_R_Offset()`, `Set_LLR_Th()` | soft-read threshold·LLR 양자화 캘리브레이션 입력으로 간접 참고 가능 (알고리즘 이식 아님) |
| LDPC 복호/부호구조 | 대응 없음 | 논문이 복호기·부호설계를 다루지 않음 |
