### B. 알고리즘 요약

1. 시스템: AWGN 채널 + polar code 인코딩, 16QAM(length 1024, rate 0.5) / 64QAM(length 1536, rate 0.33) 전송 시스템에서 검증.
2. 풀려는 문제: soft-decision 디코더 입력인 LLR을 정밀 계산하는 log-MAP은 constellation 전점 traverse+지수/로그 연산으로 복잡도가 `M`에 비례해 고차 변조에서 과도.
3. 대안 max-log-MAP은 구간별 선형함수 근사라 비선형인 정밀 LLR과 오차 발생, 매핑 규칙 제약도 있음.
4. 핵심 기법: fully-connected BP 신경망으로 (SNR, 수신신호 실/허수부) 2입력 → 각 비트 LLR 출력을 학습해 정밀 LLR 관계를 fitting.
5. 의미: NN 추론 복잡도가 `log2 M`에 비례하여, 고차 변조일수록 log-MAP 대비 복잡도 절감 효과가 커짐.
6. 망 구조: 16QAM은 hidden 2층(크기 4,4)+출력 2뉴런, 64QAM은 hidden 3층(4,16,8)+출력 3뉴런. 출력 뉴런 수 = 변조 차수.
7. 구현 디테일: hidden 활성 logsig, 출력 활성 purelin, loss MSE, epoch 50.
8. 학습 절차: SNR 1-15dB에서 80,000 세트 학습 / 20,000 세트 테스트, trainlm 훈련함수, learngdm 가중치학습, 학습률 0.1.
9. 결과: 저SNR에서는 두 알고리즘 차이 미미, 고SNR에서 NN LLR이 정밀 log-MAP BER 성능에 근접(유사 BER, 복잡도는 대폭 감소).
10. 한계: HW 미설계 순수 시뮬, polar/QAM demapping 대상이며 LDPC·NAND 부호 자체와 무관.
