### B. 알고리즘 요약

1. BSC(binary symmetric channel)에서 hard-decision 반복복호(Gallager A/B, majority-based MB0 등)로 복호되는 유한길이 LDPC 부호(수백~수천 비트)의 BER/FER을 저복잡도로 추정하는 문제를 다룬다.
2. 기존 방법([11])은 복호 실패하는 최소 무게 J와 그 개수 `|EJ|`를 구하기 위해 무게 `≤J`인 모든 오류패턴을 직접 열거(direct enumeration)해야 해 복잡도가 `n^J`로 지수적으로 증가하는 문제가 있다.
3. 핵심 관찰은 반복복호가 실패하는 저-무게 오류패턴 대부분이 Tanner graph의 짧은 cycle의 부분집합이라는 것.
4. 핵심 기법은 girth `g`부터 시작해 길이 `l=g,g+2,...`인 cycle 집합 `C_{l/2}`(무게 `l/2` 오류패턴)를 열거하고, 각 원소의 부분패턴(무게 1~`l/2`)을 복호기에 넣어 실패 여부로 `J`, `|E_i|`를 갱신한다.
5. FER/BER 추정식은 `FER ≈ P(J) + Σ|E_i|·C(n-J,i-J)/C(n,i)·ε^i(1-ε)^(n-i) + Σp_i`로, `|E_J|`를 안다는 가정 하에 나머지 오류패턴 확률을 조합론적으로 근사한다(`P(J)=|E_J|ε^J(1-ε)^(n-J)`).
6. 확장 단계로 두 번째 라운드(J+1)까지 진행 시 이중계수 방지를 위해 `|E_i|_est = |E_J|·C(n-J,i-J) + (|E_{J+1}|-|E'|)·C(n-J-1,i-J-1)`로 추정식을 개선한다.
7. cycle 열거의 복잡도는 `O(n(d_v d_c)^⌈l/4⌉)`로 direct enumeration 대비 무시 가능한 수준이며, 반복복호 호출 횟수(`Σ|C_i|(2^i-1)`)가 지배적이지만 이마저 direct enumeration 대비 수 자릿수 감소(예: τ~10^2~10^7배 절감).
8. 별도 학습 절차는 없고, 8개 regular/irregular 부호(GA/MB0 decoding)에 대해 Monte Carlo 시뮬레이션과 비교 검증했다.
9. 결과: 대부분 부호에서 초기 추정(1라운드)만으로 BER/FER이 시뮬레이션과 정확히 일치, 일부(Code 4)는 2라운드에서 정확도 개선; 극히 일부(Example 1, PEG 504,252 부호)는 실제 J를 놓치는 사례(oscillatory trapping set이 짧은 cycle에 속하지 않는 경우) 발견.
10. 한계로 추정 오차의 tight upper bound가 없고, 고밀도(고rate) 부호나 J>4인 경우 cycle 기반 탐색이 비효율적이며 별도의 trapping-set 구조 기반 탐색이 필요하다고 저자 스스로 명시; HW 설계나 min-sum/soft-decision 복호와는 무관.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Tanner graph cycle 기반 error-floor 추정 알고리즘 | 대응 없음 | Prime ECC는 min-sum soft-decision 시뮬레이터이며, 이 논문은 GA/GB/MB0 hard-decision 전용 error rate 추정(성능 예측) 방법론으로 디코더 로직 자체가 아님 |
| H-matrix/Tanner graph 구조 (cycle 분석 대상) | `ecc_top.cpp` `Load_PCM()` | 부호 구조 분석 관점에서만 참조 가능, 실제 이식 대상 아님 |
```

적용 가치: **낮음** — HW 미설계, hard-decision(GA/GB/MB0) 전용 해석적 error-floor 예측 도구로 Prime ECC의 min-sum 디코더/부호설계에 직접 이식할 알고리즘 요소가 없음. 다만 부호설계 단계에서 trapping-set/short-cycle 기반 error-floor 사전평가 아이디어는 참고 가치가 있을 수 있음.
