> 총평: Sipser-Spielman의 기념비적 expander code 논문으로 asymptotically-good LDPC 계열의 선형시간 bit-flip 복호를 이론적으로 확립했으나, NAND용 soft min-sum QC-LDPC HW ECC에 떼어 쓸 알고리즘·부호구조는 없다.

### B. 알고리즘 요약

1. 대상: expander 그래프에서 유도한 선형부호(LDPC/Tanner 계열). left `n` 변수 노드 = 부호비트, right clause 노드 = subcode(짝수패리티 or 일반 선형부호) 제약.
2. 문제: 상수 rate·거리를 유지하는 asymptotically-good 부호 중 선형시간 복호가 가능한 최초 부류를 제시.
3. 모델: 이진 대칭 오류(비트 손상), hard-decision. 채널모델 없이 오류 fraction `α`로 정정능력 규정.
4. 핵심기법1(부호구성): unbalanced bipartite `d`-regular(변수)/`c`-regular(clause) 그래프 `B`와 subcode `S`로 `C(B,S)` 정의. Theorem 1: 팽창 `>c/2`면 rate `dr-(d-1)`, 최소거리 `≥αn`.
5. 핵심식: 팽창률 조건 `|N(X)| > (3/4)d|X|` 등이 성립하면 손상 변수 대부분이 unsatisfied clause에 둘러싸여 flip으로 오류 감소.
6. 핵심기법2(병렬 복호): 각 라운드 clause가 패리티 계산→변수는 이웃 clause 과반이 unsatisfied면 flip. `O(log n)` 라운드, 라운드당 상수시간, depth `O(log n)`·size `O(n log n)` 회로.
7. 부수 기법(sequential 복호): unsatisfied 이웃 수로 변수 리스트 `L_i` 유지, 최대 `i` 변수부터 flip → 선형시간(Prop.3), `αn/2`까지 정정(Theorem 4). 역으로 복호 성공 ⇒ 그래프가 expander(Theorem 5).
8. 최적화/구성: LPS/Margulis Ramanujan 그래프 + Gilbert-Varshamov subcode로 명시적 무한족 구성(Theorem 9/12).
9. 결과: 랜덤 (20000 변수 deg5 / 10000 clause deg10, rate 1/2) 부호 실험 — sequential은 최대 ~730~750 오류, parallel은 ~950 오류까지 정정(sphere-packing 상한 2205 대비 참고).
10. 한계: 순수 이론+소규모 시뮬. hard bit-flip만, soft-info·양자화·QC 구조·HW·gate count 전무.
