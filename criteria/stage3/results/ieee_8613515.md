### B. 알고리즘 요약

1. 대상은 (2,6)-regular protograph 기반 QC-GLDPC: SPC노드의 fraction `ν`를 (6,3) shortened Hamming GC노드로 치환, 예제 N=490(s=83 prime lifting).
2. 문제: 5G URC는 저rate(R=1/12)·short block(480/2400bit)에서 낮은 BLER 필요, polar은 SC 순차성으로 throughput 한계.
3. 모델: AWGN + QPSK, `RJ,K,ν = R0 − ν(1−R0)(k−1)`; BEC asymptotic 분석으로 rate-threshold tradeoff 최적점 도출.
4. 기법1(비율 최적화): ν=0.75에서 gap-to-capacity 최소(rate 1/6), GC노드가 국소 정정능력 제공해 빠른 수렴·낮은 error floor.
5. 핵심식: GC노드 C2V 업데이트 `Λ̃j`를 component code codebook `C` 기반 log-sum-exp로 계산(식5) — SPC 대신 (n,k) 부호 제약 반영.
6. 기법2(QC lifting): all-ones base matrix에 circulant shift `si,j=a1·b^j mod s`(Power construction)로 girth g=12 확보, dominant error object(4-cycle 쌍, 8-cycle) 제거.
7. 구현 디테일: VN degree J=2라 VN은 계산 없이 message 전달만, GC노드 배치는 local SPC 밀집 회피 위해 300개 랜덤 샘플 중 선택.
8. rate 매칭: 내부 GLDPC(1/6) + outer hard-decision (83,40) shortened BCH(≤20 err 정정)로 목표 R=1/12 달성, outer가 잔여 에러 정리.
9. 결과: outer(20err) 결합 시 polar+SCL 대비 BLER 10⁻⁵에서 ~2dB 이득, error floor 미관측(10⁻⁸까지), iteration당 복잡도 23655, Imax=10에서 50 대비 ~0.05dB 열화만.
10. 한계: 순수 시뮬(HW/RTL 미설계), 유한정밀도 message passing 미분석, 초저rate·wireless 특화로 NAND 고rate와 무관.
