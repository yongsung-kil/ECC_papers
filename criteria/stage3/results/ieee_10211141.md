### B. 알고리즘 요약

1. 시스템: BICM 코드변조 - binary GC-LDPC 인코딩 → interleaver → M-ary QAM(QPSK/64QAM) 매핑 → AWGN 채널 → soft demod(LLR) → deinterleaver → LDPC 복호.
2. 문제: 스펙트럼 부족 상황에서 고차변조와 FEC를 결합할 때, 기존 girth-6 GC-LDPC는 저rate 설계 시 error-floor가 심함.
3. 부호: globally-coupled LDPC(GC-LDPC) - t개의 local code `H_iL`을 global check `HX`로 결합한 계층 구조 (식 2).
4. 핵심 기법 1단: GCD-기반 FLRM(full-length row multiplier) 행렬 `E`(식 4)로 부호 구성, PEM(partition/edge-spreading/mask)로 최종 exponential matrix 생성 (인용 [11] Construction 2).
5. 핵심 개선: girth를 6에서 8로 확장하고 CPM 최소크기 `P*_min=(a_{J-1}-a_0)(L-1)+1` 을 줄이는 알고리즘으로 error-floor 저감.
6. 부수 트릭: cyclic mask 행렬 `M`(식 5)으로 local code column weight를 4로 낮춰 채널/보호수준별 유연 조정.
7. 복호: local/global 2-phase 복호 - 고SNR에서 local code만 독립 복호하여 전체 복호량 감소 (Fig.3). BP는 Min-Sum, scale factor 0.5, 최대 iteration 50.
8. 비교대상: 5G-LDPC(QC-LDPC, BG1/BG2, quasi-double-diagonal `B`)를 유사 파라미터로 구성.
9. 결과: 64QAM에서 GC-GCD가 5G-LDPC 대비 약 0.5~0.6dB 이득(@BER=1e-6), 단 QPSK에서는 0.4dB 이상 열세. GC-RS 계열은 심한 error-floor. EXIT chart로 이득 뒷받침.
10. 한계: 시뮬레이션(BER/EXIT)만이며 HW 설계·복잡도·throughput 수치 없음. 타깃이 QAM 무선/멀티미디어 방송, NAND 언급은 인용문헌 [10] 참조 수준.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| GC-LDPC 부호구성(FLRM/PEM/mask) | 대응 없음 | Prime ECC는 QC-LDPC(base+circulant) 고정, globally-coupled 구조 미보유 |
| local/global 2-phase 복호 | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `LDPC_Decoder()` | 스케줄만 유사, GC 부호구조 전제라 그대로 이식 불가 |
| Min-Sum(scale 0.5) BP | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `CNU_Update_New_Mag()` | 이미 보유(min-sum)라 중복, 신규 기여 없음 |

적용 가치: 낮음 - QAM 코드변조용 GC-LDPC 설계로 Prime ECC의 고정 QC-LDPC 부호족·NAND 타깃과 정합되지 않고, 복호 기법(min-sum)은 이미 보유.
