### B. 알고리즘 요약

1. 대상: 3GPP 5G NR downlink transport channel의 QC-LDPC(base graph 1: 46×68, base rate 1/3~8/9; base graph 2는 미변경), 기존 최대 lifting `Zmax=384`(코드워드 최대 25344bit)를 방송(MBMS/FeMBMS)용으로 확장.
2. 문제: 5G NR은 unicast 전용으로 짧은 코드워드만 지원(최대 25344bit)하는 반면, DVB-T2/ATSC 3.0 등 방송 표준은 최대 64800bit 장문 코드워드를 사용해 스펙트럼 효율이 더 높음 - 5G 기반 방송 서비스 도입 시 성능 격차 우려.
3. 핵심 가정: AWGN 채널, QAM 변조(QPSK~256-QAM), 기존 3GPP Rel.15 표준의 shift coefficient matrix `P_set`은 그대로 유지하며 lifting size 집합만 확장.
4. 핵심 기법: lifting size set(Table II)에 상위 값(최대 `Zmax'=1920`, 기존 384의 5배)을 추가해 `K_CB,max,BG1 = Kb×Zmax = 22×1920 = 42240bit`, 최대 코드워드 길이 `N_CW,max = 66×Zmax = 126720bit`로 확장.
5. 핵심 식: transport block size 계산식 `N_info = N_RE·R·Q_m·v` 및 code block segmentation 로직(§II-A~C)은 그대로 두되 `K_CB,max,BG1`, `Zmax` 상수만 대체 - 표준 최소 변경 원칙.
6. 부수 트릭: rate matching의 circular buffer 크기를 `ζ=Zmax'/Zmax=5`배 확장, HARQ 미지원 방송 특성상 buffer 시작점은 항상 systematic bit부터 고정.
7. 구현/검증: IfN Generic SDR Toolkit(C++)으로 인코더/디코더 구현, 디코더는 offset min-sum + horizontal-layered scheduling(x86 SIMD 가속), ATSC 3.0 비교를 위해 AFF3CT 툴박스 확장 사용.
8. 학습/최적화 절차: 없음(표준 파라미터 확장 및 시뮬레이션 비교뿐, shift matrix 재설계는 future work로 남김).
9. 결과: 동일 spectral efficiency 기준 ATSC 3.0 대비 기존 5G NR은 Es/N0 0.48dB 열세였으나 확장 후 격차가 0.20dB로 절반 이상 축소; 전체 downlink 체인 BER 기준 QEF 도달점이 평균 0.3dB 개선(MCS 9~19 구간에서 특히 뚜렷).
10. 한계: shift coefficient matrix 자체는 미설계(향후 과제로 언급), HW 합성/게이트카운트 없음(x86 SDR 소프트웨어 검증), 메모리 요구량이 최대 5배 증가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Lifting size 확장(Zmax 384→1920) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 `z_sb=32` 고정 병렬단위라 5G NR식 lifting size 가변 확장 개념은 구조적으로 상이(직접 이식 불가) |
| Offset min-sum + horizontal-layered scheduling 디코더 | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | Prime ECC도 min-sum 계열이나 이미 보유한 기법이라 중복 기여 |
| Rate matching(circular buffer, shortening/puncturing) | `encoder.cpp` `LDPC_encoder_4KB()` | Prime ECC 인코더는 dual-diagonal 고정 구조로 5G NR식 circular buffer rate matching과 대응 없음 |
| 장문 코드워드(최대 126720bit)를 통한 spectral efficiency 개선 | 대응 없음 | Prime ECC는 NAND 페이지 단위(4KB급) 고rate 짧은 코드워드 대상이라 방송용 초장문 코드 확장과 무관 |

적용 가치: `낮음` — 방송(MBMS) 서비스를 위한 5G NR 표준 파라미터(lifting size) 확장 논문으로, 부호설계·디코더 알고리즘 혁신이 없고 대상 채널·코드 특성(저~중rate, 초장문, 방송 AWGN)이 Prime ECC의 NAND 고rate 짧은 QC-LDPC와 크게 다름.
