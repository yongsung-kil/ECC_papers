### B. 알고리즘 요약 (10줄 내외)

1. 대상은 [[n,k]] 양자 stabilizer code(CSS code, QLDPC), 독립 Pauli-σx 오류채널(확률 `px`) 및 depolarizing 채널(σx/σy/σz 각 p/3)에서의 syndrome decoding 문제.
2. 기존 BP 기반 QLDPC 디코더는 Tanner graph의 짧은 cycle과 코드의 degeneracy(같은 syndrome을 만드는 여러 오류패턴 존재)로 인해 비수렴(non-convergence) 실패가 잦은 문제.
3. 채널모델: 각 qubit이 독립적으로 Pauli-σx 오류(px)를 겪는 binary 모델과, σx/σy/σz 오류(p/3씩)를 겪는 quaternary(depolarizing) 모델을 각각 다룸.
4. 핵심 기법: 표준 sum-product BP(식 8,9, `tanh`기반 C2V/V2C 갱신)를 `T` iteration 라운드 단위로 실행하고, 매 라운드 비수렴 시 가장 신뢰도(`γ(vi)=|m_v^(t)|`) 높은 변수노드 하나를 골라 채널 LLR을 `±llrmax`(예: 25)로 고정(decimation, 식 12)하는 BPGD(Belief Propagation Guided Decimation) 알고리즘.
5. 핵심 식 의미: decimation은 BP가 막힌 지점(짧은 cycle/degeneracy로 인한 진동)에서 가장 확실한 변수부터 hard-fix해 나머지 그래프의 자유도를 줄여 재수렴을 유도하는 역할.
6. 확장: quaternary BP(Q-BP, CSS code σx/σz 동시 디코딩)에도 동일 원리를 적용한 Q-BPGD(식 14, 4값 중 최대확률 성분으로 고정)를 제시.
7. 부수 트릭: `llrmax=∞` 대신 유한값(25) 사용해 수치 불안정 방지, quaternary는 `ϵ=1e-10`으로 근사 hard-fix.
8. 최적화 절차 없음(학습 기반 아님, 결정론적 알고리즘).
9. 결과: B1 code([[882,24]])에서 BPGD(T=100)가 BP-OSD order-0, BP-SI(λ=10)와 대등한 성능을 선형방정식 계산 없이 달성, worst-case 복잡도 `O(Tn²)`로 BP-OSD의 `O(n³)`보다 낮음(BP-SI와 유사).
10. 한계: 하드웨어 미설계, 시뮬레이션(BER curve)만 제시, average complexity는 오류율(px)에 따라 크게 달라짐(Table 1), classical binary LDPC가 아닌 quantum stabilizer code 전용 syndrome decoding 문제.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP(sum-product) syndrome decoding 전체 | decoder.cpp LDPC_Decoder() | Tanner graph message-passing이라는 틀만 유사, Prime ECC는 min-sum이며 quantum stabilizer 문제 자체가 대응 없음 |
| Guided decimation(신뢰도 최고 변수노드 순차 hard-fix) | decoder.cpp VN_Cal_HD() | 개념적으로 비수렴 시 강제 hard-decision 고정이라는 아이디어만 유사, 실제 trapping-set 완화용 후처리 메커니즘은 Prime ECC 미보유 |
| BP-OSD/BP-SI 등 선형방정식 기반 post-processing | 대응 없음 | Prime ECC는 syndrome 선형시스템을 푸는 post-processor 없음 |
| Quaternary BP(GF(4) 상당 확장) | 대응 없음 | Prime ECC는 binary LDPC 전용, non-binary/quaternary 디코딩 미보유 |
```

적용 가치: `낮음` - 양자 오류정정 코드의 syndrome decoding 문제로 채널·코드 구조가 NAND binary LDPC와 본질적으로 다르며, decimation 아이디어의 개념적 참고 외에는 이식 가능한 모듈이 없음.
