### B. 알고리즘 요약

1. 시스템: insertion/deletion(ID) 채널(bit-patterned/racetrack 메모리의 동기화 오류), 각 입력비트가 확률 `pid/2`로 삭제/삽입, hidden Markov drift state `s_t∈[-S,+S]`, `S=8`.
2. 문제: 지난 20년간 irregular LDPC는 마커 없이 ID 오류를 거의 복호 못함, 마커 없이 SIR 근접 부호가 존재하는지.
3. 모델: state 기반 채널표현(식1 전이확률, 식2 출력확률), SIR을 Arnold-Loeliger로 수치평가하여 capacity 대용.
4. 핵심 기법: degree분포쌍 `(λ(x),ρ(x))`을 differential evolution(DE)으로 전역 최적화, 목적함수는 tolerance threshold `pth` 최대화.
5. 핵심 식: 관례적 good-degree 제약을 버리고 전 해공간 탐색, Kavcic density evolution으로 임계값 평가(FB 메시지가 비Gaussian이라 EXIT 대신 사용).
6. 확장: 복호는 채널 subgraph에 forward-backward(FB) + 부호 subgraph에 sum-product를 교대 반복(식4 factorization).
7. 구현 디테일: DE 파라미터 `G=500, Z=240, F1=F2=0.5`, min check degree 제약(식9)으로 low-degree CN 기여 분석.
8. 최적화: DE mutation/selection 반복으로 `pth` 최대 degree쌍 선택, PEG로 실제 부호 구성.
9. 결과: 마커 없는 irregular 부호가 SIR limit에 근접(R=0.417 threshold 0.1248 vs SIR 0.1278, gap 0.0030), 마커/SC-LDPC보다 우수, degree-2 CN이 SIR 근접에 필수(drmin>=5는 복호 거의 불가).
10. 한계: HW 미설계·시뮬만, ID(동기화) 채널 전용, 높은 λ2로 저BER error-floor 우려, high-degree 부호는 DE 계산량 한계.

```

| sum-product 복호(부호 subgraph) | decoder.cpp LDPC_Decoder() | Prime는 min-sum, full SP 미보유 |

| LLR/양자화 | 대응 없음 | 논문은 실수 LLR·양자화 미다룸(무관) |
```

적용 가치: 낮음 — insertion/deletion 동기화 채널 전용 irregular LDPC 부호설계 + FB/SP 결합복호로, Prime ECC의 binary QC-LDPC(고rate, AWGN/RBER, min-sum, HW)와 채널·부호구조·복호구조가 모두 달라 이식 실익 없음.
