### B. 알고리즘 요약

1. 대상은 WDM 광패킷 링 네트워크의 OOK(on-off keying) 변조 채널, LDPC(6176,5983) 코드(BIBD 구성, redundancy 5.6%~3%), 패킷 길이 100000bit 고정, data rate 10Gbps 기준.
2. 풀려는 문제는 기존 LDPC soft decoder 설계가 OOK 광채널 노이즈를 Gaussian으로 가정하지만 실제로는 광증폭기 ASE(amplified spontaneous emission) beating으로 인해 비대칭·비음수(non-negative) 분포를 가져 부정확하다는 점.
3. 핵심 모델: photocurrent를 4개 제곱 가우시안 변수의 합으로 표현, order-4 noncentral chi-square 분포 `f(y)`(식 3, non-centrality parameter s와 modified Bessel function I1 포함)로 채널을 재정의.
4. 핵심 기법 1단: 부호설계는 BIBD-LDPC(balanced incomplete block design) 결정론적 구성법[9] 사용, 두 파라미터(열 수, 소수 파라미터)로 코드 차원 결정, 이는 short girth를 회피해 수렴을 보장.
5. 핵심 식은 chi-square 채널 통계에서 유도한 새 LLR 식(식 8) — 기존 AWGN LLR(`2y/σ²`) 대신 chi-square pdf 비율로 LLR을 재계산해야 decoder가 정확한 extrinsic 정보를 사용함.
6. 확장 단계: AWGN 채널에서 먼저 코드 rate/length를 정하고(NCG 요구치 4.5dB 만족, 실제 6.5dB 확보), 이후 실제 chi-square 채널에 적용해 성능 재평가.
7. 구현 디테일 특별한 양자화/HW 근사 없음 — 순수 소프트웨어 시뮬레이션 수준의 BP 디코더.
8. 별도 학습/최적화 절차 없음(코드 파라미터는 data rate 최대화 기준으로 탐색적으로 선택, 결정론적 BIBD 구성).
9. 결과: chi-square 채널에서 정확한 LLR(식 8) 사용 시 AWGN 가정 LLR 대비 BER=10^-9에서 약 3dB 이득, chi-square와 AWGN 채널 간 effective NCG는 동일(약 6.5dB 이상).
10. 한계: HW 미설계(순수 이론/시뮬), 채널 모델이 광통신 ASE 노이즈 전용이라 NAND flash 채널과 물리적으로 무관, 코드 자체는 표준 BIBD-LDPC 구성으로 새로운 정정능력 개선 기법 없음.

|---|---|---|
| chi-square 채널 LLR 재계산식 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | 채널 통계에 맞춘 LLR 산출이라는 개념은 유사하나 대상 채널(광증폭 ASE)이 NAND read 채널과 달라 식 자체는 이식 불가 |
| BIBD-LDPC 구성법 | `ecc_top.cpp` `Load_PCM()` | 결정론적 permutation 기반 girth 확보 구성법으로 H-matrix 생성 방식 참고는 가능하나 Prime ECC는 이미 특정 QC-LDPC 고정이라 재검증 부담 큼 |
| 디코더(BP/SPA) | `decoder.cpp` `LDPC_Decoder()` | 표준 BP 디코더 그대로 사용, 신규 기법 없음(대응은 있으나 기여 없음) |
