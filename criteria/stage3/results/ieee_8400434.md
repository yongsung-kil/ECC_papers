### B. 알고리즘 요약

1. 대상은 헬리콥터-위성 채널로, 회전날개가 안테나를 주기적으로 가려 발생하는 rectangular window fading(최대 감쇠 `Ab`=10~30dB)을 blockage ratio `γ=Td/Tp`(대개 <25%)로 모델링, 채널은 SNR이 다른 두 AWGN 인스턴스로 취급.
2. 문제: 대부분 QC-LDPC는 AWGN 설계라 소거성 오류에 약하고, SPC-constrained GLDPC는 성분부호 여분 check bit 전송으로 심한 rate 손실.
3. 핵심 기법 - Check-Hybrid(CH) LDPC: `(a,b,p)` QC-LDPC의 layer 구조 `HQC=[H0;...;H_{a-1}]`에서 **한 layer(중간 layer 권장)의 SPC 제약만** 더 강력한 선형블록 성분부호 `C(nc,kc)`로 대체(super check node화).
4. rate는 `R=(b-a)/(b+nc-kc-pc+1)`(식4)로, 한 layer만 대체해 GLDPC 대비 rate 손실 완화(예: 전체 대체 rate-0.083 → CH rate-0.1875).
5. 성분부호는 SPC 제약(systematic G에 all-ones 열) 필요, RM/BCH/simplex 후보 사용, simplex는 항상 SPC 만족(전처리 불필요).
6. 복호: VND/SPC-CND는 sum-product, super check node는 trellis 기반 MAP(저복잡도 MAP 사용), VND는 통신·외부 채널 모두 수신.
7. 최적화 - EXIT chart: 기존 Monte Carlo 대신 split-encoders 모델로 성분부호 EXIT 함수 `IE,Cmp`의 해석적 표현을 BEC(식7)→AWGN(식8) 유도, base QC-LDPC 고정 후 최적 성분부호 탐색.
8. 최적화 규칙: `ΓV > ΓC^{-1}` 수렴조건 하에 `Eb/N0|0` 최소화(min over C(nc,kc)); split information function의 다양성으로 코드마다 threshold 상이.
9. 결과: 25% blockage에서 Example1(rate-1/4, len 2892)은 RM/BCH threshold 1.507dB, Example2(rate-2/7, len 427)는 simplex 1.995dB; rate-2/9 QC-LDPC 대비 대역효율 28.6% 향상, 짧은 length로도 rate-2/7 QC-LDPC보다 우수.
10. 한계: HW 미설계, 시뮬만, super check node MAP 복호가 필요하고 저rate·소거채널 특화, NAND read soft 판정과 무관.
