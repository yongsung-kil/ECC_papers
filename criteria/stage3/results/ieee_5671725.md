### B. 알고리즘 요약

1. 대상은 MIMO-BICM 수신기(16-QAM, 4×4 안테나, convolutional code rate 1/2, block 2016bit), MIMO detector와 채널 디코더 간 iterative demapping/decoding.
2. 문제: 외부 iteration 수가 energy·latency·throughput을 좌우하는데 고정 iteration은 낭비 → 채널 동특성에 맞춰 iteration 동적 제어 필요.
3. 기여 초점: 채널코드 내부가 아닌 **MIMO detector↔decoder 외부 iteration** 조기종료(기존 연구와 차별점).
4. 시스템: SISO sphere decoder가 Max-Log-Map 근사로 bit LLR `λ(xq,m)` 산출, extrinsic `λe=λ-La`만 디코더로 전달, 디코더 extrinsic `Le`를 다시 detector로 환류.
5. 제안 메트릭 `Mit`: 디코더 출력 extrinsic `Le`의 mutual information을 한 iteration LLR만으로 근사 계산(식13), 수렴 블록은 MI→1.
6. HW 친화 변형: division 회피 위해 `Nc·Mit` 사용, LLR 6bit 양자화 + 짝함수라 32-entry LUT 2회 참조로 구현.
7. 조기종료 조건: `Cit`를 threshold와 직접 비교하거나 `Cit-Cit-1` 차분 비교, threshold θ가 iteration수 vs FER trade-off 결정.
8. 비교군: Sign-Change Ratio / Re-Encoding / Mean-Reliability / Channel Capacity / (하한) Magic Genie와 알고리즘 복잡도(정보비트·iteration당 연산수) 대조.
9. 결과: FER 1% 타깃에서 mutual-information 메트릭이 평균 2 iteration 미만, 중·고 SNR에서 기존 메트릭 대비 우수, 양자화 손실 0.2dB 미만.
10. 한계: HW 미구현(알고리즘 분석·시뮬만), 채널코드가 convolutional(LDPC는 threshold 재튜닝으로 적용 가능 언급뿐), 무선/MIMO 특화.
