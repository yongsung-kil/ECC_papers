### B. 알고리즘 요약

1. 시스템: binary linear block code (n,k), BSC(crossover `p`), hard-decision. 예제 부호 다수(QC/Tanner/MacKay/Margulis LDPC), rate 0.4~0.903, length 155~18432, offset min-sum·Gallager-A/B 디코더.
2. 문제: ANN으로 임의 선형부호를 ML 복호하려면 클래스(codeword) 수가 `2^k`로 지수 증가 → 순수 ANN ML 복호는 비현실적. error-floor는 `10^-5` 이하가 필요해 학습 난도 큼.
3. 모델: 복호를 구간분할 분류문제로 정식화 — 채널출력 `r∈[0,1]`을 `2^k` 구간으로 나눠 `f(x)=ci`.
4. 핵심 기법(이론): Cybenko 정리 기반 SLFN이 ML을 `|FER_ANN−FER_ML|<p(1−p)^{n−1}/2^k` 오차로 모사(Thm1). SLFN은 `2n−2^k` 뉴런 필요(Thm2), 복잡도 `O(2^n)` → trellis Viterbi보다 대개 비효율.
5. 핵심 식: error-floor 근사 `FER ≈ Ct·p^{⌈t/2⌉}` (t=최저 uncorrectable 오류패턴 weight, Ct=그 개수) — ANN이 `Ct`개 패턴만 학습하면 됨을 정당화.
6. 확장: TLFN(2 은닉층)은 Huang 결과로 SLFN 대비 branch ~33% 감소하나 여전히 대부분 부호에서 trellis보다 복잡.
7. 구현 디테일: 실효 스킴 — 반복 디코더 `D`를 먼저 돌리고 복호 실패(syndrome≠0) 시에만 `D`의 출력을 저중량 uncorrectable 패턴 `Bt`로 학습된 ANN에 넘겨 후처리. 평균 복잡도만 미미하게 증가(`Nb,avg`).
8. 학습/최적화: 사전계산된 trapping-set 프로파일에서 저중량 오류패턴 수집(He[8]의 대량 시뮬 학습과 대조), ReLU SLFN 2M+1 weight.
9. 결과: Margulis(C5) 부호 Gallager-B에서 weight-4 잔여 패턴 교정으로 error-floor 1자릿수↓. C1 예: p=0.01에서 FER `1.9e-4→1.4e-5`.
10. 한계: BSC·hard-decision 한정, ANN 실제 아키텍처/게이트/throughput 미제시, min-sum이 아닌 Gallager-A/B 위주, HW 미설계, post-processor라 실패 시 latency 악화.
