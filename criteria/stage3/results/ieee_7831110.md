### B. 알고리즘 요약

1. 시스템: 차세대 SDON 광트랜시버용 variable-rate non-concatenated `QC-LDPC`, OH 20/18.18/16.66%(길이 24576/26624/28672 bit), 256Gb/s dual-pol 16QAM 32GBd.
2. 문제: 소프트 FEC는 iteration `Ni`↑로 coding gain↑이나 latency·전력도↑, 채널(OSNR)별 최적 `Ni`가 달라 동적 조절이 필요.
3. 핵심 기법 - 조기종료 결합: `C1`(soft-output 안정 시 중단)과 `C2`(syndrome=0 또는 max iter) 두 기준을 결합해 평균 iteration 최소화.
4. 의미: C1은 정정불가 codeword의 iteration 낭비 방지, C2는 정정가능 codeword에서 최소화 → 결합이 평균전력·throughput 최적.
5. 버퍼 설계: BER threshold 0.025에서 평균 iteration `N=9.6`, 입력/출력 큐 버퍼(10 codeword, 1.68Mbit)로 조기종료로 인한 latency 변동을 흡수하고 상수 latency 유지.
6. 성능 등가: 이 방식으로 9.6 평균 iteration HW로 29 iteration 성능 달성, 버퍼 saturation error floor를 10^-20 미만 유지.
7. HW arch: RCP(Regular Column Partition) QC 기반 recursive partial-parallel, recursive check-node + non-recursive variable-node로 고rate(열≫행) 대응.
8. 부수 트릭: 2 codeword 동시 복호(interlaced)로 pipeline penalty 회피·throughput 50%↑, CN 입출력단 timing 6배 완화 + message serialization으로 배선 혼잡 감소; 조기종료 유휴 사이클에 zero-data 삽입해 HW 부분 off.
9. 결과: 28nm CMOS ASIC 실측, NCG 11.4/11.3/11.2dB @BER 10^-15 error floor 없음, pre-FEC BER 0.025→0.012일 때 전력 약 35% 절감.
10. 한계: 광통신 OSNR/16QAM 채널 특화, gate count 미보고, NAND 실채널/soft-read 비용 미고려.
