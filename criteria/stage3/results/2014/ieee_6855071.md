### B. 알고리즘 요약

1. 문제: unsorted 리스트 `L=(x0..xk-1)` (각 `n`비트)에서 임의 순위 `g`(1~k)의 max/min 값과 인덱스를 저지연 HW로 추출; LDPC/turbo CN의 min1/min2 탐색이 대표 응용.
2. 기존 한계: 선행 HW는 `m=1`(단일 max) 또는 `m=2`(min1/min2)에 국한, 일반 `g`번째를 다루지 못함.
3. 정의: `maxg = max(L \ maxg1-1)` — 이전에 찾은 g-1개 최대값을 제거한 리스트의 최대값(재귀적 정의).
4. 핵심 기법(COMP): 각 입력 `xi`를 0~2^n-1 모든 값과 병렬비교해 thermometric 코드 `ci,j=1 if xi>j` 생성.
5. 선택(MUX_SEL): threshold `j`를 통과한 입력 수가 `g` 이상이면 `selj_g=1`; 대칭함수 `selj1,selj2,...`를 parallel-prefix로 계산.
6. 재귀식: `Sk,l = Sk,l-1 + cl,j`로 부분항을 중복 없이 누적 → 복잡도가 `k`에 선형 증가.
7. 출력: `selj_g`가 멀티플렉서 뱅크를 제어해 g번째 값을 출력; 트리형 MUX로 지연을 `2^n`→`n`으로 축소.
8. 복잡도: 면적 `A=O(g·2^n·(n+k))`, 지연 `D=O(log2 k + log2 n + n)` → 저비트폭·대집합에 유리(LDPC BP는 `n=6~8`, `k`=node degree 32+).
9. 구현: VHDL 기술, Synopsys DC + 90nm CMOS 합성; 조합형은 1사이클 처리, 파이프라인형은 클록주기 0.32ns 달성.
10. QCA: NML/molecular QCA로도 구현 — NML은 저전력, molecular은 초고주파(약 1THz 이론) 소형이나 전력 증가.
11. 한계: 면적비용 큼(부호rate/length·throughput 등 코드 스펙 없음, LDPC 통합 성능평가 없음).
