### B. 알고리즘 요약

1. 대상: sub-20nm TLC/MLC NAND Flash 컨트롤러용 multi-mode binary LDPC ECC 엔진, code rate `0.893~0.941`, code size `1KB~2KB`.
2. 문제: 기존 flow는 채널상태별로 고정된 소수의 LLR table을 쓰며, fit을 위해 optimized level tracking에 다수 read가 필요 → read disturb·throughput 저하.
3. 세 모듈 구성: Dynamic Information Scaling(DIS), low-power LDPC decoder, DSP 모듈.
4. DIS: hard-bit 정보 + parity check matrix로 high-reliable error를 특성화하여 hard-bit를 pseudo-soft-bit로 변환.
5. DIS 효과: 수렴속도 향상 → decoding throughput 최대 `53%` 증가(pure hard-bit 대비).
6. DSP 모듈: NAND soft 정보를 채널정보로 재구성 후 on-the-fly noise cancellation + reliability scaling으로 adaptive LLR table 계산.
7. 효과: optimized level tracking과 LLR retry를 생략 → 추가 latency와 read disturb 제거.
8. 알고리즘 레벨: scheduling decoding으로 iteration 수 절반 감소. 구조 레벨: CNU/VNU partial-parallel로 throughput-area 균형.
9. low-power engine: 60 error 이내는 대부분 모듈을 shut-off; 45 error 시 `21.8mW`로 `>400MB/s` 달성(40nm CMOS).
10. 결과: retention 신뢰도 BCH 대비 `20배`, 기존 LDPC 대비 `2~5배`; 평균 정정 error bit는 BCH의 `3.4~3.9배`, 기존 LDPC의 `1.6~2.1배`.
11. 한계: 부호종류/H-matrix, gate count, 부호길이 비트수 미공개; DIS/DSP 내부식이 회로지 요약 수준으로 서술 부족.
