### B. 알고리즘 요약

1. 시스템: IEEE 802.16e(WiMAX) LDPC, code size `2304`, rate `1/2`, 16QAM 4×4 MIMO 무선 채널 대상.
2. 문제: 기존 저전력 기법(DVFS[7], adaptive wordwidth[8])은 동기회로라 전압·주파수 스케일링 협조 시 타이밍 위반 위험과 제어 부담이 큼.
3. 핵심 기법 1단: min-sum(BP) 디코더의 VNU/CNU를 `precision-scalable`(4/6/8-bit)로 만들어 목표 FER에 필요한 최소 비트만 계산 -> 스위칭 감소.
4. 핵심 식: `Q_nm = y_n + Σ R_mn` (VN update), `R_mn = (Π sign)·min|Q_nm|` (CN min-sum) — 이 연산의 비트폭을 가변화.
5. 비동기회로: REQ/ACK 핸드셰이크(clock-free)로 각 stage 완료를 보장, 저정밀 계산 시 전압을 낮춰도 주파수 재조정 오버헤드 없음(Fig.3).
6. VNU 구현: adder-tree 합산 후 자기입력 뺄셈, full-adder carry-in을 control로 gating해 비트 정밀도 조절 -> critical path 단축.
7. CNU 구현: 절대값 변환 후 sorting으로 min/sub-min 추출, BCU(bit comparator unit)를 MSB->LSB로 concat해 정밀도 clip 및 조기결정.
8. 양자화/클리핑: 각 정밀도마다 최적 LLR clipping window를 exhaustive search — MSB 편향은 낮은 SNR에 유리하나 error-floor 상승, LSB 편향은 floor 완화하나 waterfall 지연.
9. 결과: high-SNR에서 8-bit->4-bit로 낮춰도 약 3 dB 이내 손실로 목표 FER 유지, 4-bit 모드에서 full-precision 대비 49% 전력(=51% 절감), Nangate 45nm 합성.
10. 한계: 무선 802.16e 특화, 정정능력/floor 개선 없음(전력 전용), throughput/gatecount 본문 수치 없음, 비동기회로는 동기 RTL과 이질적.
