### B. 알고리즘 요약

1. 대상: Galois field `GF(p^s)` 상의 d차원 Euclidean geometry `EG(d,p^s)`에서 µ-flat(hyperplane) 구조로 QC-LDPC 부호 구성, AWGN·BPSK·LLR-BP(max iter 50) 시뮬.
2. 문제: 기존 EG 기반 regular QC-LDPC는 girth가 6에 머물러 성능 제약이 있고, column/row splitting으로 irregular화하면 (a) QC성 상실, (b) column/row weight가 2종뿐이라 EXIT 최적화 불가.
3. 구성 3단계: base matrix `B`(µ1-flat 행 × µ2-flat incidence 열) → shift matrix `S`(1→shift값, 0→∞) → circulant permutation `P^a(m,n)`(q×q)로 QC PCM `H` 생성.
4. Girth 정리: `B`가 연속 두 차원(`µ1=µ2-1`)이면 base girth `gb=6`, 아니면 4(Thm1); QC 코드 girth `g`는 `µ1=µ2-1`일 때 `6≤g≤18`, else `4≤g≤12`(Thm3), 4-cycle은 shift값 조건(식11,12)으로 회피.
5. Regular 두 특수케이스: `µ1=0`(각 행 δ개 '1') 또는 `δ=전체 parallel bundle 수`(행 weight `A(d,µ2,µ1,s,p)` 일정).
6. Irregular 확장: hyperplane decomposing(Thm4) — `µ1<µ2-1`이면 column weight `N(µ2,µ1,s,p)` 열 하나를 `p^s`개의 `N(µ2-1,µ1,s,p)` weight 열로 분해(column decomposing), row도 유사 → 최대 `D=µ2-µ1`개 column weight의 irregular `BI`.
7. Degree 분포: `dV,j=N(µ2-j,µ1,s,p)`, `aV,j`·`λ,ρ`(식15~27)를 `δ,NV,j,NC,i` 조정으로 가변화.
8. 최적화: EXIT chart의 VND/CND transfer(식28~31) curve-fitting으로 `(λ,ρ)` 결정; degree-2 노드 fraction `λ2` 낮춰 error-floor 개선(식32,36, threshold SNR와 trade-off).
9. 결과: C1(4050,rate0.5,girth8), C2(3968,rate0.871), C4(13400,rate0.68)는 error-floor 없이 BER 10^-7까지, C4가 기존 geometric C5 대비 약 2.9dB coding gain(Table I, Fig.5).
10. 한계: 순수 부호구성법(HW·디코더 미설계), NAND 무관 AWGN 시뮬, minimum distance는 open issue.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| EG 기반 QC-LDPC H-matrix 구성(base+shift+circulant) | `ecc_top.cpp` `Load_PCM()` | Prime ECC도 base+circulant shift(`PCM_b`)이나 특정 QC-LDPC 고정 — 부호 교체 재검증 부담 큼 |
| large-girth·degree 최적화 부호설계 | 대응 없음 (부호구성은 이식 난이도 높음, 프로파일 §4) | 디코더 무관, 부호 재설계 레이어 |
| LLR belief propagation 복호 | `decoder.cpp` `LDPC_Decoder()` | 논문은 표준 BP 사용(개선 아님) — 기여 아님 |
| dual-diagonal 인코딩 | `encoder.cpp` `LDPC_encoder_4KB()` | 논문 부호는 QC지만 dual-diag 구조 아님, 인코더 재설계 필요 |

적용 가치: **낮음** — 순수 부호구성법으로 디코더/HW 기여가 없고, Prime ECC는 특정 고rate QC-LDPC H-matrix가 고정되어 EG 부호로 교체하면 전체 재검증 부담이 크며(프로파일 §4 "높음"), 검증도 NAND 무관 AWGN 시뮬에 그침.
