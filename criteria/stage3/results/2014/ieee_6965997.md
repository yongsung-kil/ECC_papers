### B. 알고리즘 요약

1. 시스템: FTN(Faster-than-Nyquist) 송수신기, raised-cosine 펄스(roll-off 0.4), QPSK·16QAM 고차변조, rate `R=1/2` QC-LDPC(길이 DVB-S2급) 또는 DVB-S2 표준 LDPC(길이 64800).
2. 문제: binary FTN(QPSK)은 높은 스펙트럼 효율 달성 시 `τ`를 과도히 낮춰야 해 수신기 복잡도 폭증·표준 LDPC와 부정합으로 성능 저하.
3. 모델: 정합필터 출력 `xk = Σ g_ℓ a_{k-ℓ} + n_k` (식3), 채널길이 `L`, ISI를 상태 `σk=(a_{k-1},...,a_{k-L})`로 정의.
4. 핵심기법1: BCJR(forward-backward) 기반 FTN equalizer가 branch metric `γ(σk,ak)=exp(R[a*_k(xk-isi)]/N0)·Pa(ak)`(식4)로 부호비트 soft decision 생성.
5. 식 의미: `isi(ak,σk)` 항으로 ISI를 명시 분리해 BCJR trellis에서 ML 등가 등화 수행.
6. 핵심기법2: equalizer→soft QAM demapper→bit-deinterleaver→LDPC BP decoder→다시 a priori로 되먹임하는 반복 복조-복호 루프(global iteration).
7. 구현 디테일: 내부 LDPC iteration 30, global iteration QPSK 5·16QAM 3, `L=4`, 수신기 복잡도 `∝ M^L`.
8. 부호최적화: rate 1/2 irregular QC-LDPC를 base matrix 16×32와 degree matrix로 정의, 열차수분포를 `τ`(0.75, 0.67)에 맞춰 조합최적화([8]).
9. 결과: FTN+표준 LDPC가 Nyquist 대비 ~0.5dB, 최적화 LDPC로 추가 ~0.3dB; 16QAM FTN이 QPSK 도달불가 영역까지 스펙트럼 효율 확장.
10. 한계: 순수 시뮬레이션, HW 미설계, FTN ISI·고차변조 채널 특화라 NAND binary LDPC와 채널·목적 모두 상이.
