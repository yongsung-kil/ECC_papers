### B. 알고리즘 요약

1. Bit-patterned media(BPM) 기록의 write-in error를 cascaded BSC-AWGN 채널로 모델링; BSC crossover `p_BSC`가 write process, AWGN이 read noise를 표현.
2. 문제: 기존 LDPC 디코더는 AWGN 가정으로 설계돼, BSC로 생성된 오류비트가 초기 채널정보에 남아 check node가 잘못된 메시지(faulty check node)를 생성.
3. 채널 pdf는 `p(y_i|v_i)`를 `(1-p_BSC)`·`p_BSC` 가중 두 Gaussian 혼합으로 modified Gaussian distribution 유도, LLR `L(v_i)`도 이 혼합으로 계산.
4. 핵심: check node 갱신식에 check node가 틀릴 확률 `p_LDPC`를 도입 - 기존 `∏(1-2P)` 형태의 CN 곱셈에 `(1-2·p_LDPC)` 인자를 곱해 faulty CN 확률을 반영.
5. 식 의미: `tanh` 규칙 `∏tanh(½ L)`에 `(1-2p_LDPC)` 스케일을 곱해, BSC로 훼손된 CN 신뢰도를 하향 조정(잘못된 parity 정보의 과신 방지).
6. VN 처리는 modified Gaussian LDPC와 동일; 변경은 CN outgoing message에 국한.
7. 구현 단순화: `p_LDPC`를 반복별 정밀추정 대신 `p_LDPC = p_BSC`로 고정(계산 편의).
8. 부호: PEG로 생성한 girth-큰 random LDPC, VN degree 3(binary 최적), block size 4096(섹터급), `p_BSC = 1e-2, 1e-3`(1Tb/in^2 급 write error rate).
9. 결과: BER 1e-5에서 modified distribution 대비 SNR 이득 medium-rate 및 high-rate 부호에서 conventional 대비 1dB 이상; high-rate는 BER 1e-6 부근 error-floor 존재.
10. 한계: HW 미설계·시뮬만, `p_LDPC=p_BSC` 근사·crossover mismatch에 성능 민감, BPM write-in error 채널 전용.
