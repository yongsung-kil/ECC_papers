### B. 알고리즘 요약

1. 대상은 BSC(crossover α) 위 (γ,ρ)-regular binary LDPC, hard-decision 입력, 예제 코드 (155,64) Tanner 및 QC(732,551).
2. 기존 BF는 Gallager A/B보다 성능이 낮고, unreliable 게이트(나노스케일 VLSI) 환경에서 신뢰성 분석이 부재하다는 문제.
3. Wadayama의 GDBF(AWGN용)를 BSC로 재유도: bipolar `η=(-1)^y`, `χ=(-1)^x̂` 대신 modulo-2로 변환.
4. 핵심식은 modified inverse function `Λv = (x̂v⊕yv) + Σ_c∈Nv Σ_u∈Nc x̂u` 를 최대화하는 노드를 flip.
5. `Λv` 값이 정수 `[0, γ+1]`로 제한되어 여러 노드가 동시에 최대값 `b(l)`을 만족 → 수렴엔 불리하나 게이트 고장엔 둔감.
6. PGDBF 신규성: `Λv=b(l)`이어도 자동 flip이 아니라 Bernoulli `av`(Pr=p)를 AND해 확률 p로만 flip.
7. HW 구현은 각 VN에 γ+1입력 MAJ 게이트(적응 threshold b(l)) + AND 게이트 + Bernoulli 생성기; threshold는 전 노드 공통.
8. 고장 모델: XOR 고장확률 `P⊕`, MAJ `PMAJ`, 레지스터 `PR`를 독립 von-Neumann transient로 주입.
9. 결과: perfect HW에서 PGDBF(p≈0.7)가 GDBF(p=1)보다 FER 감소; 놀랍게도 GDBF/PGDBF는 게이트 고장이 오히려 trapping set 탈출을 도와 FER 개선.
10. 한계: HW는 구조 스케치뿐 gatecount·throughput 없음, BSC hard-decision 전용, latency 소폭 증가.
