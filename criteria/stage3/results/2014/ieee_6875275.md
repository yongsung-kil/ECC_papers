### B. 알고리즘 요약

1. 대상은 binary QC-LDPC(CPM 구조), FPGA 복호 검증, 예제 rate 0.952(16120,15345) 및 rate-1/2(2640,1320)/(504,252).
2. 문제: 고rate·저 error-floor·라우팅 친화 QC-LDPC를 유연하게 구성하고 복호 HW 복잡도를 줄이는 것.
3. 핵심 구성: GF(q) 원소의 두 임의집합 `S1={α^ik}, S2={α^jl}`로 base matrix `B = [α^ik + α^jl]`(식1)를 만들고 각 원소를 (q-1)×(q-1) CPM으로 dispersion.
4. 핵심 식·성질: `B`의 2x2 submatrix 비특이(2x2 SM-constraint) → Tanner graph girth>=6 보장, 3x3 SM-constraint(Theorem 2) → girth>=8 보장.
5. Example 1: GF(131), 6x124 base로 (6,124)-regular (16120,15345) rate 0.952 코드, girth 6, scaled MSA(0.75)로 50 iter시 BER 10^-12에서 error-floor 없음(waterfall), Shannon 한계 1.33dB 근접.
6. rate-1/2 확장: masking(nonzero→0, Hadamard product `Bmask=Z⊗B`)으로 4x8 masked matrix 탐색해 col/row weight 3/6, girth 8/10 확보(Algorithm 1).
7. Example 2: GF(331)에서 74,336개 masked matrix가 3x3 SM-constraint 충족, girth-8/10 (2640,1320) 코드가 Margulis·PEG 코드 대비 낮은 error-floor.
8. CPM-RID 복호: section-wise cyclic shift로 전체 H 대신 첫 행들만 모은 `H*0`으로 복호, CN 처리유닛·wire를 q-1배 축소(Example 4: 130배, CN 6개·wire 744개), 라우팅 이점 유지·행/열 permutation 불필요.
9. 결과: CPM-RID-MSA(650~6500 sub-iter)가 전체 MSA와 성능 거의 동일하면서 HW 복잡도·전력 대폭 감소; `H*0(l)`로 복잡도-latency 트레이드오프 조절.
10. 한계: CPM-RID는 latency 증가(sub-iteration q-1배), 특정 CPM 크기·algebraic 구조 고정으로 기존 H-matrix에 곧바로 갈아끼우기 어려움.
