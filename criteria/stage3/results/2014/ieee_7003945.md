### B. 알고리즘 요약

1. DTMB 표준의 multi-rate irregular QC-LDPC(rate 0.4/0.6/0.8, codeword 7493bit, circulant size `b=127`)용 serial-in serial-out 인코더 HW 설계.
2. 문제: 기존 SRAA(Shift Register Adder Accumulator) 인코더는 매 클록 shift register 갱신으로 전력 소모가 크고, circulant generator를 여러 ROM에 저장해 메모리 낭비.
3. 부호 구조: generator matrix `Gqc`가 여러 circulant `Gρ,j`(각 행이 위 행의 우측 순환시프트)로 구성, parity `p = m × G'qc`.
4. 핵심 기법 RLA: 우측 시프트 `gr(i) = gl(b-i)` 관계를 이용해 (7)식을 좌측 시프트 반복형(8)으로 재작성 → multiply-add-rotate 반복 구조.
5. 식(8)의 의미: shift register B 없이 LUT 출력을 AND gate 입력에 직결하고, accumulator register A를 매 클록 좌측 1비트 회전시켜 parity section 누적.
6. 메모리 최적화: 3개 rate의 circulant generator 총 2196개를 단일 ROM(depth 2196, width 127)에 저장, 다중 ROM 제거.
7. 구현 디테일: rate 0.4는 35개 RLA 회로 필요, 0.6은 23개, 0.8은 11개 → 35개 RLA로 multi-rate 지원, state machine으로 rate 전환.
8. 검증: Verilog로 기술, Altera Cyclone IV E EP4CE55F29C6에 place&route, QuestaSim6.5와 MATLAB 결과 일치 확인.
9. 결과: RLA 인코더는 총 register 4518개(SRAA 9137개의 약 절반), logic elements 4617개(SRAA 9275개), 메모리 284988bit로 면적·전력 대폭 절감.
10. 한계: serial 구조로 kb 클록 소요(throughput 수치 없음), DTMB generator matrix 방식 특화, 복호/정정성능과 무관.
