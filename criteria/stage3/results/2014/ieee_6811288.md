> 총평: number-theoretic large-girth 그래프(D(m,q))에 node-splitting을 적용해 DDP를 맞추는 순수 부호구성 이론 논문 — BEC 검증뿐이고 block length가 비실용적이며 QC/HW·NAND 관련성이 낮아 이식성 하.

### B. 알고리즘 요약

1. 목표: 주어진 degree-distribution pair(DDP) `λ(x),ρ(x)`에 대해 girth가 block-length와 함께 커지는(provably large-girth) irregular·protograph LDPC 부호 수열 구성.
2. 문제: PEG는 provably large-girth 수열로 확장 불가, 기존 LPS Ramanujan `X_{p,q}` 기반 구성은 block length가 비실용적으로 큼(예: 10^20+).
3. 핵심 도구: `D(m,q)` large-girth q-regular bipartite 그래프(q=prime power), girth `≥ m+4/5`로 girth 6~18 구간에서 `X_{p,q}`보다 노드 수가 작음.
4. node-splitting: k-regular 그래프의 각 차수-k 정점을 `Σ d_i x_i = k` diophantine 해에 따라 작은 차수 `d_i` 정점들로 분할, edge 재배정 시 girth 보존.
5. 핵심 식: fraction `λ_i = (1/Nk)Σ_j s_j x_{ji} d_i` — 그룹별 분할 해와 노드 수 `s_j`로 목표 DDP를 근사(l 방정식 t 미지수).
6. 확장: `X_{p,q}` 대비 다양한 차수 혼합 분할로 같은 girth를 훨씬 짧은 block length에 달성; 초기 그래프 차수 k가 클수록 DDP 근사 정밀도↑(소수점 3자리는 작은 k로 충분).
7. protograph 확장: base matrix B의 edge 수 |E|가 소수면 `D(m,|E|)`를 |E|-regular 그래프로 쓰고 edge-coloring+node-splitting으로 protograph 확대(differential evolution 최적화 protograph 사용).
8. 최적화 절차: 최소 k 탐색 → 여러 해집합으로 근사 DDP 생성 → 목표 DDP와 Euclidean norm 최소인 해 선택.
9. 결과(예): rate-0.5 DDP를 `D(2,27)`로 2187×4376 PCM(girth 6)로 구성(`X_{p,q}`는 ~10^20 필요). BEC 시뮬에서 `(13448,6724)` protograph 부호가 random 구성 대비 block-error rate 크게 우수(단 bit-error waterfall은 차이 미미).
10. 한계: BEC만 검증(AWGN/NAND 무), block length가 girth 8 예제에서 55만까지 커 실용성 낮음, QC/순환행렬 구조 없음, HW 미설계.
