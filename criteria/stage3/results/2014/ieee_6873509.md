### B. 알고리즘 요약

1. 대상 시스템은 차세대 지상파 cloud transmission(CTxn), co-channel 간섭이 AWGN 등가라 SNR threshold -2~-3dB 요구.
2. 문제: LDPC 단독은 error-floor 존재, fading 대응 위해 복잡한 interleaver 필요.
3. 채널모델: AWGN 및 mobile TU-6(120km/h), OFDM(DVB-T2 파라미터, 4096 FFT, QPSK).
4. 핵심기법: 2D 부호 = 수직 raptor-like rate-compatible LDPC + 수평 RS. RS parity 먼저 수평 추가 후, 열 방향으로 읽어 LDPC parity 수직 계산.
5. 의미: 2D 배열 자체가 block-type interleaver 역할 → 별도 interleaver 불필요, RS가 LDPC 잔여오류(error-floor) 제거.
6. 확장: 수직 LDPC의 raptor-like 성질로 truncated codeword 복호 가능 → 수신 SNR 따라 적응적 rate.
7. 구현: quarter-rate LDPC(length 64800, parallel factor 30), block size 30으로 2160 블록 분할, 수신 블록수로 rate를 2/3·1/2·2/5·1/3로 truncate. 블록 단위 수평전송으로 time/freq interleaving.
8. 복호 순서: 수신측에서 LDPC 복호 후 RS 복호.
9. 결과: RS parity 8byte(t=4) 시 AWGN -3.33dB, TU-6 -1.68dB threshold; RS 14byte(t=7)로 LDPC 단독 대비 0.05~0.25dB 이득.
10. 한계: HW 미설계, 시뮬만, 방송 채널 특화, LDPC 디코더 자체 개선 아니고 concatenation·puncturing 아이디어 중심.
