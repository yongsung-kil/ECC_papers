> 총평: irregular Tanner 그래프의 통신 스케줄을 edge-coloring 기반 2-matching으로 최소화하는 그래프이론 논문 — 부호설계·정정성능과 무관하고 이미 정형화된 QC-LDPC 라우팅에 이식 실익 없음.

### B. 알고리즘 요약

1. 대상은 Tanner(bipartite) 그래프로 표현되는 LDPC/expander/IRA/polar 등 반복복호 부호, 특히 unbalanced·irregular 그래프.
2. 문제: irregular 그래프에서 flooding 스케줄의 한 iteration당 통신(노드 간 메시지 전달)을 최소 클럭으로 끝내는 스케줄 설계 — 계산 자체가 아닌 통신 phase만 최적화.
3. 모델: clock-synchronous VLSI 스케줄, COTS 블록메모리의 2-port 제약 → source/destination 각 노드가 cycle당 최대 2개 edge만 처리 가능.
4. ILP 정식화: boolean 변수 `s(j,k)`(edge j가 cycle k에 스케줄되면 1), edge당 1회 스케줄 + 노드당 cycle당 ≤2 edge 제약으로 파티션 수 `L` 최소화.
5. 핵심: 문제의 선형제약 polytope가 integral polyhedron이라 ILP 최적해가 그래프의 최소 edge-coloring과 일치.
6. 정리1: bipartite 그래프를 2-matching으로 분해하면 ILP 제약을 모두 만족(각 edge 정확히 1회, 노드당 ≤2 edge).
7. 정리2: (최소)edge-coloring ⇔ (최소) disjoint 1-matching 분해. bipartite에서 edge-chromatic index = `Δmax`.
8. 알고리즘: 알려진 다항시간 edge-coloring(`O(E log D)`)으로 색칠 후 두 색씩 합쳐 2-matching 구성 → 총 `Δmax/2`개(최소).
9. 결과: Alon 알고리즘 기반 python 유틸로 스케줄 생성, RTL 시뮬에서 통신 중 stall cycle 0 (conflict-free) 확인. 정확 알고리즘이라 BER/성능 시뮬 없음.
10. 한계: 정정능력·수렴속도·부호설계 무개선(통신 latency만), 대상이 일반 irregular 그래프이며 NAND/soft-read 무관, throughput/gatecount 수치 없음.
