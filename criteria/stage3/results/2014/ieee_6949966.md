### B. 알고리즘 요약

1. 대상은 circulant permutation 기반 QC-LDPC, base matrix `Hb` 5x8을 expansion factor `p=9`로 확장해 H=45x72, rate `R=3/8`, info bit 27, BPSK/AWGN.
2. 문제: irregular 부호는 성능 좋으나 VLSI 라우팅 혼잡·면적 큼 -> QC 구조로 면적/라우팅 줄인 저복잡 partial-parallel 단일모드 디코더 필요.
3. 복호는 soft-decision BP(SPA) 기반, 부호는 base matrix를 pxp identity의 우측 순환 shift로 확장.
4. CNU: `L(r_ji) = Πa_ij · max(min|β_ij| - offset, 0)` 형태(offset 보정), VNU: `L(q_ij)=L(c_i)+ΣL(r_j'i)`, 판정 `L(Q_i)` 부호로 hard decision.
5. offset 보정값은 Density Evolution으로 산출한다고 서술(수치 미제시).
6. partial-parallel 아키텍처: TDM으로 일정 수의 VN/CN을 단일 유닛에 매핑, 45 CNU + 72 VNU 구성.
7. HW 블록: Buffer IN(N개 RAM, depth=Z=p), Node RAM array, Control Unit(LUT 기반 라우터 재구성), CNU 회로(logarithmic/exp adder + comparator), VNU 회로, Buffer OUT. ITER 입력 0~11.
8. 학습/최적화: 별도 없음, offset만 DE로 결정.
9. 결과: Xilinx ISE 12.1, Virtex-5 5vlx85tff1136 타깃 합성+P&R, slice register 3816(7%), LUT 7295(14%). 면적 관점 저복잡 주장.
10. 한계: 부호 극소형(N=72), BER/throughput/수렴 iteration 수치 없음, offset BP는 신규성 낮음, 단일모드만.
