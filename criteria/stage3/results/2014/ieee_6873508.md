### B. 알고리즘 요약

1. 대상 시스템은 DTTB(디지털 지상파 방송), 다중레이트 QC-LDPC 요구. 기존 DVB-X2는 rate별 독립설계로 degree 분포 차이 커 HW 복잡.
2. 문제: 다중레이트를 동일 길이로, 낮은 HW 복잡도로 near-Shannon 성능을 동시에 만족하는 구성법 부재.
3. QC-LDPC의 parity-check H를 저차원 `Template matrix T`(circulant 유무만 0/1 표기)로 대체해 설계공간 축소.
4. 기법1 RCSEV: mother 부호의 Template row를 결합(`PM1-M2`)/분할, Edge Variation으로 열/행 차수분포 미세조정 → 동일 길이 basic 그룹 {1/2,2/3,5/6} 생성.
5. row-combining은 Row-combinable(같은 CN에 연결 안 된 노드) 조건에서만 가능, `M3` 중간값을 골라 결합/분할 2단계로 분해.
6. 기법2 CD(Column Deletion): Template 열을 점진 삭제해 확장그룹 {2/5,3/5,4/5},{1/4,1/2,3/4} 생성, nested 구조 유지.
7. 열 삭제 선택은 EXIT chart와 graph cycle 수 최소화로 결정, 후보 중 랜덤 선택.
8. 최적화: EXIT chart로 rate별 호환 열차수분포 결정, 이후 PEG 알고리즘으로 실제 H 구성.
9. 결과: sub-matrix b=128/512로 길이 15360/61440 두 클래스, DVB-S2 대비 EsN0 threshold 최대 0.18dB(rate 2/3) 개선.
10. 한계: HW 미설계, 디코더 알고리즘/양자화 무관, 방송채널(AWGN) BER threshold만 검증, error-floor·NAND 특성 미평가.
