### B. 알고리즘 요약

1. 대상: rate `R=1/2` 이진 LDPC convolutional code(CC)와 이를 tailbiting한 QC LDPC block code(BC), delay 576/2304/64800 bit 세 케이스.
2. 문제: LDPC CC가 점근적으로 우수하나 표준(WiMax, DVB-S2)의 저복잡도 인코딩·짧은 delay 제약하에서 실제로 BC를 능가하는지 불명확.
3. 모델: syndrome memory `m`의 다항 parity-check `H(D)` (monomial `D^wij`), tailbiting length `M`으로 QC LDPC BC 유도.
4. 핵심 기법: base matrix·degree distribution·girth profile를 multi-criteria 최적화(참고문헌 [8][9] 절차)로 탐색, 계산불가 문제를 search set 크기 제한으로 준최적 해결.
5. 핵심 지표: free girth profile `gfree`, sliding-window girth `gw`, column degree distribution `λs`가 BER 좌우.
6. 인코딩: `H(D)`에 bi-diagonal 구조 `Hbd` + 3원소 컬럼 `h0(D)`를 두어 partial syndrome `s(D)`로 parity 다항식을 재귀 계산하는 선형복잡도 인코딩.
7. 복호: LDPC CC는 sliding window decoding(SWD, 창 `W`, shift `S`, delay `L`), BC는 표준 BP. 버퍼 갱신은 채널·soft추정 선형결합 `r̃=(1-α)r+αs` (보통 `α=1/2`).
8. 최적화 절차: 저weight 컬럼 회피를 위해 `H0` 최적화 시도했으나 bi-diagonal의 weight-2 컬럼 다수로 오히려 BER 악화.
9. 결과: delay·complexity 모두 제약 시 QC LDPC BC가 CC를 능가; complexity 완화 시 저SNR에서 CC 우세. 신규 LDPC CC가 delay 96000에서 BER `10^-7`을 0.62dB(Shannon 한계 0.43dB 이내)에 달성.
10. 한계: HW 미설계, 시뮬만. rate 1/2·긴 delay·SWD 전제로 NAND 고rate 짧은 코드와 무관.
