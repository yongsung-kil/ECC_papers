### B. 알고리즘 요약

1. 시스템: BPSK+AWGN, 총 `n=3000`, `R=0.5`. 상위 외부부호 `nU=1000, RU=0.7`, 하위 외부부호 `nL=2000, RL=0.4`.
2. 문제: 일반 Gallager LDPC를 BP로 복호하면 동작 시작 직후 높은 error-floor가 나타남.
3. 모델: 부호어를 3행×`n/3`열 이진행렬 `C`로 보고, 길이 3·rate 1/3 내부 임베디드 부호 시스템(행렬 `H0`)의 syndrome 행렬 `S=H0·C`가 외부 LDPC 부호어가 되도록 구성(일반화 연접부호 GC).
4. 핵심기법1: 이 GC 구조 자체가 sparse parity-check 행렬을 가지므로 `H = [H(01)·HE(1); H0(23)⊗HE(23)]`로 쓰여 BP로도 복호 가능.
5. 핵심식: `s2=c1⊕c2`, `s3=c1⊕c3` — 확률분포(PD) 상에서 syndrome을 계산해 내부부호 상태를 갱신하는 것이 반복 복호의 핵심.
6. 핵심기법2: 연접부호 복호 기반 iterative soft 알고리즘 — 채널 PD로 `S` 계산→하위 외부부호 복호로 `S(23)` 갱신→쌍 확률비 유지하며 `C` 재계산→상위 외부부호 복호로 `S(1)` 갱신 반복.
7. 구현 디테일: PD 계산을 심볼 개별 방식과 벡터(8워드) 방식 두 가지로 제시, Gallager 부호의 랜덤 열치환 덕에 심볼 대응 무관.
8. 하이브리드: 먼저 BP를 돌려 부호어면 종료, 실패 시 BP 출력을 연접복호기 입력으로 전달.
9. 결과: soft 연접복호는 관측 범위 내 error-floor 없음, 하이브리드는 BP보다 크게 개선(단 순수 연접복호보다는 floor 높음). BER/BLER 곡선(그림)만 제시.
10. 한계: 정정성능 최적화 안 함(디코더 비교가 목적), HW 미설계, 특수 GC 구조라 일반 QC-LDPC와 다름.
