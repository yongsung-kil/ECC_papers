### B. 알고리즘 요약

1. 대상: binary LDPC의 Tanner graph에서 iterative(SPA/MSA) 복호의 error-floor를 좌우하는 trapping set 구조 분석, 유한기하 기반 부호설계(초청 개관 논문).
2. 문제: 좋은 error-floor를 위해 작은 trapping set이 없어야 하나, 이를 분석하려면 parity-check 행렬에 충분한 구조가 필요.
3. 정의: `(κ,τ)` trapping set = VN 부분집합 `∆`가 유도하는 subgraph에서 홀수 차수 CN이 `τ`개. small = `κ≤√n` 이고 `τ/κ≤4`.
4. 핵심 구조 1: partial geometry `PaG(γ,ρ,δ)` (점 `n=(ρ(ρ-1)(γ-1)+δ)/δ`, 선 `m`)의 점-선 incidence 행렬을 RC(no length-4 cycle) 제약 QC-LDPC로 사용.
5. 하한(Thm1): girth≥6, VN degree `γ`인 PaG 부호에서 `κ≤γ`이면 `τ ≥ (γ+1-κ)κ + (홀/짝 CN 보정항)`, `δ=ρ`에서 등호 → trapping set 크기의 구조적 하한 보증.
6. 확장(net/GQ): `δ=γ-1`이면 net, `δ=1`이면 generalized quadrangle(GQ) → girth 8. GQ 부호는 quasi-cyclic이며(Liu-Pados) 순환 블록 크기 산출 가능해 구현 친화.
7. 구현 트릭: 유한체 GF(2^s) 위 primitive element로 circulant `P`를 dispersing해 incidence 행렬 `H` 생성(RC 제약 자동), QC 구조 확보.
8. GQ 성질: girth-8이 girth-6 대비 성능 개선 여부 탐색, ovoid/spread 구성이 small trapping set을 배제함을 사례로 제시(`κ/τ=1+t>4`).
9. 결과: s=7 예제로 (16129,15372) rate 0.953 QC-LDPC, MSA 50 iter로 BER 10^-6를 Shannon 0.95dB 이내, BER 10^-15를 약 1.6dB 이내 달성, error-floor 낮음(Fig.1).
10. 한계: 정성적 개관(진행 중 연구), error-floor와 trapping set 크기의 tight한 해석적 대응 부재, HW 미설계, rate/length가 유한기하 파라미터에 고정.
