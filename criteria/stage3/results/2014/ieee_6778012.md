### B. 알고리즘 요약

1. 대상 시스템: 120Gb/s 16-QAM CO-OFDM(coherent optical OFDM), FFT size 128, 중앙 64 subcarrier 데이터, inner 부호 QC-LDPC `(2295,2042)` column weight 3, girth 6.
2. 풀려는 문제: LO 근처 near-DC noise로 몇몇 subcarrier SNR이 급락 → 전체 BER 지배, 기존 LDPC는 모든 VN noise variance가 동일하다고 가정하여 오염됨.
3. 채널 모델: 수신 `r = c + n`, `n`은 AWGN이나 subcarrier마다 variance `σ²`가 다름(varying subcarrier reliability); LO drift 범위 1.25GHz 가정.
4. 핵심 기법(디코딩): subcarrier별 `σ²`를 training symbol로 추정 후 식(1)(2)의 LLR 초기화에 반영 → 저신뢰 subcarrier의 VN LLR을 0에 가깝게 만들어 iteration에서 가중치 축소.
5. 핵심 식 의미: `LLR_j = log[p(r|b_j=0)/p(r|b_j=1)]`에 subcarrier별 `σ²`를 대입 → error-propagation 억제.
6. 핵심 기법(인코딩, UEP): LO drift 대역(middle range) subcarrier에 강한 FEC(20%), outer range에 약한 FEC(7% RS) 적용 — 비대칭 보호.
7. 부호 구성: strong FEC = 13% QC-LDPC(2295,2042) inner + 7% RS(255,239) outer concatenation + interleaver; decoding은 sum-product(또는 min-sum) 기반.
8. 최적화: 별도 학습 없음, training symbol 20개로 채널·subcarrier SNR 추정.
9. 결과: 20% middle+7% outer로 near-DC noise 하 error-free 달성, constant variance 디코딩 시 10⁻⁴ error floor 잔존, conventional 대비 수신감도 ≥1dB 개선, SE 손실 1.67%.
10. 한계: HW 미설계(실험 검증뿐), 광 CO-OFDM subcarrier 구조 특화, throughput/복잡도/gate count 미기재.
