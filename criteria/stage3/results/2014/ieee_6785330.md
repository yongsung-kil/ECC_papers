### B. 알고리즘 요약

1. 시스템: 4Tb/in² 자기기록(BPMR/HAMR/TDMR) 채널, 2D-ISI + AWGN, rate 8/9 LDPC, 코드길이 `Nc=4608/9216/13824`.
2. 문제: 최적 symbol-based 2D BCJR 검출기는 `O(N·2^(M·LN))` 복잡도로 과도, 저복잡도 검출기(GA-IRCSDFA)는 성능 손실 발생.
3. 모델: 상/하 트랙 ITI를 중심극한정리로 등가 Gaussian noise 근사 → 2D-ISI를 1D-ISI + Gaussian으로 단순화(64-state → 4-state BCJR).
4. 핵심기법1: GA-IRCSDFA row/column 검출기 소프트정보를 LDPC BP 디코더와 turbo 반복(검출-복호 joint), 검출기간 반복 3, 검출-복호 반복 10, BP 반복 30.
5. 핵심식: 등가잡음 `wr(i,j)`를 Gaussian 근사 → 1D-BCJR 적용 가능, 이로써 `2^(2·LM·LN)` → `2^(LM)+2^(LN)` 복잡도 대폭 감소.
6. 핵심기법2: GA-IRCSDFA 검출기 EXIT 곡선을 Monte Carlo로 측정, 3차 다항식 근사 후 sum-product EXIT과 매칭하여 LDPC degree distribution(`λ2,λ3,λ13,λ15`, dc=27) 재최적화.
7. 트릭: 두번째 기여로 BPMR write-in error(치환+삽입+삭제)를 BSC+insertion/deletion 채널로 모델(W-ISI), EMCS(embedded marker code) + SA-WSD 동기화 알고리즘.
8. 최적화: EXIT-chart 기반 degree 최적화(참고문헌 [8] 절차), inner BCJR HMM(drift state `Si`) forward/backward recursion으로 삽입/삭제 정정.
9. 결과: GA-IRCSDFA의 0.55dB 손실(BER 1e-5)이 재설계 LDPC로 거의 완전 회복; SA-WSD가 기존 SA 대비 큰 FER 이득.
10. 한계: HW 미설계, 시뮬만, 자기기록 2D-ISI/write-in error 채널 특화로 NAND 무관.
