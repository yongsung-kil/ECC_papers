### B. 알고리즘 요약

1. 대상은 finite field/finite geometry로 만든 base matrix `W`(RC-constraint, girth≥6)를 circulant permutation으로 dispersion한 QC-LDPC이며, 예제는 geometry decomposition으로 만든 (12,63)-regular 768x4032 `H`(64x64 순환행렬 배열).
2. 문제: `H`에 masking(`M = Z ⊙ H(γ,ρ)`)으로 얻는 irregular QC-LDPC는 SPA 복호 복잡도는 낮지만 조기 error-floor가 자주 나타남.
3. 채널/복호 가정: AWGN + BPSK, SPA, 최대 iteration 50.
4. 핵심 기법: masking matrix `Z`(γxρ, GF(2))의 cycle/girth/ACE가 masked matrix `M`의 성능을 좌우함을 규명 — Proposition 3으로 `Z`의 cycle `L`과 `M`의 sub-cycle `l`이 동일 ACE를 가짐을 증명.
5. 의미: 큰 `M` 대신 훨씬 작은 `Z`에서 ACE를 분석·최적화하면 되므로 저비용으로 고성능 irregular 부호 구성 가능.
6. 확장: 수정 ACE 알고리즘(Algorithm 1)으로 목표 degree distribution `λ(x)`, `ξ(x)`를 만족하는 `Z`를 열 단위(내림차순 degree)로 생성, 임계 `(δ2d, δACE)`를 파라미터화.
7. 구현 디테일: `Z`는 non-sparse라 length-4 cycle을 허용(masked M에는 length-4 미발생), length-4 cycle의 ACE를 최대화하면 그를 포함하는 상위 cycle의 ACE도 커짐.
8. 최적화 절차: 각 variable node마다 `NT_MAX`(=10000)회 trial-and-test로 ACE 제약을 만족하는 열벡터 선택, 미충족 시 최대 ACE 열벡터를 채택.
9. 결과: rate-0.81 길이-4032 부호 C1(random)/C2(ACE 4,3)/C3(ACE 4,4) 중 C3는 error-floor 소멸, 원본 regular QC-LDPC 대비 BER=1e-6에서 약 1.4 dB 이득, threshold 3.9851dB→2.8060dB, 평균 열무게 12→≤4.35로 SPA 복잡도도 감소.
10. 한계: HW 미설계, AWGN/SPA만 검증, ACE 최적화 이득은 목표 degree distribution 선택에 크게 의존.

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| masking/ACE로 만든 irregular QC-LDPC H-matrix 구성 | ecc_top.cpp Load_PCM() (H-matrix 로드) | 부호구조 재설계 -> 특정 QC-LDPC 고정된 Prime ECC 재검증 부담 큼 |
| masked matrix로 인한 degree distribution 변경 | encoder.cpp Generate_PCM_encoding() (부호 PCM 생성) | irregular 채택 시 인코더 PCM/dual-diagonal 재구성 필요 |
| error-floor 완화 부호구조 | decoder.cpp LDPC_Decoder() (복호 성능은 부호구조에 종속) | 부호구조로 error-floor 개선하나 min-sum과 조합 재검증 필요 |
```

적용 가치: **중간** — binary 고rate QC-LDPC의 error-floor를 부호설계로 제거하는 기법이라 방향은 맞지만, Prime ECC는 QC-LDPC base가 고정(N_b=129, z=32)이라 masking 재설계·재검증 부담이 크고 검증도 SPA/AWGN에 국한된다.
