### B. 알고리즘 요약

1. 시스템: ATSC DTV 수신기, 8-VSB(±1,±3,±5,±7) 심볼, symbol rate `10.76MHz`, SRRC 송신필터·matched filter·심볼레이트 샘플링. 데이터는 RS+trellis 부호화됨(등화 대상은 채널 ISI).
2. 문제: 지상파 다중경로로 CIR이 수백 심볼에 달하는 강한 ISI. 기존 DFE는 slicer 오판정 시 feedback filter가 오히려 unintended ISI를 주입해 error propagation(bursty error) 발생.
3. 모델: 시변 CIR `hk` + AWGN `w[k]`, Doppler fading. `y[k]=Σ hk[i]x[k-i]+w[k]`.
4. 핵심 기법 1(DFSF): DFE의 feedback filter `gk`에서 가장 큰 `(D-1)`개 tap을 0으로 만든 modified filter `g̃k`를 써서, 지배적 ISI tap은 일부러 남긴다. 결과적으로 결합응답 `ck`가 D개 nonzero tap의 sparse 채널이 됨.
5. 의미: 큰 tap을 feedback에서 제거하면 오판정 시 주입되는 unintended ISI가 줄어 error propagation이 완화됨. D는 성능/복잡도 trade-off(D=1이면 순수 DFE).
6. 핵심 기법 2(적응): startup은 MMSE-DFE 해(식 (2),(3))로 `f0,g0` 초기화; tracking은 결합응답을 초기값 `c0`에 고정하도록 DD-LMS로 `fk,gk,g̃k`를 갱신(Algorithm 1). → BP detector가 static 채널로 인식해 구현 대폭 단순화.
7. 구현 디테일: 채널이 크게 drift해 `g̃k` tap이 커지면 startup 재실행(주기적 reset).
8. 최적화: decision-directed LMS(step `µf=µg=µg̃=10^-5`), 오차항 `e[k]`(DFE 기준)·`ẽ[k]`(`c0` 기준)로 각각 갱신.
9. 결과: BP iteration 5회, `Lf=Lg=Lg̃=400`, `D=3`. Brazil A(정적)·CRC #4(fd=5Hz 동적) 채널에서 SER 10^-5 기준 기존 DFE 대비 약 1dB 이득.
10. 한계: BP는 LDPC 복호가 아니라 sparse ISI 채널 등화에 쓰임. HW 미설계(시뮬만), NAND·binary LDPC 부호와 직접 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP(belief propagation) equalizer | 대응 없음 | Prime ECC의 BP는 LDPC 부호 복호용, 본 논문은 채널 ISI 등화용으로 목적이 다름 |
| DFSF/DD-LMS 채널 적응 | 대응 없음 | NAND ECC에 채널 등화 개념 없음 |
| 8-VSB/ATSC 채널 모델 | 대응 없음 | Prime ECC 채널은 AWGN/RBER/fixed-error (NAND) |

적용 가치: **낮음** — BP를 LDPC 복호가 아닌 방송 채널 등화에 적용한 논문으로, NAND binary LDPC ECC와 대응 모듈이 없다.
