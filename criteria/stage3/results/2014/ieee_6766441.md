### B. 알고리즘 요약

1. 대상 코드: Tanner code, column-weight 3, length `N=155`, rate `2/5`, 155개 소형 trapping set으로 전면 덮인 구조.
2. 문제: BP는 Tanner graph의 loopy 구조(trapping set)에서 수렴 실패해 저잡음 영역에서 error-floor를 유발.
3. 모델: LDPC 복호를 spin glass 추론으로 대응시키고, joint distribution을 변분 자유에너지 `Fb = Ub - Sb` 최소화로 근사(Bethe → BP fixed-point 등가).
4. 핵심 기법 1단: Bethe 근사를 일반화한 Region-Based Approximation(RBA) — Tanner graph를 subgraph(region)로 묶어 region-graph를 만들고, 각 region belief `br(xr)`에 대해 GBP message `mrs`를 반복 전달(식 20).
5. counting number `cs = 1 - Σ_{r⊃s} cr`로 각 변수노드가 한 번만 기여하도록 제약.
6. 핵심 기법 2단(신규): Local loopfree principle — harmful subgraph T를 `nc`개 클러스터로 "쪼개어" 그 region-graph가 국소적으로 loopfree가 되게 함(TS(5,3)을 3클러스터로 분할, Fig.4-5).
7. 구현: region-graph loop가 Tanner graph loop보다 커져 trapping set 영향이 약화됨(전체 loopfree는 불가, 부분 개선).
8. 결합: BP를 먼저 돌리고 실패 시에만 GBP 호출하는 hybrid decoder (BP 속도 + GBP 정확도).
9. 결과: BSC p=5e-2에서 hybrid BER이 BP 대비 10배 낮음; AWGNC에서 BP는 5.8dB에서 error-floor, hybrid는 7.0dB까지 floor 없음. 단 KGBP≈49.9 vs KBP≈0.9.
10. 한계: HW 미설계, 시뮬(1e12 realization)뿐, 소형 저rate Tanner code 한정, GBP iteration/복잡도 대폭 증가, 다른 topology의 trapping set은 미해결.
