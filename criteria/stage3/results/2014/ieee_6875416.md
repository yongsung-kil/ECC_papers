### B. 알고리즘 요약

1. 대상: array LDPC 부호 `C(q,m)` (q=홀소수, m≤q), 길이 `q^2`, `(m,q)`-regular QC-LDPC, PCM는 순열행렬 `P`의 거듭제곱 블록으로 구성.
2. 문제: `m=6`의 최소거리 상계는 Mittelholzer의 `d(q,6)≤32`가 최선이었고 이를 개선하려 함.
3. 표현: 각 column을 정수쌍 `(x,y)`로 표현해 column = `(x, x+y, ..., x+(m-1)y) mod q`; codeword의 support submatrix를 `m×w` 정수행렬로 다룸.
4. 핵심개념: template support matrix — `q`의 함수로 채워진 `m×w` 행렬로, `q≥q0`인 모든 q에서 weight-w codeword의 (열치환된) support matrix가 되는 틀.
5. 3단계 heuristic 1단: 서로 다른 `q1<q2`의 minimal codeword `c1,c2`가 같은 graphical cycle structure를 갖는 쌍을 탐색.
6. cycle structure: support matrix graph `G(i,j)`에서 특정 edge를 포함하는 각 길이별 cycle 수가 두 codeword에서 일치하면 같은 구조로 정의.
7. 2단: Algorithm 1로 2×2 방정식·합동식(중국인 나머지 정리, extended Euclidean로 `κq1+ηq2=1`)을 풀어 candidate template support matrix (13) 추론.
8. 3단: 추론된 template의 모든 instance가 `q>7`에서 유효 codeword의 support matrix임을 형식 증명.
9. 결과: Theorem 1로 `d(q,6)≤20 (q>7)` 증명, 확률적 최소거리 알고리즘으로 `q≥23`에서 실제 최소거리가 20임을 시사(상계 tight), q≤79 신규 최소거리 값 제시.
10. 한계: 순수 조합론·수론적 상계 증명, 복호 알고리즘/HW/시뮬레이션 성능 없음.
