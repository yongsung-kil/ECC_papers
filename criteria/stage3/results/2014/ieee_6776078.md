### B. 알고리즘 요약

1. 대상은 DVB-S2 LDPC 복호로, 채널은 binary-input AWGN, LLR 9bit 양자화, 50 iteration을 가정한다.
2. 푸는 문제: SPA(`ln cosh` 기반)는 정확하나 LUT·메모리 비용이 크고, MSA는 `ln cosh(x)≈sgn·min`으로 단순화하나 성능 열화(BER=1e-5에서 SPA 대비 0.58dB)가 크다.
3. 핵심 기법: SPA의 `ln cosh(x)`를 step 근사 `≈{0 if x≤ln2; x−ln2 otherwise}`로 대체한 EMSA를 사용, CN 갱신을 세 구간(±C_offset, min)으로 분기.
4. 두 입력 core 함수 `CHK(Λ1,Λ2)`는 min 결과에 동작점 분산 기반 offset `C_offset=(σ²_op·ln2)/4`를 조건부로 가감하는 형태로, 덧셈·비교·비트시프트만으로 구현.
5. 핵심 식은 (5)의 `sgn(r1)sgn(r2)min(|r1|,|r2|) ± C_offset`으로, `|r1±r2|`와 `σ²_op·ln2` 비교로 offset 적용 여부를 결정해 ln cosh를 근사한다.
6. n-edge CN의 다중입력 갱신은 재귀 `CHK(Λ1,...,Λn)=CHK(Λ1,CHK(Λ2,...))`로 정의.
7. 효율 구현: binary tree search 기반 **forward-backward 알고리즘**으로 보조변수 `F_i`, `B_i`를 계산, i번째 edge 출력을 `F_{i-2}`·`B_{n-i-1}` 조합으로 산출 → CN당 `3n-6`회 2입력 CHK 연산.
8. core 연산 `sgn·min`은 Fig.3의 전용 아키텍처로 단순 실현.
9. 결과: AWGN, 50 iter, 9bit에서 EMSA는 BER=1e-5 기준 SPA 대비 0.02dB 열화에 그쳐 MSA(0.58dB 열화)보다 우수, 9bit 양자화 열화는 float 대비 무시가능.
10. 한계: 고정소수점 시뮬만 제시(BER 곡선 외 gate count/throughput 없음), DVB-S2·AWGN 특화이며 error-floor·NAND read 비용 미검증.
