### B. 알고리즘 요약

1. 시스템: FH-SS(BPSK) 통신, LDPC-BCH 부호, LDPC base `16384`b·`R=1/2`, SFH, hop당 심볼수 `T=14`.
2. 문제: 부분대역 재밍(PBJ) 환경에서 다수 대역이 재밍될 때 표준 LDPC의 복잡한 trellis 구성 문제와 성능 저하.
3. 채널 모델: full-band AWGN + Additive Partial-Band Gaussian Noise(APBGN), 재밍 비율 `P=W_J/W_SS`, 재밍된 hop의 수신값은 0으로 설정.
4. 핵심 기법: LDPC Tanner graph의 SPC check node(차수 i)를 `(n_i, i-1)` nonsystematic BCH code 제약으로 치환 → GLDPC(LDPC-BCH).
5. rate 조절: 각 BCH check node의 punctured codeword 전송길이 `n_tran`(0~n_i-2)을 선택해 부호율을 가변 조절, `R=K_L/(N_L+Σn_tran)`.
6. nonsystematic BCH: 체계적 BCH generator를 전처리(`u'(b)=u(b)⊕u(0)`)해 최대 열가중치 열을 parity-check 제약으로 활용.
7. 복호: bipartite graph 상 message-passing, BCH check node는 APP-BCH 복호(`B^-1`)로 extrinsic LLR 계산, VN은 repetition code 복호.
8. 저복잡도: 직접 APP 계산 `O(2^(i-1)·n)`을 butterfly 기반 Fast BCH Transform(FBT)으로 `O((i-1)·n)`으로 축소(Hadamard SISO 복호 유사).
9. 분석/결과: EXIT chart로 VN-BCH 곡선 매칭 분석, `R=1/8.5`에서 P=30/50/70% 시 BER=1e-6를 SNR 6.97/4.63/2.48dB 달성, Reed-Solomon-BCH 대비 error-floor 없이 일관된 성능.
10. 한계: HW 미설계, 시뮬만, PBJ 채널·초저rate 특화, non-binary BCH 성분부호로 QC-LDPC와 근본 상이.

| message-passing VN 처리 | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `VN_Cal_HD()` | VN 개념만 공유, BCH check 제약과 결합 불가 |

적용 가치: **낮음** — Prime ECC는 SPC check node의 binary QC-LDPC min-sum 디코더인데, 본 논문은 check node를 BCH 성분부호로 대체한 GLDPC로 부호구조·복호 알고리즘이 근본적으로 다르고, 대상 채널(PBJ)과 초저 부호율도 NAND ECC와 무관.
