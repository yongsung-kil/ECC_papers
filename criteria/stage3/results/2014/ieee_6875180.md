### B. 알고리즘 요약

1. 시스템: binary LDPC(rate `R=K/N`)를 CPM 변조기와 직렬연접, AWGN 채널, 수신단은 SISO CPM 검출기(BCJR) + SISO LDPC(BP) 반복복호.
2. 문제: CPM과 연접한 LDPC 부호설계 연구가 드물고, 프로토그래프 LDPC의 CPM 성능·설계법 부재.
3. 핵심아이디어: LDPC-CPM 사이에 **부분 인터리버**(VN degree별 다른 인터리빙)를 삽입 → EXIT 재귀 `Ψ`를 λ_i에 대해 선형화.
4. 비정형 설계: Rimoldi 분해로 CPM을 CPE(trellis)+MM으로 보고, CPM 검출기 EXIT `Tcpm(.)`을 Monte Carlo로 구해 다항식 근사.
5. 핵심식: 최대 달성 rate `R* = ∫₀¹ Tcpm(Ivn,cpm) d(Ivn,cpm)` (EXIT 곡선 아래 면적).
6. 최적화: `Σ λi/i` 최대화를 [C0]mixture·[C1]convergence·[C2]stability 제약 하 선형계획으로 풀어 rate-1/2 λ/ρ 프로파일 도출.
7. 확장: 프로토그래프 base matrix `H`(식13, degree-1/2/고차 혼합)에 대해 PEXIT(식8-12)로 threshold 계산, PEG+ACE로 lifting.
8. 관찰: AWGN 최적 프로토그래프(AR3A/AR4JA)가 CPM에서는 좋지 않음 → CPM 전용 설계 필요.
9. 결과: GMSK/2RC/2REC에서 capacity 0.3-0.4dB 이내 동작, AWGN 프로토그래프 대비 1.5-4dB threshold 개선(약 5000 info bits, 250 turbo iter, 10^-5까지 error floor 없음).
10. 한계: HW 미설계, 시뮬레이션만, CPM(위성/무선) 특화 채널 전제 - NAND 무기억 채널과 무관.
