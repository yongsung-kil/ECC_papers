> 총평: ICI 있는 MLC NAND 채널의 정확한 soft-output MAP detector(Viterbi/BCJR + 특성함수 FFT)를 유도해 기존 detector 대비 0.35dB soft 이득을 보이나, LDPC 디코더 밖 채널 전처리 detector라 Prime ECC의 binary QC-LDPC 디코더에 직접 이식 대상은 아니다.

### B. 알고리즘 요약

1. 시스템: MLC(4-level, `m=4`) NAND, ECC encoder→channel→detector→ECC decoder 구조에서 detector 블록만 다룸. Detector 출력이 LDPC의 soft-input이 됨.
2. 문제: 기존 문헌은 채널출력 양자화로 근사 soft-info만 제공, 양자화 없는 exact soft-output detector가 부재.
3. 모델: Dong et al. 채널모델을 tractable하게 변형. `Yk = Xk + Σ Γℓ(Yk-ℓ - Ek-ℓ) + Wk + Uk`, ICI 계수 `Γℓ~N(γℓ,gℓ)`, program noise `Uk~U(-Δ/2,Δ/2)`, PE noise `Wk~N(0,σw²)` (truncated Gaussian 대신 Gaussian 채택).
4. 핵심 기법 1단: input을 order-`M` Markov로 두고 trellis 상태 `(Xk...Xk-M+1)`, `mM` states의 Viterbi/BCJR MAP sequence/symbol detector. Branch metric `ΛMAP(xk,yk)` 계산이 관건.
5. 핵심 식: `f(yk|xk,yk-1)`를 직접 못 구해 조건부 특성함수 `GYk|Xk,Yk-1(t)`를 유도(식 10-11)하고 그 Fourier 역변환으로 pdf 획득 → branch metric. 왜: 곱 Gaussian(ICI)의 pdf가 해석적으로 안 풀려 특성함수 경유가 필요.
6. 핵심 기법 2단: 각 trellis section당 FFT 한 번(N=512)으로 모든 branch metric을 샘플링 획득.
7. 부수 트릭: sufficient statistics는 `2L+1`개(FIR 필터 출력), Gaussian Approximation(GA) detector는 `[yk, θk, νk]` 3개 통계와 Q-function 닫힌형으로 FFT/LUT 없이 곱셈기·가산기만으로 구현.
8. 확장: signal-dependent noise, input ISI, 2D anti-causal(even/odd bit-line) 채널로 일반화. 기존 post-compensation[5]·soft detector[2]가 본 GA의 추가근사임을 보임.
9. 결과: hard BER은 post-comp와 동등, soft-output은 SIQ(=random LDPC 달성가능 rate) 기준 SNR 변화 시 0.35dB 이득(SIQ 1.8bits/cell, rate 0.9 대응). N=512.
10. 한계: HW 미설계, 시뮬만, MAP은 FFT로 `O(NlogN)` 부담, 2D 최적 detector·최적 Markov input은 미해결로 남김.

| soft-output → LLR/양자화 | [channel.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `Set_LLR_Th()` | detector가 내는 exact posterior를 2~3bit LLR로 양자화해야 함, soft-4bit+ 수준이라 정합 부담 |
| MAP/BCJR/Viterbi detector 본체 | 대응 없음 | Prime ECC는 trellis detector 없음, 채널 전처리 detector 자체가 코드베이스 밖 |
| LDPC 디코더 이터레이션 | 대응 없음 | 본 논문은 디코더를 다루지 않음 (detector만) |

적용 가치: 낮음 — NAND 직접 타깃이고 soft-info 품질(SIQ)을 0.35dB 올리지만, LDPC 디코더가 아닌 채널 detector 전처리이며 exact posterior 출력이 Prime ECC의 hard/2~3bit soft 파이프라인과 맞지 않아 그대로 떼어 쓰기 어렵다.
