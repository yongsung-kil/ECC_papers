> 총평: CPM을 대체하는 새 semi-regular 서브행렬(TPM)로 large-girth 부호를 구성하는 부호설계 논문 - QC-LDPC 고정된 Prime ECC와 서브행렬 구조가 근본적으로 달라 이식성 낮음.

### B. 알고리즘 요약

1. 대상: QC-LDPC의 circulant permutation matrix(CPM)를 대체하는 새 서브행렬 "tree-permutation-matrix(TPM)"를 정의하고, 이를 base matrix `J×L`에 채워 TPM-LDPC 부호를 구성한다.
2. 문제: 랜덤 permutation(RP-CC-LDPC)은 BER 좋지만 병렬복호 불가·메모리 과다; CPM은 병렬 쉽지만 girth 선택폭이 좁다. TPM은 그 중간(semi-regular)을 노린다.
3. TPM 구성: 단일 '1'에서 시작해 각 '1'을 2×2 항등 `I2×2` 또는 anti-diagonal `Ĩ2×2`로 재귀 치환, Layer-M에서 `2^M×2^M` 크기, `2^(2M−1)`가지 존재; 각 TPM은 tree vector `V=(a1,0,...)`로 표현.
4. 핵심식/연산: 두 TPM의 곱·전치를 tree vector상의 mod-2 덧셈으로 O(트리) 계산(Algorithm 1, 2) - lookup table 없이 대형 TPM 처리 가능.
5. cycle 판정: permutation 곱이 "fixed column"을 가지면 cycle 존재(Theorem 1), TPM은 tree에 all-zero branch 있으면 fixed column → tree vector로 girth를 빠르게 평가.
6. 부호 구성: 작은 TPM에서 낮은 girth 먼저 확보 후 확장; 확장은 기존 girth를 깨지 않으므로 새로 추가된 tree 원소만 탐색 → fast hill-climbing으로 large-girth 부호 탐색.
7. 병렬복호/메모리: `2^M2 × 2^M2` 소TPM 단위로 C↔V 메시지를 `G=2^M2`개 RAM에 분산 배치하면 RAM access conflict 없이 병렬복호 가능, address RAM 수·크기가 RP 대비 감소.
8. 결과 부호: girth-8 (4×24 base, z=4096, len 98304, rate 5/6), girth-6 (z=2048, len 49152), girth-10 (3×10 base, z=4096, len 40960, rate 7/10) 구성.
9. 성능: AWGN, BP 10 iteration, 4-bit 양자화 FPGA(Altera Stratix IV E). len98304에서 CC-QC-LDPC가 TPM보다 ~0.015dB 우수하나 CC-QC는 10⁻¹³ error floor 발생, TPM은 10⁻¹³ 이하까지 floor 없음; QC 대비 약간 우수·error floor 낮음. 복잡도는 QC<CC-QC<TPM<RP-CC 순, throughput은 QC/CC-QC와 동일.
10. 한계: 타깃이 광통신(중rate·긴부호)이고 NAND용 rate 0.9/short-length(4K/1K)는 "향후 연구"로만 언급, TPM 복호기가 QC보다 자원 더 소모.
