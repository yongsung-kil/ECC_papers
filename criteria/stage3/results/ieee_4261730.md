### B. 알고리즘 요약

1. 대상 채널은 perpendicular magnetic recording (PR(5,6,0,-1) 등화), 부호는 N=1309, M=88 구조적 LDPC.
2. 기존 array code/eIRA code는 single cyclic shift 구조 때문에 ISI 채널에서 short block/high rate일 때 성능(특히 error floor)이 부족.
3. 채널은 ISI가 있는 자기기록 채널이며, RLL 코딩된 채널은 long error event를 발생시켜 LDPC 복호 후 error floor를 유발한다고 가정.
4. 제안 기법은 parity check matrix `H=[H1 H2]` 구조로, H1은 p×p 순열행렬 `α`와 shift `g`(>1)로 정의된 quasi-cyclic 행렬 `ω`를 사용해 컬럼을 거의 랜덤하게 인터리빙(RIA-LDPC).
5. H2는 column weight 3의 cyclic shift 행렬로, 첫 행의 1 위치를 신중히 선택해 역행렬 `H2^-1`을 sparse하게 만들어(M=88일 때 weight 5) low-complexity encoding을 가능하게 함.
6. 추가로 rate-16/17 quasi-MTR(QMTR, j=3) 코드를 reversed MTR/ECC 방식으로 LDPC와 결합해 RLL이 유발하는 long error event를 제거.
7. NPSOVA(noise predictive SOVA)가 QMTR의 j=3 제약에 따라 path elimination을 수행하는 것이 long error event 억제에 중요.
8. 학습/최적화 절차 없음 (구조적 구성법).
9. 결과: 기존 array+RA 코드 대비 error floor 대폭 개선, rate-200/201 HR-RLL 대비 SER=10^-4에서 1.2dB 이득, SER=10^-6 이상에서 error floor 없음.
10. 한계: 시뮬레이션만 수행(HW 미설계), 자기기록 채널 전용 실험(NAND 채널과 무관), RLL/MTR 결합이 필수 전제.
