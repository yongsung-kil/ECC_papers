### B. 알고리즘 요약

1. 시스템: 코드화된 generalized linear system(GLS) `y = Q(Ax+n)`, unitarily-invariant 채널 `A`(N=500, 조건수 κ=10), QPSK, `R_LDPC=0.5`.
2. 문제: GOAMP/GVAMP는 uncoded GLS에서 replica Bayes optimal이나, 임의 입력분포에 대한 정보이론적 한계(achievable rate)가 복잡한 SE 때문에 미해결.
3. 모델/가정: LD(LMMSE)와 두 NLD(비선형 제약 Ψ, 코딩 제약 Φ)로 구성, DIDO transfer function이 서로 결합된 multi-layer SE.
4. 핵심기법 1단: DIDO SE를 enhanced LD(ELD)로 묶어 등가 단일입출력 variational SE(VSE)로 변환 - inner-iteration으로 `v_z,*` 고정점 해결.
5. 핵심식: VSE `v̂_t^x = φ̌_SE(ρ_t^x)`, `ρ_{t+1}^x = γ̌_SE(v̂_t^x)` - achievable rate 분석을 SISO로 환원하기 위함.
6. 핵심기법 2단: Lemma 1로 II-GOAMP/GVAMP와 원 GOAMP/GVAMP가 동일 SE 고정점(수렴 MSE 동일)임을 증명.
7. 핵심식: achievable rate `R = ∫_0^{γ̌_SE(0)} φ̌_SE^C(ρ)dρ`, decoding tunnel 조건 `φ̌_SE^C < γ̌_SE^{-1}`.
8. 최적화 절차: optimal coding principle - MMSE decoding transfer `φ̌_SE^C`를 `φ̌_SE^{C*}=min{φ̌_SE^S, γ̌_SE^{-1}}`에 매칭하여 rate 최대화(Lemma 3, irregular LDPC degree profile 설계).
9. 결과: 최적화 LDPC가 유한길이에서 임계한계 1.0 dB 이내, 기존 P2P regular(3,6)/irregular 대비 BER 0.8~2.8 dB 이득(Fig.6).
10. 한계: HW 미설계, 시뮬만, clipping GLS/AMP 수신기 특화 - Bayes optimal이라도 error-free 보장 불가라고 명시.
