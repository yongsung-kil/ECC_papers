### B. 알고리즘 요약

1. 대상: AWGN 채널용 rate=1/2 QC-LDPC(GEO-LDPC), 부호길이 384~3200 bit, base matrix 16×32, expanding factor `Q = n/nb`.
2. 문제: 기존 구성법은 가변 부호길이를 지원 못하거나(802.16e 대비) error-floor/BER이 나쁨.
3. 채널/모델: AWGN, iterative decoding에서 short cycle(girth-4/6)이 belief 오차를 누적시켜 성능 열화한다는 전제.
4. 핵심기법 1: base matrix의 각 `1` 위치에 대해 모든 가능한 shift(offset) 값을 시도, girth와 EMD 정보로 최적 offset 선택(Fig.3 탐색).
5. 핵심식: cycle의 EMD `= Σ(d_i − 2)` — 큰 EMD가 error floor를 낮춘다는 지표로 offset 선택 기준.
6. 핵심기법 2: double-diagonal H2 구조로 linear-complexity fast encoding 지원(down-triangle 유사 성질).
7. 구현 디테일: 첫 행은 정보 부족으로 탐색을 2회 반복. 다른 길이는 `p_i,j = floor(p'_i,j·Q/Q')` 규칙으로 offset 스케일.
8. 최적화: density evolution으로 degree distribution `λ(x)=0.46875x²+0.34375x³+0.0625x⁷+0.125x⁹` 선택 후 PEG로 좌측 mother matrix 구성.
9. 결과: normalized min-sum(factor 0.8, 100 iter) 시뮬, 802.16e보다 낮은 BER, 3200bit에서 2.2dB에 error floor<1e-9 수렴(그림 기반).
10. 한계: HW 미설계, rate=1/2 무선 비교 전용, gate/latency 수치 없음.
