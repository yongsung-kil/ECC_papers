### B. 알고리즘 요약

1. 대상: QC-LDPC-BC에서 유도한 시불변 LDPC 컨볼루션 부호(LDPC-CC), 다항식 syndrome former `H^T(D)`로 기술, 예제 rate 2/5~3/5.
2. 문제: LDPC-CC는 low-density라 메모리 `ms`가 커서 거리스펙트럼(free distance `df`, weight enumerator `Aw`) 직접계산이 매우 복잡.
3. 모델: 부호어 v가 `v·H^T=0` 만족, Property 1(무게 l 부호어 = 합이 0인 l개 행) 기반.
4. 핵심기법 1단: `H^T(D)`의 열을 `I`개 부분행렬 `Hpi^T(D)`("super code")로 분할, 각 super code의 거리스펙트럼을 개별 계산.
5. 핵심식: super code 공통 부호어 `W(D)`는 식(12)에 의해 원부호의 부호어이기도 함 → 분할 계산이 원부호 추정으로 연결.
6. 핵심기법 2단: super code 저무게 부호어 간 주기 shift(`m_pj`) 조합의 linear dependence를 평가해 공통 부호어 탐색(식 15).
7. 디테일: 최소차수 ≤ c인 multiplexed 부호어만 포함, 비겹침 선형결합은 중복 카운트 제외(식 17), shift 최댓값으로 탐색 종료 보장.
8. 성질: 얻은 값은 free distance의 상한, `Aw`의 하한. 저무게 부호어에서 추정 정확도가 높음.
9. 결과: Tanner (21,3,5) LDPC-CC에서 `df=24` 정확 산출, (ms,3,5) 부호들이 free distance 24·부호어 5개 공유 확인, BEAST와 달리 복잡도가 `ms`/제약길이 무관.
10. 한계: 부호 성능·디코더·HW 없음, 거리스펙트럼 추정만 하는 이론 분석 도구.
