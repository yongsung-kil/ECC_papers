### B. 알고리즘 요약

1. 시스템: longitudinal magnetic recording 채널, column weight 3인 QC-LDPC 코드(`length=4623`, `rate=8/9≈0.889`)를 ERP4(EPR4) target으로 SOVA와 channel iteration하는 구조, Xilinx Virtex II 8000 FPGA에 전체 구현.
2. 문제: 기존 LDPC 성능평가는 AWGN + idealized PR 채널만 가정하나, 실제 자기기록 신호는 transition noise, nonlinear transition shift(NLTS), partial erasure(PE) 등 매체 비선형성을 추가로 겪음; 이들을 반영한 현실적 채널에서의 decoding behavior(특히 error-floor)가 불분명.
3. 채널 모델: 고립 천이응답은 Lorentzian pulse(`PW50`)로 모델링, readback 신호는 노이즈보정된 천이응답 중첩(식 (2))이며 폭 변동(transition noise)과 위치 시프트(jitter)를 포함; 매체 비선형성은 PE 비율 `alpha`(인접 천이 개수에 따라 1, ~0.9, 등)와 NLTS 시프트 계수 `delta`로 모델링(식 (4)).
4. 핵심 기법 1단: LDPC 인코더-채널-디코더 전체를 FIFO 구조로 FPGA(VHDL) 구현, PE/NLTS 효과를 비트 데이터 패턴 기반으로 결정해 신호를 왜곡시킨 뒤 11-tap 등화기로 PR target에 정합.
5. 핵심 식 의미: SNR 정의(식 (3))는 고립천이응답 파워 대 (jitter noise, width variation, AWGN을 포함한) 노이즈 파워스펙트럴밀도 비율이며, transition noise가 지배적 노이즈 성분임을 명시.
6. 핵심 기법 2단: LDPC 디코더(inner iteration)와 SOVA 검출기(channel iteration) 간 soft 정보를 반복 교환하는 이중 반복 구조 - channel iteration으로 약 3.5dB 성능 향상.
7. 구현 디테일: Box-Muller 방법으로 transition/additive 노이즈를 FPGA 상에서 생성, 소프트웨어 디코더로는 도달 불가능한 낮은 BER(~10^-11)까지 FPGA 실측으로 신뢰성 있게 추정.
8. 최적화 절차: 별도 학습 없음, PE ratio/NLTS 계수를 스윕하며 BER 영향 분석(PE ratio 0.7 이하에서 손실 급증, NLTS 0.15Tw 초과 시 precompensator 필요).
9. 결과: 정규화 기록밀도 `Dc=2`에서 SNR 20.5dB에 BER 10^-11, 전체 코딩이득 20dB 이상; error-floor는 BER 10^-8 부근에서 시작; inner iteration 50→10 감소 시 손실 0.6dB뿐이나 throughput 5배 증가(35Mb/s 수준).
10. 한계: FPGA 처리량이 채널 시뮬레이터 복잡도와 메모리 제약으로 35Mb/s에 그침, column weight 3 LDPC는 새 부호설계가 아닌 기존 문헌 재사용, 자기기록 채널 특유 노이즈 모델에 국한.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LDPC 디코더-SOVA channel iteration (inner/outer 반복) | `decoder.cpp` `LDPC_Decoder()` | 채널 검출기(SOVA)와의 반복 구조는 자기기록 PR채널 전용, NAND는 별도 채널검출기 없이 soft-read LLR을 직접 사용해 대응성 낮음 |
| transition noise/NLTS/PE 채널 모델링 | `channel.cpp` | AWGN/RBER/fixed-error 채널 모델과 근본적으로 다른 자기기록 특유 노이즈로 대응 없음 |
| FPGA 기반 저-BER(10^-11) 실측 검증 방법론 | `TV_Dec_Inter/TV_Dec/TV_Enc` | 방법론적 유사성(HW 기반 초저BER 검증)은 있으나 채널 모델 자체는 대응 없음 |
| inner iteration 축소를 통한 throughput-BER 트레이드오프 관찰 | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | iteration 수와 성능/처리량 트레이드오프라는 관찰은 일반적이나 논문 고유 기법이 아니라 이식 요소로 보기 어려움 |

적용 가치: 낮음 — 자기기록 채널의 현실적 노이즈 모델과 SOVA-LDPC 반복 구조를 FPGA로 관찰한 시스템 연구로, LDPC 알고리즘/부호설계 자체의 신규 기여가 없어 NAND binary QC-LDPC에 이식할 요소가 제한적.
