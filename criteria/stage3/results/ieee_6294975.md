### B. 알고리즘 요약

1. 대상: (3,6)-regular LDPC, `N=2000`, `Rc=1/2`, AWGN, hard-decision BF 복호.
2. 문제: WBF/MWBF/IMWBF는 최적 가중치 `α`를 SNR별 시뮬로 사전 탐색해야 하고, α가 최적 아니면 성능 급락.
3. 착안: soft BP(SPA)의 첫 iteration LLR `L(c_j|y)`에서 flipping 판정을 이론 유도 → 채널독립.
4. 근사: `ϕ(x)=ln((e^x+1)/(e^x−1))`를 3구간 piecewise-linear `ϕ_p`로 근사해 `E_j`를 단순화.
5. flipping 함수 `e_j = Σ_{i∈M(j)}(2s_i−1)w_{i,j} − y_j`, `w_{i,j}=Σ_{j'∈N(i)\j}|y_{j'}|` (기존 WBF의 "최소신뢰비트" 대신 참여비트 신뢰도 합).
6. 의미: `e_j`가 클수록(양수) 위배 신드롬 많아 해당 비트가 오류 가능성 높음 → flip 대상.
7. 절차: 초기화 시 `w_{i,j}` 저장 → 신드롬 0이면 종료, 아니면 `e^(k)` 계산 → 최대 `e_j` 비트 1개 flip → `k_max`까지 반복.
8. 이점: `α` 사전탐색 불필요, 채널독립, hard-decision BF의 낮은 복잡도 유지.
9. 결과: `BER=5e-6`에서 WBF 대비 1.5dB(MWBF 1dB, IMWBF 0.5dB) coding gain, iteration 25~48% 감소 (`Eb/N0=5dB`, CIWBF 88.03회).
10. 한계: hard-decision BF 계열이라 SPA/MSA보다 성능 낮음, 단일 (3,6)-regular 시뮬만, HW·QC-LDPC 미검증.
