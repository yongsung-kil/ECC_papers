### B. 알고리즘 요약

1. 대상 부호: 1/2·2/3·3/4 rate AR4JA protograph LDPC(P-LDPC), 짧은~중간 길이 `N={300,500,700}`; base matrix를 `K`번 copy-and-permute로 lifting.
2. 문제: 짧은~중간 길이에서 H 크기가 작아 PEG 랜덤 선택이 비영원소 간 거리를 짧게 만들어 연속 버스트 오류를 못 이김.
3. 채널 모델: 단일 버스트 소거(SBE) + AWGN 결합; 길이 `Ne` 연속 버스트, 전이확률 `p=Ne/N`, 시작점 랜덤. 버스트/펑처 비트 사전정보는 `q=0.5`로 초기화.
4. 핵심 기법: PPEG lifting에서 edge permute 시 각 행의 "평균거리"(H(x,·) 내 대상 원소와 나머지 비영원소 간 거리 평균)를 최대화하도록 제약 → IPPEG.
5. 핵심 원리: CN에 연결된 VN 간 평균거리가 버스트 길이 `Ne`보다 크면 버스트에 안 걸린 이웃 VN 정보로 오류 VN을 복원 가능.
6. 절차: base matrix 순회하며 lowest row-weight CN을 선택하고, tanner 트리 확장으로 girth 유지하면서 largest average distance를 갖는 CN에 edge 연결(step 1~6).
7. 부수: 버스트를 흩뿌리는 random interleaver 유무 두 경우 모두 비교.
8. 복호: 표준 belief propagation(BP) 그대로 사용, 디코더 변형 없음.
9. 결과: interleaver 없을 때 IPPEG가 PPEG 대비 BER 개선(p 클수록·길이 짧을수록·rate 낮을수록 이득 큼); interleaver 추가 시 짧은 길이 고SNR에서 약 2~3dB 개선, 단 IPPEG 상대이득은 감소.
10. 한계: 시뮬레이션만(HW 미설계), 무선/우주 버스트 채널 특화, gate/throughput/정량 이득 수치 없음.
