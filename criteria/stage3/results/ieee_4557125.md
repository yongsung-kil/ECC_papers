### B. 알고리즘 요약

1. 대상 부호: serial turbo 구조(외부 `Rep_r` 반복부호 + interleaver + 내부 `Φ_N = φ_N ∘ Sum_s`)로 표현되는 LDPC 코드. Parity check matrix `[H_N | K_N]`에서 `H_N`은 저밀도(행당 최대 `s`, 열당 최대 `r`개의 1), `K_N`은 내부 recursive convolutional encoder `φ(D)`에서 유도되는 저밀도 행렬.
2. 풀려는 문제: 기존 LDPC 인코딩은 block length에 대해 quadratic 복잡도인데, staircase(accumulator) 구조의 RA/IRA는 선형시간 인코딩이 가능. 본 논문은 accumulator 자리에 임의의 `k×k` non-catastrophic·recursive convolutional encoder `φ(D)`를 넣어 일반화하고 이때의 waterfall 성능을 분석.
3. 채널/가정: memoryless binary-input symmetric-output 채널(BSC 또는 BIAWGN), 복호는 ML(최우) 가정 하 coding theorem으로 증명(실제 Sum-Product 복호는 별도 논의).
4. 핵심 기법: interleaver `Π`를 `S_{rN}`(전체 순열)에서 균등 랜덤으로 뽑는 앙상블에서 평균 bit/word 오류확률 `P_b(e)`, `P_w(e)`을 union-Bhattacharyya bound와 최소거리 하한(`P(d_min < d*)`)으로 상하한 유도 (Theorem 1). `d*`는 truncated error event의 최소 출력 weight로 정의되는 effective free distance.
5. 핵심 식: `c1·γ^{d*}·N^{-μ} ≤ P_b(e) ≤ c2·γ^{d*}·N^{-μ} + O(N^{-μ-1})` — interleaver gain(코드 길이 `N`↑일수록 오류확률이 다항식으로 0에 수렴)을 보이며, 지수 `μ`는 반복률 `r`에만 의존(고전 serial turbo 결과의 확장).
6. 확장(2단): 2-cycle이 없도록 interleaver를 제한한 부분앙상블 `R_{r,s}`(정규 LDPC의 Tanner graph 2-cycle 배제와 동치)를 정의, `P(R_{r,s})`가 `N→∞`에서 0으로 수렴하지 않는 상수임을 이용해 expurgation 기법으로 Theorem 2 유도 — 이 경우 effective free distance `d*_exp`가 내부 encoder `φ(D)`의 weight-2/weight-3 입력에 대한 최소 출력 weight(`d^φ_{f,l2}`, `d^φ_{f,l3}`)에 의존하게 되어, **내부 encoder 선택이 실제 설계 파라미터가 됨**.
7. 부수 예시(E1~E4): E1은 표준 staircase(accumulator, `φ(D)=1/(1+D)`), E2~E4는 `k=1,3`인 다양한 recursive convolutional encoder로 서로 다른 `K_N` 구조(및 cycle 특성)를 생성하는 구체 사례 제시.
8. 학습/최적화 절차 없음(해석적 앙상블 분석). 다만 Section V에서 내부 encoder의 구조(Tanner graph 내 짧은 cycle 개수)가 waterfall 이후 실제 복호 성능에 영향을 줄 수 있음을 추가로 논의.
9. 결과(정성적): r≥2에서 bit 오류, r≥3에서 word 오류에 interleaver gain 존재. 2-cycle 배제 앙상블에서 effective free distance가 inner encoder 의존적으로 개선됨을 이론적으로 증명. 구체 수치(BER 곡선, gate count 등)는 본문에 없음.
10. 한계: ML 복호 가정의 이론적 코딩정리이며, 저자 스스로 "실제 Sum-Product 반복복호 시뮬레이션 결과는 기대한 명확한 계층 구조를 보이지 않는다"고 명시(진행 중 연구). HW 설계·실측 전무, error-floor 분석도 없음.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Inner encoder(accumulator 대체) 부호 구성 아이디어 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC-LDPC H-matrix 고정 구조라 staircase/RA 계열 구성법 자체는 이식 난이도 높음(프로파일 §4) |
| 선형시간 인코딩(RA 계열) 개념 | `encoder.cpp` `LDPC_encoder_4KB()` | Prime ECC는 이미 dual-diagonal(staircase류) 방식으로 선형시간 인코딩 구현 중이라 개념 자체는 기보유, 신규성 낮음 |
| 2-cycle 배제 interleaver 앙상블(Tanner graph girth 개선) | `ecc_top.cpp` `Load_PCM()` | H-matrix 설계 시 girth 고려는 일반적 기법이나 본 논문은 QC-LDPC가 아닌 일반 turbo-LDPC 프레임 |
```

적용 가치: **낮음** — waterfall 영역 이론적 앙상블 분석(ML 복호 가정)이며 실제 Sum-Product 디코더 성능·HW·NAND 관련 실험이 전무하고, Prime ECC는 이미 dual-diagonal 선형시간 인코딩을 보유해 개념적 중복도 있음.
