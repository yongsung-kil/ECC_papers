> 총평: finite-length 성능을 예측하는 통계물리학적 scaling law를 BEC에서 일반 BMS 채널로 확장한 순수 이론 논문으로, NAND binary LDPC 이식과는 거리가 멀다.

### B. 알고리즘 요약

1. 대상: regular LDPC ensemble + 유한 메시지 알파벳을 갖는 message-passing(MP) 디코더, 채널은 일반 BMS(binary memoryless output-symmetric).
2. 문제: 기존 finite-length scaling law는 BEC의 peeling decoder 구조에 강하게 의존해 일반 채널로 확장이 안 됨.
3. 핵심 가정: 채널을 엔트로피 `h`로 파라미터화하고, density evolution의 고정점 밀도로부터 EXIT-like curve `x(h)`를 정의(임계점에서 무한 기울기+non-zero 2차미분인 "regular" curve 가정).
4. 핵심 기법: BEC 유도를 EXIT-like curve 관점으로 재유도하여 scaling parameter `a`를 `2차미분/curvature`와 메시지 분산 `V`만으로 표현되는 conjecture 형태로 일반화.
5. 핵심 식: `P_B ≈ Q(sqrt(n)*(h-h*)/a)`, 여기서 `a`는 EXIT-like curve의 곡률과 임계점에서 메시지 분산 `ζ`의 조합으로 결정.
6. 확장: 메시지 분산 `V`가 임계점 근방에서 `1/(1-γλ2(x))^2` 형태로 발산함을 보이고, 이를 트리 상의 correlation을 전이행렬 `M`의 2번째 고유값 `λ2`로 표현.
7. 부수 기법: 체인(alternating variable/check node) 상의 메시지 correlation을 재귀적 확률(`p(i,j)`, `q(i,j)`) 및 행렬 `C`, `K=FD+γC^T FD C`로 정식화해 `ζ`를 근사 계산.
8. 학습/최적화 절차: 별도 학습 없음, 고유값/고유벡터 기반 해석적 계산.
9. 결과: Gallager algorithm A ((3,3)-code, threshold ε*≈0.223)에서 `a≈0.7847`, Decoder with Erasures ((3,6)-code, threshold ε*≈0.07076)에서 `a≈1.04±2` — 시뮬레이션과 잘 일치.
10. 한계: 순수 시뮬레이션/해석적 검증뿐(HW 미설계), regular ensemble·특정 디코더(Gallager A, DE)에 국한, error floor 영역은 범위 밖.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| finite-length scaling law (통계적 오류확률 예측) | 대응 없음 | 코드 구성/최적화 이론이며 Prime ECC의 실행 코드 모듈과 직접 대응 없음 |
| Gallager algorithm A / Decoder with Erasures | 대응 없음 | Prime ECC은 min-sum 기반이며 hard-decision Gallager A류 알고리즘 미보유 |

적용 가치: `낮음` — NAND용 QC-LDPC 부호/디코더 설계에 즉시 이식 가능한 기법이 아니라, ensemble 수준의 finite-length 성능 예측을 위한 통계적 도구이며 대상 코드도 regular ensemble·비-QC 구조.
