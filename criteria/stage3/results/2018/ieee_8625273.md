### B. 알고리즘 요약

1. irregular LDPC 앙상블(degree 분포 `λ(x)`, `ρ(x)`, rate 0.5, girth 6, n=500~20000) Tanner 그래프의 LETS 통계 분석.
2. 문제: LETS는 error-floor의 가장 해로운 substructure지만 탐색이 NP-hard - 앙상블 기대값으로 분포를 근사하려 함.
3. 모델: 무한 블록길이 점근 영역에서 유한·비영 평균 multiplicity를 갖는 LETS는 단일(simple) cycle 포함 구조뿐 (기존 [8] 결과).
4. Theorem 1: 특정 길이 `c`와 특정 VN/CN degree 조합 `(αc, βc)`를 가진 사이클의 기대 개수 `E(Nαc,βc)`를 configuration 모델로 유도 (식 (2)).
5. 핵심 식 `E(N)~(1/c)∏(λi(i-1))^αi·∏(ρj(j-1))^βj`: degree별 edge 비율이 사이클 발생 확률을 결정.
6. Theorem 2: 일부 노드 degree를 자유 선택하도록 일반화, Specht's ratio S(h) 배수 내 점근 상한 (식 (12)).
7. Proposition 1: (a,b) LETS의 점근 multiplicity = VN degree 합이 `2a+b`인 길이 `2a` simple cycle 수와 동일.
8. Example 1: 2-term `λ(x)` 앙상블에서 선형연립 `iαi+jαj=2a+b`, `αi+αj=a` 풀어 (a,b) LETS multiplicity 폐형식 (식 (13),(14)) 도출.
9. 결과: a=3~6, 3≤b≤2a LETS multiplicity 이론값이 유한길이(n~1000부터) 랜덤코드 실측과 잘 정합 (Table I~IV).
10. 한계: 순수 조합론/앙상블 분석 - 특정 부호·디코더·HW 미제공, stopping/absorbing set 확장만 언급.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LETS/trapping-set 점근 multiplicity 이론 | 대응 없음 | 앙상블 통계 분석, 특정 H-matrix 설계나 코드 모듈에 대응 없음 |
| irregular degree 분포 기반 사이클 계수 | [ecc_top.cpp](../Prime_ECC_3.1_Claude/) `Load_PCM()` (H-matrix) | 우리는 고정 QC-LDPC(binary, degree 17) - irregular 앙상블 분석 직접 적용 불가 |
| error-floor(LETS) 분석 | [corner.cpp](../Prime_ECC_3.1_Claude/) (수렴실패 추적) | error-floor 관심사는 공유하나 우리는 고정부호 시뮬 추적, 앙상블 예측 아님 |

적용 가치: **낮음** - error-floor 원인인 LETS를 다루나 순수 앙상블 통계 이론이라 특정 QC-LDPC(Prime ECC 고정 부호)에 떼어 쓸 설계 절차·디코더·HW가 없다. 부호설계 관점 참고 배경지식 수준.
