> 총평: layered NMSA에 double-layer 메시지 추가(NALA)로 0.1~0.15dB·저iteration 이득, column-1 후처리로 error-floor 개선. 알고리즘 이식은 쉬우나 연산자원 증가와 PCM 구조 의존이 걸림.

### B. 알고리즘 요약

1. 대상: (1120,840) irregular QC-LDPC (IEEE P802.3bn D3.0, `R=0.75`), BPSK/AWGN, layered NMSA 기반, max iter 5.
2. 문제: 기존 row-layered는 data dependency로 현재 layer 완료 전 다음 layer 시작 불가 → 특정 layer에 없는 변수는 갱신 안 된 APP로 참여, 수렴 지연.
3. 모델: sparse PCM `H`, `H·x^T=0`; C2V는 normalized min-sum `R = δ·sgn·min|Q|`(정규화계수 `δ<1`).
4. 핵심 기법1 (NALA): PCM을 layer 순서만 바꾼 part A/part B 두 벌로 동시 처리, part A의 layer n은 직전 layer(n-1) 뿐 아니라 part B의 다음 layer(n+1) 새 C2V `R'_ij^k`까지 반영: `Q = L - R^{k-1} + (R'^{k} - R^{k-1})`.
5. 핵심 의미: 한 layer 갱신에 더 많은 최신 C2V 메시지 활용 → 수렴 가속(연산자원 증가 대가).
6. 핵심 기법2 (column-1 correction): column weight=1 변수(열)에 오류 집중 관찰, max iter 후 단 하나의 unsatisfied check가 column-1 변수를 포함하면 해당 bit flip 시도, `H·d^T` 만족하면 채택·아니면 rollback.
7. 구현 디테일: part B는 part A와 `R_ij^k` 공유하고 `R'_ij^k`는 저장 안 함; column-1 전략은 유사 구조 PCM이면 타 알고리즘에도 적용 가능.
8. 학습/최적화: 없음.
9. 결과: NALA가 layered NMSA 대비 전 Eb/N0에서 +0.1~0.15dB, 낮은 max iter(3)에서 특히 우수; column-1 정정은 고 Eb/N0(2.8dB)에서 모든 알고리즘 상회.
10. 한계: HW 미설계·시뮬뿐, NALA 이중 처리로 연산/메모리 증가, column-1 정정은 PCM에 weight-1 열 존재 전제.
