### B. 알고리즘 요약

1. 대상: protograph 기반 QC-LDPC 중 base matrix 원소가 `l≥2`인 multiple-edge 부호. exponent matrix `B`의 각 벡터 원소가 CPM 합으로 lifting.
2. 문제: multiple-edge는 minimum Hamming distance 이점이 있으나 girth 보장 조건·최소 lifting degree `N` 연구가 거의 없고 탐색이 매우 느림.
3. 모델: Tanner 그래프의 4-cycle 회피 조건(식 3)을 exponent matrix 전 부분행렬에 적용해야 하는 조합 문제.
4. 기법1(차이행렬): `B`에서 원소쌍 차로 `D`(행/열 내), `Δ`(행 인덱스쌍) 두 차이행렬 정의. Definition 2/3.
5. 핵심(Theorem 1): girth 6(4-cycle free) ⟺ `D`의 각 행·열에 중복 원소 없음 AND `Δ`의 각 행에 중복 없음.
6. 기법2(하한): 파라미터 `X,Y,Z`(D 행/열, Δ 행의 최대 disjoint 원소 수)로 lifting degree 하한 `N = max{X,Y,Z}` 유도(Theorem 2).
7. 부수 트릭(탐색공간 축소): exponent matrix 최대값 `t`를 최소화하면 원소당 후보가 `N` 대신 `t+1`개로 감소. `t = max{⌈X/2⌉,⌈Y/2⌉,⌈(Z-1)/2⌉}`(Proposition 1).
8. 학습/최적화: 없음(해석적 유도 + 하한부터 시작하는 exhaustive search 가이드).
9. 결과: 제안 하한을 만족(tight)하는 column weight 3/4 부호 제시. single-edge 대비 동일 (m,n)에서 길이 훨씬 짧음(Table II). (4,8)-regular 예 N=10, rate≈0.5.
10. 한계: BER/복호 시뮬·HW 전무, girth 6에 한정, base matrix 주어진 경우에만 적용.
