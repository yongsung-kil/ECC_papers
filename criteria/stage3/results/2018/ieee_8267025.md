> 총평: rate-compatible·length-scalable 다중rate QC-LDPC 부호족을 저복잡도로 설계하는 우수한 구성법이나, 고정 고rate 단일부호를 쓰는 NAND Prime ECC에는 목적이 어긋나 이식가치 제한적.

### B. 알고리즘 요약

1. 시스템: DTTB(지상파 방송, DVB-T2/ATSC 3.0) 및 5G eMBB용, 다중 rate·다중 길이를 단일 nested PCM 집합으로 지원하는 raptor-like QC-LDPC(RL-QC-LDPC) 부호족. base matrix `B`를 lifting factor `L`로 확장.
2. 문제: 기존 QC-LDPC는 rate/길이별 PCM을 개별 설계해 HW 복잡도가 과도. rate-compatible + length-scalable을 동시에 저복잡도로 구성하는 방법 부재.
3. 구조/모델: raptor-like base matrix = 5개 서브행렬(`Z,I,A,C,D`), `C`는 dual-diagonal 가정. incremental redundancy로 부분복호 가능. 채널은 BI-AWGN.
4. 핵심기법 1단: `PMG`(Progressive Matrix Growth) — nested base matrix들을 정보변수노드·검사노드 양방향(2D)으로 entry 단위 점진 확장. `MET-DE`(multi-edge-type density evolution)로 각 `(K,M)` 쌍의 SNR 임계 예측.
5. 핵심 식: 코스트 `D(B(K,M),P)=S_Shan − S_DE` (Shannon 한계 대비 임계 gap), 부호족 코스트 `Dc_j`는 가중 평균 → 각 nested rate의 capacity 근접성을 최적화하는 목적함수.
6. 핵심기법 2단: 2D 확장에서 각 라운드마다 unique base matrix 수 `u0`가 최소인 corner entry를 선택 → MET-DE 연산 `(bmax+1)u0`을 최소화. 파라미터 `uref,u/uref,l/Jmax`로 연산량·메모리 상한 제어(복잡도 `(K,M)` 쌍 수에 선형).
7. 구현 디테일: information-bit puncturing(`P`)으로 고rate 임계 개선, base graph parallel edge(`bmax>1`)로 더 compact화, QRO(quasi-row orthogonality)로 row-parallel 복호 지원(고throughput 목적).
8. 최적화 절차: Step1 core 행렬 열거→Step2 정보변수노드 확장→Step3 검사노드 확장→Step4 2D 확장(Algorithm 1), 각 단계 최선 후보 `J`개만 저장.
9. 결과: Case 4b에서 rate 1/17~24/25, 1056개 nested base matrix, 평균 임계 gap 0.429dB(BI-AWGN). 유한길이(L=128/1024, k=1024/16384) BER=1e-5에서 MET-DE 대비 0.15~0.27dB gap. flooding, 60 iteration.
10. 한계: 순수 부호 구성법(HW 미설계, gatecount/throughput 수치 없음), 방송/5G 저~중rate 다중rate 타깃, QRO는 언급뿐 실제 병렬 HW 미구현.
