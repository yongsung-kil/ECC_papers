### B. 알고리즘 요약

1. 대상은 rate `R=1/3` 블록별 브레이드 컨볼루셔널 부호(BCC), 두 개의 `Rcc=2/3` 시스터매틱 컨볼루셔널 성분부호와 길이 `N` 인터리버 3개로 정의되는 spatially-coupled turbo-like 부호.
2. 풀려는 문제: AWGN 채널에서 BCC의 정확한 DE 방정식이 없어 임계값 계산에 Monte-Carlo 방법이 필요한데, spatial coupling으로 edge type이 많아 계산량이 크다.
3. 핵심 기법 1단: Monte-Carlo Density Evolution(MC-DE)으로 variable node 갱신 → trellis node 갱신 → 메시지 밀도에서 표본 추출을 반복해 복호 오류확률 수렴 임계값 추정.
4. 핵심 식: 소거채널 예측(ECP) `σ* ≈ CG^-1(1 - ε*)` — BEC 임계값 `ε*`에서 AWGN 임계값을 근사(`CE(ε*)≈CG(σ*)` 관찰 기반).
5. 확장: 랜덤 puncturing 앙상블에 대해 단일 파라미터 `θE=(1-R)/ε*`로 모든 상위 rate의 BEC 임계값을 예측(P2 puncturing 한정).
6. θG 예측: unpunctured AWGN 임계값이 있으면 `θG=(1-hG(σ*))/R`로 저rate에서 정확도 개선, 고/저rate 결합한 mixed prediction 제안.
7. 학습/최적화: 없음. 통계 안정화 기준은 반복 내 BER 변화 `.0001`.
8. 결과: SC-BCC에서 ECP와 MC-DE 임계값이 거의 일치, coupling memory `m=3`이면 Shannon limit에 근접(예: rate 1/3에서 -0.46 dB).
9. 시뮬: sliding window decoder(w=5, 20 iteration)로 BER 곡선이 예측 임계값과 일치 확인.
10. 한계: LDPC가 아닌 turbo-like BCC 대상, HW 설계 없음, 임계값 이론 분석에 국한.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BCC 부호구조 / trellis 성분부호 | 대응 없음 | turbo-like 부호로 QC-LDPC 구조와 무관 |
| MC-DE 임계값 분석 | 대응 없음 | DE는 설계용 이론 도구, Prime ECC 복호기와 무관 |
| ECP / θE 예측 | 대응 없음 | 임계값 근사 이론, 코드베이스 대응 없음 |
