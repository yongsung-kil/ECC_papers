> 총평: 컨볼루셔널 LDPC-C 부호의 주기성 기반 short-cycle 회피 구성법으로, 통신(가변 프레임/연속모드)용이며 NAND 블록 binary QC-LDPC ECC와는 부호 클래스가 달라 이식 가치 낮음.

### B. 알고리즘 요약

1. 대상: `(n0b0, k0b0, M)` QC-LDPC-C(컨볼루셔널) 부호, rate `R=k0/n0`, semi-infinite parity-check 행렬이 CPM(순환치환행렬)로 구성.
2. 문제: LDPC-C 부호를 블록 부호에서 unwrapping하면 length-4 cycle이 생기고, 직접 구성은 계산량이 지수적으로 증가.
3. 모델: 시변 주기 `T` LDPC-C(`B(i,t+T)=B(i,t)`), 기저행렬 `Hb` 원소는 `{-1,0,...,b-1}`(−1=영행렬, 그 외=순환치환 offset).
4. 기법1(결정부): fast encoding 위해 각 block row parity-check 부분의 행·열 비음수 개수를 1로 두고 결정적 서브행렬을 먼저 채움(선형복잡도).
5. 핵심: 주기 `T`이므로 `T` 기간 열만 구성하면 됨. `(T+2M)` 행 범위에서만 short cycle 고려.
6. 기법2(랜덤부): 각 열에 변수노드 선택 시, 주기 확장으로 생기는 위치 + 새 위치가 만드는 length-4 cycle을 모두 회피(집합 `U1`,`U`,`F0` 갱신으로 girth 관리).
7. 절차: Program 1 — 결정부 채움 → 랜덤부 채움 시 check node 후보집합 `A`에서 `F0=A\U` 선택, 도달 시 girth를 2씩 감소·`U0` 재계산.
8. 복잡도: 전체 구성 복잡도 `O(b·((T+M)mb)^3)` (check row 수의 3승).
9. 결과: BPSK/AWGN, 최대 30 iteration, LBP 복호. IEEE 1901 표준 부호(CC12/CC23-1901, Ref-CC2) 대비 동일/유사 파라미터에서 BER 우수, 인코딩·디코딩 복잡도 낮음.
10. 한계: 순수 부호 구성법·시뮬(BER 곡선)만, HW 미설계, 컨볼루셔널 부호 특화(연속 스트림).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QC-LDPC-C 기저행렬 주기적 bit-filling 구성 | 대응 없음 | 컨볼루셔널(semi-infinite) 부호 구성 — Prime ECC는 블록 QC-LDPC 고정 |
| length-4 cycle 회피 / girth≥6 | [ecc_top.cpp](../Prime_ECC_3.1_Claude/) `Load_PCM()` (부호구조 로드, 개념적) | H-matrix 설계 품질 개념은 닿으나 컨볼루셔널 절차라 직접 대응 아님 |
| CPM offset·주기 확장 | 대응 없음 | Prime ECC circulant shift와 개념 유사하나 부호 클래스 상이 |
| LBP 복호 | [decoder.cpp](../Prime_ECC_3.1_Claude/) `LDPC_Decoder()` (개념적) | 복호는 표준 BP, 논문 기여 아님 |

적용 가치: **낮음** — 부호 클래스가 컨볼루셔널 LDPC-C로 Prime ECC의 블록 binary QC-LDPC와 다르고 HW 설계도 없어, cycle-4 회피 구성 아이디어만 참고 수준.
