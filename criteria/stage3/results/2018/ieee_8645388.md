### B. 알고리즘 요약

1. 시스템: binary linear code를 AWGN 채널로 전송, 디코더는 soft output `s∈R^n` 산출 후 `x̂=sign(s)`로 경판정.
2. 문제: 기존 neural 디코더 학습이 `cross-entropy` 손실만 써서 코드 구조(부호어 유효성)를 반영하지 못함.
3. 관찰: error correction은 단순 이진분류가 아니라 부호 구조로 비트가 결합된 structured prediction 문제.
4. 핵심 기법: `synd(s)=Hĉ`의 미분불가 문제를 완화한 `soft syndrome softsynd(s)_i = (∏sign(s_j))·min|s_j|` 도입 (min-sum의 CN 식과 동일, gradient 비자명).
5. 핵심 식: syndrome loss `ℓ_synd = (1/(n-k))Σ max(1-softsynd(s)_i, 0)` — 유효 부호어가 아닌 출력을 벌점.
6. 확장: 총손실 `ℓ_total = (1-λ)ℓ_synd + λ·ℓ_ce`, `λ∈[0,1]`. λ=1 지도, 0<λ<1 정규화, λ=0 완전 비지도.
7. 구현 디테일: 추론 시 추가 비용 없음(학습 시에만 손실 계산), all-zeros codeword로 학습.
8. 학습: Adam(lr=0.01), 10,000 minibatch×120 codeword, 전 SNR 균등 noise. NNMS(neural normalized min-sum) 디코더 사용.
9. 결과: (63,45)BCH, (16,8)LDPC, (128,64)polar, (200,100)LDPC 전 SNR에서 FER 소폭·일관 개선; BER은 비일관(개선/악화 혼재).
10. 비지도: λ=0에서 all-zeros 학습 시 붕괴 위험, random codeword 학습으로 회피; (63,36)BCH에서 무학습 대비 개선.
11. 한계: HW 미설계, 시뮬만, short block code 대상, 학습 기반이라 고정형 디코더 직접 적용 불가.
