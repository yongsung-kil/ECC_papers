### B. 알고리즘 요약

1. 대상: `3×n` fully connected base graph(평행 edge 없음)에서 cyclic lifting한 girth-8 (3,n) 정규 QC-LDPC 부호.
2. 문제: 고SNR에서 error floor는 소형 trapping set(특히 ETS)에서 기인 → 이를 부호설계 단계에서 제거하려 함.
3. 모델: AWGN + BPSK, BP(belief propagation) 반복복호. permutation shift 행렬 `E`와 lifting degree `N`으로 QC 구조 규정.
4. 핵심관찰: girth-8 (3,n) 부호의 (5,3) ETS는 유일하게 `(5,3){2}` 형태이며 `(4,4)` ETS에 변수노드 1개 추가로 진화. 8-cycle의 4개 CN이 3 block row(u,v,w)에 분포.
5. 핵심조건: block row `v`에 CN 2개를 갖는 8-cycle을 Tanner graph에서 제거하면 (5,3)·(7,3) ETS가 자연 소멸.
6. 확장: (5,3),(7,3) ETS 제거는 그 후속(6,2),(6,0),(8,2),(8,0) 등 successor ETS도 동반 제거 → a≤8, b≤3 소형 ETS 전멸.
7. cycle permutation shift `pW = Σ(-1)^i p_i mod N` 로 cycle 존재 판정, 4/6-cycle 회피 조건 `mod(p_i ± p_j, N) ≠ 0`.
8. 최적화: cycle-controlling 절차를 붙인 random search로 shift 행렬 탐색(N=41, 63, 251 예시).
9. lifting degree 하한 유도: `N ≥ 2n(n-1)+1` (3×5 → N≥41, 3×6 → N≥61). 결과 코드 C1(len 205), C2(len 378), C3(len 2510).
10. 결과: FER에서 PEG·random lifting 대비 error floor 영역 우수. 한계: HW 미설계, 시뮬만, column-weight-3 정규·저rate 예제 한정.
