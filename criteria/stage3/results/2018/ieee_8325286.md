### B. 알고리즘 요약

1. 배경: affine permutation matrix(APM) `I_{s,a}`(`p_{i,j}=1 iff i=aj+s mod m`) 기반 APM-LDPC의 특수 사례로 AQC-LDPC를 정의.
2. 문제: QC-LDPC는 circulant만 사용 → minimum-distance/cycle 분포/error-rate에서 개선 여지, 명시적 고girth 구성 필요.
3. 핵심정의: CPM `I_s+ = I_{s,1}`와 anti-CPM `I_s- = I_{s,m-1}` 두 종류 순열행렬만 사용, `G_AQC={I_s+, I_s-}`는 order `2m`의 부분군.
4. 핵심기법1: 각 부호를 sign matrix `A`(원소 ±1)와 slope matrix `S`로 표현, Tanner graph의 각 cycle을 modular linear equation으로 환원(Theorem 4).
5. 핵심식: `2l`-cycle 존재조건 `s_l=0(a_l=1)` 또는 `gcd(2,m)|s_l(a_l=-1)`, 여기서 `s_l`은 slope/sign의 곱합(Eq.4).
6. 핵심정리: normal sign matrix `A`에 `-1∈A`이면 maximum-achievable girth는 8로 상한(Theorem 6); `-1∉A`이면 QC-LDPC와 동치.
7. 구성1(girth≥6): proper vector `v`(연속원소 홀짝 다름)와 `a_{ij}=(-1)^{ij}`, `s_{ij}=v_i·j`로 `m>(L-1)(v_max1+v_max2)`에서 girth≥6 보장.
8. 구성2(girth≥8): Theorem 12의 4개 GCD 부등식 만족하는 proper vector를 Algorithm 1로 생성, `m>2(v_max1+v_max2)(L-1)`에서 girth≥8.
9. 결과: Table I~III에서 동일 length/girth 조건 QC-LDPC[1]·APM-LDPC[5] 대비 6/8/10-cycle multiplicity·minimum-distance 우수, (J,L)=(3,15) AWGN BER에서 QC/APM보다 우수(단 PEG target-girth-8에는 열세).
10. 한계: 순수 이론·구성법, HW 미설계, AWGN 채널만, rate/length 실사용 스펙 미명시.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |

| sign/slope matrix로 girth 보증 명시적 구성 | 대응 없음 | 부호설계 도구 미포함(bit-exact 시뮬레이터) |

적용 가치: **낮음** — NAND 무관·HW 부재의 순수 대수적 부호구성 이론이며, anti-circulant 구조가 QC 고정 Prime ECC와 정합되지 않아 이식 가치가 낮다.
