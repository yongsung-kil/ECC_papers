> 총평: 4-cycle을 제거하는 교과서 수준의 check matrix 후처리로 UVC/MIMO 광통신에 특화되고 QC 구조·HW·min-sum 기여가 없어 NAND 이식 가치 낮음.

### B. 알고리즘 요약

1. 대상 시스템은 2×2 공간다중화 MIMO UVC(자외선 통신)로 BPPM 변조, MMSE 신호검출, BP 복호(10 iteration)를 사용.
2. 문제: 선행 연구[5]의 heuristic check matrix 구성이 Tanner graph에 4-loop(4-cycle)를 만들어 LLR이 4노드 사이를 무한 순환하며 정정 실패를 유발.
3. 채널: 대기 감쇠로 저SNR인 NLOS UVC, MIMO 채널이득 행렬 `H=[[1,a],[a,1]]`, `a`는 간섭광/신호 전력비.
4. 핵심 기법: 각 행의 1의 위치 벡터 `posi`를 비교, 두 행 대응 원소 합이 2가 되는 지점을 4-loop로 판정(4-loop positioning).
5. 제거: 해당 열 `p1/p2`에 1이 없는 행 `m`을 찾아 원소를 교환(interchange)하여 loop를 해소.
6. 원소 교환이 새 4-loop를 유발할 수 있으므로 check matrix 전체에 4-loop가 사라질 때까지 재검사·반복.
7. 구현 디테일: rate 1/2, 통계적 밀도분포 3×6, 부호길이 512 및 [128×256] 사용, 반복 종료 시 `H_r4` 출력.
8. SISO 시뮬 결과: 4-loop 제거로 부호길이별 약 5 dB 추가 coding gain, 512-length는 SNR 6 dB에서 35 dB coding gain·BER<10^-5.
9. MIMO 시뮬/실측: `a=0`일 때 실측 SNR 6 dB에서 29 dB gain·BER 2.6×10^-5, `a=0.2`에서 22.6 dB gain·BER 1.87×10^-4, 500 kbps 동작.
10. 한계: HW 미설계, 저bitrate(500 kbps) 광통신 testbed, 일반 non-QC check matrix 대상으로 정량 복잡도·throughput 미제시.
