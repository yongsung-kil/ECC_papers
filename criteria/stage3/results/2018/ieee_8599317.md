### B. 알고리즘 요약

1. 대상: `mp×np` QC-LDPC(CPM 기반, `H`=CPM 배열, 시프트 파라미터 행렬 `P`, base matrix `Hb`), 시뮬은 rate `R=0.5`·길이 512, `Hb`=16x32·확장인자 `p=16`(즉 H=256x512).
2. 풀려는 문제: QC-LDPC의 dense·구조적 `H`는 short cycle(girth-4)로 성능 열화 — masking으로 edge를 줄여 girth를 키우되 유연한 rate/length 확보.
3. 기법: masking matrix `Z`를 GF(2)에서 base matrix `Hb`와 원소곱 `Mb=Z⊗Hb=[zi,j·hi,j]`, `zi,j=0`이면 해당 CPM을 영행렬로 치환.
4. 핵심 식: loop 길이 조건 `Σ(-1)^{k+1} p_{ik,jk} ≠ 0 (mod p)`로 girth≥2(l+1) 보장, girth-4는 `l=1` 케이스 제거.
5. progressive 최적화 절차: full-rank·목표 rate 유지 위해 `Hb`의 고열중(column weight 큰) 열을 all-1로 보호(`w`개), 나머지 미정 원소를 집합 A로 둠.
6. 지역 girth 계산으로 loop가 지나는 base matrix 원소를 집합 B로 수집, 교집합 `Φ=A∩B` 원소를 0 후보로 지정.
7. `Φ`의 각 원소를 0으로 시도해 short ring이 줄면 확정, 0 과집중 방지 위해 column-by-column 진행, girth 목표 도달 또는 `Φ` 소진 시 종료.
8. 파라미터 `w`(보호 열 수)는 0~u, 대개 `w<u/2`가 최적, `g/u=ξ` 크면 `w`=3~6 권장 — `w`↑ 시 성능↑이나 복호 복잡도·메모리 요구↑.
9. 결과: AWGN/BPSK, BP 복호 100 iteration에서 girth≥6 보장, non-masked QC-LDPC 및 PEG irregular(평균 열중 3)보다 단길이에서 BER 우수(개선폭은 그림 손상으로 수치 미상, "미미"로 서술).
10. 한계: 저rate(0.5)·단길이(512) 일반부호, HW/합성/soft-read 전무, 개선폭 정성적, error-floor 영역 미검증.
