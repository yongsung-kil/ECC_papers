### B. 알고리즘 요약

1. 시스템: NGB-WS 위성+지상 결합 방송, QC-IRA LDPC, mother rate 1/2 length 64800, puncture로 rate 2/3(len 48600) 생성, AWGN/Rayleigh, QPSK, sum-product BP 50 iter.
2. 문제: DVB-S2/T2 부호는 독립 설계라 서로 다른 송신원 신호를 합쳐 복호(combined decoding)할 때 성능이 최적과 멀다.
3. 모델: degree distribution `λ(x)/ρ(x)`, punctured GADE(식2)로 puncture 비율 `π_i` 반영한 임계값 평가. IRA에서 parity node만 puncture(정보노드 무손상 시 복원 가능).
4. 핵심기법1(Algorithm1, greedy): mother code를 최저 rate에서 설계, parity puncture로 상위 rate 파생. `π_i`는 mother/puncture rate로 결정(식3).
5. 핵심식: 목적함수 `min Σ β_i·|T_ri - T_ri,shannon|`(식5) - 각 rate의 Shannon gap 가중합을 동시 최소화해 단일+결합 성능 균형.
6. 핵심기법2(Algorithm2): protograph 기반 punctured GADE로 서로 다른 puncture 패턴(mode1/mode2) 후보를 결합 임계값 기준 선택.
7. 구현 디테일: 결합복호 시 외부 LLR은 각 송신원 LLR의 합 `m_u0,j = Σ_s m_s,j`(식6), protograph V→C/C→V 업데이트(식7~9)로 결합 임계값 산출.
8. 최적화 절차: mode1×mode2 rate 조합의 임계값 합 `T_z`를 최소화하는 puncture 패턴 쌍 선택(중복 parity 최소화 = K-step recoverable node 최소화).
9. 결과: 단일 성능은 DVB-S2와 comparable, 결합복호에서 2/3 rate 부호 다른 puncture 패턴 사용 시 AWGN 0.6dB / Rayleigh 0.75dB 이득. NGB-WS 표준 채택.
10. 한계: HW 미설계, gate/throughput 없음, 위성+지상 다중송신 결합복호라는 방송 특화 시나리오, 단일 NAND 복호와 무관.
