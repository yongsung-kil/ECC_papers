### B. 알고리즘 요약

1. 시스템: binary AWGN + BPSK 채널에서 rate-adaptive FEC를 목표로 SC-LDPC를 QC-LDPC 기반으로 구성, FPGA(Kintex UltraScale kcu105) 에뮬레이션.
2. 문제: 단일 QC-LDPC block code는 rate 고정·coding gain 한계 → threshold saturation을 활용한 SC로 gain 확보하되 HW 복잡도를 base와 비슷하게 유지하려 함.
3. base 부호: permutation matrix 기반 large-girth QC-LDPC `(J=3, K=15, b=1129)`, girth-8, rate `R ≥ 1 − J/K`, codeword length `Kb`.
4. 핵심 기법: SC 구성은 base QC block을 `copy-and-shift`로 c번 복사·대각 배치. sub-block 단위 coupling으로 regular 부호에 irregularity(높은 column weight) 도입.
5. 핵심 식: SC codeword length `[(c×K) − m×(c−1)]×b`, c=coupled base 수, m=coupling length. coupling으로 rate가 감소(gain의 대가).
6. 확장: 동일 base를 재사용하므로 H-matrix MEM은 그대로, coupling length만 파라미터화 → 여러 rate 부호를 단일 아키텍처로 지원.
7. 디코더 트릭: layered scaled min-sum(scale 0.75), CNP는 tree 구조 min 탐색, VNP는 sum. 8-bit uniform 양자화로 정밀도 error-floor 방지.
8. HW: SC 디코더는 APP MEM이 c배 필요하나 layer index를 CNP/VNP에 추가해 empty sub-matrix 공간 미할당 → 메모리가 coupling length m과 독립인 unified 구조.
9. 결과: SC gain 0.3~0.6dB, error floor 미관측(≤1E-14). effective rate c=3,m=3/5/7 → 0.76/0.733/0.706. Imax QC=36, SC=108, 각 layer 12회 갱신.
10. 한계: rate loss, coupling length↑ 시 girth 감소로 cycle 유입 위험, iteration/latency 대폭 증가, 광통신 특화.
