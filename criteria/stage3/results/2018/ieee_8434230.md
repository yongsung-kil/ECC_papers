> 총평: 고rate QC-LDPC error-floor를 block-column 감쇠 재복호로 저비용 저감 — NAND binary QC-LDPC 디코더에 얹기 쉬운 이식성 높은 기법.

### B. 알고리즘 요약

1. 시스템: 고rate binary QC-LDPC (base matrix `Mp×Np`, circulant `z`), 대상 부호 `(18396,16644)`, `(35442,33294)`, WiMax rate-5/6 length-2304, Margulis rate-1/2 length-2640, AWGN/BSC 채널, `50` iteration layered 복호.
2. 문제: 고rate LDPC는 trapping set으로 high-SNR error floor 발생, 기존 대책은 trapping set 정보 저장이나 unsatisfied check 검출로 복잡/latency 큼.
3. 모델: `(v,w)` trapping set. `z`가 소수면 같은 block column/row에 `z-1`개 isomorphic trapping set 존재 → block 단위 처리로 다수 패턴 커버.
4. 핵심기법: 표준복호 실패 시 선정된 `NA`개 block column의 채널 LLR magnitude를 `μ<1`로 감쇠 후 단일 재복호 1회 실행.
5. 핵심식/근거: SPA/MSA에서 C2V 메시지가 최소 magnitude에 지배됨 → augmentation보다 attenuation이 다른 복호결과 유도해 정정 확률↑.
6. 확장: QC 구조상 같은 block column의 VN 동시 감쇠 = block-wise attenuation, 저장은 block column index만.
7. 구현 디테일: 재복호에 a posteriori LLR 사용(채널 LLR buffer 제거) → 저장복잡도↓, 성능은 채널 LLR 재사용과 유사.
8. 최적화: 2단계 오프라인 탐색 — 1단계 각 error pattern별 단일 block 감쇠 결과 표작성, 2단계 최다 패턴 정정 조합 `NA` 선정(`(35442,33294)`는 pattern당 `2^4×99=1584` 평가, exhaustive `~2.35e10` 대비 대폭 축소), `μ=2^-Q` brute-force(Q=4).
9. 결과: `μ=0.4375`, `NA=4`, 50 iter에서 두 QC 부호 error floor 약 2 orders 저감(FPGA XC7VX690T 검증), backtracking[6] 대비 유사 성능을 단일 재복호로 달성.
10. 한계: 실패 프레임만 재복호하나 1회 추가 latency, attenuation set은 부호·decoder·구현 종속(error pattern 재수집 필요), gate/throughput 수치는 표에만 있어 본문 미상.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 재복호 시 선정 VN LLR 감쇠 | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | 실패 후 block-column LLR 스케일링을 루프 앞단에 삽입 |
| a posteriori LLR로 재복호 (채널 LLR 재로드 안 함) | `decoder.cpp` `VN_Cal_HD()` 등 | VN 갱신 LLR 재사용, 우리 pipeline과 정합 검토 |
| block-column 단위 감쇠 (QC 구조) | `ecc_top.cpp` `Load_PCM()` | z=32 col-block 인덱스로 감쇠 대상 지정 |
| min-sum CN이 최소 magnitude 지배 | `decoder.cpp` `CNU_Update_New_Mag()` | 우리 min-sum과 동일 원리라 attenuation 효과 그대로 적용 |
| 오프라인 error pattern 수집/탐색 | `corner.cpp` (수렴실패 추적) | 실패 패턴 수집 파이프라인에 연결 가능, 탐색 자체는 오프라인 도구 |
| 조기종료/재복호 트리거 | `partial_CRC.cpp` / `full_CRC.cpp` | CRC 실패를 재복호 트리거로 활용 |

적용 가치: **높음** — binary QC-LDPC 전용 error-floor 대책이고, 감쇠+단일 재복호가 우리 min-sum 디코더 루프에 최소 추가 HW로 삽입 가능. 감쇠 block-column set만 우리 H-matrix로 오프라인 재탐색하면 됨.
