> 총평: girth-8 보장 + masking으로 short cycle 169배 감소해 error-floor 억제하는 binary QC-LDPC 부호설계. 방법론은 이식가능하나 순수 code-design이라 Prime의 고정 H-matrix 교체·재검증 부담이 커 추천도 중.

### B. 알고리즘 요약

1. binary QC-LDPC를 GF(2^s)상 dmin=5 RS 부호의 패리티행렬 `BRS(4,n)=[(β^i)^j]`(4행)로부터 CPM-dispersion하여 구성.
2. 문제: QC-LDPC의 Tanner graph short cycle(특히 girth 6/8)이 waterfall·error-floor 성능을 저해 - girth>=8을 대수적으로 보장하고 싶음.
3. 핵심조건: girth>=8 <=> base matrix가 `2×2/3×3 SM-constraint`(2×2·3×3 부분행렬 행렬식 전개에 동일 비영항 없음, Theorem 1).
4. 열선택: `n=3m`으로 열을 label-triplet으로 분할, 각 triplet에서 1개씩 골라 `2×2 SNS-constraint`(girth>=6) 만족(Theorem 2).
5. girth>=8은 선택 column-label `{j0..jb-1}`이 정수조건 9개(식 3, p1~p9)를 만족해야 함(Theorem 3) - 정수연산이라 탐색 복잡도 낮음.
6. Masking: 이진 masking matrix `Z(d,b)`로 base entry 일부를 0으로 `Bmask=Z⊗B`, CPM-dispersion 후 열가중치 축소·short cycle 제거.
7. 예제1: GF(2^8), β order 255, `Λ8={2,5,7,13,20,32,54,60}` -> (4,8)-regular (2040,1025) girth-8 부호, masking으로 (3,6)-regular (2040,1020) rate-1/2, length-8 cycle 53805->765(총 169배 감소).
8. 복호: 스케일드 min-sum(MSA, scaling 0.70/0.75), 50 iteration, AWGN+BPSK.
9. 결과: masked (3,6)/(3,9)/(3,12)-regular 부호가 BER 1e-9에서 error-floor 없음, unmasked 대비 0.3dB 이득, 동길이 PEG 부호와 성능 동등(단 QC 구조 유지).
10. 한계: 부호설계·시뮬(AWGN)만, HW 미설계, gate count/throughput 부재, rate가 예제마다 0.5~0.75.
