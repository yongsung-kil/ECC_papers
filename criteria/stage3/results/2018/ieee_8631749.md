### B. 알고리즘 요약

1. 대상: Distributed Transmit Beamforming(DTB), Independent Rayleigh fading, BPSK; rate 1/3~4/5, block-length 120~960 bit의 짧은 QC-LDPC.
2. 문제: 기존 rate-adaptive는 mother code를 puncturing/shortening해 파생부호 성능이 열화 → (N,R)마다 좋은 부호를 새로 구성 필요.
3. 접근: JPL AR4JA 프로토그래프(rate 1/2, CN 3·VN 5, punctured node 1개)를 base로, 노드 append로 목표 rate 확보 후 lifting(copy-and-permute).
4. 핵심식: 목표 rate `R`에 필요한 추가 노드 `nR = floor((1-3R)/(R-1))`, 목표 length `N`의 lifting order `v = floor(N/(nR+3))`.
5. 최적화: lifting 순열을 compact Genetic Algorithm으로 탐색해 lifted Tanner graph의 **girth 최대화**; block circulant으로 제한해 QC 구조·간단 encoder 확보.
6. 확장: 정의된 rate(0.33/0.5/0.6…) 사이값은 lifted PCM truncation 후 재-GA 최적화로 생성.
7. 부가: multi-optima 목적함수라 girth 큰 비동일 PCM 다수 생성 가능 → seed/step 공유로 송수신 동일 PCM 재현(물리계층 보안 활용).
8. 복호: soft-decision BP, LLR `= (2/σ_n^2)·r·a`(a=fading gain) 근사식 사용.
9. 결과: N=480 R=1/2 부호가 uncoded 대비 큰 코딩이득, DTB 10노드에서 Convolutional/BCH/RS 대비 각 1.1/2.0/3.3dB 우수; 목표 BER 달성 필요 송신노드 28→6 감소(단 DVB-S2 64800bit LDPC보다 1.8dB 열세).
10. 한계: 시뮬만(HW/ASIC 없음), 무선 fading 특화, 짧은 부호, girth만 최적화라 error-floor·trapping set 미고려.

| GA girth 최대화 PCM 탐색 | 대응 없음 | Prime ECC엔 온라인 부호구성기 없음(부호 고정), 오프라인 부호설계 참고뿐 |

적용 가치: **낮음** — QC-LDPC·protograph라는 형식적 공통점은 있으나, rate/length 실시간 가변 짧은 무선부호 생성이 목적이라 NAND용 고정 고rate 장부호에 떼어 쓸 알고리즘·HW가 없고, girth-only GA는 error-floor 보증도 없음.
