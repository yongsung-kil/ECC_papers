> 총평: 실수 LLR 기반 RPC+AVN 부호구조 확장 기법으로 정보이론(bound) 지향, NAND용 고rate QC-LDPC HW 디코더에 직접 이식가치는 낮음.

### B. 알고리즘 요약

1. AWGN 채널·BPSK 전송, Gallager 앙상블 `(J,K)` regular LDPC 및 WiMAX/최적화 QC-LDPC를 대상으로 BP 디코딩 성능을 개선한다.
2. 문제: 일반 BP는 short cycle과 낮은 열 무게 때문에 ML 성능에 못 미치며, 기존 RPC(BP-RPC) 확장은 오히려 성능을 해칠 수 있다.
3. 모델: BEC에서의 stopping set 스펙트럼 이론을 AWGN으로 확장, 정지집합(active parity check) 기반 error probability bound를 유도한다.
4. 핵심 기법1: 각 좌표 `i`마다 쌍대부호(dual code)의 저중량 코드워드로 PCM을 확장해 얻은 체크 중, 위치 `i`에서만 겹치는 **직교(orthogonal) 부분집합** `C_i^ort`을 선택해 `H_RPC`를 구성.
5. 핵심 식: `s_i^(h) = P(y_i) + Σ_{j∈C_i^ort} Z_ij` 로 직교 체크들의 soft 추정을 합산 — 겹침이 한 위치뿐이라 length-4 cycle을 사실상 제거하고 열 무게를 키운다.
6. 핵심 기법2: AVN(추가 변수노드+행) 기법을 원본 PCM에 적용해 length-4 cycle을 보조 변수노드로 제거, BP-RPC와 결합(`AVN+RPC`).
7. 구현 디테일: 긴 코드에서는 저중량 코드워드 DB를 제한 생성하고 greedy suboptimal 탐색으로 직교 체크 집합을 근사, 중복 행 삭제.
8. 이론: Gallager 앙상블의 stopping set/distance 스펙트럼을 생성함수 `f,g`로 `O(n)` 복잡도로 계산, ML의 rank bound보다 tight한 새 union-type BP bound 유도.
9. 결과: 소수의 RPC만으로도 눈에 띄는 FER 이득(정성적), 다만 BP-RPC의 ML 수렴은 코드 길이가 커질수록 매우 느려짐.
10. 한계: HW 미설계, 시뮬/이론 중심, 직교 RPC 생성이 긴 코드에서 계산 불가, 실수 LLR 가정(soft 4bit+ 상당).
