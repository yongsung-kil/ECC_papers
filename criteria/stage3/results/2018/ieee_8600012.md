### B. 알고리즘 요약

1. 대상: AWGN 채널용 QC-LDPC(통신, Wi-Fi/WiMAX 동기), 유한체 원소의 matrix dispersion으로 순환치환행렬 배열 H 구성.
2. 문제: random-like(PEG) 부호는 부호화 복잡도 quadratic, 구조적 대수 부호는 shift-register로 linear encoding·큰 최소거리 확보가 용이 → 좋은 대수적 구성법 필요.
3. 핵심 개념: 원시다항식 `π(X)`의 근 `β`와 그 conjugate `β^(q^τ)`들이 모두 근이 됨(Theorem 2)을 이용해 `e×e` base matrix `B` over `GF(q^m)` 정의.
4. 핵심 기법: `B`의 각 원소를 유한체 원소의 2차원 matrix dispersion(`J_q` q-ary / `W_b` binary)으로 치환 → `H^(1)`(non-binary), `H^(2)`(binary) 순환행렬 배열 생성.
5. 핵심 제약: δ-multiplied constraint 1·2(Theorem 3)로 RC-constraint 충족 → length-4 cycle 없음 → Tanner graph girth ≥ 6 보장.
6. rate/거리: `w_c×w_r` subarray 선택 시 rate 하한 `(w_r−w_c)/w_r`, 최소거리 `w_c·min+1`.
7. 이진화: binary dispersion `W_b`는 원소를 `(q^m−1)` 크기 순환치환행렬로 전개 → binary QC-LDPC(예: GF(64) 기반 63×63 circulant).
8. 학습/최적화: 없음(순수 대수 구성).
9. 결과: non-binary `(889,510)` rate 0.5736(GF(128)), binary `(378,191)` rate 0.5052; BP(max iter 50) AWGN 시뮬에서 PEG·기존 대수 QC-LDPC와 waterfall 동등, 고SNR에서 우세.
10. 한계: 시뮬만(HW 미설계), 작은 유한체 → 저rate·중간길이만 구성 가능(논문 명시 한계), gate/throughput 없음.
