> 총평: length adaptation lifting으로 short-cycle과 trapping set을 확률적으로 줄여 error-floor를 낮추는 QC-LDPC 부호설계 기법으로, 부호를 새로 짜는 부담은 크나 error-floor 보증이 필요한 NAND ECC에 참고가치가 있다.

### B. 알고리즘 요약

1. 시스템: CPM 기반 QC-LDPC, exponent matrix `E(H)`(원소 `aij∈AL`), circulant size `L`, mother matrix `M(H)`; 최대길이 부호에서 짧은 길이 부호를 length adaptation(lifting)으로 유도.
2. 문제: 5G eMBB BG2 표준 lifting이 일부 길이에서 error-floor로 catastrophic 성능 저하 - short cycle(length-4)·harmful trapping set이 원인.
3. 모델: exponent chain이 사이클을 이룰 조건 `Σ(-1)^i a_i ≡ 0 mod L`(Prop.1), TS(5,3)/TS(4,4)를 pseudocodeword weight로 harmfulness 정량화.
4. 기존 기법: floor lifting `E_ij(Hk)=round(Lk/L0 · E_ij(H0))`, modulo lifting `E_ij(H0) mod Lk`.
5. 확률적 비교: length-4 chain이 lifting 후 사이클로 바뀔 확률 - floor `p_fl=5/(4(2q-1))`, modulo `p_mod=1/(2q-1))`; 기대 사이클수 `ECfl=3x/4+5y/(4(2q-1))` vs `ECmod=x+y/(2q-1)`.
6. 제안 기법: floor scale modulo lifting `E_ij(Hk)=round(Lk/L0 · ((r·E_ij(H0)) mod L0))`, scale value `r`(2q와 서로소) 도입.
7. 핵심성질(Prop.4): 조건 `r1≡r2(q+1) mod 2q` 만족하는 두 scale값에 대해 원래 비사이클 chain이 두 lifting 모두에서 동시에 사이클이 될 확률 0 - 여러 `r` 중 최소 사이클 선택 가능.
8. 이론: `Nr`개 scale값 사용 시 length-4 부재 확률 `Pfsml(Nr)=1-O(q^-Nr)`로 modulo/floor lifting보다 우수(Thm.3), 기대 사이클수 점근식(Thm.2).
9. 결과: sum-product 50 iteration, QPSK/AWGN, rate 1/3 BG2에서 표준 eMBB lifting 대비 0.1~0.3 dB coding gain, harmful trapping set 감소.
10. 한계: HW 미설계, 시뮬만, 저rate(1/3)·5G BG2 예제 중심, length-4 사이클에 집중(더 긴 사이클·girth는 미확장).
