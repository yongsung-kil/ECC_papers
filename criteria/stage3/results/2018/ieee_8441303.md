### B. 알고리즘 요약

1. 대상은 CPM(circulant permutation matrix) 기반 binary QC-LDPC 부호(exponent matrix `E(H)`, circulant 크기 `L`, mother matrix `M(H)`), regular/irregular protograph 모두 적용.
2. 문제: short cycle(girth)와 trapping set이 belief-propagation 복호의 error-floor를 유발 - 기존 PEG/Hill-Climbing/Improved PEG는 greedy라 최소 circulant 크기에서 high-girth 보장 못함.
3. 핵심 개념: exponent chain이 cycle을 이루는 조건 `Σ(-1)^i·a_i ≡ 0 mod L` (Proposition 1, 식 3), cycle 연결도 지표로 EMD/ACE 사용 (girth g에서 length<g-2 cycle은 ACE=EMD).
4. 핵심기법 - Simulated Annealing lifting: 초기 랜덤 exponent matrix를 SA로 반복 개선, shift 후보 k별 cycle 수 `Θ_k`로 확률 `P(k)=w(k)/Σw(m)`, `w(k)=exp(-Θ_k/Temp)` (Alg.1).
5. 온도 냉각: `Temp = η·Φ/Nstep^2` (Φ = 전체 cycle 수), 목적함수는 girth 최대화 / 최소 EMD / 최소 pseudo-codeword weight 중 선택.
6. 확장: SA는 non-binary, generalized, spatially-coupled(convolutional), multi-edge type QC-LDPC로도 확장 가능.
7. 구현 디테일: seed에 매우 민감 → 여러 seed 병렬 실행 권장 (Observation 1), 일부 circulant 크기는 girth 조건 해가 극히 드물어 미수렴('catastrophic length', Observation 2).
8. (학습/NN 절차 없음 - metaheuristic 최적화)
9. 결과: girth 10/12에서 Hill-Climbing·Improved PEG보다 작은 최소 circulant 크기 달성(Table I,II), 5G eMBB BG2 대비 sum-product(50 iter, AWGN, rate 1/3)에서 0.1~0.3 dB error-floor 이득.
10. 한계: HW 미설계, 시뮬만, offline 부호설계 기법이며 특정 길이 미수렴 문제.
