### B. 알고리즘 요약

1. 대상은 CCSDS 근지구용 regular QC-LDPC `(8176, 7154)`, rate ~0.875, base matrix `2×16`, submatrix `511×511`, row weight 32, column weight 4, LLR 6-bit(1 sign/3 int/2 frac) 양자화.
2. 문제: 각 submatrix가 두 cyclic-shifted identity의 중첩이라 한 VN에 두 CN 메시지가 동시 도달 → TDMP(layered)에서 memory access conflict 발생.
3. 알고리즘: NMSA(normalized min-sum, `α=0.75`, shift/add로 근사) + TDMP layered 스케줄, 식(2) `R_mn = α·∏sign·min|L_mn|`.
4. 핵심 해법: delta value `D_mn = R_new - R_old`(식4)를 도입해 posterior를 `S_new = S + Σ D_mk,n`(식5)로 갱신 → 두 CN 기여를 충돌 없이 합산.
5. HW: submatrix를 layer 통째가 아닌 클록당 1블록만 처리, 각 block row를 2회 스캔(1차: prior/new extrinsic 계산, 2차: posterior 갱신).
6. 인터커넥트: CN↔VN 연결을 barrel shifter로 구현, 한 submatrix의 두 shift값에 대응해 barrel shifter 2개, 2차 스캔은 shift값을 `511 - shift`로 역치환.
7. 메모리: VNM(16 words×511×6b), CNM(2 words×511×47b, first/second min·position·32 sign 압축 저장), TR(1 word), 그리고 Am/Bm/POSAm 레지스터로 최소값 추적.
8. RU(recover unit)는 압축 CNM/TR 정보를 511개 병렬로 extrinsic 메시지 복원, CU는 tree 구조로 first/second minimum 탐색([10] 기반).
9. 결과: TSMC 65nm 합성, 380MHz, 4.1Gb/s(8 iter), core area 2.3mm2 — [6] 172Mbps·[7] 1.1Gbps 대비 throughput 우위, fixed(6-bit)와 double precision BER 차이 미미.
10. 한계: (8176,7154) 특정 구조·2겹 circulant 전제, barrel shifter의 긴 critical path로 클록 주파수 낮음(향후 과제).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| NMSA CN min-sum 갱신 (`α·min`) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC min-sum과 동일 계열, α scaling 방식 참고 |
| TDMP layered + delta-value 충돌 회피 | `decoder.cpp` `LDPC_Decoder()` 루프, Dual-Update | Dual-Update(parity 교대) 스케줄과 취지 유사, delta 갱신 아이디어 이식 검토 |
| first/second min 압축 저장 + CU tree | `decoder.cpp` `CNU_Update_New_Mag()` | min1/min2 저장은 Prime ECC와 동일, tree 구조 CU는 HW 참고 |
| barrel shifter CN↔VN 라우팅 | `GT.cpp` `Make_GT_HW()` (edge/MUX) | Prime ECC는 GT MUX 기반, barrel shifter와 라우팅 방식 상이 |
| 6-bit LLR 양자화 | `channel.cpp` `Set_LLR_Th()`, `ecc_data.h` `PARAM_LLR` | soft LLR 양자화 대응, 단 Prime ECC는 soft-4bit+ 미지원 |
| H-matrix / 부호구조 로드 | `ecc_top.cpp` `Load_PCM()` | CCSDS 특정 구조라 직접 이식 부담 큼 |

적용 가치: **중간** — conflict-free delta-value layered 갱신과 압축 min 저장은 참고할 만하나, 2겹 circulant·z=511의 CCSDS 특정 구조와 6-bit soft LLR이 Prime ECC(단일 circulant, z=32, hard/2SD/3SD)와 정합이 필요.
