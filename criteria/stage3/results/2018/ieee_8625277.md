### B. 알고리즘 요약

1. 서베이로 대상은 binary LDPC(QC-LDPC 포함, 예: Tanner code `N=155`, IEEE 802.3an 10GBase-T), 채널은 AWGN/BSC.
2. 문제: BP는 성능 우수하나 HW 비용 큼, BF는 단순하나 성능 나쁨 - 비결정(stochastic/noise) 기법으로 이 trade-off를 개선.
3. 두 부류로 분류: (1) stochastic computing으로 BP 산술을 Bernoulli 비트열로 에뮬레이트, (2) noise-enhanced로 표준 복호기 계산에 인위적 노이즈 주입.
4. stochastic VN: Muller C-element(J/K FF)로 `pQ = pApB/(pApB+(1-pA)(1-pB))`가 BP symbol node 연산에 대응, cyclic 그래프에서는 "edge memory"로 결정론적 루프 제거.
5. noise-enhanced 핵심 알고리즘 GDBF: 에너지함수 `E_k = x_k·y_k + Σ_{j∈M_k} s_j`, `E_k<θ`면 비트 플립. NGDBF는 `+q`(가우시안 노이즈) 추가, PGDBF는 확률 `p_flip`로만 플립.
6. 핵심 원리: 노이즈가 trapping set을 교란해 BF의 error-floor를 낮춤 - GDBF 대비 순수 노이즈만으로 큰 성능 도약(Fig.3), PGDBF는 BSC에서 ML 근접(Fig.4).
7. 이론: trapping set(m개 VN, `2^m` 상태)을 homogeneous Markov chain(전이행렬 `Γ`)으로 모델링, MCMC로 error 기여 평가·설계시 파라미터 최적화.
8. HW: 대부분 완전병렬(factor graph 직접 매핑), 설계 노력은 RNG/edge memory에 집중. BF는 stochastic 대비 2~3배 작은 면적, 최저 J/bit 전력. QC-LDPC용 VNSA는 RNG 없이 노드 순환 시프트로 노이즈 샘플 공급.
9. 신소자: memristor의 sub-threshold 확률적 스위칭으로 true-random RNG를 저면적 구현(Fig.6) - RNG 오버헤드 절감.
10. 한계: 서베이라 단일 부호 확정 수치 없음, 비결정 수렴이라 프레임별 latency 보장 불가(버퍼링 필요), 이론(DE 앙상블 최적화)은 아직 미비.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| GDBF/NGDBF/PGDBF 비트플립 복호 | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | 우리 min-sum 루프와 다른 계열(대체), hard-read 저비용 경로 후보 |
| trapping set Markov chain 최적화 | `corner.cpp`(수렴실패 추적) | 수렴실패 원인 분석 관점에서 개념적 연결, 노이즈 주입은 미보유 |
| edge memory / stochastic VN 연산 | 대응 없음 | full-`tanh` BP 에뮬레이션·Bernoulli 스트림은 우리 구조에 대응 없음 |
| QC-LDPC VNSA(RNG-free 시프트) | `GT.cpp` `Make_GT_HW()` | 노드 순환 시프트 HW 기법으로 QC 구조 활용 관점 참고 |
| soft-info 처리 | `channel.cpp` `Set_LLR_Th()` | BF 계열은 주로 hard/BSC라 우리 soft-read 경로와 상충 |

적용 가치: **중간** - noise로 trapping set을 깨 error-floor를 낮추고 hard-read로 저비용 복호하는 방향은 NAND에 유의미하나, min-sum LLR 파이프라인 대체 계열이라 부분 참고에 그친다.
