### B. 알고리즘 요약

1. 대상 부호: array LDPC `H(m,q)`(q 홀수소수, `Pi` q×q circulant permutation)를 n-fold 재귀곱(superposition)한 SA-LDPC `H^n(m,q)`, 길이 `q^{2n}`, 예제 `H^2(4,7)`는 length 2401·rate 0.918·col weight 4·row weight 49.
2. 문제: BEC error-floor는 최소 stopping set 크기(stopping distance `S(·)`)에 좌우되며, 이를 크게 해야 floor가 낮아짐.
3. 모델: `H(m,q)`를 transversal design `TD(m,q)` incidence matrix로 대응, RC-constraint(두 열/행 공유 1개 이하)로 4-cycle 없음(girth ≥6).
4. 핵심 기법: support matrix(비영 원소 행번호 행렬)와 TD-extending 재귀 알고리즘(Algorithm 1)으로 `SM(H^n)`을 `SM(H^{n-1})`+블록에서 구성.
5. 식 의미: stopping set을 pattern `(x1,...,xk;t)`(각 `xi≥2`)로 특성화, girth·pattern 모순으로 특정 크기 stopping set 부재를 귀납 증명.
6. 확장: girth 부등식 `min{g1,g_{n-1},8} ≤ gn ≤ min{g1,g_{n-1}}`로 `H^n` girth = `H` girth 유지.
7. 주요 결과(증명): `S(H^n(3,q))=6` (q≥3), `S(H^n(4,q))=8` (4<q≤7), `S(H^n(4,q))=10` (q≥11), 모든 n에서 `S(H^n)=S(H)`.
8. 최적화 절차 없음(순수 조합론 induction 증명).
9. 시뮬: BEC, BP decoding으로 `H^2(4,7)`이 동급 TD-LDPC/random/lattice LDPC 대비 p=0.045에서 BER 1e-8 (타 부호 1e-5/1e-4) — floor 우수.
10. 한계: BEC·stopping set 이론 전용, AWGN/soft 채널 미검증, HW·throughput·복잡도 수치 없음, m>4는 open problem.
