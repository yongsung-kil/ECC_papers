### B. 알고리즘 요약

1. 대상: rate 1/2 등 random irregular/bi-regular LDPC, random lift(protograph 기반), random cyclic-lift(QC-LDPC) 세 앙상블의 Tanner graph.
2. 문제: LDPC 성능(특히 error-floor)을 지배하는 short cycle 개수 분포를 경험적으로만 알던 것을 해석적으로 규명. cycle 세기 자체가 NP-hard.
3. 모델: cycle multiplicity `N_c`를 configuration/matching 확률공간에서 Poisson 근사(Theorem 2 joint Poisson) + adjacency matrix 거듭제곱으로 TBC walk 계수.
4. 핵심기법 1단(Theorem 1,3): random 이분그래프에서 `N_4,...,N_k`가 독립 Poisson으로 수렴, 기댓값 `E(N_c) ≈ [(2/η)Σd_i² ...]^{c/2}/c` (degree distribution·cycle 길이에만 의존, block length 무관).
5. 핵심식: bi-regular는 `E(N_c) ∼ [(d_u−1)(d_w−1)]^{c/2}/c` (Corollary 1) — 블록길이와 무관한 상수.
6. 핵심기법 2단(Section IV,V): random lift는 `E(N_c)=T(G,c)` (TBC walk 수), cyclic-lift(QC)는 exponent matrix `P` 무작위에서 `T`를 T1(prime ZP)/T2/T3로 분할.
7. 부수 결과(Corollary 3): QC cyclic-lift는 base graph에 prime ZP TBC walk가 있으면(inevitable cycle) `E(N_c)=Θ(N)`로 선형 증가, 없으면 `Θ(1)`. girth g면 `c<3g`에서 inevitable cycle 없음(Remark 4).
8. 최적화: Maclaurin 부등식 + Specht 비율 `S(h)`로 upper/lower bound 유도, 분산은 `Var[N_c] ≤ O(N)` (Theorem 8).
9. 결과: 수치 검증에서 block length 200~50000에서 이론 기댓값이 실제 cycle 개수와 근접(Table I~V), (a,a) LETS multiplicity ≈ `10^a/(2a)` 추정(du=3,dw=6 예).
10. 한계: 순수 이론(증명·수치 검증), 부호구성 알고리즘·복호·BER 시뮬·HW 전무.
