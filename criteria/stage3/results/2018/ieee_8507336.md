### B. 알고리즘 요약

1. 시스템: NAND flash SLC용 binary QC-LDPC, 사례 부호는 `(2048,1792)` rate `0.875`, submatrix `P=64`, base matrix `I=4`행·`J=32`열.
2. 문제: dual-diagonal 구조는 선형 인코딩엔 유리하나 weight-2 열이 불가피, hard/저정밀 soft 복호에서 성능 저하.
3. 모델: SLC 문턱전압을 2개 Gaussian 혼합으로 근사, 1-read(hard) / 2~3-read(soft) 입력을 LDPC 디코더에 전달.
4. 핵심 기법: dual-diag 제약 없이 `Φ = ET⁻¹B + D = I`가 되도록 패리티부 `H2`를 컴퓨터 탐색 -> Richardson-Urbanke 선형 인코딩 유지.
5. 핵심 식: base matrix에서 `Φb = EbTb⁻¹Bb + Db = 1`이면 대응 `Φ`가 GF(2)에서 가역 -> 탐색 가이드라인으로 활용.
6. 확장: `Φ`가 CPM `Px`이면 `[D E]` 행을 순환시켜 `Φ=I`로 변환.
7. 구현 디테일: 정보부 `H1`은 PEG로 base 그래프 구성 후 shift 반복 최적화로 길이 4·6 단주기 제거, weight-4 열 위주.
8. 최적화: `Tmax=100000` 탐색으로 `Φ=CPM` 해 856개 확보 후 최적 사이클 분포 선택.
9. 결과: weight-4 부호가 weight-3보다 hard/soft-3read 모두 우수, dual-diag 대비 error-floor 영역에서 약간 우수. rate 적응(행·열 확장 + shortening)으로 `(2112,1792)` 저rate 부호 파생.
10. 한계: HW 미설계, Matlab BP 시뮬만(최대 50 iteration), dual-diag 대비 이득이 slight.
