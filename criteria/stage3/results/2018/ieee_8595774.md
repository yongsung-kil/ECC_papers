### B. 알고리즘 요약

1. 대상 시스템은 BPSK·AWGN 채널, SPA(sum-product) 50 iteration 복호를 가정한 일반 통신용 QC-LDPC.
2. 유한체 GF(q)의 임의 두 부분집합 `S1,S2`로 RC-constraint 만족 base matrix `B(δ)=[α^{ik}−α^{jl}]`를 생성.
3. 문제: 유한 부호길이에서 Tanner graph의 short cycle이 SPA 성능을 저하시키므로 girth `g` 확대·short cycle 수 감소가 필요.
4. 핵심 기법: base matrix `B`에 대해 girth `g`의 cycle 분포 행렬 `D`를 계산, `D`의 최대값 좌표 원소를 masking 후보로 선정.
5. 해당 원소가 masking 가능하면 마스킹, 불가하면 차순위 좌표 선택, 실패 누적 시 `g=g+2`로 완화하며 desired degree distribution 만족까지 반복.
6. masking matrix `Z`(0/1)를 `Hmask = Z⊙B` 형태로 적용하여 `Z`의 행·열 weight 분포를 그대로 갖는 irregular LDPC 획득.
7. 알고리즘 종료 조건: `Z`의 행 weight 분포 `ρp(x)`가 목표 `ρ(x)`와 일치하고 대응 `H`의 rank가 `J·P`이면 `Z` 출력.
8. 최종적으로 masked `B`를 array dispersion(circulant permutation)으로 전개해 length `Δ(q−1)` binary QC-LDPC 부호 생성.
9. 결과: (576,288)은 code [11] 대비 WER 10^-7에서 0.15 dB, (576,432)는 WER 2×10^-7에서 최대 0.3 dB 이득, WER 10^-7 위로 error floor 미발생.
10. 한계: HW 미설계, 시뮬레이션만, 광통신 배경의 rate 0.5~0.75급 부호로 short cycle 통계(Table I) 외 정량 복잡도 없음.
