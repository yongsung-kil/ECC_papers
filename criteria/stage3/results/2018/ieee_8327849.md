### B. 알고리즘 요약

1. 대상: 시간불변 SC-LDPC-CC, polynomial parity-check matrix `H̄(x)`(F2[x]), 비대칭 rate `R=(a−c)/a`, 실험코드 rate 2/3~3/4.
2. 문제: column weight `wc>2`이면 codeword와 cycle이 1:1 대응 안 됨 → 저중량 codeword 원인 규명 난해(trapping set 지식 필요).
3. 모델: cycle을 vertical separation `dl=(q2−q1)c`(Eq.7~8)로 표현, cycle 길이 λ는 Eq.(9) 차분합=0으로 정의.
4. 핵심기법: 임의 codeword `T̄(x)`를 column weight 2인 component code codeword들의 중첩으로 분해 (Theorem 1, Eq.11) `T̄(x)=Σ t̄j(x)·2W̄⁻¹/s`.
5. 의미: cycle superposition degree `s`가 상수임을 증명 → codeword↔cycle 조합 관계를 대수적으로 확립.
6. 확장: monomial/two-polynomial/diagonal code 앙상블별 unavoidable codeword·cycle 길이(12/16 등) 유형화, 고rate로 iterative 확장.
7. 트릭: `(1+x)` 곱으로 홀수중량 열을 짝수중량 cycle-coverable 형태로 변환 (Lemma 2,3).
8. 최적화 절차: unavoidable codeword full-weight화 → 저중량 codeword 유발 짧은 cycle 탐색·제거 → free distance를 bound(Eq.5)까지 상승(5단계 heuristic).
9. 결과: packing-based SC-LDPC-CC 2개(R=3/4, R=2/3)에서 length-6 cycle 10~20% 감소, `dfree` 8→12, BPSK/AWGN, BP 100 iter에서 고SNR(error-floor) 영역 BER 개선.
10. 한계: HW 미설계, 시뮬만, convolutional(시간불변) 부호 전용 이론.
