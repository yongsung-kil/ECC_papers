### B. 알고리즘 요약

1. 시스템: STT-MRAM(비휘발성 emerging memory), 고속 read 위해 고정밀 ADC 불가 -> **1-bit(q=1) 양자화** 임계치 설계가 목표. 데모 부호는 polar `(128,110)`, R≈0.859.
2. 문제: 시뮬 기반 양자화기 설계는 느리고 저에러영역 재현 불가; 기존 MMSE(Lloyd-Max)는 STT-MRAM 채널에 준최적.
3. 채널 모델: cascaded = BAC(write+read-disturb error, 비대칭) + GMC(read decision error, 저항 Gaussian mixture). `µ0,µ1,σ0,σ1`; write error `P1=2e-4`, `σ0/µ0` 스윕.
4. 양자화 채널: q-bit 양자화가 출력을 `n=2^q` 구간으로 매핑, 전이확률 `W(y|x)`를 (2)(3)으로 유도; 1-bit는 임계치 `a1` 하나에만 의존.
5. 기법1 (capacity): `Cq=H(Y)-H(Y|X)` 최대화, `dCq/da1=0`을 bisection으로 풀어 최적 `a1*`.
6. 기법2 (cutoff rate): `R0`(ML random-coding union bound) 최대화, 유한길이 부호에 achievable -> 동일 bisection.
7. 기법3 (PPV bound): 유한블록 최대 coding rate `R≈Cq - sqrt(Vq/N) Q^-1(ε)`, BLEP `PB`를 최소화하는 `a1`을 수치미분으로 최적화 (단블록 `N` 몇백에 최적, 본 논문 첫 시도).
8. 결과: 세 정보이론 기준 모두 Lloyd-Max 대비 `σ0/µ0` 기준 ~2% 이득, FER 시뮬 최적 임계치에 근접; PPV가 미세 우위.
9. 복호 데모: AD-SCL polar decoder(list 256, CRC 4bit) - LDPC 아님. 1-bit는 full-soft 대비 성능 gap 존재, 3-bit로 확장 시 gap 해소 언급.
10. 한계: STT-MRAM 전용 채널·polar code 검증, LDPC 복호 미포함, HW 미설계, 시뮬만.
