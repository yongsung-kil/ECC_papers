### B. 알고리즘 요약

1. 저장장치용 RS+LDPC 디코더 아키텍처 논문 — LDPC 파트만 이식 관련(RS 파트는 iBM KES 2t-cycle 저지연 파이프라인으로 별개 주제).
2. LDPC 대상 부호: array-code 계열 regular LDPC, 프로토타입은 `(384,4096)`(3 block row × 32 block col, 병렬도 `p=128`), FPGA `I=5`.
3. 문제: 거의 모든 디코더가 균일 양자화를 쓰는데, 비선형 min-sum 동역학상 비균일 양자화(NUQ)로 같은 비트에서 추가 이득 가능.
4. 핵심 기법: 층화(layered) min-sum(`P=L`, `Q=P-Rold`, `Rnew=f(Q)`, `P=Q+Rnew`)에 NUQ 적용 — 비균일 양자화 레벨 `{±Lj}`를 min-sum 가·감산 LUT로 직접 구현(2's-comp 변환 유닛 제거).
5. 양자화 최적화 식: 부동소수 vs 양자화 디코더 출력 LLR의 평균제곱왜곡 `Δ=Σ(L(f)-L(q))^2/N`을 비용함수로, gradient-descent `(17)`로 레벨 갱신.
6. 최적 레벨 산출은 SW상 LMS 엔진(gradient 근사식 `(18)`)으로 수행, 결과 레벨을 HW에 고정 LUT로 탑재.
7. 부수: 채널 aging(SNR 저하) 대비 여러 bin-size LUT를 두고 code failure rate 모니터로 MUX 선택하는 적응형 디코더 구조.
8. CNU는 N1/N2(min1/min2) finder + partial/final state + sign XOR(표준 min-sum), 배럴 시프터로 QC shift 처리.
9. 결과: 층화 NUQ(4bit)가 균일(5bit) 대비 동일 성능을 1비트 적게 달성 → BRAM 20% 절감, ~0.1dB SNR 개선·error-floor 완화; 층화가 비층화 대비 ~0.3dB↑·수렴 iteration 절반. FPGA throughput 층화 2.128Gbps@150MHz, 비층화 6.64Gbps.
10. 한계: NUQ 가감산 LUT 자체가 area-extensive, 부호 rate 미명시, ISI+AWGN 근사 가정, soft 다비트(4~5bit) 채널 LLR 전제.
