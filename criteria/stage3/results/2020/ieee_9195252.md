### B. 알고리즘 요약

1. AWGN/실제 무선 채널 대상 BICM(Bit-Interleaved Coded Modulation) 시스템, 표준 802.11n LDPC(길이 1944bit)를 채널코드로 사용.
2. 기존 오토인코더 학습은 symbol-wise cross-entropy 기반이라 BMD(bit-metric decoding) 수신단과 정합이 어려운 문제를 해결하려 함.
3. 채널·송수신단을 하나의 신경망(오토인코더)으로 간주하는 end-to-end 학습 프레임워크를 전제.
4. 핵심 기법: bit-wise mutual information(BMI)을 학습 목표로 사용해 constellation shaping·labeling과 BMD LLR 출력을 동시 최적화.
5. BMI가 BMD 하에서 달성 가능한 rate임을 활용(symbol-wise CE는 부적절하다는 논리, [5] 근거).
6. 확장 기법: 완전 미분가능한 신경망 기반 반복 demapping-decoding(IDD) 구조를 도입해 AWGN에서 성능 이득 확보.
7. 부수 트릭: 학습된 통신시스템 "위에" LDPC 부호를 설계해 추가 성능 개선(구체 구성법은 본문 미기재).
8. 학습 절차: end-to-end 신경망 학습(오토인코더), 실제 채널 기반 model-free 학습 알고리즘([7] 활용)으로 SDR 상에서 훈련.
9. 결과: modulation order m=6,8에서 802.11n 기반 QAM 대비 제안 오토인코더 및 최적화 코드가 BER 개선(Fig.1, 구체 수치는 그래프 형태로만 제시).
10. 한계: Extended Abstract(2쪽) 형식으로 알고리즘 수식·HW 구현 세부사항 부재, decoder/encoder 자체의 신규 알고리즘이 아닌 물리계층 전체(변조+코딩) 공동학습이 핵심이라 순수 LDPC ECC 이식 관점과는 거리가 멂.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 신경망 기반 오토인코더 물리계층 | 대응 없음 | NN/딥러닝 전면 대체 방식으로 Prime ECC의 binary QC-LDPC 구조와 무관 |
| BMI 기반 constellation shaping/labeling | 대응 없음 | 변조 계층 기법, LDPC 디코더/인코더 대상 아님 |
| 신경망 반복 demapping-decoding(IDD) | 대응 없음 | 표준 min-sum 계열 디코더가 아닌 NN 기반 구조 |
| 802.11n LDPC(길이 1944) 사용 | `ecc_top.cpp` `Load_PCM()` | 채널코드로만 차용, 부호설계 기여 없음(대응 약함) |

적용 가치: 낮음 | 신경망 기반 물리계층 공동학습이 핵심이며 LDPC는 부가 요소로만 등장, Prime ECC의 binary QC-LDPC ECC 엔진(부호설계→디코더→HW)과 판정 기준이 근본적으로 다름.
