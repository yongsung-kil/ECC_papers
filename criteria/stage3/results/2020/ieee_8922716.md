### B. 알고리즘 요약

1. 대상은 QC-LDPC 코드의 parity part에 block dual-diagonal(BDD) 구조를 사용하는 저rate(rate 낮음, J/L 큼) 부호. 기존 BDD는 parity part가 전부 weight-1 circulant(순열행렬)로 구성됨.
2. 문제: 저rate에서는 BDD 구조상 degree-2 variable node 비율 하한이 `(1-R)(1-1/J)`로 커지고, 이는 EMD가 작아 error floor를 유발.
3. 제안: parity part 대각선의 일부 circulant를 weight-2(double-weight, `x^0 + x^{s_i}`)로 대체하는 "generalized BDD" 구조. `n2`(double-weight circulant 개수)를 조절해 degree-2 비율을 `(1-R)(1-n2/J)`로 낮춤 (degree-3 노드 증가).
4. 효율적 인코딩 조건: parity part 하삼각 인코딩(`φ = ET^-1B + D`)에서 `φ(x)`가 circulant permutation이 되도록 shift value 집합 `S`를 결정 (Theorem 1, 2: `z=2^k` 조건 하 `S={2^0,2^0,2^1,...}` 형태).
5. Girth 최대화: 논문이 정의한 protograph의 TNC(tailless non-reversing closed) walk와 dumbbell graph 이론으로 shift value 배치를 결정하는 PPC(Parity-Part Construction) 알고리즘(Algorithm 1) 제시, target girth `g`를 내려가며 탐색.
6. 메시지 part는 PGGM(Proposed Greedy Girth-Maximizing) 알고리즘 또는 simplified EMPEG 알고리즘으로 낮은 차수 열부터 순차 구성.
7. HW 친화 요소는 없음(순수 부호설계); QC-LDPC의 표준 circulant/shift-register 구조는 유지.
8. 최적화 절차: girth maximization 탐색(Algorithm 1, 2), 무작위/EMPEG 등 3가지 구성 방식 비교(PGGM/PEMPEG/PRAND vs 기존 CGGM/CEMPEG/CRAND).
9. 결과: R=1/3, N=2025 및 R=1/4, N=1720에서 제안 BDD가 기존 BDD 대비 error floor 영역 BER/FER 개선(waterfall 영역은 소폭 열화, threshold도 소폭 낮음). typical minimum distance ratio·최소 해밍거리 상한도 더 큼. IEEE 802.16e, 5G LDPC(base graph 1/2) 대비도 error floor에서 우위.
10. 한계: 순수 이론/시뮬레이션(AWGN, sum-product, 최대 100 iteration)만 수행, HW 미설계, 대상 rate가 1/5~1/2 범위(주로 저rate)로 NAND ECC의 고rate(~0.9) 영역과는 거리가 있음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Generalized BDD parity 구조(double-weight circulant) | `ecc_top.cpp` `Load_PCM()` | H-matrix 구조 자체를 바꾸는 것이라 현재 고정 QC-LDPC(PCM_b) 교체가 필요, 재검증 부담 큼 |
| 효율적 인코딩(φ 역행렬, shift value 결정) | `encoder.cpp` `Generate_PCM_encoding()`, `Column_Reordering_Dual_Update()` | 현재 dual-diagonal 인코딩과 구조적으로 유사하나 double-weight 확장은 미대응 |
| PPC/PGGM girth 최대화 알고리즘 | 대응 없음 | 오프라인 부호설계 툴 성격, 런타임 디코더/인코더 코드와 무관 |
| error-floor 개선(정정능력) | 대응 없음 (H-matrix 자체 특성) | decoder.cpp 알고리즘이 아닌 부호구조에서 기인하므로 디코더 모듈과 직접 연결 없음 |

> 적용 가치: 낮음 — 저rate 부호설계 이론 논문으로 Prime ECC의 고rate(~0.9) 고정 QC-LDPC 구조와 rate 대역이 달라 직접 이식은 어렵고, H-matrix 전면 재설계가 필요해 재검증 부담이 큼.
