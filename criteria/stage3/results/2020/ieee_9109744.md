### B. 알고리즘 요약

1. Polar code (N,K) 및 BI-AWGN 채널 대상, code rate 1/2 예시 (16,8) 기준 시스템 모델.
2. 기존 pure-NND(PNND)는 노이즈 낀 수신 심볼을 바로 신경망으로 디코딩해 BER 성능이 SC 대비 부족.
3. 채널 모델: `y1N = s1N + n1N`, `n1N ~ N(0, σ²I_N)` 가산 백색 가우시안 잡음.
4. 핵심 기법 1단: residual learning denoiser(RLD) - `ŝ1N = H(y1N) = y1N + F(y1N)`로 residual mapping을 학습해 수신 심볼 SNR 개선.
5. 손실함수 `L_denoise = (1/N)||ŝ1N - s1N||²2` (MSE)로 denoiser를 원 심볼에 근접시킴 (이미지 denoising의 1D 유사 적용).
6. 핵심 기법 2단: RLD와 NND를 직렬 연결한 RNND, multi-task loss `L_MTL = L_denoise + λ·L_decode`로 두 모듈을 공동 최적화(Joint-RLD).
7. RLD/NND는 MLP·CNN·RNN(LSTM) 3종으로 각각 구현, dimension matching 위해 출력층은 N(또는 K) 노드 fully-connected, activation 없음.
8. 학습: Adam optimizer, batch 64, lr 0.001, training SNR=0dB, 최대 2^16 epoch, λ=1 고정(하이퍼파라미터 탐색 없음).
9. 결과: RNND(MLP-MLP)가 SC 대비 근접 성능이면서 연산시간 100배 이상 단축, BER 10^-3에서 PNND 대비 0.1~0.3dB 이득; LDPC(20,8) 확장 실험에서도 RNND가 PNND보다 우세하나 BP보다는 크게 열세.
10. 한계: 모든 실험 시뮬레이션뿐(HW 미설계), 극소형 코드(N≤64)에서만 검증, LDPC 실험은 스케일 작고 BP 대비 성능 열위로 실용성 낮음.

| LDPC(20,8) 소형 코드 확장 실험 | 대응 없음 | non-QC, 극소형 코드로 `ecc_top.cpp` `Load_PCM()` 대상 부호 구조와 무관 |
| SC/BP 비교군 | 대응 없음 | 벤치마크일 뿐 이식 대상 아님 |
