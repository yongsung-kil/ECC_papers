### B. 알고리즘 요약

1. BPSK/AWGN 채널에서 quantized LDPC 디코더(SPA/MSA)의 Tanner graph 내 문제 서브그래프인 absorbing set `G(A)`가 error-floor의 지배 원인임을 전제.
2. 문제: 특정 코드에 종속적인 기존 error-floor 예측법(몬테카를로, importance sampling 등)은 코드마다 재계산이 필요하고 저FER 영역에서 시간이 오래 걸림.
3. 가정: absorbing set 외부에서 들어오는 모든 edge를 단일 보조 변수노드 `a_j`의 LLR `u_j`로 축약한 "absorbing set decoder graph" `D(A)`를 정의(코드 전체 구조 불필요).
4. 핵심 기법: 가능한 모든 채널입력 `s=x_i`와 외부입력 `U=W_k` 조합에 대해 디코딩 성공/실패를 나타내는 decodability array를 정의하고, 모든 행(`W_k`)에서 실패(`1`)인 열만 모아 `λ(A) = Σ_{x_i∈Ψ(W)} Pr(s=x_i)`로 code-independent lower bound를 유도(식 17).
5. 핵심 정리(Theorem 1): 임의의 코드가 `G(A)`를 포함하면 `Pr(E(A)) ≥ λ(A)`이며, 지배적 absorbing set의 multiplicity `N`을 알면 `FER ≥ N·λ(A)`로 근사(식 33).
6. 복잡도 축소를 위해 전체 `t^{κp}`행 대신 대표행 몇 개만 검사하는 근사: Row Set I(`U=W_max`, 최대 양자화 레벨로 즉시 수렴 가정), Row Set II(`U=W_inc`, LLR이 서서히 증가하는 경우), Row Set III(둘의 교집합)을 제안.
7. 부수 트릭: 코드rate가 다르면 `λ̂(A)`는 `Eb/N0` 축을 `10log(R)`만큼 이동시키는 것과 같다는 성질(식 43)을 이용해 rate=1에서 한 번만 계산.
8. 별도 학습 절차 없음(Algorithm 1: 각 `x_i`에 대해 `W_max`, 실패 시 `W_inc`로 absorbing set decoder를 순차 실행해 `Ψ̂(W)` 산출).
9. 결과: (3,61) array code(n=3721, R=0.9514)에서 (4,2)/(6,0) absorbing set 기반 예측이 시뮬 error-floor와 근접 일치, code-dependent 방법 대비 계산복잡도 약 0.0016배(a/n 비율)로 대폭 절감.
10. 한계: 새로운 디코딩/코드설계 기법이 아닌 순수 사후 분석 도구이며, 지배 absorbing set과 그 multiplicity를 사전에 알아야 하고 waterfall 영역에는 적용 불가(error-floor 전용).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| quantized SPA/MSA 양자화 파라미터별 error-floor 예측 | `decoder.cpp` `VN_Cal_HD()` 등 (HD/2SD/3SD), `ecc_data.h` `PARAM_LLR` | 양자화 레벨별 error-floor를 사전 추정해 LLR 테이블/양자화 비트수 설계에 참고 가능 |
| absorbing/trapping set 식별 및 multiplicity 분석 | `ecc_top.cpp` `Load_PCM()` | 로드된 QC-LDPC H-matrix에서 지배 absorbing set을 찾아 부호구조 검증에 활용 가능하나 별도 그래프 분석 도구 필요 |
| min-sum(MSA) 기반 lower bound | `decoder.cpp` `CNU_Update_New_Mag()` | Prime ECC의 min-sum CN 연산과 동일 알고리즘 계열이라 적용 근거는 있으나 코드 자체 분석 도구는 대응 없음 |
| 코드-rate 독립적 shift 성질(식 43) | 대응 없음 | 다양한 H-matrix(matrix_sel)의 error-floor를 빠르게 비교하는 데 활용 가능하나 Prime ECC 코드 내 구현은 없음 |

> 적용 가치: 중간 — 새 디코더/부호 설계 기법은 아니지만, Prime ECC의 QC-LDPC 부호(H-matrix) 선정·검증 단계에서 quantized min-sum의 error-floor를 시뮬레이션 없이 빠르게 사전 추정하는 분석 도구로 활용 가치가 있음.
