### B. 알고리즘 요약 (10줄 내외)

1. 시스템은 coherent 광통신(DP-QAM, 34 GBd, N-span dispersion-managed NZDSF 링크)에서 fiber Kerr nonlinearity로 왜곡된 신호를 등화하는 BICM-ID(iterative demodulation) 구조.
2. 문제: 기존 모델기반 비선형 등화기(MLSE, TEQ, Volterra, DBP)는 계산복잡도가 높고 모델 불일치에 취약하며, 기존 DNN 등화기 다수는 FEC 피드백을 활용하지 못함.
3. 채널모델: fiber 비선형 왜곡을 nonlinear Schrödinger equation 대신 데이터 기반 DNN이 블랙박스로 근사, 고차 QAM에 대해 multi-label BCE loss로 LLR을 직접 출력.
4. 핵심 기법: nested residual network(ResNet) 구조의 DNN-TEQ가 FEC 디코더로부터의 a priori(APR) soft LLR을 입력으로 받아 extrinsic(EXT) LLR을 출력, skip connection으로 EXT=APP-APR 관계를 구현.
5. 학습 시 APR 값은 실제 FEC decoder 대신 `N((-1)^b σ²/2, σ²)`(σ=ten Brink J-inverse function) 가우시안으로 합성 생성해 EXIT 특성을 직접 제어.
6. 확장: EXIT chart 분석으로 DNN-TEQ와 LDPC decoder(VND-CND) 간 결합 EXIT tunnel을 열어 디코더 수렴을 가속(적은 iteration으로 Iout=1 도달).
7. 부수 기법: DNN은 4개 은닉층(batch norm + ReLU + FC + skip + 50% dropout, 1000 노드), Adam 옵티마이저, APP/EXT sigmoid cross-entropy의 max-pooling(worst-case) 손실로 학습.
8. LDPC 부호설계: DNN-TEQ의 EXIT curve를 3차함수로 모델링해 삼중-degree check-concentrated distribution을 탐색, DVB-S2 표준 대비 재최적화된 degree distribution 산출(예: rate 5/6에서 `λ(x)=0.725x²+0.25x⁹+0.225x³⁰`).
9. 결과: DP-16QAM/64QAM 환경에서 DNN-TEQ가 LE 대비 achievable rate 0.85 b/s/Hz 개선(DNN 대비 0.61 b/s/Hz, LDPC 재설계로 추가 0.12~0.24 b/s/Hz), post-LDPC BER이 BCH 문턱(5×10⁻⁵) 이하로 개선.
10. 한계: dispersion-unmanaged 장거리 링크처럼 채널 메모리가 긴 경우(3-tap DNN으로 처리 불가) 성능 개선 제한적이며, 순수 시뮬레이션 결과이고 HW 구현·합성은 다루지 않음.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| DNN-TEQ (fiber nonlinearity 등화) | 대응 없음 | 광통신 채널 전용 등화기로 NAND read 채널 모델과 무관, NN 전면 대체형 |
| irregular LDPC degree distribution 재설계(EXIT 기반) | `ecc_top.cpp` `Load_PCM()` | 부호설계 레이어에 닿지만 DVB-S2 표준 LDPC(광통신용, non-QC 명시 없음)이며 Prime ECC의 고정 QC-LDPC(z=32, N_b=129) 구조와 직접 매핑 불가, 이식 난이도 높음 |
| DNN-TEQ ↔ LDPC decoder EXIT tunnel(iteration 수렴 가속) | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | iteration 감소 개념은 참고 가능하나 가속 메커니즘이 DNN 등화기와의 turbo 결합에 의존, 순수 디코더 내부 개선이 아님 |
| soft LLR 피드백 구조(APR/EXT) | `channel.cpp` LLR 관련 파라미터 | Prime ECC는 soft-2~3bit까지 지원하나 본 논문은 아날로그 등화기 LLR 출력이라 NAND soft-read 양자화 체계와 다름 |
```

적용 가치: `낮음` — 핵심 기여인 DNN-TEQ는 광섬유 비선형 채널 전용 등화기로 NAND 채널과 무관하며, LDPC degree distribution 재설계도 DVB-S2 표준 부호를 광통신 EXIT 특성에 맞춘 것이라 Prime ECC의 고정 QC-LDPC 구조에 직접 이식하기 어려움. iteration 수렴 가속이라는 결과 지표만 개념적으로 참고 가능.
