### B. 알고리즘 요약 (10줄 내외)

1. QC-LDPC 부호의 parity-check matrix는 `P×P` circulant들의 `M×N` 배열; 모든 circulant의 최대 weight가 2인 경우를 Type-II 코드로 정의(weight-1=CPM, weight-0=ZM인 경우는 Type-I/classical).
2. 풀려는 문제: Type-II QC-LDPC는 Type-I 대비 더 큰 distance 상한을 갖는 장점이 있으나, 기존 대수적 구성법(state-of-the-art, [10])은 이론 하한(`circulant size ≥ 4L`, Lemma 1)보다 약 2배 큰 circulant 크기를 요구.
3. Tanner's method(`E_tan(i,r) = (α^i·β^r)_P`, α order J, β order L인 소수체 FP 원소)를 확장하여, 지수행렬을 상/하 절반(E0, E1)으로 분할하는 새 대수적 구성 `E_k(i,r) = (α^{i+(k·J)/2}·β^r)_P` (k∈{0,1}) 제안 (식, IV절 A).
4. 핵심 성질: Lemma 6에 의해 `E1(i,r) = -E0(i,r) (mod P)`이므로 자동으로 weight-2 circulant(Type-II) 구조가 되며, girth 6 이상을 보장하려면 L이 홀수여야 함(Remark 2).
5. Theorem 1(J=4 케이스): `P = 4mL+1`(m≥1)가 소수이면 G6+ Type-II (4, 2L) QC-LDPC 코드를 얻으며, m=1일 때 이론 하한보다 겨우 1 큰 거의 최소 circulant 크기를 최초로 대수적 구성으로 달성(기존 서치 기반 방법 [9][10]만 하한 도달 가능했음).
6. J≥6 확장은 프로그램 기반 4-cycle 전수 탐색(Table 1의 16가지 패턴)으로 유효한 (J,L,P) 삼중항을 찾아 girth-6 코드 다수 확보(예: (6,27), (10,25) 등).
7. 부수 기법: gCRT(generalized Chinese Remainder Theorem)으로 Type-K(K≥1) 코드에 CRT를 일반화, 두 개의 짧은 component 코드(하나는 제안 최소-circulant 코드, 다른 하나는 무작위 생성)를 결합해 girth 저하 없이(`Gg+`) circulant 크기를 유연하게 확장(Theorem 2).
8. 학습/최적화 절차 없음(순수 대수적 구성 + 컴퓨터 탐색으로 4-cycle 없는 (J,L,P) 조합 확인).
9. 결과: 예제에서 BPSK/AWGN 채널, sum-product algorithm(SPA) 반복복호로 BER/BLER 시뮬레이션; 제안 gCRT+신규 짧은 코드 조합이 기존 CDS/무작위 구성 코드와 거의 동일한 복호성능을 보이면서 더 유연한 circulant 크기(길이 1885, 1624, 3248 등)를 가짐; Type-I 대응 코드(TAS 방법) 대비 circulant 크기가 커서(101 vs 52) 2-stage 인코더 복잡도가 더 낮음.
10. 한계: HW 미설계, 순수 시뮬레이션(이론+SPA)뿐; 부호rate 명시적 최적화나 NAND 채널모델 없음; iteration 수·복잡도·latency 수치 미보고.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Type-II(weight-2 circulant) QC-LDPC 대수적 구성 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 base matrix+shift 기반 QC-LDPC(`PCM_b`)를 이미 고정 사용 중이나 weight-1(CPM) 위주 구조로 Type-II와 상이 - H-matrix 전면 교체 필요 |
| gCRT 기반 circulant 크기 확장 | 대응 없음 | 부호 길이 유연화 기법으로 코드 구조 결정 단계에만 관련, 디코더/인코더 모듈과 직접 연결 없음 |
| girth-6 보장, 4-cycle 회피 이론 | `ecc_top.cpp` `Load_PCM()` | H-matrix 설계 시 girth 특성은 참고 가능하나 현재 코드가 특정 QC-LDPC로 고정되어 재설계 부담 큼 |

적용 가치: **낮음** — Type-II(weight-2 circulant)라는 근본적으로 다른 QC-LDPC 부호족을 대수적으로 구성하는 이론 논문으로, Prime ECC의 고정된 Type-I 기반 base matrix 구조(§4 "부호설계" 항목은 이식 난이도 "높음"으로 이미 분류됨)와 직접 대응되지 않으며 정정성능 개선도 기존 방법과 "거의 동일"한 수준에 그침.
