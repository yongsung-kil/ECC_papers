### B. 알고리즘 요약

1. 대상은 자기기록(MR)·Flash·광기록·USB/PCIe 등에 쓰이는 고정길이 이진 line code(constrained code)로, LDPC 자체가 아니라 채널 전처리(ISI/ICI 완화, DC-free 밸런싱) 목적의 별도 코드 계층.
2. 기존 RLL/Tx-constrained 코드는 유한상태머신(FSM) 기반이라 rate가 낮거나, 사전식 인덱싱 기반 선행연구([23]~[25])는 코드 길이 m>250에서만 이득이 나 복잡도·오류전파 부담이 큼.
3. 채널 가정: NRZ 신호에서 `01^y0`, `10^y1` (1≤y≤x) 패턴 금지(Tx-constrained), 자기기록에서는 PR(EPR4/E2PR4) 채널, Flash에서는 인접 셀 간섭(ICI) 패턴 `101`/`010`.
4. 핵심 기법: LOCO 코드 `C_{m,x}`를 사전식 순서로 정의하고, 재귀식 `N(m,x)=N(m-1,x)+N(m-x-1,x)`로 카디널리티를 계산해 인덱스 `g(c)`와 코드워드 `c` 간 1:1 매핑 규칙(Theorem 2)을 유도.
5. 식 `g(c) = (1/2)[a_{m-1}N(m,x) + Σ a_i N(i-x+1,x)]`이 인코딩(index→codeword)/디코딩(codeword→index)을 단순 덧셈 기반 매핑으로 축소하는 핵심 — lookup table 없이 저복잡도 실현.
6. 확장 기법: 셀프클로킹을 위한 C-LOCO(all-0/all-1 codeword 제거), 균형(balanced) LOCO 코드(부호의 상보성으로 DC-free 달성)까지 정의.
7. LDPC 결합(LDPC-LOCO): binary spatially-coupled(SC) LDPC 부호의 parity bit(n-w bits)만 C-LOCO로 인코딩해 ISI에 취약한 parity 영역의 신뢰도를 높이고, 나머지는 인터리버로 처리. 디코더 측은 LOCO 디코더가 하드결정/에러정정 후 LDPC decoder(FFT-QSPA, GF(2))에 고신뢰 LLR을 전달.
8. 결과: SC Code 1(rate 0.725, n=3774)에 C-LOCO(rate 0.6842)를 parity에만 적용 시 FER≈10^-6에서 동일 rate 대비 밀도 이득 약 20%, 동일 overall rate의 SC Code 2 대비 약 16% 이득, error floor 기울기도 더 가파름(개선).
9. 한계: HW 구현/합성 없음(순수 이론+시뮬), 검증 채널이 자기기록(PDNP/BCJR) 중심이며 LDPC는 non-binary GF(2) FFT-QSPA 디코더로 NAND의 binary QC-LDPC min-sum 구조와 다름, Flash 적용은 서론에서만 언급되고 실험/수치 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LOCO/C-LOCO 제약코드 인코더-디코더(사전식 인덱싱) | 대응 없음 | Prime ECC는 QC-LDPC 부호화/복호 엔진이며 별도 line-coding 계층은 프로파일에 대응 모듈 없음 |
| LDPC parity bit만 별도 코드로 보호(LDPC-LOCO 결합 구조) | `encoder.cpp` `LDPC_encoder_4KB()` | 개념적으로 parity 영역 신뢰도 강화 아이디어는 인코더 파이프라인과 맞닿지만 구현 방식(사전식 제약코드)은 완전히 다른 레이어라 실질 대응 없음 |
| Flash ICI 완화(패턴 101/010 금지) 언급 | 대응 없음 | 서론에서만 언급되고 본문 실험은 MR 채널 기준이라 Prime ECC의 NAND soft-read/LLR 모듈과 직접 연결되지 않음 |
| SC-LDPC(non-binary GF(2) FFT-QSPA) 디코더 | `decoder.cpp` `LDPC_Decoder()` | 부호구조(SC-LDPC)만 유사하고 알고리즘(FFT-QSPA)은 Prime ECC의 min-sum과 다름 |

> 적용 가치: 낮음 — 부호설계 이론이 line coding(제약코드) 계층에 집중되어 있고 LDPC 실험도 자기기록 채널·비이진 디코더 기준이라, NAND binary QC-LDPC ECC 엔진에 직접 이식할 모듈이 프로파일 표에 없다.
