> 총평: Absorbing set(error-floor 핵심 원인 구조)을 coset/syndrome 프레임워크로 재해석한 조합론적 이론 논문이며, 디코더 알고리즘이 아닌 코드 설계·검증 단계의 error-floor 구조 탐색 도구로서 간접적 이식 가치가 있음.

### B. 알고리즘 요약

1. 대상은 일반 이진 [n,k] 선형블록코드(LDPC 포함)의 Tanner graph 상 absorbing set(반복 디코더 실패, 특히 error-floor의 원인 구조)이며 채널 비종속적 조합론 분석.
2. 문제의식은 기존 absorbing set 탐색이 branch-and-bound 전수탐색([11],[12])에 의존해 비효율적이라는 점.
3. 핵심 정의는 `(a,b)-absorbing set` D(|D|=a, 홀수차수 체크노드 수 b)와 `fully absorbing set`(D 밖의 모든 변수노드도 O(D) 조건 만족)이며, syndrome은 `syn(x)=Hx^T`.
4. 핵심 발견(Theorem III.1)은 한 coset 안의 absorbing set support vector 집합이 코드 C의 부분공간 D의 translate `x_D + D` 형태라는 것.
5. Theorem III.4/Corollary III.5는 한 coset이 fully absorbing set을 하나라도 포함하면 그 coset 전체(codeword의 translate 전부)가 fully absorbing set이 됨을 증명 — 탐색 단위를 벡터에서 coset으로 축소 가능.
6. Lemma IV.2/IV.3은 low-weight absorbing set과 fully absorbing set의 syndrome weight 상한을 변수/체크노드 차수로 유도, Theorem IV.4는 syndrome weight가 `(min_v d_v - 1)/2` 이하면 해당 coset 전체가 fully absorbing임을 보장.
7. 이 상한들을 이용한 Algorithm 1은 저weight syndrome 구간은 자동으로 fully absorbing으로 분류(Theorem IV.4)하고, 중간 구간만 Theorem V.1(비-fully-absorbing coset의 syndrome support를 포함하는 syndrome은 자동 제외)로 가지치기하여 탐색.
8. 별도 학습/최적화 절차 없음(순수 조합론적 정리 기반 탐색).
9. 결과는 소형 예제 코드([6,3], [9,3], [10,4] 코드)에서 coset 구조 표(Table I, II)로 fully absorbing coset 개수·syndrome weight 분포를 실증; 수치적 성능(BER, iteration) 실험은 없음.
10. 한계는 QC-LDPC의 circulant 구조를 활용하지 않은 범용 선형코드 이론이며, 대형 코드에서의 알고리즘 실제 실행 시간/스케일링은 향후 연구로 남김.
