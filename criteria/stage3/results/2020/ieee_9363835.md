### B. 알고리즘 요약

1. 시스템: 장파(long-wave)/수중 통신 채널에서 필터링·오버샘플링·모터 잡음 등으로 인해 샘플 간 상관관계를 갖는 correlated Gaussian noise 모델을 가정.
2. 문제: 전통 BP 디코딩은 AWGN(비상관) 가정으로 설계되어 상관 잡음 채널에서 성능 저하 발생.
3. 채널 모델: `[n1,...,nk] = [w1,...,wk] * Σ^(1/2)`, `Σ_i,j = η^|i-j|` (η=상관계수)로 상관 가우시안 잡음 생성.
4. 핵심 기법: BP를 2라운드로 수행. 1라운드 BP 결과로 얻은 추정 잡음 `n̂`을 Gated Neural Network(Gated-NN 또는 Gated-CNN)에 입력해 실제 잡음 n에 근접하는 예측치를 학습.
5. 핵심 식: 수신신호 `y=1-2x+n`, 잔차 신호 `ŷ=1-2x+(n-ñ)`으로 잡음 성분을 제거한 뒤 2라운드 BP용 LLR을 잔차 분산 `σ²`로 재계산.
6. 확장: Gated neuron(input gate i, output gate o를 갖는 게이트 구조)을 NN/CNN 노드에 적용해 그래디언트 폭주/소실을 완화, 일반 NN/CNN 대비 학습 성능 향상.
7. 부수 트릭: 파라미터 수를 통제한 대조군(NN2, CNN2)을 별도 설계해 성능 향상이 단순 파라미터 증가 때문이 아님을 검증.
8. 학습: 데이터셋 생성 → NN 학습·파라미터 저장 → ŷ로 새 잡음분포 계산·LLR 갱신 → 전체 디코딩 4단계 파이프라인 (TensorFlow-GPU, GTX-1080Ti).
9. 결과: BER=10^-4에서 BP+Gated-CNN+BP가 기존 BP 대비 약 3dB 코딩이득, pectinate noise에서도 4dB 이상 개선.
10. 한계: 코드 스펙(rate/length)·복잡도·HW 구현 전혀 언급 없음, 순수 시뮬레이션 수준이며 상관/pectinate 잡음이라는 특수 채널 가정에 한정.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP(Sum-Product) 디코딩 알고리즘 | 대응 없음 | Prime ECC는 min-sum 기반이며 논문은 full BP(Sum-Product) 사용, 상이한 근사 방식 |
| Gated-NN/CNN 기반 잡음 추정·제거 | 대응 없음 | NN 전면 도입 기법으로 Prime ECC C++ HW 모사 구조와 무관 |
| correlated Gaussian noise 채널 모델 | `channel.cpp` | 채널이 AWGN/RBER 가정이며 상관 잡음 모델은 없어 관련성 낮음 |
| LLR 갱신(잔차 분산 기반) | `channel.cpp` `Set_LLR_Th()` | 개념적 유사성만 있고 NN 종속적 재계산이라 이식 난해 |

적용 가치: `낮음` — NAND 채널(RBER, 비상관)과 무관한 특수 상관잡음 채널 대상 NN 기반 기법으로, binary QC-LDPC HW 디코더에 직접 이식할 요소가 없음.
