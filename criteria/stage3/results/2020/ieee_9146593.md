### B. 알고리즘 요약

1. 대상: bi-regular bipartite graph(예: `(dv, dc)`-regular LDPC Tanner graph)에서 길이 `k`(`g ≤ k ≤ 2g-2`, `g`=girth)인 short cycle의 개수 `Nk`를 계산하는 문제.
2. 풀려는 문제: 기존 두 접근법 중 (1) directed edge matrix `Ae`의 spectrum `{η_i}` 이용(Theorem 1, `O(|E|^3)`)은 범용이지만 느리고, (2) adjacency matrix spectrum `{λ_i}` + degree sequence 이용(Theorem 2, `O(|V|^3)`)은 빠르지만 `k≥g+6`에서 재귀식이 지수적으로 복잡해짐(trapping-set 특성화에 필요한 `g+10`대 cycle 계산 시 실무적으로 막힘).
3. 핵심 가정/모델: bi-regular bipartite graph(`dv`, `dc` 고정)의 Tanner graph 구조, 그래프의 adjacency matrix `A`와 directed edge matrix `Ae`(`2|E|×2|E|`) 정의.
4. 핵심 기법: 보조 행렬 `𝒜`(크기 `|V|^2×|V|^2`, 식 6)를 정의하고 그 고유값 `{ξ_i}`가 `λ`와 2차방정식 `ξ^2+(-λ^2+q1+q2)ξ+q1q2=0`(식 20, `q1=dv-1, q2=dc-1`)로 연결됨을 증명.
5. 핵심 식의 의미: 이 2차방정식을 통해 `Ae`의 고유값(즉 `Nk` 계산에 필요한 `{η_i}`)을 `A`의 고유값(계산이 상대적으로 쉬운 `{λ_i}`)으로부터 상수 시간(주어진 cycle 길이 `k`에 무관)에 유도 - 기존 Theorem 2의 지수적으로 복잡한 재귀식(`Ω_i`, `Ψ_i`) 계산을 완전히 우회.
6. Theorem 3: `Ae`의 전체 고유값 스펙트럼을 3단계(Case A/B/C: `λ≠0∧ξ≠1`, `λ=0`, `λ≠0∧ξ=1`)로 나눠 `A`의 고유값·중복도로부터 명시적으로 계산.
7. 부수 트릭: bipartite 구조를 이용해 `A`를 block 형태(`[[0,D],[D^t,0]]`)로 분해, null space 차원(`Rank-Nullity`)으로 고유값 중복도를 도출.
8. 학습/최적화 절차 없음(순수 대수적 유도).
9. 결과: `Nk` 계산 복잡도가 `O(|E|^3)→O(|V|^3)`로 감소(`|E|>|V|`인 고차수 그래프에서 유리), tesseract graph `Q4`(예제 2) 등에서 기존 backtracking 알고리즘과 일치하는 수치 검증.
10. 한계: 실제 LDPC 부호의 waterfall/error-floor 성능 수치나 디코더 시뮬레이션 결과 없음(순수 계산 복잡도 이론), bi-regular graph에 한정(irregular LDPC는 Theorem 1만 적용 가능), HW 구현/양자화 개념 전혀 없음.
