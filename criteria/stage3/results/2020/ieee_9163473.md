### B. 알고리즘 요약

1. 선형 이진 코드(BCH(127,99), BCH(127,64), BCH(63,45), BCH(63,36))의 Tanner graph 기반 BP 디코딩을 신경망으로 재해석한 기존 연구([10],[11])를 확장.
2. 기존 신경망 디코더는 반복마다 variable-layer 1개 + check-layer 1개 구조라 성능 개선에 한계.
3. 채널은 AWGN, 학습은 message passing symmetry 조건을 만족시켜 all-zero codeword의 노이즈 버전만으로 수행.
4. 핵심 기법: 각 반복(iteration)에서 variable-layer를 2개(경우에 따라 그 이상) 연속 배치하는 "Multi Variable-layer Network(MVN)". check-layer는 그대로 유지.
5. 식 `μ(l)_v,ṽ,c = tanh(...)`, `μ(l)_ṽ,c,v = tanh(...)` 로 두 단의 variable-layer가 순차로 가중치(`b`, `ω`)를 적용해 메시지를 정제 — 복잡한 check-layer 연산을 일부 대체해 곱셈 수를 줄이는 것이 핵심.
6. symmetry 조건(대칭성)이 유지됨을 증명해, 학습 데이터가 zero codeword 노이즈 버전으로 충분함을 보장.
7. 학습: cross-entropy multi-loss `Γ`를 RMS-Prop optimizer(lr=0.001~0.003)로 학습, 코드별로 학습 대상 가중치(마지막 반복 ω만 학습 등) 선택적 적용.
8. 결과: BCH(127,99)/(127,64)에서 [11] 대비 BER 최대 3.54배/6.3배 감소, BCH(63,45)/(63,36)에서 최대 5.25배/2.12배 감소. 가중치·곱셈 수는 [11] 대비 BCH(127,*)에서 감소, BCH(63,*)에서는 소폭 증가하지만 [14]보다는 적음.
9. 한계: 짧은 길이(N=63,127) BCH 코드에 대한 시뮬레이션 결과만 제시, HW 구현·합성 없음, LDPC나 NAND 채널(RBER 등) 실험 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP 전체를 대체하는 신경망 디코더(MVN) | 대응 없음 | Prime ECC는 min-sum 기반 bit-exact HW 시뮬레이터로 NN 전면 대체 구조와 무관 |
| variable-layer/check-layer 가중치 학습 | 대응 없음 | 코드별 오프라인 학습된 실수 가중치는 `decoder.cpp`의 고정 min-sum/HD·2SD·3SD 양자화 테이블과 구조적으로 대응되지 않음 |
| Tanner graph 메시지 전달 구조 | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 개념적 유사성(반복 메시지 패싱)만 있고 구체 알고리즘은 대응 없음 |

> 적용 가치: 낮음 — NN 전면 대체형 디코더로 프로파일 §5의 "NN/딥러닝 디코더 = 대응 없음" 판정에 해당하며, 짧은 BCH·AWGN·시뮬뿐이라 NAND binary QC-LDPC HW 이식과 무관.
