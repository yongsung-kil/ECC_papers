### B. 알고리즘 요약

1. 시스템: SISO AWGN 채널, LDPC 인코더(rate `(k,n)`) + 성상 매핑(M=2) 후 전송, 수신단은 demapper+LDPC decoder를 대체.
2. 문제의식: 기존 belief-propagation 기반 demapping+decoding은 반복 연산 비용이 크고 저/중 SNR 구간 성능이 제한적.
3. 채널 모델: `y[t] = x[t] + w[t]` (AWGN, 별도 간섭·페이딩 없음, 완벽 동기화 가정).
4. 핵심 기법: Autoencoding Deep Neural Network(ADNN) = CNN 기반 오토인코더(denoising) + 6-layer DNN(regression)을 직렬 결합해 수신 IQ 심볼로부터 직접 비트를 추정.
5. 핵심 연산: `y = Θ(Ωx + b)` (DNN dense layer), Conv2D/MaxPooling(인코더)·UpSampling(디코더)으로 데이터 차원 축소·복원.
6. 확장: 손실함수는 MSE와 BER 두 가지를 동시에 사용해 학습(Adam optimizer, 100 epoch, 학습 SNR 7dB 고정).
7. 구현 디테일: activation은 IQ 데이터가 zero-mean이라는 이유로 tanh 사용, noise layer로 매 epoch 다른 노이즈 실현을 주입해 과적합 방지.
8. 학습 절차: TensorFlow 2.0 + GPU(RTX 2080 Ti ×2)로 오프라인 학습, 학습 완료 후 추론은 반복형 LDPC 복호보다 빠르다고 주장(수치 미제시).
9. 결과: rate 0.9/0.95 LDPC 코드에서 BER 10⁻³ 목표 시 벤치마크(BP 복호) 대비 SNR 3dB 개선.
10. 한계: 단일 학습 SNR(7dB)에서만 학습, AWGN 단일채널·SISO 가정, 실제 HW/복잡도 분석 없음, LDPC 디코더 내부 알고리즘(min-sum 등) 아님.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| ADNN 기반 demapping+decoding 전체 | NN/딥러닝 디코더 (대응 없음) | binary QC-LDPC min-sum 구조를 신경망으로 통째 대체하므로 대응 모듈 없음 |
| LDPC 인코더/디코더 벤치마크(BP) | `decoder.cpp` `LDPC_Decoder()` | 비교 대상일 뿐 본 논문 기여가 아니므로 이식 요소 아님 |
| soft-read/양자화 관련 언급 | `channel.cpp` `Set_LLR_Th()` | 없음 — 논문은 연속값 IQ 입력을 신경망에 직접 투입, 양자화·LLR 설계 논의 없음 |

> 적용 가치: 낮음 — 부호설계·디코더 알고리즘·HW 요소가 전혀 없고 종단간 신경망 대체 접근이라 binary QC-LDPC 이식 지점이 없음.
