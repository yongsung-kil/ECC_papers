### B. 알고리즘 요약

1. AWGN 채널 + BPSK, LLR 기반 수신신호 `r`, regular LDPC (code length 32/128/256bit, 밀도 12.5~1.6%)로 실험.
2. 기존 sum-product/bit-flip/IPM 등은 근사해만 제공하며, LDPC 복호를 QUBO 문제로 엄밀 변환한 선행연구는 없었다는 문제의식.
3. LDPC 복호를 조합최적화문제(COP)로 정의하고, 목적함수 `f = Σ(ri-(1-2ŵi))²`, 제약 `Hŵ=O`로 표현.
4. 패리티 제약(XOR)을 짝수합 제약으로 바꿔 정수 `l_j`를 unary/binary encoding으로 0/1 변수화, 제약항 `HS = Σ(Σ H_{j,i}ŵi - 2l_j)²` 구성.
5. 전체 목적함수 `H = HC + αHS` (QUBO 형태 `xᵀQx + C`)로 결합, 최소해가 원 LDPC 해가 되도록 함.
6. Theorem 1: `α > Σ4|r_i|`이면 QUBO의 ground state가 항상 LDPC 제약을 만족하는 해임을 증명(이론적 정확해 보장).
7. Ising machine(정수 목적함수만 지원)에 맞춰 H를 50배 후 반올림하는 실무적 변형 적용.
8. Fujitsu Digital Annealer + Replica Replacement Method(128 replica, 1억 iteration)로 QUBO를 풂; unary encoding이 binary encoding보다 정확도 높음.
9. code length 256bit, SNR 4.0dB에서 정확도 93.9%, 평균 실행시간 214.0ms 달성.
10. 한계: 실행시간이 수백 ms 수준으로 NAND ECC 실시간성과 거리가 멀고, 전용 Ising HW 필요, HW(RTL) 설계 없음, 순수 시뮬(소규모 code length)뿐.
