### B. 알고리즘 요약 (10줄 내외)

1. 시스템: K개 이산 입력 심볼(PAM)과 연속 가산잡음 `u = x_i + n_i`를 K-1개 threshold `h`로 양자화하는 DMTC(discrete memoryless thresholding channel) 모델.
2. 문제: DMC와 달리 DMTC의 mutual information `I(X;V)`는 입력분포 `p_X`와 threshold `h`에 대해 non-convex/non-concave라서 채널용량 계산이 어려움.
3. 가정: 잡음은 조건부 밀도 `φ_i(u) = p_{U|X}(u|x_i)`로 모델링, 실험에서는 i.i.d 가산 가우시안 잡음 사용.
4. 핵심 기법: 교대최적화(alternating optimization)로 `p_X` 고정 시 `h` 최적화(P2), `h` 고정 시 `p_X` 최적화(P1, gradient descent)를 반복.
5. 핵심 식: P2를 KL-divergence 기반 k-means로 풀며, `EU[D(x_u||x_v)] = I(X;U) - I(X;V)`이므로 KL-divergence 최소화가 곧 mutual information 최대화와 동치.
6. 확장: 일반 quantizer용 k-means(Algorithm 2, `O(K^2MT)`) 대신, thresholding quantizer 구조를 이용해 cluster assignment를 이분탐색으로 대체하는 Algorithm 3 제안.
7. 구현 디테일: `φ_i(u)/φ_j(u)`가 단조(log-concave/log-convex 잡음 분포에서 성립) 조건 하에 `F_k(u)=D(x_u||x_{v_{k-1}})-D(x_u||x_{v_k})`가 단조증가·유일근을 가짐을 증명, bisection/Newton법으로 근을 탐색.
8. 최적화 절차: Algorithm 1이 P1(gradient descent)과 P2(Algorithm 3)를 수렴할 때까지 반복.
9. 결과: 계산복잡도가 `O(K^2 M log M T)`에서 `O(K^2 T log M)`으로(팩터 `M log M` 절감), K=5 실험에서 평균 실행시간이 exhaustive search 대비 10.22초 vs 319725.21초.
10. 한계: 순수 이론/시뮬레이션(HW 미설계), LDPC 부호나 디코더 자체를 다루지 않고 채널 quantizer 설계(정보이론)에 국한, 실제 NAND 채널 특성 반영 없음.
