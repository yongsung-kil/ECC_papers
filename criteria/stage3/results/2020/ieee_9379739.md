> 총평: min-sum 정규화 팩터를 check ratio·degree 기반 lookup table로 적응화하는 기법으로, 우리 decoder의 LLR 테이블 구조와 접점이 있어 이식 가치 중간 수준.

### B. 알고리즘 요약

1. 5G-NR LDPC(MET, QC, raptor-like, rate-compatible/length-scalable) BG2 대상, check-node degree `dh = {3,4,5,6,8,10}`, layered scheduling(TDMP) 채널은 BI-AWGN·Rayleigh.
2. 기존 MSA는 scale-invariant하지만 성능 손실이 있고, AdjMSA는 성능은 좋으나 tanh 연산으로 scale-variant; 고정 정규화 팩터(NMSA)는 SNR·채널·CN degree별 최적값이 달라 부정확한 문제를 해결하려 함.
3. check ratio `ρ`(패리티 불일치 비율)와 CN degree에 따라 정규화 팩터 `α`가 달라진다는 것을 핵심 가정으로 삼음.
4. 핵심 기법: 정규화 팩터 fitting function `ρ-α`을 degree별로 구해 유사한 것끼리(`{3,4}`, `{5,6}`, `{8,10}`) 그룹핑해 3개 type의 lookup table만 생성(기존 방식 대비 샘플 부족·고복잡도 해소).
5. 핵심 식: `α = argmin_α E[(L^spa_mn − α·L^ms_mn)^2]` (식5) — SPA와 MSA 출력 차를 최소화하는 팩터를 학습, 복호 시 `L_mn = α × L^ms_mn`(식9)로 CN 출력에 적용.
6. 확장 없음(단일 알고리즘, layered scheduling과 결합).
7. 부수 트릭: lookup table은 오프라인 학습으로 생성해 온라인 복호 시 추가 연산 거의 없음(offline curve fitting → sampling/quantization).
8. 학습 절차: 다양한 SNR에서 Monte Carlo로 `(ρ, α)` 데이터 수집 → degree별 fitting function 도출 → 그룹핑 후 재학습·lookup table 생성.
9. 결과: BLER 10^-3 기준 SPA 대비 gap이 rate-1/4에서 0.23dB, rate-5/6에서 0.03dB; AdjMSA와 유사한 성능을 MSA 복잡도 수준에서 달성.
10. 한계: HW 미설계, 시뮬레이션(BI-AWGN/Rayleigh)만 수행, 5G-NR MET 구조 특화(BG2 한정)로 gate/throughput 수치 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 정규화 팩터 lookup table(degree/check-ratio별) | `decoder.cpp` `CNU_Update_New_Mag()`, `ecc_data.h` `PARAM_LLR` | Prime ECC도 iteration별 LLR 테이블을 이미 사용하나, check-ratio 기반 동적 정규화는 추가 검토 가치(min-sum 정확도 개선) |
| CN degree별 그룹 단순화(fitting function 유사도로 type 축소) | `decoder.cpp` `Get_VNU_Table_Idx()` | 우리 HCU는 degree 9/7 col 2개만 예외 처리하는 구조라 그룹핑 아이디어 자체는 유사하나 5G MET 특유의 degree셋(3~10)과 직접 대응 안됨 |
| layered scheduling(TDMP) | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 이미 layered(row 단위) 구조를 사용 중이라 중복 기여 |
| check ratio ρ 계산(정지조건 겸용) | `partial_CRC.cpp`, `full_CRC.cpp` | 조기종료 조건과 유사한 통계량이나 직접 대응 함수는 없음 — 대응 없음 |

적용 가치: **중간** — min-sum 정규화 팩터 lookup table 및 degree 그룹핑 방법론은 decoder LLR 테이블 개선에 참조 가치가 있으나, MET 5G-NR 특화 구조라 우리 QC-LDPC(고정 rate) H-matrix에는 방법론만 이식 가능.
