> 총평: RS-LDPC 부호(대수적 구성)의 최소거리/stopping distance 상한을 수학적으로 증명한 순수 이론 letter로, NAND용 QC-LDPC 이식 관점에서는 참고 가치가 낮음.

### B. 알고리즘 요약

1. 대상 부호: extended Reed-Solomon(RS) 부호(길이 `q=2^r`, 정보심볼 2개)로부터 구성한 이진 정규(regular) LDPC 부호 `C(γ,ρ)`, PCM `H(γ,ρ)` 크기 `γq × ρq` (`γ`=열weight, `ρ`=행weight). `ρ=q`이면 full-length, `ρ<q`이면 non-full-length.
2. 문제: 일반 LDPC 부호의 최소거리 `d(C)`·stopping distance `s(H)` 계산은 NP-hard/intractable이나, RS-LDPC처럼 대수적 구조가 있으면 해석적으로 유도 가능. 기존 연구는 `γ≤5`까지만 다룸, 본 논문은 `γ=6` 확장.
3. 핵심 가정/모델: PCM을 vector representation(`H(γ,ρ)`의 각 열을 `F_q` 원소 벡터 `[τ0,...,τ_{γ-1}]^T`로 표현, `τ_m=(1+α^{m-1})τ0+α^{m-1}τ1`)으로 다룸(대수적 접근, 채널모델 없음).
4. 핵심 기법: `q≥8`에서 `H(6,q)`가 크기 12인 stopping set(=weight-12 codeword support)을 갖도록 하는 특정 컬럼 부분집합을 명시적으로 구성(연립방정식 (7)~(9) 풀이)하여 `s(H(6,q)) ≤ d(C(6,q)) ≤ 12` (Theorem 1)를 증명.
5. `q≥32`에서는 이 상한이 tight(즉 `s(H)=d(C)=12`)하다는 것을 컴퓨터 탐색(exhaustive for q=8,16 / probabilistic for q=32,64,128)으로 뒷받침하는 Conjecture 1 제시.
6. non-full-length 부호 `C(6,ρ)` (`ρ<q`)에 대해서는 `ρ`가 `q`의 약 절반일 때 `d(C(6,ρ))`, `s(H(6,ρ))`가 원래 값(12)보다 개선(=증가)되는 경향을 컴퓨터 탐색(Table I)으로 관찰. 이론적 증명은 아니고 경험적 관찰 + affine invariance 성질을 이용한 정성적 설명.
7. 부수 트릭: IEEE 802.3an 표준에서 실제 사용되는 `(γ,ρ)=(6,32)`, 길이 2048 RS-LDPC 부호가 non-full-length 사례로 언급됨(고rate 통신 응용).
8. 학습/최적화 절차 없음(순수 대수적 증명 + 컴퓨터 탐색 검증).
9. 결과 수치: `s(H(6,8))=d(C(6,8))=8`(오타 아님, 원문 그대로 q=8은 결과 상이하게 보고되나 본문 표기 기준 재확인 필요), `s(H(6,16))=d(C(6,16))=8`; 상한 `d(C(6,q))≤12` for `q≥8`; `d(C(6,32))=14`로 알려짐(참고문헌 인용).
10. 한계: 특정 대수적 구성(RS 기반, γ=6 고정)에만 적용되는 조합론적 결과이며, 디코더/HW/실제 성능(BER, throughput 등)에 대한 논의가 전혀 없는 순수 이론 letter.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| RS-LDPC 부호 구성(대수적, PCM H(γ,ρ)) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 QC-LDPC(base matrix+circulant shift) 고정 구조이며 RS 기반 대수적 구성과 무관, 코드 교체 자체가 무의미 |
| 최소거리/stopping distance 이론 상한 증명 | 대응 없음 | 디코더/인코더 알고리즘이 아닌 순수 조합론적 부호 특성 분석으로 실행 코드 대응 없음 |
| non-full-length shortening (ρ<q) | 대응 없음 | Prime ECC의 puncturing(`punct_col_num`)과 개념적으로 유사할 수 있으나 본문에 실질적 이식 근거 없음 |

> 적용 가치: 낮음 — RS 기반 대수적 LDPC 구성에 국한된 순수 이론(최소거리 상한 증명) 논문으로 Prime ECC 3.1의 QC-LDPC 구조·디코더·인코더 어디에도 실질적으로 이식할 요소가 없음.
