### B. 알고리즘 요약

1. 대상은 [[N,K]] 양자 stabilizer 부호(check matrix `S ∈ {I,X,Y,Z}^(M×N)`), 이진 syndrome `z ∈ {0,1}^M`을 이용한 BP 복호.
2. 기존 방식은 4진 분포 벡터 메시지를 쓰는 BP4가 필요해 이진 부호용 BP2 대비 check-node 복잡도가 `q^2=16`배 크다는 문제.
3. 채널모델은 depolarizing channel (`p_n=(1-ε, ε/3, ε/3, ε/3)`), Tanner graph는 4-cycle이 많아 BP 성능이 저하된다고 가정.
4. 핵심 기법: syndrome이 이진이라는 점을 이용해 MacKay의 δ-rule(LD 기반)을 GF(4)로 확장, 메시지를 벡터 대신 스칼라 `δ_mn`으로 전달하는 parallel BP4(Algorithm 1) 제안.
5. 핵심 식 `δ2⊙δ3 = (pI2+pZ2)-(pX2+pY2)` 등으로 두 분포 결합을 O(q) 덧셈+O(1) 곱셈으로 계산(기존 FFT법 O(q log q) 대비 경량), check-node efficiency는 기존 BP4와 동일 출력을 내면서 16배 향상.
6. 확장 2단: 스칼라 메시지이므로 message normalization(파라미터 `αv`)을 도입한 Algorithm 2 및 serial schedule 적용으로 짧은 cycle에 의한 오류 억제.
7. 구현 디테일: `(·)^(1/αv)` 연산은 곱셈 1회+덧셈 2회로 근사 계산(HW 친화적 근사이나 실제 HW 설계는 없음).
8. 별도 학습/최적화 절차는 없음(DE/GA/NN 아님). heuristic한 column-weight 균일화(MIN-VAR/MIN-MAX)로 bicycle 부호 구성만 수행.
9. 결과: [[800,400]] bicycle 부호 및 [[3786,946]] bicycle 부호(row-weight 24) 시뮬에서 message normalization 적용 시 logical error rate <10^-4 구간 error-floor가 크게 개선.
10. 한계: 순수 컴퓨터 시뮬레이션(이론+시뮬)뿐이며 HW 설계·gate count·throughput 언급 없음. GF(4) 양자 부호 전용으로 binary NAND LDPC로의 직접 적용은 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| GF(4) 양자 stabilizer BP4 (스칼라 δ-rule) | 대응 없음 | non-binary/양자 부호 전용, Prime ECC는 binary QC-LDPC만 지원 |
| message normalization (αv) | `decoder.cpp` `CNU_Update_New_Mag()` | 개념(정규화 팩터)만 유사, 실제 정규화 대상(GF(4) 분포)은 다름 |
| serial/parallel schedule 비교 | `decoder.cpp` `LDPC_Decoder()` (Dual-Update 옵션) | 스케줄링 아이디어 방향성만 참고 가능, 알고리즘 자체는 별개 |
| bicycle 부호 구성(column-weight 균일화) | `ecc_top.cpp` `Load_PCM()` | 양자용 무작위 sparse 부호 구성법이라 QC-LDPC 구조와 무관 |

> 적용 가치: 낮음 — 양자 GF(4) stabilizer 부호 전용 기법으로 binary NAND QC-LDPC와 부호 모델이 근본적으로 다르며, 이미 Prime ECC가 보유한 min-sum/정규화/스케줄링과 개념적 중복 외에 실질적 이식 요소가 없음.
