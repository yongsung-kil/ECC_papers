> 총평: 이진입력·연속출력 채널에서 출력 제약부 mutual-information quantizer의 구조(convex cell)를 증명한 이론 논문으로, LDPC 디코더 soft-quantization/LLR 설계에 이론적 시사점은 있으나 본문에 부호·HW 실험이 전혀 없어 즉시 이식 가능한 구체 기법은 부재.

### B. 알고리즘 요약

1. 이진 랜덤 입력 `X∈{x1,x2}`가 연속 노이즈 채널(조건부밀도 `φ1(y),φ2(y)`)을 거쳐 연속 출력 `Y`가 되고, quantizer `Q`가 이를 이산 출력 `Z∈{z1,...,zN}`로 매핑하는 일반 채널 quantization 문제를 다룬다.
2. 순수 상호정보량 최대화(`I(X;Z)`)만 다루던 기존 연구와 달리, 출력 분포 `pZ`에 임의 비용함수 `C(pZ)`가 걸린 제약 하 `βI(X;Z)−C(pZ)` 최대화 문제를 새로 정식화한다(엔트로피 제약 압축, 전력/지연 제약 등 응용 동기).
3. 사후확률 `r_y = p_{x1|y}`를 채널 통계량으로 정의하고, KL divergence 기반 distortion `D(Q)=Σ∫ D(v_y||c_i)μ(y)dy`을 사용해 quantizer 최적화를 clustering 문제로 재해석한다.
4. 핵심 정리(Theorem 1, 2): 임의의 quantizer `Q(y)`에 대해 동일한 `pZ`(따라서 동일 `C(pZ)`)를 내면서 `I(X;Z)`는 더 크거나 같은 convex-cell quantizer `Q̄(r_y)`가 항상 존재함을 증명 — 즉 최적해는 `r_y`를 임계값들로 나누는 형태 `Q*(r_y)=z_i (a*_{i-1}≤r_y<a*_i)`.
5. 증명은 두 출력 집합 간 KL divergence 차 `F(r_y)=D(v_y||c1)−D(v_y||c2)`가 `r_y`에 대해 단조증가함을 이용해 임의 quantizer를 convex cell quantizer로 치환해도 distortion이 감소하지 않음을 보이는 방식.
6. N=2 특수해: 최적 quantizer는 단일 threshold `a*_1∈(0,1)`로 결정되며, `φ2(y)/φ1(y)`가 단조함수이면 고전적 단일 threshold quantizer(`y`축 threshold `h_1`)도 최적임을 보임.
7. N>2 확장: 제약이 `C(pZ)=Σ g_i(p_{Zi})` 형태(엔트로피 제약 등 다수 포함)이면 문제를 1차원 scalar quantization으로 환원해 dynamic programming으로 풀 수 있음(SMAWK 알고리즘 적용 가능성 논의).
8. 수치 예시로 `p1=0.7, p2=0.3, φ1=N(1,1), φ2=N(-1,1)`, β=5, `C(pZ)=H(Z)`일 때 최적 `r*_y=0.62`(단일 threshold `y*=-0.17887`)를 도출하고, β값에 따른 `I*(X;Z)` vs `H*(Z)` trade-off 곡선을 제시.
9. 정량 결과는 이론적 최적값(`βI(X;Z)−H(Z)`)과 threshold 값 자체이며, 실제 LDPC 코드나 디코더 성능(BER/BLER)에 대한 실험은 없음.
10. 한계: 완전히 이론적(증명+소규모 수치예시)이며 실제 부호·채널·HW 구현/검증이 전무. K>2 입력 채널로의 일반화는 "open problem"으로 남김.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 출력 제약 하 mutual-information quantizer 구조(convex cell) | `channel.cpp` `Set_LLR_Th()` | soft-read LLR threshold 설계 이론과 관련되나, 본문은 특정 채널모델(NAND RBER)·비트폭 제약 하 threshold 도출을 다루지 않아 직접 적용식은 없음 |
| N=2 단일 threshold quantizer | `channel.cpp` `Set_R_Offset()` | 1비트(hard-decision) read threshold 설계에 이론적 참고 가능하나 구체 파라미터 부재 |
| dynamic programming 기반 다중 threshold quantizer(N>2) | `ecc_data.h` `PARAM_LLR` | 다중 threshold(soft-read LLR 테이블) 설계와 개념적으로 유사하나 논문은 LDPC 디코딩 성능과 연결짓지 않음 |
| LDPC mutual-information lookup table 디코딩[2] (인용만, 본문 미전개) | 대응 없음 | 논문 자체 실험이 아니라 서론 인용에 그침 |

> 적용 가치: 낮음 — quantizer 구조에 대한 순수 이론 증명으로 LDPC/NAND 채널에 특화된 실험이나 구체적 threshold 설계 수치가 없어, Prime ECC의 soft-read LLR 양자화 설계에 즉시 참조할 이식 코드/파라미터가 없음.
