### B. 알고리즘 요약

1. 코드 스펙: CCSDS 표준 (2560,1024) QC-LDPC를 puncture하여 얻은 (2048,1024) rate 1/2 코드, check matrix `1536×2560`을 `512×512` 크기 순환 서브행렬(3×5)로 구성, layered scheduling에서는 12 layer(각 128행, row weight 3/6)로 재분할.
2. 문제: 기존 flooding(parallel) 스케줄은 모든 CN/VN 갱신을 매 iteration 전부 대기해야 해 수렴속도가 느리고 HW 유휴 자원 발생, LLR-BP는 hyperbolic 함수로 HW 구현 복잡.
3. 핵심 가정: AWGN 채널 + BPSK, 최대 iteration T=20(1차)/10(이후 실험).
4. 핵심 기법 1단: Min-Sum 근사(`L(rij) = Π sign(L(pi'j)) · min|L(pi'j)|`, 식(3))에 dynamic normalized correction factor `α(n) = α(n-1) + β` (β>0)를 곱해 NMSA→DNMSA로 발전, density evolution 기반 `α ≈ E(|L_LLR|)/E(|L_NM|)` 근사로 초기값 결정.
5. 핵심 식 의미: 정규화 계수 α가 반복이 진행될수록 1로 수렴해야 하므로 고정값 대신 반복별 선형 증가시켜 초기/후기 반복 모두에서 최적 신뢰도 보정.
6. 핵심 기법 2단: Layered(=layer-serial) 스케줄링에서 row weight가 작고 column weight가 큰 노드가 속한 layer를 우선 갱신하는 최적화된 update order를 채택(고정된 순서, 추가 HW 복잡도 없음), BER waterfall과 error floor를 동시 개선.
7. 부수 트릭: 식(9)의 VN 갱신이 단순 뺄셈 연산뿐이라 VN 갱신 모듈 자체를 HW에서 생략 가능; check matrix가 128×128 순환/영행렬 서브블록이라 하드웨어 저장에 유리.
8. 학습/최적화: α, β, threshold th는 컴퓨터 시뮬레이션 스윕으로 결정(α0=0.6, β=0.015, th=2), 별도 학습 알고리즘 없음.
9. 결과: BER=10⁻⁵에서 LDNMSA가 MSA 대비 약 0.5dB 이득, LLR-BP 대비 0.1dB 열세에 그침; 최대 반복횟수가 다른 알고리즘의 절반 수준(de-layering 조기종료로 신뢰 layer skip).
10. 한계: 전량 시뮬레이션(HW 미설계, gatecount/throughput 수치 없음), 파라미터(α,β,th)는 특정 부호·SNR 조건에서 튜닝된 값이라 다른 코드/채널엔 재검증 필요.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Dynamic Normalized Min-Sum(DNMSA) 보정계수 α(n) | `decoder.cpp` `CNU_Update_New_Mag()` | min-sum scaling factor를 반복별 동적으로 조정하는 기법으로 기존 정적 정규화 대비 개선안, 이식 난이도 낮음 |
| Layered scheduling + update order 최적화 | `decoder.cpp` `LDPC_Decoder()` (Dual-Update 스케줄) | Prime ECC의 Dual-Update(even/odd 교대)와 유사한 layer-serial 개념이나, row/column weight 기반 순서 결정 로직은 추가 이식 요소 |
| De-layering 기반 layer-skip 조기종료(threshold th) | `partial_CRC.cpp` `partial_crc_HW_T4()` | CRC 기반 조기종료와 별개로 layer 단위 조기 skip이라는 신규 아이디어, 이식 시 CRC 방식과 병행 검토 가능 |
| QC-LDPC 순환 서브행렬(512×512, 128×128) 구조 | `ecc_top.cpp` `Load_PCM()` | CCSDS 특정 부호로 고정된 base matrix이며 Prime ECC의 자체 H-matrix(z_sb=32)와 다름, 부호설계 이식은 난이도 높음 |

> 적용 가치: **높음** — dynamic normalized min-sum 보정계수와 layered update order, de-layering 조기종료 모두 decoder iteration 루프에 낮은 난이도로 이식 가능한 실용적 저복잡도 기법이며 QC-LDPC라는 부호 구조도 Prime ECC와 일치.
