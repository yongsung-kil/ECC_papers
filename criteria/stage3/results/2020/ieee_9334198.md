### B. 알고리즘 요약

1. 시스템: RA(repeat-accumulate) 부호를 spatial coupling한 SCRA 부호 2개를 병렬 결합한 PC-SCRA 부호 앙상블, 이진소거채널(BEC) 대상.
2. 문제: 기존 SC-LDPC는 다중 coupling chain 연결(extra edge, degree-2 check node, chain 병렬연결 등)로 threshold를 개선해왔으나, 더 단순한 인코딩 구조로 우수한 threshold를 얻을 방법이 필요.
3. 핵심 가정/모델: SCRA 부호는 SC-LDPC 대비 인코딩이 단순하고 낮은 복호복잡도와 좋은 threshold를 가진다는 전제(참고문헌 [8]), BEC 채널의 erasure probability 기반 density evolution 사용.
4. 핵심 기법: 동일 메시지 비트 `u`를 두 개의 동일한 `(J,K,L)` SCRA 서브인코더로 각각 인코딩해 `p1`, `p2`를 생성하고, `(u,p1,p2)`를 하나의 코드워드로 병합(인터리버 불필요, RA 구조의 무작위성 덕분).
5. 핵심 식: base matrix `B_PC = [[B1,B2,0],[B1,0,B2]]` (병렬결합 protograph), 설계 rate `R_PC = L/(L+2·(J/K)·(L+J-1))` — 부호율은 coupling length `L`과 J/K 비율로 결정.
6. 핵심 기법 2단: density evolution 방정식(6)~(10)으로 message bit/check/parity node 간 erasure probability를 iteration별로 갱신, BP threshold `δ_BP`를 `x_i^(ℓ)→0`이 되는 sup δ로 정의(식 11).
7. 부수 디테일: smoothing parameter `w`(또는 `μ`)로 edge connection을 무작위화한 `(J,K,L,w)` 앙상블도 제시, coupling width `μ=3,4`로 threshold 실험.
8. 최적화 절차: 명시적 학습 없음, degree distribution `(J,K)` 및 coupling length `L=30,50,100` 조합에 대해 density evolution을 반복 계산해 threshold 표(Table I, II) 생성.
9. 결과: PC-SCRA는 동일 rate SC-LDPC 대비 Shannon limit gap이 더 작음(예: `(3,6)` rate 0.4934에서 PC-SCRA gap 0.0100 vs SC-LDPC gap 0.0306; `(4,8)`에서도 0.0124 vs 0.0308).
10. 한계: BEC 채널의 이론적 threshold 분석에 국한, AWGN/실제 채널 BER 시뮬레이션 없음, 인코더/디코더 HW 설계 전무, QC-LDPC가 아닌 RA 기반 protograph 구조라 Prime ECC의 QC-LDPC 부호 프레임워크와 직접 호환되지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SCRA 부호 병렬결합(protograph, base matrix 구성) | 대응 없음 | Prime ECC는 QC-LDPC(base matrix+circulant shift) 고정 구조, RA 기반 SC 부호 구성법과 부호 프레임워크 자체가 상이 |
| BEC density evolution / BP threshold 계산 | 대응 없음 | 순수 이론 분석 도구로 실제 디코더(min-sum) HW와 무관 |
| Spatial coupling(coupling chain, coupling width) | [ecc_top.cpp](../Prime_ECC_3.1_Claude) `Load_PCM()` | H-matrix/부호구조 로드와 개념적으로만 연결(부호설계 레이어), 실질적 이식은 아님(spatial coupling 미지원) |
| RA 부호 인코딩(accumulate 구조) | 대응 없음 | Prime ECC 인코더는 dual-diagonal 고정, RA accumulate 구조와 인코딩 원리 다름 |

적용 가치: **낮음** — non-QC, RA 기반 spatially-coupled 부호의 BEC 이론 분석 논문으로 HW/디코더 설계가 전혀 없고 부호 프레임워크도 Prime ECC의 QC-LDPC와 근본적으로 달라 이식할 요소가 없다.
