### B. 알고리즘 요약 (10줄 내외)

1. 송신 비트열을 rate `1/2`, block length `2048`의 LDPC로 부호화 후 QPSK 변조, AWGN + 상관 간섭이 있는 채널로 전송.
2. 표준 BP 디코더는 상관성 있는(colored) 채널 간섭에 취약해 성능이 크게 저하되는 문제를 다룸.
3. 채널 간섭은 FIR 저역통과 필터로 생성한 상관 노이즈 모델(`n_k = Σ α_i n_{n-i}`)로 가정.
4. 1차 BP 디코딩 결과 `ŝ`와 수신 신호 `y`의 차 `n̂ = y - ŝ`를 노이즈 추정치로 삼아 CCNN(복소수 CNN) 입력으로 사용.
5. CCNN은 여러 층의 복소 컨볼루션(`c_{i,j} = F(h_{i,j} * c_{i-1} + b_{i,j})`)으로 잔차 노이즈를 줄이는 `ñ`을 추정, `ŷ = y - ñ`을 다시 BP 디코더에 입력.
6. 활성함수는 split-complex 방식 `F(z) = tanh(z_R) + j·tanh(z_I)` 채택(fully-complex 대비 학습 성능 우수).
7. 손실함수는 잔차 노이즈 `r = n - ñ`의 정규화 제곱합 `Loss = (1/2N)‖r‖²`를 사용, Adam + Xavier 초기화로 학습.
8. 네트워크 구조 `3;5,1,9;16,8,1`(3층, 커널크기/피처맵수 명시), 학습 2000블록/테스트 1000블록, 30 epoch 학습.
9. BP(25)-CCNN-BP(25) 구조가 BLER=0 기준 표준 BP 대비 약 1dB 개선, 2단(two-level) CCNN 구조는 SNR 8dB에서 완전 복호 달성(단일 CCNN 대비 큰 개선).
10. HW 설계 없음, 순수 시뮬레이션 결과만 제시하며 실환경에 가까운 채널 모델 검증은 향후 과제로 남김.
