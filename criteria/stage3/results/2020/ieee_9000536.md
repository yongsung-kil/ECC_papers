### B. 알고리즘 요약

1. 시스템: 이진 선형 코드(길이 `N`, `M×N` PC 행렬 `H`) ML 디코딩을 LP relaxation으로 근사하는 문제. AWGN+BPSK 채널, `[96,48]` MacKay LDPC(C1), `[128,64]` CCSDS LDPC(C2)로 실험.
2. 문제: 기존 LP 디코더는 BP 대비 복잡도가 높고 저-SNR 성능이 열세. ADMM-penalized 디코더가 이를 완화했으나 penalty parameter `μ`/penalty function `g(·)`를 수작업 튜닝해야 함.
3. 핵심 가정: PC 제약을 3-변수 제약으로 cascaded 분해(auxiliary variable `x̃`, 선택행렬 `Q_τ`)해 ADMM 표준형(`Au+z=b`)으로 변환, `g(·)`로 pseudocodeword에 페널티 부여.
4. 핵심 기법 1단(LADN): ADMM 반복(`u`-update, `z`-update, `y`-update 3단계, 식 (9a)-(9c))을 K단 신경망으로 unfold, `{α, μ}`(L2 penalty 계수·penalty parameter)를 학습 가능 파라미터로 설정.
5. 핵심 식: `u`-update는 `Π[0,1](η ⊙ (q+A^T(y+μ(z-b))+α/2))`(식 14a) — projection에 학습된 `α,μ`가 직접 관여해 pseudocodeword 억제 강도를 데이터로 최적화.
6. 핵심 기법 2단(LADN-I/LADN-P): LADN-I는 `μ`를 iteration별 `μ_k` 시퀀스로 확장(수렴 개선), LADN-P는 조각별 선형(piecewise-linear) penalty function `g_l(x)`(기울기 `{φ_l}`만 학습)를 도입해 L1/L2보다 유연한 penalty 형상 학습.
7. 부수 트릭: `z`-update는 `ReLU(b-Au-y/μ)`로 표현되어 DL 프레임워크의 활성함수와 동일 형태 — projection 연산을 신경망 레이어로 재해석.
8. 학습 절차: TensorFlow+Adam, training SNR 2dB(cross-validation), C1/C2 각각 stage 50/70 고정, loss는 비지도항(`‖Au+z-b‖²`, residual)과 지도항(`‖F(v)-x‖²`)의 가중합(σ=0.3/0.9).
9. 결과: 저-SNR에서 LADN-I가, 고-SNR에서 LADN-P가 최고 BLER 성능, 원 ADMM-L2 디코더 대비 전 SNR 구간 개선. 계산복잡도는 ADMM-L2와 동일 수준(`O(N+Γa)` 곱셈).
10. 한계: 길이 96~128의 소형 코드만 실험, 긴 코드로 확장 시 학습 복잡도/성능 미검증(저자 자인). HW 미설계, ADMM-LP 프레임워크 자체가 min-sum과 이질적.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| ADMM-penalized LP 디코딩 반복 구조 전체 | 대응 없음 | Prime ECC는 min-sum 기반 message-passing이며 ADMM/LP-relaxation 디코더 아키텍처를 채택하지 않아 구조적으로 대응 모듈 없음 |
| 학습 기반 penalty parameter/function 최적화 | `decoder.cpp` `Get_VNU_Table_Idx()` (LLR 테이블) | 개념적으로 "학습을 통한 파라미터 최적화"는 적응형 LLR 테이블과 유사하나 ADMM penalty와 min-sum LLR은 수학적으로 이질적, 직접 이식 불가 |
| 신경망 학습(TensorFlow, 역전파) | 대응 없음 | Prime ECC는 bit-exact C++ HW 시뮬레이터로 NN 학습/추론 파이프라인 없음 |

적용 가치: 낮음 — ADMM-LP relaxation 기반 deep-unfolding 디코더로, Prime ECC의 min-sum 반복복호 프레임워크와 알고리즘적으로 이질적이며, 짧은 코드(96~128bit)에서만 검증되어 NAND ECC급 긴 QC-LDPC 확장성도 미확인.
