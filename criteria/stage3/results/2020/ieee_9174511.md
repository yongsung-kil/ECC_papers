### B. 알고리즘 요약

1. Multi-Dimensional SC-LDPC(MD-SC-LDPC) 코드는 여러 개의 1D SC-LDPC 체인을 두 번째 차원으로 결합한 code ensemble `C(dl, dr, L1, γ1, L2, γ2, T)`이며 BEC(ε) 채널에서 BP 디코딩을 다룸.
2. 문제의식은 표준 windowed decoder를 그대로 쓰면 window 크기를 모든 constituent 코드에 균일하게(uniform) 줄여야 해서 불필요한 신뢰도 손실이 발생한다는 것.
3. 채널 모델은 binary erasure channel(BEC), coupling은 확률 `T`로 두 번째 차원 결합 여부를 선택하는 랜덤 그래프 구성.
4. 핵심 기법은 non-uniform window size 벡터 `W = [W0,...,W(L2-1)]`를 갖는 windowed decoder이며, targeted VN(TVN) 중심으로 window configuration(WC)을 정의해 BP를 수행.
5. 식 (6)의 window constellation update `z^(l+1)`은 window 밖은 고정, window 안은 density evolution 재귀식 `f(·)`로 갱신하며 반복 `I(it,jt)`까지 진행.
6. 설계 자유도는 (1) WC 처리 순서, (2) 주어진 window complexity `s(W) ≤ C`에서 최적 `W`, (3) window별 반복 횟수 3가지로 정리.
7. Worst-Case WC(Definition 1)를 이용해 실제 성능과 거의 동일한(첫 4자리 일치) 근사 임계값 `δ_worst,W`로 `W` 최적화를 단순화.
8. 최적화 절차는 이론적 density evolution 임계값 계산 기반이며 별도의 학습(GA/NN) 없음.
9. 결과: 동일 window complexity에서 uniform 대비 non-uniform decoder가 평균 반복 횟수 35% 감소(C=42/45 vs C=36), 동일 latency/복잡도에서 uniform보다 높은 채널 erasure probability에서 동작 가능.
10. 한계는 BEC 채널 및 asymptotic(M→∞) density evolution 분석에 국한되며, HW 설계·유한장 실험·AWGN/soft-decision 채널 적용은 다루지 않음.
