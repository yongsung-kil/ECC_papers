> 총평: 저rate 방송용 raptor-like LDPC의 degree distribution 최적화(부호설계) 논문으로, NAND 고rate binary QC-LDPC와 rate 영역·부호구조가 달라 이식 가치는 낮으나 H-matrix 최적화 방법론(GADE+휴리스틱 탐색) 자체는 참고 가능.

### B. 알고리즘 요약

1. ATSC 3.0의 저rate(3/15, 4/15, 5/15) QC-IRA raptor-like LDPC(`q=360`, 코드길이 `qN=16200`), core matrix(QC-IRA, Inner Encoder)와 extended matrix(systematic QC-LDPC, Outer Encoder) 직렬 encoding 구조, AWGN+QPSK 채널.
2. ATSC 3.0의 저rate 코드는 extended matrix의 check-node/variable-node degree distribution이 모두 concentrated(집중형)인데, 이 조합이 저rate에서는 SNR threshold 관점에서 최적이 아니라는 문제를 제기.
3. Gaussian Approximation Density Evolution(GADE)을 두 단계(Inner/Outer Encoder) cascaded 형태로 적용해 평균 LLR 반복식을 유도(all-zero 코드워드 가정).
4. 핵심 기법: check-node degree vector `C`와 variable-node degree vector `V`를 편집(이웃 생성, `neighbor()`)하며 convergence gain function `g(C,V)(s,t) = f(C,V)(mchannel, m^(l-1)) - t`가 모든 `t∈(0,T]`에서 양수가 되도록 SNR threshold `s`를 최소화하는 genetic-like(population=1) 탐색.
5. 핵심 식(5): `m^(l) = (1/C)·φ^-1[1-(1-1/S)^V · φ(mchannel+(V-1)m^(l-1))]^(C-1) - λ1·φ(mchannel)` — Outer Encoder의 평균 LLR 갱신식으로, 이 식의 고정점이 낮을수록 저SNR에서도 수렴함을 의미.
6. neighbor 생성(Algorithm 2)은 degree vector 원소를 ±1 조정하여 Tanner graph edge를 추가/제거하는 방식으로 세대별 후보(C,V) 생성.
7. 부수 트릭: 후보 (C,V) 각각에 대해 PEG(progressive edge growth) 알고리즘으로 prototype graph 생성 후 modified GADE로 예측, 이후 QC구조 PEG로 실제 QC-LDPC H-matrix 생성.
8. 최적화 절차: (C0,V0,s0)=ATSC 3.0 기존값 초기화 → 반복적으로 이웃 후보 중 최선(gain 최대) 선택 → 더 이상 개선 없으면 종료(4단계 heuristic search, 학습이 아닌 결정론적 탐색).
9. 결과: BER 10^-4 기준 ATSC 3.0 대비 rate 4/15에서 0.22dB, rate 3/15에서 0.10dB, rate 5/15에서 0.04dB 이득.
10. 한계: waterfall 영역만 최적화(error-floor·변조 방식 연동은 향후 과제로 명시), HW 미설계, 시뮬레이션(AWGN, QPSK)만 수행, 저rate 특화라 NAND ECC의 고rate 영역과는 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| check/variable-node degree distribution 최적화 (H-matrix 설계) | `ecc_top.cpp` `Load_PCM()` | 부호설계 레이어로 이식 난이도 "높음"(프로파일 §4 명시), 저rate 대상이라 우리 고rate QC-LDPC와 설계 목표 자체가 다름 |
| GADE 기반 SNR threshold 예측 | 대응 없음 | Prime ECC는 bit-exact HW 시뮬레이터로 density evolution 기반 이론적 예측 도구는 보유하지 않음 |
| genetic-like 휴리스틱 탐색(neighbor 생성) | 대응 없음 | H-matrix 자동탐색/최적화 도구 없음 |
| QC-IRA + PEG 기반 QC-LDPC 생성 | `ecc_top.cpp` `Load_PCM()` | PCM 로드만 대응, 생성 알고리즘 자체는 대응 없음 |

적용 가치: **낮음** — 저rate 방송용 raptor-like 부호설계 논문으로 부호설계 레이어(이식 난이도 최고)에 속하고, 우리 NAND용 고rate QC-LDPC와 rate 영역이 정반대라 직접 재사용 어려움.
