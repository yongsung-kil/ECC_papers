### B. 알고리즘 요약

1. IEEE802.11ad 표준의 QC-LDPC(N=672, z=42, rate 1/2·5/8·3/4·13/16), row degree ≤16, column degree ≤4, row-based layer scheduling.
2. 기존 MS/NMS는 C2V 메시지로 제1최소값(m1)과 제2최소값(m2)을 모두 라우팅해야 하며 m2의 낮은 이용률(1/λ)에도 배선 자원을 크게 소모, 라우팅 혼잡이 면적·타이밍 병목.
3. AWGN 채널 + 16-QAM/BPSK, 60GHz mmWave SC 베이스밴드 시스템, BER 요구 3×10⁻⁷.
4. 핵심 기법: CNU는 `m1`만 계산·전송(식 11)하고 VNU에서 `m̂2 = min(m1+γ, 2^(q-1)-1)`(식 14)로 제2최소값을 근사(SAMS: Second-minimum Approximation Min-Sum).
5. 핵심 식: `γ = 2^(q-1) - 1 - m1·σ/2`(식 15) — m1의 신뢰도(크기)가 낮을수록 큰 offset을 동적으로 부여, σ∈[0,1]로 조정, 좌시프트 연산만으로 구현.
6. 확장: shared address generator로 VNU 내 4개 레지스터의 idle(3/4 T) 구간 clock gating하여 스위칭 전력 절감, early termination 병행.
7. 구현 디테일: 3-bit quantization SAMS(3,3), 2-bit comparator로 m2 estimator 단순화, dual-diagonal 아님(QC row-layer 구조).
8. 별도 학습 절차 없음, γ 파라미터 σ는 워드렝스별 heuristic 설계 가이드라인으로 결정.
9. 결과: 28nm 1P9M CMOS, throughput 10.5Gb/s(평균 2.5회 반복 시 최대 21Gb/s), core area 0.14mm², power 81mW@312.5MHz, 면적/에너지 효율 75Gb/s/mm²·10.2pJ/bit; 배선 총 길이 19%, via count 17% 감소(NMS 대비).
10. 한계: mmWave 무선통신(IEEE802.11ad) 전용 설계로 NAND 채널 모델과 무관, 정정능력(waterfall/error-floor) 개선이 아닌 HW 라우팅/전력 최적화가 목적, m2 근사로 인한 BER 열화(최대 0.5dB, CR12는 최대 1.5dB word-length 저감분 포함) 존재.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| CNU의 m1만 계산·전송, m2 근사(SAMS) | [decoder.cpp](../../../Prime_ECC_3.1_Claude) `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | min-sum CN 연산부에 m2 라우팅 제거 및 동적 offset 근사 로직 대체 가능성 있음(단 이미 min1/min2 구조 보유) |
| VNU 동적 offset γ 계산 | `decoder.cpp` `Get_VNU_*` (VN_Cal 계열) | LLR 갱신 시 offset 추가 항 삽입 지점과 대응 |
| VNU idle 레지스터 clock gating(전력) | 대응 없음 | Prime ECC는 bit-exact 시뮬레이터로 실제 clock gating/전력 모델링 대상 아님 |
| C2V 라우팅 네트워크 배선 최적화 | 대응 없음 | Prime ECC 프로파일엔 라우팅/PR(place&route) 레이어 정보 없음 |
| row-based layer scheduling, QC 다중 rate 공유 | `ecc_top.cpp` `Load_PCM()` | QC 구조 자체는 대응하나 다중 코딩레이트 하드웨어 공유 스킴은 Prime ECC 구조(z=32, N_b=129 고정)와 상이 |

> 적용 가치: **낮음** — min-sum CN 연산은 Prime ECC가 이미 min1/min2 방식으로 구현 중복 기여이며, 핵심 기여(m2 근사에 의한 라우팅 배선 감소)는 RTL 배치·배선 최적화 문제로 bit-exact 시뮬레이터 레이어에서는 직접 이식 대상이 아님.
