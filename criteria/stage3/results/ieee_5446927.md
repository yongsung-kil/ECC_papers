> 총평: 심우주용 AR4JA LDPC를 고차 APSK/PSK 변조와 결합한 AWGN 성능 카탈로그 논문으로, NAND binary QC-LDPC(변조 무관) 관점에선 신규 이식 요소가 없어 이식성/추천도 하.

### B. 알고리즘 요약

1. 대상은 CCSDS 표준 `AR4JA`(accumulate-repeat-4-jagged-accumulate) protograph LDPC 9종: rate `{1/2, 2/3, 4/5}` × 입력길이 `{1024, 4096, 16384}`.
2. 문제: 대역제한 심우주 링크에서 BPSK/QPSK 외 8-PSK/16-APSK/32-APSK 고차변조와의 결합 성능 데이터 부재.
3. 채널모델: 임페어먼트 없는 순수 AWGN, 완전 동기, 잔류반송파 제거 가정.
4. 각 변조·비트매핑에 대한 정확 LLR을 식 (14)/(16)으로 유도(등에너지 PSK는 `|r|²,|c|²` 소거).
5. 핵심 근사식 (20): `λj ≈ <r, c*(j,0)−c*(j,1)>/σ²` — 각 비트에 대해 가장 가까운 0/1 성상점만 사용, 곱셈 2회+뺄셈 1회로 모든 변조에 통용.
6. 8-PSK/APSK는 거리계산 대신 각도(φ) 비교로 최근접 성상점 식별, 16-APSK는 Voronoi 경계(수평/수직/45도)로 곱셈 없이 처리.
7. Gray/natural/anti-Gray/DVB-S2 비트매핑 비교(8-PSK에서 natural은 Gray 대비 2.8 dB, anti-Gray는 4.1 dB 손실).
8. 구현: C 소프트웨어(인코더/변조/복조/LLR/복호), LLR 8비트 양자화, codeword 발견 시 조기종료, 최대 200 iteration.
9. 결과: 근사 LLR은 rate 1/2에서 ~0.1 dB, 2/3·4/5에서 0~0.05 dB 손실; hard-decision은 BPSK에서 ~1.6 dB 손실; 50 iteration 이상 성능 포화; APSK ring ratio 민감도 낮음(최적 r2/r1≈3.15).
10. 한계: 신규 알고리즘 없음(성능 카탈로그), 부호는 CCSDS 고정, HW 없이 데스크톱 C 시뮬만.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| AR4JA protograph LDPC 부호구조 | 대응 없음 | Prime ECC는 binary QC-LDPC(base+circulant), AR4JA accumulator 구조 미대응 |
| 근사 LLR (최근접 성상점) | 대응 없음(변조 종속) | Prime ECC LLR은 channel.cpp Set_LLR_Th()로 NAND read 기반, 변조 심볼 LLR과 무관 |
| 8비트 LLR 양자화 message-passing 복호 | decoder.cpp LDPC_Decoder() | 개념적 유사(8비트 양자화 BP)이나 min-sum 아닌 정확 LLR, 이식 이득 없음 |
| 조기종료(codeword 발견 시 중단) | partial_CRC.cpp / full_CRC.cpp | 유사 개념이나 CRC 기반 조기종료를 이미 보유 |
```

적용 가치: 낮음 — 고차변조 성능 카탈로그이며, NAND는 변조가 없어 근사 LLR·성상 매핑이 직접 적용되지 않고 부호/복호도 이미 보유 기법과 중복.
