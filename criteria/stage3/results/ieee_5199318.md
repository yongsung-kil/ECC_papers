### B. 알고리즘 요약

1. 대상은 packet 기반 LDPC (`C(n, λ, ρ)`), 각 변수노드가 l=32비트 packet(symbol)을 표현, BSC 채널.
2. Luby-Mitzenmacher의 verification-based decoding(VA)을 BSC에 적용할 때 false verification(오검증) 확률을 무시한 기존 해석이 실제 FER을 과소평가하는 문제를 다룸.
3. 채널은 이진 대칭 채널(BSC)로 크로스오버 확률 `p0`를 가정, 각 반복(iteration)에서 비트오류확률 `pk`가 갱신되는 마르코프적 모델.
4. VA는 verification 단계(check node 합이 0이면 이웃 변수노드 검증)와 correction 단계(미검증 노드 1개만 남으면 나머지 검증된 값의 합으로 정정)로 구성, 두 단계 모두 메시지 패싱 4-step으로 재정식화.
5. 핵심 식 `FER(m) = Q_f^(1) + Σ Q_f^(k) Π Q_u^(j) + Π Q_u^(k)`: false verification 사건과 unverification 사건의 누적확률로 m회 반복 후 FER을 산출.
6. `Q_f^(k)`(false verification 확률)과 `Q_c^(k)`(정정 완료 확률)을 노드 독립성 가정 하에 edge-perspective 확률(`qcv`, `qfv`)로부터 유도.
7. 오류 구조는 degree-i check node에 연결된 인접 변수노드들의 error pattern을 integer partition으로 열거해 계산(소수 오류비트만 고려, 대량 오류는 확률 무시).
8. 별도 학습/최적화 절차 없음 — 폐형(closed-form) 반복 확률식으로 계산.
9. rate-1/2 regular-(3,6), n=1008, packet=32bit 코드 50개 무작위 생성 실험: false verification 고려 시 시뮬레이션과 FER이 같은 오더로 일치(미고려 시 3자리 이상 과소평가).
10. 한계: BSC 및 소수 오류비트 근사(대량 오류 무시)에 국한, HW/디코더 구현 없이 순수 시뮬레이션·이론 검증뿐이며 NAND binary LDPC의 min-sum 계열과 무관한 packet-symbol 채널 모델.
