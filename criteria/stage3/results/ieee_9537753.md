### B. 알고리즘 요약

1. 시스템: MLC NAND flash 채널(program noise+RTN+retention noise), 4 threshold-voltage state {11,10,00,01}, 레벨 {1.4,2.6,3.2,3.93}V. ECC는 rate-`9/10` regular CW-3 protograph LDPC, length `4000`, BP 디코딩 max iter 25.
2. 문제: 기존 read-voltage/LLR 설계(uniform quant, MMI)는 threshold-voltage PDF를 정확히 안다고 가정 → 별도 채널추정 필요, PE/retention 변화에 취약.
3. 모델: 4개 state PDF를 가우시안 `psA(v)`로 모델, LLR은 read voltage 구간 적분식 `Lmsb`/`Llsb` (식2,3).
4. 핵심기법 1단(CNN detection): `T-1=2` 1D conv layer(filter 5, ReLU)+FC, 입력 read-back 전압열 `v(K=50)`, softmax로 4 state a-posteriori `PNN(s|vk)` 출력. Adam+cross-entropy 학습.
5. 핵심 식(6): 구간 `Vn`을 M등분해 average a-posteriori `PNN(si|Vn)=Σ PNN(si|qnm)/M`로 근사 → 구간 내 전압 효과 반영, 단일 양자화값 근사보다 정확.
6. LLR 산출(식7,8): 평균 a-posteriori로 MSB/LSB LLR 계산, 추론 후엔 CNN 없이 LLR 테이블만으로 read → read latency 절감.
7. 핵심기법 2단(read-voltage 설계): coded bit과 LLR 간 mutual information `I(L,b)`(식9) 최대화로 hard read-voltage `{t1*,t2*,t3*}`를 bisection search로 도출(식10).
8. 확장(soft): hard read-voltage 중심 overlap 영역 폭 `{W1*,W2*,W3*}`을 MI 최대화로 최적화해 soft read-voltage 6개 생성(식11).
9. 결과: RNN-aided 및 uniform-quant보다 우수, mismatch 없을 때 optimal MMI에 근접. PE/retention mismatch 커지면 성능 저하 → 재학습/재호출 필요.
10. 한계: HW 미설계·시뮬(keras+MATLAB)만, 디코더 자체는 무변경(채널/LLR 단), NN 학습셋·mismatch 재학습 부담.
