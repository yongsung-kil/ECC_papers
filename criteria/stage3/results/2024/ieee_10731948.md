### B. 알고리즘 요약

1. 시스템: 5G NR LDPC 표준 base code BG2(42x52 parity-check, rate 0.2, lifting factor z=10)를 이진 대칭 채널(BSC)에 적용, code length N=520(52x10), message length K=100(10x10)으로 생체인식 키 생성(fuzzy commitment)에 사용.
2. 문제: 생체측정치(얼굴 특징)의 인트라클래스 오류율(~30%)과 ECC 정정능력 사이 noise gap이 커서, 기존 handcrafted MS 디코더나 BCH 등으로는 성능 저하 발생.
3. 채널 모델: AWGN이 아닌 BSC(bit crossover rate로 표현되는 비트반전) 채널을 가정, 원 neural-MS([49] Dai et al.)를 AWGN에서 BSC로 재설계.
4. 핵심 기법: min-sum CN 업데이트에 학습형 정규화 계수 `α`와 오프셋 `β`를 결합한 `ℓ_c→v = sign(·) × max(α×min|ℓ_v→c| − β, 0)` (식 14) 형태의 neural-MS 디코더.
5. 핵심 식의 의미: 표준 MS(식 11)는 정확도가 SP(sum-product) 대비 낮고, SP는 tanh 연산으로 계산량이 큼; α(정규화)·β(오프셋)를 데이터로 학습해 SP에 근접하는 정확도를 MS 수준 복잡도로 달성.
6. 확장: 신경망은 VN/CN 업데이트를 trellis 구조로 펼친 partially-connected 구조(hidden layer i = i번째 iteration, sublayer iv/ic)이며, gradient vanishing 회피를 위해 iteration(layer) 단위로 greedy 순차 학습(한 레이어 학습 후 가중치 고정 후 다음 레이어 학습).
7. 부수 트릭: 얼굴 특징을 실수 벡터→양자화(LSSC, q=4 구간)→이진화 후, AdaMTrans라는 랜덤 마스킹(bitwise AND with mask r)으로 인트라클래스 해밍 거리를 LDPC 정정능력 범위 내로 줄임(비-LDPC 파트, 부호/디코더와 무관).
8. 학습 절차: 5G 표준 BG2 H-matrix 기반으로 iteration-by-iteration greedy layer-wise 학습(층별 순차 최적화, 이전 층 가중치 고정).
9. 결과: BSC crossover rate 기준 FER<5%를 만족하는 최대 crossover rate가 MS 15.5% → neural-MS 17.62% → SP 18.52%로 개선; 키 검색 정확도는 SP 대비 약 0.4%p 우세, MS 대비 3.42%p 우세(평균 81.33%).
10. 한계: 원 기법([49])의 채널을 BSC로 바꾼 응용이며 HW 설계/합성 없이 시뮬레이션(GPU)로만 검증, 정규화/오프셋 min-sum 자체는 이미 문헌에 존재하는 기법으로 신규성은 낮음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Neural-MS (학습형 정규화 α + 오프셋 β min-sum) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC의 min-sum CN 연산에 α/β 학습 파라미터를 대입하는 형태로 이식 가능, 단 Prime ECC는 이미 magnitude 양자화 테이블 기반 min-sum을 보유해 중복 기여 |
| Iteration별 학습 파라미터(greedy layer-wise) | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | iteration별 적응형 LLR 테이블과 개념적으로 유사한 위치, α/β를 iteration별 테이블처럼 저장 가능 |
| BSC 채널 가정 / 생체인식 fuzzy commitment 응용 | 대응 없음 | NAND 채널(RBER/soft-read)과 무관한 별도 응용 도메인 |
| AdaMTrans(양자화·이진화·랜덤마스킹) | 대응 없음 | 생체 특징 벡터 전처리로 부호/디코더 레이어가 아님 |

적용 가치: `중간` — neural-MS의 학습형 α/β min-sum 개념은 Prime ECC의 min-sum 파라미터 튜닝에 참고 가치가 있으나, 기존 min-sum+LLR 테이블 구조와 상당 부분 중복되고 코드 자체(5G BG2, rate 0.2)와 채널(BSC) 모두 NAND용 고rate QC-LDPC와 거리가 있어 직접 이식보다는 개념적 참고 수준.
