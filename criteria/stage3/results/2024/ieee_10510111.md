### B. 알고리즘 요약

1. QC-LDPC(16×32 PCM, rate≈0.5, codeword 32-bit) 또는 MN-LDPC(비체계적)로 부호화한 128-bit 메시지를 BPSK 변조하여 AWGN 채널로 전송.
2. 기존 반복 디코더(BF/BP/SPA)는 반복 횟수만큼 지연이 발생해 이미지 등 실시간 전송에 부적합하다는 문제의식.
3. 채널은 단순 AWGN, SNR 0~9dB 구간에서 성능 비교.
4. 핵심 기법은 Tanner 그래프 구조를 모사한 MSNN(Multilayer Shallow Neural Network): hidden layer1=check node 수(tansig), hidden layer2=variable node 수(logsig), hidden layer3=check node 수(tansig), 출력은 max-pooling.
5. 수신 LLR `LLR(x) = log(P(x=0)/P(x=1))`을 입력으로, 학습된 가중치로 1회 순전파해 비트 확률을 직접 출력(반복 없음).
6. 확장이 필요하면 MSNN 디코더를 cascade하여 메시지 길이를 늘리는 구조.
7. 학습 데이터: 이미지 픽셀(0~255)을 8-bit화 후 16-bit로 패딩, SNR 0/2/5/7dB에서 생성한 데이터로 70/15/15% train/test/validation 분할.
8. 학습 절차는 MATLAB 신경망 툴박스 기반 supervised 학습(구체적 옵티마이저 명시 없음).
9. 결과: 디코딩 시간은 SPA(3dB) 28.5분→MSNN 3.92분으로 단축되었으나, BER은 3dB에서 10^-3배, 7dB에서 10^-1배 SPA보다 열등.
10. HW 미설계, 소규모(32-bit codeword) 시뮬레이션에 국한되며 대형 NAND급 코드로의 확장성·정정능력은 검증되지 않음.
