### B. 알고리즘 요약

1. 시스템: LDPC(576, rate 3/4), BPSK 변조, 수신 `y = s + n`, n은 상관계수 `η`를 갖는 가우시안 상관 노이즈(`n = Θ^(1/2) n_w`).
2. 문제: 필터링/오버샘플링 등으로 발생하는 상관 노이즈는 통계적 모델링이 어려워 기존 pilot/필터 기반 기법의 성능이 제한됨.
3. 가정: 노이즈는 zero-mean 가우시안이며 상관 행렬 `Θ`(η 기반 Toeplitz 구조)를 따름.
4. 핵심 기법: Unet 구조(1D conv 기반) 신경망 denoiser(UnetNND)가 수신 신호 y로부터 노이즈 추정치 `n̂`을 산출, `ŝ = y - n̂`으로 보정 후 LLR 계산해 BP 디코더에 입력.
5. 핵심 식: `LLR_i = 2ŝ_i/σ²` — denoiser가 codeword가 아닌 노이즈 자체를 추정하는 이유는 노이즈 분포가 codeword/LLR 분포보다 단순하기 때문.
6. 확장: 지도학습 모드(노이즈 라벨 필요, SNR mismatch 시 성능 저하) + 자기지도학습 모드(mask-and-combine 기법으로 온라인 학습, 라벨 불필요).
7. 구현 디테일: 자기지도 학습은 fragment 단위(l=4)로 비트를 마스킹해 My를 생성하고, 마스킹된 위치만 결합(C(·))해 손실 계산 — identity mapping 학습 방지.
8. 학습: 손실함수 `L = L_mse + α·L_reg`, L_mse는 invisible/visible part 가중합(β), L_reg는 정규화 항으로 학습 불안정성 보정.
9. 결과: η=0.8에서 BER 1E-5 기준 iterate BP-CNN 대비 약 1.1dB, ResCNN 대비 약 0.48dB 성능 향상. 자기지도 모드는 지도 모드 대비 근소하게 낮으나 유사한 성능.
10. 한계: HW 미설계, 시뮬레이션(합성 데이터)뿐, 실측 노이즈 미검증, 계산 복잡도/자원 분석 없음, zero-mean 노이즈 가정에 의존.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Unet 기반 NN denoiser (전처리) | 대응 없음 | Prime ECC는 NN 기반 전처리/디코더를 갖지 않음 |
| BP 디코더 본체 | decoder.cpp LDPC_Decoder() | Prime ECC는 min-sum 기반이며 BP(sum-product)와 다른 근사 방식 사용 |
| soft-read LLR 계산 | channel.cpp Set_LLR_Th() | LLR 산출 개념은 유사하나 논문은 AWGN 상관노이즈 기반, Prime ECC는 NAND RBER 기반 |
| 자기지도 온라인 학습(mask-and-combine) | 대응 없음 | Prime ECC에 온라인 학습 파이프라인 없음 |
```

적용 가치: **낮음** — NN 전면 도입 기반 전처리 구조이며 통신 채널 상관 노이즈 대상으로, Prime ECC의 binary QC-LDPC min-sum HW 구조 및 NAND read 노이즈 모델과 부합하지 않음.
