### B. 알고리즘 요약

1. TLC NAND(3bit/cell, 7개 state 경계)에서 soft decision LDPC 복호를 위한 sensing level 배치를 최적화하는 문제, 목표는 read latency 감소.
2. 기존 nonuniform sensing 기법들은 state 간 차이(interstate)만 고려하거나 균등 간격을 유지해 state 내부(intrastate) 비균일성을 반영하지 못함.
3. threshold voltage 분포를 Gaussian으로 가정하고 인접 state 중첩영역의 경계값 `vl_i`, `vr_i`를 두 분포 교점 조건 `f(i)(v)/f(i+1)(v)=R`로 계산.
4. 엔트로피 `H(v) = -Σ P(state=i|v) log2 P(state=i|v)` 로 중첩영역 내부 위치별 혼란도(intrastate)를 정량화 — 중첩영역 중심에서 최대, 경계에서 최소.
5. 엔트로피가 클수록(중심부) sensing level을 촘촘히, 경계부는 성기게 배치(EBISN, intrastate 비균일 sensing).
6. 7개 중첩영역 각각의 면적 `S(j)`을 적분해 영역별 엔트로피 `H(S)`를 계산, 엔트로피 순위(예: 3>6>1>2>5>4>0)로 sensing 우선순위·횟수를 영역별 차등 배분(EBDN, interstate 비균일 sensing).
7. 두 기법을 결합한 EBDN: 엔트로피 상위 2개 영역만 soft sensing(2회) 적용, 나머지는 hard decision만 사용해 총 sensing 횟수 축소.
8. P/E cycle·retention time 구간별 offline 테이블(Table I, Search 1-4)을 사전 구축(측정+피팅), 실제 read 시엔 룩업만 수행해 latency 오버헤드 없음.
9. 결과: min-sum LDPC(rate 8/9, max 30 iter) 기준 기존 nonuniform sensing 대비 반복횟수 약 70% 감소, read latency 34.52% 감소, 저장 오버헤드 110 bytes(256Gb 칩 대비 무시 가능).
10. 한계: Micron B16A 칩 실측 데이터 기반이라 타 공정/제조사 이식 시 재보정 필요, decoder HW 자체는 변경 없이 sensing 정책만 개선(디코더 알고리즘 미설계).
