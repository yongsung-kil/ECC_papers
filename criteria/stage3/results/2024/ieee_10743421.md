### B. 알고리즘 요약

1. Uplink PDMA(pattern division multiple access) 시스템에서 각 사용자는 rate `Rc` LDPC 부호로 인코딩 후 sparse pattern matrix(PM)를 통해 OFDM RE에 매핑되며, overload `β=K/N`이 변함.
2. 문제: nested PM 구조(예: `S4,6 ⊂ S4,8`)에서 overload마다 EXIT 특성이 달라 단일 고정 LDPC 부호(WiMax 표준, rate 0.5, length 2304)로는 성능이 저하됨.
3. 채널/간섭 모델은 BP(belief-propagation) 기반 PDMA 검출기와 LDPC 복호기 간 iterative detection and decoding(IDD), a-priori LLR은 가우시안 근사 `La(ci) = μ·xi + n0`로 모델링.
4. 핵심 기법은 check-concentrated degree distribution: check-node degree를 1~2개로 고정(`ρ_q=1.0`)하고 variable-node degree distribution `λ`만 최적화 변수로 둠.
5. EXIT 매칭 문제를 손실함수 `Δ(λ,Eb/N0)=min[f1(I)-f2^-1(I)]` 최소화로 정의, 이를 `max Σλp/p` 형태의 선형계획법(LP)으로 변환해 풀이.
6. Module I(BP검출기+VND)과 Module II(CND)의 EXIT 곡선을 각각 유도 후 매칭 — module II는 duality property로 해석적 계산.
7. 결과 부호(IT++ 라이브러리로 구성)는 최대 check-node degree `Dc,max`가 WiMax-LDPC(7)보다 낮은 6으로 설계되어 복호 복잡도도 소폭 감소.
8. 최적화 절차는 EXIT chart 기반 LP(학습/GA 아님), degree distribution pair `(λ,ρ)`와 임계 `(Eb/N0)*`를 출력.
9. 결과: EXIT 예측 기준 약 1dB 이득(S4,8, rate 0.5), BER 시뮬 기준 BER=1e-2에서 0.2~0.4dB 성능 이득(rate 0.5/0.75 전 PM 조합).
10. 한계: HW 미설계, 순수 시뮬레이션/EXIT 이론 검증뿐, PDMA 무선 다중접속 특화 채널(가우시안 fading + BP 검출기 결합) 가정이라 NAND 채널과 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Check-concentrated degree distribution 최적화(LP) | `ecc_top.cpp` `Load_PCM()` | 특정 5G/PDMA 채널·검출기에 결합된 최적화라 이식 난이도 높고 대응 근거 약함 |
| EXIT chart 기반 부호 설계 절차 | 대응 없음 | Prime ECC는 고정 H-matrix 사용, EXIT 기반 자동 설계 파이프라인 없음 |
| BP-detector와 LDPC 결합 IDD(반복 검출-복호) | 대응 없음 | PDMA 다중접속 검출기는 NAND ECC와 무관한 무선통신 요소 |
| 표준 BP(Sum-Product) 디코더 | `decoder.cpp` `LDPC_Decoder()` | 논문은 표준 BP만 사용(30 iteration), Prime ECC의 min-sum 대비 개선 아님 |

> 적용 가치: 낮음 — PDMA 비직교 다중접속이라는 5G 무선통신 특화 문제이며, NAND 채널·HW와 무관한 부호설계 최적화라 이식 요소가 사실상 없음.
