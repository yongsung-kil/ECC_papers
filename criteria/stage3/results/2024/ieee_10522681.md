### B. 알고리즘 요약

1. 시스템: array-based spatially coupled LDPC convolutional code(SC-LDPC-CC), sliding window(SW) decoding 사용, base matrix `B`를 `w+1`개 component matrix로 분해해 coupling.
2. 문제: 기존 4-cycle free SC-LDPC 구성법([1])은 heuristic search + check-and-flip 랜덤 방식이라 base matrix 크기가 커질수록 복잡도가 급증하고 최소 coupling width `w` 산출이 비효율적.
3. 핵심 가정: exponent matrix `E`(spreading matrix)를 QC-LDPC의 exponent matrix로 보고, `E`의 4-cycle free 여부와 이로부터 유도된 SC-LDPC-CC PCM `H(I,E)`의 4-cycle 존재 여부 사이의 관계를 규명(`E` 정의: `e_{i,j}` 정수 성분, incidence matrix `M_e`).
4. 핵심 기법: `good sequence` `(a_n^{(p,q)})_{1≤n≤p+q-1}` 정의(조건: `a_{n1}-a_{n2}+a_{n3} ≠ a_{n1-n2+n3}`) — 이 수열로 `e_{i,j}=a_{j-i+p}^{(p,q)}`를 만들면 `E`가 자동으로 4-cycle free(Theorem 2).
5. 핵심 식: `Theorem 1` — `I=[i0,iw]` (연속 구간)일 때 `H(I,E)`가 4-cycle free ⟺ `E`가 4-cycle free. 즉 primary matrix `E`의 성질만 확인하면 전체 PCM의 girth(≥6)를 보장.
6. 핵심 기법 2단: `I`가 비연속(불규칙 간격)일 경우를 위한 `Theorem 3`(representative block matrix `BR(E)`의 2×2 부분행렬 조건)으로 girth 조건 일반화.
7. 부수 트릭: back-tracking 알고리즘(Algorithm 2/3)으로 최소 MOE(coupling width `w`)와 최소 (6,8)-cycle 수를 갖는 good sequence `A_{p,q}`를 탐색.
8. 최적화 절차: 주어진 `p,q`에 대해 `w`를 1부터 증가시키며 good sequence 집합 `S(w,p,q)`가 공집합이 아닐 때까지 탐색 후, `N_{6,8}(H)` 최소인 수열 선택.
9. 결과: `(p,q)=(3,6)`, `I=[0,3]`, lifting `m=100`인 QC-SC-LDPC(rate 0.485, length 60000)에서 `N_{6,8}=13720`(기존 [1]은 14346), BER 시뮬(SW decoding, window 12/20)에서 기존 [1],[2],[6],[12] 대비 동일 rate/length/window에서 소폭 우수.
10. 한계: 순수 조합론적 구성법으로 하드웨어/디코더 설계 없음, 예제는 저rate(0.485, 4/7) 중심이며 girth-6 보장이 핵심이라 고rate·특정 degree 분포 요구 시 적용성 미검증.
