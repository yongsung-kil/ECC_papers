### B. 알고리즘 요약

1. 시스템: CV-QKD(연속변수 양자키분배)의 정보조정(information reconciliation) 단계, AWGN 가정 classical channel, SNR이 평균 6dB 기준 ±1dB(5~7dB) 변동.
2. 문제: 고정 rate LDPC는 시변 SNR 환경에서 key rate가 최적이 아니며, 다양한 rate별 LDPC 코드 테이블을 저장하는 것은 비용이 큼.
3. 핵심 가정: 비밀키율 `k=(1-FER)(β·I_AB - χ_BE)`이며 β=R/C(reconciliation efficiency), χ_BE(Holevo bound)는 단순화를 위해 시뮬에서 무시.
4. 핵심 기법: Q-learning 기반 온라인 강화학습으로 상태(SNR), 행동(LDPC rate `R`), 보상(secret key rate `k(R,SNR)`)을 정의해 code rate를 동적 선택.
5. 핵심 식: Q-value 갱신식 `Q(s,a) ← Q(s,a) + α[r + γ·max Q(s',a') - Q(s,a)]`(식 5) — FER과 β의 상충 트레이드오프를 데이터 기반으로 학습.
6. 확장: ε-greedy 탐색(exploration ε=1→exponential decay 0.998/episode)으로 탐색-활용 균형.
7. 구현 디테일: LDPC 부호 자체(regular, length 1000bit, rate range 0.02~0.15, 정밀도 0.005)는 고정 구성이며 논문 초점은 부호설계가 아닌 rate selection 정책.
8. 학습/최적화 절차: 5000 episode 학습, α=0.1, γ=0.9, episode당 1~100 step, Q-table 0으로 초기화.
9. 결과: adaptive rate가 고정 rate(0.02/0.09/0.15) 대비 대부분 SNR 구간에서 secret key rate 우위(Fig.3, Fig.6). FER threshold 50% 설정.
10. 한계: 부호설계/디코더 알고리즘 개선 없음, HW 미설계, χ_BE 생략으로 실측 key rate 아님, 부호 rate가 0.02~0.15로 NAND ECC(고rate)와 전혀 다른 영역.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Q-learning 기반 rate 선택 정책 | 대응 없음 | Prime ECC는 부호 rate가 H-matrix에 고정되어 런타임 rate 적응 개념이 없음 |
| LDPC 코드(regular, 극저rate) 자체 | 대응 없음 | 부호설계 세부 미제시, NAND 고rate와 무관 |
| CV-QKD 정보조정 시스템 모델 | 대응 없음 | NAND ECC 도메인과 무관한 양자통신 시스템 레벨 논문 |

적용 가치: **낮음** — LDPC 부호설계·디코더·HW를 전혀 다루지 않는 CV-QKD 시스템 레벨 rate-selection 강화학습 논문으로 Prime ECC 이식 대상 자체가 없음.
