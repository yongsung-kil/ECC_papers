### B. 알고리즘 요약 (10줄 내외)

1. AWGN 채널에서 서로 다른 파워레벨 `Pℓ`로 BPSK 변조된 `L`개(2~3개)의 LDPC 코드 컴포넌트를 non-orthogonal superposition으로 합쳐 전송(`c = Σ Pℓ fℓ(mℓ)`), 최종 lifted 코드는 `k=5544, n=6006`.
2. SPARCs(sparse regression code)는 error-reducing 성능은 좋으나 복호 복잡도가 커서 실용성이 낮아, 저복잡도 LDPC 기반 대안을 설계하려는 것이 목적.
3. Error-reducing code는 concatenated FEC의 inner code로서, outer HDD 코드(Staircase, OH 6.25%)가 필요로 하는 낮은 입력 BER(목표 `D=4.7×10^-3`)을 달성하는 것이 임무.
4. 핵심 기법 1단: soft Successive Interference Cancellation(SIC) 디코더로 각 컴포넌트를 순차 디코딩하며 다른 컴포넌트의 간섭을 LLR 기반으로 추정·제거(Algorithm 1), 간섭 `ρℓ = Σ Pj·tanh(γj/2)`.
5. 핵심 식: SINR 재계산 `γ(in)ℓ = 2(y-ρℓ)×√SINR/Pℓ`로 매 outer iteration마다 입력 LLR을 갱신 — 간섭 추정 정확도가 전체 성능을 좌우.
6. 핵심 기법 2단: 양자화된 density evolution(quantized DE)을 proto-graph 위에서 수행, LLR 분포를 `2J+1`개 이산점 p.m.f.로 근사(클리핑 `γmax`)해 waterfall과 error-floor를 동시에 예측.
7. 부수 트릭: variable/check 노드 연산에 chain rule(결합법칙)을 적용해 계산량 절감, check node는 normalized min-sum(scaling factor 0.75) 사용.
8. 최적화 절차: 파워분배 `{Pℓ}`와 base parity-check matrix `{Hb,ℓ}`를 번갈아 최적화 — PCM은 그리디 랜덤 disturbance(비트 flip) 탐색으로 개선.
9. 결과: 2-컴포넌트 구성이 단일 컴포넌트 대비 목표 BER(`D=4.7×10^-3`)에서 약 0.9dB 이득(2.47dB vs 3.39dB), lifting factor를 절반으로 줄여도 성능 손실 0.1dB에 불과.
10. 한계: HW 미설계, 시뮬레이션만 수행, superposition 구조 특성상 error count 분포가 이항분포보다 heavy-tail이라 concatenated 스킴에서 성능 저하 우려(향후 과제로 남김).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Normalized min-sum (scaling 0.75) | `decoder.cpp` `CNU_Update_New_Mag()` | Prime ECC도 min-sum 기반이나 이미 보유 기법(§3)이라 신규 기여 아님 |
| Quantized DE (LLR p.m.f. 이산화, 최적화) | `ecc_top.cpp` `Load_PCM()` (H-matrix 선정 관련) | H-matrix 최적화 도구라는 점에서만 개념적 관련, 실제 코드 내 DE 최적화 파이프라인은 대응 없음 |
| Soft-SIC 간섭제거 (다중 컴포넌트 결합) | 대응 없음 | Prime ECC는 단일 채널 단일 코드 구조로 superposition/SIC 개념 자체가 적용 대상이 아님 |
| PCM 그리디 랜덤 탐색 최적화 | `ecc_top.cpp` `Load_PCM()` | PCM 구조 최적화라는 목적은 유사하나 논문의 탐색 알고리즘 자체는 코드에 미구현, 이식 시 높은 난이도(§4 기준 "부호설계" 항목) |

> 적용 가치: 낮음 — superposition coding은 NAND의 단일채널 binary LDPC 구조와 근본적으로 다른 시스템 모델이며, 핵심 기여(SIC, 다중 컴포넌트 파워분배)가 NAND 이식 대상이 아님. 다만 quantized DE의 error-floor 예측 방법론은 향후 QC-LDPC H-matrix 설계 참고자료로 간접 가치 있음.
