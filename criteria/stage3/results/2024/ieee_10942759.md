### B. 알고리즘 요약

1. (1536,1280) protograph 기반 irregular QC-LDPC 코드(4×24 base graph, lifting Z=64) 7종을 설계, BPSK+AWGN 채널에서 NMS(normalized min-sum, scaling factor 0.75) BP 디코더로 평가.
2. 문제: 반도체 공정 미세화가 둔화되며 배선(interconnect) 에너지 비중이 커져, Tanner graph의 locality(국소성)와 randomness(무작위성)가 에너지-성능 트레이드오프에 미치는 영향을 정량화할 필요.
3. permutation matrix `B_k`의 원소를 정수집합 `Ω_k = {i∈[0..63] | i mod k = 0}`로 제한해 동일 노드 차수분포·에지수(5120)를 유지한 채 `k` 클러스터 수만 다른 코드 앙상블 `L={L1,...,L64}` 생성 (PEG 알고리즘으로 사이클 최소화).
4. `L64`(64개 독립 클러스터, 최고 locality) ~ `L1`(1개 클러스터, 최고 randomness)까지 스펙트럼을 구성, fully node-parallel 아키텍처(CFU/VFU + VN-CN/CN-VN 라우팅망)로 12nm FinFET/22nm FD-SOI 두 공정에 합성.
5. 채널값·메시지 6bit 양자화, 목표 600MHz/utilization 60%, I=8 iteration 고정.
6. 결과: 랜덤성이 높을수록(클러스터 적을수록) FER 개선(사이클 감소·메시지 교환 향상)되지만, combinational logic 에너지가 최대 2.18배 증가해 전체 에너지가 6.20→10.20 pJ/bit로 1.65배 악화(12nm 기준). 면적 효율은 랜덤성과 무관.
7. Polar 코드 파트(별도 실험, LDPC와 무관)는 CA-Fast-SSCL-8부터 ExPOS Rate-1/SPC node, SCAL, row-merged code까지 5단계 cross-layer 최적화를 12nm/22nm에 구현, 동일 throughput(64Gbit/s) 기준 비교.
8. Cross-layer 최적화만으로 면적효율 10.4~10.7배, 에너지 9.6~10.0배 개선을 달성한 반면, 공정 미세화(22nm→12nm) 단독 효과는 면적효율 1.7배·에너지 개선은 미미(cross-layer가 압도적으로 지배적).
9. 결론: LDPC는 H-matrix 랜덤성-에너지 트레이드오프를 최초로 정량화, polar는 알고리즘/코드/아키텍처 공동최적화가 공정 미세화보다 훨씬 중요함을 실증.
10. 한계: 새로운 디코딩 알고리즘이나 HW 기법 제안이 아닌 기존 구조(NMS BP, fully unrolled SCL) 기반의 설계공간 탐색 논문이며, LDPC 실험은 fully node-parallel(고throughput/고면적) 구조에 한정되어 NAND 컨트롤러의 저면적·저전력 요구와는 최적화 방향이 다를 수 있음.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Tanner graph 랜덤성-에너지 트레이드오프 분석 | `ecc_top.cpp` `Load_PCM()` | H-matrix 구조 설계 시 라우팅 에너지/면적 참고자료가 될 수 있으나, Prime ECC는 이미 QC구조(base+circulant, z=32)로 고정돼 있어 직접 재설계 불가 |
| fully node-parallel 디코더 아키텍처(CFU/VFU 라우팅) | 디코더 HW 모델(z=32, HCU, GT) | Prime ECC는 row 단위 병렬(z_sb=32)+HCU/GT로 다른 병렬화 전략을 이미 채택, fully-parallel 구조와는 설계축이 다름 |
| NMS(normalized min-sum) BP 디코딩 | `decoder.cpp` `CNU_Update_New_Mag()` | Prime ECC도 이미 min-sum 기반, scaling factor 관련 참고 가능하나 신규 기여 아님 |
| polar SCL/SCAL/row-merged 최적화 | 대응 없음 | polar code 전용, binary QC-LDPC와 무관 |
```

적용 가치: **중간** — 새로운 알고리즘·모듈 이식 대상은 아니지만, H-matrix 랜덤성이 배선 에너지에 미치는 정량적 통찰은 Prime ECC의 향후 H-matrix/라우팅 설계 시 트레이드오프 참고자료로 가치 있음.
