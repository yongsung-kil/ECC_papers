### B. 알고리즘 요약

1. 대상: protograph 기반 (QC-)LDPC 부호, girth 6/8, column weight `γ`, lifting degree `N`, exponent matrix `B`.
2. 문제: 기존 trapping set 제거법은 all-one base(γ=3,4)에만 적용되고 irregular/multi-edge에 부재하며, dmin에 대한 해석적 결과가 없음.
3. 핵심 개념: cycle의 chord = 사이클 밖에서 두 정점을 잇는 edge. chord 있는 사이클 = `cycle-wc`. 짧은 cycle-wc가 여러 ETS의 뿌리임을 증명.
4. 이론1: girth 6에서 `8-cycle-wc`를 없애면 dmin 하한이 `γ+1`→`2γ`로 상승 (Theorem 3).
5. 이론2: girth 8에서 `12-cycle-wc`를 없애면 `dmin ≥ 3(γ-1)^2 / (γ ln γ - γ + 1)` (Turan 정리·독립수 이용, Theorem 4).
6. 근거식: 2k-cycle 존재조건 `Σ(b_rmi - b_rmi+1) ≡ 0 (mod N)` (Eq.1)을 chord 조건과 결합해 필요충분조건 유도.
7. 구성법: simple/compact/array-based/multi-edge protograph 각각에 8-cycle-wc, 12-cycle-wc 제거 필요충분조건을 exponent matrix 엔트리 제약으로 제시(Algorithm 1).
8. 복잡도: 8-cycle-wc 제거가 all 6-cycle 제거(girth 상승)보다 연산 수(+,-,×,≡) 적음(Table II/III), 12-cycle-wc 제거는 ETS 제거법(M1,M2)보다 적음(Table VIII).
9. 결과: sum-product 50 iter, AWGN/BPSK. C1/C5/C7 등 cycle-wc-free 부호가 counterpart보다 유의하게 우수(예: (3,18)-regular rate 0.835, (4,12) rate 0.7397); girth-8 대비 유사 성능을 짧은 길이로 달성.
10. 한계: HW/디코더 설계 없음, 순수 부호구성·이론; Raptor-like 저rate에서는 이득이 사라지거나 역전되는 혼재 결과.
