### B. 알고리즘 요약 (10줄 내외)

1. 시스템: 이진 sparse source(psrc)를 LDPC(n,k)로 부호화 후 BPSK 변조, AWGN에 상관계수 `η`를 갖는 상관노이즈가 추가된 채널(WiMAX LDPC(576,432), CCSDS LDPC(128,64) 사용).
2. 문제: 실제 채널 노이즈는 필터링·오버샘플링으로 상관성을 가져 BP 디코딩 수렴을 저해하고, 짧은 코드의 Tanner graph short cycle이 BP 성능을 추가로 열화시킴.
3. 가정: 상관노이즈는 `n = Θ^(1/2) n_a` (식 3, 4)의 one-sided correlation 모델로 생성, 소스는 비균등 확률(`psrc`)을 가질 수 있음.
4. 핵심 기법 1단: Correlated-DnCNN(1D 컨볼루션 D층)이 수신 심볼 `y=s+n`에서 노이즈 추정치 `n̂`을 출력, `ŝ=y-n̂`로 잔차노이즈 `r=n-n̂`를 최소화(`LossA=‖r‖²/n`, 식 8).
5. 핵심 식: 사전정보 결합 LLR `û = l_ch + l_src` (식 11), `l_src`는 소스 확률분포 기반 로그우도비로 non-equiprobable source 압축 효과를 부여.
6. 핵심 기법 2단: BP-DNN은 Tanner graph edge마다 학습 가능한 가중치 `w`를 부여한 Nachmani식 feedforward 신경망(2L 은닉층=L회 BP 반복)으로 short cycle 악영향을 완화.
7. 부수 트릭: 잔차노이즈가 정규분포를 따르도록 Jarque-Bera 통계량 기반 `LossC`(식 9,10)를 추가해 디코딩 성능과 정규성을 동시에 제약.
8. 학습 절차: `Loss = LossA + b·LossB + c·LossC` (식 21, LossB는 BP-DNN 크로스엔트로피)로 Correlated-DnCNN과 BP-DNN을 Adam으로 공동학습, 균형계수 b/c는 노이즈 상관도·소스 분포에 따라 시뮬로 탐색.
9. 결과: η=0.8, psrc=0.5일 때 BER=10⁻⁵에서 기존 iterative BP-CNN 대비 약 1.4~2.1dB 이득, psrc=0.1 결합모델은 BER=3×10⁻⁴에서 전통 BP(50) 대비 5.5dB 개선.
10. 한계: 시뮬레이션(Pytorch, GPU)만 수행되어 HW 미설계, 상관노이즈·비균등 소스라는 특정 무선채널 가정에 종속되며 소스 확률분포별로 별도 네트워크 재학습이 필요.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP-DNN (전체 tanh 기반 BP를 신경망 가중치로 대체) | 대응 없음 | Prime ECC는 min-sum 기반 HW 디코더이며 full-tanh Sum-Product/NN 대체는 프로파일 §6 "대응 없음" 예시에 해당 |
| Correlated-DnCNN (상관노이즈 제거) | 대응 없음 | Prime ECC 채널모델(AWGN/RBER/fixed-error, channel.cpp)은 상관노이즈를 다루지 않음 |
| 사전정보 결합 LLR(식 11, non-equiprobable source) | channel.cpp Set_LLR_Th() | 소스 통계 기반 LLR 보정이라는 개념은 유사하나 non-equiprobable 소스 압축이라는 무선통신 목적이 NAND ECC와 무관 |
```

적용 가치: `낮음` — NN이 BP 알고리즘 전체를 대체하는 구조로 min-sum 기반 Prime ECC HW 디코더와 근본적으로 이질적이며, 상관노이즈·소스압축이라는 무선통신 특화 문제는 NAND 채널과 무관.
