### B. 알고리즘 요약

1. 딥러닝(DL) 채널코딩 서베이로 크게 (1) DL 기반 부호설계, (2) DL 기반 채널 디코딩 두 영역을 커버하며, LDPC/polar/turbo/cyclic 코드를 대상으로 함.
2. 기존 부호설계(DE, EXIT chart)는 이상적 채널·무한 iteration 가정에 의존해 실환경과 괴리가 있다는 문제의식에서 출발.
3. LDPC 디코딩 핵심 절(IV-B)에서 `deep unfolding` 개념을 소개: BP iterative decoding을 unrolled Tanner graph 상의 feed-forward망으로 보고 CN-VN edge마다 trainable weight를 부여해 SGD로 학습(`[272]`가 원조).
4. 학습된 weight는 Tanner graph의 short cycle 영향을 보정하는 scaling 역할을 하며, 이를 통해 무학습 BP 대비 동일 iteration에서 BER 개선(`(63,45)` BCH 예시로 Sionna 시뮬 결과 제시).
5. Neural MS 변형들: neural NMS/OMS(정규화·오프셋 계수 학습), LAMS(protograph LDPC용 선형근사 min-sum), NOMS(layered BP + 5G LDPC), AMS 선택메커니즘, SMMS/VWMS(단일-최소값 min-sum), SCMS(부호반전 메시지 폐기) 각각에 학습가능 파라미터 도입.
6. 복잡도/메모리 절감 기법: weight sharing(엣지·CN/VN degree별 파라미터 공유, protograph lifting 구조 활용), tensor-train/tensor-ring 분해, pruning(중요도 낮은 CN 제거 + decimation), quantization(LLR·메시지 양자화, RQNN 기반 FAID).
7. error-floor 대책: 두 단계 cascaded neural NMS 디코더(1단은 waterfall 성능 개선, 2단은 잔여오류 처리)로 LDPC error floor 제거를 학습(`[345]`).
8. 학습 절차는 대부분 SGD/Adam/RMSprop 기반 supervised(cross-entropy loss) 또는 syndrome-based loss, 일부는 RL(Q-learning 등)로 스케줄링·flip 위치 최적화.
9. 결과 수치는 인용 논문별 산발적(예: (63,45) BCH에서 학습 BP가 무학습 BP 대비 개선, VWMS 학습판이 40nm CMOS TSMC로 throughput 실측)로, 서베이 자체는 통합 수치를 제공하지 않음.
10. 한계: 서베이 특성상 알고리즘 원본 상세·HW 구현은 각 인용 논문에 있으며, 이 논문 자체는 실험/HW 설계를 수행하지 않음. Explainability, 학습/추론 비용, 벤치마크 부재 등을 향후 과제로 지적.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| deep unfolding 기반 neural BP/weighted min-sum (§IV-B-1) | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | CN/edge별 학습 weight를 min-sum 스케일링에 추가하는 아이디어는 기존 min-sum 구조에 파라미터만 얹는 형태라 이식 난이도 낮음(단, 학습 인프라 필요) |
| weight sharing / degree별 파라미터 공유 (§IV-B-1(b)) | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | 기존 iteration별 LLR 테이블 구조와 유사한 형태로 대응 가능성 있으나 학습 절차 자체는 대응 없음 |
| error-floor 대책용 cascaded neural NMS (§IV-B-2) | `decoder.cpp` `LDPC_Decoder()` 반복 루프 | error-floor 개선 방향성만 참고 가능, 구체적 학습 기반 기법은 코드 이식 대상 아님 |
| quantization/FAID 기반 저정밀 디코더 (§IV-B-7(a)) | `channel.cpp` `Set_LLR_Th()` | soft-read LLR 양자화 설계에 참고 가능하나 신경망 학습 파라미터는 대응 없음 |
| GNN 기반 디코더 (§IV-B-8) | 대응 없음 | Tanner graph 기반이나 GNN 자체는 Prime ECC의 고정 min-sum 파이프라인과 구조적으로 상이 |

> 적용 가치: 중간 — 서베이 자체는 이식 가능한 구체 알고리즘을 제공하지 않지만, min-sum 개선·error-floor 대책·복잡도 절감 관련 최신 딥러닝 기법의 지도 역할을 하여 후속 개별 논문 탐색의 출발점으로 유용함.
