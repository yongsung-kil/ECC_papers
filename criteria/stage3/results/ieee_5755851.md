### B. 알고리즘 요약

1. 대상은 자기기록(magnetic recording) 채널로, 구조화 LDPC 코드를 외부 Reed-Solomon(RS) 코드와 concatenation한 시스템(BER/SFR 성능 비교).
2. 문제: 여러 연구그룹이 각기 다른 방식(FG, RA/IRA/eIRA, Array, combinatorial, RS-based, TPC/SPC)으로 구조화 LDPC를 구성해왔으나 이들 간 관계가 불명확했음.
3. 핵심 가정: 두 선형블록코드가 동일 codeword 집합이거나 동일 permutation으로 서로 변환되면 "equivalent"로 정의(교과서적 부호이론 정의).
4. 핵심 기법: Lemma 1~4로 RS-based LDPC ↔ Euclidean Geometry(EG) LDPC ↔ 정규 Array LDPC ↔ Lattice LDPC 간 등가성을 incidence vector/line 구조로 증명.
5. 핵심 식: 라인 `L={β·(1,a)+(0,b): β∈GF(q)}`의 incidence vector가 RS 부호 codeword의 symbol location vector와 동일함을 보여 등가성 성립(`y=ax+b` 관계).
6. 확장: Extended regular array LDPC(열합=0 제약 추가)도 EG-Gallager LDPC의 부분클래스임을 Lemma 4로 증명, TPC/SPC 코드도 n이 소수일 때 extended array LDPC에 포함.
7. 채널 모델: perpendicular magnetic recording 채널 `V(t)=Σb_k g(t-kT_b+Δt_k)+v_n(t)`, transition jitter + AWGN, PDNP-SOVA 검출기 사용(SSD/NAND와 무관한 아날로그 채널).
8. 최적화 절차: 없음(구성법 비교), 각 클래스별로 시뮬레이션에서 "best" 대표 코드를 선정해 비교.
9. 결과: 높은 SNR에서 random LDPC가 최상, column weight 8 EG-LDPC가 고SNR에서 강세, RS-based/Array LDPC가 SFR 특성에서 random과 유사한 성능(구체 gate/throughput 수치 없음).
10. 한계: 순수 수학적 등가성 증명 + magnetic recording 채널 시뮬레이션뿐. HW 설계, 디코더 알고리즘 개선, NAND 관련 실험 전무.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 구조화 LDPC 클래스 등가성 증명(EG/RS-based/Array/Lattice) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC-LDPC(base+circulant) 고정 구조라 이 이론적 등가성 결과 자체는 부호 재설계에 직접 쓰이지 않음 |
| Magnetic recording 채널모델(PR+jitter+PDNP-SOVA) | `channel.cpp` | NAND의 AWGN/RBER/fixed-error 모델과 상이, 대응 없음 |
| RS와의 concatenation 구조 | 대응 없음 | Prime ECC는 순수 binary QC-LDPC 단독 ECC로 RS concatenation 미보유 |

> 적용 가치: 낮음 | 순수 이론적 부호 클래스 등가성 증명과 magnetic recording 채널 특화 실험으로, NAND binary QC-LDPC 부호설계·디코더·HW 어느 레이어에도 직접 이식할 요소가 없음.
