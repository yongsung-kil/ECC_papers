### B. 알고리즘 요약

1. 시스템: SSD 4KB 페이지 ECC 대상, `4KB`(40960 bit) 길이 `rate 4/5` QC-LDPC 4종, AWGN/BPSK 채널 4-bit 양자화 soft.
2. 문제: BCH 대체용 LDPC는 SSD에서 error-floor·복잡도·throughput·페이지크기 맞춤 설계가 미해결.
3. 기본 부호: QC-LDPC는 base matrix + circulant shift(`z×z`), `J=4` block row, `L=24` block col.
4. 핵심 기법 - Cyclically-Coupled(CC) 구조: 연속 codeword를 head/tail로 연결, `K`개 sub-code의 PCM을 부분 overlap(양쪽 `W=4` block col)시켜 결합.
5. 의미: overlap 영역 변수노드는 6개 check node에 연결되어 제약이 강화, sub-code 하나보다 복호능력이 크게 향상되고 error-floor를 억제.
6. 확장: sub-code 수 `K`(2/4/8)와 `z`(1024/512/256), girth(6/8) 조합으로 4종 코드(a~d) 구성.
7. 디코더: 계층(layered) BP 복호기를 sub-decoder마다 배치, cyclic interconnect + soft-message RAM + configurable routing network으로 조립, sub-code 동시 복호.
8. 메모리: RAM을 sub-code 전용 영역과 sub-code 간 교환(overlapping) 영역 두 파티션으로 분할, sub-decoder 재사용 스케줄링.
9. 결과: FPGA에서 4종 모두 100MHz에서 `1.47Gbps`, 약 1M gate + 1Mbit mem; 코드 a,b,c는 10^-10에서 0.02dB 이내, 10^-11까지 error floor 없음, code d만 0.1dB 열화.
10. 한계: AWGN 시뮬뿐(NAND 실채널 아님), 10^-14 이하 미검증, 상세 복호기 설계는 참고문헌 [13][14]에 의존.
