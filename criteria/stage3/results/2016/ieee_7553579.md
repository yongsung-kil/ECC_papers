### B. 알고리즘 요약

1. NAND flash storage용으로 이진 SC-LDPC를 `(m,2s)` EG LDPC 블록부호에서 구성 (고정정능력, 극저 UBER, 저복잡도 복호 목표).
2. 문제: 블록 LDPC는 error-floor가 있고, SC-LDPC는 convolutional gain으로 저RBER/고SNR에서 유리해 차세대 NAND에 적합.
3. 핵심 기법: `m`차원 EG의 `K` cycle class 각 generator vector를 `θ`개 subvector로 circulant decomposition → `θ×θ` base matrix 생성.
4. 2D edge-spreading으로 regular protograph를 열·행 방향으로 불규칙 확산, column-and-row summation 제약 `Σ B(i,j)=B` 유지 (식 4,5).
5. base matrix `B(m,2s)`를 unwrapping하여 계단형 `B_uw`, 이를 `Y`회 주기 반복해 terminated SC base matrix `B_SC` 구성 (`ms=θ-1`로 주기 θ).
6. 각 원소를 `γ×γ` CPM(lifting factor `U=γ`, `θγ=2^ms-1`)으로 lifting해 최종 PCM `H_SC` 획득.
7. PCM rank 정리: EG 블록부호 rank를 Hamada 공식(식 15,16)으로 정확히 계산, EG-SC의 rank 하한 `R(H_SC) ≥ R(H_uw)+R(Q_SC)(Y-1)` 유도 (Lemma 1).
8. 복호: 저복잡도 reliability 기반 WBF(flooding schedule)로 BSC상에서 평가, 채널정보 `log((1-p)/p)` 상수 magnitude 입력, 최대 500 iter.
9. 결과: EG(3,2^3)/EG(3,2^4)에서 EG-SC가 블록 EG 대비 RBER convolutional gain(~0.0012), UBER 10^-9~10^-10에서 error-floor 없음(protograph SC는 10^-8, regular는 10^-6~10^-7에서 floor).
10. 한계: HW 미설계, WBF/BSC hard-decision 위주, window 복호 latency/throughput은 향후과제, 부호길이 28k~147k로 대형.
