### B. 알고리즘 요약

1. 시스템: BPSK/AWGN, sum-product 반복 복호(최대 30회), 제안 QC-LDPC 부호 C1(10530,0.556), C2(7176,0.5).
2. 문제: Tanner graph의 짧은 cycle이 반복 복호 성능을 저해 - girth를 8 이상으로 크게 하는 구성 필요.
3. 핵심기법 1단: Moore (q+1,8)-graph `Γ_q`로 `N=1+q+q^2+q^3` 크기의 (q+1)-regular base matrix `B` (girth 8 보장) 구성.
4. 핵심 정리: Moore graph가 girth 8인 (q+1)-regular bipartite → base matrix `B`의 girth `g_b=8` (Theorem 2.2).
5. 핵심기법 2단: `B`의 각 '1'을 L×L circulant permutation으로, '0'을 zero matrix로 치환해 mother matrix `H_M` 생성.
6. shift값 `a_{m,n}` 할당은 [12]의 Shift Values Assigning Algorithm 사용, cycle 조건 `Σ(a_{mk,nk}-a_{mk+1,nk})≡0 mod L`로 짧은 cycle 회피.
7. 핵심기법 3단: column decomposing - 각 block column(weight q+1)을 P개로 분해해 effective matrix `H_E`, rate `r_E = 1 - 1/P`로 다중rate 실현.
8. 이론 보증: 분해가 cycle을 끊어 `H_E`의 girth ≥ `H_M` girth = 8 (Theorem 4.1), check node degree 집중 유지.
9. 결과: C1은 [13]의 동일 rate geometric LDPC보다 우수, C2는 [14]의 girth-8 부호들보다 고SNR에서 우수 (BER 곡선).
10. 한계: 순수 구성법, HW 미설계, error-floor·정정능력 정량 미제시, rate 0.5~0.556으로 NAND 고rate와 거리.
