### B. 알고리즘 요약

1. 시스템: IEEE P802.16e(WIMAX) rate 1/2, size 576 (288,576) irregular LDPC, AWGN/BPSK, max iter 50, 소프트 LLR 복호.
2. 문제: MS는 check node 근사로 C2V 메시지를 과대추정, NMS는 상수 γ(0<γ<1)로 보정하나 short cycle(girth) 영향은 반영 못함.
3. 핵심 가정: Tanner graph 최소 cycle(girth)에 얽힌 check node일수록 self-confirmation이 커 extrinsic 메시지를 더 공격적으로 감쇠해야 한다.
4. 핵심 기법 - GA-NMS: 상수 γ 대신 check node m의 local girth `G(m)`에 의존하는 `γ(G(m))`를 식(3) `βm,n = γ·sgn·min|α|`에 대입.
5. girth 계산: 각 check node가 "ping"(1)을 Tanner graph에 broadcast, 다시 돌아오면 cycle 발견, `G(m) = it×2` (Algorithm 1, offline).
6. 코드 구조 분석: WIMAX rate 1/2 코드는 girth 4인 CN 144개, girth 6인 CN 120개, girth 8인 CN 24개로 분류됨.
7. 계수 배정: girth 4는 더 작은 γ1, girth 6은 γ2>γ1, girth 8은 γ2~1 사이로, γ2·γ3=0.85 고정 후 γ1을 0.1~1 스윕하는 식으로 offline 결정.
8. 결정값: SNR 2.5dB, itermax 50에서 `[γ1,γ2,γ3]=[0.8,0.9,0.95]` (기존 NMS 상수 γ=0.85 대비).
9. 결과: WIMAX 코드에서 GA-NMS가 MS/NMS 대비 BER/FER 우수(특히 error floor 영역), 고SNR(3~6dB)에서 더 빠른 수렴(iteration 감소), 복잡도 증가 작음.
10. 한계: HW 미설계 시뮬만, γ(G(m))가 특정 코드 girth 구조 종속(같은 구조 코드에만 재사용), 특정 WIMAX 코드 1종만 검증.
