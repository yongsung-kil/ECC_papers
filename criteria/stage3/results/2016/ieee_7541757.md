> 총평: absorbing set이 양자화 MP 디코더의 FER를 code-independent하게 하한한다는 진단 이론 — Prime ECC의 error-floor/양자화 한계 분석·부호선별에 간접 유용하나, 디코더 성능을 직접 개선하지는 않음.

### B. 알고리즘 요약

1. 시스템: 길이 L 이진 LDPC, BPSK+AWGN, 채널 LLR을 `Qq1.q2` 균일양자화(q=q1+q2+1 bit)로 표현하는 양자화 MP 디코더(예시는 SPA, Q3.2, Φ(0)=4.25).
2. 문제: HW 구현의 유한정밀 양자화가 error-floor를 유발 — SPA는 양자화에 특히 민감. absorbing set(AS)이 high-SNR 복호실패의 주 원인인데, 기존 error-floor 예측은 특정 부호 전체 구조 지식을 요구.
3. 모델: (m,n) absorbing set A = 각 VN이 O(A)(홀수차 CN)보다 E(A)(짝수차 CN)에 더 많이 연결된 부분집합. 수신 LLR을 AS 내부 `s`와 외부 `o`로 분리, all-zero codeword 전송 가정.
4. 핵심 기법1(code-dependent 하한): `FER ≥ P(E_A)` — AS에서 오류가 나면 프레임 오류. 외부노드가 최대신뢰 `o_max`일 때도 AS 내부 noise가 실패를 유발 → `P(E_A) ≥ P(E_A|o_max)` (Assumption I).
5. 핵심 식: `α = P(E_A|o_max) = Σ_i P(E_A|s_i,o_max)·P(s_i)` (11) — 각 `P(E_A|s_i,o_max)`는 디코더 실행으로 0/1 결정, `P(s_i)`는 양자화 확률 (3)(12)로 계산.
6. 핵심 기법2(code-independent 하한): 매 iteration마다 외부 VN이 CN에 최대신뢰 `l_t`를 전달한다고 가정(Assumption II) → AS 서브그래프만 복호. `β = P(E_A^max) = Σ_i P(E_A^max|s_i)·P(s_i)` (16), 부호 무관·부호길이 무관.
7. 다중 AS 근사: 동일 구조 AS N개 존재 시(array/QC 코드에서 흔함) union bound + 독립 근사로 `γ' = Σ P(E_A^max) − Σ P·P` (19); (3,K) array code의 (4,2) AS는 `γ' = Nβ − (N choose 2)β²` (20).
8. 파라미터 의존: 하한은 양자화 스킴과 AS의 VN/CN 차수에만 의존, 부호길이·상세구조 무관 → 부호설계자용 진단도구.
9. 결과: (3,6)-정규 L=4000 부호의 (4,2) AS 하한 β가 Eb/N0=2dB부터 시뮬 error-floor를 정확히 하한(Fig.3). (3,61) array code의 (4,2) AS(multiplicity 334,890)에서 γ'가 시뮬 error-floor를 근접 추종(Fig.4). Φ(0)↑·분수비트 q2↑는 error-floor 개선하나 waterfall 악화(Fig.5).
10. 한계: HW 미설계(시뮬 뿐), 예시는 양자화 SPA 위주(min-sum/box-plus는 향후 확장), 성능을 개선하는 게 아니라 달성 불가능 하한을 증명, 정규 코드 예시 중심.
