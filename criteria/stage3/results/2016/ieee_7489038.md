### B. 알고리즘 요약

1. 대상: 유한 블록장(short-to-medium) LDPC의 error floor를 offline 그래프 구성으로 개선. AWGN/BEC/block-fading 채널, rate `1/2` 위주.
2. 문제: 짧은 블록에서 cycle이 message 독립성을 깨 stopping set/error floor를 유발. 기존 PEG/IPEG(ACE), EMD[12] 구성법이 다중 경로 상황에서 stopping set 위험을 부정확하게 평가.
3. 핵심 개념: cycle의 `EMD`(집합에 단일 연결된 CN 수), stopping set(EMD=0). PCM 열 합의 1 개수로 경로 EMD를 계산.
4. 핵심 기법 1단: PEG 트리 확장으로 최대거리·최소가중 CN 후보(`A`,`B`) 확보 후, root VN에서 후보 CN까지 distinct path 수 `P_ci`가 최소인 CN(`C`)을 우선 선택(작은 cycle 수 최소화).
5. 의미: stopping set은 zero-EMD cycle 또는 extrinsic edge로 결합된 cycle들에서 생기므로, 작은 cycle 수를 줄이면 stopping set 생성 확률이 감소.
6. 핵심 기법 2단: `C`가 복수면 각 경로의 EMD를 (12)로 구해 평균 `γ_ci=(1/P_ci)ΣE_p,ci`가 최대인 CN을 최종 배치(연결성 큰 후보 선택).
7. 부수: cycle-oriented EMD spectrum `γ`를 부산물로 산출해 그래프 품질 정량 비교. QC-LDPC/IRA에도 적용(IRA는 accumulator dual-diagonal 초기화).
8. 확장: block-fading(F 페이딩) 완전다이버시티용 reduced-structure 코드 클래스 제안(systematic stopping set 회피, null block 배치), 단순 puncturing으로 F-1 채널 호환.
9. 결과: QC-LDPC에서 PEG 대비 0.4dB, IPEG 대비 0.3dB, QC-DO-PEG 대비 0.25dB 이득(BER<1e-7). 최단 cycle 수 PEG 대비 27% 감소. block-fading 수렴 반복수 개선.
10. 한계: HW 미설계·시뮬만, 짧은 블록/rate 1/2/block-fading 중심, 구성 시 distinct path 저장 비용 큼.
