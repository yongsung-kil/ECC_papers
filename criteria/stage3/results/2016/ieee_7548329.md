### B. 알고리즘 요약

1. 시스템은 100Gb/s+ 광전송(OTN)용 FEC, 모(母)부호는 girth-10 regular QC-LDPC `(34635, 27710)`, `γ=3, ρ=15, B=1129`(순환 permutation 행렬), rate 0.8.
2. 문제: 시변 광채널에 맞춰 rate를 바꾸려면 부호족마다 PCM 저장 비용이 크므로, 단일 통합 HW로 rate-compatible을 지원해야 함.
3. 채널 모델: BI-AWGN(ASE noise 지배 시나리오 유효), 6-bit uniform 양자화 LLR.
4. 핵심 기법: coarse-tuning은 오른쪽 열을 soft-puncturing(초기 LLR을 최대 정수로), fine-tuning은 매 `d`번째 SPC check를 (15,11) Hamming code(generalized/local code)로 치환. rate {0.6858~0.8} 18단계.
5. rate 식: `R ≥ 1 − (M + M/d×(m−1))/n`, d∈{INF,127,63}로 미세 rate 제어.
6. 디코딩: layered scaled min-sum, scaling factor `s=0.75`; SPC check는 min-sum(식 7), generalized check는 BCJR 기반 MAP/APP 업데이트(식 6).
7. HW: VNP/SMS-CNP/MAP-CNP(BCJR bidirectional recursion)/ETU + 재구성 mux(puncturing-mux, generalized-mux) 통합 아키텍처, 4-set block RAM.
8. BCJR generalized-CNP는 forward/backward α,β 동시 재귀로 latency 감소, max-star→max 근사로 메모리 절감.
9. 결과: Xilinx xc6vsx475t에서 통합 RC 디코더는 puncturing 대비 slice +6%; throughput 모부호 ~9.6Gbit/s(high SNR)/4.8Gbit/s(low SNR); rate 0.8~0.6858에서 coding gain 12.67~13.8dB @ BER 10⁻¹⁵, error floor 없음, 20~27 iteration 수렴.
10. 한계: 광통신 특화(ASE/BI-AWGN), generalized LDPC(Hamming doping) 확장은 순수 QC-LDPC와 이질, 6-bit soft LLR은 NAND read 비용 관점에서 과다.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| layered scaled min-sum(s=0.75) CN 업데이트 | decoder.cpp CNU_Update_New_Mag(), C2V_Cal_New_Sgn() | 우리 min-sum(min1/min2)과 직접 대응, scaling factor 개념 이식 가능 |
| VNP/bit decision 업데이트 | decoder.cpp VN_Cal_HD() 등, LDPC_Decoder() | layered VN 업데이트가 우리 이터레이션 루프와 정합 |
| soft LLR 6-bit 양자화 | channel.cpp Set_LLR_Th(), Set_R_Offset() | 양자화 테이블 대응, 단 6-bit는 우리 2SD/3SD 범위 초과 |
| early termination unit(ETU) | partial_CRC.cpp / full_CRC.cpp | 우리는 CRC 조기종료, 논문은 codeword 수렴판정으로 방식 상이 |
| puncturing/generalized-mux, BCJR MAP-CNP | 대응 없음 | generalized LDPC(BCJR local code doping)는 순수 QC-LDPC인 우리 구조에 대응 없음 |
| QC PCM(circulant) 로드 | ecc_top.cpp Load_PCM() | girth-10 QC 구성 로드 지점 대응, 단 부호 재검증 부담 |
```

마지막 적용 가치: **중간** — layered scaled min-sum과 6-bit soft 양자화 FPGA 구현은 우리 min-sum 디코더에 참고 가치가 있으나, 핵심 신규 기여인 puncturing+Hamming doping rate-compatibility와 BCJR generalized-CNP는 광통신 특화·우리 순수 QC-LDPC 구조와 이질적이라 부분 이식만 가능.
