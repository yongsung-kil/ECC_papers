### B. 알고리즘 요약

1. 대상은 SC-irregular LDPC(SC-iLDPC) 부호 앙상블, 시뮬 예시 `M=8192`, `L=64`, code length `2LM=1,048,576`, rate 1/5~4/5.
2. 문제: 기존 SC-LDPC의 rate 유연화는 regular 성분(puncturing/체크노드 차수 K 조정/다중 chain 병렬)에서 오며 rate 증가 시 Shannon limit과 gap이 벌어진다.
3. 구성: `(L-1+J)×qL` binary band matrix `B(J,qJ,L)`를 base로, 비영원소를 비정방 `(M-k)×M` irregular superposition matrix `S`로 치환하여 PCM `H` 생성(rate는 `k`,`M`으로 조절, `q=2` 가정).
4. 핵심 변수: `S`의 열/행 가중치 분포 `λ̃i`,`ρ̃i` → M-copy VN과 (M-k)-copy CN 사이 edge-degree 분포 `λ(x)`,`ρ(x)`로 변환. 기존 SC-LDPC의 M×M 순열행렬(가중치 1)을 irregular 행렬로 일반화한 것.
5. 핵심 식: `k/M = 1 - ∫ρ(x)dx / ∫λ(x)dx` — superposition matrix를 하나의 irregular LDPC PCM으로 보면 `k/M`이 그 부호 rate이며, 이를 통해 전체 rate `R`을 연속 조절.
6. 최적화: BEC protograph density evolution(식 4~7)으로 threshold `ε*`를 정의하고, differential evolution으로 `λ(x)`,`ρ(x)`의 최적 degree 분포 탐색.
7. 결과(수치): `J=2,L=100`에서 모든 rate에서 gap `G=εsh-ε*<0.011`; `J`를 4→2로 줄일수록 threshold가 Shannon limit에 근접(rate loss 없음, 소J 이점은 open question).
8. 관찰: SC 특유의 좌우→중앙 순차 복호가 재현됨(`J=2,L=64,R=1/2,ε=0.45` BER evolution).
9. Monte-Carlo(AWGNC, TPMP, iter 1000): rate 1/5~4/5 BER이 Shannon limit에 근접, degree-2 VN 비율이 매우 작아(예 0.0728) error floor/waterfall 우수.
10. 한계: HW 미설계, BEC/AWGN 이론·시뮬만, SC-irregular 구조·저~중 rate 위주, non-QC superposition 구성.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| superposition matrix 기반 SC-iLDPC 부호 구성 | `ecc_top.cpp` `Load_PCM()` | 낮음: Prime ECC는 고정 block QC-LDPC(base+circulant) 로드만, band+superposition 구성 파이프라인 없음 |
| degree distribution DE 최적화(differential evolution) | 대응 없음 | Prime ECC에 부호설계용 DE/degree 최적화 로직 부재 |
| error-floor(degree-2 VN 최소화) | 대응 없음 | Prime ECC는 부호구조가 고정되어 VN 차수 재설계 경로 없음 |
| message passing 복호(TPMP) | `decoder.cpp` `LDPC_Decoder()` | 낮음: Prime ECC는 min-sum layered인데 논문은 표준 TPMP, 알고리즘 기여 아님 |

적용 가치: **낮음** — rate 유연·capacity-approaching SC-irregular LDPC 구성 이론으로, 고정 block binary QC-LDPC(고rate)인 Prime ECC와 부호구조·rate 영역이 맞지 않고 HW 미설계라 직접 이식 대상이 아니다.
