> 총평: 비트패턴 자기기록(BPMR)의 2D ISI+media noise 채널에 맞춘 irregular LDPC 부호 degree 분포를 EXIT chart로 최적화한 순수 부호설계 논문. 채널·검출기가 NAND flash와 전혀 달라 Prime ECC 이식 가치 낮음.

### B. 알고리즘 요약

1. 대상: BPMR(bit-patterned magnetic recording) 채널의 rate 8/9 irregular LDPC 부호. 블록길이 약 1e4~13K bit, 기록밀도 4Tb/in2.
2. 문제: BPMR은 2D ISI(1D ISI+inter-track interference)와 media noise(island position jitter PJ, size fluctuation SF)를 겪음 - 기존 AWGN 최적화 부호로는 손실 발생.
3. 채널모델: analytical 2D island pulse response `h(z+Δz,x+Δx,Λz,Λx)`로 PJ(Δ)·SF(Λ) 반영, 수신신호 `r(i,j)=ΣΣh·x+w`, w는 AWGN.
4. 검출기: full BCJR 2D-DET 대신 저복잡도 IRCSDFA-GA(iterative row-column soft decision feedback + Gaussian approximation) 사용, trellis 상태 `2^(LN-1)`로 축소.
5. 핵심식(등가잡음): 2D ISI 잔여항을 중심극한정리로 Gaussian 근사, 등가잡음 평균 `η_r`·분산 `∂_r^2`(식6,7)를 row/column soft info `p_r,p_c`로 계산해 BCJR metric에 대입.
6. 부호설계(EXIT): variable/check node degree 분포 `λ,ρ`를 modified EXIT chart로 최적화 - CND EXIT 곡선을 (2D-DET+VND) 결합 EXIT 곡선에 최대한 근접시키되 교차 안 하도록 λ를 선형계획으로 탐색.
7. EXIT curve: 2D-DET EXIT `I_E,DET`는 Monte Carlo로 히스토그램 기반 산출, VND EXIT는 duality theorem `J(·)` 함수로 표현, `I_E,DET`는 3차 다항식 근사.
8. 시뮬레이션: 2D-DET↔LDPC BP 반복 최대 10, LDPC 내부 BP iteration 최대 30, 모든 부호 rate 8/9.
9. 결과: SF 최적화 부호가 AWGN 최적화 대비 BER 1e-5에서 약 0.2dB 이득, PJ만 있을 때 약 0.3dB 이득. PJ가 SF보다 성능 열화 더 큼. BPMR media noise 최초 최적화 부호.
10. 한계: 순수 부호설계(HW 미설계), BPMR 자기기록 2D ISI 특화 채널, PJ error floor 잔존(추가 신호처리 필요), 시뮬레이션만.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BPMR 2D ISI 채널 모델 | 대응 없음 | NAND RBER/AWGN과 다른 자기기록 2D ISI 채널 |
| IRCSDFA-GA 2D 검출기 | 대응 없음 | Prime ECC엔 채널 검출기(equalizer) 자체가 없음 |
| EXIT chart degree 최적화 | 대응 없음 (부호설계는 [ecc_top.cpp](../Prime_ECC_3.1_Claude) `Load_PCM()`에 고정 QC-LDPC 로드) | irregular degree 분포 설계 - 우리는 특정 QC-LDPC 고정 |
| BP LDPC 복호 | [decoder.cpp](../Prime_ECC_3.1_Claude) `LDPC_Decoder()` | BP는 공유하나 논문 기여는 부호설계·검출기이지 복호 아님 |

적용 가치: **낮음** - BPMR 자기기록 2D ISI+media noise 채널 특화 irregular LDPC 부호설계로, NAND flash binary QC-LDPC 및 Prime ECC의 채널/디코더 구조와 도메인이 달라 이식 불가.
