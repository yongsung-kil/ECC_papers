> 총평: 공간결합 protograph 앙상블의 asymptotic threshold 설계 이론으로, block QC-LDPC 기반 Prime ECC와 부호족·디코딩 구조가 달라 이식성 하.

### B. 알고리즘 요약

1. 시스템: SC-LDPC block code(SC-LDPC-BC) 신규 protograph 설계, target rate `R=1-bc/bv`(주로 1/2, Table IV서 0.484~0.794), lifting factor `M`, termination length `L`(16~256), coupling width `w`.
2. 문제: SC-LDPC-BC의 rate loss(유한 `L` 부산물)를 threshold·인코딩/복호 복잡도 희생 없이 줄이는 것이 미해결.
3. 핵심 모델: convolutional protograph를 `t=0..L-1`로 종단(terminate)하면 양끝 구조적 불규칙성이 wave-like convergence를 유발해 threshold가 capacity에 근접.
4. 핵심 기법1 - symmetric 앙상블: `bc=1`, check node degree를 대칭(2,4,...,16,16,...,4,2)으로 두고 모든 variable node degree ≥3 보장 → linear minimum distance growth rate 확보.
5. 핵심 식: rate loss `RL = R - 2/(L·bv)` (식 4) - 기존 `C(J,K,L)`의 `w/(L·bv)`보다 작아 rate-threshold 트레이드오프 개선.
6. 핵심 기법2 - multiple-edge: variable-check 쌍에 다중 edge 허용(`d11,d12,d21,d22` 분배)해 base matrix 밀도↑, threshold를 Shannon 한계에 더 근접(예 SNR*=0.1974 dB).
7. 구현 디테일: threshold를 DE 대신 RCA(reciprocal channel approximation) flooding으로 빠르게 계산, reciprocal energy `R(x)=C^-1(1-C(x))` 사용.
8. 최적화: exhaustive search로 `w=1` 최적 edge 분배 탐색(`d≤10` 제약).
9. 결과: R=1/2 symmetric `d=8`이 BI-AWGNC capacity 달성(SNR*=0.187 dB, L=256), `C(J,K,L)`보다 낮은 rate loss로 더 나은 L-threshold 트레이드오프(L=64서 0.329 vs 0.688 dB); BSC threshold 최초 산출.
10. 한계: asymptotic threshold 이론만 - finite-length BER, HW, gate count, throughput 전무; 공간결합은 window 디코더 전제.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SC-LDPC-BC protograph/edge-spreading 구성 | ecc_top.cpp Load_PCM() (block QC 전용) | Prime ECC는 block QC-LDPC 고정, 공간결합 대각 밴드 구조와 부정합 |
| RCA threshold 분석 | 대응 없음 | Prime ECC에 asymptotic threshold 분석 모듈 없음 (bit-exact 시뮬만) |
| wave-like convergence / window decoding 전제 | 대응 없음 | decoder.cpp LDPC_Decoder()는 block flooding, streaming/window 스케줄 아님 |
```

마지막 적용 가치: **낮음** - SC-LDPC(공간결합)는 Prime ECC의 block QC-LDPC와 부호족이 다르고 window/streaming 복호를 전제하며, 이 논문은 무한길이 threshold 설계 이론에 그쳐 NAND block 파이프라인으로 이식할 근거가 약함.
