### B. 알고리즘 요약

1. 대상은 `(dl, dr, L, w)` SC-LDPC — uncoupled PCM `Hu`를 `L`번 복사 후 smoothing parameter `w`로 이웃 체크노드에 결합해 PCM `HSC` 구성.
2. 문제: 무한길이 SC-LDPC는 BP=MAP 임계값·universality로 우수하나, 유한 codeword-length 성능은 덜 연구됨.
3. rate loss를 두 축으로 분해: coupling chain 유한길이(`L`) 손실 `fL(L)`과 uncoupled code 유한길이(`M`) 손실 `fM(M,L)`, `N=M·L`.
4. coupling chain 손실은 `fL(L)=λL·L^-1` — design rate와 true rate 차이의 tailing effect.
5. uncoupled 손실은 [9]의 FER 근사 `FER≈√(k1L/(MΔ))·exp(-k2MΔ²)`에서 `MΔ²≈const` 유도 → `fM(M,L)≈λM·M^-0.5`.
6. `N` 고정 시 `M`↑이면 `L`↓ 트레이드오프 → `Rloss(N)=min λL·L^-1+λM·M^-0.5`.
7. 최적화 해: `Lopt=(4λL²N/λM²)^1/3`, `Mopt=λM·N^2/3/(2λL)`.
8. 결과: `Rloss(N)=λN·N^-1/3`, `λN=(27/4)λLλM²` — 일반 LDPC/finite-blocklength capacity의 `O(N^-1/2)`보다 열등.
9. 함의: 일반 length-N LDPC와 동성능 내려면 SC-LDPC는 `O(N^3/2)` 길이 필요.
10. 한계: 순수 점근/scaling 이론이며 부호설계·디코더·HW 미제시, BMS/BEC 채널 가정, universality 장점만 정성 언급.
