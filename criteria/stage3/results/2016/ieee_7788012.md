### B. 알고리즘 요약

1. 시스템: 고속 광통신(100G~1Tbps)용 irregular QC-LDPC, code length `N=16092`, rate `0.778/0.806/0.833/0.889`, circulant size `p=149`, array `J×L`(L=108).
2. 문제: 큰 `dv`는 저BER 성능은 좋으나 고BER 손실, girth `g`·column weight `dv`·차수분포 `{ρi}`가 성능 좌우하나 irregular 코드는 구조성이 약해 고속 시스템 적용이 어려움.
3. 핵심 기법 1단: `g=6` regular QC-LDPC `H_reg`를 먼저 구성(circulant 계수 `c_{j,l}=mod{l·[slope(j)-1], p}`, `slope=randperm(p)`).
4. 핵심 기법 2단: EXIT chart 분석으로 최적 차수분포 `{ρi}` 설계(`1≤dv_i≤L`, `Σρi=1`, 정밀도 `1/L`), 서로 다른 column weight 수는 5개, 최소 `dv=3`(dv=2는 error-floor 유발).
5. 핵심 기법 3단: `H_reg`의 일부 circulant sub-matrix를 null matrix로 랜덤 치환(random block puncturing)해 최적 `{ρi}`를 만족하는 irregular `H_irreg` 생성, QC 구조 유지.
6. 예시: rate 0.833에서 `{dv}={3,4,5,6,7}`, `{ρ}={0.3821,0.2547,0.1651,0.0991,0.0991}`, 수렴 임계 2.82dB, 평균 `dc=23.56`.
7. 복호: BP-LLR, iteration 8/12로 Monte Carlo 시뮬(BPSK AWGN).
8. 결과: 모든 코드가 BER 10^-10 위에서 error-floor 없음, rate 0.889에서 Shannon limit 대비 0.56dB 손실, 4 iteration만으로 고rate 0.2dB / 저rate 0.5dB coding gain.
9. 한계: HW 미설계, BPSK AWGN 시뮬만, error-floor 검증도 10^-10 수준까지만.
10. NAND 무관 채널(광통신)이지만 girth+차수분포 동시 최적화 구성 아이디어 자체는 binary QC-LDPC 부호설계에 이식 여지.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| H-matrix / 부호구조 로드 (irregular QC 구성) | [ecc_top.cpp](../Prime_ECC_3.1_Claude/) `Load_PCM()` | random block puncturing으로 생성한 H를 로드하는 지점, 부호설계 이식의 진입점 |
| 차수분포/circulant 설계 (H-matrix) | 대응 없음 | Prime ECC는 특정 QC-LDPC(N_b=129, z=32) 고정 - 재검증 부담 큼 |
| BP-LLR 복호 | [decoder.cpp](../Prime_ECC_3.1_Claude/) `LDPC_Decoder()` | 논문은 표준 BP, Prime은 min-sum - 알고리즘 자체 이식 불필요 |

적용 가치: **낮음** - 부호설계 아이디어(girth·dv·분포 동시 최적화)는 흥미롭지만 광통신 타깃·HW 미설계이고 Prime ECC는 QC-LDPC가 고정되어 재설계·재검증 부담이 큼.
