### B. 알고리즘 요약

1. 시스템: 단일 인코더 PCGC, rate `1/3` 연접부호(길이 `600`), 성분부호는 rate `1/2` LDPC(`200×400`, MCW 3), 동일 parity `Pb`를 2번 전송.
2. 문제: 기존 단일인코더 PCGC는 interconnecting check node로 복호기 복잡도·지연이 크고, 성능이 동급 전용 LDPC에 못 미침.
3. 모델: AWGN 채널, MSA(Min-Sum) 반복복호, 두 성분복호기가 동일 parity check matrix를 공유.
4. 핵심기법: 두 성분복호기를 interconnecting check node 없이 병렬 연결하고, 매 local iteration 후 전체 codeword의 a posteriori LLR을 서로 교환.
5. 식(1~3): 비트→체크 메시지 `M = L - E`, 체크 extrinsic `E = ∏sign · min|M|`, 비트 갱신 `L = Y + ΣE` — 표준 MSA 갱신.
6. 확장: super iteration마다 `Y_m^q = Y_m^q + L_m^{q'}`로 상대 복호기의 a posteriori LLR을 자기 intrinsic에 더해 신뢰도 강화.
7. 디테일: super iteration당 성분복호기는 local iteration 1회만 수행해 지연 축소.
8. 학습/최적화: 없음.
9. 결과: 제안 PCGC가 기존 PCGC보다 모든 SNR에서 BER 우수, 전용 LDPC와 거의 동등(max super iteration 25).
10. 한계: HW 미설계, 시뮬(BER)만, gatecount/throughput 없음, 전용 LDPC보다 더 많은 iteration 필요.
