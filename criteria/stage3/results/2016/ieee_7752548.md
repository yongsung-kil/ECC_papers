### B. 알고리즘 요약

1. 시스템: TLC(3bit/cell) NAND, CCI 채널, rate-0.784 length-2331 binary QC-LDPC, 최대 반복 30회.
2. 문제: SPA/BP는 복잡도 과다, 기존 MLGD/SRBI-MLGD는 양자화 미고려로 read latency·수렴속도 개선 여지.
3. 채널모델: threshold voltage `Vth` 분포에 CCI `F = Σ γ_n·ΔV` (수평 γ_h, 수직 γ_v, 대각 γ_vh 결합비) 반영, 인접 state 중첩으로 오류.
4. 핵심기법: 균일 양자화 간격 `Δ = F_max/2^b`, 양자화값 `q_j = ±round((V'_i - V_hd)/Δ)`를 초기 soft reliability `R_j^(0)=q_j`로 사용.
5. MLGD 코어: extrinsic `E_j = Σ(2σ_i,j - 1)` (범위 -γ~γ, γ=열가중치), 신뢰도 갱신 `R_j^(k+1) = R_j^(k) + E_j^(k)`.
6. intra-cell 확장: TLC 심볼 (m,c,l) 페이지별 불균등 오류확률·데이터의존성으로 추가 신뢰도 `ER_j` 도입 → `R_j^(k+1)=R_j+E_j+ER_j`.
7. 선택적 갱신: overlapping 영역만 flag `f_i=1` 설정, `ER_j`가 큰 비트는 이후 반복에서 갱신 생략 → message propagation 수 감소.
8. 양자화 차등: MSB/CSB는 2bit, LSB만 4bit → 평균 2.7bit (기존 [4]는 4bit). 논리·정수 덧셈 연산만 필요(HW 친화).
9. 결과: SRBI-MLGD(4bit) 대비 더 나은 BER·빠른 수렴, 특히 결합계수 `s_c>1.16`에서 이득; 양자화 비트는 감소.
10. 한계: BP 대비 BER 손실, HW 미설계(gatecount/throughput 없음), TLC intra-cell 모델·CCI 파라미터 의존.
