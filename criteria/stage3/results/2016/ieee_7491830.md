> 총평: girth-8 (3,K) QC-LDPC exponent matrix 생성 공식만 제시한 순수 구성법 논문 - binary QC이긴 하나 저밀도·저rate라 고rate NAND ECC(Prime ECC)엔 직접 이식 대상 아님.

### B. 알고리즘 요약

1. 대상은 binary regular QC-LDPC: `J×K` 배열의 `P×P` 순환치환행렬(CPM), 지수행렬 `E(H)`로 표현.
2. 문제: 기존 (3,K) girth-8 구성법(Zhang [9] 등)은 H-matrix 계산 복잡도·메모리가 커, 더 간단·빠른 exponent matrix 생성이 필요.
3. 3가지 규칙으로 base matrix 고정: 1행·1열은 0벡터, 2행은 오름차순, 인덱스 중복 금지.
4. 2행은 `a1,l = l`로 순차 채우고, 3행은 뺄셈 기반 폐형식 공식 (7)(8)로 계산: `a2,t = a2,t-1 + a1,t + (max(a1,·) - a1,t)`.
5. 이 공식은 4-cycle/6-cycle을 배제해 girth≥8을 보장하도록 인덱스 간격을 설계.
6. CPM 크기 조건(정리1): `P ≥ max(a2,·) + 1` 이면 girth 8 (귀납법으로 4·6 cycle 부재 증명).
7. 계산은 1초 미만으로 끝나 H-matrix 생성 시간복잡도를 크게 절감.
8. 예: K=9에서 3행 최대 인덱스 73 → CPM 크기 P=74로 girth-8 부호 구성.
9. 결과: AWGN + 50-iter 메시지패싱 시뮬에서 rate 1/3·1/2 부호가 shortened array code[10]·Zhang[9] 대비 고SNR BER 유리, 약 20 iteration에서 빠른 수렴; 단 CPM 크기는 Construction I보다 큼(Table 1).
10. 한계: 열무게 J=3·girth 8까지만, HW 미설계, NAND 아닌 일반 AWGN 채널 대상.
