### B. 알고리즘 요약

1. 대상은 protograph 기반 tail-biting SC-LDPC (rate `0.5`~`0.75`, coupling window `W=2~4`, `L=50`), 16-QAM AWGN + iterative demapping.
2. 문제: terminated SC-LDPC는 종단(termination)으로 유한길이에서 rate-loss 발생, tail-biting은 rate-loss 없지만 decoding wave가 트리거되지 않으면 기저 block LDPC와 동일 threshold.
3. 핵심 아이디어: 고차변조의 병렬 bit-channel 신뢰도 차이를 이용해, 신뢰도 높은 비트를 특정 sub-block에 집중시켜 "effective termination"을 만들어 wave-like 복호를 유발.
4. 핵심기법: Gray 대신 Ungerboeck set-partitioning(SP) labeling을 도입 — SP의 최우수 bit-channel이 Gray 두 채널보다 우수해 VND 곡선을 개선, threshold 상승.
5. 척도: per-bit EXIT 곡선과 protograph density evolution(Gaussian approx, mean만 추적)으로 CND/VND/demapper 함수를 매칭, BEC MAP threshold를 EXIT area theorem으로 근사.
6. 확장기법: hybrid mapping — `T`개 sub-block만 SP+iterative demapping, 나머지 `L-T`는 Gray(비반복 demap)로 iterative demap 복잡도를 앞쪽 블록에 국한.
7. 최적화: bit fraction 행렬 `D`(2K×LN')를 differential evolution으로 최적화, `T_uni`/`L-T_uni` 두 벡터의 균일 분포로 완화 가능.
8. 결과: 최적화된 tail-biting `C1`(SP)은 rate-loss 없어 terminated SC-LDPC(Gray)를 능가; hybrid는 threshold 약간 낮지만 여전히 terminated Gray 대비 우수, iteration 수는 Gray 수준 유지.
9. 부수 이점: SP/hybrid labeling 시 encoding buffer(인코딩 큐 길이)가 작아져 latency-제약 시스템에 유리.
10. 한계: 전부 threshold/EXIT/DE 이론 예측, 실제 BER 시뮬·HW 미설계, 16-QAM 고차변조·spatial coupling 전제.
