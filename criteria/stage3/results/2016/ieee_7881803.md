### B. 알고리즘 요약

1. 대상은 IEEE 802.11ad LAN QC-LDPC, rate 3/4·length 672, base matrix `4x16`, sub-matrix `42x42`, Xilinx Virtex6 FPGA 반병렬 구현.
2. 문제: 고throughput 디코더는 permutation network가 HW 자원 대부분을 차지하고, TPMP는 clock cycle이 많아 지연이 큼.
3. 복호는 Sum-Product Log-Domain으로, CN 갱신은 `E_ij = 2 tanh^{-1}(∏ tanh(·/2))`, VN은 채널 LLR에 extrinsic 합산 후 hard decision.
4. CNU를 `tanh` LUT + multiplier + `tanh^{-1}` LUT 모듈로 분할하고, row 곱을 cyclic shift register로 수행해 HW 재사용으로 곱셈기 축소.
5. 핵심 아이디어: layer decoding 대신 TPMP를 쓰되(성능 우위) 행·열 연산을 간섭 없이 동시화하여 iteration당 clock cycle을 `m+5`로 줄임(`m`=행 수).
6. 2단 pipeline: (1) CNU 내부 `tanh`/row-mult/`tanh^{-1}` 파이프라인, (2) CNU와 VNU 연산 동시화 — 병렬 유닛 2개면 clock cycle 1/2로 감소.
7. VNU는 tree adder로 열당 1 addition/clock, permutation network를 제거하고 데이터 shift 값 변경으로 고정 배선(fixed connection) 라우팅.
8. 병렬도는 layer 수와 동일하며, 병렬 유닛 증설로 clock cycle을 추가 감축.
9. 결과: max 280MHz에서 throughput 3360Mbps, 10 iteration, 전력 150mW, area 3.4(단위 불명), permutation network 제거로 스위치 오버헤드 삭제.
10. 한계: 무선 802.11ad 특화, full-tanh SPA(LUT 의존), 양자화 비트수·gate count·error-floor 성능 미제시, 부호설계·정정능력 개선 아님.
