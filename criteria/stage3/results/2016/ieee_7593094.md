> 총평: QC-LDPC의 circulant shift를 예측적으로 선택해 large girth를 보장하는 부호구성법으로, Prime ECC H-matrix 설계 시 error-floor 개선 목적의 이식 후보(간접).

### B. 알고리즘 요약

1. 대상은 `(dv,dc)`-regular/irregular 및 protograph type-I binary QC-LDPC로, `H`가 `P x P` circulant permutation matrix `I(p_{j,i})` 블록(lifting factor `P`)으로 구성됨.
2. 문제: short cycle이 sum-product 성능(특히 error-floor)을 저하시켜 girth를 키우되 circulant size `Pmin`을 최소화해야 하는데, QC-PEG는 backtracking(사후검증)이 무겁다.
3. 모델: `2l`-cycle 존재의 필요충분조건 `sum(p_{jx,ix}-p_{jx+1,ix}) = 0 mod P` (식3)에 기반, girth `g` 보장은 `!= 0 mod P` 제약.
4. 핵심 기법: computation tree에서 안 잡히는 "PEG-undetectable cycle"(같은 circulant 블록을 2회 이상 traverse하는 cycle) 개념 도입, 새 edge 할당 전 예측적으로 배제.
5. 이 예측성으로 사후 backtracking을 제거(P·girth 클 때 계산량 절감)하고 생성되는 girth-`g` cycle 수까지 제어 가능.
6. 확장: 8-cycle은 2x2/2x3/3x2/3x3 4개 시나리오(Theorem 1, 식6), 10-cycle은 3<={r,q}<=4의 8개 시나리오(Theorem 2, 식8)로 circulant shift 부등식 유도.
7. 구현: 후보 check node `Cm = jl*P + p_{jl,il}`를 식(6)/(8) 만족분만 남겨 girth 10(식6) 또는 12(식6+8)를 사후검증 없이 구성.
8. 학습/최적화 없음 - greedy PEG 골격에 예측 제약을 삽입한 결정적 구성 절차.
9. 결과: `(3,dc)`-regular girth-10에서 Wang hill-climbing 대비 `Pmin` 개선, `4<=dc<=6`은 Karimi-Banihashemi 하한 달성; girth-12는 대수적 설계 대비 짧은 circulant; BPSK/AWGN sum-product(최대 50 iter)에서 error-floor 영역 FER 개선.
10. 한계: HW 미설계·시뮬만, 예시는 dv=3 regular 위주, NAND soft-read/throughput/latency 무관, 확정 rate/length 미기재.

| PEG-undetectable cycle 제거 조건식 | 대응 없음 | 오프라인 부호구성 도구 영역, 런타임 디코더 모듈 밖 |
