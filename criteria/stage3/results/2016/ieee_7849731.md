### B. 알고리즘 요약

1. 시스템: 3 bits/cell MLC NAND, rate-0.784 length-2331 (504,2331) binary QC-LDPC(대부분 VN degree 8, CN degree 36 constant), all-bit-line 구조.
2. 문제: MLC CCI로 raw BER 증가, SPA는 정확하나 복잡도 과다 → 저복잡도 복호+정확한 LLR 필요.
3. 채널 모델: victim cell threshold voltage가 인접 3셀 coupling(`γy`,`γxy`)으로 shift, PDF는 erased/programmed state convolution `p(k)=pp_k⊗pc`.
4. 핵심 기법1: 비균일 검출(`N-UD`) — overlap 영역에 soft-decision 기준전압 `±q` 2개만 두고, 평균 최대 mutual information으로 `q` 값 결정(q=0.06V).
5. 식 의미: LLR `L(z_i)=log(Σ_{k∈Si}p(k)/Σ_{k∉Si}p(k))`을 실제 임계전압 구간 [Rl,Rr] 적분으로 계산 → 검출 정확도가 LLR 정확도를 좌우.
6. 핵심 기법2: MSRBI-MLGD — SRBI-MLGD에 power-function 기반 비균일 양자화 적용, |LLR|이 0 근처면 고정밀·먼 값이면 저정밀 양자화.
7. 구현 디테일: 양자화 레벨 `m`, power `r=log_{Q8}Lth8`(r>1 위해 |LLR| 배율 β=100 확대), 임계 `Lth_j=Q_j^r`, 초기 신뢰도 `φi=Q×sign(L(zi))`.
8. 최적화: max|LLR|=6.5, β=100은 다수 시뮬로 결정.
9. 결과: N-UD가 UD 대비 BER 개선; MSRBI-MLGD가 SRBI-MLGD 능가(iter>15부터), m=6이면 min-sum에 근접, m>5 이후 이득 포화; 최대 iteration 100.
10. 한계: HW 미설계, gate/throughput 없음, MLGD 계열이라 min-sum 복호기와 코어 상이, N-UD는 약간의 latency 증가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| N-UD soft-read 기준전압(±q) 최적화 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | mutual-info 기반 임계전압 설정을 우리 soft-read threshold에 접목 가능 |
| MLC LLR 계산(PDF convolution) | `channel.cpp` `Set_LLR_Th()` + LLR 테이블(`PARAM_LLR`) | MLC 채널 LLR 생성 방식 참고 |
| power-function 비균일 LLR 양자화 | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | 비균일 양자화 테이블 아이디어는 이식 가능 |
| MSRBI-MLGD 복호 알고리즘 | 대응 없음 | Prime ECC는 min-sum 기반, MLGD 복호 코어 미보유 |

적용 가치: **중간** — N-UD 소프트 리드전압 최적화와 비균일 LLR 양자화는 우리 채널/LLR 테이블에 이식 가치가 있으나, 복호 코어(MLGD)는 우리 min-sum과 달라 알고리즘 전체 이식은 부적합.
