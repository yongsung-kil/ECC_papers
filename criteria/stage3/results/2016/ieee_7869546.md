### B. 알고리즘 요약

1. 시스템: 1-D magnetic recording(MR) 채널용 비이진 array-based spatially-coupled(AB-SC) LDPC, 대표 예 GF(4), `p=23`, `γ=3`, `L=5`, code length `5290`bit, design rate 0.84~0.87.
2. 문제: SC 부호의 점근(asymptotic) 분석은 cycle-free·평균화 가정 때문에 유한길이·비AWGN(MR) 설정에 직접 적용 불가.
3. 핵심 가정: 반복 복호에서 error-floor를 유발하는 조합 객체는 absorbing set(AS), MR 채널에선 balanced absorbing set(BAS)만 문제; 관심 AS는 모든 VN을 잇는 cycle이 존재한다고 가정.
4. 핵심 기법 1: SC 부호의 반복 구조를 이용해 전체 Tanner 그래프 대신 소형 search matrix `A_{n,β}`만 탐색하는 `(a,b)` AS 열거법.
5. 핵심 식: `β=⌊a/2⌋(γ-1)+1` (AS의 VN이 걸치는 최대 연속 region 수), Theorem 1로 전체 AS 수 `F`를 region별 시작 개수 합으로 계산 — 복잡도가 coupling length `L`에 무관.
6. 핵심 기법 2: `γ=3` AB-SC의 지배적 BAS `(6,0)/(8,0)/(10,0)`가 공통으로 `(4,4)` AS를 하위구조로 가짐 → 이 "공통분모"를 최소화.
7. 부수 트릭: cutting vector `ζ`로 partition H0/H1을 조절, 최적 `ζ=[5 11 18]`이 `(4,4)` AS를 degenerate 대비 ~60%, 평균 대비 ~20% 감소.
8. 최적화 절차: cutting vector 최적화 후 non-binary edge weight 최적화로 잔여 BAS 추가 제거.
9. 결과: cutting vector + edge weight 모두 최적화(OO) 시 error-floor에서 약 2 orders of magnitude(FER 2.33e-6 → 4.11e-8) 개선.
10. 한계: HW 미설계·시뮬만, non-binary GF(4)·MR 채널 특화, higher column weight/다른 underlying code로 확장은 future work.
