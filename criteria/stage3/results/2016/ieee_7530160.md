### B. 알고리즘 요약

1. 대상: `16×32` sparse PCM(message 16bit → codeword 32bit, rate 0.5), AWGN+BPSK, 25000 random 메시지로 시뮬.
2. 문제: SPA는 성능 좋으나 iteration이 많아 복잡도 높고, OSMLGD(One-Step Majority Logic Decoding)는 단순하나 성능 낮음 → 둘의 절충.
3. 채널 모델: `yj = (2vj−1) + xj`, `xj`는 평균0/분산`N0/2` Gaussian, 입력 LLR `ri = 4yi/N0`.
4. SPA(SDD): CN 외적정보 `Ej,i = log((1+∏tanh(Mj,i'/2))/(1−∏tanh(Mj,i'/2)))`, VN 갱신 `Mj,i = Σ Ej',i + ri`, syndrome=0이면 종료.
5. OSMLGD(HDD): syndrome `s(k)=z(k)H^T` 계산, reliability `Rj` 를 외적정보 `Ej=Σ(2σ−1)`로 갱신 후 hard-decision.
6. 핵심 기법(Hybrid): SPA를 40 iteration 수행 → syndrome=0이면 종료, 아니면 reliability 값을 OSMLGD에 넘겨 추가 복호.
7. 구현 트릭: quasi-uniform quantization(지수 증가 간격)을 입력 LLR·VN↔CN 메시지에 적용, tanh는 box-plus `x⊞y = sign·sign·min(|x|,|y|)+s(x,y)`로 근사.
8. (학습 절차 없음)
9. 결과: BER-SNR 그림상 Hybrid > SPA > OSMLGD 순 성능, SPA 20~50 iter에서 Hybrid 성능 거의 동일. Table I에 단일 iteration당 연산 수(wr/wc 기반) 비교.
10. 한계: 32bit 소규모 코드 MATLAB 시뮬뿐, HW 미설계, 정량 BER 수치는 그림에만 존재(본문 부재).
