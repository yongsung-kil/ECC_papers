### B. 알고리즘 요약

1. 대상은 multilevel NAND flash를 memory channel로 본 1차원 causal 모델 (식 (1)): `Yn = Xn + An·Xn-1 + Bn(Yn-1-En-1) + Wn + Un`, 입력 `Xn ∈ {v0..v_{M-1}}` (M-level).
2. 문제: intercell interference(ICI)와 P/E cycling을 포함한 memory 채널의 이론적 저장효율 한계(capacity)를 정의·계산 가능한가 (기존 [2]는 검출기 설계만).
3. 모델 가정: `A,B,E,W`는 i.i.d. Gaussian, `σ²_B<1`(시간무한대 안정성 보장), `U`는 균등분포, 시작시각 0에서 ICI 없음.
4. 핵심 결과 1 — indecomposability: `|f_{Yn|X}−f_{Ỹn|X}| < σ_B^{2n}(...)` 로 초기조건 영향이 지수 소멸 (Theorem II.1), 따라서 information capacity `C_Shannon`이 well-defined.
5. 왜 필요: 출력 process가 비정상(non-stationary)이라 entropy rate 존재가 자명하지 않음 → asymptotic mean stationarity로 해결.
6. 핵심 결과 2 — AEP: stationary·ergodic 입력에서 `(1/(n+1))log f_{Y}(Y) → H(Y)` (Theorem III.3), ergodic theory(Birkhoff)로 증명.
7. 핵심 결과 3 — capacity 등가: stationary capacity `CS = C_Shannon` (Feinstein 방법 변형, blocked process mixing).
8. Markov 근사: `m`차 Markov capacity `C^{(m)}_Markov → CS` as `m→∞` (Lemma IV.2, [4] 병렬 논증).
9. 최종: `C = C_Shannon = CS = lim C^{(m)}_Markov` (Theorem IV.3) — operational capacity가 Markov 근사로 접근 가능함 확립.
10. 한계: 부호설계·디코더·HW 전무, 실제 capacity 수치 계산 알고리즘은 향후과제, 순수 존재성/등가성 증명.
