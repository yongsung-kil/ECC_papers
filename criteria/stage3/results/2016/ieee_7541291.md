### B. 알고리즘 요약

1. TDMR(2D 자기기록) 채널, Voronoi grain 기반 media model, 3×3 ISI span. 32×32 페이지 단위 검출.
2. 풀려는 문제: TDMR은 grain 경계 media noise가 down/cross-track 상관성을 가져 2D ISI+media noise 동시 처리 필요.
3. 채널모델: read-back `y_{i,j}`는 3×3 span의 `x_{i,j}`에 의존, pattern-dependent media noise를 AWGN(`σ²_{xR}`)으로 근사.
4. 핵심 기법(GBP): region 단위 메시지 전달 알고리즘. Gibbs free energy를 Kikuchi/RBA 근사(`F̂`)로 최소화, overcounting `cR`로 region entropy 합산.
5. 핵심 식: 제약 `Σ b(xp)=b(xR)` 하에서 Lagrange multiplier로 marginal `b(xR)` 최적화 → APP/soft-LLR 산출.
6. 이론 기여: pseudo-marginal 공간 `Δ_P`에서 GBP hard decision이 FER-optimal 결정영역 `D̂_FER`로 수렴함을 convex opt으로 증명(Prop 1,2).
7. region 선택: 3×3 이하 모든 sub-region(2×3,3×2,...,1×1) 포함 시 최적, `R=R̄`가 유일해·balanced 보장.
8. LDPC 연동: GBP soft-LLR을 BP LDPC 복호에 입력. SIR(mutual info) 상·하한으로 채널용량 추정해 rate 선택.
9. 결과: region 전부 사용 시 JTED 대비 0.2dB 우수. 부호 QC-LDPC N=756, R=0.66, circulant L=126, small trapping set free(구성법 [12] 인용). TW=16.2nm에서 BCH(1023,675) 대비 FER 2자릿수 이상 이득.
10. 한계: 시뮬만, HW 미설계, 검출기(GBP) 중심이라 LDPC 복호기 자체는 표준 BP, TDMR/Voronoi 채널 전용 가정.

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |

| GBP soft-LLR → BP LDPC 복호 | decoder.cpp LDPC_Decoder() | LDPC 복호 대응하나 논문은 표준 BP만 사용(개선 없음) |
| soft-LLR(APP ratio) 생성 | channel.cpp Set_LLR_Th() | soft-read LLR 개념 유사, 단 2D 자기채널 산출식은 미대응 |

```

적용 가치: 낮음 - 논문 핵심은 TDMR 채널용 GBP 2D-ISI 검출기이며 LDPC는 표준 BP로만 사용되어, NAND용 binary QC-LDPC 복호기/부호설계를 개선하는 이식 요소가 없다.
