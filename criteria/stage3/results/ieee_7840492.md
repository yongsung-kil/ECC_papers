### B. 알고리즘 요약

1. 대상: BSC 상 irregular LDPC를 Gallager A(이진 message passing, 메시지값 0/1) 하드복호로 분석; all-zero codeword 송신 가정.
2. 문제: BP 기반 반복복호의 정정능력을 이론적으로 규명하기 어려움. Chilappagari의 column-weight-3 regular 분석을 irregular로 확장.
3. 모델: 수신값 `rv`, 반복 l의 VN→CN 메시지 `wl(v,c)`, CN→VN `wl(c,v)`; `Mrv`(수신값과 같은 입력수), `M1`(값1 입력수)로 갱신식 (1)/(2) 정의.
4. 핵심개념 `2-cycle`: good VN degree=2, bad VN degree>=2인 부분그래프; 길이 k, bad VN 수 w인 `(k,w) 2-cycle`로 실패집합 정의(degree-2 VN 역할 강조).
5. 핵심정리(Thm 2/3): 2-cycle 내 VN·CN은 이웃 메시지를 무변경 전달; `vi`는 l-hop 떨어진 VN의 수신값을 even-degree CN 경유로 수신 `wl(ci,vi)=rv_{(i+l) mod k/2}`.
6. 실패조건: `(k,1)`은 k=4에서만 실패(k>4는 1st iter 정정); `(k,2)`는 두 bad VN 거리 `d1<=d2`, `k/2=d1+d2`로 `d1>2`면 성공(Lemma 3).
7. 정정한계(Lemma 6/8): `k/2` 짝수일 때 d1=1은 k>8, d1=2는 k>16에서 정정; `k/2` 홀수는 k>8에서 정정 → Table I로 (k,w)별 성패 정리(Thm 4).
8. 학습절차: 없음 (조합론적 정리 증명).
9. 결과: PEGirReg252x504는 실패유발 2-cycle 5380개, PEGirReg504x1008은 8710개 발견; 낮은 crossover prob 영역에서 시뮬 FER을 2-cycle 이론이 하한(lower-bound)함을 확인.
10. 한계: HW 미설계, Gallager A/BSC 하드복호 전용, 정정능력 개선이 아닌 실패집합 특성화(분석 도구).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Gallager A 이진 message passing | 대응 없음 | Prime ECC는 min-sum(magnitude 양자화) 사용, Gallager A 하드 알고리즘 미채택 |
| 2-cycle 실패집합 / error-floor 예측 | [decoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `LDPC_Decoder()` (간접) | 복호실패 분석 관점은 유사하나 min-sum 수렴실패(trapping-set)와 모델이 달라 직접 대응 없음 |
| BSC hard-decision 채널 | `channel.cpp` (간접) | Prime ECC의 hard-1bit(HD) 모드와 채널 유형만 유사, 나머지 불일치 |

적용 가치: **낮음** — Gallager A/BSC 하드복호 전용 조합론적 실패집합 이론으로, min-sum·soft-read 기반 NAND ECC 디코더와 모델이 근본적으로 달라 직접 이식 대상이 아님.
