### B. 알고리즘 요약

1. 대상은 circulant permutation matrix 기반 binary (j,k)-regular QC-LDPC 부호설계(디코딩은 SPA, 최대 50 iter).
2. 문제: random LDPC는 구조 부재로 인코딩 복잡·PCM 저장 메모리 과다, 기존 QC(SFT/Fossorier) 대안 필요.
3. GF(p)(p 소수)에서 원소를 뽑은 두 시퀀스 `{a_i}`(길이 j), `{b_v}`(길이 k)로 예비행렬 `Y` 정의.
4. 각 원소 `y_uv = [d(a_u+b_v)^2 + e_u + e_v] mod p`(이차합동식), `d∈{1..p-1}` 고정.
5. `y_uv`를 shift량으로 갖는 `p×p` circulant permutation matrix `I(y_uv)`를 `j×k` 배열로 배치해 PCM `H`(크기 `jp×kp`) 생성.
6. 핵심 성질: OCS(one-coincidence sequence) 특성상 임의 두 행의 차분이 서로 달라 Tanner graph girth ≥ 6(4-cycle 제거) 정리로 보장.
7. 블록길이 `N=kp`, rate `r=K/N ≥ 1-(j/k)`; QC 구조로 PCM 저장 메모리 `1/p`배 절감, shift-register 인코딩 가능.
8. 예: (3,5) p=31 → [155,64] rate≈0.413; (4,16) rate=0.75 N=1648/4016; (3,17) rate≈0.824 N=1037/5219.
9. 결과: 짧~중간 길이에서 random보다 근소 우위, SFT 코드와 거의 동등 성능.
10. 한계: 긴 길이(3,5)에서 최소거리 한계((j+1)!)로 error floor 발생, HW 미구현·시뮬만.
