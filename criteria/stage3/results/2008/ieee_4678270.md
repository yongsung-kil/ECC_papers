### B. 알고리즘 요약

1. 시스템: BPSK/AWGN, sum-product(BP) 복호, irregular LDPC (rate 1/2 중심, 2/3·3/4 확장), length 504/1008/2016.
2. 문제: Brink의 EXIT 설계는 check node degree `dc`를 사전 고정해 degree 최적화 자유도가 낮음.
3. 핵심 모델: iterative decoder를 VND/CND 두 SISO 디코더로 분리하고 EXIT chart의 mutual information `I_A`/`I_E` 전달곡선으로 수렴 거동을 시각화.
4. EXIT 함수: CND `I_{A,CND}=J(J^{-1}(1-I_E)*sqrt(dc-1))`(SNR 무관), VND `I_{E,VND}=J(sqrt((dv-1)[J^{-1}(I_A)]^2 + σ_ch^2))`, `σ_ch^2 = 8R*Eb/N0`.
5. irregular 평균 VND 곡선 = `Σ b_i*I_{E,VND}(dv,i)`, `Σ b_i = 1` (`b_i`=degree dv,i edge 비율).
6. 핵심 기법: objective `F = Σ_j (Σ b_i I_{E,VND} - I_{A,CND})^2`를 VND>CND 제약 하에 최소화하되 `b_i`(0.0001~0.98)와 `dc`(6~15)를 **동시** exhaustive 탐색(max I_A=0.98, 9800 샘플).
7. 결과: rate 1/2에서 joint 최적 `dc=8`(우연히 Brink와 동일)이나 vn 분포 상이, `F=3.154<3.319`(non-joint), 복호 threshold 0.4413dB (Shannon 0.1868dB에서 0.2545dB gap).
8. threshold 검증: length 504/1008/2016 모두 threshold ~0.5dB로 수렴, 2dB에서 (2016,1008)은 평균 10 iteration.
9. 확장: rate 2/3(dc=13, gap 0.2285dB), 3/4(dc=15, gap 0.3304dB)로 임의 rate 설계 가능.
10. 한계: HW 미설계, waterfall threshold만 최적화(error-floor 미고려), irregular·저rate 중심.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| EXIT 기반 degree distribution 설계 | ecc_top.cpp Load_PCM() | 부호 degree 분포 결정 단계에 대응하나 Prime ECC는 고정 구조 QC-LDPC |
| irregular VND/CND degree 최적화 | 대응 없음 | Prime ECC는 구조적 QC-LDPC(VN degree ~17 고정) |
| BP/sum-product 복호 | decoder.cpp LDPC_Decoder() | Prime ECC는 min-sum 채택, EXIT 전제인 full-BP과 상이 |
| soft in/out mutual information | channel.cpp Set_LLR_Th() | LLR 설계엔 닿지만 EXIT 최적화 자체는 미대응 |

적용 가치: 낮음 — EXIT joint degree 최적화는 저rate irregular 코드 설계 도구로, Prime ECC의 고rate 구조적 QC-LDPC와 부정합하고 error-floor를 고려하지 않는다.
