### B. 알고리즘 요약

1. 시스템: 단일 relay 협력 통신, half-duplex(broadcast/MAC 2 time slot), BPSK/AWGN, 부호 `w1`(rate `R1=1/2`)에서 확장한 `w2=[w1, we]`(rate `R2=1/4`), 길이 `N1=20000`, `N2=40000`.
2. 문제: 한 codeword `w2` 내에서 region-1(`γ1`)과 region-2(`γ2`)가 서로 다른 SNR을 겪어(`γ2=2γ1`), SNR 불변을 가정한 기존 DE/Gaussian approximation이 성능 예측에 부정확.
3. 모델: rate-compatible 패리티행렬 `H2`(좌상 `H1`, 우상 0, 하단 `A,B`)로 code extension 구조를 정의, region별 노드 확률집합 `{π1(i),π2(i)}`로 표현.
4. 핵심기법1: region별 edge 분포 `λ(i,k)`를 분리 정의하고, VND/CND LLR을 region별 서로 다른 Gaussian 밀도로 전개하는 수정 Gaussian approximation DE (Algorithm 1).
5. 핵심식: `Pk(i)=iaiπk(i)/Σjaj`로 CND 입력 edge를 region별로 배분, `ψ(x)=E{tanh(y/2)}` 기반으로 region-1/2 평균 LLR을 분리 계산 (식 14,15).
6. 핵심기법2: multi-edge type LDPC framework를 relay의 rate-compatible 제약으로 확장한 것으로 해석.
7. 구현 디테일: rate-compatible 제약을 edge 분포 제약(식 20,21,23)으로 변환하여 최적화 탐색공간에 반영, 초기 노드 확률집합은 고차 VN을 region-1에 배치.
8. 최적화: differential evolution(step size `α,β<1`)로 후보 앙상블 `C(n)`을 갱신, Algorithm 1로 각 앙상블 threshold(`mdec`) 평가 후 최적 앙상블 선택 (Algorithm 2).
9. 결과: 최적화된 relay 부호가 repetition scheme 및 기존 Gaussian approximation 설계 대비 유의미한 이득, capacity bound CB-3과 0.5dB 미만 gap.
10. 한계: half-duplex relay 채널 특화, HW 미설계(시뮬만), error-floor 미고려, 랜덤 irregular 부호(QC 아님).
