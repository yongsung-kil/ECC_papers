### B. 알고리즘 요약

1. 대상은 protograph QC-LDPC(base matrix `Hb` r×s, 확장인자 `m`, circulant `R^j` 치환)로, WiMax 802.16e 부호와 랜덤 (3,6)/irregular 부호에 적용.
2. 문제: BEC에서 반복복호가 정정 못 하는 최소 burst 길이(=stopping set의 span)를 최대화하는 bit-node 순열(interleaver) 찾기.
3. 정의: `Span(S)=max|x−y|+1`, `Span(H)=Span(X)` (X=모든 stopping set 모음); `Span(H)`가 정정 불가한 최소 burst 소실 길이.
4. QC 축약: 확장행렬의 stopping set은 base matrix stopping set과 many-to-one 대응(Thm 3.2), 따라서 base matrix(예: 24열) span만 최적화하면 됨 → 탐색공간 대폭 축소.
5. 경계식: `(b−2)m+2 ≤ t ≤ bm` (Thm 3.3, b=base span, t=확장 span), 그리고 `Span(H')≤r`.
6. stopping set 열거 기반 상한: `2(2n+1) cycle bound`, `nCk subset bound`, `minimum weight bound` 3종을 제시.
7. 최적화: `n!` 순열 탐색은 NP-hard로 보고 simulated annealing 채택, 에너지=span, 상태전이=Metropolis-Hastings.
8. `GenPerm()`: 두 위치 `b1<b2`를 랜덤 선택해 사이 구간을 뒤집는 TSP식 이웃생성; 냉각 `t←0.9t`.
9. 결과: WiMax rate-1/2 span 3→12(상한 12 달성), rate-2/3A 4→6, rate-3/4 2→4; 기존 [8] 알고리즘 대비 우수. 랜덤부호도 193→207, 1066→1146으로 확장성 확인.
10. 한계: 순열등가라 랜덤오류 정정성능은 불변(설계 무변경), HW·soft 복호 미포함, 자기기록·무선 버스트 소실 채널 전용.
