### B. 알고리즘 요약

1. 대상: 구조기반 binary QC-LDPC(SFT, array, Song 등)를 topological graph theory의 voltage graph로 통일 해석.
2. 문제: 기존 구조기반 구성이 ad hoc하고 girth/minimum distance/stopping set를 설계·증명할 수학적 틀이 부족.
3. voltage graph: base graph X edge에 voltage group G 원소 `α(e)` 할당 → degree-|G| lift `Xα`가 파생그래프(=Tanner graph). permutation voltage(`Sn`)로 circulant shift 표현.
4. 핵심 Thm2.1: 닫힌·backtrackless·tailless walk W의 net voltage가 항등원이면 Xα의 lift `Wg`가 cycle → cycle/girth를 base graph walk로 환원.
5. SFT 코드 예: `[155,64,20]`(q=31, j=3, k=5), H는 j×k circulant array, rate `R≥1-j/k`, K3,5 base에 S31 voltage.
6. array 코드: 블록길이 `N=q²`, rate `R≥1-j/q`, Kj,q base + `πts` voltage로 해석.
7. abelian-forcing walk 도입: abelian group에선 edge 정/역 traversal 수가 같으면 net voltage=0 → 항상 cycle 유발(abelian-inevitable cycle).
8. 완전분류(Thm4.5): abelian-forcing subgraph는 theta `T(a,b,c)` 또는 dumbbell `D(a1,a2;b)`뿐, girth 상한 `2(a+b+c)` / `2(a1+a2)+4b`.
9. 결과: SFT 계열 base가 K2,3=`T(2,2,2)` 포함→girth≤12 간결 증명, [16][20]의 잘못된 정리 수정, girth g면 abelian-inevitable cycle `≥3g`(Thm4.7).
10. 한계: abelian voltage는 girth 상한에 묶임, 신규 우수부호(비abelian)는 ongoing work, 시뮬/HW/성능결과 없는 순수 이론.
