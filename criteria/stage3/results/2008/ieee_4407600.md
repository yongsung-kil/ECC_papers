### B. 알고리즘 요약

1. 대상은 자기/광 기록(HDD) 시스템의 sector failure rate(SFR) 추정 — outer ECC는 `t`-symbol 정정 Reed-Solomon, inner는 iterative 채널(LDPC/TPC/SPC).
2. 문제: SFR이 `10^-N`급으로 극히 작아 counting 시뮬로 못 구함 → error vector 통계모델로 analytic 추정 필요.
3. 방법을 철학적으로 macroscopic(error weight만 모델링)과 microscopic(심볼 오류 위치·상관까지 모델링)으로 분류.
4. Macroscopic: multinomial 모델 — error event가 독립, weight distribution `p(w)`을 다항분포로 표현, WFR=`sum_{w>t} p(w)`.
5. 파라미터 추정: (7)MSE·(8) 대신 ML=**최소 상대엔트로피(KL) 해** (9)가 최적 — tail 확률을 정확히 추정, gradient descent로 최적화.
6. Microscopic: 심볼오류를 Markov chain(Fig.3 상태도)으로 모델링, 전이행렬 `P(x)`의 monomial로 run-length별 오류수 열거, 다항행렬 곱으로 weight distribution 계산.
7. 확장(Sec IV): error indicator vector를 error-event(EE)와 error-free run으로 파싱, EE는 i.i.d., 개수 `Q(k)`는 별도 분포로 모델링.
8. 핵심 관찰: iterative 채널은 interleaver 때문에 오류가 흩어져 장거리 상관 발생 → macroscopic이 부정확, microscopic이 적합. `Q(k)` tail이 LDPC/TPC는 지수, SPC는 Poisson.
9. 결과: TPC 24dB 시스템(10-bit 심볼, 79711 sector) 예제에서 부분 데이터로 추정한 analytic SFR이 전체 데이터 counted 값과 잘 일치(Fig.7).
10. 한계: 순수 추정 방법론(HW 없음), 특정 code/interleaver마다 고유 microscopic 모델 필요, LDPC는 iterative 예시로만 등장.
