### B. 알고리즘 요약

1. 대상: 임의 block length·rate `(n,k,d)` 부호 중 Tanner graph의 (simple) cycle이 서로 node를 공유하지 않는(non-overlapping) 부호.
2. 동기: SPA(Sum-Product)는 cycle-free 그래프에서만 정확한 posterior를 주지만 cycle-free 부호는 최소거리가 나빠 optimal decoding과 min-distance 사이 trade-off 존재.
3. 관찰: non-overlapping cycle이면 tree inducing cut `XT=|E|-|V|+2`가 작아 minimal tree complexity `t(C)≤XT`가 낮아 합리적 복잡도로 optimal decoding 가능 → 이 부호류의 min-distance가 관심사.
4. 분해: non-overlapping cycle 그래프를 base graph/base loop/branch로 정의하고 gluing·joining 연산으로 재귀 구성.
5. 핵심 개념 critical set: base loop `L(n,m)`에 대해 `|S|=n-m+1`개 codeword 집합 `S`로 모든 좌표 `j`에서 `Σ s_ij=2`가 성립하는 집합 존재(Prop 1).
6. critical set 전파: gluing(Thm 1)·branch joining(Thm 2)·base graph joining(Thm 3)에서 critical set이 보존되어 모든 non-overlapping cycle 그래프가 critical set 보유(Cor 2).
7. 핵심 상한(Thm 4): critical set의 각 codeword weight `≥d`, 좌표별 합이 2이므로 `2n≥(n-m+1)d` → `d≤2n/(n-m+1)`.
8. full-rank이면 `d≤2n/(k+1)≤2n/k=2/R` — 즉 rate 고정 시 min-distance가 block length로 증가하지 않음(asymptotically bad).
9. 비교: cycle-free 부호 상한 대비 최대 1만 큼 → cycle이 있어도 non-overlapping이면 이점 거의 없음, 좋은 부호는 필히 overlapping cycle 필요.
10. 부록: cycle 내 degree-2 check node는 row operation(`c2→c2+c1`)으로 제거해 cycle check node degree≥3 가정을 WLOG 확보. 한계: 순수 이론, 시뮬/구성/HW 전무.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| non-overlapping cycle 최소거리 상한(이론) | 대응 없음 (부호설계 이론, 코드 무관) | Prime ECC QC-LDPC은 overlapping cycle 보유, 본 bound가 제약하지 않음 |

적용 가치: **낮음** — 순수 이론 impossibility 결과로 이식할 기법·구성법이 없고, Prime ECC의 QC-LDPC은 이미 overlapping cycle 구조라 실질적 시사점이 없다.
