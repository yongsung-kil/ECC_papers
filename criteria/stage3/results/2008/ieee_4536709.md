### B. 알고리즘 요약

1. 시스템: binary QC-LDPC, `J×L`(12×24) mother PCM `H=[HS|HP]`, 각 sub-matrix는 `q×q`(q=96) shifted identity 또는 null, code length 2304 고정.
2. 문제: multi-rate LDPC에서 puncturing은 길이 감소로 BER 열화·복호복잡도↑; row-removing은 성능 좋으나 인코딩 복잡도(quadratic) 문제.
3. 핵심: mother PCM을 triangular + dual-diagonal 형태로 설계 → row-removing으로 rate를 조정하면서도 linear encoding 유지.
4. 구성: shift 계수 `C_j,l`를 Fossorier 방식으로 short-cycle 회피, 우하단 dual-diagonal로 error-floor 저감, 최하위 4 block-row는 row degree 20으로 성능 보장.
5. 다중rate: 최상위 block-row를 순차적으로 숨겨(hide) rate 12/24~20/24의 9개 daughter code 생성 (`R=1-J/L`), 모든 rate 동일 길이(2304).
6. universal encoder: `Hx^T=0`에서 codeword를 24 파트로 분할, 삼각+dual-diagonal 구조로 parity를 순차 matrix-vector 곱(식 (6)~(9))으로 계산, daughter code는 시작 layer만 이동.
7. universal decoder: joint row-column BP로 마지막 layer부터 복호, daughter code는 `i`번째 layer에서 정지.
8. 적응 변조: multi-level QAM과 결합, spectral efficiency `η=R·log2(M)` 1~5 bit/symbol/Hz, QAM demapping LLR은 min-approx(식 (13)~(17)).
9. 결과: FPGA(Virtex-4) universal encoder가 138 XOR gate·100MHz로 1.2~3.6 Gb/s, BER curve error-floor 없음(10^-7까지), Rayleigh에서 η=1~5가 Eb/N0 5/6/10/14/16 dB에서 신뢰전송.
10. 한계: 무선(QAM/fading) 특화, rate 0.5~0.83, 디코더는 개념/선행연구 수준, dual-diagonal 인코더는 이미 표준.
