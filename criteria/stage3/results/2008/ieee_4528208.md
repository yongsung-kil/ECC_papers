### B. 알고리즘 요약

1. 대상: 40Gb/s long-haul 광통신용 FEC로, 구조적 block-circulant(array) LDPC를 고속 구현 가능한 대안으로 제안.
2. 문제: random LDPC는 성능 좋으나 구조 없어 구현 곤란, 구조적 LDPC는 성능손실 -> girth를 최적화 파라미터로 삼아 손실 회복.
3. 이론 근거: Tanner 최소거리 하한식 `(1)`에서 열가중치 `r>=3`이면 girth `g`가 클수록 최소거리 `d`가 지수적으로 증가(예 g=10, r=3 -> `d>=10`).
4. 부호 구조: H는 순열행렬 `P`의 거듭제곱 `P^(j*ci)` 블록으로 구성 `(2)`, `p`는 소수.
5. cycle 회피 조건: 길이 `2k`(k=3,4) cycle은 지수합 등식 `(3)`을 만족할 때 존재하므로, 이를 만족하지 않는 정수열 `ci`를 선택.
6. 구성 알고리즘: 초기집합 `S`에 `Q={0..p-1}`의 정수를 하나씩 추가하며 식 `(3)` 위반 시 제거하는 자동 computer-search(사용자 개입 최소).
7. rate 제어: `R>=1-r/n`, `n=r/(1-R0)` (예 R0=0.8, r=3 -> n=15). 예: q=1129, |S|=15 -> LDPC(16935,13550), girth-10.
8. 복호: min-sum-with-correction-term. 열가중치 3~4로 낮아 디코더 복잡도 낮음, 인코더는 shift-register+mod-2 adder.
9. 결과(AWGN): girth-10 LDPC(24015,19212) R=0.8이 BCH×BCH TPC 대비 0.95dB(BER 1e-10), NECG 10.95dB(BER 1e-12), RS 대비 최대 4.75dB 우수. 광 40Gb/s 시뮬서 TPC 대비 1.47dB(burst+random error).
10. 한계: HW 미설계, 성능은 시뮬(AWGN + 광 fiber model)만, rate 0.8 고정.
