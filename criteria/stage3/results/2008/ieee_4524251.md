> 총평: 작은 QC-LDPC 성분코드를 CRT-일반화 결합+row 순열로 이어붙여 6-cycle을 줄이고 waterfall 0.5dB 개선하는 순수 부호 구성법 - Prime ECC의 고정 고rate H-matrix 대체가 필요해 이식성 낮음.

### B. 알고리즘 요약

1. 시스템: `L×L` circulant permutation 행렬 기반 binary QC-LDPC. shifting matrix `S(H)=(aij)`, mother matrix `M(H)`로 부호구조 표현. SPA 복호(최대 50 iteration), AWGN/BPSK.
2. 문제: 기존 CRT 결합법[4]은 중간 파라미터가 제약식으로 **유일 결정**돼 성분코드당 부호 1개만 설계 가능, array-code 유사 구조 탓에 Tanner 그래프에 6-cycle이 많아 성능 부족.
3. 모델: 성분코드 `Ck`(k=1..s)는 mother matrix가 모두 동일하고 circulant 크기 `Lk`가 서로소(gcd(Ll,Lk)=1), `L=ΠLk`. 2l-block-cycle이 `Hk`에서 2l·rk-cycle을 유발(Theorem 1).
4. 핵심기법: 결합 shifting matrix `aij = Σk a^(k)_ij·Ak·L̄k mod L` (식 4), `L̄k=L/Lk`, `Ak`는 `gcd(Ak,L̄k)=1`로만 완화(유일 제약 제거).
5. 핵심식(Theorem 2): 결합 부호의 cycle 최소정수 `r=Σk rk` → girth(C) ≥ 각 성분코드 girth. 4-cycle 없음 보장, 최소거리 ≥ m+1.
6. 확장: 조건 완화로 `Σφ(Lk)`개, 성분코드 block-row 순열까지 포함 시 `(m!)^{s-1}·Πφ(Lk)`개 부호 설계 가능.
7. 구현 트릭: 각 `S(Hk)`의 6-block-cycle 위치를 찾아 공통 6-cycle 수 N을 최소화하는 row 순열 조합을 탐색(bubble 유사), 최적 조합 선택.
8. 최적화: 학습 없음. 순열 조합 완전탐색으로 6-cycle 최소 조합 선택.
9. 결과: (539,310)은 6-cycle 1386→154, (924,695)은 3696개 제거, (1573,1146)은 7150→858. BER 1e-6에서 CRT 대비 **0.5dB** 이득. 부호 rate 0.57~0.85, 길이 539~2860.
10. 한계: 순수 구성법(HW·복호 알고리즘 미설계), 시뮬만, 산출 rate가 중간대(0.57~0.85)로 NAND ECC 고rate와 불일치.
