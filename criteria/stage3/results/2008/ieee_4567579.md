### B. 알고리즘 요약

1. 대상: 임의 binary LDPC 부호(Tanner `(155,64)`, Margulis `(2640,1320)`, MacKay `(1008,504)`); 채널 BSC(Gallager B) / AWGN(min-sum).
2. 문제: 고SNR error floor는 Tanner 그래프의 dominant trapping set이 지배; girth 최대화만으로는 최적이 아님.
3. 핵심 정의: `(a,b)` trapping set = `a`개 variable node + `b`개 odd-degree check; critical number `k` = trapping set 진입에 필요한 최소 초기 오류 비트수.
4. 핵심 기법: 부호 `C` 두 복사본으로 `H(2)=diag(H,H)` 구성 후 복사본 간 edge swap (`H0`,`B`로 분할, `H0+B=H`)으로 dominant trapping set 파괴.
5. 알고리즘: trapping set을 critical number 순 정렬 → 최소 critical set의 edge 하나 swap → 나머지 edge freeze(같은 critical number 재도입 방지) 반복.
6. rate 보존: Theorem 1 — `r(2) ≥ r`; `H` full rank면 `r(2)=r` (Margulis rate 0.5 유지).
7. min distance: Theorem 2 — `dmin ≤ d(2)min ≤ 2·dmin` (경계 sharp).
8. 확장: `H`를 `H0`,`B`로 분할해 convolutional(SC) LDPC로 unwrap 가능, pseudoweight 상승.
9. 결과: 수정 Tanner `(310,126)` FER slope ≈20 (원본 14, random 12); Margulis `(5280,2640)`는 `(4,4)` trapping set 제거→`(5,5)`가 지배; rate penalty 0.4129→0.4065(Tanner) 또는 무손실(Margulis).
10. 한계: HW 미설계·시뮬만, 길이 2배 증가, dominant trapping set 사전 파악 필요, QC 구조 미고려.
