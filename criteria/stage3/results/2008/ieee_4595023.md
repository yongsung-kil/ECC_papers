### B. 알고리즘 요약

1. 대상: `γ-left regular` LDPC (column weight γ, (γ,ρ) regular), BSC 상 hard-decision `bit flipping`(serial/parallel) 복호.
2. 문제: 반복 복호기의 *보장* 정정능력(worst-case)은 BEC의 stopping set과 달리 BSC/AWGN에서 미해결. code 파라미터만으로 경계를 얻고자 함.
3. 접근: 필요한 expander 확장률을 만족하는 variable node 집합 크기를 Moore bound로 산정 (random expander 대신 girth·column-weight 사용).
4. 핵심 정리(Thm 3): `g(G)=2g'`일 때 `k < n0(γ/2, g')`인 모든 `k` variable node 집합은 최소 `3γ/4` 배로 확장.
5. 의미: Sipser-Spielman 정리(3γ/4 확장→정정)와 결합해 보장 정정능력 하한 도출.
6. 따름정리(Cor 1): column-weight γ, girth 2g' 부호는 `n0(γ/2, g')/2` 미만 오류를 bit flipping으로 정정 보장.
7. 상한(trapping set): fixed point로 trapping set 정의, `(a,b)` trapping set. 조건(a)만 쓴 potential trapping set으로 일반 경계화.
8. cage graph 연계(Thm 5): 최소 potential trapping set 크기 `= nc(⌈γ/2⌉, g')` (cage graph 노드 수). Thm 6: 이 크기 오류를 못 고치는 부호가 존재.
9. 결과: 정정능력이 girth에 지수적으로 증가(단, girth는 부호길이에 로그 증가 → 선형 정정은 미증명). 상·하한 모두 제시.
10. 한계: 하한이 약하고 순수 이론(시뮬 없음), hard-decision bit-flipping 전용.
