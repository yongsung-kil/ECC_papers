### B. 알고리즘 요약

1. 위성 원격제어용 short/medium QC-LDPC 부호 구성법. base matrix `84열×42행`, 열가중치 3, rate 0.5, 확장 후 length 1008 (`p=12`).
2. 문제: PEG 부호는 BER 우수하나 인코딩 복잡도 높음 → QC 구조로 shift register 인코딩을 가능케 하되 성능 유지가 목표.
3. QC 부호 cycle 구조 분석: PCM이 `p×p` circulant 블록(식1) 구조면 길이 `2t` cycle 하나 존재 시 최소 `p`개 cycle 존재(Theorem 1) → 하나 제거하면 p개 동시 제거.
4. 핵심 기법: 작은 base matrix를 PEG로 구성 → '1'을 `p×p` circulant permutation `I(S_ij)`로, '0'을 zero matrix로 치환.
5. cycle 소거 조건(Theorem 2): 닫힌경로가 `Σ(-1)^(k+1)·S_αk,βk = 0 mod p`를 만족할 때만 cycle 형성 → shift를 이 조건 불만족하도록 정하면 base matrix의 short cycle 제거.
6. 확장 알고리즘: cycle 단위로 '1'을 circulant(shift가 Theorem2 불만족)로 치환, 나머지 '1'은 random shift, girth 탐색 후 요구 미충족 시 1~5단계 반복.
7. 다중 확장으로 더 많은 cycle 제거 가능하나 인코딩 복잡도 증가 트레이드오프.
8. (별도 학습/최적화 절차 없음 — 결정론적 cycle-by-cycle shift 배정)
9. 결과: PEG-E-LDPC의 localgirth 분포(60노드 girth6, 936노드 girth8, 12노드 girth10) — B-LDPC 대비 length-6 cycle 대부분 제거; BER은 PEG-LDPC와 근사 동등(BP·MS 모두).
10. 한계: 성능이 PEG 대비 개선 아닌 동등, HW/throughput/gatecount 없음, rate 0.5 저rate·단부호, error-floor 데이터 없음.
