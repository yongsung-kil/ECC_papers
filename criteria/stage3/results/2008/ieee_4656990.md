### B. 알고리즘 요약

1. 대상: 이진 QC protograph LDPC 부호 구성법, regular γ=3, rate 0.4~0.82, N 200~3060, AWGN/Rayleigh 채널.
2. 문제: pseudo-random LDPC(MacKay 등)는 성능은 우수하나 인코딩 복잡도가 블록길이에 2차, HW 구현 비친화적.
3. 구성 1단(protograph): 소규모 base protograph를 J배 복제 후 edge permutation으로 derived graph 생성(Thorpe protograph).
4. 구성 2단(Vandermonde): base PCM `Hb`를 순열행렬 `Pq` 거듭제곱의 Vandermonde-like 블록행렬로 구성 → QC 구조, `Nb=ρq`, `R≥1−γ/ρ`.
5. 제약: `q`는 ρ,γ보다 크고 상대적 소수 or `q>(ρ−1)(γ−1)` → 같은 행/열에서 순열행렬 `Pxq` 반복 금지.
6. 수정 PEG: girth·최소거리 최대화 + 제약1(base 구조 유지) + 제약2(QC 유지: 직전 선택 CN 최근접) → allowed check 집합을 `(Jγ choose γ)`로 축소.
7. 이점: shift-register 선형시간 인코더(`r(N−K)` 연산), semi-parallel protograph 디코더, memory-shift 기반 MAG로 배선·메모리 절감.
8. 복호: SPA(sum-product), 최대 100 iteration, BPSK.
9. 결과: (1008,504) R=0.5가 BER 10⁻⁶ @2.831dB, MacKay 대비 +0.2dB, unstructured PEG 대비 −0.06dB, 복호 복잡도 Δ≈39(AWGN,I=50).
10. 한계: HW 미구현(시뮬+복잡도표), 고rate·초단길이에서 제약 미충족 성능열화, 오류정정은 벤치마크와 동등.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Vandermonde QC protograph H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | Prime은 특정 QC-LDPC H-matrix 고정, 부호설계 이식 난이도 높음 |
| QC 순열행렬(cyclic shift) 기반 PCM | `ecc_data.h` `PCM_b`(base+shift) | 구조 유사(base+circulant)하나 rate/degree 상이 |
| shift-register 선형 인코더 | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | Prime은 dual-diagonal 인코더 보유(구성 상이) |
| semi-parallel protograph 디코더 | `decoder.cpp` `LDPC_Decoder()` | Prime은 z=32 row 병렬, 구조 상이 |
| 수정 PEG girth 최적화 구성 | 대응 없음 | Prime H-matrix는 고정, 온라인 부호 구성 없음 |

적용 가치: 낮음 — 중간 rate(0.4~0.82)·규칙적 γ=3 QC protograph 구성으로 NAND 고rate(~0.9)와 맞지 않고, 오류정정 성능도 벤치마크와 동등 수준이다.
