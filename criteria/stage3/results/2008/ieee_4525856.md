### B. 알고리즘 요약

1. 대상 부호는 binary LDPC — regular `(96,48)`,`(504,252)` (ρ=6,γ=3=(3,6)-regular)와 irregular `(200,52)`; BPSK/AWGN.
2. 문제: 그래프 메시지패싱 계열(Sum-Product/Min-Sum)과 최근 제안된 최적화기반 Gradient Projection(GP) 디코더 사이의 연결을 규명하고, 실용적 저복잡도·저지연 디코더 도출.
3. 모델: GP는 비용함수 `F(x)=∏_j f_j`, `f_j=∏_{i∈Ij} x_i`를 `[-1,1]^n`에서 gradient projection(step `α`, 사영연산 `u()`)으로 최대화 → 정점(=유효 codeword)에 수렴.
4. 핵심: GP의 그래프 모델을 도출해 `modified SP` 정의 — extrinsic 원리를 포기하고 edge 메시지 대신 근사 APP 하나만 저장/갱신, `v_i^k = v_i^{k-1} + α·Σ_{j} c_ji^k`.
5. `tanh→u`, `atanh→identity`(구간 [-1,1] 조각선형화)를 modified SP에 적용하면 원래 GP로 환원됨을 보여 SP↔GP 브릿지 확립.
6. modified SP의 C-node에 고전 Min-Sum 근사(sign·min)를 적용 → `modified Min-Sum`(GP보다 단순·우수한 실용 알고리즘).
7. 복잡도(Table I): modified MS는 mult `O(n)`·저장 `O(3n)` vs MS 저장 `O(n(γ+4))` vs GP 저장 `O(2n)`; step `α`로 수렴속도 조절.
8. 학습 없음 — `α`는 정확도-수렴속도 trade-off 수동 튜닝.
9. 결과(BLER 10^-5): modified MS가 GP보다 0.5/2.5/4.4dB 우수하나 고전 MS보다 0.7/1.0/1.3dB 열세; modified MS·GP는 훨씬 적은 iteration에 수렴(예 40/10 vs MS 160).
10. 한계: modified MS는 이미 보유한 Min-Sum보다 열등, 부동소수 시뮬만·HW/양자화 없음, non-QC 단부호 통신용.
