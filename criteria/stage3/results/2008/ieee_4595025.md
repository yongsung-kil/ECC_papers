### B. 알고리즘 요약

1. 대상: 고rate array-based(QC) 구조적 LDPC (2209,1978)/(2209,2024), rate ~0.90, quantized sum-product 복호, BPSK/AWGN.
2. 문제: finite-length LDPC의 error floor(저FER서 BER-SNR 기울기 저하)를 실측 대신 빠르게, 매우 낮은 확률(`10^-30`)까지 예측.
3. 핵심 구조: `absorbing set` — near-codeword/trapping set의 일종으로 bit-flipping에 안정, 디코더/채널과 독립 정의. `(a,b)` fully absorbing set: 각 노드가 `O(D)`보다 `F\O(D)`에 이웃이 더 많음.
4. 방법1(IS): all-zero 전송, 원래 Gaussian을 absorbing set 방향으로 mean-shift한 `f_bias`로 샘플, weight `w=f/f_bias` 재가중해 absorbing 확률 `q̂` 추정, `p̂=N_sets·q̂`.
5. 신뢰구간: delta-method로 `log10(p̂)` 분산 근사, Chebyshev로 근사 95% 신뢰구간 → HW emulation 결과가 구간 내 정합.
6. 방법2(결정론적): (a) absorbing region `R`(그 set으로 수렴하는 LLR 벡터, 채널 독립) 저차원 projection 계산 → (b) generalized Chernoff 근사로 absorbing 확률 하한 (`log P≈-d²(0;R)/2σ²`, 고SNR서 exact) → (c) `card(A)` 곱해 error floor 하한.
7. 양자화 효과: quantization 개선((6,2)→(8,4)→(9,5))하면 absorbing region 수축→error floor 개선. satisfied/unsatisfied check 차가 작은 노드에서 효과 큼.
8. 채널 확장: mixture channel(good/bad state) 등 다른 채널에도 Chernoff 근사 적용 가능(재계산은 채널 의존 항만).
9. 결과: (2209,1978) 부호 (4,8)/(8,6) absorbing set에서 IS·결정론적 추정이 HW emulation과 정합, `10^-30`까지 예측, 고SNR서 보장된 하한.
10. 한계: error-floor 예측 방법(코드/디코더 개선 아님), sum-product·array code 예시, 양자화 6-9bit.
