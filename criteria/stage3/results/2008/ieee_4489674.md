### B. 알고리즘 요약

1. 대상: 정규 `(dv, dc)` Tanner 그래프, 주 illustration은 column-weight `dv=2` (cycle graph) — 비이진 LDPC용 ultra-sparse 부호.
2. 문제: 기존 PEG는 주어진 크기에서 달성 girth가 제한적이고 target girth의 하한(cage) 크기에 못 미침.
3. 핵심기법1 - Truncated spanning tree: 트리를 최대깊이 대신 `lmax = gt`(+gap)까지만 span → diameter 논증(`d = gt/2`)과 정합.
4. 식 의미: `lmax = gt`로 제한해야 크기 `N = Ng(dv,dc)`(하한)에 도달, 더 깊이 span하면 diameter 조건 위배로 실패.
5. 핵심기법2 - 목적함수: 후보 check node 중 (a) 정확히 depth `lmax`에 있는 것만, (b) 생성 cycle 수 `nbCycles` 최소, (c) 최소 degree 후보만 남기고 그중 무작위 샘플링.
6. 랜덤화(Monte Carlo): 얕게 span→후보 다수→다양한 해 탐색, 실패 시 전체 재시작. 여러 번 돌려 best 그래프 저장.
7. gap 변수: `Ng(dv,dc) < N < Ng+2` 구간에선 diameter 논증 실패 → `lmax = gt+gap`로 시작 후 실패마다 gap 감소.
8. 결과1(cage): dc up to 50에서 girth 6,8 cage 구성; dc=3,4는 girth 16까지 하한 달성 (Table I).
9. 결과2(성능): (2,4) N=160 GF(64) 코드 girth 12→16, BIAWGNC에서 error-floor 개선; 이진 (3,6) N=504 girth 다중도 808→452, N=1008은 167→31로 감소하며 성능 우위.
10. 한계: 순수 그래프 구성법(디코더/HW 없음), 비정형 그래프(QC 구조 미생성), dv=2·rate 0.5 위주 검증.
