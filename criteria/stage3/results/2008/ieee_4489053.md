### B. 알고리즘 요약

1. 시스템: `nT×1` MISO에서 `2^M`-QAM 변조, `(3,4)`-regular `(252,63)` LDPC (`N=252`, rate `0.25`), quasi-static Rayleigh flat fading 채널.
2. 문제: turbo-type 수신기(MISO 검출기+LDPC 복호기 분리)는 3-layer Tanner 그래프의 length-4 loop로 positive feedback 발생, 단거리 부호에서 성능 저하.
3. 모델: `nT`개 전송심볼이 결합돼 1개 수신심볼 생성 → 시스템 전체를 `2^(M·nT)`-ary LDPC 부호로 간주.
4. 핵심기법1: 원 이진 PCM의 `MnT`개 연속 열을 묶어 modified PCM/Tanner graph 구성, 각 변수노드가 `2^MnT` 가지 확률 메시지를 BP로 교환 → 검출·복호를 단일 message passing으로 병합.
5. 핵심식: length-4 loop가 modified graph에선 하나의 edge로 흡수돼 positive feedback 제거(식 acyclic 시 MAP 등가).
6. 핵심기법2: edge-based message passing(EBMP) - 각 edge마다 `2^MnT` 메시지 대신 edge LLR 1개만 전달(Theorem 1/2: edge LLR이 V2C/C2V 계산에 충분).
7. 구현디테일: C2V 갱신은 `r(mn)=ln[(1+∏tanh(q/2))/(1-∏tanh(q/2))]` 형태의 full-tanh Sum-Product.
8. 채널추정: 미지 채널 시 PSAM 초기화 + EM 알고리즘으로 채널·데이터 반복 추정(디코더 soft output 사용).
9. 결과: 단거리 부호·미지 채널에서 제안 EBMP가 turbo-type 대비 BER 우세, coherence time·pilot power·변조크기 클수록 격차 확대.
10. 한계: HW 미설계, 시뮬만, non-binary 알파벳·무선 MISO 특화, gate/throughput/iteration 수치 없음.

| EBMP C2V full-tanh Sum-Product 갱신 | 대응 없음 | Prime ECC는 tanh SP를 min-sum(`CNU_Update_New_Mag()`)으로 대체 이미 보유, 오히려 역행 |
| non-binary 2^MnT-ary LDPC 구조 | 대응 없음 | Prime ECC는 binary QC-LDPC 전용 |
| MISO functional node(심볼 검출기) / EM 채널추정 | 대응 없음 | NAND 채널 밖 무선 검출·추정 영역 |

적용 가치: **낮음** — non-binary·무선 MISO 검출/복호 결합 기법으로, binary NAND QC-LDPC 디코더에 떼어 쓸 요소가 없고 CN 연산도 우리가 이미 대체한 full-tanh SP다.
