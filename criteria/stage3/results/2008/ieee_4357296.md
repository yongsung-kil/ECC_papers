### B. 알고리즘 요약

1. 목표: 조합설계(BIBD) 기반으로 girth 6/8/10인 (3,k)-regular LDPC를 명시적으로 구성 — 열가중 `j=3` (최소거리 선형증가 보장).
2. 문제: 2-D cyclic lattice로는 girth-10 불가(Lemma 1), 자유도 부족.
3. 핵심: `(C,R,Q)` 3-D cyclic lattice의 점=코드비트, 선(line)=parity-check로 두고 point-block incidence matrix를 PCM으로 사용, `(QRC, QR², R, C, 1)` BIBD 생성.
4. girth 상승 트릭: 각 Q-C plane에서 평행선 하나의 slope `sqc = A·y0² + B·y0`로 설계, 파라미터 `B∉[Q-2RA,...,2RA]` 조건(식 5/6)으로 triangle(girth-8)·quadrangle(girth-10) 제거.
5. 부호 파라미터: 길이 `n=QR²`, 설계 rate `1-C/R` (full rank 아니라 실제 조금 높음), girth-10은 rate≥0.4·길이>800.
6. 구조적 이점: PCM이 `Q×Q` circulant 블록(row/col weight 1)로 구성된 block-circulant → 2차원이면 cyclic, 고차원이면 quasi-cyclic.
7. 인코더: systematic block-circulant `G(x)=[p(x⁻¹) I]`를 다항식환 `P_Q`에서 유도, RC개 recursive component encoder(LFSR형)로 `Q(R(R-C)+1)` 클록에 인코딩(선형복잡도).
8. 병렬 인코더: 정보벡터·G를 m등분해 부분 곱 후 합산(식 19), m배 자원으로 m배 속도. (1125,3,540) 인코더를 m=5로 Altera Stratix FPGA에서 681.8 Mb/s 구현.
9. 결과: (521027,120237) rate≈0.77 code가 BI-AWGN 용량 0.45 dB·Rayleigh 0.51 dB 근접, BER 1e-8까지 error floor 미관측, 고girth일수록 SPA 반복수 감소(Fig 7).
10. 한계: 복호기 상세 설계 없음(참조[20][22]), 저rate·장부호·열가중 3.
