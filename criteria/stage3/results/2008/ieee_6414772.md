### B. 알고리즘 요약

1. 대상: 소수 `m` 기반 순환치환행렬로 만든 `j×k`(예제 3×6) 이진 QC-LDPC, index matrix `P`의 shift factor `a,b∈{2,...,m-1}`로 결정, rate≈0.5.
2. 문제: 기존 QC-LDPC 구성은 small stopping set·small girth·small minimum weight를 함께 고려하지 않아 랜덤 LDPC보다 BER가 나쁘거나 복호 실패.
3. 모델: BEC/AWGN에서 성능을 좌우하는 stopping set(모든 이웃 CN이 S에 2번 이상 연결)과 stopping distance `ds`(최소 nonempty stopping set 크기) 정의.
4. Theorem 1~3 + Corollary 1~3: (2,dc) QC-LDPC가 `ds=3`(6-cycle)·`ds=2`(4-cycle) stopping set을 가질 필요충분조건을 shift factor 방정식 `(b^X'-b^X)(a^Y'-a^Y)≡0 mod m` 형태로 유도.
5. 의미: 이 방정식들이 성립하지 않도록 `a,b`를 고르면 small stopping set·small girth를 구조적으로 배제.
6. check function `F_{s,2}`(4-cycle/ds=2), `F_{s,3}`(6-cycle/ds=3): 주어진 `m,a,b`에 대입해 small stopping set/cycle 유무를 판정.
7. 생성행렬 기반 최소weight 보강(Theorem 4): `wmin(g) ≤ wmin(c)`, 후보 `T={(a,b,m)}`마다 `H_i=[A B]`→systematic `G_i=[A^-1 B | I]`의 행벡터 최소weight `wmin(i)` 계산.
8. 최적화: `i=1..M` 후보 중 `wmin(i)`가 최대인 후보를 선택해 최소 code weight가 큰 부호 채택.
9. 결과: 설계부호 `a=2,b=16,p=97`의 `wmin=115`가 랜덤 LDPC `wmin=114`보다 크고, 동일 길이에서 랜덤 LDPC 대비 BER·weight 분포 우수(Fig.3).
10. 한계: HW 미설계, rate-0.5 소형·저차수(dv=2/3) 예제뿐, 시뮬 검증 수준.
