### B. 알고리즘 요약

1. 대상: BPSK/AWGN 채널의 binary QC-LDPC, BIBD `(v,k,λ)` 기반, 예제 rate `3/4`(len 964), `5/6`(len 1446).
2. 문제: 기존 BIBD-LDPC는 모든 circulant 열가중치가 같아 regular 코드만 나오고, 기존 irregular화(difference family)는 CWD를 임의 선택해 성능 보장 없음.
3. 모델: Tanner graph degree profile `γ(x)`(변수노드), `ρ(x)`(체크노드)로 irregularity 규정, all-zero codeword·Gaussian message 가정으로 Density Evolution.
4. 핵심기법: `(v,dv,1)` BIBD 각 base block에서 원소를 골라 서로 다른 열가중치 circulant 생성 → column weight distribution(CWD) `w={w1..wt}`로 irregular H 구성.
5. 핵심식: 주어진 rate `R=(t-1)/t`, row weight `ρ`에서 `S2+..+Sdv=t`, `2S2+3S3+..+dv*Sdv=ρ` (17)을 만족하는 모든 CWD 해를 열거.
6. 각 CWD의 변수노드 degree profile `γi = i*Si / Σ i*Si` (20)를 계산.
7. 각 degree profile에 대해 DE로 threshold(수렴하는 최소 Eb/No)를 구하고, threshold 최소인 CWD 채택 (TMCWD 알고리즘).
8. 최적화: `ρ`를 변화시키며 threshold 최소값이 반전(증가 시작)하는 지점의 CWD·row weight 선택. `S2<=t/2` 제약으로 최소거리 열화 방지.
9. 결과: rate 3/4는 최적 CWD `{2 2 5 5}`가 regular `{4 4 4 4}` 대비 0.25 dB(@BER 3e-4), rate 5/6은 `{2 2 2 4 5 5}`가 0.21 dB(@BER 1e-3) coding gain, Shannon 한계와 0.41 dB 격차.
10. 한계: AWGN/BPSK 시뮬만, short block, BP(tanh) 10 iter 고정, HW·error-floor·NAND 미검증.
