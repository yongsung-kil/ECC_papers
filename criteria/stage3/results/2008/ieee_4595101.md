### B. 알고리즘 요약

1. 시스템: butterfly 멀티캐스트 + linear network coding, source가 두 binary LDPC codeword `a`,`b`(동일길이)를 multicast, sink는 `a`와 `a⊕b` 수신, BPSK/AWGN.
2. 문제: network-coded 수신어(`a`, `a⊕b`)로 두 codeword를 신뢰 복원, 기존 serial 복호는 error propagation·불균등 오류보호(UEP) 유발.
3. serial 복호: `a`를 SPA로 먼저 복호 후 `(1)`식 tanh-rule로 `b`의 LLR 유도→`b` 복호. `a` 오류가 `b`로 전파(불균등 보호).
4. 핵심(joint 복호): component decoder A(a)·B(b)가 internal/external message passing으로 extrinsic 교환, 두 interactive Tanner graph 위 복호로 확장.
5. external message: `(2)(3)`식으로 stale 메시지 차감 후 extrinsic LLR 전달 → 각 variable node에 edge 1개 추가된 효과.
6. 주의: 두 부호가 동일하면 external 메시지 상관↑로 이득 없음 → `b`를 interleave(간단히 bit reverse)해 상관 제거.
7. split-codeword 전송: codeword를 반으로 분할(a1,a2,b1,b2) 후 두 time slot 대칭 network coding → 동등오류보호(EEP) 달성.
8. 분석: 대칭조건 하 all-one codeword 가정으로 단일 SPA의 DE를 joint 복호용으로 수정, scheme1/2/3 SNR threshold 계산(Table I).
9. 결과: joint(scheme2)이 serial 대비 threshold 약 수 dB 개선, QC-LDPC(RS+array masking) 예제서 waterfall gain 및 error-floor 하강(일부 BLER 10^-6까지 floor 소멸), 유한길이서 DE 예측보다 우수.
10. 한계: 이득이 network coding(두 상관 codeword) 구조에 의존, HW 미설계·시뮬만, 저~중rate QC-LDPC 예제.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| joint iterative message passing(두 interactive Tanner graph) | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 단일 Tanner graph 복호, 두 codeword joint 구조 부재로 직접 대응 불가 |
| LLR / tanh-rule soft 복호(SPA) | `decoder.cpp` `C2V_Cal()`, `channel.cpp` | Prime ECC는 min-sum 근사(양자화 LLR)라 full-tanh SPA와 상이 |
| QC-LDPC(RS-based+array masking) 부호 | `ecc_top.cpp` `Load_PCM()` | QC-LDPC 동일 계열이나 부호 자체는 재사용 대상 아님 |

적용 가치: **낮음** — binary QC-LDPC/AWGN 공통점은 있으나 핵심 이득이 네트워크코딩의 두 상관 codeword joint 복호에 종속되어 NAND 단일 codeword ECC엔 이식 불가.
