### B. 알고리즘 요약

1. 대상: binary regular LDPC (Gallager random, `(816,408)`, rate `1/2`, column weight 5), BPSK/AWGN, 저복잡도 BF 디코딩.
2. 문제: 기존 WBF/MWBF/IMWBF는 매 iteration에서 flip reliability 최대 bit 1개만 flip → 잘못 flip 시 error propagation(chain reaction)으로 iteration 폭증·성능 저하.
3. 모델: hard-decision `b`, syndrome `s = b·H^T`, IMWBF의 부분 신뢰도 `|y|_min-i,j`(자기 기여 제외한 min |채널값|)로 bit별 flip reliability `E_i` 산출.
4. 핵심기법(CVBF): flip reliability 상위 `Nc`개 bit로 culprit set 구성 → 최상위 후보의 연결 syndrome들에 majority vote(연결 syndrome 중 1의 수 `N_{k,1}` > 0의 수 `N_{k,0}`)를 적용.
5. 의미: 다수결이 1이면 그 bit를 flip 시 syndrome 벡터가 0에 더 가까워지므로(1의 개수 감소) 잘못 flip 확률↓ → error propagation 억제.
6. 확장: 다수결 실패 시 다음 후보로 이동, culprit set 내 아무도 통과 못 하면 최상위 후보를 강제 flip해 iteration 지속.
7. 조기종료: culprit bit들이 일정 iteration 내 다수결을 통과 못 하면 이후엔 계속 오flip만 발생 → culprit number 도달 시 루프 조기 종료(저복잡도).
8. 범용성: CVBF는 임의 BF 알고리즘 위에 culprit set 구성+다수결 결정만 얹는 형태로 결합 가능.
9. 결과: `Nc=n-1`에서 IMWBF 대비 약 0.9dB 개선, RRWBF 대비 약 0.2dB 우위, 동일 iteration(50)에서 MSA(min-sum)에 근접, per-iteration 복잡도는 MSA보다 훨씬 낮음(`Nc>50`에서 성능 포화).
10. 한계: 시뮬(BER)만, 실제 HW/gate count 없음(HW 오버헤드 "작다"는 정성 주장만), 성능이 결합 대상 BF(IMWBF)에 의해 상한.
