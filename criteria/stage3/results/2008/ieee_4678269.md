### B. 알고리즘 요약

1. 시스템: BPSK/AWGN, SPA 복호(max iter 50), circulant permutation matrix 기반 binary QC-LDPC.
2. 문제: computer-generated random LDPC은 encoding 복잡도가 quadratic; QC-LDPC로 linear encoding을 확보하되 girth>=6을 보장하는 대수적 구성이 필요.
3. 핵심 조건: base matrix `P`의 원소 `p_{j,l}`를 `p x p` circulant permutation matrix `I(p_{j,l})`로 치환; 4-cycle 부재 필요조건은 `p_{i,m}-p_{j,m} = p_{i,n}-p_{j,n} (mod p)` (Thm1).
4. 방법1(QR): prime `p`에 대해 원소를 `((i-1)+(j-1))^2` quadratic residue로 `P(1)` 구성 -> girth>=6 보장(Thm2). Example: `p=251`, `4x32` subarray -> (8032,7031), rate 0.875, wc=4/wr=32.
5. 식 의미: QR 제곱차 인수분해가 `(i-j)(m-n)=0`을 강제해 `i=j` 또는 `m=n`만 허용 -> 4-cycle 불가능.
6. 방법2(inverse): `GF(q=2^m)`의 primitive `α`로 `w_{i,j}=α^{i-1}α^{j-1}+(α^{i-1}α^{j-1})^{-1}`(원소+역원 합) 행렬을 만들고 exponent로 `P` -> `H(2)`, girth>=6(Thm3). Example: `q=32`, `25x25` subarray -> (775,541), rate 0.698, wc=wr=24.
7. HW 친화: circulant 구조로 linear-time 인코딩 및 min distance `>= γ+1` 하한 보장.
8. 성능1: (8032,7031)은 BER 1e-6에서 Shannon 1dB, NASA EG-LDPC(8176,7156)와 곡선 일치이나 144bit 짧고 row 18개 적어 복잡도 소폭 낮음.
9. 성능2: (775,541)은 SNR>3.6dB에서 동일rate random Mackay보다 우수, Mackay는 min distance 열세로 error floor 발생.
10. 한계: HW 미설계, AWGN 시뮬만, 성능이 기존 EG-LDPC 동급이며 masking 기반 irregular 확장은 향후 과제로만 언급.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QC-LDPC H-matrix 대수적 구성(QR/inverse) | ecc_top.cpp Load_PCM() | 새 H-matrix 생성/로드 대상이나 부호 교체 시 재검증 부담 큼 |
| circulant permutation base matrix | ecc_data.h PCM_b (base+shift) | Prime ECC도 base+shift 구조 -> 표현 호환 |
| 효율적 QC linear 인코딩 | encoder.cpp LDPC_encoder_4KB(), Generate_PCM_encoding() | Prime ECC는 이미 dual-diagonal 효율 인코딩 보유 -> 중복 |
| SPA 복호 | decoder.cpp LDPC_Decoder() | Prime ECC는 min-sum 채택, full-tanh SPA는 미채택 |
| error-floor(min distance 하한) | 대응 없음 | 부호 구조 유래, 별도 모듈 없음 |
