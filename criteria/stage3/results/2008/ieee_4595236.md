### B. 알고리즘 요약

1. 대상: binary regular LDPC(예제 `n=204, m=102, wc=3, wr=6`, MacKay)를 linear vector channel `r = Ax+b+z`(ISI/PR/MIMO 포함, `A`=간섭행렬)로 전송.
2. 문제: memory 채널의 MLD는 코드길이 지수복잡도로 불가능하고, 기존 LP decoding은 memoryless BIOS 채널 가정에 묶여 있음.
3. 아이디어: MLD의 실행가능집합을 부호 `C(H)`에서 fundamental polytope `P`(parity + box 제약)로 relax → convex 최적화 `min_{x∈P} d(Ax+b, r)`.
4. 핵심: Gaussian 채널에서 목적함수 `f(x)=||r-(Ax+b)||^2`(convex, 미분가능)를 최소화하는 relaxed MLD.
5. 제약 처리: log-barrier `b(x)=-ln(-x)`로 parity/box 제약을 merit function `ψ(t)(x)=t·f(x)+B(x)`에 흡수, `t`가 제약 softness 제어.
6. interior point 구조: 이중 루프 — inner loop는 근사 gradient descent/Newton으로 `ψ` 최소화, outer loop는 scale `t`를 점증.
7. 하이브리드: inner loop 후 내장 normalized min-sum으로 최종 판정 — 수렴가속·결정영역 확장·stacked point 구제.
8. 채널 예: dicode/1-D PR(`h0=1,h1=-1`) 및 long-tail PR(`δ=16`, BCJR 적용불가 영역).
9. 결과: PR 채널서 joint MPD(BCJR+min-sum) 대비 BER 10^-4에서 약 1.5dB 이득(Newton), Newton이 gradient보다 적은 iteration으로 수렴, long-tail PR도 복호 가능.
10. 한계: HW 미설계·시뮬만, memoryless AWGN에선 min-sum이 이미 충분, polytope parity 제약 수가 `m·2^(wr-1)`로 row weight에 지수적.
