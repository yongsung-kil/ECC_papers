### B. 알고리즘 요약

1. 대상: finite-length binary LDPC, quantized `q-bit` soft-decision iterative(min-sum) 복호, AWGN, LLR 도메인·대칭 디코더 가정.
2. 문제: error-floor 성능이 양자화 스킴에 강하게 의존 → 다양한 `q`의 FER을 Monte Carlo 없이 빠르게 추정 필요.
3. 핵심: all-zero 전송 가정, 최신뢰 입력벡터 `R=(2^{q-1}-1,…)`로부터 Euclidean distance `d` 작은 입력벡터를 열거해 복호실패집합 `E_d` 추정 (trapping set 식별 불필요).
4. FER 식: `FER=Σ_d (P(E_d)/P(S_d))·P(S_d)`, partial ordering `r≤t`와 Assumption 1-3으로 `E_d≈S_d`(below `E_{dm}`) 근사.
5. Modified cycle enumeration(Alg 1): girth부터 cycle set `T_k={C_g,…,C_{g+2k}}` 사용, distance 오름차순으로 입력벡터 생성·복호해 첫 `d_m`·`E_{d1}` 획득. 실험상 첫 3-4 cycle set이면 충분.
6. 위치 기반 분해: `E_{d1}∪E_{d2}`를 동일 location `l`로 분할해 `P(U(E_{D1,l}))`를 효율 계산(Lemma 1).
7. q 확장(Alg 2): 작은 `q1`(2,3bit)의 문제 location `L`을 재사용해 큰 `q2` 디코더 추정 → q마다 Alg 1 재실행 불필요.
8. 확률 계산(Alg 3): `P(S_d)`를 위치 1개씩 증분하며 distance list에 누적, `P(S_{d>dth})` 산출.
9. 결과: (273,82)·(1008,504) 부호, 3/4/5-bit MS에서 Monte Carlo 대비 훨씬 적은 복호 인스턴스로 정확한 FER 추정; 5-bit ≈ floating-point MS.
10. 한계: block length ≳1000이면 여전히 고비용, 통계적 가정 근사, HW 미설계.
