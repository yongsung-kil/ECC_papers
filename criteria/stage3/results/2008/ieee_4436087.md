### B. 알고리즘 요약

1. 대상: 이진 `(γ,ρ)`-regular QC-LDPC 부호, AWGN/BPSK + binary(랜덤/burst) erasure 채널, SPA 100 iter 복호.
2. 문제: 실용 길이에서 iterative 복호에 유리한(girth>=6, 큰 최소 stopping set) 구조적·효율 인코딩 가능 QC-LDPC를 대수적으로 설계.
3. 핵심 아이디어: 소수체 `GF(p)`의 원소 `k`를 location-vector `z(k)`(cyclic shift)로 표현 → base matrix `W`의 각 엔트리를 `p×p` circulant permutation matrix로 p-fold dispersion.
4. 구성 골격: Bose type-I `(12t+1, t(12t+1), 4, 4t, 1)`-BIBD, type-II `(20t+1,·,5,5t,1)`-BIBD의 base block을 확장 → circulant 배열 `H`.
5. RC-constraint(두 행/열이 1을 공유하는 위치 최대 1개) 자동 만족 → Tanner graph girth `>=6`, 최소거리 `>=γ+1` 보장.
6. subarray 선택: `H(γ,ρ)` 부분배열의 null space로 다양한 rate/length 코드 생성 (예 `H(1)(4,24)` → (8088,6743) rate 0.834).
7. burst-erasure 확장: array dispersion + masking으로 zero-covering-span `δ`를 키워 길이 `δ+1`까지 erasure-burst 정정 (예 1368/1686/2948 bit burst).
8. 인코딩: QC 구조라 shift-register로 선형복잡도 인코딩 가능(정성적, HW 미설계).
9. 결과: (8088,6743)은 AWGN에서 Shannon 1.05dB, 최소거리 추정 30, error-floor 추정 `<10^-25`; MacKay 랜덤코드와 waterfall 동등, 10↔100 iter 간 0.2dB 미만.
10. 한계: HW·gate count 없음, SPA 시뮬만, circulant 크기가 소수(337 등)이고 column weight 4~5 저차수라 NAND용 고차수/고rate 구조와 상이.
