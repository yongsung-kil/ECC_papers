### B. 알고리즘 요약

1. 일반 그래피컬 모델(factor graph)에서 실수값 함수 곱 `u(x)=∏fj(xj)`를 최대화하는 configuration을 찾는 수신 문제를 다룬다.
2. 문제: SP(sum-product) 수신기는 심볼 단위 오류율 최소화라서 최적 configuration을 보장 못 하고, 실패 원인이 pseudocodeword로만 부분 설명됨.
3. 가정: semiring `R⊆ℝ`이고 non-pendant factor node가 local behaviour `Bj`의 indicator function이면 LP 완화 적용 가능(대부분 실제 수신기 설계가 만족).
4. 핵심: pendant node는 비용함수(채널 관측 `λ`), non-pendant node는 제약집합을 담당하는 "separation effect"로 LP를 구성.
5. `xopt = argmax_{g∈KY(B)} λg^T` — 관측(λ)과 유효 configuration 볼록껍질(KY(B))의 내적 최대화로 최적화가 환원됨.
6. 저복잡도 등가 LP(§IV, auxiliary 변수 `pj,b`)와 이론용 등가 LP(§V, 제약 (18))를 정의하고 세 LP의 출력 동일성을 증명.
7. LP 출력이 정수해면 configuration, 아니면 'receiver failure'로 보고(optimum certificate property).
8. LP pseudoconfiguration(유리수 좌표 polytope 점)과 graph-cover pseudoconfiguration의 집합 등가성 증명(Thm 6.1).
9. 응용 예: BPSK+주파수선택 AWGN 채널의 joint equalization+decoding LP 수신기 설계(trellis 상태공간, 비용 `λ~i(d)=log p(ri|d)/p(ri|0)`).
10. 한계: BER/복잡도 수치 없음, LP가 SP보다 일반적으로 복잡, HW 미설계, 순수 분석 프레임워크.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LP 기반 복호/수신 알고리즘 | decoder.cpp LDPC_Decoder() | 낮음 (Prime은 min-sum MP, LP 방식과 근본 상이) |
| pseudoconfiguration/pseudocodeword 분석 | 대응 없음 | error-floor 진단 개념 참고만 가능, 코드 대응 없음 |
| joint equalization+decoding (trellis) | 대응 없음 | NAND ECC는 채널 equalization 미포함 |
```

적용 가치: **낮음** — LP 복호는 min-sum message passing과 근본적으로 다르고 복잡도가 높아 NAND HW에 이식 부적합하며, 논문은 순수 이론·분석 프레임워크로 부호설계/HW 기여가 없다.
