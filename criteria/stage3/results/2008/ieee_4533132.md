### B. 알고리즘 요약

1. 대상: 반율(R=0.5) 규칙적 binary LDPC, block length `N=1008~8064`, BPSK, AWGN/UR 채널, 최대 `I=100` 반복 복호.
2. 문제: pseudo-random PCM은 non-zero 엔트리를 전부 LUT에 열거·저장해야 해(예 24192 edges) memory-constrained 송수신기에 부적합.
3. 핵심구조: PCM을 `base matrix Hb` + `J`개 constituent matrices `Ω={Q0..QJ-1}` + `(J×J)` adjacency matrix `PJ`로 3계층 분해(Multilevel Structured).
4. 필수제약: (C1) 각 Qj girth≥6, (C2) 모든 Qj의 non-zero가 base matrix 동일위치에만 발생·상호배타 → PCM 저장 edge 수 = `Nb·γ` (base 한 레벨분만 열거).
5. 의미: J 레벨 중 한 레벨만 pseudo-random, 나머지 J-1은 그 순열 → 기술 복잡도 `1/J` 축소 (Conjecture 1).
6. 확장: MLS는 protograph 코드의 부분집합(Conjecture 2)이나 adjacency matrix(외부구조)로 protograph보다 더 compact.
7. 두 클래스: Class I = Homogeneous Coherent Configuration(circulant adjacency, (1)식), Class II = Latin square(대칭군 SΩ) — 본 논문은 Class I 성능만 보고.
8. 추가제약(HW용): (C3) 각 Qj 행·열당 non-zero 1개(J≥ρ일 때) → 병렬처리, (C4) 각 non-zero를 크기 q circulant로 치환 → 완전 QC화, 저장 `1/qJ` 축소, shift-register 인코딩.
9. 탐색: edge-colored bipartite graph의 isotopy/conjugate 동형성을 이용해 large-girth 그래프 탐색 가속(Mao-Banihashemi 개량).
10. 결과: QC MLS(8064,4032,6,28) girth 8, MacKay 대비 BER 동등하면서 저장 edge 24192→144; MLS(2016,1008,6,7)은 EG*(3,2^3) 대비 약 0.4dB 우수. HW는 미구현.
