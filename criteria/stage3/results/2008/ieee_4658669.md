### B. 알고리즘 요약

1. 대상: 이진 선형부호(irregular LDPC (64,32)/(160,80) PEG 구성, dense BCH (63,24)), AWGN 채널, rate 0.5 중심.
2. 문제: Feldman의 LP 복호는 forbidden set 부등식이 check-node degree에 지수적(`2^{δ(i)-1}`)이라 dense 부호에 적용 불가, BP는 저오율/dense에서 성능 보장 불가.
3. 모델: ML 복호를 정수계획 `min{c^T x : Hx-2z=0, x∈{0,1}^n, z≥0 integer}` (IPD)로 정식화. `c`는 LLR 비용벡터, `z`는 패리티 위반 감지용 지시변수.
4. 핵심기법: IPD의 LP 완화(RIPD)를 풀고 비정수해면 cut을 반복 추가하는 분리(separation) 접근.
5. 의미: 지시변수 `z`의 비정수성으로 위반된 패리티를 `O(mδmax)`에 즉시 식별 → cut 생성이 Taghavi-Siegel 적응형보다 빠름.
6. 확장: `x`가 정수면 Cut Gen Alg 1(Gomory cut = forbidden set 부등식, Thm 4.1), `x`가 비정수면 행연산으로 `Ĥ` 구성 후 Cut Gen Alg 2(new forbidden set 부등식, Thm 4.2).
7. 구현 디테일: `Ĥ`는 비정수 `x_j` 열을 단위행렬 열로 변환(`O(m^2 n)`), 이는 유효 RPC(redundant parity check) cut을 유도.
8. 최적화 절차: 초기해는 hard-decision, 이후 RIPD 반복 풀이+cut 추가, (x*,z*) 정수화 또는 Alg2 error로 종료.
9. 결과: LP 복호 대비 LDPC 두 부호 모두 약 0.5dB FER 개선, BP에 근접/상회; BCH에서는 적응형 LP 대비 2dB 이상 우수(단 ML과 격차 ~1dB).
10. 한계: HW 미설계, CPLEX 솔버 의존, cut 소진 시 추가 개선 불가, 소규모 부호에서만 검증.
