### B. 알고리즘 요약

1. 배경: LDPC 반복 message-passing 복호(sum-product, min-sum)의 사이클 그래프상 동작을 pseudocodeword/computation tree로 분석하는 이론 틀.
2. 문제: min-sum 복호와 LP 복호의 관계가 일관되게 정리되지 않음 - graph cover와 computation tree를 잇는 통일 모델 필요.
3. 핵심 개념: universal cover `T̃` — 무한 depth computation tree이자 무한 graph cover로, LP와 min-sum 복호를 잇는 직관적 다리.
4. Configuration/pseudocodeword: `T̃`의 변수노드에 0/1을 할당하되 각 check node 이웃 이진합이 0인 배정(universal cover pseudocodeword).
5. Rooted cost `Gv(cv)`: Wiberg cost `G(cv^(m))`를 변수노드 수로 정규화한 limsup, 유한·수렴 보장.
6. Theorem 3.2: (dX,dF)-regular(dX,dF≥3)에서 computation tree 변수노드 분포가 iteration→∞에 균일분포로 수렴(non-backtracking random walk로 증명).
7. Theorem 3.4: connected graph cover pseudocodeword로 유도된 rooted cost = `λ·ω`(LP 복호 비용, λ=LLR 벡터).
8. Prop 3.7: minimal universal cover pseudocodeword의 rooted cost=0이며 어떤 graph cover pseudocodeword로도 유도되지 않음(Wiberg deviation 대응).
9. Def 4.1: universal cover 복호 UC(y)=(θ1..θn), θi=minimal rooted cost configuration이 root에 1을 줄 확률.
10. Prop 4.3: admissible 확률측도 존재 시 graph-cover로 제한한 UC 복호가 LP 복호와 일치 - 단, 그 측도 존재·구성은 미해결(향후 과제), 시뮬/HW 없음.
