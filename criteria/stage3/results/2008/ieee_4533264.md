### B. 알고리즘 요약

1. 시스템: AWGN 채널상 rate 1/2 비재귀 조직 convolutional(NSC)-LDPC 4종(V=B=5~8, memory order m=98~2252).
2. 문제: Tanner graph의 짧은 cycle이 반복 BP 성능을 저하 → 코드 크기·최소거리와 무관하게 큰 girth를 보장하는 대수적 구성이 필요.
3. 모델: 코드를 B×V 생성행렬 `A`(연결위치 `aij`)로 정의, 패리티 `pj,t = Σ_i ui,t−aij` (식 7), 패리티검사 `zi,j,t` (식 8).
4. 핵심기법: order-1 차분 `Dil`와 order-2 차분 `Eil`로 두 차분족 `FD`,`FE`를 정의하고 Definition 1의 세 조건을 부과.
5. 의미: 조건1이 4-cycle, 조건2가 6-cycle, 조건3이 8-cycle을 제거 → girth ≥ 10 (크기·`dmin`과 독립).
6. 결과 구조: 연속 2회 반복에서 교환 메시지가 완전 독립(2회까지 BP가 exact), row weight `B+1`·col weight `V`의 정규(regular) 패리티검사행렬.
7. 복호: pipelined 반복 BP, CN 갱신 `RB(·)`를 sum-product 또는 min-sum 규칙으로, LLR 초기화 `wu = (4/N0)y`, 마지막 반복서 `λ`로 경판정.
8. 확장: feedback 복호(반복 내 soft 출력 되먹임)로 수렴 속도 개선 — 정의복호 5회 ≈ feedback 3회.
9. 성능: AWGN에서 3~5회 반복으로 최상 BER(1e-7까지 error floor 없음), V를 5→8로 늘릴 때마다 약 0.5dB coding gain.
10. 한계: rate 1/2 고정, convolutional 구조(블록 QC-LDPC 아님), AWGN 가정, HW 설계·gate count·throughput 전무.
