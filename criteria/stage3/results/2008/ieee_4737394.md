### B. 알고리즘 요약

1. 대상 부호: base matrix의 '1'/'0'을 `p×p` circulant permutation/null matrix로 치환한 PEG-E-LDPC(QC-LDPC), shift number `S_{i,j}`로 H 결정.
2. 풀려는 문제: 기존 QC-LDPC 구성법이 short cycle을 충분히 제거·분석하지 못함 - cycle 구조가 BER·수렴·error-floor를 좌우하므로 정밀 분석 필요.
3. 핵심 개념: base matrix의 cycle을 `block-element-cycle`로 정의, 실제 Tanner graph cycle과 구분(하나의 block-element-cycle 안에 최소 1개 cycle 존재).
4. Theorem 1: 길이 `2k` block-element-cycle에서 i번째 열 시작 시 마지막 '1'의 열번호 = `[i + Σ(-1)^m S_{αm,βm}] mod p`.
5. 정의 (2): `S1 = (Σ(-1)^m S_{αm,βm}) mod p` - cycle 반복 시 열 이동량.
6. Theorem 2: `(S1,p)=1`이면 block-element-cycle은 길이 `2pk`인 cycle 하나만 포함; `S1=0`이면 길이 `2k` cycle이 p개; 그 외엔 `(S1,p)`개 cycle.
7. 따름: p가 소수면 shift가 고정될 때 base matrix cycle을 최대로 연장 가능 → short cycle 회피에 유리.
8. block-element-cycle 간 관계: 공통 노드 l개면 최단 cross-cycle < `2(2k1+2k2-l)`; chain 연결이면 `2k1+2k2+2k3-2`.
9. 결과: PEG-E-LDPC의 cycle을 3부류(내부 길이=2k, 내부 길이>2k, block-element-cycle 간)로 분류 - 정량 BER/성능 수치는 없음.
10. 한계: 순수 수학 분석, 구체 구성 알고리즘·시뮬레이션·HW·성능 곡선 전무.
