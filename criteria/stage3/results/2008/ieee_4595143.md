### B. 알고리즘 요약

1. 대상은 rate `R=1/2` protograph 기반 LDPC convolutional code로, block 대응물 대비 동일 복잡도에서 이점을 가진다.
2. 문제: irregular LDPC convolutional code 앙상블의 free distance 하한이 기존에 없었고, block 대비 우위를 정량화하려 함.
3. protograph `P`(예: RJA `[[2,1,2,1],[1,3,1,1]]`)를 copy-and-permute(`N×N` 순열행렬 치환)로 앙상블화한다.
4. protograph `P`를 하삼각 `Pl`/상삼각 `Pu`로 'cutting'하여 무한 반복 → 주기적 시변 convolutional PCM `Hcc`를 구성한다.
5. unwrapping factor `λ`로 tail-biting convolutional code `Ht(bλ)`를 만들고, `λ=1`이면 원래 block code와 같다.
6. Theorem 1: 무한 convolutional code의 `dfree ≥ dmin`(임의 tail-biting `λ`)로, `δfree = dfree/νs ≥ λ·δmin`의 하한을 준다.
7. Divsalar 방식(`r(δ) = lim rn(δ)`, `δ=d/n`)으로 앙상블 평균 weight enumerator의 첫 zero-crossing `δmin`을 계산한다.
8. `λ`를 키우면 tail-biting 성장률이 무한 convolutional 하한으로 수렴한다.
9. 결과: (3,6)regular `δfree≥0.086`, RJA `δfree≥0.057`, ARJA `δfree≥0.053`으로 대응 block의 `δmin`을 크게 상회; `νs=10000` 시뮬에서 error-floor 미관측.
10. 한계: HW 미설계, AWGN/BPSK 가정의 점근 앙상블 이론, block QC-LDPC와 무관한 convolutional 구조.

| LDPC convolutional(SC) 부호 구조 | 대응 없음 | Prime ECC는 block QC-LDPC 고정 |
