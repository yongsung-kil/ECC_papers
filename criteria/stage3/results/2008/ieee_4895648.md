### B. 알고리즘 요약

1. 대상은 BEC(소거채널)이며, 좌측 전통 variable node + 우측 일반화 check node로 구성된 2-level generalized Tanner graph 위에서 부호설계·성능분석을 수행한다.
2. 기존 단일 parity-check(SPC) check node는 소거정정 능력이 제한적이고, 더 강한 component 부호를 쓰면 복호 복잡도가 급증하는 문제를 푼다.
3. 일반화 check node는 식(1)의 `(dc, dc−2)` 이진선형부호로 정의(`x0+x1+x2=n`, `dmin=2`), 3개 그룹 binary label 기반 반복 소거정정을 적용한다.
4. 핵심 기법은 각 check node에서 MAP erasure correction을 수행하는 반복복호로, Luby의 peeling decoder를 2-level 그래프로 일반화한 것이다.
5. Lemma 1: 성공적 복호 시 각 일반화 check node의 이진덧셈 복잡도는 최대 `x0+x1+x2−1`로, `x1` 값과 무관하다(→ `x1>0`의 성능이득이 추가 복호비용 없이 얻어짐).
6. Lemma 2: 부호 C의 unnormalized information function `ẽ_g`를 closed-form으로 유도하여 EXIT 함수 `I_E(p;x0,x1,x2)` 계산에 사용한다.
7. 2차원 right degree distribution `ρ(x,y)`와 수정된 DE recursion 식(5)로 앙상블을 정의한다.
8. 선형계획법으로 `λ(x)`와 `ρ(x,y)` 파라미터를 최적화하여 고정 `ε`에서 design rate를 최대화한다.
9. 결과: `ε=0.5`에서 best achievable rate 0.4943(2-level) vs 0.4936(표준 irregular), 유한길이 8192 부호는 rate 0.4768 앙상블(7)이 error-floor에서 우월.
10. 한계: BEC 전용, HW 미설계, 일반화 check node MAP 복호가 AWGN·min-sum 프레임워크와 이질적.
