> 총평: 특정 array-based LDPC 부호족의 absorbing set을 조합론적으로 완전 분류한 순수 이론 논문으로, Prime ECC의 실제 QC-LDPC H-matrix 구조와 다르고 부호설계(H-matrix 변경)는 이식 난이도가 가장 높은 영역이라 참고 가치는 제한적.

### B. 알고리즘 요약

1. array-based LDPC 코드 C_p,γ (γ≤p, p는 홀수 소수, H_p,γ는 circulant permutation matrix σ의 거듭제곱으로 구성된 γ×p 블록 행렬) — regular 코드, girth≥6.
2. 문제: 이 부호족이 저 BER 영역(error floor)에서 보이는 성능 저하 원인이 absorbing set이라는 것을 이전 하드웨어 에뮬레이션 연구에서 관찰했으나 이론적 규명이 없었음.
3. 가정: AWGN 채널, message-passing(iterative) 디코딩에서 수렴하는 non-codeword 상태를 absorbing set으로 정의.
4. 핵심 정의: `(a,b) absorbing set` = 크기 a인 bit node 부분집합 D로, unsatisfied check 집합 O(D) 크기가 b이며 D의 모든 원소가 O(D)보다 F\O(D)에 더 많은 이웃을 가짐; 모든 D 외부 노드도 이 조건을 만족하면 `fully absorbing set`.
5. 핵심 정리(Theorem 1): γ=2에서 최소 (4,0) absorbing set(=코드워드), γ=3에서 최소 (3,3) absorbing set 및 (4,2) fully absorbing set, γ=4(p>19)에서 최소 (6,4) (fully) absorbing set — vertex/edge consistency 조건으로 존재/비존재를 modular 방정식으로 증명.
6. 확장(Theorem 2): 최소 absorbing set의 개수가 블록길이 n=p²에 대해 γ=2는 O(n²), γ=3,4(p 조건 만족)는 O(n^1.5)로 스케일링.
7. 구현 디테일 없음(순수 조합론 증명 논문); 다만 실험 검증에 4.5(4정수5소수) uniform quantization, 최대 200 iteration 하드웨어 에뮬레이터 사용.
8. 최적화 절차 없음(수학적 증명 위주, DE/GA/학습 없음).
9. 결과: C_47,4 코드에서 SNR=6.4dB, 25개 오류 샘플 중 18개(최소 weight)가 모두 이론에서 예측한 (6,4) fully absorbing set과 일치 — 이론과 실험 완전 부합.
10. 한계: HW 설계 없음(순수 이론), 특정 array-code 구조(γ≤4)에만 적용, 일반 QC-LDPC나 protograph 코드로의 일반화는 future work로 남김.
