## B. 알고리즘 요약

1. 대상은 protograph 기반 QC-GLDPC 부호로, base matrix `B(m×n)`를 lifting factor `z=pq`로 리프팅해 Tanner 그래프를 얻는다 (rate `R=0.4762`~, length `N=210~960`).
2. 문제: GLDPC에서 GC 노드에 연결된 doped VN은 강하게 보호되지만, SPC만 연결된 undoped VN은 채널 오류에 취약한데 기존 PEG는 이를 구분하지 않는다.
3. 관찰: 같은 protograph라도 undoped VN 주변 local cycle이 적을수록 복호 성능이 좋다 (BEC 최대 20배, AWGN 3.8배 FER 차이).
4. 핵심기법: CC-PEG는 리프팅 시 (i) undoped girth `g_u`를 최대화하는 edge를 먼저 고르고, (ii) 그중 concentrated CM `M_cc = Σ_{Vu}M_vs + αΣ_{Vd}M_vs` 를 최소화하는 shift 값을 선택한다.
5. `M_cc`의 의미: `α<1`로 undoped VN cycle을 doped VN cycle보다 `α^-1`배 더 해롭게 취급 → cycle을 doped 쪽으로 의도적 집중.
6. cycle 개수 계산은 PMP 알고리즘(`O(lE^2)`)으로 각 edge가 포함된 최단 cycle 길이·cycle 수를 산출한다.
7. VN 리프팅 순서 정책: degree 오름차순 정렬 후 같은 degree 내에서 doped VN을 뒤로 배치.
8. 파라미터 `α`는 0.05~0.2가 최적, 실험에서 `α=0.1` 고정 (`β=0.01`).
9. 결과: PEG/QC-PMP 대비 protograph P1~P5에서 FER 1~20배 개선, AWGN 최대 0.4dB 이득, edge 많은 큰 protograph일수록 이득 증가.
10. 한계: 순수 code-construction 논문으로 HW 설계·NAND 실험 전무, GLDPC(component code) 전제라 SPC-only binary QC-LDPC와 구조가 다름.
