### B. 알고리즘 요약 (10줄 내외)

1. AWGN/BPSK 채널에서 terminated convolutional code(CC)를 block-oriented accumulator(BA)로 결합해 spatially coupled convolutional code(BA-SC-CC)를 구성, rate `R = (1/(1+T/L))·(K/N)`.
2. 기존 SC-LDPC/BMST/SC-RCC 등은 좋은 threshold를 갖지만 free distance 하한 증명이 부족했고, MAP 최적 복호가 비현실적인 LDPC/BCH 대비 CC는 trellis 기반 BCJR로 효율적 MAP 복호가 가능하다는 점에 착안.
3. 채널모델: BPSK+AWGN, LLR 기반 Gaussian 근사 EXIT chart 분석(`I_ch = J(σ_ch)`).
4. 핵심 기법 1단: normal graph 상 sliding window decoder(윈도우 W=d+1)로 `= → + → CC → + → =` 스케줄에 따라 반복 복호, CC 노드는 BCJR/Max-Log-MAP으로 디코딩.
5. 핵심 정리: doubly dispersed interleaver 조건 하에서 `d_min ≥ 3·d_free - 2`(정리 3)로 BA-SC-CC의 free distance가 원 CC보다 최소 3배 가까이 커짐을 증명 - spatial coupling으로 코드 강도 자체가 개선됨을 보장.
6. 확장: self-recovery threshold(SRT) 개념 도입 - 특정 레이어 디코딩 실패 시 이후 레이어가 EXIT chart 기반으로 복구 가능한 최소 SNR을 정의, error propagation을 정량 분석.
7. 부수 트릭(HW 친화): threshold-based Max-Log-MAP(T-Max-Log-MAP, 식 18)과 threshold-based min 연산(TAMSA 차용, 식 19-20)으로 |L1|,|L2|가 [τ0,τ1] 구간일 때만 스케일링 적용, LUT 기반 non-uniform quantization(NUQ) 연산자 `F(·)`로 3~5bit 양자화 구현(표 IV~VI).
8. 별도 학습절차 없음(수치 최적화 대신 EXIT chart 기반 threshold 설계와 MMI 기반 계층적 NUQ 구성).
9. 결과: decoding latency 6,000bit에서 (3,6)-regular SC-LDPC 대비 0.45dB, 4bit NUQ 기준 4bit uniform quant 대비 0.15dB 성능 개선; BER 1e-5에서 평균 반복수 2.01회(비교 SBC 코드는 4.50회)로 iteration 감소.
10. 한계: 순수 시뮬레이션(플로팅포인트/양자화 모두)이며 HW(FPGA/ASIC) 미설계, decoding complexity가 BCJR 기반이라 SC-LDPC/BMST보다 본질적으로 높다고 저자 스스로 명시(Remark 5).

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Threshold-based min 연산(TAMSA, 식 19-20) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | check node min-sum 변형으로 CN 연산에 threshold 스케일링 추가 이식 가능성 있음 |
| Non-uniform quantization LUT(표 IV~VI) | `decoder.cpp` `Get_VNU_Table_Idx()`, `ecc_data.h` `PARAM_LLR` | LLR 테이블 기반 양자화 구조와 유사, 3~5bit 저해상도 NUQ 설계 아이디어 참고 가능 |
| BA-SC-CC 부호설계(convolutional+accumulator 결합) | `ecc_top.cpp` `Load_PCM()` | convolutional 코드 기반 구조라 QC-LDPC PCM 로드 구조와 직접 대응 어려움("대응 없음"에 가까움) |
| Sliding window decoder(normal graph 기반) | 대응 없음 | Prime ECC는 block 코드 단일 프레임 반복이라 spatially coupled 윈도우 슬라이딩 스케줄과 구조 상이 |
```

적용 가치: 중간 — 부호설계(BA-SC-CC) 자체는 convolutional code 기반으로 QC-LDPC와 이질적이라 이식 어려움이 크지만, threshold 기반 min-sum 변형(TAMSA)과 저비트 non-uniform quantization LUT 설계는 Prime ECC의 min-sum CN 연산·LLR 양자화 테이블에 부분적으로 참고 가치 있음.
