### B. 알고리즘 요약

1. 대상은 PD-NOMA(전력영역 비직교 다중접속) 다운링크 2-user 시스템, Rayleigh 페이딩 채널 + AWGN, M-PSK 변조(QPSK/8-PSK), 코드rate 1/2, 코드워드 길이 1008.
2. 문제의식은 기존 NOMA에서 두 사용자 변조신호를 독립 생성 후 단순 중첩해 사용자간 구조적 상관을 활용하지 못하고, 비피드백 다중접속용 코드의 반복복호(ID) extrinsic 정보 이득이 제한적이라는 점.
3. 채널모델: `y = h*x_sc + n`, `x_sc = sqrt(P1)*s1 + sqrt(P2)*s2` (전력할당계수 α로 P1,P2 결정), BICM-ID(Bit-Interleaved Coded Modulation with Iterative Decoding) 프레임워크.
4. 핵심 기법 1단: symmetric-Hamming(SH) 성상매핑 - 두 사용자 M-PSK 심볼 순열 `π_i(j)`를 반전연산자 `R(·)`과 순환시프트 `C_λ(·)`로 구성해 인접 레이블간 Hamming distance를 최소화(Algorithm 1).
5. 핵심 식: modified HMMSED(조화평균 최소유클리드거리) `D_h1`, `D_h2` (식 (5)-(6))로 성상매핑 성능을 정량화, 값이 클수록 초기 soft 정보 신뢰도 향상.
6. 핵심 기법 2단: PEXIT(protograph EXIT) 분석 기반 mutual-information-guided threshold 추정으로 TU-1(non-punctured, 3x6 base matrix)/TU-2(punctured, 3x5 base matrix) protograph LDPC 설계.
7. 부수 트릭: 단일사용자에서는 degree-2 VN 비율을 50% 미만으로 제한하는 관례를 깨고, 다중사용자 교차반복 이득을 활용해 degree-2 VN 비율을 50%까지 상향(waterfall 성능 우선, error-floor는 outer iteration이 완화).
8. 최적화 절차: PEXIT 알고리즘으로 base matrix 원소(δ_p,r ∈{0,1,2,3}) 탐색공간을 제한하며 decoding threshold 최소화.
9. 결과: TU-2가 기존 E-PLDPC 대비 threshold 0.19dB(UE-1)/0.26dB(UE-2) 개선, BER 1e-5에서 TU-1/TU-2가 E-PLDPC 대비 약 0.3~0.4dB 이득. SH 매핑은 기존 매핑 대비 0.5~2.8dB 이득.
10. 한계: Rayleigh 페이딩+2-user NOMA 시나리오 전용 시뮬레이션 결과뿐(HW 미설계), 단일사용자/유선 NAND 채널로의 일반화는 검증되지 않음.
