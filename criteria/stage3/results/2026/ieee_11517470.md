### B. 알고리즘 요약

1. 위성간 통신(AWGN, BPSK)용 rate-compatible partially-coupled staircase LDPC(PS-LDPC) 코드를 제안, 코드율은 결합깊이 `m` 조절로 `[2Rc-1, Rc)` 범위에서 가변.
2. 기존 staircase/SC-LDPC 코드는 짧은 컴포넌트 코드에서 error-floor가 크고 결합패턴 최적화 이론이 부족함이 문제.
3. 채널은 AWGN+BPSK, 컴포넌트 코드는 IEEE 802.16e/802.22, CCSDS 표준의 QC-LDPC(N=256~1056) 사용.
4. 핵심 기법 1단: sliding window decoding(SWD) - inner iteration(BP, `I`회)과 outer iteration(블록간 메시지 교환, `J`회)의 이중 구조, LLR 입력 `L_in = L_c + ε·L_a` (`ε`는 오류전파 억제 스케일링).
5. MET(multi-edge type)-LDPC 프레임워크로 3종 edge-type(결합비트/비결합비트/미래결합비트)의 node-perspective 다항식 정의, Gaussian approximation DE로 평균/분산 추적해 복호 threshold `σ_th` 도출.
6. 핵심 기법 2단: OCP(optimized coupling pattern) 인코딩 - 2단계 column permutation(`Π1`, `Π2`)으로 VN에 연결된 edge-type을 조정해 threshold를 최소화.
7. 부수 트릭: C-SWD(cascaded SWD) 알고리즘 - 초기블록 pre-decoding, 이중 syndrome(S1,S2) 기반 신뢰도 강화(계수 `κ`), error-floor 발생 시 BP 또는 OLD(ordered likelihood decoder)로 cascaded 재복호.
8. 최적화 절차: DE 기반 exhaustive-search 대신 lifting factor `Z` 단위 permutation 탐색으로 준최적 결합패턴 도출.
9. 결과: 동일 복잡도(~2×10^5 ops/info-bit)에서 기존 PIC-LDPC/SysConv-LDGM/PSC-LDPC 대비 최저 BER·복잡도 달성, OCP로 약 0.15dB 개선.
10. 한계: HW 미설계(순수 알고리즘/시뮬), spatially-coupled 다중블록 구조 전제(단일 QC-LDPC와 다름), 큰 N에서 waterfall 이득 감소.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SWD의 inner iteration(BP) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | BP 기반이나 Min-Sum이 아닌 full Sum-Product LLR 갱신이라 직접 대응 약함 |
| 결합패턴 DE/OCP 최적화 | `ecc_top.cpp` `Load_PCM()` | spatially-coupled 다중블록 구조는 Prime ECC의 단일 QC-LDPC PCM과 근본적으로 다름 - 대응 없음에 가까움 |
| C-SWD의 cascaded OLD/BP (error-floor 대책) | `decoder.cpp` `LDPC_Decoder()` | error-floor 개선 아이디어는 관련 있으나 OLD는 MRB 재부호화 기반으로 min-sum HW 구조와 이질적 |
| 조기종료(syndrome 기반) | `partial_CRC.cpp` `partial_crc_HW_T4()`, `full_CRC.cpp` | 문맥 유사(syndrome=0 종료)하나 논문은 CRC가 아닌 parity-check 직접 사용 - 대응 없음 |

> 적용 가치: 낮음 - 위성 채널 전용 spatially-coupled staircase 부호설계 논문으로 NAND 단일 QC-LDPC 구조에 이식할 요소가 제한적임.
