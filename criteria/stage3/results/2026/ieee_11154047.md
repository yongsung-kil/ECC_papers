> 총평: 추가 read voltage 1개로 얻는 저비용 강/약 비트 구분을 BF 계열 디코더의 첫 iteration에만 선택적으로 적용해 hard-decision 수준의 복잡도를 유지하면서 RBER 정정한계를 7×10⁻³→1×10⁻²로 끌어올린 실용적 하이브리드 기법으로 NAND 이식 가치가 높음.

### B. 알고리즘 요약

1. 대상: 3-D (M/T)LC NAND flash, LDPC code length `K=32768`, rate `0.89`, column weight `dv=5`, 채널모델은 Luo et al.[29]의 3-D NAND Vth drift 모델.
2. 문제: 기존 DTBF(dynamic threshold bit-flipping)는 hard information만 사용해 RBER가 증가하면(P/E·retention 누적) 정정 한계를 넘어서고 iteration/decoding latency가 급증.
3. 가정: 인접 상태(state) 분포가 겹치는 영역(overlap region)의 셀일수록 read voltage에 가까워 에러 확률이 높다는 threshold voltage distribution drift 특성을 이용.
4. 핵심 기법: 기존 read voltage 양옆에 추가 reference voltage(bound 파라미터로 위치 결정)를 두어, 그 사이 값을 읽은 비트는 `weak`(1비트 플래그 0), 그 밖은 `strong`(플래그 1)으로 표시하는 1비트 소프트 정보를 생성.
5. 핵심 식: 신뢰도 지표 `En = α(zn⊕yn) + Σ_{m∈A(n)}(2sm-1)` (기존 DTBF 식과 동일)에 대해, 첫 iteration에서만 weak 비트에 별도 임계값 `En*`를 적용해 `En > En*`이면 flip — strong 비트는 기존 `Th`로 비교.
6. 확장: `En* ∈ {-dv-α+k}`로 탐색해 P/E·retention 조건별 최적값을 사전 캘리브레이션 후 디코더에 저장(모든 조건 저장 대신 핵심 retention 지점만 저장해 복잡도 절충).
7. 구현 디테일: 2번째 iteration부터는 순수 DTBF로 복귀(추가 소프트 판정은 1회성) — 계산 비용 증가를 최소화하면서 초기 오류 밀집영역만 선별 보정.
8. 파라미터 탐색: bound(5/10/15/20/50)와 En*(-5~5) 각각 시뮬레이션 스윕으로 최적값 도출, 실칩(3-D TLC NAND, floating-gate) 측정 기반 RBER·retention 데이터로 채널 특성 보정.
9. 결과: RBER=6.8×10⁻³에서 FER이 DTBF `6×10⁻¹` → SiDTBF `4×10⁻⁵`로 개선, decoding time `0.017s`→`0.0053s`로 단축, 정정가능 RBER 상한이 `7×10⁻³`→`1×10⁻²`로 확장.
10. 한계: 알고리즘 검증은 컴퓨터 시뮬레이션 수준(HW 미설계, gatecount/throughput 없음), bound·En* 최적값이 채널/칩 특성에 의존해 사전 튜닝 필요.

### C. Prime ECC 관련 모듈 핀포인트

| 1비트 strong/weak 소프트 정보(추가 read voltage) | `channel.cpp` `Set_R_Offset()`, `decoder.h` `Get_VNU_*`(HD 확장) | 1비트 추가만으로 hard-decision 대비 정정 성능을 크게 높이는 저비용 soft-info로, Prime ECC의 HD/2SD 경계 영역에 이식 시도 가능 |
| iteration별 En*(임계값) 캘리브레이션 테이블 | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | 기존 적응형 LLR 테이블과 유사한 구조(iteration·조건별 임계값 사전계산)로 대응 용이 |
| 조기 iteration 후 표준 DTBF 복귀 | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | Partial CRC 조기종료와는 별개로, "1회성 강화 보정 후 표준 루프 복귀" 패턴 자체는 이식 검토 가치 있음 |

적용 가치: **높음** — NAND 3D flash 채널 특성을 직접 이용한 1비트 소프트 정보로 hard-decision 계열 디코더의 정정능력·iteration·latency를 동시에 개선하는 저비용 기법이며, Prime ECC가 이미 지원하는 HD/soft 경계 판정 로직에 결합 검토가 용이함.
