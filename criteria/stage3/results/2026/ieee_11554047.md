### B. 알고리즘 요약

1. 시스템: J개 UE, K개 RE의 SCMA 업링크, Rayleigh fading 채널, 각 UE는 protograph LDPC로 부호화 후 SCMA 변조.
2. 문제: 기존 코드북은 UE-RE 매핑이 균일해 간섭이 크고, 기존 repetition은 단순 중복이라 이득이 작으며, 기존 P-LDPC 설계는 turbo/non-turbo 중 한 쪽만 최적화됨.
3. 채널모델: `y = Σ diag(h_j) x_j + n`, `n ~ CN(0,N0)` AWGN, Rayleigh 페이딩 계수 `h_j`.
4. 기법1(코드북): LDFG로 비균일 factor graph `F`(cycle 최소화 제약, `dv≥2`) 구성 후, ECS로 RE별 에너지 벡터 `E_k`를 할당해 사용자 신호 중첩을 구분.
5. 기법2(전송): SWIT은 원본/1차 중복 블록을 interleaved block-delay mapping으로 결합하고, 나머지 중복블록은 이전 슬롯과 XOR하는 Markovian superposition(`S_(t,r)=C_(t,r)⊕Ĉ_(t-1,r)`)으로 시간축 정보이득을 얻음.
6. 기법3(코드): decoding-threshold-guided multi-objective optimization(DT-MOO)으로 non-turbo/turbo 임계값 `τ1,τ2`를 동시에 최소화하는 Pareto 최적 protograph base matrix(EMUC) 탐색.
7. 최적화 절차: multi-objective differential evolution(MODE)로 base matrix 원소(0~2) population을 mutation/crossover/selection 반복(`Np=10, Gmax=100`), EXIT 기반 decoding threshold를 fitness로 사용.
8. 결과: EMUC-1(rate 1/2)이 non-turbo 3.53dB, turbo 2.77dB로 기존 대비 최소 임계값 달성; 비균일 CB가 BER 5e-4에서 기존 대비 0.8~1.1dB 개선; SWIT이 repetition 대비 최대 1.3dB 이득.
9. 한계: HW 미설계, 순수 시뮬레이션(BER/FER, decoding threshold)만 존재, Rayleigh fading + SCMA 다중사용자 특화 가정으로 NAND 단일채널 환경과 근본적으로 다름.
