### B. 알고리즘 요약

1. 대상 시스템은 J명의 UE가 K개 RE를 공유하는 SCMA(overloading factor `λ=J/K`) 업링크, Rayleigh fading + AWGN 채널이다.
2. 기존 PLDPC 코드는 단일 사용자용으로 설계되어 SCMA 멀티유저 환경에서 성능이 떨어지고, 독립적 디코딩 구조는 인터유저 간섭을 효과적으로 제거하지 못하는 문제를 다룬다.
3. 채널모델은 `y = Σ diag(h_j)x_j + n` (Rayleigh 채널이득 `h_j`, AWGN `n~CN(0,N0)`)이며, SCMA sparse codebook 매핑을 가정한다.
4. 1단계로 PEXIT 알고리즘 기반 4×8(비펑처) / 4×7(펑처) 초기 base matrix에서 degree-1/2/3/최고차수 VN 배치 제약을 두고 computer search로 rate-1/2 Code-U, Code-P를 설계한다.
5. base matrix `B=(b_{i,j})`의 원소는 CN-VN 간 엣지 수이며, degree-2 VN 과다는 error floor를 유발하므로 `nc-1` 이하로 제한하는 조건이 핵심 설계식이다.
6. 2단계로 전력불균형 다단복호(PI-MLD)를 제안: UE를 L개 레벨로 분류(level classification) 후 power vector `p=[p1,...,pL]` (`p1>p2>...>pL`)를 모디파이드 PEXIT 기반 Algorithm 1로 탐색해 국소 최적화한다.
7. 3단계 power-oriented MLD는 고전력 레벨부터 저전력 레벨 순으로 SCMA 검출(log-MPA, `N_S=5`)과 BP 복호(`N_L=50`)를 순차 반복하며 외인성 LLR을 되먹임한다.
8. PIA 탐색은 스텝사이즈 `δ=0.025`로 레벨쌍 간 전력을 점진 이동시키며 AMI(평균상호정보량) 수렴을 관찰하는 방식(학습이라기보다 그리디 탐색)이다.
9. 결과: Code-U는 (3,6)/RJA 대비 BER 1e-5에서 약 0.3dB, PI-MLD-SCMA는 CB-1 기준 약 1.6~1.8dB 성능 이득을 보였고 Code-P의 복호복잡도 `O(N_L(6.0n))`는 NR코드 `O(N_L(7.0n))`보다 낮다.
10. HW 구현/합성 결과는 없고 전부 이론(PEXIT) 및 Monte-Carlo 시뮬레이션에 국한되며, Rayleigh fading·SCMA 특화 가정이라 일반 binary LDPC 채널로의 일반화는 제시되지 않는다.
