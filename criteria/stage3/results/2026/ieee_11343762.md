### B. 알고리즘 요약

1. 이 논문은 6G 무선통신 물리계층 전반(신호검출, 채널추정, 프리코더/빔포밍, ISAC, ECC 디코딩, 전력할당, 물리계층보안)에 deep unfolding(algorithm unrolling)을 적용한 기존 연구들을 정리한 서베이/튜토리얼 논문이다.
2. 문제의식: 순수 데이터 기반 DNN은 대용량 데이터셋 필요·해석 불가능인 반면, 전통 반복 알고리즘은 구조는 있으나 파라미터 튜닝이 수작업임; deep unfolding은 반복 알고리즘의 각 iteration을 DNN layer로 매핑해 두 장점을 결합.
3. 일반 프레임워크: `s_{l+1} = h(s_l; θ_l)`로 L개 layer를 구성하고, 학습가능 파라미터 `{θ_l}`을 end-to-end backpropagation으로 최적화(Section II, ADMM 기반 MIMO 검출을 예시로 튜토리얼 제공).
4. Section VII("Decoding in Error-Correcting Codes")이 LDPC/turbo/polar 디코더 관련 유일한 절이며, 자체 알고리즘 제안 없이 기존 문헌을 리뷰: 예) [43] LADN(unfolded ADMM decoder, binary linear code), [44] irregular LDPC용 unfolded ADMM, [223] 1-bit quantization polar BP unfolding, [224] penalty dual decomposition 기반 double-loop LDPC unfolding, [225] normalized min-sum LDPC decoder의 deep unfolded 버전.
5. [225]는 CN-VN 반복 복호 과정을 unfolding하여 classical normalized min-sum보다 BER 개선 및 code length/rate/channel 조건에 강건함을 보고(원논문 인용, 본 서베이는 직접 실험 없음).
6. [224]는 primal-dual 변수를 교대 갱신하는 double-loop 구조이며 ADMM 기반 디코더보다 계산복잡도가 낮다고 보고.
7. Table VI에서 위 unfolded 디코더들을 계산복잡도·학습파라미터 수·수렴속도(layer 수) 기준으로 비교(표 자체는 본문 텍스트로 추출되지 않음, 수치 확인 불가).
8. 한계(Section X, C1~C5)로 서베이가 직접 지적: 코드 구조/길이 변경 시 재학습 필요, 고정 layer 수로 인한 수렴 보장 부재, 복잡한 제약 반영 어려움, HW 병렬성·면적효율과의 부정합, 실채널 조건(이동성, HW 결함)에서 성능 저하 우려.
9. 결과 수치: 본 논문 자체 실험 결과 없음 - 모든 성능 수치(BER/복잡도 개선 등)는 인용된 원논문의 것.
10. 한계: 이 논문은 서베이이므로 검증수준·복잡도·throughput 등 모든 정량 카테고리가 미상/N/A이며, NAND용 binary QC-LDPC와 직접 관련된 부호설계/HW 내용이 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Deep unfolding 기반 LDPC decoder(NN layer화) 전반 | 대응 없음 | Prime ECC는 min-sum 기반 고정 알고리즘 시뮬레이터로 NN/학습 계층 없음 |
| Section VII 인용 [225] normalized min-sum NN LDPC decoder | `decoder.cpp` `CNU_Update_New_Mag()` | 개념적으로 min-sum normalize factor와 관련되나 본 서베이는 원논문 요약뿐, 직접 이식 근거 없음 |
| ADMM 기반 unfolded decoder([43][44][224]) | 대응 없음 | Prime ECC는 message-passing(min-sum) 구조이며 ADMM/LP 디코딩 미보유 |

> 적용 가치: 낮음 | 서베이 논문 자체는 신규 기법이 없고 LDPC 관련 부분도 타 논문 요약뿐이므로 이식 판단은 인용된 원논문([225] 등) 개별 확인이 필요.
