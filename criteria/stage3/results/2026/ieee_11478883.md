### B. 알고리즘 요약

1. 시스템: CSS 기반 QLDPC 코드(hypergraph product/HGP, lifted product/LP 코드), quantum erasure channel(큐비트가 확률적으로 소거됨), 디코더 입력은 수신벡터가 아닌 syndrome 벡터.
2. 문제: syndrome 추출 회로 자체가 오류를 일으켜(analog voltage/current를 이산화) 디코더가 노이즈 있는(noisy) soft syndrome을 받는 상황에서 소거(erasure) 오류를 복구해야 함.
3. 핵심 가정/모델: syndrome 노이즈는 Gaussian 모델(`r_i = s̄_i + n_i`, `n_i~N(0,σ²)`)을 따르며, syndrome LLR `γ_i = 2r_i/σ²`로 각 check node(신드롬 비트)에 신뢰도를 부여.
4. 핵심 기법 1단: 원래 Tanner graph에 check node마다 "virtual variable node"를 추가한 modified Tanner graph를 구성해 신드롬 신뢰도를 variable node LLR처럼 취급, normalized min-sum(n-MSA, `α=0.61`)으로 메시지패싱.
5. 핵심 식/의미: 신뢰도 `|γ_{v_c}| > Γ`인 check node 집합 `C(a)`만 활성화해 T iteration 메시지패싱을 수행하고, 나머지(`C(b)`)는 최종 메시지로 갱신 후 재분류 — "노이즈 있는 신드롬 일부를 무시하는 것이 잘못된 신드롬을 그대로 쓰는 것보다 낫다"는 경험적 관찰에 기반.
6. 핵심 기법 2단: 최대 `T'` 라운드 동안 (n-MSA on 서브그래프) → (신뢰도 재계산·`C(a)`/`C(b)` 재분류)를 반복, halting 조건은 `HÊ^T = ŝ^T` 매칭.
7. 부수 디테일: 소거 variable node LLR은 수치안정성을 위해 0 대신 `llr_min=10⁻⁵`, 비소거는 `+∞` 대신 `llr_max=25`로 설정, 메시지값은 `[-25,25]`로 클리핑; `T'` 도달 시 임계값 `Γ'=10` 초과 노드만 매칭 확인하는 추가 halting 체크 사용.
8. 학습/최적화 절차: 없음(파라미터 `α, Γ, Γ', T, T'`는 시뮬레이션으로 수동 튜닝).
9. 결과: [[882,24]] LP code에서 σ=0.4일 때 noiseless syndrome과 거의 동일한 failure rate 달성(제안 알고리즘 vs 기존 noisy-soft n-MSA 대비 큰 개선, 예: σ=0.3에서 기존 기법 failure rate 1.7×10⁻¹ vs 제안 기법은 noiseless 수준 1.38×10⁻² 근접); 복잡도는 `O(nT' log n)`.
10. 한계: HW 미설계, 이론/시뮬레이션(HGP/LP 특정 코드 2종)에 한정, degeneracy 처리는 future work로 미해결, 완전히 양자 도메인(non-binary stabilizer)이라 classical binary LDPC와 부호/채널 모델이 근본적으로 다름.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QLDPC syndrome 기반 n-MSA 디코딩 알고리즘 전체 | 대응 없음 | non-binary quantum stabilizer 코드 디코딩으로 Prime ECC의 binary QC-LDPC 디코더와 입력 형식(신드롬 vs 수신 LLR)부터 상이 |
| 신뢰도 임계값 기반 check node 부분집합 스케줄링 | 대응 없음 | Dual-Update(parity even/odd 교대)와 개념적으로만 유사할 뿐 신드롬 노이즈 대응 목적의 서브그래프 분리는 대응 모듈 없음 |
| Gaussian 노이즈 syndrome LLR 모델 | 대응 없음 | Prime ECC의 채널 모델(AWGN/RBER/fixed-error, `channel.cpp`)은 데이터 채널용이며 syndrome 채널 노이즈 개념 자체가 없음 |
| Virtual variable node를 통한 modified Tanner graph 구성 | 대응 없음 | binary QC-LDPC PCM 구조(`ecc_top.cpp` `Load_PCM()`)와 무관한 quantum 특유의 그래프 변형 기법 |

적용 가치: `낮음` — non-binary quantum stabilizer 코드 및 syndrome 노이즈 모델을 다루는 논문으로, Prime ECC 3.1(binary QC-LDPC, classical channel decoding)과 판정 기준(§5) 상 "non-binary·양자" 범주에 해당해 이식 가치가 낮음.
