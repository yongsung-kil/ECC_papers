### B. 알고리즘 요약

1. 시스템: non-ergodic block-fading(BF) 채널, F개의 독립 페이딩 이득으로 프레임이 분할, BPSK 변조, anytime 코딩 프레임워크(고정 길이 프레임 `N`, 누적 정보 인코딩).
2. 문제: 기존 anytime 코드(SC-LDPC, LDPC convolutional)는 BEC/AWGN 채널에서만 검증됐고, BF 채널에서는 anytime reliability와 full diversity를 동시에 만족하는 코드가 없음.
3. 핵심 가정: 수신 신호 `y_{p,j} = α_f x_{p,j} + z_{p,j}`, `α_f`는 i.i.d. Rayleigh 페이딩, `z_{p,j}`는 AWGN.
4. 핵심 기법 1: root-protograph(F개 root-check 타입 보유)에 exponential spatial coupling(연결확률 `Pr(h) ∝ e^{-hλ}`, λ=0.5)을 적용한 RP-anytime 앙상블, convolutional base matrix `B_[0,t-1]`.
5. 핵심 식의 의미: root-check 구조가 각 페이딩 블록별 정보 VN을 보호하여 full diversity(`d_v = F`)를 보장하고, spatial coupling이 시간에 따른 메모리 누적으로 anytime reliability를 제공.
6. 핵심 기법 2: frame-indexed 수정 density evolution(delay `d` 종속)으로 average asymptotic BER(`Ā_BER`)과 anytime SNR outage region(`D̄_SNR`) 두 성능지표 정의.
7. 부수 기법: anytime SNR/fading outage boundary 산출 알고리즘(Algorithm 1)으로 diversity order 예측.
8. 최적화 절차: 두 지표(`Ā_BER`, `D̄_SNR`)를 동시 최소화하는 multi-objective differential evolution(MODE, Algorithm 2)으로 root-protograph base matrix 설계, Pareto dominance 기반 선택.
9. 결과: 설계된 코드가 기존 root-protograph LDPC/anytime 코드 대비 outage limit에 근접, full diversity 달성(R=1/2에서 dv=2, R=1/3에서 dv=3), MODE는 random search 대비 약 7.38배 빠른 수렴(6618초 vs 48830초).
10. 한계: 순수 이론·유한장 시뮬레이션(비트 수준)뿐이며 HW 미설계, BF 채널(무선 페이딩) 특화로 NAND의 AWGN/RBER 채널 모델과 무관.
