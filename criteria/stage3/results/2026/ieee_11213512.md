### B. 알고리즘 요약

1. 시스템: cyclic LDPC(단일 circulant PCM) 및 QC-LDPC(base matrix + Z×Z CPM/ZM) 코드, 5G LDPC(BG2, Z=16/30) 포함 다양한 코드에 적용.
2. 문제: 기존 NBP(neural belief propagation)류 신경망 디코더는 block length가 커질수록 Tanner graph의 edge 수(`|E|=nγ`)에 비례해 파라미터·GPU 메모리가 급증해 배포가 어려움(예: (1057,813) 코드에서 NBP 23.5GB 필요).
3. 핵심 가정: cyclic PCM의 각 행이 이전 행의 순환 shift이고(cyclic), QC-LDPC의 row-block 내 행들도 section-wise 순환 shift 관계(QC)라는 구조적 대칭성.
4. 핵심 기법: "Revolving NBP" — 전체 PCM `Hcyc` 대신 첫 `l`개 행으로 이루어진 서브행렬 `H0`(또는 QC의 `H*`)만으로 작은 신경망(NBP/NNMS/NOMS)을 구성하고, 한 sub-iteration마다 출력 reliability vector를 `l`(또는 QC는 1) 위치만큼 순환 shift해 다음 sub-iteration 입력으로 재사용 (Algorithm 1/2).
5. 핵심 식: CN 메시지 갱신은 학습 가능한 스케일 `w_c→v`(NNMS, 식 7) 또는 스케일+오프셋(NOMS, 식 8)을 min-sum 근사에 곱해 성능·복잡도 균형을 맞춤; 파라미터 수가 `|E0|=lγ`로 원본 `|E|=nγ`의 `l/n`로 축소.
6. 확장 기법 2: QC-LDPC용은 CN 메시지 갱신 전 이전 sub-iteration의 CN 메시지를 reliability vector에서 빼는 skip-connection형 파이프라인을 추가해 성능 저하를 보완 (§IV-B).
7. 부수 트릭: grouping size `l` 선택은 일반화 오차 상한(Theorem 1)으로 `l ≤ n/d_c`를 권장, QAT/PTQ 8-bit 양자화 실험 포함, cyclic HDPC(BCH) 코드에도 그대로 적용 가능함을 시연.
8. 학습 절차: PyTorch 기반 end-to-end 훈련, all-zero codeword 데이터셋(BP 대칭성 이용), 반복(iteration) 간 가중치 공유(RNN-like).
9. 결과: (1057,813) cyclic PG-LDPC에서 CN 메시지 수 7.55배 감소하며 RID 대비 우수, 5G(800,160)/(1500,300) LDPC에서 NNOMS-CPM-RID가 CPM-RID 대비 큰 개선 및 NMSA를 능가해 SPA에 근접, Rayleigh 채널 미스매치 조건에서도 SPA보다 우수한 견고성.
10. 한계: 학습 필요(온라인 적응 아님), block length가 매우 커지면(2640) NBP 자체가 SPA보다 성능 저하되는 사례 존재, HW 합성/ASIC 검증 없이 GPU 시뮬레이션(PyTorch)에 그침.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| NNMS/NOMS 학습 가중치 min-sum 변형 | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC의 min-sum CN 연산에 학습된 스케일/오프셋을 곱하는 형태로 대응 가능 — 단, 학습된 가중치를 저장/곱셈하는 HW 오버헤드 필요 |
| revolving(서브행렬 H0/H* 재사용) 구조 | `ecc_top.cpp` `Load_PCM()` | QC-LDPC PCM 구조를 활용한 디코더 단순화 아이디어는 H-matrix 로드/구조와 연관되나, Prime ECC는 이미 z=32 lifting 기반 row 병렬 처리라 설계 사상이 다름 |
| iteration별 파라미터 공유(가중치 iteration 독립) | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | iteration별 적응형 LLR 테이블과 개념적으로 유사(반복별 파라미터 다름)하나, 여기선 학습된 가중치이지 문턱값 테이블이 아님 |
| 8-bit quantization(PTQ/QAT) 실험 | `channel.cpp` `Set_LLR_Th()` | 양자화 비트수와 성능 트레이드오프 분석은 soft-read LLR 양자화 설계에 참고 가능 |
| VN/CN Tanner graph 전체 대비 축소 (l, Z 그룹화) | `GT.cpp` `Make_GT_HW()` | Graph Thinning(엣지 축소를 통한 HW 절감)과 목적이 유사(연산량 절감)하나 메커니즘은 학습 가중치 기반 신경망이라 직접 대응 아님 |

> 적용 가치: 중간 — min-sum 계열에 학습 가중치를 부여해 성능을 SPA에 근접시키는 아이디어와 QC-LDPC 구조 대칭성을 이용한 연산량 절감은 Prime ECC decoder.cpp의 min-sum 파이프라인 개선에 참고할 여지가 있으나, 신경망 훈련·가중치 저장이라는 근본적 패러다임 차이와 HW 미검증이 이식 난이도를 높임.
