> 총평: SC-LDPC에 locality를 부여해 sub-block 랜덤액세스를 주는 순수 부호구성 이론(BEC threshold 분석)으로, Prime ECC의 고정 binary QC-LDPC와 구조가 달라 이식성 낮음.

### B. 알고리즘 요약

1. 대상: 프로토그래프 기반 SC-LDPC 부호, 저장장치용 랜덤액세스 목적. 예시 `(3,6)` R=0.389~0.5, `(5,10)` 등 regular 앙상블.
2. 문제: SC-LDPC는 threshold saturation을 얻으려면 매우 긴 블록이 필요 → 저장장치의 작은 read 단위와 상충, sub-block 단독 복호 불가.
3. 모델: 프로토그래프를 BEC로 전송, lifting `L→∞`에서 density evolution으로 임계값 `ε*` 분석(식 1a~3).
4. 핵심결과(부정): 일반 SC-LDPC를 M개 sub-block으로 나누면 local threshold `ε*_m = 0` (Theorem 4, l=2면 항상 0 — Corollary 5) → 랜덤액세스 불가.
5. 핵심식: `ε*(H(I)) ≤ ε*(H) ≤ ε*(H_J)` (Lemma 1) — 부분행렬 임계값 상하한으로 local/global 임계값을 bound.
6. 핵심기법: Construction 1(SC-LDPCL) — 커플링 행렬 B1을 `A1`(계단형 t행)+`A2`(all-one) 구조로 설계해 local CN을 확보, `(l,r,t)` 프로토그래프 생성.
7. 설계변수: 커플링 파라미터 `t∈{1,...,l-2}`가 local vs global threshold 트레이드오프를 제어(t↑ → local↓ global↑, t=l-1이면 표준 SC-LDPC).
8. 확장: semi-global 복호 — 타깃 SB를 인접 d개 SB 도움으로 순차 복호(backtrack), sub-block 변동 채널 BEC(E_m) 모델 가정.
9. 결과: `(3,6,1)` R=0.444, `ε*_G=0.4772`, `ε*_1=0.4298`; semi-global 성공확률 하한 식(10) 유도, d에 따라 최적 t 달라짐(짧은 체인 t=1, 긴 체인 t=3).
10. 한계: BEC·점근(L→∞) 이론만, AWGN/유한길이/HW 없음, 정정능력·error-floor 수치 없음, 디코더 구현 미제시.
