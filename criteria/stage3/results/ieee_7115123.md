### B. 알고리즘 요약

1. 대상은 1-D 공간결합(spatially coupled) 그래피컬 모델로, 큰 크기 극한에서 스칼라(1차원 실수) 상태 `f_t(x)∈[0,1]`로 기술된다.
2. 문제: 공간결합이 BP threshold를 uncoupled MAP threshold로 끌어올리는 threshold saturation을 BEC·정규 LDPC 넘어 일반 채널·모델로 엄밀 증명.
3. 모델: 두 EXIT-유사 함수 `h_f, h_g`로 컴포넌트 DE `u_t=h_g(v_t)`, `v_{t+1}=h_f(u_t)`를 정의하고, 공간결합은 평균화 커널 `ω`와의 convolution `(3)`으로 확장.
4. 핵심: 컴포넌트 potential `φ(h_f,h_g;u,v)=∫h_g^{-1}+∫h_f^{-1}-uv`를 도입, DE는 `φ`의 좌표별 축차 최소화와 동치(Lemma 4).
5. 판정식: EXIT 두 곡선 사이 부호면적(총 gap `A=φ(·;1,1)`)의 양/음이 성공 여부를 결정 — "strictly positive gap condition".
6. 주정리(Theorem 1): 이 조건 하에서 `(0,1)`-interpolating traveling wave 해 `f_t(x)=F(x-st)`가 존재하고 파 진행방향 `sgn(s)=sgn(A)`.
7. `A=0`이 곧 두 EXIT 곡선 부호면적 균형 → 정지파(spatial fixed point) = threshold saturation 임계.
8. 고차원/무한차원은 가우시안 근사 + EXIT 방식으로 확장(비엄밀이나 예측 정확)해 일반채널 BP·min-sum에 적용.
9. 결과: (3,6) BEC 예시 A=+0.03125(파 우이동)/−0.0254(좌이동), min-sum (3,6) BAWGNC threshold가 uncoupled 0.3818 → coupled 0.429로 상승 예측.
10. 한계: HW·부호구현·BER 곡선 전무한 점근 정보이론 증명이며, min-sum은 threshold 예측만(비대칭 가우시안 근사).

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 공간결합(SC-LDPC) threshold saturation 이론 | 대응 없음 | Prime ECC는 고정 H-matrix binary QC-LDPC 블록부호로 convolutional 구조와 상이 |
| EXIT/potential 기반 threshold 판정 | 대응 없음 | 설계 분석용 점근 도구로 디코더 코드(min-sum 루프)에 직접 이식 대상 아님 |
| min-sum threshold 예측(가우시안 근사) | decoder.cpp C2V_Cal_New_Sgn(), CNU_Update_New_Mag() | 우리 min-sum 연산은 이미 보유 — 이 논문은 알고리즘 변경 없이 threshold 값만 예측 |
```

적용 가치: **낮음** — SC-LDPC threshold saturation의 점근 존재증명으로, 고정 QC-LDPC 블록부호인 Prime ECC의 디코더·부호·HW에 떼어 쓸 구체 기법이 없다.
