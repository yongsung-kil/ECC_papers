### B. 알고리즘 요약

1. 대상: 방송용 QC-LDPC(DVB-S2 1/2·2/3, ATSC 3.0 7/15·8/15·2/3), length `64800`, BI-AWGN/BI-Rayleigh 채널.
2. 문제: 실제 QC-LDPC가 Shannon limit 대비 0.4~1.0dB SNR loss를 겪는데 그 원인을 정량 분해하는 도구 부재.
3. 핵심 도구(QC-MET-DE): 같은 QC sub-matrix 내 edge를 한 edge-type으로 두고 서로 다른 sub-matrix는 다른 type으로 PDF를 추적(식 1,4,5) - QC 대칭성으로 single-edge 가정이 항상 성립해 SNRth를 정확 예측.
4. SNR loss 3분해: base matrix / 제한 iteration / 유한 codeword length 각각을 QC-MET-DE + DE-EXIT + 수정 DE + 유한길이 capacity로 산출 후 합산.
5. 수정 DE: iteration이 유한(50/60)일 때 δ+∞ 수렴 대신 target BER(10^-5)을 성공조건으로 하여 iteration loss를 이분법으로 측정(식 6).
6. 진단 결과: DVB-S2 loss의 대부분이 base matrix(1/2·2/3 각 0.48dB), ATSC 3.0에서 base matrix 재설계로 약 0.2dB 회복; SET-DE는 QC-MET-DE 대비 최대 0.18dB 부정확.
7. 부호설계: raptor-like PCM `H=[[A O];[B I]]`(식 7, sub-matrix 360x360)로 DVB-S2/ATSC 3.0 HW 재사용 가능하게 구성.
8. 최적화: check node degree는 semi-regular 가정, variable node degree를 hill-climbing + PEG로 SNRth 기준 반복 개선.
9. 결과: 제안 1/2·2/3 부호가 BI-AWGN에서 각 0.39·0.37dB, BI-Rayleigh에서 0.60·0.65dB gap - state-of-art 수준, 예측과 시뮬 일치.
10. 한계: waterfall(threshold)만 최적화, error-floor·인코딩·HW·constellation은 명시적으로 범위 밖(순수 시뮬).
