### B. 알고리즘 요약

1. 시스템: 광통신 FEC용 regular/irregular QC-LDPC, rate 5/6(≈0.833), 길이 18432~24576, J=4 block row, L=24 block col, z=768/1024.
2. 문제: 고rate·고 CN degree에서 quantized MS가 quantized SPA 대비 정확도 저하 → 높은 NCG 미달, 기존 아키텍처는 메모리·라우팅 과다.
3. 가정: BI-AWGN, layered decoding으로 메시지 전파 가속.
4. 핵심 기법 1단: RAM 동적저장 — 각 variable-check pair당 C2V 또는 V2C 메시지 하나만 시분할 저장(같은 위치 재사용), shuffle network 불필요로 라우팅 축소.
5. 핵심 식: LUT CNP `O(i,j)=Q_Δ{2 tanh^-1(tanh(iΔ/2)tanh(jΔ/2))}` — SPA를 2입력 LUT로 근사, 재귀 결합으로 CNP 복잡도 Θ(L).
6. 핵심 기법 2단: deep pipeline scheduling — pipeline hole(J(S-1)/iter)을 추가 codeword로 채워 throughput 100% 향상(G=3 group, S=4 stage 예).
7. 구현 디테일: V2C는 sign-magnitude, C2V/channel은 2's complement 저장, address controller는 QC 구조 덕에 단순 counter.
8. 최적화: 양자화 step Δ는 density/differential evolution으로 최적화(각주), 메시지 2q+1 레벨(4bit).
9. 결과: quantized SPA가 quantized offset MS 대비 0.14dB 이득(BER 10^-6, CN degree 클수록 커짐), Altera Stratix FPGA 80MHz, 1.33~1.97Gbps, ALUT 93k~162k.
10. 한계: 광통신 rate 5/6 타깃, 4bit soft, Code C는 Eb/N0 3.6dB에서 error floor, NAND/soft-read 미검증.
