### B. 알고리즘 요약

1. 대상은 rate `R=b/c=4/5`, 길이 약 1000/2000/4000의 regular·irregular QC-LDPC block code (tailbiting convolutional code 형태).
2. 문제: girth=6이면 이론한계에 근접한다는 통설이 있으나, degree distribution과 girth의 상호작용으로 더 나은 BER/FER 여지가 있음.
3. 모델: 다항식 parity-check matrix `H(D)` (monomial `D^{w_ij}`)를 base matrix `B`로 라벨링, tailbiting 길이 `M`으로 block code 생성.
4. 핵심기법: submatrix별 girth 벡터인 **girth profile** `g_p=(g_1...g_c)`를 정의 - `i`번째 성분은 가장 가벼운 `i`개 열로 만든 부분행렬의 girth.
5. 목표식: `sliding-window girth` `g_w > g` - 무게순 정렬된 연속 `w`개 열의 window 안에서 최소 girth를 보장 (저weight 열의 보호 강화).
6. 확장: greedy 알고리즘[10]으로 좋은 degree distribution의 base matrix를 얻은 뒤, 본 라벨링 탐색으로 girth profile 개선.
7. 탐색절차: 열을 하나씩 추가하며 target girth `g`, `g_w` 제약을 만족하는 후보만 유지 (Smin~Smax 후보, Nmax 시행)하는 greedy sliding-window 방식.
8. 부수효과: girth 제약만으로도 저weight 열의 연결성이 커져 ACE(approximate cycle extrinsic) 특성이 좋아짐.
9. 결과: 개선 라벨링이 random 라벨링 대비 고SNR에서 BER/FER 크게 개선, 길이 1000/4000에서 기존 최고[9] 부호를 능가 (BER 곡선, 수치는 그림에만 존재).
10. 한계: AWGN BP 시뮬만, gate count·throughput·HW 없음; 저SNR은 irregular 우세, 고SNR은 column weight 큰 regular가 오히려 우세.
