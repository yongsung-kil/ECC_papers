### B. 알고리즘 요약

1. 채널은 AWGN이 아닌 additive symmetric α-stable(SαS) impulsive 노이즈, BPSK 심볼 `L=8000`, Mackay rate-1/2 (3,6) regular LDPC 부호.
2. 문제: SαS PDF는 닫힌형이 없고(α=2 Gaussian, α=1 Cauchy 제외) 노이즈 상태가 급변, 기존 추정기는 hard-decision만 써 정보손실·성능열화.
3. 모델: 노이즈 파라미터 `θ=(α,γ)`, α는 impulsive 정도(1≤α≤2), γ는 scale(0<γ≤1); 2차 모멘트 무한이라 GSNR로 SNR 정의.
4. 핵심: EM 프레임에서 출력 y=관측변수, 입력 x=잠재변수로 두고 ML 추정 `θ̂=argmax Σ log p(y_j|θ)`의 하한을 최대화, 입력확률을 0.5가 아닌 복호 사후확률 `p(x|y)`로 가중.
5. 핵심 식(11): `Ĝ(θ)=Σ_j Σ_x 1/(1+e^{-xΛ_app(y)}) log f_θ(y_j−x_j)` — Sum-Product 복호가 내보낸 사후 LLR `Λ_app`로 추정목적을 재가중(cross-entropy 항은 θ와 무관해 소거).
6. 확장: SαS PDF가 닫힌형이 없어 `G(θ)` 최대화가 이론적으로 convex 증명 불가 → gradient descent로 근사탐색(산점도상 near-convex).
7. 구현 디테일: 유한차분 gradient `∇G=(G(θ+Δθ)−G(θ))/Δθ`, 갱신 `θ_{n+1}=θ_n+η_θ∇G`, update rate η_α=1e-4, η_γ=1e-5, gradient iter N=20, EM iter M에 민감(boundary 이탈 주의).
8. 학습/최적화: outer EM(추정)–inner gradient descent 이중 루프로 매 EM iteration마다 Sum-Product 복호 → 사후 LLR → 파라미터 갱신 반복.
9. 결과: 추정 α·γ의 RMSE가 EM iteration 증가로 감소, EXIT curve 상 ideal/SIR과 유사 수렴, BER은 SIR 대비 0.3~0.4dB 이득이며 채널 완전기지 ideal BP 대비 0.1dB 이내.
10. 한계: HW 미설계(FPGA/GPU 구현 가능성만 언급), SαS impulsive 채널 전용, gradient update rate 튜닝 민감.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SαS 채널 LLR 계산 Λ_θ(y) | channel.cpp Set_LLR_Th() | 채널 모델이 SαS라 NAND RBER/soft-read와 상이, 직접 대응 아님 |
| Sum-Product 복호 루프 | decoder.cpp LDPC_Decoder() | 우리는 Min-Sum 기반, boxplus full Sum-Product는 미보유 |
| EM 기반 채널 파라미터 추정 | 대응 없음 | Prime ECC에 채널파라미터 온라인추정 모듈 없음 |
| 사후 LLR 되먹임 | 대응 없음 | 조기종료/LLR 테이블은 있으나 파라미터 추정 되먹임 구조 아님 |
```

적용 가치: **낮음** — 채널 모델(SαS impulsive)이 NAND flash와 근본적으로 다르고, 기여의 핵심이 채널 파라미터 EM 추정이라 binary QC-LDPC ECC 디코더 이식 대상이 아니다.
