### B. 알고리즘 요약

1. 시스템: BSC(오류율 `αM`)상 `(γ,ρ)`-regular binary LDPC, 검증은 Tanner (155,64) 부호(rate 0.41, `dmin=20`), hard-1bit 메시지.
2. 문제: 유한길이 부호에서 무결점 반복복호도 trapping set(BFE local minima) 때문에 저중량 error를 못 고치고 error-floor 발생.
3. 모델: 복호기를 noisy 논리게이트(MAJ/XOR, von Neumann 실패, Bernoulli `αMAJ`,`α⊕`)로 구성 → 메시지에 오류 주입(식 (1)).
4. 핵심기법1: noisy Gallager-B — 랜덤 게이트 오류가 message update를 교란해 decoder가 local minima에서 탈출("Error errore eicitur"), trapping set 소멸.
5. 핵심식: 에너지함수 `Λv=νv⊕yv+Σμ̃c`, threshold `b=max(Λv)`; GDBF는 `Λv=b`인 bit만 flip해 BFE 최소화.
6. 핵심기법2: rewinding `F^r(L)` — `L0` iter 후 추정 `u`를 reference로 저장, 실패 시 채널값 대신 `u`로 재초기화하여 `r`라운드 반복(오류 누적 방지).
7. 구현 디테일: 임계 게이트(bit 추출용 MAJ, 채널값 레지스터, syndrome checker)는 noiseless; syndrome check는 마지막 `Z` iter만 수행해 전력 절감.
8. 분석절차: `FER=ΣPr{e}·FERe`, 미정정확률 `MERe`를 rewinding 합성(식 (5)~(7)), Markov chain의 dominant trapping set 탈출확률로 추정.
9. 결과: noisy Gallager-B가 무결점 대응기보다 우수, `L=1000` 근최적 게이트오류율에서 noiseless min-sum(더 복잡한 soft)보다 우수; PGDBF는 `n=5000,LR=4000`에서 ML bound 도달.
10. 한계: HW 미구현(에너지분석 향후), 오직 (155,64)/BSC/hard-1bit 시뮬, 이득이 노이즈 HW·TRNG라는 특수 전제에 의존.
