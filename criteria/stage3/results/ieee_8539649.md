### B. 알고리즘 요약

1. 시스템: binary QC-LDPC 부호구성, GF(2^8) RS 기반 `BRS(4,85)`, AWGN + BPSK, SPA 복호 max iter `50`.
2. 문제: QC-LDPC는 Tanner graph 짧은 cycle(trapping set)이 error floor 유발 → girth 크게 + min Hamming distance 상한 높이는 부호 필요.
3. 모델: 두 기존 구성 결합 — Xiao[5]의 RS 기반 girth≥8 구성 + Smarandache[7]의 protograph min Hamming distance 상한.
4. 핵심기법1: RS parity-check `BRS(d,n)`(식1)에서 `t`개 열 선택(`Λt`), 임의 세 원소가 [5]의 9개 조건 충족 시 girth≥8인 `BRS(d,t)` submatrix 획득.
5. 핵심기법2: 그 위에 protograph weight matrix 개념 추가 → circulant `Mi,j=Σ mi,j,s·Hi,j`로 병렬 edge 부여한 `H(d,t)`(식3) 구성 = Proto-RS-QC-LDPC.
6. 핵심식/근거: weight matrix 원소>1(병렬 edge)이면 min Hamming distance 상한↑ → WER/BER 성능↑ (모든 원소 1인 경우보다 우수).
7. 구성 전략 3종: (i) CN·VN 모두 regular, (ii) CN regular·VN irregular, (iii) CN·VN 모두 irregular.
8. 파라미터: `Λ8={2,5,7,13,20,32,54,60}`, `r=255`, `BRS(4,8)` → 길이 `2040`, rate `1/2` 부호 3종 생성.
9. 결과: 구성 (ii)(iii)가 기존 RS-QC-LDPC 대비 coding gain — BER `10^-5`에서 약 `0.05dB`, BLER `10^-4`에서 약 `0.02dB`. (i)은 gain 없음.
10. 한계: 순수 부호구성법, HW 미설계, 시뮬만, rate-1/2 단일 예제, gain 미미, .txt 수식 손상.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| RS+protograph H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | 우리 QC-LDPC H-matrix가 고정이라 대체 시 전면 재검증 부담 큼 |
| circulant/base matrix 표현 | `ecc_data.h` `PCM_b` (base+shift) | 표현 형식은 유사하나 girth-8 RS 조건은 우리 부호와 무관 |
| protograph 병렬 edge / min Hamming distance 상한 | 대응 없음 | 우리 부호는 특정 QC 구조 고정, 오프라인 부호설계 이론 |
| SPA 복호 (max 50 iter) | `decoder.cpp` `LDPC_Decoder()` | 우리는 min-sum, full-tanh SPA는 대응 없음 |

적용 가치: **낮음** — binary QC-LDPC 부호구성법이나 rate-1/2·gain 0.05dB에 그치고, H-matrix 전면 교체(프로파일 §4 최고 난이도)를 요구해 고rate NAND 부호에 이식 실익 적음.
