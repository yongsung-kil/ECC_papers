### B. 알고리즘 요약

1. 대상은 `L×L` circulant permutation matrix 배열로 이뤄진 `(gamma, rho)`-regular binary QC-LDPC(GF(2)), 예제 부호 (539,310)·(1573,1146).
2. 문제: 짧은 cycle(특히 6-cycle)이 BP 반복복호 성능을 열화시키는데 기존 CRT/GCRT 결합법은 block-row 순열만 써서 선택지가 부족.
3. 모델: shifting matrix `S(H)=(a_ij)`로 cycle을 대수적으로 표현, 2l-cycle 존재조건 `sum(a_mi,ni - a_mi+1,ni) ≡ 0 mod L` (Theorem 1).
4. 핵심 기법: 서로소 `L1..Ls`에 대해 CRT로 `a_ij = sum(a_ij^(k) * b_k * L_k) mod L`로 성분부호를 결합해 `gammaL×rhoL` PCM 생성.
5. 개선점: column weight가 row weight보다 작으므로(고rate일수록 더 작음) block-row 대신 **block-column 순열**을 써서 순열 후보를 `(rho!)^(s-1)`로 대폭 확대.
6. 이 중 girth>=6인 부호들에서 short cycle 최소인 것을 선택 -> 순열 자유도 증가가 곧 short cycle 감소.
7. 탐색 절차: 후보가 너무 많으면 bound `T`개를 랜덤 추출, 먼저 6-cycle 최소, 그다음 8-cycle 최소인 순열 선택.
8. 학습/최적화: 별도 DE/GA 없이 조합적 순열 탐색(예: T=10^5, 401초에 (1573,1146) girth-8 구성).
9. 결과: AWGN·BPSK·BP 50 iteration에서 BER 10^-6 기준 GCRT 대비 약 1.2dB 개선, 6-cycle 0개, 8-cycle 최대 429000개 제거.
10. 한계: HW 미설계·시뮬만, 예제가 소수(L=7,11,13) 기반 소형 array code에 국한, 연판정/양자화·NAND 채널 고려 없음.
