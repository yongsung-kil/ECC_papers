### B. 알고리즘 요약

1. 대상은 `m×m` circulant permutation matrix 기반 binary QC-LDPC 및 그 변형 S-QC-LDPC(예제 `m=173`, 4×4 sub-matrix 배열 Q).
2. 문제: QC-LDPC는 sub-matrix에서 한 행/열이 정해지면 나머지가 결정되고, length-4 cycle 하나가 생기면 `m`개가 동시 발생 -> 설계 자유도 제약.
3. 모델: shift matrix `P`로 H 표현, length-4 cycle 필요충분조건 `a+b+c+d ≡ 0 mod m` (식 5), 없는 조건은 `a_i1j1 - a_i1j2 + a_i2j2 - a_i2j1 != 0 mod m` (식 6).
4. QC 기준 결정함수 `a_ij = (i·j)/2` 형태로 `m`이 소수면 length-4 cycle 회피 가능(식 7~9).
5. 핵심 기법: identity 대신 basis matrix `E1..Em`(각기 다른 위치의 단일 1)의 cyclic shift 합으로 sub-matrix를 구성하는 S-QC-LDPC 정의 -> QC보다 랜덤성 증가.
6. 0-1 programming: 변수 `x_ij∈{0,1}`(sub-matrix 유무), `y_ij`(shift량), 목적함수(식 15)와 제약(행/열당 1 개수, cycle-free 조건 식 18/19)으로 PCM 최적 설계.
7. 인코딩 보정: Q가 가역이 아닐 수 있어 matrix B를 더해 `H=[Q B]`로 만들고 `G=[I (B^-1 Q)^T]`로 유일 인코딩 보장.
8. 학습/최적화: DE/NN 없이 정수계획 제약 만족 탐색, Class I(제약 없음)/II(행열당 1개 엄격)/III(length-4 cycle 허용) 3종 설계.
9. 결과: BPSK·AWGN·BP에서 BER 10^-4 기준 Class I·II가 각각 QC-LDPC 대비 약 0.35dB·1dB 개선, Class III(length-4 cycle 포함)조차 length-4 없는 QC-LDPC보다 우수.
10. 한계: HW 미설계·시뮬만, rate/length 명시 없음(m=173만), S-QC-LDPC가 QC 준순환성을 깨어 HW 라우팅 이점 상실 우려.
