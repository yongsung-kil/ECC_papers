### B. 알고리즘 요약

1. 대상: 단·중길이(n=512~2048) rate-1/2 irregular LDPC 부호의 Tanner 그래프 분석(부호 자체가 아닌 분석 알고리즘).
2. 문제: iterative BP 성능은 global girth뿐 아니라 local girth 분포와 error-floor에 영향을 주는 ACE(사이클 외부 연결도)에 좌우됨.
3. 기존 message-passing/trellis 기반 사이클 열거(MPA/IMPA [14][15])는 간접적이고 느림.
4. 핵심 기법: `SETA`(single-edge tree-apex) 서브그래프 확장 — PEG 확장을 변형해 symbol node의 한 edge씩 확장하여 그 edge를 지나는 짧은 사이클을 직접 탐색.
5. girth 계산: 다른 edge의 check가 depth `l`에서 발견되면 `girth = 2(l+1)`.
6. local girth = symbol node의 모든 edge에 대한 최단 사이클 중 최소값.
7. ACE 정의: `ACE = Σ(d_i - 2)` (사이클 내 symbol node degree 합). 각 check vertex에 누적 ACE weight를 부여해 사이클 검출 시 즉시 산출.
8. local ACE = symbol node를 지나는 짧은 사이클들의 최소 ACE. 그래프 전체는 average ACE로 요약.
9. 결과: irregular 단길이 부호에서 MPA/IMPA보다 빠름(Code I 18s vs MPA 472s); ACE PEG·G-ACE PEG 앙상블에서 girth·ACE 분포가 표준 PEG 대비 향상됨을 재확인.
10. 한계: HW 미설계, C 구현 CPU 실행시간 비교뿐, QC-LDPC에는 미적용 명시, 복호기 자체 개선 아님.

### C. Prime ECC 관련 모듈 핀포인트
