### B. 알고리즘 요약

1. 시스템: 802.11n(rate 1/2, length 648) 및 DVB-S2(rate 1/2, length 64800) 표준 LDPC 코드를 outer code로, 학습된 autoencoder를 inner code로 결합한 concatenated 구조. 채널은 1-bit quantized(ADC) AWGN + faster-than-Nyquist(10배속) 전송으로 인한 ISI/colored noise.
2. 풀려는 문제: 기존 LDPC는 선형 AWGN 채널엔 최적이지만 1-bit 양자화 같은 강한 비선형 왜곡 하에서는(특히 16/64-QAM 등 고차 변조) 성능이 급격히 저하됨.
3. 핵심 가정/모델: 1-bit quantization을 실수부/허수부의 sign 함수로 모델링, 비선형이지만 해석적으로 다루기 쉬운 채널로 채택.
4. 핵심 기법: LDPC로 인코딩된 심볼을 블록(길이 N) 단위로 나눠 동일 파라미터의 autoencoder(6-layer: FC-Linear → channel/quantization lambda layer → FC-ReLU×3 → FC-Linear)에 통과시켜 인코딩/디코딩. 손실함수는 `W* = argmin ||d_i - d̂_i||^2`.
5. 핵심 식: 폭(width)이 무한대로 가는 autoencoder는 NTK(Neural Tangent Kernel) 이론에 의해 학습 전 과정에서 출력이 Gaussian process를 유지함(`K^(6) = F5(F4(...F0(K^(0))...)))`) → LDPC 디코더가 기대하는 Gaussian 통계를 1-bit 비선형 채널에서도 근사 제공한다는 이론적 보증.
6. 확장 기법: faster-than-Nyquist(심볼을 Nyquist rate의 10배 속도로 전송)로 대역폭 증가 없이 autoencoder 학습 용량(뉴런 수 G)을 키움, 대신 ISI/colored noise를 감수.
7. 구현 디테일: 코드워드 전체를 한 번에 학습하면 파라미터 폭발이므로 작은 블록(N=24 또는 64) 단위로 autoencoder 파라미터를 공유(1D CNN과 유사한 stride 구조), quantization은 sigmoid로 근사해 미분 가능하게 처리.
8. 학습 절차: 표준 gradient descent, Xavier/He 초기화(Gaussian). 일반화가 어려워 k개 코드워드마다 디코더 부분을 주기적으로 재학습하는 정책 채택.
9. 결과: 16-QAM/64-QAM 모두 1-bit 양자화+LDPC+autoencoder 조합이 unquantized LDPC와 거의 동등한 BER waterfall을 달성(LDPC 단독은 1-bit 양자화에서 매우 열악), 짧은(648) 및 긴(64800) 코드워드 모두에서 유사 경향.
10. 한계: 순수 시뮬레이션(HW 미설계), LDPC 코드 자체는 기존 표준 그대로 사용(부호설계/디코더 알고리즘 개선 없음), 무선 AWGN/QAM 변조 특화 채널 모델로 NAND flash 채널과 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Autoencoder inner code / NN 기반 채널 대응 | 대응 없음 | NN 전면 대체 성격의 기법으로 프로파일 §6 "대응 없음 예시"(NN/딥러닝 디코더)에 명시적으로 해당 |
| LDPC outer code (802.11n/DVB-S2 표준 그대로) | 대응 없음 | 부호설계/디코더 알고리즘 변경 없이 기존 표준 재사용, Prime ECC의 QC-LDPC 구조와 무관 |
| 1-bit 양자화 채널 모델(무선 AWGN+QAM) | 대응 없음 | `channel.cpp`는 NAND RBER/fixed-error 채널 모델이며 무선 QAM/1-bit ADC 모델과 목적이 다름 |

적용 가치: 낮음 — 무선 통신(QAM 변조, faster-than-Nyquist, 1-bit ADC) 특화 논문으로 LDPC 자체는 기존 표준을 그대로 사용하며 핵심 기여가 NN 기반 채널 등화/등가화(autoencoder)에 있어 NAND binary QC-LDPC의 부호설계·디코더·HW 어느 레이어에도 이식 근거가 없음.
