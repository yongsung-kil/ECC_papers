### B. 알고리즘 요약

1. 시스템: EPR4 PR 채널(`P(D)=(1-D)(1+D)²`)+AWGN, LDPC 단독 ECC(외부 RS 제거), rate 0.82/0.9, length ~4095-4550.
2. 문제: LDPC 실패 시 다수 비트 오류가 나 외부 RS가 못 잡음 → LDPC 하나로 residual 오류와 media-defect/thermal-asperity burst를 모두 처리 가능한지 검증.
3. 모델: media defect/thermal asperity를 "noise burst"(강한 노이즈 구간)로 모델링, 이상적 burst detector가 CSI를 제공한다고 가정.
4. 핵심 기법 1: burst 검출 CSI를 EPR4 SISO 검출기뿐 아니라 LDPC 디코더에도 전달하여 해당 비트 LLR을 0(erasure)로 설정 → error-rate 곡선 평탄화(flooring/flaring) 제거.
5. 핵심 기법 2: 최대 해결가능 erasure-burst 길이 `L`을 정의 — burst 위치 무관하게 디코더가 반드시 복원하는 최장 erasure 열.
6. L 계산 알고리즘: H의 submatrix `H(p,l)`에서 weight-1 행을 찾아 해당 열을 삭제, 반복해 전 열 삭제 성공 여부로 판정(brute-force 대비 10배 이상 빠름).
7. 부호 비교: MacKay(4095,3358), EG cyclic(4095,3367), eIRA(4095,3367)/(4550,4096), array code, RS-I/II 대조.
8. 부호 특성: eIRA는 upper-triangular 구조로 인코딩이 codeword 길이에 선형(효율적 인코딩), EG는 circulant이라 shift-register 인코딩 가능.
9. 결과: rate-0.9 LDPC가 RS 대비 측정영역에서 1.2 dB 이상 우수, burst 128bit에도 강건; `AWGN`와 `burst` 하한/상한 bound 차이 0.8 dB 이내. EG는 최소거리 커 floor 낮으나 시뮬영역 성능 열위.
10. 한계: HW 미설계, 특정 PR 타깃 가정, 고rate LDPC의 검증가능한 낮은 error-floor 설계는 미해결 과제로 남김.
