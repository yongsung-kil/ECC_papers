### B. 알고리즘 요약

1. 대상: 40Gb/s DWDM 광전송용 regular block-circulant LDPC(예 `[4608,3744]` rate~0.81, `[4608,3456]` rate 0.75), min-sum 소프트판정 복호.
2. 문제: 광통신 고속 FEC에서 pseudorandom LDPC는 encoder/decoder 복잡도 과다 → 저장·인코딩·복호가 간단한 구조적 QC-LDPC 필요.
3. 핵심객체: `(λ-1)`-fold cycle-invariant difference set(CIDS) — difference set을 순환 shift해도 difference set이 유지되는 조합구조.
4. PCM 구성: basic circulant permutation matrix `P`의 서로 다른 거듭제곱 `P^{a_ij}`을 blocks로 하는 block-circulant `H` (식 (1)); 지수는 CIDS로 선택.
5. girth/min-dist: CIDS 지수 선택으로 4-cycle 제거, girth>=6 보장(min-dist>=4~6); permutation block 재배열로 girth 8/10 이상 달성 가능.
6. 부호구성: primitive element of `GF(q)` 기반 Bose construction (식 (3)(4))로 CIDS 생성, 부분집합 삭제·재배열로 카디널리티 조정.
7. 인코딩: `H`가 block-Toeplitz/permutation-block → generator도 block-circulant, 선형시간 인코딩(shift register); girth8 부호는 역행렬이 non-Toeplitz라 인코딩 복잡.
8. 사이클 수 저감: PCM 구조 수정으로 girth 유지하면서 6-cycle 개수 대폭 감소(정성).
9. 채널모델: WDM+CSRZ+MZ modulator, ASE noise, fiber 비선형(nonlinear Schrödinger, split-step Fourier), 분산·크로스토크·ISI; Q-factor로 평가.
10. 결과: `[4608,3456]`가 BCH[128,113] turbo block code 대비 ~1dB↑, BER 목표에서 coding gain ~10dB, modified Bose 부호는 >11dB(당시 최고); min-sum은 add+min만이라 turbo 대비 복잡도 대폭 낮음.
11. 한계: HW 미설계(gate/throughput 없음), 광통신 특화 채널 평가, NAND/soft-read 무관, girth 상향 시 인코딩 부담 급증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| permutation-block QC PCM 구성(CIDS) | `ecc_top.cpp` `Load_PCM()`, `ecc_data.h` `PCM_b`(base+shift) | circulant shift 지수 구조는 QC base+shift 표현과 유사, 매핑 검토 가능 |
| min-sum 복호(add+min) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC min-sum과 동일 계열이나 신규성 없음(이미 보유) |
| block-circulant 선형시간 인코딩 | `encoder.cpp` `LDPC_encoder_4KB()` | Prime ECC는 dual-diagonal 인코딩 고정, 구조 상이 |
| girth/사이클 최적화 부호설계 | `ecc_top.cpp` `Load_PCM()` | 부호구조 오프라인 설계 관점 참고 |
| DWDM 광채널/Q-factor 모델 | 대응 없음 | 광통신 특화 채널, NAND RBER/AWGN과 무관 |

적용 가치: **낮음** — CIDS 기반 girth 보장 QC-LDPC 구성법은 이론적 참고가치가 있으나 광통신 특화·부호설계 한정이고, min-sum 복호는 Prime ECC가 이미 보유해 신규 이식 실익이 낮다.
