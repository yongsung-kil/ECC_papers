### B. 알고리즘 요약

1. 대상: 유한기하(Euclidean/projective geometry) 기반 sparse circulant을 원소로 하는 QC-LDPC 부호 구성, BPSK/AWGN에서 SPA 복호.
2. 문제: 잘 설계된 QC-LDPC가 랜덤 LDPC와 동등 성능을 내면서 인코딩 이점(FSR)을 갖도록 대수적으로 구성하고 싶다.
3. 핵심 기법: circulant `G`(EG 라인 관점 weight `ρ`)를 첫 열/행을 나눠 `κ`개(열분할 factor), `τ`개(행분할 factor)로 `cyclic shift` 하는 **array decomposition** 으로 더 sparse한 부호를 생성.
4. RC 제약(row-column, 두 행/열의 공통 1이 ≤1)을 만족하면 Tanner graph에 길이-4 cycle이 없어 `girth≥6` 보장.
5. Example 1: EG* 511점 9개 circulant 중 8개 → 2×16 array (weight 2), 1022×8176 (col wt 4, row wt 32) → (8176,7156) regular, rate 0.8752, `d_min≥5`.
6. Example 2: κ=4,τ=2로 permutation matrix 배열 후 일부를 zero로 대체 → (8176,6132) irregular, rate 0.75.
7. 구조적 호환(§IV): mother code의 앞쪽 circulant 열만 사용해 shortened 부호 계열 생성 → 하나의 디코더 회로로 여러 rate/length 복호.
8. Broad-sense(§V): RC 제약을 풀고 2-flat 기반 circulant 사용, cycle-4가 있어도(분해로 3.6M→9198개로 급감) 성능 유지.
9. 결과: (8176,7156) BER 10^-6에서 Shannon 한계 1dB 이내, error floor 없음; 등가 랜덤 MacKay 부호와 곡선 일치, irregular 대비 낮은 error floor (NASA-Goddard 위성통신 채택).
10. 한계: 순수 부호구성 letter — 디코더/HW 미설계, 양자화·soft-read 논의 없음, 성능은 full SPA 기준.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| circulant array decomposition으로 만든 H-matrix | [ecc_top.cpp](../Prime_ECC_3.1_Claude/) `Load_PCM()` | 새 QC-LDPC PCM을 로드하는 지점 — 부호 교체 시 여기로 진입 |
| QC 구조(base+shift) 부호 자체 | `ecc_data.h` `PCM_b` | Prime ECC도 base+circulant shift 구조 → 개념 호환, 단 z=32/고rate 고정 |
| 구조적 호환 shortened 부호 계열 | 대응 없음 | Prime ECC는 단일 부호 고정, 다rate 계열 미지원 |
| SPA 복호 성능 | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum(근사)라 full-SPA 결과와 직접 비교 불가 |

적용 가치: **낮음** — 개념적으로 호환되는 대수적 QC-LDPC 구성법이나, Prime ECC는 특정 H-matrix가 고정된 이식 난이도 최상위 레이어(부호설계)이고 디코더/HW/양자화가 없어 실제 이식 가치는 제한적.
