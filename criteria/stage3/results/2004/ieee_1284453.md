### B. 알고리즘 요약

1. 시스템: EPR4 PR 채널(또는 EPR4 타깃 등화 MRC) + regular (4,36) LDPC, rate 8/9, length 4608 `(4608,4,36)`.
2. 문제: BCJR/MP 복호가 노이즈 분산 `σ²`을 요구하나 실제로 정확히 알아도 성능이 최적이 아니며, `σ²`를 일부러 틀리게 쓰면(SNR mismatch) 성능이 바뀜.
3. 모델: 복호기가 쓰는 분산을 `βσ²`로 두고 `SNR_mismatch = 10log10(1/β)`로 정의. 채널 message pdf를 Monte Carlo로 히스토그램화하여 DE 입력으로 사용.
4. 핵심 기법: BCJR 채널검출이 SNR mismatch에 둔감함을 증명 — max-log-MAP/비터보 log-MAP에서 `σ`가 APP path metric에서 factor-out되어 hard-decision BER 불변.
5. 의미: 따라서 mismatch 효과의 주범은 LDPC MP 복호기(입력 LLR 스케일에 민감)임을 특정.
6. 확장: 채널 pdf의 두 속성 — Gaussianity와 VMR(variance-to-mean ratio)이 mismatch의 원인임을 가설화·검증.
7. 규칙: VMR>2면 SNR 과대추정(음의 mismatch), VMR<2면 과소추정이 유리. 경험식 `mismatch ≈ 10log10(VMR/2)`.
8. 최적화: DE로 SNR threshold를 VMR·erasure별로 계산, BER 추정식 (8)로 실용 최적값 도출.
9. 결과: AWGN 지배 MRC는 -3~-2 dB, 10%AWGN+90% media(jitter) noise는 -10~-8 dB가 최적; 시뮬과 DE 예측이 일치. 3% erasure 유무에 무관하게 동일 값 사용 가능.
10. 한계: HW 미설계, 특정 PR 타깃/채널모델 전용 수치, 반복 채널검출 미고려(BCJR 1회만).
