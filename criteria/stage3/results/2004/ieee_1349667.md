### B. 알고리즘 요약

1. 시스템: n차원 lattice `Λ`를 orthogonal sublattice `Λ'`의 coset union으로 보고 label code `L`(Abelian group block code, 알파벳 `G=Z_g1×...×Z_gn`)로 표현.
2. 문제: lattice 복호의 integer programming/trellis 접근은 고차원에서 비실용적 - iterative(min-sum) 복호로 수천 차원 lattice를 저복잡도로 풀고자 함.
3. 모델: lattice decoding = 입력 `x∈R^n`에 최근접 lattice vector `p` 찾기, label group `G_i`와 `Z_gi` 간 isomorphism으로 정수 라벨링.
4. 핵심 기법: Gallager/Tanner min-sum(max-sum)을 group code로 일반화 - symbol node에 `g_i`개 성분 weight vector `w_i^(j)=‖d_ij - ξ_i‖²` 할당, 초기화/hard decision/symbol·check node 갱신.
5. 핵심 식: symbol node 갱신 `M_{si→chk}(j)`은 incoming check message 합, check node 갱신은 locally valid configuration 최소 weight (min-sum 규칙의 group 버전).
6. 확장: Construction D' + nested LDPC code로 lattice 구성 시 label group 크기 `g_i = 2^(a+1)` 고정, congruence 해 개수를 Lemma로 정확히 산출.
7. 트릭: check node 연산 복잡도를 label group 크기·check degree·edge 수로 표현, 정수 modulo 연산(mod 6, mod 2 등)으로 parity check 표현.
8. 최적화: 없음 (학습/최적화 절차 아닌 복잡도 상·하한 유도가 핵심).
9. 결과: 복잡도가 차원 n에 선형, label group 크기에 지수적, coding gain에 다항시간 - 수천 차원 lattice도 쉽게 복호 가능하다고 주장.
10. 한계: HW 미설계, 순수 이론(복잡도 bound)이며 정정능력/BER 수치 없음, binary가 아닌 비이진 group code lattice 대상.
