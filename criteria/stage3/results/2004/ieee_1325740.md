### B. 알고리즘 요약

1. 대상은 partial response(EPR4) 자기기록 채널이며, column weight 2 LDPC인 cycle code로 girth를 최대화해 반복복호 성능을 높이는 것이 목표.
2. 문제: cycle-free Tanner graph가 최적 복호를 주므로, 동일 block length·rate에서 최대 girth(=girth 12) cycle code를 구성해야 함.
3. Gallager의 tier-Tanner-graph 하한식(식(1)(2))으로 girth 12 달성에 필요한 최소 block length를 설정.
4. 핵심 기법: 3-tier graph의 3번째 tier에서 bit node↔check node를 short cycle(길이≤10) 없이 연결하는 mapping matrix `M_l`을 구성.
5. 두 short-cycle 시나리오(동일 branch 내 공유 check, 정수쌍 중복)를 제약으로 배제 → girth 12 보장.
6. Lemma 1: row weight `p`, 특정 최소 block length의 girth-12 cycle code는 `p`가 소수일 때 존재; mapping은 `GF(p)`의 primitive element 순환시프트로 구성.
7. Lemma 2: 이 부호는 bottom tier bit가 정보비트이고 XOR/circular shift만으로 linear complexity 인코딩 가능(생성행렬 불필요, 인/디코더 HW 통합 가능).
8. Lemma 3: cycle code의 min-distance는 최단 cycle의 bit node 수 = girth/2 = 6이며, multiplicity를 12-cycle 계수로 정확히 산정(식(9)).
9. 결과: rate 3/4(len 456)·rate 8/9(len 5526) 부호에서 sum-product가 ML union bound(식(10))에 근접; EPR4 채널서 동일 random code보다 낮은 BER에서 error floor 도달, block당 잔여오류 대부분 10bit 미만(outer RS에 유리).
10. 한계: HW 미설계(시뮬만), PR/EPR4 채널 특화, gate count·throughput 없음, RS 연접 전체성능은 미평가(향후과제).
