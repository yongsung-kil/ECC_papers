### B. 알고리즘 요약

1. 시스템: 2-user 페이딩+AWGN Gaussian broadcast channel, 수신단만 페이딩 정보 인지. 예제 부호 rate `0.522`, blocklength `n=10^3, 10^4`.
2. 문제: BC 용량영역이 미지이며 중첩 전송된 두 codeword를 최적수신단에서 결합복호해야 stripping 대비 성능손실을 피함.
3. 모델: `X = U√(αP) + V√((1-α)P)` 전력분배 중첩, user-Y는 두 메시지 결합복호, user-Z는 상대 codeword를 잡음처리.
4. 핵심기법1: sum-product(message passing) 복호를 다중사용자로 확장, factor graph에 두 코드를 잇는 mapping node 추가.
5. 핵심식: mapping node 메시지 `μ = log(cosh(·+·)/cosh(·-·))` (식16)이 상대 사용자 codeword 기여분을 제거 → soft interference cancellation.
6. 확장: `f_t(x)=log cosh(x+t)-log cosh(x-t)≈2t·sign(x)` 근사로 저복잡도 hard interference cancellation(식18) 유도.
7. 안정성조건: DE로 LLR 밀도의 무한대 질량점 거동 분석, user별 안정성 조건(식19,20) 도출하여 후보 부호 선별.
8. 최적화: EXIT chart로 threshold 추정 + differential evolution으로 degree 분포 `λ(x),ρ(x)` 최적화 (user-Z 먼저, 그다음 user-Y).
9. 결과: 최적화 부호가 binary 입력 achievable region 경계에 근접, threshold 대비 약 1dB 이내에서 동작(blocklength 10^4).
10. 한계: HW 미설계, 무선 broadcast 채널 특화 시뮬레이션만, NAND read/양자화 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| degree 분포 최적화(DE+EXIT) | `ecc_top.cpp` `Load_PCM()` | 부호구조 설계 레이어이나 QC-LDPC 고정과 무관, 이식성 낮음 |
| mapping node soft interference cancellation | 대응 없음 | 다중사용자 factor graph 노드로 Prime ECC single-user 구조에 없음 |
| sum-product message passing | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum 근사 사용, full-tanh SPA는 이미 대체됨 |

적용 가치: **낮음** — 무선 2-user broadcast의 결합복호/간섭제거 이론으로, NAND single-user binary QC-LDPC 부호설계·디코더에 떼어낼 요소가 사실상 없음.
