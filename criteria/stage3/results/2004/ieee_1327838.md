### B. 알고리즘 요약

1. 매우 고rate(rate→1) LDPC를 정수격자 기반 결정론적 구성으로 설계, AWGN 채널 시뮬 평가.
2. 문제: 짧은 cycle이 message-passing 성능을 해치는데, 고rate 부호에서 girth-8 확보는 저rate보다 훨씬 드묾. random 부호 대신 deterministic sparse H 원함.
3. 핵심 모델: parity-check matrix를 `m×k` 직사각 정수격자의 점-선 incidence로 정의(1-configuration, BIBD 부분설계), 결과 H는 circulant 행렬들의 배열(column weight 1인 circulant 서브블록).
4. 핵심 기법 1단: 서로 다른 기울기(slope) `s`의 평행선 클래스를 골라 배치, 평행선 클래스 하나가 resolvability class = 순환 서브블록에 대응.
5. 핵심 식: girth-8은 slope 집합이 "arithmetically constrained sequence"(3항 등차수열 없는 수열)의 부분집합일 때 성립(Theorem 3.1), earliest sequence로 생성.
6. 핵심 기법 2단: maximal slope 집합 찾기는 지수복잡도라, select-check-disregard greedy로 다항시간에 girth-8 slope 집합 구성(girth는 Dijkstra/Bellman-Ford로 검사).
7. 구현 디테일: slope-set 크기가 클수록 code rate↑, 첫 열의 nonzero 위치는 base block 원소로 결정 → 배열 순환구조.
8. 이론 결과: girth-8, column weight 부호의 최소거리 하한식 제시(Gallager bound).
9. 결과: `m=127`류 예시서 maximal-slope 부호는 Shannon limit 2.8dB, 다항시간 부호는 3.4dB(짧은 부호치곤 양호). girth-8 부호가 length-4 제거 random 부호와 거의 동등 BER.
10. 한계: 순수 구성법(HW 미설계, 정정능력 정량이득 미제시), column weight 3~4 저degree, 고rate 실험은 maximal slope 탐색난으로 제한적.
