### B. 알고리즘 요약

1. 대상: `(dv,dc)`-regular LDPC 부호를 BSC에서 복호하는 binary message-passing(hard-decision) 알고리즘, 특히 majority-based(MB) 계열.
2. 문제: Gallager algorithm B 같은 switch-type hybrid(`H_S`)는 iteration마다 알고리즘을 바꿔 성능을 얻지만 채널 파라미터(crossover `alpha`)의 정확한 값을 알아야 하고 채널 변동에 민감.
3. 핵심 기법: time-invariant hybrid(`H_T`) — 각 iteration에서 v-node/c-node를 확률질량벡터 `~a`에 따라 N개 그룹으로 무작위 분할해 서로 다른 MB 알고리즘 `Psi_i`를 동시에 섞어 적용, 혼합비율은 iteration에 무관(time-invariant).
4. 메시지맵: hybrid 알고리즘을 `Psi = (sum a_i Phi_i, ...)` 형태의 변수/체크노드 메시지맵 혼합으로 정의(식 1).
5. 수렴 조건: density evolution 곡선 `Q_alpha(a,x)`로 표현, `Q < x` for all x ⇔ 오류확률이 iteration에 따라 0으로 수렴(식 2, A-2).
6. threshold: 각 알고리즘의 오류확률 0 수렴 최대 `alpha`(threshold)를 `~a`에 대해 최대화; hybrid `H_S`/`H_T` 모두 구성 MB 알고리즘 개별보다 threshold 크게 상승(Table I).
7. 비교: `H_T`는 `H_S`보다 threshold는 약간 낮으나, 채널을 모르거나 과소추정할 때 훨씬 robust — `H_S`는 수렴 실패 가능, `H_T`는 거의 영향 없음(Fig.2,3).
8. 보편성: `alpha_max` 목표로 설계한 `H_T`가 특정 채널 전용 설계와 거의 동등 성능 → 설계가 채널에 둔감(universal).
9. 결과: 정성적·이론적 — threshold 향상 및 수렴속도(iteration 수) 개선을 density evolution/BSC로 제시, 구체 부호길이/BER 곡선·gatecount 없음.
10. 한계: BSC hard-decision 이진 MP 전용, DE 이론분석만(HW·양자화·soft-read·특정 부호 인스턴스 없음).
