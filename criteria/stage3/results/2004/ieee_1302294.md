### B. 알고리즘 요약

1. 문제: 곱함수의 marginal(MPF)을 junction tree 메시지패싱(GDL)으로 저복잡도 계산하는 것을 측도론 틀로 재정식화.
2. 기존 GDL은 "변수" 단위로만 독립성을 만들어(moralization+triangulation) 변수보다 미세한 구조(예: weight 행렬의 low-rank/sparsity)를 놓친다.
3. 핵심 재정식화: local domain → sample space 위의 `sigma-field`, local kernel → 그에 측정가능한 random variable, marginal → 조건부기대 `E[·|F]`. signed measure 사용.
4. 독립성 판정: 원자쌍의 joint-measure 행렬 `M_a`가 rank-one이면 조건부독립 성립; 이를 이용해 finest valid partition을 정의.
5. Lifting(핵심 신기법): junction tree가 없을 때 `sigma-field`의 원자를 rank-one 분해(`M_a = sum M_a^k`)로 분할(split)하여 독립성을 인위 생성 - moralization+triangulation의 대체물.
6. signed measure 덕에 rank-one 분해를 SVD 등 표준기법으로 수행 가능(양수 rank-one 분해는 NP-hard).
7. 알고리즘 3(junction tree 존재판정)+알고리즘 4(lifting으로 구성)+알고리즘 2(메시지패싱 복호)로 구성.
8. 복잡도: junction tree 구성(lifting)은 코드/문제 크기에 지수적이나 offline 1회면 되고, 이후 인접원자 joint-measure 테이블만 저장하면 됨.
9. 결과: 소블록 LDPC 정확복호(Example 7)에서 triangulation 대비 훨씬 낮은 clique 크기 -> 선형 블록부호 최소 trellis BCJR과 등가 알고리즘을 자동 도출. MATLAB 시뮬(Pentium III, 최대 코드 30분).
10. 한계: 순수 이론/자동화 프레임워크, HW 없음, iterative message-passing 복호 개선 아님, 실용 대블록 LDPC엔 lifting 지수복잡도로 부적합.
