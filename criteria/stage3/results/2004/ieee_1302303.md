> 총평: high-rate(최대 0.96) binary QC-LDPC를 Bose-BIBD+circulant 분해로 구조적으로 설계 - Prime ECC의 base+circulant shift·shift-register 인코딩과 같은 클래스라 부호설계 방법론이 개념 이식가능하나 circulant 크기가 소수 종속·HW 미검증이라 이식성 중.

### B. 알고리즘 요약

1. 목표: 랜덤 LDPC와 달리 소수 파라미터로 완전결정되는 구조적 regular QC-LDPC를 설계, shift-register로 저복잡도 인코딩(직렬 O(parity), 병렬 O(length)) 가능케 함.
2. 핵심틀: BIBD `2-(v,gamma,1)`의 point-block 결합행렬 `Q`를 PCM으로 사용, `lambda=1`이 girth 정확히 6 보장(cycle-4 없음), 열무게 `gamma`로 `d_min>=gamma+1`.
3. 부호구성(Bose designs): Bose의 symmetrically-repeated-difference 방법으로 `2-(v,4,1)`/`2-(v,5,1)` BIBD 3클래스(Class I/II/III) 생성. 유한체 `GF` 기반이라 결합행렬이 circulant 구조.
4. Class-I: `Q=[Q1..Qt]` circulant 1행 -> `H(1)[m]=[Q1..Qm]` 선택으로 length `m(12t+1)`, rate `~(m-1)/m` QC-LDPC. base block `Bi={0,x^2i,x^2i+4t,x^2i+8t}`.
5. Class-II/III: Theorem 3.2 기반, `M,C` circulant 배열 + identity로 고rate(예 rate 0.96, `d_min>=5~6`).
6. Row splitting: 한 행을 rotating 방식으로 `s`개 행으로 분할 -> rank 증가 -> 저rate 코드 파생(같은 length, 열무게 유지).
7. Circulant decomposition(Section V, 핵심 유연화): 각 circulant `Qj`의 첫 행을 `1` 성분별로 분할·순환shift해 무게-1 circulant permutation matrix `Qi,j`로 분해(`sum Qi,j = Qj`). 이를 `c x m` 배열 `G[m]`로 재조합 -> `(c,m)`-regular, girth>=6, `d_min = c+1`(c 홀) / `c+2`(c 짝), rate·length 자유 조정.
8. 인코딩: PCM이 circulant 배열이라 QC 구조 -> feedback shift register로 인코딩.
9. 결과(시뮬, SPA max 100 iter): (2715,2535) rate 0.934 코드 Shannon 1.35dB, (7676,7375) rate 0.96 코드 0.95dB, 등가 랜덤 MacKay 코드와 대등하거나 약간 우위. avg iteration 통계(Table I,II) 보고.
10. 한계: HW 미설계, SPA 시뮬만, circulant 크기가 `GF(12t+1)`/`GF(20t+1)` 소수(prime power)에 종속 -> 임의 lifting size 불가, gate/throughput/실측 없음.

적용 가치: **중간** — Prime ECC와 동일한 binary QC-LDPC(base+circulant shift) 클래스의 고rate 부호설계 방법론(BIBD + circulant 분해)이라 개념 이식가치가 있으나, circulant 크기가 소수 종속이라 z=32 등 규격 lifting과 직접 맞지 않고 HW·실측 미검증으로 부호설계 계층 이식은 재검증 부담이 크다.
