### B. 알고리즘 요약

1. 대상: AWGN 채널·sum-product 복호 하의 irregular binary LDPC 차수분포 `λ(x), ρ(x)` 설계.
2. 문제: 기존 all-Gaussian 1-D 분석([1])은 check-node 출력까지 Gaussian 가정해 max variable degree≤10, rate 0.5~0.9로 제약, capacity 접근 곤란.
3. 핵심 관찰: variable-node 출력은 CLT로 Gaussian 근사가 정확하나 check-node 출력은 soft-min 특성으로 원점 편향된 비Gaussian.
4. semi-Gaussian: 채널·variable 메시지에만 Gaussian 가정, check 출력은 "true" pdf 사용 → 한 iteration을 depth-one tree로 통째 분석.
5. 지표: GA 매칭 measure로 mutual information이 가장 정확(ten Brink 결과와 일치), threshold 오차 수천분의 1 dB.
6. EXIT chart: `I_out=F(I_in, I_ch)`를 depth-one tree density evolution으로 몇 점만 계산·보간; 임의 stage부터 시작 가능해 DE보다 저복잡도.
7. irregular: 각 variable degree별 elementary EXIT chart의 차수분포 가중 선형결합(식 10)으로 전체 EXIT chart 구성.
8. 설계: EXIT tunnel이 `f_e(x)<x` 조건 유지 하 rate 최대화를 LP로 정식화; degree 분포 미지 문제는 single-Gaussian 초기화 후 재귀 보정.
9. 결과: rate 0.25~0.95 범위 설계, rate-0.9497 부호(check degree 120, max var degree 40)가 Shannon limit 대비 0.0340 dB gap; DE 설계 부호와 대등.
10. 한계: 순수 부호설계/분석 도구, HW·복호기 미포함, rate<0.25 부정확, sum-product/AWGN 전제.
