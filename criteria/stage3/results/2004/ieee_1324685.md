### B. 알고리즘 요약

1. 대상 시스템은 40Gb/s WDM(채널간격 100GHz) 장거리 광통신이며, 목표는 RS/turbo 대비 높은 coding gain과 high code rate의 LDPC 부호 설계.
2. 기존 pseudorandom LDPC는 인코더/디코더 복잡도가 고속 광통신 한계를 초과 → 조합론적 대상으로 순환구조 부호를 구성해 permuter 없는 단순 회로화가 목표.
3. 핵심 기법: 합성수 `n`의 소인수분해를 이용해 MacNeish–Mann 정리로 order `n`의 MOLS(mutually orthogonal Latin squares) 완전계를 구성.
4. 각 MOLS `Li`가 Galois field 연산(식 (4)(5))으로 셀을 채우고, MOLS↔BIBD 일대일 대응으로 incidence matrix를 만든다.
5. 이 incidence matrix가 곧 parity-check matrix가 되며, 각 line의 label이 H의 한 행에서 "1"의 위치를 지정.
6. 부호 length는 점 개수, code rate는 block/point 관계로 결정되며, `n`이 소수거듭제곱이면 이전 구성([8])으로 환원.
7. 부호 특성: girth 최소 6, high code rate, min-distance 하한이 존재(BIBD 특성).
8. 복호는 six-iteration min-sum 알고리즘 사용(turbo 대비 복호 복잡도 낮음).
9. 결과: redundancy 9.57% 부호가 RS 연접(rate 0.9043)을, redundancy 12.59% 부호(rate 0.888)가 redundancy 28% block turbo를 능가.
10. 한계: HW 미설계, 광통신 채널(ASE noise, fiber nonlinearity, dispersion) 특화 시뮬레이션이며 부호 length·gate count 등 구체 HW 수치 없음.
