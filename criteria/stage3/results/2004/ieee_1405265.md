### B. 알고리즘 요약

1. 대상: cycle code(모든 bit node의 degree가 2인 LDPC 특수 클래스)의 반복 message-passing 복호 성능 특성화, 채널은 memoryless.
2. 문제: 반복 디코더는 Tanner graph의 유한 cover 위 codeword(pseudo-codeword)를 구분 못 해 복호 실패 → pseudo-codeword 집합의 기하학적 특성화 필요.
3. 모델: parity-check matrix `H` → Tanner graph `T(H)`, degree-2 bit node를 단일 edge로 축약한 normal graph `N(H)` 도입.
4. 핵심개념1: fundamental cone `K(H)` = LP 완화 제약을 만족하는 벡터 집합, 반복 복호 거동을 (cycle code에서 정확히) 특성화.
5. 핵심개념2: pseudo-codeword = 유한 M-cover의 codeword를 각 bit의 M copy 평균으로 사영한 유리벡터 `w(E)`, scaling 불변(pseudo-weight 불변).
6. 핵심기법: edge zeta function `ζ_N(u1..un)` = normal graph의 backtrackless·tailless·primitive cycle을 모두 열거하는 멱급수, `ζ_N^-1 = det(I - M(N)U)`로 유리함수화.
7. 주요정리(Th.V.4): monomial `u1^p1...un^pn`이 `ζ_N`에서 nonzero 계수를 가짐 ⟺ 지수벡터 `p`가 C의 pseudo-codeword.
8. 따름정리(Cor.V.5): `ζ_N`의 Newton polytope(급수 지수의 볼록포)가 부호 C의 fundamental cone `K(H)`와 정확히 일치.
9. 결과: cycle code에 대해 pseudo-codeword 열거 = zeta function의 cycle 열거로 환원, Code A/B 두 예제로 예시(수치 성능평가 없음).
10. 한계: 순수 대수/그래프 이론, HW·시뮬 없음, 일반 LDPC로의 zeta function 존재는 추측(conjecture)에 그침.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| cycle code / pseudo-codeword 이론 | 대응 없음 | Prime ECC는 degree-17 고rate QC-LDPC, degree-2 전용 cycle code와 구조 무관 |
| fundamental cone / LP 복호 분석 | 대응 없음 | Prime ECC는 min-sum 반복 복호, LP 디코딩·zeta function 도구 미보유 |
| edge zeta function | 대응 없음 | 알고리즘/HW로 이식할 계산 절차 아님 |

적용 가치: **낮음** — cycle code에 한정된 순수 대수적 pseudo-codeword 이론으로 실용 부호·복호기·HW 기여가 없고 Prime ECC 구조와 접점 없음.
