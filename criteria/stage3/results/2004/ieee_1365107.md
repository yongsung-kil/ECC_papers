### B. 알고리즘 요약

1. 대상은 irregular LDPC 부호(차수분포 `λ(x), ρ(x)`), Tanner graph 기반, error-floor 개선이 목표.
2. 문제: random 구성 LDPC는 작은 stopping set으로 AWGN에서 error-floor가 높아 이를 낮추는 구성법의 이론적 보증이 필요.
3. 핵심 개념: ACE(Approximate Cycle EMD) — 길이 `2d` 사이클의 값 `Σ(di−2)` (`di`=사이클 내 i번째 변수노드 차수).
4. 구성(Tian et al.[1]): 입력 `(dACE, η)`로 크기 `≤2dACE`인 모든 사이클이 ACE값 `≥η`를 갖도록 보장하고, degree-2 노드만의 사이클은 금지.
5. 분석 방법: Random 앙상블 `G(n,λ,ρ)`와 ACE-제약 앙상블 `G_{dACE,η}(n,λ,ρ)`의 stopping set 기댓값을 생성함수(generating function) coeff 추출로 비교.
6. Lemma 1: 크기 `l` stopping set은 그 유도 부분그래프에 길이 `≤min(2l,2k)` 사이클을 최소 하나 포함.
7. 핵심 식: ACE 준수 시 `2dACE` 미만 stopping set의 edge 수가 `≥2l+η`로 강제됨 → 작은 stopping set 억제.
8. Theorem 2: `l>dACE`에서 `ES_RS[l]−ES_ASCE[l]≥0` (ACE 앙상블 stopping set 기댓값이 항상 더 적음), 합산은 낮은 `e`가 지배.
9. 결과: 크기 `≤dACE` stopping set 기댓값은 `η`에 대해 지수적으로 감소, ~`dc·dACE/2`까지 감소 보장. 단, cycle 수 자체는 random과 큰 차 없음.
10. 통찰: "저차수 변수노드로 이뤄진 사이클"이 AWGN 성능저하의 주범.
11. 한계: HW 미설계, 시뮬 없음(수치는 상한의 numerical 계산뿐), BEC/WER 중심 분석.

| 사이클/생성함수 기댓값 계산 | 대응 없음 | 순수 앙상블 이론, 코드 대응 없음 |
