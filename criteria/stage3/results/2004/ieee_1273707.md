> 총평: efficiently-encodable(IRA 확장) high-rate·low-error-floor 부호설계로 NAND ECC 관심사(고rate·낮은 error floor·저복잡도 인코딩)에 부합하나, irregular 구조라 QC-LDPC 이식에 재설계가 필요하고 초록만 확보됨.

### B. 알고리즘 요약 (초록 기반, 본문 없음)

1. 대상: moderate length(`10^3`~`10^4`)·high rate(`≥3/4`)의 새로운 irregular LDPC 부호 클래스 설계.
2. 문제: 기존 irregular LDPC 설계는 고rate에서 error floor가 높고 인코딩 복잡도가 크다.
3. 핵심1: 이 부호 클래스는 **low-complexity(linear-time) 인코딩**을 허용한다(systematic serial turbo/IRA 등가 구조).
4. 핵심2: **다른 irregular LDPC 설계보다 낮은 error-rate floor**를 가진다.
5. 등가성: 제안 부호가 systematic serial turbo code 클래스와 등가이며, **irregular repeat-accumulate(IRA) 부호의 확장**임을 보인다.
6. 설계 알고리즘: **density evolution + differential evolution 최적화 + 수정된 cost function** 조합으로 degree 분포/부호를 설계.
7. 수정 cost function은 threshold뿐 아니라 error floor를 함께 억제하도록 설계된 것으로 보인다(초록 근거).
8. 결과: moderate-length·high-rate 부호를 매우 낮은 BER(초록 상 `10^-?`, OCR 손상으로 정확값 `미상`)까지 error floor 없이 제시(시뮬레이션).
9. 일반성: 초점은 moderate-length·high-rate지만, 다른 length/rate의 irregular LDPC에도 적용 가능하다고 주장.
10. 한계: 추출본이 forthcoming 초록 페이지라 HW·gate count·throughput·정확 degree분포·BER 수치 부재.
