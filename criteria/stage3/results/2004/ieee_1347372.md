### B. 알고리즘 요약

1. 시스템: BEC(binary erasure channel)에서 irregular LDPC를 ML로 복호, degree 분포 `(λ,ρ)` Tanner graph로 정의된 앙상블.
2. 문제: iterative 복호는 stopping set에 걸려 ML threshold까지 못 감 - iterative와 ML 사이 threshold gap을 사용자 조절 비용으로 메우고자 함.
3. 모델: ML 복호 = erasure된 변수에 대한 GF(2) 희소 선형계 `H_K x_K^T = z^T` 풀이 (unique 해 존재 ⟺ H_K 열 독립).
4. 핵심 기법: iterative 삼각화(triangulation)가 멈추면 `βN`개 reference variable을 선언, 나머지 미지 bit을 reference의 affine 조합으로 표현 (structured Gaussian elimination 아이디어).
5. 핵심 식: straightforward 복잡도 `((1-R)+Δ')·γ·2·N^3`에서 reference 도입 시 `((1-R-δ+β)+Δ')·γ·2·N^3`로 상수 축소 (계수 최소한 `(δ/β)^2`배 개선).
6. 확장: reference 선택법 3가지(A: 사전 random, B: 삼각화 정지시마다 random, C: [18]의 greedy 벡터 `ω` 기반) 제시, `β_A ≥ β_B`, 각 방법 typical `β`를 미분방정식으로 유도.
7. 트릭: residual graph degree 분포 진화를 미분방정식 `dλ̃_i/dt`, `dρ̃_i/dt`, `dΛ_0/dt`로 추적해 필요 reference 수 `β`를 해석적으로 계산.
8. 최적화: 사용자가 `β`(reference 비율) 조절 → 성능 향상 vs 계산량 tradeoff 조정, β=0이면 표준 iterative와 동일.
9. 결과: (3,6) regular, ε=0.47에서 Method A는 naive 대비 최소 131배, Method B 286배, Method C 397배 효율; Method C로 β=0.01 허용시 threshold가 0.429→0.45로 향상.
10. 한계: HW 미설계, 순수 이론(complexity 분석)이며 BEC 소거채널 전용이라 AWGN/NAND의 bit-flip·soft-read 상황엔 부적용.
