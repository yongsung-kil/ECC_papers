### B. 알고리즘 요약

1. 목표: Tanner graph의 girth가 알려진(4-cycle 없는) LDPC 부호를 **명시적(비랜덤)** 으로 구성 - 구현·성질 규명 목적.
2. 구성원: 소수 거듭제곱 `q`에 대해 Lazebnik-Ustimenko `k-regular` bipartite graph `D(k,q)`(2q^k vertex)의 incidence matrix를 parity-check로 삼아 이진 부호 `LU(k,q)` 정의.
3. girth 보장: `D(2,q)`는 girth 6·diameter 4, `D(3,q)`는 girth 8·diameter 6 → BP 디코딩은 girth가 클수록 유리(diameter 작으면 빠름).
4. 핵심 성질: 임의 두 행(열)이 최대 한 열(행)에서만 공통 1을 가짐(Prop.1) → 4-cycle 부재; 이로부터 차원·최소중량 유도.
5. stopping set: BEC 반복복호 성능 지표인 최소 stopping set weight를 `LU(2,q)`는 ≥ q+2(짝수 q에서 정확히 q+2), `LU(3,q)`는 ≥ 2q로 하한 증명.
6. LU(3,q) 파라미터: 홀수 소수 `q`에 대해 `[q^3, ...]` 부호로 rank 상한을 Vandermonde/순환행렬 rank 보조정리(Lemma 2~7)로 규명, 차원 conjecture 제시.
7. partial row construction: `H(k,q)`의 앞부분 행만 사용해 girth 유지·rate 상승(대신 최소중량 하락 가능)하는 새 기법으로 흥미로운 부호 다수 발굴(Table II/III).
8. array/finite-geometry 관계: `q`가 소수면 LU 부호는 array code, 짝수 q면 Euclidean geometry code와 동치(단 PCM은 non-cyclic).
9. 결과: `[64,37,10]`, `[343,132]`, `[512,230]`, `[729,296]`, `[1331,560]` 등 부호를 AWGN에서 BP(최대500 iter)/OSD1/MLD로 시뮬, 동일길이 random 부호 대비 WER/BER 우수(구성적 이점).
10. 한계: 순수 대수/그래프 이론 구성법, 저rate·짧은 length, `k≥4`는 disconnected로 활용난이, HW/디코더 arch/양자화 전무.

| array code / 순환 permutation block PCM | 대응 없음 | LU 부호 PCM은 non-cyclic·저rate로 Prime ECC의 고rate binary QC-LDPC와 구조 불일치 |
| BP/OSD1/MLD 디코딩 시뮬 | 대응 없음 | Prime ECC는 min-sum HW 디코더, OSD/MLD·full-BP는 미채택 |

적용 가치: **낮음** — 대girth·4-cycle-free 부호구성의 순수 이론 성과이나, 저rate·짧은 length·HW 부재로 NAND 고rate QC-LDPC ECC에는 직접 이식 실익이 없다.
