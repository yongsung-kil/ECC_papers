### B. 알고리즘 요약

1. 대상: 5G polar 코드의 SCL(successive cancellation list) 디코더(list size 8, CRC-aided fast simplified SCL), 부가로 5G LDPC min-sum 디코더(§V case study, (1088,352)·(2720,880), 각 15/50 iteration).
2. 문제: 기존 non-uniform quantization(NUQ)은 설계자의 경험적 매핑함수 설계에 의존해 자동화된 설계 방법론이 없음.
3. 핵심 가정: 큰 크기의 LLR은 드물고 값 민감도가 낮은 반면, 작은 크기의 LLR은 성능에 큰 영향을 미친다는 관찰에 기반해 양자화 스텝이 입력 크기에 따라 단조 비감소하도록 설계.
4. 핵심 기법: M-bit 양자화에서 `2^M`개의 스텝 크기 `s`를 개별 파라미터 `q`(스텝 증분, `q_i = s_i - s_{i-1}`, `q_i ≥ 0`)로 재정의해 NUQ 설계를 비볼록(non-convex) 파라미터 최적화 문제로 변환.
5. 핵심 식: `min F = FER(q), s.t. q_i ≥ 0` — FER(frame error rate)을 목적함수로 하는 정수계획(IP) 문제(식 5). Gaussian Process 대리모델 `F(q) ~ N(μ,Σ)`(식 6, squared exponential kernel)로 목적함수를 근사.
6. 확장: acquisition function(PI/EI/LCB, 매 iteration 균등 확률로 택1)으로 다음 샘플 `q*`를 결정해 반복 탐색(Algorithm 1, iteration 수 1000).
7. 구현 트릭: 채널 입력 LLR의 수동 스케일링 불필요(파라미터화 자체가 이를 흡수), 산출된 매핑을 룩업테이블(LUT)화해 덧셈/뺄셈(`|X|+|Y|`, `||X|-|Y||`) 연산을 LUT 또는 Karnaugh map 기반 combinational logic으로 대체.
8. 최적화 절차: BayesOpt(GP 대리모델 + acquisition function) 기반 블랙박스 최적화, 목적함수 평가는 실제 SCL/min-sum 디코더 실행(FER 측정, 최대 오류 프레임 100개).
9. 결과: (512,410) polar 코드에서 UQ 대비 bitwidth 20%·BOPs 19.5% 절감, floating-point 근접 성능(FER 10^-3에서 UQ-4 대비 최대 0.26dB 이득). LDPC min-sum case에서 (2720,880) 코드 BANQ-5가 UQ-6과 동등 성능이며 UQ 대비 LLR bitwidth 16.7% 절감, 동일 5bit에서 기존 기법[19] 대비 0.10dB 개선.
10. 한계: LDPC 검증은 case study 1건뿐이며 offset/normalized min-sum으로의 확장은 future work로 명시, HW 합성·gatecount·throughput 수치 없음(LUT/조합논리 구현 가능성만 언급), 채널은 AWGN 가정으로 NAND 채널 특성(RBER 등) 미고려.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BayesOpt 기반 non-uniform LLR 양자화 자동 설계 | `channel.cpp` `Set_LLR_Th()`, `ecc_data.h` `PARAM_LLR` | LLR threshold 테이블을 수동 설계 대신 BayesOpt로 자동 탐색하는 방법론을 iteration별 LLR 파라미터 최적화에 적용 가능 |
| min-sum 덧셈/뺄셈 LUT·조합논리 구현 | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | min-sum CN 연산(min1/min2, 부호 XOR)에 대한 양자화 매핑을 LUT/조합논리로 구현하는 아이디어가 유사 |
| SCL polar 디코더(PM 계산, 트리서치, node 기반 단순화) | 대응 없음 | Prime ECC는 QC-LDPC이며 SCL/polar 특유의 path metric·트리서치 구조 없음 |

적용 가치: **중간** — LDPC min-sum 디코더에 대한 실증(§V)이 있어 Prime ECC의 LLR 양자화 테이블 자동 설계에 방법론적으로 이식 가능하나, 검증이 제한적(case study 1건)이고 HW 구현·gatecount 근거가 부족함.
