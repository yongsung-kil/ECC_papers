### B. 알고리즘 요약

1. 대상은 2단계 계층 QC-LDPC(H-QC), core matrix(m×n)를 q1×q1 순환치환행렬로 1차 확장 후 다시 q2×q2로 2차 확장, N=n×q1×q2=12288, M=m×q1×q2=6144, rate≈0.5(m=3,n=6,q1=64,q2=32).
2. 문제의식: 순수 QC-LDPC는 긴 코드길이에서 coding gain 저하·error-floor가 발생하고, 기존 partially-parallel 디코더는 CFU(row)/BFU(column) 연산이 교대로만 동작해 절반씩 유휴.
3. H-QC는 modified PEG 알고리즘으로 2단계 확장 행렬을 구성해 무작위 구성 LDPC 수준의 coding gain을 확보하면서 HW 친화적 QC 구조를 유지.
4. 핵심 기법 1(scheduling): jump-reset 알고리즘으로 CFU/BFU가 반복(iteration) 경계를 넘어 겹쳐 실행되도록 주소를 재배열(`CFU-jump`, `BFU-jump`, `CFU-reset`, `BFU-reset`), 제약조건 `W ≤ floor(q1/2)`.
5. 겹침 스케줄링으로 기존 파이프라인 무스케줄 대비 메모리 요구량을 절반으로 줄이면서도 functional unit 가동률 100% 달성.
6. 핵심 기법 2(2차 매트릭스): q2×q2 순환치환행렬을 프로그래머블 4-way rotator로 처리, 상이한 2차 행렬 종류를 4개로 최소화해 라우팅 복잡도 절감.
7. 부수 트릭: min-and-convert unit으로 2's complement→sign-magnitude 변환과 1차 minimum selection을 병렬화해 CFU 지연을 4-adder에서 3-adder로 단축, scaling min-sum 사용.
8. 최적화 절차: LDPC 코드 생성기가 최소 개수의 2차 순환행렬 종류를 갖는 코드 행렬을 탐색(coding gain 유지 조건 하).
9. 결과: FPGA(Altera Stratix II EP2S130, 96.26MHz) 298.10Mbps@15iteration, 55491 ALUT(52.33%), 메모리 307,200bits(4.5%). H-QC가 QC 대비 BER=10^-5에서 약 0.1dB 개선, HW 구현 손실은 0.05dB 미만.
10. 한계: 메모리 인터리빙은 시간 제약으로 FPGA에 미구현(향후 과제), throughput 배가는 향후 인터리빙 적용 시 기대치이며 실측치는 298Mbps 단일 경로 기준, H-QC 구성 자체가 특정 PEG 알고리즘·파라미터(q1=64,q2=32)에 종속.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| jump-reset overlapped scheduling (CFU/BFU 중첩) | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | Dual-Update(even/odd 교대) 스케줄과 유사 목적, 메모리 충돌 회피 기법으로 스케줄링 비교 참고 가능 |
| scaling min-sum + min-and-convert unit(지연 단축) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | CN 연산 지연 최적화 트릭으로 참고 가능 |
| H-QC 2단계 부호구조(PEG 기반) | `ecc_top.cpp` `Load_PCM()` | 특정 QC-LDPC 고정 구조라 이식 난이도 높음(부호설계 재검증 부담 큼) |
| 2차 순환치환행렬 rotator(4종 제한) 및 column-major 메모리 저장 | 디코더 HW 모델 (z=32, HCU, GT) | Prime ECC의 z_sb=32 병렬 구조와 개념적으로 유사하나 2-level 계층 구조는 대응 없음 |
| 메모리 인터리빙(even/odd 주소 분리) | `decoder.cpp` 관련 메모리 접근 로직 | 미구현 상태로 제안만 되어 있어 참고용 |

적용 가치: **중간** — HW 스케줄링(jump-reset)과 CFU 지연 최적화는 참고 가치가 있으나, 핵심 기여인 H-QC 부호구조는 Prime ECC의 고정 QC-LDPC 구조와 재설계 부담이 커 이식 난이도가 높음.
