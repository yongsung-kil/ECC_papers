### B. 알고리즘 요약

1. 시스템/부호: IEEE 802.16e(WiMax) 표준 irregular LDPC, 시뮬레이션 예시는 rate 1/2, length 576; base matrix로부터 실시간 parity check matrix 생성.
2. 문제: WiMax용 프로그래머블(재구성형) LDPC 디코더가 필요하며, 여러 디코딩 알고리즘(Min-Sum, LLR-BPA, Offset-BPA) 중 어느 것이 재구성형 코어에서 throughput 대비 유리한지 분석.
3. 핵심 가정: AWGN 채널, 표준 sum-product/min-sum 반복복호 프레임워크.
4. 핵심 기법 1단: check-node 메시지는 `φ(x) = -log(tanh(x/2))`를 사용하는 정식 SPA 대신 min-sum 근사(식 4)로 계산하고, Offset-BPA는 `LLR_civ ← sign(LLR_civ)·max(|LLR_civ|-β, 0)`(식 5, β=0.25)로 min-sum 출력에 오프셋 보정.
5. 핵심 식 의미: 오프셋 보정(β)은 min-sum의 과대추정(overestimation) 오차를 density evolution으로 얻은 상수로 깎아 SPA에 근접시키는 표준 보정.
6. 핵심 기법 2단: Reconfigurable instruction cell architecture(RA) 상에서 loop distribution(코드율별 서브루프 분리), if-문 제거 후 HW multiplexer 대체, 메모리 배열 4종 중 불필요 배열 제거로 memory access 축소.
7. 부수 트릭: 파이프라이닝(memory read/compute/write 3단), packed computation — 32bit cell에 8bit soft value 4개를 패킹해 한 번의 메모리 접근으로 4개 variable/check node 병렬 처리.
8. 학습/최적화 절차 없음(density evolution은 β 상수 도출에만 문헌 참조, 자체 최적화 없음).
9. 결과: 최적화 전 0.43Mbps(RA) → 최적화(loop distribution+HW mux+memory 축소+parallel+pipelining) 후 5.94Mbps → packed computation 추가로 최종 20Mbps(10 iteration 기준), StarCore DSP 대비 6.6배 속도.
10. 한계: ASIC/FPGA가 아닌 특정 reconfigurable DSP-like 코어(instruction cell architecture) 전용 최적화라 일반 ASIC 이식성 낮음, 정정성능(BER) 개선 기여 없이 기존 min-sum/offset-BPA 문헌값과 비교만 수행, gatecount/합성 결과 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Min-Sum + Offset 보정 | `decoder.cpp` `CNU_Update_New_Mag()` | Prime ECC가 이미 min-sum(min1/min2+sign XOR) 보유, offset 보정은 §3 "이미 보유" 항목과 중복 |
| soft 8비트 양자화(1sign+2int+3frac) | `channel.cpp` `Set_LLR_Th()` | Prime ECC의 2SD/3SD LLR 양자화 테이블과 유사 목적이나 표준적 수준 |
| packed computation(32bit에 8bit x4 패킹) | 대응 없음 | Prime ECC HW 모델은 z_sb=32 lifting 기반 row 병렬이라 SW instruction cell의 bit-packing 기법과 구조가 다름 |
| reconfigurable instruction cell 최적화 | 대응 없음 | Prime ECC는 고정 RTL 파이프라인(6-stage) 기반이라 재구성형 코어 최적화는 대응 없음 |

적용 가치: 낮음 | 사용된 min-sum/offset-BPA는 Prime ECC가 이미 보유한 표준 기법이며, 구현 플랫폼(reconfigurable DSP-like instruction cell)이 Prime ECC의 고정 RTL ASIC 구조와 근본적으로 달라 이식 실익이 없음.
