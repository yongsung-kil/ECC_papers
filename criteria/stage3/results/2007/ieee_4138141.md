> 총평: rate-compatible(가변 rate) LDPC를 위한 순수 부호 구성법(4-cycle 회피, 균일 row weight, full rank 확보) 논문으로, 고정 rate NAND binary QC-LDPC(Prime ECC 3.1)와 목적·구조가 달라 이식 가치가 낮음.

### B. 알고리즘 요약

1. AWGN 채널에서 ARQ 방식 패킷 통신을 위한 rate-compatible LDPC 부호 구성법을 다룸 (rate 8/16~8/19, 모체 부호 (1536,3,6)-regular).
2. 기존 extending 방식은 row와 column을 모두 추가해 저rate로 갈수록 block length가 길어지고 저장공간이 커지는 문제가 있음.
3. 본 논문은 column은 고정하고 row만 추가하는 semi-random 구성법을 제안 (block length 불변, 저장 부담 감소).
4. 핵심 제약: (1) 새 row 추가 시 길이-4 cycle이 생기지 않도록(기존 row와 1-component 공유 2개 이상 금지), (2) 새 row의 weight `ρ`를 모체 부호 row weight와 균일하게 유지(threshold 성능 위해), (3) 새 행렬이 full rank가 되도록 선형종속 row를 재생성.
5. 이론적 근거: girth와 minimum distance의 관계(Gallager, Tanner), Gaussian approximation 기반 stability 조건 `λ2 < λ2*`에서 균일 row weight가 좋은 threshold를 준다는 결론(Theorem 2)을 구성 알고리즘 설계에 사용.
6. Irregular 모체 부호의 경우 원 행렬 row weight 평균을 반올림한 정수를 새 row weight로 사용, 이후 절차는 regular case와 동일.
7. 알고리즘은 pseudocode로 제시(§4 "Algorithm for regular LDPC codes"): Δm개 새 row를 무작위 생성 후 4-cycle 검사, 선형독립성 검사 및 재생성 반복.
8. 학습/최적화 절차 없음 — 컴퓨터 서치(랜덤 생성 + 제약조건 검사) 기반.
9. 결과: (1536,3,6) 모체 부호에서 파생된 rate 8/16~8/19 부호들의 AWGN BER 시뮬레이션(Fig.1)에서 양호한 성능을 보였다고 서술(수치는 그림에만 있어 본문 서술 수치 없음, 미상).
10. 한계: HW 미설계, 시뮬레이션(AWGN)만 수행, degree distribution 최적화 없는 semi-random 기법이라 elegant한 degree distribution을 보장하지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| rate-compatible 저율 부호 구성(row 추가 기반) | 대응 없음 | Prime ECC는 고정 rate QC-LDPC(`ecc_top.cpp` `Load_PCM()`)이며 가변 rate ARQ 구성 개념 자체가 없음 |
| 4-cycle 회피, 균일 row weight 제약 | `ecc_top.cpp` `Load_PCM()` | H-matrix 구조 관련이나 QC-LDPC 특유의 circulant/protograph 구조가 아닌 순수 랜덤 구성이라 정합성 낮음 |
| 디코딩 알고리즘(표준 sum-product) | `decoder.cpp` `LDPC_Decoder()` | 교과서 수준 sum-product 서술뿐, 신규 기법 없음 |

> 적용 가치: 낮음 — non-QC 랜덤 구성 기반 rate-compatible ARQ용 논문으로, 고정 rate binary QC-LDPC ECC 이식 관점에서 재사용 가능한 요소가 없음.
