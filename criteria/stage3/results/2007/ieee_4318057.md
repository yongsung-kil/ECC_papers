### B. 알고리즘 요약 (10줄 내외)

1. 채널/시스템: 독립(memoryless) Rayleigh flat-fading 채널 `R = aX + N`(X는 이진 입력, a는 Rayleigh 분포 페이딩 계수, N은 AWGN), coherent detection과 완벽한 CSI 가정.
2. 문제: Feldman et al.의 LP(linear programming) 디코더는 BEC/BSC/AWGN에서는 pseudo-weight 개념으로 성능이 분석되어 있으나, Rayleigh flat-fading 채널에서의 동등 개념(pseudo-weight)은 미분석 상태.
3. 핵심 가정: LP 디코더는 코드워드의 convex hull 대신 완화된 polytope(Tanner graph의 stopping set에 대응하는 pseudo-codeword를 꼭짓점으로 포함)에서 최적화하며, 출력은 항상 pseudo-codeword.
4. 핵심 기법: all-zero 코드워드 전송 시, pseudo-codeword `w`로 오판정할 pairwise error probability(PEP) `Pr{0→w|r,a}`를 조건부/무조건부로 유도(식 3~5).
5. 핵심 결과 식: 무조건부 PEP는 점근적으로 `K(w)·(Es/No)^(-|χ(w)|)` 형태이며, χ(w)는 pseudo-codeword의 support set(비영 좌표 집합) — 즉 오류확률의 SNR 감쇠율이 Tanner graph의 최소 stopping set 크기로 결정됨.
6. 확장: pseudo-codeword의 비영 좌표 값이 서로 다른 개수(r_w, 서로 다른 값의 종류)에 따라 r_w=1(식 4), r_w=2(식 6), r_w≥3(일반화, 식 5)의 3가지 케이스로 나누어 닫힌 형태 점근식 유도.
7. 부수 트릭: 변수변환(z0,...,z_{r-1})으로 다중 서로 다른 진폭 값을 갖는 pseudo-codeword의 적분을 감마분포 형태로 단순화.
8. 검증 절차: (3,5)-regular (155,64) Tanner code에서 size≤21인 모든 stopping set을 알고리즘([14])으로 탐색, 최소 stopping set 크기 18(multiplicity 465) 확인.
9. 결과: min-sum(MS)/sum-product(SP) BP 디코딩과 adaptive LP 디코딩의 FER을 시뮬레이션 비교(최대 100회 반복, EbNo당 최대 200 프레임 오류), 최소 pseudo-codeword 기반 truncated union bound(TUB)가 고SNR에서 실제 FER과 근접함을 확인.
10. 한계: 이론/시뮬레이션 전용(HW 미설계), Rayleigh flat-fading이라는 무선통신 특화 채널 모델 전제이며 NAND flash의 채널 특성(AWGN/비대칭 erasure형)과는 근본적으로 다름.
