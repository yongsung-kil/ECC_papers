> 총평: min-sum을 FSR(shift-register) 기반으로 재구성해 상호연결 복잡도를 낮춘 실측 ASIC 사례로, 디코더 스케줄링/HW 구조 아이디어는 참고 가치가 있으나 특정 finite-affine-plane QC-LDPC 부호에 강하게 결합되어 있음.

### B. 알고리즘 요약

1. (4095, 3687+42i) 고rate QC-LDPC 코드(finite affine plane 기반), row weight 64, column weight 7~8인 semi-regular 구조.
2. 목표: 고rate·고throughput min-sum 디코더를 복잡한 processor-memory 상호연결 없이 저비용 ASIC으로 구현.
3. 채널모델: SDH 마이크로웨이브 라디오 시스템, 64/128-QAM 변조와 결합한 실측 BER 환경.
4. 핵심기법: bit-node/check-node 메시지를 각각 m-step feedback shift-register(FSR)에 저장하는 SRMS(shift-register-based min-sum) 구조. 메시지는 `Zw(u)`, `Gt(s)=(Lt(s),Mt(s),vt(s),qt(s))`의 4-tuple로 단순화.
5. `Lt(s)`=인접 bit-node 중 최소 magnitude, `Mt(s)`=두 번째 최소값, `vt(s)`=최소값 위치 — check-node가 min-sum 근사를 4-tuple만으로 전달하기 위한 압축 표현.
6. n개의 BNPU(bit-node processing unit)와 r개의 CNPU(check-node processing unit)가 update mode/output mode를 번갈아 수행, 한 이터레이션에 2m 클록 필요.
7. HW 단순화: LLR 채널 입력 5bit, 메시지 6bit, 정규화 상수 2 고정; `vt(s)`의 하위 비트 1개만 사용해 게이트 15% 절감(대신 0.1~0.2dB 성능 열화).
8. 별도 학습/최적화 절차 없음(코드 구성은 affine geometry 기반 대수적 설계, Theorem 1의 다항식 인코더로 시스템적 인코딩).
9. 결과: 693K gates@0.15um, 60.2MHz에서 155.52Mbps(STM-1), 730mW, BER 10^-11까지 뚜렷한 error-floor 미관측, 10회/100회 반복 성능차 0.2dB 미만.
10. 한계: 특정 (4095,3687) 계열 affine-plane 부호 전용 설계로 일반 부호에 대한 이식성 검증 없음, encoder 아키텍처 상세는 지면상 생략.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SRMS min-sum 디코딩 (Zw(u), Gt(s) 4-tuple) | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | min-sum 근사 자체는 이미 보유, FSR 기반 상호연결 단순화 아이디어만 참고 가치 |
| CN min-sum L/M/v 압축 표현 | `decoder.cpp` `C2V_Cal_New_Sgn()` | min1/min2 방식과 유사한 개념, 신규 기여도 낮음 |
| 정규화 상수(normalization=2) | `decoder.cpp` `CNU_Update_New_Mag()` | normalized min-sum 계수 설정, 이미 유사 기법 보유 |
| (4095,3687) affine-plane 부호구조 | `ecc_top.cpp` `Load_PCM()` | 특정 코드 고정 설계로 이식 난이도 높음 |
| 폴리노미얼 시스템 인코딩 | `encoder.cpp` `Generate_PCM_encoding()` | 회로 구조가 dual-diagonal과 상이, 대응 낮음 |

적용 가치: 낮음 | 이미 보유한 min-sum/정규화 기법과 중복되고, 부호설계는 특정 affine-plane 구조에 고정되어 이식 난이도가 높음.
