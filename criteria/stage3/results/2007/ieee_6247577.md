### B. 알고리즘 요약

1. 시스템은 rate-1/2 LDPC 코드와 recursive MSK(RCPE-MSK) 변조기를 직렬 연결(serial concatenation), 수신단에서 BCJR 기반 MSK 복조기와 LDPC 복호기 간 joint iterative decoding/demodulation 수행.
2. 문제: RCPE(recursive continuous phase encoder)는 메모리를 가져 복조기 출력 LLR 간 시간적 상관(error correlation)이 발생, 인터리버 없이 LDPC와 직결하면 성능 저하.
3. 채널 모델은 AWGN이나, RCPE-MSK를 거친 "가상 채널"은 memoryless가 아닌 correlated channel로 취급(2-연속/4-연속 오류가 지배적, 시뮬 Table 2로 실증).
4. 핵심 기법: RCPE와 LDPC의 Tanner graph를 하나의 joint graph로 결합해 분석, 인터리버 없이 사용 시 joint graph의 최단 루프(예: length-6 circle)가 성능을 좌우.
5. 표준 RA(repeat-accumulate) 구조는 variable node 최소거리가 1이라 correlated error에 취약 → M_RA(modified RA) 구조는 check equation 내 variable node 최소거리(`min|i_j - i_k|`)를 RCPE 수렴길이(구속장의 5~8배) 이상으로 확보.
6. 확장 없음(단일 기법).
7. 구현 디테일: 부호화 복잡도는 RA 계열 특성상 code length에 선형(인코더 자체는 단순 구조), 무작위 구성부의 variable node degree=3 고정.
8. 별도 학습/최적화 절차 없음(density evolution은 향후과제로 언급만).
9. 결과: M_RA(최소거리 3, 11)가 인터리버 사용 RA 모드보다 낮은 Eb/N0에서 오히려 더 우수한 BER을 보이나(Fig.9), error floor는 인터리버 모드보다 높음.
10. 한계: HW 미설계, 시뮬만 수행(N=2000, rate 1/2 한정), MSK 변조 특유의 joint graph 구조 전제라 일반 채널(NAND 등)에는 적용 불가.
