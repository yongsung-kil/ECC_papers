> 총평: 기존 부호의 Tanner 그래프에서 edge를 최소한으로 제거해 girth를 늘리는 off-chip 후처리 기법으로, 새 부호 구성법 없이 기존 QC-LDPC에 적용 가능한 저복잡도 girth 개선 방법.

### B. 알고리즘 요약

1. 대상: 이미 존재하는 (irregular 가능) LDPC 부호의 Tanner 그래프(bipartite graph)에서 girth(최단 사이클 길이)를 사후에 증가시키는 방법. 예시 부호는 QC-LDPC(255,510), rate 1/2, 두 개의 255x255 순환행렬로 구성.
2. 문제: 짧은 사이클은 BP/SPA 복호 시 메시지 독립성을 깨뜨려 성능을 저하시키며, 기존 Trellis 기반 girth conditioning([5])은 복잡도가 커서 긴 부호(N up to 3만)에 적용 곤란.
3. 핵심 가정: 표준 BP/SPA 반복복호, AWGN 채널.
4. 핵심 기법 1단: 각 variable node `s`를 루트로 하는 support tree `T_s`를 `g/2` 레벨까지 BFS로 탐색해, 레벨 `g/2`에서 노드가 두 번 나타나면 길이 `g` 사이클 존재로 판정(MBFS, 표준 BFS의 변형).
5. 핵심 식: 사이클을 이루는 edge를 최소로 제거하는 문제는 set-covering 문제로 정식화 — `min Σx_{i,j}` s.t. `Σx_{i,j}y_{i,j} ≥ 1 ∀cycle∈C_g` (NP-complete). 그리디 방식으로 각 edge의 등장 카운터 `K_{i,j}`가 최대인 edge부터 제거.
6. 핵심 기법 2단: girth conditioning을 반복 적용해 `g0 → g0+2 → g0+4 ...`로 순차 증가시키는 iterative 구조(G0→G1→G2→...).
7. 부수 트릭: H matrix 자체를 카운터 저장 공간으로 재사용해 별도 메모리 오버헤드 최소화(공간복잡도 `O((d_v d_c)^{g/2})` 분석, N=30000 부호도 처리 가능함을 입증).
8. 최적화 절차: 그리디 휴리스틱(정식 DE/GA 없음), 복잡도 분석으로 최대 girth 상한 결정.
9. 결과: girth 4→6 적용 시 약 0.5dB, 6→8 적용 시 약 1dB 코딩 이득; 단 8→10 확장 시 일부 Eb/N0 구간에서 오히려 성능 저하(row/column weight 분포가 저weight 쪽으로 치우쳐 구조 붕괴).
10. 한계: HW 미설계, off-chip 전처리 전용(온칩 실시간 미적용), 복호 반복횟수(maxit) 영향은 미미하다고만 정성적으로 언급되고 특정 수치의 iteration 감소 보고는 없음.
