### B. 알고리즘 요약 (10줄 내외)

1. 시스템: 정보열 `u=[u1..uK]`를 outer systematic LDGM 코드(rate 근사 1, `[I_K P1]`)로 인코딩 후 inner LDGM 코드로 재인코딩하는 직렬연결 구조(SCLDGM). 전체 rate `R=K/N`, Gray mapping으로 AWGN/Rayleigh fading 채널(perfect CSI) 전송.
2. 문제: 단일 LDGM 코드는 인코딩이 매우 단순하지만 error-floor가 높음(고rate에서만 낮음); outer 고rate 코드와 직렬연결해 error-floor를 낮추면서 저복잡도 인코딩 유지.
3. 핵심 가정: 채널은 BIAWGN(perfect CSI) 또는 완전 인터리빙된 Rayleigh fading, Gray mapping 사용 시 AWGN 최적 코드가 non-binary 변조·Rayleigh 채널에도 최적임을 가정.
4. 핵심 기법: 다층 factor graph(outer check `f1`, inner check `f2`, 변수노드 `u,p1,p2`)에 대해 Sum-Product Algorithm(SPA)으로 반복복호, check-to-variable `L_{c→v}=2·tanh^{-1}(Π tanh(L_{v'→c}/2))`.
5. 핵심 식 의미: 전형적 EXIT chart(두 곡선 거리 최소화) 대신, 다층 구조라 코드 파라미터(degree profile, `p=L1/(L1+L2)`)에 대해 mutual information 수렴 여부를 exhaustive search로 판정하는 EXIT function 기반 최적화.
6. 확장: BEC의 duality 속성을 이용해 check-to-variable mutual information을 SPC/REP 코드의 `J(·)` 함수로 근사 계산(`I_SPC≈1-J(√(d_f-1)·J^{-1}(1-I_m))`).
7. 부수 트릭: 짧은코드 실제 시뮬레이션 임계값이 EXIT 예측(무한길이 가정)보다 높게 나타나며, 정규(regular)/비정규(irregular) SCLDGM 성능 차이가 크지 않음(대조적으로 RA 코드는 차이가 큼).
8. 최적화 절차: 초기 `Eb/N0`에서 모든 degree profile 조합을 시뮬레이션해 수렴 여부 판정 후, `Eb/N0`를 점차 낮추며 마지막까지 수렴하는 코드를 최적으로 선정(convergence threshold 탐색).
9. 결과: R=1/2 optimized irregular SCLDGM이 non-optimized 대비 0.2dB 개선, capacity limit(0.19dB) 대비 threshold 0.36dB로 근접, IRA 코드(0.45dB)보다 우수.
10. 한계: HW 미설계, 순수 시뮬레이션(K=10000~100000의 매우 긴 블록 길이), AWGN/Rayleigh 무선 채널 전용 최적화이며 NAND read/write 채널 모델과 무관.
