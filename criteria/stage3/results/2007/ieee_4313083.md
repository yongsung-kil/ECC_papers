> 총평: LDPC가 아닌 RS/BCH 성분코드 기반 GLD/Tanner 그래프코드의 hard-decision 대수적 반복복호 초기 연구(threshold 해석)로, 소프트정보 없는 환경(광통신 등)을 겨냥한 것이라 NAND binary QC-LDPC 이식 관점에서 관련성이 낮음.

### B. 알고리즘 요약

1. 대상 코드는 RS 또는 BCH 성분코드로 구성된 불규칙 이분그래프 코드(GLD/Tanner code 유형), rate 0.5~0.95, RS(255,*) 또는 RS(127,*) 성분코드 사용, BEC/BSC 채널.
2. 풀려는 문제: soft information이 없는 고속 저장/광통신 환경(예 optical-electrical 변환 후)에서 belief propagation 대신 저복잡도 대수적(algebraic) bounded-distance 복호로 iterative 처리 성능을 규명.
3. 핵심 가정: 각 성분코드(RS/BCH)는 hard-decision bounded-distance decoding만 수행하며, undetected error(디코더가 오류를 못 찾고 오정정) 가능성을 명시적으로 고려.
4. 핵심 기법(BEC): check node가 자신의 정정능력보다 적은 erasure를 가지면 그 노드부터 순차 디코딩, 실패시 다른 성분코드가 먼저 정정해줄 때까지 대기하는 iterative bounded-distance 알고리즘.
5. 핵심 식: `p_{i+1} = p0·λ(φ·(1-f(p_i)))`, 여기서 `f^k(p_i)`는 RS/BCH 성분코드 k가 erasure를 못 고칠 확률 - decoding 성공조건은 `p0λ(φ·(1-f(x))) < x` (density evolution과 유사한 threshold 조건).
6. 확장(BSC): undetected error 처리 위해 message-passing 대신 check node degree별 밀도를 추적하는 차분방정식(difference equation) 시스템 `v(i,j,k;t)`, `ṽ(i,j,k,a,b,s;t)`으로 성공 fraction `h(t)`를 추적.
7. 부수 디테일: 분석 단순화를 위해 4-cycle 없는 그래프만 고려, 각 check/variable node를 1회만 처리(실제로는 반복 처리가 성능 개선 가능하다고 언급).
8. 학습/최적화 절차: 명시적 최적화 없음(성분코드 분포 최적화는 "future work"로 남김).
9. 결과: BEC에서 RS 2종 조합 threshold `ε*`가 동일 rate LDPC의 `(1-R)/(1+R)` 대비 열세; BSC에서는 고rate(R=0.85)일 때 `ε*`(0.0099~0.0105)가 BP threshold(0.0132)에 근접, 채널용량(0.0217) 대비도 상대적으로 근접.
10. 한계: 순수 threshold 해석(이론)이며 시뮬레이션·HW 구현 전무, LDPC 대비 근본적으로 열세인 hard-decision 대수적 복호 방식, error-floor 분석 없이 waterfall(threshold) 영역만 다룸.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| RS/BCH 성분코드 bounded-distance 대수적 복호 | 대응 없음 | Prime ECC는 min-sum 기반 BP LDPC 디코더로 대수적 hard-decision 디코더 구조가 없음 |
| GLD/Tanner code 부호구조 (비-LDPC 그래프코드) | 대응 없음, `ecc_top.cpp` `Load_PCM()`과 무관 | binary QC-LDPC가 아닌 RS/BCH 성분코드 그래프이므로 부호구조 자체가 다름 |
| BEC/BSC threshold density-evolution 유사 해석 | 대응 없음 | 이론적 threshold 계산이며 HW 디코더 파라미터(LLR 등)와 직접 연결되지 않음 |
```

적용 가치: `낮음` — LDPC가 아닌 RS/BCH 기반 대수적 그래프코드의 hard-decision 복호 이론 연구로 Prime ECC의 min-sum BP 디코더 구조와 접점이 없음.
