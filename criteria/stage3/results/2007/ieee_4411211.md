### B. 알고리즘 요약

1. 채널은 Binary Erasure Channel(BEC), erasure 확률 `ε`, irregular LDPC ensemble을 `(λ(x), ρ(x))` degree distribution으로 표현, rate `R=1-dv⁻¹/dc⁻¹`.
2. 풀려는 문제: 기존 방법(density evolution/EXIT chart 기반 최적화)은 variable node degree 종류가 많을수록 계산 비용이 크므로, optimization 없이 빠르게 near-optimal ensemble을 찾고자 함.
3. 핵심 가정: check node degree 분포 `ρ(x)`가 주어진 상태에서 convergence 조건 `ελ(1-ρ(1-x)) < x`(BEC 특유의 message-passing 수렴 조건)를 만족하는 `λ(x)`를 탐색.
4. 핵심 기법: `ρ⁻¹(1-x)`의 Taylor 전개 계수 `Ti`를 이용해 `λi < Ti/ε` 상한을 유도하고, 이를 등식으로 설정(`λi = Ti/ε`)해 Type A(전체 degree 사용) 앙상블을 결정론적으로 구성.
5. 핵심 식 `λi = Ti/ε, 2≤i≤N-1; λN=1-Σλi` (식 8) — 최적화 없이 낮은 차수 variable node에 우선적으로 확률을 배분해 rate를 근사 최대화하는 greedy closed-form 해.
6. 확장(Type B/Modified B): 구현상 제약으로 제한된 개수 `P`의 variable node degree만 허용할 때, 나머지는 최대차수 `N` 또는 더 작은 `Dv`에 몰아 여전히 수렴 보장되는 앙상블 구성(Proposition 1의 `Dv` 하한식).
7. 별도 HW 구현 트릭 없음(순수 수학적 구성, 양자화·근사 언급 없음).
8. 최적화 절차 대체가 논문의 핵심(= optimization-free 설계); Theorem 1~3으로 유일해 존재성과 수렴성을 증명.
9. 결과: rate 1/2, Dc=7, P=4 조건에서 설계한 Type MB 앙상블이 용량 상한 대비 약 99.9%(Dv=203, P=90 예시) 도달, 기존 최적화 코드(literature) 대비 threshold 근소한 차이로 근접.
10. 한계: BEC 전용 asymptotic(무한 blocklength) 분석뿐이며 BSC/AWGN 등 실제 채널로의 확장은 미래 과제로 명시, 유한장 코드 성능·HW 구현은 다루지 않음.
