### B. 알고리즘 요약

1. 대상 채널은 binary erasure channel(BEC); LDPC의 BP 디코딩 성공 여부는 Tanner graph의 stopping set(SS) 구조에 의해 결정된다는 기존 결과([1][2])를 배경으로 함.
2. 풀려는 문제: 동일한 코드(row space)를 정의하면서도 parity-check matrix에 linearly dependent(redundant) row를 추가하면 작은 크기의 stopping set 수를 줄일 수 있음(기존 [3][4][5] 연구) — 이 효과를 random matrix ensemble 관점에서 정량화.
3. 모델: `m x n` 이진 행렬의 redundant extension `H^(L)` — 원 행렬의 부분블록(크기 `L`)의 모든 비자명 선형결합을 row로 추가하여 row 수를 `(2^L-1)(m/L)`로 확장(row space는 불변).
4. 핵심 기법 1단: `L=2`인 경우 average SS weight distribution `S_w^{R(2)}`을 정확히 유도(Theorem 1, 식 (23)) — 두 row의 pairwise 선형결합까지 고려한 카운팅 `U`(Lemma 6, 식 (22)).
5. 핵심 식의 의미: SS weight distribution `S_w`은 weight `w`인 벡터가 stopping set이 될 (행렬 앙상블 평균) 개수이며, 값이 작을수록(특히 낮은 `w`에서 0에 가까울수록) BEC 하에서 BP 디코딩 실패 확률이 낮음을 의미.
6. 핵심 기법 2단: `L>2`(일반) 경우 정확한 식 유도가 어려워 상한/하한(Theorem 2, 식 (25)(26))으로 대체 — 상한은 `L→∞`에서도 상수배 내로 non-extended ensemble과 tight함을 보임.
7. 구현 디테일 없음(양자화·HW 근사 등 비대상) — 순수 조합론적 카운팅.
8. 최적화 절차 없음. 대신 asymptotic growth rate `α(R,μ)`(정규화 weight `ζ`, 정규화 extension degree `μ` 함수)의 상/하한을 Corollary 1/2로 유도, `ζ>μ`에서 tight함을 증명.
9. 결과: `n=100, m=50` 예제(Table I, Fig.2)에서 `L`(extension degree)이 커질수록 특정 weight `w`의 average SS distribution 상한이 감소(즉 stopping set 감소) — 단, row 수는 `L`에 지수적으로 증가하는 trade-off(Fig.2, Fig.3).
10. 한계: BEC 전용 분석이며 AWGN/soft-decision 채널이나 실제 코드 성능(BER) 결과 없음, HW/디코더 구현 논의 전무, redundant row 추가에 따른 디코딩 복잡도 증가(row 수 지수 증가)가 실용성의 주요 제약.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| redundant row 추가를 통한 stopping set 감소 | 대응 없음 | BEC 전용 이론이며 Prime ECC는 AWGN/RBER 채널의 min-sum QC-LDPC로 stopping set이 아닌 trapping set/error-floor 관점이 주 관심사 |
| average SS weight distribution 상/하한 (random matrix ensemble) | 대응 없음 | 비구조적 random ensemble 분석으로 QC-LDPC(`PCM_b`) 고정 구조와 직접 대응 없음 |
| H-matrix redundant extension 개념 | `ecc_top.cpp` `Load_PCM()` | 개념적으로만 H-matrix 로드 단계와 연관되나 실질적 이식 근거 부족 |

> 적용 가치: 낮음 — BEC 채널 전용 stopping-set 이론이며 Prime ECC가 다루는 binary QC-LDPC의 AWGN/RBER 채널·min-sum 디코더·HW 구조와 접점이 없다.
