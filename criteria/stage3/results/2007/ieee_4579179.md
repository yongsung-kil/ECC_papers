### B. 알고리즘 요약

1. AWGN 채널에서 sum-product 반복복호(최대 100회)로 동작하는 irregular LDPC 코드 설계, code rate 약 1/2, block length 211~541.
2. 목표: 낮은 저장/구현 복잡도를 가지면서 girth-6(4-cycle 없음)를 보장하는 구조적(structured) irregular LDPC 코드 설계.
3. 채널 모델은 표준 AWGN, 별도 NAND/저장매체 특화 가정 없음.
4. 핵심 기법: `(n,w,1)`-BIBD의 base block `B`를 `s`번 순환 시프트하여 `H = [B0; B1; ...; Bs-1]`을 구성 (식 9). `n = w(w-1)r+1`, rate `≈(n-s)/n`.
5. base block 하나만 저장하면 순환 시프트로 전체 H를 유도할 수 있어 디코더 메모리 요구량이 크게 줄어든다는 것이 핵심 식의 의미.
6. base block은 Wilson의 두 가지 구성법(홀수 w는 construction 1, 짝수 w는 construction 2, 원시근 기반 집합 `S_i`/`T_i`)으로 생성.
7. 4-cycle 회피를 위해 임의 두 열의 공통 1 개수가 최대 1이 되도록 시프트 값을 신중히 선택(all-zero column 회피 포함).
8. 별도 학습/최적화 절차 없음 (조합론적 파라미터 탐색으로 base block 선택).
9. 결과: N=541, rate 1/2 code1이 Eb/No=2.5dB에서 BER 2.79e-4로 동일 조건 랜덤(MacKay) 코드(BER 7.13e-4) 및 규칙 BIBD-LDPC(BER 4.34e-4) 대비 우수.
10. 한계: 소블록 길이(211~541)에서만 검증, AWGN 이론적 시뮬뿐이며 HW 구현/양자화/복잡도 분석 전무.
