### B. 알고리즘 요약

1. 시스템: irregular LDPC 부호 앙상블, BSC/BAWGN 채널, block length N=1000~100000, rate 1/2 (3,6)-regular 및 irregular 예시.
2. 문제: 기존 LDPC 인코딩은 brute-force 시 O(N²), Richardson-Urbanke(RU)의 ALT 변환은 임의 부호 프로파일에 대해 선형시간을 보장하지 않고 gap이 커질 수 있음.
3. 핵심 가정: parity-check matrix를 approximately lower triangular(ALT) 형태(Fig.1, gap `G`)로 놓고, 앙상블 자체를 이 구조만 갖도록 제약.
4. 핵심 기법: 임의의 row/column degree 분포에 ALT 제약(variable node `i=1..L-G`가 정확히 하나의 hardwired edge로 check node `G+i`에 연결, 특정 범위 check node에는 연결 금지)을 부여한 `C_ALT(λ,ρ,g)` 앙상블 정의, RU의 사후 순열 없이 애초에 ALT 구조를 갖는 부호만 샘플링.
5. 핵심 식: gap `G=gN`이 충분히 작으면(`G=O(√N)`) 인코딩 복잡도가 `O(N+G²)`로 선형이 됨; Theorem 1/2로 ALT 앙상블의 점근 threshold와 tree assumption이 표준 앙상블과 동일함을 증명.
6. 확장: gap `G=0`인 특수 케이스로 완전 lower-triangular인 LT-LDPC 앙상블 정의, RU의 역행렬 계산·전처리 단계 자체가 불필요.
7. 구현 디테일: 인코딩은 parity bit를 `L`번째부터 첫 번째까지 재귀적으로 계산(계산량 N에 선형), bit 쌍(2i,2i-1) 단위로 병렬 계산 가능(m-tuple로 확장 시 더 높은 병렬도).
8. 최적화 절차: 그래프 생성은 socket matching(변수 노드 순차 스캔, 무작위 매칭) 알고리즘으로 O(N) 시간에 구성, 매칭 실패 시 재시도.
9. 결과: gap `G=√N`인 ALT 앙상블이 N=1000~100000에서 표준(제약없는) 앙상블과 BER 성능이 거의 동일(Fig.3~11), G=10 정도의 작은 gap에서도 성능 유사; graph construction 실패확률은 gap이 작아도 낮음(Table I).
10. 한계: 순수 이론/시뮬 논문으로 HW 구현 없음, LT 앙상블(G=0)은 저인덱스 변수노드 주변 사이클 변동성이 커서 BER 그래프가 불안정, 정정능력 자체의 개선이 아니라 표준 앙상블과 동등 성능 유지가 목표.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| ALT 앙상블 구성(선형시간 인코딩용 부호 제약) | `ecc_top.cpp` `Load_PCM()` | H-matrix 구조를 ALT 형태로 새로 설계해야 함, 이미 고정 QC-LDPC를 보유해 재설계 부담 큼 |
| 선형시간 recursive 인코딩 + m-tuple 병렬화 | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | 이미 dual-diagonal 구조로 선형시간 인코딩 보유 - 중복 기여, ALT 방식 도입 실익 낮음 |
| gap G 크기와 성능/복잡도 트레이드오프 이론 | `encoder.cpp` `Column_Reordering_Dual_Update()` | 우리 dual-diagonal도 유사한 구조적 트레이드오프 개념이나 이 논문의 이론적 분석은 직접 적용 대상 아님 |
```

적용 가치: `낮음` - Prime ECC는 이미 dual-diagonal 인코더로 선형시간 인코딩을 확보했으며, 이 논문은 정정능력 개선이 아닌 인코딩 복잡도 이론(RU 대안)에 국한되어 중복 기여.
