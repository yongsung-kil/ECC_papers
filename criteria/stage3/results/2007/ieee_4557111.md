> 총평: LDPC/turbo 디코더 분석 기법(DE, EXIT chart)을 binary가 아닌 일반 exponential family 기반 Expectation Propagation으로 확장한 통계적 추론 이론 논문으로, NAND ECC와 직접 관련 없음.

### B. 알고리즘 요약

1. 배경: LDPC/turbo 반복복호(BP)는 Expectation Propagation(EP)이라는 더 큰 분산 근사 베이지안 추론 알고리즘군의 특수 사례(binary/Gaussian 등)이며, EP는 Kalman filter, forward-backward 등도 포함.
2. 문제: 기존 DE/EXIT chart 이론은 binary LDPC/turbo 코드에 한정되어 있었는데, 이를 임의의 exponential family(이산, Poisson, beta, Gaussian 등) 파라미터를 갖는 homogeneous factor graph의 EP로 일반화.
3. 모델: factor graph는 degree-1 factor(사전정보)와 degree `df` factor(관측/제약)로 구성된 homogeneous 랜덤 정규 이분그래프.
4. 핵심 기법: 메시지 밀도 `q_k`의 반복(식 (4))을 `As.1`(factor node 국소 선형화, Weak Gaussian Approximation)로 근사하면 Gaussian-to-Gaussian 맵 `G_∞(μ,Σ)`(식 (5))로 차원을 축소해 스칼라/저차원 동역학으로 분석 가능.
5. 더 강한 가정 `As.2`(Strong Gaussian Message Distribution)는 메시지를 가상 AWGN 채널 `y = θ + n(0, σ²I)`의 사후분포로 취급, 메시지 밀도를 표준편차 `σ` 하나로 특정하여 상호정보량 `I(σ)`와 결합.
6. 이 가정 하에 EXIT 함수 `T0`(파라미터 노드), `T1`(factor 노드)을 정의해 `I_{k+1} = T1∘T0(I_k)` 형태의 1차원 EXIT chart로 수렴/성능(고정점 교차) 예측.
7. 응용 예: 센서 네트워크의 랜덤 sleep 스케줄 기반 분산 데이터 퓨전 문제를 homogeneous factor graph(`df=2`)로 모델링, 상관계수 `ρ=.999`의 Gaussian 관측 모델에서 EXIT chart로 MSE 수렴을 예측.
8. 학습/최적화 절차: 별도 최적화 없음. Monte Carlo 시뮬레이션으로 EXIT 함수 `T0`, `T1`을 계산/검증.
9. 결과: 네트워크 크기 `S > 100000`일 때 DE/EXIT 예측이 실제 EP 동작(수렴 및 점근 MSE ≈ -35dB)과 일치함을 그래프로 확인 (수치 예시일 뿐 LDPC 성능 지표 아님).
10. 한계: 순수 이론/시뮬레이션 논문으로 LDPC 부호 설계나 디코더 HW 요소 전무. 예시도 센서 네트워크 응용이며 binary NAND LDPC와 무관.
