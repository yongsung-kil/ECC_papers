### B. 알고리즘 요약

1. GF(q) 유한체의 원소를 location vector(곱셈군 또는 덧셈군 기준)로 표현하여 circulant permutation matrix로 dispersion.
2. 목표는 4-cycle(girth<6) 없는 QC-LDPC 코드를 체계적(대수적)으로 구성해 기존 computer-search 기반 random LDPC 대비 낮은 error-floor를 달성하는 것.
3. 채널 모델은 AWGN(BPSK), binary random erasure channel(BEC), burst-erasure channel 세 가지.
4. 핵심 기법: `α`(원시원소)의 `M`-location vector로 만든 base matrix `B`를 RC-constraint(row-column constraint)를 만족하도록 vertical/horizontal expansion하여 `H` 어레이 생성. RC constraint 만족 시 Tanner graph girth ≥ 6 보장(Theorem 1).
5. RC constraint: 두 행(또는 열)이 두 곳 이상에서 동시에 1을 갖지 않음 → 4-cycle 자동 배제, 최소거리 하한 `2(γ+1)` 보장.
6. 확장 기법으로 Method-1(곱셈군), Method-2(RS 부호 기반), 덧셈군 기반(§VII) 세 가지 base matrix 구성법 제시.
7. Masking(§VIII): base array `B`에 masking matrix `Z`(원하는 degree distribution)를 Hadamard-곱 방식으로 곱해 밀도를 낮추고 irregular 코드 생성, degree-1 variable node를 상위 degree로 재분배해 error-floor 개선.
8. Burst-erasure 채널용으로 zero-covering span 개념 도입, 이 값으로 erasure-burst 정정 능력 하한을 이론적으로 증명(Theorem 5).
9. 결과: (961,721) rate-0.75 코드가 Shannon limit 대비 2.3dB(BER 1e-6), 다양한 예제에서 random MacKay code 대비 동등~우수한 error-floor 성능, AR4JA/turbo 코드 대비도 우수.
10. 한계: HW 구현·게이트카운트 없음, 순수 알고리즘/시뮬레이션 수준이며 SPA(sum-product, full-precision) 위주로 min-sum/양자화 등 HW 친화적 변형은 다루지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QC-LDPC H-matrix(base+circulant) 구성 | `ecc_top.cpp` `Load_PCM()` | 부호 구성 방법론이 다르나 QC-LDPC base+shift 구조 자체는 Prime ECC의 `PCM_b`와 동일 개념 |
| Masking을 통한 irregular degree distribution 설계 | `ecc_top.cpp` `Load_PCM()` | H-matrix 설계 시 degree distribution 참고 가능하나 특정 코드 교체는 재검증 부담 큼 |
| SPA(sum-product) 디코딩 | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum 사용, 본 논문은 full SPA 기준 성능 곡선이라 직접 비교 어려움 |
| Burst-erasure 정정(zero-covering span) | 대응 없음 | NAND 채널은 random bit-error 위주로 burst-erasure 모델과 무관 |

> 적용 가치: 낮음 — 순수 대수적 QC-LDPC 구성법·masking 이론이며 HW/디코더 최적화 요소가 없어 Prime ECC 이식 대상으로는 참고 수준에 그침.
