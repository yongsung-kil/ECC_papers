> 총평: MIMO V-BLAST 무선통신 시스템에서 density evolution 기반 적응 LDPC 부호설계로 채널용량에 근접하는 통신이론 논문으로, NAND 저장 채널·binary LDPC ECC HW 이식과는 무관함.

### B. 알고리즘 요약

1. 2x2 V-BLAST형 MIMO 시스템, 각 송신 안테나별 독립 LDPC 부호화(rate 0.2~0.25 mother code), M-QAM 적응변조, optimum successive detection(OSD) 채택.
2. 기존 V-BLAST는 등전력/등rate 할당이라 MIMO 용량의 일부만 달성; per-antenna rate/power adaptation이 필요하나 실제 채널코드 설계 방법이 미해결 문제였음.
3. 채널 모델: block flat-fading MIMO 채널 `Y = HX + N`, i.i.d. 채널 어댑터로 등가 이진입력 채널의 대칭성을 강제해 density evolution(DE) 적용 가능하게 함.
4. 핵심 기법: 각 채널실현·SNR마다 DE로 목표 BER(10^-5) 만족하는 최대달성 rate와 degree distribution `(λ,ρ)`를 도출해 LDPC 부호 설계.
5. 핵심 식: 등가 채널의 LAPPR 초기밀도를 Monte Carlo로 구해 DE 입력으로 사용 (비-Gaussian이라 Gaussian approximation 미적용, `Q(y|x)` 수치적으로 획득).
6. 확장: 설계 복잡도를 줄이기 위해 안테나마다 별도 최적 코드 대신 하나의 mother code(rate 0.25/0.20)에서 rate-compatible puncturing으로 다양한 rate 생성.
7. 부수 트릭: OSD의 결정지연/오류전파 문제를 완화하기 위해 parallel interference cancellation(PIC) + MMSE 필터링으로 대체, 첫 반복에서 지연된 복조(delayed demodulation)로 수렴 가속(3~4회 반복으로 목표 BER 도달).
8. 최적화 절차: puncturing 비율 `p_i`는 Gaussian approximation 기반 convergence 조건으로 SNR threshold를 최대화하도록 최적화.
9. 결과: 설계된 적응변조 LDPC 시스템의 sum rate가 V-BLAST용량(rate+power adaptation)에 근접; punctured code는 최적 code 대비 약간의 rate 손실.
10. 한계: 무선 MIMO 페이딩 채널·표준 sum-product 디코더 전제, HW 설계 전무, 순수 시뮬레이션 기반 통신이론/부호설계 연구이며 NAND 저장 채널과 무관.
