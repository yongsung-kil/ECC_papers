### B. 알고리즘 요약

1. Type-II QC-LDPC는 weight-0/1/2 circulant(각 $p \times p$)로 구성된 parity-check matrix `H` (index `L`, length `N=pL`)이며, type-I(monomial만)는 그 부분집합.
2. 문제의식: type-I QC-LDPC(순열 circulant만)는 girth-6 보장 시 최소거리·rate 범위가 제한적이므로, weight-2(binomial) circulant를 허용해 더 넓은 rate·최소거리를 얻고자 함.
3. 별도 채널/noise 모델 없음 — 순수 대수적 코드 구성 문제(Tanner graph girth, rank).
4. 핵심 기법: cycle 구조를 circulant shift 값의 합 `Σ (p^(i_2k)_{j_k,l_k} - p^(i_2k+1)_{j_k,l_k+1}) ≡ 0 mod p` 조건(식 (3),(4))으로 정식화, 이를 만족하지 않도록 shift 값 `p^(1)_{j,l}=a_l+jlf`, `p^(2)_{j,l}=a_l+jlf+d_{j,l}` 를 공식으로 직접 지정(Construction, §IV-A).
5. 핵심 정리(Theorem 6): 4-cycle이 없으려면 `d_{j0,l0} ≠ d_{j1,l0}`, `d_{j0,l0} ≠ ±d_{j0,l1}` 등 4가지 부등식이 성립해야 하며, 이를 만족하도록 `f > 2d+1`, `p > (J-1)(L-1)f + 2d`로 파라미터를 잡으면 girth≥6 보장.
6. 확장: 동일 공식을 regular/irregular weight configuration(일부 위치의 weight-2 항 생략)에도 적용 가능하도록 일반화.
7. 부수 결과(§V, Theorem 10): multi-generator(J>1) QC 행렬의 fullrank 조건을 `gcd(Δ_1,...,Δ_{(L choose J)}, x^p-1)=1` (J×J submatrix 행렬식들의 gcd)로 제시, design rate `1-J/L` 달성 여부 판정.
8. 최적화/탐색 절차 없음 — 컴퓨터 girth 탐색을 대체하는 것이 이 논문의 핵심 주장(공식만으로 결정).
9. 결과: 예제로 (3,4)-regular type-II 코드 `[N=88, k=22]`(rate 1/4, p=22)와 (6,8)-regular type-I 코드 `[N=252, k≥63]`(girth≥6)를 구체 구성. Table I/II에 최소 `p` 값 표.
10. 한계: BER/디코딩 성능 시뮬레이션 전무, HW 설계 없음, 예제 code length가 88~252로 매우 작아 NAND ECC급 고rate/긴 길이로의 확장성은 미검증.
