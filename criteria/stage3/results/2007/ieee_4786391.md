### B. 알고리즘 요약

1. 시스템은 SC-FDE(단일 반송파 with cyclic prefix) 전송 방식, BPSK 변조, 랜덤 구성 (3,6) regular LDPC 부호(rate 1/2, N=1024, K=512), cycle-4 회피 구성.
2. 기존 임펄스 노이즈 채널용 LDPC 디코딩([8])은 사전 LLR 계산에 무한급수가 필요해 계산 복잡도가 높다는 문제를 해결하려 함.
3. 채널모델은 Middleton Class A 노이즈: 배경 가우시안 잡음 `n_g`와 파라미터 p=Λ인 Bernoulli 변수 `b`로 스위칭되는 임펄스 잡음 `n_i`의 합 `y_k = x_k + n_g,k + b_k·n_i,k`.
4. 핵심 기법은 Poisson 프로세스 기반의 정확한 Class A LLR을 Bernoulli 분포로 근사하여 사전 LLR `L_c(y_i)`를 무한급수 없이 폐형식으로 계산.
5. 식 (6): `L_c(y_i) = ln[((1-Λ)exp(-(y-1)²/2σ_G²) + (Λ/σ)exp(-(y-1)²/2(σ_G²+σ²/Λ))) / (같은 형태, y+1)]` — 배경잡음과 임펄스잡음 두 항의 가중합으로 LLR을 단순화.
6. 확장으로 CP(cyclic prefix) 구간을 수신단에서 버리지 않고 MRC(Maximal Ratio Combining)로 결합하여 다이버시티 이득을 얻는 CP-combining 기법을 함께 제시, 결합신호에 대한 수정 LLR(식 8) 유도.
7. 부수 구현: 표준 LLR-BP(sum-product 계열) 반복복호, 최대 반복횟수 20회 고정.
8. 별도 학습/최적화 절차 없음(폐형식 근사식 유도만).
9. 결과: Λ=0.1 소규모 임펄스일 때 제안 방법이 기존 [8] 방법과 유사한 BER을 보이며, coded 시스템에서 저 Eb/No 영역은 CP-combining이 유리하나 고 Eb/No에서는 CP 유무 차이가 미미(수치 곡선은 그림 제시, 본문 수치 없음).
10. 한계: HW 설계 없음, 순수 시뮬레이션 결과만 제시, 임펄스 노이즈(전력선/무선) 채널 특화 가정으로 NAND flash 채널과 근본적으로 다름.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 임펄스 노이즈 채널 LLR 근사 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | NAND read 채널이 AWGN/RBER 기반이라 임펄스 노이즈 모델 자체는 대응 없음 |
| LLR-BP 반복복호 | `decoder.cpp` `LDPC_Decoder()` | 표준 sum-product 계열로 이미 보유한 min-sum과 차별점 없음 |
| CP combining (MRC) | 대응 없음 | 물리계층 다이버시티 기법으로 ECC 코어와 무관 |

> 적용 가치: 낮음 — 임펄스 노이즈 채널 특화 LLR 근사와 CP combining은 NAND flash read 채널 모델과 무관하고, 사용된 LDPC 자체도 랜덤 regular 구성으로 부호설계 기여가 없음.
