### B. 알고리즘 요약

1. Rate 1/2 BI-AWGN 채널 대상 irregular LDPC 코드의 variable/check node degree distribution λ(x), ρ(x) 설계 문제.
2. 기존 Gaussian Approximation 기반 설계는 정확도가 떨어지고, 기존 최적화(GA 등)보다 더 단순하고 우수한 분포를 찾고자 함.
3. 채널 모델은 binary-input AWGN(BI-AWGN), all-zero codeword 전송 가정.
4. 분석 기법은 Discrete Fast Density Evolution(FDE): `Lˆ = log tanh(L/2)` 변환으로 check node 연산을 특성함수(Laplace 변환) 곱셈으로 단순화 (식 8~11).
5. 비용함수 `Δδ = δ* − δ`(δ*=Butman-McEliece Shannon limit, δ=FDE로 구한 threshold)를 최소화하도록 λ, ρ를 최적화.
6. 최적화 알고리즘은 PSO: 각 입자(후보 λ 또는 ρ 벡터)의 속도/위치를 `v_{k+1}=wv_k+c1r1(pbest−x_k)+c2r2(gbest−x_k)`로 갱신 (c1=c2=2, w는 반복횟수에 따라 선형 감소).
7. λ(x), ρ(x)는 확률벡터 제약(합=1)을 만족해야 하며, λ 최적화 후 고정한 뒤 ρ를 재최적화하는 2단계 절차.
8. 학습 절차: FDE로 각 입자의 threshold 평가 → pbest/gbest 갱신 → 반복(step 2~7), dv=4,8,10,12,15,20 각각에 대해 별도 탐색.
9. 결과: dv=20일 때 기존 설계[12,13] 대비 threshold 0.1142dB 개선(δ=0.9658), FDE의 Gaussian Approximation 대비 정확도 향상.
10. 한계: 이론/시뮬레이션 결과만 존재하며 디코더 구조, HW, 정정성능(BER 곡선) 실측 없음. 순수 코드 구성(degree distribution 설계) 논문.
