### B. 알고리즘 요약

1. 이진 대칭 채널(BSC) 상의 irregular LDPC 부호 앙상블(길이 `n≈10^4`, rate 0.5 예시), 4-cycle 없는 Tanner graph 가정.
2. 문제: 채널 파라미터(crossover probability)를 정확히 모를 때 기존 switch-type 알고리즘(Gallager B, `G`)은 성능이 크게 저하되는데, 이를 개선하는 강건한 hard-decision 복호를 찾는 것.
3. 채널 모델은 memoryless BSC, 메시지는 `{-1,0,+1}` 알파벳의 majority-based(MB) 알고리즘으로 구성.
4. 핵심 기법: variable node degree 및 node 종류별로 서로 다른 order의 MB 알고리즘(`MB_ω`)을 반복 내내 고정 배정하는 time-invariant hybrid 방식(`ρ̃`벡터로 각 degree의 알고리즘 order 조합 비율 표현).
5. 핵심 식: `ε_(l+1) = f(ε_l, ε)` 형태의 density evolution 재귀(식 (3)~(5))로 오류확률 진화를 추적, threshold `ε*`는 이 곡선이 `y=x` 선과 만나는 첫 교차점.
6. 부수 기법: MB 알고리즘의 threshold 상한을 tight하게 유도(Theorem 1)하고, 오류확률의 점근적 수렴속도가 지수적 또는 초지수적(super-exponential)임을 증명(Theorem 2).
7. 최적화 절차: degree distribution `(λ,ρ)`와 MB order 배정 `ρ̃`을 MATLAB `fseminf`로 공동 최적화, 주어진 채널 파라미터에서 rate를 최대화하는 non-linear 제약 최적화 수행.
8. 결과: rate 0.5에서 최적 MB 앙상블이 Gallager A(`ε*≈0.5`대비) threshold를 능가, 유한장(n=10^4) 시뮬레이션에서 채널 미지 상황 시 MB가 최적화된 Gallager B 대비 넓은 BER/채널파라미터 범위에서 더 빠른 수렴을 보임(예: 오류율 `10^-6`에서 평균 bit error/word error가 MB 2.38 vs G 29.99/225.13).
9. iteration 측면: 동일 최대 반복수(`ℓ=10,20,50,500`) 제약 하에서 MB가 낮은 `ℓ`에서도 Gallager A/B보다 우수한 BER/WER 달성 - 사실상 더 적은 반복으로 목표 BER 도달(iteration 감소 효과).
10. 한계: 순수 BSC(hard-1bit) 채널·density evolution 기반 점근적 분석과 소규모 유한장 시뮬레이션뿐이며 HW 구현, 양자화, 게이트/throughput 설계는 전혀 다루지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| MB(majority-based) hybrid hard-decision 복호 알고리즘 | `decoder.cpp` `LDPC_Decoder()`, `VN_Cal_HD()` | Prime ECC의 HD(hard-decision) 모드와 개념적으로 유사한 카테고리이나, min-sum(soft) 위주 구조에서 MB 방식으로 교체 시 성능/복잡도 이득이 불명확 |
| degree별 알고리즘 order 고정 배정(time-invariant hybrid) | `decoder.cpp` `Get_VNU_Table_Idx()` (iteration별 LLR 테이블 참조 로직과 유사한 위치) | node 종류별 다른 규칙 적용이라는 아이디어는 LLR threshold 테이블 설계에 참고 가능하나 실제 대응 함수는 없음 |
| Gallager A/B 비교, DE 기반 threshold 최적화 | 대응 없음 | Prime ECC는 특정 H-matrix 고정이라 degree distribution 최적화 자체가 적용되지 않음 |

적용 가치: 낮음 - hard-decision(BSC) 전용 알고리즘·이론 분석 논문으로, Prime ECC의 soft min-sum 기반 HW와 직접 대응되는 요소가 제한적이며 NAND soft-read 환경과의 연계도 약함.
