> 총평: LP/iterative 디코딩에서 pseudocodeword weight를 최소 거리와 일치시키기 위한 parity-check 행렬 중복도(redundant row 수)의 이론적 상계를 다양한 부호군에 대해 유도한 순수 수학 논문으로, 특정 코드 구조·실험·HW 요소가 전혀 없어 Prime ECC 이식 가치는 낮음.

### B. 알고리즘 요약

1. 대상은 일반 이진 선형부호 `[n,k,d]`이며 특정 rate/length가 아닌 Hamming, tree-based/finite-geometry LDPC, Reed-Muller, Hadamard 등 여러 부호군을 이론적으로 다룸.
2. 풀려는 문제는 LP/iterative 디코딩에서 그래프(파리티체크 행렬) 표현에 따라 달라지는 pseudocodeword가 성능을 좌우하는데, 이를 최소화(즉 `Wmin(H) = d(C)`로 만드는)하는 데 필요한 최소 행 수(`pseudoweight redundancy`, φ(C))가 무엇인지 규명.
3. 핵심 가정은 binary symmetric channel(BSC)에서 정의된 pseudocodeword weight(§2.2)이며, 유한 커버(finite graph cover)에서 발생하는 pseudocodeword만을 다룸.
4. 핵심 개념 정의: `φ(C)` = `Wmin(H) = d(C)`를 만족하는 parity-check 행렬 H의 최소 행 수 (기존 stopping redundancy의 유사 개념).
5. 핵심 정리(Theorem 2.1): 쌍대부호(dual code)의 모든 0이 아닌 코드워드로 구성된 행렬 `H*`는 항상 `Wmin(H*) = d(C)`를 만족 — 즉 φ(C) ≤ 2^(n-k)-1이지만 실용적으로 과도하게 복잡.
6. 확장 기법: 부호 결합(구성법)에 대한 φ 상계 유도 — direct sum(Thm 4.1), (u,u) 구성(Thm 4.2), extended code(Thm 4.3) 등 컴포넌트 부호의 φ로부터 합성부호의 φ 상계를 재귀적으로 계산.
7. 부수 결과: tree-based/finite-geometry(EG/PG) LDPC 부호에 대해 특정 조건(p=2)에서 `φ(C) ≤ n`(블록길이) 상계 제시, Reed-Muller/Hadamard 부호에도 `{u,u+v}` 재귀구성 기반 상계(g(r,m)) 유도.
8. 학습/최적화 절차 없음 — 전부 조합론적 증명(귀납법, 부등식 전개)으로 상계를 도출.
9. 결과는 정량적 시뮬레이션 수치가 아닌 폐형 상계식(예: Hamming code `φ(C)=m`, extended Hamming `φ(C)≤2m`, Hadamard `φ ≤ (m-2)2^(m-1)+1`)로 제시.
10. 한계: 순수 이론(상계 유도)뿐이며 실제 반복복호/LP복호 시뮬레이션, HW, min-sum 알고리즘 변형 등이 전혀 없고, redundant row 추가에 따른 디코더 복잡도 증가 트레이드오프는 논의만 되고 정량화되지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| pseudoweight redundancy 상계 이론 | 대응 없음 | LP 디코딩 관점의 순수 이론으로 min-sum 기반 Prime ECC 디코더와 직접 대응 없음 |
| redundant row 추가를 통한 H-matrix 재구성 | `ecc_top.cpp` `Load_PCM()` | H-matrix 로드/구성 층위와 개념적으로 연결되나, 특정 QC-LDPC 구조에 이 redundant row 이론을 적용하는 방법은 논문에 없음 |
| finite-geometry(EG/PG) LDPC 부호 사례 | 대응 없음 | 특정 부호족 사례 분석일 뿐 Prime ECC의 QC-LDPC 구조와 직접 연결되지 않음 |

> 적용 가치: 낮음 — LP 디코딩 이론 틀에서 정의된 pseudoweight redundancy 상계 연구로, 실제 min-sum 반복복호 성능이나 QC-LDPC HW 설계와 무관한 순수 수학적 결과라 Prime ECC 이식 참고 가치가 낮음.
