### B. 알고리즘 요약

1. 시스템은 spread OFDM(BPSK, Hadamard-Walsh 확산, N=1024 부반송파), 부호는 rate 1/2 irregular LDPC(길이 약 1024~1000).
2. 채널은 주파수 상관(frequency-correlated) Rayleigh 페이딩(Hilly Terrain, Typical Urban COST-207 모델)이며, 기존 연구는 비상관 채널 최적화만 다뤄 상관 채널에 대한 사전 성능 예측이 부재.
3. 채널 모델: 부반송파 간 페이딩 상관계수 `ψ_i,j = (1+|i-j|Δf/(Δf)c)^-2`, coherence bandwidth `(Δf)c`로 상관도 결정.
4. 핵심 기법 1단: discretized density evolution으로 잡음 임계값(σ) 산출 후 differential evolution으로 최적 degree distribution `λ(x), ρ(x)` 탐색, eIRA와 PEG 두 구성법으로 코드 생성.
5. Girth conditioning(6000개 후보 코드 중 평균 girth 최대인 코드 선택, eIRA 평균 girth 7.59 vs PEG 8.11)으로 short-length 코드의 error floor를 완화.
6. 핵심 기법 2단: 부반송파당 오류 비트 수 `K_f{OFDM}` 분포를 유도(`μ_Kf = N(1-e^{-ρ²})`)해, 상관 채널의 `K_f` 분포가 비상관 채널과 유사할수록 비상관용 최적화 코드가 상관 채널에서도 근사 최적임을 사전 예측.
7. 부수 트릭: Hadamard-Walsh 확산을 채널 코딩 위에 추가해 다이버시티를 확보, 이를 통해 eIRA 코드의 error floor를 별도 girth conditioning 없이도 낮춤(spreading이 대안적 error-floor 완화 수단).
8. 최적화 절차: density evolution(임계값) + differential evolution(degree distribution) + girth 탐색(코드 인스턴스 선택), 총 2단계 파이프라인.
9. 결과: PEG 구성 코드가 비확산 대비 BER 10^-5에서 0.5dB 개선, eIRA 코드는 SNR 5~6dB 이상에서 error floor(비확산 시 BER ~10^-3, 확산 시 ~10^-4로 개선) 발생.
10. 한계: 무선 페이딩 채널(AWGN 아님) 전용, HW 설계 전혀 없음(순수 알고리즘/이론+시뮬), NAND/저장매체와 무관한 목표 채널.
