### B. 알고리즘 요약

1. 대상은 유한길이(N=500~10000) regular/irregular LDPC 및 systematic IRA(irregular repeat-accumulate) 코드, AWGN/fading 채널, 3GPP HSDPA 4-MCS 적용.
2. 문제의식: 무한길이 LDPC는 density evolution으로 최적화 가능하나, 유한길이에서는 waterfall region이 완만해지고 error floor가 발생해 성능 예측이 어려움.
3. 채널/모델 가정: extrinsic message가 symmetric Gaussian(또는 유한길이에서 Gaussian-mixture)로 근사됨을 시뮬레이션으로 검증.
4. 핵심 기법 1단: extrinsic information evolution chart를 확장해 transfer characteristic band(TCB)를 정의, 두 TCB의 교차점(demarcation point 't')에서 codeword error probability `Pw ≈ 1-(1-Pv)(1-Pc)` 추정.
5. 식 (14)/(23)의 tangent point `m0`은 LDPC ensemble profile에만 의존하는 고정점으로, Pw를 SNR별로 빠르게 추정하는 데 사용(Monte Carlo 없이).
6. 핵심 기법 2단(코드설계): IRA 구조 `H=[P1,P2]`에서 degree-2 변수노드를 특별히 보호하기 위해 differential girth `gd`를 도입, 타입별 cycle(2/3)의 girth를 g와 g+gd로 차등 부여.
7. 부수 트릭: modified bit-filling algorithm으로 확장집합 `Ū = U ∪ Λ`을 정의해 4-cycle 회피 및 girth 제어를 재귀적으로 수행(파라미터: N,K,w,v,g,gd).
8. 최적화 절차: (w,g,gd) 파라미터 조합을 exhaustive search로 비교, density evolution으로 얻은 degree profile λ(x)=λ1x^0+λ2x^1+λw*x^(w-1) 기반.
9. 결과: (N=532,K=266) 등에서 최적화 IRA 코드가 regular LDPC 대비 FER 10^-6~10^0 구간에서 우수, encoding 복잡도도 낮음. (N=8192) 4-MCS 조합은 ergodic capacity 대비 2.4~2.8dB 이내.
10. 한계: 순수 시뮬레이션 기반 분석/설계로 HW 구현 없음, 채널은 AWGN/fading(무선) 한정, error floor를 낮추려면 column weight w를 늘려야 하나 저SNR 성능과 trade-off 존재.
