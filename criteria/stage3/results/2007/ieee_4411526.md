### B. 알고리즘 요약

1. protograph 기반 LDPC/MET(multi-edge-type) 코드 앙상블에 대해 AWGN 및 BEC 채널에서의 반복복호 수렴 임계값을 평가.
2. 기존 EXIT chart는 degree distribution의 평균만 다뤄 degree-1 변수노드, punctured 변수노드, 동일 degree distribution이지만 다른 edge 연결을 가진 protograph들을 구분하지 못함.
3. 채널모델: AWGN(Gaussian approximation, `J(σ)` 함수)과 BEC(정확 해석 가능)를 모두 다룸.
4. 핵심기법: protograph의 개별 edge마다 mutual information `IEv(i,j)`, `IEc(i,j)`를 추적하는 PEXIT(Protograph EXIT) 알고리즘. base matrix `B`(원소 `b_i,j`=edge 개수)로 그래프 구조를 표현.
5. 핵심 식 `IEv(i,j) = J(Σ b_{s,j}[J^{-1}(IAv(s,j))]^2 + (b_{i,j}-1)[J^{-1}(IAv(i,j))]^2 + [J^{-1}(Ich_j)]^2)` — 노드별이 아닌 edge별 MI 갱신으로 protograph 구조 특이성을 반영.
6. APP-LLR mutual information `IAPP(j)`을 매 iteration마다 계산해 1로 수렴하는지로 임계값(threshold, AWGN은 `(Eb/N0)*`, BEC는 `p*`) 판정.
7. 부수 트릭: BEC는 (12)(13)의 정확식 사용, AWGN은 duality property로 check-node EXIT 함수 근사, `J(·)`/`J^-1(·)`는 다항식 근사식 활용.
8. 별도 학습절차는 없으나 differential evolution과 결합해 protograph ensemble threshold를 최적화하는 코드 설계 파이프라인에 활용 가능하다고 언급.
9. 결과: AR3A 코드 앙상블(rate 7/8~1/8)에서 실제 DE 임계값 대비 오차 0.002~0.041dB, MET/CTC 코드에서도 0.007~0.026dB 수준의 높은 정확도 확인.
10. 한계: 순수 앙상블 임계값(무한 block length 가정) 분석이며 실제 유한장 코드 구현, 디코더 HW, 양자화 등은 전혀 다루지 않음.
