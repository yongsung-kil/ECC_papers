> 총평: 디코더나 부호를 개선하는 논문이 아니라, trapping set을 이용해 매우 낮은 error floor(10^-14)를 FPGA 하이브리드 시뮬레이션으로 저비용 추정하는 분석 방법론 논문 - NAND ECC 검증/평가 단계에 참고 가치는 있으나 직접 이식 대상은 아님.

### B. 알고리즘 요약

1. 대상은 자기기록 데이터 저장 채널의 partial response(PR, 예: EPR4) 채널 + AWGN, rate 8/9, length-4623 QC-LDPC 코드(soft-output Viterbi 검출기 LLR 입력).
2. 문제: LDPC의 error floor는 FER 10^-12~10^-15까지 요구되나, direct Monte-Carlo 시뮬레이션(FPGA 포함)으로는 이 정도로 낮은 영역을 현실적 시간 내 관측할 수 없음.
3. 핵심 가정: error floor 영역의 decoding failure는 거의 전적으로 trapping set(반복 복호의 fixed point가 되는 소수 변수노드 집합)에 의해 발생하며, 서로 다른 trapping set 발생은 독립(동시발생 무시 가능)이라 가정.
4. 1단계로 FPGA 하드웨어 시뮬레이션을 고SNR에서 실행해 decoding failure의 error symptom(fixed point 변수노드 집합)을 모아 trapping set 목록 `T_k`를 탐색.
5. 각 trapping set에 대해 노이즈 벡터를 저차원 파라미터 `S`(4L차원 노이즈 G를 특정 방향으로 투영한 값, `S ~ N(0, σ²/4L)`)로 축약해 `Pr{ξ_Tk|S}`만 시뮬레이션하면 되도록 근사(`Pr{ξTk} ≈ Σ_S Pr{n∉G}Pr{ξTk|S}·(가우시안 밀도)`).
6. in-situ 시뮬레이션: 각 S값 조건에서만 FPGA로 conditional decoding failure rate를 구해 전체 계산량을 direct MC 대비 대폭 절감.
7. 전체 FER는 개별 trapping set들의 실패확률 합(`FER = Σ_k Pr{ξTk}`)으로 근사.
8. 별도의 부호 최적화/학습 절차는 없음(방법론은 순수 분석·추정).
9. 결과: length-4623 rate 8/9 QC-LDPC, EPR4 채널에서 60 iteration 기준 추정치가 direct 시뮬레이션 대비 0.3dB 이내로 일치, FER 10^-14까지 수시간 내 추정 가능(직접 시뮬로는 도달 불가능한 영역).
10. 한계: 새로운 복호 알고리즘이나 HW 설계 제안이 아니라 기존 디코더의 error floor를 추정하는 분석기법이며, PR채널(자기기록) 전용 노이즈 모델(EPR4 4-D Gaussian projection) 가정이라 NAND의 이산적 오류모델에는 재유도가 필요함.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Trapping set 탐색(고SNR FPGA 시뮬레이션) | `corner.cpp` (수렴실패 추적) | 개념적으로 유사한 목적(디코딩 실패 케이스 추적)이나 논문은 별도 trapping-set 파라미터화 방법론 제공 |
| in-situ 시뮬레이션 기반 FER 추정 | `ecc_top.cpp` (시뮬 오케스트레이션, 통계) | 낮은 FER 영역을 저비용으로 추정하는 통계기법으로 시뮬레이션 오케스트레이션에 방법론적으로 결합 가능 |
| soft-output Viterbi 기반 LLR 입력 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | PR 채널 특유의 LLR 생성이라 NAND RBER/fixed-error 채널과는 직접 대응 어려움 |
| 반복복호 자체(min-sum 등) | `decoder.cpp` `LDPC_Decoder()` | 표준 iterative decoding 가정만 사용, 알고리즘 변경 없음 |

> 적용 가치: 중간 — 새 디코더/부호 기법은 아니지만, error floor를 저비용으로 정밀 추정하는 방법론은 Prime ECC의 매우 낮은 FER 검증(코너케이스 추적, corner.cpp)에 통계적 가속 기법으로 참고할 가치가 있음. 다만 PR채널 특유의 노이즈 모델 재유도가 필요해 즉시 이식은 어려움.
