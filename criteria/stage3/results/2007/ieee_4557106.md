> 총평: NAND ECC 이식과 무관한 generalized LDPC/protograph 코드의 minimum distance 점근적 상한에 관한 순수 이론(조합론) 논문.

### B. 알고리즘 요약

1. 대상: check node가 single parity-check code에 대응되는 generalized LDPC 코드(변수노드 degree 1, non-transmitted 변수노드 포함) 및 protograph 기반 코드.
2. 문제: 어떤 조건에서 코드 패밀리가 asymptotically good(최소거리가 길이에 선형)이 아닌지 필요조건을 찾는 것이 목표.
3. degree-2 변수노드들이 이루는 그래프 `W2`(각 check node가 정점, 동일한 degree-2 변수노드에 인접한 두 check node를 잇는 edge)를 정의.
4. `W2`의 평균 degree `A > 2`이면 girth 상한(Moore bound)을 이용해 `dmin < 2 log_{A-1}(2m+1)` (Proposition 2, 로그 상한).
5. `A = 2`(disjoint cycle, dangerous cycle)인 임계 케이스에서는 "potentially bad set" `X`(크기 `< K2`, defect 크기 `< K3`)가 `K1*n^t`개 이상 존재하면 `dmin < K4 * n^{K3/(K3+2K2+1)}` (Theorem 5, 다항식 상한).
6. 확장: multi-edge 코드(예: rate 1/2, degree-3 변수노드 O(n)개)에 적용 시 `dmin = O(n^(2/3))` 수준으로, 이 코드는 error floor(10^-5~10^-3)를 가짐을 보임.
7. protograph 예제(Fig.1, designed rate 1)에서는 potentially bad set 크기 4, defect 6으로 `dmin = O(n^(6/1))` 수준 sublinear.
8. 반례: protograph의 edge 하나를 살짝 바꾸면(Fig.3) blue degree-2 노드가 길이 2n의 cycle을 이룸에도, 평균 weight distribution 분석(Lemma 8~13, Stirling 근사 기반)으로 코드의 절반 이상이 `dmin > δn`(선형)임을 증명(Proposition 14).
9. 결과: dangerous cycle 존재가 asymptotically good 여부를 항상 결정짓지 않음을 보인 반례 제시가 핵심 성과.
10. 한계: 알고리즘/HW 설계 없음, 시뮬레이션·수치 실험 없이 순수 증명 논문. NAND ECC 실무 이식 요소 전무.
