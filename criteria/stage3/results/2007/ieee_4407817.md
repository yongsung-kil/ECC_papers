### B. 알고리즘 요약

1. 입력은 IEEE Trans. Communications 2007년 12월호 "Abstracts of Forthcoming Manuscripts" 페이지(ieee_4403033과 동일)로, LDPC 관련이 2편이다.
2. ① BIBD 기반 QC-LDPC 구성(Lan Lan, Ying Yu Tai, Shu Lin, Memari, Honary): 특수 클래스의 BIBD를 이용해 효율적으로 인코딩 가능한 QC-LDPC 부호를 구성하는 신규 방법.
3. 대상 채널은 AWGN과 이진 소거채널(BEC)이며, 반복복호(iterative decoding)로 양호한 성능을 보인다고 초록에 서술.
4. QC 구조 덕에 shift-register 기반 효율 인코딩이 가능한 것이 강조점(구체 rate/length/BER 수치는 초록에 없음, 미상).
5. ② 저 error-floor QC-Generalized-LDPC(Liva, Ryan, Chiani): protograph의 check node를 Hamming 등 강한 component code로 치환("LDPC code doping")하여 구조적 QC-G-LDPC를 설계.
6. QC 구조로 shift-register 기반 효율 인코딩 가능, 단순 component code 선택으로 빠른 반복복호에 적합.
7. 결과(초록): AWGN에서 error-floor와 waterfall 양 영역 모두 우수, 단 저rate QC-G-LDPC로 귀결.
8. 두 초록 모두 부호 구성/설계 수준이며 디코더 HW 구조·양자화·gate 수·throughput(Gbps) 등 구현 수치는 초록에 없음(미상).
9. ②의 G-LDPC는 component-code(Hamming APP) 복호를 전제해 Prime의 표준 binary min-sum 디코더와 복호 구조가 다름.
10. 한계: 둘 다 AWGN/BEC(무선·소거) 대상, NAND 고정 고rate binary QC-LDPC ECC와 rate·복호구조·검증부담 측면에서 이식성 낮음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BIBD 기반 QC-LDPC H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | 낮음 - Prime 고정 QC-LDPC 대체는 전면 재설계·재검증 필요 |
| QC 순환구조 효율 인코딩 | `encoder.cpp` `Generate_PCM_encoding()` | 낮음 - Prime은 dual-diagonal 인코딩 고정 |
| G-LDPC(Hamming component) 복호 | 대응 없음 | Prime은 표준 binary min-sum, component-code 복호 미지원 |

적용 가치: 낮음 — 부호족은 binary QC-LDPC로 Prime과 동일하나, 초록만이라 rate/length/성능 검증 불가하고 고정 부호 전면 교체 부담·상이한(G-LDPC) 복호구조로 실질 이식 요소 없음.
