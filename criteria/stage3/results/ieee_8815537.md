### B. 알고리즘 요약

1. 시스템: 무선(LTE/5G 맥락) Convolutional(rate 1/2, L=100)·Turbo(rate 1/3) 부호의 뉴럴 디코더에 메타러닝을 결합, 2-layer bi-GRU 200 units.
2. 문제: 뉴럴 디코더는 non-AWGN 채널에 강하나 새 채널 적응에 방대한 데이터·계산(fine-tune K=10000 step) 필요 → 실전 비현실적.
3. 모델: 각 채널 `Ti`를 하나의 task로 보고 AWGN/ATN(T분포)/Radar 채널을 archetypal task 집합으로 사용.
4. 핵심기법1(MAML): task update `θi=θ-α∇θ L_Ti(fθ)`로 채널별 적응 후, meta update `θ←θ-β∇θ ΣL_Ti(fθi)`로 민감한 초기가중치 학습, 손실은 BCE.
5. 식 의미: 좋은 초기화 `θ`를 학습해 meta-testing에서 pilot 소량·few gradient step만으로 최적 디코더에 근접.
6. 핵심기법2: FOMAML(1차 근사) — 2차 gradient를 상수취급해 계산량 절감, MTL(단순 gradient 평균, Eq.4)을 naive 벤치마크로 대비.
7. 구현 디테일: task update용/meta update용 데이터 분리(meta-overfitting 방지), Turbo는 반복구조 민감성으로 α를 더 작게(0.0001).
8. 학습/최적화: meta batch P=10, 50000 meta step, meta-testing MIND-1(pilot 100, K=1)·MIND-10(1000, K=10)로 fine-tune 대비 데이터·step 대폭 절감.
9. 결과: trained 채널에서 MIND-1이 static(MTL-0/1)·MIND-0 능가하며 채널전용 뉴럴/최적 디코더에 근접, untrained Radar에서도 gradient step 늘리면 개선.
10. 한계: Conv/Turbo NN 디코더 한정(LDPC 미포함, 확장은 future work), HW 미설계, 시뮬만, 여전히 상당량 샘플 필요.
