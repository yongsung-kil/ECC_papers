### B. 알고리즘 요약

1. 대상: 이진 cyclic/QC-LDPC(주로 FG-LDPC), rate 0.5~0.969, length 63~66048 예제(NASA (8176,7156) rate0.875 포함).
2. 문제: cyclic 부호의 인코딩(단일 shift-register) 장점과 QC의 병렬 디코딩/wire-routing 장점을 동시에 얻고, error-floor를 구조적으로 보장하고 싶음.
3. Part1 - 순환행렬 분해: `L×L` 순환행렬을 `L=cn` 인수분해 후 열/행 치환 `π`(식(2))로 `n×n` 크기 `c×c` 순환행렬 배열로 분해(doubly-cyclic 구조, 식(3)).
4. 핵심 식: type-1 descendant 부호의 생성다항식 근 판정 (Theorem 1, `g^(0)(X)` 근 조건), dual 부호가 root polynomial 선형결합(Lemma 1)임을 이용.
5. Part2 - FG 부호 분해: 2-D/고차원 Euclidean·Projective 기하 cyclic EG/PG-LDPC의 순환 PCM을 분해해 cyclic(type-1/2/3) 및 QC descendant를 대량 생성(masking으로 irregular도).
6. CPM 분해(Theorem 2/3): FG 순환행렬이 CPM/ZM 배열로 분해됨을 증명, column/row block splitting으로 밀도 낮춘 QC-EG-LDPC 확장.
7. Part3 - trapping set 분석: RC제약 `(γ,ρ)`-regular은 `|a|≤γ` VN 집합의 홀수차 CN 수가 `≥γ(γ-t)... ≥γ` → 크기<γ 유해 trapping set 부재 증명.
8. 핵심 결론: 최소거리 미만 trapping set이 없으면 error-floor가 최소거리로 지배됨 → FG/finite-field RC제약 부호는 낮은 error-floor 보장.
9. 결과: (65536,32768) 부호가 DVB-S2 대비 0.1dB 우수, error-floor 미관측; (32768,31747) rate0.969는 Shannon 0.6dB. SPA/scaled-MS 50-iter 시뮬((8176,7156)엔 별도 FPGA 디코더로 저BER 확인 언급).
10. 한계: 부호설계+trapping set 이론 중심, 신규 디코더 알고리즘·HW 아키텍처 설계 없음. hard-decision syndrome 기반 분석(연판정 무관).
