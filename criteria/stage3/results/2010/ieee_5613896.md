> 총평: 부호구조·girth·degree를 보존한 채 mapping만 바꿔 error-floor를 낮추는 부호설계 기법으로, NAND 고rate binary QC-LDPC의 error-floor 대책으로 개념적 가치가 크나 array-code 특유 대수구조 의존이라 Prime ECC H-matrix에 직접 적용은 재설계 부담이 있음.

### B. 알고리즘 요약

1. 대상: circulant permutation 기반 array-based/QC LDPC(고rate, 예 (2209,1978) rate≈0.895, bit degree r=5, check degree 47), sum-product/soft-xor 반복복호.
2. 문제: 반복복호가 그래프 cycle 위에서 non-codeword 고정점(absorbing set)에 빠져 저BER에서 error-floor 발생.
3. 모델: `(a,b) absorbing set` = 각 bit node가 unsatisfied check(O(D))보다 satisfied 이웃이 더 많아 bit-flip에 안정한 부분그래프; error-floor 지배.
4. 핵심: parity-check `H`를 mapping rule `f(i,j)=a(i)·j`로 정의, `a(i)` 선택을 바꿔 지배적 최소 absorbing set인 (4,8)의 존재조건 congruence 식을 위배시켜 제거.
5. (4,8) 존재 필요조건 식 (3),(4)(mod p congruence) 유도 → `a(i)`의 pairwise 차·곱이 서로 distinct하도록 잡으면(예 {0,1,3,8,19}, p>충분) 조건 불성립 → (4,8) 소멸(Lemma 2).
6. 파급효과: (4,8)이 큰 absorbing set의 부분구조이므로 이를 없애면 (6,8) 등 상위 spectrum도 감소(Lemma 3), 즉 spectrum 전체 개선.
7. 구조 보존: girth(=6), node degree, PCM 구조, rate 불변 → 기존 고throughput compact HW/디코더 재사용 가능(추가 복잡도 0).
8. 최적화: 조합론적 증명으로 a(i) 후보 도출, prime p 조건(예 p>17/23) 명시.
9. 결과: Q4.2 고정소수점, 200 iter, SPA/soft-xor 및 FPGA로 BER 10^-11까지 검증 — (4,8) 오류 완전 소거, error-floor 최소 1자릿수(order of magnitude) 개선.
10. 한계: regular array-based code 특정 사례(r=5)에 집중, irregular 확장은 future work; 매핑 대수구조에 강하게 의존.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |

| absorbing set / error-floor 분석 | `corner.cpp` (수렴실패 추적) | 수렴실패 케이스가 trapping/absorbing set인지 진단에 연결 가능 |

| 고정소수점 양자화(Q4.2), sum-product | `decoder.cpp` `VN_Cal_HD()` 등 / `ecc_data.h` `PARAM_LLR` | Prime ECC는 min-sum이라 SPA와 다르나 양자화 관점 참고 |

적용 가치: **중간** — error-floor를 부호구조·girth 보존한 채 낮추는 접근은 NAND 고rate QC-LDPC ECC에 매우 매력적이나, 기법이 array-based `a(i)·j` 대수구조에 특화되어 Prime ECC의 고정 base+circulant-shift H-matrix에 그대로 이식하려면 absorbing set 스펙트럼 재분석과 shift 재설계가 필요.
