### B. 알고리즘 요약

1. 대상은 (J,K)-regular 프로토그래프 기반 LDPC convolutional code를 종단(termination)해 만든 asymptotically regular block code ensemble의 QC sub-ensemble.
2. 문제: QC sub-ensemble은 ensemble의 'typical' 멤버가 아니며(순환행렬 비율 1/(N-1)!→0), base가 0/1이면 dmin이 `(nc+1)!`로 상계됨 → 이 상계를 어떻게 키우는가.
3. 모델: 프로토그래프 base matrix `B`를 복사-치환(copy-and-permute)으로 `H` 생성, 순환치환행렬 `I_a`로 제한하면 QC 부호.
4. 핵심 기법: Smarandache-Vontobel 정리(Theorem 1)로 dmin(C) ≤ min over S의 `perm(B_{S\i})` 합 — permanent 기반 상계.
5. 식 의미: base matrix의 permanent가 QC 부호 최소거리 상한을 지배하므로, edge-spreading 선택이 상계를 좌우.
6. 확장 1: convolutional protograph를 `L` 시점에서 종단 → rate `R_L = 1-(L+ms)/L·(1-R)`, L 증가 시 rate 상승·regular 근접.
7. 확장 2: graph cover(m-cover)로 base 항목을 circulant 배열로 대체 → dminQC 상계 추가 향상.
8. 핵심 관찰: 좋은 ensemble 최소거리 성장률(δmin)을 주는 edge-spreading이 QC 상계도 크게 함 (Table I: Ex1=56, Ex2=176, Ex4 3-cover=210).
9. 결과: [294,51,56] girth-8, [152,38,30] 등 상계 달성 QC 부호 구성; graph-cover로 30→82→210 향상.
10. 한계: dmin 상계 이론·구성만, 실제 복호성능/BER·HW·NAND 언급 없음, BEC threshold 기반, 저rate 예제.
