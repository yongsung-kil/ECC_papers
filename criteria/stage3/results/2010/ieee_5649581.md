### B. 알고리즘 요약

1. 대상은 이진 소실채널(BEC)에서의 QC-LDPC 부호, base matrix `B`(a×b) + 순환시프트로 확장(factor `z`), rate `R=2/3`, source degree 5인 RRA(staircase) 구조.
2. 문제: BEC에서 유한길이 부호는 IT(반복) 복호보다 ML 복호가 정정능력이 좋으나, ML은 잔여선형계 `He Xe = Hr Xr`를 가우스소거로 풀어야 해 복잡도가 `O(k^2)`로 큼.
3. 핵심가정: 대칭 소실 채널, symbol=패킷(AL-FEC), IT 실패 시 ML 활성화하는 hybrid IT/ML 복호.
4. 핵심기법 1: QC 행렬에 단순 row/column 순열(`i=xi+yi·a`, `j=xj+yj·b`)을 가하면 `H'`가 pseudo-band 구조가 됨을 증명.
5. 밴드 파라미터 `p=a(M+1)`, `q=b(M+1)` (M=base 최대 shift값). 밴드가 얇을수록 GE 비용 감소.
6. 핵심기법 2: 밴드폭을 부호 차원에 따라 키우려 `M = C·z/(bR)` (band QC), 즉 `M ∝ √k`로 두면 GE 복잡도가 `O(k√k)`.
7. 구현 트릭: staircase 마지막 원소를 다시 z×z staircase로 확장해 degree-1 열 제거, base 비음수 엔트리는 {0..M}에서 랜덤 선택(온더플라이 생성 용이).
8. 코드앙상블 4종(band/unconstrained/constant-bandwidth/protograph) 비교로 밴드폭-성능 트레이드오프 검증.
9. 결과: band QC-LDPC가 ML 하에서 이상부호 대비 overhead 0.5%로 채널용량 근접, 복잡도는 `O(k√k)`로 protograph/unconstrained의 `O(k^2)`보다 우수. error floor는 10^-5 수준.
10. 한계: BEC 전용, HW 미설계, 시뮬만, min-sum류 message-passing 복호와 무관.
