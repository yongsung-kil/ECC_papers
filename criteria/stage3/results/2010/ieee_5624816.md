### B. 알고리즘 요약

1. 시스템: DVB-S2/T2/C2 표준 LDPC, 구조화(AA-LDPC/QC) 검사행렬(shifted identity, size P). 짧은 프레임 16200 / 긴 프레임 64800, 11개 code rate(1/4~9/10). layered(수평) 디코더.
2. 문제: layered 디코더 면적의 최대 70%가 메모리(긴 프레임). 유한정밀도 양자화가 성능·면적을 좌우하는데 SO/extrinsic saturation 방식이 문헌에 잘 설명 안 됨.
3. 모델: SO 기반 Node Processor. `Mv→c = SOv − Mc→v_old` (1), CN update sign/mag 분리 (2)(3), `SOnew = Mv→c + Mc→v_new` (4). normalized Min-Sum `|Mc→v_new| = α·min|Mv→c|`, α=0.75.
4. 핵심 기법1: SO saturation 문제 분석 — SOmax로 clip 시 SO가 감소만 가능한 비선형성 발생→pseudo-codeword·error floor 유발.
5. 식 의미: saturation 오차 ε=SOnew−SOmax가 다음 iteration `Mv→c=Mv→c−ε`로 전파, dv=13까지 누적되면 부호까지 바뀜.
6. 핵심 기법2: APP-SO saturation 알고리즘 — saturation 발생 시에만 `Mv→c = SOv` (APP 근사, 식(6)) 사용, 아니면 정상 식(1). 복잡도↓ + saturation 비선형성 회피.
7. 구현 디테일: extrinsic Mc→v를 {min, submin, indmin, sign} 4원소로 압축 저장. 저장된 Mc→v_old만 saturate(1회 사용→면적절감, 성능영향 작음).
8. 메모리 최적화: 워드 분할 — layer 계산 지연 dc cycle을 활용해 1 word를 2~3 cycle에 나눠 fetch. WRAM=9bit가 최적(874800 bits, 이론 최소 대비 +26%).
9. 결과: XQ5VLX110 Virtex-V FPGA, rate-2/3 long frame, 200MHz, 평균 20 iter, throughput 90Mbit/s(HDTV 2스트림). 5-6-5 양자화로 기존 대비 extrinsic 메모리 28%(4-6-4는 40%) 절감, 성능손실 없음.
10. 한계: HW 실측 아닌 FPGA 합성/시뮬. DVB-S2 특화(긴 프레임). CN degree<7(rate 1/4,1/3,2/5)에서 normalized Min-Sum error floor(A-min*/λ-min으로 해결 제안). 저BER 성능 미평가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| normalized Min-Sum CN 연산(min/submin/ind/sign) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC도 min1/min2+sign XOR 보유 — 알고리즘 중복 |
| SO/extrinsic 메시지 양자화·saturation | `decoder.cpp` `VN_Cal_HD()` 등, `ecc_data.h` `PARAM_LLR` | 유한정밀도 saturation 아이디어는 우리 양자화 테이블에 참고 가능 |
| APP-SO saturation(error floor 회피) | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | saturation 시 APP 근사 스위칭은 우리 VN 업데이트에 이식 검토 여지 |
| extrinsic 메모리 워드 분할 | 대응 없음 | Prime ECC HW 메모리 모델(z=32 RAM)과 구조 다름, DVB layered 전용 |
| layered(turbo) 스케줄, Barrel Shifter, P-parallel NP | 대응 없음 | Prime ECC는 z=32 row-parallel+GT/HCU 구조로 layered 아키텍처와 상이 |

적용 가치: **낮음~중간** — min-sum·layered 구조 자체는 이미 보유/상이하나, APP-SO saturation 및 유한정밀도 메시지 saturation 기법은 Prime ECC 양자화 테이블·VN 업데이트에 참고할 여지가 있음. 다만 DVB-S2 긴 프레임·layered 전제라 그대로 이식은 어려움.
