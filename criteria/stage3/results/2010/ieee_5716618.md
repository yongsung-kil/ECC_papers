> 총평: layered OMS + 5bit 양자화(message clipping) + latch 분산메모리 + L-memory bypass + per-layer PPC 조기종료로 15.8pJ/bit/iter 달성한 저전력 실측칩. 알고리즘은 우리와 중복이나 저전력 메모리/양자화 회로기법이 참고 가치.

### B. 알고리즘 요약

1. 대상: IEEE 802.11n QC-LDPC, `Np=24` col, `Z∈{27,54,81}`, rate `1/2,2/3,3/4,5/6` (12개 matrix prototype), runtime-programmable.
2. 문제: 고throughput(600Mbps) + 12개 부호구성 지원하며 에너지효율 최적화 필요.
3. 핵심기법1 (layered offset-min-sum): H를 layer(행)별 순차 갱신, flooding 대비 약 2배 빠른 수렴; OMS는 SPA 대비 복잡도 대폭 감소.
4. NCU 연산: MIN unit이 min1/min2·minIdx·sign 계산 -> SEL unit이 offset 뺀 R-message 갱신 (`max` 함수), MIN/SEL pipeline interleave로 인접 layer 동시 처리.
5. message clipping (MC): R-message 클리핑으로 L/Q word length `7bit->5bit` 축소, CS/NCU 전력 16%/19% 감소, 전체 21% 절감.
6. 수 표현: sign-magnitude 채택 (2's complement 대비 총전력 9.5% 감소).
7. 저전력 메모리: latch 기반 standard-cell 분산 스토리지, one-hot clock-gating write, L-memory bypass(rate-5/6 Z=81시 write 90% 우회로 L-memory 전력 75%·전체 7.5% 절감), 9 NCU/MCC 파티셔닝으로 data locality 확보.
8. 조기종료: per-layer partial parity check(PPC) OR 조합으로 minimal HW 조기종료, 4dB SNR에서 FER 손실 없이 에너지 58% 절감.
9. 결과: 90nm CMOS 실측, 코어 1.77mm2/398kGE, 346MHz, 680Mbps@10iter(rate-5/6,Z=81), 15.8pJ/bit/iter (선행연구 [5] 대비 7.8배 개선); 10->5iter시 에너지 50%+ 절감·SNR 0.2dB 손실.
10. 한계: 정정능력 개선 없음(수렴·전력 최적화), 802.11n 특정 부호 종속, 저전력 이득이 HW/양자화 회로기법에 의존.

적용 가치: **중간** — layered OMS·min1/min2·조기종료는 우리 코드와 중복이나, message clipping(5bit) 양자화 word length 축소와 L-memory bypass·latch 분산메모리 같은 저전력 회로기법은 실 HW(RTL) 관점에서 참고 가치가 있음. 단 우리 시뮬은 전력 모델이 없어 직접 이식보다는 RTL 설계 참고에 그침.
