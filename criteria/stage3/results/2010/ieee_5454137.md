### B. 알고리즘 요약

1. 대상: circulant permutation 기반 structured LDPC(array-based [Fan], Tanner QC-LDPC), regular, girth ≥ 6.
2. 문제: 유한길이 LDPC의 error-floor는 최소거리보다 작은 absorbing set(반복복호의 고정점)이 지배한다.
3. 모델: `(a,b)` absorbing set = 크기 `a`인 bit-node 집합 `D`, 홀수차 인접 check 수 `b`, 각 bit가 만족 check를 불만족 check보다 많이 갖는 구조(bit-flip 하에 안정).
4. 핵심 하한: bit degree `r`, girth `g`에 대해 `t = ⌈(r+1)/2⌉` 만족 check 필요 → 최소 stopping set 하한을 absorbing set으로 사상하여 `a*` 하한 유도.
5. 의미: 최소 absorbing set 크기 `a*`가 `d_min`보다 작으면 error-floor 원인이 됨을 정량화(예: c=3,g=6 → (3,3); c=5,g=6 → (4,8)).
6. 확장: array code `f(i,j)=i·j`에서 (4,8) absorbing set 존재를 증명(Lemma 2).
7. 트릭: 매핑 규칙을 `f(i,j)=Ir(i)·j mod p`로 교체 — `Ir`의 pairwise 차의 곱이 중복되지 않도록 greedy 선택(예: `Ir=(0,1,3,8,19)`).
8. 최적화: girth·node degree·rate·구조를 보존하면서 absorbing set만 제거(그리디 시퀀스 구성).
9. 결과: 수정 매핑으로 (4,8) absorbing set 제거, 길이-6 cycle은 유지하되 overlap을 sparsify. (2048,1723) RS-LDPC는 하한 (5,10)이나 실제 (5,10) 없음((8,8)이 지배).
10. 한계: 순수 조합 이론, HW·시뮬레이션 성능 검증 없음, 특정 QC 구성에 한정.
