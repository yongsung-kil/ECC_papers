### B. 알고리즘 요약

1. 대상: BSC(crossover `q`)에서 SP 복호되는 임의 이진 LDPC. 실험은 MacKay `(504,252)` rate 0.5, DSC `(273,191)`.
2. 문제: SP 복호의 정확한 WER 해석식이 없고, 관행적으로 초기화 확률을 채널값 `q`로 그대로 씀.
3. 핵심 아이디어: SP 복호에 채널 추정 crossover `p`를 별도 파라미터("initialization")로 도입 → WER가 `WER_{H,lmax}(p,q)` 2변수 함수가 됨. 기존 SP는 `p=q` 특수경우.
4. correctable error set `E_{H,p,lmax}`: 고정 `p,lmax`에서 SP가 정정 가능한 오류패턴 집합. 채널과 무관하게 결정됨.
5. 정리 II.1: 고정 `p0`에서 `WER(p0,q)`는 `q`의 다항식이며, weight enumerator `A_E[1-q,q]`로 `WER=1-A_E[1-q,q]` 표현.
6. 성질: `WER(p,1/2)·2^N`는 정수(=set 여집합 크기), 원래 SP의 `WER(q,q)`는 `p`에 대해 불연속일 수 있음(H1 예제로 (0.220,0.258) 구간 불연속 증명).
7. 계산: 소co-dimension 부호는 syndrome 복호(Corollary VI.2)로 exhaustive search 없이 correctable set 결정 가능(길이 121 array (3,11) FSA QC-LDPC에 적용).
8. Fixed Initialization Decoding(FID): `p`를 `q`가 아닌 고정값으로 사용. MacKay 코드에서 `p=0.10`이 원래 SP를 여러 `q`에서 상회.
9. 결과: FID가 `lmax=10`처럼 짧은 iteration에서 특히 SP 대비 WER 개선·수렴 가속(예: DSC에서 `p=0.07`이 원SP보다 빠른 수렴). 이론 WER 다항식으로 고SNR 외삽 가능(`WER(0.01,1e-5)=9.04e-12`).
10. 한계: BSC 1-bit 전제, HW 미설계, `p` 최적값은 예제별 수동 선택.
