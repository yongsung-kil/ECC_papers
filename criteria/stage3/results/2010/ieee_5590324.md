### B. 알고리즘 요약

1. 대상: 유한체 GF(q) 상 binary QC-LDPC, BPSK/AWGN 및 BEC, SPA/min-sum(MSA) 복호, 최대 50 iter.
2. 문제: 알고리즘적 QC-LDPC(문헌 [8],[24])는 구성만 제시되고 PCM의 rank/dimension 분석이 없어 실제 부호 차원 예측 불가.
3. 핵심 구성: GF(q) 원소 `α^i`를 (q-1)×(q-1) circulant permutation matrix(CPM) `P^i`로 분산(binary matrix dispersion), RD-constraint 만족 base matrix `W`를 array dispersion → RC-constraint(girth≥6) 만족 `H`.
4. Latin square 구성: `w_ij = α^i·η − α^j` 형태 q×q Latin square가 RD-constraint를 만족함을 Theorem 2로 증명, η 변화로 q-1개 Latin square 생성 → 기존 [8] method-1보다 훨씬 큰 부호족.
5. rank 정리(Theorem 1): Fourier 변환으로 `H`를 주파수영역 대각화, `rank(H)=Σ_{l=1}^{2^m-1} rank(W∘l)` (Hadamard 곱의 rank 합).
6. rank 폐형식(Theorem 3): `rank(W∘l(γ,ρ)) ≤ min(γ,ρ,λl)`, `ρ=2^m`이면 `=min(γ,λl)`, 여기서 `λl`=Pascal 삼각형 l행의 홀수 이항계수 개수.
7. 최소거리: RC-constraint로 min-distance ≥ γ+1(홀수 γ), 짝수 γ면 ≥ γ+2 보증 → low error-floor 근거.
8. masking: 규칙적 base array에 불규칙 masking matrix(density evolution 설계) 적용해 irregular QC-LDPC 구성(Example 6, rate-1/2 (32640,16320)).
9. 결과: (992,750) rate 0.756 code는 SPA 50iter로 sphere packing bound 0.9dB, FPGA min-sum으로 BER 1e-11까지 error-floor 無; (16200,15125) rate 0.9336는 Shannon limit 1.3dB, BER 1e-12까지 floor 無; PEG 랜덤부호와 성능 동등.
10. 한계: HW 아키텍처 설계 없음(FPGA는 성능 시뮬용), circulant 크기 q-1 고정, 부호rate가 subarray 선택에 종속.
