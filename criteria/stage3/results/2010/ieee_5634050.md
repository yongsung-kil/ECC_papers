### B. 알고리즘 요약

1. 시스템: PolMUX QAM 광전송(100Gb/s aggregate), 각 편광 스트림을 rate `Ri=Ki/N` 역연접부호로 인코딩 후 비트인터리버·QAM 매퍼.
2. 문제: 대girth(9~10) 장부호 LDPC는 성능 좋으나 codeword가 너무 길어 QC 설계·FPGA/VLSI 구현 곤란, 반면 짧은 저girth(6/8) LDPC는 error-floor 발생.
3. 핵심가정: 채널손상(PMD/분산/비선형)은 turbo equalization으로 이상 보상되었다고 가정, 심볼 LLR `Λ(q)=log P(q|r)/P(q0|r)`.
4. 핵심기법1(역연접): 내부부호=BCH/RM(soft-MAP 복호), 외부부호=LDPC. 기존 Mizuochi 방식(LDPC 내부/RS 외부, hard-decision)과 위치·복호를 뒤집음.
5. 핵심식: 내부 BCH를 BCJR/Ashikhmin MAP로 복호해 고정밀 bit-LLR을 외부 LDPC로 전달 → 외부 LDPC 입력 신뢰도 향상.
6. 확장: BCH↔LDPC 간 외부 반복(extrinsic LLR 교환) `L_BCH,e(cj^k)=L_LDPC(cj^k)−L_LDPC(cj^(k−1))`, 5회 외부 + 10회 내부 LDPC iteration.
7. 구현디테일: LDPC는 min-sum-with-correction-term 알고리즘, 외부 LDPC는 modified PEG로 생성한 girth8·col-weight3 부호.
8. 최적화: 대응 QC-LDPC(16920,14805)는 col-weight3/row-weight24/permutation block 705로 등가 성능.
9. 결과: LDPC(16935,14819)-BCH(64,57)의 NCG 9.62dB@BER10^-9(11.38dB@10^-13), Chase-II turbo-product 대비 0.94dB 우수, girth-12 장부호(345165)와 0.25dB 차이이며 복잡도 32.16배 낮음.
10. 한계: HW 미설계·시뮬만, 광통신/QAM/turbo-eq 특화, 부호는 non-QC PEG 기반(등가 QC만 언급).
