### B. 알고리즘 요약

1. 시스템: 40~100Gb/s 고속 coherent 광전송에 고차변조(8-PSK/16·64·256-QAM 등) + LDPC multilevel coding, PID(병렬독립복호) 또는 MSD(다단복호).
2. 문제: 초고속 throughput 제약으로 soft decoding이 너무 복잡 → hard-decision 복호로 한정. 기존 irregular hard 디코더(GB)는 variable node degree 정보 저장/조회가 회로 복잡도를 키움.
3. 채널모델: 등화기 출력의 등가 AWGN `y_k = s_k + n_k`, hard-demod 후 crossover 오류확률 `p0`로 표현.
4. 핵심기법(GB 동등 디코더): CN은 GB와 동일(edge 곱), VN은 채널입력과 불일치하는 extrinsic 개수의 discrepancy `m_k = -u_n·Σ u_{c→v,j}`를 계산해 임계 `d^(l)` 이상이면 flip.
5. 핵심식 `d^(l)=2b_j^(l)-(j-1)`: GB의 degree별 flipping threshold `b_j`를 discrepancy 기준으로 변환하면 threshold가 **degree j에 무관**해져 모든 VN이 동일 `d^(l)` 사용 가능.
6. 확장: switching point `{v_d(p0)}`가 degree 무관임을 증명 → GB와 정확히 동일 성능이면서 degree 저장/조회 제거.
7. finite-length 변형: K-exp-OSS(`d^(l)=d*^(floor(l/K))`, 각 threshold K회 반복), L-add-OSS(각 threshold에 L 이터 추가). 실제 EEP가 해석치보다 클 때 복호실패 방지, 안정화.
8. 학습/최적화: EEP(extrinsic error probability, EXIT류) 함수 `h(p0,x)<x` 제약으로 DE/LP로 irregular profile 최적화; switching point `v_d(p0)`만 고려해 최적화 대폭 단순화(레이트별 15개 미만).
9. 결과: 3-exp-OSS가 원 디코더 대비 length-15000 QPSK에서 약 0.6dB 개선; 5-add-OSS는 평균 iteration이 3-exp-OSS의 약 70%. multilevel(PID/MSD)이 single-level 대비 유의미 이득(광통신 전송거리 14.8~17.5% 증가 환산).
10. 한계: HW 미설계(시뮬만), hard-1bit 전용, 고차변조 multilevel/광통신 특화 가정.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| CN 이진 message-passing / VN discrepancy flip 규칙 | `decoder.cpp` `LDPC_Decoder()`, `VN_Cal_HD()` | 중: Prime ECC의 HD 경로와 대응, degree 무관 단일 threshold는 HD VN 로직 단순화 아이디어 |
| iteration별 flipping threshold `d^(l)` 테이블 | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | 중: 적응형 threshold를 iter 구간별로 두는 구조와 개념 유사 (LLR→flip-threshold 대응) |
| EEP 기반 irregular profile 최적화 (DE/LP) | `ecc_top.cpp` `Load_PCM()` (부호구조) / `local_opt.cpp` (LLR 자동최적화) | 낮음: Prime ECC는 특정 QC-LDPC 고정, irregular profile 재설계는 재검증 부담 큼 |
| soft-decision 대체(hard-only) | 대응 없음 (Prime ECC는 2SD/3SD soft 지원) | 낮음: NAND soft-read 이득을 버리는 방향 |

적용 가치: **중간** — degree 무관 hard-decision flip 규칙과 iteration별 threshold 안정화(exp/add-OSS) 아이디어는 Prime ECC HD 경로 단순화에 참고 가치가 있으나, 대상 부호가 irregular이고 hard-1bit 전용이라 soft-read 지원 고rate QC-LDPC로의 직접 이식은 제한적.
