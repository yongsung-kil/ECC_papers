### B. 알고리즘 요약

1. 대상: DVB-S2/802.16e류 binary QC-LDPC 복호기. 테스트 코드는 length 6400, rate 13/25·17/25·21/25, expansion factor Z=256.
2. 문제: 기존 멀티모드 디코더(802.16e 19모드, 802.11n 3모드 등)는 특정 표준의 QC-LDPC subset만 지원 — 임의 QC-LDPC 지원 필요.
3. 모델: base matrix 원소(-1/0/양수 shift)를 `Z×Z` circulant으로 전개한 QC-LDPC. 동일 Z면 임의 코드 configure 가능.
4. 핵심기법: offset BP-based(offset min-sum) 복호. CNU가 min1/min2 탐색 후 `offset` 차감, 최소값 위치만 min2 출력·나머지는 min1 출력, sign 부착.
5. 핵심식: `L(r_ji)=α_ij·max(min_{i'∈Rj\i}β_{i'j}-offset, 0)` — tanh 대신 min 근사로 HW 복잡도 절감(BP 대비 소폭 성능손실).
6. 확장: 멀티모드 — Control Unit이 mode register로 3개 router(input buffer↔node RAM, node RAM↔CNU, node RAM↔VNU)를 재구성, N→X·M→X 인덱스 매핑을 Matlab 사전생성.
7. 구현 디테일: node RAM 구조화 주소지정, 미사용 CNU 입력=최대값/미사용 VNU 입력=0 주입, CNU/VNU 파이프라인, 6-bit 양자화.
8. 최적화: rate별 최적 offset를 컴퓨터 탐색(0.5/0.5/0.4).
9. 결과: Altera Stratix II FPGA, 200MHz, iteration 12 고정, throughput 104/136/168 Mbps(rate별). 멀티모드 자원(ALUT 12662)은 최대 단일모드(9180) 대비 소폭 증가.
10. 한계: 성능(BER) 개선 없음(구현/멀티모드 논문), throughput 100Mbps급으로 NAND급 미달, 모든 지원코드가 동일 expansion factor여야 함.
