### B. 알고리즘 요약

1. 대상: 임의 구조 LDPC (regular/irregular 모두), Tanner graph 기반. 예제는 (13,26) 열무게 3 부호.
2. 문제: 통상 LDPC 인코딩은 `c=uG`로 `O(N^2)` (G가 dense). 이를 선형복잡도로 낮추는 범용 방법 부재.
3. 핵심 도구1 `label-and-decide`: Tanner graph에서 bit를 information/parity로 labeling 후 parity를 순차 계산. cycle-free tree·pseudo-tree에서 성공.
4. pseudo-tree 정의(A1~A4): tier 계층구조로, 각 check가 상위 tier에 유일한 parent bit를 가짐 → 선형시간 인코딩 가능(Lemma 1).
5. `encoding stopping set` 정의(B1~B3): 모든 bit가 set 내 check 2개 이상에 연결되고 check들이 독립인 연결 부분그래프. label-and-decide로 인코딩 불가(Theorem 1).
6. 핵심 도구2 `label-decide-recompute`/`label-decide-flip`: key check node 1~2개를 제거하면 pseudo-tree가 됨. reevaluated bit를 재계산해 stopping set을 선형시간 인코딩.
7. 분해 정리(Theorem 3): 최대 열무게 3인 임의 Tanner graph는 pseudo-tree와 (1-/2-fold) encoding stopping set으로 분해 가능 → 순차 인코딩.
8. 확장(Theorem 4): 고차 열무게 bit를 degree-3 bit + auxiliary check로 node splitting해 임의 열무게/행무게 부호로 일반화.
9. 결과: 전체 인코딩 복잡도 `< 4M(1/k1 - 1)(k-1)` (M=독립행 수, k=평균 행무게)로 선형. 순수 이론 증명, HW/시뮬 없음.
10. 한계: 전처리(분해·reevaluated bit 탐색)는 offline이나 인코딩 자체가 tier별 순차 순회라 병렬 HW에 불리, 특정 부호구조 이점 미활용.
