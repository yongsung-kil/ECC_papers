### B. 알고리즘 요약

1. 대상은 WiMAX(802.16e)·WiFi(802.11n) QC-LDPC(base matrix c×d, d=24, z×z circulant 확장), 실험은 length-2304 rate-0.5(WiMAX)·length-1944 rate-2/3(WiFi), AWGN·BPSK.
2. 문제: 기존 SC 중 SPE(모든 parity-check 만족)는 BER 손실 없으나 불필요 iteration 낭비, HDA(연속 iteration hard-decision 불변)는 BER 손실 유발.
3. 관찰: column weight 큰 정보비트가 check node 연결이 많아 APP(LLR) magnitude가 더 빨리 커짐 → 정보비트가 redundant(parity)비트보다 훨씬 빨리 수렴(Fig.3), 표준 PCM의 parity부는 대부분 column weight=2.
4. 결과적으로 iteration 부족으로 오복호된 codeword는 대개 오류가 parity부에 몰림 → 오류가 rightmost M-z redundant 비트에만 있는 것을 Type II, 그 외를 Type I로 정의.
5. 핵심 기법: syndrome `s`를 z 주기로 접어 더한 `SAV a`(`a_i = Σ_{k=0}^{c-1} s_{i+kz}`)를 정의.
6. 핵심 성질(수학적 귀납 증명): rightmost M-z redundant 비트에만 n개 오류가 있으면(Type II) SAV는 짝수만 포함 — two-diagonal 구조상 각 redundant 오류비트가 정확히 2개 row(같은 a_j)에만 영향.
7. 역방향: SAV가 모두 짝수이면 거의 항상 Type II(정보/first-z 비트 오류는 random 위치 3개 이상 row에 영향해 SAV를 홀수로 흐트러뜨림), 7M codeword 실험서 false detection<1e-7.
8. 제안 SC: 매 iteration마다 SAV 계산, 모두 짝수면 즉시 종료(정보비트 다 맞으면 parity 오류 남아도 중단, HW 검증 용이).
9. 결과: WiMAX/WiFi에서 SPE 대비 AIN을 최대 11% 이상 절감(rate 낮을수록 이득 큼), BER/FER 성능 손실 없음(SPE와 동등, HDA와 달리 무손실).
10. 한계: rightmost two-diagonal parity 구조 종속, HW 미설계(floating-point 시뮬), false detection 0은 아님(매우 낮으나).

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SAV 짝수성 기반 조기종료 SC | partial_CRC.cpp partial_crc_HW_T4(), full_CRC.cpp | 조기종료 지점에 대체/보완 판정 로직으로 이식 가능(Prime ECC는 CRC 기반 보유) |
| syndrome 계산 s = H·x | decoder.cpp LDPC_Decoder() (syndrome check) | 이터레이션 내 syndrome 산출은 이미 존재, SAV 매핑만 추가 |
| two-diagonal parity 구조 의존 | encoder.cpp (dual-diagonal 인코딩) | Prime ECC도 dual-diagonal이라 유사 parity 구조 존재, 성질 재검증 필요 |
| APP/LLR 수렴 관찰 | decoder.cpp VN_Cal_HD(), ecc_data.h PARAM_LLR | 정보비트 우선 수렴 관찰은 정성적, 직접 이식 요소 아님 |
| Type II(parity-only) 오류 검출 | full_CRC.cpp | 정보비트 정확성 확인 후 종료 개념은 full-CRC 조기종료와 목적 유사 |
```

적용 가치: **중간** — Prime ECC가 조기종료를 이미 partial/full CRC로 보유하나, SAV 짝수성 검사는 CRC보다 가벼운 HW로 "정보비트 정확 시 종료" 판정을 줄 수 있어 avg iteration/latency 절감 관심사와 부합. 다만 이 성질은 WiMAX/WiFi rightmost two-diagonal parity에 종속되므로 Prime ECC의 dual-diagonal PCM에서 동일 성립 여부 재검증이 전제.
