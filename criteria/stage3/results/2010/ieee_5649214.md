> 총평: cyclic EG-LDPC의 circular PCM을 순열로 CPM 배열로 분해해 고rate 포함 다양한 binary QC-LDPC를 얻는 유한기하 부호설계로, error-floor 없는 고rate(16200,15128) 실증까지 있어 Prime ECC 부호설계 후보로 중간 이식성.

### B. 알고리즘 요약

1. `n=c·l`로 인수분해되는 cyclic 부호의 `n×n` 원형(circulant) PCM `Ψ(w)`를 순열-`π`(식(2),(3))로 열/행 치환해 `c×c` 크기 `l×l` circulant 배열 `Φ(w)`로 분해(Theorem 1).
2. 문제: cyclic 부호는 인코딩(단일 시프트레지스터)이 쉽고 QC 부호는 배선(wire routing)·부분병렬 복호에 유리 — 둘의 장점을 한 부호에서 취하려 함.
3. 핵심: `Ψ(w)`와 순열 후 `Φ(w)`는 조합적으로 등가라, cyclic 형태로 인코딩하고 QC 형태로 복호 가능(양쪽 HW 장점 동시 확보).
4. descendant 부호: `Φ(w)`의 각 descendant circulant `Ψ(w_i)`의 null space가 짧은 cyclic 부호를, 부분배열 `Φ(w)(γ,ρ)`가 QC 부호를 파생.
5. Theorem 2/3으로 descendant 부호의 생성다항식 근(root)과 EG(2,q)의 CPM/ZM 배열 구조(`(q+1)×(q+1)`, 각 행/열에 ZM 정확히 1개)를 특성화.
6. cyclic EG-LDPC(2D 유클리드기하 `EG(2,q)`의 선 incidence 벡터)를 mother code로 삼아 RC-constraint(4-cycle 없음, girth≥6, `d_min≥γ_min+1`) 만족하는 이진 cyclic·QC-LDPC 파생.
7. masking: base array `π(HEG)(γ,ρ)`에 저밀도 masking 행렬 `Z(γ,ρ)`를 Hadamard-곱해 edge 축소(더 큰 girth, 다양한 regular/irregular 부호) — 예: `(4032,2016)` rate 0.5.
8. 결과 부호: `(1365,765)` rate 0.56(d_min=17, trapping set (17,0)로 error-floor가 min-distance 지배), `(4095,3771)` rate 0.9208(SPA 50iter, Shannon 1.1dB), `(16200,15128)` rate 0.9338.
9. `(16200,15128)`는 scaled MS 50iter를 FPGA 디코더로 실측해 error-floor 없이 BER 1e-12 도달, 동rate RS(1473,1375) 대비 BLER 1e-6에서 1.7dB 코딩이득.
10. 한계: 부호구성법·성능평가 위주로 인코더/디코더 HW 아키텍처는 미설계, SPA/scaled-MS 시뮬(일부 FPGA 실측)에 국한.
