### B. 알고리즘 요약

1. 대상은 (G)LDPC의 LP(linear programming) 디코딩으로, Feldman의 fundamental polytope `P` 위 `P(c)=Σ c_i γ_i` 최소화(ML certificate 성질 보유).
2. 문제: LP 디코딩은 ML certificate가 있으나 복잡도가 다항(비선형)으로 BP/min-sum보다 크고 relaxation이 느슨하다.
3. 채널 모델: 이산 무기억 채널(DMC) `Q(y|c)`, GLDPC(constituent code `C_j`) 일반화; `γ_i=log(Q(y_i|0)/Q(y_i|1))` LLR.
4. 핵심 기법 1(check-node merging): 여러 constraint node를 하나로 병합해 `Q'(C)⊆Q(C)`인 더 조인 relaxation을 만들어 오류확률을 낮춘다(`P'_e(c)≤P_e(c)`).
5. Proposition 3: 병합 대상 노드+직접이웃 부분그래프가 cycle-free면 relaxation 불변 → 병합은 **사이클을 포함하는** check node 집합에 대해서만 유효.
6. 핵심 기법 2(선형복잡도 LP): Vontobel-Koetter 소프트버전 Problem-DS를 `log-sum-exp`(파라미터 `K`)로 min 근사, Burshtein[5]의 Algorithm 1로 O(N) 반복(각 상수연산)에 `λ̃` 산출.
7. 구현 디테일: 각 edge에 `u_{i,j}, v_{i,j}, λ_{i,j}` 변수, active set `A={i:ε_i≥ε_0}` 기반 좌표선택 업데이트, `v_{i,j}=-(1/K)log(B_{i,j}/A_{i,j})`.
8. 최소거리 하한: check node `r`에 local codeword 강제 후 Algorithm 1로 최소무게 pseudocodeword 근사, dual vector L1 norm의 최소가 최소거리 하한이자 fractional distance 상한; 전체 check node 순회로 O(N^2).
9. 결과: (3,4)-regular Gallager 앙상블 10개 코드 평균, N=100~1000에서 병합 시 최소거리 하한이 미병합 대비 향상(그림상 최대 ~16 수준), exact fractional distance와 비교.
10. 한계: HW 미설계, LP/min-sum 디코더 패러다임 상이, 병합 시 constituent code 크기 증가, rate=0.25 저rate regular 예제 중심.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LP 디코딩 / check-node merging | 대응 없음 | Prime ECC는 min-sum iterative MP, LP polytope 디코더 미보유 |
| 반복적 message passing(Algorithm 1) | decoder.cpp LDPC_Decoder() | 개념적 iterative update 유사, 알고리즘·메시지 정의 상이 |
| 최소거리/pseudocodeword 하한 도구 | 대응 없음 | 부호 검증용 오프라인 도구, 코드베이스 대응 모듈 없음 |
| DMC LLR(γ_i) 소프트정보 | channel.cpp Set_LLR_Th() | LLR 개념 공유(단 LP 특유 polytope 제약과 무관) |
```

적용 가치: **낮음** — LP 디코딩 개선과 최소거리 하한 산출은 이론·검증 도구로서 가치가 있으나, Prime ECC의 HW min-sum 디코더에 이식할 알고리즘 대응이 없고 LP 자체가 HW 부적합이라 실디코더 이식성은 낮다.
