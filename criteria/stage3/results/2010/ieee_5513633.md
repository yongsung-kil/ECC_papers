### B. 알고리즘 요약

1. 대상: protograph 기반 LDPC. 예시로 irregular AR4JA protograph(design rate `R=(1+e)/(2+e)`)를 사용.
2. 문제: (J,K)-정규 부호는 선형 최소거리 성장은 좋으나 iterative threshold가 나쁨 — rate와 threshold와 최소거리를 동시에 유연 설계하고 싶음.
3. 기법: AR4JA protograph를 edge-spreading으로 component 행렬 `B0,B1`(`ms=1`)로 분해해 무한 컨볼루션 protograph `B[-∞,∞]` 구성.
4. 종단(termination): `t=0`부터 `L` time instant 후 절단해 유한 base matrix `B[0,L-1]` 생성 → LDPC 블록부호 앙상블화.
5. rate: 종단 rate `R_L = 1 - ((L+ms)/L)(1-R)`, `L`↑ 시 원 컨볼루션 rate에 수렴(예 ARJA는 `R_L=(L-1)/2L`, L→∞에서 1/2).
6. 설계 디테일: degree-1/2 check node 회피 위해 degree-3 check `c0` edge를 분할하지 않음 → 선형 최소거리 성장 보장.
7. 분석: BEC density evolution으로 iterative threshold, 점근 weight enumerator로 최소거리 성장률 `δmin` 산출.
8. 핵심 관찰: `L`↑ 시 threshold가 Shannon 한계에 근접(ARJA e=0: `ε*→0.4996` vs `εsh=0.5`)하고 더는 열화하지 않음(threshold saturation), 단 `δmin`은 감소 → 거리성장 vs threshold 트레이드오프를 `e,L`로 튜닝.
9. 결과: TAR4JA로 `1/4≤R≤6/7` 범위 asymptotically good 부호 대량 확보. 종단 후 capacity gap 예: L=9→약10%, L=20→5%, L=50→2%, L=100→1%.
10. 한계: BEC 이론(DE+enumerator)만, AWGN은 별도 [15]로만 언급, 유한길이 BER·HW·throughput 없음.

| edge-spreading / component 행렬 설계 | 대응 없음 | Prime ECC는 base+circulant QC 고정, protograph 종단설계 미탑재 |

**적용 가치**: 낮음 — spatial-coupling(종단 컨볼루션) 기반 protograph 부호구성 + BEC DE 이론으로, 고rate 고정 binary QC-LDPC + min-sum HW 디코더인 Prime ECC와 부호패밀리·분석틀이 달라 직접 이식 대상이 아님(부호설계 재검증 부담 높음).
