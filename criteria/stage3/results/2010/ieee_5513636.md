### B. 알고리즘 요약

1. 대상: 이진 소거 채널(BEC)에서 rate-1/2 regular (3,6) LDPC, code length `n = 2^i` (i=9~12).
2. 문제: BP(peeling) 디코더는 degree-1 check node가 소진되면 멈춰, 유한길이·순환그래프에서 MAP보다 성능이 크게 낮다.
3. 모델: Tanner 그래프에서 비소거 변수노드를 제거하고 BP를 돌린 잔여 그래프 `T_BP`를 대상으로 함.
4. 핵심기법: BP가 멈추면 degree-2 check node `P_j`를 골라 그것과 연결된 변수노드 하나(`V_2`)를 제거하고, 남은 변수노드(`V_1`)에 나머지 edge를 재연결(`d_V1+d_V2-2`), 제거된 check가 parity 1이면 관련 check parity를 반전.
5. 의미: Maxwell 디코더처럼 미지 변수를 가정하되, 매 iteration당 check·variable 각 1개만 제거하므로 복잡도가 BP와 동일(가정 변수 수에 무관).
6. 확장: degree-2 check의 두 변수가 degree-3 check도 공유하면 그 check를 제거해 degree-1 check를 새로 만들어 BP를 재시작.
7. 트릭: 제거된 변수는 미결정 상태로 리스트에 저장, 남은 변수가 복호되면 역추적으로 결정.
8. 분석: DD(degree distribution) 진화를 Stage A/B 미분방정식으로 모델링, degree-1 check 생성 확률 `p_B(t)`가 시간에 지수적으로 증가함을 보임.
9. 결과: regular (3,6)에서 TEP 용량 하한 `ε_TEP >= 0.4315 > ε_BP = 0.4294`, 유한길이 WER에서 항상 BP 우위.
10. 한계: BEC 전용, HW 미설계, 시뮬만, soft/hard LLR·AWGN 개념 없음.
