### B. 알고리즘 요약

1. 대상: 유클리드 기하 `EG(d,p^s)`의 평행 line/flat 다발과 순환치환행렬을 결합한 하이브리드 QC-LDPC 구성법 (regular, binary, AWGN 채널).
2. 문제: 기존 하이브리드 EG-QC-LDPC 대비 더 큰 girth와 더 낮은 밀도를 얻어 BER·인코딩/디코딩 복잡도를 개선.
3. 모델: `EG(d,p^s)`의 점/line/flat 근접구조; base matrix `B`의 열은 line/flat의 incidence vector.
4. 핵심 기법(3단계): (1) 평행다발로 base matrix `B` (Class-I `BI` line, Class-II `BII` μ-flat) 구성, (2) 각 원소를 `[1,q-1]` shift 값으로 치환해 shift matrix `S`, (3) `α^(a-1)` 위치벡터의 수직·수평 `(q-1)`-fold 확장으로 순환치환행렬 배열 `H` 생성.
5. 식 의미: QC-LDPC에서 길이 `2i` 사이클 부재 조건 `Σ((a_k-1)-(a_{k+1}-1))≡0 mod(q-1)` 위반 여부로 girth 보장.
6. 확장: base matrix가 자체 사이클 없음(Lemma) → block-cycle 탐색 복잡도 하락; `BI` girth 6, `BII` girth 4.
7. 이론 결과: `HI` girth 범위 `6≤g≤18` 밀도 `1/(p^{(d-1)s}q)`, `HII` girth `4≤g≤12` 밀도 `1/(p^{(d-\mu)s}q)`; 충분히 큰 q면 short girth 제거·저밀도 달성.
8. 최적화: shift value assigning algorithm — 열별로 `[1,q-1]` 정수를 순차 배정하며 조건 위반 시 재배정, girth `2i` 보장.
9. 결과: `EG(3,2)` 예제 2개(`q=128` line, `q=256` flat)에서 SPA 반복복호, BER=1e-6에서 uncoded BPSK 대비 코딩게인(girth 클수록 향상) 확인. 기존 [5] 대비 girth≥·밀도↓.
10. 한계: HW 미설계, 시뮬(SPA)만, rate/length 확정값 본문 서술 없음(파라미터 종속), NAND/soft-read 무관.
