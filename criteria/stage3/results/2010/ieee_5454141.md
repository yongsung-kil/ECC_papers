### B. 알고리즘 요약

1. 대상: protograph 기반 (J,K)-regular LDPC convolutional code를 termination factor `L`로 잘라 block code 앙상블 형성, rate `R_L=1-((L+m_s)/L)(1-R)`.
2. 문제: (J,K)-regular block LDPC는 선형 최소거리(asymptotically good)지만 iterative decoding threshold가 나쁘다 — threshold와 거리성장의 trade-off 개선이 목표.
3. 모델: BEC(erasure prob `ε`)에서 BP 복호, protograph 내에서 density evolution 수행 (`q`=check→var erasure, `p`=var→check erasure).
4. 핵심 기법: convolutional base matrix `B_[0,L-1]`를 성분 submatrix `B_0..B_ms`로 구성하고 시간축으로 termination → 경계의 저차 check node가 threshold를 개선.
5. 의미: termination이 만드는 저degree check(경계 효과)가 BP 수렴을 촉진해 threshold를 MAP threshold까지 saturate시킨다.
6. 확장: 성분 submatrix 복잡도(평균 node degree `J`↑)를 높이면 threshold와 거리성장률을 동시에 개선((3,6)→(4,8)→(5,10)).
7. 트릭: edge-spreading — 전체 base matrix `B`의 edge를 `B_0+...+B_ms=B`로 분산; degree-2 check 제거 시 growth rate·threshold 향상, repeated-edge로 메모리·constraint length 축소.
8. 최적화: asymptotic weight enumerator로 spectral shape `r(δ)` 계산, 첫 양의 zero-crossing `δ_min`으로 선형 거리성장 증명.
9. 결과: (3,6) family threshold `ε*→0.4881`(Shannon 0.5 근접), (5,10)은 `ε*=0.4994`; `L`↑ 시 거리성장률은 감소(trade-off).
10. 한계: BEC·asymptotic ensemble에 한정, AWGN은 유사거동 기대만, 유한길이·HW·복호 latency 미분석.
