### B. 알고리즘 요약

1. 대상: 유한체 GF(q) 위 circulant permutation matrix(CPM) 배열로 만든 binary QC-LDPC, rate≥, girth≥6. 예제 rate 0.756~0.934, length 992~16200.
2. 문제: 기존 유한체 QC-LDPC 구성([8][9])은 우수 성능이나 패리티검사행렬 rank(실제 rate·redundancy) 분석이 없었다.
3. 모델: RD-constraint(행 간 Hamming 거리 조건) 만족 base matrix `W`를 CPM으로 array dispersion하면 RC-constraint 자동 만족.
4. 핵심 기법: base matrix에 Fourier Transform(over GF(q))을 적용해 주파수영역 대각화, rank를 대각 블록의 Hadamard power `B^(⊙k)` rank 합으로 표현(Theorem 1).
5. 의미: rank 식은 부호의 실제 dimension(redundant check 수)을 정확히 주어 부호율·인코더 설계에 직접 활용된다.
6. 확장: Latin square 기반 RD-constrained 행렬 클래스 신규 제안(식(7)), GF(2^s)에서 Pascal 삼각형 홀수개수로 rank 상한 유도(Theorem 3~5).
7. 트릭: 대각선에 ZM(0-element)을 배치, subarray 선택으로 다양한 rate/length 부호 파생, shift-register로 효율적 인코딩 가능.
8. 최적화: 별도 학습 없음 — 대수적 결정론 구성(finite field parameter 선택).
9. 결과: QC(992,750)/QC(4032,3304)/QC(16200,15125) 모두 error-floor 없이 낮은 BER, SPA와 FPGA min-sum 곡선 일치, PEG 유사부호와 동등, 5~50 iteration에서 빠른 수렴.
10. 한계: 부호구성·rank 이론 중심, FPGA는 성능확인용이라 gate count/throughput/HW 아키텍처 수치 미제공, 채널은 AWGN/BEC.
