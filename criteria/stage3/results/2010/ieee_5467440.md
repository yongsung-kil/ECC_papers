### B. 알고리즘 요약

1. 시스템: PMRC(수직자기기록)에 외부 shortened RS(GF(2^10)) + 내부 LDPC 연접, user density 1.049 고정.
2. 문제: RS는 defect(burst) 정정 보증되나 랜덤노이즈 열세, LDPC는 랜덤 우세하나 error-floor·defect 취약 → 둘을 연접해 상호보완.
3. 채널: SOVA 채널 검출 + `tanh` transition response, jitter/electronics noise 50:50, media defect = 50bit half-erasure(위치 기지).
4. 내부 LDPC는 PEG 알고리즘으로 rate {0.84,0.86,0.88,0.9}, column weight {2,3}로 설계, RS 코드워드 길이에 맞춰 length 조정.
5. 복호: 내부 LDPC는 normalized Min-Sum, 외부 RS는 hard-decision, 둘 사이 turbo equalization 반복.
6. iterative scheme = (최대 내부 iteration, 최대 외부 turbo iteration) 조합 탐색; outer iteration 증가는 항상 개선, inner는 6회 이상 marginal → T6MS6 선택.
7. 결과(rate): 주어진 t,W에서 inner rate 0.88이 항상 최적, 그 이상은 정정능력 급락.
8. 결과(성능): concatenated code가 RS-only 대비 random·defect 양쪽 큰 코딩게인, column weight 낮을수록 우수, RS가 inner error-floor를 낮춤.
9. error-floor 추정: Kuznetsov microscopic method(에러이벤트 weight 분포 Markov chain + 개수 분포 exponential tail-fitting)로 W=2 연접부호 SER 하한 추정.
10. 한계: HW 미설계, 순수 시뮬, 자기기록 채널 특화, 목표 SER에서 W=2 연접부호는 부적합 결론.
