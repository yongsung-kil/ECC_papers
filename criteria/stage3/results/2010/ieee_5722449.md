### B. 알고리즘 요약

1. 문제: 이진 dirty paper channel `y=x+s+n` (`s~Ber(1/2)` 간섭, `n~Ber(p)` 잡음)에서 송신단이 간섭 `s`를 비인과적으로 알 때 용량 `R(δ,p)=h(δ)-h(p)` 달성하는 실용적 비닝 코드 설계.
2. 기존 한계: superposition coding은 고차 알파벳(비이진) 부호 필요; 본 논문은 이진부호만으로 구현 목표.
3. 코드 구조: LDGM/LDPC concatenation — LDGM의 정보비트 일부는 간섭 근사(lossy source coding), 나머지는 데이터 전송; 데이터는 고rate LDPC로 precoding 해 LDGM의 error floor(저거리 codeword) 제거.
4. 핵심 알고리즘: lossy source coding용 수정 BP — check→variable 갱신에 간섭항 `fi(si,xi)`와 자기강화항 `ξ·vjni(xj)` 추가(식 3), `ξ=0`이면 표준 BP로 환원, block length에 선형 복잡도.
5. 수식 의미: lossy source coding에서 간섭열이 다수 codeword와 등거리라 BP marginal이 1/2로 붕괴하므로, `fi` weight로 check 만족/불만족 구성을 차등 가중해 marginal 강화.
6. 인코딩: 메시지 `d`를 하위코드 `C1`으로 인코딩 후 수정BP로 `c1`에 가장 가까운 `C0` codeword `c0` 선택, 채널에 `c0+c1-s` 입력.
7. 디코딩: 수신 `c+n`을 전역 factor graph에서 표준 BP(`ξ=0`)로 복호 후 bin 매칭으로 `d` 복원.
8. 코드 최적화: 무precoding 시 dual code가 Razaghi-Yu의 bilayer expurgated LDPC와 동형임을 이용, BEC 근사 + density evolution + iterative linear programming으로 degree distribution `λi,j` 최적화(rate 최대화).
9. 결과: 채널 rate 1/2 + source rate 1/8 → DPC rate 3/8, N=1e5, 수정BP 400 iter시 평균 distortion 0.305; BER=2e-5에서 threshold `p*=0.094`로 capacity gap 0.016(superposition coding 수준).
10. 한계: 무선/side-information 채널 전용, regular check로 제한(irregular 시 추가 개선 여지), HW 미설계·시뮬만.
