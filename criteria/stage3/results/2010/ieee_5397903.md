### B. 알고리즘 요약

1. 대상: BSC 채널, 출력을 `{+1,-1}`로 매핑한 irregular binary LDPC (`N=100000`, rate `0.1~0.9`).
2. 문제: 기존 이진 메시지패싱 복호(hybrid MB 등)는 irregular 코드에서 변수노드 차수정보 저장·조회가 필요해 회로 복잡도 증가.
3. 채널 모델: BSC, 초기 오류확률 `p0`, extrinsic error probability(EEP) `p(ℓ)` 기반 반복 분석.
4. 핵심 기법: 변수노드에서 discrepancy `m_k = -u_n·Σ u_{c→v,j}`를 계산하고, `m_k ≥ d`이면 부호 flip. **모든 변수노드에 동일한 flipping threshold `d` 적용** (차수 무관).
5. 핵심 식: 시변 복호의 최적 `d(ℓ+1)`는 `(1-p0)/p0 ≤ ((1+ρ(1-2p))/(1-ρ(1-2p)))^d`를 만족하는 최소값 — 차수정보 없이 산출 가능.
6. 확장: 시변(GB) 복호의 최적 switching point `v_d^{p0}`를 식(10)의 근으로 일반화(Theorem 2.2).
7. 구현 디테일: unsatisfied check node 비율 `P_c`로부터 EEP `p̂`를 1차/2차 Taylor 근사로 추정(식 7,8) — genie 없이 threshold 갱신.
8. 최적화: differential evolution / linear programming으로 degree profile+threshold 공동 최적화, 제약 `d ≤ D_min-1`.
9. 결과: 최적화 threshold가 hybrid MB와 대부분 동일(성능손실 없음); error-floor 회피 위해 `d_max=D_min-3` 설정 시 `10^-4` 이하 floor 제거.
10. 한계: hard-decision(1-bit)·이론/시뮬 위주, 실제 HW(gate count/throughput) 설계·수치 없음.
