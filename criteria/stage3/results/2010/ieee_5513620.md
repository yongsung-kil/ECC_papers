### B. 알고리즘 요약

1. 대상: BSC 상의 3-left-regular(변수노드 차수 dv=3) binary LDPC, 예제 코드 `n=155/R=0.4`(Tanner), `n=768/R=0.75`(QC, dmin=12), `n=4085/R=0.82`(MacKay).
2. 문제: 기존 양자화 디코더는 asymptotic threshold 최적화에 치중 → 유한길이·고SNR(error-floor)에서 취약, 양자화가 error-floor를 악화.
3. 모델: 메시지는 belief의 양자화가 아니라 유한 레벨 집합 `M={0,±Li}`에서 값을 취하는 "레벨"; 채널값 `Y={±C}`.
4. 핵심기법: 디코더를 4-tuple `F=(M,Y,Φv,Φc)`로 정의. `Φc`는 min-sum과 동일(sign XOR × min|mj|). `Φv`는 trapping-set 지식으로 유도한 map.
5. 핵심식: LT 디코더 `Φv=Q(Σmj+yi)` (threshold 집합 `T`로 양자화 `Q`) — 합을 임계로 매핑해 오류 전파 억제.
6. 확장: NLT 디코더 `Φv=Q(Σmj+ωc·yi)`, 채널 가중 `ωc=Ω(m1,m2)`가 비선형 → 합이 같아도 다른 출력 가능(기존 양자 min-sum과 구별).
7. 구현 디테일: 두 대표 디코더 — 7-level LT(`L1<C<2L1`, `L2=2L1+C` 등), 5-level NLT(`C=L1, L2=3L1`, `ωc` δ함수).
8. 유도근거: isolation assumption + isolation theorem으로 trapping-set 유발 부분그래프를 고립 분석, trapping set ontology 계층으로 룰 유도(임계수↑·최소 iteration 수렴).
9. 결과: 3-bit 디코더가 세 코드 모두에서 error-floor 영역 FER 기울기 개선(BP/min-sum 능가); Tanner 코드에서 5-error 전부 정정(LP 디코딩도 실패하는 것을 정정), QC 코드 3-error 패턴을 min-sum(4 iter 필요) 대비 3 iter로 정정.
10. 한계: HW 미설계·시뮬만, BSC(hard) 채널·dv=3 한정, 임의 코드로의 룰 유도 일반화 및 gate/throughput 미제시.
