### B. 알고리즘 요약

1. 대상은 rate-1/2 regular Gallager LDPC(column weight 3), 예제 부호 C1(504,252)/C2(816,408)/C3(1008,504), AWGN 채널.
2. 문제: moderate-length LDPC의 error-floor 영역 초저 BER(예: 10^-16)는 일반 Monte Carlo로 시간상 측정 불가.
3. 모델: error-floor는 near-codeword인 `(a,b)` trapping set(a=bit weight, b=unsatisfied check 수)이 지배한다고 가정.
4. 기반: Cole et al.의 3-step FER 추정(trapping set 탐색 → error boundary 탐색 → importance sampling(IS))을 사용.
5. IS 핵심식 `Pr(x∈D)=∫I(x)(g(x)/g*(x))g*(x)dx` — 시뮬레이션 PDF `g*`로 저확률 사건을 자주 뽑아 분산 감소.
6. 기여1: Cole의 FER 추정을 BER 추정으로 확장(`nb(r)` = error 정보비트 수 포함), unbiased 유지.
7. 기여2: 새 시뮬레이션 PDF `g*_prop`를 dominant trapping set들의 선형결합으로 설계, 각 set을 확률 `Pt≈Q(√(2 d_t^2 Es/No))`(error boundary까지 제곱 유클리드 거리) 비례로 선택.
8. 효과: trapping set 개수를 시뮬레이션 run 수와 독립적으로 정할 수 있어, dominant set 증가에 따른 시간 폭증 회피.
9. 결과: BER 10^-16 추정 시 제안법 시뮬레이션 시간이 Cole-BER 확장 대비 최대 1/6.6(C2), CV 기준 run 수도 감소.
10. 한계: AWGN + mean-translation IS 전제(비가우시안 채널엔 무효), HW 미설계, 부호/디코더 성능 자체를 바꾸지 않는 평가 방법론.
