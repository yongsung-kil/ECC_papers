### B. 알고리즘 요약

1. 대상은 array-based LDPC code `C(γ,p)` (γ×p 순열행렬 배열, p 소수, 열무게 γ, girth≥6). 실험은 (2809,2600),(2209,2024),(1849,1680) 세 부호, 모두 bit degree 4.
2. 문제: 중간 블록길이 LDPC가 저-BER에서 error floor를 보이며, 이는 최소거리가 아니라 Tanner graph의 combinatorial 부분구조(absorbing set)가 지배.
3. 정의: `(a,b) absorbing set`은 bit 부분집합 D(크기 a)로, 각 원소가 satisfied check 이웃이 unsatisfied check 이웃보다 strictly 많은 집합. bit-flipping 하에 stable하면 fully absorbing set.
4. 핵심 도구: bit/check/cycle/pattern consistency 4가지 구조 제약(Lemma 1,2)을 순열행렬 배열 구조에서 유도.
5. Theorem 1(minimality): γ=2는 최소 absorbing set이 codeword(최소거리), γ=3은 (4,4), γ=4는 (6,4)가 최소이며 후자 둘은 최소거리보다 작음.
6. 증명 방식: 후보 (2,b),(3,b),(4,4),(5,4),(6,2) 등이 존재하지 않음을 pattern consistency 합동식(congruence)이 큰 소수 p에서 해가 없음으로 배제(Lemma 5~7).
7. 열거: 존재하는 최소 (fully) absorbing set을 3개 독립 파라미터(q,s,t)로 명시 해집합(Table IV~VII)으로 완전 나열(Lemma 8).
8. Theorem 2(scaling): 최소 (fully) absorbing set 개수가 codeword 길이 N에 대해 Θ(N) (γ=3,4는 Θ(N))로 스케일.
9. 결과: HW emulator로 BER<10^-11 관측, error 대부분이 (fully) absorbing set 수렴이며 minimum-distance codeword가 아님을 실증(Table I~III).
10. 한계: 순수 이론(enumeration/nonexistence 증명)이며 디코더/부호 개선책·HW설계 없음. array-based(비-QC) 구조 전용.
