> 총평: 스페이셜 커플링(LDPCC) 앙상블의 AWGN threshold 점근 분석으로, NAND용 고정 binary QC-LDPC 부호설계와 부호 클래스·디코더 구조가 달라 이식가치 낮음.

### B. 알고리즘 요약

1. 대상: `(J,K)-regular` 및 AR4JA 기반 irregular LDPC 합성곱부호(LDPCC)를 종단(termination)해 만든 블록부호 앙상블.
2. 문제: 이 앙상블의 반복복호 BP threshold가 AWGN 채널에서 얼마나 Shannon 한계에 접근하는지 정량화 필요(기존은 BEC만 증명됨).
3. 모델: protograph + base matrix `B`, `N×N` permutation lifting; 합성곱 base `B_[0,L-1]`, syndrome former memory `ms`.
4. 핵심: edge-spreading으로 블록 protograph를 합성곱 protograph로 분할, 종단 시 끝단 check node degree가 낮아져 BP 성능이 극적으로 개선(threshold saturation).
5. 종단 rate `RL = 1-((L+ms)/L)(1-R)` — L 증가 시 rate가 원래 합성곱 rate로 접근.
6. AWGN threshold는 정확 density evolution 대신 `RCA`(reciprocal channel approximation)로 근사 계산(protograph에 적용).
7. shortened tail-biting 앙상블: tail-biting에서 첫 `ns` 열 제거로 tail-biting~terminated 사이 rate·threshold 연속 트레이드오프 확보.
8. 결과: (3,6)-regular에서 L=10일 때 σ*=0.9638, L→∞에서 σ*=0.948로 블록부호 0.881보다 Shannon(0.979)에 훨씬 근접; AR4JA e=1은 Shannon gap 0.002dB.
9. 트레이드오프: L 증가 시 threshold는 좋아지나 최소거리 성장률 `δmin→0`; 대신 합성곱 free distance 성장률 `δfree`는 constraint length에 선형.
10. 한계: 순수 점근 threshold/거리 이론, HW 미설계, 유한길이 QC-LDPC 구성·시뮬레이션 없음; 실용화엔 sliding-window continuous 디코더 필요.

| protograph lifting / base matrix | `ecc_top.cpp` `Load_PCM()` | H-matrix 로드 개념만 유사, 앙상블/종단 구조는 미대응 |
