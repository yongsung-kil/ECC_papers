### B. 알고리즘 요약

1. 시스템: rate 1/2, length 4000 regular LDPC(행렬 row weight 6, col weight 3)를 OFDM(carrier 128, cyclic prefix 32bit) + 4-path Rayleigh fading 채널에 결합.
2. 문제: 무선 고속 전송에서 주파수 선택적 페이딩·ISI를 줄이면서 near-Shannon 정정능력 확보.
3. 모델: OFDM으로 채널을 M개 병렬 독립 Gaussian sub-channel로 근사, 수신 `r = bx + n`(b는 Rayleigh 진폭).
4. 핵심기법: Gallager H의 several single-parity-check 행을 묶어 "super code"로 구성, 각 super code를 2-state(`0`/`1`) trellis로 표현.
5. 식: 각 super code C(i)는 `cH_i^T=0`을 만족, trellis type I/II 두 섹션으로 codeword 경로를 표현(type II는 crossover branch 제거된 pruned 버전).
6. 확장: LLR 도메인 forward/backward 재귀 `α_k(s)`, `β_k(s)`와 branch transition으로 BCJR형 반복복호를 구성.
7. 트릭: 상태 2개뿐이므로 재귀식을 `⊞`(max*형 연산) 단일 연산으로 단순화해 저장·연산량 축소.
8. 최적화: iterative message passing으로 super-code 디코더 간 extrinsic `λ(i)` 교환, 반복 10회.
9. 결과: 제안 2-state trellis 복호가 BP 대비 성능 "약간" 우수, OFDM 결합 시 BER 크게 개선(수치 곡선은 그림 Fig.7에만 있어 미상).
10. 한계: HW 설계 없음, gatecount/throughput 없음, 무선 OFDM/Rayleigh 특화라 NAND flash 연관성 낮음.
