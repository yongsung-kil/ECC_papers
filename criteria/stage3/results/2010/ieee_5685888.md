### B. 알고리즘 요약

1. 자기기록용 고rate(>0.8, length 4k~5k) 부호로, 성분부호로 SPC 대신 column weight `dv=2`인 QC-LDPC를 쓰는 다중병렬결합(MPC) 스킴 제안.
2. 풀 문제: 기존 MPCSPC(SPC 성분부호)는 Tanner 그래프에 degree-1 parity bit node가 있어 AWGN에서 높은 error-floor(블록길이 무관)를 유발.
3. 부호구성1(BIBD/Netto): 소수체 `GF(6t+1)` 기반 base block `Bi={α^i,α^{i+2t},α^{i+4t}}`로 (v,c,λ=1)-BIBD를 만들어 girth≥6, dv=3 regular QC-LDPC(Netto code) 생성.
4. 부호구성2(array decomposition): weight-3 circulant를 row-splitting+cyclic shift로 weight-2/weight-1로 분해, weight-2 부분만 취해 `dv=2` QC-LDPC 성분부호(예 (2286,2160) rate0.945, (4401,4239) rate0.963) 획득.
5. MPC 인코딩: P개 병렬 branch(각 b개 (Nc,Kc) LDPC) + branch간 random interleaver로 결합, rate `Kc/(Kc+PMc)`; dv=2 성분부호 채택으로 결과 그래프에 degree-1 node가 사라짐(모든 bit node가 iterative 갱신 수혜).
6. MPC 복호: outer(branch간, I회) + inner(성분 LDPC SPA, J회) 이중 스케줄, branch별 extrinsic 정보를 interleave해 a priori로 교환(multiple turbo류 식 7~10).
7. 핵심 효과: degree-1 node 제거로 SPA가 고SNR에서 유효해져 error-floor 회피(BER 1e-7까지 floor 미관측).
8. 결과(AWGN): MPC(P=2)가 대응 MPCSPC 대비 BER 1e-6에서 2.6dB 이득, dv=2 성분부호 자체(Netto dv=3 대비 -1dB 약함)를 써도 MPC 결합 후 Netto 대비 +1.4dB.
9. 결과(ME2PR4 PR채널, BCJR+SPA turbo수신): eRS 기반 best 구조 LDPC와 대등하거나 약간 우수, 또한 P/b/(Nc,Kc)로 rate·length 유연 조정 가능(eRS는 field order 배수 제약).
10. 한계: HW 미구현, 시뮬(BER)뿐이며 random interleaver+outer/inner turbo 스케줄 의존, 성분부호 자체 BER은 약함(결합으로만 이득).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BIBD/Netto QC-LDPC 부호구성·array decomposition | [ecc_top.cpp](../Prime_ECC_3.1_Claude) `Load_PCM()` + H-matrix | H-matrix 교체 필요 → 특정 QC-LDPC 고정이라 재검증 부담 큼(이식 높음) |
| MPC 병렬결합/systematic 인코딩 | [encoder.cpp](../Prime_ECC_3.1_Claude) `LDPC_encoder_4KB()` | 우리는 단일 dual-diagonal 인코더, 다중 branch 결합 구조 아님(대응 약함) |
| CN/VN SPA message passing | [decoder.cpp](../Prime_ECC_3.1_Claude) `LDPC_Decoder()`, `CNU_Update_New_Mag()` | 성분부호 복호는 표준 SPA(우리 min-sum과 다름) |
| outer/inner iteration + random interleaver | 대응 없음 | 우리 디코더는 단일 PCM 파이프라인, turbo류 branch 교환 없음 |
| BCJR 채널 detector(PR채널) | 대응 없음 | NAND read 기반이라 채널 detector 미포함 |

적용 가치: **낮음** — degree-1 회피로 error-floor를 낮추는 부호설계 통찰은 참고되나, BIBD 성분부호+MPC 병렬결합+random interleaver+turbo 스케줄이 우리 단일 고정 QC-LDPC/min-sum 파이프라인과 구조적으로 달라 떼어 쓰기 어렵다.
