### B. 알고리즘 요약

1. 대상: rate-1/2 Gallager regular LDPC (열무게 3), `C1`(504), `C2`(816), `C3`(1008), AWGN + Sum-Product 복호.
2. 문제: error-floor 영역(FER 10^-12, BER 10^-14 수준)은 Monte Carlo로 추정 불가능할 만큼 시뮬 시간이 큼.
3. 모델: error-floor는 trapping set `(a,b)`(a=Hamming weight, b=unsatisfied check 수) 근처 near-codeword가 지배.
4. 핵심 기법: importance sampling으로 원 PDF `f(r)` 대신 시뮬 PDF `g*(r)`로 표본화, `Pr = E[I·f/g*]`로 분산 축소.
5. 최적 PDF: `f*_opt(r)=I·f/Pr` 이면 추정량 분산이 0 — 식(9)~(13)으로 zero-variance 증명.
6. 한계: 최적 PDF는 참 확률 `Pr`을 이미 알아야 하므로 실사용 불가.
7. 제안: 지배 trapping set 집합 `E`에 대해 최적 PDF를 근사, `P̂_e = max_w Q(√(2 d_w² Es/N0))` 상한으로 대체(식17,18,21).
8. 시뮬 PDF `f*_e,prop`은 mean-translation(평균 이동)으로 AWGN에서 각 성분 이동(식23).
9. 결과: C2(n=816)에서 BER 10^-14 추정 시 Cole et al. 대비 시뮬 시간 1/17.5, C1은 FER 1/5·BER 1/2.5로 단축.
10. 한계: 순수 성능 평가 도구 — 복호 알고리즘/부호구조/HW 미개선, AWGN·mean-translation 가정 의존.
