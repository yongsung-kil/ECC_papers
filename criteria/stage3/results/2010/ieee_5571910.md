> 총평: LDPC convolutional code(SC-LDPC)의 임계값 이론 분석으로, threshold saturation 현상의 초기 관측이나 HW·NAND 접점이 없고 부호계열도 달라 Prime ECC 블록 QC-LDPC 이식가치 낮음.

### B. 알고리즘 요약

1. 시스템: rate `R=b/c`(주로 `(3,6)`, rate 1/2) 정규 LDPC convolutional code(LDPCC) 앙상블 `C(M,2M,J)` - permutation 행렬로 구성된 syndrome former, memory `m_s=J-1`.
2. 문제: LDPCC의 iterative 복호 임계값(reliable communication 경계)을 블록 LDPC와 비교해 정량화하려 함.
3. 모델: terminated LDPCC를 특수한 LDPC 블록코드로 보고, BEC/binary-input AWGN에서 belief-propagation 수렴을 density evolution으로 추적.
4. 핵심기법: Tanner graph 시간축 `t`별로 서로 다른 pdf/erasure 확률을 개별 추적(양끝 constraint node degree가 낮아 강한 보호 → structured irregularity).
5. 핵심식: Bhattacharyya 파라미터 recursion `(12)`로 오류확률 상한을 잡고, breakout value 이하로 떨어지면 doubly-exponential 수렴 보장.
6. 확장: `sliding window` 갱신 스케줄로 termination length `L`에 무관하게 임계값 계산(계산량 대폭 절감: 예 40899→688 iteration).
7. 이론보증: Theorem 4/Corollary 5 - 창(window) 스케줄로 `L→∞` 임계값이 유효, unterminated 코드로 conjecture 확장.
8. 결과(BEC): `(3,6)` LDPCC 임계값 0.488(rate 0.5) - 대응 irregular/regular 블록코드보다 우수, ML 임계 상한에 근접.
9. 결과(AWGN): 유사 경향, 놀랍게도 parity-check 밀도(J)를 높이면 BP 임계값이 오히려 개선(블록코드와 반대).
10. 한계: 순수 점근 이론(HW/gate/throughput 없음), rate 1/2만 분석, sliding window의 AWGN 유효성은 conjecture.
