### B. 알고리즘 요약

1. 대상: 무한 블록길이 극한에서 irregular LDPC ensemble의 variable/check node degree distribution `(λ(x), ρ(x))` 설계 (BIAWGN, rate 0.5 기준 예시).
2. 문제: 기존 최적화 기반 설계(density evolution + 다변수 탐색)는 variable node degree 종류가 많아지면 계산량이 큼.
3. 모델: BIAWGN 채널, parallel BP 스케줄. rate `R = 1 - dv/dc` (식 3), check 분포는 단일 파라미터 `dc`로 표현(식 4).
4. 핵심기법: rate 최대화를 `dv^{-1} = Σ λi/i` 최대화로 환원 → 낮은 degree(λ2,λ3)에 edge 비중을 greedy하게 몰아줌.
5. 핵심식: 안정성 조건 `λ2 ≤ e^{1/2σ²}/ρ'(1)`로 `λ2,max` 결정; 이것이 상한을 넘으면 발산.
6. 확장(경험적 발견): `λ2=λ2,max` 고정 시 `λ3`에도 상한 `λ3,max`가 존재함을 실험으로 보임 → Algorithm 1로 DE 반복하며 사전계산해 lookup table화 (Fig.1, dc·σ² 함수).
7. 구현 디테일: DE는 [-50,50] 범위 10-bit quantized density로 수행; δ2≈0.002-0.003, δ3≈0.001로 상한에서 약간 빼 수렴속도 조정.
8. 설계 절차: Algorithm 2(주어진 σ²에서 최대 rate) / Algorithm 3(주어진 rate에서 최적 threshold) — degree 2,3,Dv와 필요시 중간 degree j 하나만 사용.
9. 결과: rate 0.5 예제에서 최적화 기반 [2] 대비 threshold 열화 대부분 ≤0.01dB(Dv<15), 큰 Dv도 ≤0.03dB; constituent degree 수는 3~5개로 훨씬 적음.
10. 한계: 점근적 ensemble 설계일 뿐 구체 H-matrix 구성/디코더/HW 없음; BIAWGN 전용, 유한길이·error-floor 미고려.
