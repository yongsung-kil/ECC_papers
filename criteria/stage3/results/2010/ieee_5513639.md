### B. 알고리즘 요약

1. 대상: regular QC-LDPC 블록부호를 rate R=b/c 모부호(parent convolutional code)의 tailbiting(TB)으로 생성, rate 2/5·3/6, 길이 N=Mc (144~980).
2. 문제: LDPC는 최소거리가 작아 BP 성능이 제한 - 큰 girth와 큰 minimum distance를 동시에 갖는 짧은/중간 길이 QC-LDPC를 찾고자 함.
3. 모델: parity-check matrix를 하이퍼그래프의 incidence matrix로 해석, monomial `h_ij(D)=D^{w_ij}`의 degree matrix `W`로 QC 구조를 표현.
4. 핵심기법: basic Tanner graph `G_BT`의 짧은 cycle들을 edge voltage `μ_uv = w_vk - w_uk` 방정식으로 나타내고, `μA^T ≡ 0 mod M`을 만족하는 voltage labeling·TB length M을 무작위 탐색해 girth g를 보장.
5. 의미: voltage Tanner graph의 girth `g_VT`는 voltage-zero 최소 cycle 길이이며 `g_VT = 2 g_free`, 이를 통해 부호 girth를 대수적으로 설계.
6. 확장(경계): `d_min <= d_free`, `g <= g_free`(정리1), `g <= 12`, `d_min <= (c-b+1)!`(정리3) 상한과 `g_T >= 2·max{...}`(정리2) 하한 제시.
7. 트릭: TB로 다양한 rate·길이 부호를 하나의 모부호에서 파생, dual generator matrix 기반 저복잡도 인코딩 가능.
8. 탐색: minimum distance는 `HTB` 컬럼합=0인 최소 컬럼수로, root별 트리 탐색(깊이 t-1 제한, 상태벡터 nonzero 개수로 pruning)으로 결정.
9. 결과: girth 8/10/12, minimum distance 최적 24(대부분)인 신규 QC-LDPC 획득, BP 60 iteration BER 시뮬에서 girth·length 클수록 성능 향상, edge voltage 선택이 성능 결정에 가장 중요.
10. 한계: 저rate(0.4~0.5)·짧은 부호뿐, HW 미설계, AWGN 시뮬만.
