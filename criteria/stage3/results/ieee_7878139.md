### B. 알고리즘 요약

1. 대상: AWGN/Rayleigh 채널용 고rate binary LDPC (rate 0.67~0.93, length 1323~3920), SPA 복호(max 200 iter).
2. 문제: PEG 등 컴퓨터 기반 구성은 긴 부호 생성 비용이 크고, algebraic 구성만으로는 성능 한계 — 두 방식을 결합(hybrid)하려 함.
3. 성분부호 1: (7,4) cyclic Hamming을 column splitting으로 (14,8) 확장 Hamming(EHC)으로 변환, 4-cycle 제거. column weight 2.
4. 성분부호 2: (7,7,3,3,1) BIBD의 7x7 circulant 관계행렬(incidence matrix), null space가 (7,3) cyclic LDPC. column weight 3.
5. 핵심 아이디어: base matrix `H1`의 각 1-성분을 (14,8)EHC 또는 BIBD의 한 행으로 치환, 0-성분은 14/7-tuple 영벡터로 치환.
6. 크기 관계: base가 `m×n`이면 hybrid는 `m×14n` 또는 `m×7n`. rate = `(14n-r)/14n` (r = parity 수).
7. base matrix 생성법 3종: PEG(PH), Gallager 원본(GH), difference-set(DSH). column weight 2면 마지막에 단위행렬 행 추가.
8. 근거: 성분부호가 모두 4-cycle-free이므로 결합 부호도 4-cycle-free([11] 증명), 짧은 사이클 수 표로 검증.
9. 결과: PH*가 GH*보다 우수, PEG 부호와 유사하나 Eb/N0 15dB 부근 소폭 우위; BIBD+PEG는 AWGN에서 Shannon limit 2dB 이내. avg iteration은 PH*가 15 이내로 빠른 수렴.
10. 한계: HW 미설계, SPA 시뮬만, Rayleigh/AWGN 가정, QC-LDPC 아님, error-floor 일부 관찰(3920 코드).
