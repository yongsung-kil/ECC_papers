> 총평: LP decoding 프레임워크 내에서 fractional distance(BSC 오류정정 보장 지표)를 redundant row 추가로 개선하는 이론적 구성법. Prime ECC의 min-sum 기반 디코더와 근본적으로 다른 복호 패러다임이라 이식성 낮음.

### B. 알고리즘 요약

1. 대상은 이진 선형부호 일반(Hamming[7,4,3], Golay[24,12], 짧은 (3,6)/(4,8)-regular LDPC n≤204)이며, 복호기는 min-sum/BP가 아닌 Linear Programming(LP) decoding.
2. LP decoding 성능은 fundamental polytope `P(H)`의 기하적 성질(fractional distance `dfrac(H)`)에 좌우되는데, 같은 부호라도 parity-check matrix `H` 표현에 따라 `dfrac(H)`가 달라지는 문제를 다룸.
3. 채널모델은 BSC(binary symmetric channel), `dfrac(H) ≥ 2e+1`이면 LP 디코더가 `e`개 이하 비트반전 오류를 항상 정정한다는 Feldman의 정리(Lemma 1)를 전제.
4. 핵심 기법: cutting-plane method — 원본 `H`의 dual code에 속하지만 아직 행으로 없는 "redundant row" `h`를 추가해 `H'=[H;h]`를 만들면 `P(H')=P(H)∩U(h)`로 다각형이 좁아져 fractional vertex를 제거.
5. 핵심 식 `dfrac(H) = min_{f∈V(P(H))\0} Σfi` (fractional distance = 최소weight 0이 아닌 꼭짓점의 1-weight) — redundant row 추가로 이 최소값이 커지면 정정보장 오류수 `e`가 늘어남.
6. 확장 기법: greedy cutting-plane algorithm — `Γ(H)`(최소weight 꼭짓점 집합)에서 점 `p`를 뽑아 이를 잘라내는 redundant row `h`를 찾아 반복적으로 `H`에 추가.
7. 구현 디테일: `dfrac(H)` 계산을 fundamental cone `K(H)` 기반 relaxed LP(`d_frac^relax`)로 대체해 계산량 절감(Theorem 2), redundant row 탐색도 `Supp(p)` 관련 부분행렬 `H_Q`의 row-echelon 변환으로 탐색공간을 브루트포스 `2^{(1-R)n}`에서 `|Q|`로 축소(Theorem 3).
8. 최적화 절차: LP solver(GLPK, simplex)를 반복 호출하는 것 외 별도 학습은 없음; greedy하게 redundant row를 하나씩 추가.
9. 결과: Golay code에 100개 redundant row 추가 시 fractional distance 2.625→3.895, block error probability가 crossover prob 10⁻²에서 약 2자리수 개선; LDPC(204,102) 9개 row 추가로 dfrac 5.526→5.646, 에러곡선 기울기 개선(steeper slope).
10. 한계: 계산복잡도 때문에 n≤204까지만 검증(장문 부호는 dfrac 자체 계산이 어려움), 대상 디코더가 LP decoding 전용이라 min-sum/BP 계열 HW 디코더에는 직접 적용 불가, HW 설계·throughput 관련 내용 전무.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LP decoding 알고리즘 전체 | 대응 없음 | Prime ECC는 min-sum 기반 BP 계열 디코더만 보유, LP solver 자체가 없음 |
| redundant parity-check row 추가(H-matrix 구성) | `ecc_top.cpp` `Load_PCM()` | H-matrix 구성 관점에서만 개념적 유사성, 실제 QC-LDPC 순환치환 구조와는 무관한 임의 redundant row라 이식 난이도 높음 |
| fractional distance 평가/최적화 | 대응 없음 | Prime ECC는 min-distance/trapping-set 기반 평가를 쓰지 않고 시뮬레이션 기반 BER 평가만 수행 |

> 적용 가치: 낮음 | LP decoding 전용 이론(fractional distance)이며 Prime ECC의 min-sum BP 디코더 및 QC-LDPC 순환구조와 접점이 거의 없음. 검증 규모도 n≤204로 NAND ECC 실사용 규모(수천 bit 이상)와 거리가 멂.
