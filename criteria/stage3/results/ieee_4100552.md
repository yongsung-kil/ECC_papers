### B. 알고리즘 요약

1. DTV(SBTVD) 방송 채널(AWGN 가정)을 대상으로 codeword length 9792, rate 1/2·2/3·3/4·5/6·7/8의 eIRA(extended irregular repeat-accumulate) LDPC 부호를 설계.
2. 기존 표준(DVB-T, ISDB-T)의 convolutional code 대비 오류정정 성능을 개선하려는 것이 목표.
3. 채널 모델은 BPSK + AWGN, 디코더는 sum-product algorithm(SPA) 기반.
4. 부호 설계는 두 단계: (1) `A(x)`, `p(x)` 차수분포 최적화(density evolution의 semi-Gaussian 근사, linear program), (2) H1 서브행렬을 PEG(progressive edge-growth) 알고리즘으로 구성해 짧은 사이클 최소화.
5. H2는 dual-diagonal(accumulator) 구조로 고정하여 저복잡도 systematic 인코딩 가능(`G = [I_k | H1^T H2^-T]` 유도).
6. 최적화는 linear program으로 환원: 주어진 noise threshold·length·rate에서 `A(x)`를 반복적으로 조정.
7. HW 구현: 인코더는 Altera Stratix II FPGA(VHDL), 디코더는 Xilinx Virtex II XC2V3000 FPGA(VHDL+System Generator)에 각각 별도 구현. 디코더는 constant correction term 근사 사용, serial 구조.
8. 학습/최적화 절차: density evolution + semi-Gaussian 근사 기반 linear program으로 차수분포 결정(Table I에 rate별 결과 제시).
9. 결과: 설계된 (9792,4896) rate-1/2 코드가 BER=10^-5에서 Shannon limit 대비 0.7~1dB, 기존 convolutional code 대비 약 3dB 우수. 인코더는 FPGA 면적 4~5%만 사용(저복잡도). 디코더는 clock 97MHz, throughput 1.933Mbps(본문 표기, 단위 오탈자 가능성), block RAM 96%·slice 61% 사용, 메시지 5비트 양자화, 최대 iteration 13회.
10. 한계: 채널이 AWGN(방송) 전제이고 NAND read 채널·soft-read 비용과 무관. 부호 자체는 표준적인 eIRA 구성법(교과서 수준)이며, 이식 가능한 신규 정정능력/error-floor 개선 기법은 제시되지 않음. 본문 텍스트(OCR)가 하드웨어 구현 세부 문단에서 심하게 손상되어 정확한 수치 확인이 어려움.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| eIRA 부호 구성(H1 랜덤+PEG, H2 dual-diagonal) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 QC-LDPC 고정 구조라 eIRA류 비-QC 구성법은 재검증 부담 큼 |
| dual-diagonal 인코더 구조 | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | Prime ECC도 dual-diagonal 인코딩을 이미 사용 중이라 개념 중복 |
| sum-product(SPA, tanh 기반) 디코더 | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | Prime ECC는 min-sum 기반이라 SPA는 이미 min-sum으로 대체된 상위 개념 |
| density evolution 기반 차수분포 최적화 | 대응 없음 | 부호설계 단계 이론이며 Prime ECC 코드베이스에 대응 모듈 없음 |
| FPGA HW 구현(serial decoder, memory) | 디코더 HW 모델(z=32, HCU, GT) | 구체 아키텍처(파이프라인, 병렬도)가 상이해 직접 대응 어려움 |

적용 가치: `낮음` — AWGN 방송 채널 대상 eIRA 부호의 표준적 설계·FPGA 구현 사례로, NAND binary QC-LDPC에 이식할 신규 디코더 알고리즘이나 정정능력 개선 기법이 없음.
