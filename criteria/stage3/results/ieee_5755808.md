### B. 알고리즘 요약

1. 대상은 binary LDPC 코드의 LP(linear-programming) 디코딩으로, 채널은 binary-input memoryless channel(LLR `λi=log(PY|X(yi|0)/PY|X(yi|1))`)을 가정한다.
2. 문제의식: 범용 LP solver로 LP 디코딩 문제를 풀 수 있으나 특수 구조(패리티체크 코드의 convex hull `conv(Cj)`)를 활용하지 않아 비효율적이라는 점.
3. Primal LP(`PLPD1/2`)는 Feldman의 relaxation `x∈conv(Cj), j∈J`으로 정의되며, Forney-style factor graph(FFG)로 표현해 각 체크노드/변수노드를 convex hull 제약으로 대체한다.
4. Dual LP(`DLPD2`)를 유도하고, `min` 연산을 soft-minimum `min^(κ)(z)=-log(Σexp(-κz))`으로 대체한 softened dual `SDLPD2`를 제시 — `κ→∞`이면 원래 dual과 일치.
5. 핵심 식: coordinate-ascent 갱신규칙 `u'_{i,j}=(1/2κ)((S_{i,0}-S_{i,1})-(T_{j,0}-T_{j,1}))`(Lemma 3)은 sum-product 알고리즘의 메시지 갱신과 (부호 반전 제외) 동일한 계산량을 가짐을 증명.
6. `κ→+∞` 극한(Lemma 5)에서는 갱신규칙이 min-sum 알고리즘과 유사한 형태(구간 내 임의값)로 단순화되며, Hamming weight≥3 조건 하에 순환 갱신 시 SDLPD2의 최댓값으로 수렴함을 증명(Lemma 4, block coordinate-ascent/Gauss-Seidel 근거).
7. 보조 알고리즘(Decoding Algorithm 2)으로 incremental sub-gradient method를 제시 — 매 iteration마다 체크노드/변수노드를 순차 업데이트하며, 목적함수가 매 스텝 증가함을 보장하지 않지만 적절한 스텝사이즈 수열로 수렴 보장.
8. 최적화 절차는 별도 학습 없이 좌표상승/sub-gradient의 순수 수학적 반복이며, softened LP는 Bethe free energy 최적화 문제와 밀접히 연관됨을 부가 설명(entropy 항 `H(αi)`, `H(βj)` 포함).
9. 결과: (3,6)-regular [1000,500] LDPC(4-cycle 제거)에서 LPD(최대 256 iteration)를 표준 min-sum(64/256 iteration)과 비교한 시뮬레이션(Fig. 3)만 제시, 구체적 BER/WER 수치는 본문에 명시되지 않음(그림에만 존재).
10. 한계: 단일 소규모 코드에 대한 proof-of-concept 시뮬레이션뿐이며, 수렴 시 pseudo-codeword(분수값 `x̃`)가 나올 수 있어 실제 codeword 미보장, HW 구현/양자화/조기종료 등은 "향후 연구 방향"으로만 언급.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| dual LP coordinate-ascent 갱신규칙 (sum-product/min-sum 유사) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 계산 형태가 min-sum과 유사하나 LP relaxation·pseudo-codeword 개념은 Prime ECC의 이산 message-passing과 다른 수학적 프레임 |
| CN 연산(soft-min 근사) | `decoder.cpp` `CNU_Update_New_Mag()` | soft-min이 min-sum의 min 연산과 형태적으로 유사하지만 LP 쌍대성 기반 도출이라 직접 이식 시 재검증 부담 큼 |
| sub-gradient 기반 디코딩 알고리즘 | 대응 없음 | LP 최적화 이론 특화 기법으로 HW 파이프라인 대응 없음 |
| pseudo-codeword / stopping set 판별 | 대응 없음 | Prime ECC는 CRC 기반 조기종료(`partial_CRC.cpp`)만 지원, pseudo-codeword 개념 미보유 |

> 적용 가치: 낮음 — LP 디코딩 이론(pseudo-codeword, dual polytope) 프레임 특유의 개념으로 Prime ECC의 실전 min-sum HW 구조에 직접 이식하기엔 재검증·구조 변경 부담이 크고 실증 성능 이득도 불명확.
