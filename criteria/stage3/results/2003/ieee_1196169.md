### B. 알고리즘 요약

1. 시스템: 40Gb/s/채널 CSRZ 변조 WDM 광전송, 100GHz 채널간격, 6개 인접채널 간섭 가정.
2. 문제: 광 초장거리 전송에서 turbo/기존 FEC 대비 더 강력하고 고속 구현 가능한 부호가 필요.
3. 채널 모델: AWGN 가정 대신 ASE noise, 섬유 비선형(Kerr), 잔여분산, crosstalk, ISI를 포함한 nonlinear Schrödinger 기반 현실적 모델.
4. 핵심 기법: 사영기하 `PG(m,q)` 점-선 결합행렬을 LDPC parity-check로 사용, `GF(q)` 임의 소수거듭제곱 order로 일반화(기존 GF(2) 한정 확장).
5. 구조: 각 행 weight `q+1`, 임의 두 열이 정확히 하나의 "1" 공유 → Tanner graph girth 4 cycle 없음(large minimum distance).
6. 부호 예: `PG-LDPC(1057,813)`, rate ~0.77(redundancy 30%), `m=2, q=32` (2^5) 구성.
7. 복호: sum-product 대신 정규화 min-sum(normalized min-sum) 사용 - 덧셈·최소연산만 필요, noise 표준편차 지식 불필요 → 고속·복잡 광채널 통계에 적합.
8. 검증: Monte Carlo BER 시뮬레이션, 길이 10^? bit sequence(본문 지수 손상).
9. 결과: BER 10^-? 에서 coding gain ~10dB 확보, 기존 제안 대비 우수.
10. 한계: HW 미설계, 광통신 채널 특화, analog VLSI 구현은 향후 과제로만 언급.
