### B. 알고리즘 요약

1. 시스템: BI-AWGN 채널, rate 1/2 irregular LDPC (10000,5000) 및 short (1264,456), 최대 VN degree 20/14, BP 복호 최대 200 iteration.
2. 문제: irregular 부호는 density evolution threshold에 근접하나 BER 10^-6 부근에서 error floor 발생. girth conditioning은 짧은 cycle을 모두 제거해 부호구조를 과도 제약(고degree 노드에서 특히 문제).
3. 핵심 통찰: BP 복호의 error floor는 작은 stopping set이 주원인. 모든 stopping set은 cycle을 포함(Lemma1), Ld(linearly dependent) set은 반드시 Sd stopping set(Thm2) → 작은 stopping set 방지가 곧 작은 dmin 방지.
4. 핵심 지표: extrinsic message degree(EMD) = 노드집합에 한 번만 연결된 constraint 수. stopping set의 EMD=0. EMD 큰 cycle은 stopping set 되려면 추가 'closure' 노드 필요.
5. 근사 지표(ACE): 길이 2d cycle의 ACE = `Σ(di − 2)` (di=i번째 VN degree). constraint sharing 무시한 계산가능 근사. `(dACE, η)` 성질 = 길이 2dACE 이하 모든 cycle의 ACE ≥ η.
6. 알고리즘: column을 degree 내림차순 배치(정보비트 보호). 각 VN 생성 시 parity면 Gaussian Elimination으로 full-rank 보장, ACE < η인 짧은 cycle 발견 시 재생성(goto redo).
7. 구현: ACE 탐지를 support tree/trellis 상의 Viterbi-유사 pruning으로 수행 — 각 노드에 최소 path ACE만 저장해 지수→선형 비용. 복잡도 상한 `dACE × n × (d−1)`.
8. 최적화: degree η+2 이상 VN은 자동으로 ACE≥η 만족해 생성 생략 → 런타임 개선. 10^5 bit 블록에서도 실용적.
9. 결과: (dcyc,dACE,η) 튜닝, 최적 dACE≈9,η≈4. (1,9,4) 구성이 error floor를 BER≈10^-9로 억제(무조건 대비 수 자릿수↓), RU 대비 waterfall penalty 0.07dB, MacKay regular 대비 0.6dB 우수.
10. 한계: HW 설계 없음, irregular 랜덤부호 구성용(QC/circulant 구조 미고려). low-error-floor 부호는 waterfall에서 소폭 penalty.
