### B. 알고리즘 요약

1. AWGN 채널, sum-product(belief propagation) 반복 복호를 대상으로 한 관찰적 분석 논문.
2. 문제: 최적 PCM 구성법이 미확립이라, 반복 복호 중 나타나는 bit probability transition 특성을 분석해 설계 파라미터를 찾고자 함.
3. 각 열당 1이 3개인 `30x50` regular PCM, 반복 `300`회로 단순화하여 시뮬레이션.
4. 충분한 반복 후 bit probability transition이 3가지 Type으로 분류됨: Type I(0 또는 1로 수렴), Type II(0/1 아닌 값으로 수렴), Type III(0~1 사이 진동).
5. 같은 codeword 내 각 bit은 동일 transition type을 보임 - Tanner graph로 상호 연결된 결과로 해석.
6. Type II는 저SNR, Type I은 고SNR에서 지배적; Type III는 4dB까지 증가 후 감소.
7. transition type은 채널 종속 초기확률 분포와 무관하고, PCM 구조에 의존한다고 결론.
8. 특정 bit node(3,5,16,20)가 오류에 취약; 이들은 최고차(degree 7) 검사노드 16에 연결되고 6-cycle에 위치.
9. bit node 20을 저차 검사노드 2로 이동(6-cycle·최고차 회피)하니 해당 노드 오류가 550->17로 감소, BER `10^-6`에서 약 `0.7dB` 이득.
10. 한계: 초소형 PCM·시뮬만, HW 미설계, 정성적 관찰 위주로 체계적 부호설계 알고리즘 없음.
