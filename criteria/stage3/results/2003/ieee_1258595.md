> 총평: 비균일 병렬채널(VHM)용 subchannel별 degree 분포 최적화 + error-floor 회피 설계법. NAND의 위치별 RBER 편차와 개념적 유사성은 있으나 irregular·랜덤 부호설계라 QC-LDPC 기반 Prime ECC에는 직접 이식 어려움.

### B. 알고리즘 요약

1. 시스템: 길이 `n` 부호 1개를 `kr`개 독립 binary-input 대칭 subchannel에 나눠 전송, `j`채널에 `n(j)`비트(`pj=n(j)/n`). VHM 페이지를 위치별 노이즈로 `kr`영역 분할해 병렬채널로 모델.
2. 문제: 채널마다 별도 부호 대신 단일 LDPC로 비균일 오류보호, 기존은 interleaver+RS 방식(VHM 용량 50%↑ 여지).
3. 핵심 아이디어: bit type `j`별로 다른 variable-node degree 분포 `λ^(j)(x)`를 갖는 bipartite graph ensemble `g(Λ,ρ)` 정의 — 각 비트가 어느 채널을 타는지 사전지식을 부호설계에 반영.
4. 핵심 도구: type별 density evolution `Pl^(j)=P0^(j)⊗λ^(j)(Ql)`, `Pl=Σq^(j)Pl^(j)`, `Ql=Γ^-1(ρ(Γ(Pl-1)))`.
5. Lemma 1: 임의 두 채널 `i,j`의 오류확률이 동시에 0으로 수렴 → 채널별 개별 최적화 불필요, 설계 단순화.
6. 안정성(Theorem 1): BEC 등가 `ε=Σpjεj`, 행렬 `A`(`αji=λ'^(j)(0)ρ'(1)q^(i)x0^(j)`)의 spectral radius `r(A)<1`이면 오류확률 0 수렴.
7. Rate 상한(Theorem 2): BEC에서 `1-R ≥ 1-Φ(1-ε̄)`, `Φ`는 check degree 분포.
8. 핵심 설계: **semi-regular** ensemble `g(D,ρ)` — 같은 type 노드는 동일 degree, check는 인접 두 값 집중. error-floor 회피 위해 `λ2·ρ'(1)<1` 강제 및 degree-2 노드 회피(작은 stopping set 제거, expurgation).
9. 결과: BEC 반rate `n=10^4` 부호 B가 code A/C와 달리 BER 10^-9까지 error-floor 무관측; BIAWGN VHM(`kr=4`, rate 0.85, `d1..d4=3,4,7,10, dc=40`)에서 `n=10^5`는 capacity 0.65 dB, `n=10^4`는 1.04 dB, 10^-9까지 floor 없음.
10. 한계: HW/디코더 미설계, SPA/DE 시뮬만, VHM·BEC·OFDM·multilevel 특화 채널 가정, irregular 랜덤 부호(QC 아님).
