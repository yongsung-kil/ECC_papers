### B. 알고리즘 요약

1. 대상: 구조화된 regular `(n, 2, k)` LDPC 부호(column weight `j=2`), AWGN 채널, sum-product 복호.
2. 문제: Tanner 그래프의 짧은 cycle이 sum-product 수렴을 방해 → girth를 키워 고SNR 성능 확보.
3. 핵심 모델: `j=2` 부호의 H matrix 각 열을 두 점 사이 edge로 보는 "structure graph" `G_H`로 cycle을 기하학적으로 식별.
4. 기법1(girth 12): 점집합 `X`를 두 부분집합 `X1,X2`로 나눠 서로 다른 부분집합 간 edge만 허용 → `(4m+2)`-cycle 자동 배제(Fact 1).
5. 핵심 변수: edge의 slope `s = j - i`, scissors width `w = |sij - siq|`, double-scissors의 base로 8-cycle 조건 판정.
6. 8-cycle 제거: admissible slope set `A`를 선택해 base >= p 조건을 강제 → girth 12 부호 구성(Construction 1).
7. 기법2(girth 16/20): 점을 8개(또는 10개) 부분집합의 "cylinder structure"로 배치, section별 admissible slope set을 "Viterbi-like" 탐색으로 결정.
8. 결과 rate: girth 12는 `v`에 따라 rate `3/4~8/9`, girth 16은 rate `1/2`, girth 20은 rate `1/3` 고정.
9. 성능: 고SNR에서 girth 12/16/20이 random 부호 대비 각 0.5/1.1/0.8 dB(@BER 1e-5) 우수, 저SNR에선 random이 더 좋기도 함.
10. 한계: `j=2`라 `dmin = g/2`로 작고, HW 설계 없음, sum-product 시뮬만, RS 연접 저장용으로 제안됨.

| girth 최적 H-matrix 구성(slope/scissors) | `ecc_top.cpp` `Load_PCM()` | H-matrix 설계 레이어에만 닿음, 그러나 j=2·저rate라 Prime의 고rate QC-LDPC와 부정합 |
