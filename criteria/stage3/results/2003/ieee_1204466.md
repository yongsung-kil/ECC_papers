### B. 알고리즘 요약

1. 코드: SFT(Sridhara-Fuja-Tanner) permutation 행렬 기반을 확장한 irregular QC-LDPC "IPP"(Irregular Partitioned Permutation), 다양한 rate(1/4,1/3,1/2,3/4,9/10)/block size, 예시 rate=1/2 H=2316x4632 (macro 12x24, m=193).
2. 문제: 기존 LDPC HW는 단일 코드/소수 코드에만 고정, block size·rate 유연성과 고병렬을 동시 만족하는 구성+디코더가 없음.
3. 구성: 최적 degree distribution에 맞춰 j×k macro matrix를 0/1로 채우고, 1을 circulant-shift permutation(m×m), 0을 zero 블록으로 치환.
4. 디코더: log-BP (min이 아닌 `Ψ(x)=log(tanh(x/2))` LUT 기반), 두 배열 `L(qmj)`, `Rmj`를 행/열 2-pass로 갱신 (식 1-5).
5. 핵심 식: `Rmj = -smj·Ψ(Σ Ψ(L(qmn)))` (식 3) — check node 갱신을 Ψ-LUT + adder tree로 구현; 대안으로 pairwise `[+]` LLR 덧셈(식 7,8).
6. 아키텍처: R(row) 메모리, parallel adder, PCUB(parity check update, Ψ-LUT bank+augmented adder tree), router/reverse router, bit update block(column-sum ping-pong 메모리 A/B).
7. HW 친화 트릭: circulant-shift 덕에 컬럼 주소지정이 단순 counter(modulo m wrap), 행이 순차면 열도 순차 → 랜덤 주소/permutation generator 불필요([6] 대비 단순).
8. 병렬화: single-row 내 병렬 + w-row 병렬(연산블록 w배 복제로 속도 w배), macro 컬럼을 9 section으로 그룹핑해 메모리 수 72→27로 축소, 라우터도 단순화.
9. 결과: IPP irregular code가 유사 크기 regular(RV/MacKay) 대비 10^-5 이상에서 유의미하게 우수, 3G turbo에 근접 (BER 곡선, 수치는 그림에만).
10. 한계: gatecount/throughput 실측 없음, HW는 개념 아키텍처(시뮬 BER만), rate 변경 시 row 길이 변화로 HW 공유·재구성 부담.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| IPP QC-LDPC (SFT permutation 확장) 구성 | `ecc_top.cpp` `Load_PCM()` | Prime ECC도 base+circulant QC-LDPC — 구성법 개념 참고 가능, 단 특정 H 고정으로 재검증 부담 |
| circulant-shift 순차 컬럼 주소지정 | `decoder.cpp` `LDPC_Decoder()` | z-block 순차 처리와 유사 개념, 주소지정 단순화 참고 |
| PCUB (Ψ-LUT + adder tree) CN 갱신 | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC는 min-sum, 논문은 full Ψ(tanh) LUT — CN 방식이 다름(면적 불리) |
| 블록열 그룹핑 라우팅 축소 / 메모리 수 감축 | `GT.cpp` `Make_GT_HW()` | Graph Thinning의 edge 선택과 목적 유사(라우팅/메모리 절감) — 대응 관심 요소 |
| w-row 병렬화 | 디코더 HW 모델 (z=32 row 병렬) | Prime ECC row 병렬(z_sb=32)과 정합성 검토 대상 |

적용 가치: **중간** — QC-LDPC circulant 주소지정·블록열 그룹핑 라우팅 축소는 NAND QC-LDPC HW와 개념 정합하나, CN 갱신이 Ψ-LUT(min-sum 아님)이고 rate/검증이 통신급이라 직접 이식보다 아키텍처 아이디어 참고 수준.
