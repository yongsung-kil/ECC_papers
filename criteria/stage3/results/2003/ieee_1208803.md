### B. 알고리즘 요약

1. 대상 부호: PAA(Parity Accumulator Accumulator) — 외부 부호=단일 패리티체크(SPC), 내부=두 개의 accumulator를 직렬 연접한 고rate 부호(RA 유사, storage device 겨냥).
2. 문제: 기존 PAA는 rate가 SPC로 고정되어 다양한 rate 시스템에 부적합.
3. 제안: SPC를 Etzion의 multi-rate cycle-free 부호(예 [10,8], [11,8])로 치환해 다중 rate PAA 부호를 구성.
4. 복호: factor graph에 sum-product 알고리즘 적용, tree representation으로 정보/패리티 노드 메시지 갱신(`M(0)=a+b+y(0)` 등).
5. CN 갱신은 표준 `tanh(u/2)=∏tanh(u_j/2)` 사용, VN 갱신은 incoming LLR 합.
6. 분석: Gaussian approximation으로 AWGN 채널에서 noise threshold 계산 (변수/체크 노드 차수분포 λ,ρ,γ 유도).
7. 대칭 Gaussian 가정(`σ²=2m`)으로 평균 m만 추적, `φ(m)` 함수로 accumulator별 평균 갱신식 유도.
8. threshold = iteration→∞일 때 bit error 확률이 0으로 가는 최대 noise level.
9. 결과: 정보 블록 길이 4096, iteration 10/20 BPSK 시뮬에서 rate 8/9~8/15 PAA가 Shannon limit에 근접.
10. 한계: HW 미설계, 순수 부호구성+threshold 이론, non-QC 구조라 NAND ECC 이식 부적합.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| PAA/cycle-free 부호구성 | 대응 없음 | QC-LDPC(base+shift) 구조와 무관, RA 계열 |
| sum-product(tanh) CN 갱신 | 대응 없음 | Prime ECC는 min-sum 근사, full-tanh 미사용 |
| Gaussian approximation threshold | 대응 없음 | 부호설계 이론 분석, 코드 대응 모듈 없음 |
```

적용 가치: **낮음** — RA/PAA 직렬연접 부호 구성법+threshold 이론으로 binary QC-LDPC 디코더/HW/부호설계 어느 레이어에도 이식 대상 대응이 없다.
