### B. 알고리즘 요약

1. 대상: `GF(p^s)` 위 (n,2) shortened RS 부호(정보심볼 2개, 최소거리 n-1)를 base로 한 대수적 LDPC 구성.
2. 문제: 랜덤 LDPC(MacKay) 대신 구조적·girth 보장된 regular LDPC를 대수적으로 만들고 싶다.
3. 핵심: RS 코드워드 각 심볼을 location vector(`GF(2)` q-tuple, 단일 1)로 치환 → 각 coset마다 permutation matrix 행으로 된 symbol location matrix `A_i` 생성.
4. 두 서로 다른 `A_i`, `A_j` 행은 1-성분을 최대 1개만 공유 → 이들을 쌓은 `H`는 (γ,ρ)-regular이며 4-사이클(직사각형 4-corner) 없음.
5. 결과 Tanner graph는 girth≥6, `H`는 Gallager 원형의 (γ,ρ)-regular 패리티체크행렬.
6. 최소거리: Massey 직교성 정리로 dmin ≥ γ+1(홀수) / γ+2(짝수) 보장 → error-floor 억제.
7. QC 조건: `k=q-1`이면 부호가 quasi-cyclic(HW/인코딩 친화).
8. rate는 최소 (ρ-γ)/ρ 이상, γ·k 변화로 다양한 길이/rate 부호열 생성.
9. 결과(시뮬, BPSK/AWGN, SPA): (2048,1723) rate0.841은 Shannon 1.55 dB, (4032,3307) QC rate0.82 dmin≥62로 error-floor 거의 없음, (8192,6754), (12288,10845) 등 MacKay 랜덤부호와 대등.
10. 한계: 부호구성+SPA 시뮬만, HW/디코더 설계 없음, soft-read/양자화 논의 없음.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| RS 기반 H-matrix 대수적 구성 (QC 조건) | ecc_top.cpp Load_PCM() | 고rate binary QC-LDPC PCM 후보, base+shift와 정합 검토 필요 |
| girth≥6 / dmin 보장 (error-floor) | ecc_top.cpp Load_PCM() (부호구조) | 구조적 error-floor 억제, 부호설계 레이어 |
| permutation-matrix 배열 = circulant | ecc_data.h PCM_b (base+shift) | QC일 때 circulant shift 표현과 대응 가능성 |
| SPA(sum-product) 복호 | decoder.cpp LDPC_Decoder() | Prime ECC는 min-sum 근사, 부호는 재사용 가능하나 복호는 기성 |
```

적용 가치: **중간** — 고rate binary QC-LDPC를 girth≥6·큰 dmin으로 대수적 구성하는 방식은 Prime ECC의 부호설계(`Load_PCM`) 레이어와 맞닿지만, 특정 QC-LDPC(z=32 base+shift)로 고정된 코드베이스에 교체 이식은 재검증 부담이 크고 HW 미설계다.
