### B. 알고리즘 요약

1. resolvable Steiner 2-design(BIBD, λ=1)의 incidence matrix 전치를 H로 써서 four-cycle-free regular LDPC를 대수 구성.
2. 풀려는 문제: random 구성은 irregular·고rate에서 small-cycle 제거 곤란, 기존 대수 구성은 rate/length 선택이 경직 → 둘의 장점 결합 목표.
3. 핵심: Steiner 2-design은 두 점이 한 블록에만 함께 → Tanner graph에서 four-cycle 자동 배제(단 girth=6, six-cycle은 필연 존재, 개수는 식(5)로 계산).
4. KTS(Kirkman triple system, resolvable STS)를 mixed difference system과 Construction 1/2로 생성, resolution class 단위로 블록 구성.
5. resolution class 수 `r`이 row weight·length·rate를 결정: column weight 3(또는 4), `N=b`, rate는 `1-(rank/N)`; 같은 rate에서 임의 length 선택 가능.
6. rank(2-rank)로 dimension·rate 정확 산출: Construction 1은 full-rank, Construction 2는 1개 종속행; STS rank는 n mod 12에 따라 결정/유계.
7. min-distance는 column weight+1 이상(=4); anti-Pasch KTS 선택 시 stopping set 최소크기 5, mitre까지 배제(five-sparse) 시 6으로 상향.
8. 구현 이점: difference set만 저장하고 translate를 online 생성(저장 절감), 규칙성이 VLSI 레이아웃 규칙성으로 직결, cyclic KTS는 shift-register로 QC처럼 선형시간 인코딩; translate 추가만으로 online rate/length 확장.
9. 결과: rate-1/2·2/3·0.86 KTS 부호가 동일 rate/length/nonzero의 random 부호와 유사 성능(SPA, max iter 50~500); 특히 length-2000 고rate에서 column-weight-4 resolvable 2-(160,4,1) 부호는 four-cycle-free random 구성이 불가한 영역에서 random 대비 유의미 우세.
10. 한계: HW 미설계, 순수 조합론적 구성이라 girth 6 고정·중간 rate 대부분에서 성능이 random과 동등, 채널은 AWGN 일반(NAND 미언급).
