### B. 알고리즘 요약

1. 문제: 성능 좋고 linear-time 인코딩 가능한 구조적 LDPC 부호를 조합적으로 구성.
2. 핵심기법(superposition): binary `Q×p` base matrix `B`의 각 1-entry를 sparse `k×n` constituent matrix `Gi(∈Q)`로, 0-entry를 zero matrix로 치환하여 `βk×pn` 크기 sparse `H` 생성; `H`의 null space가 LDPC 부호.
3. 제약1(row-column constraint): 같은/다른 `Gi`의 두 row(또는 column)가 공통 1-entry를 최대 1개만 가짐.
4. 제약2(replacement constraint): row/column 내 비영 submatrix들은 서로 distinct.
5. narrow-sense: `B`가 row-column constraint 만족 시 `H`도 만족 → Tanner graph에 length-4 cycle 없음; column weight `≥ γb·γg`, `d_min ≥ γb·γg+1`.
6. broad-sense: `B`가 constraint 미충족이면 4-cycle이 subgraph 사이에만 존재하나, 반직관적으로 iterative decoding에서 매우 잘 동작.
7. 정규성: `B`와 모든 constituent가 regular면 `H`도 regular → regular LDPC.
8. base/constituent 구성 소스: 유클리드 기하의 parallel line 구조, 유클리드·사영기하의 cyclic line 구조(→ QC-LDPC).
9. 결과: (6132,4602) narrow-sense·(8176,7156) broad-sense QC 부호 모두 Shannon limit 근접; (8176,7156)은 4-cycle 5110개에도 BER 1e-6에서 Shannon 1.2dB, error floor 없음; MacKay·irregular 부호와 동등, 저BER서 irregular(error-floor 발생) 대비 우수.
10. 한계: 1쪽 ISIT 요약이라 구성 세부·복잡도·throughput 부재, HW 미설계, AWGN 시뮬만.
