### B. 알고리즘 요약

1. 시스템: 2-D ISI 채널(page-oriented 광 메모리, 고밀도 자기기록), 사용자 비트를 ECC로 부호화 후 2-D matrix로 스캔, AWGN 부가.
2. 문제: 일반 2-D ISI 검출은 combining coefficient 수가 지수적이라 큰 이웃(고밀도)에 비현실적.
3. 가정: 2-D ISI matrix가 두 벡터곱으로 분리가능(separable) → row 1-D ISI + column 1-D ISI 직렬연접으로 취급.
4. 방법1: 한 차원은 ZF/MMSE 등화로 ISI 제거 후 나머지 차원에 MAP 검출, LDPC 복호와 반복(turbo equalization의 2-D 버전).
5. 방법2 (row-and-column): column ISI는 4상태 비이진 trellis → 비이진 MAP, row ISI는 이진 trellis → BCJR, 두 검출기가 extrinsic 교환.
6. row detector의 branch transition은 직접 관측이 없어 column detector의 extrinsic만으로 계산하도록 수정.
7. 반복 스케줄: type-I(column↔row subiteration) 후 type-II(row↔LDPC decoder), LDPC가 pseudo-posterior 계산해 되돌림.
8. 비교: full graph 3-level message passing(LDPC+ISI 결합 그래프)도 소개, row-and-column이 이보다 우수.
9. 결과: block length 10000, regular (3,6) LDPC, LDPC 최대 50 iter; 분리가능 채널에서 ISI 성능 손실 1 dB 미만, full graph 대비 약 1 dB 개선.
10. 한계: HW 미설계, 시뮬만, 복잡도는 ISI 길이에 지수적, LDPC 자체는 기성 부호로 개선 없음.

```
| turbo equalization 반복 스케줄 | 대응 없음 | 외부 검출기↔LDPC 스케줄, min-sum 이터레이션 루프와 무관 |

```
