### B. 알고리즘 요약

1. 시스템: `(γρ^{γ-1}, ρ^γ, ρ, γ, {0,1})` 조합설계 기반 (γ,ρ)-regular LDPC 앙상블, rate `(1-γ/ρ)`, girth `≥ 2γ+2 (≥8)`.
2. 문제: 유한기하 LDPC는 부호집합이 제한적이고 복호 복잡도가 높음 → 일부 line을 제거한 "loosened" 설계로 복잡도↓·부호집합 다양화.
3. 모델: γ차원 subscript로 라벨된 `ρ^γ` block, `γρ^{γ-1}` point; 두 block이 최대 1개 point만 공유(no 4/6-cycle 유도).
4. 핵심기법: `p^2` block을 principal plane별로 `ρ`개 bundle로 분할하는 bundle-rule로 point-block incidence 정의 → (γ,ρ)-regular H 생성.
5. bundle-rule 선택지: cyclic permutation, M-sequence(shift register), congruential sequence, GF(p^t)의 Zech logarithm.
6. 확장(γ=2 특수화): 고rate·systematic·quasi-cyclic·linear-time encodable/decodable, length-4/6 cycle-free 부호; 2D single-parity-check TPC와 등가.
7. 구조 vs randomness: 구조(HW 저비용 구현) + pseudo-randomness(사각형 등 반복 error pattern 회피)의 조합.
8. 거리스펙트럼: γ=2 부호의 output weight enumerator(OWE)를 Gallager random 앙상블 평균과 비교, 저weight codeword 수가 더 적음.
9. 결과: rate 0.88·0.94 부호를 PR4/EPR4/E²PR4 채널에서 시뮬, precoding 적용 시 random 부호와 동등하거나 약간 우수.
10. 한계: (2,ρ) 부호는 AWGN에서 iterative threshold 부재(memory 채널에서만 "good"), HW 미설계, 1쪽 ISIT 요약이라 수치·구성 세부 부족.
