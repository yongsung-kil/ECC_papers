### B. 알고리즘 요약

1. 시스템: AWGN 채널, CDMA/무선 통신 응용 대상의 LDPC 부호 구성법.
2. 문제: 기존 classic π-rotation 부호는 rate 유연성이 제한적 → 다양한 rate를 간단히 기술하는 확장이 필요.
3. 핵심 구조: parity-check matrix `H`를 단일 순열벡터로 완전 기술, 순열벡터는 3정수 key `[·,·,·]`로 생성(classic).
4. 확장 기법: irregular column stack 도입, 확장 부호는 7정수 벡터 `[key3 + I,P,H,compression]`로 정의.
5. 파라미터 의미: `I`=정보 세그먼트 수(1~4), `P`=puncturing할 parity 세그먼트 수, `H`=stack 높이(0~12), 마지막=압축계수.
6. rate 조정: parity 세그먼트 puncturing(`c` segment 제거)으로 code rate 가변, 식(2)로 `P` 결정(예: rate 1/4).
7. 인코딩: 정보→projection→parity를 4방향 register rotation + accumulation으로 수행, 복잡도 block length에 선형비례, shift-register 친화.
8. 확장 인코딩: 압축된 순열행렬(`R'` 등)에 대한 추가 projection, 마지막 정보 subvector 비트 복제로 수행.
9. 결과: 여러 rate(1/5,1/4,1/3,3/7,1/2)에서 Shannon limit 대비 1.0dB 이내, classic 및 MacKay/Sorokine 부호 대비 우수한 BER/FER.
10. 한계: HW 미설계(회로 구조만 서술), AWGN 시뮬만, 저rate 무선/CDMA 특화.
