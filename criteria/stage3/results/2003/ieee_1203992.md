### B. 알고리즘 요약

1. 시스템: BPSK/AWGN 채널에서 irregular LDPC 부호를 puncturing해 하나의 base code로 넓은 rate 범위(0.33~0.985)를 커버하는 rate-compatible 방식.
2. 문제: rate마다 별도 부호 설계 대신, 단일 base code를 puncturing해도 threshold·error-floor 열화 없이 여러 rate를 지원하고 싶음.
3. 모델: variable node를 degree별 그룹 `Gj`로 묶고, 각 그룹에서 비율 `πj(0)`만큼 랜덤 puncturing. 전체 puncturing fraction `p(0) = Σ λj·πj(0)`.
4. 핵심기법: sum-product 복호에서 puncture된 VN이 하나라도 붙은 CN 메시지는 0 LLR이 됨. Gaussian Approximation(GA)으로 이 확률 `ε(k)`의 진화를 추적.
5. 핵심식: updated mean `mu(k)` 재귀식 (식 5,7). residual puncturing fraction이 SNR과 무관하게 degree 분포 3-tuple `(λ,ρ,π(0))`에만 의존 → steady-state 방정식으로 단순화.
6. 확장: error-free 복호 조건을 `mu(k+1)>mu(k)` 부등식(식 8)으로 표현, puncturing 분포 설계 기준 도출.
7. 구현 디테일: 위 조건을 단순 linear optimization(LP)으로 풀어 주어진 Eb/N0에서 `p(0)`를 최대화. DDE+Differential Evolution(DE)로 교차검증.
8. 최적화 절차: DE는 U4 공간 탐색(TR1) + LP 초기값을 Gaussian/uniform 섭동(TR2/TR3)으로 local minima 회피.
9. 결과: base 대비 +0.1dB gap으로 rate 0.72(dl=10)/0.86(dl=20)/0.80(rate1/3)까지 확장. 최대 rate 0.92~0.985 달성, error floor 미열화(BER 10^-4까지 확인).
10. 한계: asymptotic threshold 중심의 이론 연구, 짧은 블록·iteration 수·양자화 등 실무 이슈는 미다룸. HW 설계 없음.
