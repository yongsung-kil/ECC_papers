> 총평: base+cyclic-shift 확장으로 QC-LDPC(Prime ECC 구조와 동형)를 partially-parallel 디코더 친화적으로 joint 설계한 초기 논문으로, 부호구성 철학은 참고가치 있으나 rate 1/2·AWGN 예제 한정이라 이식성은 중.

### B. 알고리즘 요약

1. 시스템: BPSK/AWGN에서 partially-parallel VLSI 디코더에 맞는 (3,6)-regular LDPC 부호(`C1`=4K, `C2`=8K, rate 1/2)를 joint code-decoder 설계.
2. 문제: 완전 랜덤 LDPC는 라우팅 혼잡·긴 전역배선으로 partially-parallel HW에 부적합. code-first 설계는 효율적 디코더 구현을 거의 불가능하게 함.
3. 3단계 구성: (1) `Ms×Ns` base matrix 다수 생성, (2) 각 base를 인자 `p`로 랜덤 확장, (3) cycle-effect 기준으로 부호 선택.
4. base matrix: bit-filling(Campello-Modha) 휴리스틱으로 girth 제약을 최대에서 시작해 점진 완화하며 1을 채워 large-girth 그래프 확보.
5. 확장: base의 각 1을 `p×p` 항등행렬을 `ku,v`만큼 우측 cyclic-shift한 서브행렬 `Tu,v`로, 각 0을 `p×p` 영행렬로 치환 → circulant 기반 QC-LDPC.
6. 확장 효과: 확장행렬 girth ≥ base girth, 랜덤 cyclic 치환이 평균 cycle 길이를 늘려 BP 실효성↑, 메모리 주소생성 제어로직 단순화.
7. cycle-effect(loopiness) `L=Σ Ni·αi` 최소 부호 선택 → 짧은 cycle 억제로 정정능력 확보.
8. partially-parallel 디코더: `Ms` CNU + `Ns` VNU + `L+Ns` 메모리블록, 한 iteration을 `2p` 클록에 완료(전반 p=CN처리, 후반 p=VN처리), `Tu,v` shift값 `ku,v`를 counter 시작값으로 사용해 주소생성.
9. 결과: `C1`(4K)/`C2`(8K) 부호가 완전랜덤 부호와 대등한 BER/FER, max iter 20에서 평균 iteration 곡선 제시(1.6~3.2dB).
10. 한계: (3,6)-regular rate 1/2·AWGN만, gatecount/throughput 미제시, 인코딩 복잡도는 향후과제, irregular 분포 미적용.
