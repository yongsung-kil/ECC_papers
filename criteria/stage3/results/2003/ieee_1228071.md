### B. 알고리즘 요약

1. 대상: 유한 affine plane `AG(2,2^s)`의 point-line incidence matrix 부분행렬 기반 QC-LDPC 부호 구성.
2. 풀려는 문제: Kou 등[2]의 finite geometry LDPC를 확장해 circulant(quasi-cyclic) 구조를 갖는 부호를 대수적으로 유도.
3. 핵심 대상: 유한체 `F_q` (`q=2^s`), 원점 제외 점집합 `A0`, trace `Tr(βy)=1`로 정의되는 원점 미포함 라인집합 `A1`.
4. 구성: 점(`A0`)-라인(`A1`) incidence matrix `H^(∞)`를 order `q+1`의 `(q-1)×(q-1)` circulant 배열로 표현.
5. 의미: norm map kernel `O`의 coset 구조 덕에 각 라인이 `O_i`를 0 또는 2 점에서 만나 circulant 블록(row weight 2)이 규칙적으로 배치됨.
6. 확장: 부분군 `G`에 대해 `O(G)`를 정의하고, incidence를 `H^{(a)}(G_a)` 서브행렬 계층으로 분할(Theorem 2로 rank_2 산출).
7. 부호 정의: `C^{(a)}(G_a)`, `C_S^{(a)}(G_a)`를 각 서브행렬의 null space로 정의, 차원은 rank_2 공식(예: `2^{2s}-3^s+2s-1`)으로 산출.
8. 성질: 이 부호들의 Tanner graph는 four-cycle이 없음(girth ≥ 6). oval design(Weller-Johnson[3])과의 연결도 제시.
9. 결과: `C^{(a)}(G1)`, `s=4,5`를 sum-product 복호(`I_max=100`)로 시뮬한 BER 곡선(Fig.1) 제시 — 수치는 본문 서술 없어 미상.
10. 한계: 1페이지 초록, rate/length·정량 BER·error-floor·HW 데이터 부재, sum-product(full-tanh) 전제.
