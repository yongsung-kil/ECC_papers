### B. 알고리즘 요약

1. 대상: 고rate·단블록 regular Gallager(LDPC) 부호를 조합론적 설계(BIBD)로 구성, AWGN/BPSK에서 평가.
2. 문제: 기존 구성법은 고rate·단길이 영역에서 파라미터 유연성·최소거리 보증이 부족.
3. 핵심: 2-`(v,k,1)` BIBD의 block-point 근접행렬을 패리티검사행렬 `H`로 사용 → 열가중 `r`, 행가중 `k`, girth 6.
4. 차분족(CDF): Abelian group `Z_v`의 base block 궤도(orbit)마다 하나의 circulant 부분행렬 → `H`가 quasi-cyclic 구조를 가짐 (`H=[H1 H2 ... Ht]`, 각 v×v).
5. 첫 구성: Netto/Buratti 차분족 (`v` 소수, `v≡1 mod 6` 등) → code rate·길이 최적 tradeoff이나 고rate·단길이 부호 수가 적음.
6. 둘째 구성: `k×m` 직사각 정수격자의 slope별 직선(line)을 block으로 → 각 부분행렬이 열가중 1인 순열행렬(circulant), 파라미터 유연·rate 낮음.
7. 최소거리: Pasch 구성(quadrilateral, (6,4) 형태) 부재를 이용, Steiner triple(k=3)에서 `d_min≤6`, 일반 `k`에서 `d_min≤2k` 상한, 열가중만의 함수로 하한 유도.
8. Netto triple system은 `v≡...mod 24` 조건에서 Pasch-free → 짧은 사이클/유해구조 억제.
9. 결과: BER 10^-x에서 유사 rate RS(240,234,7) 대비 4~5 dB 이득(log-domain message passing 복호, Monte Carlo).
10. 한계: HW 미설계, 복호기 세부 없음, rate/length 예제 종속으로 확정 불가, 이론(min-distance) 중심.
