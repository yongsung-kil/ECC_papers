### B. 알고리즘 요약

1. 시스템: ATSC 3.0 지상파 방송, LDM(Layered Division Multiplexing)으로 UHD 고정 + HD 모바일 서비스 동시 전송. 코드는 ATSC 3.0 LDPC (rate 4/15~13/15).
2. 문제: LDM UL(Upper Layer) 복호 시 최적 LLR PDF는 LL(Lower Layer) 성상 전체를 수신기에 저장해야 해 모바일 수신기 메모리/복잡도 부담이 큼.
3. 모델: AWGN + 다양한 방송 채널모델(DVBT-F/P, Ray6, Brazil, indoor/mobile PI/PA/PB/PO/VA/VB/TU6)에서 SNR threshold(BER<1e-6) 측정.
4. 핵심기법: UL LDPC 복호의 LLR을 `NAWGN`, 채널추정 `ρ`, LDM 송수신 IQ 코드워드로 정의된 최적 PDF (식 4)로 계산.
5. 식 의미: 식(4)의 최적 PDF는 LL 성상 점 `k`에 대한 합산을 포함해 다층 간섭을 반영 → LDM UL 성능 향상.
6. 확장(semi-optimal): 실제 LL 성상보다 낮은 차수 성상(QPSK/16-64 NU-QAM 여러 code-rate 조합 26종)을 가정한 근사 PDF로 성상 저장 부담 축소.
7. 구현 디테일: 최적 PDF는 IL(injection level)과 LL 성상 종속, semi-optimal은 성상 차수 독립.
8. 최적화 절차: 없음 (시뮬레이션 기반 SNR threshold 비교만).
9. 결과: LDM은 TDM 대비 항상 최소 0.9 dB 우수, 최적 PDF 시 정적 +0.9 dB / 모빌리티 +1.5 dB 추가 이득. semi-optimal은 최적 대비 최대 0.4 dB 열화에 그침.
10. 한계: HW 미설계, 시뮬만, 방송용 다층변조(LDM) 특화 LLR로 NAND single-layer 저장채널에 부적합.
