### B. 알고리즘 요약

1. 시스템: TLC NAND Flash, PEG로 구성한 binary regular LDPC `0.9-(3780,3402)`, `0.92-(8000,7360)`, 채널은 14개 non-uniform read reference로 15개 영역(8 data + 7 overlap) 분할.
2. 문제: LDPC soft-decision 복호의 read latency·복잡도, 저장밀도 증가에 따른 신뢰도 저하를 함께 줄이려 함.
3. 채널 모델: programming noise·CCI·RTN·retention noise의 convolution으로 threshold voltage 분포 `ps111`, `psp` 정의(식1,2), PE=13000 기준.
4. 핵심기법(그룹핑): 모든 VN을 `|init-llr|` 크기와 영역별 오류확률로 data1(D1,D2,D7,D8)/data2(나머지 data)/overlap 3 sub-block으로 동적 분할.
5. 근거: Table I에서 `Pdata1`(2~6%)가 가장 낮고 `Poverlap`(81~94%)가 최고 → data1은 가장 신뢰 높아 초기 반복에서 갱신 skip 가능.
6. 저정보 VN 정의: `|llr| >= Vlow`이면 low-information VN. `Vlow = avg(abs(init-llr[overlap]))`(식3)로 overlap 평균 LLR을 임계로 사용.
7. 스케줄(Block-SBP 4단계): (1)적응 분할·임계결정 (2)초기엔 data1 skip하고 data2·overlap serial 갱신하며 low-info VN을 집합 L에 저장 (3)L의 VN을 소진까지 갱신 (4)`iter >= γ·Itermax`부터 data1 갱신 시작, 반복.
8. 파라미터: `Itermax=5`, `γ=0.625`로 동일 FER에서 최고 throughput.
9. 결과: VNBP-MP 대비 convergency rate 약 28.6% 향상, avg iteration 약 27.7% 감소(PE=13000), 정정능력(FER) 손실 없음.
10. 한계: HW 미설계, 시뮬만, TLC 다중 read voltage(soft-4bit+) 전제, gatecount/throughput 미보고.
