### B. 알고리즘 요약

1. 시스템: MLC/TLC NAND flash + 소프트 LDPC 복호, 채널을 AWGN(means `{-3,-1,1,3}V`)으로 모델, word-line 단위 고정전압 read.
2. 문제: soft-decision LDPC는 다중 read로 soft 정보를 얻는데, read 개수·위치가 read latency와 BER를 좌우 -> read threshold `γ`의 개수·배치를 사전 최적화해야 함.
3. 접근1(MMI): 등가 DMC의 상호정보 `I(X;Y)=H(Y)-H(Y|X)`를 최대화하는 threshold를 gradient ascent로 탐색(`γ(l+1)=γ(l)+δ·∇g`), read 개수 무관하게 적용 가능.
4. 결과1: `m≈20` read면 추가 read당 MI 이득 0.05% 미만으로 near-optimal, m=6에서는 최적 위치가 등간격 gap으로 수렴.
5. 접근2(DDE): LDPC 구조를 반영하기 위해 discretized density evolution으로 BER를 추정, 이를 cost로 gradient descent(`γ(t+1)=γ(t)+δ·∇g`) 최소화.
6. SRQ(split ratio quantization): 표준 양자화의 LLR pmf 급변으로 생기는 BER fluctuation을 인접 양자점에 거리 역비례 분배(`Pl=b/(a+b)P`)로 완화 -> local minima trap 제거.
7. 채널 라벨링: MLC-SCL(symmetric channel labeling)로 MSB/LSB 대칭화, LSB가 더 열화되어 병목이므로 LSB 중심 최적화.
8. 구현 트릭: 유한 LLR-pmf(`∆[-N,N]`), FFT 기반 convolution, R-operation을 R-List 사전계산으로 O(n³)->2차원 축소.
9. 결과2: (3,4)-regular QC-LDPC(n=36, Code A)·802.11n irregular(n=1944, r=2/3, Code B)·n=8192 r=0.94 부호에서 DDE가 mid-SNR BER를 MMI보다 근소 우위, 2단(MMI->DDE)으로 항상 global minimum 도달. TLC는 실측 분포로 m=7->14 read 시 BER 대폭 개선.
10. 한계: DDE 이득이 복잡도 대비 미미해 실용상 MMI로 충분, HW 미설계·시뮬(DDE/BP)뿐.
