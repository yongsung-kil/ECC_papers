### B. 알고리즘 요약

1. 대상은 2D inter-symbol interference(ISI)를 갖는 자기기록 채널(2D array 데이터 x(i,j), 수평/수직 간섭 길이 LM/LN, 채널 응답 행렬 H, AWGN 가정).
2. 목적은 기존 EXIT-chart 기반 irregular LDPC([3])보다 낮은 복호 threshold와 낮은 error floor를 갖는 protograph LDPC를 2D-ISI 채널에 맞게 설계.
3. 채널 모델은 4Tb/in2 자기기록 밀도의 2D-ISI 채널, symbol-based BCJR 2D 검출기와 결합.
4. 핵심 기법은 degree-2 노드를 배제한 protograph base matrix 설계(Fig.1(a)), 부호율 R=(n+1)/(n+2), n=8일 때 rate 8/9.
5. EXIT curve 분석으로 채널검출기-LDPC디코더 곡선이 서로 교차하지 않으면서 최소 간격을 갖는 최소 Eb/N0(threshold)를 도출; 제안 부호 threshold=3.93dB, irregular 부호=4.0dB(둘 다 BCJR 검출기 기준).
6. 추가로 normalized logarithmic asymptotic weight distribution r(δ)를 계산해 최소거리비 δmin=0.031(>0)을 확인, 이는 블록길이 증가 시 error floor가 선형적으로 낮아짐을 의미.
7. 별도 HW 친화 변형·양자화 언급 없음; 순수 이론적 EXIT chart fitting + asymptotic weight enumerator 최적화.
8. 최적화 절차는 EXIT chart fitting(반복적으로 base matrix degree distribution 조정)과 weight enumerator 분석.
9. 결과: 코드길이 4608/13824, rate 8/9에서 제안 부호가 irregular 부호 대비 낮은 threshold·낮은 error floor, BER 10^-5에서 13824비트 부호가 채널 대칭정보율(symmetric information rate) 대비 0.3dB 이내.
10. HW 미설계, 시뮬레이션/이론 분석에 국한, 특정 2D-ISI 채널·BCJR 검출기 조합에 종속.
