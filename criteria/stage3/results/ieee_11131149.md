### B. 알고리즘 요약

1. binary-input AWGN/QPSK, 5G NR 관행 하에 3개 시나리오(rate·length 상이)로 5G LDPC와 공정 비교하는 GLDPC 부호를 설계.
2. 기존 GLDPC는 (a)CN 처리 복잡, (b)성분코드 길이 가변 어려움, (c)EXIT/Gaussian 근사 부정확이 문제.
3. 성분코드로 Cordaro–Wagner 부호의 **dual**을 사용: PCM이 높이 2의 서로 다른 non-zero 열로만 구성, 열을 `X,Y,Z` 세 그룹으로 균등 분배해 weight-2 codeword 최소화.
4. 각 codeword는 `⊕_{i∈X}c_i = ⊕_{i∈Y}c_i = ⊕_{i∈Z}c_i = v` 제약을 만족; 이를 Tanner 그래프 CN으로 사용.
5. SP 디코더: Hartmann–Rudolph 규칙(dual code 4개 codeword)으로 CN extrinsic을 `Lˆi = Li + 2 tanh⁻¹(...)` 닫힌형으로 도출 (식 4).
6. MS 디코더: bit별 2개 경쟁 가설(`c0,c1`)의 최소 LLR magnitude만 보고 `Ei = max(L̃X+L̃Y+L̃Z,0) − max(L̃Y+L̃Z, L̃X)` (식 7).
7. 핵심 관찰(Statement 1): PCM에 잠재변수 열을 추가한 확장 행렬(식 8)에 **표준 MS를 그대로** 적용하면 식(7)과 동일한 extrinsic이 나옴 → 표준 SP/MS 구현 재사용 가능, 단 스케줄링이 결정적.
8. 부호 설계: protograph에 quantized DE + 유전 알고리즘으로 최적화, ACE 알고리즘으로 lifting해 짧은 사이클 제거.
9. 결과: NLMS(α=0.75) 50 iter에서 5G LDPC와 동등, **10 iter에서 유의하게 우수**(빠른 수렴); CN당 비교 연산이 `2n−3`에서 `2n−9+6` 수준으로 감소.
10. 한계: HW 미설계, 시뮬(BLER)만, 무선/AWGN 타깃이며 성분코드 기반 GLDPC라 부호구조가 NAND binary QC-LDPC와 이질적.
