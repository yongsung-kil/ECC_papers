### B. 알고리즘 요약

1. 대상은 QC-LDPC 부호를 QPSK 변조 후 AWGN 채널로 전송한 데이터를 복호하는 것(구체 rate/length 미상).
2. 문제는 부호 길이 증가 시 전통 BP/행렬연산 복잡도가 커지고 Tanner 그래프의 short cycle이 복호 성능을 저하시키는 것.
3. 핵심 가정: Tanner 그래프를 "전개(unfold)"하고 edge에 가중치를 두어 경사하강으로 학습하면 short cycle 악영향을 완화(Nachmani류 접근 계승).
4. 핵심 기법: soft-decision BP 복호부를 DenseNet(CNN) 신경망으로 대체. 입력은 LDPC 부호화+QPSK+AWGN 통과 데이터.
5. DenseNet은 앞뒤 층을 dense connection(shortcut)으로 연결해 feature 재사용·gradient 소실 완화, ResNet보다 적은 파라미터로 더 나은 성능 목표.
6. CNN의 weight sharing 구조로 파라미터·복잡도를 줄여 긴 부호길이 입력을 처리하고 복호 성능 향상 주장.
7. 구조는 convolution + pooling + fully-connected(최소 5층), conv-relu 사이 batch normalization으로 학습 가속·초기화 민감도 감소.
8. 학습: 경사하강(gradient descent)으로 edge 가중치/네트워크 파라미터 훈련.
9. 결과: 시뮬레이션 BER 곡선(Fig.2)에서 신경망 기반 복호가 전통 BP보다 우수하며 SNR 증가 시 BER 감소가 더 빠름(구체 수치 미상).
10. 한계: HW 미설계, 정량 스펙 부재(rate/length/파라미터/gate 없음), 단일 채널 시뮬만, extended abstract 수준.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| DenseNet(CNN) 신경망 디코더 | 대응 없음 | 프로파일 §6에서 NN/딥러닝 디코더는 명시적 "대응 없음" |
| soft-decision BP 복호부 대체 | 대응 없음 | Prime ECC는 min-sum 근사 사용, full-BP/NN 미보유 |
| Tanner 그래프 edge 가중치 학습 | 대응 없음 | Prime ECC 이터레이션 루프는 학습 가중치 미보유 |
| soft/LLR 입력 | [channel.cpp](../Prime_ECC_3.1_Claude) `Set_LLR_Th()` | 입력단만 개념적 연관, 복호 알고리즘은 무관 |

적용 가치: **낮음** — Prime ECC는 min-sum 기반 bit-exact HW 디코더로 NN 전면 대체 방식과 근본적으로 상충하며, 논문에 HW·정량 스펙이 전무해 이식 근거가 없다.
