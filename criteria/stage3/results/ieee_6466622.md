### B. 알고리즘 요약

1. 시스템: N=155, M=93 Tanner code(rate≈0.4)의 Tanner graph를 MRF로 보고 BP 대신 GBP로 복호, error-correcting 관점에서 trapping-set 극복이 목표.
2. 풀려는 문제: BP는 Tanner graph의 짧은 loop 조합인 trapping-set(TS)에서 발산/오수렴하여 suboptimal → loop 흡수 필요.
3. 핵심 모델: Bethe 근사(=BP)를 Kikuchi 근사로 일반화. Tanner graph 노드를 묶은 region-graph 위에서 message-passing(GBP) 수행.
4. 핵심기법1(region-graph): 레벨별로 region 구성. `R0`는 각 check node cj와 이웃 Nj를 포함하는 region들, 상위 레벨 `Rl`은 이전 레벨 region들의 교집합으로 구성.
5. 핵심 식: region 메시지 `m_rq^k = F_rq({m_ps^{k-1}}, {l_i})`, region q에 대한 사후확률분포. TS(5,3)를 3개 소형 region(r,p,q)으로 분해해 복잡도 완화.
6. 핵심기법2(안정화): GBP 수렴 불량 → uniform weighting `m_rq = 0.5·F_rq + 0.5·m_rq^{k-1}` 도입(이게 없으면 BP보다 나쁨).
7. 부수: TS(5,3) 하나는 loop-free region-graph로 최적 복호되나, 전체 코드는 노드 공유로 loop-free 불가.
8. 동역학 분석: bifurcation diagram(mean square beliefs `Ej=(1/N)Σb_i^2`), Lyapunov exponent `λ=ln(dK/d0)`, 신규 hyperspheres 반경법으로 BP/GBP 궤적(fixed point/limit cycle/chaos) 비교.
9. 결과: GBP가 BP보다 작은 attractor로 수렴해 더 안정적, SNR별 임계값(GBP 2.07dB vs BP 2.19dB 등)에서 정정력 우위. BER 정량 곡선은 그림에만 존재(수치 미상).
10. 한계: HW 미설계, 양자화/soft-read 무관, region 복잡도 큼, 특정 Tanner code 대상 순수이론.
