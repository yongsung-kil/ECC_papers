### B. 알고리즘 요약

1. 대상: GF(q) 유한체 base matrix(type-1 Vandermonde/type-2 Latin square)를 masking+dispersion해 얻는 irregular binary QC-LDPC, 예제 rate 0.5(6096,12192)·0.375(5080,8128).
2. 문제: masking matrix가 부호 오류성능을 좌우하나 대개 computer search로 얻어짐 — 구조적 masking matrix 설계 부재.
3. masking 연산: `M(γ,ρ)=Z(γ,ρ)⊗H(γ,ρ)` (`z_ij h_ij`), zero 항목이 base matrix를 마스킹 -> dispersion으로 `Hdisp`(circulant/zero blocks) 생성, null space가 QC-LDPC.
4. 핵심 기법: spatial-coupling에서 착안, 원본 `Zorig`를 재배선해 구조적 `Znew = [[Z1,0,0],[0,Z1,0],[A,B,C]]`(식 6) — 동일 크기·동일 degree distribution 유지, 두 동일 `Z1` 그래프를 특수 결합.
5. `Znew` 생성: PEG로 `Z1`, `Z2=[A B C]` 각각 생성 후 결합, 요구 degree distribution `v(X),c(X)` 만족할 때까지 반복(Step1~3).
6. 분석 도구: protograph-based LDPC용 Gaussian Approximation(PGA) 제안 — `b_ij`(엣지 다중도) 반영해 V->C(식 9), C->V(식 10, `φ` 함수) LLR 평균 갱신.
7. 수렴속도 지표: 전 변수노드 APP-LLR 평균 `mAPP=(1/ρ)Σ m_APP^(L)(j)`(식 7) 도입, `mAPP>M*(~45)`이면 decodable.
8. `φ(x)` 근사식(식 13) 2구간(x∈[0,10], x>10)으로 복잡도 절감.
9. 결과: code A threshold `Eb/N0*`=0.7109(orig)/0.7067dB(new)로 거의 동일하나, threshold 이하 SNR에서 new의 `mAPP`가 항상 큼 -> 더 빠른 수렴, BER 1e-6에서 약 0.2dB(code A)/>0.1dB(code B) coding gain(SPA 50 iter, AWGN/BPSK).
10. 한계: AWGN/SPA 시뮬만, HW 미설계, error-floor 미검증, spatial-coupling과 엄밀히 동일 구조 아님(향후 이론분석 필요).
