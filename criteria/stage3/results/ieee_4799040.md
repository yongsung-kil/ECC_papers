### B. 알고리즘 요약

1. 채널: classical burst erasure channel(burst 구간=완전 소거, guard 구간=소거 없음 또는 낮은 확률 p로 랜덤 소거) 및 이를 확장한 multi-burst 채널. binary LDPC 부호(N,K), 패리티검사행렬 H, message-passing(erasure) 디코딩 대상.
2. 풀려는 문제: burst erasure(연속 구간 전체 소거, 예: 딥 페이딩·패킷 손실·자기기록매체 결함)에서 항상 정정 가능한 최대 burst 길이 `Lmax`를 이론적 상한(burstMDS, `Lmax=n-k`)에 최대한 가깝게 만드는 구조적 QC-LDPC 설계.
3. 핵심 가정: 수신기는 소거 위치를 정확히 알고, 소거되지 않은 비트는 오류 없음(guard band에 랜덤 소거 확률 p를 추가로 고려하는 변형도 다룸).
4. 핵심 기법: superposition 구성 - `M×N` base matrix `H_base`의 0 원소를 `v×v` 영행렬, 1 원소를 circulant/permutation 행렬(`A^(i)`=circulant를 i열 순환이동)로 치환해 `m=Mv, n=Nv` LDPC 행렬 `H`를 생성.
5. 핵심 식: minimum zero span `s`(행 내 두 비인접 1원소 사이 최소 0개수)와 최소 stopping set 크기 `S_min`이 `Lmax`를 결정 (`Lmax ≥ s+1`, 4-cycle-free이면 `Lmax ≥ s+2γ-1`); circulant shift 값을 조정해 stopping set 열 간격을 넓혀 `Lmax`를 극대화.
6. 확장(Construction 1~6): weight-2/3 circulant, 다중 base matrix 연결(concatenation)로 single-burst뿐 아니라 multi-burst(2개 이상) 정정 부호를 설계, 각 구성마다 `Lmax`의 닫힌 형태 하한을 증명(Lemma 3~13).
7. 부수 트릭: 4-cycle 회피를 위해 circulant의 zero span을 서로 다르게 배치, quasi-cyclic 구조로 `[20]`의 효율적 인코딩 기법 재사용 가능하게 설계.
8. 학습/최적화 절차 없음 - 결정론적(closed-form) 구성법이며 별도 탐색(GA/DE 등) 불필요.
9. 결과: 예) length-500 rate-1/2 코드에서 Construction 4가 `Lmax=248`, 효율 `η=0.992` 달성(랜덤 LDPC나 기존 pseudo-random 구성법[9] 대비 우수), length-16500 rate-0.9 코드에서 `Lmax=1648~1639`, 효율 `~0.99`.
10. 한계: 채널이 순수 erasure(위치 known)라 NAND의 하드/소프트 read-error(BSC/AWGN) 모델과 다르고, 디코딩도 min-sum/BP가 아닌 단순 unique-neighbor 소거 복원(1개 소거항만 있는 parity check을 순차 해결)이라 HW 설계나 정정능력(BER) 개념 자체가 적용되지 않음.

```
| 4-cycle-free circulant 설계(zero span 최적화) | 대응 없음 | H-matrix 구조 최적화 기법이나 목적함수가 burst 정정용이라 고rate NAND용 H-matrix 설계와 무관 |
```

적용 가치: 낮음 — burst erasure 채널·erasure 디코딩은 NAND 고전 binary LDPC의 AWGN/BSC hard/soft error 정정 문제와 오류모델·디코더가 본질적으로 다르며, 부호 구성법도 다른 목적함수(burst 길이)를 최적화해 이식 시 재설계 부담이 큼(§4 "부호설계" 항목 자체가 최고 난이도로 분류됨).
