> 총평: LDPC가 아닌 concatenated tree codes(CT codes)의 constituent tree 저복잡도 BP 디코딩과 union-bound 기반 오류율 해석 기법 논문으로, NAND binary QC-LDPC와 부호족 자체가 달라 이식 가치 낮음.

### B. 알고리즘 요약

1. 대상은 concatenated tree (CT) codes — M개의 constituent tree code를 parallel concatenation한 capacity-approaching 부호로, 각 constituent는 `(I, J', J'')` 파라미터의 cycle-free tree 구조(Tanner graph), 부호율은 `(J'+J'')/(J'+J''+M)`.
2. 문제의식: 기존 CT code 제안 논문[7]은 전체 디코딩 스케줄만 제시했고 constituent tree code 자체의 구체적 저복잡도 디코딩 알고리즘과 오류율 성능은 미제시했음.
3. 채널 모델은 AWGN, LLR 기반 sum-product(BP) 디코딩을 가정하며, 메시지 함수는 `W(a1,...,an)=Πsgn(ai)·φ(Σφ(|ai|))`, `φ(x)=-log(tanh(x/2))` (표준 SPA check-node 연산).
4. 핵심 기법: tree 구조상 대부분 bit node의 degree가 1이라는 성질을 이용, 메시지가 실제로 갱신되는 노드는 punctured parity node뿐임을 활용해 5단계(S0~S4) forward/backward 2-way scheduling으로 단순화.
5. S1(forward: FI/FO)과 S2(backward: BI/BO)는 서로 독립적으로 병렬 수행 가능(S1은 이전 forward 값에만, S2는 이전 backward 값에만 의존) — 이 부분이 병렬화 가능 포인트.
6. 부수 트릭: 전체 CT code를 직접 BP 하지 않고 constituent tree 단위로 나눠 순차 디코딩 후 extrinsic information을 교환하는 구조를 채택(원 논문[7]의 스케줄 재사용).
7. 최적화/학습 절차는 없음(구조 기반 알고리즘 단순화, GA/DE/NN 없음).
8. 오류-플로어(저오류율 영역) 해석을 위해 union bound 근사 `P_W(C)≈A_dmin·Q(√(2R·dmin·Eb/N0))`를 제시, tree code의 `dmin=2`이고 weight-2 codeword 수 `A_dmin = I×(C(J',2)+C(J''+1,2))`로 closed-form 유도.
9. 결과: (50,2,2) tree code에서 union bound 근사와 Monte-Carlo 시뮬레이션 WER/BER이 잘 일치함을 그래프(Fig.4)로 확인(구체 수치는 그래프에만 존재하여 본문 서술 수치 없음).
10. 한계: HW 구현/합성 결과 없음(순수 알고리즘+시뮬), 매우 작은 예제((50,2,2))만 검증, CT/tree code는 LDPC의 특수 사례로 소개되지만 QC-LDPC 및 min-sum 기반 Prime ECC 구조와는 이질적.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| tree code 2-way scheduling BP 디코딩 | 대응 없음 | full-tanh 기반 sum-product이며 부호구조(tree)가 QC-LDPC와 상이, Prime ECC는 min-sum만 지원 |
| union bound 오류율 해석 | 대응 없음 | 알고리즘/HW 요소가 아닌 이론 해석 기법 |
| S1/S2 forward-backward 병렬화 | 대응 없음 | tree 특유의 degree-1 구조 의존적 트릭으로 QC-LDPC circulant 구조에 일반화 안 됨 |

적용 가치: 낮음 — LDPC가 아닌 별도 부호족(concatenated tree codes)의 저복잡도 디코딩/해석 기법으로 Prime ECC의 QC-LDPC min-sum 아키텍처에 직접 이식할 요소가 없음.
