### B. 알고리즘 요약

1. 대상은 regular `CR(dv,dc,w,L,M)` SC-LDPC 앙상블 (spatial position `L`개, 위치당 VN/CN `M`개, coupling `w`), `N=LM`, rate `r=1-dc/dv-delta`.
2. 문제: SC-LDPC는 threshold saturation으로 BMS 채널 용량을 달성하나, 단일 random burst erasure 정정능력은 거의 알려지지 않음.
3. 모델: 길이 `B=bM`의 단일 burst가 랜덤 시작위치에서 비트를 erase, burst 밖 비트는 BEC(`epsilon`) 또는 BiAWGN으로 수신.
4. 핵심 기법: erasure 채널에서 LLR 분포를 스칼라 erasure 확률로 환원한 density evolution(DE) scalar recursion으로 위치별 erasure 확률 `x_z^{(t)}` 갱신.
5. 최대복구 burst 길이 `bBP(epsilon)=sup{b : lim_{T->inf} Pe(T,b,epsilon)=0}` 정의 - burst가 BP threshold까지 회복되는 임계.
6. 확장: BiAWGN에서는 양자화 LLR 분포 DE로 `bBP(No)` 계산, 동일 capacity면 `bBP(epsilon)≈bBP(No)`(채널 무관성) 관찰; `1<=bBP<=k`에 대한 `w` 필요/충분조건 유도.
7. 구현/근사: 유한 `M`에서 error-floor를 size-2 stopping set(모든 이웃 CN이 최소 2회 연결) 평균개수 `E[N2]`로 추정, `Ak ~ O(LM^{2-dv})`.
8. 학습/최적화 없음 - DE 임계 계산과 stopping set 조합 카운팅(Theorem 1) 기반 해석.
9. 결과: `Cn(3,6,w,L)`은 `epsilon≈0.488`, `Cn(4,8,w,L)`은 `≈0.497`에서 `bBP=0`; `w`↑에 `bBP`↑·`dc`↑에 `bBP`↓; 유한 `M=100`에서도 error-floor를 식(6)이 잘 예측.
10. 한계: 순수 DE/조합 이론+시뮬, HW 미설계, burst erasure 채널·SC-LDPC 전용으로 NAND read/QC-LDPC와 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SC-LDPC 앙상블 구성/DE 임계 | 대응 없음 | Prime ECC는 block QC-LDPC로 spatial coupling 구조 부재 |
| burst erasure 정정 분석 | 대응 없음 | NAND 채널은 RBER/soft-read이지 burst erasure 모델이 아님 |
| size-2 stopping set error-floor 추정 | 대응 없음 (해석 도구, 코드 모듈 아님) | 개념은 error-floor 분석에 참고되나 QC-LDPC·NAND 채널로 직접 이식 불가 |

적용 가치: **낮음** - 부호클래스(SC-LDPC)와 채널모델(단일 burst erasure)이 Prime ECC binary QC-LDPC/NAND와 모두 달라, 부호설계·디코더·HW 어디에도 직접 이식할 요소가 없다.
