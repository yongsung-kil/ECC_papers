### B. 알고리즘 요약

1. 대상 부호: NAND용 `QC-LDPC`, code length 2KB, column weight 4, row weight 36, rate 8/9, AWGN 채널로 flash 모사. 4-cycle 없는 H를 computer search로 구성.
2. 문제: LDPC의 BP 복호가 latency 커서 NAND read 성능 저하 → retention error 특성을 활용해 iteration을 줄이는 것이 목표.
3. 모델: MLC 셀 threshold voltage를 normal 분포로 모델링(식3), retention error를 `λ1=α1|μk-μi|`(transfer), `ρ1=β1|σk-σi|`(instability) factor로 정의.
4. 핵심 관찰: retention error가 numerical-correlation 특성 — 주 전이 "00→01"(46%), "01→10"(44%), "01→11"(5%), "10→11"(2%)로 P/E 무관하게 비율 일정. → 셀 내 LSB/MSB 2bit 상호 영향.
5. 6가지 관찰(Observation 1~6): 검출된 lower/upper page 값과 fuzzy area 여부로 같은 셀 다른 bit의 값 확률을 추론.
6. 핵심 기법 1(codeword layout): 기존엔 셀의 2bit가 서로 다른 codeword. REAL은 codeword를 중간에서 잘라 같은 MLC 셀의 2bit가 한 codeword에 들어가게 재배치 → intra-cell 상관 활용 가능.
7. 핵심 기법 2(REAL 복호): 표준 min-sum(odd/even iteration에서 증가/감소 순 CN 갱신, scaling `α`)에 bit node 갱신 `Vj,i = Pj + Ej + ΣCk,j`로 extra decision `Ej` 추가.
8. `Ej` 규칙: 6 observation·부호(Pc·HDc)에 따라 하드코딩 — `Ej=-1`(upper page bit '1' 유도), `Ej=3`(lower '0' 유도), `Ej=0.75·Pj`(correct 신뢰) 등.
9. 결과: LD-MS 대비 latency 26.44%, PD-BP 대비 33.05% 감소(SP/SNP 3.5dB). 낮은 SP/SNP(retention 심함)에서 iteration·decoding failure rate도 baseline보다 대폭 낮음. NAND read 성능 개선.
10. 한계: HW 미설계(Matlab 시뮬만), gate count/throughput 없음, MLC 2bit-per-cell intra-cell 상관에 강하게 의존.
