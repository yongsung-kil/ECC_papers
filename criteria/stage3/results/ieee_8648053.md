### B. 알고리즘 요약

1. 대상: AWGN+16-QAM 채널, IEEE 802.11 표준 불규칙 LDPC (`N=1296`, `R=0.5`), 50 iteration.
2. 문제: 유한정밀도 message passing은 정밀도 하락 시 성능 급락, 기존 정보병목(IB) 복호기는 regular LDPC나 특별 최적화 irregular에만 적용.
3. 핵심 아이디어: 노드 연산을 relevant-information 최대화 `LUT`로 대체, LLR 대신 정수 cluster index `t`만 교환 (IB method + discrete DE로 `p(x,y)` 생성).
4. 확장 1단: discrete DE를 임의 degree distribution `λ(z),ρ(z)`으로 일반화, degree별 `p_d(x,y)`를 평균화하여 `p_d̄(x,t)` 추적.
5. 문제 발견: degree가 다르면 같은 index `t`가 다른 의미(LLR)를 가짐 → 평균화 시 belief 왜곡 (message misalignment).
6. 정보이론 정식화: `I(X;T,D)=I(X;D|T)+I(X;T)`, 여분정보 `I(X;D|T)`가 KL divergence 기댓값으로 표현됨 (식 5,6).
7. 핵심 기법 message alignment: `I(X;D|Z)≈0`이 되도록 cluster index를 재정렬하는 결정론적 매핑 `p_d(z|t)` 탐색 (식 7,8, KL 최소화).
8. 알고리즘: degree별로 순차적으로 `p_d(x|t)`를 평균분포 `p_d̄(x|z)`에 정렬, 안정화까지 반복 (Fig.3 flow graph).
9. 결과: 정렬 후 `|Z|=|T|=16`(4-bit) 유지하며 double BP 대비 0.15 dB 열화(BER 1e-3~1e-6), min-sum 능가, alignment 없으면 BER 1e-4에서 발산.
10. 한계: 시뮬레이션만·HW/gatecount/throughput 미제시, 무선(802.11) 채널 가정, LUT는 코드·채널별 오프라인 재설계 필요.
