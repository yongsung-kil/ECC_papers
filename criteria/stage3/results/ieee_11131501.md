### B. 알고리즘 요약

1. 대상 시스템은 5G 이상의 다중 트래픽 클래스(eMBB/URLLC/mMTC)를 지원하는 coded random access 네트워크로, Poisson 도착 부하 하에서 성공적으로 수신되는 패킷 확률을 다루는 Poisson receiver(PR)/coded Poisson receiver(CPR) 프레임워크를 대상으로 한다.
2. 기존 CPR은 안정영역(stability region, 모든 패킷이 확률 1로 성공 수신되는 부하 집합)의 상한(outer bound)과 향상 기법이 불충분하다는 문제를 다룬다.
3. 채널모델은 φ-ALOHA 수신기(결정론적 부하 `n`을 성공 패킷 수 `φ(n)`으로 매핑하는 함수)로 일반화하며, SIC(successive interference cancellation) 반복 복호를 가정한다.
4. 핵심 기법은 convolutional LDPC 코드 구성법(spatial coupling)을 CPR에 적용한 convolutional coded Poisson receiver(CCPR): L개의 base CPR 이분그래프를 연결(concatenate)하고 수신단을 윈도우 `w` 내에서 재배선(rewire)한다.
5. 핵심 안정성 조건은 밀도진화(density evolution) 재귀식 `p^(i+1) = f(h(p^(i)); G)`이며, 이는 convolutional LDPC의 DE와 동일한 scalar admissible system 형태로 정리된다.
6. 확장으로 potential function `U(p;G)`를 정의해 세 가지 임계값 `G_s* < G_conv* < G_up*`(단일계 임계값/potential 임계값/potential 상한)의 관계를 증명(Theorem 9)한다.
7. shortening 기법(LDPC의 pruning과 유사, 마지막 `w-1` 단계 부하를 0으로 설정)으로 CCPR을 구성하고, threshold saturation 정리(Theorem 7)로 윈도우 크기 조건 하 안정성을 보장한다.
8. 최적화/학습 절차는 없고, 수치해석은 density evolution 재귀식을 반복평가(그리드 서치, step size `δ=0.0001~0.01`)하여 percolation threshold를 수치적으로 탐색한다.
9. 결과: convolutional IRSA(w=2~4)에서 안정영역이 outer bound에 근접하며 `|G_conv*(L,w) - G_conv*|`가 최소 0.0001 수준까지 좁혀짐을 수치 확인했다.
10. 물리계층 부호(LDPC H-matrix)나 실제 채널코딩 디코더 구현은 전혀 다루지 않으며, 전부 MAC 계층 랜덤액세스 프로토콜의 이론적 안정성 분석(수학적 증명+수치해석)에 국한된다.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| convolutional LDPC 아날로지(spatial coupling, threshold saturation) | 대응 없음 | 실제 LDPC 부호가 아닌 MAC 프로토콜 안정성 이론에 개념만 차용 |
| density evolution 기반 안정영역 분석 | 대응 없음 | 이분그래프 기반이나 대상이 패킷 랜덤액세스이며 binary LDPC 복호기와 무관 |
| SIC(successive interference cancellation) | 대응 없음 | 다중 사용자 패킷 충돌 해소 기법으로 NAND 채널 복호와 무관 |

> 적용 가치: 낮음 — LDPC 부호/디코더가 아니라 무선 다중접속 프로토콜의 수학적 안정성 이론으로 Prime ECC 코드베이스와 접점이 없음.
