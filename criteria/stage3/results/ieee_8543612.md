### B. 알고리즘 요약

1. 대상은 `2^L-PAM` 변조 + L개 nested binary LDPC 성분부호로 만드는 멀티레벨 Construction D' 격자(Poltyrev/AWGN 채널), `n`=1024~10000, 성분부호 rate `R0`~`R1`.
2. 문제: 기존 Construction D LDPC 격자는 효율적 인코딩·다단복호가 없고, 패리티행렬 nesting 제약으로 성능이 열위였음.
3. 성분부호를 개별 binary 부호로 인코딩하되, 이전 레벨 기여를 syndrome 벡터 `s_ℓ`로 흡수하는 순차 인코딩(Theorem 1)을 제시 (식 13).
4. 다단복호는 재인코딩 없이 coset code `C_ℓ(s_ℓ)`만 복호하면 되며, 확장 패리티행렬 `H'=[-I | H_ℓ]`로 affine 제약을 LLR `∞`(식 22)로 주입해 표준 BP 디코더 재사용.
5. 핵심: congruence mod `2^(ℓ+1)`가 Lemma 1 조건 하에서만 순차 인코딩 가능 -> 이 조건이 nesting 완화의 근거.
6. 일반화(Definition 1): `H_ℓ ≡ F_ℓ H_(ℓ-1) (mod 2)` 만족하면 되며, 패리티행렬 자체는 nested일 필요 없음(성분부호만 nested).
7. 구현 트릭: check splitting으로 최고rate 부호 `H_(L-1)`에서 하위 레벨 행렬을 열가중치 보존하며 생성, ALT형 유지로 선형시간 인코딩.
8. 설계 절차: PEG 기반 check splitting(girth 최대화)+삼각형 변형; rate 설계는 VNR 최소화 최적화(식 33)로 `R0,R1` 선택.
9. 결과: n=1024에서 WER=10^-5 @ VNR 2.2865dB로 polar lattice와 동등, LDLC에 근접하되 복잡도 훨씬 낮음. n=1000에서 R1 높은 부호의 4-cycle로 error floor 관측.
10. 한계: 모두 시뮬레이션(HW 미설계), variable-regular(dv=3)만 다룸, check splitting이 성분부호 degree 분포를 강제하는 제약 존재.
