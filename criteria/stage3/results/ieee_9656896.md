### B. 알고리즘 요약

1. 대상은 QC-LDPC(base matrix + `p`-lifting circulant)이며, 컬럼 레이어드=shuffled 디코더로 VN을 `n_b`개 컬럼블록 레이어로 나눠 순차 갱신한다.
2. 문제: 레이어 갱신 순서(schedule)가 error floor를 바꾸는데, row layered용 [1] 모델을 column layered로 확장한 분석이 없었다.
3. 채널/알고리즘은 [1]과 동일(BI-AWGN, box-plus SPA), 차이는 message-passing schedule(column vs row)뿐.
4. 핵심: LETS(leafless elementary TS)를 상태벡터 `x̃(ℓ,j)`(레이어 j, 반복 ℓ)로 두고 선형 상태공간 모델 `x̃ = Gj(ℓ)Aj·x̃ + Bj·L + ...`로 SPA 동역학을 근사.
5. 전이행렬 `Aj`, 게인행렬 `Gj(ℓ)`가 레이어 스케줄(TSLP)에 따라 바뀌며, 이것이 TS failure rate와 error floor를 좌우.
6. missatisfied CN의 선형 게인은 부분게인 `θk`의 곱 `ḡi=∏θk1·∏θk2`(식 8)로 계산 - 부분게인을 사전 계산해 스케줄 최적화 복잡도를 낮춤.
7. 근사 DE로 한 스케줄의 CN↔VN 분포 평균 `ψ̄j`를 구해 모든 스케줄에 재사용, `n_b!` 순열 전탐색을 회피.
8. 최적화: 지배 LETS의 전이행렬 지배 고유값 `r̃`를 최소화하는 후보 스케줄만 골라 정확 DE로 error floor를 확정.
9. 결과: C1(640,192, rate 0.3, dv=5)은 모든 컬럼 스케줄이 동일 `r̃=16.9536`(row보다 큼→평균적으로 열보다 행이 우수), C2(576,432, rate 0.75 WiMAX)는 최적 스케줄이 원래 순서보다 소폭 개선.
10. 한계: SPA(box-plus)·시뮬 전용, HW 미설계, 저rate 부호 예제, min-sum 양자화 디코더로의 직접 전이 미검증.
