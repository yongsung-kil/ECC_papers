> 총평: 양자화 min-sum의 CN 업데이트에 임계 τ 기반 선택적 attenuation/offset을 더해 waterfall 유지하면서 error-floor를 낮추는 저비용 기법 - min-sum 기반 Prime ECC 디코더에 이식성 상.

### B. 알고리즘 요약

1. AWGN/BPSK, 양자화(finite precision) LDPC 디코더 대상, 예제 부호 `(8000,4000)`, PEG `(1008,504)`, Tanner `(155,64)`, 5-bit 균일 양자화(`Δ=0.15`, `max=2.25`).
2. 문제: 양자화 MSA는 고SNR에서 error floor가 높고, waterfall 개선용 AMSA(감쇠 α)·OMSA(offset β)는 오히려 MSA보다 error floor가 더 나쁠 수 있음.
3. 가정: error floor의 주원인은 trapping/absorbing set 등 problematic object이며, 그 내부 VN들이 작은(unreliable) LLR 크기를 가짐.
4. 관찰: 고SNR에서 problematic object 바깥 CN LLR은 빠르게 max로 성장, 내부는 작은 크기에서 출발 - CN이 받은 최소 VN LLR 크기 `min|Vi'j|`로 CN이 object 내부인지 추정.
5. 핵심 기법(TAMSA): CN 업데이트를 조건부로 - `min|Vi'j| > τ`면 일반 MSA(감쇠 없음), 아니면 `α`(<1) 감쇠 적용 (식 7).
6. 핵심 식 의미: object 내부로 의심되는 CN(작은 LLR 수신)의 메시지 성장을 늦춰 채널의 잘못된 메시지 강화를 억제, 신뢰 높은 바깥 CN이 오류를 정정하도록 유도.
7. 확장(TOMSA): 감쇠 α 대신 offset `β`를 조건부 적용 (식 8) - 같은 목적.
8. 위치·구조 정보 불필요: 임계 테스트 하나만 추가, α/β는 고SNR에서 소수 CN에만 적용됨.
9. 결과: `(1008,504)`에서 TAMSA `[α=0.75,τ=1.75]`가 SNR=4dB에서 AMSA/OMSA 대비 error floor 1자리수 이상 개선, waterfall은 AMSA/OMSA와 동등(약 0.1dB 손실 수준), Tanner 부호도 고SNR에서 AMSA/MSA 상회.
10. 한계: HW 미설계·시뮬만, α/β/τ 부호별 튜닝 필요, gatecount/throughput 미제시, layered/single-min 결합은 future work.
