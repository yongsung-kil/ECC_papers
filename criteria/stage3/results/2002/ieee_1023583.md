### B. 알고리즘 요약

1. 대상: `X={x1..xn}`의 `b`개 블록으로 이루어진 BIBD의 incidence `b×n` matrix `Q`; `λ=1`일 때 `Q`의 null space가 `(n,n-r)` LDPC, `Q^T`는 `(b,b-r)` LDPC.
2. 문제: 반복 복호에 적합한 LDPC를 랜덤이 아닌 조합론적 BIBD로 체계적으로 구성.
3. BIBD 조건: 각 object가 정확히 `γ`개 블록에, 임의 두 object가 정확히 `λ`개 블록에 함께 등장, block size `ρ≪n`.
4. 구성 클래스: Bose의 방법으로 `λ=1`인 5개 클래스(Class I~V)를 파라미터 `(n,b,γ,ρ)`로 명시(예: Class-I `n=6t+1, ρ=3`).
5. 핵심 성질: BIBD-LDPC의 Tanner graph는 길이-4 사이클을 포함하지 않음(girth `≥6`).
6. 고rate 부호: `Q^T`의 null space가 고rate `(b,b-n)` 부호 `Ca`를 준다.
7. QC화: Class I/II/IV의 `Q^T = [M1,M2,...,Mt]`(각 `Mi`는 `n×n` circulant) → QC-LDPC 형태, shift-register 인코딩 가능.
8. 확장: circulant `δ`개 제거(`δ<t`)로 length `(t-δ)n`의 rate 조절된 새 부호 생성; row splitting/column deletion으로 파생 부호 구성.
9. 결과: Class-V `(1044,899)` 1.6 dB, `(3498,3233)` 1.09 dB, Class-II `(6700,6499)` 0.73 dB(Shannon limit 대비, BER≈1e-4~1e-6).
10. 한계: 순수 구성법, 복호 알고리즘·HW 미제시; `λ=2`(예 `(876,804)`, 1.43 dB)는 길이-4 사이클 포함하나 SPA 성능 양호(예비 결과).

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| circulant 기반 QC-LDPC H-matrix 구성 | ecc_top.cpp Load_PCM() | Prime도 base+circulant shift QC-LDPC이므로 부호구조 성격은 정합, 다만 부호 교체·재검증 부담 |
| shift-register(circulant) 인코딩 | encoder.cpp LDPC_encoder_4KB(), Generate_PCM_encoding() | QC 순환구조로 인한 저복잡도 인코딩 개념이 dual-diagonal 인코더 경로와 닿음 |
| circulant 제거로 rate/length 조절 | ecc_top.cpp Load_PCM() | 부호 파라미터 스윕 아이디어이나 Prime의 고정 z=32 구조와 정합 필요 |
```

적용 가치: **중간** — binary·QC·고rate라는 점에서 Prime의 부호군과 성격이 일치하고 순환 인코딩 발상이 유용하나, 복호/HW 기여가 없고 특정 QC-LDPC를 교체·재검증해야 하므로 즉시 이식 가치는 제한적이다.
