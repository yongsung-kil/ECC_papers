### B. 알고리즘 요약

1. 대상은 정보/패리티 비트가 무작위 소실되는 **erasure 채널**(정보만 소실=partial, 둘 다=general)이며, capacity는 `δ < 1-R`.
2. tornado 부호는 복잡한 cascade 다단 구조가 필요해 인코딩/디코딩이 복잡하다는 문제를 SR-LDPC로 단순화한다.
3. SR-LDPC는 PCM `H=[Hp, Hd]`를 특수구조 subblock(`Hp`)과 무작위 subblock(`Hd`)으로 분할해 정의한다.
4. 핵심은 SR-LDPC가 "패리티 완전수신 제약 하의 two-column tornado 인코더 1단계"와 **등가**임을 보이는 것.
5. 인코딩은 재귀식 `p_j - p_{j-1} = Σ_{d∈Bj} d_k mod 2`로 수행되어 tornado 서브코드와 동일 복잡도.
6. left/right degree 다항식 `λ(x)`, `ρ(x)`로부터 새로운 최적 degree 분포족(식 7,8)을 유도.
7. Theorem 1(iterative decoding PDER→0 조건 `ρ(1-δλ(x))>1-x`)을 사용해 partial 채널에서 `δ<J/K` 시 capacity 달성 증명.
8. 핵심 결과: 패리티 비트 소실에 대해 code 구조가 **불변**(Remark 4)임을 그래프 인코딩(encoding line) 논거로 증명, 패리티 추가보호 없이 general 채널 capacity 달성(Theorem 3, `δ<1-R`).
9. 부수적으로 short code용 CT(concatenated tree) SR-LDPC 인코더 변형을 제시(짝수 degree 상쇄 문제 완화).
10. 한계: 순수 이론(증명 중심), 시뮬 결과 미수록(공간제약), HW 미설계, erasure 채널 한정.
