> 총평: 고rate cycle-4-free circulant(QC) EG-LDPC 구성법으로 QC-LDPC 부호설계 관점은 참고가치 있으나, 대수 구조가 고정적이라 Prime의 특정 z=32 base-matrix로 직접 이식은 어렵다.

### B. 알고리즘 요약

1. 유한기하 중 유클리드 기하 EG(m,`p^s`)를 사용: 원점과 원점 통과선을 제거한 `EG*(m,p^s)`에는 `n = p^{ms} - 1` non-origin 점과 다수 선이 존재.
2. 문제: iterative decoding(SPA)에 잘 맞으면서 short cycle이 없는 구조적 LDPC 부호를 대수적으로 구성.
3. 핵심 가정: 유한기하의 두 선은 disjoint이거나 정확히 한 점에서 교차 → 어떤 두 행(열)도 공통 1-성분이 최대 1개(cycle-4 free 보장).
4. 선의 incidence vector를 순환 시프트하면 다른 선의 incidence vector가 되어, 선들을 `K` 개 cyclic class로 분할.
5. 각 cyclic class `Q_i`로 `n x n` circulant `H_c(i)` 구성(행/열 weight `p^s`); 교차구조상 두 circulant 간에도 공통 1-성분 ≤ 1.
6. Type-I: `y` 개 circulant을 가로로 붙인 `n x yn` 행렬 `H(1)`의 null space → quasi-cyclic(=`y=1`이면 cyclic), 최소거리 ≥ `p^s + 1`, cycle-4 free.
7. Type-II: `H(1)`의 transpose `H(2)`의 null space → cyclic 부호, 최소거리 ≥ `y p^s + 1`.
8. `H(2)`의 각 열을 `p^s`개로 column splitting(무게를 회전 분배해 열 무게 1) → Gallager 원형(열무게 `y`, 행무게 `p^s`)의 circulant EG-Gallager LDPC.
9. 결과: `EG*(3,2^3)` 기반, SPA로 복호 시 최소거리 ≥9. (4599,4227) 부호는 Shannon limit 1 dB 이내, (4088,3716) 부호는 1.1 dB; EG-Gallager 부호 2종은 0.65/0.87 dB 이내(BER=1e-5).
10. 한계: HW 미설계, 2페이지 요약, 부호 파라미터가 유한기하 대수구조에 고정되어 임의 rate/length 자유도 제한.
