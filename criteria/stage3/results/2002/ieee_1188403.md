### B. 알고리즘 요약

1. 시스템은 진폭·위상 모두 변하는 frequency-nonselective, time-selective(block-independent) 복소 fading 채널, block 길이 `N`(=coherence time)마다 pilot 심볼 1개 + 코드 심볼 `N-1`개, `(3,6)` regular LDPC (rate `R=0.5`).
2. 풀려는 문제: 무선 fading에서 pilot과 코드 심볼 간 최적 전력배분과 error-free 수렴 조건을 정량화("얼마의 pilot 에너지가 필요한가").
3. 채널 모델: fading 계수 `c`는 `E[|c|^2]=1` zero-mean circular complex Gaussian(Rayleigh 진폭 + uniform 위상), AWGN `n_k`.
4. 핵심 기법: factor graph의 **channel constraint node**에서 생성되는 메시지(variable/check 노드 간 표준 BP와 별개)에 대한 여러 알고리즘 제시 — Sum-Product(지수 복잡도), Perfect CSI, Pilot-Only(PO), PDCDF(genie 하한).
5. 제안 알고리즘 QDF(Quantized Decision Feedback): incoming 메시지를 threshold `T`로 `{+1,0,-1}` 양자화해 `|r_i|>T`인 심볼을 pilot처럼 사용, 채널 노드가 반복마다 `q_k` 재계산(joint detection/estimation).
6. 대부분 알고리즘(QDF 제외)은 채널 메시지가 반복에 불변이라 초기 메시지 밀도만 evaluate하면 됨.
7. 구현 디테일: discretized density evolution, 각 메시지를 8-bit uniform 양자화(`2^8-1` 레벨, 범위 (-35,35))로 `E_b/N_0` threshold 계산(BP 상한).
8. 최적화: `E_b/N_0` binary search(해상도 0.001 dB, ≤1000 iter, BER<1e-6)와 pilot/코드 전력배분 수치 최적화.
9. 결과: 전력배분 최적화로 1~2 dB 개선; `N=20`에서 QDF threshold ≈4.4 dB (PO 대비 1.0 dB↓, PDCDF 하한보다 0.8 dB↑), Perfect CSI는 3.06 dB.
10. 한계: 순수 DE 이론 분석(HW 미설계), 무선 fading·pilot·위상추정 전용, memoryless·scalar 아닌 vector 채널이라 표준 stability 조건을 변환해 적용.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| channel constraint node 메시지 / soft-info | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | fading+pilot용 LLR 생성이라 NAND RBER/soft-read 채널과 모델이 달라 대응 낮음 |
| QDF 3-레벨 양자화 decision feedback | 대응 없음 | NAND 복호엔 pilot/decision-feedback 개념 없음 |
| variable/check 노드 BP 반복 | `decoder.cpp` `LDPC_Decoder()` | 표준 BP만 사용(개선 없음), min-sum도 아님 |
| density evolution 수렴/stability 분석 | 대응 없음 | 부호설계 이론 도구, 코드에 이식 대상 아님 |
