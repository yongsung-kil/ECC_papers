### B. 알고리즘 요약

1. 대상: 랜덤 이분그래프 앙상블 LDPC 부호를 소실채널(erasure)에서 반복복호할 때의 유한블록길이 평균 오류확률.
2. 문제: [1](Di 등)의 재귀식은 모든 정규 LDPC에 적용되나 평가에 `O(n^5)` 연산이 필요해 큰 블록길이에 비현실적.
3. 기존 [1]은 left degree 2,3에 한해서만 효율적 재귀를 유도(Theorem 2.2).
4. 핵심 기법: stopping set이 없는 그래프 수 `N(v,c,d)`를 세는 대안 재귀식 유도 — variable node `v`개, simple check node `c`개, super check node 1개 구조 사용.
5. 식 의미: `N(v,c,d)`는 [1]의 Theorem 2.1로 부호 오류확률 평가에 사용됨.
6. 확장: block 오류확률은 super check node를 `d=0`으로 제거해 `O(n^3)`, bit 오류확률은 super check node 포함 `O(n^4)`.
7. 추가 파라미터(차수별 variable node 수) 도입으로 left-irregular right-regular 부호에도 적용.
8. 최적화 절차 없음 (해석적 재귀).
9. 결과: 다양한 블록길이의 정규/좌비정규 부호 블록오류확률 계산, 최저 left degree `l_m`과 `P_LT(G,ε)=O(ε^{R·l_m})`(작은 ε) 관계 증명.
10. 한계: erasure 채널 한정 이론, HW·시뮬 없음, 실제 min-sum/soft-read 복호와 무관.
