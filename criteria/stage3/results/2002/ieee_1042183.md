### B. 알고리즘 요약

1. 시스템: perpendicular magnetic recording 채널(PR2 타깃, 21-tap 등화, BCJR MAP 검출 + MPA LDPC 복호), 부호는 MOLR 기반 regular 고rate LDPC 5종.
2. 문제: 랜덤 LDPC는 파리티체크 행렬이 랜덤이라 인코더/디코더 복잡도가 매우 큼 → 구조적 PCM으로 구현 복잡도 절감 필요.
3. 핵심 가정: 파리티체크 orthogonality를 유지하는 조합론적 설계로 Tanner 그래프 girth를 키우면 좋은 반복복호 성능을 얻는다.
4. 핵심 기법 1단: balanced incomplete block design(BIBD)의 point-block incidence matrix를 Gallager 부호의 PCM으로 사용, 정수격자상 slope별 line(block)으로 열가중치 `j`인 orthogonal parity 구성.
5. 핵심 식 의미: index `lambda<=1`이면 두 점이 한 block 이상 공유하지 않아 length-4 cycle이 없고 girth>=6 보장(error-floor 관점 이점).
6. 핵심 기법 2단: mutually orthogonal Latin rectangles(MOLR)로 resolvable BIBD를 구성해 넓은 block size·code rate 범위를 확보, PCM이 순열(circulant column-weight-1) 부분행렬 블록들로 표현됨.
7. 부수: 순열 부분행렬 구조로 인코더 복잡도 대폭 절감, infinite slope 제외 시 성능이 다소 개선(원인 미규명).
8. 학습/최적화: 없음(결정론적 조합론 구성), rate는 정확계산 어렵고 lower bound만 제공([5]).
9. 결과: "BCJR + MPA 4회"를 10회 반복하는 구조로 시뮬, lattice code는 length 증가 시 BER 소폭 개선, 저rate lattice code는 error floor(약 1e-6)와 TPC 대비 열세, jitter 증가 시 BER 악화.
10. 한계: HW 미설계·시뮬만, PR2/자기기록 채널 및 BCJR 등화 전제, 정확 rate 산정 곤란, 짧은 저rate 부호에서 error floor.
