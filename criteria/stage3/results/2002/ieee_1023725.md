### B. 알고리즘 요약

1. 대상: 2차원 정수격자 기반 조합론적 구성의 고rate binary LDPC(Gallager code), circulant 배열 PCM.
2. 문제: 자기기록 등에서 equalization loss 최소화 위해 매우 높은 rate 부호가 필요, 구조적·거리보증 부호 요구.
3. 구성 기반: balanced incomplete block design(BIBD) `2-(v,k,1)`의 sub-design; `v`=parity bit 수, `k`=column weight.
4. 격자 정의: 직사각 격자 `L={(x,y):0≤x≤k-1, 0≤y≤m-1}`, `m`은 소수; parity bit 수 `= m·k`.
5. block(line): 기울기 `s`의 직선 `{(x, a+s·x mod m)}`로 정의, `m`개 parallel class(slope별).
6. 핵심 성질: design index `λ=1` → 같은 점쌍이 두 block에 없음 = **bipartite 그래프에 length-4 cycle 없음**(BP 유리).
7. PCM 구조: slope별 permutation 행렬 `H_{i,j}`의 array → **doubly circulant**(circulant 행렬들의 배열).
8. 거리/구조 정리: Lemma1(resolvable 2-design), Lemma2((m,k=3) lattice BIBD는 Pasch-free), Conjecture3.1(generalized Pasch 없음); min-distance bound 제시.
9. 적용/결과: PR 수직자기기록 채널에서 채널 trellis에 BCJR + LDPC message-passing turbo-equalization; 정규화 기록밀도 2에서 random 부호 및 Kirkman-system 부호와 비교(Fig.1), 성능 동등~우수. column weight 증가로 거리 개선해도 rate loss 상쇄 못함.
10. 한계: 2쪽 ISIT 요약, rate/length 구체수치 추출 손상, HW·양자화 무언급, 채널이 PR 자기기록으로 NAND와 상이.
