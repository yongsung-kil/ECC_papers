> 총평: 소수 파라미터로 결정론적 생성되는 array-code 계열 binary LDPC로 linear-time 인코딩과 error-floor 부재를 주장. QC-like 순환치환 구조라 이식 여지는 있으나 DSL/QAM 타깃 2쪽 요약이라 중.

### B. 알고리즘 요약

1. 대상: multilevel modulation(QAM, DSL 멀티캐리어)용 **binary LDPC 부호족**, 소수 파라미터로 지정·linear-time encodable·결정론적 PCM 생성.
2. 문제: 실용 시스템에서 낮은 BER 영역의 error-floor 회피 + encoding 복잡도·latency·성능 trade-off 확보.
3. PCM 구조: array code 기반. `p`(소수), 정수 `k,j≤p`, `p×p` 항등행렬 `I`, 영행렬 `0`, 단일 순환좌/우 shift 치환행렬로 구성된 **상삼각(upper-triangular) H**.
4. 핵심 성질: 상삼각 구조가 linear-time 인코딩 보장 — 약 `(N/2)[r(j+3)+(j-3)]` XOR 연산(N=length, r=rate; 추출 손상 가능).
5. min distance: H에 `0` block 도입해도 `j=3,4`에서 최소거리 감소 없음 → error-floor 부재 근거.
6. 복호: sum-product algorithm, soft input은 수신 QAM 심볼에서 유도한 각 code bit의 a posteriori 확률.
7. 심볼 매핑: "double Gray-code labeling" — subset 선택 비트와 subset 내 심볼 비트를 별도 Gray 규칙으로 라벨링, 실/허수부 독립 demapping.
8. 적용: DSL 멀티캐리어(subcarrier마다 2~2^14 심볼 constellation), bit-interleaved coded modulation capacity 분석으로 성능/복잡도 trade-off.
9. 결과: 넓은 파라미터 범위에서 관심 error-rate(~10^-?)에 error-floor 없음; 16-QAM에서 (2209,2021) 부호로 net coding gain ~6 dB, latency 10 ms 제약 하 긴 부호로 ~6.4 dB.
10. 한계: 2쪽 ISIT 요약, HW/양자화/min-sum 무언급, 채널이 AWGN+QAM(멀티레벨)로 NAND와 상이.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| array-code 순환치환 PCM 생성/로드 | `ecc_top.cpp` `Load_PCM()` | 순환 shift 기반이라 base+circulant QC와 개념 유사, 매핑 재검증 필요 |
| 상삼각 H linear-time 인코딩 | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | 우리는 dual-diagonal 인코더 보유, 상삼각 구조는 다른 방식 |
| sum-product 복호 | `decoder.cpp` `LDPC_Decoder()` | 우리는 min-sum(tanh 근사) 보유, full sum-product는 대응 없음에 가까움 |
| error-floor 부재(min distance 설계) | 대응 없음 | 부호설계 레이어(§4 이식난이도 높음) |

적용 가치: **중간** — 순환치환·상삼각 기반 error-floor-free 고rate binary LDPC 부호설계는 관심 영역이나, DSL/QAM 타깃 2쪽 요약이라 QC 매핑·정량근거·HW를 full version에서 재확인해야 함.
