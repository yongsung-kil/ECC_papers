### B. 알고리즘 요약

1. 대상: BEC(binary erasure channel)에서 belief propagation으로 복호되는 유한길이 LDPC ensemble들의 평균 erasure 확률 분석.
2. 문제: 점근적으로 capacity 달성은 알려졌으나 유한길이 거동·최적화, 특히 error-floor 예측은 미해결.
3. 핵심 관점: 고SNR(저erasure) 성능을 codeword weight spectrum이 아니라 **stopping set weight spectrum**으로 분석(BEC에서 BP 실패는 stopping set이 결정).
4. 방법: 여러 ensemble(standard, Poisson, degree-2 VN을 cycle로 추가, degree-2 VN이 서로 다른 check 이웃을 갖는 ensemble)의 stopping set 무게분포 조합식 유도.
5. error-floor 상계: 그래프의 작은 결함에서 오는 erasure 확률 기여를 tight upper bound로 제시.
6. expurgated ensemble: 특정 크기 이하 stopping set을 포함하지 않는 부분집합을 정의, 그 error-floor 상계 유도(예: (3,6)-regular length 1000에서 절반 미만이 크기 20 이하 stopping set 포함 → typical code가 훨씬 우수).
7. 성장률: `λ'(0)ρ'(1)>1`이면 최소 stopping set 크기 logarithmic, `<1`이면 expurgation으로 선형 성장 가능.
8. erasure correcting radius: 적절히 고른 ensemble의 typical code는 블록길이 `n`에 선형인 erasure 정정 반경 확보(예: (3,6)에서 ≥0.01799n).
9. 핵심 결론: 반복복호 성능은 minimum distance가 아니라 stopping set 무게분포로 결정 → degree-2 VN 구조를 신중히 제어해 설계해야 함.
10. 한계: BEC 전용 이론, HW/시뮬 없음, 1페이지 요약(수치·그림은 외부 웹 인터페이스 참조).
