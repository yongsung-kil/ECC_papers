### B. 알고리즘 요약

1. 대상: Reed-Solomon형 array code 기반 결정론적 binary QC-LDPC, 고rate·중간 length(약 5000까지)에서 랜덤 regular LDPC와 유사 성능.
2. 문제: array LDPC의 실용화를 위해 부호 length에 선형 복잡도인 fast encoding과 성능 보증(dmin)이 필요.
3. 모델: `q`를 홀수 소수, 링 `R=GF(2)[x]/(1-x^q)`에서 원소 `x`가 order `q`, RS형 parity-check로 array code `C_R(q,j)` 정의.
4. 핵심기법 1단: 링 `R`의 정규 행렬표현 `ρ`(q×q permutation matrix)로 array code를 binary화 → parity-check `ρ(Hc)`가 row/col weight `q,j`의 sparse 행렬.
5. 식 의미: `ρ(C)`가 permutation matrix이므로 `ρ(Hc)`가 constant weight sparse가 되어 QC-LDPC 조건 성립, rate 식(3) 유도.
6. 핵심기법 2단: 부호어를 `g(z)=(z-1)(z-C)...(z-C^{j-1})`로 생성, 기본대칭함수 `s_l(·)`로 이루어진 sparse binary generator matrix `G` 구성 → 선형복잡도 인코딩.
7. 구현 디테일: `G`가 `(q(q-j)+j-1) x q^2` 크기, 원소는 대칭함수 축약 표기, sparse 성질이 fast encoder 핵심.
8. 성능 보증: Theorem 1 — `G`의 첫 `j-1` 행 weight `2q`, 나머지 행 weight 상한 및 `dmin` 상한 도출.
9. 결과: `dmin` 상한이 `j=3,4,5,6`에서 각각 6,12,20,32; `j=3`은 tight하며 첫 distance spectrum 항 `dmin=6`, multiplicity `q`.
10. 한계: HW 미설계, 디코더/BER 시뮬 없이 부호 구성·거리특성 이론뿐, rate/length 확정 수치 미제공.
