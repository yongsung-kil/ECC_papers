### B. 알고리즘 요약

1. 대상: 매우 고rate(>0.9, recording/optical 채널용)·저 column-weight regular Gallager LDPC의 결정적 구성.
2. 문제: 랜덤 구성은 구조가 없어 HW 복잡, 기존 projective/affine geometry(Kou-Lin-Fossorier) 계열은 고rate·단길이 영역에서 파라미터(길이·column weight) 선택폭이 좁음.
3. 기반: `2-(v,k,1)` BIBD의 point-block incidence matrix `A`를 PCM `H`로 사용 (point=parity check, block=bit); row weight `k`, column weight `r`, rate `R=(b-rank(H))/b`.
4. 핵심 관찰: Steiner triple system에 Pasch configuration(6점 4블록, 각 점 2회)이 있으면 `H`에 선형종속 4열 → `d_min=4`; anti-Pasch면 `d_min>=6`.
5. 문제점: anti-Pasch STS는 극히 희소(2-(15,3,1) 80개 중 1개만) → 고rate 구성 어려움.
6. 핵심 기법: `m x k` 정수격자(m=prime)에서 slope `s`(0<=s<=m-1) 직선을 block으로 정의하는 affine geometry resolvable BIBD 구성.
7. 정리: (Lemma1) slope별 직선 집합은 2-design; (Lemma2) `(m,k=3)` 격자 BIBD는 anti-Pasch → `d_min>=6` 자동 보장.
8. 구조 이득: 결정적·resolvable(parallel class 분할)이라 구조화되어 저복잡 구현 가능, girth `g=6`.
9. 결과: girth-6 부호는 iteration 5~6회 제한된 고속·저복잡 응용에 적합, Fig.1/2로 rate-length 하한 제시.
10. 한계: 순수 조합론 구성(HW/BER 시뮬 없음), girth 6 고정, QC-circulant 구조 아님, 성능은 이론적 `d_min` 논거뿐.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| H-matrix/부호구조 구성(BIBD affine geometry) | [ecc_top.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md) `Load_PCM()` | PCM 로드 지점이나 우리는 고정 QC-LDPC라 재검증 부담 큼 |
| anti-Pasch girth-6/d_min 보장 (error-floor) | 대응 없음 | Prime ECC는 특정 QC-LDPC 고정, 부호설계 교체는 이식난이도 높음 |
| resolvable parallel class(구조화) | 대응 없음 | QC circulant 구조와 상이, 직접 대응 모듈 없음 |

적용 가치: **낮음** — 결정적 고rate 구성법으로 error-floor를 d_min으로 보증하는 아이디어는 흥미로우나, Prime ECC는 특정 QC-LDPC가 고정되어 부호설계 교체 부담이 크고 BIBD/affine 구조가 circulant QC와 정합되지 않는다.
