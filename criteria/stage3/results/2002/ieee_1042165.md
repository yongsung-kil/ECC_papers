### B. 알고리즘 요약

1. 시스템: 이상적 PR4 등화 + AWGN의 이진 ISI 채널, rate 4/5·블록길이 5120 LDPC 부호 사용.
2. 문제: 용량접근 부호가 저SNR로 동작하므로 동기화기가 저SNR에서 cycle-slip을 겪어 버스트 오류를 유발, 이를 제거해야 함.
3. 모델: 느리게 변하는 위상 드리프트 `theta(t)`가 있는 PR4 채널, 정합필터 후 샘플링해 (2)식 샘플 획득.
4. 핵심기법: 채널 soft-output(APP 기반 LLR)을 입력으로 cycle-slip 검출기(CSD)를 이진 ISI 채널로 일반화.
5. 검출통계: 윈도우 `M`개의 soft 정보로 통계 `T`를 구성, 중심극한정리로 `T`를 가우시안 근사하여 임계값 `gamma`와 비교.
6. 판정: `P_D`(검출확률)·`P_FA`(오경보확률)로 임계값과 최소 윈도우길이 결정, cycle-slip 위치를 알아냄.
7. 구현: forward-only APP 알고리즘(survival memory 6/30)을 타이밍루프·반복복호 검출기로 사용.
8. CSD-aided 반복 타이밍복원: CSD가 fire하면 slip 위치 기준으로 파형을 분할·재샘플, LDPC 복호로 패리티 만족 시 정지.
9. 결과: 시뮬(SNR 8.6 dB, 임계값·윈도우 90)에서 재샘플(동기화) 반복 횟수를 유의하게 줄여 수렴 가속, 성능은 [6]과 동등.
10. 한계: HW 미설계·시뮬만, 자기기록 특유의 타이밍/cycle-slip 문제 전제(블록당 slip 최대 1회 가정).
