### B. 알고리즘 요약

1. 시스템: 2-D 자기기록/차세대 저장장치의 2-D ISI 채널, 사용자데이터를 100x100 행렬로 스캔, AWGN 잡음, ECC는 block length 10000·rate 1/2 regular (3,6) LDPC coset code.
2. 문제: 트랙간격 축소·2-D 매체에서 ISI가 2-D로 발생, 이를 완화하는 등화와 LDPC 복호를 어떻게 결합할지.
3. 핵심 가정: 채널 출력 `r = point spread function * a + noise`, 2-D convolution ISI, 입력 Gaussian 가정하 Wiener 필터 설계.
4. 핵심 기법 1단(MMSE 방식): Wiener 필터로 2-D MMSE 등화 후 LDPC 복호, 복호 실패 시 LDPC가 soft(codeword 평균) 또는 hard(추정 codeword) 정보를 필터로 되돌려 반복(turbo equalization).
5. 핵심 식 의미: 각 iteration에서 Wiener 필터 연산을 LDPC로부터의 정보로 갱신, 최대 10회 반복.
6. 핵심 기법 2단(APP full-graph): LDPC bipartite 그래프와 채널 ISI 그래프를 결합한 3-level 그래프 위에서 message passing으로 등화·복호를 동시 수행.
7. 부수: full-graph 스케줄 `x->c, x->r, ...`, 2-D ISI가 만드는 length-4 cycle(`r-x-r-x-r`)이 근사 APP 품질을 떨어뜨림.
8. 학습/최적화: 없음(구조적 스케줄 비교), guard band(all-1) 삽입으로 sector 격리·알고리즘 종료 제공.
9. 결과: soft 정보 반복 Wiener(10회)가 최고 성능으로 BER 1e-5에서 ISI-free LDPC 대비 약 2 dB, full-graph는 short cycle로 성능 열화, undetected error 없음.
10. 한계: HW 미설계·시뮬만, 2-D ISI/등화기 전제, irregular 부호 이득은 대블록에서만.
