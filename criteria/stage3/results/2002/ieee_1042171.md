### B. 알고리즘 요약

1. 시스템: 자기기록(magnetic recording) PR 채널(precoded EPR4)에 고rate LDPC 부호 + 랜덤 인터리버를 결합한 코딩 시스템.
2. 문제: 관심 SNR 영역이 매우 높아 시뮬레이션으로 error-rate를 얻기 불가능 → 해석적 성능 예측 도구 필요.
3. 핵심 가정: 오류사건 시 두 codeword 간 squared Euclidean distance가 부호 제약 없는 경우와 근사적으로 같다고 가정(고rate 부호에서 정당화).
4. 핵심 기법 1단: [1]의 bounding 기법을 frame error 확률로 확장, weight distribution `A_w`와 PR 채널의 입력-Hamming/출력-Euclidean distance 쌍 수를 결합해 union bound 식 (1)(2) 계산.
5. 핵심 식 의미: bound 성립의 관건은 특정 PR 타깃에 대한 distance-쌍 분포 계산이며, 표준 PR 타깃은 직접 계산 가능.
6. 핵심 기법 2단: weight distribution을 정확히 알 수 없으므로 [3]의 Krawtchouk 다항식·MacWilliams 항등식 기반 방법으로 상·하한을 추정.
7. 대상 부호: finite-geometry LDPC(PG-I, PG-II, EG-II) — 최소거리 `d`, `d_dual`이 알려져 있어 weight moment function 경계 도출에 유리.
8. 부수: 랜덤 앙상블 `(n,j,k)` 부호(예 (5000,3,30))는 [8]의 weight distribution bound를 그대로 사용.
9. 결과: 고SNR에서 log-BP 시뮬 결과가 ML upper bound 안에 들어와 bound 유효성 확인, 추정 weight로 얻은 bound와 exact weight bound 차이 < 1 dB.
10. 한계: 앙상블 bound와 특정 부호 시뮬 비교라 주의 필요, moderate SNR과 저rate 부호(PG-I(2,3),(2,4))에선 bound 미검증, bound가 다소 loose.
