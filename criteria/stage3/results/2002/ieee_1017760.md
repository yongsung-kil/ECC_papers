### B. 알고리즘 요약

1. 시스템: 수직자기기록(perpendicular recording) PR 채널, 고rate(19/20~29/30) 규칙 LDPC + BCJR/MPA turbo-equalization.
2. 문제: 랜덤 Gallager LDPC는 인코더가 복잡하고, turbo/RA 계열은 본질적으로 저rate라 초고밀도 기록에 부적합.
3. 핵심 착상: 조합설계 BIBD `(v,k,λ)`의 point-block incidence matrix를 그대로 parity-check matrix로 사용 → column weight `r`, row weight `k` 규칙 LDPC.
4. girth 조건: index `λ=1`(2-design)으로 제한하면 어떤 두 점도 한 block에만 공존 → 4-cycle 없음(self-orthogonal), 반복복호에 필수.
5. 구성법: cyclic difference family(CDF)의 base block을 group `Z_v`에서 orbit 전개 → 각 orbit이 circulant 부분행렬 → `H = [A1|A2|...]` quasi-cyclic 구조.
6. Kirkman/Steiner triple system: `k=3, λ=1` resolvable Steiner triple system(Kirkman)을 Netto의 CDF 두 구성으로 생성.
7. 인코더: `p = A_p^{-1} * (Σ A_i d_i)` — circulant 곱은 shift register + AND/XOR/FF로, 역행렬 `A_p^{-1}`도 circulant라 한 열만 ROM 저장; parity bit 위치 자유 배치(RLL 위반 회피).
8. 복호: PR 채널 BCJR 5bit soft → MPA(min-sign 근사, 5bit likelihood) 상호 교환, "10 iter × 외부 10회" 스케줄.
9. 결과: rate 19/20~29/30 Kirkman 코드가 SNR 18.25+에서 10^-? error floor; 랜덤 LDPC(rate 16/17, 4096 info) 대비 waterfall 약 0.4dB 손실.
10. 한계: HW 인코더만 제안(디코더 HW 없음), column weight 3 고정, error floor 존재, 자기기록 PR/jitter 채널 특화(AWGN·NAND 미검증).
