### B. 알고리즘 요약

1. 광/자기 데이터저장 PR 등화채널용 LDPC 부호 설계 — 채널 `h(D)=(1+D)^n, n=1,2,3` (PR1/2/3) + AWGN 모델.
2. 문제: random PCM은 nonzero 원소 저장 메모리가 HW 구현의 병목 → 구조화된 저복잡도 부호 필요.
3. 핵심가정: RS outer code 병용이 전제이므로 inner 부호는 최소거리보다 outer RS와 호환되는 block error 통계가 중요.
4. 핵심기법: column weight `j=2` 구조 LDPC 두 가지 구성법 제안 — 6-cycle free, 넓은 rate 범위, PCM이 단순 집합으로 표현.
5. `j=2`는 min distance가 block length에 대해 로그로만 증가(Gallager)하지만, RS 병용 시 이 단점보다 저복잡도·block error 통계가 유리.
6. 결과 복잡도: `j=3`, `j=4` regular LDPC 대비 각각 3배, 4배 계산복잡도 절감.
7. 구현: structured 부호는 encoded sequence를 random-like하게 만들기 위해 interleaving 필요, precoding은 저~중 SNR 성능손실로 미사용(단 특정 precoder는 error floor 저감).
8. 결과 BER: PR1/2/3 채널에서 uncoded 대비 4~5dB coding gain(target BER 1e-4), PR1이 error-floor bound 역할.
9. `j=2`(with precoder)가 `j=3` LDPC를 능가하면서 복잡도는 약 3배 낮음(PR2 채널).
10. 한계: HW 미설계, 정량 gatecount/throughput 없음, PR 등화채널·precoder 전제, 상세 구성/rate는 final submission으로 미룸.
