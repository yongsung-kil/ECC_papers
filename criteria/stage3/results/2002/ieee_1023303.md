### B. 알고리즘 요약

1. 대상: 이진입력 AWGN 채널(noise std `σ`)에서 정규/비정규 LDPC의 message passing 복호 동역학, 무한블록길이 가정.
2. 문제: density evolution은 수치절차라 복잡 — 동역학 통찰을 위해 메시지 분포를 가우시안으로 근사하는 해석적 방법 필요.
3. 가정: 집중정리(concentration)로 각 iteration 메시지가 i.i.d., all-zero 코드워드 전송 가정, iteration `l`의 오류확률 `P_l(σ)`=variable node 메시지가 음수일 평균확률.
4. 핵심 기법: 오류확률로 표현되는 닫힌형 비선형 1-D 사상 `z ← f(z;σ)` 유도 — `σ`가 제어 파라미터, 초기값 `P_0(σ)=Q(2/σ)`.
5. 식 의미: 이 1-D 사상의 반복(iterate)이 곧 복호 오류확률을 직접 나타냄 → 동역학 정성분석 가능.
6. 안정성: 사상의 편미분으로 수렴 분석 — `z=0`이 안정 고정점일 필요충분조건 `λ'(0)ρ'(1) < e^*`.
7. threshold: `σ* = sup{σ>0 : lim_{l→∞} P_l(σ)=0}` 도출, density evolution과 일치.
8. 최적화 절차 없음 (해석), saddle-node bifurcation 발생을 (d_v=3,d_c=27) 예로 예시.
9. 결과: rate R=... d_v=3 정규 LDPC의 threshold를 DE와 비교(Table I), 정확도 0.1~0.3dB; (3,6) rate-1/2 예 σ_DE=0.880(1.11dB) vs σ_GA=0.848(1.43dB).
10. 한계: 가우시안 근사·무한길이 가정, 정성분석 위주, HW·유한길이·실제 복호 구현과 무관.
