### B. 알고리즘 요약

1. 시스템: 이진 소거채널(BEC, 파라미터 ε)에서 regular LDPC ensemble `C(n, x^{l-1}, x^{r-1})`(예 (3,6), rate 0.5)의 유한길이 성능 분석.
2. 문제: 무한길이(cycle-free) DE는 유한 블록의 error-floor를 못 잡음; 그래프의 cycle이 유한길이 해석을 난해하게 함.
3. 핵심 통찰: 반복복호 실패는 **stopping set**(모든 이웃 체크노드가 최소 2번 연결되는 변수노드 부분집합)으로 조합론적으로 특성화됨(Lemma 1.1).
4. 정리: 소거집합 내 최대 stopping set이 곧 복호 종료 시 남는 소거이므로, ensemble 평균 bit/block erasure를 정확히 계산 가능(Theorem 2.1).
5. 방법: super check node를 도입해 stopping-set-free constellation 개수의 재귀식 `A(v,c)` 유도, 각 크기 s 소거집합이 최대 stopping set일 확률로 전개.
6. 효율화: (2,r) cycle code 등 특수 경우엔 degree-one check를 peel-off하며 constellation을 한 변수노드씩 키우는 간단 재귀(2.6)로 계산.
7. ML 비교: standard random ensemble의 full-rank 행렬 개수로 ML 소거확률을 정확 계산, LDPC ensemble은 union bound 유도(Appendix B).
8. 해석: iterative vs ML 성능차 대부분이 부호 구조 자체 탓이며, ML에서도 error-floor 영역이 나타남.
9. 결과: 개별 부호가 ensemble 평균 주위로 지수적 집중, 유한길이 곡선이 cycle-free 대비 유의미하게 열화됨을 정량 확인.
10. 한계: BEC 전용, regular ensemble 평균만, 부호 개선 없음(예측 이론), HW 미설계.

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |

```

적용 가치: **낮음** — stopping set은 error-floor 이해의 핵심 개념이나 본 논문은 BEC 전용 유한길이 확률 계산에 한정되어, NAND RBER 채널의 Prime ECC 코드/디코더에 떼어 쓸 실행 요소가 없다.
