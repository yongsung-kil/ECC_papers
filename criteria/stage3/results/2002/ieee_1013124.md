### B. 알고리즘 요약

1. 시스템: AWGN/BPSK 채널, 내부 rate-1/2 feedforward convolutional code + 외부 binary block code(parity/CRC/BCH) + block interleaver 구조의 CSCC.
2. 문제: 짧은 블록(약 56~300 정보비트)에서 sphere-packing bound에 근접하는 저복잡도 반복복호 부재.
3. 모델: 전체 CSCC 코드워드의 sparse parity-check 행렬(다항식)을 유도하여 extrinsic 정보가 parity 방정식 따라 흐르게 함(TWL/Tanner-Wiberg-Loeliger graph).
4. 핵심기법: Iterative Viterbi Algorithm(IVA) — 1차 반복은 각 column을 VA로 독립 복호, 외부코드 syndrome 검사 후 branch metric을 outer code로 갱신 반복.
5. 갱신식(13): `m' = m1 + gamma*m2` 형태의 "switch-sum" — 이전 반복 hard-decision `S`가 switch 역할, `gamma`는 error propagation 억제 factor.
6. 확장: tail-biting trellis 변환(Theorem 1~3)으로 non-tail-biting CRC-CSCC도 sparse parity 다항식 확보; 2-D parity code로 장블록 대응.
7. HW 친화: `gamma` 곱셈 불필요 — metric을 "+/-" 레벨 증감으로 구현 가능(추가 회로는 shift register+few gates).
8. 최적화: `gamma`를 반복 진행에 따라 점진 증가시키는 heuristic(약 0.1dB 이득).
9. 결과: 블록길이 56~11970에서 sphere-packing bound 약 1dB 이내; 256-state VA 대비 최대 1.2dB 개선, 평균 반복 1.08~4회.
10. 성능분석: pairwise error로 side-information distance 평균/분산 유도 → weak(low-density) parity code가 강한 BCH/RS보다 IVA에서 우수(Theorem 4~6).
11. 한계: HW 실제 설계 없음(복잡도 추정만), soft-output 미제공, 장블록 수렴 느림, LDPC 미대상.
