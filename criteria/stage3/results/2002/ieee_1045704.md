### B. 알고리즘 요약

1. 대상 시스템은 IS-95 CDMA, rate 1/8, block length 1536의 Sorokine et al. `(1536,-,3)` modified Gallager (regular) LDPC ensemble.
2. 문제: 이 부호는 중간 BER에선 용량을 크게 늘리나 낮은 BER(~10⁻⁵)에서 poor minimum distance로 error-floor가 발생.
3. 원인 진단: order-4 loop 중 check-node 하나가 첫 sub-graph에 속하는 "special loop"이 weight-2 codeword(또는 syndrome weight-1 near-codeword)를 유발.
4. Theorem 1/2로 non-special/special loop 기대개수를 truncation depth `Td`로 유도: `E_ns≈(1-Td)(k-1)²`, `E_s≈(2-Td)(k-1)²`.
5. Sorokine ensemble은 `Td=0.375`에서 `E_s/E_ns≈2.6` → 무작위 선택 부호가 special-loop-free일 확률 0.036, 즉 탐색이 exhaustive.
6. 핵심 기법 SLBA(Special Loop Breaking Algorithm): 두번째 sub-graph부터 변수노드를 순차 스캔, special loop에 참여하면 같은 sub-graph 내 무작위 변수와 위치 교환(ensemble membership 유지).
7. 한 iteration = 모든 sub-graph(첫번째 제외) 스캔; special loop이 없어질 때까지 반복, 보통 2~3 iteration에 종료.
8. 표준 loop-breaking과 달리 같은 sub-graph 내 교환만 하여 결과가 여전히 ensemble 멤버임을 보장.
9. 결과: SLB 적용 code D는 FER=10⁻⁵까지 error-floor 없이 code C 대비 ~1.2dB, code B는 code A(=[11] 수준) 대비 FER=10⁻⁴에서 ~0.7dB 개선; 용량은 BER=10⁻⁵에서 orthogonal conv 대비 3배.
10. 한계: HW 설계 없음, AWGN/fading 시뮬만, IS-95 저rate CDMA 특화 구성법이며 디코더는 표준 BP 200~2000 iteration 사용.
