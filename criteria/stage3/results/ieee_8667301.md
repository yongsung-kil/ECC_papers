> 총평: LDPC error-floor의 핵심 구조(LETS/EABS) 탐색이 NP-hard임을 증명한 순수 이론으로, 이식 가능한 기법·부호·디코더가 없어 이식성 하.

### B. 알고리즘 요약

1. 대상: variable-regular `(dv,dc)`-regular binary LDPC의 Tanner graph, AWGN + iterative decoding error-floor를 유발하는 구조.
2. 문제: error-floor 주범인 LETS(leafless elementary trapping set)·EABS(elementary absorbing set)를 효율적으로 찾을 수 있는가.
3. 정의: `(a,b)` TS = 크기 `a`, 홀수차수(unsatisfied) check node 수 `b`; ETS=check 차수 1 또는 2; LETS=leaf 없는 ETS; EABS=elementary absorbing set(변수노드가 No보다 Ne에 더 많이 연결).
4. 핵심 결과: Min-b-LETS(주어진 a에서 최소 b)와 Min-a-LETS(주어진 b에서 최소 a) 모두 임의 근사인자로 NP-hard.
5. 증명 도구: Monotone 1-IN-3 SAT 등 SAT 문제의 NP-완전성을 다항시간 환원으로 이용.
6. 확장 결과: EABS에 대해서도 Min-b-EABS, Min-a-EABS가 동일하게 임의 근사인자로 NP-hard (dv=3이면 LETS=EABS).
7. 환원 절차: Monotone 1-IN-3 → Monotone 2-IN-β → Cubic Monotone 2-IN-β → α-Monotone 2-IN-β SAT을 거쳐 gadget(D_c,z, F_c,z)으로 Tanner graph 구성.
8. 강화 지점: 기존 [21]은 ETS에 상수 근사인자·RP=NP 가정이었으나, 본 논문은 bi-regular graph에서 P=NP 가정하에 "임의 근사인자" 불가로 더 강함.
9. 결과 함의: P=NP 아니면 LETS/EABS(최소 크기조차)를 찾는 다항시간 (근사)알고리즘 없음.
10. 한계: 순수 이론(하드니스 하한)만, 실제 error-floor 저감 기법·부호설계·디코더·HW·수치실험 전무.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LETS/EABS(trapping/absorbing set) error-floor 구조 | `ecc_top.cpp` `Load_PCM()` / H-matrix | error-floor은 부호구조 종속이나 본 논문은 탐색 하드니스만 다뤄 직접 개선 대응 없음 |
| trapping set 관련 수렴실패 추적 | `corner.cpp` (수렴실패 추적) | 개념적 연관, 그러나 논문에 이식할 탐색/저감 알고리즘 없음 |
| error-floor 저감 부호설계 | 대응 없음 | 논문은 저감법이 아닌 탐색 NP-hardness 증명 |

적용 가치: **낮음** — error-floor 구조(LETS/EABS) 탐색이 NP-hard라는 이론적 하한만 제공하여, Prime ECC에 떼어 쓸 부호설계·디코더·탐색 알고리즘이 없음 (설계 시 완전탐색 비용 인지에만 참고).
