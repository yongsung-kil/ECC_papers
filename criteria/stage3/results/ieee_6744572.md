### B. 알고리즘 요약

1. 시스템: cloud transmission(CTxN) 차세대 지상파 방송용 저rate LDPC, 길이 16200/64800(DVB-T2/S2와 동일), QPSK, AWGN 채널.
2. 문제: DVB-T2/S2 LDPC는 R≥1/2에서 Shannon 근접하나 quarter-rate에서 1.29 dB 손실 — 저rate 최적화 부족.
3. 모델: co-channel 간섭을 AWGN 등가로 취급, SNR threshold −2~−3 dB 목표.
4. 핵심기법 1: raptor-like cascaded PCM `H = [[A B],[C D]]` 구조. `[A B]`는 고rate 코드, `[C D]`는 단일 parity-check 코드 다수 concatenation.
5. 식 의미: `As^T+Bp1^T=0`, `C(s,p1)^T+Dp2^T=0` — `B`는 블록 dual-diagonal, `D`는 identity라 선형시간 인코딩 가능.
6. 핵심기법 2: rate 1/6~1/2 지원, `L=30` circulant lifting, density evolution으로 degree 분포 최적화.
7. 구현 디테일: `[C D]`가 단일 parity-check 다수라 mother code 마지막 부분을 puncturing하여 rate 1/3,2/5,1/2,2/3 생성(raptor 성질).
8. 학습/최적화: density evolution(Richardson-Urbanke)으로 VN/CN degree 분포 도출.
9. 결과: 길이 64800 quarter-rate가 DVB 대비 0.75 dB 우수(복잡도 43%↑), Shannon 한계 0.55~0.65 dB 이내. Puncturing으로 rate 1/2시 복잡도·지연 56%/50% 감소.
10. 한계: HW 미설계, 시뮬만, 저rate 방송 특화(R<1/2)이며 고rate에서는 이점 소멸.
