### B. 알고리즘 요약

1. IRA-like 부호는 QC 구조 부분행렬 `H1`(Information Node-Check Node 연결, p×p ICPM `I(Sα,β)` 블록으로 구성)과 zigzag 부분행렬 `H2`(Parity Node-Check Node, 삼각행렬)로 이뤄진 parity-check matrix `H`를 가짐.
2. 문제의식: QC 구조는 저장복잡도(1/p)를, zigzag 패턴은 선형시간 인코딩을 각각 제공하나, 둘을 결합할 때 short cycle과 low-weight codeword가 쉽게 발생하여 성능이 저하됨.
3. 채널모델은 BI-AWGN + BPSK, 표준 belief propagation(BP) 복호(고정 최대 iteration에서 종료).
4. 핵심 기법 1단(Theorem 1, Lemma 1): base matrix `B`의 길이-`2k` block cycle이 실제 QC 매트릭스에서는 `So = [Σ(-1)^m S_αm,βm] mod p`에 따라 더 긴 cycle(`2pk`)로 확장됨을 증명. `So=0`이고 `p`가 소수이면 확장 효과 최대.
5. 핵심 식 `(So, p)=1`이면 block cycle이 오직 하나의 길이-`2pk` cycle만 가짐 — 즉 shift value를 잘 고르면 짧은 cycle이 자동으로 늘어난다는 것이 이 식의 의미(Corollary 1).
6. 핵심 기법 2단: 서로 다른 block cycle 간 결합(공유 edge/chain)으로 생기는 cycle 길이도 정식화(Theorem 2, 3)하여 girth 상한을 유도(Corollary 2), cycle 최적화 제약 3가지(p는 소수, So=0 지향, block cycle은 길게·공통edge는 적게)를 도출.
7. 부수 트릭: H2-chain(PN-CN 교대 체인) 연결 시 degree-2/3/4 Information Node가 만드는 legal codeword의 weight를 분석해 "H2-chain 연결 제약"(CN 사이 VN 수를 충분히 크게)을 제시. 직접 연결 시 weight-5 저가중치 코드워드가 쉽게 발생하므로 "improved zigzag pattern"(CN을 넓게 분산 연결)으로 이를 회피.
8. 구성 절차: PEG(Progressive Edge Growth) 방법으로 저차원 base matrix `B` 구성 → cycle 제약을 만족하는 shift value 선택 → improved zigzag 접속 → local girth 테스트 후 불충족 시 재시도(반복적 휴리스틱 탐색, DE/GA 아님).
9. 결과: (9216,4608) IRA-like 코드가 동일 length/rate CMMB-LDPC 대비 저SNR 1~2자릿수, 고SNR 3자릿수 이상 BER 개선. (8176,7154, rate=0.875) 코드도 CCSDS-LDPC(EG-LDPC) 대비 소폭 우수.
10. 한계: 시뮬레이션(BI-AWGN, BP)만 수행, HW 구현/gatecount/throughput 데이터 전무. Error-floor 영역 곡선은 그림에만 있어 본문 수치로 확정 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QC 구조 + block cycle 확장 조건 (H-matrix 설계) | `ecc_top.cpp` `Load_PCM()` | Prime ECC도 QC-LDPC(base+circulant shift) 구조이므로 shift value 선택 조건은 H-matrix 재설계 시 참고 가능하나 특정 QC-LDPC 고정이라 이식 난이도 높음 |
| zigzag 패턴 기반 선형시간 인코딩 | `encoder.cpp` `Generate_PCM_encoding()` | Prime ECC는 dual-diagonal 인코딩을 이미 사용(zigzag와 유사 계열이나 별개 구조), 대체 도입 시 인코더 전면 재설계 필요 |
| short cycle/low-weight codeword 회피 휴리스틱 | 대응 없음 | 복호 알고리즘이 아닌 부호 구성 단계 이론이라 decoder.cpp의 실행부와 직접 대응되지 않음 |

적용 가치: **중간** — QC-LDPC H-matrix 구성 시 cycle 최적화 이론은 원리적으로 참고할 수 있으나, Prime ECC는 특정 QC-LDPC(z_sb=32, N_b=129)로 고정되어 있어 실제 이식은 H-matrix 전면 재설계·재검증이 필요.
