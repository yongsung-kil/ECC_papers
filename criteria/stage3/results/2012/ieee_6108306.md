### B. 알고리즘 요약

1. 시스템: two-way relay channel(TWRC) — 두 end node A/B가 relay R에 동시 전송(uplink), R이 network-coded 패킷 생성해 broadcast. BPSK/QPSK 변조.
2. 문제: 두 송신자 신호가 symbol offset `Δ∈[0,T)`, phase offset `φ∈[0,2π)`로 어긋나 도착 → 기존 방식은 최대 3dB(BPSK)/6dB(QPSK) BER 페널티.
3. 모델: R이 수신신호 `y_R(t)`를 `2N+1` 샘플로 oversampling(식4), 각 샘플은 인접 심볼쌍 `(x_A,x_B)`의 중첩 + AWGN.
4. 핵심기법(unchannel-coded, BP-UPNC): 결합심볼 `x_{i,j}=(x_A[i],x_B[j])`를 변수노드로, oversample 관측을 evidence로 하는 Tanner graph 구성; 인접 결합심볼 일관성을 check node `ψ_o,ψ_e`로 강제(식8).
5. 그래프가 tree 구조라 BP 1회 message pass로 exact posterior `P(x_{n,n}|Y_R)` 산출 → XOR MAP 매핑(식9).
6. QPSK message update: 결합심볼당 16조합 확률 `p^{a,b}_k`(식10,11)을 sum-product로 left/right-bound 전파.
7. 채널부호 확장: joint channel-decoding & network-coding(Jt-CNC)로 채널복호와 network coding을 하나의 factor graph에서 공동 수행(disjoint XOR-CD 대비 우수). 예시 부호는 Repeat-Accumulate(RA).
8. 핵심 통찰: half-symbol misalignment가 "diversity + certainty propagation"으로 phase 페널티를 desensitize.
9. 결과(시뮬): BPSK 3dB 페널티→<0.5dB; QPSK 6~7dB 페널티→half-symbol offset 시 <1dB; 채널부호 사용 시 비동기가 오히려 성능 향상(spread ~1dB). 패킷 2048bit×10000개.
10. 한계: 무선 릴레이 채널·변조 특화, HW 미설계, LDPC 결과는 본문 아닌 부록/기술보고서에만.
