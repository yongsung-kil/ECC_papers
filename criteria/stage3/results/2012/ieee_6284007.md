### B. 알고리즘 요약

1. MBIOS 채널에서 **Tanner 부호**(local-code node가 임의 선형부호; LDPC는 SPC local code 특수경우)의 LP-decoding 성공확률 분석.
2. 풀려는 문제: 주어진 부호어 `x`가 채널출력 `y`에 대해 ML·LP 최적이며 유일한지 판정하는 one-sided **인증서**(certificate)를 효율적으로 구성.
3. 핵심: computation tree 상의 **가중 normalized subtree**(w-weighted d-tree)로 local-optimality를 조합론적으로 특성화. 기존 skinny tree(degree 2)와 달리 local-code node degree `d`(2≤d≤d*) 허용, 높이 `h`가 girth 초과 가능.
4. `(h,w,d)-local optimality`: 모든 deviation `β∈B_d(w)`에 대해 `<λ, x⊕β> > <λ, x>` 성립. 이것이 ML-최적성(Thm.5)과 LP-최적성(Thm.10) 모두를 함의함을 증명.
5. 의미: 더 두꺼운(dense) subtree가 skinny tree보다 인증서 존재확률이 높음 → threshold 하한 개선 여지.
6. 검증 알고리즘(Alg.1 VERIFY-LO): dynamic programming/message-passing으로 각 변수노드의 최소비용 d-tree를 찾아 nonpositive cost deviation이 있으면 false. 복잡도 `O(|E|·h)`.
7. 구조적 핵심(Lemma 11): 임의 부호어는 computation tree 내 가중 d-tree projection들의 유한 합(기댓값)으로 분해됨 → min-sum 확률분석을 `sum-min-sum` process로 확장.
8. 결과: (2,16)-regular Tanner 부호(rate≥0.375, [16,11,4] 확장 Hamming local code)에서 BSC LP-decoding이 최대 `0.044N` 비트플립까지 whp 정정(기존 iterative 0.0016N 대비 개선). 실패확률은 girth에 대해 이중지수 감쇠.
9. AWGN 등 MBIOS 채널 일반에 적용 가능(BI-AWGN threshold σ*=0.0735 언급).
10. 한계: 순수 이론(증명·bound)만, HW·시뮬 없음. regular LDPC 최선 bound는 넘지 못함(open question), NAND soft-read/QC 구조와 무관.
