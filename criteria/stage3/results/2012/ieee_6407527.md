### B. 알고리즘 요약

1. 시스템: 8-level(및 16-level) MLC NAND, even/odd BL multipage 프로그래밍, ECC는 rate-19/20 (34520, 32794) QC-LDPC `[7]`.
2. 문제: MLC 레벨 증가로 cell-to-cell 간섭(parasitic coupling)이 급증, 기존 post-compensation/pre-distortion은 패턴 수가 레벨에 지수적(M=16→2176)이라 복잡.
3. 모델: 임계전압 shift로 표현되는 coupling 식 `(1)` (`x`,`y`,`xy` = 수평/수직/대각 coupling ratio), erased state Gaussian + programmed state uniform+양측 Gaussian tail 분포 `(2),(3)`.
4. 핵심기법1(추정): 이전 페이지 전압을 특정값으로 고정해 threshold shift를 `VS`/`VL`/`VSS(=0)` 세 값으로 축약 → 간섭 패턴 수가 레벨 무관하게 항상 18(Table I).
5. 의미: 심볼패턴 `(x1,x2,y)`만 sensing하면 평균 간섭 `E{F}`를 테이블 lookup으로 즉시 추정 → latency·복잡도 절감.
6. 핵심기법2(보상): scheme1은 read voltage를 `Favg`만큼 shift(`V*read=Vread+Favg`), scheme2는 oversampled bit에서 간섭 bit를 이진 감산으로 제거.
7. 구현 디테일: sensing은 scheme1이 3bit hard(BCH 적합), LDPC용은 4bit+ fine-grained sensing; scheme2는 n배 oversampling으로 `/n` 해상도.
8. LLR: 간섭이 제거되었다고 가정 후 식 `(4),(5)`로 상태 pdf 적분비 log로 LLR 산출 (무간섭 가정으로 계산 단순·신뢰도↑).
9. 결과: coupling strength factor `s` 대비 BER 곡선에서 scheme1이 최고, LDPC 결합 시 scheme1+LDPC가 scheme2보다 우수, 레벨 증가 시 신뢰도 향상 (시뮬만).
10. 한계: HW 미설계, gate/throughput 없음; LDPC 디코더/부호는 외부 인용부호 사용, 본 논문 기여는 read·LLR 전처리에 국한.
