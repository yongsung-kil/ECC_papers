### B. 알고리즘 요약

1. 대상: 특성 2 유한체 `GF(2^r)` 분할로 구성한 binary/non-binary QC-LDPC, AWGN 채널, SPA/MSA(비이진은 FFT-QSPA) 복호.
2. 문제: 좋은 error-floor·큰 min distance를 가진 QC-LDPC를 대수적으로 쉽게 구성하고 PCM rank를 정확히 알고자 함.
3. 모델: `GF(2^r)`를 disjoint 부분집합 `G1={λ}`, `G2={δ}`로 분할, base matrix `B` 원소 `b_{i,j}=λ_i+δ_j`.
4. 핵심 기법: `B`가 2×2 SM-constraint(모든 2×2 부분행렬 non-singular)를 만족(Thm 1) → 각 원소 `α^k`를 `(2^r-1)` CPM으로 array dispersion하여 RC제약 array `H` 생성, null space가 QC-LDPC.
5. 식 의미: RC제약은 girth≥6, `(γ,ρ)`-regular면 min distance≥`m+1` 보장.
6. 확장: 같은 base matrix를 α-multiplied CPM으로 분산하면 `2^r`-ary non-binary QC-LDPC 생성(rank 동일). masking(`Z∘B`)으로 CPM 밀도 낮춰 irregular·고girth 코드 구성.
7. 구현 디테일: Fourier domain에서 `H`가 `diag(B^{∘0},...)`로 대각화 → rank(H)=Σrank(B^{∘l}), Vandermonde 구조로 `rank(B^{∘l})=min(m,μ_l)`, `μ_l=2^{ω(l)}`(Lucas).
8. 학습/최적화: 없음(결정론적 대수구성). rank 조합식 `rank(H)=3^r-2^r-Σ...` 유도(식 (10),(11)).
9. 결과: (3654,3335) rate 0.9127 binary는 50-iter MSA로 BER 1e-6에서 Shannon한계 1.2dB; (15494,14807) rate 0.9557은 BER 1e-8에서 0.92dB; masking으로 rate 0.5 irregular (8128,4064)도 구성.
10. 한계: HW 미설계, AWGN 시뮬만, error-floor 정량 없음, non-binary·특정 QC구조는 이식 부담.
