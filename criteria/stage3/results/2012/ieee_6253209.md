### B. 알고리즘 요약

1. 시스템: protograph LDPC(외부부호) + PR 채널(내부부호)의 직렬연접, rate `R=0.8`, 정보길이 `k=4096`(부호길이 약 20480 bit). dicode(`1-D`)·EPR4(`1+D-D^2-D^3`) 채널.
2. 문제: AWGN에서 우수한 AR3A/AR4JA protograph 부호가 ISI 있는 PR 채널에선 성능 저하.
3. 모델: 수신 `yi = Σ hj·xi-j + ni`, turbo equalization으로 BCJR 검출기와 BP 복호기가 외재 LLR 교환.
4. 분석도구: 유한길이 EXIT 알고리즘 — SISO 출력 LLR을 대칭 가우시안 근사, EXIT band(기대곡선±3σ)로 수렴 예측.
5. 핵심관찰: AWGN에선 최고차 VN 퍼처가 유리하나, PR 채널에선 최고차 VN을 **보존**해야 함(turbo iteration에서 복원 중요). Table I로 검증.
6. 설계 2단계: (1) 최저차(degree-1) VN 퍼처, (2) degree-2 VN 비율 축소 — degree-2 VN에 edge 추가하고 최고차 VN에서 edge 제거해 복잡도 불변.
7. 결과 부호: IARA1(=AR3A base, degree-1 퍼처), IARA2(degree-2 재연결), IARA3(=AR4JA base, degree-1 퍼처).
8. 임계값(AWGN): IARA1/2 = 2.627 dB, IARA3 = 2.568 dB (용량 2.040 dB); AWGN에선 AR3A/AR4JA가 오히려 우수.
9. 결과(PR): 성능순 IARA2 > IARA1 > regular-col3 > IARA3 > AR3A > AR4JA. 2 turbo iter 후 IARA1/2 평균 오류 < 1 bit(AR3A=11, AR4JA=101). PR에서 turbo iteration 수도 감소.
10. 한계: 순수 시뮬(HW 없음), turbo equalization(BCJR+BP) 구조 종속, PR/자기기록 채널 특화.
