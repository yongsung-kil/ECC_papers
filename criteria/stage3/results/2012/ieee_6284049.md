### B. 알고리즘 요약

1. 대상: MBIOS(BSC/AWGN) 채널, min-sum 및 변형(AMS/OMS) 복호. 부호는 rate-0.3 (640,192) QC-LDPC, rate-0.5 (2640,1320) Margulis.
2. 문제: error floor의 한 원인이 코드/알고리즘이 아니라 fixed-point 구현의 **메시지 양자화 규칙**(uniform saturation)임을 규명.
3. 모델: absolute trapping set (a,b) - 유도 부분그래프에 degree-1 CN 최소 1개 필요(외부 정답 메시지 유입 경로). k-separation 가정.
4. 핵심 관찰: variable-regular 코드에서 trapping set 밖 정답 메시지 magnitude가 iteration마다 `|Ll|=|L0|+(dv-1)|Ll-1| > (dv-1)^l|L0|`로 지수 증가(Thm 1).
5. 의미: uniform quantizer는 saturation이 작아 정답 메시지가 조기 포화 -> 오답 상쇄 불가 -> min-sum이 Gallager-B로 퇴화. saturation 확장이 핵심.
6. 핵심 기법: **(q+1)-bit quasi-uniform 양자화**. `[-NΔ,NΔ]` 내는 q-bit uniform과 동일 정밀도, 밖은 추가 1비트로 step을 `d^r·NΔ`(지수 증가)로 표현. `d∈(1,dv-1]`.
7. 구현 디테일: 예) Δ=1,d=3,q=3이면 {0,±1,±2,±3,±9,±27,±81,±243}. update rule·그래프 무변경, 복잡도 증가 없음.
8. 확장: irregular는 d 조정으로 대응, SPA 근사 등 대부분 MP 디코더에 적용 가능(별도 보고 예정).
9. 결과: (640,192)/Margulis에서 3+1/4+1/5+1-bit quasi-uniform이 동일 비트 uniform 대비 error floor 대폭 저감, unsaturated 디코더에 근접. 수십억 realization에서 small trapping set 실패 미관측.
10. 한계: 시뮬만(BSC/AWGN), HW 미설계, d는 휴리스틱 선택, variable-regular 분석 전제.
