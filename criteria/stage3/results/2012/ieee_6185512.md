### B. 알고리즘 요약

1. 대상은 rate `R=(L-J)/L`의 binary LDPC **convolutional** code로, AWGN/BPSK, 최대 50 iteration BP(sliding-window) 복호를 가정한다.
2. 문제: LDPC convolutional code의 대수적 구성법이 제한적이고, 기존 QC-LDPC 유도법은 다항식 항이 많거나 인코딩이 무겁다.
3. 핵심 아이디어: `GF(q)` 곱셈군으로 만든 QC-LDPC의 이진 순환행렬이 다항식환 `F2[D]/<D^r-1>`에 환동형이라는 성질을 이용한다.
4. 기법 1단(original): `J×L` base matrix `M`(원소 `α^{m+...}-1`)을 만들고 각 원소 지수 `E_{i,j}`를 다항식 `h_{i,j}(D)=D^{E_{i,j}}`로 치환해 다항식형 검사행렬 `H(D)`를 얻는다.
5. `H(D)=H_Q(D)` 등가 해석으로 QC block code의 다항식 행렬을 그대로 convolutional code 검사행렬로 재사용한다는 것이 핵심 식의 의미다.
6. 기법 2단(fast encoding): 마지막 `J`열 대각을 `D^0`로 치환해 systematic·fast-encoding 행렬 `H_I(D)`(FE-I), 또는 초등변환 기반 `H_II(D)`(FE-II)를 만든다.
7. 구현 디테일: FE-I만 `α`-multiplied row-constraint 만족 시 최대 인코딩 memory `m_s=q-2`를 얻어 시프트레지스터 인코더로 실시간 연속 인코딩 가능.
8. (학습 절차 없음 — 순수 대수적 구성).
9. 결과: `GF(211)`, `(3,5)`-regular, `N=1050`, `R=2/5`에서 FE-I이 original/FE-II 대비 block code 0.7dB, convolutional code 0.4dB 이득(BER<1e-6). Table I로 short-cycle 수 감소 확인.
10. 한계: FE-I의 cycle 개선은 통계적 관찰뿐 증명 없음, HW 미설계, convolutional code 전용이라 NAND block ECC와 직접 대응 어려움.
