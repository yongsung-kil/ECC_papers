### B. 알고리즘 요약

1. 대상은 NAND MLC(X2/X3, 8-level) 플래시로, cell-to-cell 간섭에 의한 `Vth` shift가 asymmetric·limited-magnitude(크기 1) 오류를 유발.
2. 문제: BCH 등 기존 ECC는 임의·대칭·무제한 크기 오류용이라 MLC 특성을 못 살려 redundancy가 큼.
3. 모델: 오류 크기가 1로 제한되면 modulo-`N` 잔여값이 오류정보를 보존(`ε = (ξ̄ − ω̄) mod N`).
4. 핵심기법: q-ary 데이터를 modulo `N`으로 축약 후 `m×n` 블록의 행/열 패리티(`p_rk`, `p_ck`)를 생성하는 2D 패리티 검사(LMEPC).
5. 복호: 수신값의 행/열 패리티를 재계산해 불일치하는 행·열의 교점 `d'_{i,j}`를 오류위치로 특정.
6. asymmetric용은 mod 2(upward만 정정), symmetric용은 mod 3(upward/downward 판별 후 정정).
7. 구현 디테일: ternary 패리티는 `log2 3` bit로 표현, 하드결정(hard-decision) 기반.
8. 학습/최적화 절차 없음(구성법·규칙 기반).
9. 결과: 저rate에서 no-ECC 대비 BER 약 10^3배 감소, code rate 0.95 이하에서 asymmetric 이득은 포화(read voltage 재조정으로 오류가 대칭화).
10. 한계: 코드블록당 단일오류만 정정, HW 미설계·시뮬만, LDPC 반복복호와 무관.
