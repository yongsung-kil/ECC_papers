### B. 알고리즘 요약

1. 대상: BIOSM 채널(BEC/BSC/BIAWGN) 위 LDPC 앙상블을 degree distribution `(λ(x),ρ(x))`로 표현, rate `R=1-∫ρ/∫λ`.
2. 문제: EXIT 차트는 BEC 외 채널에서 area theorem이 성립 안 해 용량달성 앙상블 설계 불가, GEXIT는 잠재력 있으나 규칙부호·물리채널 부재 가정에 국한.
3. 모델: extrinsic LLR의 L-density `c(y)` (대칭성 `c(-y)=c(y)e^{-y}`), 엔트로피 `H(c)=∫c log2(1+e^{-y})`.
4. GEXIT 함수 `g(ε)=∂H(X_i|Y,Ω)/∂ε_i · dε_i/dε`, area theorem 만족하도록 정의된 kernel `l_c(z)`.
5. Prop.1: 임의 물리채널 파라미터 `q`에서 variable node GEXIT 함수 유도(식 6/7), `b_q(y)→Δ_0`이면 기존 [11] 식으로 환원.
6. Prop.2: BSC(δ)에 대해 variable node GEXIT의 폐형식 유도(식 9, 이항합 형태).
7. Prop.3: BIAWGN(q)에서 GEXIT 곡선 시작점의 폐형식(극한 `ε→0`) 제공.
8. 대안 표현: check node의 dual GEXIT vs variable node inverse GEXIT를 함께 플롯, area theorem 유지(Prop.4) 증명.
9. 결과: rate 1/2, `D_v=20` BIAWGN 앙상블 설계 threshold 0.3004 dB (Shannon limit 0.12 dB 이내, [2] 대비 0.01 dB 개선). 설계=선형계획 최적화로 귀결.
10. 한계: 이론만(HW·유한블록·QC 구조 없음), 무선 통신 채널 대상, hard/soft NAND read와 무관.

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| GEXIT 차트 기반 degree distribution 설계 | 대응 없음 | 점근 앙상블 설계 이론, 우리는 고정 H-matrix(ecc_top.cpp Load_PCM()) 사용 |
| DE/matching 조건 threshold 분석 | 대응 없음 | 부호설계 검증 이론일 뿐 디코더 코드와 무관 |
| BSC/BIAWGN L-density LLR | 대응 없음 | soft-read LLR 테이블(PARAM_LLR)과 개념만 겹칠 뿐 이론적 |
```

적용 가치: **낮음** — GEXIT 기반 점근적 LDPC 앙상블 설계 이론으로, Prime ECC의 고정 QC-LDPC/디코더/HW 파이프라인에 이식할 코드 요소가 없다.
