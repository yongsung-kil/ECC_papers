### B. 알고리즘 요약

1. 대상: variable-regular binary LDPC(QC-LDPC (640,192) rate 0.3, Margulis (2640,1320) rate 0.5), BSC/AWGNC, box-plus SPA 복호.
2. 문제: 고정소수점 SPA에서 φ함수/tanh 양자화 시 메시지 포화·클리핑이 error-floor를 유발 — trapping set 자체보다 양자화가 주원인.
3. 모델: `(a,b)` absolute trapping set과 `k-separation assumption`을 그래프 이론으로 정의(채널·디코더 무관).
4. 핵심 기법1: `tanh`/`φ` 대신 box-plus 연산 `⊞`로 CN 갱신 — `U⊞V = sign·{min(|U|,|V|)+s(U,V)}`, 보정항 `s(x,y)`.
5. 핵심 식 의미: `-log2 < s(x,y) ≤ 0` (Lemma 1) → box-plus SPA는 작은 보정항을 가진 min-sum으로 취급 가능해 [9]의 MS 결과를 SPA로 확장(Theorem 1).
6. 결론: 메시지 크기에 상한이 없으면 `k-separation` 만족 시 SPA는 trapping set 오류패턴을 정정 → 포화가 실패의 열쇠.
7. 핵심 기법2: `(q+1)-bit quasi-uniform quantizer` — MSB 1비트로 균일/지수(비균일) 구간 전환, 파라미터 `d∈(1, dv-1]`로 큰 크기값까지 표현.
8. 구현 디테일: Margulis 코드엔 `ln(1+e^-|x|)`의 2구간 선형근사(approximate-SPA) 적용, 원 box-plus 대비 약 5배 속도, waterfall 손실 <0.02 dB.
9. 결과: (6+1)-bit quasi-uniform SPA가 부동소수점에 근접한 FER, error-floor를 크게 낮춤(균일 양자화는 높은 error-floor). (5,5)/(5,7), (12,4)/(14,4) trapping set 강제오류도 정정.
10. 한계: HW 미설계·시뮬만, 복잡도 증가는 "essentially none"이라 정성 서술뿐(gatecount/throughput 없음).
