### B. 알고리즘 요약

1. 대상: BSC(hard-decision) 위 이진 LDPC의 sum-product 복호. 예제 부호는 array(3,11) len121, MacKay(504,252), DSC(273,191) 등 짧은 부호.
2. 문제: 주어진 LDPC의 정확한 WER 공식이 없음 → 저error-rate 영역 시뮬 비용 큼. 해석적 WER 공식을 원함.
3. 핵심 관찰: BSC에서 sum-product는 반복 `l_max` 고정 시 오직 "초기화 오확률"에만 의존하고, 관례상 이를 crossover `p`로 두지만 다른 값 선택이 가능.
4. 핵심기법 1 - FID(fixed initialization decoding): 초기화 `ε_0`를 실제 `p`와 분리한 파라미터로 도입 → `WER(ε_0, p)` 2변수 함수화.
5. 핵심 개념 - correctable error set `Corr(ε_0, l_max)`: 해당 초기화로 정정되는 오류패턴 집합, `p`와 무관.
6. 핵심 정리(Thm IV.1): 고정 `ε_0`에서 WER는 `p`의 다항식(correctable set의 weight enumerator로 표현). 원 SPA의 WER(`ε_0=p`)는 `p`에 불연속일 수 있음.
7. 복잡도 축소 1(Sec V): syndrome 복호와 correctable set 동치(Thm V.1) → co-dimension 작으면 exhaustive search 회피, FSA(3,11)에 적용해 Table I 산출.
8. 복잡도 축소 2(Sec VI): Tanner graph automorphism(orbit)로 필요한 syndrome 수를 Burnside 보조정리로 대폭 축소(예: 인자 `~n!` 축소).
9. 결과: MacKay(504) FID `ε_0=0.05`가 SPA를 crossover 0.02~0.05 전반에서 상회; DSC(273) FID가 특정 `ε_0`에서 수렴 반복수를 크게 단축(Fig.3). 양자 CSS-LDPC WER 평가 복잡도 축소에도 적용.
10. 한계: BSC(hard 1-bit)·짧은 부호 한정, correctable set 열거가 부호길이에 지수적. 고rate/장부호·soft-read 확장 미검증, HW 설계 없음.
