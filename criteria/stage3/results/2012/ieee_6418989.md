### B. 알고리즘 요약

1. 시스템: BI-AWGN 채널, regular QC-LDPC, rate 0.4~0.67, 길이 N≈1246~8300, column-weight 3.
2. 문제: random LDPC는 성능 좋으나 HW 구현 어렵고, 기존 구조적(BIBD) 구성은 rate/length 유연성 부족((4,4k)/(3,3k) 제한).
3. 가정: parity-check H는 c×c circulant permutation matrix 배열, GF(2) null space로 정의.
4. 핵심 기법 1단: BIBD `(c, p1, λ)`로 hybrid 구조행렬 `F` 생성 — 각 행이 p1개 weight, (c-p1)개 0 → 0-block으로 밀도 저감.
5. 핵심 식: `F`의 0행렬 배치가 6-cycle 수를 줄여 성능 향상(Massey bound로 최소거리 ≥ s+1).
6. 핵심 기법 2단: preliminary matrix `P` 원소를 CodeType(MPS `d_{u,v}=e_u⊙_{p2}f_{v+1}`, 또는 MWC codes)로 채워 circulant shift 값 결정.
7. 구현 디테일: I(p_{m,n})는 p2×p2 identity를 우측 순환 shift, 0은 zero block → QC 구조 유지, girth ≥ 6 보증.
8. 최적화: seed 시퀀스 e,f를 GF(p2)에서 선택, MPS는 GF(139)/GF(89) 예시.
9. 결과: MWC/(7,3,1)은 (3,6)-regular 대비 error floor 제거(BER 10^-8까지 floor 없음), MPS/(7,3,1)은 Shannon 1.1dB 이내(BER 10^-6). λ=2인 MWC/(11,5,2)는 개선 미미.
10. 한계: 시뮬(SPA log-domain, 최대 50 iter)만, HW 미설계, NAND/soft-read 무관, rate 낮음(0.4~0.67).
