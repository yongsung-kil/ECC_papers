### B. 알고리즘 요약

1. 시스템: BPMR 2 Tbits/in², 고rate LDPC(PEG 구성, col-weight 3, row-weight 26~29, block 4608, rate 8/9), BP 디코더, 20 iteration.
2. 문제: BPMR 쓰기 과정의 written-in error가 data-dependent인데 기존 LDPC는 이를 반영 못해 성능 저하.
3. 모델: cascaded WEC(write-error channel)-AWGN 채널, `r = x + n`, `n`은 zero-mean 분산 `σ²` AWGN.
4. 핵심 기법: hard decision으로 수신열을 비트화 후, 각 3비트 패턴(예: 000~111)의 middle bit written-in error 확률 `Pw`를 Table I에서 조회.
5. 핵심 식: pdf `f(r)`를 `Pw`(하드결정 의존)로 가중해 수정, LLR `= log( f(r|x=1)/f(r|x=0) )`를 Bayes rule로 재계산 (식 (3)~(5)).
6. 확장: shift 방향별 Type 1(좌)/Type 2(우)/Type 3(shift 범위 알 때 좌우 평균) 3종 패턴 의존 LLR.
7. 구현 디테일: 관례 LLR은 `Pw=0`으로 설정한 특수 케이스로 환원.
8. 최적화: 별도 학습 없음, Table I의 물리 파라미터(head field, SFD, demag field)로 `Pw` 산출.
9. 결과: BER 지점에서 SNR gain 약 0.2 dB(좌우 평균), Type별 0.1/0.6 dB, write margin(5.5 dB 고정) 최대 확대.
10. 한계: 시뮬만, HW 미설계, BPMR 물리 write-error 모델 종속으로 일반화 제한.
