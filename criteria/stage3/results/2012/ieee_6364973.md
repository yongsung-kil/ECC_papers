> 총평: 고rate(0.96) NAND용 binary LDPC + NAND 채널 최적 soft-read 양자화/LLR 설계라는 Prime ECC 핵심 관심사와 정면 일치, 부호계열(EG-LDPC)만 다를 뿐 채널·LLR 방법론은 직접 이식가치 높음.

### B. 알고리즘 요약

1. 시스템, 코드 스펙: 8KB 페이지 NAND용 rate-0.96 (68254, 65536) shortened EG-LDPC(원본 (69615,66897)), PCM은 17개 4095×4095 cyclic-shift 서브행렬, row/col weight 16.
2. 문제: MLC NAND에서 BCH 한계 → soft-decision LDPC 필요, 그러나 BI-AWGN 시뮬로는 비정상(non-stationary) NAND 채널 성능을 정확히 평가 못함.
3. 채널 모델: 2bit/cell MLC, ideal ISPP(uniform) → RTN(대칭 지수분포 `P_RTN`) → CCI(all-bit-line, 다음 word-line 3셀, 결합비 `γ_y=0.08s`, `γ_xy=0.0048s`) 순 왜곡.
4. 핵심기법 1: BI-AWGN 최적 양자화 — `n`-bit 양자화 경계 `q_i`, 대표레벨 `r_i`(식1, 부분 CDF `φ` 비의 로그), discretized DE로 noise threshold `σ*` 최대화하는 경계 결정[7].
5. 핵심기법 2: NAND 채널 비균일 sensing — 오버랩 경계 `B_l(k)`,`B_r(k)`를 확률비 `R`로 정의(식3), 각 심볼분포를 Gaussian 근사해 2차식(식4,5) 풀어 경계 산출, 오버랩 영역만 조밀 sensing.
6. 핵심 식: 양자화 LLR — 영역 `R_i`의 LSB/MSB LLR을 심볼 `p(0..3)`의 부분 CDF 비(식6,7)로 계산, error/Q-function LUT로 간단 산출([8]의 수치적분 대비 실용적).
7. 부호 성질: EG-LDPC는 min-weight보다 작은 harmful trapping set이 없어 error-floor 우수(예: (1024,781) EG-LDPC는 BER 10^-23 이하), cyclic/QC 구조로 LFSR 인코딩 가능.
8. 디코딩: BP(belief-propagation), floating-point, 최대 iteration 16.
9. 결과: BI-AWGN 3-bit는 infinite-bit 대비 BER 10^-5에서 0.04dB 차, BCH 대비 3-bit 0.92dB gain. NAND 채널은 BI-AWGN 대비 LSB 3/2/1-bit에서 0.14/0.09/0.18dB, MSB에서 0.25/0.24/0.13dB 차 → 채널 모델별 성능 상이.
10. 한계: HW 미설계·floating-point 시뮬만, 심볼분포 Gaussian 근사, 비정상 채널 적응형 sensing은 향후 과제.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| EG-LDPC PCM / 부호구조 로드 | `ecc_top.cpp` `Load_PCM()` | 고rate binary LDPC PCM 로드 경로 동일, 단 EG-LDPC≠Prime의 QC base+circulant |
| NAND 채널 / soft-read threshold | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | RTN+CCI 채널모델·오버랩 경계 산출이 Prime 채널·threshold 설정과 직접 정합 |
| 비균일 양자화 LLR 계산 | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | 영역별 LLR(식6,7) 산출법은 Prime의 LLR 테이블 설계에 이식 가능 |
| BP 디코딩 (min-weight/error-floor) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | Prime은 min-sum, 논문은 full BP·EG 부호의 trapping-set-free error-floor 특성 참고가치 |

적용 가치: **높음** — NAND용 고rate binary LDPC의 채널 모델링·최적 soft-read 양자화·양자화 LLR 계산이 Prime ECC의 `channel.cpp`/LLR 테이블 설계와 직접 맞닿아 있어, 부호계열(EG-LDPC) 차이만 고려하면 방법론 이식 가치가 크다.
