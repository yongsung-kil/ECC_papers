### B. 알고리즘 요약

1. 시스템: AWGN 채널, Tanner code `N=155`, `M=93` (155개 `TS(5,3)` trapping set으로 전체 커버됨).
2. 문제: BP는 Tanner graph의 짧은 루프(특히 `TS(5,3)`)로 인해 error-floor에 빠져 준최적.
3. 접근: 통계물리 Kikuchi 근사에서 온 Generalized BP(GBP)로 루프를 region(클러스터) 노드에 흡수.
4. region graph 구성: 각 region `r`은 check set `Cr`+variable set `Xr`로 구성, overcounting number `cr` 조건 만족.
5. novel 구성 1 (triplets): `TS(5,3)`을 3개 초기 region(각 2 check + 1 unverified check)으로 분할, region당 13 variable → 메시지 `2^13` 연산.
6. novel 구성 2 (triangles): 각 region 4 check + 2 unverified check, 24 variable → `2^24` 연산, 짧은 루프를 더 잘 흡수.
7. 메시지 갱신식(8): region 간 방향성 그래프에서 부모→자식으로 belief 전파, 분모 메시지를 `m^(k-1)`로 근사해 병렬화 촉진.
8. 감쇠계수: 기존 상수 `w=0.5` 대신 반복 종속 `wk`(constant/affine/parabolic/inverse-parabolic), `w0=0.85`, 마지막 `wK=0` 강제.
9. 결과: SNR 2.90dB에서 parabolic `wk`가 최속 수렴(k≈20), triangles 구성이 7 error 이벤트를 error 증가 없이 정정(BP는 진동).
10. 한계: 특정 Tanner code에만 검증, HW 미설계, region 지수 복잡도, binary이나 QC 구조 아님.
