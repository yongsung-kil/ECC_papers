### B. 알고리즘 요약

1. 대상: sparse graph LDPC, AWGN + binary message passing(bit-flipping) soft decision 디코더, rate 0.5, `dc=11`, 최대 VN degree 30, PEG로 `N=10^3, 10^4` 구성.
2. 문제: absorbing set이 binary message passing 디코더의 안정 고정점이 되어 error-floor를 유발 - 유한길이 성능 저하.
3. 모델: Tanner graph에서 `(a,b)` absorbing set 정의 - D의 각 VN이 O(D)(홀수차 이웃)보다 F\O(D)에 더 많은 이웃을 가짐.
4. 핵심 정리(Lemma 2,3): VN degree `γ∈{2,3}`이면 크기 `g`의 cycle ⟺ 크기 `a=g/2`의 absorbing set - 즉 저차수 노드의 absorbing set은 cycle과 동치.
5. 의미: degree 2/3 VN 서브그래프에서 cycle을 피하면 최소 absorbing set 크기를 키워 error-floor를 낮출 수 있음(임계 성능 유지하며).
6. 확장(Theorem 1): degree d까지 VN 서브그래프가 cycle-free일 조건 `Σ(i-1)/i·λ_i ≤ (1-R) - 1/N` 유도.
7. 부수 제약: stability condition `(λ2+2·c_h·λ3)(dc-1)<1` 도 동시 만족하도록 degree분포 최적화.
8. 최적화: degree분포를 세 case(d=2 / d≤3 / d≤5 cycle-free)로 최적화 후 PEG로 부호 생성.
9. 결과(N=10^4): case1(d=2)은 크기7 absorbing set 다수로 높은 floor, case2(d≤3)는 단일 크기8 set으로 floor ~`10^-8`, case3(d≤5)는 absorbing set 미발견·3.4dB 이상 오류 없음.
10. 한계: binary message passing 디코더 전제, HW 미설계, rate 0.5 단일 예제, min-sum/SPA로의 일반화는 논문 범위 밖.
