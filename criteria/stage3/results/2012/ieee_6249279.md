> 총평: QC-LDPC base matrix의 shift 값을 적응형 cost hill-climbing으로 빠르게 large-girth 최적화하는 오프라인 구성 툴로, 우리 H-matrix 설계 파이프라인에 참고 가치가 있으나 복호·HW 기여는 없다.

### B. 알고리즘 요약

1. 대상: J x K base matrix + Z x Z 순환치환행렬로 구성된 QC-LDPC 부호(`P(h_j,k)`=우측 순환이동 항등행렬), 목표 girth g 달성.
2. 문제: 고rate·구조화 QC-LDPC는 구조상 girth>6 보장 못하고, 전수탐색은 Z^(JxK) 조합으로 불가, PEG도 QC서 최대 girth 미보장.
3. 핵심 기법 1: hill-climbing으로 base matrix 원소를 순서대로 지역최적값으로 갱신, cost 최소화까지 반복.
4. 핵심 식: cycle 존재 조건 sum_{v}(h_{jv,kv} - h_{jv,kv+1}) = 0 (mod Z) (식 3)로 길이 2l cycle 판정.
5. 적응형 cost: 현재 부호 girth가 g'(<=g)일 때 cost를 "길이 g' cycle 수"로 정의, 더 짧은 cycle이 없도록 보장하며 원소 갱신 -> girth 증가에 따라 cost가 적응(Fig.2).
6. 확장/가지치기: 현재 girth 2l이면 길이 2(l+1) 이상 cycle은 무시, 길이<2l cycle을 만드는 z값은 후보서 제외해 계산량 대폭 축소.
7. 구현 디테일: 원소를 column-by-column(또는 row-by-row) 사전지정 순서로 갱신, 순서에 따라 다른 해 도출; 수렴은 전체 원소가 지역최적일 때.
8. 이론 보증: girth-2l 부호서 length-2i(i<l) cycle을 통과 안 시키는 z로 설정하면 결과 Hb도 girth 2l 유지(증명).
9. 결과: girth-8 3x18 부호서 Wang 대비 등가 iteration 10~20%, 4x24서 10% 이하 -> 5~10배 이상 가속, Z 작을수록 감소폭 큼.
10. 한계: 오프라인 부호구성 속도 개선만, BER/error-floor 실측·HW·복호 알고리즘 없음, rate/length 확정값 미제시.
