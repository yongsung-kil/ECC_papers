### B. 알고리즘 요약

1. 대상: Gaussian relay 채널용 bilayer lengthened LDPC(하위=destination, 상위=relay 그래프), rate 쌍 `(0.4877, 0.6906)`, block length `10^5`, BPSK.
2. 문제: rate-optimized bilayer 부호는 relay 디코딩 iteration이 많음 → 전력·복잡도 제약이 큰 relay에서 iteration(복잡도) 최소화 필요.
3. 모델: DF relaying, `Y1=X+Z1`, `Y=X+X1+Z2`; message-passing + EXIT chart `f(e)`로 반복별 오류확률 궤적 표현.
4. 핵심기법: 하위 그래프 degree 분포 `(λ(1),ρ(1))` 고정, 상위 그래프 `(λ(2),ρ(2))`만 최적화해 정보비트당 복잡도 `K∝N·E` 최소화.
5. 핵심식: `N ≈ ∫ de / (e·ln(f(e)))` (식 7)로 수렴 iteration 수를 EXIT chart로 추정, rate `RBL` 유지 제약 하 최소화(식 8).
6. 확장: 상위 check node degree를 6/7/8로 바꿔가며(정규 CN 가정) 각 rate 유지 복잡도 최소 부호 `C1/C2/C3` 설계.
7. 구현 디테일: discretized density evolution(13 quantization bit), 상위 VN 최대 degree 19, 목표오류 `10^-6`·최대 600 iteration.
8. 학습/최적화: DE 기반 최적화 문제 풀이 + CN degree exhaustive search.
9. 결과: `C1~C3`가 code(F) 대비 iteration 최대 31~62% 감소; 동일 iteration 시 BER=1e-4에서 0.05/0.16/0.18 dB coding gain, 고 SNR에서 waterfall 더 가파름(degree-2 VN 비율 감소 덕).
10. 한계: relay 2-layer 구조·시뮬(BER)만, HW/gate/throughput 없음, QC 구조 아님.
