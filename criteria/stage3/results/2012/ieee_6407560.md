### B. 알고리즘 요약

1. 시스템: rate-R LDPC 부호화된 2-D ISI(intersymbol interference) 기록 채널, code bit를 `l×l` 행렬로 스캔해 2-D 채널 `h={h_i,j}` 통과, AWGN 잡음 `n(i,j)` 식 `(1)`.
2. 문제: 2-D 채널 Tanner graph의 channel check node SPA 연산이 `2^dc` 벡터 전탐색이라 복잡도 과다.
3. 모델: log-domain SPA가 각 check node에 연결된 `dc` bit의 APP LLR을 식 `(2)~(4)`로 계산, 복잡도는 탐색집합 `E`의 크기 `|E|`에 비례.
4. 핵심기법: Chase 디코딩 원리로 부분집합 `E` 구성 — a priori LLR 절댓값 `|A_k|` 기준 최소 신뢰 `Q`개 비트만 모든 조합(`2^Q` 벡터, `E1`) 생성.
5. 의미: 신뢰 낮은 비트만 전탐색하면 전체 `2^dc` 대신 소수 벡터로 근사 → 복잡도 `O(2^Q + dc-Q)`.
6. 확장: `E1`에서 최적 벡터 `b`(식6) 선택 후, 신뢰 높은 `dc-Q` 위치 각각 1비트 flip한 `E2`(dc-Q 벡터)를 합쳐 `E=E1∪E2` 구성.
7. 구현: 3-layer Tanner graph(상위 2층=LDPC 부호, 하위 2층=2-D ISI 채널), 반복당 message-passing 스케줄 고정.
8. 학습/최적화: 없음(고정 규칙 기반).
9. 결과: rate-0.5 length-10000 regular(3,6) LDPC, 2×2 채널, 최대 50 iter. Q=2는 6벡터로 full 대비 0.2dB 손실·복잡도 62.5%↓, Q=3(9벡터)은 full(16벡터)과 거의 동일, CN 복잡도 43.75%↓.
10. 한계: 2-D 자기기록 채널 detector 전용, HW 미설계·시뮬만, LDPC 부호/디코더 자체는 표준.
