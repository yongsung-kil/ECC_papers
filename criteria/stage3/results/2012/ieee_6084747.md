### B. 알고리즘 요약

1. 대상은 QC `(J,K)-regular` LDPC 블록부호로, tailbiting한 convolutional code의 monomial parity-check 행렬로 표현한다.
2. 문제: 반복복호 효율을 좌우하는 girth(그리고 error-floor를 좌우하는 최소거리)가 큰 짧은 QC-LDPC를 찾는 것.
3. 핵심 대응: parity-check 행렬 = bipartite graph의 biadjacency 행렬, base matrix에 voltage(=monomial degree)를 부여해 voltage graph를 만든다.
4. girth 판정: lifted graph의 cycle ↔ voltage graph에서 voltage 합이 0인 cycle. 따라서 girth = "voltage 0인 최단 cycle" 길이.
5. 탐색 절차: base graph에서 길이 `<2g` cycle을 기술하는 부등식 목록 생성 → 이를 만족하는 voltage assignment 탐색(Algorithm A: cycle voltage≠0 확인 / Algorithm B: 동일 path-voltage 노드쌍 없음 확인, tree 정렬).
6. 복잡도: 순진한 `~n^{g/2}` path 탐색을 trellis 유사 tree로 `~sqrt(n^{g/2})` 수준까지 축소. 큰 girth엔 Algorithm B가 점근적으로 우세.
7. 상한/하한: Theorem 1~3에서 girth와 최소거리의 상·하한(Griesmer, generalized Hamming distance, Fossorier 상한 일반화) 제시.
8. 구성 3종: all-one base matrix(girth≤12), Steiner Triple System 기반(girth 14/16/18), 기존 QC 부호를 다시 base로 재적용(girth 20/22/24).
9. 최소거리 계산: sparse PCM에서 partial syndrome state tree를 키워 최소 열합=0을 찾는 전용 알고리즘 제시, [24]의 미상 최소거리 등도 확정.
10. 한계: 이론/컴퓨터탐색 결과만, 복호 알고리즘·양자화·HW·NAND soft-read 언급 없음. 큰 girth 부호는 블록길이가 커 비실용적.
