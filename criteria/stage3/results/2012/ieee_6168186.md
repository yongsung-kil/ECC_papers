### B. 알고리즘 요약

1. 시스템: binary-input AWGN 채널용 finite-length irregular LDPC, 결과 부호는 QC(quasi-cyclic), rate 1/2~7/8, length n=990~5792.
2. 문제: waterfall 최적 irregular degree 분포는 랜덤 구성 시 error floor가 높음 - 두 영역 동시 최적화가 어려움.
3. 가정/모델: error floor는 trapping set에 좌우되나 irregular 부호의 dominant trapping set 열거는 사실상 불가 → 간접지표 ACE spectrum 사용.
4. 핵심기법 1단: 작은 protograph(base graph)에서 시작해 cyclic N-lifting(circulant shift `πd`)으로 목표 block length까지 lifting → 결과가 QC 구조.
5. 핵심식: TBC(tailless backtrackless closed) walk의 permutation shift `d=Σ(-1)^i d_{i+1} mod N`, order `O(W)=N/gcd(N,d)` - lifting 후 cycle 길이/ACE를 결정.
6. 핵심기법 2단: "problematic TBC walk" 집합을 ACE spectrum 위반 순으로 정렬 후, 각 walk의 최소 edge에 nonzero shift를 부여해 order>1 강제(N을 소수로 택하면 유리).
7. 구현 디테일: DES(Deterministic Edge Swapping) 알고리즘 변형 - 입력이 dominant trapping set 대신 problematic TBC walk 순서집합.
8. 최적화 절차: greedy - `η_2`부터 `η_{2dmax}`까지 ACE spectrum 성분을 하나씩 최대화(이상적으로 모두 +∞).
9. 결과: n=990에서 ACE spectrum (+∞,+∞,17,10,5)로 기존 [11]의 (+∞,+∞,16,9,5) 상회, 시뮬에서 error floor 영역 BER/FER 우위(BP, max iter 100).
10. 한계: 순수 부호설계 - HW/디코더 미설계, 시뮬만, greedy가 전역최적 보장 안 함, 작은 base graph에 정확한 degree 분포 구현 제약.
