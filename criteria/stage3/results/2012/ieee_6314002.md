### B. 알고리즘 요약

1. 시스템: IEEE802.16e QC-LDPC(base col=24, dual-diagonal), mother code rate 1/2, BPSK/AWGN/BP(max iter 50)로 rate 2/3·3/4·5/6 서브코드 생성.
2. 문제: 기존 천공법(random/order/Ha grouping-sorting/density-evolution)은 이론·HW 복잡도가 높고 천공행렬을 수신단에 전달해야 함.
3. 핵심 가정: dual-diagonal 우측 11열이 column weight 2(최저 차수 VN)로 천공 대상 범위.
4. 핵심 기법: 저차수 VN을 우선 천공하되, 천공된 VN이 같은 검사식(공통 CN)에 인접하지 않도록 블록 단위로 선택.
5. 규칙: `b(1)`과 `b(p+1)`이 같은 검사식→홀수 블록 우선 선택으로 회복 상호간섭 최소화.
6. 확장: 삭제비트 `M_p < 6*p`면 홀수 블록만, `M_p > 6*p`면 짝수 블록 추가.
7. 구현 디테일: low-k-SR 노드 수 최대화, 천공행렬 송신 불필요(오버헤드 절감).
8. 최적화 절차: 별도 학습 없음(구조 기반 결정론적 선택).
9. 결과: 제안 천공이 random·order 천공 대비 BER 우수(Fig 6/7, rate 2/3·3/4), order가 최악.
10. 한계: HW 미설계, 시뮬만, IEEE802.16e 특정 구조 의존, 정량 gate/throughput 없음.
