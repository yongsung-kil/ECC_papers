> 총평: 무선용 rate-compatible puncturing 패턴 선택 알고리즘으로, 고정 고rate binary QC-LDPC를 쓰는 NAND ECC(Prime ECC)에는 이식 가치가 낮다.

### B. 알고리즘 요약

1. 대상: 약 `10^4` bit irregular LDPC(mother code rate 0.3~0.8), BEC/BSC/AWGN, sum-product 복호(200 iter), intentional puncturing으로 rate-compatible화.
2. 문제: 넓은 rate 범위를 노리는 puncturing 알고리즘은 작은 puncturing 비율(짧은 확장)에서 성능이 나쁘다 — 적당히 punctured된 short-length 코드 복호 최적화가 목표.
3. 모델: punctured 심볼 `v∈P`는 iteration 1에서 0 LLR 송신, `Rk`(k-step extended recoverable) 심볼은 survived check node로부터 k iter 후 복원.
4. 핵심 기법: `untainted` 심볼 = `N^2(v)` 내에 punctured 심볼이 없는 심볼 → 선택 시 모든 이웃 check가 survived node가 되도록 보장.
5. 핵심 식/성질: extended recovery error `Pe(v)=ϵv/2`; survived check node를 추가하면 `Pe(v)`가 증가하지 않는다(Theorem 1,2, BEC/일반 대칭채널 증명).
6. 확장 개념: `extended recovery tree Tv` — 단일 survived check 가정([5])을 다중 survived check로 일반화.
7. 구현 디테일: greedy — `|N^2(u)|`가 최소인 후보 Ω에서 심볼 선택→`X∞`에서 `N^2(v)` 제거 반복, 저비용. check degree가 거의 균일하면 최소차수 심볼 탐색으로 단순화.
8. 학습/최적화: 없음(그리디 조합 선택).
9. 결과: FER `10^-3`에서 [5],[6] 대비 전 rate·채널·puncturing 비율에서 우수, 동일 성능을 더 적은 punctured 심볼로 달성(Table I).
10. 한계: HW 미설계·시뮬만, untainted 제약으로 최대 달성 rate가 낮아짐, irregular non-QC 코드 대상.
