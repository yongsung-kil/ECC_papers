### B. 알고리즘 요약

1. 대상은 rateless Fountain 코드(LT/Raptor, LDGM, `x = mG`, k message bit -> n≥k encoded bit)를 AWGN/BIAWGN 상에서 BP로 복호하는 G-based Tanner 그래프.
2. 문제: error-floor를 지배하는 error-prone pattern(trapping/absorbing set)이 LDPC(H-based)에선 많이 연구됐으나 Fountain(G-based)에선 미탐구 — 개념이 자동 이전되지 않음.
3. 모델: G-based 그래프는 message node m과 encoded node x 두 종류 변수노드를 가지며, encoded node를 복제·check node로 재명명해 modulo-2 합=0 제약을 표현.
4. 핵심기법 1: encoded node는 채널 LLR `Z_j = (2/σ²)y` 상수 메시지 전달, message-check 갱신은 식(2)의 `2 tanh^-1(z_j ∏ tanh(M/2))`로 BP 수행.
5. 핵심 식: pseudo a posteriori LLR = `Σ M_{cj→mi}`, hard decision으로 message 추정 — 복호 성공/실패(고정된 unsatisfied check 집합)를 판정하는 근거.
6. 핵심기법 2: G-based absorbing set을 triplet `(α,β,z)`로 정의(α=message node 수, β=incorrect encoded node 수, z=unsatisfied check 수), 각 message node가 unsatisfied check에 satisfied check 이상 연결되면 흡수집합.
7. 부수 정의: error-free absorbing set(α=0, β=z), elementary absorbing set(check degree 1 또는 2) — 고SNR에선 대부분 elementary이며 decoder가 탈출 불가.
8. 학습/최적화: 대규모 Monte-Carlo 시뮬 + 그래프 분석으로 case ii)(unsatisfied/incorrect 집합 영구 고정)를 absorbing set으로 식별.
9. 결과: robust soliton 분포 k=1000 AWGN에서 복호 복잡도 20% 감소, BER 10^-5/10^-6에서 각 0.6dB/0.9dB coding gain 관찰.
10. 한계: HW 미설계, 시뮬뿐, encoder/decoder 실제 회피/탈출 기법은 future work, G-based LDGM 전용 정의.
