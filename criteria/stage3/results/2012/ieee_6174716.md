### B. 알고리즘 요약

1. 대상: PEG 정규 LDPC (`PEGReg504x1008` rate 0.5 col-deg 3, `PEGReg252x504`), AWGN+BPSK, hard-decision bit-flipping 계열.
2. 문제: BPA는 성능 좋으나 복잡도 큼, 기본 BF는 iteration 많음 — 저복잡도로 수렴 가속 필요.
3. 각 bit에 inversion function `E_n`(GDBF식: syndrome 합 + `x_n*y_n` 상관항)을 계산, 임계 `λ_n` 초과 시 flip.
4. 적응형 임계: flip 안 되면 `λ_n(k+1)=θ·λ_n(k)`로 낮춤 → 자동으로 multi-flip에서 single-flip으로 전이.
5. 스케일 `θ=2^-dA + 2^-dB` (두 shift 합)으로 곱셈기 제거 (예 0.75=0.5+0.25), HW 친화.
6. 핵심 기법: dual-scaling algorithm(DSA) — coarse(`θ1=0.5`)/fine(`θ2=0.97`) 두 분기 병행 후 syndrome 합이 작은 분기 채택, 이후 나머지 분기 비활성으로 전력 절감.
7. 트랩 탈출: syndrome 합이 window W(=10)간 불변이면 `λ_n += c·z_n` (균등난수, `c`=0.1~0.4) perturbation 주입.
8. 조기종료: `f(k)=f(k-10)` 또는 `f(k-11)`이면 정지 (진동 대비 두 조건).
9. 결과: DSA(θ=0.63/0.97)가 단일 θ=0.97과 동등 BER이면서 평균 iteration 66→39 (4dB), early-stop 시 17까지; perturbation으로 BER 10^-3에서 0.5dB 개선.
10. 한계: 시뮬레이션(AWGN)만, gate count/throughput 없음, 저rate col-deg 3 wireless 코드, 향후 MIMO/OFDM 지향.
