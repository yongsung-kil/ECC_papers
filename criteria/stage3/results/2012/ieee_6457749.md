> 총평: Vandermonde+산술급수 lift로 regular protograph QC-LDPC를 낮은 인코딩 복잡도로 생성하는 순수 구성법 논문으로, 디코더·HW·정정성능 평가가 없어 Prime ECC 이식 가치 낮음.

### B. 알고리즘 요약

1. 대상은 regular binary QC-LDPC 부호 (예제 γ=3, ρ=5), protograph 기반으로 낮은 인코딩/디코딩 복잡도를 노린다.
2. 문제: 기존 protograph lift는 모든 edge를 순회하며 허용/금지 edge를 판정해야 해 구성 복잡도가 크다.
3. 기본 protograph를 Vandermonde 순열행렬 `P`(식 `Pmn=1 if m=(n-1)mod q`)의 블록으로 구성, 크기 `(γq × ρq)`.
4. 순열행렬 조건 `q > (ρ-1)(γ-1)`로 girth 확보, 블록행렬 `Iq, Pq, Pq^2,...`로 basic matrix H 정의(식 3).
5. lift(enlarging) 시 파생그래프에서 특정 control node 위치를 산술급수 `Cn = C0 + T(n-1)`(식 5)로 사전 결정.
6. 이 사전결정으로 PEG처럼 전 edge를 순회하지 않고 protograph 구조를 보존하며 copy&permutation 수행.
7. 남은 edge는 수정 PEG(modified PEG, [9])로 `n·γ·(T-1)`개 연결을 최적 배치.
8. 학습/최적화 절차 없음 (결정론적 대수 구성 + PEG 후처리).
9. 결과: basic protograph (27,45)를 T=5배 lift해 QC-LDPC G'(edge 675개) 생성, girth 8, 최대 (2790,3255)/19530 edge, 구성 실행시간 약 6초, 열무게 ≤4에서 선형·>4에서 2차 복잡도.
10. 한계: BER/정정능력 시뮬레이션 전무, HW 미설계, 채널·연판정 무관, 복호 성능 이득 미제시.
