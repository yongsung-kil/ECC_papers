### B. 알고리즘 요약

1. 대상은 사영평면 `PG(2,2^s)`와 유클리드평면 `EG(2,2^s)`의 incidence 구조로 만든 정규(uniform column-weight) 이진 LDPC 부호이며, Tanner graph girth는 6이다.
2. 문제: absorbing set(MPI 디코딩이 빠져나오지 못하는 near-codeword 상태)이 error-floor를 지배하므로, 이 부호족의 최소 absorbing set 파라미터 `(a,b)`를 규명한다.
3. 모델: AWGN 채널 + bit-flipping/MPI 디코딩. absorbing set은 각 원소가 이웃 중 짝수차수 이웃이 홀수차수 이웃보다 많은 bit-node 집합 `D`로 정의.
4. 핵심 결과 1: `H_PG`로 기술된 부호의 최소 absorbing set 파라미터가 Dolecek 하한(Lemma 2)과 정확히 일치함을 증명(Theorem 4).
5. 핵심 결과 2: `H_EG`에 대해서도 동일하게 하한이 tight함을 증명(Theorem 5).
6. 핵심 식/의미: 최소 absorbing set은 평면 내 `(a, a(q+2-a)/...)`-arc에 대응 - 세 점이 한 직선 위에 있지 않은 점집합(arc)이 곧 최소 absorbing set의 필요충분조건(Theorem 6).
7. 확장: PG 부호에서 fully absorbing set은 complete arc와 1:1 대응(Theorem 16), EG 부호는 일부만 fully.
8. 예제: PG(2,4) 부호는 최소 absorbing set (6,24)가 182,280개 등, arc 개수로 absorbing set 수를 계산.
9. 결과(수치): 이 부호들의 최소 absorbing set은 `b/a` 비율이 크기 증가에 따라 급감 -> 점근적으로 unstable(안정하지 않음)임을 보여 낮은 error-floor를 설명. 50M 프레임 시뮬 결과 SNR 4.5dB에서 FER ~1.9e-6, 오류 프레임 대부분이 random error로 최소 absorbing set이 원인이 아님.
10. 한계: HW 미설계, 순수 이론/조합론이며 디코더 알고리즘·부호 최적화 절차를 제시하지 않음.
