### B. 알고리즘 요약

1. 배경: LDPC 부호의 error-floor는 반복 BP 디코더의 Tanner graph 내 trapping set(및 elementary trapping set)에 의해 발생.
2. 문제: 기존 stopping redundancy 개념을 trapping set으로 일반화하여, `(a,b)` trapping set을 모두 제거하는 parity-check matrix의 최소 row 수(trapping redundancy `T_{a,b}(C)`)를 정의하고 그 상한을 구하는 것이 목표.
3. 모델: 이진 선형 블록 부호 `C`, 그 dual code의 codeword를 무작위/독립적으로 뽑아 구성한 redundant parity-check matrix 앙상블(`M_m`)을 분석 대상으로 함.
4. 핵심 기법 1단: Lovász Local Lemma(LLL, 표준형)를 이용해 `(a,b)` trapping set이 전혀 없는 matrix가 존재하기 위한 row 수 `m`의 충분조건 `Theorem 3.4`를 유도 (`e·m^a·(2p)^b(1+D)≤1` 형태 조건).
5. 핵심 식 의미: dual code codeword가 강도 `d^⊥-1`의 orthogonal array를 이룬다는 성질로, 임의 열 부분집합의 restriction에서 홀/짝 weight row 확률이 균등함을 이용해 사건 독립성을 확보.
6. 확장: LLL의 high-probability variation을 적용해 특정 확률 `1-ε` 이상으로 trapping-set-free한 matrix를 얻는 데 필요한 row 수(`Theorem 3.6`, `3.9`)도 유도, elementary trapping set에 대해서도 별도 정리(`3.8`) 제시.
7. 구성적(deterministic) 대안: 임의의 rank-`k` full-rank parity-check matrix에 2개~`t`개 row의 모든 선형결합을 추가하는 방식으로 `T_{a,b}` 상한을 구성적으로 도출 (`Theorem 3.10`), 무작위 방법보다 loose함.
8. 점근식: `m,d^⊥→∞`인 점근 영역에서 Shannon entropy 함수 `h_2(·)`를 이용한 닫힌 형태 근사식을 제시.
9. 결과: `[2640,1320]` Margulis code에서 elementary `(a,3)` trapping set 회피에 최대 1336~1394 row, PG 부호는 최대 175~309 row로 가능함을 수치로 제시. 또한 Margulis code의 특정 `(12,4)`→`(14,7)` trapping set 확장 구조를 이용해 단 1개의 redundant row로 특정 trapping set 하나를 제거하는 구성적 예제 제시.
10. 한계: 특정 코드(Margulis, PG)의 사례 연구뿐이며 일반 QC-LDPC 구성 알고리즘·디코더 구현·HW 검증은 전혀 다루지 않음. LLL 기반 상한은 존재성만 보장하며 실제 탐색 알고리즘은 제시하지 않음.
