### B. 알고리즘 요약

1. 대상: 확장 `[q,2,q-1]` RS 부호(정보심볼 2개)로 만든 `(γ,ρ)`-regular **RS-LDPC** 부호 `C(γ,ρ)`, rate ≥ `1-γ/ρ`. 대표례는 10G Base-T의 `(2048,1723)` (rate≈0.84).
2. 문제: RS-LDPC의 PCM은 circulant가 아닌 일반 permutation matrix라 인코더/디코더 HW 구현·이론분석이 어려움(10G Base-T 칩의 silicon/전력 난제).
3. 핵심기법1(QC 표현): 열치환 `πc`(식11)+행치환 `πr`(식14)을 PCM에 적용하면 대부분이 `(q-1)×(q-1)` CPM과 zero-matrix로 이뤄진 QC 부분행렬 `H̃_QC(γ,q)`로 변환됨(Theorem 1).
4. 식 의미: 변환된 `H̃_QC`는 `γ×(q+1)` base matrix `B̃`의 `(q-1)-fold dispersion`으로 표현(Theorem 2). 비영원소가 `(δi+ωj)^-1` 형태.
5. 응용: QC 표현으로 인코딩을 shift-register/GFT 방식으로 수행 가능(복잡도 길이에 선형↓), 디코더는 permutation network 기반 configurable 구조로 wire routing·critical path 감소.
6. 핵심기법2(랭크해석): GFT 대각화 `rank(H)=Σrank(Bl)`(식6) + Lucas 정리로 `rank(H(γ,q))=γ+3^r-2^r-Σ(...)` 폐형식 도출(Theorem 3), 임의 `γ≤q`에 적용.
7. 핵심기법3(벡터 표현): 각 permutation matrix를 `Fq` 원소 벡터로 압축, `Aˆ_{j+1}=α·Aˆ_j`(Theorem 6).
8. 성질: full-length RS-LDPC에 **affine invariance** 증명(Theorem 5) → 최소무게codeword·stopping/trapping set 등 구조분석을 한 위치 분석으로 환원.
9. 결과: 4-cycle 없는 RC-constraint 유지, `(2048,1723)` 등 비-QC 표준부호를 QC 구현 가능케 함. (수치 BER/gatecount 없음)
10. 한계: 순수 구조/랭크 이론(대수적)만, 실제 HW 합성·throughput·오정정 성능 검증 부재. 정보심볼 2개 RS 기반으로 한정.
