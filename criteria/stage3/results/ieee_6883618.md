> 총평: girth-4 cycle의 double counting을 확률영역 조건부 보정으로 없애 error floor를 낮추는 이론 연구지만, 확률영역 재귀연산이라 min-sum 기반 Prime ECC에 이식 어려움.

### B. 알고리즘 요약

1. 대상: block length `n=1200`, rate `0.5`, regular (col-weight 3, row-weight 6), girth `g=4` isolated cycle 200개를 심은 H-matrix, AWGN 채널.
2. 문제: loopy BP는 cycle이 있는 Tanner graph에서 message가 시간에 따라 종속되어 독립성 가정이 깨지고 error floor 유발.
3. Lemma 1: girth `g` graph에서 message는 `t = g/2` iteration 이후 자기 자신에 종속됨 (`v1(t+g/2)`가 `v1(t)`에 의존).
4. Double counting: girth-4에서 채널 신뢰도 `p2`가 결정 단계에서 두 번 반영(`p1*p2*p2`)되어 gain saturation 발생.
5. 핵심 기법: cycle set(`va,vb,ca,cb`)의 message 종속성을 확률영역에서 조건부 확률로 명시 처리, `P r(c2=0|c1=0)=1`을 이용해 `p1*p2`로 보정.
6. 종속 항 `γ` 정의: `γ_va = P r(ca=0, cb=0 | xa=0)`, cycle이 유발한 종속을 포착.
7. `γ = α + β`로 분해, `α`,`β`를 이전 iteration으로 거슬러 올라가는 재귀식(`Δ(t)` 함수 포함)으로 계산.
8. 수정된 variable node update: `P r(va_j=0) = pa·γ_va·∏P r(ck=0)` — cycle 외부 message는 기존 update, cycle 내부만 γ로 보정.
9. 결과: 200개 girth-4 cycle에서 1 iteration 후 고SNR(>5dB) 약 0.2dB, 10 iteration 후 약 0.5dB SNR 이득. Theorem 1로 `Pe(ML) ≤ Pe(MMPA) ≤ Pe(MPA)` 경계 주장.
10. 한계: 확률영역 SPA·재귀연산, girth-4 isolated cycle에만 적용, HW 미설계, ML 증명 생략, 복잡도 오버헤드 인정.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 확률영역 수정 message passing (γ 보정) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | Prime ECC는 min-sum(LLR 도메인)이라 확률영역 SPA 재귀식과 도메인 불일치 |
| double counting 제거 / error floor 대책 | `decoder.cpp` `VN_Cal_HD()` 등 | 개념적으로 VN 결정 단계 대응이나 확률영역 계산은 이식 부담 큼 |
| girth-4 cycle 구조 인식 | 대응 없음 | Prime ECC는 특정 QC-LDPC 고정, cycle isolated set 식별 로직 없음 |

적용 가치: **낮음** — error floor 완화 아이디어는 흥미로우나 확률영역 SPA 재귀연산이 min-sum HW 엔진과 도메인·복잡도 면에서 정합하지 않고 HW 미설계.
