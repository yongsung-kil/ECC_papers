### B. 알고리즘 요약

1. 시스템: 2D 그리드 저장 매체, bipolar 신호 `Y = C∗X` (`∗`=성분별 곱, `X`=에러패턴, `C`=codeword), rate 0.05~0.5 generic regular LDPC.
2. 문제: NAND식 랜덤노이즈 가정과 달리 실제 매체 에러는 인접 영역을 덮는 2D 버스트/클러스터로 발생 → 표준 반복복호로는 부족.
3. 모델: 2D 에러를 2차 근방(second-order neighborhood) Markov Random Field(MRF)로 모델링, 결합확률은 Gibbs 분포 `P(x)=Z^{-1}exp(-U(x)/T)`.
4. 핵심기법: LDPC SISO의 입출력 LLR 차 `Δi = Yi(L_I - L_O)`를 에러 관측치로 삼아 에러패턴 `X`를 MAP 추정.
5. 핵심 식: `V(x)=Σ[α x_i + β Σ_{Ni} x_i x_i' + (μ/σ²)δ_i x_i]` — 각 비트값·근방상관·SISO 관측치를 결합한 목적함수(bipolar `x_i²=1`로 단순화).
6. 확장: `Δi | Xi`를 평균 `x_iμ`·분산 `σ`의 정규분포로 근사(식 3)하여 지수형 우도로 표현.
7. 구현 디테일: 전역최적 대신 노드별 `α x_i + β Σ x_i' + (δ_iμ/σ²)`의 부호를 양수화하는 국소최적(보통 10회 미만 수렴).
8. 최적화: MRF 클러스터 추정치 `X̂`를 다시 LLR `Li=P(Ci|Yi,X̂i)`로 변환해 SISO에 a priori로 재주입, 유효 codeword까지 반복.
9. 결과: rate가 낮을수록 MRF-LDPC가 표준 LDPC 대비 BER 대폭 개선(Fig.6/7), 에러율 상계 `Eh` 클수록·클러스터 클수록 이득 큼.
10. 한계: 2D 공간상관 채널 특화, 국소최적(전역보장 없음), HW 미설계·시뮬뿐, β/ρ 등 파라미터 사전지식 필요.

