### B. 알고리즘 요약

1. 대상은 binary CPM-QC-LDPC: base matrix `B`를 GF(q)의 두 임의 부분집합 `S1`, `S2`와 곱셈인자 `η`로 `B(η)=[ηα^ik + α^jl]` 형태로 구성.
2. 문제: 고rate·저 error-floor QC-LDPC를 유연하게 구성하고, 디코더 HW의 CN 유닛/배선 수를 줄이는 것.
3. `B`의 각 원소 `α^l`을 (q-1)×(q-1) CPM으로 dispersion → parity-check array `H` 생성. 기존 유한체·조합 구성법들이 이 프레임의 특수 경우.
4. girth 조건: 2×2 SM-constraint(girth≥6), 3×3 SM-constraint(girth≥8)를 base matrix 상에서 대수적으로 판정.
5. masking: base matrix의 일부 nonzero를 0으로 치환(Hadamard 곱 `Bmask=Z⊗B`)해 short cycle 제거·girth 확대.
6. 예: 4×8 base를 `Z(4,8)`로 masking → (3,6)-regular (2640,1320) 부호, girth 6→8, length-8 cycle 24,750→330.
7. 3×3 SM-MMSA 탐색 알고리즘으로 girth 8/10 만족 masked matrix를 체계적으로 탐색, `Z(4,4k)`·`Z(4+l,4t)` 확장으로 rate 1/2~9/10 부호열 생성.
8. 디코딩: CPM-RID(CPM-RMSA) - `H`의 section-wise cyclic 구조를 이용해 첫 row-block 대응 decoding matrix `H0(1)`만으로 revolving 복호.
9. 결과: (16120,15345) rate 0.952 부호가 50 iter scaled-MSA로 BER 10^-14, error-floor 없음; CPM-RID는 CN 유닛/배선을 (q-1)=130배 감소(6 CN, 744 wire).
10. 한계: RTL/게이트카운트 없음(FPGA 측정만), CPM-RID는 복잡도↓ 대가로 sub-iteration 증가 → latency 증가(l 조절로 trade-off).
