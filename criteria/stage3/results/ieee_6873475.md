### B. 알고리즘 요약

1. 대상: 차세대 지상파 방송(8K SHV), dual-polarized MIMO-OFDM, 1024/4096QAM, 32k-FFT, FEC로 SC-LDPC 제안.
2. 문제: DVB-T2 LDPC block 부호가 Shannon 근접이나 BER 추가 개선 여지가 있음.
3. 채널: AWGN(이상 조건, cross-polarization 없음), Zero-forcing MIMO 복호.
4. 핵심 기법: DVB-T2 irregular LDPC를 component base matrix `H`(길이 `n`, parity `m`)로 삼아 cycle을 최소화.
5. SC 구성: base의 parity부를 `HL`,`HU`로 랜덤 분해 후 대각 근처에 `L`개 결합 → 부호길이 `L(n-m)+(L+1)m`, rate `R'=(n-m)/(n+m/L)`.
6. rate loss: `(L+1)m` parity로 rate가 base보다 소폭 감소, `L→∞`이면 `R'→R`. 실험은 `L=20`, base 길이 12960.
7. 복호: 기존 sum-product 그대로. 대각 국소 연결로 windowed(row/column op를 `HU_i,HL_i` 순차 처리) 복호, 처리 완료 블록 메모리 해제·재사용.
8. 메모리 이점: 복호 메모리가 SC 전체 길이(~26만)가 아니라 base matrix 크기에 비례 → HW 구현 유리.
9. 결과: QPSK AWGN에서 DVB-T2 대비 required CNR(BER<1e-7) 0.1/0.15/0.1dB(R=2/3,3/4,5/6) 개선. pseudo-LLR 반복계산은 4096QAM에서 최대 0.8dB 추가 개선.
10. 한계: 시뮬만(HW 미설계), 이상 채널 가정, 초장 SC-LDPC, 이득 소폭, 방송 QAM 특화.
