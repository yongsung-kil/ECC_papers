### B. 알고리즘 요약

1. 시스템: mother code rate-1/2, length `N=2048`, irregular LDPC, BPSK+AWGN, Layered BP 복호, IR-HARQ(펑처링으로 rate 0.5~0.9 가변).
2. 문제: 무선 ACM/HARQ는 여러 rate·길이를 균일 구조로 필요로 하는데, 짧은 길이에서 정확 프로파일(RCT/ARCT)과 펑처링 성능(E2RC)을 동시에 만족하기 어려움.
3. 모델: Tanner graph, degree distribution `(λ(x), ρ(x))`, k-SR(k-Step Recoverable) 노드 - 펑처된 VN이 k iteration 후 복원되는 구조.
4. 핵심기법: 2RC 알고리즘 - PCM을 systematic `H1`과 non-systematic `H2`로 분할, `H1=I(M,M)` 초기화로 girth cycle 방지·구성 가속.
5. H2 구성: E2RC로 k-SR 행렬을 `M×(M/2^k)` 크기로 순차 배치(식 (3)(4)), depth `d=log2 M`, 저-SR 노드를 많게 해 펑처링 성능 확보.
6. H1 구성: SUT(Speed-Up Technique)로 supplier `u`에서 무작위 행 선택해 1 배치, length-4 cycle 형성 시 재선택.
7. 프로파일 보정: H2에서 생긴 고차 row를 막기 위해 modified supplier로 row index 재계산(NCD/NRD/RI 정의).
8. 최적화: rate-1/2 AWGN 최적 degree 분포 `λ(x)=0.00015+0.30235x+0.27132x^2+0.42618x^6` 등 사용, H2 보정 위해 실제 분포 약간 조정.
9. 결과: rate 0.8, BER 1e-5에서 SUT 대비 2.8dB 펑처링 이득; IR-HARQ throughput 0.8에서 2.5dB 이득, throughput 0.9까지 동작(SUT는 불가).
10. 한계: HW 미설계, 무선 IR-HARQ 타깃, rate-compatible 펑처링 구성법이라 NAND ECC 관점 신규성 없음.
