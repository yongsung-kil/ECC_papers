### B. 알고리즘 요약

1. 시스템: 모바일 통신용 rate-compatible(RC) length-scalable(LS) QC-LDPC 부호군, raptor-like nested 구조, BI-AWGN 채널.
2. 문제: 다중 rate·length를 지원하려면 단일 encoder/decoder 쌍이 바람직한데 기존 puncturing/shortening은 많이 뚫을수록 성능 열화, QC 임의길이 구성은 HW 부담.
3. 모델: `(N,K)` raptor-like PCM = `[[A B 0],[... C I]]`, 후단 check bit은 상호 비연결(확장 용이). template matrix `T`(circulant shift 정수)에 lifting factor `b`로 length 조절.
4. 핵심기법: check-node extension(하향 rate) + information-node extension(상향 rate)을 progressive하게 적용해 **단일 nested template matrix** 하나로 부호군 생성.
5. 식 의미: 확장 시 MET-DE의 SNR threshold metric `d`를 rate 집합에 대해 누적(식2) → 여러 rate에 균형 잡힌 sub-optimal edge 선택.
6. 확장: information-node는 열 1개, check-node는 행+열 1개씩 추가하며 각 단계 MET-DE로 최적 edge 후보 평가.
7. 구현 디테일: 탐색공간이 큰 경우(예 227 후보) edge switch algorithm으로 조기종료(X회) 기반 축소; lifting factor `b=2^j` 단순 집합 사용.
8. 최적화 절차: MET-DE(multi-edge-type density evolution) 분석으로 template별 threshold 예측 후 sub-optimal edge 그룹 선정.
9. 결과: 예제 template(Kb 12~24, Mb 4~73)로 rate 0.143~0.889 커버(0.25~0.8 완전커버), MET-DE상 Shannon limit 대비 0.30~0.48 dB @ BER=2e-9, 60 iteration.
10. 한계: HW/디코더 미구현, 결과는 MET-DE 이론 예측(BER 곡선 실측 아님), flooding sum-product 가정.
