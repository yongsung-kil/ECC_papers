### B. 알고리즘 요약

1. 시스템: 4 Tb/in² 초고밀도 자기기록(BPMR/TDMR) 2D-ISI 채널, rate `Rc=8/9`, 코드길이 `Nc=4608, 9216`.
2. 문제: AWGN용 최적화 LDPC는 채널 비대칭 때문에 자기기록에서 비최적, 기존 2D-ISI 최적화는 detector마다 재실행 필요하고 EXIT 곡선이 폐형식이 아니어서 Monte Carlo 부담이 큼.
3. 가정: 특정 2D-DET(symbol-based BCJR)의 출력 LLR이 대칭 가우시안(mean `k`, var `2k`)으로 근사된다는 관측.
4. 핵심 기법: 2D-DET 출력 LLR의 추정 분산이라는 단일 파라미터로 ISI채널을 AWGN 최적화 프레임에 연결, VND EXIT 함수 `IE,VND` 를 이 분산으로 재구성(식 5).
5. 핵심 식: `IE,VND = Σ λi × J(...)` 에서 detector 분산을 J⁻¹ 항에 삽입 — 채널 오차 특성을 하나의 분산값으로 포착.
6. 최적화: 선형계획으로 degree sequence `λ,ρ` 최적화(Algorithm 1), 2D-DET을 `Lλ·Lh` 회 대신 1회만 실행해 계산량 절감.
7. 구현 디테일: 최적 degree로 protograph base matrix `Hp` 구성 후 PEG 변형으로 edge permute, QC 구조 유지하며 girth 최대화 → 저복잡도 QC 인코더/병렬 protograph 디코더.
8. 결과: Scheme2(AWGN최적화) 대비 BER 10⁻⁵에서 약 0.2 dB 이득, random 대비 최대 1 order magnitude BER 이득, BER 10⁻⁸까지 error floor 미관측.
9. Scheme1(BCJR 최적화) 대비 성능 손실 미미하나 훨씬 낮은 복잡도, high-SNR에서 더 낮은 error floor.
10. 한계: HW 미설계, 시뮬만, 특정 2D-DET·자기기록 채널 가정에 강하게 종속.
