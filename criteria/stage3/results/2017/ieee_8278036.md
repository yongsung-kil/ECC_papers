### B. 알고리즘 요약

1. 시스템: TLC NAND의 셀 전압을 8-PAM 신호로 보고 LDPC coded modulation + IDD 수신기 구성, code length `n=9216`/`18432`.
2. 문제: 기존 Gray 매핑 coded-modulation은 검출기가 LDPC 디코더에 여분의 extrinsic 정보를 못 주는 경우가 있어 IDD 이득이 제한됨.
3. 모델: `y = x + n` (Gaussian noise, 분산 `σ²`), 심볼당 `m=3` 비트, 비대칭 채널(0/1 pdf 분리) 가정.
4. 핵심기법1: 이웃 심볼끼리 `m-1=2` 비트가 다르도록 하는 신규 8-PAM 매핑을 [14]의 cost function `Dcost` 최소화로 설계.
5. 식 의미: 검출기의 extrinsic LLR `Le(c_ib)` (식1)을 최대화해 IDD 외부 루프가 LDPC 내부 루프에 더 많은 정보를 전달하게 함.
6. 핵심기법2: 비대칭 채널용 수정 density evolution(식13~16)으로 매핑별 최적 degree distribution `λ,ρ` 도출 → 제안 매핑은 degree-2 VN이 90% 이상인 초희소 부호.
7. 구현 디테일: 초희소 그래프 → 인코딩·디코딩 복잡도(edge 수)가 Gray(43200) 대비 최소(24480)로 감소.
8. 최적화: 비대칭 DE로 decoding threshold(≈14.6) 기준 degree distribution 탐색.
9. 결과: 인터리버로 2-chain/2-cycle degree-2 VN을 서로 다른 심볼로 분리 → error-floor를 한 자릿수(>1 order) 이상 낮춤, FER 10^-6 근처 개선.
10. 한계: 순수 시뮬레이션(HW 미설계), 8-PAM/IDD coded-modulation 구조에 강하게 종속, throughput·gate count 미제시.
