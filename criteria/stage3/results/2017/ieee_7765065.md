> 총평: 802.15.3c용 binary QC-LDPC layered NMS 디코더 실측칩. TD-MVIG 자체는 아날로그성이라 이식불가지만, Q-기반 layered 메모리축소·불균형 M1/M2 양자화는 아이디어 참고 가능.

### B. 알고리즘 요약

1. 대상 부호: IEEE 802.15.3c binary QC-LDPC, `N=672`, 4가지 rate(`1/2, 5/8, 3/4, 7/8`), circulant `z=21`, `dc=32`, base matrix 간 구조관계로 멀티모드 지원.
2. 문제: multi-Gb/s NMS 디코더의 critical path가 CN의 min1/min2/index 생성(MVIG)에 몰림 → 이를 빠르고 작게 만들면서 긴 부호까지 scalable하게 해야 함.
3. 복호 알고리즘: normalized min-sum(NMS), `R = α·S·min`, `α=0.75`, block-row layered scheduling(레이어당 1사이클 갱신), layered 기반 조기종료(syndrome check).
4. 핵심기법1(TD-MVIG): DTC로 값을 시간영역 지연으로 변환하고 arbiter로 "먼저 도착=최소" index를 병렬 검출, M1/Idx1을 AND-mask 후 2차 arbiter로 M2/Idx2까지 1사이클에 산출. TDC 불필요 → DTC-TDC 매칭문제 제거.
5. 의미: min1/min2/index를 짧은 latency·작은 면적으로 뽑아 [19]의 short-code 한계(scalability)를 극복하고 긴 실용 부호 지원.
6. 핵심기법2(면적효율): `(k-1)`-b DTC array로 MSB 최대지연 셀 제거(dc=32에서 면적 18.9%↓, latency 52%↓), 불균형 양자화(M1은 M2의 절반 clipping → M1 양자화비트 1b 절감).
7. 구현 디테일: APP/Q magnitude는 `(4,1)` 양자화, M1=`(2,1)`, M2=`(3,1)`; 불균형 양자화로 약 50k gate 절감(BER 열화 미미).
8. Q-기반 layered 아키텍처: `Λ` 대신 V2C(`Q`)를 저장, Q-value/Λ-value를 인터리브로 갱신해 압축 C2V를 4 layer 대신 3 layer만 저장 → [3] 대비 메모리 약 40%로 축소.
9. 결과(실측): 90nm CMOS, 495k gate, 2.25mm², 157MHz, 5.28Gb/s(5 iter), 조기종료 시 BER=1e-5에서 30.08Gb/s, 182mW, 34.47pJ/b — 긴 부호 지원 TD/analog 디코더 중 최고 HW효율.
10. 한계: 정정능력 개선이 아닌 HW 효율·면적·throughput 개선, TD-MVIG는 공정변이·metastability 대응이 필요한 아날로그성 회로, 무선표준(802.15.3c) 타깃.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| NMS CN min1/min2·sign | decoder.cpp CNU_Update_New_Mag(), C2V_Cal_New_Sgn() | 중: Prime도 min1/min2 min-sum 보유, α=0.75 및 M1/M2 불균형 양자화 아이디어 참고 가능 |
| VN 처리·LLR 양자화((4,1) 등) | decoder.cpp VN_Cal_HD() 등, ecc_data.h PARAM_LLR | 중: 양자화비트 절감(불균형 M1/M2) 개념이 magnitude 양자화 테이블에 응용 여지 |
| layered 조기종료(syndrome) | partial_CRC.cpp partial_crc_HW_T4(), full_CRC.cpp | 낮음: Prime은 partial/full CRC 조기종료 이미 보유, 방식 상이 |
| TD-MVIG(arbiter/DTC 타이밍 회로) | 대응 없음 | 아날로그성 회로라 bit-exact C++ 시뮬로 모사 불가 |
| QC-LDPC PCM/circulant 로드 | ecc_top.cpp Load_PCM() | 낮음: 부호구조는 802.15.3c 고정, Prime과 별개 |
```

적용 가치: 중간 — binary QC-LDPC layered NMS라는 공통점 덕에 불균형 M1/M2 양자화와 Q-기반 layered 메모리 축소는 알고리즘/HW 아이디어로 참고 가치가 있으나, 핵심 신규성인 TD-MVIG는 아날로그성 회로라 Prime ECC의 디지털 bit-exact 모델에 직접 이식할 수 없다.
