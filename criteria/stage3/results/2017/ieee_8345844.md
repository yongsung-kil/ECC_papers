### B. 알고리즘 요약

1. 시스템: coherent optical FEC용 (36000,30000) regular QC-LDPC(dv=4, dc=24, rate 5/6, 20% OH), finite-field subset법 구성, BPSK/AWGN, 5bit 고정소수(1sign·3int·1frac) 양자화.
2. 문제: SD-LDPC 디코더가 long-haul 100Gbps 링크 에너지의 15~20% 소비하고 ASIC 면적 큼 → 성능/복잡도 절충 필요, 기존 AD 알고리즘 개선.
3. 모델: VN 메모리 `Mv`를 채널 LLR로 초기화, VN↔CN이 **binary 메시지** `bv→c/bc→v` 교환(bit-flipping 계열), CN은 parity `pc = ⊕ bv→c`.
4. 핵심기법(PAD): AD에 **prior 정보 보존** 추가 — 초기 부호 `κv=sgn(Mv)`와 크기 `λv=max(γ1,|Mv|)`를 저장, VN 갱신 합 `σ`에 prior 기여 `ρ`를 조건부 주입.
5. 핵심식: `Mv ← Mv + s·σ ∓ δ` (degeneration factor δ, scaling s); 진전 없으면 δ를 γ0→γ1로 키워 trapping set 탈출.
6. 확장: prior 부호 κv를 λv≥T1이면 매 iteration, T0<λv<T1이면 매 4번째 iteration만 σ에 기여; 미충족 parity 수가 τ 미만이면 register로 비활성(잘못된 prior 간섭 방지).
7. 트릭: `|Mv| ≤ λv`로 클램핑 → 잘못된 VN이 최대 magnitude로 커지는 특정 trapping set 정정력↑, λv는 γ1 이상으로 하한(정답 bit mass-flip 방지).
8. 최적화: free param(s=0.5, γ0/γ1=0.5/3.0, τ=64, T0/T1=4.0/7.5, im=74)은 구현서 실험적 고정.
9. 결과: BER 10^-15에서 NCG 11.0dB 추정(Shannon 1.65dB 이내), AD 대비 +0.4dB·NMSA와 격차 0.4dB; 28nm FD-SOI 7.72mm2, 160Gbps, 3.37 pJ/bit(타 SD-LDPC 20~60 pJ/bit 대비 낮음).
10. 한계: AD 대비 면적·에너지 ~50%↑, dv=4의 느린 수렴으로 im=74·throughput 저하, 광통신 특화·prior 기반이라 error-floor 이득이 코드 종속.
