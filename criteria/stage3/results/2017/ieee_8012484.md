### B. 알고리즘 요약

1. 대상은 순환행렬(circulant) 블록으로 구성된 binary `QC-LDPC`, base matrix `B`를 masking matrix `Z(m,l)`로 곱해 일부 submatrix를 0으로 만드는 masking 기법.
2. 문제: 기존 UEP 코드(puncturing, irregular degree 설계, Plotkin형)는 short cycle 유발·다중 디코더 필요 등 단점 -> masking으로 UEP를 얻으려 함.
3. 핵심 지표는 `blockwise separation` `d_j` = j번째 레벨 비트가 0이 아닌 codeword의 최소 Hamming weight로 정정능력을 규정.
4. shift criterion(Theorem 1): 각 레벨 최소 codeword `c_j^min`의 left-shift `c~_j^min`이 `S_{j-1}`에 속하면 `d_1 <= d_2 <= ... <= d_l` (UEP) 보장.
5. 이 부등식이 필요한 이유: 상위 레벨일수록 separation이 커져 잡음 내성이 강한 계층적 보호를 수학적으로 확보.
6. Corollary 1: base matrix가 modified blockwise circulant, masking matrix가 상삼각+각 k-대각 원소 동일이면 Theorem 1 자동 충족 -> 쉽게 구성 가능.
7. 세 가지 구조 제시: triangle / terminal triangle / removal triangle form (masking 대각 일부를 0으로 치환).
8. strict UEP(Theorem 2, Corollary 2): `nullity([E_uep]2)=nullity(M_uep)`이고 circulant 크기 `p`가 충분히 크면 `d_1 < d_2 < ... < d_l` 엄격 보장 -> nullity 비교로 손쉽게 검증.
9. 결과: 예제로 length 6986/rate 0.4286, length 7376/rate 0.5 등 다양한 rate/length UEP 코드 구성; AWGN·BPSK·sum-product 100 iter 시뮬에서 time-sharing 대비 우수.
10. 한계: 순수 부호 구성 이론(HW 미설계), UEP 목적 자체가 NAND 페이지 균등보호와 부합하지 않음.
