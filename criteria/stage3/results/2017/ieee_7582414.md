### B. 알고리즘 요약

1. 시스템: binary QC-LDPC(WIMAX/WIFI 표준, rate-1/2, block 1152/1296, cyclic-permutation+all-zero 서브행렬), BI-AWGN·Rayleigh 채널의 sequential BP 복호.
2. 문제: 기존 동적 스케줄(RBP/NW-RBP 등)은 소수 노드가 자원을 독점하는 greedy group과, 전혀 참여 못하는 silent VN 문제 발생.
3. 배경: BP는 `Ccm→vn = 2tanh⁻¹(∏ tanh(Vvj→cm/2))`, `Vvn→cm = Lvn + ΣCcj→vn` 갱신, C2V residual `r = ||C_pre − C_new||`로 다음 갱신 대상 선택.
4. 핵심1(SVNF-free 유지): binary 벡터 `u`로 VN 참여상태 표시, 한 round 내 각 VN 1회만 참여하게 강제해 silent VN 제거, round 종료 시 u 리셋.
5. 핵심2(동적화): 고정순서 대신 방금 갱신된(최신·신뢰도 높은) VN을 다음 참여 노드로 선택 → 최신 메시지 활용으로 수렴 가속.
6. 핵심3(오류 우선): 선택된 VN v_p의 unsatisfied 이웃 CN만 대상으로 on-demand residual 계산·최대값 선택, 오류 가능 VN으로 갱신을 우선 전파.
7. 복잡도: unsatisfied CN에서만 residual을 on-demand 계산하므로 SVNF 대비 residual 계산·real comparison 수 감소(Table I).
8. 결과: SVNF 대비 message update·residual·comparison을 중~고 SNR에서 60~70% 절감(Fig.4), FER 성능 무손실이며 iteration<25 구간에선 SVNF보다 우수(Fig.2/3).
9. 부가: 오류 VN으로 향하는 C2V 갱신 비율이 SVNF보다 높아(Fig.5) 오류 VN이 더 빨리 정정됨.
10. 한계: 시뮬만(HW 미구현), full tanh SPA·sequential 스케줄 기반, 표준 무선 채널 대상(NAND/SSD 미검증).
