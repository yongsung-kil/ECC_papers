### B. 알고리즘 요약

1. 대상: BSC 상 고rate regime, `t`-error BCH 성분부호로 만든 **spatially-coupled GLDPC**(=generalized product / braided block code), design rate `R=2k/n-1`(고rate~1).
2. 문제: capacity 근접 부호는 모두 soft(APP/BP) 복호에 의존 - 광/wireline 초고속 시스템엔 복잡도 과다. hard-decision으로 capacity 근접 가능한가.
3. 모델: `(C,m,L,w)` SC-GLDPC 앙상블, degree-2 bit node + degree-`n` BCH constraint node, BSC(p) all-zero 가정.
4. 핵심 기법: **extrinsic message passing HDD(EMP)** - constraint node로 가는 후보벡터의 `k`번째 원소를 채널값 `r_i`로 치환해 extrinsic만 전달 → miscorrection 있어도 DE 엄밀 분석 가능.
5. 핵심 식: 이상(ideal) 복호 DE `x^{(l+1)}=p·f̂_n(x^{(l)})`, `f̂_n(x)=Σ_{i≥t}C(n-1,i)x^i(1-x)^{n-1-i}` - constraint node 입력에 `t`개 초과 오류 시에만 오류 전파.
6. BDD 확장: miscorrection 확률 `P_n(i),Q_n(i)`를 BCH weight spectrum `A_l`로 폐형화(식 13,14), DE에 반영해 miscorrection까지 엄밀 처리.
7. 고rate 스케일링: `p=ρ/(n-1)`로 두면 DE가 Poisson 재귀 `λ^{(l+1)}=ρφ(λ;t-1)`로 수렴, k-core threshold와 동치.
8. 이론 결과: Theorem 5 - 임의 `ε>0`에 대해 `(t,n,L,w)` 존재하여 SC-GLDPC iterative HDD가 `ε`-redundancy achieving(고rate에서 capacity 근접).
9. 구현: 저복잡도 EMP(Algorithm 1) - constraint node를 `{0,1}` 2회 BDD + 채널값 선택으로 근사, IMP 대비 extrinsic 등가성(Table I)만 제시.
10. 한계: 순수 이론(DE threshold·증명), HW 미설계, hard-decision·BCH 성분부호 기반이라 binary LDPC와 패러다임 상이, 고속 광통신 타깃.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| iterative HDD / EMP 메시지 스케줄 | decoder.cpp LDPC_Decoder() | 반복복호 루프 층위는 유사하나 min-sum이 아닌 BCH BDD 기반이라 직접 대체 불가 |
| BCH 성분부호 syndrome/BDD 복호 | 대응 없음 | Prime ECC는 SPC-check 기반 binary QC-LDPC, 성분부호 복호기 없음 |
| SC-GLDPC / product code 부호구조 | 대응 없음 | QC-LDPC(base+shift) 고정, generalized/product 구조 미지원 |
| hard-decision(hard-1bit) 채널 | channel.cpp Set_R_Offset() | HD 지원은 있으나 부호구조가 달라 성능/양자화 재사용 불가 |
```

적용 가치: **낮음** - 강력한 이론이지만 BCH 성분부호 기반 GLDPC/product code로 Prime ECC의 binary QC-LDPC + min-sum과 부호·복호 아키텍처가 근본적으로 달라 이식할 코드 요소가 없다.
