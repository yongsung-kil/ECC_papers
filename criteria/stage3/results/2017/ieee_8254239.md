### B. 알고리즘 요약

1. 시스템: 외부 오류정정부호 + 내부 변조(coded modulation/FTN/QAM)로 구성된 직렬연접 BICM을 AWGN에서 iterative turbo 수신(SISO demod + BCJR outer) 처리; rate `R=k/n`.
2. 문제: SC-LDPC의 threshold saturation 이점을 일반 직렬연접(turbo-like) 시스템으로 일반화하고 수렴 속도(iteration)까지 최적화하려 함.
3. 모델: 연접구조를 MET/protograph LDPC의 compact graph로 벡터화 표현, 결합은 edge spreading rule 변형으로 정의.
4. 핵심기법1: coupling matrix `B=[b0..b_ms]`(bundle 폭)을 도입, coupling length `L`·syndrome former memory `ms`로 SC-TC 구성; termination/tail-biting으로 경계 처리.
5. 핵심식: 종단 시 rate `R_L = LR/(L+ms)`로 rate loss `ms/(L+ms)·R` (L→∞에서 소멸) - 경계의 known bit가 wave effect로 신뢰 LLR을 중심으로 전파.
6. 확장: `B`를 연속값 행렬(`b_i∈[0,1]`, `∑b_i=1`)로 재정의해 최적화를 feasible하게 만듦.
7. 구현 디테일: SISO 전달함수 `TM/TO`는 해석식 부재로 Monte Carlo로 추정, SC-EXIT는 P-EXIT처럼 edge에 `{b_k}` 가중.
8. 최적화: threshold 고정 하에 iteration 최소화하는 `B_opt`를 interval halving + differential evolution으로 탐색.
9. 결과: CC+FTN에서 uncoupled 대비 결합이득 1dB, Area theorem bound 대비 0.45dB(16-QAM 0.29dB); `B_opt`로 ms=2~6에서 수렴 1.7~3배 가속.
10. 한계: 순수 이론/시뮬(EXIT·threshold), HW 미설계, 대상이 turbo-like 연접부호·FTN/QAM으로 NAND binary LDPC와 무관.
