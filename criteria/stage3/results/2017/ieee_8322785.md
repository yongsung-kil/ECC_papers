### B. 알고리즘 요약

1. 시스템: FTN(faster-than-Nyquist) 위성통신, AWGN 채널, LDPC CC rate `R=1/2`, base matrix `12×24`, MAP equalizer + iterative LDPC 디코더 결합.
2. 문제: FTN은 Nyquist 대비 대역 ~80%로 줄여 전송률↑하지만 심한 ISI 발생 → 이 채널에 맞는 LDPC 부호 최적화 필요.
3. 모델: FTN 샘플링 간격 `τ=0.74`, roll-off 0.4 RRC 펄스, QPSK, Ungerboeck 모델 MAP equalizer, iterative demod-decoding 루프.
4. 핵심 기법1: 시간불변 LDPC CC를 tailbiting해 QC LDPC block code의 parent code로 취급, `H(D)` 다항식 parity-check로 정의.
5. 핵심 지표: girth profile `gGP`(submatrix 별 girth 열)과 sliding-window girth `gW`(윈도 길이 W 내 최소 girth)로 error-floor 저감을 정량화.
6. 핵심 기법2: 최적화 3단계 — (1)degree분포, (2)base matrix, (3)degree matrix 탐색.
7. degree분포: EXIT chart로 구체적 H 구성 없이 성능 예측, GA·small-Gauss increment로 FTN 특화 분포 탐색(작은 base matrix는 exhaustive).
8. base/degree matrix: 5개 circle criteria(공통위치·triple·gGP·ACE-like·최소ACE cycle 수)와 GP/SWG target 만족까지 열 단위 랜덤 생성·재생성.
9. 결과: FTN 시스템에서 최적화 LDPC CC(C1)가 WiMAX LDPC BC 대비 약 2.9dB gain, 최적화 LDPC BC(B1) 대비 추가 이득, 고 SNR에서 error floor 지연/미발생.
10. 한계: HW 미설계, 시뮬만, FTN/위성 채널 특화 부호 구성법으로 부호가 시스템 파라미터에 강하게 종속.
