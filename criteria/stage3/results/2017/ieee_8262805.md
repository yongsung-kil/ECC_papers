### B. 알고리즘 요약

1. 시스템: binary-input AWGN 채널, `q`-bit 양자화 MSA 디코더, array-code 예시(rate 0.48 `n`=25 / rate 0.95 `n`=3721).
2. 문제: 기존 SPA용 absorbing set error-floor 하한을 MSA에 그대로 쓰면 sub-optimality 때문에 하한이 시뮬을 과대추정(잘못됨).
3. 모델: absorbing set `A`의 채널 LLR `s`와 나머지 `o=(s,o)`로 분해, 각 원소는 `t=2^q` 양자화 레벨, all-zero codeword 가정.
4. 핵심: FER 하한 `γ≈N·Pr(E_A)`, `Pr(E_A)`는 복호 실패하는 `xi` 집합 `Ω(Y)`의 확률합으로 근사.
5. 핵심 식: `FER ≳ N·Pr(E_A) ≳ Nδ`, `δ=Σ_{xi∈Ω̂(Y)}Pr(s=xi)` — absorbing set에 갇히는 채널 패턴 확률.
6. MSA 보정: ML/SPA에서 성립하던 max-LLR 가정(`Ω̂(Y)=Ω(ymax)`)이 MSA에선 틀림 — `o=ymax`가 아닐 때만 실패하는 "fake" `xi` 존재.
7. 개선 기법: `M+1`개 벡터 `yjm=(t-m,...)`로 `Ω̂(Y)=∩Ω(yjm)`을 구성(식21,22), fake xi를 제거해 하한 정확도↑.
8. 절차: Algorithm 1 — 각 `xi`를 `yj0=ymax`부터 `yjM`까지 복호 시도, 모두 실패(`m=M`)해야 `Ω̂(Y)`에 포함.
9. 결과: (3,61) array code 5-bit quasi-uniform 양자화 MSA에서 `Ω(ymax)∩Ω(yj3)∩Ω(yj6)`가 시뮬과 잘 맞는 하한 제공; 복잡도 `t^a` 프레임, Monte-Carlo 대비 고SNR에서 저렴.
10. 한계: HW 미설계, absorbing set 외부 code 구조 종속(code-dependent), 성능 개선이 아닌 error-floor 예측 도구, quasi-uniform 5bit 전제.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|

| 양자화/LLR 레벨 (quasi-uniform) | decoder.cpp Get_VNU_Table_Idx(), ecc_data.h PARAM_LLR | 양자화 레벨 설계 검증에 참고 |
| absorbing set error-floor 하한 | 대응 없음 | 분석/예측 도구, Prime ECC 코드 모듈 아님 |
| 부호구조(array code) multiplicity | ecc_top.cpp Load_PCM() | 부호 구조 종속, QC-LDPC로 재산정 필요 |
```

적용 가치: 중간 - Prime ECC의 min-sum·양자화 error-floor를 예측/검증하는 분석 도구로는 유용하나, 정정성능·HW를 직접 개선하지 않고 부호 구조에 종속되어 이식 대상은 아님.
