### B. 알고리즘 요약

1. 대상은 AWGN + 버스트 소거 채널의 binary QC-LDPC로, rate 0.82~0.95, length 7225~9261 예제(GF(2^8)/GF(2^6) 기반).
2. 문제: 랜덤에러와 (짧은 phased/긴 single) 버스트 소거를 하나의 QC-LDPC로 동시에 정정하고 싶다.
3. 기반: RS 부호의 conventional 패리티행렬 `BRS(d,n)`을 GF(2^s)의 순환부분군 `S` 원소로 구성, `2×2 SM-constraint` 만족 시 base matrix로 사용.
4. 핵심 정리(Theorem 1): base matrix의 모든 2×2 부분행렬이 zero 포함 or nonsingular이면 CPM-dispersion 후 Tanner graph girth≥6 보장(RC-constraint).
5. 구성: base matrix를 `k×k` CPM(circulant permutation matrix)으로 dispersion하여 binary QC-LDPC 패리티행렬 생성.
6. Globally-coupled(GC): 로컬 base matrix `BRS,1(d1,λ)`의 p개 복사본을 대각 배치 + 연결행렬 `BRS,2(d2,λp)`로 결합 -> type-1 `(λ,λp)`-construction.
7. type-2: 두 RS 행렬의 direct product로 구성, cascade+interleave 구조로 로컬 코드워드 1개 전체 소실도 복구(분산스토리지 지향).
8. 소거정정: 컬럼치환 `πcol`로 zero-span을 늘려 단일 버스트 소거 길이 최대 λp(p−1)+1까지 정정.
9. 복호: 전체 H 대신 로컬/글로벌 2단 반복복호(MSA), 높은 global connectivity `d2λ(p−1)`로 빠른 수렴 주장. 예제: (7225,6888) 코드가 BER 10^-9에서 Shannon limit 대비 1.3dB.
10. 한계: 시뮬(AWGN)만, HW·게이트카운트·throughput·양자화·NAND read 모델 전무, 다수 미결과제(효과적 2단 복호·trapping set 분석)를 향후과제로 남김.

### C. Prime ECC 관련 모듈 핀포인트

| globally-coupled 로컬/글로벌 구조 | 대응 없음 | Prime ECC는 단일 통합 PCM, GC 로컬/글로벌 분할 구조 없음 |

| 버스트 소거 정정(컬럼치환) | 대응 없음 | Prime ECC 채널은 AWGN/RBER/fixed-error, 소거 버스트 모델 없음 |
