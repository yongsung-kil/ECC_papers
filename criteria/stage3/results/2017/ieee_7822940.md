### B. 알고리즘 요약

1. 대상: binary QC-LDPC (base matrix + circulant), AWGN·BPSK, rate 0.5~0.87, length 1778~8192, 예: `(4080,3319)` rate 0.813, `(8192)` rate 0.87.
2. 문제: 대수부호(cyclic)는 인코딩 간단하나 성능 부족, LDPC는 성능 좋으나 인코딩 복잡 - 둘의 장점을 group ring 구성으로 동시 달성.
3. 모델: group ring `R[G]`와 `n×n` 행렬환의 동형사상(`M(RG,w)`)을 이용해 원소 `w` 하나로 PCM 생성.
4. 핵심기법1: 유한체 위 `α`-multiplied 제약(row 조건 1,2) 충족하는 exponent matrix `W`를 group ring 원소로 구성, CPM 치환해 girth>=6 QC-LDPC 획득 (Theorem 4).
5. 핵심식: Abelian group의 S2-set(모든 쌍합이 distinct) 사용 시 4-cycle-free 보장(Prop.11), non-cyclic Abelian group으로 lifting degree b 선택폭 57% 확대·고rate 달성.
6. 핵심기법2: QCPM = 여러 CPM의 Kronecker product(식11)로 non-cyclic Abelian group 지원, `W|W^-1` 배치로 rate 확장(Theorem 18).
7. 구현디테일: 인코더 회로 - n개 PMC, 각 b개 monomial multiplier(MM), RAM 기반(shift register 대신)·LUT, kb 클록에 인코딩, 나이브 구현 gate/RAM 수식 명시.
8. 최적화: group ring 곱셈을 FFT 컨볼루션으로 처리 - `O(nb log2 n · Σlog2(qi-1))` AND 게이트, GFT/ETD 인코딩보다 낮은 복잡도.
9. 결과: 유한체 기반 부호(C7·C10·C11 등)와 BER/WER 성능 동등(예 C4는 Shannon limit 1.32dB), 인코딩 시간복잡도 `O((log2 b)^2)` (기존 [18] `O(b^2)` 대비 대폭 감소).
10. 한계: HW 합성/gatecount/throughput 없음(수식 복잡도만), 부호 정정성능은 기존 대비 개선 아닌 동등, 순수 이론·시뮬.
