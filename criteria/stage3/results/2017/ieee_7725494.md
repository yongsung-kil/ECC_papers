### B. 알고리즘 요약

1. 시스템: rate-0.905 irregular (18396,16644) binary QC-LDPC(M=6,N=63,z=292,EB=246), RS-based LDPC에 masking 적용, 90nm CMOS 디코더.
2. 문제: 고rate·장부호 layered 디코더는 저장(APP/FIFO/C2V/V2C)이 실리콘 면적의 절반 이상 차지 -> area 병목.
3. 기법1(FIFO-free reorder): layer 연산을 재배열해 `Qi,j = λj - Ri,j^{k-1}` 갱신을 V2C 메모리에 누적, APP 메모리와 FIFO 버퍼 제거([26] 특허 기반).
4. 핵심식: NMSA `Ri,j = α·Si,j·min|Qi,j|`로 C2V 생성, reorder는 동일 메시지 생성 -> error-rate 불변.
5. 기법2(graph coloring): V2C sign 메모리 동시접근 3회 -> 최소 뱅크 수를 coloring으로 증명, Theorem 1(`gcd(a-b,3b)|b`)로 pipeline (a,b)=(2,2), 4-stage로 뱅크 3개 확정.
6. 기법3(ILP): V2C sign을 뱅크에 균등 분배(`3·Σwi,max - EB` stall 최소화), 스케줄과 뱅크배치 공동 설계로 stall 15->9->3 감소.
7. 트릭(modified MSA): 1st-min 미저장, 2nd-min>Tγ면 1st-min을 작은 γ로 근사(Tγ=γ=0.5, 5-bit step0.5) -> C2V 저장·MUX 대폭 감축(gated-clock).
8. 양자화: qAPP=6, qV2C=5, qC2V=5; float 대비 0.1 dB 열화.
9. 결과: gate 926K@90nm, 4.19mm², 8 iter에서 4.674 Gb/s@200MHz, 299mW; baseline 대비 gate 36.5%↓, TAR 43%↑, direct-reorder 대비 TAR 58.9%↑.
10. 한계: modified MSA는 수렴이 느려 iteration 부족(8) 시 BER 열화, error-rate 향상 목적 아님(area/throughput 최적화).
