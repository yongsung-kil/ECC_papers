### B. 알고리즘 요약

1. 부호: isolated shifted identity 기반 구조적 (3,6) regular QC-LDPC, `wc=3`, `wr=6`, rate `R=1−wc/wr=1/2`, 길이 222/546/642/648/1152 (permutation size `p`=37/91/107/108/192).
2. PCM: 3×6 블록 순환 배치(`H`, 식 1), 각 블록은 크기 `p` permutation matrix, 인접 두 행 간 길이-4 short cycle 없음.
3. 문제: Wi-Fi/Wi-Max/DVB-S2용 QC-LDPC를 낮은 iteration에서 고throughput으로 FPGA 구현.
4. 인코딩: Gauss-Jordan elimination으로 generator matrix 생성 후 AWGN 채널 전송.
5. 복호: min-sum soft-decision iterative message passing, 4-bit fixed-point LLR 양자화(sign 1/integer 2/fraction 1 bit).
6. 종료 조건: `c·H^T=0` 또는 지정 iteration 도달, 실측상 약 30 iteration에서 수렴.
7. HW 구조: 완전병렬(fully parallel) — VN/CN 각 1 clock, 30 iteration에 `30×2=60` clock, Input/Output network(pipeline register)+VN-CN permutation network+Control unit.
8. Input network folding factor로 FPGA 핀 수에 맞춰 `n/Foldfactor`씩 입력.
9. 결과: 모든 길이에서 BER `10^-5` @ ~4 dB coding gain, 완전병렬로 최대 throughput 2 Gbps(1152), Virtex-5 slice/LUT 자원표 제시, 648 부호는 기존 [10] 대비 coding gain 1 dB 우수.
10. 한계: waterfall만 보고 error-floor 없음이라 정성 서술, gate count(공정) 없음, rate-1/2 통신 부호, min-sum은 표준 기법.
