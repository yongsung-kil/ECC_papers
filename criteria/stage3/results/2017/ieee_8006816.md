### B. 알고리즘 요약

1. 대상은 block protograph를 무한 chain으로 coupling한 뒤 lifting하는 QC spatially-coupled LDPC(SC-LDPC)로, base matrix `B=1_{nc×nv}`(예: 3×6, 3×8)에서 coupling width `ω`로 edge-spreading해 SC base matrix `B_SC`(식3, 대각 밴드 구조)를 만든다.
2. 문제: coupling·lifting 2단계에서 4-cycle이 생겨 short cycle이 error-floor를 악화시키므로, girth를 체계적으로 키우는 설계가 필요하다.
3. 핵심 가정/근거(Lemma1): lifted graph의 girth는 base graph(protograph)의 girth로 하한되므로, 먼저 protograph 단계에서 4-cycle을 없애면 lifting 후에도 girth가 유지된다.
4. 핵심 기법: `B_SC`를 representative block `B_R`, constituent block `B_C`, excluded patterns `B_E(j)`로 분해하고(식4~6), Theorem3에 따라 `B_C`와 `B_E(j)`에 4-cycle이 없으면 전체 coupled protograph에 4-cycle이 없음을 보인다.
5. 이 분해 덕분에 전체 `B_SC`가 아닌 `B_C`·`B_E(j)`만 검사하면 되어 4-cycle 판정 범위가 축소된다.
6. 2단계 설계: Stage 1(Design Rule 1)은 `B_0=[I Ξ]`, `B_ω`(min row weight 2, max col weight 1) 등으로 submatrix를 초기화해 excluded pattern의 4-cycle을 제거, Stage 2(Design Rule 2)는 `B_C`에 남은 4-cycle을 entry flip-down/flip-up(고weight submatrix 우선)으로 제거한다.
7. QC lifting: 설계된 `B_SC`를 circulant `I_M^θ`로 M-fold lifting하며, 6-cycle(3×3 grid) 제거를 위해 Fossorier 조건 `(θ1-θ7)=(θ2-θ5)+(θ6-θ9) mod M`(식12)으로 shift factor를 선택, 충분히 큰 M에서 girth>=8 QC-SC-LDPC 획득.
8. 학습/최적화: 없음(조합적·규칙 기반 설계, GA/NN 미사용).
9. 결과: (3,6) ω=3 L=30 R=0.45, (3,8) ω=4 R=0.575, AWGN/BPSK, sliding window 복호(W별, window당 max 100 iter; FSD는 max 1000 iter). 설계 코드가 undesigned 대비 특히 error-floor에서 큰 이득, QC lifting(g=8)이 g=6보다 추가 개선. W=10처럼 window가 너무 작으면 SW 복호 이득 제한.
10. 한계: HW 미설계·AWGN 시뮬만, block-length·gate count 등 구현 수치 미제시, 저rate(0.45~0.575) SC 구조 특화.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| QC-SC-LDPC protograph / edge-spreading 부호설계 | `ecc_top.cpp` `Load_PCM()` + H-matrix | Prime ECC는 block QC-LDPC 고정, SC(convolutional) 구조와 상이 |
| circulant lifting / Fossorier girth 조건 | 대응 없음 | Prime ECC의 base+shift(`PCM_b`) lifting에 girth 조건 참고 가능하나 논문은 SC 밴드행렬 전용 |
| sliding window 복호(SW) | 대응 없음 | Prime ECC는 block flooding/dual-update 복호, SW 스케줄 없음 |

적용 가치: **낮음** - spatially-coupled(LDPC convolutional)+sliding window 복호 전용 부호설계라 block 기반 고rate QC-LDPC(Prime ECC)와 부호구조·복호 스케줄이 근본적으로 달라 직접 이식 대상이 아니며, girth>=8·4-cycle 제거 원리 정도만 개념적 참고 가치가 있다.
