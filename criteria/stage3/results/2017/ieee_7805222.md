### B. 알고리즘 요약

1. 시스템: BSC(파라미터 α) 상 regular LDPC, Gallager B 디코더 최대 `L` iteration, flooded scheduling.
2. 문제: 저전력·고클럭 회로에서 발생하는 timing error(게이트가 clock edge 전 스위칭 실패)가 디코더 성능에 미치는 영향을 이론적으로 규명.
3. 모델: timing error는 `z(l) = D(l)·y(l) ⊕ D̄(l)·y(l-1)` — 확률 ε로 이전 iteration 출력이 유지되는 memory 있는 오류(Amaricai의 SPICE 기반 모델).
4. 핵심기법: error-free 디코더와 timing-error 디코더의 computation tree를 비교. tree inclusion 관계 `N1 ⊆ N2` 정의.
5. 핵심식(Th.1): `N_e(l+1) ⊆ Ñ_e(3l) ⊆ N_e(3l)` — timing error 디코더의 computation tree가 error-free tree보다 작지만 하한을 가짐.
6. 확장(Th.2): `Pe(l)`가 극한을 가지면 `P̃e(+∞) = Pe(+∞)` — 무한 iteration에서 오류확률이 error-free와 동일하게 수렴.
7. 구현 디테일: 첫 iteration은 error-free로 가정(이전 codeword 의존성 제거), εMAJ=ε⊕=ε로 단순화.
8. 최적화: 없음(순수 해석적 증명, density evolution으로 Pe(l) 산출).
9. 결과: (3,5)/(3,6)/(4,16)/(4,8) regular code에서 100 iteration Monte Carlo로 ε=0과 ε=0.2 BER가 사실상 동일(error floor에서만 미세 차이).
10. 한계: HW 미설계, cycle-free tree 가정, Gallager B(hard-decision) 전용, cycle 포함 그래프 예측은 future work.
