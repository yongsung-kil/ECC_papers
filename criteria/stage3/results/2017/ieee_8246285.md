### B. 알고리즘 요약

1. 대상: BSC 상 binary LDPC의 hard-decision 반복복호(Gallager B message-passing, Gradient Descent Bit-Flipping), 저장/암호 시스템 지향.
2. 문제: 유한길이 부호의 trapping set이 완벽한(잡음 없는) 디코더의 error-floor를 유발.
3. 핵심 관찰: 논리게이트를 의도적으로 unreliable하게 만들면 stochastic resonance로 trapping set을 오히려 깨뜨려 성능 향상(추가 HW 비용 없음).
4. 노이즈 모델: XOR/MAJ 게이트 출력에 Bernoulli 잡음 `e_XOR~α_⊕`, `e_MAJ~α_MAJ` 주입, 단 최종 판정·syndrome checker 게이트는 완벽하다고 가정.
5. GDBF 기준함수: `Λ_v = (ν_v⊕y_v) + Σ_{c∈Nv} μ_c`, 최대값 `b=max Λ_v` 인 노드를 flip.
6. 해석: 복호 동역학을 absorbing Markov chain으로 모델링, 상태 = check→variable 메시지 벡터, 전이확률은 Hamming distance·α_⊕·α_MAJ·L 종속.
7. error-floor 원리: elementary trapping set(check degree≤2)은 α_⊕>0이면 소멸 가능, L→∞에서 `FER_e(∞)=MER_e(∞)`.
8. 최적화: 노이즈 레벨을 부위별로 조절(trapping set 관여 노드엔 잡음↑, 나머지↓)하는 것이 본질적으로 deep-learning 문제(향후과제), GDBF는 rewinding(20 iter마다 재시작)으로 추가 개선.
9. 결과: QC Tanner (155,64) 부호에서 α_MAJ 조절로 noiseless 대비 FER 개선; FER=1e-5·α_M=1e-2에서 noisy Gallager B는 L=1000, noisy GDBF는 L=100 필요(GDBF가 latency 유리).
10. 한계: 서베이/이론+시뮬, HW 미설계, hard-decision 전용, latency 대폭 증가, 전용 unreliable-gate 하드웨어 전제.
