### B. 알고리즘 요약

1. 대상은 (3,K) regular binary QC-LDPC의 exponent matrix `E(H)` 구성 (CPM 크기 `P`, block-column 수 `K`).
2. 문제: 기존 구성법은 girth 4 회피만 보장하거나 특정 rate/length/girth에 제약이 있어 유연성이 낮음.
3. 목표: girth 8/10/12를 보장하면서 CPM 크기 `P`를 최소화하고 계산시간을 줄이는 것.
4. 핵심 규칙: 1행·1열은 0 벡터 고정, 2행은 오름차순, 인덱스 중복 금지.
5. 2행 생성: 인덱스마다 1부터 `P`까지 값을 대입하며 부분행렬 girth가 목표치가 될 때까지 탐욕 탐색(greedy search).
6. 3행 생성: 완성된 2행 값에 +1을 더해 (`(2,K)+1`) 대응 인덱스로 사용 → 별도 탐색 없이 즉시 결정.
7. 재귀 확장: `(2,5)` base matrix를 시작으로 `(2,6)`, `(2,7)`... 이전 행렬을 base로 재사용해 마지막 원소만 새로 탐색.
8. 사이클 회피 조건: shifting chain 합 `Σ(a_ik,jk − a_ik+1,jk) ≢ 0 mod L` (식 3)로 짧은 사이클 제거.
9. 결과: girth 8에서 K=5~12의 CPM 크기가 기준 [9] 대비 절반 이하(예 K=9: 17 vs 30), girth 10도 개선.
10. 한계: BER/디코딩 성능·FER·HW 구현 전무, girth와 CPM 크기 비교만. 타깃은 Wi-Fi/Wi-Max, NAND 무관.
