### B. 알고리즘 요약

1. 대상은 6-bit 양자화 partially-parallel QC-LDPC decoder(VNU/CNU/interleaver, VRAM/CRAM/LRAM), (4,32) regular 및 (8176,7154) CCSDS 코드.
2. 문제: 우주 방사선 환경에서 decoder 내부 message RAM의 bit-flip(unreliable memory)이 복호 성능을 크게 열화시킴.
3. 모델: 메시지 저장 오류를 BSC(cross-over `α`)로 모델링. 채널 메시지 `m0`는 read-only라 iteration 누적으로 `α_r(n)=(1-(1-2α0)^n)/2`, 내부 메시지는 `α_v=α_c=α0`.
4. 핵심 기법 1: 이산 density evolution으로 각 양자화 bit 위치별 오류가 decoding threshold에 주는 영향 분석. PMF 벡터 `P`와 전이행렬 `Π(E)` 사용.
5. 핵심 관찰: 오직 sign bit 오류만 threshold를 크게 좌우하고, magnitude bit 보호는 이득이 미미함 (식 (6)~(14)).
6. 핵심 기법 2: partially-parallel 구조로 동시에 나오는 sign bit들을 block으로 묶어 embedded 코드로 보호 (UEP - unequal error protection).
7. 부수 트릭: embedded 코드로 modified Hamming product code `(48,16)` 채택 - row/column subcode 모두 (8,4) Hamming, byte 정렬.
8. 복호: Hamming product code의 iterative decoding(2 iter) + fixed/erasure bit 상태 정의, 3rd-order error pattern용 enhanced decision logic 추가.
9. 결과: memory error ratio 0.001에서 성능 열화 0.1 dB, TMR 대비 0.1 dB(0.001)~0.4 dB(0.004) 이득. (8176,7154), 50 iteration 시뮬.
10. 한계: HW 미설계(gate/throughput 없음), 방사선 memory-fault 특화, magnitude 오류는 무시.
