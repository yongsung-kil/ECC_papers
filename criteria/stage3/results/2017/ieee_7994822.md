### B. 알고리즘 요약

1. 문제: irregular LDPC 설계에서 변수노드 λ(x)·검사노드 ρ(x) degree분포를 동시(joint) 최적화하면 조합폭발로 구현 난해 - 좋은 부호 특성(저복잡도·고rate·고threshold·안정성·낮은 gap) 동시달성 어려움.
2. 채널/모델: Binary Erasure Channel(BEC), 용량 `C=1-ε`, degree분포쌍 `(λ(x),ρ(x))`로 부호 앙상블 특성화.
3. 좋은 부호 기준: edge 수↓, degree-2 비율 `λ2`↓(short cycle/stopping set 억제→error floor↓), rate `r(λ,ρ)`↑, threshold `εBP`↑, capacity 대비 multiplicative gap `δ`↓.
4. 핵심기법: GA로 `λi` population 최적화, 동시에 LP로 각 population의 `ρ(x)`(degree-2 모델, `ravg` 조정)를 탐색 → 순위매겨 상위 선택·반복(iteration `M`).
5. 최적화 제약식: threshold ≥ε 위해 `f(ε,x)-x≤0`, `f1`이 `λi`의 선형함수 → LP로 rate 최대화 `max Σλi/i`.
6. 수렴조건: `c(x)<vε⁻¹(x)`, `εBP`는 이를 만족하는 ε의 상한(EXIT chart로 계산).
7. 결과(Case1, ρ=x^5, lmax=8): GA-LP가 threshold 0.48339, rate 0.50053, gap 0.03112로 LP(gap 0.0359)·GA(gap 0.036)보다 우수.
8. 결과(Case2, lmax=13): GA-LP threshold 0.49277, gap 0.024553로 개선.
9. FER(bit-flipping 복호): GA-LP가 higher εBP로 waterfall 소폭 우수, 낮은 λ2로 error floor도 소폭 개선; 단 ρ(x) 모델 복잡화로 실행시간↑(MATLAB tic/toc 5.43→239초).
10. 한계: BEC 전용, irregular(비QC), HW 미설계, bit-flipping/시뮬만, 부호길이 n=2000 예제.
