### B. 알고리즘 요약

1. 대상: `F_q` 상 M-level rate-compatible 부호 `C1≺C2≺…≺CM` (고rate부호가 저rate부호의 앞부분, 식 1), q-ary symmetric channel(crossover `p`), BSC 예시. NAND 직접 언급 없음, Seagate 공저자.
2. 문제: 기존 rate-compatible은 puncturing(패턴 탐색 난해) 또는 extending(Construction X/XX)에 의존, 이론적 보증이 부족.
3. 기여1(하한): GV-like lower bound(Thm 2, Cor 3) 및 weight-enumerator 기반 2-level bound(Thm 4)로 좋은 rate-compatible 부호 존재성 증명.
4. 핵심 기법(Construction 1): base code `C1`을 nested component code들의 syndrome `si=c1 H^T_{C1i\C1i-1}`을 순차 생성·인코딩해 확장 → "syndrome-coupled" 구성.
5. 부호 파라미터(Thm 5): `Ci` 길이 `Ni=Σnj`, 차원 `Ki=k`, 보조부호 min-dist 조건 만족 시 `Ci` 최소거리 `Di=di`, `di/2` 미만 오류 정정 디코더 존재.
6. 예시: extended Hamming 등으로 `C1=[15,11,3]≺C2=[20,11,5]≺C3=[26,11,7]` (C1,C2 optimal), nested MDS/BCH 예시도 제시.
7. 복호(Algorithm 1): `yM→y1` 순으로 각 sub-block을 component error-erasure decoder `DC1i`/`DAij`(bounded-distance)로 정정, `2ti+τi ≤ δiM-1` 조건 만족 시 성공(Thm 6).
8. 기여2(capacity): degraded q-ary symmetric 채널열에 대해 nested capacity-achieving component(예: polar)를 쓰면 각 `Ci`가 해당 채널에서 `Pe→0` (Thm 9), [5]의 polar 구성과 등가임을 syndrome↔frozen bit 매핑으로 증명.
9. 결과(수치): BCH 예 `C1=[8191,7671]≺C2=[8550,7671]`(rate 0.8972), BSC p=0.0035에서 `Pe≈1.049e-7`, shortened BCH 대비 근접(약간 열위).
10. 한계: 순수 이론(HW/throughput/gatecount 없음), min-dist·hard-decision bounded-distance 관점, LDPC soft 반복복호 성능은 미다룸.
