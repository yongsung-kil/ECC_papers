### B. 알고리즘 요약

1. 대상: circulant permutation matrix(CPM) 기반 binary QC-LDPC, rate 1/2, IEEE 802.16 12x24 모수행렬 사례.
2. 문제: 하나의 최대길이 QC-LDPC에서 여러 짧은 길이 부호를 파생(length adaptation)할 때 기존 floor/modulo lifting은 harmful short cycle 제거가 부족.
3. 핵심 정의: exponent 행렬 `E(H)`, cycle 조건 `Σ(-1)^i·a_i ≡ 0 mod L`, trapping set `TS(a,b)`와 EMD/ACE(`ACE=Σ(d(v)-2)`)로 유해도 평가.
4. 핵심 기법: floor-scale-modulo lifting `E_ij(H_k) = floor( (L_k/L_0)·((r·E_ij(H_0)) mod L_0) )`, 특수 파라미터 `r`(scale value)로 CPM 값 선택 cardinality 증가.
5. 식 의미: `r` 도입으로 최단 cycle 수를 줄이는 방향으로 lifting 결과를 조정 가능 — 메모리엔 상수 `r` 하나만 저장.
6. 부호 설계 절차: (1) PEXIT-chart + covariance evolution으로 mother matrix 선택, (2) 각 circulant size L_k마다 simulated annealing으로 r 탐색해 ACE=EMD인 cycle 최소화.
7. 코드 sieving: FER<10^-4 목표 시 pseudocodeword weighting + Smarandache-Vontobel/Butler-Siegel 코드거리 상한으로 후보 재정렬 후 시뮬 선별.
8. 학습/최적화: simulated annealing이 Hill Climbing 대비 girth 10 달성에 더 작은 circulant L 필요(예 n=6에서 97 vs 103).
9. 결과: layered NOMS(15 iter) 디코더 하에서 정보길이 2000 미만 구간 5G 제안 부호 대비 약 0.1dB coding gain(block error 10^-2).
10. 한계: HW 설계 없음, rate 1/2·짧은 길이 예시만, 실측/합성 없음, hard/soft read 논의 없음(디코더는 min-sum류로 무관).
