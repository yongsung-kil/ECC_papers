### B. 알고리즘 요약

1. 대상은 (3,6) regular LDPC(예제 length-10, rate 0.4), AWGN 통신채널용 소프트판정 디코더.
2. 문제: HDL 코드로 check/variable node를 직접 구현하는 전통 방식이 복잡하고 시간소모적 → 설계 공수 절감이 목표.
3. 채널/모델: 표준 LLR 입력 message-passing, 별도 신규 채널모델 없음.
4. 핵심기법: SPA의 비선형 check node 연산을 Min-Sum으로 근사 — CN은 `Ck = Sign(Vl) · min|Vl|` (l≠k), VN은 수신 LLR 합산 `Vi = LLRn + ΣCj`.
5. 식 의미: SPA의 tanh 계열 고정밀 비선형을 min 근사로 대체해 HW 계산복잡도를 낮춤(표준 min-sum 동기).
6. 확장: 신규 알고리즘 확장 없음. 대신 설계 자동화(Simulink→HDL)에 초점.
7. 구현 디테일: (3,6) CN 6입력을 Fin3min으로 3세트(min1234/min1256/min3456) 나눠 min 계산, fin3min 출력 재사용으로 find-minimum 하드웨어 자원 절감.
8. 학습/최적화 절차 없음 — Simulink 라이브러리 기본 블록으로 CN/VN 모델링 후 Simulink HDL Coder로 VHDL 자동생성.
9. 결과: FPGA device utilization만 보고 — CN 96 slice LUT(1%), VN 16 slice register + 43 LUT(1%). BER/throughput 수치 없음.
10. 한계: 표준 Min-Sum + toy 부호, BER 성능·정정능력 결과 없음, NAND 무관, 방법론(설계흐름) 논문.
