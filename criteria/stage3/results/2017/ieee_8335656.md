### B. 알고리즘 요약

1. 시스템: 2-user, symbol-synchronous, binary-input Gaussian MAC(`Y = X1 + X2 + Z`), blocklength `N=10^4`, symmetric capacity 접근이 목표.
2. 문제: time-sharing/rate-splitting 없이 두 LDPC 코드워드의 잡음 합을 공동복호(joint decoding)해 대칭 용량에 근접하려 함.
3. 모델: 이진 입력 `x_k(0)/x_k(1)`에 전력제약 `x_k(0)²+x_k(1)² ≤ 2`, 채널출력 pdf는 4개 가우시안 혼합.
4. 핵심기법1: 고전 BP에 **MAC node**를 추가한 수정 Tanner graph로 두 코드워드를 동시 복호(joint variable node degree distribution `Λ(χ1,χ2)`).
5. 식 의미: MAC-to-variable 메시지 `m_mv`(식12)가 상대 사용자 정보를 반영해 두 코드가 서로의 복호를 보조.
6. 핵심기법2: 메시지를 가우시안 근사(GA)하고 상호정보 transfer function `J(·)`로 DE 수렴조건(식16~20)을 유도.
7. 구현 디테일: check node degree는 단일 고정, variable node는 최대 degree 200까지, PCM은 PEG 알고리즘으로 구성.
8. 최적화: GA-DE 수렴조건을 제약으로 하는 선형계획(LP, CVX)으로 대칭 rate `Rsym`(식19) 최대화 → `Λ` 대각 구조가 최적.
9. 결과: 이론성능 대비 대부분 SNR에서 1dB 이내; 저SNR=antipodal, 고SNR=max-min-distance, 중SNR=비대칭 성상이 최적(놀라운 결과). 4-PAM superposition으로 rate 확장.
10. 한계: 순수 시뮬(HW 미설계), 다중접속 채널 전용, 고차 성상 확장은 antipodal에만 국한.
