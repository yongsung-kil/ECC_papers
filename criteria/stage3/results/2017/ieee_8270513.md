### B. 알고리즘 요약

1. 대상: FG-LDPC(outer)와 FF-QC-LDPC(inner)를 concatenate한 globally-coupled LDPC, AWGN/BEC. 예시 최종 코드 `(8000,5600)` rate 0.7, 플래시메모리·광통신 응용 언급.
2. 문제: 일반 LDPC는 iterative 복호 시 small trapping set으로 인한 error-floor가 심각. FF-QC-LDPC는 유연하나 작은 column weight면 small trapping set 존재.
3. 핵심 관찰: FG-LDPC는 최소거리 크고 `d_min-3`보다 작은 trapping set이 없음(예 (1057,813) PG-LDPC, `d_min=34`, trapping set 크기 <31 없음) - error-floor-free.
4. 기법 1(inner 구성): GF(q) 위 base matrix `B(m,n)`(2x2 SM-constraint)를 CPM-dispersion해 RC-constraint QC-FF-LDPC 생성. 예 GF(101), `(6,80)`-regular `(8000,7400)` rate 0.925.
5. 핵심 식: `B(m,n)=[α^{i_k}+ηα^{j_l}]`의 2x2 non-singular 조건이 girth>=6(RC-constraint)을 보장.
6. 기법 2(concatenation): outer FG codeword c개를 cascade(type-1) 또는 interleave(type-2)한 뒤 inner FF로 부호화 → CN-based globally-coupled 구조. inner의 r global CN이 c개 local FG 그래프를 연결.
7. 복호: 2-phase serial-turbo iterative(inner/outer 또는 outer/inner), MSA scaling factor inner 0.275 / outer 0.65, syndrome 기반 전환. outer FG의 큰 `d_min` 덕에 크기 `d_0-3` 이하 trapping set은 outer가 해제.
8. 부수: 대형 column weight FG는 RID(revolving iterative decoding) 서브매트릭스(l=244, 심지어 l=1)로 HW 복잡도 약 5배 절감. inner PCM masking으로 column weight 6→4, 0.1 dB 이득.
9. 결과: `(8000,5600)` type-1, `Iin=Iout=Iin/out=5`에서 BER 10^-15 threshold 약 3 dB, NCG 약 12 dB. type-2는 burst erasure(길이 231까지) 정정.
10. 한계: HW 미설계(gate/throughput 없음), 2-phase turbo 복호라 latency·복잡도 증가, underwater fiber-optic(10^-15 BER) 타깃.
