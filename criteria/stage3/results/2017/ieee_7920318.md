### B. 알고리즘 요약

1. 대상: multilevel NAND Flash(SLC/MLC/TLC) ECC 전반 서베이. NVM 채널의 비대칭(retention은 하향, overshoot/ICI는 상향 오류)을 핵심 화두로 삼음.
2. 문제: 기존 ECC는 대칭·Hamming metric 최적화라 modern NVM의 비대칭 구조 오류에 grossly inefficient.
3. 알제브라 코드: BCH/RS(Berlekamp-Massey + Chien search), rate-compatibility로 wear-out 대응. Chien search 병렬화가 병목.
4. 비대칭 magnitude-1 오류용: 하위 bit만 BCH 보호 후 error 위치에서 level -1 복원(gray-map 대비 parity 절감), NCC(nonconsecutive constraint)로 rate/정정 tradeoff 제어(rate 0.77 예시).
5. Tensor-Product Code(TPC): `H=[A⊗B]`(nonbinary A=cell, binary B=bit), 4-구성 확장 `[A C ⊗ B D]`로 cell당 bit오류 세분 제어, TLC 실측 데이터에서 lifetime ≥40%↑ (length 4096, rate 0.86).
6. LDPC/graph 코드: sparse PCM bipartite graph + message-passing(LLR). 유한길이 error-floor는 trapping/absorbing set 때문. NVM은 채널 비대칭이라 AWGN용 absorbing-set 제거는 무용.
7. channel-aware QC-LDPC 최적화: circulant 순서 + edge label 지정으로 Flash-detrimental absorbing set((a,b), 4≤a≤7, 1≤b≤4) 제거 → nonbinary GF(4), rate 0.9, length 4000, VN degree 3에서 order-of-magnitude 개선.
8. nonbinary multibit: q-ary partial-erasure channel 정의, density evolution 확장으로 degree distribution·edge label 최적화(예: ε1≫ε2면 label을 {1,2}만 사용), stopping-set 제거. q-ary가 3배 긴 binary보다 우수.
9. rewrite 코드(WOM): q-ary WOM `(q,n,t,M)`으로 in-place update·write amplification 절감(짧은 inner WOM + long outer ECC 권장), 예 `(8,2,4,8)` 최적 코드. 그 외 constrained coding·rank modulation 소개.
10. 한계: 서베이라 HW 미설계, 상당수 기법이 이론 단계, LDPC 신기법은 nonbinary 중심이며 결과는 시뮬/실측 인용에 그침.
