> 총평: MLC 비대칭 채널을 MI 기반 EXIT로 반영해 error-floor·waterfall 모두 개선한 NAND 직결 부호설계로 가치가 높으나, 부호설계 이식은 H-matrix 재검증 부담이 커 이식성은 중.

### B. 알고리즘 요약

1. 대상은 MLC(2bit/cell) NAND flash, 4개 임계전압 상태 Gaussian 모델(`μ11=0,μ10=3.25,μ00=4.55,μ01=6.5`, `δ11=2δ01=4δ10=4δ00`), 3bit 양자화 soft read.
2. 문제: AWGN 최적 LDPC는 flash 채널의 비대칭성 때문에 최적이 아니고, 양자화 LLR은 Gaussian 근사가 깨져 통상 EXIT chart 직접 적용 불가.
3. 모델: sense-amp 다중 read로 얻은 양자화 신호를 입출력 상호정보(MI)로 특성화(`Eq.3`), Monte Carlo 히스토그램으로 `p(ξ|x)` 추정.
4. 핵심기법: 측정한 MI로부터 등가분산 `δ²mlc`를 역산해 VND EXIT 함수 `IEdvi,VND = J(√((div-1)[J⁻¹(IA)]²+δ²mlc))`(`Eq.4`)에 주입.
5. 의미: 비Gaussian 양자화 채널을 MI 등가 분산으로 치환해 curve-fitting EXIT의 강건성을 활용, 채널 비대칭을 degree 최적화에 반영.
6. 확장: 선형계획(`Eq.5`)으로 VN degree sequence `λ` 최적화 후 QC 제약 부여한 PEG로 protograph base matrix 구성(Algorithm 1), nested protograph로 rate `(3n+1)/(3n+4)` rate-compatible family 생성.
7. 구현: protograph 기반 QC 구조로 저복잡도 encoder + 병렬화 가능 decoder, degree-2 VN 비율을 줄여 error-floor 억제.
8. 분석: asymptotic weight distribution으로 최소거리 선형증가 검증, 전형 최소거리비 `ζmin=0.026`.
9. 결과: N=4032,R=0.8095에서 EG-QC LDPC 대비 BER 10⁻⁶에서 0.7dB gain, 비구조 최적 random 코드와 유사 성능이면서 저장공간 절감.
10. 한계: HW 미구현(시뮬만), 단순화된 MLC 채널모델 가정, throughput/gatecount 수치 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| protograph/QC H-matrix 설계·로드 | `ecc_top.cpp` `Load_PCM()` | 부호구조 대응, 다만 Prime ECC는 특정 QC-LDPC 고정이라 재설계 부담 큼 |
| QC 인코더(dual-diagonal 관련) | `encoder.cpp` `Generate_PCM_encoding()`, `LDPC_encoder_4KB()` | 저복잡도 QC encoder 개념 대응 |
| 양자화 LLR / MMI soft-read 설계 | `channel.cpp` `Set_LLR_Th()`, `Set_R_Offset()` | MLC MI 기반 3bit 양자화가 우리 LLR threshold 설계와 직접 맞닿음 |
| iteration별 LLR 파라미터 | `ecc_data.h` `PARAM_LLR` | degree/채널별 LLR 최적화 사상 참고 |
| degree distribution·EXIT 최적화 | 대응 없음 | 오프라인 부호설계 절차로 코드베이스 직접 대응 모듈 없음 |

적용 가치: **중간** — NAND MLC 채널의 MI 기반 LLR 모델링과 3bit soft-read 설계는 `channel.cpp` LLR threshold에 직접 참고할 가치가 있으나, protograph 부호설계 자체는 Prime ECC의 고정 QC-LDPC 재검증 부담이 커 부분 채택.
