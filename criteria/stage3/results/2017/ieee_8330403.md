### B. 알고리즘 요약

1. 시스템: BPSK 변조 + 비상관 flat Rayleigh fading 채널(`yj = αj·xj + nj`), rate `R=1/2` protograph LDPC, 정보길이 `K=1024`(→ length 약 2048).
2. 문제: AR4JA 등 기존 protograph 부호는 AWGN엔 좋으나 Rayleigh 고SNR(error-floor) 영역 성능이 부족.
3. 채널 모델: `αj` 정규화 Rayleigh 페이딩(`E(α²)=1`), `f(α)=2α·exp(−α²)`, AWGN `σ²=N0/2` 가정, SISO BP 복호.
4. 핵심기법1: base matrix `B`(3×5, 원소 ≤3, edge 수를 AR4JA와 동일 고정)에 대해 컴퓨터 탐색으로 IARA1/IARA2 protograph 구성.
5. 핵심식: 고SNR 성능은 TMDR(typical minimum distance ratio)이 지배 → TMDR이 클수록 error-floor 낮음.
6. 확장: threshold 차이를 AR4JA 대비 ≤0.1dB로 제약해 waterfall은 유지하면서 TMDR만 키우는 절충.
7. 구현 디테일: degree-2 VN 비율을 적정(40%)으로 두고 punctured 고차 VN 1개 유지로 linear-min-distance-growth 확보.
8. 최적화 절차: 수정 PEXIT(threshold) + AWD 함수(TMDR 추정) 결합 EXIT-AWD 컴퓨터 탐색.
9. 결과: TMDR AR4JA 0.0144 → IARA1 0.0153 / IARA2 0.0157; Rayleigh BER=2×10⁻⁶에서 AR4JA 대비 약 0.4dB 이득, error-floor 미출현.
10. 한계: HW 미설계·시뮬만, Rayleigh 무선 특화, rate-1/2 저rate·비QC protograph.
