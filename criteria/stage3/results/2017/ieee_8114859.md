### B. 알고리즘 요약

1. 시스템은 광통신용 16QAM MLC-PID(multi-level coding, parallel independent decoding), 각 서브채널마다 독립 QC-LDPC 인코더/디코더, 총 code length 16200 bit, 총 rate 3/4.
2. 문제: 기존 regular QC-LDPC rate-adaptive는 mother code array 크기 확장 한계로 고정 rate만 가능해 서브채널 용량 초과·성능 저하 발생.
3. 모델: gray mapping 16QAM에서 GMI 용량이 MI 용량에 근접, 서브채널 용량 `Ci=I(bi;Y)` 기준으로 rate 배정.
4. 핵심 기법: 서브채널 용량 곡선에서 33.33% overhead 하 최적 rate 배정(b1/b3=5/6, b2/b4=2/3) 도출, UEP(4/5,7/10)보다 용량 초과 없음.
5. 식 의미: rate `R=1-J/L`을 정확히 맞추려면 array 폭 `L`을 6,5,10,3의 LCM=30으로 잡아 모든 목표 rate를 shortening으로 실현.
6. 확장: mother code `(J,L,p)=(10,30,540)` (circulant 크기 540, 길이 16200)를 finite-geometry local optimal heuristic 탐색으로 girth/cycle 최적화.
7. 구현 디테일: EXIT 차트로 열가중치분포 `λ(x)` 최적화(최소 열가중치 3, 열가중치 2는 error-floor 유발로 배제), random masking matrix로 irregular 구조 생성.
8. 최적화: 각 rate별 수렴 threshold와 Shannon limit 격차 0.45~0.5dB 수준 확인.
9. 결과: 제안 MLC-PID가 UEP 대비 BER 10^-8에서 0.1dB coding gain 향상.
10. 한계: 시뮬(BPSK/16QAM AWGN, BP-LLR, max iter 20)뿐, HW 미설계, 광통신 특화 rate/mapping 가정.

적용 가치: **낮음** — 광통신 16QAM MLCM용 유연 rate irregular QC-LDPC 구성법이라 NAND 고정 고rate binary QC-LDPC(min-sum) 기반 Prime ECC에 이식할 요소가 없음.
