### B. 알고리즘 요약

1. 시스템: 2x2 MIMO i.i.d. Rayleigh 페이딩 채널 + BICM/BICM-ID, 32APSK Gray 매핑, 스펙트럼효율 4.83 bits/ch.
2. 문제: 기존 QC-LDPC 기반 MIMO는 independent demapping(BICM) 최적화 시 iterative demapping(BICM-ID)에서 성능 저하, 반대도 마찬가지 - 두 방식 동시 만족 어려움.
3. 접근: SC-LDPC의 universality(BP 복호가 uncoupled MAP 성능에 도달, 동일 용량 대칭채널 그룹에 공통 최적)를 활용해 BICM/BICM-ID 트레이드오프 해소.
4. 핵심기법: base matrix `B`를 spatial coupling 형태로 구성(smoothing parameter `w`, coupling chain length `L`), 각 `m×n` 서브행렬 `Bi` 엔트리는 {0,1}.
5. 리프팅: `B`의 0을 `b×b` 영행렬, 비영을 `b×b` 순환치환행렬로 치환 → 부호길이 `Lnb`, 패리티길이 `(L+w-1)mb`.
6. rate loss 대응: 유한 `L`로 rate가 `1-(L+w-1)m/(Ln)`로 감소 → `B`의 마지막 몇 행 제거로 rate 회복.
7. 최적화 도구: MET-DE(multi-edge type density evolution)로 uncoupled component base matrix `Bunc`를 결합 최적화(서로 다른 degree 노드에 연결된 edge를 type별로 구분 추적).
8. 설계 예: `L=60`, `w=7`, `Bunc`(5x3), 마지막 4행 제거로 rate=58/120≈0.483, 순환행렬 크기 512x512 → 부호길이 61440.
9. 결과: BER=1e-5에서 MIMO BICM gap 1.48dB, BICM-ID gap 1.52dB(제약용량 대비), 두 방식 gap차 0.04dB로 단일 부호가 양쪽 동시 근접, interleaver 불필요.
10. 한계: HW 미설계, 시뮬만, 저rate·무선 MIMO 방송 특화, MAP demapping 전제.
