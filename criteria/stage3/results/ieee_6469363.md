### B. 알고리즘 요약

1. 시스템: 100Gbit/s 광통신 장거리 전송에서 coherent detection과 phase-slip 대응용 differential modulation(BCJR 기반 turbo detection)을 모두 지원해야 하는 hybrid 시스템, irregular LDPC 부호 사용.
2. 문제: coherent 검출과 differential 검출 각각에 최적인 LDPC 부호가 서로 다른데, 운영자는 채널 조건에 따라 FEC 코드 변경 없이 두 검출 방식 사이를 전환하고 싶어함 — 동일 코드로 양쪽 모두 우수해야 함.
3. 채널/모델: AWGN 채널, differential modulation은 BCJR 알고리즘으로 연판정 복호 후 LDPC와 turbo 반복(extrinsic information 교환); coherent는 LDPC(VN-CN)만 반복.
4. 핵심 기법: EXIT(Extrinsic Information Transfer) chart로 inner decoder(coherent: VN, differential: VN+BCJR)와 outer decoder(CN) 곡선이 만나는 turbo cliff SNR(`SNR_C`, `SNR_D`)을 비교해 differential penalty 극복 가능 여부 판정.
5. 핵심 식: coherent EXIT `I_CN(I)=Σ ρ_i J(√(i-1)J^-1(I)^2+8R·Eb/N0)`, differential EXIT `I_VN,BCJR(I)=Σ λ_i J(√((i-1)J^-1(I)^2+J^-1(I_B,out)^2))` — check/variable edge degree distribution `ρ`,`λ`로 상호정보 전이 계산.
6. 확장: 두 SNR 동작점에서 동시에 열리는 조건 `Ĩ_VN = min{I_VN,BCJR(I,SNR_D), I_VN(I,SNR_C)}`을 outer(CN) 제약으로 사용하는 결합 최적화(coherent+differential 동시 만족).
7. 구현 디테일: variable node degree는 고정(regular)하고 check node edge degree distribution `ρ`만 선형계획(LP류) 최적화 대상으로 단순화, 최소/최대 check·variable 노드 차수로 복잡도 근사.
8. 최적화 절차: rate 최대화를 목적함수로 하는 최적화 문제(P)를 EXIT chart 제약 하에 풀어 achievable rate 도출(수치 최적화, DE/GA 아님).
9. 결과: 낮은 variable node degree(약체 코드)에서는 differential penalty 극복 가능(differential이 coherent보다 우수), 반대로 높은 variable node degree에서는 coherent가 우수; 결합 최적화 코드는 두 케이스 중 더 나쁜 쪽과 거의 동일 성능.
10. 한계: EXIT chart는 waterfall(turbo cliff)만 예측하며 error floor는 다루지 않음, HW 미설계, BER 시뮬 없이 이론적 EXIT 분석만 수행, variable node degree distribution은 최적화 대상에서 제외(향후 과제).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| EXIT chart 기반 check node degree distribution 최적화(coherent+differential 결합) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 고정 QC-LDPC 구조라 degree distribution 재최적화 대상이 아니며, differential modulation/BCJR turbo detection 개념 자체가 NAND read 파이프라인에 대응 없음 |
| BCJR 기반 differential demodulation과 LDPC 간 turbo iteration | 대응 없음 | NAND read 채널은 위상 슬립(phase slip) 문제가 없어 관련성 낮음 |

적용 가치: 낮음 — 광통신 coherent/differential 검출 전환이라는 특수 문제를 위한 부호 설계로, NAND binary QC-LDPC의 read 채널·복호 구조와 무관.
