### B. 알고리즘 요약

1. 시스템: LDPC 부호화→random bit interleaving→M-ary 변조(16/32/64QAM, 8PSK) BICM, AWGN 채널; DVB-S2 rate 0.44/0.5, length 504·16200·64800.
2. 문제: 기존 BICM-SPA는 같은 심볼 내 비트 간 통계적 종속성을 무시해 MAP로 수렴 못 하고 변조 크기 커질수록 BICM 용량 gap 확대, 특히 Gray 매핑 불가(32QAM)에서 열화.
3. 가정: LDPC parity-check 행렬이 sparse·cycle-free라 한 심볼의 여러 비트가 동일 check node에 연결되는(multi-edge) 심볼 수가 매우 적어 무시 가능.
4. 핵심 기법: 최적 generalized symbol SPA를 GDL(Generalized Distributive Law)로 단순화한 `BICM-ISPA` — 복조와 복호를 결합해 심볼 APP를 iterative 반영.
5. 핵심 식: check node 갱신에서 심볼 로컬 비용 `W(m)`를 log 도메인(Jacobi logarithm, `max*`)으로 근사해 두 상태만 갖는 unilateral 연결 비트에 대해 빠르게 계산.
6. 확장: multi-edge(한 심볼의 2개 이상 비트가 같은 check node) 경우 dual edge 비트를 "새 비트"로 결합해 check node 갱신을 수정, 미고려분은 iteration 중 asymptotic 자기보정.
7. 구현: unilateral 케이스만 처리해도 되므로 복잡도가 `M`과 check node degree에 대해 선형(기존 symbol SPA는 `M^dc` 지수 증가), Table II로 비교.
8. 학습절차: 없음(순수 message passing 알고리즘).
9. 결과: 16QAM 0.12dB, 64QAM 0.2dB, 32QAM(quasi-Gray) 0.5dB, non-Gray 매핑 시 최대 1.5dB BICM-SPA 대비 개선, 목표 BER 도달 평균 iteration 더 적음(최대 50 iter).
10. 한계: 순수 시뮬(HW/gate count 없음), M-ary 변조·BICM에 고유하여 BPSK성 저장 채널과 무관.

| BICM-ISPA 반복 복호(심볼 결합) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 반복 message passing 위치는 같으나 심볼 노드/APP 결합은 binary NAND 복호에 부재 |
| log-domain check node 갱신(Jacobi/max*) | `decoder.cpp` `CNU_Update_New_Mag()` | min-sum과 유사 근사 계열이나 Prime ECC는 이미 min-sum 보유(중복), 심볼 결합분은 불필요 |
| 심볼 노드/M-ary 복조 결합, GDL | 대응 없음 | Prime ECC는 심볼/변조 계층 없는 순수 binary QC-LDPC |

적용 가치: **낮음** — 고차 변조 BICM 복조-복호 결합 알고리즘으로, NAND binary LDPC에는 결합할 심볼 상관 구조 자체가 없어 이식 요소가 사실상 없음.
