### B. 알고리즘 요약

1. 목표: 정수격자(integer lattice) `L(γ×q)` (q는 소수)를 이용해 girth-6, 고rate, low encoding complexity를 갖는 QC-LDPC 부호를 재귀적·체계적으로 구성.
2. 문제의식: 기존 random-like LDPC는 고rate 중간길이 부호에서 우수한 최소거리를 얻기 어렵고 인코딩 복잡도도 높음; 대수적/기하적 구성법은 girth·최소거리는 좋으나 유연성이 제한적.
3. 핵심 모델: `γ×q` 격자의 incidence structure를 (1,2)-configuration으로 보고, 두 configuration 간 partial(weak) Kronecker product `⊗_w`를 정의(기존 [12] 기법을 격자에 적용).
4. 핵심 기법: `L(γ×q)`를 자기 자신과 `n-1`회 재귀적으로 product하여 `L^n(γ×q)`를 얻고, 이로부터 QC-LDPC 부호 `C^n_{γ×q}`를 생성. 부호 파라미터: 길이 `b_i=q^{2i}`, column weight `γ`, row weight `q^i`, 밀도 `1/q^i`, rate `≥1-γ/q^i`.
5. 핵심 정리: `d(C^n_{γ×q}) = d(C^1_{γ×q})` — n-fold product해도 최소거리가 core code(n=1)와 항상 동일하게 유지됨(Theorem 4). 최소중량 codeword의 support도 core code의 support로 특성화됨(Proposition 2).
6. 확장: `d(C^n_{3×q})=6` (q≥5 소수), `d(C^n_{4×q})≥8` 등 γ,q별 최소거리 하한을 구체적으로 도출.
7. 구현 디테일: `Z_q`-representation으로 순열행렬을 정수로 표현해 행렬 연산을 단순화, 인코딩은 shift register 기반으로 저복잡도.
8. 최적화 절차 없음(순수 조합론적 구성, 파라미터 (γ,q) 선택만 존재).
9. 결과: BPSK-AWGN 채널, 최대 50 iteration 시뮬레이션에서 구성 부호가 동일 rate/length의 random-like 부호와 대등하거나 일부 우수한 BER 성능(예: length 15625, rate 0.976에서 waterfall/error-floor 모두 경쟁력).
10. 한계: 디코더 알고리즘·HW 구현·양자화 논의 전무(순수 부호구성+시뮬레이션 검증), 부호 길이가 `q`에 따라 기하급수적으로 증가해 rate 세밀 조정이 어려움.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 격자 기반 재귀적 QC-LDPC H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | 특정 QC-LDPC 구조가 고정되어 있어 이식 시 H-matrix 전면 교체·재검증 필요, 난이도 높음 |
| girth-6 보장, 최소거리=core code 이론 | 대응 없음 | 부호 이론적 성질 증명일 뿐 코드/HW 모듈과 직접 대응 없음 |
| shift register 기반 저복잡도 인코딩 | `encoder.cpp` `Generate_PCM_encoding()` | dual-diagonal과 다른 구조이나 개념적으로만 참고 가능, 실제 인코더 교체는 중~높음 난이도 |

> 적용 가치: 낮음 — girth-6·최소거리 보증이라는 이론적 미덕은 있으나 Prime ECC의 특정 QC-LDPC 구조(z=32 lifting, base matrix 고정)와 무관한 별도 부호족이라 H-matrix 전면 재설계 없이는 이식 불가.
