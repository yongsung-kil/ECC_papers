### B. 알고리즘 요약

1. CubeSat용 (512,384) IRA-LDPC, rate 3/4, DBPSK 변조 + AWGN(LOS FSPL) 채널 가정, 목표는 저전력·고속·소형 하드웨어에서 동작.
2. 기존 QC-LDPC dual-diagonal 인코더 대비, IRA(반복+누적) 구성 기반 dual-diagonal `H=[A|B]` 구조로 인코딩 연산량을 `(n²-k²)/2`에서 `k×(n-k)`로 감소(8192회 절감).
3. 인코딩 재귀식 `p_i = p_{i-1} ⊕ (⊕_j a_{i-1,j}·d_j)`(식(6))으로 이전 parity bit만 참조, 서브행렬 `A`만 저장하면 되어 하드웨어 자원 절감.
4. degree distribution은 trial-and-error 알고리즘(Algorithm 1)으로 `ρ(x)`, `λ(x)` 조정, IRA 구성(반복→누적→대각 인터리빙)으로 parity check 행렬 `A` 생성(Algorithm 2).
5. 디코더는 MS(min-sum)와 BF(bit-flipping)를 결합한 하이브리드: variable node degree가 높은 비트는 MS(soft, LLR/tanh 기반), 낮은 비트는 BF(hard)로 처리해 복잡도-성능 균형.
6. 복잡도는 `O(n²(n-k))`로 MS의 `O(n³)`보다 낮고, code rate가 높을수록(`n-k` 감소) 복잡도가 더 낮아짐(식 `O(n³(1-R))`).
7. BER은 Gaussian Approximation(GA)으로 이론식 유도(식(29)~(36)), check node 메시지 평균 `m_c`를 반복적으로 갱신해 시뮬레이션과 근사 일치 확인.
8. 부수 트릭: DBPSK 매핑을 단일 XOR 게이트로 구현, RCF pulse shaping filter 48곱셈기/41덧셈기로 최소화(비교 대상 260/253 대비 대폭 절감).
9. 결과: 10 iteration으로 BER 10⁻⁶ 도달(Eb/N0 7.8dB @ DBPSK), 코드rate 3/4에서 GDBF/NGDBF 대비 복잡도 66%↓·Eb/N0 0.3~0.5dB↓; 정보율 22.5Mbps로 비교 CubeSat LDPC 대비 최고.
10. 한계: 완전병렬 HW 아키텍처/FPGA 구현은 "향후 별도 논문"으로 예정, 본 논문의 1Gbps@3W(Cyclone V)는 MATLAB 분석 기반 추정치이며 실측/합성 결과 아님.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| IRA dual-diagonal 재귀 인코딩 (`p_i = p_{i-1} ⊕ Σa·d`) | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | Prime ECC도 dual-diagonal 인코딩 방식을 이미 사용 중이나, IRA 구성(비-QC)이라 회로 구조/PCM 로드 방식이 다름 - 개념적 참고 수준 |
| MS-BF 하이브리드 디코딩 (VN degree 기반 MS/BF 분기) | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | Prime ECC는 순수 min-sum(soft) 구조이며 hard BF 분기 스케줄링은 대응 모듈 없음 - 이식 시 스케줄러 재설계 필요 |
| IRA degree distribution 최적화(trial-and-error, Algorithm 1) | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC-LDPC H-matrix 고정이라 IRA 재구성은 이식 난이도 높음(§4 부호설계=높음) |
| LLR 기반 GA(Gaussian Approximation) 이론 BER 분석 | 대응 없음 | 순수 이론 분석 도구로 코드 모듈과 무관 |

> 적용 가치: 중간 - dual-diagonal 재귀 인코딩 아이디어는 Prime ECC 인코더와 개념적으로 유사해 참고 가치가 있으나, 핵심 디코더는 hard-decision BF를 혼합한 구조로 Prime ECC의 순수 soft min-sum 파이프라인과 이질적이며 HW 검증도 미완료.
