### B. 알고리즘 요약

1. 대상은 cycle-reduced BCH(63,36), BCH(63,45) 코드이며 AWGN 채널에서 BPSK 전송, 모두 all-zero codeword 전송 가정.
2. 신경망 디코더(WBP/BP-FF)의 성능은 학습 데이터 품질에 크게 좌우되는데, decision boundary 근방(너무 쉽지도 어렵지도 않은) 샘플을 만드는 기존 방식(거리/신뢰도 기준)이 비효율적이라는 문제를 다룸.
3. 관측공간을 반지름 `r`(수신벡터 norm, scaled Chi 분포 `g(r)`) 기준 shell `Y_l`로 분할하고 각 shell의 오류확률(acquisition function) `θ_l = Pr(I(z)=1; z∈Y_l)`을 정의.
4. 핵심 기법은 Importance Sampling 분포 `P*_l = θ_l P_l / Σ θ_l P_l`(`식 6`)로 샘플링 반지름을 선택, 오류확률이 높은 shell을 우선 샘플링해 가장 어려운(디코더가 불확실한) 학습 샘플을 생성.
5. `θ_l`은 학습 반복마다 Monte Carlo 추정(`식 5`)으로 갱신되며 이는 shell별 조건부 오류확률 자체가 디코더 상태에 따라 변하는 것을 반영하는 적응형 샘플링 분포임.
6. 확장으로 active learning 루프: T번째 iteration마다 `P*_l(T)`로 학습/테스트셋 생성 → N2 epoch 학습 → 테스트로 `θ_l(T)` 재추정 → 갱신을 N1회까지 반복(Algorithm 1).
7. 부수 트릭: 초기 `θ_l=0`이 되는(에러 미관측) 문제를 막기 위해 비영(non-zero) `θ_l`들 사이를 선형보간하는 보정식(`식 8`), 임계값 `γ` 초과 시 0으로 클리핑해 과도한 노이즈 샘플 배제.
8. 학습/최적화 절차 자체가 핵심: WBP(Tanner graph unroll, VN/CN layer trainable weight, BCE loss, SGD)를 오프라인으로 훈련, 테스트/실사용 시 추가 연산 없음.
9. 결과: BCH(63,36/45)에서 기존 BP-FF 대비 waterfall 영역 BER 최대 0.62dB, error-floor 영역 BER 최대 1.9dB 개선(FER도 유사).
10. 한계: 신경망 BP(학습 가중치 기반) 전용 오프라인 학습 기법이며, HW 설계·복잡도·throughput 분석 없음, 짧은 길이(n=63) BCH 코드로만 검증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| WBP(Neural BP) 학습 가중치 | 대응 없음 | Prime ECC는 학습 가중치가 없는 고정 min-sum 알고리즘(`decoder.cpp` `CNU_Update_New_Mag()`) |
| IS 기반 학습 데이터 샘플링(shell 기반 오류확률) | 대응 없음 | Prime ECC는 신경망 학습 파이프라인 자체가 없음 |
| error-floor 개선 목표 | `decoder.cpp` `LDPC_Decoder()` | 개선 목표(error-floor)는 개념적으로 관련되나 접근법(학습 데이터 생성)이 min-sum 튜닝과 무관 |

> 적용 가치: 낮음 — 신경망 기반 BP 디코더(WBP)의 오프라인 학습 데이터 생성 기법으로, 학습 가중치가 없는 Prime ECC의 고정 min-sum 디코더 구조에 이식할 접점이 없음.
