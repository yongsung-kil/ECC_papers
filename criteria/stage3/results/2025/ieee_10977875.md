### B. 알고리즘 요약 (10줄 내외)

1. 3D TLC NAND flash(EoL 단계, 128 active/141 full layer)를 채널로 하는 QC-LDPC(protograph 12×80, circulant size 128, `λ(x)=0.0125x²+0.9875x³`, `ρ(x)=0.41667x²⁵+0.58333x²⁶`) 부호를 대상.
2. 기존 정적 read threshold와 IBIS/Verilog 저수준 모델은 EoL 단계의 retention noise·layer-to-layer 공정편차를 반영하지 못해 디코딩 성능이 저하되는 문제를 해결하고자 함.
3. 채널 모델: 각 레이어 k의 문턱전압 분포를 erase/program state별 컨볼루션(`p_s0,k = p_u0 * p_npe * p_nr * p_nlk`)으로 모델링, retention 분산 `Bk`를 지수감쇠 항 포함 식(4)로 개선 제안(기존 3차 다항식 모델보다 정확).
4. 핵심 기법(RCQ layered decoder, Algorithm 1): CN/VN 업데이트에 `γ`(채널 LLR), `β`(C2V 메시지)를 사용하되, 매 iteration·row별 사전계산된 reconstruction 테이블 `R`과 quantization 테이블 `Q`로 CN/VN 메시지를 압축·복원.
5. 핵심 식 `α_{m,r} = sign(l_{m,r})·Q(m+It·M, min(|l_{m,r}|,VNmax))`, `β_{m,r}=min(R(m+It·M,|α|),CNmax)·Π sign` — 일반 min-sum의 min/sum 연산을 LUT 참조로 대체해 연산·라우팅 복잡도 절감.
6. 확장 기법: layer-to-layer variation 모델(식 2~4)을 반영한 동적 read threshold 조정 메커니즘으로 레이어별 이질적 오류 특성에 맞춰 LLR 산출.
7. 구현 디테일: R/Q 테이블은 CNbitwise/VNbitwise 및 최대 iteration 수에 따라 크기 결정되는 저비트폭 양자화 LUT(부호 비트 분리, symmetric 설계).
8. 최적화 절차: symmetric AWGN 채널 기반 Density Evolution으로 초기 R/Q 테이블 설계 후, mutual-information 최대화(Kurkovskiy 방법, Monte Carlo PDF 추정) 및 error-floor 대응 importance sampling으로 fine-tuning, read threshold는 레이어별 mutual information을 dynamic programming으로 최대화.
9. 결과: RBER 0.004 이상 개선, throughput-to-area 비율 최대 16% 향상, TLC 3D NAND 내구성이 single-layer 모델 대비 30% 이상, 신경망(MLP) 기반 방법 대비 12%, cubic 모델 대비 21% 개선.
10. 한계: 전부 시뮬레이션 결과이며 FPGA/ASIC 합성·실측 검증 없음, gatecount/throughput 절대수치 미기재, 향후 4KB QC-LDPC 확장은 future work로 남김(현재는 1KB 구현).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| RCQ layered decoder (LUT 기반 CN/VN 업데이트) | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | min-sum 연산을 LUT 참조로 대체하는 구조로, 기존 min-sum 파이프라인에 R/Q 테이블만 추가하면 이식 가능성 높음 |
| iteration별 quantization/reconstruction 테이블 | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | 이미 iteration별 LLR 테이블 구조 보유 - R/Q 테이블 개념과 직접 대응 |
| soft-read threshold / layer-variation 동적 조정 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | 3D NAND layer 특화 threshold 최적화는 채널 LLR 양자화 로직에 참조 가능 |
| Mutual-information 기반 R/Q 테이블 최적화(오프라인 DE/MC) | `local_opt.cpp` | LLR 자동최적화 모듈과 개념적으로 유사(오프라인 파라미터 튜닝) |
| H-matrix protograph/circulant 구조 (12×80, L=128) | `ecc_top.cpp` `Load_PCM()` | 특정 QC-LDPC 고정 구조로 재검증 부담 있으나 부호구조 참고용 |

> 적용 가치: 높음 — NAND 3D TLC를 직접 타깃한 QC-LDPC LUT 기반 저비트폭 디코더로, Prime ECC의 기존 min-sum + iteration별 LLR 테이블 구조와 개념적으로 정합되어 R/Q 테이블 설계·layer-aware threshold 조정 아이디어를 직접 참고할 수 있음.
