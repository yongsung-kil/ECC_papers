### B. 알고리즘 요약

1. 시스템은 BPSK+AWGN 채널을 통한 이진 linear block code C(n,k)이며, rate-compatible(RC) 방식으로 parity bit를 뒤에서부터 puncturing해 코드율 R=k/n을 조절.
2. 기존 puncturing 기반 RC 부호는 precode·puncture pattern이 별도 최적화되어 서브옵티멀하고, 기존 AI 부호는 rate별로 별도 모델이 필요해 저장 비용이 큰 문제를 다룸.
3. 채널/디코더 모델: 표준 AWGN + `RC-NBP`(rate-compatible neural belief propagation) 디코더로, rate에 따라 Tanner 그래프 뉴런(VN/CN) 활성/비활성.
4. 핵심 기법: `Matrix-Gen` 모듈이 학습 파라미터 `W`로부터 `H=[H1|H2]`(H2는 하삼각 고정 대각 1) 생성, differentiable step function(DSF)으로 실수 W를 이진 H로 미분 가능하게 매핑.
5. 핵심 식 `dσ(x)/dx = σ(x)σ(1−x)` (식4): non-differentiable step function의 backward pass를 sigmoid로 근사해 Gauss-Jordan 소거 없이 H 학습을 gradient 기반으로 가능케 함.
6. 확장: Multi-Task Learning(MTL)으로 여러 rate를 동시 학습, 낮은 rate 데이터가 높은 rate 파라미터 학습에도 관여하는 parameter-sharing 방식.
7. 부수 트릭: H를 시스템적(`H=[P|I]`) 뿐 아니라 non-systematic 구조로도 학습 가능하게 하여 짧은 사이클을 줄임(Fig.7에서 non-systematic이 systematic보다 우수).
8. 학습 절차: precode(최저rate)를 5,000epoch 단일rate 학습 후 5,000epoch 혼합rate(MTL) 학습, Adam(lr=0.001), batch 256, loss는 BCE(식5).
9. 결과: k=11, n=31 기준 BER 1e-4에서 BCH 대비 3dB, 5G BG2 LDPC 대비 2dB 이득; punctured AI-RC-LBC가 punctured 단일rate AI-LBC보다 우수.
10. 한계: 검증 코드 길이가 n=16~100 수준(curse of dimensionality로 긴 코드 학습 어려움), decoder iteration 5회 고정 시뮬뿐이며 HW 구현·QC 구조·NAND 채널 모델 전혀 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| AE 기반 H-matrix(비-QC, dense) 학습 설계 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 QC-LDPC(base+circulant shift) 고정 구조인데 본 논문은 QC 제약 없는 dense H를 학습, 구조 자체가 달라 이식 시 QC화 재설계·재검증 필요(난이도 높음) |
| RC-NBP 디코더(학습형 min-sum 유사 가중치) | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 개념적으로 min-sum에 learnable weight/bias를 추가하는 형태이나, 본 논문은 짧은 코드(n≤100)·5 iteration 고정 시뮬로 NAND 규모 검증 없음 |
| Rate-compatible puncturing(코드율 가변) | `encoder.cpp` `LDPC_encoder_4KB()` | Prime ECC 인코더는 dual-diagonal 고정 구조로 puncture 기반 가변 rate 미지원, 개념 이식 시 인코더 전면 재설계 필요 |
| (대응 없음) DSF/미분가능 H 학습, MTL 파라미터 공유 | 대응 없음 | Prime ECC는 학습 기반 부호설계 파이프라인 자체가 없음 |

적용 가치: 낮음 — 짧은 길이(n≤100) code-design 논문으로 QC-LDPC 구조가 아닌 dense H를 학습하며, NAND 채널·HW·긴 코드 확장성에 대한 근거가 전혀 없어 Prime ECC의 QC-LDPC 고정 구조에 이식하려면 사실상 전면 재설계가 필요함.
