### B. 알고리즘 요약

1. 5G NR LDPC(raptor-like QC-LDPC, BG1/BG2, rate-1/2, length 1440 예시)를 basic code로 사용, BMST 방식으로 여러 시간슬롯의 코드워드를 중첩 전송해 spatially coupled LDPC(SC-LDPC)로 변환.
2. 기존 SC-LDPC는 인코더/디코더가 기본 LDPC와 구조적으로 달라 6G 하위호환성 요구를 충족 못함; BMST는 basic code 아키텍처를 그대로 재사용 가능해 HW-friendly.
3. 채널 모델: AWGN, fast fading(Rayleigh, `Tf=1`), quasi-static block fading(Rayleigh, `Tf>1`) 3종.
4. 핵심 기법: memory size(coupled width) `m`개의 이전 코드워드를 selection matrix `Si`(superposition fraction `αi`)로 마스킹해 현재 코드워드에 중첩. `c(t) = v(t) + Σ w(t-i)`.
5. 일반화 EXIT chart 분석으로 디코딩 threshold를 계산. `m`을 키우면 threshold가 개선됨(m=1: 0.4515dB → m=3: 0.4414dB, d=6 기준).
6. 부수 트릭: 코드율 손실은 `R = kL / (n(L + Σαi))`로 L→∞일수록 손실 미미. 5G BG1/BG2 protograph 기반 lifting factor Z로 base matrix 확장.
7. 학습/최적화: superposition fraction `αi`는 EXIT chart 기반 exhaustive search로 근사 최적화(m=1,2는 완전탐색, m=3은 특정값 사용).
8. 핵심 기법 2단: adaptive sliding window decoding(ASWD) - 기본 window d=1부터 시작, 디코딩 실패 시(`vHT≠0`) 윈도우를 점진 확장(최대 dmax=6), 성공 시 즉시 확정. m>1 시 고정 SWD는 d≥2m+1 필요해 latency 문제 발생.
9. 결과: m=2 코드가 5G LDPC 대비 AWGN 0.4dB, fast fading 0.5dB, quasi-static block fading 1.7dB 이득. ASWD는 중~고SNR에서 평균 윈도우 크기를 SWD 대비 크게 줄이며(고SNR에서 1에 근접) 성능 손실 없음.
10. 한계: 무선 페이딩 채널 대상 시뮬레이션뿐, HW 미설계. 다중 시간슬롯 코드워드 간 중첩 구조로 NAND의 단일 코드워드 페이지 단위 ECC와 시스템 모델 자체가 다름.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BMST 시간축 spatial coupling (다중 코드워드 중첩) | 대응 없음 | Prime ECC는 단일 페이지/코드워드 단위 QC-LDPC 디코더로, 여러 코드워드를 시간축으로 결합하는 BMST 구조 자체가 대응 없음 |
| adaptive sliding window decoding(ASWD, 실패 시 윈도우 확장) | `partial_CRC.cpp` `partial_crc_HW_T4()` (조기종료 CRC)와 개념적 유사성만 있음 | 실패 감지 후 추가 자원(윈도우/재시도) 투입이라는 아이디어 차원에서 조기종료 로직과 유사하나, ASWD는 다중 코드워드 간 윈도우 확장이라 구조가 다름 → "대응 없음"에 가까움 |
| EXIT chart 기반 threshold/superposition fraction 최적화 | 대응 없음 | Prime ECC는 고정 H-matrix(`Load_PCM()`) 기반이며 EXIT 기반 앙상블/파라미터 설계 도구가 없음 |
| 5G BG1/BG2 protograph QC-LDPC(기본 코드) | `ecc_top.cpp` `Load_PCM()` (H-matrix 구조는 다르나 QC-LDPC라는 점만 참조 가능) | 기본 코드 자체는 QC-LDPC지만 NAND용 고rate 구조와 파라미터가 상이 |

**적용 가치**: 낮음 — 무선 페이딩 채널 대상 시간축 spatial coupling 통신 시스템 논문으로, NAND 단일 블록 binary QC-LDPC 디코더/HW 구조와 시스템 전제가 근본적으로 다름.
