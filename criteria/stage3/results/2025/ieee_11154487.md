### B. 알고리즘 요약

1. Quantum CSS 부호(Calderbank-Shor-Steane)를 2-block group-algebra(2BGA) 프레임워크로 구성, `HX=[A B]`, `HZ=[B^T A^T]` 형태의 패리티체크행렬 사용.
2. 기존 bivariate bicycle(BB) 코드는 정정에 ordered statistics decoding(OSD, `O(n^3)`)이 필요해 비효율적이라는 문제 제기.
3. 채널모델은 depolarizing noise(확률 `ε/3`로 X/Z/Y 오류)이며 syndrome measurement는 완벽하다고 가정.
4. 고전 Margulis 구성(`H=[M0 M1]`, `G=SL(2,Zp)`의 Cayley graph 기반)을 quantum 영역으로 확장해 quantum Margulis code 정의.
5. 핵심 주장: quantum Margulis 코드의 Tanner graph는 group symmetry가 없어(qubit群 대칭 부재) 표준 normalized min-sum(nMS) 디코더로도 OSD 없이 수렴 가능.
6. girth 제어 알고리즘(Algorithm 1,2)으로 4-cycle/6-cycle을 제거하며 generator set을 탐색, girth 6(n=240) 또는 8(n=642) 부호 구성.
7. nMS 디코더는 flooding schedule, normalization factor `β=0.875` 사용, 최대 iteration 300(n=240)/700(n=642).
8. 학습 절차 없음(구성 알고리즘은 탐색적 generator set 교체 방식).
9. 결과: n=240, ε=0.01에서 logical error rate ~`10^-8` 달성(k=2), BB코드 대비 error-floor에서 크게 우수. 디코딩 복잡도가 `O(n^3)`(OSD)에서 `O(n)`(MS)로 감소.
10. 한계: circuit-level noise(실제 회로 결함 모델)에서는 이 우위가 사라져 OSD 후처리가 여전히 필요; HW 구현 없이 시뮬레이션에 한정.

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| min-sum/normalized min-sum 디코딩 | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | 알고리즘 형태는 유사하나 quantum stabilizer 대상이라 직접 이식 불가 |
| girth 제어 부호구성 (2BGA/Cayley graph) | `ecc_top.cpp` `Load_PCM()` | binary QC-LDPC 구성과 무관한 quantum 특화 구성법이라 대응 없음 |
| flooding schedule / 정규화 β | `decoder.cpp` `Get_VNU_Table_Idx()` | 개념적 유사성만 존재, non-binary 채널모델 |
```

> 적용 가치: 낮음 — quantum non-binary CSS 부호 구성·디코딩 논문으로 binary NAND QC-LDPC 부호설계/디코더와 구조적으로 이질적이며, min-sum 자체는 이미 Prime ECC에 보유.
