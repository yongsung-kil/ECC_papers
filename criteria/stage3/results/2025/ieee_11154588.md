### B. 알고리즘 요약

1. 시스템: 이진 선형 블록코드 C(n,k,dmin), BSC 채널(교차확률 α)에서 hard-decision bit-flipping 복호를 다룸. 실험은 Tanner 코드 (155, 64, 20)로 수행.
2. 문제의식: ML 복호는 NP-hard이며, 기존 학습 기반 bit-flipping 디코더(Deep Q-Network)는 Q-value 근사 오차로 정책이 진동(oscillation)하여 수렴 실패가 발생.
3. 모델링: 복호 과정을 MDP `(S,A,P,R,γ)`로 정의 - 상태 `s`는 syndrome(길이 m), 행동 `a∈{1,...,n}`은 비트 플립, 전이 `s'=s+h_ca`는 결정적, 보상은 `R=1`(종료) 또는 `-1/L`(매 스텝 페널티).
4. 핵심 기법 1단: Deep Q-learning으로 `Q_θ(s,a)`를 학습(primary/target 네트워크, replay memory, ε-greedy), loss `L(θ)=E[(r+γ max_a' Q^t_θ-(s',a') - Q^p_θ(s,a))^2]`.
5. 핵심 기법 2단(action-list decoding): 각 depth에서 top-k Q-value 행동으로 후보 경로 집합 `B_l`을 유지, 조건 `Q(s_l^i,a) > v_i`(부모보다 개선되는 행동만) 만족 시에만 경로 확장, 매 depth마다 top-k 경로만 유지(beam search와 유사).
6. 자동형군 확장: QC-LDPC의 circulant 구조에서 오는 순환 자기동형군(Z_p의 곱셈/덧셈 부분군)을 이용해 syndrome을 순환 이동시키며 재시도, `S(Γ_v(e))=Γ_s(S(v))` 성질을 활용.
7. 부수 트릭: 탐색은 all-zero syndrome 도달 또는 최대 깊이 `D_max`에서 종료, 최고 점수 후보 경로를 최종 출력으로 선택.
8. 학습 절차: 은닉층 1개(512 뉴런) 신경망, 10^6 에피소드, batch 128, γ=0.9, ε을 0.9→0.05로 선형 감쇠, learning rate 10^-4, `D_max=10`.
9. 결과: list size 증가(1→40)에 따라 FER/BER가 크게 개선되며, 자기동형군을 결합한 list size=5가 BDD(radius=10)에 근접하는 성능을 달성.
10. 한계: 짧은 코드(n=155)·BSC·hard-decision만 다루었고 HW 설계·복잡도·지연시간 분석은 전무하며 Monte Carlo 시뮬 결과에 국한.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Deep-Q 기반 bit-flipping 디코딩 전체 | 대응 없음 | Prime ECC는 min-sum 기반 soft-info 디코더로 RL/bit-flipping 알고리즘 계열이 아님. |
| Action-list(beam search) 후보 경로 관리 | 대응 없음 | 조기종료(`partial_CRC.cpp`)와 목적은 유사하나 구조적으로 무관. |
| QC-LDPC 자기동형군(circulant shift) 활용 | `ecc_top.cpp` `Load_PCM()` | H-matrix가 base+circulant 구조(`PCM_b`)라는 점에서 구조적 유사성은 있으나 자기동형군 재시도 전략은 미구현. |
| BSC/hard-decision 채널 모델 | `channel.cpp` `Set_R_Offset()` | hard-1bit(HD) 지원과 연관되나 논문은 soft-info를 전혀 다루지 않음. |

> 적용 가치: 낮음 — RL 기반 hard-decision bit-flipping 계열로 Prime ECC의 soft min-sum HW 아키텍처와 알고리즘 패러다임이 상이하고, HW/복잡도 분석이 없어 이식 시 재검증 부담이 큼.
