### B. 알고리즘 요약

1. 5G TDD MIMO-OFDM 시스템에서 Alice/Bob이 DM-RS 채널추정으로 상호 채널특성을 추출해 물리계층 키를 생성하고, 5G 표준 QC-LDPC 코딩모듈로 키 재조정(reconciliation)한다.
2. 기존 PLKG 기법은 5G 표준 물리계층과 결합이 미흡하고, 5G LDPC 디코더가 soft-input을 요구하는데 기존 방식은 binary quantization 위주라는 문제를 지적한다.
3. 채널 모델은 3GPP CDL-A/B/C(UMi/UMa/RMa)이며, DM-RS는 프리코딩 행렬과 결합돼 있어 `W_A^T ĥ_A = W_A^T h_BA W_B + ñ_A` 형태로 reciprocity 복원 전처리가 필요하다.
4. 핵심 기법: MAQ(Multibit Adaptive Quantization)의 Gray코드 비트별로 flip 임계값만 모은 부분집합 `γ^m`을 구성하고, 채널진폭 `X(n)`과 해당 부분집합의 최소거리 `d_i[m] = min_{γ∈γ^m}|X(n)-γ|`를 신뢰도로 정의한다.
5. `d_i[m]`이 클수록 양자화 오류 확률이 낮다는 의미이며, soft-bit은 `r_i[m] = (1-2b_i[m])·d_i[m]`으로 부호화해 LDPC decoder의 LLR 대응 입력으로 사용한다.
6. 확장: Alice가 보낸 parity bit `w`의 복조 LLR(`LLR'_w`)과 Bob의 soft-bit(`R'_b`)을 결합한 `[R'_b | LLR'_w]`를 LDPC decoder 입력으로 구성해 syndrome 기반 키 복구를 수행한다.
7. 구현 디테일: LDPC 디코딩 알고리즘은 표준 Normalized Min-Sum(NMS) 그대로 사용, base graph는 3GPP TS 38.212 BG1/BG2(rate 1/3~8/9 및 1/5~2/3)를 그대로 채택.
8. 별도 학습/최적화 절차는 없음(구성법은 결정론적 임계값 분할).
9. 결과: 3-bit quantization·rate 1/3 LDPC에서 soft-decision이 hard-decision 대비 낮은 KDR을 달성하며(SNR별 CDL-C가 -2dB, CDL-A가 10dB에서 완전 재조정 도달), Eve는 모든 SNR에서 KDR≈0.5 유지(보안 검증).
10. 한계: LDPC 자체(min-sum 변형, 부호구성)는 개선 없이 표준 재사용뿐이며 HW 미설계, MATLAB 5G Toolbox 시뮬레이션에 한정, 목적이 보안 키 재조정으로 NAND ECC 정정 성능과 무관.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Normalized Min-Sum 디코딩 | `decoder.cpp` `LDPC_Decoder()`, `CNU_Update_New_Mag()` | 표준 NMS 재사용뿐 개선 없음, 이미 Prime ECC가 min-sum 보유(§3) — 중복 |
| Gray코드 bit-flip 임계값 분할 기반 soft-bit(신뢰도) 생성 | `channel.cpp` `Set_LLR_Th()` | 원리(진폭-임계값 거리 기반 신뢰도)는 LLR 양자화 threshold 설계와 유사하나 목적이 채널추정 키생성이라 직접 이식엔 재해석 필요 |
| 3GPP base graph(BG1/BG2) H-matrix | `ecc_top.cpp` `Load_PCM()` | 5G 표준 고정 구조, Prime ECC의 NAND 전용 H-matrix와 무관 |

> 적용 가치: 낮음 — LDPC 디코더/부호 자체의 개선이 전혀 없고 5G 표준을 물리계층 보안(키 재조정) 응용에 재사용한 시스템 논문으로, Prime ECC의 정정성능·HW 개선과 목적이 다름.
