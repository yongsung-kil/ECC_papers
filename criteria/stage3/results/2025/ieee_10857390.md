### B. 알고리즘 요약

1. 시스템은 패킷 손실(erasure) 네트워크에서 `K`개 입력 심볼을 `n`개 배치(batch)로 전송하는 BATS(Batched Sparse) 코드로, outer code는 fountain code(LT 행렬형), inner code는 random linear network coding(RLNC), base field는 `F_q`.
2. 기존 연구는 BATS 코드와 precode를 분리해서 분석했거나 precode 없이만 다뤘고, precode 결합 유한길이 오류확률의 정확한 이론 분석이 없었던 것이 문제.
3. LDPC를 precode로 사용하는 LDPC-BATS 코드를 정의(`(K,n,M,Ψ,h,l,r,νmin)_q` LDPC-BATS 앙상블), BATS Tanner그래프와 LDPC Tanner그래프를 공유 VN으로 결합한 joint 그래프에서 BP(peeling)와 ML(inactivation) 복호를 수행.
4. 핵심 정리: BP 복호 오류확률 상한(Theorem 1) `P_E^BP(C_pre) <= union bound over stopping-set-like structures`; ML 복호 오류확률 상한(Theorem 2)은 precode의 weight enumerator `A_l`을 이용한 union bound.
5. Theorem 1의 핵심 아이디어는 BP(peeling) 복호 실패 시 잔여 VN 집합 `A`가 LDPC precode의 stopping set이자 동시에 BATS 코드의 "stopping set"이 되는 그래프 구조를 규명하고 그 확률을 union bound로 상한.
6. 확장/응용: 두 상한을 목적함수로 하여 LDPC-BATS 코드의 degree distribution `Ψ`를 최적화하는 문제(P1)를 구성 — BP 성능 상한을 최소화하며 ML 성능 상한을 임계값 `ϵ*` 이하로 제약.
7. 구현 디테일: precode는 이론분석 편의상 q-ary regular LDPC 앙상블(및 stopping set 크기 `ν_min` 미만을 제거한 expurgated 앙상블)에서 무작위 선택된 것으로 가정, 실제 코드는 시뮬레이션으로 검증.
8. 최적화 절차: 유도된 상한을 목적함수로 반복적(iterative) 수치 최적화를 수행, 소수(1~5회) 반복 만에 초기 분포 대비 유의미하게 개선되는 것을 관찰.
9. 결과: 128개 패킷을 packet-loss 라인 네트워크로 전송 시 최적화된 LDPC-BATS 코드가 표준 BATS 코드 대비 전송 오버헤드를 50% 미만으로 감소(예: 37.8%까지); inactivation 복호 복잡도(비활성 심볼 수)도 함께 낮게 유지.
10. 한계: 순수 이론(상한 유도 + Monte-Carlo 검증)이며 HW 미설계, erasure 채널(패킷 손실) 및 네트워크 코딩(RLNC) 특화 모델로 NAND read 채널·binary LDPC 구조와 무관, ML 상한은 유한체 차수 `q`가 클수록(예: F256) 느슨해짐.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| LDPC precode (q-ary regular LDPC) | 대응 없음 | Prime ECC는 binary 전용, q-ary(GF(q)) LDPC는 판정 기준상 이식성 하 |
| BP(peeling) decoding, ML(inactivation) decoding | 대응 없음 | erasure 채널 전용 그래프 복호(peeling/Gaussian elimination)로 min-sum LLR 복호와 메커니즘 완전히 다름 |
| degree distribution 최적화 | `ecc_top.cpp` `Load_PCM()` | 부호 degree 최적화라는 개념만 유사, 채널·목표(erasure vs AWGN/NAND read) 상이해 실질 대응 없음 |
| finite-length 오류확률 상한 이론 | 대응 없음 | error-floor 해석과 목적은 유사하나 유도 기법이 stopping-set 기반 network-coding 이론으로 별개 |
```

적용 가치: **낮음** — erasure 네트워크(fountain/network coding) 환경의 q-ary LDPC-precoded BATS 코드 이론 분석으로, NAND binary QC-LDPC ECC의 채널·부호·복호 구조와 접점이 없음.
