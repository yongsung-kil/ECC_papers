> 총평: transformer 어텐션에 belief-propagation의 Tanner-graph 구조와 syndrome 정보를 결합한 신경망 디코더 연구로, 짧은 LDPC/Polar 코드에서 BP 대비 소폭 성능 개선을 보이나 NAND용 고전 binary QC-LDPC HW 디코더(min-sum 기반)로의 이식은 구조적으로 어려움.

### B. 알고리즘 요약

1. 대상은 AWGN+BPSK 채널에서 짧은-중간 길이 LDPC(n=128~816) 및 Polar(n=128) 코드를 위한 transformer 기반 신경망 디코더(DiffMPT).
2. 기존 transformer 디코더(ECCT, CrossMPT)는 attention의 softmax 정규화로 인해 무관한 위치 간에도 배경 어텐션 노이즈가 남아 실제 오류 패턴을 가리는 문제가 있음.
3. 채널 모델은 표준 BPSK-AWGN이며 LLR `y† = 2y/σ²`, soft syndrome `s†_j = 2·arctanh(Π tanh(y†_i/2))`로 SPA check-node 연산을 모사해 입력 특징으로 사용.
4. 핵심 기법 1단: mean-field 근사 기반 미분가능 syndrome loss `L = L_transport + λ·L_validation`, `Pr(S_j=0) = 0.5(1+Π(Pr(Ĉ_i=0)-Pr(Ĉ_i=1)))`로 기존 비미분 syndrome loss 대체.
5. 핵심 식의 의미: syndrome이 0이 되어야 유효 코드워드라는 제약을 gradient 학습에 직접 반영, block error rate 개선 목적.
6. 핵심 기법 2단: differential cross-attention `DiffAttention(Q,K,V;M) = [softmax(QK^T/√d + ψ(M)) - λ_diff·softmax(QK^T/√d)]V`로 배경 어텐션 노이즈를 제거, 파리티체크행렬 H/H^T를 마스크로 사용해 bit-embedding과 syndrome-embedding 간 message-passing 형태의 반복 업데이트 수행.
7. 부수 트릭: bit embedding에 절대 LLR `|y†|`, syndrome embedding에 soft syndrome `s†` 사용(곱셈 잡음 z̃ 추정), attention 모듈 간 weight sharing, GELU+AdamW+cosine annealing 학습.
8. 학습 절차: 6-layer, 128차원 임베딩, 1000 epoch × 1000 minibatch(128 samples), SNR 2~7dB 균등샘플, gradient clip 0.1.
9. 결과: LDPC(128,60)에서 BP 대비 FER 10⁻²에서 약 `0.2dB` 이득, Polar(128,64)에서 CrossMPT 대비 `0.2dB`, ECCT 대비 `0.3dB` 이득; 단 LDPC(408,204)/(816,408) 등 긴 코드에서는 여전히 BP 대비 열세.
10. 한계: HW 미구현·시뮬레이션만, 임베딩 차원(128) 제약으로 긴 코드에서 성능 갭 존재, 완전 신경망 구조라 NAND ECC의 min-sum HW 파이프라인과 이질적.

| soft syndrome(SPA 기반) 계산 | `decoder.cpp` `C2V_Cal()` | SPA 형태의 check-node 연산이라는 점에서만 유사, min-sum 근사와 무관 |
