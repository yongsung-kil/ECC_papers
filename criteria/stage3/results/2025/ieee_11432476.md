### B. 알고리즘 요약

1. 시스템: 다운링크 NOMA, K 사용자 이종 blocklength(`n_k≤n_{k+1}`), BPSK superposition, 강한 사용자는 첫 `n_1` 심볼만 받고 조기복호. rate R=5/11(≈0.4545), length 242~2288.
2. 문제: 기존 NOMA SIC은 강한 사용자가 전체 프레임 수신을 기다려야 해 지연 발생, 이상적 Gaussian signaling 가정의 early decoding을 실제 부호로 구현 불가.
3. 모델: 2-user GBC, 미수신 심볼의 채널 LLR을 0으로 두고 조기복호.
4. 핵심기법 1단: GC-LDPC의 local code + global CN 구조를 확장한 partial globally coupled(PGC) base matrix `B_PGC`(식4) 설계 — 인접 두 local code만 잇는 부분 global check `B_PG^i` 도입.
5. 핵심 식: rate `R=1-(Σl_i + t-1 + l_G)/Σm_i`(식5), 부분 결합으로 일부 local code만 수신해도 message 교환 유지가 핵심.
6. 핵심기법 2단: early-decoding SIC 3종 — TIN(baseline), E-H-SIC(경판정 후 간섭 제거), E-S-SIC(function node로 두 디코더 간 extrinsic 반복 교환).
7. 구현 디테일: 부분 parity-check(PC) 메커니즘 — 수신된 local code에 해당하는 submatrix만 검증해 조기복호 허용(Algorithm 1,2), 5G NR base matrix 12×22 크기·CPM(lifting) 정합.
8. 최적화 절차: E-S-SIC outer iteration 최대 6, 단일 디코더 min-sum 최대 20 iteration.
9. 결과: (α,n) 설정별 E-S-SIC PC가 E-H-SIC/TIN 대비 4.3/5.1 dB, 1/4 미만 심볼만 SIC 시 약 3 dB, 5G NR LDPC 대비 최대 2.4~2.5 dB 개선.
10. 한계: HW 미설계·시뮬만, 다중사용자 NOMA SIC/조기복호 전제, 작은 lifting의 short cycle·유한길이 성능은 future work.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| PGC base matrix / 부분 global check 구조 | ecc_top.cpp Load_PCM() | H-matrix 구조 로드에 닿으나 NOMA 다중코드 결합 구조라 단일 codeword ECC와 불일치 |
| 부분 parity-check로 조기복호 | partial_CRC.cpp partial_crc_HW_T4() | 부분 검증으로 조기종료란 점만 유사, 목적(간섭제거 vs 수렴판정)은 상이 |
| min-sum 반복복호 | decoder.cpp LDPC_Decoder(), CNU_Update_New_Mag() | 동일 min-sum 계열이나 다중디코더 SIC 스케줄이라 재사용 어려움 |
| 두 디코더 extrinsic 교환(E-S-SIC) | 대응 없음 | 다중사용자 간섭제거 전용, NAND 단일 복호에 대응 없음 |
```

적용 가치: **낮음** — 다중사용자 NOMA 이종 blocklength/SIC early-decoding에 특화된 부분 결합 구조로, NAND 단일 codeword 완전수신 binary QC-LDPC ECC에 이식할 요소가 사실상 없음.
