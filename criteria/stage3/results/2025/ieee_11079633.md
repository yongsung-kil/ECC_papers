### B. 알고리즘 요약

1. TLC NAND 채널, AWGN 기반 min-sum LDPC 디코더 대상. 검증 코드는 `(18656,16748)` QC-LDPC(rate 0.9, 2KB page 상당)와 `(2304,1152)` WiMAX(rate 0.5).
2. Absorbing set 내 잘못된 LLR이 순환 전파되며 수렴을 막아 error floor를 유발하고, quantization은 이를 악화시킴 - 기존 uniform/quasi-uniform quantization만으로는 부족.
3. 관찰: 정확히 복호된 노드는 initial LLR과 APP LLR의 부호가 98% 일치, check node LLR과 APP LLR 부호는 89.3% 일치 - 부호 불일치를 absorbing set 진입 플래그로 사용.
4. Step 1(초기 LLR 최적화): `sign(Λk[lbit]) × sign(p[lbit]) < 0` 이고 `ncheck[lbit] ≤ wcol[lbit]/2` 이면 `p[lbit] = a` (상수)로 리셋.
5. 식의 의미: 부호 불일치 + unsatisfied check node가 column weight 절반 이하 = absorbing set 내부 특유 패턴(cycle 밖 unsatisfied node가 적음) 검출 조건.
6. Step 2(check node LLR 최적화): `sign(Λk[lbit]) × sign(r[s,lbit]) < 0` 이면 `r[s,lbit] = b` (상수)로 리셋 - Step1에서 잘못 satisfied 판정된 check node의 메시지를 교정.
7. 시작 시점 결정: 매 iteration 오류 비트 수를 추적해 더 이상 감소하지 않는 지점(실험상 29번째 iteration, Imax=32)부터 AsLDPC 활성화 - 이미 정정 가능한 비트의 오염 방지.
8. 파라미터 a, b는 "controlling variables" 방식 시뮬레이션으로 탐색, 두 코드 모두 a=b=0으로 최적화됨. (q+1)-bit quasi-uniform quantizer(선행연구 [18]) 기반 위에 구현, LUT로 fixed-point 변환.
9. 결과: quasi-uniform 대비 FER 58%(QC-LDPC)/48.1%(WiMAX) 감소, BER 60.7%/38.2% 감소, 저장공간은 uniform 6-bit 대비 11.9~17.4% 절감(오버헤드는 원 알고리즘 대비 1.76~2.05%만 증가), WiMAX 기준 Eb/N0=4.1dB에서 decoding latency 17% 감소.
10. 한계: HW 구현/합성 없이 C 시뮬레이션(floating/fixed-point)만 수행, a/b/m 파라미터가 H-matrix마다 재탐색 필요, 두 코드에서만 검증되어 일반화 근거 제한적.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Absorbing set 인식 후처리(Step1/Step2 LLR 리셋) | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | late-iteration에서 min-sum 루프 내 조건부 LLR 재설정 로직 추가 지점 |
| 부호 불일치 검출(초기 LLR vs APP LLR) | `decoder.cpp` `VN_Cal_HD()`, `decoder.h` `Get_VNU_*` | VN 갱신 시 APP과 initial LLR 부호 비교 로직 삽입 위치 |
| check node LLR 조정 | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | CN 메시지 갱신 후 조건부 리셋 적용 대상 |
| AsLDPC 시작 iteration(m) 결정 | `ecc_data.h` `PARAM_LLR`, `decoder.cpp` `Get_VNU_Table_Idx()` | iteration별 threshold 테이블에 시작 시점(m) 추가 가능 |
| flag 저장(1-bit per VN) | `ecc_data.h` (구조체 확장) | 대응 없음(신규 flag 배열 필요, 기존 구조체에 없음) |
| (q+1)-bit quasi-uniform quantizer, LLR 양자화 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()` | soft-read LLR 테이블과 연계 가능 |

> 적용 가치: **높음** — 기존 min-sum 루프 변경 최소화(O(n) 조건 판정 + 1bit flag)로 error-floor와 waterfall을 동시에 개선하고 iteration/latency를 낮추는 저비용 후처리이며, Prime ECC의 decoder.cpp 이터레이션 루프에 직접 삽입 가능한 구조.
