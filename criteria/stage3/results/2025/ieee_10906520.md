### B. 알고리즘 요약

1. 대상은 5G NR QC-LDPC 인코더로, base graph BG1/BG2를 lifting size `Z`로 확장한 PCM 기반 systematic 부호를 생성한다.
2. 문제: 기존 인코더는 배럴시프터, base graph 계수용 RAM, 다중 사이클에 의존해 throughput과 면적이 비효율적이다.
3. 부호구조는 submatrix `A,B,D,E`로 나뉘고, `B`는 dual-diagonal 구조를 가져 parity 계산을 단순화한다.
4. 핵심: `H C^T = 0^T`에서 `Px^T = B^{-1} A S^T`, `Py^T = E Px^T + D S^T`로 parity block을 유도한다(식 7,8).
5. `A S^T`는 4개 성분 `λ_i`(shifted message block의 modulo-2 합)로 표현되며, `B` 4종류별로 `Px1~Px4`를 폐형식으로 계산(식 11–14).
6. `Py`는 `Px` 확정 후 모든 block을 동시 계산(식 15).
7. 핵심 트릭: MATLAB 기반 VeCoGen이 base graph별 맞춤 Verilog를 생성, on-the-fly 시프트 대신 HDL 생성 단계에서 wire index를 재배치해 배럴시프터/RAM을 제거한다.
8. base graph 희소성을 활용해 nonzero block만 XOR 트리로 처리, XOR 연산을 BG1 1012Z→265Z(74% 감소), BG2 420Z(65% 감소)로 줄인다.
9. 결과: 입력·출력 레지스터 사이가 완전 조합회로가 되어 단일 클록에 codeword 완성, Zynq UltraScale+ ZCU111 FPGA에서 SOTA 대비 최고 throughput 달성(본문 서술).
10. 한계: 5G NR 표준 base graph 전용, 정정능력/복호와 무관하며 gatecount·throughput 수치는 표에만 존재.
