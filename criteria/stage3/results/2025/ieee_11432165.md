### B. 알고리즘 요약

1. 시스템: AWGN/BPSK 채널용 short-length binary QC-LDPC, base(model) matrix + right-circulant permutation `P`로 구성, rate 0.52~0.67·length 256~1687.
2. 문제: short block에서 girth<8이면 short cycle이 message correlation을 유발해 수렴 지연·error floor 발생, 임의 열가중치용 girth≥8 유연 구성법 부재.
3. 모델: 정수 수열 이론(Behrend 수열 일반화) `S(n,k,d,λ)`를 정의, 원소 norm 조건으로 distinct 보장(Lemma 1,2).
4. 핵심기법 1단: model matrix `M`(1행 랜덤 `b_1j`, 2~λ행은 `i·b_1j`)에서 `H=[P^{b_ij}]` 구성, Theorem 1로 Tanner graph가 4-·6-cycle-free임을 증명 → girth≥8.
5. 핵심 식: circulant order `p ≥ max{b_ij}+1`, Type-I 6-cycle 부재 조건 `(n-l)a=(n-m)c+(m-l)b (mod p)`를 Lemma 2가 배제함이 핵심.
6. 핵심기법 2단: 열가중치 3·4용 search-guided 구성, 6-cycle 조건 9개(C1~C9) 중 회피해야 할 C2(3항 등차수열 금지)·C6·C7만 남김.
7. 구현 디테일: Algorithm 1이 C2/C6/C7 미충족 증가수열 생성, Algorithm 2가 여러 후보 중 8-cycle(또는 6-cycle) 최소인 `H` 선택.
8. 최적화 절차: 실패 시 sequence 최대값 상향 재시도, 엄격 rate/length 제약 시 girth-6(6-cycle 최소)로 타협.
9. 결과: CCSDS TC (512,256)/(256,128) 대비 FER 10^-5에서 0.1/0.25 dB, TD·ES girth-8 short code 대비 0.25 dB 개선, girth≥8·열가중치≥4는 BER 10^-6까지 error floor 없음.
10. 한계: HW 미구현(NMS/OMS 시뮬만), AWGN best-case 가정, short-length 저~중rate 우주/IoT 타깃.
