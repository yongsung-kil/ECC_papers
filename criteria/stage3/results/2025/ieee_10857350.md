### B. 알고리즘 요약

1. 대상은 hypergraph product(HP) 기반 variable degree-3(dv-3) QLDPC 코드(CSS 코드, HZ/HX parity check matrix), 채널은 binary symmetric channel(BSC), 예시 코드는 B1[[882,24]], B3[[1270,28]], BB[[288,12,18]] 등.
2. 풀려는 문제: dv-3 QLDPC는 정정능력은 좋으나 낮은 variable degree로 인해 반복복호 시 error-floor가 심하고, 양자 하드웨어의 엄격한 latency 제약과 충돌.
3. 핵심 모델: classical trapping set(고정점)과 "quantum trapping set"(=symmetric stabilizer, 대칭 구조의 degenerate error) 두 종류의 harmful configuration을 정의해 반복복호 실패를 특성화.
4. 핵심 기법 1단: Two-Bit Bit Flipping(TBF) — variable node를 4상태(00/01/10/11, MSB=값 LSB=strong/weak), check node를 4상태(0old/0new/1old/1new)로 표현해 기존 BF(1bit) 대비 세밀한 갱신 규칙 `Ψ(w_v^ℓ, χ1^ℓ(v))` 제공.
5. 핵심 식: 각 TBF 디코더는 `D=(Ψ,f)`, 9개 플래그 `f=(I_Δv,I_Δc,W110,W200,...,W020)`로 정의되며, 이 플래그가 특정 unsatisfied-check 패턴에서 flip 여부를 세밀 제어해 trapping set 내 오실레이션(진동)을 방지.
6. 핵심 기법 2단(확장): 단일 TBF가 아니라 `D_{ND}={D1,...,D_ND}` 여러 TBF 디코더를 병렬(collective) 배치, Algorithm 3(탐욕적 선택)으로 weight-t까지의 오류패턴을 모두 교정하는 최소 디코더 집합을 순차 생성.
7. 부수 트릭: 두 circulant 행렬(A, B)에 서로 다른 Ψ 함수를 적용해 비대칭성을 유도, symmetric stabilizer((6,0) trapping set) 내 오류를 교정 가능하게 함(D9, D10 등).
8. 최적화 절차: 별도 학습 없이 weight-3~5 오류패턴에 대해 511개 후보 디코더를 전수 시뮬레이션하여 FER 최소 디코더를 그리디하게 선택(Algorithm 3).
9. 결과: normalized min-sum(nMS) 대비 error-floor 영역에서 D24(24개 디코더 집합)가 crossover probability 0.01에서 로지컬에러율 약 1000배 개선, 평균 반복횟수 약 2.5회(nMS는 1.2회이나 반복당 latency는 TBF가 더 낮다고 주장); BB 코드에서 SSF 대비 threshold 약 3% 달성(SSF는 threshold 없음). TBF/BF는 O(n) 복잡도로 변수노드당 카운팅+XOR, MS는 dv회 덧셈+최소값 연산이 필요해 TBF가 연산은 더 단순(Table IV).
10. 한계: 시뮬레이션(BSC, ideal circuit) 결과뿐이며 HW 설계 없음, dv-3/dc-6 특정 구조(6-cycle grid, symmetric stabilizer)에 특화되어 일반 QC-LDPC/고전 binary LDPC로의 일반화는 논문 범위 밖(저자도 향후 과제로 명시).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Two-Bit Bit Flipping (hard-decision, 2bit 상태) | `decoder.cpp` `VN_Cal_HD()` | Prime ECC의 HD(hard-decision) 경로와 개념적으로 유사하나 4상태 TBF 로직은 별도 구현 필요, 이식 난이도 중 |
| 병렬 앙상블(collective) 디코더 구성 | 대응 없음 | 여러 디코더 병렬 실행/선택은 Prime ECC의 단일 파이프라인 구조와 맞지 않음 |
| Trapping-set 기반 error-floor 분석/설계 | `ecc_top.cpp` `Load_PCM()` | 양자 stabilizer 코드 특화 분석이라 QC-LDPC H-matrix 설계에 직접 대응 어려움 |
| nMS/layered nMS 비교 대상 | `decoder.cpp` `CNU_Update_New_Mag()` | 이미 보유한 min-sum 계열과 비교 기준일 뿐 신규 기여 아님 |

적용 가치: 낮음 — 양자 QLDPC(stabilizer 코드) 전용 이론/시뮬레이션 연구로 NAND binary QC-LDPC의 H-matrix·HW 구조와 직접 대응이 없으나, hard-decision 2bit 상태 기반 bit-flipping의 error-floor 대책 아이디어는 개념적으로 참고 가능.
