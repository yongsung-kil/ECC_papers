### B. 알고리즘 요약

1. LEO 위성 inter-satellite FSO 링크용 BIAWGN 채널에서 regular/irregular LDPC 코드의 설계·최적화를 돕는 Density Evolution(DE) 분석 논문.
2. 기존 Gaussian Approximation(GA)은 check node degree `dc`가 크거나 입력 메시지가 작을 때(저 SNR) check node 출력 분포를 부정확하게 근사하는 문제가 있음.
3. check node 업데이트 `u = 2tanh^-1(prod tanh(vi/2))`를 `phi` 함수로 재표현하면 `S = sum phi(|vi|)`가 되며, 이 `S`의 분포를 정확히 근사하는 것이 목표.
4. 핵심 기법: `S`를 평균·분산이 같은 Gamma 분포로 근사(Gamma-based method). Gamma 분포는 로그정규분포보다 가산성(additivity)이 좋아 `dc`가 커도 정확도 유지.
5. `S`가 spiky한 경우(입력 메시지가 크고 `dc`가 작을 때)는 Gamma 근사 오차가 크므로, skewness 차이 기반 가중치 `c`로 Gamma와 GA를 혼합(MGA, mixed Gamma approximation).
6. irregular 코드 확장: 입력 메시지를 variable node degree 분포에 따른 Gaussian mixture로 취급, k차 모멘트를 lookup table 기반으로 계산.
7. 구현 시 1차원 lookup table(`f1, f2, f3` 함수)로 각 order moment를 미리 계산해 실시간 연산 부담을 줄임(교과서적 numerical technique).
8. 응용: 기존 GCA(Gaussian Capacity Approximation)의 `dc-1`차 수치적분을 MGA 기반 분포로 대체해 1차 적분만으로 근사 가능(복잡도 절감).
9. 결과: regular LDPC 9종 (dv,dc) 조합에서 noise threshold 오차를 GA 대비 최대 0.1035dB→0.0377dB, 평균 0.0560dB→0.0290dB로 감소. KL divergence도 0.2 이하로 개선.
10. 한계: 순수 시뮬레이션/이론 분석뿐이며 HW 미설계, 실제 디코더 알고리즘이 아니라 코드 설계 단계의 threshold 계산 도구. SC-LDPC 확장은 future work로 미구현.
