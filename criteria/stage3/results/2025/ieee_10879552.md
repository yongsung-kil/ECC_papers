### B. 알고리즘 요약

1. 5G NR LDPC(QC-LDPC, base graph 1/2, rate 1/3~22/24, K=8448 대표, AWGN+BPSK 채널)의 초저해상도(2~4bit) 복호 성능 개선을 다룬다.
2. 기존 mutual-information 최대화 양자화 설계는 노드 degree가 다양한 5G 코드에서 정렬(alignment) 부담이 크고, LUT 기반 복호기는 다단 압축 손실이 발생하는 문제가 있었다.
3. 채널은 AWGN, VN/CN 메시지를 이산 확률변수 `T`로 모델링해 density evolution으로 확률분포를 추적하는 probabilistic memory model을 사용한다.
4. 핵심 기법 1: **정렬영역(alignment region)** `α⋆(n)`으로 메모리 위치들을 묶어 공통 양자화기 `Q_a`를 information bottleneck(IB)으로 설계, `I(X̄;T̄)` 최대화.
5. 핵심 식 `L(x|y) = Σ sgn(L_k) φ⁻¹(Σφ(|L_k|))` (식 7)은 CN 갱신의 정보최적 결합규칙이며, min-sum은 이의 근사(식 9,10)이다.
6. 핵심 기법 2: **CN-aware VN 양자화** — VN 양자화기 설계 시 후단 CN 갱신까지 고려해 `I(X;T^c)`를 최대화하도록 임계값을 결정(§V), 기존 CN-unaware 대비 최대 0.2dB 개선.
7. 구현 디테일: 양자화 임계값은 대칭 제약(`τ_k=-τ_{|T|-k}`), 정수 표현·LUT 기반 재구성, 2bit에서는 thermometer 코드 인코딩 생략 가능.
8. 학습/최적화: schedule 최적화(Alg.9)는 매 레이어 갱신 시 mutual information 이득 `ΔI` 최대인 레이어를 우선 선택하는 정적 스케줄 구성.
9. 결과: 정렬영역 개선으로 최대 0.4dB, CN-aware 설계로 추가 0.2dB, row-layered 스케줄 최적화로 평균 iteration 최대 35% 감소, 2bit 복호가 4bit 대비 면적효율 2배.
10. 한계: 5G 표준 코드·AWGN 채널 전제, 실측(칩 제작) 없이 gate-count 추정 기반 ASIC 합성/시뮬 수준 검증에 그침.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 정렬영역 기반 양자화기 설계(IB) | `channel.cpp` `Set_LLR_Th()` | soft-read LLR 임계값 설계 참조 가능, 단 설계 방법론 자체는 오프라인 최적화이며 코드 이식은 부분적 |
| CN-aware VN 양자화 | `decoder.cpp` `Get_VNU_Table_Idx()`, `ecc_data.h` `PARAM_LLR` | iteration별 LLR 테이블 설계 방식과 유사한 목적, 설계 알고리즘만 참조 가능 |
| min-sum CN 연산 (식 9,10) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | 이미 Prime ECC가 min-sum 보유 → 중복 기여 |
| row/column-layered 스케줄 최적화 | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | 정적 스케줄 재배열 아이디어는 Dual-Update 스케줄과 유사 목적이나 5G 특화 |
| rate-compatible 정렬영역 확장 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 특정 QC-LDPC 고정이라 다중 rate 지원 구조 자체가 대응 없음 |
| block/row-parallel HW 아키텍처 | 디코더 HW 모델 (z=32, HCU, GT) | shift network·partial row-parallel 개념은 Prime ECC의 z_sb=32 병렬구조와 유사하나 5G 전용 설계 |

> 적용 가치: 중간. NAND 표준 코드와 무관한 5G 표준 코드 기반이지만, coarse quantization 설계 방법론(정렬영역, CN-aware 양자화)은 Prime ECC의 soft-read LLR 임계값/테이블 설계에 참조할 만한 일반적 기법이다.
