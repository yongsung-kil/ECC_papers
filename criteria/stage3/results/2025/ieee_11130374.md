### B. 알고리즘 요약 (10줄 내외)

1. 시스템은 K개 단일안테나 UE, M개 안테나 BS의 업링크 massive MIMO이며, 채널은 I개 서브블록으로 나뉘고 서브블록 간 1차 AR(auto-regressive) 프로세스로 시간에 따라 변화(channel aging)한다.
2. 문제: 채널 에이징으로 CSI가 outdated되어 MIMO 검출 error floor가 높아지는데, 기존 detection-aided 채널추정은 검출된 심볼만 이용하고 복호 정보를 활용하지 못한다.
3. 채널 모델: `h[i] = Λh[i-1] + Λ̄e[i]`(AR 프로세스), 파일럿+데이터 전송 프레임, Q-QAM 변조.
4. 핵심 기법 1단(부호): GC-LDPC(globally-coupled LDPC) 부호 — I개의 독립적 로컬 base matrix `Blocal_i`와 이들을 연결하는 global matrix `Bglobal`로 구성된 base matrix `BGC`.
5. 부호 설계: PEXIT(protograph EXIT) 분석으로 global 서브행렬 `Bgp`를 최적화하고, cycle length·EMD(extrinsic message degree) 기준의 간소화 EMPEG(error minimization PEG) 알고리즘으로 lifting factor(exponent matrix) `Pgp`를 최적화 — 5G 표준 base graph(BG1/BG2)를 재활용해 탐색공간을 대폭 축소.
6. 핵심 기법 2단(채널추정): Dec-Aided KF(Kalman filter) — 로컬 복호 성공 시 hard-decision, 실패 시 soft-decision(LLR 기반 posterior 확률)으로 복원한 데이터 심볼을 KF 관측모델에 재대입해 채널 추정 오차를 줄이고, 모든 서브블록 처리 후 로컬 복호 실패분에 한해 global 복호를 트리거해 error floor를 낮춤.
7. 부수 트릭: LLR 계산은 max-log 근사(`min` 연산으로 log-sum-exp 근사), 채널추정 오차 공분산 `ΣH`를 LMMSE 데이터 검출식에 반영.
8. 결과 수치: Q=4, RGC=0.664 조건에서 제안 기법이 detection-aided 대비 FER 10^-3에서 2.0dB 이득; global decoding 사용 시 local-only 대비 0.6~0.8dB 추가 이득; 부호 탐색공간을 110,055개에서 11개로 축소하면서도 성능 우세.
9. 학습/최적화 절차: PEXIT 문턱값 최소화로 global matrix 선택, EMPEG로 순환길이/EMD 기준 exponent 최적화(둘 다 부호 설계 단계 최적화이며 온라인 학습 아님).
10. 한계: MIMO 무선 페이딩 채널·Kalman filter 채널추정 특화 시스템이며 HW 구현 미제시, 복호기 자체는 표준 5G BP/min-sum 계열로 알고리즘적 신규성 없음(부호 구성·시스템 결합이 기여의 핵심).

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| GC-LDPC 부호(global+local base matrix) 구성 | `ecc_top.cpp` `Load_PCM()` | H-matrix 구조 자체를 이식하려면 재검증 부담 큼; Prime ECC는 단일 QC-LDPC base+circulant 구조로 global/local 2단계 개념 없음 |
| PEXIT 기반 부호 최적화, EMPEG lifting 설계 | 대응 없음 | 부호설계 단계 도구이며 Prime ECC 코드베이스는 고정 H-matrix 로드만 담당 |
| 로컬/글로벌 2단계 복호(local decode 실패 시만 global 복호) | `decoder.cpp` `LDPC_Decoder()` | 조기종료(Partial CRC) 개념과 유사하나, GC-LDPC의 "부분 성공 시 해당 블록만 correct로 고정하고 나머지만 재복호"하는 구조는 대응 없음 |
| Kalman filter 기반 채널추정 되먹임 | 대응 없음 | NAND는 무선 페이딩 채널 추정이 아닌 read-channel(RBER/threshold)이라 무관 |
```

적용 가치: **낮음** — MIMO 무선 채널 추정 결합형 부호 구성 논문으로, 복호 알고리즘 자체의 신규 기여가 없고 시스템 모델이 NAND read-channel과 근본적으로 달라 이식 가치가 낮음.
