> 총평: 무선 계층변조(HM) 다중 데이터스트림 전송에 특화된 protograph LDPC 기반 SC coding/ILC decoding/EXIT 분석 기법으로, NAND 단일스트림 binary QC-LDPC ECC와는 목적·채널·부호 구조가 근본적으로 달라 이식 가치가 낮음.

### B. 알고리즘 요약

1. `M`-ary QAM 계층변조(HM) 기반 시스템으로 `L`개의 서로 다른 우선순위 정보 스트림을 protograph LDPC(P-LDPC)로 각각 부호화 후 하나의 심볼로 매핑, AWGN 채널을 가정.
2. 기존 BIC-HM은 스트림 간 독립적이라 저우선순위 스트림 성능 개선이 고우선순위 스트림 희생을 요구하는 트레이드오프 한계가 있음.
3. 제안 SC coding: layer-1(최고우선순위) 정보비트 일부를 나머지 `L-1`개 layer 정보스트림에 다중화(multiplex)하여 layer 간 의존관계 형성, lookup table로 매핑 관리.
4. 제안 ILC decoding: layer-1을 먼저 BP 복호 후 실패 시 다중화된 정보비트들의 사후 LLR 평균 `la1v,te = ||l1mul||1/(p1+...+pL-1)`로 조기종료 판단(`DBP_t = |la_t - la_{t-1}| < lpre`).
5. 조기종료 임계값 `lpre`으로 반복 간 LLR 변화가 작으면 복호 중단, 이후 layer-i(`i=2..L`) 복호기들이 병렬로 BP 수행하며 layer-1과 외부반복(outer iteration)으로 soft info 교환.
6. VND multiplexing: irregular P-LDPC의 최저-degree VN에 다중화 정보비트를 매핑하여 layer-i 수렴성능을 개선(고degree VN이 저degree보다 수렴 우수하다는 원리 이용).
7. 구현 디테일: 다중화 정보비트 위치는 `Hp`의 컬럼 순열(내림차순 degree 정렬) 및 시스터매틱 `H'p=[IH|A]` 구조로 결정, full-rank 아니면 인터리브 버전 `HInt_p` 사용.
8. ILC-EXIT: 기존 EXIT을 L개 스트림 동시분석·조기종료(Step 5,8) 가능하도록 확장한 MI 분석 알고리즘, `Iavg = E[IAPP(j) x Fmul(j)]`로 다중화 VN 평균 MI 계산.
9. 결과: layer-2 decoding threshold gain 약 0.19dB(R2=0.4), 0.16dB(R2=0.3) 개선; BER 1e-5에서 약 0.4dB 이득; 조기종료로 연산량(덧셈/곱셈/tanh) 저SNR 구간에서 유의미하게 감소.
10. 한계: 순수 시뮬레이션(이론+Monte-Carlo)뿐 HW 미설계, 무선 AWGN·계층변조 채널 전용 가정으로 NAND read 채널 모델과 무관, decoder는 full BP(tanh 기반)로 min-sum 계열이 아님.

### C. Prime ECC 관련 모듈 핀포인트

```
| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| ILC 조기종료(LLR 임계값 기반) | `partial_CRC.cpp` `partial_crc_HW_T4()` | 조기종료라는 목적만 유사, CRC가 아닌 LLR 통계 기반이라 메커니즘 상이 |
| Log-BP 복호(full tanh) | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum 근사이므로 full-BP는 대응 없음 |
| P-LDPC(protograph) 구조/부호설계 | `ecc_top.cpp` `Load_PCM()` | protograph 구성법 자체는 대응 있으나 계층변조 특화 다중화 구조는 대응 없음 |
| SC coding / VND multiplexing | 대응 없음 | 다중 데이터스트림 다중화는 NAND 단일스트림 ECC와 무관 |
| ILC-EXIT 이론분석 도구 | 대응 없음 | 코드 설계용 이론 툴, HW 시뮬레이터와 레이어 다름 |
```

적용 가치: **낮음** — 무선 계층변조 다중스트림 전송 특화 기법으로 NAND 단일스트림 binary QC-LDPC 디코더/부호설계와 채널·목적이 근본적으로 달라 이식 접점이 거의 없음.
