### B. 알고리즘 요약

1. 시스템: 고이동성 OTFS(N=16 subcarrier, M=128 slot, Δf=15kHz)에 LDPC-Hadamard concatenated code + BPSK/16~256QAM, 3GPP TDL-C(L=8 tap) 채널.
2. 문제: Doppler shift·delay spread의 doubly-selective 채널에서 기존 convolutional/polar/non-binary LDPC-OTFS의 유연성·복잡도 한계.
3. 모델: DD도메인 ISFFT/Heisenberg 변환으로 시변채널을 근사 시불변화, block-fading 가정.
4. 핵심기법1(부호): 외부 LDPC + 내부 Hadamard 2계층 concatenation, Tanner graph의 degree-1 leaf node에 Hadamard 직교제약, degree>1 node에 LDPC 제약.
5. 핵심식 의미: Hadamard order를 LDPC CN degree에 정렬해 SPC를 Hadamard로 대체, 수렴 가속·burst noise 내성.
6. 핵심기법2(구성): 저복잡도 QC-LDPC 구성 - H를 A/B/T/C/D/E 6블록 분할, T를 lower bi-diagonal로 만들어 인코딩 복잡도 `O(n+g²)≈O(n)`, block-circulant로 FFT 기반 곱 `O(n log n)`.
7. 부수트릭: 복호 시 CN은 Hadamard APP(Fast Hadamard Transform)로, VN은 LLR-BP `L(r_ji)=2tanh⁻¹(∏tanh(...))`로 반복 교환.
8. 최적화: PEG로 girth 확대(short cycle 회피), rate-compatible puncturing으로 1/12~1/6 rate, EXIT chart로 수렴 검증.
9. 결과: R=1/12에서 BER `10^-6`을 -0.4dB(AWGN)·1.1dB SNR(TDL-C)에 달성, 기존 QC-LDPC 대비 0.2dB gain, 120~500km/h 전속도에서 우세.
10. 한계: HW 미설계·MATLAB 시뮬만, 초저 code rate, block-fading 가정(Doppler>15% frame이면 열화), 무선 특화라 NAND 부적합.
