### B. 알고리즘 요약

1. 시스템: systematic linear block code `(n,k)`, H를 표준형 `H=[W_{(n-k),k} | I_{(n-k),(n-k)}]`로 제한, 학습대상은 오직 `W`(이진 행렬, 파라미터 수 `(n-k)×k`); BP 반복횟수는 학습 5회 고정(단기 iteration 저복잡도 지향, IoT/URLLC 타깃).
2. 문제: LDPC/블록부호 자동설계(ML 기반)는 최근 연구주제이나, 짧은~중간 길이 영역에서 학습 기법이 무작위 탐색보다 통계적으로 우월함을 엄밀히 검증한 사례가 부족하고, binary-valued weight의 미분 불가능성이 gradient 학습을 어렵게 함.
3. 핵심 가정: 훈련 시 채널을 AWGN이 아닌 "controlled error channel"(코드워드 내 오류 비트 수 `N_errors`를 고정)로 사용해 학습 신호의 유효성을 극대화, 평가 시에는 AWGN 사용.
4. 핵심 기법: 미분 가능한 BP를 RNN cell로 펼쳐(differentiable Tanner graph) 오토인코더로 구성, loss는 binary cross-entropy `ℓ(x̃)=log(1-x̃)`(all-zero codeword 학습).
5. 핵심 식: 이산 가중치 갱신 `w ← w + g(∇w,φ,w)`를 그래디언트의 부호(sign)만으로 결정하는 GQLA(Gradient Quantization LeArning) 최적화기 제안 — Mini-Batch GQLA(MB-GQLA)는 배치당 gradient sign을 Update Matrix(누적 카운터, threshold `T`)에 축적 후 임계값 도달 시에만 0/1 비트 플립.
6. 확장: 샘플 단위 gradient sign을 합산하는 Stochastic GQLA(S-GQLA)도 제안하나 MB-GQLA 대비 유의미한 성능차 없음; DSF(Differentiable Step Function, 실수값+pass-through gradient) 방식과도 비교해 유사 성능 확인.
7. 부수 트릭: BP 내 `arctanh` gradient를 pass-through로 대체해 gradient 전파 개선, 학습은 3회 iteration으로 진행하되 평가는 5회 iteration 사용(둘의 불일치가 오히려 성능에 유리).
8. 학습/최적화 절차: 하이퍼파라미터(스케일링 α, N_errors, threshold T, 초기 밀도 D) coarse search 후 fine search(코드 크기별 5회 학습), 무작위 탐색 대비 우월성 검증을 위한 CDF 기반 통계 프레임워크(Agresti-Coull 신뢰구간) 별도 제안.
9. 결과: (64,32) 부호에서 이전 연구[8] 대비 약 1dB 개선, (128,64) 부호에서 5회 iteration만으로 CCSDS LDPC(200회 iteration) 대비 동등 성능(iteration 40배 감소); 학습된 부호는 무작위 탐색 대비 girth 증가(VN 최대 8, CN 최대 6)·밀도 감소 경향.
10. 한계: 고SNR 영역에서 AR3A 등 SOTA LDPC 대비 완만한 기울기(error-floor 진입 가능성 시사), non-systematic 부호로의 확장 미검증, HW 미설계, 검증된 code length가 최대 128bit로 NAND ECC급(수천~수만 bit) 미검증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| BP 최적화 parity-check matrix(H) 학습 | `ecc_top.cpp` `Load_PCM()` | H-matrix 구성법이라는 개념만 대응, 학습 산출 H는 QC 구조·특정 크기(z=32) 제약이 없어 직접 대체 불가 |
| girth/degree 분포 개선을 통한 BP 성능 향상 | 대응 없음 | 논문은 BP(sum-product) 기준 최적화이며 Prime ECC의 min-sum 및 특정 col-degree(17, HCU 2개 col) 구조와 직접 연결되지 않음 |
| gradient quantization 학습 최적화기(GQLA) | 대응 없음 | 오프라인 신경망 학습 절차로 Prime ECC의 실시간 디코더/HW 로직과 무관 |

> 적용 가치: 낮음 — 짧은 길이(≤128bit) BP 최적화 code-design 논문으로 NAND용 고rate·대용량 QC-LDPC 구조와 규모·목적이 달라 직접 이식 근거가 약함.
