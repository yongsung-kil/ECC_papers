### B. 알고리즘 요약

1. 시스템: 코히런트 광통신 QAM/PS-QAM, BICM 기반 다중 B-MC(binary-input memoryless channel) 프레임, 부호어 길이 `n=21600` 예시 구성.
2. 문제: 기존 MLC(TSD, two-stage decoding)는 변조차수 `m`에 종속적으로만 복잡도 절감이 가능해, 저차 변조에서는 절감 효과가 감소.
3. 채널모델: AWGN 하 BICM으로 변환된 병렬 B-MC `W:{0,1}→R`.
4. 핵심기법: CP-XOR `T⊕`로 `d`개 B-MC를 1개 비신뢰 채널 `W1`과 `d-1`개 신뢰 채널 `Wj`로 변환(`I(W1)≤I(W)≤I(Wj)`), SDD는 `W1`에만 적용.
5. 핵심식: 비신뢰 LLR `λ1 = l1 ⊞ l2 ⊞ ... ⊞ ld`(boxplus 연산), 신뢰 LLR `γj = lj + (-1)^z1(...)`— log-domain 재귀 계산으로 계산량을 `O(2^d)`에서 `O(d)`로 축소.
6. 확장: systematic CP-MLC는 A-XOR `A⊕`를 추가해 정보비트를 보존, PAS(probabilistic amplitude shaping)의 component code로 사용 가능.
7. 구현 디테일: outer code로 zipper/staircase code, inner code로 (7,4,3)-Hamming 예시; outer decoder는 LDPC(SPA, sum-product algorithm) 사용.
8. 복잡도 지표 `η = (1-R1)(d̄c-ν)I·(d/(d-1+R1)) + P`로 정의, 코드 파라미터 최적화는 별도 학습절차 없음(수치 시뮬레이션 기반 비교).
9. 결과: 4/16/64/256QAM에서 Δ=3.0dB 기준 복잡도 최대 54%(4QAM), PS-QAM에서 Δ=2.0dB 기준 최대 34% 절감.
10. 한계: 알고리즘/시뮬레이션 수준(HW 미설계), 코히런트 광통신 특유의 QAM 성상·PAS 프레임 전제로 NAND 채널 모델과 무관.
