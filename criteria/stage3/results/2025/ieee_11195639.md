### B. 알고리즘 요약

1. 시스템: Tanner code `T(G,C0)` — `(c,d,α,δ)`-bipartite expander `G`의 각 우측 정점에 최소거리 `d0`인 선형 inner code `C0`를 배정해 정의되는 이진 선형 부호.
2. 문제: expander code(inner code가 parity-check인 특수경우) 선형시간 복호는 활발히 연구됐으나, 일반 inner code `C0`에 대해 `δd0`의 어느 조건에서 선형시간 복호가 가능한지 최선 조건(`δd0>1` 필요, `δd0>3` 충분)에 간극 존재.
3. 핵심 가정: 부호는 이진 선형(`F2`)이며 채널은 명시된 노이즈 모델 없이 임의의 `≤αn` bit 오류(worst-case Hamming error)로 가정.
4. 핵심 기법 1단: 무작위 `RandFlip` — 각 unsatisfied check `v`가 자신의 로컬 디코딩 결과 `wv=Decode(xN(v))`와 `dH(wv,xN(v))<d0/2` 조건에서 가중치 `(t-dH)/ct`(`t=d0/2`)를 비트에 투표, 확률적으로 flip.
5. 핵심 식: `E[dH(x',y)] ≤ (1-ε0δ/t)dH(x,y)` (Theorem 3.1) — 매 라운드 오류 개수가 상수배로 감소함을 보장, 조건은 `δd0>2`.
6. 핵심 기법 2단: 탈무작위화(`DeterFlip`+`DeepFlip`+`MainDecode`) — flip 확률 `pi`값을 유한집합 `W`로 이산화하고, 후보 `q` 값들을 상수 개수만 전수탐색해 결정론적 `O(n)`시간 복호(복호반경 `αn`)를 구현.
7. 부수 트릭: Size-Expansion Function `fδ(k)`(LP로 정의)를 이용해 큰 집합의 expansion을 작은 집합으로부터 추론, 복호반경을 `αn`에서 `f_δ^{-1}(2/d0)αn`로 확장.
8. 결과(수치 아님, 정리): 기존(Cheng et al., RANDOM 2024) `δd0>3` 조건을 `δd0>2`로 완화, 결정론적 복호반경을 `d0/(2α(1+0.5cδ))n`에서 `αn`으로 개선; 최소거리 하한 `f_δ^{-1}(1/d0)αn` 증명(tight).
9. 학습/최적화: 해당 없음 (조합론적 증명, 파라미터 학습 없음).
10. 한계: 전부 이론적 증명이며 시뮬레이션/BER 실험, HW 설계, 구체적 rate·length·gatecount 수치 전혀 없음. 부호가 일반 expander/Tanner 구조라 QC-LDPC로의 구성적 변환이 별도로 필요.
