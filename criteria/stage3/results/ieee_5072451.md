### B. 알고리즘 요약

1. AWGN 채널, BPSK 변조를 가정하고 단순 다항식 성분 부호를 직렬 결합해 LDPC 부호를 구성하는 새 기법을 제안.
2. 기존 구조적(QC-LDPC) 설계는 길이/rate에 제약이 있고, 비구조적(PEG 등) 설계는 세밀한 granularity는 얻지만 HW 구현에 불리한 문제를 해결하고자 함.
3. 채널 모델은 표준 AWGN, decoding은 BP 기반 알고리즘(LLR-SPA 또는 normalized min-sum, NMS) 가정.
4. 성분 부호는 생성다항식 `g(i)(x) = 1 + x^ri` (ri=redundancy)를 갖는 매우 단순한 shortened cyclic 부호이며, `M`개 성분 부호를 systematic 형태로 직렬 결합해 하위삼각(lower triangular) 패리티검사행렬 `H`를 구성.
5. `ri`들을 서로소(coprime)·오름차순으로 선택하면 4-cycle-free를 보장하는 최대 길이 `n_max = r1r2 + Σr_j`(Corollary 1, 식(8))가 유도되어, 이 범위 내 임의 `n`에서 세밀한 길이 granularity 확보.
6. 인코딩은 LFSR 기반 다항식 곱셈 또는 병렬 SPC(single parity-check) 인코더 뱅크로 구현 가능(그림 3, 4); 하위삼각 `H`가 sparse라 back-substitution 인코딩도 저복잡도.
7. 최소거리 상한 `d_min ≤ 2^M`(Lemma 2)을 유도, systematic 직렬 결합 구조상 성분 부호를 점진적으로 추가/제거하면 rate-compatible 부호군(식 10) 형성 가능해 Type-II HARQ에 적합.
8. 별도 학습/최적화 절차 없음(순수 대수적 파라미터 선택 `ri`).
9. 결과: medium code(n=8208, k=7182, R=0.875)가 CCSDS 준최적 부호(n=8176, k=7156, FPGA 시뮬 50 iteration)와 거의 동일한 BER/FER 곡선; large code(n=10972)는 BER<10⁻⁷에서 error floor 발생, n=27430 부호(R=0.9)는 우수한 waterfall.
10. 한계: 시뮬레이션(소프트웨어)만 수행, HW 합성·gatecount 없음, CCSDS 대비 성능 우위가 아니라 "유사 성능+저복잡도 인코딩"이 주된 이점, NAND 특화 실험 없음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 직렬 결합 다항식 부호 H-matrix 구성 | `ecc_top.cpp` `Load_PCM()` | Prime ECC의 QC-LDPC 구조와 근본적으로 다른(비-QC, lower-triangular) 부호설계라 이식 난이도 높음 |
| lower-triangular 기반 저복잡도 인코딩(back-substitution/LFSR) | `encoder.cpp` `LDPC_encoder_4KB()`, `Generate_PCM_encoding()` | Prime ECC는 이미 dual-diagonal 인코딩을 사용 중이라 중복 기여, 구조 상이로 직접 대체 어려움 |
| NMS(normalized min-sum) 복호 | `decoder.cpp` `CNU_Update_New_Mag()` | 이미 Prime ECC가 min-sum 계열 사용 중이라 중복 |
| rate-compatible / HARQ용 incremental redundancy | 대응 없음 | NAND ECC는 고정 rate 사용, HARQ 개념 자체가 무관 |

> 적용 가치: 낮음 — 부호구조가 Prime ECC의 QC-LDPC와 이질적이며(직렬결합·비-QC), 정정성능 우위 없이 인코딩 저복잡도·rate 유연성이 주된 기여라 NAND 이식 필요성이 낮음.
