> 총평: DVB-S2 short frame MAT의 불규칙 row weight를 illegal 주소값으로 우회하는 표준 특화 구현 트릭으로, NAND binary QC-LDPC에 이식 가치 없음.

### B. 알고리즘 요약

1. 시스템: DVB-S2 BCH+LDPC FEC의 short frame(`N=16200`, 10 rate), eIRA(extended irregular accumulation) 부호, QPSK.
2. 문제: short frame 일부 rate(1/4,1/2,3/4,4/5,5/6)에서 H 행렬의 row weight가 불규칙 → 표준 디코더가 최대 row weight로 일괄 처리 시 zero를 nonzero로 오인해 복호 실패.
3. 발견: `H=[A|B]`에서 B의 기여 row weight는 항상 2, `(N-K)·row_weight = col기여 + 2`가 정수가 아닌 rate에서 불규칙 발생.
4. 저장구조: row-based MAT을 360행 단위 `q`-set으로 압축, block table(360열 블록 인덱스)+offset table(우측 오프셋)로 저장.
5. 핵심기법: 빈(vacant) 위치의 column 정보를 illegal(`col>16199`)로 세팅해 max-row-weight로 균일 계산하되 illegal이면 update/decode 스킵.
6. 식의 의미: block 값 `>45.99`이면 `(block-1)·360>16199`가 성립 → 빈 위치 block을 `46`으로 채워 illegal 보장(식8,9).
7. 구현 디테일: 별도 불규칙행 위치/weight 저장 없이 재구성된 MAT + 비교연산 1회만 추가.
8. 검증: Matlab, iteration 20, QPSK, `Eb/N0`는 표준 이상값 ±0.2dB 이내로 복원.
9. 결과: 관습적 max-row-weight 직접 복호는 성능 급락, 제안법은 DVB-S2 표준 성능에 근접하며 rate 변화에 둔감.
10. 한계: HW 미설계, 정정능력 개선이 아닌 표준 준수 복원, DVB-S2 MAT 저장구조 종속.
