### B. 알고리즘 요약

1. 시스템: binary-input AWGN 채널(BPSK ±1), irregular LDPC를 n-bit 균일양자화된 min-sum 복호기로 복호. rate 1/2, length 4096, `n`-bit(3~7bit 검토, 실험은 4-bit) 대칭 균일양자화(2^n-1 레벨).
2. 문제: 기존 설계법은 sum-product 알고리즘 기준으로 threshold-최적화된 degree distribution(예: LdpcOpt)을 구한 뒤 min-sum 등 저복잡도 복호기로 복호하는데, 이 degree distribution이 quantized min-sum에서는 더 이상 최적이 아님.
3. 핵심 가정: 메시지는 원점 대칭 균일양자화, quantizer는 최대 채널출력값(`max channel output`) 하나로 완전히 특정됨(정규 code는 mutual information 최대화 quantizer로 충분하나 irregular code는 아님).
4. 핵심 기법: check node 갱신을 `tanh^-1(Πtanh(m/2))` 대신 `min|m|·sign(m)` 근사(min-sum, 식 2)로 단순화하고, 이를 discretized(양자화) density evolution으로 확장.
5. 핵심 식: EXIT chart(메시지 오류율 기반)를 이용해 irregular code의 EXIT 곡선을 regular code(elementary EXIT curve)의 선형결합으로 표현, `λ(x)`가 elementary curve에 관여하므로 threshold 최대화는 반복 선형계획법(iterative LP)으로 풂.
6. 확장/공동최적화: `(Qc, λ(x), ρ(x)=x^{dc-1})` 트리플을 결합 탐색 — 고정 `dc`,`n`에서 threshold가 quantizer 최대출력값에 대해 unimodal, 그리고 `dc`에 대해서도 unimodal이라는 경험적 성질을 이용해 반복 탐색.
7. 부수 관찰: quantized 복호기에서는 degree-2 variable node가 상한이 걸린(capped) reliability 때문에 무한정밀도 복호기보다 더 해로워, degree distribution을 quantizer에 특화해 재설계해야 함.
8. 학습/최적화 절차: EXIT chart 기반 반복 선형계획법 + quantizer 파라미터에 대한 unimodal 탐색(GA/DE 아님, closed-form 반복 최적화).
9. 결과: 4-bit min-sum, rate 1/2, degree-15 제한 하에서 quantized-min-sum 최적화 코드(Q.M.S.T.O.)가 sum-product 최적화 코드(S.P.T.O.)보다 우수, 심지어 (3,6) regular code보다도 0.14dB밖에 차이 안 남(비교하면 S.P.T.O.는 훨씬 열화).
10. 한계: 균일양자화만 고려(비균일 양자화는 더 나을 수 있으나 미검토), rate 1/2·단일 code 사례만 실험, HW 미설계, degree 분포 설계는 이론+시뮬 수준.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 양자화 min-sum(check node min 근사) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC의 min-sum 연산과 동일 알고리즘 범주, quantizer 최적화는 개선 여지 |
| n-bit 균일양자화 LLR / quantizer 파라미터 | `channel.cpp` `Set_R_Offset()`, `Set_LLR_Th()`, `ecc_data.h` `PARAM_LLR` | soft-read 비용과 직결되는 LLR 양자화 설계, Prime ECC의 HD/2SD/3SD 임계값 최적화에 방법론 적용 가능 |
| degree distribution(quantizer 특화) 재설계 | `ecc_top.cpp` `Load_PCM()` | irregular degree 재설계는 고정 QC-LDPC base matrix 변경을 요구해 난이도 있음 |
| EXIT chart 기반 반복 LP 최적화 절차 | 대응 없음 | 코드 설계 툴체인(오프라인) 성격, 실행 코드에 직접 대응 모듈 없음 |

> 적용 가치: 높음 — Prime ECC가 이미 min-sum + soft quantization(2SD/3SD) 구조를 갖고 있어, 본 논문의 "복호기 양자화 방식에 맞춘 degree/quantizer 공동 최적화"라는 방법론을 LLR 임계값 테이블 및 향후 H-matrix 개선 설계에 직접 참고 가능.
