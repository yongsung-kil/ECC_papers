### B. 알고리즘 요약

1. 대상: structured LDPC(EG-LDPC), (1023,781) 및 (4095,3367), BPSK over AWGN, soft-decision 수신 magnitude `|rn|` 활용.
2. 문제: WBF 및 변형들은 성능은 좋으나 반복마다 real operation을 도입해 복잡도·지연이 큼, 표준 BF는 단순하나 hard-decision 성능 손실.
3. 핵심가정: AWGN에서 `|rn|`이 클수록 hard-decision `zn` 신뢰도 높음 → 큰 magnitude엔 큰 임계값(덜 flip), 작은 magnitude엔 작은 임계값(더 flip).
4. 핵심기법(MTBF): 반복 전 magnitude로 code bit를 `⌈γ/2⌉` 그룹으로 나눠 그룹별 threshold `Tn` 부여, 반복 중엔 unsatisfied check-sum 수 `fn`이 `Tn` 초과 시 flip(논리연산만).
5. 핵심식: 최소 threshold `⌈γ/2⌉`, 최대 `⌈γ/2⌉+⌊γ/2⌋-1`, `kα≤|rn|<(k+1)α`로 그룹 결정(`α`는 경험적 최적화).
6. 확장: flip 횟수가 홀수면 `Tn=Tn-1`(신뢰도↓), 짝수면 `Tn=Tn+1`로 동적 조정해 error flipping 억제.
7. 부수트릭: 초기화 시 `⌈γ/2⌉N` real 비교만 필요, `α`는 SNR 비의존·정밀도 둔감하여 WBF보다 적은 양자화 비트로 구현 가능.
8. 복잡도: WBF `M(ρ-1)`+반복당 `(N-1+ργ)` real op 대비, MTBF는 반복 중 real op 0, 다중 bit 동시 flip으로 빠른 수렴.
9. 결과: (1023,781)에서 WBF 대비 0.25dB·GDBF 대비 0.15dB 이득(BER 1e-6), 10/20 iteration 성능 거의 동일(빠른 수렴), IMWBF/AWMBF/PWBF보다는 성능 낮으나 복잡도 우위.
10. 한계: HW 미설계(시뮬만), column weight가 큰 부호에 적합하다고 명시, IMWBF/AWMBF 대비 정정성능은 열세.
