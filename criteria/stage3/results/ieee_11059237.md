### B. 알고리즘 요약

1. 대상 채널은 M×N ReRAM crossbar array이며, `(255,175)` EG-LDPC 코드(rate `0.69`, length `255`)를 사용, 각 셀 저항값에 Gaussian noise가 더해지는 read 모델(식 3).
2. 문제: crossbar 구조 특유의 sneak path interference(SPI, 식 4~6)로 인해 BP 기반 디코딩이 성능 저하되고 계산복잡도도 높아, ReRAM 환경에 맞는 저복잡도 디코딩이 필요.
3. 채널 모델: SPI 발생 시 셀 저항이 `Rs`(식 5)로 왜곡되며, sneak path 확률 `psp`(식 15)로 그 영향 정도를 정량화.
4. 핵심 기법 1단(HRB-IMLGD, Gallager-B 계열): 초기 신뢰도 `Rj(0)=±γ`, 매 반복마다 extrinsic info `Ej(k)=Σ(2σ_ij^(k)-1)`을 더해 갱신(식 7~9), syndrome이 0이면 종료.
5. 핵심 식: sneak path distance `d_ij^sp = e^{...} - psp`(식 16)로 셀별 sneak path 영향을 정량화해 초기 hard decision과 신뢰도(`βsp`, 식 17~18)에 반영 — SPI 영향을 받은 셀의 신뢰도를 낮춰 오류 정정 성능을 개선.
6. 확장(ESPR-IMLGD): Step 3의 초기 신뢰도 설정만 `IR0/IR1/기타` 3구간으로 세분화(식 19)해 성능 추가 개선.
7. 부수 트릭: `d_ij^sp` 계산에 필요한 지수함수를 1차 Taylor 전개로 근사(식 20~22)해 실수 연산을 상수 뺄셈+정수 비교 1회로 축소, HRB-IMLGD 대비 반복당 추가 연산은 정수 비교 1회뿐(Table I).
8. 별도 학습/최적화 절차 없음(파라미터 `βsp∈{3,4}`는 실험적으로 설정).
9. 결과: `σ=34, βsp=3`에서 ESPR-IMLGD가 HRB-IMLGD 대비 BER 4자리, SPR-IMLGD는 1자리 이상 개선. `βsp=4`에서는 SPR/ESPR-IMLGD가 BP 디코딩보다도 우수한 구간 존재. 평균 반복 횟수도 HRB-IMLGD보다 적음(Fig. 3).
10. 한계: 시뮬레이션(BER/FER/평균반복수)뿐이며 HW 설계·gatecount·throughput 없음. sneak path 채널 특유의 `psp`, `Rs` 등 파라미터가 ReRAM 저항 모델에 종속되어 NAND flash RBER 채널에는 직접 적용 불가.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| HRB-IMLGD 반복 신뢰도 갱신(식 7~9) | `decoder.cpp` `VN_Cal_HD()` (HD 계열) | Gallager-B류 hard-decision majority-logic 갱신은 Prime ECC의 HD 모드 VN 처리와 개념적으로 유사하나 min-sum 기반과는 다른 알고리즘 |
| syndrome check 조기종료 | `partial_CRC.cpp` `partial_crc_HW_T4()` (개념적 유사) | 매 iteration syndrome=0 검사는 유사한 조기종료 목적이나 CRC가 아닌 syndrome 자체 검사로 구현 방식 상이 |
| sneak path distance / 채널 특화 초기 신뢰도(식 16~19) | `channel.cpp` `Set_LLR_Th()` (개념적 유사) | ReRAM 특화 채널 모델로 NAND RBER 채널에는 그대로 대응 불가, 초기 신뢰도 설계 아이디어만 참고 가능 |
| iteration별 성능/반복수 비교 | `decoder.cpp` `LDPC_Decoder()` | 반복 루프 구조 자체는 Prime ECC 디코더 루프와 대응 |

적용 가치: **중간** — hard-decision 기반 저복잡도 majority-logic 디코더 개선이라는 알고리즘적 아이디어(신뢰도 초기화·갱신 세분화로 성능 개선)는 참고 가치가 있으나, ReRAM sneak path 채널 모델에 강하게 결합되어 있어 NAND 채널로 이식 시 채널 특화 부분(식 10~22)을 전면 재설계해야 함.
