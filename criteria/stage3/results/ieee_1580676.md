### B. 알고리즘 요약

1. 시스템: high-density magnetic recording을 겨냥한 regular structured LDPC 부호 설계, AWGN 및 EPR4(partial response) 채널에서 sum-product 알고리즘으로 디코딩.
2. 문제: cyclic/quasi-cyclic LDPC는 girth ≤ 12로 상한이 증명되어 있어(Fossorier), 매우 긴 코드에서 girth가 log(N) 관계를 못 따라가 성능이 떨어짐. 임의로 큰 girth를 갖는 regular structured LDPC 설계법이 부재.
3. 핵심 가정: check node 집합 `M`과 bit node 집합 `N`을 각각 크기 `m`(cardinality)인 부분집합들로 분할하고, 부분집합 간 연결을 `m x m` shift 행렬 `S`의 원소(모듈로-m shift)로 정의 — 이를 partition-and-shift LDPC(PS-LDPC)라 명명.
4. 핵심 기법 1단: Theorem 1(2t-cycle 조건)로 Tanner graph의 `2t`-cycle 존재 여부를 shift 행렬 `S` 안의 닫힌 경로(closed path)가 만족하는 `Cycle Condition`(모듈로-m 합)과 동치 관계로 환원.
5. 핵심 식 의미: 큰 Tanner graph의 girth 문제를 훨씬 작은 shift 행렬 `S`(크기 `J x K`, `J,K << M,N`) 위의 조합 문제로 축소해 girth 상한(Theorem 2)과 그 상한을 달성하는 구성(Theorem 3, Corollary 1)을 증명.
6. 핵심 기법 2단: Corollary 1을 이용해 "girth `g`를 갖는 큰 PS-LDPC 코드 설계"를 "girth `g-4` (or 이하)를 갖는 더 작은 PS-LDPC 코드 설계"로 재귀적으로 축소하는 Algorithm 1/2 (PEG(progressive edge growth)와 유사한 탐색적 shift 값 선택).
7. 부수 트릭: shift 행렬 `S`만 저장하면 되므로(파리티검사행렬 대비 훨씬 작음) 저장 메모리 요구량이 크게 감소한다고 언급(구현 관점 이점, HW 설계는 아님).
8. 최적화 절차: girth-감소 재귀 + Algorithm 1의 shift값 후보 탐색(도달 가능성은 조합론적 카운팅 부등식(14)으로 존재성 증명, PEG 방식으로 실제 값 탐색).
9. 결과: girth-8, rate-0.86, length-4536 PS-LDPC가 랜덤 LDPC(no-4cycle) 대비 고SNR에서 BER 약 0.6dB 이득(그림4, 랜덤코드는 error floor 도달, PS-LDPC는 미도달), EPR4 채널(girth-8, length 32760, rate 0.875~0.92)에서도 랜덤 대비 우수하고 finite geometry LDPC와 유사 성능.
10. 한계: 코드 구성/BER 시뮬 수준 논문으로 디코더 알고리즘(min-sum 등) 개선이나 HW 파이프라인·양자화·throughput 논의가 전혀 없음, magnetic recording 채널(AWGN/EPR4) 특화 검증.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| PS-LDPC shift matrix 기반 H-matrix 구성(girth 확보) | `ecc_top.cpp` `Load_PCM()` | 새로운 QC-LDPC 유사 구성법이나 Prime ECC는 특정 base+circulant PCM 고정 사용 중이라 재검증 부담 큼(이식 난이도 높음, §4 프로파일 기준) |
| shift matrix 저장으로 메모리 절감 | `ecc_data.h` (PCM 구조체) | H-matrix 압축 저장 아이디어는 참고 가능하나 이미 base+shift 구조(`PCM_b`)로 유사하게 압축 저장 중 |
| error-floor 개선(girth 확보) | 대응 없음 (디코더 로직 무관, 순수 부호구조 문제) | 디코더 알고리즘 변경이 아닌 코드 구성 자체의 특성이라 decoder.cpp에 직접 대응 없음 |

> 적용 가치: 중간 — girth 확보를 통한 error-floor 개선은 Prime ECC의 실제 관심사(§5 "관심 높음")와 부합하나, 부호설계 자체를 통째로 교체해야 하므로 이식 난이도가 높고 QC-LDPC circulant 구조와의 정합 검증이 추가로 필요.
