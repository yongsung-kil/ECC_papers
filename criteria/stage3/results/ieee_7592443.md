### B. 알고리즘 요약

1. 대상: 시불변(time-invariant) QC SC-LDPC 컨볼루셔널 부호(SC-LDPC-CC), AWGN+BPSK, sliding-window SPA(최대 100 iter) 복호.
2. 문제: 기존 시불변 SC-LDPC-CC는 시변(time-varying) 대비 성능 열화가 크고, HW 친화 QC·시불변 구조를 동시 만족하는 유연한 부호 부족.
3. 핵심 도구: Bose의 두 BIBD(`(12t+4,4,1)`, `(20t+5,5,1)`)를 완화해 `λ=1` packing 2종 유도 — 임의의 두 객체가 최대 1블록에만 공존(local connection만 유지).
4. 핵심 기법: packing의 base block을 GF(4t+1) 원소 가산으로 순환 생성 → incidence matrix가 CPM(circulant permutation)+ZM 블록으로 구성되어 시불변 CPM-QC, RC-제약(girth ≥6) 만족.
5. 핵심 의미: BIBD의 전역연결을 packing으로 끊어 컨볼루셔널 구조(구속길이 내에서만 CN 공유)를 얻음, `λ=1`이 4-cycle 부재를 보증.
6. 확장: block 제거(=incidence 열 삭제)로 rate 고정 하에 구속길이 확장 등 유연한 파라미터 확보 (Type-I `(4,2ts)`-regular, Type-II `(5,5ts/2)`-regular).
7. 부수: convolutional base matrix `P_packing`가 protograph `P`와 근소 차이지만 성능 우위(EG(3,2^3) CPM 치환 실험으로 확인).
8. 최적화 절차: 없음(조합론적 결정론 구성), cycle counting 알고리즘으로 girth 검증.
9. 결과: 기존 시불변 SC-LDPC-CC 대비 BER 10^-6에서 0.6~1.2 dB 우수, 시변 protograph 부호와 근접(구속길이 클수록 threshold 0.1 dB 이내).
10. 한계: 순수 부호구성 이론(시뮬만), HW·인코더/디코더 설계 없음, rate 0.5~0.93 통신용 SC-LDPC-CC로 NAND block QC-LDPC와 부호계열 상이.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| packing 기반 CPM-QC 부호구성 | `ecc_top.cpp` `Load_PCM()` | Prime ECC는 고정 block QC-LDPC(base+shift)이며 SC 컨볼루셔널 구조 미지원, 재검증 부담 큼 |
| SC-LDPC-CC sliding-window SPA 복호 | 대응 없음 | Prime ECC는 block 단위 min-sum 복호로 window 복호 미지원 |
| BIBD/GF(4t+1) 조합론 base block | 대응 없음 | Prime ECC 부호설계는 특정 H-matrix 고정, 조합론 생성 경로 없음 |
| convolutional base matrix 성능 비교 | 대응 없음 | 컨볼루셔널 특유 요소로 block 부호에 무관 |

적용 가치: **낮음** — 시불변 QC-SC-LDPC 컨볼루셔널 부호구성법으로, block binary QC-LDPC가 고정된 Prime ECC의 부호설계 레이어(이식 난이도 최상)와 부호계열·복호방식(window SPA) 모두 어긋나 이식 가치가 낮다.
