### B. 알고리즘 요약

1. 시스템: 100 Gbit/s급 광통신 coherent 수신기 soft-decision FEC, overhead 25%(rate 0.8), 채널출력 15레벨(≈4bit) 양자화, BPSK/AWGN, target BER 10^-15.
2. 문제: 기존 블록 LDPC는 저 error-floor를 위해 hard-decision outer cleanup code+interleaver 필요 → outer 없이 soft-only로 낮은 error-floor 달성하고 싶음.
3. 핵심: Spatially Coupled LDPC — 무한 확장 parity-check `H_SC`에 convolution형 구조 superimpose, syndrome former memory `μ`, terminated(length `N=Ln`)로 유한화. threshold saturation으로 BP가 ML에 근접.
4. 강결합(strongly coupled): `μ`를 크게(15~42), `m` 작게 → protograph `(μ+1)×c`에서 girth 6 하한 `μ ≥ c·(dv 2)/2`, time-varying shift `s_ij(t)`(주기 T)로 unavoidable cycle 제거해 girth 8~10 확보.
5. 식 의미: `dmin ≤ (m+1)(μm+1)` — 최소거리(정정능력)를 `m` 또는 `μ`로 키움. girth↑ → 작은 absorbing set 억제 → error-floor↓.
6. 약결합(weakly coupled): `μ=1 or 2`, 큰 sub-matrix에 slightly irregular(degree-3+degree-6) 분포로 decoding wave 전파속도(Ireq) 제어. density evolution으로 `av,6=0.2`가 Ireq=1 최적.
7. 구현: protograph를 32×32 circulant(P^s)로 lifting → QC-LDPC화, partially-parallel layered 디코더 + programmable barrel shifter(time-varying shift용).
8. 디코딩: windowed fixed-point 디코더(window w=13~25 copies), window당 I=1~3 iteration, offset/scaled min-sum row-layered. 등가 iteration (w+μ-1)≈14로 NCG 확보.
9. 결과: SC Code B(dv=4,μ=1) 외삽 NCG 11.91 dB@BER 10^-15, SC Code A(I=3) NCG 12.14~12.18 dB. FPGA(Virtex-7) SC μ=2/w=13/I=1에서 ≈1.5 Gbit/s, ≈25k LUT. 동일 복잡도/메모리에서 블록 LDPC·Polar 능가.
10. 한계: windowed 디코더가 블록 대비 메모리(wn) 증가로 throughput↓, 광통신 저rate(0.8) 전제, error-floor 완전제거 위해 post-processing 잔존 가능성 언급.

| offset/scaled min-sum row-layered CN | [decoder.cpp](../Prime_ECC_3.1_Claude/) `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | 우리 min-sum과 동일 계열, offset/scaling만 차이 |
