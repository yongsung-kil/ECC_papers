### B. 알고리즘 요약

1. 시스템: point-to-point AWGN `y=x+z`, 5888 정보비트를 7350 채널사용에 전송(`R_SR-LDPC≈0.80`), QPSK류 아닌 CS 기반.
2. 문제: SPARC는 점근적으로 capacity 달성하나 유한 블록에서 성능 저하, 표준 LDPC도 단독으로는 부족 - 둘을 연접해 유한길이 성능 개선.
3. 모델/가정: inner CS 인코딩 `x=As` (`A_ij~N(0,1/n)`, `n≪qL`), outer는 GF(q=2^m) non-binary LDPC.
4. 핵심기법 1단: 3단 인코딩(q-ary LDPC 인코딩 → LDPC symbol indexing으로 sparse `s` 생성 → `x=As`), 복호는 AMP 복합 알고리즘 식(5)~(7)의 residual/Onsager/denoiser 반복.
5. 핵심식: AMP 유효관측 `r=A^T z+s`가 Condition 1로 `s+τζ`(가우시안)로 근사됨 - denoiser 설계 근거.
6. 핵심기법 2단: BP-N denoiser - 유효관측 `r`로 outer LDPC factor graph 초기화 후 N회(girth 미만) BP 수행하여 state estimate `Ŝ` 갱신, AMP에 재투입.
7. 부수 트릭: non-binary CN 갱신을 Fq convolution으로 표현하고 `q=2^m`일 때 FWHT로 고속화(식 19, Remark 4).
8. 학습/최적화: 별도 학습 없음 - outer LDPC는 PEG로 생성, 가중치는 F256\0에서 균등 랜덤; AMP 25회, i번째 AMP 내 i+1회 BP, 이후 추가 BP 100회.
9. 결과: BER 10^-3에서 기존 SPARC+LDPC[32] 대비 약 1 dB 개선, 5G-NR LDPC+4-PAM iterative 대비도 중간 BER 우수(steep waterfall).
10. 한계: HW 미설계, 시뮬만, 선택 SR-LDPC가 비교군보다 error floor가 더 높음(추가조사 필요), non-binary GF(256) 전용.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| AMP 복합 복호(residual/Onsager) | 대응 없음 | 압축센싱 기반 AMP로 min-sum message passing과 무관 |
| outer non-binary GF(256) LDPC의 BP denoiser | 대응 없음 | non-binary LDPC BP(Fq convolution/FWHT)는 binary min-sum(`decoder.cpp` `CNU_Update_New_Mag()`)과 연산 자체가 상이 |
| inner CS 인코딩 `x=As`(sensing matrix) | 대응 없음 | dual-diagonal QC-LDPC 인코더([encoder.cpp](../Prime_ECC_3.1_Claude/paper_screening_profile.md))와 구조 무관 |

적용 가치: 낮음 - non-binary GF(q) LDPC + SPARC/AMP 연접 패러다임으로, §5 필터의 "non-binary LDPC는 이식성 하"에 정확히 해당하며 Prime ECC의 binary QC-LDPC 디코더·인코더·HW 어느 것과도 대응이 없다.
