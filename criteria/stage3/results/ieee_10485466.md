### B. 알고리즘 요약

1. Product Code(PC)는 `(n,k,t)` BCH 컴포넌트 코드로 구성된 `n×n` 배열 부호, GLDPC(각 VN degree 2, CN degree n)의 특수 클래스로 취급되며 Bi-AWGN+BPSK 채널을 가정.
2. 기존 iBDD(iterative Bounded Distance Decoding)는 낮은 복잡도이나 미보정(miscorrection)으로 성능 저하, 소프트 정보를 결합한 iBDD-SR(scaled reliability)이 개선책이지만 이전 반복 결과를 재사용하지 않음.
3. 채널 LLR `L=2y/σ²`로 계산, BDD 출력은 성공/실패/미보정의 3가지 케이스를 가지며 하드결정 메시지 `ψ∈{0,1}`만 컴포넌트 디코더 간 교환.
4. 핵심 기법(TB-BMP): 반복 `ℓ`마다 memory unit `B(ℓ)`을 두어, soft reliability `u(ℓ)=w(ℓ)ū(ℓ)+β(ℓ-1)w(ℓ)B(ℓ-1)+L`를 계산(`ū(ℓ)`=현재 BDD 출력, `w`=스케일 팩터).
5. `|u(ℓ)|<T`(flipping threshold)이면 `B(ℓ)=-sgn(u(ℓ))`로 "반대 힘(opposite force)"을 부여해 local trap 탈출을 유도하고, 그렇지 않으면 `B(ℓ)=ū(ℓ)`로 이전 결과를 유지.
6. 분석 확장: memory로 인해 표준 DE(Density Evolution) 적용이 불가하여 Bayesian network(DAG, d-separation)로 메시지·memory 간 의존성을 모델링하고 EMP(extrinsic message passing) 제약 하 DE 방정식을 유도.
7. 구현 디테일: 계산 복잡도가 반복수에 지수적으로 증가하는 문제를 memory truncation(`δ=3` 근사)으로 완화, threshold `T`와 스케일팩터 `w(ℓ)`, `β(ℓ)`는 DE 결과 기반 근사식(식 27, 28)으로 산출해 시간 소모적 Monte-Carlo 최적화를 회피.
8. 학습/최적화 절차: DE로 도출한 결정임계값 `T_DE`를 기준으로 하되 실제로는 EMP/IMP 시뮬레이션 기반 최적 `T_opt`를 별도 탐색(이론과 유한길이 시뮬 간 괴리 존재).
9. 결과: (256,239,2) extended BCH 컴포넌트 PC에서 BER=10⁻⁷ 기준 iBDD-SR 대비 0.28dB, iBDD 대비 0.53dB 개선. 복잡도 측면 iBDD-SR 대비 추가 메모리 `n²+q'ℓ` bits, 추가 binary-addition `qn²(ℓmax-1)`회.
10. 한계: HW 설계·gatecount·throughput 전무(순수 알고리즘+DE 이론+소프트웨어 시뮬), 유한 길이 코드(길이-4 cycle 존재)에서 DE 이론값과 실제 성능 간 gap 존재, avg iteration 감소 효과는 언급 없음(오히려 반복수 고정 10+2iBDD 사용).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| memory unit 기반 반복 정보 재활용(threshold flip) | `decoder.cpp` `LDPC_Decoder()` 이터레이션 루프 | HD 디코딩 경로에 이전 반복 결과를 저장·재사용하는 아이디어로 개념적 참고 가능(직접 이식은 구조 차이로 낮음) |
| Product Code(BCH 기반) 부호구조 | 대응 없음 | Prime ECC는 QC-LDPC 전용이며 PC/BCH 구조와 근본적으로 다름 |
| hard-decision 메시지 교환(binary message passing) | `decoder.cpp` `VN_Cal_HD()` | Prime ECC도 HD 모드 지원하므로 개념적 유사성 있으나 CN 연산(BDD vs min-sum)이 상이 |

> 적용 가치: 중간 — 부호 자체(Product Code/BCH)는 Prime ECC와 이식 불가하지만, "memory+threshold로 hard-decision 디코딩에 반복 관성/반대힘을 부여"하는 아이디어는 HD 모드 min-sum 디코더의 bit-flipping 보조기법으로 원용 검토 가치가 있음.
