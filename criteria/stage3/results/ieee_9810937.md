### B. 알고리즘 요약

1. 대상은 column weight `J=3`, row weight `L` 임의인 AWGN용 (3,L) regular QC-LDPC 부호, CPM 크기 `N`(소수).
2. 문제: 반복 디코더의 error floor는 주로 trapping set(특히 ETS)에서 기인하므로 이를 구조적으로 제거하려 함.
3. 패리티검사행렬 `H`를 submatrix `Hi`(2N×LN, 2×L circulant)와 대각 submatrix `Di`(LN×LN)로 조합하여 구성.
4. 각 circulant shift는 정수 수열 `SL=(a0,...,a_{L-1})` 하나로 결정: `Hi`는 `I(a_l)`, `Di`는 `I(2i·a_l)` 사용.
5. 사이클 존재조건 식 `Σ(-1)^{k+1} p_{jk,lk} ≡ 0 mod N`을 이용해 8-cycle 존재조건(식6~14)을 유도.
6. Theorem1: `SL`의 원소가 모두 다르면 girth≥8, 8-cycle은 `Hi` 내부에만 존재. `N`이 소수이고 `N≥2L-1`이 핵심 가정.
7. Theorem2: `SL`이 Golomb ruler이고 `N≥2max{SL}`이면 girth=10.
8. Theorem3: 원소가 다르면 (a≤10, b≤3)인 소형 ETS((5,3),(7,3),(9,3) 및 후속) 전부 제거됨을 ETS evolution 계층으로 증명.
9. 구성 예: C1=(3,5) length-275 (N=11), C2=(3,6) length-468 (N=13); trapping set 탐색으로 (a≤10,b≤3) ETS 부재 확인.
10. 결과: SPA(max 100 iter) BLER에서 기존 방법(C3~C6) 대비 error-floor 영역 성능 우수. 한계: HW 미설계, 시뮬만, column weight 3 소형 예제 한정.
