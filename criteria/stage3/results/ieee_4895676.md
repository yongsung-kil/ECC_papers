### B. 알고리즘 요약

1. 대상은 DVB 계열 표준과 유사한 QC-LDPC 코드(길이 1504, RDF 방식 설계, rate 약 0.89)를 16-QAM으로 변조해 AWGN 채널로 전송하는 시스템.
2. 문제: 고차 변조(M-QAM)에서는 수신 신호와 디코더 메시지(LLR) 양자화를 이진 변조와 다르게 다뤄야 하며, demapper의 clipping 효과가 예기치 않은 error floor를 유발.
3. 채널/모델: AWGN + M-QAM(Gray labeling), MAP demapper가 LLR `L(b_k)`을 계산해 LLR-SPA(sum-product) 디코더에 입력.
4. 핵심 기법 1단: 균일 midtread 양자화(`b`비트, threshold `T`)의 saturation이 클수록 clipping이 줄지만 비트수가 늘어나는 trade-off를 실험적으로 분석.
5. 핵심 식: 로그형 비균일 양자화 `q(x) = sign(x)·A·(exp(|x|/A·ln(1+α))-1)/α` 형태(factor `α`)로 작은 LLR에 조밀한 레벨을 배정해, 적은 비트수로도 clipping error floor를 제거.
6. 확장: 수신 신호 양자화 비트수 `b_r`을 디코더 메시지 양자화 오차와 정합시키는 조건식(식 5~8)을 유도해 demapper 쪽 과잉 정밀도를 방지.
7. 구현 디테일: SNR에 선형 의존하는 근사(max-log 근사, 식 11)로 LUT를 SNR-독립적으로 설계하고, 16-QAM 대칭성을 이용해 LUT 주소폭을 최대 4배(order M) 축소하는 회로(Fig.10~12) 제안.
8. 별도 학습 절차 없음(수치 시뮬레이션 기반 파라미터 튜닝).
9. 결과: 비균일 양자화(factor 적용)가 균일 양자화 대비 훨씬 적은 비트수(예: b=5)로도 error floor 없이 성능 확보, LUT 크기를 최대 M배 축소.
10. 한계: 순수 이론/시뮬레이션 분석(HW 미설계), M-QAM·AWGN 특화 시스템 모델이라 NAND의 이진 채널·min-sum 하드웨어와 직접 연결되지 않음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| 비균일 양자화(clipping 방지 error-floor 대책) | `channel.cpp` `Set_LLR_Th()` | LLR threshold/양자화 테이블 설계에 개념적으로 참고 가능하나 M-QAM 특화 |
| demapper-디코더 양자화 비트수 정합 조건식 | `channel.cpp` `Set_R_Offset()` | soft-read LLR 양자화 설계와 유사한 문제이나 M-QAM demapper 전용 |
| LDPC 디코딩(SPA) 메시지 갱신 | `decoder.cpp` `LDPC_Decoder()` | Prime ECC는 min-sum 기반이라 full SPA/tanh 논의는 대응 약함 |
| LUT 기반 demapper 크기 축소 | 대응 없음 | NAND 채널은 M-QAM demapper 자체가 없음 |

적용 가치: **낮음** — M-QAM 방송 채널의 demapper/양자화 정합 문제로 NAND 이진 채널과 문제 설정이 다르고, HW 미설계·SPA(비 min-sum) 기반이라 Prime ECC로 직접 이식할 요소가 제한적.
