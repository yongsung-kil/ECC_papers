> 총평: Spatial-coupling과 compressed-sensing을 결합해 Gaussian capacity/범용성/짧은 per-user 블록을 노리는 무선 코딩변조 이론 논문으로, LDPC 디코더/부호설계가 아니라 NAND ECC 이식가치 없음.

### B. 알고리즘 요약

1. 시스템: coded compressed sensing `y=Ac+n`, 내부 FEC 부호 `C` + 압축행렬 `A`, 실수 AWGN 채널(Gaussian signaling 목표).
2. 문제: turbo/LDPC/polar가 Gaussian signaling gap·범용성·짧은 블록에서 한계 → 이 세 도전을 동시에 완화.
3. 기법: SC(spatial coupling)와 CS를 결합한 SCCS — `c`를 `W`번 반복·각기 다른 `A_i`로 압축, `K`개 copy를 spatial-couple.
4. 복호: AMP/OAMP(orthogonal AMP)로 신호복원, state evolution `phi(v)`(de-compressor SNR)와 `psi(p)`(decoder output variance) 전달함수로 수렴 추적.
5. 핵심식: potential function `U(v)=∫[phi(v)-psi^-1(v)]dv`의 최소점이 `v=0`이면 error-free (Lemma 3, K,W→무한).
6. rate: `Rc=delta·C_CH - c`, 압축 후 전체 rate `Rcs=C_CH - c/delta` (Lemma 4).
7. 이상부호 정리: 이상 binary `C`에 대해 `delta,Rc→0`(비 고정) 시 `Rcs→C_CH` (Theorem 1) — binary 부호로 Gaussian capacity 달성 가능, random puncture로 범용.
8. 실전부호: concatenated/zigzag-Hadamard(CZH/ZH) 부호, Hadamard sensing matrix + FHT(복잡도 `log2(NH)/bit`), OAMP로 복호.
9. 결과: 시뮬에서 Rcs=0.4902가 Gaussian limit에서 0.7dB, 다중사용자 SCCS-IDMA가 WiMax rate-1/2 LDPC(1152 info bit) 대비 우수, K 증가 시 성능 향상.
10. 한계: HW 미설계·시뮬만, `W→무한`·큰 sensing matrix 가정, 수정 SCCS 구조는 potential-function 해석 미완성.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| SCCS(SC+CS) 송수신 구조 | 대응 없음 | compressed sensing 기반 코딩변조로 QC-LDPC 부호/디코더와 무관 |
| AMP/OAMP + state evolution | 대응 없음 | AMP 계열은 우리 min-sum BP 디코더와 다른 복원 알고리즘 |
| zigzag-Hadamard 내부부호 | 대응 없음 | binary QC-LDPC 아님 |
| potential-function 임계 해석 | 대응 없음 | DE/threshold 이론이나 이식 가능한 코드/HW 산출물 없음 |

적용 가치: **낮음** — SC+CS 기반 무선 다중사용자 코딩변조 이론으로, NAND용 binary QC-LDPC의 부호설계·디코더·HW 어디에도 대응되지 않는다.
