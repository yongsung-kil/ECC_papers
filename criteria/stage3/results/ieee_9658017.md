### B. 알고리즘 요약

1. 시스템: 6G 신뢰성 목표, BI-AWGN 채널에서 5G-NR-LDPC 대비 성능 우위를 노린 time-invariant SC-LDPC 구성법.
2. 문제: 5G-NR-LDPC보다 나은 SC-LDPC를 체계적으로 구성하되 check 행렬 구조는 유지.
3. base 구성: PEG(Progressive Edge-Growth)로 큰 girth의 base matrix `H_0` 생성(주어진 variable/check degree 분포).
4. 핵심 기여(결합 알고리즘): `H_0` degree 불변 상태에서 결합행렬 `H_1`에 coupling 엣지 추가; 제약 - `H_0` 이진 full rank, `H_1`의 1은 `H_y`의 10~20%로 최소화, 각 행당 1엣지 추가, `H_1`/`H_0`의 1 위치 상이.
5. 효과: 새 검증관계 추가로 Tanner 트리 깊이↑ → 제한 iteration 내 confidence 전파 증가로 복호 유리.
6. QC 확장: 결합행렬을 CPM으로 lifting해 목표 부호길이 달성, cyclic shift 계수 도입.
7. 단환 제거: 링존재 정리 `Σ(-1)^l p_l ≡ 0 mod p` 기반, `p1-p2+p3-p4 ≠ 0 mod p`로 4-loop 제거·6-loop 감소(Algorithm 1의 대규모 shift 계수 스크리닝, 좌→우 1회 순차).
8. 인코딩: 윈도우별 3가지 인코딩(첫 윈도는 `H_0`만으로 check code 생성), `G·H^T=0`으로 생성행렬 유도.
9. 결과(시뮬): AWGN/BPSK, BP 20-iter sliding window; rate 2/3 약 0.5 dB, rate 1/2·1/3 약 0.75 dB(N=256) BER 개선, N=128/512에서도 5G-NR-LDPC 대비 waterfall 우위.
10. 한계: 시뮬뿐(HW 미설계), 단코드(N=128~512)·저rate 위주, throughput/복잡도 미제공.
