### B. 알고리즘 요약

1. 대상은 random regular/irregular binary LDPC 부호 앙상블이며, AWGN/BEC 채널에서 반복복호 시 error-floor를 유발하는 Tanner 그래프 부분집합인 trapping set과 stopping set의 점근적 분포를 분석.
2. 풀려는 문제는 특정 부호의 모든 trapping set을 계산적으로 열거하는 것이 일반적으로 계산 불가능(intractable)하다는 점 — 기존 방법(Richardson의 computer-search)은 유한 길이 개별 부호에서만 가능하고 비효율적.
3. 핵심 정의는 `(a,b)` general trapping set(유도된 서브그래프에 홀수차수 체크노드 `b`개), `(a,b)` elementary trapping set(체크노드가 차수 1 또는 2뿐이고 정확히 `b`개가 차수 1)이며, FER은 `sum_{a,b} T(a,b) * P(a,b)`(수식 (1))로 근사됨을 인용.
4. 핵심 기법은 Sanov's theorem(대편차이론)과 지정된 행/열 합을 갖는 0-1 행렬 점근적 개수 정리(`Theorem 2.3`)를 결합해, 파라미터 `(a,b)`가 상수 또는 `N`에 비례해 커지는 두 경우 모두에 대해 trapping set 개수의 점근적 로그값(rate function)을 유도.
5. 핵심 식은 `Theorem 3.1`(regular 앙상블 elementary trapping set 분포)과 `Theorem 4.1/4.2`(irregular 앙상블, Varadhan's lemma로 상한 supremum 형태 유도) — 열/행 가중치 분포 벡터 `f`, `h`에 대한 상대엔트로피 최소화 문제로 귀결.
6. 확장 기법으로 "critical elementary trapping exponent" `κ(f,h)`(수식 (3))를 정의: 이 값 이하 크기의 trapping set은 지수적으로 많이 존재, 이상이면 확률이 0으로 수렴(`Theorem 3.2`) — 부호 앙상블의 error-floor 취약성을 나타내는 단일 지표.
7. 부수 기법으로 Lagrangian multiplier를 이용해 상대엔트로피 최소화 분포를 폐형식(고정점 방정식)으로 도출, 이를 stopping set 분석(Section V)에도 동일 프레임워크로 적용.
8. 별도 학습 절차는 없으며, 수치해석적으로 고정점 방정식을 풀어 특정 rate/degree 앙상블의 spectra를 계산(Section VI, Fig.2-6).
9. 결과: column weight 3, rate 1/3·1/2·4/5 정규 앙상블에서 critical trapping exponent 각각 `κ≈0.35, 0.28, 0.13`(예시), row weight 증가 시 지수 감소, column weight 감소 시에도 지수 감소(즉 error-floor에 더 취약) — capacity에 가장 근접한 `(3,6)` 앙상블이 오히려 error-floor 특성이 나쁠 수 있음을 시사.
10. 한계: 결과가 앙상블 평균이며 무한 길이 점근적(asymptotic) 분석이라 특정 유한 QC-LDPC 부호(circulant/protograph 구조)에 직접 적용하기 어렵고, HW/시뮬레이션 검증이 전혀 없는 순수 조합론적 이론.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| trapping set / error-floor 원인 분석 | 대응 없음 | Prime ECC는 min-sum 디코더 동작(`decoder.cpp`)만 시뮬레이션하며 H-matrix의 trapping set 구조 자체를 분석하는 도구는 없음 |
| degree distribution과 critical trapping exponent 관계 | `ecc_top.cpp` `Load_PCM()` | H-matrix/degree 분포 설계 시 error-floor 취약성 사전 예측 지표로 참고 가능하나 QC-LDPC 특유의 circulant 구조는 논문의 random 앙상블 가정과 다름 |
| stopping set 분석 (BEC용) | 대응 없음 | Prime ECC 채널 모델(AWGN/RBER/fixed-error)은 BEC를 다루지 않음 |

> 적용 가치: 중간 — 특정 알고리즘/HW 이식 요소는 없으나, degree distribution 선택 시 error-floor 취약성을 사전 진단하는 이론적 지표(critical trapping exponent)로서 향후 H-matrix 재설계 시 참고 가치가 있음.
