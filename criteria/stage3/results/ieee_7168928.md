### B. 알고리즘 요약

1. 대상은 이진 대칭채널(BSC, crossover probability `p0`) 기반 QC-LDPC 코드(Tanner code, N=155, K=64, dv=3, dc=5), hard-decision bit-flipping(BF) 계열 디코더.
2. 문제: 기존 GDBF(gradient-descent bit-flipping)는 매 iteration마다 inversion function 최댓값을 갖는 모든 비트를 flip하는데, 이 결정론적 flip이 수렴을 방해해 성능 손실을 유발.
3. 핵심 가정: GDBF의 inversion function `Δ(v_k(i)) = v_i(k) XOR y_i + Σ_{c(j)∈N(v(i))} c(j)(k)`을 최대화하는 비트 집합 중, 확률 `p`로만 일부만 flip하는 PGDBF(Probabilistic GDBF)를 채택([2] 알고리즘 재현).
4. 핵심 기법 1단(LFSR 방식): 각 VN 유닛마다 서로 다른 길이의 LFSR(l=3~10)로 랜덤 이진열을 생성, 임계값 비교로 flip 확률 `p`(최적값 0.9)를 제어 — 그러나 VN마다 로컬 RG가 필요해 등록기(register) 오버헤드 868.4%.
5. 핵심 기법 2단(제안, IVRG): 디코더 내부 CN 출력값(1차 iteration의 unsatisfied 확률 `p(c(j1)=1)=F(p0,1,dc)=1/2-1/2(1-2p0)^dc`)을 재활용, 함수 `G`(선택형 OR 게이트 + shift 체인)로 CN 값을 원하는 확률 `p`의 유사난수열로 변환 — 전용 RG 블록 없이 기존 CN 하드웨어를 재사용.
6. IVRG는 글로벌(공유) 랜덤 생성기 1개로 전체 VN을 커버, LFSR은 VN마다 분산 구현 — 이 구조 차이가 HW 비용 격차(9.7% vs 868.4% register 오버헤드)의 근원.
7. 부수 디테일: comparator tree 기반 maximum-finder는 GDBF/PGDBF 공통 구조 재사용, RG 오버헤드가 throughput에 미치는 영향은 2% 미만.
8. 학습/최적화 절차 없음(고정 알고리즘, FPGA synthesis 기반 설계공간 탐색만 — LFSR 적용 VN 비율 0~100% 스윕).
9. 결과: Xilinx Virtex-6(40nm) FPGA, IVRG-PGDBF는 4114.3→4202.36Mbps(비확률 GDBF 대비 throughput 거의 동일), register 9.7%·LUT 12.1% 오버헤드만으로 waterfall 영역에서 non-probabilistic GDBF 대비 큰 FER 개선; offset min-sum(6bit)은 197.5Mbps로 BF 계열 대비 훨씬 느림.
10. 한계: IVRG는 error-floor 영역에서 LFSR-PGDBF 대비 성능 열화(flattening) 확인, hard-decision·소형 코드(N=155)만 검증, ASIC 합성/실측 아님(FPGA synthesis 결과만).

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| PGDBF 알고리즘(확률적 bit-flipping) | 대응 없음 | Prime ECC는 min-sum 계열만 보유, bit-flipping 알고리즘 자체는 없음 |
| CN 연산(unsatisfied parity 계산) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | CN 유닛 구조는 유사하나 본 논문은 min-sum이 아닌 XOR 기반 parity 검사 |
| VN hard-decision 처리 | `decoder.cpp` `VN_Cal_HD()` | Prime ECC HD 모드와 대상 정합(hard-1bit) |
| IVRG(내부값 재활용 랜덤 생성기) | 대응 없음 | 새로운 HW 블록 개념으로 기존 Prime ECC 모듈에 대응 없음, 단 CN 신호 재활용이라는 아이디어는 GT.cpp의 MUX 기반 신호 재사용 철학과 유사 |

적용 가치: `중간` — min-sum과 알고리즘 패밀리가 달라 직접 이식은 어렵지만, "기존 회로 신호를 재활용해 전용 블록 없이 확률적 요소를 저비용으로 구현"하는 IVRG의 회로 아이디어는 조기 저비용 디코딩 단(HD 모드) 설계 시 참고할 가치가 있음.
