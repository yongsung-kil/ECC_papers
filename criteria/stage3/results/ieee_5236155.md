### B. 알고리즘 요약

1. IEEE 802.16e WiMAX 표준의 systematic irregular LDPC 코드(최대 block length 2304, rate 1/2~5/6)를 대상으로 유연(flexible) 하드웨어 인코더를 설계.
2. 기존 Richardson-Urbanke(RU)법은 parity를 p1, p2 두 단계로 나눠 여러 행렬곱을 순차 수행해야 하며 코드 rate/length 변경 시 재합성이 필요한 구조가 많아 유연성이 부족함이 문제.
3. Gaussian elimination으로 parity check H를 systematic form(G^T=[I_k P^T])으로 오프라인 변환한다고 가정하고, 이 P^T 서브행렬 자체를 메모리에 저장.
4. 핵심 기법: 매 클록마다 `p = P^T s`의 한 행씩을 읽어 message 벡터와 AND한 뒤 XOR-reduce하여 parity bit 1개를 생성(`P_l = Σ H_{l,j} S_j`).
5. 이 식은 RU법의 다단계 행렬곱 대신 벡터-행렬곱 1회로 축소해 parity bit 간 상호의존을 없애고 병렬화를 가능케 함(복잡도 O(p), p=parity 길이).
6. Rate/length 변경은 메모리 내 P^T 재기록만으로 가능(controller가 rate로부터 parity bit 수 `p=n(1-r)` 계산).
7. 최대 block length nmax 기준으로 설계하고 짧은 codeword는 앞쪽을 0-padding하여 동일 구조로 처리.
8. 별도 학습/최적화 절차 없음 (Gaussian elimination은 오프라인 1회 Matlab 수행).
9. Altera Stratix FPGA에서 11,430 LE, 60MHz, throughput 119.7~357.2Mbps(rate 1/2~5/6), 선행 RU 구현 대비 2.5~6배 속도 향상.
10. 한계: 유연성 확보를 위해 P^T 전체를 dense 상태로 메모리에 저장(3.9M memory bits)해야 하므로 메모리 비용이 매우 크고, QC 구조의 순환특성을 활용하지 않음.

### C. Prime ECC 관련 모듈 핀포인트
