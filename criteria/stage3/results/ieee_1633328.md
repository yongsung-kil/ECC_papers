### B. 알고리즘 요약

1. Multilevel Modulated Signal(MMS)을 Discrete Memoryless Channel(DMC)로 전송, `q=2^L` 심볼 constellation, Single-Level Coding(SLC) LDPC 사용. 시뮬레이션은 rate 1/2 regular (3,6)-LDPC, length 10000.
2. 기존 BICM+Marginal BP(MBP) 디코딩은 같은 심볼에서 유래한 비트 간 통계적 의존성을 무시해 MAP 대비 손실 발생, 특히 변조 크기가 크거나 Gray mapping이 불가능할 때 capacity gap 확대(예: 32-QAM quasi-Gray에서 2.6dB).
3. 채널은 memoryless DMC 가정, 심볼 노드 간 사이클이 없다고 가정(subgraph T, depth 2l).
4. 핵심 기법: Coded Modulation BP(CMBP) — 표준 BP의 bit-to-check 메시지 `V_(k,j),i`에 같은 심볼 내 인접 비트로부터의 extrinsic 정보 항 `U^E2_(k,j)`을 추가.
5. `U^E2_(k,j) = log{sum_s Pr[...|s^j_k=0]·exp(...) / sum_s Pr[...|s^j_k=1]·exp(...)}` — 심볼 내 다른 비트들의 확률을 marginalize하여 비트 간 의존성을 명시적으로 반영하는 항.
6. check-to-bit 단계는 표준 BP와 동일(`tanh` 기반), bit-to-check 단계에만 심볼-레벨 extrinsic 항 추가.
7. 부수 트릭: 코드 그래프 구성 시 심볼 간 length-4 cycle 제거.
8. 학습/최적화 절차 없음 — 알고리즘 유도는 그래프 상 심볼 노드 간 무사이클 가정 하 recursive factorization.
9. 결과: 32-PAM AWGN에서 Gray mapping 대비 0.75dB, semi-Gray 대비 1.75dB 개선; 32-QAM quasi-Gray에서 1.7dB 개선(BER 곡선 기준).
10. 한계: 순수 시뮬레이션(AWGN, PAM/QAM 채널)뿐 HW 미설계, regular LDPC만 검증, 심볼 내 사이클 없음 가정이 유한장 코드에서는 근사적일 수 있음.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| CMBP (심볼 내 비트 의존성 반영 BP) | 대응 없음 | multilevel modulation(PAM/QAM) 전용, Prime ECC는 NAND 이진 채널·binary LDPC로 심볼 개념 자체가 없음 |
| check-to-bit 표준 BP 연산(tanh 기반) | `decoder.cpp` `CNU_Update_New_Mag()`, `C2V_Cal_New_Sgn()` | Prime ECC는 이미 min-sum(장방향 근사)으로 tanh 완전연산 대체, 개념적 유사성만 있음 |
| MMS-DMC 채널 모델 / LLR 초기화 | `channel.cpp` `Set_R_Offset()` | NAND read 채널(AWGN/RBER/fixed)과 다른 다중레벨 변조 채널이라 직접 대응 어려움 |

> 적용 가치: 낮음 — multilevel modulation(비이진 심볼) 채널 전용 디코더 개선으로 NAND의 이진 binary LDPC 구조와 근본적으로 다른 채널/부호 모델을 다루어 이식 가치가 낮음.
