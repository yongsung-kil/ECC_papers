### B. 알고리즘 요약

1. 채널: NBNDC-QB(nonbinary queue-based) 모델 - 2q진 출력, M차 마르코프 burst noise, 상관 Rayleigh 페이딩을 근사.
2. 문제: 기존 interleaving은 채널 메모리를 버리고 memoryless로 만들어 코딩이득을 잃음 - 메모리를 활용한 디코딩으로 이득을 얻고자 함.
3. 채널 그래프(state-space)와 코드 그래프(패리티체크)를 하나의 factor graph로 결합, SPA로 joint decoding/채널추정.
4. 핵심 변수: 노이즈 상태 `S_k=(Z_{k-1},...,Z_{k-M})`, 큐 파라미터 `(M, α, ε)`, 상관계수 `Cor_QB`.
5. 핵심 식: LLR `U_n = ln[Σ P_QB(z_n=y_n|s_n)π / Σ P_QB(z_n=ȳ_n|s_n)π]` - 채널 서브그래프에서 코드 서브그래프로 전달되는 메시지.
6. 채널 서브그래프는 forward-backward(α_n, β_n) 메시지로 노이즈 상태 분포를 갱신, 코드 서브그래프는 표준 tanh-rule SPA 사용.
7. 스케줄: 코드 서브그래프 1회 반복 후 채널 서브그래프 1회 반복을 번갈아 수행(균등 시간 배분).
8. 최적화 없음 - QB 파라미터는 실제 DFC(discrete fading channel)에 대해 KL divergence 최소화로 사전 피팅.
9. 결과: regular LDPC (N=15000, K=7500, PEG 생성)에서 BSC 대비 최대 5.8dB 코딩이득(matched decoding, q=2), mismatched decoding에서도 0.57dB 이득.
10. 한계: HW 미설계, 순수 시뮬레이션/이론 결과뿐이며 복잡도가 qM에 지수적으로 증가(저자가 향후 과제로 명시), 무선 페이딩 채널 전용 모델.

### C. Prime ECC 관련 모듈 핀포인트

| 논문 요소 | Prime ECC 모듈 | 관련성 |
|---|---|---|
| Joint channel-code SPA (burst noise state 그래프) | 대응 없음 | NAND read 채널은 독립 오류 가정이라 상태전이 기반 채널 서브그래프 개념 자체가 부재 |
| 코드 서브그래프 tanh-rule SPA | `decoder.cpp` `LDPC_Decoder()`, `C2V_Cal()` | 표준 sum-product 자체는 이미 min-sum으로 근사 구현되어 있어 tanh 회귀는 불필요 |
| non-binary(2q-ary) LLR 양자화 | `channel.cpp` `Set_LLR_Th()` | binary 채널 전용이라 non-binary 출력 알파벳 대응 안 됨 |

> 적용 가치: **낮음** — 무선 페이딩 채널의 마르코프 burst noise 모델링·joint estimation이 핵심이며, NAND flash의 독립 오류 채널 가정과 부합하지 않고 HW 설계도 없음.
